Setting annotations twice causes crash

Hello all,

So I am working on iOS with a map view and attempting to refresh a maps pins when during points of the app and am having major troubles. Here is what I am trying to do:

  1. Create a map view
  2. Add annotations to this map view (between 1 and 5)
  3. Remove all annotations from map view
  4. Load new annotations (some may be the same location)

I haven't made it past step 3 because the iOS simulator keeps crashing when I try and clear the pins. I have used:

  • map.removeAllAnnotations();
  • map.annotations = [];
  • A loop through all of the annotations using map.removeAnnotation()

Each one of these methods removes the pins and the immediately freezes the simulator causing a crash seconds later.

Any help would be appreciated!
Thank you in advance.

  • Just do the following steps:

    //Create Mapview Instance :

    var mapview = Titanium.Map.createView({
    mapType: Titanium.Map.STANDARD_TYPE,
    //region:{latitude:evt.latitude, longitude:evt.longitude,latitudeDelta:0.5, longitudeDelta:0.5},
    animate:true,
    regionFit:true,
    userLocation:true,
    //annotations:[discription]
});

function SetAnnotation(){
mapview.setMapType(Titanium.Map.STANDARD_TYPE);
       mapview.setRegion({
              latitude:lat, longitude:lon, animate:true,
              latitudeDelta:0.5,longitudeDelta:0.5});

       var discription = Titanium.Map.createAnnotation({
                    latitude:lat,
                    longitude:lon,
                    title:" ",
                    //subtitle:' ',
                    pincolor:Titanium.Map.ANNOTATION_RED,
                    animate:true,
                    //leftButton: '../images/appcelerator_small.png',
                    myid:1 // Custom property to uniquely identify this annotation.
                 });

mapview.annotations = [discription];
                mapview.selectAnnotation(discription);
                mapview.show();
}

    You can use SetAnnotation method to set annotation, this approach we use.
