  • Hi

    You have a number of problems here;

    xhr.open('POST', 'http://@/amfphp/Amfphp/?contentType=application/json',callData,onSuccess);

    onSuccess refers to a function, yet this property does NOT accept a function and if it did you have not actually defined it before trying to us it.

    You appear to have two functions serving as the success callback, onSuccess and the actual onload method. You define a callback for the error handler but never set it. You also appear to be setting the content header as a parameter in the query string and not an actual header. Hoping your domain is not as written here and just for example purposes as otherwise - likely problems there as well.

    You are calling the open before you have defined the callbacks., work out which call back you are expecting and then order your code more like this;

    var dado = {"name":"occisor", "pw":"123456"};
var callData = {
    "serviceName": "usersMobile",
    "methodName": "alert",
    "parameters": [dado]
};

var xhr = Titanium.Network.createHTTPClient();
xhr.onload = function()  {
    var json = JSON.parse(this.responseText);  
    alert("Success: " + JSON.stringify(json));
}
xhr.onerror = function (e) {
    alert("Error");
}
xhr.open('POST', 'http://@/amfphp/Amfphp/');
xhr.setRequestHeader('contentType', 'application/json');
xhr.send(callData);

    As I do not know the real URL of the purpose of the data you will need to tweak this - but should move you on.
