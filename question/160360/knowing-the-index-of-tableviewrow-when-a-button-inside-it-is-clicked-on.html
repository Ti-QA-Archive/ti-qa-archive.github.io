Knowing the index of TableViewRow when a button inside it is clicked on

Hi all,

I have a tableViewRows that contains a button, I need to add a new row after the one that contains the button when the button is clicked.

So, in btnPressedFunction, I need to know the index of the row that contains this button, I tried to capture the value of e.index because bubbleParent of the button is set to true, but the problem is that both events are fired in the same thread and the first one fires is the button's event not the row's one.

Here's some code that may explain what I tried to do…

var rowPressed;
tableViewRow.addEventListener('click', function(e){
    rowPressed = e.index;
});

buttonInsideTableViewRow.addEventListener('click', function(e){
    var newRow = Ti.UI.createTableViewRow({ title:"test" });
    myTableView.insertRowAfter( rowPressed, newRow );
});

Note: That's not the exact code I have, I just typed it to explain what I want so, It may contains some spelling Mistakes

When I used that technique, I thought that events are fired in another threads, I did a lot of tests giving results that the TableViewRow Click event is fired after finishing the Button Click event so rowPressed will always remain null.

It's not a must to help me fixing this code, I just want to know the index of the row containing a button when the button is clicked.

Thanks in advance !

  • Its usually a better technique to listen out for clicks on the tableView itself, instead of the rows or elements within the row. You can then use the e.row value to work out which row was clicked

  • Hi,

    you can get row index by using tableview's click event instead of listening of tableviewRow's click event.

    tableView.addEventListener('click', function(e){
    rowPressed = e.index;
    alert(e.index);
});
