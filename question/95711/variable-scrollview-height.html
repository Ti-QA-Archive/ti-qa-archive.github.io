Variable Scrollview height

I use Titanium to make a Disneyland Paris guide; and the attractions views are created with webviews.

  • Regarding your first question. If you are looking for something similar to HTML/CSS's display: fixed property, I don't believe that exists in Titanium.

    However, you can keep updating the position of the blue bar on the window's scroll event. So, something like:

    // assume the blue bar is the object "bar"
win.addEventListener('scroll',function() {
    bar.bottom = <some calculated position>;
});

    You will have to come up with the algorithm to calculate the position. It would be something like the height of the platform (Ti.Platform.displayCaps.platformHeight) plus the window's vertical offset.

I have just finished something similar to this, it took me a little while to figure out, but if you use the 'layout: vertical' (its not in the api docs but works for me on iphone) property on the scrollView, the 'top' property of all the elements inside it become relative to the element above.
