  • Try this sample code:

    var f = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory, 'test.xml');
if ( !f.exists() ) {
    f.write("<items><item>Test 1</item><item>Test 2</item></items>");
}

var xmlStr = f.read().text;
var xmlObj = Ti.XML.parseString(xmlStr);

    See here for how to parse XML.
