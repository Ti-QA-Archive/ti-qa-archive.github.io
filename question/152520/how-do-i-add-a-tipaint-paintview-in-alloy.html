How do I add a ti.paint PaintView in Alloy?

I've tried the below but am getting an error: 'Couldn't find variable: ti at index.js'

Controller:

var paint = require('ti.paint');

View:

<Alloy>
    <Window>
        <PaintView ns="ti.paint" />
    </Window>
</Alloy>

I've also tried this as the view:

<Alloy>
    <Window>
        <PaintView ns="paint" />
    </Window>
</Alloy>

    This isn't how you would include native modules in markup. In fact, native modules in markup isn't officially going to be supported until 1.2.0. When it is supported, it will look like this:

    <Alloy>
    <Window>
        <!-- 
            "module" your UI module's ID.

            "method" is the create method use to create an
            instance of your UI module. "createView" will be 
            used if none is specified
        --> 
        <Module id="paint" module="ti.paint" method="createPaintView"/>
    </Window>
</Alloy>

    If you want to try out the cutting edge version of Alloy, you can always try installing straight from the github repo:

    [sudo] npm install -g git://github.com/appcelerator/alloy.git

    If you find the cutting edge version not to your liking, you can switch back to any released version in a second just by executing the following:

    [sudo] npm install -g alloy@1.1.2

    If you do give the latest version a shot and work with a native module, let me know, I'd like the additional feedback.
