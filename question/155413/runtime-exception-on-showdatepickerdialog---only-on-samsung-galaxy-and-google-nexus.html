Runtime exception on showDatePickerDialog - only on Samsung Galaxy and Google Nexus

Hi,

We are getting runtime exception on click of Date picker dialog. It's happening only on Samsung Galaxy S4 and Google Nexus.

Exception: Uncaught Error:Binary XML file line#20: Error inflating class <unknown>

Source: this.showDatePickerDialog()

Source Code where it throws this error:


if (utils.isAndroid()) {

        var datePicker = Titanium.UI.createPicker({
            type : Ti.UI.PICKER_TYPE_DATE,
            minDate : new Date(2009, 0, 1),
            maxDate : new Date(2014, 11, 31),
            value : new Date(),
            top : '-200dp', // hidden on screen,
            lastValue : null
        });

        //datePicker.hide();

        datePicker.display = function(callBack, currentDate) {

            this.showDatePickerDialog({
                value : this.lastValue ? this.lastValue: new Date(),
                callback : function(e) {
                    if (e.cancel) {
                        //Ti.API.info('User canceled dialog');
                    } else {
                        //Ti.API.info('User selected date: ' + e.value);
                        callBack(e.value);
                        this.lastValue = e.value;
                    }
                }
            });
        }
        return datePicker;
    }

Environment:

Application type: mobile
Titanium SDK:3.1.1.GA
Platform & version: Android 4.2.2
Device: Samsung Galaxy S4 and Google Nexus.
Titanium Studio: 3.0

Any help would be greatly appreciated.
