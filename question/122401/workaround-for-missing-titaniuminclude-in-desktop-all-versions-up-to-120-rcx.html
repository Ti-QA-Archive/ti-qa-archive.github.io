Workaround for missing Titanium.include in Desktop (all versions up to 1.2.0 RCx)

While Titanium.include has been implemented for PHP, Ruby, and Python files, attempting to use it inside Javascript files gives an error message. Here's a workaround I've found that works much better than using eval(). The problem with eval() is that anything you execute inside an included file has a very limited scope, so you can't create functions inside included files and then use those functions later. The solution below works well to overcome the deficiencies in eval():

Ti = Titanium; // globally allow use of Ti, so Ti.include will work as well
Titanium.include = function(url)
{
    var theFile = Titanium.Filesystem.getFile(Titanium.Filesystem.getResourcesDirectory(),url).read();
    try {
         var myFucn = new Function(theFile);
         myFucn();    
        } catch(ex){Titanium.API.debug("INCLUDE ERROR in " + url + ":" + ex );}    
};
