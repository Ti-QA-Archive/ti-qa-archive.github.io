createScrollableView.setViews not working on 3.x.x

I am porting an app from 2.1.4GA to 3.1.0GA and I have run into a small issue with createScrollableView. A call to set the views using createScrollableView.setViews(...) does not result in the scrollable view showing any views. If I add the views as an argument to the createScrollableView call is works just fine.

This doesn't work

var views = [view1, view2, view3];
var view=Ti.UI.createScrollableView();
view.setViews(views)

This does work

var views = [view1, view2, view3];
var view=Ti.UI.createScrollableView(views:views);

    How about

var view=Ti.UI.createScrollableView();
view.views=views;
