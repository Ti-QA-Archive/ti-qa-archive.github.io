Gap between list view items

Is it possible to have a gap between list view items in alloy, like the way we can set dividerHeight property of android list view? I am on 3.3.0 GA

  • There is no divider height property on list items. However, when setting the data for your list, you can pass in a properties object, one of which's properties would be a height. Something like:

    someLabelBindId: { text: 'foo'},
properties: { height: '200dp'}

    You can also use TSS on the template to add padding. Your item components can be positioned ## points from the top and bottom of the cell, etc.
