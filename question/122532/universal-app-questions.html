Universal App Questions

Perhaps I misunderstood, but I thought having a universal app (1.7.1) for iPhone and iPad meant I could have one code base and it would "scale" for both devices appropriately (assuming I didn't conditionalize the code to behave differently) - is that not the case? It seems to do it for KitchenSink, but on my app it just shows the small iphone version on the iPad.

  • ensure you use relative properties (like top/left/right/bottom or percentages) instead of absolute width/height in pixels

    i.e.:

    //fullscreen view
Ti.UI.createView({
    top:0,
    bottom:0,
    right:0,
    left:0
});

  • When you create a project in Titanium Developer, you specify in the project settings whether its for iPhone or iPad. If you've created a new project for iPad and just copied the project files over from an iPhone project, I'm assuming that it would scale except in cases where you have 480 x 320 resources, which wouldn't look nice scaled on iPad anyway.

    Studio might be a bit different, but if you're having an issue getting a universal app to work (never done that myself), you could always go back to Ti Developer and create a seperate project for iPhone and iPad that just shares the same project files.
