Can I get image blob data from a thumbnail?

The API documentation doesn't really say what a thumbnail is, exactly. Code like the following suggests that it's an image:

var thumbnailImage = activeMovie.thumbnailImageAtTime(4.0, Titanium.Media.VIDEO_TIME_OPTION_EXACT);
win.add(Titanium.UI.createImageView({
    image:thumbnailImage,
    bottom:10,
    width:100,
    height:100
}));

However, in my code, when I try and do something like thumbnailImage.toBlob() I get an undefined, not a function type of message. What I'm trying to do is capture a thumbnail and save it to disk (using the GenericImageCache I shared yesterday).

Any thoughts?

  • The image returned from the thumbnail function is already a blob, so there is no need to try and call toBlob() on it.
