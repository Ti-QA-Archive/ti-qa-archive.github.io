    Ahmed,

    After the postlayout event is complete, you can retrieve the width and height of an object using the rect property: rect.width and rect.height.

    So, it would be something like this in code,

    label.addEventListener('postLayout', function(e) {
    image.width = label.rect.width + 50;
});

    Ray
