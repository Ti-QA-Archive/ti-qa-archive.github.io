Script Error = invalid method '(null)'

Hi all.
I have the following error in my app.

[ERROR] Script Error = invalid method &#39;(null)&#39; at testTabs.js (line 3).

Line 3:

var window = require(&#39;views&#x2F;main&#39;).setWindow();

File views/main.js:

exports.setWindow = setWindowFunct;

function setWindowFunct()
{ 
    var win = Titanium.UI.currentWindow;
    var window = null;
    if (!win){
        window = Ti.UI.createWindow({ title:'Main',    backgroundColor:'#FFF'});
    }else{
        win.title            = 'Main';
        win.backgroundColor = '#FFF';
        window                = win;
    }

}

This app works fine on 2 different computer (one with mac and one with ubuntu) and on another one (with mac) throw this error.

I hope any one can help me with this issue.

  • You should stay away from variable names like window and function names like `setWindow', because they most likely conflict with other Ti indentifiers. Specifically, Titanium states that function names that start with "set" and "get" should be avoided because they map to native functions.
