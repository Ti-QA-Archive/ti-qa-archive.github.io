Window is opening only sometimes

I have the following code:

$.jobCompanyView.addEventListener("click", function(){
    var companyDetailsWindow = Alloy.createController("companyDetails", { job: job}).getView();
    companyDetailsWindow.open();        
    $.lblCompanyLink.color = "#3c3c3c"; // simulate click by changing the font color
    setTimeout(function(){ $.lblCompanyLink.color = "#898888"; },  100);
});

When I click the view, you see that I change the color of the text label inside, as a way to let the user know they clicked the link. That works on every click, so the event fires correctly.

However, the companyDetailsWindows opens only 1 out of 5 clicks maybe, really randomly.

What can be wrong here? Looks to me like that's a Titanium bug.
