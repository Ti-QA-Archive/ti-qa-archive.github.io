Orientation Modes

Hi,
I have an iphone /android app that I have fixed in portrait mode in the main window

I have a few videos that I want show but want to enable landscape orientation for just those windows is there a way to do this? for both android and iphone but return to fixed portrait mode?

regards

— asked March 20th 2012 by derek johnston

  • just do win.orientationModes = [Ti.UI.PORTRAIT]; or any of the following constants

    Ti.UI.PORTRAIT
Ti.UI.UPSIDE_PORTRAIT
Ti.UI.LANDSCAPE_LEFT
Ti.UI.LANDSCAPE_RIGHT

    • and if you need more than one orientationModes takes and array

      win.orientationModes = [Ti.UI.PORTRAIT, Ti.UI.UPSIDE_PORTRAIT];
