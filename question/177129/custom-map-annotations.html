Custom Map Annotations

The app that I am working on has a map view with annotations on it. The annotations represent doctors at as location. There can be more that one doctor at a location. Is it possible to have an annotation bubble with a list of the doctors at that particular location with a listView of tableView. If not, are there any other possible solutions to this problem. Thanks.

  • Yes, if you look to the docs, you can specify a customView instead of a title for an annotation. Note that events are not supported within that custom view, however. So, users won't be able to tap a row in your table. Also, I haven't tested this but I'm going to guess that the annotation height is probably fixed (so a tall table won't fit).
