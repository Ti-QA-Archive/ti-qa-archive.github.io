  • I think the 'answer' is that Titatnium does not offer API level support for graphics at this time (even incredibly trivial drawing commands). You can use Javascript and an HTML5 canvas, and that gets the job done for basic stuff, but unfortunately it's not enough since I then need to output the contents of the WebView as an array for further process (or otherwise generate an image from the view that can be sent off to a server).
    It's basically a deal breaker. Very lame. I'd think that getting basic drawing up and running would be a matter of a couple weeks of work for a decent developer- can't believe it was overlooked, especially given the volume of questions on this topic in the forum.
