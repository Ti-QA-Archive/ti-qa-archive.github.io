  • For delivering of php files you need a server which can do this. A classic solution is apache server. Maybe it is better to write:

    <?
header('Content-type: text/json');
echo '{"users":[{"usre":"bob","pass":"bob"},{"user":"alice","pass":"alice"}]}';
?>

    Normally you retrieve mySQL Dazabase, fill an array/object and in the end you send the php stuff by json_encode(); to client. You have a type in joer json (user/usre). In the end is your code OK, but I would catch the JSON parse error in a try-catch-block.
