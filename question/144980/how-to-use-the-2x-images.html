    Hi

    As far as I know, Titanium Studio does all the work for you, when using images. If Titanium finds the @2x file of the image then it will be used instead of normal resolution when you app is running on devices with retina.

    If you use @2x.png files by default, then I'm not sure that your app will look good on iPhone3 and other non retina devices.

  • Can you clarify what you mean by: "This method of images with suffix @2x can not be used even for images internal to the app? If yes, how? thanks"

    On iOS, any image that ends in the suffix "@2x" is deemed to be a retina image, and will be used on devices with retina screens. So if you have an image called header.png that is 320x40px, then the file header@2x.png should be 640x80px and it is this 2x image that will be used in all retina devices.

    Make sense?
