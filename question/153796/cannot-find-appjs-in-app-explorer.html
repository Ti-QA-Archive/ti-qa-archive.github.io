Cannot find app.js in App Explorer

Hi. I'm new to using Titanium, and the initial README file tells me that to begin editing my app (targeted to Android), I should edit app.js. I tried finding app.js through the App Explorer pane, but I couldn't find it. Eventually, I did a search through Windows Explorer, and it told me that app.js was located in the Resources folder, a folder that does not appear in the Package Explorer. I opened the app.js file from Windows Explorer in Titanium and tried to edit it. As soon as I hit enter, I got a pop up saying, "This is a derived file. Do you really want to edit it?" I'm guessing that I shouldn't be opening the file through Windows Explorer, but I'm not sure why the Resources folder isn't appearing in App Explorer (or Package Explorer), and I'm wondering how to make it appear, and whether app.js is really what I should be editing.
