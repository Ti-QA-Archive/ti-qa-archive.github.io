    Since you've given almost no details, here's a very generic way to do exactly what you are asking:

    index.xml

    <Alloy>
  <Window>
    <TableView id="table"/>
  </Window>
</Allo>

    index.js

    var names = ['name1', 'name2', 'nam3'];
for (var i = 0; i < names.length; i++) {
  $.table.add(Ti.UI.createTableViewRow({
    title: names[i]
  }));
}

    I'm betting this doesn't completely answer your question, but based on the info given this should be a start.
