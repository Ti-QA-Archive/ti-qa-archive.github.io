Come back to the first window (root window) in a Titanium.UI.iOS.NavigationWindow

Hi, I open various windows under a Titanium.UI.iOS.NavigationWindow: sometimes, they are a fourth or first "level" starting from the first window. How to I directly come back to the first Window -the unique root of the entire Titanium.UI.iOS.NavigationWindow- so that the user has not to click four times a back button? Any help is appreciated.

  • Another way would be to keep your root window out of the NaviagationWindow (open it as simple window rather than through NavigationWindow). From the root window onwards open all other windows in navigation window and simply close the navigation window on "Home" button press. All windows will be closed and root window will be visible. I've done this recently in my project. See if this works for you too.
