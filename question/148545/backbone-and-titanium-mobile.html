Backbone and Titanium-Mobile

What are the steps involved in using Backbone in a titanium mobile app?
I searched the docs and found this: Backbone Objects without Alloy

But when I try to create some dummy models and collections as shown the code below, I get an error in the iOS simulator: [ERROR] : Script Error = Can't find variable: Backbone at app.js
Even after inserting the require(&#39;Backbone&#39;) statement

// Initialize a collection class
var collection = Backbone.Collection.extend({
    comparator: function(model){
        return model.get('title');
    }
});

// Create a new collection
var library = new collection([
    {title: 'To Kill a Mockingbird', author:'Harper Lee'},
    {title: 'The Catcher in the Rye', author:'J. D. Salinger'},
    {title: 'Of Mice and Men', author:'John Steinbeck'},
    {title: 'Lord of the Flies', author:'William Golding'},
    {title: 'The Great Gatsby', author:'F. Scott Fitzgerald'},
    {title: 'Tom Sawyer', author:'Mark Twain'},
    {title: 'Animal Farm', author:'George Orwell'}
]);

// Initialize a model class
var model = Backbone.Model.extend();

// Create a new model and add it to the collection
var book = new model({title:'Bossypants', author:'Tina Fey'});
library.add(book);

// Remove the very first model from the collection
model = library.at(0);
library.remove(model);

    I'm working on the same thing. Backbone without Alloy.

    You need to require both underscore.js and backbone.js like this:
    var = require("underscore").;
    var Backbone = require("backbone");

    Notice however that you need to write a new Backbone.sync method because the native one uses Ajax calls.

  • Thanks to Erik, got it working by:

    1. From the Alloy source, copied backbone.js and underscore.js into a lib folder
    2. Modified backbone.js to find the correct underscore
      require(&#39;alloy&#x2F;underscore&#39;) to require(&#39;lib&#x2F;underscore&#39;)
    3. Require them:
    var _ = require('lib/underscore')._;
var Backbone = require('lib/backbone');
