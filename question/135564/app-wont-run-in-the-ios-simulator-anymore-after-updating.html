  • If you haven't uninstalled xCode 4.2, downgrade to this version. Configure xCode 4.2 as default, as explained here, then configure Titanium studio correctly (select Titanium Studio preferences > Aptana Studio > Titanium > IOs SDK Home). This would allow you work with 1.8.2 or prior versions again.

    If you don't want to downgrade, you can try building latest 1.8.3 version available here, but not sure if will work. Of course, clean build after each change.
