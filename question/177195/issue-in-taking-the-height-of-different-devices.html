Issue in Taking the Height of different devices

Hi All,

In my project i want handle the Views Top, Height Dynamically for that i want to get Device Height,

I am using below code for getting height

Titanium.Platform.displayCaps.platformHeight

for iOS i getting nice like for 5 inch i am getting 568 which is height of device and i am able to work fine.

But when i am trying to get for Note 2 Andriod i am getting 1280, I think that is resolution, How can i get exact height.

    Try this:

    var deviceHeight = parseInt(Ti.Platform.displayCaps.platformHeight / (Ti.Platform.displayCaps.logicalDensityFactor || 1), 10);

    Hope this helps.
