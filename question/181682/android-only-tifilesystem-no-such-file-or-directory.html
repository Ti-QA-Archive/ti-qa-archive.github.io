  • Here is sample code,

    
    var skipTourFile;
        if(Ti.Filesystem.isExternalStoragePresent()) {
        skipTourFile = Ti.Filesystem.getFile(Ti.Filesystem.externalStorageDirectory,'skiptourusernames.txt');
        } else {
        skipTourFile = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory,'skiptourusernames.txt');  
    }

        if(skipTourFile.exists()) {
        var skipTourFileContents = skipTourFile.read();
        }

    Let me know if you are still facing the issues.
