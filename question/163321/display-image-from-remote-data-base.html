  • Accepted Answer

    Hi

    If you are returning a byte array in the JSON feed then you are going to need to reconstruct the data in the app yourself.

    I must say that returning image data as a byte array in a JSON feed in not very common. It is more common to return the image as a path to the physical file stored on the server. This allows so many performance benefits they are hard to count.

    If your image is stored inside the column then the smarter move would be to pass the image data back as base 64 encoded and then recode from the base 64 on the app.
