  • THIS IS HOW I SOLVED THIS PROBLEM

    The module that was included in this project had to be rebuilt with architecture armv6 instead of armv7.

    The steps to rebuild it are as follows:

    • In Xcode 4.2 open the module project file
    • In Build Settings under Architectures click more and add armv6
    • Save and close the project
    • Rebuild the module using the build.py script in the module folder
    • Replace the current module in the Titanium Module directory
    • Rebuild in Ti Studio
