how to dynamically create Annotations

Hello,

I can't see Annotation dynamically with alloy

<Alloy>
    <Window id="index">
        <View id="mapview" ns="Ti.Map" animate="true" regionFit="true" userLocation="true" />
    </Window>
</Alloy>

$.index.open();

$.mapview.region = {latitude:53.5686, longitude:10.0386, latitudeDelta:0.01, longitudeDelta:0.01};


$.mapview.addAnnotations(
 Ti.Map.createAnnotation({
        latitude:54,
        longitude:10,
        title:'xxxx',
        subtitle:'zzzzz',
        pincolor:Ti.Map.ANNOTATION_RED,
        animate:true,
        id:'new01'
    })

);

    Hi

    You almost had it. You have used the method addAnnotations (plural) but you only provided a single annotation.

    You have two choices;

    Singular

    $.mapview.addAnnotation(Ti.Map.createAnnotation({
        latitude:54,
        longitude:10,
        title:'xxxx',
        subtitle:'zzzzz',
        pincolor:Ti.Map.ANNOTATION_RED,
        animate:true,
        id:'new01'
    })
);

    Plural

    $.mapview.addAnnotations([ Ti.Map.createAnnotation({
        latitude:54,
        longitude:10,
        title:'xxxx',
        subtitle:'zzzzz',
        pincolor:Ti.Map.ANNOTATION_RED,
        animate:true,
        id:'new01'
    })
]);

    If you use addAnnotation you only pass one.

    If you are using addAnnotations then you must provides annotations in an array even if you only have one.
