  • Build your project for the emulator.

    Copy the AndroidManifest.xml from {project}&#x2F;build&#x2F;android to {project}&#x2F;platform&#x2F;android

    You have now created a custom AndroidManifest.xml file.

    Now edit your new AndroidManifest.xml:

    For every activity:

    <activity android:configChanges="keyboardHidden|orientation" android:label="Crestwood" android:name=".CrestwoodActivity" android:theme="@style/Theme.Titanium">

    add

    android:screenOrientation="portrait"

    as so

    <activity android:screenOrientation="portrait" android:configChanges="keyboardHidden|orientation" android:label="Crestwood" android:name=".CrestwoodActivity" android:theme="@style/Theme.Titanium">

    This is a sample from a project here, so your activity names will be different
