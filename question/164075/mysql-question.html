  • If you want to select the last record, change the order and limit to 1 as per:

    SELECT * 
FROM  `felfe_love`, felfe_friends 
WHERE (felfe_friends.facebookid = felfe_love.UserId) AND (felfe_friends.id = "999") 
group by felfe_love.UserId
ORDER BY  `felfe_love`.`datefav` ASC
LIMIT 1
