Pinterest like display with imageview doesn't work. Any solutions?

Hi,
I am trying to display images like pinterest, but I am running into an issue. if the container of the imageView has either an 'auto' height or a vertical layout, it won't display the imageview.

var win = Ti.UI.createWindow({});
var container = Ti.UI.createView({
    width: 100,
    height: 'auto',// 1000,
    //layout: 'vertical'
});

var picture = Ti.UI.createImageView({
    backgroundColor: 'red',
    image: 'IMG_0036.jpg',
    //width: 100,
    //height: 140,
    //canScale: true,
    top: 2
});

container.add(picture);

win.add(container);

win.open();

Test out this code. You will see. Change the height to 1000. It will work. Change the layout to vertical. It won't.

Any suggestions?

Thanks
