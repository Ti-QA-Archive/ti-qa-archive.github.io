    Hi Ronny, you are not calling openWin1 function, you are just defining it.
    Just call it:

    function openWin1(){
   var Window1 = require('win1');
   var win1 = new Window1();
   win1.open();
}

openWin1();

    or simply

    var Window1 = require('win1');
var win1 = new Window1();
win1.open();

    Both should work. Hth.
