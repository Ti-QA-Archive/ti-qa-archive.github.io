  • Owen

    The Remote Images test in the KS works for me, so I am not sure what is causing your issue. I am using ubuntu also. Note that Android SDK 2.3 is not yet supported, so you must use 2.2 or lower.

    Here are some steps to follow:

    Download and install the latest 1.5.X SDK from the Continuous Builds page. Use these instructions if you need them.

    Close your emulators, delete the contents of the subdirectories of the $PROJECT&#x2F;build&#x2F;android&#x2F; directory. Hence, $PROJECT&#x2F;build&#x2F;android&#x2F;[subdir]&#x2F;*.

    Restart TiDev

    Set the new SDK on the Edit Project screen, press the "Save Changes" button to ensure that tiapp.xml is touched and hence the project rebuilt.

    On the Run emulator screen, ensure that SDK "APIs 2.2" is selected, and launch your emu.

    Hope this helps
