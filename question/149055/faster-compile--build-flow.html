Faster Compile / Build Flow

Using the Titanium IDE is painful with testing. I generally see 40-60s on build times, depending on which system I use, plus another 15-25s for the iTunes transfer. If you compile 20 times an hour, you're wasting 33% of your hour waiting on the compile. For me, this is unacceptable as twiddling thumbs is not the best way to spend my time.

So…I'm looking for ideas on how to better improve my Ti workflow so I can regain my ~33% loss.

Any suggestions?

Something I've tried is building directly through XCode but this iPad only app (only deployment target checked in tiapp.xml) shows up as a 2X iPhone app. It does speed up the time to device but I still have to go through the 40-60s build time.

Any ideas are welcomed. Thanks!
