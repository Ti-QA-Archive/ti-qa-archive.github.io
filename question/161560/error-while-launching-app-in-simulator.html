  • Try compiling from the command line, using an older version of iOS. For example, to compile in iOS 6.1:

    sudo titanium build --platform ios --ios-version 6.1

    Run this command from the root of the Titanium project.

    If it says that iOS version 6.1 is not valid, you can download iOS 6.1 from Xcode / Preferences / Downloads tab.
