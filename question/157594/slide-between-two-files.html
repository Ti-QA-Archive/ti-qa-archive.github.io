  • Ok. I have two files.

    I can make a slider using Ti.UI.createScrollableView like this:

    var scrollableView = Ti.UI.createScrollableView({ views:[view1,view2,view3], showPagingControl:true });

    But the code above allows me to do that only with views.

    I want to know if there is any way to do the same but with two or more .js
