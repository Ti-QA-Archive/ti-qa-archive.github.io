How to completely uninstall Titanium Studio so a fresh install is done?

After uninstalling Titanium Studio in an effort to get it to run the "hello world" demo with error, I was surprised to see that the user files had not been deleted; though it sort of makes sense too. When uninstalling Titanium Studio on Windows XP-SP3, beside doing a remove program the the control panel, and removing the "Titanium Studio Workspace" files under "My Documents", what else do I need to do to completely remove all files so that I know I'm doing a fresh install? (Note: This does not include, nor is it intended to included the removal of files related to the Java-SDK or Andriod-SDK.)

System Profile:

  • Host OS: Windows XP
  • OS Arch: x86
  • Install Directory: file:/C:/Program Files/Appcelerator/Titanium Studio/
    Version: 2.0.1.201204132053
  • Workspace Directory: file:/C:/Documents and Settings/user/My Documents/Titanium Studio Workspace/
  • Titanium Desktop SDK Location: C:\Documents and Settings\user\Application Data\Titanium\sdk\win32\1.1.0
