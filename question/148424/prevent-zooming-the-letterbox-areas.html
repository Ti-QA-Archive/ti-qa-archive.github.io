Prevent zooming the letterbox areas

I have a scrollview, let's say it's in portrait orientation. I have an imageView inside it, which fills the entire screen. The image inside the imageView is in landscape, so there are transparent areas below and over the image. This is fine, until you start to zoom: scrollView expands the whole content area, with the blank letterbox areas. You can scroll the scrollView to the top in the way that your image is not visible, only the letterbox area over it. How can you prevent this? I want it to be able to zoom only the image, not the blank areas, so if the user zooms the image, and it's height is greater than the scrollView's, the blank areas should be gone.

  • Any Titanium version (including 2.x and 3.0), iOS platform (Android is not affected because it does not have 2-direction scrollviews at all). Code snippet:

    var scrollView = Ti.UI.createScrollView({
    width: 320,
    height: 480, // portrait
});

var imageView = Ti.UI.createImageView({
    image: 'url' // landscape image,
});

scrollView.add(imageView);
