  • hi Gopinath

    About your question "How to disable Zoom Functionality in webview"

    Put this line in html file <head> portion.
    <meta name = "viewport" content = "user-scalable = no">

    This you have to figure out how you add it, if the source coming from other webpage you can search for head tag and append this line in the code.

    Hope this solve your problem.
