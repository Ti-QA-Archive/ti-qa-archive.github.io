iOS UrbanAirship Push Notification sound only

I'm running into a really strange issue that I'm not sure how to solve: my app is using the UrbanAirship push notification module, and it's working perfectly on the first iPod device that I tested it on, but on the second iPod device UrbanAirship is only registering for sound notifications.

I thought that maybe it was something to do with this code:


    Ti.Network.registerForPushNotifications({
        types:[Ti.Network.NOTIFICATION_TYPE_BADGE |
               Ti.Network.NOTIFICATION_TYPE_ALERT |
               Ti.Network.NOTIFICATION_TYPE_SOUND],

But no matter what I change that to be my first iPod always registers for all three notification types and my second iPod only registers for sound notifications. What am I doing wrong?
