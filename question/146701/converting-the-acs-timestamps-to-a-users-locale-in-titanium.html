Converting the ACS timestamps to a users locale in Titanium

ACS returns timestamps for created at stored in UTC, ie. 2013-01-10T11:29:01+0000.
While I could implement a function which creates a JS Date object from this by using substr for year, month and day and then converting this to the users locale using String.formatDate(), it seems to be a rather complicated approach to something very very common.

I have seen a few 3rd party library references but these are older posts. Have I missed something? Is there a core way to convert the dates returned from ACS to local readable formats in Titanium?

Any help appreciated.
