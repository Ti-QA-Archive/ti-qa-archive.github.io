    Hi

    You have not specified your target platform, so here is the answer for both;

    • iOS = Yes; use scrollToIndex, but use a pixel value in the brackets - for example;
    tbl.scrollToIndex(100);
// moves the table content position 100 down
    • Android = No; you can only set things based on row indexes
