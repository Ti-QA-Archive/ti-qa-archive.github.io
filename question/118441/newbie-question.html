Newbie question....

Hi, i have been pointed to appcelerator by a friend. I need to build an app that I can run on iPhone and Android mobile devices.

Can appcelerator let me do this???

Do I simply design the app then compile for different platforms???

thanks

  • Yes, that is correct, Harry, developing with Titanium enables you to have one codebase that builds versions of your app for both Android and iOS. Be aware, however, that users of the respective platform expect a different style of user interface, so while much of the logic of your application will be the same, there will still be some slight differences in the look and feel of the UI. Furthermore, although most of the functionality is common to both platforms, there is some platform-specific functionality too, detailed in the API docs under Titanium.UI.iPhone, Titanium.UI.iPad and Titanium.UI.Android etc. Also look out for notes in the API docs that state "Android only" or "iPhone only" etc.

    The best place to learn more is in the Documentation Guides, starting at How does Titanium work? and Designing the User Interface. Then work through the articles in the Getting Started drop-down menu of the Guides Homepage.

    Good luck! :)
