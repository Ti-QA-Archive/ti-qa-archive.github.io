How to change the ActionBar Tab Text color When using the AppCompat Library ?

I'm in Titanium SDK version 3.3.0.GA and i have customized my Android Action Bar Style using the generator and i set the Style compatibility to appCompat as it's recommended for that version.

In both this android documentation page about Styling the Action Bar and also this question there are only solutions to customize the ActionBar title Text color when using parent="@style/Theme.Holo.Light"or parent="@style/Theme.Holo" for the custom style but as i said i have parent="@style/Theme.AppCompat.Light"

  • This question helped me.

    In res/values-14/styles_myCustomStyle.xml , i added : 

    <item name="android:actionBarStyle" >@style/ActionBar.Solid.myCustomStyle</item>

    to the style tag with name="Theme.myCustomStyle"

    and 

    <item name="android:titleTextStyle">@style/ActionBar.CustomTitle</item>

    to style tag with name="ActionBar.Solid.myCustomStyle"

    and this style tag : 

    <style name="ActionBar.CustomTitle" parent="@style/TextAppearance.AppCompat.Widget.ActionBar.Title">
        <item name="android:textColor">@color/actionbar_text</item>
        <!-- The textColor property is backward compatible with the Support Library -->
 </style>

    and to the res/values/styles_myCustomStyle.xml i added the same but i removed all ' android: ' prefixes except the ' android:textColor ' in the last style tag ' ActionBar.CustomTitle '

    and i added the colors.xml to both res/values-14 and res/values directories with the following code : 

    <resources>
    <color name="actionbar_text">#ffffff</color>
</resources>
