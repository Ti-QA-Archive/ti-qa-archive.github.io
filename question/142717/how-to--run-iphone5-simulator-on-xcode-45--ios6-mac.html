How to : Run iphone5 simulator on Xcode 4.5 / iOS6 Mac

Installed Xcode 4.5 to test the new iphone 5 simulator and like always, you can't set different device using titanium studio unless you launch the simulator manually first, set the desired simulator device, close it then launch your the studio.

This time i can't find the newly installed iPhone simulator using spotlight, so here's location just in case anyone run into the same issue.

  • Go to ~/Applications, and right-click Xcode and select “Show Package Contents”
  • Browse to “Contents/Developer/Platforms/iPhoneSimulator.platform/Developer/Applications/iOS Simulator.app”
  • Launch the app (Make an alias/short cut to desktop if you like.)
