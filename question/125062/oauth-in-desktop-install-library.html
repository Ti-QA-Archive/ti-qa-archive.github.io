  • To install php-extensions and use dl()-function Iuse this in my php-code:

    $file = 'app://dll/php_imap.dll';
$newfile = 'C:\\php5\\php_imap.dll';
mkdir('C:\php5');
copy($file, $newfile);

dl('php_imap');

    For me it works fine.

    Maybe, you findout, where is the right dll-location at the different plattforms.
    Hope, it helps
