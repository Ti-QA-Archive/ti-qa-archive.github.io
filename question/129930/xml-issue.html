  •  i dont know how to get same row attribute data

    in the event listeners, you can catch the data coming off the row:

    mytableview.addEventListener('click', function(e){
    var mymanufacturer = e.row.manufacturer;
    var myproductname = e.row.productname;
}

    This assumes that when parsing the xml, you added the data to each of the rows:

    yourrow.manufacturer = xml.parsingcall.to.get.manufacturer;
yourrow.productname = xml.parsingcall.to.get.productname;
