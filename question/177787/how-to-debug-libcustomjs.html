How to debug lib/custom.js

I've added a lib/customfoo.js

Inside it does exports.createCustomFoo = function(args) { … }

Inside my xml I put <CustomFoo id="foo1"></CustomFoo>

Clean, delete build folder, run the project:

[WARN] : Attempted to load TiUICustomFooProxy: Could not find class definition.

Then my app goes red & complains: invalid method (createCustomFoo) passed to UIModule at controllercode.js (line 43). Line 43 doesn't exist (it's in the build product temp file, apparently, and I the path in the console is no longer there.)

How to debug this?

  • It looks like you are trying to include a reference to code inside a view xml controller. When you reference something from an XML view, you need to return a UI object that is embedded in the view. Check out the XML Markup for Alloy.

    The file located in lib/customfoo.js likely doesn't include the view. Even if it does, you need to reference an external view using the require in your xml file.

    Ray
