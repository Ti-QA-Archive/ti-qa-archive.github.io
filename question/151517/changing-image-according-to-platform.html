Changing image according to platform

Hi everyone!

I'm trying to change the image of my app according to the device, mobile or tablet. What am I doing wrong?

"#header[platform=ios formFactor=handheld]" :{
    image:"/images/header.png"
}
"#header[platform=ios formFactor=tablet]" :{
    image:"/images/header-ipad.png"
}

Thanks!

    Actually, you don't need to do anything special like that. Define your image as normal:

    '/images/header.png'

    Then, in your Resources/iphone/images directory, create as many of these files as you need:

    • header.png (for non-retina displays)
    • header@2x.png (retina 3.5" displays)
    • header-586h@2x.png (retina 4" displays)
    • header~ipad.png (ipad)
    • header~ipad@2x.png (ipad retina)

    With the exception of the 4" retina display, this is an iOS-level feature. The retina display file name pattern is a Titanium SDK feature.

    Similarly, on the Android side, you would have all of your various sized images named "header.png" but put them into the proper sub-directory under Resources/android/images/
