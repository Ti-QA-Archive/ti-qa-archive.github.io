  • Distribute to who? And which platforms?

    If you want it public, then iOS applications must be done through the Apple App Store. For Android, it would be the Android Marketplace.

    If you want to distribute iOS apps to a limited number of testers, then you can add their device UDIDs into the provisioning profile (which must be made on the apple webesite). The *.ipa can then be distributed using something like testflightapp.
