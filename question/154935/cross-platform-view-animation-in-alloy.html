Cross Platform View Animation in Alloy?

I'm trying to show some kind of animation (slide-in, fade-in, whatever) when clicking on a button to go from one view (index.xml) to another (logon.xml). It needs to work in iPhone and Android. I've tried the following code that gets called on button click on index.xml:

var win = controller.getView();
win.open();
