2.1.0: Problem with tableview and Ti.UI.SIZE

When using a table view (with height set to Ti.UI.SIZE) inside a view (with height set to Ti.UI.SIZE) the tableview is not displayed.

When using a table view (without height setting) inside a view (with height set to Ti.UI.SIZE) the view is displayed till the end of the screen (too much height).

The problem occurs since Titanium SDK 2.x.

Tested on kitchensink table_view_layout_3.js by adding the following lines of code at the end:

var vwTV = Ti.UI.createView({
width : '80%',
height : Ti.UI.SIZE,
left : '10%',
layout : 'vertical',
backgroundColor : 'white',
opacity : 0.8,
borderRadius : 5,
top : 100
});

vwTV.add(tableview);
win.add(vwTV);
