  • You should use string format function. "The String object in Titanium contains a format function which formats strings according to the IEEE printf specification.". It should be

    var formatted = String.format(L(&#39;ordered&#39;), &#39;nico&#39;, &#39;nico2&#39;);.

    Check Replacing values in a localized string section https://wiki.appcelerator.org/display/guides/Internationalization#Internationalization-Replacingvaluesinalocalizedstring
