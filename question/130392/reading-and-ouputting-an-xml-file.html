Reading and Ouputting an xml file

Hi I have the following xml file:

brand.xml

<?xml version="1.0" encoding="UTF-8"?>
<brands>
    <store>Brand A</store>
    <store>Brand B</store>
    <store>Brand C</store>
</brands>

I am trying to output the contents of this file to the console screen for now.

//load local xml file
var file = Ti.Filesystem.getFile(Ti.Filesystem.resourcesDirectory,'brands.xml');

// Check if file exists
if(file.exists()){
    var xmltext = file.read().text;

    var xmldoc = Ti.XML.parseString(xmltext);

    var stores = xmldoc.documentElement.getElementsByTagName("stores");

    for(var i = 0; i<stores.length;i++)
    {
            Ti.API.info([i] + "-" + stores[i]);
    }
});

The file is found however when it gets to the following line of code

var xmldoc = Ti.XML.parseString(xmltext);

I get the following error message:

[ERROR] Error Domain=com.google.GDataXML Code=-1 "The operation couldn’t be completed. (com.google.GDataXML error -1.)". in -[TiDOMDocumentProxy parseString:] (TiDOMDocumentProxy.m:57)

I have searched high and low for an answer and haven't yet found a solution that works. If anyone has experienced this problem and managed to get it working then please could you post your solution!

Thanks in advance!
