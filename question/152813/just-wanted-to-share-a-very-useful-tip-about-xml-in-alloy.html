Just wanted to share a very useful tip about XML in Alloy

Alloy default XML editor is very lacking to say the least. Since I had previously developed Android applications I decided to download the ADT Plugin into Titanium Studio and then set my default XML editor to Android in the preferences section. This works great! ADT Plugin can be found @ the following location: http://developer.android.com/tools/sdk/eclipse-adt.html Just be sure to editor your file associations for XML in the preferences section of eclipse…
