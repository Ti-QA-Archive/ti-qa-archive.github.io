Android httpclient post image problem

Hi guys

I am using the following approach to upload a picture and a few fields. It works fine on iPhone, but on Android it fails with an "invalid multipart boundary" error. (current_img_data is obviously an image, the other two are strings)

var xhr = Titanium.Network.createHTTPClient();
xhr.timeout = 45000;
xhr.open("POST",url); 
xhr.setRequestHeader('Content-Type', 'multipart/form-data');
xhr.setRequestHeader('Accept', 'application/json');
var send_data = {
            receiptImage:current_img_data, 
            amount:receipt_data.Amount,
            description: receipt_data.Description,
        };
xhr.send(send_data);

I have looked around, and the suggestions to fix this seem to be to either write my own request handler (!) or take away the various headers. I have tinkered with the headers, but where I remove xhr.setRequestHeader(&#39;Accept&#39;, &#39;application&#x2F;json&#39;); I get "bad request".

Does anyone have any suggestions?

  • Hi Alister Pate,

    Please change the request header like this : 

    xhr.setRequestHeader('Content-Type', current_img_data.mimeType);
