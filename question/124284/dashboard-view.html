I tried changing the left,right values in the item properties, but it does not affect the position. Could you help me out or point me in the right direction please??

  • You need to make changes at the mobileSDK level. /Users/YOURUSERNAME/Library/Application Support/Titanium/mobilesdk/osx/1.7.5/iphone/Classes

    TiUIDashboardView.m

    launcher = [[LauncherView alloc] initWithFrame:CGRectMake(0, 0, 300, 480)];

    LauncherView.m

    static const NSInteger kLauncherViewDefaultColumnCount = 4;

    Try play around with the value. Make sure to remove your iphone build folder for fresh build after making the changes.
