  • I am also using a Galaxy S3 to test my android app.

    I have put the regular images in this folder:

    &#x2F;Resources&#x2F;images&#x2F;

    The density images (twice as large as regular image as Galaxy S3 is xhdpi) are located in this folder:

    &#x2F;Resources&#x2F;android&#x2F;images&#x2F;res-xhdpi&#x2F;

    I am only using portrait mode so ignoring landscape mode for now.

    Reference the image like this:

    var image = Ti.UI.createImageView({
    image:'/images/test.png',
    heigh:'75dp',
    width:'75dp'
});

    I have noticed that in the android emulator background images are not taken from the density image folder (seems like a bug) but on real device it works and the images are shown in best resolution.
