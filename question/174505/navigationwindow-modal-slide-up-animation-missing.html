NavigationWindow modal slide up animation missing

Hi all,

I am obviously missing something. I am trying to open a modal window (in a navigation window) from another navigation window and I am missing the slide up animation. Slide down seems to be here, although rather fast. I am sure it's supposed to work and I would like to see if anyone can tell the problem from the sample code below before I go and try to fake it.

Before the code, here is my environment:

Mac OS X 10.8.5, Node.js Version  0.10.13

Titanium CLI 3.2.3
SDK 3.2.2.GA
iOS7.1 (reproductible with iOS6.1) both in simulator and device

Here is a quick and dirty sample to reproduce

var win2 = Titanium.UI.createWindow({
    layout: 'composite',
    backgroundColor: 'red'
});
var win1 = Titanium.UI.iOS.createNavigationWindow({
   window: win2
});

var win3 = Ti.UI.createWindow({
    barImage: '/images_final/test-logo-nav-bar-ios7.png',
    backgroundColor: 'blue'
});
var win4 = Ti.UI.iOS.createNavigationWindow({
   window: win3,
   modal: true
});

var btn = Ti.UI.createButton({
    title: 'click here'
});

btn.addEventListener('click', function() { 'use strict';
    win4.open({
        animated: true,
        modalTransitionStyle: Ti.UI.iPhone.MODAL_TRANSITION_STYLE_COVER_VERTICAL
    });
});

win2.add(btn);

win1.open();

Thanks in advance for any help or hint.

    Hi

    Your sample code works perfectly for me and it also works with a simplified open.

    • A blue window sliding from the bottom to the top of the device - covering the previous red window
    // your code
win4.open({
    animated: true,
    modalTransitionStyle: Ti.UI.iPhone.MODAL_TRANSITION_STYLE_COVER_VERTICAL
});
    • A blue window sliding from the bottom to the top of the device - covering the previous red window
    // this code does the same job as yours
win4.open();

    MODAL_TRANSITION_STYLE_COVER_VERTICAL is the default action

    Your statement below and your code appear to tell two different stories;

    • Statement
