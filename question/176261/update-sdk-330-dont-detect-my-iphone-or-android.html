    Do the following:

    1. in your home directory, go into the hidden .titanium folder
    2. open the config.json
    3. make it valid JSON and save it
    4. try again / restart Titanium Studio

  • I fixed this problem by

    1. changing the SDK in tiapp.xml to the latest version 3.3.0.GA

    2. changing the last SDK: selected entry in ~&#x2F;.titanium&#x2F;config.json to 3.3.0.GA as follows

    "sdk": {
    "selected": "3.3.0.GA"
}
