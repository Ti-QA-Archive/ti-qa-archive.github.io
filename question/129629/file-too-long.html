  • I have 3000 lines long file and I have syntax highlighted (don't know about breakpoints).

    Maybe you have Java stack issue. All Eclipse based products "eat" a lot of memory. If there are a lot of files in your project, sometimes they can "choke". In that cases, increase of Java's stack size would solve my problems.

    To increase Java's stack size, modify "TitaniumStudio.ini" file in "Titanium Studio" folder.

    You need to modify (I think):

    –launcher.XXMaxPermSize

    256m

    -Xms40m

    -Xmx1024m

    I don't know will this help, but it's worth a try.
