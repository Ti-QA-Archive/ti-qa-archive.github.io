    I'm wondering if you have 500000 users ;-)) On the other hand "it depends".
    If you use a cloud provider like parse.com, apiomat or ACS it depends on the conditions of cloud provider. These providers offers a wrapper to native services like GCS or apples Push Notification.

    If in your project dance so much users then it is a good idea to build an own wrapper to the both native providers. If you use a wellknown search engine you will found description,how to implement.
