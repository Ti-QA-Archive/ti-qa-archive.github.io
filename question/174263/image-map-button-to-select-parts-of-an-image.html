Image map button to select parts of an image.

I have requirement to display a human body image and select parts of the body for a health related application.

I would like to know if this is possible in Titanium.

    Hi

    Yes

    The imageView has; click, touchstart, touchmove, touchend and many other events that provide the x/y or the touched location. In fact almost all windows, views, buttons and labels as return x/y.

    So you simply gain the x or y and check against the area you have mapped out.

    Imagine;

    • Your "body" was a rectangle of 300x400
    • The arm was 50,100 at offset 25,25
    • The events return the x or y so you use this to check;
      • is x greater than or equal to arm x offset?
      • is y greater than or equal to arm y offset?
      • is x less than or equal to arm width plus arm x offset?
      • is y less than or equal to arm height plus arm y offset?
    • If yes - then this is the arm region.

    Build those regions into an array and loop the array each time someone touches the image. If you have odd shaped regions due to not fitting into a single rectangle - use more than one to denote a single body part. For example a leg bent at the knee would have a narrow and tall region PLUS a wide but short region - either one says you touched the leg.
