Delete Button Title in API Docs? + Feedback

I had a hard time finding how to change the delete button text when a row is swiped. Finally found this answer that does the trick.

var tableView = Titanium.UI.createTableView({
    data: someData,
    editable: true,
    deleteButtonTitle: 'Archive'
});

The thing is, why is deleteButtonTitle not in the API docs? What gives? Am I not looking at the API right? What else am I missing? The docs seriously needs to be reworked! Please!

It's not just the aforementioned. You can't change the background image for a toolbar as well, but if you look at the docs for Titanium.UI.iOS.Toolbar you will find a setBackgroundImage method and backgroundImage property. What's worse is that when I set the visibility for the top border of the toolbar to false, I can see about a pixel in height of my assigned border, and setting the toolbar to translucent does not do the trick.

Anyway, I hope this post will help others in my situation, even though this rant is not in the best of tones. I do get frustrated with small but important things like these.
