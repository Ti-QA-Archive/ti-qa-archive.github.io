Alloy : Pass arguments to tss

I have a searchbar wrapper that I am using in multiple places in my app. Every time I use it , I do the following

<Require type="view" src="search" id="searchBar" />

searchBar.xml

<Alloy>
    <View id="container">
        <View id="left"/>
        <View id="right"/>
        <TextField id="field">
    </View>
</Alloy>

searchBar.tss

"#container": {
    top:0,
    left:0,
    right:0,
    height:44,
    backgroundImage:"/img/control/searchbar_mid.png"
},
"#left": {
    top:0,
    left:0,
    width:30,
    backgroundImage:"/img/control/searchbar_left.png"
},
"#right": {
    top:0,
    right:0,
    width:22,
    backgroundImage:"/img/control/searchbar_right.png"
},
"#field": {
    top:0,
    left:0,
    right:10,
    height:44,
    paddingLeft:32,
    font: {
        fontSize : 16
    },
    autocorrect:false,
    clearButtonMode:Ti.UI.INPUT_BUTTONMODE_ALWAYS,
    borderStyle:Ti.UI.INPUT_BORDERSTYLE_NONE,
    hintText:"Type in the code",
}

My question is how can I dynamically set the hintText according to different purposes? I tried the following, but no luck.


<Require type="view" src="search" id="searchBar" hintText="Zoom BB 1.5"/>

Then


"#field": {
    top:0,
    left:0,
    right:10,
    height:44,
    paddingLeft:32,
    font: {
        fontSize : 16
    },
    autocorrect:false,
    clearButtonMode:Ti.UI.INPUT_BUTTONMODE_ALWAYS,
    borderStyle:Ti.UI.INPUT_BORDERSTYLE_NONE,
    hintText: hintText,  //If accessible, would be very useful for code reuse
}

Any idea?

Thanks

    Hi jiale,

    You can pass arguments through tss/xml, but the actual mapping to that needs to be done in js file through the use of params.
