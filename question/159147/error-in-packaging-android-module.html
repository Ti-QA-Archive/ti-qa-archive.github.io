  • Have you set the android ndk folder path in studio?

    titanium preferences -&gt; studio -&gt; platforms - android -&gt; ndk home

    Also set the environment variable on bash_profile. I'm having like below

    export ANDROID_NDK=&#x2F;Applications&#x2F;android-ndk

    I'm using android ndk 8rb with titanium 3.1 GA works fine. I'm not sure what 2.1.2GA will support, you may check about that.
