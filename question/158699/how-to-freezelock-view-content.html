  • I am using this model - https://bitbucket.org/MostThingsWeb/virtualscroller/wiki/Home

    It creates 5 Views dynamically and adds them to a scrollableview - as you scroll to the next view it does the following:

    1. Removes the first view in the 5 view string
    2. Adds a new view to the end of the 5 view string
    3. Changes the currentPage of the scrollableView to the the old currentPage -1
    4. Reloads the scrollableView with the new views.

    Thats what it comes down to if I put it very simply - it does alot more but thats the basics.
