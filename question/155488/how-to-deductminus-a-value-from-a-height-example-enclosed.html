    Hi

    As you have not provided a "use case" to help provide an answer I will give you a common scenario that this is used with.

    This will create a view with a fixed height of 50 and a tableView that occupies the remaining space - which will automatically adjusts.

    var view = Ti.UI.createView({
    height: 50,
    top: 0,
    width: Ti.UI.FILL
});
var tbl = Ti.UI.createTableView({
    height: Ti.UI.FILL,
    top; 50,
    width: Ti.UI.FILL
});
win.add(view);
win.add(tbl);
