Annotation with image crashes the app

Hey guys,
I have added an annotation to my map.
if the annotation has an imageview as image attached, my app crashes instantly.
displaying the imageView in any other window works fine.

some code: http://pastebin.com/EEcRQWzi

    The image should be a string like so:

    var mountainView = Titanium.Map.createAnnotation({
    latitude:48.08214555,
    longitude:11.48501065,
    title:"Appcelerator Headquarters",
    subtitle:'Mountain View, CA',
    image: 'img/human-position-red.png',
});

var mapview = Titanium.Map.createView({
    mapType: Titanium.Map.STANDARD_TYPE,
    animate:true,
    regionFit:true,
    userLocation:true,
    height: '100%',
    width: '100%',
    region: {latitude:48.08214555, longitude:11.48501065,
            latitudeDelta:0.01, longitudeDelta:0.01},
    annotations:[mountainView]
});
