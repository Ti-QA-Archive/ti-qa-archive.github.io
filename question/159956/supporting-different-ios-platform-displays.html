Supporting different iOS platform displays

is there a way in the styling file to specify if non-retina display then do so and so, and if retina-tall display then do so and so ? I need to do different styling for different displays in iOS, I am using Alloy, so how can I do this ?

    Hello Ahmad :

    you can use device Height and Width to detect iPhone5 device like :

    alloy.js

    Alloy.Globals.isIphone5 = (Ti.Platform.osname == 'iphone' && Ti.Platform.displayCaps.platformWidth == 568) ? true : false;

    index.xml

    <Alloy>
    <Window id=win>
        <View id="dispView"/>
    </Window>
</Alloy>

    index.tss

    "#dispView:{
    heigth : "150dp",
    width : "200dp",
    backgroundColor : 'red',
    top : "50dp"
}

    index.js

    if(Alloy.Globals.isIphone5)
{
    $.dispView.top = "75dp";
    $.dispView.height = "175dp";

}

$.win.open();

    ###change your code accourding to this example..

    in iphone non-ratina & ratina device display same design only changes for iPhone5 device..

    I hope this is helpful to you and solved your problem. thanks.
