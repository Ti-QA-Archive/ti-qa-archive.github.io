  • Hello Virasinh!

    Your problem looks more related to java, in the way it handles certs.

    Problem Description

    When trying to connect via SSL with Java, you will get something like this:

    Error:
avax.net.ssl.SSLHandshakeException: sun.security.validator.ValidatorException: PKIX path building failed: sun.security.provider.certpath.SunCertPathBuilderException: unable to find valid certification path to requested target
at com.sun.net.ssl.internal.ssl.Alerts.getSSLException(Alerts.java:150)

    Solution

    • Download InstallCert.jar to a local directory;
    • CD into the directory and run "java -classpath InstallCert.jar InstallCert api.appcelerator.net";
    • It should output the same exception stacktrace as above and detect what certificate needs to be added; enter '1'.
    • Let it run to the finish, and afterward, Java should be able to complete a handshake with the host, and you could verify by running the program again.

    Extra info

    You can find more info here.

    Let us know how it goes!

    Best,

    Mauro
