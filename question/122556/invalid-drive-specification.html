  • Titanium is not compatible with Vista, check the Compatibility Matrix for more on that. As for the machines running WinXP, make sure that its not the Starter Edition and then open up your preferences and make sure that the Titanium SDK location is pointing to the correct place and that you are building to a location that exists.
