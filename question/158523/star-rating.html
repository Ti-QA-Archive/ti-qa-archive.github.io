Star rating

Hi, anyone can recommend how to install a "star rating" page whereby users can click to rate 1 to 5 (1 being low 5 being best). Just a jumpstart as a guide. I only have html or jquery mobile samples but not a native one. Thanks.

  • Please try this:

    var strip = Ti.UI.createView({layout:'horizontal',height:'30dp'});
for (var i=0;i<7;i++) {
    strip.add()Ti.UI.createImageView({left:0,image:'star.png', rating:i});
}
strip.addEventListener('click',function(_e){
    var rating = _e.source.rating;
});
