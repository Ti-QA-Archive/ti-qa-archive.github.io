  • Hi Tim, Sails.js could save you from PHP-$'s nightmares. It works with node so you would be writing the server in javascript as well, as the Titanium client app.

    Sails builds JSON REST APIs with very little code: you define your models and get basic CRUD services out of the box.

    If you use Alloy (recomended!) then you could be interested in Mads Møller's restapi adapter and the Models and Collections documentation. Hth
