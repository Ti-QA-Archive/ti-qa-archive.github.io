    Hi Maurits,

    the trick is to set an empty barImage and then hide the shadow of the navbar. In lack of having one, I created it dynamically. But a real image with 0px should also work.

    Cheers

    "Window[platform=ios]": {
    navGroup: true,
    // barColor: 'transparent', // not needed
    translucent: true,
    backButtonTitle: '', // No back button title by default
    statusBarStyle: Ti.UI.iPhone.StatusBar.LIGHT_CONTENT,
    navTintColor: '#fff',
    extendEdges: [
        Ti.UI.EXTEND_EDGE_TOP
    ],
    titleAttributes: {
        font: {
            fontFamily: 'RobotoSlab-Bold',
            fontSize: '17dp'
        },
        color: '#fff'
    },
    // new stuff
    barImage:Ti.UI.createView().toImage(),
    hideShadow:true,
    title:"Title",

}

  • Thanks Moritz, with your help we came to this solution that works, we made a 1 x 1 transparent pixel and added this to the .tss

    "Window[platform=ios]": {
    // Tell the windowManager this window is part of the NavGroup by default
    navGroup: true,
    backButtonTitle: '', // No back button title by default
    statusBarStyle: Ti.UI.iPhone.StatusBar.LIGHT_CONTENT,
    barColor: 'transparent',
    translucent: true,
    navTintColor: '#fff',
    backgroundColor: 'transparent',
    extendEdges: [
        Ti.UI.EXTEND_EDGE_TOP
    ],
    barImage:'generic/transparent.png',
    hideShadow:true,
}
