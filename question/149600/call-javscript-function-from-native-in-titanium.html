  • Hello Tarak,

    if you want to add events to the application and call them from any part of your application you must add a eventListener to "Ti.App":

    Ti.App.addEventListener ("app:something", function (data) {
    / * Code * /
    Ti.API.info("variable : " + data.variable)
})

    when your event is ready to be fired :

    Ti.App.fireEventListener("app:something", {
    /* here you can pass values */
    variable : "test"
})
