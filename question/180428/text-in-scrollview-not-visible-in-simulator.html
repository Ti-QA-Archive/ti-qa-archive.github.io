Text in scrollview not visible in simulator

I am trying to place a fair amount of text in a textArea and then adding that text area to a scrollView. Once the text reaches reaches a certain size the text is no longer visible. It is still there because you can copy it out of the text view and scroll the scrollView, but you cannot see it. I changed the background color of the textView to make sure the text was not the same color as the textView background. This seems to happen when the contentHeight of the scrollView gets between 8100 and 8200 and the text requires larger than 8200 to be completely viewed. Weird thing is that it seems to work fine on an actual 5s. I can see the text. Does anyone know if this is a Titanium issue or a simulator issue?

Titanium: 3.4.1GA
iOS 8 iPhone 5s or 6+ Simulator
