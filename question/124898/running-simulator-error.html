  • Try to open the xcode project and build from there, it might give you a more descriptive error message.

    You might also want to look at the build log file (I think in build/iPhone/build, I'm not near my Mac so I can't say for sure).

    If I recall correctly, some people faced this error due to broken PNG images. Try moving your images out of the folder and see if it runs or not.

    Others faced it due to some files in the resource folder having special characters in their names. Make sure you have nothing like that.
