  • I have been looking for this a few weeks ago as well when I started developing with Titanium and would like to help you as good as I can.

    It's not really clear if you mean Titanium or just only the IDE, so to answer as completely as possible for you I'll try to answer for both.

    As far as their documentation says it it completely free to develop and distrubute apps:
    > Appcelerator licenses Titanium under the Apache 2 license and is free for both personal and commercial use.
    > Titanium Studio is Appcelerator's free IDE (integrated development environment). You can use Studio to write, test, and debug your mobile applications. Studio also has integrated templates and sample applications to make it even easier to get started creating your own apps. In addition, Titanium Studio will help you manage Titanium SDK updates and module usage.

    See this link for more information on Titanium SDK.
    See this link for more information on Titanium Studio as well as this link*

    *Note that Titanium Studio is an extended version of Aptana Studio.

    So to summarize:
    YES, you can use Titanium totally for free without any limitations (at least not other than the cross platform limitations it might have).

    I hope this answers your question.

  • As of March 6, 2015:

    1. No, no limitations due to licensing.
    2. Nope, no limitations on types of apps.

    Appcelerator Platform, which provides all sorts of additional features and services, is the non-free product. It's aimed at enterprise customers.
