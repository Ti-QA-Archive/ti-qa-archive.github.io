  • Accepted Answer

    You're getting the error because some of those apps have builtin runtime testing and they require loading in a specific manner. Those apps are meant to be run following these instructions to make sure all dependencies are pulled in: https://github.com/appcelerator/alloy#running-sample-test-apps

    If you don't feel like bothering, though, you can always remove the test runner block from the bottom of each controller, that looks something like this:

    // runtime unit tests
if (!ENV_PROD) {
    require('specs/index')($);
}
