    Your error log is complaining about a class casting error to double. What it means is that you try to set a number to a property that is not meant to be a number.
    The docs say that the message of the notification should be a a string but you're sending a Float - that's your Double ;)

    Try to change 

    message:totalValue

    to

    message:totalValue+'' // empty string

    This will convert your float into a string and the error should disappear.
