Toolbar overlaps to WebView

iOS App SDK 2.1.4 - iOS simulator

Hello,

I have a problem with a toolbar, it overlaps on the webview and keeps a small part of the webview hidden, my app is intended to be universal. This is my code.

    var self = Ti.UI.createWindow({
        backgroundColor : 'white',
        title: "Resultados"
    });

    var WebView = Titanium.UI.createWebView({
        height:'auto',
        url : remurl,
        hideLoadIndicator: true
    });
    self.add(WebView);

    var btnHomeScreen = Titanium.UI.createButton({
        title : 'Volver a empezar',
        style: Titanium.UI.iPhone.SystemButtonStyle.BORDERED,
    });
    var btnShare = Titanium.UI.createButton({
        title : 'Enviar por correo',
        style: Titanium.UI.iPhone.SystemButtonStyle.BORDERED,
    });
    var flexSpace = Titanium.UI.createButton({
        systemButton: Titanium.UI.iPhone.SystemButton.FLEXIBLE_SPACE,
    });
    var toolbar = Titanium.UI.iOS.createToolbar({
        items:[flexSpace,btnHomeScreen,flexSpace,btnShare,flexSpace],
        bottom:0,
        borderTop:true,
        borderBottom:false
    }); 
    self.add(toolbar);

Thanks in advance.

    hi,

    you should set top and bottom property.

    var WebView = Titanium.UI.createWebView({
    top :0,
    bottom : 44,
        url : remurl,
        hideLoadIndicator: true
});
self.add(WebView);
//44 is a height of toolbar
