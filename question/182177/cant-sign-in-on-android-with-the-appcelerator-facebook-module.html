  • Welcome to the wonderful world of using pre-release software! :)

    Have you followed all the android-specific [instructions for the module]? There's three things you need to do:

    1. Modify the android manifest portion of your tiapp.xml file
    2. Create a special strings.xml file in the right spot with the correct value
    3. Use the brand new Ti.UI.fbProxy property on your windows
