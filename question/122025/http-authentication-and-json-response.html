HTTP authentication and JSON response

I have a rails app using Devise with HTTP authentication. I want to authenticate the user from the mobile app and then send/receive JSON data between my rails web app and the mobile app.

If I use cURL and use the following code:

curl -H 'Content-Type: application/json' \
  -H 'Accept: application/json' \
  -X POST http://localhost:3000/users/sign_in \
  -d "{'user' : { 'email' : 'test@example.com', 'password' : '1234'}}"

If the password is correct, it will return something like

{"user":{"email":"test@example.com"}}

If it is incorrect then it will return something like

{"error":"Invalid email or password."}

So, I'm simply trying to reproduce what I have in cURL using Xhr http client. I have something like this

xhr.onload = function(){
 Ti.API.info(this.responseText);
};

xhr.setRequestHeader("Content-Type", "application/json");
xhr.setRequestHeader("Accepts","application/json");

authstr = 'Basic ' + Titanium.Utils.base64encode('test@example.com' + ':' + '1234');
xhr.setRequestHeader('Authorization', authstr);

xhr.open('POST','http://localhost:3000/users/sign_in');
xhr.send();

But it always returns XML data of the webpage instead of the JSON data I am getting using cURL.

I get something along the lines of

<html >
<head>
Action Controller: Exception caught

body { background-color: #fff; color: #333; }
body, p, ol, ul, td

How do I get the JSON data?

I've tried responseText and responseXML. I've read all the questions on the forums. Help please!
