    Mert

    It's best to use a javascript validator during development. Titanium Studio has this built in, but there are other options listed in the Editing section of the Third Party Tools guide. If you want to quickly determine what is causing your app.js script to fail, you can use one of the online validators under the Debugging section.

    Personally, I would never think of launching or building an app that does not validate.

    Hope this helps
