  • Are you testing this on device or in the emulator? My understanding is that there's an emulator bug that causes it to fail to correctly render HTML emails. Depending on your version of the Android tooling and the Android SDK, the emulator might or might not actually send a correctly HTML formatted message. Also, Android SDK 1.6 doesn't support HTML formatted messages at all.

    On-device with an SDK newer than 1.6 should work corrctly.
