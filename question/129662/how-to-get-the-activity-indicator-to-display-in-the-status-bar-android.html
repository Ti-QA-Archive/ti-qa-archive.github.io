How to get the activity indicator to display in the status bar (Android)

In today's (12/20/2011) Appcelerator Office Hours webinar, one of the presenters answered a question about the Android activity indicator by saying that you can place a simple "spinner"-type activity indicator (iPhone style) in the status bar. Does anyone know how to do this?

    • Cool! Thanks Tony!

      To summarize, after a window is open, this code will show the activity indicator and message in the nav bar (even though the code says "STATUS_BAR"). The message will clobber the app name in the nav bar, and the spinner will appear on the right side of the nav bar. You have to have navBarHidden:false for this to work:

      var spinner=Ti.UI.createActivityIndicator({
              location: Ti.UI.ActivityIndicator.STATUS_BAR,
              type: Ti.UI.ActivityIndicator.DETERMINANT,
              message:'Loading something...'
             });
spinner.show();

      The key is the undocumented location property of Ti.UI.ActivityIndicator.

      What are the other possible values of the location property? What are the other possible values of the type property?
