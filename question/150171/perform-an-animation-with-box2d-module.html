  • You should keep the redBodyRef as a dynamic type but make sure to set the awake to false. This way, the dynamic object won't move around with forces/gravity, but you can still apply your manual movement to it.

    var redBodyRef = world.addBody(iview1, {
    density: 12.0,
    friction: 0.3,
    restitution: 0.4,
    type: "dynamic"
});
redBodyRef.setAwake(false);
