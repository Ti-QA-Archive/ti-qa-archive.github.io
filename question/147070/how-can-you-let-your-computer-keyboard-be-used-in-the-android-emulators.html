How can you let your computer keyboard be used in the android emulators?

Whenever I run my app in the android emulator, I cannot use my computer keyboard to type text into the inputs. The android keyboard comes up, and I can click the keys, but typing on my keyboard doesnt do anything. What setting do I change to fix this? I'm using Titanium Studio 3

— asked January 17th 2013 by Dan Oved

  • Dan,

    Find your <sdk>/tools/android and launch from the command line. Once launched, select Manage AVDs… from the Tools menu.

    There you will find your AVD list (emulators). Select your emulator that you want keyboard support for. Click edit and in the hardware listing click new and add 'Keyboard support' if not there and select yes.
