  • Use ACS to store an url of the file saved on Amazon S3.
    I'm using this approach on one of my apps ( own nodejs server, not ACS ).

    Try to find out if ACs suports streams, so you can upload on S3 using knox (a nodejs module for S3) without saving the file on disk, but stream it directly to S3.
    Or you could send the file from the device directly to S3 and send the url to ACS.

    Hope it helps.
