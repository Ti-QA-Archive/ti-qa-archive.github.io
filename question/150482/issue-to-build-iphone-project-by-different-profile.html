Issue to build iphone project by different profile

Somebody has built a Titanium iPhone app and generated an iPhone XCode project using his developer profile, it has not problem to deployed to his device with his XCode. With the same generated XCode project he gave to me, I am not able to build it on device via XCode although I can build in in XCode simulator, it looks a profile issue in Studio. This becomes serious issue since I will be the developer to publish this app. Can anybody reproduce it and has a similar issue? I can email or put the sample project in a website if anyone like to try. Big thanks. Bill
