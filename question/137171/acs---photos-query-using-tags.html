ACS - Photos Query using tags

Hi! I'm playing around with ACS and I'm looking for photos with a special tag.

When I do the same with coordinates, everything works, but when I try with tags I get No Results

Here is my code.

 Cloud.Photos.query({
    page: 1,
    per_page: 20,
    where: {
        tags: 'Hello'
    }
}, function (e) {
            if (e.success) {
                if (e.photos.length == 0) {
                    table.setData([
                        { title: 'No Results!' }
                    ]);
                }
                else {
                    var data = [];
                    for (var i = 0, l = e.photos.length; i < l; i++) {
                        data.push(Ti.UI.createTableViewRow({
                            leftImage: e.photos[i].urls.small_240,
                            title: e.photos[i].filename,
                            id: e.photos[i].id
                        }));
                    }
                    table.setData(data);
                }
            }
            else {
                error(e);
            }
        });
    });

I have 1 photo with that tag… and I get No Results

  • I do not believe you can query data in a pre defined field that is a hash or an array; I think you need to use search

    see

    http://cloud.appcelerator.com/docs/customfields#query
    http://cloud.appcelerator.com/docs/search_query

    This is directly from the documentation:
    You could also store more complex data types such as Array and Hash. Hash and Array can be embedded into each other. Currently, data stored inside an Array or Hash is not queryable.
