Constructing a JSON object from JSON string

The following code converts a string to JSON object.. According to my undersanding it should show the string "world" in the alert. But it is not showing… I wonder how you convert JSON format string to JSON object in Titanium.

var str = "{ hello: 'world', places: ['Africa', 'America', 'Asia', 'Australia'] }";
var json = JSON.stringify(eval("(" + str + ")"));
alert(json.hello);

    Hi

    You are getting a little confused there, no need for eval (which is evil) and stringify converts an object to a string - you need to convert a string to an object which is parse.

    So your code becomes;

    var str = '{ "hello": "world", "places": [ "Africa", "America", "Asia", "Australia" ] }';
var json = JSON.parse(str);
alert(json.hello);

    • Quick follow-up;

      To be correctly formatted you strings (values and properties) must be encapsulated with quotes ".

      See below;

      // correct
{ "hello": "world", "places": [ "Africa", "America", "Asia" ] }
// both wrong
{ 'hello': 'world', 'places': [ 'Africa', 'America', 'Asia' ] } 
{ hello: 'world', places: [ 'Africa', 'America', 'Asia' ] }
