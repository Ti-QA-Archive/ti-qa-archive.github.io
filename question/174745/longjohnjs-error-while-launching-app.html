  • Longjohn errors normally occur when a config file cannot be parsed, because it is damaged, or missing.

    The file could be Titanium's own config, could be a list of ios provisioning profiles or certificates, or a list of android AVDs.

    Since you are on ubuntu, I'm not about to spin up a box to test your configuration, but the above should give you some placed to start looking.
