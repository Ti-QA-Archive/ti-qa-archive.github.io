Below is the code:
var car = Ti.UI.createImageView({
image:'default.png',
height:242,
width:267,
defaultImage:'default.png',
backgroundImage:'default.png'
});
win.add(car);

Even i tried with CreateView :
var img = Ti.UI.createView({

animating:false,

height:242,
backgroundImage:'default.png',
width:267,
top:2
});
win.add(img);

    • Using the default image that comes with each new mobile project in Titanium Studio I had no problems getting an image to display using this code. Again, paths are relative to your resources directory. Use createImageView.

      Titanium.UI.setBackgroundColor('#000'); 
var win = Ti.UI.createWindow({
    title:'My App', 
    backgroundColor:'#fff'
}); 

var winwidth = Ti.Platform.displayCaps.platformWidth; 
var img = Ti.UI.createImageView({ image:'KS_nav_ui.png', height:100, width:100, left:0, top:0 }); 
win.add(img); 
win.open();
