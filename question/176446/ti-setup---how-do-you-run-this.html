ti setup - how do you run this?

I'm running Windows7 64 bit.

I keep seeing references to running "ti setup" to fix config issues.

I keep seeing instructions to run the Terminal and run "ti setup"

When I open the Apccelerator 'terminal' from the Appcelerator menu and type "ti setup' it responds with

'sh.exe": ti: command not found

Can someone please explain? Is there a different terminal I need to run?

    I tend to do all my fixes outside of studio.

    Open a standard windows terminal, and try to run ti setup.

    If that doesn't work, then we have a problem.

    I would then try to run node -v and see if nodejs is properly installed.

    IF it is, then npm install -g titanium to install titanium command line tools (CLI)

    Then install alloy with npm install -g alloy

    You should then be able to run ti setup.
