The iPhone/Android Delta

I'm in the embryonic stages of developing my first mobile app and the little work I've done looks great in the iPhone simulator and…not so much in the Android simulator. Specifically:

  1. I've used backgroundGradient in the createWindow() options and createButton() options. A little Google Fu indicates that these simply aren't supported on the Android platform. Fair enough, if that's the case. I can use background images unless/until gradients become supported.
  2. Buttons which, on the iPhone, are as wide as their container are not so much that way on the Android platform. Do I need to be more explicit for Android?

Here is a snippet of button code:

var buttons = Ti.UI.createView({ 
  height: 'auto', 
  layout: 'vertical', 
  left: 10, 
  right: 10, 
  top: 10 
}); 
var btn_appointment = Ti.UI.createButton({
  backgroundGradient: {
    type:'linear',
    colors: ['#4a9ecd','#1075a0'],
    startPoint: { x:0,y:0 },
    endPoint: { x: 0,y: 50 },
    backFillStart: false
  },
  borderRadius: 5,
  color:'#fff',
  font:{ fontSize: 20, fontWeight: 'bold', fontFamily: 'Helvetica Neue' },
  height: 50,
  style: Titanium.UI.iPhone.SystemButtonStyle.PLAIN,
  title: 'Schedule an Appointment',
  top: 10,
});
buttons.add( btn_appointment );
