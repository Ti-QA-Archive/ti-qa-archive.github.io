Ti 3.1.3 & support for ios6

Hi, I am currently using Ti 3.1.3 SDK. With ios7, my app UI renders correctly. However if I take the same app and run it on a IOS6 device, I have noticed that the buttons colors & styling (via the .tss files) do not render correctly or rather the styling (such as background color, font color,…) do NOT take effect. . The issue only happens when I run the app in IOS6. (no issues with IOS7).

If I do button styling at run time (as opposed to using .tss) with IOS6, then too there is NO issue. The issue only happens when buttons styling need to be done at app startup via the .tss file - essentially none of the button styling takes effect.

Any suggestions to fix this problem?

Thanks.

  • I don't know if you solved this or not, but for future reference, this is how I solved this issue. For iOS 6 devices, buttons must have the style property set. In Alloy, you can put this line in app.tss:

    "Button":{
    style: Ti.UI.iPhone.SystemButtonStyle.PLAIN
}

    This will cause all buttons to be rendered as they're styled, rather than using the standard system button styling. YMMV as far as button bars and such go.
