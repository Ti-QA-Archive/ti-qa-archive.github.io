Mobile (iOS/Android), latest SDK (3.2.x), building on Mac OSX (10.9)

In my Alloy view, I have :

<Button class="possibleLetter" onClick="insertLetter">B</Button>

When someone click on it, I want to change the backgroundColor of the button and its title :

In my Controller I have this :

function insertLetter(e)
{
     e.source.setTitle('');

}
