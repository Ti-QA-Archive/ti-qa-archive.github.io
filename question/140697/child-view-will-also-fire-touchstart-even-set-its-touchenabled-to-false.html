  • Maybe this is a bug of touchstart, not occurs at touch move , end and cancel.
    I add these code at first touchstart, and worked well.

    var onViewTouchstart = function(e) {
    if (!e.source.id || e.source.id != "t_box") {
        return;
    }
    ......
    ......
}

    Note that I add "id" attribute to ViewP. you can identify the view other ways.
