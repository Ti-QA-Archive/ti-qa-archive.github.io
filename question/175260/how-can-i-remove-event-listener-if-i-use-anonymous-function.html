How can I remove event listener if I use anonymous function?

btn.addEventListener('click', function(){... do something ...});

I read
this interesting post. I tried but it seems that addEventListener returns always 0. From the docs it seems that addEventListener has not a returned value (void). Thanks for attention.

    You can no longer remove an eventListener in that manner. You cannot remove an anonymously defined callback , you must provide a function name for it to work.
