  • Use footerTitle:

    var data = [
    { title: 'Row 1' },
    { title: 'Row 2' },
    { title: 'Row 3' }
];
var tableview = Titanium.UI.createTableView ({
    data:data,
    headerTitle:'TableView examples and test cases',
    footerTitle:"Wow. That was cool!",
});

    Check this tutorial.
