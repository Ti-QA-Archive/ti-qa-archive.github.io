View height is not correctly scaled

Hi.

I've got a scrollview (width 100%, height 100%, layout vertical).

Inside there is a view (width 100%, height auto, default layout (no layout is set) ).

If I put anything in the view, big or small, the view does not scale. It takes everytime the height of my mobile screen.

  • Since version 2.* you have to use Ti.UI.SIZE …

    In you case a good pattern is:

    exports.create = function() {
    return Ti.UI.createScrollView({
        width : Ti.UI.FILL,
        height : Ti.UI.FILL,
        contentWidth : Ti.UI.FILL,
        contentHeight : Ti.UI.SIZE,
        layout: 'vertical'
    });
}

    Now you can staple all your child views.

    Rainer
