  • One way to do it is to open the XCode project file from the build directory and delete the targets you don't want. Then you can Build and 'Archive' it from there and verify/submit to iTunes Connect using the 'Organizer' window.

    • in XCode, you should see the 'Archive' option in the 'Product' menu.
