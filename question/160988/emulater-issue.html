  • First, a handy tool to use is the markdown syntax. It makes reading your console output a lot easier. Put 3 tildes (that's this char ~) immediately above and below your code paste.

    Next, it appears that you have a read/write permissions or some kind of access error on this directory:

     Error: ENOTDIR, not a directory 'C:\Program Files\Microsoft Security Client\EppManifest.dll' at fs.readdirSync

    Check what that file is or does and try to resolve that error. Looks like it's a Microsoft product. So you may need to contact them first. Hope that helps.
