  • Need to know your operating system.

    If Windows Vista, grant the emulator access to the net in the built in firewall.
    In Vista or Win7 try running the emulator with administrator permissions.
    If on a wireless connection, sometimes the DNS is not properly handed to the emulator so name resolution does not take place.
    If behind a proxy, you may have to set up the APN settings on the emulators AVD.

    Now My Emulator is able to connect to internet.
