    Hi

    Looking at your output you are using Titanium SDK version: 2.0.1 and not the 2.1.2 you think you are using.

    You may have already installed the updates but for safety Studio will not automatically change your app to the newest version.

    • In studio locate your project and expand the project listing (if it is not already)

    • double-click the tiapp.xml file

    • find Titanium SDK on the left and use the drop down to locate a more recent version - all SDK versions you have installed will be listed here.

    • Save the file and close the tab
    • menu; Project > Clean (I always do this is changing versions)

    Try again.

    If it still does not work then;

    • create a brand NEW project and leave all options as defaults.
    • run this app

    Does it appear in the simulator? If yes then there is something wrong in your code, if not there is something wrong with your installation.
