TextField in a ListView

I've got a textfield in my listview, like below:

        <ListView id="storyboard" defaultItemTemplate="template">
            <Templates>
                <ItemTemplate name="template" layout="horizontal">
                    <Label bindId="hidden" id="hidden" />
                    <Label bindId="shown" id="shown" />
                </ItemTemplate>
                <ItemTemplate name="textfield" >
                    <TextField bindId="line" id="line" onReturn="doEnterLine"/>
                </ItemTemplate>                
            </Templates>
            <ListSection id="story">
            </ListSection>
        </ListView>

I'm reading the value of the textfield with e.value like below:

function doEnterLine(e) {
    alert(e.value);
}

Is there a way i can clear the value of the textfield? i tried to use $.line.value = &#39; &#39; but it says its undefined

    Try something like this…

    function doEnterLine(e) {
    var item = e.section.getItemAt(e.itemIndex);
    alert(e.line.value); // use bindid not id
    e.line.value = ' ';
    e.section.updateItemAt(e.itemIndex, item);
}

    I have had wierd issues when bindid and id are the same, as in your example. I typically do something like this to avoid confusion:

     <TextField bindId="line" id="tfLine" onReturn="doEnterLine"/>
