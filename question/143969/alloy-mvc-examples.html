    Super basic navigation group test app from the alloy repo: https://github.com/appcelerator/alloy/tree/master/test/apps/ui/navgroup

    I am actively working on a more robust SQLite based model example, since Alloy's persistence adapters seem to be one of the most popular features so far. Expect that hopefully within the next week or so. In the meantime, there's some very basic model test apps that can be found here: https://github.com/appcelerator/alloy/tree/master/test/apps/models

    Also, be sure to check out the upcoming 3.0 docs that include some good overview content on Alloy: http://docs.appcelerator.com/titanium/3.0/index.html#!/guide/Alloy_Framework
