Popover docs

From the documentation it shows this line in the example:

popover.show({view:button});

but in the documentation for the show method it doesn't list any parameters. In the list of properties, there is no "view" property. What is "button" suppose to be?

Also, all the get methods say they return void. What good is a get method that doesn't return anything?

Am I missing something here?

  • If you for some reason did not figure this out, or if someone else is looking for the answer to this - here it is;

    1) The "button" is the name of the button you want to "attach" the popover to - i.e. where the arrow should point to when the popover is opened.
    2) The docs for 2.1 still does not include this info, so they should be updated.

    Here is a working code example:

    var win = Ti.UI.createWindow();
var b1 = Titanium.UI.createButton({
    title:'View popover',
    left:10,
    top:3
});    

b1.addEventListener('click', function(){
    if (popover.showing) {
        popover.hide();
        popover.showing = false;
    } else {
        popover.show({view:b1});
        popover.showing = true;
    }
});

var view_p = Ti.UI.createView({
    backgroundColor:'#fff',
    width:250,
    height:250
});

var popover = Ti.UI.iPad.createPopover({
    height:250,
    width:250,
    title:'Settings',
});

popover.add(view_p);
win.add(b1);
win.open();

    Incidently, I chose to use a custom property called "showing" instead of the "visible" property, as it seems to allways return <null>.

    Hope this helps!
