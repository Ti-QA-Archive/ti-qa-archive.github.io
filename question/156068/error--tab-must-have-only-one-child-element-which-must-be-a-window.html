[ERROR] : Tab must have only one child element, which must be a Window

The code is very simple.

<Alloy>
    <TabGroup>
        <Tab id="leftTab">
            <Require type="view" src="rss" id="rssTab"/>        
        </Tab>
    </TabGroup>
</Alloy>

Copied, pasted, and slightly altered, from this appcelerator.com URL.

Produces this error, which directly contradicts the documentation:

[ERROR] : Tab must have only one child element, which must be a Window

Huh?

Your basic data:

Application type: mobile

Titanium SDK: 3.1.1.GA

Platform & version: iOS (version unknown as unable to locate this info, please tell me where to look)

Device: iOS simulator

Host Operating System: OSX 10.7.5

Titanium Studio, build: 3.1.2.201308091617

    The error is saying that a Window must be inside the tab, which you are requiring from a file called rss.xml. So you also need to have the rss.xml file and code as listed in the link:

    <Alloy>
    <Window id='rssWindow'>
        <WebView id='rssView' />
    </Window>
</Alloy>
