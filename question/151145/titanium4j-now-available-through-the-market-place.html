Titanium4j now available through the market place

Hello folks,

We just uploaded Titanium4j to the marketplace (https://marketplace.appcelerator.com/apps/5420?2037282889).
Titanium4j lets you write Titanium application in Java and used the GWT and Closure compiler to output a high optimized JavaScript.
We are looking forward to reading your feedbacks and reviews.

Cheers

    • We also started with JavaScript too. The problem is that the language dont scale well with big scale projects. Java simply offers a bit more in term of software engineering patterns. Also most of our backends were Java, so beeing able to use the same language on the client and the server was a big plus

      To you question.

      1) How do we handle memory management
      First the GWT compiler will generate the most optimized JS you can get out there. The output is also on a single file so that the momory footprint is really low.

      2) We also wrap a lot of existing modules. wrapping your own module should be pretty easy to do since Ti4j is opensource or you can contact us and we do it for you.

      "And i think it would be very annoying and would slow down the app much
      java–>javascript—>objective c or java"

      Well titanium already does "javascript—>objective c or java". What we offer is "Java–>JavaScript"
      In the case of ti4j this process is very fast because we implemented our own GWT Linker.
      Also this about what you get on the other side. Strong typing, Refactoring, Compile time errors, etc..
      We believe it s a fair trade :)

  • I'm currently using it and you can think of it this way, so you have a mobile application to develop (for Android and iOS platforms) and given this conceptual API which allows you to write once and deploy to both platforms,you can either develop it by making use of this API via JavaScript (via Titanium) or Java (via Titanium4j). Essentially this is how seamless it feels. The benefits of developing it in Java over JavaScript are but not limited to: IDE Tooling (code refactoring), Source Code analytics and optimisation of readable code, Compile-time safety of the Java language, you can even use Google's great 'Guava library' that's for GWT. Ti4j all the way!!!
