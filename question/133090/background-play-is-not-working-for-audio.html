background play is not working for Audio

Hi,

I have a standard audio player:

var audioPlayer = Ti.Media.createAudioPlayer({ 
    url: 'www.example.com/podcast.mp3',
    allowBackground: true
});

a very typical setup: there is no background playing here. If the user changes the screen, or moves out of scope, it stops playing. How can i fix this?

  • Accepted Answer

    If this is for iOS, add the following to a custom Info.plist

    <key>UIBackgroundModes</key>
    <array>
        <string>audio</string>
    </array>
