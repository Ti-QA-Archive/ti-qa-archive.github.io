  • I wouldn't worry about either of those errors as long as the simulator runs your app. I've never seen the Thumbs.db error but it looks like it's just skipping over that file while compiling, which is good. The second error I see all the time with the 1.4 builds and it has never affected the outcome of the build. If you want to get rid of it then you can upgrade to one of the 1.5 CI builds, although then you'll be introducing a whole other batch of issues.
