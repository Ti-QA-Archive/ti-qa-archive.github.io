    Hi Chris, i use the command line interface to create and then eclipse to make customization…. to try that way you can refer this link

    also remember to add all the path in build.properties file. which becomes like this..

    titanium.platform=/Users/user.name/Library/Application Support/Titanium/mobilesdk/osx/2.1.1.GA/android
android.platform=/Applications/android-sdk/platforms/android-8
google.apis=/Applications/android-sdk/add-ons/addon-google_apis-google_inc_-8
android.ndk=/Applications/android-ndk-r8b
