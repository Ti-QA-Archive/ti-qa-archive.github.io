    I find Alloy cool as much as any other guy here. But let me be honest here, I don't really think Alloy is 100|% ready for production code. Sure it's great and it has a lot of very neat features, but if you have a short development cycle and you want to be able to use everything that has already been made available by other titanium devs. (modules, etc.) I strongly suggest you go the "classic" route.

    Because if you encounter any errors down the road, there will be no way to know if your problem is Allow related or coming from your code itself.

    Allow is getting there and you can do many things with it. But for real app, today go with what you know.

    Just my 2 cents.
