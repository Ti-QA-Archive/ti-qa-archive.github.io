JSON parse - trying to handle the response data

Hi guys,

Quick one here, and most likely a late night syntax issue. Using SDK 3.1.2 on iOS 6.

I'm calling an API which returns an access token if the request is successful.

This is the JSON that gets returned from the API response

[
    [
        "result",
        "success"
    ],
    [
        "access_token",
        "1dfcc030a0d31eec0d1eabbff0cca2a3"
    ]
]

And this is how I parse it, just setting a variable to store to use use later on.

// get the JSON response from the API
var jsonObject = JSON.parse(this.responseText);

// get the access token from the API reponse
var APIResponder = jsonObject.access_token;

I just keep getting NULL returned, my TI.API output returns the access token, so the API is working correctly.

Any pointers?

Simon

    Hello Simon,

    IN your XHR return data is an two dimensional array not object.So you can access following way:-

    var APIResponder = jsonObject[0][0] ;  //then it will display  result;
var APIResponder = jsonObject[0][1] ;  //then it will display  success;
var APIResponder = jsonObject[1][0] ;  //then it will display  access_token;
var APIResponder = jsonObject[1][1] ;  //then it will display  1dfcc030a0d31eec0d1eabbff0cca2a3;
