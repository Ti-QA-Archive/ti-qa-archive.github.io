  • Is there a reason why you want to use the "focus" event instead of the "open" event? I've seen the same behavior with "focus" in my app, that it gets called multiple times when the window opens, and I'm not using tabs so those shouldn't have anything to do with what you're seeing. If you want to keep using the "focus" event then just have an isLoaded variable that you set to true when you load the views, and only load the views if isLoaded == false.
