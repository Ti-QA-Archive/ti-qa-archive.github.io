Titanium.Network.online does not work on Android

When I try to access the Titanium.Network.online property in the Android emulator, I get the following error:

Type Error : Cannot read property "online" from undefined

This also happens when I try to access other properties in the Titanium.Network class. However, I do not have this issue in the Iphone Simulator. Is there an assembly I need to include to use the Titanium.Network class?

I have seen another question about this same issue on this site and the solution was to add lines to the manifest.xml file. When I check this file, the line were already there.

  • It's common for a module to show as missing or undefined the first time it's being used in the application because Titanium only includes the modules you're using when it builds, and doesn't do a full rebuild of the app every time you build.

    If you touch tiapp.xml (or make a trivial change to the file and save it), it will force a full rebuild with all required modules.
