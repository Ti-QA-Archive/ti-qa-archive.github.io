How to get datePicker(Titanium.UI.PICKER_TYPE_DATE) value?

My picker definition

var bDayPicker = Titanium.UI.createPicker({
    top:43,
    type:Titanium.UI.PICKER_TYPE_DATE,
});

Trying to retrieve value

done.addEventListener('click', function(e) {
    picker_view.hide();
    alert(bDayPicker.value);
});

nothing returns.

Looking at the docs it says
> value Date the Date/Time value for date pickers

Any help please?

    Here you go

    // Listen for when the datetime picker changes value
        picker.addEventListener("change", function(e){
            alert(e.value)
        });
