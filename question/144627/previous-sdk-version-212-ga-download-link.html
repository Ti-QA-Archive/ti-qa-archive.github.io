Previous SDK version 2.1.2 GA download link

Were can i find link to the previous versions of titanium SDK,
i'm aware of http://builds.appcelerator.com.s3.amazonaws.com/index.html - but I can not find 2.1.2 GA version.

  • Accepted Answer

    Go to: Titanium Studio -> Help -> Install Specific Titanium SDK -> Install from Update Stream and choose 2.1.2.GA
