  • You change these warnings in the manifest XML file (think Ti uses Tiapp.xml). Look into the preferences XML element. If you don't define any permissions, android assumes you need them all. In a more native app it looks like this:

    <preference name="permissions" value="android.permission.INTERNET"/>

    you can google for all the permission names.
