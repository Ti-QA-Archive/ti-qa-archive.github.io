JSS files not recognized by app.js

Hi, I'm learning how to use the JavaScript Style Sheets without any success. I have the following simple project:

app.js

// Call modules
var ui=require('ui');
ui.win.open();

ui.js

// Program interface
var win=Ti.UI.createWindow({
    title:'Test',
    id:'uiWindow'
});
var label=Ti.UI.createLabel({
    text:'This is a sample label',
    id:'uiLabel'
});
win.add(label);
exports.win=win;

And in the some folder where ui.js is, I have ui.jss:

#uiWindow{
    background-color:'#990000';
}
#uiLabel{
    width:150;
    color:'#ffffff';
    text-align:'center';
    font-size:'12';
}

When I run the project a black screen show up with nothing, like if the styles in the JSS file were not read. I have checked the code explicitly adding color formats in the win and label objects and they work.

Do I need to do anything else in order to tell Titanium that ui.js is linked to a file called ui.jss or should Titanium do it automatically?

What is wrong?

Thanks.

    • I have no idea why it isn't working, however you probably shouldn't use JSS. We're working on an alternative that isn't quite ready yet. Once it is, I imagine JSS will begin a deprecation period and marked for removal in the next major version. In the meantime, you can continue to try and get JSS to work or write some helper functions to apply the properties to whichever UI elements you want.
