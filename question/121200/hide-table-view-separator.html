Hide Table View separator

i have a TableView control , on the TableView control am adding TableViewRow.
On the UI, the each row is coming with a separator (default) , but according to my requirement need to hide the separator.
How is poosible ?

Thanks in advance

    You can make the separator transparent or set it to the window background Cole with the separatorColor property, as in :
    separatorColor:'transparent',

  • You have got two options - if you are developing to both android and iphone, you have toset the separator color as the same color as the backgroundcolor of the tableview.

    If it is only for iPhone, you can use the separatorStyle and set i to Titanium.UI.iPhone.TableViewSeparatorStyle.NONE

    var tableView = Ti.UI.createTableView({
    separatorStyle: Titanium.UI.iPhone.TableViewSeparatorStyle.NONE,
    separatorColor: '#FFFFFFF',
    backgroundColor: '#FFFFFF'
});
