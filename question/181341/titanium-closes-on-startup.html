  • Hello Rodrigo,

    Seems the problem is with your Java setup .

    Titanium supports the Minimum JDK Version 6 (aka 1.6) rev 10 and Max JDK Version 8 latest revision for Linux Operating System. Please have a look on Installing Oracle JDK.

    Note that,

    1. This package must be installed to launch Studio on Linux systems.

    2. If using the 64-bit JDK, the ia32-libs package is required.

    Hope this helps. Thanks.
