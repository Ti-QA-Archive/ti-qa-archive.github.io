Ti.addEventListener, Ti.App.addEventListener and Ti.API.addEventListener!

Hello,

What is the difference between these functions: Ti.addEventListener, Ti.App.addEventListener and Ti.API.addEventListener?

  • They are each methods for adding listeners for events fired by their respective modules. Doesn't look like Titanium (Ti) or Ti.API modules actually fire any events, so adding an event listener wouldn't be very useful there.

    Ti.App.AddEventListener is very useful, however, and can be used for working with application level events. See more info on using events here.
