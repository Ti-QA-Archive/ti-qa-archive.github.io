How could you set a password on an application?

Sorry if this is a stupid question!

I'm trying to allow users to set a password on their app the first time they open it, so that all subsequent opens require a password to be entered.

Is there a way to permanently store a variable on the phone?

    To store just a few pieces of data, use the Properties module. For large amounts of data a relational database would be better but Properties are perfect for something like this.

    Generally speaking though, you do not want to store a password. Instead consider storing a MD5 hash of it or some encryption or obfuscation technique. Then to validate, perform the same algorithm against the login request and compare encrypted/hash values. You never really need to be able to decrypt to plain text.
