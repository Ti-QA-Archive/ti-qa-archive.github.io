Titanium Map Annotation customView not appearing on Android

I've tried using the 'customView' of Map Annotations and assigned a Ti.UI.Label as the custom view. The label appears on iPhone Map but it does not appear on Android. Basically the 'customView' is not visible on the Android.

Environment :
Ti SDK 3.4.0
Android OS 4.3

  • Accepted Answer

    Somehow, it seems that a label can't be used directly as a customView, so you need to wrap it with a Ti.UI.View to make this work:

    var label = Ti.UI.createLabel({ text: "my annotation" });
var annotationView = Ti.UI.createView({ backgroundColor: 'yellow' });
annotationView.add(label);

var annotation = Map.createAnnotation({
  customView: annotationView
});
