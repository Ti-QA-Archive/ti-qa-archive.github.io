Custom Row selection highlight

Hi,
In a TableView I am crearing a custom row with some label and imageviews inside. When I click and hold in an empty space in the row view, it hightlights to selected color but when I click any component (imageview or label that I added to custom row), it does not hightlight the background. How can I make sure it will highlight the background whereever I click in the row?

    Hey Burak,

    if you need this labels and imageviews only for display then just set

    touchEnabled : false

    so now it works like as you want…
