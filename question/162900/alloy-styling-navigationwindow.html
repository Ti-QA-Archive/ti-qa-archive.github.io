Alloy Styling NavigationWindow

Hello,

I am building an iPhone App using Alloy, and i need to make a custom navigation bar for styling reason… this is my code:

<Alloy>
        <NavigationWindow id="nav" platform="ios">
            <Window class="container" id="appCont" navBarHidden="true">
                <View id="topBar">
                    <Button id="back" />
                    <Label id="title" />
                    <Button id="expandMenu" />
                </View>
            </Window>
        </NavigationWindow>
</Alloy>

how can i make the back button work like the back button in the navigationWindow (move back with effect)?

Best Regards,

    In the controller file of the window do the following

    $.back.addEventListener('click',function(e){

     $.appCont.close();

});
