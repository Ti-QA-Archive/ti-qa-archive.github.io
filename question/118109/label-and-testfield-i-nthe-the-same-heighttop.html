Label and TestField i nthe the same height/top

Hi,

I need to put a label (left aligned) and in the right of the label a textField. I.e.
Name: __
Age: ___ Years
(where the underline stands for the text field).
How can I do it? I've tried to put them both in the same 'top' level etc. and nothing help.

Thanks…

code:

    var enterNameLabel = Ti.UI.createLabel({
        text:'Name',
        top:'100',
        width:'50',
        height:'auto',
        textAlign:'left',
        color:'white    '
    });

    var nameField = Titanium.UI.createTextField({
        color:'#336699',
        top:'auto',
        height:35,
        top:'100',
        left:80,
        width:250,
        keyboardType:Titanium.UI.KEYBOARD_NAMEPHONE_PAD,
        returnKeyType:Titanium.UI.RETURNKEY_DEFAULT,
        borderStyle:Titanium.UI.INPUT_BORDERSTYLE_ROUNDED

    });
    nameField.addEventListener('return', function()
    {
        nameField.blur();
    });

    infoWindow.add(enterNameLabel);
    infoWindow.add(nameField);

  • try like this:

    
var enterNameLabel = Ti.UI.createLabel({
        text:'Name',
        top:100,
        left:30,
        width:50,
        height:35,
        textAlign:'left',
        color:'white    '
    });

    var nameField = Titanium.UI.createTextField({
        color:'#336699',
        height:35,
        top:100,
        left:80,
        width:250,
        keyboardType:Titanium.UI.KEYBOARD_NAMEPHONE_PAD,
        returnKeyType:Titanium.UI.RETURNKEY_DEFAULT,
        borderStyle:Titanium.UI.INPUT_BORDERSTYLE_ROUNDED

    The dimensions specified by numbers don't need to have quotes around the number value.
    If you don't specify a left/right or a top/bottom values Titanium will try to center the element on that direction in the parent view/window.
