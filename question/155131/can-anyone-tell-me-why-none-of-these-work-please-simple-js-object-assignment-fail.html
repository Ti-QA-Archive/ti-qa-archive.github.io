    Hi

    As you have probably guessed that all 3 of your JSON syntax examples as correct.

    So the problem is not that, so you need to work out why either this code appears to be causing a problem or if this section is the actual problem.

    • First thing to check is that adMobView is not used for anything else in this code section.
    • Second thing add extra tests to work out if this is where the actual problem is;
    Ti.API.info('before');
adMobView = {
    rect : {
        height: 0
    }
};
Ti.API.info('after');

    If you see both before and after then this is not where the code is actually dying.
