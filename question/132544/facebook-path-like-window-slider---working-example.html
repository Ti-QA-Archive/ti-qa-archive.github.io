Facebook/ Path like Window Slider - working example

Hello community,

I'm glad the day, has finally come when I contribute to this community.

Well, now this isn't a question. I am not sure if this is the right place to post this, but I thought I'll give it a try.

Let's cut to the chase. For a personal project, I thought of implementing the Path like window slider. I found a few working examples in the Q/A section but did not have the same finesse.
So I carefully studied the Path app, and realized that it isn't utilizing the "swipe" listener. It checks when "touchstart" happens, records every "touchmove" and takes a decision upon "touchend".
So I went ahead and wrote my code to mimic it. Turns out, it works like a bliss.

Also, this is purely native to Titanium. No obj-C dependencies. So it works out of the box.

You can find the code on the GitHub page along with a video of the working demo. Cheers.
