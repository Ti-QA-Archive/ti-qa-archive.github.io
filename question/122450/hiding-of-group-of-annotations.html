Hiding of group of annotations

Hi *,

I try to hide/unhide a group of annotations.

The group of annotations is in a array stations.

The switcher has the following onclick-function: 

mySwitch.addEventListener('change',function(e) {
    velo.options.stations = e.value; // copy in options  
    Ti.App.fireEvent('saveoptions'); // signal2ctrl
    switchStations(e.value);         // switching
});

And here the implementation of hider/unhider:

function switchStations(onoff) {
    for (var i=0;i<stations.length;i++) {
    if (onoff=='1') {    
        mapView.selectAnnotation(stations[i]);
    } else {
        mapView.deselectAnnotation(stations[i]);
    }
}

But nothing happens ;-((
