  • Hi

    It is possible to test on a device, but achieving it is dependant on which version of the android OS your device has.

    A few things to check

    • enable unknown sources on your android device
    • close the emulator
    • when you connect via USB, if you have the option, use connect as power/charge only (usually only 2.x versions)
    • if option above is missing make sure it is set for data transfer, not camera mode
    • two options above only changeable when device physically connected
