    Hi

    One way I would try is to use the convertPointToView method of the child view.

    • convertPointToView

    The way to try this is to add the views in this way;

    scrollView
  view  // container
    view
    view
    view

    Then you would use the convertPointToView method and reference the specific child view and the container - the container is required to give an internal height position.

    I have not tried this but this should work.

    var pos = childView.convertPointToView({ x: 0, y: 0}, containerView);
scrollView.scrollTo(pos);

    I am not sure which way round the first line of code should be - so if it does not do as you hope swap it as follows;

    var pos = containerView.convertPointToView({ x: 0, y: 0}, childView);
scrollView.scrollTo(pos);

    Let me know how it goes.

    • I am able to make it work. Here it goes like this.

       subView.addEventListener("postlayout", function (e) {
                    var pos = this.convertPointToView({ x:0, y:0}, _topView);
                    _topView.scrollTo(0, pos.y);
                });
