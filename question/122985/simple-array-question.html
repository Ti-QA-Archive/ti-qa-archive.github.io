    Both approaches work fine for me:

    var myArray = [];

myArray[0] = "hello 0";
myArray.push("hello 1");

Ti.API.info('myArray[0] = ' + myArray[0]);
Ti.API.info('myArray[1] = ' + myArray[1]);

    This outputs the following, using Android 2.1, Ti 1.7.1:

    TiAPI  I  (kroll$1: app://app.js) [126,1260] myArray[0] = hello 0
TiAPI  I  (kroll$1: app://app.js) [1,1261] myArray[1] = hello 1

    That said, I would usually use the array push method.

    Cheers
