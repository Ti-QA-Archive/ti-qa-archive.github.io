App crashes after opening and closing a window.

I am creating this test application to get familiar with appcelerator but my app is crashing on the iOS iPhone simulator after I open and close a window with a button.

This is my code:

//
//
// Main window creation
//

var mainWin = Titanium.UI.createWindow({
        backgroundColor:'#336699',
        title:'Database App',
        navBarHidden:false
});

mainWin.open({modal:true});

// Image of the MySQL logo

var titleView = Titanium.UI.createImageView({
    image:'images/logo.png',
    width:'auto',
    height:'auto',
});


// Title window containing the MySQL image

var titleWin = Titanium.UI.createWindow({
    backgroundColor:'white',
    borderColor:'black',
    borderWidth:2,
    height:130,
    width:320,
    top:0,
});

// Window containing the main menu

var menuWin = Titanium.UI.createWindow({

    backgroundColor:'white',
    width: 260,
    height: 240,
    top: 150,
    borderColor: 'black',
    borderWidth: 6,
    borderRadius: 15,

});

var menuLabel = Titanium.UI.createLabel({
    text: 'MENU',
    textAlign: 'center',
    width:'auto',
    height:'auto',
    top: 15
});


// Buttons for the main page menu

var btnServerInfo = Titanium.UI.createButton({
    title:'Server Info',
    width: 200,
    height: 25,
    top: 60
});

var btnServerGet = Titanium.UI.createButton({
    title:'Server Query Get',
    width: 200,
    height: 25,
    top: 100
});

var btnServerPut = Titanium.UI.createButton({
    title:'Server Query Put',
    width: 200,
    height: 25,
    top: 140
});

var btnAbout = Titanium.UI.createButton({
    title:'About',
    width: 200,
    height: 25,
    top: 180
});


titleWin.add(titleView);
mainWin.add(titleWin);
mainWin.add(menuWin);
menuWin.add(menuLabel);
menuWin.add(btnServerInfo);
menuWin.add(btnServerGet);
menuWin.add(btnServerPut);
menuWin.add(btnAbout);

// Server Info Button Even listener for Click - Opens window with server information

btnServerInfo.addEventListener('click', function()
{

    var w = Titanium.UI.createWindow({
        backgroundColor:'#336699'
    });

    // create a button to close window
    var b = Titanium.UI.createButton({
        title:'Close',
        height:30,
        width:150,
        bottom: 10
    });
    w.add(b);
    b.addEventListener('click', function()
    {
        w.close();
    });

    w.open();
});

After I click the close button and the window closes the app doesn't respond. I try to click any button and nothing happens. I have to reload the app for the button to work again but it goes and crashes again. On Titanium Studio's Console it shows this message on yellow "[WARN] We still care about isRootViewAttached!!!!!!!". Does anyone know what is wrong?

  • Well, I think I figured it out. It seems that the main window can't be in modal view. After I removed that then it seemed to work fine. Not sure why but that's what worked for me.

  • Technique Tip:

    For performance, you should Window.open() after adding the elements, this example here:

    Window.open();
a = new Date();
for (loop=0;loop<1000;loop++) {
   Window.add(Titanium.UI.createButton({title:loop}));
}
b = new Date();
alert(b-a);

    runs in 203ms on my simulator, however, changing the open until everything has built (buffered):

    a = new Date();
for (loop=0;loop<1000;loop++) {
   Window.add(Titanium.UI.createButton({title:loop}));
}
b = new Date();
alert(b-a);
Window.open();

    only took 104ms. It makes a more dramatic difference on 800mhz (or slower) phones.
