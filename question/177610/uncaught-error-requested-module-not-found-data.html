    • In the future, please post follow ups as comments to the original answer rather than adding a new answer. This will make it easier to keep conversations together.

      Of the error log you posted, this looks to be the key part:

      ComponentInfo{me.project.Alloy2TabbedAndroidUI/org.appcelerator.titanium.TiActivity}: java.lang.IllegalStateException: You need to use a Theme.AppCompat theme (or descendant) with this activity.

      I missed that when looking at the sample app's code. It's for a previous version of Titanium. Starting with 3.3.0, we've switched base themes to the AppCompat series. In fact, it uses a couple of outdated tags in its tiapp.xml. I'd recommend changing this:

      <android xmlns:android="http://schemas.android.com/apk/res/android">
  <tool-api-level>14</tool-api-level>
  <manifest>
    <uses-sdk android:maxSdkVersion="17"
      android:minSdkVersion="8" android:targetSdkVersion="14"/>
    <application android:theme="@android:style/Theme.Holo.Light"/>
  </manifest>
</android>

      To this:

      <android xmlns:android="http://schemas.android.com/apk/res/android"/>

      (Just make it an empty tag) With the newer SDK versions, the default values are typically sufficient. In any case, the values set in that old example block would not be suitable for current development.

      With that change, the sample app works for me.
