Could not find all required Titanium Modules: ti.cloud in Kitchen Sink

I am trying out Titanium and created the Kitchen Sink demo app. When I try to run it, either on Mobile Web or on Android, I get the error:

[ERROR] :  Could not find all required Titanium Modules:
[ERROR] :     id: ti.cloud     version: latest     platform: commonjs     deploy-type: development

Is there anything I need to do to include the Titanium Modules? I thought they would be installed by default.

  • I met the same problem with you. The solution is : don't declare the ti.cloud as the platform='commonjs'. declare them in the specific platform instead, e.g:

    my code with error: 

    <modules>
    <module platform="commonjs">ti.cloud</module>
</modules>

    should be changed to: 

    <modules>
    <module platform="ios">ti.cloud</module>
</modules>
