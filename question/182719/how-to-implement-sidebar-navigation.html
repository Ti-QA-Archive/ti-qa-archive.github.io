  • Accepted Answer

    • You can have two views added in a window(layout property should not be set for the window), then give the width and height as Ti.UI.FILL for both the views. End Result should be one view(View2) is siting on top of the other (View1).

    • Next you can have a button(Menu).

    • When you click the Menu Button, it should animate the view 2 to move right, this will expose the underlying view1.

    • Now you can design view1 to have the contents of the side bar.

    • Add Event Listener to the options you just created for the side bar, now when you click on the option View2 will move back to its original position.

    — answered June 18th 2015 by Himanshu Justa

    • index.xml

<Alloy>
    <Window id = "win">
        <View id="view1"  width="Ti.UI.FILL" height="TI.UI.FILL" />
     <View id="view2"  width="Ti.UI.FILL" height="TI.UI.FILL" >
         <Button id="nav" onClick="showsideBar" title="Nav" top="10" width="100" height="50" />
    </View>
    </Window>
</Alloy>


index.js

function showsideBar(){
    $.view2.animate(
    {
      left: 100,
      duration:1500});
    });
}

      — commented June 20th 2015 by Himanshu Justa

    • This a very basic way to move the view.
      Amount by which you want the view2 to move can be adjusted by changing the left parameter in the showsideBar function
      Basic idea, here is, when your app launches View2 will be above view 1, then when you click on the button you move the view2 by changing either its left or the right property, to show the underlying view1.

      This should give you the basic idea of how you can implement the side bar navigation, this is how i had done.and then when you click on the button again then view2 should move back to its original position by making left as 0.
