Is animatedCenter or center supported by Android using alloy?

Using alloy on SDK 3.0.2.GA the following code (based on kitchen sink) works on iOS (5+) but fails on Android (tried 2.2 and 4.0) with the msg "uncaught typeError: Cannot read property 'x' of undefined". After various test trying to display centre property I'm coming to the conclusion that this isnt support by Android even though the documents (I think) imply it should be. Can someone confirm whether this is correct and if so suggest an alternative. Many Thanks.

index.js

function tmvImage(e) {
    Ti.API.info('Our event tells us the center is ' + e.x + ', ' + e.y);
    Ti.API.info('Animated center is ' + $.imgCam.animatedCenter.x + ', ' + $.imgCam.animatedCenter.y);
    Ti.API.info('Width is ' + $.imgCam.width + ', Height is ' + $.imgCam.height);
    var newX = e.x + $.imgCam.animatedCenter.x - $.imgCam.rect.width;
    var newY = e.y + $.imgCam.animatedCenter.y - $.imgCam.rect.height;
    Ti.API.info('newX is ' + newX + ', newY is ' + newY);

    $.imgCam.animate({
        left : newX,
        top : newY,
        duration : 1
    });
}

$.index.open();

index.xml

<Alloy>
    <Window class="container">
        <ImageView id="imgCam"  onTouchmove="tmvImage" image="http://farm9.staticflickr.com/8241/8553540375_7bcb942e26_n.jpg" />
    </Window>
</Alloy>
