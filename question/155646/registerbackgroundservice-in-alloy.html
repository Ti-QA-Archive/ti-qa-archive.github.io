How can I use the registerBackgroundService with the Alloy MVC pattern?

Tom

  • Exactly how you would in a traditional Titanium app. The only differences I can think of off hand would be:

    • The "url" JS file you would be referencing should be placed in your "app/lib" folder
    • You should likely register the service in the initialization of your app, before even the index is opened. To do that, put your registering code in the "app/alloy.js" file.
