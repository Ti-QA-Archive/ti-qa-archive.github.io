  • I want to do an iTunes sync so i can grab the IPA and put on TestFlight

    Well, thats not the only way to do it. The cli offers you plenty of possibilities.

    testflight.sh

    ti build --platform iphone -T dist-adhoc --log-level trace -R "Your Developer Cert" -P {provisioning profile id} -O $HOME/Desktop
curl http://testflightapp.com/api/builds.json --progress-bar -F file=@/Users/stephenfeather/Desktop/yourAppName.ipa -F api_token='your_api_token_here' -F team_token='your_team_token' -F notify=False -F distribution_lists='Internal' -F notes="$comment"

    We have also moved from testflight to Installr. It was developed by a titanium development firm, helps manage your provisioning profiles a lot better than Testflight ever did.
