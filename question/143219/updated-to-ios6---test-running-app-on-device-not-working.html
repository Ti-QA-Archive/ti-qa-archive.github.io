Updated to iOS6 - Test Running app on device not working

Hi

I have recently been having problems with appcelerator and xcode after updating to xcode 4.5.1 and the 6.0 SDK.

Another problem i am having is that when testing my app out on my iPad through Xcode - it's picking up an older version of the app files not the very latest.

Is this a problem for anyone else?

It seems to be that xcode is running the build as it was just before i had updated the software. As for example i now have an alert dialog and a progress bar within my app (that were not in place before updating). The simulator in appcelerator picks this up and runs the latest app version but anytime i go to actually running on device (through xcode) it is running an older version without both of these elements on the ipad.

I have tried:

  • removing all traces on the simulator
  • several clean builds in the folder structure
  • deleting the previous app from the iPad device
  • even running the exact same code as a seperate new app with a different name

All of the above result in the same outcome

    Read the 2.1.3 SDK release notes.
