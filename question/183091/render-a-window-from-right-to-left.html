Render a window from right to left

Hi,
When I render a window using animation it always renders from left to right (I set the initial hight = 0, final height to 220). I was wondering if it's at all possible to render a window from right to left?

Any help is greatly appreciated.

  • Try this:

    win.left = Ti.Platform.displayCaps.platformWidth;

win.open();

win.addEventListener('open',function() {

    win.animate({
        left:0,
        duration:500
    });
});

    Hi Ray,

    I hope this will still help you.

    var win = Ti.UI.createWindow({
    left:'100%',
    width:'100%',
    height:'2%'
});

win.open();

win.animate({
    left:0,
    height:'220px'
    duration:500
});

    If you don't fix the window's width, the right position will stay out of the screen and your window'll be larger than the screen.
    I already tried to set the height to 0 but it appears that the animation doesn't work and with 1% neither. So with '2%' it works.
