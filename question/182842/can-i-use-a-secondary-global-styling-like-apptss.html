Can I use a secondary global styling like app.tss?

I would like to create another TSS file in addition to app.tss that is used globally. The reason for this is because my company has multiple apps that need to use the same style rules. I want to extend the app.tss global styling to include a secondary file. The only way I know to use the same styles is to copy-paste our style rules into app.tss, but this causes issues with source control. It's difficult to tell if you have the latest version and difficult to tell why a change was made.

I see in my build folder, under the app, there is the file MyApp.app&#x2F;alloy&#x2F;constants.js which has the line:

I'm wondering if I can extend this somehow. However, I don't see anywhere that GLOBAL_STYLE is used.
