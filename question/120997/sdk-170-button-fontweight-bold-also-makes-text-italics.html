SDK 1.7.0 Button fontWeight bold also makes text italics

Using SDK 1.7.0 and assigning a button this code makes it italics too:

font: {fontSize: 14, fontFamily: "Helvetica", fontWeight: 'bold', fontStyle: 'normal'},

I tried all different combinations but, as soon as you assign fontWeight: 'bold' the text in the button is bold and italicized.

Any ideas?
