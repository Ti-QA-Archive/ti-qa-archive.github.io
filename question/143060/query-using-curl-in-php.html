Query using cURL in PHP

How to perform Query using cURL in PHP?

I can run the actual cURL (similar to Example A below) via command line and it works fine. Now, I am aiming to write something similar to Example B below. Can anyone help with this?

Thank you in advance.

Example A:
curl -X GET "https://api.cloud.appcelerator.com/v1/places/query.json?key=XXXXXX" –data-urlencode 'where={"user_id":"A0B1C2D3E4F5G6XXX"}'

Example B:

$var = 'curl -X GET "https://api.cloud.appcelerator.com/v1/places/query.json?key=XXXXXX" –data-urlencode \'where={"user_id":"A0B1C2D3E4F5G6XXX"}\'';

$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, $var);
curl_setopt($ch, CURLOPT_GET, true);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, TRUE);
$buffer = curl_exec($ch);
curl_close($ch);
