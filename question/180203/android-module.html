i wish use AudioTrack module of android sdk.

but when i run then

This module is in /android-sdk/android/media/AudioTrack.java

How i can use it??

    Modules need to be compiled for use in Titanium. You cannot simply reference a .java file and use the module. You can package a module that would expose the functionality you want. Here is a link to the documentation on what you need to do to create a module for use in Titanium. http://docs.appcelerator.com/titanium/3.0/#!/guide/Creating_a_New_Titanium_Module
