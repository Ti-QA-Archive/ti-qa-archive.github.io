  • You cannot hide the Status Bar at the top, but you can influence the color of it using the <statusbar-style> setting in the tiapp.xml.

    As for the navigation bar at the bottom of the browser, you cannot hide that either, but if you add the page to the home screen, it will launch the app without the navigation bar.

    Titanium doesn't have an API to detect if running Mobile Safari or if running in a fullscreen browser via the home screen. However, you can do something like this:

    if (Ti.Platform.osname == 'mobileweb' && !navigator.standalone) {
    // show a view that says to bookmark the app to the home screen
}
