  • Nested views do work on android. I often put a view onto a window and add views like buttons to that view.

    Your code is obviously part of a much larger body of work, so it's very hard to see what might be going wrong without seeing everything (which I'm sure you don't want to do, nor do many readers here have the time to sift through multiple modules).

    Can you boil this down to a minimal case (preferably one that fits into a single app.js)? It's a tedious exercise, but I find that it's part of life with Titanium. What you'll probably find is that when you try to make a minimal case, it does in fact work. Then you can look at what you're doing wrong with your richer codebase.

    If you can build a minimal case that exhibits the problem, I'm sure somebody can help you out!
