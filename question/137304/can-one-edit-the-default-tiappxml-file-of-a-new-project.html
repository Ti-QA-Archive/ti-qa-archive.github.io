Can one edit the default tiapp.xml file of a new project?

Can one edit the default tiapp.xml file of a new project?

What I mean by that is, when I open a new project the first thing I do is edit the tiapp.xml file to include this piece of code:

    <android xmlns:android="http://schemas.android.com/apk/res/android">
   <tool-api-level>8</tool-api-level>
   <manifest android:installLocation="preferExternal">
      <uses-sdk android:minSdkVersion="7" />
   </manifest>
</android>

All that code does is place the app on the memory card of my Android device instead of the internal memory, which is typically very small.

And I want that code in every new project I start.
Can I edit the default code of the tiapp.xml file to include that code automatically every time I start a new project in Titanium?

Is there a place to edit the preferences of the default files generated by starting a new project?

    The answer is yes, you CAN edit the default project files.

    You failed to mention the platform you are building on, so I'll give the answer for OSX and you can figure it out if you aren't on that.

    $HOME&#x2F;Library&#x2F;Application Support&#x2F;Titanium&#x2F;mobilesdk&#x2F;osx&#x2F;{sdk_version}&#x2F;all&#x2F;

    I have 8gb of internal storage, and I like my apps where I want to put them.
