Loading Javascript dynamically

I am making an application where we have a very elaborate effects system which can be customized using an FX file format much like the Cg and HLSL fx formats. This code is loading dynamically from disk and essentially converted into equivalent Javascript code. However, there doesn't appear to be a convenient way of loading Javascript files dynamically from a runtime source string.

Previously I was writing files to Resources directory and doing a dynamic Ti.include(…). However, this doesn't work on iPhone which is fair enough. The problem is that if I extract the javascript file to another path (say Documents), I cannot give an absolute path to it on iPhone (at least). Javascript eval is not an option because an eval really isn't the same.

Can someone help me please?

Thank you.

  • You can 

    var file= Ti.Filesystem.getFile('path');
Ti.API.debug(file.nativePath);

    also there are options like
    Ti.Filesystem.documentsDirectory;
    check the filesystem API for further details.
