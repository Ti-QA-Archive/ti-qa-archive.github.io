  • Hi - you might have solved this by now, but I had a problem calling evaluate on xhr.responseXML.documentElement. Are you doing this? It worked fine when compiled against Ti SDK 1.6.2 and run in the simulator, but would cause unpredictable crashing in later Ti SDKs or when published to a device. I think evaluate is only meant to be called on the document object - i.e. xhr.responseXML.
