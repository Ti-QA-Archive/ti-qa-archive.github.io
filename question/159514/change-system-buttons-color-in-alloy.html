Change system buttons color in alloy

How can I change the color of the systemButton in alloy for iOS 7 ?
Has no effect: <Button systemButton="Titanium.UI.iPhone.SystemButton.COMPOSE" tintColor="#FF0000"></Button>

In xcode the property is calles tintcolor but it has no effect in titanium.

<Alloy>
    <TabGroup id="tabgroup">
        <Tab>
            <Window class="container">
                <RightNavButton>
                    <Button systemButton="Titanium.UI.iPhone.SystemButton.COMPOSE" ></Button>
                </RightNavButton>
                <View>
                    <Label text="Hallo !"></Label>
                </View>
            </Window>
        </Tab>
    </TabGroup>
</Alloy>

".container": {
    backgroundColor:"white"
},
"Label": {
    width: Ti.UI.SIZE,
    height: Ti.UI.SIZE,
    color: "#000"
},
