  • I found that if you edit the following file:

    ~/Library/Application Support/Titanium/mobilesdk/osx/3.1.0.GA/iphone/cli/commands/_build.js

    Go to line 2455 which should look something like this:

    } else if (fstat.isDirectory()) {
    recursivelyCopy(f, t, p);

    Modify that line to look like:

    } else if (fstat.isDirectory() || fstat.isSymbolicLink()) {
    recursivelyCopy(f, t, p);

    After the above change it was able to work for me.
