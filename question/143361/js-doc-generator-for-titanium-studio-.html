JS Doc Generator for Titanium Studio ?

Hi,

I'm looking for an efficient way to generate documentation from my javascript (titanium mobile) source code. Is there an easy way to do that and generate, for example, html pages out of my js files and their functions? Assuming that I am using a specific comment syntax of course.

Maybe a JSDoc plugin for Titanium Studio?

Thank you in advance.

PS: Currently Using Titanium Mobile SDK 2.1.3.GA on Titanium Studio 2.1.3.201208311808 on MacOS 10.7.5

  • It seems like the easiest approach is to run JSDoc toolkit from Titanium Studio.

    I followed this instructions here: http://code.google.com/p/jsdoc-toolkit/wiki/OtherTools
    I use a variable for project path, in order to target the Resources folder of the project I currently work on like this: 

    -jar jsrun.jar app/run.js "${project_loc}/Resources/" -t=templates/jsdoc/ -r

    Therefore with two clicks, you can generate an html based tree of documentation for your API/source code.
