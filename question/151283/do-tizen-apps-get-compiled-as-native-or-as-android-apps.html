Do Tizen apps get compiled as NATIVE or as ANDROID apps?

We all know Tizen's best feature is its ability to run Android apps, and I was just curious. The Titanium implementation on Tizen, does it simply run the Android build of your app, or does it compile your project as a 'native' Tizen app? I only ask, as the time from when you announced you were going to be adding Tizen, to when it was released was really fast! :)

Cheers

  • Titanium for Tizen generates neither a "native" Tizen or Android app. Instead it generates a Tizen "web-based" app. We took the Titanium Mobile Web platform, fixed a few compatibility issues, and expose the Tizen API's so that you can build your Tizen apps using the Titanium API's. Titanium does not support Tizen's "native" API's.
