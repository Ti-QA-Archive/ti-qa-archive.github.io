Background app: images animation with smooth transition

Hi.

I want to have a "dynamic" background in my ipad app. I mean, to have an animation in the background that shows 5 different images in an endless loop. I was able to do it with this:

fondoipad = Ti.UI.createImageView({
    images: ['../images/Fondos/FondoIpad01.jpg', '../images/Fondos/FondoIpad02.jpg', '../images/Fondos/FondoIpad03.jpg', '../images/Fondos/FondoIpad04.jpg'],
    duration: 3000, // milliseconds for each frame to show
    repeatCount: 0, // number of times the animation will repeat. 0 is indefinitely
});
win.add(fondoipad);
fondoipad.start(); // starts the animation

But it changes the image very "hard". I want it to have a transition or something smoother.

Is there a way to do it?

Would this eat the battery in a sec?

Thanks.
