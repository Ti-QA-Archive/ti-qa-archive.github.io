Delete app data on appstore/playstore update

Hi, I want to delete the app data including shared preferences, db…, when user updates the app to a new version. Basically every update should be like fresh install. The app is for android and ios

  • Check app version in alloy.js (For alloy project) or app.js (classic project) and if it is found not matching with your locally cached version then do your stuff. Here is example,

    alloy.js/app.js

    
if(Alloy.Globals.appversion && Ti.App.version !== Alloy.Globals.appversion) {
    // Deletion code goes over here
    Alloy.Globals.appversion = Ti.App.version;
} else if(! Alloy.Globals.appversion){
    Alloy.Globals.appversion = Ti.App.version;
}
