Update value/Run function when tab activated?

I have an iPad app with 4 tabs, loaded via URL property.

I have a global value the tabs can set in user data via Titanium.App.Properties.setList.

The problem I have now is that each tab only checks the value when it's first activated, and the value of that user data can be changed by the other tabs.

I would like to fire a function to check this value every time a tab is activated. I don't see this in the docs or KS.

TS 1.0, OS 10.7.1, SDK 1.7.2

    In each of your 4 *.js files for the tabs, add something like:

    var win = Ti.UI.currentWindow;
win.addEventListener('focus',function(e) {
    // do something here to refresh whatever you want
});
