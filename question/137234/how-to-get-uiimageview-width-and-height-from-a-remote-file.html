How to get UI.ImageView width and height from a remote file?

Hello,

Using the "old" 1.82 sdk, when I loaded a remote image into an ImageView (getting the file from a remote server, by example) letting the width and height properties empty I could then read these properties from the ImageView to know its size (which was the image size obtained from the file), but since I updated to 2.01 I'm not able to do that anymore since it returns 'undefined'.

I need to be able to know the size of the image before displaying it on screen so I can adjust the main View size and orientation to the image size and orientation (which can be portrait/landscape -but I dont't know it in advance, as I also don't know the image size- ).

Does somebody know how to achieve this? (Knowing the width and height of a remote image that I load into and Image View) ?

I'm trying this under Titanium 2.01 Mobile and IOS (Iphone/Ipad).

Thanks.

    hi,

    you can get the size of the image on load event of the imageview..like

    imageview.toImage().width
imageview.toImage().height
