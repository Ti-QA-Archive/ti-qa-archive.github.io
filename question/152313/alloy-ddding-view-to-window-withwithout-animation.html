Alloy ddding view to window with/without animation

Been wasting my day trying to do one of the most basic things ever.
I'm trying to add a view that will popup over an certain area of my window (body).

The window starts with a grid menu from a widget and i want to create a view that will come over that one from the left or right once I press a button.

I keep getting this error
on windows.js
Message:
expected.type.Object
Source:
this.view.add(view);

this is my mainPage.js

function openCalendar()
{
    $.test2 = Alloy.createController('win2');
    $.mainPage.add($.test2.container);
    $.test2.container.left = -320;
    $.test2.container.animate({left:320, duration:200});
}

win2.xml

<Alloy>
    <View backgroundColor:"red">
        <Button>LOL</Button>
    </View>
</Alloy>

and my mainPage.xml

<Alloy>
    <Window layout="vertical">
        <TableView height="15%" width="100%">
            <TableViewRow layout="horizontal">
                <Button width="50%" onClick="openCalendar">Calendar</Button>
                <Button width="50%" onClick="logout">Logout</Button>
            </TableViewRow>
        </TableView>
    <Require type="widget" src="com.appcelerator.buttongrid" id="buttongrid"/>
    </Window>
</Alloy>

what am I forgetting?

if i try to remove container from $.test2.container it just crashes no error or anything (app stopped working)

    I don't see "container" mentioned anywhere in your win2 controller. Also, the markup has errors.

    <Alloy>
    <View id="container" backgroundColor="red">
        <Button>LOL</Button>
    </View>
</Alloy>

    Why are you creating the instance of the controller as part of the parent controller? By this I mean the following line of code: $.test2 = Alloy.createController(&#39;win2&#39;); That seems like a potential for a memory leak if you are clearing that reference. I would just redo that with a local variable instead unless you have a really good reason to attach it to the controller:

    function openCalendar()
{
    var test2 = Alloy.createController('win2');
    $.mainPage.add(test2.container);
    test2.container.left = -320;
    test2.container.animate({left:320, duration:200});
}
