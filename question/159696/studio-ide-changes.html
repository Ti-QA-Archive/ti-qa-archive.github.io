  • Accepted Answer

    Check the project nature…maybe it somehow lost track of the fact that it was an Alloy project. (You are working with an Alloy, not traditional project, correct?)

    From the Project menu, choose Properties. On the left, select Project Natures. Make sure Alloy is checked and is the Primary.
