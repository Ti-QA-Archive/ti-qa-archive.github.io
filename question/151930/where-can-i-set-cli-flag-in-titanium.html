Where can I set CLI flag in titanium.

The CLI flag is –skip-js-minify. How do I add this flag? I want tobe able to run/debug app directly from the IDE, and not from the command line. How can I do this?

    When you select 'Run on iOS Device', the launch window that appears has an option to Select/Deselect 'Enable JavaScript Minification'.

    I'm on Ti Studio 3.1.0, not sure if this is on previous versions.
