Editing model columns

Is it possible to make changes to a model's columns, specifically adding and removing them? If I edit the config in the model file itself, I just get "invalid SQL statement" errors, so it seems the table hasn't actually been altered. Is there a way to update the table?   • Migrations can be used to modify the structure of a database. There is support for automatic changes (where you specify the new structure and the migration implements the code to update the DB) and custom changes (where you write the specific SQL).

    If you write your own SQL, keep in mind that the underlying dbms on mobile is SQLite, which offers less robust table modification options than the "big" dbmses. For example, there is no DROP COLUMN support in SQLite (reference). You have to use workarounds, such as temporary tables, to accomplish such changes … or just let the built-in migration code do that for you.
