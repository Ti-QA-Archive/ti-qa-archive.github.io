Determining clicks in Tableview Row - hasDetail

I need help on how to do the following:

In a tableview row, I have an image, then the name of a person, then I set the hasDetail property of the row and I get a detail view button appear on my tableview row. If the user selects the image, I want to do something, if the user taps the name I want to do something else and if the user taps the detail view row button I want to do yet a third different thing.

from the "click" function on the tableview I can get the function(e) values, but I don't know how to evaluate or set an event for the detailviewbutton as it is a property of the row, not an actual object as far as I can tell. Anyone?

    You have two basic choices:

    • Set event listeners on each object you care about in each row
    • Use the x offset of the row click to determine where they tapped

    • Typical behavior for a row with a hasDetail button is to have a tap anywhere on the row open the detail view. Since you evidently want to override that behavior, yes, I would suggest using e.x to determine where they click, then act accordingly. Remember to account for the device orientation and screen width.
