barcode scanner module

hi, i'm triyng to add this barcode scanner. i downloaded it and copy into my project folder. i wrote in the 'tiapp.xml' this code:

<modules>
<module platform="android">titanium-barcode</module>
</modules>

but when i run the app, the console shows me this warning:
'[WARN] : Could not find Titanium module id=titanium-barcode version=latest platform=android deploy-type=development'. why? I've searched for the solution on the site of titanium but i haven't found anything. can someone help me?

  • Accepted Answer

    It hasn't been compiled with the newest version of the api, that is why you are getting the errors. Many of the open source modules will work once recompiled with the current version of the api. If you download the source and compile, it should work. Your other option would be to try another module. I use the Acktie barcode module without any problems with the latest api. Do a search in the marketplace for Acktie or check their website here.

    Ray

    • What I do is install the module for the specific app. So, in the root directory of your app, there should be the following folder structure:

      modules\android\com.acktie.mobile.android.qr\1.3

      Then, instead of putting your module code manually into the tiapp.xml, you should be able to 'add' the module by clicking the green plus beside the module list. Then the acktie module should show up for you.
      BTW, I was compiling with sdk 3.2.3GA until just a few days ago. Updated my app to 3.30GA and no problems with either.

      Ray
