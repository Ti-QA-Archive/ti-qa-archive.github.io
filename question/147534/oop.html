    • Everything you do in Titanium is client side, which includes the Javascript and the PHP module above.

      You can use any server side scripting language you like, these are usually used to provide an API access to data on the server. You would pass fields to a URL, the server processes the request and then passes back data in the form of; JSON, XML, plain text or raw data.

      Script run on you server will not be directly run on your mobile device. YOu would also need to make sure you either have a constant data connection (unlikely) or some way to run without the connection.
