measure label height on runtime

I have a label on a window and want to create a second label under the first label. The first label have the height property set to 'auto'. Ho can I meassure the height on runtime?

var lbl1 = Ti.UI.createLabel({
            text:txt,
            top:70,
            left:5,
            right:5,
            bottom:5,
            height:'auto',
            color: '#fff',
            textAlign:'left',
            font:{fontSize:16, fontStyle:'bold'}
});

var lbl2 = Ti.UI.createLabel({
            text:'more text',
            top:lbl1.height+20, //lbl1.height is always 0 on runtime
            left:5,
            right:5,
            bottom:5,
            height:'auto',
            color: '#fff',
            textAlign:'left',
            font:{fontSize:16, fontStyle:'bold'}
});

I am using MobileSDK 1.7.2

    You can wrap both label into a view with layout:vertical or you can make
    lbl2.height = lbl1.toImage().height.

    The second option should be done after you add the lbl1 to it's parent.
