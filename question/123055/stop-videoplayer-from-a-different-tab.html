Stop videoplayer from a different tab

Hi.

I have a tabgroup with 3 tabs. 2 of them have a videoplayer streaming from 2 different sources. The 3d one is just text.

What I want is:

  1. When you go to the 2 tab, stop the videoplayer of the 1st one and start the video of the 2nd, and viceversa.
  2. I dont want to stop the players when the parent tab loses focus, because I want to get to the 3rd tab and be able to hear the audio in background regardless of the one playing.

My question:
How can I stop the player of a different tab? Or can i declare the players as "global" variables?

Thanks. (I hope i didn't make it so confiusing)

    Try this:
    In tab1 ( in the window of tab1 ) set an event

    
Ti.App.addEventListener('stop_player_one', function(){
    player_one.stop() // not sure this is the syntax
});

Ti.UI.currentWindow.addEvent('focus', function(){

    Ti.App.fireEvent('stop_player_two',{});
    player_one.start() // not sure this is the syntax

})

    in tab2 do this ( in the window of tab2 )

    Ti.App.addEventListener('stop_player_two', function(){
    player_two.stop() // not sure this is the syntax
});

Ti.UI.currentWindow.addEvent('focus', function(){

    Ti.App.fireEvent('stop_player_one',{});     
    player_two.start() // not sure this is the syntax

});

    You might have to add more stuff to check if the player is started or no, or to pause instead of stop, but I hope u get the idea.
