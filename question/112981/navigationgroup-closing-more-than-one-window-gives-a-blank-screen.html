NavigationGroup closing more than one window gives a blank screen

I'm having a problem when i'm trying to close 2 windows at a time thats been opened under a navigation group.

For example:

navgroup.open(window1);
navgroup.open(window2);
navgroup.open(window3);

in window3 i have a button with an event listener like:

closeButton.addEventLister("click", fuction(e) {
    navgroup.close(window3);
    navgroup.close(window2);
});

I'm expecting this to return me back to window 1, but instead i get a blank screen.

    Try like this:

    
navgroup.close(window3, {animated:false});
navgroup.close(window2, {animated:false});
