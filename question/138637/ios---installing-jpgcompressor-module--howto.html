  • The git contains only the important bits - the compression itself.
    You may wanna try to do it yourself, is a great way to learn :)

    Follow the info here

    https://wiki.appcelerator.org/display/guides/Android+Module+Development+Guide

    then inside your java file in the src folder (dig to reach it) copy the compress function and import the missing import libs in the header.

    have a look in app.js how to call compress, do it in your module's app.js and build the module.

    Is not too hard :)

    PS. didn't test it :)
