quicktigame2d creating an instance

So i'm using the quicktigame2d module and I essentially want to create a bunch objects that follow along a path. In order to do this I need to instance creating an object which I've done using a simple function.

function create(pathnumber){
    pathnumber = pathnumber+pathnumber;
    var startx = xpath[pathnumber];
    var starty = ypath[pathnumber];
    var endx = xpath[pathnumber+1];
    var endy = ypath[pathnumber+1];
    var currentx = startx;
    var currenty = starty;
    tiles = gamemod.createSpriteSheet({image:'tiles.png', width:32, height:32, x:startx, y:starty});
    tiles.animate(0, 6, 50, -1);
    game.addEventListener('enterframe', function(e) {
        if(currentx != endx){
            tiles.x = currentx + speed;
            currentx+=speed;
        }
        if(currenty != endy){
            tiles.y = currenty + speed;
            currenty+=speed;
        }
    });
    scene.add(tiles);
}

This works pretty well. It creates a new object everytime I press a button. The problem I'm having with this code is that if the animation is not complete, when I press the button to generate another object, the previous object's animation is stopped. Is there a better way to instance an object using this module?

Any help would be greatly appreciated. Thanks!
