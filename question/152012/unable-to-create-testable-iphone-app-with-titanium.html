Unable to create testable Iphone-App with Titanium

Hello!

Since a while I'm not longer able to build Apps for iPhone. There seems something wrong with the Developer-Key but i did not change it.

"Error details: Code Sign error: The identity 'iPhone Developer: XXX (xxx)' doesn't match any valid, non-expired certificate/private key pair in your keychains"

Has someone an idea how to fix this? Any tips?

    Hi

    • How long have you had your developer account?
      • If more than 12 months you need to renew and download renewed certificates
    • If you are just testing for the simulator then focus on the developer certificate
    • If you are testing on physical devices - review all and confirm you have a matching certificate for the app id yuo are using.
    • Delete all iOS related certificated - BE VERY CAREFUL and do one at a time, and replace with a new version from the apple developer site.
      • Delete certificates using xcode and then Menu > Window > Organiser
    • Have you changed computers?
      • If so you need to add all the certificates back in.
