Is it possible to change preferredProvider on runtime?

Hello all

I would like to listen to the "location" event, and if after X seconds I'm not getting a satisfactory accuracy, try and get my location from the cellular network. if I change the preferredProvider on runtime, will the next "location" event be from my cellular network?

Thanks!

  • Set the Titanium.Geolocation.preferredProvider property. You can set it to either Titanium.Geolocation.PROVIDER_GPS or Titanium.Geolocation.PROVIDER_NETWORK

    Titanium.Geolocation.preferredProvider = Titanium.Geolocation.PROVIDER_NETWORK;

    Titanium.Geolocation in API reference
