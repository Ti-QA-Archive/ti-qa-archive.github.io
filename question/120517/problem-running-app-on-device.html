Problem Running App on Device

Hello,

I've just purchased a new Mac Book Pro and I've been testing my latest app built with Titanium 1.2.2, Mobile SDK 1.6.2, I've had no problems testing in the simulator.

This morning when trying to load onto a device I get the following error:

[BEROR]Code Sign error: The identity 'iPhone Developer: xxxxxxxxxxxx (xxxxxxxxx)' doesn't match any valid certificate/private key pair in the default keychain

I've downloaded all certificates and provisioning profiles, so I'm not sure what's going on.

I've also deleted my build and recompiled again but to no availl.

Any help would be greatly appreciated.
