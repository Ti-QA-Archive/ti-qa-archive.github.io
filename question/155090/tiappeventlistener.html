    Hi,

    you need to remove the previous event listener before you add the new event listener with the same now,
    for doing this,

    when events added in pages check the property eventsAdded if already added remove and then add.

    if(Ti.App.Properties.getBool('eventsAdded', false)){
    Ti.App.removeEventListener('app:fromwebview', function()); 
    Ti.App.removeEventListener('app:savedata', function());
}

Ti.App.addEventListener('app:fromwebview', function()); 
Ti.App.addEventListener('app:savedata', function());
Ti.App.Properties.setBool('eventsAdded', true);

    Hope this will helps you.
