What does mean when a value or variable is undefined?

I have seen this undefined using Ti.API.info(some_variable);

I would like to know why happens that.

Thank you in advance.

  • It's undefined because you haven't defined it yet. Note that it returns the value undefined because JavaScript can check to see if it exists.

    However, if you try to access a property of something that is undefined, say, some_variable.thisProperty, you will get an error, because the object some_variable is undefined, so it can't even check for the property thisProperty.
