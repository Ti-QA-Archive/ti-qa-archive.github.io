how to include underscore in alloy

Hi,
I am trying to use com.appcelerator.buttongrid widget and in the docs there is _.bind operation which handles click event. But I could not bing the button scope to click function. Any help is appreciated. 

$.buttongrid.init({
    buttons: [
        { id: 'mm_btn1',title:"Title1" },
        { id: 'mm_btn2', title: 'Title2',btnCustomAction:"sample action",click: _.bind( function(){ alert('clicked: ' + this.btnCustomAction); }, this) }
    ],

When I click alert dialog says clicked: undefined.

    "this" value that you're trying to bind is actually global object (I think), therefore you're binding wrong "this" value. What you want is that "this" points to button to which click event is attached. To do that, just don't use "bind" :)
