Navigation

Hi,

I am making a application where i am going to use a Window divided into two views.
The views would be as right & left. The left view would be static and there would be images on it.
I want to show different views, in the right view, on a click on the image in left view.
Is that possible?
Or is there any other way I should follow for performing the before mentioned navigation?

Thanks,
Prafful

  • This is off course possible, because you are doing everything in the same window.
    First of all you need to create a view with a horizontal layout, which will include the two views you want to show. with this horizontal layout the views will be placed automatically next to each other. simply place the images you want in the left view and add eventlisteners to these images. what i should do is the next thing: when you click one of the images in the left view, i would first delete all children of the right view like this:

    image1.addEventListener("click" , function(e){
var children = rightView.getChilderen();
    for(i in children){
        rightView.remove(children[i]);
    }
    // AND NOW PLACE NOW STUFF ON THE RIGHT WINDOW
});

    Or when you will generate the right views earlier, you can simply first remove the view that is vissible, and then add the new rightView to the normal view.

    So you will have a window, within this window you will have a view with a horizontal layout. in this view you have a left view with images(that containts the click event) and you will have a right view which holds the content you want to show.
