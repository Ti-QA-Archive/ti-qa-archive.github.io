  • i never use the titanium app to install my apps. I just use adb to push the .apk over to my phone. Not sure if this not recommended but i find it faster and more reliable. If you are not familiar with adb here's a quick overview (for windows).

    1:) Open command prompt and navigate to your android\tools directory. Mine is located in c:\android\tools, i moved to this location for other reasons, yours will most likely be in your program files directory.

    2:) Test your app so the emulator launches (this creates the latest .apk for your app in the "build\android\bin" folder. By default the app is just called app.apk

    3:) in the command prompt type "adb -d push "PATH TO APP.APK FILE" /system/app/anyname.apk

    An example for me would be "adb -d push "G:\apps\BenchItUp\build\android\bin\app.apk" /system/app/BenchItUp.apk

    4:) I keep this window open so after i make changes to my app, i just rebuild with a test and then just hit the up arrow in the cmd prompt, that last command comes back up and i just hit enter.
