Create an image from a canvas and upload it to a server

Hi,

I'm a newbie to appcelerator, and I have a project that could be the perfect candidate to use it.

The problem is, I have to develop an app that accepts signatures from clients and I thought that could be solved by using a capacitive stylus and something like a canvas where the client could write his signature.

With the signature I then have to create an image (like a png for example) and upload it to a server.

Is this possible on appcelerator? Can anyone point me on some direction? Or is this a case where we can only do it natively?

Thanks in advance,
