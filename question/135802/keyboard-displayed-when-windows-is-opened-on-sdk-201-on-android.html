Keyboard displayed when windows is opened on SDK 2.0.1 on Android

Hi,

I have a window that has an input field in it, and when this window is opened that field immediately get the focus and shows the keyboard (only happens on Android). There is no focus that is given to that field any where in the code. I don't want that to happen because the keyboard hides the content below.

Anyone knows how to solve that?

Thanks

  • Hide your textfield at time of creation and show it when window opened.

    Hope this trick will help you out.

    • var tf = Ti.UI.createTextField();
tf.hide();

win.addEventListener('open', function() {
    setTimeout(function(){
        tf.show();    
    }, 1000) ;
});

  • Hello ophir,

    Try this…

    
var tf = Ti.UI.createTextField();

win.addEventListener('focus', function() {
    setTimeout(function(){
        tf.blur();    
    }, 1000) ;
});
