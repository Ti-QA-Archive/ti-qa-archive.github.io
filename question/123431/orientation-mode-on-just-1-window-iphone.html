Orientation mode on just 1 window iPhone

Hello

I have all orientation modes enabled on my app, except on my homepage I only want to use Portrait mode. When I go open this page it gets set on portrait mode, but when i hold my ipad straight and on the side again, it switches from orientation mode again.

How can I lock it that it won't switch to any other mode than portrait?

The code I am using is:

Titanium.UI.currentWindow.orientationModes = [Titanium.UI.PORTRAIT];
Titanium.UI.orientation = Titanium.UI.PORTRAIT;

i'm using iPhone SDK 4.2 and titanium SDK 1.6.2.

Tjeu

  • Is there a specific reason that you're using such an old SDK? If not, I suggest updating to 1.7.2 or at least 1.7.1. Aside from that, check your tiapp.xml and make sure the orientations node reads like this:

    <iphone>
  <orientations device="ipad">
    <orientation>Ti.UI.PORTRAIT</orientation>
  </orientations>
</iphone>

    Apple's HIG says your app should work in all orientations on iPad. Be aware that you might face some extra scrutiny from Apple's reviewers when you submit your app to the App Store.
