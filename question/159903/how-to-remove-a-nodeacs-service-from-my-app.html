How to remove a Node.ACS service from my app

Hi.

I have an existing iOS Ti App with Cloud enabled.
I wanted to create a server-side procedure to return data from ACS objects.

So… I created a Node.ACS service inside the app. (Right button click -> New -> Node.ACS Service).
I named the service "test" (my bad).

Then I created a Node.ACS method (Right button click -> New -> Node.ACS Method) just to see what it does as it was the first time I worked with Node.ACS.

I think I understand now after digging into the files that were created … so… I want to start over, I don't want a service called "test" in my app, so I just want to remove or de-integrate the service and create a new one.

I can't find the way… I ended removing the files manually (deleting those files related to Node.ACS from my resources folder).

I'm stuck now.

I can't create a new service as the option is not present. It only displays New Method (to be created into "test" service).
But as I deleted those files, I can't even create a new method.

So, how do I remove the integration of this service from my app?
I want my app to be clean as it was before I started messing up with Node :(

Thank you very much!

  • Here are the steps that worked for me:

    1. Remove the acs folder from the root of your project
    2. From the tiapp.xml, remove all the ACS keys
    3. Edit the hidden .project file in the root of your project and delete the following line: 
      <nature>com.appcelerator.titanium.acs.core.nodejsnature</nature>
    4. Then, in Studio, select a folder and press F5 to refresh the view

    After that, I could right-click, choose New, ACS Service
