Full height on 4 inch simulator but not on iPhone 5S device

I know that we need to have a 640 x 1136 default-568h@2x.png splash screen.

The default-568h@2x.png file is OK and at the right place.

My app is perfectly full height in 4 inch simulator but… not on iPhone 5S device and I can't find any solution to this (I tried with SDK 3.2.1 and 3.2.0).

Does anyone have an idea ?

Thanks a lot.

  • The simulator is not case sensitive, but devices are. Make sure the file's name is "Default-568h@2x.png" (with a capital D). Also, confirm that your file is not named -586 (I've done that dyslexic flip of the 8 & 6 more than once). If that's not the case, rename the IPA file to .zip, open it up, and confirm that the file is being copied into the archive.
