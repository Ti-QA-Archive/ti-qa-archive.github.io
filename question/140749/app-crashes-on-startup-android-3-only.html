App Crashes on Startup. Android 3+ only.

Well i've tried for weeks to solve this issue on my own, but have had no luck. I'm having a startup crash with my app on certain versions of Android (3+). I've examined the console output, but i'm not really sure how to accurately debug with Android because the output errors are a bit greek to me.

Essentially there are a lot of warnings, then the actual errors. I'm not sure if the warnings turn into the error or if they are separate.

The second warning line is something that stood out to me though. The "NetworkOnMainThreadException". I did some looking into this and found an article that talks about this. It states this.

"The reason why your application crashes on Android versions 3.0 and above, but works fine on Android 2.x is because HoneyComb and Ice Cream Sandwich is much stricter about abuse against the UI Thread. For example, when an Android device running HoneyComb or above detects a network access on the UI thread, a NetworkOnMainThreadException will be thrown"

Source

So i'm not sure if i'm looking into the correct issue or not. Not to mention that to add further confusion, The app seems to work on certain devices running Android 3+ so i'm really not sure where to even start to track down this problem.

Hope the community can help me out on this one. Thanks in advance.
