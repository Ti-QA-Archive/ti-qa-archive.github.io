    • As per your type of object or datatype you can store your data in to property,

      Ti.App.Properties.setString('Key','value');

      for more help you can see this link.

      and you can get its value anywhere in the code by accessing the property.

      Ti.App.Properties.getString('Key');

      and another method to store data into variable is

      Ti.App.variableName = "Value" or "object";

      you can access it by simply assign its value to another variable or direct access it like any variable.

      var temp = Ti.App.variableName;
//or
Ti.API.info("Testing Application Variable : " + Ti.App.variableName);

      you can choose any of the method but second one is not advisable, because it lost data sometimes. both have its limitations.

      when you are going to choose Property to store user data then remember one thing that properties are wipe out when your application uninstalled from device.
