Event controller in different .js when event is declared by XML

Hi guys!
I have this situation:

index.xml:

<Alloy>
    <Window>
        <Require id="iconGroupAlfa" src="iconGroupToken" />
        <Require id="iconGroupBeta" src="iconGroupToken" />
        <Require id="iconGroupGamma" src="iconGroupToken" />
        <Require id="iconGroupDelta" src="iconGroupToken" />
        <Require id="iconGroupEpsilon" src="iconGroupToken" />
    </Window>
</Alloy>

iconGroupToken.xml:

<Alloy>
    <ImageView id="iconAustria" onClick="doThis" />
    <ImageView id="iconBelgium" onClick="doThat" />
</Alloy>

and index.js:

function doThis()
{
    // do something
}

function doThat()
{
    // do something else
}

I get an Uncaught ReferenceError: doThis is not defined. Same with doThat. I also tried using onClick=&#39;index&#x2F;doThis&#39; with no result and same error. Do I have to make an iconGroupToken.js controller or there is another way?
Thanks!
