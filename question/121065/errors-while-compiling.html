  • It appears to be caused by the file permissions.
    Go to the "/Library/Application Support/Titanium/mobilesdk/osx/1.7.0/iphone/" directory in terminal (easiest is to use finder to go to that folder then drop it into terminal) and then type "chmod a+rwx ." to change permissions.
