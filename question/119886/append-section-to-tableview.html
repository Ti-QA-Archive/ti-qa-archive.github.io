Append section to tableview

Hello

How can I append an entire section to a tableview?

I've read this post but it says to use the tableview.setData(data) command. This refreshes the entire tableview, and i want to keep the data that is already in it.

Can anyone help me out?

  • The only proper way to add a row to an existing TableView is by using one of those 3 methods.

    • appendRow
    • insertRowBefore
    • insertRowAfter

    Now, if you have many rows to add, you could call one of those methods inside a loop and add your rows.

    If the number of rows you what to add is too great, you may have no other choice to clear the data collection and replace it with a new one.

    One could imagine code that:

    1. Copies the existing data into a new array variable.
    2. Adds the new rows to the array.
    3. Clears the data collection from the TableView.
    4. Sets the array to the TableView using the setData method.
