Button wont dispatch click

Here is my code for the button creation

var fields = Ti.UI.createView({ top:250, left:38, width:250, height:80 });

&#x2F;&#x2F; login button creation var loginBtn = Ti.UI.createButton({ width:250, height:40, top:100, title:&#39;test&#39; });

loginBtn.addEventListener(&#39;click&#39;,function(){ Ti.API.log(&#39;test&#39;) })

fields.add(loginBtn);

the event never fires :/
I have tried singletap, touchstart etc dont get a single event

  • Accepted Answer

    The problem is that your "fields" View is specified with a height of 80, but the top of the Button is positioned at 100 (relative to the top edge of the View). Thus the button is located outside the boundaries of the View and is basically disallowed any touch events.

    If you remove the "top" parameter for the Button, or increase the height of the View, the Button will now show the events you are expecting.
