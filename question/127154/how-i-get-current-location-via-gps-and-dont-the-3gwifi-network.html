how I get current location via GPS and don´t the (3g/wifi) network.

Hi all … I have some issue with Titanium Studio, i need to get the exactly location in order to mark as an issue in my app. I´m already use this code (Kitchenshrink) to get the location ( latitude and longitude ) but I get from the API is the nearest location from my provider internet (3G). The idea is get the gps (lat,lon) around 5/10 meters of distance.

This is an example from my code … I hope you can help me i´m so excited to publish this app on the apple store! thanks a lot !!!




Ti.Geolocation.preferredProvider = Titanium.Geolocation.PROVIDER_GPS;
Ti.Geolocation.purpose = "GPS"; 
Titanium.Geolocation.accuracy=Titanium.Geolocation.ACCURACY_BEST;
Titanium.Geolocation.distanceFilter=0;

if(Titanium.Geolocation.locationServicesEnabled==false){ 
    Titanium.UI.createAlertDialog({title:'gps', message:'No puedo conseguir su ubicación actual, active GPS',buttonNames:['Aceptar'] }).show();
} else {
        Titanium.Geolocation.getCurrentPosition(function(e){
            if (e.error) {
                    // Show error
                return;
                } else {
                    longitude = e.coords.longitude;
                    latitude = e.coords.latitude;  
                       Ti.App.Properties.setDouble('Latitude',latitude);
                Ti.App.Properties.setDouble('Longitude',longitude);     
            ShowMap(latitude,longitude);
                }
        });
}

  • How many times do you fire the get location?.

    Normally a gotcha on getting the location is that the first time will not always return your exact location. I always call it twice first time around this should guarantee getting the actual location.
