  • You would need to construct your SQL statement using the LIKE operator, return the filtered dataset and then populate your TableView.

    For example, I want to show all customers with the first name of James, it would look like this:

    SELECT * FROM tblCustomer WHERE firstName LIKE '%JAMES%' OR lastName LIKE '%JAMES%'

    That would return me rows where the first or last name columns contained the word James.

    I think this is what you are asking for.

    You can read more on the LIKE operator here: http://www.sqlite.org/lang_expr.html
