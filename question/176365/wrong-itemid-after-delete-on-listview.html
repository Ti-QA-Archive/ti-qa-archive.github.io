Wrong ItemID After delete on Listview

hi how i can retrive the item deleted by the delete event with listview?

i have a listview with delete event

$.listaLotti.addEventListener('delete',report);

but in the function with

function report (e) {
    APP.log('info',e.itemId);
   APP.logJson('info',$.listaLotti.sections[e.sectionIndex].getItemAt(e.itemIndex));
}

i have the item at index but after the delete (like the item is deleted before this event)

i need the correct id to remove the item from the collection

    The documentation is correct. The sectionIndex and itemIndex do correspond to listview state before the user action. There is no sectionIndex or itemIndex for the item after it has been deleted from the listview.

    The two properties are sent out for the developer to update their data set on the javascript side. Listview automatically updates its internal data set on delete event. Querying the listview or section for deleted data is incorrect.
