  • Hello, Amit, In sample TextField code hintText property is working. I checked in 3.4.0.RC

    var win = Ti.UI.createWindow({
  backgroundColor: 'white'
});

var textField = Ti.UI.createTextField({
  borderStyle: Ti.UI.INPUT_BORDERSTYLE_ROUNDED,
  color: '#336699',
  hintText: 'TEXTFIELD',  
  width: 250, height: 60
});

win.add(textField);
win.open();
