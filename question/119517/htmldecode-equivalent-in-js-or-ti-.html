  • This isn't exactly ideal, but something along the lines of this should work :

    var
    decStr,
    decoder=Ti.UI.createWebView(),
        // htmlentities encoded string
    encStr= "A 'quote' is <b>bold</b&gt"; 

decStr=decoder.evalJS("" + 
    "var dec=document.createElement('div');" +
    "dec.innerHTML=\""  + encStr + "\";"     +
    "dec.innerText;"
);

    decStr should resolve to: "A 'quote' is <b>bold</b>"

    If you're in a crunch… refining the above into a proper method can get you through… but in the long run you should look at refining your feed source.
