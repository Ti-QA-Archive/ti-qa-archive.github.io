Cursor color in a textbox

Hi
When I use textbox with a white backgroundColor, I cannot see my text cursor as it is white as well. This seems to be a problem with Android 4 and SDK 3.2.x, because lower versions did not give me this problem. Is there a way to change the color of the cursor?

Thanks
Gino

    Hi Gino,

    The easiest way I have found of doing this is to use your own Android styles.

    Modify your tiapp.xml with the following:

    <android xmlns:android="http://schemas.android.com/apk/res/android">
    <manifest>
        <application android:theme="@style/Theme.Custom">
        </application>
    </manifest>
</android>

    Then create a styles.xml file and a colors.xml file at platform&#x2F;android&#x2F;res&#x2F;values, relative to the main project folder

    In the styles.xml file add the following code:

    <?xml version="1.0" encoding="utf-8"?>

<style name="Theme.Custom" parent="@android:style/Theme.Holo.Light">
        <item name="android:editTextStyle">@style/editText</item>
</style>

<style name="editText" parent="@android:style/Widget.Holo.Light.EditText">
    <item name="android:textCursorDrawable">@null</item>
    <item name="android:textColorHint">@color/cursor</item>
</style>

    and in the colors.xml file add in the cursor color

    <?xml version="1.0" encoding="utf-8"?>

<resources>
    <color name="cursor">#FFBEBEBE</color>
</resources>

    Hope this helps!

    • Been diggin' a bit more. If I extend your styles.xml with a <resources> element, it works!

      So: 

      <resources>

    <style name="Theme.Custom" parent="@android:style/Theme.Holo.Light">
            <item name="android:editTextStyle">@style/editText</item>
    </style>

    <style name="editText" parent="@android:style/Widget.Holo.Light.EditText">
        <item name="android:textCursorDrawable">@null</item>
        <item name="android:textColorHint">@color/cursor</item>
    </style>
</resources>

      Thanks!!
