Cannot Enable Cloud Service on Titanium

Hi,
I want to enable my existing app to use Appcelerator Cloud Services so I clicked 'Enable' button under cloud services in tiapp.xml. But I can't find my application in my.appcelerator.com/apps.
When I create another new project and enable cloud services, it works.
How can I enable my existing app to use ACS?

As an additional information, the account who created this app and enable cloud services are different. The first account was not activated, so I tried with second new account.
(I tried to activate the first one but I cannot get email which has activate link…)
