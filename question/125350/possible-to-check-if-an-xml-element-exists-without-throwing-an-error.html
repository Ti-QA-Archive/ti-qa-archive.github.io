Possible to check if an XML element exists without throwing an error?

If I run this command:

var item = subitem.getElementsByTagName("item").item(0).text;

If the item Tag doesn't actually exist in the XML feed, then the program throws an error obviously.

Is there a way to check to see if the tag exists without throwing an error?

I tried this 

if (subitem.getElementsByTagName("item").length>0)

But that doesn't work for me…

  • what I did was write a helper function which wraps it with a try catch block

    function getItem(_doc, _tag) {
    try {
        return _doc.getElementsByTagName(_tag);
    } catch (ee) {
        return null;
    }
}
