ACS Push Notifications

We are trying to add push notification support, but the instructions tell us to log into the Dashboard, which as far as I know is used for paid clients. Normally we manage all of our apps via the developer.appcelerator.com "control panel". It has a section for push notifications for each app, but does not have any way to access this step:

Click the Services tab, then click Show Key under Cloud / Performance / Analytics.

Since the control panel does not have a Services tab.

Does this mean you cannot use the free (low monthly API calls) version of ACS with Push Notifications?

  • Yes, you can send push notifications with a free account from the 'Push Notifications' tab of the "Community" web console:

    my.appcelerator.com/apps

    The instructions you mention are for Platform subscribers using the Platform Services SDK for native iOS and Android applications.

    • Ok, it seems you're using the old ACS iOS SDK https://github.com/appcelerator/acs-ios-sdk/?

      That SDK was deprecated in favor of the Platform Services SDK, and the docs were updated for the new SDK. For now, you can view the old SDK docs (which contain more info about setting up push notifications) on my acs-ios-sdk fork:

      https://github.com/tstatler/acs-ios-sdk/blob/readme-update/Readme.mdown

      I've opened a PR to make this the new main repo readme. But please note that Appcelerator is no longer updating the ACS iOS SDK, fixing bugs, etc. as the focus is on the Platform Services SDK.
