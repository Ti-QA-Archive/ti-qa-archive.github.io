Window's 'open event' broken on Android with Ti 1.8.1?

Hi,

The following code used to work on IPhone and Android with Ti 1.8.0.1.
I switched to Ti 1.8.1 and the 'open' event is not triggered anymore on Android (works fine on IPhone).

The only thing I did was updating Titatnium SDK to 1.8.1 without any code changes.
Using ti 1.8.0.1 (by changing it into Tiapp.xml) makes this works fine, i.e., the 'open event' is triggered.

var myWindow = Ti.UI.createWindow({
    title: 'Title',
    backgroundColor: '#ff0000',
});

// open event not triggered on Android with Ti 1.8.1
myWindow.addEventListener('open', function(event){
    alert("My window opened: "+event.toString()); 
});

myWindow.open();

Am I the first to encounter this?

Note: FYI, 'myWindow' is passed to a TabGroup tab as the 'window' property'.
