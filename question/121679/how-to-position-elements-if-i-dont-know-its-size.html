How to position elements if I don't know its size?

Hello,

I am developing an events app, and for that I need to insert some data into the view. My problem is that each event data length is different and I want to divide data blocks with an image separator.

So my question is how can I know where I should put the separator? With html it would be like this

<div id="text">the text</div>
<div id="separator">image separator</div>

So, is there any way of positioning elements below each other and not in an absolute position?

Any help is much appreciated,
thanks

    Sure. Check out the "Vertical Layout" example in the Kitchen Sink. Though, you'll have to make a unique 'separator' view for each instance. Ugh.

    Consider populating a Table View with your data, then maybe you could jigger the background image of the Table Row(s) to display your separators.
