Image 'carousel' scrolling: how to make images stop in the middle of the screen ?

Hi,

I'm trying a small example which allow to scroll a set of images horizontally. I succeeded partially, but I need some help for fine tuning. See, the images are scrollable but there are two quirks:

  • The paddings are inconsistant. First image has no left padding, and padding between images is not always the same.

  • I would like the images to stop in the middle of the screen, the closest image in view sticking to the middle automatically once the touch is released.

Here are samples from my code :

XML

<ScrollView id="mainView" layout="horizontal" scrollType="horizontal" contentHeight="auto"></ScrollView>

JS

var imageCollection = [
'/dummy_poster.png',
'/dummy_poster.png',
'/dummy_poster.png'
];

var viewCollection = [];
for (var i = 0; i < imageCollection.length; i++) {      
  var img = Ti.UI.createImageView({
    image:imageCollection[i],
    left: i * 25
  });
  $.mainView.contentWidth = (imageCollection.length * 185) + 270;
  $.mainView.height = 285;
  viewCollection.push(img);
}

$.mainView.add(viewCollection);

(note: portions of this code has been found on this forum)

So, are my requests easy to implement ? Or does it need rewriting or something ?
