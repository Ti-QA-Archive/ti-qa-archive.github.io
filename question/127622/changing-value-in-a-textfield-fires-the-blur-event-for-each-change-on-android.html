Changing value in a textfield fires the blur event for each change? on Android

I'm using SDK 1.8 from 18 october.

I have several textfields in a tableview, and when I change the text inside a textfield the blur event is also fired each time. Why is this happening? Which leads to my application crashing if I erase all text in the textfield..

Anyone else experienced this problem?

  • I think I found out why it crashed.
    I had this check:

                        if(e.source.value == "") {
                        e.source.value = 0;
                    }

    If I changed the above to this:

                        if(e.source.value == null) {
                        e.source.value = 0;
                    }

    It works, but the blur event is still fired every time I change the value in the textfield, which it shouldn't imo.
