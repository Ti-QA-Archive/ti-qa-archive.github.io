    • If you want to change application name than you need to create app.xml file in your project and put it into i18n folder.In your app.xml file put this code and clean build and test it.

      
<?xml version="1.0" encoding="UTF-8"?>
<resources>
    <string name="appname">My App Name</string>
</resources>
