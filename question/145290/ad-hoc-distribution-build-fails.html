    Hi

    Some things for you to check;

    • Have you run Project > Clean (after closing the simulator), before you try to compile to adhoc or appstore?
    • Have you created a specific certificate for either ad-hoc or appstore as you cannot use the same one as you have for your own device?

    Either of these moved you on?

    • You are welcome. Do not forget to mark this as solved so others can benefit from the process to solution.

      Build log can be found in the following path.

      [app folder]\build\iphone\build\build.log

      This is up one level from your resources folder.

      It is created each time you build, for whatever target you built the app for. The contents can be complicated as it is a log, but jump to the bottom and work up and things may jump out at you.
