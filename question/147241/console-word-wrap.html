    • Sorry about that!

      To wrap the console go to:

      Preferences > Run/Debug > Console. Check the "Fixed width console" box.

      While digging around for the above answer I stumbled upon how to set Studio to word wrap by default.

      To have studio word wrap all files go to:

      Preferences > Editors. Check the "Enable word wrap (requires editor restart)" box.
