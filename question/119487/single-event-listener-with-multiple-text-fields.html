Single Event Listener with Multiple Text Fields

Hi,

I am wondering how I can have one event listener (onclick) which is fired when any of the text fields are clicked (clicked and the keyboard pops up).

Thanks in advance.

  • Hi Jay.

    I think that you can achieve that by using named function (or by assigning anonymous function to variable) and than attach that function to any text field you want. For example:

    
var eventHandler = function(e) {
    //do something with "e"
    Ti.API.debug(e);
};

var my_tf1 = Ti.UI.createTextField();
var my_tf2 = Ti.UI.createTextField();
var my_tf3 = Ti.UI.createTextField();

my_tf1.addEventListener('click', eventHandler);
my_tf2.addEventListener('click', eventHandler);
my_tf3.addEventListener('click', eventHandler);

    In "eventHandler" function, "e" contains information which text field is clicked. I think "e.source" should be the source of the event (in particular, one of three text fields that I created).

    Hope this helps (I'm not sure have I understood the question right :) ).
