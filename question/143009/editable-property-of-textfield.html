Editable property of textfield

Hi guys,

I am working on a mobile app, using 2.1.3 version of titanium SDK and using ios 6 platform for ios simulator.

In my app, i have a textfield in which i have used rightbutton propoerty to accomodate a button. on the click of the button i opening popover which has some values and on selecting among those values, selected value gets assigned to the textfield.
Now i want this textfield non-editable. I set editable property of textfield false, but it didn't workout as i wanted. If i set enable property false, then it disables the text field i.e. textfield is unclickable now. but side-effect of this is it disables the rightbutton also.
Is there is any way so the textfield will be uneditable and rightbutton will be clickable too..!!!

Please help..I will be thankful.

  • Hi Harshad, you can try both of these solution. solution one may not be the exact solution but if solution two does not work then try solution one. i did not try these solution, so you try and let me know.
