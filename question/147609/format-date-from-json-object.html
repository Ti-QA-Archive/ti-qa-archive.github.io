Format Date from JSON object

I'm getting a datetime value from a web service and populating the return into a JSON object. When I put that value into a label, as so

json.PreviousDate

It displays like this: 2/11/2013 12:00:00 AM. I'm having a hard time figuring out the format to make it simply: 2/11/2013. I'm sure it's simple, just not finding it.

Thanks!
