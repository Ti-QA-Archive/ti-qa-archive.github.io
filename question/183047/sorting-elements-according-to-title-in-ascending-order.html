    Hi Rujutu, actually your question is related to Javascript, so there are so many ways to achieve your goal.

    The most simple one is using inbuilt Javascript sort() function.

    Here is the sample code of your problem.

    var titles = ['Rajutu', 'Prashant', 'Titanium', 'It works'];

Ti.API.info(titles.sort());    // It works,Prashant,Rajutu,Titanium

    You can try this code here Sort Array

    One more link W3Schools Javascript Arrays

    That's it for sorting the titles, & now you can make custom views in Titanium according to your requirements.

    • Thank you Prashant for a quick reply… I tried the custom sort function but it didn;t work.. what worked for me is:

      objs.sort(function (a,b) {

      var nameA=a.last_nom.toLowerCase(), nameB=b.last_nom.toLowerCase()

      if (nameA < nameB)
      return -1;
      if (nameA > nameB)
      return 1;
      return 0; //no sorting

      }); // where obj is the array which you need to sort
