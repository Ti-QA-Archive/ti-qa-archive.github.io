  • Yes, it is possible, but you'll have to write a Titanium module to expose the component with a javascript API. This will require you to write native code (Obj-C for iOS, Java for Android), but it's reasonably well documented for both iOS and Android. I suggest you start with the iOS guide, even if you want to get into Android, because it covers the background theory regarding Titanium module architecture better.
