    Hi

    In order for the buttons to do something we need to add an event listener and give it the swap code. This is easy enough when you have seen the process;

    b1.addEventListener('click', function (e) {
   imageView0.setImage('/images/red.jpg');
});
b2.addEventListener('click', function (e) {
   imageView0.setImage('/images/blue.jpg');
});
// repeat

    Please note to save yourself pain and trouble later; always prefix images with /. You will thank me later.

    This is the most basic way to achieve your goal. There are lots of ways to achieve most things. I will add a comment to give you a better way.

    • Tweak your image properties to include a custom property;

      var b1 = Titanium.UI.createButton({
        swatch: '/images/red.jpg', // add this specific to the button

    });
    win.add(b1);
// repeat the addition if the swatch value

      Use this generic handler for clicks, it takes the property from the source button that clicked and then set the placeholder image to that file a,e.

      var buttonHandler = function (e) {
   imageView0.setImage(e.source.swatch);
});
b1.addEventListener('click', buttonHandler);
b2.addEventListener('click', buttonHandler);

  • I've figured out how to do it. It wasn't really that hard. I probably should've tried harder before posting the question. Anyway, in case anyone else is looking for how to do this, here's the code.

    var win = Titanium.UI.currentWindow;
win.backgroundColor = '#ffffff';


//IMAGE VIEW STARTS

var imageView0 = Titanium.UI.createImageView({
    image:'images/red.jpg',
    width:475,
    height:200,
    top:10
});

win.add(imageView0);


//SWATCH BUTTONS

var b1 = Titanium.UI.createButton({
        backgroundImage:'images/red_swatch.jpg',
        borderRadius:5,
        width:44,
        height:20,
        top:230,
        left:173
    });

win.add(b1);

b1.addEventListener('click',function(e)
{
    var imageSwap = Titanium.UI.createImageView({
    image:'images/red.jpg',
    width:475,
    height:200,
    top:10
    });

    win.add(imageSwap);
});

var b2 = Titanium.UI.createButton({
        backgroundImage:'images/blue_swatch.jpg',
        borderRadius:5,
        width:44,
        height:20,
        top:230,
        left:235
    });

win.add(b2);

b2.addEventListener('click',function(e)
{
    var imageSwap = Titanium.UI.createImageView({
    image:'images/blue.jpg',
    width:475,
    height:200,
    top:10
    });

    win.add(imageSwap);
});
