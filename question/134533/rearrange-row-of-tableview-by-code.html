  • Thank you for your explanation akihiro,

    if you put your child rows nested in to row then it moves automatically otherwise you can set data by setData method after move event complete with parameter e.fromIndex and e.index parameter. for that you have to move data in array and then set updated data with animation that looks like dynamic changing.

    tableView.setData(data,{animationStyle:Titanium.UI.iPhone.RowAnimationStyle.DOWN});
