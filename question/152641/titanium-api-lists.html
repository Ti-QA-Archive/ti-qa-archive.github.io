    Here is a short answer item by item.

    • Accelerometer YES
    • Barcode YES (but using a native external module)
    • Calendar YES
    • Camera YES
    • Compass YES
    • Contacts YES
    • File System YES
    • GPS YES
    • Media YES
    • Menu YES
    • NFC YES (but using a native external module)
    • Network YES
    • Notification YES
    • Screen rotation YES
    • Storage YES
    • Globalization YES
    • InAppBrowser YES
    • Splashscreen YES
    • Devices YES
    • Events YES
    • Connection YES

    Hope this clears this up for you.
