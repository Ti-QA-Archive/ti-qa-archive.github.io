  • It appears to me that as long as you do a full compile each time before you distribute the app, you could use the modification timestamp of any file you know to be in the resources folder. Untested, something like this:

    var f = Ti.Filesystem.getFile(Ti.Filesystem.resourcesDirectory, 'Default.png');
var dt = new Date(f.modificationTimestamp());
alert('Compiled ' + dt.toString();

    Force a full compile in Ti Studio by using menu Project > Clean. In Ti Developer switch to the Edit tab and click the Save button prior to building the project.
