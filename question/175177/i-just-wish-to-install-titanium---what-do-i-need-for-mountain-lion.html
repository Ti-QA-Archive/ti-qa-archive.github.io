    Hi

    I cannot work out why you are getting so many errors. I have seen the many questions you have posted (I have tried to assist in one or two) and all point to a big install issue somewhere.

    • Titanium Compatibility Matrix

    Your setup does appear to be fine - I have Mavericks running with xcode 5.1 and latest Titanium (in fact beyond latest in the the alphas) so you should be fine.

    But clearly all is not fine. The two biggest culprits I would go for right now (given you have tried lots and lots so far);

    • Check your java install - it MUST be 1.6 (AKA 6) NOT 1.7 (AKA 7).
    • Remove node and re-install it.

    The answer I gave yesterday has the details to started;

    • http://developer.appcelerator.com/question/175150/how-do-i-uninstall-completely-titanium

    But also check out Node - jump to the website for install/uninstall details;

    • Installing Node

    Teh only other thing that jumps to mind is permissions - are you configured as admin when you are installing and do any related folders have sufficient rights?
