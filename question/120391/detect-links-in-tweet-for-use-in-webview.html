  • Here is your code for the link extraction:

    http://pastie.org/1998618

    You will see that it's pretty straight forward.

    As for loading the different links in multiple WebViews, you could create one Tab per link. Each Tab would contain its own WebView (kinda like a tabbed browser).

    Simply pass the link as the url parameter when creating the WebView.

    var wb = Ti.UI.createWebView({
   url: link_from_tweet
});
