  • To test 'camera functionality' on the simulator, you can 'fake it' by loading a file into the error: function of yourshowCamera() call.

    error:function(error){
    //alert('error');
    // normal error codes would go here, but we use this on the simulator to test subscreens since there is no camera
    var event = {};
    event.media = {};
    var f = Ti.Filesystem.resourcesDirectory + 'images/dinner.jpg';
    event.media.file = {};
    event.media.file.name = 'dinner.jpg';
    event.media.length = 10000;

    // Make your calls to play with the event data

}
