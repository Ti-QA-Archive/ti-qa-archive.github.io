  • What do you mean "cross button?"

    You can use a standard view (Ti.UI.createView) created as a child element of a larger view, and then listen for the click event on the child view, if you'd like. For instance:

    // This makes the holder. You can add a background image/etc. to this view
var holder = Ti.UI.createView({
    top: 0,
    bottom: 0,
    left: 0,
    right: 0
});

// This would be a menu item. 
var button = Ti.UI.createView({
    top: 10,
    left: 10,
    width: 100,
    height: 100,
    backgroundColor: "#0000FF"
});

// Here is where the click is registered
button.addEventListener("click",function(eventObject){
    // Do stuff here
});

holder.add(button);

    You would then need to add the holder to your main window. Hope this helps!

    • Ah, I see what you mean. You're talking about a "close" or "delete" button.

      That's not a standard interface item. You'll need to use either a button or standard view with the cross (or X) graphic as the background. Then you'll need to attach an event listener to the click event on that view and close your message view in the callback function. Does that make sense? So, assuming closeGraphic.png is your "cross" graphic, window is your active window and messageView the view with your message in it:

      var closeButton = Ti.UI.createImageView({
    image: "closeGraphic.png"
    width: 30,
    height: 30,
    top: -15,
    left: -15
});

closeButton.addEventListener("click",function(eventObject){
    window.remove(messageView);
});

messageView.add(closeButton);

      Untested code, but that should at least get you started.
