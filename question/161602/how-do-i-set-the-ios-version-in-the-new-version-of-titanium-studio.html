How do I set the ios version in the new version of Titanium Studio?

I recently upgraded to build: 3.2.0.201312191547, and now I cant seem to find a way to change my run configuration, where I would previously would have been able to change the ios version to run the app with.

Is this gone, is has it just been moved?

  • Accepted Answer

    In 3.2.0 you should have two buttons just above your project list panel. These buttons also have dropdowns, one for choosing run type and a second one for choosing the device. It is the second dropdown you want, here you will find run configurations

    There is also the run dropdown at the top of your titanium window, you can find run configurations here too.

    Hope this helps!

    • Ok, that's a weird issue. Another way to get to the run configurations is to go to the properties of a project, either by right-clicking or going to file at the top of the window.

      Once in properties select the Run/Debug Settings and you should be able to edit the launch configurations for the project.

      I would check that everything is up to date just in case.

      — commented January 17th 2014 by Luke Taylor
