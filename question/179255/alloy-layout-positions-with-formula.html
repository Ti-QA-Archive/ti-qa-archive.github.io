Alloy Layout positions with formula

Hello,
is it possible to do my layout positioning in Alloy using a calculation?

I know Ti.UI.SIZE and Ti.UI.FILL but can I can not say "Ti.UI.SIZE / 2".

I have a vertical layout and want a GAP between two labels being a fraction of screen height. If I do

"#Label2" : {
top : "Alloy.Globals.Style.RoundButtonHeight"
}

… it works…but if I use: "Alloy.Globals.Style.RoundButtonHeight*4" it does not work.

Of course I could precaclucate the value and use a new global variable but it would be much nicer to do such calcuations in the stylesheet.

    Hi

    The TSS isn't like a LESS/SASS compiler so you can't do calculations in the stylesheet. Any calculations will have to be done beforehand in the controller/JS.

    You should use Ti.Platform.DisplayCaps to get the width and height of the screen and then you can do your calculations from those values. If you are using Android you will have to take the scale factor into consideration using the logical density factory.

    — answered November 25th 2014 by Luke Taylor

    • In addition you can calculate values in your JS and assign them to Alloy.Globals.someObject which can then be referenced inside your TSS.

      example:

      var width = parseInt(Ti.Platform.displayCaps.platformWidth / (Ti.Platform.displayCaps.logicalDensityFactor || 1), 10);

var height = parseInt(Ti.Platform.displayCaps.platformHeight / (Ti.Platform.displayCaps.logicalDensityFactor || 1), 10);

Alloy.Globals.deviceWidth = width;
Alloy.Globals.deviceHeight = height;
