Custom fonts from the same family

Hi, I'm using the Helvetica Neue LT Pro family of fonts in my app (wiki Custom Fonts). However, I can't set the weight of the font by using the normal fontWeight property because each weight and style of the font is held in a separate file. For example, I have 'HelveticaNeueLTPro-Cn.otf' [condensed], and 'HelveticaNeueLTPro-BdCn.otf' [bold condensed] and so on. Therefore they all have the same 'friendly iOS name' which is just 'Helvetica Neue LT Pro' which means I have no way of differentiating between them in my code. I have tried to change the fontWeight attribute but this has no effect.

Does anyone have any idea how to use multiple custom fonts from the same family?

SDK 1.8.0.1, iOS 5.0

  • If they are separate fonts for each fontWeight then you need to specify the whole font name and cant use the fontWeight property. Eg IOS has HelveticaNeue and HelveticaNeue-Italic but you cant add the fontWeight:&#39;Italic&#39; to the HelveticaNeue font and expect it to work, you must use the fontFamily:&#39;HelveticaNeue-Italic&#39; with no fontWeight
