  • Hi there,

    If you want to change the title's font, size and color, you can use the titleControl or titleImage proprieties see more info in the UI.window docs.

    titleImage, as the name suggests, is just an image in the title bar. IMHO, I think it's great if you want to use an logo or a custom title and dont want to import the font to the iOS.

    A little more flexible is titleControl. With this, you can actually change the font's values.
    An excellent example on how to use titleControl, from the blog of Aaron Saunders.

    Note: These 2 features are only available for iOS.

    Note2: From time to time, check Aaron Saunders's blog. He has snippets and other awesome stuff about Titanium, so follow his blog :)

    Hope it helps!
