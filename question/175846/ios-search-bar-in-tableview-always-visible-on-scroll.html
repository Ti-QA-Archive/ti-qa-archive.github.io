iOS Search Bar in Tableview always visible on scroll

Hi, I'm using search bars in tableview to filter data but when I scroll to bottom the search bar disappear it's only visible when I'm on the top of the tableview, there is some attribute or anything that I can do to make always visible the search bar on scroll, I have even tried to add the search bar directly to the view but the first result it's covered by the search bar

    Hi Gabriel,

    Thanks for your concern.

    Please try my test code and see if it helps you.

    var win = Ti.UI.createWindow({
    title:"Search Test"
});

var table = Ti.UI.createTableView({
    top: 45
});

var search = Ti.UI.createSearchBar({});

table.setData([{title:"A"},{title:"B"},{title:"C"},{title:"D"},{title:"E"},{title:"F"},{title:"A"},{title:"B"},{title:"C"},{title:"D"},{title:"E"},{title:"F"},{title:"A"},{title:"B"},{title:"C"},{title:"D"},{title:"E"},{title:"F"}]);

table.searchHidden = true;
table.search = search;
search.top = 0;
win.add(search);

win.add(table);
win.open();

    In this code i've added a searchBar to tableView to search the table data. To prevent first result/data covered by the search bar, use tableview top to 45.

    Thanks
