ButtonBar for Android?

Hi, I'm curious if anyone has implemented something similar to a button bar for Android?

I find myself in need of the ability to horizontally place 1-N buttons on the screen inside of a view. It appears (from my testing) that I need to take care of the calculations of where to place the buttons to space them properly (left to right).

A few questions
1) Am I missing something where TI can help with placing them?

2) Has anyone built any reusable code for this?

3) Do others think it would be useful for me to try to build this as reusable class?

Thanks.

  • This is far from an elegant solution but given that better support is coming from native TI in 1.6 I think this will work. It's basically a function that takes some info about the layout and generates the "left" values for your buttons.

        /*
     * This method will return an array of "left" values
           to equally space buttons horizontally.
     */
    styles.spaceButtons = function(containerWidth, numButtons,
                                       buttonWidth, buttonSpacer)
    {

        var padding = containerWidth - 
        ((numButtons*buttonWidth)+(buttonSpacer*(numButtons-1)));
        var lefts = [];
        for (var i = 0; i < numButtons; i++)
        {
            var thisLeft = padding/2 + 
                          (buttonWidth*(i))+(buttonSpacer*(i));    
            lefts[i] = thisLeft;
        }
        return lefts;
    }
