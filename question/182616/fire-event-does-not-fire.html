    Hi Nilmar Castro,

    Please find below the solution,

    
var myArrayDate = [];

Ti.App.addEventListener('app:updateDate', function(e)
{
    label.text = e.myArrayDate[0] + '/' + e.myArrayDate[1] + '/' + e.myArrayDate[2];
});
and child.js
btn1.addEventListener('click', function() 
{
    Ti.App.fireEvent('app:updateDate',
    {
        myArrayDate: pickerData         
    });
    win.close();  
});

    I have changed the myArrayDate to e.myArrayDate since you are passing an array inside object to app:updateDate event. Hope this is clear to you.

    Thanks,

    Narendra Raorane
