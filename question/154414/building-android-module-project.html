  • Build Problem Solved

    Download and install gperf as given in the documents of Installing gperf.
    Then configure the environment variables,
    In sysytem variables select "Path" and to it add "C:\Program Files (x86)\GnuWin32\bin;"…
    Yes the path should be given till bin folder of gperf.
