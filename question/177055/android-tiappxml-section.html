Android TiApp.xml section

Below is the code I have for android in the TiApp.xml file

    <android xmlns:android="http://schemas.android.com/apk/res/android">
        <tool-api-level>14</tool-api-level>
        <manifest> 
                 <supports-screens
                android:anyDensity="false" android:largeScreens="false"
                android:normalScreens="true" android:smallScreens="false"/>
            <application android:theme="@style/Theme.Action"/>
            <uses-sdk android:minSdkVersion="14" android:targetSdkVersion="16"/>
        </manifest>
    </android>

How do I had 

  <android:versionCode="2" android:versionName="1.0.1"/>

to it?

Titanium 3.2.3

    <android xmlns:android="http://schemas.android.com/apk/res/android">
        <tool-api-level>14</tool-api-level>
        <manifest android:versionCode="2" android:versionName="1.0.1"> 
                <supports-screens
                android:anyDensity="false" android:largeScreens="false"
                android:normalScreens="true" android:smallScreens="false"/>
            <application android:theme="@style/Theme.Action"/>
            <uses-sdk android:minSdkVersion="14" android:targetSdkVersion="16"/>
        </manifest>
    </android>
