  • Hi

    The alert method is the short form (convenience) version of the AlertDialog.

    If you want to change change the caption for the 'ok' button or provide your own 'title' - this is the place to be.

    This is a very simple example;

    Ti.UI.createAlertDialog({
  message: 'The file has been deleted',
  ok: 'Okay',
  title: 'File Deleted'
}).show();

    You can however create and 'alert' with more than one button and then respond the the choice the user makes;

    var dialog = Ti.UI.createAlertDialog({
    cancel: 1,
    buttonNames: ['Confirm', 'Cancel', 'Help'],
    message: 'Would you like to delete the file?',
    title: 'Delete'
});
dialog.addEventListener('click', function(e){
    if (e.index === e.source.cancel){
        Ti.API.info('The cancel button was clicked');
    }
    Ti.API.info('e.cancel: ' + e.cancel);
    Ti.API.info('e.source.cancel: ' + e.source.cancel);
    Ti.API.info('e.index: ' + e.index);
});
dialog.show();

    This creates a new alert dialog with three buttons that are shown in the order you enter them. You then add an event listener to the dialog to respond to the choice the user makes. So you could have a Yes/No alert and do one thing for yes and another for no.

    If you need many choices then the AlertDialog may be insufficent - if so then the; OptionDialog takes provides a UI more suited to more buttons.
