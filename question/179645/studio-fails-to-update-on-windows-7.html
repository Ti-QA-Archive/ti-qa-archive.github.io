  • Hello Andrey,

    Thanks for your inquiry. It´s recommended that you update npm.

    So, in command line do:

    $ npm config set strict-ssl false
$ npm install npm -g
$ npm config set strict-ssl true

    Now, update Titanium. If it doesn't work, same problem occurred during update, then,

    Only run

    $ npm config set strict-ssl false
$ npm install npm -g

    First, and then try updating. After finished update run,

    $ npm config set strict-ssl true

    Hope this helps, also for more info, go through this similar community link. Thanks.
