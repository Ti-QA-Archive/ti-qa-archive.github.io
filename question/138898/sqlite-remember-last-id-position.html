SQLITE remember last ID position

I have a a database with scrollview. How can I get it to remember the last position it was on after restarting the app? It saves the day after closing the app, but if you fully exit out of it, it loses its day.

  • Mike, you can use this to get the value using Ti.App.Properties.

    Save the value

    Ti.App.Properties.setString("fiIdDistribuidor",somevalue);

    Get the value

    var distribuidor = Ti.App.Properties.getString("fiIdDistribuidor");
if (distribuidor == null ) {
    Ti.App.Properties.setString("fiIdDistribuidor","0");
    distribuidor = 0;
}
