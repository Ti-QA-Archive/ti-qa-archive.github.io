  • <h3>After several days of searching, I finally found the solution !</h3>

    Here, all the steps to see the application in a Android system :

    1/ Download the latest version of Titanium (for me it was 3.3.0).

    2/ Launch Titanium, update it.

    3/ Create a project (ctrl + n): "Mobile App project", follow the steps and install the Android SDK. You can download the Android NDK and add in Titanium too.

    4/ Download Java 1.6: Java SE Development Kit 6u10 (on http://www.oracle.com/technetwork/java/javasebusiness/downloads/java-archive-downloads-javase6-419409.html) and install it.

    5/ Right-click on "Computer" > "Properties" > "Advanced system Settings > "Environment Variables" :
    Add Variable: 

    Name : JAVA_HOME
Value = your path to the directory java 1.6

    Edit the PATH variable : change the path to the directory of your current version of java, or add it without erasing the other variables (separated by " ; ").

    6/ Download and install Genymotion. Create a device. Run it.

    7/ On the device, click the Genymotion application to have the IP of the device.

    8/ On Titanium
    Add Genymotion (Window > Preferences > Studio > Platforms > Android)
    Install the SDK 3.2, start the console and type:

    ti sdk install -b 3_2_X --default

    9/ Restart Titanium

    10 / In the console, select Genymotion :

    ti config genymotion.enabled true

    11/ Add the device:

    p ti build android --device-id "Xperia Z - 4.3 - API 18 - 1080x1920"

    Xperia Z - 4.3 - API 18-1080 × 1920 is the name of my Genymotion Emulator

    12/ In the menu bar, at the right of the run button, the IP of the device appears.

    13/ Click on run (or in the menu : Run). If necessary, choose the device genymotion.

    14/ The launch may be blocked at 57%, in this case, change the window.js file:

    C:\ProgramData\Titanium\mobilesdk\win32\3.3.0.GA\node_modules\titanium-sdk\lib\

    Go to last function "exports.getSelectedVisualStudio" and comment:

    if (env.visualstudio) {
    return env.visualstudio [Object.keys (env.visualstudio) .Sort () filter (function (v) {
        return env.visualstudio [v] .selected;
    }). pop ().]
 }

    15/ Launch Titanium and click on Run.

    16/ The console turn on a few seconds later … my app is display on my genymotion device !
