  • Accepted Answer

    Hello Chung ,

    In above code you add ImageView in ImageView. This is not proper. Android ONLY allows child views to be place inside container views. See This. Link.

    Try To add ImageView in View like :

    // i think as you declare book in above code is a view like :
var book = Ti.UI.createView({
    width : 250,   // set as you need
    height : 340   // set as you need
});
var thumbnail = Ti.UI.createImageView({
        left: padding,
        top: '5%',
        width: bookWidth,//250
        height:bookHeight,//300
        image: params.thumbnail,
        touchEnabled: false
    });
    book.add(thumbnail);

    var newLogo = Ti.UI.createImageView({
        image: '../images/New.png',
        top: '5%',
        left: '25%',
        width: 100,
        height: 100,
        touchEnabled: false,
        zIndex:10
    });
book.add(newLogo);

    i hope this is useful to get solution and solved your problem. thanks.
