Read content of webview on Android

Hi all,

I need to get a string from a webview to my app, and I have tried a few ways to do it. On iOS everything works fine, but on Android I've got problems. Basically the idea is:

webview.addEventListener("load", function() {
   var html = ....;
   var data = html.split("stringtolook");

   if(data.length == 2) {
      /* We are on the correct page, split the data to get string we want */
   }
}

So, I try to search string "stringtolook" from the webview to know that now users is on the correct page, and then I would cut the string I want to my app.

I have tried to do this in these two ways:

1) 

var html = webview.html;

This would be an ideal way of course, but it is undefined on Android, so no go.

2) 

var html = webview.evalJS("document.documentElement.innerHTML");

This would also be OK for me, but at some point it throws an exception: "Only original thread that created a view hierarchy can touch its view".

How should I implement this? I'm using Titanium 1.8.2 and exception occurs on both emulator and real device.
