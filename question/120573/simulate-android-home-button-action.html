Simulate android home button action.

Hi,

Is it possible to trigger the "home" button event to return the user to their desktop (in effect, "minimizing" the app).

I basically want to catch the android:back event and convert it into a question of:

-Do you want to leave the app running and return to the home screen {yes: Fire "home" button event}

-Do you want to quit entirely {yes: close the app}

The reason for this is less savvy users need to be given the question, a lot don't realize that "back" often quits apps and "home" often leaves them running.

Hope someone can help.

  • Awesome.. After some googling on how to do it on the Android native language I managed to then use the Ti documentation to convert it to Ti code:

    backButtonConfirm.addEventListener('click',function(e) {
    if (e.index == 0) {
        var intent = Ti.Android.createIntent({
        action: Ti.Android.ACTION_MAIN
        });
        intent.addCategory(Ti.Android.CATEGORY_HOME);
        Ti.Android.currentActivity.startActivity(intent);
    } else if (e.index == 1) {
        win.close();
    }
});

    Hope this helps someone else. (I am now on to solve other problems! :D)
