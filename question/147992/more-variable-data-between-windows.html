  • When you are creating the new window, just add it as another variable, like

    var self = Ti.UI.createWindow({
        title:title,
    newV=variable_to_pass,
        //backgroundColor:'white'
        backgroundImage: 'defensive-zone-of-the-court-tennis.jpg'
    });

    and in your new window, access them by

    var win = Titanium.UI.currentWindow;
var passedV = win.newV;

    If you have a lot, just put them in an array first, and pass and manage the same way.
