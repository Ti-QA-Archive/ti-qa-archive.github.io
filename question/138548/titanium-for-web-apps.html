Titanium for web apps

Hello, I'm pretty new to Titanium and I have to develop a web app and also a mobile app both remaining almost the same with a few functionalities more on the mobile app.

Please, could anybody tell me if I can make the web app also with Titanium and get a native look and feel or I should think in something like Sencha?

Thanks a lot in advance.

    You're in luck! Titanium Mobile supports not only native Android and iOS, but Mobile Web. Using the same Titanium code that is used for native apps, you can build your app for the web and it will generate HTML, CSS, and JavaScript. Titanium Mobile Web supports all modern browsers on both handheld devices and desktop browsers.

    You need to be using the latest 2.0.2 release to get started, however v2.1 is going to be out in a couple short weeks and has many bug fixes and performance enhancements. I'm the lead engineer for the Mobile Web platform and I monitor this Q&A daily, so give it a try and if you have any questions, just ask away! :)

    — answered June 19th 2012 by Chris Barber
