    • That should be easy, "setTimeout" as first argument takes function reference that is going to be executed after number of milliseconds (second argument):

      setTimeout(function() { tf1.editable = true; }, 10000);

    • I don't have iPhone, so I can't test on it. I've tried that code on Android and it works fine, it should also work on iPhone (as property name says, setting it to true should disable text field editing).

      If that doesn't work on iPhone, you should report it as a bug.

      As a workaround (if you target iPhone only) you can add event listener to the focus event and then call "blur" method of the text field:

      var tf = Titanium.UI.createTextField(...);

var blurHandlerId = tf.addEventListener('focus', function() { this.blur(); });

setTimeout(function() { tf.removeEventListener('focus', blurHandlerId) }, 10000);
