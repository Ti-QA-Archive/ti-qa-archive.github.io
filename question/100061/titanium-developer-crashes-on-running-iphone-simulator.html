    Where did you get the "sample" project? Try creating a new, unmodified project, and see if that will launch.

    There may be a file called build.log in the $PROJECT&#x2F;build&#x2F;iphone directory or one of its subdirectories. If there is, would you pastebin/gist that, please?

    I am not sure if the trace log is different to the build.log. Anyway, would you pastebin/gist that also?

    Use the tools available to you to provide information about where things are failing. Read Troubleshooting Your Installation for more info. Although it is very Android-centric, it should still give you a few pointers.
