Titanium Desktop for Linux - exe compiling

Hey guys,

Im on Ubuntu 11.10 trying to compile my application in to a .exe file. I have downloaded both the windows and linux sdk, but when i compile it only creates a linux application folder in the "dist" folder of my app.

How can i also compile a .exe version of my application when on linux?

    You cannot do it.
    The Titanium Desktop was discontinued.
    The compiler for any platforms was build in each system by a cloud servers, but Appcelerator shutdown this.
    In this moment the only way to do is build on each system locally.
