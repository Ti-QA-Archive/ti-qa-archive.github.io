Can't find Android SDK

When I try to set path to Android SDK Home in T Studio it complains with "No platform SDK is found". I have installed Android SDK Tools and also SDK Manager. What path should I set, to the directory Android SDK or an executable file?

  • Simply set it to the root of the Android SDK installation. (it requires a directory, not an executable)

    Also, make sure you have the 2.3.3 package installed as well as one of the latest ones 4.2.2 or 4.3. The old one is required for compatibility reasons. The latest one is to have access to the latest features.

    Hope this helps
