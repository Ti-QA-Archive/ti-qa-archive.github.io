Hashcode

In Titanium how to generate hashcode?

— asked October 5th 2011 by Lala Waghmode

  • var uname='Lala'; var password='waghmode'; var str=password+uname;
    var hashcode=Titanium.Utils.md5HexDigest=function(str){return str;} alert(hashcode);

    when I run this I am getting different hashcode return value.Instead of that it should be same for each execution.

    can you please correct this code so I will get proper hashcode.

  • Accepted Answer

    That's not the way to do it, you redefined hash function!

    Should be:

    var uname='Lala';
var password='waghmode';
var str=password+uname;
var hashcode=Titanium.Utils.md5HexDigest(str); 
alert(hashcode);
