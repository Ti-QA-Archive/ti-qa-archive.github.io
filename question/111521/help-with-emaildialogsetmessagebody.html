Help with email.Dialog.setMessageBody

Platform: iOS, Latest Titanium Version <br><br>

Im trying to paste together a formatted message body from several App.properties strings. My problem is adding several strings together.

The goal that im trying to achieve is:<br>
Driver: Your Name<br>
Customer: Customer Name<br>
Address: 123 Any Street<br>

etc…

The code below, obviously, does not work. SO how do I use the &lt;br&gt; and &lt;p&gt; tags without it breaking the app?

Thanks to all in advance!

if (Ti.Platform.name == 'iPhone OS') {
            emailDialog.setMessageBody('<b>Driver</b>' + 
Titanium.App.Properties.getString('drivers_name') + <BR> +
Titanium.App.Properties.getString('customer_name')
);
            emailDialog.setHtml(true);
            emailDialog.setBarColor('#336699');
        } else {
            emailDialog.setMessageBody('NorEaster Lite Rocks!');
        }

    Not sure about you, but I don't like the "set" functions cause they make things pretty messy. Here's the code as I would write it:

    if (Ti.Platform.name == 'iPhone OS') {
    emailDialog.messageBody = '<b>Driver</b> '+Ti.App.Properties.getString("drivers_name")+'<br />'+Ti.App.Properties.getString("customer_name");
    emailDialog.html = true;
    emailDialog.barColor = '#336699';
} else {
    emailDialog.messageBody = 'NorEaster Lite Rocks!';
}

    I noticed there was no "s around the second <br /> so I added them in. That should work :)
