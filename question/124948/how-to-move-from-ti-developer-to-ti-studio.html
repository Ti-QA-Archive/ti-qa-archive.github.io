How to move from TI Developer to TI Studio

I've decided to switch to the new Titanium Studio as it looks like Developer is not able to build 1.7.2 projects.

At any rate, I can't figure out how to build in studio. When I click the Run button I get Selection cannot be launched.

Can somebody kindly point me to documentation that explains how to use Studio? Unless I missed it, I didn't see anything on the Docs page.

Even better would be a transition how-to for for former Developer users.

Thanks!

    Try creating a new project in Studio. That way you can specify what platforms you'd like to be able to deploy on. After it creates it, just move your old resources file over to the new project. I don't think it should be anything more than that, really. That's what we had to do.
