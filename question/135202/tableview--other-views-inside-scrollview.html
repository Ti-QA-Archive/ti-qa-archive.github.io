TableView + other views inside ScrollView

I have a need to place TableView and View inside ScrollView. I found a similar question on how to get it working here, but the solution there was to define static contentHeight to a ScrollView. Is there any way to get contentHeight autocalculated depends on TableView.height+View.height? I tried

ScrollView.contentHeight = TableView.size.height + View.size.height;

after seting the rows to a table (so the TableView should have correct size.height value), but with no result. If there any best practice on how to achieve that?

Is there any other way to get a view (with some controls inside) under a scrollable TableView?

  • this code work

    var scrollView = Titanium.UI.createScrollView({
    contentWidth:'auto',
    contentHeight:'auto',
    top:0,
    showVerticalScrollIndicator:true,
    showHorizontalScrollIndicator:true,

    layout: 'vertical'
});

var data = [{title:"Row 1"},{title:"Row 2"}];
var table = Titanium.UI.createTableView({data:data, rowHeight:40});
table.height = 2 * 40; ////set height for table with 2 rows
scrollView.add(table);

var view = Ti.UI.createView({
    height:100,//set height for view
    top:10
});
scrollView.add(view);
