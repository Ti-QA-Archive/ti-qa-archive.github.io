Conditional Alloy Styles for tablet formFactor Not Being Applied

Hi there-

I am attempting to apply a conditional style, as indicated below. When attempting to apply a conditional style for a handheld device, all is fine. For some reason, the tablet formFactor styles are not being applied. Just curious if there is something I am doing in the code that needs to be tweaked to actually apply styles to tablet devices, and why these styles are not being applied appropriately to tablets. By default, the sample code always takes the non conditioned style.

"Window": {
    backgroundColor: '#000000'
},
"Window [platform=ios formFactor=tablet]": {
    backgroundColor: '#ff0000'
},
"Window [platform=ios formFactor=handheld]": {
    backgroundColor: '#00ff00'
}
— asked April 1st 2013 by Christopher McGuire
