  • I use the following in order to resolve different path management between Android and iOS:

    if (Ti.Platform.osname == 'android')
    Ti.include('../suds.js');
else
    Ti.include('suds.js');

    My files:

    Resources\suds.js

    Resources\ui\WebserviceWindow.js (where I use the above code)
