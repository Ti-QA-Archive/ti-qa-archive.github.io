  • Had a simular issue with the iOS simulator not launching or closing directly after launching in Titanium Studio. In my case the issue was caused by a recent upgrade to XCode 4.3. Upgraded Titanium Studio to the new 2.x version with the 2.x mobile SDK.

    • Please visit (this page)[http://ec2-50-16-19-245.compute-1.amazonaws.com/studio/] for Titanium Studio upgrades.

    • Please visit (this page)[http://builds.appcelerator.com.s3.amazonaws.com/index.html] to obtain a master 2.1.x build link.

    • Copy link into Titanium Studio > Help > Install Update from URL

    • Open tiapp.xml in Titanium Studio > Select Titanium SDK 2.1.x > Save

    • Clean (delete files in) build folder for application and build again.
