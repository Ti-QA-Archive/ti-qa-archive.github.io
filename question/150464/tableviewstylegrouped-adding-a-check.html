    • when you create tableViewRow at that time set this property and on tableView's click event you can check it 

      //in .xml file
<Alloy>
        <TableView id = "tblView" onClick="tableClick">
            <TableViewRow hasCheck = "true" ></TableViewRow>
            <TableViewRow hasCheck = "true" ></TableViewRow>
        </TableView>
</Alloy>

      and in .js file

      
function tableClick(e){
    alert(e.row.hasCheck);
}

      //in .xml file
<Alloy>
        <TableView id = "tblView" onClick="tableClick">
            <TableViewRow></TableViewRow>
            <TableViewRow></TableViewRow>
        </TableView>

function tableClick(e){
    e.row.hasCheck = true;
}

    • One last question. How do i make it that its just one row clicked at a time

      — commented April 4th 2013 by Conor Byrne

    • in .js file

      var preRow = null;

function tableClick(e){
    if(preRow){
        preRow.hasCheck = false;
    }

    e.row.hasCheck = true;
    preRow = e.row;
}
