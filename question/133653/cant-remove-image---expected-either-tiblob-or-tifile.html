Can't remove image - expected either TiBlob or TiFile

I have a table containing rows in which I add a checkmark image when that row's task is complete. Adding the checkmark works great, but whenever I try to remove the check, I get this error: "[ERROR] invalid image type. expected either TiBlob or TiFile, was: NSNull in -[TiUIImageView setImage_:] (TiUIImageView.m:693)". I have tried setting the image to '' (empty string) and to null. I don't understand why this crashes. It actually does remove the 1st row's checkmark, then crashes, then doesn't remove any more checkmarks.

Here's my code:

for (var i = 0; i < tableMain.data[0].rows.length; i++){
        if(tableMain.data[0].rows[i].children){
                if(tableMain.data[0].rows[i].children[0]){
                    if(tableMain.data[0].rows[i].children[0].image){
                        tableMain.data[0].rows[i].children[0].image = null;
                    }
                }
        }
    }

    Setting image property of ImageView to null would cause above error. To remove the check image, you can either hide() or remove() it:

    
if(tableMain.data[0].rows[i].children[0]){
    tableMain.data[0].rows[i].children[0].hide();             
}
