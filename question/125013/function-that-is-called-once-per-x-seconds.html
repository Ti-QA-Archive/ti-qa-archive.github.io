    • If you also need to remove the timer then you can do it this way:

      //activate the timer
var timer = setInterval(do_something, X * 1000);

//...

//win is a reference to your window  object
win.addEventListener('close', function() {
    //remove the timer
    clearInterval(timer);
});

  • If you want to stop the interval when the window closes, then set a variable to the interval instance, and clear it in the window's close event:

    var myInterval = setInterval({ function() {
    // do something here
}, 1000 );

Ti.UI.currentWindow.addEventListener('click',function(e) {
    clearInterval(myInterval);
});
