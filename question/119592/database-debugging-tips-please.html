Database debugging tips please

I am having few problems debugging my sqlite database storage and retrieval. So I am looking for some general tips and advice on ways to make life easier. All suggestions welcome.

A have a few specific questions too.

  1. Is there a sqlite browser app that I can install on Android? Would the android security model even let me browse my app's database?

  2. Where on my filesystem is android emulator data stored? Presumably in one of the android SDK folders, but which one?

For those with similar queries about Iphone, this answer tells you how to find the sqlite file on Iphone emulator.
http://developer.appcelerator.com/question/8261/sqlite-database-storage

    Caspar

    The Android SDK and AVD Manager states where the virtual device images are located on your filesystem; see the path at the top of this screen. The data is stored within them.

    The easiest way to browse the Android filesystem is using Android Development Tools for Titanium Studio. See Installing the ADT Plugin.

    To access the sqlite database directly, I typically use adb from the command line.

    Hope this helps
