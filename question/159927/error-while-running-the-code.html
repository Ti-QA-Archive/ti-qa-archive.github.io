  • First, ensure that your android environment is configured properly. Open the Studio dashboard (red house w/ letter A icon on toolbar), scroll down to Configure Native SDKs. Is there a red X or green checkmark next to Android SDK option? A red X means the sdk is not properly configured. Click that link for instructions on what to do to properly install the sdk.
