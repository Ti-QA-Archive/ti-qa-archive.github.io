Individual appicon and splash screen

Hi,
I try to build a white label app for iPad. Inside the app I can modify the assets depending on customer number.
But is it possible to modify icon and start screen in same app (without recompile with other resources)?

  • I don't think you can change the icon, but you can have a custom splash. Make the default splash white (or any other color) and at startup execute your custom splash that is a fullscreen window, close it after some time and finally run the main window. I use this method to display my animated spalsh screen.
