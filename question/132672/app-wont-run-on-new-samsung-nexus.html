App won't run on new Samsung Nexus

Have an app that was running fine on various android devices. Got a new phone, Samsung Galaxy Nexus, which has Android 4.0.2 and my app fails to even get out of app.js

I am using the Cross-Platform Navigation Controller as described here:
http://developer.appcelerator.com/blog/2011/08/forging-titanium-episode-2-a-cross-platform-navigation-controller.html

and it is failing on the launch of my controller.

Here is the line of code it fails on:

controller.open(new assessmentsWindow(controller));

With the error message:
"undefined is not a function"

here is the rest of the code leading up to where if fails in app.js:

var assessmentsWindow; //first window of app
var NavigationController = require('ui/common/NavigationController').NavigationController;
var controller = new NavigationController();

assessmentsWindow = require('ui/common/Assessments').Assessments;        
controller.open(new assessmentsWindow(controller));

Is there something I am missing for Android 4.0? This one has me stumped. No changes to any of the code, it just won't run on the new phone.

Please help, thanks!
