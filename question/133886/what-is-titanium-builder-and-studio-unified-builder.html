What is Titanium builder and studio Unified Builder

I was just exploring Titanium Studio and under project properties-> builders I found two builders present there i.e Titanium Builder and Studio Unified builder.
I tried searching about them but didnt got any document explaining there role.

I then unchecked both of them and then tried to build my project and it got build successfully.

can anyone explain me the role of those builder and can we create our own custom buider

    The builders are responsible for indexing, validating, and cleaning projects. While you will still be able to build without them, it's more likely that your project will be out of sync or have errors you won't catch until deploy time.

    The purpose of a builder is to run any task in one of two circumstances:

    • On delay after typing (i.e. indexing or validating the current file)
    • While building the project (i.e. after performing a clean). This commonly does the same things you do on reconcile, with perhaps some additional tasks.

    You can create your own builder. If you'd like to do so, it does require some Eclipse knowledge. See https://wiki.appcelerator.org/display/tis/Participating+in+the+build+and+reconcile+process for more info.
