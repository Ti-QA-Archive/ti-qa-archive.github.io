    Hi

    As you are using a Mac you should be able to simply download the file, open up the downloads folder and double-click the downloaded file.

    Tar is compressed file format (think zip), but you need special software on a Windows machine to open it (WinRar is a good choice), but the Mac supports it natively.

    If you still cannot do this, then simply press the <> symbol in the top right of each of the two code snippets files, copy the code and then save use the name at the top of each of those two.

    The compressed file simply contains those two files.

    • Thank you for your update, I may have taken your words as tone more than I should.

      CommonJS = a way to modularise code into reusable blocks (files in this case)

      • Guide for CommonJS Modules in Titanium

      The link you provided…

      • Using a module

      … was for native compiled modules, this is why I thought you knew more than you did.

      There are three main sorts of modules;

      • CommonJS - JavaScript
      • Compiled CommonJS - Compiled CommonJS
      • Native Module - Java or ObjectC code compiled into a format that can then be used in Titanium

      All three of these are accessed by the require keyword.

      • The first one is the same type as used with Codebird, you will be able to tell these in the future as you can read and edit the code, they are placed inside your app code folder(s) and contain the phrase exports (usually at the bottom of the file
      • The second one you can create yourself, you take a straight CommonJS file and create a module project - this requires a little more experience - not because it is hard (tiny learning curve) but there needs to be a business rule reason to compile your CommonJS files.
      • The third version means you have experience in either native Java code or Native ObjectiveC code and you wish to extend the features of Titanium by exposing features from the Android or iOS SDKs - this requires much more experience and not for the faint hearted

      If you use the second or third versions then the module is installed in either your Titanium installation or your apps modules folder.

      Many apps take advantage of these last two types as they provide things like iOS printing, barcode scanning, social networking and much more.
