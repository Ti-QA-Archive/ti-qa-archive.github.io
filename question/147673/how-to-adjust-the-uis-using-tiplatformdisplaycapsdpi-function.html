  • Hi, Ti.Platform.displayCaps.dpi function returns you the dpi of the device.

    to use it practically, you have to set height and width in dp

    example:

    var view = Titanium.UI.createView({
   borderRadius:10,
   backgroundColor:'red',
   width:"50dp",
   height:"50dp"
});
window.add(view);

    this way height and width are automatically adjusted as per device DPI

  • 
var screenWidth = Titanium.Platform.displayCaps.platformWidth;
var screenHeight = Titanium.Platform.displayCaps.platformHeight;

    Then 

    desiredUnit=  (screenWidth OR  screenHeight )*MultiplyingFactor  
// for example   
 50= Math.floor(screenHeight*0.1042)

    left and right are handled by width calculations while top or botton and handled with height values
