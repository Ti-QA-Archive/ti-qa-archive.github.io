Installing a 3rd Party Module

Hi,
I found this third party module for Android push notifications: https://github.com/IGx89/titanium-c2dm

I want to install the module so I can use it in my app.
I tried following these guides:
http://titaniumtips.com/files/modules_installation.php
http://titaniumtips.com/files/modules_testing.php

and other guides I found on the web. I changed my tiapp.xml file and added:
<modules>
<module platform="android" version="0.1">com.my_company_name.titanium.c2dm</module>
</modules>

and I copied the module to the directory in the article and also to the project directory under modules (like I read in a different post).
It doesn't work.
The file I copied is in the dist directory - is that what I really should copy?

The error I'm getting is: "Wrapped java.lang IllegalArgumentException: size must be >= 0 (file:///android_asset/Resources/app.js#302)

Can anyone explain how to install a compiled module?

Thank you

  • I don't think you need the 'com.my_company_name' for a module name (unless a particular module for some reason requires it). If the instructions say to copy to the Library folder (e.g., /Library/Application Support/Titanium/modules/android) make sure it's the root Library folder, not that of the user. I don't believe you need to copy to somewhere within the project AND within Library - usually it's one or the other, depending on the type of module.

    Also, see this Blog Entry on Modules
