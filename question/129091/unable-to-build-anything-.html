    • I'm sorry - this is so long ago I don't know how I managed to get it working again.

      One thing to check is that all of your text-files is encoded with UTF-8, if they're not, you'll get errors like this. However the Titanium build system has a habit of barfing out errors with little clues to what the problem really is.

      If you're using Mac - check for copies of your files that are called ._FILENAME - ie if you have a thing.js - it's possible you have a ._thing.js file somewhere. They can cause problems even though they're not supposed to any longer.
