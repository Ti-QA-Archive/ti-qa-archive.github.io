How to scroll a single image to the right?

I have a single image…. I want to scroll the image to the right on Android.. When the app opens, user should see only left part of the image, and he should be able to scroll the image to the right to see the complete image..
Please some one help me with code on this…

  • The magic world is 'animation'.

    var i = Ti.UI.createImageView({
        image: image,
        left:-100
})
i.animate{{left:0,duration:1000}}
