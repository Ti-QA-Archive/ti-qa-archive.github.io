enable cloud services via CLI

Is there a way to create a new titanium app via the CLI that has cloud services enabled by default?

I found an old spec for the CLI going back to may 2012 that has a –cloud parameter in the ti create command, but it looks like this did not make it into the released application

Would be interested in any success in doing this

    At this time, cloud-enabling an app via the CLI is not supported. This process is handled internally by Studio rather than passing parameters or commands to the CLI as is done with some other operations (e.g. compiling an app).
