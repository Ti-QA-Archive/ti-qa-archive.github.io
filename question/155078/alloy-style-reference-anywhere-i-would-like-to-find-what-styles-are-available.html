Alloy style reference anywhere? I would like to find what styles are available.

I'm new to Alloy and need a TSS reference guide to know what style declarations I can use in my .tss files.

How would I go about doing this and is there a reference anywhere exposing what elements we can style for each UI object? Would be a HUGE help.

    You can apply styles across your entire app in the app.tss file.

    "Label":{
 top:10,
 color:"#666"
}
"ScrollView":{
 layout:"vertical"
}
