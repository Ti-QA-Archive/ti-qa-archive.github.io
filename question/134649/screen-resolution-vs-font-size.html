Screen Resolution vs Font Size

Morning all,
I have a question regarding Screen Resolution and Font Sizes.
How can I tell what the users Screen resolution is and then serve them the correct font size?

In my latest project, I have hard-coded all my font sizes e.g. 12px,
but when viewed on a higher resolution screen, the text is obviously tiny and impossible to read.

What is the best way to serve different font sizes based on screen resolution?
Any help us much appreciated.

Thanks,
Lee

    L F, the best way is to set your font size to '12dp' instead of '12px'. If the 12 looks good on a medium-density device, it will look the same relative size on devices of all densities.

  • Hi L F,

    Try to use this code that may solve your problem.

    var screenWidth =  Ti.Platform.displayCaps.platformWidth;
var screenHeight = Ti.Platform.displayCaps.platformHeight;

function GetHeight(value) {
    var temp = (value * 100) / 480;
    return parseInt((screenHeight * temp) / 100);
}

function GetWidth(value) {
    var temp = (value * 100) / 320;
    return parseInt((screenWidth * temp) / 100);
}

var label = Ti.UI.createLabel({
    text : 'label text',
    height : GetHeight(30),
    width : GetWidth(50),
    fontSize : GetHeight(12),
});
