    Hi guys,

    Simply clean your kitchen Sink project, then try to run it. If that doesn't work you can try various ways.

    1. Check you tiapp.xml file and make sure error free and project deployment target exists or not.
    2. Or create a new project and replace this resource folder by kitchenSink resource folder, then attempt to run it.
    3. You can also newly download it to completely delete previous then try to run it.

    After all, let me know if you have any question about this query.
