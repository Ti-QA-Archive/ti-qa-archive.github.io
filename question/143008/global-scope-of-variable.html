Global scope of variable

Hi
I have two windows "win1" and "win2". I have declared a variable in win1 as
var patient;

the win2 has been created like this 

var win2 = Ti.UI.createWindow({
url:'home.js'
});

the problem is I am not getting the value of the variable patient in home.js.

can somebody help me

    Way to Fetch in home.js

    var win = Titanium.UI.currentWindow;
alert(win._patient);

  • Hi, There are many ways to do this one way is pass that variable to that window like

    win2.variable = "abc";
win2.open();

    or you can also define global variable which is accessible to all window like

    Ti.App.variable = "abc";

    or you can also use set property like

    Titanium.App.Properties.setString("my_prop","cool");

// and you can use this value 

var vari1 = Titanium.App.Properties.getString("my_prop");

    if you use first method passing variable to window it is accessible to only that window.
    if you use second method then that variable is accessible to all window.Changes made into that variable also available to all window but this variable is not persist after app close.. but if you use third window then that set property persist after app close for eg. storing username and password or auth token so that avoid every time login process…..
