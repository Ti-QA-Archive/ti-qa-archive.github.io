Recursive Directory Listing with RegEx name matching (Desktop)

This is for Titanium Desktop. I don't think it uses anything exclusive to Desktop, though, so it might work in mobile but I haven't tried it. Wanted to share it here because when I started working on it I found nothing that helped.

Recursive Directory Listing with RegEx name matching

function recFileList(path, pattern){
    p = fs.getFile(path.toString());        //Make sure we have a Titanium File object
    var ret = [];                            //Initiate return variable
    var fl = p.getDirectoryListing();        //Get listing for current directory
    for(var fn=0; fn<fl.length; fn++){
        if(fl[fn].isDirectory()){            //If current item is a directory then make recursive call to drill down
            ret = ret.concat(recFileList(fl[fn], pattern));
        } else {
            if(pattern.test(fl[fn].name())){
            //If file matches the regex pattern then add to list
                ret.push(fl[fn]);
            }
        }
    }

    return ret;            //Return recursive directory listing
}
