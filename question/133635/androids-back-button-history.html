Android's back button history

Is it possible to reset android's back button history?

What I am doing in my app is:

  1. Launch the app
  2. //do stuff
  3. Check if the content available on the app is the same as the available on website
  4. if needed, download the updated contents

once the contents are updated I update the interface of the app, and I have to prevent the user to go back to the old interface.

Would be nice if, somehow, I could "block" the app, and just let the users navigate through this new interface.

    You can add a event listener for 'android:back' to the window and define you own action instead of the default one. See http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.UI.Window.android-back-event.html
