ios NavigationWindow styling

I'm trying to use an image for a button in the left top nav of an ios app in the navigation window. I can't seem to get the button styled in any way though. I can't change the color, can't change the background, can't change anything about it other than what the button says when I edit the XML. Any ideas?

This is the markup. How would I style the button?

    <NavigationWindow id="iNav">
        <Window id="home" title="">
            <LeftNavButton>
                <Button id="menuButton">Options</Button>
            </LeftNavButton>
        </Window>
    </NavigationWindow>

    Hello Brandon,

    When you will add a navigation button, you will add under a view, then it will work properly. For clear understand check this following code. 

    <Alloy>
    <NavigationWindow id="navWin" platform="ios">
        <Window id="win1" title="Red Window" backgroundColor="red">
            <LeftNavButton>
                <View width="Ti.UI.SIZE" height="Ti.UI.SIZE">
                    <Button id="menuButton" image="appicon.png" backgroundColor="transparent"></Button>
                </View>
            </LeftNavButton>

            <RightNavButton >
                <View width="Ti.UI.SIZE" height="Ti.UI.SIZE">
                    <Button id="rightButton" backgroundColor="green" color="red" height="30" width="50">Done</Button>
                </View>    
            </RightNavButton>
        </Window>
    </NavigationWindow>
</Alloy>

    If you have any question about this issue then feel free to ask me.

    Thanks.
