    Currently, backgroundColor and backgroundGradient for both tabs and tabGroup unfortunately don't work for Android. You could try using a backgroundImage filled with your desired color instead, though.

    Alternatively, you could synthesize tabGroups using views, as this kind user has done and shared with the community.
