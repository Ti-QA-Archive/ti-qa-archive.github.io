setting installLocation to auto

hi,

is it possible to set the installLocation in manifest.xml to "auto"?

i tried to edit tiapp.xml adding android:installLocation=&quot;auto&quot;
but when compiling i get this error:
No resource identifier found for attribute &#39;installLocation&#39; in package &#39;android&#39;

i use titanium 1.6 sdk, and android 2.3

— asked March 7th 2011 by marcin kolonko

    I think you need to include the Tool API Level, the below works for me

         <tool-api-level>8</tool-api-level>
    <manifest android:installLocation="auto">
        <uses-sdk android:minSdkVersion="4"/>
    </manifest>
