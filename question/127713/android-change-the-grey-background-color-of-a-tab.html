Android: Change the grey background color of a tab

Hi,

I'm struggling to change the background color of a tab in android. As you can see from the screenshot( http://imageshack.us/photo/my-images/573/annoyinggrey.png/ ), I load an icon and I would like to use white background color.
My first approach was to use the backgroundImage property for the tab (using an image with white background that it would cover the whole tab). It doesn't work, it does not load the image at all. So I use the icon property, but like this, the image occupies only a part of the tab.
I tried setting all available properties like backgroundColor, backgroundDisabledColor, backgroundDisabledImage, backgroundFocusedColor, backgroundFocusedImage, backgroundImage, backgroundSelectedColor (with focusable enabled) for both tabgroup, tab and window but the grey insists.

Is there ANY way to use different color or I'm stuck with this grey?

PS: I have also noticed that when setting the focusable true for window, the app is crashing.
