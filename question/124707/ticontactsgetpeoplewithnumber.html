Ti.Contacts.getPeopleWithNumber

If anyone is interested or needs it I've modified the SDK to provide a function where you pass in the phone number you're looking for and it'll return all matching contacts with that number.

You can select exact or partial match as well.

Code is here:

http://pastie.org/2423303

You need to modify the ContactsModule.m file and stick that code in. I put it just above 

-(NSArray*)getAllPeople:(id)unused

You also need to modify ContactsModule.h

And put this

-(NSArray*)getPeopleWithNumber:(id)arg;            /* CSM: new API */

just above 

-(NSArray*)getAllPeople:(id)unused;

Then you can call it in Javascript as

var contacts = Ti.Contacts.getPeopleWithNumber("123", false);
if (contacts) {
    for (var i=0; i<contacts.length; i++) {
        alert('Contact name is ' + contacts[i].fullName);
    }
}
else {
    alert('Contacts is empty');    
}

the second parameter is for "exact" match.

Note that you just provide a number ie: 123456123 and don't worry about the + or the ( ) that show up in the normal phone number fields as they'll be stripped for comparison.

If you don't supply the second parameter it assumes exact match by default.

Enjoy.
