Textfield: doesn't raise tap event

Hi,

I'm facing a strange issue I'm sure it's quite newbe-ish.

I have this view, which is quite straightforward:


<Alloy>
    <View class="container">            
        <TextField id="myField"  onChange="onChange"></TextField>            
    </View>        
</Alloy>

I want the keyboard to appear when I tap on the field, and disappear when I blur. I thought this to be a normal behaviour, but so is not. Moreover, cursor blinks in the textfield, no matter if I execute the blur() method or not.

I also tried some dummy-code for testing purpose, such as:

  1. I handled both onClick nand onTouchend events, bot no one raised
  2. I set softKeyboardOnFocus =&quot;Titanium.UI.Android.SOFT_KEYBOARD_SHOW_ON_FOCUS&quot; so I proved that keyboard is working correctly, but since I return and I get the keyboard disappeared, it's not appearing anymore
  3. I also set touchEnabled=&quot;true&quot;, with no effect at all

I'm working with Titanium SDK 3.1.0, Alloy 1.2.0, Android 2.3

Is there any known issue about it, or it's jut me?

Thanks

        <TextField id="myField" backgroundColor="orange" width="150" height="40" bubbleParent="false" />

    JS

    $.myField.blur();
});

$.container.open();

    The main thing to notice here is that I have set the bubbleParent property for textField to false, to stop its event bubbling to the parent.

  • I've made something different cause if you have touchable views on the window, event can be not fired.

    Just add fullscreen view (remove_blur_view) when textfield is in focus and keyboard opened.. addListener click on the new view (remove_blur_view, top of all others view) and just blur and remove view when it's clicked.

    You can pass a variable to defined wich textfiled is active to use this method with all your textfield in app…

    I hope it's helpful.

    $.t_birth.addEventListener('focus',activeBlur);

    function activeBlur()
    {
    $.view_blur.visible = true;
    $.view_blur.touchEnabled = true;
    $.view_blur.addEventListener('click',closeKeyboard);

    }

    function closeKeyboard()
    {
    $.view_blur.removeEventListener('click',closeKeyboard);
    $.view_blur.visible = false;
    $.view_blur.touchEnabled = false;
    $.t_birth.blur();

    }
