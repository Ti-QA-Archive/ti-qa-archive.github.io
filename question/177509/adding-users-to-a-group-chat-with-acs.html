Adding users to a group chat with ACS

Hello,
I'm using ACS to adding a simple group chatting functionality to an app that I'm building, everything is working well, but I'm stuck with one thing, I want to have the ability to add users to a chat, for instance let's say I have a group chat with A, B, and C, now I want to add D to the conversation, what would be the best way to establish that? I tried many ways but ACS always ends up creating a new group message with the new users.
