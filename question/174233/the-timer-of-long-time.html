The timer of long time

Hello,
I plan to make a long time timer function in my application.

It operates by a stand-alone without network,
the interval of a week or the month can be set up,
and even if a device reboots,
it will continue operating.

In my understanding,
I think that I should use iOS Background fetch event and Background service of Android.

Is my understanding right?

Thank you for your kindly support. :)

— asked May 15th 2014 by Kazuhiro Kusuyama

  • Can you you add more details about what the service is intended to do in the background? For a common timer app you don't need to set up a service that runs in the background all the time - For notifying the user when the timer has ended I would recommend to use Titanium.App.iOS.LocalNotification on iOS and benCoding.AlarmManager on Android.
