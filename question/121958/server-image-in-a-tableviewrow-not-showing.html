Server Image in a TableViewRow not showing

I'm trying to show a server based image in a tableviewrow and I am having no luck at all. I create the tableviewrow with a description and an image url, then add this to my createtableview expecting to see a row with an image and text, all I see is the text

mydata.push = (Ti.UI.createTableViewRow({title:'Table row', leftImage:'http://xxx.xxx.xxx.xxx/image1.png'}));

var tableview = Titanium.UI.createTableView({data:mydata});

I am sure it must be a basic mistake I am doing, so if someone can point it out for me it would much appreciated.

Thanks.

    I have found success when using remote images in a table view when I use an imageView not the leftImage property.

    var row = Ti.UI.createTableViewRow({
   height:'auto',
   hasChild:true
});

var nameLabel = Ti.UI.createLabel({
    text:item.name,
    top:8,
    left:68,
    width:220,
    height:16,
    textAlign:'left'
});
row.add(nameLabel);

var imageView = Ti.UI.createImageView({
    image:'http://xxx.xxx.xxx.xxx/image1.png',
    top:0,
    left:0,
    width:48,
    height:48,
    preventDefaultImage:true
});

row.add(imageView);
