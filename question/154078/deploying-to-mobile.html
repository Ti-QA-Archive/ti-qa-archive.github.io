    On iphone you can do it from Titanium Studio. This is different from how it's done on Android, there is no file to manually cop:

    First you'll need to have the appropriate development certificate and provisioning profile. you can find more here

    Then from Appcelerator Titanium you just right click on the project in project explorer and choose the option to install the app on iPhone and if everything was configured correctly you'll have the app installed on device.

    detailed docs http://docs.appcelerator.com/titanium/latest/#!/guide/Deploying_to_iOS_devices
