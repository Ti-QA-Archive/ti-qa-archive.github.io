    Hi.

    First, add checkbox reference to table view row after it is created:

    thisRow.thisCheckbox = thisCheckbox;

    Then in table view click event handler simply do this:

     table.addEventListener('click', function(e)
{
    Ti.alert(e.row.thisCheckbox.value);
});

    Basically, in table view click event handler you have reference to clicked row and you just need to get checkbox easily and that's why I have added it's reference to the table view row.
