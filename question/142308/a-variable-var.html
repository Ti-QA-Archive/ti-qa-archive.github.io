    Hi Jesse,

    As Christian mentioned above, you'll need to add the buttons individually. So, something like this:

    var data = [];

for(var x = 1;x<6;x++){
    var myButton = Ti.UI.createButton({title:L('string'+x), custom_item: x, width:btnWidth, height:btnHeight});
    myButton.addEventListener("click",btnCallback);
    v1cont.add(myButton);
    data.push(myButton);
}
function btnCallback(e)
{
    view.currentPage = e.custom_item;
}

    Hope that helps!

  • You will to assign an eventListener for each button. Keep in mind that data is of type Array. And there is no support for the click event for an array.

    Also, if you look at the documentation for the add method, you will see that it returns void (nothing). So trying to set the a variable with the return value of this method won't work.

    The proper way you should approach this would be to:

    • Create your buttons
    • Add them individually to your container (vlcont in your case)
    • Add the click eventListener to each button.

    A good approach for that would be to encapsulate these 3 operations in a loop or function.
