Detect Any Touch/Click App Wide?

Is there a way to attach an event at the App level that listens for any touches/clicks? I want to have a timer that does something only if the app has been idle for a given number of seconds. I'd rather not attach an event to every part of my application… thanks!

  • I believe you can Ti.APP.addEventListner('click', function() { reset timer() });

    should then work.

    Other than that you can add the reset timer function to the call of all specific clicks, which would be a real pain..
