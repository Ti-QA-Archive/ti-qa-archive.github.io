  • In my app, I've worked around similar issues by using hide() and show() after the window has already been opened, instead of close() and open(). I set a bool called "initialized" in each window controller that is set to true the first time the window is opened. Then whenever I want to open the window again, I check that bool and determine whether I need to show() or open().
