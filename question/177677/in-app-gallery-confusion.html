In-app gallery confusion

Hi all,

I've been searching docs, google and the q&as for a couple of days and I cannot get my head around it.
Building for ios + android with Alloy.

  2. Then the user can add photos from the gallery or the camera (this part I've got figured out, except I just add them to the screen without any formatting/thumbnailing).
  3. The user should be able to delete each photo in this gallery, maybe even reorder them.

    Gary,

    You will need to "roll your own" gallery to get what you want. There is no "gallery view" to leverage natively. You might try checking gitt.io
    Maybe something there that might be a good starting point for what you want. I did a little app called Magicpad for iOS earlier this year. Uses a 'grid control' that let's you add, delete, save (with some code modifications) that might be a good starting point for you. Check out photogrid
