ti.android.useLegacyWindow not working after receiving titanium update

Hi,
I am facing a very strange issue. Till today morning I was able to run mobile application for android and iphone using titanium classic framework 3.2.3GA properly, but after receiving notification that new update 3.3 is available. I just check for updates to see new additions under 3.3 and saw support for lightweight winsdow is removed so didnt update. But since then when I am running the application it is creating heavyweight window even though ti.android.useLegacyWindow is set to true. Any idea what is the issue? Is it a Titanium Studio bug?

  • As you can read in the Release Notes lightweight windows were dropped in 3.3.0:

    "As of this Release, support for lightweight windows (windows without an Android Activity) is removed and using the ti.android.useLegacyWindow property in the tiapp.xml file has no effect."

    — answered July 18th 2014 by Manuel Lehner
