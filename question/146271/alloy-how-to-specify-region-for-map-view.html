Alloy: How to specify region for Map view?

The Titanium 3.0 documentation mentions how to build a Map view with the help of the new Alloy framework, it even mentions regionFit=true unfortunately it does not seem possible (yet?) to set the region with its latitude, longitude and delta. Could someone please clarify?

    • Thanks for pointing to this example, in which you declare the map annotations as part of the style declaration. Actually (as least for my use case) I consider the annotations not part of the styling, but more of the view definition. Anyways I ended up setting the region programmatically inside of my controller logic, like:

      $.mapview.region = {latitude:53.5686, longitude:10.0386, latitudeDelta:0.01, longitudeDelta:0.01};
