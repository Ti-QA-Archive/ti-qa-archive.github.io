Basic layout of appcelerator app

I'm creating my first appcelerator app and looking for some info on whether something is possible, and if so is it easy to implement.

I'm using Titanium SDK 2.0.1.

I'm trying to create a layout similar to the Stream window in the Google+ app for Android.

  1. 1 A large header (Can the navbar property of Ti.UI.createWindow be adjusted?)

  2. 2 Two tabs under the header (Would like to be able to adjust height of them if possible? Are there alternatives to Ti.UI.createTabGroup if this is not possible?)

The main content I can figure out easily enough I reckon.

Any examples or links on how to achieve this layout would be great.

  • Accepted Answer

    1 A large header (Can the navbar property of Ti.UI.createWindow be adjusted?)

    • no, but you can use a custom Ti.UI.View

    2 Two tabs under the header (Would like to be able to adjust height of them if possible? Are there alternatives to Ti.UI.createTabGroup if this is not possible?)

    • No you cannot, but hey there are still Ti.UI.View components :)

    The Tabs are at the top on android by default, in iOS they are at the bottom.

    Basically you can do all imaginable layout, some cost more effort, some less.
    I also did a custom TabGroup, where my Tabs were at the top (iOS) including custom navGroup to guarantee platform-independence. Therefore its all possible, but may not that easy as a beginner.
