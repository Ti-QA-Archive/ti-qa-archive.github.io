TableView.search in iphone

In android, the below code displays a tableView with a search bar.

searchBar = Titanium.UI.createSearchBar({
    hintText: 'type to filter rows',
    showCancel: true 
});

var headerLabel = Ti.UI.createLabel({
    backgroundColor: '#6d84a2',
    color: "white",
    font: { fontSize: 15, fontWeight: "bold" },
    text: "Incidents - support1",
    textAlign: "center",
    height: 35,
    width: 320 
}); 

tableView = Ti.UI.createTableView({
    top: 42,
    borderWidth: 2,
    separatorColor: '#000',
    search: searchBar,
    headerView: headerLabel, 
    backgroundSelectedColor: '#c8f'
});
tableView.addEventListener('click', viewDetails);
win.add(tableView);

//set tableView.data
showResultSet();

But in iphone, the search bar is missing. Whats wrong with this code??
