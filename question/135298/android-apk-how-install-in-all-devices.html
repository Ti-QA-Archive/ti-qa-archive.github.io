  • Daniel has you headed in the right direction, so will just add some comments instead of muddying the answer waters.

    The Xperia mini ships with 2.3 (gingerbread) so it should meet the min sdk requirements. Sony also claims they intend to build up to 4.0 (ICS) on it, that be interesting to see if it happens.

    If you can, tether the xperia to your system via usb, and connect to it using adb/ddms and get the log file. You should be able to watch what happens while you are trying to install the app.
