How To Decrypt MD5 Encrypted Value In Titanium Alloy

I have the MD5 encrypted value, and also Key 'PrivateKey' and 'PublicKey'

Now locally I want to decrypt the encrypted MD5 value in titanium alloy.
Since encryption have done in asp .net MicroSoft, I have the key where i can use this key for decrypt locally at my side using javascript titanium.

Can any one advice me how to do decrypt in titanium alloy framework.

  • yes - you cannot.

    MD5 is a cryptographic hash function designed to go one way (i.e. there is nothing to decrypt). It does not use Public/Private Keys as it is designed to be used as a digital signature (for example). See http://en.wikipedia.org/wiki/MD5
