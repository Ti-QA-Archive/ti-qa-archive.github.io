  • Is there a reason to get it through webView?

    I don't think Android security models, allows you open it automagically once it has been downloaded. If the app is beta testing phase and available only for specific users, you can always get it by doing a HTTPCall request and save it to a ExternalStorageDirectory.

    But, at the end of the day I think it's much better to link users to Google Play page of your app.
