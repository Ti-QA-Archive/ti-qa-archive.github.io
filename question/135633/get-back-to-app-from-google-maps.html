Get back to app from google maps

I've got a mapview, with a Get Directions button that opens Google Maps and shows the user driving directions. This works great, accept that once in the directions window, there's no way to get back to my app, besides killing the app. Anybody know of a way to get back to the app from google maps? Here's my google maps openurl code:

Ti.Platform.openURL(&#39;http:&#x2F;&#x2F;maps.google.com&#x2F;maps?saddr=&#39;+latitude+&#39;,&#39;+longitude+&#39;&amp;daddr=&#39;+claimLatitude+&#39;,&#39;+claimLongitude);

  • Hi Tim,

    Instead to using openURL you can create one window and open that window adding webview that work for you.
    Try Below code…

    var win = Ti.UI.createWindow({
            title: 'webview'
        });
webview = Ti.UI.createWebView({
url : 'http://maps.google.com/maps?saddr='+latitude+','+longitude+'&amp;daddr='+claimLatitude+','+claimLongitude
});
win.add(webview);

Ti.UI.currentTab.open(win, {
            animated : true
        });
