    • var contactosMarcados = Contactos.where( { marcado : 'true', interesado:'false' } );

      So IT IS an alloy collection after all !!

      Then you could try to use something like:

      var c = contactosMarcados[i].toJSON();
console.log(c.fullName)

      toJSON() is perhaps a confusing name for that method which actually returns you an object with just the models attributes;

      or you can get them with:

      var c = contactosMarcados[i].attributes;
console.log(c.fullName)

      Also see http://docs.appcelerator.com/backbone/0.9.2/#Model for more informations on alloy/bb models

      hth.
