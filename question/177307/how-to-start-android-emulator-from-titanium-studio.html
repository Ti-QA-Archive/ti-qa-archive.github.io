  • Hi

    You need to make sure that you have the Android Platform SDK installed.

    • Load Titanium Studio up and sign in.
    • Near the top of your window should be a symbol/button of a 'house', this will launch Dashboard if it isn't already open.
    • Click 'get Started' and then scroll down to 'Configure Native SDK's', there should be an option for 'Android SDK' which you should click.
    • To the right should be another button that will bring up another window allowing you to configure your Android SDK's. It should look like this Installing Platform SDKs, the link will also give you more information about installing SDK's.

    Once you have installed the Android SDK's, you now need to set up or 'manage AVDs', which are the emulators themselves. More information can be found here Installing the Android SDK

    Below is the route you should take:

    Titanium Studio > Preferences > Studio > Android > Manage AVDs

    Once you have created an emulator, you should now be able to select 'Android Emulator' from your run options and the emulator which you just created.

    Now, like many you will probably come across a lot of problems using Titanium Studios native emulator. Many are now using something called GenyMotion and I would recommend this myself. It's free to sign up and it's faster than the native emulators which is good for development. There is a guide here also for Installing Genymotion

    Hope this information helps.
