Make a timer that does something on an increment?

I am trying to create a timer that will call a method every 50 milliseconds or so, but I can't seem to find any way to do it. Does anyone have any ideas?

    You can do the following:

    
var myFunction = function()
{
    // do something here
}


setInterval(myFunction, 50); // runs the function every 50ms

    • If the user completely quits the app, then the timer will also stop. It will restart again from 0, next time the app is started. So you can't have something set to 1 day etc, since it probably won't execute in a real world scenario (no one will have an app running in the foreground for 24 hours or more, unless it's a super, awesome, mega-addictive, life changing app :)

      If the app is only put into the background, I'm not really sure what would happen. But I wouldn't recommend setting something like 24 hours anyway, because if an app is in the background, the OS can reclaim memory at any time (if required), which could mean your app might force quit and next time the user runs the app, the timer will start at 0 again. This could happen after 5 mins or even after a few hours, it totally depends on how the end-user uses his phone, which you have no control of.

      Hope this helps.
