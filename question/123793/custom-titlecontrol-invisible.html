Custom titleControl invisible?!

I'm trying to build a custom title bar using titleControl.

As simple test I wrote:

var lbl = Ti.UI.createLabel({
        text: "Nur ein Test",
        height: 30,
        width: 50
    });
    var vi = Ti.UI.createView({
        backgroundColor: '#f00',
        height: 40
    })
    vi.add(lbl);

    var win = Ti.UI.createWindow({
        backgroundColor: "#00ff00",
        titleControl: vi
    });

    win.open();

But everything I get is my green window without any title bar. Why? Any Idea?

  • Accepted Answer

    If you open a window without being part of a navigationGroup or tabGroup you do not have the navBar.
    In this case you will have to fake it with a normal toolbar.

      var win = Ti.UI.createWindow();
var my_title_control = Ti.UI.createLabel({
    text:'Title Control'
});

var my_nav_bar = Ti.UI.createToolbar({
    top:0,
    items:[my_title_control]
});

win.add(my_nav_bar);
win.open();

      not tested but you should get the idea :)
