Where .csv file can saved?

Hi,
I generated one csv file from array, but I dont know how to see that was created or not. How can I see my csv file, where it is stored?

This is my code…

report1.addEventListener('click',function(e)
{
var result = [];
result.push('"content"\n');
for(var i=0,i<=checkData.length; i++)
{
 result.push('"');
 result.push(checkData[i].name);
 result.push('"\n');
}
var dataString = result.join('');
var exportFile = Titanium.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory,'rpt.csv');
exportFile.write(dataString);
alert("export.csv created: " + String(new Date(exportFile.createTimestamp())));
});

Here checkData is one array contains data.Someone can help me pls…

regards,
Lokesh.

  • The file rpt.csv is being created in your app applicationDataDirectory - you ca read the values of that file and if you just want to check you could Ti.App.info the contents to the console to check that it is being written correctly.

    You should take a look at the FileSystem API Docs and the Wiki entry

    You could do something like this:

    var csvData = exportFile.read().text;
Ti.API.info(csvData);

    After the file has been written. Then you can check the contents are what you expect them to be.
