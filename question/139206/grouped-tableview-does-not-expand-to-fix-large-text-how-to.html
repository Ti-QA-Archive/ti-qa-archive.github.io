Grouped TableView does not expand to fix large text? How to?

I have a grouped tableView, like this:

var inputData = [
    {title:'Title', header:''},
    {title:"This is the title"},
    {title:'Content', header:''},
    {title:"This is the content. This is the content. This is the content. This is the content. "}
];
var tableView = Titanium.UI.createTableView({
    data:inputData,
    style:Titanium.UI.iPhone.TableViewStyle.GROUPED
});

This works fine, apart from the fact that only 1 line of the content is visible.

QUESTION 1:

How do I allow the entire content to be visible within the tableView?

QUESTION 2:

Is it possible to change things like the font size, font color, font family etc of each row in the tableview? For example have the title in red bold, and the content in green… etc

Thanks guys…

  • Hi Donal,

    Try this code.

    var arrData = [{
    title : 'Title',
    header : 'First'
}, {
    title : "This is the title"
}, {
    title : 'Content',
    header : 'second'
}, {
    title : "This is the content. This is the content. This is the content. This is the content. "
}];
var data = [];
for (var k = 0; k < arrData.length; k++) {
    var row = Ti.UI.createTableViewRow({
        height : 'auto'
    });
    if (arrData[k].header) {
        row.header = arrData[k].header;
        row.color = 'green';
    }
    var lblView = Ti.UI.createView({
        width : 'auto',
        height : 65,
        // backgroundColor : 'green'
    });
    row.add(lblView);

    var lblEventName = Ti.UI.createLabel({
        text : arrData[k].title,
        left : 0,
        // top : 0,
        color : 'red',
        width : 'auto',
        height : 'auto',
        font : {
            fontSize : 17
        },
        // color : '#000044',
        // backgroundColor : '#dff2f6',
    });
    lblView.add(lblEventName);
    data.push(row);
}
// tableview object
var tableView = Titanium.UI.createTableView({
    width : 300,
    backgroundColor : 'transparent',
    data : data,
    separatorStyle : Ti.UI.iPhone.TableViewSeparatorStyle.NONE,
    top : 10,
    style : Titanium.UI.iPhone.TableViewStyle.GROUPED

});

win.add(tableView);
