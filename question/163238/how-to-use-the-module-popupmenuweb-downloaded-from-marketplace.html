  • Here is the description of using the module (found in the document):

    PopupMenuWeb Module

    Description

    This module extends the Titanium TiUIWebView to allow custom buttons in UIMenuController.

    Accessing the PopupMenuWeb Module

    To access this module from JavaScript, you would do the following:

    var PopupMenuWeb = require("com.mywebrank.popupmenuweb");

    Methods

    The module provides no public api. This module extends the Titanium TiUIWebView.

    Events

    selection

    Fired when the user select text on the webview.

    Properties

    index : Number Index of the button.
    text : Selection text
    html: Selection HTML
    rect: Rect for selected text

    Usage

    See example.

  • It's not clear what the final solution is from the previous discussions. Here's an example that I had developed when the plugin was first released that I have just tested in Titanium SDK 3.2.3.GA.

    var popupMenuWeb = require("com.mywebrank.popupmenuweb");
var topicWebView = Ti.UI.createWebView({popupMenu: ["Note", "Search", "Share"]});
topicWebView.addEventListener("selection", function(e) {
    Ti.API.info("Button: "+ e.index);
    Ti.API.info("Selection Text: "+ e.text);
    Ti.API.info("Selection HTML: "+ e.html);
    Ti.API.info("Selection Parent ID: "+ e.source.evalJS("window.getSelection().anchorNode.parentNode.getAttribute('id')"));
});
topicWebView.html = topicHTML;

    I used the evalJS() method of the webview to then query the selection.

    Hope this helps
