Mixing relative and absolute positioning

Is it possible to have a view with vertical/horizontal layout, and then have child content that positions itself absolutely, rather than obeying the layout rule of its parent? In the same way you can in CSS with 'position: absolute'?

    can't you put the child items to be positioned differently in their own view (which would have a layout of composite) and simply add this "container" view into the parent, vertically layed out container ?

    Something like:

    var parent = Ti.UI.createView({ layout: 'vertical' });
for (i=0; i<4;i++) {
  // stack some views vertically
  var lbl = Ti.UI.createLabel({text: "label"});
  parent.add(lbl);
}
var container = Ti.UI.createView({layout: 'composite'});
var otherlabel = Ti.UI.createLabel({text: "postioned", top: 23, right:5});
container.add(otherlabel);
// … more of the same ...
parent.add(container); // vertically

    I didn't test the code, but that's the idea
