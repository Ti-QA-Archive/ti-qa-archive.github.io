};

  • Thank you for eveyone who tried to help, but I have solved this problem.

    It turns out, that passing data through a TCPsocket adds an invisible ' to the end of the string. It does not show in any logging. All I had to do was remove the last character of the string, and then I could parse it without issue.

    function(e){
    if (!server) return;
    var json_string = e.data.text.substring(0,e.data.text.length-1);
    var submitted=JSON.parse(json_string);
    return submitted;
};
