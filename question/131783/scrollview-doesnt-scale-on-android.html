ScrollView doesn't scale on Android

Does ScrollView no longer scale on Android? I have an application that scales 4 images at once using the pinch event on iOS. However, that event is not supported on Android, and attempts to get the scale event on a scroll view have been fruitless. Is this behavior no longer supported on Android? I am using 1.8.0.1. Documentation indicates that "scale" is fired on Android…but I can pinch it all day long and it does nothing (while working fine on iOS).

fullMapView.masterView.addEventListener('scale', function(e) {
    Ti.API.debug(e.scale);
});

This event never comes…probably because the scale never changes.

    • 2.2 is required, because it uses ScaleGestureRecognizer (which came with API Level 8). However, Titanium itself has not supported 2.0 since version 1.6.x so it isn't that big of a deal. A lot of (all?) devices that can't upgrade to 2.2 are not multi touch capable anyway.
