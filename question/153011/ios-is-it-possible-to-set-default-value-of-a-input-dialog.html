  • Hi

    The short answer is no.

    The alertDialog is mostly use to announce things, the ability to include the input boxes is only deemed an assistance rather than a full blown UI element in its own right.

    You will need to create your own dialog UI if you need to provide default values or your own hint text values.

    Sorry.
