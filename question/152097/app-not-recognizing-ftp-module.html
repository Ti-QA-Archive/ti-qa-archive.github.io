App not recognizing FTP Module

Hi,

I recently installed the Titanium FTP Module since I am trying to copy pictures from my website to display on my app. However, when I run the app I get an error that says: “Requested module not found”.

I have followed the installation instructions on these two links: http://developer.appcelerator.com/question/125954/how-do-you-install-a-module-into-titanium-studio and https://wiki.appcelerator.org/display/tis/Using+Titanium+Modules, but still it does not work.

Just to make sure here are the steps I followed:

  1. Downloaded and unzipped the FTP module

  2. Copied the Modules directory of the unzipped download to the root of the project (it on top of the Resources directory, that is, at the same level)

  3. Modified the modules portion of the tiapp.xml file as follows:

    <modules>

     <module platform="android" version="0.1">com.apiit.FTPModule</module>

    </modules>

  1. Added the following to the .js file where I need to make the FTP call:

var ftpmodule = require('com.apiit.FTPModule');

  1. When I need to do FTP on my app I am using the following statement:

              var filenameToDownload = data['item']['pic1'];

         var localFilenameToStore="currentplantpic.jpg";

         var downloadResult = ftpmodule.accessFTP("ui/common/"+localFilenameToStore,   "ftp://username:password@FTP Host:Port/Directory if it is there/File To Download,"get");

         alert(downloadResult);

I am using a PC with Windows 7 to run the emulator. I am using Titanium 3.0, to run the app. Can you see anything wrong I may be missing? Do you have any other things I should be looking at?

Francisco
