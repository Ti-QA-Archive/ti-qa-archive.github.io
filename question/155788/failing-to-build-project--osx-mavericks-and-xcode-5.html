  • Hi Matt,

    it is possible to install Xcode 4.6.3 on Mavericks (to use the current Titanium SDK) with a little trick: open Appstore, then edit /System/Library/CoreServices/SystemVersion.plist and change the ProductVersion to 10.8, download Xcode, then change the ProductVersion back to 10.9.

    Cheers
