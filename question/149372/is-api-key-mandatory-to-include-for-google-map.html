Is API key mandatory to include for Google Map

I am creating an Application for IOS and Android. I am using Titanium.Map.createView method for creating Map view.Do I mandatory to generate API key for Google API?

  • Hi

    Yes and No.

    • iOS = No, simply use it as you want, no API key is used.
    • Android = Yes, follow the instructions in the module map and Google Maps v2 for Android.

    Please note: you now MUST use the new version of Google Maps v2 for Android as you cannot obtain any new API keys from Google for v1. This changed just last week.

    BTW the guide to gaining a new Android API key is very easy, just follow the guide exactly.

    • Yes, because the built-in map view REQUIRES a version 1 API key and those are no longer being issued.

      Existing projects where you have already generated a map api key will work until it is deprecated in the future.

      The only way you can continue to use the Map View in new projects is to add the custom module to your project. It looks like it you do need to upgrade to Ti SDK 3.0.2 in order to run it according to the docs.
