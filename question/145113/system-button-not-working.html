  • Hi

    Unless you are creating windows using the url property or the createWindow then Titanium.UI.currentTab will not return the currentTab.

    Your code does not include a property called url so I suspect you are not using this older style technique that is no longer considered the best way to create apps as each window become its own context.

    So this will not work.

    Titanium.UI.currentTab.open(winCrea, {animation:true} );

    You will need to keep access to the current tab variable and then change the line to something like;

    tab1.open(winCrea, {animation:true} );

    Which should be easy unless the window you create is used in several different tabs. If it is, then just requires planning.
