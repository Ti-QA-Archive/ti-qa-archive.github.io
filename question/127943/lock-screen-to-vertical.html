Lock screen to vertical

Hi,
Is it possible to lock the app to the vertical layout? I dont want to give the user the ability to use horizontal layout. Is that possible?

    Yes …

    for iPhone 

        var MSWindow                                 =     Ti.UI.createWindow
    ({
        navBarHidden:                                true,
        backgroundColor:                            FSUT.GS.LO.screenBKCL,
        orientationModes:                             [Ti.UI.PORTRAIT]
    });

    but for android you need to set the orientation mode outside of the window creation..

    orientationModes = [Ti.UI.PORTRAIT]..

    You can also set the iPhone orientation if the tiapp.xml file. Think you can do the android one as well but not got there yet.
