  • That option is not from XCode, it's a template added by the installation of Titanium. It just creates an empty template for creating modules and not wrapping an existing one. There's no easy way to wrap an existing native iOS App with Titanium because the Ti Runtime stores compiled/obfucated javascript code that is used to communicate with the proxies via the Kroll runtime (a closed-source component of the Ti Architecture)
