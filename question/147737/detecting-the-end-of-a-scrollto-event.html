Detecting the end of a scrollto event...

I have a large grid image within a scrollview which, when the user stops dragging (or swiping) the image around, it activates a scrollto event which locks on to the nearest square.

The problem is, i am then trying to detect the end of that automatic scrollto event so i can bring up other elements on screen.

But how do i detect that specific scrollto scrollend event?

This is the function thats called when a user finishes dragging around the image or when the main scrollview has finished scrolling after a swipe event:

function moveover() {

    var valx = scrollView.contentOffset.x / 50;
    var offsettingx = Math.round(valx) * 50;

    var valy = scrollView.contentOffset.y / 50;
    var offsettingy = Math.round(valy) * 50;

    scrollView.scrollTo(offsettingx, offsettingy);

}

Its that last scrollTo event that im trying to track the end of so i can call some other functions.

Thanks in advance…
