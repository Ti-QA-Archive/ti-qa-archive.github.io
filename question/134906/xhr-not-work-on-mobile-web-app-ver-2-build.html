  • Mobile Web has some limitations when using XHR requests imposed by the browser. If you are doing a cross-domain request, the destination must either have CORS (Cross Origin Resource Sharing) enabled, or you need to use a proxy in conjunction with Ti.Network.httpURLFormatter. For more information, check out http://docs.appcelerator.com/titanium/2.0/index.html#!/api/Titanium.Network and http://docs.appcelerator.com/titanium/2.0/index.html#!/api/Titanium.Network.HTTPClient
