Complex set of events, is it possible to execute this....

I have a window with a scrollview.

If you click an option in the scrollview, it will open a new window with data related to the item.

I created a button in the subwindow. If the button is clicked, I want the subwindow to close (subwindow.close()) which I can do, but after its closed, I want a function called scrolltobottom (which scrolls my scrollview down) to execute.

The function is defined in the primary window, but it has to somehow be executed by the subwindow AFTER the subwindow is closed.

I hope this makes sense. If it does make sense, I hope its possible :)

THanks!

    fire an event from the subWindow before you close it and have the scrollView listening for the event

    // in the main window
Ti.App.addEventListener('subwindow.closed', function(data) 
{ 
     // do the scroll thing here
});

    in the sub window

    closeBtnSubWindow.addEventListener('close', function(data) 
{ 
     Ti.App.fireEvent('subwindow.closed', {});
     subwindow.close();
});
