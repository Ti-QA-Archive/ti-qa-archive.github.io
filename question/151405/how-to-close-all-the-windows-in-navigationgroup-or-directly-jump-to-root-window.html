Hi,

I had already tried using "Alloy.CFG.navgroup.close();" but it actually make my screen blank.

    hi,

    so there is no any functionality to redirect to its rootview controller as we have in native iphone but you can handle an array of windows so when you click logout button , just close all the windows pushed in the array.

    To do that first declare a global array then whenever you open any new window just push that window in an array except login window because you want to redirect to login screen then also on close event of window you have pop that window too so when user click logout button just get the windows from an array and close it with navigation group. in this way you can redirect user to login screen.
