  • Well, you cannot retrieve a device's phone number as it is explained here. So calling them back is not an option. (unless the user gives you this information)

    One option though, would be to send a mail to a support address using the Titanium.UI.EmailDialog. You could pre-fill some details in the mail body and the users could add any additional info they want.

    since this is email, there is absolutely no cost to your customer.
