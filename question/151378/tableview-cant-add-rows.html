Tableview cant add rows

I have a table declared in my xml, which is as follows: 

<TableView id="goalsTable">

</TableView>

And then in my controller file i am trying to add rows, using this code:

for(var i = 0; i < 5; i++) {
    var row = Ti.UI.createTableViewRow({
        touchEnabled: true,
        height: '20',
        className: 'row',
        objName: 'row'
    });
    $.goalsTable.add(row);
}

For some reason, when i run my app and try open the page containing the table, the app "unexpecdantly quits". I am left with no error message, and i have no idea why it wont work. Any help?

  • Hi Dom,
    Try this way : 

    var data=[];
for(var i = 0; i < 5; i++) {
    var row = Ti.UI.createTableViewRow({
        touchEnabled: true,
        height: '20',
        className: 'row',
        objName: 'row'
    });
    data.push(row)
    $.goalsTable.setData(data);
}

    Thanks

    var rowData=[];
for(var i = 0; i < 5; i++) {
    var row = Ti.UI.createTableViewRow({
        height: 40,
    title : 'row ' + i.toString()

}
