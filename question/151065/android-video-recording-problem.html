Android video recording problem

Hi all,

I have recorded video in Android and now want to save the recorded file in my app's data directory. Currently the file is being saved inside DCIM (Gallery) folder with its own naming format. How can I save the recorded file in app data directory ?

My code is as; 

var intent = Titanium.Android.createIntent({
        action : 'android.media.action.VIDEO_CAPTURE'
    });
    var videoUri = null;
    var currentFileName= 'myRecord';

    var videoDir = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory, 'videos');
                if (! videoDir.exists()) {
                    videoDir.createDirectory();
                }

    Titanium.Android.currentActivity.startActivityForResult(intent, function(e) {
        if (e.error) {
            Ti.UI.createNotification({
                duration : Ti.UI.NOTIFICATION_DURATION_LONG,
                message : 'Error: ' + e.error
            }).show();
        } else {
            alert('success');
            if (e.resultCode === Titanium.Android.RESULT_OK) {
                videoUri = e.intent.data;
                Ti.UI.createNotification({
                    duration : Ti.UI.NOTIFICATION_DURATION_LONG,
                    message : 'Video captured; now share or save it!'
                }).show();

                var file = Ti.Filesystem.getFile(videoDir.resolve(), "video||" + currentFileName + ".mp4");
                file.write(videoUri);

            } else {
                Ti.UI.createNotification({
                    duration : Ti.UI.NOTIFICATION_DURATION_LONG,
                    message : 'Canceled/Error? Result code: ' + e.resultCode
                }).show();
            }
        }
    });

I have gone through the code in https://gist.github.com/dawsontoth/832488 but this doesn't work all the time. It works randomly. The saving/sending button appears at random time. The code I provided above records the video all the time but the only thing is it saves in its own folder. How can I change it ?

Thanks… looking for some response.
