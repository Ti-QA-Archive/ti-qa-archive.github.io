Video Thumbnails On Android

I'm trying to create an image thumbnails (using imageView) for a video that was either selected from the Video gallery or captured on an ANDROID DEVICE.
As far as i encountered so far , this can be easily done on Iphone using Titanium.Media.VideoPlayer.thumbnailImageAtTime-method.
I have thought about using intents and found the following function on the media store but dont know exactly how to use it MediaStore.Video.Thumbnails.html
and found the following code 

int id = **"The Video's ID"**
ImageView iv = (ImageView ) convertView.findViewById(R.id.imagePreview);
ContentResolver crThumb = getContentResolver();
BitmapFactory.Options options=new BitmapFactory.Options();
options.inSampleSize = 1;
Bitmap curThumb = MediaStore.Video.Thumbnails.getThumbnail(crThumb, id, MediaStore.Video.Thumbnails.MICRO_KIND, options);
iv.setImageBitmap(curThumb);
