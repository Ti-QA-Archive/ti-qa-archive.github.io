Access self-signed certificate https in WebView

Hi, I am trying to access a self-signed certificate url using WebView. The code is like this:

var webView = Titanium.UI.createWebView({
    width:'100%',
    height:430,
    top:80,
    url:'https://myurl'
});

Unlike others, I do not get any errors. Instead, the statements inside a 'load' event could be printed, as shown below:

webView.addEventListener('load',function(e){
    Ti.API.info('++++++++++++++++Page is loaded');
});

It means the page is loaded (or not?). However, there is nothing shown on the webview. It is totally blank. However, I could open the url using the device browser. So it is not something wrong with the web page itself.

Could anyone help me?
Thanks a lot!
