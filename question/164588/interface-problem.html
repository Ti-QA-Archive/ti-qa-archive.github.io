  • Accepted Answer

    on iOS7, the toolbar now consists of 20dp of the status bar (where you have your cell status, time etc.) and the toolbar itself (44dp).

    Depending on how you build the window structure, you can add the native iOS toolbar by applying Ti.UI.iOS.Toolbar to a window.

    Please look at the following API doc: http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.iOS.Toolbar
