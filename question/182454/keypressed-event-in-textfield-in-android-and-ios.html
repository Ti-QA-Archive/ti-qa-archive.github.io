Keypressed event in textfield in android and ios

Application type:mobile,Device:emulator and simulator,os:Mac OSX 10.9.5,titanium sdk:3.5.1,titanium studio:3.4.1

I am trying to change the amount entered in the text field as a number format of Currency.

I have found String.formatDecimal to change the amount.

I want to change this when the user enters the value in the textfield I have used `onKeyPressed' event but it not working as expected.

index.xml

<Alloy>
    <Window backgroundColor="white" id="win">
        <TextField id="text" onKeypressed="applyCommaFormat"></TextField>
    </Window>
</Alloy>

index.tss

'#text':{
    hintText:'Amount',
    width:'200dp',
    height:'60dp',
    top:'300dp',
    backgroundColor:'black',
    color:'white',
    textAlign:Titanium.UI.TEXT_ALIGNMENT_RIGHT,
    focusable:true
}

index.js

function applyCommaFormat(e) {
    $.text.value = String.formatDecimal($.text.value, 'en-US', '##,##,000.00');
}

  • Try to this,

    index.xml

    
<Alloy>
    <Window backgroundColor="white" id="win">
        <TextField id="text" onBlur="applyCommaFormat"></TextField>
    </Window>
</Alloy>

    index.tss

    
'#text':{
    hintText:'Amount',
    width:'200dp',
    height:'60dp',
    top:'100dp',
    backgroundColor:'black',
    color:'white',
    textAlign:Titanium.UI.TEXT_ALIGNMENT_RIGHT,
    focusable:true
}

    index.js

    
function currencyFormat (num) {
    return "$" + num.toFixed(2).replace(/(\d)(?=(\d{3})+(?!\d))/g, "$1,");
}

function applyCommaFormat(e) {
      e.cancelBubble = true;
    $.text.value = currencyFormat(parseFloat(e.value));
}

$.win.open();

    Thanks,

    Narendra Raorane
