    Assuming your images are in the /Resources folder, you need to 

     image: '/locker.png'

    for it to work on Android as well as iOS

    • @Cristian: maybe you've found this by now, but for the reference of other Alloy devs ending up here.
      If you are developing in Alloy, leave the Resources folder alone. It is automatically generated by Alloy based on the files you have inside the app/ directory. If something is to be placed within Resources/ in Titanium, the equivalent location for things like images in Alloy is app/assets/. If you see Titanium tutorials mentioning paths, just replace the "Resources/" with "app/assets/".
