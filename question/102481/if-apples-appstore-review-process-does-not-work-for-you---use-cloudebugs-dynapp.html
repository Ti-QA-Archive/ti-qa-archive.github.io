If Apple's Appstore review process does not work for you - use cloudebug's DYNAPP

If it is not acceptable for you to wait days (sometimes - weeks!) until your app's update is being reviewed by the Apple's Appstore - cloudebug's DYNAPP is for you (www.cloudebug.com).

With DYNAPP you can create native Titanium apps which, just like web pages, can be updated at any time (even if they are already published and available in the Appstore), and still are true native Titanium apps which can use full Titanium SDK!

See more details in this post:

http://developer.appcelerator.com/question/102101/create-dynamic-titanium-apps—under-your-full-control-even-after-published-to-the-app-store
