iPhone Retina: platformWidth & platformHeight return 320x480

Hello,
I test my application with iPhone Retine but I always get 320x480, how can I fix that ?

  • The logical coordinates on the display are the same whether you are working with a retina display or not, so Titanium.Platform.displayCaps.platformHeight and .platformWidth will return the same numbers in either case. If you need to determine whether the physical resolution is doubled or not, you can inspect the Titanium.Platform.displayCaps.density or .dpi property.
