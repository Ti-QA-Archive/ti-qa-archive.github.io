  • Had the same problem after updating my Titanium to 3.3.0
    The solution of deleting the config.json file worked for me but I couldn't locate the file via finder so after digging up info online I came up with this:

    Exit Titanium Studio if open

    1. In terminal (under your username folder) type: cd .titanium/

    2. Check that the config file exist using: ls -l

    3. Rename (instead of deleting) config.json: mv config.json conf.json

    4. Confirm rename: ls -l

    5. Restart Titanium.

    That worked for me.
