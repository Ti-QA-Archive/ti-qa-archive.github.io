How to customize positions for all android screen

I am using android
When i test my project in a different screen, i got a different view for controls positions like (top, left, width,… etc).

how can i get a value to be suitable for all screen
can i put a value in percentage like (top:20%) ?

    All your screen measurements can take 2 types of values, a number (ie:20), or a string (ie: '20px', '20%'). Numbers are taken as pixels and transposed as such. Your entire layout can be percentage based if you wish. Hint: set min/max values where available.
