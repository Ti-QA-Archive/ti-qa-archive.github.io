    Hello Bunty,

    Thanks for your inquiry, You need to create a custom title bar. for that you need to hide the Android action bar.

    To do that, please follow the link here. There are three ways to hide the action bar. Try following the number two. Now if you finished doing it you will see your app will run with full screen with no action bar. Now you need to create a custom title bar like the below code.

    var win2 = Ti.UI.createWindow({
    backgroundImage : '1.jpeg',
    navBarHidden : true
});

var titlebar = Ti.UI.createView({
    backgroundImage : '2.jpeg',
    top : '0%',
    height : '6%',
    width : '100%',
});

var searchbutton = Ti.UI.createButton({
    image : '3.jpeg',
    backgroundColor : '#CE1420',
    height : 'auto',
    Width : 'auto',
    top : '0%',
    right : '0%',
});

var aLabel = Ti.UI.createLabel({
    text : '????? ?? ?????? ',
    color : '#FFCC66',
    font : {

        fontSize : 30,
        fontWeight : 'bold',
        fontFamily : 'Siyamrupali'
    },
    height : 'auto',
    width : 'auto',
    top : '0%',
});

var imagelogo = Ti.UI.createImageView({
    image : '4.jpeg',
    top : '0%',
    height : 'auto',
    width : 'auto',
    left: "0%"
});

titlebar.add(imagelogo);
titlebar.add(aLabel);
titlebar.add(searchbutton);

searchbutton.addEventListener('click', function(e) {

Ti.API.info('Button clicked');

});
win2.add(titlebar);
win2.open();

    Hope this helps. Thanks.

    The simplest way to do that, is to get rid of the standard title bar put (android:theme="@android:style/Theme.NoTitleBar" in the <activity> element in the manifest) and put your own "title bar" at the top of the activity.

    • Ok, follow below steps.

      1.Run your titanium app for android.
      2.Go to yourapp/build/android folder and copy the android manifest file.
      3.Go to yourapp/platform/android folder and paste copied file there.
      4.open that file and look for the main activity and change android:theme value to "@android:style/Theme.NoTitleBar".
      5.use property navBarHidden as true for every window.
      6.create a view and add a button on that view,add view to window like a navigation bar.this view will work as a custom title bar.
