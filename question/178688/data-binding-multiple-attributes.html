Data Binding multiple attributes

I am trying to display multiple model data values in a single view label as described here.

However if I put anything other than the model attribute in in the text string i.e.
<Label text="Hole {holeNum}"/>

I get this fatal error during compilation:
[ERROR] Application Installer abnormal process termination. Process exit value was 1

Separating them into two labels though works fine i.e.
<Label text="Hole"/>
<Label text="{holeNum}"/>

It seems anything other than a single model value within the quotes wont compile.

  • You can create custom attributes in the trasnformFunction() in the controller, i.e.

    function transformFunction(model) {
    var transform = model.toJSON();
    transform.holeLabel = "Hole" + transform.holeNum;
    return transform;
}

    and use as <Label text="{holeLabel}"/> in the view.

    Check the guide for more details. Hth.
