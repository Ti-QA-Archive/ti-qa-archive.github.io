  • This is not possible with the sms: URL Scheme. The way it works is that you specify the phone number(s) you want to send to. Once you call the URL, the SMS dialog will appear with the number pre filled.

    You cannot send an SMS automatically with just URL schemes. Your user will have to do the interaction.

    Now if you want to send SMS without the user interaction, you can use one of the available Module from the Marketplace.
