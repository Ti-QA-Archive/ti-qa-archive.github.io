    Hi

    This is a simple fix, you have not triggered support for Retina Tall for your app.

    Make sure in the iphone folder you have the following files (match the initial caps) with the associated resolutions.

    • Standard, Default.png = 320x480
    • Retina Default@2x.png = 640x960
    • Retina Tall Default-568h@2x.png = 640x1136 this one is almost certainly missing

    Also make sure you are running a newer SDK ideally the very latest 3.1.2 but at least 3.0.
