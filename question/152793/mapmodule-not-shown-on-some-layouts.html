MapModule not shown on some layouts

I'm using Titanium SDK 3.1.0GA with Ti.map module (2.1) on Android.

I have several issues with the maps on Android, with API v2. The issue is only reproducible on a few of devices (such a Samsung Galaxy Ace Duos, a Motorola Delfy and a Samsung Galaxy S3 mini).

For example, with this layout:

<Alloy>
    <View class="container">
        <View id="mapview" ns="Alloy.Globals.Map" animate="true" regionFit="true" userLocation="true" />
    </View>
</Alloy>

The map is not shown on a Samsung Galaxy Ace, however it works perfectly on high-end phones (such as Motorola Atrix HD, HTC One X, etc).

Yet, on the Samsung Galaxy Ace, it works perfectly on the following layout:

// .... <Alloy> etc etc

<View width="90%" left="5%" height="200dp">
    <View id="mapview" ns="Alloy.Globals.Map" animate="true" regionFit="true" userLocation="true" />
    <View width="100%" height="100%" touchEnabled="false" opacity="0" backgroundColor="#ffffff"></View>
</View>

// ...

Thanks in advance!
