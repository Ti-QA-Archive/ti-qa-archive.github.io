Find out IP assigned to iphone

How can we find out what IP is being assigned to an iphone when its connected on wifi or 3 G network using titanium?

  • Accepted Answer

    var myIP = Titanium.Platform.address;

//The system's WIFI IP address. No other network types are supported.

    This will give you the local (LAN) IP address. If you are looking for your public IP, the only way is to retrieve it from the outside world, i.e. using a service like http://www.whatismyip.com (or even MoanMyIP.com LOL). You can also set up your own script into an application server (such as PHP) like :

    <?php 
 $ip = getenv("REMOTE_ADDR") ; 
 Echo $ip; 
 ?>

    and grab it through an httpClient call.

    George
