Using YQL & xpath to scrap a website

Hi folks, first time trying this as I have no access to a websites database content. I want to use YQL and the xpath to scrape the website and export the specified part in JSON.

On the Yahoo YQL console I have the desired query working, but when i put it into Titanium and test the app in the iphone simulator it crashes the app with out running.

Desired code in YQL console 

Titanium.Yahoo.yql(select * from html where url='http://www.shinty.com/news/' and xpath='//div[@class='topStories']/ul/li',function(e){

    var data = eval('('+e.data+')');
    Ti.API.info(data);
});

Anyone know what I am doing wrong?

  • Hi Crhis, it´s problem of query string.
    try this:

    var query = ' select * from html where url="http://www.shinty.com/news/" and xpath="//div[@class=\'topStories\']/ul/li" ';

    and then replace query in Titanium.Yahoo.yql to this like:

    Titanium.Yahoo.yql(query,function(e){
    var data = eval('('+e.data+')');
    Ti.API.info(data);
}
