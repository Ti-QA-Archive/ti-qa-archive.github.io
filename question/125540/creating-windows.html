Creating Windows

I want to create windows so that I can use the back button on Android phones to return to a previous screen.

Each window shares similar elements in common like a header and footer on a site. The only part that is different between each window is the content that goes in the center body port of each window.

How can I create windows so that only the body portion of each window changes?

I noticed that when I create windows, I am recreating the header and footer which takes a while to build in the application.
