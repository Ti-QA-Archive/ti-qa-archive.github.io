  • Native objects are created at runtime, not compile time. The javascript is also interpreted at runtime. It's the Titanium framework that's written in the platform's native language.

    Here's the native iOS code for dealing with a Ti.UI.Button: https://github.com/appcelerator/titanium_mobile/blob/master/iphone/Classes/TiUIButton.m
