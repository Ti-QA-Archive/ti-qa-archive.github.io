tableView header and search

Hi, everyone. Is there a bug that headerView and search doesn't work together in tableView?

var loadInd = Ti.UI.createActivityIndicator({
    color: "white",
    style: Ti.UI.iPhone.ActivityIndicatorStyle.DARK,
    left: 20
});
var loadView = Ti.UI.createView({
   left:20,
   width:"auto",
   height:50
});
loadView.add(loadInd);
var loadRow = Ti.UI.createTableViewRow({
    width:"auto",
   height:"auto"
});
loadRow.add(loadView);
loadInd.show();
//searchView is a variable that contains search filed

   top: 0,
   bottom: toolbar.size.height,

Can they work together?
