How to set root level variables via reference

Hi there,

If I want to set a top level variable via reference, in a browser I would use…

window[var_name] = &#39;123&#39;

Can I do this in Appcelerator without any problems?

  • Hi Toby

    Yes - the same way.

    var window = {};  // start off with this bit so the object is ready for the parameters
window["worked"] = "Answer";
alert(window.worked);

    Nice and simple

  • Maybe I'm reading Toby's question differently, but I think he's talking about using window as a sort of global variable repository. In this case, while window[&quot;worked&quot;] and window.worked are functionally equivalent, there's no "magic" about window that makes these values globally available.

    In fact, the use of globals is discouraged in Titanium, and you should encapsulate these variables in CommonJS modules.
