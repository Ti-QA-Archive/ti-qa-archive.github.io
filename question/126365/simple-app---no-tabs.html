Simple App - No Tabs?

I am trying to learn how to make apps with titanium and I just can not figure out how to go about things without tabs. I have spent hours going through the Q&A, Kitchen sink, etc and just haven't figured it out.

So here is what I need help with. Can anyone post the code for a simple app that uses no tabs. I want to load the app have a background image with my logo, and three buttons under it. When you click one of the three buttons I need to open a new .js file in a new window and have a back button which takes you back to the original logo/button page.

I need the buttons to open new .js files since they are going to have a lot of code in them, one will pull random data from the db, one will have a table view, etc.

  • from Titanium.UI.iPhone.NavigationGroup

    var win = Titanium.UI.createWindow();
var win1 = Titanium.UI.createWindow({
    backgroundColor:"red",
    title:"Red Window"
});
var nav = Titanium.UI.iPhone.createNavigationGroup({
   window: win1
});
win.add(nav);
win.open();
var win2 = Titanium.UI.createWindow({
    backgroundColor:"blue",
    title:"Blue Window"
});
nav.open(win2,{animated:true});

  • Try this:

    app.js

    var mainWindow = Titanium.UI.createWindow({
        url:'../Screen1.js'
});
mainWindow.open();

    screen1.js

    follow below structure also..

    var Screen1= {
    ui:{
        view:null,
        create:function(){    

            var objView = Titanium.UI.createView({
                backgroundColor:'black',
                top:'0%',
                bottom:'0%',
                left:'0%',
                right:'0%'
            });

        //here you create your buttons.
        //under button events you create window with your required .js file i.e url:'screen2.js' 
        //if you dont want tab to the new widnow, then use this property, `navBarHidden:true,`
        //finally you add your controls to the `objview`    
            this.view = objView;
        },
        render:function(){    
            Titanium.UI.currentWindow.add(this.view);        
        }                
    },
    init:function(){    
        Screen1.ui.create();
        Screen1.ui.render();    
    }
};
Screen1.init();

    hope this helps to u…
