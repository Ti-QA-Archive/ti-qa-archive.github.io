  • For each object within the documentation are samples for creating the objects using Classic Titanium and Alloy.

    For example the documentation for a button is here and under the examples you will see classic:

    var button = Titanium.UI.createButton({
   title: 'Hello',
   top: 10,
   width: 100,
   height: 50
});
button.addEventListener('click',function(e)
{
   Titanium.API.info("You clicked the button");
});

    and Alloy:

    <Alloy>
    <Window id="win" backgroundColor="white">
        <!-- The title property can also be defined as node text. -->
        <Button id="button" onClick="doClick" title="Hello"
            top="10" width="100" height="50" />
    </Window>
</Alloy>

    So you need to look around the documentation!

    • Nice clear answer;

      I would recommend using the XML in conjunction with the TSS files (think style sheets for Titanium). This gives the greatest flexibility and means you can define a single style that will affect all versions of that thing (app.tss) but can also be altered inside the specific controller (specific.tss).

      Additionally events should be in the controller JS.

      index.xml

      <Alloy>
    <Window id="win">
        <!-- The title property can also be defined as node text. -->
        <Button id="button1" title="Button 1" />
        <Button id="button2" title="Button 2" />
        <Button id="button3" title="Button 3" />
        <Label id="lbl1" text="Label 1" />
        <Label id="lbl2" text="Label 2" />
    </Window>
</Alloy>

      app.tss

      "Window": {
    backgroundColor: "white"
}

      index.tss

      "Window": {
    layout: "vertical"
}
"Button": {
    top: 30,
    height: 50,
    width:100
}
"Label": {
    backgroundColor: "orange",
    top:10,
    height: 40,
    width:120
}
"#button3": {
    width: 200
}
"#lbl2": {
    backgroundColor: "green"
}

      index.js

      var buttonHandler = function (e) {
   alert("Pressed " + e.getApiName() + " id = " + e.getId());
});
$.button1.addEventListener('click', buttonHandler);
$.button2.addEventListener('click', buttonHandler);
$.button3.addEventListener('click', buttonHandler);
$.lbl1.addEventListener('click', buttonHandler);
$.lbl2.addEventListener('click', buttonHandler);
$.win.open();

      I have taken this much further than the example needed to be, as I said the one from Larrie was very clear. However this gives you an idea of the true separation. You may not understand why this is important until playing with Alloy for a while but when you do - everything becomes faster, easier, more reliable and easier code to read.
