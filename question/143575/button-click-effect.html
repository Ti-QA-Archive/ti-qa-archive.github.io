Button click effect

Hello!
I have this code:

var button = Titanium.UI.createButton({
        title : 'Close',
        width : 150,
        height : 40,
        backgroundImage : 'none',
        borderRadius : 10,
        font : {
            fontSize : 16,
            fontFamily : 'Optima',//_fontFamily,
            fontWeight : 'bold'
        },
        backgroundGradient : {
            type : 'linear',
            colors : ['#000001', '#666666'],
            startPoint : {
                x : 0,
                y : 0
            },
            endPoint : {
                x : 2,
                y : 50
            },
            backFillStart : false
        },
        borderWidth : 1,
        borderColor : '#666'
    });

Then I add an event listener:

button.addEventListener('click', function(e) {
        console.log('click');
});

The problem is that when I press the button there is no effect that indicates that the button is being pressed. What can I do?

Some suggestions??

Thanks in advance.

    Hi German,

    actually backgroundGradient is not supported by buttons on iOS.
    What is happening is that you are setting " backgroundImage : 'none', " which result in actually setting the backgroundImage of the app to 'transparent/invisible' , since 'none' it's not a valid value.
    At that point, backgroundGradient is being applied to the View below the Button (in fact if you comment out the " backgroundImage : 'none', " you will see no gradient are being applied to the button).

    So, what you have to do is to create an image for your button and apply it using the "image" or the " backgroundImage" properties of the button.
    In that way you are also going to see the effect of the button being pressed.

    var button = Titanium.UI.createButton({
        title : 'Close',
        width : 150,
        height : 40,
    // image: 'my_button_image.png',
        backgroundImage :'my_BackgroundI_Image.png',
        borderRadius : 10,
        font : {
            fontSize : 16,
            fontFamily : 'Optima',//_fontFamily,
            fontWeight : 'bold'
        },

        borderWidth : 1,
        borderColor : '#666'
    });

    Thanks,
    Federico
