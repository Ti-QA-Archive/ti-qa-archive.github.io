How to disable back swipe gesture on iOS 7

Hi.
In iOS 7 Apple add new default navigation behavior. You can swipe from left border of the screen to go back on navigation stack. But in my app, this behavior conflict with my custom left menu. So, is it possible to disable this new gesture?

— asked October 10th 2013 by Masao Suzumi

    Example code of Alberto's answer:

    var win = Ti.UI.currentWindow;

var backButton = Ti.UI.createButton({
    title:'Back'
});

win.setLeftNavButton(backButton);

backButton.addEventListener('click', function() {
    win.close();
    //Unless you're using NavigationWindow, then use:
    //NavigationWindow.closeWindow(win);
});
