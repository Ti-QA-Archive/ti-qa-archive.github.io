    • This is because the html you are loading is trying to fit best to the page. You probably want to wrap your webView content in a complete html document instead of 

      html : '<div><span  style = "color:#f0f;">' + tweet + '</span></div>',

      ie

      
<html>
<head>
</head>
<body>
<div><span  style = "color:#f0f;">Style your content here</span></div>
</body>
</html>

      and include some CSS to make the content align better
