Format date and time to the country specific style

Hi

I want to show the user the date and time of the latest update.

Is there already a function (maybe from the iOS) that formats the time?
Because for example in the USA you say 7pm, in Europe you say 19:00

Would great if someone knows a solution!

Thanks
Felix

    Hi.

    var mydate = Date();
mydate.setHours(19);
mydate.setMinutes(30);
String.formatTime(mydate);

    Returns - depending on your settings - 19:30 (Germany) or 7:30 PM (US)

    Take a look at the bottom of the Titanium-module API docs for more information.

    Cheers,
    Florian
