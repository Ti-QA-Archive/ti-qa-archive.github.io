  • Nermin

    Judging from your screenshot, I presume you are installing to OSX. Is that correct?

    The Titanium SDKs are installed to either &#x2F;Library&#x2F;Application Support&#x2F;Titanium or ~&#x2F;Library&#x2F;Application Support&#x2F;Titanium. Would you check to see that these directories exist, and that you have the appropriate permissions to write to them?

    It may be best to run Titanium Developer as administrator in order to install the updates. Is this possible?
