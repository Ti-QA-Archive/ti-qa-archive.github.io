Android Calendar Intent --> Help needed !

Hi, I am developing an app using Android Calendar Intent but facing problem to 'write' data into 'event_location'. I am able to write data into 'title' and 'description' but not 'event_location'.

My code is as follow:

var content_title = 'some title';
var content_description = 'some description';
var content_location = 'somewhere';

//== Android Intent !!    
var activity = Ti.Android.currentActivity;
var intent = Ti.Android.createIntent({
    action: Ti.Android.ACTION_INSERT,         
    type:'vnd.android.cursor.item/event'
    });

intent.putExtra('title', content_title);
intent.putExtra('description', content_description);
intent.putExtra('event_location', content_location);         

activity.startActivity(Ti.Android.createIntentChooser(intent,'Add Event Using:'));

Can anyone enlighten me on this matter? Am I doing something wrong in coding?

Thank you.

Note: I am using Android Device with SDK 4.4.2 and Ti.SDK 3.2.2
