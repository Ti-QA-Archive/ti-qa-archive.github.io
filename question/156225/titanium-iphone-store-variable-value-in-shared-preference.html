Titanium iPhone store variable value in shared preference

Hi
I am new on titanium and iPhone development, i want to know how to store value of variable in share preference and get that value on another js?

    Sajid,

    In Titanium you can use App properties store the data. It will work in both iOS and Android

    Store a property

    Ti.App.Properties.setString('userName', 'Paul');

    Fetch saved properties

    Ti.App.Properties.getString('userName');

    For more example refer this blog

    http://appcodingeasy.com/Titanium-Mobile/Storing-data-in-Titanium-application
