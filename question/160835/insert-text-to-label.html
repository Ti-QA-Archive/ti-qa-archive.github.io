Insert text to label?

Hello, I am trying to follow the "create your first titanium app tutorial (http://docs.appcelerator.com/titanium/latest/#!/guide/Creating_Your_First_Titanium_Apps).

Fist, I found an error on it:

At the step "Create a book detail view" we are asked to create a new Alloy Controller called bookdetails, but then in the JS function used to call it, the code shown is 

var bookview = Alloy.createController("bookview", args).getView();
bookview.open();

So obviously it wont work, i ve then deleted the bookdetails and created a bookview instead.

I'm using this exact function:

function showBook(event) {
    var selectedBook = event.source;
    var args = {
        title: selectedBook.title,
        author: selectedBook.author
    };
    alert(selectedBook.title);
    alert(selectedBook.author);
    var bookview = Alloy.createController("bookview", args).getView();
    bookview.open();
}

I reiceve the alert so the function is well launched, the alert contains the title and author.

But when it launch the bookview controller; it's just a blank screen, So i've tried to insert random text and I can see it. If i am right it means the variable are not well passed from index to bookview, do you know why is this happening? Is somewhere a better starter guide cause this one seems bugged.

Here is the bookview xml:

<Alloy> 
   <Window class="container"> 
    <View layout='vertical'> 
          <Label id="titleLabel">aaaa</Label>
          <Label id="authorLabel">aaaa</Label> 
    </View>
   </Window> 
</Alloy>

Also, sorry to ask so many question but I am running on a galaxy s3 and I don't have a navigation menu at top, there is just a little "test3" at the top that isn t clickable, and when I click the back button it close the app :/

Thank you very much!

    $.titleLabel.text = arguments[0].title;
$.authorLabel.text = arguments[0].author;

  • I found that indeed the author was not being passed in the args for the onclick event - the answer I discovered was to include the author in the index.xml file as follows:

    <Alloy>
    <Collection src="books"/>
    <Window class="container">
        <TableView dataCollection="books">
            <TableViewRow title="{title}" author="{author}" onClick="showBook"></TableViewRow>
        </TableView>
    </Window>
</Alloy>
