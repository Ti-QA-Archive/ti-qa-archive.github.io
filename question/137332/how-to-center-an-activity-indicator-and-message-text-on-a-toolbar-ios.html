how to center an Activity Indicator and message text on a toolbar (iOS)

seems to align left by default. textAlign:'centre' is not an available property and width:'auto' and width:TI.UI.SIZE have no effect. Any ideas?

  • There is a special button for iOS toolbars that works as an ActivityIndicator, here is an example of how can be used.

    Is documented here

    About message text, add a button without border.

    
        var _toolbar = Titanium.UI.iOS.createToolbar({left:0, right:0, height:40});

        var _btnMail = Ti.UI.createButton({
            style:Titanium.UI.iPhone.SystemButtonStyle.BORDERED,
            systemButton:Titanium.UI.iPhone.SystemButton.COMPOSE
        });

        var _btnSave = Ti.UI.createButton({
            systemButton:Titanium.UI.iPhone.SystemButton.SAVE,
            style:Titanium.UI.iPhone.SystemButtonStyle.BORDERED    
        });

        var _btnActivity = Ti.UI.createButton({
            systemButton:Titanium.UI.iPhone.SystemButton.ACTIVITY,
            style:Titanium.UI.iPhone.SystemButtonStyle.PLAIN
        });

        var flexSpace = Titanium.UI.createButton({
            systemButton:Titanium.UI.iPhone.SystemButton.FLEXIBLE_SPACE
        });


        //define custom functions to show/hide indicator on demand...
        _toolbar.showActivity = function(){
            _toolbar.items = [_btnSave,flexSpace,_btnActivity, flexSpace,_btnMail];
        };

        _toolbar.hideActivity = function(){        
            _toolbar.items = [_btnSave,flexSpace,_btnMail];
        }

        //initialize
        _toolbar.showActivity();
