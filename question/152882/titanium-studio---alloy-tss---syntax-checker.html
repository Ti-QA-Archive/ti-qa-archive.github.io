    Hi

    I think the problem is because you are using the L macro to provide a localised version of your string.

    Instead of using text with the L macro use the textid property and just the name of the localised string.

    ".title": {
    left: "5dp",
    textid:"'title"
}

    This does the thing you want but the way Titanium wants it.

    You will notice all text & title fields have an optional version with a suffix of id, use these instead. Never use both text AND textid (or the equivalents) as you can only provide one or the other.
