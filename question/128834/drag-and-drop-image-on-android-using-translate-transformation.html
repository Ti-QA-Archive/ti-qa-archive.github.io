Drag and Drop Image on android using translate transformation

Hi all,

I spent many hours trying to implement drag and drop on imageView. It works fine on IOS but still not work in android.

This is how I implement my drag and drop :

this.view = Ti.UI.createImageView({
        image : imageSrc,
        width : 400,
        height : 200
    });

    //object used to transform picture
    var olt = Titanium.UI.create2DMatrix();

    //touch position when move start
    var curX, curY;

    this.view.addEventListener('touchstart', function(e) {
        //record position on touch start
        curX = e.x;
        curY = e.y;
    });
    //perform drag and drop
    this.view.addEventListener("touchmove", function(e) {
        var deltaX = e.x - curX, deltaY = e.y - curY;
        olt = olt.translate(deltaX, deltaY);

        e.source.transform = olt;
        e.source.duration = 30;
        curX = e.x;
        curY = e.y;

I use translate transformation on my image view on touchMove. The problem on android is that the 'touchable' area remain at the initial image position.

For example, if i add an image on top left corner then translate it on bottom, the only way I have to drag it again is to touch and move my finger in the top left corner.

Is there something wrong in my code or any work around?
