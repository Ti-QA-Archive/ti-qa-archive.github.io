Feature request - 'Force Rebuild' button and key commands

Not sure if this is the proper place to make feature requests (feel free to delete if not), but I would love to see 2 features in Titanium Developer:

  1. A button to force delete and recompile the build folder (so rather than having to manually go move the files to the trash and delete them and then come back and build, just hit 'Force Rebuild' and it does it).

  2. Key commands for the 'Stop' and 'Launch' buttons. Being able to launch/relaunch via keyboard rather than having to physically click a button would be nice.

Great program, keep up the good work!

— asked March 26th 2011 by Bryan Wills

  • In theory, all you need to do to force a full rebuild is to switch to the Edit tab and click "Save Changes" even if you don't change any values like the version number. This will "touch" the tiapp.xml file and update its last modified timestamp. The build scripts sees this as needing a full rebuild.

    In practice, while that is generally good enough, I have seen rare instances where things still did not work right until I cleared out the build directory as well. But typically this is all you need to do.

    Regarding the Stop and Launch keyboard accelerators, I am hoping the forthcoming IDE integration with Aptana will make things easier.
