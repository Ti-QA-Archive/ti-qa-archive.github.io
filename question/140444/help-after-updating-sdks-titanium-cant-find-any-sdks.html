  • Accepted Answer

    The desktop SDK is no longer supported by Appcelerator.

    It has been spun off to an open source project TideSDK.

    This occurred quite a while ago in the grand scheme of Net time with the last Appcelerator desktop SDK being released over 9 months ago.
