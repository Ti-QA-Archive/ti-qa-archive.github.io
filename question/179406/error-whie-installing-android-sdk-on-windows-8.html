  • Hello,

    Please follow these steps to set up the android sdk for titanium studio.

    1. Get the android sdk.After downloading it extract it to your comfortable location.
    2. Inside the adt bundle you will find sdk manager, double click to run it, and install the desired android api level.
    3. After installing the desired API level, now click on the tools and select manage AVD…. in the sdk manager. Now Android Virtual Devise (AVD) mamager will show up. Now click the Devise Definition and choose the devise you need and click Create avd.
    4. Now open your Titanium Studio , window > preferences. The preferences window will open with some collapsible thing. On your left hand side of the preferences window you will see studio click on that, now you will see platforms and choose android. Once you select android from the left hand side of the preferences window, the right hand side window will automatically get updated with the details. Now you need to enter the android sdk's path (which you have configured in step 3). So click browse and navigate to that path. The default emulator will automatically updated.
    5. Click Apply, Ok.

    Thanks
