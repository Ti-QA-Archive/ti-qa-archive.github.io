    • The Alloy framework is a way of creating your app in a structured, highly re-usable way with the benefit of code, presentation and logic being separated.

      JSON is a way to organise data into an object that can be created by a web server as text and read by your app using httpClient where you convert it back to an actual object. JSON can also be used within your app as an object for many different reasons.

      You do not have to worry about all of that though, check out the sample app in the documentation to see how to handle the client side of things. The app explains how to read and react to the data.
