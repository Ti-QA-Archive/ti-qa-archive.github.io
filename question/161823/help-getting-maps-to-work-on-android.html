Help Getting Maps to Work on Android

Hello,

I been trying to make maps work on Android without any luck. I think I am doing all the steps right, but it looks like I am not because I just see a grey box with the controls and everything but no map on in. My app works fine on iOS but trying to port to android is proving more complicated than what I thought.

My code for the map view is the demo one, like:

    var mapview = Map.createView({
     mapType: Map.NORMAL_TYPE,
    region: {latitude:33.74511, longitude:-84.38993,
            latitudeDelta:0.01, longitudeDelta:0.01},
    animate:true,
    regionFit:true,
    userLocation:true,
    });

Where map is Map = require(&#39;ti.map&#39;);

And this is the code in my tiapp.xml

<android xmlns:android="http://schemas.android.com/apk/res/android">
    <manifest>
            <!-- Allows the API to download data from Google Map servers -->
            <uses-permission android:name="android.permission.INTERNET"/>

            <uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE"/>

            <uses-permission android:name="android.permission.ACCESS_FINE_LOCATION"/>

            <!-- Specify OpenGL ES 2.0 as a requirement -->
            <uses-feature android:glEsVersion="0x00020000" android:required="true"/>
            <!-- Replace com.domain.appid with your application ID -->
            <uses-permission android:name="com.myapp.app.permission.MAPS_RECEIVE"/>
            <permission android:name="com.myapp.app.permission.MAPS_RECEIVE"
                android:protectionLevel="signature"/>
            <application>
                <!-- Replace "PASTE YOUR GOOGLE MAPS API KEY HERE" with the Google API key you obtained -->

    </android>

Is there anything else I am missing to do? Its strange that maps seem to be loading, at least the controls,google logo and box are, but then there is no map on it.

I am really stuck on this so any tip will be super really appreciated! :-D

Thanks!

  • Accepted Answer

    I think the best way to make sure, if you are still in development, is to create a new SHA1 and add your application to the allowed list in the Google API console, then generate a new API key and use that in tiapp.xml.

    Make sure you are testing on device, as maps won't display on the emulator.

  • If you are using ios you can use ti.map but if you are using android you need only the old instruction.

    In your tiapp.xml

     <modules>
        <module platform="iphone">ti.map</module>
    </modules>

    in your index.js wirte:

    Alloy.Globals.Map = require(&#39;ti.map&#39;);

    in your view:

    <View id="viewMapa" ns="Alloy.Globals.Map" animate="true" regionFit="true" userLocation="true"  width="100%" platform="ios" bottom='14.7%' top="0"  top="0"></View>


    <View id="viewMapa" ns="Ti.Map" animate="true" regionFit="true" userLocation="true"  width="100%" platform="android"bottom='14.7%' top="0" mapType="Titanium.Map.STANDARD_TYPE" top="0"></View>

    in the xml you can use the attribut platform.
