backgroundRepeat: true causes black background in Android

Hi,

I have the following code and in iOS everything works as intended but in Android I get a black solid color background when backgroundRepeat is set to true. The image is there (and stretched) when backgroundRepeat is set to false. I am using density folders, so my actual image path is /android/images/high/bg-main.png I also tried giving the view a height and a width (and removing contentHeight and contentWidth altogether).

This is SDK 3.0.0 GA but I tried it in earlier versions of the SDK and I get the same result.

    var scroll = Ti.UI.createScrollView({
        backgroundImage:'/images/bg-main.png',
        backgroundRepeat: true,
        layout: 'vertical',
        contentWidth: 'auto',
        contentHeight: 'auto',
        showVerticalScrollIndicator: true,
        showHorizontalScrollIndicator: false,
        top: zwh.os({
            iphone: uSize(44),
            ipod: uSize(44),
            android: uSize(88),
        })
    });
    self.add(scroll);
