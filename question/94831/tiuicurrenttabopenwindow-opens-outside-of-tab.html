Ti.UI.currentTab.open(window) opens outside of tab

I have an application with a basic tabgroup - 5 tabs to be exact - Inside those tabs i have the need to sometimes open a subwindow and still stay on the tab or at least still have the tabbar showing.
So whenever i need to open a window from a tab i use Ti.UI.currentTab.open(window) - This opens the window just fine but it opens in fullscreen and not inside the tab so i loose the ability to switch to other tabs - setting fullscreen: false on the window doesn't help either.
How do i make the window open inside the tab?

    the trick is to set the window's width and height properly so it fits, yet leaves title bar and tab bar visible.

    in the code below, I set the width and height based on the window that is currently visible in the tab. I have created a button that when clicked, creates the new window, adds it to the tab and then opens the window

    button1.addEventListener('click', function(e) {
    var childWin = Titanium.UI.createWindow({  
        title:'Tab 1 - Child Window',
        backgroundColor:'#fff',
        width: win1.width,
        height: win1.height
    });    
    tab1.add(childWin);
    childWin.open();
});
