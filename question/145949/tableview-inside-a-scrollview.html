TableView inside a ScrollView

I'm trying to write a chat window, similar to WhatsApp's chat.

For that, I initially wanted to put a bug ScrollView, and in it two elements: a TableView for all the chat bubbles (90% height for example), and a view for the TextField/SendButton (10% height for example).

However, from what I've read, it's not a good idea to put a TableView (inherently scrolls) inside a ScrollView.

Can anyone give me a hint of how chat windows are made?

  • Hi

    No need to use a table for what you want, not just because a tableView is not a good idea in a scrollView.

    If you add the property layout: 'vertical' to the scrollView you will get what you need with much more control.

    Let me know if you want a quick example.
