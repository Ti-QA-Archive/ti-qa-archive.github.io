reload and repaint not refreshing webview in simulator

Hi I have a web view, which I wish to change the url of and refresh, so:

var webview = Titanium.UI.createWebView({url:'http://www.google.com'});    

// i call the following on a button press

webview.url='http://www.twitter.com';
webview.reload();
webview.repaint();

yet the webview will not change to the new url.

I've skimmed loads and loads of pages but have not found anything to work.

Can anybody help please?

I'm running this in ios simulator as iphone 4.3 and using 1.7.2 sdk

Thanks in advance!

  • Hi Brett, there is no need to call reload() and repaint() after assign an url. Remove these lines and it should work.

    • In fact, if you assign an url and immediatelly you call reload(), webView will reload the prior url, since the new one has not been loaded yet.

      And about repaint(), AFAIK, may be used only when the component is resized, so in this case is useless.
