Problem with long text inside a label on iOS

Hi,
I'm building an App that retrieves a text from an internal database and shows it inside a label.
When the text is long, here comes the problem!
The text is truncated with ellipsis, apparently without reason, because the height of the label is set to 'auto' and the properties like wordwrap or ellipsize on iOS are not working.

Here is my code:

var scrollabletext = Ti.UI.createScrollView({
    showVerticalScrollIndicator:true,
    scrollType:'vertical',
    contentHeight:'auto',
    width:300,
    height:364,
    left:10,
    top:58
});
win.add(scrollabletext);

var textview = Ti.UI.createView({
    width:280,
    left:10,
    right:10,
    top:5
});

var textlabel=Ti.UI.createLabel({
    text:rows.fieldByName(win.text_to_open),
    font:{fontSize:15,fontFamily:"Gotham"},
    shadowColor:'#aaa',
    shadowOffset:{x:0,y:1},
    height:'auto',
    color:'#000'
});

textview.height=textlabel.toImage().height;
textview.add(textlabel);
textlabel.show();
scrollabletext.add(textview);

I'm working on the latest Titanium SDK (2.0.1.GA2) and iOS 5.0 SDK.

Thank you very much!

  • Ok, thanks to the tip given by Trevor, I changed my code into this:

    var scrollabletext = Ti.UI.createTextArea({
    value:rows.fieldByName(win.text_to_open),
    font:{fontSize:15,fontFamily:"Gotham"},
    color:'#000',
    shadowColor:'#aaa',
    shadowOffset:{x:0,y:1},
    backgroundColor:'#dcdcdc',
    editable:false,
    width:300,
    height:364,
    left:10,
    top:58
});
win.add(scrollabletext);
