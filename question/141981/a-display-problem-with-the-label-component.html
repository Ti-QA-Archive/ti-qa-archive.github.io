A display problem with the label component

Hello everybody,

I tested this example about implementing the datePicker component:

var myWindow=Titanium.UI.createWindow({

    title:"Test Date Picker",
    backgroundColor:"#15F",
    exitOnClose:true,

});


var results = Titanium.UI.createLabel({

    title:"Date Picker display",
    width:"auto",
    height:"35",
    top:10,
    color:"#FFFFE0",
    borderColor : 'black',
    borderWidth : 2

});


var minDate = new Date();
    minDate.setFullYear(1984);
    minDate.setMonth(0);
    minDate.setDate(01);

var maxDate = new Date();
    maxDate.setFullYear(2060);
    maxDate.setMonth(11);
    maxDate.setDate(31);

var setValue = new Date();
    setValue.setFullYear(2012);
    setValue.setMonth(08);
    setValue.setDate(07);

var myPicker = Titanium.UI.createPicker({

    selectionIndicator:true,
    type: Titanium.UI.PICKER_TYPE_DATE,
    minDate:minDate,
    maxDate:maxDate,
    value:setValue

});

myWindow.add(results);
myWindow.add(myPicker);

myWindow.open();

the only problem is that the label component doesn't showen in the simulator, even with the border !!!

Thanks in advance!

  • Accepted Answer

    That's because you put "Date Picker display" into the label's title property, not the text property. Easy mistake to make, since other views in the API have a title property (like the Button, for example). But the label gets its text from the text property. I've made the same mistake myself.
