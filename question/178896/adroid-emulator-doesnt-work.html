Adroid Emulator doesn't work

Hello guys,
I am having a problem when I try to install my app on Android emulator, I have just downloaded titanium studio and set up android SDK I have created a hello world app but when I run my app on android I got this error:
[ERROR] Application Installer abnormal process termination. Process exit value was 1
[ERROR] : Application failed to install

Sometimes happens timeout while titanium waits for Emulator be ready

  • Hi Jefferson Cardoso,

    Thanks for your concern.

    You can also follow a few steps for your problem.

    • First of all you can install genymotion, a virtual emulator for android, it resolves process exit value was 1 problem and try to use GA version of Ti SDK, it's stable.

    • Launch your android emulator first from AVD manager, wait for your emulator to become ready then run your project. Clean your project before you.

    • Please check which version of JDK you are using now because 32-bit version of the JDK is required regardless of whether titanium is running on a 32-bit or 64-bit windows system. Check the link installing oracle JDK for details.

    • Check your environment variable path configuration, clean your project and run again.

    • If anything doesn't work, then just uninstall node.js, titanium studio and delete the supported files (.titanium, .android, .eclipse) from C –> User –>Your PC name and titanium studio folder from C–> User –>Your PC name –>appData. After all re-install, update node and CLI completely and set the android SDK path then restart PC and run again.

    Hope this resolve your problem.
