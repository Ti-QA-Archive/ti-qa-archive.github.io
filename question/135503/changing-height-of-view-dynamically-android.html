Changing Height of view dynamically Android

Hi, I am developing android application. In my application there is one outer view with height 'auto'.I am displaying 3 view inside this outer view but one at a time mean view1.show() and view2.hide(), view3.hide(). Height of all 3 inside views not fixed. On first load It displaying view1 and on click of button2 and button3 it displaying view2 and view3 respectively.
But my problem is that on first load height of outer view adjust according to inner view1.When i click on button2 it prepare view2 and display in outer view. This time my outer view considering height of view1.Not adjusting height according to height of view2
my code looks like:

    var user_todos_view1 = Ti.UI.createView(
    {
        clickName:'user_todos_view',

        left:0,
        right:0,
        height:'auto',
        borderColor:'transparent',
        borderWidth:1,
        borderRadius:3,
        backgroundColor:'transparent'
    });scrollView.add(user_todos_view1);

    var view1 = Ti.UI.createView(
    {
        top:0,

        borderWidth:1,
        borderRadius:3,

    var view2 = Ti.UI.createView(
    {
        top:0,
        right:0,
        left:0,
        borderColor:'#aaa',
        layout:'vertical',
        height:'auto',
        borderWidth:1,
        borderRadius:3,
        backgroundColor:'#fff'
    });user_todos_view1.add(view2);

    var view3 = Ti.UI.createView(
    {
        top:0,
        right:0,
        left:0,
        borderColor:'#aaa',
        layout:'vertical',
        height:'auto',
        borderWidth:1,
        borderRadius:3,
        backgroundColor:'#fff'
    });user_todos_view1.add(view3);

// on first load display view1
//on button2 click show view2 and hide view1 and view3
//on button3 click show view3 and hide view1 and view2

I want to adjust height of user_todos_view1 according to view displayed inside it like view1 ,view2 or view3. But it always talking height of view1.It mean it not releasing height of view1 after hiding it.Is there any way to handle this situation.Thank you.
