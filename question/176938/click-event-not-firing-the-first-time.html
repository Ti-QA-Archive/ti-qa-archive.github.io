Click event not firing the first time

I am using CLI version 3.3.0, Titanium SDK version 3.3.0.GA, Titanium Studio version 3.3, and OSX 10.9. I have some textFields that have onClick events that are not firing when I click on them. Then, after the textField has focus, the click event can fire. I have one disabled textfield that fires the click event the first time I click on it without problem. This issue is primarily on Android, but occasionally presents itself on iOS. It appears that the first click is giving the textField focus instead of firing the click event. Is that how it is supposed to work, or is could there be something else that is causing the issue? Some of the event listeners are added in the view and others are added in the controller, and both react the same way.
