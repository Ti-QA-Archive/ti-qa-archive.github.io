require a module inside a commonjs module

I've discovered that we can create commonjs modules instead of native or web one (take a look at your titanium folder and search the ti.cloud module). There is no documentation at all that explains it…

So, I try to add my own commonjs module inside the titanium folder "commonjs" and import it to my application using "require". All runs like a charm.

Now, I want to add some other files to my commonjs module and make files referencing each other inside the module.

For example in my module I have :

  • main.js
  • ui
    • CommonView.js

in this structure main.js is my main module and have a method createCommonView() that tries to require the module CommonView (require('./ui/CommonView')) but the compiler complains that the module is not found.

How to create a commonjs module with multiple files and referencing each other ?
I precise that I don't want to copy these modules inside my project but keep them in the titanium folder to make it available to all projects.
