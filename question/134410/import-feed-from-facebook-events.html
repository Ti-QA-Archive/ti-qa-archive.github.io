  • You have to get the list of eventIds from the page then you can pass them through that API to get the details. To get all your events you do something like

    graph.facebook.com&#x2F;[user_id]&#x2F;events?fields=id,owner&amp;limit=1000 then loop though them;
    graph.facebook.com&#x2F;?ids=[event_ids for the user&#39;s events
