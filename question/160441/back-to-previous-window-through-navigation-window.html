Back to previous window through navigation window

In my app, i've a login window at first. Then after login, there is a Home window. From home window there is a plenty of child window. Each child window may have another child window and so on. In every child window, there are two buttons as 'Home' button and 'Logout' button. if i click 'Home' button then the app will be back to the Home window and if i click 'Logout' button then the app will back to the login window. i tried few ways but that always take me to the login window. i can't go to the Home window ever by clicking 'Logout' button.

In login window.js :

Ti.App.addEventListener('backtoLogin', function() {
        _navigation.close(homeWindow, {
            animated : true
        });
    });

and in homeWindow.js:

Ti.App.addEventListener('backtoHome', function() {
        _navigation.close(childWindow, {
            animated : true
        });
    });

In child window i tried like -

btnHome.addEventListener('click', function(e) {
        Ti.App.fireEvent('backtoHome');// Back to the root/login window
    });

btnLogout.addEventListener('click', function(e) {
        Ti.App.fireEvent('backtoLogin');// it also back to the root/login window
    });

i tried like avobe but does not work for back to home window but always back to the login window for both button EventListener. can anyone tell me what should i do now ?

      // window id is win3
var winStack = Ti.App.Properties.getList("winStack");

winStack.push(win3);
Ti.App.Properties.setList("winStack", winStack);
