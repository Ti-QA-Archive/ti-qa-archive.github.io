movieControlStyle versus mediaControlStyle & movieControlMode for iPhone, iPad

Mobile API docs at developer.appcelerator.com/apidoc/mobile/latest/Titanium.Media.VideoPlayer-object.html, say that movieControlMode is deprecated and that mediaControlStyle should be used. The use of mediaControlStyle produces NO results. "movieControlStyle" is mentioned in the docs' Titanium.Media section under properties (see "VIDEO_CONTROL_EMBEDDED" for example). The use of the property movieControlStyle does work for a videoObject created with Titanium.Media.createVideoPlayer.

Apple's iOS API docs for MPMoviePlayerController Class also show MPMovieControlStyle as an enum constant property.

— asked February 12th 2011 by Andy Felong
