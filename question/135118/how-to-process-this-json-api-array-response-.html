How to process this JSON API array response ?

Been working with some JSON APIs, and ran into one API that looks to be placing each object into an array, but with random numbers. How do I access them in a loop. I'm trying to apply them back to a tableview.

{"0":{"Available_Date":"Apr 20 2012 12:00:00:000AM",
"Start_Time":850,
"End_Time":1300,
"Sched_Loc_Abbr":"GFO     ",
"Sched_Dept_Abbr":"IM      ",
"Resource_Desc":"Tim Smith",
"Resource_Abbr":"SMA     ",
"Activity_Type_Abbr":"XZ      ",
"Usual_Appt_Duration":10,
"Booking_Factor":1,
"Scheduling_Location_ID":2,
"Scheduling_Department_ID":1,
"Resource_ID":21,
"Location":"DC Office",
"start_timestamp":1334926200,
"end_timestamp":1334927400},

"7":{"Available_Date":"Apr 20 2012 12:00:00:000AM",

I was handling the there results with:

for (var c=0;c<data.length;c++)....
....
text:    data[c].Resource_Desc
....

At this point I just can not figure how to access the objects as I can't even get a length on it currently or anything other than a manual hardcoded data[7].Available_Date, any ideas?

    This is not an array, is an object. The fact that the keys are set as numbers instead of a string (actually is a string but has a number instead of letters) doesn't make it an array.
    You can use the for…in loop

    for (var key in data) {
    Ti.API.info(data[key].Resource_Desc);

}
