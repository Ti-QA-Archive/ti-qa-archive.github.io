  • I think you are missing a separator in the path:

    var filename = Ti.Filesystem.applicationDataDirectory +  Ti.Filesystem.separator + vfilename;

    you can also call getFile with two arguments: the path and the filename

    var bgImage = Titanium.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory, vfilename);

    I always suggest to check if file exists, so any related problem is easier to debug

    var bgImage = Titanium.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory, vfilename);

if(bgImage.exists()){
    videoPlayer.media = bgImage;
}else{
    alert('file does not exists');
}

    var videoFile = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory,'Video_file_name');
