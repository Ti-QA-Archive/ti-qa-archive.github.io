[Android] CSS file doesn't work anymore with WebView and 3.0.2.GA

Hey,

I've just upgraded to 3.0.2GA and tested my app. I noticed that the webview doesn't apply the style of a loaded css file anymore. Anyone having similar issues?

I have a WebView like this:

    var noteTextView = Ti.UI.createWebView({
        top : '94 dp',
        width : Ti.UI.FILL,
        html : me.prepareNoteText(),
        backgroundColor : 'white',
        color : 'black',
        left : '0 dp',
        right : '0 dp',
        bottom : '40 dp',
        enableZoomControls : false,
    });

where me.prepareNoteText() is generated HTML containing a reference to a local css file.

With 3.0.0GA everything works fine, with 3.0.2GA the style is not applied, whole text is black.
