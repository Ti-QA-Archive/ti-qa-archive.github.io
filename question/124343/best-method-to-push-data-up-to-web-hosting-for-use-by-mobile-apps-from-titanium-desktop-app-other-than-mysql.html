Best method to push data up to web hosting for use by mobile apps from Titanium Desktop App other than mySQL

Basically the Titanium Desktop app will monitor local hardware and upload data to the web server where a mobile app can interact with the data. My first thought was mySQL but my host requires you add each IP that will be connection to mySQL. With dynamic IPs and new users that won't be efficient. Are there any other lighter methods I could use to push the data up?

    YOu could use something like PHP or Node.js on the server side to interact with the DB and just expose an API to your app. This would remove that need for each IP to register, and provide a lot of extra security.
