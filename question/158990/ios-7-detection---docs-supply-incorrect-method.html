iOS 7 detection - docs supply incorrect method?

Hi i need to detect whether the iOS version is 7 or greater and change the top value of a window if it is however the example shown in the docs gives me an errors message: "Can't find variable: isIOS7Plus at iOS.js (line 82)"

I got the code from here: http://docs.appcelerator.com/titanium/latest/#!/guide/iOS_7_Migration_Guide

This is what i used:

// Function to test if device is iOS 7 or later
function isiOS7Plus()
{
    // iOS-specific test
    if (Titanium.Platform.name == 'iPhone OS')
    {
        var version = Titanium.Platform.version.split(".");
        var major = parseInt(version[0],10);

        // Can only test this support on a 3.2+ device
        if (major >= 7)
        {
            return true;
        }
    }
    return false;
}

var iOS7 = isIOS7Plus();  // THIS IS THE ERROR LINE. LINE 82
var theTop = iOS7 ? 20 : 0;

Please help.
