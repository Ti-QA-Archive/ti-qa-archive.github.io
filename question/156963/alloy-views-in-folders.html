[Alloy] Views in folders?

Hey guys,

is it possible to create folders inside the "view" folder (created by alloy) to structure the app?

Let's say we have 100 View files - it would be easier/nice to sort/structure them in folders inside the main view folder.

BUT how does the reference-mechanic work? is this even possible? Couldn't find anything related in the docs… ?

    Yes, just put the folders in there. There's nothing to it.

    If you had the view "app/views/items/item.xml", you would reference it in XML as:

    <Require src="items/item"/>

    or in a controller as:

    var controller = Alloy.createController('items/item');
