  • I have found the solution in this link

    purchaseState : The purchase state of the order. Possible values are 0 (purchased), 1 (canceled), or 2 (refunded).

    this value return from PURCHASE_STATE_CHANGE you need to check:
    if(response.orders[0].purchaseState == 0){ &#x2F;&#x2F;purchase code }

    Hope that help…
