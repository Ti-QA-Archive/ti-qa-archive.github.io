Matching multiple values for a single column in alloy

Hi All,

I am getting problem in fetching data from database, I am using alloy collection for storing data locally.
Now my problem is that, I have one column named "flag" in database. Now in this column, there are four kinds of flag : 0,1,2,3. Now I have to fetch flag value of 1 and 2 in the column. how can I fetch this value?

inboxLocal.fetch()
    var flag1Records = inboxLocal.where({
        flag : 1
    })

By this, I am able to fetch single flag value records.

How can I fetch for 1 and 2 both in a single shot.

My image link is : https://dl.dropboxusercontent.com/u/72782351/1.png

Thanks in advance

    Hi Sunny,

    Try following code.

    var detail = inboxLocal.filter(function(obj) {
       return obj.get("flag") === 2 || obj.get("flag") === 3 ;
   });

    Use filter command to get multiple values from single column.

    Let me know, if it works.
