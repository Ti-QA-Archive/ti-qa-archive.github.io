User Location own pin

Hi,

I want when i detect my own location , a pin is placed on my current position in map, without using 

userLocation:true;

it is possible?

Thanks in advance

    yes you can do that, you need to do is get the user current location coordiantes (latitude, longitude) and create annotation and push annotation on mapView:

    var anno=[];

anno.push(Titanium.Map.createAnnotation({
    latitude  : latitude,
    longitude : longitude,
     image     :   'img/map_point_h@2x.png',
   }))

       $.mapview.annotations = anno;
