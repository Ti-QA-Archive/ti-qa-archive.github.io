    Alessio

    I build a Windows 7 Professional 64 bit machine with Titanium Developer and android SDK yesterday, and it's working without an issue.

    Have you followed the advice in Preparing for Android development? If so, I am surprised that you even attempted to use 64 bit JDK.

    Try deleting all your AVDs, as the guide suggests.

    Like you, I also see the "Error loading preferences" message, but it does not cause any problems in my case.

    Incidentally, remember that the current KitchenSink won't necessarily work with old versions of the Ti SDK.
