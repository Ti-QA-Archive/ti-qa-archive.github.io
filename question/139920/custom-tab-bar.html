Custom Tab bar

I am currently using the customtabbar.js that is on github and although it works perfectly, for some reason the windows for the tabs are opening fullscreen instead of at the height I want, making things appear behind the tabbar - does anyone know why?

Titanium.UI.setBackgroundColor('#fff');

var tabGroup = Titanium.UI.createTabGroup();

var win1 = Titanium.UI.createWindow({  
    title:'Window 1',
    height:430,
    tabBarHidden:true,
    fullscreen:false,
       url:'/main_windows/window1.js',
});
var tab1 = Titanium.UI.createTab({  
    window:win1
});

var win2 = Titanium.UI.createWindow({  
    title:'Window 2',
    height: 430,
    tabBarHidden:true,
    fullscreen:false,
    url:'/main_windows/window2.js',
});
var tab2 = Titanium.UI.createTab({  
    window:win2
});

tabGroup.addTab(tab1);  
tabGroup.addTab(tab2);  

tabGroup.open();

Ti.include("/customTabBar/customTabBar.js");

var ctb = new CustomTabBar({
    tabBar:tabGroup,
    imagePath: '/images/',
    width: 160,
    height: 50,
    items: [
        { image: 'win1.png', selected: 'win1_over.png' },
        { image: 'win2.png', selected: 'win2_over.png' },
    ]
});

  • There seem to be a number of customTabBar.js files on github, so you may want to contact the author of the version you are using. However, looking at 2 of them that are pretty similar, it looks as though the tab bar works by overlaying a window at the bottom of the app window, so items in the window may end up behind the tab bar. This however does not offer a fix as I'd have to actually see to what extent things appear behind the tab bar in order to come up with something more to offer. Regardless, the author of the script may know the fix immediately.
