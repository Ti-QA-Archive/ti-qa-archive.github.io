    Brandon, looks like we are only seeing info/error. I'd like to see trace, as it will tell you what occured in the build right before the error pops up.

    Open up a terminal, run ti setup. You can do the quick setup and change the default logging level to trace.

    I'm concerned about this [ERROR] Error parsing XML file.. Kind of want to know what file is failing.

    Also, keep in mind, the 1.4.0 alloy is less gracious about errors in your xml and tss.
