Get push notification data if app is not in foreground - Android

Hey Folks,
I am busy implementing push notifications in my application.
Registering/Subscribing/Sending push notifications is not a problem.

Also catching the notifications if the app is in the foreground is not a problem.

But when the application is not in the foreground, I cannot retrieve the notification data when the user clicks on the tray notification. I tried to do it by getting the launcher intent from Ti.App.Android.launchIntent.

But this is never the intent from the notification tray.
It's always:

Anyone tried this before? I know there is this module: GCM I am yellow, But I don't want to rely on third party modules too much.
