    The problem is you have a bad module. The following log output is the clue:

    [DEBUG] Detected module for None: None None @ /Users/mdavidlow/Library/Application Support/Titanium/modules/compression/mobile/ios/test/ZipIt

    For starters, the module is not in the correct path format. It should be:

    /Users/mdavidlow/Library/Application Support/Titanium/modules/<platform>/<module name>/<version>/

    Where platform is either "ios", "iphone", "android", "mobileweb", "commonjs", or "blackbery".

    There should be a file called manifest in the module's directory and it needs to contain a number of required properties including "name" and "guid".

    You can manually fix it, then build your app again.
