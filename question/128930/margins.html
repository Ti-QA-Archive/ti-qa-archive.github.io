  • Hi David.

    Although Titanium does not have in-built mechanism of defining margins, they can be simulated by defining position properties of particular element. Check this simple example:

    var win = Ti.UI.createWindow({
    backgroundColor: '#000'
});

var mainView = Ti.UI.createView({
    height: '100%',
    width: '100%',
    backgroundColor: '#f00',
    layout: 'vertical'
});

var view1 = Ti.UI.createView({
    width: '100%',
    height: '10%',
    top: '5%',
    backgroundColor: '#f90'
});

var view2 = Ti.UI.createView({
    width: '100%',
    height: '10%',
    top: '5%',
    backgroundColor: '#09f'
});

win.add(mainView);
mainView.add(view1);
mainView.add(view2);

win.open();

    Here, I simulated top-margin by defining "top" property of "view1" and "view2". Note that "mainView" has layout property set to "vertical". Similar functionality can be achieved for "horizontal" layout.

    This works for Views only, ScrollView for example does not function in same way. I reported that so you can watch the ticket if you like to support it's resolution: http://jira.appcelerator.org/browse/TC-413

    This is for Android only, don't know do iOS function in a same way, but I suppose it does.
