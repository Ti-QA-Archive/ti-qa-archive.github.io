  • Accepted Answer

    My guess is that you don't have to include the old icons, but I would still do it, to provide the best possible quality to pre-iOS7 users. For my latest universal app i have included the following icons:

    • appicon.png (57 x 57)
    • appicon@2x.png (114 x 114)
    • appicon-60.png (60 x 60)
    • appicon-60@2x.png (120 x 120)
    • appicon-72.png (72 x 72)
    • appicon-72@2x.png (144 x 144)
    • appicon-76.png (76 x 76)
    • appicon-76@2x.png (152 x 152)
    • appicon-Small-40.png (40 x 40)
    • appicon-Small-40@2x.png (80 x 80)
