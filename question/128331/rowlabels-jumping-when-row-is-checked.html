  • San

    Be aware that the font* properties are not available for TableViewRow, so you have to add a label to display text on one. This is a mistake in the documentation that is addressed in ticket TIMOB-3061. Although it's been outstanding for a long time, I'm happy to say that, as we speak, we have someone working to resolve these types of APIDoc issues.

    Are you using the checkbox style of switch, which may be useful?

    Are you using the className property on rows, which will prevent it being re-rendered in some cases, and thus can vastly improve performance?

    Cheers

      I needed to see a script that included the changes that I had proposed. I certainly hadn't seen one that used the className, or had the font properties deleted from the row view.

      Note that your script did not include the window's creation. Admittedly it's a trivial thing, and easy to fix, but by providing the script that you are running makes it more likely that we can reproduce the issue.

      I have run the following with Android 2.2 and Titanium 1.7.5, but it seems to run perfectly. What are you developing with? Please remember to include all of this kind of information when posting a question, as explained in the Q&A Question Checklist.

      var win = Ti.UI.createWindow({
  backgroundColor:'#ddd',
  fullscreen:false,
  exitOnClose:true
});
var all = ["Hello", "Hi", "Bye", "MoreStuff"];

function getTableData(){
  var data = [];
  for(var i=0, ilen=all.length; i<ilen; i++)
  {
    var label = Ti.UI.createLabel({
      height:'auto',
      width:280,
      color:'#454545',
      left:20,
      font:{fontSize:15, fontWeight:"bold"}
    });

    var row = Ti.UI.createTableViewRow({
      className:'rows',
      color:'454545',
      height: 65
    });
    label.text = all[i];
    row.add(label);

    data.push(row);
  }

  return data;
}

var tableview = Ti.UI.createTableView({
  data:getTableData(),
  top:0
});

tableview.addEventListener('click', function(e){
  if(e.row.hasCheck){
    e.row.hasCheck = false;
  } else {
    e.row.hasCheck = true;
  }
});

win.add(tableview);
win.open();

      See this video for a demo.

      Let me know if this resolves it for you.

      var win = Ti.UI.createWindow({
      backgroundColor:'#ddd',
      fullscreen:false,
      exitOnClose:true
    });
    var all = ["Hello", "Hi", "Bye", "MoreStuff"];

function getTableData(){
    var data = [];
    for(var i=0, ilen=all.length; i<ilen; i++){
        var label = Ti.UI.createLabel({
            color:'#454545',
            text: all[i],
            left:20,
            font:{fontSize:15, fontWeight:"bold"},
            width:280
        });
        var row = Ti.UI.createTableViewRow({
            className:'rows',
            height: 65
        });
        row.add(label);
        data.push(row);
    }
    return data;
}

var tableview = Ti.UI.createTableView({
    data:getTableData(),
    top:0
});

tableview.addEventListener('click', function(e){
    if(e.row.hasCheck){
        e.row.hasCheck = false;
    } else {
        e.row.hasCheck = true;
    }
});

win.add(tableview);
win.open();
