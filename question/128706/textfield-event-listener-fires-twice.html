textField event listener fires twice

I have a search text field with returKeyType as search. When I press the search button on keypad the eventlistener fires twice and gives me duplicate output.

var searchInput = Titanium.UI.createTextField({
    top:'2dp',
    left:'0dp',
    right:'97dp',
    hintText:'Search items',
    font:{fontSize:'14dp',fontWeight:'normal'},
    borderStyle:Titanium.UI.INPUT_BORDERSTYLE_ROUNDED
});

searchInput.returnKeyType = Titanium.UI.RETURNKEY_SEARCH;

The eventlistener code is as follows

searchInput.addEventListener('return', function(e){
    alert('ok');
});

Here the "ok" message is displayed twice. Any suggestion?

Thanks

  • I tried ur code n the solution is not to specify a returnKey becozz when u specify a returnKey , there will b two keyboard on the screen on the focus of text field one is of android's default and another is ur returnKeyType soft keyboard.That's why ur event is firing two times.I got the solution is:

    var searchInput = Titanium.UI.createTextField({
    top:'2dp',
    left:'0dp',
    right:'97dp',
    hintText:'Search items',
    font:{fontSize:'14dp',fontWeight:'normal'},
    borderStyle:Titanium.UI.INPUT_BORDERSTYLE_ROUNDED
});

//searchInput.returnKeyType = Titanium.UI.RETURNKEY_SEARCH;

searchInput.addEventListener('return', function(e){
    alert('ok');
});
