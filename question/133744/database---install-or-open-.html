    If you know the sqlite file exists, use .open()

    If unsure, use .install() as it will install the passed db file by copying it to the devices internal storage and then opening it. If a file already exists with the same name, it opens it.

    In most cases, you will use .install()

    If, lets say, you download a sqlite file from the net to the SD card, you could just open it by using Ti.filesystem to reference the file and then passing that to .open()
