  • Accepted Answer

    Hello Ankita,

    Check this code. It is working in all resolution. Use same height and same width for imageView and for borderRadius i used ( width * 1.5) only for android. Check below code :

    Try this :

    var testWin = Titanium.UI.createWindow({
    backgroundColor : '#ccc'
});

var imageView = Titanium.UI.createImageView({
    image : 'baby.jpeg',
    width : 250,
    height : 250,
    backgroundColor : 'red',
    borderColor : 'red',
    borderRadius : 375,
    borderWidth : 2,  //(250 * 1.5)
});

testWin.add(imageView);
testWin.open();

    i hope this helps. thanks.
