Add three control horizontally

I want to add three control horizontally in single row side by side. Android support linear layout . Is there any linear layout in titanium .

  • Accepted Answer

    var view = Ti.UI.createView({
    height: '10%',
    layout: 'horizontal'
});

view.add(button1);
view.add(button2);
view.add(button3);

    use of layout property will arrange the child of row or view to the right of the last one.

  • There is "layout" property which can be set to "horizontal".

    var view = Ti.UI.createView({
    height: '10%',
    layout: 'horizontal'
});

view.add(button1);
view.add(button2);
view.add(button3);
