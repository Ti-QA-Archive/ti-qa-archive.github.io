Maps on Android

I've seen a lot of questions about Google maps API keys on here, but I've noticed that maps work fine both on my emulator and on my Droid X (in development mode) without adding any key whatsoever. I'm using mobile SDK 1.5.1 and Android APIs 2.2. When I submit my app to the Android Market, will the maps stop working?

— asked January 19th 2011 by John Williams

    Yes you must add your own personal API key before submitting to the market. In the 'Emulator' and 'Run on Device' modes of Titanium, you are using their API key so it is working. You won't be using theirs for a distributable version, so you must get your own before deploying to the market.
