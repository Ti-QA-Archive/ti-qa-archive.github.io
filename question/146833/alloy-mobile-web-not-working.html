  • Are you opening the generated index.html file directly in the browser (with the file:// protocol)?

    If so, there's a bug with loading resources from local disk in Titanium Mobile Web. You'll have to set up a web server such as nginx or Apache. Titanium Studio does have a built-in web server which also works, but it can't proxy requests or stream video.

  • Well, it seems i've solved my problem just adjusting zOrder in the index file just before open it.

    
var c = Alloy.createController('cView');
c.setDelegate($);

$.a.setZIndex(1);
$.b.setZIndex(2);
c.getView().setZIndex(3);

function showC (evt) {
    c.getView().open();
}

$.index.open();
