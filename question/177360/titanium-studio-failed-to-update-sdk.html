  • Accepted Answer

    Hello, kholid, Thanks for your query. Seems you have a connection error. Please try to follow the solutions below.

    1 From the command Prompt use

    ti sdk install

    2.You may be behind a proxy, Go to Titanium Studio> Preference> General > Network Connections. In Proxy Entries there should be 

    HTTP, HTTPS, SOCKS

    . in Proxy bypass there is,

    localhost

    add your 

    ip

    in there. add 

    127.0.0.1

    if not there.

    Click OK, Then try to update again.

    1. If you have nodeJS installed properly nodeJS.com and are SURE that you have your proxy setup, or are not behind a proxy, then you can try to bypass the studio updater and install manually:
      ~~~
      [sudo] npm install -g titanium

    [sudo] npm install -g alloy

    [sudo] npm install -g acs

    ~~~

    Try these solutions, one of them should work.

    Thanks.
