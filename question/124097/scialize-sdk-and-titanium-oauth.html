  • I tried, what is implemented, seems like it works. I could create entities. However, the socialize.js uses http for logging in, and it should be https, so when I tried With FB authentication, it did not work. The problem was in the strange fork, that in case of https doesnt create the oauth headers. Look for https in lib/socialize.js, it's hard to miss.
