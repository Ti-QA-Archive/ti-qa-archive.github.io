How To set multiple RequestHeader for a service call

Hi All,

I want to know how to set multiple request header for a API call.
for single header we can do like this,

xhr.setRequestHeader("Content-Type", "application/json; charset=utf-8");

what if I have to add custom data to the header

For 3 custom data,can it be like this ? 

xhr.setRequestHeader("key1": "value1");
xhr.setRequestHeader("key2": "value2");
xhr.setRequestHeader("key3": "value3");

or we have to put the key value pair in an array and then pass it.. ?

  • You can also code:

    var headers = [{k1:v1},{k2:v2} ...];
for (var k in keaders) {
    xhr.setRequestHeader(k : headers[k]);
}

    Hint: a good idea is to track the traffic between smartphone and net, because the http client modified headers.
