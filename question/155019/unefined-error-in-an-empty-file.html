    You are using binding notation in bookRow.xml but you haven't given its container (book1Table in book1.xml) a dataCollection. You can't use binding notation without a dataCollection. Something like this:

    book1.xml

    <Alloy>
    <Collection src="Book" />
    <Tab id="book1Tab" title="Book1">
        <Window id="book1Win">
            <View>
                <TableView id="book1Table" dataCollection="Book">
                    <Require src="bookRow"/>
                </TableView>
            </View>
        </Window>
    </Tab>
</Alloy>

    Of course, this will only show rows if your Book collection has data in it.
