How do you specify the url of a TableViewRow onClick event??

I have a tableView, with multiple tableViewRows, as follows:

var row = Titanium.UI.createTableViewRow({
width:"auto",
height:42,
backgroundColor:"white",
hasChild:true
});

On Click, each tableViewRow should open a particular url [pages/viewRow.js]. How do I set the url of these tableViewRows, as I have tried adding 'url:"pages/viewRow.js"' in the createTableViewRow definition [above], but nothing happens on click.

Cheers guys….

    hi Donal,

    Try this code,

    var row = Titanium.UI.createTableViewRow({
    width : "auto",
    height : 42,
    backgroundColor : "white",
    hasChild : true,
    url:"pages/viewRow.js"
});
tableView.addEventListener('click', function(e) {
    var winRow = Ti.UI.createWindow({
        url : e.row.url,//if ios else e.rowData.url
    });
    Ti.UI.currentTab.open(winRow);
});
