'Could not find the file app.js'

    Hi

    Whilst I do not know why the Kitchen Sink app is not working for you, but I can guess why you are receiving the error you are currently seeing.

    This problem occurred a few minor versions back, it only occurred on apps compiled for devices though - you have not said specifically if you are a simulator or physical device.

    Either way, previously the problem occurred due to errors in the code that failed to minify during the compile process. This is very possible with KS as the app has been built over years and has been updated often so could easily have unforeseen errors.

    The errors that killed code before were often trailing commas in the object definition - the last property entry had a comma afterwards which is very wrong.

    Ti SDK 3.1.2 does have preliminary support for iOS7 so you should be able to start your initial testing, but obviously only a limited set of things will work as iOS7, the rest will work as before.

    It would be worth checking jira to see if the problem has been logged, if not log the error and it will be added to the queue.
