  • Ok, finally found an answer on these forums that actually worked. It's in one of the comments here:

    The jist is fixing a line in the build script. Should be in 

    /Library/Application Support/Titanium/mobilesdk/osx/[version]/iphone/builder.py:

    Change

    if target_build_dir!=build_dir:

    to

    if target_build_dir.lower() != build_dir.lower():

    Hope this helps somebody else. And uh, Appcelerator team… what's up?
