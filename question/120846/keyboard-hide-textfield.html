keyboard hide textfield

I have a windows with a view and a textfield in the view (near the bottom). When the textfield receives focus, the keyboard pops up and goes over top of the textfield.

I can't see the text in the textfield I'm editing. Why is this?

    you can solve this problem by placing all of your views items in a scrollView. When this is done, the framework will scroll the view the appropriate amount to ensure the text field is visible when it receives focus

  • I found the best solution for that problem:

    The first this:

    textName.addEventListener(&#39;focus&#39;, function() { configWindow.animate({bottom: &#39;30%&#39;, duration:500}); });

    And the end this:

    textName.addEventListener(&#39;blur&#39;, function() { configWindow.animate({bottom: 0, duration:500}); });

    This do that the windows compress and expand when you hide the keyboard
