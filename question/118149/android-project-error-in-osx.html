  • Asterism

    Try the following:

    • close your emulator
    • close Titanium Developer
    • start the Android AVD and package manager, and delete all existing AVDs
    • delete your project build files (see instructions below)
    • start Titanium Developer
    • select your project in the left panel
    • on the Edit project screen, select Titanium SDK 1.6.1
    • on the Run Emulator screen, ensure that SDK "APIs 2.2" is selected

    To delete your build files and force a full rebuild:

    Caution, though; due to an Android SDK bug, currently the process for Android and iOS is different. Hence:

    For iOS/iphone, delete the contents of the project's build&#x2F;iphone directory, select Edit Project in Titanium Developer and click the Save Changes button to force a full rebuild. Then launch the app.

    For android, only delete the contents of the immediate subdirectories of &#x2F;build&#x2F;android, otherwise you may encounter problems. Then click the Save Changes button to force a full rebuild and launch the app, as above.

    Let us know how you get on.

    Cheers
