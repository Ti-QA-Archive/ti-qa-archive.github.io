  • I started having this problem after upgrading to the current beta SDK (iOS 5). Following the comments I downloaded again the SDK 4.0.2 and installed it to no avail. The problem was solved only after running the following command:

    sudo <Xcode>/Library/uninstall-devtools --mode=all

    restarting the system and re-installing the latest production version of SDK.
