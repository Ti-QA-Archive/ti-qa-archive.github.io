XCODE SDK 4.3 no longer showing in Titanium Run Config menu

Hi,

I was happy coding away, and building my app with XCODE 4.3.

I would go to my Run Config menu, and select Device - Iphone, iOS SDK : 4.3.

The choices I had were 4.3, 5.0 and 5.1.

I tried to build my project again, and I got an error message in the Ti compiler saying "Unknown SDK 4.3".

I restarted the computer, and now the only SDK showing is 5.1

What happened to 4.3 and 5.0?

How can I get them back?

I know there's a way to get them back without re-installing the SDK, but I forgot. Do you know?

Thanks

Ps. I'm running release 2.0.2.201205311912 of titanium.

    In Xcode (v4.3), go to Xcode > Preferences, select Downloads, Components. Next to iOS 5.0 Simulator and iOS 4.3 Simulator, click the install button. After that is complete, close and reopen Studio.
