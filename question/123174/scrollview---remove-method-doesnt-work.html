ScrollView - "remove" method doesn't work

Hi all.

My environment is: Android 2.2 (Samsung Galaxy Tab), Titanium SDK 1.7.1 (06/21/11 14:28 1293a6d), Windows XP

I'm trying to remove component added to the "ScrollView" component, but it seems it doesn't work.

Here is simple example that demonstrates the issue:

var win = Ti.UI.createWindow({
    modal: true,
    backgroundColor: '#000',
    layout: 'vertical' 
});

var view = Ti.UI.createScrollView({
    layout: 'horizontal',
    height: 'auto'
});

var label = Ti.UI.createLabel({ text: 'Test' });

var tf = Ti.UI.createTextField({ width: 300 });

view.add(label);
view.add(tf);

win.add(view);

win.addEventListener('open', function() {
    setTimeout(function() { view.remove(label) }, 3000);
});

win.open();

Label should be removed 3 seconds after window opens, but if you run example you'll see that nothing happens.

Change "ScrollView" with "View" and label will be removed.

Can anyone confirm this is a bug?
