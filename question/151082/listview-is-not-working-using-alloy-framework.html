ListView is not working using Alloy framework

  • Accepted Answer

    Full markup integration with ListView won't be supported in Alloy until 1.2.0. In the meantime you can specify the <ListView> in the markup, but the rest of it needs to be composed in the controller code. I am actually working on this functionality right now.
