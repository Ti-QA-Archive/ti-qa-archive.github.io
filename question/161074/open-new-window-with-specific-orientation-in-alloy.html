  • I would add something to the global config object that will allow you to selectively set the orientation. I haven't used this specifically, but I think it'll work. In your index.js add

    Alloy.CFG = {
  "leftLandscape": Ti.UI.LANDSCAPE_LEFT
};

    Then in the module that uses it, set the value something like this:

    var myScreen = Alloy.CFG.leftLandscape;

    That may not be the exact syntax, but the idea is to use Alloy.CFG.leftLandscape as the assigned value.

    Hope that helps.

    Regards.
