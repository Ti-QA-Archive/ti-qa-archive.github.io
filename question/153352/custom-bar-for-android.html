    Hi

    This is not the difficult if you think of it as a number of views sandwiched together.

    • outer view = contains all the inner views, use a background image to simulate the black line
    • gradient image view = fixed width image already created with the gradient effect add to the outer view
    • Set the left side of the gradient image to a minus value equal to the width and set touchEnabled to false.
    • Add a touchmove event to the outerview and move the gradient based on this

    You can play with the process by adding an overlay mask PNG where the inside is transparent, this could provide the drop shadow effects and the non-standard wedge shape.
