Change backgroundImage on Button

I am developing for iOS. I have a Button that just uses backgroundImage. It is a play/pause button so I need to toggle between the two.

function playbackStateChange(event)
{
    if (event.playbackState == 1)
    {
        Ti.API.warn('Playing');
        playPauseBtn.backgroundImage = '../images/pause.png';
    }
    else
    {
        Ti.API.warn('Paused');
        playPauseBtn.backgroundImage = '../images/play.png';        
    }
}
vidPlayer.addEventListener('playbackState',playbackStateChange);

The above code isn't changing the image though. Any idea why?

  • Alright what I ended up doing was making the play button and pause button separate buttons. Rather than toggling the backgroundImage property on one button, which wasn't working, I changed the items property on the Toolbar. Code below.

    function playbackStateChange(event)
{
    if (event.playbackState == 1)
    {
        Ti.API.warn('Playing');
        controls.items = [flexSpace,backBtn,flexSpace,pauseBtn,flexSpace,slideListBtn,flexSpace,zoomBtn,flexSpace,linkBtn,flexSpace];
    }
    else
    {
        Ti.API.warn('Pause');        
        controls.items = [flexSpace,backBtn,flexSpace,playBtn,flexSpace,slideListBtn,flexSpace,zoomBtn,flexSpace,linkBtn,flexSpace];        
    }
}
vidPlayer.addEventListener('playbackState',playbackStateChange);

    controls is my Toolbar.
