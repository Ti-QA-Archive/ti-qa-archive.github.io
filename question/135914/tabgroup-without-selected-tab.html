  • You can simply overlap the default start-up tab with a view. Then on focus event of the TabGroup you would remove that view.

    But I suggest you Not to do this, everybody wants to Start fast and such an implementation would slow that down.

  • hi michal,

    it is not possible to display tabgroup with no selected tab.

    but you can customize it.

    follow the steps.

    • create one image exactly like tabGroup but without selected any tabs.
    • create one separate window and add this image in that and set this window on the tabs.
    • then on click event of that image, close that window and set the tab index as clicked on the image tab
