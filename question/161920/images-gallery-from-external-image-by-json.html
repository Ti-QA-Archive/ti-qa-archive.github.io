    Hi

    To retrieve your JSON object you would use Titanium.Network.HTTPClient, examples are given in the documentation.

    Next create a function which can accept a URL string and create an ImageView. ImageView can accept a remote URL string as seen here Titanium.UI.ImageView-property-image

    Now you loop through your JSON object and grab your image nodes one at a time, passing each one into the function you created, allowing you to create a new image from each node.

    You now have your ImageView's to do with what you want, an example would be to add each ImageView into an array and then pass that array to a ScrollableView.

    Hope this helps!
