  • I think I might be able to start on this. .net is an integrated framework which combines the backend database, code and front end within itself, using and MVC architecture.

    To use titanium you would need to use it to create the front end either Web / Desktop or mobile application, talking via http requests or sockets to the api interface of the backend.

    Depending on how the .net application has been written depends on how you can directly interface into it. But simply if you can make a http request to the backend code, then titanium will provide a framework for you to develop a new front end.

    But as Alan has asked a bit more detail of the exact requirements would enable a more direct answer to your specific needs.

    Hope this helps

    T.
