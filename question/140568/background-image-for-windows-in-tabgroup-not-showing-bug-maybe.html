Background image for windows in tabgroup, not showing. Bug maybe?

When i try this code:

});

var tabGroup = Ti.UI.createTabGroup({
    backgroundImage: '/images/background.jpg'

Is there any reason for that? When i add the backgroundImage code to the windows that are in the tabgroup, then the background images display just fine. It just doesn't seem to be working when i place that line in the tabgroup self.

Maybe a bug, or is this intended?

  • Hi David

    I agree with Alexander, I can see no issues with the backgroundImage property. I have tried 2.1.1 (latest public) and 1.8.2 (in case something was different previously).

    Try this code in a brand new project, replace all the app.js contents with this code.

    This has been fully tested on iOS

    Titanium.UI.setBackgroundColor('#000');

var win1 = Titanium.UI.createWindow({  
    title: 'Tab 1'
});
var tab1 = Titanium.UI.createTab({  
    icon: 'KS_nav_views.png',
    title: 'Tab 1',
    window: win1
});

var tabGroup = Ti.UI.createTabGroup({
    backgroundImage: 'Default.png'
});    

tabGroup.addTab(tab1);  
tabGroup.open();

    This will use the existing image found the iPhone folder - you do not need to move or supply anything other than this code.
