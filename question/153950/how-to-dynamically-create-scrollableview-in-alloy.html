How to dynamically create scrollableview in alloy

Hi,

using Ti Studio 3.1.1, developing app for iphone. I am having problem creating scrollableview dynamically.

here is my xml

<Alloy>
    <Window id="winOverlay"></Window>
</Alloy>

my attempt in js

var scrollView = Titanium.UI.createScrollableView({
        views: []
});

function createCustomLabel(){
    for (i=0; i<4; i++) {

        var customView = Ti.UI.createView();

        var customLabelA = Ti.UI.createLabel({
            text: "I'm top label",              
        });
        var customLabelB = Ti.UI.createLabel({
            text: "I'm bottom label",              
        });

         customView.add(customLabelA);
         customView.add(customLabelB);

         scrollView.add(customView);
    }
    $.winOverlay.add(scrollView);
}
createCustomLabel();

    Hi

    If I am right then you almost had this, one small change.

    scrollView.add(customView);
// becomes
scrollView.addView(customView);

    As you are adding a page to the scrollable view which is a distinct command from adding a normal child.
