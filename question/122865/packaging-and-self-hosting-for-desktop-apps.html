Packaging and self-hosting for desktop apps

Deploying desktop apps in Appcelerator uploads the apps to the cloud and hosts them there.

I was curious if there was a way to package and distribute apps internally on LAN servers. I'd like to package the application for Mac and PC, upload them to a server in house, then have users deploy the applications from the in house server rather than the web. Is this possible?
