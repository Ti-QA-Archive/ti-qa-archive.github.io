addEventListener based on class rather than id?

Question seems simple enough but I've had trouble locating any kind of examples.

I have several different elements, in the same window, that I want to assign the same event listener to. Can I do it by class? If not by class, what other possibilities are there?

So, the xml markup would be something like this:

   <Label class="listen_to_me">Im some label</Label>
   <Button class="listen_to_me">Im Some button</Button>
   <ImageView class="listen_to_me"> // im some image view
   </ImageView>

And the listener code would be something like this?


$.listen_to_me.addEventListener('click', function(e) {

alert('i hear ya!!');
});

    <Label onClick="clickEvent">Im some label</Label>
   <Button onClick="clickEvent">Im Some button</Button>
   <ImageView onClick="clickEvent"> // im some image view
   </ImageView>

    function clickEvent(e){
    alert('clicked me');
}
