Cannot "Select SDK Version" other than 4.3 for on device testing

Hi there,
just discovering the possibilities offered by Appcelerator! Nice and exciting :)

I did purchase an a developer account (iTunes connect) last night and I'm trying to test on my device (iPhone 3G iOS 4.2.1).

I did successfully test on my device an app made only with Xcode but when trying to do the same with Titanium I only have the "4.3" option in the "Select SDK Version".

I have Titanium Studio, build: 1.0.4.201108101535
and Xcode 4.1 4B110
on Lion

I've read the following Q&A: http://developer.appcelerator.com/question/45481/use-iphone-sdk4-to-develop-for-3x-devices

I changed the deployment target but when trying to test on device from Xcode I get :
Xcode cannot run using the selected device.
This executable doesn't support iPhone, so it can't be used with “iPhone…”. Connect an iPad or choose an iOS simulator to use.

On Titanium Studio's side, I'm still stuck with 4.3 "only choice"…

Thank you for your help!
