    • Ti.APi.info should be Ti.API.info

      concerning url syntax i mean this:

      var win = Ti.UI.createWindow({
        url : 'main.js'
    });

// Ti.UI.currentWindow, etc.

      this a BAD and not recommended practice, and will lead to many errors because it creates multiple js execution contexts, every window called with an url will be executed in a different thread. So interaction between windows is a nightmare. Please use commonjs (with or without alloy)

      to migrate your code: take a look at this thread I posted an example in one of the comments
