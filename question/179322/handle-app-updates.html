  • Yes. Its possible.

    To update you need to check for application version in app.js if your project contain titanium classic code or alloy.js in case of alloy project. Below is the code to update version name.

    if(Ti.App.version == "4.0") {
    Ti.App.Properties.setString('VersionName', 'JellyBean');
}
if(Ti.App.version == "5.0") {
    Ti.App.Properties.setString('VersionName', 'Lolipop');
}

    This version is the version that you are specifying in tiapp.xml. When updating the app, you will be changing the version that version will reflect in the code when user update there app.
