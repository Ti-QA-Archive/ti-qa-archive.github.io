Go to root window from last window in iOS and android

Dear all,

I am using Alloy for my project. There are 4 modal windows in my app which gets open one after another. For iOS, I am using NavigationWindow for multiple window navigation and in android i am just declaring the windows as modals and it solves the purpose.

My problem is to navigate back to root window from the last window of the app.

My app is a feedback app, so after taking the text inputs and audio inputs from the user, on the last window I submit these details on a server and then display a thank you note to user. After displaying thank you note I want my app to shift to root window and reset all the details user entered previously so that user can enter a new feedback again.

Please guide me how should i implement such backward navigation. Please reply at your earliest as its urgent. I am a student and my project deadline is approaching.

Thanks in advance.
