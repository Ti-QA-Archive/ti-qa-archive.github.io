    • then you can pass module's object to TCDeviceProxy like.

      -(void)startup { // this method is called when the module is first loaded // you must call the superclass [super startup];

// Create singleton TCDevice
self.Device = [[TCDeviceProxy alloc] init];
self.Device.parent = self;

NSLog(@"[INFO] %@ loadedaaa",self);

}

      and in proxy just import Module.h file and synthesize parent variable.
