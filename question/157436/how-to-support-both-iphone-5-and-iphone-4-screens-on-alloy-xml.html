How to support both iPhone 5 and iPhone 4 screens on Alloy XML?

Hi,

On my Alloy XML files I have tags such as:

<Window id="winHome" barImage='/images/img_navbar.png'>

However, this img_navbar.png image is for the iPhone 4 screen. How can I support iPhone 5 images in the same XML file?

Or even, if I have this tag:

<ScrollView layout="vertical" width="320" height="434" backgroundImage="/images/home/img_bg.png">

How can I support different height properties for both iPhone 4 and iPhone 5 screens in the same XML file?

If it's not possible, what do you guys recommend as a best practice?

Thanks.

    Hello Bernardo Oliveira,

    For auto height and width you can set Ti.UI.FILL as height and width value. Ti.UI.FILL will grow its size to fill its parent

    "ScrollView":{
    height: Ti.UI.FILL,
    width: Ti.UI.FILL
}

    I hope that will help to you.

    Thanks
