  • I had the same issue today. It turned out the js file where I was using the Ti.Gesture module had a weird format and the builder could not understand that it had to pick the gesture library that is needed.

    Workaround: I user the Ti.Gesture on my app.js to force the builder pick it up.
    So my 1st app.js line was: 

    var forceOr = Ti.Gesture.orientation;
