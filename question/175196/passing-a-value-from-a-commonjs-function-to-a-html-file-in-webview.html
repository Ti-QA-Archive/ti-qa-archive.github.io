Passing a value from a commonjs function to a html file in webview

I have a webview in which i am displaying charts. Along with the charts i want to add a label
The text for this label is in a commonjs function which returns the value.
However for some reason the value is not getting returned.
Is there any work around for this…

    Hi

    Two ways;

    • Build the page as a file and then display it. This will give you the most control. But be a pain if the file itself is parameter based.
    • Use evalJS

    Using evalJS you can call functions inside your page and provide data as function arguments. Your internal HTML js code will then do whatever you need with it. Quick example;

    web.evalJS('setTitle("hello world");');

    The logic here is your page has a JS function that accepts a parameter that you provide and it uses that to update whatever you need it to.

    var title = 'hello world';
web.evalJS('setTitle("' + title + '");');

    This does the same but passes a variable instead - much more what you are looking for.
