Ti.API.addEventListener/fireEvent not available on Android???

Hello!

I am developing an app using MacOSX Snow Leopard, Titanium Studio 1.0.8.201201262211. Titanium SDK 1.8.1, Android 4.0.3 and XCode 4.0.2.
Application type: mobile, device Android emulator.

I ran into problems when cmpiling my app for Android, so I made a small testapp.
I created a new project using the Dashboard, I used the "Tabbed Application" template, and it works fine in both Android and iPhone emulators.

If I then add the following code in AppWindow.js, for example in the button click handler:

Titanium.fireEvent(&#39;my.special.event&#39;);

I get a Runtime Error (when clicking the button) with the following info:

[264,26] ti:/titanium.js:
Message: Uncaught TypeError: Cannot read property '_hasJavaListener' of undefined
Source: return this_properties[property];

Just adding an Titanium.API.addEventListener() for the event gives another type of error…

Can anyone shed any light on this for me?? I really need to use application events for communicating internally.
