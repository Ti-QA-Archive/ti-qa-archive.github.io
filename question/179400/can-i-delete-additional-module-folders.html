Can I delete additional module folders?

Hello,

This may sound like a silly question, but I figured its worth asking before I shoot myself in the foot ;-)

My app uses a few different modules and I have installed them via "Titanium Studio -> Help -> Install Mobile Module".

As part of these installs, my project got several folders that don't seem required such as documentation, example etc. Can I delete these folders?

I checked the Appcelerator docs and did not find anything on this.
http://docs.appcelerator.com/titanium/latest/#!/guide/Using_a_Module

Since I would like to keep my application size to the minimum, any tips that folks may have to limit module sizes will be great. Thanks!
