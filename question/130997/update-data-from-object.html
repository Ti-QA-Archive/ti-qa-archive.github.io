  • update

    CustomData[1].status = "7:00 PM";

    add new property

    CustomData[0].newprop = "new value";

    add new element

    CustomData.push({name:"Next event",status:"10:00 PM", hasChild:true});
//or
CustomData[2] = {name:"Next event",status:"10:00 PM", hasChild:true};

  • this is a javascript topic. For updating a field do:

    CustomData[0].status = "10:00 PM"; //ends
CustomData[1].status = "01:00 PM"; //starts

    you can add an entry at the end of array using push() method

    CustomData.push({name:"whatever", status:"8:00 PM", hasChild:false});

    google "javascript arrays" for more info
