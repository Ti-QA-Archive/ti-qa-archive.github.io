  • It seems that this build error has something to do with the new windows build functionality in 3.3.0. Navigate to your titanium sdk folder:

    Titanium\mobilesdk\win32\3.3.0.GA\node_modules\titanium-sdk\lib and open windows.js. Go to last function

    exports.getSelectedVisualStudio

    And comment out or delete:

    if (env.visualstudio) {
    return env.visualstudio[Object.keys(env.visualstudio).sort().filter(function (v) {
    return env.visualstudio[v].selected;
    }).pop()];
    }

    Then build should work.

  • To change the permissions do the follow:

    • open Applications > Utilities > terminal.app;
    • write cd ~/Library/Application\ Support/Titanium/ and press enter;
    • write chmod -R 777 mobilesdk and press enter;
    • write chmod -R 777 sdk and press enter.
