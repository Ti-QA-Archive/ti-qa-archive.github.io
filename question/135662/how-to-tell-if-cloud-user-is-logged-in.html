[code]
Ti.App.Properties.setProperty('loggedIn',true);

Ti.App.Properties.setProperty('userID',e.users[0]['id']);
[/code]

I can see there is a show me function but that seems long-winded also I'm guessing it makes a network call?

  • Matt, it looks to me that any call requiring authentication that I try to make while logged out will return an error with a code of 401. You could either check for that on your calls, or like you mentioned, make a show current user call that would also return the error with code 401.
