  • Hello Fddie,

    Try to increase the emulator timeout time by running the below command in terminal.

    titanium config android.emulatorStartTimeout 0

    Here set the value "0" to 150000 or more.

    You could also let the emulator start up and unlock the screen, when it is ready launch the app. Also, Try connecting android device and click run as android device. Running app in device takes less time than in emulator. Thanks.
