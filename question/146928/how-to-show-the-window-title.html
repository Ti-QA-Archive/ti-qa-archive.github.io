How to show the window title

Hello everyone,

My question is very simple: I would like to have my window title shown instead of the project title on the emulator:

when I use this code in a project named "My Project":

var win = Titanium.UI.createWindow({

    title: "First window",
    backgroundColor:"red",
    exitOnClose:true
});

I got "MyProject" into the Andrpod emulator! so how to do to have "First window" ?

Thank you in advance!

    Hi

    Assuming "Andrpod emulator" means "Android Emulator" and not some weird offspring of an Android and iPod - we are talking Android here.

    The "title bar" displays the applications title and is not designed to display the window title in its place.

    If you are using Ti SDK 3.0.0 and targeting Android 3 or higher you can make use of the ActionBar implementation now available - this will provide a better looking window title area with more functionality but would mean your app would not work on Android 2.x.

    You could and many do (including myself) hide the "title bar" and creating your own version using labels and views - you do this by changing the following setting in the file tiapp.xml found in the folder up one level from your app.js.

    <navbar-hidden>false</navbar-hidden>

    becomes

    <navbar-hidden>true</navbar-hidden>

    The "title bar" will then not display on your windows, you will also need to set navBarHidden: true in your window creation for other windows, using your own example;

    var win = Titanium.UI.createWindow({
    title: "First window",
    backgroundColor:"red",
    navBarHidden: true,
    exitOnClose:true
});
