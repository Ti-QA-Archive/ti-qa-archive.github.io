    • If you built the module yourself, you should be ok.

      There are some instructions you need to follow to update the module's XCode project file.
      Just open it in XCode. If it needs updating you'll see a yellow triangle warning right away. Click this. It will then ask if it can automatically update some settings. Say yes, then close it.

      You also need to make you have a version of the manifest file which contains the the line :
      architectures: armv7 arm64 i386 x86_64

      Another thing I found I needed to do was create a new dummy module project in Ti Studio and copy the latest build.py to my old project.

      That seems to work for me.

      If all else fails, (and it's worth doing, since the module project have a different folder structure now), recreate your module from scratch using Ti Studio and copy your classes over. (I've had to do this a few time now too).

      HTH.
