    The logging levels in that drop-down list are cumulative. So, selecting Debug tells Studio to include debug level messages plus all the others below it in that list. In a quick scan of the Studio docs, I don't see how to filter a level exclusively.

    You don't say what platform you're building for. If it's Android, you could execute the following in a terminal window:

    adb logcat | grep [DEBUG]

    It would display the logs, filtering just the debug messages. You could then turn off the console display in Studio. Sorry, I don't know the equivalent of grep on Windows… filter maybe?

    If you're building for iOS, perhaps the OS X Console utility would offer similar functionality.
