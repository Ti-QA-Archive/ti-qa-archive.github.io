Alloy - display a truncated string in a Label

Brand new to Titanium/Alloy and so far so good! However, I can't seem to find a way to make any modifications to text (from a column) before displaying it in a view. Essentially, I have a 'name' field that I want to abbreviate by truncating. I figure one option is doing this in the model definition using a custom attribute but I don't know how to access/bind to this attribute from within the XML template.

I tried creating my own commonJS module to do this and put the value into a Label, but this sent me down another rabbit hole with the build attempting to load a non-existent TiUITruncatorProxy class.

Suggestions?

    Hello Bruno,
    with Alloy Data Binding you can create custom(calculated) attributes within the dataTansform function in the view's controller, then use them in the view's XML markup. Find all details in the linked documentations. Hth.
