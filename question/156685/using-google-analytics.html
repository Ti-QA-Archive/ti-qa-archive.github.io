Using google Analytics

How do you implement google analytics tracking into your App?

Just recently we have revised our App. Prior to this, we simply loaded a url into a webview. Now we use a httprequest and load html into a webview. Since this change google analytics doesn't show any tracking data.
We tried out a free module from the marketplace, but it doesn't work, probably because of the new GA Version.

We have to decide if we buy a module or make our own module.

So the question is: how do you implement the last version of GA in your App?
Do you even use it?

Any advice would be much appreciated.

    The important question is:

    Do you want to track
    a) a Website from inside the HTML-Code (as you did before) or
    b) a mobile App inside of Titanium-JS?

    This is a difference in the usage and in the reporting.

    If your answer is a) you don't need a module. You just have to put the Google-Trackingcode inside your HTML like on a website.

    If your answer is b) you can use the free module, register a mobile App in your Analytics account and use the new UA.
    You then track users instead of visitors and screens instead of pages (and events, social, timinge optionally).
    You can use this module from the marketplace: https://marketplace.appcelerator.com/apps/5081?1183763187
    It works fine for me on Android and iOS.
    See the documentation of the module.
    If you track a screen the code is like this:

    Tracker.trackScreen(name);

    If you track an event the code is like this:

    Tracker.trackEvent({
    category : String,
    action : String,
    label : String,
    value : Integer,
});

    I recommend b) as you are more flexible and get much better analytics data (like devices, OS, app version, provider,…).
    It even tracks when your device is offline.
