  • Note to Appcelerator folks: it would be really, really helpful if you included "What's Changed" notes in the README when you update a module, so that folks using an earlier version (say, ti.storekit 1.0) know what has to change when upgrading (to, say, ti.storekit 1.2). OTHERWISE we're left to opening up the documentation and sample for each (old and new) and doing a visual comparison, which is a slow and buggy process. If you did document these changes somewhere… sorry, I didn't see them where you'd expect (in the module directory, documented on a per-release basis). Thanks for listening.

    To answer the poster's question: it appears that in addition to the "sharedSecret" support mentioned by Talal, there is a 1.0->1.2 breaking API change to Ti.Storekit.purchase, and perhaps a change to what's returned in a "restoreCompletedTransactions". There may be other changes, but these are the ones I've noticed so far.
