    You can simplify your whole push to array, by using this single line of code.

    songs = jsonObject.songs;

    So change your onload as follows;

    xhr.onload = function () {
    var jsonObject = JSON.parse(this.responseText);
    songs = jsonObject.songs;
    Ti.API.info(songs.length); // just for confirmation
}

      By using this line index &lt; songs.length before you incremented the index, you allowed an extra index value of 1+ the total actual array elements.

      If you had 20 songs, then songs.length = 20, but the index of the array is zero based - 0 to 19. This line index &lt; songs.length allows 19 (the last actual array element) to be lower than 20 (the total songs) so it increments the index to 20 which is one larger than the array. So it was not until you pressed the button again that it reset to zero.
