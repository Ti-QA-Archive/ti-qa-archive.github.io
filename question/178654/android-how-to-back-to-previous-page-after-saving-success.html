  • Accepted Answer

    Assuming createNewInboxView is a Window, then closing this should take you back to the previous window.

    $.btnCreateInbox.addEventListener("click",function(){
  /*
  ** Some code to Save New Inbox
  */

  $. adrCreateInbox.close();
});
