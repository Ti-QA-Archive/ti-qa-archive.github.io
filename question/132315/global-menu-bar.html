Global Menu Bar

Hi,

I need to create a menu bar(horizontal) on the top with the options "Accounts,Contacts,Opportunities,Settings and Products"..I have created it in the app.js file.. But i want this to appear in all the above pages also(accounts page and other pages)..How do i create it globally? One more thing is, I have created new window on top of the default window for each of the menu item.. Please help me with this.. My code snippet in app.js is as follows:

var navBar = Titanium.UI.createButtonBar({
labels:['Accounts','Contacts', 'Opportunities','Products','Settings'],
top:150,
height:40,
style:Titanium.UI.iPhone.SystemButtonStyle.BAR
});

navBar.addEventListener('click', function(e)
{
var idx = e.index;
Ti.API.info(e.index);
switch(idx)
{
case 0:
{
var newWindow = Ti.UI.createWindow({
url:'accounts.js'
});
newWindow.open();
}

url:'accounts.js'
});
newWindow.open();
}

{

}

});

    To create a global menubar,make a separate js file with UI of menu bar and use following code in each file to which you want to add

    Ti.include("/folder/file.js");
