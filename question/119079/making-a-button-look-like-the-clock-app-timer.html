Making a button look like the clock app timer

In the clock app on the iPhone under timer, there is a button to select the sound to play when the timer ends. The button is flat, has a slight gradient, has an arrow on the right hand side, and has two labels of differing sizes, but with drop shadows.

I want to duplicate this. I'm thinking that I need an image of the button that includes the arrow, both with selected coloring and unselected coloring. The labels shouldn't be too big of a deal (I can make labels with drop shadows on buttons in code). The problem is that I'm a complete dunce when it comes to image editing. I have no clue how to go about making the transparent PNG images of the button.

How do I do this? It seems to menial a task for most graphic designers, but out of my league completely…

    Hi,

    You can easy replicate using an VIEW with gradient background, rounded corners, and 3 labels in it.

    and you can add a click event to this view to use like a button.

    Sample (Identical to button do you need):

    
// Window
var w = Ti.UI.createWindow({
    backgroundColor:'#000'
});
w.open();


// View (Button)
var v = Ti.UI.createView({
    height:40,
    width:300,
    top:100,
    borderRadius:8,
    backgroundGradient:{type:'linear', colors:['#AAA','#DDD'], startPoint:{x:0,y:0}, endPoint:{x:0,y:40}, backFillStart:false}
});
w.add(v);


//Label 1 
l1 = Ti.UI.createLabel({
    text:'On Finished',
    font:{fontSize:16, fontWeight:'bold'},
    top:0,
    left:10,
    height:40,
    zIndex:1,
    shadowColor:'#fff',
    shadowOffset:{x:0,y:1}
});
v.add(l1);


//Label 2
l2 = Ti.UI.createLabel({
    text:'Marimba',
    font:{fontSize:14, fontWeight:'normal'},
    top:0,
    left:v.size.width-100,
    height:40,
    zIndex:2,
    shadowColor:'#fff',
    shadowOffset:{x:0,y:1}
});
v.add(l2);


//Label 3 
l3 = Ti.UI.createLabel({
    text:'>',
    font:{fontSize:16, fontWeight:'bold'},
    top:0,
    left:v.size.width-16,
    height:40,
    zIndex:3
});
v.add(l3);


//Events
v.addEventListener('touchstart', function() {
    v.backgroundGradient={type:'linear', colors:['#55F','#33D'], startPoint:{x:0,y:0}, endPoint:{x:0,y:40}, backFillStart:false};
    l1.color='#fff';
    l1.shadowOffset={x:0,y:0};
    l2.color='#fff';
    l2.shadowOffset={x:0,y:0};
    l3.color='#fff';
});

v.addEventListener('touchend', function() {
    v.backgroundGradient={type:'linear', colors:['#AAA','#DDD'], startPoint:{x:0,y:0}, endPoint:{x:0,y:40}, backFillStart:false};
    l1.color='#000';
    l1.shadowOffset={x:0,y:1};
    l2.color='#000';
    l2.shadowOffset={x:0,y:1};
    l3.color='#000';



    alert('Here your function!');


});
