    The Build.py script is located in the iPhone folder within the Titanium SDK folder (in your case 1.2.0), however two things come to mind:

    1. You don't need to use Ad-Hoc, you can run the "Install on device" which creates a *.app file in the build folder that you can send to people via email to test the app (I'm guessing that is what you want to do?)

    2. Update your SDK to 1.5.1! All SDKs will run on iPhoneOS 3.1.3, just the new iOS features will not be available/work.

    3. Only use the build.py script if you know what you are doing, you should be able to do everything with Titanium Developer.
