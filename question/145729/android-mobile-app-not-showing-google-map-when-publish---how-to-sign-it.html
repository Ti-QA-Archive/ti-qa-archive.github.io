Hi all!

Application type: mobile

Device: physical devices

My app integrates google maps but when published I only see the grid and not the map.

Someone can say me where to insert it? In the titanium xml (tiapp.xml) or directly in the android custom manifest (manifest.custom.xml)?

    • Finally i solved my problem, looking more carefully at the default values that titanium uses, when building the app for distribution.

      In the distribute window, titanium auto suggests as the alias key, the value from the <id> in the tiapp.xml

      In my case the alias key doesn't match my app id.

      My app id was com.test.App and the alias key in the android.keystore was com.test.app

      Changing the alias key, in the build for distribution window, to its lower case (com.test.app) did the trick.

      You can verify the actual alias key name using:

      keytool -list -v -keystore android.keystore
