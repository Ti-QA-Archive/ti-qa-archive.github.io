can not get modules working in 2.0

I have a native android module that used to work in 1.7 and earlier. I have used this since 1.2 and have not done anything in titanium in awhile. If I try to use the old module from 1.7, it says that this is not supported in 2.0. So I created a new module in 2.0. I finally got this compiling properly, however my app crashes as soon as I include the module (even without using a require). I tried just creating the default code included when you create a new mobile module project, and this causes apps to stop unexpectedly upon initialization as well. Do android modules even work in 2.0 or am I just wasting my time?
