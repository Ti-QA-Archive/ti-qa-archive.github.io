HTML Password-type TextField?

Would there be anyway to re-create the:

<input type="password" name="password" />

HTML code in Titanium 1.6.1 for iOS 4.3, without using a WebView and putting that in the HTML? I have a Login/Register part to my App and instead of just plainly putting in their Email/Password, I would like to change the text of the password as their typing it to "•"s, but I do not know how. Is there already a way to do this, I just don't know the flag or API for it?

Could somebody enlighten me on how this is done? Thanks :)
