Is it possible to scale these photos down, or to resize them, perhaps make them half of the original size, before sending??

  • You are looking for the imageAsResized() property..

    Try this:

    var filename = Titanium.Filesystem.tempDirectory + Titanium.Platform.createUUID();
f = Titanium.Filesystem.getFile(filename);
f.write(event.media.imageAsResized(320,240));
f.remoteBackup = false;

    For Android use this: https://github.com/novelys/titanium-jpegencoder

    I've found the ImageAsResized module in the Marketplace for Android doesn't work properly compared to this. Watch out for memory though..

  • This is possible to do via a Ti module.

    But one quick hack you can try to is to edit TiBlob.m in your SDK. Look for the 'data' function and change the compression down from 100% ('1.0') thus:

    
-(NSData*)data
{
    switch(type)
    {
        case TiBlobTypeFile:
        {
            NSError *error = nil;
            return [NSData dataWithContentsOfFile:path options:0 error:&error];
        }
        case TiBlobTypeImage:
        {
            if ([@"image/png" isEqualToString:mimetype]) {
                return UIImagePNGRepresentation(image);
            }

    // changed to 0.5, which is 50% compression.
    // much smaller files are saved and very little quality is lost.

            return UIImageJPEGRepresentation(image,0.5);
        }
        default: {
            break;
        }
    }
    return data;
}
