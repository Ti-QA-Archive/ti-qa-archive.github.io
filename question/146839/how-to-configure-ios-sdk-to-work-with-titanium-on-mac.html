How to configure iOS SDK to work with titanium on mac?

Dear all,

I'm working on max OS X Lion 10,7,4 and I have a fresh install of Titanium Studio 3.0.1. I have installed Android SDK and XCode 4.5.2.

The problem is when I create a new project, the wizard does not allow me to choose iPad and iPhone as deployment target. When I click on the "set-up/Configure SDK" link, in the dialog box the iOS SDK Home field is empty. I don't know what I should have in this field.

Could you please help me?

Many thanks in advance

    Hi execute this command in terminal and it will select Xcode in Titanium.

    sudo xcode-select -switch <xcode_folder_path>

    xcode_folder_path :: is generally.

    /Applications/Xcode.app/Contents/Developer for Xcode 4.3

    for more info visit here.
