  • Apparently I was using the wrong version of node.js (v.0.10) which is now the default version that is installed when you follow the official instructions. Downloading v.0.8.22 (http://nodejs.org/dist/v0.8.22/) resolved this problem for me.https://developer.appcelerator.com/question/149920/creation-problem-with-alloy#
