    Hello Faisal,
    app.js is the main (and possibly unique) executed file in a Titanium project.
    You can put all your code in there, or split your code into different .js and include them in app.js. I'd suggest you make some test just with the single app.js at fist, if you check the default template code created with a new mobile project you should find some useful hints for you quest.
