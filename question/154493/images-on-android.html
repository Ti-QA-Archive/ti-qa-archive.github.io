Images on Android

var circulitoPro = Ti.UI.createImageView({
        image : "images/tablet/android/image.png",

var btonImagen= Ti.UI.createButton({
        backgroundImage: "images/tablet/android/image.png"

    Hi,

    First try to set width and height as integer value and also for android
    set &#x2F; before path like

    &#x2F;images&#x2F;tablet&#x2F;android&#x2F;image.png

    its for android only.
