    • Oops..accidentally posted without completing the message.

      You need to update PATH Environment Variables in Windows.

      1. Uninstall all versions of the JDK.
      2. Reinstall the latest version ( I installed the version Titanium was trying to install Java 6 u25)
      3. Add an environment variable %JAVA_HOME% path C:/Program Files/Java/javasdk_someversion (In XP Start>ControlPanel>System>Advanced a quick google will help you figure out the rest but its pretty simple)
        4.Update path using the newly added variable ( Same menu as env. variable, just click PATH from the bottom menu and edit add the following %JAVA_HOME%\bin )

      Steps 3 and 4 are very general but once you see the menus, it should be clear as to what I am trying to say. This is in XP if you're in 7 then you may just need to use a little google magic to figure it out. Goodluck! (Even though I know this may be a little too late)
