    Hi, please try the code below.

    In your app.js file use code like,

    var win1 = Titanium.UI.createWindow({
    backgroundColor:'#000', 
});
win1.open();


// Create a Label.
var Lbl = Ti.UI.createLabel({
    text : 'Label',
    height : 'auto',
    width : 'auto',
    top:50
});

// Listen for click events.
Lbl.addEventListener('click', function() {
        var newWindow = Titanium.UI.createWindow({
            modal:true,
            url:"back.js"
    });
    newWindow.open();

});

// Add to the parent view.
win1.add(Lbl);

    and another back.js file use,

    var newWindow = Titanium.UI.currentWindow;

// Create a Button.
var btn= Ti.UI.createButton({
    title : 'Click here to go back',
    height : 100,
    width : 100,
    top : 30
});


btn.addEventListener('click', function() {
    newWindow.close();

});

newWindow.add(btn);

    This should work.

    Thanks

    • Ref: follow-up question about window load animation.

      As mentioned Android does not support animation when loading windows - at least not sliding in - this is an iOS specific UX element. Whilst animation can be performed on a loading heavyweight window - it is not as easy and not common. See other Android apss to see what I mean, sliding is contained to views most of the time. Beyond that you need more experience.

      — commented March 21st 2014 by Malcolm Hollingsworth

    • Ref: scrolling text.

      You need to add a scrollView to the window with a contentHeight of Ti.UI.SIZE. You then add the label or whatever other child views you require. If you have more than one child element on the scrollView you need to set the layout property to vertical.
