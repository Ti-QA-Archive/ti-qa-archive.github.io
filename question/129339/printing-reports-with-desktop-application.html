  • I don't know anything about using Crystal Reports, etc., but a basic way to print what's on-screen in your Appcelerator desktop app is to use JavaScript's built-in window.print() method. It will open a "Print Setup" dialog where the user can choose a printer (but there's no page-range option…?). When the user clicks OK, it should print the window's contents the same way it prints a web page.
