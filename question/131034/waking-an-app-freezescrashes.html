Waking an app freezes/crashes

Hi there,

I've only noticed this with Titanium built apps; I open the app and then close it using the home button (so it's still running in the background) when I come back to it after a few minutes later the app is not responsive.

If you wait a little, it eventually comes back or if you get a little click happy on the tabs while it's froze it then crashes completely.

Any thoughts?

  • Hi Craig,
    I had the same issue and solved by setting analytics to false from tiap.xml. You will be surprised but that is the reason of application getting frozen on wakeup etc. Go to tiap.xml > look for <analytics> > set it to false > clean the project and compile again. Voila, your app is finally acting like native one!
