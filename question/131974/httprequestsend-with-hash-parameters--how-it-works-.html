HttpRequest.send with hash parameters : how it works ?

Hello,

I'm new in Titanium and i'm building a mobile authentification application for android. I want to send data to a web server (Rails) with hash but it doesn't work.

That's I try to send: 

var params = {
    user : { email : 'test',
        password : 'test',
        client : 'client'
    }
}

But the user params value is converted into a string… Server side, i keep :

Parameters: {"user"=>"{client=client, password=test, email=test}"}

And I want to have : 

Parameters: {"user"=>{"client"="client", "password"="test", "email"="test"}}

Maybe I have a property to set for my xhr…

Thank's

    Your best bet is to use JSON.

    In Titanium:

    xhr.setRequestHeader('Content-Type', 'application/json;charset=utf-8');
xhr.send(JSON.stringify(params));

    Then in Ruby just decode the JSON.
