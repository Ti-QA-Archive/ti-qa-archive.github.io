Event sounds?

I searched on the QA site but couldn't find anything relating to event sounds. For example, if there is a short sound effect I want to play when a button is pushed or some other function is triggered.

The kitchen sink seems to have only long sound demonstrations.

Are there any examples of this?

  • All you have to do to accomplish this is to load your sound(s) and play them on the event you want them to be attached to. ie:

    var sound = Ti.Media.createSound({url:"somesound.wav", preload: true});
var btn = Ti.UI.createButton();
btn.addEventListener('click', function() {
  sound.play();
});
