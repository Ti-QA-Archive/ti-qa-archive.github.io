    Hi this is ACS web dev,

    apparently we had an update but didn't update the docs.
    There is no search, but there is query method which basically works pretty much same.

    query method which works like other query method provided on other objects(http://cloud.appcelerator.com/docs/search_query)

    example:

    curl -c cookies.txt -b cookies.txt -X GET --data-urlencode 'where={"name":"asdf"}' --data-urlencode 'order=created_at' "http://api.cloud.appcelerator.com/v1/keyvalues/query.json?key=<app key>"
