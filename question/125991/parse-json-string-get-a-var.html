      Anyway, you have one error that I haven't seen before:

      alert(json[0].['id']);

      Should be:

      alert(json[0]['id']);

      So, without dot, because you can't use both syntaxes (dot notation and square-bracket notation) at the same time.

    { id : 39, name : me }

    Fix that and you'll get "id" with "json.id".
