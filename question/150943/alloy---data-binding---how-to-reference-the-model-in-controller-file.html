    You created a controller-specific model in textField.xml. it will not be available in in another controller, in this case "record". If you wanted to expose this controller-specific model you can do so by creating a public reference to it from your "textField" controller. Something like this:

    textField.js

    // this exposes "myLottery" as a reference to the controller-specific model
exports.myLottery = $.myLottery

    record.js

    // "tf" was the ID you gave to the "textField" controller in your markup
$.tf.myLottery.set({
    title : "home",
    content : "Warriors"
});
