How can I install a previous version of Android SDK?

Hi all, I'm stuck with the bug Android SDK 22.6.2. How can I intall a previous version? I've downloaded the r22 but when i install it (obviusly uninstalling the other first) automatically is downloaded the damn'it last version. Thanks for the help.

  • I found a solution. It is possibile to download a zip file of ADK tools. So, enter in your android sdk path and rename (or delete) the tools folder. Put the tools folder contained into the zip file and you get the tools version you want. The path for downloading tools zip file is:
    http://dl-ssl.google.com/android/repository/tools_rNUMBERVERSIONYOUWANT-windows.zip
    In my case the version is tools_r22-windows.zip
    For Linux and Mac you have to subtitute the word 'windows' with 'linux' or 'macosx'.
    I hope it can be useful for others with my same problem. Bye
