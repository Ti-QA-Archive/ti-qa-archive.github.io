Transparent TextArea?

Is it possible to have a TextArea without a background? My view has an image as a background and I'd like to have text on it without drawing an ugly white text box over the image. I've done my googling but failed to find a solution.

The app is for iPad if that's relevant.

    var my_txt_field = Ti.UI.createTextField({
    // other props,
    backgroundColor:'transparent'
})
