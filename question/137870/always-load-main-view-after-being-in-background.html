Always Load Main View After Being In Background

Essentially, I want the app to only load the Main View whenever it starts up. It will by default do this when you start up the app for the first time or when you load the app when it's not running in the background.

All I need to do is figure out how to bring it back to the main view, even if the user was in a separate view, whenever the app comes out of the background.

Other info:
Mobile App, Titanium SDK 2.0.2, For both iPhone and Android

Thanks!

    Best way to do this would be to use the app resumed event listener. See below.

    Ti.App.addEventListener('resumed', function(){
    //Add code here to close all potential windows and open the main one.
    window1.close();
    window2.close();
    main_window.open();
});

    Hope this helps!
