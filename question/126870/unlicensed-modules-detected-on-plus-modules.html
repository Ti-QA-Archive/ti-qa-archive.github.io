Unlicensed module(s) detected. On Plus modules

I am an Indie, trying to add the ti.storekit which I downloaded from the HelpDesk. I added it to my project (same as other modules I use) and when I run I get the following message: "Unlicensed module(s) detected. You must be a subscriber…".
I have seen that others got the same problem but didn't understand how to solve this !!!

Please note that I tried to restart the Studio, clean the project - nothing helped.

Please Advise !!!

  • You need to follow steps as follows:

    1. Open Terminal and run “uuidgen”
    2. Copy the code you see on the terminal
    3. Open tiapp.xml, replace what’s in the “guid” tag with the the code you copied.
    4. In Titanium Studio, do Project –> Clean
    5. Run

    Concept is to change GUID of module which is mentioned in manifest.

  • I had the same problem on a project I was working on!
    I followed the suggestion posted here and created a new project as a test and actually the module was working there…
    So I made a diff between the folders of the test project and my project… and I found that the problem was in the tiapp.xml file… the project was created by one of my colleagues, so in the tiapp.xml file was set my colleague's guid instead of mine.
    So when I was trying to build it I was unable to use the module, because my colleague didn't purchase it on the marketplace.
    When I changed the guid it worked perfectly!!
    That could happen if many people are working on the same project!!

  • Resolved. Just to let you know, anyone who encountered the same problem (with other modules as well) -
    If the Titanium Studio project was created before you signed into the Indie program you would get the "Unlicensed.." msg for every Plus+ modules. You should create a new project (after signing as Indie) and then add your modules to the project.
    Took me a while to figure this out. Would have been nice to find it on TroubleShooting or Q&A.
    Anyway, hope that helps.
