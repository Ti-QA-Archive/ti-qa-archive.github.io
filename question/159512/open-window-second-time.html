    win2.open();

    win2.close();
    win1.open();

  • Hi Alzhrani,
    Please find the below code :

    but1.addEventListener('click',function(){
    win2.open();
    win1.close();
});
but2.addEventListener('click',function(){
    win1.open();
    win2.close();
});

    Just make sure you open the next window before closing the current one.
