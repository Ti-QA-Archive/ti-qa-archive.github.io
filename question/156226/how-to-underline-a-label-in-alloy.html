  • Hi

    The font object does not provide an underline option for Alloy or non-alloy projects.

    Available properties;

    • fontFamily
    • fontSize
    • fontStyle
    • fontWeight

    If you really do need underline (not a common UI element these days) then you would need to combine a label with a single pixel high view underneath the label.

    You could try using try using the html property of the label view, this is only available in Android though - which should be fine for you.

    Example usage

    <Label id="label2" html="<u>Underlined?</u>"></Label>

    I have not tried this but worth a go.
