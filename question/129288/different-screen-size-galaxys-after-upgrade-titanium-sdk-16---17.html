different screen size galaxyS after upgrade Titanium SDK 1.6 -> 1.7

I got a very small font and small icons after upgrading Titanium SDK 1.6.2 -> 1.7.[2,3,5] if I run my application on my Galaxy S. In emulation mode it looks ok. What do I wrong/ how to correct?

    there is an article about this on the wiki. The issue is that by default in 1.6 was to handle all screen sizes within the Sdk, now this is not the case and you have to override the default in the tiapp.xml file. I will go and try and find the article and post the link here.

    you need this section Disabling Multiple Density Handling of this document

    http://wiki.appcelerator.org/display/guides/Using+density-specific+resources+on+Android
