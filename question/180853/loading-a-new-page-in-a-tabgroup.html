  • It seems you have been out of the game for a while and it looks like you are trying to create a new app with some starting tabs.

    I suggest you take a moment to review the documentation as a lot has happened in the last 3 years, whilst most things are the same - many of the ways they work together have changed. Given this looks like a new app - it is worth starting with Alloy, this might seem like a big difference from how you previously created apps - but given you need a refresher - this is a good time take a look. It will make it much much faster to get back into things and help you create better apps because of it. If you were updating an app you would need to stop using the old url property anyway.

    If you create a default Alloy app with tabs - you will be half way there already - you can then start adding your own extras and work out how they affect things.

    As Stephen Feather has pointed out - the older way of the url property on the Window is basically dead. CommonJS was a better way, but that should now be used as a logic only method of creating libraries of code. Alloy and Widgets are the way forward.

    Give yourself a little time to play and your whole app will come together so much faster with some good old fashioned planning.

  • Hello L F….Try the below code. Say you have a two tab application and in each tab there is a window with a button. When you press a button it opens a new window which code is in a separate js file.

    app.js

    // this sets the background color of the master UIView (when there are no windows/tab groups on it)
Titanium.UI.setBackgroundColor('#000');

// create tab group
var tabGroup = Titanium.UI.createTabGroup();


//
// create base UI tab and root window
//
var win1 = Titanium.UI.createWindow({  
    title:'Tab 1',
    backgroundColor:'#fff'
});
var tab1 = Titanium.UI.createTab({  
    icon:'KS_nav_views.png',
    title:'Tab 1',
    window:win1
});

var button1 = Titanium.UI.createButton({
    color:'#999',
    title:'Go Home!',
    font:{fontSize:20,fontFamily:'Helvetica Neue'},
    textAlign:'center',
    width:'auto'
});

win1.add(button1);

// Listen for click events.
button1.addEventListener('click', function() {
    var RescueWin = require('Home'),
    win = new RescueWin();
    win.open();
    Ti.API.info('\'Go Home!\' was clicked!');
});

//
// create controls tab and root window
//
var win2 = Titanium.UI.createWindow({  
    title:'Tab 2',
    backgroundColor:'#fff'
});
var tab2 = Titanium.UI.createTab({  
    icon:'KS_nav_ui.png',
    title:'Tab 2',
    window:win2
});

var button2 = Titanium.UI.createButton({
    color:'#999',
    title:'Go Next!',
    font:{fontSize:20,fontFamily:'Helvetica Neue'},
    textAlign:'center',
    width:'auto'
});

win2.add(button2);

button2.addEventListener('click', function() {
    var RescueWin = require('Next'),
    win = new RescueWin();
    win.open();
    Ti.API.info('\'Go Next!\' was clicked!');
});

//
//  add tabs
//
tabGroup.addTab(tab1);  
tabGroup.addTab(tab2);  


// open tab group
tabGroup.open();

    Home.js

    function Home(){

Ti.UI.backgroundColor = 'white';
var win1 = Ti.UI.createWindow();

var tableData = [ {title: 'Apples'}, {title: 'Bananas'}, {title: 'Carrots'}, {title: 'Potatoes'} ];

var table = Ti.UI.createTableView({
    bottom:10,
      data: tableData
});
win1.add(table);
var button1 = Titanium.UI.createButton({
    color:'#999',
    title:'Go Back!',
    bottom:0,
    font:{fontSize:20,fontFamily:'Helvetica Neue'},
    textAlign:'center',
    width:'auto'
});

win1.add(button1);

// Listen for click events.
button1.addEventListener('click', function() {
    win1.close();
    Ti.API.info('\'Go Back!\' was clicked!');
});

return win1;


}

module.exports = Home;

    Next.js

    function Next(){

Ti.UI.backgroundColor = 'white';
var win2 = Ti.UI.createWindow();

var tableData = [ {title: 'Apples'}, {title: 'Bananas'}, {title: 'Carrots'}, {title: 'Potatoes'} ];

var table = Ti.UI.createTableView({
    bottom:10,
      data: tableData
});
win2.add(table);
var button2 = Titanium.UI.createButton({
    color:'#999',
    title:'Go Back!',
    bottom:0,
    font:{fontSize:20,fontFamily:'Helvetica Neue'},
    textAlign:'center',
    width:'auto'
});

win2.add(button2);

// Listen for click events.
button2.addEventListener('click', function() {
    win2.close();
    Ti.API.info('\'Go Back!\' was clicked!');
});
return win2;

}

module.exports = Next;

    Hope this helps. Thanks.

  • You can open new window using below syntax,

    
var tabGroup = Titanium.UI.createTabGroup();

var win1 = Titanium.UI.createWindow({
    title:'HOME',
    url: 'home_main.js'
});

var buttonHome1Login = Titanium.UI.createButton({
    title: 'Home'
});

buttonHome1Login.addEventListener('click',function(e) {
    var window = Ti.UI.createWindow({
        title: "Register",
        url: "home_register.js"
    });

    tab.openWindow(window)

});
win.add(buttonHome1Login); 

var tab = Titanium.UI.createTab({
    window: win1
});

    Note - Using url property is not recommended. It is not only antiquated, but also soon to be deprecated. You can use commonjs module instead url property.
