    • Jason, the tutorial shows how to remove the shadow of a NavigationGroup (http://docs.appcelerator.com/titanium/3.0/#!/api/Titanium.UI.iPhone.NavigationGroup), the file I modified in the Titanium framework lives in Classes>Modules>UI>iOs>iPhone>Nav Group>TiUIiPhoneNavigationGroup, I track down what file to modified based on create the navbar, in my case: Titanium.UI.iPhone.createNavigationGroup. If you are using a different type of window, its navbar is probably not created in by the same file, you will have to find what class in Xcode creates your window and add the same "clipsToBounds = YES;" to that window. Hopes this makes sense.
