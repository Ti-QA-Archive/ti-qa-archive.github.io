Alloy, MVC - how to load another view

I've gone through the docs a dozen times, gone over all the example projects and only found one that shows how to go from one controller/view to the next - what I would consider pretty basic requirement for MVC. I'm running into lots of problems. First, my Index sets up a TabGroup and there is a separate view for each tab. Lets take my profile view as an example, it shows the user's profile. There is an edit button and I want to load a new view for editing certain fields.

I call the "action" so to speak in my profile controller to open a view for the profileedit controller and its view, this appears to have to be a window and so when it opens it hides the tabgroup. If I make the view for the editprofile an actual View, open() doesn't exist and show() does nothing, even when added to the views stack in the profile controller.

If I can only have 1 view for a tab, that has to have groups of hidden views as its "sub-pages" this would create one huge controller and doesn't seem to be the correct use for mvc.

Am I missing something? thanks
