Alloy TabGroup event listener

How do I identify the TabGroup in the controller? I want to add a listener to capture click events on the tabs, but I can't seem to find how to call it.

index.xml

<Alloy>
    <TabGroup>
        <Require src="homeView"/>        
        <Require src="contactView"/>
        <Require src="digsafeView"/>
    </TabGroup>    
</Alloy>

I tried $.index.TabGroup.addEventListener() but that failed. Since I didn't find any direct reference in the forums to this issue, it usually means I'm overlooking something obvious. I output the JSON to the console and found nothing that helped.

Any ideas? Seems like it would be easy if I knew how to ID the object.

  • Give the tab group an id (file.xml):

    <TabGroupd id="tabGroupID">

    …and access it like this (file.js):

    $.tabGroupID.addEventListener(...);
