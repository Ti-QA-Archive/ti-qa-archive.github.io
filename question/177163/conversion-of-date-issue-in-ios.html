Conversion of Date issue in iOS

Hi All,

We are getting some date conversion issue in iOS but in Andrios it is working fine.

We are getting some date format like this "1992-08-20T00:00:00"

We wrote a method to convert it to 1992-08-20,

function dateFormatYyyyMmDd(a_date) {

var d = new Date(a_date);

alert('This is new Date',d);
var mo    = d.getMonth() + 1;
var day   = d.getDate();
var yr    = d.getFullYear();
if (mo < 10) { mo = "0" + mo; }
if (day < 10) { day = "0" + day; }    
var vDate = yr + "-" + mo + "-" + day;

var tmp   = vDate;
return tmp;

}

The above method is fine for Andriod and returning correctly but for iOS we are getting NNaNaN some thing like this. 

var d = new Date(a_date); This is giving empty.

Can any one explain the issue here.
