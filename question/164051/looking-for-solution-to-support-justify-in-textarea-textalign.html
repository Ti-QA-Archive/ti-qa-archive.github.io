  • Hi,

    Label has the attribute "textAlign" but the only supported values are "left", "center", "right" and the TEXT_ALIGNMENT_constants(TEXT_ALIGNMENT_LEFT, TEXT_ALIGNMENT_CENTER, TEXT_ALIGNMENT_RIGHT). "Justify" is not supported.

    • Hi,

      For text align you can use custom module (TZ Label Styled Label)[https://marketplace.appcelerator.com/apps/2732?] Which is available in AppC marketplace. Please use this module and my code. I hope it help you for solving your issue. 

      var StyledLabel = require('ti.styledlabel');

var win = Ti.UI.createWindow({
    backgroundColor : '#fff'
});
var container = Ti.UI.createScrollView({
    scrollType : 'vertical',
    contentHeight : 'auto',
    disableBounce : false,
    horizontalBounce : false,
    verticalBounce : true,
    showVerticalScrollIndicator : true
});
var label = StyledLabel.createLabel({
    height : Ti.UI.SIZE || 'auto',
    top : 5,
    right : 5,
    bottom : 5,
    left : 5,
    width : Ti.UI.FILL,
    html : '<div style="font-size: 30px; color: #000; text-align: justify; background-color: red">Thanks for writing us. We have tested your issue. Please check my code for justify text area.</div>'
});
container.add(label);
win.add(container);
win.open();

    • This custom label module will provide an answer but keep these things in mind;

      • Fully Justified text is a very poor user experience when reading
      • Users with visual impairments find justified text harder to read
      • This module provides a mini web view for each label, this means overuse will be a memory hog. There is NOTHING wrong with using the module but do not relpace all other labels with this one or you will have problems.
