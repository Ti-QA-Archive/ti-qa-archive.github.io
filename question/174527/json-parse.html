    Hi

    Not really sure what you are after, we have no way of testing the URL, we can only assume the returned data is as you have presented.

    If you are simply trying to work out if the link debug link is the correct path based on the supplied JSON object then - no.

    {
  "unn_link": [ 
    {
      "unn_link": 
        {
          "link":"http://216.245.200.114:1935"
        }
     }
  ]
}

    So

    unn_link[0].unn_link.link

    Your object references two nodes with the name unn_link, as you will see from my expanded object.
