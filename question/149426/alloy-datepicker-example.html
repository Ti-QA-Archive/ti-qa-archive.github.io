  • You can put in your view:

    <Picker id="picker" type="Ti.UI.PICKER_TYPE_DATE" selectionIndicator="true" ></Picker>

    but you still have to set minDate, maxDate, and value in your controller:

    $.picker.minDate = new Date(2014, 0, 1);
$.picker.maxDate = new Date(2020, 11, 31);
$.picker.value = new Date();

    and could also add an event on date change: 

    $.picker.addEventListener('change', function(e) {
        console.log("Selected date: " + e.value);
    //doSomething
});
