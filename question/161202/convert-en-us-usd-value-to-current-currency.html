    • The thrown error is because you didn't define the fx.base and fx.rates ; this module is not for real time and doesn't deal with any webservice, you have to do a remote xhr request and load the rates into a JSON array then you will not see this error, or you can manually do something like this offline and define a static rate for each currency like this:

      fx.base = "USD";
fx.rates = {
    "EUR" : 0.745101, // eg. 1 USD === 0.745101 EUR
    "GBP" : 0.647710, // etc...
    "HKD" : 7.781919,
    "USD" : 1,        // always include the base rate (1:1)
    /* etc */
}

      as you can see this module is useless if you don't get the rates online or define the values yourself!
