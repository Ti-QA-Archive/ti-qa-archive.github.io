Variable in EventListener not being set

Hi,

When i get to the Ti.API.info line, the variable theSelected is being present as null - can anyone help with this??

var theSelected;
Ti.App.addEventListener("app:fromHomeScreen", function(e) {
    theSelected = e.selectedItem;
});

Ti.API.info(theSelected);

My fireEvent looks like the below

Ti.App.fireEvent('app:fromHomeScreen', {selectedItem: selectedSection});

Thanks.

    In your code example, Ti.API.info will be called before your eventlistener has any chance to get fired.

    what you are doing is basically 

    var theSelected; // means undefined
Ti.API.info(theSelected); // will be undefined

    output the value in your event callback

    
var theSelected;
Ti.App.addEventListener("app:fromHomeScreen", function(e) {
    theSelected = e.selectedItem;
    Ti.API.info(theSelected);
});

  • What is selectedSection? I'm assuming it's a table or list section. You can pass any object that can be JSON serialized – that means no objects with methods (like any of the Titanium proxies). If an object has methods, it ends up passing through as null. Thus, your code isn't going to work. You'll need to pass through a custom object without methods.
