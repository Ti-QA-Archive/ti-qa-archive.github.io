Android/iPhone map app using map image with drag, pinch and touch hotspots to open new page.

Background

I have not done any Titanium development yet so I am just asking to see if this is the best solution for me. I could develop this app in game salad but that costs $299. Using Titanium would have a larger learning curve especially if the drag and pinch code has to be written from scratch. I would like to know how easy this app would be to develop for someone with limited JS knowledge. I have written a reaction time test in JS but nothing like a full application.

The App

A large image map of a city that the user is able to drag around the screen. Pinching would zoom in to the image. The map would contain location marked with a red dot. Upon pressing one of these dots the user will be taken into a new screen with details about the location.

As much information and help is greatly appreciated. Thank you.

  • Hello Steve,

    You could just use a mapView and the provided maps from google to use the pinch to zoom and drag functions (which are natively provided with a map view). Then you could use a custom annotation which would be on the positions that you want the user to click on and these could take you to a new page. Having not developed an app before the learning curve will be large but it is doable for someone of limited knowledge.

    Hope this answers your question!

    Carter
