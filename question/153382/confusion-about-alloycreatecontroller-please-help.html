  • This command will create an instance of the indicated controller along with its GUI and style.

    A controller needs to be created from command line using : Alloy generate controller controllername before you can use it with the createController function.
    This creates 3 files in your project: an xml file to declare the view items, a js file where you put your javascript code and a tss file where you put the styling.
