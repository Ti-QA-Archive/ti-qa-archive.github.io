  • Couple of things.

    First, remove the height declaration from your container view. Second, put a height on the buttons that isn't in percentage. If you set the height to 50%, you'll never get more than 2 buttons in the container view. If you set it to 10%, you'll get 10, etc…

    Try something like this:

    var container = Ti.UI.createView({ 
    layout: 'horizontal',
    backgroundColor:"gray",
    //bottom:"1%",
    //height:"40%"
});
var button;
for (var i=0;i<60;i++) {
    button = Ti.UI.createButton({ 
        //backgroundImage:"/Images/any.png",
        title: "Button "+i, 
        width: '50%',
        height: 100
    });
    container.add(button);
}
scrollView.add(container);

    Hope that helps!
