How to copy a new object which is isolate with old one ?

I have a JSON object and a copy of it

var abc = JSON.parse(' { "a1": "ab1" ,  "a2": "ab2" ,  "a3": "ab3" ,  "a4": "ab4" ,  "a5": "ab5" ,  "a6": "ab6" ,  "a7": "ab7"}');
var abc1 = abc;

When I change data of abc1, I want to keep data of abc. What should I do ? Please help me… thanks so much !

    ECMA specifies Javascript objects are passed by reference. That's why changing abc1 changes abc. So if you create a clone of abc, then you'll be good to go.

    var abc1 = JSON.parse( JSON.stringify( abc ) );
