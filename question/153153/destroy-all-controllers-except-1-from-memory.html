    • $.destroy() is only responsible for cleaning up any collection data binding that was used. It does not explicitly free up any other controller resources.

      — commented June 3rd 2013 by Tony Lukasavage

    • As Sunny stated, if you are going to have multiple windows/controllers open, you need to manage them in an array yourself. When your logout is triggered you'll need to close any open windows contained in those controllers. You'll also need to null out any global references you created in any of those controllers in order to ensure that they will be cleaned up by garbage collection.
