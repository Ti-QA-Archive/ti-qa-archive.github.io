Button - Scale background image

Is it possible to scale the height accordingly when the width has been set?

var menu = Titanium.UI.createView({
    top:"220dp",
    left:0,
    height:Ti.UI.FILL,
    width:Ti.UI.FILL
});
var menu_option = Ti.UI.createButton({
    backgroundImage:'/images/my.png',
    height:"auto",
    width:Ti.UI.FILL,
    top:62
});
menu.add(menu_option);

I want to height of the menu_option to be set based on the width, i.e. maintain the aspect ratio of my.png.
Is this possible?
