  • Your table can be of a solid color. Say #000.

    Each cell, in the descending order, can have opacity ranging from 1.0 all the way down to 0.1

    The only problem is, that the OS has to do a lot of rendering for this. So it would be wise to use solid colors, like "#fff" all the way to "#111". Opacity will just slow down your app.
