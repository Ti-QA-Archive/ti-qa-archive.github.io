Hi - I am using Titanium Alloy to built an android app. I have many images at a very high scale 1000 x1000 for example. On all devices I would like the image to take up 50% width and 50% height. How would i go about doing this. Including scaling and compressing the image to a reasonable size. The images are png files if that is of any interest

    Hi

    These two previous answer I gave usually cover most of this sort of question;

    • How to achieve same layout for Android and iPhone using Titanium
    • images in Android (also iOS)

    In addition though you need to keep in mind that there are so many Android device variations that you will never manage to cope with each variant. The best idea is to supply the best resolution for each density you are supporting (explained in the two links above).

    Density independent pixels (dp) work by determining the ratio of density to physical dimensions.

    • mdpi = 100%
    • hdpi = 150%
    • xhdpi = 200%
    • xxhdpi = 300%

    So if you had an image that works great on an mdpi device then would need versions that were scaled as above to that size. You place these into the respective folders - again outlined in the two answers.
