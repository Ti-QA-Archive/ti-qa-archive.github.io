(SDK 2.1) Label: Not firing "postlayout" event

Hi!

As the title should expect I'm using Ti SDK 2.1.0.GA and trying to get the label height with the postlayout event.

This is the code im using: (iOS)

var contentView = Ti.UI.createView ({
    width: Ti.UI.FILL,
    height: Ti.UI.FILL,
    left: 0,
    right: 0,
    top: 100,
    layout: 'vertical'
});

var textView = Ti.UI.createLabel ({
    text: 'long text',
    width: Ti.UI.FILL,
    height: Ti.UI.SIZE,
    font: {
        fontSize: 12
    },
    left: 10,
    right: 5,
    bottom: 5,
    top: 5
});

//adding the view
contentView.add(textView);

var postLayoutCallback = function (e) {
    Ti.API.info ("Layout done, left: "+textView.rect.x+", width: " + textView.rect.width);
    textView.removeEventListener ('postlayout', postLayoutCallback);
};

textView.addEventListener ('postlayout', postLayoutCallback);

// w/ or w/o this method the postlayout ist never firing
textView.updateLayout ({
    height: 30
});

Neither the event is firing nor textView.getRect() or textView.getHeight() gives me any information about the real sizes of the view.

Could someone give me some hint I'm missing or is there some bug in the sdk?
