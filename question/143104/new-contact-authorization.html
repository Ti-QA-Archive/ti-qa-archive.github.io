New Contact Authorization

So it seems after you ask the user for authorization to access contacts the first time… you can NEVER ask again. I have a switch in my app where the user can save the contacts in the app to their address book if they like.

In the event they at one time denied the app access, on purpose or by accident, I cannot ask them to authorize my app again. I have to instruct them to go into their privacy settings and change the setting there?

That is crazy. Please let me know if I understand correct. The requestAuthorization function does indeed say it works only if the status is Unknown.

    It looks like this is the behavior Apple set, nothing to do about.

    http://developer.apple.com/library/ios/#releasenotes/General/RN-iOSSDK-6_0/_index.html#//apple_ref/doc/uid/TP40012166-CH1-SW19

    Checking access authorization status:
    An app can use the authorization status API to check if it can access contacts, calendars, reminders, or the photo library. This API is independent from the request access API and will not prompt the user to grant or deny access.
