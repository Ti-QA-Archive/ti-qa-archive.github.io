Storing values

Is there a way to store your values so the next time you use the api you don't have to put in the values? Can I use cookies or something else?

  • Accepted Answer

    For simple storing:

    Ti.App.Properties.setString("key", "value");
var value = Ti.App.Properties.getString("key");

    For more complex storing, you can use the database. There is an example of this in Tweetanium.
