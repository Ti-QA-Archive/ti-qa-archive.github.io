Viewing on demand training, "Building Native Mobile Apps With Titanium". I've downloaded the sample code projects "bnapps-content.zip". But I get an error when trying to import any of the projects. Error = "Selected directory doesn't contain manifest. Please select a directory containing Titanium app." Same error when trying to open the root folder (unzipped), \Social, \Location, \Remote-Data, etc. Has anyone else been able to open or import these projects???

    Where did you find the bnapps-content.zip? Is it a project, or just the resources folder? What you can do if it's only the resources folder is make a new project, and copy the files into your project.

    Tjeu

    • I just found the bnapps-content. All the files are just the resources folders and the tiapp.xml. Just create a new project, and name it the same as the example in the bnapps-content you chose. Then when created the project go to its resources folder, and copy all the files from the resources folder in your bnapps-content into the resources folder of your new project folder. and also copy the tiapp.xml into your new project folder (overwrite the original one).

      Hope this helps!

      — commented July 25th 2011 by Tjeu Vdw
