iOS app never exits the map

In my app I have a button that gives direction to a specific location from the current location, using the openurl method:

Ti.Platform.openURL(&#39;http:&#x2F;&#x2F;maps.google.com&#x2F;maps?saddr=&#39;+latitude+&#39;,&#39;+longitude+&#39;&amp;daddr=&#39;+claimLatitude+&#39;,&#39;+claimLongitude);

On the iphone when the button is taped it loads the map, and so you would press the iphone button to exit the map but when I tap on my app to open it again, it loads the map again even after an hour!

    when you use Ti.Platform.openURL, it opens the native safari browser application. so you can not go back from that application.

    solution…

    you can open this url in your webView
