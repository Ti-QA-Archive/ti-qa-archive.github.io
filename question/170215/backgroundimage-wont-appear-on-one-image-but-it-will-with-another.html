backgroundImage won't appear on one image but it will with another

Hi,

I'm dropping a background image into a createWindow like this:

var eventDetailWindow = Ti.UI.createWindow({
    title : 'Cabaret',
    barColor : '#BF1818',
    orientationModes : [Titanium.UI.PORTRAIT],
        translucent:false,
           tintColor:'#ffffff',
        navTintColor:'#ffffff',
        backgroundImage: eventToDisplay.blurred,
        extendEdges:[]
});

However the image isn't appearing. However what makes this really odd, is if I change the value to eventToDisplay.icon (the image that's used in the list view calling this detail page) it works as expected. I've written the value of eventToDisplay of both .icon and .blurred to the console and they are the full URLs as I would expect. The image formats are both the same, and I've triple checked the files exist on the server.

Anyone got any advice? I'm pulling my hair out trying to figure out why one works and the other doesn't.

Pete
