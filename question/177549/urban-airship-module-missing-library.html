    • And maybe it is a good idea to install the module not globally but inside your project.

      Also please recheck the Breaking Changes and Setup inside the documentation. You need a AirshipConfig.plist inside your project to specify your Urban Airship credentials and options.

      The path looks like YourApp/platform/iphone/AirshipConfig.plist

      The deprecated options property has been removed but is still present inside the example on the documentation page!
