  • Hi,

    In the event that you're using a tabGroup or a navigationGroup , you need to create a button, style it with the images, title, etc. and then set it to the window like this

    var win = Ti.UI.currentWindow, //In case you are using the url property for windows. If not, directly reference to your window

leftButton = Ti.UI.createButton({
    title: 'Edit'
}),

rightButton = Ti.UI.createButton({
    image: '/addFriend.png'
});

// ... event listeners for both buttons need to be added before setting them to the window.

win.setLeftNavButton(leftButton);
win.setRightNavButton(rightButton);

    The preferred way on iOS is to get the button created by the system, but unfortunately, Titanium does not allow this at the moment, or I simply haven't figured out the right way to do it. But if I could, my instincts tell me it should be win.getLeftNavButton() and then, be able to style it, and set it back using the above method.

    Hope this helps.
