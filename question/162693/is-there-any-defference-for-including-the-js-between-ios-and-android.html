Is there any defference for including the .js between iOS and Android

This is my script, it works perfectly on iOS.
However, it shows error on Android.

Uncaught Reference Error : createAdView is not defined

this is my createAdView.js it is located /app/assets/myfunc/createAdView.js

Titanium.Admob = require('ti.admob');

function createAdView(){
    var adview = Titanium.Admob.createView({
        width:  320,
        height: 50,
        bottom: 0,
        adBackgroundColor:'black',
        publisherId: '************'
    });
    return adview;
}

I include this script like this in 

Ti.include('myfunc/createAdView.js');
//
//
//
$.byFavWin.add(createAdView());

Somehow, it shows error only on Android

  • Titanium.Admob = require('ti.admob');

    NO NO NO NO NO NO NO NO.

    DO NOT EXTEND THE TI NAMESPACE.

    iOS Docs:

    var Admob = require('ti.admob');

    Android Docs:

    var Admob = require('ti.admob');
