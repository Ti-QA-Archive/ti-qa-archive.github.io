How to change Emulator on new Ti SDK

I am using latest Ti Studio (3.2.0.2…) with Ti SDK 3.2.0.GA and trying to run my App on Android Emulators (platform 4.4.2 , API level 19) the same dimensions and dpi as Galaxy S2 and S3 (if S2 is not supported then I am ok with S3 and S4).

When I first clicked run with android, it opened a WVGA800 Emulator so since it is not what I need I had to find a way to change it. I opened the preferences and opened the AVD Edit. I changed it to XVGA854 (S2) and WXGA720 (S3) but the Emulator starts (most of the times at least) and the App fails to install or sometimes it says that it is installed and it is trying to run, but it doesn't show on the Emulator.

I have been trying all day to run it on S3's dimensions and dpi but it didn't work. Am I doing something wrong?
