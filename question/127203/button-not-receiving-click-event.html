Button not receiving click event

I have a custom table row, wherein if a variable value is true I want to include a button. Code is below. The button is added if the condition id true, I can see the image I am using as a background image. However, the click does not occur (the open url nor the info trace fires). What am I missing?


    // sponsored by
    var sponsLink = sponsURL;
    if(sponsIMG != '' || sponsIMG != null) {

        var bSponsor = Ti.UI.createButton({
            width : 150,
            height : 60,
            right : 0,
            top : 5,
            backgroundImage : 'images/' + sponsIMG
        });

        vExtras.add(bSponsor);

        bSponsor.addEventListener('click', function(e) {
            Ti.Platform.openURL(sponsLink);
            Ti.API.info('opening '+sponsLink)
        });
    }

sdk 1.7.3, os 10.7.2, ios 5

  • Have you tried simply making the 'button' an image view instead. Within the layout, the element looks more like an image than a button anyhow. You can still capture the click event on an image view as well. It just seems that the use of an image view would be more appropriate than a button and would also not suffer from any nested view issues.

  • Try looking at the Kitchen Sink tables and how they handle click events. As Anthony above said, it's easier/better to use a simple image view. Then you attach your listener to the entire table and trace what was clicked from the "source" variable. It's all in their demos.
