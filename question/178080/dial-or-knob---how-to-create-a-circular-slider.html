  • Yes, what I did in the end was to setup a webview on my window and have a instance of http://anthonyterrien.com/knob/ I then passed the value to the window and manipulate any assets that you need as you would normally do. It works quite well. I only use Knob in the background to calculate the rotation and value, is not shown on my window as all the element displayed are Titanium elements. You can see a demo at www.goph.recipes
