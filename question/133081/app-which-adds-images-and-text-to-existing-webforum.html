App which adds images and text to existing webforum

We have a webforum where you can add photos and ask "what plant is this" or "what kind of butterfly is this". We want an app where you can take pictures with your smartphone and upload picture and questions to the forum. It must be able to write to the MYsql-database at our server and add the image file to a specific folder on our server. Before we start I like to know if this is possible with Titanium?

  • Sure, this can be done – I currently have an app in the iOS App Store that does something very similar that I built with Titanium. The KitchenSink demo app contains an example of how to use the Titanium.Media API to capture images and the Titanium.Network API can be used to upload those images to a server.
