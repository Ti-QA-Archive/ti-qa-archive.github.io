    • Hi Jorge,

      I have finally made other choice.

      But if you still want to use a ScrollView, use instead a table view with the property selectionStyle:Titanium.UI.iPhone.TableViewCellSelectionStyle.NONE. Then, add one row to the tableview and put all your content in it. Thus, it will work like a ScrollView.
