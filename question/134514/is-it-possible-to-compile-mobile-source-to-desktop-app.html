Is it possible to compile mobile source to desktop app?

As u know Titanium is cross-platform, and I have coded some source that can execute on Android(almost is written by javascript).

Now I want to use these source to make win32-exe app, is it possible/is it easy?

Also I want to know the difference between mobile app & desktop app that written by Titanium, or there if is a way to "write once, apply everywhere"?

    The short answer is no, you can't.

    The reason for this is due to to the fact that Ti Mobile and TiDE (the new Name for Titanium Desktop) are very different in terms of structure and API functionality.

    Ti Mobile creates a bridge between your JavaScript code an the Native widgets from the mobile platform.

    TiDE provides you with what you could call a skinless Browser that hosts your them and JavaScript files. It also provides you with additional connectors to access some native features. Such as File system, Database, Notification, JSON, logs, etc.)
