    Have a look at ScrollView:

    **Android
    On Android, a scroll view can only scroll in one direction, either vertical or horizontal, and not both at the same time. Hence, the scrollType property may be used to set the scroll direction explicitly.

    If the scrollType property is not defined, the scroll view attempts to deduce the scroll direction based on some of the other properties that have been set. Specifically, if contentWidth and width are both set and are equal to each other, or if they are both set and showVerticalScrollIndicator is set to true, then the scroll direction is set to "vertical". If contentHeight and height are both set and are equal, or if they are both set and showHorizontalScrollIndicator is set to true, then the scroll direction is set to "horizontal". If scrollType is set, it overrides the deduced setting.**

  • Yes, on On Android, a scroll view can only scroll in one direction, either vertical or horizontal, and not both at the same time. Hence, the scrollType property may be used to set the scroll direction explicitly.

    But on iphone, you can scroll in both direction at the same time.For scrolling the view, the 4 properties (width,height,contentWidth,ContentHeight) should set properly. Refer sample:

    var win = Ti.UI.createWindow({
    backgroundColor : 'white'
    });
    var parentView = Ti.UI.createView({
    width: '100%',
    height:'100%'
    });
    var scrollView = Ti.UI.createScrollView({

    showVerticalScrollIndicator : true,
width: '250',
height: '200',
contentHeight: 'auto',
showHorizontalScrollIndicator : true,    
contentWidth: '350'

    });

    var data = [
    Ti.UI.createTableViewRow({title:'Title 1', header:'Header 1' }),
    Ti.UI.createTableViewRow({title:"sample data…………Test…………"}),
    Ti.UI.createTableViewRow({title:'Title 2', header:'Header 2'}),
    Ti.UI.createTableViewRow({title:"sample data 2…", height:100}),
    Ti.UI.createTableViewRow({title:'Click Me!', header:'Header 3'})
    ];

    var tableView = Ti.UI.createTableView({
    data: data,
    width:'100%',
    height:'100%'
    });
    scrollView.add(tableView);
    parentView.add(scrollView);
    win.add(parentView);
    win.open();
