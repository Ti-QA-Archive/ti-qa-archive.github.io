Resizing remote images to pre-defined dimensions

Hi all,

I'm currently trying to build a tableview with remote images. Each image displayed in the table needs to have the same dimensions. However, since the original remote images are of different sizes this task is proving to be difficult. I'm certain there is a simple fix, but i somehow am just not able to implement it.

I tried using a parent view for the image but that doesn't seem to do the trick. I next tried resizing the images once they were loaded using an event listener. This seem to fix the issue with the image sizes but the performance gets hammered with the app lagging terribly even on the simulator.

    i have run into this problem and in the end, the best solution is to have thumbnails created on the server since loading the images into the the tableView and then attempting to resize them is going to provide for a bad user experience.

    with that being said, you can look at these solutions

    https://github.com/appcelerator/titanium_mobile/blob/master/iphone/Classes/TiBlob.m#L305

    http://developer.appcelerator.com/question/118022/resize-photo-from-camera-before-save

    http://developer.appcelerator.com/question/13851/can-photos-be-resized-on-the-device
