    Firstly, it is bad form to ask the same question twice.

    Secondly, I imagine the reason people have not replied to any of your recent questions is because they are too broad and scream "Please write my app for me!". These aren't really specific help questions and truthfully, it seems a little like you want to be spoon fed the answers. A previous question asked about documentation for iOS/Android…. Nobody is going to spend their time handholding you through 'How to create a flipboard type UI' or a 'Fully workable bluetooth sample project' or 'build a Pinterest UI'.

    Titanium is a programming… All of the things you want to do are achievable. If you want to develop apps, then welcome and start learning. If you don't want to dedicate your time, then don't expect others to dedicate theirs.
