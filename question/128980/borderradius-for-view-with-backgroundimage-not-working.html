borderRadius for view with backgroundImage not working

I have the following code that displays backgroundImage in a view. If I want to have borderRadius property for view, it does not display the image.

var imgLogo = Titanium.UI.createView({
    backgroundImage:image,
    backgroundColor:'transparent',
    borderRadius:5.0,
    borderColor:'#eee',
    borderWidth:2.0,
    height:'70dp',
    width:'90dp',
    left:'5dp',
    top:'5dp'
});

I tried using imageView too but was not successful.

    You didn't specify your platform, but the behavior you describe is a known deficiency for Android. Fortunately the Appcelerator platform folks are aware of it.

    As a workaround, you should use an ImageView instead of a View, and set the backgroundColor to the same color as the view behind it.

    Here is a slightly modified version of your code to demonstrate:

    var win = Ti.UI.createWindow({
    backgroundColor : '#357'
});
var imgLogo = Ti.UI.createImageView({
    image : image,
    backgroundColor : '#357', // Match the parent View
    borderRadius : 5.0,
    borderColor : '#eee',
    borderWidth : 2.0,
    height : '70dp',
    width : '90dp',
    right : '5dp',
    top : '5dp'
});
win.add(imgLogo);
