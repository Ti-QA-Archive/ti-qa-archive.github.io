add a view to a window when a button in another view is clicked

window win1 contains
view1, view2 as direct children.
in view2 I have a child that when clicked, I want it to create a new view (view3) that will be added as a field to win1 (so it will be displayed over the other 2 views).

How can I get access to win1 object? what is the best way of accomplishing this? Do you recommend a different practice?

  • you can remember a reference to to view object.

    var v1 = Ti.Ti.createView();
var v2 = Ti.Ti.createView();
v1.add(v2);
v1.foo = {};
v1.foo.v2 = v2;

    Now you cann access v2 over v1.foo.v2
