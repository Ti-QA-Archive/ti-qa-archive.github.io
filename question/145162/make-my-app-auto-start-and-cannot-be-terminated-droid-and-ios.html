Make My APP Auto Start and Cannot be Terminated? (Droid and iOS)

Hello, we are working on a project where we buy the devices for the customer, and want it to only run our app. We install the App before shipping the devices, and want to keep them in our APP only.

Anyone know how to achieve this? I know I can disable many of the android:* events, however, I need to AUTO launch, and not allow exit - and if the APP crashes for some reason, recover.

  • Android doesn't provide you with a Kiosk Mode out of the box I'm afraid.

  • For iphone and ipad, there is a way to put it into 'Store Demo Mode' where the home button, all gestures, and the notification center are disabled. There are a number of online tutorials that show you how to upload a mobile config file to a phone in order to do this. There is no way to auto-launch an app on startup though.

    For Android, there are a couple software installs that I've seen that allow you to run in Kiosk Mode.
