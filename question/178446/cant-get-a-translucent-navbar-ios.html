Can't get a translucent NavBar iOS

I'm using 3.4.0.GA on iOS8 and can simply not get a translucent nav bar! Have been pulling my hair out over this.

My view:

<Alloy>
    <Tab id="tab_places">
        <Window id="window_places" title="Places">
            <TableView id="table_places">
            </TableView>
        </Window>
    </Tab>
</Alloy>

My tss file:

"Window": {
    tintColor: "#34aadc",
    barColor: "#99FFFFFF",
    extendEdges:[Ti.UI.EXTEND_EDGE_TOP],
    autoAdjustScrollViewInsets: true,
    includeOpaqueBars: true,
    translucent:true
}

I am aiming for a translucent white navbar where i can see the table rows as a user scrolls, but i currently get a solid white bar.

The way may app is set up, i include each tab into the index file: 

<TabGroup id="navTabs"><Require src="places" /></TabGroup>

I have tried changing the windows background to see if that shows through, but still get a solid white bar.

Any help is greatly appreciated on this!

  • Hi Jamie,

    the trick is to set an 1x1 px transparent image as barImage and hide the shadow of the navBar

    tss

    "Window[platform=ios]": {
    statusBarStyle: Ti.UI.iPhone.StatusBar.LIGHT_CONTENT,
    barColor: 'transparent',
    backgroundImage:"images/start.jpg",
    navTintColor: '#fff',
    extendEdges: [
        Ti.UI.EXTEND_EDGE_TOP
    ],
    title:"Title",
    backgroundColor:'white',
    barImage:"images/transparent2.png", // <-- important
    hideShadow:true,
},
