How to display a list of content (text)

I have e.g. 10 restaurant tips where each tip is about 2 sentences long.
This is too long to use as a title in a tableviewrow, so I'm looking for a way to display these items in Titanium.

What's the best way to produce such a list? I can't predict how long a tip is so setting height of items might be difficult

    • Create label and then add the label to the row. Leave the title blank on the row.

      What you also must remember is you may need to change the selectedColor and selectedBackground color on the rows. When a title is used, the selectedColor (selected text color) changes to white as the background changes to blue. This will not be the case for the label. So you must either programatically change the colors, or my favorite solution, is change the selectedBackgroundColor of the row to a lighter shade of whatever color you want to use, so that way Black text still looks good on it.
