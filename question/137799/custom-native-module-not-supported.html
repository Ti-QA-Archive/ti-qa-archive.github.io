Custom native module not supported

Hello,

I developed a custom native iOS module which I want to use in my Titanium mobile project. I tested my module using Titanium run in console. Everything worked fine. Building my module with the build.py script did not throw any errors as well. I followed all steps in this guide Appcelerator Wiki. When I want to include my module in my project I can not find it. I mean I click the plus-icon in TiApp.xml and I can not see my module. When I modify the xml-file using

<modules>
<module version="0.1">app.id</module>
</modules>

Titanium Studio tells me the module is not supported by Titanium SDK 2.0.2.GA. I build the module with Titanium SDK 2.0.1.GA2 and set the minimum SDK to this version as well. The path to the titanium command line command is set to the 2.0.1.GA2 Path. When I create a new unmodified module it does work as well in Titanium Studio.

What am I missing that it does not work?

Martin
