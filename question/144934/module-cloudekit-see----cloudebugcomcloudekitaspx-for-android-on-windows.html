Module Cloudekit (see --> cloudebug.com/cloudekit.aspx) for Android on Windows

Applications Type: mobile, Target: Android 2.2 and newer

Titanium SDK: see screenshots

Devices: Android Emulator (tested many versions) and Android 4.0.2 tablet

Host-OS: Windows7 64

#Description

[START] —————————————————-

After using this great module successfully on iOS now we have to build the Android-Version of our app. CloudeKit supports Android – but so far, we are not able to get it up and running. We tried it installing globally and at the projects modules directory. Tried it in all combinations on SDK 2.0.1.GA2 and more recent with Runtime V8 and Rhino.

Maybe anyone has the Android Version working in his own project and can help us.

[END] —————————————————-
