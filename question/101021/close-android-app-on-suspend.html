Close android app on suspend

On the iPhone I was able to add a line in the info.plist to have the app exit on suspend so it never went to the background, is there a way to do this same thing on the android?

If a user is in my app and they hit the home button, I don't want it to pause or go to the background, I want it to exit the app so the next time they launch it, it will start fresh.

    <key>UIApplicationExitsOnSuspend</key>

  • Take a look here, but just for the "back button" you can kill the app.

    exitOnClose boolean
    (Android only.) Boolean indicates if the application should exit when the Android back button is pressed while the window is being shown. You can only set this as a createWindow({…}) option. Setting it after window creation will no effect.
