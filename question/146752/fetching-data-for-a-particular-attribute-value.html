Fetching Data for a particular attribute value

Hello Everone,

I am facing one issue,When I am trying to fetch data for a Particular value of a attribute.So I am trying

var categoryData = Alloy.createCollection('category');
        alert(categoryData.fetch({name:'baseball'}));
        categoryData.map(function(category) {
            var newsUrl = category.get('newsUrl');
            var headlineUrl = category.get('headlineUrl');
            alert("value of newsUrl is "+newsUrl+"value of headlineUrl is "+headlineUrl);
        })

I have model with name category,and want to get all the rows from the table that satisfies the criteria means fro which name is basketball.

The above code is giving me full data,not the conditional data that i want

Can anyone help Please???

Thanks

        categoryData.fetch();
