  • set the default sdk

    From command line, run the following and select 3.2.0.GA as your default sdk

    ti sdk select

    edit your tiapp.xml

    Open up your tiapp.xml for each older project, and change 

    <sdk-version>3.1.3.GA</sdk-version>

    to

    <sdk-version>3.2.0.GA</sdk-version>
