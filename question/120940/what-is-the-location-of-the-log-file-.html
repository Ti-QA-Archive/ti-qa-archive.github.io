  • iOS or Android? Mac or Windows?

    It is usually found in the 'Documents' folder within an app, which can be found locally (or on the device) by getting the full path to Ti.Filesystem.applicationDataDirectory (which is the Documents folder inside an iOS app) - the log file is stored there.

    It will usually be somewhere like (for the iOS simulator on OSX) at ~/Library/Application Support/iPhone Simulator/[iOS version]/Applications/[UNIQUE_APPLICATION-HASH]/Documents/

    hope this helps.
