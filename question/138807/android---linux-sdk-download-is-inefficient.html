  • Titanium Studio is only going to download a version of the SDK if you don't point it to one initially.

    So, the process is either:

    • Download Android SDK
    • Open Titanium Studio, start configuring Android and point to the existing Android SDK

    or

    • Download Titanium Studio
    • Point to a folder on disk, and it will download the Android SDK into that location.

    There is no hard-coded check inside Titanium Studio for a version of the Android SDK. There is only a link to download a known good version if the user doesn't have one installed. That link is updated regularly, but Google has rolled out a number of SDK releases very quickly.
