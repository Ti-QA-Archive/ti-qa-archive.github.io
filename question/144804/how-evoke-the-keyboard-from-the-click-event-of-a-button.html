How evoke the keyboard from the click event of a button

Hi, there

I want to build a feature that when the user clicks a button, a keyboard would then show up. And on the top of the keyboard, there is a toolbar that contains a textfield and two other buttons.

I don't know how to directly evoke the keyboard from the click event of the button. Do you have any idea how to make the keyboard popup?

Thanks!
Best,
Phil

    If I am not mistaken you want the textfield and some buttons to appear after you click on a button that opens the keyboard.

    So, what you have to do is:

    • create a button (the button that opens the keyboard and the textfield with the two buttons)
    • create a textfield and the other two buttons and make them hidden
    • create an event listener for the first button, that checks 'click' and makes the textbox and the two buttons hidden=0; and textfield.focus();

    That will do the job.

    I hope I helped.
