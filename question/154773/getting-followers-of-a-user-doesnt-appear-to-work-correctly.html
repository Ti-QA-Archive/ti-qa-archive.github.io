Getting followers of a user doesn't appear to work correctly

I'm using ACS to manage friends in my app, and I'm trying to get the followers of a user. For the sake of this example, consider the following three users:

Jack (friends with Sue)
Sue (friends with Jack)
John (friends with Sue)

In this setup, Sue should have two followers, and be following one person.

If I use the following query to ACS, I only get one "friend" back, whereas my understanding is passing the followers=true flag should return two friends instead of the one.

The call I'm using is: 


Cloud.Friends.search({
    user_id: currentUser.id, //let's pretend this is Sue's ID
    followers: true
    },
    function(e){
        if(e.success){
            console.log(e);
        }
        else {
            console.log("error");
            console.log(e);
        }
    });

Seems like I get the same response whether or not I pass the followers=true flag in that query. Am I setting this call up wrong, or does this seem correct and a possible bug?

Thanks!

Justin

    It's correct if you set this as one way (as twitter). Since Sue is only following one person, there is just one person there.

    If you set as two way (as facebook), you will get two person as friends.
