  • There is an example here from the documentation on how to GET data from a web server and if you view the this.responseText in the onload function, you will see your data represented as a string.

    You then just need to convert this string to a JSON object as follows:

    var myObject = JSON.parse(this.responseText);

    Once you have the data in an object, you can access the properties and assign it to an Alloy object using:

    $.myLabel.text = myObject.content;

    Again, all this is described in this.

    You need to use the online documentation more which is quite good for beginners and for more difficult code, try a good search engine.

    Hope this helps
