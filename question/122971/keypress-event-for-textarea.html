Keypress event for TextArea

Is there a way to get/filter keypress events for a TextArea?

For instance, on an iPad with an external Bluetooth keyboard, I need to know when the Tab key is pressed by a user.

— asked July 21st 2011 by K T

    There are no keypress events for the TextArea, however you can listen for the change event and pull the last character off of the string in the change.
