A bit of clarity about the implementation of the Google Maps API

Hi,
I'm trying to understand what I have to do in order to implement a native map into my application.
The documentation says that "Google Maps API v1" aren't supported anymore and that I need a Google key and the "Google Maps v2 add-on module".
Reading the Google documentation about the Key I read that "The Google Maps Javascript API Version 2 has been officially deprecated" and "The Google Maps JavaScript API v3 does not require an API key to function correctly" [check this link given by the Titanium Documentation ).

Now the question: "Shall I still use the "Google Maps v2 add-on module" and can I omit the Google Key ?"

Thank you in advance,

Luca

    Hi

    You were right to ask the question as you have fallen into the trap that Google has unintentionally created.

    Your information is based on the web based implementation there is a different but similar scenario for native Android applications.

    The correct version to use for your Titanium app is Google Maps API v2 and you DO require the API key or it will not work.

    The internal Ti.Map is not the correct one, instead you use the module provided with Titanium.

    • Modules.Map
    • Google Maps v2 for Android

    Follow the instructions carefully in the two links above and you will be running maps in 10 minutes. Keep in mind you cannot test maps on the emulator you HAVE to test on a physical device.

    If your map view starts but you cannot see any actual map tiles then you did not correctly set up or enable the Google APIs key for Android Maps v2 for devices. Tweak that and they will appear.

    It may seem a bit fiddly, but as long as you take your time it will all come together.

    • If you are using the latest version of the Ti SDK, 3.1.1 (or the last couple of versions) it is now built in.

      • In Titanium Studio locate your Project
      • Expand the tree view until you see the tiapp.xml listed
      • Double click to edit the file
      • In the visual editor (should be the first one to appear, locate the modules section (top right)
      • Click the add button
      • Choose the ti.map
      • Confirm and save the tiapp.xml
      • Project > Clean

      Have fun
