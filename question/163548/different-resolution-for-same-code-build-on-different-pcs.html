different resolution for same code build on different pc's

I am building an app for my client , when i build it on my system and test it , ( i am testing on two different phones), the resolution looks great and works fine, but when i import the code i have written in a machine which my client has(inside titanium the resolution gets changed , to something really large and ugly). what i am amazed about is the fact that when i build the same code on my machine and install that app on my clients phone the app again looks good as desired. is there any system specific setting or some thing that needs to be changed in the code i am developing only for android.

    Actually, there is no system that you're told. For this problem you must test your apps in different devices, like tab or phone in different sizes. But sometime you have to avoid Absolute measurement. you can use Relative measurements. For example
    You can use

    var view = Ti.UI.createView({

top: '10mm', // avoid this
left: '5px', //avoid this
width: '30%', // use this
height: 50 //avoid this
});
