Chart Module - createBarPlot - How to change each bars fill color with fillColor?

Is it possible to have different fill colors for each bar?

Does anyone have an example, can't find anything about this in the documentation.

I can't believe this feature is missing! , hope its just poor documentation.

barPlot = demoChart.createBarPlot({
        name:'bar plot',
        lineColor:'grey',
        lineWidth:1.0,
        fillColor:'green',

        barDirection:Charts.DIRECTION_VERTICAL,
        barWidth:0.75,
        barOffset:0.5,
        barCornerRadius:2.0,
        labels:{
            offset:1.0,
            angle:0.0,
            color:'blue',
            font:{ fontFamily:'Helvetica', fontSize:12 }
        },
        data:[ 1, 4, 2, 3, 7, 6, 7 ]
    });

fillColors:['green','blue','red','green','blue','red','green']
