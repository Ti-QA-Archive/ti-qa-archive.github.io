Error Titanium SDK 3.3.0

i recently updated to new version and i created a emulator (nexus one with 4.4.2)
but when i run the aplications throws this error:
[ERROR] : Invalid selected Titanium SDK "3.2.3.GA"
but when in console
Ti.sdk.config returns me 3.3.0
i dont know what to do

    Two things to check:

    ti sdk select – make sure to select 3.3.0.GA

    Then, open your project's tiapp.xml. Make sure it is also set to use 3.3.0.GA.

    They don't technically have to match, but you would need both installed. If you were to use 3.3.0 for your selected SDK, and put 3.2.3 in your tiapp, the flow would go like this: the 3.3.0 SDK would be used to parse your tiapp.xml and locate the base project resources, then, control would be passed off to 3.2.3 for the remainder of the build.
