How to get the values from all the text fields in a table view

I have a table view which every row of it contains two text fields and a button under the table view.

How can I get all the values of the text fields from every row at once and put it inside an array.

    You need to loop through table view data and push each and every value of textfield in array.

    Here is sample code,

    
var txtArr = [];

for(var i=0, len=tableview.data.length; i < len; i++) {
    for(var j=0, jlen=tableview.data[i].rows.length; jlen < len; j++) {
        txtArr.push(tableview.data[i].rows[j].children[0].value); // 1st Textfield Value
        txtArr.push(tableview.data[i].rows[j].children[1].value); // 2nd Textfield Value
    }
}

    Note - Assumption in above code is your row contain two textfield as first two childrens.

    Hope this helps.
