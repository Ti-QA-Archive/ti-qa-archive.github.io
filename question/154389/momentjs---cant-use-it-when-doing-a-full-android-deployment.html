momentJS - cant use it when doing a full Android deployment?

I use momentJS in my app. It works when you debug on device etc..

When I try to deploy for the Google Play store through studio, even this minify javascript turned off the app will not work when I try to use it.

The only way I can deploy and use momentJS together is with this property in tiapp.xml:

<property name="ti.android.compilejs" type="bool">false</property>

The problem is I don't want to leave the JS uncompiled in my .apk, how did anyone else get around this?
