  • Titanium SDK Home:

    C:\Documents and Settings\[USER]\Application Data\Titanium

    And the structure of that directory:

    Blackberry SDK Home:

    C:\Program Files\Appcelerator\Titanium Studio

    Blackberry Keys Directory:

    C:\Documents and Settings\[USER]\My Documents\BBKeys
