Modal Window - blur will not hide the keyboard for a textfield

I have a iPad app with a Modal Window - blurring a textfield in the modal will not hide the keyboard. I've seen other people complaining about this with no answers. Has anyone else experienced this? Are there any workarounds?

  • i have the following code and is working fine

    
var win = Ti.UI.createWindow({
    modal: true
});

var passTF = Ti.UI.createTextField({
    width: 400,
    height: 50,
});

win.add(passTF);

win.addEventListener('click', function() {
    passTF.blur();
})

    using sdk 1.8.2
