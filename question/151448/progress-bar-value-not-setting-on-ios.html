Progress Bar Value Not Setting on iOS

Hey Everyone, I am trying to put a progress bar into my app. I have this code in the Alloy XML for it.

<ProgressBar id="pb" min="0" max="100" value="10" />

This code takes a string from a JSON data set, parses it to an integer, and sets the value of that progress bar to it.

//Let Me Know What You're Setting It To
Ti.API.info("Setting progress bar to " + parseInt(args.percent));
//Set the Progress Bar Value
$.pb.value = (parseInt(args.percent));
//Show the Progress Bar
$.pb.show();
//Spit Out the Progress Bar's Value
Ti.API.info($.pb.value + " set");

Console Output:

[INFO] :   Setting progress bar to 53
[INFO] :   53 set

In Android, it works fine. In iOS, the bar always looks 100% full. Why is this? Thank you.

    On iPhone you'll need to divide by 100

    $.pb.value = (parseInt(args.percent)/100);

    so a workaround would be to set max to 1 instead of 100 and divide the value by 100, this way you'll get it working on both Android and iPhone
