  • Accepted Answer

    • Download Titanium Studio.
    • Configure the project for iphone and ipad - this should give you a universal app
    • In your code use conditionals based on Titanium.Platform.model to detect if it's an ipad or an iphone and based on this use specific graphics, interface elements and dimensions.

  • There are 2 ways to do this.

    1. Create an iPhone app. The app will be available on the iPad as an iPhone app. You will be viewed the app as an iPhone on your ipad (with the possibility to zoom it to get a bigger 'iPhone'). Here's a screenshot of it.

    2. Create an app specific for the iPad. If you do this you can use the specific interfaces for the iPad (such as the split screen). If you have created an iPhone app you can re-use most of the code, and modify some parts of your code to make it specific for the iPad.

    So with the second option you can start from scratch, or you can make an iPad project based on the iPhone project, and edit it.

    Hope this info is useful!
