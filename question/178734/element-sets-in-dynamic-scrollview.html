Element Sets in Dynamic ScrollView

I used to have a ScrollView in xml structure like this:

<ScrollView layout='vertical'...>

<View ...>
   <Label ... />
</View>
<View layout='horizontal'...>
... some nice gridview with icon & labels ...
</View>

<View ...>
   <Label ... />
</View>
<View layout='horizontal'...>
... some nice gridview with icon & labels ...
</View>

</ScrollView>

The idea is to have sections of gridview menu. It's all fine in xml but I need to make it dynamic by reading some JSON definitions. I leave the xml to have one empty <ScrollView id='homeScrollView' layout='vertical' height='100%' … /> as container.

My loop looks like this:

function buildMenu() {
    var elements = [];
    for section ... {
        var sectionTitle ...
        var titleText ...
        sectionTitle.add(titleText) ...

        var sectionGrids ...
        for items ... {
            var gridItem ...
            var iconLabel ...
            var textLabel ...
            gridItem.add(iconLabel);
            gridItem.add(textLabel);
            sectionGrids.add(gridItem);
        }
        elements.push(sectionTitle,sectionGrids);
    }
    $.homeScrollView.add(elements);    
}

However no matter how I try, I got the sections overlay something like this:

Title of 1st section followed by it's grids
Title of 2nd section followed by it's grids but stacking below first title

It seems that each section is good on it's own but adding 2nd section stacks instead of expected vertical layout . Is there any secret to add multiple elements to a vertical ScrollView dynamically? I also tried creating the ScrollView in JS but same result.

  • Ah Ha … finally, it turns out that I used:

            var sectionGrids = Ti.UI.createView({
            layout: 'horizontal',
            height: 'Ti.UI.SIZE',
        });

    where Ti.UI.SIZE should NOT be quoted ….
