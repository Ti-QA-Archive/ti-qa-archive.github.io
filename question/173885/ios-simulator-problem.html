IOS Simulator Problem

Please help figure why I cant launch my IOS simulator after updated the XCODE and TI. The error below:

[INFO] : Launching application in iOS Simulator
[INFO] : Focusing the iOS Simulator
[ERROR] : An error occurred running the iOS Simulator
[ERROR] : dyld: Library not loaded: @rpath/iPhoneSimulatorRemoteClient.framework/Versions/A/iPhoneSimulatorRemoteClient
[ERROR] : Referenced from: /Users/rtan/Library/Application Support/Titanium/mobilesdk/osx/3.1.3.GA/iphone/ios-sim
[ERROR] : Reason: image not found
[ERROR] :
[ERROR] : Project failed to build after 1s 942ms

Am using latest Ti, XCODE 5.1.1

Also after updated my Ti to latest I now cannot open tiapp.xml. The error shows:

Could not open the editor: null argument:

Please help.

  • Accepted Answer

    Go to ~/Library/Application Support/Titanium/mobilesdk/osx/ your latest version if it is 3.2.2.GA or 3.2.3.GA i.e.,
    ~/Library/Application Support/Titanium/mobilesdk/osx/3.2.3.GA/iphone and in that copy ios-sim (Executable file) into 3.1.3.GA (replace with the existing ios-sim file.).

    Clean your build and try again. This should works out.
