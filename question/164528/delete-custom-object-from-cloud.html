  • Hi,
    You can delete custom objects via following way : But only the owner of the object or user who is granted write permission by the object's ACL can delete the object.

    Also an application admin can delete any custom object.

    User login is required to perform the operation.

    Cloud.Objects.remove({
    classname: 'cars',
    id: savedCar1.id
}, function (e) {
    if (e.success) {
        alert('Success');
    } else {
        alert('Error:\n' +
            ((e.error && e.message) || JSON.stringify(e)));
    }
});
