    If thats the data you are getting back from the php file then the json is not valid and thats why is crashing. This is how the json should look like to be valid

    [
    {
        "numero": "1",
        "data_g": "10 gennaio 2012",
        "immagini": "1.jpg"
    },
    {
        "numero": "2",
        "data_g": "25 gennaio 2012",
        "immagini": "2.jpg"
    },
    {
        "numero": "3",
        "data_g": "4 febbraio 2012",
        "immagini": "3.jpg"
    }
]
