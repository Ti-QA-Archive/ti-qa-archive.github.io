  • Hi Paul (and anyone else finding this as I did),

    Apparently it was a problem with latest android tools, and the fix was to downgrade android tools, or as mentioned in comments, upgrade to SDK 3.2.3 Alpha:

    http://www.appcelerator.com/blog/2014/03/updating-to-android-22-6-1-tools/
