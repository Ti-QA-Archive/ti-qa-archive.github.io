Require only once

I need to use scanning module many places across my app. Every time I use it I need to require it. Is there a way to require module only once? And everywhere can share it? (My app is running under multiple context though)

Many thanks

  • you could try requiring it in your "parent" context and passing it as a property to each new context you open. It is used mainly with variables - I also successfully passed objects - but it something that you need to try to see if it works and that the object still function correctly.
