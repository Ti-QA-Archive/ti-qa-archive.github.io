  • At the command line, run titanium info -t ios and at the top it will print out the detected Xcode versions. If no versions are displayed, you may have a bad Xcode install or maybe it's in a location that Titanium can't detect. You can also confirm the Xcode location by running xcode-select --print-path.

    — commented December 18th 2012 by Chris Barber

  • Hy! I'm new with titanium, I'm from Argentina. Where can I put the correct location of xcode so that titanium detect it? TKS!

    — commented December 27th 2012 by Ignacio Rolandelli

  • You can put Xcode in the /Applications folder and it will automatically be detected. Otherwise you can put it anywhere and run:

    sudo xcode-select --switch /path/to/Xcode.app/Contents/Developer
