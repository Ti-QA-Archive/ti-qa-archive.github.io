Ti.Media.Sound -

hi there, is there a way, e.g. an event, to find out when a sound, created with 

Ti.Media.createSound({url: url_from_server});

has loaded, or is currently loading? any ideas? thanks, uno

— asked September 13th 2011 by u no
  • ti.media.sound

  • There is not a loaded event that you can track using the createSound() method, however you can pass the preload property to the object constructor to ensure that the file is loaded when you call the play() method.
