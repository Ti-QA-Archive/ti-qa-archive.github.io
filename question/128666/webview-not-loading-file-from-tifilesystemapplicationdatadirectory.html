Webview not loading file from Ti.Filesystem.applicationDataDirectory

Hi folks,

Is there anyone can help me to let my Webview to read the images/videos from applicationDataDirectory ?

I did this

f = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory,'20111101/media/demo.jpg');
Ti.API.debug f.nativePath

Then i simply copy over the path to my Webview as 

<img data='{"a":"download","t":"image"}' height="308" width="440" src="file://xxxx/Documents/20111101/media/demo.jpg">

But it's not loading at all.

Any clue ?

Thanks
Eric

    Hi Eric

    Try out this If you set the img src in your webview to the following it works in 1.7.0+

    getFile.nativePath.replace('file://', 'app://');
