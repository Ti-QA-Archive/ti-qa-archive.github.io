  • you'll need 2 api keys; one for local development/unsigned and one for the signed with your key store. here are the instructions Obtain and Add a Google API Key

    you'll use the command (put in your sdk version)

    keytool -list -v -keystore ~&#x2F;Library&#x2F;Application\ Support&#x2F;Titanium&#x2F;mobilesdk&#x2F;osx&#x2F;3.1.0.GA&#x2F;android&#x2F;dev_keystore

    to get the sha value for local dev (emulator/build to device)

    then you will use the same

    keytool -list -v -keystore **path to your production key store**

    to get the sha to generate the api key for signed apks
