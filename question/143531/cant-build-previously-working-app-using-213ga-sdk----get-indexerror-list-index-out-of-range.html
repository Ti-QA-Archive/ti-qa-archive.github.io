  • I switched back to the Xcode 4.2 path, and then back to the Xcode 4.5 path like this:

    sudo xcode-select -switch /Applications/Xcode.app/Contents/Developer/

    Then I was able to build, but the simulator crashed before appearing. I followed the advice here:

    http://developer.appcelerator.com/question/8501/application-has-exited-from-simulator

    Now the app runs in the simulator from Xcode, but still not from Titanium Studio.
