    In the sdk directory, there is an api.jsca that includes Titanium and ACS api references. It would be a starting point. Its designed to be used for content assist though, not full docs. http://docs.appcelerator.com/titanium/latest/#!/guide/JSCA_1.0_Specification

    The documentation on docs.appcelerator.com are generated from the source. Some others have been workign towards a similar end as you. http://stackoverflow.com/questions/13434478/download-documentation-for-appcelerator
