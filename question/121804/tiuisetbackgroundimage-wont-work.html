I'm trying to set a background image for my application (not windows).

My code is simple:

Ti.UI.setBackgroundImage('assets/bg.jpg');

It's the first line of my app.js file.

I triple-checked the url, it's fine.
The jpg file is fine too and I emptied my build folder, nothing works.

  • You need to add at least one window to your app and open it to invoke the setBackgroundImage() method on the root UIView.

    This doesn't work

    
Ti.UI.setBackgroundIMage('assets/bg.jpg');

    But this does

    Titanium.UI.setBackgroundImage('assets/bg.jpg');
var w = Titanium.UI.createWindow();
w.open();

    You could accomplish this initial window by using a tabGroup as well.

  • You can't achieve exactly what you're wanting to, but you can get close. The app backgroundImage, Window1 and window2 would have the backgroundImage set.. When you animate between the two windows, the image will appear visible.

    Or, a better way..

    Instead of using two windows, you can use a view container to hold the contents of both windows.. then animate the container, passing in the resulting view object that you want to have visible. 

    Ti.UI.setBackgroundImage('assets/bg.jpg');
var win = Ti.UI.createWindow({
  backgroundColor: 'transparent',
  url: 'windows/main.js'
});
var container = Ti.UI.createView();
var view1 = Ti.UI.createView();
container.add(view1);

win.add(container);
win.open();

container.animate({
  view:view2,
  duration:500,
  transform:Ti.UI.iPhone.MODAL_TRANSITION_STYLE_FLIP_HORIZONTAL
});

    This will display the background image of the app at all times, assuming your container is sized appropriately.
