'android:back' not closing my window.

Hi,I am developing android application.In my application I need to close one window on click of back button of android device. My code looks like

var win = Ti.UI.currentWindow;
var label1 = Ti.UI.createLabel(
{
    text:'Edit ',
});

var img_win = Titanium.UI.createWindow
({
    backgroundColor:'#000',
    opacity:0.5
});    

label1.addEventListener('click', function(e)
{
        img_win.open();
});

img_win.addEventListener('android:back', function(e) 
{
    img_win.close();    
});

as in code i want to close img_win on android back button click. But it not closing my img_win. It closing my win window. I want to close img_win not win. I am using titanium sdk 2.0.1.GA2. Any solution to solve this problem? Thank you …..

    The very simple solution is that when you open your img_win window you have used img_win.open()
    This code actually creates a light weight window and when you press back button on the light weight window it will close the other window too, if they are light weight window.

    To solve this issue you should open a window in a modal way,this will reate a heavy weight window.
    just do img_win.open({modal:true}).

    Now when your press back on it, It wil only close your img_win window not your main window.

    hope that works.
