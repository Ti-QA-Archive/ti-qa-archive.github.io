  • Hi

    The current iOS maps does not have the option to turn off the compass.

    Ti SDK 3.2 introduces a new Map Module that means Ti.Map will no longer be used. The new version is based on the latest MapKit and introduces a number of new features. However none of these appear to be an option to turn off the compass.

    To check further I have reviewed the MapKit Class in the apple documentation. This does not carry any information about a compass setting as either a property or creation only setting.

    Sorry it does not look like you can turn the compass off.
