Disable Pinch in Webview

I am using a webview to display map on an android application. However since android has a issue with pinch functionality, the application crashes if I user pinch to zoom into map. Is it possible to disable pinch in a web view?

Thanks

Apurva

  • You can try setting the viewport meta tag after you add the WebView:

    myWebView.evalJS('(function(){' +
    'var m = document.createElement("meta");' +
    'm.setAttribute("name", "viewport");' +
    'm.setAttribute("content", "width=device-width,initial-scale=1.0,minimum-scale=1.0,maximum-scale=1.0,user-scalable=no");' +
    'document.getElementsByTagName("head")[0].appendChild(m);' +
'}())');

    I did not test his code, so it may not work. It definitely will not work for cross-domain web pages if you build your app for Mobile Web.
