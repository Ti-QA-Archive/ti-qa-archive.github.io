FTP 500 error - PASSIVE mode required

Dear All,

I have created a simple FTP client to upload files to my FTP server.

var controlSocket = Ti.Network.createTCPSocket({
hostName:'example.com',
port:21,
mode:Titanium.Network.READ_WRITE_MODE
});
controlSocket.addEventListener('read', function(e) {
controlSocket.connect();
};

The essential part of my code is above. It works pretty well, I login, send my password.. no worries, but then I get this 500 error.

Can someone tell me what settings I need to solve this issue? I'm establishing a TCP connection at port 21.

I know that the listening port on the FTP server is 21. However, the return port (from server to client) may not be 21. Perhaps this is the problem?
