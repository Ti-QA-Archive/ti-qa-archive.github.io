  • Its a dynamic url generated from the server..using MaD5Hash implementation..
    You need to post the url & in response you will get the url as per the id provided. & then you can use that url.

    eg id=387829

    you need to get the MD5Hash of ("387829|1|HUNG#25$12") by following way

    var temp = Ti.Utils.md5HexDigest("387829|1|HUNG#25$12");

    The URL that needed to be posted is url = "http://api.hungama.com/streaming/stream.php?"
    parameters that needs to be provided are as below. temp is value of MD5Hash that needs to be implemented as mentioned above

        store_front_id : '1',
    app_id : '287',
    content_id : '387829',
    content_type_id : '1',
    content_sub_type_id : '8',
    format : 'xml',
    auth : ''+temp

    & if it is right you can get url as response & when I am trying to provide the url to the audioplayer I am getting the message as mentioned in the earlier query.
