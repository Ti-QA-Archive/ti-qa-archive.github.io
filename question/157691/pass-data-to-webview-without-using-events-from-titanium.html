pass data to webview without using events from titanium

Hi, is it possible to pass data to a webview without using events? I have a unique webview loaded many times during the lifecycle of the app. Part of the content of the page is updated anytime I load the webview: I use the JQuery load method to load dozens of different HTML forms without creating full HTML pages (in the HTML I only have the forms).

Problem is, being the page dynamic, I need to pass the name of an HTML page that the page contained in the webview must load. Events half work, but I would prefer something different (a get parameter or something similar, if this is possible). Any help is appreciated.
