Fastdev not working for one app...

….all other apps work just fine….until I run the one app, then all other apps have issues too.

The error message I get is "Warning: Fastdev mode disabled Error message 10.0.2.2:52014 - connection refused"

If I run the 1 app that is having fastdev issues, the other 3 will begin to have fastdev issues as well. The fastdev issues go away for the 3 apps if I shut down the emulator and restart it.

Titanium Studio, build: 1.0.6.201110251616

The other 3 apps are using 2.2 Google Api's.

This seemed to start happening after I updated from 1.7.1 to 1.7.3. I've also searched the Q&A forum for my issue. It doesn't seem that the problem is that fastdev isn't running, it just seems to lose connection to the emulator.
