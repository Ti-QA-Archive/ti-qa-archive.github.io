[ERROR] Application Installer abnormal process termination. Process exit value was 1
[ERROR] :  Failed to package application:

and the only thing i did was install android's sdks

    This was fixed in SDK 3.4.1 – it's an incompatibility between our tools and the newer Android build tools. Upgrade to the newer SDK or downgrade your Android build tools versions.
