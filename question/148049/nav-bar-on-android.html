Nav bar on Android

I've developed an app for iPhone and are now making sure it works on Android as well, but i've run in to a small issue. The nav bar holds two buttons and a background image. On Android, those two buttons doesn't render. I understand that it doesn't exists any navigation buttons in the nav bar on Android, but i still want them. Sure i could just create a view and present it, but i thought "what would be the best solution that works on both Android and iPhone with the same code and renders the same". And that is my question :)

  • Hi

    The answer to this one is not easy as there is no "one solution fits all".

    Android and iOS interfaces are different and users expect to use apps they are comfortable with. So if you introduce interface elements that are predominately iOS to Android and vice-versa then users will not be as comfortable with the app on their chosen platform.

    One of the main differences between the two platforms is the way in which main navigation is handled.

    • iOS uses a navigation bar and tabs
    • Android
      • Older apps; tabs
      • Newer apps; Action Bars

    For Android the Action Bar is by far the most common (in my opinion) interface element now being used to provide navigation for apps. It was introduced in Android 3.0 and has become a main stay ever since.

    As it was introduced in 3.0 you cannot use it to solve navigation problems for earlier editions. If you are targeting 4.x then no problem, but if you still aim to provide support for 2.2.x and 2.3.x then the Action Bar will not work.

    You can make your own action bar with views, labels and imageViews if your needs are simple. It can end up looking the same but will not have the flexibility of the native control (unless you dedicate lots of time to your own solution). But if all you are after is the basic look then this approach will work for you.

    Android does not slide windows in, each one is brought in or just appears over the top, so a new ActionBar (native or fake) is simple added to each window.

    Check out the documentation for ActionBar.
