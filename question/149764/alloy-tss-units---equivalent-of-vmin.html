Alloy tss units - equivalent of vmin?

One of my favorite new features in css3 is the vmin unit, where 1vmin = 1/100 of the smaller dimension of the viewport (ie in a 800x600 viewport 1vmin == 6px). Is there anything like this in tss? If not is there a straightforward way to implement something like it?

  • You want to add a new unit to sizes in TSS? In the same vein as "dp"? I would say the best way to implement that would be a TSS preprocessor using alloy.jmk files), but the preprocessing would happen at compile time, and you would need to wait til runtime to determine the dimensions of the device. Something like this would probably have to wait until dynamic styling is incorporated into Alloy, which is slated to start development after 1.1.0 is released.

    Then again, the community generally proves very clever with workarounds, so I'd be interested to hear if anyone else has any ideas.
