Emulator version for iOS and/or virtual device for Android

With Titanium Studio you can Run As "iPhone" or "iPad" or "Android" easily, however how do you select which version of the iPhone, iPad or Android you wish to target?

I know with Android Virtual Device Manager you can set up multiple Virtual devices, but it seems that Titanium always uses the profile called "titanium_8_WXGA".

With XCode you can select which version of the iPhone or iPad you wish to use.

How do you do this with Titaniun?

    Hi Matthew

    As you can only run one Android emulator at a time, you must close the emulator and relaunch the app in order for the changes to take effect.
