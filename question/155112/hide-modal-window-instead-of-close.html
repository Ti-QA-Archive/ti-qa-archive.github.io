  • Hi

    Modal windows need to be closed rather than hidden due to their nature.

    I am going to assume your typos (fixed below) were accidental.

    var window = Ti.UI.createWindow({ modal: true }); // here
var btn = Ti.UI.createButton({ text: 'Hide' }); // here

window.add(btn);

btn.addEventListener("click", function(){
    window.hide(); // here
});
window.open();
