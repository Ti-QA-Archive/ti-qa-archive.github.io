What do I do if the Android Screen Resolutions dropdown says "loading..."?

What do I do if I can see the Android SDK versions I have installed, but the available screen resolutions dropdown just says "Loading…"?

— asked June 3rd 2010 by Kevin Whinnery

  • Accepted Answer

    It looks like Developer may be storing some incorrect Android SDK information in its local database if you've changed the location of your Android SDK directory after using the Android SDK from a different location. To resolve the issue, I deleted the appdata directory in ~/Library/Application\ Support/Titanium on OS X and relaunched Developer. I will need to re-import all of my Developer projects, but I am now able to run the Android emulator.
