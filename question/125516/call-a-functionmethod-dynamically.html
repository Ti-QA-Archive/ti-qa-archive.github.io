Call a function/method dynamically

this mainMenuData is used to populate a tableRow within a tableView

var mainMenuData = [{icon:'imgs/m_1.png',mainLb:'BUY OR\RENT',subLb:'Which would be a better fit?',action:'createBuyMenu'}
];

tableView.addEventListener('click',function(e)
{    
    var numRow = e.index;
    var action = [numRow]['action'];
    mortgageApp.ui[action]();

});

I need to call mortgageApp.ui.createBuyMenu();

How should I reference this method in 

mortgageApp.ui[action]();  ?

  • nevermind, got it. some of my code was wrong, this fixed it:

    var page = mainMenuData[e.index]['action'];
eval(mortgageApp.ui[page](e.index));
