  • Creating symbolic link for jarsigner in /usr/bin fixed the issue. So, java, javac and jarsigner should be in /usr/bin. If they don't exist there, create the symbolic links in /usr/bin because they exist in $JAVA_HOME/bin.

    sudo update-alternatives --config java
sudo update-alternatives --config javac
