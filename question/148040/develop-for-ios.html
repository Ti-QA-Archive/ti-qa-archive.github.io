    • This is a little confusing, so I hope this clears it up.

      That clause only means that any app created for the iPhone only must not take measures to stop working on iPad.

      There is nothing wrong with creating a iPhone only app, and iPad only app or a universal app (same code but internally determines what to put where when on the device).

      When an iPhone only app is running on an iPad is simply runs as a "emulated" using the same resolution and dimensions that the would on the phone. With no changes to present the app specifically for the iPad.
