Xcode 4.5 and iOS6

Hi,

I need to install the xcode 4.5 so I can test all my apps. Now that iOS 6 GM seed and Xcode 4.5 GM are available, it seems like a good time to upgrade.

Will this effect Appcelerator?

    You can temporarily switch to Xcode 4.5 using

    sudo xcode-select -switch /Volumes/Xcode/Xcode.app/Contents/Developer/

  • On the Appcelerator side of things:

    1. There are some rendering issues with TabbedBars (it now requires to specify a height of 30 to work like before). This affects current apps and exists up to sdk 3.0.
    2. It seems there are some memory issues (displaying a large number of images causes a crash, sdk 3.0 doesn't cause a crash though, although there are a bunch of odd rendering issues with 3.0).
    3. The current GA release is not able to deploy to a device (sdk 3.0 works fine).

    It would be nice if we saw a patch for items 1 and 3. As I have a feeling #2 is my fault…
