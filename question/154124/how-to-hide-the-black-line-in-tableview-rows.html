How to hide the black line in tableview rows?

Is there a way to hide the black lines in the table view rows? I data array and created buttons inside the table rows and set the table view row background as transparent. But there are lines in the bottom which are not quite adorable. Anyone can suggest please? Thanks.

  • Accepted Answer

    Hi Renato,

    For iphone you have Titanium.UI.iPhone.TableViewSeparatorStyle property.
    And for android have you tried using separatorColor of tableView? To make the line invisible, set this property to transparent, or the same value as the backgroundColor property.

    Meenakshi
