String functions in Android

Hi,
I am using SDK 1.4.2 on a Mac, whenever I use any string function in the Android emulator, I get errors like the simple code snippet below:

TypeError: Cannot find function split in object [object Object].
The same error shows up for all string functions. I tried using the String type instead of 'var' but still get the same error.
Note that the string functions work perfectly on the iPhone simulator.
Please Help.
