Napp Drawer module - How to set the right window for iOS app

Hi All,

I am really struggling with a problem for the last few days and would appreciate if some one can help me.

I am using the Napp drawer module

for developing an iPhone app using Titanium SDK version 3.2.3.GA, Alloy version 1.3.1, iOS SDK: 7.1

My problem is that I am not be being able to set the right drawer window.

My requirement is that by clicking a table view row of the left drawer window, a new navigation window is opened, and only some of these navigation windows will have a sliding menu on the right.

Hence, I don't set the right window at the very beginning while declaring the module.

If I set the right drawer in the very beginning while declaring the module, I am stuck with the first right window that is created and can not change it later.

My code -

main.js - Here, I declare the Napp Drawer module.

var NappDrawerModule = require('dk.napp.drawer');

var leftWindow = Alloy.createController("leftWindow").getView()

var centerWindow = Alloy.createController("category").getView();

var drawer = Alloy.Globals.drawer = NappDrawerModule.createDrawer({
    leftWindow : leftWindow,
    centerWindow : centerWindow,
    closeDrawerGestureMode : NappDrawerModule.CLOSE_MODE_MARGIN,
    openDrawerGestureMode : NappDrawerModule.OPEN_MODE_ALL,
    showShadow : false, //no shadow in iOS7
    leftDrawerWidth : 200,
    rightDrawerWidth : 200
});

// lets open it
drawer.open();

I have tried to set the right navigation drawer in the following ways -

  1. var rightWindow = Alloy.createController("rightWindow").getView();
Alloy.Globals.drawer.setRightWindow(rightWindow);

  2. var rightWindow = Alloy.createController("rightWindow").getView();
Alloy.Globals.drawer.rightWindow = rightWindow;

But I always get an error of the type

'undefined' is not an object (evaluating 'Alloy.Globals.drawer.setRightWindow')";

Please guide as I am not able to find a solution myself.

Thanks in advance.
Bably
