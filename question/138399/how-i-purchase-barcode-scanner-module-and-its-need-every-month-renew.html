    There should be a Buy Now button on the page you linked. In my case it doesn't show up because I already have the module, but it should be near the upper right in your case. If it does not appear, perhaps you need to be logged in.

    Secondly, you need a subscription in any month you need to BUILD the application. Once compiled, that version will run indefinitely. Even if you put in the App Store and it gets downloaded a year from now. However, if / when you wish to make changes to the application and BUILD it again, it will need a valid license at that point in time.
