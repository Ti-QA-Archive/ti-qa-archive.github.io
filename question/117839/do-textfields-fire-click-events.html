Do TextFields fire click events?

I can't seem to get mine to. I'm trying to open a Picker when the user clicks the TextField. Here's the code more or less:

var text = Titanium.UI.createTextField({
height:100,
value:'text',
width:100
});

function doSomething(e){
alert('success');
});

text.addEventListener('click',doSomething);

Any suggestions?

  • You don't need a function. Put the alert in the event listener

    text.addEventListener('click',function(){
alert('clicked');
});
