Sending email attachment in Android

Hi,

I want to send a CSV file as email attachment in Android. I am facing 2 issues.

1> The File write call returns false in Android but true in iOS

2> When I send the file as attachment, the Gmail app opens but I cant see the attachment

Here is the code snippet:

var jsfile = Titanium.Filesystem.getFile(Titanium.Filesystem.externalStorageDirectory,'example.csv');
            if (jsfile)
            {
                             var header = "example text";

                var x = jsfile.write(header); // this returns false

                {
                    var emailDialog = Ti.UI.createEmailDialog();

                    if (!emailDialog.isSupported()) {
                        alert("Not Supported");
                        return;
                    }    

                    emailDialog.subject = "My mail";
                    emailDialog.toRecipients = ['foo@yahoo.com'];
                    emailDialog.ccRecipients = ["foo1@gmail.com'];

                    emailDialog.addAttachment(jsfile);
                    emailDialog.open();// gmail opens but with no attachment
                }
            }

Any pointers?
