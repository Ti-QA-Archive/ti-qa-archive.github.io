Toast notification duration

hi,
i have the next issue:
when my app is fired throws a toast notification with a large text , and don't have a duration required
ive tried with miliseconds:

40 seconds

    Ti.UI.createNotification({duration:40000,message:'blablabla'}).show();

android's constant

    Ti.UI.createNotification({duration:Titanium.UI.NOTIFICATION_DURATION_LONG,message:'blablabla'}).show();

but in the second way only appears about 10 seconds.
what im doing wrong?
thanks in advance

    This is described in the docs: Titanium.UI.Notification.duration

    Only long and short is possible. Short is default, so the only value that would make sense to set the duration is Titanium.UI.NOTIFICATION_DURATION_LONG

    If you create Toasts with large text, you are doing it wrong. If you want the user to read this then you should consider using a Label, a Dialog, a Notification or some custom component. Whatever makes sense for your use-case / design.
