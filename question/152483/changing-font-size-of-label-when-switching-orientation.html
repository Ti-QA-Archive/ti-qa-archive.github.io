Changing font size of label when switching orientation

What's the easiest way to change the font size when switching orientation?
I'd like to maintain a layout when switching…

  • I just found the answer myself: 

    // "content" is the container of the text, in my case a label created with "Ti.UI.createLabel"
// tested on iOS
// if switching orientation modes, change font size

    Ti.Gesture.addEventListener('orientationchange', function(e){
        if(e.orientation == "3" || e.orientation == "4") {
            content.font = {
                fontSize:'16dp',
                fontFamily:'monospace'
            };
        }
        else {
            content.font = {
                fontSize:'11dp',
                fontFamily:'monospace'
            };
        }
    });
