Extract data from Twitter search tweet result

Hello all

I have succeed in getting the result from the Tweet search API 1.1

Here's the link : https://dev.twitter.com/docs/api/1.1/get/search/tweets
follow the link and go to the "Example Request GET"

I have got the result but I don't know how to get the different values.

For example, how can I get all the nickname of the tweet author ?

Thank you very much for helping me.

    you can parse it as a JSON:

    var myNewObject = JSON.parse(myObjectString);

    where myObjectString is the result of the http request.

    then you can simply treat it as a dictionary:

    var authors = [];
for (var a in myObjectString.statuses) {
    currStatus = myObjectString.statuses[a];
    var author = {name:currStatus.user.name, screen_name:currStatus.user. screen_name};
    authors.push(author);
}
