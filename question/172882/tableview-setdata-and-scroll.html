TableView setData and Scroll.

Hi!

I'm using Alloy, latest Ti SDK build, developing for iPhone.

The case is in dynamically set data (the list of rows) to the TableView and scroll it to the end (to the last index). The thing I've found in docs is the following TableView method:

setData( data, animation )

and the animation is the one I need:

animation : TableViewAnimationProperties
Animation properties. (iOS only.)

But the problem is that it doesn't work properly (messages.js):

var rows = [...]; // Array of TableViewRows objects inside it.
$.list.setData(rows, {
    position:         Titanium.UI.iPhone.TableViewScrollPosition.BOTTOM
});

The template is (messages.xml):

<TableView id="list" />

It is simply shows me the list of rows without scrolling it to the BOTTOM.

How could I make my plan happens?
Appreciate any help!

  • Hi,

    Please you can use the following approach.

    index.xml

    <Alloy>
  <Window id="win" >
        <TableView id="list" />
  </Window>
</Alloy>

    index.js

    var win = $.win;

var data = [
    {title:'Row 1'},
    {title:'Row 2'},
    {title:'Row 3'},
    {title:'Row 4'},
    {title:'Row 5'},
    {title:'Row 6'},
    {title:'Row 7'},
    {title:'Row 8'},
    {title:'Row 9'},
    {title:'Row 10'},
    {title:'Row 11'},
    {title:'Row 12'},
    {title:'Row 13'},
    {title:'Row 14'},
    {title:'Row 15'},
    {title:'Row 16'},
];
$.list.setData(data);
$.list.scrollToIndex(data.length-1);
win.open();
