Assign a specific CSS class to an object in Mobile Web?

I was wondering if there was any way to assign a class (as a property) to an object in Titanium so that when the application is compiled for mobile web I can create CSS to style the object further (in ways not possible in Titanium).

I can see several classes and IDs are applied, but I'd like to be able to add one of my own while still in Studio.

— asked December 18th 2012 by Steve Lombardi
  • css
  • mobile
  • mobileweb
  • webview
4 Comments

  • While it's technically possible, it is strongly discouraged because there are no documented Titanium APIs for doing so (no Android or iOS compatibility plus the undocumented APIs are subject to change without notice). The recommended way of doing custom styling is to create a mobile web module (not CommonJS).

    — commented December 18th 2012 by Bryan Hughes

  • OK, so basically the answer is no. It's too bad, since it would be nice to be able to apply font styles such as leading and web fonts via css to a web layout which now lacks those things.
