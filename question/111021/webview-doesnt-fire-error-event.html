WebView doesn't fire error event

I've just tried to catch a loading error from the WebView on iPad by using the following code:

var browser = Ti.UI.createWebView({url:startpage, top:0, 
height:win1.height - 44});

browser.addEventListener("error", function(e){
    Ti.API.log("Error: " + e.message);
//do something
});

But the event doesn't get fired, when an invalid URL gets opened. I can only see

>[WARN] couldn't load URL: app://…

in the Test&Package console…

Has anybody got the same problem or a solution for it?
