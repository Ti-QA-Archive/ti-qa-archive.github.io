Utilize the app screen shot by taking iphone 5 picture resolution issue

Taking a screenshot of app, imported to desktop, the size is 640x1136, but resolution is 28.35. The apple store requires 72dp resolution and up. Is there a way to change resolution setting?

    Hi

    If you take a screenshot of the phone either in the simulator or the actual device - that is the correct resolution for the AppStore.

    I tend to use the simulator and the screenshot option myself, this way I can provide the; non-retina, retina and retina tall versions quickly and simply.

    The images you get are 72dpi, do not worry - they are just trying to make sure you do not provide a lower res image that has been stretched.

    • I am going to assume you are using Ti Studio 3.0.2 or 3.1 - if you are using an earlier version let me know which and I will detail that.

      Ti Studio 3.0.2

      • Locate your app in the left side of Studio - Project Explorer
      • Expand the app tree list - click the triangle
      • Edit the tiapp.xml file by double clicking it
      • Find the section of the main tab Deployment Targets
      • Click the Configure button
      • Expand the Platforms option on the left
      • Select iOS
      • Select the Default Display option at the top of the screen

      Ti Studio 3.1

      • Locate your app in the left side of Studio - Project Explorer
      • Select Run Configurations from the play button at the top
      • Select Display Type from the drop down choices

      Ti Studio 3.1 allows you to set a different choice per app you are working on, the 3.0.2 version required you to set all apps to the same choice.
