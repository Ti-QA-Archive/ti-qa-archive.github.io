Image flickers while dragging on Android

Hi ,
I want to implement the drag and drop functionality. Here is the code :

                        var curY = 0;
            var curX = 0;
            var deltaY = 0;
            var deltaX = 0;
            var olt = Titanium.UI.create2DMatrix();

        imageView.addEventListener('touchstart', function(e) {
                curY = e.y;
                curX = e.x;
        });

        imageView.addEventListener('touchmove', function(e) {
                 var deltaX = e.x - curX;
                              var deltaY = e.y -curY;
                              olt = olt.translate(deltaX,deltaY);
                              imageView.animate({transform: olt, duration: 10}); 
        });

but i can't get it working as i start dragging it, image flickers and does not dragged smoothly . App runs in landscape mode. Please help me out,Thanks in advance.

Application type: mobile app

Titanium SDK: 2.1.1

  • You can get a slight improvement by not responding to every single pixel the image has moved by. Try only updating after, for example 10 pixels. In my experience, however, Titanium does not work very well with this kind of javascript animation. Native apps almost certainly use lower-level Cocoa drawing api's to to this in a performant manner.
