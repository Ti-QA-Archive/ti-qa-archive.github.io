  • Accepted Answer

    It doesn't look like the build script is finding your Titanium SDK:

    "TiPaintModule.h:7:21: error: TiModule.h: No such file or…" There are several other errors like that.

    Make sure your titanium.xcconfig is set up properly and that the SDK you are pointing at actually exists at the folder you specified. Also try using something earlier than 1.7.2 (I built with 1.7.0 when I was testing).
