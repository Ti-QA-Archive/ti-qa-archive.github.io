How to Repeat the same Imageview multiple times in the same window

Hiya

My iphone app takes a photo from the camera or camera reel and i then add it into an imageview quite simply like below:

//Create imageview from Chosen Image
var image = Titanium.UI.createImageView({
   image:event.media,
   height:220,
   width:283,
   top:40
});

I need to be able to repeat the imageview so the window contains 5 images rather than just the 1. I currently have it set up by defining each individual imageview by number i.e 

var image1 = Titanium.UI.createImageView({... 
var image2 = Titanium.UI.createImageView({... 
var image3 = Titanium.UI.createImageView({... 
var image4 = Titanium.UI.createImageView({... 
var image5 = Titanium.UI.createImageView({...

but this seems very messy. Is there a function or way in Titanium that i can repeat an imageview by a set amount of times without having to define and add each one one at a time?

Thank you for any help with this,

    I suggest using an array of images. Note the addition of an index field. This enables you to know about events for the images in the array. Example below with event listeners, as well:

    var image = [];

for (i=0;i<=4;i++) {

  image[i] = Titanium.UI.createImageView({
    image:event.media,
    height:220,
    width:283,
    top:40
    my_index: i
  });

  image[i].addEventListener('click', function(e) {    
    Titanium.API.info("image event listener - my_index: " + e.source.my_index);
  });

}
