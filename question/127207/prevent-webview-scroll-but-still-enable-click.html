Prevent WebView Scroll but still enable Click

Hey guys,

I'm trying to prevent my WebView from scrolling - at all.

I'm currently using this code to do so:

        videoView.addEventListener("touchmove", function(e) {

           e.preventDefault();
        });

But this also disables my click event.
How can I prevent it from scrolling but still enable it to be clicked?

It's an YouTube video and the only event I need on it is the click event.

Thanks!

  • add this code to the html page you have in your webview

    document.ontouchmove = function(event) {
   if (document.body.scrollHeight == document.body.clientHeight) {
      event.preventDefault();
   }
}
