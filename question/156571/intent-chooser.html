  • Accepted Answer

    Our APIs don't specifically expose a method for adding a ShareActionProvider to a menu, which is how that "mini share" menu is created natively. Looking at our docs, I don't see a way to set the necessary properties either. I don't think you can do this currently. Please submit a feature request at jira.appcelerator.org and hopefully the engineers will add it in a future release.
