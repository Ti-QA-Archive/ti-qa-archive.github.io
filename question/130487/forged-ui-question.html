  • Hi Josh

    Forged UI was introduced in the Titanium Developer week, so I can give you what they said. Using Forged UI will enable you to develop by using a GUI the screens / windows required. At the moment you would build a separate screen / window for IOS, Android and the different resolution so effectively 2 for IOS and 3 for Android, then code to display the one required.

    going forward they plan to combine this into a single design p screen/window.

    I haven't tried it but I believe you can try Forged UI for free but not have all the functionality.

    This product is still in its infancy and will develop very quickly into a much more mature product.

    Hope this helps

    T.
