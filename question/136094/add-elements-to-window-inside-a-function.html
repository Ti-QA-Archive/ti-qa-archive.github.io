Add elements to window inside a function

Mobile, Android, Titanium Studio, build: 2.0.1, Windows 7 64x

So, I'm trying to add some elements to my window, dynamically.
I have a query, and the answer of that query will come in rows.
I want to be created as much labels as rows I recieve.

I have this (where "i" will be the number of rows)

var answers = [];
for(var i=0; i<=3; i++){

    answers[i] = Titanium.UI.createLabel({
    color:'#000',
    text:'hello',
    top:'500 px',
    left: '0 px',
    width:'420 px',
    height:'150 px',
    visible: 'true',
    font: FontNormal,
    })
win1.add(answers);

}

But then I need to add the element "answers" to the current window, but I have no idea how to do it…
