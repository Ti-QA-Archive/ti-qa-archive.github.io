    Hi

    You say you want to sort the array using the function I provided during our last question, at least twice, but your code appears to attempt to add the table every single time you run the function.

    I would suggest you split the table creation and the array management to two separate functions, then when you want to sort the data you simply use the setData method.

    The problem you are experiencing is likely due to the attempt to recreate the table each time.
