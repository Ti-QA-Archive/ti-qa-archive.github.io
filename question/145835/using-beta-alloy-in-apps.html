Using beta Alloy in apps

I've taken a look at Alloy and I really like it. However I also read it is still in beta. I got some questions regarding the beta status:

  • Is it stable enough to use for developing iOS and Android apps on Windows?

  • Would you recommend Alloy or the 'old' way to develop apps using Titanium?

  • When will Alloy be officialy released (no beta)?

Thanks in advance for answering these questions.

  • In brief:

    Alloy is stable, but since it is still beta, it is evolving quickly. People are building production apps with it. Most of these people are top titanium developers and early adopters of Alloy. As Christian said, if you need to build an app today, then you should stick with what you know. If you make the investment in Alloy, it will likely take longer at first, but it will make things faster for you in the long run.

    We don't have an official date for a GA release of Alloy yet. There's a number of features and integrations with existing Titanium tools that we want to nail down before considering it GA. We will keep the community posted on this progress.
