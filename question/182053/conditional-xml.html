Conditional XML

So, I have been playing with the conditional XML 'IF' statement and originally used the condition as specified here. According to the documentation, you should be able to reference the arguments that are passed in using $.args.hereIsTheArgumentName

After trying this and getting an error that it could not find the argument object, I changed my xml to read if=&#39;arguments[0].hereIsTheArgumentName&#39;. Based on the code generated in the Resources directory, this works. Just wondering if I am missing the syntax somehow (don't want this to break in the future) or if this is a problem with the documentation… or should I log this in JIRA?

Ray

  • The documentation states:

    To use a custom query (available since Alloy 1.4), assign the if attribute to a conditional statement in the Alloy.Globals namespace. This conditional statement must return a boolean value. You may only assign one query to the if attribute.

    If you read the statement above it says

    to a conditional statement in the Alloy.Globals namespace

    Which means the IF must be as follows:

    if='Alloy.Globals.myVariable'

    where myVariable is something you have set up prior to this script running.

    Don't log a JIRA.
