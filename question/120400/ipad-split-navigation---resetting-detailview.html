  • Root window of the navigation view cannot be changed, closed or reset. What you're doing right now is probably adding more views to navgroup hierarchy (I don't see your code, so I'm guessing).

    What you CAN do is make a clean (vanilla) window as a detail view and then create navigation group within dynamically when masterview changes. Make navgroup var accessible within detailview, so you can set it to null before creating new navgroup (I'm guessing again).

    Hope this helps.
