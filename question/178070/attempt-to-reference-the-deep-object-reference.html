Attempt to reference the deep object reference

Hi,

My app was working but I've updated to Titanium SDK 3.4.0 and now I receive this error:
[ERROR] : Attempt to reference the deep object reference : "productData.name".
[ERROR] : Instead, please map the object property to an attribute of the model.
[ERROR] : Alloy compiler failed

alloy.js

Alloy.Models.productData = new Backbone.Model({});

productRow.js (click event)

var detailController = Alloy.createController('editItemModal', {
        parent: $,
        data : products.get( products.models[e.index] )
    });

editItemModal.js

productData.set(args.data.attributes);

editItemModal.xml

<TextField id="textField" value="{productData.name}"/>

How can I solve this ?

Thanks!

  • Accepted Answer

    What you're seeing is a side effect of the resolution of ALOY-932. Accessing properties of objects was never expected to work, and the fact that it did for you was a bit of luck. One possible solution is the technique noted in the linked Q&A post. You could also use a dataTransform function or extend the collection to expose the child properties as direct values.
