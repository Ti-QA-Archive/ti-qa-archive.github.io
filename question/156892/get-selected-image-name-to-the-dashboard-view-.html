  • Hello,

    you can add a custom field into the DashboardItem, so you can actually print or know which one is:

    var item = Ti.UI.createDashboardItem({
    badge: itemData[i].badge,
    image: 'KS_nav_ui.png',
    selectedImage: 'KS_nav_views.png',
    label: itemData[i].name,
    customID: 'this is my id',
  });

    Check the customID, which you can assign to a number or function.

    Mauro
