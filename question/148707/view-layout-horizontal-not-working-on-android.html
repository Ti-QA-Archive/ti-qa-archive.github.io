View: Layout "horizontal" not working on Android

I have a problem with a view with a remote image (unknown with) and layout "horizontal" on Android. On the iPhone everything is working pretty good but not on Android. Here is my Code:

             var imgview = Ti.UI.createView({
                    height : 100,
                    zIndex : 90,
                    width : '95%',
                    layout : 'horizontal',
                });

                var image = Ti.UI.createImageView({
                    image : remote_url,
                    hires : true,
                    top : 5,
                    left : 5,
                    height : 90,
                    backgroundColor : 'white',
                });

                var label = Ti.UI.createLabel({
                    text : data.title,
                    font : {
                        fontSize : '14sp',
                    },
                    color : 'black',
                    width: Ti.UI.SIZE,
                    height: Ti.UI.SIZE,
                    left : 140,
                    top : 5,
                });

                imgview.add(image);
                imgview.add(label);

On Android the label is missing. If I remove "layout: 'horizontal'" from the imgview, the label is shown.

Does anyone know why this happens? The problem I don't know the width of the remote image and image.width is undefined. So I don't know the left position of the label.

    if you do not know the width of the image then you should set width : Ti.UI.SIZE property of image

    var image = Ti.UI.createImageView({
                    image : remote_url,
                    hires : true,
                    top : 5,
                    left : 5,
            width : Ti.UI.SIZE,
                    height : 90,
                    backgroundColor : 'white',
});
