Finishing of opentok-session

Hi,

opentok is a face2face video conferende tool for titanium as well. In this documentation is wonderful described how I can establish a conference. But what is with finishing?

Here my code:

OpenTok.prototype.finishSession = function() {
    if (self.session) {
        if (self.subscriber) {
            self.subscriber.close();
            self.remove(self.subscriberView);
        }
        if (self.publisher) {
            self.session.unpublish();
            self.remove(self.publisherView);
        }
        self.session.disconnect();
        self.removeAllChildren();
    }
}

The first member is publisher, all worls fine. But the second member (subscriber): if he finish sesion the app crashes.
