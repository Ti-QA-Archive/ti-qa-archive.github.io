App resizing for screen

So now I have an app that is working correctly on the iPhone simulator in its default portrait mode. I simulate rotation - the app stays firmly stuck in its portrait orientation. I run it on the iPad simulator and it stays with the pixel dimensions I had given the various controls for the 320 x 480 iPhone screen. I found that it is possible to use %age units so I tried that only to end up with a blank screen.

I guess there is something I am doing wrong but I haven't got a clue. Any help would be appreciated.
