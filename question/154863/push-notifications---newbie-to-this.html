Push Notifications - Newbie to this

Hi All

I was wondering if you can send a push notification, which tells our app to sync some data from the server.

As we may change some data on the server, and then issue command to our app to fetch the new data.

So process is:

  1. Change data on server.
  2. Issue push notification.
  3. App receives notification.
  4. App calls sync process on server to fetch new data.

This will allow all apps to have the correct data.

Any advice on this would be great.
