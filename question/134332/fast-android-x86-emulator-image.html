Fast Android x86 emulator image

Recently, Intel released an Android x86 emulator image for Android 2.3.7 which runs much faster then the ARM images we normally use:
Installing the Intel Atom Android x86 Emulator Image Add-on from the Android SDK Manager

It works on my Mac nicely. Two drawbacks however:

  • No support for the Google API
  • Titaniums V8 libraries are ARM-only (hence you need to use Rhino)

Thought I would share it with you.
