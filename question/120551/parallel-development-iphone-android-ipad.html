    Ti Studio allows you to make universal apps (iOs).
    In theory you can use the same code to make on app for all the platforms trying to figure out what device is.
    Something like this:

    if (Ti.Platform.osname == 'android' ) {
    // we are on android
    // use Titanium.Platform.displayCaps to get the screen size
}

else if ( Ti.Platform.osname == 'iphone' ) {
    // ....
}

else if ( Ti.Platform.osname == 'ipad') {
    // ....
}

else {
    // where are we?
}
