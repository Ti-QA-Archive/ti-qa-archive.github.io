  • Accepted Answer

    You can still use it without any problem, but as Andrew mentioned, it is now deprecated so you will have to go with no support. If you want to go ahead and do something more professional, then you should have a look at Lanica's new implementation (still in beta, but more powerful that the previous version) There is a hefty price tag attached to it though…

    So, in a nutshell:

    • Keep using quicktigame2d for small projects. It works well, and it's free.
    • Go with the Lanica professional offering for big commercial projects.

    Hope this clears it up for you.
