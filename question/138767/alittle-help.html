  • hi Ronny,

    you can create tableview row with the help of for loop like

    var arrRows = [];

for(var i = 0; i < 10; i++){
var tvr = Ti.UI.createTableViewRow
({title:'Celeberty'});

arrRows.push(tvr);

}

tv.data=arrRows;

    and instead of row's click event , you can use tableview's click event…
