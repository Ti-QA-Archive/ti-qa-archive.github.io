Error in map

When i write mapview code run in iphone but in android it shows error

[ERROR] Failed installing com.map.example: pkg: /data/local/tmp/app.apk

  • Prasanna

    Only the Google enhanced SDKs include support for maps. See Preparing for Android development for information about this.

    To resolve your problem, make sure you choose one of the SDKs which have the "APIs" prefix on the "Run Emulator" screen. At the moment, "APIs 2.2" is the most up to date SDK supported by Titanium. See the first screenshot in the Titanium Developer Debug Tools section of the Troubleshooting guide for an example.

    If this doesn't work, delete all your build files and try launching your app again, as explained in this post.

    Cheers
