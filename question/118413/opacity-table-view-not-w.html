  • The tableView is going to display in front of the window element. Thus, if you have a background set for the tableView the window's background will not be seen.

    To have a transparent background for the tableView in order to see that window background, simply leave out the backgroundColor property.

    If you are wanting a semi-transparent background of the tableView I would suggest making three elements: a tableView with a transparent background, a view with a backgroundColor of '#000' and opacity 0.2, and then the window element with the backgroudnImage.
