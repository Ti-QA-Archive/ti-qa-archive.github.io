Mobile web default page not ture in Firefox

I create a project with two tab UI, the default page will turn to the tab page when debug with local browser.
But after making it the package, open the index.html in Firefox, it just stay in default page.
Is that i miss some step? May i open the tab page as the start up page? Thanks for help.

    Titanium Mobile Web currently requires a web server. When you preview a mobile web app, Titanium Studio uses a built-in web server. When you package your Titanium Mobile Web app, you need to supply the web server.
