Session - Global Variables

Hi all,
The question is as in the title. How can I create and use session variables for my app? In order to make myself clear I mean session variables like the ones used in web applications, global variables that all windows of the application can use.
Thanks in advance

  • Accepted Answer

    Ti.App.myVar

    will be accessible anywhere in your app for the current session.

    Ti.App.Properties.setString("mystr", "mystr")

    will be accessible even after you close your app.
