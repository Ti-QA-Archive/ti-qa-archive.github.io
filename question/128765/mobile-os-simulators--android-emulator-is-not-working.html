Mobile OS Simulators + Android Emulator is not working

I started working on a new project and every time I ran, it was opening the simulator for the old project that I created. I thought erasing old 2 (iPhone and iPad) simulators from the Run Configurations dialog box would allow Appcelerator to create new ones for the new project but didn't work that way.

Both simulators worked fine on the first project but didn't work after I erased them.

I'm using MAC Lion and XCode 4.2 for the Titanium Studio.

Is there anything I can do to fix this?
Thanks!

    Hi Mehmet

    The possibility is that when more than one projects are open this lead to reference the last project run on the device will run every time to run the project from the top toolbar.

    So i suggest you to run the project from the project explorer select the project then right and select run As -> android emulator / Ios emulator as per your choice.
