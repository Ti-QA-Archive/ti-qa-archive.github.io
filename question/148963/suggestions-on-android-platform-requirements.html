Suggestions on Android platform requirements?

I don't know where to start on this. ANy ideas? I need to have from android first.

Ability to program on the following platforms:

Android
iOS
Windows

For Android

Ability to interface with and read from the built-in NFC reader in Android devices (phones/tablets)
Ability to decode an ISO NFC card, encrypted with a specific key
Decrypted information will be: stored on the device locally, displayed on the device, and transmitted to a website
Transmission can be by any means available (cellular/WiFi)

For Android and iOS

Program the system for a licensing scheme to allow one software license per device
Secure configuration screen to setup device for a conference: conference logo, lead retrieval method used (NFC/QR), license key, website URLs, encryption key, etc.
Ability to e-mail the local database as a CSV file
Ability to use built-in camera to read and decode QR codes
Transmit to and retrieve detail data from the website, store locally and display detail data on the device

For Windows

Ability to interface with and read from a USB-based NFC reader
Ability to query a database for session details (via website calls)
Ability to decode an ISO NFC card, encrypted with a specific key
Ability to query a website for confirmation on attendance status (allowed/disallowed/unknown card)
Display that status and allow for override
Ability to send scanned cards linked with current session data to database (via website calls)

The need for a programmer to support all the options listed under Android, all the options under Windows and the first three options listed under Android and iOS is the immediate need. The ability to incorporate QR codes is secondary . Programming language(s) used are at the discretion of the programmer but must be maintainable and supportable if programmer is no longer retained.
