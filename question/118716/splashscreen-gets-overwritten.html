Splashscreen gets overwritten?!

How do i make the splash screen stick on android?
In resources/android i have made a folder called res that I have made all my spashscreens, however, this does not get copied to build/android/res.

The only way i can find to do it is to copy it manually, which works, but then when i build (only sometimes) it deletes my custom drawables and reinserts an appcelerator one.

This is a real pain because i do not know wether building will overwrite it or not. Where do i put the res folder i really want to use so it gets copied.

  • Hi,

    Lets get it short and sweet, if you wanna get rid of default titanium splash screens then replace all the default Titanium splash screens or atleast the ones that corresponds to the Screen size you are basically testing, with your custom designed Splash screen …

    Path => <Project Path>\Resources\android\images

    And do not forget to delete the contents of <Project Path>\Build\android\ and perform a fresh build…
