Multilanguage with i18 xml on iOS

Hi,
I wrote a little app that uses i18 xml file in Titanium to support metalanguages on my app (currently english only). When I run from appcelerator I see everything is fine - and I see my text from the xml file.

The problem is that when I the iPhone project from xCode and run it on the emulator, instead of the text from the xml file I see the text id name. How can I make sure xCode loads this file correctly and use it?

Thanks

  • I believe the internationalization files are only packaged with your project when run through Titanium Studio, but not if you attempt to build natively from XCode. You can find more details on this in the wiki documentation: http://wiki.appcelerator.org/display/guides/Internationalization
