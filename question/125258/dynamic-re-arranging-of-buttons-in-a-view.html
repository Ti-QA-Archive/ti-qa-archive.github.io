Dynamic re-arranging of buttons in a view

I need to create a view full of horizontal buttons, that dynamically adjusts the buttons positions and the height of the view as buttons are hidden or shown. I can accomplish this with manual JavaScript coding, but I first wanted to see if there was an easier, more in-built method to do this.

To explain further, I have a view like this (using XML sudo code):

<view>
   <button1 />
   <button2 />
   <button3 />
   <button4 />
</view>

Under normal circumstances, using a vertical layout, if I were to hide button3 there would be a gap in between button2 and button4 where button3 used to be. I need button4 to move up so that there is no gap. It should look like:

<view>
   <button1 />
   <button2 />
   <button4 />
</view>

I also need the View object to adjust its height accordingly so that it fits exactly as many buttons as it contains. Then, I need to be able to show button3 again in the same place, and everything should re-adjust itself to its original state. Do I need to do all this logic with JavaScript, or can I accomplish it with some kind of layout or control?
