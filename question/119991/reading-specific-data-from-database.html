reading specific data from database

Hi, my database looks something like this.

area item

  • north 2
  • north a
  • south 3
  • south b
  • east 1
  • east c
  • west 4
  • west d

i use this command to populate my menu on {N,S,E,W}.

var rows = db.execute('SELECT DISTINCT area FROM list');

However, i would like to choose North to be shown on the menu instead of all NSEW.

Anyone can teach me to do just that?

many thanks

  • iOS 4.3, Ti 1.2.2

    var db = Ti.Database.install('list.sqlite','list');

    var prodCat = Ti.UI.currentWindow.prodCat;

var rows = db.execute('SELECT * FROM list WHERE area = "' + prodCat + '"');

// create the array
var dataArray = [];

while (rows.isValidRow())
{
    dataArray.push({title:'' + rows.fieldByName('name') + '', hasChild:true, path:'address.js'});
    rows.next();    
};
