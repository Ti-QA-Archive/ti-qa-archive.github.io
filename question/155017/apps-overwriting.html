apps overwriting...

I have created two copies of same project(for ipad in titanium studio) then create different ids for both with different provisioning profile. then by the ad-hoc distribution creates two .ipa files, But when I install that both app using itunes , its overwrites on one another. What will I do for installing its as seperate apps..

    Hi

    To create two different apps you should only need to create two different app ids and each with an associated certificate, so as long as your app ids are actually different, like;

    • com.yourcompany.appname1
    • com.yourcompany.appname2

    Then the only other thing that is likely is the actual ipa file has the same physical name, thus the file is actually overwriting the other rather than the ids are overwriting each other.

    Make sure each has a different filename - you can rename after the first one has been created or I would change the app name as the filename is usually generated from that,
