Access a label of tablerow in controller

Hi..

I've a table in a view like this. How to access the label in controllor?


<View id="tableview">
        <TableView dataCollection="RestModel" dataTransform="transformFunction" id="table">
            <TableViewRow id="row">                           
                <ImageView id="photo" image="{photo}"/>
                <Label id="text" text="{text}"></Label>
            </TableViewRow>
        </TableView> 
</View>

I'd tried to access like


$.text.left ='5dp';

But it shows error "cannot set left of undefined.

  • You're trying to access a single element in a data bound list of elements. That will never work. The TableViewRow you have in the markup represents a template to be rendered for every model in the bound collection. If you want to have finer control over the components that will be repeated (the row and its image and label) then you need to move the TableViewRow (and its children) into its own view/controller. In the separate controller you'll be able to reference $.text.left no problem.
