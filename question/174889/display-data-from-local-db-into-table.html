  • But how data can be populated based on the the category and display into different tableview section ?

    Ok, couple of ways to do this.

    1. Request all the records, and iterate through them, sorting into different arrays of rows, push each array into a section, add the sections to the table.

    2. If the categories are known in advance, query just for records with that category, shove rows created with these into a section, add section to table, repeat for subsequent categories.
