  • Since Titanium Windows don't behave like webpages, you con't really need wait for the entire window to load. (if you see the window on screen, it is loaded)

    Then again, if you ever want to trigger an event once the Window is loaded, you can add a listener on the postlayout event.

    win.addEventlistener('postlayout', function() {
     alert("The window is ready!");
});
