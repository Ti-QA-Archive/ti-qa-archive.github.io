iOS5 and toolBar

Hi,
since iOS5 (SDK 1.7.3) the toolBar is not visible.
Only in the construct

var tab4 = Titanium.UI.createTab({
    icon : 'assets/bottombar/latest.png',
    title : 'Latest',
    window : view.latestWindow
});

In the context:

var winoptions = {
    title : '',
    width : 320,
    barImage : Ti.Filesystem.resourcesDirectory + 'assets/start/toplogo.png',
    barColor : 'black',
    backgroundColor : '#000'
};

switch (selectedWindow) {
  case 0: w.tab.open(view.createNewsWindow(winoptions)); break;
  case 1: w.tab.open(view.createRentalsWindow(winoptions)); break;
  case 2: w.tab.open(view.createSalesWindow(winoptions)); break;
  case 3: w.tab.open(view.checklistWindow); break;
}

the toolBars are hidden. This problem appears since new version.

What is wrong?
