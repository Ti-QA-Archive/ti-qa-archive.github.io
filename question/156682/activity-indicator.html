Activity indicator

Hi all. I've tried to implement a simple activity indicator to show and hide once a webview is loaded. It is not coming up in my app. Can anyone tell me, what I've done wrong here:

var actInd;

actInd = Titanium.UI.createActivityIndicator({
    style:Titanium.UI.iPhone.ActivityIndicatorStyle.DARK,
    height:60,
    width:60,
    zIndex:400
});

//scheduleWebView.add(actInd);
actInd.show();


    scheduleWebView.addEventListener('load',function(e){ 
        actInd.hide();
    });

    Hi,
    You also need to add activityIndicator to the window, then only it will be visible.

    
window.add(actInd);
actInd.show();
