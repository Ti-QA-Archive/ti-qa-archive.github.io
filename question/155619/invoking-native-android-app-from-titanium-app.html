invoking native android app from titanium app

Hi ,
Im trying to invoke android and Iphone native app from titanium application. While im searched on the web i could get that we can open native application from titanium app using url scheme name. For that im created One sample native app in android. Below code in native AndroidManifest.xml:

<?xml version="1.0" encoding="utf-8"?>
<manifest xmlns:android="http://schemas.android.com/apk/res/android"
    package="com.example.sample"
    android:versionCode="1"
    android:versionName="1.0" >

    <uses-sdk
        android:minSdkVersion="8"
        android:targetSdkVersion="8" />

    <application
        android:allowBackup="true"
        android:icon="@drawable/ic_launcher"
        android:label="@string/app_name"
        android:theme="@style/AppTheme" >
        <activity
            android:name="com.example.sample.MainActivity"
            android:label="@string/app_name" >
            <intent-filter>
                <action android:name="android.intent.action.MAIN" />

                <category android:name="android.intent.category.LAUNCHER" />

                <data android:scheme="Mobileapp" />
            </intent-filter>
        </activity>
    </application>

</manifest>

I have used android:scheme name 'Mobileapp'
I also created another titanium sample application. Im calling the native app in click event by using 

label.addEventListener('click', function(e) {
        //alert(e.source.text);
        Ti.Platform.openURL('Mobileapp://');
    });

While using Ti.Platform.openURL('Mobileapp://'); is not working . But when i tried to open the telephone using Ti.Platform.openURL('tel://'); is working . What wrong in above code ?
