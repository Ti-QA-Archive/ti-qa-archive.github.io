How to access to children elements in a view inserted with <Required> tag

Hello everyone! I'm new to Titanium and Javascript, so I need your help!
Can you explain me how to access to children elements in a view inserted with the <Require> tag?
I have this code:

views/index.xml:

<Alloy>
    <Window>
        <Require id="req1" src="reqView" type="view" />
        <Require id="req2" src="reqView" type="view" />
    </Window>
</Alloy>

views/reqView.xml:

<Alloy>
    <View id="view">
        <Label id="name" />
        <Label id="surname" />
    </View>
</Alloy>

how can I access from the index.js for example to the name Label from req1 and then to the surname Label from req2 and modify the text, if it's possible?

I'm using Titanium Studio, build: 3.1.0.201304151600 with the latest updates to develop an Android app.

    index.js

    $.req1.getView('name')
$.req2.getView('surname')

    • You need to give the Require an id then you can access its childrens.For eg :

      <Alloy>
    <Window id="register">
        <View id="registerBody" class="container">
            <Require src="header" type="view"  id="header"/>
            <Require src="buttonBar" type="view"  id="buttonBar"/>     
        </View>
    </Window>
</Alloy>

      So now in the code you can access the buttonBar.xml elements via this id easily.
      for eg :

      
$.buttonBar. userButton.title="New Title";
