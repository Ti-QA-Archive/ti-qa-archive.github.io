    • ok then there is another custom solution to solve the problem:

      try to put a view over the tab which one you want to disable. such as:

      var tabOnDemand = Titanium.UI.createTab({  
    icon:'images/ondemand.png',
    title:'On Demand',
    window:win2,

    backgroundDisabledColor:'gray'
});

var view = Ti.UI.creatView({
    top:'//according to the top of disabled tab'
    bottom:'//according to the bottom of disabled tab'
    height:'//according to the height of disabled tab'
    width:'//according to the width of disabled tab'
    zIndex://mustbe greater than tab
});
// and dont write any event listener for this view.

      Hope this logic work for you.
