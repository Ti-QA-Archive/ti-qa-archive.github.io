  • 1.1 hasn't been released as of right now, but is expected tonight. TI Developer is not the same thing as a SDK drop. when the guys out it up in GitHub, you will download the zip, and then (assuming it's still a manual process), copy the 1.1 sdk folder into your:

    Library/Application Support/Titanium/mobilesdk/osx folder along side whichever older versions you previously installed.
