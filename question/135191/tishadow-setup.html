I'm trying to run TiShadow, which seems a fantastic tool for fast coding. I'm stuck at this part of the instruction

> "A custom build of the Titanium SDK. These can be retrieved from the tishadow branch of my fork of the titanium mobile sdk."

I forked the mentioned project, but I'm totally clueless about what to do next. How do I "install" or use that mobilesdk version?

  • Hello Esteban.

    I think that you need to use custom build of Titanium SDK provided in TiShadow Git repository. To do that, you can unzip "https://github.com/dbankier/TiShadow/blob/master/mobilesdk-tishadow-osx.zip" file and copy it to location of Titanium SDKs on your hard drive. Or you can go to TiStudio and use option to install SDK from URL.
