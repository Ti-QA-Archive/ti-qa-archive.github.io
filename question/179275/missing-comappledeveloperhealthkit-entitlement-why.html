missing com.apple.developer.healthkit entitlement (Why?)

Hello everybody, Im currently developing an iPhone-module, to integrate Apple Healthkit with my current Titanium app. Latest version of Ti And iOS. (3.4.1).

Im trying to add Healthkit in xcode first, ofcourse. But somehow, I cant get it to work in Titanium. Everything seems added the right way, in the module and app-project in Xcode.

The errormsg from the Log: "missing com.apple.developer.healthkit entitlement".

(The module)
1) Added Healthkit.framework in Xcode ("build" -> "Link Binary With Libraries")
2) Added "framwork HealthKit" to module.xconfig
3) Added <ios> <plist><dict> <key>UIRequiredDeviceCapabilities</key><array> <string>healthkit</string> …
to timodule.xml
4) Added the code (see image) http://i.imgur.com/nhQFpC3.png

In the App:
1) Added <module platform="iphone" version="0.1">healthkitModule</module> tiapp.xml
2) Added "framwork HealthKit" to module.xconfig
