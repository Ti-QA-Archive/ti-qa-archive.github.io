    You just need to escape single quotes because single quotes are special characters in SQL (SQL strings are quoted with single quotes, so if string contains single quotes, they should be escaped).

    To escape single quotes, you can use "replace" function from JavaScript. In Sqlite, single quotes are escaped this way: A string constant is formed by enclosing the string in single quotes ('). A single quote within the string can be encoded by putting two single quotes in a row.

    So …

    stringValue = stringValue.replace(/'/g, "''");
