iOS, tabbed interface, buttons in the top toolbararea?

Hi!

I am doing an app with a tabbed interface.

In iOS I would like to be able to place buttons in the top area (where the window title is), just like it's seen in the app-store app. I'd like them to look very native to iOS.

I've tried adding to the windows toolbar array, but that creates a toolbar in the bottom of the window.

    You can do things like:

    Button bar:

    var buttonBar = Ti.UI.createButtonBar({
    labels:['One', 'Two'],
    style : Titanium.UI.iPhone.SystemButtonStyle.BAR,
    backgroundColor : '#ff0000',
    width:Ti.UI.SIZE
});        
myWindow.setLeftNavButton(buttonBar);

    Text button:

    var saveButton = Ti.UI.createButton({
    title : 'Save'
});
myWindow.setRightNavButton(saveButton);

    System button:

    var composeBtn = Ti.UI.createButton({
    systemButton : Titanium.UI.iPhone.SystemButton.COMPOSE
});

myWindow.setRightNavButton(composeBtn);

    Your own custom graphic button:

    var customButton = Ti.UI.createButton({
    backgroundImage: 'images/buttonImage.png',
    height:31,
    width:123
});
myWindow.setLeftNavButton(customButton);

    Then, of course, set your event listener:

    myButton.addEventListener('click', function(e){
    //do my bidding
});
