    Hi Sabri,

    this error means either phoneNumbers[i].value or numMobile[key] is not defined.
    So there are two solutions.

    1. Check your variables before trying to call a prototype method like toString()

    e.g.

    if (phoneNumbers[i].value && numMobile[key]) {
    if (phoneNumbers[i].value.toString() == numMobile[key].toString()) {
    ....
}
    1. don't call toString() because its unnecessary in this context

    in javascript the "==" operator doesn't do a type conversion so 

    var number = 10;
var someString = "10";
if (number == someString) // will be true

    if you like to compare two values and make sure they are using the same type use the === operator

    var number = 10;
var someString = "10";
if (number === someString) // will be false

    Hope this helps
