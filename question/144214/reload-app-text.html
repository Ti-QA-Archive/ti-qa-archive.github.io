"reload" app text

Hi,

I am working on an app that supports multiple languages. Basically there is a picker that allows the user to change the language.

The way it works that instead of using static text throughout the app, I'm calling a function that returns the text according to the currently selected language.

The problem is that when changing the language, the tabgroup windows (and every other window that was created before the language change) are still using the previous language and only newly opened windows use the new language.

Is there a way to "refresh" the window's text? Or initialize/restart the app so that the new language will take effect?

Thanks,
Idan

    There is no "function" to do this on it's own… You need to create the functionality yourself when inside the "click" event listener of that button

    button.addEventListener("click", function(e){
  currentLanguage = "ES"; // I assume you want to keep track of this

  yourLabels.text = yourNewValue;
