    however.
    If you want to update a row, you have to execute the following sqlite command:

    var db = Ti.Database.open('database_name');
db.execute("UPDATE table SET field = '?' WHERE id = '1'", newValue);
db.close();

    this will give a newValue to the row with the id equal to 1.
