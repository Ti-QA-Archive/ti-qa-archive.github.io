  • it should be found at <Macintosh HD>/Library/Application Support/Titanium/

    Note that is the Library folder in root, not in the users folder. If is there and still doesn't work, I suggest you, first to look for updates in Titanium Studio (I think the option is in Help menu) or reinstall Titanium Studio.

  • This has been resolved. I had to create a Titanium folder and then grant it write permissions. I would have been to nice to know that it was a permissions issues the whole time, not just that it was missing. Once I set the permissions, and reopened Ti it wrote the sdk.
