Cannot change font from inside a listener

I cannot seem to change the font of any object from inside a listener. I have tried many things to get this to work. The listener is being fired properly and "label.font.fontSize = 2" works fine outside of listeners. 

var label = Titanium.UI.createLabel({
    text: "Font Size",
    color: '#ffffff',
    backgroundColor: '#000000',
    height: 100,
    width: 100,
    font: {fontSize:20}
});

label.addEventListener('click', function() {
    label.font.fontSize = 2;
});

    Try setting:

    label.font = {
  fontSize:2
};

    In the handler - that should do the trick.
