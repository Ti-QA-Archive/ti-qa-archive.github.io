  • Developing native extension modules requires a very good knowledge of Titanium and well as the underlying platform. So this is not for everyone.

    That being said, your error message seems pretty much self explanatory. Your ANDROID_SDK environment variable has to be set to the location where you installed the Android SDK.

    Hope this clears that up for you.
