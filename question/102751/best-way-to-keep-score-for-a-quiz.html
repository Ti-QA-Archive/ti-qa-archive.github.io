Best way to keep score for a quiz?

I can't seem to find a good way to keep score for a quiz I am making. I have one screen with three choices and one correct answer and then it goes to a new screen with a new question. I cant seem to get it to tally the total score at the end of ten questions.

Does anyone have a possible solution to this problem?

  • The Working With Local Data guide explains properties, and other ways to persist data.

    Your code will probably look something like this:

    // on new test, clear previous score
Ti.App.Properties.removeProperty('totalScore')

// get totalScore, return zero if does not exist
var currentScore = Ti.App.Properties.getInt('totalScore',0);

// on correct answer
Ti.App.Properties.setInt('totalScore',currentScore + 1);
