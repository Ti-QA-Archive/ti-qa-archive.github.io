Add more than one contact

Hi,

(iOS)

I have a textfield where the user can add an email and a button to add email form the contacts list.

Can we add more than one contact at time, I want to let the user select the emails of multiple contacts and add them to a textfield ?

var contactsTextField = Ti.UI.createTextField({
        top: 3,
        left: 3,
        height: 34,
        borderRadius: 14,
        backgroundColor: 'white',
        hintText: ' Search',
        borderColor:'black',
        borderWidth: 1,
        width: 265
    });

    var contactsBtn = Ti.UI.createButton({
        title: 'add',
        color: 'black',
        width: 34,
        height: 34,
        left: 10,
        top: 3,
        style:Titanium.UI.iPhone.SystemButtonStyle.PLAIN,
        borderRadius: 10,
        borderWidth: 1,
        borderColor:'grey'
    });

    contactsBtn.addEventListener('click', function(e){
        //indicateur.show();
        Ti.Contacts.showContacts({
            selectedProperty: function(e){
               contactsTextField.value += e.value + ', ';
            }
         });
    });
