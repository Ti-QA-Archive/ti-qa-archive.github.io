Verification Issue using a Ti+ Plus module

I am trying to compile the example Ti.Barcode app, and upon startup, I get an error dialog with says "Verification Issue: Unlicensed module(s) detected. You must be a subscriber to use…"

However, I am an Indie subscriber with a paid annual subscription. I should be able to use this module.

Also, I notice that even though this screen says I'm a premium customer and that I should post my issue on the premium support portal, I only read only access to that.

  • I've got same problem and I've find way to fix it

    1. go to your module manifest file
    2. copy guid number
    3. go to your tiapp.xml file and insert <guid> string from module manifest file
    4. save tiapp.xml file
    5. clean project
    6. build and run
