    • Rainer is right. Use Ti.Media.AudioPlayer. It allows for background playing.

      On iOS you'll have to set the proper value for Ti.Media.audioSessionCategory and put this in your tiapp.xml:

      <key>UIBackgroundModes</key>
<array>
    <string>audio</string>
</array>
