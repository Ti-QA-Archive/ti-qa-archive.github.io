Get Facebook Email Adress

Hello everyone, I am building a iOS application with Titanium Studio SDK build: 3.0.0.201212131639. 2.1.4 GA. Using Mac OS X 10.7.5, now onto the problem.

I am currently trying to get a Facebook Email Address from a logged in user, I have the following permissions in my application.

Titanium.Facebook.permissions = ['publish_stream', 'user_photos', 'read_stream', 'email'];

I can get the user's profile picture, first name / last name from the graph.facebook API, but in that JSON file there is no reference for a email address(using my own account which I gave permission for email like above)

Underneath is my code.

`

Ti.App.addEventListener('fblogin', function() {
Titanium.Facebook.authorize();
Titanium.Facebook.addEventListener('login', function(e) {
if (e.success) {
Cloud.SocialIntegrations.externalAccountLogin({
type : 'facebook',
token : Ti.Facebook.accessToken
}, function(e) {
if (e.success) {
var user = e.users[0];
Ti.API.info('User = ' + JSON.stringify(user));
Ti.App.Properties.setString('currentUserId', user.id);
getUserData();
GHC.profile.firstname = user.first_name;
GHC.profile.lastname = user.last_name;
alert('Success: ' + 'id: ' + user.id + '\n' + 'first name: ' + user.first_name + '\n' + 'last name: ' + user.last_name);
} else {
alert('Error: ' + ((e.error && e.message) || JSON.stringify(e)));
}
});

    } else if (e.error) {
        alert("Error = " + e.error);
    } else if (e.cancelled) {
        alert("canceld");
    }
});

});

`

I am using the "GHC.profile.firstname" for storage of the first(and thus also last name) in my application which works great, I also get the success alert.

Thanks in advance,

Dave

    I pull the email in from a Ti.Facebook.requestWithGraphPath call.

    Titanium.Facebook.requestWithGraphPath('me', {}, 'GET', function(e) {
   if (e.success) {
      var data = JSON.parse(e.result);
      Ti.API.info('FB login event');
      Ti.API.info('FB accessToken : ' + Ti.Facebook.accessToken);
      Ti.API.info('FB expirationDate : ' + Ti.Facebook.expirationDate);
      Ti.API.info('FB uid : ' + Ti.Facebook.uid);
      Ti.API.info('FB email : ' + data.email);
      Ti.API.info('FB firstname : ' + data.first_name);
      Ti.API.info('FB lastname : ' + data.last_name);
   }
});

    in the callback function of Cloud.SocialIntegrations.externalAccountLogin()
