  • Accepted Answer

    Just add the property to the window creation, then reference it in the new window.

    var local = Titanium.UI.createWindow({  
            url:'restaurant.js',
            backgroundColor:"#fff",
        variable1:'myVariable'
});

    restaurant.js

    var win = Ti.UI.currentWindow;
alert(win.variable1);

    Just remember that you are creating a multi-context application versus a single context application which can have some ramifications down the road. Take a look here for more info on that topic: https://wiki.appcelerator.org/display/guides2/Coding+Strategies
