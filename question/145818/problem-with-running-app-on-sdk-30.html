  • Accepted Answer

    It appears the Titanium CLI can't find iOS SDK 5.0. iOS SDKs are located inside the Xcode.app package. Run the following to view the available SDKs installed:

    titanium info -t ios

    At the top it will print out all Xcode installs that it found. If 5.0 isn't listed as a valid iOS SDK, then you don't have 5.0 installed. Keep in mind that you can build using iOS SDK 6.0 and it'll run on iOS 5.

    You may need to run xcode-select to pick your default Xcode install.
