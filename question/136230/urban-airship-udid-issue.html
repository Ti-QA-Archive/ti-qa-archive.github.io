  • 1.2.1. is the latest version of the iPhone Urban Airship module for Titanium. In the documentation changelog for this version (1.2.1) of the module, it says it updated to Urban Airship's 1.2.0.a library build. Urban Airship got rid of all UDID usage in it's 1.1.5 build.

    So, anyone using the 1.2.1 build of the Urban Airship module for Titanium shouldn't have any problems with Apple.
