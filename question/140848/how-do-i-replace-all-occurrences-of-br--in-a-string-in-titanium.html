How do I replace all occurrences of “<br />” in a string in Titanium?

Hello Friends, I am working on an Titanium Application which deals with Wordpress's JSON API. I am facing one problem of Replacing some codes from the json data. Example:

Content 1

<p>Come everyone let&#8217;s give a cheer,<br \/>\nIn celebration of a couple dear.<br \/>\nA couple who in love unite,<br \/>\nAnd are seldom cross and hardly fight.<br \/>\nTo a couple who bring joy to all,<br \/>\nMay you laugh and sing and\u00a0have a ball.<br \/>\nMay happiness surround you two,<br \/>\nThat is what I wish for you<\/p>\n

I was using str.replace, but it removes only first occurrence but not the other ones. Can someone please help me.

I am using a function like:

function htmlSpecialChars(unsafe) {
    return unsafe
    .replace("&#8217;", "`")
    .replace("<p>", "")
    .replace("</p>", "")
    .replace("<br />", "")
    .replace("<p>", "");    
}

Can someone please share a code to replace all the occurances from the above like Content 1?

Help!

  • Hi,

    Are you looking to remove specific tags, or all tags?
    It is (relatively) simple to remove all tags:

     var clean = html.replace( /<[^>]+>/g, '' );

    If you're looking to remove specific tags, something like this might work .

     var pattern = new RegExp( '</?p|div|br ?[^>]*>', 'g' );
var clean = html.replace( pattern, '' );
