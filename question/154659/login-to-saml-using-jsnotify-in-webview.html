Login to SAML using jsnotify in webview

I'm working on setting up the authentication token from my webpage using jsnotify technique. I'm using alloy framework.

my index.xml goes here (login url below is a fake one to have security)

<Alloy>
    <Window id="authWindow" class="container">
        <WebView id="webview" url="http://myloginserver.com/login.html"/>
    </Window>
</Alloy>

my index.js goes here



$.webview.external = {};
$.webview.external.notify = function catchPopUp (token, jsToken) {
    ajaxheaders = { VSAML: jsToken.access_token };

    Ti.API.info(ajaxheaders.VSAML);
};
$.index.open();

I'm able to see my login page but the popup comes over and webview doesn't handle it. Does any one worked on jsnotify in Ti?
