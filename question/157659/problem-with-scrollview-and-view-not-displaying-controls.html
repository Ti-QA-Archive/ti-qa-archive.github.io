Problem with ScrollView and View not displaying controls

I am using Titanium 3.1.3 and Xcode 5.

When creating a View within a ScrollView, and adding controls to said view, it is not displaying the labels on first launch. It does, however, display the Switch and the Save button. It also, as intended, displays the navBar and tabGroup as translucent, with the background and controls able to be seen behind them.
However, when clicking on another tab and returning, the controls appear (with Labels), but the translucent effect is gone.

Does anyone have any idea or can they provide me with any explanation on how to fix this error?

Cheers

// Create first window //
var settingsWindow = Titanium.UI.createWindow({
    title: "Settings",
    extendEdges:[Ti.UI.EXTEND_EDGE_TOP, Ti.UI.EXTEND_EDGE_BOTTOM]
});

Titanium.UI.iPhone.StatusBar.LIGHT_CONTENT;

var scrollView = Ti.UI.createScrollView({
