  • Do you have a Studio 3.4.0 build installed? The Titanium CLI has changed it's output for ti info in 3.4.0, and as a result Studio 3.4.0 has adapted how it handles the resulting JSON.

    Typically if you move up to the latest CLI, you need to be sure that you're also on the latest Studio.
