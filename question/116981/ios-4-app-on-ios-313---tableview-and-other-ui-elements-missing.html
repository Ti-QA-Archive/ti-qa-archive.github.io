iOS 4 app on iOS 3.1.3 - tableView and other UI elements missing?

Hi,

It seems that the HTTP request doesn't appear to fire, the subsequent table view with the request data isn't appearing and also the 'refresh' button in the top right nav bar is missing.

Has anyone else had any experience of issues with iOS 3?
