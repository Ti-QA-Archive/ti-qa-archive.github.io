Great tool for live debugging on an iPhone with console access

I've recently started using AppSwitch on my iPhone to catch and review debugging messages my Titanium app generates. It's been useful to watch memory usage and report on any other things that I've needed during the course of development. I felt it was $0.99US well spent.
