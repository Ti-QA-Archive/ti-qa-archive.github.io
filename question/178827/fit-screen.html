  • You don't have a lot of details in your question here. However, another way to detect modes is to set your view width and height dynamically:

    For example an Alloy view could be set like this:

    $.myView.width = Ti.Platform.displayCaps.platformWidth;
$.myView.height = Ti.Platform.displayCaps.getPlatformHeight;

    Then the view will automatically adjust based on the device. You could then add a listener to fire every time the device is rotated and fire an event to update the width and height.
