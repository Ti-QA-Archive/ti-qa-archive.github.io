  • Hi

    Try using the separatorStyle property of table view and provide it with a number constant like in the following example.

    "#tableview": {
    separatorStyle: Titanium.UI.iPhone.TableViewSeparatorStyle.NONE
}

    You could also try setting the 'separatorColor' to 'transparent'.

    If either of these aren't working like you have shown in your example, try using one or the other.

    Finally, if the issue persists, you should try different SDK's to work out if it is a bug with a particular SDK.

    Hope this helps!
