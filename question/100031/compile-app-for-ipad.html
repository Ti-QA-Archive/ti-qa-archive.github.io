Compile app for iPad

Hi

When I compile the Titanium iPhone Project with XCode for the iPad on the simulator I see an iPad optimized app.
When I want to transfer it on my real iPad, it only installs the iPhone version. (It is small and I see a button to double the size of the app)
Can I transfer it on the iPad with Titanium?

Thank you
Felix

  • Accepted Answer

    Yes, just don't use XCode to do it.

    What you want to do is in Titanium Developer select "Import Project", then select your current iPhone app. It will come up with an alert saying something like "Press Continue to import as mobile project or press cancel to import as iPad app".

    Currently Titanium does not allow for universal binary so you have to maintain two projects. Hopeful Universal Binaries will be added soon(ish).
