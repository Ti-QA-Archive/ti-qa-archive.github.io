    2 things:

    • Use the markdown syntax for formatting your code properly. It is unreadable as it is.
    • Full ListView integration in markup will not be available in Alloy until version 1.2.0. If you're feeling brave you can try out the bleeding edge version on the master branch of the github repo which currently supports this. You can install the bleeding edge version by installing via npm from the repo directly, like this:
    [sudo] npm install -g git://github.com/appcelerator/alloy.git#master

    Again, this is bleeding edge and has not gone through our pre-release testing so I can make no certification of its stability. That said, I always use the bleeding edge stuff and it helps me test if others do as well. If you do try it out, you can go back to the latest stable release at any time with a typical alloy install:

    [sudo] npm install -g alloy
