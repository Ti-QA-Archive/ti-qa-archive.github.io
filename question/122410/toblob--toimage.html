    Rather than using the database and a blob, I recommend using the file system. I assume that the source of your image is either the gallery or camera. If so, do something like the following:

    Ti.Media.showCamera({ // or Ti.Media.openPhotoGallery({
    success:function(event) {
        var fileName = // your file name here;
        var imageFile = event.media;
        var tempFile = Ti.Filesystem.getFile(Ti.Filesystem.resourcesDirectory,fileName + '.png');
        // change 'resourcesDirectory' to 'tempDirectory' for use on device
        tempFile.write(imageFile);
    },
});

    If a blob is supplied and you want to store it, just yourBlob.toImage();.
