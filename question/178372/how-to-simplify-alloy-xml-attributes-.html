How to simplify Alloy XML attributes ?

I have some structure like this:

<ScrollView layout='vertical'...>
<View layout='horizontal' ...>
   <Label ... />
   <Label ... />
</View>
<View layout='horizontal' ...>
   <Label ...  />
   <Label ... />
</View>
</ScrollView ...>

That way I got a nice grid view. Each grid acts like a button to be clicked to trigger some action. I have some attributes to be passed as parameter. I end up with something like this for each grid. E.g.:

            <View class='gridview' onClick='openItem' layout='vertical'
                    controller='contentList' cat='ALL'>
                <Label id='list' class='gridicon' text="some_icon_text" 
                    controller='contentList' cat='ALL'/>
                <Label id='settings' class='gridlabel' text="some_text"
                    controller='contentList' cat='ALL'/>
            </View>

I need to pass "controller" & "cat" to the "openItem" function in controller. Problem is that I need to repeat them 3 times in order to get the proper behavior. If I omit them in View, clicking empty area will not pass the attributes as argument. If I omit them in the Labels, clicking the Labels won't pass the attributes. I tried combinations of zIndex, bubbleParent etc… but none worked to simplify the above code.

Any pointer?

    Have you tried setting touchEnabled = 'false' on the labels? Then the click should bubble to the parent view.
