Hello,
is there a way to include google Earth in a webview and then open an kml file via google earth?
I tried to include it - viewing google Earth in the webbrowser works fine but my mobile device says that there is no google earth plugin installed. I can import the .kml in the Google Earth app but I like to write an own app.

Thanks and best Regards,

  • Hi Martin,
    googleEarth cannot run in webview (plugin). But good news: It is very simple the display KML-datas in native map view. If the KML is in net you can use YQL to get and parse it. In other case you can use the XML-parser of titanium.

    Best regards Rainer
