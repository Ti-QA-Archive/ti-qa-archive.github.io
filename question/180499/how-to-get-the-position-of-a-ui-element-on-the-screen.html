    Hi Tony,

    see you assign space_lable_l in postlayout event of label. but just next you assign left : space_lable_l-13, but that time space_lable_l undefined.. because postlayout called after creating space_img.

    so try this…

    var space_img = Ti.UI.createImageView({
        image : "/images/space.png",
        top:38,
        width : 18,
        height : 14
});
space_lable.addEventListener("postlayout",function(e){
        space_lable_l=e.source.rect.left;
    space_img.left = space_lable_l-13;
});
