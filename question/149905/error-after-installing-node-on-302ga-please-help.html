  • Titanium SDK 3.0.0.GA and 3.0.2.GA work only on Node.js 0.8.x. There were two bugs that became apparent when 0.10 shipped. We have since fixed the bugs and they will be available in the upcoming 3.1.0.GA release works with Node.js 0.8.x and Node.js 0.10.1.
