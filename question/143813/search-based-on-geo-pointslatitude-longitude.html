search based on geo points(latitude, longitude).

I have list of records (name, address, latitude, longitude) in my sqlite database table. I want to filter the records based on the user current position (latitude, longitude).

I think I need to use Haversine formula for this one.

Can any one help me, how to search sqlite database table based on geo points (latitude, longitude).

I am using Titanium 2.1.3 and Android 2.3

Thanks in Advance.

  • I would define a rectangular bounding box so you can use standard SQL queries. You can define a bounding box like this:

    var lat_min = curr_lat - radius;
var lat_max = curr_lat + radius;
var lon_min = curr_lon - radius;
var lon_max = curr_lon + radius;

    Then the query would look something like this:

    SELECT * FROM location
WHERE latitude > lat_min AND latitude < lat_max
AND longitude > lon_min AND longitude < lon_max

    Then if you really need to use a circular region, you can apply the Haversine formula to each of the results, discarding those that are not within the circle.
