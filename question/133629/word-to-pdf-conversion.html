  • I have used to try to concert doc to pdf using the following code:

    using RasterEdge.Imaging.Basic.Core;
using RasterEdge.Imaging.Basic;
using RasterEdge.Imaging.MSWordDocx;
using RasterEdge.Imaging.PDF;    

RasterEdgeImaging doc = new RasterEdgeImaging();
public void WordConvertToPdf();
{
WordInputFile = (@"C:/1.docx");
ImageOutputFile = ImageFormat.pdf;
}
doc.Save(@"C:/1.docx", 1, @"C:/1.pdf");

    I hope you success. Good luck.

    Best regards,
    Arron
