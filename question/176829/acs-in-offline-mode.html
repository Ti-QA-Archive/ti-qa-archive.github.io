ACS in offline mode

How ACS manages offline devices ?
For example if a Cloud.Places.create is called and the callback has returned error for unavailable network. In this case do the Cloud module of titanium cache the entries on device?

Is there any way to easily store it locally and again call it to sync it with ACS cloud?

  • ACS only works and would return a success to ANY cloud call if you are connected to the network. You could cache your call information in an arra, then occasionally check to see if you are connected to retry the call.
