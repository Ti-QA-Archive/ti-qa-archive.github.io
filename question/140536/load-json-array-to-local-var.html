    Hi Gerd

    This example will show you what to do.

    As Anthony said you cannot directly run PHP of course, but as long as you create a PHP page that outputs JSON and that URL is accessible by the app - then you are set.

    var xhr = Ti.Network.createHTTPClient();
xhr.onload = function (e) {
    var response = this.responseText;
    var json = JSON.parse(response);

    //json now contains the returned data parsed into json
    //as long as your php return true json this will work.
};
xhr.onerror = function (e) {
    Ti.API.log('error', JSON.stringify(e));
};
xhr.open('POST', 'http://www.someurl.com/api/something.php');
xhr.send();

    If you need to send parameters to your URL change the bottom line to something like this;

    xhr.send({
  id: 67,
  login: 'username',
  password: 'somethingelse'
});

  • You can't use PHP within the app, but if you are returning an array from the server using PHP, simply turn it into a json array, get the array from your request, then use JSON.parse(returnvalue) to set the json data to a local variable.
