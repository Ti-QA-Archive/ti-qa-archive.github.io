Animate BackgroundColor or Opacity of View's does not work on Android.

Has anyone been able to animate the backgroundColor or opacity property of a View on Android? The following code works fine in the iPhone simulator.

Note: I am currently running this animation on a random selection of views up to 9 maximum at a given time.

Property: BackgroundColor

var view = Ti.UI.createView({
     backgroundColor: "#ffffff"
});

var a = Ti.UI.createAnimation();
a.backgroundColor = '#000000';
a.duration = 100;

view.animate( a );

Property: Opacity

var iView = Ti.UI.createImageView({
     image: "image.png",
     opacity: 0
});

var a = Ti.UI.createAnimation();
a.opacity = 1.0;
a.duration = 400;

iView.animate( a );

Note: Android simulator does not throw any errors during the process. It simply ignores my animation execution.

Any suggestions as to why android ignores this?

Thanks
