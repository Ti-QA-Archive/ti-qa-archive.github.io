  • JSON.parse is used to convert a string that may contain valid JSON into a JavaScript object.
    JSON.stringify is to convert a JavaScript object into a string.

    Your declaration of the JavaScript object itemId should be:

    itemId = {
    to: null,
    month: 1,
    year: 1435,
    remaining: 4600,
    from: null,
    total_income: 5000,
    total_expenses: 400
};

    and using that you can JSON.stringify it as follows:

    var myString = JSON.stringfy(itemId);

Ti.API.debug(myString);

    Once you have it in string format, you can then parse it back into an object, and refer to its properties:

    var myNewObject = JSON.parse(myString);

Ti.API.debug(myNewObject.year);

    Hope this helps…
