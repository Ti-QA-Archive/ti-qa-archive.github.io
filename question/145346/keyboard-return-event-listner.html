  • textarea = Ti.UI.createTextArea({
suppressReturn: false,  
width: 60,
height: 60,
top: 0
});

textarea.addEventListener('return', function(e) {
Ti.API.info(' Return key  Detected');
});

    • you can do whatever you want in return event listener . if you want to change background color on return key press then do this

      textarea.addEventListener('return', function(e) {
e.source.getParent().backgroundColor = 'yellow';
});

  • Hi Ilan,

    You have to handle some code manually for correct output according to your need. here some hint from you can get return key.

    var txt= Ti.UI.createTextField({
    value : 'Test',
    height : 45,
    width : 200
});
var funReturn = function (e) {
      alert(JSON.stringify(e));
    e.source.getParent().backgroundColor = 'blue';
}
txt.addEventListener("return", funReturn);
txt.addEventListener("blur", funReturn);

    Good Luck
