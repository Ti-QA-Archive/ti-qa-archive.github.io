  • ok, i figured it out, kind of:
    (my findings only apply to a tiapp.xml where i specify an android activity)

    the project name and the app name in tiapp.xml has to be the same. so when you name your project My_Project, the app name must be My_Project as well. the activity name attribute is also tied to the project/app name and has to be My_projectActivity
