How to clear user selection in WebView

I have a iPad App which has a Button and a WebView in the same Window.

The webview will display a html page and user is allowed to tap and select text in it.
The button is used to get user's selected text and save it in database.

After selection has been saved, I wanna clear user's selection inside the webview.

I am using Javascript to do the "deselection" but no luck.

Here is the event handler of the button:

button.addEventListener('click', function(){
    webview.evalJS('deselect(); );
});

And here is the deselect function defined in the html page:

function deselect(){
    if ( document.selection ) {
        document.selection.empty();
    } else if ( window.getSelection ) {
        window.getSelection().removeAllRanges();
    }
}

After execute the deselect() function, I can see the `Copy|Define' popover menu disappear, but the selection overlay is still there.

How to remove that selection overlay?
