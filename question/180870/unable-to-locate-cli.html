Unable to locate CLI

Attempting to create an alloy application, receiving "unable to locate CLI" on Windows 7, selecting only mobile web type. Completely uninstalled studio and node, reinstalled with your installer. Titanium 3.4.1 and node v.0.10.13 installs properly from package installer. I added the path to titanium so "titanium -v" on the command line yields 3.4.1. So if it can be found by DOS, why can't it be found by studio?

I also spent about 6 hours trying to get an ubuntu VM working, suffering the exact same issue. It would be stellar to have a vm image available for download.

    Hello Daaron Dwyer,

    I think your CLI is not installed properly. After installing node try to install CLI using this command.

    npm install -g titanium

    Then try this command for quick setup

    titanium setup quick

    If your problem is not solved then use this command and post your log file for further investigation

    ti info

    For more info follow this link.

    if you have any concern, then let me know.
