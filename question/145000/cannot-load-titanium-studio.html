Cannot load Titanium Studio

I cannot load Titanium Studio in OSX 10.7.5 , cause I have this alert:
The JVM shared library "/System/Library/Frameworks/JavaVM.framework"
does not contain the JNI_CreateJavaVM symbol.

Anyone knows how to solve the problem?

    In case somebody comes across this I fixed it by re-installing Java. I spent all day trying to uninstall Java 1.7 and 1.6. I deleted the virtual machines and everything. After that I was getting a different error (A Java Runtime Environment (JRE) or Java Development Kit (JDK) must be available in order to run Eclipse. No Java virtual machine was found after searching the following locations…). It didn't work until one random restart the software update tool recognized it was missing and installed it. It worked after that. Good luck everyone!

  • I had the same problem. I downgraded my java from 1.7 to 1.6 because of a bug when I wanted to upload the app on iTunes Store.

    Just put this in your terminal :

    cd /System/Library/Frameworks/JavaVM.framework/Versions
sudo ln -s /System/Library/Frameworks/JavaVM.framework/Versions/A Current
