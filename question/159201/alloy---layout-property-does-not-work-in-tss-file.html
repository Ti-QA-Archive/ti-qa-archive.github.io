".container" : {
    height : Ti.UI.FILL,

}

Have to set it outside in the controller OR markup

  • You do not use tss inside the controller, it is suppose to be applied to the view like so: 

    //Inside the View
<Alloy>
    <Button class="small_button" id="small_button"/>
</Alloy>

    //Inside TSS
".small_button":{
    height: 50,
