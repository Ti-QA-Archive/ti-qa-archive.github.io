getTiContext() is undefined for the type KrollInvocation

Hi there,

I've started a module as per the instructions, imported the project into eclipse and added the libs kroll-apt.jar and kroll-common.jar to the build path.

This has worked fine. However, I added a method defined like so:

@Kroll.method
    public void toastString(KrollInvocation invocation){
        Activity activity = invocation.getTiContext().getActivity();
        Toast.makeText(activity.getApplicationContext(), "Hello World", Toast.LENGTH_LONG);
    }

And on the line Activity activity = invocation.getTiContext().getActivity(); I'm getting the error "The method getTiContext() is undefined for the type KrollInvocation".

Any idea why this could be. Its as if the method just doesn't exist. This is ALL I have added from the example code that was generated when I made the project on the commandline. I'm using titanium sdk version 1.8.1. I haven't deleted any code either.

This code I added to the Module class (public class AnotherModule extends KrollModule).

What's wrong here, and how do I fix it.

Thank you.
