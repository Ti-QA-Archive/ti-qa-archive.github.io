I've used the code below that I found on another post but when we use this the font size of the picker rows are decreased as well:

    var transformPicker = Titanium.UI.create2DMatrix().scale(1.0);
    var pick_shelf_life = Ti.UI.createPicker({
        top: 30,
        left: 230,
        width: 150,
        height: 150,
        transform: transformPicker
    });

Is this not possible? If not, are there any alternatives to the picker control that perform similar functionality that can be set to a specific height?
