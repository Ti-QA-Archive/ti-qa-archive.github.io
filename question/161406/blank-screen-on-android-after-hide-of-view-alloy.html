Blank screen on android after hide() of view alloy

I open a new controller using the 

Alloy.createController('detail', {
        name : currVehicleInfo.vehicleID,
        vehicleid : currVehicleInfo.vehicleID
    }).getView().open();

I run 

$.detail.hide();

on the file that is opened. It hides correctly on iPhone but on android it shows a blank screen instead of going back to the index screen that opened it.

— asked January 10th 2014 by David Clymer
