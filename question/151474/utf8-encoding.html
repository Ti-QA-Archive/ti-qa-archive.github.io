UTF8-encoding

Hi,
unfortunately, there are XML documents in Latin-1. After parsing by XMLTools.js I get ugly texts. How can I convert like utf8_encode for PHP?

iOS/Android SDK 3.1 OSX 10.8

— asked April 24th 2013 by Rainer Schleevoigt

  • The only way you could achieve this would be to ask specifically to the server to give you UTF-8 content. You can do this by adding the following header to your HTTP call.

    Content-Type: text/html;charset="utf-8"

    There is no way you could convert a charset using JavaScript. Because if your server gives you Latin-1, you will be stuck with it forever. So you must address this conversion before it reaches your device.
