  • Immediately roll back your setup - if you're up against it and need to get a release out you can still build to iOS7 SDK using 3.13.

    So,

    use Time Machine to rollback your Titanium Studio install

    then to uninstall the CLI

    $ sudo npm uninstall titanium

    and uninstall Alloy

    $ sudo npm uninstall alloy

    then

    $ sudo npm install -g titanium@3.1

    and

    $ sudo npm install -g alloy@1.2

    and Hey Presto, back to your old setup.
