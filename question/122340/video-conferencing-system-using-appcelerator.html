Video conferencing system using appcelerator

I would like to create a cross platform video conferencing system.

1) Can I use Appcelerator to stream video from a web camera to an HTML based window ?

2) If the answer is yes, would it support Windows and Mac ?

3) If the answer is yes, would it be possible to show the Appcelerator application inside a web browser, say IE, Firefox, Chrome, etc. ?

4) I can develop a cross browser cross platform npapi plugin (using Firebreath) that would enable me to capture and stream video. Is it possible to make this plugin work inside Appcelerator ?

    1. 1) Yes
    2. 2) It would be your job to make it cross compatible, but its possible.
    3. 3) It sounds a bit counter-intuitive to the idea behind building a native app, so I'm not sure this is possible in the way that you are thinking.
    4. 4) This may not work out the box the way you want it to, but Titanium is completely extensible and I believe the functionality you are looking to achieve can be added as a module extension.

    With all that said, everything you are looking to do can be done outside of Titanium and then wrapped up using Titanium. I mean, you could have a camcorder streaming audio/video to an HTML5 media player in about 20 minutes, add another 20 and you can wrap all the functionality inside a Titanium application. The idea is that you don't necessarily have to rely on Titanium specific functionality in order to accomplish your goals. Rely on your existing web development skills to rock that functionality.
