Display app name next to logo in android app?

I am working on an alloy project , and I'm wondering how I can get the app name next to the icon on android ? We're talking about the upper left icon. If I open a new window on-click it opens a new window without navigation and then it displays the app name , so I think it has something to do with the tabgroup ? I want to show the title at any given time.

To sum it up : win1 and win2 need to display a title , the "details" window that is opened on-click displays it by default.

simplified version of my alloy xml markup : 

<!-- index.xml  -->
<Alloy> 
        <TabGroup id="mainTabGroup"> 

            <Tab id="tab1" > 
                <Window id="win1"> 
                     <WebView id="webview" url="http://192.168.1.28/a6/twitter.html" />
                </Window> 
            </Tab>

            <Tab id="tab2" > 
                <Window id="win2"> 
                    <TableView id="tableView" onclick="details"/>
                </Window> 
            </Tab>
            </TabGroup> 
</Alloy>

Setting the title in-line or in the .tss does not seem to have any effect. I also tried setting the title in the controller but that also gave no results.

// index.js ( controller )
$.readWin.title="Groot in auto's";

    Here is the code I use to customise my actionbar, should be pretty self explanitory but any problems just ask

    if (OS_ANDROID) {
    $.centerView.addEventListener("open", function() {        
        if (!$.centerView.activity) {
            Ti.API.error("Can't access action bar on a lightweight window.");
        } else {
            var actionBar = $.centerView.activity.actionBar;
            if (actionBar) {
                actionBar.displayHomeAsUp = false;
                actionBar.icon = "/appicon.png";
                actionBar.title = Ti.App.name;
                actionBar.onHomeIconItemSelected = function() {
                    Ti.API.info('Home Icon Item Selected');
                 };
            }
        }
    });
}
