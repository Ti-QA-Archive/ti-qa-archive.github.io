Titanium 3.1.3 get device token

I am doing push notification with titanium, When I use Titanium sdk 3.2.x, the device token is found but when I change to 3.1.3 , but the code are totally same, however it cannot get the device token anymore. So what is the reason for this ?

    Hello Ronn,

    Thanks for your inquiry. The reason you can't get the device token with Ti SDK 3.1.3 is ''ti.cloudpush" module is not supported with SDK 3.1.3.GA. As of Release Ti SDK 3.2.0.GA, the "ti.cloudpush" module version 3.2.1 supports Google Cloud Messaging (GCM) for push notifications.

    For more info on "ti. cloudpush" module, please follow this Link.

    Thanks.
