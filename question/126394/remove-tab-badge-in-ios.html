Remove tab badge in iOS?

I know how to set a tab badge, as in tg.tabs[2].badge = foo.length;

But I'd like to remove it if the variable is zero, as in 

if (foo.length > 0) {
  tg.tabs[2].badge = foo.length;
} else {
  // remove badge altogether instead of '0' badge
}

What's the syntax for this?

SDK 1.7.2, OS 10.7, Studio 1.0, iOS 4.x

    set the value of the badge to null, so:

    if (foo.length > 0) {
  tg.tabs[2].badge = foo.length;
} else {
  tg.tabs[2].badge = null;
}
