    • Because what you want to do is to implement the HTTP protocol. The browser (client) sends a HTTP GET request, something like 

      GET /index.html HTTP/1.1
 Host: www.example.com

      and waits for a standard HTTP response. You, the server, should answer sonmething like 

      HTTP/1.1 200 OK
 Date: Mon, 23 May 2005 22:38:34 GMT
 Server: MyTitaniumServer 1.0
 Last-Modified: Wed, 18 Jun 2011 23:11:55 GMT
 Etag: "3f80f-1b6-3e1cb03b"
 Accept-Ranges: bytes
 Content-Length: 12
 Connection: close
 Content-Type: text/html; charset=UTF-8
 Hello world!

      (this is a sample, do not expect to work properly at first try…!). Take a look to http://www.jmarshall.com/easy/http/ for a brief intro to HTTP.
