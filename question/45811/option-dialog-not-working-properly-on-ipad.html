  • I had to set up my options this way to get the last option, Cancel, to show on the iPad:

    var a = Titanium.UI.createOptionDialog({
        title:    customerName,
        options: ['Sign Out', 'Delete', 'Cancel', ''],
        destructive:  1,
        cancel:  2
    });
