TabGroup within NavGroup title bar - ipad

Opening a navGroup and moving to a second window as per this tutorial shows the title of the second window in the title bar, and a nice back button.

If I were to replace this second window with a tabGroup: pastie

I get 2 title bars. I can hide the second of these title bars by setting navBarHidden:false on win1 and win2, but I can't figure out how to set the title of the top bar, nor add the rightNavButton to it.

How do I consolidate these title bars so that when I open my tabGroup I end up with one title bar, with the title of the tab, the back button, and the rightNavButton?

Any assistance anyone could provide would be greatly appreciated :)

    I'm not sure this approach is ok.
    Usually the tabGroup needs to be a root object, so not attached to another window. Maybe you should consider opening the tabgroup as a separate entity ( with a flip animation to be nice ) and use your own custom back button to go back to the main window.
