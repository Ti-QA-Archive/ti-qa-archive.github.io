Add-on Google Apis version 8 required

TitaniumStudio can't find any Google APIs. I installed SDK Android for mac.

When I run my SDK manager he shows a "SDK platform" to install and "Samples for SDK" to install, but no "Add-on Google APIs" to install. My Add-ons folder after installation of SDK is empty as well.

Version mac 10.5.8.
Titanium build 3.0.1.201212181159
Titanium build date 18 December 2012
Platform Android 17

  • I solved the problem. It seems to be a known problem: http://code.google.com/p/android/issues/detail?id=30659

    I changed the "Java 5" to "Java 6" environment in the application "Java Preferences" and restarted Android.
