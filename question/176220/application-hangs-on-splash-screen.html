  • Hi

    A hanging splash screen simply means that something in your code that occurs during the start up is broken. Do not go off and try and remove things before you have performed sufficient tests.

    Create a brand new default app and compile this - does that work on your device? If yes then it is neither the SDK setup or your device. So it is going to be the code.

    If it is the code you need to work out what you added since it last started on the device and comment out that code until it starts again correctly. When it does then review your code and add back in small section at a time until it no longer works - that section will be the issue. Either fix it or post it back here as a comment on your own question. NOT the whole app the bit that causes the error.
