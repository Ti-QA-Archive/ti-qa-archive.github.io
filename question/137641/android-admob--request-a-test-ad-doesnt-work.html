  • Hello, Hopes this helps someone… My Solution was as follows…

    • Download & Install AdMob Module for Android - Latest - https://github.com/appcelerator/titanium_modules/tree/master/admob/mobile/android/dist
    • Edit tiapp.xml - On the GUI Page(Overview tab), Add the module on the right and select the build. Now click on the tiapp.xml tab on the bottom left, replace the following :

    this —>
    <android xmlns:android="http://schemas.android.com/apk/res/android"/>
    with –>
    <android xmlns:android="http://schemas.android.com/apk/res/android">
    <manifest>
    <uses-sdk android:targetSdkVersion="13"/>
    </manifest>
    </android>

    • Save & then close everything and Clean the project.
    • Log in to AdMOb and goto Monetize section, there you can manually add your app. There you will get a NEW pub ID for that App - example - ca-app-pub-8240072909759242/5483057458
      *Use that number as the Views Pub ID whereever you want to place that ad -

    // then create an adMob view like so
    var adMobView = admob.createView({
    publisherId:"ca-app-pub-8240072909759242/5483057458",
    testing:false, // default is false
    bottom: 0, // optional
    });

    You can now install or run in emulator and see admob ads. Again create campaigns and use wisely throughout your apps and games.
