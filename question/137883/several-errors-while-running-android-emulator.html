  • I thought that using titanium was a good way to make apps, but i only get errors and errors.

    on windows 7

    Install the sdks and your workspace to a location without spaces in the path.

    The window dev boxes in our shop all have titanium, android, and blackberry tools and sdks in \dev so there are no spaces anyplace. Doing that eliminates nearly all our build problems. If after you do that you still have problems, let me know.
