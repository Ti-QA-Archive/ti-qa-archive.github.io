  • Hi

    All image references should start with a / prefix, so these;

    backgroundImage:'images/btn.png',
backgroundFocusedImage:'images/btn_focus.png'
// become
backgroundImage:'/images/btn.png',
backgroundFocusedImage:'/images/btn_focus.png'

    The same is true of this;

    <View class="container" id="view_Login" backgroundImage="images/bg.png">   
// becomes
<View class="container" id="view_Login" backgroundImage="/images/bg.png">

    Although any visual property should be in the TSS not the XML, so this would look like;

    <View class="container" id = "view_Login">
// background in the tss

    The image files would be found in any of these places;

    {appname}/app/assets/images/bg.png // any platform
{appname}/app/assets/android/images/bg.png // platform specific
{appname}/app/assets/iphone/images/bg.png // platform specific
// or a theme folder - if you are not using these do not worry
