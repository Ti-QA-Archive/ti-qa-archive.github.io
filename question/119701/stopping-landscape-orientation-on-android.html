  • I just noticed there is still a really old bug for Android.
    If you create your window with: 

    var win1 = createWindow({  
    orientationModes : [Titanium.UI.PORTRAIT ]
});

    It will still allow you to rotate in landscape.

    But if you do:

    var win1 = createWindow();
win1.orientationModes = [Titanium.UI.PORTRAIT ];

    That will work!
