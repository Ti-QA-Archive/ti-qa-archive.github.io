Is there a documentation only for Android ?

    There are icons in the top-right-hand corner of the API docs, to indicate whether the functionality relates to iOS or android. For example, Ti.UI.DashboardView is iOS-only whereas Ti.UI.TableView works on both platforms.

    The code you pasted will not work because it contains mistakes:

    var win = Titanium.UI.createWindow()
var view = Titanuium.UI.createView()
var btn = Tintanium.CreateButton()
win.add(btn)
veiw.add(btn)

    It should be:

    var win = Titanium.UI.createWindow();
var view = Titanium.UI.createView();
var btn = Titanium.UI.createButton();
win.add(view);
view.add(btn)

    Hope this helps
