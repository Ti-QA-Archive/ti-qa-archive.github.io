    var d1 = moment.utc(datestr).format();

    Hello Dinesh,

    Download this file moment.min.js / moment-with-langs.js from Moment. And copy this file to your application resource folder. Resource&#x2F;moment.js. 

    

    var moment = require('moment');   //  File name only
    //you also use ('/moment').
    var datestr = '2012-11-30T01:30:00+0000';

    var d1 = moment(datestr);
    Ti.API.debug("d1: " + d1.toString());

    var d2 = moment(datestr, "YYYY-MM-DDTHH:mm:ssZ");
    Ti.API.debug("d2: " + d2.toString());

    For more information about Moment.js file see it's Documentation . I hope this is very useful to you and solved your problem. thanks.
