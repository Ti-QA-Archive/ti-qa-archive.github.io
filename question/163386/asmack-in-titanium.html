Asmack In Titanium

Hi, I would like to develop an app that allows you to chat or have communication between the mobile and the server using the XMPP protocol.

For being more especific here is the library: https://github.com/flowdalic/asmack

So the trouble that I got is that I cant find the way to include or import it to a Mobile Proyect in Titanium.
