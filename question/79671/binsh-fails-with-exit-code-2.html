  • Xcode will show you what the problem is, usually. When Xcode fails on build, you can click the exclamation point in a red circle and it will take you to the log. In a line or two above the "failed with exit code 2" will give you your answer.

    In my experience, the most common culprit is a PNGCrush error, where PNGCrush thinks one of your images is corrupted. This is easily fixed by opening the image file in an editor and resaving it (you don't even have to do anything, just tell the editor to "Save As…" and select the file and confirm the overwrite).
