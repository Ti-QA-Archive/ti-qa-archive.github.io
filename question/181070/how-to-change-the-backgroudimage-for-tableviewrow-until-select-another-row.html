    Hi Husserin

    @narendra what about selected row image or background and as per your suggestion on first row selection nothing will happen..

    try this one

    ...

var row = Ti.UI.createTableViewRow({
    height : 35,
    backgroundColor : "white",
});
...
var preRow = undefined;
$.tblView.addEventListener("click", function(e) {

    if(preRow != undefined){
        preRow.backgroundColor = "white";
    }
    e.source.backgroundColor = "red";
    preRow = e.source;
});

    Hope this is helps you..

  • Put the a reference in the code for the row clicked and then change the background color. Here is sample code,

    
var oldRow;

tableViewRow.addEventListener("click", function(e) {
    if(oldRow)
        oldRow.setBackgroudImage("old.png");
    oldRow = e.source;
});
