row rightImage size

Hello,
Im creating a table dynamicaly and im setting a rightImage for each row.

var row = Ti.UI.createTableViewRow({
                id: i,
                height:"50dp",
                color:'white',
                height:'auto',
                rightImage: 'marker.png'
          });

Is there any way to set the rightImage size?
I want to do this in order to give it a relative size to show it the same in all screen sizes.

thank you

  • you can just add the rightImage as an image on top of the row:

    var rightImage = Ti.UI.createImageView({width:"50dp", height:"30dp", image:"marker.png");
row.add(rightImage);

    and make sure to set the size in dp, so that the image size will be relative (as you said) to all the screen sizes.
