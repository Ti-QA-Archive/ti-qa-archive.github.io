Testing for devices < iOS 7 Simulator

Just a heads up on something I discovered that may help some people with down-testing for lower iOS devices when building for iOS 7. As Ingo's blog post discusses, there is a lot of backwards compatibility conditionals and testing that will need to be done. After you install the latest Xcode with iOS7 SDK, you'll may need to manually install the older simulators for iOS 6 and less.

To install older simulators, open Xcode and go to Xcode > Preferences and select DOWNLOADS. In the Components section there's a list of older simulators you can install. Click the ones you want to install. After install is complete, restart Studio.

When Studio opens, select RUN > Run Configurations and under simulators you should see the newly installed options.

I hope this helps someone.
