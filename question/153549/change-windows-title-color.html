    You can do this using Android themes. Create a Example project and in Example/platform/android/res/values/theme.xml put below code. Create the folders.

    <?xml version="1.0" encoding="utf-8"?>
<resources xmlns:android="http://schemas.android.com/apk/res/android">
    <style name="MyTheme" parent="android:Theme">
        <item name="android:windowTitleBackgroundStyle">@style/WindowTitleBackground</item>
        <item name="android:windowTitleStyle">@style/WindowTitle</item>
    </style>

    <style name="WindowTitleBackground"> 
        <item name="android:background">#f00</item>
    </style>

    <style name="WindowTitle" parent="@android:style/TextAppearance.WindowTitle">
        <item name="android:textColor">#0f0</item>
    </style>
</resources>

    In tiapp.xml replace the line 

    <android xmlns:android="http://schemas.android.com/apk/res/android" />

    for

    <android xmlns:android="http://schemas.android.com/apk/res/android">
    <manifest>
        <application 
            android:icon="@drawable/appicon" android:label="Example"
            android:name="ExampleApplication" android:theme="@style/MyTheme"
            android:debuggable="false">
            <activity
                android:configChanges="keyboardHidden|orientation"
                android:label="Example" android:name=".ExampleActivity">
                <intent-filter>
                    <action android:name="android.intent.action.MAIN"/>
                    <category android:name="android.intent.category.LAUNCHER"/>
                </intent-filter>
            </activity>
        </application>
    </manifest>
</android>

    This will apply a custom title for all windows. If necessary, make a Project -> Clean and use navBarHidden:false.
    For more information refer to this link
    Android Themes
