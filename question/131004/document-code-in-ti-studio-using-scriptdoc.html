  • Mads,

    After much frustration, we went with the jsDoc Toolkit.

    Called from a ruble that calls the following:

    java -jar /android-sdk-mac_86/jsdoc-toolkit/jsrun.jar /android-sdk-mac_86/jsdoc-toolkit/app/run.js Resources/ -t=/android-sdk-mac_86/jsdoc-toolkit/templates/jsdoc -r=4 -d=Documentation/

    I know this doesn't solve your ScriptDoc issues, but its how we resolved ours.

    • Mads,

      Sorry for the silence. I must have unsubscribed. The DevMail was effective, however. grin

      You will need to check your paths.
      This particular system had jsdoc-toolkit installed into the android-sdk directory.

      By default, we create a Documentation&#x2F; folder in the project root at the same level as Resources.

      Download the ruble from DropBox:
      jsDocs.ruble

      The jsDocs.ruble directory goes into $HOME&#x2F;Documents&#x2F;Aptana Rubles

      After a restart of TiStudio, you should have a jsDocs menu under Commands.
      Again, edit the bundle.rb to reflect your correct paths.
