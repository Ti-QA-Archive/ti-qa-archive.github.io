hide show functionality

Perhaps a defect exists for this already, but my experience with hide() & show() on Android and SDK 1.6 is that those methods only work if the view object was added directly to a window.

Example 1 - the following code works in hiding and showing b1:

var b1 = Titanium.UI.createButton({title:'button'});
Titanium.UI.currentWindow.add(b1);
bi.hide();
// some other code
bi.show();

Example 2 - This piece of code does not work, hide()/show() have no affect, and b1 is always visible.

var b1 = Titanium.UI.createButton({title:'button'});
var v1 = Titanium.UI.createView({height:100});
v1.add(b1);
Titanium.UI.currentWindow.add(v1);

bi.hide(); 
// some other code
bi.show();

All of the Kitchen Sink examples are of the first variety and do work on Android & SDK1.6. Anyone else get hide()/show() to work within the context of other container views on Android, or is this a known issue?

  • It appears that setting the visible property directly works better, so experiment with both methods if you run into problems.

    b1.visible=false;
// some other code
b1.visible=true;
