  • Your jsObj.alerts is not an array, is an object, therefore you cannot use a length loop on it as the objects don't have length. Use for in or an uderscorejs utility for collections (each in this case).

    Then when you got each element of the alerts you can use a "normal" for to get all the alerts in that day.
