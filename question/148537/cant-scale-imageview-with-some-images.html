Can't scale imageView with some images

Hi,
here is problem
i create imageView 

var head = Titanium.UI.createImageView({
    images: ['images/head.png', 'images/head1.png','images/head2.png','images/head3.png'],
    duration:100,
    repeateCount:0,
    top: 10
})
head.stop();

head.width =  Math.round( head.toImage().width*Ti.App.SCALE ); // SCALE is realSize/defaultSize
head.height = Math.round( head.toImage().height*Ti.App.SCALE );

window.add(head);
head.play()

but images are not scaled. What did i miss?
