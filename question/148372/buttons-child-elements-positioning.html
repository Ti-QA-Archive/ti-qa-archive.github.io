    Hi

    Android is not as easy as iOS as you cannot add child views to other views that are not designed to accept them.

    However it is possible;

    var outer = Ti.UI.createView({
    height: 50,
    width: 200
});
var btn = Ti.UI.createButton({
    height: 50,
    title: '',
    width: 200
});
outer.add(btn);
var inner = Ti.UI.createView({
    height: Ti.UI.SIZE,
    layout: 'horizontal',
    touchEnabled: false,
    width: Ti.UI.SIZE
});
outer.add(inner);
var img = Ti.UI.createImageView({
    height: Ti.UI.SIZE,
    image: '/KS_nav_ui.png',
    touchEnabled: false,
    width: Ti.UI.SIZE
});
inner.add(img);
var title = Ti.UI.createLabel({
    color: '#000',
    height: Ti.UI.SIZE,
    text: 'Title',
    touchEnabled: false,
    width: Ti.UI.SIZE
});
inner.add(title);
win1.add(outer);

    This will create an outer view add a button and then overlay an inner view that will display its children as items that run side by side, this will site over the button and not respond individually to touch events. This allows the button only to receive the events.

    You can wrap this up as a function if you need to re-use it.

    The code is tested on iOS and Android and works perfectly.

    Have fun.
