SearchBar Text Color - Android

I have a searchBar attached to a tableview that works great except the text color on Android is too light, very difficult to see the font. This is not an issue on iOS. There doesn't seem to be a color setting for the font… any ideas on how this can be changed?

  • According to the docs for the TextField object there is a colour: property when creating the textfield. I assume that is what you are using?

    This is the example they give:

    var textField = Ti.UI.createTextField({
  borderStyle: Ti.UI.INPUT_BORDERSTYLE_ROUNDED,
  color: '#336699',
  top: 10, left: 10,
  width: 250, height: 60
});

win.add(textField);
