Highlight variable useage in a file

Anyone knows how to change the settings in Titanium Studio to be able to mark a variable/word and immediately see where that variable is used elsewhere in the file? Like default behaviour in notepad++

  • Accepted Answer

    Preferences > Titanium Studio > Editors > JavaScript, check "Mark occurrences with background color".
