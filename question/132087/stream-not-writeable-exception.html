Stream not writeable exception

Hi,

Im making an app which can control a device in my network. The commands can be send trough a socket to the device. It works fine with PHP. See code below.

$fh = fsockopen('192.168.2.3', 30000, $var1, $var2, 30);
fwrite($fh, 'x');
fclose($fh);

Now i want to do the same with an iphone app. See code below:

var connectSocket = Ti.Network.Socket.createTCP({

        port: 30000,
        mode: Titanium.Network.READ_WRITE_MODE,
        connected: function(e) {
            Ti.API.info('connected to ' + e.socket.host);
            //Ti.Stream.write(connectSocket, Ti.createBuffer({value: 'x'}), writeCallback);

            Ti.API.error(" error description <" + e.error + ">");
        }
    });

connectSocket.connect();
connectSocket.write(Ti.createBuffer({value: 'x'}));
connectSocket.close();
