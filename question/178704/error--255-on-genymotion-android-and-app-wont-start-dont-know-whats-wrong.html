[ERROR] : 255 on Genymotion Android and app won't start. Don't know what's wrong

So I have been using Titanium for a while and all have been fine. However today I installed Titanium on a new computer, and it won't launch the app on a Genymotion Android emulator. The log:

[INFO] :   Writing build manifest: /Users/JX/Dropbox/Titanium/spacious-mobile/build/android/build-manifest.json
[INFO] :   Making sure the adb server is running
[ERROR] :  255
[INFO] :   Project built successfully in 23s 272ms

It's totally weird because it seems that all things are done before launch and even the signed apk is created. I can even go into build folder and drag-and-install the signed apk onto Genymotion to install and run it perfectly.

Does it mean that Titanium failed to transfer the apk onto Genymotion emulator for some reason? If so, how can I resolve it.

Thanks!

  • OK I found the error: It's actually a new bug introduced by Genymotion. Solution is here:

    Simply, just set "use custom Android SDK tools" in Settings -> ADB of Genymotion, instead of using Genymotion's default tools.
