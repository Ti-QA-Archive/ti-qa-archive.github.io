  • open the API: http://docs.appcelerator.com/titanium/latest/ and have a look at the Button, TextField and Label example. There is some code you can use to start. Basically you have to add elements to the views (<Label id="lbl1"></Label>…), then style them in the .tss files ("#lbl1":{left:10,top:10,..}) and add some functions in the controller ($.lbl1.value = "test";).
