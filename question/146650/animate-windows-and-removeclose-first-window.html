Animate windows and remove/close first window

Hi

In my app the frontpage consists of 2 pages and a tabgroup inbetween. The first page is a login page, the next is a "whats new" page which is avaiable even when you are not logged in.

As I understand it I should be able to close the tabgrouped content with a $.login.close; or $.login.remove; What happens is that the application crashes.

I'm using the Alloy framework. Please have a look at my code.

index.xml

<Alloy>

    <!-- Welcome Window -->
    <TabGroup>
        <Require src="login"/>
        <Require src="news"/>
    </TabGroup>

    <!-- Main Window -->
    <Window id="appWin" title="Breure PlantDeco">
        <TabGroup>
            <Require src="categories"/>
            <Require src="details"/>
            <Require src="contact"/>
        </TabGroup>
    </Window>

</Alloy>

login.xml

<Alloy>
    <Tab id="tab_one" title="Inloggen">
        <Window id="login" title="Inloggen" class="container">
            <ImageView id="logo"/>
            <ImageView class="login_wrapper"/>
            <TextField id="text_email" class="text_input" value="E-mailadres"/>
            <TextField id="text_password" class="text_input" value="Password"/>

            <Label id="noAccount" text="Nog geen account, klik dan hier" />

</Alloy>

var t = Ti.UI.iPhone.AnimationStyle.FLIP_FROM_LEFT;

function login() {
    $.login.animate({view:$.appWin,transition:t});

    $.login.remove();
}

    You can't close a window that is attached to a tab, as that tab would no longer have a corresponding window. I always suggest that you don't attach a login window to a tab at all. You open up your tabgroup, then check if the user is logged in, if not, then you open the login window as a modal window over the tabgroup. Once the user logs in, then you can close the modal window and return to your tabgroup.
