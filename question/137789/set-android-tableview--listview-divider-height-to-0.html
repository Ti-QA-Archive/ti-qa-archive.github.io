Set android tableview / listview divider height to 0

I have a design that works on the iPhone, using a faux grouped table view style. Its a normal table with a background image on each row that doesn't fill the whole width, and separator style set to none. The trouble is that I cannot hide the divider lines on Android. Setting the tableView separatorColor to "transparent" leaves a 1dp gap where the background image shows through.

I don't know much about native Android development, but I know native ListViews (which I presume Ti.UI.TableViews are based on) have a setDividerHeight() method.

Is there any way of actually setting the height of the separator (to zero) or including a xml layout in platform/Android/res/layout/


function TheWindow()
{
    // create window
    self = Ti.UI.createWindow
    ({
        backgroundImage:"windowBackground.png",
        title:L("more_window")
    });

    // create table
    var tableView = Ti.UI.createTableView();

    // create rows
    var rows = []
    rows[0] = Ti.UI.createTableViewRow //new ui.TableViewRow
    ({
        backgroundColor:"transparent",
        backgroundImage:"paperBackground.png",
        height:60
    });
    var rowView = Ti.UI.createView({left:20,right:20})
    var rowTitle = Ti.UI.createLabel({left:10,right:10,text:"row1"});
    rowView.add(rowTitle);
    rows[0].add(rowView);
    rows[1] = Ti.UI.createTableViewRow //new ui.TableViewRow
    ({
        backgroundColor:"transparent",
        backgroundImage:"tableViewRowBackground.png"
    });
    var rowView = Ti.UI.createView({left:20,right:20})
    var rowTitle = Ti.UI.createLabel({left:10,right:10,text:"row2"});
    rowView.add(rowTitle);
    rows[1].add(rowView);

    // add data
    tableView.setData(rows);
    self.add(tableView);
}
module.exports = TheWindow

  • Accepted Answer

    I managed to get rid of the separator on Android by creating an Android Module with the 2.1.4 SDK.

    I added the following imports (titanium studio may complain that it can't find the ti.modules.titanium.ui namespace when compiling the module, but just skip that, it compiles fine):

    import ti.modules.titanium.ui.TableViewProxy;
import ti.modules.titanium.ui.widget.TiUITableView;
import ti.modules.titanium.ui.widget.tableview.TiTableView;
import android.widget.ListView;

    And following code added to the AndroidModule class:

    @Kroll.method
public void hideTableRowSepartor(TableViewProxy tableViewProxy)
{
    ListView tableListView = tableViewProxy.getTableView().getListView();
    tableListView.setDividerHeight(0);

    Log.d(LCAT, "Divider height set to zero!");
}

    And then in javascript, creating the tableView and then calling the above code:

    var table = Ti.UI.createTableView(...);

var module = require('mymodule');
module.hideTableRowSepartor(table);
