Blackberry Dev within Titanium Studio

I'm trying to get going with Blackberry dev on Windows 7 x64 however am having issues. I followed this guide:

http://support.appcelerator.com/kb/1/tickets

However it is for Titanium Developer, whereas I am using Titanium Studio. It has you install a separate instance of Eclipse, which doesn't make much sense when Titanium Studio is itself a version of Eclipse.

Within Ti Studio I go to Window > Preferences and click the Get SDK link, which then lets me pick the Blackberry SDK v5.0.0.25 to install. After that it reloads Ti Studio and in preferences it shows the Blackberry SDK Directory set to:

C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Appcelerator

The Blackberry Keys Directory is empty and the Default Blackberry Simulator dropdown is also empty. I am using the latest 1.7 version of the Ti Mobile SDK.

Has anyone gotten Blackberry dev working with Ti Studio? I see multiple other threads where people are having the same issue, the only reason I got Indie was for Blackberry, thanks…
