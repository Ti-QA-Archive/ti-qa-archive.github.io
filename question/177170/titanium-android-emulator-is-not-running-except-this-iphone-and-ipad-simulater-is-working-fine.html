  • There is no "Titanium Android Emulator". Titanium doesn't have it's own proprietary stuff… it uses the actual Android and iOS developer tools. The answer is right there in your error:

    [ERROR] : Target Android SDK 14 is not installed To target Android SDK 14, you first must install it using the Android SDK manager.

    You didn't install the Android SDK properly. See how to properly set up your machine.
