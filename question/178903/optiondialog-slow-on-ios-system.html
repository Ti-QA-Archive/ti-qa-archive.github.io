OptionDialog slow on iOS system

I am trying to use an OptionDialog to choose an option from a list of about 50 elements.
On android it works well but on IOS 8 the dialog is very slow, it takes abour 5 seconds to appear.
Is it a known problem ?

    Hello Roberto,

    I have tested the "Titanium.UI.OptionDialog" functionality with 50 elements as option in iOS platform. It took less than a second to load. Here is my code

    Ti.UI.setBackgroundColor('white');
var win = Ti.UI.createWindow({
  title: 'Click window to test',
  backgroundColor: 'white',
  exitOnClose: true,
  fullscreen: false
});

var options=[];

for(var i=0,j=50; i<j; i++){
 options.push("Option "+i) ;
};

options.push("cancel") ;

var opts = {
  cancel: 50,
  options: options,
  selectedIndex: 2,
  destructive: 0,
  title: 'Delete File?'
};

win.addEventListener('click', function(e){
  var dialog = Ti.UI.createOptionDialog(opts).show();
});

win.open();

    Here is my platform specification

    Mac OSX 10.9.5

    Titanium SDK 3.4.0.GA

    Titanium CLI 3.4.0

    Xcode 6.0.1

    iOS simulator 8.0

    Try cleaning your project and rebuild. Thanks.
