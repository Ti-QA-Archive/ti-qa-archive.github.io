iPhone StatusBar Styling Errors, Need help!

I have searched and searched the Q&A trying to find an answer this this issue with no luck. All the suggestions people have posted to try and change the iPhone statusbar style in an application do not work.

I need to change the StatusBar to "opaque_black". It works perfectly in the Kitchen Sink app, but when I copy the same code from the Kitchen Sink "statusbar.js" file and paste it into my "app.js" it doesn't work. It still shows up as the default grey.

Here is the code I copied directly from the "statusbar.js" file in the Kitchen Sink.

Titanium.UI.iPhone.statusBarStyle = Titanium.UI.iPhone.StatusBar.OPAQUE_BLACK;

Here is the Error that keeps coming up in the Red Screen on the iPhone Simulator and in the Titanium info screen.
"[ERROR] Script Error = Result of expression 'Titanium.UI.iPhone.StatusBar' [undefined] is not an object. at app.js (line 1)."

I have also tried the following code, which I found from seraching the Q&A.

Titanium.UI.iPhone.statusBarStyle(Titanium.UI.iPhone.StatusBar.OPAQUE_BLACK);

Also Tried.

Titanium.UI.iPhone.setStatusBarStyle(Titanium.UI.iPhone.StatusBar.OPAQUE_BLACK);

And last I even tried setting &lt;statusbar-style&gt;opaque&lt;&#x2F;statusbar-style&gt; in the tiapp.xml file.

NOTHING WORKS! In need of assistance here. I'm lost. Why would it work in the Kitchen Sink app but not my app???

Thanks for your help!
