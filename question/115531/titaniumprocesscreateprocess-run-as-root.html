Titanium.Process.createProcess run as root

Particularly in OS X, is it possible to run commands with elevated privileges using the security framework?

I'm trying to run some commands which require root access to run. Using Platypus I can package a script and have it present me with the OS X authentication dialog. Is there a way to do this with Titanium?

Thanks.

  • For OS X :

    osascript -e 'do shell script "/bin/echo irc.freenode.net" with administrator priveleges'

    or run multiple

    osascript -e 'do shell script "/bin/echo irc.freenode.net & /bin/echo titanium-app" with administrator priveleges'

    It's late, but hope this helps you or anyone else who ran into this problem.
