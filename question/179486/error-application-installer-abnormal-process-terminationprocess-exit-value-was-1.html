  • Hello Amaresh,

    Thanks for your inquiry. There are some reasons for "Application Installer abnormal process termination. Process exit value was 1" this error to occur. Maybe emulator is getting timed out, or you need a clean build.

    Here are some steps you should follow.

    1. Restart your PC and try to run it.

    2. just clean your project and try to run it.

    3. Be careful of path follow this link after installing node if show node problem just restart your PC.

    if anything not work……

    You just uninstall node and titanium studio and delete the supported file .titanium, .android, .eclipse from C> User >your PC Name this location and titanium studio folder also this location C> User >your PC Name >appData. And after all re-install it and update the node and CLI completely, and set the android sdk path then restart PC and run it again. This Document can help you. If you are building alloy project then install Alloy 1.5.1 with Titanium. To do that Run 

    sudo npm install -g alloy@1.5.1
sudo npm install -g titanium@3.4.1

    Also it's faster to use Genymotion virtual devices (or emulators) to test Android applications. It is an alternative to the Android SDK's native Android Virtual Device manager. Genymotion emulators run a lot faster than the native Android emulators. Here is the link to configure Genymotion with Titanium.
