hello,
i have several pdf files that i have inside my resources folder. what i would like to do is open them up in using openURL.

i am currently using

_file = Ti.Filesystem.getFile(Ti.Filesystem.resourcesDirectory, 'file.pdf')
Ti.Platform.openURL(_file);

but this does nothing.

  • On iOS it is quite easy to open an PDF in a web view. I don't think you need to use getFile in your example though. Try this:

    Ti.Platform.openURL(Ti.Filesystem.resourcesDirectory + 'file.pdf');

    On Android you would need to use intents instead.

  • That would require you to use an URL Scheme. You can find out elsewhere, how to do that. Also you need to take into account that maybe the user doesn't have Acrobat Reader installed. You would end up with something like this:

    if(Ti.Platform.canOpenURL('adobereader://') {
    Ti.Platform.openURL('adobereader://' + Ti.Filesystem.resourcesDirectory + 'file.pdf');
} else {
    Ti.Platform.openURL(Ti.Filesystem.resourcesDirectory + 'file.pdf');
}

    I don't know if this will work, and you may have to tweak the code. Maybe there's no URL scheme for Acrobat and maybe it does not support parameters in the URL.
