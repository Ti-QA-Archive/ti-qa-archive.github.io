Adjust width of TextArea in orientation for IOS

How to adjust the width of TextArea in orientation for IOS.


    var commentsBackgroundView = Ti.UI.createView({
        top:'110',
            left:'0',
            height:'140',
           width:Ti.UI.FILL,
            backgroundColor:'#FFFFFF',
            borderColor:'#00000',
            borderWidth:'1',
            borderRadius:'1'
    });

    var txtArea_Comments = Ti.UI.createTextArea({
        top:'10',
        height:'120',
        left:15,
        width:Ti.Platform.displayCaps.platformWidth-30,
        borderColor:'#000000',
        borderRadius:'1',
        borderWidth:'1'
    });

    commentsBackgroundView.add(txtArea_Comments);
    self.add(commentsBackgroundView);

After orientation from portrait to landscape TextArea width size remain same. Its not incremented in landscape automatically. I did not written any gesture for orientation.
Please advice me for the auto adjust the size of the width for textArea. 

@All

  • Accepted Answer

    var txtArea_Comments = Ti.UI.createTextArea({
        top:'10',
        height:'120',
        left:15,
        right:15,
        borderColor:'#000000',
        borderRadius:'1',
        borderWidth:'1'
    });

  • It is easy: 

    width: Ti.UI.FILL,
left: '20dp',
right:'20dp'

    If you use you code with Ti.Platform.displayCaps.platformWidth- the calculation is right without rotation of device ;-)
