    There isn't a favourite button component. The ones you see on other apps are just normal buttons with appropriate graphics (i.e. a hart icon). You will need to implement the logic to make the app remember user's favourite items (database or local variables).

  • Hey E B

    There is no such component, module or any which works as favourite. What ever record you wants to set as favourite you can either update that record in the Database to be as favourite or you can save the id of that particular record in the Ti.App. Properties. After that once you again open the same screen first check either in the DB or in the Ti. App.Properties for the favourite id's and set the particular row with that id as favourite or you can even show the different list as favourite too.

    • What you are trying to do is call a CRUD operation. This blog entry may be of good use.

      The important part is to use an ID column for each record. This can be an automatic column. So once added each record will have an ID value and this should also be populated in the TableView as a custom property. So when a user clicks on a row you can get the ID value of that row and use it for your WHERE clause.
