    • There is no such thing as painting in mobile web. If there is a black space, then that means the window is not covering the entire screen (i.e you are seeing the bounds of the window). Modal windows do not cover the entire screen by design, much like they don't on iOS. Whatever is behind the window then becomes visible around the edges. You can override this behavior by setting left/top/right/bottom to 0 or by setting left/top to 0 and width/height to 100%. If you want to verify the behavior, you can use a desktop browser's debugger to inspect the window.
