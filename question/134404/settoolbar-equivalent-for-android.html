setToolbar equivalent for Android

Hi,
I have an app that works very well in iOS (iPhone and iPAD) etc.
My problem is that when I compile in Android, objects such as setToolBar etc, are totally unrecognized by this OS.

I am of course assuming that this setToolBar feature is one of many that is native only to iOS.

What is the equivalent of this for Android??

I'm finding porting from iOS to Android quite a challenge.

Thanks

  • Hello E B,

    Unfortunately, there is no Toolbar equivalent on Android, so you will have to implement your own. (That's what I did)

    There are many approaches you can take, but here's the one I followed:

    • Create a view 100% wide (that's you toolbar container)
    • Set it to layout: &#39;horizontal&#39; (that way, the button will stack from left to right)
    • Add the Buttons (or Views if you prefer)
    • Add the click eventListener on every button.

    If you use CommonJS, you can encapsulate all this behavior in a single module. And depending on the platform, you call one implementation or the other.
