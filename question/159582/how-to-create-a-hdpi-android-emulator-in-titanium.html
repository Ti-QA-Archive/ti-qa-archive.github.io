How to create a hdpi android emulator in titanium

Hi all,
Is it possible to create a hdpi android emulator in titanium?

In the run / debug configuration, we have different screen size to choose, but all of them runs in mdpi.

I'd like to see how the ui looks on a hdpi screen…

Thanks.

    Hello Zongque,

    Here is the reference like that you are looking for. http://developer.android.com/tools/devices/managing-avds.html

    First right click on your project and select your desired emulator and then run your project.

    Open AVD Manager from your android directory and edit your AVD to your desired resolution.

    You have to change your emulator's Abstract LCD Density to desired criteria.
