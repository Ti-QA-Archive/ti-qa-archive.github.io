Highlight Location on Floor Map - iOS and Android

I have a floor plan with about 200 rooms. I would like to be able to highlight specific locations on the map based on sections from the user.

It seems my options are these but I would some feed back or other suggestions… Any sample code would be awesome too.

Option 1.) My first thought was to create overlay images for each room. I would have the map and then another image the same size with a transparent background and a colored box the shape of the room that can be overlaid. This seems simple technically but would require a designer to create 200 extra images and would also increase the size of the app.

Option 2.) Get the x,y coordinates for each room and overlay a flag icon or something. I'm not sure how this would work with all the different screen sizes.

Option 3.) Use a webview with an imageMap, CSS, and JavaScript.

Thoughts?

Thanks,
