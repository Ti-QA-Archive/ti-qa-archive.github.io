  • Hello Frank,

    We have been using Titanium4j about a year now. And like you we are a bunch of enterprise Java Devs writing mobile apps with Titanium.

    Beeing able to leverage our Java knowledge was a big productivity plus.

    Here is my experience with Titanium4j

    Pros

    1) Java.
    Tooling around Java are simply awesome compare to pure JS(No offense to Titanium studio).
    Beeing able to catch errors at compile time rather than runtime is something you learn appreciate once you dont have it anymore.
    Before Titanium4j we developed using JavaScript and it s was a bit painful

    2) Softqare Engineering pattern
    We are able to use real software engineering pattern(Interface design, Design pattern, etc) in Titanium development.
    Something that s not easy to do with pure JavaScript

    3) Good support + Open Source
    The guyz behind the tool are very responsive and Titanium4j is opensource ansd Apache 2

    4) Same result as with JS
    We are able to get the same result as with JavaScript while leveraging a more robust language.
    Plus the GWT compiler generate a single optimized JS output saving us some extra build process.

    5) Easy to pick up
    Well this is bit subjective because we are Java Devs. JavaScripters will probably think the opposite.

    Cons

    1) Development process
    As for now one needs to compile to JS first before seeing the result. Something like an incremental compilation of the Java code would be awesome.

    2) No Debugging
    There is currently no way to debug the Java Code and debugging with the console only(or in JS) is not enough.

    3) Not much tutorials
    I wish the team would put more tutorials out there. Something like a Titanium4j version of the Kitchensink would be terrific

    So to me Titanium4j is probably to best add on to Titanium. I can only recommend you giving it a try.
    You wont regret

    I m very surprised Appcelerator has nt had a closer look at that tool. Can only be good for the platform

    Good luck and let us know how it goes.
