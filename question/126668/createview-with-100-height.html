createView with 100% height

I have a window composed of 2 views (header, body). The body view is composed of other components, but it seems that it has no height. I'd like to specify a backgroundImage, but couldn't get it to show up until I realized that my height: &#39;100%&#39; value wasn't being rendered the way I expected. If I set the value to 100, I get a body, but of course it doesn't stretch the way I expected.

How do I specify a view height that will render properly on multiple devices so that my background image will show up?

Thanks.
