iPad splitWindow masterview width

How is the width of the masterview determined? It looks like the width can't be changed, but is always set to 320dp. Where does this number come from?

  • The width is defined here:

    ~/Library/Application Support/Titanium/mobilesdk/osx/3.4.0.GA/iphone/Classes/MGSplitView

    #define MG_DEFAULT_SPLIT_POSITION        320.0    // default width of master view in UISplitViewController.

    I can post some code how to set the width using js, if s.o. is interested.
