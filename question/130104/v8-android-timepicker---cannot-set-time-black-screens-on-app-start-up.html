V8 Android TimePicker - Cannot set time. Black screens on app start up

Hi, I'm trying to set the value of a time picker on Android, using a line of code like this:

timePicker1.Value = new ( DateTime.Now.Year, DateTime.Now.Month, DateTime.Now.Day, 5, 30, 0 );

And it is ignored. The time picker simply shows the current time. It works fine in SDK 1.7.5 but 1.8.0.1 it seems to be broken. Also experiencing random black screens on starting up in 1.8.0.1 under Android.. are these known issues? They're complete show-stoppers for me to upgrade my app at this point.. :(

    It should be "value", not "Value" (property name is case sensitive).

    • Thanks Ivan, that got me a bit closer to the answer. It seems from experimentation that under 1.8.0.1 you cannot set the value of a timepicker after it has been created.
