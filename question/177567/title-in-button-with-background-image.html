Title in Button with Background Image

Hi Everyone

I'm having trouble trying to set up a button with a backgroundimage and a title, either the title shows or the backgroundimage but not both. Based on certain logic the title changes, but the backgroundimage must stay the same.

My TSS file has for this button the following code

"#txtOperacion" : {
top: '170 dp',
height: '35 dp',
textAlign: 'center',
left: '0 dp',
right: '0 dp',
title: 'Venta, Renta?',
font: {
color: '#111',
size: '20 dp'
},
backgroundImage:'textfield_60x640.png'

}

XML has the following
&lt;Button id=&quot;txtTipoPropiedad&quot; title=&quot;Tipo Propiedad&quot; &#x2F;&gt;

As you can see the tss file has title and backgroundimage but here i can only see the backgroundimage and not the title.

Any suggestions here? Thanks!

  • Hello, Xavier, Here is a sample button code with title and a background image.

    var win = Ti.UI.createWindow({backgroundColor: 'white'});
var button = Titanium.UI.createButton({
   title: 'Hello',
   top: 10,
   width: 100,
   height: 50,
   backgroundImage: "KS_nav_ui.png"

});


Titanium.UI.iPhone.SystemButtonStylePLAIN;

button.addEventListener('click',function(e)
{
   Titanium.API.info("You clicked the button");
});
win.add(button);
win.open();

    Use 

    Titanium.UI.iPhone.SystemButtonStylePLAIN;

    for assigning a custom image to backgroundImage.

    Thanks.
