    • If you want to create a native module, you can find the module development guides here: http://docs.appcelerator.com/titanium/2.1/index.html#!/guide/Extending_Titanium_Mobile

      To recreate the functionality using low level view components is basically just writing an app that builds out each of those views and recreates the functionality. There is no guide for that, all you are trying to do is use the components from the api to recreate the functionality that you want.
