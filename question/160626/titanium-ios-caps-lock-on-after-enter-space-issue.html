HI All,
I want to know how to caps lock on after enter space in "edittext" or "textfield". Right now my "edittext" is working like that "Titanium development" and i want result "Titanium Development".
My line of code is. var name = createEditTextView('Enter Name', 30, 40, 35, 250);

    The autocapitalization property accepts a constant (integer) not a boolean. The code you have used will work on iOS, though not necessarily provide the solution you want. It will most certainly crash on Android and perhaps other platforms. (I know, I've wasted time tracking down the cryptic error message that results when you do this on Android.)

    See http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.TextField-property-autocapitalization for the various constants that you can set for the autocapitalization property.
