Clear Array on another window

Is it possible to clear an array of tabledata that is set in another tab/window and rebuild that table.

I have a tabbed application with 2 tabs.

The first tab displaying a list of items that drill down to a sub window which shows the information and a "FAV" button.

var favButton = Titanium.UI.createButton({
    image : '../images/not-favourited-button.png',
    //backgroundColor:'#c20115',
    backgroundSelectedColor : '#000',
    height : 30,
    width : 36,
    right : 10,
    borderRadius : 5,
    borderWidth : 1,
});

And the second tab displaying all the "FAV"'s stored in a local DB and displayed in a table with each item getting it's own row.

var tabledata = [];

What i am looking to achieve is to be able to click the "FAV" button to toggle favouriting that specific item. I have that part set up and works to a point. This is where I am stuck.

If my database holds more than 1 item, when i click on the ´favButton´ button of an already fav'd item the script works fine, it removes the item from the DB so when i refresh that page it looks as it should with one less item.

Now if i have only one item in the database, when i click on the ´favButton´ button of that fav'd item, my script runs and removes the item from the db, but the data from the table doesn't clear. It still allows me to select the item. But if i query the DB it has 0 rows.

I have my favTable set to editable, which allows me to swipe to delete. This works perfectly. So it seems that i can not clear the array from a separate window.

What methods do you use for adding and removing favs to a local db?

  • You should move the code that loads the database and fills the table in the focus event of the window. 

    Titanium.UI.currentWindow.addEventListener('focus', TableLoadFunctionHere);

    That way you will be sure it is reloaded every single time the window is shown on the screen.
