How to implement unsubscribe_token function for Push notification?

Hello,

I am trying to implement unsubscribe_token function. However the function does not have a Titanium SDK method. Can anyone show me how to implement it in Titanium?
http://docs.appcelerator.com/cloud/latest/#!/api/PushNotifications-method-unsubscribe_token

The rest API code is
curl -X DELETE -F "channel=<channel>" -F "device_token=<token>" "https://api.cloud.appcelerator.com/v1/push_notification/unsubsribe_token.json?key=<key>"

but I am not sure what <key> is.

Thanks,
Wentao Ji

  • Hi Wentao,

    You can implement using curl where, key is your api-key. :

    curl -X DELETE -F "channel=<channel>" -F "device_token=<token>" "https://api.cloud.appcelerator.com/v1/push_notification/unsubsribe_token.json?key=<key>"
