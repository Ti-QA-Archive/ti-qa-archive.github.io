  • Hello Robert.

    To answer your -very valid- question of "how it works", first start thinking that all platforms are different; similar in many ways, but different. Titanium provides the developer with a high-level SDK, which has function calls that map directly to the actual calls in the mobile OS. This means that if iOS and Android have a native object called a button, Titanium will provide you with a way of "using" a button, but in the end, Titanium will tell the platform to use the native one. Naturally there will be functions that either exist only in one platform, or will display differently; but that's the very nature of cross-platform.

    Starting out is easy. When you download Titanium Studio, make sure you download the "Kitchen Sink" project. This sample app demonstrates most of the functionalities of the platform. Then, make sure you check out Alloy, the MVC Framework for Titanium, which will allow you to build your UI using XML and TSS (like CSS), and CommonJS modules for your app logic, all with Backbone.js and Underscore.js built-in.

    I have a few slide decks published at http://speakerdeck.com/ricardoalcocer

    I'm sure you'll love Titanium! Have fun!

    R
