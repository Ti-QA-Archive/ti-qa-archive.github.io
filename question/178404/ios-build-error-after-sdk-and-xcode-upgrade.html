iOS Build error after sdk and xcode upgrade

After upgrading the XCode, Titanium SDK and Studio (Alloy 1.5.1, Xcode 6.0, Ti SDK 3.4.0) whenever i try to build it fails giving this errors.
[ERROR] : ** BUILD FAILED ** [ERROR] : The following build commands failed: [ERROR] : Ld build&#x2F;Debug-iphonesimulator&#x2F;SCH.app&#x2F;SCH normal i386 [ERROR] : (1 failure)
I can build using Ti SDK 3.3.0. I have tried deleting the Resource and build folders too but it doesnt help.
