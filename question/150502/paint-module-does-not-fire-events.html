Paint module does not fire events

I would like to listen for events fired by a Paint view (Ti.Paint module) in order to know if something as been drawn or not. I tried adding standard listeners (click, touchstart, …) but nothing happens. Does not Paint fire anything?

target: iOS, SDK 3.0.2, Ti.Paint 1.3

  • Try this:

    $.paint.addEventListener('touchstart',function(e){
    event_Started = "yes";
});

function clear(){
    event_Started = "no";
    paint.clear();
}

    This code works fine.
