Android - Adding menu items to Camera window/view with onCreateOptionsMenu

I am currently trying to add menu items to a camera window/view but I cannot seem to add them successfully.
I want to add menu items to the menu that is displayed when the menu button is pressed.

With the other windows I can add them in the onCreateOptionsMenu event, like so:

var self = Ti.UI.createWindow();

var activity = self.activity;
activity.onCreateOptionsMenu = function(e) {
    var menu = e.menu;
    var menuItem = menu.add({
        title: 'My Menu Item',
        icon: 'mymenuItem.png',
        showAsAction: Ti.Android.SHOW_AS_ACTION_IF_ROOM
    });
    menuItem.addEventListener('click', function(e) {
        Ti.API.debug("Menu Item clicked");
    });
};

When I use Ti.Media.showCamera(), I cannot seem to get hold of the activity in order to set the onCreateOptionsMenu function.

I have tried creating a Window before calling showCamera, but I end up with a window that is not visible until the Back button is pressed to close the Camera window/view.

When trying to get the activity of the current window or attempting to get the current activity, I end up with either nothing or the menu gets added to the window I created prior to the showCamera call.

How can I add items to the menu whilst the Camera is being displayed?
