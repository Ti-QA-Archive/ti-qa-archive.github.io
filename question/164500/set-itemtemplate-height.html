Set ItemTemplate height

Hello!
I am using Titanium 3.2.2, android. I have this xml with custom listView:

<Alloy>
 <Window id="container">
   <ListView id="list" onItemclick="listItemClick"  defaultItemTemplate="template">
    <Templates>
     <ItemTemplate name="template">
      <View>
         <Label bindId="info" id="title" /> 
         <ImageView bindId="icon" id="icon" right="15" /> 
       </View>
        <View id="separator_line" /> 
       </ItemTemplate>
    </Templates>
  <ListSection id="section" /> 
  </ListView>
  </Window>
  </Alloy>

I need to set listItem height "15dp", could someone suggest how can I do that?

  • In the*.tss file associated with your view, you can reference the ItemTemplate element and assign a height property to it. Additionally, if you give the ItemTemplate an ID you can reference the ID in the titanium style sheet vs the element. This is handy when you are using multiple templates.
