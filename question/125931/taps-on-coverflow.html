Taps on Coverflow

How could I know if user tapped image1, image2, etc… and call a function sending the index of the image as the argument of the function.

var view = Titanium.UI.createWindow({
    title: 'Some Window'
});
var view = Titanium.UI.createCoverFlowView({
    images:['image1.png','image2.png','image3.png'],
    backgroundColor:'#000'
});
window.add(view);



function x('The Image Index'){
    alert('Do Something');
}

  • Just add a click event listener for the view and check the index property on the event object.

    view.addEventListener('click', function(e) {
    Ti.API.log('image index: ' + e.index);
});
