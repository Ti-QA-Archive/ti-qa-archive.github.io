Building on Genymotion (on another machine & using TiShadow)?

Hi guys,

Is it possible to deploy builds to Genymotion if it's installed on another machine?

i.e. I'm using a really old Mac and Genymotion runs super slowly.
Also I've got a second Linux machine that I have to use for PHP stuff and it's sat idle most of the time.
If I could send the build to Genymotion on the Linux machine it would be amazing.

Bonus points if I can do it with TiShadow.

If it helps I'm using the latest versions of Titanium Studio, the CLI, Genymotion etc.

Thanks!!!

    Ashley if you want to use Genymotion you will need to first install Google App.
    See this stack overflow question

    Next you will need to install the TiShadow app. If you are running genymotion on a different machine that will still work. You can install the apk from the ts builds site. Pick the build that matches the ts version you are using. Then from the app connect to the ip address of the machine that is running the ts server.

  • The answer is yes. Instead of appifying your project and installing to device/emulator,

    1. build the TiShadow standalone app.
    2. Deploy it to your Genymotion emulator.
    3. Start the TiShadow server.
    4. Run the stand alone app, point it via ip/port to the TiShadow Server.
    5. Now build your project toward the TiShadow server.

    Updates will be pushed to the server, which puhes them out to all attached client sessions.
