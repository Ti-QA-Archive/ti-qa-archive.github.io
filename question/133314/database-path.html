Database Path

i have developed my first app.. thanks Titanium !!… now, before send it, i have a few questions..

in what path i should save the Database ?
when i make an update to my App… the data will be erased ?

what i have to do so that dont happend?

In my code i have at the begginning of app.js

// ABRIMOS LA BD SIEMPRE AL INICIO Y
// CERRAMOS LA BD CADA VEZ QUE SALIMOS DE LA APLICACION
Ti.App.db = Ti.Database.open( "medicaldatacommx" );
Ti.App.addEventListener('close',function(e)
{
    if ( Ti.App.db ) {
        Ti.App.db.close();
    }
});

Is that correct ? or i must assign a path to the Database ?
