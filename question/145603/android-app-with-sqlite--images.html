    You should use the leftImage property if you want to display an image instead of the title.

    So your first row should read something more like:

    { leftImage:'../recipes/'+rows.fieldByName('image') }

    Now, if ou want the image to have another format, you will have to create a custom table view row yourself. It quite simple actually, just think of it as a regular view.

    var customRow = Ti.UI.createTableViewRow();
var img = Ti.UI.createImageView({ image: 'yourimage.png', left: ... });
customRow.add(img);
...
data.push(customRow);

    Hope this helps!

    PS. If you ever need to post code again, simply use the three tildes ~~~

    • Ah ok, you can't add an ImageView directly to a row.

      var sectionImage = Ti.UI.createTableViewSection({ headerTitle: '' });
var row = Ti.UI.createTableViewRow();
var imageView = Ti.UI.createImageView({image:'../recipes/'+rows.fieldByName('image') });
row.add(imageView);
sectionImage.add(row);

      That seems to work, I'll be back with more questions!
