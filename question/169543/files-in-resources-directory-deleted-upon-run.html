Files in resources directory deleted upon run

It doesn't seem to matter what type of file I add to my app's resources director, as soon as I choose Run to test the app in the simulator, all new files in the resources directory vanish. It's as if the Resources directory is being reverted back to its original form when I created the project, every time I run the app.

The files really do seem to vanish, too. I can't find them anywhere once they disappear from the directory.

Any idea how I can stop this from happening?

Mobile app - iOS 7.1
CLI version 3.2.1, Titanium SDK version 3.2.2.GA
Host system is OS X 10.9.2

Thanks.

    If, this is an alloy project, you shouldn't be touching anything in the Resources directory. Resource directory in alloy contains generated files that are then built into the native binary.

    If this is a classical project, well, your system is borked because this is abnormal behavior.
