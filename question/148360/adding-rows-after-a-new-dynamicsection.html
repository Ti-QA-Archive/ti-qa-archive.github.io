  • Hi

    Did you know that since Titanium SDK 3.0.0 there have been many new methods added to assist you to manipulate tables and their sections?

    • appendSection
    • deleteSection
    • getSectionCount
    • getSections
    • insertSectionAfter
    • insertSectionBefore
    • setSections
    • updateSection

    Add these to being able to set sections using setData with sections as Rainer pointed out and you can manage some complex needs very simply.
