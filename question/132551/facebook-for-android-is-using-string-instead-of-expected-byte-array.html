Facebook for Android is using string instead of expected byte array

It appears as though the Facebook API is not converting strings into byte arrays as expected by the Android FB SDK. I'm getting class cast exceptions with messages like "Key message expected byte[] but value was a java.lang.String." This makes the Titanium functionality useless for Android apps. Not sure if there is a work around for this.
