Search Bar for a non tableview window

Hi everyone!

I'm writing an app where i need a searchbar over a map o a view, where i can query google maps and get sugestions for places I'm looking.

All info I've read talks about search bar over tableview, and documentation tells that you can create a searchbar not only on tableview.

Currently I haven't been able to do this. My next option is creating a textfield on the window title, to behave like google maps app. where the searchbar on title turns to a tableview as soon as you touch it.

Any advice?

Thanks and have a great weekend

  • Ok, I found a way around to do this:

    var searchField = Titanium.UI.createTextField({
    color:'#000000',
    backgroundColor:"#ffffff",
    hintText : 'Search places',
    height:35,
    width:300,
    borderStyle:Titanium.UI.INPUT_BORDERSTYLE_ROUNDED
});

$.win.titleControl = searchField;

    • A searchbar doesn't require a tableview and is a much better choice than a textfield, e.g. you have also a "cancel" property.

      var search = Titanium.UI.createSearchBar({
    barColor:'#000', 
    showCancel:true,
    height:43,
    top:0,
});

      http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.SearchBar
