Searching photos problem

Hi everyone and thanks in advance….

I have to search the photos of a user (Clous Services). I´m using this example code:

Cloud.Photos.search({
user_id: savedUserId
}, function (e) {
if (e.success) {
alert('Success:\n' +
'Count: ' + e.photos.length);
for (var i = 0; i < e.photos.length; i++) {
var photo = e.photos[i];
alert('id: ' + photo.id + '\n' +
'name: ' + photo.name + '\n' +
'filename: ' + photo.filename + '\n' +
'updated_at: ' + photo.updated_at);
}
} else {
alert('Error:\n' +
((e.error && e.message) || JSON.stringify(e)));
}
});

The response include all the photos even for users different of the "savedUserId" variable.
Does anybody how to use search or query methods for photos objects.

Thanks a lot.

  • Hello Lopez,

    For Photo search you can use Cloud.Photos.query instead of Cloud.Photos.search. Although, both script are working good for me. Please follow my code for your solution: 

    
Cloud.Photos.query({
        page : 1,
        per_page : 20,
        where : {
            user_id : savedUserId
        }
    }, function(e) {
        if (e.success) {
            alert('Success:\n' + 'Count: ' + e.photos.length);
            for (var i = 0; i < e.photos.length; i++) {
                var photo = e.photos[i];
                alert('id: ' + photo.id + '\n' + 'name: ' + photo.name + '\n' + 'filename: ' + photo.filename + '\n' + 'updated_at: ' + photo.updated_at);
            }
        } else {
            alert('Error:\n' + ((e.error && e.message) || JSON.stringify(e)));
        }
    });

    And visit for more help http://docs.appcelerator.com/cloud/latest/#!/api/Photos-method-query
