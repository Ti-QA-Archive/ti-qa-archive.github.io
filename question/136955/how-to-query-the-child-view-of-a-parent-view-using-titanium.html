How to query the child view of a parent view using Titanium?

How do we acess the child items for a particular view in titanium by classname,id or name thing ??

Here is the code I am trying to implement :

var win = Ti.UI.createWindow({
    className : "myWindow",
    backgroundColor : '#fff'
});

win.open();

var label = Ti.UI.createLabel({
    className : "mylabel",
    text : "hello",
    top : 10,
    left : 20,
    color : '#000'
});
win.add(label);

for( i = 1; i < 10; i++) {
    var button = Ti.UI.createButton({
        className : "button",
        id : i,
        title : 'button' + i,
        top : i * 40,
        height : 30,
    });
    win.add(button);
    button.addEventListener('click', function() {
        alert("You clicked the button" + button.id);
    }); 
}

Ideally on clicking buttons it should return buttons id.

But clicking on any button it returns always 9.

What is the workaround for this. I don't want to do this via adding table rows and then finding out row index.

Also , I was wondering if there is something like Jquery in Titanium :

$('.myWindow .button').click(){
    alert ( "my id is " + $(this).id );
};


or 

$('.myWindow .button').addeventlistener('click',function(){
    alert ( "my id is " + $(this).id );
});
