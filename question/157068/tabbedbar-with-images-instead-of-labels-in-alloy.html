TabbedBar with images instead of labels in Alloy

Hi,

I want to create a TabbedBar with images (instead of labels) like this example from old KitchenSink app:
http://i.imgur.com/PZ6JYDQ.png

I had a look at the old way of achieving this on the kitchen sink code and this is what I found:

var buttonObjects = [
        {title:'Toggle Style', width:110, enabled:false},
        {image:'/images/slider_thumb.png', width:50},
        {title:'Toggle Enabled', width:140}
    ];
    var tb4 = Titanium.UI.iOS.createTabbedBar({
        labels:buttonObjects,
        backgroundColor:'#999',
        top:100,
        style:Titanium.UI.iPhone.SystemButtonStyle.BAR,
        height:40,
        index:1
    });

How do you do this in Alloy view xml markup?

  • Hi,
    This is for iOS. Check the documentation here - http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.iOS.TabbedBar

    <Alloy>
    <Window id="win">
        <TabbedBar id="bb1" platform="ios" backgroundColor="#369" top="50" height="25" width="200">

            <!-- The Labels tag sets the TabbedBar.labels property. -->
            <Labels>

                <!-- Specify text with node text or the title attribute. -->
                <!-- Can also specify the enabled, image and width attributes. -->

                <Label>One</Label>
                <Label>Two</Label>
                <Label>Three</Label>

            </Labels>

            <!-- Place additional views for the TabbedBar here. -->

        </TabbedBar>
    </Window>
</Alloy>

    You can replace the above Labels with ImageView or Buttons or what ever the controller

    <ImageView id='image' image='/images/myimage.png' />
