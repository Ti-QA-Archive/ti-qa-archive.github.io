Return count or quantity of cloud objects

I'm using Titanium 3.1, ACS, and Alloy MVC.

Say I have photo objects stored in ACS and I want to know how many I have, can I query for just the quantity?

I don't want to waste bandwidth and time returning all the photo objects just to count them locally on the users device.

I see for pagination this exists:

meta : {
    total_results:''
}

Could I use that to return how many objects exist?

If there's no way to get an exact count, is there a way to check if objects exist or don't exist?
