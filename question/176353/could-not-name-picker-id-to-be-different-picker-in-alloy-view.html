Could not name Picker ID to be different 'picker' in Alloy view

I want to add a Picker into my Alloy view (xml file) with id='datePicker' but it through an error Uncaught ReferenceError: datePicker is not defined. The generated Javascript code is:

$.__views.datePicker = Ti.UI.createPicker({
        ...
        useSpinner: true,
        calendarViewShown: false,
        id: datePicker,
        type: Ti.UI.PICKER_TYPE_DATE
    });

You see, the generated code of ID is not a string literal. I have no problem if I use id='picker' but it causes I could not add more than one Picker in an Alloy view.

I am using SDK 3.3.0.GA on Windows OS. Please advice.
