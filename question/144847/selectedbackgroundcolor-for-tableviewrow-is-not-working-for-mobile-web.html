The document says, it supports selectedBackgroundColor. I am not able to have this property in my application. Can anyone explain ?

  • BackgroundSelected color is rendered when the row is selected 

    var data=[];
var tblview= Ti.UI.createTableView({
      data: data
  });
  var tbrow1= Ti.UI.createTableViewRow({
      title:'i am row1 ',
      selectedBackgroundColor:'red'
  })

  var tbrow2= Ti.UI.createTableViewRow({
      title:'i am row2',
      selectedBackgroundColor:'pink'
  })
  data[0]=tbrow1;
  data[1]=tbrow2;

  tblview.setData(data);

    when you click any row then backgroundselected color is is visible
