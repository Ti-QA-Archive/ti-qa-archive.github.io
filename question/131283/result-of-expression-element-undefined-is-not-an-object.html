    This works for me:

    var win = Ti.UI.createWindow();

var statusField = Titanium.UI.createTextArea({
    value: "HELLO WORLD"
})
win.add(statusField)

win.open();

function foo(){
    var val = statusField.value
    return val.toLowerCase()
}

alert(foo());

    A couple things that may or may not be obvious.

    1. You must add the TextArea to a window and open that window.
    2. The value will be null if it isn't set beforehand and hasn't been typed in by the user.
