How to dynamically update collection?

I have a category model that has the following structure:

exports.definition = {
    config: {
        "columns": {
            "id": "TEXT PRIMARY KEY",
            "name": "TEXT",
            "parent": "TEXT"
        },
        "adapter": {
            "type": "sql",
            "collection_name": "categories",
            "idAttribute": "id"
        }
    }   
};

This data is presented using the the listview.

What I'm trying to do is to load child categories when a category (i.e. list item) is clicked. This means to load all categories where parent is equal to the id of clicked category. So I use the following code:

function categoryItemClicked(e) {
    var item = $.categoriesListView.sections[0].getItemAt(e.itemIndex);
    var _catId = item.properties.catid;

    var catsList = Alloy.createController('categories', {catid: _catId}).getView();
    Alloy.CFG.win1.openWindow(catsList);

    categories.fetch({
        statement: 'SELECT * FROM categories WHERE parent = ?',
        params: catid
    });
}

My problem is I want to clear the collection before fetching child categories, so that only child categories should appear in list, not parent and child categories together. I don't want to delete them from database, but just to remove them from listview. I tried categories.reset() before categories.fetch... but it doesn't do anything.
