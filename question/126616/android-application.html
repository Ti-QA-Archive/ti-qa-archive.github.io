  • I'm not completely sure I understand. Why do you need an algorithm to transfer a file? If you are connected via a socket or other service, then use that service to transfer the file. ie: if you connect via a tcp socket, then transfer your data via tcp, if you connect via ftp, then transfer via ftp etc.

    Is the second part a statement or a question? If you transfer a file from the phone to a server, the file format wont change because you transferred it. It will be the same on the phone as it is on the server. If you want to convert that file to another file type, then you will need to know both the file type of the original file and the file type that you want to convert to.
