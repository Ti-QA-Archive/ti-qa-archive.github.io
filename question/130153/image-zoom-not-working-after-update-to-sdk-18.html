Image zoom not working after update to SDK 1.8

Hello Appcelerator,

I have an image gallery which is a scrollView that contains an array of images. With the SDK 1.7.5, i can zoom in and out the images. After upgrading to 1.8. i can't zoom anymore. My scrollView looks like this:

photosView = Ti.UI.createScrollableView({
    backgroundColor:'#000',
    width : 320,
    height : 460,
    top : 0,
    showPagingControl : false,
    maxZoomScale : 2.0,
    currentPage : 0
});

Then i have a list of images:

for(z = 0; z<images.length; z++){            
    view = Ti.UI.createImageView({
        backgroundColor : '#000',
        image : images[z].url,
        defaultImage:'images/img_not_found.png',
        width : 320,
        height : 460,
        top : 0,
        photoId : z
    });
    viewArray[z] = view;
....

The images are added to the scrollView:

photosView.views = viewArray;

And that's all. Anyone can tell me why the zoom not working anymore?
