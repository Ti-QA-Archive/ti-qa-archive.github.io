iPhone 4, 3G and IP Addresses

Hi,

Just in case others have a similar issue to me …

I have recently been working with some RESTful web services that worked perfectly well with Wifi connections but but when switching to 3G on the iPhone.

I could not figure out why the same app would not work when using 3G.

It turned out that the iPhone resets its IP address for every request that is made.

Because the web services I was using used a hash of the IP address to create a session key that I then pass to all other requests everything stopped working because the IP address no longer matched.

You can go to any "what's my IP" site in Safari on the iPhone with 3G connection and refresh the window. The IP address will change.

Luckily the web services I am using are our own and so we could fix the issue which was added in an attempt to further protect sessions in the web browser.

Cheers
Mark
