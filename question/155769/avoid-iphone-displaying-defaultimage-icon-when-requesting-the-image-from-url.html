Avoid iPhone displaying defaultimage icon when requesting the image from url

Hi there,

I'm loading images from urls using imageViews (image property).

I've tried with the backgroundImage property, the backgroundColor, etc., but till the mobile downloads the image, the imageView will show the default iphone icon for images.

How can i avoid that and show a transparent background?

Thanks in advance.
