How to display a view in the center of another?

I have a view that has height Ti.UI.SIZE. It may be between 200 and 100px in height.

I want to display this view in the center (height) of a 300px heigh view.

How do I do it?

  • The most straightforward way to do this would be to set the center property of the view to the x and y values of the center of your parent view. In this case, that might be:

    { x: Ti.Platform.displayCaps.platformWidth / 2, y: 150 }

    You don't specify the layout in the x direction in your question, so in the above code I've assumed that both the parent and the child take up the full screen width.
