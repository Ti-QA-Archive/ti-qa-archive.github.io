  • Hi,

    the apple api does not accept any special control characters like * and # in the "tel:"-call. We also tried it - without success.

    Refer to Apple URL Scheme Reference:

    "Specifically, if a URL contains the * or # characters, the Phone application does not attempt to dial the corresponding phone number."
