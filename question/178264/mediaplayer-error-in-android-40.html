MediaPlayer error in Android 4.0+

Hello, I have an app that works on an Android device with sdk 2.3.5 on it (gingerbread). I am using Titanium sdk 3.3.0. The problem is that the streaming audio for our college's local radio station is breaking in newer versions of Android 4.0+. Here is the code I'm using to play the stream when the radio station window is opened:

player.stop();
player.setUrl(e.audio);
player.addEventListener('change', function (e){
    console.log(e.state);
});
player.play()

  • Well I got it working in genymotion on Android 4.1.1 finally.. I think there may have been a mismatch between the Titanium SDK in my build path and the SDK we were using in our manifest. I noticed earlier in the build path there were 2 SDKS selected no matter what and when I finally figured it out and matched the two it worked.
