    Hi

    You may be better off with a helper function to remove all the children. There a number of inconsistencies between the docs and reality - this might be one.

    function removeChildren(obj) {
    for (i in obj.children) {
        var child = obj.children[0];
        removeChildren(child);
        obj.remove(child);
        child = null;
    }
}

    Usage;

    removeChildren(nameofparentview);

    I use this successfully in an Android app.
