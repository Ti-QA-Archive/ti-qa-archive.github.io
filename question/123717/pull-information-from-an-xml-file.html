pull information from an xml file

HI ,
I have an xml file in this format

<vehilcle>
<two wheeler></two wheeler>
<four wheeler></four wheeler>
<Truck></Truck>
</vehicle>

I want to pull only the value contained in truck only.

When i use this

xml1.documentElement.getElementsByTagName("vehicle").item(2).text;

It returns me the value but i dont want to use positioning like item(2) because the xml file which i m taking from the server returns different tags based on the query so position of the truck tag may change.So if i use item(0) then in this case it may show me the wrong info..

So how do i take the value contained in the truck tag without using item(2)..

Thanks.
