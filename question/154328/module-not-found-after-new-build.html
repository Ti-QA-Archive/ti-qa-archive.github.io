  • Hi

    I am going to guess as there is not a lot of info here (chances are you have no more info to give).

    You are on Android, the error complains about a map. I am wondering if the problem is due to either;

    • You are still using the old version of Ti Maps which has limited support and only for apps that already have a version 1 Google Maps api key.
    • You are using the new Google Maps for Android Module but the module reference has not been set or has somehow been unset.

    Possible solutions;

    • If you are creating a new app or do NOT have an existing Google Maps for Android API key already issued before Google stopped handing them out - then you need to swap to the Google Maps for Android version 2 module
    • If you have an existing app AND an existing already issue version 1 api key, try cleaning the the project un-installing from the device and trying again.

    References for Google Maps version 2;

    • Google Maps version 2 - Modules.Map
    • Google Maps v2 for Android guide
