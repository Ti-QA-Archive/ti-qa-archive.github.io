App Restrictions?

Will an app that simply creates a portal to a custom made app version of a website on a web server be denied acceptance into the Apple App Store?

If the app loads this:

var window = Titanium.UI.createWindow();

var webview = Titanium.UI.createWebView({url:'login.html'});

window.add(webview);

window.open();

After the user logs in they view pages set up inside the app from a web server.

  • Accepted Answer

    Add some functionality or added-value not present in the web version (or with a different user experience). There are no warranties that your app will be rejected, but don't give them a good excuse to do it.
