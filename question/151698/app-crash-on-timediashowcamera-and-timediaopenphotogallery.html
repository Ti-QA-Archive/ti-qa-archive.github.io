App crash on TI.Media.showCamera and Ti.Media.openPhotoGallery

Hi,

I am working on the iPhone build for my app using Titanium SDK 3.0.0.

For images, the user can either click a picture or select from a gallery. The following code works fine for Android. but the app crashes in iOS(emulator as well as device).

Here is the code

function image_select(image_row, key) {
    var opt = Titanium.UI.createOptionDialog({
        options : ['Capture an Image', 'Select from Gallery', 'Cancel'],
        title : 'Select an Option'
    });
    form_win.add(opt);
    opt.show();
    opt.addEventListener('click', function(e) {
        opt.hide();
        changeImg = [];
        removeImg = [];
        if (e.index == 0) {
            //camera
            Titanium.Media.showCamera({
            });
        } else if (e.index == 1) {
            //gallery
            Titanium.Media.openPhotoGallery({
            });
        } 


    });
}

I have temporarily removed the callback options of the functions. The app is crashing in both cases(with and without the options) anyways.

Here is the error log

Could you please tell me what is going wrong?
