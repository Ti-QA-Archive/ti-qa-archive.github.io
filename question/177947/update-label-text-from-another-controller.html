Update label text from another controller

Hi,

I want to change the label text from another controller, but the view never changes…

Here is my code:

view1.xml

......
<Label id="pickerLabel" onClick="showPicker">The text</Label>
.....

view1.js

exports.updateDateLabel = function(value){

    Ti.API.info("Before: " + JSON.stringify($.pickerLabel));

    $.pickerLabel.text = value;

    Ti.API.info("After:" + JSON.stringify($.pickerLabel));
};

view2.js

function usePicker(e){
    var controller = Alloy.createController('view1');

    controller.updateDateLabel("new text!");
}

The output:

Before: {"text":"The text","left":"16","textAlign":"Titanium.UI.TEXT_ALIGNMENT_LEFT","id":"pickerLabel"}
After:{"text":"new text!","left":"16","textAlign":"Titanium.UI.TEXT_ALIGNMENT_LEFT","id":"pickerLabel"}

If I hit the button again, the output is the same:

Before: {"text":"The text","left":"16","textAlign":"Titanium.UI.TEXT_ALIGNMENT_LEFT","id":"pickerLabel"}
After:{"text":"new text!","left":"16","textAlign":"Titanium.UI.TEXT_ALIGNMENT_LEFT","id":"pickerLabel"}

What's wrong ?

Thanks!

  • in this code:

    function usePicker(e){
    var controller = Alloy.createController('view1');

    controller.updateDateLabel("new text!");

    }

    you are creating a brand new instance of view1, not reusing the same view1. Think of it like view1_2 or some such. Rather then creating a new controller, pass in the reference to the previous controller.
