configuration for the mail composer

Hi
createEmailDialog for activating mail composer, on mobile device if mail box is not configured then it will not shown the Emaildialog. If I configured on mobile device then it shows.

      var     emailDialog = Titanium.UI.createEmailDialog();
        emailDialog.subject = "Subject";
        emailDialog.toRecipients = "demo@demo.com";
        emailDialog.messageBody = "Hello World";
        emailDialog.open();

I want to show alert message if emailDialog is not configured on device "Configure email to send mail" so end user able to know about his email box is configured or not. How to know that emailD is not configured on device?

  • Hi

    You wrap the code you have with a request to the isSupported method.

    var emailDialog = Titanium.UI.createEmailDialog();
if (emailDialog.isSupported()) {
    emailDialog.subject = "Subject";
    emailDialog.toRecipients = "demo@demo.com";
    emailDialog.messageBody = "Hello World";
    emailDialog.open();
} else {
    alert('No email support');
}
