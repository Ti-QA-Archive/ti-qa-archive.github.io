App2sd android 1.5.1

Hi,

Im having a problem getting app2sd to work with the new version. I found this and it seems it should be possible to set this on a project level now.

But I'm having a hard time getting it to work, and does this change the target in default.properties as well?

My tiapp.xml contains this:


<android xmlns:android="http://schemas.android.com/apk/res/android">
        <tool-api-level>8</tool-api-level>
        <manifest>            
            <uses-sdk android:minSdkVersion="4" android:targetSdkVersion="8"/>    
        </manifest>
    </android>

What am I doing wrong? (I used to do this with the older versions and that worked)

  • I realize this is 10 months old, but it's the only post I've found that's relevant. Based on the original code posted I was able to get this to work with the current Titanium Studio just by adding this for the android section of the tiapp.xml file:

    <android xmlns:android="http://schemas.android.com/apk/res/android">
        <tool-api-level>8</tool-api-level>
        <manifest android:installLocation="auto">          
            <uses-sdk android:minSdkVersion="4" android:targetSdkVersion="8"/>    
        </manifest>
    </android>

    This worked as desired, I was able to move the app between the phone's local storage and the SD card, but the app crashes when I actually install it on to my phone - HTC Desire running 2.2
