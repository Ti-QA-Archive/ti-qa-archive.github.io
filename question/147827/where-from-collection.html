how to do where from collection

below code shows both the same length , when it should not 

var u = Alloy.createCollection("friends");
    u.fetch();

    Ti.API.info(u.length);

     u.where({
        block : false
    });

    Ti.API.info(u.length);

    http://backbonejs.org/#Collection-where

    the documentation

    Return an array of all the models in a collection that match the passed attributes.

    var new_u = u.where({
    block : false
});
Ti.API.info(new_u.length);
