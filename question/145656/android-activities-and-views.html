  • hi,

    i think you want to use default back button provided in emulator or in device so in that case you can override back button's click event by adding window's android:back event.

    so just do that

    
win.addEventListener('android:back',function(e){
    //with this event, window can not be closed automatically but here you can do all the stuff.
});
