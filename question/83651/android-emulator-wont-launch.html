  • Anthony

    Firstly, you seem to have installed your android sdk into C:\Temp. If this is the system temporary directory, then this probably is not a good idea. C:\android-sdk or C:\programs\android-sdk would be better locations.

    Can you run the android tool to see whether a virtual device has been created by Titanium? It should have a name beginning with "titanium_".

    See this screenshot:

    If one exists, select it and click on the start button to see if it boots successfully.

    Try rebooting after you have installed, just to ensure there isn't any windows funny business going on. ;)

    Report back and either I, or someone else, will help you further.
