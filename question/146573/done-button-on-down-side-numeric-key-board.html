  • Accepted Answer

    The key is not missing, it was never intended to be there in the original spec and is actually a placeholder for the decimal separator if required.

    Any changes to this are a hack and not supported by titanium.

    Follow the link Mitul gave you or the latest docs version - this shows you how to add a toolbar above the keypad where you can place a done button and (even next/previous) which is the accepted practise.
