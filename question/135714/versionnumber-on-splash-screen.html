Versionnumber on splash screen

Hi,
How can I add the version number of the app on the splashscreen? I don't want to update the images each time a new version is released.

  • Accepted Answer

    Yah it's possible. Follow below steps:

    • Create transparent splash screen named Default.png in iOS or default.png in Android.
    • Create window with your desired splash screen + a label with version number.
    • Close this window when finish loading

    • var win = Titanium.UI.createWindow();

var splash = Ti.UI.createImageView({
    image: 'path/to/your/splashsceen',
    width: '100%',
    height: '100%'
});
var versionLabel = Ti.UI.createLabel({
    text: 'Version 1.1',
    width: '200',
    height: '40',
    zIndex: 1,
    bottom: 10
});

win.add(splash);
win.add(versionLabel);

win.open();
