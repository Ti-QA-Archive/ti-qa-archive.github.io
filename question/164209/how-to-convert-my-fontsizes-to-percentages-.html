How to convert my fontSizes to percentages %

in order to make my app having a responsive design i would like to convert my fontSizes value to percentages (%) value because i think this will be better to suit all screen's dimensions. if this is wrong and that the pixels values are ok for every screen i need this being confirmed.

I've tested a % value depending on the parent dimension (eg. to have a 10px fontSize under a 200px heigthed parent i will write 5% but this didn't work)

    It won't work. The font size needs to be a specific defined size.

    If you set your height to say 200, you could set your fontsize to be calculated

    i.e. fontSize : (200 * 0.5);

    that would give you 5%.

    Generally though you would set a specific size for phones and tablets.

Always try to allow the system to do the complex calculations for you first, and if you arent happy with the results, resort to complex algorythsm, because, after all, the average dev is so much smarter than a system engineer.

    fontSize: "16sp"
