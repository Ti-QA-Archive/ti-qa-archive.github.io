    • @kevin : you're the man :)

      so:

      I added as a service with the following code in app.js

      var intent = Titanium.Android.createServiceIntent({
        url: 'geo.js'
    });

var service = Titanium.Android.createService(intent);
service.start();

      the geo.js file resides in Resources/android folder

      and I had to add in tiapp.xml the next lines:

          <android xmlns:android="http://schemas.android.com/apk/res/android">
        <services>
            <service url="geo.js" />
        </services>
    </android>

      The geo.js file communicates with the app using events.
      One small question though: the example in KS has the service in tiapp.xml declared type="interval". Where do I find this types?
