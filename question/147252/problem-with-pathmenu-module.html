    Hi

    I "think" I know what the problem is. I prefix this with I have just gone through your process - purposely did not do one step (the one I think you need) and got the exact same error you have.

    Good news - do the missing step and it works.

    Seems like I have been holding back the answer for some weird reason - so here it is;

    You need to add the module into the modules section of the tiapp.xml file. There are two ways to do this;

    • Hard way; Edit the file tiapp.xml in a text editor and add the reference manually
    • Easy way;
      • double click the file tiapp.xml from the project explorer
      • locate the "modules" section on the top right of the "Overview"
      • press the green plus (add) button
      • choose the "path.animator" entry

    Either way close the simulator/emulator then from the menu "porject" > "clean" (choose your project)

    Then run again and you are there - path menu greatness.
