How to Print "Hello World" in python plugin?

Hi. I understand in the creation of plug-in python.

  1. Create titanium project "Hello".

  2. Create a file in \Hello\plugins\ti.hello\plugin.py
    ~~~
    #!/usr/bin/env python

    -- coding: utf-8 --

    #

import os, re

print "[INFO] —————- Hello ——————"


3. Added to in tiapp.xml:

<plugins>
<plugin version="1.0">ti.hello</plugin>
</plugins>
~~~

  1. Run Project. Everything compiles, but no line "—————- Hello ——————" console

What is wrong?

p.s. At the top of the window App explorer is displayed "Puthon not configured".

  • Titanium SDK version 3.0.2.GA.
  • Mac os 10.8
  • Ios simulator

  • This is not an appcelerator specific problem, and the solution could have been found with about a minute of GOOGLING.

    Your environment is fouled up (or python really isn't installed, but unlikely on a mac)

    Change the first line to #!&#x2F;usr&#x2F;bin&#x2F;python (or point to the python executable you want to use)
