Alloy focus event not firing

I have the following view and controller, but the focus event is not firing on the iOS simulator or device.
This is with Alloy 1.3.0, SDK 3.2.0.GA on Mac 10.9.1, Studio 3.2.1, simulator running iOS 7.0.3

Is this the correct way to run this event?

library.xml

<Alloy>
    <View id="library" layout="vertical" onFocus="refreshView" onOpen="refreshView">
        <View id="toolbar" layout="vertical" height="50" backgroundColor="blue">
            <Label id="label" top="30">toolbar</Label>
        </View>
        <TableView id="table" onClick="rowClicked" onFocus="refreshView" onOpen="refreshView" />
    </View>
</Alloy>

library.js

var refreshView = function() {
    Ti.API.info('library refresh view');
};

refreshView();

$.table.addEventListener('focus', function() {
    Ti.API.info('table event listener refresh view');
    refreshView();
});

    Two things;

    1. A View object does not have an open event.
    2. The focus event on a View only applies to Android
