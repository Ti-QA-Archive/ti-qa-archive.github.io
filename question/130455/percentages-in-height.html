Percentages in Height

Why doesn't this work:

       var placewin = Titanium.UI.createWindow({
            id: 'placewin',
            title: 'Place '+mid,
            backgroundColor: '#FFF',
            barColor: '#414444',
            layout:'vertical',
            fullscreen: false
            });

        var infoView = Ti.UI.createView({
            top:0,
            height:'30%'
        });

        var menuView = Ti.UI.createView({
            top:0,
            height:'70%',
            backgroundColor:'blue'
        })
        placewin.add(infoView);
        placewin.add(menuView);

I'm trying to break the screen up into sections by % not hardcoding the value. If I switch to static integers the blue shows up, with the percentages it doesn't.
