ImageView - Change current index

Hello,
I've created an ImageView that is running an animation.
I need to "change" at runtime the current image showed by the animation.
I'm using the following code:


var images = [];
for (var i=0; i<10; i++) {
    images[i] = "/image_"+i+".jpg";
}

var loaderImage = Ti.UI.createImageView({
  width:  'auto',
  height: 500,
  images: images,
  duration: 400,
});

$.index.add(loaderImage);

$.index.open();

loaderImage.start();

loaderImage.addEventListener('click',function(e) {
    loaderImage.stop();
    loaderImage.image = images[3]; // jump to image index '3'
});

The problem is that the imageView blinks… like showing the background for a second.

Any sudgestion?

  • The fact you haven't provided information on the OS, I'm going to guess that you are using iOS?

    If you are then try removing loaderImage.stop();, this resets the index to 0 on iOS which may be causing the blinking you are experiencing.

    Also have a look at Titanium.UI.ImageView-property-animating as the animating property defaults to false on iOS.
