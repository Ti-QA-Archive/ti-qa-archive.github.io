How to internationalize textfields hints

Hi, i am making internationalization of project and replace label text and button title with textid and titleid, but how to set hintText in alloy xml views ? I try with hintTextid and hintTextId but it does not work.
I know i can set it in controller file like:

$.textFieldId.hintText = L("hint_text_id");

but this needs additional code wich I want to avoid. I want to set string id directly in alloy xml file.

  • Actually you can do it in the TSS file directly.

    Creating a class with hinttext and using the internationalisation.

    "#homeLinkLabel": {
    height: Ti.UI.FILL,
    width: Ti.UI.FILL,
    font: {
        fontSize: 14,
        fontFamily: Alloy.CFG.fontTextBold
    },
    color: Alloy.CFG.fontColorLight,
    textAlign: "center",
    touchEnabled: false,
    text: L("return_home_link_label")
}

  • You can use the textid for labels or titleid for buttons in the XML. But the TextField and TextArea objects do not have internationalized properties for value or hintText. You'll have to do that via the TSS like Trevor suggests or in the controller as you stated. This is not a limitation of Alloy – as I stated, the TextField object has no "hintTextId" property, so there isn't anything for us to expose in Alloy.
