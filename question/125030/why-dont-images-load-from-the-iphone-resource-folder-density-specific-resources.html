  • Don't put the images in the iphone folder. This folder won't be accessible after the app is built.

    Put the images in a folder inside Resources.

    something like Resources/images/ and then set the path like this:

    var logoImage = Ti.UI.createImageView({
    width   : 172,
    height  : 116,
    top     : 30,
    image   : '/images/logo.png'
});

    / is the Resources folder.
