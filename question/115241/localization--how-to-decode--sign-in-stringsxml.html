Localization // How to decode &-sign in strings.xml

I have a strings.xml file with the following content:

<string name="kategorien_data_Villen_title">Villen & Fincas</string>

When trying to run the app in the iPhone Simulator i get the following errors: Log output. If I remove the & in the string everything works as expected.

So how can i mask special chars like & % and others in strings.xml?

My setup is: Titanium Developer 1.2.2, Mobile SDK 1.6.0 (18.02.11) and iOS 4.2.

    Hi Stefan,

    you need to replace XML entities, so & needs to be &amp;
    This will work:

    <string name="kategorien_data_Villen_title">Villen &amp; Fincas</string>

    Gruss
    Thomas
