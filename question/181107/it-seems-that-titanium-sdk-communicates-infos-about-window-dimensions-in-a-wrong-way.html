It seems that Titanium SDK communicates infos about Window dimensions in a wrong way!

I made other tests and it seems that Ti SDK 3.5.0 and 3.4.1 communicates dimensions like widht and height of a Window in a wrong way: if you try to set width and height to the actual screen size using Ti.Platform…. you can see that these measures are treated as double. Try to put a simple colored view 50x50px in the window (both Alloy and Classic mode are affected) without modifing "layout" attribute of the window and without setting size for the window or putting its half : you'll see the view in the center (correctly). Try to set width and height to real size and you'll se that the view is not at the center but below at the right: the window acts as if its size were doubled. Is there a solution to this?
