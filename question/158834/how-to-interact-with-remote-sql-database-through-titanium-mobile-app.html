  • Hi

    You cannot connect directly to remote databases using drivers like ODBC.

    Mobile apps interact with remote databases using APIs that you create on your web server.

    These are usually created using PHP, ASP.net, ASP or other server scripting language. These scripts interact with the data and stored procedures, you manage the parsing and cleansing of data in and out here.
