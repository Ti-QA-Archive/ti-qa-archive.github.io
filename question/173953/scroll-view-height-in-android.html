  • Hi Wahib,

    you have to set the 'contentHeight' property of your scrollView

    xml (add an id to the scrollView)

    <ScrollView id="scrollView1" showVerticalScrollIndicator="true" contentHeight="auto" scrollType="vertical">
...

    tss

    #scrollView1 : {
    contentHeight:Ti.UI.SIZE
}
