  • Hi Siddharth,

    Ti.UI.View dont't have a load event, you can check the docs here.

    But you can add and fire your own events.

    win = Ti.UI.createWindow({
    backgroundColor : 'white'
})

view = Ti.UI.createView({
    title : "my view",
    backgroundColor : 'blue',
    width : 100,
    height : 100
});
view.addEventListener("myevent", function(e) {
    alert("my event");
})

win.add(view)

win.open();

view.fireEvent("myevent");

    Hope that helps,

    Matthias

    • Hi Siddharth,

      if you use the postlayout event and want it to be fired only once, you have to remove it also.

      function getAlbum() {
    scrollView.removeEventListener('postlayout', getAlbum);

    Ti.API.info('getAlbum');
    // more code
    // ....
};

scrollView.addEventListener('postlayout', getAlbum);
