Black splash screen appears instead of my own on Android

Hi all,

The issue is when I run my android application I see the window with title above but instead of my splash screen I see black background. When I change device orientation splash screen appears. It looks like it is not drawing properly at first time. Same app works fine on iOS. SDK 1.7.0

Does anyone know a workaround? Any help will be appreciated.

  • I've found a workaround. I looked on this line in my app.js:

    Titanium.UI.setBackgroundColor('#FFF')

    When I changed FFF to 000 I got white screen so I deleted those line and all became to work fine.
