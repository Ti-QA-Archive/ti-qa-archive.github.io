  • There are couple of ways. You can select based on what is perfect match for your application. Below are few options that you can try,

    1. You can store it in database and fetch it from DB each time when you require.
    2. You can store it in Global variable. Alloy.Globals.token = &#39;12AB3ddF&#39;;
    3. You can put in app properties like Ti.App.setString(&#39;token&#39;, &#39;12AB3ddF&#39;); This is available across app session also.
