    Hello,

    that's a different bug. Please go to run configurations (below your button of run on the emulator) and select run configurations. Check the android 2.2 with google api. The reason is not working is that you are using android 2.2 without the google stuff needed for the map.

    Just a quick test: Go and create a new mobile project, and try to run it. It will run, since it's only a window with tabs.
