  • Ok thanks for replay.
    i use this function :

    ...
this.addSlashes = function(str) {
    return (str + '').replace(/[\\"']/g, '\\$&').replace(/\u0000/g, '\\0');
}

this.stripSlashes = function(str) {
    return (str + '').replace(/\\(.?)/g, function(s, n1) {
        switch (n1) {
            case '\\':
                return '\\';
            case '0':
                return '\u0000';
            case '':
                return '';
            default:
                return n1;
        }
    });
}

  • First, you need to have in mind that you need "escape" function for Sqlite database, not MySql database.

    So, you should check in Sqlite documentation what you need to escape.

    Here is function that I wrote for that purpose:

    function escape (value) {
    if (value && typeof value.replace == "function") {
        return value.replace(/'/g, "''");
    }
    else {
        return value;

    }
};
