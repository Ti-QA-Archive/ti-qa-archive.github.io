Two title bars when using navigation window inside tab.

I have created a navigation window inside a tab but it is still showing two title bars even if I set titleBarHidden equal to true for all windows involved with the nav window.

My code for the windows:

var self = Titanium.UI.createWindow({
    titleBarHidden: true
});

var nav = Titanium.UI.iOS.createNavigationWindow({
   window: self,
   title: 'Upload Test',
});

self.barColor="#005100";

nav.open();

I have played around with a lot of settings but can't seem to hide the top title bar, I added the text to highlight where it is being generated:

var uploadTab = Ti.UI.createTab({
    icon:'plantcombos_nav_upload.png',
    title:'Upload',
    window:uploadWin,
    titleBarHidden: true    
});

I am running iOS 7.1 and 3.2.2.GA, any help would be greatly appreciated.

Thanks,
James

    • when you creating your window and tabs in tabs.js add the tab as a reference to the window like this.

      var win = Ti.UI.createWindow();
var tab = Ti.UI.createTab({window:win});
win.containingTab = tab;

      then you can access the tab from your window.

      uploadWin.containgTab.open(photoWin);
