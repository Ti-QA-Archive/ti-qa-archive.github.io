Session Handling in titanium

Hi Everyone,

             i hve login and signup functionality in my app. After login i want user to interact with server data . And after logout i want to refresh my app and only show local data available. Is there any way to maintain session of users?

  • Hi Gagan,
    You can do this by giving user a token in login response and save it in the app and pass this token in every request you make to return the data to the user, validating this token from your service db. So when user logs out the you just simply flush the token from the database. So when next time user request a request yoy ask him to login again.
