Android Animation Crashes with Border

  • App Type: mobile
  • SDK: 3.1.3.GA
  • Platform: OSX v.10.8.5
  • Device: Android Simulator
  • Studio Build: 3.1.3.201309132423

If a view has a borderRadius set or a borderWidth set, it crashes the entire app once animated. Even if those items are set to 0, it still crashes. Removing those attributes completely causes app to work flawlessly again.

I found a bug fix here: https://jira.appcelerator.org/browse/TIMOB-13671?page=com.atlassian.jira.plugin.system.issuetabpanels:comment-tabpanel

However, when I check for updates, no updates are found. Therefore I can't get the "fixed" version so I can finally launch my new app. How do I get the fixed version?

Thanks.
