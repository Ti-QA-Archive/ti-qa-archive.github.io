Licensing and distributing Titanium based desktop apps

Hi,

I have used Titanium to create an IOS and Android version of my app which currently sell in the app store and the android market. I want to be able to create a desktop version of my app for both windows and OSX, and sell them in a similar type of store. I assume I can use the app store for the OSX version, but how can I license and distribute a desktop app for windows? Thoughts?

Also, how can I prevent copying of the app onto other desktops?

Thank you.
Martin.

  • For Windows you're going to have to set up the purchase and hosting yourself, ie: PayPal Download or the similar.
