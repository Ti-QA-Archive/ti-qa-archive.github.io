How to query ACS selecting fields?

Right now an ACS query looks like a select *.
There is a way to do a select field1, field2 alike query so I can get only the fields I want at a give moment?

  • You can achieve this using the Cloud.Objects.query function.

    It's quite different than what you might be used to though since it doesn't rely on SQL-like syntax.. (I prefer to give you the heads up). The basic principle is that you specify:

    • What class object you want to retrieve
    • A list of attributes matching certain criteria

    And ACS will do the heavy lifting for you. This is a good thing since it frees you from the burden of optimizing the query yourself.
