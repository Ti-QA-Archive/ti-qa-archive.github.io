Applications will crash when using webviews on Android 2.2.1 and 2.3

I'm using Titanium SDK 1.4.2, and have built an app using webviews (so that it will be simple to port from Android to iPhone). Everything works great up to Android version 2.2.

However, versions above that, 2.2.1 and 2.3, will crash. Not always, but only when you make calls to the Java layer from Javascript. Running simple code like this will crash the app:

Ti.API.info('>>> Details. DocumentReady');

I read somewhere that you should manually include binding.js, but that doesn't make any difference.

I'm not really sure if this is fixable, since I believe it was changed because of security reasons, but it should cause a lot of problems for a lot of programmers.

Does anybody know a fix, besides going native?
