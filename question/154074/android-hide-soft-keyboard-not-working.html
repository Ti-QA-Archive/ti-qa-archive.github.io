Android Hide Soft Keyboard not working

In the following block of code:

        var appPassword = 'grantaccess';

        var passwordTextField = Titanium.UI.createTextField({
            height: 45,
            width: 75,
            clearOnEdit: true,
            passwordMask: true,
            softKeyboardOnFocus: Titanium.UI.Android.SOFT_KEYBOARD_SHOW_ON_FOCUS,
            keyboardType:Titanium.UI.KEYBOARD_ASCII
        });

        var alertDialog = Titanium.UI.createAlertDialog({
            title: 'Login Password',
            androidView: passwordTextField,
            message: 'Password is to required to grant access',
            buttonNames: ['OK']
        });

        alertDialog.show();

        alertDialog.addEventListener('click', function(e){
            if(passwordTextField.value != appPassword) {
                alertDialog.message = 'Incorrect password! Please try again.';
                alertDialog.show();
            }
            else Ti.UI.Android.hideSoftKeyboard();
        });

The Ti.UI.Android.hideSoftKeyboard() does not work. The keyboard is still showing after the code exectues. However, when I step through the code in debug mode it DOES work! The keyboard does go away.

Any idea how to resolve this issue? Should I just file a bug in JIRA?
