Which version of Node.js works?

I've installed the lates version of the CLI, 3.0.24, and am trying to build a hello world project from the command line in Windows 7. I've followed the stepes outlined here: http://docs.appcelerator.com/titanium/3.0/#!/guide/Setting_up_the_Titanium_CLI.

The problem is that when I try to run the build command - titanium build -p android –log-level-trace - I'm getting an error that reads: "CreateProcessW: The system cannot find the file specified."

All of the explanations of this error that I've found point to some incompatability between the CLI and some version of node js. So far I've tried: 0.8.3, 0.8.9, 0.9.9, and 0.10.4 but I still get the same error. Can someone point me to a version of node that is known to be compatable with 3.0.24, or an alternate explanation for this error?

Thanks in advance for any help.

  • Node.js 0.8.22 is the official supported version. Titanium SDK 3.0.0.GA and 3.0.2.GA will ONLY work with Node.js 0.8.x and they will NOT work with Node.js 0.10.x or newer. This compatibility issue was fixed in Titanium SDK 3.1.0.GA.

    I highly recommend you updating your Titanium CLI from 3.0.24 to 3.1.0. There were a handful of bugs that were fixed. To update, run:

    sudo npm install -g titanium
