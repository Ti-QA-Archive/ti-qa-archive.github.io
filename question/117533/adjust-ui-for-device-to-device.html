    Soon you will be able to use percentage dimensions, implemented in ticket #1176. However, there are still outstanding issues with it, as this list shows.

    Until these are resolved, you must implement a function that will calculate the correct widths based on the Ti.Platform.displayCaps.platformWidth and Ti.Platform.displayCaps.platformHeight values, unfortunately.

    Hope this helps
