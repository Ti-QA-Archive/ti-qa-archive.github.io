How can I scroll a scrollview (to the right or left) as long as the user touches a button?

Usage scenario: Have 2 arrows on each end of the scrollview (one pointing to the left, one to the right). If the user touches / holds the button, the scrollview should scroll in that direction as long as the user touches the button.

  • Thought this would be nice for my app so here is a rough version (scrolls every 750 ms):

    I tried testing it but for some reason the touchstart and touchend aren't firing on a button I have, didn't spend too much time looking into it though.

    var scrollingRight = true;
function scrollRight()
{
    if (scrollingRight)
    {    scrollable.scrollToView(scrollable.currentPage+1);
        setTimeout(function(){  scrollRight();  },750);
    }
}
rightButton.addEventListener('touchstart', function()
{    
    scrollingRight = true;
    setTimeout(function(){  scrollRight();  },500);
});
rightButton.addEventListener('touchend', function()
{
    scrollingRight = false;
});
