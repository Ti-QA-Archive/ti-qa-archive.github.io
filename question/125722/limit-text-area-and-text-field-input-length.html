Limit Text Area and Text Field input length?

Hi all,

I was wondering if its possible at all to limit how many characters can be entered into an input field. So for example, you could limit a username to say 50 characters in length, and it wont allow you to enter more characters.

I know it can be done using native/direct Objective-C however wasn't sure if the functionality existed in Titanium yet. I did look through the API docs, and searched the Q&A however wasn't able to find anything of use for this.

Cheers

    Use the change event of the textarea/textfield to read the value and if it passes the 50 chars reset it to the first 50 chars.

    the_textarea.addEventListener('change', function(e){
    e.source.value = e.source.value.slice(0,50);
});
