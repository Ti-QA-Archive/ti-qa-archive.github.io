  • Don't think this is possible in Titanium.

    Another solution is to get the touch x/y position from imageView, then send the position to a webView and use <canvas> to get the pixel value.

    • After getting the touch from the imageView, you can call imageView.toBlob() and then save it to the filesystem. Then create the webView, send the path to the saved image, do something with the image in the webView. Then pass the result back from the webView, and show the result.
