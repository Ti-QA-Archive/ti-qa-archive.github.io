  • Hi,
    You can query all users through below sample code from ACS :

    Cloud.Users.query({
    page: 1,
    per_page: 10,
    where: {}
}, function (e) {
    if (e.success) {
        alert('Success:\n' +
            'Count: ' + e.users.length);
        for (var i = 0; i < e.users.length; i++) {
//Now here you can get the image of users

         }
    } else {
        alert('Error:\n' +
            ((e.error && e.message) || JSON.stringify(e)));
    }
});

    Now under success callback you can get the image of users and can put it in a list view with the null check on each if user has set the profile picture or not, otherwise you can set your default one.

    Hope it helps
