  • Very simple, but I'm not going to write the app for you. Create a ListView and a SearchBar. Add them to the window. Listen to the SearchBar "change" event and there create some logic to look for results and populate the ListView accordingly. With the "change" event you can get the text that was entered. It's that easy.
