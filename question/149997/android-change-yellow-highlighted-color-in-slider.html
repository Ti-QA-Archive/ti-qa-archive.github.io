    Hi

    This is an example for you and left and right track image work good may be you are not giving the image path correctly

    var slider = Ti.UI.createSlider({
    left : '11%',
    right : '12%',
    min : 0,
    max : 199,
    top : '45%',
    value : 98,
    width : '76%',
    leftTrackImage : 'images/sliderprogress.png',
    rightTrackImage : 'images/sliderbackground.png',
    //thumbImage : '../images/slider_thumb.png'
});

    Thanks
