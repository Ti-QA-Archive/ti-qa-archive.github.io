  • Hi Deepan,

    I am using a lazy loading/ infinite scrolling method. I display 20 rows at first in a table View and as the user scrolls to the bottom, I load another 20. This continues till I reach the end of the data to display in the database.

    In my opinion displaying 100 rows all together will slow down your app and might even crash for Android if you also display images in the Table View row.
