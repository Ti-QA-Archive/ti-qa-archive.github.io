HTTPClient - Send file on Android problem

Hi,

I'm trying to send a file to my server using HTTPClient. Here's my code:

var file = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory + '\log.txt');
var blob = file.read();


var httpClient = Ti.Network.createHTTPClient();
httpClient.open("POST", 'http://mywebserver.com/uploader.ashx');
httpClient.send({ 'logFile': blob });

on iPhone it works fine and I get the file on the server side.

on Android the client throws an error: "Uncaught Error: org.apache.http.entity.mime.content.FileBody".

Is there a way to send files on android?

Thanks
