  • Accepted Answer

    In the Ti.Filesystem.File class, there is a method to getDirectoryListing

    > A file object can point to an ordinary file, a directory or a symbolic link. Use createDirectory to create a directory. Use the getDirectoryListing method to retrieve a list of the directory's contents.

    Then loop through the files, and include each file with Ti.include

  • Hi

    Please elaborate on "…include an entire directory of js files…"

    • If you mean you have created lots of JavaScript files and you want them to compile within your app

    That happens automatically.

    • If you mean "include" as in Ti.include(&#39;some_module.js&#39;);

    Then no; one at a time.

    If this is the case and you have so many JS files that you Ti.include(), you might want to refer to the CommonJS model of working. Do not go re-writing things now but take a look and use it and Alloy to help you manage larger projects and work-flows.

    Note: Alloy will be 'baked' in to 3.0
