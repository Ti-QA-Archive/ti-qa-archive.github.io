Full IOS builds everytime since upgrade to 4.1.0

IOS builds used to be very quick once they were initially compiled. Ever since I upgraded to 4.1.0 (and node.js 4.1.2) every time I run it invokes a full xcode build. This means every small change takes about a minute to 2 minutes to launch. Before, these small changes would relaunch the simulator in 10 seconds. Was there a default setting changed?

  • Have you checked you preference/settings of Appc studio?

    Go to Appc studio -> Run -> Run Configurations… -> Select your run configuration and see whether the checkobx inside Build section is checked or not. If it is checked, unchecked it and click Apply.
