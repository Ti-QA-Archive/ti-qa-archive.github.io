  • Since we are on a ball you can't use simple flat trigonometrics. There's a function called the Haversine formula. Please refer to the wikipedia article on that @
    Wikipedia://Haversine Formula

    It looks complicated but boils down to:

    @earthRadius * acos(cos(@hereLat) * cos(@destLat) * cos(@destLon - @hereLon ) + sin(@hereLat) * sin(@destLat ))

    This is actualy a SQL snippet from the query I use to get data from a mySQL db.

    The result is in the unit you provide earth radius in and longitude and latitude need to be in radians.

    Hope this helps
