Place a view on top of the navigation window

I have a navigation window set up in my application. The first screen you come to is a login screen. When your credentials are being authenticated, I want to display a loading window. I have this working except the loading window is beneath my navbar. Is there a way to place this view on top of the navbar? Possibly a zIndex workaround?

My window object

mainWin = Ti.UI.createWindow({
    backgroundColor:'#ededed',
    width:'auto',
    height:'auto',
    title:'Welcome',
    leftNavButton: Ti.UI.createButton({
        title: 'Register',
        color:'#fff',
        font:{
            fontSize:17,
            fontFamily:"Helvetica Neue"
        }
    }),

    rightNavButton: Ti.UI.createButton({
        title: 'Sign In',
        color:'#fff',
        font:{
            fontSize:17,
            fontFamily:"Helvetica Neue"
        },
        enabled:false
    }),
    navTintColor:'#fff',
    barColor:'#1179a0'
});

The view I want to set up

indicatorWin = Ti.UI.createView({
    width:'auto',
    height:'auto',
    backgroundColor:'#000',
    opacity:0.5
});

  • Hi,

    you can create a separate .js file. In that create one window and add activity indicator plus create 2 custom add event listeners. one is for show indicator and second is for hide indicator.

    in show indicator, open that window and on hide indicator, close that window.
    include that .js in your app.js file to register those 2 custom listeners then you can fire event from any where in your application.
