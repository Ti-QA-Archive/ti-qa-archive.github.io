Titanium Studio and OS X Mavericks - Error generating AST file with jQuery

Hi

I've just updated to OS X Mavericks and updated to Titanium Studio 3.1.4.201310152018 as recommended on the Appcelerator blog. I'm building an Alloy app which uses jQuery to create charts in a webView and I'm now getting an error when compiling my app:

[WARN] :   : ERROR: Unexpected character '' [jquery.1.9.js:5,28407]
[ERROR] :  Error generating AST for "/Volumes/IT_Department/Design/_team/James/THW-IOS-App-Local/Resources/jquery.1.9.js"
[ERROR] :  Unexpected character ''
[ERROR] :  line 5, column 28407, position 92629
[ERROR] :  Alloy compiler failed

I don't think there's a need for any code snippets here as all my views, styles and controllers are compiling fine, it's just any third party JS files I have in my "lib" folder. These files were not causing any errors before I updated my software, so it must be related to the new update. Has anyone else seen this? I think it might be related to JS linting but I don't know how to modify this. Any ideas?
