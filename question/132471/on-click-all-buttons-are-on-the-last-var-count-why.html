On Click all buttons are on the last var count, why?

Newbie to Titanium.. Some code I developed for a rating system, under the for loop, the bn variable keeps updating to whatever number it is set to, so when you click on one of the button, its always the last number for all buttons in the Ti.API.info or in the click event under button[bn].setTitle('*') always update's the last button.

/**
 * create a view object to hold the buttons
 * and add the buttons into the view
 */
function createRatingButtons(numButtons,BarTitle,topspace) {
    // set vars
    var bn=0;
    var left = 5;
    var top    =    5;
    /*
     * create a view for the buttons
     */
    var ratingView = Titanium.UI.createView({
        height:        100,
        color:        'white',
        top:        topspace,
    });

    /*
     * create a label to put into the view
     */
    var ratingLabel = Titanium.UI.createLabel({
                text:                            BarTitle,
                color:                        '#fff',
                backgroundColor:     'transparent',
                textAlign:                'left',
                height:                        'auto',
                width:                        'auto',
                top:                            0,
                left:                            left,        
    })
    ratingView.add(ratingLabel);
    /*
     * do the for loop and add
     * the buttons to the view
     */
        var button = [];
    for(bn==0;bn<numButtons;bn++) {
                     button[bn] = Titanium.UI.createButton({
                title:              bn,
                width:              50,
                height:             50,
                color:              "black",
                // backgroundColor:    "blue",
                left:               left,
                top:                                top+ratingLabel.getHeight(),
            });
            /*
             * Add event handler for this button
             */
            button[bn].addEventListener('click', function(e)
                        {
                            Ti.API.info("Rating Button Click #: " + bn);
                            /*
                             * Update buttons below this count for this object
                             * to have colored stars, and all starts after this
                             * to be uncolored.
                             */
                            button[bn].setTitle('*')
                        });


            ratingView.add(button[bn]);
            left = left + 50 + 5;
    }
    //bn = 0;

  // return the entire block for this view  
    return ratingView;
}

Method call

scrollView.add(createRatingButtons(5,"test1",0));
scrollView.add(createRatingButtons(5,"test2",95));

    You need a closure around the bn variable in your event handler to give it local scope (see this article for details on why). Try this:

    button[bn].addEventListener('click', (function(x) {
  return function(e) {
    button[x].setTitle('*');
  };
)(bn));
