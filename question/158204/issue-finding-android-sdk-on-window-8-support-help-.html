    I've just installed Studio onto a freshly installed Win 7 without any installations from other SDKs (eg. JDK). Don't worry. You only need to take a few more steps:

    1. When installed Studio is only able to develop on a web app mode. Nothing to panic about here.
    2. From the Dashboard tab select Get Started sub-tab.
    3. Scroll down to Configure Native SDKs.
    4. Select Android SDK. Studio will install a whole bunch of stuff it needs, including all the Java, Node.js, and everything you need to dev on Android.
    5. Once done it will open the Android SDK Manager. On a fresh install the SDK Manager does not come with all the libraries Studio needs. The important ones are already selected, so you know what to install. Remember that you need to install API 10 as well. What you need to install are stated in the Studio page.

    That's it! I simplified step 5 a little but that's really the kookiness of the SDK Manager, not within Studio's control. Ask me again if you're tripping over that step. Will suggest you try a little more so you understand how things work.
