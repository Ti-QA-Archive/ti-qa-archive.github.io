fetch single record by id via alloy models

Hello,
I'm trying to find the correct/best/cleanest way to load a single alloy model from an ID. Eg

var myModel =  Alloy.createModel('myModel');
myModel.fetch({id: 123});

This return a collection with all the records.

myModel.fetch({query: &#39;select * from ... where id = &#39; + 123 });

This works, but is pretty verbose. Is it correct or does exists a better way?

Thanks in advance

(using 3.1.2GA, Alloy 1.2, android)
