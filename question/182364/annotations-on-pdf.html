  • Out of the box support for annotation is not possible using Appcelerator. For it you need to take support from 3rd party libraries, or if you intend to create it on your own then the answer is modules. You can try Foxit, they have both Android and iOS modules (not free, but free to try). Use those libraries, create modules and then use it in your application. Also note that, in iOS, by default WebView might not display annotations properly.
