  • Without looking at code, it is difficult to say exactly what the issue is. However I suspect that you have an inbuilt navigation system which has back buttons to go to and load previous windows. Which is fine.

    But the android hard back button will not know what this is so you will need to handle this within your code by adding the android:back event listener and processing it through you navigation system,

    Hope this helps

    T.
