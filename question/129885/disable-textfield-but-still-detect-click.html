Disable textfield but still detect click

Hello, I'm stuck on this …

I want to disable a textfield but still have a listener detect a click/focus on that field. I've tried adding a rightbutton in the textfield with it's own listener but disabling the textfield disables the rightbutton as well. I guess I'd expect it to work that way.

I've also tried editable:false but that doesn't seem to have any effect.

The reason for all of this is that I don't want the keyboard appearing when the field gets focus or is touched (I want the event to handle things) but that seems unavoidable. It seems that disabled:true is the only way to stop the keyboard but that also prevents listeners from receiving events from that field.

As a work around, the first thing the listener does is call a blur on that field to hide the keyboard but this seems ugly.

Thanks!
