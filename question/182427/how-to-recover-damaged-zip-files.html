    Please try Recovery Toolbox for Zip is used to recover files from corrupted ZIP archives. The program scans the archive, detects the data structure and tries to recover as much information from the corrupted file as possible. Using several different recovery algorithms, the tool makes it possible to achieve the minimal loss of useful data stored in the corrupted archive. Also, it checks data integrity, which allows it to increase the quality of recovery. The zip recover program shows the full data recovery report during its operation.

    For more: https://zip.recoverytoolbox.com/
