Dashboardview without delete function

Hey developers,

I've made myself a dashboard view with the createDashboardView function. When I hold an icon on the view it starts to wobble and shows a delete button. Is it possible to remove that delete button? I do want the user to be able to change the order of the icons, but I don't want to show the delete button..

Kind regards,
Niels

    Just set the canDelete Property of your DashboardItems to false.

    for (var i=0; i<items.length; i++){
  var item = Ti.UI.createDashboardItem({
    badge: items[i].badge,
    image: '/images/dashboard/' + items[i].name + '.png',
    label: items[i].name,
    canDelete: false
  });
  dashboardData.push(item);
}
