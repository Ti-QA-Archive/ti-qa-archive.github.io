How to design TableViewRows ??

I have a table view, which I populate with data from a database.
For each row in the database, I put a single row into the table view.

Each row in the table view is required to be composed of two separate rows of text, with a single line of text in each row.

Currently I am using two labels, called topLabel and bottomLabel, however both of these labels appear to be displayed on top of each other and it looks a mess!

Question:

How do I re-design this in such a way as to have each label on their own separate row, within each table view row?

var row = Titanium.UI.createTableViewRow({
    width:"auto",
    height:42,
    hasChild:true,
    url:"view_object.js"
});

var topLabel = Titanium.UI.createLabel({
    text:"I am the top label"
});                     
var bottomLabel =  Titanium.UI.createLabel({
    text:"I am the bottom label"
});

row.add(topLabel);
row.add(bottomLabel);

Thanks guys..

  • hello Donal,

    you can manage label into tableView rows like below,

    var win = Ti.UI.createWindow({

});

var table = Ti.UI.createTableView({
    backgroundColor:'blue',
    top:50,
    height : 160
});

win.add(table);

var data = [];

for (var i = 0; i < 4; i++) {

    var row = Ti.UI.createTableViewRow({
        height : 40
    });

    var label1 = Ti.UI.createLabel({
        text : 'label1',
        top : 5,
        left : 10,
        width : 80,
        height : 30
    });

    row.add(label1);

    var label2 = Ti.UI.createLabel({
        text : 'label2',
        top : 5,
        left : 130,
        width : 80,
        height : 30
    });

    row.add(label2);

    var label3 = Ti.UI.createLabel({
        text : 'label3',
        top : 0,
        left : 240,
        width : 80,
        height : 30
    });

    row.add(label3);

    data.push(row);
}

table.data = data;

win.open();

    for more just refer kitchensink so you can get better idea.
