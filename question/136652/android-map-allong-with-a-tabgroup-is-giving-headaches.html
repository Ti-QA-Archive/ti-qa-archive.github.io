Android map allong with a tabGroup is giving headaches

Hello everyone,

We are making a little app for school.
But we ran into a little strange problem.

We have a (google) map in our app and the window also has a tabGroup.
The app will sometimes show the map, but othertimes (most of the time) it won't.
It does work on Apple though, so only for android
However if we disable the tabGroup the map will show up.
Is there some colission with using the tabGroup along with a map?
Is there some solution to this problem or workaround?

Our partialy code for the ui.

var tabMenu = Titanium.UI.createTabGroup(); // de container van het menu

var MapTab = Titanium.UI.createTab({
    icon: '../img/Map.png',
    title: 'Map',
    window: MapWindow     
});

var ListTab = Titanium.UI.createTab({
    title: 'Lijst',        
    icon: '../img/lijstje.png',
    window: ListWindow 
});


tabMenu.addTab(MapTab);
tabMenu.addTab(ListTab);

tabMenu.open();

and the code for the map:

var MapWindow = Titanium.UI.createWindow({
    title: 'Map',    
    backgroundColor: '#FFFFFF',
    navBarHidden: true
});

var mapView = Titanium.Map.createView({
    bottom: 0,
    left: 0,
    height: "100%",
    width: "100%",
    mapType: Titanium.Map.STANDARD_TYPE,
    animate: true, 
    regionFit: true,
    userLocation: true
});

MapWindow.add(mapView);
MapWindow.open();

We are using the newest SDK.

Any help would be appreciated.

Greetings
