Making major changes to an iPhone app's main toolbar

I'm about to implement the design for an iPhone app I'm creating and the design calls for completely custom looking window toolbars. The background of the toolbar is completely transparent and the toolbar is quite a bit wider vertically than the standard iPhone toolbar. The buttons also have to be customized.

In order to accomplish this will I need to completely write my own toolbar for each screen in my app, or is there a way to customize the standard, automatically generated toolbar? So far I've only been able to affect the toolbar's transparency by setting backgroundColor to 'transparent', but the toolbar's gradient still shows through.
