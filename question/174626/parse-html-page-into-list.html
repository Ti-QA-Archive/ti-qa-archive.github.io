    Hi

    It is possible - but not easy and not ideal. You should really have the data come back as an API. But I guess this web page is not in your control.

    You could use a webView, load the page you are after into an iframe within that page and use jQuery to grab the content you need. Then use fireEvent to pass the information back to the app. Not ideal, not easy and I have no code to show you - but if you do some digging it can be done.

    It is only not easy because it is not for beginners. But it you can do it on the server and get it back as an API of just the data.
