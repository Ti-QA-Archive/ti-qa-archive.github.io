Problem in installing NDK

I am using 64 bit MAC machine

I downloaded NDK from the following site

http://developer.android.com/tools/sdk/ndk/index.html

So it gave me the file–android-ndk-r10d-darwin-x86_64.bin

which after double click converted into –android-ndk-r10d-darwin-x86_64.bin.cpgz

I have prepared module but when i run build.xml through Ant it gives me the following error :-

BUILD FAILED
/Users/bunty/Library/Application Support/Titanium/mobilesdk/osx/3.4.1.GA/module/android/build.xml:297: The following error occurred while executing this line:
/Users/bunty/Library/Application Support/Titanium/mobilesdk/osx/3.4.1.GA/module/android/build.xml:247: Neither the ANDROID_NDK environment variable, or the android.ndk property is not set to an existing Android NDK installation (check your module's build.properties).

In build.properties following is added

module.ndkbuild=true

android.ndk=/Users/bunty/Downloads/android-ndk-r10d-darwin-x86_64

But still I am getting this error.

Can anyone provide me the solution of above query?

Thanks in advance :-)

    Hi Bunty Rohra ,

    Thanks for your concern. Please flow the steps :

    Step 1. Download android-ndk64-r10-darwin-x86_64.tar.bz2

    Step 2. Extract the file .

    Step 3: Set NDK path: Titanium studio > Preferences> Platforms>Android >NDK .

    Step 4. Configure your build.properties

    titanium.platform=/Users/------------------/Library/Application Support/Titanium/mobilesdk/osx/3.5.0.GA/android
android.platform=/Users/------------------/android-10
google.apis=/Users/------------------------/addon-google_apis-google-10
android.ndk=/Users/----------------------/android-ndk-r9d

    Please Check your module's build.properties and set the correct path.

    Thanks
