    Hello Sabari,

    Try this :

    var data = [{
        title : 'A',
        no : '10'
    }, {
        title : 'E',
        no : '130'
    }, {
        title : 'C',
        no : '110'
    }, {
        title : 'D',
        no : '190'
    }, {
        title : 'B',
        no : '100'
    }];

    data.sort(function(a, b) {
        var keyA = a.title, keyB = b.title;
        if (keyA < keyB)
            return -1;
        if (keyA > keyB)
            return 1;
        return 0;
    });

    Ti.API.info('>>>' + JSON.stringify(data));

    i hope this helps. thanks.

    • @Sanat; some constructive criticism (Good Answer though) .

      You have not used braces in your code for if statements, whilst perfectly legitimate this technique should not be used to explain something to someone who is learning. This is because the potential to not fully understand this short-cut could lead to developers extending the code and not realising that successive code inside the rows will break the code.

      This is a better and safer example;

      var arrSortedData =  data.sort(function(a, b) {
    var keyA = a.title,
        keyB = b.title;
    if (keyA < keyB) {
        return -1;
    }
    if (keyA > keyB) {
        return 1;
    }
    return 0;
});

    • If you use the underscore.js library - included with Titanium with Alloy but also can be used with Titanium without Alloy (copy the file and require it, then you can do the sorting in one line;

      // sort by title
data = _.sortBy(data, function (element) { return element.title; });
// sort by number
data = _.sortBy(data, function (element) { return element.no; });
