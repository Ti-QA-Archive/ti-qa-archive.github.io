Webview url not loading

I have the following error, can't find what to do…

[WARN] couldn't load URL: app://com.XX.XXX/pages/www.XXXX.com

var w = Ti.UI.currentWindow;
var url = w.title;
 Titanium.API.info("url:" + url);
 Titanium.API.info("W.url:" + w.title);

w.orientationModes = [Titanium.UI.PORTRAIT]; 

var webview = Ti.UI.createWebView();
    webview.url = url;
    webview.scalesPageToFit = false;
    w.add(webview);

    Hi Julien, i think your url is not valid in this case.

    it is like :: app://com.XX.XXX/pages/www.XXXX.com

    where it should start with http/https and then go on example : http://www.google.com

          webview.url = 'htpp://' + url;
