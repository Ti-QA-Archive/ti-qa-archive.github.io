  • Making sure the adb server is running:

    alte-flunder:~ rainerschleevoigt$ cd android-sdk/platform-tools/
alte-flunder:platform-tools rainerschleevoigt$ ./adb kill-server
alte-flunder:platform-tools rainerschleevoigt$ ./adb start-server
* daemon not running. starting it now on port 5037 *

    Now you can test with:

    alte-flunder:platform-tools rainerschleevoigt$ netstat -an | grep 5037
tcp4       0      0  127.0.0.1.5037         *.*                    LISTEN
