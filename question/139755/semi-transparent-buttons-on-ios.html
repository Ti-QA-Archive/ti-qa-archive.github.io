Semi-Transparent buttons on iOS?

I was trying to use the color specs in the SDK to give my buttons a semi-transparent background color, as laid out under "Color" in the SDK

However, no matter what format I use, the button backgrounds are alway opaque white. For example, even this:

width : 230,
height : 115,
left: 10,
title : 'New Roof',
font : {
    fontFamily : 'Helvetica Neue',
    fontWeight : 'bold',
    fontSize : 18
},
backgroundColor : '#42759D',
});

Gives me this:

Note in the screenshot, the area outside the button's corners is correct.

Can this even be done with a button, or do I need to use a view?
