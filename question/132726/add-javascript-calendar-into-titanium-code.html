  • There used to be a JavaScript calendar available for Titanium - how well it works under Android I can't tell you, but you can give it a go.

    Note that this is a very old (at least 18+ months) library and I frankly don't think it's even available on GitHub anymore, and I would really recommend you look around for either a native module or something that doesn't rely on HTML/WebViews. I am posting it so you can try it out and maybe hack it to your needs anyway.

    Here's the library:
    http://hotfile.com/dl/147666966/29c5e58/libraries.zip.html

    Here's a sample implementation:

    Ti.include('../libraries/calendar/calendar2.js');

var dateView = Titanium.UI.createView({
    width: 310,
    left: 0,
    top: 0,
    height: 320
});
win.add(dateView);
selectDate(dateView);
