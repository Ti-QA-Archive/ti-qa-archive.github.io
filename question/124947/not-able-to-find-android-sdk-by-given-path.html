Not able to find Android SDK by given path

I installed Titanium Studio on WindowsXP, then installed Android SDK in the path C:\Program Files\Addroid\androdi-sdk as the window's installer suggested. When configuring Android SDK in Titanium studio, it always gives error "Could not locate the Andorid SDK at the give path." I tried uninstall/reinstall many times, still no luck. Very frustrating.

    • Thanks for pointing to the helpful thread. After reading answers/comments of the mentioned thread, I got it resolved. Here are what I did,

      1. I checked the log file (Help -> Titanium ->View log) to make sure that my problem is similar as what others had
      2. Make sure env. var is set and referenced in the path
      3. Installed the required SDK version that complained in the log file, in this case is SDK2.1 update API7
        Finally, SDK version showed up after entering the path.
