Make things work for Xcode 6 and latest IOS SDK

Hi, I just recently updated my Xcode to 6.0.1 and installed the latest version for SDK and simulator.

However, I cannot build and run ios application directly in Studio since then. It will always tell me the following error:

[INFO] :   Focusing the iOS Simulator
[ERROR] :  Project failed to build after 22s 765ms

I found the following article provided a clever walk around for this problem:
http://www.tidev.io/2014/06/03/using-xcode-6-for-titanium/

which simply build the ios application in studio and run it using 'build/ios/XXX.xproject' file in Xcode because the reason leading to this question is the studio is not compatible with the latest simulator.

However, my question is:
Is that the only way solving the problem? I haven't yet find and question related to Xcode 6 in the community and any issue trackers related to this problem. Hope it can be solve soon. Thanks!

  • Accepted Answer

    We have seen occasional reports of this problem. However, prior to the release we were unable to reliably reproduce the issue and determine an exact cause. I believe that TIMOB-17601 is the related ticket. Our workaround will be to extend the time we wait for the simulator to launch, as the issue seems most common on slower computers.

    BTW, we've had reports that Objective-C / Xcode projects (no Titanium involved) also occasionally experience this same issue. The iOS8 simulators take a long time to load and sometimes simply fail to load.
