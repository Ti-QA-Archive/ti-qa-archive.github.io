Making the status bar text white everywhere in alloy

Hi, i need to make the status bar text white in alloy. I have tried setting LIGHT_CONTENT in tiapp.xml but nothing changed. How do i do this??

    Hi

    Can only be used on Ti SDK 3.1.3 and higher - assuming you have that - just double checking.

    Set statusBarStyle on the main window you open to the value you require. Then do not set this on any other window.

    • Whats the markup though? is it:

      <Window id="win" class="container" statusBarStyle="Ti.UI.iPhone.StatusBar.LIGHT_CONTENT">

      — commented March 13th 2014 by Matt Pilott

    • That should work but it is best to put it in the TSS as this is a style thing rather than the xml as a property.

      ".container": {
    statusBarStyle:  Ti.UI.iPhone.StatusBar.LIGHT_CONTENT
}
// or
"#win": {
    statusBarStyle:  Ti.UI.iPhone.StatusBar.LIGHT_CONTENT
}
// or
"Window": {
    statusBarStyle:  Ti.UI.iPhone.StatusBar.LIGHT_CONTENT
}

      Depending on your scope, it should be referenced once of course so pick the best of the ones above for your needs.
