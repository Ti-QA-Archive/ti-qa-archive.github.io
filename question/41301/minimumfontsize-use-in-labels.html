minimumFontSize use in labels?

Hi, the KS example only shows use of minimumFontSize in a textfield. minimumFontSize is also listed as a property of Labels in the docs. However, it doesn't seem to work. Can anyone confirm/deny whether or not Labels support this property? (I have a label element which I dynamically assign the text; it would be great if it would auto size the font to fit the label's width property).

Thanks!

    This is supported for labels. It is important to set a width or both left & right properties for it to work though.

    I am having some trouble with it on Android right now, but iPhone works great.
