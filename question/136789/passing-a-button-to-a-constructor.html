Passing a button to a constructor?

I have been using this method to create repetitive objects in my iOS app, with items like labels, views, and buttons. I am now trying to use this to have a constructor for a toolbar where I pass the button, but it fails, complaining that "btn" is undefined

var aButton = Ti.UI.createButton({
    title : 'Done',
    height : 50,
    width : 150,
    style : Titanium.UI.iPhone.SystemButtonStyle.DONE
});
var introBar = new botBar(aButton);
var botBar = function(btn) {
    return Ti.UI.iOS.createToolbar({
        items : [flexSpace, btn],
        barColor : '#ee3c03',
        bottom : 0,
        borderTop : true,
        borderBottom : false
    })
}

SDK 2.0.1, iOS 5.1
