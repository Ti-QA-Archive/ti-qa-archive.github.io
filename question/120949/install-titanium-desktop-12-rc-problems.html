  • Always make sure your SDKs go in the root /Library/Application Support/Titanium/sdk/osx/[sdk version] folders and not your user ~/Library/…. one

    naturally the desktop SDKs go in the 'sdk' folder with the Titanium one and the mobile ones go in the 'mobilesdk' folder.
