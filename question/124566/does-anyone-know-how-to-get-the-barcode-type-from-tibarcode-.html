  • In the success function, if you log the data it gets, you will see something like this:

    [INFO] TiBar success callback!
[INFO] {
     barcode = 8717163279038;
     source = "[object TibarModule]";
     symbology = "EAN-13";
     type = success;
 }

    So data.symbology is what you need :)
