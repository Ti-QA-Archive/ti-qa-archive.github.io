Installing extra files on users system(on app install)

Hi, was wondering if anyone knows if it is possible to install additional files onto a users system. e.g installing a certain font. esp in Windows(When your app first installs on their system), and is there any way of customizing the defaut installer?

  • Yes, this is entirely possible. For example if you have large video/sound/image files that you do not want included in the packaged app that, for example, make it larger than 20MB (The limit for downloading an app via 3G on the iTunes App Store), you could host that media file and then download it on the first opening of your app.

    For more information on storing these files, check out the Ti.Filesystem module.
