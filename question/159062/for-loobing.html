  • Hi

    Really now sure what view means in this context as you have have not indicated, but if you want a loop to repeat more than once there are several ways;

    • Very simple method allows you simple repeat the loop exactly as it stands
    var loop = function () {
    for (var x = 0; x < 5; x++) {
        $.bubbels = x;
    }
};
loop(); // run first time
loop(); // run second time
    • More re-usable - allows you to call the loop multiple times but with the ability to call the loop using different start and end points;
    var loop = function (from, to) {
    for (var x = from; x < to; x++) {
        $.bubbels = x;
    }
};
loop(0, 5); // run first time
loop(0, 10); // run second time
    • Classic method;
    for (var y = 0; y < 3; y++) {
    for (var x = 0; x < 5; x++) {
        $.bubbels = x;
    }
}
// outer loop runs the inner loop 3 times
