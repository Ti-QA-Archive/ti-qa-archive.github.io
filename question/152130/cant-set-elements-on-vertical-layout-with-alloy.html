    A table view doesn't have a vertical layout. only normal views, windows and scrollviews have.
    Vertical layout is used when you want to add multiple controls to a single view and specify the top of each of them relative to the previous one instead of relative to the parent view.

    • Here you go. it's better that you put the style in the appropriate tss file

      
<Alloy>
    <Window class="container">
        <View layout="vertical">
            <TextField title="username" top="10" left="20" width="50%" hintText="Username" backgroundColor="gray"/>
            <TextField id="password"  top="10" left="20" width="50%" passwordMask="true" hintText="Password" backgroundColor="gray"/>
            <Button id="myButton" top="10" left="20" width="50%" title="login" onClick="openMenu"/>
        </View>
    </Window>
</Alloy>
