  • Hi

    If you are using Alloy then there is a built in library for handling date and time called moment.js, and here is the example for your question:

    var moment = require('alloy/moment');

var a = moment();
var b = a.hours();
var c = b * 60;
Ti.API.info(c);

    If you aren't using Alloy then you can still use the moment.js library by downloading the JS and adding it to your app's library. Which I would advise because it's so simple!
