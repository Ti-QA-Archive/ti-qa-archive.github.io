scrollTo slide on Android

Hi, I'm trying to implement scrollTo on a Ti.UI.scrollView object. It executes the function well, but I'd like it to actually slide, instead of jumping to the scrollTo position immediately.

Is there a way to implement this? If the scrollTo method doesn't allow me to do this, then I'd like to get the current position of the scrollView. Is there a way to do this? I tried contentOffset but it doesn't seem to be correct.

Here's my code:

navigationBar.addEventListener('customClick',function(e){
    scrollView.scrollTo(0,e.position); // scrolls to immediately w/o sliding
};

    • For anyone getting here by search: The scroll event's x and y properties contain the scroll offset of the scrollView. One workaround for the missing contentOffset would therefore be to have a variable somewhere that contains the scroll offsets and update it on the scrollView's scroll event.
