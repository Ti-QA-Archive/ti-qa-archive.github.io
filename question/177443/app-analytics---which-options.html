    • Hi Michael,

      a little note from the analytics module readme.

      A Warning About Tracking IDs

      Google Analytics Website Tracking IDs will NOT work with this module (because they don't work with the underlying Google SDK). Instead, you must generate new App specific Tracking IDs:

      • Go to analytics.google.com.
      • Sign in.
      • Go to Admin > Accounts > Create New Account.
      • Choose "App" beside "What would you like to track?"
      • Create the account, and get the tracking ID.

      Hope that helps,

      Matthias
