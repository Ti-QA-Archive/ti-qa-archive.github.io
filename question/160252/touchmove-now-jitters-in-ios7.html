TouchMove Now Jitters in iOS7

While updating my app to make it be compatible with iOS7 i've run into some issues. I have a notification drawer pulldown in my app that a user can drag down to find out more information. Anyways in my previous versions it worked flawlessly, but with iOS7 and 3.1.3GA the view jitters around. You'll see an example in the video link at the bottom. Below is a snippet of the code that is being used. I've tried animating the view with a duration of 1, which made it smoother and worked before. I've even tried just setting the view bottom to the current touch position but it still jitters like crazy. Does anyone know a fix for this, or found a better way to do this.

I'm using iOS 7.0.3, TI SDK 3.1.3GA and Studio 3.1.4

itemsInfoDrawer.addEventListener('touchmove',function(e){

        var localPoint = {x:e.x, y:e.y};
        var convPoint = itemsInfoDrawer.convertPointToView(localPoint, Window);

        itemsInfoDrawer.animate({bottom:convPoint.y, duration:1});
        //itemsInfoDrawer.bottom = convPoint.y;
    });

  • It's jittering because you are responding to every single touchmove event. Try animating when the x/y coordinates have moved more than (for example 5 pixels, then fine tune)
