Send Push Notifications with ACS using command line?

I have my push notifications setup and they work fine when using the cloud console. The problem is I want to automate the push notification process and create the push notifications as a batch job or some sort of trigger from a database and cannot figure out how to do it command line.

http://docs.appcelerator.com/cloud/latest/#!/api/PushNotifications shows that you can do this but doesn't explain where you put your credentials in so you can send it.

How do you use this example and what are the cookies.txt?

$ curl -b cookies.txt -c cookies.txt -F "channel=friend_request" -F "to_ids=123456,34567" -F "payload=test" https://api.cloud.appcelerator.com/v1/push_notification/notify.json?key=<YOUR APP APP KEY>

  • Heres the answer… you need to do two calls… first one to log in second one to send the send out the notification

    curl -b cookies.txt -c cookies.txt -F"login=myid" -F "password=mypw"  https://api.cloud.appcelerator.com/v1/users/login.json?key=myappid

    curl -b cookies.txt -c cookies.txt -F  "channel=test" -F "payload=new alert" https://api.cloud.appcelerator.com/v1/push_notification/notify.json?key=myappid
