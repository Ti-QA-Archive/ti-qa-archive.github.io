  • Polylines are supported in Titanium for iOS, including polygons. But, there isn't a directions API. You'll need to generate an array of lat/lng from some data provider, then use this array as points. Use the mapview.addRoute() method.

    Also note, you can shade an array of points to create a boundary area as well.
    https://skitch.com/tzmartin/fq74f/titanium-mobile-mapview-polygon

    Ping me if you want the modified code that adds a fillColor property.
