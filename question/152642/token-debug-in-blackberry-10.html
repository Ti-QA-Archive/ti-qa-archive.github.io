    Hi Alvaro,

    before using a debug token, you need to register for signing keys and then create and install that debug token.

    You will find instructions at Titanium Mobile BlackBerry - README.md on github.

    The register/signing/token part is here: How to Create Titanium Applications using the BlackBerry SDK
