    Firstly let me explain tableView and TableView Row:

    Table View : A table view is used to present information, organized in sections and rows, in a vertically-scrolling view.A TableView object is a container for Table Section objects that are, in turn, containers for TableViewRow objects.

    Table view row: A table view row is an individual item in a table, organized into table view sections.

    Now moving on to the functions of the view:
    As per your question I think you are getting confused with programmatically selecting a row and user selection

    1) Table View Functions: Firstly if you want the user to select a row , then you need to add a event listener which listen to click event of the row. 

    table.addEventListener(‘click, function(e){
    Ti.API.info('Row swiped: ' + e.source); 
    Ti.API.info('Row ID : ' + e.source.rowID);
});

    If you want to programmatically select a row in the table then you use selectRow(row) function , where you pass the row index.
    Similarly programmatically to deselect a row in the table you use deselectedRow(row)

    2) Table View Row Functions: the function getHasChecked() and setHasChecked(boolean) works with the property hasCheck ( Determines whether a system-provided checkmark is displayed on the right-hand side of the row. ) .
    We use setHasChecked(true) if we want a check mark to appear on that row and setHasChecked(false) if we don want it.
    And programmatically to check what is the value of the HasCheck property of the row we use the function getHasChecked() .

    I have tried to explain your first question here , in my next post will try to explain 2 and 3 queries.
    I hope this is clear to you.

      Ti.UI.backgroundColor = 'white';
var win = Ti.UI.createWindow();

var tableData = [  'Apples',  'Bananas', 'Carrots', 'Potatoes' ];

  });

    color:'#000',
    text:tableData[i],
    left:20,

  });

  tableData.push(row);
}


var table = Ti.UI.createTableView({
  data: tableData
});


//click event called when user presses the row of the table
table.addEventListener('click', function(e){

 Ti.API.info('Row clicked: ' + e.source); // row clicked

 // programmatically use get and set HasCheck to give toggling effect of check image 
 if(e.source.getHasCheck()){
     e.source.setHasCheck(false);
 }else{
     e.source.setHasCheck(true);
 }
  table.selectRow(e.source.rowIndex); // when we click the row that row gets highlighed with selectedBackgroundColor which is red in our case   
});


win.add(table);
win.open();
