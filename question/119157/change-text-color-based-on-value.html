Change text color based on value

My label text comes from a remote JSON feed, and I want the color of the text to be different based on whether the data contains a minus symbol or not.

In particular how would I go about identifying if the '-' character is present.

Thanks for your help !

  • var negative = myString.substring(0,1);

    var color = negative == "-" ? "red":"green";
