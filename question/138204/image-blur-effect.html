Image blur Effect

I need to put blur effect on Imageview on button click.
I have one image view and two buttons VIZ (1) Btn_blur (2) Btn_reset.
I need to blur a image on Btn_blur click event and reset blur image to orignal image when clicking on Btn_reset.
any way to achieve this .

  • What you want to do is make use of the touch events.

    • When the user touches the view (touchstart), you change the image.
    • When the user releases his touch (touchend) on the view, you change back to the original image. and perform the action you need.
    • If the user moves his finger outside of the view (touchcancel), we simply revert to the original image and take no action.
      ~~~
      img.addEventListener('touchstart', function(e) {
      img.backgroundImage = 'clicked.png';
      });

    img.addEventListener('touchend', function(e) {
    img.backgroundImage = 'reset.png';
    // Perform the code you would have normally done in the click event
    });

    img.addEventListener('touchcancel', function(e) {
    img.backgroundImage = 'reset.png';
    });
    ~~~

    Don't forget to remove your click event handler. Otherwise, they will conflict with one another.

    Hope this helps
