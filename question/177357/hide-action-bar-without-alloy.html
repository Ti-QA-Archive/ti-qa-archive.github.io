    Hello, Lu Pin.

    There are two ways to hide the android action bar

    1.Add this Following Code to your your JavaScript code: Get the activity's action bar instance and call the hide method.

    // Wait for the Window (or TabGroup) to render
win.addEventListener('open', function(){
    // Grab the window's action bar instance and call the hide method
    win.activity.actionBar.hide();
});

    Or,

    1. To modify the theme to hide the action bar: To do so, add a custom theme file to your project.

    • Create a directory in your project named "platform/android/res/values/custom_theme.xml"

    • Copy below code to "custom_theme.xml" file

      <?xml version="1.0" encoding="utf-8"?>
<resources>
  <style name="Theme.NoActionBar" parent="@style/Theme.Titanium">
      <!-- Depending on the parent theme, this may be called android:windowActionBar instead of windowActionBar -->
      <item name="windowActionBar">false</item>
  </style>
</resources>
    • Copy below code to your "tiapp.xml" file
      <android xmlns:android="http://schemas.android.com/apk/res/android">
  <manifest>
      <application android:theme="@style/Theme.NoActionBar"/>
      <!-- Need to specify at least API level 11 for Titanium SDK 3.2.x and prior -->
      <uses-sdk android:minSdkVersion="14" android:targetSdkVersion="19"/>
  </manifest>
</android>

    Thanks.
