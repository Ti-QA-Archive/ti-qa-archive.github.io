Error starting Fastdev server

I have installed everything on my windows 7 pc like the manual suggest and when i am trying to launch the Kitchen Sink app in Titanium Studio i get this error:

Error starting Fastdev server: 'ascii' codec can't decode byte 0xf6 in position 24: ordinal not in range(128)
[TRACE] app property, ti.android.debug : false
[TRACE] app property, ti.facebook.appid : 134793934930
[TRACE] app property, ti.android.google.map.api.key.development : 0ZnKXkWA2dIAu2EM-OV4ZD2lJY3sEWE5TSgjJNg
[TRACE] app property, ti.android.google.map.api.key.production : GET_ME_FROM_GOOGLE

Does anyone know how to fix this?

  • Kenneth

    Make certain that you are using an Android SDK that is named in the form "Google APIs Android x.x". Try "Google APIs Android 2.1-update1" to begin with. This is important, because the KS uses Google's maps, which are only available in these versions of the SDK. See the last colored textbox in the Android SDK / Target Android Platform section of the "Titanium Compatibility Matrix" guide for more information.

    Cheers
