How to change view's property from another tab

titanium mobile sdk 2.0

iphone sdk 5.1

Hi,

Basically, my problem is how to change object's property from another tab.

I've got tabgroup with 2 tabs. the first tab has a button the second tab has a view. The idea is that when the button is clicked, the view's backgroundcolor on the 2nd tab should change to, say 'white'.

thanks all.

  • Hi Nash,

    You do it using fireEvent like this way.

    put this code where your view is

    Ti.App.addEventListener('changeColor', function(evet) {
    view.backgroundColor = 'white';
});

    and put this line on button click event.

    Ti.App.fireEvent('changeColor');
