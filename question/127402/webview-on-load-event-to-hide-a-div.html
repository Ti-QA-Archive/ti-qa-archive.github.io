webView on load event to hide a div

I'm loading a webView and want to hide a title bar div when the webview loads. Can I do this? I've looked through the forum and have found a few examples of the 'load' event on the webView but I can't get this to work.

Here's my code:

var webView=Titanium.UI.createWebView({url:myURL})

webView.addEventListener('load',function(e)
{
webView.evalJS("document.getElementById('titleBar').style.display='none';");
});

The page all loads ok but the titleBar div is still displayed.

Any help, much appreciated!

  • I'm not sure, but web view's finish of loading (so, when "load" event gets fired) doesn't mean that DOM is loaded, therefore you would need to wait until DOM is loaded, something like this:

    webView.evalJS("window.onload=function(){document.getElementById('titleBar').style.display='none';}"); });
