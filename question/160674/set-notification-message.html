Set Notification Message

How can i set notification message dynamically? Also wants to change it's position. I have tried following:

index.xml


<Notification id="notificationMessage" platform="android" duration="Ti.UI.NOTIFICATION_DURATION_LONG" gravity="80"/>

index.js


$.notificationMessage.message = "Hello World !!! ";
$.notificationMessage.show();

Please help me.

  • You can set the message with setMessage

    $.notificationMessage.setMessage("hello");
$.notificationMessage.show();

    You can change the position as answered at yesterdays similar question: http://developer.appcelerator.com/question/160595/notification-position

    If it does not work for you, please describe the error. You have not mentioned what you get with your code.
