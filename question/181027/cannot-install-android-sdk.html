Cannot install Android SDK

I have installed Appcelerator Titanium (based on Eclipse) on my Mac (OS X 10.10), as well as the CLI tools including Node.js. I have Android Studio installed. This is my first attempt at using Titanium.

When I startup Titanium, I see the dashboard, and then click the Get Started tab. Only iOS has a green check, both Android and Blackberry are red, meaning no SDK is configured or installed.

I can click the Android link in the dashboard, then click the configuration button there. This shows the screen. I click configure and wait. It seems to succeed, but then it didn't.

It downloads the SDK, seems to install it, and I can find the installed SDK in my user library (Library/android-sdk-macosx). I see platforms and system images in there with subfolders like android-19 and android-20 and more.

Am I missing something? I want to get started, especially with Android apps, and I have no idea how to get the SDK installed.
