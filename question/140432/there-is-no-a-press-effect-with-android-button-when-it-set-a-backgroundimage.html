There is no a press effect with android button when it set a backgroundImage?

I have set a button's backgroundImage, and I found that there is no a press effect(highlight the button) with it in android, but it's fine in iphone, how can I make it work in android?

thanks!

    Hi Winson

    If you set a background image with Android you should also set a selected versions as well.

    backgroundImage: 'back.png', // yours
backgroundSelectedImage: 'selected.png'  // new one

    This should do what you need.
