  • Desktop apps built via Appcelerator are based on the Webkit engine, so anything you can do normally via HTML, PHP and Javascript (not Java!) on a website you can in "Desktop".

    Sor far I've developed two apps using the inbuilt SQLite database and it all works pretty well.

    As for online resources? Although they are trying, Appcelerator's documentation is still poor, spread all over the place and incomplete.

    Try http://developer.appcelerator.com/doc/desktop/guides

    Somewhere there is also a full guide to the SQLite database but I can't rememeber where they've hidden it.
