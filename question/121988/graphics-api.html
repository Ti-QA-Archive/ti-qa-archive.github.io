  • You can create graphics with the standard views as long as you do not need anything too complex, just use one as the canvas and add others to represent the lines, circles, blocks and points you need. You have control of a views colour, width, height, radius and position so you can do quite a bit with them.

    I did a similar thing to create an on screen barcode, I only used black bars but you can colour the the views as well. (link to barcode info). It was good enough to allow the bar code to be scanned and read.

    I guess it just depends on how complex your requirement is and how much memory you would consume if you create many views for your graphic.

    Though it would be good to see a generic Titanium Canvas Object with a simple drawing API of some sort ( point, circle, line and rectangle) at some point.
