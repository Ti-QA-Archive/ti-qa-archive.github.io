Button attached to bottom of the screen is sliding up with the keyboard

I have a window.

In this window, I have a button. I want the button to be at the very bottom of the screen, so I have the button setup like this

var Button = Titanium.UI.createButton({
   bottom: 10,
   left: 10,
   right: 10
});

Then I added the button to the window with window.add(Button);

On the same screen, I have a textfield. When selecting the textfield, the soft keyboard pops up, and pushes the button on the bottom of the screen, back to the middle of the screen, above the keyboard.

Is there a way to prevent this behavior?

  • The following code has the elements you describe and does not exhibit the behavior you stated on iPhone sim. Try this:

    
Ti.UI.backgroundColor = '#333333';

var win = Ti.UI.createWindow();
var button = Ti.UI.createButton({
    title:'button',
    right:10,
    bottom:10,
    height:40,
    width:100
});
var textfield = Ti.UI.createTextField({
    height:30,
    width:'80%',
    top:50,
    borderStyle:Titanium.UI.INPUT_BORDERSTYLE_ROUNDED
})

win.add(textfield);
win.add(button);

win.open();

  • I figured out why it does this. If you orient your button from the bottom of the screen, say with bottom:10, your button will stay 10px above the bottom of the screen. On iOS, that means what you think, the bottom of the actual window, but on Android, it is measuring from the keyboard. So it will then be 10px above the keyboard when its displayed. Just orient your button from the top and you'll be all set.
