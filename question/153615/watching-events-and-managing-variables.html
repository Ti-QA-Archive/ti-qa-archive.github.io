Watching Events and Managing Variables

  • Application type: mobile
  • Titanium SDK: 3.0.0.GA
  • Platform & version: iOS 6.1
  • Device: iOS simulator
  • Host Operating System: OSX 10.8.3
  • Titanium Studio: CLI 3.1.0

  1. How can I watch an event bubbling in progress? I clicked a button that's inside a view that's inside another view, I would like to see how it travels through the layers and where it stops.

  2. How can I easily dump all of a view's current variables/attributes during runtime on click so I can see what's inside?

  3. Can you assign custom variables/attributes to View & ImageView just like you can with Button?

  • Accepted Answer

    1. Without you adding a click event to each view in the stack, there is no other way as far as I know.

    2. You can try a 

      console.log(JSON.stringify(my_view));

      Anyway you must be aware that the proxies are not exactly javascript objects and you might not get all the info you need.
      Read more about here: What is a Titanium “proxy” object?

    3. Of course
      ~~~

    var my_view = Ti.UI.createView({
    top:10,
    width:100,
    my_prop:'Whooray'
    });

    
Keep in mind that you will have to find some names for the custom attributes that are **very unlikely** to overwrite a native one (because they won't ;))

For example

    my_view.parent

    
So come up with names you know the Appc engineers couldn't use. Usually a weird prefix would do it, like

    my_view.my_parent
    myview.ab_parent
    ~~~

    Hope it helps.
