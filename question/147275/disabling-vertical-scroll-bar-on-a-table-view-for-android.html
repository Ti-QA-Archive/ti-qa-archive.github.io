Disabling vertical scroll bar on a table view for android

Is there a way to disable/hide the vertical scroll bar on a table view for android ?
I am using Titanium Studio, build: 3.0.1.201212181159.
Host OS is Apple OS X Mountain Lion.

  • As for as my experience is concerned there is no way to hide vertical scroll bar in table view. In scrollView you can do that. by setting the showVerticalScrollIndicator property as false but that property doesn't work in tableView actually this property doesn't exist in tableView. So, you can't hide it.
