  • This problem was resolved

    1 - download again titanium

    1.1 - install titanium

    2 - download again xcode

    2.1 - install xcode

    3 - open xcode/window/organizer/archives right click on your application and delete it

    4 - go to titanium/distribute click on package

    For me it worked!

  • I was getting a similar error using Titanium 3.4.1 with Xcode 6.2, this answer is what fixed it:

    Right click the archive in the Organizer, click Show in Finder, Right click the archive and Show Package Contents, and rename the dSYM folder to dSYMs. The archive should then validate in the Organizer.

    from: http://developer.appcelerator.com/question/179244/ipa-does-not-exist-while-trying-to-distribute-to-ios#answer-285236
