How to remove first model after fetching collection.

Hello,

How can I remove a model from a collection. This does not remove it (works fine to add it).
I think it´s this part: col.models (need to fetch the first model for the fetched collection).

favorites.fetch({  
        sql: {  
            where:{  
                parent: parent.id
            }
        },
        success : function(col) {

            if (col.models.length == 0) {

                var garden = Alloy.createModel('favorite', args.$model); 

                garden.save({}, {localOnly:true});

            } else {

                var favorite = favorites.get(col.models.id);

                favorites.remove(favorite);

            }
        },
        localOnly:true
    });

  • Try either

    var favorite = favorites.first();

    or

    var favorite = favorites.at(0);

    then

    favorites.remove(favorite);

  • Hi Dainel,

    you can also use destroy() method to remove model from the collection;
    as shown in below example:

    var favorite = favorites.at(0);
favorite.destroy();

    or

    var favorite = favorites.at(0).destroy();
