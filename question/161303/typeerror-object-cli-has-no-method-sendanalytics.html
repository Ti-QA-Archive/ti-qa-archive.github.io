  • The error is caused because you're using a mix of versions. You have Studio 3.2 with the 3.0.2 SDK, and while you didn't say, I'll guess you're using the 3.2 version of the CLI. You will need to use the 3.2.0.GA SDK. If you absolutely must use the 3.0.2 SDK, you'll need to downgrade Studio and the CLI to a matching version.
