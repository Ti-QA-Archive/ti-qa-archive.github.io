Facebook Connect Plugin showing user's password! - IOS

If you are using the social share plugin in iOS, try the following use case:

  1. bring up the fb connect screen
  2. type an email or phone number in the username field
  3. type an incorrect password in the password field
  4. click on 'login
  5. type anything into the password field
  6. you will notice that you can see the characters
