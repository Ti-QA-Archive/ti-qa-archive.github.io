[Beginner]Displaying a random image on click

Hi

I'm trying to display an image (calling it through a variable) once a label is clicked.
I've figured out how to display a image from a single source (i.e. the path), but now I try to randomly selectan image from an Array.

The issue I have is with the createImageView. I found that i can pass a variable as a parameter for it, but this doesn't show anything when clicked (no errors occuring).

I'm using Alloy.

My index.js is the following:

function doClick(e) {

Ti.UI.createImageView({
    width: 100,
    height: 100,
    top: 20,
    image:randomCard()

});

};


$.index.open();

function randomCard() {
    var card = cards[Math.floor(Math.random()*cards.length)];
    return card;
}

var cards = new Array();
cards[0] = '/images/1.png';
cards[1] = '/images/2.png';
cards[2] = '/images/3.png';

index.xml has the onClick="doClick" and haven't made changes to index.tss

Thanks in advance for your help.

    Hello, I'll layout a simple example.
    First place an ImageView and a Button tag in your index.xml (here I choose to layout the two window's children vertically):

    index.xml

    <Alloy>
    <Window layout="vertical">
            <ImageView id="img"/>
            <Button id="btn" onClick="changeImage"/>                
    </Window>
</Alloy>

    then in your index.js, when button is clicked, you could display a random image from the cards array by setting the ImageView.image property, as shown in the changeImage function:

    index.js

    var cards = [
    '/images/1.png',
    '/images/2.png',
    '/images/3.png'
];

function changeImage() {
    var card = cards[Math.floor(Math.random()*cards.length)];
    $.img.setImage(card);
}; 

$.index.open();

    hth.
