When will we get an overall stable Titanium release?

My app was built with sdk 1.3.2 several months ago. It works okay as it is. I wanted to use sdk 1.4 when it came out but table views were extremely slow for me. Unfortunately nobody really cared.

https://appcelerator.lighthouseapp.com/projects/32238/tickets/1431-tableview-extremely-slow-with-sdk-140#ticket-1431-11

Then version 1.5 came out and the table views seem to be working again for me. Unfortunately the events of the audio player seem to be broken. Looking at the ticket, again nobody really seems to care.

https://appcelerator.lighthouseapp.com/projects/32238-titanium-mobile/tickets/2135

I followed the advice, fetched the latest sources from github, compiled the sdk and installed it. Just to find out that now calling AudioPlayer.start() starts playback but freezes the complete app. I wonder if this is a new bug or something in the threading code changed or whatever.

Unfortunately I cannot even stay with sdk 1.3.2 since - at least for me - the app crashes right after the start when using sdk 1.3.2 with iOS SDK 4.2.

Bottom line: I have not only been stuck with an old version for several months now but I cannot work at all anymore.

Any advice?
