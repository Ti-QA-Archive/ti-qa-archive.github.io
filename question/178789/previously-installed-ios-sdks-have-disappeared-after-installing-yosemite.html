Previously installed iOS SDK's have disappeared after installing Yosemite.

Hi.

I have recently installed Yosemite, Xcode v6.1, and Titanium SDK v3.4.0.GA. The Titanium Studio build I’m using is 3.4.0.201409261227. Prior to doing this I’ve had older versions of Xcode and Titanium SDKs installed with multiple Run/Debug configurations for my project for different versions of iOS so I could for example test my app against iOS 6.1, and 7.1.

However, since performing the aforementioned upgrades, my Debug/Run configurations have all changed, in that for all my configurations the ‘iOS SDK’ and ‘Simulator Version’ settings have changed to ‘8.1’; and 8.1 is the only option in the dropdown, where previously it would’ve been ‘7.1’ for example. I presume this is because somehow Titanium Studio can no longer see the old versions of Xcode I previously installed? Does anybody know how I can get it see my older Xcode versions?

  • Xcode 6.x doesn't officially support anything other than SDK 8.x

    If you have older versions of Xcode installed, you can switch between them using xcode-select

    Select the Active Xcode Version
