Android CloudPush Implementation

Hello all,

I have been trying to follow the steps described in this tutorial

http://www.titaniumtutorial.com/2012/06/appcelerator-cloud-push-notification-in.html

to implement Push notifications in an Android app. The tutorial is nice but there are steps that appear to be missing. From what I understand the steps involved are

  1. Register with ACS
  2. Add an application to ACS
  3. In the app, retrieve a device token
  4. Log in to ACS. Here I hit the first issue - the login code requires a login and password. I tried using my ACS login but that did not work. So I tried to create a user from the ACS control panel but that did not work either
  5. Subscribe for push notifications - this requires the device token obtained earlier and a "channel". Just where do I get this? Is there some way to define the channel in ACS?

Finally, I want to send out push messages from a PHP script. Does ACS have an API I can access from the script? I am new to ACS and Android Push but nevertheless it seems to me that the Titnium documentation on the subject, and on Cloud/Push, is somewhat lacking.
