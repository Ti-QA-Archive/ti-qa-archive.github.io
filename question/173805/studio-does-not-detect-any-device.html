Studio does not detect any device

Good day,

I am having a problem with the last update of Titanium Studio, since then, it does not recognize any devices or emulators attached to run/debud even when "adb devices" shows them.

in "Run Configuration" there isn't any device detected, only emulators from the AVD.

Before the update it was fine. Any device was detected and running without problem.

There is a way to fix Studio? Thanks

Application type: mobile Titanium
SDK: 3.2.3GA
Titanium Studio, build: 3.2.3
Platform & version: Android 4.0
Device: Amazon Kindle Fire, Genymotion emulator
Host Operating System: Windows

  • Got the problem fixed. I tried everything but what worked was:

    Close Ti Studio,

    delete C:/Users/MyName/.titanium/config.json

    then restart Ti Studio.

    To check to see if your system has any other issues, in the Ti Studio Terminal execute these commands. Very helpful for trouble shooting problems.

    Titanium info

    Titanium setup

    Ti -v

    — commented May 20th 2014 by Michael Lake

  • Got the problem fixed. The method Michael described didn't work for me. What I did is this.
    Close the Genymotion virtual device. Run the program against Google emulator. Close the emulator. Start the Genymotion virtual device, now you can see the IP of the device besides the run button and it works. It is obvious that something somewhere didn't update correctly despite of Titanium IDE restarts and device refreshs.
