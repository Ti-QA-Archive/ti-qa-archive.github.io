    Hi

    The zoomScale is a factor value so it is the different between the parent and the child.

    So if you have a child view (or image) that has a width or 640x640 and the parent view is 320x320 then the initial zoomFactor would be 0.5 to show the whole image scaled to fit and 1.0 to show the image at the correct size where you would only see a quarter of the whole image.

    If the parent view is 320x320 and your have three child views; 320x320, 640x640 and 960x960 then the zoomFactors (or at least scaled initial ones) would be; 1.0, 0.5, 0.3333.

    If you have aspect ratios that differ between the parent and the child then you can choose to scale down to the greater or the lesser difference, so you either have empty bars either side (top/bottom or left/right) or have overlapping clips.

    So choose the dimensions you want to scale to suit your needs.
