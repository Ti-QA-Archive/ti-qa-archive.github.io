Implementing x-callback-url in Titanium, sample code provided

I've working on protocol, called x-callback-url, intended to help iOS developers standardize formatting and usage of URL scheme-based interactions between apps.

Details on the protocol are provided here:

http://x-callback-url.com

Also, a great overview was posted on The Changelog:

http://thechangelog.com/post/3426369462/xcallbackurl-iphone-app-for-testing-the-x-callback-url-i

Long story short, I'm posting here because I've now I've got up an open source sample app demonstrating the use of x-callback-url in Titanium up on GitHub:

http://x-callback-url.com/2011/04/tixcallbackurl-implementing-x-callback-url-in-appcelerator-titanium/

I hope iOS devs using Titanium will take a look and consider implementing x-callback-url in their app.

Thanks!
