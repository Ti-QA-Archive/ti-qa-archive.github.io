Android apk not installed

Hi!

I've got a probem. If I pack my Android app into signed (the same problem with unsigned) apk and trying to distribute it throught the direct link on Android devices 4.0-4.2 when I install I've got an error:
"Application not installed". Sometimes I've got an error "Not enough space" even I've got more the 500mb.
Tryied on Samsung S2 4.0.4, Samsung Note (mb not not but tablet) with 4.0.4, on Samsung S 4.2.2 ond on htc 601 (4.2).

On 4.3+ even on 5.0.1 everyting is fine.

Of course I've allowed installations from other then Google Play Sources.

I've tryed different SDK versions from 4.0.3 to 4.4.3 both with Google APIs and without. I've used Titanium SDK 3.4.0, 3.4.1, 3.5.0 all GA.
Titanium strudio version is 3.4.1.201410281727

  • The problem was with jarsigner. To solve the problem I've used command line jarsigner:

    jarsigner -verbose -keystore pathto.keystore path_to.apk keystore_alias
