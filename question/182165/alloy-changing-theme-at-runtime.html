Alloy changing theme at runtime

Hi,

Apologies in advance if this question is duplicate. While I found a few related questions, I couldn't yet find a definitive answer.

The question is how to change the (style) theme at runtime? I have multiple themes under app/themes folder, say

app/theme folder

??? themes
?   ??? blue
?   ?   ??? assets
?   ?   ?   ??? button_blue.png
?   ?   ??? styles
?   ?       ??? index.tss
?   ??? dark
?   ?   ??? assets
?   ?   ?   ??? button_dark.png
?   ?   ??? styles
?   ?       ??? index.tss

and config.json is something like:

...
    "global": {
       "theme": "blue"
    }
...

At runtime, I wanted the app to switch to use/apply 'dark' theme. As you can see, this is not applying dynamic styles to particular element, but apply several styles to the whole app. Is this possible at all, or are we limited to use createStyle and applyProperties and apply them to each element that requires new styling?

I see that a few others have had similar needs and I can imagine this to be somewhat a common need.

I'm using 3.5.1.GA Ti SDK. I'm trying to do this for iOS now, but will eventually need this for Android too.

Appreciate any pointers.
