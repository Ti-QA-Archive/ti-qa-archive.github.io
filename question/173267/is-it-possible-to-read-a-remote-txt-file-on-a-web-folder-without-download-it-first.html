    • The example code for the httpClient is a good example of what you need/.

       var url = "http://www.appcelerator.com";
 var client = Ti.Network.createHTTPClient({
     // function called when the response data is available
     onload : function(e) {
         Ti.API.info("Received text: " + this.responseText);
         alert('success');
     },
     // function called when an error occurs, including a timeout
     onerror : function(e) {
         Ti.API.debug(e.error);
         alert('error');
     },
     timeout : 5000  // in milliseconds
 });
 // Prepare the connection.
 client.open("GET", url);
 // Send the request.
 client.send();

      This line shows you what you received;

      Ti.API.info("Received text: " + this.responseText);

      You provide your path to the specific document by changing this url;

      var url = "http://www.appcelerator.com";

      Be aware that the httpClient call is asynchronous, this means that your text will appear after the request was made, so as the above code shows you handle the response in the onload event - not after the send line.
