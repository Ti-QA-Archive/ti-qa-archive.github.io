Is it safe to use files to store app data

Hi all

I am using files to store parts of my application data (using Ti.Filesystem.applicationDataDirectory).

But I read somewhere that Apple changed some settings that will delete "temporary" app data if it takes up too much space (or something like that).

Are files stored (written to) in applcationDataDirectory safe from being deleted by iOS?

Regards

  • Files in Ti.Filesystem.applicationDataDirectory won't be deleted, but they will be backed up to iCloud if the user has iCloud enabled. If there are too many files stored there that Apple thinks could be 'recreated' (i.e. downloaded files) if might not approve your app. But they won't be deleted.
