Facebook AppID not copied to Info.plist

Hey there,

I've defined my Facebook AppID as follows in tiapp.xml:

<property name="ti.facebook.appid">APPID</property>

However, the property is never copied over to info.plist, causing Facebook auth to fail. Any reason for this? Is there a better way to get the ID into the final build? If I try to manually insert the property into Info.plist, it's overwritten when I build the app.

BTW: Ti SDK 3.2.3

Justin Davis

    • Ah, hang on… Instead of your custom info.pList you can now put those in the plist section. Then delete the custom info.plist, clean the project and try again.

      eg

      <ios>
        <plist>
            <dict>
                <key>UIAppFonts</key>
                <array>
                    <string>fonts/your-font-file.otf</string>
                </array>
            </dict>
        </plist>
    </ios>
