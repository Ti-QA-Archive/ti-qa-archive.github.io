  • Hi,
    In the last line of MainWindow(), you return instance - it's a Titanium's Window object. Hence your extension on MainWindow will not work.

    My suggestion - remove that line:

    function MainWindow(viewToShow) {
    ...
    // return instance;
}

    Next, you need to declare open() method for the MainWindow, or else below code in app.js would raise error

    /// app.js
mainWindow.open();

    Extend MainWindow:

    function MainWindow(viewToShow) {
    ...

    this.open = function() {
        instance.open();
    }

    //return instance;
}

    Hope that would help.
