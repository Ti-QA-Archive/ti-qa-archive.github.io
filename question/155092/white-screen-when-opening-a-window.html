    • OK so I know see what you are referring to. The slight delay between asking for the new window and the new window displaying.

      The reason for this is because you are still using the now out of favour multiple context app method which is actioned by the use of the url property. Also you are not caching the or pre-loading the new window prior to displaying it.

      Your app will not move to the next window until the next window context has been completed, so the more you put on the second window the longer the delay.

      You should consider using CommonJS or delayed content loading on the second windows.
