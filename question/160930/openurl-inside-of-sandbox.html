  • Hi, EB,

    the can only use API statements. The filesystem API can only access files and directories in document (titanium=applicationdata) and in resources folder. Every app has it own app document folder. That's why an app cannot open files of other app.

    Exception: calendar, contacts and photos are objects, which can share between apps.

    If you call openURL, then the system decides by string ended with ':',which app is starting (schema). This app can only open his datas.

    Other exception: 'open in' (safari, dropbox). For this functionality you need a module. (@EB last year we found a link for objectivC developer)

    Rainer
