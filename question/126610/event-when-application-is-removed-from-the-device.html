Event when application is removed from the device?

Hi,
I'm writing an app for both iPhone and Android. My app requires a users registration. I was wondering if there is a way that I can know when a user removes the application (both Android and iPhone) from the device so I can unregister them?

Thanks

    There are no built in events to know when someone uninstalls your app. On a side note though, it seems like it would be better practice to simply mark an account as inactive after a certain amount of time of inactivity rather than unregistering them when they remove an app. Not everyone removes an app for permanent reasons. Maybe it was an accident, or maybe they are troubleshooting, in which case when they reinstall the app from iTunes, they will have to re-register again.
