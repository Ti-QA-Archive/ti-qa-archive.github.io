  • Android web view supports native plugins such as Flash Player. To use the Flash Player plugin, hardware acceleration must be enabled for your application. To enable hardware acceleration, add the tool-api-level and manifest elements shown below inside the android element in your tiapp.xml file.

    <android xmlns:android="http://schemas.android.com/apk/res/android">
    <tool-api-level>11</tool-api-level>
    <manifest>
        <application android:hardwareAccelerated="true"/>
    </manifest>
</android>

    also set WebView.pluginState as Titanium.UI.Android.WEBVIEW_PLUGINS_ON or Titanium.UI.Android.WEBVIEW_PLUGINS_ON_DEMAND.
    This property(pluginState) only works on Android devices at API Level 8 or greater.
    See Android documentation for WebSettings.PluginState.
