    I'm going to guess that the problem is your Node JS version. Are you running Node 0.12? (Run node -v) I'm afraid we've found some compatibility issues with 0.12 and we'll be releasing a fix for that shortly (it's in QE testing now). In the meantime, you can use a node version manager, such as https://www.npmjs.com/package/n to install multiple versions – use a 0.10.x branch until we release our fix.
