    Hi

    Try this;

    • Select your project on the left side - project explorer
    • Click the run icon (green circle play button, small yellow folder)
    • Click Run configurations…
    • The right side should show iOS SDK
    • This is probably blank, change this to 6.0
    • Apply
    • Run

    Report back either way.
