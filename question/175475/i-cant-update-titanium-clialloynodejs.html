  • Hi isola Abeeb,

    To update the CLI tool, run the npm install command. On Mac OS X and Linux, you may need to prefix this commands with sudo.

    npm install -g titanium

    And For updating Alloy run this command in the terminal.

    npm install -g alloy

    When you will update CLI, node.js will be automatically updated.

    If you have any concern about this query then let me know

    Thanks.
