  • According to the compatibility matrix, Titanium 2.0 should work with iOS 4.0, which is pretty much the last iOS version that worked (I use that term in the broadest possible sense) with the iPhone 3G.

    Now, if that doesn't work, you can find builds of all the older Titanium versions here. I've created an app that worked perfectly on a 3G with Titanium 1.6 back in the days, so if I were you I'd start from there and work your way up until you start getting problems.
