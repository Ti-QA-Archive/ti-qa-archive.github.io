Picker does not work with iOS 7 SDK?

I cannot make my old picker control run under iOS7. Here's the code:

var win = Ti.UI.createWindow({ backgroundColor : "white" });
var picker = Ti.UI.createPicker();

var data = [];
data.push(Ti.UI.createPickerRow({ title : "AAA", }));
data.push(Ti.UI.createPickerRow({ title : "BBB", }));
data.push(Ti.UI.createPickerRow({ title : "CCC", }));
data.push(Ti.UI.createPickerRow({ title : "DDD", }));
data.push(Ti.UI.createPickerRow({ title : "EEE", }));
picker.add(data);

var view = Ti.UI.createView({width:300, height:200});
win.add(view);
view.add(picker);
win.open();

Works fine with SDK 6.1 but renders an error on SDK 7: "[ERROR] Script Error = * -[__NSArrayM objectAtIndex:]: index 0 beyond bounds for empty array".

Any ideas?
