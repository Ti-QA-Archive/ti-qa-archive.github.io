    • Hi
      I resolved this problem , Titanium module project now working fine in my application. Reason for My project error is wrongly I set the path in windows environment path settings

      Before my environment path:

      path=%ANDROID_SDK%tools and %ANDROID_SDK%platform-tools

      In that above path settings I miss "\" symbol ,After I have added "\" symbol to my path settings now its working fine .

      After my environment path:

      path=%ANDROID_SDK%\tools and %ANDROID_SDK%\platform-tools
