  • Hi

    The documentation is pretty clear on what it does;

    Use Titanium.Network.createCookie to create a new Cookie object.

    Given it is in the Network API family, we can make a guess it relates to httpClient. But the documentation gives us some comfort in our guess;

    Cookie object used to manage the system cookie store and HTTP client cookie store.

    As it notes the support is available from Ti 3.3, a quick check of the release notes;
