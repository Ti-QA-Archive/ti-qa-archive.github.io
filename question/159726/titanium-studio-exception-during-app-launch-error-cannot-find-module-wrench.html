    To run all Appcelerator components, you must have Node.js 0.10.13 or later. Can you check what is your Node version: 

    $ node -v
v0.10.20

    If this is correctly install at your local system I'd suggest to re-install Node and start over. Furthermore, if you can replicate this error consistently you may want to log a report at jira.appcelerator.org so the team can investigate more about this problem.
