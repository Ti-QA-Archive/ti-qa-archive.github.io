Interactive Text

I am looking at creating an application in which there is a sentence or two on a page and if an individual clicks on one of the words in that sentence the word they clicked on is read back to them using a sound byte. For example: A sentence on the screen sais, "Hello my name is Scott". If I click or touch the word "Scott" a sound byte plays back saying "Scott". If I click on the word "my" a sound byte plays back saying "my". I have done in the past using flash, is this possible using Appcelerator?

  • Hi

    Very easy, provided you are supplying the audio.

    • create your sentence as a string
    • split the string on spaces and punctuation using JavaScript split command
    • create a parent view with layout horizontal
    • loop the array
    • add a label for each word with some right padding to the parent view
    • add an event to each label to detect click
    • that event calls a function with the word as a parameter
    • the function plays the audio file for that word
    • check the docs under media for suitable code examples to play back audio

    Done.

    The layout of horizontal allows the individual words to flow as he would in a word processor/printed letter, splitting them gives the unique click event.

    It is also possible with a click map that determines the word based on x/y position but would require the fixed display of the sentences (not good for multiple devices) and significantly less effort working out those coordinates..
