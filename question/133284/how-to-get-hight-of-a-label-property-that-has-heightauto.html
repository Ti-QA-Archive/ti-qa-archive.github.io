How to get Hight of a label property that has height:'auto'

What i'm trying to do is this:
I got a label Title with auto height. Under that I want to place a other label where the property bottom is:
bottom:label.bottom.tostring()
I also Tried label.getbottom
but nothing seems to work it always says '[Object label]

Is it possible to retrieve the bottom?

  • Accepted Answer

    try this one

    setTimeout(function(e){
    NewsDetailAanmaak.top = NewsDetailTitle.top + NewsDetailTitle.size.height + 5;
},500);

    or

    setTimeout(function(e){
    NewsDetailAanmaak.top = NewsDetailTitle.top + NewsDetailTitle.toImage().height + 5;
},500);

  • Wrap your auto height labels into a view with layout: &#39;vertical&#39; would help:

    var win = Ti.UI.createWindow({backgroundColor: 'white'});

var view = Ti.UI.createView({
    layout: 'vertical',
});
var label1 = Ti.UI.createLabel({
    text: 'Label with auto height',
    height: 'auto'
});
var label2 = Ti.UI.createLabel({
    text: 'Another Label with auto height',
    height: 'auto'
});

view.add(label1);
view.add(label2);
win.add(view);
win.open();
