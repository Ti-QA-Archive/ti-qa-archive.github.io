Titanium studio with Titanium SDK 3.5.1, Xcode 7 and OS X Yosemite (10.10.5) can't able to see iOS simulators ann iOS device

Hi All,

As iOS9 and Xcode 7 released I updated my machine and found in my titanium studio I am unable to see iOS simulators and iOS devices. As I am using Titanium Studio with titanium sdks 3.5.1. Do I need to update my Titanium SDK in my Titanium studio or Is it necessary to move to Accelerator studio for Xcode 7 and iOS9 support.

  • Yes, you do need to update titanium sdk - only sdk 5 will allow you to compile and build for iOS 9! But do not despair, I was having this exact problem. I tried reinstalling node, appcelerator studio, everything i could think of - it didn't help at all. But then I downloaded the latest build of sdk 5 and installed it via the CLI. This worked perfectly! Go to: http://builds.appcelerator.com.s3.amazonaws.com/index.html#master. Important note - I could NOT install it using Studio. It had to be done from the command prompt to work.

  • Same here. But you can work around this by going to Run configurations -> run. Afterwards you can repeat this by doing run -> run last launched.

    The same works for packaging.

    Another thing: Compilation failed for me, because the default xcode project template has bit code compilation on, but my modules don't support this. Editing the default project fixed this for me. (~/Library/Application Support/Titanium/mobilesdk/osx/4.0.0.GA/iphone/iphone/Titanium.xcodeproj)

    Good luck!
