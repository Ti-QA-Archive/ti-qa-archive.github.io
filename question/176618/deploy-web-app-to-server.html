  • Hi D,

    Titanium Studio provides the integrated local browser to test your web app and helps to package your mobile app for final publishing.

    Follow this link for 'deploying web app' help.

    I assume you have a purchased hosting server with valid credentials. Then install any FTP client app(Filezilla etc.) to connect to your domain and upload the package. Thats it. You are all set.
