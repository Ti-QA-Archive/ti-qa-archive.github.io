  • Hi

    The solution to this is nice and easy; the addAttachment method only accepts a file or a blob.. Your attachment is a remote url.

    You have to download the file locally and then attach it. The example code in the EmailDialog api explains how you attach a local file. You will need to use the HTTPClient api to download the file first of course - example in this documentation as well.
