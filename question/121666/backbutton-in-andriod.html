Backbutton in Andriod

I make application of many windows.
I want to use back button of Android to return to previous page.
When I clicked the back button the application closed.

  • Accepted Answer

    You need to add an event handler to the window.

    win.addEventListener('android:back', function (e) {
  Ti.API.Info("Pressing Back Button");
  win.close();  
});

  • Hi Hema.

    Most probably you are using lightweight windows instead of heavyweight (google for difference between them). Back button on Android closes only heavyweight windows as they are considered as new context. Try to create heavyweight window instead of lightweight, something like:

    var mywin = Ti.UI.createWindow({
    modal: true //this will force heavyweight window creation
});
