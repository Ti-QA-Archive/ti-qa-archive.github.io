  • Do you mean something like this:
    for example increasing top value by 10px

    for(var i in views)
{
    views[i].top += 10
}

    or if you know the index (for example 3) of view in array, then simply views[3].top += 10

    if you need to determine if thats a specific view, for example nbview from your code snippet equals 3, then something like this would do the trick

    for(var i in views)
{
    if(views[i].nbview == 3)
    {
        views[i].top += 10
    }
}

    Hope this helps
