iOS Back button

Is there any way to make a button use the Back shape (i.e. looks like an arrow on the left side of the button, but the right side is like normal button) ? I can't seem to figure out any way to do this - except for creating the button as graphics myself…

    Well, if you are using a NavigationGroup or a TabGroup, simply setting the backButtonTitle property will display the button with the look you want.

    If you are in a different scenario, you will have to provide your own button and assign it using the leftNavButton property.
