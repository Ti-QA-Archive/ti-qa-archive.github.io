  • I found that if the width property is 'auto' the message display correctly, but if I set to 200, for example, the message doesn't display.
    Update to SDK 1.8.2

    var actInd = Titanium.UI.createActivityIndicator({
    height: 50,
    width: 200,
    color: '#fff',
    backgroundColor: '#000',
    opacity: 0.75,
    borderRadius: 5,
    borderColor: '#fff',
    message:' Loading...',
    style:Titanium.UI.iPhone.ActivityIndicatorStyle.PLAIN
});
