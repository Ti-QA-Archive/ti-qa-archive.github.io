    • You would hate?! I thought you had some technical issues with modal windows … okay, if you don't like them, it doesn't mean that using them is a bad practice. Modal window are perfectly normal to use and I don't see a reason why someone wouldn't use them. Modal windows are good choice for Android applicaton because hardware back button works well with them (last window is always closed first, so there is no mess when one has that algorithm in the mind). On the other hand "NavigationGroup" is iOS only (although, it seems to me that they are functionally similar to modal window), so if you are targeting Android also, you will have to write Android-specific code …

    • in fact I said that it was a "bad practise", not that you have any particular issues. Yeah, it's iOS only, I just noticed it today. The cool thing about NavigationGroup is that the windows are all connected and you can close one for all.
