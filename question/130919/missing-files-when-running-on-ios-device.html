Missing files when running on iOS device

My project is running perfectly on iPad. But when I run on iPhone, it cannot find any of the files I specified. I have a function that checks the OS name. If it's an iPad, it imports a set of files. If it's an iPhone, it loads a different set of files. I know that my IF statement is working, because it's running code inside, but it refuses to load my .js files. I've tried everything I can think of. I've used Ti.Filesystem.resourcesDirectory, includeAbsolute, etc:

var ipad = false;
var iphone = false;
var iOSCheck = function(){
    if(Titanium.Platform.osname == 'ipad'){
        ipad = true;
        Titanium.include('games/views/ipad/options.js');
    } else if(Titanium.Platform.osname == 'iphone'){
        iphone = true;
        Titanium.include('games/views/iphone/options.js');

    } 
    if(iphone || ipad){
        Titanium.UI.iPhone.statusBarStyle = Titanium.UI.iPhone.StatusBar.OPAQUE_BLACK;
    }
}

All file paths are absolutely correct, I've checked my upper/lower casing, I've cleaned the project 1000 times, I'm going crazy….
