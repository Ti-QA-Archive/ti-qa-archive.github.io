Titanium CLI Use

Does anyone know if it possible to access and run the Ti CLI commands within a node.js script?

Something akin to 

var cli = require('titanium');
cli.run('info');

so that it issues the command 'ti info' ( i realise there is no run command, just used for example)

I have tried with the exec('ti info') command which does not work and I have also looked at using require as in the example above, but so far have had no success.

Is what I an attempting possible, and if so has anyone else got this to work?

What I am trying to do is run the CLI from within a node script on my local machine.

  • Looks like I found an answer, at least using the CLI with require('child_process')exec.

    Seems like OSX does not always read the .bashrc when using child_process (for GUI apps anyway) or so my research finds. Anyway I needed to add /usr/local/bin to the process PATH, i.e.:

    if(process.platform==='darwin'){
    process.env.PATH= process.env.PATH + ":/usr/local/bin"
 }

    et voilà! - I can now run the CLI.
