Switches always animating to on state (iOS)

In my app I have a table of switches that allow the user to opt in or out of various tags (which are dynamic and sync with our backend). Whenever the view is rendered, though, the switches all start at off and instantly animate to on. Is it possible to have these render to their final state without any animation? The function that builds the table looks like:

buildTable = function(tags) {
    $.tags.visible = false;
    Ti.API.info(tags);
    var count = tags.length,
        i = 0,
        rows = [];
    Ti.API.info(tags);
    for (var tag in tags) {
        if (!tags.hasOwnProperty(tag) || tags[tag].hidden) continue;
        var    currentTag = tags[tag], 
            row = Alloy.createController('tagRow', {
                id: currentTag.id,
                name: currentTag.name,
                value: currentTag.value,
                tag: tag
            }).getView();
        $.tagSection.add(row);
    };
    $.tags.visible = true;
}
