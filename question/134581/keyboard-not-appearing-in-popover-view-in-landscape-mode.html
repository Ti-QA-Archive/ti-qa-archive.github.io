Also I noticed whenever keyboard is appearing in this case, the topover view is truncated on area which is overlapped with keyboard. Some head burning stuff..

The keyboard works well with Windows in landscape. I have issue only with Popover view and only in landscape mode (Portrait is fine)..

  • Hi Virendra.

    Try this hack

    // substitute TEXTFIELD with your own TextField variable
TEXTFIELD.addEventListener('focus', function(e) {
    var setFocus = function(){
        TEXTFIELD.focus();
        TEXTFIELD.removeEventListener(setFocus);
    }
    setTimeout(setFocus,180);
});

    It works for me.

    Cheers

    PS. I also tried resizing the popover as mine was quite large and it could have been a battle of screen estate but to no avail. Perhaps this is fixed in 2.0.0 but running a continues build totally ruined the app for me. Your mileage may vary of course.
