No events fired in SplitView

I'm using a SplitView with a table in the sidebar, and a detail view in the main area. However, Titanium doesn't seem to be emitting any events.

The table view doesn't register clicks, or event scroll when it's dragged.

I've reduced the code down to the very basics for testing and still haven't been able to capture a simple label click event:

//SIDEBAR
var listWin = Ti.UI.createWindow();
listWin.title = 'List';

var label = Ti.UI.createLabel({text: 'TEST', color: '#fff'});
listWin.add(label);
label.addEventListener('click', function(event) {
    alert('click');
});


//DETAIL
var detailWin = Ti.UI.createWindow();
detailWin.title = 'Details';


//SPLIT WINDOW
var listNavGroup = Ti.UI.iPhone.createNavigationGroup({
    window: listWin
});

var detailNavGroup = Ti.UI.iPhone.createNavigationGroup({
    window: detailWin
});

var splitWin = Ti.UI.iPad.createSplitWindow({
    masterView: listNavGroup,
    detailView: detailNavGroup
});

splitWin.open();

Anyone had any luck with this? I get the impression from other questions that the SplitView implementation is quite flakey and am considering just creating my own version with a couple of regular views in a window… Or is it worth sticking with the SplitView?
