  • Hi Mubarak Al-Mutawa,

    Create Ti.App.fireEvent for that and onclick event put your code inside that click event see below code as a reference. Best luck

    <html>
    <head>
        <script>
            Ti.App.addEventListener("app:fromTitanium", function(e) {
                alert(e.message);
            });
        </script>
    </head>
    <body>
        <button onclick="Ti.App.fireEvent('app:fromWebView', { message: 'event fired from WebView, handled in Titanium' });">fromWebView</button>
    </body>
</html>
