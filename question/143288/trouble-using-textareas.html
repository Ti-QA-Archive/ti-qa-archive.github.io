    Hi Eric,

    There might be a chance that you missed something in the code. You can refer below code to check if everything is right on place, though i used a single TextArea but similarly you can add other textareas as well:

    // this sets the background color of the master UIView (when there are no windows/tab groups on it)
Titanium.UI.setBackgroundColor('#000');

var win1 = Titanium.UI.createWindow({
    title : 'Tab 1',
    backgroundColor : '#fff'
});

var row = Ti.UI.createTableViewRow({
    className : 'forumEvent', // used to improve table performance
    backgroundColor: 'green',
    height : 110
});

var textArea = Ti.UI.createTextArea({
    width: 100,
    height: 50,
    hintText: 'Enter text here'
}); 
row.add(textArea); 

var tableView = Ti.UI.createTableView({
  backgroundColor:'white',
});

tableView.setData([row]);

win1.add(tableView);
win1.open();

    I hope this helps.
