  • Hi

    You have not said which Ti SDK you are using. If it is Ti SDK 3.3 (released a few days back) then there have been a number of ActionBar related changes and it is worth refreshing your knowledge with the latest documentation updates;

    • ActionBa
    • Android.Menu

    BTW it is always a good idea when de-bugging to remove all variables that are not important to the code, so these can be discounted from any issues;

    var menuItem = menu.add({
    title : "test",
    showAsAction : Ti.Android.SHOW_AS_ACTION_ALWAYS
});

    This is a better test as args.data may not be returning a string or even be defined. This way you know the title will have a string not matter what else is happening.
