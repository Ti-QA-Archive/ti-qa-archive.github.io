Closing a tabGroup opened from a dashboard

I have a dashboard of 9 buttons from each different tabGroups are called (opened). Here's the flow: my app.js calls ui.js (main interface with the dashboard icons). Clicking a news icon (for example) opens a news tabGroup (newsTabGroup.js) with its first tab being a bible RSS feed. I attached a button to the window of the tab to enable closing the tabGroup and returning to the dashboard. But whenever I click on that menu button, I get an error "cannot close the root window". Not sure why this is a root window since the tabGroup was opened from the dashboard (ui.js).

I have tried all means to return to the dashboard from an opened tabGroup but no luck. I tired declaring a public variable that is assigned to the tabGroup and hoping I can close it in the child window. Still no luck.

function BIBLEWindow(title){

// Builds the RSS reader window using Charisma RSS News feed channel  
    require('includes/getRSSData').fetchRSSData('http://charismamag.com/index.php/news?format=feed&type=rss');

    var bibleWindow = Titanium.UI.createWindow({  
        title: title,
        backgroundColor:'#13386c',
        barColor:'#9b05ad'
    });

    var b = Titanium.UI.createButton({title:'Menu'});
    bibleWindow.leftNavButton = b;

    // create the tableview
    var tvBible = Titanium.UI.createTableView();
    bibleWindow.add(tvBible);

    // helper function for adding rows, we'll need this later
    var addTbRows = function(tbData){
        var tbRows = []; // an array to hold our rows
        for(var i=0; i<tbData.length; i++) {
            tbRows.push(Ti.UI.createTableViewRow({
                title: tbData[i].postTitle,
                hasChild:true,
                link: tbData[i].postLink,
            }));
        }
        tvBible.setData(tbRows);            
    };

    // add click event listener to open blog post when row is tapped

    tvBible.addEventListener('click',function(e) {
        var ArticleWindow = require('includes/ArticleWindow');
        var articlewindow = new ArticleWindow(e.rowData);
        articlewindow.open({modal:true});
    });

    // set up an event listener to populate data when it's available from the network and remove same when window closes
    //to conserve memory

    var bibleEventHandler = function(e) {
            // Set the window's title
        bibleWindow.title = e.blogTitle;
        // Add the table rows
        addTbRows(e.blogPosts);
    };


    Ti.App.addEventListener('net:bibleRssDataReturned', bibleEventHandler);

    b.addEventListener('click', function()
    {

    Ti.App.removeEventListener('net:rssDataReturned', bibleEventHandler);
    bibleWindow.close();

    });
    return bibleWindow;

};

module.exports = BIBLEWindow;

  • Hi

    You are trying to close the bibleWindow but from the rest of your code it looks like bibleWindow is the very FIRST window you actually open. If this is the case then bibleWindow is the root window, which is why you are struggling to close it.

    Make sure it is not the first window - maybe that should be the one for the dashboard in this case.

    Re-arrange the order of window opening and see what happens.
