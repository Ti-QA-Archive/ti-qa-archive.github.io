  • Bad method is to create custom Info.plist, by copying the Info.plist from the build/iphone after the first build, remove all automatically generated entries and add just CFBundleShortVersionString. Apparently, its content shuld be rewritten by hand each time the version changes, and if forgotten, it could fail verification/submitting. Need a solution to do it in some script at build time
