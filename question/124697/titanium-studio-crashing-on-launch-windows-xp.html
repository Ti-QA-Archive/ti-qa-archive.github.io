Titanium Studio Crashing On Launch (Windows XP)

I've just installed Titanium Studio and I have not been able to get it to launch at all. The splash screen comes up, then goes away and I get the message: "An error has occurred. See the log file."

I've run through all the documentation and I really have no idea what the issue is. I'm running Windows XP Pro SP3 and have both JDK 6.27 and 7 installed–I've changed %path% to try launching with each by default. I don't understand the log file at all–other than that some assertNotNulls are failing–which leads me to think some files are missing, but I don't know what they would be. Any help is appreciated.

  • Appcelerator support provided a solution to this problem. I've tested it and it works:

    To fix it, you need
    to delete the file:

    C:\Documents and
    Settings/<user>/.eclipse/org.eclipse.equinox.security/secure_storage

    After deletion, restart studio and it should work.
