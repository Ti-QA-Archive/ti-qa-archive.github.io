VideoPlayer gives ClassCastException in android

I create a movie upon the window 'open' event 

activeMovie = Titanium.Media.createVideoPlayer({
    contentURL:'../movies/' + moviefilename,
    backgroundColor:'#111',
    movieControlMode:Titanium.Media.VIDEO_CONTROL_DEFAULT,
    scalingMode:Titanium.Media.VIDEO_SCALING_MODE_FILL
});

When I run it either on the simulator or on an android device (APIs 2.1 or 2.2), I receive this popup "Runtime Error"

Message:
Wrapped java.lang.ClassCastException: 
ti.modules.titanium.media.VideoPlayerProxy (app://path/to/file.js)

I keep butting my head on this problem, meanwhile the kitchensink example runs beautifully, same parameters, same simulator, same video file…

I am using Titanium SDK 1.6.0. Any suggestions?
