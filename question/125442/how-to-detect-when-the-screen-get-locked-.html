  • Android:
    It is possible by doing the following in your app.js:

    Ti.Android.currentActivity.addEventListener("stop", function() {
    // your code
});

    But this event is also fired when you press the home button in Android. I haven't looked out for fixing that.

    For iOS you can use the pause and resume events.
