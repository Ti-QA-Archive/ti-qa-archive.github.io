  • Vishnu

    Have you read the Preparing for Android development chapter of the Getting Started guide? You don't need to "move or copy" adb.exe - you can simply create a symbolic link for it in the android tools directory. For windows, you can do so as follows:

    cd \path\to\android-sdk\tools
mklink adb.exe ..\platform-tools\adb.exe

    By the way, Titanium does not support Android 2.3 yet, so you are likely to cause yourself problems if you use it.

    Cheers
