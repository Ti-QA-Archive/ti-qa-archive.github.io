  • Hello Levani,

    May by your update is getting broken. Try manually update your CLI, Node.ACS, and Alloy through terminal.

    sudo npm install -g alloy@1.5.1            To install alloy
sudo npm install -g titanium@3.4.1         To install Titanium CLI
sudo npm install -g acs                    To install Node.ACS

    Restart Titanium studio. Hope this helps. Thanks.

  • Hi, we are tracking this on the npm issue tracker here: https://github.com/npm/npm/issues/6776

    The ENAMETOOLONG seems to be caused by a very long value for the prefix value in &#x2F;usr&#x2F;local&#x2F;etc&#x2F;npmrc. If you have that file, open it with a text editor and check to see if you have something like

    prefix='npm info it worked if it ends with ok......'

    It is not necessary to delete the whole file; just removing that one line will allow npm to run

    • Here comes my solution: Just delete the /usr/local/etc/npmrc file!

      rm /usr/local/etc/npmrc

      After this, upgrading worked perfectly! Update windows won't appear anymore.

      This was my source: https://github.com/npm/npm/issues/6776
