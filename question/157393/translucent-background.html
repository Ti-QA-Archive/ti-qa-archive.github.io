Translucent background

Hi All,

I really want to know how to do the fancy iOS 7 translucent background with a view or window. My current code: 


var mainWindow = Titanium.UI.createWindow({
    backgroundColor : '#FFFFFF'
})
var mapView = Titanium.Map.createView({
    mapType : Titanium.Map.STANDARD_TYPE,
    region : {
        latitude : 33.74511,
        longitude : -84.38993,
        latitudeDelta : 0.5,
        longitudeDelta : 0.5
    },
    animate : true,
    regionFit : true,
    userLocation : true
});
mainWindow.add(mapView);

var translucentView = Titanium.UI.createView({
    height : 200,
    bottom : 0,
    tintColor:'#ffffff',
    backgroundColor : '#2E2EFE',
    translucent : true,
});
mainWindow.add(translucentView);

mainWindow.open();

Not working tho :-(

Who can help me?

Regards, Marc

SDK 3.1.3 GA / Xcode 5 / SDK 7.0
