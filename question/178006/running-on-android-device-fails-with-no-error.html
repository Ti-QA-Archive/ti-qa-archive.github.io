I'm trying to run my app on an Android device (LG-E975). It used to work, but today it doesn't anymore : the process halts at random points, anywhere between

Titanium Command-Line Interface, CLI version 3.3.0, Titanium SDK version 3.2.3.GA

(sometimes the launch process just terminates here, with no error message)

and

[INFO] :   Making sure the adb server is running
[INFO] :   Removing deploy.json from SD card
[INFO] :   Installing apk: C:\path\build\android\bin\myapp.apk
[INFO] :   Installing app on device: LG-E975

There is no error message. The launch process simply terminates with no reason given. If I wait long enough, I eventually get a popup saying
"Session initialization failed
connect timed out"

I tried restarting my device, uninstalling the previous build, upgrading the Android SDK: launching the app directly on the device (either in "run" mode or in "debug" mode) keeps failing. Anyone having the same issue? (and possibly a fix?)
