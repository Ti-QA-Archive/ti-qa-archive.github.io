  • What you need to do is remove the "complete" event listener when you want to stop the animation. So, first, make the callback its own function:

    function spinMe() {
    t = t.rotate(20);
    spin.transform = t;
    image.animate(spin);
}

    Then, create your event listener:

    anim_but.addEventListener('click',function(e){
    spin.addEventListener('complete', spinMe);
});

    Finally, in your stop button's click callback:

    stop_but.addEventListener('click',function(e){
    spin.removeEventListener('complete', spinMe);
    Titanium.API.info("rotate stop"); 
 });

    Untested, but should help you out.
