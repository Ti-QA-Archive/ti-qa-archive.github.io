Invalis SQL statement

Hi Everybody,

Does anyone knows what is wrong on this query?

"UPDATE tbList SET checked = " + "'" + status + "'" + " WHERE section = " + "'" + Ti.UI.myObject + "'" + " AND shopItem = " + "'" + item + "'";

On appceletor it gives a error "Invalid SQL statement" but on SQLite Manager.app it runs OK.

— asked January 16th 2014 by Nilmar Castro

  • Accepted Answer

    I'm pretty sure you shouldn't use things like Ti.UI.myObject inside a string/query. If you need to save a global var use Ti.App.myObject.

    Furthermore, write your query like this:

    db.execute("UPDATE tbList SET checked = ? WHERE section = ? AND shopItem = ?", value1, value2, value3);

    it would result more readable.
