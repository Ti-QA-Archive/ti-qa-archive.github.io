  • Unfortunately, you are going to need to update your code. A lot has changed since 2.1.4 was released.

    First step is to change the tiapp.xml to it tries to compile with 3.2.3.GA.
    Then just work through your code as it breaks and fix.

    If you used NavigationGroups, those all will need to come out. There were some namespace changes since then, so some constants may no longer exist if they were relocated into a platform specific namespace.

    If you are building for android, the min supported API was increased as well, so check your tiapp.xml to be sure its 10 not 8 now in the <android> section
