Save password in device

Hi guys,

I'm creating an app for iPhone. And in my app, there is option to lock using password. The problem I'm having is that:

1) How can I save the password when I create the lock? (the password is saved even when the app is closed so that i can access it when reopen the app)
2) How do I get the password back to compare back when I want to unlock?
3) If there is muliple lock, how do I know which password is for which lock?
4) How do I delete the password after successful comparison?

I'm using 3.0.2.GA and hope that someone can help me by providing example…

    When click on that lock at that time set that password value using this setString.

    Ti.App.Properties.setString(myVariable,password.value); // set password value

    when you need back that password at that time using getString you can get back it.

    Ti.App.Properties.getString(myVariable);
