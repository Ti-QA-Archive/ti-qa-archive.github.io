    Hi

    Yes it is a standard tableView with the style property set to the grouped setting. Then you add sections of small numbers of rows, some sections have footer views with labels inside. Some of the rows are custom tableViewRows so you can add a switch or caption in addition to the title.

    • Having done a lot of research on this subject since your question I have found out a few things;

      • This is not an error
      • This is by design
      • The new grouped view IS this one
      • The grouped table view with rounded corners is only in the iPad settings app the iPhone version is the new format as are all other tables
      • The iOS settings app presentation of a grouped view is not a standard view
      • This may be a restricted property that is not yet public or simply a custom table

      There is more information on tableView styles including the grouped one in the following set of slides; tables are slide 33 onwards;

      • iOS 7 UI Transition Guide
