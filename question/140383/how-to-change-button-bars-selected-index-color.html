How to change Button Bar's selected index color

I need to know how i can make difference between selected button bar's index and unselected button bar's index by changing backgroundcolor or image.

  • Hi Sumit

    Unfortunately you cannot.

    The only color you can change is backgroundColor, the selected items color is simply a darker version of that background color.

    It is a shame as often the it is hard to work out which is selected and which is not if there are only two items.
