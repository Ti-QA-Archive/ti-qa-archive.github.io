Filesystem Android

I'm and building for Android using build 1.7.5. I am attempting to write some information to a file but it does not seem to be working. I created an iOS version of this app and the file creation worked there. I found a question on the community forum similar to mine and implemented the solutions given there but it still did not work. Here is my code:

var newDir = Titanium.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory,'Dir1');
newDir.createDirectory();

var newFile = Titanium.Filesystem.getFile(newDir.nativePath,'newfile.txt');
newFile.write('This is new text in a file');

I am, however, able to read from an existing file using the following code:

f = Ti.Filesystem.getFile(Ti.Filesystem.resourcesDirectory, 'uid');
var uid = f.read();    
return uid.text;

Any Suggestions? Also, is it better practice for an Android app to write files to "Ti.Filesystem.resourcesDirectory", or "Ti.Filesystem.applicationDataDirectory"?
