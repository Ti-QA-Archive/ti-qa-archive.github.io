    • Ok, it turns out the location of the file is the reason, with Alloy, you need to put the i18n folder on the same level as your app folder, not under it.

      — commented October 6th 2013 by Moshe Marciano

    • The i18n folder has always been placed at the same level as resources folder and it is the same with the app folder;

      Traditional structure

      {appname}/i18n/
{appname}/resources/

      Alloy structure

      {appname}/app/
{appname}/i18n/

      Note the location of the i18n folder is the same in both examples.

      This is shown in the international guide documentation using the traditional app structure.

      In this example the {appname} is KitchenSink and resources and i18n are both at the same level.

      The alloy version of this folder layout can be found in the Convention over Configuration documentation.
