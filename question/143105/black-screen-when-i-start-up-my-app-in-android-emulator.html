  • how long did you wait?

    Depending on the speed of your system, could take a minute or two for first boot of the android VM.

    What you could do is run 'android' from inside the android-sdk directory. Find the VM called Titanium_??? and start it, see if it will run there.

    Also, to see more of whats going on inside of the android vm, need to use adb or better yet ddms, a utility found in the android-sdk/tools directory to watch the vm logs.
