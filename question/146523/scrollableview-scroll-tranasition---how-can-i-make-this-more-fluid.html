  • Hi

    Not a lot of code to work through there, have you consider adding all the views at the same time?

    var views = [];
var intView = 0, intViews = 100;
for (intView = 0; intView < intViews; intView = intView + 1) {
    views.push(coreView[intView]);
}
var scrollableView = Ti.UI.createScrollableView({
  views: views,
  showPagingControl: true
});

    This will put all the views into the scrollableView at the same time.

    Without seeing the contents of the coreView that is very little else I can do for you if this does not improve things (which it should a bit).
