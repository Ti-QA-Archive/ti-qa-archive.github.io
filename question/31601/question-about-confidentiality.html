  • If you'd like to use our cloud based service for packaging for multiple platforms, then yes, your source code will be sent to our servers. We will create downloadable binaries that you can use for distributing your application. This is not strictly necessary however - it is possible to package your application manually, though it is a manual and undocumented process at this point (which we need to rectify). For now, you can stop by #titanium_dev on irc.freenode.net to chat up the Titanium core team on how to go about this process.
