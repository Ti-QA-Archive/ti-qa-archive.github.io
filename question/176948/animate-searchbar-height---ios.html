Animate Searchbar Height - iOS

Hello,

Is it possible to animate a searchbar's height at all? It's possible to adjust the height using setHeight.
It's also possible to animate it's opacity. But when I'm trying to animate it's height nothing happens.

PoC:

Ti.UI.backgroundColor = 'white';
var win = Ti.UI.createWindow({
  exitOnClose: true,
  layout: 'vertical'
});

var search = Ti.UI.createSearchBar({
    height: 0 
});

var list = Ti.UI.createListView({
    top: 10
});
list.search = search;

var btn = Ti.UI.createButton({
    top: 20,
    width: "60%",
    height: 40,
    title: "Click"
});
btn.addEventListener("click", function() {
   search.animate({
       height: 40,
       duration: 250
   });
});
win.add(btn);
win.add(list);
win.open();

Regards,
Patrick
