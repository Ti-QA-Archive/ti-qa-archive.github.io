    • Here is a PHP version you can use as a starting point, with this is takes an image and then resize the image down and the crops the resulting image to match the required the dimensions.

      You can use this on the server when images are added into your admin and you can then have the correctly sized images as you need them - these then are served to your app.

      It uses the superb PHP library Gravitywell Image with some tweaks of my own.

      http://pastie.org/5674329

      This is the important part, you will find many functions in the library you can use.

      $image = new Gravitywell_Image("color.jpg");
$image->thumbnail(320, 120);
$image->save("resized.jpg", 100);
