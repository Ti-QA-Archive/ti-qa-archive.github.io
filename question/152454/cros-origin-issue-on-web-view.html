  • Ti.Network.httpURLFormatter is only used with Ti.Network.HTTPClient calls. It does not work with Ti.UI.WebViews.

    If you are using Ti.Network.HTTPClient, the server you are connecting to must have CORS enabled and it must be configure to respond to OPTIONS requests with a 200.

    Please refer to this post for more info: http://developer.appcelerator.com/question/149900/tinetworkhttpurlformatter-for-web-app.
