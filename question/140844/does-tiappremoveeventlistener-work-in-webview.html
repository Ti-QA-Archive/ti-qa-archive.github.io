Does Ti.App.removeEventListener work in webView?

I use webView with following listener Ti.App.addEventListener(&#39;fetched-transitions&#39;, appendTransitions); which is binded when DOM is ready. But problem is that this listener is chained when I redirect to another url and back like this:

function goToOverview() {
    Ti.API.info('goToOverview()');
    win.webViewUrl = "/pages/activity_show.html";
    webView.setUrl(win.webViewUrl);
    webView.reload();
}

I tried Ti.App.removeEventListener(&#39;fetched-transitions&#39;, appendTransitions); but it does't work for me… Is there a possibility to use it in webView?

  • I found solution for it. Have to remove this listener on window unload() event.

    $(window).unload(function() {
    Ti.App.removeEventListener("fetched-transitions", appendTransitions);
});
