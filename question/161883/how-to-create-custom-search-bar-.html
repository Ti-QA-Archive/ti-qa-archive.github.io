  • Hi

    You have not actually stated what is not working with this code, but here are a few things I can see wrong;

    Change as follows;

    var tableView = Ti.UI.createTableView({
    separatorColor : '#C0C0C0',
    backgroundColor : 'white',
    // do not set this as this does not use the filter search bar
    // filterAttribute : 'title', 
    // do not set this as this does not use the filter search bar
    // filterCaseInsensitive : true,
    // do not set this as this does not use the filter search bar
    // searchHidden : false,
    top : 44 // should be the same as the search view height
});
// the tableView should take up less space than that available to it make room
// for the custom search
// you should NOT be covering anything up - by not setting search settings,
// the built-in search should not appear

    This code is causing a visual problem. Not sure why you are doing this?

    // this adds a label over the row
var lbl = Ti.UI.createLabel({
    text : "Row " + x
});
// this set the title of the row to be the same as the label above it
// really not sure why you have both - only one should be required and used
row.title = lbl.text;

    You have not told this function what to do;

    customSearchField.addEventListener('change', function(e) {
    customSearchField.value = e.value;
});

    This function should be looping through the data from the table (originally from how you added it). Then reducing the results to only those that match the enter search field. The problem is you example code does no load any real data - it simply provides a generic caption with a number after it. Nothing I can do here to help.

    You need to explain WHAT does not work and WHAT you think it should be doing that is is currently not doing.
