    Hey David,

    First of all I am considering your environment to be a Mac system but similar workaround is for Windows as well.

    1. Please uninstall node from your system and install node again from the titanium studio update. The latest version of node somehow causes issues with the titanium studio. Ideally it should be 10 and the latest version is 12.
    2. If thats does not work you will have to uninstall titanium studio and install another version or setup since maybe the downloaded setup has issues.

    Thanks.

    2. rename the extracted folder to android-sdk, if you wish to keep it in accordance with the Software Locations section above
