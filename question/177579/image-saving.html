  • Accepted Answer

    When the image is loaded in the webView you can use following code to save the image on android

    var imageFile = $.webview.toImage();
// $.webView is the object handle of the webView, since im creating the webView in xml Im using this way.

if(Ti.Filesystem.isExternalStoragePresent()){
    var file = Ti.Filesystem.getFile(Ti.Filesystem.externalStorageDirectory, 'image.jpeg');
    file.write(imageFile.media);
    Ti.Media.Android.scanMediaFiles([file.nativePath], null, function(e){});
}

    After this you should be able to see the saved image in your gallery.
