Accessing custom properties given to Ti.UI.TextField in iOS module

Hi,
I am creating a custom module in iOS for TextField being used in titanium.
Task of module is to let me know when user focuses on a text field, along with some calculation in module itself.
The code to require the module is as below

var module = require("com.textFieldDelegate");

module.setCallbackFunction(function(e) {
alert(JSON.stringify(e));
});

module.setValueForTextField({
"valueFromServer" : _theValue
});

// iOS TextField delegate code

-(void)textFieldDidBeginEditing:(UITextField *)textField{

NSString *value = textField.valueFromServer;

// Do something.
// Call the callback method to return the computed value.

}

My problem here is that I am unable to access the property "valueFromServer" in iOS module's delegate method.

Any help will be helpful and thanks in advance!
