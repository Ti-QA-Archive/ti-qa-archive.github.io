Kindle Fire detect screen size problem

Greetings,

I am having one of those 3 hour headache problems. I can't seem to get the if statement to detect which version of the Kindle Fire is running. It always runs the lesser screensize no matter what. The code is as follows:

if (Titanium.Platform.displayCaps.platformHeight == 1920){
    //content 1 goes here
}

if (Titanium.Platform.displayCaps.platformHeight == 1280){
    //content 2 goes here
}

It always displays content 2, even if I'm running the emulator for the Kindle Fire 8.9 (1920x1200). Any insight would be most appreciated.

  • Ok guys, I solved the problem. I'll illustrate it below so maybe someone else will save some time.
    The problem is that I had variable names that were the same in both if statements, so I had:

    if (Titanium.Platform.displayCaps.platformHeight == 1920){
   var image1 = //stuff
}

if (Titanium.Platform.displayCaps.platformHeight == 1280){
    var image1 = // different stuff
}

    It appears that even though the value was encased in the if statement, it was still being written for some reason. Ah, time for a drink..
