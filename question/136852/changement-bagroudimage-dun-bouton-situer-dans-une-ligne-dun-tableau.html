    Le plus simple je pense, sans trop modifier ton code :

    function changeIMG(theButton){
    theButton.backgroundImage = "yourImage.png";
}

// Puis tu appelles cette méthode comme avant mais avec le paramètre (dans ta fonction display()) :
changeIMG(imageConnect);

    Comme l'a dit Rainer, la structure de ton code n'est pas super, mais c'est peut être du au fait que tu as simplifié ton code pour que l'on puisse comprendre ta problématique.
