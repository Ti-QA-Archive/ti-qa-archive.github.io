Launch Titanium Developer in small resolution

Hello,

I'm trying to run Titanium Developer on Mac that runs in 1024 x 576 resolution.

The app doesn't scale to this resolution and goes out of my screen. I cannot access the toolbar buttons so I can't really use it at all.

Why haven't you included the new project, import project, perspectives and update actions to the app main toolbar? That way it could at least be used somehow on smaller screens.

Is there any way to launch Titanium Developer in smaller size? Is it possible to use it fully from console?

Currently it cannot compile Android or iPhone at all, because it loaded just 1.0.0 sdk, so the new Android and iPhone sdk's aren't supported. It would have been nice for it to install the latest titanium sdk on first install, but it got 1.0.0 instead.

Any help for us with small screens and resolutions?

    The Titanium Developer app can easily be resized if you have access to the bottom right corner. It seems to tolerate small sizes well, scrolling the middle contents and keeping necessary controls visible.

    If you can't reach that corner look for a third party tool that will let you resize windows with the keyboard. Found this in a quick search but have not tried it:

    http://mindflow.dk/software/windowwrangler/

    Since TI doesn't appear to save size or position you'll need to do this on every launch if there isn't another way to change the startup size.
