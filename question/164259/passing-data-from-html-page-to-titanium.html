  • Unless you have good reason to use a webView to build a Login window (and not knowing how to do it the Titanium way should not be one), I'd suggest you to create a Ti.UI.Window and use very basic Ti.UI elements as Titanium.UI.TextField (which also offers you a passwordMask property) and Titanium.UI.Button; then use an eventListener to the button to handle the login process. (Or even better dedicate some time to learn about how to use Alloy in your projects).

    Find more in-depth informations in the UI documentation, it should become a good friend of yours.

  • To follow my previous comment with something more useful, here's a bare-bones example in 'classic' titanium . Also missing is the authentication process (I guess you'd want to make a POST http request to a dedicated API on you server). 

    // create a window with a vertical layout
var win = Ti.UI.createWindow({
    layout:"vertical"
}),

// create children elements equally spaced by 20px (vertically)
// the title will be placed at 50px from window's top
var title = Ti.UI.createLabel({
    text: "Sign In Into your Account",
    top: 50,
});

var username = Ti.UI.createTextField({
    hintText: "username",
    top: 20
});

var password = Ti.UI.createTextField({
    hintText: "password",
    passwordMask: true,
    top: 20
});

var login = Ti.UI.createButton({
    title: "LogIn",
    top: 20
});

// add some functionality to the login button
login.addEventListener("click", function(){
    // validate login fields
    if (username.value != "" && password.value.length > 7) {
        // do user authentication and handle success/error
    } else {
        alert("Login fields cannot be empty");
    }
});

// add children elements to the parent window
win.add(title);
win.add(username);
win.add(password);
win.add(login);

// all is set, then 
win.open();

    You can style each UI element with CSS-like properties, as explained in the documentation; here I just show the use of top property to position the elements in a vertical layout (as defined in their parent window).

  • Hello Mainul Hasan,

    Thanks for raising this issue. I have added sample code for your issue. In my code, I add an onclick function to pass data from html page to titanium window. Please test my code in your environments. I hope it solves your problem.

    Check attachment

    Test Code:

    App.js

    var win = Ti.UI.createWindow();
var webview = Ti.UI.createWebView({
    url: 'logging.html'
});


Ti.App.addEventListener('app:fromWebView', function(e) {
    alert('User name is '+e.message);
});

win.add(webview);

win.open();

    logging.html

    <html>
    <head>
        <script>

            function myFunction()
            {
            var username=document.getElementById("userName");    
            Ti.App.fireEvent('app:fromWebView', { message: username.value });

            }

       </script>
    </head>

    <body>
        <table>
            <tr>
                <td class="tdAlignLeft">
                    <label>
                        Your Name
                    </label>
                </td>
                <td class="tdAlignRight">
                    <input type="text" id="userName"  type="text"/>
                </td>
            </tr>
            <tr>
                <td colspan="2" class="tdAlignRight">
                    <button onclick="myFunction()">Login</button>
                </td>
            </tr>
