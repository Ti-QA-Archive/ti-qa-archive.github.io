Container Box with different color - Alloy

I'm new to this. So bear with me !

I've created one creen with some text on top and then some buttons. Now I want to put the UI controls (buttons) inside another box container with different background color.

<Alloy>
    <Window id="win" backgroundColor="white">
        <View id="screenContainer">
        <Label id="title" >Welcome</Label>
        <Label id="text">Lorem Ipsum Lorem Ipsum Lorem Ipsum Lorem Ipsum!</Label>
        <View id="buttonContainer" backgroundColor="red">
        <Button id="button" onClick="openPage" title="Hey" 
         top =110/>
        <Button id="button" onClick="openPage" title="Hello" 
         top =180 />
        <Button id="button" onClick="openPage" title="What" 
         top =250  />
         </View>
         <ImageView id="image" image="/android/images/kool.png" />
         </View>
    </Window>
</Alloy>

This is making the whole screen go red in color. How can I make that box with buttons in it only with red color ?

    Hi

    We need to make some tweaks here

    // change this
<View id="buttonContainer" backgroundColor="red">
// to this
<View id="buttonContainer" class="buttonbox">

    Then in the associated TSS file (same name as XML file) add this class entry.

    "buttonbox": {
    backgroundColor: 'red'
    height: Ti.UI.SIZE,
    width: Ti.UI.SIZE
}

    You can change the width and height to specific values but this will get you started - chances are this view is assuming a full device width and height - this should stop that.
