Error trying to use Titanium.Utils

Hello -

I'm trying to use a method in Titanium.Utils but I keep getting this error when running the application:

Error = Result of expression 'Titanium.Utils' [undefined] is not an object. at app.js (line 381)

All of the following permutations cause the error:

var password2 = Titanium.Utils.md5HexDigest("password");

or

var password2 = Ti.Utils.md5HexDigest("password");

I also tried base64encode…seems like anything in the Utils package.

How do I use the Utils package in my code? I must be missing something.

Thanks!

    I had this problem after the move to 1.5.1. I resolved this by trashing the build/iphone directory (don't forget to save any custom Info.plist entries) and letting Titanium rebuild the project.
