Mapevent (click) from within a view?

I have a map within a view. I add annotations to it. I want to let the user click on an annotation and get more info. As I understand it I can't get the clickevent from the map/annotation when I have a mapview within a view.

Has anyone a workaround?

    Hi.. as far as i know.. you can get the click event in annotation and not in the map.. whether you put it in a view or window..
    to create an infoWindow… you have to set the title, and subtitle of annotation…

    var annot = Titanium.Map.createAnnotation({
    latitude: data[i].latitude,    //latitude
    longitude: data[i].longitude,    //longitude
    animate:true,
    title: "This is title",
    subtitle: "This is subtitle",
    pincolor: Titanium.Map.ANNOTATION_RED    //the pin
});

    just a tip also.. in android there is bug if the map is in heavyweight it don't update the maptiles.. a work around in this is…

    var win = Ti.UI.createWindow({
    modal : true,        //set first the modal to true
    navBarHidden: true    //then set the others
});

    I hope this helps…
