Layout in Titanium

I searched a lot but could not find a good tutorial on layout.

I have below queries.

  1. Is there any way to write a custom layout?

  2. Can I use TableView to divide screen to three section (just like swing borderlayout without west and east component). I want a toolbar on top with a specified height and 100% width. Center portion of the screen should be working area. component placed in center of the screen will occupy rest of the screen if required after leaving space for footer. Footer should always be placed at the bottom of the screen even the center view uses very small portion of the working area.

  3. Is there any other option to meet the above requirement.

    I don't know what the swing borderlayout is, but here is how you would do what you described in terms of positioning.

    var win = Ti.UI.createWindow();
win.open();

var topHeader = Ti.UI.createView({ top: 0, left: 0, right: 0, height: 80, backgroundColor: "#000" });
//TODO: add stuff to header
win.add(topHeader);

var table = Ti.UI.createTableView({ top: 80, left: 0, right: 0, bottom: 80 });
//TODO: add rows to table
win.add(table);

var bottomFooter = Ti.UI.createView({ bottom: 0, left: 0, right: 0, height: 80, backgroundColor: "#808080" });
//TODO: add stuff to footer
win.add(bottomFooter);

    The TableView object also has Headers and Footers that you can set which may fit your needs, see the API for info on that.
