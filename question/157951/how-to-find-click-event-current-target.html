How to find click event "current target"?

Consider a view A with a few child views. I need to listen for clicks anywhere on the view A and read a property from A in the handler. When I add a click listener to A, the event.source is whichever one of the child views was clicked. How can I get a reference to the parent view A to which the listener was attached?

    If the handler is attached to the parent 'A' then the value of this ought to be set to 'A' . This is true as long as the bubblesParent property on the children is set to True (which is the default)

    eg

    var w = Ti.UI.createWindow();

w.customProp = 'I am A';

var b = Ti.UI.createButton({
    width : 200,
    height : 200,
    title : 'Click me'
});

w.add(b);

w.addEventListener('click', function(e) {
    alert(this.customProp);
});

w.open();
