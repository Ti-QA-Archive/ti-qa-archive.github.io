Alloy SQLite DB

Hello,
I am just trying to select only one field from table.
Below code works and it returns whole row.

var subscribers = Alloy.Collections.subscriber;
 subscribers.fetch({query:'SELECT * from subscriber where id="' + args.id + '"'});

but I want to select only one field from table. Im trying so but it does not work:

var subscribers = Alloy.Collections.subscriber;
var name = subscribers.fetch({query:'SELECT name from subscriber where id="' + args.id + '"'});
alert(name);

Please, help me.

    You will need to do it like this:

    subscribers.fetch({query:'SELECT name from subscriber where id="' + args.id + '"'});
var j = subscribers.toJSON();
var name = j.length ? j[0].name : null;
