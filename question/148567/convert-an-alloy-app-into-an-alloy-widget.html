Convert an Alloy app into an Alloy widget

Hi,

I'm testing Alloy (v1) and Titanium Studio (v3.0.2) on Ubuntu (v12.10). I've read all the documentation about Alloy but haven't found / understand how to transform an Alloy app into an Alloy widget. You can correct me if I'm wrong, but I think that there could be some cases where this could be interesting to reuse the code easily. As an example, I've an eye over the RSS sample app we can find in the Alloy's official repository: https://github.com/appcelerator/alloy/tree/master/samples/rss

Can anyone give me some advice on how to accomplish this? Can the Appcelerator team add a new documentation page with a migration guide (app –> widget)?

I've tried to accomplish this myself, but I haven't got strong skills on Javascript and I'm still a bit lost with Alloy. For example, after different code changes, I get stopped on compiling the app for Android because of the views/widget.xml (old views/index.xml) can't find the correct route to the required "master" controller/style.

Thank you in advance!
