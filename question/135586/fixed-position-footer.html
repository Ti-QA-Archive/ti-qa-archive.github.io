Fixed position footer

I have an app loaded into a tab group for which I need to create a footer. The footer will appear on every window and should sit with bottom: 0, but that property doesn't seem to be enough (if/when the window layout is set to vertical). Is there a useful approach that can be recommended for creating such a thing?

Thanks.

  • Accepted Answer

    if the footer is common to all tabs, add that footer to the tabGroup instead of to the window.

    If the footer is only for one window, do not use a vertical layout. Instead, use a main view with all your content in a vertical layout and add that view to the window. Then, add the footer to the window with bottom:0
