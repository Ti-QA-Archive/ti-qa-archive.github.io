ios TabGroup with rightNavButton

Hi all,
I'm using Alloy 1.3, TItanium SDK 3.2.0.v20131112180422 on iphone 7.0.3

I'm trying to show a button on the top right of a tabgroup, but not really sure how to do it.
My code is like this:

$.tabgroup.setRightNavButton(Ti.UI.createButton({ title: L('Settings') }));

Is it correct?

    You can just add the button to the tab group in the controller using add, making sure you get the positioning correct:

    $.tabgroup.add(Ti.UI.createButton({ 
      title: L('settings')
    , bottom: 40
    , right: 2 
}));

    Although I would put the button in another view/controller…

    • Are you trying to add a button to the top/right of the tabgroup or the top/right of the screen (window)?

      A tabgroup contains tabs, each tab should contain its own window. The window then has a rightnavbutton which you can set:

      $.windowName.setRightButton(myButton);

      or 

      <Alloy>
    <Window id="windowName">
        <RightNavButton>
            <Button id="myButton" onClick="handleMyButtonClick">
        </RightNavButton>
    </Window>
</Alloy>
