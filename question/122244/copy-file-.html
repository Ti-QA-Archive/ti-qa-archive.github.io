  • Hi Erwin,

    in the following code snippet I get a pdf and save it on phone:

    c = Titanium.Network.createHTTPClient();
c.setTimeout(60000);
c.onload = function() {
  // here you can use the cached file

}
c.open('GET','http://www.hamburg.adfc.de/uploads/media/' + issue);
c.file = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory,issue);
c.send();

    Hope it helps.
