Alloy android preferences

Very silly question, but I can't find anything in the docs …
Where does the preferences.xml file go when in an alloy project. I tried putting it under assets/android/res/xml … but that didn't work. I tried creating a platform folder under /app and place it there like my non-alloy projects, but that didn't work. Anyone have any clues about the structure?

Alloy 1.0

3.0.2GA

Titanium Studio, build: 3.0.2.201302191606

    Create the folder in the same exact place you did in the old titanium projects, in the project root's "platforms" folder, not in the "app" folder.
