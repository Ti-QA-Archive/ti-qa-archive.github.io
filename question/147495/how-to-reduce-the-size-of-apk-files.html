  • Hi

    I think you will find that a base standard Android app has a largish footprint without any additional resources.

    Create a new default project and then WITHOUT adding or changing anything compile that in the same way as you id with your main app.

    Then provide the following details;

    • What size is that app?
    • What is the total size of your resources folder including all scripts and all of the images you have?

    Let us know, I suspect the two figures added together are very close the the current size of our app.
