  • Note that 0FFF is octal, not hexadecimal.

    hexadecimal is only a form of representate a value. But the value itself will be sent in the same manner by a socket if you send an decimal integer such as 4095 or an hexadecimal integer as 0xFFF, both are the same values.

    Maybe you want to send hexadecimal values as string values. In this case, you can send a string like "0xFFF". To convert a numeric variable to its hexadecimal string, do this:

    var x = 4096;
var hexString = x.toString(16); //hexString is 'FFF'

  • function decodeFromHex(str){
    var r="";
    var e=str.length;
    var s;
    while(e>=0){
    s=e-3;
    r=String.fromCharCode("0x"+str.substring(s,e))+r;
    e=s;
    }
    return r;
    }
