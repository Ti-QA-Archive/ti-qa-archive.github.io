  • There's no need for a click listener on the tableview itself. You should instead add a click listener to tabbedbar. Add this code after you define tabbedbar:

    // definition of tabbedbar...

tabbedbar.addEventListener('click', function(e) {
    row.title = e.source.labels[e.source.index];
});

    Your labels for that row will change when tabbedbar does now.
