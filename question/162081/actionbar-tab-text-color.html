  • By default, if you set the Holo Light Theme as parent of your custom Theme, youl'll get black Tab titles. You can change the font color of the ActionBar Tabs with this:

    <resources>
    <style name="CustomActionBarTheme"
           parent="@style/Theme.Holo.Light">
        <item name="android:actionBarTabTextStyle">@style/MyActionBarTabText</item>
    </style>

    <style name="MyActionBarTabText"
           parent="@style/Widget.Holo.ActionBar.TabText">
        <item name="android:textColor">@color/actionbar_tabs</item>
    </style>
</resources>

    Don't forget to define the color in res/values/colors.xml

    <resources>
    <color name="actionbar_tabs">#CC043142</color>
</resources>
