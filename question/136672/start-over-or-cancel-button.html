'Start Over' or 'Cancel' Button

  • The app is a tabGroup
  • tab[1] contains a tableview list of items
  • Clicking an item builds and opens a new window with a sublist tableview, continues for X levels
  • I have a CANCEL rightNavButton on all windows loaded after the original tableView
  • Clicking CANCEL needs to "reload" the first tableview, closing all other subwindows i've built

the effect i want is the same as if you physically click the currently active tab[1]. it reloads the tab.

i tired to fake that programmaticaly with this firing when CANCEL is clicked:

tabGroup.close();
tabGroup = W.TabGroup();
tabGroup.open();
tabGroup.setActiveTab(tabGroup.tabs[1]);

it gets the job done, but the following warning irks me:

[UITabBarController setSelectedViewController:] only a view controller in the tab bar controller's list of view controllers can be selected.

any tips of a better way to start over once you go down a determined path of tableViews?
