Alloy : set Right Nav Button issue

I would like to set left nav button to window and I did something as follows

win2.xml

<Alloy>
    <Window id="index" modal="true">
        <View backgroundColor="green" />
        <Button id="lbtn" onClick="closeWin" />
    </Window>
</Alloy>

win2.js (Controller)

function closeWin() {
    $.index.close();
}


function initView() {
    $.index.leftNavButton = $.lbtn;
}

initView();

$.index.open();

$.index.addEventListener("close", function() {
    $.destroy();
});

The problem is that the button is added twice - top left and in the middle of the window.

I try modifying it a bit like :


<Alloy>
    <Window id="index" modal="true" leftNavButton="$.lbtn">
        <View backgroundColor="green" />
        <Button id="lbtn" onClick="closeWin" />
    </Window>
</Alloy>

But, it does not work… Any idea?
