  • Hi

    If you mean can you make your app run in the background then the short answer is - no.

    However; no does not always mean no.

    • You can run background audio in you app
    • You can have code run as a background service - limited functionality and time allowances
    • You can have local notification run to trigger things.

    You cannot run a true background app as that is not how iOS apps work. When your app is placed into the background because the home button has been pressed or the lock screen has been activated or another app has the focus, you cannot have your normal activities still run in the same way.
