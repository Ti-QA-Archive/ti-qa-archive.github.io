Label in Alloy TableViewRow disappears when using percentage width

Within a TableViewRow, I'm trying to have a Label as 25% width (left="0") and a TextArea as 75% width (right="0"). If I have width="25" and "75" (just pixels or "dp"s) they display fine. But as soon as I make them a percentage, they disappear. It doesn't seem to be a "width" issue, though, as I can make them "5%" and they still disappear.

I'm using Alloy.
Titanium SDK 3.1.3.GA
Titanium Studio, build: 3.1.3.201309132423
iOS Simulator Version 7.0 (463.9.4.2)

It works fine on Android 2.3.3 emulator (well, they display, still working on lining them up).

team.xml:

<Alloy>
    <Window id="team" class="container" title="Modify Team" layout="vertical">
        <View width="100%" height="Ti.UI.SIZE" layout="horizontal">
            <View width="50%" height="Ti.UI.SIZE" layout="vertical">
                <Button id="rosterBtn" class="btn" left="5%" onClick="openRoster">Manage Roster</Button>

                <Button id="saveBtn" class="btn" right="5%" onClick="saveTeam">Save/Update</Button>
            </View>
        </View>
        <TableView width="100%" height="100%" backgroundColor="transparent" separatorColor="#EEEEEE">
            <TableViewRow class="tableRow">
                <Label class="leftCol">Team Name:</Label>
                <TextArea id="teamName" class="valueCol" editable="true"/>
            </TableViewRow>
            <TableViewRow class="tableRow">
                <Label class="leftCol">City:</Label>
                <TextArea id="city" class="valueCol" editable="true"/>
            </TableViewRow>
            <TableViewRow class="tableRow">
                <Label class="leftCol">State:</Label>
                <TextArea id="state" class="valueCol" editable="true"/>
            </TableViewRow>
        </TableView>
    </Window>
</Alloy>

    I found the issue here: http://developer.appcelerator.com/question/143154/horizontal-layout-and-percentage-width-do-not-work-well

    A TableViewRow can only have one object/View. So the below solved itâ¦ instead of having a Label and TextArea inside the TableViewRow, I just added one View inside the TableViewRow, and the Label/TextArea were inside that View, so the TableViewRow only had the 1 child element.

    <TableViewRow class="tableRow">
    <View layout="horizontal">
    <Label class="leftCol">Team Name:</Label>
    <TextArea id="teamName" class="valueCol" editable="true"/>
    </View>
</TableViewRow>
