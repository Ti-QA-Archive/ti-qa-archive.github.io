How to call module method in js code

I have a module class:

@Kroll.module(name="TestHelpers", id="com.home.testhelpers")
public class TestHelpersModule extends KrollModule
{

    // Standard Debugging variables
    private static final String LCAT = "TestHelpersModule";
    private static final boolean DBG = TiConfig.LOGD;

    // You can define constants with @Kroll.constant, for example:
    // @Kroll.constant public static final String EXTERNAL_NAME = value;

    public TestHelpersModule()
    {
        super();
    }

    @Kroll.onAppCreate
    public static void onAppCreate(TiApplication app)
    {
        Log.d(LCAT, "inside onAppCreate");
        // put module init code that needs to run when the application is created
    }

    // Properties
    @Kroll.getProperty
    public int getNumber()
    {
        return 0;
    }

}

I've built it and loaded into my project just as the guide tells. Now, in my index.js file I do this:

$.index.open();

var module = require("com.home.testhelpers");

alert(module.getNumber());

When running this code, I'm getting: Uncaught TypeError: Object #&lt;TestHelpers&gt; has no method &#39;getNumber&#39;. So what should I do to call this code?
