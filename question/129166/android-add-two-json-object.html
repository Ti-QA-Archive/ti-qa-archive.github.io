[ANDROID] Add Two JSON Object

Hello,
How can i add two JSON object… to make them 1..

var json1 = {
    user : "Ken",
    msg : "Hello" 
};
var json2 = {
    user : "Unknown",
    msg : "Hi"
};
//how can i add these two to come up with these result
var result = {
    {
        user : "Ken",
        msg : "Hello" 
    },
    {
        user : "Unknown",
        msg : "Hi"
    }    
};

Thanks for the help.

    "result" code is not possible - it's not syntactically valid JavaScript code.

    What you need is an array:

    var result = [ json1, json2 ];

    • If they are arrays (and it seems they are from JSON string you posted), you can use "concat" function:

      var myjson = myjson1.concat(myjson2);

      The result would be:

      [
    0: {
        "user": "ken",
        "msg": "hi"
    },
    1: {
        "user": "Unknown",
        "msg": "Hi"
    }
]
