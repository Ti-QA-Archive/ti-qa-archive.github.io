  • Hello Roseanne,

    Can you tell me which data you want to read? There are no difference between basic table and custom table for reading data form table row. And hasChild is a table row property. You can set like 

            var row=Titanium.UI.createTableViewRow({
            height: "auto",
        hasChild: true
        });

    For more information please visit following link http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.TableViewRow-property-hasChild
