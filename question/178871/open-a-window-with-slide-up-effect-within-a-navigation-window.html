open a window with slide up effect within a navigation window

Hi,

I am trying to open a window from a navigationwindow. I want the window to slide-up. but as of now it always slides left even thou' i give Ti.UI.iPhone.MODAL_TRANSITION_STYLE_COVER_VERTICAL.

here is what i am trying


var navWin = Titanium.UI.iOS.createNavigationWindow({
        window : winMain
    });


var addeditproduct = Titanium.UI.createWindow({
            translucent : false,
            barImage : 'none',
            barColor : '#e0595a',
            navTintColor : 'white',
            statusBarStyle : Titanium.UI.iPhone.StatusBar.OPAQUE_BLACK,
            backgroundColor : '#f4f4f4',

        });


//on a button click i am trying to open addeditproduct window. Here I want slide up effect
        navWin.openWindow(addeditproduct, {
            animated : true
        });

Please advise.

I am using 3.2.2 GA. and checkin in IOS 7 and 8 devices

Regards,

    • When you use a NavigationWindow it handles the appearance of the windows and uses the correct UI for this - which is what you have been experiencing.

      The use of Ti.UI.iPhone.MODAL_TRANSITION_STYLE_COVER_VERTICAL and similar constants is for those times you open a window OUTSIDE of a navigation or TabGroup.

      If you are opening a window outside of the Navigation stack then you do not use the openWindow method as that creates a child window, you open it by itself and optionally use those constants and include the modal property if it is suitable.
