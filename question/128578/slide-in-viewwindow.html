Slide in view/window

Hi, im trying to replicate the default slide in / out of windows in the iPhone, but without using the navigationGroup because I need to compile for android, so the question is this:

How is the best way to do this, Ive tried using 1 containing view and adding child views to the container, these child views contain the content of each "window", but the problem with this is that a view thats created out of the screens visible space does not get any events (I cant slick on any button, cant scroll the scrollView, etc). I would animate the position of the container view the get the slide effect.

The other way that I tried was creating 2 windows at the same time, but I didnt get to far with this method.

Any ideas how to solve this?

    There's nothing to replicate, android supports this effect using normal windows. If you try kitchenSink samples, you'll see that new windows open with that slide effect.

    I suggest you to use windows instead of views, is the normal way.

    If you are looking for a multiplatform solution, take a look to this implementation of a custom navigation group:

    https://gist.github.com/1040334
