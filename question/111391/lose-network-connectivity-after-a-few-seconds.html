    See the advice in PC behind proxy server.

    Also, disable your local firewall software and antivirus to see if it makes a difference. If it resolves it, you will need to create some rules to ignore Titanium.

    Note that I've never heard anyone say that you should amend appcelerator.xml. I would change this back to the original while you follow the information above.
