  • Accepted Answer

    Titanium SDK 3.0.2.GA and older will only work on Node.js 0.8.x. You are running Node.js 0.10.0.

    We have fixed the problem (https://jira.appcelerator.org/browse/TIMOB-13025) and the upcoming Titanium SDK 3.1.0 will work on Node.js 0.10.0.

    Your options are to either:

    1. Downgrade to Node.js 0.8.22 (http://nodejs.org/dist/v0.8.22/
    2. Use a CI build (http://builds.appcelerator.com/#master). If you use a CI build, note that it's not fully tested and should be used for development only.
