  • Hi Chung, hope you are good.
    Here is my suggestion to accomplish this task:

    • Use "defer" function from underscore.js library to show the progress of both downloading files.
    • To use the above function, you will need to use the "ondatastream" function for all HTTP calls(which i hope you will be using in current version).
    • actually defer function provides you the mechanism of handling events at same time instantly, by postponing the current call, e.g. the button click event during another process in progress.
    • hence, this way you could achieve what you want because defer function will be acting somewhat like multithreading.

    Let me know if it helps you, otherwise i will write the code for you in coming days.
