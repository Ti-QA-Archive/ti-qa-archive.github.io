  • Whoops. Just realized this is a Stack Exchange type of site. Sorry for the double post in the answers section. Won't happen again.

    The fix that worked for me to build my app into an APK correctly was that I didn't set my alias correctly. So the correct step by step for me was this for generating my key:

    • keytool -genkey -v -key
      store mycompany.keystore -alias myalias_goes_here -keyalg RSA -keysize
      2048 -validity 10000
    • Fill out your info as it asks, and I just hit enter to make the second key password the same as the first one
    • When you compile your app, tell Ti where you want to save your app, where your key file is, what your password is that you used, and type your alias name

    The big issue for me was that I hadn't specified a correct alias name that was used when creating the key file, so the build-time alias and password need to match perfectly.

    I hope this helps! I'm new at this too and learning a lot quickly from this communities previous posts.
