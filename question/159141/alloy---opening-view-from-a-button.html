Alloy - opening view from a button

I have a view which contains 3 buttons. When a button is clicked, I want it to open the associated view. I've been away from Titanium for awhile and I can't seem to remember how to accomplish this.

Here's the homeView.xml:

<Alloy>
<Tab title="Home">
    <Window class="container">
        <Label>This is the Home View</Label>
        <ImageView id="kplLogo"/>
        <View id="homeNav">
            <Button class="navButton" id="recognizeButton" title="Recognize" onClick="doClick" />
            <Button class="navButton" id="reactButton" title="React"/>
            <Button class="navButton" id="reportButton" title="Report"/>
        </View>
    </Window>
</Tab>
</Alloy>

I've tried so many different variations on how to do this that I lost track. None worked. So, I'm starting from scratch.

In one incarnation, I tried this but received an error that view is undefined:

function doClick(){
    var clickView = Alloy.Controller('recognizeView').getView();
    clickView.show();
}

So, I'm out of ideas. I know I'm overlooking something fundamental, but I can't seem to recall what that is.
