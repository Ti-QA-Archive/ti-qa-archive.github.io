  • You can und must create the keystore file by yourself. You can use the studio or you can create the keystore on command line. The details you can find here

    $ keytool -genkey -v -keystore my-release-key.keystore -alias alias_name -keyalg RSA -keysize 2048 -validity 10000

    On the other hand you can read all details in Titanium documentation
