    Did you intentionally set the npm config prefix value to "/Users/sebastian/npm_install" ? If you haven't intentionally set this value, then can you remove/reset the env variable NPM_CONFIG_PREFIX to /usr/local and verify the command "npm config prefix" returns "/usr/local" ?

    Also, you mentioned that the command "npm -p -g list alloy" returns a lot of information ? This command should ideally return only the path of the alloy location (at least the last line of the output should return the alloy install location). Can you post the entire output of this command ?
