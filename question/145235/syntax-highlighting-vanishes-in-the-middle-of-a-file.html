Syntax highlighting vanishes in the middle of a file

I'm using Titanium Studio 3 beta (build: 3.1.0.201211281925) and I have a problem with the syntax highlighting. In some files it just vanishes abrudly at some random point. I haven't found anything weird or out of the ordinary in these files. I've tried closing and opening the files again but the problem persists. Titanium or OS restarts do not fix the issue either.     • I found the reason (I think). I'll post a simple example here. It seems that having a function like below causes the syntax highlighting to vanish at random point. I was able to remedy this by removing the ; from the exports call end.

      exports.createUser = function() {
    var user = {};

    user.exampleMethod = function(exampleAttribute) {
     ...
    };
    ...
   return user;
};

      So, with same code but no ; at the exports call end, there is no problem with the syntax highligting:

      exports.createUser = function() {
...
}

      I've structured my code so that one file exports one object. And they go under application namespace.
