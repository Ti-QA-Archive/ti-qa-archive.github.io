ERROR: Found Conflicting Resources

Hi, I am facing an issue when i try to run my application in titanium studio using android emulator.
Below is the error i am getting

[INFO] :   Launching emulator: FourNex
[INFO] :   Emulator already running
[ERROR] Application Installer abnormal process termination. Process exit value was 1
[ERROR] :  Found conflicting resources:

I dont have any same folder/files name.

Below are the details of the Setups
Titanium - Latest(3.2.3.xxx)
Titanium SDK - 3.2.3.GA

I have installed all Android SDK's. I am importing a JavaScript based Android app which was developed in older versions of Titanium.

I am stuck with this from past 2 days. Will really be thankful if anyone could help me out with this.
