How to retrieve List of Files in a directory and print it with Titanium Desktop?

Hi,
I'm trying to retrieve the list of files in my application's Resource directory and print the list within a DIV tag called "output". I tried with the following code, but it failed; (I was using jQuery)

$(function() {
        var resourcesDir = Titanium.Filesystem.resourcesDirectory();
        var dir = Titanium.Filesystem.getFile(resourcesDir);
        var dir_files = dir.getDirectoryListing();
        for (var i=1;i<dir_files.length;i++){ 
            $("#output").html(dir_files[i]);
        }
    });

When I execute this script it prints nothing. I tried several examples described in Q&A section, but had no luck.

I'm using Titanium Developer 1.2.2 (Desktop).

Pls help!

-Nilaksha

<script>
$(function() {
        var resourcesDir = Titanium.Filesystem.getResourcesDirectory();
        var dir = Titanium.Filesystem.getFile(resourcesDir);
        var dir_files = dir.getDirectoryListing();
        for (var i=1;i<dir_files.length;i++){ 
            $("#output").append(dir_files[i].toString() + '<br>');
        }
    });
  </script>
 <div id="output" style="border:1px solid #CDCDCD" />
