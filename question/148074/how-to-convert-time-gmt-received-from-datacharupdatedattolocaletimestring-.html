How to convert Time GMT received from data(char.updated_at).toLocaleTimeString() ?

I'm using the chat system from titanium cloud and I want to display the time that an message are displayed but when I use

container.add(Ti.UI.createLabel({
            text: new Date(chat.updated_at).toLocaleTimeString(),
            top: 10, 
            right: 10,
            font: { fontSize: 10 },
            height: 10,
            width: 'auto'
}));

all I get is something like that:

-8:-31:-23 GMT-03:06:28

  • var date = new Date(chat.updated_at);

container.add(Ti.UI.createLabel({
            text: date.getHours().toString() + ":" + date.getMinutes().toString(),
            top: 10, 
            right: 10,
            font: { fontSize: 10 },
            height: 10,
            width: 'auto'
}));
