  • Well, there are two things here.

    1. Your userinfo variable is not an array. If you want it to be, simply surround it by quire brackets. ([])
    2. You should define a default value when you are invoking getList.

    Example:

    var data = Ti.App.Properties.getList('logininfo', []);

    This will return an empty array in cases where there are no property saved under the logininfo key.
