I tryed to use Titanium.UI.KEYBOARD_NUMBER_PAD Titanium.UI.KEYBOARD_NUMBERS_PUNCTUATION Titanium.UI.KEYBOARD_PHONE_PAD and Titanium.UI.KEYBOARD_DECIMAL_PAD all of them showed the same Keyboard, numbers with small alphanumeris (ABC,DEF…) and a Sym key that show the puctuation, slash, +, - and others.

Titanium: 3.2.0 with Alloy

  • This is not a "solution" per se but more of a workaround. I'm now detecting the manufacturer and if it is Samsung I change the keyboard to KEYBOARD_NUMBERS_PUNCTUATION.

    keyboardType: Ti.Platform.manufacturer.toLowerCase() === 'samsung' ? Ti.UI.KEYBOARD_NUMBERS_PUNCTUATION : Ti.UI.KEYBOARD_DECIMAL_PAD

    As far as I can tell there is no other way around it and users are OK with this.
