Build for older iOS sdk but run on iOS 8 simulator

Hi,

Is there any way to build my project for an older iOS version (iOS6 or iOS7) but run it on the iOS8 simulator?

I want to test if an already submitted app will have any problems running on iOS8. Currently I am successfully building with iOS8 beta and running on iOS8 iPhone simulator.

  • Hi

    Yes, assuming you have the other iOS SDKs still available;

    • Select your project in Studio
    • On the toolbar where you change device, select the menu
    • Choose Run configurations
    • Make sure your app is still the selected one
    • And the iOS entry is listed (should be if the last sim was iOS)
    • Choose a different iOS SDK form the drop down
    • Leave the Simulator set to 8

    If you do not have the other SDKs available you will need to start xcode and then download them.
