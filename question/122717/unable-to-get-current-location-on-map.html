  • I tested your code on the iPhone simulator by modifying the default mobile app cre ted by Titanium this way:

    • I put your all code in a file called 'toto.js' in the 'Resources' folder;
    • In the 'app.js' file, I modified win2 creation by adding a 'url' property:

    var win2 = Titanium.UI.createWindow({ title:&#39;Tab 2&#39;, backgroundColor:&#39;#fff&#39;, url: &#39;&#x2F;toto.js&#39; });

    It's working fine on iOS.
