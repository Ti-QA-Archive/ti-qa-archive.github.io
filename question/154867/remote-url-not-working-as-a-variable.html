    Hi

    You have asked another related question which gives a better way of providing the property than using regular expressions but this one is easily fixed.

    This reg has two parts

    ([a-z\-_0-9\/\:\.]*\.(jpg|jpeg|png|gif))
//  breaks down to
(one(two))

    So your result becomes;

    one,two

    The the result is provided as an array.

    You only need the first part which is actually the whole thing so change your line as follows;

    leftImage: "'"+str.match(patt1)+"'",
// becomes
leftImage: str.match(patt1)[0],

    You should not require the quotes either as the returned value is already a string.
