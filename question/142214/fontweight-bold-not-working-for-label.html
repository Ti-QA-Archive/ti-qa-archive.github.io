fontWeight bold not working for Label

I am using following Font Attributes but it is not making any effect to make it bold

font:{fontSize:24,fontFamily:'Lucida Grande',fontWeight:'bold'},

  • Accepted Answer

    if you are using custom fonts, you'll probably have to add one file for each style (regular, bold, italic…). In this case, the bold type is selected loading by its name, not by fontWeight property

    font:{fontSize:24,fontFamily:'Lucida Grande-Bold'},

    the exact name will depend on the font filename.

    • Yes. A browser will "fake" the bold font if the font package doesn't include a bold variant. However, native code will not. Since Lucida Grande doesn't include the bold variant in the font package, it must be declared separately.
