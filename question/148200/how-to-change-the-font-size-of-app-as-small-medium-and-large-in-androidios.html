    Hi

    If you are trying to provide various font sizes for the benefit of your user and not to simply provide multiple density support then - don't.

    You do not and should not include font size switching inside your app as an accessibility feature as your app should react to the built-in OS features that the user will set to make the changes across the entire device.

    Your tags indicate iOS support even if your dev setup does not support it so specific advice on that. Apple will almost certainly ban the app from being in the store if you included such a feature if the aim is to provide additional accessibility support which duplicates functionality already provided.

    Check the accessibility settings section of each device to see what I mean.
