default splash screen shows, not custom

I checked the entire project structure and there are no other splash screens. I've tried fresh installs in multiple languages. Where does the default splash come from, since I can't seem to be able to find the file to delete it? Using 3.1.0.GA on Android 2.2.3 and 4.0.4

    • I have just created a brand new project and added a custom splash screen and it works a treat.

      Create a set of folder in;

      {appname}/android/images/

      As follows;

      {appname}/android/images/res-ldpi
{appname}/android/images/res-hdpi
{appname}/android/images/res-mdpi
{appname}/android/images/res-xhdpi

      Then add your image in the respective folders and call it default.png

      Check out a previous answer I gave that goes further into images folders;
