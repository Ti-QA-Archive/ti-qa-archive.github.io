Get HTML of a WebView

Hello I have a WebView where I load some local html which queries a remote website and show results; the problem is that when the WebView is reloaded with the results is I access the html property is unchanged (has the form code).

I tried to load the page remotely but same problem, the html property is always undefined.

I tried also to use the webview.evalJS(" document.documentElement.innerHTML;"); but still returns undefined.

Any idea? I really need to solve this!

    Accessing the html property after the load event fires works for me perfectly on iOS. I catch it in an event listener to the load event. Maybe you have to wait for the load event to fire to get the actual html after your form submit.
