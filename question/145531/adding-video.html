    Hi Jay,

    Firefox support WEBM which you can create from MP4 with Firefogg.

    Video formats and conversion are a wide area and for getting some background you may
    look at this page for some good beginners video encoding tutorial stuff.

    You will find information about various supported video formats for desktop and mobile here.

    You can use a video format converter/encoder (google for that) and there are also cloud services like zencoder which will encode to target format on the fly. All depends on your needs and budget.

    Hope that helps,

    Matthias

    • Thanks for that Matthias.
      Btw I played little with the example code you provided me previously and it turned out that it worked very well.

      The issue was that we were actually providing two video links instead of one. So, we were providing link for the .mp4 format as well as .webm format.

      I believe that the code should be like this. 

      var vidWin = Titanium.UI.createWindow();
var vidView = Titanium.UI.createWebView();

var html = '<link href="http://vjs.zencdn.net/c/video-js.css" rel="stylesheet">';
html += '<script src="http://vjs.zencdn.net/c/video.js"></script>';
html += '</head>';
html += '<body>';
html += '<video id="my_video_1" class="video-js vjs-default-skin" controls';
html += '  preload="auto" width="640" height="264" poster="my_video_poster.png"';
html += '  data-setup="{}">';
html += '  <source src="media/movie.mp4" type="video/mp4">';
html += '</video>';
html += '</body></html>'

vidView.html = html;
vidWin.add(vidView);
vidWin.open();

      Works successfully for me.

    • You can also consider using for example something like Video.js inside a WebView.

      code example:

      var vidWin = Titanium.UI.createWindow();
var vidView = Titanium.UI.createWebView();

var html = '<html><head>Video View Demo';
html += '<link href="http://vjs.zencdn.net/c/video-js.css" rel="stylesheet">';
html += '<script src="http://vjs.zencdn.net/c/video.js"></script>';
html += '</head>';
html += '<body>';
html += '<video id="my_video_1" class="video-js vjs-default-skin" controls';
html += '  preload="auto" width="640" height="264" poster="my_video_poster.png"';
html += '  data-setup="{}">';
html += '  <source src="movie.mp4" type="video/mp4">';
html += '  <source src="movie.webm" type="video/webm">;'
html += '</video>';
html += '</body></html>'

vidView.html = html;
vidWin.add(vidView);
vidWin.open();
