    When i did the follwing 3 things, it worked:

    1st. Do something on command prompt:

    cd C:\android-sdk-windows\tools
mklink adb.exe ..\platform-tools\adb.exe
mklink AdbWinApi.dll ..\platform-tools\AdbWinApi.dll

    2nd. Exec the titanium studio by adminitrator level.

    3rd. Clear all of the avd(s), under the folder C:\Users\<your user name>.android\avd.
    After you clear, exec app you will get error "directory can not found"(near this), just create a folder under the avd folder like titanium_n_WXGA.avd, and then restart your app.

    Maybe helps.
