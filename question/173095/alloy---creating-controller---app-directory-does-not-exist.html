  • Stupid question first: This is an Alloy-based project, right? Check the project nature: from the Project menu, choose Properties. Then, select Project Natures. Alloy should be both checked and Primary.

    Have you tried at the command line? Inside the project's directory, alloy generate controller XYZ

    Are you using the Project Explorer view? Make sure you have a file within your project selected before attempting to create the controller. Try right-clicking on the app folder and choosing New, Alloy Controller (maybe it works when the command key shortcut doesn't).

    • Is your app is called 'app' and thus its directory is called 'app'? If so, that's probably the issue. The scripts are looking for a directory called app inside your app's folder and could be getting confused if your app's root folder is also named app.

      If that's not the case, what is your app's name, its folder name, and its app ID?
