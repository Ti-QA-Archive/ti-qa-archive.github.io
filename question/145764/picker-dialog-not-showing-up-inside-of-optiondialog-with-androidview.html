Picker dialog not showing up inside of OptionDialog with androidView

What I am trying to do is create a custom OptionDialog on Android. I create my OptionDialog and pass it the following androidView: 

var viewAddEvent = Ti.UI.createView({});

var v_ae = Ti.UI.createView({
width : 300, height: '100'
});
viewAddEvent.add(v_ae);

var data = [];
data[0]=Ti.UI.createPickerRow({title:'Bananas'});
data[1]=Ti.UI.createPickerRow({title:'Strawberries'});
data[2]=Ti.UI.createPickerRow({title:'Mangos'});
data[3]=Ti.UI.createPickerRow({title:'Grapes'});

var picker = Ti.UI.createPicker();
picker.selectionIndicator = true;
picker.add(data);

v_ae.add(picker);

The OptionDialog displays the picker fine, but when you click on it nothing happens. It does not display it's options. Is there something here that I am missing?

    backgroundColor:'#fff'

var viewAddEvent = Ti.UI.createView({});

var v_ae = Ti.UI.createView({
width : 300, height: '100'

v_ae.add(picker);
