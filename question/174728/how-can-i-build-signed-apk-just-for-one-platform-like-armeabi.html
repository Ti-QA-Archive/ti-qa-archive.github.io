How can I build signed APK just for one platform like armeabi?

Target OS: Android
Titanium SDK: 3.2.2GA

Hi, when I build an APK file Titanium builds for armeabi, armeabi-v7a, x86. For reducing file size I wish to avoid to build for platforms other than armeabi. I saw that the APK file is a simple ZIP file so if I open it with a ZIP manager I can remove from 'lib' the folders I don't need. In this way I can reduce the size of about 30%. and all works well. Do you know if this method is useble also with signed APK? Or do you know a method for build a signed APK just for one platform? Thanks for attention.

    So inside the android tag in tiapp.xml put something like

    <abi>armeabi-v7a,armeabi</abi>
