  • If you're getting a 

    Error extracting zip file
No such file or directory

    The problem is that /Library/Application Support/Titanium is owned by 'root' rather than by your user account.

    Run this

    sudo chown [your username].admin /Library/Application\ Support/Titanium/

    Replace [your username] with your short username. Should fix the problem.

    Can test to make sure it works by creating a test file.

    touch /Library/Application\ Support/Titanium/test.txt
