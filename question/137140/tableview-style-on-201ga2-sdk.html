TableView style on 2.0.1GA2 SDK

Hi all.

Are you having any problems setting the style of a tableView to a grouped table?

LEt me try to explain what is happening to me from the 2.0.1GA2 SDK:

This works:

var myTableView = Ti.UI.createTableView({
    rowHeight:57,
    style:Ti.UI.iPhone.TableViewStyle.GROUPED
});

This doesn't:

var platform = Ti.Platform.osname;

var myTableView = Ti.UI.createTableView({
    rowHeight:57
});

if(platform == 'iphone'){
    myTableView.style = Ti.UI.iPhone.TableViewStyle.GROUPED
};

What am I missing here??
