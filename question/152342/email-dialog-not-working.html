    Here is the full code of app.js:

    var win = Ti.UI.createWindow();
var button = Ti.UI.createButton({
    title: 'E-Mail'
});
button.addEventListener('click', function(){
    var emailDialog = Ti.UI.createEmailDialog();
    emailDialog.subject = "testSubject";
    emailDialog.messageBody = "testBody";
    emailDialog.open();
});
win.add(button);

win.open();

    Paste this into your app.js and run the project. I should work. If not, there is happening something else strange…
