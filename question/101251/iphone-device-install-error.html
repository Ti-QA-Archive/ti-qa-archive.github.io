  • I had the same problem and managed to work around it.

    I opened up the generated xcodeproj in xcode and noticed that the os was set to 4.1 even though I had specifically setup the application as 4.2 in titanium.
    I corrected this, cleaned and rebuilt and eventually got my app installed on my iphone.
