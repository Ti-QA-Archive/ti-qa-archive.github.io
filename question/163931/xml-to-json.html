    • I think your problem is the encoding of the BriefingContent node. It has formatted HTML which should be inside a CDATA wrapper.

      Example;

      <BriefingContent><![CDATA[<html>
        <head>
            <script/>
        <head>
        <body>
        Your HTML's body
        </body>
        </html>
     ]]>
</BriefingContent>

      ^^^ Not your actual code obviously.

      Without this declaration, the html tags conflict with the xml tags and it breaks.

      Also you fail to close the BriefingContent, order and orders tags before closing the data tag. Your xml is very malformed.

    • If you mean you have given up on converting xml to json, That is very strange.

      To check to see if a property exists in a json object after converting it from xml you can use a couple of methods;

      • First one;
      var json = {
    firstname: 'Malcolm',
    lastname: 'Hollingsworth'
};
alert(json.hasOwnProperty(lastname)); // true
alert(json.hasOwnProperty(ages)); // false
// value
alert(json['lastname') // Hollingsworth
      • Second one;
      var json = {
    firstname: 'Malcolm',
    lastname: 'Hollingsworth'
};
alert('lastname' in json); // true
alert('alert' in json); // false
// value
alert(json['lastname') // Hollingsworth
