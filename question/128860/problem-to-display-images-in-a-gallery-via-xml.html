  • Problem solved, thanks for your help. It was something in the com.codeboxed.imagegalley.js

    To create the thumbs they were creating a label with a background image like that

    _img = Ti.UI.createLabel({({
            backgroundImage: _images[i].path,
            width: 75,
            height: 75,
            myID: i,
            left: _rowPosition,
            top: _columnPosition
        });

    so I changed like that and it works

    _img = Ti.UI.createImageView({
            image:_images[i].path,
            width: 75,
            height: 75,
            myID: i,
            left: _rowPosition,
            top: _columnPosition
        });
