What to do with Appcelerator...

I have been developing an app for the last few months, using my android handset to test and overall it has been going pretty well, except the constant 'run on device' was frustrating for seeing the results of code changes. It was with GREAT ENTHUSIASM that I signed up as an Indie developer to start using an actual debugger to move development ahead more quickly… sadly the experience has been more than disappointing.

Initially, I had issues with getting the new TStudio up and running, but that was to be expected, I was learning and things were new… Now after three weeks of trial and error, I have decided that debugging my current project in TStudio is a waste of time and I'm back to code change/run on device/make a change/repeat. Not the ideal scenario, but the only one that seems to work.

The various issues I have had are as follows:

  • Initially couldn't figure out why ALL of my screens had changed layout. Turns out this was an artifact of some things Appcelerator had changed in 1.7 (see this posting for details New Defaults for android layouts in 1.7 So, it was frustrating, but at least there was a fix.

  • Database not installing on emulator. Again a weird problem that I never did figure out why it was happening. See my Q&A posting

  • Last Straw, I am trying to debug on the emulator, but when I set a breakpoint and switch to the 'Variable' view tab so I can try and see what is going on, TStudio fires an error Debug Error that causes the debugger to become USELESS because you can't see the values of ANYTHING you're trying to debug!

Now, I may be slow or daft, but the indie subscription does nothing for me as I can't get the help I need to either get this setup properly or figure out if this is a true product error or not. So it would be helpful if there was some way to submit issues around setup or errors or bugs that the system is raising (not my APP, again it works fine/GREAT as a matter of fact, on my handset).

Not sure where to go from here, what to do… If anyone from Appcelerator has any suggestions, I'm open to them. In the meantime I really feel like you have my money and I don't really have any way of getting the answers I need to move forward with the product.

Ray Belisle
