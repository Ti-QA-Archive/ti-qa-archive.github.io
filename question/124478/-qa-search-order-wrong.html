    Seb,

    Yes, we totally agree that there needs to be better sorting controls in this Q&A. We have it on our work list, but unfortunately other issues are taking priority at the moment. To help with your situation, you can actually add plus and minus symbol modifiers, to narrow down the search results. For example, to find all the questions that include a post by Kevin Whinnery and include the terms tableview and json but not database, enter +tableview +json -database +&quot;kevin whinnery&quot;.

    That said, before you make these venomously demands, it would be appreciated if you, yourself, would first respect the guidelines of this forum. Specifically, try to keep it friendly, along with the other suggestions in the Participating in the Q&A guide.

    Please stop answering your own questions - use the comment facility, unless you are providing a definitive answer.

    I thank you, in advance, for showing more consideration in future.
