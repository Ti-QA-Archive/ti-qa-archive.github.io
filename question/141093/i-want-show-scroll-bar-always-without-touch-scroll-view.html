  • Hi Ashish

    It is not possible to make the scrollbar always visible - you can create your own 'mock' scrollbar' and adjust its position when the user interacts - but that is a lot of work and not required.

    The great thing about modern phones is they are built using a common set of visual cues - the scroll region (and tableview) is one of those, the first time a user sees a screen with content that exceeds the visible space a visual scrollbar appears and then fades - this is what tells people, and if they do not know they very soon learn.

    Whilst I agree it might be nice to have the option - it is a problem not really needing a solution.
