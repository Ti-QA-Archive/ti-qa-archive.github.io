Ti.UI.MaskedImage class not found on device (Mobile SDK 3.1.3.GA)

I want to use a MaskedImage instead of an ImageView. This is in the template definition of a ListView. All works fine in the simulator but on the device the app always crashes with Ti.UI.MaskedImage class not found in the console logs. I rebuilt this project multiple times, manually removed the build folder, but still I cannot get this to work on the actual device (iPhone 5s with iOS 7.0.2) while it always works fine in the simulator. Any ideas?
