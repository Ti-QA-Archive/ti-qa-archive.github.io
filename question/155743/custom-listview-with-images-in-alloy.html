  • What have you tried so far? It helps if you provide your initial effort, rather than just requesting for someone else to do it. You could reference the API docs for ListView or the extensive ListView guide. Additionally there's multiple examples of ListView test apps in the Alloy repository that are supported in the current 1.2.0 alpha release. Here's one of them:

    https://github.com/appcelerator/alloy/tree/master/test/apps/ui/listview

    You can install the 1.2.0 alpha like this (it's currently on its 6th revision):

    [sudo] npm install -g alloy@1.2.0-alpha6

    And now that I've taught you how to go digging for info in the future, I'll be nice and provide an extremely basic example based on your extremely minimal description. (tested):

    index.xml

    <Alloy>
    <Window backgroundColor="#fff">
        <ListView defaultItemTemplate="template1">
            <Templates>
                <ItemTemplate name="template1" height="60">
                    <ImageView bindId="image1" left="5"/>
                    <ImageView bindId="image2" right="5"/>
                    <Label bindId="label1" top="5"/>
                    <Label bindId="label2" bottom="5"/>
                </ItemTemplate>
            </Templates>

            <ListSection id="section">
                <ListItem image1:image="/appicon.png" image2:image="/appicon.png"
                    label1:text="static label1 text" label2:text=" static label2 text"/>
            </ListSection>
        </ListView>
    </Window>
</Alloy>

    index.js

    var items = [];
for (var i = 1; i < 31; i++) {
    items.push({
        image1: {
            image: '/appicon.png'
        },
        image2: {
            image: '/appicon.png'
        },
        label1: {
            text: 'index ' + i + ': label1 text'
        },
        label2: {
            text: 'index ' + i + ': label2 text'
        }
    });
}

// add the new items to the end of the existing list
$.section.items = $.section.items.concat(items);

$.index.open();

    index.tss

    'Label': {
    left: 60 ,
    right: 60,
    height: Ti.UI.SIZE,
    color: '#000'
}

'ImageView': {
    height: 50,
    width: 50
}

'#index': {
    backgroundColor: '#fff',
    modal: false,
    exitOnClose: true
}

  • Hi Raqueeb,

    You have to make template of two imageViews and two labels.Like:

    var myTemplate = {
    childTemplates: [
        {                            // Image justified left
            type: 'Ti.UI.ImageView', // Use an image view for the image
            bindId: 'pic_1',           // Maps to a custom pic_1 property of the item data
            properties: {            // Sets the image view  properties
                width: '50dp', height: '50dp', left: 0
            }
        },
        {                            // Image justified left
            type: 'Ti.UI.ImageView', // Use an image view for the image
            bindId: 'pic_2',           // Maps to a custom pic_2 property of the item data
            properties: {            // Sets the image view  properties
                width: '50dp', height: '50dp', left: 0
            }
        },
        {                            // Title 
            type: 'Ti.UI.Label',     // Use a label for the title 
            bindId: 'info',          // Maps to a custom info property of the item data
            properties: {            // Sets the label properties
                color: 'black',
                font: { fontFamily:'Arial', fontSize: '20dp', fontWeight:'bold' },
                left: '60dp', top: 0,
            }
        },
        {                            // Subtitle
            type: 'Ti.UI.Label',     // Use a label for the subtitle
            bindId: 'es_info',       // Maps to a custom es_info property of the item data
            properties: {            // Sets the label properties
                color: 'gray',
                font: { fontFamily:'Arial', fontSize: '14dp' },
                left: '60dp', top: '25dp',
            }
        }
    ]
};

    I hope this helps you.For more details you can go through the Appc doc: http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.ListView
