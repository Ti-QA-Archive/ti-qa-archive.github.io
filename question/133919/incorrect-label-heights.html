Incorrect label heights

I've had a lot of trouble with label heights, when the text length isn't known. The height is often displayed wrong and results in overlapping labels.

I've found a few tricks to fix the problem, but it then gives other issues.

The solution seems to be setting width:&#39;auto&#39; as well as height:&#39;auto&#39;, but the problem is that adding an auto width then makes the label fill the width of the container… I can offset it with left:20 and right:20 - but the problem now is that any label that doesn't fill the 100% width is now centered within the containing view.

Is there not a way to set the label to align to the left of the container?

The only way I can think, is set another view within the container view with -20 off the width, then add the labels to this view and don't set a right value on the labels. This works fine, but seems far too overkill when I just wanted an equal space between my labels to begin with.

Thanks!

  • if you just want to align the labels one ontop of the other and adjust the height and width you are on the right track by setting the height to auto just add them to a container and set the lay out property to vertical

    var view = Titanium.UI.createView({
        height: win.height,
        width: win.width,
        layout: 'vertical'

    })
    var placeAddressLabel = Titanium.UI.createLabel({
        text:'texttextte xttexttex ttexttextte xttexttexttexttext texttexttextte xttexttex ttexttextte xttexttexttexttext texttexttextte xttexttex ttexttextte xttexttexttexttext text',
        left:'75dp',
        width:'auto',
        height:'auto',  
        font:{fontSize:'14dp'}                  
    });
    var checkInDate = Titanium.UI.createLabel({
        text:'texttextte xttexttex ttexttextte xttexttexttexttext texttexttextte xttexttex ttexttextte xttexttexttexttext texttexttextte xttexttex ttexttextte xttexttexttexttext text',
        left:'75dp',
        width:'auto',
        height:'auto', 
        font:{fontSize:'14dp'}       
    });
    view.add(placeAddressLabel);
    view.add(checkInDate);

    if you need more space between the labels just adjust the top property to the number you like
