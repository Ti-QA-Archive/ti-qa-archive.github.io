How to pass current view or window to titanium ios module.

I want to use the current view in my module.How to do pass current view to module.

  • Hi Vishnu,
    it's actually pretty simple to pass a View to an ios module

    Set up a method in your module, with a TiUIView Class parameter, then pass your Titanium View instance to that method.

    In your module:

    -(void) doSomething: (TiUIView*) view {
    // do whatever you want with view
    // for example:
    view.frame = CGRectMake(10, 10, 100, 100);

    // If you want to get the parent of your View, try something like this (not tested) 
    TiUIView *parentView = (TiUIView*)[view superview];
}

    in your Titanium App:

    var view = Ti.UI.createView();
var myModule = require('myModule');
myModule.doSomething(view);

    Hope this helps you to get started
