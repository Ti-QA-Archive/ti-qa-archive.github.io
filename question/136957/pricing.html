  • You dont have to worry, you can use Ti without being charged.

    What pricing basically gives you are:

    • bunch of modules 4 free (build by Ti)
    • support (employees of Ti)
    • more ACS API calls

    Making own API calls (http) does not affect this.
    In ACS its restricted, because you use their resources (traffic/power/maintenance/…).

  • Simply put:

    What you get for free:

    • You can develop and distribute applications for free (IDE included).
    • Appcelerator Cloud Services (ACS)
    1. (Quotas are per month)
    2. 1,250,000 Push and Email
      3, 250,000 Tier 1 API calls
    3. 250,000 Tier 2 API calls
    4. 5 GB storage
    • No modules from the Marketplace
    • Community support

    What you get by paying:

    • Bigger quotas on you ACS Plan (the more you pay, the more you get)
    • A set of modules from the Marketplace
    • Enterprise support (faster response time)
