  • Accepted Answer

    The Titanium Blackberry SDK directory is.. mobilesdk/win32/1.7.0RC1/blackberry/

    You may need to obtain the 1.7.0RC1 SDK from support, which is part of the Blackberry early access. This version will contain the BB plugins needed to build apps.
