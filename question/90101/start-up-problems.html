  2. Closed Titanium and then restarted it. Under SDK and Screen it says Loading… (forever). So I cannot duplicate the problem.

  3. If I select new project and then Mobile, then it shows me that I have both the Android and iOS SDKs installed. However, when I go to test and package only Android shows up under Run Emulator. I would expect to see iPhone as an option also.
