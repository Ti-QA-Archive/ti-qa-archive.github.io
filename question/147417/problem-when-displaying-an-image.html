Problem when displaying an image

Hello everyone,

I am trying to display an image according to a specific number, just like that:

This is the onload function after connecting to a remote php file:

onload : function(e){

         Ti.API.info(this.responseText);
         alert("Response Text: "+this.responseText);
         alert("Success!");

         var winres = Titanium.UI.createWindow({
             backgroundColor:"#FFFFFF",
             answer:this.responseText,
             url:"conn2.js"
         });

         winres.open();
        },

and this the code od the "conn2.js" file:

var win = Titanium.UI.currentWindow;

var valr = win.answer;

 var picres = Titanium.UI.createImageView({    
             top:50
         });

         if(valr<50){
             picres.setImage="images/pic1.png";
         }

         if(valr>50){
             picres.setImage="images/pic2.png";
         }

         win.add(picres);

the problem is that the picture didn't show up !!! the alerts which contains the value are fine, but after that I got just a white background!

Did I do something wrong?
Thanks you!

Application type: mobile

Titanium SDK: 3.0.0

Platform & version: Android 2.3.3

Device: Android emulator

Titanium Studio: Titanium Studio, build: 3.0.1.201212181159
