Beginner question: Layouts

Hi, I'm trying to get layouts working in XML-Markup. I got this:

index.xml:

<Alloy>
    <Window class="container">
        <View class="horizontalView">
            <Label text="Username: "/><TextField id="usernameTextField" maxLength="50" width="150"/>
        </View>
        <View class="horizontalView">
            <Label text="Password: "/><TextField id="passwordTextField" maxLength="50" width="150" passwordMask="true"/>
        </View>
        <View>
            <Button title="Login" onClick="login"/>
        </View>
    </Window>
</Alloy>

index.tss

".container": {
    backgroundColor:"white",
    layout:"vertical"
},
"Label": {
    width: Ti.UI.SIZE,
    height: Ti.UI.SIZE,
    color: "#000"
},
".horizontalView": {
    layout:"horizontal"
}

But I only see:
Username: [ ]

There is no Password label&textfield and button. Whats wrong?
cu Floh

  • Accepted Answer

    You should specify height and width properties of child views of vertical / horizontal containers…

    Adding this to your tss will fix the issue:

    "View" : {
    width: Ti.UI.FILL,
    height: Ti.UI.SIZE
}
