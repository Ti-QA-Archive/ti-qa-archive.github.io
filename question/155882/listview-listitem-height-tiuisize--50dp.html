Listview: Listitem height (Ti.UI.SIZE) + 50dp?

Hi,

I'm searching for a way to add 50dp to a listview listitem which is set to Ti.UI.SIZE height. I want to add some space if a certain value is set (image available) to put it below the text. I've set the height of each row to Ti.UI.SIZE to have it as height as the text is (which is working). But I can't find a way to add some height to it.

Is it possible to change add some space? toImage, rect.height and height isn't working/supported/giving the right result.
