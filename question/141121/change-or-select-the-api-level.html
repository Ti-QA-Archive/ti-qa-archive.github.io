Change or select the API level

It is possible select (or edit) the API level of a Project or Module Project, without create a custom manifest.xml?

I create a module a if I use API 10 don´t compile:

MediaRecorder.AudioEncoder.DEFAULT //API 1 OK
MediaRecorder.AudioEncoder.AAC //API 10 WRONG Don´t compile

  • HI Alberto

    You can edit your projects own tiapi.xml file and include (or update) this section.

    <android xmlns:android="http://schemas.android.com/apk/res/android">
   <tool-api-level>8</tool-api-level>
   <manifest android:installLocation="preferExternal">
      <uses-sdk android:minSdkVersion="7" />
   </manifest>
</android>

    Note the tool-api-level and minSdkVersion properties. More information can be found in the tiapp.xml and timodule.xml Reference
