  • sets the length, width of the label, and the label minimumFontSize

    in this way when the text of label exceeds the length (width) of the label that you set, the text gets smaller by only

    an example

    var name = Titanium.UI.createLabel({
                text : "test",
                left:10,
                color : '#FFFFFF',
                font : {
                    fontSize : 20,
                    fontWeight : 'bold'
                },
                height : 70,
                width : 270,
                minimumFontSize : 8
});
