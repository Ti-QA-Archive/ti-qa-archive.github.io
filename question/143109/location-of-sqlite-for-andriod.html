  • if you are using Mac then you must follow the following steps to find out the database.

    1. Open SQLite Browser
    2. From menu go to File->Open database
    3. open this directory
      (USER->Library->Application Support->Iphone Simulator->version (5.0 in my case)->Applications)
    4. In applications folder you can find some Temporary folders open them and check them one by one
    5. You can find your filled database here (From Applications->Temporary Folder Name -> Library->private documents)
