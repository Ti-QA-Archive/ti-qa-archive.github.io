Hi,

I have a question, with iOS you can simply handle an localnotification with:

Ti.App.iOS.addEventListener('notification',function(e) {});

But now i want to do the same for Android, i'm using this code the create an localnotification:

           var AppIntent = Ti.Android.createIntent({
                    action: Ti.Android.ACTION_MAIN,
                    packageName: Ti.App.id,

                    flags: Ti.Android.FLAG_ACTIVITY_RESET_TASK_IF_NEEDED | Ti.Android.FLAG_ACTIVITY_SINGLE_TOP,
                });

                AppIntent.addCategory(Ti.Android.CATEGORY_LAUNCHER);

                var NotificationClickAction = Ti.Android.createPendingIntent({
                    activity: Ti.Android.currentActivity,

                    flags: Ti.Android.FLAG_UPDATE_CURRENT,
                    type: Ti.Android.PENDING_INTENT_FOR_ACTIVITY

                Ti.Android.NotificationManager.notify(1, Ti.Android.createNotification({
                    contentTitle: Ti.App.name,
                    contentText: 'Hi there!',
                    tickerText: '',
                    icon: Ti.App.Android.R.drawable.appicon,
                    flags: Titanium.Android.ACTION_DEFAULT | Titanium.Android.FLAG_AUTO_CANCEL | Titanium.Android.FLAG_SHOW_LIGHTS,
                    contentIntent: NotificationClickAction,
                }));

Everything works perfect, but i need an eventlistener when this notification is beeing clicked from the home screen, does somebody has the answer?
