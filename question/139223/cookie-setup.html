I need to store the login information in cookies and need to retrieve that when user opens the application again..

how can i do this

    Here are not cookies, but you can store session infos in properties (or in files or in DB)

    Example:

    var options: {"firstname":'Otto',"brand": 'Trabant',"bornin":'GDR'}
Ti.App.Properties.setString("options",JSON.stringify(options));  //saving

    and with JSON.parse(Ti.App.Properties.getString(&quot;options&quot;); you can restore.

    If you want, can can store the properteis also single. In our example three properties.

    Hope this helps.

    Rainer
