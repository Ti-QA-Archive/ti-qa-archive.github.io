    • I found the problem.

      First I created a new project and compiled it to run on devices - just to make sure it works for a new project. It worked.

      Then I was compiling my projects one at a time. Some worked fine - some did not.

      So for the project that did not work I was checking all the files and found icon_x.js file in img/ folder. Somehow the icon_x.png file was renamed as .js file. And the file is not even used anymore in the app.

      So the tip here - remove all un-used files and check the rest of the files.

      I'll check other broken projects the same way now.

      I hope this is helpful.
