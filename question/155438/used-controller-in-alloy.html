Used controller in alloy?

Ex. i have 2 file controller a.js vs b.js. I want to access 1 function in b.js when i stay at a.js but not used Alloy.createController('b').getView().function(). How can i do it?

  • You should use exports object in controller.

    In b.js use 

    exports.myFunction = function(e){ 
    alert("Start");
}

    and from a.js use 

    Alloy.createController('b').myFunction();
