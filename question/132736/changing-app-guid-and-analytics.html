Changing App GUID and Analytics

I have an App that I originally created from an existing app as a starting point. The only problem is that I ended up using the same GUID as the original app. So, I suspect that Analytics is collecting data from both apps under the original apps name.

I deleted the GUID tag from tiapp.xml and Titanium Studio generated a new one on the next build. I also updated the manifest with that new GUID…but the app doesn't show up under my apps under my developer account.

Is there anything I can do in Studio to have it "phone home" to register this app under my account?

I know I can create a new app and paste my components into it, and I'll probably end up doing that. However, that feels so wrong. Seems like an update should trigger during the build following a change of tiapp.xml.
