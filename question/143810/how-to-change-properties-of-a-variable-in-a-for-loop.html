    BTW, you know that by using the same variable over and over in a loop. It will be considered as the same reference. So all objects will share the last updated properties since they all share the same name. (b)

    If you want to dynamically create objects in a loop, it is recommended to use an array. Then, you fill the array in the loop

    ####Like so 

    var buttons = [];

for (var i = 1; i < (x+1) ; i++){
    buttons.push(Titanium.UI.createButton(...));
}

    With this, each item from the array will be a different object.
