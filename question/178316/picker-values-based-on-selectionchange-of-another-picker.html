Picker values based on selection/change of another picker

How can I set the dynamic values of picker A after change in Picker B? How can I support this on android and IOS? I did not find any method but is there a way we could clear the picker items and reassign to some other values at runtime?

  • There's no clear method for pickers. You have to do it yourself something like:

    function clearPicker(_picker){
  if(_picker.columns[0]) {
    var _col = _picker.columns[0];
    var len = _col.rowCount;
    for(var x = len-1; x >= 0; x-- ){
      var _row = _col.rows[x];
      _col.removeRow(_row);
    }
  }
}

    Assigning a new set of rows is simple. Just do:

    // rows is an Array of Ti.UI.PickerRow items
picker.add(rows);

    Of course you can do this at the change event of another picker.
