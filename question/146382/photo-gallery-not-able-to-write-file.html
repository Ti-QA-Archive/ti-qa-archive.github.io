Photo Gallery not able to write file

I am able to open the photo gallery and select a file, but I am not able to successfully write the file. For example:

        Titanium.Media.openPhotoGallery({
             mediaTypes:[Ti.Media.MEDIA_TYPE_PHOTO],
            success:function(event)
            {
                var image = event.media;

                var file = Ti.Filesystem.getFile(components.dir, "detailsImage");

                var worked = file.write(image); 

                file.remoteBackup = false;

            },
            cancel:function(){},
            error:function(){},
            allowImageEditing:true
        });

the value of worked is always false. it doesn't matter if i give the file an extension or not. What I have found though is that even though the method is claiming to be false, if i rebuild the program the file is available and written correctly. any suggestions of how to fix the writing error?
