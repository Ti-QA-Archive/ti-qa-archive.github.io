Cannot nested View inside ImageView in Android.

When I have this markup in the view,

<ImageView id="image">
    <View id="crop_view"></View>
</ImageView>

In my Style,

"#image" : {
    width: '250dip',
        height: '300dip'
},
"#crop_view" : {
    borderRadius: "5dip",
    borderWidth: "1dip",
    borderColor: "red",
    width: "150dip",
    height: "200dip"
}

This is render as expected in iOS So I have Red Border Box on top of the image.
But in Android, this is not the case, the Red Border Box has not shown into the view.

So I just curious about how to have View render on top of the ImageView please.

  1. Application type: mobile
  2. Titanium SDK: 3.0.2.GA
  3. Platform & version: Android 4.2.2
  4. Device: Android emulator
  5. Host Operating System: OSX 10.8.4
  6. Titanium Studio: 3.1.0.201304151600

    in android, you can only add child to that parent view which is a container control so image view is not a container control in android instead you can set image in a view and then add another view which is crop_view in your case in a parent view.
