How to style tab bar?

Hi,
I've been looking around here in appcelerator's Q&A and the web for a good solution how to create at custom tab bar. The default is black with white text but a want to set the color of my own.

The best solution I came up with, which meet my demands is:

// create a view at the bottom of the window
var tabView = Titanium.UI.createView({
    height: 50,
    width: 1000,
    bottom: 0,
    left: 0,
    width: 320,
    height: 10

});

// create some buttons like the code below
var btnToDrink = Titanium.UI.createButton({
    title: 'to drink',
    bottom: 0,
    left: 0,
    width: 107,
    height: 50,
    style:Titanium.UI.iPhone.SystemButtonStyle.PLAIN,
    backgroundColor: '#a07936',
    font: {size:9, fontWeight:'bold'}
});

// add buttons to my view        
tabView.add(btnToDrink);
tabView.add(btnWinecella);
tabView.add(btnAddWine);

// add view to window
win.add(tabView);

Anybody who has a better solution?
Thanks in advance!
