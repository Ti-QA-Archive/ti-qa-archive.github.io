TI seeing Android SDK under Android but not under Titanium

I'm getting weirdness with the Android SDK. Under Preferences > Android the sdk is recognised at C:\Android but when I try to set it under Preferences > Titanium Studio > Titanium it says "Could not locate the Android SDK at the given path". I tried copying adb.exe to tools dir but no go. I installed the Google API as suggested in the blog mentioned in another post on this. No go. I've installed the recommended SDK for my version 1.7.2 so I'm at a loss. Any help here is appreciated.

    Execute SDK and AVD Manager and download all the updates needed, it could take long time depending of your connection and updates. After that try again. Make sure you have followed all the steps to install the requirements such as JVM, set System variables, etc. Then Titanium recognizes Android SDK.
