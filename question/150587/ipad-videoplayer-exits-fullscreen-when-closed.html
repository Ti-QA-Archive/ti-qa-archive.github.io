iPad videoplayer exits fullscreen when closed

I've developed a simple videoplayer-app to be used at an exhibition. It consists of six icons at the home page of the app, and when you press an icon a video is played. The home screen is ran in fullscreen mode. If i play a movie and it is finished (or someone presses the "Done"-button) with the video controls present, then the home screen of the app exits the full screen mode, and the title bar is shown.

Next time i play a video, it's like the videplayer window isn't filling up the entire screen anymore. It looks like it's having a bottom value the same height as the title bar, but it's shown at the bottom of the window.

The windows look the same no matter how many videos i play UNTIL i finish watching a video with the controls closed. Then the "bottom-value" of the video player disappears, and the titlebar in the home screen is hidden like it's supposed to.

Here is my code for the videoplayer

function playVideo(url, scale){
    var container = Ti.UI.createWindow({backgroundColor:"#000"});
    var videoPlayer = Ti.Media.createVideoPlayer({
            movieControlMode : Ti.Media.VIDEO_CONTROL_FULLSCREEN,
            mediaTypes       : Ti.Media.VIDEO_MEDIA_TYPE_VIDEO,
            autoplay         : true,
            sourceType       : Ti.Media.VIDEO_SOURCE_TYPE_FILE,
            scalingMode      : (scale) ? Ti.Media.VIDEO_SCALING_ASPECT_FILL : Ti.Media.VIDEO_SCALING_ASPECT_FIT,
            backgroundColor  : "#000"
    });

    videoPlayer.addEventListener("complete",function(e){
            videoPlayer.stop();
            videoPlayer.release();
            container.remove(videoPlayer);
            container.close();
    });

    container.add(videoPlayer);
    container.open();
    //container.orientationModes = [Titanium.UI.LANDSCAPE_LEFT,Titanium.UI.LANDSCAPE_RIGHT];
    videoPlayer.setUrl(url);
}

And i call the function like this

playVideo('myvideo.m4v',false);

I have tested the app on iOS6 and iOS4 and it turns out that this issue only occurs on iOS6.
SDK version is 3.0.2.GA
