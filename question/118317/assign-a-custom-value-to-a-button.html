assign a custom value to a button

I i was wondering what is the best way to assign a value to a button, so that when it gets clicked i can retrieve this value as a property of the button. So far i have something like:

answer3Btn = Ti.UI.createButton({
    title:answersArray[2],
    result:"result to question",
    width:300,
    height:50,
    top:270,
    left:100
});
answer3Btn.addEventiListener("click",function(e){
    alert(e.source.result);
});

but of course doesn't work

I have also tried:


answer3Btn = Ti.UI.createButton({
    title:answersArray[2],
    width:300,
    height:50,
    top:270,
    left:100
});
answer3Btn.result="answer to question";
answer3Btn.addEventiListener("click",function(e){
    alert(e.source.result);
});

The idea is that i have three button dynamically generated and when are clicked i need to know if it's the right answer to an random question, it's a multiple choice thingy.
Cheers.

    Hi,
    That's very strange. The way you are doing this is exactly the correct way to do it, adding a custom property and then reading it from e.source.property_name in the click event handler.

    I just tested your code and both ways work fine.

    There must be something else going wrong else where in your code.

    Out of interest, what happens if you replace the alert with :

    Ti.API.info('the button result value is : '+e.source.result);

    You should see the result in your debug window in Titanium developer under [INFO].

    Is your alert message popping up at all? Or are you seeing a lot of green error code in the debug window?

    If this is the case, try deleting the 'contents' of your [projectfolder]/build/iphone/ folder (leaving the 'iphone' folder there, just delete the contents of it) and this will force a full rebuild of your code.
