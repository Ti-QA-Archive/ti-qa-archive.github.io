Disable keyboard when switching between tab views on Android

Android is so annoying in that whenever I switch between tabs and there is a textfield in the window, it automatically shows the keyboard and focuses everytime. It's the most annoying thing to have it show everytime when you don't want it to. I have tried, txtField.blur in the 'open' event of the window, Ti.UI.Android.hideSoftKeyboard() and nothing works. Any suggestion please? Thanks!

  • You need to set the windowSoftInputMode property of the window to Ti.UI.Android.SOFT_INPUT_STATE_ALWAYS_HIDDEN, to keep the keyboard from automatically coming up when a textfield initially has focus. windowSoftInputMode must be set in the constructor of the window…not later:

    var TheWindow=Ti.UI.createWindow({
                  windowSoftInputMode:true,
                  ...other properties
                });

    The keyboard will still appear when the user taps a textfield.
