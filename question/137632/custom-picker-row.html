Custom picker row

Hi, in my iPad app i have a picker, every row of this picker is a country, now i want to add the country's flag on every rows, how can i do?

    Yes you can do it pretty easily. Check http://docs.appcelerator.com/titanium/2.0/index.html#!/api/Titanium.UI.PickerRow

    This code will work with new project in studio as it'll use the KS_nav_ui.png image that is included by default:

    
var win                 = Titanium.UI.currentWindow;

var column1 = Ti.UI.createPickerColumn();
var countries = [ 'Usa', 'China', 'Estonia', 'Mexico' ];
for(var i=0; i<countries.length; i++){
  var row = Ti.UI.createPickerRow();

 var label = Ti.UI.createLabel({
    text: countries[i],
    width:'126',
    left:50
 });

  row.add(label);
  var img = Ti.UI.createImageView({
      image:"images/KS_nav_ui.png",
      width:46,
      height:43,
      left:0,
      top:0
  });
  row.add(img);
  column1.addRow(row);
}

var picker = Ti.UI.createPicker({
      bottom:0,
    columns: [column1],
      visibleItems: 3,
      selectionIndicator: true
});

win.add(picker);

    Enjoy.

  • You can add an imageView to a pickerRow (IOS only)

                var row = Ti.UI.createPickerRow({
                title : 'Title'
            });

            var imageView =  Ti.UI.createImageView({
                    image : '<path_to_image>',
                    width : <value>,
                    height : <value>
                });
            row.add(imageView);
