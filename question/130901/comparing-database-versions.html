Comparing Database Versions

Hello all! I'm back with another question.

I'm developing for mobile apps, but I think this question applies across the board.

My app is pulling information from a database online. I have all that code working. The problem is, I don't want my app to re-download the database when there is no new information. So, I'm using Titanium.App.Properties to store a database version.

What is the best way to keep track of the version of the database on my servers? It's a MySQL database.

Thanks!

  • Update - I found this query that returns quite a few columns of information, on of which is UPDATE_TIME, but only as it applies to a table. And it only works for innodb types of db's.

    SHOW TABLE STATUS FROM your_database_name LIKE  'your_table_name'

    Another one that goes right into the information_schema, as long as you have privileges:

    SELECT UPDATE_TIME
FROM   information_schema.tables
WHERE  TABLE_SCHEMA = 'dbname'
   AND TABLE_NAME = 'tabname'

    but I believe it only works on innodb as well.
