  • Your best approach would be to simply append a new row to your table when the iser clicks the Send button.

    Here is what the event handler would look like:

    sendButton.addEventListener('click', function() {
    var newRow = Ti.UI.createTebleViewRow({
        title: myTextBox.value;
    });
    table.appendRow(newRow);    
});

    Of course, if you want to actually send data to a chat server component, you would have to do these tasks in the event handler as well.
