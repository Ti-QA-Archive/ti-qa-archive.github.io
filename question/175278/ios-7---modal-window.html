iOS 7 - modal window

I remember if I set modal:true, the simple window (not in Navigation or Tab Group) will have a title bar. Is this behavior changed? Now I set modal:true when opening the window, there is no title bar at all. Tested on iOS 7, SDK 3.2.3.

— asked June 17th 2014 by Shawn Lan

  • Accepted Answer

    This behaviour changed with iOS7, SDK 3.1.3 and the introduction of Navigation Windows described here.

    You need to wrap the window in a Navigation Window now.

    <Alloy>
    <NavigationWindow>
        <Window id="myWindow" title="My Window">
        ...
        </Window>
    </NavigationWindow>
</Alloy>
