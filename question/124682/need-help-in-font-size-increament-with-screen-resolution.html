    You just meed to specify the fontSize property using density independent pixels, via the suffix 'dp'. For exmaple:

    var label = Ti.UI.createLabel({
    text:'test',
    font: {
        fontSize:'18dp'
    }
});

    More details can be found in this Appcelerator developer blog post: http://developer.appcelerator.com/blog/2011/06/new-defaults-for-android-layouts-in-1-7.html
