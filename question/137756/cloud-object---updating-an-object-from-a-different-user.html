Cloud Object - updating an object from a different user

I have a situation where i had created a cloud objectb and need to update that object, but from a different user then who had created it. When i try to update from a different user i get an error 'you do not permission to update this object'.

Does anyone know if there is a way around this…?

Thanks

    I think the point of requiring user login for creating and updating objects is to prevent one user from changing another user's data.

    Users can add reviews or comments to other user's objects (using the Review object), but they can't create or update objects for another user.
