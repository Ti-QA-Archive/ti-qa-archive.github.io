Finding a field in a form.

I am unfamiliar with how one would go about finding a field in a 'UI.View' based on a property of the field.
IE: Inside my view i have 10 fields, I need to retrieve the textfield with an id of 'mytxtfield'.
how does one go about this in titanium ?

Thanks in advance.

  • Accepted Answer

    That text field object has a name. Reference it by name.

    var MyTextField = Titanium.UI.createTextField({
    color:'#336699',
    height:35,
    top:10,
    left:10,
    width:250,
    borderStyle:Titanium.UI.INPUT_BORDERSTYLE_ROUNDED
});

    To get the value of that would be

    var myTextFieldValue = MyTextField.value;

    • Objects added to TheView are children.

      The first item added TheView.add(textfield1) would be TheView.children[0]

      You can access the number of children with TheView.children.length

      Using that value you can write a loop to go through each and try to match MyView.id === &#39;mytxtfield&#39;
