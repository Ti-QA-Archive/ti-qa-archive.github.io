Hi All,

Scrolling if we give tableVIew height : Ti.UI.SIZE if we give tableView height is LIke 420 now scrolling

  • Hi Balakrishna,

    If you set height:Ti.UI.SIZE then it will not scroll because it's default behavior. it scroll only when it's content height it greater then it's actual height but if you set it's actual height is Ti.UI.SIZE it means content height == it's actual height..

    but you can scroll it using scroll view.. Add your tableview into scrollview.

    if you have any doubt let me know.
