I installed xcode 4.2 with the iOS5 sdk, and titanium still works, etc etc.

However, I can't submit apps, because Apple isnt allowing it with that iOS version. Can I install JUST the iOS 4.3 sdk alongside, so I can choose it when I'm compiling?   • Accepted Answer

    It is more than the SDK. Apple isn't accepting apps compiled with xCode 4.2 even if you target iOS 4.3 instead of iOS 5.0. You can check the apple dev forums for a couple of threads concerning this.
