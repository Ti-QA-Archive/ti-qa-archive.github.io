xml parsing broken on 1.8.1 ?

In my app I'm parsing an xml document with an xhr request.

Using sdk 1.7.6

var xml = this.responseXML;
var elements = xml.documentElement.getElementsByTagName("elements");
Ti.API.info(elements.length); //return 1, it is correct

Using sdk 1.8.1

var xml = this.responseXML;
var elements = xml.documentElement.getElementsByTagName("elements");
Ti.API.info(elements.length); //return 0, impossible to to proceed with the rest of parsing

this is the xml structure

<?xml version="1.0" encoding="UTF-8"?>
<elements version="1">
  <element name="element_one">
    <master>text</master>
    <node>text</node>
  </element>
  <element name="element_two">
    <master>text</master>
    <node>text</node>
  </element>
  ..
  ..
</elements>

Why???
