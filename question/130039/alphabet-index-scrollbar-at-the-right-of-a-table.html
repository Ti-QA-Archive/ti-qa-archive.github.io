  • I'm adding an answer for future Googlers of this issue, as the previous answers ("Check KitchenSink!") aren't particularly helpful.

    The KitchenSink section to check is Base UI > Table Views > Headers with Filter. It's iOS only, and the source is in Resources/ui/handheld/ios/baseui/table_views_header_filter.js

    Basically, you need to have the rows in alpha order already, adding a 'header' property to the first row of each section. Then you set up an index, which an array of objects, which container two properties: title, which is the text on the right of the screen (for Alpha, it's "A", "B", "C", etc.) and index, which indicates what row index each section starts at. Then you set that index array to the tableview.index property.
