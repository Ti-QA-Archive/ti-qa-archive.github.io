    • is pretty simple, I'll give you some ideas but you'll have to implement it yourself.

      So create the cancel and the ok image as in the tutorial.
      The only thing you need to add to each of them is the role it plays with your own variable.

      var ok_img = Ti.UI.createImageview({
    image:'ok.png',
    right:10,
    is_ok:true // <- this is the role
});

var cancel_img = Ti.UI.createImageview({
    image:'cancel.png',
    left:10,
    is_cancel:true // <- this is the role
});

      Now depending on your logic you can add the event on the row ar on the table

      var row = Ti.UI.createTableViewRow({
    className:'my_row' // this is to optimize the rendering
});

row.add(ok_img);
row.add(cancel_img);



row.addEventListener('click', function(e) {

    if (e.source.is_cancel) {
        alert('Cancel clicked');
    }
    else if {
        (e.source.is_ok) alert('Ok clicked');
    }
    else {
        alert('Row clicked');
    }

});

      Makes sense?
