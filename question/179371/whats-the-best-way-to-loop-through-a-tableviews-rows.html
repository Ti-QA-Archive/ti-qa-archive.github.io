What's the best way to loop through a TableView's rows?

Hi folks,

What's the best way to loop through TableViewRows and perform some functions on the controls they contain? For example, right now I'm working on a checklist feature which will change the background color of a specific switch, depending on whether or not it is checked. I believe I know how to do this upon initially populating the table, and upon changing the state of one button. But I have a clear button which reverts checklists to the default state of everything being unchecked, and I need to reset the color of every switch accordingly. How can I loop through the rows and update the switch background color when the checklist is cleared?

Alternatively, is there an event listener that would accomplish this? My table is populated using a collection, would the onChange event pick up on a collection.fetch changing a switch's state for instance?

I'm guessing the solution I seek is relatively context agnostic, but I can provide code if needed.

    Iterating over the rows in a table is covered in the docs. However, I know from experience that it can be very slow, especially on Android. Depending on your needs, you might be better off updating the source data and resetting the table's contents entirely.
