Country Code

Hi,
I am developing an app for both android and iOS where I want to use country code to display terms and conditions. I used Ti.Locale.getCurrentCountry() to get the country code. In iOS it is fine, but when it comes to android it using Device Language to get the current Country. But I don't to use that.
SO, Please help me how to get the country code in android without using location services.

  • Maybe you need to read the android docs a bit, to understand the purpose of Locale.

    http://developer.android.com/reference/java/util/Locale.html

    Ti.Locale.getCurrentCountry() wraps the following native code:

    Locale.getDefault().getCountry();

    From the android docs:
    Locale represents a language/country/variant combination.

    but when it comes to android it using Device Language to get the current Country

    Exactly as the android docs indicate.
