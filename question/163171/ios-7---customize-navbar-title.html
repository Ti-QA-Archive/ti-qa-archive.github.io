iOS 7 - Customize NavBar Title

Hi all, how can I customize navbar title in a webview? 

    winOffer = Titanium.UI.createWindow({
        title: L('offer'),
        backgroundColor: '#fff'
    });

I would like change the color title and the background color; how can I do?

— asked March 4th 2014 by henry manco

    Hi,

    You can try this code to customise nav bar title: for doing this you can play with titleControl property and navTintColor, and to change background colour of title bar you can use barcolor property.

    var winOffer = Titanium.UI.createWindow({
    backgroundColor: 'white',
    titleControl : Ti.UI.createLabel({
            text : L("offer"),
            Color : '#FFFFFF',
        }),
    barColor : '#385292',
    navTintColor : '#FFFFFF',
    backgroundColor : "#EFEFF4",

});
