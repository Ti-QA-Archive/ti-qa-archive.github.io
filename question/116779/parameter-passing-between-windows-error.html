Parameter Passing Between Windows Error

This may be a really simple answer, but I can't seem to fix it. I am trying to pass a parameter between two javascript files. In contacts.js, I run the following code:

var win = Titanium.UI.createWindow({
                url:e.rowData.test, //contactsPage.js
                title:e.rowData.title,
                selectedIndex:this.selectedIndex
            });
        }        
        Titanium.UI.currentTab.open(win,{animated:true});

Then in the page it opens, contactsPage.js, I declare the variable, but it never gets changed and stays at whatever I initialized it to. What am I missing?

var selectedIndex = 0;

Note: I also tried the method as described in http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.UI.Window-object, but no luck.

    I think what you want, if I understand it right, is the property selectedIndex of your window object, not a variable selectedIndex in the file: they're not the same thing.

    Assuming you're doing something like:

    var win = Ti.UI.currentWindow;

    in your contactsPage.js, you can then access it as win.selectedIndex, not var selectedIndex.
