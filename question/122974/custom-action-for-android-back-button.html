Custom action for android back button

Is there anyway to add a custom event on the android back button?
I've built a custom tabgroup for my app, and its all based on lightweight windows.
So I would like to be able to add a custom event, or add a window to the 'back' history. is that possible?

    You cannot replace the events that take place when any given button is used, however you can listen for those events, and add your custom functionality. For example, listening for the 'android:back' event will allow you to run your custom functionality and then handle the action intuitively. Does this make sense?
