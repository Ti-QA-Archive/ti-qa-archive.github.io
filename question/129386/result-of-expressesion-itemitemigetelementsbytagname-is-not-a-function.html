Result of expressesion item.item(i).getElementsByTagName is not a function

Hello!

I have i weird issue when im running my app on iOS. I am loading an array of data from an XML-file using this code 

        var i = 0;
        var readFile = Ti.Filesystem.getFile('compositions.xml');
        var doc = Titanium.XML.parseString(readFile.read().text);
        var items = doc.getElementsByTagName("composition");
        for (i = 0; i < items.length; i++)
        {
            var artist = items.item(i).getElementsByTagName('Artist').item(0).text;
            var imageURL = items.item(i).getElementsByTagName('Image').item(0).text;
            var artistDescription = items.item(i).getElementsByTagName('AboutArtist').item(0).text;
            var compositionTitle = items.item(i).getElementsByTagName('CompositionTitle').item(0).text;
            var lon = items.item(i).getElementsByTagName('longitude').item(0).text;
            var lat = items.item(i).getElementsByTagName('latitude').item(0).text;
            var ID = items.item(i).getElementsByTagName('ID').item(0).text;
            var aboutComposition = items.item(i).getElementsByTagName('AboutComposition').item(0).text;
            var distance = getDistance(lati,lat,longi,lon);


            data.push({
                artist: artist,
                imageURL: imageURL,
                artistDescription: artistDescription,
                compositionTitle: compositionTitle,
                longitude: lon,
                latitude: lat,
                ID: ID,
                aboutComposition: aboutComposition,
                distance: distance
            });
        }

But when im running the app it gets stuck on the "var lon" row with the error 

Result of expression 'items.item(i).getElementsByTagName'[] is not a function.

I've tried replacing the function that picks up the data from the XML file with a static value and then it works great…
Also tried cleaning the project and rebuilding it with no luck :(
The two xml-rows which i tries to get data from looks like this

        <longitude>19.627177</longitude>
