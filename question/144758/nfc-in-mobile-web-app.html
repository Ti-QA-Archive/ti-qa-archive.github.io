NFC in Mobile Web App

I am developing a mobile app for both iPhone and Android. I would prefer to configure it as a mobile web app - I already know HTML/JS - and that will save learning new platform. I want the app to also work without internet, so all html/js will be saved offline.

My question is: can I use NFC within the mobile web app framework. I know it only works on Android, and not iPhone - can an API be used for that within HTML/JS or a workaround.

— asked November 15th 2012 by Blah Linens

  • In short, you can't. Titanium Mobile Web is designed for mobile web browsers and since there is no NFC web API, no browser supports it. You could write an Android app with a WebView and have your WebView send messages to your Titanium code that talks to the NFC APIs.
