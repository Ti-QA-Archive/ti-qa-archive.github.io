Newbie Button Question

Just trying to figure out why this doesn't work:

if (b2.title == 'Turn On') 
  {alert("The toggle is now on")};
  {b2.title = 'Turn Off'};
elseif (b2.title == 'Turn Off') 
  alert("The toggle is now off");
  {b2.title = 'Turn On'}; 
});

b2 is a button with a default title of 'Turn On'
click it once, and it switches to 'Turn Off' as I want it to.
click it again, and it stays on 'Turn Off'

  • var btn = Ti.UI.createButton({title:'Turn On'});
btn.addEventListener('click', function(e) {
    if (e.source.title == 'Turn On') 
    {
        alert("The toggle is now on");
        e.source.title = 'Turn Off';
    }
    else
    {
        alert("The toggle is now off");
        e.source.title = 'Turn On';
    } 
});
