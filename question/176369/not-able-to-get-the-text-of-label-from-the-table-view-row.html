  $.pc_listtable.addEventListener('click', function(e){

    Hi

    In this case your child is actually a grandchild.

    TableViewRow
    View // child of row
        Label // child of View

    So this would be the way to access the grandchild.

    e.row.children[0].children[0].getText();
