Get the name of a photo that has just been taken?

Hey Everyone,

I am taking a photo using the built in camera on my iPad, everything is working fine, but I am unsure how
to get the name of the photo after it has been taken?
I do not want to change the name, I just want the name given to the photo by the device.

Thanks.

    When the media is returned from the camera, if you have set it to save to the gallery, it does that, then hands you the blob. You are responsible for managing that blob, saving it to a file. When you pull an existing image from the gallery, same thing. Apple will not give you direct access to the file (sandboxing) but will return the blob.
