  • Eduardo, please put your answers as answers not comments so that the original questioner can mark the issue resolved if it fixed their problem.

    The link Eduardo pointed you to http://developer.appcelerator.com/question/159730/unable-to-import-the-kitchen-sink-project mentions that git is not installed by default on Windows. You have to install is separately. But I also provide installation instructions for the Kitchen Sink that don't involve installing git.

    1. Visit https://github.com/appcelerator-developer-relations/KitchenSink
    2. Click the Download Zip button
    3. Unzip the resulting file into your Titanium_Mobile_Workspace folder
    4. In Titanium Studio, choose File, Import. Expand Titanium and select Titanium Mobile Project, then click Next.
    5. Browse to your KitchenSink folder and click Finish. Accept the defaults
