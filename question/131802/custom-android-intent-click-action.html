Custom Android intent click action

Hi,
I've implemented c2dm in my app - everything is fine (but I had to go a long way before I could write this :-) ).

The push arrives from the server with the data I want and activates the callback function with that data.

In the callback function I create a notification for the user:

var AppIntent=Ti.Android.createIntent({
    flags: Ti.Android.FLAG_ACTIVITY_CLEAR_TOP | Ti.Android.FLAG_ACTIVITY_SINGLE_TOP,
    className:'org.appcelerator.titanium.TiActivity',
    packageName:Ti.App.id
});
AppIntent.addCategory(Ti.Android.CATEGORY_LAUNCHER);

var NotificationClickAction=Ti.Android.createPendingIntent({
    activity:Ti.Android.currentActivity,
    intent:AppIntent,  
    flags:Ti.Android.FLAG_UPDATE_CURRENT,
    type:Ti.Android.PENDING_INTENT_FOR_ACTIVITY
});

var NotificationMembers = {
    contentTitle: 'New POV message',
    contentText: e.data.message,
    tickerText: "New POV message",
    icon: 0x7f020000,
    contentIntent: NotificationClickAction
};

Ti.Android.NotificationManager.notify(2, Ti.Android.createNotification(NotificationMembers));

This notification brings the focus to the app.

What I want to do is once the notification is clicked, in addition to giving the app focus is call another JS function I have that does something related to the data received by the push notification - my own custom action.

Can this be done? if so, how?

Thanks
