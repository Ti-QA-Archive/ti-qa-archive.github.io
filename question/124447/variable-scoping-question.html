Variable scoping question

I I declare a variable in a js file, and then include this file in all the views of my application, will changes to this variable in one view be carried into all views?

In other words, if I have var myList = [] in the include, and in view 1 a button handler sets myList.push(&#39;foo&#39;) and then in view 2 there is a label that is supposed to display myList[0] will this display "foo"?

iOS4, SDK 1.7.2, Studio 1.0 OS 10.7.1
