How to get a short app name on home screen with long app name everywhere else?

The name of my app is long and displays with ellipsis on the home screen (beneath the app icon). Therefore, I want to use an app short name ONLY on the home screen. I want the full name to display everywhere else.

I tried dropping a custom Info.plist in my project root and set the following properties:

<key>CFBundleDisplayName</key>
<string>Long Name</string>

<key>CFBundleName</key>
<string>Short Name</string>

This does not work. The app uses the CFBundleDisplayName value everywhere, including the home screen.

What is the proper way to get a short app name on the home screen with a longer app name everywhere else? If this cannot be done in Titanium, how do I do it directly in Xcode?
