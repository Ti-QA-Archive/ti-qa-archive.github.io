Formatting (strong, italic, new lines) a label

Hi all :)
Is there a way to formatting with bold, italic, new lines on a text of a label?

For example:

var label = Ti.UI.createLabel({
    text:'Lorem Ipsum Lorem Ipsum Lorem Ipsum Lorem Ipsum Lorem Ipsum Lorem Ipsum Lorem Ipsum Lorem Ipsum Lorem Ipsum Lorem Ipsum Lorem Ipsum Lorem Ipsum Lorem Ipsum Lorem Ipsum ',
    textAlign:'center',
    font:{
        fontFamily:'Georgia',
        fontWeight:'bold',
        fontColr:'#333',
        fontSize:12
    }
});

Thank you very much :)

    Different font for a new line of a the same label?
    If this is what you want this is not possible.
    Each label have the same font attribute, you can't define a different font attribute for a new line.
