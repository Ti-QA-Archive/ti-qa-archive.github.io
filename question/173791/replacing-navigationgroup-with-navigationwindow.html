  • Hi

    I am confused by this question as your code suggests you currently have a navigation group inside a tab - is that correct?

    Each Tab automatically has a navigation controller so you simply call tab.open(childwindow), this gives the title bar, left and right nav buttons and automatic back button when needed.

    The NavigationWindow now replaces the NavigiationGroup as you know. The big different is that NavigiationWIndow IS a window in its own right so you open it and use any rules like full screen and modal as you wish. You are required to provide a root child window.

    The previous NavigationGroup was added to AS a child container to a window and then a root child window.

    So using a NavigationGroup or NavigationWindow inside a template is pointless.

    Have you tried the Tab focus & blur events - NOT on the windows? This should tell you when the tab is pressed and automatically shows the chosen tab.
