Styling upper border radius

Hey,

how would i acomplish this?
https://www.dropbox.com/s/6gyb2xwggf9qeu8/Screenshot%202013-12-02%2021.35.40.png

As you see, those are two text fields divided by a simple grey line. The upper input has a upper border radius, the lower input has bottom border radius.
Those two controls should have around 10dp / 10px space to the left and to the right.

How would the XML and TSS look like?
I'm stuck with this:

XML:

<Alloy>
<Window id="container">
<View id="formWrapper">
<TextField id="username" hintText="Username" />
<TextField id="password" hintText="Password" passwordMask="true" />
</View>
</Window>

</Alloy>

TSS:

"#formWrapper": {
width: Ti.UI.SIZE,
height: Ti.UI.SIZE,
top:100,
}

"#username":{
backgroundColor: '#ffffff',
borderRadius: 2,
borderWidth: 0,
color: "black",
left: 10, right: 10,
width:Ti.UI.FILL,
height:50
}

"#password":{
backgroundColor: '#ffffff',
borderRadius: 2,
borderWidth: 0,
color: "black",
top: 97,
left: 10, right: 10,
width:Ti.UI.FILL,
height:50

}

  <TextField/>
  <View id="divider"></View>
  <Textfield/>
</View>

    style and size as needed
