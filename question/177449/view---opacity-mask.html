    var border = Ti.UI.createView({
    width: "80%", height: 4,
    backgroundGradient: {
        startPoint: {x: "0%", y: "50%"},
        endPoint: {x: "100%", y: "50%"},
        colors: [
            {color: "transparent", offset: 0},
            {color: "#7B999999", offset: 0.5}, // white color, +- 0.5 opacity
            {color: "transparent", offset: 1}
        ]
   }

    Hi Matej,

    i think i figured it out, the dark color comes from two sources:

    1. the color you chose 999999 is a grey color and already kind of dark,

    2. the so called "transparent" color: "transparent" seems to be a transparent black color, in hex this would be #00000000. If you want to fade into a grey or white color you have to use 8 digits hex colors everywhere.

    here is example for white:

    var border = Ti.UI.createView({
    width: "80%", height: 4,
    backgroundGradient: {
        startPoint: {x: "0%", y: "50%"},
        endPoint: {x: "100%", y: "50%"},
        colors: [
            {color: "#00FFFFFF", offset: 0},
            {color: "#B3FFFFFF", offset: 0.5}, 
            {color: "#00FFFFFF", offset: 1}
        ]
   }
});
