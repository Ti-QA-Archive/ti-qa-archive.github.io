  • Hey Joel I don't believe thats possible. You have to leave it up to the user. What I do is I create a prefs screen where users can "enable" an app from a list of apps. This way if the app is enabled I will send the user to that app using the apps custom URL scheme. (note not all apps use this feature though). Its commonly used for SAFARI to PUSH a url to an app.
