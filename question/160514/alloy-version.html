Alloy Version

I noticed Alloy version 1.3.0 was released however how can I even tell what version of Alloy I currently have running? When I run an app on a Android Device I see this in the build log:

[INFO] plugin=C:\Users\UsernameAppData\Roaming\Titanium\plugins\ti.alloy\1.0\plugin.py

[INFO] Detected compiler plugin: ti.alloy/1.0

However if I look in C:\Users\UsernameAppData\Roaming\Titanium\ there isn't even a plugins folder. For all I know I am still running version 1.0 of alloy that is what the log info above would suggest. Is there anyway to update it through Titanium Studio? I have gone to Help -> Check for Titanium Updates and it doesn't show any. So do I actually have alloy version 1.3 but the logger is incorrect in its output or am I not getting an update or something else? Any idea?
