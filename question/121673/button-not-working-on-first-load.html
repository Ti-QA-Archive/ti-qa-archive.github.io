Button Not Working on First Load

I have a weird problem. I have a button that will not click when I display a window that contains following button:

var prod_button = Titanium.UI.createButton({
width:81,
height:21,
top:125,
left:150,
backgroundImage:'images/OnlineButton.png',title:'View Online',
font:{fontSize:10,fontWeight:'bold'},
});

prod_button.addEventListener('click',function(e)
{
Titanium.Platform.openURL('http://www.url.com');
});

The weird thing is if I go back and reopen the window the button works. This happens every time. No matter how long I leave the app open the button works, but the first time the window is displayed it does not.

I have tried it on iPhone simulator/app and Android simulator/app. Is this common?

Thanks

Jon

  • Jon

    One problem is that there is a validation error in your code. The following code works fine for me every time. Let me know if it also does for you.

    Note: running on Titanium 1.7.1 (2011/06/17 00:13 293a6d…) and Android 2.1 (APIs).

    Ti.UI.backgroundColor = "white";
var win = Ti.UI.createWindow({
  exitOnClose: true
});

var prod_button = Ti.UI.createButton({
  width:81,
  height:21,
  top:125,
  left:150,
  // backgroundImage:'images/OnlineButton.png',
  title:'View Online',
  font:{fontSize:10,fontWeight:'bold'}
});

win.add(prod_button);
win.open();
prod_button.addEventListener('click',function(e){
  Ti.Platform.openURL('http://www.google.com');
});
