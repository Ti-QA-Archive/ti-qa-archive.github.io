  • Alloy, for the sake of performance and simplicity, does not manage its own DOM. For this reason, you can only query the controller $ for elements by id. There may be a lighter-weight solution to achieve a similar functionality, but yes, I would need a feature request logged for that, which could be done here: Titanium Community Jira Project
