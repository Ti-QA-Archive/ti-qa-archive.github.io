Example code for Android push notifications?

There is plenty of documentation and Q&A iPhone push notifications going. It only took me a few hours to wrap my head around the entire process, add push notification to an example app and write my simple own push notification PHP server. Very easy. However, I'm having a really hard time with Android push notifications. I've spent a few hours looking at android documentation, Titanium documentation (there is none for android push), and this Q&A section, but I can't figure out if push notifications are supported by Titanium yet, and if so, what needs to be done to register with a C2DM server, etc. There are a staggering number of unanswered questions on this site about this, and any info I do find seems to be about a year old now. Are push notifications supported with Titanium? If so, does anyone know of tutorials or example code?
