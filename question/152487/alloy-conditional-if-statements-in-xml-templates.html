Alloy conditional (if) statements in XML templates

How do I use if statements / conditionals in the Alloy XML templates? And if they don't exist, what are the best ways of substituting this?

  • No, there is no current syntax for doing arbitrary "if" statements in the XML. There are however currently 2 conditional attributes you can use in the XML: "platform" and "formFactor". You can read about them in more detail in the docs: http://docs.appcelerator.com/titanium/latest/#!/guide/Alloy_XML_Markup-section-35621528_AlloyXMLMarkup-ConditionalCode

  • to determine whether to show / hide a part of UI in a template,

    at the time to form any row in the dataset, you can add the “visible" property using the object's bindId you want to show / hide

    DataSet = [

    { pic: {image: ‘images/someimage.png’, visible:false}, info: {text: ‘blabla’}}

    ];
