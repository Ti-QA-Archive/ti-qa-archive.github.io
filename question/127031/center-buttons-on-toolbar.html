Center buttons on Toolbar

I've searched and its looking like using something called FLEXIBLE_SPACE, however, when I do that, my buttons disappear. Here is my code:

var backBtn = Ti.UI.createButton({
    backgroundImage:'../images/back.png',
    width:30,
    height:30        
});

var zoomBtn = Ti.UI.createButton({
    backgroundImage:'../images/2x.png',
    width:30,
    height:30        
});

var playPauseBtn = Ti.UI.createButton({
    backgroundImage:'../images/pause.png',
    width:30,
    height:30    
});

var slideListBtn = Ti.UI.createButton({
    backgroundImage:'../images/list.png',
    width:30,
    height:30
});

var toolbar = Ti.UI.createToolbar({
    items:[backBtn,playPauseBtn,slideListBtn,zoomBtn],
    bottom:'base',
    borderTop:true,
    borderBottom:true,
    barColor:'#336699'
});
win.add(toolbar);

Right now they are all aligned left. If I apply the FLEXIBLE_SPACE style to the button, they just disappear. Any idea?

    You create a new button that is to be the spaced button and style it with flex space. Dont style any of your actual buttons with the flex space styling.

    It should look like this:

    var toolbar = Ti.UI.createToolbar({
    items:[flexSpace,backBtn,playPauseBtn,slideListBtn,zoomBtn,flexSpace],
    bottom:'base',
    borderTop:true,
    borderBottom:true,
    barColor:'#336699'
});
