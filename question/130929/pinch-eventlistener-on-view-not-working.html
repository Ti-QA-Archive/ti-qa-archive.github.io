Pinch Eventlistener on view not working...

I am pretty much new to the titanium environment. I have a view and i have added the pinch eventlistener to the view according to this wiki link::
https://wiki.appcelerator.org/display/guides/Supporting+Gestures#SupportingGestures-Pinch
I have written the following simple code for it and somehow it does not react to the event pinch:


var win = Titanium.UI.createWindow({  

     orientationModes:[Ti.UI.LANDSCAPE_LEFT, Ti.UI.LANDSCAPE_RIGHT],
     backgroundColor:'white'

});

win.open();

var view5 = Ti.UI.createImageView({
    url:'view5_background1.png',
    borderRadius : 10,
    width : 1030,
    height : 600,
    top : 0,

});

view5.addEventListener('pinch', function(e){
    alert('You pinched to ' + e.scale*100 + '%');
});

win.add(view5);

Can anybody tell me why is this happening? I am running this code for Ipad simulator and using Titanium SDK 1.8.0.1.

  • Webview automatically listen pinch gesture and by default provide zoom in and out functionality , for this you don't want to put any efforts . All this is possible in titanium using webview. Accept answer if you find usefull … This things work for me.

    var webview = Ti.UI.createWebView({

    borderRadius : 10,
    width : 1030,
    height : 600,
    top : 0,
    backgroundColor:'#00000000',
    scalesPageToFit:true,
    touchEnabled: true,

    });

    wdt = 100;

    hgt = 200;

    img_pth = 'view5_background.png'; // Insert your image url.

    webview.html = '<html><body> <img src="images/'+img_pth+'" width='+wdt+' height='+hgt+'></body></html>';

    win.add(webview);
