Hi guys,

I've read the getting started guide, the appcelerator API, I've read the Q&A and I googled about this but I couldn't find anything. My i18n installation simply can't work as described in the doc.

Here is what I've done :

  1. I created a folder named 'i18n' at the same place as the tiapp.xml file.
  2. I filled the file en/strings.xml with the following code:
    <?xml version="1.0" encoding="UTF-8"?>
