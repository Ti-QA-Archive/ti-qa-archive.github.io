Help with .jss

I can't seem to get jss to work. I have the following simple example. 

//In the app.js

Titanium.UI.setBackgroundColor('#000');

// create tab group
var tabGroup = Titanium.UI.createTabGroup();

var win1 = Titanium.UI.createWindow({  
    title:'Tab 1',
    backgroundColor:'#fff'
});
var tab1 = Titanium.UI.createTab({  
    icon:'KS_nav_views.png',
    title:'Tab 1',
    window:win1
});

// Title Field
var title= Titanium.UI.createTextField({
    id:'header1',
    top: 0,
    left: 0,
    height: 40,
    textAlign:'left',
    backgroundColor:'white',
    hintText: 'Untitled ',    
    });

win1.add(title)

tabGroup.addTab(tab1);  
tabGroup.open();


//In the app.jss
#header1 {
    font:{fontSize:16,fontWeight:'bold',fontFamily:'ArialRoundedMTBold'};
}

I want to have one JSS file for all the formatting will naming it app.jss work for all the files I have set up? Thanks

    It should work if you change your jss to be the below. You might need to clear your build directory or update your tiapp.xml if you are using Android

    #header1 {
    font-size:16;
    font-weight:'bold';
    font-family:'ArialRoundedMTBold';
    }
