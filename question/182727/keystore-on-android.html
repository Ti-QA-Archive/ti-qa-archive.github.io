  • For debugging you have to use a dedicated Titanium debug keystore.
    It is located ~/Library/Application\ Support/Titanium/mobilesdk/osx/3.5.1.GA/android/dev_keystore

    Use this command to generate a correct fingerprint:
    keytool -list -v -keystore ~/Library/Application\ Support/Titanium/mobilesdk/osx/3.5.1.GA/android/dev_keystore

    For further details
    http://docs.appcelerator.com/platform/latest/#!/guide/Google_Maps_v2_for_Android
