    • @Issac, good news, proxy support has not only been added, but has been released as of 2 days ago!

      To setup the Titanium CLI to work with a proxy server, first you need to update to v3.0.23:

      # Mac OS X and Linux
sudo npm install -g titanium

# Windows
npm install -g titanium

      Next, you need to set the proxy server setting:

      titanium config cli.httpProxyServer http://<IP>:<PORT>

      That should do it! You can log in, create projects, and perform builds now. Depending on your proxy server you may be blocked from downloading large files, so you may not be able to use the "titanium sdk update" feature to upgrade the Titanium SDK when the next version is released.

    • That's because the bug was fixed in the node-appc library and we haven't released the latest version yet. It's still being tested.

      If you are willing to risk breaking things, then you could edit /usr/local/lib/node_modules/titanium/package.json and under dependencies carefully change the "node-appc" version from ">=0.1.13" to "https://github.com/appcelerator/node-appc.git".

      Then cd &#x2F;usr&#x2F;local&#x2F;lib&#x2F;node_modules&#x2F;titanium and run sudo npm update.
