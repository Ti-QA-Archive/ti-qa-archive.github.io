applicationDataDirectory path from within local html file

What is the path that should be used to access the applicationDataDirectory from within local html files in a webview? I tried app://Documents/ but this points to my app resources, not the applicationDataDirectory where my app saves data and images. I'm developing this app for iPhone/iPad/Android.

I built a path in my app.js file using the applicationDataDirectory and the image file name and pass this to my webview but I get the following path that will not work in an "img src".

Path in simulator:
/Users/myUserName/Library/Application%20Support/iPhone%20Simulator/4.2/Applications/427D2830-A50E-47BA-80B6-E3422AA72AFC/Documents/myImageName.jpg

A similar path is created on the device. They don't work

  • my suggestion would be to set the variable in a webView from inside your application prepend or append a javascript snippet to put it in.

    appDir_html = "<script>var applicationDataDirectory='" + Ti.Filesystem.applicationDataDirectory + "'</script>";
var webView = Ti.UI.createWebView({html:text_of_html +appDir_html });
win1.add(webView);
