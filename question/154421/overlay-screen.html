Overlay screen

Hi.

At the moment i want to add a overlay screen to certain pages on my app where it will help guide the user on how to use certain buttons. Im not sure the best way to do this in alloy.

What ive done so far is add a class of Overlay and then add some tss with the following

".Overlay":{
top:0,
opacity: 0.9,
backgroundImage:"/images/OVERLAY.png",
height:Ti.UI.FILL,
weight:Ti.UI.FILL,
}

But instead of cover the whole page is only covering the end part because i put the main window a layout of vertical. It there any way i can make the background image of overlay cover everything?

  • Open a new window and set the background Image to whatever you're trying to show. I realize this isn't alloy, but the idea is the same:

        var overlayWin = Titanium.UI.createWindow({
        top:0,
        height:568,
        width:320,
        backgroundImage: Ti.Filesystem.resourcesDirectory + 'images/loadingOverlay.png'
    });   
    overlayWin.open();
