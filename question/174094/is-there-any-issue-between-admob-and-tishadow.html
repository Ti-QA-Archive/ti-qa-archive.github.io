    In order to use your modules, you have to include them into your tishadow client app in order for them to be recognised. Simply:

    • Open the tishadow app project
    • Import the adMob module (by copying the zip file into the project's root directory)
    • Add it's reference to the module section of the tiapp.xml file
    • Launch the app to install the updated version

    Once the tishadow app includes your module, you hosted app will also be able to use it.

    Hope this helps

    • tishadow is comprised of 2 components.

      • The tishadow server
      • The client application that is installed on your device (or your emulator) that handles the communication between your app and the server.

      What I am talking here is the code of the client application that must be modified so it includes your admob module. The app is usually located in your home directory under tishadowapp. This is the default location from the documentation. If you look closely, you will see that this is a Titanium project and that you can import it into Titanium Studio just like you would a regular app.

      Simply open the tiapp.xml file from this project and add the reference to your module. Then recompile, install this new version** of the client* app and it will recognise your modules.

      Simply put, if you want tishadow to use your module within your app. It has to know about it in its own app.
