Override onCreate for Android

So I'm working on dealing with the "Application restart required" bug in Android, and I've seen some work arounds to put in the onCreate method for the main activity. Specifically, I've seen this suggested:

if (!isTaskRoot()) {
    final Intent intent = getIntent();
    final String intentAction = intent.getAction(); 
    if (intent.hasCategory(Intent.CATEGORY_LAUNCHER) && intentAction != null && intentAction.equals(Intent.ACTION_MAIN)) {
        Log.w(LOG_TAG, "Main Activity is not the root.  Finishing Main Activity instead of launching.");
        finish();
        return;       
    }
}

Is there a way I can get this code into the onCreate for my app?
