Pass arguments from Require tag to another xml

Hi,

I am populating data in tableview and on click of the row, i want to populate data on other table (on different window).

Following is my code :

window1.xml

<TableView dataCollection="Mydata" onClick="showId" dataTransform="transformData">
    <Require src="row"></Require>
</TableView>

row.xml

<Alloy>
    <TableViewRow>
        <Label id="row" text="{Name}"/>        
    </TableViewRow>
</Alloy>

row.js

if ($model) {
    $.row.model = $model.toJSON();
}

window2.xml

<TableView dataCollection="Mydata" onClick="showId" dataTransform="transformData">
    <Require src="row1"></Require>
</TableView>

row1.xml

<Alloy>
    <TableViewRow>
        <Label id="row1" text="{Surname}"/>        
    </TableViewRow>
</Alloy>

row1.js

if ($model) {
    $.row1.model = $model.toJSON();
}

This code is working fine, but can I populate data in both the tables using only row.xml ,row.tss and row.js files?

can we pass arguments using Require tag to row.xml?
