  • setInterval is your best shot. I suggest putting it on a interval of 1000 (ms) and just have a variable +1 every hit.

    e.g.: 

    timer = setInterval(function() {
                        t++;
                        Ti.API.info(t);
                        if(t==10) {
                        //10 seconds have past...
                        }                     
                    },1000);
