    Hi Emrah.

    I'm developing on Samsung Galaxy tab v7, should soon get v10 (or something similar) … so, yes Titanium works on tablets. It's very similar, Android (and Titanium) API is the same, only screen size is different.

    Titanium lacks support for new Android Honeycomb features, but you can develop apps for Honeycomb, only without that new features.
