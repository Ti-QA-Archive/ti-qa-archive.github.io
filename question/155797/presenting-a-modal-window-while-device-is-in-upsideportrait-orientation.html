Presenting a modal window while device is in UPSIDE_PORTRAIT orientation

Hi everyone,

I am sure this has been asked before, however I wasn't able to find it in the Q&A or elsewhere.

The problem is: Whenever a modal window is presented while the device is in UPSIDE_PORTRAIT orientation, the modal window is presented upside down. Instead of sliding up from the bottom of the screen the modal window slides down from the top of the screen. The effect is somehow different on the iOS-Simulator, however it also doesn't work there.

This once used to work, but it somehow doesn't anymore or I am missing something here. All other orientations work fine.

My questions: Am I missing something here? Is this a bug? Is there a workaround for this?

Please note that I do not intend to use the UPSIDE_PORTRAIT orientation on the iPhone, this is just for testing purposes. The UPSIDE_PORTRAIT orientation is only supported on the iPad in our production app.

Test case: You can test this by simply adding a new Mobile Project using the classic 'Single Window Application' template and amending two files:

Amend FirstView.js like this:

//FirstView Component Constructor
function FirstView() {
    //create object instance, a parasitic subclass of Observable
    var self = Ti.UI.createView();

    // Create a Button.
    var aButton = Ti.UI.createButton({
        title : 'aButton',
        height : 44,
        width : 132,
        top : 200
    });

    // Listen for click events.
    aButton.addEventListener('click', function() {
        var modalWin = Ti.UI.createWindow({
            modal : true,
            backgroundColor : 'blue',
        });
        // Create a Button.
        var aButton = Ti.UI.createButton({
            title : 'Close',
            height : 44,
            width : 132,
            top : 200
        });

        // Listen for click events.
        aButton.addEventListener('click', function() {
            modalWin.close();
        });

        // Add to the parent view.
        modalWin.add(aButton);

        modalWin.open();
    });

    // Add to the parent view.
    self.add(aButton);

    return self;
}

module.exports = FirstView;

Set both iPhone and iPad orientations in tiapp.xml to

Ti.UI.PORTRAIT
Ti.UI.UPSIDE_PORTRAIT
Ti.UI.LANDSCAPE_LEFT
Ti.UI.LANDSCAPE_RIGHT
