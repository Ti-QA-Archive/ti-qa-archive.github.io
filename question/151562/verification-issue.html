    • The module should be in (depending on your settings)

      in your user folder
      ~&#x2F;Library&#x2F;Application Support&#x2F;Titanium&#x2F; ..

      or in your system root:
      &#x2F;Library&#x2F;Application Support&#x2F;Titanium&#x2F;

      You should only have one copy of the module files.

      Another possible reason for this issue is that the developer of the module has not built it correctly and it has a duplicate GUID to another module (another reason to only have one copy)

      You should also try using the feedback support option on the Marketplace page for this module.
