iOS7 windows slide up under navigation bar

Hi all,
Since switching to iOS7, my App's displays behave inconsistently. The app has five tabs which, on first opening, all display fine, but on moving between tabs, the window has moved up the screen by the vertical height of the nav bar. This vertical shift only occurs once rather than every time the tab is re-visited. Any ideas?
Titanium build 3.1.3.201309132423, iOS 7.0.3, Simulator version 7.0, iOS SDK 7.0.

  • Success! After much trial and error, I have got a result! If a window has a navigation bar, it's top property needs to be approx 40px rather than 0. This stops the sliding up behind. Now I just have to apply this to all the pages. Thanks for your interest.
