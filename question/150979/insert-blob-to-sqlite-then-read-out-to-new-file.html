Insert BLOB to sqlite then read out to new file

I have audio/video/images I need to extract blob data out and insert into database, then delete the original files. This is for a mobile app, I then need to read the blob data back into a new file. Could someone tell me how (or if) I can read the video/etc to a Ti Blob, and insert it to SQLite. Then once from install of my app I need to extract the blob data to new files, outside of my app bundle so that they are not read-only and I can set some special properties on them. There will not be outside connection to do this via server request, so it must be done this way.

Titanium: 3.0

Any help appreciated.

  • Don't put a blob in the db, it will make your app very slow and crashy due the memory consumption.
    Save in the db only the path to the file, then edit the file in it's place.
