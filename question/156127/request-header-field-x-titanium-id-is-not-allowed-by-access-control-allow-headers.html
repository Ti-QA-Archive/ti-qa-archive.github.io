Request header field X-Titanium-Id is not allowed by Access-Control-Allow-Headers?

After update to Titanium SDK 3.1.2 GA, I get the following error message when trying to connect to ACS via mobile web (Safari). This message occurs both on localhost but also when I upload the package to a server. The app work fine when started in iOS Simulator. Any ideas. Before with SDK 3.0 it worked fine, but with the last Titanium Studio Update (3.1.2.201308091617) this also stopped working (the I get some JSON.parse error…).
http://developer.appcelerator.com/question/ask#

  • This happens because "X-Titanium-Id" header is always included in response.

    Only solution I found in order to remove it, was to open "build/mobileweb/index.html" and comment this line:

    this._xhr.setRequestHeader('X-Titanium-Id', App.guid);

    Of course this needs to be done each time you build.
