    If you you do not define a requestCode, it will use a default requestCode. That means, if you doing this twice, it overrides the first one. To solve this, simply add a requestCode property:

    // first
alarmManager.addAlarmService({
    requestCode: 1001,
    service : 'com.midade.alfSona.NotficationService',
    // ...
});

// second
alarmManager.addAlarmService({
    requestCode: 1002,
    service : 'com.midade.alfSona.NotficationService',
    // ...
});
