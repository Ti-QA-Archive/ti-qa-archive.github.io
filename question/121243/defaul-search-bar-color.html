Defaul search bar color

System default search bars have a nice light blue gradient backgrounds. If I create a search bar by this code:

var search = Titanium.UI.createSearchBar({
    hintText: L('places_search_hint_text', 'Keresés a helyek közt')
});

var table = Titanium.UI.createTableView({
    search: search,
    filterAttribute: 'searchTitle'
});

it will be simply white. I know, that I can use my own gradients, but they seem to be slow, and the search bars in the Kitchen Sink are looking good (and they are used just like mine).

Please tell me, what is the trick! :)

  • Try barColor:

    var search = Titanium.UI.createSearchBar({
    hintText: L('places_search_hint_text', 'Keresés a helyek közt'),
    barColor:'#385292'
});
