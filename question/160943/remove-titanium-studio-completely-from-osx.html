  • I ran into the same issue. Here is what I did on Mac.

    1. Download previous version: https://wiki.appcelerator.org/display/tis/Reverting+to+a+Previous+Version
    2. Delete Titanium from Applications/Titanium Studio
    3. Install new version.
    4. You may run into an issue with the Android SDK so when you open Titanium go to Get Started. Click Android SDK and install in a different location. This will take a while but it should get you back to where you want.
