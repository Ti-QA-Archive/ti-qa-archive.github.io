Get height of Label

If I try to get the height of a label, which has no particular height definition, I get the values 0 or 'auto'. It doesn't matter if the height is unset or set to 'auto'.
However this problem only apears on Android devices or the Android simulator, it works perfectly fine on iOS devices.

Example programm (I tried to use a workaround with the view component):

var label = Titanium.UI.createLabel({
    text:'TESTHEIGHT',
    width:'auto',
    height:'auto'
});
var label2 = Titanium.UI.createLabel({
    text:'TESTHEIGHT2',
    width:'auto',
    height:null
});
var view = Titanium.UI.createView({
    width:'auto',
    height:'auto'
});

view.add(label2)

win2.add(label)
win2.add(view);

win2.addEventListener('open', function(e){  
    alert('labelHEIGHT: '+label.height+'-'+view.height);
    alert('labelHEIGHT.size: '+label.size.height+'-'+view.size.height);
});
win2.open();

Output:

labelHEIGHT: auto-0
labelHEIGHT.size: 0-0

Does anyone have a solution for this problem?
