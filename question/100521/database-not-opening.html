Database not opening

I just can't seem to figure out the pathing just yet.

The file is located at "Resources/data/dbfile.db"

I have:

var db = Ti.database.open( Titanium.Filesystem.resourcesDirectory + "dbfile" );

but all I get is:

IllegalArgumentException: File app://dbfile contains a path separetor.

I've tried: "dbfile.db"
"PATH + 'dbfile.db'"
"dbfile"
"data/dbfile"
"data/dbfile.db"

The only one that worked was "dbfile" when I placed the file in the root of the resources folder. It would be nice to put this in a subfolder as I also will be using subfolders for storing images and whatnot.

Can someone shed some light as to what I'm doing wrong and what I should be doing?

    Paul

    You haven't included all the information. Please read Participating in the Q&A to help yourself and others get more out of the Q&A.

    You need to install the db, to copy it from the Resources folder to applicationDataDirectory&#x2F;..&#x2F;databases&#x2F;, and then you can open it.

    Hence:

    Ti.Database.install('/path/to/', 'dbfile.db');
var db = Ti.Database.open('dbfile.db');

    Hope this helps
