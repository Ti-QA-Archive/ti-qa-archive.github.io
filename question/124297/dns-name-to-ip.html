DNS name to IP

Hello friends,

Can someone please help me on how to resolve dns name into an ip. For instance, if given john.noe to resolve to 10.0.2.80 if dns server is aware of john.noe

  • Your best bet is going to be using Titanium.Network.HTTPClient to make a server side query and return the DNS data as JSON back to your app. You can use JSON.parse() to handle the returned values.

  • Jacob

    You can either use the dig.jsondns.org service directly, if the owner allows, or run the same code, available under the jsondns github project, on your own server. See this query for www.appcelerator.com.

    Then, as Tony says, you can use Titanium.Network.HTTPClient to retrieve the json request from your application.

    Hope this helps.
