Remove windows from iPhone NavigationGroup

Hi,

I'm using iPhone NavigationGroup control for my messaging app.

On my inbox window (the first window) the user clicks the new message button, which brings him to the message window (window number 2). After filling the message form and sending it opens the message window (same one the user gets when he clicks an item in the inbox) - that's window number 3.

Now when the user clicks the back button he gets to the "new message" window, but I really want him to get back to the inbox. So My question is how can I remove middle windows in a NavigationGroup?

So here's what we've got:
Window 1: Inbox
Window 2: Create message
Window 3: Message

When window 3 opens I want to remove window 2.

can this be done?

Thanks

  • Hi Ophir,
    Try out this to go back to root window.

    Root Window :

    var Rootwin = Ti.UI.createWindow({
        url : 'next.js',
        _parent : Titanium.UI.currentWindow,
        navGroup : win.navGroup
    });

    Next window :

    var win = Ti.UI.currentWindow;
var Thirdwin = Ti.UI.createWindow({
        url : 'Third.js',
        _parent : Titanium.UI.currentWindow,
        navGroup : win.navGroup,
        rootWindow : win
    });

    Third Window : on button ClickEvent put this code. that will close all window and go to root window.

    win.navGroup.close(win.rootWindow);
    win.navGroup.close(win._parent);
    win.navGroup.close(win);
