How to update the UI when there is a change to the table in a DB

Could someone advice me on how I can update the UI as and when a table on the DB is updated. What should my approach be for doing this.

For example, a new row is inserted on a table, then I need to append the tableview automatically.

Thanks and Regards

    You cannot do that automatically. What you can is firing an Event. Lets say you have two UI components tblA and tblB, now on any row insert you can Ti.App.fireEvent(&#39;db:tableName:rowAdded&#39; , rowObject); and in your UI Code you can listen for this and table.appendRow(..).

  • Hi,
    Lets take a sample declaration of tableView

    var tableView = Titanium.UI.createTableView({
            backgroundColor : '#fff',
            data : rows,            //rows is the dataSource, of type array
            borderWidth : 1,
            borderColor : '#576996',

        });

    After you done adding new row or deleting on your table dataSource that is rows, means all your insert, delete operation should be on rows.

    After that just do tableView.setData([ ]); //This will empty your current table
    and do
    tableView.setData(rows); // This will update your table view with the new data.

    Hope that helps
