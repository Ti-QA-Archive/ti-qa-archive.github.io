  • Marked as -1 because a simple search and/or reading the install requirements would have solved this problem.

    [ERROR] JDK version &#39;javac&#39; is not recognized as an internal or external command,

    Means either your java is not installed et al. or the path has not been setup on your system correctly.

    Please refer to the instructions regarding paths and environment variables:
    http://docs.appcelerator.com/titanium/latest/#!/guide/Software_Locations_and_Environment_Variables

  • Hi Stephen, i hereby would like to thanks for your helping, i have resolved all of the error.

    I have found, actually i have wrong assign the value of PATH in the Environment variable…

    I just edited the PATH become C:\Program Files (x86)\Java\jdk1.6.0_25\bin at the first time..

    Now i copied path of value of before, and add the C:\Program Files (x86)\Java\jdk1.6.0_25\bin

    Then the problem are very clearly to resolved…

    For who still cant resolve this problem, you can see the step below.

    Computer > Properties > Advanced system settings > Environment Variables > System Variable > Path>Edit> Variable Value:

    %SystemRoot%\system32;%SystemRoot%;%SystemRoot%\System32\Wbem;%SYSTEMROOT%\System32\WindowsPowerShell\v1.0\;C:\Program Files\Microsoft\Web Platform Installer\;C:\Program Files (x86)\Microsoft ASP.NET\ASP.NET Web Pages\v1.0\;C:\Program Files (x86)\Windows Kits\8.0\Windows Performance Toolkit\;C:\Program Files\Microsoft SQL Server\110\Tools\Binn\;c:\Program Files (x86)\Microsoft SQL Server\110\Tools\Binn\;c:\Program Files\Microsoft SQL Server\110\DTS\Binn\;C:\Program Files (x86)\nodejs\

    *above value are the original, and you need to add the path of you Java installation file.

    ;C:\Program Files (x86)\Java\jdk1.6.0_25\bin

    jdk1.6.0_25 you need to follow your java version
