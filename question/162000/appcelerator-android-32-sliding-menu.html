  • Accepted Answer

    For what it's worth, the -right- way of implementing a slide-in menu on Android is using Android's Navigation Layout, which comes pre-baked with the SDK http://developer.android.com/design/patterns/navigation-drawer.html. There's a Jira ticket for it https://jira.appcelerator.org/browse/TIMOB-15910, and there's also a module at http://goo.gl/R1UwIq you can use today, so you can have a TRUE native Android slide-in menu.

      <property name="ti.android.useLegacyWindow" type="bool">true</property>

      Enables or disables the legacy window behavior that a heavyweight window is created if any of the following properties are defined on creation: fullscreen, navBarHidden, modal, or windowSoftInputMode. By default, the legacy behavior is disabled, which means all the windows are heavyweight. (since 3.2.0)
