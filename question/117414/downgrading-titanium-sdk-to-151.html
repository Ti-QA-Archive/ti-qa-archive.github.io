Downgrading Titanium SDK to 1.5.1

I wanted to develop using Titanium 1.5.1. Now when I download Titanium I am having SDK 1.2. and when update to 1.6 which is the only option I get to update, I don't get 1.5.1 SDK.

How can I get SDK 1.5.1.?

    Really, the major release of 1.5 should be installed automatically by Titanium Developer. However, you can also install the latest build by following the Continuous Builds guide. On the download page, just select 1.5.X from the drop-down menu.
