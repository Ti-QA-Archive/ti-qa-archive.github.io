  • The Sound object loads the entire media resource in memory before playing, and there's no preload property. Try this:

    var sound = Titanium.Media.createSound({
    // url : url,
    // preload : false// This wont effect
});

......

win.open();
sound.url = url; // Loading Media after win opened
