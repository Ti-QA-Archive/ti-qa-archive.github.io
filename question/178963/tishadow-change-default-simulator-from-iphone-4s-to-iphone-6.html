TiShadow: Change default simulator from iPhone 4s to iPhone 6?

I am using TiShadow Express for live reload, how do I change the default simulator device? Mine is iPhone 4s, I want it to be 6.

ti build -p ios --shadow

If I use this, it builds but not with live reload

ti build -p ios -C &quot;iPhone 6&quot;

    from my .bash_profile

    alias geny-n5='ti build -p android --device-id "Nexus 5 - 4.4.2 - API 19 - 1080x1920"'
alias geny-n4='ti build -p android --device-id "Nexus 4 - 4.2.2 - API 17 - 768x1280"'
alias geny-n5s='ti build -p android --device-id "Nexus 5 - 4.4.2 - API 19 - 1080x1920" --shadow'
alias geny-n4s='ti build -p android --device-id "Nexus 4 - 4.2.2 - API 17 - 768x1280" --shadow'
alias shadow-iphone='ti build -p iphone -I 7.1 --retina --tall --shadow' 

## xCode 6 and iOS 8 commands
alias iphone4s='ti build -p ios -C 85403F52-1516-424E-B60A-5A8532A3C381' 
alias iphone4s-shadow='ti build -p ios -C 85403F52-1516-424E-B60A-5A8532A3C381 --shadow' 

alias iphone5s='ti build -p ios -C D10ADF0C-DF4C-4B3D-BB13-A25F35B0CFC2' 
alias iphone5s-shadow='ti build -p ios -C D10ADF0C-DF4C-4B3D-BB13-A25F35B0CFC2 --shadow' 

alias iphone6='ti build -p ios -C F231C192-9EA2-4C19-957A-38F9148A37A2' 
alias iphone6-shadow='ti build -p ios -C F231C192-9EA2-4C19-957A-38F9148A37A2 --shadow' 

alias iphone6plus='ti build -p ios -C DE15B488-A082-48F0-8C2B-8B5EBC4CC195' 
alias iphone6plus-shadow='ti build -p ios -C DE15B488-A082-48F0-8C2B-8B5EBC4CC195 --shadow'

    Your simulator UDIDs will be different, so you will need to run ti info and pull the UDIDs from the results. Keep in mind, that every xcode update now seems to change the simulator UDIDs.
