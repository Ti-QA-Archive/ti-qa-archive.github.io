  • > Maybe you ought to go write your own framework then, or, stick with phonegap.

    I am using the html/js (whatever) framework that i did not write and because i did not write the framework that is why i have to look for better options. Found Ti and thought it was better. The framework name is sencha touch. No need to have any ios/android sdk installed. I write code and then use phonegap cloud sevice to build native app(not browser based). The sencha cmd was supposed to build without ios/sdk installed on windows but they are always fixing bugs instead of adding features to the cmd (sorry sencha). Phonegap one click build the native app from uploaded sencha code in zip format. Then one tap is enough to start installation on device. All on windows without any ios/android sdk. That makes life a lot easier of a developer like me who build for ios without mac.
