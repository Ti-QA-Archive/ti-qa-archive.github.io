Fail to run on iPad Retina Simulator

Hi there,

My app can run on iPad Simulator (non-retina) without any problem and can deploy to iPad 2 device successfully.

But once I try to run on iPad (retina) simulator, by selecting iOS Simulator's menu: Hardware > Device > iPad (Retina), my app quit at once without reporting any error. I only got the message in Titanium Console:
"Application has exited from Simulator"

I am using Titanium Studio 2.0, XCode 4.3 on Lion. Run Configuration set as below:
Device=iPad, iOS SDK=5.1, Log Level=Trace

What's wrong?

    Try closing iOS simulator, then opening it without using Studio (ie: type in iOS Simulator in Spotlight). From there open it up, select iPad Retina, close the simulator and relaunch the app via studio. I've had to do this for iPhone retina previously and this was the solution.

    Hopefully it helps.
