Changing the height of a tableViewRow

HI all,
I am wondering if there is a way to change the height of each row in a table view after the table has been added to the window.

My reasoning for this is that the auto height on a row does not work since I have a label dynamically loaded so the device does not know the height of the label until it is all added to the window.

So I will need to target each row of the table and the labels inside of each row after the table is displayed. Is this possible?

    Yes, it is possible. Something like this:

    
...
//suppose you add label to the row like this
tableViewRow.add(label);
tableViewRow.Label = label;
...

win.addEventListener('open', function() {
    //suppose there is just one section
    var rows = tableView.data[0].rows;
    for (var i = 0; i < rows.length; ++i) {
        row.height = row.Label.size.height;
    }
});

    Didn't test, hope it works! :)
