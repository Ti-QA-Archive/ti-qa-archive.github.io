  • Yes you can, depending on the changes. If you're going from one window to the next. by using a button listener or something, just make the changes in the new window/js file, and they will reflect. The only time you have to clean/recompile is if you are making file-level changes (adding new js files, images, databases, etc)

  • There are 2 ways of doing this:

    1. as Shebin said, try TiShadow, is a very nice tool.

    2. (ios only) - build your app once with studio, run the objC project in the build folder with xcode but profiling the app for allocations, then from the instruments UI just start stop the app. It will take any change you do in the js side (simulator only)
