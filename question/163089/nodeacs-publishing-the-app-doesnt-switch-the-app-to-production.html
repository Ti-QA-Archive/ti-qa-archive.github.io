Node.acs publishing the app doesn't switch the app to production

Hi, I just tried publishing my app to the cloud however my app is still using the development settings. How can i change it so that it uses production settings?

    That’s because Development and Production setting use the different App Keys. Changing the App key in you app code, you may be able to user the production setting.
