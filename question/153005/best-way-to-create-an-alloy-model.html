Best way to create an Alloy model

Hi,

I am looking for the best practice advice on creating a model when I only have the 'id' of it for instance. What I've done so far to get a model is:

var myCollection = Alloy.createCollection("mycollection");
myCollection.fetch();
var mymodel = myCollection.where({id: collection_id});

….and then mymodel[0] is the model instance I can work with.

Is there a better way to go with this? I would appreciate if anybody can share better ideas so I learn the proper way of creating a model. I also fear that fetch() selects all my data, which should not be a good practice during page load especially on a heavy loaded database! I am using sqlite

Thanks in advance!

    If the id you are searching for is the idAttribute for your model, then you can just do this, which will be much more efficient:

    var myCollection = Alloy.createCollection("mycollection");
myCollection.fetch();
var mymodel = myCollection.get(collection_id);
