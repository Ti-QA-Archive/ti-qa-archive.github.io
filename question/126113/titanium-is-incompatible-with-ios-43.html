Titanium is incompatible with iOS 4.3?

After having installed the latest of Titanium (Titanium Studio, build: 1.0.4.201108101535) and Xcode 4.1 (Build: 4B110f, iOS SDK: 4.3), and Titanium insisting the iOS SDK is not installed, I can only conclude that Titanium is not compatible with the latest release of Xcode / iOS SDK.

The latest release of Titanium was released on Aug 10, 2011, while the latest release of Xcode was released on Aug 29, 2011.

I hope the developers are looking, because this is pretty annoying.
