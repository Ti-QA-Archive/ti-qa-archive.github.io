Confirm delete of a tableViewRow with an AlertDialog

Setting my tableView.editable to true I get an out-of-the-box working behavior with my rows showing a swipe triggered left-side red circle and a right-side delete-button; now I'd like to add a custom confim/cancel dialog to this delete action.

So I need to show an alertDialog when the user clicks on the delete button in the editable tableView, as a confirm/cancel dialog for the deletion which is going to occur.

The questions is how to detect the click event on the delete button in the tableViewRow to show the alert dialog itself.

Thanks in advance.

    There is a delete event on the tableView that gets fire when the row is deleted. i havent use this event but you might have to save the row data on a variable and then insert it back in if the user doesn't want to delete.
