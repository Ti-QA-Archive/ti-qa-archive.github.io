android and UI.FILL

Hi there,
I'm trying to create a view with horizontal layout with two view inside; the first is an imageview with a specified width and the second is a label which should take all the remaining available space.

Here is the code:

    var self = Ti.UI.createView({
        height:Titanium.UI.SIZE, 
        width:'49%',
        layout: 'horizontal',
        left:0,
        top:10,
        border: 1,
        borderColor:'blue',
    });

    var pathobj='/images/dirctory.png';


    var icon =  Titanium.UI.createImageView({
        image: pathobj,
        width:65,
        height:65,
        left:10,
    });

    var title = Titanium.UI.createLabel({
        text:details.text,
        color:'#E00821',
        font:{fontSize:16,fontWeight:'bold'},
        width:Titanium.UI.FILL,
        height:55,
        border: 1,
        borderColor:'#888',
        backgroundColor:'#DDD',
        textAlign:'left',
        right:2,
        left:5,
    });

    self.add(icon);
    self.add(title);

On the IOS emulator (ipad) it works like a charm but on an android device (samsung galaxy) the result is quite different; the UI.FILL directive takes the whole space, not considering the width of the previous imageview, and also borders are completely missing (colors are only for example). What is wrong?

Thanks.

  • What you describe sounds like a bug.

    However, I would simplify the code to be more efficient anyway, doing away with some of the layout calculations.

    var self = Ti.UI.createView({
        height:65, // no need to have a dynamic height, children are all hard-coded
        width:'49%',
//        layout: 'horizontal', // we don't need this, width is hard-coded too
        left:0,
        top:10,
        border: 1,
        borderColor:'blue',
    });

    var pathobj='/images/dirctory.png';


    var icon =  Titanium.UI.createImageView({
        image: pathobj,
        width:65,
        height:65,
        left:10,
    });

    var title = Titanium.UI.createLabel({
        text:details.text,
        color:'#E00821',
        font:{fontSize:16,fontWeight:'bold'},
//        width:Titanium.UI.FILL, // use left & right instead
        left:75, // icon width + icon left
        right:0, // take up the rest of the space
        height:55,
        border: 1,
        borderColor:'#888',
        backgroundColor:'#DDD',
        textAlign:'left',
        right:2,
        left:5,
    });

    self.add(icon);
    self.add(title);
