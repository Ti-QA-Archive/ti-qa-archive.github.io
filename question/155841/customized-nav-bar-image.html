Customized Nav Bar Image

I have a left nav bar image that appears in the nav bar, but with a system style button that appears behind the image. My question is, can you override that system style button, so it doesn't appear behind my image. Here is my code:


function detail(data) {
    var self = Ti.UI.createWindow({
        backgroundColor:'#fff',
    });

    self.barColor = '#3b5e34';
    self.titleImage = 'tap.png';
    self.barImage = '/images/navBar.png';



 var ButtonRetour = Ti.UI.createButton({
    image:'/images/backButton.png' 

});             

ButtonRetour.addEventListener('click', function(){
    self.close();
});

self.leftNavButton = ButtonRetour;

    Hi

    Firstly - I misread this question and thought you meant the leftNavButton was displayed behind the navBar, your image has cleared up that you only mean the image on the button.

    The answer to this question would normally be to set the style property to Ti.UI.iPhone.SystemButtonStyle.PLAIN and the button would then only show as an image (in your case), however this button is in the navigation/tabgroup title area - so the rules are different.

    What you need is actually an imageView instead of the button - whilst the control name would imply a button it can take any view (sorry if you already know this).

    So try this instead;

    var ButtonRetour = Ti.UI.createImageView({
    image:'/images/backButton.png'
});

    The event listeners work the same on both and it works correctly - I have just mocked-up a test app to prove it.
