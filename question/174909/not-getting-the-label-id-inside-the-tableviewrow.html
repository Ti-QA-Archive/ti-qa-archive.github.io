    Hi

    You need to access the row that was selected and then from there access the children array. Accessing the specific named requirs adding each row into the TableView with a controller. But here is what you are after - you just need to make sure you know the order you added them or loop the array to find the match'

    $.listtable.addEventListener('click', function(e){ 
    alert(e.row.children[0].getText());  // first child
    // alert(e.row.children[1].getText());  // second child
});
