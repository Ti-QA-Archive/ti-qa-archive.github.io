    To access the data in your table, you need to do something like the following:

    // For rowCount
Ti.API.info(tableview.data[0].rowCount);

// This is equivalent to "data", returns all the rows
Ti.API.info(tableview.data[0].rows);

// Accesses the first row
Ti.API.info(tableview.data[0].rows[0]);

// This would access row ones "pp.text"
Ti.API.info(tableview.data[0].rows[0].pp.text;);
