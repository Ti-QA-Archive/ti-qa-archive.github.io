  • Yes, I believe Titanium will only include the module(s) it finds referenced in your code. For Android, I think you need to use the new 1.5.x SDK in order for this to occur. For iPhone, I think it happened in earlier releases too.

    The .ipa file mentioned by Peter is a special type of .zip file containing the *.app folder mentioned by Alberto.

    By way of example, I have a fairly complex iphone app and the *.ipa file is under 2.5MB in size.
