Dynamically replace part of a View

Hi, I want to replace part of a View dynamically at run time depending on certain condition. For example, if condition is true, I will show 2 pickers and a text field. if it is false, I want to replace the text field with another picker. What is the best way to achieve this? I am thinking of encapsulating the 3rd picker in a javascript include file and swap the text field with the js file at runtime. What I don't want to do is hardcode if..else in my View since the same idea will used on a wider scale in my application. Thanks in advance.

  • Okay, something like this:

    var item;
if (condition) {
    item = Ti.UI.createTextField({...});
}
else {
    item = Ti.UI.createPicker({...});
}

var picker = Ti.UI.createPicker({...});
var textField = Ti.UI.createTextField({...});

win.add(item);
win.add(picker);
win.add(textField);
