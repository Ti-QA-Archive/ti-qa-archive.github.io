  • Here's the relavent code. As I mentioned, works fine the first time the modal window is shown but afterwards the webview refuses to show up.

    Here's where I'm adding two views to my "about" window. (still have the webview inside a regular view in this version)
    //about
    MC.ui.aboutWindow.add(MC.ui.aboutView);
    MC.ui.aboutView.add(MC.ui.aboutWebView);

    Binding the about button graphic:

    aboutGraphic.addEventListener('touchend', loadAbout);

    And the function that gets called:

    //swap windows
    var loadAbout = function() {
    MC.ui.aboutWindow.open();
    MC.ui.aboutView.show();
    };
