  • Provided that you configured everything correctly for mediation in the Admob console, the native module you are using (ti.admob, ti.dfp, etc.) will be able to show ads coming from other networks ONLY IF it has been built INCLUDING the specific extensions/libs for those networks.

    None of the above mentioned modules, as you find them on github, comes prepackaged with support for additional networks.

    You should download the module source from GitHub, then add each extension/lib following Google instructions here and rebuild.
