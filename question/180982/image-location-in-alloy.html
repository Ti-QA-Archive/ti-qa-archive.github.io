Image Location In Alloy`

So I'm creating an Alloy Project. I have an ImageView in my view, which I need to render a static image. This may be a really stupid question, but where in my project do I put said image file?

I'm using (or attempting to use) the Alloy framework.

    You need to put it in assets directory. If you want any specific images for platforms you are free to put it in iphone/android directories.

    For e.g. If you are placing test.png file inside app/assets directory. here is the example of how you can refere this image.

    index.xml

    
<Alloy>
    <Window>
        <ImageView image="/test.png"> </ImageView>
    </Window>
</Alloy>
