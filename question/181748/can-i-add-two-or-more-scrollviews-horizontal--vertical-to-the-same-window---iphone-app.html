Can I add two or more ScrollViews (horizontal + vertical) to the same window - iPhone app

Hi All,

I am creating the home page of my iPhone app which is quite long. So I have added a vertical scrollView to the Window which is fine.

But there is a requirement that I need to show data in horizontal scrolling way for some of the sub-sections in the window.

So I am using a horizontal Scroll View for the same, but my window is not behaving well i.e. if I add the horizontal scrollView to the window my whole window is scrolling horizontally whereas I want only that subsection to scroll horizontally.

So is it a good idea to add two/ or more ScrollViews to a window? How best can that be worked.

Is there some sample code which I can refer?

A part of the my code :

<Alloy>
<NavigationWindow id="homeWin">
<Window id="myVantageWin">
<ScrollView id ="parentScrollView" class="scrollView" >
        <View id="ParentView" class="detailView">
        <ScrollView id="childScrollView"/>
    </View>
</ScrollView>
</Window>
</NavigationWindow>
</Alloy>

My classes "scrollView" and "detailView" are as below :

'.scrollView':{
layout: 'vertical',
showVerticalScrollIndicator: 'true',
contentWidth: Ti.UI.SIZE,
horizontalWrap: false,
left: "0dp",
right: "0dp",
width: Ti.UI.SIZE,
scrollType: "vertical"
},

'.detailView':{
layout: 'vertical',
width: Ti.UI.SIZE
}

For horizontal scrolling I am using a common JS function as below :

function createHorizontalScrollr(scrollView) {
var viewWidth = 100;
var viewLeftPadding = 20;
var viewQty = 10;

var scrollViewWidth = (viewWidth * viewQty) + (viewLeftPadding * viewQty) + viewLeftPadding;
Ti.API.info("scrollViewWidth = " + scrollViewWidth);
scrollView.backgroundColor = "red";
scrollView.contentWidth = scrollViewWidth;
scrollView.contentHeight = "auto";
scrollView.backgroundColor = "red";

for (var i = 0; i < viewQty; i++) {
var thisView = Ti.UI.createView({
backgroundColor : 'green',
width : viewWidth,
height : 100,
left : viewLeftPadding + (viewWidth * i) + (viewLeftPadding * i),
//       left:viewLeftPadding, // for relative layout
top : 20
});
scrollView.add(thisView);
}
return scrollView;
}
exports.createHorizontalScrollr = createHorizontalScrollr;

Other details :
Mac OS X Version = 10.10,Titanium CLI Version = 3.4.2, Titanium SDK Version= 3.5.1.GA

    Hi Das, yes we can use two or more nested scrollviews but we have to be aware of making the UI user-friendly.

    I think that we can implement the nested scrollviews by just properly setting their scrolltype property & should work without any issues.

    Just try to create a sample app using the original Ti docs as the original docs are the most comprehensive material to accomplish the task in much detail.
