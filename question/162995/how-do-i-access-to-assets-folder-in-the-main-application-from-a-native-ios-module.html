How do i access to assets folder in the main application from a native ios module?

Application type: mobile

Titanium SDK: 3.2.0.GA

Platform & version: iOS 6.0

Hi, i need access to an image that is located on the main application from an native ios module.

    #import "TiHost.h"

NSString *imgName = @"KS_nav_ui.png";
NSString *resourcesPath = [TiHost resourcePath];
UIImage *image1 = [UIImage imageWithContentsOfFile:[resourcesPath stringByAppendingPathComponent:imgName]];

    or 

     UIImage *image2 = [UIImage imageNamed:@"KS_nav_ui.png"];
