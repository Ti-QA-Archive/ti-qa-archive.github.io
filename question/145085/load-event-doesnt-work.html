Load event doesn't work

Hello everybody,

I have this code:

var win = Ti.UI.currentWindow;

win.backgroundImage="/images/img1.png";

win.addEventListener("load",function(e){

// Some tasks!
});

The some tasks part didn't work ! …. I wander if there is a mistake with the event listener!

    Try the "open" event listener:

    win.addEventListener("open",function(e){
// Your code
});
