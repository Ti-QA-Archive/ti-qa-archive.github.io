    • Hello Vijay,

      Please see my code. You can add this in your app.js file. I hope it solved your problem. Its automatically call when your network connection changes. 

      
    Ti.Network.addEventListener('change', function() {

        if (Ti.Network.online == false) {

            var alertDialog = Titanium.UI.createAlertDialog({
                title : 'WARNING!',
                message : 'Your device is not online.',
                buttonNames : ['OK']
            });
            alertDialog.show();

        }
    });
