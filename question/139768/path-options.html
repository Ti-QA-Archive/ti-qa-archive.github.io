hi all - I am trying to build my first project in Titanium Studio and getting the following error:

Logfile initialized
[INFO] logfile = D:\Projects\Titanium\TiBountyHunter\build.log

  • You can have a look at the Installation Troubleshooting section from the Titanium Guides.

    If that doesn't work, you can paste the following in your command prompt:

    cls
cd \
echo %PROGRAMFILES%
echo %PROGRAMDATA%
echo %USERPROFILE%
echo %PATH%
echo %JAVA_HOME%
java -version
javac -version
python -V
git --version
rem if the ANDROID_SDK variable is configured, please delete it and run this script again
echo %ANDROID_SDK%
adb version
cd %PROGRAMFILES%
android list target
cd \
rem --- The following are optional ---
rem ----------------------------------
scons -v
rem ----------------------------------
echo %HTTP_PROXY%
echo %HTTPS_PROXY%
rem ----------------------------------
echo %SOCKS_PROXY%
rem ----------------------------------
rem

    And paste the output here, this will give everyone a better understanding of what might be the problem.
