Increasing Button Size While Sound Playing

hey all,
I wanted to increase the size of a button while a sound is playing. This is the event listener for the button. The button increases in size but does not go back to its original size when not playing.

goatButton.addEventListener('click', function (e){
    var goatsound =  Ti.Media.createSound({url:"sounds/goat.wav"});
        goatsound.play();

    if (goatsound.STATE_PLAYING){
        goatButton.width = 110; //incresing size
    };

    if (goatsound.STATE_STOPPED){
        goatButton.width = 100; //decreasing size (error is here)
    };

});

    You need to add a 'complete' event listener on the sound that changes the width:

    var goatsound =  Ti.Media.createSound({url:"sounds/goat.wav"});

goatsound.addEventListener('complete', function() {
    goatButton.width = 100; //decreasing size
});

goatButton.addEventListener('click', function (e){
    goatsound.play();

    if (goatsound.STATE_PLAYING) {
        goatButton.width = 110; //increasing size
    };

});

    I have moved the creation of the sound outside the button click event as you don't want to create this every time the button is clicked.

    You might want to think about what happens if the user clicks the goatButton twice - like testing for the sound already playing or disabling the button until the sound is complete.

    Hope this helps
