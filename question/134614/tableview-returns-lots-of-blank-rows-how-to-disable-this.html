TableView returns lots of blank rows, how to disable this?

I have a sample array with 4 records which I use as the datasource for a tableview. I have noticed that all 4 rows show in my tableview but for some reason my tableview keeps showing blank rows below this until it reaches the end of my screen. Is there any way I can only display the rows that have records and do not display blank rows?

  • Using the 2.0.0.v… SDK or later, try setting your TableView's height to Ti.UI.SIZE. This should make it only as tall as it needs to be to contain your rows.

    I'm assuming you're running this in iOS, since Android doesn't add the extra rows.
