How can I copy a binary file from Resources folder to another in appDataDir?

Hi, on the official guide there is this example for copying files (because Ti doesn't have a copy method)

var oldfile = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory,'old.txt');
var newfile = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory, 'new.txt');
newfile.write(oldfile.read()); // both old.txt and new.txt exist now

Is it possible to use it for binaty files?
Thanks for attention.

  • I found a solution. I don't know why in the official guide 'Building Mobile Applications with Titanium' there is this:

    "Titanium doesn't include a specific copy() method. Instead, you copy a file by combining reading and writing…"

    I made this

    var fileDaCopiare;
fileDaCopiare = Titanium.Filesystem.getFile(Titanium.Filesystem.resourcesDirectory, 'img/miniature/img01.jpg');
fileDaCopiare.copy('img/miniature/img01.jpg');

    and it works perfectly.
    I hope it can be useful for others. Bye.
