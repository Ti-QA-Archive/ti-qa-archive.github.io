tableView Click fireing twice.

I am opening my windows like this:

        var winEuro = new CorretorasWindow('Euro', tabGroup);
        tabGroup.activeTab.open(winEuro);

And inside this CorretorasWindow I have a tableView with an click event.

The first time I entered and I click a row the event fires just once. But if I go back on the navigation and enter the CorretorasWindow again, when I click the tableView row the event fires twice.

If go back and foward again, three times and so on.

  • Hi

    The problem is in the code you have not yet provided. However it is a common problem.

    You are creating the event listener that handles the click multiple times. This either means the code to create the event listener is added each time you go back to the 'navigation' or you have not destroyed the table once you no longer need it.

    Not sure which as I have no idea how your code adds the event listener and how the tableview is attached to the window or how the window is closed.

    Hopefully the simple point "the code to add the event listener is called multiple times" will point you in the correct direction. If not please provide a sufficient amount of code focused on this one point - NOT all the code.
