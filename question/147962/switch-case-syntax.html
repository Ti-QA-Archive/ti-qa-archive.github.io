  • Hi

    The answer to your question is this;

    switch (color) {
case 'red':
    alert('red');
    break;
case 'green':
    alert('green');
    break;
default:
    alert('none of the above');
    break;
}

    But this is NOT the answer to your problem.

    You should consider making your code (and life) easer by handling the logic first then dealing with the responses.

    As I have no idea what your code is trying to solve think about this sort of logic and apply it to your function.

    // logic first
var isTop = (point.y < (space.height / 2));
var isBottom = (point.y > (space.height / 2));
var isLeft = (point.x < (space.width / 2));
var isRight = (point.x > (space.width / 2));
// branches second
if (isTop && isLeft) {
   // do things that ONLY apply if top and left
}
if (isBottom) {
   // do things that ONLY apply if bottom
}

    This pattern will allow you to greatly simplify your code.
