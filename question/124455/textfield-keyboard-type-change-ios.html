TextField keyboard type change (iOS)

Hi,

I was having trouble changing the keyboardType when the keyboard is displayed and having it switch to something else. ie: move from ascii to dialpad.

I was having to change it, then blur and focus, the problem was it looked great on simulator but not on 3gs, the KB moves down then changes and moves back up.

If you want it to instantly change then in the SDK change this file

TiUITextWidget.m

-(void)setKeyboardType_:(id)value
{
    [[self textWidgetView] setKeyboardType:[TiUtils intValue:value]];

    /* CSM: If we're currently the first responsder (active) and we're changing the type we need to flush it so it displays the new KB type */
    if ([textWidgetView isFirstResponder])
    {
        [[self textWidgetView] resignFirstResponder];
        [[self textWidgetView] becomeFirstResponder];
    }

}

This removes the flicker and the KB type changes instantly without the blur/focus needing to be called up in JS land.
