How do you make multiple paragraphs in a single Titantium.UI.Label?

I'm wanting to create a label with multiple paragraphs. Can i do this in a single label or do I need to make a label for each paragraph?

    Hey,
    I also came across this scenario today. Thankfully it's really easy as all you do is add '\n' to the text property of the label. For example:

    { text: &#39;this is the first paragraph \n\n this is the second paragraph&#39;}

    I don't know if this works for all devices but I tested on iPad with Titanium 1.5.1.

    Hope that helps,
    Pat
