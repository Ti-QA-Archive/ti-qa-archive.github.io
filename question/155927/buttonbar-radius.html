  • Hi

    Short answer - no.

    This the default and expected behaviour and there is no way to change it. You can create your own variation, but this will require much more code than using the standard version.

  • Hi Wentao Ji,

    As above said you can't do it with ButtonBar, But, If you would like you can do it so by creating your own view with Buttons in it.

    see the Documentation here and also notes in it. It says as below
    http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.ButtonBar

    "Note that you cannot style individual buttons in a button bar. If you want to give a distinct visual style to individual buttons, to use an icon and text on the same button, or to use a button-bar type component on a platform other than iOS,you can use a set of individual Button controls wrapped in a View to create the appearance of a button bar."
