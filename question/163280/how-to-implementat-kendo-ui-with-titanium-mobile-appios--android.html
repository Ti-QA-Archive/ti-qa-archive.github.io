  • Hi

    This is a guess only, try renaming the file extensions of the following js files;

    <script src="media/kendo/js/jquery.min.js"></script>
<script src="media/kendo/js/kendo.all.min.js"></script>
<script src="media/kendo/console.js"></script>

    To these;

    jquery.min.tjs
kendo.all.min.tjs
console.tjs

    Then reference these as;

    <script src="media/kendo/js/jquery.min.tjs"></script>
<script src="media/kendo/js/kendo.all.min.tjs"></script>
<script src="media/kendo/console.tjs"></script>

    I never try to add a third-party UI via web views so no personal experience here. But I wonder if the compile process is seeing your webview JavaScript files and trying to parse them as if they are Titanium files. Changing the file extension means they will no longer be considered Titanium JavaScript files.
