  • Here is an example that pulls a a query of posts from a wordpress installation (php) and passes it as json:

    <?php
/** Absolute path to the WordPress directory. */
if ( !defined('ABSPATH') )
    define('ABSPATH', dirname(__FILE__) . '/');

/** Sets up WordPress vars and included files. */
require_once(ABSPATH . 'wp-config.php');
header('Content-type: application/json');
$results = $wpdb->get_results("SELECT p.ID as id, p.post_author as author_id, u.display_name as author, p.post_title as title, p.post_date as post_date, p.post_content as post_content FROM {$wpdb->prefix}posts as  p  JOIN {$wpdb->prefix}users as u on p.post_author = u.ID WHERE p.post_type='post' and p.post_status='publish'");
echo json_encode($results);
?>
