  • If it's not a true .PNG file (Such as you just renamed a .JPG or .GIF file to .PNG), you will get errors. Open it up in Paint or Photoshop and simply convert to a PNG and all your errors will go away. No need to bash Appcelerator/Titanium for a file problem caused by the user.
