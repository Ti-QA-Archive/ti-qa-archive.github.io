Getting Platform Width as an int

I am creating a custom tabBar for android and to make each tabs width size dynamically i need to divide the width of the device by the number of tabs to that no matter what the width in pixels the device is they will always display correctly.

Now I know that I can get the platform with this

Ti.Platform.displayCaps.getPlatformWidth

but if display the value in a label i get this

org.appcelerator.kroll.runtime.v8.V8Function@40615ed8

My code for display the label with the value of the tabWidth variable is as follows

var tabWidth = Ti.Platform.displayCaps.getPlatformWidth;

var label = Ti.UI.createLabel({
    text: tabWidth
});

main.add(label);

How do I get an integer value of the platform width?
