Compiling Multiple Projects at Same Time

Is it possible to compile two or more projects at the same time?

I manage a bunch of Apps and use a single core folder. It would be nice if I could press a button and compile all of them instead of going through each one when I make a change.

  • There is not a built in way to do that in Titanium Studio that I am aware of. That said, Titanium Studio is eclipse based, so it affords you a lot of opportunities to extend it to your needs. You could create your own build chain by using the existing python build scripts leverages by Titanium Studio, and putting a series of them in a single "run configuration" via a "python run".
