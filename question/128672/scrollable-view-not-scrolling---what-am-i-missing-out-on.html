Scrollable view not scrolling - what am I missing out on?

Hi All.
I am trying to make a scrollable view in my window containing text. The text is quite long and therefore I would like to place in its own scrollable view in order to make the rest of the window visible - the rest of the window contains a button and 2 other labels. Below is my code with the label and scrollable view - but the text is not scrolling… Can anyone help me on this? I cannot find any information as to why the view shouldn't be scrolling - the text goes beyond the length of the view, so the scrolling should be enabled autamatically -or?

Thanks a lot in advance!!

Dearest
Louise 

function labelMaker (myText, w, h, pos, fontS, fontC, fontF) {
    var myLabel = Ti.UI.createLabel({
        text:myText,
        width:w,
        height:h,
        top:pos,
        color:fontC,
        textAlign:'center',
        zindex:10,
        font:{fontSize:fontS, fontFamily:fontF}

    return myLabel;
    };

  • Set contentHeight/Width to 'auto' or to a fixed number greater then the height/width.
    Setting it in percents will take 90% from the current height therefore being smaller then the height will not scroll.
