  • Mohammad

    Have you seen the official guides? I would recommend you begin by working through the Getting Started menu at the top of that page, namely Getting Started with Titanium, Getting Started with Kitchen Sink, The Titanium Architecture and The Application Project Structure, and then move on to The Titanium JavaScript Environment, Introduction to JavaScript and JavaScript Best Practices.

    Personally, I find Mozilla's MDC Doc Center an invaluable general-purpose Javascript resource, not only for the guides and other documents it contains, but also the language reference, which I search using Google, like this.

    As others have recommended, the KitchenSink is an great working example of a Titanium project, and comtains almost all functionality provided by Titanium.

    Best of luck with your learning! :)
