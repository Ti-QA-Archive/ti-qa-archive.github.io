    Hi

    In order to dismiss the keyboard you have to blur to object that currently has it focussed. If you do not know which object had the focus last time add a routine to each of your textfields focus events and reference a cache variable with the current textfield reference. Then on the window click event blur that reference.

    Something like this - change to suit your needs.

    var cacheField; // global to THIS function
var saveCache = function (e) {
    cacheField = e.source;
};
txtA.addEventListener('focus', saveCache);
txtB.addEventListener('focus', saveCache);
win.addEventListener('click', function (e) {
    cacheField.blur();
});

    Something like that should do the job.

  • Or the brute force:

    for(var i=0, j=win.children.length; i<j; i++) {
  try {
    win.children[i].blur();
  } catch(err) {}
}

    Not super memory efficient but it works across platforms.
