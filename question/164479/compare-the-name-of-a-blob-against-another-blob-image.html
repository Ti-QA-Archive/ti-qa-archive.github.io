Compare the name of a blob against another blob? ("image")

Hello, is it possible to compare two blob agains eachother to see if they are actually the same image?

My structure of a function is based on that a image has to be provided when checking out (sending data to the server). And after, it's possible to re-send the data, but with another image. But I dont want to send another image if it's still the same image provided.

So basicly I just want to compare and see if the first uploaded image == the image which could be uploaded the second time when I run the same function?

(Developing for iPhone, Ti 3.2)

  • Hey Mathew,

    use Ti.Utils.md5HexDigest() to generate a md5 hashes of the blobs, then compare the hashes.
