No data showing in TableView

Hi.

I am developing an Android app using Titanium SDK 3.4.0.201409261227-26092014122818 on Windows 7, testing on a HTC One device. My Titanium Studio build info is:
Titanium Studio, build: 3.4.0.201409261227 Build: jenkins-titanium-rcp-master-197 (origin/master) Date: 26 September 2014, 12:28:18

I have a controller:

function cleanup() {
    $.destroy();
}

var transactions = Alloy.createCollection("transactions");
transactions.fetch();
Ti.API.info(JSON.stringify(transactions));

and a view:

<Alloy>
    <Collection src="transactions" />
    <Window class="container" onClose="cleanup">
        <TableView color="#000000" id="transactionTable" dataCollection="transactions">
            <TableViewRow color="#000000" created="{created}" amount="{amount}"></TableViewRow>
        </TableView>
    </Window>
</Alloy>

I also have a model:

exports.definition = {
    config: {
        columns: {
            "created": "datetime",
            "account_id": "bigint",
            "amount": "int",
            "card_number": "bigint",
            "account_name": "string",
            "reference": "string"
        },
        adapter: {
            type: "sql",
            collection_name: "transactions"
        }
    },
    extendModel: function(Model) {
        _.extend(Model.prototype, {
            // extended functions and properties go here
        });

        return Model;
    },
    extendCollection: function(Collection) {
        _.extend(Collection.prototype, {
            // extended functions and properties go here
        });

        return Collection;
    }
};

However, I just get a blank screen when I try to view the TableView even though this line:

Ti.API.info(JSON.stringify(transactions));

in my controller shows a valid JSON representation of my SQLlite database rows.

I have read numerous articles and I cannot see what I am doing wrong. I get no runtime errors when testing on my HTC One.

Can anyone help me to understand what I am doing wrong please?

Thank you.

  • Ok, having read this article:

    http://docs.appcelerator.com/titanium/latest/#!/guide/Creating_Your_First_Titanium_App

    I was under the impression you could simply add an attribute for each item of data in to the TableViewRow XML element as they did with this code:

    <TableView dataCollection="books">
            <TableViewRow title="{title}" author="{author}"></TableViewRow>
        </TableView>

    However, it seems this is not possible and in fact I should be using either the "title" attribute or a custom View element within the TableViewRow element.

    It's working ok now.
