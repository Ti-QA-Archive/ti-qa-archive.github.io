navigation stack rather than navigation group

I'm making an app (android / ipad) that has a complex navigation flow, nonlinear, but can be circular. The user must be able to move back and forward and vice versa.

Is there any way to stack and unstack views (or the views variables) so "global" to manage the "flow" of the app. Since android does not have navigation group, I think I can not take that option. Any other solution or example?

all suggestions are welcome! Thank you.
