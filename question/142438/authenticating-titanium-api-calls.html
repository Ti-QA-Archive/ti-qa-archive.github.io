authenticating titanium API calls

I would like for my titanium app to contact my main server. I would also want to authenticate the API call to make sure it is coming from titanium app, so that the outside world is not able to directly access the page. How can I do that? I am using PHP server side, and trying to build app for iphone

    All you have to do is create some unique identifier on the device and pass that along with your requests. Only the phone app and the web app will know the correct key or key format, so you can be sure that the request is coming from your app.
