A newer version of Titanium Mobile SDK

I'm trying to get started on my first app and have a few questions. I've installed Studio and the Kitchen Sink app appears to be working as expected, but when I create a new mobile project the dialog "warns" me that

A newer version of Titanium Mobile SDK is required for the Mobile Web deployment target.

I only have one SDK version installed (downloaded as recently as late last week) and it's 1.7.2. Is there a newer version? Should I just ignore this message or is there something I'm supposed to do? Studio's "check for updates" indicates that I have the latest.

Thanks.

    The Mobile Web SDK is currently in beta and only available to paid subscribers. You can safely ignore this error message. If you would like, you can open the raw tiapp.xml file for your project and change the mobileweb boolean to false, and the message will no longer be displayed.
