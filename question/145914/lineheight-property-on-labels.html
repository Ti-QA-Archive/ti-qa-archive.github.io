LineHeight property on labels?

I seem to recall being able to set the lineHeight (aka "leading") on a text label. That currently has no effect or an unexpected one. I have a label with fontSize: 14 and wanted lineHeight:16 but when I tried that only the first line appeared in the text label (perhaps the lineHeight was enormous?).

I searched the API for lineHeight and came up empty, and don't see it listed along with font etc.

  • Hi

    The answer is no - currently no native support for line height on labels, or any other text based elements.

    As to why you only see the first line, that may be related to either no width specified or one that is too wide for the device, try fixing a shorter width values. If you are using width: Ti.UI.SIZE - change it to a value for a test.
