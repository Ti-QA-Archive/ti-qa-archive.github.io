    Set at the first start a property with the current timestamp…then at every other start up compare with that timestamp the time it has been used. When it exceeded 30 days, you can throw your alert.

    // during startup....
if (!Ti.App.Properties.hasProperty('firstStart'))    {
    // set current date as string property 'firstStart'
}
else    {
    // check if 30 days have passed
    // by comparing the current date (new Date()) with the date from firstStart
}
