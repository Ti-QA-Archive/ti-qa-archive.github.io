  • Hola Kao Nima, I had the same problem and I've just read that the last ERROR show this: No such file or directory titanium_5_HVGA.avd so I look throught the directories C:\Documents and Settings\Administrator.android\avd\ and instead of titanium_5_HVGA.avd I had a titanium_7_HVGA.avd so I went to C:\Users\USER_NAME.android\avd and I found the titanium_5_HVGA.avd there so I just copied it and paste it in C:\Documents and Settings\Administrator.android\avd\ then I restart Titanium, run Android Emulator and the problem was solved.
