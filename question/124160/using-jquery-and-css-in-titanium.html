    1. You can call jQuery methods in your webview like any other webpage. You cannot, however use jQuery in the native Titanium code as jQuery relies on the DOM which is not used by Titanium for mobile development. This question from Q&A offers multiple methods for including local JS files in your webview.
    2. Again, you cab use CSS in the webview but not native Titanium code. Here's another Q&A question that covers the topic.
    3. Yes it is possible to call $.ajax({}); if you have jQuery properly loaded in your webview.
    4. Use application level events. Add these lines to your Titanium code and HTML respectively.

    in your Titanium code

    Ti.App.addEventListener('app:webview_event', function(e) {
    // handle event sent from webview
});

    in your webview

    <input id="search_Btn" type="button" title="Search" \onclick="Ti.App.fireEvent('app:webview_event', 'this can be any JSON serializable data');">

    Remove the backslash that is before the onclick
