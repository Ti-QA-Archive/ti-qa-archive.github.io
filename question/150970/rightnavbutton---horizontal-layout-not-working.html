rightNavButton - Horizontal Layout not working

I'm using the code below to try and set a View as the rightNavButton for a Window. The view does appear, but not as I'd expect. What I'd like to do is have the view's layout set to horizontal so it works if there is 1 button, 2 buttons, etc. and they can all be spaced out accordingly. Instead, Titanium just stacks the two buttons on top of each other. I tried putting borders on the rightNavButton and the view inside of it to determine where the problem lies but the borders do not appear either. What am I doing wrong here?

view.xml

<Alloy>
    <Window id="blah_window" class="tab_window">
        <TitleControl>
            <Label class="window_title" text="Window Title" />
        </TitleControl>
        <LeftNavButton>
            <Button id="back_button" onClick="back_button_click" />
        </LeftNavButton>
        <RightNavButton>
            <View id="right_nav_view">
                <Button title="English" class="right_nav_button" />
                <Button title="Font Size" class="right_nav_button" />
            </View>
        </RightNavButton>

        <Label text="Testing!" />
    </Window>
</Alloy>

view.tss

RightNavButton: {
    borderColor: "#00ff00",
    borderWidth: 1
}

".right_nav_view": {
    borderColor: "#ff0000",
    borderWidth: 1,
    layout: "horizontal",
    horizontalWrap: false
}

".right_nav_button": {
    left: 10,
    backgroundImage: "navBar_button_background.png",
    backgroundLeftCap: 4,
    height: 30,
    font: {fontFamily: "Open Sans [Regular]", fontSize: "13"},
    color: "#252525"
}
