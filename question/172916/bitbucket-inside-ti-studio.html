    We use bitbucket in our projects all the time. The key is to remember that git is git. The repo could be hosted on github, bitbucket, your own git server, doesn't matter.

    If this is an existing repo, you can import it just like you would from github.

    If this is a new project, be sure it is open. Right click on the project in Project Explorer. Your git functions are under 'Team'. You first 'create' (which is just git init). Then add your remote.

    I spend most (99.9999999%) of my time in sublime and the cli.

    In a project folder git init gets me started locally, create the repo on bitbucket, and then add the repo as a remote.
