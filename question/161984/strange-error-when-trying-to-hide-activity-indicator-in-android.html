Strange error when trying to hide activity indicator in Android

Hello,

I am getting a strange error when porting an app from iOS to Android. The app works fine in iOS but on Android when I call

actInd.hide()

I get the following error.

Uncaught TypeError: Property 'hide' of object #<ActivityIndicator> is not a function

This is my code:

        var actInd;
        actInd = Titanium.UI.createActivityIndicator({
            //style : Titanium.UI.iPhone.ActivityIndicatorStyle.DARK,
            height : '60dp',
            width : '60dp',
            hide : true,
        });

        each.addEventListener('beforeload', function(e) {
            each.add(actInd);
            actInd.show();
        });
        each.addEventListener('load', function(e) {
            actInd.hide();
        });

As you can see it is really simple. Any idea why this might be happening and how to solve it?

Thanks in advance!

  • Check your logs to see how many times each event is being fired. Android likes to fire events multiple times, which leaves you having semaphores sitting around, or checking to see if an object exists before hiding/removing.
