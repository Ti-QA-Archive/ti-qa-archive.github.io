How to add a custom rightButton to a TextField?

Hi!. I've tried to custom the right Button for a textField, but there is no image.

I can click the button, but there is no image shown.

This is the code I've been using.

var rightButton = Titanium.UI.createButton({
    color:'#fff',
    backgroundImage:'images/toolbaricon_cancel.png',
    backgroundSelectedImage:'images/toolbaricon_cancel.png',
    backgroundDisabledImage: 'images/toolbaricon_cancel.png'
});

rightButton.addEventListener('click',function()
{
    Titanium.UI.createAlertDialog({
        title:'Text Fields',
        message:'You clicked the right button'
    }).show();
});

var tx_keyword = Titanium.UI.createTextField({
    color:'#336699',
    top:5,
    left:70,
    width:245,
    height:30,
    hintText:'Restaurantes, Hoteles, etc.',
    keyboardType:Titanium.UI.KEYBOARD_DEFAULT,
    returnKeyType:Titanium.UI.RETURNKEY_DEFAULT,
    borderStyle:Titanium.UI.INPUT_BORDERSTYLE_ROUNDED,
    rightButton:rightButton,
    rightButtonMode:Titanium.UI.INPUT_BUTTONMODE_ONFOCUS
});

  • This is obvious, but just to eliminate the possibility I'll say it.

    The code you gave above is it in <yourapp>/Resources/ or is it in a folder, say <yourapp>/Resources/views/? If it is within a folder, I typically do:

    ../images/toolbaricon_cancel.png

    instead of 

    images/toolbaricon_cancel.png

    If that is not the case, please send a comment on this answer. My suspicion is that the script is having trouble finding the image file.
