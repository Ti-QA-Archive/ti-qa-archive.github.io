    • I found this in documentation

      Android Platform Implementation Notes
      On Android, the tab does not maintain a stack of windows. Calling open opens a new, heavyweight window, which by default covers the tab group entirely. This seems quite different from the iOS model, but it is the standard model for Android applications. Users can use the Back button to close the window and return to the tab group.

    • I am very sorry - I failed to notice you are using Android - not sure why - I thought this was iOS.

      In android you CANNOT open a window as a child of a tab group they always open as a window over the top of the tab group. Whilst it may appear some other apps can open a window inside a tab - they are simply showing views over the top of the previous one to give that effect.

      Your code should be;

      var newView = Alloy.createController("new_view", {}).getView();
newView.open(newView);

      You simply open the window.

    • Android apps do not have navigation in the way you may have experienced in iOS apps. They manage a similar approach in Android 4.0 and higher (technically 3.x but nobody uses it) using ActionBar and windows.

      You would have a root window, a tabGroup is fine that has an ActionBar - so you can add titles and menu options. The tabs do normal things like hosting a main app section. When you need to show different information you create a new window that sits over the top. It also has an ActionBar - the ActionBar can be set to include a navigate back icon so when you click it you can tell the window to close - showing the previous thing that called it.

      This is normal and the expected behaviour on Android these days.

      You can do other things if yu want to create a "facebook style side menu" but that takes a bit of extra effort. It is worth checking out the modules repository for versions already created - there are several types.

      • gitt.io search for "drawer" as a good keyword to get you going

      The NappDrawer is a favourite of many.
