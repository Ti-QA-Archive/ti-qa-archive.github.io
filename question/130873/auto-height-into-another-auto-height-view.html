auto height into another auto height view

Hello, I have a view with height:&#39;auto&#39;, containing other views, images and text, and the height increases and decreases correctly based on the content.

This view in container into another main view with height:&#39;auto&#39; but this view is not resizing if the internal view height change.
I can't use fixed height because content is very dynamic.

Any solution? Thank you.
