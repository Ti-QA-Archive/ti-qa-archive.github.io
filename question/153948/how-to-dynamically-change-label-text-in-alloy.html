How to dynamically change label text in alloy

Hi,

currently using Ti Studio 3.1.1, creating app for iphone. I am having problem changing label's text dynamically in alloy. If I have a label setup like this 


<Label id="textA"></Label>

how can I dynamically change the text from my js file?

I tried $.textA.setText(&#39;test&#39;) but this didn't work

thanks in advance.

    Hi

    That is the correct syntax for both parts. I just tried in my test app and it works fine.

    I added;

    $.textA.setText('test')

    To a button click as follows;

    function doClick(e) {  
    $.textA.setText('test');
}
    • The previous text changed to the new text.
    • I have tried it with and without original text in the Label.
    • I have tried it on the line above $.index.open() - also worked.

    Try adding some test text in the label mark-up to start with to make sure the label is actually visible on the window you are expecting to see it - chances are it is either too small to display or not within the visible section you can see.
