  • Actually, both of the links address his question. That page contains the answer though it's not very obvious.

    You can, in fact, call a php function from javascript.

    Simply define your php function as you normally would in the php section of your code. Then, in the javascript section call the function as you normally would. The Kroll bridge handles it smoothly.

    So, for instance, here's a simple php function:
    function test(){ return "Testing"; }

    In the javascript section do something like:
    var t = test();
    alert(t);

    It will generate an alert with the string generated in the PHP function.
