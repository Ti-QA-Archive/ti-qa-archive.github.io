Hint text of a text field in android is not coming

I have a text field in which i have given a hint text. The hint text is showing in iphone but not in android. Does hint text property is even there in android or not??

This code is showing the hint text in iphone but when i put it in android it only shows a text field but not any hint text

var tf1 = Titanium.UI.createTextField({

        height:100,
        top:245,
        hintText:'ABCD',
        textAlign:'right',
        left:154,
        width:105,font:{fontSize:20},
        enabled : false,

        });//End of create Text field

        win.add(tf1);

I am using 1.7.1 titanium sdk , 2.3.3 as android sdk and 4.3 as iphone sdk.
