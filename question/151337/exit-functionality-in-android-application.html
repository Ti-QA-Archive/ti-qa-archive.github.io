exit functionality in android application

hi,

i am developing an android application in 3.0.2 sdk

in this app i want exit from application on click of a button , i have used the following code for that :

var activity = Titanium.Android.currentActivity;

activity.finish();

when i click on the button i am getting following errors:

Error triggering 'click' event: Error: Invalid index 0, size is 0

thanks

  • You can't do in the Titanium context since you are still in a context (so to speak) so if you try to kill off the current activity by forcing it, you will get the error you just spoke of.

    The recommended way (maybe not the best, but this is the one that will always work) is to basically close the main window of your application.

    mainWin.close();

    From there, Titanium will know there is no point in running the application, since the main window isn't even there to interact with. So it will free the objects and close the activity.
