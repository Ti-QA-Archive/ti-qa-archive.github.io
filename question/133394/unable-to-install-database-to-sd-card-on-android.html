Unable to install database to SD card on Android

I'm trying to install database, that was shipped in APK into sdcard using example from
this page

$db = Ti.Database.install('/main.db', Ti.Filesystem.externalStorageDirectory + 'main.db.v1');

and I got this error:

Uncaught Error: File appdata:///main.db.v1 contains a path separator

Then if I change the code to look like:

var $db_file = Ti.Filesystem.externalStorageDirectory + 'main.db.v1';

$db = Ti.Database.install('/main.db', Ti.Filesystem.getFile($db_file).nativePath);

Then the error changes to

Uncaught Error: File file:///mnt/sdcard/com.alextime.alextest/main.db.v1 contains a path separator

I'm trying this in Android emulator and using Titanium SDK 1.8.2 on Windows 7x64.
