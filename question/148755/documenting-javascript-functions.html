Documenting javascript functions

Hi,

I just figured out that there is a default way for documenting javascript functions in appcelerator titanium function, which i will demonstrate in the next bit of code:

/**
 * Function foo is a funny bit of function
 */
function foo()
{
    alert("foo");
}

When doing &#x2F;** as the start of the documentation, it becomes blue (well in my theme it does), and when i am typing the function, the auto complete functionality of titanium shows the documentation i put in the comments.

All very awesome this is possible, but is it also somehow possible to comment attributes required by the function?

example function with arguments:

/**
 * Function foo requires an argument
 */
function foo(argument)
{
    alert("foo says: " + argument);
}

Soo, what do do in the comments to also let the argument be documented in the auto complete.

Hylke

  • You could also use the following in your comment block:

    @author
@param
@version
@return
@deprecated
@since
@throws
@exception
@see
@serial
@serialField
@serialData

    Find more info here: http://www.oracle.com/technetwork/java/javase/documentation/index-137868.html
