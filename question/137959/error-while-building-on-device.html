  • Like all external modules in Titanium, TiBar module needs to be included in the /Modules/iPhone folder and referenced in the TiApp.xml

    TiBar may need to be recompiled to work with the 2.0.x releases of Titamium but that only needs to be done once. Refer to this guide for consuming Ti Modules http://docs.appcelerator.com/titanium/2.0/index.html#!/guide/Using_Titanium_Modules
