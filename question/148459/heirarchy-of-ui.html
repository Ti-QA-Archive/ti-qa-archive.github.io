var parentView = Titanium.UI.createView({
    width : Titanium.Platform.displayCaps.platformWidth,

    Hello Rahal,
    your second label inherits the opacity of his parent and lives direct over his father. In Titanium all elements has other defaults as in web world. All elements has absolute position, are centered and fill parent.
    Hint: Please read API of views.

    Rainer
