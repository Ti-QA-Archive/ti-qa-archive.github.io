    Would you please tell me the Titanium SDK version (and date, if you are using the continuous builds), and Android SDK version that is exhibiting this issue? Really, you should include this information as a bare minimum for every single question you post to the Q&A.

    Remember that you can always view the logcat output directly, using the following command at the command line:

    adb -e logcat

    The adb executable is available in your android sdk directory.
