    As you are new in Titanium Studio Development please follow the Installation and Configuration instructions. And check the following steps:

    Step-1: Installing Oracle JDK

    Step-2: Installing Node

    Step-3: Setting up the Titanium CLI

    Step-4: Installing Platform SDKs
