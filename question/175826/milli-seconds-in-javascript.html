milli seconds in javascript

Hi i have a date picker

//xml
<Picker id="datepicker"></Picker>
//tss
"#datepicker":{
    top:"8%",
     minDate:Alloy.Globals.todaydate,
    type:Ti.UI.PICKER_TYPE_DATE
},
//js
// when i click on a button i'm getting value of picker

Ti.API.info($.datepicker.getValue());
//output  Mon Jul 07 2014 11:02:06 GMT+0000 (GMT)

//good im getting the correct value in milli seconds too 
Ti.API.info($.datepicker.getValue().valueOf());

//output 1404730926834

but the problem is the picker value should be Mon Jul 07 2014 00:00:00 GMT+0000 (GMT) pls help
