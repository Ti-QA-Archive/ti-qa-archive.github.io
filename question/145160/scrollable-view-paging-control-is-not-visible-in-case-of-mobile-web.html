Scrollable View Paging Control is not Visible in case of Mobile Web

Hi, I am using Titanium to develop an application which uses scrollable view in a window. There are two web view added to the views. And the views are binded to the scrollable view. I am able to scroll the web views from left to right and vice versa. But the problem is I am not visible with pagingControl at the bottom of the screen. Below is my code…

var win = Ti.UI.createWindow();

var view1 = Ti.UI.createView({

backgroundColor : 'white',

width : '90%',

height : '90%'

});

var view2 = Ti.UI.createView({

backgroundColor : 'red',

width : '90%',

height : '90%'

});

var WebView1 = Titanium.UI.createWebView({

width : '90%',

height : '90%',

backgroundColor : 'red',    

showScrollbars : false,

scalesPageToFit : false,

enableZoomControls : false

});

var WebView2 = Titanium.UI.createWebView({

width : '90%',

height : '90%',

backgroundColor : 'white',    

showScrollbars : false,

scalesPageToFit : false,

enableZoomControls : false

});

view1.add(idCardFrontWebView);

view2.add(idCardDepFrontWebView);

var scrollableView = Ti.UI.createScrollableView({

views : [view1, view2],

showPagingControl : true,

currentPage : 0,

scrollingEnabled : true

});

win.add(scrollableView);

win.open();

I want to execute this code in mobile web. I am able to scroll the web views. But the paging control is not visible at the bottom of the screen. Can anyone share a solution…

Thanks in Advance
