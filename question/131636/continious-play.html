  • if you want it to play the next on in the array you would need to reassign the sound.url to that of the next variable in the array. The way you have it set up it will have to preload before it starts playing. If you are looking into streaming it you would need to use the audio player for it.

    sound.addEventListener('complete',function()
    {
    sound.url="next value";
    sound.play();
    }
