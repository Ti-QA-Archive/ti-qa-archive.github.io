  • Hi

    If you mean the bit where the title normally goes then you use titleControl.

    var lbl = Ti.UI.createLabel({
   backgroundColor: 'orange',
   color: '#000',
   height: Ti.UI.SIZE,
   text: 'Hello',
   textAlign: 'center',
   width: 200
});
win.setTitleControl(lbl);

    You pass a view, label, button and it displays it. Keep in mind you can pass a view, that has a switch, button, label or any combination of them.

    Keep in mind you have a limited space to play with as the leftButton, rightButton and backButton all take up space.

    • Here is a technique to show a "mask" window over the top of the whole existing window, this way nothing underneath can be effected until it is removed.

      For testing purposes I have added a simple translucent mask and click window to remove, you should choose your own colour or translucent and then close the window when you need to.

      var win = Ti.UI.createWindow({
    backgroundColor: '#fff',
    exitOnClose: true,
    height: Ti.UI.FILL,
    modal: true,
    title: 'OptionDialog',
    width: Ti.UI.FILL
});
var btnL = Ti.UI.createButton({
    title: 'Left'
});
var btnR = Ti.UI.createButton({
    title: 'Right'
});
btnL.addEventListener('click', function (e) {
    alert('left');
    var win = Ti.UI.createWindow({
        backgroundColor: '#99000000',
        height: Ti.UI.FILL,
        modal: true,
        navBarHidden: true,
        width: Ti.UI.FILL
    });
    win.addEventListener('click', function (e) {
        win.close({animated: false });
    });
    win.open({animated: false });
});
btnR.addEventListener('click', function (e) {
    alert('right');
});
win.setLeftNavButton(btnL);
win.setRightNavButton(btnR);
win.open();
