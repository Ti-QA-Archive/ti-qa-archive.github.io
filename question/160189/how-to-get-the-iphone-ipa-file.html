Is there a way to get the IPA file without distribution method. The IPA has to be somewhere as the simulator is able run it. Any help would be appreciated.

  • Run you App on an iOS device. You can drag and drop your IPA from iTunes into the Finder. Therefor you don't need to distribute the app. AFAIK the simulator is running a *.app file, but not an IPA. Also note, that you will need a Provisioning Profile to package an IPA.
