Alloy change tss class depending on model value

I want to change views "tss" class depending on model value.

<Alloy>
    <TableViewRow>
        <ImageView id="myImageView" class="{myIcon}"></ImageView>
    </TableViewRow>
</Alloy>

Alloy generates following code for imageview:

    $.__views.contactIcon = Ti.UI.createImageView({
        width: 35,
        height: 35,
        left: 5,
        id: "contactIcon"
    });

The value containing "myIcon" get ignored.

I solved it this way:

if ($model) {
    $.contactsTableViewRow.model = $model.toJSON();

    var style = $.createStyle({
        classes: $model.get('myIcon')
    });
    $.contactIcon.applyProperties(style);
}

Does anyone have ab more dynamic approach ?
