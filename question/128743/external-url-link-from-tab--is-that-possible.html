External URL link from Tab : Is that possible?

Hi All,
I have a question which you may be able to help me with.

I have a tabGroup, in iPhone, and I would like to open an external URL directly from the tab.

So, I don't want to click on the tab and then open an internal JS file first,
I'd like the tab click to open the web browser. Is that possible?

So basically I'd like to do something like this (openURL doesn't work ;-)

var win3 = Titanium.UI.createWindow({  
    title:'Visit us on Facebook',
    Titanium.Platform.openURL: 'http://www.facebook.com',
});
var tab3 = Titanium.UI.createTab({  
    title:'Visit us on Facebook',
    window:win3
});


//  add tab
tabGroup.addTab(tab3);

Any thoughts?
Much appreciated!

Thanks,
Lee

    Any reason why you don't want to open an internal file first? Because if you did, you could easily do:

    var win3 = Titanium.UI.createWindow({  
    title:'Visit us on Facebook',
    url:'facebookwindow.js',
});

    then, in facebookwindow.js:

    Titanium.Platform.openURL('http://www.facebook.com');

    Of course, once the user returns to the app, then that tab is just blank. So you would probably want something like this instead for facebookwindow.js:

    openFace();

var win = Ti.UI.currentWindow;

var btn = Ti.UI.createButton({
    title:'Visit us on Facebook',
    height:100,
    width:200
});
win.add(btn);

btn.addEventListener('click', openFace);

function openFace() {
    Titanium.Platform.openURL('http://www.facebook.com');
}
