Getting the index of the current tab

Hello all,

As far as I read, the getCurrentTab() function of the TabGroup Object returns a Tab Object representing the current tab.

I'd live to ask if there is a way to get the index instead of the actual current tab?

    When you create your tab, add a new property (eg index) to it. Then, when you call getCurrentTab() you can retrieve your custom Index property.

    eg

    
Ti.UI.createTab({
    .... other properties...,
    index: 1
});
