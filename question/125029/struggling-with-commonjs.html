Struggling with commonJS...

I'm trying to set my project up using the commonJS methodology.

Everything works fine on the IoS simulator, but as soon as I try it on Android it crashes on the first line of app.js where the require('ui') command lies with the following error message:

E/TiJSError( 345): (kroll$1: app://app.js) [3,1179] - In file:///android_asset/Resources/app.js:1,0
E/TiJSError( 345): (kroll$1: app://app.js) [1,1180] - Message: Wrapped java.lang.IllegalArgumentException: size must be >= 0 (file:///android_asset/Resources/app.js#1)

I downloaded the code for commonJS (Forging Titanium Ep 2) and this works fine on both platforms, yet when I simply copy+paste the same code into my project I get the same error as above…

Have I simply got something like a build setting wrong perhaps? Any help would be extremely appreciated.

    There's a problem in 1.7.2 with CommonJS modules and the Android FastDev server. It should work fine on the device. That particular issue is fixed in the 1.7.x branch and master, so you can take a new build from there. Otherwise, you can turn FastDev off in your tiapp.xml.
