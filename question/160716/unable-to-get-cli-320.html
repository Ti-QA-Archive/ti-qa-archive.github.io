Unable to get CLI 3.2.0

  • Hi Fabio, this should be easy to fix. Run:

    sudo npm cache clean
sudo unlink /usr/local/bin/ti
sudo unlink /usr/local/bin/titanium
sudo npm install -g titanium

    Then verify you have Titanium 3.2.0 by running:

    ti -v
