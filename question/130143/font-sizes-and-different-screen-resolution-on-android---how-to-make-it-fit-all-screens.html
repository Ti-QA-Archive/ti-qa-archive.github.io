font sizes and different screen resolution on Android - how to make it fit all screens?

Hi all.
A very small but annoying piece of information missing in my knowledge (a bit embarrassing)…. I have a problem with font sizes and screen resolutions… How should I write the size of text so that it is the same size regardless of screen resolution?

    Use the sp unit. (which is the same as dp, but for fonts) This will ensure that the size is scales based on dots-per-inch (dpi).

    var label = Ti.UI.createLabel({
    width: 'auto',
    height: 'auto',
    font: {
        fontSize: '10.5sp'
    },
    text: 'Hello world!'
});
