  • Accepted Answer

    If you're using Titanium Studio try this:
    Go to Project -> Clean -> Select your project (or select to clean all projects) -> Click Ok.

    This will force a fresh build (and will remove your old compiled files and generate new ones). This will take a while.

    Regards.
