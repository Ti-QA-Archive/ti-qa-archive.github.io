i want to remove it's auto focus of textArea and searchBar Android

SDK : 3.1.1

OS : android

My android app contains one tableview with search bar so when i open that window SearchBar automatically focus i want to remove it's auto focus.

The second window contains the TextArea that also auto focus on window open i want to remove it's auto focus.

any solutions ???

  • Set softKeyboardOnFocus: Ti.UI.Android.SOFT_KEYBOARD_HIDE_ON_FOCUS while creating textArea.
    Then you have to manually give focus on click event:

    textArea.addEventListener('click', function() {
    textArea.softKeyboardOnFocus = Ti.UI.Android.SOFT_KEYBOARD_SHOW_ON_FOCUS;
    textArea.focus();
});

    and set softKeyboardOnFocus back to initial value

    textArea.addEventListener('blur', function() {
    textArea.softKeyboardOnFocus = Ti.UI.Android.SOFT_KEYBOARD_HIDE_ON_FOCUS;
});
