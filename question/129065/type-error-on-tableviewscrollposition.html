    • You'll need to add a 'name' to it when you're building the table. Can make it same as 'id'.

      row.name = row.id

      or when you declare

      var row = Ti.UI.createTableViewRow({
    name: 'someName123'
    //etc
});

      And then use name to get the index

      var row_index = view.getIndexByName(row_name);

      Where view is the TableView with that row.
      And then use index for scrollToIndex similar to what I'm doing in the question above.
