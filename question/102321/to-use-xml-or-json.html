  • JSON.

    adding some more reasons near @Alan's :)

    • The xml contains more information (it's a bigger file) that means loads slower if is remote and uses more memory.
    • On the android the XML parser is very slow.
    • Accessing an element in xml requires you to write more code than if it's json :)
