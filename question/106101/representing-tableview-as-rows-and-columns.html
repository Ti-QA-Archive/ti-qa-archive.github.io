  • To create the look of columns in a TableView, do not specify a title property. In each TableViewRow object, create a View to use as a container then place whatever objects you need such as multiple labels or text fields.

    By setting the placement of those labels/fields/etc, you will achieve the column look. Each object can use the usual plethora of properties like font size, text alignment, etc to achieve any look you want.

    Although not laid out in columns, the basic technique is demonstrated in some kitchen sink examples such as this.
