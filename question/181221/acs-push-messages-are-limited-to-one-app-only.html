    • An app is still only going to handle one APNs cert. So you would need two apps defined in ACS (I'm guessing that the two versions of your app have different bundle IDs, right?).

      So you define two apps in ACS, give each one the appropriate APNs cert that is tied to that bundle ID at apple, and then when it's time to send a notification, you use the ACS API to send it to each of the apps.

      I don't know if your push notifications are generated by automation software or if you send them manually.

      If you send them manually, you might have to build yourself a simple web app somewhere that takes the text of the notification from a web form and sends it via the ACS API to each app.

      If you're sending via automation, just add the appropriate HTTP calls to your automation code.

      In the answer to this question is an example of using the API to send a push notification.
