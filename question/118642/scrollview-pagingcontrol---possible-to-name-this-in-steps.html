ScrollView pagingControl - Possible to name this in steps?

When using ScrollView, is it possible to change the pagingControl to labels (Step 1, Step2. etc) instead of the dots and display to the right of the ScrollView?

How would this be done?

  • No, but you could just use a custom View that checks for the currentPage of the ScrollableView and changes accordingly?

    The pagingControl is Apple's own module that Titanium just takes advantage of. You can't change it beyond it's height and color.

  • Hi Dav M,

    var Sc = Ti.UI.createScrollableView({
     showPagingControl:false //hides the dots
});

    Then on the View Construction add a label at the bottom of every view to Read "Step 1","Step 2" … and so on
