2 questions about images

Hello,

I'm developing an iPhone app that takes a photo of someone and then lets the user drag imageViews over that photo.

  1. Is it possible to use a zoom/pinch gesture on a view/imageview in order to change its dimensions? I was only able to find a way to do it on a webview.
    I would like the user to be able to scale the imageViews that are dragged over the photo.

  2. Once the user finished dragging the imageViews over his photo, is it possible to copy the manipulated image (the photo with all the imageviews on it) and save it / post it to Facebook etc? I was thinking about doing a "print screen" but I'm not sure if thats possible with Titanium.

Thank you!
Idan

  • I don't know about the pinch-zooming, but wrt the 'print screen', you can do a toImage() of the containing view, or use takeScreenshot() to capture the whole screen (you can then use imageAsCropped() to then get the area of the image you want)
