TVOut working, but with black borders

I have gotten the TVOut module working in my app, and overall, it's working great.

But on the monitor that I'm using to test, the image consistently has a 1" black border around all 4 edges. It doesn't matter how I set the monitor, that border is always there. The TVOut module reports a monitor size of 1280x720, and the image/background on the screen is 1280x720, but there's still that black border.

I found someone else having this trouble in Objective-C, and their solution was to set the overscanCompensation on the UIScreen to 3. This will allow the TV to process the overscan correctly.
[stackoverflow answer][http://stackoverflow.com/questions/9465712/ios-screen-mirroring-shows-overscan-borders-on-selected-apps]

I'm far from an Appcelerator expert, so I'm trying to figure out if there's any way to set this property.

Right now, I'm using basically the example TVOut code:

var viewForTV = Ti.UI.createView({ backgroundColor: '#A00000' });
TVOut.setView(viewForTV);

Any suggestions? Thanks in advance!
