Titanium Stylesheet (.tss) Syntax error

Hi Folks,

I'm trying to set the width and height properties of a view like this:

"#someView": {
    width: Alloy.Globals.someValue + 'dp',
    height: Alloy.Globals.getDpValue(Alloy.Globals.someValue)
}

The first try is obviously causing a syntax error (unexpected token "+"). So I thought, I can write a function that returns me that value + 'dp' as a string. But I can't call functions from .tss files, so the second try (setting the height via function) also causes a syntax error.

Do you guys have an idea how to solve this?

Do I have to create this view dynamically in the controller or is there any chance to set density-independent size-values?

Thanks in advance.
