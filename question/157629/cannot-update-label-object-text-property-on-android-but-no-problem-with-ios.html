cannot update label object text property on Android but no problem with iOS

Been scratching my head on an runtime error i've been receiving only on Android and not iOS. I am trying to change the text of a label object dynamically after the object had been created. Below is the code which runs perfectly on iOS devices. But when run from an Android phone (not emulator), I get an uncaught typeerror; cannot call method 'applyProperties' of undefined. It appears distanceLabel is deemed undefined by Android devices. i have tried two different methods (version 1 and version 2 in the code below) to change the text property to no avail ….

I have even tried putting a dummy text property in the declaration part of the distanceLabel variable , just in case android needs there's a text property already present. still no luck. This seems like a fairly straightforward thing I am trying to do.

I would be most grateful if help could be offered here.

Thanks.

    var distanceLabel = Ti.UI.createLabel({
        color: 'blue',
    //    text: 'anything',    
        font: {
            fontSize: '12dp'
        },
        right: maxWidth * 0.02,
        width: maxWidth * 0.2,
        textAlign: 'right'

    });

    tableRow.add(distanceLabel);

    function gpsCallback(_coords) {

        if(_coords.latitude && _coords.longitude) {
            var lat1 = parseFloat(item.lat);
            var long1 = parseFloat(item.longitude);            
            var lat2 = _coords.latitude;
            var long2 = _coords.longitude;

            var distance = getDistance(lat1,long1,lat2,long2);

            // version 1
            distanceLabel.text = distance.toFixed(1) + 'km';

            // version 2
            distanceLabel.applyProperties({
                text: distance    
            });
        }
    }
