Update View/Add View Realtime for Current Window

I am struggling to find any code that will add views to another view once the window is created. I have been searching for a while but am struggling to find out how?

Currently I have a scroll view and I am adding another view on a button click, but it is not appearing anywhere. I think I have to update the view some how. Below is the extracts of my code at the moment:

How I created the 'holder' view

var subjectHolder = Ti.UI.createScrollView({
    showVerticalScrollIndicator: true,
    contentHeight: 'auto',
      scrollType: 'vertical',
    top: Ti.Platform.displayCaps.platformWidth*0.65+"px",
    width: Ti.Platform.displayCaps.platformWidths
});

Run on button click

function DisplaySubject(holder){
    var viewHolder = Ti.UI.createView({
        top: Ti.Platform.displayCaps.platformWidth*0.02,
        width: Ti.Platform.displayCaps.platformWidth,
        height: Ti.Platform.displayCaps.platformWidth*0.13,
        color: "#0066CC"
    });

    holder.add(viewHolder);
    holder.show();
}
