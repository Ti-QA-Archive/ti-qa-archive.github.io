  • Mark, despite the fact that you code in javascript, the applications you are making are 100% native. So what you are asking is not what Titanium offers.
    If you don't want to go native you can try to use PhoneGap, but if you want real performance then you should look at what you need from the html5 features and try to find the similar ones in the native APIs ti gives us.

    If you insist going with a webview the html5 features supported by the webview Ti can create are the same as what the an webview will give you coding in objC or java.
    The above is available for mobile of course.

    As for desktop you should have all the html5 features available in webkit, but again you can also combine this with native and unrestricted access to the filesystem, sqlite db, etc
