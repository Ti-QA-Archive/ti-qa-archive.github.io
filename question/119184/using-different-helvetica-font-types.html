Using Different Helvetica Font Types?

Hello,

I'm trying to use different Helevetica Font Types i.e. Regular, UltraLight, Light and Condensed Bold but couldn't get it through.

I successfully managed to install them using the custom font parameter but when I am trying to define them in the code it just shows up Helvetica Neue Light.

That is probably because whole Helvetica Family shares the same font name "Helvetica Neue" while there are different types within that family such as "Condensed Bold" etc.

So, do anyone have any idea how to use all these different types of font family within Titanium Mobile iPhone app?

Titanium SDK Version: 1.6.2
iOS SDK : 4.3
Platform: iOS

  • I've managed to get this to work, you just have to alter how you reference the font variants.

    With Helvetica Neue, you can use the default iOS font for the regular weight, then include the Condensed Bold as a custom font.

    //Default found available in iOS
var label1 = Ti.UI.createLabel({ 
    text:'Regular', 
    font:{fontSize:22,fontFamily:'HelveticaNeue'} 
});

//Custom font - I've included HelveticaNeueCondensedBold.ttf
var label2 = Ti.UI.createLabel({ 
    text:'Condensed Bold', 
    font:{fontSize:22,fontFamily:'HelveticaNeue-CondensedBold'} 
});

    Here's another example where I've pulled it off with two different font weights where the font name in Font Book is the same.

    My info.plist file looks like this:

    <array>
    <string>ui/fonts/MuseoSans700.ttf</string>
    <string>ui/fonts/MuseoSans900.otf</string>
</array>

    And my code looks like this:

    //Museo 700
var label3 = Ti.UI.createLabel({ 
    text:'Museo 700', 
    font:{fontSize:22,fontFamily:'Museo Sans'} 
});

//Museo 900
var label4 = Ti.UI.createLabel({ 
    text:'Museo 900 (a bit heavier)', 
    font:{fontSize:22,fontFamily:'MuseoSans-900'} 
});

    Hopefully that helps.

  • Altaf I think the issue is actually pretty complex.

    Actually there seem to be a pretty severe limitation in iOS's UIKit itself: see this iOS dev question.

    The solution is to rename the fonts so you can use a different name for light/regular/condensed variants and then just select normal/bold/italics styles as usual.
