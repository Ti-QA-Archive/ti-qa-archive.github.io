Newbie! Trying to add scrollableView to window, no dice

I am stumped and have no idea what to do so I ask you, o' great Ti devs, what the hell am I doing wrong!??!

Here's my code, all in app.js:

var url = "http://test.com/myservice/getdata";
var win = Ti.UI.createWindow();
var scrollView = Ti.UI.createScrollableView(
    {
        showPagingControl: true
    }
);

var title, view;
var xhr = Ti.Network.createHTTPClient(
    {
        onload: function()
        {
            var titles = JSON.parse(this.responseText);

            for(var i=0; i<titles.length; i++)
            {
                title = titles[i];
                view = Ti.UI.createView(
                    {
                        backgroundImage: title.BackdropUrl
                    }
                );                

                scrollView.addView(view);
            }
        }
    }
);

xhr.open("GET", url);
xhr.send();

win.add(scrollView);
win.open();

The service is working because I was able to do Ti.API.debug(title.BackdropUrl) and see results. That prop is an image url. When opened all I see is a big grey window. Are there required props to createView that I'm missing? Ideas?

Thanks a ton!

Jeff

    I'm afraid you cannot set a remote url as a background image.
    Build an imageview instead of a view as a page for scrollableview.
