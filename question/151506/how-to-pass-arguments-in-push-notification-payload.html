How to pass arguments in Push Notification Payload?

How can I pass arguments for my app in Push notifications, so that the app receives the information?

My App event receives and can only interact with push notification actions which was send local.

Ti.App.addEventListener('resumed', function(e) {
    if (Ti.App.Properties.getString('sync') != 1) {
        if (Ti.App.getArguments().hasOwnProperty('url')) {
            Ti.App.launchURL = '';
            pushURLAufrufen(Ti.App.getArguments().url);
        }
    }
    Ti.App.Properties.setString('localPush', 0);
});

Ti.App.iOS.addEventListener('notification', function(e) {
    if (Ti.App.Properties.getString('sync') != 1) {
        pushURLAufrufen(e.alertAction);
    }
});

I am using Titanium SDK 3.1 and I am developing for iOS. The app receives the notifications via APNS. But when I click on a notification, no arguments will be pushed to the app?

The JSON at this time:
{"aps":{"alert":{"action-loc-key":"open","body":"Hallo","loc-args":{"url":"thisIsTheURL"}},"sound":"soundeffects\/push.aiff","badge":0}}
