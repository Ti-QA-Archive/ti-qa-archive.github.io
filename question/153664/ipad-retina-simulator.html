iPad retina simulator

Hi, i'm trying to run my app on the ipad retina simulator, but Studio always launches the regular simulador. I try switching to retina running the simulator sepatedly and then from within the studio, but it doesn't work.

Any suggetion? I'm using studio 3.1.0 and SDK 3.1.1

PS: i followed this steps:

  • Run to the iPad in the current resolution

  • (If your simulator is not normally on your dock - right click it and select options > keep in dock)

  • In the simulator find "hardware" menu

  • Then device and change as you need

  • The simulator will close - this is normal (stupid but normal)

  • Run the simulator manually (from the dock)

  • Change the option again (is still stupid but is still normal)

  • The simulator will now be in the new resolution

  • Run your app from Titanium again

  • It will now run your app in the new resolution

    Hi

    This process has now changed and is much easier;

    • Select your app in the Project Explorer
    • Press the play icon on the toolbar
    • Choose the Run Configuration option
    • On the right;
      • Choose the Device as you require it
      • Choose the Display Type as you require it
    • Run

    You can now set these up per project.
