Custom font not aligning properly within window titles

When embedding a custom font (TradeGothicLTStd-BoldObl) in a window (by setting Window.titleControl to a custom Label) I'm unable to get the font text to center within the title area, or to force it down vertically.

I have tried setting top, bottom, width & height, auto height, etc. I tried setting the Label height to much larger than the navbar height. Nothing I've tried gets the font to budge downward.

Here is a screenshot

Also note the right edge of the oblique text is cut off, which might be a separate issue.

The source:

Titanium.UI.setBackgroundColor('#fff');
var titleLabel = Ti.UI.createLabel({
    text:'Hello World',
    color:'#ffffff',
    font:{fontSize:40,fontFamily:'TradeGothicLTStd-BoldObl'},
    height:44, top:0
});

var theWindow = Titanium.UI.createWindow({titleControl:titleLabel});

var tabGroup = Titanium.UI.createTabGroup({ barColor:'#5050d0' });
tabGroup.addTab(Titanium.UI.createTab({ window: theWindow }));
tabGroup.open();

Probably a related issue:
When I used the custom font in a Label in a Window, there was about 10 points of mystery padding added to the bottom of the text.

So, is this a bug or does someone have a workaround?
