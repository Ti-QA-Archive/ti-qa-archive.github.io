Changing the color of the table text

How to change color of the table text?

I tried like this:


var tbl_data = [
    {title:'MenuItem1'},
    {title:'MenuItem2'},
    {title:'MenuItem3'},
    {title:'MenuItem4'},
    {title:'MenuItem5'}
];

var table = Titanium.UI.createTableView({
    data:tbl_data,
    backgroundColor: '#00F',
    color: '#FF0'

});

I managed to change the background color, however I was unsuccessful with the text color.

    try this:

    var tbl_data = [
    {title:'MenuItem1',color:'green'},
    {title:'MenuItem2',color:'green'},
    {title:'MenuItem3',color:'green'},
    {title:'MenuItem4',color:'green'},
    {title:'MenuItem5',color:'green'}
];

var table = Titanium.UI.createTableView({
    data:tbl_data,
    backgroundColor: '#00F',
    color: '#FF0'

});

    this code is working here, hope this will work for u.
