Problems with blur() on searchbar attached to tableview

Hi all,

I'm having some problems with a searchbar attached to a tableview.

I want the searchbar to search my sqlite database and change the tableview data to the data returned by my sql search. And that works just fine - my problem is, that after the sql search is finished I try to fire search.blur(); but nothing happens. If I add an eventListener, the blur function is fired, but the searchbar isn't blurred.

Furthermore, I would like to avoid the "No results" and just have the semi-transparent black background, is the possible?

I'm building with TiDev 1.2.2, using Mobile SDK 1.5.1 and iOS SDK 4.2.
