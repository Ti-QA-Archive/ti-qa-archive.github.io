    hi,

    in alloy , you can get current window by its id defined in .xml file.

    like in your case i think search is a window so just check search.xml where you set window tag with its so you can use it in .js file

    something like below

    in search.xml

    <Window id="winSearch">
</Window>

    in search,js

    var currentWin = $.winSearch

    by this way you can get current window in Alloy.

      var tab = null;
