Android: Pop up text input?

                      var dialog = Ti.UI.createAlertDialog({
                        title: 'Recipient Username',
                        style: Ti.UI.iPhone.AlertDialogStyle.PLAIN_TEXT_INPUT,
                        buttonNames: ['OK']        
                        });

    1. Create a view with a transparent background that covers the entire device screen width/height (to prevent taps being processed)
    2. create an inner view. Give it a background color and border radius (for the rounded corner effect)
    3. Add a label
    4. Add a textField
    5. Add ok/cancel buttons
