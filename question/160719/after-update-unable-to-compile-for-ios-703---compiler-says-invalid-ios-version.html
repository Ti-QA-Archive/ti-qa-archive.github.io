  • You are using Titanium CLI 3.2.0 which is good, but you are trying to build an iOS 7.0.3 app using Titanium SDK 3.1.3.GA which is bad.

    Titanium SDK 3.1.3.GA only supports up to iOS 7.0.0. It does not support iOS 7.0.3 or newer. Likewise, Titanium SDK 3.2.0.GA only supports iOS 7.0.0 and newer. You can still target iOS 6.x devices, but SDK 3.2.0 does not support Xcode 4.x.

    To build for iOS 7.0.3 or newer, you must install Titanium SDK 3.2.0.GA, then set the <sdk-version> in the tiapp.xml to "3.2.0.GA" and rebuild.
