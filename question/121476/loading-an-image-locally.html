Loading an image locally

Hi,

I'm trying to show an image on a window using this code:

var imgs = ['coin01.png', 'coin02.png', 'coin03.png'];
var images = new Array();
Ti.API.info('imgs.length: ' + imgs.length);
Ti.API.info('imgs[0]: ' + imgs[0]);
for (var i = 0; i < imgs.length ; i++){
  Ti.API.info('image['+ i +']: ' + Titanium.Filesystem.resourcesDirectory + Titanium.Filesystem.separator + 'images' + Titanium.Filesystem.separator + imgs[i]);
  var myimg = Ti.UI.createImageView({
      image: 'coin03.png',
      width : 34,
    height : 34,
    myID : i,
    left : ((75*(i+1))+10),
    top : 20
  });
  Ti.API.info('xmyimg '+ i +': ' + myimg.toImage());

  images[i] = myimg;

  win1.add(images[i]);
};

The image dont show up, and I get this in my log:
> Bitmap bounds could not be determined. If bitmap is loaded, it won't be scaled.

I have my images in Resources&#x2F;images but I have tried to place one of the images in Resources too - nothing helped…

My system is:
Windows 7 x64
Newest Titanium Studio and Android simulator. Android API is 2.2

Must be trivial to anyone that has used Titanium for more than some hours. Thanks in advance
