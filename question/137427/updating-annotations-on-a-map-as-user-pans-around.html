Updating annotations on a map as user pans around

I'm working on an app that uses a map view. At first a number of annotations are added to the map around the user's current location from a database. When they move around the map more annotations should be added on the new areas of map.

What is the best way to only add annotations that are not already on the map? Currently this is my thinking:

- retrieve points from database for the visible map bounds

- create annotations and add to an array and to the map

- If user pans the map to a different area, retrieve points within the visible map bounds from database again

- loop through the array and filter out any points that are already in the annotation array

- add any new points to the annotation array

- remove the all annotations from map

- add the updated array of annotations to the map

Do you think this is the best approach especially considering performance and memory usage?

Thanks
