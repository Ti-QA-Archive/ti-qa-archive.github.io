  • [ERROR] : message = "'undefined' is not an object (evaluating 'Ti.Map.createAnnotation')";

    means: Ti.Map is undefined. Map is not part of standard namespace. You must import it by require.
    API: http://docs.appcelerator.com/titanium/3.0/#!/api/Titanium.Map
