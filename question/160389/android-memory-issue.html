  • Hi Nash,

    If you are concerned about the android emulator, then go Menu -> Manage apps -> 'Your app' -> Force Stop -> Clear cache. This will help if really your app is using cache memory.

    For efficient memory management, try to avoid creating projects with multiple windows, this always occupies memory. Instead do use multiple views in single window, so that you only switch views and remove them from parent window if you don't need them any more.

    Hope this helps.
