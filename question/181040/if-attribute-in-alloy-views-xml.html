if attribute in Alloy Views XML

In the Alloy documentation there is an example for using the if attribute in Alloy Views:

function openBar (e) {
    Alloy.createController('win2', {'fooBar': true}).getView().open();
};

<Alloy>
    <Window>
        <Label if="$.args.fooBar" color="blue">Foobar</Label>
        <Label if="$.args.fooBaz" color="red">Foobaz</Label>
    </Window>
</Alloy>

However, using the sample code results in an error:

message = "undefined is not an object (evaluating '$.args.fooBar')";

Putting the boolean into Alloy.Globals.fooBar works.
Is this broken or am I missing something?

  • You can indeed use the if-attribute for this but since the UI comes before the controller code, fooBar won't work.It should be either a global or an arg (e.g. arguments[0]. fooBar === 1)
