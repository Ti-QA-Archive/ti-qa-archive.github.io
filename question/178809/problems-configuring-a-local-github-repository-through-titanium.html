  • Hi Fred Strout,

    Thanks for your concern. You can do it manually and this will be better just follow this.

    After creating your project login to the guthub and create a new repository project using the same name, which is created you before. Then select your project and open the studio terminal and paste this command respectively. When you will create repository, you will get this command also.

    [Note: At first close the terminal selects the project, then open the terminal so that this project is selected for uploading]

    git init
git add --all
git commit -m "first commit"
Git remote add origin "URL" // you will get the URL  when you will create the repository just copy and paste without " "
git push -u origin master

    If you have any question then feel free to ask me.

    Thanks.
