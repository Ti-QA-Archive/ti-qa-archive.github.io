  • Sorry for the late reply everyone, after I ran into more issues with Titanium, I have given up on it and disregarded coming to this forum.

    I did quite a few things so I am sure which one actually fixed it, hopefully it will help everyone out.

    First I installed a fresh version of Titanium. Then I used this patch to fix the libcore file: http://developer.appcelerator.com/blog/2010/05/titanium-and-iphone-os-4.html

    After that, I updated the logger.py file using this: http://developer.appcelerator.com/blog/2010/07/introducing-titanium-mobile-1-4.html

    After that I deleted the iphone build folder or everything under the build folder in KitchenSink so that everything will be built with your SDK.

    Good luck and Happy Monday.

    Hai
