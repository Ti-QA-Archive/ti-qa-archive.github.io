Delete row with header doesn't delete header

Hi

I have a row like this:

var row = Ti.UI.createTableViewRow({
    title: 'test',
    header: 'delete me plz'
});

when I do do myTableView.deleteRow( row ); the header "delete me plz" remains. How do I make sure the header gets deleted as well WITHOUT using .setData() again?

Thank you.
