Android e-mail dialog with recipient pre-populated

Does anyone know how I can open an e-mail dialog on Android and pre-populate the recipient field.

On iOS I use:

Titanium.UI.createEmailDialog()

but this does not appear to work on Android (it doesn't seem to for me anyway)

so instead I am using Intents e.g

var intent = Ti.Android.createIntent({
                action : Ti.Android.ACTION_SEND,
                type : 'plain/text',

            });

            intent.addCategory(Ti.Android.CATEGORY_DEFAULT);
            try {
                Ti.Android.currentActivity.startActivity(intent);
            } catch (ex) {
                Ti.UI.createNotification({
                    message : 'No sharing apps installed!'
                }).show();
            }

but I can't see how to set the recipient automatically. I have tried:

intent.putExtra(Ti.Android.EXTRA_RECIPIENT, 'test@test.com');

but the recipient field remains blank. Does anyone know?

    Unfortunately as it stands, the recipient extra for Android intents expects an array value. Titanium is currently only capable of sending strings and numbers for most extras. Adding this via module is possible, and/or you may want to log it as a ticket in the Titanium Community Jira project.
