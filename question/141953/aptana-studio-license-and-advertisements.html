Aptana Studio license and advertisements

Hello,

Reading the terms and conditions here: http://www.appcelerator.com.s3.amazonaws.com/blog/www/appc-appexplore-agreement.pdf it says (last few sentences of point 2.3) that Appcelerator can place ads within the Apps that are built using Appcelerator products.

Please could you tell me if this is applicable only if we use Titanium Studio as the IDE for development or does Appcelerator still hold the right to place random ads in the apps if we use the Aptana Studio as the IDE (instead of the Titanium Studio).

Since Aptana is open source under GPL , while Titanium Studio is under Appcelerator's license - which has a free and commercial plan, I would believe that Appcelerator would not place any ads if we use Aptana Studio.

Also, technically, is the origin of the ads into the Apps the Titanium SDK or the Studio/IDE?

Please let me know as this is a very important issue for us.

  • Vineet,

    Thanks for your inquiry. As you suggested, the Aptana Studio SDK is under the terms of the GPL license (http://www.gnu.org/copyleft/gpl.html). The AppExplore EULA, which Titanium Studio uses, does not apply with use of the Aptana Studio SDK. You can refer to the GPL license for more information on using Aptana Studio SDK.

    Thanks,
    Greg

  • Vineet,

    The AppExplore EULA (and the right to display ads) would not apply to the set of components you're referring to. Taking this approach though would prevent you from utilizing key features of TiStudio (debugging for instance).

    If you are concerned about any stipulations in the AppExplore EULA, I would recommend looking at the App Accelerate Standard or other enterprise versions of Studio.

    Thanks,
    Greg
