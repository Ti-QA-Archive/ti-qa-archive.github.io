  • As emerged in the comments discussion, to solve this error: 

    [ERROR] :  Failed to run dexer:
[ERROR] :  
[ERROR] :  trouble processing "javax/xml/namespace/QName.class":
[ERROR] : 
...

    You have to add the maps.jar library to your build path:

    In Titanium Studio do:

    • right click > Build Path > Configure Build Path
    • in libraries tab click "add external JAR"
    • choose [your-android-sdk]/add-ons/addon-google_apis-google-19/libs/maps.jar
