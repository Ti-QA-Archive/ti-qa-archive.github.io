  • This error only means that the Android teams needs to use updated libraries when building the emulator. It is simply a warning. Your start up problems are unrelated to this.

    As you move over to Android, I suggest that DDMS becomes your friend. You can run DDMS manually from the command line {android-sdk-directory}/tools/ddms.

    Alternatively, you should probably install the Java Development Tools and the Android Development Tools. This will give you a DDMS view within Titanium Studio. Allows you to see how far through the install process you get before your app fails to install/start.
