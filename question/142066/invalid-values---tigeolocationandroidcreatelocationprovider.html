Invalid values - Ti.Geolocation.Android.createLocationProvider

Hi,

I want to implement the manual mode for geolocation events on android. It works at this point but I get error messages on each event stating that minUpdateTime and minUpdateDistance are invalid values and default values are used.

Any hints how these values have to be formatted or whats goin on? Thanks!

Titanium.Geolocation.Android.manualMode = true;

var _gpsProvider = Ti.Geolocation.Android.createLocationProvider( {
  name : Ti.Geolocation.PROVIDER_GPS//,
  minUpdateTime : 10, // Max location events per given seconds
  minUpdateDistance : 50 // Fire only if distance has changed for 50m at least
} );

Ti.Geolocation.Android.addLocationProvider( _gpsProvider );

  name : Ti.Geolocation.PROVIDER_GPS//,
  minUpdateTime : 10, // Max location events per given seconds
  minUpdateDistance : 50 // Fire only if distance has changed for 50m at least
} );

Ti.Geolocation.Android.addLocationProvider( _gpsProvider );

gpsProvider.minUpdateDistance = 10;
gpsProvider.minUpdateTime = 50;
