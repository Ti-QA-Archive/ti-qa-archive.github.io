Get Selected Contact Titanium

Hello Guys,

I am working on my first Titanium app, I need to show the phone's contact to user and allow to select a contact from list

I use following code

function showContacts() {
        Titanium.Contacts.showContacts({
            selectedProperty : function(e) {
                alert("some thing selected");
                alert("Selected phone #: " + e.value);
                Ti.API.info("Selected phone #: " + e.value);
                alert("Type of phone #: " + e.label);
            },
            fields : ['phone', 'name']
        });
    }

The issue is that I am not able to get the 'select event' I am able to show contacts list to user but when user selects the contacts nothing happens no event fires, the view returns to my app

Any ideas on how to get name and phone number of selected person

  • Hi

    I have just tried your code and it works correctly for me, so I wonder if the problem is how you think it should work.

    As you have selected to use the method selectedProperty the expected user bahavious is as follows;

    • Contacts appear
    • Select Contact
    • Select Property being sought (phone in your case would be a good start)

    If you are expecting to simply select the contact then you need to stop using selectedProperty and use selectPerson instead.
