Arranging Text Elements..

Hello,

I'm currently using the mobile SDK to create an iPhone app.

The app fetches text data (title, abstract, date etc.) from a webservice (among other different things). I'm currently failing at arranging these texts in a "native" manner, i.e. putting them into text-views (rather than putting them into an html-dom and simply using a webview). I tried using either labels or uneditable textareas with different "top" values wrapped in a scrollview and they never would land where I intended to put them. The problem seems in the missing "height" values, which I can't set beforehand due to the arbitrary length of the fetched text.

var someWindow = Ti.UI.createWindow({title: 'Some Window'});

var container = Ti.UI.createScrollView({
    showVerticalScrollIndicator : true,
    verticalBounce: true,
});

var titleLabel = Ti.UI.createLabel({
    top: 0,
    text: titleWithArbitraryLength,
    font: {fontSize: 16}
});


var abstractLabel = Ti.UI.createLabel({
    text: abstractWithArbitraryLength,
    top: titleLabel.height
});

container.add(titleLabel);
container.add(abstractLabel);

someWindow.add(container);

In this example, the titleLabel shows up in the vertical center of the view and the abstractLabel can't even be seen, while I expected titleLabel to show up on the top of the view and the abstractLabel directly beneath it..

What is the best way to achieve what I'm trying to do?

Greetings,
Danni
