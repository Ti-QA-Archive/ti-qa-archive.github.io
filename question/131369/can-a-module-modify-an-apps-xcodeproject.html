Can a module modify an app's xcodeproject?

Is there any supported method to modify the generated xcode project and source? In particular, I'd like to do this from an appcelerator module, but I suspect the module code isn't executed until it's compiled and loaded onto the phone, and there's no way to modify the target app's settings (libraries, etc) from a module. Is that correct?

Thanks!

  • You can write a python plugin script that might be able to do that.

    Write your script, and add it to $HOME&#x2F;Library&#x2F;Application Support&#x2F;Titanium&#x2F;plugins&#x2F;{plugin_name}&#x2F;{plugin_version}&#x2F;plugin.py

    Alternatively, you can put a plugin directory in your project folder.

    Take a look at the kitchen_sink project for some inspiration. There is a little plugin that demonstrates logging.
