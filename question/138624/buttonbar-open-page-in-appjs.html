    From your code it seems that you want to open the window inside that tab only, i too had this problem and the activetab never worked for me so I endup opening a window with the size so that my bottom tab bar is visible.

    But if you open that window using navigationGroup then you wont be getting that problem and your next window will open in the same tab.
