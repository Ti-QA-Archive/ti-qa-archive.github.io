After upgrade to 3.4.0.GA: Titanium could not detect valid SDK with version '6' or higher

So I've upgraded to the latest version of Studio and SDK 3.4.0.GA. Problem:

Could not detect a valid SDK with version '6' or higher at '/Applications/Xcode.app/Contents/Developer/Platforms/iPhoneOS.platform/Developer/SDKs'

I've tried refreshing it in the platform settings - no luck

I've reinstalled Titanium Studio and Xcode 6.0.1 - no luck

I've also tried "xcode-select -switch /Applications/Xcode.app/Contents/Developer/" - no luck

I've updated node, npm, alloy - no luck

Working on: OSX 10.9.5 Macbook pro.

Any one got an good idea that works?

Thanks.

  • Try ti sdk select and you get to choose which Titanium SDK is used by Studio. Somehow this is different from selecting which SDK to use in your specific project. When you pick the 3.4.0 SDK, suddenly Studio no longer has the Xcode / iOS SDK / iOS Sim issue.

    Got it here
