Open app or go to app Play page

Hi,

How can i open an application from within my application, and if the application is not installed how can i redirect the user to the market?

I know that

Ti.Platform.openURL('market://details?id=com.package.myapp');

opens myapp page in google play, but is there a way to customize this URL so that if myapp is installed, myapp is launched instead of launching the google play?

— asked September 20th 2012 by Andre Lopes

  • You can only open an app if it has a custom url scheme. ie: fb:&#x2F;&#x2F;profile&#x2F;id launches facebook.
    So once you know that url scheme, you can wrap it in a try/catch block, where if you try to open the url scheme and it fails, then you can open the Play url.
