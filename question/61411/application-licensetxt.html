Application license.txt

When preparing an application for distribution, Titanium allows the inclusion of a LICENSE.txt file alongside the Resources folder. When/where is this license visible to the user?

I tried looking for some license information in other published apps, and couldn't find anything in iTunes or the app store.

    Check out this "Guide". It explains all those files (tiapp, license, changelog, etc.)

    LICENSE - this file just describes the Appcelerator license. This is for your reference and not included in your application.

LICENSE.txt - this is your end-user application license. This is not currently used for mobile project but will be used in the future.
