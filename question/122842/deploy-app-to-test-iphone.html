Deploy app to test iphone

Hello,

I have created a iphone app using Titanium Studio. Now I would like to deploy this app to a test iphone to see how it looks on the iphone. Could you please help me with this?

Thanks,
Vijay

    • You will need an Apple Developer account
    • Visit the iOS Provisioning Portal (apple has some great videos to walk you through the next few steps)
    • Create a Development Certificate (might as well create a distribution one as well)
    • Register your device
    • Create a new app id (this needs to match the one you use in Titanium)
    • Create a Development Provisioning Profile (tie the device and app to the profile)
    • Install the certificate and provisioning profile on your OSX system.
    • In studio either "Run on Device" or "Distribute - App Store". Get the app file and the provisioning profile onto the device via iTunes. (drag and drop)
    • Run app (if all has gone well)
