Dynamically adding elements in alloy view

My app requires a variable number of items that the user will control. They will be able to go to their settings, pick the number of items, then go to a view where they will see their items (represented by an icon), each with it's own control (button, text box, etc). So how can I create that in the view dynamically using alloy? As far as I know there is no conditional formatting where I can get the number of items from memory, then create the xml elements in a loop. Can I do this with alloy or do I need to use the traditional method?

    You don't need to add to the view. If you have dynamic elements, add them through the controller using a for loop.
