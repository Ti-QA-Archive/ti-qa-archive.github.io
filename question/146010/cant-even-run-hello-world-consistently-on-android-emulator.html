  • Accepted Answer

    try one thing, delete the previous emulator and then launch with the fresh new project.

    it will create the new emulator, that will take time. when finished, if app is launched then all is well else close the emulator and kill the adb.exe using task manager.

    After that use the AVD manager to launch the emulator and then deploy the project already build so it won't give you timeout error or something.

    i know it seems boring, but once you get the resolution it will be handy. in Windows system it generally creates problem.
