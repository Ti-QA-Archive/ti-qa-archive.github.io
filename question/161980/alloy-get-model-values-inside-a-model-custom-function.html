[alloy] Get model Values inside a model-custom function

Hey.

how is it possible to acces values of a model, inside a model function?

extendModel: function(Model) {
        _.extend(Model.prototype, {
            // extended functions and properties go here
             customFunction: function() {
                Ti.API.info( Model.value1 );
            },    
        });

        return Model;
    },

I've tried several things, but nothing really works - it is possible at all, i guess ?!

best,
nico

    extendModel: function(Model) {
        _.extend(Model.prototype, {
            // extended functions and properties go here
             customFunction: function() {
                var value = this.get("value");
                Ti.API.info(value);
            },  
        });

        return Model;
    },
