Add a custom table row at end of table?

I have a page I originally worked out in HTML with jQuery, you can see how it behaves here

I would guess that to do this in Titanium I would use a table view, with each item being a row. I want to default with Row "A", and allow users to add B-Z as needed. I then (as I do in my jQuery) need to do math on each row and add it all up.

I see the examples in the kitchen sink for adding a row at a specified index, but how would I add a new row to the END of the table?

Also, how would I have each row to multiply it's fields' values and add it to a running total?

SDK 2.0.1, Studio 2.0.2, IOS 5.1, OS 10.7.3

    To add a a row to the end of the table use the appenRow method

    table.appendRow(Ti.UI.createTableViewRow({
        color:'black',
        title: 'new Row',
        height: 140
      }));

    To get the total add a property to the table (for easy access) that is going to keep track of the multiplication and the addition

    table.customfield = x*y + total;
