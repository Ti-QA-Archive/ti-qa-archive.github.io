Get number of event push notification sent

Hello all

I succeed in configuring Push Notification in android using GCM with this tutorial : http://www.tidev.io/2013/12/20/using-gcm-for-android-push-notifications-with-acs/

Now I would like to know how to get the value of badge ?

I use this code to retrieve the value : 

CloudPush.addEventListener('callback', function(evt) {
    alert(evt.payload);
});

And here's the text I have in evt.payload :

{"android":{"title":{"body":"My title","sound":"default"},"alert":"My alert","badge":2},}

I would like to do some action if badge is 2, another action if badge is 6.

Hope I am clear

  • Hello Pat U,

    Thanks for raising this issue. I have tested your issue with my sample code. In my code I have parsed the payload data and added two conditions for retrieving the badge value 2 and 6. Please test my sample code in your environments.

    Check attachment

    Test Code:

    
 CloudPush.addEventListener('callback', function(evt) {

    alert(evt.payload);

    var data =JSON.parse(evt.payload);

    if (data.android.badge =='2'){

        alert("Badge is "+data.android.badge);

    }else if(data.android.badge =='6'){

        alert("Badge is "+data.android.badge);

    }
});
