Add tabgroup from a NappDrawer View using Titanium Android

Hi I'm currently developing Android app that uses NappDrawer. NappDrawer uses a view instead of a window. So what I need to do is like this:

so here's my code

var tabGroup = Titanium.UI.createTabGroup();
var myTab = Titanium.UI.createTab({
    title: 'Tab1'
});
var myTab2 = Titanium.UI.createTab({
    title: 'Tab2'
});
var myTab3 = Titanium.UI.createTab({
    title: 'Tab3'
});
tabGroup.addTab(myTab1;
tabGroup.addTab(myTab2);
tabGroup.addTab(myTab3);

//nappdrawer's view
var centerWin = Ti.UI.createView();
centerWin.add(tabGroup.open());

I am able to create a tabgroup but instead of just adding it into the centerWin, a tabgroup was created into a new window. How can I add tabgroup into a Titanium View?

  • Tabgroups are top level window elements. With the exception of a splitview on ipad, should not be placed INTO any other container. On Android, because of the way the android layouts are constructed, this isn't even a suggestion by a requirement.

    Talk about a busy UX, you want a tab group on a drawer?
