    Hello Devanjali,

    Thanks for your inquiry. When you set the height using the height property, you're changing the height of the whole ProgressBar View which includes the label.

    If you want to change the height of the actual bar itself, you can only do that using a custom Android Theme:

    1. Increase height of progress bar in Android.

    2. Link to how to make custom Android themes in Titanium.

    Also, height=auto has been deprecated and you should use either Ti.UI.SIZE or Ti.UI.FILL

    Also for iOS, looks like a custom module is the only solution. Also, here is a community link on custom progress bar.

    Hope this helps. Thanks.

  • Height of bar and label is fixed. Below is the sample code to demonstrate the same.

    
var checkStatusWin = Ti.UI.createWindow({ width : '100%', title:'Check Status', height : '100%', backgroundColor:'#D4D4D4', });
var checkStatusView = Ti.UI.createView({ width : '100%', title:'Check Status', top : '8%', height : '80%', backgroundColor:'white'});

var statusBar = Titanium.UI.createProgressBar({
                width : 200,
                min : 0,
                max : 10,
                value : 0,
                height : 370,
                color : 'red',
                backgroundColor: 'green',
                message : 'Downloading 0 of 10',
                font : {
                    fontSize : 14,
                    fontWeight : 'bold'
                },
                style : Titanium.UI.iPhone.ProgressBarStyle.BAR,
                top : 120
}); 

checkStatusView.add(statusBar);
statusBar.show();
checkStatusWin.add(checkStatusView);

    You have to use show method of progress bar to make it visible.
