  • I guess you meant "Ti.App" events, or so called app-level events.

    If you add more than one identical event listener (that means, same event name and same function reference as event listener), they will all execute.

    You can test this with simple example with button event listeners (because same thing applies for component level event):

    var win = Ti.UI.createWindow();

var button = Ti.UI.createButton({ title: 'Click me!' });

var onClick = function(e) {
    Ti.alert('On click!');
};

button.addEventListener('click', onClick);

button.addEventListener('click', onClick);

win.add(button);

win.open();

    You should get two alert dialogs with identical message.
