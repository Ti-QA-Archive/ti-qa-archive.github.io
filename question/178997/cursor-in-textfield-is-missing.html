Cursor in TextField is missing

Hi,

i have a screen with 2 text field:

        <Label id="LtxtNombre" top="12%">Nombre</Label>
        <TextField id="TNombre" top="18%" hintText="Introduzca el Nombre" maxLength="15"/>
        <Label id="LtxtClave" top="32%">Clave</Label>
        <TextField id="TClave" top="38%" hintText="Introduzca la Clave" maxLength="30"/>

The problem is when i click in any TextField, then keyboard in my android phone is show, but i haven't a cursor in this TextField. This is only if TextField is empty, when i have any text then the cursor is visible.

How i can show the cursor when a TextField is empty??

Thanks

    Hi Carlos,

    Thanks for your concern. I've tested your issue and default white cursor color change to black and vesisible even when the textfield is empty.

    First, Modify your tiapp.xml using the code below.

    <android xmlns:android="http://schemas.android.com/apk/res/android">
        <manifest>
            <application android:theme="@style/Theme.AppCompat.Light"/>
            <!-- Need to specify at least API level 11 for Titanium SDK 3.2.x and prior -->
            <uses-sdk android:minSdkVersion="14" android:targetSdkVersion="19"/>
        </manifest>
</android>

    Second, Create a styles.xml file and a colors.xml file in [Project]/platform/android/res/values, relative to the main project folder.

    In styles.xml, add the following code.

    <resources>
    <style name="Theme.Custom" parent="@android:style/Theme.Holo.Light">
      <item name="android:editTextStyle">@style/editText</item>
    </style>
    <style name="editText" parent="@android:style/Widget.Holo.Light.EditText">
       <item name="android:textCursorDrawable">@null</item>
       <item name="android:textColorHint">@color/cursor</item>
    </style>
</resources>

    And in colors.xml, add the following code.

    <?xml version="1.0" encoding="utf-8"?>
<resources>
    <color name="cursor">#000</color>
</resources>
