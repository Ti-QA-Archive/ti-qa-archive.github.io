  • I don't think you understand how webviews work. You have to either pass the var value during the beforeload like this:

    webview.addEventListener('beforeload',function(e)
{
    webview.evalJS("var myVar='" + myVar + "';");
});

    or you have to fire an event from the app.js to the chart.html when the html is loaded.
