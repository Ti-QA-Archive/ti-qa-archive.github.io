Unable to work with modules on Ubuntu (12.04 LTS)

Hi,
I'm running titanium on Ubuntu 12.04 LTS OS. Lately I've run into a problem where i am not able to work with any 3rd party module. When i run the app either in simulator or device i get the error as requested module not found.

Ex. for one of the project the module files are located at:
modules-> android->com.activate.gcm->0.3

Also I have linked the module in tiapp.xml as follow.

<module platform="android" version="0.3">com.activate.gcm</module>

What could possibly be wrong? The app does work with module in other systems but seems something is wrong with my Ubuntu setup.

  • I managed to solve this issue by hours and days of struggle. The issue was that the jar file which was created after installing the module was in mixed letter case. Eg. ImageAsResized. I renamed it to all small case, imageasresized.

    Otherwise the module was being detected by titanium but it wasn't getting executed in app.
