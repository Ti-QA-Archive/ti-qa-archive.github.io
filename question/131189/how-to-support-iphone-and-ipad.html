how to support iphone and ipad

Hi all, I am creating project in titanium using android. I ran that project in iphone and ipad. But the alignment is not coming correctly for iphone and ipad. what should I do now to support all these three formats with same coding. Is there any possible for this or we should do three separate coding for this. can anyone suggest me please?

  • What I do is use the CommonJS method. And then separating models, services en UI components. Then it is possible with some if..then in app.js to include the right views. And then use Ti.platform.displayCaps.platformWidth as mentioned earlier.

      //create different UI files named iphone.js, ipad.js, android.js
