    Hello,

    You cannot increase the height of the annotation, as the map API provide by appcelerator is not fully mature. The annotation you get on the map is the same annotation you get on native iOS map, but with very limited feature. The only solution you left with is to present a custom view when user taps on the annotation. but this also has a problem, as the map API does not allows us to add map overlays, so you have to add a transparent window on top of your map with your custom view on it.

    In short you can only add images, as you said 30px.
