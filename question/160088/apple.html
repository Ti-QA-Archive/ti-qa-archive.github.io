  • Hi,

    Minimum requirement for developing app supported on iOS 7 devices is:

    OS X Version 10.8.4

    Ram minimum 2 GB

    System should have Xcode 5 installed.

    If you are going to use Titanium Appcelerator the You require Titanium Studio and titanium SDK.
