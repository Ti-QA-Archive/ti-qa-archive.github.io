Emulator failed to start in a timely manner

hello,

I have created avd emulator for android with following configuration,
target : Android 2.3.3 and API 10
CPU :Intel Atom (x86)

I have titanium studio 3.2 and android 2.3.3 and 4.1.2 version is installed in it.

But on running application emulator get start very late and i get [ERROR] : Emulator failed to start in a timely manner.
The current timeout is set to 120000 ms

if I again run the app after emulator is ready then also process create new emulator every time and shows same error. like Emulator 5554 then 5556 then 5558 like wise

  • Hi Ritesh.

    I have the same problem, but I found 2 workarounds:

    1. Launch and wait the first time until the emulator is ready, then you will find a new option in the launch section (in the left upper corner), named SDK (seems like Studio believes is a new "device"). If you select that SDK, then the new launch will re-use the emulator.

    2. Launch and wait the first time until the emulator is ready, then launch again. As you know, it will open a new emulator (first a DOS window, then the emulator window). If you CLOSE the DOS window before the new emulator window is shown then, incredibly, Studio will use the existing emulator (I don't know if this is a bug or what, but works).

    One note: remember to close the other launched processes in the console, I found that they are always opened. Just stop it and all will work.

  • Unless you have a compelling reason to be running an emulator with 2.3.3, I'd suggest you look at running the genymotion emulator (genymotion.com). It is also x86 based, but is much faster and easier to operate (along with camera access) than android x86 emulator.

    I have yet to have it timeout on me. Ti 3.2.0.GA (along with 3.2.0 cli) supports genymotion very well.

    See: http://www.appcelerator.com/blog/2013/12/using-genymotion-with-titanium-3-2/ (note: you do not need to install beta anything, as 3.2.0 is GA since that article was written)

    If you need the google apis (and you will) follow the instructions here: http://blog.zeezonline.com/2013/11/install-google-play-on-genymotion-2-0/
