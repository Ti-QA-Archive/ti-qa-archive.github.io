copyright notice on map

I am using the built-in Ti map. On android in the bottom left corner I get a sign that reads : Google but on iOS it says Copyright. I know that I have to start using v2 maps but is there something I can do to change that sign/label currently?

Thanks,

    You can not change this copyright hint. The reason that it is not the same on both iOS and Android is easy: Android uses Google Maps but iOS uses iOS Maps.
