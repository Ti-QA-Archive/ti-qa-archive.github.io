Hide Keyboard On Blur When Using Multiple TextFields

I have a simple form that has a TableView, and inside of the table view there are ten rows which contain a TextField and TextArea. I want to make it so when a keyboard is active, and the user taps anywhere in the table, it should hide the keyboard. Currently I have to keep track of the current selected keyboard with:

costs[i].addEventListener("click", function() {
    currentActiveKeyboard = costs[i]
}

When the above code is called, it only assigns the currentActiveKeyboard to null. I'm guessing this is because costs[i] will always be the last cost item in the array (but still it shouldn't be null).

I have this on the tableView to say when the currentActiveKeyboard should be hidden:

tableView.addEventListener("click", function(e) {
    currentActiveKeyboard.blur();
    currentActiveKeyboard = null;
})

The odd thing is that this works one time only when I directly assign the currentActiveKeyboard to be the first cost textField
currentActiveKeyboard = costs[0];

Is there a way to check if the user just simply didn't tap on the keyboard? That would probably be the easiest way to know the keyboard should be hidden.

Any help would be greatly appreciated.
