Memory management

Hi,
I have the below code

var win_cur = Ti.UI.currentWindow;
var view_welcom = Titanium.UI.createView({ width : "100%", backgroundColor : "#EFEADD", top : '50dp', height : '70dp' });
var welcom_lbl = Titanium.UI.createLabel({ text : 'Welcome Back', color:globalfont_color, font : { fontSize : '22dp', fontWeight : "bold" }, top : 0, textAlign : 'center' }); view_welcom.add(welcom_lbl);
var larr_img = Ti.UI.createImageView({ image : 'images/News-Feed_03.png', left : GetWidth(20), top : GetHeight(20) }); view_welcom.add(larr_img);
win_cur.remove(view_welcom); view_welcom = null;

i have added this line
win_cur.remove(view_welcom); view_welcom = null;
to improve the memory management, but the contents inside the view(view_welcom) is not shown, it is empty
any body please share about how to make view null and to improve memory in andorid

    Problem 1

    Because of this var win_cur = Ti.UI.currentWindow; everything else you write is suspect to me because it only works with crappy code.

    Problem 2

    I see you removed the view win_cur.remove(view_welcom); but where, pray tell, did you actually add the view to the window in the first place?

    Problem 3

    You can't null something you hope to use win_cur.remove(view_welcom); view_welcom = null; If you want to null it out in the hopes of having it garbage collected, do it in an event AFTER you are done with it.
