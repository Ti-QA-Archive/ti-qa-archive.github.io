iOS 7 Native Sharing Support

Is there a way to access that "share screen" in iOS 7 with the list of compatible apps? You see this screen if you open a PDF or something in Safari and tap the share option. Here is an image of what I'm referring to: Link.

I have an app that lets you view PDFs and Excel files in WebViews, but I would like to let users share the files, and more importantly, open files in other applications, like Pages and Numbers. The "share screen" would be perfect for that.

Is there a way to do something like this with Titanium?

  • You can try the iOS documentviewer. If you pass in a valid type (PDF, HTML, etc) it will provide a share button that will let you send the file to any application that is registered to handle that you're on the device. Check out http://docs.appcelerator.com/titanium/3.0/#!/api/Titanium.UI.iOS.DocumentViewer in the documentation to see the options.
