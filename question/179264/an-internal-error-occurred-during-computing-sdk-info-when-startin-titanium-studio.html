An internal error occurred during: “Computing SDK Info…” When Startin Titanium Studio

Since upgrading TitaniumStudio, I am getting a java error when starting the TitaniumStudio application. Full error message is:

An internal error occurred during: "Computing SDK Info...".
java.lang.NullPointerException

I have no idea how to bypass this error. I tried installing and reinstalling the application, installing and reinstalling CLI via command line with no luck.

I am running osx with Yosemite.

Titanium 3.4.1
Node v0.10.33
npm 1.4.28

Any suggestion on where else to look in order to solve this error?

  • Hello,

    It looks like there is a problem with your nodejs. Try uninstalling just the Titanium node.js package using npm titanium uninstall command. Once uninstalled,open Titanium studio.It will prompt you for installing Titanium CLI.Follow the steps and install Titanium CLI.

    Note: Sometimes to fix this problem you need to remove node.js and installing an older version (v0.10.23).

    Let me know how it goes.
