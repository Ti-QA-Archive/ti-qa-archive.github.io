Activity Indicator in Android

Hi

I want to have an activity Indicator on Android and I used this code.

var button = Ti.UI.createButton({
    title : 'button',
    height : 70,
    width : 100,
    top : 1,
    left : 4
});

// Listen for click events.
button.addEventListener('click', function() {
    //alert('\'button\' was clicked!');
    actInd.show();
});

// Add to the parent view.
win.add(button);

var actInd = Titanium.UI.createActivityIndicator({
    bottom:0, 
    height:55,
    width:180,
    color:'#FFFFFF',
    backgroundColor:'#000',
    opacity:0.9,
    borderRadius:5,
    borderColor:'#000',
    font:{fontFamily:'Helvetica Neue', fontSize:13},
  })

  win.add(actInd);

The indicator appears correctly. But my concern is I want to change the location of the indicator may be to the top or bottom. But it appears at the centre. Can we change the location of the Indicator.

And Secondly, When I click on the back-button, the indicator doesn't disappear.

Please help me on this.
Thanks.

  • 1 you should have the indicator .hide() when a particular activity, for which it was shown, ends

    2 you can have an android:back event and .hide() the indicator there
