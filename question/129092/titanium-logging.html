Hi all!

In Titanium Studio I don't get the log messages. I writes everything until the deployed message is written. After that nothing appears. Can anyone help me with this? Is there a way to look at the log messages if I test on and Android device? (Titanium SDK 1.7.5, Android SDK 4.0)

  • You can use the use the Android DDMS tool. Run DDMS from the command line and the debugger will show all of the Ti.API.Info messages you write out. Is that what you were looking for?
