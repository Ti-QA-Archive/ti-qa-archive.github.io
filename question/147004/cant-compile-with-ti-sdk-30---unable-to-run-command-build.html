  • Yeah, there's a bug in 3.0.0.GA with parsing plist and xml files. It's been fixed in 3.0.x. You can give the latest 3.0.x CI build a try, but just know that it's NOT tested and should NOT be used in production. To download the latest, simply run:

    titanium sdk install --branch 3_0_X --default

    Be sure to update your selected sdk-version in the tiapp.xml, then build your app again.

    • From the terminal, if you run which titanium, it should say "/usr/local/bin/titanium". But by the sounds of things, you probably have an alias to the old CLI. Open up ~/.bash_profile in an editor and search for a titanium alias. Remove the alias and relaunch your terminal.

      Next try running titanium -v. It should print "3.0.22". Now you should be able to create projects.
