Problems getting Titanium to see Android SDK Directory (OSX)

Hi,

Just getting started with Titanium, and I am having some problems getting it to recognize the Android SDK directory.

I have installed the Android SDK into /users/USERNAME/Android_SDK.

I have opened the SDK Manager and downloaded/installed the SDK Tools, the SDK Platform-tools, and the following Platforms: 3.2 (API 13), 2.2 (API 8), and 1.6 (API 4).

I have created the symbolic link to adb as described in the "getting started" guide. (And re-created it after installing new platforms.)

I have confirmed that I have java and javac versions 1.6.

However, every time I go into the preferences to point Titanium at the Android SDK directory, it tells me "Could not locate the Android SDK at the given path."

Any ideas? Am I missing a step somewhere?
