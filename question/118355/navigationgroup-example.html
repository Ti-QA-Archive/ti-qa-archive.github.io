NavigationGroup example?

I don't understand how to use NavigationGroup, and I haven't seen an example taken from a real-world app. The only examples I've found, such as the KitchenSink, this one, and this one are "toy" examples that assume that the entire app is managed by this navigation group.

I'm looking for an example that uses both a tab group a nav group. This is behavior that many iPhone apps display.

Note that the KitchenSink NavGroup test is broken. This particular portion of it doesn't work (Tap done, then relaunch the NavGroup: it's not correctly displayed.) And since the KitchenSink test is broken, I'm tending to think that nav groups can't be used in tab group-based apps. (?)

This tabgroup+navigationgroup might be easy to do; I just might not have played around long enough.

Example - Momento:

The app is based on a tabbed view. Tap the gear in the upper left and a modal settings window displays:

Note how this modal window sets up its own local "navigation group". Tap the Feeds row, and the Feeds settings window slides in from the right:

And so that's a setup I've seen in a lot of apps: (1) a tab group to control the global nav, and a (2) a navigation group to control a subset of pages within a modal window where needed.

In my case, I need this for a simpler settings page (the first modal window), which will then have explanatory help pages.
