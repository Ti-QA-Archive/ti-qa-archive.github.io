I have looked through a lot of documentation and examples on github and I have also been looking at the backbone docs as the suggestion is you need to understand backbone.js to use models in Alloy.

What I need is a very very very simple example of an app built in Alloy which contains a local sqlite database which reads information into a simple tableview and then saves back to the database.

  • have a look at:

    https://github.com/appcelerator/alloy/tree/master/test/apps/models/binding_tableview

    and

    http://docs.appcelerator.com/titanium/latest/#!/guide/Alloy_Data_Binding

    Then you can use backbone to get the data or use normal sql commands with:

    var db = Ti.Database.open('_alloy_');
var deleteRecords = db.execute('DELETE FROM tbl');
Ti.API.info('Affected Rows    ' + db.getRowsAffected());
db.close();
