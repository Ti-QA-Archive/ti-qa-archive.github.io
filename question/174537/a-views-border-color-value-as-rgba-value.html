A view's border color value as rgba value?

Hi

I was thinking to have a border color value as rgba color value instead of normal hex-code so that I can have transparency in border color,
is it possible to have rgba like example below.

var view = Ti.UI.createView({
        borderColor : rgba(100, 100, 100, 0.5)
});

Thanks,

    Hi

    There are a couple of ways to achieve this and you were almost there one of them

    borderColor: '#44000000', // hex aarrggbb

 or

borderColor: 'rgba(0, 0, 0, .25)', // rgba

    • Hex

      This takes 4 sets of two hex values, this is normal three sets of two you use already plus another set at the start. 44 in the example is the amount of opacity to add

    • rgba

      As I said you were almost there. rgba is not a function of Titanium but a function of colours -so you place it in quotes. This is the same as if you said "red".

    Important

    ONLY the hex versions works on both Android and iOS devices. The rgba way will simply be ignored and show a transparent border.

  • Hi Maruti,
    there is no need to pass this as rgba, which is not supported anyway. You can set transparency with 8 digit long hexcodes. The first 2 digits correspond to to opactiy

    100% — FF
95% — F2
90% — E6
85% — D9
80% — CC
75% — BF
70% — B3
65% — A6
60% — 99
55% — 8C
50% — 80
45% — 73
40% — 66
35% — 59
30% — 4D
25% — 40
20% — 33
15% — 26
10% — 1A
5% — 0D
0% — 00

    e.g. 75% opacity

    '#BF000000'
