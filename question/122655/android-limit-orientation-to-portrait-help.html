Android limit Orientation to portrait Help

I'm trying to make my app (or even just a window) lock into portrait mode on android. I've tried setting the android section of my tiapp.xml file to this:

<android >
<manifest>
<supports-screens
android:smallScreens="false"
android:normalScreens="true"
android:largeScreens="true"
android:anyDensity="false"
/>
<uses-sdk android:minSdkVersion="7" />
<application>
<activity android:name="MyAppName" android:screenOrientation="portrait" android:configChanges="keyboardHidden|orientation">
</activity>
</application>
</manifest>
</android>

but nothing I've tried seems to work. I've tried sdk 1.7.0, 1.7.1 and the latest nightly builds for 1.7.x and 1.8.x. I don't have any problem with iOS, so hopefully I can get this issue corrected on android soon, too.
