  • As I remembered this is caused due to Android SDK add-ons module(It still causes even if you try running for iPhone). Follow the below steps

    • Locate your Android SDK folder.

    • Goto add-ons folder.

    • Delete addon_google_apis_google_19 folder.

    • Make a copy of addon_google_apis_google_19_2 folder and rename it as addon_google_apis_google_19

    • Finaly, you should have both addon_google_apis_google_19 and addon_google_apis_google_19_2 folders in the add-ons folder.

    Now, clean your project and try building again.
