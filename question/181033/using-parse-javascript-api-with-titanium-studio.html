    • Parse has a rest api that you can access via Ti.Network.HTTPClient.

      For android push, you can use the eu.rebelcorp.parse module. iOS you simply send the token to Parse over rest.

      There are a couple of titanium modules wrapping the Parse native sdks out there as well.
