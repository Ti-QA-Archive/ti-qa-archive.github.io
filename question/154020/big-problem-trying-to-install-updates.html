  • What you need to enter in this Dialog window is not your complete name, but your technical user name linked to you Mac account. If you don't know your user name, there is a very simple command you can run to retrieve it. From the Terminal, simply type this:

    $ whoami

    This will return your username. (usually all lower cased)

    As for your password, it's basically the password you use everyday to unlock your Mac. If your doesn't have that kind of mechanism in place (no prompt for password when you turn it on) then simply hitting the Return should do the trick.

    Hope this helps.
