How To Access Files With PHP In The Resource Folder Of The Packaged & Installed Desktop Program?

I asked a question here about fopen() and the use of "app://", I am now guessing that this whole time I have been confused on its real use. It is NOT used for desktop environments at all?

If this is the case, how do I access files within the Resource folder of the end result, packaged, and then installed program folder?

I need to write some simple data to a Setup.ini that resides in the Resource folder, this works during testing if I use a direct path, such as "C:\Users\ComputerName\Desktop\ExampleProg\ExampleProg\Resources\Setup.ini", but this obviously doesn't work, because the file isn't going to be there after the install happens to the end user. So what do I do to access files within the Resource folder of the installed program?

Any help is greatly appreciated! Thanks in advance.
