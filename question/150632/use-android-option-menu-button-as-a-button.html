Use Android option menu button as a "button"

I'm using this code to fire a click event when the Android option menu button is clicked.

function androidToggleOptionMenu() { menuButton_options.fireEvent(&#39;click&#39;); Ti.Android.currentActivity.onCreateOptionsMenu = androidToggleOptionMenu; } Ti.Android.currentActivity.onCreateOptionsMenu = androidToggleOptionMenu;

By repeating the last row within the function, I would hope that next time the button is clicked, it would run the function again.

But it only works once - the first time. Second click generates no action at all.

Any advice on this?

  • Here is an example for you

    var self = Ti.UI.createWindow({
        title:'RSS Reader',
        backgroundColor:'#fff',
        exitOnClose:true,
        navBarHidden:false,
        activity: {
            onCreateOptionsMenu: function(e) {
                var menu = e.menu;
                var menuItem = menu.add({ title: "Refresh" });
                menuItem.setIcon("images/refresh_icon.png");
                menuItem.addEventListener("click", function(e) {
                        refreshRSS();
                });
            }
        }
    });
self.open();

    just add an icon
