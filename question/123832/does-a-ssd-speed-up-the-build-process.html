  • More or less, your computer will run faster but only for processes that require hdd read/write.
    On compiling the app there are this kinds of operations, but the majority is still CPU.

    You will have though an improvement because overall the computer will run faster.
    I had one on my previous MBP and I can tell you is waaaay faster than when I had a normal hdd.

  • I have both a standard HD and a SSD in my Mac Book Pro, and I recently moved my titanium studio workspace to the solid state drive to improve speed, and it helped a lot. I didn't track it or anything, but it felt like it takes about half the time on the SSD as it did on a standard HDD.

    What I would LOVE to be able to actually get working is "LiveTainium", which I haven't seen much talk about lately, but seemed like a HUGE timesaver. I couldn't get it to work, and would love to see the community try to get it working on the current version of Titianium, as I think it would save me a HUGE amount of development time.
