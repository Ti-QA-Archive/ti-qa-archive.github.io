Window titleControl in Alloy

I've just started the process of converting an older Ti app to use Alloy, but I'm finding some of its conventions a little confusing, and very not much guidance from the docs.

Specifically, how do you create a label node and reference it as the titleControl for a window? In the past I've done this:

var videoWindow = Titanium.UI.createWindow({
                    titleControl:        Titanium.UI.createLabel({
                                color:        '#fff',
                                text:        'Video Player',
                                textAlign:    'center',
                                font:        {
                                    fontFamily:    'Open Sans',
                                    fontSize:    18,
                                    fontWeight:    'bold'
                                },
                                shadowColor:    '#555',
                                shadowOffset:    {
                                    x:    -1,
                                    y:    -1
                                }
                            }),
                    barColor:            '#0066AE',
                    backgroundColor:    'transparent',
                    backgroundImage:    '../images/gallery-bg.jpg'
                });

In Alloy should I be doing something along the lines of this?


<Alloy>
       <Label id="titleControl" class="titleControl">Video Player</Label>
    <Window title="Bookings" id="bookingsWindow" titleControl="#titleControl">

    </Window>
</Alloy>

(obviously the styles for #titleControl would be in the stylesheet under the .titleControl class)

Any pointers would be much appreciated!

Ryan

    Hi,

    You can use the title control object as follows

    <Window>
<TitleControl></TitleControl>
</Window>

    You can look under Currently, the following Titanium proxies and properties implemented using this syntax are: this section
    http://docs.appcelerator.com/titanium/3.0/#!/guide/Alloy_XML_Markup-section-35621528_AlloyXMLMarkup-PropertyMapping

  • Brilliant- that was it, thanks Ranjith I'd missed that.

    For anyone else, here is the full XML code in the view:

    
<Alloy>
    <Window title="Bookings" id="bookingsWindow">
        <TitleControl>
            <Label class="titleControl">Bookings!</Label>
        </TitleControl>
        <Label>This is the Bookings tab</Label>
    </Window>
</Alloy>
