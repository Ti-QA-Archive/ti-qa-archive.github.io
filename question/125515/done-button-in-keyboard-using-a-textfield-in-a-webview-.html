    If the webview is local you can trigger a cross-app event with Ti.App.fireEvent from the webview, in the main app to show you the Done button.

    • In the window you want to add the button:

      
// win is the window we are talking about

Ti.App.addEventListener('add_done', function(){
    var done_btn = Ti.UI.createButton({title:'Done'});
    win.add(done_btn);
});

      in the html of the webpage:

      
<input id="the_input" onfocus="function() {Ti.App.fireEvent('add_done')" />

      or something like this :)
