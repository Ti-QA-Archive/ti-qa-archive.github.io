  • The best way to give everyone a better idea of what your issue actually is to run this little environment script.

    Simply paste the following into your Command Prompt:

    cls
cd \
echo %PROGRAMFILES%
echo %PROGRAMDATA%
echo %USERPROFILE%
echo %PATH%
echo %JAVA_HOME%
java -version
javac -version
python -V
git --version
rem if the ANDROID_SDK variable is configured, please delete it and run this script again
echo %ANDROID_SDK%
adb version
cd %PROGRAMFILES%
android list target
cd \
rem --- The following are optional ---
rem ----------------------------------
scons -v
rem ----------------------------------
echo %HTTP_PROXY%
echo %HTTPS_PROXY%
rem ----------------------------------
echo %SOCKS_PROXY%
rem ----------------------------------
rem

    And then share the output here. This will help us identify whatever configuration errors you might have.
