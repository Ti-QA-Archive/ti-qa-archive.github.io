Adding lable and switch to the table view data.

Hi,
I created table view with some rows and adding label,switch to this rows. In this , If I want to add 15 rows to the table. BY using my process , I can add total 15 labels and switches. Then it seems to be huge code. Is there any solution for this like using for loops to reduce the code. Here is my code:

var basicSwitch = Titanium.UI.createSwitch({
    style:Titanium.UI.Android.SWITCH_STYLE_CHECKBOX,
    value:false,left:270
});
var table1 =  Titanium.UI.createTableView({});
var row1 = Titanium.UI.createTableViewRow({});
var row2 = Titanium.UI.createTableViewRow({});
row1.add(basicSwitch);
row2.add(basicSwitch);
table1.setData([row1,row2]);
win1.add(table1);
var lab1 = Titanium.UI.createLabel({title:'Nokia',left:0});
var lab2 = Titanium.UI.createLabel({title:'Samsung',left:0});
row1.add(lab1);
row2.add(lab2);

Is there any possibility to add this controls to the rows, like creating the table view data like this way:

var table1 =  Titanium.UI.createTableView({
    data:[
        {title:"Row 1"},
        {title:"Row 2"},
        {title:"Row 3"},
        {title:"Row 4"},
        {title:"Row 5"}
    ]    
});
