  • hi,

    just got the solution..

    first save your book page with Filesystem in sdcard with page no ……like

    var f = Ti.Filesystem.getFile(Ti.Filesystem.externalStorageDirectory,'page1.png');
f.write(paintView.toImage());

    and then set that page in paintView like

    var f = Ti.Filesystem.getFile(Ti.Filesystem.externalStorageDirectory,'page1.png'); 
paintView.image = f.read();

    and when you want to clear that page then

    var f = Ti.Filesystem.getFile(Ti.Filesystem.externalStorageDirectory,'page1.png'); 
if(f.exists()){
    f.deleteFile();
}
