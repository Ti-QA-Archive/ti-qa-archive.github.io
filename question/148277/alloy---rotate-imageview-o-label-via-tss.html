Alloy - Rotate imageview o label via tss

Hi, there is a way to rotate every imageview or a label inside a tableview(row) using tss or something similar? Just like you can do with "opacity".
Thanks

  • You can not because a rotation is something that is calculated from a transform matrix that must be created.

    Like this:

    $.myLabel.transform = Titanium.UI.create2DMatrix({ rotate: -45 })

    But we can't create a Matrix in the TSS at this time (well I don't think so) , maybe in future versions.

    But I wish we had something nice like:

    "element" : {
    transform : {matrix2D :  {rotate: -45}}
}

  • Define a constant in Alloy.Globals for the rotate method, then reference the constant in .tss files.

    Alloy.Globals.rotate90 = Ti.UI.create2DMatrix().rotate(-90);

    "element":{
  transform: Alloy.Globals.rotate90
}
