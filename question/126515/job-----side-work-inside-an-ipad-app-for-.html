Job - - Side work inside an iPad app for $$

We've got an iPad app with more work than we can handle so we need some of smaller bits to be done as a "Per item" job.

This will be done simply as an isolated job. You will not be receiving our source code, and to make this easier on both keeping our app a secret(until release) and to make your job easier.

###Some of the things we need done:

  1. Popup styled… Using the stock/provided iPad popover doesn't work with our iPad's styled UI. All images will be provided in the exact slices/sizes required but you'll have to let me know what those are. (ideally… I want an auto resizing popup which has a nice smooth transition in/out of the screen and when getting larger and smaller(vertically… horizontal width would remain the same).

  2. Image styling… Rounded corners and a png overlay with multiple sizes from 32px up to 65px.

  3. Transitions… There are multiple parts of our application that needs transitions. It masks the loading of the application as well as enhances the user experience. There are multiple areas that need transitions, and the way that we'll accomplish this is I tell you the screen or area thats changing, you give me the code to add for the requested transition and we put it in.

  4. Tables and Chair styling & transitions… Our app is for restaurants and we have tables and chairs as the focus of our app. The functionality is already there but needs to look great in every version. They're currently being drawn so there are no images. We need gradients, inner shadows, drop shadows, stroke, and alternating color pallets. They are also clickable, so having an "Touch" event and a transition between states are crucial. We have a lot of different tables to be designed as well… meaning different types of table and chair combinations.

  5. Statistics… This app relies heavily on statistics and we have a slew of information that needs to be displayed in a google analytics type way. Standard, but this too will need to be stylized as well.

  6. Avatar Creator A single page with png avatar pieces. Using swipe left and right to navigate the different noses/hair styles/etc… And then to combine them all when finished for users avatars. Should be as simple as possible.

  7. Background selection & photo uploader Using a popover, the ability to select either "provided backgrounds"(large background images) or "upload" from the camera roll. The background would be fit inside of a top toolbar and bottom navbar.

If you would like to do 1, multiple, or all of these please contact me ASAP with a PER ITEM price.

Those that help will also get recognition for their work. Message me for details.
