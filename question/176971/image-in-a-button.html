Image in a button

Hi, I need put a image in a button, but no run.

<Alloy>

    When I had issues with this I needed to use "/images/<image_name>.jpg" as the path after putting the image into all of the different density folders in app/assets/android/images. Also, did you try just adding a "/" before BAJAR.jpg. Finally, if that doesn't work, I have had more success with making an ImageView and giving that an onClick function instead of using a button.
