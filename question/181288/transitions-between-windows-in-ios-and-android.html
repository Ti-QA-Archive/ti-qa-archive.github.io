Transitions between windows in iOS and Android

Hi all,

I'm trying to do a transition between windows with the createAnimation function, but it doesn't work in Andoid despite work fine in iOS. This is my code:

function doClick(e) {

    var slide_it_left = Titanium.UI.createAnimation();
    slide_it_left.left = 0;
    slide_it_left.duration = 300;

    var winHome = Alloy.createController('home').getView();
    winHome.left = 500;

    winHome.open(slide_it_left);
}

iOs do the transition ok! But it's not the same in Android where only change the window without any transition. What's the problem? Is there another way to do transitions for both systems?

Thanks a lot!

        winHome.open({
            activityEnterAnimation: Ti.Android.R.anim.slide_in_left,
