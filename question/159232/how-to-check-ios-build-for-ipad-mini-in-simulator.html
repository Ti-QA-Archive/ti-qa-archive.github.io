How to check ios build for ipad mini in simulator

am having an app using titanium appcelerator. i need to test it for ipad mini but am not having the device to test. Is there any possibility to test it in simulator?

  • You can use the iPad Simulator to accomplish this. The screen resolution of the iPad mini and the iPad is the same for both retina and non-retina versions (1024 x 768 for non-retina and 2048 x 1536 for retina). The only difference between the two is the pixel density. The iPad mini's pixel density is greater than that of the iPad, so the iPad mini's screen resolution gets compressed into a smaller space.
