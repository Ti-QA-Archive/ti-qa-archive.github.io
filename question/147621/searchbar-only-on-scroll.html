Searchbar only on scroll

Hi,
i'm new in Titanium.
I use searchbar in my tableview (iOS), but I need to show it only when user scroll down.
Same behavior of iPhone message list.
Is it possible ?

    Hi

    Set the searchHidden property of the tableView to false. This hides the searchBar initially requiring a pull down to reveal it.

    • I thought I would give you a full code example.

      To test this code create a new default project and replace the contents of app.js with the code below.

      var win = Titanium.UI.createWindow({
    backgroundColor: '#fff',
    title: 'Window'
});
var data = [
    { title: 'One' },
    { title: 'Two' },
    { title: 'Three' },
    { title: 'Four' },
    { title: 'Five' },
    { title: 'Six' },
    { title: 'Seven' },
    { title: 'Eight' },
    { title: 'Nine' },
    { title: 'Ten' },
    { title: 'Eleven' },
    { title: 'Twelve' },
    { title: 'Thirteen' }    
];
var search = Ti.UI.createSearchBar({});
var tbl = Ti.UI.createTableView({
    data: data,
    hideSearchOnSelection: true,
    search: search,
    searchHidden: true
});
win.add(tbl);
win.open();

      This property makes the searchBar initially hidden from view, pull the table down and the searchBar is revealed. If the tableView is pushed back up and enough rows exist to exceed the current portal size then the searchBar is hidden again.

      searchHidden: true

      This one is a bonus property, when true it makes the searchBar hide once the one of the filtered tableViewRows are clicked. If this is set to false then the searchBar remains in place and the selection remains in focus.

      hideSearchOnSelection: true,
