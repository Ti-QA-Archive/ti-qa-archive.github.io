Automatic Facebook access token

Hi.

Maybe a stupid question but here it goes…

I hav an app where i want to show statuses and events from a FB page.

In order to show them i have to authorize and get an access token. That is no problem but what if the user dont want to have to log in with their FB account just to read those posts?

Is it possible to make some sort of auto login with a FB app or similar to get a token?

I have seen many mobile apps out there that shows posts and events from FB without the need to login and authorize through a FB app.

How is that possible?

  • You can use the Facebook Graph API to query public facing information from Facebook, ie: posts, events, etc. As long as you don't have to be signed in to view the info, then you don't need to be signed in to request the info either. Use the Ti.Facebook api to make graph requests.
