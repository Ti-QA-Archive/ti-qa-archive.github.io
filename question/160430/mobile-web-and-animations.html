Mobile Web and animations

Can I do animations with labels on Mobile Web? I am having no luck getting this to work. Any help is appreciated.


var gentleText = Ti.UI.createLabel({
    text: '• High-level',
    color: '#000000',
    font: { 
        fontSize:22
    },
    top: '165',
    right: '0'
});

buttonOne.addEventListener('click',function() {
    gentleText.animate({
        right: 155,
        duration: 1000
    });
});

~~~

  • The syntax seems ok and it will work on iOS and Android. If you have an issue, you can pack the label in a view and animate the view (container).

    • Like this?

      var gentleContainer = Ti.UI.createView({
    height: '200px',
    width: '200px',
    left: 0
});

gentleContainer.add(gentleText);

buttonOne.addEventListener('click',function() {

    var anim1 = Ti.UI.createAnimation({
        left: 320,
        duration: 1000
    });

    gentleContainer.animate(anim1);

});
