TableView - Update ImageView within Row

Hiya

My iPhone app uses a tableview where i want to add an imageview to each row. I am using the kitchensink example (layout 5) as shown below to create each row dynamically from the images array.

All works fine, my problem is that i now at different stages of the app have to change the image in a specific row.

So for example i need to update the imageview within row 3 and 4 to new one's. Is it possible to target these images despite them being added in this way?

var images = [
    'http://t2.gstatic.com/images?q=tbn:BjUSCfLZ6aMORM:http://mariusbardan.files.wordpress.com/2009/10/carbonapple_mono1024.jpg',
    'http://t0.gstatic.com/images?q=tbn:HhYIr-Oz_5sLkM:http://blog.volgyiattila.hu/wp-content/uploads/2009/01/apple-logo1.jpg',
    'http://t3.gstatic.com/images?q=tbn:u18bgKxb_hjznM:http://www.applephone.hu/wp-content/gallery/miki315/apple-marbles.jpg',
    'http://t1.gstatic.com/images?q=tbn:GzVd4vgzFJCrdM:http://muzzak.freeblog.hu/files/apple.jpg',
// and so on...
];

var table = Ti.UI.createTableView({});
w.add(table);

var idx = 1;

for (var i in images) {
    var row = Ti.UI.createTableViewRow({
        height: 150,
        className: 'image'
    });

    var imgView = Ti.UI.createImageView({
        image: images[i],
        width: 128,
        height: 100,
        row: idx++
    });
    row.add(imgView);

    table.appendRow(row);
}

    Try pushing the imageViews onto an array after you create them. Then if you need to update the images on certain rows, you can access the imageViews via:

    imgArray[3].image = "myimage.png";
