Alternative API for Titanium Mobile

Hello folks,

During build of my next app I create kind of new API to Titanium Mobile objects which can speed up development because of less typing & more readable & more object oriented code.

Here is short example.

Ti.include('framework.js');

var btn=new Button({    
    title:'About'
},{
    'click':function(){

        alert('Example');
    }
});

var win = new Win({
    title: "Test app",
    backgroundColor: '#bbb',
    leftNavButton:btn
});

var table=new TableView({
    grouped:true // no more Ti.UI.iPhone.TableViewStyle.GROUPED etc
});

var row=table.createRow({
    height:40
},'top');

var label=row.attach('label',{
    text:'Lala'
});

label.set('color','#FF00CC');

win.add(t);
win.open();

What it does for you:

  • no more typing Ti.UI.create …
  • no more addEventListener - it attaches events on creation or using 'on' method in objects
  • attach method for ui objects - it creates new object & automatically attaches it to parent object. Useful for labels

What do you think about this approach ? Should I document it & publish ? :)

  • Hi Paul.

    I would like to see what and how did you done your framework.

    I'm using similar approach. I don't like factory methods, I prefer OOP approach. I've wrote my own framework that is component and event based. It could be a lot better, I'm still learning a lot of things. But I like some things, I can build my custom components and add them to other components in a simple way, for example:

    var view = new Ad.components.ui.View({
    items: {
        'MyLabel': function() {
            return new Ad.components.ui.Label({ text: 'MyText' });
        }
    }
});

    creates "View" that has one label and I can easily access it as "view.MyLabel".

    Also, I have "FormView" component, which is very cool because it automatically collects form values. For example:

    var formView = new Ad.components.ui.FormView({
    items: {
        'SomeText': function() {
            return new Ad.components.ui.TextField();
        },
        'Option': {
            return new Ad.components.ui.Switch();
        }
    }
});

//when user clicks on save button, form values can be fetched easily
var values = formView.getFormValues();
//{ 'SomeText': 'TextInput', 'Option': true }

    Maybe you can use some ideas. :)
