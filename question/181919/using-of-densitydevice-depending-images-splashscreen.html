Using of density/device depending images (splashscreen)

Hi coders,

after the splash screen I want to use the same image from folder /Ressources/android/images/ to show data synchronisation progress. Following this answer from Malcom I use this:

var self = Ti.UI.createWindow({
        fullscreen : true,
        exitOnClose : true,
        theme : 'Theme.NoActionBar',
        orientationModes : [Ti.UI.LANDSCAPE_LEFT, Ti.UI.LANDSCAPE_RIGHT],
        backgroundImage : '/images/default.png'
});

But I don't see my image. The same issue if I add an image view with this "URL".

Whats going wrong? Yes there are a lot of default.png in different folders, yes I see this images on splash screen.

  • Just try to put the image file you need to display in background at this location:

    • project_folder/assets/android/background.png
    • & use this image as backgroundImage : '/background.png'

    Since, i do not know your project directory status, so above thing will work for any backgroundImage or ImageView..

    *You can also create a new folder under ../assets/android/ directory & put the images over there like this:

    // file path is ../assets/android/konrad/image.jpg
// konrad is the folder inside android folder, & contains the 'image.jpg' file
var self = Ti.UI.createWindow({
        fullscreen : true,
        exitOnClose : true,
        theme : 'Theme.NoActionBar',
        orientationModes : [Ti.UI.LANDSCAPE_LEFT, Ti.UI.LANDSCAPE_RIGHT],
        backgroundImage : '/konrad/image.jpg'
});

    It will definitely work.

    Good Luck!
