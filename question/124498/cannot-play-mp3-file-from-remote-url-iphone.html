  • Accepted Answer

    I was able to play the MP3 you listed in the iPhone simulator with this code, though you do need to wait for the whole MP3 to load into memory before it will start to play:

    var url = "http://pyephyohan.info/audio/pahtan_intro.mp3";
var sound = Titanium.Media.createSound({url:url,preload:true});
sound.play();

    I was also able to play it with streaming with this code, no wait:

    var url = "http://pyephyohan.info/audio/pahtan_intro.mp3";
var sound = Titanium.Media.createAudioPlayer({url:url,preload:true});
sound.start();
