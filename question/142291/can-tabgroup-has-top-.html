  • So the documentation says tabgroup has a propert top and heres the description of the property 

    The view's top position.

This position is relative to the view's parent. Exact interpretation depends on the parent view's layout property. Can be either a float value or a dimension string (for example, '50%' or '10px').

This is an input property for specifying where the view should be positioned, and does not represent the view's calculated position.

    As far as in the code, you would pass it to the createTabGroup method call like this 

    var tabGroup = Titanium.UI.createTabGroup({top:100;});
