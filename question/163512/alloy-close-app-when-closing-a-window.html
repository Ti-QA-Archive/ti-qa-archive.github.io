Alloy, close app when closing a window

Hi, I'd like to close app, do not show title screen, I've tried something like this

$.loginWindow.addEventListener('close', function() {

if(!my_check_if_logged_in) {
Alloy.createController("index").destroy();
}
});

but it doesn't work

  • You need to set the exitOnClose property of your main window. When you do close the window, the app wll stop as well. BTW, this only works in Android.
