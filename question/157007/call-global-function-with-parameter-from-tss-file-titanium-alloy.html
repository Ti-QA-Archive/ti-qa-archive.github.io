Call global function with parameter from tss file Titanium Alloy

Hi.. i have this code:

Alloy.js

Alloy.Globals.rotateLeft = function(degree){
    Ti.UI.create2DMatrix().rotate(degree);
}

"myLabel": {
    color: "#000000",
    transform: Alloy.Globals.rotateLeft(90)

Does anyone know how to call that global function from tss file? i've tried that code but it give me error Syntax Error: unexpected token ( and it not works for me.. :( Really need your helps guys.. many thanks..

  • Accepted Answer

    The other answers are right, you can't use functions in TSS. You could do it like this:

    alloy.js

    Alloy.Globals.rotateLeft90 = Ti.UI.create2DMatrix().rotate(90);

    index.tss

    "myLabel": {
    color: "#000000",
    transform: Alloy.Globals.rotateLeft90
}
