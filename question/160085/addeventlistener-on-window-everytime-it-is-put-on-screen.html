addEventListener on window everytime it is put on screen

Hi, I am new to appcelerator and I was hoping to call a function everytime a page is put on screen.

I tried using this code:

var self = Ti.UI.createWindow({
    title:'Hello World!',
    backgroundColor:'black'
});

self.addEventListener('open', function() {
    alert('hi');
});

It does the alert on call when going to page, however I would also like the function to be called when page is shown when a back call is made and you are returned to the page.

Thanks

  • Accepted Answer

    var self = Ti.UI.createWindow({
    title:'Hello World!',
    backgroundColor:'black'
});

self.addEventListener('focus', function() {
    alert('hi');
});
