  • Hello Ryan,

    I just implemented something similar for the app I'm working on. If you're developing for Ipad, you can use the Popover object, it works just great. You can tell where the object should appear on the screen, next to an UI object choose. You have some snippets below: 

    button.addEventListener('click',function(e){
var popover = Ti.UI.iPad.createPopover({
    width: 110, 
    height: 250,
    title: 'Sample Popover',
    arrowDirection: Titanium.UI.iPad.POPOVER_ARROW_DIRECTION_DOWN,
    navBarHidden:false,
    backgroundColor: '#FFF'
});
    popover.show({view: e.source, animated:true});
}

  • I just want to leave a solution here since I faced same challenge recently.

    First you need to set the container (window or view) to layout: composite. This allows you to set the position of each view regardless of other views.

    To ensure which view would be on top you need to give it the appropriate zIndex. e.g. zIndex: 20

  • Hi Ryan Popa,

    Yes, It is possible.

    You can create a View with the zIndex property.

    like this : 

    var guideView= Ti.UI.createView({
    backgroundColor : 'black',
    opacity : 0.8,
    zindex : 20
});

    Refer this link for zIndex Property : http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.View-property-zIndex
