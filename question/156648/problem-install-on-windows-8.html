    Well the last line from you logs mentions this:

    [ERROR] UnicodeDecodeError: 'utf8' codec can't decode byte 0xb0 in position 37: invalid start byte

    This can happen when you are using accented characters (common in Spanish or French) when the files aren't saved using UTF-8 format. Just to be sure, try building a standard app using the wizard. If it works, you will know it is related to something you wrote in your files. If not, you will know it is related to your configuration.
