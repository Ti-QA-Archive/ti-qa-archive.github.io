Code for Android app updates.

Titanium sdk-3.1.0, Android 3.1

I have developed an app for android. When i installed it for first time, the app works. But when i do some changes in source code, and recreate the .apk file , store it on google store. I cant update code which check the version of device and google store for same app. If version of google store is greater than device app version.Then the application on device gives popup for update app. After click ok on the popup then updated version from google store is updates app on device . Any Solutions?

Note : My app consists of database which i dont want to delete so want to update new app on existing one

  • In your tiapp.xml:

    <android xmlns:android="http://schemas.android.com/apk/res/android">
   <manifest android:versionCode="3" android:versionName="1.05">
   </manifest>
</android>

    versionName is what the user sees

    versionCode is used to increment your builds (as recognized by Play Store
