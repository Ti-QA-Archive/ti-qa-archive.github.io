TextField in TableView doesn't get focus when clicking on it

I was just trying to use a TextField in a TableView, but when i click on it, it doesn't get focused. Here's the code I'm using:

var URLField = Ti.UI.createTextField({
    backgroundColor:"red"
});

var URLRow = Ti.UI.createTableViewRow({selectionStyle:Ti.UI.iPhone.TableViewCellSelectionStyle.NONE, height:50});

URLRow.add(URLField);

var quickdata = [];

quickdata[0] = Ti.UI.createTableViewSection({headerTitle:"URL"});

quickdata[0].add(URLRow);

var quicklist = Ti.UI.createTableView({
    data:quickdata,
    style:Ti.UI.iPhone.TableViewStyle.GROUPED
});

win.add(quicklist);

What am I doing wrong?
