    Hi Nicola

    Yes, you need to use the SDK with the extended Google APIs because it includes support for maps, as you rightly state. To do this, follow these steps:

    • verify that you have installed the packages shown in the screenshot at Summary of Installed Android Packages
    • configure your Titanium Studio Default Android SDK as shown in Configuring Titanium Studio with the Platform SDKs
    • when you launch the app via the toolbar button, there is a drop-down menu. Click Run Configurations, and verify that the configuration you are using has Android API: Google APIs Android 2.2 selected, as the screenshot below shows

    Hope this resolves it for you
