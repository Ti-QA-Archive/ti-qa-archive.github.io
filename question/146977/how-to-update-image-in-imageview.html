How to update image in imageView

I have a window with an imageView in it. Depending on which button I click, a new image should appearin the place of the older. The image is supposed to be a remote link (string). In the button click listener event I tried with:

banner.image = "www.blablabla.com/newimage.jpg";

and also with the method:

banner.setImage("www.blablabla.com/newimage.jpg");

banner is the imageView, and the image propriety starts as an empty string ""

It doesn't work, it appears the default image. If I set the image propriety from the beginning, when creating the imageView, to a remote link string, it works.

Any advise?

  • Accepted Answer

    banner.image = &quot;www.blablabla.com&#x2F;newimage.jpg&quot;;

    A protocol would be nice…

    banner.image = "http://www.blablabla.com/newimage.jpg";
