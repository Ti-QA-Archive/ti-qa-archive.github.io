Parse a json response

Hi all ,

My json response is following

{
   "parties":{
      "2012-02-21":[
    {
        "name":"Miss Midnight",
            "ID":"1",
            "eventdate":"2012-02-21",
            "time":"22:00:00",
            "place":"Le rafiot",
            "club_id":"7",
    }
    ],
      "2012-02-22":[],
      "2012-02-28":[]
   }
}

How can i get the value of "name"??

When i write the following i get the parse error.

var response =JSON.parse(this.responseText);
alert(response.parties.2012-02-21.name);

Tried this too but nothing happened

alert(response.parties[0].name);

Thanks

  • Accepted Answer

    You can iterate through json response like this:

    var parties = response.parties;
for(var item in parties) {
    if (parties[item][0]) {
        Ti.API.info( parties[item][0].name + ' - ' + parties[item][0].time + ' - ' + parties[item][0].place );
    }
}
