  • message.Message is a text string. You will need to parse it before continuing.
    var data = JSON.parse(message.Message)

    Also, items: [{owner name:&#39;john&#39;,number:12345,car:&#39;KA 12 HD 7894&#39;}] is not valid JSON. I suggest using an online JSON tool to debug your JSON issues.
