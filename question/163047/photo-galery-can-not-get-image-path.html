PHOTO GALERY... CAN NOT GET IMAGE PATH

Hello .
Open the picture gallery.
Click an image.
can not get the image path

My code : 

Titanium.Media.openPhotoGallery({
            success : function(event) {
                Titanium.API.info(JSON.stringify(event));
            }
        });

return info:

[INFO] :   {"code":0,"type":"success","source":{},"mediaType":"public.image","success":true,"media":{}}

please help. (IOS)

    Hi Serkan KOCAKAFA,
    FYI you cannot get path of the image of the image via gallery in IOS because you cannot access the local Storage. So in IOS you get the object for the image like this 

    event.media//This will give you a blob object representing the image 
event.media.nativePath// This will give you the path of the image in android

    so in ios if you just want to show the image then you can directly pass this blob object to the image property of the imageView, else save it to local applicationDataDirectory the get the path.

    Please feel free to ask if any confusion or mark the answer to help others
