Best way to add Bar with logo on top

I'm trying to add a bar on top of my android/iphone app, which is black, 100% wide and about 1cm high. It has the logo in the middle, which can be a label with a custom font

I have 5 windows and I'm wondering what's the best way to add a bar like I described above. I'd like to create it only once, not in every window I'm creating.

    hi,

    you can create a navBar with view and labels and other views you want to add. then store it with Ti.App.navBar = navBarView;

    you can access that by
    var navBar = Ti.App.navBarView;

    and whenever you want to change any controls' property then access that by its children property.

    like

    
var navView = Ti.UI.createView({
    top : 0,
    left : 0 ,
    width : 320,
    height : 44
});

var lbl = Ti.UI.createLabel({
    text : 'test'
});
navView.add(lbl);

Ti.App.navBar = navView;

    then when you want to change the text of the label

    var navView = Ti.App.navBar;
var lbl = navView.children[0];
lbl.text = 'Test >> 1';

win.add(navView);

    try it and let me know if you have any doubt
