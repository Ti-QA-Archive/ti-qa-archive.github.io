Basic Example to switch windows

Hi,

All examples i've seen are to open windows using tab groups.

I want to create an application which has a main menu which doesn't consist of tabs, then from there be able to click on buttons which open other windows (as child windows so I can use the back button)

Can anyone give a very simple example starting on app.js how to open for example about.js from a link/button so then when you press the back button on the about.js page it goes back to the app.js page.

Thanks for your help

Kevin

    You can look the kitchen sink application on the BaseUI->WindowStandalone

    var main = Titanium.UI.createWindow({
    backgroundColor:'#336699',
    title:'Main Window'
});
var b3 = Titanium.UI.createButton({
    title:'Open New Win',
    width:200,
    height:40,
    top:110
});
main.add(b3);

b3.addEventListener('click', function()
{
    var w = Titanium.UI.createWindow({
        backgroundColor:'#336699',
        title:'New Window',
        barColor:'black',
        url:'newWin.js'
    });
    var b = Titanium.UI.createButton({
        title:'Close',
        style:Titanium.UI.iPhone.SystemButtonStyle.PLAIN
    });
    w.add(b);
    b.addEventListener('click',function()
    {
        w.close();
    });
    w.open();
});

main.open();

  • I've put the following:

    var main = Titanium.UI.createWindow({
    backgroundColor:'#000000',
    title:'Main Menu'
});

var button = Titanium.UI.createButton({
    title:'Open New Win',
    width:200,
    height:40,
    top:110
});

button.addEventListener('click', function()
{
    var w = Titanium.UI.createWindow({
        backgroundColor:'#000000',
        title:'Second Window',
        url:'pages/second.js',
    });
    w.open({modal:true});
});

main.add(button); 
main.open();

    Then on second.js i've put 

    var w = Titanium.UI.currentWindow;

var label = Ti.UI.createLabel({
    text: 'Second Page',
});

w.add(label);

w.addEventListener('android:back',function(e) {
    w.close();
});

    That seems to work, can someone confirm that is the correct practice? if so, I think i've got it!! :)
