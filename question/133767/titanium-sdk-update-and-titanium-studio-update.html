  • Here is what I had to do to fix errors updating 3.x. I would always get an error during the update process somewhere in npm.

    It ends up I had previously installed both NodeJS as well as Git, and I had old versions. I updated both NodeJS and Git to the latest versions, and now all works fine.

    So lesson learned - be careful you're not running old stuff! A suggested Titanium Studio feature would be something that confirms versions of previous installations are sufficient.

  • The problem for me was 2 fold:

    1. I am behind a corp firewall, and have to use a proxy, so I had to set nodejs proxy and http_proxy as documented here https://npmjs.org/doc/config.html.
      I my case I just set HTTP_PROXY and HTTPS_PROXY environment variables to http://hostname:port
    2. I was also having problem using https nodejs registry, so had to point to an http registry to get it working,
      as described in this post
      http://stackoverflow.com/questions/11773509/npm-behind-a-proxy-fails-with-status-403
