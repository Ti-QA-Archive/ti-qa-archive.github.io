  • Hi Gagan,
    The problem is with your provisioning profiles,
    I suggest you to do the following steps :-
    1)Go to the developer.apple.com.
    2) Revoke the current developer certificate.
    3) then click on add new certificate.
    4) it will prompt you for a certSigningRequest file for that:
    a) Open keychain Access.
    b) go to the keychain menu.
    c) their select certificate assistant -> request new certSigningRequest .
    d) enter you email and select save on disk from the radio button and save.
    5) Give the path of this file in the add certificate in your account and create.
    6) Now go to the provisioning profiles tab and click edit and select your certificate from the list and save.
    7) Download both and install again, but before installing delete the previous certificate and profile.
    8) Thats it your problem is solved.

  • Hi

    It sounds like the profiles have become messed up. Whilst previous versions of xcode allowed you to resolve this easily, this is now not available in xcode.

    However the iPhone Configuration Utility 3.5 for Mac OS X (not just for iPhone) allows you to manage the profiles, remove old ones and clean up things. Use it intelligently as removing things yuo should not will cause problems.

    So use the utility, clean things up, create new profiles if needed and import them into your Mac and try again.
