Filesystem - How to rename a directory

Dears,

i didn't find into api docs any method useful to rename a directory.

I'm developing a function that, after a correct download of a huge amount of files it write in append "_OK" on the name of directory.

I don't want to move file (at the end of download) in a new folder with the correct name, then delete previous temp directory just because the procedure may occur 20 seconds (from older tablet models) to move over 300 files from a directory to another.

any ideas?

    There's a method called rename under the Filesystem API

    var newDir = Titanium.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory,'mydir');
newDir.createDirectory();
newDir.rename("mydir2");
