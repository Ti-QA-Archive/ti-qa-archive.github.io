Titanium cannot find android adb

I have been using Ti Dev for a while and have created an Iphone and Ipad app successfully.

When I attempt to set the environmental settings in TI.Dev I get the following message:

  • I think I just went through the same thing… in the newest release of the Android SDK adb.exe is located in the "platform-tools" folder. If you copy adb.exe to the "tools" folder I think you'll find Titanium will work. I'm on Win7.
