Change a class dynamically

Hi,

Do you know if it's possible to change dynamically the classes associated to a TabViewRow (or any other kind of view)?

I have tried :

row = Ti.UI.createTableViewRow({
   classNames: ['class1', 'class2']
});
row.classNames = ['class3', 'class4'];

But the classes used are 'class1' and 'class2' :-/

    I think you have some typos in your question. Do you mean TableViewRow rather than TabViewRow and className rather than classNames?

      Changing the color of a font does not change the layout, and so would not require a different class. I think the same is for hasCheck, as it is a property rather than a view.

      — commented February 16th 2011 by Paul Dowsett

    • Yes it works. I have tried this code in my app:

      tabView.addEventListener('click', function (e) {
    currentRow.hasCheck = false;
    currentRow.color = '#000000';

    currentRow = e.row;
    currentRow.hasCheck = true;
    currentRow.color = '#2222EE';
});

      And the view change.
      But I don't like this method because I hardcode the style in my code.

      I would prefer to do :

      tabView.addEventListener('click', function (e) {
    currentRow.classNames = ['class1'];

    currentRow = e.row;
    currentRow.classNames = ['class1', 'class2'];
});

      Or better:

      tabView.addEventListener('click', function (e) {
    currentRow.removeClass('class2');

    currentRow = e.row;
    currentRow.addClass('class2');
});

      — commented February 16th 2011 by Romain Salles
