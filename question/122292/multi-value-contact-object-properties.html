Multi-value Contact object properties?

If I interpret my Person object and pass it as follows:

{
    email =     {
        home =         (
            "me@home.com"
        );
        work =         (
            "me@work.com"
        );
    };
    emailCount = 2;
    firstName = Micah;
    lastName = Alcorn;
}

I can produce the email values using…

for (each in contact.email) {
    Ti.API.debug(contact.email[each]);
}
// output === [DEBUG] ('me@home.com')
// output === [DEBUG] ('me@work.com')

But how do I produce the label strings 'home' and 'work'?

        work : "me@work.com"
    },
    emailCount : 2,
    firstName:  'Micah',
    lastName: 'Alcorn'
}
