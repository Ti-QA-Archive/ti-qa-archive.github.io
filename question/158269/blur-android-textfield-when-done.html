Blur Android textfield when done.

Hi!

When i am editing some textfields in an Android app and i come to the last field and press "Done" on my keyboard the cursor still stands in the field.

That is not ok. I have functions that needs to make some magic when field is blurred and most important when i save the values from the textfields it wont "Stuck". So am i forced to make an invisible, outside of the view, textfield that the cursor jumps to to get the cursor away from my last field?

    Hi Richard Harrysson,

    when you press done button on keyboard you can use 'return' event and blur() method like this code, i think it will be work.

    aTextField.addEventListener('return', function(e) {
    aTextField.blur();
    alert('Input was: ' + aTextField.value);
});
