TextArea maxHeight ?

How can I set a maxHeight to my textArea?

I have set it to height: &#39;auto&#39; on textArea`s creation… to able user to make a new line by typing return key and upgrade textArea size…

var escreverTexto = Ti.UI.createTextArea({
    borderWidth: 2,
    borderColor: '#fff',
    borderRadius: 10,
    backgroundColor: '#bababa',
    width: 280,
    height: 'auto',
    font: {
        fontSize: 15
    },
    suppressReturn:false,
    bottom: 15,
    paddingLeft: 10,
    paddingRight: 10
})

but HOW can I set a limit for height?

I have tried something like that:

escreverTexto.addEventListener( 'change', function(e){
    if(escreverTexto.height >= 50 ){
        alert( 'a' )
    }    
})

but don't work… there's is a maxHeight command? So I can set it and prevent my textArea to grow more than 50.

Thanks

    var win = Ti.UI.createWindow();

var t = Ti.UI.createTextArea({
    borderWidth : 2,
    borderColor : '#fff',
    borderRadius : 10,
    backgroundColor : '#bababa',
    width : 280,
    height : 'auto',
    font : {
        fontSize : 15
    },
    suppressReturn : false,
    top : 15,
    paddingLeft : 10,
    paddingRight : 10
})

t.addEventListener('return', function() {
    if (t.rect.height > 50) {
        t.height = 50;
    }
});
win.add(t);
win.open();
