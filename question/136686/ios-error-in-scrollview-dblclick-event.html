iOS ERROR in scrollView dblclick event

I've got a simple ScrollView, with a remote image inside
when I double click the image, it checks for scroll view scale value, and then sets it to 1 or 2 accordingly.

    var fullImage = Ti.UI.createImageView({
       image: image,
    });

    var scrollView = Titanium.UI.createScrollView({
        contentWidth:320,
        contentHeight:200,
        backgroundColor:'black',
        showVerticalScrollIndicator:false,
        showHorizontalScrollIndicator:false,
        maxZoomScale:2,
        minZoomScale:1,
        zoomScale:1
    });

    scrollView.add(fullImage);
    win.add(scrollView);

    win.open();

    fullImage.addEventListener('dblclick',function(e){
        (scrollView.scale <= 1.0) ? scrollView.zoomScale = 2.0 : scrollView.zoomScale = 1.0;
    });

on the simulator, and on real iPhone it works just great.
the problem is that once I dblclick I get an error in the console:

[ERROR] Adding an event listener to a proxy that isn't already in the context

and when i drag the zoomed pictuer (after clicked), this error returns

if I remove the dblclick event listener the error doesn't show.

I'm using the latest software and SDK.

I have tried moving the eventListner to different parts of the code, none seems to fix it.

any ideas what is wrong?

and if I sumbit the app like this to apple, will it get reject because of this error ?

thanks guys!
