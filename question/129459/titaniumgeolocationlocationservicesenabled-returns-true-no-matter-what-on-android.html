Titanium.Geolocation.locationServicesEnabled returns true no matter what on Android.

Does anyone have any idea why the code below always returns true? I have disabled every location setting yet when I alert Titanium.Geolocation.locationServicesEnabled it will always be true. Can anyone test or confirm this? I have tried this on a droid x2, htc thunderbolt and droid 3. Using 1.7.5 sdk

if (Titanium.Geolocation.locationServicesEnabled === false)
{
    Titanium.UI.createAlertDialog({title:'Kitchen Sink', message:'Your device has geo turned off - turn it on.'}).show();
}

  • Hi,

    This is because Android always returns true.

    On Android OS 2.2 and above, there is a new, "passive" location provider that is enabled at all times, even when the user disables both the GPS and Network location providers. Therefore, this method always returns true on these devices.

    http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.Geolocation-method-getCurrentPosition
