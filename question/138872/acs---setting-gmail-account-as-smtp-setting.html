  • Hi,

    Here is my setting for gmail.

    Username : myid@gmail.com
Password : gmail password
TLS : true
SMTP : smtp.gmail.com
Port : 587
domain : gmail.com
authentication : login
user sender : false
test mode : false

    Please check your setting against mine.
    Be careful that "user sender" to true will cause your gmail account blocked.
