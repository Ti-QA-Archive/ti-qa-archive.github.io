What format does "start_time" for Events in ACS accept?

I want to create an event in ACS using its REST API from my server. From what I've read to create an event, I only need 2 fields(name and start_time). After coding a bit and running my code, I received "Invalid calender schedule undefined method `to_time' for nil:NilClass" error. I think it might be my date format. I tried following the format in the ACS but there is still the invalid calender error. In the docs, it says Date only, and I am a little confused at this. What format does "start_time" for Events in ACS accept?

Sample Code:
$fields = array(
"name" => "SampleName",
"start_time" => "2015-04-06T02:13:00+0000"
//"start_time" => "2015-04-06"
);

    $c_opt         = array(CURLOPT_URL => "https://api.cloud.appcelerator.com/v1/events/create.json?key=".$key,

    CURLOPT_COOKIEJAR         => $tmp_fname,

    CURLOPT_RETURNTRANSFER     => true, 
    CURLOPT_POST             => 1,
    CURLOPT_FOLLOWLOCATION  =>  1,
    CURLOPT_POSTFIELDS        => array("fields" => json_encode($fields)),
    CURLOPT_TIMEOUT         => 60);
