    Not all View types "close", only a limited number of them do, like Window for example.
    to execute a function on window closing you can listen to the window's close event

    in normal javascript that would be: 

    win.addEventListener('close', function(){
    // logic here
});

    in Alloy you can do the following in the view xml:

    <Window onClose="myCloseHandler" />
