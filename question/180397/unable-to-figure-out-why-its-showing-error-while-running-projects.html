  • Hello Subhajit,

    Thanks for your inquiry, Try Updating your SDK to 3.5.0.GA. Try manually update your SDK. Download SDK from this Link. After download, extract the zip file and put 3.5.0.GA file in …/Titanium/mobilesdk/win../ folder. Now update your CLI with this command

    npm install -g titanium@3.4.1

    Restart your studio, Go to studio/help/check for updates. Install any update that asked you to do so. Hope this helps. Thanks.
