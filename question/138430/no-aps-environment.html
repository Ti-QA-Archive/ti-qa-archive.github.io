No aps environment

I am trying to get push notifications to work and i keep getting the error "no aps environment"

I have deleted and and reinstalled all certificates and provisioning profiles and it is till giving me an issue when i try and run it on my ipad.

Both the Push Service certificates for development and production is in my keychain.

This is driving me NUTS!!!

anyone?
SDK 5.0.2 ios 5.0

  • So the answer to this question is simply that there is an issue with Apple provisioning certificate process. IF you created a certificate without push notifications added and then later on you added Push Notification service Apple DOESN'T update your provisioning profile. You need to create a NEW provisioning profile.

    I wouldn't delete the current certificate unless you haven't distributed your app yet.

    To double check that your provisioning profile has the push notification embeded in it, open the provisionoing file in a text editor and search for "environment" you should see the words

    <key>aps-environment</key>

    if you see this your provisioning profile has been updated.

    Try registering your device and hopefully things should work now.

    If anyone has issues let me know.
