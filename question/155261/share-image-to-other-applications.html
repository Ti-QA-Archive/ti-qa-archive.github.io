  • If you want to share images amongst multiple application, they must be stored in a location that is available to every application. (In other words, not your application directory) A temp directory would probably be the best way to do this.

    Stephen Feather wrote a blog post about doing something similar but for PDF files. Surely you could apply the same logic using images.
