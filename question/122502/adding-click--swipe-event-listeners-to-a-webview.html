Adding 'click' & 'swipe' event listeners to a webview

I need to handle tap and swipe events on a webview. Adding event listeners freezes the local HTML content and prevents the user from scrolling the web page. Is there a workaround?

  • As documented at the bottom of the [webview][1] docs:

    "Since a webview internally wants to handle its own events, scrolling and other related touch events against it's own view surface, you cannot have both Titanium style events against the webview instance and internal Javascript events in the DOM. You must choose between one or the other."

    In other words, no, you can't. You can still use non-UI events such as beforeload or load on the webview, but as soon as you add any UI related event such as tap or swipe, they usurp the internal events.

    [1]http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.UI.WebView-object
