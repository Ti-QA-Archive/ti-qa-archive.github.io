alloy 3.0.0.GA cannot find app.js

compiling and running my app in alloy 3.0.0.GA via. Xcode the app will run in the simulator but not on device. If I compile and package the app from within titanium studio and deploy to device via. iTunes the app will run.

Any suggestions as to what might be the problem? I like to test on device via. an Xcode build so I can monitor output via. the Xcode debug console before packaging for distribution.

  • Hi Martin,

    that's a known issue with SDK 3.0.0GA.

    It looks like, there are JavaScript syntax errors inside your app.js .

    SDK 3.0.0GA continues building, even if there are JavaScript errors and says it was successful.

    As a workaround to easily find that errors you can install the latest CI build of 3.0.1.
    Have a look at this thread and read the comment from Chris Barber.

    Hope that helps,

    Matthias
