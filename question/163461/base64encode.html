I want to pass an image converted in base64 here is my code
tempImageView = Titanium.UI.createImageView({
image:image
});

               var image2 = tempImageView.toImage();
             var   str = Ti.Utils.base64encode(image2);

but I have an error in the instruction var str = Ti.Utils.base64encode(image2); invalid type of argument any help please
