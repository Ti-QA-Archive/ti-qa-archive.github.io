  • firstly please look at the Q & A guidelines about asking a question.

    Having said that I think you might have a small gap in your knowledge and differences between web development and Titanium.

    There is no float: paramater and TSS is not CSS.

    For all Titanium objects and views they have a set of properties, items like top, left, right, width etc. The docs will show the properties associated with a specific object.

    All you can do in TSS is set those properties.

    Now understanding that, if you want an image a specific size in a listview item and to the left, you have to build a custom view inside the listview, which has an imageview positioned left and then a second view to contain the other data you wish.

    I hope this makes sense
