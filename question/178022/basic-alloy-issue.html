    Frank,

    Looks like I figured it out. I didn't know this but, it seems you cannot have the dash character (-) in the name of the id's in your xml files. Get rid of them in welcome-btns and btn-play.

    Change your xml to:

    <Alloy>
    <Window class="container">
        <Label id="title">App Title</Label>
        <View id="welcomebtns"> 
            <Button id="btnplay" title="Play"></Button> 
        </View>
    </Window>
</Alloy>

    That seems to solve it.

    Ray

    • Right, dashes aren't allowed. The ids you assign are turned into variable names. So $.btn-display would be the result, and would be interpreted as $.btn minus display. Same for spaces, can't use them either.

      Don't forget that Alloy is simply a pre-processor that is used to generate standard JavaScript code in the end.
