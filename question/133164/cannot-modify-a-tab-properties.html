Cannot modify a Tab properties

I have created the most minimal application for Android (created a new project, selected the "Tabbed Template") and it works fine.
But whenever I change the title in here :

var tab1 = Ti.UI.createTab({
        title: L('home'),
        icon: '/images/KS_nav_ui.png',
        window: win1
    });

from "home" to say, "map", the title is set to null instead. How does it even make sense ?

    Did you have map defined in strings.xml (located under i18n folder)?

    <?xml version="1.0" encoding="UTF-8"?>
<resources>
    <string name="home">Home</string>
    <string name="map">Map</string>
</resources>

    Checkout this guide for Internationalization.

    For quick testing purpose (without the need to change strings.xml), I'd suggest you to use non-internationalization like this:

    var tab1 = Ti.UI.createTab({
    title: 'map',
        icon: '/images/KS_nav_ui.png',
        window: win1
});
