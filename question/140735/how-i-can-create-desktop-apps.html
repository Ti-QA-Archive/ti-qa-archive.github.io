    If you wish to manage your existing desktop apps, you can install the Open-Source version of the existing Titanium Desktop plugins for Titanium Studio. Thos plugins are available here:

    • Stable versions: http://studio.appcelerator.com/plugins/desktop/index.html
    • Betas: http://preview.appcelerator.com

  • Hi Javier. The desktop SDK is by no means dead and there has strong effort underway to ensure the long term viability of the project. You can see the project at its home on github at https://github.com/TideSDK. You will find some help in our wiki, and mailing list. Visit http://tidesdk.org for details. There are helpful folks and the TideSDK devs on #tidesdk on freenode. Our development version of TideSDK builds for OSX Lion, Mountain Lion, Windows 7 and 8 and Ubuntu 12.04. We are not yet at a release but will be announcing a date soon. You can get started with downloads of Titanium Desktop from our site in the interim.
