  • I have done all steps above without any luck. Finally a friend told me for some reason I have to install the old Android 2.3.3 as well on top of the latest SDK. And indeed now finally it shows it as ticked green. On another note having to downgrade to Java 1.6 didn't help at all. I have installed everything on a new laptop with Java 1.7, Android SDK 2.3.3 + 4.2.2 + Android SDK tools 22 and edited ~/.bashrc and added these lines 

    export ANDROID_HOME=/home/hooman/Android-sdk
export PATH=$PATH:/home/hooman/Android-sdk/tools

    and added the symlinks:

    cd $ANDROID_HOME/tools
ln -s ../build-tools/17.0.0/aapt
ln -s ../build-tools/17.0.0/dx

    and now it finally seems to work. (Why is Android 2.3.3 mandatory to have?) What a mess !!!
