  • Hey Prashant,

    Try going to '/Library/Application Support/Titanium/sdk/osx' you should have all your SDKs installed there. It's not inside your user directory (not inside ~), but instead inside the root/main partition.

    Open up Finder, select your hard drive, and navigate to the above mentioned path. It's important that you don't navigate to the Library folder inside your user area.

    Let me know if this works for you.
