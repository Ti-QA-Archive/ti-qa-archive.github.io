    • OK Taylor, I would recommend you to completely uninstall and reinstall Titanium Developer as follows:

      • close all Titanium and android programs
      • run the Titanium Developer uninstaller, which is located in the folders where Titanium Developer is installed (usually C:\Program Files)
      • delete the Titanium application settings directory. On Win7, this is C:\Users\appcel\AppData\Roaming\Titanium
      • delete the directory where Titanium stores the android sdcard. On Win7, this is C:\Users\appcel\.titanium
      • delete the .deltafy directory. On Win7, this is C:\Users\appcel\.deltafy
      • Once Titanium Developer has been completely removed, reboot your machine
      • disable your antivirus and local firewall (not your router firewall, or else your system will be completely unprotected from Internet threats)
      • reinstall Titanium Developer

      Let me know if that improves matters.

      — commented February 9th 2011 by Paul Dowsett
