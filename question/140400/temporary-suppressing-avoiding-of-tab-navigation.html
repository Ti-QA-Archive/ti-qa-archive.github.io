Temporary suppressing (avoiding) of tab navigation

Hi,
In my project I want to supress changing of tab during xhr process.

Any ideas?

Rainer

  • Accepted Answer

    Put a view over the entire screen (with the highest zIndex) and it will cover all the UI behind it and prevent all user interaction. If you don't want it to be visible (but still do the job) set the opacity to 0.0.

    It might be a good idea to include a loading indicator inside this same view while the XHR is processing so the user knows they need to wait. :)
