  • Not sure if it will help you… but in my case i solved it with this…(i have only tested it on android, but should work also on iPhone)

    So i have a new window, where i get the location (with Titanium.Geolocation.getCurrentPosition and Titanium.Geolocation.addEventListener(&#39;location&#39;) and when i close that window i call the removeEventListener with location, and the gps stops…

    wmd.addEventListener('close',function(e){
    Ti.Geolocation.removeEventListener('location',function(e){});
});

    ( if this didn't work for me, my device battery would last like 2 or 3 hours…fully charged…)

    Hope it helps!!!
