    • If it helps, I was able to copy and paste the contents of a sample (assets, controllers, views, models, styles, views, migrations, etc) into a new Titanium Project using the Alloy template. Then I was able to build successfully at that point. It has worked for me for the two different samples I have tried.

      I also tried the import "Existing folder as New project" option to no avail. I think Eclipse just creates it as a regular project instead of a Titanium Alloy project which has the necessary bindings for building. Hopefully that helps you.
