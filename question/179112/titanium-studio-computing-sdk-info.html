  • Hi Petr Malakhaltsev,

    I have seen that other people are getting the same issues. Looks like your update is broken. There are some workarounds that might work for you.

    If you are not using the latest release of titanium, then you should update your Ti CLI, Alloy and node.ACS using the following commands.

    • Alloy - command sudo npm update -g alloy
    • node.ACS - command sudo npm update -g acs
    • Titanium CLI - command sudo npm update -g titanium

    Then reopened Titanium Studio, went to Help –>Check for Titanium Updates, and the only remaining item was Titanium Studio itself. Update studio, then it should work.

    Remember: After updating your Ti Studio, you will need to restart the Ti Studio.

    You can also check the link: Titanium SDK : An internal error occurred during: "Computing SDK Info…". java.lang.NullPointerException

    Let us know how it goes.

    Thanks
