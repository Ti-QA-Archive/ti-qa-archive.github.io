Webview pass Ti.App to remote page

Hi,
I have a problem with webview. I want to show a remote page and give that remote page access to the local Ti.App-object. Now i understand that this is prevented for security reasons but that wont be an issue in this application. I have tried the iframe-hack but as iframes doesn't seem to accept cookies, that will not work. The problems appear on iOS.

  • This wont work i think. Regardless of using cookies. The cross domain restriction on reading them (localhost and your remote domain) means you cant. The restriction on remote code execution on local js objects is there to ensure you CANNOT do this as it would violate the Apple guidelines on apps that contain self-modifying code.
