How to Change the Default (Blue) Color of a TableView Row When Selected

Hi All,

How do I change the default color of a TableView row when clicked/selected?

UPDATE:

Found it. Set (any of) the following properties of the TableViewRow object:

selectedBackgroundColor
selectedBackgroundImage
selectedColor
selectionStyle
