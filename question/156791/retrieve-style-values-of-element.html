    we can.

    Let say element as button

    var button = Ti.UI.createButton({
    height:44,
    width:200,
    title:L('openWindow'),
    top:20,
    color:'red'
});

    on event listener you can get the source values. 

    button.addEventListener('click', function(event) {
    Ti.API.info(JSON.stringify(event.source)); //look at the console.
});
