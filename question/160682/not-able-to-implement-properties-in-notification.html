  • I think the question is about Ti.UI.Notification, not Ti.Android.Notification.

    Ti.UI.Notification is Titaniums Proxy of an Android Toast. Unless Ti.UI.Notification (wich inherits from Ti.UI.View) the native Toast doesn't extend android.widget.view. A Toast isn't intended to be customized, it should provide an easy way to notify users.

    Now Ti.UI.Notification has all the properties of an usual Ti.UI.View like top, left, background, border etc. but setting those will not effect the toast message. There are ways to manipulate the appearance of toast notifications but in most cases this isn't possible in Titanium.

    So I would suggest you to choose a custom implementation like Toasty
