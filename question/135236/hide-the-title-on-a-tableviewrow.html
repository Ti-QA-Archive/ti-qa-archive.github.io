Hide the title on a TableViewRow

I want to hide the title on a TableViewRow.
However I must set the title because I'm using a search bar and it can't set it to blank because the searchbar searches into the row title itself…

So unless I can "say" to the searchbar to look into another property of the row or just hide the title or put it in white or something…

    In table creation you can use parameter filterAttribute:

    var tableView = Ti.UI.createTableView({
        separatorColor : '#533',
        selectedColor : '#ddd',
        search : search,
        filterAttribute : 'filter',
        searchHidden : false,
        top : 40,
        backgroundImage : Ti.Filesystem.resourcesDirectory + 'assets/parts/bg.png'
    });

    And in row you can set it:

    row = Ti.UI.createTableViewRow({filter:'Text to search'})

    Hope this example helps.
