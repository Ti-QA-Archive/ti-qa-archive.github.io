Android heavyweight window with tabs

I need to create an android heavyweight window so that the back button does not close the application. When I do this, I lose the tab bar along the top of the window. Is there a way to get the tab bar back? I've tried using tabBarHidden:false.

Perhaps there is another way to get get the back button functionality without creating a heaveyweight window?

I'm using the continuous build 1.6.0 and APIs 2.2.

Thanks
Brian

Here is my code:

var categoryWin = Ti.UI.createWindow({
    backgroundColor: "#000b43",
    url: "main.js",
    barColor: '#000644',
    barImage: Titanium.App.Properties.getString("resourceFolder")+'images/header/header_logo.png',
    titleImage: '',
    modal:true,
    tabBarHidden:false
});

categoryWin.open();
