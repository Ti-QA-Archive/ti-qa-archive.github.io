    You didn't mention which type of barcodes you need to scan. Personally, I don't think camera based scans are suitable for high-volume scan environments. They also seem to have an easier time with adequate size QR codes than say UPC or simulator.

    For my purposes, I went with the Linea Pro line of scanners. If you have been in an Apple store since about Oct 2009, this is the same scanner which Apple now uses in their own stores for POS scans and credit card swipes. They use the model which attaches to an ipod touch but the company just released 6 different models for the iphone4 as well.

    I had to write a small custom obj-C module to integrate their SDK library with Appcelerator Titanium. But the resulting laser scans are very fast and reliable. IMHO, laser scans are the only reasonable way to go for high volume environments.

  • I don't have any specific metrics, but the recently released Ti+plus barcode scanner module works great and looks quite fast to me. It's based on the ZXing library, which is kind of standard.
    You'll need at least an Indie subscription to be able to test it on device. It is prety straightforward to use. Below is code from the example code:

    
// This is a test harness for your module
// You should do something interesting in this harness 
// to test out the module and to provide instructions 
// to users on how to use it by example.


// open a single window
var window = Ti.UI.createWindow({
  backgroundColor:'white'
});
window.open();


var button = Ti.UI.createButton({
    title:"Scan QR Code",
    width:150,
    height:40,
    top:20
});

window.add(button);

button.addEventListener('click',function()
{
    Titanium.Barcode.capture({
        success: function(event)
        {
            alert("success = "+event.result);
        },
        cancel: function(event)
        {
            alert("cancel");
        },
        error: function(event)
        {
            alert("Error. "+event.message);
        },
    });
});

    At the moment is IOS only. You can implement an Android method using Intents and calling the zxing activity.
