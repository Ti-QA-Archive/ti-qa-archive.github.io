How to conditionally disable table row click?

I have a table view where originally I had a switch in each row, and the switch was the only event I responded to.

I modified the code using custom tableViewRows, and no respond to the row click and set an image to toggle instead of a switch.

I have conditional logic that once a given number of rows have been clicked, all the remaining rows should be disabled. I had a companion function to re-enable them if users de-selected switches (so they could change their minds).

With the switches, this function worked fine:


function disableUnusedSwitches() {
    for (i=0; i<tableView.data[0].rows.length;i++) {
        if ( tableView.data[0].rows[i].children[0].value == false ) {
            tableView.data[0].rows[i].children[0].enabled = false;
        }
    }
}

However, now that there are no switches, how would I disable the ROW from responding to a click?

    You could probably set touchEnabled to false for the row, but it seems it would be easier to just let the click occur. Then in your table event handler for the clicks, determine whether or not you should toggle the image based on how many rows are "on". If already at the max, just ignore the click or display a message to deselect another row first.
