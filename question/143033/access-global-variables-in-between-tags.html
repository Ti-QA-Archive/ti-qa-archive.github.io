Access global variables in between tags

Hi I am new to titanium. I want to bind html content to a webview. In this regard I declared a variable and initialised the variable with html content like

var action = ['hi', 'hello', 'how are you'];

var sample = "<html><head><script></script></head><body></body></html>";

prior to this sample variable I declared an array variable which I want to access in between the <script> tags of sample variable. I want to define a function in between <script> tags using the action array variable. I am able to declare and define function in between <script> tags by declaring array directly with in <script> tags. For example:

var sample = "<html><head><script>var array = ['hi', 'hello' , ' how are you']; </script></head><body></body></html>";

In this way I can declare the array variable statically within the tags and can access it. But if I use as follows:

var sample = "<html><head><script>var array ="+action+"; </script></head><body></body></html>";

the array variable within tags is not getting assigned. I tried in so many ways. As of now I am hardcoding the action array variable. In furthur I will initialize the action array with dynamic data. Thats why I need to pass action array variable with in the <script> tags. But I am not able access action array variable with in <script> tags. Can anyone share the solution ASAP.

Thanks in advance…

    Try this:

    var sample = "<html><head><script>var array ="
    + JSON.stringify (action) 
    + "; </script></head><body></body></html>";
