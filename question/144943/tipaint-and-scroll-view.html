Ti.Paint and Scroll View

Hi folks,

I have a scroll view which I allow the user to add extra views too on a button press.

When they press the button a view is created with a PaintView added to it, then it is added to the ScrollView so that they can create a collection of frames that they can draw onto individually then scroll through. (Something like how a flip book would work).

I am having a problem when adding more than 4 views to the ScrollView, what happens is when the user reaches 4 views in the scrollable view the PaintView is cleared of there drawings, but the information on the 3rd one stays. As you add more and scroll back and forth it seems that the when you go 2 views away from one that is when the information is cleared.

The users are also able to add ImageViews onto the main view in the scrollable view, but these stay no matter how far you go.

Has anyone any idea why a scrollable view would clear information after scrolling past 2 views?
