get php parameters

Im trying to implement a GET request to a PHP file which is just echo the parameters that get. The app gets the value of a textArea and make a get request to my php file, the request is working properly but if im trying to send "error" form the php file, the alert("not found!") is not firing.. here is my code

    userId = Ti.Network.createHTTPClient({
        onload: function(e){
            activityIndicator.hide();
            ID = this.responseText;
            Ti.API.info(ID);
            if(ID == "error"){
                alert("not found!");    
            }
            else{
                alert(ID);
            }

        },
        onerror: function(e){
            alert("error");
            activityIndicator.hide();
        },
        timeout : 5000
    });

    userId.open("GET","http://myserver.com/login.php");
    var params = {
        id: textInput.value
    };
    userId.send(params);

and here is the php code

<?php

$id = $_GET['id'];

if($id) {
    echo $id;
}
else {
    echo 'error';
}

?>
