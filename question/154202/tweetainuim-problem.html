  • Tweetanium was written in 2010.

    So it would make sense that the codebase isn't on par with the latest version of the SDK.

    And even if you (or someone) managed to get it working with the current Titanium SDK, it would not be compatible with the latest version of the Twitter API. (Twitter completely discontinued their old API a few weeks ago)

    My best advice to you would be to find another sample application to base your work on.
