  • Lets say you have a database {project}/Resources/mydb.sqlite3 and you install the app.

    If you then modify {project}/Resources/mydb.sqlite3 and run the app again, the old database still exists on the device/simulator, it is NOT overwritten.

    You can delete the app from the device/simulator.

    You can delete the file from the file system (in finder for iOS or through DDMS on android)

    Your code can do an update of the database.

    Your code can delete the old database and then install the new one.
