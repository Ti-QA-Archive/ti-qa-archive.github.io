Announcement: open-source tutorial app, GrillTime

I'm happy to tell you all that I've finished up my open-source tutorial app, GrillTime. I've been working on this for about two months (with the bulk of that spent wrestling with alarms, services, and notifications on android).

If you want to see what it looks like, check out these images: ios, android

The app itself is a multi-stage timer designed for grilling, along with a reference guide for safe cooking temperatures. I walk you through the entire process of building the app, sharing all the challenges I faced along the way.

The tutorial covers such topics as:

  • dynamic UI construction (fitting UI elements to the available space in the
    given orientation)
  • image processing for different screen densities (I provide the original
    artwork along with the command-line scripts I use to process the images)
  • Local Notifications on iOS
  • AlarmManager and Notifications on android
  • Services on android
  • persistent data with Ti.App.Properties
  • fitting background images to the full window without distortion

I hope somebody finds it useful. Check it out here.
