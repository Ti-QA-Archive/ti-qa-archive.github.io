Use same view in all windows

Hi guys

I want to place a view with a height of 20 pixels at the bottom of the screen (above the tabGroup) that's used for all tab windows.

Instead of having to add the view for each window, is there a way to do this when I create my tabGroup and windows? The view has a label that can be changed in each window. So when I go to another window, the same text should be there.

I hope you understand - thanks!

    Your best bet would probably be to create a module for generating a window with this label. Here's a quick and dirty sample:

    var createLabelWindow = function() {
    var win = Ti.UI.createWindow();
    var content = Ti.UI.createView({
        height:'auto',
        width:'auto'
    });
    var label = Ti.UI.createLabel({
        text:'your text',
        height:20,
        width:'auto',
        bottom:0
    });

    win.add(content);
    win.add(label);
    win.content = content;
    win.label = label;

    return win;
};

    And now you can add whatever you need to add to that window via the content property, which will always be behind the label. You can access your label via the label property. There are undoubtedly cleaner ways to do this, but this should point you in the right direction.
