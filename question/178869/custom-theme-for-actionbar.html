Custom theme for actionbar

Hi,

I've generated the holo theme and the action bar theme by following the procedure: Using Both the Android Holo Colors Generator and Android Action Bar Style Generator and everythings works. My tabGroup has the color that I defined.

I want to hide the actionBar so I added the two lines in the themes_mythemes.xml

<item name="android:windowActionBar">false</item>
<item name="android:windowNoTitle">true</item>

The result is that the action bar is hidden BUT the color of the tabGroup is the default blue and not the color from my theme.

So, how to hide the actionBar but still have the theme for the tabgroup?

Thank you.

  • Hello, Vincent

    Thanks for your inquiry. To automatically hide the action bar when opening a window or tab group add a custom theme file to your project.

    platform/android/res/values/custom_theme.xml

    <?xml version="1.0" encoding="utf-8"?>
<resources>
    <style name="Theme.NoActionBar" parent="@style/Theme.AppCompat">
        <item name="android:windowActionBar">false</item>
        <item name="android:windowNoTitle">true</item>
    </style>
</resources>

    Now, Modify your tiapp.xml file to use the modified theme. This will globally hide the action bar in all of your activities.

     <android xmlns:android="http://schemas.android.com/apk/res/android">
    <manifest>
        <application android:theme="@style/Theme.NoActionBar"/>
        <!-- Need to specify at least API level 11 for Titanium SDK 3.2.x and prior -->
        <uses-sdk android:minSdkVersion="14" android:targetSdkVersion="19"/>
    </manifest>
</android>

    Now for different tabGroup color, use backgroungColor property in tabGroup. Hope this helps. Thanks.
