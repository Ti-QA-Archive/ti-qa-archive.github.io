  • Hi

    If all you are wanting to do is add the text over the top of an image you can simply add a label to that image.

    var img = Ti.UI.createImageView({
    height: Ti.UI.SIZE,
    image: 'http://placekitten.com/200/200',
    width: Ti.UI.SIZE
});
var lbl = Ti.UI.createLabel({
    color: '#fff',
    font: {
        fontSize: 24
    },
    height: Ti.UI.SIZE,
    text: 'Text on Image',
    textAlign: 'center',
    width: Ti.UI.SIZE
});
img.add(lbl);
win.add(img);

    If you then wanted that combination in an image you could upload or save to the local file system;

    var newImage = img.toImage();
