Variable TableViewRow height

I've read all that I can find online about TableViewRow's and how to set the height to auto and can't find a solution that works for me. I have an array of data and loop through it calling a method which creates and returns a TableViewRow. Each row could be of variable height because of the 'NamelessContent' field shown below. This iteration of my code tries to set the rowWrapper height to auto in using a class name and the same issue still occurs: when set to auto, the TableViewRow is 100% so it takes up the entire screen. I can scroll through but each row is the entire height. The height of the TableViewRow depends upon the content in the 'NamelessContent' field so sometimes it will be 60 pixels and others it could be 250 pixels. Any help or things to try are very appreciated.

  • I assume you're using Ti SDK v2? If so, then you should set the height of your rowWrapper to Ti.UI.Size. This will make it so that it fits the size of the contents and doesn't take up the whole height of the window. Don't set a height on row, it will auto fit to the size of the rowWrapper view.
