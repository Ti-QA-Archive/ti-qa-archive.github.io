  • Hi Kiran

    I think your requirement is to access the inner value as a different variable.For that you can use Json object.

    Example :
    var testValues = {
    name : 'abc',
    token :'test'
    }

    Ti.API.info(testValues.name); // Will return : abc

    Ti.API.info(testValues.token); // Will return : test

    In this way , different values can be accessed.Hope this will help .If this is not required , completely specify requirement so that i can provide my help at best.
