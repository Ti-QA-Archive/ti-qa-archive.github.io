    • Dan,

      I tried using background image and it works: thanks a lot!

      var coloredMiddleImage =  Titanium.UI.createView({
    backgroundImage: '/images/qbar_'+barColor+'_mid_slice_10@2x.png',
    top: topDp,
    left: '30dp',
    width: imageWidth+'dp',
    height: height,
    touchEnabled: false,
});
view.add(coloredMiddleImage);

  • All you need: 2 Views, one for background (empty bar) and one for the overlay (progress)

    onProgress: function(e) { 
this._view2.width = e.percentage/100 * this._view1.width; 
}
