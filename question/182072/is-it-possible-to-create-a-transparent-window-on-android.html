  • Hi Arjan Brinkman,

    Thanks for your concern . You can use opacity to create a popup with a semi/transparent background.

    Example code

    var win = Ti.UI.createWindow({
    backgroundColor : 'white'
});

win.open();

var aButton = Ti.UI.createButton({
    title : 'open popup',
    height : "auto",
    width : "auto",
});

aButton.addEventListener('click', function() {
    openModalWindow();
});

win.add(aButton);

function openModalWindow() {
    var view = Titanium.UI.createView({
        backgroundColor : '#000',
        opacity : 0.3,
        height:'70%',
        width:"70%",
        borderRadius:5,
    });


    var aLabel = Ti.UI.createLabel({
        text : 'Popup Window',
        color : 'white',
        font : {fontSize:20},
        height : "auto",
        width : "auto",
        top : 2,
        textAlign : 'center'
    });

    view.add(aLabel);

    win.add(view);
}

  • Arjan, if i am right then you are trying to display a view with a transparent background AROUND the view.
    You can do it this way:

     var win = Ti.UI.createWindow({
    backgroundColor : '#80000000'
});

var view = Titanium.UI.createView({
        height : '70%',
        width : "70%",
        borderRadius : 5,
    backgroundColor : 'green'
    });

var aLabel = Ti.UI.createLabel({
        text : 'Popup Window',
        color : 'white',
        font : {fontSize:20},
        height : Ti.UI.SIZE,     
        width : Ti.UI.SIZE,
    });

    view.add(aLabel);

    win.add(view);

win.open();

    Above code will display a view of height=width=70% with a green background & containing a Label in middle of it.
    This green View will have a semi-transparent background around itself, which i hope what you want to achieve.

    Below are the some modifications from Md. Harish Uz Jaman Mridha answer:

    • don't use auto property, it has been deprecated
    • setting the OPACITY property will make the whole view transparent (even its children ) which will not be good for UX.

    Hope my answer helps you :)
