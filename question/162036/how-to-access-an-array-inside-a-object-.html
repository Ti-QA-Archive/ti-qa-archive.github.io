  • Complex Object:

    var myObject = {
    basicProperty: 'basicProperty',
    basicArray: [1,2,3,4],
    arrayOfObjects: [
     {arrayObjectProperty: 'object1Property'},
     {arrayObjectProperty: 'object2Property'}
    ]
}

    console.log(myObject.basicProperty); // basicProperty
    console.log(myObject.basicArray[1]); // 2
    console.log(myObject.arrayOfObjects[1].arrayObjectProperty); // object2Property

    that should be right. Not sure why you are messing with Object.keys

    The only time you would have to mess with the keys, is if you have named objects, usually returned by the lousy json output from a php server.
