    • There is too much referenced code here that is not supplied for me to be able to check everything of course, for example you have a whole app name space I have no code for and each window code is not here.

      But there are two things that jumps out;

      selectTab is referenced before it is actually defined, so this event listener will never work.

      self.addEventListener("focus", selectTab);
function selectTab(e) {
    app.envr.infor("@ApplicationTabGroup\selectTab " + e.index);
    app.ui.currentTab = e.tab;
}

      Change to

      function selectTab(e) {
    app.envr.infor("@ApplicationTabGroup\selectTab " + e.index);
    app.ui.currentTab = e.tab;
}
self.addEventListener("focus", selectTab);

      Setting window outside the tab create is not always successful. This should be set at creation time ideally.
