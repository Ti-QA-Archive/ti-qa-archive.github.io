  • I'm not sure it's the same bug as when I last looked but back then the problem was that Titanium is trying to load a native module and failing. Appcelerator apparently forgot that they're using the same function name (require) for both including titanium modules and CommonJS-modules on android so the app chokes. I suggest just using Ti.includes and self-calling anonymous functions to get namespace separation. It'll work better for now.
