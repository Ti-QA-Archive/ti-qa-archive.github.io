AlertDialog does on fit on Android screen in Portrait

Hello,

I am trying to open an alertdialog in a window on my android app and it appears as it is always sizing itself for landscape mode. I have tried setting the width property but it does not seem to make any difference. I am using the code directly from the API example (alloy):

    <AlertDialog id="dialog" title="Delete"
            message="Would you like to delete the file?" cancel="1" >
            <!-- The ButtonNames tag sets the buttonNames property. -->
            <ButtonNames>
                <ButtonName>Confirm</ButtonName>
                <ButtonName>Cancel</ButtonName>
            </ButtonNames>
        </AlertDialog>

Any Ideas?

  • ButtonNames are create only mode. http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.AlertDialog-property-buttonNames. That's why you can't able to set width.

    If something that you have more buttons I suggest you to create your own views and add event listener to open up the view.
