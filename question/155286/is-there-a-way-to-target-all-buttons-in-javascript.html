Is there a way to target all buttons in Javascript?

I have a group of buttons that need to get enabled after a user clicks on the first radio button at the top of a table but I'd like to enable all buttons on the page and set the opacity to 1 after the user clicks this first radio button though. I don't want to manually set all the button properties.

I know you can target a certain button, like so:

$['mybutton_id'].opacity = 1;
$['mybutton_id'].enabled = true;

I would like to target all buttons on the page in the javascript after a user clicks on the first radio button. Any ideas how I could go about doing that?

    Put your buttons in an array

    var myButtonArray = [];
myButtonArray.push($['mybutton_id']);
myButtonArray.push($['mybutton_id1']);
myButtonArray.push($['mybutton_id2']);

etc....

    then

    for (var x=0;x<myButtonArray.length;x++){

myButtonArray[x].opacity = 1;
myButtonArray[x].enabled= 1;
}
