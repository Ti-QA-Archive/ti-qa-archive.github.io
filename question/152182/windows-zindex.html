  • Hello,

    the zIndex won't work in Windows. Only with views. 

    var win = Ti.UI.createWindow({
    backgroundColor:'black',
});

var view = Ti.UI.createView({
    zIndex:1,
    backgroundColor:'white',
});

win.add(view);

var view2 = Ti.UI.createView({
    zIndex:2,
    backgroundColor:'red',
});

win.add(view2);

win.open();

    I'm adding a bug for docs, since zIndex in windows won't work. The order to show windows is will show you the last one you open.

    If you want to play with which window you want on top, you use window.show or window.hide

    Best,
    Mauro
