how to abort an upload

Hi,

im uploading an image to a server. i have cancel button, when i click on it, i execute this : 

backButton.addEventListener('click', function(e){
        //cancel download
        if(clientUpload != null){
            Ti.API.info('client not null');
            if(!isNaN(clientUpload.readyState)){
                clientUpload.abort();
                Ti.API.info('aborted');
            }
        }
        //close window
        winProgressBar.visible = false;
    });

When i debug, the client is aborted (after 70% of upload for example) but the image is uploaded successfly…

what should i do ?

thanks

  • Accepted Answer

    You can abort a network process using the abort() method, but it doesn't change the fact that you've uploaded 70% of an image to the server. So what did get uploaded is still there. If you call an abort, you may want to make a secondary call to the server to run a script that removes that file .. or only write to a temp directory as you upload and then move the file only when upload is complete.
