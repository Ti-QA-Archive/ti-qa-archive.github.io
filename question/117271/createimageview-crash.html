createImageView crash

Hi All,

I'm an experienced programmer but new to Titanium. I must be doing something very silly. I can't get an image to load up. The following code just causes the iPhone emulator to exit with no error messages etc:

var win1 = Titanium.UI.createWindow();
var imageView = Titanium.UI.createImageView({image:'home.png',
        width:261, height:178});
win1.add(imageView);
win1.open();

If I swap the ImageView for a plain View it works. So it must be the ImageView causing it. I have tried url property instead of image. I have the image in my Resources folder. I have checked that it is being copied into the app bundle.

Thanks in advance,
John
