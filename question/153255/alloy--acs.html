    • No, you are not required to use models if you are not using an adapter. You can however use plain old backbone models that you populate with data from your ACS queries and still take advantage of Alloy's model functionality, like data binding. You can check this simple example of how you can use plain backbone models and still use data binding: https://github.com/appcelerator/alloy/tree/master/test/apps/models/binding_no_persistence
