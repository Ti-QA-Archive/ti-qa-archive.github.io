  • for that you should try to create a dictionary and pass that dic to each row
    like…

    var dic = {
    title:title,
    date:date,
    media : media,
    desc : desc
}
row.thisObj = dic;

    this stores each object to the row when you want to get at the time of click event of tablview , you can get this back by using the following code

    var obj = e.row.thisObj;

    and you can get this by using…

    obj.title,

    obj.date
