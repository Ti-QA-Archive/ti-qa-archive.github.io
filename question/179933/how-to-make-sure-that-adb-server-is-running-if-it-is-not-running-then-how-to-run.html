  • Hi Mahmudul Hasan,

    Thanks for your concern.

    You can follow the link to Restart adb

    Please check which version of JDK you are using now because The 32-bit version of the JDK is required regardless of whether Titanium is running on a 32-bit or 64-bit Windows system.

    Check the link Installing_Oracle_JDK for details.

    If this doesn't resolve your problem then try some steps below.

    • Check your environment variable path configuration, clean your project and run again.

    • If anything doesn't work, then just uninstall node.js, titanium studio and delete the supported files (.titanium, .android, .eclipse) from C –> User –>Your PC name and titanium studio folder from C–> User –>Your PC name –>appData. After all re-install, update node and CLI completely and set the android SDK path then restart PC and run again.

    Hope this helps.

    Thanks
