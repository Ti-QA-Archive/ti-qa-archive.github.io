  • You can do flip transition by following ways 

    var window = Titanium.UI.currentWindow;
var windowView = Titanium.UI.createView({
    height: WindowHeight,
    top: 0,
    width: WindowWidth,
});
window.animate({view:windowView,transition:Ti.UI.iPhone.AnimationStyle.FLIP_FROM_LEFT});
// add all the elements to the view.

    Second way to flip is :

    var window = Titanium.UI.currentWindow;
window.open({transition:Titanium.UI.iPhone.AnimationStyle.FLIP_FROM_LEFT})
