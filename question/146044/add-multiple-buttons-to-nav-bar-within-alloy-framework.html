Add multiple buttons to nav bar within Alloy framework

The general way to add multiple buttons to the navigation bar using the 2.0 framework was to build a view that contains two buttons and then add the view to the rightnav or leftnav property of the Window. I've tried the same solution using the Alloy (3.0 framework) but did not get the same results. Anyone have a solution for something like this?

    Do this instead:

    index.xml

    <Alloy>
    <Window id="index">
        <NavigationGroup id="navgroup" platform="ios,mobileweb">
            <Window title="Window 1" id="win1">
                <Label>This is the first window</Label>
                <LeftNavButton>
                    <View id="view">
                        <Button id="leftOne" title="leftOne" top="-50dp"></Button>
                        <Button id="leftTwo" title="leftOne" top="-50dp"></Button>
                    </View>
                </LeftNavButton>
                <RightNavButton>
                    <Button onClick="functionToOpenWin2">next</Button>
                </RightNavButton>
            </Window>
        </NavigationGroup>
        <Label platform="android">NavigationGroup not available on Android</Label>
    </Window>
</Alloy>

    index.js

    if (OS_IOS || OS_MOBILEWEB) {
    Alloy.Globals.navgroup = $.navgroup;
}

function functionToOpenWin2() {
    $.navgroup.open(Alloy.createController('win2').getView());
}

$.index.open();

  • I used the navgroup project from alloy/test/apps/ui as base starting point link here if needed. Modified that project so the index.xml looks like this:

    <Alloy>
    <Window id="index">
        <NavigationGroup id="navgroup" platform="ios,mobileweb">
            <Window title="Window 1" id="win1">
                <Label>This is the first window</Label>
                <View id="view">
                    <Button id="leftOne" title="leftOne" top="-50dp"></Button>
                    <Button id="leftTwo" title="leftOne" top="-50dp"></Button>
                </View>
            </Window>
        </NavigationGroup>
        <Label platform="android">NavigationGroup not available on Android</Label>
    </Window>
</Alloy>

    index.js (controller)

    if (OS_IOS || OS_MOBILEWEB) {
    // attach the navgroup to Alloy.CFG so it can be accessed globally
    Alloy.CFG.navgroup = $.navgroup;

    // We can do most of this in markup once we resolve:
    // https://jira.appcelerator.org/browse/ALOY-202
    //
    // TODO: find out why nav buttons and titles don't work for mobileweb
    var button = Ti.UI.createButton({ title: 'next' });
    button.addEventListener('click', function(e) {
        $.navgroup.open(Alloy.createController('win2').getView());
    });
    $.win1.rightNavButton = button;
    $.win1.leftNavButton = $.view;
}

$.index.open();
