include any file when needed

Is that a good practice including any file when needed instead of including them at the beginning?

Instead of doing this….

Ti.include(file1.js);
Ti.include(file2.js);
Ti.include(file3.js);

We do something like this.

function getData() {
   Ti.include('file1.js');
}

function getName() {

    Ti.include('file2.js');
}

  • I tried to use second approach, but it didn't work well. It would be great if it worked because you could load resources when they are needed. Other way (the first approach) you need to preload all necessary resources at the beginning which makes application load much slower.
