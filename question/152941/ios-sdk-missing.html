iOS SDK missing

Hello,

I have installed the latest Titanium SDK version 3.1.0.2 and Xcode 4.6 on my Airbook (Apple), I use OS X version 10.8.3.

When I click on 'iOS SDK' in the 'Get Started' page in the 'Configure Native SDK's' I get the error 'An iOS SDK is missing'. How can I resolve this issue?

Thank you so much for your help.

Best regards,
Eric

  • Try to select the Xcode version

    sudo xcode-select -switch /Applications/Xcode.app/Contents/Developer/

  • I have resolved the error. The icon in the applications folder was named differently than the default one, so I used:
    sudo xcode-select -switch /Applications/Xcode46-DP2.app/Contents/Developer/
