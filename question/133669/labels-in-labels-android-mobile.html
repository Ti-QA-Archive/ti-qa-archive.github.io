Labels in Labels [Android] [mobile]

I am currently building for iOS and Android and the code (Very Basic) runs fine on iOS but things are missing on Android

I here is what I was doing:

  • Create "dashboard" view
  • Create Label w/ rounded corners + .8 opacity black background (Background of icon) for each icon
  • Create Label with background image of the icon image (Image is same size as label with rounded corners but mainly transparent with the icon in the center)
  • Create Label with the name of the icon
  • Add both above labels to the background of icon
  • Add label with the other 2 labels inside to the dashboard

Rinse and Repeat for all icons

This works perfectly on iOS yet the two "top" labels do not show up on android. I tested on a clean new mobile project and I seem to be having issues adding labels to labels. This will be a hugh problem if this is an actual limitation on Android…

Thanks for your help!

    The problem may well be due to you using labels as parents to other things.

    Instead of adding labels to labels make the 'parent' label a view instead, as a label is just a specific purpose view.

    For each item;

    • create outer view, including; rounded corners and opacity
    • add imageview to view, including; rounded corners if required
    • add label to view (for caption)

    Much neater and uses the appropriate UI items.

    The other thing to remember is to set width and height when approriate.
