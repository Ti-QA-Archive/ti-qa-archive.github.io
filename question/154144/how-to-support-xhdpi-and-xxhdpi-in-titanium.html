How to support xhdpi and xxhdpi in Titanium?

Can anybody please tell me does titanium support xhdpi, xxhdpi. What is the folder structure to follow the same?

I have set following properties in tiapp.xml:
<manifest>
<uses-sdk
android:minSdkVersion="7" android:targetSdkVersion="9"/>
<supports-screens android:anyDensity="true"/>
</manifest>
<tool-api-level>16</tool-api-level>

I am using Titanium SDK v2.1.4 GA

    Yes, you can add xhdpi and xxhdpi resources as you would do for other resolutions.
    See Appcelerator docs here and Android docs here.

    The basic folder structure is:

    • res-ldpi
    • res-mdpi
    • res-hdpi
    • res-xhdpi
    • res-xxhdpi

    But you can also specify orientation, aspect ratio and screen size, e.g.

    • res-long-land-xhdpi
    • res-long-port-xhdpi
    • res-long-land-xxhdpi
    • res-long-port-xxhdpi
    • res-notlong-land-xhdpi
    • res-notlong-port-xhdpi
    • res-notlong-land-xxhdpi
    • res-notlong-port-xxhdpi

    etc.
