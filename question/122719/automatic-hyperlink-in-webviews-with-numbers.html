Automatic hyperlink in webviews with numbers

Hi,

So i'm using a webview with local html in it. When i'm using numbers in my html, like that :

<p> 1234 - 455 </p>

(you can see i'm NOT using hyperlink with <a></a>)

Hyperlink are automatically created in my webview… And I don't want any hyperlink !

I didn't found anything about that in the Q & A… or in the doc.

    Set to the webview this property:

    webview.autoDetect = [];

  • Hi,

    In iOs, telephone number detection is active by default. Use this meta in the local file.

    <meta name = "format-detection" content = "telephone=no">
