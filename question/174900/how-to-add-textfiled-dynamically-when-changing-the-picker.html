How to add textfiled dynamically when changing the picker

Hi All ,
I am using picker , in this picker I am providing different data.
I am providing one of the data is Others text.
when i select that Others name in picker , i want display one more text field below of the picker .
this can be display only after selecting Others name in picker.

previously i am adding picker to view and button to view , then view to window.
picker and button comes one by one in window ,
but i want add text field in between picker and button when we select Others option of picker.

If any one knows give me the instructions how to achieve above situation for android ?

  • Hi

    This would be as simple as adding the TextField to your view to begin with, and then simply giving it the property of visible: false.

    Then on either the 'change' or 'click' event of the picker when you have a selected value/option, you would simply set the TextFields property to visible: true

    Hope this helps!
