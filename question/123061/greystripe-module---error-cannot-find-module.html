GreyStripe Module - Error Cannot find Module

I am trying to use GreyStripe Module.
I have downloaded the file, dropped the folder in Titanium/modules/iphone
Added below into my tiapp.xml for my project.

<modules>
    <module version="1.0">ti.greystripe</module>
</modules>

tried to call the module as explained in the example app.js,

Titanium.Greystripe = Ti.Greystripe = require("ti.greystripe");

Tried to launch the simulator (ios, sdk 1.7.2)

receive error: "could not find module: ti.greystripe

I see in the console that it is finding the module and loading it.
I have cleaned my build folder, tried to run it under 1.7.1 still receive the same error message.

Any ideas on how to get this working?

  • The module has to be unzipped in

    /Library/Application Support/Titanium/modules/iphone

    where / is your root ( the "Machintosh HDD" if you want)
