  • I am developing a fairly big application using Alloy. I am happy with my decision so far. To me I like alloy because of the following reasons

    1. Clear separation of MVC
    2. Having backbone as a library for working with remote resources (I have one custom sync adapter to talk to our server).
    3. Able to use plain old titanium API where alloy is either not able to support or won't support
