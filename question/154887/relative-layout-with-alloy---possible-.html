    Hi

    It is very simple, just follow the same properties and translate them.

    index.xml

    <Alloy>
    <Window id="win">
        <View class="box" />
        <View class="box" />
        <View class="box" />
        <View class="box" />
    </Window>
</Alloy>

    index.tss

    "Window": {
    layout: 'vertical'
},
"View": {
    backgroundColor: 'orange',
    bottom: 10,
    height: 100,
    left: 10,
    right: 10,
    top: 10,
    width: 100
}

    The layout property tells all child windows to use this layout instead of the default absolute.

    • This should work, by placing the smaller boxes in an outer box, you move the layout to this box and then tell this box to sit at the bottom, the inner boxes should be one on top of each other but at the bottom of the parent - which is the window in this case.

      index.xml

      <Alloy>
    <Window id="win">
        <View id="outer" />
            <View class="box" />
            <View class="box" />
            <View class="box" />
            <View class="box" />
        </View>
    </Window>
</Alloy>

      index.tss

      "Window": {
},
"#outer": {
    bottom: 0,
    height: Ti.UI.SIZE,
    layout: 'vertical',
    width: Ti.UI.FILL
},
"View": {
    backgroundColor: 'orange',
    bottom: 10,
    height: 100,
    left: 10,
    right: 10,
    top: 10,
    width: 100
}
