What events are supported on Ti.Map.View?

I gather from links like this: http://developer.appcelerator.com/question/93331/mapview-click—add-an-annotation-on-selected-point that the 'click' event is not actually supported on the Ti.Map.View event. The documentation is not exactly clear on this, making it seem a lovely variety of events is supported: http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.Map.View.click-event.html. I assume from what I see there that they must actually be supported on annotations only.

I've found a workaround using 'longpress' (which is supported) for now. Are there any plans to expand the set of supported events, or is this an inherent effect of the way the API classes are designed?
