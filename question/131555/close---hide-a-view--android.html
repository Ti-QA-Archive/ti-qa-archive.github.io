Close - hide a view / android

Hi,

I display a view over my current window.
In this view there is an activityIndicator.

However, I want to hide that at some point.
I run this code:

myView.hide();

On the iPhone this code works perfectly on android it does not work. My view is always displayed in the foreground

Thank's for your help
