How to set the date of a picker in Android after it is created?

I have a picker created as type "type: Ti.UI.PICKER_TYPE_DATE" When my window first displays I am fine with the current date showing. After some user interaction I might need to get a date from my data base and set the picker to the DB date.

Setting the property .value at creation seems to work but only at creation. The method setValue says it only works for iPhone and iPad. How do I set the date of a picker on Android after it is created?

Android 2.2 SDK 1.9.X

  • I had the same problem - for some strange reason in Android (works fine on iPhone) the datepicker.value property doesn't get updated when you change the value of the picker.

    I'm using the following workaround for this:

        date_picker.addEventListener('change', function(e) {
        date_picker.value = e.value;
    });

    This change listener just does what you'd expect to happen automatically: update the date_picker value whenever it is changed by the user, and then when you exit the page or the user clicks 'save' or whatever you can just use date_picker.value to get the selected date.

    Hope this saves someone a bit of time!

  • Try adding the view in this code below to a window. The picker will come up with current date. The button will set the value of the picker and display an alert. The alert will show you both the myDate variable and the picker.value are set to 4/29/1995, but the picker control on the UI will remain as the current date. How can I update the visual picker on the view to reflect the current .value???

    function FirstView() {

    var self = Ti.UI.createView();
    var myDate = new Date();
    myDate.setFullYear(1995,3,29);

    var myPicker = Ti.UI.createPicker({
        type: Ti.UI.PICKER_TYPE_DATE,
        height: 'auto',
        width: 'auto',
        top:'10dp'
    });

    var myButton = Ti.UI.createButton({
        title:'Set Date',
        height: 'auto',
        width: 'auto',
        top:'200dp'
    });

    myButton.addEventListener('click', function(e) {
        myPicker.value = myDate;
        alert('myDate = ' +  myDate + 'picker = ' + myPicker.value);
    });

    self.add(myPicker);
    self.add(myButton);

    return self;
}
module.exports = FirstView;
