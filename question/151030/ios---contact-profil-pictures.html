    • Ok.

      Titanium.Contacts.getPersonByID(0).getImage()

      using this you find ti.blob data.

      for display that image you can simple display that image in imageView

      alert(peopleArray[i].image); // this alerts ti.blob

$.imageView1.setImage(peopleArray[i].image); // simply set that **ti.blob** in imageView

      so you can visible that image

      using this you can display that Contact Profil Pictures.

      I hope this helps you.
