Use function in different .js

Hey there,

I'm using the latest Titanium Studio(3.0.1) and developing with a default android SDK of 2.2.

I was wondering if i could create a function in my app.js which can be used in different JS files aswell. Is this possible, and if so: How?

For example:
App.js has a function with shows/throws an error if there is no connect: A label saying: "No connection available". I'd want to put this in a function: noConnection().

If i'm in a different view and js i'd like to call on that same function if there is no connection again.

Can this be done?

Thanks in advance,

    Engwer yes you can do this.You have to use the common js method for this purpose
    Suppose it is Home.js

    function alertfun() {
    alert("I am function in other js file");
}
exports.alertfun = alertfun;

    And acess in other js file like this

    var Home=require("Home");
Home.alertfun();
