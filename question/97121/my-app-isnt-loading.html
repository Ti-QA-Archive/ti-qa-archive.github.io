    CZ

    iAd is only supported on iOS 3.2 and above, so anyone running your app on 3.1 will experience these problems.

    The exception:

    Script Error = Result of expression 'Ti.UI.iOS' [null]

    can be mitigated by checking that Ti.UI.iOS is not null before using it.

    Thanks to Blain for clarifying this for me.

    Cheers
