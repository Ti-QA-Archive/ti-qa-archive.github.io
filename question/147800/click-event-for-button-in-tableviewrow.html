Click event for Button in TableViewRow

I am using mobile SDK 3.0.0.GA. I have a TableView with some TableViewSections. In one of the sections there are TableViewRows which contain Buttons. Here is some sample code for creating the rows and the buttons.

var testButton1 = Ti.UI.createButton({title:"test 1"});
testButton1.addEventListener("click", function() {Ti.API.info("button1 clicked!")});
var testRow1 = Ti.UI.createTableViewRow();
testRow1.add(testButton1);
section.add(testRow1);

var testButton2 = Ti.UI.createButton({title:"test 2"});
testButton2.addEventListener("click", function() {Ti.API.info("button2 clicked!")});
var testRow2 = Ti.UI.createTableViewRow({children:[testButton2]});
section.add(testRow2);

The above sample code misses the TableView and TableViewSection creation but is enough to explain what I see.

Running in the simulator and on my iPhone I get the click event for button 1 but not for button 2. Running in the simulator in DEBUG mode I get both events. The solution is to use the code like in the button 1 sample but I am pretty sure the code for button 2 worked before (maybe in previous SDK versions).

    Hi

    Your second button uses the property children to add the button, but as you will see from the documentation it is read-only so this will not work.

    Add it the same way as you did the first button.
