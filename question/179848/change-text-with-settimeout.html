Change text with setTimeOut

Hi, i'm looking for a function to change a text in a label. I need change automatically text repeatedly.

function CambioTexto() {
    setTimeout(function () {
        $.label.text="On";
     }, 3000);
    setTimeout(function () {
        $.label.text="Off";
     }, 3000);
    setTimeout(function () {
        $.label.text="On";
     }, 3000);
    setTimeout(function () {
        $.label.text="Off";
     }, 3000);
    setTimeout(function () {
        $.label.text="On";
     }, 3000);
    setTimeout(function () {
        $.label.text="Off";
     }, 3000);
    setTimeout(function () {
        $.label.text="On";
     }, 3000);
    setTimeout(function () {
        $.label.text="Off";
     }, 3000);
}

Execute the function, but i only see text "Off".
Where is the problem??

    It is simple: all processes start to the same time and end at the same time. You must chaine it or use setInterval and store the state in a variable.

    function CambioTexto() {
    var cron;    
    var counter = 0;
    cron  = setInterval(function () {
        $.label.text= (counter%2)? "On" : "Off";
    counter++;
    if (counter == 10)  clearInterval(cron);
     }, 3000);
};
