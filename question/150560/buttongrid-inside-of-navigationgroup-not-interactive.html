  • This may be a moot point by now, but I thought I'd give my experience. Maybe it'll help someone else later on.
    As of the Studio 3.2 RC announcement yesterday, NavigationGroup is deprecated. However, I got buttongrid working in a TabGroup on iOS (having issues with Android currently). One note, my grid is not launched from the index controller, but within one of the tabs. Don't know if that makes a difference or not.

    Here's my code. Hope it'll help others.

    index.xml

    <Alloy>
    <TabGroup>
        <Require src="homeView"/>        
        <Require src="venue"/>
        <Require src="cuisine"/>
    </TabGroup>    
</Alloy>

    homeView.js

    function openView(childView) {

    if (childView) {
        controller = Alloy.createController(childView);
        $.homeTab.open(controller.getView());
    }
}

$.homeButtonGrid.init({
    buttons : [{
        id : 'register',
        title : "Register",
        backgroundColor : "#009900",
        backgroundSelectedColor : "#006b00",
        click : function(e) {
            openView('registerView');
        }
    }, {
        id : 'login',
        title : "Login",
        backgroundColor : "#ff0000",
        backgroundSelectedColor : "#990000",
        click : function(e) {
            openView('loginView');
        }
    }],
    buttonWidth : Alloy.isTablet ? 200 : 150,
    buttonHeight : Alloy.isTablet ? 200 : 75
});
$.homeView.open();
