  • Hi

    The short answer is no UNLESS you are using the property window property url, which as you will not from my comment on Mituls answer Titanium is moving away from that quickly towards CommonJS and also Alloy.

    Given the short amount of time to run the simulator, about 10 - 15 seconds in my experience is it really a big problem?

    I am so old (40) I used to write code on paper - debug it and then type it into computers, we have lost that art these days, most likely as the code is MUCH larger. But the need to run and re-run the simulator is reduced by breaking functionality onto simple small CommonJS - test them and move on, then stitch everything together. Makes things much more reliable and faster - both to develop and for the end app.
