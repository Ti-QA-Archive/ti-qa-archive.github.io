  • Good news! This bug has been fixed! Here's the ticket: https://jira.appcelerator.org/browse/TIMOB-13944. It will be available in Titanium SDK 3.1.1.GA.
    You can test drive a 3_1_X continuous integration (CI) build if you'd like to check it out. Note that CI builds are not recommended for production use. You can download it from http://builds.appcelerator.com/#3_1_X or run the following at the command line:

    ti sdk install --branch 3_1_X

    Don't forget to update your <sdk-version> in the tiapp.xml to the 3.1.1.v2012xxxxxxx build that you just installed.
