What exactly IS "mobileweb"

I created a project for Android and iPhone.

Now I'm compiling for Android, I see the folder and message info from the compiler called "MobileWeb" - what is this?

Am I doing something wrong?

    Mobile Web is a beta product that implements the Ti App javascript runtime as a web based project. One that can be served from a Web Server and is targetting mobile devices that have a web browser but are not smartphones.
