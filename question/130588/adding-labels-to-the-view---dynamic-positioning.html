Adding Labels to the View - Dynamic Positioning

Hi

On my iPhone App i am struggling to add labels to a view that don't overlap eachother and all have different top position values

I am able to loop through my Items stored in an Array and create a label for each but the labels all have the same position (top:120) so just overlap. I need to be able to have each label go below the other.

At the start of my app i define the basketItems Array:

basketItems = new Array();

I then have an AddtoBasket Function that gets called that will add the item data into a new Array called item. The item Array then gets added to basketItems with the key itemUnique:

function addToBasket() {

    var item = new Array();
    item["number"] = number variable;      
    item["color"] = color variable;
    item["engine"] = engine variable;
    item["shape"] = shape variable;
    item["seats"] = seats variable;
    item["price"] = price variable;

    var itemUnique = unique variable;

    // Add Item into BasketItems Array
    basketItems[itemUnique] = item;

}

Finally i then need to loop through the basketItems (using for(key in basketItems) { ) and create a label for each item spec as the below example shows for to show the price for each item in basketItems:

for(key in basketItems) {
    var label = Ti.UI.createLabel({top:120, text:basketItems[key]["price"], textAlign:'center'});
    view.add(label);
}

Is there anything i can put in place of top:120 that will mean each label gets positioned further down?

Thank you for any help on this in advance

    Couple of options…

    Use a "vertical" layout and remove the top:120 parameter.

    Or, create a variable 

    var topPosition = 120;

    Then in the loop;

    topPosition += 50;

    and use that as the value of top

    top:topPosition
