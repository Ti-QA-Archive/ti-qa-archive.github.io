Use same Appcelerator cloud service in more than one app

I want to set same appcelerator cloud service in more than one, can this be do-able, as both the app will share the same user.

    Hi,

    You can copy the old projects TiApp.xml API keys from to the new one. It means all the project should share same cloud key. This will share same data to all the user of different apps.

    This keys you should copy:

    
<property name="acs-oauth-secret-production" type="string">aaay71PhrrrHLNOIW</property>
    <property name="acs-oauth-key-production" type="string">aaaM5mrrrJx7Nk</property>
    <property name="acs-api-key-production" type="string">aaapOOxUorrrqMDK</property>
    <property name="acs-oauth-secret-development" type="string">aaaXrrmb3oEhJJUyu4</property>
    <property name="acs-oauth-key-development" type="string">aaaaaaS2535CRnxUbKWW</property>
    <property name="acs-api-key-development" type="string">p2j7pOOxUroqM</property>

    Hope this will help you!
