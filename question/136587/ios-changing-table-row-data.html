iOS changing table row data

Hi guys, I got a simple question .

I have a table , with a few rows , and some of them are checked (with hasCheck proprty).

Im setting the proprty on and off with an event listener that is called when you click the row
(changing e.row.hasCheck ).

I want to write a peace of code that resets all check values of the current table rows
how can i access the rows without an event listener?

  • Try this:

    function uncheck() {
    var sections = table.data; // grab the array of sections
    var section = sections[0]; // If your table has only 1 section.. or else you need to loops through sections

    for(var x = 0, y = section.rowCount; x < y; x++) {
        var theRow = section.rows[x];
        theRow.hasCheck = false;
    }

}
