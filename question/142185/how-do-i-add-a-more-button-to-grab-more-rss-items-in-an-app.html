How do I add a 'more' button to grab more rss items in an app

I am developing an android and iphone app that pulls an rss feed for videos. I am using the latest version of titanium 2.1.2.GA. I would like to know how to list only 15 items, then show a button at the bottom of the feed table that says 'more videos' and then replace those 15 recent items, with 15 more items further down in the list. Does anyone know how to do this?

  • There are examples of this the kitchen sink too. But basically, this is what you can do

    1- On initial load, you will create the window and call a method that will accept the starting index (the starting point, from which you will read the array to load 15 items). Needless to say you can either store the number of items to be loaded as a constant or parameterize that so that you can let user select it at some point. Anyways, this method will read the 15 items, create the table with 16 rows - 15 data rows and 1 row for the 'Load More' action element and return the table object.

    2- Add a click event listener to the table and within the event listener, check the index and if that is the 16th row (e.index==15) then call the method again with the incremented value of starting index, receive the new table object, remove the old table from the window and add the new one..

    Hopefully it makes sense.
