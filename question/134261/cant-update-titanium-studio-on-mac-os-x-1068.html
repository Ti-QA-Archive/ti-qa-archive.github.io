Can't Update Titanium Studio on Mac OS X 10.6.8

Hello,

I'm using Titanium Studio on Mac OS X 10.6.8. It's been several months since I last used Titanium Studio, so when I launched it I went to check for software updates. There were several listed and I did one at a time.

The last available update on the list is this one:

Titanium Studio 1.0.7.201112281834-28122011183948

However, the update always fails with this message:

    OK, never mind, sorry. I went into Preferences –> Install/Update –> Available Software Sites and clicked Reload a couple of times. Then I restarted Titanium Studio and the update worked.
