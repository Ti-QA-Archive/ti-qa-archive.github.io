Swapping between Alloy windows

I just started using titanium the other day, I just ran into a problem swapping between windows. I can do this if the windows are all in the same xml file, but when I have separate xml files with separate controllers I could not get it to work.

index.js

Ti.include("test.js");

$.signin.addEventListener('click', function(e) {
    if($.username.value == 'test') {
        $.second.open();
        $.first.close();
    } else {
        alert('Do not pass')
    }

});

$.first.open();

test.js

Ti.include('index.js');

$.logout.addEventListener('click', function(e) {
    $.second.close();
    $.first.open();
});

$.second.open();

index.xml

<Alloy>
    <Window class="container" id="first" url="../controller/index.js">

        <Label id="username_label">Username: </Label>
        <TextField id="username"></TextField>

        <Label id="password_label">Password: </Label>
        <TextField id="password"></TextField>

        <Button id="signin" action="doClick" title="Sign in" />
    </Window>
</Alloy>

test.xml

<Alloy>    
    <Window class="container" id="second" url="../controller/test.js">
        <Button id="logout" title="Logout"/>
    </Window>
</Alloy>

Can anyone help me out and tell me where I am going wrong?
