Local notifications when application is closed

Hello,

Does anyone know how get userInfo value of local notifications when application is closed from background.

For example I have a this code 

var set = Titanium.UI.createButton ();
var notificationId = 1;

set.addEventListener('click' , function(arg) {
    var notification = Ti.App.iOS.scheduleLocalNotification({
        alertBody: "Hello word",
        alertAction:"Show",
        userInfo: {"notificationId": notificationId},
        date: new Date(new Date().getTime() + 5000)
    });
});

Ti.App.iOS.addEventListener('notification',function(e){
    Ti.API.info('userInfo = '+ e.userInfo.notificationId );
});

I see info when
1) application opened and come local notification.
2) local notification came and you click in "Show" button if application in background.

I need to get userInfo value when application closed from background and local notification came. Anyone know how?
