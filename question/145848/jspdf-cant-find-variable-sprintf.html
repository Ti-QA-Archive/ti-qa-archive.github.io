JsPdf can't find variable sprintf

I am trying to generate PDF's using the JsPDF library, but when i call new jsPDF() i get application error Can&#39;t find variable: sprintf at jspdf.js (line 254)
On that row in jspfd.js i find the following row

out(sprintf('%.2f w', (lineWidth * k)));

sprintf isn't a variable, it's a function. Yet it claims that. What could the problem be?
I've seen that other people's been able to use JsPDF, why isn't this working for me?

Titanium SDK 3.0, iPhone simulator

    If you downloaded the jspdf lib yourself, then you'll see that the sprintf library is a 3rd party lib that jspdf includes in its functionality. Make sure that you have all the scripts that jspdf comes with in order to use the lib itself.
