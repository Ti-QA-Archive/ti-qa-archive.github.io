  • I solved it!

    In case anyone outthere encounter the same problem. Here's how I get it work again:

    1. Run "ti setup" in terminal
    2. select 8) Android Settings
    3. enter the path for SDK and NDK
    4. exit setup when finish

    Then, open the Android configuration again, I can see the drop down option now.
