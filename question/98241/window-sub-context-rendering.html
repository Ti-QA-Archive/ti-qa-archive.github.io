Window sub-context rendering

I am using windows with sub-context feature through out my application. When navigating from window to window, i just use open function and to get back i call the close function. When the new window shows, it starts blank and the controls render one by one as the javascript is parsed. This is really annoying.

Is it possible to render the controls in a window and then show it when the rendering is done?
