'like' button problem

hi, im creating android application about business directory, so im putting 'like' button for each place, so if the user like that place..the click the like button, click it again when they want to dislike. all the likes for each place will be counted,and put it under favourite place

my problem right now is.. everytime the user open the application, the like button can be clicked as like again, eventhough the user already click the button before.. as they click it again, the like will be added again from the same user..

so is there a way to recognise which user that is already click the like button, so that when they click again, it will change to dislike- is there may be some code like recognise the device, so that i can check whether the button is already being click by the user or not

sorry for my bad english, hope u guys understand.. thank you in advance :)

    You can use the Properties API to store some identifier in an array for each place that the user has 'liked'. When you relaunch the app, if the id for the place is in your array, then you can set the button to the correct state.

    function getDetail(){
        var txt;
        if (Ti.App.Properties.hasProperty(idb,"liked")){
            txt="dislike";
        }else if(Ti.App.Properties.hasProperty(idb,"dislike")){
            txt="like";
        }else {
            txt="like";
        }
        return txt;
    }


    lbl_like.addEventListener('click',function(e){

        if (Ti.App.Properties.hasProperty(idb,"liked")){
            Ti.App.Properties.setString(idb,"dislike")
            lbl_like.text="like";
        }else{
            Ti.App.Properties.setString(idb,"liked")
            lbl_like.text="dislike";
        }
