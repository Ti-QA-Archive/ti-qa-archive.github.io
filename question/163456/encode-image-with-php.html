What php function to encode the image, so it can be used in imageView?

    Hi

    This really depends on how you think you will be returning the image.

    • Best solution

      If you are returning a file path from a database or other property then simply provide the path to the image and use that with the image property of the imageView

    • If you really have to

      If you have stored the image in the database on the server you are talking to then return the database of that entry with base 64 encoding. On the client base 64 decode the value before passing as the image property. Although depending on how you stored the image you may need to wrap the call in a blob.

      I would suggest caching the image if it will be used more than once and then re-use that local file entry if it exists rather than going to another external call.
