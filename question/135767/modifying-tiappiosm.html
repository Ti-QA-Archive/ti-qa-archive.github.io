Modifying TiAppiOS.m

Hello all,

I edited the TiAppiOSproxy.m file in the Application Support/…/mobilesdk/…/2.0.1.GA2 directory to recognize minutely and hourly repeat intervals for local notifications. Something along the lines of

  if ([repeat isEqual:@"minutely"])
        {
            localNotif.repeatInterval = NSMinuteCalendarUnit;
        }
        else if ([repeat isEqual:@"hourly"])
        {
            localNotif.repeatInterval = NSHourCalendarUnit;
        }
        if ([repeat isEqual:@"weekly"])
        {
            localNotif.repeatInterval = NSWeekCalendarUnit;
        }
        else if ([repeat isEqual:@"daily"])
        {
            localNotif.repeatInterval = NSDayCalendarUnit;
        }
        else if ([repeat isEqual:@"yearly"])
        {
            localNotif.repeatInterval = NSYearCalendarUnit;
        }
        else if ([repeat isEqual:@"monthly"])
        {
            localNotif.repeatInterval = NSMonthCalendarUnit;
        }
    }

Then I cleaned and rebuilt the project. As far as I can tell my minutely repeat

var notiData = {date:theDate,sound:'aqua.caf',badge:0,repeat:'minutely'};
Ti.App.iOS.scheduleLocalNotification(notiData);

shows up the first notification but it does not repeat. Am I doing something wrong or do I need to rebuild Titanium for the source code changes to take effect.

This is enormously frustrating. I am one step away from finishing this project - but no minutely/hourly repeats of the local notification = no project! I would much appreciate any help
