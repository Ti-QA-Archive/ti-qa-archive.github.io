<html>

            $("#sampleTable").html(sampleTable);    
            Ti.API.log('TEst TEst TEst Test'+ sampleTable);                                
        });    



    </script>
</head> 

<body> 
        <a href='#'>TEST LINK</a>        
        <div id="sampleTable"></div>

As you can see I'm trying to fire the event 'detailWebview'. Now the 'TEST LINK' works with no issues, I'm displaying some details on the log screen so I know its working.

    Hi S. :)

    No need to use JQuery for simple things! In script tag you can simply do this:

    window.onload = function() {
    var links = document.links;

    for (var i = 0; i < links.length; ++i) {
        links[i].addEventListener('click', function(e) {
            Ti.App.fireEvent('detailWebview');
        }, false);
    }
}

    On Titanium side you need to listen to appropriate app-level event.

    Ti.App.addEventListener('detailWebview', function() {
    Ti.API.info('Link clicked!');
});

    Hope this works! :)
