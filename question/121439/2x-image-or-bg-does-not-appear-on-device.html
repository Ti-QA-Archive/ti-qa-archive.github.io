@2x image or bg does not appear on device

This is weird. In simulator everything is fine.
But on the device all local images are missing:

var win4 = Titanium.UI.createWindow({
    url:'blalba.js',
    titleid:'blalba',
            hires:true,
    backgroundImage:'/iphone/winbg.png',
    barColor:'#000'
});

Any idead?

  • The content of the iphone folder replaces root when building for that device, so if your image is in &#x2F;iphone&#x2F;winbg.png in your code you would have &#x2F;winbg.png

  • Don't put the images in the iphone folder. This folder will not be accessible after the build. Move everything ( except the files set by Appcelerator) in other place, like /images/iphone for example.
