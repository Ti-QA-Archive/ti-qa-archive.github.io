  • I don't know what you are trying to do but I access a row's data using the children array, like this:

    <tbl_name>.data[<section_no>].rows[<row_index>.children[<index_of_child_view_added_to_row]

    If your table has only one section then it would be .....data[0]...... For example you can access the name of row 5 like this:

    tbl.data[0].rows[5].children[0].text

    Hope this helps.
