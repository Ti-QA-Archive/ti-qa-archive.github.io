Problem calling SOAP webservice with Authorization

Hi,

I need to call a webservice with authorization… but when I send the xhr.send({xml:soapRequest}) I always get a "500 Internal Server Error".

I compared the request values with a similar request done using SOAPUI and they seem to be similar, even the athorization value is the same.

I have been trying to solve this problem during the last 2 or 3 days but I can't do so much.

I don't know where is the problem.

This is an example of my code:

// Definitions for the HttpClient               
var xhr = Ti.Network.createHTTPClient();                      

xhr.open(operation,url);
xhr.setTimeout(15000);    

authstr = 'Basic ' + Titanium.Utils.base64encode(username + ':' + password);

xhr.setRequestHeader("Authorization",authstr);        

xhr.setRequestHeader('Content-Type',typeContent);
xhr.setRequestHeader("Content-Length", SoapRequest.length);     
xhr.setRequestHeader('SOAPAction',soapAction);    
xhr.send({username : username});
xhr.send({password : password});

xhr.send({xml: SoapRequest});

Thanks!!

  • Thanks to all of you.

    At the end we got it with your help and other partner on the server side. Now we are able to connect remotely the web service and return "on load" status.

    The solution is a little bit diferent and I post it here because it could be useful for someother people. I hope so.

            // Definitions for the HttpClient               
        var xhr = Ti.Network.createHTTPClient();                      

        var authstr = 'Basic ' + Titanium.Utils.base64encode(user+':' + pass);

        xhr.open(operation,url);
        xhr.setRequestHeader("Authorization",authstr);        
        xhr.setRequestHeader("Content-Type",typeContent);
        xhr.setRequestHeader("Content-Length", SoapRequest.length);     
        xhr.setRequestHeader("Soapaction",soapAction);    

        xhr.setUsername(user);
        xhr.setPassword(pass);

        xhr.send(SoapRequest);

        xhr.setTimeout(15000);

    Maybe the key is: typeContent is 'text/xml;charset=UTF-8'.
    So I only have to put the soapRequest on the xhr.send sentence.

    It seems to work fine, now I have to check the document that it returns.
