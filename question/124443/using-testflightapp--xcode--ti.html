Using TestFlightApp + Xcode + Ti.

Here is what I've had success with:

1.build/run project on IOS Device from within Titanium.

2.open Xcode and open <yourappname>.xcodeproj file under build directory created by step #1.

3.Build for IOS device using Xcode. I don't turn any parameters on/off or change anything, just build. There is a note about setting a Can Be Debugged parameter to NO but I haven't bothered to do it.

4.In Xcode, select Project then Archive.

5.Under Organizer/Archive, Share using team development profile. Mine is managed by Xcode.

You have to make sure all of your testers are included as devices in Team Development Profile. TestFlightApp helps with this and
allows you to select who is eligible to receive the beta.

Hopes this helps someone.
