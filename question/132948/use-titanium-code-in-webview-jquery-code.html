  • You can use whatever js libs in webview since weview is a HTML5 based view.

    To communicate between html code in webview and JS code:

    • From JS code To html code in webview: webview.evalJS(&#39;some js statements&#39;) to call some js code ( can be your custom js code ).

    • From html code To JS code: Ti.App.fireEvent

    • Ti.Network namespace is not available in html/webview context.

      Try this:

      HTML Code

      <script type='text/javascript'>

        $(document).ready(function() {

        Ti.App.fireEvent( 'fetch', {url: 'http://somewebsite.com/all.json'} );

    }

</script>

      Titanium JS Code*

      Ti.App.addEventListener('fetch', function(e) {
    alert( 'Called from HTML Code' );

    var loader = Ti.Network.createHTTPClient();
    loader.setTimeout(10000);

    loader.onload = function() {
        var json = JSON.parse(this.responseText);
        alert(json.length);
    }
    loader.open("GET", e.url);
    loader.send();

    // alert(loader.length);
});
