How to create python plugin for mobile titanium?

I want to create plugin for Titanium Mobile project, which will be when the application runs on the Apple simulator to create text file with launch date.

  1. Create new project in Titanium Mobile

  2. In the project folder to create plugins/ti.time/1.0/plugin.py

import time
start = time.time()
open("time.txt", 'w').write("%s" % start)
  1. is added to the tiapp.xml
<plugins>
    <plugin version="1.0">ti.time</plugin>
</plugins>
  1. Run the project in the simulator, file plugins/ti.time/1.0/time.txt is not created

If I select file plugins/ti.time/1.0/plugin.py in AppExplorer window and Run Python, file plugins/ti.time/1.0/time.txt is created

How to get the plugin to run with the launch of the emulator?
