TiDOMAttr Properties

Hi

Trying to evaluate an xpath expression by using the Ti.XML.parseString.

    var xml;
    function ReadXml()
    {
        var f = Titanium.Filesystem.getFile("config.xml");
        var textString = f.read().text;
        xml = Ti.XML.parseString(textString).documentElement;    
    }    
    function GetAttributes()
    {
        if(xml == null)
        {
            ReadXml();
        }
        if (xml.evaluate) {
            var val = xml.evaluate("//Configuration/tabGroup/tab/@id");

            alert(val.item(0));

        }
    }

The alert(val.item(0) gives me the TiDOMAttr object. How to get the value of the same. And what properties and methods are available for the TiDomAttr object.

Sample xml provided below

<?xml version="1.0" encoding="UTF-8"?>
<Configuration>
    <tabGroup barColor="#336699">
        <tab id="1">
            <Window Title="Main Window" backColor="#000">

            </Window>
        </tab>
                <tab id="2">
            <Window Title="Second Window" backColor="#336699">

            </Window>
        </tab>
    </tabGroup>
</Configuration>
