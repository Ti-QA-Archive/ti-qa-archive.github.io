Possible to sort rows in a tableView?

I have a table view that gets built with rows that have a distance property (a number). That property doesn't get set until after the tableView has been instantiated (due to some asynchronous work going on). Is it possible to sort the rows after they're part of a tableView or am I perhaps just not doing it right? It's been a long day, so I'm really hoping that the latter is the case.

table.data[0].rows.sort( function( row1, row2 ) {
  return row1.distance - row2.distance;
});

With that code, nothing happens. The ordering never gets changed.

Thanks.

    What I did on my app is to sort the data object (it is an array after all) and build the table view again. I don't know if that's the right way but I couldn't find any other way…

    • That's essentially what I ended up doing:

      var data = table.data[0].rows; // table references a tableView

data.sort( function( row1, row2 ) {
  return row1.distance - row2.distance;
});

table.setData( data );
