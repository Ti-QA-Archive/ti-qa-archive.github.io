    Hi

    You have not said what you want to do with the database other than open it. To use a database that you have already created you need to install it first;

    var db = Ti.Database.install('dbfilename', 'installedname');

    If you have already installed the database you can open it up later using;

    var db = Ti.Database.open('installedname');

    installedname = the reference name you used in the install line previously.

    All the database code can be found in the documentation;

    • Ti.Database
    • db install
    • db open
    • Ti.Database.ResultSet
