create and modify anonymous labels

Hey guys,

This is probably simple but I don't know how to do it.

I need to create X amount of labels, right now it's a simply array but I want to use a database in the future.

I loop through the array and add each label to a view.

Much simplified, it looks like this:

for (var i=0; i < allLabels.length; i++) {
    var title = allLabels[i].title;
    var positionTop = 50 * i;
    var singleLabel = Titanium.UI.createLabel({
        text: title,
        top: positionTop,
    });
    view.add(singleLabel);
}

It's working nicely but I have to modify the labels afterwards so it would be nice to have an own object for each label.

How can I do so?

    create an array object and associate it to the view, store your labels there for retrieval later

    view.myLabels = [];
for (var i=0; i < allLabels.length; i++) {
    var title = allLabels[i].title;
    var positionTop = 50 * i;
    var singleLabel = Titanium.UI.createLabel({
        text: title,
        top: positionTop,
    });
    view.add(singleLabel);
    view.myLabels[i] = singleLabel;
}

    • Thanks, I tried it before but either the emulator crashed or the array was empty.

      This one works:

      var myLabels = [];
for (var i=0; i < allLabels.length; i++) {
    var title = allLabels[i].title;
    var positionTop = 50 * i;
    var singleLabel = Titanium.UI.createLabel({
        text: title,
        top: positionTop,
        width: 'auto'
    });
    myLabels.push(singleLabel);
    view.add(singleLabel);
}
