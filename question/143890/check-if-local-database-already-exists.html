Check if local database already exists

Hi, I want to create a sqlite database programatically. I don't want to execute every time on startup the sql-statement to create the tables. How can I check if the database already exists? It should work on Android and on iOS

    var fname=Ti.Filesystem.getFile("file://data/data/"+ Ti.App.getId()+"/databases/"+ Nameyoudatabase);
    alert(fname.nativePath+":"+ fname.size);
    If is -1 o 0 doesn't exist
    What you get is the size
    That is for android
