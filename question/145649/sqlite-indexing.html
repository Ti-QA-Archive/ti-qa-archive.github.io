SQLITE Indexing

'CREATE TABLE IF NOT EXISTS ' + user_favourites_table_name + ' (favourite_id INTEGER PRIMARY KEY, place_id INT,  timestamp_added INT);'

How do I add an index to the timestamp_added? Following the SQLite site, http://www.sqlite.org/lang_createindex.html, I produced the following additional query:

'CREATE INDEX IF NOT EXISTS timestamp_added_index ON ' + user_favourites_table_name + ' "timestamp_added"';

But it failed.. can't see how to add index's in the appcellerator docs either…

    Hi

    It may be your quoted last parameter - the syntax should be;

    CREATE INDEX uniqueindexname ON tablename (fieldname);

    So in your case;

    sql = 'CREATE INDEX timestamp_added_index ON ' + user_favourites_table_name + ' (timestamp_added);';

    or with the optional exists 

    sql = 'CREATE INDEX IF NOT EXISTS timestamp_added_index ON ' + user_favourites_table_name + ' (timestamp_added);';
