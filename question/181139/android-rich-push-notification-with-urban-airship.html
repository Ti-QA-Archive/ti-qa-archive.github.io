Android Rich Push Notification with Urban Airship

Hi, I noticed that the Ti.UrbanAirship module only supports Rich Push notifications with iOS. This is because they did not support android at the time of creation. I see now though that Urban Airship does have Rich Push notifications for Android. http://docs.urbanairship.com/topic-guides/android-rp-tutorial.html

How would we go about setting this up using Titanium's Javascript? Unfortunately their example code is native and isn't going to help us Titanium devs much. Here is the Urban Airship module for reference: https://marketplace.appcelerator.com/apps/4984#!overview

Thank you in advance!
