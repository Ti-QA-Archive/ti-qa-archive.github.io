    • Well, from what I read and understand, you should simply pass the values as dynamic properties.

      Here's how you could do it:

      When creating your new window, simply add your values using custom properties.

      In your fav.js file:

      // In your button click
var win2 = Titanium.UI.createWindow({
    url: e.rowData.test,
    title: e.rowData.title,
    description: description,
    secteur: secteur
    // you can pass every value you desire using this way
});

      Once you are in your other file, simply retrieve those custom properties as you would with any other variable. (remember that JavaScript allows you to dynamically add properties to objects, which is a very powerful feature)

      In your descriptionfav.js file:

      var win = Titanium.UI.currentWindow;
// The current window (that was just created in the other file) 
// have all the properties you defined.
Ti.API.info(win.title);
Ti.API.info(win.description);
Ti.API.info(win.secteur);

      There is absolutely no difference between the windows native properties (title) and the custom you created (description, secteur, etc.).
