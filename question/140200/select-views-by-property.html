Select views by property

Hi,

how can I select some group of views by some of their properties? For exemple:

 var optionCheck = Ti.UI.createView({
            width: sz(26),
            height: sz(26),
            backgroundImage : '../resources/routes/check_disabled.png',
            left:sz(25),
            top: sz(5),
            isEnabled : false,
            idQuestion : results[i].ID,
            idOption : j
        });

Imagine that I've created 10 optionsChecks: how can I hide the optionChecks which it's idQuestion = 1?

    Well, for that, you will have to create an array and push these same optionChecks into this array.

    Now, when you want to check their status (hide them or not), you simply loop through the array.

    ####Here is what the code would look like (more or less):

    for (var i = 0; i < arrayOfChecks.length; i++) {
    var optCheck = arrayOfChecks[i];

    if (optCheck.idQuestion == 1) {
        optCheck.hide();
    }
}

    Hope this helps
