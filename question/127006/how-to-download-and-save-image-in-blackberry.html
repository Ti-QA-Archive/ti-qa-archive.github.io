How to download and save image in Blackberry

Hi!

How can i download and save an image file from the internet to a file in my file system in the Blackberry Titanium sdk?

I've seen the download example from Kitchen sink but i don't see any data downloading, only a text saying it's downloading data:

httpClient.ondatastream = function(e) {
    var text = 'Progress: ' + (e.progress*100) + '%';
    Ti.API.debug(text);
    label.text = text;
}

How do i get the downloaded data?

Thanks.

    The blackberry KitchenSink example is incomplete, but you use the same approach for downloading a file that you do with iOS or Android.

    Ti.API.debug("downloading " + url);
var http = Ti.Network.createHTTPClient();
http.open("GET", url);

http.onload = function() {
   var f = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory, _path);
   f.write(http.responseData);
   Ti.API.debug("downloaded message " + f.nativePath);
};
http.onerror = function(e) {
   Ti.API.debug("ERROR: downloading file " + JSON.stringify(e));
};
http.ondatastream = function() {
};

http.send();
