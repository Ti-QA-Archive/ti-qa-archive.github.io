Text cropped with high res device !

Hello

I'm create an app for android. With "normal" device (galaxy note 2), the text appear correctly but with high res device (nexus 4, nexus 7) the text look little cropped.
I take this screenshot from a nexus 4 and we can see the text at the end of the label is cropped.! Image with text cropped

My code look good :


    var wb = Titanium.UI.createScrollView({
        contentWidth:'auto',
        contentHeight:'auto',
        top:50,
        showVerticalScrollIndicator:true,
        showHorizontalScrollIndicator:true
    });

    var wtext = Titanium.UI.createView({
        width:'auto',
        left:2,
        right:10
    });
wb.add(wtext);

    var wc = Ti.UI.createLabel({
        text:rowdata.definition + '\u000A\u000A',
        font:{fontFamily:'Roboto-Regular',fontSize:15},
        color:'#F2F2F2',
        top:0
    });
wtext.add(wc);

What's is wrong ? How i can show the text correctly ?

Thnaks, regards.

PS : Sorry for my bad english !

Platform android, Titanium SDK 3.0.0 GA

  • You could try using device independent font sizing by using 'sp'. See this link

    var wc = Ti.UI.createLabel({
        text:rowdata.definition + '\u000A\u000A',
        font:{fontFamily:'Roboto-Regular',fontSize:'15sp'},
        color:'#F2F2F2',
        top:0
    });
