    • Prasanna

      The following code:

      Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory + "/config.xml")

      should be:

      Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory, "config.xml")

      If you are still having issues, clear your build directories and force a full rebuild.

      Cheers
