  • My understanding is that for ios the app name is defined in iTunes Connect - which checks to see if that App Name is already in use (you may want to check this before you worry to much about how it looks in the simulator/dev device). Therefore there is no possible way to define a name for each platform.

    We have found that for long names its best to join them together: MarketRoundup
