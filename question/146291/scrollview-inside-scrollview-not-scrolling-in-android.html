Scrollview inside scrollview not scrolling in android

Hi all.

I've got a window just like whatsapp chat window is.

To scroll everything up except the top banner when the soft keyboard comes out i need to use a scrollview in android as a base for chat and textfield, but then, i can't scroll the scrollview that has the chat (only the base scrollview scrolls). How can i achieve to scroll only the topmost scrollview (chat one) when touching it?

Thanks in advance for any hint.

(Using latest sdk on android)
