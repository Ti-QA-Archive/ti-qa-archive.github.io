  • <receiver android:name="com.appcelerator.aps.IntentReceiver"/>

    Needs to be in you tiapp.xml or AndroidManifest.xml

    Older versions is still triggering because you probably have the old IntentReceiver in your manifest.

    the older version is :

    <receiver android:name="ti.cloudpush.IntentReceiver"/>

    Just replace this with the one I suggested above.
