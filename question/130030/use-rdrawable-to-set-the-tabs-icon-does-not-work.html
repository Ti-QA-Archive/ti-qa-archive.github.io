  • Chi, the R.drawable images are built into the Android operating system. You do not have to put pngs for them anywhere. Are you sure that the "popular_title" drawable is present in the version of the operating system that you are running on?

  • Hi chi xuan

    Please make one folder name images in the resource folder then put your images in this.

    now you make change in your code as below.

    var tab1 = Titanium.UI.createTab({
    icon : "../images/popular.png",// this will be you image path
    // title : 'Popular',
    window : win_popular
});
