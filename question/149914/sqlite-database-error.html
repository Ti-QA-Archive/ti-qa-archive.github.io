    Hi

    Looks like you need to confirm the file is actually readable BEFORE you use it in the app.

    Download the Firefox web browser plugin for your desktop computer called SQLite Manager

    Open the database in the plug-in and see if you can query it.

    • If you can, the problem is in the code
    • If it is unreadable, the problem is the database

    Report back.
