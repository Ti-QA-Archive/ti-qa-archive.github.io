How to get rid of silly (unwanted) warnings in Titanium Studio

Those little triangle exclamation point ! warnings in the margin of the Titanium Studio editor can be helpful, but I was getting dozens of "false" warnings for things that were complete valid. Such as using a call to "alert()", which shows up as undefined. Using "require" or "exports" for CommonJS is valid, too, but the editor complains over and over.

I found the fix, though, which makes the remaining warnings actually very useful. To get rid of warnings you don't want to see, do this:

  • Click Titanium Studio in the menu at the top.
  • Click Preferences…
  • Click Titanium Studio / Validation / JavaScript
  • In the Filter box at the bottom right, click the [+] button to add a filter.
  • Create 3 filters for the following items:
  1. 'alert' is not defined.
  2. 'exports' is not defined.
  3. 'require' is not defined.
  • Click Apply

Basically, you can create a filter for any warning message you don't want to see.
