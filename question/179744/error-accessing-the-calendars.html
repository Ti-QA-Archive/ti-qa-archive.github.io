Error accessing the calendars

I want to list all the calendars of the iOS- device in a tableView. But accessing the calendar I get the following error:
[ERROR] : getting all calendars: Error Domain=EKCADErrorDomain Code=1013 &quot;The operation couldn’t be completed. (EKCADErrorDomain error 1013.)&quot;
I was never asked for permissions for the access to the calendar.

I use the following code:

    var calendars = Ti.Calendar.allCalendars;

    var tblViewRows = [];
    for (var i = 0; i < calendars.length; i++) {
        var tblViewRow = Ti.UI.createTableViewRow({
            title : calendars[i].name
        });
        tblViewRows.push(tblViewRow);
    }
    $.tblView.data = tblViewRows;

I use the SKD-Version 3.4.1., iOS 8.1, the error comes up at the device and at the simulator.

What can I do to fix this error?

    You need to request permissions

    Ti.Calendar.requestEventsAuthorization(function(e) {

    if (e.success) {
        var calendars = Ti.Calendar.allCalendars;

        alert(calendars.length);
    }

});
