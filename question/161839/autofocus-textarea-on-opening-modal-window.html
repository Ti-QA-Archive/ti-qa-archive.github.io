Autofocus textarea on opening modal window

I have a simple modal window with a textarea that pops up similar to composing an email in the iOS Mail app or writing a status in the Facebook app. I just want the textarea to be autofocused when the window is displayed. Any search I do on this issue only returns people trying to disable autofocus on Android.

I can't see any properties of TextArea to set autofocus and putting $.myTextArea.focus(); in the JS file does nothing.

    The $.window.on() notation is deprecated and should not be used. Try

    $.window.addEventListener('open', function() {
  $.myTextArea.focus();
});
