Alloy data binding, concatenate attributes in label

Hi,
I have a model that contains the address, city, zip code and more things.

In a tableView i can display one property with this code: 

<Label text="{address}" />

I want to concatenate the city and zip code in one label. I've tried the below code but it does not work

<Label text="{zip}  {city}" />

Result: {zip} {city}

I found an example in the documentation at the point 4. 

<View layout="vertical">
    <ImageView image="{cover}" />
    <Label text="{title} by {artist}" />
</View>

The concatenation is possible with the alloy data binding ?

    I'm afraid our documentation is slightly ahead of our releases. The feature you're seeing in the docs is included in Alloy 1.5. The release candidate is available, but the final version has not yet been shipped. Sorry about that!
