Open Email App in Android

I want to open email app from my Titanium app. I know this is posible in Titanium using Android Intent.

        var intent = Ti.Android.createIntent({
            action: Ti.Android.ACTION_SEND,
            type: "plain/text"
        });

    Ti.Android.currentActivity.startActivity(intent);

The above code will open a email composer. Actually just want to launch the mail app not email composer. Is it possible?

  • Hey Prabhu,

    The Following code works to open gmail inbox in Android. Consider that if google update the given class name to some other this may not work..

    

    var intent = Ti.Android.createIntent({
        className : "com.google.android.gm.ConversationListActivityGmail",
        packageName : "com.google.android.gm"
    });

    var activity = Ti.Android.currentActivity;
    activity.startActivity(intent);

    But as Jason mentioned there is no reliable way to do this…!!
