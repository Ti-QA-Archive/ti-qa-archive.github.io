JSON loop problem

Hello,

I hope somebody is able to help me, while I got JSON working in my project as I wanted, I have one problem with a specific format:

{
     "firstName": "John",
     "lastName": "Smith",
     "age": 25,
     "address":
     {
         "streetAddress": "21 2nd Street",
         "city": "New York",
         "state": "NY",
         "postalCode": "10021"
     },
     "phoneNumber":
     [
         {
           "type": "home",
           "number": "212 555-1234"
         },
         {
           "type": "fax",
           "number": "646 555-4567"
         }
     ]
 }

This is from wikipedia, by the way. So, I can read everything out of the JSON object, as long as it has one entry, but as you see this example has more than one phonenumber, I am not able to get to those. Maybe someone can show me a code snippet?

Thanks!

    If you just want to get the phone number you need to do something like this:

    for (var i = 0; i < json.phoneNumber.length; i++){
    Ti.API.info(json.phoneNumber[i].type);
    Ti.API.info(json.phoneNumber[i].number);
}
