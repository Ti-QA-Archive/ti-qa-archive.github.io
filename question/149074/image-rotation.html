image rotation

I am using the below code, to upload image from gallery or camera and i will rotate that image and i will
store that in DB in the same rotated position

var img_view = Ti.UI.createView({width:380, height:380,});           

            var img = Ti.UI.createImageView({

                             transform: Ti.UI.create2DMatrix().rotate(30)
            });            

            img_view.add(img);


       var image = img_view.toImage().media;

  • Hi

    The problem you are coming up against is a common one and it is because of the way transformations affect things.

    When you run a transform, it changes the visual appearance only and not the underlying view. Which why when you upload you get the original.

    Good news though a simple tweak usually solves the problem.

    • place the view you wish rotate inside another view parent
    • rotate the view as you did before
    • run the toBlob or toImage on the parent view not the rotated view

    Should solve it for you.
