Titanium.UI.createWindow:

it states:

> If this function is called outside of the context of a window it will create a new top-level window.

How can I do this? It seems all ability to access the Titanium object is within a webkit window… am I missing awesome pre-webkit place to execute code?

Can use Python, would prefer to stay in Javascript only if possible.

Because if that module is closed by the user, it's going to try to exit the app, which I could catch and prompt the user (or somehow prevent it), but then I'm going to be dealing with unnecessary if-then-else logic on when to actually let the user exit the app.
