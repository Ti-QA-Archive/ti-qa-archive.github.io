Using titanium-jasmine with Titanium Studio

I've started a project, and would like to include unit tests. After having a look at various options here, the most promising seems to be titanium-jasmine.

However, most of the examples I see for including titanium-jasmine into projects seem to need to be run via a make call. Is there anyway to use Titanium Studio Run Configurations to call Test / Real versions of the application when running Titanium Studio?

If you think the answer is too simple, think again. I've VERY new to Titanium / Jasmine / etc. Explanations in words of one syllable or fewer will do nicely. :-)
