How to detect no GPS signal?

Hello all

if I'm in a building, is that even possible I will not get a gps signal at all from the location callback? right now I'm in a building but I'm still getting a "location" event, with high accuracy (between 600 and 2000).

Is there a way to check if there's a signal at all?

  • You can try explicitly setting the preferred provider for your location events to GPS. If you're not getting those events, then you know you aren't getting a signal.

    // set your preferred provider
Titanium.Geolocation.preferredProvider = Titanium.Geolocation.PROVIDER_GPS;

// if this doesn't fire, you don't have a GPS signal
Titanium.Geolocation.addEventListener('location', function(e) {
    // handle location 
});
