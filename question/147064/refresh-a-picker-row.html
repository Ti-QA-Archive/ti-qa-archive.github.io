  • simplest way to do is, on any row selection in Picker A, in its event Listener change and click, set the selected row in picker B

    example method.

    
pickerB.setSelectedRow(column:selectedColumn, row:selectedRow);

    reference doc :: http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.Picker-method-setSelectedRow
