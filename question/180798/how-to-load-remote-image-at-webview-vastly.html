  • Hi Hussein Elganzory,

    Thanks for your concern .

    To Load Remote Image at Webview vastly you can try this example code.

    var xhr = Ti.Network.createHTTPClient();
xhr.onload = function() {
    var f = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory, 'browse.html');
    var htmlFile = xhr.responseText;

    f.write(htmlFile);
    var webview = Ti.UI.createWebView({
        data : f.read(),
    });

}
xhr.open("GET", "http://remote image url ");
xhr.send();

    You can use url too, but in this case need the correct path, to applicationdatadirectory.
