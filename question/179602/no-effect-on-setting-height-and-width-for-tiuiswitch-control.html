No effect on setting height and width for Ti.UI.Switch control

I created a new alloy project (1.5.1) and added a switch control. Why does the height and width have no effect on the switch no matter what size i give?. It just sticks to a definite size. Trying to do this on the iOS.

index.xml

<Alloy>
    <Window class="container">
        <Label id="label" onClick="doClick">Hello, World</Label>
        <Switch value="false" top='100' height="100" width="200"></Switch>
    </Window>
</Alloy>

  • Accepted Answer

    The Ti.UI.Switch component simply exposes the native equivalent UI component. Those components are of a pre-defined size. Setting height/width on them will have no effect. For example, in the Apple API reference for the switch component there is no height property. If you must have something larger, you'll need to create a custom UI component that implements the needed functionality.
