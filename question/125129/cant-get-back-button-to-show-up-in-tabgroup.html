Can't get "back" button to show up in tabgroup

Hi,

It is my understanding that if you create a tabgroup then when you click through the windows the 'back' button will show automatically. (Maybe I am wrong).

I have 3 tabs - One main tab, and two others that I would like to go "back" to the main tab.

In app.js I have:

var tabGroup = gg.ui.createApplicationTabGroup();
tabGroup.open();

Then in my ui.js file I have:

// create tab group
    gg.ui.createApplicationTabGroup = function() {
        var tabGroup = Titanium.UI.createTabGroup();
        var homeWin = gg.ui.createHomeWindow();
        var faqWin = gg.ui.createFAQWindow();
        var formWin = gg.ui.createFormWindow();



        gg.mainTab = Titanium.UI.createTab({
          title: L('home'),
          window: homeWin
        });

        gg.faqTab = Titanium.UI.createTab({
          title: L('FAQs'),
          window: faqWin
        });

        gg.formTab = Titanium.UI.createTab({
          title: L('form'),
          window: formWin
        });
        tabGroup.addTab(gg.mainTab);
        tabGroup.addTab(gg.faqTab);
        tabGroup.addTab(gg.formTab);
        return tabGroup;
    };

And an example of a window would be:

gg.ui.createHomeWindow = function() {
        var win = Titanium.UI.createWindow({
            title:'home',
            layout:'vertical',
            backgroundColor:'black',
            backgroundImage:'iphone/Default-Portrait.png',
            tabBarHidden:true,
            barColor:'#5ea451'
            });

        win.add(gg.ui.createFAQText());
        win.add(gg.ui.createFormButton());
        win.add(gg.ui.createFAQButton());

        return win;
    };

Can you please help clarify what I am doing wrong?

Thanks for any help!

    Back buttons are not attached to tab group windows. There is no need for them as the tabs are visible allowing you to switch windows.

    A navigation group on iOS, however, is different and as you decend down into subpages, a button appears to navigate back up and out of the subpages.

    • But, if you really wanted them, I suppose you could set a left nav button that changed the tabs for you.

      Example, adjust as needed:

      var b = Titanium.UI.createButton({
                title:L('back'),
                style:Titanium.UI.iPhone.SystemButtonStyle.PLAIN
            });
            w.setLeftNavButton(b);
            b.addEventListener('click',function()
            {
                tabGroup.setActiveTab(tabGroup.tabs[1])
            });
