DataGrid Control in the Titanium / Alloy Framework

Has anyone had experience building a datagrid control in the Titanium / Alloy framework? Ideally, I am looking for something that works akin to the Fidelity iPhone/iPad application, that simply:

  1. allows for multiple columns
  2. allows a user to freeze/lock the first column while horizontally scroll the remaining columns
  3. vertically scrolls to show data rows
  4. allows for a sort when clicking column headers
  5. refreshes on pull down (akin to a typical TableView refresh)

This seems like a control I would believe other projects have tackled in the past, and just wanted some pointers on how people tackled it. I would love to utilize the Alloy databinding, but, not sure if this would even be possible, since a datagrid does not exist as a native Alloy control.

Any advice, examples of people that have done Fidelity style datagrids would be great.
