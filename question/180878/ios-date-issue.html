  • In here, it shows the correct date if i run it.

    Maybe you can try and use "moment", which is build-in in Titanium Alloy, and see what happens. It's anyhow a more easy way to work with dates (in my opinion).

    var moment = require('alloy/moment');
// your local timezone
Ti.API.warn( moment("1436/4/26", "YYYY/M/DD").format('YYYY/M/DD') );
// UTC timezone (+0000)
Ti.API.warn( moment.utc("1436/4/26", "YYYY/M/DD").format('YYYY/M/DD') );
