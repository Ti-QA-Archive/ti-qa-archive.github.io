createAlertDialog does not appear

I have an event listener that when triggers, it should open an alert dialog.

With this code, the alert DOES appears

            var opts = {
            cancel: 1,
            buttonNames: ['Confirm', 'Cancel', 'Help'],
            message: 'Would you like to delete the file?',
            title: 'Delete'
            };
            Ti.UI.createAlertDialog(opts).show();

With this code, the alert DOES NOT appear

          var dialog = Ti.UI.createAlertDialog({
            cancel: 1,
            buttonNames: ['Confirm', 'Cancel', 'Help'],
            message: 'Would you like to delete the file?',
            title: 'Delete'
          });
          dialog.show();

I need the second method in order to add event listeners on the options (not part of the code above).

SDK 3.3.0
