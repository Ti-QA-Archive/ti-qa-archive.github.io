Custom fonts and alloy textid or titleid text

I didn't find anything about this in Q&A, but when we have a custom font and pretend use a string from the internationalization files, don't works. If we set the string by text or title property directly (or set methods), works.

Titanium SDK version 3.1.2.GA
IOS: 5.0, 5.1 and 6.0 tested.
Device: simulator and device.

  • Hi,

    you need to create one folder inside app -> assets folder then you need to set font name in plist so for that add the below tag in your tiapp.xml file.

    <ios>
        <plist>
            <dict>
                <key>UIAppFonts</key>
                <array>
                    <string>fonts/(set font file name with extention here )</string>
                </array>
            </dict>
        </plist>
    </ios>

    then clean build your project and also delete app from simulator or from device and then test it.
