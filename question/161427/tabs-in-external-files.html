Tabs in external files

Hi all, the tab organization by external file is a good practise?

var tabGroup = Titanium.UI.createTabGroup();

var win1 = Ti.UI.createWindow({  
    title:'win1',
    backgroundColor: '#FFFFFF',
    tbGrp: tabGroup,
    url:'win1.js'   
});

// create section tab.
var tab1 = Titanium.UI.createTab({         
    title:'win1',
    icon: 'images/tabgroup_home.png',
    window:win1
});


tabGroup.addTab(tab1);

In my simulator iOS simulator I see: [WARN] : OPEN ABORTED. _handleOpen returned NO

  • The answer is do not code your Titanium App like this. Using the url property to manage your window stack is considered very bad practise for a number of reasons (spawning multiple contexts is one of them). Instead, the best practise is to adopt the CommonJS style of coding
