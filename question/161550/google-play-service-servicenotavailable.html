  • Hi,
    Make sure your application is not using MQTT protocol. As of Release 3.2.0, the CloudPush module only supports Google Cloud Messaging (GCM) for push notifications. MQTT support is deprecated and support was removed in Release 3.2.0. Upgrade your application to use the GCM protocol if not done already.

    Regards,
    Rupesh
