    • Actually, it is missing. Ti.Media.showCamera is used to show the camera.

      Ti.Media.camera is used to determine which camera is showing. You can see it in use in the Kitchen Sink: https://github.com/appcelerator/KitchenSink/blob/master/Resources/ui/handheld/ios/phone/camera_video.js#L87.

      — commented March 2nd 2013 by John C. Bland II

    • Exactly Aaron. Referring to the guide page should not be necessary as it is a property on Ti.Media.

      — commented March 2nd 2013 by John C. Bland II

    • Here is a usage example;

      Ti.Media.showCamera({
    success: function (e) {
        if (e.mediaType === Ti.Media.MEDIA_TYPE_PHOTO) {
            var img = Ti.UI.createImage({
                height: Ti.UI.SIZE,
                image: e.media,
                width: Ti.UI.SIZE
            });
            win.add(img);
        }
    },
    cancel: function () {
        alert('Cancelled');
    },
    error: function (error) {
        if (error.code === Ti.Media.NO_CAMERA) {
            alert('Not support on simulator');
        }
    },
    saveToPhotoGallery: false,
    allowEditing: false,
    mediaTypes: [
        Ti.Media.MEDIA_TYPE_PHOTO
    ]
});
