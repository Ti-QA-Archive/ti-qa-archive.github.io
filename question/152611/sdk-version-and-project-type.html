  • Hello,

    you should go to mobile project, if you want to do anything mobile. General project is, as the name suggest, a generic programming project, nothing to do with mobile app.

    Why are you using such old SDK? You should be using 3.1.0.GA in new projects (if you have old projects, the ideal would be to move them to newer versions of the SDK, but that likely will require some work).
