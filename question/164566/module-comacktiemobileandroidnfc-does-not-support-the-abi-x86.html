module com.acktie.mobile.android.nfc does not support the ABI: x86

Titanium SDK version 3.3.3.GA
Platform Android 4.1.2
Device: GT-I9105P
Host SO OSX 10.9.2
This is a warning I get building my Android app:
"The module com.acktie.mobile.android.nfc does not support the ABI: x86
It only supports the following ABIs: armeabi, armeabi-v7a
Your application will most likely encounter issues"
In fact, the compilation returns an error:
"TiExceptionHandler: (main) [1,246] - Message: Uncaught Error: Requested module not found: ti.nfc"
Is there any new ti.nfc module compatible with x86?
