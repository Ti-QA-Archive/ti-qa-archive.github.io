Ti.UI.iPhone.createNavigationGroup question?

Hi All,

Can I attach this function to the Current Window?

I see a lot of examples of its usage but i just cant seem to get it running correctly.

From my app.js I create a tabgroup which opens the window (See below snippit from app.js) 

var mainWin2 = Titanium.UI.createWindow ({
    backgroundColor: "#fff", // Set the background color to white
    url: "views/testing.js" // Link to file which will handle the code for the window
});

testing.js is below

var w = Ti.UI.currentWindow;

//Here's the first window...
var first = Ti.UI.createWindow({
  backgroundColor:"#fff",
  title:"My App"
});
var label = Ti.UI.createLabel({ text: "poke me to open the next window" });
first.add(label);

//Here's the nav group that will hold them both...
var navGroup = Ti.UI.iPhone.createNavigationGroup({
  window:w
});

//Here's a window we want to push onto the stack...
var second = Ti.UI.createWindow({
  background:"#fff",
  title:"Child Window"
});
second.add(Ti.UI.createLabel({text:"Here's the child"}));

//When the label on the first window receives a touch, open the second
label.addEventListener("click", function(e) {
  navGroup.open(second);
});

//This is the main window of the application
w.add(navGroup);
w.open();

If I change the window in NavGroup to be first then I get another window with another title bar displayed inside the current window. I would rather not use the hideNavbar() functionality as it looks dirty. Can I alternatively get rid of the title bar in the create window function that is used in app.js but still show the tabbed navigtion.
