ProgressBar crashes scrolling

Hello,

I have a window in which I have a scroll view. I would like to add a progress bar to it. The problem I am facing is that when I add the ProgressBar to the scrollview, the scrolling doesn't work at all. Anyone had an issue like this before, please help me out.

var scrollView = Ti.UI.createScrollView({
    contentWidth:'auto',
    contentHeight:'auto',
    top:0,
    showVerticalScrollIndicator:true,
    showHorizontalScrollIndicator:false
});

var progressBar = Titanium.UI.createProgressBar({
    width:240,
    min:0,
    max:10,
    value:4,
    color:'#aaa',
    style:Titanium.UI.iPhone.ProgressBarStyle.BAR
});

progressBar.show();
scrollView.add(progressBar);
