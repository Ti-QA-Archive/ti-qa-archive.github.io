    HI Deepthi,
    You can store the level information in app properties, and when user reopen the app then get he value from app properties the open the 10th level window for example.

    Doc Link:http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.App.Properties

    Example

    Ti.App.Properties.setString('lastLevel', '10');

Ti.App.Properties.getString('lastLevel'); // result is 10
