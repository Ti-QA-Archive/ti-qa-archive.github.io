  • Hi Greg.

    You could wrap the set methods with one which fires an event?

    function setStringProperty(property, value)
{
    Ti.fireEvent("property:changed", {property: property, value: value});
    return Ti.Properties.setString(property, value);
}
function setBoolProperty(property, value)
{
    Ti.fireEvent("property:changed", {property: property, value: value});
    return Ti.Properties.setBool(property, value);
}
