Titanium.UI.currentWindow is undefined

I have app.js and anotherfile.js in my sample project. I created a windows in app.js and opened it. I included "anotherfile.js" in app.js. In anotherfile.js, I tried to get the current window that I created from app.js. but I always get "undefined"..

How can I fix this issue? Thanks.

    Titanium.UI.currentWindow is only available in .js files that where used to create windows through the url parameter

  • Are you sure you have opened a window by the time you try to reference
    Ti.UI.currentWindow?

    When app.js starts there is no current window because none has been opened yet. Be sure that "anotherfile.js" does not run until AFTER you have opened a window.

    If that does not clear it up for you, please post the parts of app.js where you open a window, and where you include anotherfile.js
