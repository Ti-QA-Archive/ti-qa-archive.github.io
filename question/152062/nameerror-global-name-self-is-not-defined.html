  • Accepted Answer

    You are using Java 7 which is not supported by Titanium. You need to use version 6 (aka 1.6) in order to be able to work with Titanium. (you can have a look at the compatibility matrix from the documentation for further details)

    Also, it appears you are using Windows XP, which is also no longer supported. (as detailed in the documentation). Contrary to Java7, you can still get it to work on XP, but you won't be 100% sure everything will work flawlessly.

    Hope this helps.
