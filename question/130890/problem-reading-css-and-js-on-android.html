Problem reading CSS and JS on Android

Hi,

I have a local html file called 'index.html' which contains a <p> tag saying 'Hello'. I am viewing 'index.html' through a Ti.UI.WebView.

I also have a CSS file which simply changes the body text to red and a JS file containing jQuery code which simply displays an alert saying 'Hello World' when the page has loaded.

When I launch my app in an iPhone or iPad emulator everything works fine. I get a page with 'Hello' in red and an alert message from jQuery saying 'Hello World'. However, when I launch it in the Android emulator I get no styling and no jQuery functionality. I simply get a page with 'Hello' in black and no alert message.

I am new to Titanium and am not sure what I am doing wrong?

Here is my code:

app.js

var win = Titanium.UI.createWindow();

var myUrl = "index.htm";

var webview = Titanium.UI.createWebView({
    url: myUrl,
});
win.add(webview);
win.open();

index.html

<html>
   <head>
      <title>Demo</title>

      <!-- CSS -->
      <link rel="stylesheet" type="text/css" href="sample.css" />

      <!-- JS -->
      <script src="jquery-1.7.1.js"></script>
      <script src="demo.js"></script>
   </head>
   <body>
      <p>Hello</p>
   </body>
</html>

sample.css

body{
   color: red;
   font-size: 24px;
}

demo.js

$(document).ready(function() {
   alert("Hello World");
});

Please help!

Many Thanks in Advance!!!

  • Sorry made a mistake! Didn't mention that the CSS and JS files are located in another directory. If the JS and CSS files are locate in the same folder as the 'index.html' then everything is fine however as soon as I move the JS and CSS files into a separate folder then this is where the Android emulator has an issue but works fine on iPhone and iPad!

    CSS and JS references in 'index.html' should read:

    <!-- CSS -->
<link rel="stylesheet" type="text/css" href="includes/css/sample.css" />

<!-- JS -->
<script src="includes/js/jquery-1.7.1.js"></script>
<script src="includes/js/demo.js"></script>
