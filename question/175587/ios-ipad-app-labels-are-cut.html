iOS iPad App Labels are cut

Hi I have developed an app for iPads using titanium SDK 3.1.2 GA and now I'm doing an update of the app using 3.2.3 GA but in this version my labels are cut doesn't show all the text

this is how I create a Label

exports.createLabel=function(text,fontSize){
    var component=Ti.UI.createLabel({
        text : text,
        color : '#000',
        font : {fontSize:fontSize},
        height : 'auto',
        width : 'auto',
        textAlign : 'center'
    });
    return component;    
};

And this is a call to that method with the custom data

var lblHeader=components.createLabel('some large text.......',18);
lblHeader.top=50;
lblHeader.left=20;
lblHeader.right=20;
lblHeader.height=40;
lblHeader.textAlign='left';
lblHeader.font={fontSize:18,fontWeight:'bold'};

In 3.1.2 GA it was showing all text but in this version doesn't show all good
any suggestion?
