  • The correct code to remove all children from a window is this:
    Paul had the loop going up, but when removing it should always go down..

    function cleanWindow( winObj )
{
    if (winObj.children) {
        Ti.API.info('Has children! Len: ' + winObj.children.length);
        for (var i = winObj.children.length; i > 0; i--){
            Ti.API.info( (i-1) + ") " + winObj.children[i-1]);
            winObj.remove(winObj.children[i-1]);
        }
    }
}

  • Benaim

    Indiscriminately removing all child views from a window sounds instinctively bad to me. As a window's main purpose is to contain views, if I didn't need any views in a particular window any more I would consider closing the window and recreating it, to guarantee the cleanup process.

    That said, it is possible, and here's the code:

    var win = Ti.UI.currentWindow;
if (win.children) {
for (var i = 0, ilen = win.children.length; i < ilen; i++){
 win.remove(win.children[i]);
}}

    Hope this helps
