  • I personally would not waste any time trying to support BB with Titanium. The implementation is incomplete and the recent news about Ti.Next - Quote

    We need to make progress on Windows and Android backends before we can start the work of re-building the Ti.Next platform. However, we’ve done enough early work on Ti.Next APIs in our R&D phase, that we’re confident we’ll be able to go fast once we get Hyperloop to that stage. I can’t commit to any dates just yet, but our hope is to have all three first class platforms (iOS, Android and Windows) stable enough by the end of the year we can start working in earnest on Ti.Next APIs

    Indicate the BB is not on the initial target OS list, although Windows Phone is that currently isnt supported at all.
