XHR Send

Hello All,

I have a question regarding sending data with an xhr connection.
I have successfully made connections and sent data using the below code.

xhr.send({my_username: variable_username ,my_password: variable_password });

I am now making a connection which is expecting certain variables of similar names

char[1], char[2], char[3].

The server will always expect to recieve a value for each of the above mentioned. I think that the number and the square brackets in the names causes issues.

xhr.send({char[1]: variablecharacter1 ,char[2]: variablecharacter2 , char[3]: variablecharacter3});

Is there a different way I should be formatting the request.
Unfortunately, I can not change the fact the server is expecting char[1], char[2] and char[3], as giving them seperate names would be the simplest option.

Thanks,

Lee
