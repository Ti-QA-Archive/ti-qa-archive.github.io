Dynamically generated image list (SQLite + split + push)

Hi
I have a SQLite database with a field IMAGES that have:

A,B,C

B,D,A,F

A,C

And I have a image gallery that are set up like this:

var images = ['../images/A.png', '../images/B.png];

I have set up a code to dynamically generate the images, and it generates the right number of images (e.g 2 for "A,C"), but it seems like the URL of the image is wrong. What do I do wrong?

var images = [];
var txt = rows.fieldByName('IMAGES');
var arr = txt.split(', ');
    for(var i=0; i<arr.length; i++) {
            images.push('../images/' + arr+ '.png');
}
