Titanium DB open gets error : contains a path separator

unfortunately this code doesn´t work

A database, with a filename of mydb2Installed and located at the
absolute path provided, is opened.

if (Ti.Platform.name === 'android' && Ti.Filesystem.isExternalStoragePresent()) {
  var db2 = Ti.Database.open(Ti.Filesystem.externalStorageDirectory + 'path' + Ti.Filesystem.separator + 'to' + Ti.Filesystem.separator + 'mydb2Installed');
}

Unlike on iOS, no file extension is added. The file is copied to the
absolute path, as follows.

file:&#x2F;&#x2F;&#x2F;sdcard&#x2F;path&#x2F;to&#x2F;mydb2Installed

Every time i try to open / install ive got :
06-29 17:29:58.850: I/TiAPI(17009): ##### Database error : Error: File file:///mnt/sdcard/app/databases/app.db contains a path separator

The path exists.. file exists..

general info…
Application type: mobile - Android
Titanium SDK : 2.02 / 2.1
Device : Galaxy Tab 7 Plus
Host Operating System : Windows 7

  • I was having this error. You need to pass the file path string to Ti.Database.open but without the 'file://' part, i.e. '/sdcard/path/to/mydb2Installed'. This worked for me.
