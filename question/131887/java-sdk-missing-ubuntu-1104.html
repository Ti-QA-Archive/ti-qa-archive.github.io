  • Check that you have a jdk installed and not just a jre.

    'javac -version' should give output like:
    javac 1.6.0_23

    If you get an error instead, try installing the jdk:
    apt-get install openjdk-6-jdk

    I didn't need to set up any paths to make it work.

    I know you are running 32 bit, but for others reading the posts - this is working fine on 64 bit Ubuntu for me. YMMV.

  • For local development and testing, you do not need anything except Titanium and the Android sdk (and their requirements). No Android pay account is required.

    I haven't run into the problem you are seeing, but I would check that the android sdk tools are working first.

    If the android simulator is running, the command 'adb devices' should should show it. My system (with simulator running and a phone connected) shows:

    $adb devices

List of devices attached 
emulator-5560    device
043c624944c00557    device
