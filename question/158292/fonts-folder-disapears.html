    The Resources folder gets regenerated on each alloy compile. Assets that you want to be carried over to the Resources folder should be put in Alloy's app&#x2F;assets folder.

    You'll probably want to check out these docs: http://docs.appcelerator.com/titanium/latest/#!/guide/Alloy_Concepts-section-34636240_AlloyConcepts-ConventionoverConfiguration
