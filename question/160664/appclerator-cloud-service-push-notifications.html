Appclerator cloud service push notifications

I'm using titanium push notifications using Appclerator cloud service and I send these notifications through the ACS console which is by default used to send to a specific channel or all channels subscribed to the service , Can anyone help me to get to know how to send to a device through the console to a specific "device token" or a device mainly not a channel ?

  • I'm not sure there is a way to avoid specifying a channel, but you can specify user IDs in the console to limit your notifications to those users. Outside of the console, you can use the notify_token command to target specific devices (I believe you still need to specify a channel however).

    An example using curl:

    First login in:

    curl -k -b cookies.txt -c cookies.txt -F "login=<aUser>" -F "password=<password>" https://api.cloud.appcelerator.com/v1/users/login.json?key=<yourAppKey

    Then send the push to a device via notify_tokens

    curl -v -b cookies.txt -c cookies.txt https://api.cloud.appcelerator.com/v1/push_notification/notify_tokens.json?key=<yourAppKey> -F "to_tokens=<deviceToken>" -F "channel=<channel>" -F "payload={alert:'Test notification'}"
