  • I do this in the "journal" test app in the github repo: https://github.com/appcelerator/alloy/tree/master/test/apps/models/journal

    WARNING: Read the data binding docs section in full. Due to some shortcomings in Titanium's view processing, the behavior of data binding a collection to an arbitrary View (like a ScrollView) is a bit unstable and might generate unexpected results. It is reliable on Android, but iOS may not behave as you would expect. Consider it experimental at this point.

    font : {
        fontSize : myfontsize * 1.2,
        //    fontWeight : 'bold',

    width : Ti.UI.SIZE,
    top : '0%',
    right : '12%',
    left:'12%',
    height :Ti.UI.SIZE,
    color : '#666'
});
scroll.add(labInstruction);
