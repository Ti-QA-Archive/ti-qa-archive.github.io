  • I don't think it's possible to record "internal audio" (but you need to explain exactly what you mean by that. I'm assuming you mean things like phonecalls? (not possible as far as I know)

    Regarding the question about combining video and audio files, there is a module here called Timedia - https://github.com/urish/TiMedia (which needs updating to the latest 64bit SDK. Ask the developer), which exposes the 'AVFoundation' API to Titanium.

    The module contains an example of how to join together some audio files. If you follow this and read the AVFoundation documentation for how to construct a 'composition', add tracks and mixes, then export the session, you might get what you need. ;)
