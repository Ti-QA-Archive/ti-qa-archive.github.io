App crashes when animate() is called on a view

When testing code on the android emulator, the following code throws an exception and ends the process.

    var self = Ti.UI.createWindow({
        backgroundColor:'#ffffff',
        navBarHidden:true,
        exitOnClose:true
    });

    var vX = Ti.UI.createView({
            width: '80%',
            height: 100,
            borderRadius: 30,
            borderColor: '#1a0033',
            backgroundColor: '#8400ff',
            bottom: 330
    });
    self.add(vX);
    vX.addEventListener('click', function(){
        vX.animate({left:'20',height:'400'})
    });

Any advice appreciated.
