Dynamically added Table contents in titanium iphone?

Hi,
I want to create a dynamic table view in which it contains 3 rows and eachrow will have different elements like textbox,lables,picker respectively.
I have add button below the table onclicking on this add button ,Each time i want to append this 3 rows.I dnt know how to do this.
Can anybody have the solution?

  • Simply wrap the creation of your 3 rows in a function. You simply pass the TableViewObject to the function so it can append the rows to it when the user presses the button.

    Here is what the code could look like:

    function appendRows(myTableView) {
    var rowOne = Ti.UI.createTableViewRow();
    var rowTwo = Ti.UI.createTableViewRow();
    var rowThree = Ti.UI.createTableViewRow();

    /* Add the different controls to each row as you see fit */

    myTableView.appendRow(rowOne);
    myTableView.appendRow(rowTwo);
    myTableView.appendRow(rowThree);
}

    Hope his helps!
