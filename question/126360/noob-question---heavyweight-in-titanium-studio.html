    • If you have any of these properties in your window definition, then the window will be heavyweight:

      url
fullscreen (true or false)
navBarHidden (true or false)

      Since fullscreen or navBarHidden could be either true or false and still make the window heavyweight, seeing the bar at the top may not be an accurate indicator of heavyweight..ness.

      Now that you are using Studio, you will be able to run debugger and check to see threads being created for the heavyweight windows. This will tell you for sure. Check out the Debug view and expand Emulator > Android JS Debugger and see the threads listed for the running app.
