Android: Blur/Focus events do not fire on TabGroup

I was trying to work around some issues and saw in another Q&A that listening to blur/focus was a good way to detect changing of activeTab. This worked great on iOS, but these events did not fire on the TabGroup on Android.

    Geesh this is wierd. I can't get the window's listen event to fire on android (though I do see it working in the kitchen sink), so I had to add a event listener to the tab… which sounds like the complete opposite of your experience?

    win.addEventListener('focus', MyApp.UI.Checkout.CartDisplay);

if ( Ti.Platform.osname == 'android' ) {
  tabGroup.addEventListener('focus', function(e) {
    if ( e.index == 3 ) {
      MyApp.UI.Checkout.CartDisplay();
    }
  });
}
