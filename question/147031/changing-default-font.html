Changing default font

Hey Developers,

I just got my custom font working, but now I want to have this font set on every label in my app. Is there a way to change the default font into my custom font or do I really have to set the fontFamily property for every single label in my app?

Regards,
Niels

    You will have to set the custom font For each of your label you have used. like

    // label is the name of label for which you want to change the label.
label.font = {fontSize: 24, fontWeight: 'bold', fontFamily: 'Arial'};
