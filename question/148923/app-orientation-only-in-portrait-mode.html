    so for iphone , you can set that from tiapp.xml file

    <orientations device="iphone">
            <orientation>Ti.UI.PORTRAIT</orientation>
</orientations>

    and for android , you should copy AndroidManifest.xml file from android source and put that file in project/platform/android folder then replace the 2 properties with below code

    android:screenOrientation="portrait" 
android:configChanges="keyboardHidden"

    for more reference , follow this link
