  • Hi

    Guessing you are talking about the new project dialog;

    • App id

    This is up to you but usually it is a reverse domain name format - this is usually but not always a domain name in reverse. This however your choice - so how about something like this;

    us.mattjohn.appname

    Simply replace john with your friends actual name, replace appname with a lowercase all one word version of your final actual name. So "word with Friends" would be wordswithfriends

    When you have another app;

    us.mattjohn.nextappname
    • URL

    Leave it blank
