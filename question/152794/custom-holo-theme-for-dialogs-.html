Custom Holo theme for Dialogs ?

Hi there,

When customizing the Android Holo Theme I found that the OptionDialog still uses the default Holo.Light theme.

I have tried the following.


<style name="CustomTheme" parent="@android:style/Theme.Holo.Light.DarkActionBar">
        <item name="android:alertDialogStyle">@style/CustomDialogTheme</item>
        <item name="android:dialogTheme">@style/CustomDialogTheme</item>
</style>

<style name="CustomDialogTheme" parent="@android:style/Theme.Dialog">
        <item name="@android:background">@color/grey</item>
        <item name="@android:textColor">@color/blue</item>  
        <item name="android:bottomBright">@color/grey</item>
        <item name="android:bottomDark">@color/grey</item>
        <item name="android:bottomMedium">@color/grey</item>
        <item name="android:centerBright">@color/grey</item>
        <item name="android:centerDark">@color/grey</item>
        <item name="android:centerMedium">@color/grey</item>
        <item name="android:fullBright">@color/blue</item>
        <item name="android:fullDark">@color/blue</item>
        <item name="android:topBright">@color/blue</item>
        <item name="android:topDark">@color/blue</item>
</style>

Anyone have custom theming working on dialogs ?

  • I going to answer my own question. This is how you do it:

    styles.xml

    create a file in /platform/android/res/values/styles.xml

    <resources>
  <style name="CustomTheme" parent="@android:style/Theme.Holo.Light">
        <item name="android:alertDialogTheme">@style/CustomDialogTheme</item>
        <item name="android:dialogTheme">@style/CustomDialogTheme</item>
  </style>

  <style name="CustomDialogTheme" parent="@android:style/Theme.Dialog">
        <item name="@android:background">#9933CC</item>
        <item name="@android:textColor">#FAF5F7</item>  
  </style>
</resources>

    TiApp:

    Its important to use minSDK 14, due to custom xml is not supported before this SDK. Remember to add the custom theme name (here its CustomTheme), to both the application and main app activity. 

    <android xmlns:android="http://schemas.android.com/apk/res/android">
        <manifest android:versionCode="1" android:versionName="1.0">
            <uses-sdk android:minSdkVersion="14" android:targetSdkVersion="17"/>
            <application 
                android:debuggable="false" 
                android:theme="@style/CustomTheme"
                android:label="ThemeApp">
                <activity android:configChanges="keyboardHidden|orientation|screenSize" 
                    android:label="ThemeApp" 
                    android:name=".ThemeappActivity" 
                    android:theme="@style/CustomTheme">
                    <intent-filter>
                        <action android:name="android.intent.action.MAIN"/>
                        <category android:name="android.intent.category.LAUNCHER"/>
                    </intent-filter>
                </activity>
            </application>
        </manifest>
    </android>

    That should do it.
