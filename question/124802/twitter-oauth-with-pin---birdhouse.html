  • If you haven't specified a callback_url, do it like this:

    var BH = new BirdHouse({
    consumer_key:      "XXXXXXXXXXXXXXXXXXXXXX",
    consumer_secret: "XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX",
    callback_url:      "http://example.com/twitter/oauth" // NEEDED FOR OVERRIDING. Can be any URL.
});
