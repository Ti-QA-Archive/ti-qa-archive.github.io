Custom Module error: "Couldn't find module:..."

Mobile Application

Titanium SDK version 3.1.1.GA

Titanium Studio, build: 3.1.1.201306112235

iOS Simulator

OSX 10.8.3

I have found similar questions, but no answers that have helped solve my specific problem. This is my first time creating a custom module, so I'm hoping the answer is an easy one. Basically, when I try to run the app in the iOS simulator, I receive the following error:

"Application Error - Couldn't find module: CustomerAppointmentStubClass"

I'm using the Alloy Framework, and this is a newly created project so all directories/files are currently in their default location as specified below.

[app/views/index.xml] My index view with a basic button: 

<Alloy>
    <Window class="container">
        <Label>Welcome!</Label>
        <Button id="createNew"  class="interface-button" onClick="doCreateAppt">Create New Appointment</Button>
    </Window>
</Alloy>

[app/controllers/index.js] Corresponding 'Controller' class:

var CustomerApptClass = require('/CustomerAppointmentStubClass');
var ca = new CustomerApptClass();

function doCreateAppt(e) {  
    var data = { /* Populate necessary Data  */};
    ca.create(data);
}
$.index.open();

[app/controllers/CustomerAppointmentStubClass.js] My custom module:

function CustomerAppointmentStubClass ()
{
    var _customerData = {    name:    "John Smith",
                            shirt:    "0xDD2222",
                            phone:    "978-123-4567",
                            appt:    "MM-DD-YY H:MM:SS"
                        }


    this.create = function (data) 
    {
        Ti.UI.createAlertDialog({ message: 'Create New Appointment' }).show();
    };
}

module.exports = CustomerAppointmentStubClass;

  • Your controller and module are in the same folder so just do this;

    var CustomerApptClass = require('CustomerAppointmentStubClass');

    Note: No leading slash
