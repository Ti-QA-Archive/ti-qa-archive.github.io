Get real Image dimensions

Hi all,

I try to create a crossplattform app.

So I have an image 1000x200 and I wont to position it as image on background.

I try to calculate a real image size, to get it perfect on each screen.

menuBottomImageView.toImage().width works fine on iOS but on android I get not the rela size, only teh displayd size and thats not what I nead.

Is there a way to get a real dimensions of file for both systems?

  • var testImg = Ti.UI.createImageView({
        image : this.imagepath,
        height : Ti.UI.FILL,
        width : Ti.UI.SIZE || 'auto',
        visible : true
    });
    var img = testImg.toImage();
    var width = img.getSize().width;    
    var height = img.getSize().height;
    testImg = null;
