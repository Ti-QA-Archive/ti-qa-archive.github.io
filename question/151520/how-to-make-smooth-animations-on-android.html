How to make smooth animations on Android

The real question here is how can we improve the actual animations for android, when we build iOS apps, we can do beauty ones, very smooth and cool even with ease, but then when we run our apps on android it looks very strange, not even close to smooth and without the ease possibility.

Bellow I put a very simple code animating a view with colors to make it easy to see.

Run it on iOS it will be very smooth, run it on android and you will notice it's laggy, it "jump" frames, I can't really explain that, it's more noticeable in the middle of the animation.

1 - I know there is a very similar question to it and I'm sorry, but it never came up with a real answer and it's old.

2 - It's not a hardware problem, tested it on S3,S2,N4 and iPhone 5

3 - Using latest Titanium version 3.1

Code for testing:

var window = Ti.UI.createWindow();

var animation1 = Ti.UI.createButton({
    title:"Animation 1",    
    top:'0dp',
    left:'160dp',    
})

animation1.addEventListener("click",function(e){
    Ti.API.info('clicked animation 1')
    var mView = Ti.UI.createView({
        backgroundColor:'white',
        left:'0dp',
        width:'160dp',
    })
    var sView1 = Ti.UI.createView({
        backgroundColor:'blue',
        left:'0dp',
        width:'50dp',
        height:'100dp',
    })
    var sView2 = Ti.UI.createView({
        backgroundColor:'green',
        top:'0dp',
        left:'0dp',
        width:'50dp',
        height:'100dp',
    })
    mView.add(sView1)
    mView.add(sView2)
    window.add(mView)
    mView.animate({
        duration:3000,
        left:'320dp',
    })
})

window.add(animation1)
window.open();
