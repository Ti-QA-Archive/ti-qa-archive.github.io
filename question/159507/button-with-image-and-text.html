Button with image and text

I want to create simple button with image and text.

So far, I succed to create layout wich I want it is like on picture

![text](http://postimg.org/image/ylvt1m2c9/ "button")

or here : http://postimg.org/image/ylvt1m2c9/

problem is, that I achieve this by button in combination with imageview and view.

I have view -> I placed button with text into view -> i placed imageview into view.

This is looking good so far, BUT, of course when I click image on button button dies not appear to be pressed. (change from grey to orange), it appear to be pressed only if I click grey part of button.

I wan t to change this to behave as users expect (when they tap on button it looks pressed, even if they tap on picture)

here is my code

 var buttonview = Ti.UI.createView();
    buttonview.width = '50pt';
    buttonview.height = '50pt';

    var button = Ti.UI.createButton();
    button.width = Ti.UI.FILL;
    button.height = Ti.UI.FILL;
    button.title = 'Activity';
    button.verticalAlign = Ti.UI.TEXT_VERTICAL_ALIGNMENT_BOTTOM;

    imageview.image = '/activities.png';
    imageview.height = '80%';
    imageview.width = '80%';
    imageview.top = '1pt';
    buttonview.add(imageview);

    scroll.add(buttonview);

  • Create a button and add the imageview and a label to it, positioning them correctly:

    var button = Ti.UI.createButton();
...
var img = Ti.UI.createImageView();
...
button.add(img);
...
var lbl = Ti.UI.createLabel();
...
button.add(lbl);
...
scroll.add(button);

    where … is all the code that sets the properties of the objects;
