Alloy, loading separate controllers for tabs in TabGroup

Hi,
Given the (simplified) example alloy code:

<Alloy>
    <TabGroup id="tabGroup">
        <Tab id="tab1" title="First Tab" icon="KS_nav_views.png"></Tab>
        <Tab id="tab2" title="Second Tab" icon="KS_nav_views.png"></Tab>
        <Tab id="tab3" title="Third Tab" icon="KS_nav_views.png"></Tab>
    </TabGroup>
</Alloy>

Is there a way, either in the Alloy or in the example code's main .js file, to specify a separate Alloy controller to load for each tab?

Note that the seperate Alloy controllers appear something like this with multiple windows so I would need to specify a specific window in the controller to use for the tab window, however the entire controller would needed to be loaded obviously. An example of the secondary Alloy controller:

<!-- External alloy controller xml file for one of the tabs, -->
<Alloy>
    <Window id="mainWin" class="container" title="Main"></Window>
    <!-- These other windows will be opened as modal windows -->
    <Window id="modalWin" class="container" layout="vertical"></Window>
    <Window id="commonMoleculesWin"></Window>
</Alloy>

  • Problem solved,

    I just used the <Require> function in the Alloy file, and specified the Alloy controller name in the src field. It uses the first window in the chosen controller so the other windows in the file are ignored, works perfect.

    Example:

    index.xml:

    <Alloy>
    <TabGroup id="tabGroup">
        <Tab id="tab1" title="Balancer" icon="KS_nav_views.png">
            <Require type="view" src="balancer" id="balancerView"/>
        </Tab>
        <Tab id="tab2" title="Common Molecules" icon="KS_nav_views.png">
            <Require type="view" src="commonmolecules" id="commonMoleculesView"/>
        </Tab>
    </TabGroup>
</Alloy>

    balancer.xml:

    <Alloy>
    <Window id="index" class="container" title="Main">
        <Label>Hello World!</Label>
    </Window>
    <!-- These other windows are ignored! Only the first one above is used. -->
    <Window id="modalWin" class="container" layout="vertical">
        <ImageView id="equationImg"></ImageView>
        <TableView id="elementTable"></TableView>
    </Window>
    <Window id="commonMoleculesWin">
        <TableView></TableView>
    </Window>
</Alloy>
