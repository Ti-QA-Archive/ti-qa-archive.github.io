Nested layout on Iphone

Hi everyone,

I'm new to titanium and I'm getting stucked with layouts. My test have a vertical layout containing a horizontal layout, which in turn has 2 labels. On android, everything works fine, but not on iphone.

I'm using sdk 1.8.2 with iOS sdk 4.3
Here is the code:

var win = Ti.UI.createWindow({
    layout: 'vertical',
    backgroundColor: 'white',
    height: '100%'
});

var ho = Ti.UI.createView({
    layout: 'horizontal',
    backgroundColor: 'red',
    height: 'auto'
});

ho.add(Ti.UI.createLabel({text: 'first label', height: 'auto', width: '50%'}));
ho.add(Ti.UI.createLabel({text: 'second label', height: 'auto', width: '50%'}));
win.add(ho);
win.open();

Can someone tell me if I'm doing something wrong?

Thanks.

    You need to set label's width to a specific value - In your case would be: Ti.Platform.displayCaps.platformWidth&#x2F;2

    Try this:

    var win = Ti.UI.createWindow({
    layout: 'vertical',
    backgroundColor: 'white',
    height: '100%'
});

var ho = Ti.UI.createView({
    layout: 'horizontal',
    backgroundColor: 'red',
    height: 'auto'
});

var width = Ti.Platform.displayCaps.platformWidth;
ho.add(Ti.UI.createLabel({text: 'first label', height: 'auto', width: width/2}));
ho.add(Ti.UI.createLabel({text: 'second label', height: 'auto', width: width/2})); 

win.add(ho);
win.open();
