  • Hi Karray,

    doing it the reverse order is easier.

    In Titanium Studio start from menu: Run -> Run Configurations…

    • On the left side under Titanium Android Emulator create a new configuration
    • give it a name, select your Project, make your Platform settings
    • click Apply
    • click Run
    • Titanium now creates a new AVD for you which you can run from Titanium Studio run menu and run dropdown box
    • this AVD you can now tweak to your specific needs using AVD Manager

    Same you can do for Debug Configurations…

    Greetings,

    Matthias
