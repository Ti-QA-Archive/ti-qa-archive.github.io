  • I know this is a little late. But there is an easy way to get to say "a" in a programmatic way:

    As stated the normal 'Alloy way' is:

    $.a

    … but following the Javascript notation you can as well do:

    $['a']

    … and in this latter example you then have the opportunity to calculate the id (or grab it from a variable). I am not quite sure that is what you wanted. But this actually answers the original question - and therefore, I left the answer here - should other come looking for it ;-)
