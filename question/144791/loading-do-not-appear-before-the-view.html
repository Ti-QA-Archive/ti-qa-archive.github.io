    • Hi

      If this is for IOS you have to put the activity indicator to a view.
      You dont have to do this on Android, because the activity indicator is a type of modal window in itself.

      Try to make a view that holds the tableview. and put the act ind in that view

      Tom

      var win = Ti.UI.currentWindow;
var tableviewholder= new Ti.UI.CreateView({
    width:300,
    height:300
});
win.add(tableviewholder);
var actInd = Titanium.UI.createActivityIndicator({
    style:Titanium.UI.iPhone.ActivityIndicatorStyle.PLAIN,
});
tableviewholder.add(actInd);
actInd.show();

      You could also try to put the activity indicator in the tableview directly.
