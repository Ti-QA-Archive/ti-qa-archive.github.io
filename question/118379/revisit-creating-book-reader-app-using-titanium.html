    AFAIK, the epub format is really a zip file with a different extension, which contains html and css etc.

    So, you could try using a module I helped get working from here : https://github.com/kosso/ZipFile

    .. to extract the contents. Then you would use a webView to present the content.

    From there, you could look at using another 'component' someone has created here for the page effect.

    http://developer.appcelerator.com/question/93151/flipbook-component

    Though, it appears that you'd need to use images for that to work, so you could try using the .toImage() method on a UI element to create the required image for the 'flip' effect.

    could work! :) good luck!
