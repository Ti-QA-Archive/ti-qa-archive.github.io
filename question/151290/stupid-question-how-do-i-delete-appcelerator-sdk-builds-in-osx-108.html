Stupid question, how do I delete appcelerator SDK builds in OSX 10.8?

I use the continuous builds, but my list has gotten a bit huge! I wanted to delete some of my old builds that I no longer use, but cant find it anywhere on my file system no matter where I look. Any ideas?

Thanks!!

    Hi

    Not surprised you have not found it, the folder you are looking for is in the hidden library folder that OSX now deems not safe for public to see without jumping through a little hoop first.

    • finder
    • devices
    • {your main storage drive}
    • Users
    • {your user name}

    Now the hidden folder;

    • Menu > Go > Go to Folder… > Library (not the first letter is upper-case)

    Back to the folders

    • library
    • application support
    • titanium
    • mobilesdk
    • osx

    Now you have a list of the SDKs you have installed, be smart and back things up BEFORE you do anything.

    • Or you can open the Terminal app and list all installed SDKs by running:

      titanium sdk

      Then to remove a specific SDK, run:

      titanium sdk uninstall <version>
