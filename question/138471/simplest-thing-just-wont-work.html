Simplest thing just wont work

////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
TYPE: Mobile

PLATFORM: IOS

DEVICE: Simulator

HOST: OSX Lion
////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

So I have an array that holds everything about my buttons because I dynamically create them.

example of an entry:

var buttonPropertyArray = [
{rest:'images/parts_of_speech/noun_rest.png',active:'images/parts_of_speech/noun_pressed.png',x:8,y:246,width:149,height:27, cEvent:'PoemTextArea.value = PoemTextArea.value + " " + arrays.noun[ Math.floor (Math.random()*967 )];'},
];

Then in a for loop I am creating the buttons so I would think this works:

button.addEventListener('click', function(e) {

    eval(buttonPropertyArray[i].cEvent);
});

But it obviously doesn't so I try to put it into a holder var but that didnt work, only way I could get it to at least use one of the functions was to hardcode the position in the array to pull it from like:

var test = buttonPropertyArray[0].cEvent.toString();

button.addEventListener('click', function(e) {

    eval(test);
});

But whats the easiest way to do this dynamically?

  • Hey Andrew,

    I used the following code, it works for me

    

var button1 = Titanium.UI.createButton({
    color:'#999',
    title:'Click Me',
    font:{fontSize:20,fontFamily:'Helvetica Neue'},
    textAlign:'center',
    width:'auto'
});

PoemTextArea={value:10};

var buttonPropertyArray = [
{rest:'images/parts_of_speech/noun_rest.png',active:'images/parts_of_speech/noun_pressed.png',x:8,y:246,width:149,height:27, cEvent:'PoemTextArea.value = PoemTextArea.value + " " + Math.floor (Math.random()*967 );'},
];


button1.addEventListener('click', function(e){

    eval(buttonPropertyArray[0].cEvent);

    alert(PoemTextArea.value);

});

win1.add(button1);

    If this also not works try

    eval( &#39;(&#39; + buttonPropertyArray[0].cEvent + &#39;)&#39; ) instead of eval(buttonPropertyArray[0].cEvent)

    I hope this will work for you..
