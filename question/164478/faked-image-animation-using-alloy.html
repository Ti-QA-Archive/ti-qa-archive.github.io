    You can use builtin image animation of the ImageView

    xml part 

    <Alloy>
    <ImageView id="animImgView"/>
</Alloy>

    tss code

    "#animImgView": {
    duration: 200,
    repeatCount: 0,
    images: [
        'anim01.png',
        'anim02.png',
        'anim03.png',
        'anim04.png',
        'anim05.png',
        'anim06.png',
    ]
}
