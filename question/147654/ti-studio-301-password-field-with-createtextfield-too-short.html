Ti Studio 3.0.1: Password Field with createTextField too short

Hi experts,
When I create a password field like this

    var l = Ti.UI.createTextField({
        passwordMask: true,
        borderWidth: 1,
        borderColor: '#FF0000',
        hintText: 'hint',
        width: '200dp'
    });

The resulting field accepts only one character.
If I set passwordMask: false, the field accepts several characters.

Using Titanium Studio 3.0.1
It works with Titanium Studio 3.0.0

Am I doing something wrong?

TIA Stephan

    Hi

    I have created a stand-alone simple app with your single textField and it works perfectly, just in-case it is a height issue I have added a height property as the code you supplied was VERY short. I am wondering if you have a global settings or system unit tweak.

    Try the code below EXACTLY as it is and report back.

    To test this code; create a new default project and replace the contents of the app.js file with the code below.

    var win = Titanium.UI.createWindow({
    backgroundColor: '#fff',
    title: 'Window'
});
var l = Ti.UI.createTextField({
    passwordMask: true,
    borderWidth: 1,
    borderColor: '#FF0000',
    hintText: 'hint',
    height: 30,
    width: '200dp'
});
win.add(l);
win.open();
