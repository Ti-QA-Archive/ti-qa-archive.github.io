  • Titanium doesn't have an API for fetching URL parameters, but you can do it using the browser's built-in location API:

    if (Ti.Platform.osname == 'mobileweb') {
    var params = {};
    location.search.split('&').forEach(function(s) {
        var m = s.match(/([^=?]+)\=([^&]*)/);
        if (m) {
            params[m[1]] = m[2];
        }
    });
    console.log(params);
}

    Give that a shot and see if it works for you.
