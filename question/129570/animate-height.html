Animate Height

Hi,

I have an object and I want to animate its height. The code I have is the following:


    var revealmask = Ti.UI.createView({
        zIndex: 10000,
        backgroundImage: 'revealmask.png',
        width: 300,
        height: 390,
        left: 10,
        bottom: 71,
        anchorPoint:{x:1,y:1}
    });
    win.add(revealmask);

    var up = Titanium.UI.createAnimation();
    up.height = 0;
    up.duration = 800;
    revealmask.animate(up);

Unfortunately, despite my best attempts, this always results in the bottom coordinate to be reset and the view ends up having its height animated properly, but the view glides across the screen as it goes to the proper bottom coordinate. If I remove the animation, it sits in its proper position unanimated.

Does anyone see what I'm doing wrong? This seems like a pretty simple issue, but I can't seem to find anything.

Thanks for your help!
