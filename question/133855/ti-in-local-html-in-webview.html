Ti in local HTML in webview

Can I use the complete Titanium framework in the webview with a local html file?

— asked March 15th 2012 by Sebastian Klaus

  • Accepted Answer

    Not sure you can.

    What you could do is have an event "handshake" of sorts. Javascript within the webview listens for a "data" event and fires a "ready" event when it's ready to receive the data. My main Titanium code listens for this "ready" event and when triggered fires the "data" event, passing the data via the event parameters. The webview can then receive any data and deal with it how you want.

  • Nope. As the API documents state for webviews;

    1. You can use Titanium.App.addEventListener and Titanium.App.fireEvent to receive and send application-level events.
    2. Events can be logged using the Titanium.API logging methods.

    http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.UI.WebView-object.html
