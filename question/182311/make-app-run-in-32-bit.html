Make app run in 32 bit

Hello, I inherited a prototype app that runs on old iPads, but not on iPad Air or 64 bit systems.
One of the modules is not 64 bit… so I was wondering if there was a way to force the App to work in 32 bit mode on a 64 bit processor?

e.g. the emulator works in everything but iPad Air and iPad Retina 64 bit.

It worked on my iPad Air with the old version of Test Flight, but the iOS 8 stopped that.

When I put the app into the new test flight, it doesn't work on iPad Air.

Any thoughtS?

  • As quoted from Apple "As we announced in October, beginning on February 1, 2015 new iOS apps submitted to the App Store must include 64-bit support and be built with the iOS 8 SDK. Beginning June 1, 2015 app updates will also need to follow the same requirements. To enable 64-bit in your project, we recommend using the default Xcode build setting of “Standard architectures” to build a single binary with both 32-bit and 64-bit code." You need to update any outstanding modules to run in 64bit. If you open the modules Xcode project and follow the steps HERE, you should be good to go.
