Module licenses question (and all of Titanium, actually)

I am trying to use the Urban Airship modules for both iOS and Android in an app for work. We bought Indie licenses a few months ago. Getting push working on iOS was easy, on Android, after several days I don't have it working yet.

However this is not my current question. While watching the debug console on Android, I saw a message as the app starts, "verifying module licenses" which got me thinking.

If we deploy an app to the App Store that uses one of these modules, if we decide not to renew the Indie license next year, does that mean that the app will quit working for anyone who downloaded it in the meantime? Will the modules suddenly believe they are unlicensed?

And furthermore, will ANY apps built using Appcelerator suddenly cease to work, if we decide not to renew the Indie license? Is an Internet connection required for an app to run?

Thanks.

  • The way it has always been explained in the past is a current license is needed to BUILD a project. Already deployed apps will continue to work if you do not renew a license from Appcelerator. However, you would not be able to make any changes to the program and rebuild until you had a current license again.

    AFAIK, this is still true even though the marketplace is now open.
