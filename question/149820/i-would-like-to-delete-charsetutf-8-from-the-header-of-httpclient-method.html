I would like to delete "charset=utf-8" from the header of HTTPClient method.

Hello, everyone.

I'm creating App which use Cloud Service now.
The header has been a problem although data is transmitted using HTTPClient in it.
If multi-part data is sent by HTTPClient, "charset=utf-8" will be added to a header.
This generates a problem with some services.
Can't this charset be erased?

Test program:

xhr.open('POST', 'http://127.0.0.1:8080/test/');
xhr.setRequestHeader('Content-Type','multipart/form-data');
xhr.setRequestHeader('User-Agent','');
xhr.setRequestHeader('Accept','application/json');
xhr.send({image:blob});

Problem header part:

Content-Type: multipart/form-data; charset=utf-8; boundary=0xKhTmLbOuNdArY-04313890-E52F-4C7C-81B6-48D076E3517D
