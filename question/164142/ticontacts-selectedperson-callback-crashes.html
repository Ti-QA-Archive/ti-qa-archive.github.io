Ti.Contacts selectedperson callback crashes

Ti SDK 3.2.1, iOS 7, testing on iPhone 5C

I'm attempting to use Ti.Contacts to allow a contact selection and passing email onto function

Here's code

function sendEmail(addr) {
        var self = Ti.UI.createView();
        var emailBox = Ti.UI.createEmailDialog({
            toRecipients : [addr],
            html : true,
            subject : "Test!",
        });

        emailBox.open();
    };

    function emailToContacts() {
        Ti.Contacts.showContacts({
            selectedProperty : function(e) {
                alert(e);
                var addr = e.value;
                sendEmail(addr);
            },
            animated : true,
            fields : ["email"],
        });
    };

emailToContacts() works - opens contact list, sendEmail works - emaildialog opens, but calling sendEmail from emailToContacts as the callback causes app to crash
