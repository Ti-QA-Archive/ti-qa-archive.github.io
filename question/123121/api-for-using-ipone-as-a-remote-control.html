API for using IPone as a remote control?

Does Titanium provide and API for programming device remote control. An example would be if I wanted to use my IPhone to turn my landscape lighting on and off remotely.

  • If you can control the device via a TCP network connection
    (e.g. WiFi), then assuming you have the documentation on the protocols it supports you could use a socket and communicate with its controller.

    Most devices like TVs use Infra-Red (IR), and the iPhone does not have an IR port. Most devices which do not use IR use Radio Frequency (RF) transmissions which are not in the same band as Wifi.

    For IR devices, you could buy the hardware needed plus an iOS app here or here. For RF devices, a quick search did not show up an equivalent product that is off-the-shelf. But I only did a cursory search.

    My SWAG is that your landscape lighting uses RF.
