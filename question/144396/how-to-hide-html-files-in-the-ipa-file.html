How to hide html files in the .IPA file.

I looked in the IPA file of one of my apps, and saw I about 150 of my html files that are in the app. How can I hide them without putting them into an sqlite file that is encrypted? Converting the html files to a js file would take me the rest of the year, so it is not an option.

  • There is no simple solution to what you are trying to do. You're either going to have to hack something gross or hack something really gross. You could write some sort of compile-time hook that converts your assets into some encrypted format, but that will most likely require some native magic in a Titanium Module. The good news is if you write a module that encrypts assets, you could sell it in the Marketplace and make some money.
