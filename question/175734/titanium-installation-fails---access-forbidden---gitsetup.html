    I have installed git separately from git official site. The installation has completed without errors. When installing git you have to do it as administrator on windows. Propably studio setup didn't passed administrator rights to git setup, that's why we end up with 403 forbidden.

    1. Install git from [here] (http://git-scm.com/download/win) with Admin rights,
    2. Run Studio setup. Should end without errors. (I have run both with admin rights)
      Hope this helps, cheers!
