  • Hi Balakrishna,

    There is no need to change at app setting side.. means like ipad.

    you need to change you UI according to your requirement. in alloy there is created one property for identifying it is tablet or mobile device e.g Alloy.isTablet. (use in controller .js file)

    if you need to put condition .tss the use formFactor
    e.g

    "#btnTest":{
    width : 60
}

"#btnTest[formFactor=tablet]":{
    width : 60
}

    if you want to learn more about it [see this](
