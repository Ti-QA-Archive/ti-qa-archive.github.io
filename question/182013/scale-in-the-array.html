Scale in the array

Hello, I need help. This is my code 

var size = Ti.App.globalSize;
var widthSize;
if (size == 3){
    widthSize = '30%';
}else if (size == 4){
    widthSize = '23%';
}else if (size == 5){
    widthSize = '19%';
}
var J = [];
    for (var i = 0; i < size; i++) {
        J[i]=[];
        for (var j = 0; j < size; j++) {
            J[i][j] = Ti.UI.createImageView({
                image : 'lamp0.jpg',
                left : (widthSize * i), // it does not work
                top: (60 * j), // it work, but i need %
                width : widthSize,
                height : 'auto',
                backgroundColor : 'red',
                xindx : i, 
                yindx: j,
             });

Can you help me. With left and width

    Hello,

    The values '30%', '23%' and '19%' are recognized as String so you can't use multiplications. Just store the number (without quotes) in widthSize like this: widthSize = 30; You can add the percentage later:

    J[i][j] = Ti.UI.createImageView({
    ...
    left : (widthSize * i) + '%',
    top: (60 * j) + '%',
    ...
});
