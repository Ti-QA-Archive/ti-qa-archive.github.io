    • You can indeed create games with it however the problem i found was when you destroy box 2d objects they are not removed from memory.

      I had a conversation with someone from the appcelerator market place and they confirmed this issue and stated that it was to be added into future versions.

      The appcelerator Box2d module has yet to be updated.

      So if you created a game with lots of interactivity then you would almost certainly run into memory issues.

      — commented March 22nd 2013 by Gary George
