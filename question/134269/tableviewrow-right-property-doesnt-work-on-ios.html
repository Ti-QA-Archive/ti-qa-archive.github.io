tableViewRow right property doesn't work on iOS

I have a label in a tableViewRow on the right side of the row. On android works fine, but on iOS the label is on the left.
The top and left property works fine on iOS, but not right and bottom. Any idea?
Here is the code, the problem is in var fecha:

function createRow(item,tableData,color){

    var row = Ti.UI.createTableViewRow({
        height: '80dp',
        item: item,
    });
    var texto = Ti.UI.createLabel({
        text: item.titulo,
        color: '#000',
        left: '20dp',
        right: '50dp',
        top: '5dp',
        bottom: '15dp',
        font: {
            fontSize: '16dp'
        }
    });
    var fecha = Ti.UI.createLabel({
           text: item.fecha,
           right: '5dp',
           bottom: '1dp'
    });
    var leftColor = Ti.UI.createView({
           backgroundColor: color,
           height: '100%',
           width: '5dp',
           left: 0,
           top: 0,
           bottom: 0
    });

    row.add(texto);
    row.add(fecha);
    row.add(leftColor);

    return row;
}

    The problem is that you dont have a width or height for your label, so the label are the same width and height as their parent. Thats why you can only see the left and top property move because the bottom and right are moving the label but you dont see it cause your text is align to the left

    this should fix the problem

            var texto = Ti.UI.createLabel({
                text: 'text',
                color: '#000',
                left: '20dp',
                right: '50dp',
                top: '5dp',
                bottom: '15dp',
                height: 'auto',
                width: 'auto',
                font: {
                    fontSize: '16dp'
                }
            });
            var fecha = Ti.UI.createLabel({
                text: 'text',
                height: 'auto',
                width: 'auto',
                right: '5dp',
                bottom: '1dp'
            });

    just set the width and height to auto
