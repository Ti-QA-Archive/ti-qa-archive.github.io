Passing datetime to a fireEvent

Hi,
I have a problem when passing picker.value to events :
I do :

Ti.API.info(picker.value);
Ti.App.fireEvent('startDateSelected', {selectedDate:picker.value});

and for receiving the events :

Ti.App.addEventListener('startDateSelected', function(event) {    
    var endDate = event.selectedDate;
    Ti.API.info(endDate);
}

and I get in the console 2 different date (not too much but different).
[INFO] 2011-10-24 07:00:55 +0000 (sent)
[INFO] 2011-10-24 07:01:35 +0000 (received)

Any idea ?
Regards

  • try assigning the date to a var and then passing it

    var time = picker.value;
Ti.API.info(time);
Ti.App.fireEvent('startDateSelected', {selectedDate:time});
