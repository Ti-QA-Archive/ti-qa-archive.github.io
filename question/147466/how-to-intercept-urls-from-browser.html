How to intercept URLs from browser?

Is there any way to intercept all URLs requested from any browser on Android, iPhone, or iPad, check some rules, and then either halt the web request or allow it to go through? I do not want to use webview, build a custom browser, etc. This needs to work for any browser on the device (lower level than all browsers). I'm thinking maybe something like a proxy server, but I need to be able to lock it down so that all traffic MUST go through the proxy, so that my business rules can be invoked.

I'm not asking anyone to give me code on a platter, just point me in the right direction. I basically need to build a very lightweight, no-frills whitelist manager.

I have already looked in the following places, but I'm not sure I'm looking up the right tree:
http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.Network.HTTPClient
http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.Network.Socket

    Hi

    Just be clear you want to intercept any web traffic so you can choose to let it continue or block it. This must be done so that whatever browser is used on the device (not your app - outside of your app) your intercept can filter.

    You cannot do it on iOS at as everything is sand-boxed. No app can get this low-level. Google is unable with the Crome browser so you get the idea.

    You may be able to create a custom module for Android to achieve it, but there is no direct way in Titanium.
