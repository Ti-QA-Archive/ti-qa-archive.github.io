  • Hello Ram. Since webView is itself scrollable so I think you don't need to scrollView inside a scrollView.
    Use the following structure of code for this: 

    var scroll = Titanium.UI.createScrollView({

        contentWidth: 'auto',
        contentHeight: 'auto',
        showHorizontalScrollIndicator: true,

    });

var web = Titanium.UI.createWebView({
        url: "your url",        // give your url here.
});
scroll.add(web);

    Hope it will help.
