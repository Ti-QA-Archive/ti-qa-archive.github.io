  • It would be helpful if you could also post the "Details" section of the message.

    In my case, the reason was that my Windows user name had non-ASCII characters, so I went to Window-Preferences-Aptana Studio, and changed the Ti SDK directory from C:\Win...\??\Titanium to C:\Titanium, and I moved the original directory to C:\Titanium accordingly.

    After that everything worked, but again, it all depends on the error details.
