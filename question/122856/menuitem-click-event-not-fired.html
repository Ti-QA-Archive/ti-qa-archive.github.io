menuitem click event not fired

hi, consider following code:

var activity = Ti.Android.currentActivity;

activity.onCreateOptionsMenu = function(e){
    Ti.API.debug('on create');
    var menu = e.menu;
    var button = menu.add({title:'myButton'});
    button.addEventListener('click',function(e){
        Ti.API.debug('click event');
    }
};

this code being executed on window open event. the strange thing: the first open all works fine but reopening the window the "on create" is printed. but not the "click event" on button click.
