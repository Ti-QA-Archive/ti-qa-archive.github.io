    $ is reference to the current window, in the alloy code we get the instance of window using $.

    Secondly here if $.table is used it means that table object is been bind to window object:

    Let show you simple example to make you understand better:

    here you can use table.setData(tableData)

    
$ // this is window current instance

var tblview = Ti.UI.createTableView({});

$.add(tblview); // here table is added to window but table object is not added to window object so you can only use

table.setData(tableData);

    here you can use $.table.setData(tableData) 

    
var tblview = Ti.UI.createTableView({});

$.table = table; here table object is added to window object so you can write

$.table.add($.table);

$.table.setData(tableData);
