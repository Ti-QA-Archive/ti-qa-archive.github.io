  • Accepted Answer

    You have to create a new user for each device, i tried putting 2 devices under the same user. When sending a notification it only the first device received this notification.

    So yes you have to do this.

    You can also send a push notification to a specific url, so that you can send a notification from your own webservice for instance.

    Besides appcelerators cloud services there is also urbanairship which is similar to appcelerators push notifications.
