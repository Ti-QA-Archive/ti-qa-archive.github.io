I have a set of Schedule timings as -

08:00 PM,
06:30 AM,
12:01 PM,
03:03 PM,

I need to order them as -

06:30 AM,
12:01 PM,
03:03 PM,
08:00 PM, In chronological order

    Hi Rakesh,

    Try with sorting date object.

    e.g

    var date1 = new Date();
var date2 = new Date(date1.getTime() + 30 * 60000);
var date3 = new Date(date2.getTime() + 30 * 60000);
var date4 = new Date(date3.getTime() + 30 * 60000);

var arrDate = [date2, date4, date3, date1];
Ti.API.info('JSON = ' + JSON.stringify(arrDate));
arrDate.sort(function(a, b) {
    return (a - b);
});
Ti.API.info('After JSON = ' + JSON.stringify(arrDate));

    Hope this is helps you
