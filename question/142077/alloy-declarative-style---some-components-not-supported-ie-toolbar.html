    For what platform are you attempting to build? Ti.UI.iOS.Toolbar is an iOS-only Titanium API. It will throw an error similar to your's if you attempt to build it on Android or Mobileweb.

    If you are building for iOS, can you give some information about your environment? Titanium SDK version? TiStudio version? iOS SDK version? Alloy version?
