Label alignment inside table view

Hi,

I am loading data from webservice to label inside table view. The text is very big . Im having five labels inside the tableview. How to customize the each text such as not to overlap each label text. I have used wordwrap:false and ellipsize:true in android. But still it is showing text in two line. I want it to be text in oneline. In portrait mode as well as in landscape mode. How to acheive this?

  • You can insert in each table view row an image view and on left site a (container-) view with layout='vertical'. In this container you can put your labels with :

    top: 5,
height: Ti.UI.SIZE

    The container has a height of Ti.UI.SIZE

    Hope this help you.
