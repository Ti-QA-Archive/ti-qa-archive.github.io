  • You are having trouble with images, and your solution is to unmount the sdcard in the android emulator?

    Titanium copies the APK to the sdcard from which it initiates an INSTALL of the APK to the main system. Images packaged with your application are stored on the main system, not on the sdcard, unless you specifically copy them over.

    Your code segment means nothing without telling us where your image (background.png) begins its life. Is it located in the Resources directory? Is it in a subfolder? In what file does the code segment above live? Where does that file reside?

    If you are using subfolders to sort out your components, consider:

    backgroundImage: '/background.png' //If in Resources

    backgroundImage: '/path/to/background.png'  //If someplace else
