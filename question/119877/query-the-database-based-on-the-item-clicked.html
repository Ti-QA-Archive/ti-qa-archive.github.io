Query the database based on the item clicked

UPDATE– I was able to solve this problem.

I am working on an iPhone app that uses a local database. I can query and display the category names in a TableView. I want to be able to click on a category and query the database for names in that category. 

 var listtips = Titanium.UI.createTableView({
        data:data,
        searchHidden:true
        });
        listtips.addEventListener('click', function(e)
        {
            if (e.rowData.test)
            {
                var win = Titanium.UI.createWindow({
                    title:e.rowData.title
                                     /* I'd like to put a query like this:
 db.execute('SELECT bName FROM mktplc WHERE category = "the title they clicked" */
                    });
                Titanium.UI.currentTab.open(win,{animated:true});
            }
        });

But I don't know how or exactly where my query needs to go. Can someone help me?

    save the title as a variable and then your SQL statement should be something like this..

    var title = 'title';
//the questionmark is a place holder and the variables go from left to right (if you have multiples)
query= db.execute('SELECT bName FROM mktplc WHERE category =?', title);
