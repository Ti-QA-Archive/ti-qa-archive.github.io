    • Not all test apps in the alloy repo will work for all platforms. I've tried to indicate that where ever possible. This is just the nature of the fact that some will do certain things that other platforms don't support. In this case mobileweb is not supported because it does not support the SQLite adapter, which is critical to many of the model-based test apps.It would probably help to know the goal of your app in order to better assess if you should be using models and what type of model you should be using.

      As for alloy fugitive, that app is not maintained by appcelerator so I can't really account for its current functionality with respect to Alloy.
