Android Array, WTF ? (debugging : java.lang.object)

My problem is very simple but yet unexplainable.
Here is a simple code example :

var result = [];
Ti.API.debug(result);

In the console, I get :

[DEBUG] [2,1310] [Ljava.lang.Object;@43f853e8

Everything seems to work fine, I can read and write my array. But I can't use Ti.API.debug to view the content of it.

I tried to create my array with var result = new Array(); resulting in the same bug.

I tried to delete the build folder, resulting in nothing different.

I tried to use an older version of the Titanium SDK, resulting in nothing different neither.

I tried to run a blank app with this piece of code, and it just did the same.

My configuration :

  • Mac OSX 10.6.8
  • Titanium SDK 1.7.5
  • Android 2.2 with APIs in Simulator with HVGA screen

    That's fixed in new SDK versions. Anyway, use "JSON.stringify" prior to printing out.

    Ti.API.debug(JSON.stringify(result));
