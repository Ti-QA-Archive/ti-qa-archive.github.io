animation not performed correctly

hi I have 2 view groups, I want to animate them left to right so only one view group is shown on the screen at a time (the other goes out of screen) and vice versa on an iPhone screen.

I assign an initial value to view group 2 so that its off the screen on the right side, when button is clicked on vg 1, it'll animate vg 1 off to the left side and animate vg 2 onto screen.

XML

    <Window platform = 'ios'>
        <NavigationGroup id="nav">
            <Window>
            <Require src = 'ViewFacebookLogin' id = 'facebookLoginPanel'> // view group 1
            <Require src = 'ViewMBDLogin' id = 'myBigDayLoginPanel'> // view group 2
            </Window>
        </NavigationGroup>
    </Window>

ViewMBDLogin.tss

'#container':{
    left: 319.5 // setting vg's initial value and make it hiding on the right
}

onClick Method

    vg1.animate({

        left: 320, // off screen to the left
        duration: 300

    }, function(){

        mbdLoginPanel.animate({
            left: 0,
            duration: 300
        })
    });

notice how I set vg 2's initial value to 319.5 instead of 320 (completely off screen), if I set it to 320, the animation will work halfway by moving vg1 off screen, but vg2 will not appear. Interestedly if I set it to 319.5, then it would work.

what am I doing wrong here? any suggestion is appreciated!!

thanks!!
