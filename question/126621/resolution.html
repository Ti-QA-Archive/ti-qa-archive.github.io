When designing an app for iPhone (3&4), how do you deal with the different resolutions (640x480 vs 960x640)? For example if I want to create a background for a window, what resolution do I set for this background?

  • Accepted Answer

    iPhone uses a 320 X 480 resolution, and when setting distances between views you will always use these numbers (it will auto double for the retina display).

    For images, you have to add @2x to your iPhone 4 image.

    So if you have a 30x30 image named myButton.png, make a 60x60 version and name it myButton@2x.png

    The choosing what image to use happens automatically, so you would in this example set your image property to myButton.png, NOT myButton@2x.png.

    Make sense?
