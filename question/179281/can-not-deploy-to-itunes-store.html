    I have seen two reasons as to why the menus are greyed out.

    The first is because a JS, XML, TSS or similar file hasn't been selected within the project explorer. Expanding the project and selecting one of these types of files normally activates the menus and populates the options according to the project type and deployment targets in the TIAPP.XML.

    The second is because the project natures, as you stated aren't correct. Amend them under Project Natures, apply the changes and then try restarting Titanium, then selecting a file as per first option.

    Hope this helps.
