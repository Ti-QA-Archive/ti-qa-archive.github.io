  • Are you using Ti.include()? If so, you need a leading slash on your paths so Titanium knows to use the Resources folder as the root. You do not have to do this with require(). Also, you should be using require() instead of Ti.include() anywhere possible.

    var moment = Ti.include('/alloy/moment.js');
