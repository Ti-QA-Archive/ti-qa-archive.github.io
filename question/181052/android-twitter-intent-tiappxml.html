Android twitter intent tiapp.xml

Titanium SDK: 3.5.0
Platform: Android
Device: Android device

Hello i found this code to share a twitter message trough an Android intent:

                    var intTwitter = Ti.Android.createIntent({
                        action: Ti.Android.ACTION_SEND,
                        packageName: "com.twitter.android",
                        className: "com.twitter.android.PostActivity",
                        flags: Ti.Android.FLAG_ACTIVITY_NEW_TASK,
                        type: "text/plain"
                    });

                    intTwitter.putExtra( Ti.Android.EXTRA_TEXT, "http://www.google.com with some message");
                    Ti.Android.currentActivity.startActivity(intTwitter);

But i keep getting the error: "Unable to find explicit class {com.twitter.android/com.twitter.android.Postactivity}"

Do i need to put something in my tiapp.xml? I never workes with intents before.

    The error is accurate… it won't work if the user doesn't have Twitter installed… or if Twitter changes the package name of the post intent.

    Check out the shareToTwitter module.
