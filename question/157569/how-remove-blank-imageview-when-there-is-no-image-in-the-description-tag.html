  • Hi harsha kumar,

    You can use condition when you will set your description. If it gets image it will set but when it will not get image it will set discretion from the start point. Like this code.

    var title = Ti.UI.createLabel({
                        color : '#000',
                        font : {
                            fontFamily : 'Arial',
                            fontSize : 15,
                            fontWeight : 'bold'
//here set text that means description.                        },
text : items.item(i).getElementsByTagName("title").item(0).text,
//here check image empty or not if empty set left 5 or 65
left : items.item(i).getElementsByTagName("image").item(0).text.length != 0 ? 65 : 5,
                        top : 0,
                        width : 360
                    });
