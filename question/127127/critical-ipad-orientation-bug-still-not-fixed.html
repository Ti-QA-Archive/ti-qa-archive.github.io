Hi

I just have to get this out. I've been waiting for months for the iPad orientation bug to be fixed. So when 1.7.3 was released I thought: "Yes, finally!" but I was quickly dissapointed.

  • Hello -

    I just tried to duplicate this in 1.7.3 as i had been having problems previously as well.

    I was able to get this to work fine in 1.7.3 but I had to make a few adjustments. Try this and see if it works for you.

    in tiapp.xml

    
<orientations device="ipad">
    <orientation>Ti.UI.LANDSCAPE_LEFT</orientation>
        <orientation>Ti.UI.LANDSCAPE_RIGHT</orientation>
</orientations>

    now within your code where you create the main window remove any references you may have to orientationModes.

    making that one little change did the trick for me.

    Good luck and post if this worked for you.
