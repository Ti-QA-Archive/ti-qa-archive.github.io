Passing data into a mapview

Right now I have an app that will open a mapview and fetch annotations from a remote database. That seems to be working perfect. The problem I'm having now is that I want to have a "Filters" button on the top right nav that will open a new window with a defined form for filtering the data from the database. When the use makes their filters they will click a button that should close the filters window and reopen the mapview and fetch the data again with the filters applied to the XHR request.

Any idea on how to pass data back and forth between a window and a mapview for what I'm looking to do?

    What youll want is to pass custom properties between windows. Check out custom_properties.js for that. Also if you want to pass data from the filter window back to the mapview you could do something like below (not exact syntax).

    filterpage.js
ti.app.fireevent('filterdata', data);


mapview.js
ti.app.addeventlistener('filterdata', function (e){
do something with e.data
});
