    Hi

    If you mean the top of this image;

    Then you can use the features already built-in;

    var view1 = Ti.UI.createView({ backgroundColor:'#123' });
var view2 = Ti.UI.createView({ backgroundColor:'#246' });
var view3 = Ti.UI.createView({ backgroundColor:'#48b' });

var scrollableView = Ti.UI.createScrollableView({
    views: [
        view1,
        view2,
        view3
  ],
  showPagingControl: false
});

win.add(scrollableView);

    The example views used here can be images instead or any number of nested elements for example;

    view1
  image
  label

    If you are not sure let me know.
