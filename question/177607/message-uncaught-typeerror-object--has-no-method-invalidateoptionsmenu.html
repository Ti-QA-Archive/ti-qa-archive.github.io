    You are getting this error because have call this line in the code:

     activity.invalidateOptionsMenu();

    So activity.invalidateOptionsMenu method is not define any where and you are tring to call it therefore you are getting error message as no method 'invalidateOptionsMenu'

    Just comment the above line and your code will work fine without any issue.

  • Hi

    I did a quick check of the docs for invalidateOptionsMenu(). This tells you to check Changing The Menu. The following information can be found on this page:

    On Android 3.0 and higher, the options menu is considered to always be open when menu items are presented in the action bar. When an event occurs and you want to perform a menu update, you must call invalidateOptionsMenu() to request that the system call onPrepareOptionsMenu().

    This would suggest that you need to define 'onPrepareOptionsMenu' in your activity.

    I am not saying this is the solution, but maybe try adding this code above activity.invalidateOptionMenu() to see if it continues to throw an error?

    activity.onPrepareOptionsMenu = function(e) {
    var menu = e.menu;
    var menuItem1 = menu.add({
        title: 'Added Test Item',
        showAsAction: Ti.Android.SHOW_AS_ACTION_NEVER
    });
};

    Hope this helps
