Navigation control - sharing variables and controls

Hi,
I've implemented the cross platform navigation control as shown in Forging Titanium Episode 2.

It's working fine, but I was wondering how can I share controls on one window with another. I'm building a navigation based registration, where each window is a step in the registration. In window 2 I need to access the textField controls from the first window - how can I do that?

Thanks

  • You can use cross-app events.
    In the first window set an event that will be triggered in the second window, and so on.

  • There could be a second option. Do you really need to access the actual objects on the prior windows or do you just need the values? It seems to me that the text field values aren't going to change when they aren't on the screen, so you could just store the values of the text fields as you move from screen to screen. If they do change, then you can just store the value that they change to and update each screen as it comes back into view.
