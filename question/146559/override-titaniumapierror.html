Override Titanium.API.error

I'd like to override Titanium.API.error so I can have error sent elsewhere… I've tried Ti.API.error = function() { … } but it doesn't work. Is there a way to do this in Javascript?

Thanks!

    You can't override the error method this way. Just create your own logging class and send you're errors there rather than to Ti.API.error();
