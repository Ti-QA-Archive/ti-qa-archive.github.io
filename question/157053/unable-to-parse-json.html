  • I see from your code that you are attempting to access your server using the http:&#x2F;&#x2F;localhost URL. This will not fly since localhost represents the actual device. Even though you are running through the emulator on the same machine.

    My advice to you is to access your url using the actual IP address of your server.
    A second advice would also be to try to access the URL using the device's web browser first. Once you have established that your device (emulator) *can access the server, then you can start testing with your application.

    Hope this helps.
