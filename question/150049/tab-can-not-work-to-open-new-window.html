    • Hey Muhammad,

      Just see. In Titanium we never add any tabgroup to any window because A tab group can contain one or more Tab objects, each of which has an associated tab control that is used to bring it into focus.
      You can add tabs to the group using addTab, and programmatically switch to a specific tab using setActiveTab and Tab contain the Window it mean tabgroup contain the number of windows.

      I want to say that Tabgroup have high Priority compare to Window.
      So thats y we can not add tabgroup to any window and also not opened in child window and in your code you use a currentWindow which is not good. You can use currentWindow() instead of currentWindow.
