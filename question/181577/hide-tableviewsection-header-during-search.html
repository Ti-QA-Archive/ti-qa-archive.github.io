Hide TableViewSection Header during search

I have a list of TableViewSections which correspond to product categories (beer, moscato, rum, etc.). Under each header, the products are listed that fit that category. The user can also search for the specific product(s) that they want which dynamically changes the view to only show the product(s) containing the searched for substring. Unfortunately, the headers (beet, mosacto, run, etc.) do not go away if there are no products below it which results in around 10 headers being shown without any products. How can I hide the headers if there are no products underneath them to display? It works automatically in iOS, but this problem exists in the Android version.
