    hi mohamed,

    the mistake is visible in this line

    /Users/mohamedbadr/Library/Application Support/Titanium/mobilesdk/osx/3.0.2.GA/iphone/ios-sim [ERROR]

    you have ti sdk 3.2.2 installed, but your project uses 3.0.2, update sdk-version tag in your tiapp.xml 

    <sdk-version>3.2.2.GA</sdk-version>

    then clean your project

    Thanks
