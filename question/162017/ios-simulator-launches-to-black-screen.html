iOS Simulator Launches to Black Screen

Hello. I am working on an iOS application using the latest version of Titanium Studio (3.2). My problem is t hat since I upgraded to the latest version my app launches in the simulator but only displays a black screen. At first I thought it was a problem with the Navigation Group, which has been deprecated and is now called Navigation Window. I changed the code to reflect this, but I still get a black screen. I'm really stumped on where to begin to fix this problem, since I don't have a clue as to what is causing it. I know I haven't given very much to go on, but I'd appreciate any ideas that the community could provide to help me fix this. Thank you!

    Anytime you have a setup thats 'pretending' to something its not, ie, you have an x86 computer app thats pretending to be an iphone, you have the potential for things to unwind.

    My first go to fix for simulator issues is to reset the content and settings.
