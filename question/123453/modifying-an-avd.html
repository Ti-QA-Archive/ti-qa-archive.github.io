Modifying an AVD

I need to modify the AVD that titanium is choosing to emulate with. Everytime I clear the entire AVD list, by default, titanium creates a new one using ID 8. How do I change this so it uses ID 1?

— asked July 29th 2011 by Eric Dean

  • Accepted Answer

    The id is based upon the AVD ids in the android SDK.
    It applies to the android base sdk, whether it has the Google APIs installed, etc.
    Choose a different android level to target and you will get a different id.

    Show possible AVD targets

    adb list target

    Will show existing AVDs

    adb list avd

    You want id1 which is:

    id: 1 or "android-3"
     Name: Android 1.5
     Type: Platform
     API level: 3
     Revision: 4
     Skins: HVGA (default), HVGA-L, HVGA-P, QVGA-L, QVGA-P

    Where as Titanium is handing you id 8:

    id: 8 or "android-8"
     Name: Android 2.2
     Type: Platform
     API level: 8
     Revision: 2
     Skins: EVO 4G, HVGA, QVGA, WQVGA400, WQVGA432, WVGA800 (default), WVGA854

    Target Android 1.5 and you will get your id 1
