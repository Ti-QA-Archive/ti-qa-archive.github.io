I am wondering where i can find the database file created when running the android app.

I found that it is under /data/data/NAME_APP/db.sql. But Where is this /data file? I can't seem to find it anywhere?

    The file is in the emulator image itself. you can access it using the Android SDK shell.

    Use the adb devices command to list all devices connected right now (this list includes the emulators that are running).

    Once you have the name of the device you want to access, simply use it in the adb shell command.

    Here is an example (here, the device's name is emulator-5554)

    $ adb -s emulator-5554 shell
# sqlite3 /data/data/NAME_APP/databases/your_db_file.db
SQLite version 3.3.12
Enter ".help" for instructions
.... enter commands, then quit...
sqlite> .exit

    • Use the ddms (Dalvik Debug Monitor,located in /your/sdklocation/tools).
    • Select your device or emulator in the left pane.
    • From the menu, choose Device, then File Explorer…
    • From there you can browse down into /data/data/your.app.id to look around.
    • You can upload/download/delete files in those folders as well
    • The databases get their own folder and the applicationDataDirectory gets a folder and the applicationTempDirectory gets a folder. If you haven't written to them, they probably won't exist.
