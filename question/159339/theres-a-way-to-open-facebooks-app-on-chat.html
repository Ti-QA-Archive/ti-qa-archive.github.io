    I solve this problem like this:
    Firstly, on Facebook developer site I created my app and retrieve appid
    Then I used Facebook module and login button into app and finally used this code

    Ti.Platform.openUrl("fb://messaging?id=ID_OF_PERSON_TO_CHAT")

    Now, after I logged in via Facebook button my app opens facebook Messenger and I can chat with my friend.
