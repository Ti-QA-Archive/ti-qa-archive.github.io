Keeping the <Back label on a Navigation Window

Using iOS Navigation Windows, is there a way to prevent the Back function from inheriting the Title from the Previous window. I am trying to keep the generic "Back" text. I'm sure there is a simple way, I just haven't found it. Can anyone help?

    Give the backbuttonTitle property in window like this and make the message customized

    
    var win = Titanium.UI.createWindow({
        backgroundColor : 'white',
        title : 'vålj produkt',
        backButtonTitle : "Back"
    });
