  • It looks like you're making an app with umlauts. We've made some swedish apps and figured out a workaround for this issue (we've reported this issue in helpdesk, hopefully it will be supported by Titanium soon).

    What we do is that we firstly create the app in Titanium Studio with only a-z characters. When we are done and everything seems to work in iOS Simulator, we open the project build folder in Finder and open the generated Xcode-project.

    In there we can change the "Bundle display name" in Info.plist so the correct name is displayed. We also build and archive the app via Xcode (archive lets you upload the app to the App Store).
