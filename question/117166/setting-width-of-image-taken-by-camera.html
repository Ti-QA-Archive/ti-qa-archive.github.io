Setting width of Image taken by Camera

Hi

I used the code from the URL for enabling Camera feature and displaying it in Image View(Ref: http://developer.appcelerator.com/question/95431/imageview-load-photo-gallery–camera_. However it does not show Image in proper width and height, specially width is not set at all though I believe the width of picture is more than 300px which I set for ImageView.

Visit the link: http://www.flickr.com/photos/kadnan/5522469346/

As you can see the empty space around Image within Imageview(with Pink border)

    When you set the "image" property for your ImageView, set the "width" and "height" properties as well. Such as something like:

    imageView.image = event.media;
imageView.height = 300;
imageView.width = 300;

    So when you set the image, you also immediately rescale it. I have this problem as well and there seems to be a problem that the image rescales the ImageView to whatever it's own dimensions are, so you just have to re-define the width & height properties so it resizes.
