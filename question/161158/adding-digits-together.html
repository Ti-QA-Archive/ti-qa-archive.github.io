Adding Digits Together

Hi, how do i adding digits together to get a single digit? For example say I input a number 213 and the result should be 2+1+3 = 6

This javascript would do it, but how can I modify it to work on Titanium?

function addDigits(s) {
var l = s.length;
var w = 0;
for (var i=0; i < l; i++) {
w += +s.charAt(i);
}
if (w > 9) w = addDigits(w.toString());
return w;
}

Thank you.
Rishi
