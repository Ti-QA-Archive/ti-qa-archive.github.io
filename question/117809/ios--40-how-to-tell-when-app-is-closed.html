Is there an event that's fired?

I need to know when the app is being forced to close so I can save the document currently being worked on.

  • Hi,

    if you have not touched the plist to make the app totally exit when the home button is hit then this should help you

    Titanium.App.addEventListener('pause', function(){
 Titanium.API.info('paused');
});

Titanium.App.addEventListener('resume', function(){
 Titanium.API.info('resumed');
});

    as by default the apps will move to paused state when the home button is hit…

    Hope it helps …
