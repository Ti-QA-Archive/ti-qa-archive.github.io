Android Emulator on Mac OSX Error

Every time I try to run my app (or any app, including a blank project) inside of the Android emulator, I get the following error:

[ERROR] : Build process exited with code 1

[ERROR] : Project failed to build after 242ms

[ERROR] Application Installer abnormal process termination. Process exit value was 1

I'm new to Titanium, but I configured the Android SDK to include 4.3 and 4.2.2. I added a custom virtual device in the Android VDM. I tried running 4.3 [x86 & armeabi], Google APIs (Android 4.3), and all versions of 4.2.2.

The Android SDK is in my Applications folder.

  • Hi

    The Google Android Emulator is notoriously unstable, but here are some things to try;

    • Inside the Android SDK Manager
      • Make sure you have downloaded anything in the Tools folder
      • In the Extras folder make sure you have the "Android Support Library"
      • Install as many API levels as yuo are going to use and all elements in each of those
    • In your tiapp.xml add the following entry (manually edit this file - backup first);
    <property name="ti.android.fastdev" type="bool">false</property>
    • If it already exists, toggle the value.
      • Make sure you use Project > Clean whenever you hit an error, sometimes old code does not get removed.
      • Make sure you are always using the latest Ti SDK - currently 3.1.2 GA, set this in the tiapp.xml choose the latest SDK in the Titanium SDK dropdown, if you do not have 3.1.2 use the help menu to download the latest version.

    Things still may not work after this - but at least you will have tried much more.
