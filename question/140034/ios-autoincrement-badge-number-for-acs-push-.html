[iOS] autoincrement badge number for acs push ?

Hi , my ACS push notification already works.
I can set "badge": 1 in push notification and later reset it to 0 on app startup.

But how can i auto increment "unreaded" badges, like on facebook app or mail app?

  • Use Ti.App.Properties to store an 'unread' count. Then use that (incremented) to set the badge count.

    Then clear / set to null when the app is opened.

    See : http://docs.appcelerator.com/titanium/2.1/index.html#!/api/Titanium.App.Properties
