Network and resumed event

Hi, in my iPad app i make a get http request on a page clicking on a button. I need to do the same get http request when the user comes back from the lock screen, in the resume event, but in this case it doesn't work, here is some example code:

Titanium.App.addEventListener("resumed",function(e){
var xhr = Titanium.Network.createHTTPClient();
url3="www.mysite.com/myusername/mypassword";
xhr.open('GET', url3);
xhr.send();
xhr.onload=function(response){
url4="www.mysitewithusernameandpassword.com"
}
xhr.onerror=function(){
alert("Connection Error::"+ this.statusText);
}
});
i used the same code in the click event of a button and it works, but in the resumed event it does not, do you know why?Thanks
