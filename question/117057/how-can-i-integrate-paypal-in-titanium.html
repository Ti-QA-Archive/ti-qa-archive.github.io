  • You could implement your own PayPal integration module. If you're already programming in Titanium then connecting what you're building with PayPal should be very straightforward. I recommend exploring their Express Checkout system, which can be implemented entirely through simple HTTP calls and browser redirects. https://www.x.com/docs/DOC-1122?ct=t(x). If you want something ready to go then I'd examine the licensed option from Appcelerator. There may also be an example in KitchenSink worth reviewing.
