9Patch Image in iOS

I'm trying to figure out how to get 9patch png files, or some equivalent, to work on iOS. 9patch works wonderfully with android, but I cannot seem to find a comparable equivalent in iOS. All it does is stretch the image out. Any information would be appreciated. I would really prefer to not have to distribute different images for every variation of iOS screen resolution…

Any information would be greatly appreciated.

    9patch is android only.

    For iOS, look at the backgroundTopCap and backgroundLeftCap properties of the view you are working with.

    • Thanks Stephen, I got a similar effect to 9png with code similar to…

      mylabel.applyProperties({
    backgroundTopCap: "40px",
    backgroundLeftCap: "40px"
 });

      It looks like the top cap and left cap start from the bottom right, so these pixel values are offsets from that point. If your image is larger than the size of the component it will get squished. I need to play with it a little more but it looks like a good solution.
