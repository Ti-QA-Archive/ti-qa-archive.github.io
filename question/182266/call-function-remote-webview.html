Call function remote webview

I want to call a function onload in a remote webview to set a variable on the webview, for now i have the following code:
index.xml

<Alloy>
  <Tab title="Livestream" onClick="initialize">
    <Window>
       <WebView id="webview" url="http://urltowebsite.com"/>
   </Window>
  </Tab>
</Alloy>

And the controller of index.js:

$.webview.addEventListener('load', function(){
  var data = "Hello world!";
  $.webview.evalJS("foo('" + data + "'););
});

and in the index.html of the remote webview: 

<script type="text/javascript">
  function foo(data) {
    alert(data);
  }
</script>

  • Did you make a copy and paste error in the index.js code? You didn't close your second set of double quotes in the evalJS() call.

    Your code looks correct to me.

    If it's still not working, keep stripping things down to the bare minimum. For example, just make your code like this:

    remote HTML:

    <script type="text/javascript">
  function foo() {
    alert("Hello, world");
  }
</script>

    index.js: 

    $.webview.addEventListener('load', function(){
  Ti.API.debug ("HTML loaded, calling foo()...");
  $.webview.evalJS("foo();");
});

    This way, you don't have to worry about passing the variable across the boundary to the webview, and you can make sure that the basics of evalJS() are working. Note that I also put a debug statement into the load event listener – make sure that this is really getting called.

    Depending on the contents of the webview, I imagine it is possible that the load event might take a lot longer to fire than you would expect (if the page pulls in lots of external JS like ad units, the actual load event might happen 15-20 seconds after the page looks like it is loaded).
