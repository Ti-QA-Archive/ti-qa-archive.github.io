Saving a PNG screenshot to camera roll changes filetype to JPG?

I'm saving a screenshot to the file system then saving to the camera roll but for some reason the savetoPhotoGallery function saves my PNG as a JPG, is it even possible to save PNG's to the devices camera roll?

My app is intended only for iPad. This is the code I'm using:-

Ti.Media.takeScreenshot(function(e) {
                filename = 'screenshot.png';
                var f = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory, filename);
                f.write(e.media);
                Ti.Media.saveToPhotoGallery(f.read());
            });
