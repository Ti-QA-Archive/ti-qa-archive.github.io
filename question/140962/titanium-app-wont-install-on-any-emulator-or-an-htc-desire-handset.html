Titanium App won't install on any Emulator or an HTC desire handset

I'm getting the an error when I try to install my app on any of the following:

Google API 2.1 Emulator
Google API 2.2 Emulator
Google API 2.3.3 Emulator
HTC Desire (factory) handset

The error is:

Failure [INSTALL_FAILED_MISSING_FEATURE]

My app uses the ti.Bracode plugin: I tried removing this, but I still get the error. Apart from that, app just uses a webview and an SQL Lite database. I'm using KitchenSicksample code to make a four tab single app. I've not done anything drastic or incredibly clever.

In case it helps, here's the uses-feature section of my AndroidManifest file. Googling elsewhere suggests this might be where I need to look?

<uses-feature android:name="android.hardware.camera"/>
<uses-feature android:name="android.hardware.camera.autofocus" android:required="false"/>
<uses-feature android:name="android.hardware.camera.flash" android:required="false"/>
<uses-feature android:name="android.hardware.screen.landscape"/>
<uses-feature android:name="android.hardware.wifi" android:required="false"/>
<uses-feature android:name="android.hardware.touchscreen" android:required="false"/>

  • I use scandit barcode scanner and occurs the same. I've gotten it to work by adding the following uses:

    
<uses-feature android:name="android.hardware.camera" 
    android:required="false"/>
<uses-feature android:name="android.hardware.camera.any" 
    android:required="false"/>
<uses-feature android:name="android.hardware.camera.flash" 
    android:required="false"/>
<uses-feature android:name="android.hardware.flash" 
    android:required="false"/>
<uses-feature android:name="android.hardware.camera.autofocus" 
    android:required="false"/>
<uses-feature android:name="android.hardware.autofocus" 
    android:required="false"/>

    If your AndroidManifest.xml in build directory add this lines like "required" automatically omitting your tiapp.xml lines, you'll have to build a custom AndroidManifest.xml in platform/android directory with the correct lines above.
