Using Absolute URL with HTTPClient

I am trying to use HTTPClient with Titanium SDK 1.7.2. I am attempting to POST to a php file on the internet using an absolute url, but it is tacking on the local file system to the front of it. In other words, here is my code (fake URL inserted for security reasons):

loggedCheck.open("POST",'www.location.com/post_logged_check.php');
Ti.API.info("location:" + loggedCheck.location);

And it is printing out the location as: file://localhost/Users/jonathanjekeli/Documents/Titanium%20Studio%20Workspace/CruTrack/Resources/www.location.com/post_logged_check.php

How can I fix this to use the absolute URL of the php file on the web?

Thanks!

    Try with http:// before address

    loggedCheck.open("POST",'http://www.location.com/post_logged_check.php');
Ti.API.info("location:" + loggedCheck.location);
