Compression module probem under Android

Hi,
I'm trying to use the Compression module under an "only" iPad app.
SDK 1.8.2 on OSX
in tiapp.xml I have

<deployment-targets>
        <target device="mobileweb">false</target>
        <target device="iphone">false</target>
        <target device="ipad">true</target>
        <target device="android">false</target>
        <target device="blackberry">false</target>
    </deployment-targets>

and

<iphone>
        <orientations device="iphone">
            <orientation>Ti.UI.PORTRAIT</orientation>
        </orientations>
        <orientations device="ipad">
            <orientation>Ti.UI.PORTRAIT</orientation>
            <orientation>Ti.UI.UPSIDE_PORTRAIT</orientation>
            <orientation>Ti.UI.LANDSCAPE_LEFT</orientation>
            <orientation>Ti.UI.LANDSCAPE_RIGHT</orientation>
        </orientations>
    </iphone>
    <android xmlns:android="http://schemas.android.com/apk/res/android"/>
    <modules>
        <module version="1.1">ti.airprint</module>
        <module version="1.7">ti.pageflip</module>
        <module version="1.0">ti.compression</module>
    </modules>

And I get an error :

Third-party module: ti.compression/1.0 missing library at /Library/Application Support/Titanium/modules/android/ti.compression/1.0/libti.compression.a

It seems that it try to find an Android Module (but here I use only an iPad version).
And Android and IOS version number are different.

Any idea ?
Regards

    Change to this:

    <module platform="iphone" version="1.0">ti.compression</module>
<module platform="android" version="2.0">ti.compression</module>
