Documentation error

As I've tried to delete a custom object from my HTML project, I had to find out that the documentation at http://cloud.appcelerator.com/docs/api/v1/custom_objects/delete is wrong. As stated in the "required parameters" section you need to provide the id. In the code example you can find "fields". Replace that with "id" and it works. 'class_name' doesn't seem to be required, since you already provide it with your call to "objects/<classname>/delete.json".

Please update the documentation.
Markus
