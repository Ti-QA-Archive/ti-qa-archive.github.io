About separations (TabGroup.js, Window.js, etc.) introduced in appcelerator-titans app

Link: git repo of appcelerator-titans

I'm trying to copy this CommonJS template for my app, which does this thing: main window with 2 buttons, each button opens a different tabgroup, each tabgroup has some some windows inside. I've set for these windows a leftNavButton that should do tabgroup.close(), but I don't know how to do it: because of the separation introduced by modules, I don't know how to target the tabgroup.

Summary:

bootstrap.js
  new HomeWindow,
  new HomeView (two buttons),
  homeWindow.add(homeView);
  homeWindow.open();

  click one of the buttons in homeView:
    new TabGroup,
    tabs.open();

      one of the windows created inside tabs:
        new win1,
        new win1View,
        win1.add(win1View);
        win1.leftNavButton = myBackButton <-- how to add click
