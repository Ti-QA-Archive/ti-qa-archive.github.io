How do I remove old ACS project reference?

I have some rogue ACS projects that were originally created to test node.js and are now throwing errors. See https://jira.appcelerator.org/browse/TC-2204 for reference.

After discovering that the latest version of node.js was not working (at that time) I reverted to an older, stable version of node but the ACS packages I'd created in Studio continue to show error pop up windows.

I have not added any code/service to the packages, two packages were unpublished, one published once the correct node.js version was restored and none have ever been connected to application project files or cloud instances. They are empty, unconnected files that upon closing, changed the nature of my error alerts.

My question is, how do I delete any reference to the packages as well as delete the ACS files?
