Titanium object missing in webview when refreshing the page

I have a webpage inside a webview and using some Titanium calls inside the page (events, check if I'm mobile with typeof(Titanium))
But everything I reload the page to update some data the Titanium object is undefined inside the webpage!

Is there a way to initialize the Titanium object again?

iOS, TI 2.1.3

    Here is a little trick you can do to inject the limitted WebView Ti object into any page. Just use the following code web.html = web.html.

    Here is a working example:

    var win = Ti.UI.createWindow();
var web = Ti.UI.createWebView({
    url: "http://www.appcelerator.com"
});

var injected = false;
web.addEventListener('load', function() {
    Ti.API.info("LOADING");
    if (!injected) {
        Ti.API.info("INJECTING");
        web.html=web.html;
        injected = true;
    }
    web.evalJS("Ti.App.fireEvent('injected');");
});

Ti.App.addEventListener("injected", function(){
    Ti.API.info("I'M HIT");
});

win.add(web);
win.open;
