  • Well you got sqlite available, so dig into how to storing binary data into that, and then you just have to store the PaintView into an image, lets say with an button:

    
var paint_view = paint.createPaintView({
    strokeWidth : 5,
    strokeColor : '#000000',
    useBezierCorrection: true,
    backgroundColor: '#ffffff',
    width: 'auto',
    height: 'auto'
});

btn.addEventListener('click', function(e) {
    paint_view.setImage(path_to_image);
});

    That should give you a head start.
