The window has 2 views. Each view has few buttons.
The on click event on the buttons work normally (if I don't specify any layout)

But the on click events of the buttons doesn't work inside the vertical layout.
If I comment the above line of code, the click events work again…

  • i had the same issue and this code below fixed my issue…i bet u can try it without the proxy-view "$win"

        var $winMain = Ti.UI.createWindow({
        navBarHidden : true,
        //layout: 'auto',
        backgroundColor:'#FdFD00',
        height:'50%',
        top: 600
    });

    var $win = Ti.UI.createView({
        left: 10,
        right:10, 
        layout: 'auto',
        height:'100%',
        backgroundcolor: '#FF00FF'
    });
    $win.addEventListener('click',function(e){return false;});
    $winMain.add($win);
