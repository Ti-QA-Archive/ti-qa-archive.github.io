App Freezing (Hang) when i make big SQL queries

Hi,

I have a desktop app that make a query to my Titanium SQL Database (500+ datasets). I put out the content from the database to my DOM (i use jQuery Datatables) so my app freezes when i make the query and that is bad. How can i refresh the UI to show a Loading or so? In C++ there was a method named Application.ProcessMessages() is there a similar one in JS or Titanium?

    Your best bet would be to display a loading dialog or activity indicator of some sort before querying your database. Sounds like your users are going to be waiting no matter what, so be sure to give them a visual cue while loading.
