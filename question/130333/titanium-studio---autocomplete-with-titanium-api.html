Titanium Studio - Autocomplete with Titanium API

I checked out an SVN project, which i then enabled with "project natures" to be "titanium mobile". This allowed me to build the project as a mobile app. But now i can't use auto-complete with any of the titanium API keywords. I do receive the JS core keywords.

Under properties i do notice that the other projects have "Studio Unified Builder" under "builders". The current SVN project does not. If i turned this off in my other projects, then auto complete fails as well.

How do i add "Studio Unified Builder" to the project that doesn't currently have it? Or any other ideas on how to get auto complete to work?

    Edit the hidden .project file as followed:

    <?xml version="1.0" encoding="UTF-8"?>
<projectDescription>
        <name>KitchenSink</name>
        <comment></comment>
        <projects>
        </projects>
        <buildSpec>
                <buildCommand>
                        <name>com.aptana.ide.core.unifiedBuilder</name>
                        <arguments>
                        </arguments>
                </buildCommand>
                <buildCommand>
                        <name>com.appcelerator.titanium.core.builder</name>
                        <arguments>
                        </arguments>
                </buildCommand>
        </buildSpec>
        <natures>
                <nature>com.appcelerator.titanium.mobile.nature</nature>
                <nature>com.aptana.projects.webnature</nature>
        </natures>
</projectDescription>
