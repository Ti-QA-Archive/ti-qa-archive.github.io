Viewing build in xCode

Hi All,

I am interested in viewing some of the apps I've build in Titanium within xCode so I can get an idea how what I've built looks in Objective C. (Not sure it works exactly like that).

I've opened the xCode Project within the build folder in xCode and all I see is:

Classes
Thirdparty
Other Sources
Resources
Frameworks
Products

Can someone advise exactly where is the code I entered in Titanium stored in the xCode project?

If I'm completely looking at it wrong, then an overview of how Titanium compiles the code we write in Ti Studio/Developer into xCode would be great.

    The Xcode project has all the Titanium classes and libraries which read, understand and display all your JS code in the Resources folder.

    So, you can't actually see all your Javascript converted to Objective-C, if that's what you were expecting.
