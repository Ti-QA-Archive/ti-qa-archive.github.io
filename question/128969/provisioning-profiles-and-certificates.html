Provisioning profiles and Certificates

Hi all. I am an iPhone developer and a beginner in using Titanium Studio. I worked on some samples and they work fine in emulator and I need to run those applications on iOS device. Do I need to create a separate certificate and provisioning profile to work in Titanium studio, or shall I use the same which I used for developing applications using Xcode?

  • Hi, you should be able to use your current certificates within titanium. you may need to download it again and when you put your app to the device select upload file from within titanium and select the downloaded file. Easiest way I have found to add certificates for use in Titanium.

    Also if you have distribution certificates they will also work within Titanium.
