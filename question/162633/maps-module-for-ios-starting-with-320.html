    Hi

    As long as your current app is set to Ti SDK 3.2 or higher then you can select the iOS maps module by using the Tiapp.xml editor in studio.

    • Open tiapp.xml in studio
    • Make sure you are in the visual side of the editor
    • Select the green plus next to the modules list
    • Choose the Maps module

    You do not need to download it - it comes with titanium now, but only shows up if your app is set to Ti SDK 3.2 or higher. This module will come from the grouped modules within Titanium and you do not need to install it locally.
