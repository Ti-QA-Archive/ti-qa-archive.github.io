    hi david,

    you can pass your data or any other value using custom variable

    like

    var win = Ti.UI.createWindow({
    c_date : <date value>,
    url : <url of window>
});
win.open();

    and in that window you can get this value by 

    var cwin = Ti.UI.currentWindow;

var dt = cwin.c_date;
