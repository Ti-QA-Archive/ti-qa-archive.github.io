  • Ok, so you have your existing code base on your PC.
    You bought this mini so you can do iOS development.

    You can remote screen the MAC to the PC using VNC or LogMeIn (I prefer logmein, because the same interface/setup allows access from outside the office AND from the iPad)

    You will WANT to remote view the mac because you need to SEE the iOS simulator if you are using that.

    If you are testing on hardware, you can use edit the shared files from the PC, use SSH to run Matt Apperson's MakeTI command line build utility to deploy to your device or to testflight.
