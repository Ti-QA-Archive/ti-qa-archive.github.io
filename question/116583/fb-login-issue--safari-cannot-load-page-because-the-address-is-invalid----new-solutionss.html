  • I've the same problem so I change the source code of Facebook.m to open login in a pop like 1.5.1 version.

    /Library/Application Support/Titanium/mobilesdk/osx/1.6.0.RC1/iphone/Classes/FBConnect/Facebook.m

    Change the line 214 to:
    [self authorizeWithFBAppAuth:NO safariAuth:NO];

    Se more in this link http://developer.appcelerator.com/question/112551/small-facebook-login-window-on-titanium-160rc1
