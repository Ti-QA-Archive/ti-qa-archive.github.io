Set an active tab which is in the More section

I have an iPad app, portrait with more than 9 tabs, so by default you get the More button.

Now I would like to directly open the 9th tab, but if you call:

setActiveTab(8);

You get the More tab, any ideas to directly open the 9th tab?

  • Accepted Answer

    Open the window that belongs to that tab after u activate the tab. I do this in a project and works nice and smooth.

        tabGroup.setActiveTab(4);
    win_settings.open();

    where win_settings is the window assigned to the 5th tab.
