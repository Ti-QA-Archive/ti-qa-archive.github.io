How to set the splash screen always portrait orientation in the application?

I have to set the splash screen image orientation set to portrait mode.How to do this.please help…
Here is my code in the app.js…

Ti.UI.orientation = Ti.UI.PORTRAIT;
Titanium.UI.setBackgroundColor('#DBF4FF');

var admn = {
navBarHidden: true,
scrollType : 'vertical',
backgroundImage : '/images/login_bg.png',
opacity : 1,
title : "Lifeline360",
url : '/main_windows/loginForm.js',
exitOnClose : true
};

var login = Titanium.UI.createWindow(admn);
login.open();

In this code,first the splash screen come into the landscape mode and then changed to the portrait mode.
I have to set only the portrait mode.

  • Hi, If you want to splash screen always portrait orientation then do following steps :

    • Go to you project build — > iphone —> info.plist .
    • Copy info.plist and past into your resource folder.
    • Now Edit info.plist which you kept in your resource folder:
    • Under in UISupportedInterfaceOrientations tag within info.plist remove : UIInterfaceOrientationLandscapeLeft and UIInterfaceOrientationLandscapeRight
    • Now again clean and build the application , This will block your splash screen to portrait mode.

    Hope , this solves your problem :)

    • Hello, Just follow this steps and get it done:

      One option to get Splash screen in portrait mode is goto android project folder —> build —- > android ——> here you can see AndroidManifest.xml file .

      Copy AndroidMenifest.xml file text and put in AndroidManifest.custom.xml file in same folder where AndroidManifest.xml file is kept and in activity tag write android:screenOrientation="portrait" for portrait mode and replace android:configChanges="keyboardHidden|orientation" to android:configChanges="keyboardHidden" in whole file in each activity .

      Example

      <activity>
android:configChanges="keyboardHidden"
android:screenOrientation="portrait"
</activity>

      This will solve your problem.
