    Hello Sourabh,

    Thanks for your inquiry. Try doing a project -> clean, this should clean out the build folder for your project. If cleaning doesn't work. Try removing any Android Build-tools version "21" and "21+" from Android SDK Manager.

    To run Android SDK Manager in windows platform, go to your "android-sdk\tools" directory and run android.bat and deselect all Android SDK Build Tools numbered above 20 (as stated above). Restart Titanium studio after the change made with the Android SDK Manager. Try rebuilding your app.
