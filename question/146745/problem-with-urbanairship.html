Problem with UrbanAirship

Hi,
I have an issue with an iPhone app and urbanairship. I used urbanairship.js
https://wiki.appcelerator.org/pages/viewpage.action?pageId=27164863
and when I start the app I see in the log that deviceToken has been released and I can push notifiation from urbanairship console by point it to the deviceToken.

Something wrong when I change from development mode to production mode (I mean I switch the urbanairship's app from dev to prod), of course, I changed my keys and I've made all the apple's certificates process.

I've published my app in apple store, downloaded it, no error when it starts but there is no devices have been registered in urbanairship.

So I linked my iPhone to xcode to check the log:

What can i check to debug it?

Here the titanium code that we use to register and retrive pushes:

// URBANAIRSHIP
var APP_KEY =    'xxxx;
var APP_SECRET =  xxxx;
var MASTER_SECRET = 'xxxx';

Ti.include('urbanairship.js');

UrbanAirship.key = APP_KEY;
UrbanAirship.secret = APP_SECRET;
UrbanAirship.master_secret = MASTER_SECRET;
UrbanAirship.baseurl = 'https://go.urbanairship.com';


Ti.Network.registerForPushNotifications({
    types: [
      Ti.Network.NOTIFICATION_TYPE_BADGE,
      Ti.Network.NOTIFICATION_TYPE_ALERT,
      Ti.Network.NOTIFICATION_TYPE_SOUND
    ],
    success:function(e) {
      var deviceToken = e.deviceToken;
      Ti.API.info('successfully registered for apple device token with '+e.deviceToken);

      var params = {
        tags: ['it'],
      };

      UrbanAirship.register(params, function(data) {
        Ti.API.debug("registerUrban success: " + JSON.stringify(data));
      }, function(errorregistration) {
        Ti.API.warn("Couldn't register for Urban Airship");
      });
    },
    error:function(e) {
      Ti.API.warn("--> push notifications disabled: "+JSON.stringify(e));
    },
    callback:function(e) {
      Ti.API.log("callback!");
      var a = Ti.UI.createAlertDialog({
        title:'title',
        message:e.data.alert
      });
     });
      a.show();
    }
});
// END URBANAIRSHIP
