iPad simulator opening app as iPhone app

Hi all

This is probably pretty basic stuff, but what setting makes the iPad simulator treat your app as an iPhone app?

My app is for the iPad but when I launch it in the iPad simulator it shows up as an iPhone app (meaning it doesn't fill the screen but is displayed with iPhone resolution and scaled down)

    Hi Martin

    • Go to the "Project Explorer" - should be on the left side of the Studio window.
    • Expand your project with the grey arrow.
    • Look for the file tiapp.xml
    • Double click it
    • On this new screen find deployment type (bottom left)
    • Change the iPhone tick to iPad tick (or have both if you want a universal app.
    • Save using the toolbar or file menu
    • Close the tab it appeared in.
    • [I would do this step but might not be needed] Project Menu > Clean (this option is your friend)
    • Re-run in the simulator.

    When you created the app you will have been asked which platforms you were targeting, or you may have missed that if if you skipped the last steps. But easily changed.

  • Hi Martin,

    Try to remove below orientation type from your tiapp.xml file…

    <iphone>
        <orientations device="iphone">
            <orientation>Ti.UI.PORTRAIT</orientation>
        </orientations>
</iphone>

    and only set ipad orientation like…

    <iphone>
        <orientations device="ipad">
            <orientation>Ti.UI.PORTRAIT</orientation>
            <orientation>Ti.UI.UPSIDE_PORTRAIT</orientation>
            <orientation>Ti.UI.LANDSCAPE_LEFT</orientation>
            <orientation>Ti.UI.LANDSCAPE_RIGHT</orientation>
        </orientations>
</iphone>

    Clean the build from simulator and resource folder and run it again with ipad….
