Hi,

I have a little problem working with webview and button interface android.

I need to surf the webview back button using the button interface android.

I'm using alloy
Below is the code.

$.WebView.addEventListener('android:back', function() {
    if ($.WebView.canGoBack()) {
        $.WebView.goBack();
    } else {
        $.index.close();
    }
});

  • Your code helped me a lot, but i had to make some changes to make it work

    I give an id to webview and the window in xml, and put this code in index.js

    $.win.addEventListener('android:back', function() {
    if ($.main.canGoBack()) {
        $.main.goBack();
    } else {
        $.win.close();
    }
});

    Where "win" is the id of window and "main" is the id of webview

    index.xml

    <Alloy>
    <Window id="win" exitOnClose="true" navBarHidden="true" class="container">
        <WebView id="main" url="http://www.google.es" />
    </Window>
</Alloy>
