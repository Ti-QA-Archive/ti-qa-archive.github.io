Getting A List Of Unique Values From An Array

Hi,

I have a database with a field which can contain multiple values separated by a semi-colon.

What I'm trying to do is to list the unique values which will then be clicked to get values from another field in the respective row, thus listing all rows which contain a given value.

The way I'm approaching this is to try and concant the values from the database into a single string, then split the string on the semi-colon, and then do some stuff to de-duplicate the resulting array.

originalData = [1,"apples;",2,"apples;oranges;",3,"oranges;bananas;",4,"bananas;"]

string = "apples;apples;oranges;oranges;bananas;bananas;"

newData = [1,"apples",2,"oranges",3,"bananas"]

Can anybody point me at a similar example or correct my basic approach please? I'm not a programmer, but I understand basic coding concepts.

Many thanks

  • If it were a legit JS array, you could use this utility function:

    var unique = function(origArr) {  
    var newArr = [],  
        origLen = origArr.length,  
        found,  
        x, y;  
    for ( x = 0; x < origLen; x++ ) {  
        found = undefined;  
        for ( y = 0; y < newArr.length; y++ ) {  
            if ( origArr[x] === newArr[y] ) {  
              found = true;  
              break;  
            }  
        }  
        if ( !found) newArr.push( origArr[x] );  
    }  
   return newArr;  
};

    called as:

    var myUniqueArray = unique(myDirtyArray);
