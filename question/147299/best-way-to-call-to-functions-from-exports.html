Best way to call to functions from exports

Ok i am working on a basic loader which i have in a file called loader.js

here is my loader.js

exports.loader = function() {
    self = Ti.UI.createWindow({
        width : Ti.Platform.displayCaps.platformWidth,
        height : Ti.Platform.displayCaps.platformHeight,
        top : 0,
        left : 0
    });
     return self;
}

exports.hide = function() {
    setTimeout(function(){
        self.close();
    },1000);
}

Ok and im calling this from another file with.

var checker = require('loader');

And in that file i am calling it with 

checker.loader().open();

Works fine and opens then to close im calling 

checker.hide();

I am asking if this is the best practise to do this it works but should i have to set self as a global

Looking for help with best practises

Any Help

    This is fine, however, in this particular case, your hide function just closes the original window. Once you've returned the self object in the original function, you can act directly upon that window, so the second function isn't really necessary.
