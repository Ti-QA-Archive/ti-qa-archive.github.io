Android -- textInputs are all cutoff...has anyone seen this?

Hey there – for some reason, when i create text inputs, and check them in the android simulator, they look like this:

Does anyone know whats up? Here's the code im using to create them…

Ti.UI.createTextField( { font: { fontSize:16 }, color:"#666", textAlign:"left", borderColor:"#CCC", height:30, paddingLeft:5, paddingRight:5, borderWidth:2, width:"65%", right:10, top:0 } )

Bonus question: does anyone know why the fonts on the Android simulator look so….shitty?

  • Based on Android's default unit of "px" (pixels), the height=30 on your textField isn't tall enough to show all the text. You can set a taller height. Better is to specify the units. You can set height=&#39;30dip&#39; to specify density independent pixels. Or, set it for your whole app by adding the following to your tiapp.xml

    <property name="ti.ui.defaultunit">dip</property>
