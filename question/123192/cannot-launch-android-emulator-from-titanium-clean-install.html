  • So i found a fix to the problem. If you read the the DIR for the SDcard file there is a space in it.

    Reading on the forum people are saying that some problem with spaces in the path name will throw errors.

    In builder.py i changed this.

    emulator_cmd = [
            self.sdk.get_emulator(),
            '-avd',
            avd_name,
            '-port',
            '5560',
            '-sdcard',
            '"'+self.sdcard+'"',
            '-logcat',
            #"\"*:d *\"",
            '*:d,*',
            '-no-boot-anim',
            '-partition-size',
            '128' # in between nexusone and droid
        ]

    NOTE the line '"'+self.sdcard+'"',
