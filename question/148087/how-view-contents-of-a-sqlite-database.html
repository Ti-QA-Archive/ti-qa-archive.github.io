  • Well, couple of things.

    You can use ddms/monitor in the tools directory of your android-sdk to connect to a device/emulator session and within it use its functions to copy the sqlite file over to your system and open it with an sqlite editor.

    You can use a native android sqlite editor on the device, provided you have correct permissions to access said sqlite file.
