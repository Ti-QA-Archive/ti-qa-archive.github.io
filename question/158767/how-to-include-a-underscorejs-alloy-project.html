I start with Titanium and I'm trying to create a project with Alloy.

I put this code in my controller index.js but underscore is not included, I get an error …

Alloy var = require ('alloy') Backbone = require ('alloy / backbone), = require (' alloy / underscore ') .;

Do I need to download the library underscore.js? If so, or do I have to include the file?

    You don't need to include them explicitly as "Currently, these modules are automatically available in the global scope and these APIs can be used without loading the modules."http://docs.appcelerator.com/titanium/latest/#!/guide/Alloy_Controllers.

    To make sure whether there are defined or not. check before you define it.

    var Alloy = require('alloy'), _ = require("alloy/underscore")._, Backbone = require("alloy/backbone");

if(Alloy === 'undefined'){

Alloy = require('alloy');
}
if(_ === 'undefined'){

 _ = require("alloy/underscore")._;
}
if(Backbone === 'undefined'){ 

Backbone = require("alloy/backbone");
}
