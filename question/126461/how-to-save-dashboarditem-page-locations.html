How to save dashboardItem page locations

I understand how to save the order of the dashboard items after an edit is made using a simple for loop, however, I cannot figure out how to save the page that the item is on.

Here is my code to save the order:

dashboardView.addEventListener('commit',function()
    {
        mainWindow.rightNavButton = null;

        Ti.API.info('data ' + dashboardView.data);
        db = Ti.Database.open('myDatabase');
        for (var i=0;i<dashboardView.data.length;i++)
        {
            db.execute('UPDATE DashboardItems SET orderNum=? WHERE name=?', i, dashboardView.data[i].label);
            Ti.API.info('label ' + dashboardView.data[i].label + ":" + i);
        }
        db.close();
    });

This code works just great to keep track of the order of the items but it doesn't preserve the page the icon was located on if it was moved to a new page and the previous page is not full.

Is there a property in the dashboardItem that keeps track of the order and page?

I am building against Titanium SDK Mobile 1.7.2

Thanks in advance!

After completely closing the app, although the new order is correct, all the icons are added to the first page rather than remaining on the page they were previously on.
