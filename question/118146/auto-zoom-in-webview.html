The thing is that when I load it whith default settings the page is to wide and when I use scalePageToFit it zooms to much.

Can I some how set a zoom amount or set the webViewWidth or something?

    Johan

    You have not stated what platform you are using, so I am not sure if this will help. However, you can use a meta tag to set the initial scale of the view on iOS as follows:

    <meta name = "viewport" content = " initial-scale = 2.3">

    There are other metatag options you can use in the webpage, as explained in the Safari HTML Reference.

    Hope it's useful
