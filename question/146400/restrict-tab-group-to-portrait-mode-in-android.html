Restrict Tab Group to Portrait Mode in Android

Hi I am using Titanium to develop an application. I am using a tab group which contains some tabs along with some windows of tabs. But the issue is I am unable to restrict the tab group to portrait mode. I am using Titanium SDK(3.0GA). I tried by doing changes in custom Android Manifest file like

<activity android:configChanges="keyboardHidden" android:name="ti.modules.titanium.ui.TiTabActivity" android:screenOrientation="portrait"/>

But still the tab group is getting oriented. I tried programatically as follows:

Ti.Gesture.addEventListener('orientationchange', function(e) {

Ti.Android.currentActivity.setRequestedOrientation(Ti.Android.SCREEN_ORIENTATION_PORTRAIT);

});

Even though tab group is not restricting to portrait mode. The below is the code I am trying:
