tab focus and blur

hi,

i have this event on tab : 

    var tab5 = Titanium.UI.createTab({
        icon: 'images/sondage.png',
        title: '???????',
        window: winSondage
    });

    tab5.addEventListener('focus', function(e){
            alert('focus');
    });

    tab5.addEventListener('i', function(e){
            alert('blur');
    });

bizarrely, the focus is executed just one time on the first click on tab5…
the same for blur, it is executed only the first time i leave tab5 !!!!

normally it should be executed on each focus, blur ?

thanks for help.

    This is best practice:

    tabgroup.addEventListener('focus', function(_e) {
    alert(_e.index);
});
