  • You can roll your own HMAC using something like this http://jssha.sourceforge.net/

    The barcode scanner module is available for IOS and Android.

    Barcode for iOS now supports the following formats:
    QR Code
    Data Matrix
    UPC_A
    UPC_E
    EAN_8
    EAN_13
    Code 39
    Code 128
    ITF

    Barcode for Android supports the following formats:
    QR Code
    Data Matrix
    UPC_A
    UPC_E
    EAN_8
    EAN_13
    RSS14
    Code 39
    Code 93
    Code 128

    It uses Google's ZXing (Zebra Crossing) library, an open-source multi-format 1D/2D barcode image processing library.
