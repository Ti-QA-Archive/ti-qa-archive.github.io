Run iPhone app in iPad Simulator

Hi,

I have created a Mobile application. I installed it on iPhone and iPad. On the iPad it runs in a smaller portion of the screen. I specified the fixed position for every controls.

var lblUserName=Titanium.UI.createLabel({        
    text:"Username",
    font: {fontSize:17,fontFamily:'Helvetica Neue'},
    textAlign:'right',
    top:30,
    color:'black',
    height:30,    
    width:'auto',
    left:25,        
});

Is there any way to set the position which works for both? (iPhone and iPad)

Thanks

  • Hi,
    You can do platform base coding :

    
if (Ti.Platform.osname == 'ipad') {
    // ipad stuff
}
else {
    // iphone stuff
}

    Refer Api for more details. Titanium.Platform.osname

    lblUserName.left = 50;

    lblUserName.left = 25;
}
