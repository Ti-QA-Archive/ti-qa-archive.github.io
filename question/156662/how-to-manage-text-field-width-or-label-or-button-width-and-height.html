  • have you checked module Ti.Plateform ? Using this you can render device's height and width and then can set your view's height , width and other dictionary properties according to screen resolution in % .
    Use :Titanium.Platform.DisplayCaps.platformHeight to get device's height and Titanium.Platform.DisplayCaps.platformWidth to get device's width and then apply work around to set views.
