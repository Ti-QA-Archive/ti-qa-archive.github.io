Splash screen orientation only portrait

Hello, i'm trying to set only portrait orientation in my splash screen, here is the code that i use in manifest but is not working 

<android xmlns:android="http://schemas.android.com/apk/res/android">        
        <manifest>
            <activity
                android:name="com.example"
                android:configChanges="keyboardHidden"
                android:screenOrientation="portrait"
            />          
        </manifest>
    </android>

  • Accepted Answer

    The procedure to follow is documented in our Guides. However, don't remove the |orientation from configChanges as described in that guide (I've filed a bug report to have that removed). The configChanges attribute controls whether your activity is restarted when the listed events occur … by removing orientation from the list, your activity (app) could be restarted each time the user rotates the device.
