Pause music

Hello! I need help please.

I put a background music, and now I want to create a button to pause the music. I try with the following code, the music plays, but not the pause button.

(Sorry my english) Thanks!

Titanium Studio, build: 3.1.0.201304151600
Application type: mobile (iOs and Android)
MacOSX 10.8.3


var sound = Ti.Media.createSound({
        url: 'music2.mp3',
        looping:true,
        preload: true
    });
 sound.play();

var button = Ti.UI.createButton({
    title:'Pause',
    top:20,
    width:50,
    height:40
});
self.add(button);

button.addEventListener('click', function() {
    sound.pause();
});

    Hi

    Your code does work.

    The pause button only stops the music - it does not start it again - if you want that try this;

    button.addEventListener('click', function (e) {
    if (sound.isPlaying()) {
        sound.pause();
        e.source.setTitle('Play');
    } else {
        sound.play();
        e.source.setTitle('Pause');
    }
});
