Multiple devices

I am developing an app that is going to target the Iphone,Ipad and the Android platform. I am using some code similar to the follwing snippet:

if (Ti.Platform.name == 'android') 
{
do something for android here
} else if (Ti.Platform.name == 'iPhone OS') {
Do something here for the iPhone
} if (Ti.Platform.name == 'ipad') {
do something for the iPad here
}

I have been successful for the iPhone and android, however whenever I run the iPad simulator the Ti.Platform.Name always returns "iPhone OS".

Does this require an actual device to work properly or am I missing a setting while running the iPad emulator?

Thanks
