Density Scale Factor for Android?

Hi all,

My question is about multiple screen sizes support…

So i have an app and some square buttons on the screen. For example:

var squareButton = Titanium.UI.createButton({
                        height: '110dp', 
                        width: '110dp',
                        top: 1,
                        left: 1,
                        title: 'hello',
                        color: 'red',
                        borderRadius: 10,
                        opacity: 1
});
squareButton.addEventListener('click', function(e)
{
         Ti.API.info('screenDPI: ' + Ti.Platform.displayCaps.dpi + ', density: ' + Ti.Platform.displayCaps.density);
});

As you can see the height and the width are '110dp' and my problem is that the size is different in screens with different density. For example for a low density screen (~120) this size looks okay but for a high density screen the size is huge and occupies almost the half of the screen! For even higher density the size is even bigger!

So i want to ask if there is any scale factor i can apply in order to appear normal in different screens?
