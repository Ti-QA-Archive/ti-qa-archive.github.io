    • got it… I change the split ( &#39;_&#39; ) to ( &#39;&#x2F;&#39; ) and make this:

      function fetchTwitter(){
    cb.__call(
        'statuses/user_timeline',
        "screen_name=RadarBlitzPOA",
        function (reply) {
            // ...
            Ti.API.info(reply);
        },
        true // this parameter required
    );
}

    • Sorry about the mess… I figure it out

      function fetchTwitter(){
    cb.__call(
        'statuses/user_timeline',
        "screen_name=" +Ti.Network.encodeURIComponent("@RadarBlitzPOA"),
        function (reply) {
            // ...
            Ti.API.info(reply[0].text)
        },
        true // this parameter required
    );
}

      Thanks for this code!
