  • The question doesnt make sense. WCF is a generalised transport for web communication and is equally capable of delivering data to end point using XML/SOAP (basicHttpBinding) as well as other transports like HTTP GET, POX, JSON (webHttpBinding)

    So my answer would always be to use webHttpbinding for an Appcelerator App as it is better comsumed in javascript
