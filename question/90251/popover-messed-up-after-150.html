Popover Messed up after 1.5.0

I updated a few things with one of my apps today including building it now with 1.5.0 most of the issues I have had were fixed. But I ran across a new and pretty serious issue and I would like to know if anyone else has seen this or know of a fix.

I have a popover view with a table inside it. In 1.5.0 it is pushing the first 2 rows of the table up out of view automatically, I can pull it down to see them but as soon I would let go to click it just springs back up out of view.

The other problem with it is, when I select a item from the table view then go back to it, the table view is blank.

Here is a video of my screen of the problem happening: http://uploadingit.com/test/TI_Problem.mov

Does anyone know how to fix this?

Here is the code for the popover 

buttonPlaylists.addEventListener('click',function(){
    if(online)
    {
        popover = Ti.UI.iPad.createPopover({height:500,minHeight:300,maxHeight:500,width:'320',title:'Playlists'});
        popover.add(playlistsView);
        popover.show({view:buttonPlaylists});
    }
    else{Utils.alert('No Network Connection','You must be connected to the internet to access this feature.');}
});
