Image error android email dialog

Mobile
Titanium 3.1.2
Android SDK 2.3.3 Device: Samsung Galaxy Player
Mac OS X

I have some image code that works fine in ios but I get an error in android. I have a view (I have to use a view on android, as imageview will not show images on top) that I need to change to a blob and then resize to attach to an email dialog. Here is my code:

        var toBlob = image.toImage();
var pic = toBlob.imageAsResized(200,200);            
email.addAttachment(pic);

On android I get an error: Object #<Object> has no method 'imageAsResized'. From the documentation, my understanding is that .toImage returns a blob, which then has a method imageAsResized…. ?

Incidentally when I did have the image as an imageView instead of a plain view it still gave me the same error with image.toBlob();

  • I figured it out! I finally found out that in android, .toImage() does not return a blob, it returns an object. Using .toImage().media in android gets the blob, which then works with imageAsResized. Works!
