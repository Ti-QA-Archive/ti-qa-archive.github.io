how to use filesystem in a Alloy model extending from backbone?

I need to use an alloy model (extending from Backbone) that works with the filesystem storage (so, i just want to persist my model to filesystem and back - not sqllite, not anything else). However, in alloy docs at http://docs.appcelerator.com/titanium/latest/#!/guide/Alloy_Models I dont see how to use the filesystem. it only explains "type" for HTML5 localStorage and for sql. How to simply use filesystem?

    There is currently no adapter for using the filesystem for sync. If you want to use the filesystem, you'd have to write your own adapter for it. You can see the current adapters and how they work in the github source for alloy.
