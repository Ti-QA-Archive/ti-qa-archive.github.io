The Disappearing View

I'm amusing myself with the title, but it's not really accurate. The view never appears at all. One of the concepts I've really struggled with trying to build a Titanium app is the idea that I have to explicitly declare the dimensions of nearly everything. In this case, I have a window which I'd like to include a header and a scrollable body. The header shows up nicely, but when I define the body like this:

var body = Ti.UI.createScrollView({ 
  backgroundColor:'#000',
  backgroundImage: '/images/body.png',
  borderColor: '#f00',
  bottom: 0,
  layout: 'vertical',
  left: 0,
  right: 0,
  top: 0,
});

It doesn't appear. Nothing. On the other hand, if I replace the top, right, bottom and left properties with height: &#39;auto&#39;, the the body appears, but it's height becomes that of its content so it still doesn't scroll.

Can someone help me understand why?

Thanks.

  • The solution I have to this is to set the height of the scrollable View to be what I need. If I need it full screen use the height off the current window.

    Then set contentHeight: auto.

    A scrollable container needs to have its height otherwise it will keep expanding to the height of the content and go off the window. If the height isn't set it will have a height of 0 until components are added to it.
