var users = [{id: 1, price: 50}, {id: 2, price: 20}];

  • There are many ways to accomplish this. Without knowing what you want to do with the result, its hard to suggest the best method.

    To answer your question as it is stated, here is a way to get a list of id's not in the comments array using the underscore library. If you have an Alloy project the underscore library is included. 

    var userIds = _.pluck(users, 'id');
var commentIds = _.pluck(comments, 'id');
var uniqueIds = _.difference(userIds, commentIds);

    uniqueIds will now be an array of id values that are not present in commentIds.
