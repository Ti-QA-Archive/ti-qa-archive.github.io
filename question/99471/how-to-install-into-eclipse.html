  • Robert

    The only way the software has a hope of working successfully is if the pre-requisites are satisfied. As you have mentioned, there is a problem with your python installation (Titanium requires 32 bit python 2.6+). Also, I can see from the following that you are using OpenJava, whereas Sun's Java is required:

    java version "1.6.0_20"
OpenJDK Runtime Environment (IcedTea6 1.9.2) (6b20-1.9.2-0ubuntu2)
OpenJDK Server VM (build 19.0-b09, mixed mode)

    Please read Preparing for Android development, and the linux install guide again if necessary.

    Also, this page for configuring Java is very useful.

    sudo update-alternatives --config java
sudo update-alternatives --config javac

    Hope this helps
