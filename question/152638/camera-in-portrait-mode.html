    That fix is marked to be included in the SDK 3.1.1 release. So it wouldn't be working in 3.1.0. If you can't wait, you could try one of the nightly builds

    From that site, copy the URL to the SDK version you want to install. In Studio, choose Help > Install Titanium SDK. Paste in that URL and click OK. Update your tiapp.xml to use the new SDK version.
