Unable to launch iOS Simulator

I have a fresh new Macbook Pro, OSX version 10.9.4, on which I have installed Titanium Studio 3.3.0, Xcode 6.0.1 with iOS Simulator 8.0.

When I go to launch the "Hello World" mobile app to the iOS Sumultaor (any device) I get the following error in the console:

[INFO]:  Running application in iOS Simulator
[INFO]:  Launching application in iOS Simulator
[INFO]:  Focusing the iOS Simulator
[ERROR]: Project failed to build after 1s 639ms

I am able to launch the iOS Simulator manually through Xcode, however it seems that Titanium Studio is unable to. Anyone have the CLI commands that TI is calling so I can try to diagnose this further?

The Android emulator is working fine, which indicates there is not an issue with the "Hello World" app.

  • Accepted Answer

    To work properly with Xcode 6 and iOS8, you should be using the 3.4.0-RC versions of Studio, CLI, and SDK. The 3.3.0 versions are not fully compatible with Xcode 6/iOS8 and that's likely the root of the issue you're seeing.
