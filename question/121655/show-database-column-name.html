Show database column name

Hi, is it possible to show in a picker the name of the column in my database.

Like this: I have a database with 2 tables and 2 columns.
I'd like to retrieve the name of the 2 column in a picker

Table: Personne

Column: - id / - nom

And i want to return "id" and "nom" in a picker

    First off, you should know that unless you have reason to believe your schema will change dynamically on the device, you'll find the performance of your app is much better by just hardcoding this sort of thing.

    That said, another approach to accomplish getting the column names of a table is to select one row from the table (SELECT * FROM your_table_name LIMIT 1) and then just use the result.fieldCount() and result.fieldName(fieldIndex) methods to loop over the columns and read each of their names.

  • This command will return you the column information you need.

    PRAGMA table_info(&#39;YOUR_TABLE_NAME&#39;) Where YOUR_TABLE_NAME is replaced by your table's real name (Personne if I recall)

    From there, you can retrieve the column names.
