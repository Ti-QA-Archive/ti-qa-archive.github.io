  • Hi

    Do you mean you have pointed your url to a webpage that includes PHP code inside it?

    If so then this web page needs to sit on an actual web server that has PHP running as the server side language. This is NOT something you can do in Titanium natively. If your web page is inside your project then it will certainly NOT run PHP as there are no PHP servers running inside your titanium app.
