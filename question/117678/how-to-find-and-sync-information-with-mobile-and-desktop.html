How to find and sync information with mobile and desktop

Hi,

I want to develop a app for mobile and desktop use. I want to be able to sync information between then without the use of a internet connection just the pure Wi-Fi.

How can I do that?

    You can use the HTTPServer feature in desktop to create a http server.

    // Create a HTTP server that listens on port 8080
    var httpServer = Titanium.Network.createHTTPServer();
    httpServer.bind(8080, function(request, response)
    {
        Titanium.API.debug("Got request for URI: " + request.getURI());
        response.setContentType("text/html");
        response.write("<h1>Hello!</h1>");
    });
    1. Desktop Creates HttpServer
    2. Mobile app checks to see if server is online, then displays to the user it has found the desktop app
    3. Sync over HTTP
