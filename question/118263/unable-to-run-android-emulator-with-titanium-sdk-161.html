      You have a version of Java that is not compatible with Titanium. Please remove your 64bit version and install 32bit JDK 1.6. This is explained in the Sun Java Development Kit section of the Titanium Compatibility Matrix.

To rectify this, close your emulator, install these missing packages, delete all the existing AVDs, then try to launch your app again.

    • Jim

      Close your emulator, remove the obsolete Android packages, "SDK Platform Android 2.3.1 API 9…" and "Google APIs by Google Inc Android API 9", delete existing AVDs, delete your project build files (see steps below), and then launch the app (ensuring that you have selected "APIs 2.2" in the "Run Emulator" screen of Titanium Developer).

      To delete your build files and force a full rebuild, for Android, delete the contents of the immediate subdirectories of &#x2F;build&#x2F;android. Do not delete the directories themselves, otherwise you may encounter problems. Watch ticket #2852 to be updated by email of a resolution to this issue. Then click the Save Changes button to force a full rebuild.

      If this does not resolve the issue, try Titanium SDK version 1.5.X, to see if it works.

      Let us know :)
