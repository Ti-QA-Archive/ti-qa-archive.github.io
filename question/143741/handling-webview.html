handling webview

Hey,

I want to catch an event in a webview fired from Titanium. Unfortunately, I get no message… Can anyone help me? I would really appreciate it!!

WindowHome.js

var webview = Titanium.UI.createWebView({
         url: '/tablet/example1.html',
         width: 'auto',
         height: 'auto'
    });


win1.add(webview);

webview.addEventListener('beforeload', function(e){
    Ti.App.fireEvent('app:getSize', { message: 'event fired from Titanium, handled in WebView' });
});

example1.html

 });
