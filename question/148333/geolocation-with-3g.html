    Titanium.Geolocation.getCurrentPosition doesn't get the current position, but the last position known by the device. And this is done without really starting the gps module on the device.
    You will have to start listening to the location event in Titanium.Geolocation.

    Keep in mind that this is async and it can take a lot of time until it fires (has to start the gps, lock the satellites, etc)
