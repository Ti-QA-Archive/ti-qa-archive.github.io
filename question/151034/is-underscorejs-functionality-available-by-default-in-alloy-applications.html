Is UnderscoreJS Functionality Available By Default in Alloy Applications?

Hi there-

Are UnderscoreJS functions/methods available by default in the Alloy framework for use? I have been trying to access the _.findWhere() function call, and I am receiving an undefined method for an iOS app. I was not sure if UnderscoreJS methods came with the Alloy framework, or if I needed a require (and if so, how I needed to do the require, and where the proper location in my app the require should be located? My guess would be in the alloy.js.

Thanks for any help
Chris

    Ah, Alloy has underscore.js 1.4.2 builtin. findWhere() was introduced in 1.4.4. So Alloy won't support it out of the box. You can however override the builtin version pretty simply. Download the underscore.js version you want then copy into your app structure like this:

    PROJECT_ROOT/app/lib/alloy/underscore.js

    That will overwrite the builtin underscore.js and use the more recent one.
