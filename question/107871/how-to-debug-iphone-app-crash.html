how to debug iPhone app crash?

I have an app that runs fine in the emulator but crashes randomly on the device(usually about once a day). How does one determine what caused the crash? My app mainly consists of http network requests, tableviews and webviews.

Heres whats reported in xcode (always the same):

Exception Type: EXC_CRASH (SIGABRT)
Exception Codes: 0x00000000, 0x00000000
Crashed Thread: 0

Using mobile sdk 1.5.1

  • With the phone plugged in to your development machine, run XCode and open the 'Organizer' window. Then choose the 'console' tab. You should then see everything going on, including any debug (using Ti.API.info('hello') ) text you get your app to spit out.
