Controlling app from server.

HI, is it possible to control the native UI on the app from a remote server through something like a webview or something? I want to be able to update my app's tableviews from a website. Is this possible? I checked the documentation and it's not clear to me.

    Hi,

    you can always use the webview to load the HTML Content from the web through which you can make the design and other changes on the server when needed. In other alternative you can also use the webservices which provides important change from the server and for that you have to code your app in a way that it adopt the changes which came from the webservice result.

    But in short there is no trick or hack in Titanium to do that automatically.
