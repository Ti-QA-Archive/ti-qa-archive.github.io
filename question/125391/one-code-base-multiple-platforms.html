One code base multiple platforms?

We are looking to switch from jquery mobile to Titanium but had a few questions. From looking around I am under the impression we can maintain one code base and have that one code base be compiled to the iOS, Android as well as a mobile web app to be created for users that have neither of those devices. Is this true? Or will we need to have a separate code base for the mobile web app?
