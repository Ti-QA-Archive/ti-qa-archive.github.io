Different Graphics For Different Platforms In Universal iOS App?

I am creating an app with some images in it and I want to display a different resolution image depending upon whether they are viewing my app on an iPhone (pre retina display), an iPhone 4 upwards or an iPad 1 or 2.

An example piece of my code is

var compsys = Ti.UI.createImageView({
    image:'menu/1information.png',   
});

Is it a matter of simply renaming the png files in the same way you do with app icon when submitting it to iTunes (obviously I have tried that with no luck but may be doing something silly wrong) or do I need to somehow detect which device they are running and point to a different image file 3 times?

Cheers.

  • Accepted Answer

    var compsys = Ti.UI.createImageView({
    image: Titanium.Platform.osname == 'ipad' ? 'menu/information_ipad.png' : 'menu/information_iphone.png'   
});
