Updating a section

Hi all.

I'm working on a project that has a tableView with a section of 3 rows for each yearly month.

...
[section]
    [row1] April [/row1]
    [row2] Related data [/row2]
    [row3] Footer [/row3]
[/section]

[section]
    [row1] May [/row1]
    [row2] Related data [/row2]
    [row3] Footer [/row3]
[/section]
...

Now I need to update "MAY" section with new data.

I know how easy is to update a row but what about a section?
I don't even have access to the section index, or do I?

If not possible, that should be something that Appcelerator team should look into to add to the SDK.

What are your suggestions?

    • Actually, after posting this, I went again to the documentation and there is a way to see the sections index, however there is no way to get the selected section index.

      tableViewName.section = array of sections
tableViewName.section[0].rows = array of rows in section index 0

      The only way is like you suggested, creating a variable within each section and pull that value every time I select a row inside that section.

      
var tableViewData = [];

for(var i = 0; i < monthArray.length; i++){
    tableViewData.push(createSection(monthArray[i]));
}

tableView = Ti.UI.createTableView({
    data:tableViewData
});

function createSection(sectionData){
    var section = Ti.UI.createTableViewSection()

    var row1 = Ti.UI.createTableViewRow({
        ... // code
    });
    section.add(row1);

    var row2 = Ti.UI.createTableViewRow({
        ... // code
    });
    section.add(row2);

    var row3 = Ti.UI.createTableViewRow({
        ... // code
    });
    section.add(row3);

    return section;    
}

      This is probably not a best practice, but since I'm starting and still learning with Titanium, this was the best approach I found. I could probably use advantage of reusable objects, buy like I said, I'm still learning and counting with your experience to help guide me to the right path :)

      Even so, I still didn't stop to think how an I going to update those rows since I have a function that creates the entire section. I have to invest some time to think how an I going to do this.
