Webview Pinch and Zoom not work on local HTML file

Hi,

I've tried to load a remote HTML into the webview and it can be zoom in/out and pinch.

However, when I use the same HTML and load at as local file, it cannot zoom or pinch, does anyone have this problem also?

Here's my code fragment:

app.js

var webView1 = Titanium.UI.createWebView({
    url:'http://www.goodaftermoon.com/media/content/html/index.html'
});
var webView2 = Titanium.UI.createWebView({
    url:'20110123_2.html'
});

index.html

<html>
<head>
<meta http-equiv="Content-Type" content="text/html; charset=UTF-8" />
<style type="text/css">
body {
    margin-left: 0px;
    margin-top: 0px;
    margin-right: 0px;
    margin-bottom: 0px;
}
</style>
</head>

<body>
    This is a testing page.
</style>
</body>
</html>

I also found that the KitchekSink webview local HTML page demo have no zoom and pinch function

    Try to play with scalesPageToFit property of the webview ( set it false or true - don't remember which one allows you to pinch to zoom :) )

    • Thanks Dan! Completely solve the problem!

      var webView3 = Titanium.UI.createWebView({
    scalesPageToFit:true,
    url:'20110123_3.html'
});

  • Could be caused by conflicting touch events. Try disabling touch events on the webView:

    var myWebView = Titanium.UI.createWebView({
    url: url,
    touchEnabled: false,
});
