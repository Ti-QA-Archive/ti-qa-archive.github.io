    If your Titanium studio is crashing on the launch then try this.

    go to your titanium workspace in finder. There will be a hidden folder "metadata"
    open that folder now open plugins folder which will be hidden too. inside plugins go to the folder org.eclipse.core.resources inside it you will find a .snap file delete that file. and launch your studio.

    This fixed the issue for me.. hope that helps :)
