  • Hi

    This can happen if you have either installed two different versions of Xcode in different locations in the past or chosen to install Xcode in a location that is different from the standard location.

    Type this in the terminal:

    > sudo xcode-select -switch /Developer

    Or

    sudo xcode-select -switch /Applications/Xcode.app/Contents/Developer/

    Checking the path matches the location of your Xcode. Re-start studio afterwards and try again.

  • If you run titanium info -t ios, you'll see all of the Xcode installations and supported iOS SDKs that Titanium found on your computer. If your Xcode installation is not in the /Applications folder, you'll have to use xcode-select to select your preferred Xcode installation.

    According do the error, you have iOS SDK 5.1 installed. You can build for iOS 5.0 using iOS SDK 5.1. If you want to launch the 5.0 simulator, then you need to run:

    titanium build --platform ios --ios-version 5.1 --sim-version 5.0
