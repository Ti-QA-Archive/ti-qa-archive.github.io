Installing Titanium for beginners (like me)

Hi everyone,

I had a few problems setting up titanium thought I would share my experiences.

I work on Windows XP, and android SDK(hopefully).

OK, so I installed Android SDK, JAVA SE SDK and titanium with little to no problems. Got an error Titanium cannot find Android SDK, even after pointing it to SDK.

The problem:

In more recent editions of the android SDK the adb.exe file has been moved to a new folder called platform-tools, along with any associated dependencies. Titanium however is still looking for the adb file in the tools folder.

The resolution: -

Simply right click on my computer and go to environment variables button, Find the Global Path variable and add: -

c:\program files\android\android-sdk\platform-tools;c:\program files\android\android-sdk\tools;

You may get an error about titanium not being able to find java when you clearly installed it. Again you simply need to add the correct text strings to the path variable: -

%JAVA_HOME%\bin

Again you will need to restart titanium (if %JAVA_HOME%) doesnt work manually locate your java\bin directory on your computer.

Finally if you are trying to compile for android please make sure you select a platform followed by API i.e. 1.6 apis.

Good luck
