Disabling Gradient on the SearchBar

I'm trying to utilize the SearchBar, calling the following:

var search = Titanium.UI.createSearchBar({
        backgroundColor:'transparent',
        showCancel:false,
        height:43,
        top:0,
        width:450
    });

I'm trying to use this in the iOS Navigation Bar along with a title prompt to give a search screen similar to native apple apps such as Weather (when you add a new city).

The above code has the right color (for the native nav bar), but still has the gradient around it. I want to just get rid of that. I tried a null background image and even a transparent png, that didn't help. I tried to mess with the backgroundGradient, but I can't find any documentation on the different values (other than the examples in Kitchen Sink, which didn't help).

Does someone know how to simply disable the gradient?

  • Nevermind - i found a different approach - I didn't realize the search bar had that capability in it, so I just disabled the navbar for that window and added the search bar as follows:

        var search = Titanium.UI.createSearchBar({
        showCancel:true,
        prompt:"Your State",
        top:0,
        height:68
    });
