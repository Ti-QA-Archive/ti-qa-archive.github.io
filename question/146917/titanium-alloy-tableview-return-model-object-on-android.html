Titanium Alloy TableView return model object on Android

I can return the model object on a selected tableViewRow on iOS but when running on android 2.2 i get the error - Uncaught TypeError: Object has no method &#39;get when calling


function selectRow(e) {

    var title = e.rowData.model.get("title");    
    Ti.API.info("data model title: " + title);

}

I've tried this running Titanium 2.1.4GA and 3.0.0GA

Has anyone got any suggestion?

regards

Dave

    I believe that to be safe cross-platform, the data attached to a row must be serializable as JSON, meaning no functions. Models obviously contain functions. I assume you are at some point attaching the model to the row. Instead of attaching the whole model, just attach its attributes. Like this:

    // if you were doing this...
row.model = myModel;

// do this instead...
row.model = myModel.toJSON();

    e.rowData.model should then contain a plain JS object. Then you would access the properties simply as e.rowData.model.title.
