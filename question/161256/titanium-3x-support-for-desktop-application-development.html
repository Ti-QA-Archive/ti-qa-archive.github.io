    Hi

    If yo mean developing for desktop apps rather than mobile apps - then no - whilst there was support for desktop in early versions of Titanium, the direction focussed on mobile and the desktop dev splintered off to a community project - tidesdk.org.

    There are no current public plans for any form of desktop development, although mobile web development provides a certain level of support but not true desktop support.

    • The last version of Titanium that supported desktop is not something you want, it is very old (in version terms) and had very few features.

      The splinter to Tidesdk.org (link above) and the link @Stephen gave for TideKit (link above) they are much newer versions with many more features. The core logic is still based on Titanium so should be familiar but keep in mind the point at which they splintered means that each has gone down their own path - so the code is not going to be cross compatible between Titanium and the splinters.

      The versions that would have supported the desktop with Titanium are now simply too old to be able to be run these days as everything else moved on without them - which is why there are so many version upgrades each year - to keep up to date with it all.
