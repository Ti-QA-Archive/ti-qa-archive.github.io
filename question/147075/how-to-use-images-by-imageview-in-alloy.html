how to use images by imageView in ALLOY

i just want display images in imageView,and i use code:

code block (<ImageView id="loadingImgView" images="['/loading/01.png','/loading/02.png','loading/03.png','/loading/04.png','/loading/05.png']" duration="100" repeatCount="0"/>)

it's not work!

  • You can't put arbitrary Javascript in the XML attributes. It would be much better to put your ImageView's properties in a corresponding TSS file.

    index.xml

    <Alloy>
    <ImageView id="loadingImgView"/>
</Alloy>

    index.tss

    "#loadingImgView": {
    duration: 100,
    repeatCount: 0,
    images: [
        '/loading/01.png',
        '/loading/02.png',
        '/loading/03.png',
        '/loading/04.png',
        '/loading/05.png'
    ]
}
