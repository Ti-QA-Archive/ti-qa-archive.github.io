    • It does work on the simulator, just confirmed this. It does only work with local files. You did not originally indicate the need for viewing remote as opposed to local files - so I did not take this into account.

      Here is my test code;

      var btnPDF = Ti.UI.createButton({
    title: 'PDF',
    top: 300
});
win.add(btnPDF);
btnPDF.addEventListener('click', function (e) {
    var doc = Ti.UI.iOS.createDocumentViewer({url:'example.pdf'});
    doc.show();
});

      Where example.pdf is local to the root (resources).

      If your need to view files is mostly remote then the webView method is likely your best bet, however you could always download the file locally and then view it.
