  • You can't like your own app with the requestWithGraphPath method. For "me/likes" you need the "user_likes" permission, but this only returns the pages you have already liked.

    You have to show the native like button, but currently this is not available in the official Appcelerator Facebook SDK.

    You might implement the like button with another module, e.g.

    iOS: https://github.com/timanrebel/Facebook

    Android: https://github.com/mokesmokes/titanium-android-facebook
