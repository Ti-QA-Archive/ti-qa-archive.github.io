Drag an image

Should be an easy task but I can't figure out how to smoothly drag an image/view in Titanium (android and iOS). Found some threads but they all don't seem to work for both platforms or jump around.

one solution was:

var olt = Titanium.UI.create2DMatrix(), curX, curY;

$.view1.addEventListener('touchstart', function(e) {
    curX = e.x;
    curY = e.y;
});

$.view1.addEventListener('touchmove', function(e) {
    var deltaX = e.x - curX, deltaY = e.y - curY;
    olt = olt.translate(deltaX, deltaY);
    $.view1.animate({
        transform : olt, duration : 100
    });

});

But it doesn't work on android. And the solution with animatedCenter is not supported on android (documentation is wrong, but will be fixed with 3.2; there is a ticket for that already)

Anyone got a solution to drag an image around? It should work on Android and iOS, Alloy, 3.1.3. Thanks in advance
