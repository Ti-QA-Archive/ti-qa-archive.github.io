Horizontal scrollview with dynamic content loading from right to left

Is it possible to get a Scrollview to start its view from the right? 

<ScrollView id="scrollView" height="100" width="100%" top="0" scrollType="horizontal">
            <View id="view" height="100" width="1000" layout="horizontal" >

            </View>
</ScrollView>

I'm dynamically loading buttons onto the view and i want the last button to be viewed first as it will represent today with older dates available to the left.

Ive tried using offsets but i still start on the left and can only scroll to the right?

    I think you would need to use scrollToBottom to achieve that. In the case of your horizontal layout, scrollToBottom will scroll right
