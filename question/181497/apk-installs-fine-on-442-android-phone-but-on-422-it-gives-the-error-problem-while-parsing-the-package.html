  • Ok so I got it working. Still not sure what the problem was/is.

    The error in logcat was this:

    [ 03-19 10:34:01.432 7444: 7444 W/PackageInstaller ]
    Parse error when parsing manifest. Discontinuing installation

    [ 03-19 10:34:01.432 7444: 7444 E/PackageParser ]
    Package com.schrader.app has no certificates at entry AndroidManifest.xml; ignoring!

    I connected the phone through usb, did a run on device and it installed and worked fine. Then, I went to the build folder, copied the apk and put it in the server. Downloaded it to the phone like before, but now it worked. I am now working on a run on device basis so it's alright. I just hope I won't have this problem anymore because I'm not really sure what caused it.
