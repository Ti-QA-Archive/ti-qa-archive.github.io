Custom Variable on Row?

Hey,

is it possible to assign a custom variable to a component like TableViewRow?

In my case a variable named "id" would be perfect.

Is this possible?

    Yes, in the tableViewRow just assign a property id to the value that you need. Then in the click event for the table you can pull out that property via the e.rowData.id;

  • Hi Nico

    Yes this can be done fairly easily.

    If you are using a simple table;

    var data = [
 { title: 'One', id: 123 }
];

    Or a custom row

    var data = [];
var row = Ti.UI.createTableViewRow({
    height: 40,
    id: 123,
    width: Ti.UI.FILL,
});
var lbl = Ti.UI.createLabel({
    height: Ti.UI.SIZE,
    left: 10,
    text: 'One'
    width: Ti.UI.FILL
});
row.add(lbl);
data.push(row);

    Then you can use the data as before;

    var tbl = Ti.UI.createTableView({
    data: data,
    height: Ti.UI.FILL,
    width: Ti.UI.FILL
});

    And then when you are listening for the row you can access the id;

    tbl.addEventListener('click', function (e) {
    Ti.API.info('id', e.rowData.id);
});

  • Yes is possible, but try to use a variable name that does not have any chance to clash with some properties put by Appcelerator:

    var row = Ti.UI.createTableViewRow({
    title:'my title',
    nico_id: 1
});


row.addEventListener('click', function(e) {
    alert(this.nico_id);
});
