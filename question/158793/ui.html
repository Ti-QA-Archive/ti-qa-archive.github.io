  • Hi,

    first of all do not set layout property for notification view then try to make add one view that is for notificationImage instead of making imageview so now you need to set backgroundImage for that view and add notificationCount label on that view with top property to 0. In this way you can set notification counter above on the image.

    .xml

    <View id="notification">
    <View id="notificationImage">
        <Label id="notificationCount">25</Label>
    </View>
</View>

    .tss

    '#notification': {
    width: '18%',
},
'#notificationCount': {
    backgroundColor: 'red',
    borderRadius: 10,
    color: '#FFF',
    top : 0,
    font: {
        fontSize: 20
    },
    touchEnabled: false
},
'#notificationImage': {
    backgroundImage: '/images/notification.png',
    touchEnabled: false,
}
