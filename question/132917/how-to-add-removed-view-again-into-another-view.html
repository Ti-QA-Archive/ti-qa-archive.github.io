    If you want to reuse the view again then rather then removing the view you may also hide the view.
    By using :

    function abc(condition)

    {
    if(condition)
    {

    view1.hide();

    view2.show();

    }
    else
    {

    view2.hide();

    view1.show();

    }
    }
