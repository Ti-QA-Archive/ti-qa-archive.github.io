Dynamic label using a function creates a duplicate, how to remove element

Hi,

I have a view that displays different texts depending on what button is pressed. The first display is correct but once the 2nd button is pressed the second item overlaps the previous one creating a very messy result.

this is my function

function content(text) {
    var icontent = Ti.UI.createLabel({
        text: text,
        font: {
            fontSize: 14,
            fontFamily: 'AppleGothic',
            fontWeight: 'normal'
        },
        width: 260,
        top: 10
    });
    return icontent;
}

Below is how I use the function, adding it to boxContent view displaying first or second content when first and second button is clicked, respectively.

boxContent.add(content('First content'));
boxContent.add(content('Second content'));

Am thinking of removing the previous content from the view first but I have no idea how to do this. I tried content=null but this didn't do anything.

Please help. Thanks.

  • var contentone = content('first content');
var contenttwo = content('second content');

boxcontent.add(contentone);
boxcontent.remove(contentone);
boxcontent.add(contenttwo);

    Build you component into a variable, then you can add and remove that to the parent component.

    oh null out the variables after use to clear the memory.

    Hope this helps
