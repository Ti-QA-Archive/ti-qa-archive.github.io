    Hi

    You have two perfectly reasonable example of how to achieve what you want here. Unfortunately I think the problem lies inside your actual code.

    var params={
    checkbox:e.source.value,
};

    This assumes that checkbox will become the result of the value e.source.value. However your scope of e has changed, you probably think it is related to the checkbox,

    For simplicity of testing (tidy up afterwards) change as follows;

    // this
submit.addEventListener("click", function (e) {
// becomes
submit.addEventListener("click", function (f) {


// and this
var params={
    checkbox:e.source.value,
};
// becomes
var params={
    checkbox:f.source.value,
};

    You should consider renaming e and f to something that has meaning to your code.
