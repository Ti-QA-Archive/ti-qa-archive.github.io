  • Hi

    This isn't possible with the optionDialog for iOS. You can create a custom view for Android and pass it to the 'androidView' property of optionDialog…but I don't think this will solve your problem.

    I suggest that you create a custom view to replace your optionDialog so that you can create it exactly how you need it and create the buttons how you want them with coloured text you're after.

    If you create it as an Alloy_Widget or Alloy_Controller then you will be able to reuse this view whenever you need it.

    Hope this helps!
