Is there any way to save application settings? (Titanium mobile)

Hi Guys,

I'm just wonder - is there any way to save application settings ? I don't mean into database.
For example - in NimbleKit you can use something called NKSettings. Example:

var settings = new NKSettings();
settings.setValueForKey("UserName", "user");
var username = settings.getValueForKey("user");

Is there any similar function in Titanium mobile ?

— asked August 5th 2012 by Jakub Chudarek

  • Accepted Answer

    Ti.App.Properties.setString(&#39;key&#39;, &#39;value&#39;);

    and

    var value = Ti.App.Properties.getString(&#39;key&#39;);
