Add controls at the beginning of a view with a vertical layout

Hi,
I have a scroll view that's defined with layout: &#39;vertical&#39;.

I add stuff to it and their being added one by one just as they should.

I have a situation in which after I've added a few controls I need to add more controls but make them appear at the top of that view. But if I add them then they are added at the bottom.

How can I add the controls at the top of a vertical layout?

Thank you

  • OK got your needed. Try this code:

    var win = Ti.UI.createWindow({
    backgroundColor : 'white'
});

var sv = Ti.UI.createScrollView({
    contentWidth:'auto',
    contentHeight:'auto',
    top:0,
});

var v = Ti.UI.createView({
    width: '100%',
    height: '100%',
    layout : 'vertical'    
});

for(var i = 1; i < 10; i++) {
    var lbl = Ti.UI.createLabel({
        text : 'Label ' + i,
        width : 200,
        height : 40
    });
    v.add(lbl);
}

sv.add(v);
win.add(sv);
win.open();

// After 1 sec, adding 2 new labels red & green at top 
setTimeout(function() {
    var lbl1 = Ti.UI.createLabel({
        text : 'I am red',
        width : 200,
        height : 40,
        top: 0,
        backgroundColor: 'red'
    });
    var lbl2 = Ti.UI.createLabel({
        text : 'I am green',
        width : 200,
        height : 40,
        top: 40,
        backgroundColor: 'green'
    });

    sv.add(lbl1);
    sv.add(lbl2);
    v.top = 80;
}, 1000);

    Idea is NOT adding controls directly to scrollview, but adding to a normal View with layout: &#39;vertical&#39; - in above sample code is v. The scrollview sv is not layout in vertical. So when you want to add new controls to the top, just adjust v.top to move down existed controls, then adding new controls to top.

    Hope this can give you some hints,

    Minh
