  • why don't you convert the file to a blob or a base64 string and fire an event that will be caught by your Titanium app (not the webview itself) ?
    I use webviews for several things, but then with events I can achieve anything I want because they're caught by the underlying part.

    Example:
    I have a map manager developed in leafletjs. The webview calls the tile needed, that will be processed by the javascript code outside the webview, and then sent back to the webview.
