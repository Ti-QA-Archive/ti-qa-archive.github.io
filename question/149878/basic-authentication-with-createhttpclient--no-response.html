Basic Authentication with createHTTPClient – no response

I have to load ressources (text) from a secured directory. So I set the request headers of the client accordingly, but to no avail. Unfortunately, not even the onerror callback is called, I simply get no response whatsoever. Here is my code:

var client = Titanium.Network.createHTTPClient( {timeout:10000});
var url = "mysite.com/mysecuredir/myfile.json";
client.open("POST",url);
var authString = "Basic "+Titanium.Utils.base64encode("myuser:mypass");
client.setRequestHeader("Authorization", authString );
client.send();

I also tried:

var url = "http://myuser:mypass@mysite.com/mysecuredir/myfile.json;

but with the same result. This url works in the browser and in a Flash app, so it has to be right.

What am I missing?

    • Also add the content type header after the setRequestHeader for username and password:
      It's also important to note that Titanium.Utils.base64encode() adds line breaks every 72 characters (bug?). This can cause your authorization to fail! Here's the fix with a regex:
      So try like this: 

      var authstr = 'Basic ' + Titanium.Utils.base64encode(username + ':' + password);
authstr = authstr.replace(/(\r\n|\n|\r)/gm,"");
xhr.setRequestHeader('Authorization', authstr);
xhr.setRequestHeader("Content-Type", "text/html; charset=utf-8");
xhr.send();

      Hope it will help

  • hi,

    if this url mysite.com/mysecuredir/myfile.json is correct then you should add http:// before url and try it again.

    plus you should add onerror and onload method so you can get better idea of what error is.

    var client = Titanium.Network.createHTTPClient( {timeout:10000});
var url = "http://mysite.com/mysecuredir/myfile.json";

client.onload = function(e) {
    Ti.API.info('Response >>>> ' + this.responseText);
};

client.onerror = function(e) {
    Ti.API.info('Error >>>> ' + JSON.stringify(e));

};

client.open("POST",url);
var authString = "Basic "+Titanium.Utils.base64encode("myuser:mypass");
client.setRequestHeader("Authorization", authString );
client.send();
