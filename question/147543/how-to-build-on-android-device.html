How to build on Android Device

So I own an android phone and I've found the emulator just runs too slow or not at all despite using a brand new computer, so I would rather build straight to my phone, but with my phone plugged in (developer mode on) when ever I run the phone installer adb finds 0 devices/emulators, how do I get around this?

    Hi Richard,

    If you have installed the drivers, then please enable USB Debugging in your Nexus 4 by going to Settings -> Developer Options -> USB Debugging.

    After following the above step unplug and replug your device to the windows machine.
