Specifying font weights using custom fonts in iOS

Hi all,

I'm tearing my hair out over custom fonts on an iOS app. I've used them before with no issue, but only ever using 1 font and not using various weights.

So I've done the Plist update, added the 2 fonts I want to use to the Resources folder and also the reference in the PList, but it will not display the font at all.

Any ideas?

I've tried clean building etc, it can only be a syntax thing i'm sure!

So here is where I'm trying to display my font….

var titleLabel1 = Titanium.UI.createLabel({

    height:18,
    width:600,
    top:10,

    textAlign:'center',
    font:{fontFamily:'Frutiger LT Std',fontSize:18},
    shadowColor:'#eee',shadowOffset:{x:0,y:0.1}
