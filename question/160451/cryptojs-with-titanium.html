CryptoJS with Titanium

Hi,
I have a problem using CryptoJS with my Titanium project. I downloaded CryptoJS from google, included it(aes.js) to my project. The problem is, when I call CryptoJS.AES.encrypt("message", "password"), my app crashes on iOS simulator(crashes on device too). Strange thing is, that there is no error log in console, just this:

[ERROR] :  An error occurred running the iOS Simulator
[ERROR] :  [DEBUG] using retina iphone retina tall ios 7
[ERROR] :  
[ERROR] :  Project failed to build after 44s 387ms

My code:

Ti.include('/aes.js');
var window = Ti.UI.createWindow();
window.open();
var encrypted = CryptoJS.AES.encrypt('message', 'password');
alert(encrypted);

My config: Titanium SDK 3.1.3, iOS SDK 7.0
I tried older Titanium SDK(3.1.1), no change.
Thanks for any help.

    • If you reference to 'crypto-js', then the crypto-js.js must reside direct in resource folder.
      Idea: you have a vendor folder under resources and the crypto stuff is named 'crypto.js'. In this case you must write:

      require('vendor/crypto');

      — commented December 13th 2013 by Rainer Schleevoigt

    1. Don't use suffix 'js',
    2. the path doesn't start with '.': the path is relative to Resource folder
    3. the path starts with '.': the path is relative to the caller 
     require('foo');     // Resources/foo.js
