Webview link - open in Mobile Safari instead?

I have a local iPad HTML file displayed in a webview. I have a link on it that I want to open in Mobile Safari instead of in the web view itself. How would I accomplish this?

SDK 1.8.1, ipad os 5, studio 1.0.10.201202241328, OS 10.7.3

    I did this by adding onClick events to the links in the HTML that performed a ~
    Ti.App.fireEvent('openmyurl', {url:<your link>});
    ~

    which I caught and used the openUrl method to open in the browser.
