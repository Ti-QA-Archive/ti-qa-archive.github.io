Unable to show animated images in Ti.UI.ImageView

Hi, I am developing android app using Titanium. I wanted to show an animated image in one of my views. Therefore I planned to use images property of Ti.UI.ImageView. I have all the frames stored in one folder and following is the code I am using to feed the array of these images to achieve and animated image.

  var images = [];
  for(var i=0; i<=7; i++) {
    var file = '/images/activityInd/actInd-f'+i+'.png';
    images.push(file);
  }

  var indicator = Ti.UI.createImageView({
    image: images[2],
    images: images,
    width: 16,
    height: 16,
    duration: 1000,
    borderColor: '#888'
  });

  indicator.addEventListener('click', function() {
    indicator.start();
  });

  myView.add(indicator);

Output:
In output I see a box (as I have given broderColor). And also a static image which happens to be the third frame of the sequence (as I have given image: images[2]). After I click on the image, I dont find it to be animated. How do I get it working?
