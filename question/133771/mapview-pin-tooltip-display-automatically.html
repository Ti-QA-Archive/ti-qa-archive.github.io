MapView pin tooltip display automatically

Hi guys,

How can I make the tooltip appear automatically after the pin drops on the map?

var location = Ti.Map.createAnnotation({
        latitude: geoEvent.latitude,
        longitude: geoEvent.longitude,
        title: pinTitle,
        subtitle: pinSubtitle,
        pincolor: Ti.Map.ANNOTATION_RED,
        animate: true,
        myId: 1,
        leftButton: '/images/organisation.png',
        rightButton: Titanium.UI.iPhone.SystemButton.DISCLOSURE,
    });

    var mapView = Ti.Map.createView({
        mapType: Ti.Map.STANDARD_TYPE,
        region: {
            latitude: geoEvent.latitude,
            longitude: geoEvent.longitude,
            latitudeDelta: 0.1,
            longitudeDelta: 0.1,
        },
        animate: true,
        userLocation: false,
        regionFit: true,
        annotations: [ location ]
    });

Thanks!
