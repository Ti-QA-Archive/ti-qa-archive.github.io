    Hi

    Without having any real scope for where these other JS files will be and if they are in; controllers, widgets or CommonJS modules - this is going to be a universal answer.

    Firstly I am about to suggest using Alloy.Globals. I do NOT recomment using Alloy.Globals without a very good reason, there are very FEW good reasons - this is one of them.

    Tweak your XML a little to give you access to the TabGroup;

    <Alloy>
<TabGroup id="tabs"> <!-- tweaked here -->
    <Tab title="Tab 1" icon="KS_nav_ui.png">
        <Window class="tab1" title="Tab 1">
            <Label>I am Window 1</Label>
            <Button class="exampleBut">Button </Button>
        </Window>
    </Tab>
</TabGroup>
</Alloy>

    Then in your index.js file;

    // add this line;
Alloy.Globals.tabs = $.tabs;
// this creates a reference you can access later

    In any other file you can do something like this;

    // This will set the caption of the second tab (zero index array of course)
Alloy.Globals.tabs.getTabs()[1].setTitle('Hello');

    Again - for your information anyone else's, Alloy.Globals is to be used rarely if ever. Only for exceptions and something cool you will be able to achieve with Alloy 1.4 which comes with Titanium SDK 3.3 - which is currently in beta.
