Ti.Platform.openURL('http://map.google.com/maps?q=loc:lat+lng')

Ti.Platform.openURL('comgooglemaps://?q=loc:lat+lng');

  • Accepted Answer

    With iOS 6 Apple has removed Google Maps from default Apps.
    So you have to check if Google Maps is installed. If not, you should open the iOS Maps:

    if (Ti.Platform.canOpenURL('comgooglemaps://?center=37.788463,-122.392545&zoom=12')){
    Ti.Platform.openURL('comgooglemaps://?center=37.788463,-122.392545&zoom=12');
}else{
    Ti.Platform.openURL('http://maps.apple.com/maps?center=37.788463,-122.392545&zoom=12');
}
