createWebView throws exception

Just getting started with Ti and I've created a simple navigation based app.

On the first page is a grouped table list. The user selects an item that creates a new window and adds it to the stack.

This window should display a web view, but it crashes the app. The stacktrace says 'invalid method (createWebView) passed to UIModule'

app.js:

var olb = Titanium.UI.createWindow({
    url:  "TestLoad.js",
    title:  "Google"
});

nav.open(olb);

TestLoad.js:

var win = Ti.UI.currentWindow;

var webview = Titanium.UI.createWebView({
    url:  'http://google.com'
});

win.add(webview);

  • Well, I'm not sure what happened but after I installed the 1.6 SDK and rebuilt my project, the webView worked.

    Just for fun, I downgraded the project to 1.5, rebuilt and it still works.

    No clue why it started working unless Titanium Developer was bugging out and needed a restart or something.
