    Hi

    I am going to show you your new best friend layout.

    This is a basic example but you should get the idea;

    var outer = Ti.UI.createView({
    backgroundColor: 'red',
    height: Ti.UI.SIZE,
    layout: 'vertical',
    width: Ti.UI.FILL
});
var inner = Ti.UI.createView({
    backgroundColor: 'green',
    height: Ti.UI.SIZE,
    width: Ti.UI.FILL
});
outer.add(inner);
var img = Ti.UI.createImageView({
    height: 100,
    image: 'imagefile.png',
    left: 10,
    width: 100
});
inner.add(img);
var title = Ti.UI.createLabel({
    backgroundColor: 'blue',
    left: 10 + 100 + 10, // sum to show you how it was calculated
    right: 10,
    text: 'whatever you like goes here',
    textAlign: 'left',
    width: Ti.UI.FILL
});
inner.add(title);
var desc = Ti.UI.createLabel({
    backgroundColor: 'yellow',
    left: 10,
    right: 10,
    text: 'big body of text goes here',
    textAlign: 'left',
    top: 10, // acts as margin when a child of a layout
    width: Ti.UI.FILL
});

    The backgroundColor properties are just there to show you where things end up.

    • When you set layout to vertical or horizontal the child views then layout based on that type. If you have ever created any web code before think of these layout views as DIVs and the contents simply sit inside.

      The image on the left and label on the right uses a technique of known image width but unknown device width, this way the text allows for the fixed width of 100 plus two margins of 10 on the left and a margin of 10 on the right to be subtracted from the total width automatically.

      You will also notice lots of Ti.UI.SIZE = equals the width of the contents and Ti.UI.FILL = equals the width of the parent. Helps you to manage your layouts better.

  • You can use layout modes to accomplish your goals. Views (and components that inherit from View like ScrollView) support the layout property, which you can set to 'horizontal' or 'vertical'. With vertical, for example, the top property is treated as a padding setting (offset from previous element).

    You'll probably want to add some extra views as wrappers.

    var scrollview = Ti.UI.createScrollView({
  layout: 'vertical'
});

var wrapper = Ti.UI.createView({
  layout: 'horizontal'
});

var ImageView = Titanium.UI.createImageView({
  left : 5, top: 5,
  width: 100, height: 100
});
wrapper.add(ImageView);

var Titlelabel= Titanium.UI.createLabel({
    top: 5, left: 105,
    wordWrap: true
});
wrapper.add(Titlelabel);
scrollview.add(wrapper);

var articlebody = Ti.UI.createLabel({
  top: 10, left: 5,
  wordWrap: true
});
scrollview.add(articlebody);
