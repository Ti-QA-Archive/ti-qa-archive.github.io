  • Hi

    1. If you are trying to forward all emails that iOS is in control of then the short answer is - No. There are no hooks available to allow you to handle the system events that would be required to achieve this. I believe this is a limitation of iOS and not of Titanium.

    2. If you want to create a service that will allow others to provide their email details and you then offer a forwarding service then yes it is possible, but the iPhone would only be used to manage the account settings as having emails come to the phone only to be redirected elsewhere would be pointless.

    If you were going to try number 2 then know that this will be a big project and require dedicated servers and your own server based software to achieve your goal.

    This is a very big project if you are not sure what you are doing and will require advanced knowledge of email systems and protocols to get it to work.

    Trust me - I have written a system to handle email marketing and email auto response handling some years back. Whilst things have become easier - it is still not a simple project.
