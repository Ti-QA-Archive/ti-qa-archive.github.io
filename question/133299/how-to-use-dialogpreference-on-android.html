How to use DialogPreference on Android

I'm using following preferences.xml file:

<?xml version="1.0" encoding="utf-8"?>
<PreferenceScreen xmlns:android="http://schemas.android.com/apk/res/android"
    android:title="Settings">
    <PreferenceCategory android:title="Authentication">
        <CheckBoxPreference
            android:title="Foo"
            android:defaultValue="true"            
            android:summary="Bar"
            android:key="FooBarKey" />

        <DialogPreference
            android:dialogIcon="@android:drawable/ic_dialog_alert"      
            android:dialogTitle="Second PreferenceScreen"
            android:dialogMessage="This is a second PreferenceScreen"
            android:positiveButtonText="OK"
            android:negativeButtonText="Cancel">

            <EditTextPreference
                android:name="Username Preference"
                android:summary="Username from System"
                android:title="Username"
                android:key="ApiUsername" />
            <EditTextPreference
                android:name="Password Preference"
                android:summary="Remote Password from System"
                android:title="Password"
                android:key="ApiPassword" />
        </DialogPreference>          
    </PreferenceCategory>    
</PreferenceScreen>

and getting Exception, when trying to open preferences dialog. XML seems to be valid and according to Android docs. Any ideas?

I also don't get why there is a need to specify both android:name and android:key properties if Ti only uses android:key to access this property value?
