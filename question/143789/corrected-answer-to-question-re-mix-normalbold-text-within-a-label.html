Corrected answer to question re: mix normal/bold text within a label

In the thread titled
Different fontWeight in same label text content, which has been locked for some reason, the answer given is incorrect.

@vilasini patel: if you use the HTML field of a label, you can use the html tag "<b> xxxx </b> to make a portion of your label bold. The rest remains in the 'normal' weighting. The html field over-rides the text field when rendering.
You can still use the font object within the label, but be careful about the setting of fontWeight.

Hope this helps,
DJ
