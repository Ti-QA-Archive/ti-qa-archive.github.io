  • Accepted Answer

    Looks like you've got some commas in the wrong place.

    Try:-

    "Label": { 
    width: Ti.UI.SIZE,
    height: Ti.UI.SIZE,
    font: {
        fontSize: '12dp'    
    },
    color:"#000"
}

    • Refer to the Alloy TSS guide at

      http://docs.appcelerator.com/titanium/3.0/#!/guide/Alloy_Styles_and_Themes-section-35621526_AlloyStylesandThemes-TitaniumStyleSheets

      You separate properties with commas unless it's the last property. In your example you had a comma after the '12dp' when it was not required and you'd omitted the comma after the } of the font declaration that separated the font from the color property.
