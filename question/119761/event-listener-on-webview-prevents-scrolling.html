Event Listener on Webview prevents scrolling?

Hi, Ti 1.6.2, iOS 4.3. I have a very simple webview, which references a rather large (in width/height) local html file, attached directly to the window. With this simple setup, I am able to easily scroll the web page vertically and horizontally. Works great. However, if I try to add any sort of event listener for touches (swipe, click, etc., etc), the web page no longer scrolls. Non-touch events don't seem to cause this, e.g., I can add an event listener for "load" and the page still scrolls, but if I add a listener for "swipe" or even "click", the page is "frozen" from scrolling. Any ideas? Thanks!

I am trying to allow the page to scroll, but I want to capture swipe events and use those to trigger some additional functionality.
