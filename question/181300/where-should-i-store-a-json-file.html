var file = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory,fileName);

    the resource directory is read only and the applicationDataDirectory is empty if you install the app. You can read initially from resourcesDirectory and save every time into pplicationDataDirectory.
