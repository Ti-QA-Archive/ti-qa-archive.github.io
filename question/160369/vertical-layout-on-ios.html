vertical layout on ios

I can't get vertical layout to work on ios (all views are piled).

This code works on android but not on ios.

What am I missing?

var colonne1 = Ti.UI.createView({
    top: 0, left:0,
    width : '25%',
    height : Ti.UI.SIZE,
    layout : 'vertical'
});

var moduleSSALTO = Ti.UI.createView({
    width : '96%',
    left:'4%',
    top:10,
    height : Ti.UI.SIZE,
    layout : 'vertical',
    borderRadius:10
});

//............

var moduleIndicateurs = Ti.UI.createView({

colonne1.add(moduleSSALTO);
colonne1.add(moduleIndicateurs);
contenus.add(colonne1);

    I think your assumption is correct. When you use Ti.UI.SIZE it should resize accordingly, however I have come across problems before on IOS.

    SO by defining two views, which auto size and putting them into a view which also auto sizes will cause issues like this.

    So the height of the two views when added is zero, so it is placing the first view at zero and the next view at zero plus a bit of padding, add a height value to the two views and it will work.

    Until you add the views to the window they are not sized.

    Hope this helps

    T
