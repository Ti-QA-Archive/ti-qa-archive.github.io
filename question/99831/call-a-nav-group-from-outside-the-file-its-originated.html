call a nav group from outside the file it's originated

I'm having an issue where i have a nav group for my app and in iOS from any controller that's controlled by the nav group you can pop a new view on top of the stack.
I see that i can do that as long as I'm in the file that I call it from. how do i add a new view to the navgroup stack from a secondary js file? i can't find any examples of this..

    you need to pass the NavGroup as a parameter to the new, secondary js file. An example is here

    var navGroup = Ti.UI.iPhone.createNavigationGroup( {
    window : win1
});

var win2 = Titanium.UI.createWindow({  
    title:'Window Two',
    backgroundColor:'#fff',
    url:'secondary.js',
    _theNavGroup:navGroup // <-- passed as parameter
});
