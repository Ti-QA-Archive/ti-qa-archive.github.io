Atom.io editor - Ti-Alloy package

Hi,

there is a new editor currently in beta-testing very similiar to Sublime: Atom.io (http://atom.io) and today the first ti-related package was created: https://atom.io/packages/ti-alloy-related. It opens the tss and view file if you are inside a controller. Very nice already, no ti-language support yet but I'm sure someone is already working on it.

The editor is currently only available for OSX but its looking good already. If someone wants an invite I have a spare one.

I'm still testing it but it feels like a good alternative for sublime and ti-studio

  • Big update here:

    Atom.io is now an Open Source project and source is released so you can build a version for linux and windows now! And since I have it running now on Fedora I'm switching from TIStudio to Atom.io!

    I've started a TSS language package here:

    Language TSS

    at the moment I've used the CSS language pack and added TSS syntax. Not complete but it will autocomplete some TSS properties.

    Recommended packages:

    • https://atom.io/packages/ti-alloy-related (open controller related tss and xml)
    • https://atom.io/packages/ti-build-atom (build for iOS; submitted a patch for build on android device)
    • https://github.com/abe33/atom-color-highlight (will show colored highlights)
    • https://github.com/richrace/highlight-selected (highlight to current selected words)
