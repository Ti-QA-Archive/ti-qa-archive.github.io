Android Custom Font in WebView

I have a custom font included with my app that works on both iphone and android. on iPhone I can even use it in a web view, but I don't seem to be able to do this on Android.

Any one had any success with custom fonts in webview on Android? any tricks?

Ti SDK: 3.1.1,

Android : 4.1.2

  • If you are using custom fonts according to the article provided here: http://docs.appcelerator.com/titanium/latest/#!/guide/Custom_Fonts, then you fonts are in Resources/fonts.

    All you need to do is add the following to your stylesheet:

    <style>
       @font-face { font-family: 'MyCustomFont'; src: url('file:///android_asset/Resources/fonts/mycustomfont.otf'); }
       body {-webkit-user-select: none; user-select: none; font-family: MyCustomFont; font-size:14px; background-color: #ffffff}
       p {font-family: MyCustomFont; font-size:14px;}
    </style>

    Note: if you are using the same page for iOS, you may want to wrap it around an if statement since iOS will try to load the file too, but on iOS this works without the need to declare the font-face.
