  • Hi mohammad,

    Instead writing a single value to a file, use Ti.App.Properties.

    // read
var totalunreadcount = Ti.App.Properties.getInt("totalunreadcount");
// write
Ti.App.Properties.setInt("totalunreadcount", totalunreadcount);
