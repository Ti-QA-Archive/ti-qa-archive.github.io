  • Did you try to delete the app from ur iPad and restart it?

    Deinstalled/deleted apps don't have to be cache cleared until restart.

    Delete from iPad -> deconnect iPad -> restart iPad-> Clean Target in Xcode -> reconnct iPad -> build & run on device

    The app name is not taken from the prov profile. ( make sure you have the right bundle ID, when tryin to update ur app to appstore ;) )
