Have button display horizontally in a vertical view?

I'm trying to get a button to display and be positioned horizontally in a window that has a vertical view and orientation. I've tried:

layout:'horizontal'

in the buttons properties, but that had no effect.

    You need to create a 2d matrix and transform it.

    var someButton = ...;
var t = Ti.UI.create2DMatrix();
    t = t.rotate(-90);
    someButton.transform = t;

  • depends on orientation of window/view(s)… and other factors but this might work for you regardless….
    Just keep in mind many ways to do this but a simple way and one that also works for screen density (for the most part -thinking button fonts here as images usually scale) is to…

    Try setting your positional properties using percentages.

    height: '15%',
    width: '50%'

    if want to take up whole width of vertical view then set width to '100%'.

    if want button at top of view then set top; '0%'

    for bottom then bottom: '0%'

    for left side or right then use left: '0%' right: 0%.

    if want to be from edge of view/screen further then increase top or bottom % value.

    adjsut the % values accordingly for height and width to get button size desired.

    using negative values like -50 (as suggested above) runs into potential issues.
    first screen density. second if view has navBar visible or navBarhidden: false - your button can be obscured by navBar. :)

    Hope this helps…
