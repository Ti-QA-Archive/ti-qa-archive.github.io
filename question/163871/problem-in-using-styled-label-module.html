Problem in using Styled Label Module

Hi,

I am building a mobile app both for Android and iPhone by using Titanium Alloy.

CLI version 3.2.1, Titanium SDK version 3.2.2.GA, Alloy version 1.3.1

I am using the Styled-label-module

to format the html tags that I am getting by calling a REST webservice.

When I use the Styled Label module in the following response from the REST webservice, I get a empty blank area in the iOS Simulator?

 <p>Location:</p>  
  <p>Gurgaon Centre:</p>
  <p>A27/2, Next to DT Mega Mall</p>
  <p>DLF Phase - 1</p>
  <p>Gurgaon - 122001</p>

I am using the styled Label module in the following way -

var writeupLabel = StyledLabel.createLabel({
        height : Ti.UI.SIZE,
        width : Ti.UI.SIZE,
        top : "0dp",
        left : "2dp",
        //   right:"2dp",
        font : {
            fontWeight : 'normal',
            fontSize : "12dp",
            fontStyle : "Arial"
        },
        html : param.writeup // where param.writeup is the response from the REST webservice which is described above.
    });

I think that the Styled Label module is not able to recognise the <p> tag but <p> tag is a simple HTML tag so the problem could be else where.

Can some one please help me over come this problem.
Thanks in advance.
Bably
