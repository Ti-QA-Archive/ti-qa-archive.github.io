Apps doesn't run on device since SDK 1.7.5

I create a new project from Ti Studio. It runs in the iPad simulator but exits at startup on the iPad device - you can shortly see the splash screen. I use Xcode 3.2.5, target is iOS 4.2. Up to SDK 1.7.3 everything was fine. But I need the fix of the orientation bug at startup, so I cannot use 1.7.3 anymore! I hoped, that the bug would be fixed in a future version, but the same problem occurs with 1.8.0.

Any ideas?
