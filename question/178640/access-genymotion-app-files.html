Access Genymotion app files

Hello! I'm using Genymotion as my Android emulator to test an app built initially for iOS. I'm getting errors every time I try access any file saved into applicationDataDirectory by the app. The file never actually exists apparently.

I can't find any documentation on where the Geny files are stored on my mac, as I can do with iOS to troubleshoot where files have gone, can anyone point me in the right direction? I just want to look into the files as I would with iOS by going to Library/ApplicationSupport/iPhoneSimulator etc

Thanks!

  • Hi Luke,

    You can run the emulator in your computer.

    $ adb shell
shell@android:/ $ su
shell@android:/ $ cd /
shell@android:/ $ ll
