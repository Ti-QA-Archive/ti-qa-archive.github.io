Query Posts in ACS using an array of User Ids

I am using ti.cloud 2.0.5 with SDK 2.0.2 GA on OSX Lion, developing for iOS.

I can successfully query posts using a single user_id but I have been unable to pass an array. My overall goal here is to show all posts from my friends. I have successfully used the REST API to retrieve an array of user_ids that correlate to my friends but using it in the "where" query doesn't work.

Any ideas?

This works

Cloud.Posts.query({
    page: 1,
    per_page: 10,
    where: {
        user_id: 'XxxxxxxXxxxxxXxxxx'
    }
}

This returns 0 results

    Do you want to query a list of user_ids

    you can do something like

    user_id: {'$in' : [user_id1, user_id2]}
