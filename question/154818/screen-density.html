  • try using only '%' and at the fontSize:

     (Ti.Platform.displayCaps.getPlatformHeight()/100)*3)

    it is same that '3%' the Height your display
    so 

    (Ti.Platform.displayCaps.getPlatformWidth()/100)*3)

    (it is same that '3%' the Width your display

    remember if you use % is of your container and can be your window or your view.

    try it: 

    buttonCount[i] = Ti.UI.createButton({
        title: '',
        titleName: rows.fieldByName('Activities_Desc'),
        backgroundSelectedColor: '#FF8000', 
        width: '80%',
        height: '70%',
        left:'5%',
        backgroundImage:rows.fieldByName('imagePath'),
        id:rows.fieldByName('Activities_Id'),
        top:'60%',
        font:{fontFamily:'Arial',fontWeight:'bold',fontSize:(Ti.Platform.displayCaps.getPlatformHeight()/100)*3) }
    });
