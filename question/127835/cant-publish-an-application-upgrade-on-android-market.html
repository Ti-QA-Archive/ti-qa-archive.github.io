Can't Publish an application Upgrade on Android Market

Hello,
I have an app on Android Market, now I want to upload an update, I change the version to 1.1 on tiapp.xml, clear and rebuild, but when I try to upload the new apk it says that version 1 is already loaded, acting like the version 1 is hardcoded somewhere, somebody post a similar question long time ago but I try changing Manifest.xml manually but every time I rebuild the project it changes again to version 1, any idea? thanks!

  • For update on Market you must change in you AndroidManifest.xml

    <?xml version="1.0" encoding="utf-8"?>
    <manifest xmlns:android="http://schemas.android.com/apk/res/android"
    package="com.myprogram"
    android:versionCode="2" //from 1 to 2
    android:versionName="2" //from 1 to 2

    AndroidManifest.xml must be created in yourapp/platform/android you can get a copy of manifest from yourapp/build/android and then create subfolder yourapp/platform/android and copy the file and modify it.
