How to customize buttons on Tabbed Bar?

How may I display different windows when a button is clicked on a IOS tabbed bar? Here is my tabbed bar code:

var bb1 = Titanium.UI.iOS.createTabbedBar({
    labels:['National', 'Local'],
    backgroundColor:'#007FFF',
    top:100,
    style:Titanium.UI.iPhone.SystemButtonStyle.BAR,
    height:25,
    width:200

});

How can I customize the labels 'National' and 'Local' to show different windows when I click them?
Thanks in advance,

    Because the tabbedBar maintains state, you wouldn't necessarily use it to show different windows, but different views. If you were wanting to pop open a new window, I'd use a buttonBar instead.

    To use the tabbedBar,

    Create 2 views, one to match each of your buttons.

    var nationalViewContainer = Ti.UI.createView({
    // properties you desire
});

var localViewContainer = Ti.UI.createView({
    // properties you desire

});

    You can then either use the zIndex property of your views to raise or lower them into view, or you can use the visible property to hide/show them based upon the login in your tabbedBar event handler.
