Some logic in alloy view?

Hi there,
Is there any support for app logic inside the alloy view?
Currently I'm developing an app that shows 4 OR 5 tabs inside the TabGroup depending on a xhr request.
I'm missing support for view helpers or the ability to assign a variable to my view like in other MVC Frameworks. Something like this would be very nice:

<if condtion="xxx">
    <then>
        <Tab ... />
    </then>
    <else>
        <Label ... />
    </else>
</if>

Thanks in advance.

  • No, that's not supported at this time. For logic-driven UI construction like that, you'd probably want to be doing so in code anyway. Adding logic in the view markup has the potential to start making it too complex and difficult to understand, which was the whole thing Alloy was trying to clean up in the first place.
