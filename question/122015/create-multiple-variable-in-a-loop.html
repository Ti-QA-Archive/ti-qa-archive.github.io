Create multiple variable in a loop

Hi, i want to how can i do to create multiple variable with a loop.

Like this:

for (var j = 0; j < selectedData.length; j = j + 1) {
    var basicSwitch = Titanium.UI.createSwitch({
        right:0,
        style:Titanium.UI.Android.SWITCH_STYLE_CHECKBOX,
         value:false
        });
    row.add(basicSwitch);
}

I want to create as many switch as that there are values in selectedData.

Example: selectedData.lenght = 4;

I Want basicSwitch1, basicSwitch2, basicSwitch3 et basicSwitch4.

  • There are two ways you van go around this. The idea of dynamically naming your variables is not possible in JavaScript. so you will have to rely on an array to contain your values.

    var basicSwitch[]; // empty array

for (var j = 0; j < selectedData.length; j = j + 1) {
    basicSwitch[j].push(Titanium.UI.createSwitch({
        right:0,
        style:Titanium.UI.Android.SWITCH_STYLE_CHECKBOX,
        value:false
        }));
    row.add(basicSwitch[j]);
}

    Although, this will only be useful if you want to access those variables later on. If you just want to add your switches to the row in a loop, you can always do it like so.

    for (var j = 0; j < selectedData.length; j = j + 1) {
    row.add(Titanium.UI.createSwitch({
        right:0,
        style:Titanium.UI.Android.SWITCH_STYLE_CHECKBOX,
        value:false
        }));
}
