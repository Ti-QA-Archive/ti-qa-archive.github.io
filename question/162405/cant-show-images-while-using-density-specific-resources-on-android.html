    • Hi Nuibb Hasan,

      I have understood your problem in before. I have also no any images folder under the Resources directory. The above code is working well in my emulator for android specific screen. Please you only copy a image and paste it in Resources/android/images/res-long-land-hdpi(and so on folder). If you use android specific image then only use the /images/imageFileName you have no need any images folder under the Resources directory.

      like

      var image = Ti.UI.createImageView({

                width:100,
                height:100,
                image:'/images/Tulips.jpg',
                opacity:1,
            });

      Here I have used Tulips.jpg. I have pasted this image in Resources/android/images/res-notlong-port-mdpi/Tulips.jpg and I have no any Resources/images. For android specific image you only use image:'/images/Tulips.jpg' and it will automatic call Resources/android/images/res-notlong-port-mdpi/Tulips.jpg
