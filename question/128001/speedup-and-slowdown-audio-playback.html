Speedup and slowdown audio playback

Hi,

I need to be able to speed up and slowdown the playbackrate of mp3 files. Do you have any idea how to achieve this on preferabel both iOS and Android?

Currently I use the HTML5 audio in a webview (I also need to display html content) because I need to be able to handle mp3 streams and local playback with seek functionality.
I know webkit on the desktop supports the plackbackrate option, but unfortunately that is not supported by the mobile webkit.
In the iOS 5 AVAudioPlayer Class Reference I noticed the "rate" property. I there any way to use this property from Titanium?
Any change a new mobile webkit version will support the playbackrate feature?

Any help is very much appreciated.

Edmar.

  • If it's not supported in the webview, then you should submit a feature request via the Titanium Community jira project. Otherwise, you do have the option of creating native modules for ios and/or android to make the audioplayer support the properties you need. Guides for building native modules are listed here:

    http://wiki.appcelerator.org/display/guides/Android+Module+Development+Guide

    http://wiki.appcelerator.org/display/guides/iOS+Module+Development+Guide
