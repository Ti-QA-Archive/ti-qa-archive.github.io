  • Hi

    Short answer - you should not try.

    Longer answer;

    With Android there is a phrase "do not put scrollable on scrollable". This is what you are trying here, as both the scrollView and TableView are scrollable.

    Whilst it can be achieved in some circumstances it is extremely code and function dependant so I would see what other ways you can achieve your goal.

    The idea of having a scrollView that during the scroll will reveal a tableView that can scroll within its space and in the same direction is a rather strange UI choice so I imagine you have a good design reason - but it might be at the expense of the user experience if you manage to get it to work.
