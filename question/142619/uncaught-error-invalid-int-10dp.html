  • Accepted Answer

    if no units are specified, a system-default unit is assumed. The system default unit is:Pixels on Android.
    To use DIPs as the default on all platforms, set defaultunit to dip by setting the ti.ui.defaultunit property in tiapp.xml.

    <property name="ti.ui.defaultunit" type="string">dip</property>
