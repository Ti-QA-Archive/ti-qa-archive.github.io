TabbedBar problems

  • Hi Mirly

    If you simply asking - can I add a tabbedbar into the position usually used by the title of a windows then - Yes.

    Simple Example

    var bb1 = Ti.UI.iOS.createTabbedBar({
    backgroundColor: '#336699',
    height: 25,
    labels: [
        'One', 
        'Two', 
        'Three'
    ],
    style: Ti.UI.iPhone.SystemButtonStyle.BAR,
    top: 50,
    width: 200
});

var win = Ti.UI.createWindow({
    backgroundColor: '#fff',
    titleControl: bb1 // note this line
});

    • Is this for Android, iPhone, iPad or a combination of those?

    • Your explanation without context is making it hard to work out.

    • When you say tabbedbar do you mean;

      • the iOS tabbar that appears on the bottom of iPhones
      • Android tabs that appear on the top of the screen
      • The tabbedbar only available in iOS? But this should be used to toggle state, rather than be a button.
