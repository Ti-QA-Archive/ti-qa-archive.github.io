  • The table needs and array of row objects e.g. {title: &#39;Row 1&#39;} or tableViewRows e.g. Ti.UI.createTableViewRow({title: &#39;Row 1&#39;}).

    Have you tried using tableView.search (a searchBar) and tableView.filterAttribute (the attribute in each row to use in the search)?

    Untested example:

    var searchBar = enlatele.createSearchBar({
    hintText: 'Filter this list'
});

var tableView = Ti.UI.createTableView({
    search: searchBar,
    filterAttribute: 'filter',
    data: [
        {title:'Row 1', filter:'one 1'},
        {title:'Row 2', filter:'two 2'}
    ]
});
