  • Hola Javier. If I understand your point correctly, you basically want to have some sort of a custom navBar for all your windows

    If this would be the case, I can think of some workarounds:

    • Use
      Titanium.UI.iOS.Toolbar
      instead (which would be more propper since its closer to the telephone's menu button)

    • Use a Ti.UI.createView( { layout: 'vertical' } ) with your menu and a Titanium.UI.ScrollView

    • Finally, each window you create, instead of adding the contents directly, put them into some sort of function createViewWithMenu(view) and use the same technique as above, only changing the ScrollView for your contents
