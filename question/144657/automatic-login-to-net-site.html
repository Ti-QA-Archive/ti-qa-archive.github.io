Automatic login to .NET site

I have an application with a webview that takes the user to a .NET based site. The initial page is a /Login.aspx page with username and password fields and submit button. I want to store the username and password for this site in my app and allow the user to be automatically logged in rather than be presented with this login screen.

I was hoping I could create my own html form with the username and password on which I submit onLoad (as I have done in the past for a PHP based login form) e.g 

webView.html = "<body onload='document.myform.submit()'><form name = 'myform' action='" + url + "' method='post' name='login'><input type='hidden' name='username' value='" + un + "'/><input type='hidden' name='password' value='" + pw + "' />Logging in...</form></body>";

This method doesn't seem to work for the aspx login page. Looking at the source of the site's login page I can see a __VIEWSTATE field - I'm guessing maybe I have to get to this? Does anyone have any advice on what I need to do here?
