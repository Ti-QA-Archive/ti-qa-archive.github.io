Difference between Titanium Mobile and Titanium Desktop

Are Titanium Mobile and Titanium Desktop two differnt frameworks? So if I only want to build mobile Apps do I have to use Titanium Desktop?

  • Allan is correct in his statements, but I think your question denotes a small confusion about the Appcelerator product line:

    • Titanium Desktop = A SDK that lets you develop Desktop based applications

    • Titanium Mobile = A SDK that lets you develop for mobile devices (and now web)

    • Titanium Developer = A no longer supported application that compiles both Titanium mobile and Titanium desktop code into apps

    • Titanium Studio = The replacement to Titanium Developer that is everything Titanium Developer was plus a full IDE and so much more!

    All of these products are separate products developed by Appcelerator
