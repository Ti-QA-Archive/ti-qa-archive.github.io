Ti.Blob does not return media width and height in Android

Hi,

I'm capturing a picture from the camera and I need to verify the layout of the media.

The problem is that the width and height of the Ti.Blob object are always 0.

How can I get the correct image width and height?

Thanks.

  • Hi Paolo,

    Try this code.

    
    success:function(event) {
        var image = event.media;

        Ti.API.info(image.height +' x '+ image.width);

        var imageView = Titanium.UI.createImageView({
            image:image,
            width:'auto',
            height:'auto'
        });

        image = imageView.toImage();
        Ti.API.info(image.height + " x " + image.width);
 }
