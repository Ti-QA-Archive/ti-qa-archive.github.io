Disable default zoomin/zoomout buttons on a Mapview

Application type: Mobile
Titanium SDK version: 1.9.0 (01/06/12 16:01 a3a1da4)
Device: Android emulator - Google APIs 2.2 
Host Operating System: Windows XP
Titanium Studio, build: 1.0.7.201112281340

I'm trying to create an application with a mapview. When the user interact with the map the zoomin and zoomout buttons automatically appear and disappear.

Is there any way to disable them appearing all together? I have created custom buttons to handle the zoomin and zoomout. So I would really like to disable the default ones.

Thanks

  • Total stab in the dark, set the mapview's "focusable" property to false. In native android you would set android:clickable="false" and this would do the trick, maybe focusable will do the same.

    Another idea would be to create a transparent view and place it on top of the mapview so that the mapview itself is never clicked. Then place your custom zoom controls over top of the transparent view. This obviously is a bad idea if you need to click other controls on the map, like annotations.

    I have to add, however, that it may be a bad idea to try to disable these and use your own, as you are going to be taking away the consistent look and feel of maps on Android.
