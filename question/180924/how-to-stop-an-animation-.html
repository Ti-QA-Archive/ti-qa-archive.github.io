How to stop an animation ?

Hello !

I have a problem with my animation.
I've searched on the Internet, but I can't solve my problem.
I have an application (actually for Android), like Flappy Bird, and I animate my background.
This is the code for the animation : 

var t1 = Ti.UI.create2DMatrix();
var a1 = Ti.UI.createAnimation();
var decalage = -((screenWidth + background.width) * (dpi / 160));
t1 = t1.translate(decalage, 0);
a1.transform = t1;
a1.duration = 3000*6;
background.animate(a1);

I have a penguin who is flying, and when this penguin touch the floor, I loose the game. When I loose, I can restart.
I want that when I loose, the animation stops, and when I restart, the animation restarts.
So, how can I stop the animation ?

I hope I'm clear. Thanks a lot !

  • It seems you can overwrite the current active animation. But you have to use the same animation type i guess. If for example you move the "left" and "top" properties with a translate matrix animation, you have to create a new translate matrix animation that sets the "left" and "top" properties again.

    To give you an idea, i've included a stand-alone example. It moves a view to left and back (5 seconds each). After 2 seconds, i overwrite the animation with a "fixed point animation" (duration is 0, so it just sets the view to the given position).

    I hope this idea may help you a little.

    $.win.open();

//create the background to animate
var background = Ti.UI.createView({
    backgroundGradient : {
        type: 'linear',
        startPoint: { x: '0%', y: '50%' },
        endPoint: { x: '100%', y: '50%' },
        colors: ['black','gray']
    }
});
$.win.add( background );

//create end point
var matrixEnd = Ti.UI.create2DMatrix();
matrixEnd = matrixEnd.translate( -500, 0 );

//create animation to end point
var animationEnd = Ti.UI.createAnimation({
    transform : matrixEnd,
    duration : 5000,
    autoreverse : true
});

//start animation
background.animate( animationEnd, function(){console.log('end');});

//stop the animation after 2 seconds
setTimeout(function() {
    //create matrix at start point
    var matrixStart = Ti.UI.create2DMatrix();
    matrixStart = matrixStart.translate( 0, 0 );

    //create "animation" to start point
    var animationStart = Ti.UI.createAnimation({
        transform : matrixStart,
        duration : 0
    });

    //overwrite current animation (will stop previous animation)
    background.animate( animationStart );
}, 2000);
