How to hide application name

I want to hide what you can see in the top part of my application: !(http://oi62.tinypic.com/2moe34y.jpg) - alloy logo + my app name. How do I do that? I found this question: (http://developer.appcelerator.com/question/153803/to-remove-app-name-on-the-top), and so I set:

<fullscreen>true</fullscreen>
<navbar-hidden>true</navbar-hidden>

in my tiapp.xml

<Window id="index" class="main-container" fullscreen="true" navBarHidden="true"></Window>

in my index.xml

$.index.fullscreen = true;
$.index.navBarHidden = true;

$.index.open();

in my index.js

Nothing works…

    Hi Marek,

    since Release 3.3.0, due to changes to support the appcompat library, this property has no effect. By default, the action bar is always displayed. To hide the action bar, see the Android Action Bar guide.

    There is also a nice blogpost from Ricardo Alcocer, about Hiding the Android ActionBar

    Greetings,

    Matthias
