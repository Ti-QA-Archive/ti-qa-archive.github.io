Likes Api of ACS

According to the Likes api: http://cloud.appcelerator.com/docs/api/v1/likes/info
We can create and delete a like. But how can we find out whether a user has liked that object or not?

— asked September 21st 2012 by Ammar Githam

  • Accepted Answer

    Likes only seem to keep count on an object. If you want to track who liked something, you may have to create a custom association between the object and the person who liked it.

    — answered September 21st 2012 by Anthony Decena
    permalink

    • How do I create that?

      — commented September 22nd 2012 by Ammar Githam

    • Create a Custom Object that holds the id of the object and the id of the users who liked the object. Then when you display how many likes something has, you can check the custom object to see if that logged in user is one of those likes.
