  • Hi Bunty,

    You can do it by so many way.. you can use default template of tableview row(leftImage & title Property). And if you need to customize it then also you can do it.. create imageview and label inside tableviewRow.

    And for storing reference of rowData you can set custom property of row and based on this property you can identify which row is clicked and which data should be passed to next screen

    e.g

    var arrRows = [];
for(var i=0;i<arr.length;i++){
    var tblRow = Ti.UI.createTableViewRow({
    height  : 100,
    width : "100%",
    layout : "horizontal",
    obj : arr[i], //CUSTOM OBJECT

});
var img = Ti.UI.createImageView({
    left : 10,
    height : 80,
    width : 80,
    image : arr[i].rowImage,
    touchEnabled : false
});
tblRow.add(img);
var lblTitle = Ti.UI.createLabel({
    left : 10,
    right : 10,
    top : 10,
    text : arr[i].title,
    touchEnabled : false    
});
tblRow.add(lblTitle);
arrRows.push(tblRow)
}
$.tblData.data = arrRows;

$.tblData.addEventListener("click",function(e){
    Ti.API.info('Clicked data = ' + JSON.stringify(e.source.obj));
});

    This code is not compiled because i don't have actual data.. so if you have any doubt then let me know..

    Thanks
