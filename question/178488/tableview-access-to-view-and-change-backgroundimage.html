TableView, access to view and change backgroundimage

Hi,
I create a tableview grid 3x5 with about 15 views. I want to change the view backgroundimage but I dont know I can access to the single view I need

here is my code:

for (var y=0; y<5; y++){
    var thisRow = Ti.UI.createTableViewRow({
        className: "grid",
        layout: "horizontal",
        height: cellHeight+(2*ySpacer),
        backgroundSelectedColor:"red"
    });

    for (var x=0; x<3; x++){
        var thisView = Ti.UI.createView({
            objName:"grid-view",
            objIndex:cellIndex.toString(),
            backgroundColor: colorSet[colorSetIndex],
            borderRadius:10,
            left: ySpacer,
            height: cellHeight,
            width: cellWidth
        });

        thisRow.add(thisView);
        cellIndex++;
        colorSetIndex++;
    }
    tableData.push(thisRow);
}

var tableview = Ti.UI.createTableView({
    data:tableData
});

I can access the single row with tableview.data[0] but how can I access to the three diffent "thisView" inside each row ??

Regards,
Matteo

  • I see two possible options, you can store each of your views in an array when adding them and use that to change them or you can call the .children property which will return an array of the children and then use that to change it.
