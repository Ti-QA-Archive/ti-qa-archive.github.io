    hi,

    actually , there is no code for background service in kitchensink 2.1.3 but just put the below code in your app.js file
    so you will get notification in notification bar

    if (osname === 'iphone' || osname === 'ipad') {

var service = Ti.App.iOS.registerBackgroundService({
        url : 'bg.js'
});
}

    and just see that bg.js is there in your resources folder..
