Start Animation

Hey,
I like to run an image down and when its at the bottom, it should reset and start from top again.
But if I take the animation method the reset also animates, although it shouldnt. My code:

var aktTop = 0;
win1.addEventListener('click', function(e){

aktTop +=200;    

image1.animate({        
      top:aktTop,
      duration:3000        
});

if (aktTop >= 400){
    image1.top =  0;    
    aktTop = 0;
}

});

Has anybody a solution? Thanks…

  • Animation support a callbacks for when the animation is complete. So you can combine one with a timeout to get your delay. For example:

    var win = Ti.UI.createWindow({
    backgroundColor : 'white'
});
var image = Ti.UI.createView({
    backgroundColor : 'black',
    height : 50,
    width : 50,
    top : 0
});
image.addEventListener('click', function() {
    image.animate({
        top : 200,
        duration : 1000
    }, function() {
        setTimeout(function() {
            image.animate({
                top : 0,
                duration : 1000
            });
        }, 500);
    });
});
win.add(image);
win.open();
