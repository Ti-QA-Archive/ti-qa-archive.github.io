  • I have the same problem. I am on linux.

    I can run java and javac from command line:

    [marc@arch .titanium]$ java -version
java version "1.6.0_24"
Java(TM) SE Runtime Environment (build 1.6.0_24-b07)
Java HotSpot(TM) 64-Bit Server VM (build 19.1-b02, mixed mode)

    [marc@arch .titanium]$ javac -version
javac 1.6.0_24

    I have JAVA_HOME set:

    [marc@arch .titanium]$ echo $JAVA_HOME
/opt/java/

    As well as android_sdk:

    [marc@arch .titanium]$ echo $android_sdk
/opt/android-sdk/

    I don't know how it can't find java on my machine in this condition.
