  • From that I read, it seems that your JSON is invalid.

    If you test the expression using http://jsonlint.com/, you will notice that there is an extra coma at the end of the second item of the array.

    Alexander is right when he says that Titanium will accept objects with string hash keys. Somehow, the parser seems to be less permissive when dealing with string keys. But once you resolve this, your code should work fine.
