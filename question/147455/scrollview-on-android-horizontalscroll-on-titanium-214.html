scrollView on android horizontalScroll on titanium 2.1.4

Hi,

I have a scrollView which scrolls horizontally,

my app was turning on titanium 2.1.2 on android and it worked fine but since i changed to 2.1.4..my scroll doesn't sroll.

here is my code :

var scrollAlbums = Ti.UI.createScrollView({
        bottom: 10,
        height: 95,
        contentWidth: 'auto',
          layout: 'horizontal',
          scrollType: 'horizontal'
    });

i tested with width: 'auto', contentWidth: Ti.UI.SIZE or contentWidth: Ti.UI.FILL without results….

thanks

    Hi

    Without seeing what you are adding to the view to confirm that the width of the items actually exceeds the width of the scrollView all I can do is tidy your code, which should improve things.

    var scrollAlbums = Ti.UI.createScrollView({
   bottom: 10,
   contentHeight: Ti.UI.SIZE, // add this
   contentWidth: Ti.UI.SIZE, // change this
   height: 95,
   layout: 'horizontal',
   showHorizontalScrollIndicator: false,
   showVerticalScrollIndicator: true, // should be a visual indication if can scroll
   scrollType: 'horizontal',
   width: Ti.UI.FILL // assuming you need it full width - if not specify a width
});

    Remember if the internal size is not bigger than the external size the nothing scrolls anyway.
