App splash freezes on device but not in emulator.

I am not really sure how to get any sort of debug information from a device, so I don't know how much information I can give. I've seen a few other people ask similar questions, but no answer was posted.

The app works fine in emulator. I've tried changing the iOS SDK from 4.2 to 4.1, to 4.0, same problem, the splash screen just stalls and never moves.

Using Titanium 1.5.1.

I installed a default app on the device to make sure it would work, and it did. I'll be happy to provide any more information, I'm just not sure what's relevant in this situation.

  • Figured it out. If you create a tab group and then never use it, it freezes the app on the phone but not in the emulator.

    This code had been left in from the default app:

    // create tab group
var tabGroup = Titanium.UI.createTabGroup();

    Hope this helps anyone else who might be having the same problem.
