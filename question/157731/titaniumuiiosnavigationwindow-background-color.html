Titanium.UI.iOS.NavigationWindow background color

Hello to all,
in my project I have a opaque navigation bar that has always a white color.
How can you give a specific color of the navigation bar on ios7?

  • Accepted Answer

    I encourage you to read the iOS7 Migration Guide as it covers this issue and many others for iOS 7 (and backward compatibility).

    NavBar color is controlled in one of these ways:

    • Set the Ti.UI.backgroundColor value
    • The barColor or tintColor properties can be passed as a parameter to the window's setToolbar() method (they do slightly different things)
    • You can set the window's navTintColor property

    The status bar inherits the navbar's color. It appears you have a choice of black (default) or white text on the status bar (at least I don't know of a way to have arbitrary status bar text color). If you choose a dark color for your navbar, make sure to set the window's statusBarStyle to Titanium.UI.iPhone.StatusBar.TRANSLUCENT_BLACK (if you're supporting only iOS7, you can use the Titanium.UI.iPhone.StatusBar.LIGHT_CONTENT value instead, but BLACK is equivalent and supported on iOS6/7).

    You could also add the following to your tiapp.xml to set all status bars to the light text style (though this won't have any effect on iOS 6):

    <ios>
    <plist>
        <dict>
            <key>UIStatusBarStyle</key><string>UIStatusBarStyleLightContent</string>
        </dict>
    </plist>
</ios>

    Subtleties, details, etc. are covered in the migration guide and docs.
