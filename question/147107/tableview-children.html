  • Hi

    Please explain your question with a little more detail as currently this is a little confusing.

    Do you mean the following;

    • Window displays a tableView containing several tableViewRows
    • Clicking on a single tableViewRow opens another window
    • More rows appear UNDER this tableViewRow

    or

    • Window displays a tableView containing several tableViewRows
    • Clicking on a single tableViewRow opens another window
    • The new window contains another tableView (or whatever you want)

    If it is the first one then - why? This is not a common technique but is possible. You would achieve this by adding data to a specific index position in the table using methods such as;

    • insertRowBefore
    • insertRowAfter

    If it is the second one then provide the code you use to create the table and I will take a look. In the mean-time this is the basic techqnique;

    tbl.addEventListener('click', function (e) {
    var newWin = Ti.UI.createWindow({
        height: Ti.UI.FILL,
        title: 'New Window',
        width: Ti.UI.FILL
    });
    var lbl = Ti.UI.createLabel({
        height: Ti.UI.SIZE,
        text: 'Put something on this window',
        width: Ti.UI.SIZE
    });
    newWin.open();
});

    If yourtableViewRow contained extra information then you can also read this in the event;

    tbl.addEventListener('click', function (e) {
    var newWin = Ti.UI.createWindow({
        height: Ti.UI.FILL,
        title: e.rowData.title, // see this line
        width: Ti.UI.FILL
    });
    var lbl = Ti.UI.createLabel({
        height: Ti.UI.SIZE,
        text: 'Put something on this window',
        width: Ti.UI.SIZE
    });
    newWin.open();
});

    Hopefully this will set you on the right direction.
