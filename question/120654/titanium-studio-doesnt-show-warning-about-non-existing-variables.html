Titanium Studio doesn't show warning about non existing variables

Hello,

if I code with other IDEs, I get warnings if I use a variable that is written wrong or that doesn't exist.

Example:

Titanium.UI.setBackgroundColor('#FFF'); 

var app = Ti.UI.createWindow({
    title:"main window"
});  

var butt = Titanium.UI.createButton({
    title:"myButton"
});

app.add(butt_wrong);
app.open();

The butt_wrong variable doesn't exist, but Ti Studio doesn't warn me. So I complile and the iOS Simulator gives me a warning. Thats too time consuming.

Same is true, if forget the parenthesis around the myButton String at the title property.

Q: Is there a way to activate this feature or is it simply not there (yet)? Since Studio is now a releases candidate, all features should be implemented.

Thanks in advance

Ray

    Ray

    Titanium Studio certainly includes js validation, and for my installation it was enabled by default.

    In your case, first confirm that validation is enabled, in TiStudio / Window / Preferences / Titanium Studio / Validation / JavaScript. I have both Mozilla and JSLint validators ticked. Then click Apply.

    Try right-clicking on your project, and click Indexing / Rebuild Index.

    Now all syntax js issues should be listed in the "Problems" tab.

    Hope this helps
