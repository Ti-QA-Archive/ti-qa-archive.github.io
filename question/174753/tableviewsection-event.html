  • Hi

    Quick answer before I leave;

    // outside loop
var sectionHandler = function (e) {
    alert(e.source.action);
});

// inside loop
var sectionHeader = Ti.UI.createView({
    action: 'Hi',
    backgroundColor: 'yellow',
    height: 50,
    width: Ti.UI.FILL
});
sectionHeader.addEventListener('click', sectionHandler);
var section = Ti.UI.createTableViewSection({
    headerView: sectionHeader
});

    I am assuming you are correctly using an event listener on the table for the rows. Sections for some weird reason do not accept events bubbling up into that.

    This is a quick example of how to add an event handler - first you need a view so you add one to the section by adding a view - this is how you grab the event.

    I have created a reusable handler so you can handler multiple sections if you need - just swap out the action.
