2.1.3 - Table View Stripped won't go away

var settingsWin = Ti.UI.createWindow({
    title: 'Settings',
    layout: 'composite',
    backgroundImage: 'mainbg.png'
});
var mainSettings = Ti.UI.createTableView({
    style: 1,
    scrollable: false,
    backgroundColor: 'transparent',
    backgroundImage: 'transparent'
});

I've tried throwing transparent everywhere, the window's background image is still the striped background. If I remove the tableview from the window, my custom background image shows up, but if the table is there, the window has the striped pattern. If I set the table's backgroundColor to something, the table itself does indeed change, but the window still has the stripped pattern.
