  • One word again: the config file (~.acs) must be encoded in the "UTF-8 without BOM" format. For example, with a proxy settings set to use the default port of a loca "Ctntlm Authentication Proxy" (a very good proxy!), it looks like this:
    {
    "proxy": "http://127.0.0.1:3128"
    }
