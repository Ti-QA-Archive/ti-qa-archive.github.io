Blur screen without dialog

When you pop up a dialog, the screen behind it is disabled and blurred.

Is there a way to disable/blur the screen like this, but without opening a dialog?

    Hi

    Yes and it is much easier than you think.

    Method 1

    • Create a view and give the background a colour with translucency.
    • This can cover everything but the title area, actionbar etc
    var view = Ti.UI.createView({
    backgroundColor: '#66000000',
    height: Ti.UI.FILL,
    width: Ti.UI.FILL,
});
    • The most important part here is the use of a colour based on argb (alpha, red, green, blue). This makes the black in our example partially see-through.

    Method 2

    • Create a window and give the background a colour with translucency.
    • This can cover everything including the title area, actionbar etc
    var winCover = Ti.UI.createWindow({
    backgroundColor: '#66000000',
    height: Ti.UI.FILL,
    width: Ti.UI.FILL,
});

    In both examples you MUST provide a way to dismiss the screen in your code or by the user pressing something on the cover and you having an event listener ready to hide/close the cover.

    Obviously you must show and the view/window when required. Also the cover MUST be added last so it will actually cover everything before it.
