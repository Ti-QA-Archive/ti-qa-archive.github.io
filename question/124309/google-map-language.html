Google Map language

Is there any method to let MapView display the other language?

The MapView defaults to display an english map.

I cannot found any about the display language function or property in API document.

    According to the Google Maps documentation you can't change the language displayed on the maps. The place names will always use the local languages. You can search in different languages, but they will always be displayed as the local language.
