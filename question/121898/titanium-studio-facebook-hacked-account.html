Titanium Studio, Facebook hacked account

Has anyone experienced, while using Titanium Studio, a hack on their Facebook account?

I've been logged in to my Facebook account while using Titanium Studio on my mac and I'm seeing a lot of strange activity.

I get e-mails from my facebook security settings with an enclosed message that I've requested a password change, which I have not.

Then once I recover my account I see that a 'workspace' has changed my password and logged in to my account from my IP address.

I'm not sure if anyone else has experienced this or if TS is only the 'host' in the attack.

Either way I would like to know of a way to prevent this from happening. Obviously, logging out of FB would probably work, but the fact that TS is either the host or seemingly involved in the hack is very troubling.

Thanks in advance!

  • I'm sorry to hear that's happened. Thank you for alerting us. We're pretty stumped how it could even be possible with Titanium Studio. One thought is password sniffing, but we use Eclipse's encrypted secure storage mechanism for password storage, and we send all authentication requests over HTTPS, so that basically eliminates that attack possibility.

    We'd like to investigate regardless. Could you please file a ticket here with any log files (Help > Titanium Studio > View Log File), relevant screen shots of the emails, and what projects were you working on. I would be interested to know if any of them were web projects or mobile projects, if you had been previewing them, and if they involved any Facebook integration.

    http://jira.appcelerator.org/browse/TC
