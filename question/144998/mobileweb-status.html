MobileWeb status

Hi, I wonder if anybody from Appcelerator can share their thoughts about mobileweb platform future. I've been experimenting with it for a while and found it to be unusable for anything beyond toy samples. I've created TC-1544 and TC-1543 with my recent findings. In short I've found three problems:

  • ScrollableView is unusable, it flickers, behaves strangely, sometimes show background instead of a next view
  • TableView deleteRow method does not work if I set table.data to array of sections.
  • ImageView size always equal to image size despite that I passed {width : 100, height : 100} to createLabel function.

These are pretty basic examples and I believe there are many more problems.

I'm ok with bugs, titanium is open source after all. What really bothers me is if Appcelerator and Titanium community really interested in MobileWeb at all? Is going to be production ready anytime soon?
