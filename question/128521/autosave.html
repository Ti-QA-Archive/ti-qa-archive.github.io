  • You can use an interval:

    //Auto-save every 10 seconds
setInterval(function () {
//Do saving such as to database or local file
}, 10000);

    You can detect when the user puts the app in the background:

    Ti.App.addEventListener("pause", function () {
//Do the saving because the app is going into the background
});
