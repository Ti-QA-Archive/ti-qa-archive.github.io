Titanium with Google Analytics

We created a module for google analytics, but when using this module, we get a build error.
[ERROR] : BUILD FAILED
[ERROR] : The following build commands failed:
[ERROR] : Ld build/Debug-iphonesimulator/GreenLightMobile.app/GreenLightMobile normal i386
[ERROR] : (1 failure)

The steps are:
a. download GoogleAnalyticsiOS_2.0beta4 from google website.
b. create a module project.
c. add sdk and libGoogleAnalytics.a into the module project.
d. add a line code "OTHER_LDFLAGS=$(inherited) /Library/libGoogleAnalytics.a" in module.xcconfig.
e. run ./build.py in terminal, it said build successfully.
f. But when we using this module in our application project. It gave the above error.
