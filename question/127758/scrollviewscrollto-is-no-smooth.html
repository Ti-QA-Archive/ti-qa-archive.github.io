  • Animation speed and smoothness is largely a function of the device you are running the animation on. Are you sure you are only running one scrollTo() at a time? It seems from your snippet above that you have the potential to be firing them in parallel. You may want to try adding logic to ensure that you only have one scrollTo() call at any given time.
