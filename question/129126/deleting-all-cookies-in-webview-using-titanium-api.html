Deleting all cookies in webview using Titanium API

Team,
My requirement is , a portion of my iphone window is loaded with Webview. This webview is configured to point to an existing web application. This web application is expected to have some cookie information in it. I have a exit button above the uiwebview, when i press that button, i wanted to clear all the cookies that has been present in the webview.

Kindly let me know how to clear all the cookies present in the webview from the button click or press event.

Point to note here is, i cant change the source of existing web application.

Thanks

  • Accepted Answer

    var path  = Titanium.Filesystem.applicationDataDirectory;
var searchKey = path.search('Documents');
path = path.substring(0,searchKey);
path = path + 'Library/Cookies/'
alert(path);
