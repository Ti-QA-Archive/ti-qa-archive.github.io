Configuring an android app for ACS.

I'm using the CloudPush service, and my code contains this :


var CloudPush = require('ti.cloudpush');
    CloudPush.retrieveDeviceToken({
        success: function deviceTokenSuccess(e) {
            Ti.API.info('Device Token: ' + e.deviceToken);
        },
        error: function deviceTokenError(e) {
            alert('Failed to register for push! ' + e.error);
        }
    });

when i test the app in my android device i get this error :

"Failed to register for push! Server refused request. Have you configured this app for ACS?"

Though i already enabled ACS for my app in (https://my.appcelerator.com/apps).

What else can i do to configure my app for ACS ?

PS : i'm working on an android app to use push notification using this ACS service.
