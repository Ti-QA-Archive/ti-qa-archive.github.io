Problem on android when add two label in a tableviewrow

I am new to Titanium and I have problem when I add two label into a tableviewrow. IPhone works well on that but Android prompt "closes unexpectedly" and kill the apps.
Here are the code:

        for (var c=0;c<items.length;c++)
        {
            var item = items.item(c);
            var title = item.getElementsByTagName("Title").item(0).text;
            var content = item.getElementsByTagName("Content").item(0).text;
            var row = Ti.UI.createTableViewRow({id:c,height:50});
            var view = Ti.UI.createView();
            var label1 = Ti.UI.createLabel({
                    text:title,
                    left:10,
                    top:0,
                    height:'auto'
            });
            var label2 = Ti.UI.createLabel({
                    text:content,
                    left:10,
                    top:row.height/2,
                    height:'auto',
                    font:8,
                    //color:'#F00'
            });
            if(content.length>=28)label2.text=content.substring(0,28)+'...';
            else label2.text=content;
            view.add(label1);
            view.add(label2);
            row.add(view);
            dataContent.push({title:title,content:content});
            data[x++] = row;
        }

There are no problem when I comment the line view.add(label2);
but I really need two.

I have also tried without using a view, it doesn't work too.

Can any one help?

Thanks a lot.

  • You don't need View to add labels to the table view row. Remove View and add labels directly to the row.

    row.add(label1);
row.add(label2);

    Also, define row's layout as "horizontal":

    var row = Ti.UI.createTableViewRow({id:c,height:50, layout: 'horizontal' });

    and remove position properties (left, right, top, bottom) from labels.

    Note that "font" property is object, if you want to define component font size, you'll need to do that this way:

    font: {
    fontSize: 8
}

    Documentation is wrong regarding that part.

    Also, row height is property of row's "size" property, not row's property:

    Ti.API.info(row.size.height);

    Use "push" to add element to the array:

    data.push(row);

    no need to worry about array index.
