I'm using 1.5.1 and trying to create a package to upload to iTunes. Upon clicking "Package", the message "Creating Distribution Package …" appears for a while. Then, that message is replaced with the Titanium logo. But, my package (zip) is not in the directory specified and nowhere to be found.

My distribution provisioning profile is correct and the app id in the provisioning profile matches the app id of my app.

I would greatly appreciate any help.
