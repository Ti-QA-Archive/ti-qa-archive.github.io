Namespace Problem

I have a namespace
v = {};
Another namespace v.ui = {}; as
(function() {
v.ui = {}
})();

Now how can i set v.ui.properties.labelView.height = 3
Thanks

  • (function() {
    var v = {
        ui: {
            properties: {
                labelView: {
                    height: 3,
                    width: 3,
                    etc...
                }
            }
        }
    }

    // Use line below if you want to use as CommonJS module
    //exports = v;
})();

    If you decide to use CommonJS module - include like this:

    var v = require('/file');

    Then you can call these properties like this:

    var labelView = Ti.UI.createLabel({
    height: v.properties.labelView.height,
    width: v.properties.labelView.width
});
