Android window navigation without TabGroup

Hello, wondering if anyone have any idea about my case.
I want to build an Android app that has 3 windows (let say: A, B and C) where A is shown first when the application starts.

In all windows, it has a navigation bar that gives you a link to all windows.

So A->B, A->C and B->A, B->C and C->A, C->B

Without using tabGroup, is there an alternative way to do this?

Thank you.

    fullscreen: false,

Ti.App.addEventListener ('openB', function(){ B.open(); });
Ti.App.addEventListener ('openC', function(){ C.open(); });

A.open();

      A.js/B.js/C.js

      var linkA = Ti.UI.createButton({ text: 'A' });
var linkB = Ti.UI.createButton({ text: 'B' });
var linkC = Ti.UI.createButton({ text: 'C' });
var currentWindow = Ti.UI.currentWindow;

linkA.addEventListener('click', function(){currentWindow.close(); Ti.App.fireEvent('openA')}); 
linkB.addEventListener('click', function(){currentWindow.close(); Ti.App.fireEvent('openB')});
linkC.addEventListener('click', function(){currentWindow.close(); Ti.App.fireEvent('openC')});

  • you are right Edwin,

    hide method is not working properly,but i have custom solution for that.

    In app.js create Tabgroup Like This

    var tabGroup = Titanium.UI.createTabGroup({
    visible : false
});

    and also write below code into same file

    var tab1 = Titanium.UI.createTab({
    //icon : 'KS_nav_views.png',
    title : 'A',
    window : A
});
tabGroup.addTab(tab1);

Ti.App.tabGroup = tabGroup;
tabGroup.open();

    Now There is no need to open A window.

    now write code for A.js

    var tabGroup = Ti.App.tabGroup;

A.addEventListener('open', function(e) {
    tabGroup.animate({
        top : -65,
        duration : 1000
    });

    setTimeout(function(e) {
        tabGroup.visible = true;
    }, 600);

});

    So This Way you Can hide your Tabgroup and also you can use it's navigationBar.

    Hope This will work for you.

    It's working fine for me.
