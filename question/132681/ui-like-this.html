    Definitely you will need images but what type of images you need and where you need that you have to decide.

    Basically If I want to achieve the design provided in the screen shot I will do following thing (Just a basic overview).

    • Above Tab Bar is not looking like a native built in tab bar. If you want to achieve same tab bar than you have to create your own custom tab bar (You can create it using views and windows whatever way you prefer better).
    • Below that there is view and inside that you can have buttons or labels and you can change the background images of that labels or buttons according to state of it (whether it is selected or not)

    • Below that you can have a view and inside that view you can have a label to show the heading.

      -And finally you will need a table view with custom rows.

    Hope this will make the things clear.
