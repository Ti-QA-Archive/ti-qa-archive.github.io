Android : How to display an online pdf file

Hi,

In IOS I can use a WebView.
What about Android ?

I have tryed something like this but it doesn't work :

try {
                Ti.Android.currentActivity.startActivity(Ti.Android.createIntent({

                    type : 'application/pdf',
                    data : pdf_url
                }));
            } catch (err) {
                var alertDialog = Titanium.UI.createAlertDialog({
                    title : 'Pas de visualiseur PDF',
                    message : "Impossible d'ouvir un fichier pdf. Voulez-vous installer un lecteur PDF ?",
                    buttonNames : ['Oui', 'Non'],

                alertDialog.show();
                alertDialog.addEventListener('click', function(evt) {
                    if (evt.index == 0) {

                            action : Ti.Android.ACTION_VIEW,
                            data : 'market://details?id=com.adobe.reader'
