  • this use to happen when you use the back button and there's only one window on stage. The main window should use the exitOnClose property to ensure the apps closes when there's no more windows open. More info in UI.Window documentation here

  • You can try added the modal: true on the second window. This is to ensure that you will go back to the first window.

    eg:

    var secondWindow= Ti.UI.createWindow({
    backgroundColor: '#FFFFFF',
    modal: true
});
