NavBar without tabs?

I've searched the forums and got mixed messages, so I was wondering if anyone knows if we can generate a window that has a navbar w/o using a tabgroup? I would think the simple code below should generate a window with a navbar, yes? Or am I missing something:

var win = Ti.UI.createWindow({title:'test window');

  • Hi,

    I think the way to do it is like this:

    // Create your tabGroup
var tabGroup = Titanium.UI.createTabGroup();

// Create window and set the visibility of your tabGroup to false and navBar to true
var win = Ti.UI.createWindow({
    title:'test window', 
    navBarHidden: false, 
    tabBarHidden: true
});

// Open the tabGroup
tabGroup.open();

    Hope that helps? :)

    /Anders
