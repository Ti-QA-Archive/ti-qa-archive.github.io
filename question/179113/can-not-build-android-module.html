    Hello Georgi,

    I think your build.properties file configuration is not prepared correctly. At first download android NDK and browse it to the studio preference >Studio >Android >Android NDK Home.

    And then open the build.properties file and add the Android NDK location like this
    android.ndk=/Users/gsl001/Library/android-ndk-r10

    You will get this location from Android NDK Home easily.

    After all run your project
    Right click on buid.xml -> Run as -> Ant build

    Just looks this image.

    If you have any question then feel free to ask me.

    Thanks.
