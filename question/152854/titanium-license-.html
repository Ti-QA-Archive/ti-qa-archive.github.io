  • Hi

    The free and paid versions use the same SDK, the main differences for the paid versions are the extra support, extra debugging and profiling tools and increased ACS (Appcelerator Cloud Services) allowances. There are more differences but only in the extras including enterprise only modules but you get the same main SDK.

    So get the free edition and then write your app, give it away, charge for it - you choose. But if you want or need the extras then pay for for the paid versions instead.
