Imageview Does not display remote jpeg image

Hi,

As the title says remote jpeg image does not get loaded,the default image shows up then disappears but it works for png images.

here is the code

// create tab group
var tabGroup = Titanium.UI.createTabGroup();

var win1 = Titanium.UI.createWindow({
    title:'Tab 1',
    backgroundColor:'#fff'
});
var tab1 = Titanium.UI.createTab({
    icon:'KS_nav_views.png',
    title:'Tab 1',
    window:win1
});

var imageView = Titanium.UI.createImageView({
    image:'http://topnews.in/files/stephen-hawking1.jpeg',
    width: 'auto',
    height: 'auto',
    top: 10,
    bottom:10
});
win1.add(imageView);

tabGroup.addTab(tab1);

tabGroup.open();

Environment - Ti studio, Ti SDK - 1.6.2, ios - 4.3

is there any work around ???

    I might suggest fiddling with those width/height top/bottom values as you might find a png image for versitile than a jpeg. Try something like:

    var imageView = Titanium.UI.createImageView({
    image:'http://topnews.in/files/stephen-hawking1.jpeg',
    height: theImageHeight,
    width: theImageWidth,
    top: 10
});

    I find providing 'auto' for height and width is not always a great combination when providing a top and bottom value.
