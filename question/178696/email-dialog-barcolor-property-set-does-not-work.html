Email Dialog barcolor property set does not work

I'm trying to use the code below to set the barcolor of the email dialog modal window, I know it's redundant but these are the two ways I used to try and set it:

var emailDialog = Ti.UI.createEmailDialog({
    barColor : 'green'
});
emailDialog.barColor = 'green';

This is being tested on the iOS 7.1 simulator and Titanium SDK 3.3.0.GA.

    Hi Obando,

    Thanks for your concern.

    The barColor property works as expected in iOS 6 but fails in iOS 7. This is either an Apple bug or a decision they have made. The email composer is provided as is by the OS with very little options for customization.

    So check the Jira Ticket for this issue: iOS: Email Dialog barColor does not work

    Hope it'll help you to understand why it's not working.
