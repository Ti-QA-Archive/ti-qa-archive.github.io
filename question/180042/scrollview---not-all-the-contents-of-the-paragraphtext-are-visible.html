ScrollView - Not all the contents of the paragraph(text) are visible

Hi guys,

I have created a scroll view for android application, with the scrollType as vertical, here I have two problems,

  1. I can't see all the contents of my paragraph(text) inside the scroll view and I could not scroll further down
  2. Here only the top half of the text is visible when I scroll to the bottom and I could not scroll further down.
    ~~~
    var contentScrollView = Ti.UI.createScrollView({
    contentWidth: "auto",
    contentHeight: "auto",
    borderColor: "#c6c6c6",
    scrollType: "vertical",
    // showVerticalScrollIndicator: true,
    // showHorizontalScrollIndicator: true,
    height: "60%",
    width: "70%"
    });

var lblStaticText = Ti.UI.createLabel({
text : "A view that contains a horizontally and/or vertically-scrollable region of content. Use the Titanium.UI.createScrollView method or <ScrollView> Alloy element to create a scroll view. Views added to the scroll view will be scrolled based on the size of the scrollable region of content. If a scrollable region fits within the size of its scroll view, the view will not scroll. ",
textAlign : Ti.UI.TEXT_ALIGNMENT_CENTER,
color : "#1f753a",
top : "5%",
left : "3%",
right : "3%",
font : {
fontSize : "14%"
}
});

contentScrollView.add(lblStaticText);
// parent.add(contentScrollView);
~~~

In the first case I am seeing the text till If a scrollable region In the second case I am only seeing the top half of the If a scrollable region. Where lies the problem ?

Thanks
