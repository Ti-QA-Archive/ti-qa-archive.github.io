    You have to understand that, on Android, the application's life-cycle is different.

    And simply clicking the home button won't pause your application's activity. (although it will eventually).

    One good way go get a feel of how Android applications behave on the device is to log something on every item. (you can view those using DDMS).

    Simply put, your application will have to adhere to the Android logic since there is no way to fake the same behavior on both environments.

    • On android there is a similar thing using the currentActivity. Once you get that, you can add event handlers to the pause event.

      There is no "cross-platform" way to achieve this, so you will have to apply a different method depending of the OS you are running.
