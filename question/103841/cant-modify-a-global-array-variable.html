Can't modify a global array variable

Hi,

I'm using a global array variable for books categories:
(Cat is category name, used is a 0/1 int for internal use)

Ti.App.TabCategorie=[
    { Cat:"Roman", used:1 },
    { Cat:"Théatre", used:1},
    { Cat:"Poésie", used:1},
    { Cat:"Essai", used:1},
    { Cat:"Documentaire", used:1},
    { Cat:"DB Manga", used:1},
    { Cat:"Roman Classique", used:1},
    { Cat:"Roman Jeunesse", used:1},
    { Cat:"Roman bit litt", used:1}
];

Ok, it works fine, and I can read this values like this for example:

text = Ti.App.TabCategorie[i].Cat

But now I want to modify this category name, like this for test:

Ti.App.TabCategorie[2].Cat="test";
alert(Ti.App.TabCategorie[2].Cat);

Nothing appends, no error, but the value isn't modified !
I don't understand where is the code error…

Do you have an idea ?

thanks you.

    I have no idea why and YMMV. Just passing along my own experience.

  • You're right Doug, it works with basic variable and objects (label, etc) but not with array.

    After some working hours, here is the solution, by using Properties.
    That's not as easy as a simple Array but it's a better code :-)

    Here is some simple code for using Properties with an array:
    Creating a standard array:

    var TabCategorie=[
{ ID:0, Cat:"Roman", used:1 },
{ ID:2, Cat:"Théatre", used:1},
{ ID:4, Cat:"Poésie", used:1},
{ ID:6, Cat:"Essai", used:1},
{ ID:8, Cat:"Documentaire", used:1},
{ ID:10, Cat:"DB Manga", used:1},
{ ID:12, Cat:"Roman Classique", used:1},
{ ID:14, Cat:"Roman Jeunesse", used:1},
{ ID:16, Cat:"Roman Vampire", used:1}
];

    Sending the Array to Properties (Categorie is the Array/List name)

    Ti.App.Properties.setList('Categorie',TabCategorie);

    Read directly an array cell:

    alert("Catégorie: " + Ti.App.Properties.getList('Categorie')[0].Cat );

    Array length ?

    Titanium.App.Properties.getList('Categorie').length;

    Testing if an existing or not cell:

    if ( Ti.App.Properties.getList('Categorie')[2]==undefined)
