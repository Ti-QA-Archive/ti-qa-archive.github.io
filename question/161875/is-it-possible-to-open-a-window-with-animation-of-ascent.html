Is it possible to open a window with animation of ascent?

As far as I know, the only way is to set model:true when creating a window. However, this is not what I want to do since it also covers my loading mask which is a window - Model window will open on top of any other windows.

So, can I somehow achieve the same animation but without using a model window?

    Hi

    • Create your window without animation.
    • Work out device width and height and set your window to these two values (do not use percentages or Ti.UI.FILL).
    • Set the window to have a top equal to the device height.
    • Add the loading mask as a normal window over the top of your animating window.
    • You then animate your ascending window by setting top to 0
    • Add an event to this animation and on 'complete' remove the loading mask window.

    Hope this helps!
