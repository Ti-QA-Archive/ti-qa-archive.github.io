How to close previous window after an action had been executed in the current window.

  • Hi

    Your question is either not very clear or is clear but what you want is not a good idea. You have not said if the windows are opened modally, within a tabgroup or distinctly.

    • If the new window is modal;

      If you open a new modal window, you should not be closing a previous a window opened below this one

    • tabGroup

      If you open a window as part of an iOS tabgroup nav stack then you should not close one previously added to this tab

    • You open a new distinct window

      You should not close a window opened previously below this window

    You may have spotted a pattern here. If you open a new window, actions on this window should not close a window you have already opened within this navigational path the user has be taken through.
