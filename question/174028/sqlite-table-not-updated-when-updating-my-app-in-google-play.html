Sqlite Table Not Updated when updating my app in Google Play

Hi All,

i am using external sqlite file in my app.and i published my app last time.

now i am updating my app.and added one field to table in same sqlite file.then updated this app in google play.

but after updating my app.the field not reflected in new app.what is the solution for this?

###note : if i am Clear Data of my updated app,it works fine.but i thought its not correct way.because we can't tell to users to use Clear Data option.

please give you suggestions for my problem.

Thank you.

    If the SQLLite DB has already been created, when you update the app, the DB isnt going to be created again, so you need to update the table with the additional field.

    If you are using Alloy, then you will have to follow the migration methods described here.

    If you are not using Alloy, you need to check the table has got the field, and if not ALTER the table to have the field.
