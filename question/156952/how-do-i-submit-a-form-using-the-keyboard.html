How do I submit a form using the keyboard?

I have two TextFields, one a password, and I'm using returnKeyType="Titanium.UI.RETURNKEY_GO". But I want it to actually fire the same function as what my Submit button fires.

Right now pressing "Go" doesn't do anything. I'd like it to send the data to the server (I have another function to handle all that, and it works fine, I just need to be able to fire it from the keyboard's "Go" button). I'm thinking it has to do with the "fireEvent()" method of TextField? Not sure of the syntax to implement that, though.

<TextField id="loginEmail" borderStyle="Ti.UI.INPUT_BORDERSTYLE_ROUNDED" keyboardType="KEYBOARD_EMAIL" autocapitalization="TEXT_AUTOCAPITALIZATION_NONE" title="Email Address" hintText="Email Address" maxLength="80" width="220" height="30"></TextField>

<TextField id="loginPassword" borderStyle="Ti.UI.INPUT_BORDERSTYLE_ROUNDED" title="Password" hintText="Password" passwordMask="true" maxLength="30" width="220" height="30" returnKeyType="Titanium.UI.RETURNKEY_GO"></TextField>

Please help!
Thanks.

    On the last textfield(loginPassword), add a listener on return key like:

    <TextField id="loginPassword" onReturn="submit()"/>
