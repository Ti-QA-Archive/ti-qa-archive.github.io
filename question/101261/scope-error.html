Scope Error

I seem to be running into a scope problem with Titanium. To the best of my knowledge the scope of a variable is the set of brackets {} in which it is declared and any nested brackets. If the item is declared in a for loop the scope is the nested statement(s). So:

        if(true){
            var ss = true;
            Ti.API.info(ss);
        }
        if(ss){
            Ti.API.info("This shouldn't print");
        }

However, the variable ss is accessible outside it's own scope and is available to the second if statement.

Can anyone confirm that this is incorrect according the the ecma script spec?
