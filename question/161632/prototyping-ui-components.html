Prototyping UI components

App Type: Android SDK: 3.2.0 Device: Galaxy S3 i9300 Version: Android CM 4.3.1 Host: OSX 10.9.1 Alloy

Alloy: how can I extend some UI components? for example:

TI.UI.ImageView.prototype = {
    move: function(myParam){
        // manipulate the imageviewobject    
    }    
};

And then, put this on xml

<ImageView move="doIt" image="/images/logo.png"></ImageView>

Tks

    The Titanium proxies are not true JavaScript objects, instead they represent native objects. Therefore, you cannot extend them, or at least should not. Doing so can cause race conditions and crashes.

    For what you seem to be attempting in your sample above, you could do:

    // in the controller
function move() {
   // do whatever...
}

    In the XML

    <ImageView onLoad="move" image="/images/logo.png"/>

    This will fire the move() function once the image file is loaded into the ImageView.
