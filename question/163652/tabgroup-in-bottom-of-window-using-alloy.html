TabGroup in bottom of Window using Alloy

Hello!

Could someone suggest how can I move tabGroup at bottom of Window using Alloy?I use this code for TabGroup

index.xml

<Alloy>
    <TabGroup activeTab="makerequest_tab" >
        <Tab id="makerequest_tab" title="Make A request" icon="KS_nav_ui.png">
            <Require src="makerequestTab" type="view" /> 
        </Tab>
        <Tab title="Settings Tab" id="settings_tab" icon="KS_nav_views.png">
            <Require src="settingTab" type="view" /> 
        </Tab>
    </TabGroup>
</Alloy>

makerequestTab.xml

<Alloy>
    <Window title="Tab 1" backgroundColor="white">
        <Label>I am Window 1</Label>
    </Window>
</Alloy>

settingTab.xml

<Alloy>
    <Window title="Tab 2" backgroundColor="white">
        <Label>I am Window 2</Label>
    </Window>
</Alloy>

    On iOS they are by default at the bottom.
    On Android I wouldn't recommend this for 2 reasons:

    1. You are breaking the user's expectations. Android users are uses to have the tabbar at the top.
    2. Android 3+ uses the actionBar that integrates the tabs in a better way then the older versions.

    • The reason Android tabs are at the top is because the hardware buttons (Back, Home, Recent Apps) are at the bottom – it would be too easy for someone to hit the wrong button if tabs were also at the bottom. What I'm saying is that it's a bit more than "breaking users' expectations" as Dan says. You'd be hurting user experience and increasing user frustration if you put tabs at the bottom.
