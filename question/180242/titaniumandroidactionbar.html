  • Hi Husain John Rizvi, For expanding and collapsing action items, the target SDK version must be at least 14. To set the target SDK version, add code like this in your tiapp.xml file.

    <android xmlns:android="http://schemas.android.com/apk/res/android"> 
    <tool-api-level>14</tool-api-level> 
    <manifest> 
        <uses-sdk android:targetSdkVersion="14"/> 
        <!-- other manifest entries --> 
    </manifest> 
</android>
