    Hopefully people can add their comments to this question, up vote the question to help it stay in focus.

    First thing to note, the official support for a new OS is 1 month from release and usually in the form of compatibility. New features come after this and are provided based on ease of provision and majority benefit. Some may come as modules which is the new way Appcelerator/Titanium is now providing things, unless of course the "new" thing is system wide or an extension to the old thing.

    Should you start testing your apps on iOS 7 (compile them), yes - start checking things out, but right now you will not know if any issues are your code, Titanium needing a tweak or the fact iOS7 is a beta.

    • This seems to be a change in iOS7 that no longer includes 'localhost' in the file path when referencing a locally stored file. I was able to resolve this issue for now by changing the pathFromComponents function in FilesystemModule.m to no longer check for "file://localhost/" and instead just look for the prefix of "file://".

      Original Function in FilesystemModule.m:

      -(NSString*)pathFromComponents:(NSArray*)args
{
    NSString * newpath;
    id first = [args objectAtIndex:0];
    if ([first hasPrefix:@"file://localhost/"])
    {
        NSURL * fileUrl = [NSURL URLWithString:first];
        newpath =[fileUrl path];
    }

      Function after change:

      -(NSString*)pathFromComponents:(NSArray*)args
{
    NSString * newpath;
    id first = [args objectAtIndex:0];
    if ([first hasPrefix:@"file://"])
    {
        NSURL * fileUrl = [NSURL URLWithString:first];
        newpath =[fileUrl path];
    }

      Hope this helps someone with testing for iOS7 beta!

  • In FilesystemModule.m I alter to support iOS 7 and above.

    if (v == 7){
    if ([first hasPrefix:@"file://"])
    {
    NSURL fileUrl = [NSURL URLWithString:first];
    //Why not just crop? Because the url may have some things escaped that need to be unescaped.
    newpath =[fileUrl path];
    }
    else if ([first characterAtIndex:0]!='/')
    {
    NSURL     url = [NSURL URLWithString:[self resourcesDirectory]];
    newpath = [[url path] stringByAppendingPathComponent:[self resolveFile:first]];
    }
    else
    {
    newpath = [self resolveFile:first];
    }

        if ([args count] > 1)
    {
        for (int c=1;c<[args count];c++)
        {
            newpath = [newpath stringByAppendingPathComponent:[self resolveFile:[args objectAtIndex:c]]];
        }
    }

    return [newpath stringByStandardizingPath];
}
else{

    if ([first hasPrefix:@"file://localhost/"])
    {
        NSURL * fileUrl = [NSURL URLWithString:first];
        //Why not just crop? Because the url may have some things escaped that need to be unescaped.
        newpath =[fileUrl path];
    }
    else if ([first characterAtIndex:0]!='/')
    {
        NSURL* url = [NSURL URLWithString:[self resourcesDirectory]];
        newpath = [[url path] stringByAppendingPathComponent:[self resolveFile:first]];
    }
    else
    {
        newpath = [self resolveFile:first];
    }

    if ([args count] > 1)
    {
        for (int c=1;c<[args count];c++)
        {
            newpath = [newpath stringByAppendingPathComponent:[self resolveFile:[args objectAtIndex:c]]];
        }
    }

    return [newpath stringByStandardizingPath];

}
