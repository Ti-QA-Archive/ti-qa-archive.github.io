Change back button text

SDK 1.7.2 mobile

I like to change the text of a back button. But I need the text of the current back button before I can do that. But backButtonTitle is always empty.

alert(Titanium.UI.currentWindow.backButtonTitle);

Any Ideas?

— asked September 3rd 2011 by Holge Klein

    If backButtonTitle is empty (undefined), it means the back button's title is the title of the previous window in your navigation stack.

    The backButtonTitle property will only return a value if you set this property yourself.
