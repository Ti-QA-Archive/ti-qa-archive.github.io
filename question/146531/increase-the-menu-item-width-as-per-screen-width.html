  • Hi. When developing with Alloy I believe the following could work:

    if (Alloy.isTablet) {
    menuItem1.width = Ti.Platform.displayCaps.platformWidth;
    menuItem.width = Ti.Platform.displayCaps.platformWidth;
}

    Otherwise:

    if (Ti.Platform.displayCaps.platformWidth > 320)
{
    menuItem1.width = Ti.Platform.displayCaps.platformWidth;
    menuItem.width = Ti.Platform.displayCaps.platformWidth;
}
