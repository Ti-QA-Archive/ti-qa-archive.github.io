Announcement: Cloud Integration Module

This is not a question so I will try to keep it brief. I wanted to let the Titanium Community know about a new Cloud Module that we have developed at the OpenMobster: Open Source Mobile Cloud Platform Community.

The module provides the following features:

App Data Synchronization: Automatic bidirectional data sync between the device and the Cloud.

Push Notifications: A platform-agnostic Cloud-initiated Push Notification System.

Mobile RPC (Remote Procedure Call): A simple name/value pair based method for invoking service components in the Cloud.

Both iPhone and Android are supported.

You can play around with this using the Sample App found here:
http://code.google.com/p/openmobster/wiki/TitaniumGetStarted

We would really like your feedback on the module so we can work towards refining it even more

Thanks
Sohil
Chief Engineer, OpenMobster
