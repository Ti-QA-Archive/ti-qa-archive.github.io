  • I believe it is stored in the db in unixepoch format. That would be the easiest format to use in your between clause.

    In my apps I've taken two strategies.

    1. I use the Julian Sqlite function, then convert my query parameter in JavaScript into that format.

    2. I just store my "now" column as getTime(). This is by far the easiest.

    Here are all of the Sqlite date functions, Titanium supports almost all of them http://www.sqlite.org/lang_datefunc.html
