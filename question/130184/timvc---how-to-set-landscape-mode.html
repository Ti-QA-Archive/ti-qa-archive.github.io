TiMVC - How to set Landscape mode

How is the best way to set Landscape mode in TiMVC? In view file or layout? And how do I do that? I tried self.App.window.orientationModes = [Ti.UI.LANDSCAPE_LEFT, Ti.UI.LANDSCAPE_RIGHT]; but it didn't work.

  • In normal way,
    I do this:

    //rootWindow
var rootWin = Ti.UI.createWindow({
   orientationModes:[Ti.UI.LANDSCAPE_LEFT, Ti.UI.LANDSCAPE_RIGHT]
});

  • u can do changes in Tiapp.xml…………

     <iphone>
        <orientations device="iphone">
            <orientation>Ti.UI.LANDSCAPE_LEFT</orientation>
            <orientation>Ti.UI.LANDSCAPE_RIGHT</orientation>
        </orientations>
        <orientations device="ipad">
            <orientation>Ti.UI.LANDSCAPE_LEFT</orientation>
            <orientation>Ti.UI.LANDSCAPE_RIGHT</orientation>
        </orientations>
    </iphone>

    and for android:
    window.orientationModes = [Ti.UI.LANDSCAPE_LEFT,Ti.UI.LANDSCAPE_RIGHT];
