  • Right click 'Computer' -> 'Properties' -> 'Advanced system settings' -> 'Environment Variables'

    Under System variables, look for 'android_sdk' variable and enter the path to your android sdk (e.g. C:\android). If variable does not exist, create new.

    I had the same issue as well but this method resolve it. I hope it works for you as well.
