  • Phanindra

    I have installed Titanium on Ubuntu 10.10 in the last few weeks and it worked fine, so I can definitely help you get it running.

    First, make sure you follow the Preparing for Android development guide.

    After this, copy and paste the following into your terminal/shell. Copy the output and paste it to your preferred pastebin service. It's important that you do not modify it in any way, as this could mask problems with your configuration. Do NOT paste it here! This is because the system will change the formatting.

    clear
cd /
uname -a
echo "$PATH"
echo "$JAVA_HOME"
java -version
javac -version
python -V
echo "$android_sdk"
adb version
android list target
git --version
echo " --- The following is optional ---"
scons -v
echo "$HTTP_PROXY"
echo "$HTTPS_PROXY"
echo "$SOCKS_PROXY"

    Then I will help you further.

    Cheers
