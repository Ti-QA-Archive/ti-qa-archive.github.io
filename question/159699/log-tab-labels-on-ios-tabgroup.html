Log tab labels on iOS TabGroup

I have two tabs that appear right next to each other on the iPad with zero padding, but are evenly laid out across the width of the screen on the iPhone. Is there any way to spread out the tab buttons on the iPad so that the text doesn't run onto each other? I don't see anything in the docs about this, and the 'left' or 'width' properties don't have any effect. I am creating the tab like this:

    var phraseTab = Ti.UI.createTab({
        title: L('Daily Words of the Buddha'),
        window: phraseWin,
    });

Thanks!

    Hi

    Short answer - no (but a little bit yes).

    If you are using the built in tabs they are designed to use an equal width each and each tab has a fixed width when on the ipad and those are all grouped together in the centre. This is not something that Titanium does - this is part of the human interface guidelines by Apple and built directly into the iOS.

    You CAN however create custom tabs and these can look how you want them - it is much more fiddly than the built in way but might be a solution to this problem - just keep in mind the minimum width and keep the human interface guidelines from app in mind (available via your dev account with apple login).

    This is AN example - not necessarily the best but you can google "appcelerator custom tabs" for more;
