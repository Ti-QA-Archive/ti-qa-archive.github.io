  • On Android, if the height of the scrollview is sufficient to show all its contents, scrolling is disabled. If there's another solution than that, I don't know it.

    In other words, if scrollview.height >= scrollview content height == no scrolling

    (btw, same applies to tables and lists)

  • Hi

    I have read this question a little differently than Tim, so just in case.

    If you want to stop a ScrollView from scrolling on Android or IOS you use the property scrollingEnabled. It has been available since Ti SDK 3.0 on both platforms.

    However on a TableView only iOS supports turning off the scroll using scrollable.
