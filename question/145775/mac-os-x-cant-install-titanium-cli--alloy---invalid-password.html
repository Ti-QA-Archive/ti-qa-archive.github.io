  • I had the same problem, and nothing here seemed to help 100% but everything got fixed when I discovered "homebrew". http://brew.sh. As the website says, paste this at a command prompt:

    ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)"

    This will install homebrew. After that you can just say "brew install this" and "brew install that" for instance,

    brew install wget

    brew install npm

    brew install node

    So everytime Titanium Studio takes you into the endless loop of not being able to install something, just kill it and brew install whatever it was unable to do. Then start up Titanium Studio again and it seems to be happy.
