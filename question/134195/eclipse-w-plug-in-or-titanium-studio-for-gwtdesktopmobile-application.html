Eclipse w/ Plug-In or Titanium Studio for GWT/Desktop/Mobile Application?

I am configuring my environment for Titanium development and have installed Titanium Studio. I have been developing a GWT (Java compiled to Javascript) web application in Eclipse with GWT and Android ADT plug-ins. Now I have the requirement to develop a desktop client version of the app, that has access to system.

Based on http://code.google.com/p/gwt4air/wiki/TitaniumDesktop, the strategy is to create a GWT project (include the gwt4air jar), put the compiled(JS) output into the /resources of a "shell" titanium application, and then run/test/deploy using Titanium Studio.

This is a workable solution, but I would like to have the same features in Eclipse. Does installing the Eclipse plug-in give me all of the features of Titanium, or should I look into installing my existing plug-ins into Titanium Studio and work exclusively from there? IOW, are there any native features of Titanium Studio that can't be included into Eclipse via plug-ins?
