How to smoothly track the current user's location on a map, and how to detect a user map drag?

iOS Application with Alloy

I'm currently tracking the user's location with the following code but it doesn't follow the user's location smoothly like other iOS apps:

Titanium.Geolocation.accuracy = Titanium.Geolocation.ACCURACY_BEST;
Titanium.Geolocation.distanceFilter = 0; // Have tried 1 as well
Titanium.Geolocation.preferredProvider = "gps";

Titanium.Geolocation.addEventListener('location', function(event) {
    setCurrentLocation(event);
});

function setCurrentLocation(event) {
    var current_region = {
        animate: true,
        latitude: event.coords.latitude,
        longitude: event.coords.longitude,
        latitudeDelta: .005,
        longitudeDelta: .005
    };

    $.mapview.setLocation(current_region);
};

I also want to stop tracking when the user drags the map around, I can't see any drag events on the map view. Does anyone have any suggestions?

  • You can use the regionChanged event to monitor for drag events. You should also consider using the activityType property to help iOS provide you with the most efficient type of location updates and to save battery life.
