Long text align beginning of text to top of TextArea

I am developing a Alloy Mobile App with Ti SDK 3.1.3-GA for IOS and Android Handheld and tablet devices.

On an Info page I have a simple Label and a TextArea with text that will scroll off most handheld screens.

In info.tss

"Label":{
  text:'View Title',
  font: {
    fontSize: 20,
    color: "white", 
  },
}

"infoArea": {
  color: "white",
  editable: false,
  scrollable: true,
  backgroundColor: 'Transparent',
  value: "Long text that scrolls off the screen with small device screen sizes",
  top: 50,
  bottom: 10,
  left: '7%',
  right:'7%',
  verticalAlign: 'top',
}

In info.xml

<Alloy>
   <Window>
        <Label/>
        <TextArea id="infoArea"/>
   </Window>
</Alloy>

The problem is when the view is loaded the scrollable text is not aligned with the beginning text at the top, but it appears the end of the text is aligned with the bottom of the defined TextArea. There has to be a parameter which allows Alignment of the Text in a TextArea of Beginning Text align to top.

Any help would be greatly appreciated.
