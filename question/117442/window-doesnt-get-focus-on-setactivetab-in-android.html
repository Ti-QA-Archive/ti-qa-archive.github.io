Window doesn't get focus on setActiveTab() in Android

I use setActiveTab() in a tableview click event in order to change tab. When I change back to the tab I was and click another row in the tableview, although the tab changes, the window doesn't get the focus. I have a focus event in the window and that doesn't fire. In iPhone all work great. What can I do? I'm frustrated…
PS. I use Titanium SDK 1.5.1. In later versions the window doesn't get the focus at all, even the first time. In both versions all look great in iPhone.

  • Hi Luc,
    i had the same problem, you have to fired focus event from the tabgroup.

    // tabgroup.js

var tabGroupApp = Ti.UI.createTabGroup({});
//...

tabGroupApp.addEventListener('focus' , function(e){
   (e.tab.getWindow()).fireEvent('focus');;
});

//...
tabGroupApp.open();

    // win.js
var self = Ti.UI.createWindow({});

//...

self.addEventListener('focus' , function(){
  alert("focus ! ");
});

self.open();

    You have to set focus event on tabGroup object before open it!

    Hope it help!
