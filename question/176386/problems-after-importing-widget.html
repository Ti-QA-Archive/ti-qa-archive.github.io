    Hi

    It sounds like you have not used a cross-platform dimension unit. DP (device independant pixel) units allow you to set dimensions a d positions relative to the density of the device.

    As all that probably sounds a little confusing, good news here is an answer I gave before that explains the whole process in detail but very simply with full examples.

    • How to achieve same layout for Android and iPhone using Titanium

    The answer was written long before Alloy, but every point is the same, just keep in mind all the object definitions are now simply tss classes.

    So this sort of thing;

    var lbl = Ti.UI.createLabel({
    color: '#000',
    font: {
        fontSize: 14
    },
    height: Ti.UI.SIZE,
    text: 'Insert your name',
    top: 10,
    width: Ti.UI.SIZE
});

    Would become;

    ".lbl": {
    color: '#000',
    font: {
        fontSize: 14
    },
    height: Ti.UI.SIZE,
    text: 'Insert your name',
    top: 10,
    width: Ti.UI.SIZE
};

    Very simple changes.
