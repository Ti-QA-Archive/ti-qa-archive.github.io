iOS plist Customization

Studio build: 3.1.3.201309132423
Titanium SDK: 3.1.3.GA
XCODE: Latest Releases
MAC OSX Mavericks

I want to customize the distribution info.plist file and haven't had much success. How do I customize the CFBundleIconFiles and the CFBundleVersion numbers?

It seems that Titanium is skewing the <version>3.03</version> in my tiapp.xml by adding a .0 at the end of it. 3.03.0 doesn't match the version number I registered in iTunes Connect.

Copying the info.plist file from the build directory to my project root directory and modifying it there doesn't seem to work any longer. Making changes in the build directory's info.plist doesn't work either.

So I modified my tiapp.xml to include:

<ios>
        <plist>
            <dict>
                <key>CFBundleVersion</key>
                <string>3.03</string>
                <key>CFBundleShortVersionString</key>
                <string>3.03</string>
                <key>UISupportedInterfaceOrientations</key>
                <array>
                    <string>UIInterfaceOrientationPortrait</string>
                    <string>UIInterfaceOrientationPortraitUpsideDown</string>
                </array>
                <key>UIBackgroundModes</key>
                <array>
                    <string>location</string>
                </array>
                <key>CFBundleIconFiles</key>
                <array>
                    <string>appicon.png</string>
                    <string>appicon@2x.png</string>
                    <string>appicon-72.png</string>
                    <string>appicon-75.png</string>
                    <string>appicon-120.png</string>
                    <string>appicon-152.png</string>
                    <string>appicon-Small-50.png</string>
                    <string>appicon-Small.png</string>
                    <string>appicon-Small@2x.png</string>
                </array>
            </dict>
        </plist>
    </ios>

That didn't work. Any suggestions?
