getFileStream returns null?

Hi, I am trying to read a file which is located in Resource folder line by line, in a iPhone simulator. Here is my code, very simple:

var userDir = Titanium.Filesystem.getResourcesDirectory();

var sample_1 = Titanium.Filesystem.getFile(userDir, 'sample_1.txt');
if (sample_1.exists()) {
    alert('sample 1 is safe.');
    alert(sample_1.read());
} else {
    alert('sample 1 is not here!');
}

var sampleStream_1 = Titanium.Filesystem.getFileStream(sample_1);
sampleStream_1.open(Titanium.Filesystem.MODE_APPEND);

sample_1.txt indeed exists, and the content is able to show from alert(sample_1.read()). However, an error jump out after with the sampleStream_1.open function, which saying:

[ERROR] Script Error = Result of expression 'sampleStream_1' [null] is not an object. at find.js (line 141).

So seems the getFileStream didn't return anything back. I also tried delete the build folder and re-build it but it doesn't work.

Would you please help me out to see which step is wrong? Thank you very much!
