  • Hello lalit paliwal,

    Please install Java SDK and change your environment variable.
    If you don’t have access to change environment variable then please contact with your System administrator.

    And you need to add 2 variables

    1. New variable name JAVA_HOME and value “JDK folder location like C:\Java\jdk1.6.0_25”
    2. Update you path variable with Java executable directory C:\Java\jdk1.6.0_25\bin

    I hope it will help you.
