  • Unless you have very specific needs in the way you want to use Intents with your application, you should modify the AndroidManifest.xml file yourself. Because it is generated every time, so you will have to keep it some place safe.

    Probably the best approach would be to include you changes in the android section from your tiapp.xml file.
