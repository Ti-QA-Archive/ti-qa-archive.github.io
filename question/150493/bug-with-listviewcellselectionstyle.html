Bug with ListViewCellSelectionStyle

Hi,

this is not a question, but a helpful hint maybe.

It took me so time to figure out why I couldn't build an app with the new ListView function on my device with SDK 3.1.0 Beta.

I always receive this error message:

[ERROR] :  ** BUILD FAILED **
[ERROR] :  The following build commands failed:
[ERROR] :      CompileC build/testApp.build/Debug-iphoneos/testApp-universal.build/Objects-normal/armv7/TiUIiPhoneProxy.o Classes/TiUIiPhoneProxy.m normal armv7 objective-c com.apple.compilers.llvm.clang.1_0.compiler
[ERROR] :  (1 failure)

Looking through the source code, I have probably found the reason:

https://github.com/appcelerator/titanium_mobile/blob/master/iphone/Classes/TiUIiPhoneProxy.m

in line 213:

DEFINE_SUBPROXY_AS(ListViewCellSelectionStyle, TableViewCellSelectionStyle,tableViewCellSelectionStyle);

has to be changed to

DEFINE_SUBPROXY_AS(ListViewCellSelectionStyle, TableViewCellSelectionStyle,listViewCellSelectionStyle);

Now build works finally.

Best
Danny

— asked April 4th 2013 by Danny Pham
