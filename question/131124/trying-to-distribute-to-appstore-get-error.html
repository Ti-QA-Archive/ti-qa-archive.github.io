  • Sounds pretty self-explanatory, your mobileprovision file doesn't exist at the path - /Users/kristian/Library/MobileDevice/Provisioning Profiles/3307CDEB-F44C-4347-9D3D-25157B276BC2.mobileprovision. Make sure you follow Apple's instructions within the provisioning profile for setting up certificates, provisioning profiles, and then importing them into xcode.

    I would also suggest that you build for the app store using xcode itself (your xproj file is in your build/iphone directory), that way you can easily set everything such as icons, provisioning profile, release build, and target (iphone, ipad, or universal.)
