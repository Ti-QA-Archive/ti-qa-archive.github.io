    Hi

    Please review the Titanium Compatibility Matrix Android Section this tells you what can and cannot be done with each version.

    You state you are using Ti SDK 1.7.5 which only supports Android SDKs up to 3.x and NOT 4.x.

    As Muhammad has suggested you must upgrade your SDK in order to have support for Android devices in the 4.x branch.

    At a minimum you require Ti SDK 2.x but if you are upgrading go straight to Ti SDK 3.x and you will be able to support ALL the Android features Titanium provides access to and many SIGNIFICANT language and feature advances also provided by Titanium.

    For reference any number quoted here as; 2.x, 3.x or 4.x means ANY version within that group for example 2.x includes; 2.1.3 and 2.1.4.
