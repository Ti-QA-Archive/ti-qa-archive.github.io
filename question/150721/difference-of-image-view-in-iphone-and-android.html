Difference of image view in iPhone and android

I've got a question to ask….I've got a image view with width = 300dp and height = 200dp. Now I got a portrait image.

On iphone, the image doesn't stretch to fit the image view, it will show as it is. But on android, the portrait image will be stretched to fit the image view.

Is there a way i can prevent this? for landscape picture, both are displaying fine.

I'm using 2.1.4.GA and the image is locally stored in the resource folder.

    Hi Chung,

    You have to set imageView property 

    height : Ti.UI.SIZE and width:Ti.UI.SIZE
