GUI Builder recommendations

How do you build your UI for Titanium, if you are working with an app with many windows?

I thought ForgedUI is the way to go, but is it still alive? The last update on their site was more than a year ago and the announced tablet support never appeared.

Then, there is xib2js, but it seems it has many rough edges.

Any other ideas & pointers will be greatly appreciated.
Thanks!

    Hi

    I have never used a UI tool of any sort with Titanium and never found it a hindrance. Careful planning and good old fashioned wire-framing and prototyping allows you manage the process.

    The biggest speed boost is to split all code into manageable chunks, ideally use CommonJS and Alloy if you are going all out.

    Make everything as re-usable as possible, for example common window templates and table rows. Your app will use these many times over, create one great method and use it often.

    Today I spent five minutes mocking up a full app windows with custom tableViewRows just so I could take a screen-shot - much faster than using Photoshop (and yes I know how to use Photoshop).

    As to ForgedUI as a paid product I would imagine they are interested in keeping it running.
