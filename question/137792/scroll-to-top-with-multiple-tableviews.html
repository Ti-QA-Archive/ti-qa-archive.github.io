Scroll to top with multiple tableViews

Hi,

i have a tabGroup with three tabs and a tableView in every tab.
This causes the "scroll to top" feature to stop working, because iOS can't decide which table to scroll.

I tried to set the "scrollsToTop" property to false on the two tables in the inactive tabs, but that doesn't change anything. It seems like this property is not exposed.

Finally i tried it with a ScrollView ("Scroll View as a Table View") where this property works. When set to false on the two inactive "tables", the active table scrolls to the top when taping the statusbar.
But this is no solution for me, because i need features of tableView which are not included in scrollView.

Is there any way to get this working with tableViews?
And where can i find more info about the scrollsToTop property? Because it works for the scrollView but it isn't mentioned in the api documentation.

Thanks in advance, regards Joe.
