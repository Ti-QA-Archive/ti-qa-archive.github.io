Hover a view like on web devellopment

Hi all,

For my Iphone app, i created view (in green for the test).

i would like that when someone has his finger on the view, the color change to blue.

I used the event click, but the problem, the color change but don't come back like before.

It's like:

[green] => the user click => [blue] but after, the color stay blue, i would like that the color stay blue after the click, but just during the click, the view change, but don't stay blue. Just the time to see that the color changed.

I hope you will understand me and can help me.

Thx

    You need to use the events touchstart and touchend, change the color to blue on start and BACK to green on end. Might be worth duplicating the touchend call with touchcancel as well as there are instances where that will fire instead of touchend.
