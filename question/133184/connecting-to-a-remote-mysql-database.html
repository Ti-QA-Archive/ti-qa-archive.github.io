Connecting to a remote MYSQL database

Hey guys. (Sorry in advance for the long read.)

I'm trying to understand how I would connect to a database hosted on an external (web)server. I'm needing to pull data from different tables that include hundreds of companies and their information. I guess what I'm really looking for is functionality similar to the PHP method of database connection - configure the username and password strings, connect, query the db for what you need, then close the connection. I've seen a few things on SOAP requests and even Kevin Whinnery's suds.js library, but that seems to be severely out-of-date. I'm trying to use CommonJS modules so as to do this the right way; so any suggestions, advice, examples, links, etc. would be extremely appreciated! Thanks! :)
