detect when app is ready?

"[INFO] Application started" is reported when the app is ready for interaction.
How do I listen out for this event in my app?

  • I know this is quite old, but in your app.js you can do it with:

    Titanium.App.addEventListener('open', function (e) {
    Ti.API.info('app.js: Open event on '+Ti.Platform.osname);
});

    more details here: http://developer.appcelerator.com/question/45711/global–app-events-list
