Extending objects not working on iPhone

It looks like extending an object in JavaScript is not working on iPhone.
In the example code below, I have extended the object outputSwitch with the method setTopValue. When setTopValue is called on Android the info message is displayed and the switch gets its "top" value set to 50. On iPhone the method is not being invoked at all. The info message is not displayed in Developer and the "top" value remains unchanged.

Extending an object this way is inherent in JavaScript. Shouldn't this work on both platforms? Any ideas on a workaround?

Although the example is simplistic, we need this functionality to build reusable wrappers that extend base classes.

    var useCaseView = Ti.UI.createView({
        layout: 'vertical',
        right:0,
        left:0,            
        top:0,
        height:200,
        backgroundColor:'#fff'
    });

    var outputSwitch = Ti.UI.createSwitch({                    
        value: true
    });    

    outputSwitch.setTopValue = function(newValue) {
        Ti.API.info('setTop' + newValue);
        outputSwitch.top = newValue;        
    };

    outputSwitch.setTopValue(50);

    useCaseView.add(outputSwitch);

  • Titanium Developer 1.2.2

  • Titanium SDK 1.7

  • iPhone SDK 4.2
