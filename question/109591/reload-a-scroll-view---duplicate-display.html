    This code adds a new scrollview on each function call

    win.add(scroll_view); // Scroll View sur la window

    I don't see any place where you remove the previous scrollview.

    Try like this:

    if ( !!win.scroll_view ) win.remove( win.scroll_view);

// ScrollView container
    win.scroll_view = Titanium.UI.createScrollView({
        contentWidth:'auto',
        contentHeight:'auto',
        height:'100%',
        top:0,
        showVerticalScrollIndicator:true
    });

    ….

    win.add(win.scroll_view); // Scroll View sur la window

    you will have to change all you refs from scroll_view to win.scroll_view
