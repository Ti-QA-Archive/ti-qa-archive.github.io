    Nourah, did you try outputting any other Arabic text to the screen?

    I'm asking because I think the issue might be with the version of the emulator you're running, rather than your code. Remember that versions prior to 2.2 didn't even have an Arabic font in them.

    I tried to run your code, and it showed up alright http://i.imgur.com/q5iWU.jpg (this is as good as it would appear on a 2.2 emulator). The only change I did was to remove the "strip_tags" function because I wouldn't have been able to run the app without strip_tags.js.

    So it's one or the other: either the emulator version, or the strip_tags function. I would also advice that you think about testing on an Android device with proper Arabic support. Sadly the emulators still aren't there in terms of Arabic support.
