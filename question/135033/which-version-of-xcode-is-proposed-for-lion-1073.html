    • Rainer… I'm using SDK 4.2.1 and had the same error for the last 3 days: 'An error occurred uploading to the iTuens Store'.

      And yesterday, I fixed it by doing this:

      cd /System/Library/Frameworks/JavaVM.framework/Versions
sudo ln -s CurrentJDK /System/Library/Frameworks/JavaVM.framework/Versions/1.5
sudo ln -s CurrentJDK /System/Library/Frameworks/JavaVM.framework/Versions/1.5.0

      I found the solution in a post by Diana here: stackoverflow

      I hope it helps you.
