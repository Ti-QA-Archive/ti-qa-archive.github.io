Location of Titanium SQL database on Android

I can't find the location of the database file I create with:

bvxApp.bvxDB = Titanium.Database.open(&#39;bvxData&#39;);

I've tried:

bvxApp.temp_databaseToCopy = Titanium.Filesystem.getFile(Titanium.Filesystem.resourcesDirectory, &#39;bvxData.db&#39;);

(And as suggested by: http://developer.appcelerator.com/question/81271/where-is-the-location-of-my-sqlite-database-file ):

Titanium.Filesystem.getFile(&#39;&#x2F;data&#x2F;data&#x2F;com.AJPcode.TM2&#x2F;databases&#x2F;bvxData.db&#39;);

Nothing seems to work. I know it must be something very simple but I can't seem to find or figure out the answer. Your help will be greatly appreciated. Thanks!

    Here is how I locate sql DB's on either platform:

    var dbName = 'bvxData';
var dbPath;
var dbFile;
if ( Ti.Platform.osname == 'android' ) {
    dbPath = 'file:///data/data/' + Ti.App.getID() + '/databases/';
    dbFile = Ti.Filesystem.getFile( dbPath + dbName ); 
}
else {
    dbPath = Ti.Filesystem.applicationSupportDirectory + '/database/';
    dbFile = Ti.Filesystem.getFile( dbPath + dbName + '.sql' );
}

    Note that not only is the path different, but the extension on the actual database file itself is not the same.
