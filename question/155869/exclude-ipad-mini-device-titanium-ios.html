    • The simulators are specification based rather than models so you need to test on "iPad - Non Retina".

      From studio;

      • Click the Run icon (green circle - white play icon)
      • Choose Run Configurations…
      • Select Titanium iPad Simulator from the left hand side if not already chosen
      • From the Display Type question - choose Non-Retina
      • Then Run

      This is where you toggle between the different display types for the simulator. You can also toggle the Simulator version as well if you have more than one option.
