Thorax sorting collections ascending and descending

I am trying to sort a collection of results coming from an api call. I can't seem to get the sort method to work with Thorax. What am I doing wrong? Here is my code:

Template:

<script id="sorting-template" type="text/x-handlebars-template">
    <span><strong class="pageCountResults"></strong>1 - 10 of <strong id="resultsTotal"></strong> Sort By: <select id="sorting">
        <option id="asc" value="ascending">Model number ascending</option>
        <option id="dsc" value="descending">Model number descending</option>
    </select> </span>
</script>

Javascript:

modelSearchResults.items = new Thorax.Collection ([{
    url: 'models.json',
    model: modelSearchResults.item,
    comparator: 'modelNumber'
}]);

modelSearchResults.sortedView = new Thorax.View({

        events: {
            'change #sorting': 'updateSortBy'
        },

        template: Handlebars.compile($('#sorting-template').html()),

        updateSortBy: function (e) {
            var value = e.currentTarget;
            if (value.value == 'descending') {
                console.log('Descending!');
                modelSearchResults.listView.collection.reset();
                modelSearchResults.listView.collection.fetch({url: 'models.json'});
            } else {
                console.log('Ascending!');
            }
        }

    });

How do I properly sort these results with a select option menu and render them?

  • From what I can see Thorax is an extension of Backbone, so the collection comparator needs to be defined as a function:

    comparator: function(item) {
    return item.get('modelNumber');
}
