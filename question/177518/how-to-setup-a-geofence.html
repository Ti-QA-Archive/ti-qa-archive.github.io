How to setup a geofence

Hi, I have seen that there is a geofence module in the Ti.Cloud section of the docs. http://docs.appcelerator.com/cloud/latest/#!/api/GeoFences

However I don't seem to be able to make anything work. All i need is to create a geofence and check whether the user is inside it or not.

Here's my code:
~~~
var Cloud = require('ti.cloud');

Cloud.GeoFences.create({
geo_fence: {
"loc":{"coordinates":[-122.4167,37.7833], "radius":"10/3959"},
"payload":{"alert":"24-hour sale at our SF flagship store on 12/26!"},
"start_time": "2014-12-25T00:00",
"end_time":"2014-12-26T19:00"
}
}, function (e) {
if (e.success) {
Ti.API.info(JSON.stringify(e.geo_fences));
} else {
alert('Error:\n' +
((e.error && e.message) || JSON.stringify(e)));
}
});
~~

I currently get an error that says = "Requested API endpoint does not exist. Check that you are using the correct GET, POST, PUT or DELETE HTTP method."

How can i achieve what i want?
