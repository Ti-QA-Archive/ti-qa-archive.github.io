    iOS Support

    This release supports iOS 7 and introduces many changes to the Titanium SDK. Some of these changes also affect previous versions of iOS. iOS 5.x will no longer be officially supported starting with Release 3.2.0.

    iOS 7 and Release 3.1.3 require the use o Xcode 5.0 and later. You will still be able to compile iOS 6 apps with Xcode 5. Additionally, Xcode 5.0 requires Mac OS X 10.8.4 or later.

    This Release targets and supports 32-bit iOS apps, which will run correctly on all Apple iOS devices including the 64-bit iPhone 5S. A future SDK will provide support for 64-bit iOS apps.
