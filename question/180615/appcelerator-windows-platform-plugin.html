    Hi Kas,

    Before installing any components, make sure your Windows OS is up-to-date with the latest updates. If you have an existing version of Visual Studio installed, make sure it is up-to-date with the latest updates.
    Also make sure you have TiSDK and CLI with latest updates.

    Then when you configure the windows support, By default, Windows SDK Home textbox should automatically be set to C:\Program Files(x86)\Microsoft SDKs, unless someone might want to change it. So please make sure you have all the prerequisite environments up to date.

    If you have checked every point as directed and still doesn't work, then I would recommend you to uninstall and install the whole set up once again and install updates from Command. Sometimes updates from studio doesn't work.

    All the best. Let know your findings here, so that someone else might help your cause.
