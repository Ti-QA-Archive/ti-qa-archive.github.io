  • If I am not mistaken, Android understands the tel: and sms: 'protocols' when placed in a web view as a link and launches the default dialer and/or sms application as required.

    So the following should call Appcelerator: tel:18779094589 

    <a href=tel:18779094589>tel:18779094589</a>
