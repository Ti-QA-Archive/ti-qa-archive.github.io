  • Exception occurred at undefined:172: Uncaught SyntaxError is your real error. You have a javascript syntax error somewhere. It's entirely possible that the javascript minification that happens on release builds is causing it. There's an option to non-minify the code… but only use that for debugging. Don't want clear-text javascript code in your store app!
