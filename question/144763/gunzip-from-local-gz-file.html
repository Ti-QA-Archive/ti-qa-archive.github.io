gunzip from local .gz file

This requires me to buffer it, and remove the first 4 bytes to get it to a normal gzip file. Therefore, I cannot use the requestHeader to deflate it because it is not a valid gzip file over the wire. So my question really is: Is there a way to gunzip a .gz file within titanium. I have been trying the Compression Module with no luck as it appears to only process .zip files and not .gz files. Device: iOS simulator, android simulator
Host OS: Mac OSX 10.8
Titanium Studio: 2.1.2.201208301612

Test Case:

This is what I have tried.

var Compression = require('ti.compression');

var result = Compression.unzip(Ti.Filesystem.applicationDataDirectory+'versions', Ti.Filesystem.applicationDataDirectory+'versions.gz', true);
