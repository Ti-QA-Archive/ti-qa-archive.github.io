Calendar for Mobile Web in Titanium

Hello, I want to ask about how could i add calendar to my mobile web application for android and web, as it seems to be only supported for android, any solutions will appreciated.

  • Titanium Mobile Web unfortunately does not have access to the device's calendar.

    If you just need to create new calendar events, then you might be able to manually construct an .ics file, base64 encode it into a data URI, then do a Ti.Platform.openURL(). If that works, then it would probably only work in iOS5 or newer.

    If you need an actual calendar, then you'll have to manually construct a calendar using Views and Labels and pull the data from the cloud.
