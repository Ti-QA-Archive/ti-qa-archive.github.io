  • Using Ti.include(&#39;app.js&#39;); is a bad idea anyway.

    If you want your variable to be always available in your project from everywhere, just place it in one of the Titanium properties.

    For example:

    Titanium.API.MY_ABC = 'test';
or
Titanium.API.MY_JSON = { abc: 'sdfsdf' , numberrr: 555, .....};

    If you want just to store serialized data, you can write it to your own files or store in Ti properties.

    For example:

    Ti.App.Properties.getString(...)
Ti.App.Properties.setString(...)

// or boolean values for example:
Ti.App.Properties.getBool('isABC', true);
Ti.App.Properties.setBool('isABC', false);

    One thing you can do is pass the variable from one window to another window by making custom variable

    button.addEventListener('click', function() {
    win.containingTab.open(Ti.UI.createWindow({
        title:"Search by Code",
        customarray:list_length,// made custom variable of array
        url:"page2.js",
        backgroundColor: 'white'
    }));
});

    On page2.js you can retrieve it using :

    window =Ti.UI.currentWindow;
custarray = window.customarray;
var len = custarray.length; // this will retrive length.
