How to check the string contains character @

I tried to do the below code but it says error:
if(txtnumber.value.contains('@')){
alert('Email Exists');
}
else
{
alert('no email');
}

Anyone please help me

  • Learn Regex

    if (txtnumber.value.match(/^[A-Z0-9._%-]+@[A-Z0-9.-]+\.[A-Z]{2,4}$/i)){
    alert('Email Exists');
}else{
    alert('no email');
}
