  • Hi

    I think you are trying to solve a problem in the wrong place and using native path when you shouldn't need to.

    ...
var getFile = function (strFile) {
  var strNoImage = '/images/noimage.jpg';
  var strFileName = strFile;
  var f = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory, strFileName);
  if (f.exists() === true) {
     strFileName = Ti.Filesystem.applicationDataDirectory + strFileName;
  } else {
     strFileName = strNoImage;
  }
  f = null;
  return strFileName;
};
...
var imgView = Ti.UI.createImageView({
    image : getFile('yourimage1.png'),
    width : 220,
    height : 220,
    top : 0
});

    Try to simplify it - something like this, tweak to suit your own needs.

    The idea here is you create a function that tests for the existence of the file you are looking for and if it does not exist - return the "no image" file instead.
