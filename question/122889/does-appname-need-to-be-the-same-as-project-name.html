Does AppName need to be the same as Project Name?

Hi,
For a few reasons I needed to create a new project name for an existing app. So my apps name is 'AppName' but my project name (e.g. the folder in Studio) is 'AppName2'. In general, is this expected to be supported? What I'm seeing is that in my android-manifest.xml if I put AppName2Activity it works on the simulator but not on device, and if I put AppNameActivity it works on the device but not the simulator.

Changing the app back to app name causes me a few extra headaches with some scripts, source repositories, etc - but if it will make everything else easier I'll do it.
