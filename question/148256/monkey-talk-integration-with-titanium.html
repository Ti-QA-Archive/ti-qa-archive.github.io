  • Accepted Answer

    Monkey talk agent has a compile time waved, point cuts witch is waiting for on create event of an activity class. This point cut then capture the entire bundle object and pass it to monkey talk. If this tasks fail monkey talk IDE will not function. I don’t think titanium module can manage this.

    However monkey talk test automation is possible with titanium android applications.
