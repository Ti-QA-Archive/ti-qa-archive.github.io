  • First of all, it is not Titanium who retrieves these images, this is done by the Android OS.

    But to answer your question, I have calculated the screen density of your device: and this is about 150 dpi.

    According to the Android API Guide this is a mdpi device. That means the right image is picked. You shouldn't mistake the screen size with the screen density.

    • You can calculate the size with this formula: px = dp * (dpi &#x2F; 160)

      Example:

      • You want to show an 200dp x 100dp image
      • dpi = Ti.Platform.displayCaps.dpi (Let's say hdpi ~240)
      • width = 200 * (240 / 160) = 300 px
      • height = 100 * (240 / 160) = 150 px
      • provide the image resource as 300x150 in the hdpi folder

      You can also use Ti.Platform.displayCaps.logicalDensityFactor instaed of (dpi / 160)
