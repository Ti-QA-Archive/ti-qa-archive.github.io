What's wrong with this code?

Hi,

I was wonder why I don't see any images on my screen with this code :

App.js :

// create tab group
var tabGroup = Titanium.UI.createTabGroup({
    id : 'tabGroup',
});

// create base UI tab and root window
var win1 = Titanium.UI.createWindow({
    className : 'window',
    backgroundImage : 'Content/Images/bg.jpg',
    barColor : '#9c371d',
    url : 'Views/Calculator_Page1.js',
    orientationModes : [Titanium.UI.PORTRAIT, Titanium.UI.UPSIDE_PORTRAIT]
});
var tab1 = Titanium.UI.createTab({
    id : 'tab1',
    window : win1,
    icon : 'Content/Images/Tabs/calculator.png',
    titleid:'Calculator',

});
tabGroup.addTab(tab1);

tabGroup.open({
    transition:Titanium.UI.iPhone.AnimationStyle.FLIP_FROM_LEFT

    Joeri

    The two main reasons why images do not show up are because either the paths are wrong (note, they are case sensitive) or the image files are of the incorrect format. There may be something in the log to explain the problem. Alternatively, simplify your paths and use the image files that ship with Titanium default projects, just to prove to yourself that it can indeed work. Then add back the complexity.

    Cheers

  • Try adding slashes (/) to the beginning of all your image links. If you are using Titanium SDK 1.7.0+ it should work.

    Also make sure that they are case-sensitive. If your folder is named "content" and you reference it as "Content" in your code, it will not show up.
