Can I use external javascript libraries for mobile development? (SVG manipulation needed)

Is it possible to use external javascript libraries with Titanium Studio?

I develop a mobile app which uses SVG format in web view. I need more than just displaying a certain SVG file as it is. I need to do some manipulation with it such as rotation, transformation scaling etc. I want to use raphael javascript library for that manipulation. I added raphael's .js file in my project and included it at the app.js begin but Titanium fails to start as it finds some errors in raphael.js: 

Can't find variable: document
at raphael.js (line 259)

I don't know what it is but it looks like raphael does't see web view's document or something.

How can I make this to work?

    libraries like that are designed for use in the browser as they need a DOM… so you can use them in a webview, but not in native code.
