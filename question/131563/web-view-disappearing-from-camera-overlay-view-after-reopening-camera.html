Web view disappearing from camera overlay view after reopening camera

I have MyView which contains in it WebView and Label.

I use MyView as camera overlay.

I create only one MyView on application start.

When I open camera first, it shows it good. When I hide camera and open again, only that Label is on it and WebView not.

If I am in simulator I just create new window and add MyView to it and open it.
When I again want to show it I create new window and add MyView to it and open window but the problem is the same as on the device, WebView is not shown, only label.

I figured on simulator that if I call SimulatorWindow.remove(MyView) before closing, everything is OK.

But there is no way to tell something like this camera.remove(MyView);

So the problem is in the WebView itself. Why is it disappearing all the time? This looks like a major bug because you are expected from WebView to behave as any other View but it is not.
