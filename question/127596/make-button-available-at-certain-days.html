make button available at certain days

Hi all

Is it possible to make that happen? I'd like to make a button available only on 2 days of the week at a certain hour, the rest of the time the button should be unavailable.

I want to check on internet the day and hour, not the divice to avoid the problem of miss configured the time on it.

So it be like this:

  1. Check on the net the day and hour
  2. Compare with my conditions of time
    2,1 if true, make the button available
    2,2 if false, keep the button unavailable.

Thanks for help.

    Using time from the web may cause even more issues as you have no idea what timezone the user is in. Its not the same day and same hour for everyone in the world. Im going to go out on a limb and say that 99.9% of people will have the time correctly configured on their device because it is probably their primary method of timekeeping when on the move. On top of that, you will require that the user have a web connection to use the app. What happens when there is not connection to use to make the call to check the time?

    Nonetheless, it is possible to do what you are attempting to do. And you explained it perfectly. Find a place to check the time ( I would strongly suggest using the current device time ) and if it falls inside the parameters that have set, then display the button, if not, then don't
