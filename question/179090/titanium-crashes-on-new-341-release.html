  • Hello Bharal,

    Thanks for your inquiry. There is a reason for this "titanium an internal error occurred during SDK info" error to happen. Maybe your SDK update wasn't finished successfully. You should follow these steps.

    1. Restart the Ti Studio connected to the Internet. It will try to update itself and will update and work.

    2. Click on “Ti Studio/help/check for Titanium updates” and let the Ti Studio update.

    Remember: After updating your Ti Studio, you will need to restart the Ti Studio.

    Also, "Checking titanium updates" might tell you to update your Titanium CLI to 3.4.1. If not, and if your CLI is currently not 3.4.1 then you can manually update your CLI through command line. Run this command

    sudo npm install titanium -g

    Also try updating your studio version to 3.4.1.201410281727. Hope this helps. Please respond if the problem didn't solve. Thanks.
