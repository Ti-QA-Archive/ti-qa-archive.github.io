Get model for listview click event

Is there any way to retrieve the model that used to render the listitem?

For example, my model have 3 fields, A B and C. When I render the listitem, I just used bindId for A and B. I want to be able to get the C value when handling onClick event.
However, if I use e.section.getItemAt(e.itemIndex), it only gives me the data that used to render the view, which is A and B in this case. I want to retrieve the model's data instead of only the ones that used for rendering the view.

Any thought on that? Thanks!

  • The model is not directly available because only its data is used to render the list item. You'll have to look up the associated model as part of the itemclick event. How you do so will depend on how your code is structured. For example, if you set the itemId of each list item to the id/primary-key of your models table, you could do something like:

    var model = Alloy.createModel('yourModel');
model.fetch({id: itemId});

  • You probably created the ListView from a collection. If so, you can do it the nice way which is to set a property on each row to match the item id and then get the item selected from the Collection… dont do a fetch, not efficient if you already have the Collection lying around.

    You could also set the whole model as a property on the ListItem, Not so nice, but I believe it has worked in the past.
