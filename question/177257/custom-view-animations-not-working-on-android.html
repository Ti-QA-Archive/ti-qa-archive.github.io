Custom view animations not working on Android

3.3.0 SDK. Android 4.4.4. Moto X.

Per the Ti.UI.Window docs, I'm trying to create a custom view animations for heavyweight windows in Android. So, I create an xml file like platform/android/res/anim/foo.xml with a defined animation. When I try opening a window with this:

    view.open({
      activityEnterAnimation: Ti.Android.R.anim.foo,
      activityExitAnimation: Ti.Android.R.anim.fade_out
    });

…where foo is my custom animation, fade_out is one of the defaults. Currently the new window just appears on top of the old, no animations on either window. If I use an existing/default animation, like fade_in, it works as expected.

I've tried several different animation files, and copied definitions directly from working examples, but I can't get one custom animation to work. Any idea where I may be going wrong?
