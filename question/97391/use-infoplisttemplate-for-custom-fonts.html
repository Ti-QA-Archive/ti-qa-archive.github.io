Use info.plist.template for custom fonts?

Hi!

I want to use a custom font in my project. Everything works fine, when copying the Info.plist file from the build-directory to the "project root" and adding those lines:

<key>UIAppFonts</key>
<array>
   <string>MyFont.ttf</string>
</array>

But: This is a rather dirty solution, because I want to merge the custom font into the generated info.plist file.

I tried to add a info.plist.template file to my "project root"/Resources/iphone directory with following lines of code:

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
   <key>UIAppFonts</key>
   <array>
      <string>MyFont.ttf</string>
   </array>
</dict>
</plist>

The Emulator shows the message "Detected custom font: MyFont.ttf", but the Emulator doesn't show the right font.

Any suggestions?

Thanks for support!
