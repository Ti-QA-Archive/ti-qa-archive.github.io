Promo code for In-App Purchases

As you probably knows, it not possible to get promo codes for In-App Purchases. So your app reviewers (not testers) cannot review those features unless they purchase them.

Is there any workaround for this accepted by Apple? I know this is technically possible. I could check the device against my server and set the product as purchased if that is on my list. But I guess Apple would not allow this and reject the app.

Any experience about it?
