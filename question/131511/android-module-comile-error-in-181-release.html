    I'm guessing you're developing on Windows. Read the release notes for 1.8.1:

    http://developer.appcelerator.com/apidoc/mobile/1.8.1/changelog.html

    Specifically the section "Requirements for Building Android Modules on Windows", which links to the download for gperf.
