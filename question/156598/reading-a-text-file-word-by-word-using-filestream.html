Reading a Text File Word by Word Using FileStream

Hi all, I was wondering if anyone knew how to separate individual words from a text file using FileStream. I know it's possible to read a definite number of bytes but what if I don't know the length of the word. So basically I'd like to read the text file into a string, word by word. Any help would be much appreciated. Thanks. :-)

    You can do something like:

    var wordList = [];
var readFile = Titanium.Filesystem.getFile(Titanium.Filesystem.resourcesDirectory, "wordlist.txt");  

// If the file exists
if (readFile.exists()){  
    // Split the text file by line
    var text = readFile.read().text.split(" ");

    // Add the words into the wordList array    
    for(var i=0;i<text.length;i++)
    {
        wordList[i] = text[i]
    }
}
