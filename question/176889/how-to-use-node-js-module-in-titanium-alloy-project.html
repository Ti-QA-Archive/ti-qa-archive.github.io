How to use Node JS module in Titanium Alloy Project

Am using 3.3.0.GA sdk, Alloys and Android Device running 4.0. I need to use node module for logentries but unable to use it with titanium app require function.

$ npm install node-logentries

ran he above cmd in project path and it created a module in the path

node_modules/node-logentries/lib/logentries.js

I have tried to access it through require('/lib/node_modules/node-logentries/lib/logentries) and also tried require('/lib/node_modules/node-logentries). But unable to make it run.

    This module won't run on Titanium. You will manage to use only modules that don't have any "node native" or binary dependency.

    This module uses some modules that are specific to nodejs.
