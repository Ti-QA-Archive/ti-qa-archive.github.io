    Hi

    Here you go, you just need to add the correct attributes to a tag, a bit of styling and you are good to go. This supports the exact same editing capabilities and context menu interaction as the mail app you used as an example.

    editor.html

    <!DOCTYPE HTML>
<html lang="en-US">
<head>
    <meta charset="UTF-8">
    <title>Editor</title>
    <meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1">
    <style type="text/css">
        body, html {
            padding: 0;
            margin: 0;
        }
        #editor {
            background: #eee;
            border: 2px solid #666;
            border-radius: 2px;
            color: #000;
            font-family: verdana, sans-serif;
            font-size: 12pt;
            height: 200px;
            padding: 5px;
            margin: 3px;
            width: 300px;
        }
    </style>
</head>
<body>
    <div id="editor" contenteditable="true">
        <p>This is an <b>editable block</b> of content</p>
    </div>
</body>
</html>

    webview code

    var web = Ti.UI.createWebView({
    height: Ti.UI.FILL,
    url: '/editor.html',
    width: Ti.UI.FILL
});
win.add(web);

    Nothing special needed on the webView. You will need to handle setting and getting the content out yourself but I trust you are capable of that.
