  • Hi

    This one is easy, you just have to remember that the JS library has setInterval and setTimeout.

    You create a function that uses scrollToView to move to the next frame and when it gets to the last view it resets to the first view. You then call this function in the setInterval, or an intelligent use of setTimeout.

    I would suggest you either disable the touch events for the scrollableView OR pause the automation when touched and restart afterwards.
