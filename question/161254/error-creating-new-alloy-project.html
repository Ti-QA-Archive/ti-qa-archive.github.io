  • Accepted Answer

    Have you:

    • checked your configuration by running ti setup at the command line, and corrected any errors its "check" function finds?
    • confirmed your node version is supported? compare to the version support matrix in the docs
    • confirmed that you've properly updated Xcode, configured it, run it at least once to accept its license, etc.?
    • confirmed that Studio is properly setup? use the setup wizard on the dashboard page that's shown when you start Studio or the Preferences > Platforms page.

    Can you create a new project using the CLI? If so, it's a Studio configuration problem. Otherwise, make sure the CLI is configured correctly along with the other steps I mention above.

    If this info doesn't help, please post your CLI version, Alloy version, node version.
