Problem with tableview?

I created one row, and adding switch to it. again added this row to the table view.
In this, switch appeared on the row, but row title was disappeared.when I commented the adding of switch to the row, then row title was appeared.
This is my code:

var basicSwitch = Titanium.UI.createSwitch({
style:Titanium.UI.Android.SWITCH_STYLE_CHECKBOX,
value:false
});
var table1 = Titanium.UI.createTableView({
style:Titanium.UI.iPhone.TableViewStyle.GROUPED
});
var section1 = Titanium.UI.createTableViewSection();
var mrow = Titanium.UI.createTableViewRow({title:"Hello 1"});
section1.add(mrow);
mrow.add(basicSwitch);

table1.setData([section1]);
win1.add(table1);

I want to add this row on the table with appearance of switch and row title with maximum length.

  • Lokesh, please use the Markdown Syntax when posting code snippets. Failing to do so makes it really hard to read for everyone. Hence, people are less likely to take the time to answer your question. (there is even a toolbar that gives some help on how to apply proper formatting)

    Now back to your question, the reason your row title disappears when you add the switch is because you are trying to mix default and custom row design. You cannot have both, either you create a full blown custom row with your stitch and (I assume) Label for the title. Or you go the default route, but you cannot add any additional controls to your roe.
