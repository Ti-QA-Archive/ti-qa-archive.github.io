  • For anything other then square and rectangle buttons, I would suggest using an image and adding an event listener to link it.

    For example:

    var circleB = Ti.UI.createImageView({
    width   : 100,
    height  : 100,
    image   : 'circlebutton.png'
});
win.add(circleB);

circleB.addEventListener('click', function(e) {
    var circleB = Titanium.UI.createWindow({  

        url     : 'otherpage.js', 
    });
});
