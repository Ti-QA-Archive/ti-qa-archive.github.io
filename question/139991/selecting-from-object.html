Selecting from object

Hi guys,

I'm having a hard time selecting a property from an object. In essence, I've made a foreach, generating some TableViewRows and inside them is a label with a text. Then I've attached a click listener on each TableViewRow like this

row.addEventListener('click', function(e) {
                        Ti.API.info('test' + JSON.stringify(e.source.children));
});

In e.source.children I have the row clicked. I want to get the text from the label. This is what e.source.children contains:

[
   {
      "keepScreenOn":false,
      "children":[

      ],
      "top":"13dp",
      "font":{
         "fontSize":"15dp"
      },
      "color":"#fff",
      "left":"5dp",
      "backgroundRepeat":false,
      "rect":{
         "height":31,
         "y":20,
         "x":8,
         "width":156
      },
      "touchEnabled":false,
      "text":"+40744649625",
      "size":{
         "height":31,
         "y":0,
         "width":156,
         "x":0
      }
   }
]

I want to get that text value. I've tried in different ways, always giving me the undefined error. As far as I can see, this is an array. I've tried selecting it like this: e.source.children[text], e.source.children['text'], even tried e.source.children.text. Nothing works.

Do you have any ideas? Or maybe a better way to do this?

Thank you,
Mihai
