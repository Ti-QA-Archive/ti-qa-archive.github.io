Local web page not getting cached in webview

I have a webview in my application. Before opening the window containing the webview, I am checking whether the internet connection is available in the device or not. If network connection is available, then I am using:

web.url=html_code;

Where html_code is the url of the form to be opened in the webview. I am also saving the html source code of this webpage in a SQLite table.

Now, if there is no internet connection, then I am fetching the data from the local database and opening the form locally using the html source code. This is how I am doing this:

web.html=html_code;

In this case, the html_code variable contains the HTML source code.

The problem is that, when the webview is opening up the form in online mode, the page is getting cached, which is the expected behavior. But, in the offline mode, when the webview is getting its source from the database, then the web page is not getting cached, which is not expected. How to cache the local webpages as online webpages?
