    Your whole code here depends on the returning API call passing something back to your httpClient call. I am not seeing an example of that returned data so the best I can do is say that this.responseData which will be the entire API response must equal;

    • image blog
    • local to the device image reference
    • Internet URL of the image reference

    If the API returns any other data than those then it will fail.

    When you say it returns back a qr code, I hope you mean a reference to an image? If not what is it actually returning?
