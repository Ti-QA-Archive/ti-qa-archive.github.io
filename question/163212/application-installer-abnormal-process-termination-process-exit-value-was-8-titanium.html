  • You are using Titanium SDK 3.1.1.GA (as set in your tiapp.xml <sdk-version>) which does not support Android SDK Tools 22.6, nor does it support 64-bit Java. You should update to Titanium SDK 3.2.2.GA if you want to use either of those versions.
