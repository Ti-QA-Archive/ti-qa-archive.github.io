Setting default value on TabbedBar

I can't figure out how to set the default active item on a Tabbed Bar… I know this must be a pretty basic thing to do, but I've scoured the docs and seen nothing..

Here is my tabbedbar code…

var tabLabels = [{title: 'All'}, {title:'Mobile'}];

var tb1 = Titanium.UI.createTabbedBar({
    labels: tabLabels,
    backgroundColor:'#666',
    left:165,
    top:140,
    height:25,
    width:150,
    style:Titanium.UI.iPhone.SystemButtonStyle.BAR
});
view.add(tb1);

    Include the index property while creating the object, or set it later:

    var tb1 = Titanium.UI.createTabbedBar({
    labels: tabLabels,
    backgroundColor:'#666',
    left:165,
    top:140,
    height:25,
    width:150,
    style:Titanium.UI.iPhone.SystemButtonStyle.BAR,
    index: 0    // Default to first button selected
});

tb1.index = 1;  // Select second button in tabbed bar

    As typical, this could be gleaned from the kitchen sink example. The index property is also listed in the object docs as "the selected index".
