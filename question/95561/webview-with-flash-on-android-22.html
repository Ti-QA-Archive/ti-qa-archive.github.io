WebView with Flash on Android 2.2?

Hello, I am using Titanium 1.2.2 with SDK 1.5.1 targeting Android 2.2. Compiles great and runs live on my phone (HTC Desire Z).

I am using a WebView to view local html content using the following code:

var window = Titanium.UI.createWindow();
var www = Titanium.UI.createWebView({url:'index.html'});
window.add(www);
www.show();
window.open({fullscreen:true,navBarHidden:true});

The above works great for html but any Flash files embedded within do not get rendered, even though Flash is included with Android 2.2.

Is the WebView using an older version of the WebKit browser than the one that came with my phone?

Or have I done something wrong in the html and Flash does indeed work?

Any hints? Any way to "turn on" plugins?

I am most grateful for any help. Thank you!

  • I don't mean to necro, but you can definitely run Flash in a WebView for Android. By default, Titanium turns off plugins, but you can enable it by setting pluginState like so:

    var view = Ti.UI.createWebView({
    url: "http://www.adobe.com/flash",
    backgroundColor: "transparent",
    pluginState: Ti.UI.Android.WEBVIEW_PLUGINS_ON
});
