How to access tabgroup created in app.js in another js file?

Hi,

So I'm trying to hide my iPhone app's tabgroup (created in app.js) on a seperate js file (user.js).

I've tried win.hideTabBar() to no avail. Is it possible to access the tabgroup in this seperate js file?

Thanks.

  • Accepted Answer

    If you are using commonJS approach than it may be possible. In case of different context approach it will not be possible directly Hiding your tab group from another context where you have not defined your tab group.

    You can do one thing. Register one global event listener on app.js and write code to hide your tab group in that event. And fire that event from your users.js file when you want to hide your tab bar.
