  • Hi

    A few things to check;

    • Have you previously installed more than one copy of xcode? If so then;
      Try this in terminal;
    sudo xcode-select -switch /Applications/Xcode.app/Contents/Developer/

    This "should" convince your mac where the Xcode install is - provided it is in the same location as this sudo path. Re-start studio afterwards and try again.

    • You can check to see if you have multiple versions by following these instructions from Chris Barber

    • This could be down to SDK versions and your operating system, so please provide the information requested above.
