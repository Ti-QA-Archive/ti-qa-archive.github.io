  • Your row objects have height: auto it is not valid since SDK 2.x, you will need to use Ti.UI.SIZE not only on the label but also on all other components.

    Since TableView is a scrollable component, it might ignore Ti.UI.SIZE which requires you to set the height to a numeric value.
