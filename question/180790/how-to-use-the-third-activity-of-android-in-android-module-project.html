How to use the third activity of Android in Android module project?

I have a problem ,when I use Android module project. But I don't know how to use a third activity.
For example,
<activity android:name="com.test.TestActivity"
android:configChanges="orientation|keyboardHidden|navigation"
android:exported="false"
android:screenOrientation="behind" >
</activity>

  • Hello ,

    Thanks for your concern.

    Please see this example :

    <android xmlns:android="http://schemas.android.com/apk/res/android">
        <manifest>

            <!-- Activities and permissions Required by Socialize -->

            <application>

                <activity android:name="com.test.TestActivity" />
                <activity android:configChanges="orientation|keyboardHidden|navigation"/>
                <activity android:exported="false" />
                <activity android:screenOrientation="behind"/>

            </application>

        </manifest>
    </android>

    Thanks
