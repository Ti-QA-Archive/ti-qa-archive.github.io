  • That's not possible, as the Titanium.Contacts doesn't possess a method to achieve that.

    Unfortunately, the work around it might be using removeGroup first, then createGroup as per needed name. But it would require storing data prior removing that group so you avoid loosing your contacts info.

    Currently, Titanium.Contacts is An object which represents a group in the contacts database. (iPhone, iPad only.)

    Further info here: Titanium.Contacts API
