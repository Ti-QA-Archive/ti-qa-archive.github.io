  • without checking your code in detail… in which context does it run? is it all in app.js? i've had trouble to get GPS working on android if the geolocation listener is in the "root" window. if that's the case, put the geocode in a file e.g. geo.js and call it from within app.js, e.g.

    var win = Titanium.UI.createWindow({
    title: 'test',
    url: 'geo.js',
    exitOnClose: true    
});
win.open({fullscreen:false, navBarHidden:false});

    at least that works for me. let us know. –u
