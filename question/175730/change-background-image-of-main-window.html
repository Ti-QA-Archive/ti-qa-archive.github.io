Change background image of main window

First question here, just trying to learn appcelerator for school :) 

<Alloy>
    <Window class="container">
        <Button onClick="startTest" class='fullTest' id="remove3">Full Test</Button>
    </Window>
</Alloy>

in the function for startTest I would like to be able to change the background image the class container. Is this easy enough to do?

    Can you do it? Yes. Is it the easiest way? No. Instead, assign an id to your window, then change it's background via the ID.

    <!-- assume views/index.xml -->
<Alloy>
    <Window class="container" id="index">
        <Button onClick="startTest" class='fullTest' id="remove3">Full Test</Button>
    </Window>
</Alloy>

    // this would be in controllers/index.js

function startTest() {
  $.index.backgroundImage = "/path/to/your/image";
}
$.index.open();

    BTW, by default, the top level container (e.g. <Window>) gets an ID named after the file's name. So, in my example above, you could omit the id="index" from the Window tag and the controller code example I gave would still work. However, I recommend you steer clear of such tricks and implied functionality, especially while you're learning.

    If you want to do it by class, you'll need to look into dynamic styles.

  • Hi

    The easiest way so it makes sense going forward it to give window an ID, can be anything you like;

    <Alloy>
    <Window id="win" class="container">
        <Button onClick="startTest" class='fullTest' id="remove3">Full Test</Button>
    </Window>
</Alloy>

    Inside the controller for this view;

    // add this code 
// or merge if the 'startTest' already exists 
function startTest() {
    $.win.setBackgroundColor('orange');
}
