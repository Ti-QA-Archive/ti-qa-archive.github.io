Local time from time picker in android

Hi all,

I have a time picker, but it gives the time in GMT , i want to get the local time (i.e. Hong Kong time ). How can i do this ?

var time_picker=Titanium.UI.createPicker({
                              top:250,
                              type:Ti.UI.PICKER_TYPE_TIME
                          });

 time_picker.addEventListener('change',function(e){
                              time_value=e.value;
                              alert(time_value)
                          });

  • I'd suggest you to use moment.js - http://momentjs.com/.

    var moment = require('moment');//assume moment.js located under Resources

...
time_picker.addEventListener('change', function(e) {
    var time_value = e.value;
    alert(moment(time_value).format("dddd, MMMM Do YYYY, h:mm:ss a"));
});

  • my guess is you can use a Date object and use the Date.toLocalString() function to get the current date for your system like this:

    time_picker.addEventListener('change',function(e){
    time_value = new Date(e.value);
    alert(time_value.toLocalString());
});
