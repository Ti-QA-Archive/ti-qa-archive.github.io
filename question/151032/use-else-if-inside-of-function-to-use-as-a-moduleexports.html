Use "else if" inside of function to use as a module.exports

I used this a version of this in a ti.include, without an problems. I am trying to figure out how use it in the ti.require module.

[ERROR] : Script Error = &#39;undefined&#39; is not a function (evaluating &#39;Ti.require(&quot;Calculators&#x2F;height&quot;)&#39;)

exports.height = function(h) {

var test = function (h) {
    var inch;
    if (h == "m") {
        inch = 100
    } else if (h == "in") {
        inch = 2.54
    } else {
        inch = 1
    }
    return inch;
};

return test;

};

    There is no require function in the Ti namespace. You can use require straight from the global scope:

    var myModule = require('MyModule');

    Have a look at CommonJS Modules in Titanium
