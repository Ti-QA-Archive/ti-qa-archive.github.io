webview fails to import js files

<script src="//ajax.googleapis.com/ajax/libs/jquery/1.7.1/jquery.min.js" type="text/javascript"></script>
<script src="modernizr.custom.34982.jsf" type="text/javascript"></script>
<script src="tester.jsf" type ="text/javascript"></script>

Using Ti V1.8 all I'm trying to do is just like they do in the kitchensink, include some external java script files; simple enough right? There are other threads about this that date back quite a bit. One suggestion is change the file extension, which I did with .jsf; didn't work. Tried putting them in their own directory; didn't work. Even tried external sourcing, and still didn't work. I know that it's not loading the script files because I wrote a little test file that had one function that executes alert(&#39;Hi&#39;); when called, and that didn't work. I'm at wit's end; I spent that last 3 days on developing the app outside of the SDK and thought "hey all I have to do is dump the files and viola!" The only think I haven't tried is writing a function that opens the files and writes their code directly to the head of the page.
