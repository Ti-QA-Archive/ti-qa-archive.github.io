Titanium Studio, how to get playing duration in time format (hh:mm:ss)

I am creating an audio player (local files), i am currently using Ti.UI.createSound to play audio files. Is there a proper way of getting the currently playing position of the audio track in time format (hh:mm:ss) or shall i write a method of my own?

    Ti.Media.Sound doesn't have a string formatting function like that. You could pass the duration of the Sound object to a Date object, call toString(), and parse that string to get the hh:mm:ss format. That, however, would limit the hours to 23 and truncate the rest. The function below should do what you need and allow for a duration greater than 23 hours:

    function formatTime(decimalTime){
    var time = parseInt(decimalTime);    
    var hours = parseInt(time / 3600);
    var minutes =  parseInt(duration / 60 % 60);
    var seconds = parseInt(time % 60);
    return (hours > 9 ? hours : "0" + hours) + ":" + (minutes > 9 ? minutes : "0" + minutes) + ":" + (seconds > 9 ? seconds : "0" + seconds);
}
