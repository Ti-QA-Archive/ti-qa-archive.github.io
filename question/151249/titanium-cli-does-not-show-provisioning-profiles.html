I am trying to execute the .sh script from http request in php file using browser
The titanium CLI command runs fine but it does not show the iOS provisioning profiles.

anyone has any clue what's happening here?

  • You should actually be calling titanium info -o json so it outputs all the info in a JSON string that you can parse and use. I'm not sure what all your shell script is doing, but you probably could do everything in PHP without a shell script:

    $json = shell_exec('titanium info -o json');
$info = json_decode($json);
var_dump($info);
