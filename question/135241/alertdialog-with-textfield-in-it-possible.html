I don't want to be able to submit if the textfield doesn't have something written in it..

The dialog must have Ok and Cancel buttons as well

    On iOS 5 and later you can use this:

    var dialog = Ti.UI.createAlertDialog({
    title: 'Enter text',
    style: Ti.UI.iPhone.AlertDialogStyle.PLAIN_TEXT_INPUT,
    buttonNames: ['OK', 'cancel']
});
dialog.addEventListener('click', function(e){
    Ti.API.info('e.text: ' + e.text);
});
dialog.show();

    I don't know if you can disable the OK Button, if the textfield is empty, but I don't think so.

    • Yes. The AlertDialog provides an androidView propertiy for customizing the content of your alertDialog. You can do something like this:

      var textfield = Ti.UI.createTextField();

var dialog = Ti.UI.createAlertDialog({
    title: 'Enter text',
    androidView: textfield,
    buttonNames: ['OK', 'cancel']
});
dialog.addEventListener('click', function(e){
    Ti.API.info(textfield.value);
});
dialog.show();
