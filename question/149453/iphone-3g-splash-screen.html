Iphone 3G splash screen

We have build application for all iPhone devices and not for Ipad, so we had our custom splash screen and its working on all the iphone version 4 and 5 accept iphone 3G, in iPhone 3G we had got default original splash screen of Titanium, we have follow all the steps and had given all resolution image for splash screen. can you please give me some idea about the possible solution for this issue.

    Hi Biren Rao,

    permalink
    2 Comments

    • Hi Guys
      we found the solution it was due to Titanium SDK, previously we were using SDK 3.0.0 GA. so this version is creating default resource files which also included default.png for all the platform and devices.
      so when we change it to 3.0.2.GA then the issue of auto creating of default.png is not happening and hence its resolve our problem for 3G splash screen.
      thanks guys for your posts and suggestions.

      regards Biren
