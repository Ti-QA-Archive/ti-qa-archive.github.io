  • Hi, i just faced this same issue (like Studio hangs) right before commenting my comments & i was able to update my SDK after trying these few steps:

    • If you are logged-in from a different user, then login back from root user or just try to update with every user account,
      it is because that any user account can have the right permission to update( mine was root account).
    • try to check the SSL certificate as it can also create issues with updates.

    first line is most likely to solve your problem, if it doesn't works then you could manually download the latest version of SDK & put it inside where all other SDKs are.

    Good Luck!

  • Hello everyone, i just came up with a solution which i used to update the Studio & its components:

    • first of all, try to update things using different User Accounts coz this is the problem related to root user or any other user.
    • i use two accounts (root & personal).. from root account i use Titanium & only from my personal account i was able to update just Ti SDK(if ACS & CLI are also available to be updated, then uncheck them & hit install button)
    • to update CLI & ACS, i use terminal from any user account

    I hope it will help everyone facing this issue as i am thinking to issue a JIRA ticket coz this might be the Ti Studio bug
