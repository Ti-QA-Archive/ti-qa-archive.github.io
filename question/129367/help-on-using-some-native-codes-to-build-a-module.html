    • Hi Paul,

      The links are really useful… I created the Module, with the id named com.ramkumar, in that I created a new class file named com.ramkumar.PushService which extents the Service class on Android SDK.. The module built successfully via "ant" and I imported the module on my Titanium project..

      Now in my tiapp.xml I used as follows

      .
.
.
<android xmlns:android="http://schemas.android.com/apk/res/android">
         <service android:name="com.ramkumar.PushService" android:label="Tokudu Push Notifications Service"></service>
    </android>
    <modules>
        <module version="0.1">com.ramkumar</module>
    </modules>
.
.
.

      The problem now I'm facing is the Service is not at all running (so when I send a notification the app is not able to receive that).. Please guide where I'm doing wrong.
