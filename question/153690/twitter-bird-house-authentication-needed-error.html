  • That's because Twitter API v1 is not deprecated and you need to use 1.1 instead. Try the following:

    change line 967 to 

    api('http://api.twitter.com/1.1/statuses/update.json','POST',params,function(resp){

    and line 1045 to 

    
api("https://api.twitter.com/1.1/statuses/friends_timeline.json","GET",params,function(tweets){
