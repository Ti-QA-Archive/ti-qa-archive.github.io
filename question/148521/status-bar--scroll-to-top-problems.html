Status bar & scroll to top problems

I have 2 windows with 2 table views: 1 window is sidebar (called A), 1 window is main content (called B).
Both these windows are opened at the same time.

When I tap status bar, tableview in window A is scrolled to top. But I need to scroll the tableview in window B. How to do that?

    You need to apply the ScrollsToTop = True on the tableview you want that behaviour with
