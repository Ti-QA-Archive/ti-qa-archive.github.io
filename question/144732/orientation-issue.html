Orientation Issue

Hi guys,

I'm developing app in titanium. I'm facing a strange issue. I have window having portrait mode and this window orientation is fixed using(Ti.UI.portrait and orientation mode of this window is set to portrait). I have a button on this window on clicking the button a new window is opened. The new window have a movie player i set this window orientation mode to TI.UI.LANDSCAPE_LEFT. Problem is when this window opened device set the orientation to landscape that's fine but when i rotate the device it set the orientation to portrait mode.

Next thing if i'm at landscape mode and i press a back button from device(android) and back button from (iPhone top navigation) it will set the previous screen orientation to landscape.

Can any one help me out what i have to do or what i'm doing wrong.

Thanks

  • I assume you're using iOS? (You didn't say) - I have a feeling this down to changes made since iOS6 which seems to force fullscreen videos into Portrait - particularly when using a webView.

    If you're using a native moviePlayer, try forcing the orientation during the 'fullscreen' event : 

    
activeMovie.addEventListener('fullscreen',function(e){
    Ti.API.info('fullscreen entering event : '+e.entering);
    if(e.entering==true){
        win.orientationModes = [ 
            Titanium.UI.PORTRAIT,
            Titanium.UI.LANDSCAPE_LEFT, 
            Titanium.UI.LANDSCAPE_RIGHT
        ];                            
    } else {
        win.orientationModes = [ 
            Titanium.UI.PORTRAIT
        ];                        
    }
});
