    • Hi ,

      On Mac Mini with OS X Yosemite you can do this by Modifying the INI Configuration File.

      Steps

      
1. Navigate to the  /Applications/Titanium Studio  folder in Finder.
2. Right-click (or control-click) on TitaniumStudio.app  and choose Show package contents.
3. In the new Finder window navigate to Contents/MacOS.

      Add the following at the end of the file TitaniumStudio.ini

      -Dtitanium.bypassAuthentication=true
