  • Modules are your path to extending the core Titanium platform with additional functionality. Modules are written in native code (i.e. Objective-C for iOS) and provide the bridge between your JavaScript and the native code.By making modules, 3rd party libraries can easily be accessible to JS developer without actually doing code in objective C.
    Usefulness of iOS Module Development Guide for Titanium is that you will get to know all the required basics to design your module.
