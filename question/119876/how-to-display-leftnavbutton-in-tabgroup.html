how to display 'leftNavButton' in tabGroup?

Hi all,

I'm having problems displaying a back-button in the form of a 'leftNavButton' within all the pages of a tab group.

var tabGroup = Ti.UI.createTabGroup();
var tab1 = Ti.UI.createTab({title:'tab1',url:'page1.js'});
var tab2 = Ti.UI.createTab({title:'tab2',url:'page2.js'});

tabGroup.addTab(tab1);
tabGroup.addTab(tab2);
tabGroup.open();

Above, I have created a 2-tab tabgroup without problems. now I shall create a button:

var backButton = Ti.UI.createButton({title: 'Back'});

The problem now is how do I attach this to the entire tabGroup? I tried:

tabGroup.leftNavButton = backButton;

That didn't work. I also tried: 

Ti.UI.currentWindow.leftNavButton = backButton;

This only displays the button on 'page1.js', and not the other.

Any hints for me?

    Attach it to the window when you create it, but before you .open() it.

    var win1 = Ti.UI.createWindow({
  leftNavButton:backButton
});
