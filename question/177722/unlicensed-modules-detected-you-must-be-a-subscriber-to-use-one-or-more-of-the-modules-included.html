Unlicensed module(s) detected. You must be a subscriber to use one or more of the modules included.

I bought a Flurry module from Appcelerator's Marketplace using the appropriate Appcelerator account, and I am having some verification issues. I am given the error message, "Unlicensed module(s) detected. You must be a subscriber to use one or more of the modules included with this project." Flurry is the only module I am using for this project. Please help! Thank you.

    I had a problem like this before with a paid module, I got in contact with the developer of the module who told me to copy the guid from the manifest file in the module and put this as the guid value in the tiapp.xml of your project. It worked for me, hope it does for you too
