Possible solution for blurry text

It annoys me when my labels or buttons are blurry.

Here are a few things you should check if you are having the problem:

  • Make sure the width/height are even numbers.
    Seems like Titanium tries to scale the text and it causes a blur.

  • You should avoid height='auto' on the label/button.

  • (sometimes) When the parent(s) has height='auto' it causes blur.

  • (sometimes) When positioning is not defined (top, bottom, left, right)

Those are all I've come across, feel free to add other possible causes.

  • Also, using buttons in a tablerow can cause blurred buttons.
