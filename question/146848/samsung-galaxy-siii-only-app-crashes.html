Samsung Galaxy SIII only app crashes

Hi all,
my app is on the store since last year, and I've never received user comments about crashes.
Now I'm receiving a mail from a user, he write that my app crash after the splash screen.
It has a Samsung Galaxy S III, with Android 4.1.2 installed.
To compile the app I've used 2.1.3.GA SDK.
I've tried to run the app on android 4.0 device and on android 4.1 simulator, but I have no crashes. I have not a Galaxy S III to test it directly to this device.
Do you a known bug for this release?
What could I ask to the user to understand the problem?
Thanks for every helps

Manuela

  • S3 devices on which the user has fiddled with the default developer settings, including hardware acceleration, will crash titanium apps. (Aaron Saunders came across this back before the holidays).

    Maybe an app link so we can test (if its a free app) would be helpful?
