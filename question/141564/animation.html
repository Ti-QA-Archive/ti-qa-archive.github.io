Animation

I have a player sprite that is defined like this.

var player = Titanium.UI.createView ({
    backgroundImage:'Move 1.png' //Move 1.png is an image of a guy moving
})

I made a button that when clicked, it should animate the player to change its image like so.

var godude = Titanium.UI.createButton({  //this is the button named godude
    title : 'GO!',
    width : 60,
    height : 60,
    top : 10,
    left : 200
});

godude.addEventListener('click', function(e) {

    player.animate({
        backgroundImage : 'Running.png',  //Running.png is also another different image of a guy running
        duration : 10000
    });

The problem is when I click the button godude nothing happens; the sprite player stays the same image.
Can anyone tell me what's wrong?

  • The problem is that you have to create an animation object to animate like

    var animation = Ti.UI.createAnimation({
    backgroundColor:'blue',
    duration : 2000
    });

    godude.addEventListener('click', function(e) {

    player.animate(animation);

    });

    and the second thing, * titanium does not support backgroundImage animation on animation object.
