problem with switchbox when it added to the table row.

Hi,
I created one table row with adding of switch control. switch was enabled on the table row, but the table row title was disabled. Here is my code:

var basicSwitch = Titanium.UI.createSwitch({
    style:Titanium.UI.Android.SWITCH_STYLE_CHECKBOX,
    value:false,left:270
});
var table1 =  Titanium.UI.createTableView({});
var row1 = Titanium.UI.createTableViewRow({title:"Hello 1"});
var row2 = Titanium.UI.createTableViewRow({title:"Hello 2"});
row1.add(basicSwitch);
table1.setData([row1,row2]);
win1.add(table1);

  • As I explained in my previous answer, you cannot expect the title property to work when using custom rows.

    If you want to display the title, you will need to create a Label, position it where you want it displayed and add this Label to your row (row1 in your sample code).

  • Use labels instead of "title" property to display row title. You can't have other components and row title at the same time.

    var basicSwitch = Titanium.UI.createSwitch({
    style:Titanium.UI.Android.SWITCH_STYLE_CHECKBOX,
    value:false,left:270
});
var table1 =  Titanium.UI.createTableView({});
var row1 = Titanium.UI.createTableViewRow();
var label1 = Ti.UI.createLabel({ text: "Hello 1", left: 0 });
var row2 = Titanium.UI.createTableViewRow();
var label2 = Ti.UI.createLabel({ text: "Hello 2" });
row1.add(basicSwitch);
row1.add(label1);
row2.add(label2);
table1.setData([row1,row2]);
win1.add(table1);
