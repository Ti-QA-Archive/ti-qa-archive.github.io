how to get property/parameter from a wiget, inside a controller

in an "index" controller I'm loading up a widget, that has some parameters/properties associated with it:

<Require type="widget" id="NavigationTop" src="NavigationTop" title="x" backButton="true">

That works great, but now I want to add a back button or close button to that widget, depending on wether the "backButton" parameter was set to true.

I'm unable to access the property called "backButton" from inside the "index.js" controller. Is that possible?

    hi,

    you can access widget's property wiht its id. so if you want to change any property or parameter then you can change that with <widget id>.<control id>.<property>

    means… in your case you can use NavigationTop as widget id then with . parameter you can access object

    like

    NavigationTop.<object id>.<property name>
