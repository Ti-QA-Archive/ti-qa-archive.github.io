addEventListener to view array

I have a for loop, that creates an array of views.

var viewArray = [];
for(var c = 0; c < items.length; c++ {
    viewArray[c] = Titanium.UI.createView({
        height: 200,
        width: 200,
        index: c
    });
}

How would I add an event listener to the views?

I want to achieve something along the lines of:

viewArray.addEventListener('click', function(e){
    alert(e.index);
});
