Error when inserting tableViewRow into TableView

I have a data array, I want to insert this data into TableViewRow. This is my code :

var bindingData= function (_data) {
        var _iData;
        for (_iData = 0; _iData < _data.length; _iData++) {
            var _row = Titanium.UI.createTableViewRow({ title: _data[_iData].Name});
            if (_iData == 0 ) {
                tableData.data = [_row];
            }
            else {
                tableData.appendRow(_row);
            }
        }
    };

But sometime, tableView do not show all rows which were inserted (one row or some rows but not all rows). Why ? I don't know. Please help me.

    HI Thu Trang Nguyen Thi

    You are doing wrong on each iterate of loop you are appending the data in table well the table cannot update automatically(means refresh) well see the below code and make the changes

    
var bindingData= function (_data) {
        var _iData;
    var rowData = [];
        for (_iData = 0; _iData < _data.length; _iData++) {
            var _row = Titanium.UI.createTableViewRow({ title: _data[_iData].Name});
            rowData[_iData]= _row;
        }
    tableData.data = rowData;
    };

    Hope this will help.

    • See if we think logically then the append Data is the right method but the problem is In Titanium. Its not refreshing the Table automatically when the new Data have been Added. So We have to go this way. Even after all the Row added you try to append a new one or try to update an old one its reflection will not come on the tablerow.

      At that time you have to remove whole table again and set the Data to the row again with the TableData.data = rowData;
