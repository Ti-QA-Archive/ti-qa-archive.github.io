how to get updated user input from textarea

I'm currently stuck on retrieving the updated input made into a textarea on another window.

I can see that the data is being read & stored to the variables via the EventListener, but when i open the new window & call a function, it doesn't have the new updated information; just the default.

It seems like when the new window opens, and it calls the function back on the other window, it doesn't fire the listener & just uses the old value.

Is there any way to get the listener value to?

    Use the Ti.App.Properties APIs and set it as a String, or just set it as a straight variable in the Event Listener and call upon that variable in your new window.
