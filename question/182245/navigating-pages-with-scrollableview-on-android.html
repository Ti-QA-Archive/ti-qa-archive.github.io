Navigating pages with ScrollableView on android

I am using navigation between pages as usually other developers here would suggest using
historyStack.push(controller) - push and goto next page
historyStack.pop() - on click of back button show previous page

Now it works just fine on everything else.
But on ScrollableView, since ScrollableView keeps many views inside it, the above way of push/pop kind of forgets to push views inside the ScrollableView,

Therefore, it pushes only empty ScrollableView but doesn't pushes views inside it.
However on ios it works fine, all views are shown on going back on ios, but this problem is only on android.

I'm using this https://github.com/nsrosenqvist/AlloySingleWindowNavigation

Am i doing something wrong here?
