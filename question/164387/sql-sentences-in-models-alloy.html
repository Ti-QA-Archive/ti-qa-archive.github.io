SQL Sentences in Models Alloy

Hello,

I'm new working with Alloy and SQLite. I want to fill a TableView from a table but I don't need all the table, I just need the rows that have the column Active = 1.

For example:

Database: Rainbow
Table: Colors
          Columns: Id, Name, Active

I need to fill a TableView with the colors that have Active = 1

So I have:
-Views-

index.xml

<Alloy>
    <Window>
        <TableView id="table" dataCollection="color">
            <Require src="row"/>
        </TableView>
    </Window>
</Alloy>

row.xml
<Alloy>
    <TableViewRow>
        <Label id="name" text="{name}"/>
    </TableViewRow>
</Alloy>

-Controllers-

index.js
Alloy.Collections.color.fetch();

row.js
if ($model) {
    $.row.model = $model.toJSON();
}

-Models-

color.js
exports.definition = {
    config: {
        "adapter": {
            "type": "sql",
            "db_file": "/Rainbow.sqlite",
            "collection_name": "color"
        }
    }
};

Alloy.js

Alloy.Collections. color = Alloy.createCollection('color');

In this case I'd get all the table Colors in the TableView. What I need to change to only bring the colors that have the column Active = 1 ? I need something like Select * from colors where Active = 1 but I don't know how to do this in Alloy

Thanks

  • You should use something like to filter your collection

    var active = Alloy.Collections.color.where({Active: 1})

    active will contain an array of models where the "Active" attribute/field is equal to 1.

    hth.

    • also you could create a 'scoped' collection extending it as:

      extendCollection: function(Collection) {
    _.extend(Collection.prototype, {

        isActive: function(model) {
            return (model.get("Active") == 1);
        } 

        }); 
    return Collection;
}

      then use as:

      Alloy.Collections.color.isActive();
