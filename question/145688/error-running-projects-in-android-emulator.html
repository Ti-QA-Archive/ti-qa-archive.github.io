    • Namitha,

      your PATH settings for Java looks good so far.

      But i noticed that xcopy don't work.

      So i suspect your PATH settings for Windows is somewhat misconfigured.

      For PATH settings there is a PATH in System Variables and sometimes one more PATH in User Variables.

      It looks like the PATH in System Variables don't contain the primary Windows paths anymore.

      A basic PATH setting in System Variables should look like:

      C:\Program Files\Common Files\Microsoft Shared\Windows Live;C:\Program Files (x86)\Common Files\Microsoft Shared\Windows Live;%SystemRoot%\system32;%SystemRoot%;%SystemRoot%\System32\Wbem;%SYSTEMROOT%\System32\WindowsPowerShell\v1.0\;C:\Program Files (x86)\Windows Live\Shared

      This is for a x64 System, if yours is x86, then you won't need c:\Program Files (x86), it should just be c:\Program Files

      So if the above is missing on your system, that may the reason for xcopy not to function.

      Please check your path for that.

      before changing the path, please copy the original settings to an editor and save it for later, if a restore is needed!
      If you change that PATH in System Variables, please restart your system after.

      Greetings,

      Matthias
