    Hi

    Everything in Titanium will accept events, for example click. touchEnabled lets you set if you want to turn off the ability for this specific view, button etc to accept touch events.

    So if you have a label and there is no need to accept events you can set touchEnabled to false for that label. Then ALL events that would have gone to the label are no longer passed to the label (even if you added a listener) and be passed to the thing behind the label instead (bubbling).

    You do not need this for tableViewRows as you should be using the selectionStyle to achieve the same results. But you would use it for any child views in a custom TableViewRow.
