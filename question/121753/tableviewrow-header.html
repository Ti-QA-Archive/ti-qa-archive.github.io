TableViewRow header

Hi, i want to know how can i modify the height of the header of a table view.

    sectionLabel = Titanium.Locale.getString('team_driver');
    row = Titanium.UI.createTableViewRow({
        header: sectionLabel, // === HERE === 
        leftImage: '/img/employees/' + teamDriver[0].img,
        title: teamDriver[0].title,
        height:70,
        color:'black'
    });

And what does Ti.Locale.getString is reference to.

  • You would have to use the headerView property of the TableView, I am afraid. The "header" property of a TableViewRow is simply a string that Appcelerator uses to set onto a pre-defined set header that is default on iOS. If you wish to change the way that is shown, you either have to set the "headerView" to a view with a label in the TableView itself, or create multiple TableViewSections and set it there.

    Ti.Locale.getString (Macro: L) is a function that returns a localized string set in your localization files. For more help on that, I suggest you take a look at the Wiki page.
