Pdf reader don't work

Hello I have a little problem with a simple app, I just want read a pdf by the easiest way ^^

My code is:


var window = Titanium.UI.createWindow({
    backgroundColor:"#fff"

});
var file= Ti.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory+'w4.pdf');
// === for application directory
var webview = Titanium.UI.createWebView({
    data: file.read()    
});

alert("Bonjour");
window.add(webview);
window.open({modal:true});

I've tested with a simple data or url without the Filesystem.applicationDataDirectory, but that doesn't work too. When I launch this on my iPhone I have my alert"Bonjour" but no pdf ><

Thank you for your help ;)

    No need to reference the file system;

    var webview = Titanium.UI.createWebView({
    url:'/mypdf.pdf'
});

    where mypdf.pdf is placed in the /Resources folder
