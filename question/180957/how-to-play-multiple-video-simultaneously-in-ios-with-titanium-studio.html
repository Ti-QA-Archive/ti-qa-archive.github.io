  • I have a feeling that it's not possible to have two embedded video players running at the same time on iOS using the Titanium.Media.VideoPlayer using the CONTROL_EMBEDDED style.

    IDEA: It might be possible to do it using a webView with two HTML5 video tags and the webkit-playsinline attribute (which should stop a webview embedded video from using fullscreen). I have not tested this idea though.
