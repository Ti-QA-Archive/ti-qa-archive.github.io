Help with Android notifications

Hello!

This is probably supereasy for you experts, but i couldn't find how to do it in the documentation so here it goes. I want the the notification to play a sound, can be some default sound for Android that indicates that something was pushed to the device. I want the device LED to flash in purple. Also, i want the apps icon to be shown instead of the warning icon in the notification bar.

My code looks like this

var intent = Ti.Android.createIntent({
            action: Ti.Android.ACTION_MAIN,
            flags: Ti.Android.FLAG_ACTIVITY_NEW_TASK | Ti.Android.FLAG_ACTIVITY_RESET_TASK_IF_NEEDED,
            className: 'com.mycompany.myapp'+Ti.Android.currentActivity,
            packageName: 'com.mycompany.myapp'
        });
        intent.addCategory(Ti.Android.CATEGORY_LAUNCHER);

        // This is fairly static: Not much need to be altered here
        var pending = Ti.Android.createPendingIntent({
            activity: Ti.Android.currentActivity,
            intent: intent,
            type: Ti.Android.PENDING_INTENT_FOR_ACTIVITY,
        });

        var notification = Ti.Android.createNotification({
            contentIntent: pending,
            contentTitle: 'Nytt meddelande',
            contentText: e.data.message,
            tickerText: "Nytt meddelande",
            ledARGB: 3,
            ledOffMS: 500,
            ledOnMS: 500
        });

        Ti.Android.NotificationManager.notify(1, notification);
