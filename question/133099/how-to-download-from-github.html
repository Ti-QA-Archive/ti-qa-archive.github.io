How to download from GitHub?

Dumb question, but I'm new to Mac and Appcelerator, and I'm having trouble downloading source from GitHub. Could someone please outline the basic steps to do this, and then be able to open the project in Titanium?

Example: https://github.com/appcelerator/KitchenSink

  • Accepted Answer

    On your menu bar click File->Import open the Git folder select Git Repository As New Project click Next paste the git url in the uri field and hit Finish. The download can take a lil while. Hope this helps
