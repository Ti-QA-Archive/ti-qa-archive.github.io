  • Hi,

    You can identify it's updated or not using app version.

    Suppose your updated application version is 2.0. then you can compare it with existing property.

    for example you created property Ti.App.Properties.setObject(&quot;objData&quot;,SOME DATA OBJECT) in app store
    so you can compare it if current app version is 2.0 and "objData" property is exists it means it's updated application.

    
var currentVersion = Titanium.App.version;

if(currentVersion == 2.0 && !Ti.App.Properties.hasProperty("objData")){
    //EXECUTE CODE WHATEVER YOU WANT FOR FRESH VERSION
}
