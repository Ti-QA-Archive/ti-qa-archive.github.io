How to mimic iOS notification center?

Hi,

I'm trying to make some easy access panel for my iOS app. It's basically the same behavior as the iOS noitification center but from the bottom->up. I'd like to create a window and show just about 30px of it on the bottom of the screen. (I already know how to do that part.)

What I'm looking for is a way to catch the slide up event of that window and make the window follow the user's finger as the window slides up. (If you don't understand go try iOS notification center and you'll see what I mean).

Any suggestions of how I should do this? I'm very new with the animation part of Titanium so any references to proper docs or code snippet are welcome.

Thanks and have a nice day!

    Tommy,

    What you could do is just add an event listener to that window or view that is acting like the notification center panel. You would most likely want to listen for the 'touchmove' event. This event will give you the x and y coordinates of the touch. You would want to listen for the windows 'touchstart' event to see if the touch originated over the notification window or view. If it did, you would then fire a function that would pin the x coordinate to the top value of the view or window that is acting as the notification center tray.

    From there you can give it some if statement to tell if the view is close to the bottom or top of the screen to make it either snap to the top, or snap to the bottom.
