Autocapitilization of characters in text field focus

Hi, I am developing an application using titanium in which I developed a login screen consists of login field. I set autocapitilization property to 'Titanium.UI.TEXT_AUTOCAPITALIZATION_ALL' . So when I launch the application by default I am able to enter upper case letters. But if I intensionally type small letters also it is accepting. I mean whatever the case may be the characters should be autocapitilize. If I type small case letters also they should be autocapitilized. Kindly provide a possible solution. Below is the code for the textfield created…

var username = Ti.UI.createTextField({

    id : 'InputUsername',

    width : '90%',

    height : '35%',

    top : '12%',

    bottom : '3%',

    autocorrect : false,

    autocapitalization : Titanium.UI.TEXT_AUTOCAPITALIZATION_ALL,

    borderRadius : 2,

    borderWidth : 0,

    backgroundColor :'white',

    maxLength : 3,

    color : 'black',

    textAlign : Ti.UI.TEXT_ALIGNMENT_CENTER,

    verticalAlign : Ti.UI.TEXT_VERTICAL_ALIGNMENT_CENTER,

    font : {

        fontSize : 26

    },

    returnKeyType : Ti.UI.RETURNKEY_GO

});

Thanks in Advance

  • Hi Prasad,

    It seems that autocapitalization property has been broken. But you can do it by another way.

     username.addEventListener('change', function (e) {
    username.value=username.value.toUpperCase();
        });

    Hope this helps…
