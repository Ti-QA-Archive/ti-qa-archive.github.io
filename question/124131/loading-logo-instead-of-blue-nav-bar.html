Loading Logo instead of Blue Nav Bar

I'm new to Javascript and Titanium. I've been playing around now with the software for about a week. My question is how would I go about loading a logo on the main page instead of the blue nav bar that is standard.

I know how to hide the navbar, but have yet to figure out how to load my own logo in it's place. Another solution would be for me to load the logo outside of the navbar and just hide the navbar, but not really sure how to do that either and position it where i need it.

Make sense?

Any help appreciated as I trudge along trying to learn javascript.

  • Try hiding the nav bar, and then setting a background image to whatever your logo is on a screen size background. You can also use the titleImage property of the nav bar. There are great examples of this in the Kitchen Sink demo, you should have a look if you have not yet done so.

  • If I understand you correctly you can use the barImage property on the window to replace the navbar colour with an image of your choice

    win.barImage = 'mylogo.png'
