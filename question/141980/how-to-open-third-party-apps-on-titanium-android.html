  • Accepted Answer

    Ti.Platform.openURL() will work if that 3rd party app is set up for it. If not, you might need to use an Intent:

    var intent = Ti.Android.createIntent({
    action: Ti.Android.ACTION_VIEW,
    data: 'whatever data you are supposed to send',
});
Ti.Android.currentActivity.startActivity(intent);
