  • An RSS feed is just xml. So as you're looking through the feed, pull out the category name and if it matches the category that you want to display then add it to the table, if it does not, then skip it. How you grab the category will depend on how your xml is formatted. Regardless, its and xml document, so all you'll have to do is grab it in the same fashion that you are grabbing the title or the image for each row.

    • Try not to post answers to your own questions unless its actually an answer. The comment system should suffice for posting followup to either an answer or your own question.

      That said, it looks like you've already got everything you need to filter. You're already grabbing the category. Just make the category the first thing you grab in the loop and filter accordingly. ie:

      for(var i = 0; i < items.length; i++) {
    var rssCategory = item.getElementsByTagName("category").item(0).text;
    if (rssCategory == "{THE CAT YOU WANT TO SHOW}") {
        // build out your row view
    }
}

      On a side note, you're redeclaring all those xml vars on each iteration of the loop. Declare them all once before the loop and you don't have to worry about redeclaring with that var every time.
