How can I remove other ACS users which I've created?

I'm trying to remove a user from the ACS cloud. But the operation ACS.remove removes the current user from which I'm logged in. How can I remove other users which I have created?

  • You can't do what you want using the published api because you can only delete the currently logged in user.
    , as you have discovered. You can manually delete a user by logging into I the web ACS tools and deleting from there. Login to the appcelerator site, then click'My Apps' from the user menu, select your app, select users, find the user and click the delete button beside their name.
