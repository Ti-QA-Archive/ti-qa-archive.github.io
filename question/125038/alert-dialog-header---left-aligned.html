Alert Dialog header - left aligned

Hi,

When I added an alertDialog in Android, I found that the title in the alertdialog box is left aligned. Is it possible to align it to the center instead?

It is aligned to the center by default in case of iPhone.

I am using Ti 1.7.2, Android 2.2.
Thanks.

  • It's not pretty, but this works - you should probably get fancier and play with screen orientation and widths to calc the true 'offset'.

    var osName = Ti.Platform.osname;
var isAndroid = false;

if (osName === 'android') {

    isAndroid = true;

}

var androidAlertTitle = '          ';  //offset spaces

androidAlertTitle += 'My Title';


label2.addEventListener('click', function(e) {

    var alertDialog = Titanium.UI.createAlertDialog({     
        title: isAndroid ? androidAlertTitle : 'My Title',     
        message: 'Hello, dude',     
        buttonNames: ['OK','What?', 'OMG'] 
    }); 

    alertDialog.show();


});
