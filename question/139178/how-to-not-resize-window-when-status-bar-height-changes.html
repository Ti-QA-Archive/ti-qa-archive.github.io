How to not resize window when status bar height changes?

I'm building an iOS5 app and have trouble with some layout issues when the status bar height changes (e.g. during a phone call - press CMD+T in the simulator for the effect). Whenever the status bar height changes, the window height is adjusted, so it shrinks from 460 to 440 pixels. I use percentages (100%) for the window height, but the problem is the same when I set the height in pixels (460).

I would prefer the window not to shrink and stay exactly where it is before. Is that possible with Titanium?
I have not found a "status bar" event to listen to or some property where to specify the desired behavior.

  • I realized I also need to set the height of the surrounding TabGroup in pixels, than everything is fine. To get the overlapping effect, one needs to set "bottom: 0" instead of "top: 0".
