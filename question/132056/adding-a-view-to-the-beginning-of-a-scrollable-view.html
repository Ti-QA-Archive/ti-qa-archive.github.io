adding a view to the beginning of a scrollable view

Hi,

As you know it, we need to optimize the number of views loaded in a mobile app. The problem is i need to load about between 80 and 120 views in a scrollView. So to do that i thought i can only load 3 views (the current, and the left and right ones). Obviously, i need to add a view to the end when i go to the right and reciprocally add a view at the beginning when i go to the left like this :

myScrolableView.views[0] = myNewView;

or like

myScrolableView.add(myNewView, 0);

Is there a way to do that ?

Regards

    I suppose you can use similar approach as it is described in this gist: https://gist.github.com/1485751.

    The only difference is "add" and "addView" method.

    Remember that "add" always adds at the end, so to insert a view at specific position, you need to remove all views and re-add them with view inserted manually to views array at specific position.
