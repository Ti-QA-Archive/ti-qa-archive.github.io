    • Thank you. These are some of my findings.
      If we can send SMS from the background or pragmatically without showing the dialogue box, then we can send this Pin code by the user to himself and match the code. We can send SMS on the background on Android, but we cannot do this on iOS. So this not a complete solution.
      Other than using a third-party SMS gateway service, we can build our own SMS gateway server in a dedicated web server. I think paying server cost is much cheaper than paying for a SMS gateway.
