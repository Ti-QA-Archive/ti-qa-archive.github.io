Encapsulation and Instantiation in Titanium - best method?

Hi all.

I was wondering what would be considered the best method to encapsulate and instantiate objects in Titanium using JavaScript. I guess I'm thinking more about performance and efficiency than anything else. The 3 methods I know of are:

  1. JSON
var myObj = {
    x: 50,
    y: 75
};
  1. Reflection
var myObj = {};
myObj.x = 50;
myObj.y = 75;
  1. 'new' keyword
function clsObj() {
    this.x = 50;
    this.y = 75;
}
var myObj = new clsObj();

Assuming that myObj only needs to be declared and used once, are any of the above methods less computationally / memory expensive than the others? I thought that method 3 would use more memory since JavaScript functions are Objects so we essentially have the same Object being declared twice. But performance tests on Android show this actually wasn't the case.

Cheers,
Jay.
