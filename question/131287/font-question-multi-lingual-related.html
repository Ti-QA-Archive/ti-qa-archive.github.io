Font Question (Multi-lingual related).

I am working on a project which needs to be deployed world wide. Their current product was developed for Windows mobile phones, and they mentioned tremendous headaches in supporting all of the languages. So, I would like to ask here, what font(s) do I need to support for which countries?

Currently, I just use Helvetica Neue for everything. Will that render Chinese, Arabic, and others?

Thanks,
Ozz

  • By using system fonts, you should not have any issues when dealing with other languages. For iphone, that means helvetica, verdana, courier new, etc. For Android, thats obviously anything from the Droid family. The idea is that if you stick with the default fonts for each device, then you will likely cover the languages that you need to cover.
