Loading Remote Json data in the background

I have an app that has a number of tables contained in views. The table data is downloaded from Json files on my server and this is done as soon as the app loads by including the views as functions in my app.js, when the user opens a view the table is preloaded and there is no wait time, this works really well.

For a new application I want to use a much increased set of data so I don't want to load it all on startup, maybe just the first 20 items and associated data to create the table and subsequent views.

What I'd like to do is while the user is browsing items 1-20 items bring down the next 20 items in the background so when the user clicks a button at the bottom of the table "load next 20 items" its already loaded and just needs adding to the table, then as they browse items 20-40 bring down 40-60 in the background etc etc.

Anyone any thoughts on how to do this
