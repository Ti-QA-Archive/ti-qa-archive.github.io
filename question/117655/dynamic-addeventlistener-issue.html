Dynamic addEventListener issue

Hi - I'm just a few days into this stuff, and I have some JSON called via REST and have added it to a sqlite table. I am then taking data from the table and putting it in a TableViewRow. I want to take some of the data and use it in an EventListener, so that when the event is triggered, I can make another REST call to a specific record. I can't figure out how to get the data into the function triggered by the EventListener. I can see it in [info] in the loop that is creating the TableViewRow, but not inside the addEventListener function(e). If I try to look at the [info] inside the function it says it is null . Any help would be appreciated!

EDIT:I found my answer - to that question - I needed to add nid:rows.field(0) to end up with this:


listing[rownum] = Ti.UI.createTableViewRow({
   hasDetail:true,
   title:rows.field(1),
   nid:rows.field(0)
});

So that seemed to work, and then I got bit by the dreaded "Result of expression 'Titanium.UI.currentTab' [undefined] is not an object." issue, which seems to be a pretty common complaint from people trying to launch new windows from a TableViewRow…..
