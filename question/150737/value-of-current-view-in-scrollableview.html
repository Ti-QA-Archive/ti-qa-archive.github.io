  • TiUIScrollableView has a method getCurrentPage(). This returns the index (integer) of the currently displayed view.

    To access the currently displayed view, you can access the view array of the scrollableview.

    myTiUIScrollablevView.views[myTiUIScrollableView.getCurrentPage()].children[x] //Do something with the particular child of the view displayed in your scrollableview
