Displaying of secret PDFs

Hi,
I want to show secret PDFs. OK. I protect the access to the PDF on the web server with login/password and the http-client on titanium sends this credentails. It seems to be save.
But: the evil user uses a net sniffer (like Burb suite) and can see the password, retrieve the PDF and distributes the PDF.

How can I avoid?

  • password for PDF access (how can I open this inside the app)
  • I can use a password protected zip – how can I open this save zip insid app,
  • other ideas?

Rainer
