  • The problem is that it places them in the incorrect directory.
    I fixed the issue by by simply moving them via terminal:

    Go into the directory where they're placed, move them up one, and into /Templates/File Templates/

    cd &#x2F;Developer&#x2F;Platforms&#x2F;iPhoneOS.platform&#x2F;Developer&#x2F;Library&#x2F;Xcode&#x2F;File\ Templates&#x2F;
    mv Appcelerator&#x2F; ..&#x2F;Templates&#x2F;File\ Templates&#x2F;
