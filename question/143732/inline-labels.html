inline labels?

Hey,

i have an array, and a view. I would like to be able to scroll through the array and create a new label for each item in the array.
Then, to place these labels side-by-side in my given view, with the idea that labels too wide would "wrap" and take the "next line".

So i'd like the view to look like this, say

view:

____label1____ ____label2__
__ ____label3____ ____label
4____

How do i achieve this?

Here is my looping code!

for (var j = 0; j < questionArray.length; j++){
    var questionLabel ;
    if (questionArray[j].isQuestion){
        questionLabel= Ti.UI.createLabel({
            text: unescape(questionArray[j].text),
            font : {fontSize:'16sp'},
            color: 'black',
        });
    } else {
        questionLabel= Ti.UI.createLabel({
            text: "",
            font : {fontSize:'16sp'},
            color: 'black',
            backgroundColor: 'yellow',
            width: "40sp",
            borderRadius: 15,
        });
   }
   questionView.add(questionLabel);        
}
