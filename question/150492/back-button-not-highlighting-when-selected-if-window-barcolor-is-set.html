Back button not highlighting when selected if window barColor is set?

I have a Tab group, which has a tab, which has a window, which has TableView. If I set the window's barColor to something (i had it at '#000') then the back button would no longer highlight when pressed. Is there a way to get it to highlight as usual?

My layout code is similar to this -

<Alloy>
    <TabGroup>
        <Tab>
            <Window barColor='#000'>
                <TableView>
                    <TableViewRow>
                    </TableViewRow>
                </TableView>
            </Window>
        </Tab>
    </TabGroup
</Alloy>

Titanium Studio, build: 3.0.2.201302191606 Titanium SDK: 3.0.2.GA Platform: iOS Device: Simulator+iPhone5
