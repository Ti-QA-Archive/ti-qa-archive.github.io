    • I understand now.

      The yellow icon to the left of a class name inside TSS is NOT a true error. It is unfortunately noted as "error" but more correctly it should say "just so you know".

      It is simply saying that class name does not exist inside the scope of the xml files are far as it can see.

      This is very normal. It is more for those people who never create widgets, use global styles, assign classes inside controllers.

      I have many of these in my apps - perfectly normal. Trust me - I have been doing this a while.
