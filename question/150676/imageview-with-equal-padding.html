  • Try this…

    var imageView = Ti.UI.createImageView({image:'image'});

var padding = 10;

var imageViewWrapper = Ti.UI.createView({
    width             : imageView.toImage().width + padding,
    height             : imageView.toImage().height + padding,
    backgroundColor    : '#FFF'
}); 

imageViewWrapper.add(imageView);

    It takes your imageView width and height and adds a padding to it.
