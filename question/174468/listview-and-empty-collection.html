ListView and empty collection

Hi

I'm creating an app using Alloy that returns a collection from a Rest call and displays the resultant models in a listview which works very well. What I'm struggling with is what to do if there are zero models returned.

At the moment the listview just displays with zero rows. What I would like to do is display "There are no offers available at this time.".

I thought that perhaps the default item template would automatically be displayed if the collection was empty but that isn't the case.

Is there a simple way to do this?
