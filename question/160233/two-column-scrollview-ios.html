Two Column ScrollView IOS

I am trying to make a two column scroll view in titanium. I can get it to work on Android but not on iPhone. My XML looks like this 

<ScrollView id="scrollView" >
    <View id="leftColumn" class="column" left="0dp" />
    <View id="rightColumn" class="column" right="0dp" />
</ScrollView>

".column": {
    width:"50%",
    backgroundColor: 'transparent',
    height:Ti.UI.SIZE,
    width: Ti.UI.SIZE,
    layout:'vertical'
}

The reason it isn't working in ios is because the inner views are grabbing the scroll event and not the outer view. Is there a way that I can tell the inner views to ignore the event or pass it off to its parent?

    Hi,

    you can use layout to horizontal for scrollview to set view as a column so you do not need to set left or right property.

    and you can set touchEnabled to false to inner views so that you can get scroll event of parent view.
