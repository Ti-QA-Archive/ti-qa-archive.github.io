Change Color of Title Bar

Hi there,

what value has to be in the tiapp.xml (or somewhere else) to get the title bar of my main window black?

  • Accepted Answer

    Simple.

    Set it (barColor) during creation:

    var win = Titanium.UI.createWindow({
    title: 'My Title',
    url: 'my.js',
    barColor: 'orange'
    });

    or later using

    win.barColor = 'red';
