  • if you are a new coder, I am going to suggest you avoid shortcut coding until you understand the language.

    if(val ==0) return;

    and

    if(val ==0)
  return;

    and

    if(val ==0){
  return;
}

    all get interpreted the same way, however, as a new coder, the odds that you will mistype/misread by dropping your brackets is high. So, use your brackets.

  • Defining className in a tableViewRow can increate the tableView performance. also it will help you check the source row of the the event.
    in this case return will return undefined. The point is stopping the following code from being executed. For example if you wrote

    function myFunction(val){
    if(val ==0)
        return;
    alert('hello');

}

    In this case the function will check the value. If it is 0 it will not execute the next line which means it will not display the alert
