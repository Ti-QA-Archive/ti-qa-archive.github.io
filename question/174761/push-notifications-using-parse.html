  • For Android , you have to use a module :
    https://marketplace.appcelerator.com/apps/7893?restoreSearch=true#!overview

    Steps :
    1)Create an app on parse.com
    In settings tab , you can find application keys i.e Application Id and Client Key

    2) In app.js : require a module and add Keys–

    var ParseService = require('com.jepadilla.parse');
    ParseService.ParsePushService("Application Id", "Client Key");

    3)Connect Android Device, with these keys your device will be registered with the App on the parse server.Under Data Browser Tab , you can see DeviceID for your device.

    4)Go to Push Notification Tab - Send a push - select OS - send a message or a JSON Object.

    5)You should be able to see a push notification .

    Let me know if you have any issue.
