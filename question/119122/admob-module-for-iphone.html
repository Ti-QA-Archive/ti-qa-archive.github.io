I am getting an error "Couldn't find module: ti.admob"

<module version="1.0">ti.admob</module>

    I just tested it out and all is fine. Which means you have dome something wrong.

    1: Extract the module zip so the module exists at /Library/Application Support/Titanium/modules/iphone/ti.admob/1.0 .. etc

    2: Edit your tiapp.xml to include the module inside the <modules> element:

    <modules>
    <module version="1.0">ti.admob</module>
</modules>

    3: In your application, 'require' the module when you need it. To test this, try simply adding it at the top of your app.js file

    Titanium.Admob = require('ti.admob');

    4: Delete the CONTENTS (not the folder itself) inside [your_project]/build/iphone

    5: Rebuild your application

    As the app build, you should see a message in the debug console to show that the module is loaded ok. 

    [INFO] Detected third-party module: ti.admob/1.0

    and once the app is running, you should see :

    [INFO] AdMob module loaded

    if that doesn't work, paste on the code you are using here. Also, what Titanium SDK are you using, etc.
