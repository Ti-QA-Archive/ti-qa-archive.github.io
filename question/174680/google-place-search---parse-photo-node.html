    Hi

    The photos node is an array, so your line would need to be;

    var photos = placesObj.results[cc].photos[0].photo_reference //works

    This is the photo node;

    {
    "photos": [
        {
            "height": 256,
            "html_attributions": [
                "FromaGoogleUser"
            ],
            "photo_reference": "CoQBewAAAH40GzSr7008QM8tMhWVC7Un0FrZaY_SpdLWLgNiFXVii3Mgnfr5zzqoKg8kS_Hlo4FTpjC5oHLeWcFfs3pV-bmFc2OJg0RhDYXDg2QFGoFiUWNd48DmQ_DbOtJmrx5FpDvF3w4ddhvg8yj_Ww6Z37svQrFc7vlqPybaNgDGn3shEhAWtYhAZfwKkWp3x3kPwi9WGhSmkYMgAIvbZ-5ZwMQtz4CDdo8GHA",
            "width": 256
        }
    ]
}

    So in this truncated example it is;

    ...photo[index].photo_reference

    Correctly handled this would be accessed as follows;

    for (var i =0, j = placesObj.results[cc].photos.length; i < j; i = i + 1) {
    Ti.API.info('ref: ' + placesObj.results[cc].photos[i].photo_reference);
}
