Unable to pass a variable from a Java Script method call on a webpage hosted in a remote server

Hi,
I am unable to pass a variable from a Java Script method call on a webpage hosted in a remote server to the Ti webview.

My objective is as follows:
1.My app does some prerequisites with different native windoes.

  1. Then fires an URL (of a remote server) in a webview.
  2. After doing something in the remote web application, i want to give the control back to a native window on click of a button in the remote server.

Is it doable?
Following is the webview code:

    Ah, I see what you're saying. What I do for my app is on that remote page, the button clicked will redirect to a special URL. I then listen for that URL in Titanium and close the window:

    webview.addEventListener('load',function(loadEvent){
    if (loadEvent.url.indexOf('closethiswindow.html') > 0) {
        win.close();
    }
}

  • Remote webpages don't get access to Titanium, and vice-versa. You have a couple options:

    1. Download the remote HTML, and load the file into a webview.
    2. Have your remote webpage load a URL instead of trying to fire an event. Use the load event listener on webview to detect the URL, and parse from there.
