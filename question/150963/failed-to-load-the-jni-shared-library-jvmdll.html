  • i had the same problem on win7 64bit with java 7 64bit. The solution was to install additional 32bit version of Java (without removing the 64bit). The two things which I had to change were:

    • the system PATH variable (i added "C:\Program Files (x86)\Java\jre7\bin" - place where i have installed my 32bit java).
    • little mod inTitaniumStudio.ini I added:

    "-vm

    C:\Program Files (x86)\Java\jre7\bin\javaw.exe"

    Now Titanium works fine.
