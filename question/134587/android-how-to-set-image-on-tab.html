Android How to set image on tab?

Hello Friends,

I am working a simple demo app in iPhone and Android and in that I have set image ontab but its not set perfectly on default tab in Android and also please tell me tab icon image size for HVGA , QVGA , WVGA800 and Samsung Galaxy Tab so please give me suggestions or any idea.

var tabGroup = Titanium.UI.createTabGroup();

var win1 = Titanium.UI.createWindow
({  
    url:'TabClasses/Tab1/Tab1.js',
    navBarHidden:false,      
});

var tab1 = Titanium.UI.createTab
({  
     icon:'/images/nearme.png',
     window:win1
});

I am running in HVGA emulator and tab icon nearme.png image size is 64x64 but its not set in tab so please give me idea.

Thanks in advance.

    Hello Nikunj,

    I think image path problem…Remove images from "images" folder and put in "Resource" folder and set below code for tab.

    var tab1 = Titanium.UI.createTab
({  
     icon:'nearme.png',
     title : 'Tab 1',
     window:win1
});

    clean the build and try this….
