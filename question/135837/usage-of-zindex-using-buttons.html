  • First off all, try to use the Ti.UI.ScrollableView instead of Ti.UI.View and you won't need to set a left increment on every button. Then try to debug the zIndex by adding the eventListener only to one of the buttons. If this works try to set the zIndex with the new implemented method btn.setZIndex(1);
