  • Do not wrap the SplitWindow in another Window. SplitWindow is a top level container and does not need to be in another Window.

    <Alloy>
    <Window>
        <NavigationGroup>        
            <SplitWindow id="editionWindow" title="Stack Name" barColor="darkgray">
                <Require src="master"></Require>
                <Require src="detail"></Require>
            </SplitWindow>
        </NavigationGroup>
    </Window>    
</Alloy>

    Having said that, I noticed you used leftNavButton and rightNavButton. Unfortunately those aren't implemented in the markup right now for SplitWindow. You'd have to add those programmatically in the controller code for the time being. If you log a Titanium Community Jira Project ticket regarding leftNavButton, rightNavButton, and titleControl for SplitWindow, I'll get to it ASAP. Be sure to label the ticket with "alloy".
