tableViewRow multiline footer

Hi All

Just wondering if any knows how to create a multiline footer in a tableviewrow.

I have tried.

myRow.footer = 'I would like this footer to span over';
myRow.footer = 'two lines.';

and

myRow.footer = 'I would like this footer to span over\ two lines';

No luck unfortunately. Any ideas?

Jamesy

  • You can put an own view into the table's fooer like this:

    view = Ti.UI.createView({});

label = Ti.UI.createLabel({
    text: 'Two lines of text'
});

view.add(label);

tableView.footerView = view;
