Fetch data from sql, Alloy

Well i just wanted to fetch data from persons table without using Query

This is what im doing right now but i have no idea where i am wrong

function showPerson(event) {
    var selectedPerson = event.source;
    var recoverDatabase = Alloy.createCollection("persons");;
    alert("Starting fetching data");
    recoverDatabase.fetch({'name': selectedPerson.title });
    // recoverDatabase.fetch({query:"SELECT * FROM persons WHERE name = '" + selectedPerson.title + "'" });
    alert("Successfully retrive data");
    alert(recoverDatabase.get('name'));

i am not getting any name. Its not like that, that name does not exist in db something is wrong
Thanks

    What is wrong is you are using recoverDatabase like an instance of the row, instead of what it is - a collection.

    So you need to iterate through the rows as follows:

    recoverDatabase.each(function(row) {
    alert(row.get('name'));
});

    Without having your database/data, I cannot confirm the above code works, but it should.
