Local webview with Android and SDK 3.2.1

Hi all, in my app I load a local html (with jquery) in a webview. If I use the SDK 3.2.1 seem that the page doesn't load correctly; there are some errors (seem css or html code). With SDK 3.2.0 the webview works perfectly. Is there any suggestion?

  • Ok, I find my solution. In the local HTML pages I ned to call .js and .css files in this way:

    <script src="../html/jquery-1.8.2.min.js"></script>

    and don't

    <script src="jquery-1.8.2.min.js"></script>
