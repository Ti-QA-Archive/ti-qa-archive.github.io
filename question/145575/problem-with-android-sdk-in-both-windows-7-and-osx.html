  • When starting Titanium Studio, you are presented a Dashboard.

    If you go into the Configure section, you should be presented 2 big buttons. One on the them is to configure Android development. If you click it, it should download all the necessary components for you (or update if needed). This is very useful when you need to "repair" a partial install.

    Hope this helps.
