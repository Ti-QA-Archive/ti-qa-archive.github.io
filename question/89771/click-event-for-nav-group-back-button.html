  • How about using event listeners to pick up the blur of the child screen, which is triggered just after the user clicks the button? And then another for the focus on the parent screen as that is displayed to allow you to reflect the updates from the child screen data?

    on child screen:

    currWindow.addEventListener('blur', function(){
    Ti.API.info('Leaving child screen - going back to parent');
    runFunctionToStoreData();    // or other clean up actions?
});

    on parent screen:

    currWindow.addEventListener('focus', function(){
    Ti.API.info('We have returned from child screen');
    loadTableData();        
    Titanium.API.info('Refreshing table from db');
    tableView.setData(data);
});
