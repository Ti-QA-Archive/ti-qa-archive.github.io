canOpenURL and iOS 8.

I have used canOpenURL to check if a URL containing a phone number can be dialled. This worked fine under iOS 7, but run under iOS 8 (although still built under the iOS 7 SDK as I don't want to upgrade in the middle of a major project) I am finding that canOpenURL returns 'true' for a phone number on an iPod Touch which obviously can't dial a number it not being a phone!

Has anybody else seen this? Test example:


            if (Ti.Platform.canOpenURL('tel:0845XXXXXXX')) {
                alert("Thinks it can dial!");
                Ti.Platform.openURL('tel: 0845XXXXXXX');
            } else {
                alert("Sorry, your device cannot dial\nthis number!");
            }

Titanium SDK version 3.2.3.GA, OS X 10.9.5
