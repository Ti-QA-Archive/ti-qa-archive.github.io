Publishing for Android never finish

Hello,

Firstly, please excuse my poor english.

My application runs correctly on Android emulator. I'm trying to distribute it by following this tuto : http://docs.appcelerator.com/titanium/3.0/#!/guide/Distributing_Android_apps

I have created and verified the certificate. Now I try to build the app. I completed the form about keystore informations, and clicked on "publish". About logs, all is OK for compiling files.
The last log I have is :

[DEBUG] "C:/path\to\my\jarsigner.exe" -sigalg MD5withRSA -digestalg SHA1 -storepass ************ -keystore D:\path\to\the\certificate -signedjar D:\path\to\distribution\folder\for\app.apk D:\path\to\app-unsigned.apk myAlias

But it seems the build stay locked at this moment, the progress bar indicate "Creating Android distribution", i haven't apk file in my distribution location. I started the build yesterday (more than 24 hours, now) and there is no progress.

Anyone has an idea about this problem ?
