  • Hi Ajay,

    According to the error report, you're not giving us the good sql request.
    I think you should read some documentation because your request doesn't mean anything but here a tips for the insertion.

    You have 2 possibilities for an insertion:

    db.execute('INSERT INTO tableName (id, name, age) VALUES (0, Sims, 34));

or

db.execute('INSERT INTO tableName VALUES (0, Sims, 34); //If you know the right order.

    If you are using variable : 

    db.execute('INSERT INTO tableName (id, name, age) VALUES (?, ?, ?), varID, varName, varAge);
