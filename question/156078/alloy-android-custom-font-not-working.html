Alloy Android: Custom font not working

I'm trying to include the following font inside my android project:

http://www.dafont.com/dj-gross.font

I've put it inside app/assets and have the following test app:

Tss:

"Label":{
    font:{
        fontFamily:"DJ Gross"
    }
}

I've tried "DJGROSS" too (filename without ending)

XML:

<Alloy>
    <Window class="container">
        <Label id="lbl">test test test</Label>
    </Window>
</Alloy>

container:

$.index.open();

But it's not working, the font isn't used inside the label. Is the font the problem? Or do I need to change something for alloy?

SDK: 3.1.2

Android: 4.2

    Hi

    You appear to be using the family name of the font, but in Android you are meant to declare the name part of the filename.

    Your font choice; DJ Gross has the family name of DJ Gross but the filename is DJGROSS.ttf assuming you have kept that filename and maintained the uppercase formatting then the fontFamily value should be DJGROSS instead.

    If you have renamed the original font you download then you can work out the file section of the filename by ignoring the .ext, so if you call the font dj-gross.ttf the android value would be dj-gross.

    More information can be font in the;

    • CustomFonts Guide

    If you are creating an Android only app then tweak your TSS as follows;

    "Label": {
    font: {
        fontFamily: "DJGROSS"
    }
}

    If you are creating a cross-platform app then tweak as follows;

    "Label[platform=android]": {
    font: {
        fontFamily: "DJGROSS"
    }
}
"Label[platform=ios]": {
    font: {
        fontFamily: "DJ Gross"
    }
}

    This creates a platform specific style that chooses the correct version depending on the target platform.

    • I have got it working. I did experience your lack of font usage when following the normal advice.

      However when I adapted my non-Alloy advice (previously given to many on the Q&A) it worked straight off.

      The simple tweak is - move your fonts to the following directory;

      {appname}/app/assets/fonts/

      You can also have platform specific fonts if you wish;

      {appname}/app/assets/iphone/fonts/
{appname}/app/assets/android/fonts/

      This is the exact TSS I used with YOUR font choice;

      "Label[platform=android]": {
    color: "#c60000",
    width: Ti.UI.SIZE,
    height: Ti.UI.SIZE,
    font: {
        fontSize: 36,
        fontFamily: "DJGROSS"
    }
}
"Label[platform=ios]": {
    color: "#000099",
    width: Ti.UI.SIZE,
    height: Ti.UI.SIZE,
    font: {
        fontSize: 36,
        fontFamily: "DJ Gross"
    }
}

      Fully Tested on Android and iPhone.
