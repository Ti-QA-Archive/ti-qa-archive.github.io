    • Hi Trevor

      Would you try the following, and then provide the information again?

      These packages are obsolete. You should remove them:

      SDK Platform Android 2.3.1, API 9
Google APIs by Google Inc, Android API 9

      The following package seems to be installed 3 times, which could indicate that it is corrupted. Thus, you should remove all occurrences of it:

      GALAXY Tab Addon by Samsung Electronics Co Ltd Android API 8, revision 1

      Ensure you follow the advice at Preparing for Android development regarding adb.

      Set the $ANDROID_SDK environment variable to the path of your android sdk. On my system, this is &#x2F;opt&#x2F;android-sdk-linux_86. This directory contains the android tools directory.
