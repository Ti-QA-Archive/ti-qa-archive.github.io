    You need to add it to your Build Path (import from sdk). If you are using Titanium Studio, then
    right click > Build Path > Configure Build Path

    Select Libraries and click Add External JARs

    It should be under /Users/manu/Library/Application Support/Titanium/mobilesdk/osx/3.2.3.GA/android/modules/titanium-android.jar
