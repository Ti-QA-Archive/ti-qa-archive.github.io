Can I use custom lib function in alloy tss style file?

I have created a custom lib function and put in lib folder, but how can I use it in the tss file?

example, my function is:

// lib/ui.js
exports.customStyle = {
        width : 200,
        height: 300
}

within index.tss file, how should i use it? something like this:

var ui= require('ui');
"#content":{
       width: ui.customStyle.width
}

cause the above code is doesn't work.

    You can't just execute arbitrary JS in TSS files. Keeping code out of styles is important. You can however use properties on the Alloy namespace. Try this:

    app/alloy.js

    Alloy.Globals.customWidth = require('ui').customStyle.width;

    Then in any of your TSS files you can use the Alloy.Globals.customWidth property
