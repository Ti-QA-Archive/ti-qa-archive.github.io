    Dear Das,

    There may be many reason which are causing to this error.

    Don't know if below will work for you.

    1.Filter your log to just Error and look for FATAL EXCEPTION.

    OR

    2.Change a line in TiApp.xml file then clean your project, build and run.

    OR

    3.Run a simple code just try to open single tab window and check if it works, if working then there is some
    code in your which is causing this error.

    OR

    4.Kill adb server from terminal and start it again then run your application in emulator.

    OR

    5.Use breakpoint to debug your application in Titanium Studio.

    OR

    6.If nothing is working then create a new project and run it in Android Emulator.

    if any above solution is not working then share your source code i will fix the issue and will send you running code.
