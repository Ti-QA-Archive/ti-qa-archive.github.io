      I'm sorry you're having a hard time. That said, your first issue is addressed in the first section of the Alloy Controllers guide and the passwordMask field is found right in the Titanium API guide for TextFields. The API docs are kept in sync with the current version of Titanium so they should be a pretty reliable source of this type of info.

      As far as Alloy, which is at version 0.3.6 not 3.0, the guides get into a lot more than basic topics. Pretty much all features of alloy from basic to advanced are covered. Additional, there's tons of test apps to look at in the repo. TiStudio is also currently working on a lot of code assist functionality for XML, TSS, and the standard JS files, which should help.
