Logging out of the app and returning to the login page?

How do you redirect back to the login page, removing any session variables which have been set as well as reseting any open windows which have been openned prior to the Logout button being clicked?

    simple, Just show and hide login page and put an animation on it. when user log outs then remove the current window and show login window. when user login again then hide the login window and create you default window again.

    • You can close previous window after some delay of transition complete,

      for example : I am hiding tabGroup when log out button clicked. so transition let in the log in screen and
      hide the tabGroup.

      login.open({
    transition : Titanium.UI.iPhone.AnimationStyle.FLIP_FROM_LEFT,
    duration : 1000
})
setTimeout(function(e) {
    tabGroup.hide(); // so here you can close your current window.
}, 1100);

  • We use a tabgroup for most of our current apps. What I have been doing is firing the login window method as the main entry point into my app. Once we have a successful login, it opens the tabgroup thus loading the top window and running their logic. If I need to force the user to login again I fire the login method and start the process over again.
