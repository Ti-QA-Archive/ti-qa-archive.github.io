Device won't locate local image

Hello!

So, I have an application where a user has to upload a post through the ACS. I am giving them the choice of uploading one of their own pictures or if the want to use a default one, which I have located locally. When they chose the default picture, the simulator uploads the image just fine, and everything works. But when im trying on my iPhone, it says that the image cant be located, how come?

I am using this code to grab the image:

var f = Ti.Filesystem.getFile(Ti.Filesystem.resourcesDirectory, 'images', 'bgpost.png');
var defaultblob = f.read();
defaultimage.image = defaultblob;
