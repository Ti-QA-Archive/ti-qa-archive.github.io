    Terry

    Note that the entries property contains an array, so your Ti.API.info command should be written as the following to output data in the first index:

    Ti.API.info(news.entries[0].headline);

    I am not sure where you are getting the story data - this data is not returned from the json service you are using. Hence var story = news.entries seems to be redundant, and hence var titleLabel = Ti.UI.createLabel({text:story}) is unlikely to work.

    Hope this helps

  • Instead of using eval() to parse your incoming JSON, Titanium includes a built-in JSON parser that works much better. (It also means you don't run the risk of running arbitrary code.

    Instead of var news = eval(&#39;(&#39;+this.responseText+&#39;)&#39;);, try:

    var news = JSON.parse(this.responseText);.
