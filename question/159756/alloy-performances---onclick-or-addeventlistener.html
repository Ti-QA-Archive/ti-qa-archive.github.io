Alloy performances - onClick or addEventListener?

Hello,

I was wondering if it's better to use onClick in views or addEventListener in controllers to get better performances?

I know that in ActionScript it's good practice to use removeEventListener when an event won't be used anymore, does something like this apply in Titanium?

Thanks.

  • onClick is just going to generate an addEventListener for you. The performance is identical. And yes, use removeEventListener to get rid of events you are no longer using. This works exactly the same whether you use onClick or addEventListener.
