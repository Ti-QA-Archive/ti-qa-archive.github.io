    Regarding a code sample, I create a simple Alloy Tabbed Application (with Alloy) and put one ImageView and one button in it, both sharing the same "image" property (original image size is 135x135 pixels).

    Through TiShadow, the ImageView displays the image with the right size but it's tiny in the button.

    views/index.xml

    <Alloy>
        <TabGroup exitOnClose="true">
                 <Tab title="Actus">
                        <Window id="winNews" title="News">
                                <Button top="0" left="0" title="sample text" />
                        </Window>
                </Tab>

                <Tab title="Services">
                        <Window id="winServices" title="Services">
                                <!--Widget id="serviceGrid" src="com.appcelerator.buttongrid"/-->
                        </Window>
                </Tab>

                <Tab title="Agenda">
                        <Window id="winAgenda" title="Agenda">
                        </Window>
                </Tab>
        </TabGroup>
</Alloy>

    controllers/index.js

    
var testImg = Ti.UI.createImageView({
    image: '/images/buttons/btnSettings.png',
    top: 10
});

$.winAgenda.add(testImg);

var testBtn = Ti.UI.createButton({
    image: '/images/buttons/btnSettings.png',
    title: Ti.Platform.displayCaps.density + ' ' + Ti.Platform.displayCaps.dpi + ' ' +Ti.Platform.displayCaps.xdpi + ' ' +Ti.Platform.displayCaps.logicalDensityFactor,
    center: {
        x: "50%",
        y: "50%"
    },
    width: 300,
    height: 300
});

$.winAgenda.add(testBtn);

$.index.open();

    I'm also using "dip" as UI default unit in tiapp.xml :

    <property name="ti.ui.defaultunit" type="string">dip</property>
