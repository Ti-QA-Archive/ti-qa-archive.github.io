Orientation problems

How do I only allow specific orientations on various windows when in a navigation group?

Seems like a simple question, but… my code:

var mainWindow = Titanium.UI.createWindow(
{
  backgroundColor: '#fff',
  orientationModes:
  [
    Ti.UI.Portrait,
    Ti.UI.UPSIDE_PORTRAIT,
    Ti.UI.LANDSCAPE_LEFT,
    Ti.UI.LANDSCAPE_RIGHT
  ]
});

var win = Titanium.UI.createWindow(
{
  title: 'test',
  backgroundColor: '#fff'
});

var navGroup = Titanium.UI.createNavigationGroup( { window: win } );

mainWindow.add(navGroup);
mainWindow.open();

If I set the orientation modes in the 'mainWindow' as I do above, it works as expected, i.e. I support all four orientations. But if I want to override the available settings in another window (e.g. the 'win' window above), it doesn't work. I set win.orientationModes, and it seems to ignore me.

I've tried removing the orientation modes from the 'mainWindow', and setting them only on 'win', but that does nothing.

How do I get this to work?

  • When you set the second window with restricted orientation set it like this:
    eg.

    var win = Titanium.UI.createWindow(
{
  title: 'test',
  backgroundColor: '#fff',
  orientationModes:
  [
    Ti.UI.PORTRAIT
  ]

});

    Just choose the modes you want to support for the window at the time of the window creation.

  • You could try to change the orientation mode each time a navgroup window was opened/closed:

    I have a timeout at the end to close the navGroup window and restore the orientation to the main window. I do not think it is possible to restrict the internal windows.

    var mainWindow = Titanium.UI.createWindow(
{
  backgroundColor: '#fff',
  title: 'Root Window'
});

var win = Titanium.UI.createWindow(
{
  title: 'test',
  backgroundColor: '#fff'

});
 mainWindow.orientationModes =   [
    Ti.UI.PORTRAIT
  ];
var navGroup = Titanium.UI.iPhone.createNavigationGroup( { window: win } );

mainWindow.add(navGroup);
mainWindow.open();

setTimeout(function(){
    navGroup.close(win);
     mainWindow.orientationModes =   [
    Ti.UI.PORTRAIT,
    Ti.UI.UPSIDE_PORTRAIT,
    Ti.UI.LANDSCAPE_LEFT,
    Ti.UI.LANDSCAPE_RIGHT
  ];
}, 4000);
