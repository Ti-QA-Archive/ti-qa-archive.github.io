Browser detection (In-App Browser - WebView)

Hi all,

I have a mobile site and I am building a stub app (putting mobile site in WebView).

Is there any way I can detect if my mobile site is being viewed in standalone mobile browser (Safari, Chrome) or in-app browser (which is basically Web View)?

Many Thanks for replies.

Vishal

  • You could grab the user-agents from the webview and compare them to mobile browsers, but the differences might be very small and hard to check for. The easiest way would be to send either a special header when you're using a webview, or to just send the webview to a different location and have it set some parameters so you know that a webview is being used.
