  • Try resizing your controls with an orientation change event handler.

    Ti.Gesture.addEventHandler('orientationchange', function(e) {
    // code to resize your controls
});

    Also, have you attempted to use the left and right constraints to size your components, rather than using width? Same goes for top and bottom instead of height. This can sometimes yield better results when supporting multiple orientations.
