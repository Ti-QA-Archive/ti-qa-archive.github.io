How can I add a contact in Android

Per the forum answers there is no way to add a contact in Android via the Titanium.Contacts interface.

Can it be done with Intents?

Here is my testing code for creating a contact using the intent Ti.Android.ACTION_INSERT_OR_EDIT.

It compiles and runs but throws an error "No Applications can perform this action". I have added android.permission.WRITE_CONTACTS to a custom AndroidManifest.xml. Am I missing something?

        var activity = Ti.Android.currentActivity;
        var intent = Ti.Android.createIntent({
            action: Ti.Android.ACTION_INSERT_OR_EDIT,
            type:"vnd.android.cursor.item/person"
        });

        //http://developer.android.com/reference/android/provider/Contacts.Intents.Insert.html
        intent.addCategory(Ti.Android.CATEGORY_LAUNCHER);
        intent.putExtra( "name",'test');
        intent.putExtra("phone",'1231234');
        intent.putExtra("phone_type" ,'1'); // type mobile
        intent.putExtra("email",'test@test.com');
        intent.putExtra("email_type",'1');//type home
     activity.startActivity(Ti.Android.createIntentChooser(intent,'Add contact'));

Titanium 1.6 :: API 2.3 :: Win 7
