Open-source module is not supported by Ti SDK 3.2.3 . How to make it work?

I have downloaded the open source module from https://github.com/websiteburo/androzip

It is successfully packed and output to my Mobile App project.

However in the TiApp editor (tiapp.xml), the module is added but dimmed with a exception icon saying "the module is not supported by Ti SDK 3.2.3"

The module is open source and I already build it using Ti SDK 3.2.3 (I specify this in module's project > "build.properties".

Here is the module's build.properties file

titanium.platform=/Library/Application Support/Titanium/mobilesdk/osx/3.2.3.GA/android
android.platform=/Users/xuser/android-sdk-macosx/platforms/android-10
android.ndk=/Users/xuser/android-ndk-r9
google.apis=/Users/xuser/android-sdk-macosx/add-ons/addon-google_apis-google-10

How can I make it work for SDK 3.2.3 ?
