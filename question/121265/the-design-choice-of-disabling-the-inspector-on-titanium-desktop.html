  • Jeremy,

    This was enabled somewhere during the 1.2.0.RCx release candidates. I just tested packaging and app with 1.2.0.RC4 which is the latest on the packaging servers (will try to get 1.2.0.RC6d up in the next day or two), and it works fine after installing.

    You could manually edit the Info.plist and add WebKitDeveloperExtras into it and set it for a boolean of true, or have your client execute this on the command line before running your app:

    $ defaults write app.id.from.your.app WebKitDeveloperExtras -bool true

    Let me know if that works for you. This is actually a problem with either the recent webkit or on Lion, I can't remember which. If your running on Lion you should not be using 1.1.0, it doesn't work properly and you should use 1.2.0.RC4. My guess is that your running on Snow Leopard???
