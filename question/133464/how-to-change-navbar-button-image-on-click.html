How to change navbar button image on click?

This is probably just me not thinking clearly today, but why is the code below not working to change a navbar button on click?

var location = Ti.UI.createButton({
    width:25,
    heigth:25,
    backgroundImage:'../images/buttons/locate1.png',
});
location.addEventListener('click',function(){
    location.backgroundImage = '../image/buttons/locate2.png';
}
win.leftNavButton = location;

I have also tried to set the backgroundSelectedImage property, but with no success… I'm developing for iOS btw.

    What you need to do is replace oldButton with newButton.

    
var oldButton = ...;
var newButton = ...;

oldButton.addEventListener('click', function() {
win.leftNavButton = newButton;
})
