  • Thanks for the link. I was able to workaround the issue by editing the following block in C:\ProgramData\Titanium\mobilesdk\win32\1.7.1\android\builder.py

            # start the emulator
        emulator_cmd = [
            self.sdk.get_emulator(),
            '-avd',
            avd_name,
            '-port',
            '5560',
            '-sdcard',
            self.sdcard,
            '-logcat',
            "'*:d *'",
            '-no-boot-anim',
            '-partition-size',
            '128' # in between nexusone and droid
        ]

    I replaced this:

    "'*:d *'",

    with this:

    "\"*:d *\"",
