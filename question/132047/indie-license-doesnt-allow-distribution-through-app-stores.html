Indie license doesn't allow distribution through app stores?

I am considering using Titanium Mobile but am concerned about the Indie Edition Agreement at http://www.appcelerator.com/company/agreements/indie-edition-agreement-2011-11-10/. It states:

"Customer understands and agrees that under no circumstances will Customer re-sell, re-distribute or re-package any portions of the Products, including without limitation, the portions of the Products included in the Applications, to another third-party. For the avoidance of doubt, redistribution of the Applications in a third-party application marketplace (such as Apple iTunes) is expressly prohibited."

"Applications" is defined as "the software application products that Customer will develop using the Products" so it appears Appcelerator is saying that I would not be permitted to sell applications I build using Titanium Mobile through the App Store or Android Market.

Can somebody from Appcelerator please explain this? Nothing else on the Appcelerator website suggests that commercial distribution of an app you develop using Titanium Mobile is forbidden so I'm hoping this is a misunderstanding!

We'll be publishing a formal clarification and updates to those docs ASAP.
