  • Fixed:

    For me this happened when I updated to 3.3.0.GA.
    I am running OSX 10.9.3

    The error occurred when I tried building for my Android device.

    The error message looked something like this:

    [ERROR] Failed to run dexer:
[ERROR]   
[ERROR] UNEXPECTED TOP-LEVEL EXCEPTION:
[ERROR] java.lang.IllegalArgumentException: already added: Lcom/appcelerator/cloudpush/APSCallbackHandler;
[ERROR]         at com.android.dx.dex.file.ClassDefsSection.add(ClassDefsSection.java:122)

    When I looked around in the modules I noticed that "aps-cloudpush-0.8.2.jar" and "aps-cloudpush-0.8.4.jar" were both in the lib folder of the cloud push module. I deleted "aps-cloudpush-0.8.2.jar" and now it builds fine.

    To find those files at least on a Mac go to: ~/Library/Application Support/Titanium/modules/android/ti.cloudpush/3.3.0/lib/

    Have a look at your Android modules to check to see if you have any duplicate libraries.

    Hope that helps someone…
