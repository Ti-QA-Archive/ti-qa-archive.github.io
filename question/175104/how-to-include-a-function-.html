How to include a function ?

Hi

I need to override the function L() so I created a new function called L() and index.js and it works. I have a second controller and I also need to override the function L() but I don't want to copy-past my code.

How can I do ?

I always see that I need to use a module. But I don't want to do like this because that means I have to call "mymodule.L()" and it doesn't override L().

Thank you

    Create a commonJS module with your overwrite function, require that in as your L() replacement.

    • Easy, thank you!

      index.js :

      var L = require('Functions');
alert(L(' my tag'));

      lib/Functions.js :

      function L(text) {
// ...
}
module.exports = L;
