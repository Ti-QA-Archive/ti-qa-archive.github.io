  • Actually Vincenty formula is more accurate than Haversine. This is because Haversine considers the earth to be a great sphere while Vincenty consider that the earth is actually an oblate spheroid, thus much more accurate.

    If you want, you can study Vincenty formula here: http://www.movable-type.co.uk/scripts/latlong-vincenty.html

    It has two variations where you can calculate the distance between two points, or given a point, bearing and distance, return the lat/long of point 2.

    There is a JS implementation that you can use on the page too.
