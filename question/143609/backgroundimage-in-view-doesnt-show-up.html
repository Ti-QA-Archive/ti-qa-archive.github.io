backgroundImage in View doesn't show up

I got an backgroundImage in a View but it doesn't show up.. I've no idea why. I want to place the image as background in a View in Test.js. All the code beneath is from Test.js.

The map structure is:

-resources
--app.js
--images
---progress.png (this is the images where it all about)
--ui
---Test.js

var statusView = Ti.UI.createView({
    width: '100dp',
    height: '20dp',
    top: '3dp',
    left: '3dp',
    backgroundImage: '/../images/progress.png',
    backgroundRepeat: true,
    borderRadius: 12,
    borderWidth: 1
});

The statusView is placed in statusViewContainer. statusViewContainer is placed in the window.

var statusViewContainer = Ti.UI.createView({
    width: (Ti.Platform.displayCaps.platformWidth - (Ti.Platform.displayCaps.platformWidth - screenWidth)),
    height: '26dp',
    top: '5dp',
    left: ((Ti.Platform.displayCaps.platformWidth - screenWidth) / 2),
    right: ((Ti.Platform.displayCaps.platformWidth - screenWidth) / 2),
    backgroundColor: "#555",
    borderRadius: 10,
    borderWidth: 1,
});

If I use a backgroundColor in statusView, it show the color as baclground. But if I use backgroundImage nothing, it doesn't show anything..

  • Views are pathed from the Resources directory. If your image is in the Resources/images folder, try changing your image path to: images&#x2F;progress.png
