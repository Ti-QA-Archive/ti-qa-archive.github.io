Get value of multidimensional array with string variable

Hi, i have next code:

var gasolinerasArray = new Array();

                    for (var c=0;c<gasolineras.length;c++)
                    {
                        gasolinerasArray[c] = {
                                                'id': gasolineras[c].id_gasolinera,
                                                'provincia': gasolineras[c].provincia, 
                                                'localidad': gasolineras[c].localidad,
                                                'distancia': gasolineras[c].distance,
                                                //'fecha':gasolineras[c].+this.config.tipoName,
                                                'rotulo': gasolineras[c].rotulo
                                        };
                    }
                datos.gasolinerasList = gasolinerasArray;

When i try to get gasolineras[c].field value when i use a variable instead field like commented line, doesn´t work. If my "this.config.tipoName" string variable = "distance" why doesn´t work like previous line
&#39;distancia&#39;: gasolineras[c].distance,
??

How can i access to an array value with a variable string?
thanks a lot
