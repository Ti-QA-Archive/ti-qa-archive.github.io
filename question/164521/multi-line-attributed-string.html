  • Hi Mauro,

    unfortunately this is technically not possible, even with pure Obj-C code. There a several obj-c threads on stackoverflow which are related to this, but nobody got a working solution.

    From a user interaction design perspective the text "Pull to Refresh." will only be visible if somebody already pulled the refreshControl, therefore it is redundant to show that line.
