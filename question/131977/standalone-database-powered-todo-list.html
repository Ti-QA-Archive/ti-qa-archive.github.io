Standalone database-powered todo list

I'd like to build a todo list manager that can run on the desktop (maybe in a browser?) that reads and stores it's data from some type of database backend. I need the database because I have meta data that I'd like to attach to the tasks for sorting and prioritization. Most importantly, I want the application to be able to run from a USB stick. I don't want to require installation, and I don't want it to live online. I'd simply like a system (similar perhaps to tiddlywiki) that could utilize a database and leverage CSS and maybe HTML5. Is this possible through appcelerator?

    Appcelerator actually won't officially be supporting the desktop project going forward, as it will be a community led effort. Even with the desktop product though, it does require installation of an embedded runtime. For something like what you're describing, you could probably distribute an HTML file which used local storage to provide the experience you're targeting, similar to tiddlywiki.
