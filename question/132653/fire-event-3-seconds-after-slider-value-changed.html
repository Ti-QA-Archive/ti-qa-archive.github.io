Fire event 3 seconds after slider value changed

I want to fire an event 3 seconds after user has finished using a slider. I can't use the touchend event as the user may still be touching the slider.

Its important that the event isn't queued up multiple times ie. if the user moves the slider the event is scheduled for 3 seconds time. If the user moves it again, the previous scheduled event is cancelled and a new one event is scheduled.

Can someone tell me if this is possible and/or how to do it?

Thanks.
