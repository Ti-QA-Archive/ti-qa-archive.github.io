  • We are having the same issue here too.

    I suspect it is directly related to changes made to Appcelerator Studio 4.0.0. I think that the CLI has changed. In the directory where the npm modules are usually stored, there is a new directory called /appcelerator (The usual Titanium directory is no longer there if installing on a clean machine)

    See:
    /usr/local/lib/node_modules

    My guess is that the tiCalabash plugin needs to be ported to use the CLI for 4.0.0
