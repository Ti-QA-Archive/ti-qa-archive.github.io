  • Accepted Answer

    With your S4 plugged in, from a command prompt, within the project directory, try the following:

    ti build -p android -T device

    Also, be sure that the USB connection type is MTP and NOT PTP.

  • Another method I found that worked for me is to open Terminal

    $> cd [into android sdk folder]

$> adb kill-server
