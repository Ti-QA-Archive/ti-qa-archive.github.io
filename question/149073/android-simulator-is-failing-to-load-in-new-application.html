    Hi

    Make sure the emulator is closed and then run the project > clean option on your project.

    Another thing to try is running the uninstall option inside the emulator, this will remove the app in the same way as it would on a physical device. Keep in mind this will remove any databases, files and settings you created.

    • To delete AVDs (Android Virtual Devices);

      • Navigate to the Dashboard tab in Titanium Studio
      • Select Geting Started
      • Choose Android
      • Choose Update Android SDK
      • From the Android SDK Manager menu;
        • Tools
        • Manage AVDs…
        • Select the AVDs and remove them all

      Appcelerator will recreate new AVDs as it needs it.
