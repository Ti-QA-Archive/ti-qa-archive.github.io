"Image" property of TI.UI.imageView, is not working for android in titanium 1.7.5

Android Mobile Project, Titanium 1.7.5, android emulator 2.3.3 ,

Hi ,

In my application "image" property of Ti.UI.imageView, is not working for "ping" and "jpeg" images. Its not giving any error. It's working fine only for blob images.

The code is as follows:

var win  = Ti.UI.currentWindow;
var img = Ti.UI.createImageView({
               image:'images/rec-tran.png',
            //backgroundImage:'images/rec-tran.png',
            width:'100dp',
            height:'100dp',
            left:'15%',
            top:'10%',
            focusable:true,
      });
win.add(img);

The backgroundImage property is working fine.

Is there any solution..?

Thanks.

    • It's possible to obtain proportional dimensions by only setting one, such as the width, as follows: 

      
var win = Ti.UI.createWindow({
    backgroundColor:'#ddd',
    fullscreen:false,
    exitOnClose:true
});

var imageView1 = Ti.UI.createImageView({
    image:'KS_nav_ui.png',
    top:10,
    width:50,
    left:5
});

var imageView2 = Ti.UI.createImageView({
    image:'KS_nav_ui.png',
    top:10,
    width:'50dp',
    left:105
});

var imageView3 = Ti.UI.createImageView({
    image:'KS_nav_ui.png',
    top:10,
    width:'50%',
    left:205
});

win.add(imageView1);
win.add(imageView2);
win.add(imageView3);
win.open();
