Temporary Photo

I have an app that allows user to take picture and upload to our server. My question is: Is it possible for the picture to be temporarily stored on device and then once its uploaded remove itself automatically, so that it does not save to device?

  • Hi F.F,

    write that picture to device.

    Delete that picture after successful upload.

    Thats it, there is no magic automatic thing for that use case.

    Because if you would use Titanium.Filesystem.tempDirectory (createTempDirectory( )and createTempFile( ) the Files in this directory may not persist when the application is shut down and restarted. So if you close your App and upload was not finished, the files are gone.
