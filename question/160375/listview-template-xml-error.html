ListView Template XML Error

I downloaded the newest Alloy version 1.2.2 using npm as described in the docs.

I am able to build and run if I generate a template from .js code but when I attempt to create the template in the XML I get the following error.

[ERROR] : Child element must be one of the following: [Ti.UI.ListItem]

My XML is the following:


<Alloy>
    <Window>
        <ListView id="alertList" defaultItemTemplate="alertTemplate">
            <ListSection id="alertsSection">
                <Templates>
                    <ItemTemplate id="alertTemplate">
                        <ImageView id="pic" bindId="alertImage"></ImageView>
                        <Label bindId="alertLabel"></Label>
                    </ItemTemplate>
                </Templates>
            </ListSection>
        </ListView>
    </Window>
</Alloy>

I modified the tiapp.xml file to have the following plugin information:

<plugins>
    <plugin version="1.2.2">ti.alloy</plugin>
</plugins>

Is there more required to update an existing project to use the newest version of Alloy?

Thank you.

    Hi

    I am nowhere near my development machine to check - but this may be something simple.

    You have nested the template xml INSIDE the ListView section, when it should be the same level as sections if you have then or rows if you do not. As sections come from a single pool per list you choose then per row not per section.
