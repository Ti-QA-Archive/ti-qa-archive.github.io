Connect to local machine with the android emulator

I would like to, while developing, connect to my local machine. But, it looks like 127.0.0.1 connects to the emulator itself, and setting my hosts file on my windows computer does not seem to be working either.
Is there a solution for this? So far I have been unable to find it.

  • The Android emulator acts as it own device (with it's own IP Address. So when you access 127.0.0.1, you are actually trying to ping the emulator itself.

    The best way to access the local machine is by using the machine's IP address.
