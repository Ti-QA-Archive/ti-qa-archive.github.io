A view's left, top, height and width values in inch?

###Hi

I am using CLI version 3.2.3, Titanium SDK version 3.2.2.GA

Developing a app for Android and iOS devices, I was wondering can we have values in inches?
I am making use of Annotorious.js library in my app to have image annotations, and that library is storing values in inches I belive(At least not in pixels).

Like 

"geometry": {
        "x": 0.11580882352941177,
        "y": 0.185589519650655,
        "width": 0.04044117647058824,
        "height": 0.05240174672489083
}

Here values are given in inches(Annotorious default values)

Thanks,

    Hi Maruti,

    Yes it is possible to set these values to inch. You just have to set the value to a string suffixed by "in".

    see the docs for more information.

    var view = Ti.UI.createView({
    top: "0.185589519650655in",
    left: "0.185589519650655in",
     width: "0.04044117647058824in",
    height: "0.05240174672489083in"
});
