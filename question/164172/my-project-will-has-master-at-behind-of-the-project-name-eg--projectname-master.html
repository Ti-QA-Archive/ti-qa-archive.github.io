  • Accepted Answer

    This indicates the git branch that you're currently working on. If you do not use git you could just remove the local repository by deleting the .git file in your root directory. If you just don't want the git management within Titanium Studio because you are using the git command line (or another git GUI) then you could remove the git Aptana plugin from TiStudio.

    • Hi joyce,
      This problem of your's probably because of the android node you have in your tiapp.xml. Please check when you run your application in emulator then in console , there should be a log like —- staringt .xyz activity. So this .xyz activity is the one should be there in you tiapp.xml like this

       <activity
                    android:configChanges="keyboardHidden|orientation|screenSize"
                    android:label="@string/app_name"
                    android:name=".xyz" android:theme="@style/Theme.Titanium">
                    <intent-filter>
                        <action android:name="android.intent.action.MAIN"/>
                        <category android:name="android.intent.category.LAUNCHER"/>
                    </intent-filter>
                </activity>

      please check that this xyz is same in the tiapp.xml also, if this is different then there would be 2 icons in your android.

      Please ask if any confusion
