MapView Annotations with custom image - Animations don't work

I am wanting to support more pin colors on the Annotations than red/green/purple. I can do this by specifying an image to use for the pin like so:

var annot = Ti.Map.createAnnotation({
   latitude: addresses[i].latitude,
   longitude: addresses[i].longitude,
   title: addresses[i].address,
   subtitle: '',
   animate: true,
   image: '/images/pins/'+addresses[i].pincolor+'.png'
});

The problem is that the animation no longer works when I use a custom image.

Is this a bug?

If I comment out the "image" attribute and specify a pincolor then the animation works again.
