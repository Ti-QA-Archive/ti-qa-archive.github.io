  • function capitaliseFirstLetter(string)
    {
    return string.charAt(0).toUpperCase() + string.slice(1);
    }

  • Hi

    As you are using Alloy you might as well take advantage of the built in string library;

    • Alloy.builtins.string

    Usage example;

    // assumes that "json[i]" hold the string
var string = require('alloy/string');
for (i = 0; i < json.length; i++) {
    nameLabel = Ti.UI.createLabel({
        color: '#000',
        height: Ti.UI.SIZE,
        text: string.ucfirst(json[i]),
        touchEnabled: false
    });         
}

    The string library contains many useful functions not available in plain JavaScript.
