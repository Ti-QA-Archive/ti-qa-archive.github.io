Images not displayed on device - Alloy Framework

I'm using alloy framework and For android OS, I'm keeping my images under the folder - /assets/android/images/

When I'm running the app in the simulator the images are coming correctly, but when I tried the same in my device, the images are not getting displayed.

Code snippet

<View id="headerBanner" class="header">
            <ImageView id="headerImage" image="/android/images/bannerImage.png" />
        </View>

Could anyone please tell what is wrong in here ?

  • Accepted Answer

    image="/android/images/bannerImage.png" />

    android and iphone are magic words in paths that get striped out on build but their contents are referened without that part. So a file in android/images/ would be referenced as images/ and would take precedence on the android platform over anything physically in images/*
