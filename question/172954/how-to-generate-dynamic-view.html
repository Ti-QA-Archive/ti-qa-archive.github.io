How to generate dynamic view

var view  = Titanium.UI.createView({
  top: 10,
  left: 10,
  height: 'auto',
  width: 'auto',

});

    Hello Dev eloper,

    Something like below :

    var arrView = [];

for (i=0; i <10 ; i++){
    var view  = Titanium.UI.createView({
      top: 10,
      left: 10,
      height: 'auto',
      width: 'auto',
     });

    arrView.push(view);
}


// get view 

alert(arrView[0]);
