TableViewRow Show/Hide

Hello,

For showing or hiding the TableViewRow in the API Reference there is two methodes and a property :

  • show() or hide() : these functions does not work.
  • visible : Setting this property to true or false does not work neither

The suggestion between the questions that i read is to set the height to 0, but if i set the height to 0 the TableViewRow becomes bigger !
BUT if i set the height to 1 it works correctly.

Is it a bug or i missed something in TableView or TableViewRow configuration ?

In my TableViewRow there is ViewImage, so if i set the of TableViewRow to 1, it must hide() the ViewImage too, otherwise, we can still see the contain of TableViewRow

Is it the best solution for hiding a TableViewRow ?

My code :

var main_win = Titanium.UI.createWindow();

var tbl_view = Titanium.UI.createTableView({
    backgroundColor : 'white',
    top                : 60
});

data = []
for (var i=0; i < 4; i++) {
    row = Ti.UI.createTableViewRow({
        title : "hello #" + i.toString(), height : 30
    });
    data.push(row);
}

var row_remove_button = Titanium.UI.createButton({
    left : 0, top : 0, width : 48, height : 48
});

row_remove_button.addEventListener('click', function(e){
    tbl_view.data[0].rows[3].setHeight(0);
})

tbl_view.data = data;
main_win.add(tbl_view);
main_win.add(row_remove_button);
main_win.open();
