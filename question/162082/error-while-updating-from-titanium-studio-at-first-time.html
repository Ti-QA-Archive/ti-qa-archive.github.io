    • If you have nodeJS installed properly (nodeJS.com) and are SURE that you have your proxy setup, or are not behind a proxy, then you can try to bypass the studio updater and install manually:

      [sudo] npm install -g titanium

[sudo] npm install -g alloy

[sudo] npm install -g acs

      If you have failed to properly setup your proxy, the one mentioned in the log, but not mentioned by you in your question, then you can try to set it up for the cli:

      ti config cli.httpProxyServer "proxyServerSettingsGoHere"
