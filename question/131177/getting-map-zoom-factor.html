  • What about something like this?

    var zoomLevel = 5;//or whatever you set the default zoom level to.

mapView.addEventListener("zoom", function(e) {
  zoomLevel = e.level;

  • It should be possible to compute the zoom level using the info reported by the "regionChanged" event, which is basically the currently visible region. However this computation is a bit tricky and involves knowing exactly the dimensions of the map view in pixels. I don't have some ready to use Javascript code for this, but here's a pointer to more info on the topic:
    http://troybrant.net/blog/2010/01/mkmapview-and-zoom-levels-a-visual-guide/
