Alloy dynamic tab creation

I am trying to create dynamically a set of tabs that would be added on top of an existing tabgroup defined in a view with no luck.

Here is what I 've got so far:

index.xml

<Alloy>
    <TabGroup id="tabgroup">
        <Tab id="tabHome">
            <Require id="home" src="home" type="view" />
        </Tab>
    </TabGroup>
</Alloy>

index.js

for (var i = 1; i < tabs.length; i++) {
    var tab = Ti.UI.createTab({
        title : tabs[i],
        window : Alloy.createController(tabs[i]).getView()
    });

    $.tabgroup.add(tab);
};

$.tabgroup.open();

The app loads up and displays only the first (home) tab which renders correctly. Apart from that there are no other tabs. Meaning that the tabs that are created in the code, are not added to the tabgroup.

I have also tried to create a simple empty window instead of loading a controller but it's still not working.
i.e. replace window : Alloy.createController(tabs[i]).getView() with window : Ti.UI.createWindow()

Any food for thought welcome.

    never mind, I found the culprit. it's $.tabgroup.addTab(tab); and not just $.tabgroup.add(tab);
