  • Hi devyani,
    You can get value of your textfield using the following code. 

    var value = surname.value;     // where surname is the name of your textfield.

    and then you can display it using labelView like this

    var label = Ti.UI.createLabel({
    text: value      // variable you used to get text field value
    height: "auto",
    width: "auto",
    font: {fontSize: 20, fontWeight: 'bold'},
    top: "10%"
});

    Hope it will help.
