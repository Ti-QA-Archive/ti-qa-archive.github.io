Production deployment with iOS 8 support

We built our app with Titanium 3.4 Release Candidate for iOS 8. The blurb on the RC says we cannot use it for production builds. When will the GA version be available so we can build and deploy to the store?

— asked September 22nd 2014 by Gopi Reddy

  • You can use 3.4.0 for production, with the following two assumptions:

    1. None of the current known issues affect you, and you are comfortable using a build that is not yet certified as GA.
    2. We have not yet completed testing on iPhone 6/6+ devices, so it is possible there is an issue there (but none have yet been reported by the community).
