Dial a number and then send additional numbers

I know you can dial a phone number with Ti.Platform.openURL() but can you then send additional numbers? This would be for calling a number, and then entering say a calling card code once it connected.. so that would also require detecting that the first call connected before sending the additional codes.

Is that possible in titanium? Is this possible for an Objective-C native app even?
