Copy or move a directory and all of it's contents inside

I need to move a folder and it's contents from resourcesDirectory to applicationDataDirectory and i was wondering if there was any way to do it by just interacting with the parent directory…

in the past i've done the same thing this way:

    var file1Source = Titanium.filesystem.getFile(titanium.filesystem.resourcesdirectory, 'file1.txt');
    var file1Destination = Titanium.filesystem.getFile(titanium.filesystem.applicationdatadirectory, 'file1.txt');

file1Destination.write(file1source.read());

^^^ but this way i have to write one by one every single file i need to copy…

I figured out today that in iOS i can do the same with directories:

    var dir1Source = Titanium.filesystem.getFile(titanium.filesystem.resourcesdirectory, 'dir');
    var dir1Destination = Titanium.filesystem.getFile(titanium.filesystem.applicationdatadirectory, 'dir');

dir1Destination.write(dir1source.read());

and this wau i create the directory i want to copy and 'copy' also everything what's inside the directory

I need something that works just about the same way for Android…
if i use the secon example i showed above in Android it says that there was an filenotfoundexception…

Any help?
thnx in advance…
