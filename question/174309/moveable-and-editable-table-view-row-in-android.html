Moveable and Editable table view row in android.

I have created an app in which I used moveable=true and editable=true for deleting row. It is working fine in IOS, I need that feature in android. Do you guys know how can I achieve this functionality in Android. 

<Alloy>
    <TableViewRow hasChild="true" editable="true" moveable="true" id="rowView" class="row" backgroundColor="white">
    </TableViewRow>

</Alloy>

  • Hi

    Both of these properties are iOS only.

    • editable
    • moveable

    Android has no native variation of these iOS only facilities.
