  • It's been fixed in the latest build which you can download from http://preview.appcelerator.com/studio/

    OR

    You can edit the TitaniumStudio.ini file by adding the following line at the end of the file.
    -Dtitanium.bypassAuthentication=true

    TitaniumStudio.ini location:

    Windows 7 - C:\Users\Administrator\AppData\Local\Titanium Studio\

    OSX - /Applications/Titanium Studio/Titanium Studio <Right click show package contents>/Contents/MacOS
