  • There is a sample in the KitchenSink that does just that.

    https://github.com/appcelerator/KitchenSink/blob/master/Resources/examples/options_dialog.js#L117

    The basic principle is the following:

    • You create a view and add the controls you want.
    • You create your AlertDialog.
    • You set the androidView property with the view you just created above.
    • You show the AlertDialog.
