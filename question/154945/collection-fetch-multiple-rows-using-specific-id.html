Collection fetch multiple rows using specific id

HI Friends,

 I am creating an android app (chat) using alloy framework, i am using sqllite to store chat history of my app. i have table to store chat history with respect to chat topic id, that is chat history table contain two coloumns one is chat_topic_id and other is message. So each chat topic may contain multiple rows. Here i want to fetch rows of specific chat topic and display using table views.

 var x = Alloy.createCollection(history)
      x.fetch();

the above code will list all messages in the table i need only messages of a specific chat topic, how can i fetch using and id?
I know i can fetch a single row using model.fetch(). but here i have multiple rows
Please any one help.

  • Just pass a SQL query into the fetch. I'll assume your table name is "history":

    var someId = 123;
var x = Alloy.createCollection('history');
x.fetch({
  query: 'SELECT * FROM history WHERE chat_topic_id = ' + someId
});
