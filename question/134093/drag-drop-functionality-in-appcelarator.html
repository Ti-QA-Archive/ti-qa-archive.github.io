  • Hi Harini,

    If you want to drag and drop your tableView Row then you have to set 

    var tableview = Titanium.UI.createTableView({
    data:data
});

tableview.moving = true;

    or you can set in tableview like this,

    var tableview = Titanium.UI.createTableView({
    data:data,
    moving : true
});

    Hope this may help you.
