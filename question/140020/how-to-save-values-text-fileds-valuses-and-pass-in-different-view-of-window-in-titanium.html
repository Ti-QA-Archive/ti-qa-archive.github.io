  • hello sumit,

    you can do it with the many ways

    let's try this…

    create array of the values of textField

    var arr = [];

arr.push(textField1.value);
arr.push(textField2.value);
arr.push(textField3.value);

    now when you are creating new window just pass this array as property

        var win = Ti.UI.createWindow({
        url : 'newwin.js',
        Myarr : arr    
    });
    win.open();

    now you can access this array at your newwin.js file

    var win = Ti.UI.currentWindow;

alert(win.Myarr);

    so try this and let me know the results…i am sure it will working.
