  • Hi

    The closest match to your request is the following;

    Assuming this layout xml

    <Alloy>
    <View id="view" borderRadius="10" backgroundColor="red" width="50" height="50" />
</Alloy>

    Then;

    $.view.setBackgroundColor('green');

    Where view is the id in the view.

    Keep in mind that this will not always work as you want, for example there should only be one ID per thing and they must be unique - same as your jQuery reference.

    • It is as simple as this;

      function alertIdVal(id) {
    alert($[id].getValue());
}

      Obviously you can only use the correct getters and setters for the object you are referencing - in this case it would be a; switch, textfield, textarea but not a view, window etc.

      Remember that in JavaScript the following are logically equal;

      obj.firstname = 'Malcolm';
obj['firstname'] = 'Malcolm';
