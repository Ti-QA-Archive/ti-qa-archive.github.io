Going back to a stable build.

I currently get the nightly builds of Studio but would like to revert back to the latest stable build as I am about to work on a project for a customer. When I try this via the install software option it won't install with the error message saying it is because I have a later version installed. What's the easiest way for me to revert back to the stable build?

  • Due to a limitation in Eclipse, you can only revert back to an update site which is still available. Thus, you can revert from a beta back to the current stable (assuming you installed that), but not from one stable back to the previous one. We've updated the docs to reflect that. This may be possible to work around, and we are investigating possibilities.

    — commented January 6th 2012 by Ingo Muschenetz
