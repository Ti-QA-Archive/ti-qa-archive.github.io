  • Hello Palpandian,

    The easiest way to change the cursor color on textfield is to use your own Android styles.
    Modify your tiapp.xml with the following:

    <android xmlns:android="http://schemas.android.com/apk/res/android">
        <manifest>
            <application android:theme="@style/Theme.AppCompat.Light"/>
            <!-- Need to specify at least API level 11 for Titanium SDK 3.2.x and prior -->
            <uses-sdk android:minSdkVersion="14" android:targetSdkVersion="19"/>
        </manifest>
</android>

    Second, Create a styles.xml file and a colors.xml file in [Project]/platform/android/res/values, relative to the main project folder.
    In styles.xml, add the following code.

    <resources>
    <style name="Theme.Custom" parent="@android:style/Theme.Holo.Light">
        <item name="android:editTextStyle">@style/editText</item>
    </style>
    <style name="editText" parent="@android:style/Widget.Holo.Light.EditText">
        <item name="android:textCursorDrawable">@null</item>
        <item name="android:textColorHint">@color/cursor</item>
    </style>
</resources>

    And in colors.xml, add the following code.

    <?xml version="1.0" encoding="utf-8"?>
<resources>
    <color name="cursor">#fff</color>
</resources>

    Test in the main project use following code

    index.xml

    <Alloy>
    <Window layout="vertical">
        <View class="wrapper">
            <TextField id="whiteInput" hintText="input here"></TextField>
        </View>
        <View class="wrapper">
            <TextField id="blackInput" hintText="input here"></TextField>
        </View>
    </Window>
</Alloy>

    index.tss

    ".wrapper":
{
    height: '200dp',
    width: Ti.UI.FILL,
    backgroundColor: 'blue'
}
"#blackInput":
{
    backgroundColor: 'red',
    color: '#fff'
}
"#whiteInput":
{
    backgroundColor: "#000",
    color: '#fff'
}

    For more information about Android theme . Thanks
