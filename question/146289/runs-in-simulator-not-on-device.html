  • Hi

    A few things to check;

    • Make sure you have run Project > Clean BEFORE compiling for device.
    • Make sure you have development certificate for your app and that it matches your Application Id
    • If Test.Alloy.one is your Application Id then you need to make sure you have development certificate that matches that - although the convention is reverse domain - for example com.apple.test1 or com.yourcompany.test1.
    • If you have a custom plist, try it without it.
    • Check the buildlog
