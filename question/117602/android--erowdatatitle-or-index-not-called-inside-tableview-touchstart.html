  • You have to know what element is dispatching the event.
    I had a similar problem with tableView, the event was dispatched by the table row content,
    I was trying to get the value of the row data, the label was dispatching the event. 

    // you need to add an Id to your elements, including everything inside your row.
// then you can get the id of the element

myTableView.addEventListener('touchStart',function(e){
 alert(e.source.id);
})
