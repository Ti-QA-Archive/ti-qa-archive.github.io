  • Yes. You an use xhdpi density specific resources in Titanium, but the android/res-*-xhdpi directories are not created by default in new projects. You will need to add them yourself. You don't need to make any changes in tiapp.xml. See https://github.com/jeffbonnes/dipscheck for an example.
