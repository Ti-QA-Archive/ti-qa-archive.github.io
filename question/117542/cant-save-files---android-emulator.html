Can't save files - Android emulator

Hi,

I'm currently working through the kitchen sink app using the android emulator, and have currently hit a wall with the file system example.

I've have copied and tried running the following code:

var newDir = Titanium.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory,'mydir');
Ti.API.info("Created mydir: " + newDir.createDirectory());
Ti.API.info('newdir ' + newDir);

This doesn't work as the newDir.createDirectory() function returns false.

This also happens when trying to create and save files.

I'm just wondering if there's some setting or something related to the android emulator. For example when saving to Titanium.Filesystem.applicationDataDirectory, where would I find this directory on my computer.
