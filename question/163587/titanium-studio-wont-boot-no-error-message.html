  • Okay - no idea what had actually happened but I moved the .metadata folder out of the Titanium Studio Workspace folder and it loaded first time! Obviously, I'd lost my preferences etc but at least it got it loading again. Just in case anyone else has a similar problem - try moving (or renaming) the /metadata folder and Ti Studio will create a new one…
