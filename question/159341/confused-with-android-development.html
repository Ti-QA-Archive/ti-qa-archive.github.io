  • Hello Dave and welcome.

    To write Titanium apps (any target) you can use Titanium Studio (the eclipse based ide) or you can use a text-editor of your choice and the terminal with the titanium command line interface. In any case to monitor/debug android apps on devices you can invoke ddms before building, then use it the way you are used too. hth, regards.

  • Although it is entirely possible to use eclipse or a basic text editor, it's a much better idea to start off using Ti Studio - the IDE that is supplied with the SDK. Within this editor you can compile, run and debug your app and set breakpoints etc as you would expect from an IDE. I would start off by reading this mini-guide to get an understanding of how to develop and debug Apps with Ti Studio
