TableView "Zebra Stripe" Effect

Is it possible to add a "zebra stripe" effect to a table view?

My first idea was to just manually change the background color of every other table cell, but this will cause a problem when using the table's search. Any other ideas?

  • Hi Jacob

    It's simpler than you might think! ;) Programmatically create your rows. With each iteration of the loop, set a boolean value to true or false depending on its current value, and toggle the boolean value.

    Try this:

    Ti.UI.setBackgroundColor('white');

var win = Ti.UI.createWindow(); 

var switchColor = true;

var data = [];

for(var i=0; i<20; i++){
    var thisRowData = {
            color:"white",
            title:"Row "+i
    };
    if(switchColor){
        thisRowData.backgroundColor = "#CC0204";
        switchColor = false;
    } else {
        thisRowData.backgroundColor = "#DC4E4C";
        switchColor = true;
    }
    data.push(thisRowData);
}

var tableview = Titanium.UI.createTableView({
    data:data
});

win.add(tableview);
win.open();
