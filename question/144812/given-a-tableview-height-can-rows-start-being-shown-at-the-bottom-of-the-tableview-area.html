  • Hi Imanol

    Please provide some code for use to review and ideally a picture of what you are trying achieve.

    For example your short description does not explain if the screen has a keyboard, textfield and table showing at the same time and how they are positioned on the screen.

    However if you are adding the rows one at a time look at the following methods of the tableView;

    • appendRow
    • scrollToIndex

    Using these two methods you can add each row to the table one row at a time and then jump to the last index. However this will add the rows to the top of the table first and only once the rows exceed the visible space will they 'appear to show' as if they came from the bottom.

    Possibly a better approach would be to use a scrollView, add items to it and then use scrollToBottom.
