Need help converting a curl call to Ti.Network.createHTTPClient

Hi,

I am trying to include Paypal payments into my app, but the Ti module makes my app crash with the newest version of Titanium, so I am now trying to use their REST API.
In the paypal docs they give an example curl call:

https://api.sandbox.paypal.com/v1/oauth2/token \
 -H 'Accept: application/json' 
 -H "Accept-Language: en_US" 
 -u "userid:password" 
 -d "grant_type=client_credentials"

I can successfully make this call in my terminal using my own credentials, etc, but I cannot get it to work using the Ti.Network.createHTTPClient call. No matter what I try I get the 'Authentication Needed' error response. Could someone help me construct this call ?

Thanks so much.

  • Accepted Answer

    var data = {grant_type: 'client_credentials'}
var xhr = Titanium.Network.createHTTPClient();
xhr.open('POST', 'https://username:password@api.sandbox.paypal.com/v1/oauth2/token');
xhr.setRequestHeader('Accept','application/json');
xhr.setRequestHeader('Accept-Language','en_us');
xhr.send(data)
