Using canvas into titanium

So I made a tiny little game in HTML5 and I want to import it into titanium studio
and hopefully get an android .apk file out of it.

I have an index.html file, which contains a canvas and a link to my javascript code, a js file, and a png file which contains all my graphics.
It works fine in the browser but when I import the html file, the png file and the
js file….I get errors. PS: I put all of these files in the "Resources" folder.

I then noticed there was an "app.js" file in the folder, so I simply replaced all of that code with the one in my js file. Still got errors.

All my JS code does is use the canvas functions. Again, works perfectly in an HTML5 supported browser, but gets errors in Titanium Studio.
Am I supposed to be replacing app.js with my js file? Am I not supposed to include my .html file in the Resources folder? Please help I'm very new.

  • You would have to create a window, a webview, attach the webview to the window. Set the url of the webview to your index.html then open the window. It should work out of the box.

    What are you trying to do it won't work because Titanium is not just simple html/js but you need to build native components.

    Maybe you should start reading some docs before you put yourself to work :)
    Here is a good start:
    http://wiki.appcelerator.org/display/guides/Home
