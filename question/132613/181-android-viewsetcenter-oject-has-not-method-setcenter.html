1.8.1 Android view.setCenter() "Oject has not method 'setCenter'"

This may be a bug. I'm using sdk 1.8.1 and trying to set the center on a view as follows:

var buttonView = Ti.UI.createView({
    height : 206,
    width : 320
});
buttonView.setCenter({x:'50%',x:'50%'});

On android I get an error "Uncaught TypeError: Object[object View] has new method 'setCenter'"
