Apostrophe in my app name causes it not to build for android.

Hi, I am making an app and the project name (Let's Chat) has an apostrophe in it. I have only been building for the iOS emulator until today when I tried to build for an android device. It threw this error:

> Failed to package application:
> error: Apostrophe not preceded by \ (in Let's Chat)

How do I work around this? I tried just renaming the project and that completely breaks it. I also tried creating a new app named correctly and then copied the code over but then it wouldn't build for anything.
