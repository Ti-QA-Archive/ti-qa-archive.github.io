Pass in Label text to function

I have the following:

<Label id="LocalNumber" class="phoneNumber" onClick="dial">123.456.1111</Label>
<Label id="LocalNumber" class="phoneNumber" onClick="dial">123.456.2222</Label>

function dial(e)
{
    // pass in Label text and open phone dialer
    Titanium.Platform.openURL("tel:e.source.???");
}

I'm new to Titanium. What is the best way to go about passing in variables to a common function like above? Thanks for any tips.

    Hi

    Try this;

    function dial (e) {
    // pass in Label text and open phone dialer
    Ti.Platform.openURL("tel:' + e.source.getText());
}
