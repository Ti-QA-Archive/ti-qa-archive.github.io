  • You need to first create an Appcelerator Network account here: https://my.appcelerator.com/auth/signup. After you've confirmed your email address, you can then log in with the Titanium CLI via the titanium login command.

    If you already have an Appcelerator Network account, but forgot your password, you can reset it here: https://my.appcelerator.com/auth/login.
