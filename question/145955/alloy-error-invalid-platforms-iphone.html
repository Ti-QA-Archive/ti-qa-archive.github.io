    • Format

      alloy run PATH PLATFORM

      Examples

      alloy run . android
alloy run /path/to/project iphone

      NOTE: alloy run will be deprecated soon in favor of using the new titanium CLI command to run apps. alloy run is not full featured and was added as a simple convenience to those testing apps with Alloy. Now that TiStudio integration is underway and the new CLI can run Alloy apps, there's no need to have yet another way to run alloy apps that deviates from the standard CLI's path.
