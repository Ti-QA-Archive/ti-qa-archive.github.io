  • You could make a custom toolbar, putting regular buttons in a view.
    If you have 3 buttons in a view, you can set all 3 buttons backgroundColor property when one of them is clicked.

    buttonONE.addEventListener('click',function()
{
    buttonONE.backgroundColor = '#fff';
    buttonTWO.backgroundColor = '#000';
    buttonTHREE.backgroundColor = '#000';
});
buttonTWO.addEventListener('click',function()
{
    buttonONE.backgroundColor = '#000';
    buttonTWO.backgroundColor = '#fff';
    buttonTHREE.backgroundColor = '#000';
});
buttonTHREE.addEventListener('click',function()
{
    buttonONE.backgroundColor = '#000';
    buttonTWO.backgroundColor = '#000';
    buttonTHREE.backgroundColor = '#fff';
});

    May not be the most efficient method so I would be happy to see alternatives.

  • How about this?

    var btn = Ti.UI.createButtonBar({
  labels : ['Btn Name'],
  style : Ti.UI.iPhone.SystemButtonStyle.BAR,
  backgroundColor : 'red'
});
win.leftNavButton = btn;
btn.addEventListener('click', function(e) {
  e.source.backgroundColor = 'green';
});
