  • The geo datas are part of EXIF.You can open the images in a buffer and parse the infos. Here the code to read photos in buffer:

    exports.get = function(foobuffer now the byte) {
    var instream = Titanium.Stream.createStream({
        mode : Titanium.Stream.MODE_READ,
        source : foo // e.media is a Blob
    });
    var buffer = Ti.createBuffer({
        length : foo.length
    });
    while ((instream.read(buffer)) > 0) {}
    instream.close();
    return buffer;
}

    In buffer are no the bytes and ready to analyze.
