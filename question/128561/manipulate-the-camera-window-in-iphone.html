  • you can use overlay and transform properties of Ti.Media.ShowCamera

    use

    var tr = Ti.UI.create2DMatrix({scale : 0.6});//for 60 % here
Ti.Media.showCamera({
transform : tr,//it will resize camera
overlay : view // any view you want to overlap camera window but it will not resize camera 
});
