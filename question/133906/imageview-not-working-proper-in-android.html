  • Accepted Answer

    hi Sarafaraz,

    i think that is the problem of scaling …

    because when you set the image larger then your imageView size then fails to scale that image

    or

    you should try to set the imageView's size in the ratio of the image size

    or

    if you want to set that image exactly in your imageView then you can use it's backgroundImage property to set the image

    or

    you can use a simple view with backgroundImage property

    if your requirement is to show the image only then it's better to use imageView
    or if it contains more than one controls then it is better to use view instead

    thnx
