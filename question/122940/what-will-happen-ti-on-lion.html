  • So much BAD information. And OLD information at that.

    • Install Lion.
    • Update Software (should be a Java runtime update)
    • Uninstall xCode 3.x or 4.0.x (sudo /Developer/Library/uninstall-devtools –mode=all) This is from Christian Sigl.
    • Download and install xCode 4.1
    • Run Titanium, install any updates when prompted.
    • Possibly setup your provisioning profiles in Titanium again.
    • Compile your apps.

    Have seen problems from folks where the xCode install seems to have forgotten the command line prompts, so the titanium python scripts fail. Do NOT assume your problems are Titanium based.

    A number of us are running fine. Paul Caldwell, Christian Sigl
