Size ok APK over megabyte

Hi,
for my first app for android SDK I see that size of apk is over 6MB (not in develop mode) and on to my device size of installed app are over 10MB.
Image in this package are about 200KB.
There is a method to reduce size of package? I hope that in apk there are library or package not used… there is a way to not include them?
Tnx

    The smallest a titanium app for android can be is about 6mb, this is because the JS engine V8 must be bundled with the app to process the JS at runtime…
    Titanium is already only using the needed libraries… it filters those when building the APK
