  • It seems the problem solves when you wrap things up in a container, ex. a Ti.UI.view.
    like:

    function createDashboardItem(label, image){
    var item = Ti.UI.createDashboardItem({});
    var container = Ti.UI.createView({
        height : 100,
        width : 150
    });
    container.add(Ti.UI.createImageView({
        image : image,
        width : 80,
        height : 80,
        top : 0
    }));
    container.add(Ti.UI.createLabel({
        text : label,
        textAlign : 'center',
        color : 'white',
        font : {fontSize:12},
        height : 14,
        bottom : 10
    }));
    item.add(container);

    return item;
}

    add the item to the dashboard then
