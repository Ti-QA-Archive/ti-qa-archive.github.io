    Hello Clifton,

    Very probably you've already seen this guide: Alloy_Styles_and_Themes which explains the Titanium Style Sheets (TSS) being used to styling elements.

    I'm not seeing the actual OS platform where this has to be implemented, however there is a Feature request filed already to get padding to work e.g. TiAPI: Add Padding to Views.

    Everybody is encouraged to Submit a Bug Report if you consider lack of details on docs are missing important info so they can be enriched or even if there are further discrepancies or any other outstanding issues.

    I have not tried to do padding on a view, so you may want to employ a Native WebView so CSS styles can get it working.
