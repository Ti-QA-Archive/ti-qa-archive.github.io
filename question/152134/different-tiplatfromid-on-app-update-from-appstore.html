  • Accepted Answer

    It stays the same :

    • if the user update your app
    • if the user make a backup and reinstall it on another device (devs never think about this one)

    The id is regenerated on iOS :

    • if the user uninstall and then reinstall your app (on iOS)

    For android, it depends because the Ti.Platfrom.id :

    • may be generated (=> same behaviour as iOS)
    • may be obtained from the UDID of the device (cf. Ti.Platfrom.id) (=> the Ti.Platfrom.id will stay the same)
