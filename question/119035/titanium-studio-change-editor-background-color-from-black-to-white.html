Titanium Studio change editor background color from black to white?

Hello dear Titans!

Anyone figured out how to change the background color of the editor in the new studio?

I know that it is suposed to be in the editor appearance tab, but I don't seem to be able to make the changes to append.

    Appearance tab isn't the right place for changing the editor's background color - you need to check out the Preferences -> Titanium Studio -> Themes section.

    There you can tweak all the colors (including background color) for your current theme, select another theme or create your own. For a quick change from dark to light theme, I suggest you try the Aptana 2.x theme.

    For more information check out the Wiki
