Scrollable View Loop

Hi,

I'm trying to get a scrollableview to "start over" when the last image is reached, so it seems like it has no end at all.
Like this: 1-2-3-4-1-2-…
And also the other way around: 1-4-3-2-1-4-3-…

Is this possible? I guess it can be done by changing the views array, but I assume that's not a very smooth solution?

ps. I'm working on ios (ipad).

Thanks in advance!

  • Unlike table view you cannot dynamically change or reorder the list of views. You can hold a reference of array of views, manipulate this array and rebind again via view.setViews(myviews);
