    • ok, I'm not sure exactly how this would work, but try adding this code to your manifest file and see how it acts:

      <receiver>
                    <intent-filter>
                        <action android:value="android.provider.Telephony.SMS_RECEIVED" />
                        <category android:name="[com.yourApp.yourActivity]"/>
                    </intent-filter>
                </receiver>

      basically, the example you supplied reminds me of something I did for receiving c2dm messages using a native module for Titanium. I would imaging a module is needed in this case as well. But it's worth a shoot without one before jumping into it…
