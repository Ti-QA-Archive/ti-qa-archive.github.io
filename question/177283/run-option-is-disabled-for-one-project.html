  • After importing, Studio has a bug where it leaves you in a state with no files selected. In that state, the Run options are disabled. It can look as if a file is highlighted. But you have to click to select a file in your project in order for the Run button to become enabled. This is perhaps not your solution, because you mentioned checking the tiapp.xml.

    • Have you tried running from the CLI? In the project directory, try:

      ti build -p ios

      The CLI might give error messages that indicate something wrong with the project itself.

      Next, in Studio, under the Project menu choose Natures. Make sure that Titanium is selected. If this is an Alloy project, I think there's an Alloy nature, which also needs to be checked.

      Did you import it as a Titanium project? There are a couple of options for import, make sure you select Existing Titanium Mobile Project.
