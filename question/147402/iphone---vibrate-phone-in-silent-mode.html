    • No, there is to programmatic way to determine or change this setting within appcelerator and certainly no way to do it in iOS. It might be possible to do it in android but you would have create a custom module if it is. Unlikely though.

      — commented January 28th 2013 by Malcolm Hollingsworth

    • As Malcolm said, there is no way to do this programmatically on iOS, but if its a necessary feature, you could try and walk the user through it.
      Create a button that would vibrate the phone. Ask the user if they felt the phone vibrate. If yes, you're good to go, if not, then give them directions on how to change the settings on their phone. If they change it, great, if not, then you're kind of out of luck on that front.
      In the end, if you are filling a need with your app, and they need it to function in a particular way, then they'll change the setting.

      — commented January 28th 2013 by Anthony Decena
