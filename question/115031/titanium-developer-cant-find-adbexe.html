Titanium Developer can't find adb.exe

I have problems to set the android sdk. The dev environment tells me not finding adb.exe or android.bat in /tools directory. But in SDK r09 downloaded from http://developer.android.com/sdk/index.html , adb.exe has been moved to /platform-tools. Any ideas to solve this problem?

Thanks
Lars

  • You need to "trick" Titanium in seeing the files in the &#x2F;tools directory. The best way to do this is to create a logical link between &#x2F;tools and &#x2F;platform-tools.

    Under Windows 7 or Vista

    cd C:\Program Files (x86)\Android\android-sdk-windows\tools 

mklink adb.exe ..\platform-tools\adb.exe 
mklink AdbWinApi.dll ..\platform-tools\AdbWinApi.dll

    Under Windows XP

    Since Windows XP doesn't support links, simply copy (not move) the two files (adb.exe and AdbWinApi.dll) into the &#x2F;tools directory.

    Under Linux

    cd [SDK HOME]/tools 
ln -s ../platform-tools/adb
