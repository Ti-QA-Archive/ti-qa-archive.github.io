  • Hi Cristina,

    Whatsapp uses a modified XMPP (Jabber) server so it's basically a sibling to Google Talk and many other services. So you have to implement/deploy a server to which your apps connect to, see http://www.jabber.org/ for reference, and use a PubSub API to send/receive messages. Or you can investigate some Chat APIs to see if some fits your needs, see this article for some directions: 57 Chat APIs.

    Hope it helps,
    Marco
