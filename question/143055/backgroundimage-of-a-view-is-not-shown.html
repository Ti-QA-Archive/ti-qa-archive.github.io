backgroundImage of a view is not shown

titanium:2.1.1
os: mac
platform: iphone simulator 5.1

My view has backgroundImage as a remote url, it's now shown. but if change the view to ImageView with a image property, it can be shown. The reason i don't use imageview is that the image view doesn't show the image in square.

var avatar1 = Ti.UI.createView({
    left:10,
    width:35,

    backgroundImage:post.user.photo.urls.thumb_100

// avatar show the image, but the image is not shown in square
var avatar = Ti.UI.createImageView({
        left:10,
        //top:10,
