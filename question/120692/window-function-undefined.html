    • no, currentWindow is a property letting you know what the current window is.

      you can set the title of the current Window 

      Ti.UI.currentWindow.title ="A Title";

      so based on your comment I think you are trying to do this

      var windd = Titanium.UI. currentWindow;
windd.title = "Window Title 1";

    • But If I create several windows, and then use Titanium.UI.currentWindow to get the current window, How can I know which one I get?

      — commented June 8th 2011 by Jiale He

    • currentWindow only works in a .js file which was assigned to a createWindow function

      var windd = Titanium.UI.createWindow({
   title:"window 1";
   url:"window_1.js"
});

      in window_1.js

      var windd = Titanium.UI.currentWindow;
Ti.API.debug(windd.title); // this should equal "window 1"

      See this lesson and download the associated code samples
