Android Tab Bar

I have an app – one that I'm trying to make cross-platform – whose core construct is a tab group with a single tab. It's built this way so that I get the nav bar with "back" navigation on the iPhone. On my home screen, though, I want to hide the nav bar since it has nothing to offer on that particular screen. To do all of this, I'm setting the tabBarHidden and navBarHidden properties to true.

As seems to be the universal case thusfar in my short Titanium development experience, this works great on iPhone, but bombs on Android. The tab bar doesn't hide. I now have a lovely grey bar, maybe 100px tall at the top of my screen. I've seen indicators that somehow the tab group/bar isn't a native Android construct, but nothing that really helps me work out how to hide it on this one screen of my app.

Any answers out there?

Thanks.

    • I'm sorry if I gave you the wrong answer previously, but I tried so many thing to get this thing working. that I lost track.

      I re-checked my code and I managed to get around the problem by directly opening the window when I'm on Android.

      Like so:

      // open tab group or window
if (isAndroid) {
    win.open();
} else {
    tabGroup.open();
}
