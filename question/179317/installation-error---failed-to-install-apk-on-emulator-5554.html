  • After much screwing around, I downloaded eclipse, went to android sdk manager, deleted the titanium AVD available. Closed Titanium. opened again, hit Run (play button), it complained about missing AVD. I went to the windows -> preferences - platforms - android and manage avds and then I saw the titanium qvga created again. hit close and run again. everything worked.
