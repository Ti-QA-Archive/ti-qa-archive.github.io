  • Hi,

    You can add 

    layout: "horizontal",

    Property in Scroll view, secondly, do not provide the left of the imageView which you are setting in image view. Because horizontal layout handle left of image automatically.

    For width and height you can give 

    Ti.UI.SIZE

    property of image view, it automatically set required width and height of imageView according to size of image.
