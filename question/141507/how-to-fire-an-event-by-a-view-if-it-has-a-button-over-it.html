how to fire an event by a view if it has a button over it?

I have a view, that has a swipe event added o it. However it is compleely covered by a button. ow can I enable the swipe event. The alternative would be hat I have a button and I add an invisible view over it, in order to handle events like swipe events. How can I make the button under the view able to respond to click events?

  • Ti.UI.Button also has a swipe event. Add your swipe event listener to the button, then re-fire it on the view:

    
var v = Ti.UI.createView({...});

var btn = Ti.UI.createButton({...});
v.add (btn);

v.addEventListener ('swipe', function (e) {
    // do whatever you want to do in your view in response to
    // a swipe here...
});

btn.addEventListener ('swipe', function (e) {
    v.fireEvent ('swipe', e);
});
