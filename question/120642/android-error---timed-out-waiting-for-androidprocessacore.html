  • Make sure you wait for SEVERAL minutes. I was seeing this error on Windows when I first messed with Appcelerator. It sounds like it didn't work… but just let it churn for a while longer and it (at least for me) will eventually start up.

    Once it is going then don't "stop" the emmulator, just click "launch" again on next update. It's just the intial boot up of the emmulator that is painfully long.
