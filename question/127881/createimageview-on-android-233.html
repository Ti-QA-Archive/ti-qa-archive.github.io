createImageView on Android 2.3.3

When I try to load an image from an url, it doesn't work on Android 2.3.3, but the same code works perfectly on Android 2.2

var img = Titanium.UI.createImageView({
url: "http://oradeahub.com/upload/anunturi/6863332_thumb.jpg",
height: height,
width: width,
left: 0,
top: 0
});

How can I solved the problem?
