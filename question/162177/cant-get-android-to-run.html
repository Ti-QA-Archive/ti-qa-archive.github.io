    Hi Keith,

    check the document Installing Oracle JDK for using and configuring the correct Oracle JDK.

    Check Installing the Android SDK for installing and configuration of the Android SDK.

    So for Titanium SDK Version 3.2.0 - latest, you need Min Android/SDK Version 2.3.3 (API 10).

    Android SDK Tools Rev 22

    Android SDK Platform-tools/Build-tools Rev 17.

    You are using:

     ?  tools              unsupported version 21.1
