  • The first and for most thing is that you should not use row's click event instead you can use table's click event to get selected row data.

    Second thing is you are creating new object of imageView in a loop so you should put imageView's click even in a loop.

    and if you want to get your image index as you did in image's click event…you should use e.source instead of using e.index.myId
