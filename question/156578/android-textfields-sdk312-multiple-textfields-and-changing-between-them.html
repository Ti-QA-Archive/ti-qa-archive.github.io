Android textfields, SDK3.12, multiple textfields and changing between them

Hi all,

Having major issues with textfields and Android.

I've two textfields for user registration, imaginatively named username and password

First issue. Clicking in field doesn't clear hinttext
Second issue. Trying to swap between fields doesn't work. As soon as you click into second field, the focus jumps back to the first! Need to do some frantic double tapping to get round this

Here's code

var self = Ti.UI.createView();
self.layout = "vertical";

var userName = Ti.UI.createTextField({
top : "16dp",
width : "75%",
borderStyle : Ti.UI.INPUT_BORDERSTYLE_ROUNDED,
hintText : "Enter a valid email address here",
keyboardType : Ti.UI.KEYBOARD_EMAIL,
});

var password = Ti.UI.createTextField({
top : "16dp",
width : "75%",
borderStyle : Ti.UI.INPUT_BORDERSTYLE_ROUNDED,
hintText : "Enter a password here",
passwordMask : true,
});

self.add(userName);
self.add(password);
self.add(register);
self.add(login);
self.add(accountHelp);

  • Running the following code without any problems (Ti SDK 3.1.2.GA Android: 4.0.4 Host: Mac)

    var win = Ti.UI.createWindow();

var self = Ti.UI.createView();
self.layout = "vertical";

var userName = Ti.UI.createTextField({
top : "16dp",
width : "75%",
borderStyle : Ti.UI.INPUT_BORDERSTYLE_ROUNDED,
hintText : "Enter a valid email address here",
keyboardType : Ti.UI.KEYBOARD_EMAIL,
});

var password = Ti.UI.createTextField({
top : "16dp",
width : "75%",
borderStyle : Ti.UI.INPUT_BORDERSTYLE_ROUNDED,
hintText : "Enter a password here",
passwordMask : true,
});

self.add(userName);
self.add(password);

win.add(self);
win.open();
