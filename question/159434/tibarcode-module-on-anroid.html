  • Accepted Answer

    if you are downloading from github appcelerator / titanium_modules/barcode then your tiapp.xml should be 

    <modules>
<module platform="iphone" version="1.8.3">ti.barcode</module>
<module platform="android" version="2.3.4">ti.barcode</module>
</modules>

    download iphone barcode module and android barcode module.

    unzip iphone and android modules and place in iphone and android folders accordingly

    modules ->iphone -> ti.barcode

    module -> android -> ti.barcode
