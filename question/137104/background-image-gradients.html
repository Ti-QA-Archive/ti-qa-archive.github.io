Background Image Gradients

It would be outstanding if, in a future version of Appcelerator, we could specify a backgroundImage as a CSS3 gradient and have it converted to a PNG on compile. Especially on buttons.

Thank you.

sdk 2.0.1, ios 5.1, os 10.7.3

    Processing gradients as background images is not usually something that is added to compile time processes in any application development environment as this is usually done during the design phase. However, if you like to work with images created from your own gradients, you can simply create a view, set the backgroundGradient on that view, then use the toImage() method of the view to generate and save an image. However, simply using the backgroundGradient is probably much more efficient. Check out the docs for the backgroundGradient on views: http://docs.appcelerator.com/titanium/2.0/index.html#!/api/Titanium.UI.View-property-backgroundGradient

    Also, if you need help turning your css3 (webkit more specifically) gradients into Ti gradients, check out this handy little tool: http://tech.ironhearts.com/sample/titanium/gradient/index.php
