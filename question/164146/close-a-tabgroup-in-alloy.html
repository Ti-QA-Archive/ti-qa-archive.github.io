Close a tabGroup in Alloy

Continuing my app conversion to Alloy framework. With thanks to Stephen I am able to open a tabGroup (about.xml) from a controller (dashboard.js) using this code segment:

case 'iconImage7':

                exports.tabGroup = Alloy.createController('about').getView();
                exports.tabGroup.open();
break;

Now trying to close the same opened tabGroup by clicking a menu button defined in about.xml using this code in about.js (controller):

function closeWindow () {
    $.exports.tabGroup.close();
}

Now I am getting an error of: not an object evaluating …tabGroup.close(). I have tried various versions of this line code without the $. and without the exports prefix thinking that its earlier declaration as a public var will let me use just tabGroup.

Thanks for your help with pointers on ow I might execute a call to close the tabGroup that is opened in one view from another view.

    Hi Adebisi,
    What you can do is , create a global variable like Alloy.Gloals.tabgroup. Then assign this variable reference of your tabgroup in the controller in whose xml you defined the tabgroup. The you can access it anywhere u want and even close it. Let me know if this helped
