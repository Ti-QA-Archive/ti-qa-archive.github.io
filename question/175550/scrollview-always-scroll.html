ScrollView Always Scroll

When using a ScrollView how do I make it so I can still swipe the view up and down and the y value will go below 0. It seems to only work when the content height is greater than the view height.

  • Correct. Scrollviews dont scroll if the contents dont need to.. If you want to scroll a larger area then set the contentHeight higher than the content…
