  • I ran into the same error when trying to run titanium setup on OS X and I think this is an issue when you don't have an existing Titanium SDK installed.

    In my case, I was running Titanium CLI 3.1.2. I had to run titanium sdk update --install to get a Titanium SDK (3.1.3.GA). After the installing the SDK, I was able to see that the question after "What would you like as your default locale?" was related to the default SDK to use (What Titanium SDK would you like to use by default?). Since the question listed my SDK installation, I'm assuming initially not having an SDK caused the TypeError.

    FWIW, I'm running node.js 0.10.19. But again, I don't think its a node version issue and is actually related to not having a Titanium SDK installed.
