const B = "123";

B  = "456"; // will not complain

alert(B) // Still shows 123 which means const works

    First, const is not part of the ECMAscript 5 specs.

    The v8 engine on android and the javascriptcore engine on ios are both at 5.1 specs.

    If it works, heh. But don't complain if it breaks.

    Yup, it kept its value. Did you try to redeclare it under both engines?
