  • Hi Peter, first of all you should calm down to sort out things sooner…i can understand what predicament you are in…..

    Now, if Studio is saying that there is no SDK 2.3.3 installed, then first of all open the SDK manager either from Studio or directly from Android folder…….. i don't know which OS you are using so please open the SDK manager from yourself….

    Then check that whether SDK 2.3.3 is installed or not….

    If it is installed then cross check all environment variables needed to setup Studio coz it may be possible that you have all SDKs installed but the path to Studio is not correct which is showing this error.

    I also faced this issue in beginning, almost every person face this, & the error was that path variables were not set & there were some component missing for the correct SDK version needed to run the app.

    I hope this might help you, if not then get back to us with some helpful screenshots of installed SDKs, environment variables paths & setting of paths inside Titanium Studio.
