Will Titanium apps growing to 64bit?

As the new Apple presentation shows, iOS7 supports 64bit. Will appcelerator do this also?

  • (I am posting this as a comment, since it is just a partial answer.)

    Many of the UI components and behind-the-scenes elements in XCode should have their internals already developed as hybrid 64-bit and 32-bit, so you will most likely have immediate support for apps run on the iPhone 5S (and maybe the next iPod Touch or iPad, if they use the A7). The developers of Titanium would just need to update their own built-in "bridge" modules to make full use of the 64-bit processor.

  • Hi

    As Titanium code is currently run through xcode and mapped onto the API endpoints, yes. This will happen behind the scenes and there should be no changes required by you for this to take place. Once the final xcode 5 build is approved for titanium use and your apps are built for iOS7 then this process should be automatic.
