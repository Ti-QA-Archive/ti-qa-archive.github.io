    I think I figured it out. It appears you can't use the install to device as it doesn't sign the app (or not the way Google wants you to). When I did a build for the Market and then installed onto the device using adb the message stopped appearing.

    • Look ok, wondering if some of the minSDK stuff I had for other issues maid a difference, here's mine:

      <android xmlns:android="http://schemas.android.com/apk/res/android">
        <tool-api-level>8</tool-api-level>
        <manifest android:installLocation="auto">
            <uses-sdk android:minSdkVersion="7"/>
            <uses-permission android:name="com.android.vending.BILLING" />
            <application>
                <service android:name="ti.inappbilling.BillingService"></service>
                <receiver android:name="ti.inappbilling.BillingReceiver">
                    <intent-filter>
                        <action android:name="com.android.vending.billing.IN_APP_NOTIFY" />
                        <action android:name="com.android.vending.billing.RESPONSE_CODE" />
                        <action android:name="com.android.vending.billing.PURCHASE_STATE_CHANGED" />

                </receiver>
            </application>
