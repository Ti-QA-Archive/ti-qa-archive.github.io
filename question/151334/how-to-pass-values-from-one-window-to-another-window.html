  • Hi Deepthi,

    In Alloy you can pass value using this : 

    var downloadOffreWindow = Alloy.createController('downloadOffre',{"categoryID" : catID ,"categoryName" : catName}).getView();

    other side : 

    var args = arguments[0] || {};

alert(args.categoryID);

    In normal structure use this for pass value : 

    var win = Ti.UI.createWindow({
    categoryID : catID,
})

    other side : 

    var win2 = Ti.UI.currentWindow;
alert(win2.categoryID)

    Thanks
