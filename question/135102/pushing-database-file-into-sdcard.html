Pushing database file into SDCard

Hi, I created an AVD with the size of 1GB in sdcard. Now i want to push my database file,which is having 30mb to the sdcard using command prompt. I know how to push into internal phone storage by using the cmd: "adb push [filename] /data/data". But i want to push it into sdcard as my database file is having more than 30mb. Can anybody please provide me the answer.
