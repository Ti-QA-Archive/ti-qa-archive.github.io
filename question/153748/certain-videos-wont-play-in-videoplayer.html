Certain videos won't play in VideoPlayer

I'm writing an app that plays Facebook videos inside a VideoPlayer control. There are times when certain videos just won't play while others will, and I can consistently reproduce this behavior both in the iOS simulator and on an iPhone running iOS6. When I create the VideoPlayer control, add it to a new Window and open that window, I just get a black screen for certain videos while other videos play with no problems. I've tried to add an event listener to the player's error event but the event never fires. I've verified that the URL is a valid one because I can download that same video to the device and save it to the camera roll. The only explanation I can come up with is that the URL that Facebook provides to me for a particular video changes over time, and certain permutations of that URL cause the VideoPlayer to be unable to play the video. Does anyone have any suggestions as to how I might fix this?

Here's the code:

function playVideo(url) {
    var player,
        win;

    win = Ti.UI.createWindow();

    player = Ti.Media.createVideoPlayer({
        autoplay: false,
        fullscreen: true,
        sourceType: Ti.Media.VIDEO_SOURCE_TYPE_STREAMING,
        url: url
    });

    win.add(player);
    win.open();
}
