Android 3.0 Hardware Acceleration

Hi all,

I'm wondering whether "Android 3.0 Hardware Accleration" does work in combination with a Titanium Mobile app? See here for more information:
http://android-developers.blogspot.com/2011/03/android-30-hardware-acceleration.html

Has anyone successfully added the hardwareAccelerated attribute to the manifest? Are there benefits or disadvantages?

Cheers

  • As long as the device supports the Hardware acceleration, this tag can be added to application node on manifest section:

     <application
   android:debuggable="false" 
   android:hardwareAccelerated="false"
   android:label="MBApp"
   android:screenOrientation="portrait">
