Stop the tableview scroll in Android

Hi,
I am trying to stop scrolling tableview inside view. I have done successfully in Iphone by using property Scrollable :true in table view. Im placing tableview inside view. Im trying to achieve same in android. But i couldn't able to stop scrolling table view inside view in android.   • The only way I know of to do this is to set the height of the table equal to the height of its rows so that there's no need for the tableview to scroll. Doing so is an extra challenge if you use calculated heights for the rows (Ti.UI.SIZE) as your code sample shows.
