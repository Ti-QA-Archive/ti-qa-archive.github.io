Max value in array

Hi.

I have this for my android gallery where the win.count is the number of pictures in the album showed.

 for (var i = 0; i < win.count; i++) {
        scrollView.add(square(i));
    }

FB will just show 25 pics so if there are more then 25 pics the value would 76 or similar and it will just get 25 so i will get an error.

Is there any way i could write the variable with a max value?

Perhaps something like win.count (max.value=25)

    How about:

    var i=0;
while(i<25) {
   scrollView.add(square(i));
   i++;
}

    ref: http://www.w3schools.com/js/js_loop_while.asp

    • You're welcome. Based on your subject line, some folks might come looking for a means to determine the max value of the members in an array. For that, try this:

      var arr = [1, 2, 3, 4, 5]; // some array of numbers
var maxVal = Math.max.apply(null, arr);
