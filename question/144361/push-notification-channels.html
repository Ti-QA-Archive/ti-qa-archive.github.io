Push Notification Channels

Hi,

I need to send push notifications to users that decide they want to subscribe to notifications for a specific object. My idea is to create a channel per object, and subscribe the user to a list of channels based on the objects they have subscribed to.

Is it possible to join multiple channels for notifications, or is there another method to achieve this?

Thanks!

    yes, you can suscribe and unsubscribe for specific channels and send notifications only to those channels using ACS subscribe and unsubscribe methods as described here. If you don't specify any channel when using unsubscribe, user is unsubscribed from all channels.
