Slider in AlertDialog?

I am trying to have a button launch a slider in order to have the user rate a product using the slider control.
Ultimately I would like to have a slider control hover. For now an AlertDialog with a slider would work.
I can't get the slider control to work inside the AlertDialog. Is this possible?

I have also tried putting the slider in a View within the AlertDialog with no luck:

<Alloy>
    <AlertDialog id="dialog" onClick="doClick" title="Rate" message="Rate" cancel="1">
        <Slider id="slider" top="50" min="0" max="5" width="100%" value="0" onChange="updateLabel"/>
        <Label id="label" width="100%" top="30" left="0" />
        <ButtonNames>
            <ButtonName>Confirm</ButtonName>
            <ButtonName>Cancel</ButtonName>
        </ButtonNames>
    </AlertDialog>
</Alloy>

This produces the following error:
[ERROR] : Script Error = 'undefined' is not a function (evaluating '$.views.dialog.add($.views.slider)') at ratingDialog.js (line 25).

Thanks in advance for any help!

  • It ended up being pretty simple once I figured it out. Here is the code if anyone is interested.

    View:

    <Alloy>
    <Window id="winRate" backgroundColor="#D1D0CC" height="110" width="200dp" 
            top="70" opacity="0.8">
    <Label id="label" width="100%" top="10" center="0" 
            textAlign="Ti.UI.TEXT_ALIGNMENT_CENTER"/>
    <Slider id="slider" top="40" min="0" max="5" width="100%" value="0" 
            onChange="updateLabel"/>
    <Button id="btnDone" onClick="done" title="Done" top="70" left="20" 
            height="30"/>
    <Button id="btnCancel" onClick="cancel" title="Cancel" top="70" right="20" 
            height="30"/>
    </Window>
</Alloy>

    Controller:

    function done(e) {
    //do something
    $.winRate.close();
};

function cancel(e) {
    $.winRate.close();
};

$.slider.text = $.slider.value;
function updateLabel(e){
    $.label.text = String.format("%3.1f", e.value);
}
