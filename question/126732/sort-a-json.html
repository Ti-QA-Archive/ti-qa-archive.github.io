    First, your JSON is not valid.

    Second, you can use JSON.parse, Array.sort and then, if you need you can get JSON string again with JSON.stringify

    Something like this:

    var reservations = JSON.parse(jsonString);
reservations.GetReservationsReponse.lesReservations.sort(function(a, b) {
    return (new Date(a.DateFieldThatYouDidntSpecify) - new Date(a.DateFieldThatYouDidntSpecify);
});

Ti.API.debug(JSON.stringify(reservations.GetReservationsReponse.lesReservations.join(', '));

    Just replace "DateFieldThatYouDidntSpecify" with one of four that there are.
