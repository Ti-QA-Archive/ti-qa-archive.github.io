How do I read characters from the blob returned by getFile on a UTF8 text file?

I've been struggling with this for 4 days now. I'm new to Appcelerator, but pretty good with Java and other languages.

I'm trying to read a text file in a mobile, Android app, but cannot find any way to slice the resulting string when I use the .text function. I am able to get the length of the string, and I can list it in Titanium Developer using the API info functions….

var f = Titanium.Filesystem.getFile(Titanium.Filesystem.resourcesDirectory, 'HDRS.txt');
Ti.API.info('file = ' + f.name);
var contents = f.read();
Ti.API.info('file = ' + f);
var contentText = contents.text;
Ti.API.info('Content Text is: ' + contentText);
var len = contents.text.length;
Ti.API.info('Content Text length is: ' + len);

Having got the file contents, I want to index through them, using something like….

for ( var i = 0; i < len; i++ ) {
    var char = contentText[i];
    Ti.API.info('Char = ' + char);

    // do something useful with substrings here
}

I cannot find a way to make the for loop work…whenever I try to index into the string, it just crashes on the Android emulator. I've tried many ways, and cannot get it after 4 days. At this point, I'm on the verge of giving up with Appcelerator completely, as this seems a very basic thing to want to do. Can anyone point me in the right direction? I must be missing something very simple.

TIA
Paul

    Hi Paul

    You can simply use the javascript charAt() method to obtain a string character at a specific position. The following code works as you intend:

    var win1 = Ti.UI.createWindow({
                title: "Main Window",
                exitOnClose: true,
                layout: 'vertical',
                navBarHidden: false
        });     

win1.open();

var f = Ti.Filesystem.getFile(Ti.Filesystem.resourcesDirectory, 'HDRS.txt');
Ti.API.info('file = ' + f.name);
var contents = f.read();
Ti.API.info('file = ' + f);
var contentText = contents.text;
Ti.API.info('Content Text is: ' + contentText);
var len = contentText.length;
Ti.API.info('Content Text length is: ' + len);

for (var i=0; i<len; i++){
    Ti.API.info('Char ' + i + ' : ' + contentText.charAt(i));
}

    A list of other Javascript methods for string manipulation is available in the Mozilla Doc Center.

    Hope this helps
