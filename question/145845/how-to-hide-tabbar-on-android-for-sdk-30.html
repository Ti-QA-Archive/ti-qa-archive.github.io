  • In SDK 2.1.4 or lower?Use "platform/android/res/layout/titanium_tabgroup.xml" hidden tabBar.

    <?xml version="1.0" encoding="utf-8"?>
<TabHost xmlns:android="http://schemas.android.com/apk/res/android"
    android:id="@android:id/tabhost"
    android:layout_width="fill_parent"
    android:layout_height="fill_parent">

    <LinearLayout
        android:orientation="vertical"
        android:layout_width="fill_parent"
        android:layout_height="fill_parent"
        android:padding="0dp">

        <FrameLayout
            android:id="@android:id/tabcontent"
            android:layout_width="fill_parent"
            android:layout_height="wrap_content"
            android:padding="0dp"
            android:layout_weight="1"/>

        <TabWidget
            android:id="@android:id/tabs"
            android:layout_width="fill_parent"
            android:layout_height="0dp"
            android:layout_weight="0"/>

    </LinearLayout>

</TabHost>

  • That's a very interesting article, but I think that doesn't quite solve the problem. Let me restate, as we're seeing this issue in SDK 3.0 as well.

    In SDK 2.1.4 and earlier, when you created a window, you could set the property of tabBarHidden = true. This would hide the tab bar. Now it does not hide the tab bar. So how do we hide the tab bar?

    Example code, which hides the tab bar in earlier versions but not in SDK 3.0:

    var win = Ti.UI.createWindow({
    title: 'My New WIndow',
    url: 'mywin.js',
    modal: false,
    tabBarHidden: true,
    navBarHidden: true,
    fullscreen: false,
    backgroundImage: '../images/background.jpg',
});

    • Oops, ignore my post above, we were actually using titanium_tabgroup.xml as well, which does not seem to be working in SDK 3.0.

      That file is placed under Platform/android/res/layout. Here is ours, which works in 2.1.4, not in 3.0:

      <?xml version="1.0" encoding="utf-8"?>
<TabHost xmlns:android="http://schemas.android.com/apk/res/android"
    android:id="@android:id/tabhost"
    android:layout_width="fill_parent"
    android:layout_height="fill_parent">

    <LinearLayout
        android:orientation="vertical"
        android:layout_width="fill_parent"
        android:layout_height="fill_parent"
        android:padding="0dp">

        <FrameLayout
            android:id="@android:id/tabcontent"
            android:layout_width="fill_parent"
            android:layout_height="wrap_content"
            android:padding="0dp"
            android:layout_weight="1"/>

        <TabWidget
            android:id="@android:id/tabs"
            android:layout_width="fill_parent"
            android:layout_height="0dp"
            android:layout_weight="0"/>

    </LinearLayout>

</TabHost>
