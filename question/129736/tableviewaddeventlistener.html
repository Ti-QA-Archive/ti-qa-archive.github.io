tableView.addEventListener('click',function(e){
var rowImage=Ti.UI.createImageView({image:e.rowData.rowImage});
var subWindow=CreateSubWindow(rowImage);

});

CreateSubWindow("Give me an image"); will be a function which creates sub window..
Can I access all contents of table row using e??

  • Accepted Answer

    You can use "e.row" to reference table view's row.

    In row creation code, you can add references of label and image to the row:

    row.Label = label;
row.Image = image;

    so it's easier to retrieve them in table view "click" listener.

    e.row.Label
e.row.Image

    If that's what you want. :)

  • You can definitely access all contents of row using "e". But for that you have to handle click listener of row not the table view.

    you should do like this,

    row.addEventListener('click',function(e){
var rowImage=Ti.UI.createImageView({image:e.rowData.rowImage});
var subWindow=CreateSubWindow(rowImage);
subWindow.open();
});

    conclusion is … you have to handle row's click event instead of tableview.
