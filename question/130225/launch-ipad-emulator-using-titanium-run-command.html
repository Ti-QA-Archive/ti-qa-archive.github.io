Launch iPad emulator using Titanium run command

Hi,

I'm developing an iOS module for iPad. For testing the module using the example from the example folder, I launch the app using titanium run in command line. But this launches the iPhone simulator, and I can't figure out how to launch the iPad version instead.

If I change the hardware option in emulator, I see the iPad emulated, but the app is opened with the iPhone emulation screen inside (and not fullscreen).

any idea?

Now I'm testing with a project from titanium studio, but I have to copy each time the module and recompile the project, wich is much slower than launch from command line.

would be nice to have some parameters in the titanium command to select emulator, i.e., titanium run --device:ipad --version:5.0

  • Here's what I do:

    • Close the simulator
    • Start the simulator using finder or the search (top right magnifying glass, type IOS and the simulator will show up.
    • Select Hardware/Device/iPad from the menu.
    • The next time you start the simulator from the Titanium Emulator, it will start with the last selected hardware.

    Ray Belisle
