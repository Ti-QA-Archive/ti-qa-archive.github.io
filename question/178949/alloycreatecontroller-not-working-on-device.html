Alloy.createController not working on device

Hi!
My app runs very fine on emulator, but when i install it on my device (iPhone 5s iOS 8.1) I get the following error:
"Couldn't find module: alloy/controllers/controllername"
Why? controller name is right!
I run on Mac OS 10.10 Yosemite, Titanium 3.4.0

  • Check the case of the filename and controller name. On the simulator, it doesn't care, on the device it does. So, if your controller name is MainScreen.js, calling createController('mainscreen') will work on the simulator. On a device you would need to call createController('MainScreen').

    Ray
