Getting Properties from TableViewRow

I am having troubles with a tableViewRow for some reason. It worked on 1.7.5 but now that I upgraded to 1.8.0.1 I am not able to pull the properties out of the row. I am running it on the Android emulator 2.2. I have made it as simple as possible but I still get an undefined property every time I try to get it.

The table will display the titles so I know that it is getting the correct properties sent to it. The table is getting its data through a loop that is adding all the row data to a data array.
my event listener is as follows: 

tableView.addEventListener('click',function(e){
    Ti.API.debug(e.row.tester);
}

tester is a junk property I added to see if I could get it. it has a value of 1 for all rows. But it doesn't matter what row i click on, they all come out undefined. same if I try to get the e.row.title.

The listener will work if I build for iOS.

        //connection to database here and query
    var data = [];

    while(result.isValidRow()){
        var plabel = result.fieldByName('primarylabel');
        var haschild = result.fieldByName('haschild');
        data.push({
            title:plabel,
            color:'#000000',
            hasChild:(haschild == 1) ? true : false,
            nextTitle:plabel,
            tester:1
            });

        result.next();
    }
    result.close();
    db.close();

    return data;
