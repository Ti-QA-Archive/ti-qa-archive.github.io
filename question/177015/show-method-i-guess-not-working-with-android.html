  • Hi,

    So this is a slight problem with Titanium SDK, where not all the properties are fully applied to every object.

    use 

    $.slidingWindow.visible = true;
$.slidingWindow.visible = false;

    There is a constant implementation of these across all view objects

    Hope this helps
