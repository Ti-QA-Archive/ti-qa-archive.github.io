TextField not showing in View

Hi,

The text field in the below code is not showing in the view

    var usernamecont = new ui.View({
        width: 280,
        height: 40,
        top: 200,
        backgroundColor: '#ffffff',
        borderColor: theme.appDarkGray,
        borderRadius: 3
    });
    var username = Titanium.UI.createTextField({  
        backgroundColor: '#ffffff',
        color : '#333333',
        top:200,  
        width:220,  
        height:40, 
        left:60,
        hintText:'Username',    
    });  
    var userimg = Ti.UI.createImageView({
        image: '/images/login_user.png',
        left: 10,
        height: 25,
        width: 25,
    })
    usernamecont.add(userimg);
    usernamecont.add(username);
    self.add(usernamecont);

However, if i change to to be included by self.add(username) then the textfield shows up, however I want it in the view.

Thanks

    Hi

    Prepare to kick yourself (I think).

    usernamecont height = 40
username top = 200

    User name is being added - but it is being clipped because you have told it to appear 160 lower than the actual available height.
