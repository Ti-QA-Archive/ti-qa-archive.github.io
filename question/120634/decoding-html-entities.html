Decoding HTML Entities

I am trying to convert HTML entities and this works in the browser but i'm not sure how to achieve the same effect without being able to create a textarea.

How can I achieve the same result in my TI code?

function html_entity_decode(str) {
    var ta=document.createElement("textarea");
    ta.innerHTML=str.replace(/</g,"<").replace(/>/g,">");
    return ta.value;
}

    It's easy. Remove "textarea" code, it's not necessary.

    
function html_entity_decode(str) {
    return str.replace(/</g,"<").replace(/>/g,">");
}
