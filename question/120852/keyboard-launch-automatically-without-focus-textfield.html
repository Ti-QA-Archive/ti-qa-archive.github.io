  • SOLVED: In tiapp.xml's Android tag <activity> set the android:windowSoftInputMode=&#39;stateHidden|adjustResize&#39;

    <activity android:configChanges="keyboardHidden"
                    android:name="org.appcelerator.titanium.TiActivity"
                    android:screenOrientation="portrait" 
                    android:windowSoftInputMode="stateHidden|adjustResize" />

  • I found the best solution for that problem:

    The first this:

    textName.addEventListener(&#39;focus&#39;, function() { configWindow.animate({bottom: &#39;30%&#39;, duration:500}); });

    And the end this:

    textName.addEventListener(&#39;blur&#39;, function() { configWindow.animate({bottom: 0, duration:500}); });

    This do that the windows compress and expand when you hide the keyboard
