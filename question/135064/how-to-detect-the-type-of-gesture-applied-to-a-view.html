How to detect the type of gesture applied to a view

Titanium Mobile SDK 1.8.2

iPHone SDK 4.3

Hi,

I am creating a view that holds several photos. How can I create a listener that listens to both 'click' and 'swipe' gesture? Both performing different actions without overlapping the other?

    hi Nash,

    try this …

    var view = Titanium.UI.createView({
    borderRadius : 10,
    backgroundColor : 'white',
    width : 250,
    height : 250
});
var x;
var y;
view.addEventListener('touchstart', function(e) {
    x = e.x;
    y = e.y;
});
view.addEventListener('touchend', function(e) {
    if(x == e.x && y == e.y) {
        alert('CLICK');
    }
});

view.addEventListener('swipe', function(e) {
    alert('<< SWIPE >>');
});
win.add(view);
