Failure [INSTALL_FAILED_OLDER_SDK] when running test project

2 December 2011

After following the setup instructions for Android development on Windows 7 Getting Started with Titanium Studio, building a new project and then running it in the Android emulator ends with the error Failure [INSTALL_FAILED_OLDER_SDK].

Here are my specifics:

  • Windows 7 64 bit
  • Titanium Studio, build: 1.0.6.201110251616, SDK version 1.7.5
  • JDK version 1.6.0_29 installed into C:\Program Files (x86)\Java\jdk1.6.0_29
  • Android SDK installed to C:\Program Files (x86)\Android\android-sdk
  • The following Android APIs installed: 1.6 (API 4), 2.1 (API 7), 2.2 (API 8), and 3.2 (API 13), as well as the Android SDK Tools, Android SDK Platform-tools, and all of the Extras (as prescribed in Installing and Updating Android SDK ).
