  • It's not what Titanium is really about, but yes, you could probably do this. You'd use the WebView to load the HTML of your web site. The files could be stored locally on the mobile device or remotely on the web. There are limitations which you'll have to live with or work around. You'll very likely need to update the HTML/CSS to render nicely on small devices like handhelds. And you won't have easy access to device features like the camera, GPS, etc.

    This is the model that Phonegap/Cordova uses (along with frameworks that build atop Cordova, such as Sencha, jQuery Mobile, etc.) Titanium is about creating native apps .. you just happen to use JavaScript rather than Objective C or Java. Titanium does not use HTML & CSS (except as noted above).
