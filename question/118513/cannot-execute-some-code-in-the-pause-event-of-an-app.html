Cannot execute some code in the 'pause' event of an app

Hi guys,

I have the following code:

var blackView = Titanium.UI.createView({
    backgroundColor:'#000000',
    view.zIndex = -999
});
Titanium.UI.currentWindow.add(blackView);
Titanium.App.addEventListener('pause', function() {
    Titanium.API.info('1');
    blackView.zIndex = 999;
    Titanium.API.info('2');
});
  • When I close the app:

I can only see the '1' in the Titanium console.
I suppose that the black view kind of stop the event to run

  • When I open the app:

I want to see the black view straight away as I changed the zIndex. BUT the black view only appears after a second or 2. Then I can see the '2' in the Titanium console
It seems like all of the actions on the UI are executed when we open the app (and not when we close the app)

My problem is that I want to PIN lock my app, so I want to show this black view during the 'pause' event so no one can see the content of the app right after re-opening it

My question is: How can I make this black view appeared when I close the app.

Thanks for your help
