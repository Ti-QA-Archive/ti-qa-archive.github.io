  • Hi,
    You can do something following to write the content to file :

    var txtField=Ti.UI.createTextField();

var f = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory,'emptyfile.txt');
f.write('txtField.value);

    Hope it helps.
