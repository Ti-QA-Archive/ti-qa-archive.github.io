Splash screen incorrect res folder

Hi people!
I've created Splash Screen for my app through http://ticons.fokkezb.nl . On smartphone, splash screens are ok, but on tablet they look out of resolution (it seems it loads default.png splash screen). How can I fix that? Thank you!

  • Accepted Answer

    There are no established patterns for the naming of the drawables folders for tablets. Hence, there's no way to specify the correct graphics to use for the varied tablet sizes/densities/aspect-ratios. Google and our docs both recommend using 9-patch images instead of static graphics for splash screens (on tablets especially, but phones too).

    Fokke's command-line version of TiCons supports creating 9-patch images but his web version does not. See our docs for the manual steps you would take to create them.
