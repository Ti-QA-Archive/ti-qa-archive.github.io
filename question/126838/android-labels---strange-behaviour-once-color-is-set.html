Android Labels - Strange behaviour once color is set?

Hey there,

Having found strange issue.

Some of my Labels have a click eventListener - however I've noticed if I add a color to the label it'll react slow.

eg
With the following code - the alert doesn't always pop up sometimes I find myself spamming clicks to get it working.

   var lbl = Ti.UI.createLabel({color: "#000"});

   lbl.addEventListener('click', function(e) {
     alert("k");
   });

Where as, this one works all the time without fail.

   var lbl = Ti.UI.createLabel();

   lbl.addEventListener('click', function(e) {
     alert("k");
   });

Can anyone shed some light?
