    • Hi Luciano,

      You can use pause and resume event for calling callback function when the app is in background and you can alo use background service for calling callback function.

      Try like this :

      Ti.App.addEventListener('resume', function(e) {
    //When app is resume then this event is fired
});


Ti.App.addEventListener('pause', function(e) {
    //When app is in pause state  then this event is fired
});

      If your app is going in backgroung then you also use the background service.
