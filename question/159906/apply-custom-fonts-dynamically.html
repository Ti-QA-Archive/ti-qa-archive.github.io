Apply custom fonts dynamically

I'm creating a page dynamically by retrieving data from JSON. Each time a Ti.UI.Label can have other font which user has added in backend.

I can store font files in local storage but I can't apply this font to the label from there.

Does anyone has an idea how I can manage it?
It should works on both platforms

Thanks in advance

  • At this point in time, with Titanium, no. Custom fonts have to be declared at build time so they are placed into the info.plist for the app (on ios).

    You could build your own module, with your own Ti.Label type implementation and load the fonts dynamically that way.

    iOS already has a ton of fonts available by default http://iosfonts.com/

    Android, not so much.

  • You have two options.

    1. package the fonts into your app

    It is not possible to add new fonts dynamically. You have to ship it within the ipa / apk. Of course, it is possible to download a font file but I'm afraid there is no way to use it on a lable.

    2. use a webview

    This seems to be the most dynamically solution to use remote fonts without downloading them. Do it with HTML / CSS.
