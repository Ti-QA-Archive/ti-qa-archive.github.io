  • Accepted Answer

    I would suggest moving the 'click' event listener into a function in the nav.xml controller (nav.js) and then use a custom Ti.App event listener to listen for events. Then in your other view controllers, use Ti.App.fireEvent to communicate the data you need to the nav.js controller

  • This seems to work nicely, thank you. Heres the abridged code for anyone else looking

    nav.js

    
var menuToggle = false;

Ti.App.addEventListener('toggleMenu', function(e){
    // If the menu is opened
    if(menuToggle == true){
        $.nav.animate({
            left:  "100%",    
            duration: 300,
            curve:Ti.UI.ANIMATION_CURVE_EASE_IN_OUT
        });

        menuToggle = false;
    }
    // If the menu isn't opened
    else{
       $.nav.animate({
            left: "20%",
            duration: 400,
            curve:Ti.UI.ANIMATION_CURVE_EASE_IN_OUT
        });

        menuToggle = true;

    }
});

    nav.xml

    
<Alloy>
    <View id="navView"  layout="vertical" zIndex="1000">
        <View class="navRowBorder"></View>
