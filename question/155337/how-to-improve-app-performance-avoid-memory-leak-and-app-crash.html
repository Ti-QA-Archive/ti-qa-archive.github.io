How to improve app performance avoid memory leak and App Crash

Hi All,

How to improve Titanium app performance, Avoid memory leak and App Crash?

Other than this Coding_Best_Practices if know anything else please share here,

I think this will improve app performance,

  1. Use try/catch only where required
  2. Don't call Ti.platform multiple time, instead assign in one variable and reuse it
  3. use Webview only when required and if you use it, call webView.release() after to release memory
  4. Dont Keep more windows in stack , Load when required
  5. Keep String constants in /i18n/en/string.xml

Thanks,
