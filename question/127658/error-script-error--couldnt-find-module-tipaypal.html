  • 1.Make sure the modules added is same level with Resource folder, LICENSE.txt, CHANGELOG.txt, tiapp.xml. How to add and install modules into project.

    2.Added the line below :

    /*--- Remove the line <modules/> if occur ---*/
    <modules>
    /*--- Example below is Paypal module version 1.4 for iPhone ---*/
        <module version="1.4" platform="iphone">ti.paypal</module>
    </modules>

    3.Clean the project and recompile / rebuild.
