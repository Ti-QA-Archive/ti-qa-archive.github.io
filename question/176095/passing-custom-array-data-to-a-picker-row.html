    • and one more thing..

      in your code, you are putting the condition for the Picker value

      if (view.children[i].apiName === 'Ti.UI.Label' && view.children[i].id.indexOf('heading') >= 0)

      Suppose, first time, this condition is true, then value of i is 0 and currentRows is executed

      currentRows[i] = Ti.UI.createPickerRow({title: view.children[i].text}); //i=0

      Now, Suppose second time this condition is false, then value of i is 1 and currentRows is not executed..

      So array is, 

      //currentRows[0] = "blah blah"
//currentRows[1] = undefined or null

      and we must need to provide the Array to Picker without undefined or null value..So, picker will be exicuted as it needed..
