  • Hi Marwa mamon,
    Since you are using windows 7,
    Go to your command prompt. by type cmd on your Run.

    And then go to your android sdk folder-> platform-tools
    like below

    C:\Users\Simbu> cd C:\ android-sdk-win\platform-tools

    Then you have to run the following two commands on the command prompt separately

    adb kill-server

    adb start-server

    Then you will get this

    * daemon not running. starting it now on port 5037 *
* daemon started successfully *

    Now you can run your App into Android emulator. You could use this at any time when you are getting Emulator failed to start in a timely manner Error
