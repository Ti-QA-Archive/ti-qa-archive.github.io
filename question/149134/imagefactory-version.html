  • Hi

    If you only need the standard cropping, resizing functions then you can use the Titanium built-in functions now available to both platforms;

    • imageAsCropped
    • imageAsResized
    • imageAsThumbnail
    • imageWithAlpha
    • imageWithTransparentBorder

    Example usage;

    var img = Ti.UI.createImageView({
    height: Ti.UI.SIZE,
    image: 'example.png',
    width: Ti.UI.SIZE
});
var cropped = img.imageAsResized(400, 300);
var thumbnail = img.imageAsThumbnail(50);
// etc
