  • Hi

    There is no such thing - the goal posts on this are constantly moving. You should concentrate on what you can do and not what you cannot do.

    Be aware that with the future version of Titanium (code named Ti.Next) brings Hyperloop as part of it. With Hyperloop you will be able to access ALL APIs, even if Titanium does not directly support them, due to the new architecture you will be able to connect directly with the native API but still remain in JS. This will be 100% optional - if you never need or never want to delve that deep - that is your choice, but you will be able to.
