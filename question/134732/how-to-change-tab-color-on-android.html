How to change tab color on android??

Hi all,

I currently develop the android app using titanium mobile, I try to change the look of tab, however, The property "tabBackgroundColor" seem not work in titanium, is there any other way to implement it?? Thx

  • Tab colors on android are controlled by the apps theme, or device theme.
    The ability to control these settings have as of yet not been included in the API.

    You ought to be able to change some of the settings with a custom styles.xml or layout.xml. However, a lot of the hacks depend on actual java changes that would require hacking your SDK.
