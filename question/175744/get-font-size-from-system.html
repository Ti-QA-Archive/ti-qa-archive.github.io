Get font Size from system

Is possible to get font size from system in IOS 7, i want in my app font size of the labels change when changes the text size in settings (Settings>General>Text Size>larger sizes) in device, there is some function or way to get?

Titanium Studio, build: 3.2.3.201404181442 XCode Version 5.1.1 (5B1008)

    Hi

    To achieve your end goal, you do not ask what the font size is, you simply set each label to a new iOS 7+ font constant and then each time the user makes those changes! your app adjusts automatically. You will have to make sure you do not fix heights for labels though as the font size will change the amount of space required.

    • textStyle

    When using this you do NOT set any other font property as this handles how each style is presented to the user.

    Example usage;

    var Ti.UI.createLabel({
    font: {
        textStyle: Ti.UI.TEXT_STYLE_BODY
    }
});

    Or Alloy TSS syntax 

    "Label": {
    font: {
        textStyle: Ti.UI.TEXT_STYLE_BODY
    }
}
