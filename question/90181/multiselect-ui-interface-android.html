Multiselect UI Interface (Android)

Hi,

For an Android app I'd like to have a dialog that pops up and allows a user to select 1-X items in a list (that will have about 10-15 total options). So far it does not seem like there are any APIs that help out with this. My current thought is using a separate window to pop up a tableview and handle the selection of items via the tableview. Is there a better/simpler way that I'm missing?

— asked December 15th 2010 by Frank A

    You are correct, doing a tableview will be the way to do this. You'll be opening a 'modal window', which has the effect of a native dialog. Something like (untested, but should be basically right):

    var pop = Ti.UI.createWindow();
pop.open({modal:true});
