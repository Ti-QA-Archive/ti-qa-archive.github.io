  • First off you should really look at the KitchenSink for some examples of window opening. Because it doesn't look like you understand what is going on.

    You never assign Titanium.UI.currentWindow, you reference it. So you would have var win = Titanium.UI.currentWindow, then you can do things like add views on top of the current window.

    Briefly looking at your code it has some issues. You should be able to put the following in your button event listener and open a window.

    var winWebView = Ti.UI.createWindow({
    title: 'Web View',
    url: 'webView.js'
});
winWebView.open();

    If you are trying to open the window in the current tab, you need to:

    var winWebView = Ti.UI.createWindow({
    title: 'Web View',
    url: 'webView.js'
});
Titanium.UI.currentTab.open(winWebView, {animatied: true});
