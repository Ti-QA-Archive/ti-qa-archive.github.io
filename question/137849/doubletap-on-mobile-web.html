  • You can simply listen for the doubletap event. You don't need to do anything special. Note that when you double tap, it will also fire single tap events too. Some UI elements, like WebView, have undefined behavior with the touch gestures since they are competing with other gestures.

    var myView = Ti.UI.createView({ /* args go here */ });
myView.addEventListener("doubletap", function(evt) {
    Ti.API.log("Double tap");
});
