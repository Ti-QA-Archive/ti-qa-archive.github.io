Moving To Common JS modules

Application Type: Mobile
Titanium SDK :3.1.1
Platform :IOS,Android
I am trying to move to common JS Modules from the initial Include modules

I have an application that had few .js files that used based on Ti.include(); strategy ,I am trying to move to common Js modules where I wish to use require and make it more object oriented .

Please let me know how I should go about with this

    It depends on the content of the js files.

    In case of helper libraries you can use the following structure:

    exports.fn1 = function(a,b,c){
    // your function here
}

exports.fn2 = function(a, b, c){
    // your second function
}

    In case the js file exposes a View, you can do the following:

    function myJs(a, b , c){
    var self = {};
    var win = Ti.UI.createWindow() ;// and you continue your view code

    // access the view through your self object
    self.getView = function(){
        return win;
    };

    self.doSomethingWithObject(a,b,c){
        // Here you can add other functions to communicate and control the view
    };

    // and at the end return the self object
    return self;
}

module.exports = myJs;
