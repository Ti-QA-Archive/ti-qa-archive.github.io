  • Here is appcelerator Database Guide.

    Follow the steps

    1) Create database (.sqlite) file and copy it into your resource directory

    2) Install Database

    var db = Ti.Database.install('database.sqlite', 'database');
db.close();

    3) Perform Query like

    var conn = Ti.Database.open('database');
var q = 'SELECT * FROM tablename';
var resultSet = conn.execute(q);
conn.close();

while (resultSet.isValidRow()) {
//store data
var val1 = resultSet.fieldByName('field1');
}
