Alloy not working in Android.

Hi all. I am making application for iOS and Android. In iOS is no problrem but In Android works unexpected behavior.

I reappear the behavior following this:

  1. Run titanium create and Alloy new.

  2. Edit controller/index.js following this:
    ~~~
    function doClick(e) {
    alert("1");
    alert("2");
    alert("3");
    }

$.index.open();
~~~

  1. Run "titanium build -p android".

I expected alert "1", "2", "3", but alert "3", "2'", "1". Like this, execute function in inverse order. Do I something wrong?

Environment: Titanium 3.0.24, Alloy 1.0.0, Android SDK 4.2 and 2.3
