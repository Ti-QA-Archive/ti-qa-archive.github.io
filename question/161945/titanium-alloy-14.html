  • Hi

    Some point in the future.

    If that seems vague it is meant to be. There are no public road maps for upcoming versions. This is less about not wanting to be transparent and more about the complexities of making changes, updating features, removing bugs and managing all the parts that make up Titanium.

    So in short there are no firm public dates, there are internal dates they aim for (I am not privy to these and if I was would not be able to share them) but as I say other factors affect things all the time.
