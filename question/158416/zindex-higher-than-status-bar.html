  • Hi Achim Hoth,

    If you don't want statusbar display then you can hide it,so it will not display in your screen.

    In your project folder there is a tiapp.xml file, in that file there is one tag like this.

    <statusbar-hidden>false</statusbar-hidden>

    you need to change false to true like this.

    <statusbar-hidden>true</statusbar-hidden>

    after that save and clean your project and remove application icon from your simulator and run again.You will not see statusbar in your top of window after doing this.

    • Hide statusbar when you don't want using this code 

      Titanium.UI.iPhone.hideStatusBar();

      show when you want to display again.

      Titanium.UI.iPhone.showStatusBar();

      try this demo code to hide and show on click event.

      var viewTop = Ti.UI.createView({
    height : 50,
    width : 320,
    top : 0,
    backgroundColor : 'green'
});
viewTop.addEventListener('click', function(e) {
    Ti.App.isSetStatus = !Ti.App.isSetStatus;
    if (Ti.App.isSetStatus)
        Titanium.UI.iPhone.hideStatusBar();
    else
        Titanium.UI.iPhone.showStatusBar();
});
Ti.UI.currentWindow.add(viewTop);

    • Hiding/Showing the statusbar on the fly is not working in iOS7:

      > On iOS 7, the status bar cannot be controlled on the fly with Titanium.UI.iPhone.setStatusBarStyle, Titanium.UI.iPhone.setStatusBarHidden, Titanium.UI.iPhone.showStatusBar or Titanium.UI.iPhone.hideStatusBar. You need to either set the fullscreen property to hide the status bar or statusBarStyle to change the status bar style before opening the window.

    • You can set your window top using below condition.Try below code you may don't need to hide statusbar after trying this.

      function isiOS7Plus() {
    // iOS-specific test
    if (Titanium.Platform.name == 'iPhone OS') {
        var version = Titanium.Platform.version.split(".");
        var major = parseInt(version[0], 10);

        // Can only test this support on a 3.2+ device
        if (major >= 7) {
            return true;
        }
    }
    return false;
}

var iOS7 = isiOS7Plus();
var theTop = iOS7 ? 20 : 0

var window = Ti.UI.createWindow({
    top : theTop,
    backgroundColor : 'red',
    statusBarStyle : iOS7 ? Titanium.UI.iPhone.StatusBar.LIGHT_CONTENT : Titanium.UI.iPhone.StatusBar.TRANSLUCENT_BLACK
});
window.open();
