Can we apply styles in tss for dynamic components generated in .js files.?

Hi all !

I am generating a component in a .js file.

var imageView = Ti.UI.createImageView({
    id: "img"
});

Now can I apply styles in .tss using its id like this?

"#img": {
    height: 80,
    width: 80,
    image: "/images/img.png"
}

TIA

Maruti.

    Hi

    There are a couple of ways to do it but first you need to know everything is about adding classes, your TSS shows an id reference which should only affect one item and dynamically created views do not have ids, so simply change your code as follows.

    ".img": {
    height: 80,
    width: 80,
    image: "/images/img.png"
}

    Then in your controller;

    var style = $.createStyle({
    classes: 'img',
    apiName: 'ImageView'
});
imageView.applyProperties(style);

    You can also addClass and removeClass, more details and example;
