Can't get Hello World to work on Android

I have been trying for two days now to get the Hello World example to run in an Android emulator but I'm completely stuck. I have been searching the Q&A section for possible solutions but nothing so far has worked.

I'm running Windows 7, 64-bit and Titanium Studio 3.2.1

I followed the instructions in the Quick Start guide to install Titanium Studio and then create the Hello World example. From what I can read the installer downloads and installs all the necessary third party stuff (Java, Node, Git) and it seems to have been installed properly. I left all install options as default. I used Titanium Studio to get and install the Android SDK (again using default settings as presented in Studio).

When I try to run the project on Android (using e.g. the 'Galaxy Nexus' option) I just keep getting an error 'Application Installer abnormal process termination'. The whole console output is available here: console.txt.

Here's the output from 'ti info': ti_info.txt

The console output looks like it's trying to start an AVD with the name 'Galaxy Nexus' but Android manager doesn't allow me to use spaces in the name for an AVD som not sure how that fits together.

Any ideas before I get rid of this application forever?

    Hi

    The Android Emulator is a horrible horrible thing. I gave up on it some time ago as you spent more time trying to get it started than you do on the app itself.

    Two solutions - I recommend using both;

    • Physical Device

    Connect a device up using USB and you can install directly to the device. You need to enable developer mode - a quick google will explain how for you device, but usually the settings - about page and then multiple tapping the last entry - it then tells you it is enabled. Only problem though - you may not have enough devices to match the things yu want to test for. So…

    • Genymotion

      A much better emulator that everybody loves - it now works with Titanium 3.2 and higher;

      Using Genymotion with Titanium 3.2
