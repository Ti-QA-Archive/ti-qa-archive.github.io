    Hi Infy

    I haven't tried this but putting the piece of code together using the API docs.

    I found this code from one site for the native and its say its uninstall the app its on this link

    Uri packageURI = Uri.parse("package:com.android.myapp");
Intent uninstallIntent = new Intent(Intent.ACTION_DELETE, packageURI);
startActivity(uninstallIntent);

    based on the above code and with the help of API docs trying a piece of code "Not Tested" i am just making the guess.

    var intent = Ti.Android.createIntent({
    action: Ti.Android.ACTION_DELETE
});
intent.putExtraUri("package.name");
intent.addCategory(Ti.Android.CATEGORY_DEFAULT);
Ti.Android.currentActivity.startActivity(intent);

    well i don't know this is working or not but i guess at least you might get the direction from this.
