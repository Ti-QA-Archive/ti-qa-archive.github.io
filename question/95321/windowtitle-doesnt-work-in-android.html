  • Fernando

    It's good practice to include the Google and Titanium SDK versions when posting a question, because it can make a big difference to the answers.

    This issue is related to setting the title attribute of lightweight windows at creation, and ticket #2361 has been opened to address it.

    Until it's fixed, you can either make the window heavyweight, by using a modal attribute like navBarHidden:false or create the window and set the property afterwards, such as:

    var win1 = Ti.UI.createWindow({
    backgroundColor: "red",
    exitOnClose: true
});
win1.open();
win1.title = "This is the window title";

    Hope this helps

  • hi! You should declare a label and then add to your window for example:

    var titleLabel = Titanium.UI.createLabel({
    color:'#000',
    height:18,
    width:210,
    top:10,
    text:'',
    textAlign:'center',
    font:{fontFamily:'Trebuchet MS',fontSize:10,fontWeight:'bold'},
    shadowColor:'#eee',shadowOffset:{x:0,y:1}
});

// associate label to title
win1.setTitleControl(titleLabel)

    hope it helps
