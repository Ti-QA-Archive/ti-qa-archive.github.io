    What you are seeing is a current bug that is being worked on. I know this because I have been waiting for a solution to this myself. This single thing make any apps look weird.

    I have tried every single beta that has been added in the last few days and so far the work that has been reportedly fixed has yet to make it into a merged live build.

    I imagine the problem will be resolved very shortly.
