How to open link in browser window using remote jQueryMobile website in webview

I have created an app and my app.js is very simple (it just loads a remote website in webview):

var webview = Titanium.UI.createWebView({url:'http://www.website.com/mobile.html'});
var thewindow = Titanium.UI.createWindow({
    fullscreen:true,
    modal:true,
    tabBarHidden:true,
    navBarHidden:true
});
thewindow.add(webview);
thewindow.open();

I use the jQuery Mobile 1.1.1 framework to make it look all pretty. My problem is that I'm trying to get some simple HTML external links to open in the mobile browser (instead of inside the app) and no matter what I do, the links only open inside the app full-screen and therefore lock the user out of going back to the app. Here is what my current external link looks like:

<a href="http://www.google.com" onclick="window.open('http://www.google.com', '_blank')" target="_blank" rel="external" data-ajax="false">www.google.com</a>

So far I have only tested it in the iPhone/iPad simulator and have not had a chance to test it on a real device to see if it behaves differently.

Anyone have any ideas on how to make these links open in the mobile browser and not within the app itself?

Thanks.
