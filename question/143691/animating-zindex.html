Animating zIndex.

Hi everyone!

I'm having a problem when animating using zIndex property, because it doesn't have the expected behaviour.

I first create a view.

var self = Ti.UI.createView({
        height : '100%',
        width : '100%'
});

Inside the view I add a veil "not visible". This is just a black view.

var veil = Ti.UI.createView({
        backgroundColor : 'black',
        height : '100%',
        width : '100%',
        touchEnabled : false,
        opacity : 0.,
        zIndex : -10
});
self.add(veil);

Then I add an image thumbnail to self.

var image = Ti.UI.createImageView({
    backgroundImage : 'myImageThumbanil.png',
    zIndex :  2
});
self.add(image);

And now I want two animations. First one, the image is gone be scaled and put it on top. Second one, the veil is gone be visible but BELOW the image.

image.addEventListener('click', function(e) {
    veil.animate(Titanium.UI.createAnimation({
                            duration : 200,
                            opacity : 1.,
                            zIndex : 3
    }));
    image.animate(Ti.UI.createAnimation({
                            duration : 300,
                            transform : zoomTransform,
                            zIndex : 20
    }));
});

But, the result is that the veil is ON TOP of the scaled image and the image is BELOW the veil. Why?
