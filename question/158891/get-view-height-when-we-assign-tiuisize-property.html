Get View Height when We Assign Ti.UI.SIZE Property

Hello,

I want to get view height while views are created asynchronously and I have set view.height = Ti.UI.SIZE. I want to get view height each time after objects are added into it. I have also used "postlayout" Event also but it does not gives me actual height of view after addition of other objects into it.
