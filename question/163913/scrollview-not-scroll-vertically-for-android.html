Scrollview Not Scroll Vertically For Android

Scrollview is not scroll vertically for android

widget.tss

"#scrollview": {
    left: 0,
    top: 0, 
    right: 0,     
    contentWidth: Ti.UI.FILL,
    contentHeight:'auto',
    showVerticalScrollIndicator: true
}

For IOS Its scrolling vertically,
when run on Android its not scrolling.

widget.js
$.scrollview.add($._buttons[index].b);

  • Hi

    Ideally you should always provide the full basic properties for scrollView and avoid using keywords that are no longer considered correct.

    "#scrollview": {
    contentHeight: Ti.UI.SIZE,
    contentWidth: Ti.UI.FILL,
    height: Ti.UI.FILL,
    showHorizontalScrollIndicator: false,
    showVerticalScrollIndicator: true
    width: Ti.UI.FILL
}

    Be aware that scrollView will not scroll the contents of a view if the total height of the child views does not exceed the height of the scrollVIew.

    In English; the inside views need to take up more space than the scrollView or they will not show any scrolling animation.

    For the above TSS to work you MUST have a scrollView tag to work in the XML;

    <ScrollView id="scrollview>
  <Label text="Hello">
  <Label text="World">
</ScrollView>

    Something like this but with your child views.
