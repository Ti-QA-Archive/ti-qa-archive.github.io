Hyperlink text in Titanium

I'm pretty new to app development, and I'm creating a tabbed Android application. I'm trying to carry out what I suspect is a very simple task in hyperlinking some text. I would like to create a label such that when the text is clicked, the website opens up in the user's browser. I've searched for some tutorials online but can't quite seem to work out how to do it. Thanks in advance, and apologies for what is probably a very simple question!

    Hi

    In your click event you need to use this method Ti.Platform.openURL() and pass the url you wish the link to point to. Unfortunately there is no standard styling pre-defined for this, so if you want to style it to look like a link then you will have to do that yourself.

    Hope this helps.
