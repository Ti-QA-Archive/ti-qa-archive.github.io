    If you're just trying to go back to the homescreen, you don't recreate the entire thing again. Simply close the current window and the original window (the homescreen) will be displayed again.

    • Again, close the windows that have been opened over the homescreen and you will be back at the homescreen. In any user interface, to get back to the main screen, you simply close or hide the windows that you've already opened. This is the same thing. Close or hide the windows that you've already opened and you'll get back to the main screen.

      If you were to keep re-creating the homescreen and opening new ones, then all other windows would do the same. It would not take very long before you had too many windows open and the performance of the app suffered.
