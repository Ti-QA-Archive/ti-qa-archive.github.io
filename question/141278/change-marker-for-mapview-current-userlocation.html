Change marker for Mapview current userLocation

The mapview works fine and displays the current user's location with a blue default pin. I want to use a custom marker image instead of that blue pin, does anyone know how to do this for the userLocation?

I can set the userlocation to false and add my own custom marker to the current position where the device is detected, but then I don't know how to move that marker when the user moves.

Any of those two scenarios would work.

Thank you.

  • Hi

    You cannot change the default marker unfortunately, but you are close with your idea.

    Create an annotation with the exact same lat/lon and set it to the current location. But instead of a pin use the image property, this will lock image to the map when it moves.
