  • Hi,

    navBarHidden:true is not work for Titanium 3.3.0

    For hiding the Application BarName, Please use the below code.

    function inizializzaElementi(e){
    var width=Ti.Platform.displayCaps.platformWidth;
    $.username.width =width-40;
    $.password.width =width-40;
    $.login.width = width-40;
};

var win=$.index;
win.activity.actionBar.hide();
win.open();
