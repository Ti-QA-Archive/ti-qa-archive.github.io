  • Here is the compiled ti.paint module for Android which was recently added to github: https://github.com/appcelerator/titanium_modules/tree/master/paint/mobile

    I don't believe the modules for iOS and Android are identical, as the example for Android has more options like changing alpha and stroke width and an erase mode, but I'm sure if you need to you could recompile the iPhone source and add more features.

    Here is a zip of both compiled modules with an example application http://www.iammarkgoldsmith.com/titanium/Ti.Paint_Module.zip

    Be sure to add both modules to your tiapp.xml file like this:

    <modules>
     <module version="1.0" platform="iphone">ti.paint</module>
     <module version="1.0" platform="android">ti.paint</module>
</modules>
