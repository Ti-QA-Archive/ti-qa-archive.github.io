How to Remove an Image?

Hi

I feel stupid for asking but how to i remove an image from an imageView?

If i have:

var imagePhoto = Ti.UI.createImageView({width:587, height:881, top:0, left:0, image:'image.png'});

and now in my app i get to a point where i have to remove the image.png - so keep the ImageView itself but just set the actual image to nothing.

I've tried both:

imagePhoto.image=' ';

imagePhoto.image=null;`

but I always get this warning outputted:

[ERROR] invalid image type. expected either TiBlob or TiFile, was: NSNull  in -[TiUIImageView setImage_:] (TiUIImageView.m:690)

Is there a way of safely setting the image to nothing while still keeping the actual ImageView or is my only option to remove the ImageView itself and add it again next time it's needed?
