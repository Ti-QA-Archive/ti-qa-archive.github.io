Application path change when install to iOS device

I have an application need store a file path to sqllite database, form example save an image path to sqllite as file://localhost/Users/user01/Library/Application Support/iPhone Simulator/5.0/Applications/F85D40FF-CD88-4BB7-8025-DC92F1866F70/Library/Private Documents/abc.png

but when i build to iOS device second time, the folder F85D40FF-CD88-4BB7-8025-DC92F1866F70 has been changed to a new one, but my sqllite still store the old one, so that i can't access my photo after build to iOS device second time, any one idea to solve my problem? or the logic of store the image path to database is wrong? thanks.
