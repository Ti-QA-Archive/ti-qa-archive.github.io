Contacts API

This question is two fold:

1) how do you get the ID for a contact to use the get contactbyID api call? The person object does not seem to have an ID field, and if you try to iterate over its properties (for p in person) to find any undocumented properties, the app crashes.

2) Why is there no find contactByPhoneNumber? I would think this would be a common issue where you have a phone and you want to get a list of the contacts that have that number.

— asked January 3rd 2011 by Kouroche Sedaghatian

        Ti.API.info(e.person.recordId);
    }
