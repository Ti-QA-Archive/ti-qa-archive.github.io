Disabling keyboard on search bar focus?

Hi,

Love love love the Titanium! Have been enjoying coding with it :)

Question:
Is there a way to disable keyboard from displaying when search bar gets focus?

The reason I want this to happen is I push values to search bar from picker and filter a table view rows this way.

(If it's not possible to do this without keyboard popping up, is there better way to filter table rows with picker??)

  • >Is there a way to disable keyboard from displaying when search bar gets focus?

    Not that I know of, but if you create a listener for the focus event of the search bar, you should be able to just immediately do a blur() method to make the keyboard go away. I am not sure if this will cause any screen flicker though. :(
