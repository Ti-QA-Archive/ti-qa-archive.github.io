    • Of course it can be done using just Views (anything is possible after all)
      But using windows will ensure that the memory is released properly when you close a Window. If you used Views, you would have to manage these yourself.

      Also, if you want to use transitions between screens or things like that. Windows will allow you a lot more flexibility in that regard,

      Finally, Windows are the de facto UI item to use when you want to user more complex navigation mechanisms like TabbedBars or NavigationGroups.
