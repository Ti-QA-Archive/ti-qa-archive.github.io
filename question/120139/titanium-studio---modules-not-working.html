Titanium studio - modules not working

Hey,

I added the my existing project that has external calendar module to titanium studio but when i build my project it cannot find the calendar module.

i get this:
[ERROR] Script Error = Couldn't find module: com.ti.calendar.

It works fine on the titanium developer but not on studio.

Is there a different folder to store the modules in for Titanium studio?

  • Ok,

    It seems that (at least in my case) Titanium Developer used the folder:
    MYUSER/Library/Application Support/Titanium/modules/iphone/
    to get the modules.
    but Titanium Studio is using:
    Macintosh HD/Library/Application Support/Titanium/modules/iphone/

    Hope it helps to others who have that problem
