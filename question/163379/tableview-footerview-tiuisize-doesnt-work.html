TableView FooterView Ti.UI.SIZE doesn't work

Using iOS on Titanium Appcelerator 3.2.1

After multiple tests, it seems as if the Ti.UI.SIZE property for the FooterView defaults to the size of the parent TableView which means I'm unable to have footers that are larger than the height defined in the paret TableView. You might suggest simply defining the TableView with a height of Ti.UI.SIZE, but that won't work as the TableView doesn't add the size of the HeaderView and FooterView, so I have to use Ti.UI.FILL as the size of the TableView, which creates the above problem.

Does anyone know of a way to either 1) Integrate the HeaderView and FooterView height into the Ti.UI.SIZE of a TableView; or 2) Ignore the parent TableView height when defining the variable height of the FooterView. If I define it with a fixed height, it works fine. Unfortunately, I don't know what the height will be so it needs to be variable.

  • Hi,Scott Blum…To Integrate the HeaderView and FooterView height into the Ti.UI.SIZE of a TableView you can try this code:

    
var win = Ti.UI.createWindow();
win.open();


var header = Ti.UI.createView({
    height:Ti.UI.SIZE,
    backgroundColor:'green'
});
header.add(Ti.UI.createLabel({
    text:"approaches to creating and using TableView object.approaches to creating and using TableView object.approaches to creating and using TableView object..",
    top:0,
    height:Ti.UI.SIZE
})); 

var footer = Ti.UI.createView({
    height:Ti.UI.SIZE,
    backgroundColor:'red'
});
footer.add(Ti.UI.createLabel({
    text:"approaches to creating and using TableView object.approaches to creating and using TableView object.approaches to creating and using TableView object.",
    top:0,
    height:Ti.UI.SIZE
})); 

var tableData = [ {title: 'Apples'}, {title: 'Bananas'}, {title: 'Carrots'}, {title: 'Potatoes'} ];

var table = Ti.UI.createTableView({
    index:[{index:0,title:'a'},{index:1,title:'b'}],
    data:tableData,
    headerView:header,
    footerView:footer
});

win.add(table);
