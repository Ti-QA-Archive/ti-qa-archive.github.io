    Hi

    To resize an image you can use one of the many Titanium modules;Image Factory;

    • Titanium modules

    Download option on right and install like all other modules.

    newBlob = ImageFactory.imageAsCropped(blob, { width:100, height:100, x:50, y:50 });
newBlob = ImageFactory.imageAsResized(blob, { width:140, height:140, quality:ImageFactory.QUALITY_HIGH, hires:true });

    Two example functions available.
