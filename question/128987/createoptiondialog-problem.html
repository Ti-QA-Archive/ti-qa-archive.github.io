createOptionDialog problem.

Hi, I am using Android SDK and Titanium to create an app for the university scope.

After creating a "createOptionDialog" that I spent an array of data, I want to match a click event for each element of my array.

There any way to do it?
I want to individually control the various elements of my array?

var opp = new Array(2);
opp[0]="New photo";
opp[1]="Select from the library";

var options = Ti.UI.createOptionDialog({
    title: 'choice',
    options:opp, 
    cancel:1,
});
options.show();

options.addEventListener('click', function(e){
    if(...)
    { alert("One");    }
    else 
    { alert("Two");}
});

Thanks
Good Work

  • Accepted Answer

    Check for index in the event data:

    options.addEventListener('click', function(e){

    if(e.index === 0)
    { alert("One"); }
    else 
    { alert("Two");}
});
