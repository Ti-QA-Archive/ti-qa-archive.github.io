[Alloy] defining global Styles per Variable?

in my application i have different style properties - for example fix color-codes, that are used several times in the app.

Is there any "alloy" like way to store and handle them? What i am thinking about is a way to use variables in my *.tss files, but this does not seem to be possible ?!

Any hint would be great

  • You can use properties from the Alloy.Globals and Alloy.CFG namespace in your TSS files. So you could set a value in your "app/alloy.js" file and then use it in any style.

    alloy.js

    Alloy.Globals.someColor = '#fff';

    index.tss

    "#someLabel": {
    color: Alloy.Globals.someColor
}
