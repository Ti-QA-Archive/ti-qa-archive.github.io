I am creating an app that uses geolocation. I need heading, location and gyroscope values.

I put my iPhone on table to stay still and values are not changing (as expected) except x,y, and z values of gyroscope. They are acting crazy. They constantly change although iPhone is not moving.

What could be the cause of this? I need them to be correct.

  • Hi, I use this for my Accelerometer, and it works perfect.

    AppName.Properties.Accelerometer = function() {
    Titanium.Accelerometer.addEventListener('update',function(e)
    {
        Ti.API.debug("accelerometer - x:"+e.x+",y:"+e.y+",z:"+e.z);
        return e;
    });
}

    I believe the Accelerometer is that sensitive, that's why it is always moving. It senses the smallest movement/vibration/etc. I can paste in some sample code if you would like.
