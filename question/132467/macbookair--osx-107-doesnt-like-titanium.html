  • I have the exact same problem. This has occurred for me since the first time I tried downloading Ti Studio when it was first released last year. Happens every single time I download/install it (have tried 4 times). Happened on previous MBP and current one, on both Snow Leopard and Lion. My current MBP 17" has 16GB ram.     UPDATE: Just found another post that suggests turning off Titanium Studio -> Preferences -> Team -> Git -> [X] Automatically attach our git support to projects …

    (see http://developer.appcelerator.com/question/130542/titanium-studio-freezes-on-mac-osx)

    That seems to have worked!
