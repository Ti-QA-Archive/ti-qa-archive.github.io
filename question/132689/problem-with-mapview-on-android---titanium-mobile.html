Problem with mapview on Android - Titanium Mobile

Hello Dear Friends

I have an ask or problem, I'm try to develop an application where i use a mapview, the aplication is for android, but when i test the mapview, doesnt work, only show the google maps logo in the left bottom and the typical grid used on google maps, but never show the map, already try test on the simulator also on a physical device, but the result is the same.

rare is that on IOS devices works fine.

the code that i use is the next:

var win = Titanium.UI.currentWindow;

var mapview = Titanium.Map.createView({
    mapType: Titanium.Map.STANDARD_TYPE,
    region:{latitude:33.74511,
        longitude:-84.38993,
        latitudeDelta:0.5,
        longitudeDelta:0.5},
    animate:true,
    regionFit:true,
    userLocation:true,
    annotations:[atlanta,apple]
});

win.add(mapview);

I do not know if I have to add something to work on Android

I hope I can help, no more for now thank you very much for the help.

    As Alan Said, you are missing Google API keys in your tiapp.xml. You should have one API key for development and one distribution.

    According to my experience if you copy paste development google api key from tiapp.xml of Kitchensink than it will work fine in emulators for development of your application. You have register one for distribution of your application. For this you can follow the guide (link provided by alan)

    And yahh as brian said you have to use APIs version of android SDK.
