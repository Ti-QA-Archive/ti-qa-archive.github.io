  • Accepted Answer

    This is something that forces the full build each time:

     Forcing rebuild: custom Info.plist CFBundleIdentifier not equal to tiapp.xml <id>

    check the bundle id in tiapp.xml with the one in the custom info.plist.
    After this is fixed, clean the project and rebuild.

    The first rebuild will be slow, but the next ones should be faster.
