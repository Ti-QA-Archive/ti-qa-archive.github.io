    Hi

    Some time ago I had similar problem, the only way to find the problem was to build for ad-hoc put that on my device before I could find the actual issue. I am not sure why that gave me more information but it did.

    My problem (in case the one in a million chance it is similar) was an extra trailing comma in an oauth library we did not write.

    So we found code like this;

    var lbl = Ti.UI.createLabel({
  height: 40,
  text: 'hello',
  width: 100,
});

    Note the trailing comma in this example - that was our problem and a lesson well learned.

    Again this was my problem and may have no relationship to yours but - just in case.
