Asynchronous toImage not responding

Hi,

I have been using view.toImage() in the synchronous manner where I do not pass a callback function and it has been working fine. I am now trying to switch to the asynchronous usage and am having issues. Here is some example code:

Ti.API.info('start');
var tagBarView = Ti.UI.createView({
    height: 10,
    width: 10,
    backgroundColor: "black"
});
tagBarView.toImage(function(blob) {
    Ti.API.info('image created');
});

"Start" is printed to the logs, but "image created" is not. I am using the Ti 3.0 SDK with Alloy on android. The asynchronous functionality does not seem to be working. Any ideas?
