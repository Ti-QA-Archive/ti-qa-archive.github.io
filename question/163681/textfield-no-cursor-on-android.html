  • Hi please try to use the below code, as simple as possible like,

    var win = Ti.UI.createWindow({
  backgroundColor: 'white'
});

var textField = Ti.UI.createTextField({
  borderStyle: Ti.UI.INPUT_BORDERSTYLE_ROUNDED,
  color: '#336699',
  top: 10, left: 10,
  width: 250, height: 60
});

win.add(textField);
win.open();

    This works fine for me and should work for you too.
