    The widget is meant to be a very simple example of how widgets are composed. It does not, and likely will not, provide a lot of customization. This may be all moot, though, as the new ActivityIndicator in Titanium 3.0.0 will do what this widget can do and more. Check out the docs for the new ActivityIndicator API here: http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.ActivityIndicator
