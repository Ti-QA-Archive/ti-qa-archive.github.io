  • Hi,

    Web views are more expensive to create than other native views because of the requirement to load the HTML browser into memory so.

    Alternative option is you can use default browser of phone. 

    Titanium.Platform.openURL('http://www.example.com');

    But one problem is their

    if the device has two browser and you execute "Titanium.Platform.openURL('http://www.example.com');" it would force user to choose one browser . if you have no problem with this than problem solve.
