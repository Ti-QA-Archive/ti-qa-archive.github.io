Different style sheets for controller

Hi ,

I am new to appcelerator. I am trying to use different style sheet for a single controller .
Suppose , I have a controller A.js , the corresponding tss file is A.jss , view is A.xml. I have another style sheet B.tss.

Will it be possible to include A.tss and B.tss for the same controller with a certain condition .
Example ,
if (true)
{ include A.tss }
else
{ include B.tss } .

Please help .

Thanks in advance .
Shreyas

  • Hi Shreyas,

    I think there is no any method to use two tss for single controller.
    but if you want to put condition in your .tss file then its is possible.
    e.g

    A.tss

    "#imgList[if=Alloy.Globals.is7InchTablet]":{
    top : 20,
    left : 0,
    right : 0
    }

Other than that, use the various selectors (platform, formFactor, and custom TSS conditionals as shown by Nilesh) or override styles in your JavaScript.
