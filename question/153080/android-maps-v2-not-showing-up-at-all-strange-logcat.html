  • Accepted Answer

    Aha… Sorted… Kind of… So, to make the window actually show up, I needed to make the new window a heavyweight window.

    Modal makes for a black screen on maps v2 due to a bug. However, if you use navBarHidden:true to get a heavyweight window it works.

    Hopefully anyone using this solution doesn't need the nav bar :)
