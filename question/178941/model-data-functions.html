Model Data Functions

So I have a model called acontacts, and I want a custom function in the model to transform the data for viewing. Then I want to use that function to display data in the view.

custom function from the model:

    extendModel: function(Model) {
        _.extend(Model.prototype, {
            // extended functions and properties go here
             customFunction: function() {
                return "I am a book model." +  this.get('lname');
            }
        });

view code:

<TableView dataCollection="acontacts">
            <TableViewRow title="{customFunction()}" ></TableViewRow>
</TableView>

Obviously I have copied this from one of the alloy samples…. But "customFunction" or "customFunction()" doesn't return anything to the view. Wat am I missing?

Thanks

  • try using data transformation function instead?

    <TableView dataCollection="acontacts"  dataTransform="transformFunction">
   <TableViewRow title="{customValue}" ></TableViewRow>
</TableView>

    in the transformation function, you can call the function on the model I believe

    function transformFunction(model) {
    // Need to convert the model to a JSON object
    var transform = model.toJSON();
    transform.customValue = model.customFunction();
    return transform;
}
