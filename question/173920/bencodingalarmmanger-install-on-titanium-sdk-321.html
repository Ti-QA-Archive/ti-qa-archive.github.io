    I'd think one of these two options:

    • not defined in tiapp.xml properly
    • not installed in the modules folder properly

    However, you state you can run the example code ok.

    • I have solved the problem. When unzip de module in the folder "dist" are many distributions, I have chossed de last "0.9", and oly unzip this folder in the folder modules in "C:\Users\myuser\AppData\Roaming\Titanium\modules\android" Then install de module in titanium and modify tiapp.xml whith version 0.9.
