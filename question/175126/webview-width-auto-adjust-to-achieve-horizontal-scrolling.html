Webview width auto adjust to achieve horizontal scrolling

hello i am displaying dynamic content on web view, i want web view to auto adjust width according to content so that whenever content is more of screen width there must be horizontal scrolling so that user can scroll it horizontally and see the rest of data.
i am using this code but there is no horizontal scrolling.

var scrollView1 = Ti.UI.createScrollView({
                contentWidth : 'auto',
                contentHeight : 'auto',
                top : 1,
                showVerticalScrollIndicator : true,
                showHorizontalScrollIndicator : true,
                height : '100%',
                backgroundColor : '#FFFFFF',
                width : '100%'
            });

            var chartView = Ti.UI.createWebView({
                height : 700,
                width : "100%",
                left : 0,
                top : 0,
                showScrollbars : false,
                touchEnabled : false,
                url : 'graph.html',
                backgroundColor : "white"

            });

            scrollView1.add(chartView);
            win.add(scrollView1);

    First you do not need to add web view to any scroll view, because it automatically support scrolling.
    Second scrolling can happen only when the content shown is greater than phone screen width or height.
