Aligning an object in the middle of the screen

Hey Guys,

I'm been away from Titanium for a while and after a couple updates of the Studio my code seems to be a little off. Now all the objects that i used to align in the center of the screen with "left:0, right:0" now doesnt work anymore. Anyone could help me out what is the solution for that?

Basically i have an image that i need it aligned in the middle of the screen. In the past i have the following code:

    var logo = Ti.UI.createImageView({
        image:'/images/logo.png',
        width:138,
        height:43,
        left:10,
        right:10,
        top:10,
        bottom:10
    });

This would place the logo exactly in the middle, now its on the top left and that's it.

Any thoughts?

    Don't include any position setting. Remove the top, bottom, left, right…by default it will position in the middle.
