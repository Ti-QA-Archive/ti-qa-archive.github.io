  • Hi

    The console is debug log and not an audit trail of everything. You should be using this to provide debug information for your code and not entire copies of huge amounts of data.

    If your web service is passing that amount of data your app should validate the information and present checksums, validity checks in the console only as seeing that amount of data is not going to help anything.

    If you are testing that the web service is actually working then you should be testing this in a web browser of other testing tool to validate.
