hasCheck and custom check image in a row table

Hello,

In mobile app I use hasCheck property in a tableview row.
The problem is that my backround and the check image is quite dark also so not easy to see it.
Is there a way to change the color of the check image or use a custom check image ?

Regards

  • the hasCheck, renders the system check image in the rightImage location for the row. To use your own image you can do something like this.

    var row = Ti.UI.createTableViewRow();
row.rightImage = '../images/myCheckImage.png';
