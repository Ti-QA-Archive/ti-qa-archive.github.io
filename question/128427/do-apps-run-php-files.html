    The mobile platform cannot run PHP. If you are keen to use it, there is a module that uses a subset of php and is based on http://phpjs.org/

    Here is the module:

    https://github.com/appcelerator/titanium_modules/tree/master/php

    Keep in mind that is not a full stack php, but more a library that tries to emulate php with js.
