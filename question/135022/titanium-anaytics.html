  • To activate your app analytic you can log in to your appcelerator account. there you will have Apps section, you can see all your app listed there that were build by you..
    Listed App will have the option "Launch Analytic". that will help you to find that which user clicked which events etc..

    for that you have to configure your app first. see tha API doc for Titanium.Analytic implementation.

    or if you dont have to use the titanium analytic you can use the google analytic for the same purpose,..
    here is the sample application that implements google analytic..
