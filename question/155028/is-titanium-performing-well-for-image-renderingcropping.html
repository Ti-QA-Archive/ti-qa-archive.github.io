Is Titanium performing well for image rendering/cropping?

Hi,

we have an App for Android and iOS to order imaging products like prints, posters, photo books, and others. This means a lot of cropping high quality images and rendering images into each other for collages. The apps are able to handle images with 12+MP which consumes a lot of memory especially when creating collages. Memory usage on iOS easily gets up to 100-200MB and can also hit sometimes 300+MB.

Since we are thinking about supporting Windows Phone 8, we thought about using cross platform frameworks like Titanium to ease the development.

Our iOS App can be found at

  • USA: https://itunes.apple.com/us/app/photo2home/id601397001?mt=8

  • Australia: https://itunes.apple.com/au/app/photo2home/id601397001?mt=8

Does anyone have an opinion if Titanium can provide what is needed for our App?

Regards,

Chris
