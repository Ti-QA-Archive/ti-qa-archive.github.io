Pre made sqlite database inaccessible

Hi,

I'm working on an iphone app on Titanium Studio 3.1.2.201308091617

I've created an sqlite database with Firefox's sqlite Manager.
I've added it to my resources folder.
The file is called myname.sqlite and as far as I know its actual name is myname but I'm not 100% sure cause I cannot check it.

I'm trying to call it this way:

var db = Ti.Database.install('myname.sqlite', 'myname');
var res = db.execute('SELECT * FROM mytable');
        while (res.isValidRow())
        {
        Ti.API.info('*');
        }

But after a few hours hitting a wall, I still get errors. Mainly
Invalid SQL statement at app.js
and
com.plausiblelabs.pldatabase.error.query.string=SELECT * FROM mytable, com.plausiblelabs.pldatabase.error.vendor.string=no such table: mytable}

I've tried a few different things including this

Ti.Database.install('myname.sqlite', 'myname');
var db = Ti.Database.open('mytable');  
var res = db.execute('SELECT * FROM mytable');

to no avail.

  • I think you are missing something out. You should open the database which you have created. In your case, database name is 'myname' and you are trying to open a table named 'mytable'.

    Try following code,

    var db = Ti.Database.install('/myname.sqlite', 'myname'); 
db.open('myname'); 
var res = db.execute('SELECT * FROM mytable');
