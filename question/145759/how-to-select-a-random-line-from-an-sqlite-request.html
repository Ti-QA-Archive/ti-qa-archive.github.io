How to select a random line from an sqlite request

Hello everyone,

I am trying to select a random record from a sqlite request, like that:

var rows = db.execute('SELECT * FROM testdb ORDER BY RANDOM() where level="'+ levelq + '" LIMIT 1' );
    if (rows.getRowCount() > 0){
        while (rows.isValidRow()){
        labelq.text=rows.fieldByName("question");
        }
    db.close();
    }

What I wrote, unfortunately, didn't work!!! but I hope that you have understand what I want to do (The SELECT request will return many records, I want just to obtain one of thess records)

Any suggestions?

Thanks!

    Hi

    Your sql statement is the wrong way round. Order By is after Where clause.

    So your code becomes;

    var rows = db.execute('SELECT * FROM testdb where level="'+ levelq + '" ORDER BY RANDOM() LIMIT 1' );
    if (rows.getRowCount() > 0){
        while (rows.isValidRow()){
        labelq.text=rows.fieldByName("question");
        }
    db.close();
    }
