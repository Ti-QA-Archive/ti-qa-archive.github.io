How do I pass Integers to intents?

I'm trying to enable a user to crop his user image using android intents and it works fairly well except for the fact that I have to live with the defaults. When I run the code below the activity pops up as expected but the extra data for aspect and output size is ignored

    var intent = Ti.Android.createIntent({
        action: "com.android.camera.action.CROP",
        data: 'file:///mnt/sdcard/download/images.jpeg',
        type: 'image/*'
    });
    intent.putExtra("crop", "true");
    intent.putExtra("aspectX", 1);
    intent.putExtra("aspectY", 1);
    intent.putExtra("outputX", 60);
    intent.putExtra("outputY", 80);
    intent.putExtra("return-data", true);

When I check the logs it becomes apparent that the intent gets a double and defaults to 0 for all values. This isn't strange since javascript only has a double numeric type but since the intent really needs integers I'd like to know if there's any way to send something in that titanium will interpret as an Integer in the Kroll layer.
