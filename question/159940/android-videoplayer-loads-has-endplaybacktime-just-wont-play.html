  • Video is really weird on Android. For instance you can't record a video on iOS and play it on Android. It's very stringent with regards to encoding etc. of the files it will play.

    Here's a few ideas for troubleshooting:

    1. Try settings scalingMode to Ti.Media.VIDEO_SCALING_ASPECT_FILL when creating the videoPlayer
    2. Try adding complete callback - does it get called?
    3. Add the 'interrupted' callback - does it get called?
    4. Android has trouble playing videos with large dimensions, like 2000*1800 pixels, etc. - try experimenting with this, maybe scale your video down.
    5. Also some issues with videos and modal windows - try adding the videoplayer to a simple view, and to a simple window - experiment.
    6. Having a videoplayer in a view with rounded corners, and other things, can (not sure if this has been fixed) also cause black screen playback
    7. Experiment with putting the video file in other locations, for instance directly in your resources dir and using url: "/myvideo.mp4"
