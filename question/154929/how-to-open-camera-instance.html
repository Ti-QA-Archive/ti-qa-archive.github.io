    When it comes to getting hands on with the API, nothing beats Kitchen sink or the alloy tests and samples for right now.

    https://github.com/appcelerator/alloy/tree/master/samples
    https://github.com/appcelerator/alloy/tree/master/test/apps

    kitchensink:
    https://github.com/appcelerator/kitchensink

    Specifically what you are looking for.
    https://github.com/appcelerator/KitchenSink/blob/master/Resources/ui/common/phone/camera.js

    One note on KS (kitchensink), the the way the code is structured is horrible. Use the KS as an executable api documentation only. Copy and pasting large sections of KS is sure to make a kitten somewhere in the world cry.
