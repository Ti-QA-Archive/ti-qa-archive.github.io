HTTP client trouble

Hello, I am trying to connect to a database and I am repeatedly getting an 'undefined' alert with httpClient errors saying "setting ready state to 2,3,4". All my php code is correct, am I perhaps coding something wrong in titanium? Thanks!

var loginReq = Titanium.Network.createHTTPClient();

loginReq.onload = function()
{
var json = this.responseText;
var response = JSON.parse(json);
if (response.logged == true)
{
alert("Welcome " + response.name + ". Your email is: " + response.email);
}
else
{
alert(response.message);
}
};

loginBtn.addEventListener('click',function(e)
{
if (username.value != '' && password.value != '')
{

username: username.value,
password: Ti.Utils.md5HexDigest(password.value)
};
loginReq.send(params);
}
else

alert("Username/Password are required");
}
});

  • You might want to try moving all of your http client code under the button. This would ensure everything is in the same scope. I've also found order matters when adding your open and error statements. KitchenSink has a pretty good example here https://github.com/appcelerator/titanium_mobile/blob/master/demos/KitchenSink/Resources/examples/xhr_fileupload.js You want to look at line 26 thru 48
