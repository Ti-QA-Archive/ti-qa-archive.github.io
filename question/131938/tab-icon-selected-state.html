Tab Icon Selected State

Has there been any progress in this area: Android Tab Icon Guide "Providing Icons for Two Tab States" ??

If not, then is it even possible to change an Android icon after it's been added to a tab? I have this code:

tabGroup.addEventListener('focus', function(e) {
    Ti.API.info("CHANGING TAB");
    var tab = tabGroup.activeTab;
    Ti.API.info(tab.icon);
    tab.icon = 'KS_nav_ui.png';
    Ti.API.info(tab.icon);
});

The string changes like it should (the event IS coming), so I see something like:

  • CHANGING TAB
  • icon_zentai.png
  • KS_nav_ui.png

However, the actual tab never changes…the next time I click that tab the output is:

  • CHANGING TAB
  • KS_nav_ui.png
  • KS_nav_ui.png

Do I have to give up on two tab states?
