Android SDK path recognized in Preferences->Android but not Preferences->Titanium Studio->Titanium

I have an Android SDK directory with a number of releases in it. In Preferences->Android these show up fine. If I set the Android SDK Directory to this path under Preference->Titanium Studio->Titanium I get a "Could not locate the Android SDK at the given path" error.

Why?
