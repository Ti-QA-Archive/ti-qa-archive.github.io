Image zoom

Hi all,

I am using below code to zoom image on the clicked position. The problem is on click of left corner(south east) the image zooms to right corner(north west) and all other sides it works fine.

var image_view = Ti.UI.createImageView({
            image :image_path,            
            width : GetWidth(320),
            height :GetHeight(320),            
            //canScale : true,
            //enableZoomControls : false,            
            zIndex : 0
        });
        view_imgfull.add(image_view);

        var center1 = {y:160,x:160};
        var zoomScale = false;
        image_view.addEventListener('doubletap', function(e)
        {
        var t = Titanium.UI.create2DMatrix();
            if (!zoomScale){    
            t = t.scale(2.0);
            var newY = image_view.height-e.x; 
            var newX = image_view.width-e.y; //alert(e.x); alert(e.y);

            image_view.animate({transform:t,center:{y:newY,x:newX},duration:500});
            zoomScale = true;

            }else{

            image_view.animate({transform:t,center:center1,duration:500})
            zoomScale = false;
            }
            Ti.API.info('Our event tells us the center is ' + e.x + ', ' + e.y );

        });

And when i change the below code 

  var newY = image_view.height-e.x; 
   var newX = image_view.width-e.y;

to

  var newY = e.x; 
  var newX =e.y;

the problem occurs in opposite side(like on click south west image zooms to north east)
