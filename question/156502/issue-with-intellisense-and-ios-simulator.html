Issue with intellisense and iOS simulator

Hi peeps,

Technical specifications:

Titanium Studio version 2.1.2

Titanium Mobile SDK version 3.0.0 and above

Mac OSX version 10.7.5 (2.5GHz Intel Core i5 and 4GB RAM)

So I have these 2 issues.

First issue is the intellisense is not working properly.

I have to enable Titanium Mobile SDK 2.1.4 to make the intellisense working if I have a project using SDK version 3.0.0 and above. I don't know exactly why but if I don't enable the 2.1.4 SDK in Project Path, the intellisense won't work.

It's not a big deal for me but I'm just wondering if there is any bug related to this?

Second issue is the iOS simulator is very slow unlike the earlier time I used it.

Back then it only took 30 seconds at most to run the app in the simulator after cleaning the project. Then it only take a mere 6 seconds at most on subsequent runs without cleaning the project. Now, it averagely takes 50 - 60 seconds to run the app in simulator after cleaning the project. Then it takes 20 - 40 seconds for subsequent runs. Not only that, the Console also responding very slow at times. The "Ti.API.warn" could appear in the console like 10 seconds after it's executed in the app.

I admit my computer now runs a bit slower but I don't really think it will affect Titanium that much since it is just slightly slower.

This is my biggest concern, is there anything I can do to resolve this issue?

Feel free to ask me if I missed any important information.
Thanks in advance.
