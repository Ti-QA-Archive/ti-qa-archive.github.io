  • Hi Seema

    Try deleting your AVDs and clearing your build files.

    To delete the AVDs:

    • close emulators
    • go to Android SDK and AVD Manager / virtual devices, and delete all existing AVDs

    To clear your build files and force a full rebuild:

    Delete the contents of the project's build&#x2F;android directory, select Edit Project in Titanium Developer and click the Save Changes button to force a full rebuild. Then launch the app.

    See this thread for tips on speeding up the development process for Android.

    If none of this improves the situation, please post your tracelog to pastebin.

    Cheers
