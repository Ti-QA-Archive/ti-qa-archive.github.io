Accessing really complicated JSON

I have a really complex JSON file I am accessing. It doesn't matter what I'm doing it just seems that I cannot access any data. I hope some one has some suggestions.

I have tried every suggestion after reading every JSON related entry but nothing seems to be working.
The code is here:
json File

my code is:

var xhr = Ti.Network.createHTTPClient();

xhr.open("GET","http://www.waybu.com/handa/scoresjson.json");

xhr.onload = function(e)
{
    try
    {
    var data = JSON.parse(this.responseText);
    // I have tried the evaluate and I get the same result as above

    Ti.API.info(data); // this outputs the entire JSON file

        // UNDEFINED or NULL
        Ti.API.info("tournament "+data.tournament.begindate); 

        // Throws Error as UNDEFINED
         Ti.API.info("country: " + data.event[0].tournament[0].country); 

        Ti.API.info("hole no: " + data.hole.no);
         }
        catch(E)
    {
        alert(E);
    }

};
xhr.send();

As you can see I have tried to output some straight forward attributes but I get NULL or UNDEFINED. I know there is something in the data object because if I output the data variable it outputs the entire JSON file.

I appreciate anytime spent helping me out on this one!

    It looks to me that you are not searching for the right path.

    it should be something like:

    data.event.@attributes.tournament.@attributes.begindate

    the problem is that the @attributes stuff will break your javascript.
    If you parse this from an xml try to get rid of the @ part.
