Possible in 3.4.0 to remove action bar from Android in tiapp.xml?

I want to create my own custom navbar, to do that I need to remove the Android built in navbar. I tried to do this in tiapp.xml using:

<android xmlns:android="http://schemas.android.com/apk/res/android">
     <manifest>
        <application
            android:theme="@android:style/Theme.Holo.Light.NoActionBar.Fullscreen"/>
      </manifest>
</android>

But I get the following error:

java.lang.RuntimeException: Unable to start activity ComponentInfo{com.uland.navigationcontrollertestone/org.appcelerator.titanium.TiActivity}: java.lang.IllegalStateException: You need to use a Theme.AppCompat theme (or descendant) with this activity.

Caused by: java.lang.IllegalStateException: You need to use a Theme.AppCompat theme (or descendant) with this activity.

Documentation tells me that this approach is outdated as of 3.3.0, but I still want to be able to remove the navbar.

I know of ways to do with styling, but I'd rather avoid that if possible. So is it possible to do what I'm attempting?

  • So after another 20 minutes of research, I found the answer to my own question, but I thought I'd share it here for everyone's benefit.

    The solution I found can be seen here: stackoverflow

    The fix is to add the following code to modify the android manifest in tiapp.xml like so:

    <android xmlns:android="http://schemas.android.com/apk/res/android">
    <manifest>
            <application android:theme="@style/Theme.Titanium"/>
    </manifest>
</android>

    Hope this helps someone

  • This is actually Android Action Bar, There are two ways to hide the Action Bar.

    1. You can either hide the activity's Action Bar in your JavaScript code or modify the default theme to hide the Action Bar.
    win.activity.actionBar.hide();

    This hides the action bar once it appears on screen.

    1. To automatically hide the action bar when opening a window or tab group, you need to modify the theme to hide the action bar:

    First To automatically hide the action bar when opening a window or tab group, you need to modify the theme to hide the action bar:

    a. Create a theme fife "custom_theme.xml" and put the file in this directory "platform/android/res/values/". FYI, you need to create this directory.

    <?xml version="1.0" encoding="utf-8"?>
<resources>
    <style name="Theme.NoActionBar" parent="@style/Theme.AppCompat">
        <item name="android:windowActionBar">false</item>
        <item name="android:windowNoTitle">true</item>
    </style>
</resources>

    As your theme file is now in "platform/android/res/values/custom_theme.xml".

    b. Put the above code to the theme file.

    Second, Modify your tiapp.xml file to use the modified theme. This will globally hide the action bar in all of your activities:

     <android xmlns:android="http://schemas.android.com/apk/res/android">
    <manifest>
        <application android:theme="@style/Theme.NoActionBar"/>
        <!-- Need to specify at least API level 11 for Titanium SDK 3.2.x and prior -->
        <uses-sdk android:minSdkVersion="14" android:targetSdkVersion="19"/>
    </manifest>
</android>

    a. Put the above code in your tiapp.xml file.
