TableViewRow Tag

I have a table view with a list of cities. The cities are populated automatically from an online JSON feed in table rows. After clicking on a city, I'd like the app's city or region to change to that city. My problem is that in the table's event listener, I am not sure how to pass that row's city to do whatever I have to do there.

So:

tableview.addEventListener('click', function(e)
{
    e.row.id;
}

How can I achieve something like that where I can retrieve the ID or a string?

Below is the for loop which I use to populate the table rows:

for (var i in citiesData) {

        var city = citiesData[i];
        var id = city.id;
        var short_name = city.short_name;

        var row = Ti.UI.createTableViewRow({
            height: 44,
            rightImage: '/images/settings/'+city.short_name+'.png'
        });

        var nameLabel = Ti.UI.createLabel({
            text: city.name_ar,
            textAlign: 'right',
            height: 'auto',
            right: 25
        });
        row.add(nameLabel);

        // Enter check next to city selected        
        if (currentCity == city.name) {
            row.leftImage = '/images/settings/check.png'
            rowCitySelected = row;
        }

        data.push(row);
    }
