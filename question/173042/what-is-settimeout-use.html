    Hello Rama ,

    setTimeout is used to Executes code or a function after a delay.

    funAlert()
{
    alert("function call after 3 second ");
}

setTimeout(function(e){
    funAlert();
},3000);  //  3000 milliseconds

    as above example funAlert function call after 3 second. for more details see this link Global-method-setTimeout.
