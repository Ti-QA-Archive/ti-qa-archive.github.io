    Hi

    You have two problems;;

    • You are calling your parameter menu_link and assigning it to a new window, however the menu_link is an image and NOT a local JavaScript file that would be required for this window object to function,
    • Two of your example url properties have formatting in them \n, this would have to be stripped out.

    You need to create a window that includes and image - the image is assigned to the menu_link property.

    You could do this by passing a custom property with the menu_link to a generic JS file if you are intent on using the url property. In this you add a single image and use the custom property to be the image reference.
