  • Not normal at all. Warning are just that: warnings! so, I suggest you to review each warning and solve it. Maybe lot of them are ';' missing at the end of line and things like that, but my experience is that Titanium works better if you fix that warnings (less builder hangs).

    If these warnings are because you are including some kind of library from internet (such as Raphäel or any other javascript lib), you can change the extension of the library files. Rename files to '.lib' instead of '.js' and Javascript interpreter will not verbose warnings from these files.

    Of course, you will have to change also your include line to Ti.include('whatever.lib') instead of '.js'.
