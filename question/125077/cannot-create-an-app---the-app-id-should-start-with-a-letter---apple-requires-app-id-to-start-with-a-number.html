Cannot Create An App - "The App Id Should Start With A Letter" - Apple requires App Id to start with a number

I can't create an application in Titanium studio with an App Id that starts with the id prefix provided by Apple.

All I can find is this Jira item.

"Titanium Studio does not allow App Ids that begin with numbers. Apple issues App Ids that begin with numbers."

In which I can mostly see people complaining that the bug was filed the one wrong place, but nobody has bothered to link the Jira to the correct issue - so I have no way of hunting down a solution for myself.

Is there a workaround for this?

Thanks.

  • Accepted Answer

    You should not include your Bundle Identifier with your App Id. For example, if your full ID looks something like this:

    3E6A7AFZ7E.com.mycompany.myapplication

    Your application ID inside Titanium Studio should be set without the first portion, in this example:

    com.mycompany.myapplication
