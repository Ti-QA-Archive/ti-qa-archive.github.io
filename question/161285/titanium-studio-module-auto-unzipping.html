Titanium Studio module auto unzipping?

Hey,

So it's been a while since I've worked with Titanium Studio and even longer since I wrote here… Some have changed… especially Titanium Studio that I had to get used to some of the changes.

It used to be that when I downloaded a module from the marketplace I'd drop the zip file in the main project directory, and upon first run Titanium Studio would automatically unzip the folders in the modules directory.

Now it doesn't seem to do that… is it my configuration or did Titanium Studio don't do that anymore?

Thanks

    I wasn't aware that technique was no longer working. But there's actually an easier way in the newer versions of Studio. From the Help menu, choose Install Mobile Module. You can paste a URL to the zip file in the box and it will be downloaded and installed. Or, if you've already downloaded the zip file, use the Browse button to locate it. You can choose to install for all projects (choose Titanium SDK) or a single project. Either way, the zip file will be downloaded, copied to the correct location, unzipped and installed.
