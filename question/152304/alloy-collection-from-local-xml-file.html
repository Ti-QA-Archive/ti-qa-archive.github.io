Alloy Collection from local XML file

Hi,

I want to load a local XML file. The file contains a few models like

<models>
    <model>
        <name>A</name>
    </model>
    <model>
        <name>B</name>
    </model>
...
</models>

Read only would be enough for me.

I didn't found a xml sync-adapter for it…
Is there a simple way to do it?

Thanks.

  • No, there's no XML sync adapter, since this is essentially just a file-based read. A sync adapter would be overkill. I would just read it as you would any other incoming data or file (depending on how you get it) and use the Titanium.XML API to read it as necessary into whatever structure or model you see fit. Better yet, convert the data to something more performant, like JSON, before it even reaches your app and then just parse it with JSON.parse().
