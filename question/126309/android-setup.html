Android setup

Hi guys, I wish to get started with Android development.

I'm trying to follow the instructions on the Preparing for Android development but am getting nowhere - this is in no way written for a beginner.

1st in mentions that I need Java JDK for development, so off I go to the Oracle site & cannot even find the right download for my MAC, good start.

2nd it says to check the Titanium Compat. Matrix, so it says I need a min of '1.6 (rev 10)'. I run the java/javac -version in Terminal & I have 'javac 1.6.0_26'. Is this good? Do I still need the JDK? On the matrix page is stays: 'Oracle JDK can be downloaded from here' - but this links to a 404 page!

I can see why so many people struggle with Android dev on Ti now, any help greatly appreciated.

Regards
Sharry

Running: MacbookPro, Snow Leopard

    You should already have the JDK on Mac. Now you just need to install the Android SDK and setup it location in Titanium Studio in Preferences -> Titanium Studio -> Titanium and set the Android SDK directory.
