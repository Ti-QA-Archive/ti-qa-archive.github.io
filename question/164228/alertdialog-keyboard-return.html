AlertDialog Keyboard Return

I have an AlertDialog created with a textfield inside it using the style property set with Ti.UI.iPhone.AlertDialogStyle.SECURE_TEXT_INPUT. Is it possible to attach an event listener to the alert/textfield so that I know when the keyboard return key is pressed?

    Hi Keith,
    Sorry to say but you cannot add an event to the input field in the alert dialogue. However you can add event to the dialogue itself and get the text entered in the dialogue like this

    var alertDiag = Ti.UI.createAlertDialog({
    style : Ti.UI.iPhone.AlertDialogStyle.SECURE_TEXT_INPUT,
    title : 'Enter Secure Input'
});

alertDiag.addEventListener('click', function(e) {
    alert(e.text);
});

alertDiag.show();

    Please feel free to ask if any confusion
