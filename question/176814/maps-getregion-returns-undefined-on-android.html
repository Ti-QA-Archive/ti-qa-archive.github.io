Maps getRegion() returns undefined on Android

Pretty simple, calling getRegion() on android returns undefined. On iOS it returns the expected object.

Titanium 3.3.0, Alloy 1.4.1

Anyone know any workarounds? 

<Module id="mapView" module="ti.map" method="createView"/>

Then

    var tRegion = $.mapView.getRegion();

tRegion is just undefined on Android. This is across multiple devices as well.
