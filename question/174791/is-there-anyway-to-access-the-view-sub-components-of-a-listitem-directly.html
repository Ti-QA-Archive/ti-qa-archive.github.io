Is there anyway to access the view sub-components of a ListItem directly?

Is this possible, or can we only pull the ListItemData representation directly from the ListSection and operate on that? It seems that unless a view sub-component in a ListItem has an event attached to it, there is absolutely no way to access it directly.

I understand that data updates to a ListItem should be handled via ListSection.updateItemAt/etc., but there are times when we just want to make simply display updates to view sub-components in a ListItem, and the current structure of ListView makes this incredibly difficult; i.e. we have to loop each section and each item to not only find the target, but to make not of its index in the section.
