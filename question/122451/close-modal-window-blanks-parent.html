Close modal window blanks parent

Hi I'm obviously doing something stupid here, but ..

I'm developing for Android.

My app has a main window with a menu. The menu is used to show a second page for maintaining some category codes. The menu loads the second page as modal window.

The second page runs fine, but when it is closed using the android back button the main window shows as blank :(

Here are the relevant pieces of code:

In my main window -

// set up the options menu
thisWin.activity.onCreateOptionsMenu = function(e){

var m = e.menu;

var option = m.add({title:'Categories'});

option.addEventListener('click', function(e){        

    var categoryWin = Titanium.UI.createWindow({
        title: 'Categories',
        url:'../windows/categoryWindow.js'
    });        

    categoryWin.open();

});

}

the category window -

// category maintenance window
var thisWin = Titanium.UI.currentWindow;

thisWin.backgroundColor = '#000';

thisWin.fullscreen = true;

thisWin.modal = true;

What am I doing wrong?

Thanks

  • Not sure, doesn't look like you're doing anything wrong. One fix would be to listen for the android:back event while that window is open. When you receive that event, just close that window yourself as if the close button were pushed.
