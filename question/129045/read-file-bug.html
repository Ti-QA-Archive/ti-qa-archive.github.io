read file bug?

I'm trying to read a resource file with the Titanium file stream.
This works on the iPhone Simulator, BUT not on my iDevice (iPod touch 4g).
I'm currently using SDK 1.8 / 1.7.5 and iOS 5 (iPhone Simulator and iPod Touch).

Here's my code:

    function loadFile(file) {
        // create new file stream
        fileStream = Ti.Filesystem.getFile(file);

        // does the file exist?
        if (fileStream.exists()) {
            // -- method: buffer --
            file = fileStream.open(Ti.Filesystem.MODE_READ);

            // create buffer
            buffer = Ti.Buffer.createBuffer();

            // load data in buffer
            file.read(buffer);

            fileContents = buffer.toString(); // null on Device

            // -- method: file stream --
            fileContents = fileStream.read();
            fileContents = fileContents.text; // null on Device
            fileContents = fileContents.toString(); // object TiBlob on Device

            // return fileContents;
        }

        return false;
    }

Thanks for help.
