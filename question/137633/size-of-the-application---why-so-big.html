Size of the application - why so big?

I'm using Titanium Studio, build: 2.0.1.201204132053
and generating applications for Android.

I've just generated the apk of the sample todo list app that is included with Titanium Studio and it is over 4Mb size. Why is it so big? I've done some Android apps with more functionality and they are under 150 Kb.

Is there a way to make it smaller? What does into the app to make it that big, is there a link where they break down the content?

Thanks

  • Titanium packages the v8 or Rhino runtime with all projects so the minimum size of the app will be around that amount (Rhino is smaller but has worse performance)
