    I ask because your version of node is busted somehow.

    first, completely uninstall NodeJS

    Next, I suggest using NVM (node version manager)

    finally do 

    npm install -g titanium

    you should be working again. I just had this exact issue and this is how I fixed it.

    I am using OSX 10.10.2
