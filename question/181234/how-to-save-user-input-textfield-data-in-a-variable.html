How to save user input textfield data in a variable

var u=tname.value.toString(); // tname is my textfield name
var p=tpass.value.toString(); tpass is my textfield name

and
i have also tried other method of textfield still not working

var u=tname.getValue();
var p=tpass.getValue();

can any one help me. how should i get this.

  • i guess you have no event to put the textFields value to variable.
    Something like a button which the user should click after he or she finished the input.
    If you want to pass the actual input in local variable you could use the "change" Event of the textField.
    Maybe something like that:

    var input;
textField.addEventListener('change',function(e){
       input = e.value;
});

    or you simply place a save button:

    var input;

var btn = Titanium.UI.createButton({
  title:'Save'
});
btn.addEventListener('click',function(){
   input = textField.value;
});
