  • Hello Sabari,

    Check the processed status

    in your response, you need to check if the "processed" value is set to true. Check this:

    urls : Hash
A hash containing image URLs for the photo at different sizes. The keys in the hash correspond to the photo sizes described in Photo Uploading & Resizing:

square_75
thumb_100
small_240
medium_500
medium_640
large_1024
original
The URLs are only available after the photo has been processed (processed is true).

    Documentation

    You can check more on this in the photo's show method here.

    And more details in the Photo Size and Status.

    Let me know if this helps.

    Mauro
