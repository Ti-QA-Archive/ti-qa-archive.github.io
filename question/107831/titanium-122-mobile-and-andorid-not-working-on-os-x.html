    In order to determine whether your system is configured correctly, copy and paste the following code into your terminal/shell. Copy the output and paste it to your preferred pastebin service. It's important that you do not modify it in any way, as this could mask problems with your configuration. Do NOT paste it here! This is because the system will change the formatting.

    clear
cd /
uname -a
echo "$PATH"
echo "$JAVA_HOME"
java -version
javac -version
python -V
echo "$android_sdk"
adb version
android list target
git --version
echo " --- The following is optional ---"
scons -v
echo "$HTTP_PROXY"
echo "$HTTPS_PROXY"
echo "$SOCKS_PROXY"

    When you have posted this, I will take a look and try to help you.

    Cheers
