Getting attribute value of tags: XML parsing

Let's say I need to parse the following xml file :

<doc>
   <price currency = "dollar">1.34</price>
</doc>

I can get the "1.34" from getElementsByTagName() , and then using the "text" variable.
How do I go about if I need to find out what the currency is?
That is, how do I find out the value of attributes of the tags?
BTW, I'm developing for android, using titanium sdk 1.6.2 and android 2.2

  • Instead of .text, try .attributes.getNamedItem(&quot;currency&quot;).nodeValue

    This is the code I used to test:

    var string = "<doc><price currency = \"dollar\">1.34</price></doc>"
var xml = Titanium.XML.parseString(string);
var nodes = xml.documentElement.getElementsByTagName("price");
Ti.API.info(nodes.item(0).text);
Ti.API.info(nodes.item(0).attributes.getNamedItem("currency").nodeValue);
