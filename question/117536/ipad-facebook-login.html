iPad Facebook Login

Hi

I want to use the Facebook login on the iPad. The button opens Safari with the proper url, the user clicks on "OK". Now the user should be redirected to the app, but I only get a popup: "Safari cannot open the page because the address ist invalid"

The url is: http://www.facebook.com/connect/uiserver.php

Anyone else experiencing the same problem?
Thanks

Felix

    Add this in your tiapp.xml file (change the facebook app id to the one you created on FB)

    <property name="ti.facebook.appid">1234567890</property>
