Input box crashes iOs emulator

Any input box using Ti.UI.createTextField crashes the iOs emulator upon typing into it. It loads and renders fine, it's just as soon as any typing happens in the box a crash occurs with only the debug info below.

Setting passwordMask to true actually works fine.

I'm using Mac OS X Lion and have tried the latest build re5c56c17.
Titanium Studio, build: 1.0.4.201108101535

var android = Ti.Platform.name == 'android';

var win = Ti.UI.createWindow({
//url:'javascript/login.js',
navBarHidden:false,
title: 'Idea2'
});

var username = Ti.UI.createTextField({
autocapitalization:Titanium.UI.TEXT_AUTOCAPITALIZATION_NONE,
borderStyle:Titanium.UI.INPUT_BORDERSTYLE_ROUNDED,
top:110,
height: android ? 45 : 35,
width:300,
passwordMask:false,
hintText:'Username'
});
win.add(username);

win.open();
