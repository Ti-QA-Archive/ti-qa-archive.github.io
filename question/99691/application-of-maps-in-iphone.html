  • Your code works fine for me as posted (Titanium Mobile 1.5.1, iOS emulator 4.2) when I make the map URL a single line string. Remember that in JavaScript, multi-line strings must be separated by exactly one space and a \:

    var myString = 'foo \
bar \
baz';

    usually this is more painful than just doing:

    var myString = 'foo'+
  'bar'+
  'baz';

    In which case you're assured of a more consistent result. If you need to concatenate a really big string, you should use an array and join, which will perform much better:

    var buffer = []; 
for (var i = 0; i < 1000; i++)  {
   buffer.push('piece of my big string');
}
var str = buffer.join('');
