  • Hello luca,

    Thanks for your Post. Yes, you can get all subscribed device to a channel push with php REST API. Please make a REST API call using CURL. I hope you will get your expected output.

    Required Parameters:

    name : String required

Name of the push channel.

The name of the push channel cannot start with a hash symbol ('#').
pretty_json : Boolean

Determines if the JSON response is formatted for readability (true), or displayed on a single line (false). Default is false.

    $ curl -b cookies.txt -c cookies.txt -X GET --data-urlencode "name=chat" "https://api.cloud.appcelerator.com/v1/push_notification/channels/show.json?key=<YOUR APP APP KEY>&pretty_json=true"

    For more information please visit PushNotifications
