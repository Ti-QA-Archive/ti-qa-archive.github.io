If I start a window with the device in landscape mode, if i turn it into portrait mode, the contents of the window doesn't turn, but the keyboard and status bar do.

If I turn back to horizontal and then to portrait again, it works properly.

    • Can't really share my sample code as it's pretty involved.

      The bug only occurs when you start the first window in Portrait only mode. orientationModes:[Ti.UI.PORTRAIT]

      1. Open Window A in Portrait Mode
      2. Open Window B from Window A with all 4 orientations enabled, while holding the device horizontal.
      3. Window B should start in landscape mode. Make sure a keyboard is showing.
      4. Turn your device to portrait mode. This results the first time: http://i.imgur.com/uzrcrSv.png
      5. If you turn it back horizontal and back to portrait, it works correctly.
