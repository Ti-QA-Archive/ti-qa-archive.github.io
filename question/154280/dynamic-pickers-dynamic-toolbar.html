Dynamic pickers, dynamic toolbar

Hello my friends, I have the following code which is not working. The buttons that I generate dynamically in an array are not picked up by the dynamic toolbars.

optPicker[i] = Titanium.UI.createPicker({
                        top:43,
                        zIndex:100,
                });
                optPicker[i].selectionIndicator=true;

                pickers[i] = Titanium.UI.createView({
                    height:251,
                    bottom:-251
                });

                cancel[i] =  Titanium.UI.createButton({
                    title:'Cancel',
                    style:Titanium.UI.iPhone.SystemButtonStyle.BORDERED
                });

                done[i] =  Titanium.UI.createButton({
                    title:'Done',
                    style:Titanium.UI.iPhone.SystemButtonStyle.DONE
                });

                spacer[i] =  Titanium.UI.createButton({
                    systemButton:Titanium.UI.iPhone.SystemButton.FLEXIBLE_SPACE
                });

                toolbar[i] =  Titanium.UI.iOS.createToolbar({
                    top:0,
                    items:[cancel[i],spacer[i],done[i]]
                });

                pickers[i].add(toolbar[i]);
                pickers[i].add(optPicker[i]);

If I dont put the buttons in an array then it works fine, anyone have any idea? Is this a scope issue?

    Hi

    I suspect you have not declared the variables as arrays to start with.

    Make sure you have declared the variables like the following;

    var spacer = [], 
    cancel = [],
    toolbar = [],
    done = [];
