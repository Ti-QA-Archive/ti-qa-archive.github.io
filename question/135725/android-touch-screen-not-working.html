  • Often when a touch event is not registering as one would expect, there is another view's event handler that is intercepting the event because of overlay/zindex conflicts.

    In 2.0.1, the layout defaults for most of the UI items has changed, which may place a newly added object in front of an existing item. Titanium 2.0 Layout Changes
