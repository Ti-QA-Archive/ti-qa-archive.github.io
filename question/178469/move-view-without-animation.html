Move view without animation

With this code I move a view from a point to another one animating it:

var m = Ti.UI.create2DMatrix();
m = m.translate(0, 175);
var a = Ti.UI.createAnimation();
a.transform = m;
a.duration = 300;
myView.animate(a);

Now, I want to do a different thing: move this view from the current point to another one without animating it. Is this possible? How?

  • Hi Gab,

    yes it's possible, simply by changing the properties of the view

    var view = Ti.UI.createView({
    top:100,
    left:100
});
win.add(view);

view.applyProperties({
    left:10,
    top:10
});

    • here is a working example

      var win = Ti.UI.createWindow({
        backgroundColor: 'white'
    });

    var view = Ti.UI.createView({
        top: 100,
        left: 100,
        width: 100,
        height: 100,
        backgroundColor: 'red'
    });

    win.add(view);

    view.addEventListener('click', function() {
        view.top = 10;
        view.left = 10;
        setTimeout(function() {
            view.top = 100;
            view.left = 100;
        }, 1000)
    });

    win.open();

    • Moritz, I SOLVED this way:

      var m = Ti.UI.create2DMatrix();
m = m.translate(0, 0);
myView.transform = m;
// I haven't tested if this is required
myView.top = m.tx;
