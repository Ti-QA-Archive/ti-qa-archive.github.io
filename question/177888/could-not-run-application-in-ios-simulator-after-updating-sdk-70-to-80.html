    Hello, Nuibb, You need to Update your Titanium SDK, CLI and Configure your Titanium using your new Xcode. To do that, Please follow these steps.

    I guess you olready have Xcode6 Installed for iOS 8.0.

    First from the Turminal Please Run these Commend

    sudo npm install -g alloy@1.5.0-beta2        
sudo npm install -g titanium@3.4.0-beta

    Download Titanium SDK 3.4.0.v20140905152516 And install through studio.

    Run 

    titanium sdk select

    To check if the new sdk is selected.

    Now if you have Xcode6 installed in your mac then Run

    sudo xcode-select -switch /Applications/XcodeVersion.app/Contents/Developer/

    Here put the "Xcodeversion" of Xcode that is installed.

    Now you can run application on iOS 8.0. If this helps you please select as best answer. Thanks.
