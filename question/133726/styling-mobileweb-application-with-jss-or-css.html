  • You can modify some stuff. If you look in index.html, there is a bunch of CSS at the top that you can tweak, however keep in mind that subsequent builds will overwrite your changes. You can also modify a file css file at <titanium mobile SDK install folder>/mobileweb/themes/common.css, however note that this will affect ALL mobile web projects.
