Android Image "allowEditing" Alternatives?

I have a completed iOS app that I am trying to get to work with Android. The application uses square images and we have the user crop them on iOS using "allowEditing: true". This must be done before sent to the server because the server expects only square images.

It seems Android does not support "allowEditing". While I could make a system to try and auto crop it for them I would like to know if there is something on git or the marketplace I don't know of that would allow this. I have searched for a few hours now and I can't seem to find anything.

Has anyone worked with this issue before and/or can recommend a way to resolve it?
