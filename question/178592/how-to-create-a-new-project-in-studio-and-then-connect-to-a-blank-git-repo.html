  • Ok, from the "App Explorer" (not Project Explorer) click on the settings button to open the menu. Click there on "Initialize Git repository". Now you should notice that your files are red marked.

    Next click again on the settings button, go to "More => Add Remote". Change the remote URI, that should be something like git@git.assembla.com:[yourname]/[project].git

    Now if you click on the settings button once again, you should see links to "Commit …" and later "Push"
