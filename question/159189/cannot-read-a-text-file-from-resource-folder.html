var f = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory,'myfile.txt');
f.write("hi all");
alert(f.read());

  • Hi Maruti,

    Try this :

    var f = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory, 'myfile.txt');
f.write("hi all");
alert(f.read().text);
