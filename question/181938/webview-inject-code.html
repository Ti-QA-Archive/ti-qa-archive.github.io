webview inject code

Hello, i have an html form under this url http://www.mysite.com, anyone knows if there is a way to load this url via a webview and inject some js code to fillup my remote form?
thank you

  • You can use evalJS of webView for the same. Here is an example,

    
$.webview.addEventListener('load', function(e) {
    $.webview.evalJS("document.getElementById('txtName').value = 'Mattia'; ");
});
