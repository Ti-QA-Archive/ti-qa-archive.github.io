  • Hi

    Here is a sort function that allows you sort an object array based on the node.

    var sortBy = function (prop) {
    return function (a, b) {
        if( a[prop] > b[prop]) {
            return 1;
        } else if (a[prop] < b[prop]) {
            return -1;
        }
        return 0;
    }
};

    You use it as follows;

    arr.sort(sortBy('header'));

    So to sort your array you sort by the node you require. However I suspect that you also require the title to be sorted within the header sort. If so you sort by the LAST one first then the next one and so on.

    arr.sort(sortBy('title'));
arr.sort(sortBy('header'));

    This is a working example;

    var sortBy = function (prop) {
    return function (a, b) {
        if( a[prop] > b[prop]) {
            return 1;
        } else if (a[prop] < b[prop]) {
            return -1;
        }
        return 0;
    }
};

var arr = [
    { header: 'Red', title: 'One' },
    { header: 'Red', title: 'Two' },
    { header: 'Green', title: 'Three' },
    { header: 'Blue', title: 'Four' },
    { header: 'Orange', title: 'Five' },
    { header: 'Orange', title: 'Six' },
    { header: 'Green', title: 'Seven' }
];

arr.sort(sortBy('title'));
arr.sort(sortBy('header'));
