Managing List of items

Hello there, I am working on a shopping list App. The features of my app include (1) managing a list of items (figure 1), and (2) creating a shopping list by selecting one or more items from the list of items. I want to build the application around a tab bar controller to make switching between the two views fast.
Any ideas how this can be done that only marked items in one tab appears in the next tab as shopping list. Thanks in advance.. :)

  • Hi Zain,

    Use Ti.App.addEventListener for adding an item in to list that you are selecting in the list for cart.

    forExample:

    Ti.App.addEventListener('addItem', function(obj){
    // should be your code here for adding a item in the list.
}

    at other side you can fire event while user selected item from other list to add it to cart listing. for example:

    Ti.App.fireEvent('addItem', { itemId : ###, itemName : "itemName", itemDetail : "itemDetail" });

    Note that this is not the only way you can do this but this is a hint for passing a data within an application, It is dependent on you logic. You can get lot more logic on your question's thread but main point is that you should choose your best match that fit best in your logic scenario.

    best of luck.
