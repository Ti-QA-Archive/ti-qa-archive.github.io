How to modify a DOM document?

Traversing the XML-Tree is no problem, this works fine.

I try to modify a DOM documment, which is created from a XML-File using "parseString". I tried to change the node-text of a node that I read with

var KDNrNodes = DomIni.documentElement.getElementsByTagName('KDNR');
var KDNrNode = KDNrNodes.item(0);

KDNrNode.text is transfered to a textfield. When I try to store the textfields value to as node tex, that does't work. I tried

KDNrNode.text = EditKundenNummer.value;
KDNrNode.nodeValue = EditKundenNummer.value;
KDNrNode.setNodeValue(EditKundenNummer.value);

but afterwords KDNrNode.text is unchanged. Is it possible, that the SOM document is "read-only" or what else I am doing wrong? Because of the VERY poor documentation it is very hard to find out something useful…

  • Maybe something like this:

    //first remove old text node
KDNrNode.removeChild(KDNrNode.firstChild);
//create new text node
var textNode = DomIni.createTextNode(EditKundenNummer.value);
//add new text node
KDNrNode.appendChild(textNode);
