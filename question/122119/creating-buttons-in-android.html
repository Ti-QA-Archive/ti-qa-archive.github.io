Creating Buttons in Android

Hi,

Im trying to create a navigation bar in Android. Im using a view and adding buttons to it.

This is my code:

var currentWin = Ti.UI.currentWindow;

var top_header = Ti.UI.createView({top: 0, height: 40, layout:'vertical'});

var action = Titanium.UI.createButton({
    borderColor: "#FF0000",
    backgroundColor: "#FF0000",
    borderRadius: 10,
    title: "Mas fotos",
    top: 1,
    right: 2
});
top_header.add(action);

var action2 = Titanium.UI.createButton({
    borderColor: "#FF0000",
    backgroundColor: "#FF0000",
    borderRadius: 10,
    title: "Volver",
    top: 1,
    left: 2
});
top_header.add(action2);

currentWin.add(top_header);

    Well yes that is what I would expect to see. If you are trying to place your buttons top left and top right as I suspect, remove the layout: 'vertical' from the createView container.

    It has done exactly what it should laid out the buttons based on the location of the previous button. the layout will add it below the last item.
