Tablew view in nested view not firing click events

I have a tableview in a view which is added to a window. If I add the table directly to the window, everything works fine, if I ad the Table to a view component and add them to the window, the click event does not work. Thanks

3 Comments

  • Can you post the code you are trying?

    Also, make sure the view you are adding to has width and height set. Depending on what version of the SDK you're using, it's possible to have a table added to a view with no width or height. The entire table would overflow the view and would be unable to receive any click events.
