How to split up a table view?

Hi,

I've got an image file which shows a book. On each side of the book, a list of animal names should be displayed.

I got 32 animals, divised along the characters of the alphabet. How can I split up the list in such a way that there will be two lists of it layed out right next to each other in the book?

Effectively i would like to get my list split up in such a height that the content of the list matches a certain height (e.g. 500 pixels).

Is there any way to check or measure the length of a table currently being build in order to stop adding data rows to it?

Thanks in advance,
Christophe Leske

  • When building the table, build also custom rows with:

    Ti.UI.ceateTableViewRow({
    height: 30 // just an example
});

    In the tableView you set separatorStyle to 0.

    If there are no entries for a char, you do not build a row.
    That way you're able to calculate the height (countRows * heightPerRow).
    When the calculated result reaches 510 px or 17 rows you are done with it and continue the second table.

    If you need to calculate the height before adding it, you'll need to calculate the non-empty entries i.e there are 20 => you can not use 17 on each table since the second would only have 3.
    You will do nonEmptyEntries / 2, the result is the count of rows per table.

  • It might be easiest for you to calculate the height of each row as you add it to your table views to determine the current height of the table view. So as you iterate through your rows, keep a tally of the cumulative height of rows processed until you hit your threshold. Once you hit it, reset and start adding rows to the next adjacent table view.
