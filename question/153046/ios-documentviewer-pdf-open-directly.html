    Hi

    If all you want is to view a pdf directly in your app then all you need is;

    var viewer = Ti.UI.iOS.createDocumentViewer({
    url: 'test.pdf'
});
viewer.show();

    This shows the PDF in your own app taking up the whole screen without any additional user interaction. The user can share or print the print using the top right button or close the view with the top left button.

    I have just tested this code as working as described above on the iPad and the iPhone. The iPad UI for page thumbs is better than the iPhone version - but otherwise they are the same.
