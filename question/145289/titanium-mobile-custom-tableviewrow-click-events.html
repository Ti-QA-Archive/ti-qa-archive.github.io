Titanium Mobile Custom TableViewRow click events

Hi,

I have a custom TableViewRow which is generated in a loop, each row contains an image and a label.

The table has a click eventListener which opens a new window.

I want the whole row to be clickable, but if one clicks on the label or image the row won't be "clicked".
The eventListener does get fired however the dataToPass is empty..

So I was wondering, what is the best to make sure that no matter what is clicked in the row the window will be opened?

Thanks in advance!

    Hi Vincent

    You have not supplied any code which would have been very useful, but this may be a simple one.

    For every view, label, button or other item added to the TableViewRow add the property touchEnabled: false.

    For example;

    var row = Ti.UI.createTableViewRow({
    height: Ti.UI.SIZE,
    width: Ti.UI.FILL
});
var lbl = Ti.UI.createLabel({
    height: Ti.UI.SIZE,
    text: 'Example On Row Label',
    touchEnabled: false, // add this to everything you ADD to the row
    width: Ti.UI.SIZE
});
row.add(lbl);

    This will stop the views that are not the row being involved in the event, this way when you click it will be as if the label did not actually exist from an event perspective.
