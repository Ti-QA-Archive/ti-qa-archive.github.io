  • None of the existing answers worked for me in Appcelerator Studio 4.3.0.201509151348 on my Mac.
    However, if you go in Menu Commands>Bundle Development>Show Command Bindings, you will se existing shortcuts that are not in Preferences>General>Keys.
    The Comment Line/Selection is one of them.

    Go in Commands>Source>Edit this bundle. That will make the project Source appear in Project Exmplorer. Under folder "commands" you will find toggle_comment.rb. Choose open with Text Editor (Default editor would work for me, it opens in Xcode and changes are not reflected). Then you can edit cmd.key_binding value.
    I put " cmd.key_binding = ["CONTROL+C"]"

    Restart Appcelerator Studio and you're fine.
