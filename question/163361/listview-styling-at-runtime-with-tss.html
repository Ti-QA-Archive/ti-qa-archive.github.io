ListView styling at runtime with tss

I created a listview and ItemTemplate at runtime. Is it in any way possible to style the items using tss-classes?

Because the "concrete" items will be created at runtime from the template,… how to tell alloy to use my tss-class xy for a given view?

  • Hi

    Inside your ItemTemplate you can provide ID's, like so:

    <ItemTemplate name="template">
    <ImageView bindId="pic" id="icon" />
    <Label bindId="info" id="title" />
    <Label bindId="es_info" id="subtitle" />
</ItemTemplate>

    You can then provide TSS styles to the ID's:

    "#icon" : {
    width: '50dp', height: '50dp', left: 0
},
"#title" : {
    color: 'black',
    font: { fontFamily:'Arial', fontSize: '20dp', fontWeight:'bold' },
    left: '60dp', top: 0
},
"#subtitle" : {
    color: 'gray',
    font: { fontFamily:'Arial', fontSize: '14dp' },
    left: '60dp', top: '25dp'
}

    For more information check the documentation here on ListView's
