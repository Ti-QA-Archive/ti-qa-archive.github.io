Deploy an app to a local Android device from a remote Mac. Can it be done?

Hello. I'm currently trialing a remote iOS development environment using vmOSX (similar to MacinCloud but a little cheaper), and so far it's ok (not perfect but better than nothing). I hope to buy a new Mac later this year but for now it's easier on the cashflow if I go this route. I can deploy (for testing purposes) apps for iOS by grabbing the ipa using ftp and opening them locally using my copy of iTunes. However I'm a little stumped as to how I can do this for Android - my understanding is that Titanium needs me to connect my Android phone to the Mac so it can copy the app onto the phone.

Am I right or is there a way I can compile an Android app remotely and then load onto my Android phone locally?

Thanks

Phil

    Under developer options or application install settings, enable non-store or insecure apps (wording different by hardware brand).

    Then you can install from email, web link, drop box, etc.
