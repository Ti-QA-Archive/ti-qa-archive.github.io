Integrate NodeJS Modules with Alloy

Hi all,

Since Alloy is developed on nodejs, so I was trying to integrate some of the core apis of nodejs with 1 module. But it was giving an error.

Please suggest, how could we integrate nodejs core apis in our application, for creating custom modules.

— asked February 22nd 2013 by Ankit Thakur

  • Alloy apps are compiled into Titanium apps with nodejs. There is no runtime nodejs environment available in these apps. Nodejs creates the ap, it is not present in it. In short, you cannot use nodejs code in Alloy apps.
