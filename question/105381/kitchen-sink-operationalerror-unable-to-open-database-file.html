  • I, too, was having this issue. My work computer has Microsoft Security Essentials. Disabling Real-time protection allowed KitchenSink to run. After that, I added c:\Program Files\Titanium Developer\Titanium Developer.exe to the "Excluded processes" list and turned real-time protection back on and haven't had an issue since. … Maybe only needed to disable real-time protection for first run though… I dunno.
