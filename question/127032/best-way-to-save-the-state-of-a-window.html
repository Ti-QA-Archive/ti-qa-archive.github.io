Best way to save the state of a window

I have a table view where I'm allowing the user to check and uncheck certain options. I'm then using these option to build a query. I would like to save the state of this window, in other words what is checked and what is not checked in the simplest way possible. I know it's possible to save to the database, but that seems to complicated. Any suggestions?

  • Use the Properties API to store any state information related to your window. Any JSON serialized information can be stored as a string using the Properties API to make this easy for you.

      var arr = [true, true, false, true, false];
