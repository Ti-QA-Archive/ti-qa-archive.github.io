Ti.Android.currentActivity.getIntent() returning undefined?

I'm using the following code to check the action string of the intent currently active:

var intent = Ti.Android.currentActivity.getIntent();
var alert = Ti.UI.createAlertDialog({
message: intent.action
});
alert.show();

The alert dialog is displaying "undefined". I run this code during the first opening of the app, and inside the app resume event callback to see if by chance anything changes. In both cases it displays "undefined".

Any ideas?
