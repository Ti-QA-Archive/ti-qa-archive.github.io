  • Found the answer - for anyone else with the same problem: it's a USB mic in the system that Android can't cope with. On macs, apparently, if you have certain USB mics selected as the default line in option, Android emulator crashes. Set that back to line in, and you're good to go, it seems…

    cheers, all…
