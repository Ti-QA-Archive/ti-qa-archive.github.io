  • Hello Raffaele,

    you can save it into document directory like 

     var f = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory, 'imageName.png');

 f.write(imageData);

    you can use it as same way like

    var f = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory, 'imageName.png');

    here f is your image as blob.
