  • You can remove styling by applying:

    backgroundImage: 'none',
backgroundColor: 'transparent'

    to the tableviewrows and the tableview.

    However, keep in mind that there is currently no way to style group header when using a grouped tableview.

    To have fully customized grouped tableviews, I have used a standard table, adjusted the width, and used custom background images to simulate the grouped style.

  • Not exactly sure what you mean by 'flat, but here is an example of a grouped tableview with no separator or background:

    var win = Ti.UI.createWindow({
    title : 'window',
    backgroundColor:'#ffffff'
});

var table = Ti.UI.createTableView({
            top : 0,
            style : Titanium.UI.iPhone.TableViewStyle.GROUPED,
            separatorColor : 'transparent',
            backgroundColor : 'transparent'
        });
win.add(table);
