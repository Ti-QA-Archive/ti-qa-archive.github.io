Window closing doesn't go back to previous window

Hello

I am trying to build an application that has a main window from which I open a settings window, that has a navigation controller with a "DONE" button. When clicking on the DONE button, I want to close this settings window and go back to the main window.

This is how I'm building the settings window:

var navWindow = Ti.UI.createWindow({
    url: "navWindow.js",
    //navBarHidden: true
}).open({
    transition:Titanium.UI.iPhone.AnimationStyle.FLIP_FROM_LEFT,
    //modal: true
});

The DONE button in the settings window just calls "close()" on itself.
When the settings window is modal, everything works just fine, but the animation isn't FLIP, but the default one for modals (popping up from bottom).
When the window isn't modal, the FLIP works but when closing settings, all I get is a black screen, without returning to the main view.
Is there a way to go back to the previous (main) window without using modals?

Thank you
