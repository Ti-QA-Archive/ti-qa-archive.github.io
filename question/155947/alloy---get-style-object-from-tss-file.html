Alloy - get style object from tss file

Is it possible to applicate tss class in code to an UI element ?

For example:

    var style = $.getClass('myThemButtonClass'); // myThemButtonClass is a style defined in my app.tss
    $.button.applyProperties(style);
— asked August 13th 2013 by Markus Bahl

  • This works:

        var style = $.createStyle({
        classes: ['myThemButtonClass']
    });

    $.button.applyProperties(style);

    The problem was that Titanium didn't download the latest alloy version,
    I had to download latest version via cli: sudo npm install -g alloy
