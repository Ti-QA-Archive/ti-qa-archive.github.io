Color Picker - Github

Hello,

This isn't a question - I wanted to share some code I'm open sourcing. It's simply a color picker that you use for Titanium. It's super simple, so hope it provides some help. It's open source as well, so feel free to modify and it to make it better.

NOTE, there are UI problems with Android, so it only works for iPhone at the moment. I am working on getting it to work for it too.

https://github.com/t3hi3x/Appcelerator-Titanium-Mobile-Color-Picker

Example App.js:

Titanium.UI.setBackgroundColor('#000');
Ti.include('color_picker.js');

var win1 = createColorPicker({hexColor:"#621A84"});
win1.addEventListener("colorselect", function(e){
Ti.API.log('info', e);
});
win1.addEventListener("selectcancel", function(e){
Ti.API.log('info', "user cancelled selection");
});
win1.open();
