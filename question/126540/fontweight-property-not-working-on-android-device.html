fontWeight property not working on Android Device

Im creating the following label:

var title = Ti.UI.createLabel({
text: 'helloWorld',
font : {fontWeight: 'bold'}
});

The font property is working on emulator, but not on device..

Ti SDK 1.7.2, Android 2.2, MacOS X 10.7.1

  • I have hit the same problem.

    The way I resolved this was to setup two fonts, one being the normal view, and one being bold. (Most fonts have two different definitions).

    Then when I want to use bold I set the font face to the one I want. Works on both IOS and Android.

    Hope this helps
