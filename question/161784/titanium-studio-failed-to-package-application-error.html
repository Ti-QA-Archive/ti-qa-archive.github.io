Titanium Studio Failed to package application error

Hi everyone.

This is extremely frustrating.

My project was building with no problems at all, but then suddenly I started getting the following error when trying to run the project on a device or in an emulator.

[ERROR] : Failed to package application:
[ERROR] Application Installer abnormal process termination. Process exit value was 1

there is no other information in the console. I've tried the following steps.

1: Close and re-open titanium studio.
2: Restart PC.
3: Delete and reimport project.
4: Uninstall and Reinstall node.js and Titanium Studio.

Other projects run fine so I'm sure it must be something I did within the project but I made no major changes to it.

I've posted my project here - http://sdrv.ms/1hKIBDD . If anyone can have a look and help me I would be eternally grateful!!
