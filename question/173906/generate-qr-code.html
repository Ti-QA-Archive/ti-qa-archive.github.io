  • Hi

    The easiest way was to use this Google API;

    • Google QR Codes

    It returned an image that you displayed to users in an imageView - superb, however this is now deprecated.

    There are a number of modules available that provide the ability to include a QR code inside your app - but most of those relied on the QR code API from Google - so they will not work now.

    You are left with two choices;

    • Google search "Appcelerator module QR code" and try them all, keep in mind that some "may" work as the Google API is about to be terminated but not quite yet - so double check how they have provided the ability.
    • Write your own facility; this sounds hard but there are a number of example JavaScript code examples out there that provide the ability. You will be left with how to display it though as the returned information is 0's and 1's these will need to be turned into white and black squares. Titanium does not have any built in canvas - so look for a suitable module for that.

    Here is a starting point - it uses a web page thought to provide the canvas object.

    • QR code encoder - View the source of the page to see the code.

      I have put together a quick example for you to use to see what happens;

      • qrcode.js

      You can try this in a web page by adding the script tag in and using developer tools for your web browser or call it in a test app and change ONE line of code;

      // change line 728 from this;
console.log(strLine);
// to this
Ti.API.info(strLine);

      Run your app and as long as you have called the function (bottom line of the script) you will see the output. What you will see is a bunch of zeros and ones;

      • 0 = black
      • 1 = white

      Now you should see the pattern easily. Now it is a case of drawing small squares next to each other for each line and a new line for the lines shown.

      Not easy - but not so hard either.
