Open camera with forced Resolution?

Application type: mobile Titanium SDK: 3.1.1 Device: physical device (HTC One X+/ Samsung S2) Host Operating System: Windows 7 Titanium Studio: Titanium Studio 3.1.0

Is it possible, to preset the cameras resolution? I open the camera from my app and want to import a picture, but on some devices my app restarts after taking a picture with too high resolution.

— asked October 22nd 2013 by Jörg Etzkorn

  • I'm afraid the stock camera APIs don't enable that sort of functionality. You could check the Marketplace to see if perhaps there's an add-on module that would give you this functionality.

    You could try resizing the image after capturing it using the imageAsResized() method. Do so within the success event handler, before displaying the image in your UI. And make sure to null out the original blob data to minimize memory use. I'm sure the app crashes on some devices are related to memory consumption.
