    Hello Cifa,

    It's a typo mistake. CoverFlowView have a images property not image so change it to images. and when you run in iOS than no need to write folder name like iphone&#x2F;appicon.png just write appicon.png it's automatic take images from iphone folder. check below code. it's working fine.

    Try this :

    function ApplicationStart() {

    //create component instance
  var principaleWin = Ti.UI.createWindow({
    navBarHidden : true,
    backgroundColor : "#000000",

});

var coverFlow = Titanium.UI.iOS.createCoverFlowView({
    top : 100,
    left : 0,
    backgroundColor : '#000',
    images : ['appicon.png', 'Default.png', 'Default-Portrait.png']
});

principaleWin.add(coverFlow);

    return principaleWin;
}

module.exports = ApplicationStart;

    i hope this helps and soved your problem.
