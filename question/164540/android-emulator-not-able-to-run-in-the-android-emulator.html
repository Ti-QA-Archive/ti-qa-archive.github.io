  • Ensure that android is properly installed first. From Studio Dashboard, check environments at bottom of page. If Android has a red X by it, it's misconfigured. Also, double-check that you have API 10 installed (don't remember if that was changed in Studio 3.2).

    If android is properly installed, create a new, single-window project. Then compile and run in the emulator. This will tell you whether it's the app (test app runs) or the emulator (test app fails).
