Additional Facebook permissions

Currently using iOS at the moment.

Anyone have any take on the best method for requesting additional fb permissions. Currently I use the new auth dialog and track which permissions a user authorises and then if I get an oAuth error when performing a graph request I check the permissions again (via graph request) - in case they have disabled the permissions but not the app or I require more.

The best way (only way) I have found to request additional permissions is to log the user out and then ask them to log back in again (with the extra permission)?

The documentation shows that you should just be able to set the permissions and call authorise again - hopefully bringing up the permissions dialog. This doesn't work for me using the example code in the API docs.

I'm having to ask the user to jump though an extra hope by re-authenticating.

To quote Facebook:

"There is a strong inverse correlation between the number of permissions your app requests and the number of users that will allow those permissions. The greater the number of permissions you ask for, the lower the number of users that will grant them; so we recommend that you only request the permissions you absolutely need for your app."

For the less used social/more intrusive features of my app I only want to ask the user permission if they decide to use them - once the love is there!

Thanks for any advice.
