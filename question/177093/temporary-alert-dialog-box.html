Temporary Alert Dialog box

Hi all,

I need a alert dialog just a few seconds, then must be closed automatically.

I use alloy, index.xml:

<AlertDialog id="Actualizado" persistent="false" message="Información recibida correctamente" />

Then in index.js:

$.Actualizado.show();

How i can close it in 5 seconds for example??

Thanks

    Hi

    Why don't you just use a setTimeout?

    $.Actualizado.show();
// Place after you show

setTimeout(function () {
    $.Actualizado.hide();
}, 5000);

    Hope this helps.
