selectAnnotation dont works everytime

Hi,
I have a tabelview with velo dealer. On click I jump to an prepared mapView with all dealers. The paning to the place works fine, but the annotation opening is time to time.

Here the code after adding annotations to the map. The event is fired by clicking on tableview row:

Ti.App.addEventListener('onschrauberlistenclick',function(e){
   tabs.setActiveTab(1);
   // get index from DB
   // veloid comes from click
   for (var ndx=0; ndx<velo.schrauberdb.length; ndx++) {
      if (velo.schrauberdb[ndx].id == e.veloid) break;
   } // now we have index for the reference to elem of array
   Ti.API.log(ndx);
   // following paning works fine:
   mapView.region = {
       latitude:e.lat, longitude:e.lng,
       latitudeDelta:0.01, longitudeDelta:0.01}; 
   mapView.selectAnnotation(velo.ui.annotations[ndx]);
});

The command on last line works time to time.

Any ideas? What is wrong?
