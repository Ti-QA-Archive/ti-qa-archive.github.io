    Hi Muthu

    You can use 

    var devid = Titanium.Platform.createUUID();
Titanium.App.Properties.setString("globaldevid",devid);

    And then use this devid where you want this..

    var deviceid = Titanium.App.Properties.getString("globaldevid");

    This is safe method..And apple will not reject your app for the reason of accessing UDID .

    And this UUID will generate once the user uninstall your app..And then install it again into the same device.

    Both UUID will be different.(This is only the small issue).

    Because when the app.js is called UUID is created by Titanium.

    Hope this helps you
