  • The problem with the code you are referencing is that it uses a ScrollableView…to stop the flickering you have to use a regular view and track the "touch" events yourself. Here is the code I use for our iphone app…it doesn't flicker and it supports pretty much an infinite number of images. You will need to adjust for iPad screen size though.

    http://pastie.org/1718532

    Just create the namespace:

    var app = {ui:{}};

    Pass the following params to app.ui.createImageScroller(params)

    height: the height of the view

    urls: an array of image urls

    loadingImage: the image to use when loading ('images/loading.png')

    backgroundColor: the background color of the view
