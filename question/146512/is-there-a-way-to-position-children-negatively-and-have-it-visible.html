    Hi

    Here is a simple example;

    var outer = Ti.UI.createView({
    backgroundColor: 'orange',
    height: Ti.UI.FILL,
    width: Ti.UI.FILL
});
win.add(outer);
var inner = Ti.UI.createView({
    backgroundColor: 'yellow',
    height: 100,
    bottom: -80,
    width: Ti.UI.FILL
});
var lbl = Ti.UI.createLabel({
    bottom: 5,
    color: '#000',
    height: Ti.UI.FILL,
    left: 5,
    right: 5,
    text: 'Peek-a-boo',
    textAlign: 'center',
    top: 5,
    touchEnabled: false,
    width: Ti.UI.FILL    
});
inner.add(lbl);
inner.addEventListener('click', function (e) {
    e.source.setBottom((e.source.getBottom() === -80) ? 0 : -80);
});
win.add(inner);

    • OK short answer - No

      The clipping is here to stay and is much closer to the way Apps/Applications are created rather than web/HTML with CSS. Which is both good as it is much more respective of the way things should be but also a pain when your used to a different way.

      That said there is always (almost) a way to achieve what you want.

      From your example you need the two views stacked on top of each other, I am guessing that the height of each could change.

      One way to achieve this is;

      var outer = Ti.UI.createView({
    backgroundColor: 'orange',
    height: Ti.UI.SIZE,
    layout: "vertical",
    width: Ti.UI.FILL
});
var innerTop = Ti.UI.createView({
    backgroundColor: 'yellow',
    height: "100dp",
    width: "100dp"
});
outer.add(innerTop);
var innerBottom = Ti.UI.createView({
    backgroundColor: 'orange',
    height:'80dp',
    width: Ti.UI.FILL
});
outer.add(innerBottom);

      This will place both inner view on top of each other but the locational positioning is handled by the outer view.

    • Another favourite technique of mine is to have one view fill up the entire height EXCEPT for a fixed value of a second view, WITHOUT needing to know the height of the bigger main view.

      var topBar = Ti.UI.createView({
    backgroundColor: 'orange',
    height: 60,
    top: 0,
    width: Ti.UI.FILL
});
win.add(topBar);
var data = [
    { tile: 'One' },
    { tile: 'Two' },
    { tile: 'Three' }
];
var tbl = Ti.UI.createTableView({
    data: data,
    height: Ti.UI.FILL,
    top: 60, // equal to topBar height
    width: Ti.UI.FILL
});
win.add(tbl);

      You can do this on the bottom as well as the top.

      var topBar = Ti.UI.createView({
    backgroundColor: 'orange',
    height: 60,
    top: 0,
    width: Ti.UI.FILL
});
win.add(topBar);
var bottomBar = Ti.UI.createView({
    backgroundColor: 'yellow',
    bottom: 0,
    height: 40,
    width: Ti.UI.FILL
});
win.add(bottomBar);
var data = [
    { tile: 'One' },
    { tile: 'Two' },
    { tile: 'Three' }
];
var tbl = Ti.UI.createTableView({
    bottom: 40, // equal to bottomBar height
    data: data,
    height: Ti.UI.FILL,
    top: 60, // equal to topBar height
    width: Ti.UI.FILL
});
win.add(tbl);
