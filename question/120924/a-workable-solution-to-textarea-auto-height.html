A workable solution to textarea auto height

Here is some code that I used to get the text area height to adjust automatically. The width is fixed. So, with the font I used, it worked out that 16 characters = 1 line and height=36 covered about 1 line. Then just split the lines, count how many there are and allow for lines longer than 16 characters. Hope this helps someone. If you have a better way, please post to this question.

noteText[newnoteidx].addEventListener('blur', function() {
    // now we want to create the label and set its text to noteText
var linecount = 0;
var splitresult = noteText[newnoteidx].value.split("\n");

for (var x=0;x<splitresult.length;x++){

linecount += Math.ceil( splitresult.length / 16 );    

}

noteText[newnoteidx].height = (linecount + 1) * 36;

});

  • I'm not sure what are you guys trying to archive but in my case i need to let the user to select the text. Coudnt find the property to do that with labels so i used textarea. To find the height i needed i created a label, got it's height and applied to the textarea. I know this isnt the prettiest way to do it but worked like a charm.

    var lbl_quote = Ti.UI.createLabel({
    text:quotes_list[i].quote,
    left:0,
    right:0,
    top:20,
    width:300,
    height:'auto',
    textAlign:'center',
    font:{fontFamily:'HelveticaNeue',fontSize:23},
    zIndex: 3
});


var txt_quote = Titanium.UI.createTextArea({
    value:quotes_list[i].quote,
    height:200,
    width:320,
    top:20,
    font:{fontSize:23,fontFamily:'Helvetica Neue'},
    backgroundColor:'transparent',
    color:'#000',
    textAlign:'center',
    editable:false,
    scrollable:false,
    zIndex:3
});            

txt_quote.height = lbl_quote.height+5;
