Conditional TSS based on iPhone screen height

App basics: Mobile (iPhone) | SDK v 3.0.2 | Using Alloy

I'm curious if it's possible to conditionally include styles based on the height of the iPhone screen. I'm using Alloy, and need to be able to adjust positioning of elements based on whether or not the device has a 3.5" or 4" screen.

As far as I can tell, it's only possible to include conditional styles based on OS (styles/iphone/index.tss), but is it possible to do that inclusion based on a specific phone model?

If not, is it possible to programmatically set which stylesheet a view should use? Barring the ability to use the folder structure to do this, it would be nice to let a controller assign a stylesheet to the view its bound to.

Anyone have a solution for this? With the size diff in these handsets, I can't possibly be the first one to run across this issue.
