Function inside a JSON Object generate reference error in android device

I have a json object like this: 

// file MyObj.js
var MyObj = {
    attr : 'someString', 
    val : someValue, 
    act : function(){
        // •••
    }
}


// file app.js
Ti.include('MyObj.js');
MyObj.act( somePars );

when i test it into iOS simulator it work, but when i test it in my Android Device i have the error: Uncaught ReferenceError: MyObj is not defined

  • I solved!

    The problem was: Android device doesn't find the file with the object, but iOS simulator find it.

    I solved in this way: 

    Ti.include( Ti.Filesystem.resourcesDirectory + '/MyFolder/MyObj.js');
