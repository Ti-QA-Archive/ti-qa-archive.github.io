Push notifications callback is it possible to have multiple callbacks or a centralised one?

Hi,

For my users I ask for their permission for the push notifications after they have loggedin.
In the callback I notify them that there is a new personal message by an alert.

But this is only shown on the Tab where I have the registerForPushNotifications. Is it possible to show this alert shown on all tabs, so no matter on what tab the user is the notification will be shown?
