Getting values of AlertDialog when using LOGIN_AND_PASSWORD_INPUT

Hi guys,

I've got a nice simple alert dialog which pops up when a button is tapped on iOS.

In that alert dialog Im using the LOGIN_AND_PASSWORD_INPUT style to include the login/password textfields as a way of logging into my app.

This all works fine, but the main issue is how do you actually get the data that is typed into the alert dialog fields?

I need to be able to pass these to my web service.

Here is the alert dialog code.

var logindialog = Ti.UI.createAlertDialog({
        cancel: 1,
        buttonNames: ['Login', 'Cancel'],
        message: 'Enter your details to continue',
        style: Ti.UI.iPhone.AlertDialogStyle.LOGIN_AND_PASSWORD_INPUT,
        title: 'Login to my App'
    });

Any tips would be much appreciated!

Simon

    Hello Simon Hume,

    Here i have added test code for getting login and password data. you can login and password input value in click event on logindialog.I hope this will help you.

    
var win1 = Ti.UI.createWindow({
    title : 'win1',
    backgroundColor : '#fff'
});

var logindialog = Ti.UI.createAlertDialog({
    cancel : 1,
    buttonNames : ['Login', 'Cancel'],
    message : 'Enter your details to continue',
    style : Ti.UI.iPhone.AlertDialogStyle.LOGIN_AND_PASSWORD_INPUT,
    title : 'Login to my App'
});

win1.addEventListener('click', function() {
    logindialog.show();
});

logindialog.addEventListener('click', function(e) {

    Ti.API.info('e.login ' + e.login + ' e.password ' + e.password);
});
win1.open();
