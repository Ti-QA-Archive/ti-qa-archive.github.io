Closing the top level window from a few windows in

How do you make it possible to close a window defined in the app.js from a few windows deep?

For example:

  • app.js (Successful Login Opens Window1)
  • —> Window1 (Opens Window2 which contains a SplitWindow)
  • ——-> Window2 / Contains SplitWindow
  • ————>MasterWin
  • ————>DetailWin
  • —————-> DetailWin=Attempted Logout.. (So we want to close Window1, and show Login Prompt again)

This issue is holding up our ability to logout of the split window scenario.

    you can set an event listener on window 1 in app,js called like doLogout then inside of DetailWin when they click the logout button, do a fireEvent of doLogout
