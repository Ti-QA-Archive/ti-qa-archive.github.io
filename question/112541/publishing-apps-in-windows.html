Publishing Apps in Windows

Hi there,

I am new to using Appcelerator, and have recently downloaded Titanium Developer for Windows. I have built a very basic app using Javascript, but when I go to test the app within Titanium Developer nothing happens. Am I missing something else that I need to install or is it even possible to test and compile any apps on a windows machine.Thanks in advance for any help.

  • Denis

    When you say, "built a very basic app using Javascript", did you start with a default Titanium project, as in the guide, Test a default project, and create it from there? Did you use any of the Titanium Mobile API? Have you launched it in an emulator yet?

    To answer your questions, you can test and compile only Android applications on Windows.

    To configure windows correctly for Titanium development, I would recommend you read the Installing on Windows 7 guide from the top.
