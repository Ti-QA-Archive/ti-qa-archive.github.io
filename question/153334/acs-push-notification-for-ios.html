ACS Push Notification for iOS

Hi

I am reading thru samples and docs but still can't really 'get it'
Previously, I am using Urban Airship for Push and it doesn't require registration of users.

But for ACS, in many examples, there is this user registration.
Is it necessary and why?

  • Accepted Answer

    ACS manages the devices token for you so if you have a user owning multiple devices, sending a push to that user will automatically deliver pushes to all the devices without you having to worry about it.

    However, we understand the concern people have with the mandary user login. In the upcoming release, we will provide some new methods that will let you use push without having to login any ACS users.
