Blackberry problem, app won't start

Hi!

I'm developing with Titanium Blackberry in 1.7.0. I can execute KitchenSinkBB, but i can't execute a new proyect. It allways throw an error like: Application terminated:Project has been denied the "Cross application communication"permission.

Is there any solution for it?

Thanks!

    I've run into this. You need to adjust the permissions granted to your application. To do so, tap the Blackberry key, go to Options, then Applications. Find your application in the list and tap on it and then select Allow where appropriate. Save your changes and it should work.
