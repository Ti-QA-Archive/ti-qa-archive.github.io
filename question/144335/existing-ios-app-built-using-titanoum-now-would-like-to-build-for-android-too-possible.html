Existing iOS App built using Titanoum, now would like to build for Android too. Possible?

Hi, I've built a fully functioning iOS App using Titanium 2.1.3 and everything works fine - have submitted it to the App Store without any issue.

Now, I'd like to add an Android build too. Am building on a Mac (ML 10.8.2), I have the android SDK installed. I'm able to push the app to the Android Emulator and can see the App Icon. Upon launch I get a red screen with the words "Error".

The .apk files are being built and instaleld on the emulator/device but can't be run.

Any advice from anyone on what should I be checking?

FYI, the App was earlier built as iPhone only.

  • Well… yes! that's the point.

    Of course, take into consideration that some controls that are available for iOS devices are not available on Android and vis versa.

    I recommend checking out Forging Titanium videos to see some cross platform development techniques. Also, since android and ios tend to act differently on cretonne situations, you might want to check the platform and decide which code to run based on the platform.

    I would start by running the project on an android device and see where it doesn't work and adjust what's needed.
