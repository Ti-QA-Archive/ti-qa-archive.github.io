  • Hi

    You need to create a toggle state. A normal button does not provide a toggle state, in the way a switch does - but if you need a button that does simulate a toggle state then it is not too hard.

    You will have to swap the visual style between pressed and not pressed - using background images or colours. You will swap these depending on the toggled state. You could then add a custom property to the button to remember the state and refer back when pressed.

    Very simple example;

    var btn = Ti.UI.createButton({
    backgroundColor: 'red',
    height: 50,
    pressed: false, // custom property
    width: 200
});
var btnToggle = function (e) {
    switch (e.source.pressed) {
    case true:
        // currently pressed
        // swap to not pressed 
        e.source.backgroundColor = 'red';
        // your code here
        break;
    case false:
        // currently not pressed
        // swap to pressed 
        e.source.backgroundColor = 'blue';
        // your code here
        break;
    }
    e.source.pressed = !e.source.pressed;
};
btn.addEventListener('click', btnToggle);

    This will simply swap the background colour of a button from red to blue. You can also add any code you need it to perform for each state in the blocks I left for you.
