Why can't I control modal window size?

I have an app where I want to open a modal window with instructions.

I can get the window to open, but it's always a set size even though I thought I was correctly setting the size

var wHelp = Ti.UI.createWindow({
    title : "How to Use This Calculator",
    width : 400,
    height : 200,
    barColor: '#803901',
    orientationModes : [Titanium.UI.LANDSCAPE_LEFT, Titanium.UI.LANDSCAPE_RIGHT],
    backgroundColor: '#fee78c'
});

tHelp = Ti.UI.createLabel({
    width: 'auto',
    height: 'auto',
    text: 'blah blah'
});

wHelp.add(tHelp);

winCalc.addEventListener('focus', function(e) {
    if(!seenIt) {
        wHelp.open({
            modal : true,
            modalTransitionStyle : Ti.UI.iPhone.MODAL_TRANSITION_STYLE_COVER_VERTICAL,
            modalStyle : Ti.UI.iPhone.MODAL_PRESENTATION_FORMSHEET
        });
        seenIt = true;
    }
});

Gives me the image below. The yellow area is the 400x200 I specified for the window size. What am I missing here?

SDK 1.7.2, iPad 4.x,OS 10.7.1
