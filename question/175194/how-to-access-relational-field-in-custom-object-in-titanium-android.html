  • I just had the same problem - solved it by removing the expression in the brackets that is returned by the query event. For example:
    var str = JSON.stringify(e.YOUR_OBJECT).replace('[CUSTOM_category]','');
    str = JSON.parse(str);

    Now "str" holds your JSON object and you can access categorydata object by using: str.categorydata

  • You can access a relational field like so:

    var customData = food["[CUSTOM_category]categorydata"][0].property;
