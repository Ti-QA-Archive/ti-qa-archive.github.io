  • I had this problem when I made reference to a file that had portuguese chars (ã and ç) in the path of the file.

    OperationalError: Could not decode to UTF-8 column 'path' with text 'E:\Documentos\Titanium 2.0\Formação Android\chart\Resources._.DS_Store'

    I found that the file

    C:\Documents and Settings\Administrador.X.deltafy\deltas

    was referencing that path. I closed everything, changed the name from deltas to deltas1 and relaunched the titanium and the emulator. A new deltas was built by titanium and the emulator started working again.

    Hope it helps

    Antonio Leite
