Alloy listView with children expandable

It there available an example of an alloy expandable multi section listView?

Something like:

<Alloy>
    <Window>
        <ListView onItemclick="onItemClick" >
            <ListSection headerTitle="Germany" >
                <ListItem title="Borkum"/>
                <ListItem title="Brake"/>
                <ListItem title="Bremen"/>
                <ListItem title="Bremerhaven"/>
            </ListSection>
        </ListView>
        <ListView onItemclick="onItemClick" >
            <ListSection headerTitle="Netherlands">
                <ListItem title="Huibertgat"/>
                <ListItem title="Schiemonnikoog"/>
            </ListSection>
        </ListView>
    </Window>
</Alloy>

Where the initial items shown would be "Germany" and "Netherlands" in list form and with a navigation arrow to the left of the each item indicating child items. Once the item was clicked the child list would be available and clickable. When this code is run only the second ListView is displayed.

    • Shuvankar, I ended up using a TableView rather than a ListView to get the child effect I wanted
      For my Project I used something like this:

      <Alloy>
    <Tab icon="001.png">
        <Window  id="winTvLocale" onOpening="onOpening" >
            <TableView id="listTvLocale" >
                <TableViewRow id="rowInfo" hasChild="true" onClick="openLocation01"><Label id="lblLocal">L('tabtitle_loc001')</Label></TableViewRow>
                <TableViewRow id="rowInfo" hasChild="true" onClick="openLocation02"><Label id="lblLocal">L('tabtitle_loc002')</Label></TableViewRow>
                <TableViewRow id="rowInfo" hasChild="true" onClick="openLocation03"><Label id="lblLocal">L('tabtitle_loc003')</Label></TableViewRow>
                <TableViewRow id="rowInfo" hasChild="true" onClick="openLocation04"><Label id="lblLocal">L('tabtitle_loc004')</Label></TableViewRow>
                <TableViewRow id="rowInfo" hasChild="true" onClick="openLocation05"><Label id="lblLocal">L('tabtitle_loc005')</Label></TableViewRow>
                <TableViewRow id="rowInfo" hasChild="true" onClick="openLocation06"><Label id="lblLocal">L('tabtitle_loc006')</Label></TableViewRow>
            </TableView>
        </Window>
    </Tab>
</Alloy>
