iTunes Sync App failed to install

Hey.
When i install ipa then i want to install that ipa on my device, but it gives me error on iPod device
"iTunes Sync MyAppName failed to install"
Please help me that how can i install that ipa on my device
Thanks.

  • Make sure your Application Id in tiap.xml matches the App Id in https://developer.apple.com/ios/manage/bundles/index.action Otherwise, it will give you this error.

    If your App Id has .*, then you shouldn't have this issue, but you will not be able to do push notifications and in app purchases.

    I would also recommend trying the install through XCode, and install builds into Testflight.
