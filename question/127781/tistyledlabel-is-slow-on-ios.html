Ti.StyledLabel is slow on iOS

Hi there

I'm facing some performance issues with the Tit+ module StyledLabel for iPhone. The module is working as expected, but rendering is painfully slow in the iPhone simulator and even much worse on the device(Takes about a minute until view is rendered with the code pasted below, same view without StyledLabel just with normal controls is very fast). For Android i'm using the same module(1.0) and it performs pretty well in the emulator.

  • Environement: Titanium Studio, build: 1.0.6.201110251616 on OSX Lion
  • Framework: 1.7.3
  • iOS: 4.3

Any hints? Anyone has experienced similar issues?

Code: 

var css = '<style>p { font-size:13px; font-family: "Helvetica Neue"; }'
var html = '<p>Hello this is a paragraph. I am <strong>bold</strong>. I am <em>italic</em> and i am a <a href="http://www.appcelerator.com">Link</a></p>';

var text = Ti.StyledLabel.createLabel(api.combine($$.ContentTextLabel,
{
   html: css + html, //item.value,
   width:$$.platformWidth - $$.ContentTextLabel.left - $$.ContentTextLabel.right,
   top: topMargin,
   filteredTags: ['p', 'strong', 'em', 'a'],
   filteredTagsMode : Ti.StyledLabel.INCLUDE_SPECIFIED_TAGS
}
));
parent.add(text); // parent is a ScrollView

Module include in tiapp.xml:

<module version="1.1" platform="iphone">ti.styledlabel</module>
