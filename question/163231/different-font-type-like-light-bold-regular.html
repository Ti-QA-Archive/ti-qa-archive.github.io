    Hi

    This can be a real problem as setting the postscript name works for some font families but not for all.

    It is worth taking a look at these links to see a work that is annoying but seemingly successful.

    • Trouble with font families
    • Using TTX to rename fonts

    By doing this the fonts themselves have unique names so you can specify them uniquely. Not ideal but it does work.

    One very important point, despite what some people here are saying - when you use a custom font you should set the weight to normal even if the font is bold. As leaving it as normal allows the font to provide the correct render.

    Try this :

    var isIpad = Ti.Platform.osname == 'ipad' ? true : false;

font: { 
    fontSize:40,
    fontFamily: isIpad ? 'Burnstown Dam' : (Ti.Platform.osname=="android") ?  'burnstown_dam' : 'Burnstown Dam',
}
