Getting 2015-02-11T20:07:05+0000 into a date object

So, in ACS we have a field updated_at which has value 2015-02-11T20:07:05+0000

Now i wish to turn this into a date object.
But using something like var d = new Date("2015-02-11T20:07:05+0000") gives NaN issues.

How do we turn this into a date obj?
