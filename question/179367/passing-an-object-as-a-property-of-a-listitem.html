Passing an object as a property of a listitem

When using tableviews, I often passed the row object as a tableview row custom property so that I would have access to it in the row object. Using Listviews now, I can get all of the individual properties set for my row templates, but I'm wondering if there is a way to create a custom property on the listview that I can assign the javascript object? Seems that you can only bind to view objects, not create a custom property and add a JS Object. Am I missing something?

Ray

  • It is as simple as just passing in your data as a custom property to the ListItem you create. You didn't give any example code, so I'll just paste some in:

            var row = {
            template:'someListItemTemplate',
            myCustomProperty:myCustomData,
            nameLabel:{
                text:myCustomData.fullName
            },
            company:{
                text:myCustomData.company
            },
        };
