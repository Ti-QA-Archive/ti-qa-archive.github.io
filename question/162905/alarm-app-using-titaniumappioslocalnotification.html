    • Yes. I have used it in a project to date, along with the AlarmManager for the Android implementation. The module takes the same parameters as the Titanium implementation for scheduling a local notification, so a repeat value can be set, like this:

      var now = new Date();

notify.scheduleLocalNotification({
    date: now,    
    alertAction: "Alarm",    
    alertBody: "Do Something!",
    userInfo: { "id": 1 },
    repeat: "daily"
});
