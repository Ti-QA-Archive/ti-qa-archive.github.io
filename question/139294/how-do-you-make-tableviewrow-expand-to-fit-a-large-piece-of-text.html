How do you make TableViewRow expand to fit a large piece of text??

I have a tableview, like this:

var data = [
    Ti.UI.createTableViewRow({title:'Title1', header:'' }),
    Ti.UI.createTableViewRow({title:"content1"}),
    Ti.UI.createTableViewRow({title:'Title2', header:''}),
    Ti.UI.createTableViewRow({title:content})
];

var tableView = Titanium.UI.createTableView({
    data:data,
    style:Titanium.UI.iPhone.TableViewStyle.GROUPED
});

QUESTION:

How do you make a row of the table expand to fit a large piece of text?
Each Content is a different size and currently only 1 line of the content is displayed in the row?
So is it possible to give the row an AUTO height, and dump the entire content into it?

Help me out guys!!

    Instead of using the title property, add a label to the TableViewRow. Set the height for this label and for the TableViewRow to Ti.UI.SIZE.

    • Hi thanks for the advice! But could you please show me how to add a label to the tableviewrow??

      — commented July 5th 2012 by Donal Lynch 

    • var row = Ti.UI.createTableViewRow({height:Ti.UI.SIZE});

var label = Ti.UI.createLabel({height:Ti.UI.SIZE, left:0, right:0, top:10, text: 'Title 1'});

row.add(label);

      then you can create an array of row objects like this.
