Flicking through screens/windows

Hi,
Developing an app which will have ten screens (windows), each screen will have various components (imagview, buttons, etc.) What is a good way of being able to flick through these 'screens', for example, on first screen, when user clicks start button, next screen appears. I would like these screens to be in their own separate js file.

Regards

  • Flicking through windows and clicking on a button to open a window are 2 very different methods of navigation, however the idea of creating and opening the windows doesn't change. Create your window code in a commonjs module, then listen for either the swipe event on the window or the click on a button and open the new window in the event listener for each action.
