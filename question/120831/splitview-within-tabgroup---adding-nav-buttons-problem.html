SplitView within TabGroup - adding nav buttons problem

I'm working on an iPad app which has a TabGroup as the main navigation and a SplitView inside it.

I only want one title bar at the top of this screen, and I want to be able to add left or right buttons to it. By default, this scenario shows two title bars, one each for the TabGroup and the SplitView.

I can hide the title bars on the SplitView panes with navBarHidden: true. Sweet. But then I can't find a way to add an buttons to the title bar on the TabGroup. Is there any way to do this?

Otherwise, I can add buttons to the SplitView pane title bars by setting rightNavButton/leftNavButton. Sweet. But then I can't find a way to hide the title bar of the TabGroup. If I don't set the title attribute when creating the tab, the title bar stays there and is just blank (plus it takes away the title from the tab at the bottom of the screen, which is also bad). Is there any way to do this?

A solution to either problem would be great - they both seem like they should be simple.

Thanks!
