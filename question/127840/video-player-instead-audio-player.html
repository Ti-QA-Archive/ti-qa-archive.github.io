  • Platform: IOS

    i used this code for the video player 

    
var activeMovie = Titanium.Media.createVideoPlayer({
    contentURL: "mp3_file_url",
    backgroundColor:'#fff',
    playing:false,
    movieControlMode:Titanium.Media.VIDEO_CONTROL_DEFAULT,
    scalingMode:Titanium.Media.VIDEO_SCALING_MODE_FILL,
    height: 30,

});

    this player offer us the stop and play button also the progresse bar (like a slider)…
    on the other side with the "Titanium.Media.createAudioPlayer" we must create all the player controls (Play, Stop, progress bar,…)

  • Technically, no, there shouldn't be an issue with you using the VideoPlayer for sound so long as it works for the sounds you need and you really want to make use of the existing controls. As this is an uncommon use case, though, I'd personally suggest you look into creating yourself a UI for the audioplayer.
