Center a View in Parent - SDK 2.0+

I've been trying to wrap my head around all the UI changes introduced in Ti Mobile SDK 2.0. Yes, I've been putting it off upgrading from 1.8.2 until now. :) In 2.0, my app looks like it got hit by a truck.

The first issue I'm grappling with is that I used to be able to center a view by setting left:0 and right:0 like this:

var logo = Ti.UI.createImageView({ 
    image:'ui/images/logo-ipad.png', 
    width:222, 
    height:66, 
    top:0, 
    left:0,
    right:0
});

Now in 2.0, the logo view is way off to the left. I tried center:{x:0,y:0} but that doesn't seem to do anything.

How do I center this view?

  • You can use the displayCaps property similar to the following :

    var logo = Ti.UI.createImageView({
    image: 'ui/images/logo-ipad.png',
    width: 222,
    height: 66,
    top: Ti.Platform.displayCaps.platformHeight/2 - 33,
    left: Ti.Platform.displayCaps.platformWidth/2 - 111
});

    Hope this helps!
