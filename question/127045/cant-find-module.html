Can't find Module

I get this error message:

can't find module com.0x82.testflight

I did include 

<modules>
        <module version="0.3">testflight</module>
    </modules>

in the tiapp.xml

Anyone has the same issue?

— asked October 19th 2011 by Leonardo Amigoni

  • You need to use the exact name of the module which is the name of the directory it is installed in. Try the below.: 

    <modules>
    <module version="0.3">com.0x82.testflight</module>
</modules>
