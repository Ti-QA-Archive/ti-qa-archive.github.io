calling Appcelerator from Android code

I want to build an Android application that can launch Accelerator code, i.e., native screen to call Accelerator code that will launch screens built in Javascript/HTML. How do I accomplish this bridge between java classes (Android) and Javascript (app.js) that Appcelerator is packaged in? Thanks.

  • Accepted Answer

    In theory, what you can do is to create a Titanium mobile application that manages the components that are now part of your native application. Managing it the other way (i.e. your native application that manages all and wraps Titanium) is close to impossible.

    Also doing it in the straight way poses a bunch of issues that need to be addressed. Primarily, you need to be able to factor out portions of the existing app and wrap them in what Titanium calls "native modules" that you can instantiate and invoke from you titanium app.

    See here:

    http://wiki.appcelerator.org/display/guides/Module+Developer+Guide+for+Android
    http://wiki.appcelerator.org/display/guides/Module+Developer+Guide+for+iOS

    for an introduction to native module development for Android an iOS respectively
