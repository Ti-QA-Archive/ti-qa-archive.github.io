Process Incoming push notification

Hi, I am implementing push notifications using titanium cloud. I am successfully able to receive the push notification in my callback method which is in my alloy.js. I am making a network call inside this call back. I wanted to know which is the best way to process the incoming push notification because the user can be on any screen of the app when the push notification is received.

Thanks.

  • you can use registerForPushNotifications of network namespace to process what you want.

    There are 3 methods.

    • Success - function invoked upon receiving a new push notification.
    • Error - function called when an error occurs during registration.
    • Callback - function called when the push registration is successfully completed.
