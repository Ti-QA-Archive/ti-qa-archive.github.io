  • Hi

    Judging for your member since date you are most likely trying this for the first time - you did not say either way.

    The easiest thing to do is uninstall everything and then start by running through the steps in this guide.

    • Setting up Titanium Studio

    It is most likely that you installed Java 1.7 (sometimes called 7) but Titanium requires 1.6 (sometimes called 6). Makes sure you download the correct one if you do it manually or better still use the Dashboard tab and Get Started button in studio when you start it up. It will walk you through the process of the main SDKs that you need and their dependants.

    It is worth the short term pain, which was likely caused the Java section. Once you have done this your first "Hello World" app will be running in a few minutes.

    Good luck.
