Android App stuck at loading screen with distribution build

I made an android app with titanium studio. I works like a charm as long as i debug on my device but as soon as i create an android market distribuition build with the built in functionality and install the application on my device the app gets stuck at loading screen.

i also verified my apk and the apk gets verified without any errors.

i am using Titanium Studio, build: 3.1.3.201309132423

my Device: Huawei Ascend p6 Android 4.2.2

android SDK Version selected for distribution build: 2.3.3

i also tried to pack the apk without js Minification but it changes nothing.

Any suggestions how i can solve this problem?
