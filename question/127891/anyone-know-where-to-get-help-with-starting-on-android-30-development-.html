  • I'm involved in a real project for the Samsung 10" tablet with android 3, and this is what I can tell you:

    • Titanium does not support specific android 3 API yet. But all the 2.x API is available and works fine.
    • This means that you can install the regular KitchenSink sample in an Android 3 without problems.
    • Forget work with the android simulator for version 3. Really. If you think the android emulator was slow, with v. 3 you will discover a new concept of "SLOW". In my macbook i7 8GB ram, the emulator used to launch pretty well. But, I have to insist one more time, the emulator at 1280x800 pixels is impossible to work.
    • Finally, I can't wait anymore to see the new android implementation with the V8 JavaScript machine. Bigger screen means bigger images, more controls and more resources and Rhino is absolutely not prepared for this, as I have verified. AFAIK, this will happen in a very short time. I hope!
