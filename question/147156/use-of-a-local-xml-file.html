    In which level are you storing local xml. If it is in Resource Directory, then you can find it as

var fileName = "local.xml";
var file = Ti.Filesystem.getFile(Ti.Filesystem.resourcesDirectory,fileName);
var xmltext = file.read().text;
var doc = Ti.XML.parseString(xmltext);


now you can read xml nodes.. :)
