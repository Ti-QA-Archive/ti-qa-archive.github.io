    Hi lal Mia,

    If you want to support bangla in your apps at first create a folder named fonts in resource folder then paste here bangla font like .ttf file and call this in a font properties where you want like this code.

    font : {
        fontSize : 20,
        fontFamily : 'Siyamrupali'
    },

    Here Siyamrupali is the font name.

  • Hi

    Make sure you have cleaned the project since you have started using the i18n folder and updated strings as these may not become available until you have.

    • Menu > Project > Clean > Choose Project > Confirm.

    There is an internationalisation guide if yo uhave not found it already.

    Be aware that the strings will ONLY display the correct language files when the device is set to use that language - this is not something you set, you simply react to the users choice of language.
