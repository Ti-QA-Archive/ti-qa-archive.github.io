Ti.UI.currentWindow == null

I've recently noticed that Ti.UI.currentWindow appears to return null. Here's a simple example:

app.js:

var win = Ti.UI.createWindow();
win.open();

Ti.include('foo.js');

foo.js:

var type = typeof(Ti.UI.currentWindow);

alert(type);

results in 'undefined'

details of the build:
Titanium SDK version: 2.0.2,
iPhone SDK version: 4.3

— asked June 17th 2012 by James White
