    Hi,

    If your application contains large amount of information, then its more preferable to us Sqllite database, because storing information in database will need less memory space comparing to using Titanium.App.Properties , because it always store information in preference memory . But speed wise Titanium.App.Properties mechanism is more faster.

    If you want to start with Sqllite then you can go through :
    https://wiki.appcelerator.org/display/guides/Working+with+a+SQLite+Database
