  • [WARN] :   The module ti.admob does not support the ABI: x86
[WARN] :   It only supports the following ABIs: armeabi, armeabi-v7a

    You are trying to load a library that does not support the x86 platform. Need to test with that binary on an ARM based platform OR get a version of the module built with x86 support.

    That seems pretty clear from the information you posted.
