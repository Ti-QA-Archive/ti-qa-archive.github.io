    • I 100% understand where you are coming from, and if you are seeing issues with the GA builds I completely encourage you to post here, and to file bugs in JIRA.

      The reason for my statement, is this: No matter how good a platform is, it will always have bugs, and so users will always need to search the Q&A for answers to issues they find. When questions are posted regarding ours, or another companies beta software that is released with the understanding that it has bugs, and is provided to test and develop for features that are still being built. The issue comes into play where users down the road are searching for an answer about 1.8 when its released and they find a bunch of things about the beta, it can cause confusion, especially for new users. We always want and encourage questions and bug tickets for released versions when an issue is found, but for the CI builds it just causes issues for other users.
