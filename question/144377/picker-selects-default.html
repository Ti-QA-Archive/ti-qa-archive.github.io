picker selects default

Hi,

I have the following code:

var picker = Titanium.UI.createPicker({
                bottom : 0
            });

            var data = [];

            var cities = eval('(' + this.responseText + ')');

            for (var i = 0; i < cities.length; i += 1) {
                data[i] = Titanium.UI.createPickerRow({
                    title : cities[i].title
                });
            }

            if(state1.value == '')
            {
                state1.value = cities[0].title;
            }

How can I make it so that the picker selects the input the user already selected?

I have a picker that is used to list all of the states. And the user already chose a state that is saved into the DB. I want it so that when he clicks on the picker field, he sees the picker with what he had already picked as a state selected. So if he had chosen New York, that should be selected in the picker by default… How can I do that?

    The picker should hold its value, unless it's part of a view that you're destroying and recreating.

    If you're creating a picker and need to set its value to a specific value, you should use Ti.UI.Picker.setSelectedRow().
