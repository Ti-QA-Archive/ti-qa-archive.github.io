Remembering the iOS Simulator Device in Titanium Studio

Hi all-

Is there a way to remember the iOS Simulator device last tested against in Titanium Studio for a project? Prior to the last version upgrade, and start of a new Alloy project, I was able to use XCode, start the iOS Simulator, choose an iPhone device type, and on the next run of the Titanium simulator for my project, it would consistently remember if I was running my iPhone 5 simulator versus an older, non Retina display. This seems to no longer work, and I was wondering if there is a setting I am forgetting or missing to remember device simulators from test to test run?

Talk soon
Chris

    Hi

    • Open your tiapp.xml in studio
    • In the visual side of the editor on the bottom left – find configure (below iPad, iPhone ,Android etc)
    • Expand the Platforms option from the left
    • Choose iOS
    • Second drop down on the right Default Display
    • Choose the one you want.

    Agreed – buried and difficult to find first time.
