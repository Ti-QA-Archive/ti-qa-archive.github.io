How to refresh the database in an iPhone App?

I have recently added about 6000 rows to an Sqlite table.
However, none of these newly added rows are showing up in the application.

I simply did a MySQL Export of a query and then imported the CSV file into Firefox SQLite Manager.
All of the newly added rows are showing correctly in SQLite Manager, but there is no sign of them in the iPhone Application.

If anyone has any suggestions I would like to hear them!!

Thanks…

    How did you add them? Through SQL Statements within the app or did you modify the SQL file directly?
    In this case you need to delete the app First. The DB File will be installed once upon First app Installation. no matter how often you call Ti.Database.install
