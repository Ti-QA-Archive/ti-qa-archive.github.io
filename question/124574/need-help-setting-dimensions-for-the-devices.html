    Balu

    If you want your objects to be precisely aligned, then I think you are using the right approach with pixels.

    However, you can also define relative dimensions with percentages, ie left: &#39;4%&#39;.

    That said, please be aware of ticket TIMOB-3363. So, while this issue is being fixed, you will need to implement a function that will calculate the relative widths based on the Ti.Platform.displayCaps.platformWidth and Ti.Platform.displayCaps.platformHeight values.

    Hope this helps
