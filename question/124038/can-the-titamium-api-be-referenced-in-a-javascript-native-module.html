Can the Titamium API be referenced in a Javascript Native Module

I am looking at creating an iPhone Javascript Native Module and wondered if the Titanium API is available for use with it?

I am working through the doc example on the Wiki where there is reference to Titanium in a native C module, but not mention of Titanium in the Javascript section.

  • Sorry but I have answered my own question and thought I should share it in case anyone else was wondering.

    I have tried the following functions in a native module and all worked fine, so I think the answer is yes Titanium API's can be used in the JS native modules.

    I have also been using TiStudio to develop the module code once the template has been created. It would be great if actually we could use Ti Studio to do all this (create and build JS native modules from scratch), it seems to me that it would be a great addition.

    exports.test = function(pt){
    return Titanium.Utils.base64encode(pt);    
};

exports.createWindow = function(){
    var win = Ti.UI.createWindow({backgroundColor:'red'});
    return win;    
};

exports.showWindow = function(){
    var win = Ti.UI.createWindow({backgroundColor:'green'});
    win.open();    
};
