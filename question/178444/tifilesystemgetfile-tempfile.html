Ti.Filesystem.getFile(), TempFile()

Why i can correctly use

var f = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory,'ti.png');
f.write(response.img);
$.Img.image = f;

and if i try to change 'ti.png' with a different name such as 'prove.png' I receive the following error ?

[ERROR] :  TiDrawableReference: (pool-3-thread-2) [821,415886] Problem opening stream from file prove.png: /data/data..../prove.png: open failed: ENOENT (No such file or directory)

I tryed to work with temporary file but doesn't work again:

var tempFile = Titanium.Filesystem.createTempFile();
tempFile.write(response.avatar);
$.Img.image = tempFile;

What's wrong???

  • If you read the documentation for getFile it states:

    Note that getFile does not create a file if one does not exist at the specified path. It returns a new File object referring to the specified file path. The application can create a file at that path by calling write or create a directory by calling createDirectory on the File object.

    So if it doesn't exist, you need to open it to write to.

    var f = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory,'prove.png');
f.open(Ti.Filesystem.MODE_WRITE);
f.write(response.img);

$.Img.image = f;
