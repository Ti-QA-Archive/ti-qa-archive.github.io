    I was having the same issue.

    I suspect it was something to do with my npm install (v0.10.0) based on:
    http://stackoverflow.com/questions/15357274/yeoman-yo-webapp-fails-with-typeerror

    I was unable to get the bower fix working so instead, I uninstalled npm:

    > sudo npm rm npm -g

    and then installed the previous release:
    http://nodejs.org/dist/v0.9.9/node-v0.9.9.pkg

    and that got me going!
