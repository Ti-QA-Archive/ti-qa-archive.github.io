How to create new user with image in ACS?

I have an imageView that contains an image user selected. I'm not sure how to upload that photo to the cloud when executing Cloud.Users.create, can someone show me an example to create a user with an image?

Thanks

— asked July 7th 2013 by Wentao Ji
  • acs
1 Comment

  • Hi,

    You can attach an image with the user. if you want to attach an existing image to the user , then give the ID of a photo in photo_id parameter while creating a user. And use photo parameter if you want to upload a new photo with the user.

    Please refer to the link for more details: http://docs.appcelerator.com/cloud/latest/#!/api/Users-method-create .

    Thanks.
