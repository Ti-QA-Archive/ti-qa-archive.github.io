  • You will need to use an old version of the SDK. I'm not sure of exact version, but 2.1-vintage. You can check the release notes on the docs site for each of the versions to see where the armv6 architecture was removed then use a version older than that. You'll also need an older version of Xcode to support the older architectures (again, I'm not sure of the exact version). Supporting armv6 and iOS4 was deprecated by Apple, then by us, quite a while back.
