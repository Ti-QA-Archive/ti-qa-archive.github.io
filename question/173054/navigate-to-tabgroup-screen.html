  • Without seeing some code, I will have to guess with the following:

    If you want to set an active tab after navigating to a window with a tab group displayed and then display another tab other than the first, use the following:

    myTabGroup.setActiveTab(3); //What ever the tab number is

    Alternatively you can use the variable of the tab

    myTabGroup.setActiveTab(tabThree); //What ever the name of the tab is

    If the above is not what you are looking for, post some code…
