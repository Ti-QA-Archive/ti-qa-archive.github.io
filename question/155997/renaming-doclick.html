Renaming doClick

I'm using alloy and I have multiple buttons with the same onClick="doClick". I was told by someone else to rename my doClicks so that I can put all the functions in the same index.js. How do I rename them?

  • You specify the doClick function at the XML Level. You can call it whatever you want. In the controller for that file, that is the .js file, you create a function that corresponds to that name. Alloy will let you know if you have referenced a function that you have not created in the controller.

    Having said this, you can add an onClick with a specific function name for each one of your buttons, as long as there's an actual function with that name inside your controller file.
