  • You dont need a book for it. Use PHP, its easy to build JSON output.

    <?php
$output = array();

$sql="select * from table";

$result=mysql_query($sql,$conn) or die(mysql_error());

while($row=mysql_fetch_assoc($result)):

$output[]=$row;

endwhile;

echo json_encode($output);

?>

    Thats all there is to it. Then just use the standard JSON.parse to get the elements. Something like

    var data=JSON.parse(this.responseText);

for(i=0;i<data.length;I++)
{

alert(data[i].fieldname_from_query);

}
