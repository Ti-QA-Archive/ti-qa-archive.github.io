    Hi, You can get event through tableview "delete" listener.

    
tableview.addEventListener('delete',function(e)
{
    alert(e.index); // return the index of row which is going to delete
    alert(e.rowData); // return the row object which is going to delete. 
});

    • There is an editable property for row, so when it value is false, user cannot delete the row, so for the row, n which you do not want delete event set that row editable to false.

      Here is an example of how to do

      var row = Ti.UI.createTableViewRow({
        className: 'CategoryRow',
        height: 60,
        editable:false
    });
