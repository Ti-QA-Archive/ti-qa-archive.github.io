calling a fuction from different .js file

how can i call a function which is in diferent .js file. For example i want to call a function on a button click named check function , which is in file parent.js And i m calling that function in page1.js and also passing some integer values to the check function. how can i do this.

  • You can also use application event listeners as Ti.App.addEventListener('<name-of-event>','<function-to-be-called>'); This approach will free you from including the file where you want to call the defined function from another .js file. :)

  • use include:

    Titanium.include("parent.js");

    at the beginning of page1.js, and then you'll have access to its functions and variables.
