    Now, more than anything, it is all about the use of the new format of the TiAppXML where you include this:

    <iphone>
 <fileTypes>
            <type>
                <name>PDF</name>
                <icon>pdf_type_icon.png</icon>
                <uti>com.adobe.pdf</uti>
                <owner>false</owner>
            </type>
        </fileTypes>
</iphone>

  • Based on the information found on Apple's Developer site, you would want to modify your declaration in the tiapp.xml to look more like the below code. Instead of referencing iphone you want to use ios - this is where you would then add your additions to the application plist. 

    <ios>
<dict>
   <key>CFBundleTypeName</key>
   <string>My File Format</string>
   <key>CFBundleTypeIconFiles</key>
       <array>
           <string>MySmallIcon.png</string>
           <string>MyLargeIcon.png</string>
       </array>
   <key>LSItemContentTypes</key>
       <array>
           <string>com.example.myformat</string>
       </array>
   <key>LSHandlerRank</key>
   <string>Owner</string>
</dict>
<ios>

    Hope this helps!
