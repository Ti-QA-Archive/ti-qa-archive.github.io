iOS push notification unsubscribe using Notification Center

Is it possible to detect changes to the Notification Center settings to subscribe and unsubscribe a user from push notifications. My app successfully registers the user to push notifications but even when the Notification center is turned off they still receive notifications.

Ideally I'd like to be able to detect when the notification center is turned off so that I can unsubscribe them them and visa versa.

I can't find plenty of documentation on subscribing but very little on unsubscribing.

  • This is actually done server-side. You must poll Apple's feedback service and ask for devices that should be removed from your push notifications.

    See the documentation

    If you don't regularly poll this service, Apple will stop allowing you to send push notifications.
