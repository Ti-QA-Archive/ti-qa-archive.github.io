OSX Desktop - Info calls not showing in console

I'm trying to get going on a desktop app in Titanium Studio 1.0.7 (had this problem in older versions of studio, too). Whenever I call a Titanium.API.info( "anything" ), all that happens is the following:

Evaluating JavaScript file at: /Library/Application Support/Titanium/modules/osx/tiui/1.0.0/ui.js

Nothing else. Am I doing something wrong?

    You need to check the module you have installed:
    It is shown as being listed here:
    &#x2F;Library&#x2F;Application Support&#x2F;Titanium&#x2F;modules&#x2F;osx&#x2F;tiui&#x2F;1.0.0&#x2F;ui.js

    That is causing the app to crash before it can reach your code.

    try removing the reference to the module in your tiapp.xml file and then check your code. Next step would be to contact the module developer directly and ask them if there is a problem with it.
