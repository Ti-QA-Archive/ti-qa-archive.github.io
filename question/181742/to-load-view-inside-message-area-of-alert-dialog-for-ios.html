    Hi Jeevan, as far as i know it is not possible to load another view inside the AlertDialog.

    However you can create a fully custom view which can be modified further according to your needs & to create such a view you could do something like this:

    • create a new window with backgroundColor="#80000000" (first 2 digits are to set the transparency of window) & do not set the layout property(leave it to composite by default)
    • now create a view inside this window & set the height & width property as you need.
    • at the bottom of this view, you can add as many buttons as you want
    • to remove this window, set a click listener on window to close it.
    • make sure that the view's bubbleParent property is set to false in order to close the window whenever the user touch outside the view.

    Now you can do further steps to make your alert dialog fully customized by just using simple views & properties.

    If you need some sample code, then let me know if i could provide you by extracting some time for you :)

    Good luck!

    • Hi Himanshu, put the below files in respective folders :

      1- forgotpassword.xml

      <Alloy>
    <Window id="forgotPW" fullscreen="true" backgroundColor="#80000000" width="100%" height="100%" onPostlayout="showView" onClick="close">
        <View opacity="0" width="90%" id="fpwView" height="Ti.UI.SIZE" layout="vertical" bubbleParent="false">
            <Label width="90%" height="Ti.UI.SIZE" top="20" text="Please enter your Email Address" />

            <TextField hintText="Email address..." id="formValue" width="90%" height="35"></TextField>

            <View width="90%" height="Ti.UI.SIZE" top="20" bottom="20">
                <Button title="Proceed" onClick="proceed" id="buttonProceed"></Button>
                <Button title="Cancel" onClick="close"></Button>
            </View>
        </View>
    </Window>
</Alloy>

      3- forgotpassword.js

      var pointY = 0;

function showView(e){
    if(Ti.Platform.osname == "android"){ 
        $.fpwView.setOpacity(1);
    }else{
       var viewH = $.fpwView.getRect().height;
        $.fpwView.setTop( -viewH );

        var winH = e.source.getRect().height;

        pointY = Math.floor( (winH-viewH)/2 );

        $.fpwView.animate({
           duration : 350,
           top : pointY+20
        },function(){
             $.fpwView.animate({
               duration : 300,
               top : pointY-20
            },function(){
                $.fpwView.animate({
                   duration : 200,
                   top : pointY
                });
            });   
        });
        $.fpwView.setOpacity(1); 
    }
}


function proceed(e){
    close();
}

function close(){
    if(Ti.Platform.osname == "android"){ 
        $.forgotPW.close();
    }else{
        var width = $.fpwView.getRect().width;

        var m = Ti.UI.create2DMatrix();
            m = m.rotate(-90);

        $.fpwView.animate({
           duration : 500,
           transform : m,
           left : -width
        },function(){
            $.forgotPW.close();
        });
    }   
}


function showDialog(msg){
    var dialog = Ti.UI.createAlertDialog({
                message: ""+msg,
                ok: 'OK',
                title: 'Himanshu'
    });                              
    dialog.show();
}

      Now you have to do only one thing : call this line of code from wherever you want to open the custom alert dialog

      Alloy.createController('forgotpassword').getView().open();

      & see the magic of opening a new window to show the dialog with a beautiful animation

      Note that you have to add an extra view to show the separator line of the required color
