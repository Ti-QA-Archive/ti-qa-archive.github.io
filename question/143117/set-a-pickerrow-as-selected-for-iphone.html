Set a PickerRow as selected for iPhone

I'm using PickerView in my app. I have to set a predefined value as selected row but requirement is that user can change it by picking another value from picker. How it could be done?

    HI, I think this code could be helpful for you.

    picker.addEventListener('change',function(e){

    picker.setSelectedRow(e.columnIndex,e.rowIndex,false);

    });
