Locking screen aborts file upload (1.8.0.1)

Here's logs from the iPhone:

<Debug>: ALS: [AppleTSL2581] setPropertiesGated: requested ReportInterval=0 turn OFF sensor
<Notice>: MultitouchHID: detection mode: 0->255
<Notice>: Posting 'com.apple.iokit.hid.displayStatus' notifyState=1

2012-02-14T18:47:04+00:00 heroku[router]: Error H12 (Request timeout) -> POST [url].herokuapp.com/image/new dyno=web.1 queue= wait= service=30000ms status=503 bytes=0

If I go into the background, or switch apps, upload continues on both 3Ga nd WiFi. If anybody has this experience or knows how to fix it, please advise – thanks!
