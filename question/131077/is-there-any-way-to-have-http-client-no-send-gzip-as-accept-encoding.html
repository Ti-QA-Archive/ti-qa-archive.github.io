Is there any way to have HTTP Client no send GZip as Accept-Encoding?

Hi
I'm working on a HTTP Client (on iOS) and I've realized that no matter what I set the Accept-Encoding header to, Titanium will always send the Accept-Encoding to gzip.
Is there any way of changing this as the default behaviour? I tried setting it to an empty string..

Thanks in advance for any help

    xhr.setRequestHeader('Accept-Encoding', 'yourvalue');
