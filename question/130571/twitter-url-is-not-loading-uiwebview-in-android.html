  • Hello,

    you can fix it setting the user agent:

    
var win=Ti.UI.createWindow({
    backgroundColor:'white',
});
var wv = Ti.UI.createWebView();
wv.setUserAgent("custom user agent");
Ti.API.info(wv.getUserAgent());
wv.setUrl('https://mobile.twitter.com');

win.add(wv);
win.open();

    Best,

    Mauro
