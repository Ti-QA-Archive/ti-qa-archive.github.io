Database not installing

I am trying to install a sqlist database in my app but for some reason, the database isn't being copied over as far as I can tell. I'm reusing code that works in another app but everything I try doesn't work. Can someone take a look and tell me what I'm missing?

The database is in the correct folder and the name is correct. I'm at a loss.

var db = Ti.Database.install('/database/Trivia.db', 'triviaDB');
var triviaDB = Titanium.Database.open('triviaDB');

var sql = "SELECT count(name) as mycount FROM sqlite_master WHERE type='table' ORDER BY name";
var gameSearch = db.execute(sql);

// Should return 4.  
alert(gameSearch.fieldByName('mycount'));

  • Database.install() only installs the database if it does not already exist. I assume you ran that code before your data was final, and ran it again expecting a different result.

    Delete the app from the simulator/emulator/device, delete the build directory, and try running it again.
