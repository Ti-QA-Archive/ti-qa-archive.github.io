Separators on empty rows

How can I avoid separators on empty rows?

I know that I can create a tableview with separatorStyle NONE and add a border to every row. But if the row has child mark the border will not be painted to the whole width of the row.

Any ideas?

    Sven,

    If you add a table footer it will rid the empty rows. For example:

    var tblFooterView = Ti.UI.createView({
  width: 300,
  height: 1,
  backgroundColor: '#FFF'
});

tblView.footerView = tblFooterView;

    Let me know if you need anything else.
