How to close all parent windows of an alloy navgroup ?

Hi:

I have a navgroup that has 2 detail level windows. From ROOT Window I open window STATES and from sstates i open CITIES … The problem is that when I click a object in CITIES I want to return to the root window and I can't close STATES… so I have to go back from STATES to ROOT.

I use this code to open detail (STATES / CITIES) windows:

    Alloy.CFG.navgroup.open(Alloy.createController('states').getView());
// or cities 
    Alloy.CFG.navgroup.open(Alloy.createController('cities',{state_id:id}).getView());

I use this code to close CITIES when I click object.

        $.citiesTableView.addEventListener('click',function(e){
            Alloy.CFG.navgroup.close($.winCities);
            }
        );

I couldn't close states window by doing ~ Alloy.CFG.navgroup.close($.winStates);-~ … So I'm stuck at this…

At STATES level I tried to obtain the Titanium.UI.currentWindow but it always returns undefined. Also tried Titanium.UI.getCurrentWindow and same problem.

Anyone helps?

Regards
Joao

  • I had a similar problem and what I did to get around it was to add the window references to an array as they are created. Then, on your table view click event, you can loop through the collection and pass each to the navigation group close method. This may not be the ideal solution, but it worked for me.
