  • Check that the JAVA_HOME environment variable is set up and pointing to the base installation of your java JDK. Also, make sure you have installed the 32bit version and if you are using windows make sure you install the JDK to a path without spaces in it. eg C:\java

  • I get this error from time to time on my Windows 7 pc. I find that if I reboot my pc the error goes away the next time I start Titanium Studio. So if Java is installed correctly, and your environment variables are also set correctly, try a reboot!
