Remove HTML from a string

  • Hi if you want to remove Html text from string then you can use this function

    function convertHtmlToString(stringtoReplace){
var cleanText = "";
if(stringtoReplace!= null && stringtoReplace != undefined){
     cleanText = stringtoReplace.replace(/<\/?[^>]+(>|$)/g, "");
}
return cleanText;

}

    You can call the above function to get the text from html.
