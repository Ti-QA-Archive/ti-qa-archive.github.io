Can't Get Titanium up and running for Android!

Forgive me I'm new at this….

So I downloaded Titanium today for the third time, when through the process on installing the Android SDK, so that I'm getting the green light on the getting started page (my third time). I created a default Alloy project and tried to run in for the Android emulator and it bombs!

Looking back over the output in the console window the only error message I see is:

error importing function definition for `__rvm_date'

which comes after a call to builder.py, I've no knowledge of Python and have no idea how to go on from here! I might add that the exact same project runs just fine on the iPhone emulator…

My set up details are as follows:

  • Max OSX version 10.8.4
  • Android SDK latest download (set to 2.3.3 in Titanium)
  • Titanium SDK 3.1.2 (also tried 3.1.1)
  • Python 2.7.2

I should also add the the Android SDK is working fine for a native 'hello' program, so I really fell the problem is in the Python stuff, but what??

    I was able to fix this. Although I don't understand why this Problem occurred only now since I install RVM a few days ago. I also don't see why Titanium would be using anything from Ruby but maybe there is a path issue where something is also called RVM.

    I removed RVM from my OSX system by issuing the following:

    osx$ rvm implode
