barImage for "More" Window?

Hi there.

I currently have a "barImage" assigned to all my windows except for the "More" Window.

I tried adding it to the createTabGroup like here - but that didn't work:

var tabGroup = Titanium.UI.createTabGroup(
{
    barColor:'#cc0000',
    allowUserCustomization:false,
    barImage: '../images/HWC/common_header.png'
});

I'm guessing that if I can't add a custom barImage to the "More" Window, I'll need to create a custom tab called "More" with a custom Window, etc.
