Is there a way to get Android.R resources?

Hello, i'm trying to use native Android resources in my Titanium application. F.e. i need to add Ti.Android.R.drawable.checkbox_on_background as leftImage to TableRowView. How i can do this?

Ti.Android.currentActivity.getString(Ti.Android.R.drawable.checkbox_on_background); gives me following path: res/drawable-hdpi/checkbox_on_background.png, but it's relative to application Resource directory and because of this will not work.

  • if your problem is the relativity, then you can add the static path (until the folder /res/) before the Ti.Android.currentActivity.getString(Ti.Android.R.drawable.checkbox_on_background); command.

    E.G. path='c:/myPathHere/'+Ti.Android.currentActivity.getString(Ti.Android.R.drawable.checkbox_on_background);

    this will return —> c:/myPathHere/res/drawable-hdpi/checkbox_on_background.png (full path which is what you need)
