Alloy: how to set properties in a dynamic way?

Hi, I'm trying to fugure out how Alloy works so excuse me if this answer could be silly. How can I set a property of an object, i.e. a View, via js code? The index.tss file contains the properties of ui elements like backgroundColor, top, width and so on. But if I want to set the dimension of an element in a calculated mode how can I do this? Thanks for the attention.

    Hi

    Not a silly question. Here is a quick test case to show you how;

    index.xml

    <Alloy>
  <Window id="win">
     <View id="test">
       <Label id="lbl" class="cool" text="hello world" />
     </View>
   </Window>
</Alloy>

    index.tss

    "Window"; {
  backgroundColor: "#fff",
  height: Ti.UI.FILL,
  width: Ti.UI.FILL
},
"#test"; {
  backgroundColor: "#ccc",
  height: Ti.UI.FILL,
  width: Ti.UI.FILL
},
".cool"; {
  backgroundColor: "#000",
  height: Ti.UI.SIZE,
  width: Ti.UI.SIZE
}

    index.js

    $.lbl.setColor('#c60000');
$.lbl.setWidth(2*100);

$.win.open();

    Here we set the main object Window, an id test and a class cool in the TSS. In the controller we use $ to access the id properties of the object inside the Alloy view XML. From here we can use any property or method in the documentation for each specific object.
