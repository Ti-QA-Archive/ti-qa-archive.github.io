How to set toast notification position?

Firstly, Notification can only be used in Android, right? In Android, we can set Toast position. How to set it in Titanium? I want to place some notification message at the top of the screen instead of at the near bottom. — Thanks

    Hello Born,

    Try this :

    var toast = Ti.UI.createNotification({
    message:"Toast Notification Message",
    duration: Ti.UI.NOTIFICATION_DURATION_LONG,
    offsetX: 50,
    offsetY: 50,
    gravity: 48 | 3
});
toast.show();

    i hope this helps. thanks.
