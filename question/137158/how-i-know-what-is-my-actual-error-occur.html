I build a application in Android & iphone both it work fine. but some time crash in Android and flash the message

    for iPhone : Debug your project in XCode. You can find your error in debug window.

    for android : Start DDMS from 

    AndroidSDK_path&gt;tools&gt;DDMS

    Connect your device and select your device from left top corner in DDMS. It shows your device status in the bottom window. see error logs in bottom window you can find the error message red by default.
