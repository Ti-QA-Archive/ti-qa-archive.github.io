How do I send a message to objective-C class with paramaters and return result

I just need a simple example on how to send a string or webview to an module and return the result. I already have managed to create a module. I know i can message a method but i can't manage how to send a parameter and return it back.
I'm new to objective C but here is my simple testing method i can call within my javascript module

-(id)tester:(id)val
{
    //NSString *myString = @"testing";
    return val;        
}

If i wanted to return myString, i can easily uncomment that code and return that string. But my val parameter i sent to it returns null and breaks my app.
My javascript looks like this.

var myModule = require('com.moduleTest');
var foo = myModule.tester("I want to be able to use this parameter in my tester method");

Can someone please direct me to a simple example of how this is done?

Thanks.

    I finally figured it out. I posted this question as a cry for help and decided to re explore everything on this site once more. I found an app on github and found out my solution
    I replace myString comment with this one. 

    NSString *myString = [TiUtils stringValue:val_];

    The string is converted and returns to my javascript function.
    It makes so much sense that this must be done but i assumed appcelerator would do this for me. Anyways.. Hope this helps someone else saves the hours I wish i had now.
