How to append string values in a text area

I've got a text area with a value property in my tss file…how do I append the string values in the value property…like so that is from my i18n strings.xml file (using the L shorthand): 

top: 8,
width: 80, 
height : 70,
editable: false,
value: L(software) + L(other_text),

    You will have to do this in the controller.

    ####Like so:

    $.myTextArea.value = L(software) + L(other_text);
