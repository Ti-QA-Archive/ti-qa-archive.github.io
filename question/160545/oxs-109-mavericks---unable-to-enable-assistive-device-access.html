OXS 10.9 Mavericks - Unable to enable assistive device access

Hi,

I have just got a new mac which has mavericks pre installed, and now I've come to releasing a new version of my application for the iOS Store and I get the "Enable access for assistive devices" error message when Xcode loads up.

For anybody who is looking, the option is not in the same place as it was in previous OSX releases, it is now under System Preferences > Security and Privacy > Privacy.

You then get the option to enable access to specific apps.

However, titanium studio, or Xcode, are not in the list. I only have "System Events" and checking this does not enable access for building iOS apps.

My previous mac was upgraded from OSX 10.8 to 10.9, I guess because it was already enabled for all applications on 10.8 I never had the issue. But on my new mac I can only enable this per app.

Has anybody had any success enabling this option when installing titanium studio onto a mac which is already on mavericks? Its put a bit of a dead stop to my release schedule…

Thanks
