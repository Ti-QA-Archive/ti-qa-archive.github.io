Read Mifare using NFC

Ti SDK: 3.1.2.GA

Hello

I am new to NFC but have been working through the examples for the Ti.Nfc module along with referencing the Android Nfc docs.
Using the TagViewer example, I'm trying to get the app to launch when a Mifare 1K card is tapped but am having no success. (I have success using the TagForeground example.)
Reading the Android Nfc docs I see that you have to add the following:

<activity android:name=".Tagviewer" android:label="TagViewer">
<!– Add a technology filter –>
<intent-filter>
<action android:name="android.nfc.action.TECH_DISCOVERED" />
</intent-filter>

 <meta-data android:name="android.nfc.action.TECH_DISCOVERED"
     android:resource="@xml/filter_nfc"
 />

</activity>

Which I have added to the project tiapp.xml file.
I also created the filter_nfc.xml file but don't know where this should go. I have tried various places but always get a compile error - file not found!

Does anyone know how to launch an app using Mifare tags?

Many thanks in advance,
Vic

    The filter_nfc.xml file lives in res/xml.

    so in a classic titanium config it will go in {project_root}/platform/android/res/xml
