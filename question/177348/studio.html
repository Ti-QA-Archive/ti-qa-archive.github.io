  • Hy, Roger, Thanks for your Query, This problem has occurred to me and many others, and there are different solutions to resolve this problem. Please try to perform these solutions below, one of them should work for you.

    1. Try Project > Clean to clean you project build, and then try again.
      • Go to Window > Preferences > General > Editors > File Associations
    • Select the "*.js" file extension on the file types window and "JavaScript Source Editor" in the Associated editors window. Set it as the default editor.
    • Close the js files that were already open and reopen again.

    1. Try going to Titanium Studio preferences, typing the words "assist" and "complet" (short for "code assist" and "code completion") into the filter at the top, and going through each config screen in the results and clicking on "Restore Defaults" and "Apply". Then restart Titanium Studio.

    2. create a new, blank, project, and copy into its resources folder the code from the resource folder of an existing project that does not work. Again, you may need to restart TiStudio.

    Hope you find your solution. thanks.
