  • Use the ip address of your localhost, and if you have a port number other than port 80 configured for your server.

    A few examples:

    
xhr.open("GET","http://192.168.1.10/test.php");

    If you are using a MAMP server which doesn't use port 80 out of the box:

    
xhr.open("GET","http://your_servers_ip_address:8888/test.php");
