Cannot see cursor in a textArea using SDK 3.1.3 iOS7

I have a text area created like so…

// Create the text area
    textArea = Titanium.UI.createTextArea({
        editable : true,
        top : 0,
        left : 0,
        value : note,
        height : '100%',
        width : '100%',
        font : {
            fontSize : 19,
            fontFamily : 'Marker Felt'
        },
        color : '#5c0000',//'#181818',
        backgroundColor : '#ffffff',//'transparent',
        textAlign : 'left',
        suppressReturn : false,
        tintColor: '#5c0000',
    });

    view.add(textArea);

    textArea.addEventListener('textAreaEvent', function() {
            text.focus();
    });

    textArea.fireEvent('textAreaEvent');

However I cannot see a cursor. The text area lets me click on it and it displays a keyboard but I still see no cursor.

Can anyone help me out?

Thanks,
George

  • The Markup is correct here apart from the event not targeting 'textArea' but 'text' instead.

    The cursor does now appear.
