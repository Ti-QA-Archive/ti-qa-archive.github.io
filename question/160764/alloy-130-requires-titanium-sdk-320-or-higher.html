Alloy 1.3.0 requires Titanium SDK 3.2.0 or higher

Hi Everyone.

I'm getting the above error when trying to deploy to the android emulator.

Here's the full story.

When I opened Titanium this morning it said that updates were available. I duly allowed the updates to install and from then on I was unable to run my project in the emulator. It kept telling me it couldn't find the Android SDK's (I have ALL the Android platforms installed). It kept saying that Android Platform 5.0 was required.
