Adding views to alloy widget drawermenu - Titanium Appcelerator Alloy

Maybe someone of you guys already worked with the alloy widget drawermenu (https://github.com/ricardoalcocer/alloy-widget-drawermenu) and can help me. I'm trying to add another View to the Project and added a helpview.xml to the views folder. In the menu i have a point called Help.

Inside the menuview.xml:

<TableViewRow class="tableRow" id="row1">
    <View id="rowContainer">

    </View>
</TableViewRow>

This is what the helpview.xml looks like:

<Alloy>
    <View id="helpView">

controls.js:

exports.getHelpView = function(){ return Alloy.createController('helpview'); };
and finally the index.js, unfortunately it doesn't change view and i don't know why. can someone help me?

var helpView = controls.getHelpView();

    do you want to replace the whole mainview with this helpview one ? if so

    you have to add helpView to the drawermainview
    $.drawermenu.drawermainview.add(helpView.getView());

    and don't forget your tss file (helpView.tss) to properly style your menu and the main view

    but if you want to change the content of the mainview by clicking the rows you have to do this inside the eventlistener 

    menuView.menuTable.addEventListener('click',function(e){
    $.drawermenu.showhidemenu();
    if(e.rowData.id == "row1")
        // add/remove or do anything if the help row was clicked    
})
