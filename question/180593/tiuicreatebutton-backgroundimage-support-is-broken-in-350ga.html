  • Yes, I am developing on Mac OS X 10.9.5, with Titanium Studio 3.4.1.

    The code is pretty simple:

    var btn = Ti.UI.createButton({
  backgroundImage: '/images/about.png',
  width:150,
  height: 40
});

    while the button images is placed in the following directory for Android:

    ~/Resources/android/images/res-hdpi/about.png
~/Resources/android/images/res-mdpi/about.png
~/Resources/android/images/res-xhdpi/about.png

    and for iOS:

    ~/Resources/iphone/images/about.png

    Hope this help.
