Density specific graphics for Android Map Annotations

Is it possible to provide density specific graphics for android map annotations (the pins)?

Rather than going down the 'auto scale route' ie

<manifest>
<supports-screens android:anyDensity="false"/>
</manifest>

I've set all my measurements in dp's and provided a set of graphics for high, medium and low. Trouble is on a large screen my pin graphic will only work if it's in a generic folder ie images as opposed to an android specific folder like android>images>high/low/medium. The net result is very tiny pins on a big big map.

Any advice on this would be appreciated.

  • Hi Armindo, I think I solved it in the following way - its hard to be sure, I haven't worked on it for a while.

    Basically it wouldn't automatically pick the image from the correct resolution folder, so I just check the device's dpi and pick the image based on that.

    I hope this helps,
    Leigh

    var dpi = Titanium.Platform.displayCaps.dpi;


if (dpi >= 240){
    marker.pinImage = "../images/pinpurple_high.png";
}else{
    marker.pinImage = "../images/pinpurple.png";
}
