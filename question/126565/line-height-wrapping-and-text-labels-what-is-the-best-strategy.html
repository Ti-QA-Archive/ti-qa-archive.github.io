Line Height, wrapping and text labels. What is the best strategy?

var titleLabel = Ti.UI.createLabel({
text:data.titletxt,
font:{fontSize:14,fontFamily:"BorisBlackBloxx"},
color:'#000000',
top:34,
left:15,
height:'auto',
width:'auto'
});

What is the best way to wrap text and control the space between the lines? Is the only method to build a custom one that just creates multiple labels representing each line and space them yourself? Thanks

    The Label itself does not have properties to control leading (line spacing). The best way to get full control over the styling of a Label is to use the StyledLabel module and roll your own CSS.

  • Unfortunately there is no way to adjust spacing in labels with Titanium and there is no spacing parameter in UILabel natively in iOS either.

    • The shortest answer is to simply create two Ti.UI.Labels and space them whatever you need.

    • Whenever you have more than 2 labels, separate tokens and assign single labels each of them.

    • Use third party libraries to embed it as a module i.e. https://github.com/Tuszy/MTLabel

    See along the features adds line spacing.
