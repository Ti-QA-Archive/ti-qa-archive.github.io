Copy editor color preferences between workspaces?

Is there any way to copy editor color preferences from one workspace to another, like you can with key bindings?

I keep separate workspaces for test projects vs. active development projects, and I've made some editor color preference changes that I'd like to be the same in both workspaces. I'd also like to be able to copy these color preferences from my desktop PC at work to my laptop at home.

    Under Preferences/Titanium Studio/Themes you can export your editor customizations. See if that doesn't fit your needs.

    • Yea! That worked like a charm. Ironic thing is, that's the screen I used to make the color changes…I never noticed the Import/Export buttons up there.
