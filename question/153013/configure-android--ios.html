  • Hi

    If you have not already created the app;

    • From Studio
    • File > New > Mobile Projectt
    • Choose Template, then Next
    • Select your platform in the Deployment Targets question

    If you have already created the app and now want to change or verify your choices

    • From Studio
    • Select your app from the left side Project Explorer
    • Expand the tree-view listing by selecting the triangle to the left of the name
    • Double click tiapp.xml
    • Select your platform in the Deployment Targets question
    • Save and close the file
    • Menu ? Project > Clean > (select your app) and confirm (make sure all emulators, simulators and debug physical devices are no longer running)

    Done

    • You appear to have found the deployments section now which is good. Now we need to work through the disabled options.

      Titanium itself is a SDK that works with the platforms you mentioned. You need to install support for those so titanium can see them. This means downloading and configuring the target platforms SDKs themselves. Good news titanium helps you do this.

      Before we get there I suspect the lack of iPhone/iPad means you are using either windows or Linux as your development machine. If this is true then you will NOT be able to develop for iOS as apple development requires the use of Xcode in the background which is only available on the mac. There is also a licence restriction by apple only allowing dev via a mac, but as Xcode cannot be run without one it is hard to break the licence.

      You can learn all about what you need in the compatability matrix.

      To install the other platform SDKs you wish to work with;

      • in studio, select the home tab
      • choose the getting started option
      • click your platform choice
      • choose the setup option that appears to the right
      • follow the instructions

      When you have completed the process this page will confirm that and your new platform choice will appear as you have now found it.

    • One VERY important note.

      You are using Windows, however you can ONLY create iOS apps of any sort using a mac as the desktop OS. This is a restriction enforced by Apple due to two reasons;

      • xcode is used behind the scenes to compile the app and to push apps to the AppStore - this is ONLY available on OSX
      • Apple ONLY allows development to be done on a mac for iOS apps, which given the first point is mute as it is not actually possible to compile without XCODE for OSX

      You can find out more about this in the Compatibility Matrix.
