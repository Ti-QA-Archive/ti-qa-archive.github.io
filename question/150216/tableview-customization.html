TableView customization

I have two customization questions relating to my TableView. The first question I have is how do you create a TableView that does not scroll, but the window is capable to scroll and reveal the rest of the table? Right now I have disabled the scroll on the table, but I am unable to have the window scroll to reveal the hidden aspects of the table. Is there a better way to do this? My second question is what is the best way to customize the headers for my tableView rows? The following is my code:

  • hi,

    if you do not want to scroll tableview then you should use scrollview to scroll to reveal the hidden aspects of the table.

    and to create a header , you can create a array for week like var arrWeek = [&#39;Sunday&#39;,&#39;Monday&#39;,.....];

    and if your data object has a only _special values then you can use for loop to create a row.

    var arrWeek = ['Sunday','Monday',.....];

var c = 0;

for(var key in data){

var specialsRow = Titanium.UI.createTableViewRow({
    title: data[key],
    height:60,
    header: arrWeek[c],
    font: {
                fontSize: 10,
                fontWeight: 'normal'
            }
});

barData.push(specialsMon);
c++;

}
barsTable.setData(barData);
