  • I have been wanting to know how to trigger the "back" button event which clears one window from the stack myself but I can't figure that out either.

    If you have a stack for example win1, win2, win3, you can just call win3.close() and win2.close() and that will remove them from the stack and you'll be back at win1. but it isn't quite the same as whatever the back button is doing and the visual animation is a little different on the iPhone.

    you are also supposed to be able to call tab.remove(win3) which works on the iPhone but not on the Android.

  • I did this by storing an array stack of the opened windows in app.js

    Do this using a custom event passing a reference to the window you are opening and 'push'ing it on to the array.

    Then when you want to close all windows, you can send a custom 'close_all_windows' event back to app.js and loop through the window array stack to .close() them and 'pop' them off the stack.

    That will get you back to your tab 'home'.

    (Make sure that when the user actually closes a window manually with the back button, that you also create and send another custom event to individually pop the window references off the stack too.)
