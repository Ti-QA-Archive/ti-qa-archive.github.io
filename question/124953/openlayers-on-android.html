Does anyone know if it's possible to use openlayers javascript library in TI mobile? I've tried an include of the OpenLayers.js and I get memory problems on the emulator. I don't want to use openlayers in a webview, that works ok but is sluggish. I would like to use openlayers natively if possible. Thanks in advance!

  • I worked with openlayers in a webview only. It's a pain.
    To make it work in Ti natively you would need to heavy modify it to use imageviews instead of DOM elements.
    Is doable but there is much work to do.
    I'm still concerned on the performance though, because android is pretty stupid when is about images (on the load part, you might end with sluggish movements too) and how the scrollview control has only one direction to scroll ( horizontal OR vertical, not both) you would have to control yourself the touchmove part.
