Enable / Disable a button is not working

Hi !

I try to disable a button in order to not call the event. But this not working …

I have something like this :

button.addEventListener("touchstart", function() { 
alert("touchstart"); 
...
}

and further in code :

button.setEnabled(false); // Of course my function is still called when i click the button and I have my alert "touchstart"
button.setOpacity(0.5); // This is working on my button, it is grayed out !

I don't get it. TY

  • button = Ti.UI.createButton({
    //your parameter
    id : 0
});

button.addEventListener("touchstart", yourFunction);

function yourFunction(e){
    alert("touchstart");
    ///....
    e.id += 1;
}

//....
for (i = 0; i <= button.id; i++){
    button.removeEventListener("touchstart", yourFunction);
}

    I did something like that, it worked
