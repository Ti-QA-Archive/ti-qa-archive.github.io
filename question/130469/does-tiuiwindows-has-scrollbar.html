Does Ti.UI.Windows has scrollbar?

I create a large window.
Does Ti.UI.Windows has scrollbar? So that i can see the beyond part.
Thanks!

  • You would normally use a scrollable view to put your content into. Leaving the window as a single screen size.

    i.e.

    
var win = Ti.UI.createWindwo({ ... )}

var winView = Ti.UI.createScrollView({ 
  height: 300,
   width: 200,
   contentHeight: 'auto'
})

win.add(winView);

    You can then add all your content to winView.

    Hope this helps
