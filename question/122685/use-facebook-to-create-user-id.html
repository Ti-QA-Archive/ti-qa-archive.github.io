Use Facebook to create user id

Hi. This is a high level question.

I am working on an app that allows the user to create their own ID (user name, etc). For convenience, I would also like to offer logging in with their Facebook ID. I've played with the Facebook functionality in Titanium (i.e.: allowing users to post to their FB page through the app) but I'm still not sure if I can use the FB sign-on as a substitute for allowing the user to log on to my app.

To be clear (as I'm probably not explaining this very well), I want users to create their own user name as part of logging in but to save time would like to allow them to log in via their FB sign on.

At the moment I know what content in the database belongs to a user - when they log in I get their user id and return data for that ID, but how would I do this if they logged in via FB?

Does FB return a permanent ID that is valid forever (as long as the person's FB credentials remain valid)? And I would then store this ID in the database in the place of the existing user ID?

Thanks.
