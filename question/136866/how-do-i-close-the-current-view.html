How do I close the current view?

It should be simple. My application window opes a view called Home. On button click, Home opens a view called Main. Main loads and displays an RSS feed item. Under the item I want a button that closes the main view and returns the user to the Home view. I added the following code to Main:

var l1 = Titanium.UI.createLabel({                    
                text:description,
                color:'#000',
                shadowColor:'#ddd',
                shadowOffset:{x:2,y:2},
                font:{fontSize:20},
                top:100,
                left:20,
                right:20,
                height:'auto',
                textAlign:'center'
});
main.add(l1);

var restart = Ti.UI.createButton({
                title:'Back',        
                height:30,
                width:100,
                top:100
});             
main.add(restart);
restart.addEventListener('click', function(){
                main.close();
});

When I run it, I get this error:

 message = "Result of expression 'main.close' [undefined] is not a function.";

What am I doing wrong?
