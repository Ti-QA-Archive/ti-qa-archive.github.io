Ti.Platform.username

Hi

I have get username/system name of my android device using Ti.Platform.username. but is there any way to set it manually in android device. because I have gone through documentation it is readonly property. So I have to set it manually from device.

I have done research on this but it is responding me with hostname. Is there any difference between hostname and system name?

thanks in advance.

  • the sad truth: readonly means readonly.

    there is no way even with native Java code (without rooting the device).The username property is declared as a static final variable see native reference, therefore it cannot be set.

    The hostname refers to the network identifer and adb identifier as far as i know.
