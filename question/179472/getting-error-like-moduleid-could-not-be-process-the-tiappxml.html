Getting error like ModuleID could not be process the tiapp.xml

I have created one simple module
RightClick -> New -> Mobile Module Project

Then simply publish as Android Module

All is done well,Build and module zip file successfully created but after build success message throwing an alert "Error attaching the testmodule module to the {1}/tiapp.xml., moduleID could not be process the tiapp.xml" .

  • Try to copy zip in modules directory of titanium mobile project. Also include the module entry in tiapp.xml.

    <modules>
           <module platform="android">com.test.module</module>
</modules>

    Try to run mobile project. If it runs then there is no issue with your module.
