  • If I understand you correctly, you want a global "page loaded" function? You can use what we call synthetic events. Create a custom function like

    function pageLoaded(){
  doStuff;
}

    In that same module, create a GLOBAL event listener to call that function:

    Ti.App.addEventListener('pageLoaded', function(){
  pageLoaded();
});

    Then, whenever you to want call that method, use fireEvent() in any other module

    Ti.App.fireEvent('pageLoaded');

    Hope that helps.
