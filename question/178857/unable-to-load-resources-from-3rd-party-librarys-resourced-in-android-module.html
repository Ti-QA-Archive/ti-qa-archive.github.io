  • Finally I got the answer:

    just like the official document said, copy the res folder from native android project's root path , to titanium module's android&#x2F;platform&#x2F;android folder.

    If it doesn't work, check if you missed some of the resource files!

    I also asked this question on StackOverflow, maybe you want to take a look: http://stackoverflow.com/questions/29627175/in-titanium-is-it-possible-to-access-the-3rd-party-jar-bundled-resource-files

    e.g. my module project structure is shown as below:

        # name: test_ti_module_qq_login:
    android
      \--assets
      \--build
      \--dist
      \--lib
      \--libs
      \--src
      \--platform
         \--android
            \--res    # just put the resources here in your module! 
               \--drawable
               \--drawable-hdpi
               \--drawable-xhdpi
               \--layout
               \--values
    assets
    documentation
    example
    LICENSE
    ...

    and FYI, the native android resource folder look like:

        # A native Android project structure   
    ? assets/          
    ? bin/             
    ? gen/             
    ? libs/           
    ? res/     # these are the bundled resource files
      ? drawable-hdpi/     
      ? drawable-xhdpi/    
      ? layout/     
      ? values/    
      ? src/
