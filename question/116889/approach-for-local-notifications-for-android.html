  • PubNub provides a clean and simple API for cross-platform Push notifications.

    Pure JavaScript if you're looking at Titanium Mobile (although they provide API's for a large number of languages).

    No need for extraneous modules, which is really beautiful.

    I'm presently employing it as a background service (on the Android platform) and it works great.
    Highly recommended.
