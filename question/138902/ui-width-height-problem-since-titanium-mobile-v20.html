  • I am able to get the height and width of the window by defining a postlayout event listener and testing it there. I have found that sometimes I get a call to the postlayout event listener function and when I check height and width, I get values of 0. When that happens, I immediately return from my event listener; the event will fire again, and I will get the proper values.
