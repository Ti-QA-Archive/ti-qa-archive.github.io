  • Your question can be confusing.

    Titanium allows you to develop standalone mobile applications that will be run on a device or an emulator (for development). Developing these applications is done entirely using JavaScript and there is no PHP involved.

    Now, that being said, you can have your mobile application interact with a web service to post or retrieve information. This server part can be developed using any web technology of your choice (including PHP). All this is done through the HTTPClient object.

    Hope this clarifies it for you.
