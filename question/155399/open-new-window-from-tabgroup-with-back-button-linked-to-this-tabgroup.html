Open new window from tabgroup with back button linked to this tabgroup

I have the following simple tabgroup:

<Alloy>
    <TabGroup>
        <Tab title="tab 1">
            <Window id="authors" onOpen="getAuthors">
                <TableView dataCollection="authors">
                    <TableViewRow id="row" onClick="showAuthor">
                        <Label id="name" text="{name}"></Label>
                    </TableViewRow>
                </TableView>
            </Window>
        </Tab>
        <Tab title="tab 2">
            <Window id="single_theory">
                    <Label>tab 2</Label>
            </Window>
        </Tab>
        <Tab title="tab 3">
            <Window id="cheories_list">
                <Label>tab 3</Label>
            </Window>
        </Tab>
    </TabGroup>
</Alloy>

What I'm trying to do is to open a new window, when someone clicks the TableViewRow in first window, which will have a ios default back button linked to this TabGroup. I there any method to automatically create this type of window or I should create an ordinary window and manually add back button at the top?

    Back buttons are created by default when opening new window inside a tab (In tab group each tab is same like a navigation group) or navigation group. Use this method Ti.UI.currentTab.Open
