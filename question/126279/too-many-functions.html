Too Many Functions?

Hi All,

Hopefully this is simple.

My app's been running fine but I added a new function and the app crashes. I remove the function and it works. In fact if I remove any function at all it runs again.

Is there a limit to the number of functions an app can have?

  • For Android, there is a limit of functions you can nest inside another (nest levels). If this is your case, overwrite this property in your tiapp.xml file:

    <property name="ti.android.threadstacksize" type="int">32768</property>

    Look for 'android threadstacksize' for more info.
