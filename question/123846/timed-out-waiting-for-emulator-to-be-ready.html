  • Accepted Answer

    run the android.exe form inside the android-sdk/platform-tools directory to make sure that the sdk tools can run.

    When it launches, choose the titanium_7_hvga VDM if it exists to make sure it actually will launch for you. If it does, then try to build to the emulator.

    Steve is right, the emulator is flakey. More so on windows than osx, but still flakey.
