  • My solution is a workaround: attach a new function to the searchBar's focus and blur events and set the table's background color to something/transparent.

    — answered January 5th 2012 by Kristof Gruber
    permalink
    2 Comments

    • Hi, Kristof,

      — commented January 29th 2012 by Leopard Qt 

    • 
var search = Titanium.UI.createSearchBar({
    hintText: L('search_hint_message', 'Keresés'),
    backgroundColor: (traffi.nightMode ? '#444444' : '#ACBED0')
});

search.addEventListener('focus', function() {
    traffi.windows.nearbyTable.backgroundImage = 'background.png';
});
search.addEventListener('blur', function() {
    traffi.windows.nearbyTable.backgroundImage = null;
});

traffi.windows.nearbyTable.search = search;
traffi.windows.nearbyTable.filterAttribute = 'searchString';
