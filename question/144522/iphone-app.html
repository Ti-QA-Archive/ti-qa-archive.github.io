  • In the TiStudio Distribute drop down, there is an option, distribute AdHoc/Enterprise.

    You will need to sign it with an adhoc distribution provisioning profile which must have the device id of the other device in it.

    You can use testflightapp.com to facilitate a lot of the distribution overhead for you.
