Strange Problem with a Tableview when height set to Ti.UI.Size

Hi all,
I have a ScrollView and a Tableview in it.

To keep the size of the Tableview at the minimum I write Ti.UI.Size in the height property.

At this point I run the App and I start to add rows dinamically.

Teorically when I add a Row, the height of the Tableview will increase with each row added.

Effectively it works like I said, but when the Tableview reach the same height of the screen, stops increasing the height.

All other rows will be viewed only scrolling the table.

Someone knows why this happen?

This is the code of my TableView:

var tableView = Ti.UI.createTableView({
    data: [],
    top: 15,
    width: "94%",
    height : Ti.UI.Size,
    scrollable:false,
    border: 4,
    borderColor: "orange",
    borderRadius: 6,
});
