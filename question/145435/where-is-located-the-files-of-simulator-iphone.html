  • If you are looking for the structure on your Mac, its in:

    $HOME&#x2F;Library&#x2F;Application\ Support&#x2F;iPhone\ Simulator

    Note that the Library directory is hidden by default.

    If you need to browse the file system of an iOS device, iexplorer.app is a nice, not free, utility helpful during on device development
