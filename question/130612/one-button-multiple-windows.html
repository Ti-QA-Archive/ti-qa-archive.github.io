One button. Multiple windows.

I have a button that I want on multiple windows of my app. For some reason, it will show up on window 1 (tab 1), then if I click tab 2 and open window 2, it shows up there. BUT if I click back to tab 1, it is gone! The same goes for an imageView I would like to use on multiple windows.

What is the secret to having one defined button, but using it in several places?

    • If you need exactly same button with exactly same functionality in exactly same position for every window, try using something like this:

      var newBtn = function(win){
   //create your button
   var button = Ti.UI.createButton({
        height:50,
        width:"50%",
        color:'#000',
        title: "My button",
        font: {
            fontSize:14,
            fontFamily:'Helvetica Neue',
            fontWeight:'bold'
        },
        left:"25%",
        right:"25%",
        top:"20%"
    });
    //create functionality
    button.addEventListener('click', function(){alert("me");});
    //add button to window
    win.add(button);
};

      If buttons will differ in any kind, just pass different things as function arguments and apply them to button.

      After that, where you define windows for your tabgroup or simply multiple windows, use it like this:

      //for every window you have
newBtn(win1);
newBtn(win2);
newBtn(win3);

    • You can't declare variable with square bracket notation. You can create some object-like (Array, Object, String, even Function) value and store it in variable and than use square-bracket notation to get some property of that object-like value:

      var arr = [ 1, 2, 3 ];
arr[1]; //ok

var obj = { prop: 'Test' };
obj['prop']; //ok

var aa[2] = 4; //wrong!!!

      …

      What I meant is, something like, this:

      function createBlueButton(options) {
    if (!options) {
        options = {};
    }
    options.backgroundColor = '#00f';
    return Ti.UI.createButton(options);
}

//if you use CommonJS, you can export it
exports.BlueButton = createBlueButton;

      Now you can call it everywhere where you include or require it and you can call it as many times as you want:

      var win1 = Ti.UI.createWindow({ layout: 'vertical' });
for (var i = 0; i < 5; i++){
    win1.add(createBlueButton());
}

...

var win2 = Ti.UI.createWindow({ layout: 'vertical' });
for (var i = 0; i < 5; i++){
    win2.add(createBlueButton());
}

      Windows will contain 5 blue buttons.
