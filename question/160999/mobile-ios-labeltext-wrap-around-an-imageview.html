Mobile iOS Label/Text Wrap Around An ImageView

Hello,
I am trying to get a label to wrap around an ImageView element. Both are 'added' to a view. When I have the Image added to the view, the text is on top of the image. And when I try to add the image to the label the text is behind the image. I have height and width set on the image and the label height and width are set to auto.

Am I missing something easy or is this a more difficult task? How do I get this label to wrap around the image. (The look is similar to the first letter of the first word of a paragraph being over sized and spanning over 3-4 rows)

Thanks in advance.

  • So here's the temporary work around I'm using. It isn't pretty but for now, with static text, it gets the job done. The method I'm using makes use of .substring()

    viewBody = Ti.UI.createView({
    top: 0,
    left: 0,
    bottom: 125,
    height: 'auto',
    width: '100%'
});

var bodyString = 'Lorem ipsum dolor sit amet, consectetur adipiscing elit. Cras suscipit elementu  haretra. Praesent eleifend tempus diam non pharetra. Vestibulum ante ipsum primis in faucibus orci luctus et ultrices posuere cubilia Curae; Aliquam metus quam, vehicula non libero lacinia, dapibus convallis purus. Vestibulum ornare et dui vehicula dapibus. Fusce fermentum aliquam arcu. Maecenas vitae mauris nec eros pharetra vehicula. Aliquam porttitor elit scelerisque lorem blandit, venenatis semper mi mattis. Maecenas tincidunt vehicula elit in elementum. Pellentesque molestie enim quis turpis rhoncus, eu fermentum justo lobortis.';

labelBody = Ti.UI.createLabel({
    text: bodyString.substring(0,151),
    height: 'auto',
    width: 'auto',
    top: 0,
    left: 95,
    right: 15,
    color: '#000',
    font: { fontSize: 14 }
});

labelBody2 = Ti.UI.createLabel({
    text: bodyString.substring(151),
    height: 'auto',
    width: 'auto',
    top: 85,
    left: 15,
    right: 15,
    color: '#000',
    font: { fontSize: 14 }
});


viewBody.add(labelBody);

logo = Ti.UI.createImageView({
    image: '/images/logo.png',
    height: 80,
    width: 65,
    top: 0
});

viewBody.add(logo);

    This does the trick for now. Carefully truncating the string so that it doesn't splice a word into two parts incorrectly. Of course this isn't very responsive, looking to improve this later.

  • The best I can think of is

    var w = Ti.UI.createWindow({
    top : 20,
    backgroundColor : 'white'
});

var v = Ti.UI.createView({
    top : 0,
    layout : 'horizontal',
    height : 200
});

var i = Ti.UI.createImageView({
    top : 0,
    image : 'KS_nav_ui.png',
    width : 65,
    height : 80,
    left : 0
});

var l = Ti.UI.createLabel({
    top : 0,
    left : 0,
    right : 0,
    height : Ti.UI.SIZE,
    text : 'This is a long label that wraps across more than one row down the side of the image view................'
});
v.add(i);
v.add(l)
w.add(v);
w.open();

    Which wraps the label down the right side of the imageview, but this isnt CSS so you cannot continue the bounds of the label so that is wraps under the image too, i'm afraid
