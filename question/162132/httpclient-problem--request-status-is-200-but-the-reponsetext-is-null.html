    A few thoughts, questions and follow ups.

    • The server you are connected to - is this a local server (to your office and only on your network) or available via the world wide web (if you chose to share the url then I would be able to view it as well - not asking you to share - just confirming it would be viewable by me)

    • When you checked the JSON did you use a service jsonlint.com or manually? Always use a service.
    • When you tried in the browser did you add any parameters on the url line other than the domain, folders, filename?
    • What server side scripting language are you using to serve up the json?
