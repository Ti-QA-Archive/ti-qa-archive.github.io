Bigger font in email messageData

Hi.

I have an email function in my app.

How do i get bigger font on some of the text?

This is my code

var messageData='Anställningsnr : '+Ti.App.myUsername+'\nProjektnamn: '+Ti.App.projectName+'\nProjektnr: '+Ti.App.projectNumber+'\n\n\n';

    var emailDialog = Ti.UI.createEmailDialog();
    emailDialog.subject = 'Subject';
    emailDialog.toRecipients = ['hello@wassup.com'];
    emailDialog.messageBody = messageData;
    emailDialog.open();

I want the words Anställningsnr, Projektnamn etc in bolder bigger font and the Ti.App.myUsername etc to be as they are.

Is this possible?

    Hi Richard,

    You can large the size of fonts in Email by using HTML tags only. You have to enable the method setHTML to true in order to use HTML tags else tags would be considered as part of a string.

    var messageData='Anställningsnr : <H1>'+Ti.App.myUsername+'</H1>\nProjektnamn: <h2>'+Ti.App.projectName+'</h2>\nProjektnr: <b><i>'+Ti.App.projectNumber+'</i></b>\n\n\n';

    var emailDialog = Ti.UI.createEmailDialog();
    emailDialog.subject = 'Subject';
    emailDialog.toRecipients = ['hello@wassup.com'];
    emailDialog.setHtml(true);
    emailDialog.messageBody = messageData;
    emailDialog.open();

    Let me know, if this works

    Thanks
