SQLITE:select query like expression

Application type:mobile,titanium studio:3.3.0,titanium sdk:3.3.0,OSX:Mac osx.

How can I select the exact value from the database select query:

var resultset = db.execute('SELECT * FROM members WHERE  LOWER(bloodgroup) like  "%' + win.bloodgroup + '%"  ORDER BY membername');

I have used this query.
If I say AB+ it exactly selects the AB+.but If I say B+ it gives me both AB+ and B+ results.
how can I get exact query results.

    If you want exact result then use = instead of like. Below is sample query,

    
var resultset = db.execute('SELECT * FROM members WHERE  LOWER(bloodgroup) =' + win.bloodgroup + ' ORDER BY membername');

    • I will suggest first form the whole query string and store it in variable. Print that variable using Ti.API.info. Copy the whole query string and run it in sqlite to check whether it is giving desired output or not.

      Or Other way is first make query in any of the sqlite manager/console. Test the query over there and then integrate the same in your application code.

      Here is a sample,

      
var sqlQuery = 'SELECT * FROM members WHERE LOWER(membername) like "%' + win.name + '%" AND LOWER(business) LIKE "%' + win.profession + '%" AND bloodgroup="' + win.bloodgroup + '" AND (LOWER(mobile) like "%' + win.mobile + '%" OR LOWER(res_phone) like "%' + win.mobile + '%" OR LOWER(off_phone) like "%' + win.mobile + '%") ORDER BY membername';

Ti.API.info("sqlQuery ==> " + sqlQuery);

var resultrows = db.execute(sqlQuery);

      Hope this will give you fare idea about how to debug.

      Thanks.
