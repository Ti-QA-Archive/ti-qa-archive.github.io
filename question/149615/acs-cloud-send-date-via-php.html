ACS Cloud Send Date via PHP

Hi,

I have a little problem sending date to ACS cloud via php.

When i send a date like this: "2013-03-14T23:59:00+0000" ACS cut the "+" sign.

So my date isn't interpreted as a date. I tried to insert a "-" sign like this "2013-03-14T23:59:00-0000" it works well, but i really need the plus…

Anyone can explain that ?

Thanks

  • Solved the problem by myself :)

    If anyone want to know, you have to encode the "+" character since it passed in url. So i replaced it by "%2B"
