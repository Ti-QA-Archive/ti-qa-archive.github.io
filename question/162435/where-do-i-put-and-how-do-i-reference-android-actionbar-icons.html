Where do I put and how do I reference Android Actionbar icons?

Ti 3.2.1, Android Emulator 4.2

I have an Android Actionbar like so:

tabGroup.addEventListener("open", function(event) {
    tabGroup.activity.onCreateOptionsMenu = function(e) {
        var menu = e.menu;
        menu.clear();
        if (tabGroup.activeTab == listTab || tabGroup.activeTab == mapTab) {
            var item = menu.add({
                icon: 'images/refresh.png',
                showAsAction: Ti.Android.SHOW_AS_ACTION_ALWAYS
            }).addEventListener("click", function(event) {
               ....

        }
    };
});

I want to provide a density specific icon for my actionbar item. I have now referenced it as 'images/refresh.png' and stored it in Resources/android/images/refresh.png.

    Hi

    Try this;

    // change this
icon: 'images/refresh.png',
// to this
icon: '/images/refresh.png',

    Place your images as follows;

    {appname}/resources/android/images/res-ldpi/here.png
{appname}/resources/android/images/res-mdpi/here.png
{appname}/resources/android/images/res-hdpi/here.png
{appname}/resources/android/images/res-xhdpi/here.png
{appname}/resources/android/images/res-xxhdpi/here.png

    Use Project > Clean (select your project) and confirm.

    This is how you normally organise images.
