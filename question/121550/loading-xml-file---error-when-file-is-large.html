  • What do you consider to be a large XML file size? Does you XML file save complete to the file system when it is downloaded?

    When I load XML files from a web service in my apps I tend to write them to the file system first and then load them. The onload fires an event, and that event then loads the XML back up.

    I do this mainly for caching purposes as I use them as an alternative to sqlite databases.
