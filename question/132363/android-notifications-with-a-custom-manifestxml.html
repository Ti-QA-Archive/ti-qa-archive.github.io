Android Notifications with a custom Manifest.xml

I'm trying to get notifications working in Android and I've followed the instructions at https://wiki.appcelerator.org/display/guides/Android+Notifications. I put the extra stuff in my tiapp.xml file as specified in the tutorial (scroll to the very bottom) and the external JS file that specifies the service is a copy and paste from that tutorial. But I still can't make it work.

I think what is happening is that there is a conflict between adding the stuff to the tiapp.xml and the custom AndroidManifest.xml file I've added to my project. Basically, I think it's ignoring the tiapp.xml stuff. But adding the code from the tiapp.xml to the AndroidManifest.xml file doesn't work. I get errors that bunch of things are not declared or are invalid.

So is there an alternative way to declare the service in the AndroidManifest.xml file that WILL work? I've tried reading the Android documentation, but it doesn't seem to match up with what I'm passing into Ti. Thanks.
