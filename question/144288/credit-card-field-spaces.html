  • Hello Weina,

    you can do it on your textField's change event like..

    text.addEventListener('change', function(e) {

    if (e.value.length > 19){
        e.source.value = e.source.oldValue;
    }
    else{
        e.source.oldValue = e.value;
        if (text.value.length == 4 || text.value.length == 9 || text.value.length == 14) {

            text.value = text.value + ' ';
            e.source.value = e.source.oldValue;
        }    
    }

});

    just try this code and do appropriate change as your requirement.

    But i think this is what you are looking for…
