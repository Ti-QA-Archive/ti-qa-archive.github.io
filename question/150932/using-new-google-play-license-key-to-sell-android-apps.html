Using New Google Play 'license key' To Sell Android Apps

Google seem to have recently changed the way paid apps are added to the developer console.

There is now a 'License Key' generated by Google for you to use in apps that use certain services: Paid Apps, In-App Sales etc.

Can someone tell me how to include this License Key into my app when publishing.

Do I need to put it into the tiapp.xml file ?

Or is it used when I am signing the app somehow ?

  • Hi,

    Google play set up this licencing this way to put the onus on you as the developer to protect your own app.
    Appcelerator does not include any method by which you can include the licence key into your app, however there is a module in Marketplace to enable you to use the licence key.

    Licencing Module

    I purchased this module and it is very easy to use following the example included with the module.

    Having said that it is also very easy to bypass the licence key if you care to Google Licence Key.

    One of the main reasons why I sell more apps through Apple than Google Play.

    Good Luck.

    John
