  • Ok, so after playing with the code as Paul suggested, I noticed that we had an animated intro:

    window.open = function() {
  tabGroup.open({
    transition: Titanium.UI.iPhone.AnimationStyle.FLIP_FROM_LEFT,
    delay: 500,
    width: 768,
    height: 1004
  });
}

    And after taking out the width and height values, the tabs and header graphic showed back up in the landscape view.
