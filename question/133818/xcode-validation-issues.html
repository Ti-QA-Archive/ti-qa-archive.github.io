XCode Validation Issues

Hi guys,

I've used Titanium for a while and submitted various versions and apps to the AppStore with no issues.

Today however I've published my app for distribution from Titanium Studio (1.0.9.201202141208) and It's failing validation.

This bundle is invalid. The key CFBundleShortVersionString in the Info.plist file must contain a higher version than that of the previously uploaded version.

My version numbers are correct. Current version in the AppStore is 3.1, my iTunes Connect update is set to 3.2 and my tiapp.xml version number is set to 3.2

my Info.plist in Xcode says 3.2, though the short version number does say 1.0. I've tried changing that manually to 3.2 as well with no luck.

Any ideas how to fix this? I need to get this update into the store quick! Important bug fixes etc

Hope someone can help!
