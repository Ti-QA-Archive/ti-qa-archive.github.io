Facebook 'login' event handler not working on Android

I recently switched from Titanium to Appcelerator and changed the SDK from 3.5.0 GA to 4.1.1.v20150807121809 and probably went from Facebook module 3.X.X to 4.0.5. I managed to adjust most of the things for Facebook. It works nicely in iOS, but I can' t seem to get it working in Android.

Normally the login scenario works like this:

  1. The user wants to share a news item and "clicks" the share button.
  2. The app checks if the user is logged into Facebook.
  3. The user is not logged in, so the app will use the authorize() function to show the login screen.
  4. The user logs into facebook and authorizes the app to use the email and public_profile permissions.
  5. The login eventlistener is fired off and, on a successful login, will show the share dialog of Facebook.

As stated before, this works perfectly on iOS, but on Android is login eventlistener never gets fired off.
I seriously got no idea what I did wrong and it has been bugging me for several days now. The code that I use is as followed:

function authorizeFacebook() {
    if(G.hasNetwork()) {
        if (!F.loggedIn) {
            F.addEventListener("login", function loginCallback(e) {
                if (e.success) {
                    F.removeEventListener("login", loginCallback);
                    shareFacebook();
                }
            });
            F.authorize();
        } else {
            shareFacebook();
        }
    } else {
        errorDialog(L('errorConnection'));
    }
}

function shareFacebook() {
    var currentData = {
        "link" : $.itemTitle.webUrl || G.COMPANY.web,
        "name" : F.subStringAttribute($.itemTitle.text),
        "message" : F.subStringAttribute($.mainText.text),
        "caption" : F.subStringAttribute($.introText.text),
        "picture" : $.screenshot && $.screenshot.image || "",
        "description" : F.subStringAttribute($.mainText.text)
    };

    if(F.getCanPresentShareDialog()) {
        F.presentShareDialog(currentData);
    } else {
        F.presentWebShareDialog(currentData);
    }
}

I would really appreciate it if some one could help me out with this.
