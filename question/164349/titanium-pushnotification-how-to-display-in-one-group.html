  • Hi,
    Not a direct complete substitute for big view style for notification but still can add a grouped notification template into. You can implement this by adding following line in your tiapp.xml :

    <property name="acs-grouped-notification-message" type="string">you have received $number$ push</property>

    Hope it helps.
