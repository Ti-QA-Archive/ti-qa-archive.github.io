trigger table scroll from image

I'm working on iPhone (4.3) and I'm trying to place two semi transparent images on the top of a tableview. The images make the tableview look like a picker, however once the images are on top of the tableview no events are triggered.

Is there a way to have normal table touch functionality, but with the images on top?

    James,

    I would need to see your code to make sure there are no other issues, but you should be able to simply add the touchEnabled: false property to your images views. This will allow all all events to pass through to your table view.
