How can I call other apps from my app and send variables?

I have an app that has a map showing various destinations(annotation), I want to be able to call the user's chosen map app to handle directions.

Like for example when you share a location in WhatsApp, when you click on it, it ask if you wanna open with Google Maps, Waze or any other app that handles that type of information(as long as it is installed), at the same time, WPP send latitude and longitude to those apps.

Using the latest version of Titanium and it's for an iOS app.

    • Thanks Stephen, that was very helpful, I'm gonna leave the code I used than I think others might find useful, tested on iOS only.

      Tries to use Waze, if it doesn't have, it uses Google Maps, if it doesn't have it, it uses the web version of google maps.

      if (Ti.Platform.canOpenURL('waze://?ll='+latitude+','+longitude+'&navigate=yes')) Ti.Platform.openURL('waze://?ll='+latitude+','+longitude+'&navigate=yes');

else if(Ti.Platform.canOpenURL('comgooglemaps://?q='+latitude+','+longitude)) Ti.Platform.openURL('comgooglemaps://?q='+latitude+','+longitude);

else Ti.Platform.openURL('maps://maps.google.com/maps?q='+latitude+','+longitude);
