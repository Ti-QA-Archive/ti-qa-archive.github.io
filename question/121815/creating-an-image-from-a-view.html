Creating an image from a view?

Does Titanium have a way to turn a view and it's subviews into an image? I need to take a photo with the camera, apply an overlay view, and then save the composite image (imageview + overlay) to the photo library.

I know it's fairly trivial to do in obj-c and a quick Google search shows that it doesn't appear to be too difficult in java either… but I just wanted to see if I was missing an easier way to do this before I start creating modules.
