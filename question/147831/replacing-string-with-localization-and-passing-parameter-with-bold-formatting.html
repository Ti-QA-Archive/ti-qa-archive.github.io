  • Hi

    As you have not provided the platform you are creating your app for there are two answers;

    • Android & Mobile Web

      Yes you can use the label property html which supports minimal html support.

    • The rest

      No the standard label view does not provide native support for formatting. You can use custom modules to provide the support for example Styled Label Module, this should be used to provide limited formatting.
