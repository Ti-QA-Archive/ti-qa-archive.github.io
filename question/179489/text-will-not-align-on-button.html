Text will not align on button

I am using the following code. The text on the button centers itself. What do I need to do to get it to left align? Thank you.

        var btn = Ti.UI.createButton ({
            backgroundColor: '#444444',
            title: 'Just some text',
            textAlign: Ti.UI.TEXT_ALIGNMENT_LEFT,
            color: '#FFFFFF',
            font: {
                fontSize:20,
                fontFamily: 'HelveticaNeue'
            },
            top: 200,
            left: 0,
            width: Ti.UI.FILL,
            height: 55
        });
    self.add(btn);

    Hello Paul,

    Thanks for your inquiry. Try using 

    textAlign: "left"

    instead of 

    textAlign: Ti.UI.TEXT_ALIGNMENT_LEFT,

    According to titanium documentation, it doesn't specified that u have to use "Ti.UI.TEXT_ALIGNMENT_RIGHT" in button. It should be used in textfield.

    Thanks.
