Can I auto sync my commits that are done in Titanium to github?

I have just set up a github account and checked in my app source code. I wanted some sort of version control and offsite storage for my apps so I went with a github account. My question is can I auto sync my commits that are done in Titanium? From what I can tell I need to go into the github desktop application and click sync after each commit or else they will stay on my local computer.

Thanks,

David

  • You can control your git repo from within the titanium studio.

    This is a knowledge issue on GIT. When you have committed, yes you need to push your changes to the repo or they will stay local only.

    Have a look HERE to get a good understanding of git.

    Then you can use the git tools in studio to manage your project. (I personally use SourceTree as I have many repos for work and personal).
