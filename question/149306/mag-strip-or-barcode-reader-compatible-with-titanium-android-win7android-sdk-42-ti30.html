  • Although I never got this to work, I learned that the appropriate was of using a java based module in a Titanium android project, is to create a Titanium module, add the .jar to that, then include the module in your project. There is plenty of Appcelerator documentation describing this process. But one note to anyone considering using the Unimag sdk; it doesn't support my device SGS3 and who knows what others!
