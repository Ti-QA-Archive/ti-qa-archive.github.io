More then one VideoPlayer seems not work?

My app has 2 tabs, TabA and TabB, in a tabgroup.
And each tab will open a window containing a VideoPlayer. So a videoPlayer on each tab.

When I open TabA, video can be played successfully.
Then open TabB, video can be played successfully too.
But now, when switch back to TabA, the video player there disappear, while player on TabB still there, working properly.

Does anyone encounter similar problem before?

My app is for iOS, using Titanium Studio 3.0.2, SDK 3.1.1.GA
