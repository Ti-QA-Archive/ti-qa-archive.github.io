    • Hi

      Two things you need to do.

      Firstly, always add an error handler as it may be running the onload.

      xhr.onerror = function(e){
     Ti.API.info(alert(e));
};

      As a starting point change your onload event to use responseXML as the property responseText is not used for the xml document.

      xhr.onload = function(e) {
     var doc = this.responseXML.documentElement;
     var items = doc.getElementsByTagName("Playlist");
     Ti.API.info(items.length);
};

      Use this as a starting point then review this one about RSS which will show you an example similar to your own you can work through.
