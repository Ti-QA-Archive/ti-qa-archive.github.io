Event listener within an Event listener...possible?

I have a custom button that directs the user to a website. The website comes up in a new window via the click of an event listener. However, once there, I need for them to be able to push a "back" button on the navbar of that window. So, it seems like I need to create an event listener within an event listener??

This is an iPhone,iPad, and Android app and Im working on the latest version of Appcelerator on a Mac running the latest OS.

Here is my code so far:

var button11 = Ti.UI.createButton({
 backgroundImage:'button3.png',
 title: 'Google Search Engine',
 //font:{fontsize:40, fontFamily:'Helvetica Newe'},
 top: 130,
 width: 220,
 height:70

});

button11.addEventListener('click', function(e) {
    var webview = Titanium.UI.createWebView({url:'http://www.google.com'});
    var window = Titanium.UI.createWindow({
        title: 'Google Page'
    });
    window.add(webview);
    window.open({modal:true});
     });

So to reiterate, in the nav bar, next to "Google Page", on either side, I need the back button but am not sure how to code that since this is already coming from a custom button's event listener. If there is documentation on it, I would love to read up on it, but cant find anything on this navigation option.

Many thanks,
John

    Hi John.

    Because you are creating iphone and android application together you have to Create custom navBar of that you have to create one view and add that view in window and set top 0 and add button into that view and on click of that button close that window. Try below code as a refrence

    var navBar = Ti.UI.createView({
        backgroundImage : '../imgs/navBar.png',
        width : 320,
        height : 44,
        top : 0
    });

var btnBack = Ti.UI.createButton({
        width : 47,
        height : 44,
        left : 0,
        title : 'Back'
    });

    btnBack.addEventListener('click', function(e) {
        win.close();
    });

navBar.add(btnBack);
win.add(navBar);
