  • Hi Prasad M.R.C,

    It occurs several causes so you should install properly to avoid this issue.

    Follow this procedure

    At first install Jdk in your pc then install Titanium Studio then follow this

    • Install update Firefox version in your pc

    • Set Android SDK path with minimum SDK version API 10 2.3.3

    In this location Windows > Preference> Studio> platform> android> Android SDK Home

    • Set Titanium SDk path

    In this location Windows > Preference>Studio> platform> Titanium SDK Home

    • Then Check for Titanium Update This location Help> Check For Titanium Update

    After all check your environment variable then set JAVA-HOME and path in the system variable.

    Then Restart your PC’

    And Try to Run It

    You can set your environment variable path this way.

    • Right-click the My Computer icon on your desktop and select Properties.

    • Click the Advanced tab.

    • Click the Environment Variables button.

    • Under System Variables, click New.

    • Enter the variable name as JAVA_HOME. Then JAVA-HOME=C:\Program Files\Java\jdk1.7.0_17
    or C:\Program Files (x86)\Java\jdk1.7.0_17

    (note: this is your computer jdk path where you installed it )
    Again click new

    • Again Enter the variable name as path Then path = C:\Program Files\Java\jdk1.7.0_17\bin
    or C:\ Program Files (x86)\Java\jdk1.7.0_17\bin

    • Click OK.

    • Click Apply Changes.

    You might need to restart windows.

    I thank JDK problem would not show again. Now you can run your project.
