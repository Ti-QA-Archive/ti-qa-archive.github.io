  • Hi Justin,

    I don't know if this will help or not, but I was having a similar problem yesterday and found a way past it.

    In my case, I traced the problem to my deciding to use a regular expression in my app. This shouldn't be a problem, but for some reason it is.

    When I wrote it like this, it worked fine in the simulator but would give install errors when trying to deploy to my iPhone.

    var exp = /blahblahblah/;

    When I changed it to this the install error went away but I'd get a red screen error in the simulator.

    var exp = RegEx('blahblahblah');

    BUT, even thought it did install to my iPhone the app would halt when it hit that line, so it wasn't really working, it just wouldn't crash. LOL

    So for my app, I got rid of the regular expression completely and am using string.indexOf instead and now things work as expected.

    Hope this helps!
