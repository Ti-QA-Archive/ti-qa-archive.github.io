    • Have you tried it with the alwaysRetainTaskState parameter in tiapp.xml?

      Unfortunately, Android's primitive kill process sometimes leads to a known problem where the app is only partially shut down. The solution for Titanium apps is to set the alwaysRetainTaskState parameter in the AndroidManifest.xml file

      http://docs.appcelerator.com/titanium/latest/#!/guide/Maintaining_a_Custom_AndroidManifest.xml
