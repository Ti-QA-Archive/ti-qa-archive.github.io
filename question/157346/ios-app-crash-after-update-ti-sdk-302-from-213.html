    If you are using node version 0.8 then please update it to node version 0.10, From sdk 3.0.2 or greater, it supports only node version 0.10,

    you can check you current version in terminal using below command;

    node -v

    you can download the node version 0.10 from below link:

    nodejs.org/dist/v0.10.12/node-v0.10.12.pkg
