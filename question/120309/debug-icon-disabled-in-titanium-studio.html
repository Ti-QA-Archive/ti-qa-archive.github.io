Debug icon disabled in Titanium Studio

Hi all
This thingh is driving me mad.

The debugger worked perfectly for almost a month with the pre releases of Titanium Studio but now, after upgraded to 1.7.0 RC1 the debug wont start anymore.

Pratically the Dedug icon in the left pane is greyed out and there is nothingh I can do.

I reinstalled everything from scratch, I create a new project, I also tried the KitchenSink but nothingh.
I deleted the build directory but still nothingh.

I read in a previous post that if the SDK is lower that 1.7.0 the debug is disabled. So why is disabled in 1.7.0 RC1?

Thanks in advance to anyone that have a hint!

Vincenzo

  • Vincenzo

    Do you have either the Beta or Nightly update sites enabled? If so, the upcoming changes to the way debugging works in TiStudio have been included in them, in order for them to be tested.

    To be clear, be aware that debugging will only be available to developers with a subscription after the official TiStudio release at the beginning of July. For details of the differences between the upcoming Titanium Studio Standard and Premium versions, see the announcement about them here.

    To resolve your problem in the meantime, you can disable the Beta and Nightly sites and roll back the last, most recent, updates using the plugin manager. Alternatively, my preference would be to reinstall with only the Stable update site enabled.

    Hope this helps
