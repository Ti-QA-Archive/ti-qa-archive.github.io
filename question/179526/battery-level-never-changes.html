Battery level never changes

I'm using 3.4.1GA with iOS8.1.1 (this applies since iOS8.0) with iPhone5/6 and iPad 2/3 and have set Ti.Platform.batteryMonitoring to true. When the app first loads, I get a correct battery level using Ti.Platform.getBatteryLevel(). But then this value never changes for the life of the app. The battery change event doesn't seem to fire.

Since there doesn't seem to be any other API or parameter or config, what am I doing wrong? This used to work well with iOS6. Ever since iOS 7, it seems to have stopped working. Is this a Titanium bug or a new iOS restriction? Thanks in advance
