How can we test apps with Jasmine without running TiShadow?

There is a great article about using TiShadow to test with Jasmine, but people have been talking about testing without the cumbersome process of running TiShadow.

    Here are some options:

    1. Aaron Saunders shared a slide share using one technique with Alloy.
    2. These two repositories, although being old, will also be useful: titanium-jasmin and jasmin-titanium

    All the above however require an application rebuild every time you want to test and include your tests along side the source code. (They need to be removed when distributing the application.) This is not the case with TiShadow and despite the initial setup, ends up being faster.
