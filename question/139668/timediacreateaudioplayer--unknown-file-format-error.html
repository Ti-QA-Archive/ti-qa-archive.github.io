  • If you are playing a Shoutcast stream it's worth putting a /; (forward slash and semi-colon) on the end of your URL and then an mp3 extension to force Shoutcast into MP3 OVERIDE mode. I had the same where stream played OK on some Android devices but not tablets.
    So the URL would be http://example.net:8000/;.mp3
