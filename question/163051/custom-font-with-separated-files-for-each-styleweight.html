Custom font with separated files for each style/weight

I have read Custom Fonts, and it says that the (base) file name should be used in tss files for Android. If I have "Font.ttf", it should be written "Font".

"Label": {
    font: {
        fontFamily: "Font"
    }
}

If I have "Font-Regular.ttf", "Font-Italics.ttf", "Font-Bold.ttf", and "Font-Bold-Italics.ttf" (each style/weight in separated files), how do I write it?

  • Hi,

    First create a "fonts" folder in the Resources directory then paste your font.tts file in that folder. After that use it like,

    var content = Ti.UI.createLabel({
    text: "Hello You"
    font: {
        fontFamily: 'Times New Roman'
        },
    color: 'black'
});

    I think it'll solve your problem.
