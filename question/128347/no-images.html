  • There are two possible reasons I have come across for this.

    1) The image needs to be rendered. Not being a graphic bod, I let the designer do it. Where the image is not form correctly.

    2) It actually can't find the image..
    image: Ti.Filesystem.resourcesDirectory + "img/bg-image.png".

    So Suggestion is to set the path first, then try regenerating the image.
