doc file does not open in android

I want to open my application when click on *.doc file.

I wrote the following code in AndroidManifest.custom.xml and put it in to the build/android directory, when i try to open Doc file then it give me an error message activity not found.


<activity
            android:name=".ThirdActivity"
            android:label="Third"
            android:theme="@style/Theme.Titanium"
            android:configChanges="keyboardHidden|orientation">
            <intent-filter>
                <action android:name="android.intent.action.MAIN" />
                <action android:name="android.intent.action.VIEW" />

                <category android:name="android.intent.category.DEFAULT" />
                <category android:name="android.intent.category.BROWSABLE" />
                <category android:name="android.intent.category.LAUNCHER" />

                 <data android:scheme="http" android:host="*"
                    android:pathPattern=".*\\.doc" />
                <data android:scheme="https" android:host="*"
                    android:pathPattern=".*\\.doc" />
                <data android:scheme="content" android:host="*"
                    android:pathPattern=".*\\.doc" />
                <data android:scheme="file" android:host="*"
                    android:pathPattern=".*\\.doc" />

            </intent-filter>
</activity>

  • I got the solution open custom file.

    <activity android:name=".MainActivity" android:label="@string/app_name">
            <intent-filter>
                <action android:name="android.intent.action.MAIN" />
                <category android:name="android.intent.category.LAUNCHER" />
            </intent-filter>
            <intent-filter>
                <action android:name="android.intent.action.VIEW" />
                <category android:name="android.intent.category.DEFAULT" />
                <category android:name="android.intent.category.BROWSABLE" />
                <data android:host="*" />
                <data android:mimeType="*/*" />
                <data android:pathPattern=".*\\.txt" />
            </intent-filter>
        </activity>
