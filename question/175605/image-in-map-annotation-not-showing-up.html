Image in map annotation not showing up

Hi, this small (?) issue is driving me nuts…. why is the image not showing up?

   Map.createAnnotation({
        latitude: lat,
        longitude: lon,
        title: 'MyTitle',
        subtitle: 'MySubTitle',
        animate: true,
           myid:1,
        image: 'images/pin.png',  //pin image, works fine

        rightButton: 'images/map_arrow.png' // button image that does not show up
    })

The annotation itself works fine. It´s just that rightButton that is not displayed on it.

The image itself is a valid image residing in project > Resources > images.
The file from which the image is called is in project > Resources

I have tried with different images, this particular one is 30 x 30 px.
I have tried adding / in front of image url

I am currently testing this on Iphone (will move on to Android later).

Any hints would be sooo much appreciated.

Thanks in advance /Tove

  • Hi

    Try;

    Map.createAnnotation({
    latitude: lat,
    longitude: lon,
    title: 'MyTitle',
    subtitle: 'MySubTitle',
    animate: true,
    myid:1,
    image: '/images/pin.png',  // added initial slash - you should always do this
    rightButton: '/images/map_arrow.png' // added initial slash
});

    The images must either be in (non-alloy project);

    • {appname}/resources/images/
    • {appname}/resources/iphone/images/
    • {appname}/resources/android/images/ (when you choose to support it)
