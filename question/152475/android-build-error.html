  • Accepted Answer

    Seems build tools got moved to another directory with the latest Android SDK update. Create symlinks to aapt and dx in /Applications/Android-sdk/platform-tools:

    ln -s /Applications/Android-sdk/build-tools/17.0.0/aapt aapt
    ln -s /Applications/Android-sdk/build-tools/17.0.0/dx dx

    This solved it for me (after some digging in their Python code).
