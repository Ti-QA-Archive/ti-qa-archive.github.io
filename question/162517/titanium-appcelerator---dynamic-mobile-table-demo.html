  • If you click through the links in the comments on that video you'll find code at https://github.com/erictopia512/titanium_mobile_dynamic_table_demo. However, big big warning. That code is 4 years old. It's highly unlikely to even come close to working on Titanium now. Titanium was a very different product back then.

    You're better served starting from scratch. Check out our guides and API docs. Consider using the ListView instead of the TableView (it's a lot faster and works well on both iOS & Android). If there's a technique you're totally stumped on how to implement, ask it in a new question.
