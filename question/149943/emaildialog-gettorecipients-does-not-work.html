EmailDialog getToRecipients does not work

Hi,

I would like to get the list of Recipients of an email send by the user in the TI.UI.EmailDialog.
getToRecipients just returns "undefined" after the dialog is closed. Setting recipients works, the getToRecipients than returns the values I provided. However, changes by the user (adding, removing recipients) in the Dialog are not reflected. Sounds like a bug?

on Iphone iOS SDK 6.1 both simulator and 4GS phone

Titanium SDK 3.0.2GA

emailDialog.addEventListener('complete', function(e)
{

        var receivers=e.source.getToRecipients();
        Ti.API.log("send email to ? list:" +receivers);
        if (receivers) for (i=0;i<receivers.length;i++)
        {
            Ti.API.log("send email to "+receivers[i]);
        }
    }

  • Hi

    You have completely misunderstood the method getToRecipients. It is designed to give you programmatic control of the property toRecipients, which you can set with setToRecipients.

    But ONLY before the dialog is shown. The user entered email addresses are not and should not be accessible by the developer - this would a huge security issue that the user would not be aware off as they would correctly believe this is a system dialog and this NOT passed back to the app.

    If you truly need to capture email addresses you would need to build your own version if the dialog and handle the email send yourself.
