    • Hello, I have tested your code above, it's working, The only thing I changed was android:targetSdkVersion=&quot;21&quot; to android:targetSdkVersion=&quot;19&quot;

      Also I tried with

      custom_theme.xml

      <?xml version="1.0" encoding="utf-8"?>

        <item name="android:windowActionBar">false</item>

      tiapp.xml

      
    <android xmlns:android="http://schemas.android.com/apk/res/android">

        <manifest>
            <application android:theme="@style/Theme.NoActionBar"/>
            <!-- Need to specify at least API level 11 for Titanium SDK 3.2.x and prior -->
            <uses-sdk android:minSdkVersion="14" android:targetSdkVersion="19"/>
        </manifest>
    </android>

      Both of them is working. I am running SDK 3.4.1.GA, Mac 10.9.5, Hope this helps. Thanks.
