  • Ok, i found the answer to my own earlier question… so, here it is for all WindowsHeads like me:

    You can download earlier versions of the Ti SDK from: http://builds.appcelerator.com.s3.amazonaws.com/index.html

    And to install it… you need to copy the SDK to: C:\ProgramData\Titanium\mobilesdk\win32

    This last step was the strange bit since I installed Ti in C:\Ti the SDK is not installed in the same folder… somehow, no matter where you install Ti, the SDK is installed in C:\ProgramData\Titanium...

    So, now we can enjoy older versions of Ti SDK :-)

  • You can download old versions of the "Continuous" builds (nightly, development versions) from http://builds.appcelerator.com.s3.amazonaws.com/index.html. In the Branch drop-down list, select 1_4_x. The top-most 1.4.x build is internally called 1.4.3 and seems to be quite stable.

    Older than that, I don't know. I don't know that there's a repository of the older SDKs or TI Developer apps.
