    Hi, you have to iterate over the window children collection

    var cant = MyCurrentWindow.children.length;
while (cant > 0){
    cant--;
    /* check here whether MyCurrentWindow.children[cant] is the view 
       you're looking for, you can use any unique property or use something like this
    */
    if (MyCurrentWindow.children[cant] == [object TiUITableView]){
        var CurrTableView = MyCurrentWindow.children[cant]    
    }
}

    hope it helps
