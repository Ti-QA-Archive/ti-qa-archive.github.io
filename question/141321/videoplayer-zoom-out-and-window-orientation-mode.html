videoPlayer zoom out and window orientation mode

Hi all,

I’m adding a videoPlayer in a window how has landscape orientation mode, It works perfectly.
I can zoom my video without any problem, th issu is, when i remove zooming mode the window change it’s orientation mode to « Portait » instead of « Landscape », I’ve tried to force the orientation to « Landscape » but no effect.

my application type: mobile iPad, iOS 5.1, Titanium Studio 2.1.1, Titanium SDK 2.1.0.GA

Is there someone has the same issu?

Many thanks in advance for any help.

    I have the same issue. Moreover - following code has the same issue:

    var win1 = Titanium.UI.createWindow({  
    title:'Tab 1',
    backgroundColor:'#fff'
});

var aButton = Ti.UI.createButton({
    title : 'Open Window',
    height : 100,
    width : 100,
    top : 100,
    left : 100
});
win1.add(aButton);
win1.open();

// Listen for click events.
aButton.addEventListener('click', function() {
    var win = Ti.UI.createWindow({
        backgroundColor: "red",
        fullscreen: true,
        navBarHidden : true,
    });
    var btnClose = Ti.UI.createButton({
        title : 'Close',
        height : 100,
        width : 100,
        top : 100,
        left : 100
    });
    win.add(btnClose);
    btnClose.addEventListener('click', function() {
        win.close();
    });
    win.open({modal: true});
});

    To reproduce - go to Portrait orientation, click "Open Window": on white window and red window will be opened.
    then change orientation to landscape and click "Close" - white window will be Portrait on landscape screen…

    If remove fullscreen: true - issue is gone.

    The same happens with fullscreen video player.

  • Hi Sergey,
    me too, i need to keep my window in a fullscreen, so i solved that by using Titanium.Platform.displayCaps.platformHeight and Titanium.Platform.displayCaps.platformWidth

    here is my code :

    var myScreenHeight = Titanium.Platform.displayCaps.platformHeight;
    var myScreenWidth = Titanium.Platform.displayCaps.platformWidth;
    var win= Titanium.UI.createWindow({

            width : myScreenWidth,
        height : myScreenHeight

    });

    hope that can help you and thank you for your advises.
