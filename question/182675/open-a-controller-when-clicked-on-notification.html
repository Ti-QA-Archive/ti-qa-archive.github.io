  • Hi Abhishek,

    App crash issue is occurring due to the android activity will not found by the intent. You can carefully check activity tag your tiapp.xml file.
    It should be like <activity
    android:name="com.example.appname.appnameActivity" …
    ex. if appname= titaniumDemo and your package id com.example.titaniumDemo then activity tag like
    android:name="com.example.titaniumDemo.titaniumDemoActivity"

    After that you need to add the class name in the intent like "com.example.appname.appnameActivity"
