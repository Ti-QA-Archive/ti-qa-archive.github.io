In app SMS. How to install module?

Hello!

Titanium SDK version: 1.6.2
iPhone SDK version: 4.2

I am building an iOS app and there is a module to support in app SMS that I want to install and use. I cannot understand how to do this from the instructions on the website.

Is there anyone that know how to do it, someone that perhaps have installed it themselfs?

  • Solved it.

    Make sure you start by changing the Ti TDK version to you current in the file titanium.xcconfig.

    1. Move to the folder containing the build.py file.
    2. Run: ./build.py
    3. Run: unzip -u -o com.omorandi-iphone-0.1.zip -d /Library/Application\ Support/Titanium/
    4. Add to tiapp.xml: <modules>
      <module version="0.1">com.omorandi</module>
      </modules>
    5. Include the code in you file.

    Good luck!
