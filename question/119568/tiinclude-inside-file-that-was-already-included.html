Ti.include inside file that was already included

It seems that Ti.include doesn't work inside a file that has been included. Is this not possible, or am I doing something wrong?

For example:

contents of app.js:

Ti.include('Common.js');
...

contents of Common.js:

Ti.include('Base64.js');
...

The above gives me a "file not found" error even though Base64.js is there. I have to do it like this:

contents of app.js:

Ti.include('Base64.js');
Ti.include('Common.js');
...

This works fine but I have to include a bunch of files that are required by Common.js on every page that uses it. Not a huge deal but Common.js is not really self contained and it seems prone to errors down the road.
