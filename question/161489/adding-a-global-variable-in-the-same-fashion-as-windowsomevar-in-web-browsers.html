  • If you are creating an Alloy app, you have Alloy.Globals to which you can cautiously, and with great thought, ad global variables that work on ios and android.

    If you are on a classic project, variables created in app.js are 'global' in nature on ios, but not on Android.

    If you are on a classic project, using commonJS, take a look at http://www.thewarpedcoder.net/2011/12/apccelrator-titanium-commonjs-global-variables/ by Trevor Ward.

    I must say, I'm rather gobsmacked that you are using both require() and Ti.include in the same project. To be honest, I don't hold out much hope regarding the quality of your project.
