Possible to isolate functions between tabs?

My app has two tabs and inside each tab I define several functions which are of the same name.

For example. I have two tabs, inside both of these tabs I declare function ABC.

If I start in tab #1, switch to tab #2 and run function ABC, then switch back to tab #1 and run function ABC, the app actually tries to run function ABC that was declared in tab #2 (instead of the earlier declared function ABC in tab #1).

I assumed data was isolated between tabs as they are between windows. is there a way to isolate this data, or do I have to redeclare all my functions with new names between the tabs?

Thanks!

    If you are working in a single context app, then yes… if your using multipal context (using the URL property on the windows) then this should be working for you, and I would need to see some code samples to isolate the issue
