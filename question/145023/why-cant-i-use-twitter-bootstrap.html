    Hey Shane.

    Titanium Desktop doesn't exist anymore.

    Titanium Mobile is now the only product supported by Appcelerator. But if you've looked only a little before thinking I'm an idiot, you may have discovered it by yourself.

    So the code you are showing here is for TideSDK, the new name of Titanium Desktop. You can learn more about it here:

    http://www.tidesdk.org/

    As I said, HTML, CSS and every other languages you mentionned are not parts of Titanium anymore.

    Good luck;

    Ivan_

  • Hey Shane!

    I think you don't really understand how Titanium works.

    There is no CSS or HTML in Titanium, so you can't use any of those in your app (except in a webview, but this is a baaaaaad idea.)

    You may want to read the quick start guides here : http://docs.appcelerator.com/

    If you have any questions, just post here!

    (and if, I'm wrong and you try to embed Bootstrap in a webview, please post a more specific log so we can debug it together)
