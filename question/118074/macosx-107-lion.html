    Greetings,

    I am using Mac OS X 10.7 (Lion) with latest Titanium Studio and it works great. I had to update the Java SDK (it did it for me when I updated the Titanium Studio) and go to the AppStore and install Xcode 4.1 for Lion. That was it, no errors compiling and using either Android or iOS simulators.
