    Hi

    There is property to achieve this deleteButtonTitle, for some reason it is undocumented.

    var tbl = Ti.UI.createTableView({
    data: rows,
    deleteButtonTitle: 'xxx', // enter caption here
    editable: true,
    height: Ti.UI.FILL,
    width: Ti.UI.FILL,
});

    This will change the title on the red button. Works great in iOS.
