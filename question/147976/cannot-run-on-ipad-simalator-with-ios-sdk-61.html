Cannot run on iPad Simalator with IOS SDK 6.1

Hi,
I have migrated IOS SDK from 6.0 to 6.1 and I'm using Titanium SDK 3.1, but I can run on simulator, I have the error message saying that SDK 6.0 is not found.
Any idea ?

  • Accepted Answer

    Run titanium info -t ios and towards the top it will list all Xcode installs that Titanium found including the iOS SDKs. If none of the Xcode installs show iOS SDK 6.0, then you need to either A) install the iOS SDK 6.0 for one of the Xcode installs that Titanium found, B) "xcode-select" your Xcode install that has iOS SDK 6.0 installed, or C) make sure the Xcode install that has iOS SDK 6.0 is located in the /Applications folder and must start with "Xcode" and end with ".app" (i.e. Xcode.app, Xcode452.app, Xcode46.app).
