    • Why are you formatting the value like this?

      sessionObject.powerswitch.state = "\"off\"";

      Why not like this?

      sessionObject.powerswitch.state = "off";

      The quotes you see in the object just tell you those are string - you do not need to include them when setting the new value. As the new value IS a string the object will show those if you stringify as a confirmation. In reality those quotes to not exist inside the object internally.
