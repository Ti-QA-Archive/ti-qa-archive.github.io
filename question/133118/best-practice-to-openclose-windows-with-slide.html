Best practice to open/close windows with slide

What would be the best practice with CommonJS to open and close one window from another with slide animation?

I would prefer ideas for both iPhone and android, but if it is not possible then at least something for iPhone for now.

I don’t want to use TabGroup or navGroup. Also, unfortunately Titanium.UI.iPhone.AnimationStyle has only fancy default animations but not the basic slide.

If there is a way to use the functionality of TabGroup without having it visible would be the best solution. If not, I guess building my own navigation, but then I need to know how to do it, navigate between windows, open, close, and not stacking windows?

Any help would be appreciated.

  • Accepted Answer

    var win = Ti.UI.createWindow({left:800});
var animation = Ti.UI.createAnimation({left : 0,duration : 500});
win.open(animation);
