How to position the image dynamically below the text

Hi all,

I have a situation like a full paragraph and below that there is an image in the form of the line. I want to move the line below or up with respect to change in the the words of the paragraph. Like if there is a decrease in the words then the line should move up and if the words are more then the line should go down. The paragraph is just a label. The development is only for iphone and i am using titanium sdk 1.7.2

    You can set an automatic height on your label by using height: &quot;auto&quot;. To position an image under the label, the easiest way would probably be to put the label and the image in a view with layout: &quot;vertical&quot;

    var view = Ti.UI.createView({
  height: "auto",
  layout: "vertical"
});
var label = Ti.UI.createLabel({
  height: "auto",
  text: "Lorem ipsum dolor sit amet."
});
var image = Ti.UI.createImageView({
  url: "/path/to/image.jpg",
  width: 100,
  height: 100
});
view.add(label);
view.add(image);

    Code not tested, but in this case it isn't rocket science to it should work
