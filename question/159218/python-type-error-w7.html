  • Solved. Need to find out where W7 ( Matt Langston in TiMob 13944 said W7 was not tested) stores those 3 files & copy them. The files (aapt, dx, /lib) are stored in C:\Progra~2\Android\android_sdk\build=tools\18.1.1 need to be copied to \platform-tools. And since these files are stored under the sdk verison you have, using symlink would not work
