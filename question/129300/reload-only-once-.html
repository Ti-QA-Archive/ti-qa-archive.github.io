  • I am using that function. The thing is it keeps reloading and reloading continuously..

    In app.js
    function reopen()
    {
    var window = Ti.UI.createWindow({url:"myapp.js"});
    window.open();
    }
    reopen();

    Ti.App.addEventListener('reload',reopen);

    Then in myapp.js when you want to reload the app, call
    Ti.App.fireEvent('reload');

    • your app is leaking, please watch this
    • probably you're adding event listener again and again (happened to me once, I create the same commonJS object twice hence the event was called twice)
    • it would be best to have some functionality in myapp.js to restore to the original state.

    BUT, if you're that lazy, try this way instead (I didn't test this)

    var win = Ti.UI.createWindow({url:'win.js'});
win.open();

Ti.App.addEvenetListener('reload', function() {
  win.close();
  win.open(); // you probably could chain this
});
