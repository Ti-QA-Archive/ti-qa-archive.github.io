How do I programatically add components to an Alloy window

If i have an Alloy "viewfile.xml" file like this:

<Alloy>
    <Window>
    <!-- Some programmatically defined componets should go here -->
   </Window>
</Alloy>

How do I add components such as buttons, or labels to that window programmatically

I have tried things like putting the following in the corresponding "viewfile.js" controller file:

var label=Ti.UI.createLabel({text:'Hello World'});
$.viewfile.add(label);

But that doesn't work, Any ideas?

  • Try $.index.add(label);

    In general:

    You would give your views IDs like

    <Alloy>
    <Window id="myWindow">
    <!-- Some programmatically defined componets should go here -->
   </Window>
</Alloy>

    and access them from your controller by that ID like

    $.myWindow.add(label);
