Accessing XML

I have and XML like this:

<languages>
    <lang>ca</lang>
    <lang>es</lang>
</languages>

How can I iterate through "lang"? I need to insert all <lang> values into an array.

    Try this:

    var xmlStr = '<languages><lang>ca</lang><lang>es</lang></languages>';
var xml = Ti.XML.parseString(xmlStr);
var items = xml.documentElement.getElementsByTagName('lang');

var lang = [];
for (var i = 0, len = items.length; i < len; i++) {
    lang.push( items.item(i).text );
}

Ti.API.info(lang);
