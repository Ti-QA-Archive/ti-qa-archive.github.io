Tableview and modal windows data

Hi there,

I have created a tableview, and then when a row is clicked it opens up a modal window. This all works perfectly and as it should.

But, I am having trouble sending data to the modal window, can this be done? I want to send the date that is clicked (as the tableview is a range of dates). This is what I have done in the tableview section:

var row = Ti.UI.createTableViewRow({
    height: 70,
    className: 'calendarDates',
    selectedBackgroundColor: '#dddddd',
    rowIndex: i,
    currentMonth: newDate.getMonth()
});

Obviously in the above code newDate is a date object set above this.

Then I have this code for the modal window. In the title I have been trying to put table.index and that returns the row index fine, but when I try to access the table.currentMonthis says undefined. Even though I have console logged it and the table object has currentMonth in it with the appropriate value.

    calendarTable.addEventListener('click', function(table){

        var save = Titanium.UI.createButton({systemButton:Titanium.UI.iPhone.SystemButton.SAVE});

        var update_details = Titanium.UI.createWindow({
            title: 'Records',
            backgroundColor: Ti.UI.iOS.COLOR_SCROLLVIEW_BACKGROUND,
            barColor:'#00659E',
            rightNavButton: save
        });

        save.addEventListener('click',function(){
            update_details.close();
        });


        update_details.open({modal: true, modalTransitionStyle: Ti.UI.iPhone.MODAL_TRANSITION_STYLE_FLIP_HORIZONTAL});

    });

Help is greatly appreciated :)

    If you want to send data to the next window, then you can make the custom property of window and send date in that, here is an example below, suppose if you want to send the currentMonth of table view row then you can do the following :

    
calendarTable.addEventListener('click', function(table){

        var selectedRowValue = e.rowData. currentMonth;// getting value of row which is clicked 

        var save = Titanium.UI.createButton({systemButton:Titanium.UI.iPhone.SystemButton.SAVE});

        var update_details = Titanium.UI.createWindow({
            title: 'Records',
            backgroundColor: Ti.UI.iOS.COLOR_SCROLLVIEW_BACKGROUND,
            barColor:'#00659E',
            rightNavButton: save,
            custDate:selectedRowValue,
        });

        save.addEventListener('click',function(){
            update_details.close();
        });


        update_details.open({modal: true, modalTransitionStyle: Ti.UI.iPhone.MODAL_TRANSITION_STYLE_FLIP_HORIZONTAL});

    });

    you can also reterive date in the next window using:

    
var dt = update_details. custDate;

    or by using 

    
var win = Titanium.UI.currentWindow;
var dt = win. custDate;

    Let me know if this help you.
