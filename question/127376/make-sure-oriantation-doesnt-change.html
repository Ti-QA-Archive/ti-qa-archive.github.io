Make sure oriantation doesn't change

  • "orientationModes" does not work if you pass that option to "createWindow" function constructor. You need to set it on window instance:

    var win = Ti.UI.createWindow({
    //does not work
    orientationModes: [ Ti.UI.PORTRAIT ]
});

//works
win.orientationModes = [ Ti.UI.PORTRAIT ];
