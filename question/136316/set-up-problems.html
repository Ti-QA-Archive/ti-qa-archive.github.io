  • What often happens during the Xcode 4.3 install is that Apple installs a second set of Developer Tools without removing the original install. So sometimes you have to switch the 'active' tools being used. Try running:

    sudo xcode-select -switch &#x2F;Applications&#x2F;Xcode.app&#x2F;Contents&#x2F;Developer

    For more information on the hows and whys, check out this blog post here: http://developer.appcelerator.com/blog/2012/03/titanium-and-xcode-4-3-revisited.html
