Problem running app on Simulator

When I try to show specified apps on Simulator I show this erorr

[ERROR] : An error occurred running the iOS Simulator
[ERROR] : [DEBUG] Sessiocould not be started: Error Domain=DTiPhoneSimulatorErrorDomain Code=1 "Simulatore iOS non è riuscito a installare l'applicazione." UserInfo=0x100123c00 {NSLocalizedDescription=Simulatore iOS non è riuscito a installare l'applicazione., DTiPhoneSimulatorUnderlyingErrorCodeKey=-1}

Can you help me?

Thanks

  • Accepted Answer

    There seems to be an issue with some configurations that cause this problem. There's a trouble ticket filed for it, with a fix in testing for the next release of Titanium. In the meantime, the most reliable way to work around this that I've found is to reset the simulator.

    Open the simulator. Choose iOS Simulator > Reset Content and Settings. Once you do that, you'll be able to build & install your app.
