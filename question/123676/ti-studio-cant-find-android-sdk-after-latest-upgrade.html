Ti Studio can't find Android SDK after latest upgrade

This morning I opened up Ti Studio and it prompted that it needed to upgrade. I let it do so, and after it relaunched Ti Studio. Now the tiapp.xml says "Missing the Android SDK that is needed for the Android deployment target" so I can no longer launch the Android simulator.

When I open my Preferences, I can see that the Android SDK Directory is set correctly (C:\Program Files\android-sdk-windows) and in the Default Android SDK dropdown it shows all of the Google API options, so it obviously knows where it is.

Has anyone found a workaround yet for this issue? I'm on Win7 x64.
