Move and Scale label not displaying while taking Photo : Titanium iOS7

Hi my iOS application uses camera to take photo. I'm using the following code for camera

    Ti.Media.showCamera({
        success:function(event) {
           //Success functionality
        },
        cancel:function() {
           // called when user cancels taking a picture
        },
        error:function(error) {
           //Error callback
        },
        saveToPhotoGallery:true,
        allowEditing:true,
        mediaTypes:[Ti.Media.MEDIA_TYPE_PHOTO]
    });

Camera is working fine, but there is an issue. Please look at the image

The left one is the taken using iOS 7 and right one is taken using iOS 6. You can see that the label Move and Scale is missing in the left one.

Now my questions

1. How can I display the label in iOS 7? Am I doing anything wrong there?

2. After taking the image, I'm setting it to an imageView of size 100x100. But the photo get squeezed while it is taken at landscape mode. It's working fine at Portrait. I need it to be resized to fit for entire height. How can I do that?

    1. you can add an overlay with labels, button and click actions
    2. the module imageFactory is your friend. You can use toImage() to determine the width and height and after then you can use the imagefactory to crop or resize.
