Android Disable Auto-focus on TextField

Hi,

My problem is whenever there is a textField in the window, Android will automatic select and focus on the first one. I had found many suggestion on this but no matter how i do it, Android will always focus on one of the textField even after i call blur() in all textfield. How to make it only focus when tab on it?

  • My stupid workaround is to add a dummy TextField out of view as the very first element to the window.

    var win = Ti.UI.createWindow(..);
win.add(Ti.UI.createTextField({hintText: 'DUMMY', top: -200}));
// add the real textFields here
