Full table row tap/click

Hi I have a question I have a code like this:

        for (i = 0; i < json.length; i++){

          var bgcolor = (i % 2) == 0 ? '#FFF' : '#EEE';

            var row = Ti.UI.createTableViewRow({
              height:250,
              backgroundColor:bgcolor,
              className: "nowShowingRow"
            });

            var movie = json[i];


              var movie_thumb = Ti.UI.createImageView({
              image: movie.thumb_poster,
              left:15,
              top:20,
              bottom: 15,
              height: 210,
              width: 200
            });


          var movie_label = Ti.UI.createLabel({
              text: movie.title,
              left:240,
              width: 'auto',
              height: '40' ,
              top: 15,
              textAlign:'left',
              color: '#000',
              ellipsize: 'true',
              font: {
                fontFamily:'HelveticaNeue-Bold',
                fontSize:30,
                fontWeight:'bold'
                }
     });
     }

I managed to display the UI objects on my TableViewRow. I can tap and highlight a row but when I tap on the label or the imageView it doesn't highlight or tap the tableViewRow. My question is how can I tap/click on the UI objects inside my tableViewRow even I tap/click on imageView or label?

Thanks.

    Hi,

    you can set touchEnabled to false to every child objects of row so that you can tap your tableview row object.
