change fontSize on tableview

hello guys, I can't change the size of text on a tableview. how can I do ? this is a code of my tabel:

var tablewin1 = Titanium.UI.createTableView({

data:[
            {title:"row1", hasDetail:true},
            {title:"row2 ", hasDetail:true}
         ]
});
— asked September 29th 2012 by Giuseppe Parasiliti

1 Comment

  • You can set it using the font property on the tableviewrows you are creating there:

    http://docs.appcelerator.com/titanium/2.1/#!/api/Titanium.UI.TableViewRow

    data:[
            {title:"row1", hasDetail:true,font:{...}},
            {title:"row2 ", hasDetail:true,font:{...}}
         ]
});
