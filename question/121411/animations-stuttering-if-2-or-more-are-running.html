  • Ben,

    First things first:

    I see that you are trying to animate on the top of the page 2 different views at the same time (one red, one blue). If you are trying to syncronize the two I would advise you to use a Scrollableview with touchenable disabled. OR you can add this 2+ views to a wrapper view and then animate the wrapper view.

    AND This particular animation is returning "[DEBUG] ignoring new layout while animating in layout Child.." which as far as i can remmember refers to a view that is not completly loaded yet trying to be animated. (you should do a search on this particular issue)

    As a general rule i would never try to create a view and animate it immediately unless i am pretty sure the view is loaded. And you are doing exacly that.

    Hope it helps.

    Daniel
