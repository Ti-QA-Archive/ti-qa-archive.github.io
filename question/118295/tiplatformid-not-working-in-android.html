  • Aruna

    There is no reliable way to determine the device ID, particularly due to a bug in Android 2.2 where the string "DROID2" is returned regardless of the device. See the countless discussions about it in this Google search. That said, this issue was somewhat mitigated by the Titanium core team, in ticket TIMOB-2327.

    However, the official Android Developers advice is to generate an installation ID on first boot. Read this article for a full explanation.

    Cheers
