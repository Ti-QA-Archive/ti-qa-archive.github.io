    If your table view contains more than one section, you need to iterate over the rows to count them - you should be able to adapt the example in this post.

    • Yay! It works! Okay, there's a few ways you can go about this:

      ####Method 1: If there is one section (or no section) in the table

      As Aaron said,

      Titanium.API.info("item count" + table.data[0].rows.length);

      Or, you can

      Titanium.API.info("item count" + table.data[0].rowCount);

      ####Method 2: If there is more than one section in the table

      var total = 0;
for(var i = 0; i < theTable.data.length; i++){
    total = total + theTable.data[i].rowCount;
}

      ####BONUS: What to do when you want to get each row of the table

      var sections = theTable.data;
for(var i = 0; i < sections.length; i++){
    var section = sections[i];
    for(var j = 0; j < section.rowCount; j++){
        var row = section.rows[j];
        Ti.API.debug(JSON.stringify(row));
    }
}

      Or, if you have just one section you can do

      for(var i = 0; i < theTable.data[0].rowCount; i++){
    var row = theTable.data[0].rows[i];
    Ti.API.debug(JSON.stringify(row));
}

      Thank you Titanium Developer community!
