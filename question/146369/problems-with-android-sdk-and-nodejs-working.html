Problems with Android SDK and NodeJS working

I constantly have problems with Titanium working, it will ether not detect any Android SDK platforms under preferences/Titanium Studio/Titanium even though it recognizes them in preferences/Android or when I try to start a new project it doesn't recognize 'node' as a command preventing it from building the project correctly.

I am running Win 7 64.

I have the files in a location without spaces in the path. I know that I have all the environment variables are defined to the right location for android_sdk, JAVA_HOME, NODE_PATH, NODEJS, and PATH. Also I know that I am running the current version of Titanium and NodeJS, with Android 17, 16, 8, and 7.

What am I missing or why does it seem that Titanium will never recognize one of these if the other is working.

    If it doesn't, then node isn't configured properly.
