Video played in simulator but not on the device

Hi,

I added a videoplayer in a window using 

video = Titanium.Media.createVideoPlayer({
        contentURL:'christmas.mp4',
        backgroundColor:'black',
        movieControlMode:Titanium.Media.VIDEO_CONTROL_DEFAULT,
        scalingMode:Titanium.Media.VIDEO_SCALING_MODE_FILL,
        top:10,
        movieControlStyle:Titanium.Media.VIDEO_CONTROL_EMBEDDED,
        height:200,
        width:400,
        autoplay:false
    });
    window1.add(video)

Everything works fine in the simulator. When I run the app in my iphone, the video player frame shows but there's no content playing, only black screen and a loading sign there forever.

I put the christmas.mp4 under /resources
Does anybody know how to fix the problem?

Thanks in advance.

  • I've come across a similar problem before, and it turned out to be the case of the filename. On the simulator it's not case sensitive so (in my case) 'Video.mp4' worked, but on an actual device it did not. Turns out the file was called 'video.mp4' and once I'd corrected my code to use that, it worked fine on both the simulator and actual devices.
