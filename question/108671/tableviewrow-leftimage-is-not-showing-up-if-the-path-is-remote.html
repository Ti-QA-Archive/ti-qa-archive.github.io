TableViewRow leftImage is not showing up if the path is remote.

Hi,

I am creating a table view and for each row, I am adding custom labels and leftImage as follows.

var row = Titanium.UI.createTableViewRow(
      {
        leftImage: 'http://vsalisbury.com/wp-content/uploads/2010/01/91546417.png',
        layout: 'vertical',
        hasChild: true
      });

      var titleLabel = Titanium.UI.createLabel(
      {
        font: {fontsize: 12, fontWeight: 'bold'},
        text: 'Sample Title',
        height: 20,
        left: 50,
        textAlign: 'left'
      });
      row.add(titleLabel);

      var dateLabel = Titanium.UI.createLabel(
      {
        font: {fontsize: 12},
        text: '29th Jan',
        height: 20,
        left: 50,
        textAlign: 'left',
        color: '#aaa'
      });
      row.add(dateLabel);

The rows are being shown up only with two labels that I have added but no image is being shown. If I give a local path for the image, it is showing up. Kindly let me know how to show up the image.
