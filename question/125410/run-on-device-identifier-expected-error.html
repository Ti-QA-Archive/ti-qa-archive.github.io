[ERROR] Error(s) compiling generated Java code

    public static final int 308_fclass_tr_361063647d=0x7f020000;

                           ^

C:\Titanium\PSA\build\android\gen\com\clee\psa\R.java:14: <identifier> expected

    public static final int 308_fclass_tr_361063647d=0x7f020000;

                                                    ^

  • I had this same problem. Check this out: http://jira.appcelerator.org/browse/TIMOB-5964

    It looks like if you have any images or any references to those images within your code, Titanium won't compile your code for Android. Go figure! I had an image named 75off.png and changed it to sevenfiveoff.png and that finally made it work.
