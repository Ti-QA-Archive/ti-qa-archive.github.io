Unable to create new project - Run 'node help' for available commands

Unable to create new project
Run 'node help' for available commands

When I open Titanium Studio the update window always appear with Titanium CLI, Alloy and Node.ACS are not update and all the time that I try to update them I get this error

'Installing Titanium Updates…' has encountered a proble.
Error Encountered during update

Error encountered during update
Failed to install titanium. Please see the Studio log for more information.
Failed to install alloy. Please see the Studio log for more information.
Failed to install acs. Please see the Studio log for more information.
