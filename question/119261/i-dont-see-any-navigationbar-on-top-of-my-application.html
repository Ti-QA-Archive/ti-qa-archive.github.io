I don't see any navigationbar on top of my application

I'm very new to Titanium, and my first question is : what is the reason that I don't see any navigationbar/titlebar on top of my window with this code :

var win = Titanium.UI.createWindow({
title:'that is my title',
backgroundImage: 'Content/Images/bg.jpg',
barColor: '#474436',
navBarHidden: false,
});
win.open();

    i believe you need to place your window in either a navigation group OR a tabGroup with at least one tab to get the navigation bar.

    when using the tabGroup, you can set the tabBar to hidden and I think that will give you the desired results.
