  • Hey Antonie,

    onerror function is to detect any error regarding service call. Yes it will definitely called when there is no internet connection but it will call after a specified time out. And also all the time when onerror called doesn't mean that it's called because of network time out. So I would suggest you to use function (API) provided by titanium for specifically checking internet connection state.

    if (Ti.Network.online == true) {

}

    Please refer documentation here.
