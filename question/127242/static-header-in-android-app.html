Static header in Android app?

I'm trying to set up a really basic layout for heirarchical data, mostly using table views and windows to switch between them.

That works fine. But I would also like to have a static header (and maybe footer) that doesn't move when I animate the transition between windows. However, I can't see how I am supposed to do this- I can only have one window open at a time, and adding a view to the tabGroup (as suggested here) doesn't seem to work with Android.

Is there any solution for this?
