Cannot Create Multiple Background Service

I Want to create multiple schedulelocalnotification. I am doing this via Background Service. When creating the second service, the first one is stops. The second notification shows up. It overwrites the first one.

Any idea. How to create multiple Background Service?

The code is below:

Main.js

  Ti.App.Properties.setString("description", "test");
    Ti.App.Properties.setInt("timer", 15000);    
    Ti.App.iOS.registerBackgroundService({ url:'bg.js' });

bg.js


var description = Ti.App.Properties.getString("description");
Ti.API.info(description);
var timer = Ti.App.Properties.getInt("timer");
Ti.API.info(timer);

Ti.App.iOS.scheduleLocalNotification({
    alertBody:description,
    alertAction:"Re-Launch!",
    sound:"default",
    date:new Date(new Date().getTime() + timer) // 3 seconds after backgrounding
});
