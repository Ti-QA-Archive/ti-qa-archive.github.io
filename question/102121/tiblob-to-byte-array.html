TiBlob to Byte Array

I have seen this question asked a few times, but no definite answer.

I am attempting to send an image to a MS (Soap) Web Service, and the service requires files to be sent as byte arrays. Is there a way to convert TiBlobs to byte arrays?

  • When the Titanium HTTPClient sends a binary file as a part of a POST or PUT, it automatically sets the relevant headers in the request, so if the web service is set up to accept file uploads, it should receive the data as expected.

    With SOAP, I would assume file uploads are handled the same way, but I couldn't say for certain. If you have a working client for your upload service, I would use an HTTP sniffer like Charles or similar to detect the contents of a 'working' HTTP request to your service. That will give you the best starting point for understanding what needs to get sent to your server. Remember that SOAP is just a layer of crap piled on top of HTTP.

    Suds may or may not be helpful to you in this case - you may need to manually construct the SOAP envelope to send to your server.

  • try this:

    var blobStream = Ti.Stream.createStream({ source: myBlob, mode: Ti.Stream.MODE_READ });
    var newBuffer = Ti.createBuffer({ length: myBlob.length });
    var bytes = blobStream.read(newBuffer);
