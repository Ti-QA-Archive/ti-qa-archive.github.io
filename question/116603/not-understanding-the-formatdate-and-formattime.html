Not understanding the formatDate and formatTime

Can someone give more detail on the formatDate and format Time functions? I'm new to this and I'm just don't get it. I do the following:

var foo = Date();
var foo2 = foo;
Ti.API.info('Orriginal Date: ' + foo);
foo = String.formatDate(foo);
foo2 = String.formatTime(foo2);
Ti.API.info('Date: ' + foo + " Time: " + foo2);

My output is Date: null Time: null

What am I not getting?

    John

    I think those built-in functions are intended to be used with localized strings, retrived using Titanium.Locale.getString.

    In your example, I would probably just do the following:

    var foo = new Date();
Ti.API.info('Original Date: ' + foo);
Ti.API.info('Date: ' + foo.getDate()+"/"+foo.getMonth()+"/"+foo.getFullYear() + " Time: "+foo.getHours()+":"+foo.getMinutes());
