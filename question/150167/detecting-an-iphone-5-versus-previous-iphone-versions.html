Detecting an iPhone 5 versus previous iPhone versions

Hi there-

I am looking to create an iPhone app utilizing a ScrollableView control. With this control, I am looking to place a fixed margin at the top and bottom as well as the sides of the control. I would like this margin to appear consistent on all iPhone devices, but since the iPhone 5 has a larger height screen, if I set the control height, the bottom margin would increase on the iPhone 5 (since the control does not have a margin property). Is there any way to set a margin property on the ScrollableView control so I could stretch the height and make the top/bottom margins consistent, regardless of device?

Thanks in advance for any guidance
Chris

    A simple function to doit…

    function IsIphone5()
{
  if( Ti.Platform.displayCaps.platformHeight == 568 )
  {
      return true;
  }
  return false;
}
