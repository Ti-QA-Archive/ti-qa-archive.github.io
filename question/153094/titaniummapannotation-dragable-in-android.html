Titanium.Map.Annotation dragable in Android

Hi,

Is there any way to have an annotation dragable in Android?

I'm ussing this example that don't works:

    var mapview = Ti.Map.createView({
        height : '100%',
        width : '100%',
        mapType : Ti.Map.STANDARD_TYPE,
        region :{
            latitude : 40.41631,
            longitude : -3.70471,
            latitudeDelta : 0.005,
            longitudeDelta : 0.005
        }
    });

    var pin = Ti.Map.createAnnotation({
        latitude :40.41631,
        longitude : -3.70471,
        pincolor : Titanium.Map.ANNOTATION_RED,
        draggable : true
    });

    mapview.addAnnotation(pin);

SDK 3.1.0.GA
Titanium Studio, build: 3.1.0.201304151600

    Hi

    The bad news (only a little bad news though)

    Firstly you must move away from using Google Maps v1 (the one you are using right now) as Google no longer issues API keys for v1.

    You must now use Google Maps v2;

    • Modules.Map
    • Google Maps v2 for Android

    The code is almost identical, but if yu do not swap over you will not be able to put your app in the store unless you have ALREADY obtained the older v1 key.

    The good news

    You can make the annotations draggable by adding the draggable property to the annotation create object. Very easy.
