Unable to make a toolbar translucent

I'm having an issue with making a toolbar translucent. I'm testing on iOS 4.3 and Titanium 1.6.1.

I'm using the following code to create the toolbar and set it as translucent but it always appears solid. Tried a few different colors, etc but it always appears solid.

    toolbar = Ti.UI.createToolbar({
        items: [btnComment, btnSpace, ..., btnSpace, btnAction],
        height: 40,
        bottom: 0,
        borderTop: true,
        borderBottom: false,
        translucent: true,
        barColor: '#999'
    });

The issue happens in both the simulator and on the iPhone 4. The KitchenSink app also shows the same issue for me.

All ideas and suggestions appreciated :)

  • Found the cause of the issue. I had set the scrollview above the toolbar to be the height of the window - the height of the toolbar as the toolbar was originally a solid color.

    Unfortunately, the background of the window happened to be a almost black so the toolbar was transparent, I just couldn't see it.

    Noticed it when I moved the toolbar up a bit when testing.
