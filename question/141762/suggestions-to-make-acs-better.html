Suggestions to make ACS better.

Hello,

I am building an iOS app with ACS and I got a few request and suggestions.

  1. When using Facebook to create a new account it would be perfect if the name of the user was saved in lowercase otherwise querying is very hard (case sensitive). It would be also be perfect if there was a choice to save username and to import the avatar.

  2. When fetching a users profile it would be very good if there was a choice to get the amount of events, checkins and so on that the user has done and the amount of friends he och she got. It is very common for apps to display this information.

  3. I want to be able to add tags to events. I know I can add an array as a custom field but it looks like I cannot query it according to the documentation.

  4. I want to be able to administer friends in the ACS backend. Like all other objects.

If anyone know any solutions to the above please share!
