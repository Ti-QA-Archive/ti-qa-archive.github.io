Open a webview only on a specific date

I would like to have a webview opening once I start my app, but this webview should only appear on a specific date. I guess this is possible, but could someone help me in the right direction?

  • Learn more about Javascript Date Object

    var date = new Date();
var todays_date = date.getMonth()+"/"+date.getDate()+"/"+date.getFullYear();

if(todays_date == '1/7/2012')
{
    Ti.Platform.openURL('http://google.com/');
}
