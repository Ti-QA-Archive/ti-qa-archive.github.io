    Hello Franck

    Your object is not properly constructed, it should be this way: 

    var myObject = [{
    "part": [{
        "name": "John",
            "myAccounts": ['account0', 'account1', 'account2'],
            "lastName": "Doe",
            "age": "35"
    },
    {
        "name": "kate",
            "lastName": "jobs",
            "age": "35"
    }]
}];

    To modify myAccounts for example, you do the following:

    myObject[0].part[0].myAccounts[1] = "This is a new account";

    To access the new value, you do the following:

    console.log(myObject[0].part[0].myAccounts[1]);

    You can check a working example here:

    https://jsfiddle.net/zpud4nm1/3/

    Hope this helps!
