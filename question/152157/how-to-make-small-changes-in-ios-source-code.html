How to make small changes in iOS source code?

I am working with the new ListView component (on iOS 6.1, Titanium 3.1) and have found a reason to make a small change to how it is implemented. I want to make a change in TiUIListView.m, but cannot make the change to turn up in the compiled app. I have tried altering the file that is within the build/iphone folder of my project, but the change seems to be ignored (and it would probably be overwritten pretty fast anyway).

1) How do I make the change?

2) How do I make sure it is not overwritten?

    1. Find where titanium installed the current sdk you are using - usually in (/ or ~/)Library/Application Support/Titanium/mobilesdk/osx)

    2. duplicate the folder with a new name (eg 3.1.mine)

    3. change in 3.1.mine what you need
    4. make a clean project
    5. select the new sdk and compile
    6. tada :)
