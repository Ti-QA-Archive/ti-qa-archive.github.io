  • You could try setting this when creating a window:

    windowSoftInputMode: Ti.UI.Android.SOFT_INPUT_ADJUST_PAN,

    If you are developing the app for other platforms too, then do something like this:

    var softInputMode;
if (Titanium.Platform.osname == 'android')
    softInputMode = Ti.UI.Android.SOFT_INPUT_ADJUST_PAN;
else
    softInputMode = "";

    And in arguments of createWindow, write:

    windowSoftInputMode: softInputMode
