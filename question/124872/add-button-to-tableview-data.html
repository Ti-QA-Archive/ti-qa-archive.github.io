add button to tableview data[]

hi,
i would like to know if it is possible to add a button to a tableview data.
here's my code:

var data =[
{
    title:"Inserisci Nuovo", backgroundColor:'#AFEEEE', clickName:'New',editable:false
},];

var tableview = Titanium.UI.createTableView({
    data:data,
    editable:true,
    moveable:false,
    style:Titanium.UI.iPhone.TableViewStyle.GROUPED,
    backgroundColor:'transparent',
    separatorColor:'#390A0E'
});

        var buttonrs = Ti.UI.createButton({
        style:Titanium.UI.iPhone.SystemButton.CONTACT_ADD,
        left:630,
        clickName:'nuovo'
    });

can i add the button directly in the data[] row somehow?
Thanks

    Yes you can.

    Here is an example:

    // first you need to get row.
// data property of table view component
// consists of array of sections, 
// which consists of array of rows
// note that if table view doesn't have 
// any section defined explicitly, 
// one default is created
var sectionIndex = 0;
var rowIndex = 0;
var row = tableview.data[sectionIndex][rowIndex];
row.add(buttonrs);

    • Just small correction. If you add button to the table view row, you'll loose title (because, you can have title in the row and noting else, or you can add other components in which case you can't have row's title displayed). Because of that, you need to create complex layout to the table view row. It's not hard, you just need one label and you need to set table view row layout to 'horizontal'.

      var sectionIndex = 0;
var rowIndex = 0;
var row = tableview.data[sectionIndex][rowIndex];
var title = row.title;
row.layout = 'horizontal';
var label = Ti.UI.createLabel({ text: title });
row.add(label);
row.add(buttonrs);

  • Create a custom view with the button in that and add that to each row.
    Like this:

    var data = [];
var row = Ti.UI.createRow({
    title:"Inserisci Nuovo", 
    backgroundColor:'#AFEEEE', 
    clickName:'New',
    editable:false
});
var view = Ti.UI.createView({});
var button = Ti.UI.createButton({
    style:Titanium.UI.iPhone.SystemButton.CONTACT_ADD,
        right: 10,
        clickName:'nuovo'
});
view.add(button);
row.add(view);
row.classname = 'rowWithButton';
data.push(row);
