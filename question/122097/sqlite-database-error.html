SQLite Database Error

I am trying to connect to my database so I can insert/replace records from the remote server. I keep getting a null pointer exception error. I have a SQLite database file that I created with RazorSQL that is in my Resources folder. Below is the code I am using to try to connect.

if (realdata[i].errormessage = "SUCCESS") {
var db = Ti.Database.install('../mydb.sqlite', 'mydb');

        db.execute('INSERT OR REPLACE INTO Friends (FriendsID, UserID, Color, PermissionType, CreateDate, Rejected, FirstName, LastName, EmailAddress, PhoneNumber ) VALUES(?,?,?,?,?,?,?,?,?,?)', realdata[i].friendsid, realdata[i].userid, realdata[i].color, realdata[i].permissiontype, realdata[i].createdate, realdata[i].rejected, realdata[i].firstname, realdata[i].lastname, realdata[i].email, realdata[i].phonenumber);
        db.close();
        };

    }

When I use the following:

var db = Ti.Database.install('mydb');

Titanium tells me I do not have a table named 'Friends' when compiling my insert/replace statement.

Any help would be appreciated.

Thanks.

Josh

  • Josh,

    This usually happens when you try to access a database that doesn't exist. Double check the name of the database the your Friends table is in. Also, make sure that you have the correct path to your sqlite database file.
