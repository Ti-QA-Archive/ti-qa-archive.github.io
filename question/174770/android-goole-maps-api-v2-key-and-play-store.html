  • Hi Daniele,

    As you correctly point you have to use the production SHA1 value for the value attribute of
    &lt;meta-data android:name=&quot;com.google.android.maps.v2.API_KEY&quot; android:value=&quot;GOOGLEAPYKEY&quot;&#x2F;&gt;

    1. First you have to create a production keystore, if you have not already. This is necessary to publish an app successfully on Play Store. The procedure of using this key is called "signing" of your application.
    2. Then retrieve the SHA1 of the key and
    3. enter this to google developers console where you have to create a record for your project.
    4. There you will obtain the key that you have to use for your map.

    You can find more info about all these here.

    P.S. You cannot delete an app from the store (just unpublish it) once published so you will have to change the app id or just update the version number of your app (e.g. version 1.0.1).
