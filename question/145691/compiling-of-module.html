    Yeah, not like its in the DOCS or anything:
    https://wiki.appcelerator.org/display/guides/iOS+Module+Development+Guide

    
The markdown2 module is not on the PythonPath by default for the Mac's pre-installed Python interpreter. Without this installed, build.py will fail when it tries to generate the module docs. If you have not already done so, then follow these steps to install markdown2:

Download the latest release from https://github.com/trentm/python-markdown2
Unzip the resulting download*
Using the setup.py script in the directory that was created from expanding the zip file you downloaded, install with `sudo python setup.py install`
Re-run build.py

  • Hi Rainer,

    the original titanium.xcconfig looks like:

    //
//
// CHANGE THESE VALUES TO REFLECT THE VERSION (AND LOCATION IF DIFFERENT)
// OF YOUR TITANIUM SDK YOU'RE BUILDING FOR
//
//
TITANIUM_SDK_VERSION = 1.8.1


// 
// THESE SHOULD BE OK GENERALLY AS-IS
//
TITANIUM_SDK = /Library/Application Support/Titanium/mobilesdk/osx/$(TITANIUM_SDK_VERSION)
TITANIUM_BASE_SDK = "$(TITANIUM_SDK)/iphone/include"
TITANIUM_BASE_SDK2 = "$(TITANIUM_SDK)/iphone/include/TiCore"
HEADER_SEARCH_PATHS= $(TITANIUM_BASE_SDK) $(TITANIUM_BASE_SDK2)

    so you only need to change 

    TITANIUM_SDK_VERSION = 1.8.1

    to 

    TITANIUM_SDK_VERSION = 2.1.4.GA

    Please check, if you need to prefix /Library/Application Support/Titanium/mobilesdk/osx/$(TITANIUM_SDK_VERSION) with ~.

    Without ~ it's OSX pre-Lion!

    Greetings,

    Matthias
