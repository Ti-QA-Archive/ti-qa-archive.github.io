Dependancy / Module Management

Here is a suggestion. I think it would be great if Titanium had some form of dependancy management of native and js modules. I'm thinking on the lines of ruby gems and apache ivy.

In broad terms this is the proposal, either the cli command would parse the tiapp.xml file and automatically download and install the required, open sourced, modules. Alternatively, a separate dependancy file would be parsed and then modify the tiapp.xml file as required.

I'm not sure if Appcelerator are interested in developing it as they are working on the paid market. Perhaps it could integrate with it (?).
