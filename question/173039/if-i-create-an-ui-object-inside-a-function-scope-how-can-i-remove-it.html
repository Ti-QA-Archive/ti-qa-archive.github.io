If I create an UI object inside a function scope how can I remove it?

Hi, for garbage collector which is better to remove: an UI global or an UI created inside a function? And, in this case, how can I remove it if when I try to use parent.remove(objName) I get of course an error because it is out of scope?
Example:

function customView(){
    var myView = Ti.UI.createView({
    some stuff....
    });
    return myView;
};
...
win.add(customView());
...
win.remove(HOW CAN I POINT TO myView?);

    Define myView outside the function 

    var myView = '';

function customView(){
    myView = Ti.UI.createView({
    some stuff....
    });
    return myView;
};
...
win.add(customView());
...
// win.remove(HOW CAN I POINT TO myView?);
win.remove(myView);

    Defining outside a function doesn't mean a global, It mean the scope of variable live and dies with in that controller.
