Module cannot access to its resources

Hi All,

I'm creating a module that uses an external SDK for Android. The SDK is in form of a project, so in Eclipse I added it by right-clicking the module's project and adding from project section in Java Build Path. I also copied the SDK's res folder into the module's platform folder.

The module builds successfully and I added it to my Titanium app, but the app's build failed because it could not access resources under the module's platform/res folder. So, I copied the files from the module's platform/res folder to the app's platform/res folder. Then the app's build succeeded.

Now, the app runs and crashes when it tries to use the module's function. Looking at logcat, the app seems to be unable to access those resource files.The log says

VFY: unable to resolve static field 1417(<field's name>) in Lnet/<sdk's name>/sdk/view/R$string

as such.

This may be a hard problem, but does anyone know how to add the resource files from a module to an app?

Plase notify me if any more info is needed. Thanks a lot in advance…
