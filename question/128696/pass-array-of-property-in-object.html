Pass array of property in object

Hello,

I want to pass a new property of object in array format

to show you.. it goes like this

var btn = Ti.UI.createButton({
    title: 'This is button',
    width: 100,
    height: 100
});

//I want to add property in array format
var adProperty = {
    top: 5,
    left: 5
}

//how can i insert this new property?? to the btn?
//any ideas??

Thanks =)…CodeStrong…

  • Accepted Answer

    var btn = Ti.UI.createButton({
    title: 'This is button',
    width: 100,
    height: 100,
    adProperty : {
        top: 5,
        left: 5
    }
});

    or

    var btn = Ti.UI.createButton({
    title: 'This is button',
    width: 100,
    height: 100
});

btn.adProperty = { top: 5, left: 5 };
