  • Hello Sven Wiese-Solty,

    Thanks for the question. Here I have added my sample code for your issues. The function of android: home is to return immediately to the home screen, pausing any currently opened apps The Home button behavior cannot be overridden.

    Test Case :

    var win = Ti.UI.createWindow({
    backgroundColor : '#36A9E1',
    barColor : '#66B2FF',
    fullscreen : true
});

 win.addEventListener('home', function(e) {
 });
win.open();

    For more information
    Android Home
