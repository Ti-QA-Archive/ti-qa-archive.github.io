Titanium.Bytes writing binary file formats

I've been trying to write a binary format using Titanium Desktop, essentially a byte stream. Yet so far I've been unable to succeed. Does anyone have an idea?

This is what I have so far.

var bytes = Titanium.API.createBytes('');

bytes[0] = 3;
bytes[1] = 0;
bytes[2] = 0;
bytes[3] = 0;

var file = Titanium.Filesystem.getFile('test.file');
var stream = Titanium.Filesystem.getFileStream($file);

stream.open(stream.MODE_WRITE, true);
if (stream.isOpen()) {
    try {
        stream.write(bytes);
    } catch (e) {
    console.error(e);
    }

    stream.close();
} else {
    console.error('Could not open stream');
}

When I hex edit the file I get 04, which is only 1 byte. I can only assume I'm using Titanium.Bytes incorrectly.
