    • Here's a scenario for you…

      User A downloads the app, and logs in. They are subscribed to the channel.

      User A deletes the app WITHOUT logging out

      User A sells phone to User B

      User B downloads same app, and logs in.

      There are now two users subscribed to that channel that have the same device token, which means that user B will receive notifications for user A

    • Not going to happen. User B will have reset the phone they bought to associate with their Apple ID. So contents will have been wiped and in any case if user B re-downloaded App all traces of the sign-in token will have been removed from device
