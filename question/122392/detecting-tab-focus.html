Detecting tab focus

I have a tab group that consists of windows with webviews. I'd like to be able to reload the webviews when each tab is selected since they only load on app launch.

(The webviews point to television listings, so currently you have to completely quit the app to update them)

Is there a way to detect that a given tab has been selected? If I was doing this in Obj-C, I'd use viewDidAppear - is there anything like that exposed to Titanium?

I've looked into trying to detect the tab click event, but didn't get very far with that.
