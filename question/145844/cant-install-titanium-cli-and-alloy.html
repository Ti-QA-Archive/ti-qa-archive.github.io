    • From the Terminal, run node -v. It should be 0.8.16 or newer.

      Next, run npm -v. It should be 1.1.69 or newer.

      Next, run sudo npm install -g titanium. That will install the latest Titanium CLI.

      To verify the Titanium CLI installed correctly, run titanium -v and it should display 3.0.22.

      Next grab a SDK by running titanium sdk install 3.0.0.GA --force --default. After a minute or two, you will have the 3.0 SDK installed and you should now be able to create and build projects!

    • @Ron, looks like npm is either not installed or not in your path.

      Run ls &#x2F;usr&#x2F;local&#x2F;bin&#x2F;npm to verify npm does not exist.

      Next run ls &#x2F;usr&#x2F;local&#x2F;lib&#x2F;node_modules&#x2F;npm to see if npm is even installed. If that npm directory exists and there are some files, you simply need to create the symlink again by running:

      sudo ln -s /usr/local/lib/node_modules/npm/bin/npm-cli.js /usr/local/bin/npm

      If npm does not exist in /usr/local/lib/node_modules, then you need to download and re-install Node.js which comes with npm.

  • I did get this to work!

    You have to create a ".bash_profile" in your home directory with: PATH="$PATH:/usr/local/bin/"
    By default the PATH doesn't include the /usr/local/bin/ location, so if you add manually the installation will work.

    I hope this help anybody.
