  • hi

    I had basically same problem solution was to add environment variable "Path" like this

    %SystemRoot%\system32;%SystemRoot%;%SystemRoot%\System32\Wbem;C:\Program Files (x86)\Java\jdk1.6.0_29\bin

    And after that restart is necessary

    — answered May 4th 2012 by Mikk Laos

    • I helped someone yesterday who was having no end of problems due to this path being missing from their Windows system.

      Btw, be careful with the java path - if you install a new version and the jdk1.6.0_29 part of the path changes, it may cause problems. Always verify that this path is correct after updating Java.
