    by default keyboard has return key as enter key 

    textarea = Ti.UI.createTextArea({
suppressReturn: false,  
width: 60,
height: 60,
top: 0
});

textarea.addEventListener('return', function(e) {
Ti.API.info(' Return key  Detected');
});

  • Simply add an event listener for the return event. This will be triggered every time the user clicks the return (next) button.

    Also don't forget to set the Next attribute to your keyboard type so the Done button label is replaced.

    yourTextField.returnKeyType = Titanium.UI.RETURNKEY_NEXT
