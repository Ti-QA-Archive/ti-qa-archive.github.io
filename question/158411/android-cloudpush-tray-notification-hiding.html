Android cloudpush tray notification hiding

On Android is there any way to hide the tray notifications when my app is in foreground (but still receive push callback events in the app)? And only show tray notifications when the app is closed or in background? I tried using any combination of:

Titanium.CloudPush.showTrayNotification
Titanium.CloudPush.showTrayNotificationsWhenFocused

Using SDK 3.1.3GA and GCM. Thanks.

  • Don't set Titanium.CloudPush.showTrayNotification Titanium.CloudPush.showTrayNotificationsWhenFocused to any boolean value. Reason is by default showTrayNotification: true and showTrayNotificationsWhenFocused:false that means tray notification would be seen only when the app is in background or closed.
