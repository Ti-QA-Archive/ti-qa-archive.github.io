How to use Alloy to move image from gallery to app and then display

I am completely lost on this.
I can get an image from the gallery (or camera) and save it to my directory, but I can't figure out how to then pass that image to Alloy (the tss file) to display. It seems like it should be simple… what am I missing?

  • how can I tell the tss what image to display?
    or
  • how can I tell the xml what image to display?
    All the examples are hard coded, pre-existing images. How do we get a new image, and pass that path?

    Since you don't have any code… Hard to tell you exactly what to do, but basically, if you had an imageview defined in your XML file, with the id of "myimage" then in your code, your js file would have code where you would put

    $.myimage.image = (insert the path to your file here)

    Hope that makes sense.
