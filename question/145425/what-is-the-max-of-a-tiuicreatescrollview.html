  • There are two sizes for this. The size of the viewable area (height), which I would set within the screen, and the size of the content (contentHeight). Now if you using a scroll view and a layout of vertical, it will just keep adding the views into it.

    So a very simple answer is that if you add to many then it will become very slow and possibly crash the app because of lack of memory. You really need to make sure that the content of a scrollview is relevant and obvious to the user.

    Hope this helps

    T.
