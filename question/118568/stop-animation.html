Stop Animation

Platform: iOS 4.2, Titanium SDK 1.5

Is it possible to stop an animation before it completes? I've tried invoking animate() without arguments on the view that is being animated but no luck. This is absolutely essential and I just cannot figure it out. Please help.

thanks,

jason

  • The best I have been able to manage is to send an empty animation

    var stopAnim  = Titanium.UI.createAnimation();    
myView.animate(stopAnim);

    But this resets view to it's original state, which probably isn't what you want to do.
