  • Hi Gopinath

    Normally webview is used in many different ways I am mentioned below for which so far i have used.

    1) Displaying the HTML text in the app with all HTML formatting.
    2) Displaying the page of content from any webURL.

    I would suggest not to make the full app inside the webview. else there is no use of appcelerator titanium you can simply use the Phonegap Which supports the same. I would rather prefer to use webview when you have to display some content which are in HTML source or when you need to display any weburl, youtube or any other video.

    Sorry if i have missed any things.
