  • I think you are talking about custom property.

    var view =Ti.UI.createView({
      nameValue :  ' dev' , // this is custom property
      myValue :' Dev eloper'   
})


alert(view.myValue); // accessing custom property

  • Hi,

    You can set custom variable in following manner:

    var view = Titanium.UI.createView({    
        customProp:"Custom Prop Value", // this way you can set custom property.
        width:100,
        height:100,
});

    You can retrive custom value from view in following manner:

    var custval  = view.customProp;
