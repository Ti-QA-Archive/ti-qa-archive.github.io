How to get path for sd card (/storage/extSdCard/) ?

I have used 

Ti.Filesystem.getExternalStorageDirectory()

to get sd card path. In htc device, the path is correct.

/mnt/sdcard/

However, in samsung device, it seems to be device memory path instead of sd card path. 

/storage/emulated/0/

My question is how can i get the exact sd card path? 

/storage/extSdCard/

  • I was able to get the SD card path using the following:

      var mounts = Ti.Filesystem.getFile('file:///proc/mounts').read().toString().split('\n')

  var sdCard

  for (var i in mounts)
  {

    var mount = mounts[i].split(' ')

    if (mount[0].match(/^\/mnt/))
    {
      sdCard = mount[1]
    }

  }
