ACS Cloud Data Export

Hello,

I'm considering using Appcelerator's 'Explorer' package for a small start-up project, along with the ACS option. I like the sound of some of the ACS features and speed of getting up-and-running.

I've not been able to find anywhere to export data though. And I've been unable to find any Q&A topics on it either. Could someone please confirm that there is a way to export data from ACS wether it be for personal backup or for potential future needs of moving it to another system?

Also, from what I read it sounds like ACS data APIs could just as easily be consumed by a website. Is that correct? It's not limited to being used by only other Appcelerator systems, right?

Thank you for any help and advice!

    There is an export button in the ACS web maintenance screen for each app, currently you can only export once every 7 days. The interfaces for all of the acs objects are available through calls to the ACS servers. The docs are available here
