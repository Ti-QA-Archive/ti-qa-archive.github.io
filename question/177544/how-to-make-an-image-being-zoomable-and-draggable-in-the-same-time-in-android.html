How to make an image being zoomable and draggable in the same time in Android

I was looking for all possibilities for pinching and dragging image in android and the best solution for it was to use a webView which lets the user to zoom and drag the photo but this solution wasn't great for my customer as there is a big empty area around the photo and i also couldn't center it or to reduce that empty space around the photo as it seems being added for the zoom effect can work. So right now i'm trying to use both the draggable plugin (ti.draggable) and the pinch event as shown in this question (in this question already someone was complaining that once he zooms he cannot move the image to see the hidden parts of the photo : 

        I have tried this and some other ways also, I am able to zoom with 3 different ways, including u mentioned above. But the problem is after zooming you can not move image. (Nabeel Munawar)

) but for what i tried to test only the draggable property was working not the pinch in this case :


var imgvH = 750;
var imgvW = 750;

    var free = Draggable.createView({
        top:0,
        left:0,
        width:imgvW,
        height:imgvH,
        backgroundColor:'blue'
    });




    free.addEventListener('pinch', function(e) {

        Ti.API.error('e.scale  : '+e.scale);

        var scale = e.scale;
        this.height = imgvH * scale;
        this.width = imgvW * scale;


    });

    free.addEventListener('touchstart', function(e) {
         imgvH = free.height;
         imgvW = free.width;
    });



    bodyView.add(free);

and also i tried another solution by using a scrollView with the maxZoomScale property which let the zoom possible but in this case the result was worst as all was blocked and nothing works.

So i hope someone tells me if in android there is a better new solution to get those effects working together.
