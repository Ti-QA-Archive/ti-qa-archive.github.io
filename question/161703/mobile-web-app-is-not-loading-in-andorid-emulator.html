Mobile Web App is Not Loading in Andorid Emulator

I am new in titanium appcelerator i have developed one app in native code but only the content loaded in html and jquery mobile. Its perfectly running in Mobile web Browser. Then i have loaded the same app in Emulator but its not running. Please advice.

  • Hello Shiva!

    We need to check if your emulator and the Android SDK is properly configured.

    Test Instructions

    Create a new mobile project (classic), with support for android, and compile it. If everything is fine, you should see the app running in the emulator.

    Workaround

    • Try to run your new mobile project.
    • If you get an error -1, after getting the error, go to the "play" button and select "Run Configurations". Select other version of Android, and click Apply and then OK.
    • Run it again.
    • Should be working now.

    We added a bug about it, you can find it here in TIMOB-16061.

    You can find more information here.

    Let me know how it goes.

    Best,

    Mauro
