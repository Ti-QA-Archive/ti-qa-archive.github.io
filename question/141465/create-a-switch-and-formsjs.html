  • You just need to add the switch type to the form creator.

    forms.js

    //add this to the top types
exports.TYPE_SWITCH = 'switch';

//add this to the addField function
else if (type === exports.TYPE_SWITCH) {
    fieldObject = Ti.UI.createSwitch({value:0});
    handleStyle(form, undefined, title);
}

    app.js

    //add this to the fields array, right above submit
{ title:'Agree To Terms', type:'switch', id:'agree'},

    You may have to play with the styling in particular for the switch and the label to get them to look how you want them to, but thats the basic idea of it all.
