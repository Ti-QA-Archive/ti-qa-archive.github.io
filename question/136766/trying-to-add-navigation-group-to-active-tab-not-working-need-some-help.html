Trying to add navigation group to active tab, not working, need some help

I have a tabbed app. For one of the tab windows, I have a button which I would like to open a navigation group in that tab, with a sequence of windows. Someone already pointed our to me that a window has a "built in" nav group, which I can call with tabGroup.activeTab.open(win2); which works, but I seem to be unable to add a group sequence to this. Here's my code, which keeps failing with undefined is not an object evaluating win.(add)


win = Ti.UI.currentWindow;
var tabGroup = Titanium.UI.createTabGroup();

var win1 = Titanium.UI.createWindow({
    title : 'Start Here',
    backgroundColor : '#fff'
});
var tab1 = Titanium.UI.createTab({
    icon : 'KS_nav_views.png',
    title : 'Start Here',
    window : win1
});

var simpleButton = Titanium.UI.createButton({
    width : 160, height : 40, top : 10, title : "Simple"
});
simpleButton.addEventListener('click', function(e) {
    tabGroup.activeTab.open(win2);
});

// root window for simple
var win2 = Titanium.UI.createWindow({
    title : 'Estimating a Simple Roof',
    backgroundColor : '#000'
});

var nav = Titanium.UI.iPhone.createNavigationGroup({
   window: win2
});

win.add(nav);

var win3 = Titanium.UI.createWindow({
    backgroundColor: 'blue',
    title: 'Blue Window'
});

// ... other stuff including window for tab 2

//  add tabs
//
tabGroup.addTab(tab1);
tabGroup.addTab(tab2);

// open tab group
tabGroup.open();

What am I doing wrong, please?

SDK 2.0.1, iOS 5.1, Studio 2, OS 10.7.3

    Just keep calling tabGroup.activeTab.open(...); when you want to open your deeper windows. The tabGroup will automatically handle a navigation group for you with no need to define one.
