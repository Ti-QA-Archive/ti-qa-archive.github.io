Ti.Cloud - Searching custom_fields

Having issues retrieving results using the Cloud.events.search function.
Its almost as if it ignores the search fields and pulls back everything regardless. 

var _search = {
    "custom_fields": {
        "type": "" +sport+ "",
    }
}

var showEvents = Cloud.Events.search({foo:bar}, function (e) {
    Ti.API.log(e);
});
var showEvents = Cloud.Events.search(_search, function (e) {
    Ti.API.log(e);
});
var showEvents = Cloud.Events.search(function (e) {
    Ti.API.log(e);
});

The lines above all yield the same results. I just want to query events for a specific sport in the custom fields by type.
Maybe I'm going about it all wrong but, I'm wondering if anyone has had success with this.
As a workaround, I can pull down all the events and loop through but, its very counter productive.
Thanks in advance.

Ti.Cloud 2.3.0

    you need to do a query not a search…

    http://cloud.appcelerator.com/docs/search_query#where

    • proper structure is 

      var _search = {
    "where":  JSON.stringify({
        "type":  sport
    })
}
Cloud.Events.query(_search, function(e){
    Ti.API.log(e);
});
