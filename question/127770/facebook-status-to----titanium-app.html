  • Jonny,
    Titanium has a Facebook API for using the Social Graph provided by Facebook. I'm not exactly sure what you want to do but you can get a users FB status from the social graph by using this path: https:&#x2F;&#x2F;graph.facebook.com&#x2F;[STATUS ID]

    Check out http://developers.facebook.com/docs/reference/api/status/
