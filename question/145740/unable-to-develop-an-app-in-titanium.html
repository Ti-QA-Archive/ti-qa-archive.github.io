    • Hi Afeeza,

      i was searching Q&A and google for that problem.

      Are you really sure your setup is correct according Setting up Titanium Studio, Oracle JDK 32 Bit, Android SDK Min Android/SDK Version ?

      Also check path and environment settings

      Have you rebooted your machine after installation?

      If all of the above is double checked:

      • delete all existing avd's using android virtual device manager.
        From command prompt start android inside your android-sdk\tools folder. From menu Tools->Manage AVDs..
      • from Titanium Studio delete all Android run configurations.
        Menu Run->Run Configurations…
      • create a new project in Studio and run on Android Emulator (Studio will create a new AVD for you)
