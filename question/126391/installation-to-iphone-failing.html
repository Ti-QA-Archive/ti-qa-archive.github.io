  • Accepted Answer

    When you install to device, you are typically prompted by Titanium Studio to upload a provisioning profile through Studio. Did you install your new provisioning profile through Studio? If not, you may need to do it this way.

    If you can't get it resolved, you may also want to try creating a new project, copy your existing project's resources to it, then trying to install to the device again. There's a possibility something got messed up with the project settings when you switched profiles.

    • Installing via Titanium Studio would indeed use iTunes syncing, and it would resolve the issue with you changing your provisioning profile as you would be able to assign a new one to the build. What you could try is to attempt to install from TiStudio, wait for it to prompt you about your new provisioning profile, upload the profile that way, then cancel out the install from TiStudio. You may need to build your project again. Try your xcode install again after that.
