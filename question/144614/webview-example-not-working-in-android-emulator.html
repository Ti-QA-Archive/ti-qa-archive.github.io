WebView example not working in Android Emulator

Hi,

I have created a sample application using WebView component.

Application Details are as follows:

Application type: Mobile

Titanium SDK: Titanium SDK version: 2.1.3 (10/02/12 16:16 15997d0)

Platform & version: Android V8

Device: Android emulator

Host Operating System: Windows XP

Titanium Studio: 2.1.2.201208301612

Code(FirstView.js):

    var win = Ti.UI.createWindow();

    • Pankaj, if the error [INFO][Launcher.Model( 127)] not binding apps: no Launcher activity still exists, also another solution may help.

      Anthony described this procedure in question Can't run Android App in Emulator.

      • delete all existing virtual devices using AVD Manager
      • inside Titanium Studio clean your project
      • launch your project from Titanium Studio again (Studio will create a new virtual device for you)

      Good luck,

      Matthias
