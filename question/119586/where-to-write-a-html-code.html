  • HI Manu Ranjan,

    The best way is create with Titanium code for native speedy, but if you really need create your app from a HTML, you need create a HTML container. To do this, open app.js, clear actual code and paste this:

    
// Create Window
var myWindow = Ti.UI.createWindow();

// Create WebView loading "/resources/index.html"
var myWebView = Ti.UI.createWebView({
        url:'index.html'
});

// Add WebView to Window
myWindow.add(myWebView);


// Finally show window
myWindow.open();

    Now save your page inside resource folder with name "index.html"

    Run and Enjoy!

  • Titanium is meant to be written purely in javascript. Where in html you might create a label like so:

    <label id="myLabel">Hello World</label>

    In Titanium you would do:

    var label = Ti.UI.createWindow({ id: "myLabel", text: "Hello World" });
Ti.UI.currentWindow.add(label);

    If you want to have an app written purely with html, javascript, and css then your best bet is using some other framework that is only a wrapper and then writing your code in something like jquery mobile. But like Adriano mentioned, if you go this route then you don't get the look and feel of the native controls.
