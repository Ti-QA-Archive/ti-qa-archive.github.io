Attach iOS toolbar to textfield keyboard in Alloy

I'm confused as to how a toolbar (Titanium.UI.iOS.Toolbar) is meant to be attached to a textfield or textarea's keyboard in Alloy.

In particular, I don't understand how two textfields are meant to share one toolbar. How is the keyboardToolbar property meant to be used? Do you just put in the id of the toolbar? I tried that and it had no effect.

  • First off, there's a complete example of using the keyboardToolbar property in Alloy in the docs: http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.TextField

    To use the same toolbar in multiple textfields, just make the textfield in its own view and controller:

    index.xml

    <Alloy>
    <Window>
        <Require src="kt"/>
        <Require src="kt"/>
    </Window>
</Alloy>

    index.tss

    "#index": {
    backgroundColor: "white",
    fullscreen: false,
    layout: 'vertical'
}

    kt.xml

    <Alloy>
    <TextField>
        <KeyboardToolbar>
            <Toolbar>
                <Items>
                    <Button systemButton="Ti.UI.iPhone.SystemButton.CANCEL" />
                    <FlexSpace/>
                    <Button systemButton="Ti.UI.iPhone.SystemButton.CAMERA" />
                    <FlexSpace/>
                    <Button style="Ti.UI.iPhone.SystemButtonStyle.DONE">Send</Button>
                </Items>
            </Toolbar>
        </KeyboardToolbar>
    </TextField>
</Alloy>

    kt.tss

    "TextField[platform=ios]": {
    borderStyle: Ti.UI.INPUT_BORDERSTYLE_BEZEL,
    keyboardToolbarColor: "#999",
    keyboardToolbarHeight: 40,
    top: 20,
  height: 35,
  width: 300,
  value: "Focus to see keyboard with toolbar"
}

  • I usually just do it in the controller

    var toolbarDone = Ti.UI.createButton({
  systemButton: Titanium.UI.iPhone.SystemButton.DONE
});

toolbarDone.addEventListener('click', function() {
  targetWeightText.blur();
});

var flexSpace = Titanium.UI.createButton({
  systemButton: Titanium.UI.iPhone.SystemButton.FLEXIBLE_SPACE
});

$.myTextfield.keyboardToolbar = [flexSpace, toolbarDone];
