How Can I Send From HTTPClient in GZip Format?

Hi,
Using HTTPClient I'm sending out an XML-RPC request to a Java server from an iOS app. All seems to be working well, until the server throws an exception because the message isn't in GZip Format: 

Error: java.io.IOException: Not in GZIP format     at java.util.zip.GZIPInputStream.readHeader

Is there a way that I can get the message to be send in GZip format - is it just a matter of changing the headers?

Thanks
James

  • Hi,

    Accept-Encoding header should help.

    var xhr = Ti.Network.createHTTPClient();
...
xhr.setRequestHeader('Accept-Encoding', 'gzip, deflate');

    Regards,

    Minh
