Changing only one label attribute on Android

I use the if (Ti.Platform.name == &#39;iPhone OS&#39;) a lot but it only works with single commands. However sometimes I want labels to look only slightly different on Android, but I don't want to have to recreate the whole label for Android.

For example:

var label4 = Titanium.UI.createLabel({
    color:'#000',
    text:'My text',
    font:{fontSize:14,fontFamily:'Helvetica Neue'},
    textAlign:'center',
    width:'auto',
    top:300
});

If I wanted the font size to be 16 on Android, I can't just put if (Ti.Platform.name == &#39;Android&#39;) in the middle of thecreateLabel` method. Is there a way to modify only a single attribute for a label without recreating it for each platform?

  • One option would be something like this:

    var label4 = Titanium.UI.createLabel({
    color: (Ti.Platform.name == 'Android') ? '#000' : '#fff',
    text:'My text',
    font:{fontSize:14,fontFamily:'Helvetica Neue'},
    textAlign:'center',
    width:'auto',
    top:300
});

    The above would set the color to #000 for android and #fff for everything else. Another option would be to use the *.jss support added in SDK 1.5.0, which you may find more convenient for styling differences for each platform. If you have not seen it, I recommend you watch this webinar.
