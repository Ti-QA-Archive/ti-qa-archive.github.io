  • Hi Adid,

    Take a look at the Navigation Window, which will let you create the exact window which was present on the link you provided in your question.

    Navigation Window let you create a group of any number of windows & to create a page like the Sounds one on screenshot, you would create a root Window in Alloy & then you have all elements available in Alloy framework to create the page exactly like the Sounds one.
