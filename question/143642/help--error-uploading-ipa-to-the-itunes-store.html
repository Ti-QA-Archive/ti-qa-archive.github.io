  • Okay… I finally found where the issue was…
    As I supposed from the beginning, it comes from the Java Library !

    After having tried every updates on the web, nothing worked. If you don't see the "Java preferences" tool under "Application/Utilities" folder, you have to manually remove every files of Java 1.7
    In my case, I only keep the 1.6 version. To be sure that the version is 1.6 you can check the current version by typing : $ java -version.

    The 1.6 jdk is here : /System/Library/Java/JavaVirtualMachines. So remove every files in : /Library/Java/JavaVirtualMachines
