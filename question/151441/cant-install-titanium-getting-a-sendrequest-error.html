  • I got the same error when installing Titanium Studio on Windows 7 32 bit OS. To solve this issue, I downloaded and manually installed the prerequisites (Avoid downloading prerequisites using Titanium Installation wizard!!!)

    1. Download and install 32bit JDK from http://www.oracle.com
    2. Download and install node.js from http://nodejs.org/download/
    3. Download and install Git from http://git-scm.com/download/win
      When done with the above, run Titanium Installation, everything should be fine
