Upload image to parse using Titanium

Hi, What is the parse class url that I should set to upload image from Titanium to parse?
Currently my url is https:&#x2F;&#x2F;api.parse.com&#x2F;1&#x2F;classes&#x2F;UserInfo. But I know I shouldn't be using this and must be using something like https:&#x2F;&#x2F;api.parse.com&#x2F;1&#x2F;files&#x2F;photo.jpg. The issue is that I don't understand how the second url works. Can anyone please explain to me that?

  • Hey whats up,

    So to perform a Profile Upload you need to have you Image Blob, after that, execute this code:

    exports.uploadProfileImage = function(image){

    var request = Titanium.Network.createHTTPClient({

        onload: function(e) {

            var result=JSON.parse(this.responseText);
        user.set("profileImage",result.url);
            user.save();
        },
        onerror: function(e) {
            alert(e.message);
        }
    });

    // Register device token with Parse
    request.open('POST', 'https://api.parse.com/1/files/pic.jpg', true);
    request.setRequestHeader('X-Parse-Application-Id', 'YOUR_APP_ID');
    request.setRequestHeader('X-Parse-REST-API-Key', 'YOUR_REST_API_KEY');
    request.setRequestHeader('Content-Type', 'image/jpeg');
    request.send(image);
}

    the "result.url" is the url of your picture, add this to your user object and you are good to go! :)
