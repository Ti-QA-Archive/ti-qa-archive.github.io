  • I'm working on a way to do this.

    It's possible to add and position UI elements (buttons, images, etc) to the tabGroup. You then use a 'blank.png' (transparent image) as the 'official' icon and set the title to ' ' (empty) for each tab.

    Then create some listeners for the clicking on the elements to set the active tab.

    Things get a little tricky when you want to open a new window, so you need to move/animate/hide the 'tabs' when required.

    Hope this helps.
