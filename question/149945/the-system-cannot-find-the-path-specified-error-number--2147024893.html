  • I had the same problem and after hours of trial and error I found out that it's a problem with the IE10. To solve the problem simply open Internet Explorer and remove browsers history (Ctrl+Alt+Del). Then reset IE to it's Default Settings. Go to IE Tools -> Internet options, open Advanced tab and click Reset… button.

    The Titanium Studio Dashboard should work fine from now on.

    Peace,
    Andrew
