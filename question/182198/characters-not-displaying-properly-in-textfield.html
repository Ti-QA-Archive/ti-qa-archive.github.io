  • I tried something else that worked out just fine :)

    How i had set up my screen until now (old hierarchy) :

    • Window

    • View > added to the window

    • 2nd View (regional smaller view for the TextField) > added to the 1st View

    • TextField > added to the 2nd View.

    Modifying my layout worked out just fine and below is what i did :

    • Removed the 2nd view

    • Created a TableViewRow > added the TextField to it

    • Create a TableView > added the TableViewRow to it

    Below is the code i used just in case someone finds it helpful :

    //Create Table Row
var selectVehicleData = [];
var selectVehicleRow = Ti.UI.createTableViewRow({
    height:'auto',
    backgroundColor:'#FFF',
    hasDetail: true
});

selectVehicleRow.add(selectVehicleTextField); //add TextField to the Row
selectVehicleData.push(selectVehicleRow);

//Create the TableView
var selectVehicleTable = Ti.UI.createTableView({
    data:selectVehicleData, // added  the Value from the Row into the TableView
    rowHeight: 48,
    height: 70,
    top:-20,
    scrollable: false,
    backgroundColor: 'rgba(255, 255, 255, 0)',
    bottom:15
});
newTripScrollView.add(selectVehicleTable); //added the TableView into the window main view (1st view)

    Keep coding and helping out
