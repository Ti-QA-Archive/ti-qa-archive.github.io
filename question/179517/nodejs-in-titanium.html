nodejs in titanium

Hello everyone! am new to titanium I installed this:npm install mcrypt and I put this code in alloy.js

var mcrypt = require('mcrypt');

var algos = mcrypt.getAlgorithmNames();
console.log(algos);

I get an error 404, says that the library does not exist (mcrypt.js) However, I have created an application nodejs external titanium and everything works fine. why can not I use library node in titanium? Thanks.

  • You need to copy and paste mcrypt.js file in resource directory in case of classic titanium and lib directory in case of alloy, then reference the file.

    You mobile project is all about the files which are inside resource directory. It can't be like web project where you can access any library files which are there on your/server machine. In care of mobile app you need to put in project because by using that code the bundle (.ipa/.apk) is generated.

    Hope this clarifies your doubt.
