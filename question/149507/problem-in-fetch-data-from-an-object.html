Problem in Fetch Data From an Object

Hello,

I have one Object like : 

var a = {
    "application/vnd.openxmlformats-officedocument.wordprocessingml.document" : "https://docs.google.com/feeds/download/documents",
    "application/vnd.oasis.opendocument.text" : "https://docs.google.com/feeds/download",
    "text/html" : "https://docs.google.com/feeds/download/documents",
    "application/rtf" : "https://docs.google.com/feeds/download/document",
    "text/plain" : "https://docs.google.com/feeds/download/documents/export",
    "application/pdf" : "https://docs.google.com/feeds/download/documents/"
}

and i want to access the object value.
This object contain Key, pair value and all the key name contain "/" and suppose i want to access one of the value of this object "a".

I have tried a lot but no success..
Can you please help me.

Regards Jigar

    Hi Jigar,

    You can try this way:

    Ti.API.info('exportLinks is : ' + a['application/pdf']);

    You can get the value for "application/pdf" using above mention line.
