  • There is nothing in the documentation saying that the scroll event will be called only once. Actually, it says the opposite here: "Fired repeatedly as the view is being scrolled". So seeing multiple alerts looks god to me.

    As suggested in the doc, if you want to have only one scroll event try listening for event "scrollend"
