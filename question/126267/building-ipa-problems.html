Building .ipa problems

I'm new to all this and getting a demo done in a week got dumped on me Monday. I have the thing running - amazingly - but now I can't find out how to actually get built for the device.

I have a TestFlight account ready to roll but I see mentioned of "build for iOS" here I don't see any way to do that from within Titanium - is there something I need to install in Titanium to let it build an IPA because I can't find it anywhere.

Assuming I need to use Xcode to build the IPA, it keeps erroring out about "The identity 'iPhone Developer' doesn't match any valid certificate/private key pair in the default keychain". Does this mean I have no way at all to even build the thing for deployment an app even for testing until Apple approves our paperwork?

Any help is much appreciated as this is way urgent.
