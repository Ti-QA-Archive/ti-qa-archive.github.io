    • @Ashish, you can play with it today! Just run titanium sdk install --branch 3_0_X, select the SDK that was installed in your tiapp.xml, clean your project, build it, and then you can open the Xcode file. This is a CI build, not thoroughly tested, and not production ready, so please only use in a development environment until the next stable SDK based on the 3.0.x branch ships… in a short while. :)
