Titanium Developer 1.2.2 Install Error with 1.5.1 SDK install

I've been doing development with OSX Titanium Developer 1.2.2, Titanium SDK 1.4.2 and iOS SDK 4.0 and after the Titanium SDK 1.5.1 installed, I could no longer Run on Device. I now get a cryptic 'Install Error' message. I didn't have this issue when 1.5.0 installed. I've reboot and tried rebuilding with different SDK's and the 'Install Error' continues with each. Deleting the Resources/build/iPhone/build folder didn't help.

Any thoughts?

  • Accepted Answer

    Load up the generated project into XCode and try to build/install from there. Make sure you have the XCode console window open. You should be able to find the reason for the issue either in the console window or by examining the build output by clicking on whatever errors appear in the lower right of XCode.
