jQuery Slider Navigation Equivalent

Is there an equivalent navigation UI control to the jQuery slider (I am actually not sure of the true control name, but it is the navigation control as used in www.hulu.com where a user can left/right scroll elements or click the underlying dot to navigate a mini app/image/video then select the image to launch)?

    Hi

    It is called scrollableView.

    There is some simple code to get you started as well, very simple to create, basically allows you to create "pages" of views, each page view, can be an imageView or a main with with other child views.

    Works in iOS and Android, the dots are iOS specific though.

  • The scrollableView gets you there but as Malcolm mentioned the dots are only on iOS.

    I created a free custom wrapper for scrollableView called ViewScrollr which will give you the same UI on both platforms. You also get to customize the page controls and set it to auto scroll.

    It is packaged in a nice CommonJS module and easy to use. Hope it helps.
