    Is this an Alloy or non-Alloy project? Alloy source graphics go in app/assets (or app/assets/iphone).

    What is your build target – retina/tall, retina, or non-retina? If you're building to the retina/tall (iPhone 5) form factor, you'll also need to replace the Default-568h@2x.png graphic.

    And I assume you're building for the phone, not tablet form factor. If you're building for iPad, you'll need to include additional graphics. See the docs for a complete list. As of this writing, that page hasn't been updated for iOS7 so some of the sizes are wrong. The best way to generate your graphics is to use Fokke Zandbergen's TiCons tool.
