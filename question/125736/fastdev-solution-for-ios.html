FastDev solution for iOS

Hi fellow devs,

I thought you might be interested in a small project I've just completed.

Since testing Ti Mobile applications on device can be quite consuming, because the app needs to be rebuilt, signed & redeployed after each significant modification to the code, I've created a bunch of scripts that allow to patch the 1.8.0 SDK, in order to allow fast deployment of your project Resource files (JS, images, whatever), through a simple http server (developed in node.js) running in your project's Resources dir.

Just check this post on my blog for details and a demo video: http://titaniumninja.com/post/10360108035/titanium-hack-fastdev-for-ios

The project is on github here: https://github.com/omorandi/TiiOSFastDev
