Titanium.Codec.extractZip problem!

Hello all!

I'm trying to unzip a file downloaded from an URL (a zipped file). The downloading part works perfectly, but when I try to extract the file I get the following error:

[Titanium.Codec] [Error] exception decompressing: Illegal state

Here is the code I'm using:

var zip = Titanium.Filesystem.getFile((
Titanium.Filesystem.getDesktopDirectory().toString()+
Titanium.Filesystem.getSeparator()),"bigfile.zip");

var dest = Titanium.Filesystem.getFile((
Titanium.Filesystem.getDesktopDirectory().toString()+
Titanium.Filesystem.getSeparator()));

Titanium.Codec.extractZip(zip, dest, function(destFile) {
   alert("Extracted zip file to: " + destFile);
});

Can anyone help me please? Thanks in advance!

  • Solved it! :) For those who are having problems with this, you must be really careful with the file you want to unzip. I only got this working when I took the file, copied it into a folder and then zipped the folder. When I had the file zipped directly, the code didn't work.

    Here is the code I used:

    var zip = Titanium.Filesystem.getFile(
Titanium.Filesystem.getDesktopDirectory(), "test.zip");

var dest = Titanium.Filesystem.getFile(
Titanium.Filesystem.getDesktopDirectory());

Titanium.Codec.extractZip(zip, dest, function(destFile) {
   alert("Extracted zip file to: " + destFile);
});

    Hope this helps those who are headbanging on this!
