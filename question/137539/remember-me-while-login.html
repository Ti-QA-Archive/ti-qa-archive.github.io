  • Hi Lokesh,

    Try using property or Global variable doing this and create image view to check and uncheck and call when you load that view.

    if(Ti.App.remember == true) {
        imgCheck.image = 'check.png';
        Ti.App.remember == true;
    } else {
        imgCheck.image = 'uncheck.png';
        Ti.App.remember == false;
    }

    On click event of imageview if check then uncheck the Ti.App.remember and uncheck then check them.
