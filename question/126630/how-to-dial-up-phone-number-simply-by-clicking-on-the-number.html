How to dial up phone number simply by clicking on the number?

I am creating a mobile directory apps. To make it more dynamic, I wanted to put phone number where the users can simply dial up by clicking on the number. How is this possible?

  • Attach a click event listener to the object with the phone number. In it use a variation of:

    Ti.Platform.openURL("tel:" + e.text);

    The above assumes e is your event object and it has a text property with the phone number to be dialed.

  • Hi,

    try creating the label first:

    var myPhoneLabel = Titanium.UI.createLabel({
  title:'Click me',
  width:200,
  height:50
});

    then adding the event listener regarding the click action:

    myPhoneLabel.addEventListener('click', function(){
  Ti.API.warn('you clicked me');
  Ti.Platform.openURL('tel:1234567');
});

    In the emulator you should see the warning, to see the real results you need to test on a phone.
