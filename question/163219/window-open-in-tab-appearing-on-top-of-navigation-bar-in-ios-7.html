Window open in tab appearing on top of navigation bar in iOS 7

  • Application Type: mobile
  • Titanium SDK: 3.2.1.GA
  • Platform & version: iOS 7.0.3
  • Device: iOS Simulator
  • Host Operating System: OSX 10.9.2
  • Titanium Studio: 3.2.1.201402041146

We have a master/detail window setup in some tabs of our application. Here's a screenshot of the master-view window.

When the detail-view window opens it appears on top of the navigation bar of the master-view window and doesn't show it's own navigation bar. Here's a screenshot of the detail-view window.

Here's the code that opens the detail window:

var newWin = Alloy.createController('recipe', {
    id: e.row.data.id,
    onLoad: function() {
        $.recipes.deselectRow(e.index);
    }
});

Alloy.Globals.tabGroup.activeTab.open(newWin.getView());

What am I doing wrong?

  • I'm not an expert, but I had a similar issue and it was because you have a tabGroup, no a NavigationController, so the navigationBar is not persistent across the windows, I've "fixed" it doing a new tabgroup in the detail window, as follow.

    <TabGroup id="tabGroupDetail" barColor="#e33124">
    <Tab>
        <Window id="detailWin" class="container" title="Detalle" tabBarHidden="true">
            <ScrollView id="detailMainView">
            </ScrollView>
        </Window>
    </Tab>
</TabGroup>
