{ leftImage:'fr.png', title:"France" }

];

var TheTable = Titanium.UI.createTableView({});

{ flag:'es.png', country:"Spain", trend:'up.png', percent:'28%'  },

{ flag:'us.png', country:"United States", trend:'up.png', percent:'8%' },
{ flag:'fr.png', country:"France", trend:'down.png', percent:'-40%'}
];

  • To modify the image associated with the hasChild property, you will need to modify the Titanium SDK itself. It's open source, and you're certainly welcome to make a custom build. See the Platform Development guide for the info you'll need to get started.

    If this is not what you wanted, you will need to not use hasChild. Instead, you can add/remove a graphic using the rightImage property of each table row. See the TableView documentation for more info on that.
