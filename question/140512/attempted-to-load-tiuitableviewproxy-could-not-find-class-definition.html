"Attempted to load TiUITableViewProxy: Could not find class definition."

I'm using the latest version of the Titanium SDK and want to populate a simple TableView with some rows.

I get the following error "Attempted to load TiUITableViewProxy: Could not find class definition."

This is my code:

var tblTips = Ti.UI.createTableView({backgroundColor: "#fff"});
var row = Ti.UI.createTableViewRow({height: 40, title:"test"});
tblTips.add(row);

Any idea on what can be wrong here?

  • I believe you are adding the row as a child to the table, instead of adding it to the rowdata of the table.

    Try something like this:

    var tblTips = Ti.UI.createTableView({backgroundColor: "#fff"});
var row = Ti.UI.createTableViewRow({height: 40, title:"test"});

// Your data for the tableview
var data = [];
data.push(row);

lblTips.setData(row);
