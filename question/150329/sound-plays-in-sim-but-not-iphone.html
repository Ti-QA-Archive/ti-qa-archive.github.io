Sound Plays in Sim but not iPhone

I have been trying to get a sound to play all night….i need help. It works perfect in simulator…but doesn't work on iPhone..I tried an alert in the events and that works so its not the events.


var player = Ti.Media.createSound({
        looping: true,
        preload: true,
        url:"/images/buzzer.aiff"});

    Titanium.Media.audioSessionMode = Ti.Media.AUDIO_SESSION_MODE_PLAYBACK;

    //Add behavior for UI
    imagev.addEventListener('touchstart', function(e) {
        player.play();
    });
    imagev.addEventListener('touchend', function(e) {
        player.stop();
    });

  • Here my code in my last project:

    self.sound = Ti.Media.createSound({
    url : '/assets/noise-and-heartbeat.mp3',
    allowBackground : true,
    looping : true,
    audioSessionMode : Ti.Media.AUDIO_SESSION_MODE_PLAYBACK
});
self.sound.play();

    and in Info.plist:

    <key>UIBackgroundModes</key>
    <array>
            <string>audio</string>
    </array>
