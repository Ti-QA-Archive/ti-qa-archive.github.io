    • I got it going! I think I know what the problem was.

      The problem was I had added a lot of images in the Resources/<preload> folder, so I dont have to go to the server to get them -atleast while testing (because I am having problems with Ti.Network.HttpClient - but thats another problem!!). I forgot about this till just now. I dont know if Titanium (or Android) has a limit on the size of the app. (I know iOS has).

      But I thought it would be ok on the emulator, because it was for testing. Apparently not. So, i guess thats why it didnt deploy the new build file.

      So, now, I can see the changes, though I still cant load one part of my app.

      But this is a start! i can do anything else, once I know the build is being deployed! Phew! I wasted so much time on this!
