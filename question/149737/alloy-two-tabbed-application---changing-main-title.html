Alloy Two-tabbed application - changing main title

Hi,

I'm just starting with Titanium - so, please, bear with me…

I started off with Alloy Two-tabbed Application template as a starting point. For the starters the target is Android tablet (1024x600 if that matters). I managed to make a menu which is accessible on HW menu button using this code:

$.tabGroup.addEventListener('open', function(e) {
    var activity = $.tabGroup.activity;
    activity.onCreateOptionsMenu = function(e) {
        var menu = e.menu;
        var menuItem = menu.add({
            title : L('login')
        });
        menuItem.addEventListener("click", function(e) {
            Alloy.createController("login");
        });
    };
});

For now the main menu just has one option: login.

The thing that I would like to do now is: to update the "main title bar" upon successful login on webservice.
The code which is connecting to the webservice is working fine and it triggers custom event I added but, for the life of me, I can't find how to update this "main title bar" which by default shows the application name.

Is this even possible?

Please advise.
