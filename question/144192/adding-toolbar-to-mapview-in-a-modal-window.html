adding toolbar to mapview in a modal window

i have a modal window that im adding a mapview to - i'm now trying to add a toolbar to that modal so i can show the sat type, zoom etc

this is the normal code i'm using elsewhere in a normal window containing a mapview

Titanium.UI.currentWindow.setToolbar([std,sat,hyb,zoomin,zoomout]);

this is what i assumed would work but doesnt: (modalWin is the name of the modal thats being opened)

Titanium.UI.modalWin.setToolbar([std,sat,hyb,zoomin,zoomout]);

i get an "result of expression 'titanium.ui.modalWin' [undefined] is not an object"

  • Titanium.UI.currentWindow is a special property in the Ti namespace, as per the docs

    The active window associated with the executing JavaScript context

    where as modalWin is not, which is why you get the 'undefined' value.

    You need to find some other way of referencing your modalWindow.
