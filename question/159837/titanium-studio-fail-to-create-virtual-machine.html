Titanium Studio: Fail to create Virtual Machine

Greetings:
I just installed Titanium Studio on my PC and it keeps giving me an error that reads:
"Failed to create JAVA Virtual Machine"
I'm new to appcelerator and I would appreciate some help. Do I need to uninstall Titanium and re-install it in a different way?
Thanks in advance for your help.

  • This error message is usually due to the fast that your Java Virtual Machine is not installed or improperly configured.

    First of all, make sure that you install the 1.6 version (not the 1.7). Also make sure that you choose the 32-bit version; even though you are running a 64-bit version of Windows.

    Once it is installed, make sure that it is properly configured by typing the following command at the command prompt:

    java --version

    You should see the version of your virtual machine. Once this is properly done, you shouldn't have any problem launching Titanium Studio,

    Hope this helps
