  • Yes:
    Try to use a normal view on top on window. The text you can animate:

    var topView = Ti.UI.createView({
    height : 40,
    top : 0
});
var topText = Ti.UI.createLabel({
    right : -320,
    text : 'I\m a running text with very long lenght ………………………'
});
topText.animate(Ti.UI.createAnimation({
    duration : 2000,
    right : 320
}));

    ~~~
