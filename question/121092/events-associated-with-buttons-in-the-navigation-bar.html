    dont really understand the purpose of the code you pasted in the comment, but the answer to the original question is

    buttonBar.addEventListener('click', function(event) {
    switch (event.index) {
        case 0:
            //up clicked
            break;
        case 1:
            // down clicked
            break;
    }
});
