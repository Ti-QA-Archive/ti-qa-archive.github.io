Unable to display images from custom ios module

Hi, I am building a custom ios module that wraps a library that display some images. It was alright if it runs in native xcode, but once it is wrapped in titanium module, the images can't be displayed. The library is accessing a list of images in a directory. If it runs directly in native xcode, the images are bundled into the
app, but not when it is compiled for titanium ios module. Is there any way/method or step to include the images. I have tried adding build phase to copy bundled resources. Appreciate any help, thanks.
