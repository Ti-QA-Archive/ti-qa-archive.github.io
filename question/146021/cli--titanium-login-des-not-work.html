CLI $ titanium login des not work

Hi,

I tried installing the CLI via Titanium Studio first, then with NPM. I sudoed my /usr/local sudo chmod 777 &#x2F;usr&#x2F;local changed the NPM install directory export NPM_CONFIG_PREFIX=$HOME and added the bins to my path.

Anyway I get this message:

$ titanium login
Appcelerator Titanium
Copyright (c) 2010-2012 by Appcelerator, Inc.

Unknown command: login

I´m using zsh if this might be an issue.

    You probably have an alias in your ~/.bash_profile to the old titanium.py. Remove the alias and restart your terminal session, then run titanium --version. It should output 3.0.21.
