  • I have found a temporarily solution which is not the best, but either way it works okay.

    Using this code deleting contents and adding custom features I have been able to create a tableview with mp3-enclosures, so that "it looks" like a playlist.

    For the controls I have created a label on which I have applied buttons to animate the label in and out of the screen - quite complicated when you're new to JavaScript and Titanium :)

    To make the controls (the label on which they a applied) appear on top of all windows I have used the syntax:

    tabGroup.add(your_item)

    • Hi Adebesi

      Unfortunately I don't have that code anymore, but maybe I can give you some tips.

      1. You need to create a XML-file with your selected attributes, which shall then be located on your server (or any other) - you then need to link this to the url in you script.

      2. Use the Titanium.Media.AudioPlayer as the back-end of your audio player and put a listener on your RSS-tableview - then initialize your audio player with your_item.play();

      3. Now you have your audio playing, but to control the audio you need to add an eventlistener for each control-instance of your player. These could be your_item.pause(); or/and your_item.stop(); - whichever you prefer

      4. Attach these listeners to your button-objects - which could be a play and stop.

      5. Next you need to add your buttons to a view, which should be added to the tabGroup or your top window. Set the top-parameter (or bottom) as top : - your_item.height - that way it is possible to animate your view in and out of the screen.

      I hope that was helpful, but please write if you have further questions.
