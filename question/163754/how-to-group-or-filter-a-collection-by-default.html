How to group or filter a Collection by default:

i need to filter a collection, I have this code:

    var model = Alloy.createModel('measurements', { 
      time:    Alloy.Globals.sessiontime, 
      usv: valuerecived[3],  
      mrh: valuerecived[4],
      cpm: valuerecived[2],
      session: Alloy.Globals.session,
      latitude: Alloy.Globals.latitude,
      longitude: Alloy.Globals.longitude,
      position: Alloy.Globals.position
    });

    Alloy.Collections.measurements.add(model);  
     model.save();

this code a now record in my tables correctly, but at the same time refresh the tablerowview that use this collection.
I need to apply the "group by" attribute on the data that return model.save(), i don't need all records..
there is a way to do that?

thank you

  • I'm assuming you've bound the measurements collection to your tableview. Thus, whenever you update the collection, the change event fires, reloading the collection and redrawing your table. You don't have to add the model to the collection to save it. You could leave out the Alloy.Collections.measurements.add() statement and your model would still be saved with model.save().
