    Go to your Android SDK directory and look in the “tools” directory. Run the “ddms.bat” file. it will launch the Dalvik Debug Monitor.Then Select the emulator that is currently running.then go to “Device” menu and select “Screen capture…” or simply pressing CTRL+S.

    • This is how you would do the same thing using the CLI:

      adb shell screencap -p /sdcard/screen.png
adb pull /sdcard/screen.png
adb shell rm /sdcard/screen.png

      • OSX

        • Then space bar
        • Select window
        • Image saved to desktop
