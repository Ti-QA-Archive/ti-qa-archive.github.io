Android createIntentChooser size on screen android 4.4.x

Hello,

I am trying to use the Intent for social sharing on Android. Everything looks good on android version < 4.4 but when trying it on 4.4.X the intent screen is too big.

Below is the code I am using.

var intShare = Ti.Android.createIntent({
        action : Ti.Android.ACTION_SEND,
        type : "text/plain",
        width: "80%"
    });

    intShare.putExtra(Ti.Android.EXTRA_SUBJECT, "My Subject");
    intShare.putExtra(Ti.Android.EXTRA_TITLE, "myTitle");
    intShare.putExtra(Ti.Android.EXTRA_TEXT,"My content");
    var chooser = Ti.Android.createIntentChooser(intShare, "Share and Spread");
    var activity = Ti.Android.currentActivity.startActivity(chooser);

Any idea on what is wrong ?

  • Hi Ankesh,

    It's bug and reported in JIRA.

    Try below code that help you to short out that problem for now.

    var activity = Ti.Android.currentActivity;
        var intent = Ti.Android.createIntent({
                    action: Ti.Android.ACTION_SEND,
                    type: 'text/plain'
                });

        intent.putExtra(Ti.Android.EXTRA_TEXT, "My Text");
        intent.putExtra(Ti.Android.EXTRA_SUBJECT, "My Subject");
        activity.startActivity(intent);
