Override L in 3.5.0.GA

Hello everyone,

I am using an override of the L() method to be able to dynamically change language in my app. It is not possible anymore in 3.5.0.GA as the override L() method is not called anymore. Why?
Is there another way to dynamically change language strings also using Alloy's title="L(…)" functionality?

Thanks a lot.

  • Hi Tobias,

    I found this issue 3 weeks ago. In my project I have embedded the link to overwrite code in every module. It is dirty, but it works.

    // in head of every module:
var L = function(key) {
    return require('L')(key);
};
