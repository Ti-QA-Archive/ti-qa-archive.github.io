Change Android emulator?

Why is it so difficult to use Android tools? :S

My app uses WebView and uses events in it (WebView.evalJS() and Ti.App.fireEvent())

On iOS it works, on Android it crashes and this stack says that it is 2.3 emulator bug.

My tiapp.xml is configured to use Google API 2.3.3.
I changed to use Google API 2.2 but the same error is showing and app crashes.
I am not sure if this is the place to change emulator or somewhere in ddms?

    The tiapp.xml is the api that it is compiled against, to change the emulator click the "play" button in TI Studio, then click Run Configurations. From their choose the android emulator and you select the Android version (and screen form factor).
