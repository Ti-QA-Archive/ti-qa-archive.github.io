title above TabGroup for Android

Dear all,

Does any one know how to display title text(in the nav bar) above TabGroup in Android? I know I can set title to display at each tab, but how about the one on top of the tab group? It only has an application icon at left and rather strange there if it's android. I tried to set navBarHidden : true so that nav bar won't show, but the icon inside tab also goes hidden…

Please kindly advise me here…thanks!

    If you mean something like this:

    You need to first target an api that supports the updated android layouts by editing your tiapp.xml:
    (in this case targetted api 15)

    <android><manifest>
<tool-api-level>15</tool-api-level>
<uses-sdk android:minSdkVersion="10" android:targetSdkVersion="15"/>
</manifest></android>

    This example comes from alloy, but same applies to classic, just replace $.tabs (in this case the tabgroup) with your tabgroup object name:

    $.tabs.addEventListener('open', function(){
  if (OS_ANDROID){
    if($.tabs.activity){
      actionBar = $.tabs.activity.actionBar;
        if (actionBar){
          actionBar.title = "Conversion Center";
        }
    }
  }
});

  • No, you cant do this via a tabGroup, the title is supposed to come from the active window. As for the navBarHidden property As the docs state for Android:

    On Android, the nav bar can only be shown or hidden for the whole tab group. Setting this property to true disables the Action Bar since it is part of the navigation title bar.
