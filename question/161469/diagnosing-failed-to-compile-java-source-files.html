  • I got the same identical error, I solved launching the command you see in the line:

    [INFO] :   Building Java source files: C:\Program Files\Java\jdk1.7.0_45\bin\javac.exe "-J-Xmx256M" ...

    (adding quotes on "C:\Program Files\Java\jdk1.7.0_45\bin\javac.exe")
    directly on command prompt, so I can see the error, in my case:

    ...\app\R.java:17: error: <identifier> expected
public static final int 128x128_24=0x7f020000;

    removing the "x" in my icon file name let my project works again
