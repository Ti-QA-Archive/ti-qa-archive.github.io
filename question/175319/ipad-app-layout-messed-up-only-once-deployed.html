iPad app layout messed up only once deployed

Hi,
I've deployed an app with an entreprise profile onto a retina ipad.

When I run it on the simulator it looks perfect.
When I run it on the device it looks perfect.
When I use Xcode to generate the manifest file, then archive and export it for development everything works except the app looks zoomed in 200% and elements are all over the place.

Has it happened to anyone? Why would it be doing that? I would lean towards the settings in Xcode, since I package a working app in titanium.

    Sounds like the build target is iPhone. Try changing it to Build 'Universal'

    • Thank you. I actually had 3 Targets in Xcode so I removed 2, selected iPad rather than universal in the build, and all went well :)
