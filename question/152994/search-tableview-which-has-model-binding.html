Search TableView which has model binding

Hi,
I have following view. It has collection association. Data in table view is coming from collection. My table view row is customized. Now i want to apply search to this table. so i added search bar.

Now I want to set filterAttribute to {number} which is one of the label in table view row. Please tell me how to set filterAttribute in this case ?
Following is Label which is child view of TableviewRow & its text is {number} which is set with the collection data.:

<Label id="reportNo"   text="{number}"/>

I want to apply search on the basis of this {number} data. Please tell me how to set it with filterAttribute of TableView?
Complete xml code:


<Alloy>
    <Collection src="reports"/>
    <Window class="container">
        <View class="container" id="rep">

        <TableView id="listReports" dataCollection="reports" filterAttribute="">
                <SearchBar></SearchBar>
                <TableViewSection id="listReportsSection" >

                    <TableViewRow id="report" >
                        <View id="firstRow">
                            <Label id="reportNo" text="{purpose}"/>
                            <Label id="reportCost" text="{custom}"/>
                        </View>
                        <View id="secondRow">
                            <Label id="approverName" text="{number}"/>
                            <Label id="reportDate" text="{approver1}"/>
                            <Label id="reportData" text="{reportDate}"/>
                        </View>
                    </TableViewRow>

                </TableViewSection>
        </TableView>
        </View>
    </Window>
</Alloy>

    Hi

    You do not set the filterAttribute to the label but to a property of the tableViewRow, so add a custom property to each row using the same name and that becomes the filterAttribute name you assign.

    The filterAttribute documentation explains it here;

    This property maps to your data object or a property on the row object. For example, if you set filterAttribute to 'title', the table view shows only those rows that contain the current search text in the title property.
