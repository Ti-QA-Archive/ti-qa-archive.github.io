  • OK, with Minh's guidance I solved my problem. It's pretty simple once you find the solution. For others:

    • Select Titanium Studio > File > New > Titanium Mobile Module Project
    • Read all the READMEs that are created. This is the crucial step as the contain key information that I was unable to find in the documentation.
    • Package the module by clicking on the package icon and selecting Titanium iOS Package Manager.
