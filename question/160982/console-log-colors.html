Hi, if the google chrome is updated, when I open a html page, if I have a console log like this:

console.log("message to show", "color:red");

It works, the log row change the text color to red, but this dont work on titanium, there is any workaround ?

This is the console result:

message to show, color:red

    I don't think we have support like the way you specified. "The message to log can either be a single argument, or any number of arguments, which will be converted to strings and then concatenated together with a space character.".

    But, you have different colors based on log level like console.error as red, console.debug as yellowish
