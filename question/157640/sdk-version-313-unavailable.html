SDK-Version 3.1.3 Unavailable

Hello,

I recently upgraded Titanium Studio to build 3.1.3.201309132423. I am under the impression that this includes the 3.1.3 SDK, however I am unable to select that version of the Titanium SDK when creating a new Mobile Project. Likewise, when manually editing the tiapp.xml of an existing project for sdk-version 3.1.3GA I get an error when I attempt to build.

[ERROR] &#39;build&#39; is an unrecognized command.

The highest SDK version unavailable to me is 3.1.0GA. I have attempted to manually update many times, with no indication there are updates I am missing.

Thank you in advance for any help.

    That;s because Titanium Studio and the SDK are two separate things. (even though the first installation installs both at the same time.

    If you want to upgrade your SDK using titanium Studio, select the Help | Check for Titanium Updates menu. This will download the latest version of the SDK (if it not already there)

    Hope this helps.
