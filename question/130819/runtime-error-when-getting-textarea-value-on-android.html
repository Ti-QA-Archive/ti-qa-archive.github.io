Runtime Error when getting textArea Value on Android

I have an app that has been successfully compiled for iOS and now I am trying to make it work for Android. I am getting this error in regards to capturing the value of a textArea:

Runtime Error
MESSAGE: Uncaught Error: Requires property name as first argument.
SOURCE: but_close.addEventListener("click", function(){noteholder=tf1.getValue();

My actual source code for this line is:

var noteholder='';
but_close.addEventListener('click',function(){
noteholder = tf1.getValue();
Titanium.App.Properties.setString("notetakertext",noteholder);
});

Any ideas what is triggering this error?
