Collections & Data Bindings

Hi All,

Newbie Question Number 2.

I have a collection:

exports.definition = {
    config: {
        columns: {
            "player_id": "INTEGER PRIMARY KEY AUTOINCREMENT",
            "first_name": "TEXT",
            "surname": "TEXT",
            "handicap": "INTEGER"
        },
        adapter: {
            type: "sql",
            collection_name: "players",
            idAttribute: 'player_id'
        },

    },        
    extendModel: function(Model) {        
        _.extend(Model.prototype, {
            // extended functions and properties go here
        });

        return Model;
    },
    extendCollection: function(Collection) {        
        _.extend(Collection.prototype, {
            // extended functions and properties go here
        });

        return Collection;
    }
}

I have records added to this collection as I looked up the sql_lite database on my Mac HDD and it corresponded with the two entries I had added.

So here is my players.xml code:

<Alloy>
    <Collection src="players" />
    <Window class="container" title="Select Players">
        <RightNavButton>
            <Button id="new_player" onClick="openNewPlayerWindow">
        </RightNavButton>
        <TableView id="myTable" dataCollection="players" filterAttribute="title">
            <SearchBar />
            <TableViewRow title="{first_name}" playerId="{player_id}" hasDetail="true" />
        </TableView>
    </Window>
</Alloy>

My Controller Looks like this (players.js) - Nothing too kinky

function openNewPlayerWindow() {
     var win=Alloy.createController('editPlayer').getView();
     Alloy.Globals.tabGroup.activeTab.open(win);
}

My issue is the table view show's no results. Im pulling my hair out with lack of obvious understanding. I'm picking it up more and more and sure this is one of those things I will be shown once and have a eureka moment.

  • Ive worked it out:

    Had not added a reference to the collection in the controller:

    // -- Need To Make a Collection
var players = Alloy.Collections.players;
// -- Call Fetch
players.fetch();
    — commented May 19th 2013 by Steve Clark
