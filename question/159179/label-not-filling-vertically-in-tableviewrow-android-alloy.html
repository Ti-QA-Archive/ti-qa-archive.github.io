Label not filling vertically in TableViewRow (Android Alloy)

Hi,

I am having difficulty getting a label to expand vertically so that it matches the same height of horizontally adjacent labels in a TableViewRow.

For example, there are 3 labels placed horizontally in a TableViewRow. One of the labels wraps to 2 lines high, and the other 2 labels remain at one line high. I need the one line high labels to fill vertically so that they are the same height as the 2 lines high label. Ultimately I want to add borders to the table cells, so all of the cells in a row need to be the same height.

A snapshot illustrating the issue can be found here:

Here is the sample code that created the image above:

<Alloy>

        <TableView >

                    <Label class="leftLabelBackground" text="Schedule 10"/>
                    <Label class="centerLabelBackground" text="0900"/>
                    <Label class="rightLabelBackground" text="1000"/>

    The reason being is the height not set. Try to set height to Ti.UI.FILL which will match the height of its parent - tableViewRow
