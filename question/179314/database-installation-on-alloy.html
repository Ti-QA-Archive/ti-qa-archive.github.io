Database installation on alloy

I am trying to install a database that allready have some tables on it. I try to do that on alloy.js.

db = Ti.Database.install('/database.sqlite', 'database');

After i try to execute a query

db.execute('INSERT INTO CITIES (city_id, city) VALUES (?, ?)', city_id, city);

i have the database in assets folder. When i run the app i get the following error when i try to run a query

[ERROR] : SQLiteLog: (1) no such table: CITIES

Any ideas what i do wrong ?

  • I found the problem myself. I shouldn`t use the same name for the database installation. I should use

    db = Ti.Database.install('/database.sqlite', 'database_blabla');
