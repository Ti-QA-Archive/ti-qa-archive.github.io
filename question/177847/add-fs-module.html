  • When Titanium's require function can not find the module, then it is not installed correctly.

    Modules must be added to to tiapp.xml before you can require it in your code. You can add Titanium modules or other modules (native or CommonJs) that are installed globally in your SDK or locally in your project. Modules can be obtained (e.g.) from:

    • the Marketplace
    • gittio

    Please read this guide to learn how to install and use modules.
