Ti.UI.SIZE not working properly with width

Hey guys, the title is pretty self-explanatory, I have a web view with dynamic content, so I need to create a webview with 'auto' width and height, since that's deprecated, I decided to use Ti.UI.SIZE, for height, it works beautiful, however for width, it simply doesn't, it cuts of after a few bytes, below is a very simple sample and a screenshot.

I'm running the latest version of Titanium.

// this sets the background color of the master UIView (when there are no windows/tab groups on it)
Titanium.UI.setBackgroundColor('blue');

var win = Ti.UI.createWindow({});

var webview = Ti.UI.createWebView({
    width:Ti.UI.SIZE,
    height:Ti.UI.SIZE,
    backgroundColor:'white',
    html:"<center>Test of Ti.UI.FILL that's how we do things here and now!'</center>",
});

win.add(webview);
win.open();

    • While you may have a good reason for wanting to do it this way, I think you might find more success with Titanium by trying to develop within the API to handle these things via native controls, versus try to do it with html and a webview.     <div style="text-align: center;"><span style="display: inline-block; padding: 0 5px; background: #fff; ">Test of Ti.UI.FILL that's how we do things here and now!'</span></div>

    Inline-block works like what you are wanting in a normal web sense.
