  • This is really easy with npm. Only add sudo on Mac or Linux. You can confirm version at any time by running alloy --version. Any one of these installs will overwrite the previous one, making it trivial to try out different versions.

    latest stable alloy

    sudo npm install -g alloy

    specific version

    sudo npm install -g alloy@1.1.3

    cutting edge from github

    sudo npm install -g git://github.com/appcelerator/alloy.git
