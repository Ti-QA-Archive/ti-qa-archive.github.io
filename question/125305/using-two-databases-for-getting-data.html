Using two databases for getting data

In my App, I need to use two tables but they are located in two different databases.
Is that possible to get data from table A in DB A and then close it, open DB B and store retrieved data in table B in DB B? OR any other way to do that?

What about using attach statement?

Any help is highly appreciated.

    You can open both at the same time, though that will not allow a JOIN between the tables:

    var db1 = Ti.Database.open('dbA');
var db2 = Ti.Database.open('dbB');
var rs1 = db1.execute('SELECT * FROM tableA');
var rs2 = db2.execute('SELECT * FROM tableB');

    If you want to join the tables you would need to use ATTACH. The last time I tried that I did not get it to work. But see this thread, and this too because others got it to work.
