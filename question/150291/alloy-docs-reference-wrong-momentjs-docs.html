  • Accepted Answer

    Please log moment.js and Backbone.js updates as separate feature requests in the Titanium Community Jira project. Be sure to add the "alloy" label.

    FYI, moment.js update will likely be simple, but I don't think Backbone.js will be updated soon. I attempted updating from 0.9.2 to 0.9.9 a couple weeks ago and it caused breaking changes in Alloy's adapters. If you just can't wait, as Aaron already stated, you can always copy in whatever version you want to use in yourself. You could just put the Backbone.js 1.0.0 file in your app&#x2F;assets&#x2F;alloy folder and it'll get copied in when you build, overwriting the default Alloy version.
