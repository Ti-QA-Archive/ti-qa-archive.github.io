    • I am going to assume you are using Ti SDK 3.2 GA - if you are not - start there and install that from the downloads page.

      If you are using Ti SDK 3.2 GA then check that you have the correct SDKs for Android installed - there are many - one for each version of Android that should be supported (Even if you are not directly supporting them). Keep in mind you are required to have Android 2.3.3 even if you are only targeting Android 4.x devices. This is because it is a current requirement.

      Steps to check;

      • From Titanium Studio
      • Locate the Dashboard Tab
      • If you cannot find it press the red "house" with the letter "a" that should appear on the toolbar
      • In Dashboard press the "Get Started" button under your name and icon
      • Scroll down to SDKs
      • If this is red, you have not configured at least the basics
      • Press the "Android SDK" entry
      • Press the "Update Android Button" that appeared to the right side
      • Press the "Settings" link on the right side of "Android"
      • This will show you all the SDKs that can be installed
      • Select them all and press the "Configure" button
      • This may take some time

      Once complete reboot even if not asked - a reboot is always a good idea.

      Try again.
