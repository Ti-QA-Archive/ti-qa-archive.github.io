    • Did you try to append a slash before the image name like this:

      <ImageView image="/default.png"></ImageView>

      This is some sort of a fix for Android platform where you are unable to see the images.

      If you want to use the default.png image from Resource folder, then just copy paste the file into the /assets/android/ & /assets/iphone/ folders & use the same code line as i mentioned above
