  • Accepted Answer

    The iOS implementation of javascript scoping when it comes to commonJS is a little bit 'looser' than what Android allows. Once you understand the scoping (Android is strict) you can make sure you 'require' the modules you need, then my experience is that Android works flawlessly using a commonJS approach.
