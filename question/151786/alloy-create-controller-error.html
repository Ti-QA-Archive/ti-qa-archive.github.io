Alloy Create Controller Error

I have been beating my head against a wall all day trying to figure this out. I like the idea of Alloy, but have been running into issue after issue - mostly because it's so different from the "old" way of doing things.

All I am trying to do is open an new window from the active tab, here is the relevant code.

index.js

Alloy.Globals.tabGroup = $.index;
$.index.open();

Relevant code from Dashboard.js

function CreateWarranty(){
    var controller = Alloy.createController("CreateWarranty");
    Alloy.Globals.tabGroup.activeTab.open(controller.getView());
}

CreateWarranty.js is blank, as I assumed not code was required to just open the window.
CreateWarranty.xml has just some basic view code.

I am getting the following error when the CreateWarranty method fires.

[ERROR] Script Error = Parse error at CreateWarranty.js (line 27).
[ERROR] Script Error = Module "alloy/controllers/CreateWarranty" failed to leave a valid exports object.
[WARN]  ing: Attempt to present <TiErrorController: 0xb2a2260> on <TiRootViewController: 0xb2715f0> while a presentation is in progress!

Everything I have read online says this should work. I have even tried adding some dummy exports methods to CreateWarranty.js, but get the same issue.

I have also tried this approach (http://www.slideshare.net/aaronksaunders/alloy-simpleapp), and am getting the same error.

Any ideas?

    The problem is the values you are using for IDs. IDs in Alloy, unlike classes, need to follow the restrictions for variable naming in Javascript, meaning that the dashes "-" you are using are causing the error. Remove the dashes from your ID names and everything will work as expected.
