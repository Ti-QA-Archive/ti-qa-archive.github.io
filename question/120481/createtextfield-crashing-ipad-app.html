createTextField crashing iPad app

I am experiencing a very strange issue.

I've been developing an iPad app for a few weeks now and have had no problems. Been using labels, views, buttons, imageViews, tableViews, etc. Today I went to use a textField for the first time and my app would attempt to boot up and then exit immediately in the simulator.

I tried all sorts of things to no avail. In the end I stripped my application right back - so much so that the only code being run is

var tabGroup = Titanium.UI.createTabGroup();
tabGroup.open();

var aa = Ti.UI.createTextField({
    value:'a'
});

But still it crashes!

As soon as I comment the createTextField() call it works fine. If I change it to createLabel() or anything else, it works fine. No amount of changing the parameters seems to make a difference.

I have created an iPhone app before and had no issues using textFields.

What is going on?!?

Any and all help appreciated!

  • When you compile your project, titanium removes many APIs do you not still use to make a small app.
    This cause problems like this. To correct, simple do a Full compile again.

    If you don't know, open on your project folder "Build/iphone/" and delete all files/folders in it and run again.
