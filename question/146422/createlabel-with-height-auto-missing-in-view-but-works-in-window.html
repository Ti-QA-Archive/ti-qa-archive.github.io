createLabel with height 'auto' missing in view, but works in window

I am trying to create a dynamic label inside a view. If I specify a height the label shows, if I specify 'auto' the label disappears;

    var view = Ti.UI.createView({
        backgroundColor: "#f7eddd",
        width: 185,
        height: 182,
        bottom: 90,
        left: 9
    });

    var label = Ti.UI.createLabel({
        text: "dynamic text here",
        height: 'auto',
        width: 175,
        top : 5,
        left: 5,
        font:{
            fontSize: 15,
            fontFamily: "TrashHand"
        }
    });

    view.add(label);

    win.add(view);

label.size.height returns 0. I assume it loads prior to retrieving the height of the view, I tried to add the label with a open event handler and a setTimeout but I get the same results. If I add the label to the window it shows. I am only concerned with iOS.

    Hi

    Change auto to Ti.UI.SIZE.

        var view = Ti.UI.createView({
        backgroundColor: "#f7eddd",
        width: 185,
        height: 182,
        bottom: 90,
        left: 9
    });

    var label = Ti.UI.createLabel({
        text: "dynamic text here",
        height: Ti.UI.SIZE,
        width: 175,
        top : 5,
        left: 5,
        font:{
            fontSize: 15,
            fontFamily: "TrashHand"
        }
    });

    view.add(label);

    win.add(view);
