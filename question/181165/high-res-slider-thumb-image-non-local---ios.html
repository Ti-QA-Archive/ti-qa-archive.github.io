High Res Slider Thumb Image (non local) - iOS

Is there a way of setting a slider thumb image to an appropriate resolution when using a Ti Blob?

Currently it renders at the given size, so if i provide an @2x blob (just doubling the dimentions) it renders double the size.
As expected, with my current code, the thumb image is very blurry on a retina device.

Below is a summary of my code, the thumb image needs to be generated from toImage as the image will be unique for each slider.

     // image con
    var imageCon = Ti.UI.createView({
        width: 66,
        height: 66,
        borderRadius: 33
    });

    // radial band
    var band = Ti.UI.createImageView({
        image: 'ring.png',
        height:66, width:66
    });
    imageCon.add(band);

    // product image
    var image = Ti.UI.createImageView({
        image: 'placeholder.png',
        height:54, width:54,
        borderRadius: 27
    });
    imageCon.add(image);

    var sliderThumb = imageCon.toImage();

    var slider = Titanium.UI.createSlider({
        thumbImage: sliderThumb
    });
