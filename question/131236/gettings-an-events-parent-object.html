Gettings an events' parent object

Hello all,

How would one fetch a variable from and events' parent object/class?

foo = function() {

    this.button = Titanium.UI.createButton();

    this.text = 'testing';

    this.button.addEventListener('click', function(e) {

        Ti.API.info(/* How would I get the value of 'this.text'?*/);

    });

}

Is there a function like 'parent' to get the parent object?

thanks

  • Fixed:

    this.button = Titanium.UI.createButton({
    mother: this,
});

Ti.API.info(e.source.mother.text);

  • Try e.source.text

    In general, check the API docs for the object you're adding the event listener to. At the end of the docs is an Events section. Click on an event and you'll get a listing of the properties available to the e (event object name you specify) within the listener function. Most events offer an e.source property, which points to the object that received the event.
