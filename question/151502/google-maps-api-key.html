  • Yes, you still need an API key. If you don't already have a Google Maps API v1 key, you'll need to use the new Titanium Maps module, which supports Google's new v2 API & key. Details on the new module and obtaining a v2 API key are described here:

    http://docs.appcelerator.com/titanium/latest/#!/guide/Google_Maps_v2_for_Android
