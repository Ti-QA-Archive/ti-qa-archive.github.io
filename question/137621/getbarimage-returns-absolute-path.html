getBarImage returns absolute path

hi,

i have defined backgroundColor, backgroundImage and also barImage in a window:

var win1 = Ti.UI.createWindow({
    backgroundColor:'#000',
    backgroundImage:'images/image.png',
    barImage: 'images/header.png' 
});

if i now call win1.getBackgroundImage, it returns relative path: images/image.png
if i call win1.getBarImage, it returns absolute path: file://localhost/var/mobile/Application……….

and header image is not showing on Ipad real device.
it is showing only in Ipad simulator.

any suggestions how to solve this ?

thanks

  • Don't use relative path, use absolute ones 

    barImage: '/images/header.png'

    where / folder is Resources.

    Also take care that the device is case sensitive, so if you have the filename Header.jpg instead of header.jpg it won't work.
