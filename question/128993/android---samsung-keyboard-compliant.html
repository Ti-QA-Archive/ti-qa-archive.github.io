Android - Samsung Keyboard (Compliant?)

Hello there,

I'm having a problem with Samsung tablets (both 7" and 10.1"). I need to display a numeric keyboard that allows decimals and I'm using the following code:

Titanium.UI.createTextField({
    color:'#336699',
    height:step,
    top:t,
    left:left2,
    width: textfieldWidth,
    name: name,
    passwordMask:false,
    keyboardType:Titanium.UI.KEYBOARD_NUMBERS_PUNCTUATION,
    returnKeyType:Titanium.UI.RETURNKEY_DEFAULT,
    borderStyle:Titanium.UI.INPUT_BORDERSTYLE_ROUNDED    
});

If I use "English (US) Keyboard (Android Keyboard)" this works as expected allowing the user to enter decimal values.
On the other hand, if I use the default keyboard that comes with this device "Samsung Keypad" it doesn't allow the user to input the decimal point (or any other character but 0-9).

I have instructed the users of the app to use the Android Keyboard while I look for a fix but I would like to either:

  • Confirm this keyboard is not 100% compliant.
  • See if I have the option in code to select the Android Keyboard so no user action is required while using the app.

Thanks in advance,
RR.
