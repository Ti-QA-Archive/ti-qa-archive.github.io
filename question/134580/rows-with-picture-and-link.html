    • To open link when click row:

      • Add url property to tableview.data:
        ~~~
        var RegData = [

      { leftImage:'es.png', title:"Spain", hasChild:true, url: 'http://google.com' }
      ];

      
* Then use [Titanium.Platform.openURL](http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.Platform.openURL-method.html) to open url:

      tableview.addEventListener('click', function(e){
      var url = e.row.url;
      alert(url);
      Titanium.Platform.openURL(url);
      });
      ~~~
