    When you call Ti.App.Properties.setObject();, you need to pass a two parameters, the first is the property name, the second is your property value or object. Here is the documentation: Titanium.App.Properties-method-setObject

    example:

    // Setting the object
Ti.App.Properties.setObject('town', { tN: townName });

// Retrieving the object
var town = Ti.App.Properties.getObject('town');
Ti.API.info(JSON.stringify(town));
