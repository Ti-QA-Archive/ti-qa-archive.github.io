Change element style on click/touch

I'm working on an iOS/Android application in Titanium. I want my buttons to react to touches by changing their background and font color, for instance. What is the best way to do it? Currently I'm doing that by passing each button to an alloy.js function that attaches event listeners to these buttons. But, I think there might be an easier way to do it.

The function I currently use:

function addButtonDefaultUIFunctionality(button)
{
    button.addEventListener('touchstart', function()
    {
        button.setBackgroundColor("#85FF8400");
        button.setBorderColor("#FF9C00");
        button.setBorderWidth("1");    
        button.setColor("#FFFFFF");
    });

    button.addEventListener('touchend', function()
    {
        button.setBackgroundColor("#B8B6B6");
        button.setBorderWidth("0");
        button.setColor("#000000");
    });

    button.addEventListener('touchcancel', function()
    {
        button.setBackgroundColor("#B8B6B6");
        button.setBorderWidth("0");
        button.setColor("#000000");
    });
}
