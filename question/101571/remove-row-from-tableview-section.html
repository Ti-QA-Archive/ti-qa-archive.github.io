  • There's at least 3 ways of removing rows from a section. One was already mentioned, which is resetting the data property with the new table data, this will redraw the whole tableView again.

    2nd option is to set a .name property to the rows and removing like so:

    var row = Titanium.UI.createTableViewRow({ name: 'myrow' });
var section = Titanium.UI.createTableViewSection();
section.add(row);
var table = Titanium.UI.createTableView({ data: [section] });

// will remove the row from the table/section
var index = table.getIndexByName('myrow');
table.deleteRow(index);

    3rd option, using the Titanium.UI.TableViewSection.remove:

    var row = Titanium.UI.createTableViewRow();
var section = Titanium.UI.createTableViewSection();
section.add(row);
var table = Titanium.UI.createTableView({ data: [section] });

// will remove the row from the table/section
section.remove(row)

    Hope that's clear. Cheers
