  • ExternalStorageDirectory is for Android only. So, you can have blob data with Image.toImage().media.

    var blobData = image.toImage().media;
var file = Titanium.Filesystem.getFile(Titanium.Filesystem.externalStorageDirectory,'Image.jpeg');
file.write(blobData);
