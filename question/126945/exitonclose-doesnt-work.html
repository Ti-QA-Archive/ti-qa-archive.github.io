exitOnClose doesn't work

I don't know if I'm doing anything wrong, but this is my simple code (naturally in android) :

var win = Ti.UI.createWindow({url:"/pages/test.js", 
                              exitOnClose:true, 
                              navBarHidden:false});

win.open({modal:true});

// or

Titanium.UI.currentTab.open(win, {animated: true});

if I press the back button on the phone when this windows is open, the app won't close, but it will go back to the previous page.

Is this a bug?

  • exitOnClose property only works in the main app window, usually the first to be opened. On inner windows does not work as you could expect.
