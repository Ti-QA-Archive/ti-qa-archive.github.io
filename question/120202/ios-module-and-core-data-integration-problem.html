  • It turns out the xcdatamodel way will not work. You have to build your Core Data mapping layer programmatically. Its tedious but the only way if it needs to be picked up by the module.

    How to do this? See: http://developer.apple.com/library/mac/#documentation/Cocoa/Conceptual/CoreData/Articles/cdUsingMOM.html
