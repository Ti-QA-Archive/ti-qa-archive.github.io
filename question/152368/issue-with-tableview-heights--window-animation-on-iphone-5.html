  • Looks like you have the table's height set to Ti.UI.SIZE, which should set the height to be just enough to show its contents. The logic behind that autosizing with a table is perhaps getting messed up on reload of your window. Try Ti.UI.FILL instead. That specifies it should fill its parent container.

    Depending on how/when you're creating your button bar at the bottom, you might need to add that to the window first so the table doesn't overlay it. Or, you might have to calculate the height of the table (window height minus button bar height minus other stuff) and set it explicitly.
