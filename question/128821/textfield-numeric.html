textfield numeric

Hi everyone!

I have a proble when setting the value of a text field. I have an int and I want it to appear in the text field but it always appears with decimal point (e.g. 5 => 5.0).
I use a keyboardType of Titanium.UI.KEYBOARD_NUMBER_PAD. The variable I want to put there is for sure an int. Can anyone help?

Thanks

  • Use "toString" method:

    var intNum = 5.0;

textField.value = intNum.toString();

    • No problem Nikunj. But you should be aware that in your test case "intNum" variable is a string and not integer value. In this case it does not make a difference, but in general it is very important to work with real test cases. Also, as I said to Alberto, "parseInt" is useless because converting integer to integer makes no sense. Calling "toString" is actually explained in ECMAScript specification and for round float numbers it will produce string with integer value (didn't know that till recently). You can check that in any browser: alert(5.0.toString())
