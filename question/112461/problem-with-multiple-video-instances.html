Problem with multiple video instances

Hi, I'm having a problem with displaying multiple videos. Specifically, I'm displaying multiple videos in each view of a ScrollableView. Strangely, the first video does not show. I know it is not a specific video because I've changed the order of them and I still have the same result. Anyone else run into this?

  • Just found this note at the bottom of the API:

    "Starting in iPhone 3.2 SDK, the video player can be embedded in any normal view. However, if you create multiple video players at one time, only the last one to have its URL set will display video content or behave as if it is a view for a video player. Prior to 3.2, only fullscreen views were available on iPhone."

    Guess I'll need to come up with a clever work-around. If anyone else has an idea, let me know.
