Custom Dashboard animation?

Hi,
since I develop for Android & iPhone and would like to use dashboard as an app menustructure, i build my own.
It works just fine, but the transistions between the windows are rocky.
right now i have the following in my app.js:

if (Titanium.Platform.name != 'android') {    
    //iOS
    var back = Titanium.UI.createButton({
        title:'Home',            
        style:Titanium.UI.iPhone.SystemButtonStyle.PLAIN
    });
    w.setLeftNavButton(back);
    back.addEventListener('click',function()
    {
        var t = Ti.UI.create2DMatrix().translate(w.width,0);
        DashboardView.opacity = 0;
        DashboardView.visible = true;
         w.animate({left:320,duration:100,opacity:0.1}, function(){
            nav.close(w);
         });
        DashboardView.left = 0 - DashboardView.width;
        DashboardView.opacity = 1;
        DashboardView.animate({left:0, duration:300, opacity:1});
    });
    nav.open(w,{animated:true});
    DashboardView.visible = false;    
} else {
    //android
    w.open({animated:true});
}

Its kind of a hack, because I need to set the mainwindow's opacity to 0, in order to see the sub view.
It seems that Titanium is optimised to tabGroups, would it be better to implement a hidden tab, and control all the transistions there?
Or am I missing something in my custom dashboardview?
