Encrypted Values in Sqlite database

Ok, so as far as I can tell there's no way to encrypt a database (automatically anyway). So I'm looking for other ways to make it difficult for someone to extrat information from the Sqlite database we're bundling with our application.

On caveat: I still need to be able to search the database, so simply base64 encoding the values won't work.

Consider this example sql statement:

select * from mytable where myfield like &#39;EncryptedSearchString%&#39;

Say, the value in the field I'm searching is the base 64 encoded 'examplestring', the value stored int he table would be 'ZXhhbXBsZXN0cmluZw=='

This is fine if someone searches for 'examplestring', but if they only search for 'example', it won't work because the encoded version of 'example' is 'ZXhhbXBsZQ=='

So now we're left with only very simplistic, not very secure methods, such as substitution cyphers, which can be cracked very easily.

Can anyone offer any other solutions?
