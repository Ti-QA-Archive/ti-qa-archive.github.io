How to use Google Maps on iOS devices ?

I'm working on a app with a Titanium map module. Building this app for Android and iOS. Currently, on Android devices map module uses Google Maps, on iOS devices (iphone,ipad..etc) map module uses iMaps. Is there a way that I can use Google maps in the iOS devices as well (instead of using iMaps) ?

I'm using standard titanium map view.

var mapview = Titanium.Map.createView({
    mapType: Titanium.Map.STANDARD_TYPE,
    region: MyRegion,
    regionFit:true,
    userLocation:true,
    annotations:[annotation]
});

  • Your choices would be to embed a Google map in a webview or launch the Google Maps native app (which would put your app into the background). There's a possibility that Apple would reject your app if you embedded a Google map in your app rather than using Apple maps.

    Launching the native map app:

    if(!Ti.Platform.openURL('comgooglemaps://?q=440+Bernardo+Ave,+Mountain+View,+CA')) {
   // if the Google Maps app isn't installed, open the Apple Maps app instead
   Ti.Platform.openURL('maps://maps.google.com/maps?q=440+Bernardo+Ave,+Mountain+View,+CA');
}
