    • You need to edit Titanium.xcconfig to get the correct path and version of the Ti SDK.
      I changed the TITANIUM_SDK from local library to the main one ( so ~/Library to /Library) and the TITANIUM_SDK_VERSION from 1.5.1 to 1.6.2
