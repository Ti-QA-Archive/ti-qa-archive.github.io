App not working on multiple computers

I have created an application and it works great. Compiles fine on Android and on iPhone. Ive tested it on multiple computers, compiled and it works fine.

When i send it off to my client, it fails. It just crashes at the beginning of the application. The beginning of the application downloads a number of 'assets' from a website, but it just crashes on the clients computer. There running the same SDK as myself. We've cleared and reset the iPhone simulator and erased the build directories, but no luck. Does anyone know where i should start to troubleshoot this? The emulator or Titanium dosent provide any useful debugging as to the crash! :(

  • Have your client try to run it with the debugger and set a break point in the beginning of the app to see where exactly it fails.

    Also have a look at the log files. Still not sure how to do that on iPhone, but for Android it's (Window => Show View => Other => Android => Logcat).
