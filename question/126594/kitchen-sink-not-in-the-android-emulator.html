    Try cleaning the project, using TiStudio's Project > Clean… function. Although this probably isn't relevant to your particular issue, always ensure that you are using Google's enhanced SDK, labeled "Google APIs by Google Inc Android API x", when launching the KS, as it utilizes its maps APIs.

    Hope this helps
