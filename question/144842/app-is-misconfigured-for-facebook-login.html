  • Your code looks good. I also had FB login issues on the device, but most of my problems were coming from not configuring my app properly on developer.facebook.com/apps. Here's couple things that was missing on my side:

    • Make sure Sandbox mode is disabled
    • Under permissions you need to set the "User friend Permission"
    • Switching domains will change appid, so make sure you still use the same one
    • Also try to try to test FB outside of your application logic to prove that it has nothing to do with your code somewhere else

  • The problem was de Key Hash of the app, I didnt put the right certificate, my bad >.<.

    this code generate a keystore named cert.keystore and put the hash in cert.txt, this hash is the one is needed by facebook in Key Hashes field

    keytool -exportcert -alias cert -keystore cert.keystore | openssl sha1 -binary | openssl base64 > cert.txt
