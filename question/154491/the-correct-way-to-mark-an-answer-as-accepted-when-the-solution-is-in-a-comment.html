The correct way to mark an answer as accepted when the solution is in a comment

Sometimes I find that the answer to a question in this forum is not correct, but a user in a comment for this response, provides the right solution. If I put it as the correct answer will be providing a false solution, but then I can not make a comment as the accepted solution. Any suggestions on how to act in this situation?

    Hi Xabi,

    This thing happen in very rare case, but what you can do is you can ask the developer who has provided the correct answer to make a separate answer for it, so that you can mark it as correct answer.

    Other option is you can add a comment at last with conclusion that this guy answer is correct one. If you mark answer as accepted it provide help for others, and other can look into your answer as reference.
