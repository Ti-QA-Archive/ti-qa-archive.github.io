    • @Md.Harish-Uz-Jaman Mridha; You can only use xcode on Apple computers as noted in their agreement (see link in my answer), so attempting to use xcode on any other type of computer breaches the agreement and may mean the Apple iOS developer account will be terminated and any apps in the store will be removed. Not ideal, better not to risk it.

      Additionally; whilst you have suggested Adobe Air as an alternative, keep in mind that it does not provide native UI and it is still not possible to upload the app to the Apple App store without a mac - as detailed in the Adobe Air documentation.

      • Important: Apple requires that you use the Apple Application Loader program in order to upload applications to the App Store. Apple only publishes Application Loader for Mac OS X. Thus, while you can develop an AIR application for the iPhone using a Windows computer, you must have access to a computer running OS X..

      So the answer remains the same - needs a mac to create iOS apps.
