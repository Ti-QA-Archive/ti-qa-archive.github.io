    HI Nash,

    use selectedPerson instead of selectedProperty.

    Try this and check again:

    Ti.Contacts.showContacts({
        selectedPerson : function(e) {
            alert(e);
            Ti.API.info('>>>>>>' + JSON.stringify(e));
        }
    });
