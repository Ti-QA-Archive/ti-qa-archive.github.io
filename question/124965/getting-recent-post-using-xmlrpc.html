Getting recent post using XMLRPC

I'm using the XMLRPC on Wordpress to get the recent post using the method metaWeblog.getRecentPosts, but somehow it only returns 20 recent post.
The parameter I pass is blogid, username, password and no. of post.
for example,

var url="http://example.com/xmlrpc.php";
var params = [1,'admin','admin',5];

and then calling the below function to send the HTTPClient() request

xmlrpc(url,"metaWeblog.getRecentPosts",params,callback);

which is referred from http://developer.appcelerator.com/question/1061/xmlrpc

and the callback function is

function callback()
{
Titanium.API.info("XML response is "+this.responseText);
}

it always return exactly 20 recent post not more or less than 20, even I tried to change the last parameter(i.e no. of post to 40) but it still return me 20 recent post.

I also tested the same code on the android and it worked perfectly fine, it only returned me 5 post, even I change the parameters as
var params = [1,'admin','admin',40];

and it returned me 40 recent post.

I also traced and compare the HTTP request of android and iPhone using the HTTPScoop_1.4.3 tool and their was no problem in the request also, both the platform requested the same HTTP request.

I dont understand the problem, am i missing anything or it is a limitation of iPhone or Titanium.
