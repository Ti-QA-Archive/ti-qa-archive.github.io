  • See Android's Supporting Multiple Screens page. Table 3 seems useful. Generally, you'd provide a different splash screen for each size different in aspect ratio, that way you don't necessarily have to clip anything since you have control in whatever graphics package you use. On iOS I have a splash screen for iPhone/iPod Touch, a splash screen for iPad and a splash screen for iPhone 5.
