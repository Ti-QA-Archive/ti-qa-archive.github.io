  • Manu

    What OS are you using? What version of the Titanium SDK for Android are you using?

    Bearing in mind you only have 1GB installed, I am surprised that you are even able to launch an app, due to the Titanium SDK Minimum Memory requirement.

    The time it takes to boot an Android emulator or launch an app to an emulator is almost entirely dependent on Google's SDK. However, with this in mind and to give you an example for comparison, on my Ubuntu (i7 processor, 6GB memory) machine it takes 40 seconds to launch an application to an already-booted emulator, as this video demonstrates.

    This is without using the new Fastdev for Android that has been developed, which would speed turnaround times considerably.

    See the rest of this thread for lots of good advice from other users about improving your development process.
