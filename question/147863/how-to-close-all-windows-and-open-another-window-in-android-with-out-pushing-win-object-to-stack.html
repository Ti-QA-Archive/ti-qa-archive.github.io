How to close all Windows and open another Window in Android with out pushing win object to stack

Hi,

I need a better solution for closing all opened windows and open new window only in android.

My case :-
User can click on logout button in any page. That time I have to close all opened (model:true) windows and show login window.

In iOS I used Titanium.UI.iPhone.NavigationGroup so I called close on main window object.

For Android I am push all windows objects to array and using that I am close the all windows when user click logout. But, I really don't like to have all windows in array because of memory issues and ect..
Because when we comparing iOS and Android.. most of the apps (sample apps and load big images) crashing in Android only.

So, I want find a better solution to close windows with out any app level events nor array of windows.

Thank you,
