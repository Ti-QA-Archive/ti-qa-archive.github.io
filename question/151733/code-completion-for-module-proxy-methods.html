Code completion for module proxy methods

Perhaps I missed the documentation, but is there a way to define the proxy methods in a module I am developing so that code completion works? Ti has code completion for all of its internal modules, but when I test with my own module, code completion does not work.

    That's because Appcelerator does both the modules an the Titanium Studio IDE. So those completions are actually hard coded. You could always extend TiStudio of you wish, but then you would have to provide this extension to everyone that wants to use your custom module.

    Hope this cleared this up for you.
