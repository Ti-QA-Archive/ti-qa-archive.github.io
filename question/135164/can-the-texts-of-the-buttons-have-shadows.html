    • instead of creating a button:

      1> create a view with the backgroundImage which looks like a button.

      2> then add a label to it (set text as the name of the button ) and set label backgroundColor to transparent.

      3> then add another label as shadow of above label with backgroundcolor to transparent and just a bit left to the above lable to make it the shadow of the above.

      — commented April 10th 2012 by Gagan Tiwari

    • Yes, Gagan is right, the Label only have shadow property so you can use in it.

      — commented April 10th 2012 by Gaurang Chhatbar

    • the code for the above solution is here : 

      var view = Ti.UI.createView({
backgroundImage:'button.png'
});
var label1 = Ti.UI.createView({
backgroundColor : 'transparent',
left : 10,
top: 10,
text : 'click',
zIndex: 2,
// add other specifications according to need
});
var label2 = Ti.UI.createView({
backgroundColor : 'transparent',
left : 11,
top: 11,
text : 'click',
zindex : 1
// add other specifications according to need
});
view.add(label1);
view.add(label2);

      dont forget to set the text color of both the label as real for the first one and as shadow for the second one.
