iOS 7 statusBar and navigationBar

Hi,

I am wondering it it is possible to have statusBar and navigationBar two separate color in iOS 7 using (3.1.3):

    var win1 = Titanium.UI.createWindow({
        backgroundColor : 'white',
        title : 'Profile',
        left : 0,
        zIndex : 10,
        barColor : "FFCCFF",
    });
    bm.app.navigationGroup = Titanium.UI.iPhone.createNavigationGroup({
        window : win1,
        left : 0,
        width : Ti.Platform.displayCaps.platformWidth
    });

in iOS 6 only the navigation bar have the color "FFCCFF",

thanks

  • Right now with 3.1.3 and xCode 5.x, a barColor sets the navigationBar and the status bar to the same color on iOS 7. On iOS 6, the status bar is still black.

    If you set a backgroundImage for the navBar, but no color, the statusBar on iOS 7 behaves like it does on ios 6. Yeah, its all weird.

  • Try this one:

    1. Make an image which is 64pt tall (64px for non-retina, and 128px for retina).
    2. The top 20pt of the image should be filled with a color for your status bar.
    3. The remaining 44pt should be filled with your navBar color (as you mentioned, #ffccff)
    4. Use this image for the barImage property.

    Should work, as you expect it to.

    Note that when using an image for the navBar, the translucent property doesn't work.
