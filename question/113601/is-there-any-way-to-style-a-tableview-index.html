  • In the latest Titanium SDK & for iOS7 only. It is possible to change the text colour of the tableview index bar. It's using the TableView property - tintColor. See http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.TableView-property-tintColor
