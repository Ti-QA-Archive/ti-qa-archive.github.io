    Sounds like you are referencing an older version of the facebook module that does not have the 64bit slices in it.

    Version 3.2.0 of the facebook module has that support in it. Be sure you haven't forced a version in your tiapp.xml

    facebook
  3.2.0 armv7 arm64 i386 x86_64
