How to fire the 'click' event in Tabgroup?

In my case, I need to go back the tab's home window.
I use my finger to click the tab and it will back root window. But i use the method "setActiveTab" or the event "click",it cant work..

  • I'm assuming you are building for iOS? There is currently a bug for tabgroup click event on iOS, however, the functionality you are looking for should happen automatically. I've found that it depends on how you have loaded the child windows into the tab. Try loading them with tabGroup.activeTab.open(win);.
