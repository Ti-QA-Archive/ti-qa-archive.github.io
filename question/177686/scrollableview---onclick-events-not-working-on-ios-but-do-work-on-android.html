    Hello, Adam. According to this LINK there is no "onClick" event on "ScrollableView" instead there is "singletap", which is similar.

    Here is a code segment 

    var imageGallery = Titanium.UI.createScrollableView({
    views:images,
    top:0,
    height:450,
    backgroundColor:'transparent',
    showPagingControl:false,
    pagingControlColor:'transparent',
    pagingControlHeight:30,
    maxZoomScale:4.0,
    currentPage:0
});

imageGallery.addEventListener("singletap",function(e){
    alert("test");
});

win.add(imageGallery);

    Hope this helps you. if it does, please mark it as best answer. Thanks.
