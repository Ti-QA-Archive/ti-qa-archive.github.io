How to set an attribute in a view from the controller

Alloy 3.2

Given the following view, how might I set a variable to true or false and make the view exitOnClose use that value?

<Alloy>
    <Window id="win" exitOnClose="<some value set in the controller>" />
</Alloy>

    Your first problem, is that exitOnClose can only be set at creation. So in your styles file or in the xml directly.

    • grab the args[0] that are passed in through

      getController(&#39;myview&#39;, {myExitOnCloseValue: this, bar: that})

      in your controller file, and in there something like this before you open the window

      $.win.exitOnClose =args[0].myExitOnCloseValue;

      — commented January 26th 2014 by Stephen Feather

    • That's the one! Thank you for your time. I was under the impression that the window was created before the "open".
