  • John

    I can sympathise with the frustration, but bear in mind that Titanium unavoidably relies on a number of third party components, so it is quite a complex setup.

    When you were getting this error, "No android-4 or android-1.6 in the Android SDK", you were most likely choosing the old Android SDK (1.6). You should choose 2.2 (2.3 is not yet supported), as in this screenshot:

    .

    Regarding the "loading…" menus, ensure that your Android SDK path is set correctly on the profile page, for example:

    You can often deduce what is going wrong with an installation by recognising the hints that Titanium Developer gives, but you have to try to understand the mechanics of how it works. Try the Troubleshooting Your Installation which gives a better insight.

    Hope this helps
