iPad orientation bug

Hi guys,

Running 1.7.2 on 4.3. I have a blank project. Nothing in app.js.

With my iPad in normal portrait orientation, if I boot the app everything works great. The splash screen comes up and that's the end.

With my iPad in either landscape OR upside down portrait, if I boot the app, the splash screen comes up as it should and orientated as it should. Then, after only a moment, the splash image switches to normal portrait orientation.

In other words, if for some reason I am holding my iPad upside down, and boot my app, I will see it boot correctly for a moment and then it will suddenly switch to being upside down.

Adding a bit of content to app.js I can confirm that it is not just the splash but in fact the entire app is actually displaying the incorrect orientation. If I rotate it a little bit it will catch up and show up right. Also, if I put the app into the background it will pull back in correctly. This only happens when the app is first booted.

Anyone have any suggestions or seen this bug before?

  • Guys,

    This is working for me with TiSDK 1.7.2 on iOS 4.x

    Tested On Emulator 4.x and iPad 1

    I had to copy Info.plist manually to my root project folder (from Build/iphone/Info.plist)

    The key that should be set is:

    <key>UISupportedInterfaceOrientations~ipad</key>
<array>
    <string>UIInterfaceOrientationLandscapeLeft</string>
    <string>UIInterfaceOrientationLandscapeRight</string>
</array>

    And the simple sample code allows the orientation be whether landscape LEFT-RIGHT depending on device's rotation (Home button direction).

    var window = Ti.UI.createWindow({
  left:0,
  top:0,
  width:1024,
  height:768,
  backgroundColor:"red"
});

window.orientationModes = [Titanium.UI.LANDSCAPE_LEFT, Titanium.UI.LANDSCAPE_RIGHT];

var root_view = Ti.UI.createView({
  left:0,
  top:25,
  width:1024,
  height:768
});

var visible = true;
var view1 = Ti.UI.createImageView({
  top:0,
  height:300,
  width:1024,
  left:0,
  image:"http://agro.biodiver.se/wp-content/uploads/2010/09/catie-pano2-1024x300.jpg"
});

var view2 = Ti.UI.createImageView({
  bottom:0,
  left:0,
  width:1024,
  height:300,
  image:"http://kiarasbastelwelt.ki.funpic.de/masken/1024x300DMaskms.jpg"
});

root_view.add(view1);
root_view.add(view2);

window.add(root_view);
window.open();
