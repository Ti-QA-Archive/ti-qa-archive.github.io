    It looks like you are using a not-yet-released version of Alloy, since the errors you are referencing are from the latest code on the master branch of Alloy. I'm assuming alloy --version returns Alloy 1.2.0, which is not yet stable. You'll get more expected behavior perhaps using the latest candidate release:

    sudo npm install -g alloy@1.1.0-cr
