Cannot find Google add on APIs

I've installed Titanium 2.0.1 and installed the Android SDK and Google add on APIs version 8 but when I try to configure for Android, I get the error about needing to install the Add on APIs version 8. The SDK manager says they are installed. I've copied adb.exe and related DLLs from "C:\Program Files (x86)\Android\platform-tools" to "C:\Program Files (x86)\Android\tools". The "Android SDK Home" property in Preferences is "C:\Program Files (x86)\Android\"
I've run the cut and paste command line commands and it thinks the version 8 APIs are installed. Is there something obvious I am missing?
Thank you

  • Eventually, this worked after changing the Android SDK Home in Preferences->Titanium to:
    "C:\Program Files (x86)\Android\android-sdk" and restarting. I can't say for sure if that was the one thing that didid because I installed a bunch of Android and Google stuff over the day and when starting up I found that Titanium finally recognized that it had all the APIs it wanted.
