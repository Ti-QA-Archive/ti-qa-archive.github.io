Android: support for camera overlay while recording video?

Hi,

I'm evaluating Titanium and so far I'm extremely impressed, however I need to be able to create a camera overlay and record video while that overlay is displayed. To date I've explored using the Ti.Media.openCamera() method to create an overlay but it seems that Android doesn't support the startVideoCapture() and stopVideoCapture() methods available on iOS (from the Kitchensink examples) which aren't even dodocumented.

The only way to record video on Android that I have discovered seems to be to use Android specific code and create an intent (eg: Titanium.Android.createIntent({ action: 'android.media.action.VIDEO_CAPTURE' });) but this passes control back to the OS video recorder and hence one cannot create a live video overlay with that (as far as I know).

So far I haven't solved this problem. Is there any other way to do this? Is it perhaps possible by writing one's own plugin or mixing in native Android code?

I haven't provided a test case because this is a very general question about the capabilities of the SDK. Thanks.

Titanium SDK: 2.1.3.GA
Titanium Studio, build: 2.1.2.201208301612
Android SDKs: 2.2 / 2.3.3 / 4.1.2
Samsung Galaxy S2 / AVD Emulator
Windows 7 64 bit

  • Several developers would like to use Titanium APIs for recording video directly from their apps.

    A Feature request has been created on their behalf here:

    Android: Implement Video Recording

    https://jira.appcelerator.org/browse/TIMOB-2122

    I'd like to suggest you to watch that Jira ticket (probably adding a watch or just visiting the report) to have any updates.

    As you have seen (including me) the standard Intent action that can be sent to have the camera application capture a video and return it it wouldn't work to do an overlay (as the OS takes the whole control by camera application)

    e.g. public static final String ACTION_VIDEO_CAPTURE

    Being said, in sake of your requirement I'd like to point out a reliable way for recording video that is using a native module.

    If ignored, may want to have a look at this Android Media Recorder (It is an android module for recording audio and video files) available through out Market Place e.g. or you might want to create your own Module following: http://docs.appcelerator.com/titanium/latest/#!/guide/Android_Module_Development_Guide.

    Hope you or anyone else get over this issue or unless get few pointers to get started quicker, thanks.
