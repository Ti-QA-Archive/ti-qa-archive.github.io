Trouble with binding a model instance to a component

Having trouble getting binding to work.

Got a small demo application containing the following files

Controller: location.js
View: location.xml
Model: coordinate.js

In my view I declare a model instance. This instance is available in the controller and I can see default values and set values on it. My problem is how to bind a the model to a label. I want the label updated when the value of the property changes.

I think the problem is the following line in the view:

<Label text="$.currentCoordinate.speed"/>

How do I bind a model instance?

Here are my files

###View - location.xml

<Alloy>
    <Model src="coordinate" id="currentCoordinate" instance="true"/>
    <View class="container">
        <Label text="Speed:"/>
        <Label text="$.currentCoordinate.speed"/>
        <Button onClick="doClick">Update speed</Button>
    </View>
</Alloy>

Model - coordinate.js

exports.definition = {
    config: {
        columns: {
            "speed": "String",
            "lat" : "String",
            "lon" : "String"
           },
        adapter: {
            type: "properties",
            collection_name: "coordinate"
        },
        defaults: {
            "speed": "10"
        }
    },
    extendModel: function(Model) {
        _.extend(Model.prototype, {
            // extended functions and properties go here
        });

        return Model;
    },
    extendCollection: function(Collection) {
        _.extend(Collection.prototype, {
            // extended functions and properties go here
        });

        return Collection;
    }
};

Controller - location.js

function doClick(e) {
    var coord = $.currentCoordinate;
    coord.set("speed", "110");
}

Im using iOS and SDK 3.1.2.GA

    Use the documented binding notation.

    <Label text="{$.currentCoordinate.speed}"/>
