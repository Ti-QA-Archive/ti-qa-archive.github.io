Dynamic titanium mobile app

Hello folks,

Just wanted to show you guys a draft version of what I'm currently building. It's a dynamic titanium mobile app, configurable by a webinterface.

The app is built in a way, that each view is part of an application structuring tree, which is stored in a database. After each "click" or app start, a check is done, testing whether an update has occured through the webinterface or not. This way, only necessary data will be transferred to the mobile phone.

Note that it's just a "proof of concept", not a version that should be shipped.
Note also that only iOS is currently supported.
Also note, that the webinterface only shows BASIC functionality. I did not spend any effort in making it looking good. It's just a quick&dirty fix for maintaining the sqlite database.

Feel free to contact me in case you have some questions.

Clone it on: https://github.com/flo-bm/TiApp
