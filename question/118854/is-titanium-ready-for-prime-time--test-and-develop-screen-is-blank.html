    • Hi again, Mark

      It's very rare, but I am unable to fault your software configuration! ;) My only suggestion would be to install the Android packages SDK Platform Android 2.1-updatex API 7 and Google APIs by Google Inc., Android API 7, which will allow you to use the next version of the Titanium SDK 1.7.X (or the development version, named "master", already available from the Continuous Builds site).

      However, the reason why Titanium will not work when you launch apps using Titanium SDK 1.5.X and above is that, with only 1GB of physical memory, your system does not meet the minimum requirements. See Titanium SDK Minimum Memory that explains the reasons for this.

      As a temporary workaround, you may be able to use Titanium SDK 1.4.X, although obviously you won't be able to take advantage of the bugfixes and enhancements of the later releases. That said, I have quite happily developed apps using 1.4.X in the past, so it is indeed possible.

      Best of luck with it!
