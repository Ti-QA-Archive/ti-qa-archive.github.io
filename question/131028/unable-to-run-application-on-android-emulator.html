  • Now Finally after a day and understanding and playing with Titanium Sdk I successfully installed it on Emulator. What I did is to click tiapp.xml in project and make Android Runtime as V8 instead of Rhino which is very slow, and upgrade the sdk from Help –> check for Update and then clean the project and change the device to Google API 2.3.3 on WVGA854 and then run it on Emulator and it works like a charm.
