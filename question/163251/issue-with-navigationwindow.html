Issue with NavigationWindow

I created my application in Android and now trying to port it to iOS.

I've 2 index.xml files (for iOS and Android). Inorder to add handle navigations in iOS, I tried like this in index.xml file

<Alloy>
    <NavigationWindow id="navMenu">
        <Require src="home" id="home"/>
    </NavigationWindow>
</Alloy>

And in index.js file, 

Alloy.Globals.navgroup = $.navMenu; // for using the nav window in all the screens

if (Alloy.Globals.isAndroid) {
    $.home.getView().open();
} else {
    $.navMenu.open();
}

Using this, I was able to get the navigation window in my home screen. But the problem is in the child screens of home screen. To open another screen from home screen, I tried like this

var nav = Alloy.Globals.navgroup;
var win = Alloy.createController('child').getView();
nav.open(win,{animated:true});

The child screen is getting opened, but the navigation bar is missing. Please let me know what I'm doing wrong in here.
Thanks !!

//setting action bar in case of android
$.win.addEventListener("open", function(e) {
    var actionBar;
    if (Ti.Platform.osname === "android") {
        if (!$.win.activity) {
            Ti.API.error("Can't access action bar on a lightweight window.");
        } else {
            actionBar = $.win.getActivity().actionBar;
            if (actionBar) {
                actionBar.displayHomeAsUp = true;
                actionBar.onHomeIconItemSelected = function() {
                    $.win.getActivity().finish();
