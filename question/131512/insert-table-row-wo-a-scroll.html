Insert table row w/o a scroll

All,

I'm trying to insert X number of rows above the first row. This is very simple just by calling

                        tableView.insertRowBefore(0, newRowList[i], {
                            animated : false
                        });

My problem here is the tableview scrolls to the new row. I really don't want it to scroll, I just want to stack the rows up top w/o scrolling to the new row.

This seems like it should be very simple and I'm missing something obvious, but I can't seem to figure out why the Titanium table wrapper is scrolling w/o me telling it to.

Any thoughts would be great.

Thanks!
