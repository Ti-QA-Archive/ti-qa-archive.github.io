Adding a TextArea to a TableViewRow

I have a TableView with custom rows which contain Images, Labels, TextFields and TextAreas. Instead of adding a 'click' EventListener for each UI element, or even each row, I'm using one for the whole table. This works fine for most rows, but click events on the TextAreas are not registered.

If I add something like:

tableView.addEventListener('click', function(e) {
    Ti.API.info(e.source);
})

then nothing appears in the Console if I click the TextArea. If I click a different UI Element, I get the source no problem.

Has anyone else come across this problem? Is there any restriction on adding TextAreas to TableViewRows? I'm using iOS 8, Classic, SDK 3.4.1
