Clear textfield in tableview

I have a textfield inside a tablerow. When the user enters some text and then hits return, I want this textfield to clear. It should work like the way Wunderlist adds new folders/tasks.

I've tried updaterow but it takes in data, not textfield. I've tried just setting the textfield to empty but the view doesn't get updated.

How do I go about doing this? I want the clear the textfield on Blur or return event.

  • To build on what Nikunj said above, you can gain access to the appropriate textfield from the children of the row. The children's index is based on when views are added to the row. Putting comments for the child help like below:

    tablerow.add(label1) //child 0;
tablerow.add(textbox1)//child 1;

    Assuming you are using a click listener with a return variable of e on the table, you can gain access to the textbox by e.row.children[1]

    Hope this helps.
