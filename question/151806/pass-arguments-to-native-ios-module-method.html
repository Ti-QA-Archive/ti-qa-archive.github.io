Pass arguments to native iOS module method

Hello all,

I try to find out how to get some argument values passed from Appcelerator project to my Module.
This is what I have in the module.

- (id) someMethodWithArguments: (id) args;

This is what I need to pass from JavaScript.

myModule.someMethodWithArguments({
"aString":"SomeValue",
"theNSDictionary":"How do I represent this to JavaScript?",
"aBoolean":"false"
});

What I do inside the '- (id) someMethodWithArguments: (id) args;' is

NSString* aString;
NSDictionary* objectKeyPairs;
BOOL aBoolean;

aString = [TiUtils stringValue:[args objectAtIndex: 0]];
objectKeyPairs = HOW DO I GET THE NSDictionary to iOS code??
aBoolean = [TiUtils boolValue:[args objectAtIndex: 2]];

// The above are not working....

So what I need is,

  1. How do I pass a NSDictionary from JavaScript to iOS Module method.
  2. How do I proper parse the values of the JSON representation passed.
  3. How do I get the NSDictionary to the iOS code.

Can you please guide me to the correct way?

Thank you.

  • I haven't tested it, but I think you can just cast the args to an NSDictionary, like so:

    - (id)someMethodWithArguments:(id)args
{
    NSDictionary* dict = (NSDictionary*)args;
    NSString* aString = (NSString*)[dict objectAtIndex:0];
    // Do something
}
