TabBar background images problem

Hi everyone, I'm developing an app with a TabGroup that has 4 tabs, I've been trying to use a custom image for my tabs on the "tab bar" (I'm not sure thats the real name but many post call it that way) What I want is to have my 4 different images instead of the normal tabs and I want them to change to a different image once selected.

This is what I did the first time:

var myTab = Titanium.UI.createTab({
        backgroundImage: '/images/rec.png',
        backgroundColor: 'transparent',
        titleid: myTitleId,        
        window: myWinName
    });

later I tried with:

var myTab = Titanium.UI.createTab({        
        backgroundDisabledImage: 'images/rec.png',
        backgroundFocusedImage: 'images/rec_selected.png',
        backgroundColor: 'transparent',
        //icon: "/images/recomendados.png",
        titleid: myTitleId,

        window: myWinName
    });

I now the second one is a android only solution but it didn't work either… any idea?

    Hi Sergio

    The TabBar and Tabs do not support the background image properties even if the documentation still suggests they can. There is no standard way of changing them as the TabBar is a standard iOS object.

    But there is a custom way explained in a previous blog post. That should solve your problem.
