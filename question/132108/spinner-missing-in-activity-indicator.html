    • For a large activity indicator you can use:

      actInd.setStyle(Titanium.UI.iPhone.ActivityIndicatorStyle.BIG);

      beware that the big act ind is white, so you should use it on a dark background.
      You can also set the font & color of the message text. For more info look here: http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.UI.ActivityIndicator-object
