How can we implement handleOpenURL functionality of iPhone in Titanium

How can we implement handleOpenURL functionality of iPhone in Titanium?

iPhone Code is below:

- (BOOL)application:(UIApplication *)application handleOpenURL:(NSURL *)url
{
    if (!url) {  return NO; }

    NSString *URLString = [url absoluteString];
    [[NSUserDefaults standardUserDefaults] setObject:URLString forKey:@"url"];
    [[NSUserDefaults standardUserDefaults] synchronize];
    return YES;
}

  • This can easily be solved by the following code without implementing the "handleOpenURL"

    Place the following code in your app.js

    var startupWithArguments = function() {
    var args = Ti.App.getArguments();
    if(args.url) {
        // Do cool stuff here
    }
}

// Call when app is started
startupWithArguments();

Ti.App.addEventListener("resumed", function() {
    // Call when app is resumed from background
    startupWithArguments();
});
