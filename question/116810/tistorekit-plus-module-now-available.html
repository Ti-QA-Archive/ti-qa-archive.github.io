  • I think details are sparse so far. There is a webinar on Mar 9 that you can / should register for here which should answer lots of questions, plus there is time for Q&A set aside from what I read.

    Don't worry too much about the version 0.1 – as someone who has developed a few iOS modules myself, I can tell you that is just the default version number assigned to a project until you change it. That said, I doubt any of these modules have been tested outside Appcelerator internal projects, because I don't think any of them have been publicly available in the past. Perhaps some Pro or higher subscribers were able to get early access copies, but since I am not in that crowd I could not say.

    "As you move into commercial development, we have created new small business and enterprise packages that feature the new Titanium+Plus modules for commercial deployment. Pricing starts at $2,500/year."

    It is unclear to me if there are any module license alternatives separate from a Pro/Small Business or higher subscription. I am sure details will be forthcoming within a week, or less.

  • You can download the free modules listed in the blog linked below. Unzip them into "/Library/Application Support/Titanium". There is documentation in the source code for the free/open source modules linked below.

    For those with a Professional license, login and click on each module listed on the Ti+Plus page and download each file separately. So far I count 16 modules including the free ones. There doesn't seem to be any documentation in the module, on the live Documentation or in the Developer page. I expect it'll all come out with the webinar on March 9.

    For fast unzipping of all 16 modules if you download them to the Downloads folder and from "/Library/Application Support/Titanium" within a terminal:
    for file in ~&#x2F;Downloads&#x2F;ti.*.zip ; do unzip $file ; done

    Good luck!
