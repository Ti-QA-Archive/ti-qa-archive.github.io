Global Namespace

I've been trying to set this up, but after a few hours, I just can figure it out. Pleas help me out.

So, in my app.js file, I made an object var cloud = {}; like so.

In a file cloud.js, I defined functions in the cloud namespace:

cloud.login = function() {
...
}

cloud.logout = function() {
...
}

Then I did a Ti.include(&#39;cloud.js&#39;) inside app.js to make it available.

Inside another file logic.js, which holds my app's primary logic, I can call the functions defined under the cloud namespace perfectly fine.

I have a window, which uses a file, main.js defined by the window's url property. I can't seem to access the cloud namespace inside that window.

I know I'm not doing something right, but I can't figure out what. If there is an alternative method you can suggest, please do. I've just started working on this app, so making changes shouldn't be a problem.

META:
iOS: 5.1
Ti SDK: 2.1.1.GA

    When you use the url property to open a new window, it creates a new context which is separate from the context that you've already go open. This is one of the reasons that we consider it a best practice to use a single context when creating your apps.

    The suggestion would be to use the commonjs approach when creating your apps. This includes using require rather than include and not using the url property when creating new windows.
