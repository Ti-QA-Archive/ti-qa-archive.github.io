    Hi Victor

    You will need to add this listener to your window.

    win.addEventListener('doubletap', function (e) {  //  chane win to suit
    Ti.App.fireEvent('showspecial', {  
        optional: 'properties'
     });
});

    Then whereever you have created the scrollView add this;

    Ti.App.addEventListener('showspecial', function (e) {
    scrollView.show(); // change scrollView to suit
});
