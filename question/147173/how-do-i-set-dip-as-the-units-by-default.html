How do I set dip as the units by default?

Hi guys. I'm working with dip units in iPhone and want to do the same with Android. The app was set as "system", so iPhone was using dip and Android was using pixels. I didn't want to specify the units in every value, so I tried to declare the units by default in the tiapp.xml file. I replaced the line:

<property name="ti.ui.defaultunit">system</property>

by the line:

<property name="ti.ui.defaultunit">dip</property>

but it didn't work. The Android emulator is still using pixels. So I cleaned the project and even restart Titanium, but nothing.

Is there something else I have to do to set dip as the units by default so I don't have to declare the units in every value?

Thanks
