  • Use "Ti.JSON.parse" to convert JSON string to JavaScript object.

    var feed = Ti.JSON.parse(jsonString);

    Now you can access object's properties as it is usual in JavaScript (dot notation or square-bracket notation):

    Ti.API.info(feed.data.mat_78.id);
Ti.API.info(feed["data"]["mat_78"]["title"]);
