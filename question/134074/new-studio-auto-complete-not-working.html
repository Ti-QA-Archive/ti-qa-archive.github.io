    I figured it out. Should i post the problem any place in addition to here.

    The problem is if you have the project also set up as mobile web and built it so that it is In the build/mobileweb folder the software is getting the TI from those files (titanium.js in the titanium folder)

  • Accepted Answer

    For those interested in the issue, the fact is that we were indexing the items under the build/mobileweb folder. In Studio 2.0.0+ we mark those resources as "derived" and don't index them. It may be that in 1.0.9 you can also right-click on the build folder, mark it as derived, and the same fix will work. If not, 2.0.0 is available on the preview website.
