backgroundImage don't updates in click event of table row (Android 1.6.x)

When setting the backgroundImage of a row in a table click event like this:

tableView.addEventListener('click', function(e){    
    try {
        e.row.backgroundImage = '/images/background.selected.png';
    } catch(e){}
});

the row don't updates itself (Android 1.6.x). After scrolling the row out of view and back in the row has the new background image! Is there a way to force the row to update itself (hide and show don't work)?
