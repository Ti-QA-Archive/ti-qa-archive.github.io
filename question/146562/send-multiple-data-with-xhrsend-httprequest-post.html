    Hi

    Short answer; try not to repeatedly poll xhr as you may find it difficult to handle the number of send/success/failure handles.

    The best way is to build up an array or stream of data and send in one go.

    If you have LOTS of data that this is impractical then I would ask what sort of data and how much are you actually sending?

    Another method would be to create a temporary file containing the data and then upload the file for the server to process.
