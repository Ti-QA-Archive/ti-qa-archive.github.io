  • For the new window, use activity.onCreateOptionsMenu

    http://docs.appcelerator.com/titanium/2.1/index.html#!/api/Titanium.Android.Menu

    Keep in mind you need to make a heavyweight window to be able to set a menu to it.

    • A heavyweight window is a window that has set one of the next properties (true or false, it doesn't matter): navBarHidden, fullScreen, url <- don't use this one.

      It will open anew activity and will generate a new context for the app.

      var activity = win.activity;
