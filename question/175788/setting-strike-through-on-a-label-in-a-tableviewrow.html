Setting Strike Through on a Label in a TableViewRow

Hi,

I am trying to figure out how to set the text inside a label within a tableviewrow when that row is clicked. I am using Alloy to create my list and bind it to my data source but I am not sure how to go about setting the text to being strikethrough. I tried setting the style but that doesn't seem to work.

    You don't mention which platform you're building for. If you're building for iOS, you can use attributedStrings to apply strike-through formatting to a string. There's an example in the guides.

    For Android, you might be able to find a custom font that includes the line in its glyphs. Or, on each Label, set the html property equal to <strike>your text</strike> rather than using the label's text property.

    Short answer; You can not.

    Longer answer;

    The only properties available for font are;

    • fontFamily - name of font
    • fontSize - number denoted size
    • fontStyle - italic or not
    • fontWeight - bold or not
    • textStyle (iOS only; links standard styles like headline and body)
