Struggling to get a pulse effect.

I want a label I have to appear like its pulsing.

I have the label added to the window called glow, and its opacity set to 0.

The following code is in place, but seems to only run once. Is there an easy way I can get it to run continuously, or get a pulsing effect another way?

The label… 

var glow = Titanium.UI.createLabel({

    backgroundImage:"glow2.png",
    height:94,
    bottom:0,
    left:0,
    width:320,
    opacity:1

});

var glowing = 1;

The code in the main body…

if (glowing == 1){
            glow.animate({opacity:0, duration:600});
            setTimeout(function() 
                { 
                glowing = 2;
                },600);
            }
if (glowing == 2){
            glow.animate({opacity:1, duration:600});
            setTimeout(function() 
                { 
                glowing = 1;
                },600);
            }

Cheers

    Giving it a function and running the function after the setTimeout it should do the trick!

    
function glowfunc() {

    if (glowing == 1){
        glow.animate({opacity:0, duration:600});
        setTimeout(function() { 
            glowfunc();
        },600);
    }

    if (glowing == 2){
        glow.animate({opacity:1, duration:600});
        setTimeout(function() { 
            glowfunc();
        },600);
    }

}
