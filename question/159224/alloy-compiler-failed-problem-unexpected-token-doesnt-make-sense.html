  • I actually found the problem. I had to look in the Home.xml file not Home.js as the error mentioned. I was using

    extendEdges="Ti.UI.EXTEND_EDGE_BOTTOM, Ti.UI.EXTEND_EDGE_TOP"

    instead of

    extendEdges="[Ti.UI.EXTEND_EDGE_BOTTOM, Ti.UI.EXTEND_EDGE_TOP]"
