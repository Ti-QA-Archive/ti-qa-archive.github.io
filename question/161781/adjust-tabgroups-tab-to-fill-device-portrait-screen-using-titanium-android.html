Adjust tabgroup's tab to fill device (portrait) screen using Titanium Android

Hi I have a tab group with 3 tabs, and when viewing this in larger screen the tabs width fill the device screen(portrait) but when viewing my app on smaller screen the tabs width looks compressed and the tab group having a horizontal scroll. How can I automatically adjust tab group's width to fill the device (portrait) screen.

  • You can hardly influence the tabgroups default behavior on Android. It is designed to always fill the screens width. When the Tabs width gets too small (because there are so many Tabs or its titles are that long) they become scrollable if they don't fit. ActionBar Tabs could also collapse into a dropdown navigation in landscape mode. It all depends on screen size and orientation. A possible workaround would be using a custom tabgroup
