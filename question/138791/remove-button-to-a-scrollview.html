  • zozo,

    It be a lot easier for use to assist you if you provided a simplified example of your code. Often just taking the time to simplify you code will provide you with your own answer.

    Here is an example of removing a button.

    var win = Ti.UI.createWindow({
    backgroundColor:'yellow',
});
var button = Ti.UI.createButton({
    title: 'Remove Me'
});
win.add(button);
button.addEventListener('click', function(e) {
        win.remove(button);
});
win.open();
