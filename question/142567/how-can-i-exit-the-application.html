How can I exit the application?

I am trying to find a way to have a menu item where user can exit the application. My implementation looks similar to:

            var activity = m_main_window.activity;
            activity.onCreateOptionsMenu = function(e){
                var menu = e.menu;
                var menu_item = menu.add({title: 'Exit Application'});
                menu_item.addEventListener("click", function(e){
                    var activity = Ti.Android.currentActivity;
                    activity.finish();
                });

However, nothing seems to happen when user selects that menu item. I do reach inside the click callback function though, but activity.finish() doesn't look like it's doing anything.

Any thoughts?

  • Instead, close the main window. If m_main_window is the base window for your entire application, just call m_main_window.close(). If not, fire off an app level event to the main window, and close it there.
