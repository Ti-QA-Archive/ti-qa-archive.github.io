New Ti GA 3.1.2 Android Back Button no longer works!

Hi there,

Ti Studio prompted me to install the new 3.1.3 GA and when I tried it, it reminded me that as of 3.1.2GA the Back Button event no longer works.

I've not changed the code and it has always worked. The only thing that's different is the SDK version chosen in my tiapp.xml

When I roll it back to 3.1.2.v20130624104600 it works just fine!

gameWindow.addEventListener(&#39;android:back&#39;, menuExit);

Running on both an LT18i (Sony Ericsson Arc S) Android 2.3.4 and on an Asus Transformer TF300T - Android 4.2.1
