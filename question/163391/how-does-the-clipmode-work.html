How does the clipMode work?

Hey guys, I'm trying to wrap my head around the clipMode property of a view.

Here is what I want to achieve: I have a green view inside a wrapper. The wrapper is smaller than the green view but I want the green view to be fully visible anyway.

Here's the code:

var win = Ti.UI.createWindow();

var wrapper = Ti.UI.createView({
    width: 320, top: 40,
    height: 160,
    backgroundColor: 'transparent',
    clipMode: Titanium.UI.iOS.CLIP_MODE_DISABLED
});

var photoView = Ti.UI.createImageView({
    width: 320, top: -40,
    height: 200,
    backgroundColor: 'green'
});

wrapper.add(photoView);

win.add(wrapper);

win.open();

The green view is clipped by the wrapper, resulting in a height of 160 instead of 200. In fact the clipMode property seems to be doing nothing at all.

Tested on iOS simulator, Titanium SDK 3.2.1 GA.

best, Tom

    This one might turn out to be something simple and something annoying at the same time.

    • clipMode: CLIP MODE DISABLED

    This is the correct property and the correct value. The issue is that it will be available in iOS only and note the "will" - it is a 3.3 feature only that is currently in beta state right now.
