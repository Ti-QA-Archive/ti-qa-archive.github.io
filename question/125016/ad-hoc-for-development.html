Ad Hoc for Development

Hey! I'd like to know if an Ad Hoc profile is safe to use for development testing? I don't know why, but I never get on with the 'Run on device' option, so for testing I always distribute to an ad hoc profile, and install on my iphone.

This works absolutely fine, no problems, but I'd just like to know if it is safe to keep using ad hoc profiles for development (I might be re-installing about 5 times a day)

    Yes, it is safe to use a (distribution) ad hoc profile for development testing. It doesn't matter how many times the profile gets installed – it only matters that one is available at the time of the install and when the application is launched.

    It technically doesn't matter if the profile is even in the .app or .ipa bundle if the profile is already on the device and not expired. However it doesn't hurt to have it included all the time and thus that is the easier thing to do because that way you don't have to care if the profile already exists.
