Can't Install Titanium CLI???

Hi Folks.

Today I decided to upgrade to the latest version of Titanium (3.0.1) and the latest version of XCode (4.5.2)
On my mac running OSX 10.8.2

My only problem is that in Titanium there is a pop-up that prompts me to download 'Titanium CLI 3.0.22'.
When I click 'Install' it goes through the motions and then I get an error message: 'Installing Titanium Updates…' has encountered a problem. When I click on the details box it tells me: Failed to install titanium.

So I have tried uninstalling and reinstalling Titanium a couple of times now but the exact same stuff keeps happening to me. Because of this error I am unable to run the iPad Simulator (I am primarily working on an iPad app).

Does anyone know what is going on here or how to fix it?

  • Can you verify that the CLI is installed by running titanium -v at the command line? It should output "3.0.22". If that works, then run titanium sdk and verify that the correct SDK version is marked as "active". Lastly, makes sure the <sdk-version> in the tiapp.xml is set to the correct version.
