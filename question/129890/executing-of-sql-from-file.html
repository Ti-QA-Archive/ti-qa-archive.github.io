Executing of SQL from file

Hi,
I try create sql table.
Here my code:

var sqlfile = Ti.Filesystem.getFile( Ti.Filesystem.resourcesDirectory,'web/data/table.sql');
var sql = sqlfile.read();
conn.execute(sql);

In this file is:

CREATE TABLE IF NOT EXISTS users (
    id INTEGER PRIMARY KEY,
    vname TEXT, 
    fname TEXT
);

On the console is logged:

[ERROR] Script Error = -[TiBlob stringByTrimmingCharactersInSet:]: unrecognized selector sent to instance 0x9817060 at ctrl.clients.js (line 15).

The problem is the linefeed in sql …

Any ideas?

    Hello Rainer.
    as you can see the Console (and there for the compiler) is treating your file as Blob. you should use toString function and after that trim function.
    try this:
    conn.execute(sql.toString().trim());
