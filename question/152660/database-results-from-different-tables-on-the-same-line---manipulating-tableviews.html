Database results from different tables on the same line - manipulating TableViews

I can return one result per line but cant work out how to manipulate tableviews/rows to include different pieces of data on the same line. Can anyone tell me where to go from here, please?

    • Thanks very much, Trevor - I couldn't see for looking (and, yes, my Javascript is still pretty basic!) This works fine - though I could probably tidy it up:

      var sql1 = db.execute('SELECT * FROM table1 ORDER BY RANDOM() LIMIT 1;');
var sql2 = db.execute('SELECT * FROM table2 ORDER BY RANDOM() LIMIT 1;');
var data = [{
    title:'A ' + sql1.fieldByName('trait') + sql2.fieldByName('jobtitle'),
    }];
