Offical Font Awesome 4.1.0 for Titanium Alloy

Hi all,

Font Awesome has been a long time favorite of mine with web development. I did buy FontAwesome AppIcons from Titanium Marketplace but it wasn't what I was looking for. So I decided to make a widget that uses the latest version (4.1.0 released 5/14/2014) and lets you add icons in a similar way done with html. With the widget installed, you only have to work with the Views.

The font used is a direct legitimate copy of the one found at fontawesome.io - and is free to use commercially.

The widget I created is also 100% free.

With the widget installed and included properly, all you have to do is <Label icon="fa-whatever"/> and all icon names match the fontawesome.io/icons names 100% - so reference is super easy.

I intend to keep it up to date with fontawesome.io.

Please have a look at the github repo and let me know what you think!

Thanks!
