  • It certainly doesn't help that builder.py forces you to use device arguments. I got my build system working by modifying builder.py to allow for no device args instead of defaulting on -d, -e, and -s.

    Just search around for '-d', '-s', and '-e' removing any case where it's just defaulting, then remove self.device_args from any calls if it is set to None.

    Also note that adbWireless doesn't work when your device goes to sleep so keep it awake during the build and run and it will work.
