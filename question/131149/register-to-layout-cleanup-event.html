register to layout cleanup event

hi, I'm looking for a way to create custom events on Titanium.
inorder to do that I would need to register to a global var array every time the window opens (not a problem) however, I would have to unregister every time the window is cleaned from the memory, the event that fires then is the E&#x2F;TiBaseActivity(10448): (main) Layout cleanup.
I would like to register to that event with a unregister function for the specific window.

is there a way to do that?
