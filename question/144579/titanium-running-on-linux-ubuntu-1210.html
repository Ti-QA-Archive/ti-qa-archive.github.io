Titanium running on Linux Ubuntu 12.10

Hello.

I just wondered if anyone else was running Titanium on Ubuntu 12.10 and if so was the install straightforward? I have it installed (I believe) 95% successfully but have hit a stumbling block.

I have a fresh install of Ubuntu, successfully installed and setup JDK 6u37, set paths etc (or the equivalent in Linux), installed Android SDK and required platforms etc. Titanium starts and says that Android SDK is installed correctly (yay). I open a new project using the default 2 tab template, but when I go to run the app (to see that my install is really successful) the Android Emulator is not an option on the little dropdown button, only mobile web. Android is DEFINITELY ticked on the list of deployment targets on the project setup page and Titanium is configured to find the Android SDK in the correct folder.

Anyone else had this occur? I've been back and double checked that Android is selected as a deployment target and it is. I've also opened another new project but no luck.

BTW, I've been using Titanium successfully on a MAC when at work but wanted to use it at home too, hence the Ubuntu option (I don't have a MAC at home and WinXP isn't listed as being a supported platform for Titanium). I thought I knew my way around Titanium but this has me stumped.

Thanks in advance for any advice.

Phil
