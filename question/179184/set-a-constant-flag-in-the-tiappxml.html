Set a constant flag in the tiapp.xml

Hi all.
Is it possible to set a constant variable in the tiapp.xml file?

Basically I would like to set a 'Development' flag in the file that when set to true I can switch the app into dev mode where it point to our UAT URLs.

Any ideas?

    You realize that Alloy provides such flags for you automatically, right? So with that usual plug out of the way…

    In tiapp.xml, add:

    <property name="foo" type="string">value</property>

    In your app:

    var default = 'dev';
var environment = Ti.App.Properties.getString("foo", default);

    See the Properties docs for more info.
