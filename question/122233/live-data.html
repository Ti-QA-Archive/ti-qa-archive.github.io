  • The questions seems rather abstract and a bit vague, so I'm just going to throw this out there for you. Since multiple people will be working on the database, that means the database must be remote. With constantly changing information you can either push those changes to the device or have the device query for the data while it is open. The best way to execute as well as the best way to display the data really depends on the data and your service.
