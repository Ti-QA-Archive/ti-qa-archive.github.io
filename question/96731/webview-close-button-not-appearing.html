Webview close button not appearing

I have a "Tweet This" button in my app. I want it to launch a webview that links to Twitter and has the user log in to tweet some text from my app. This is the code I have but it's not working properly.

     tweetButton.addEventListener('click', function(data) {
         var tweetWindow = Titanium.UI.createWindow();

         var webview = Titanium.UI.createWebView({
                 url:'http://bit.ly/e6dhTW'
                 }); 
                 tweetWindow.add(webview);
                 tweetWindow.open({
                     modal:true
                     });

                 var closeTweetButton = Titanium.UI.createButton({
                    title:'Close'
                    });
                closeTweetButton.addEventListener('click', function() {
                        tweetWindow.close();
                        });
                tweetWindow.setLeftNavButton(closeTweetButton);
         });

Basically, I want a "Close" button to appear at the top left of the webview, but the webview is appearing completely full screen and I have to force quit my app in order to go back. I'm doing everything according to the API like setLeftNavButton and it's not working.
