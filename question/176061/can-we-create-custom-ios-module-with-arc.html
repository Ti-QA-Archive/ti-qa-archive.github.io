Can we create Custom iOS Module with ARC

Application Type: Mobile

Titanium SDK: 3.2.3

Platform: iOS 7.1.x

Device: iPad Simulator

Host OS: OS X 10.9.3

###Description:-
I have an iOS Application which allows me to view PDFs and perform annotations on them. I tried to create a Titanium Module to encapsulate this application for my Titanium Projects. I have created the module and I am able to build this just fine. But when I try to test the application, it is crashing. I don't see any errors in the "build.log" file either. I later found that the existing iOS Application was developed using ARC and this was causing run time errors to occur.

###Question:-
Is it possible to create an iOS Module where the Native Code is built with ARC?

Thanks,
Nirumal.

    Hi Nirumal,

    Yes it's possible, just turn on ARC in the build settings of your Xcode project and remove the dealloc method from the default module template. For reference see TIMOB-12558
