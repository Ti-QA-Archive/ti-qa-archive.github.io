Titanium sdk updates

Hi! I developed an application using Titanium sdk 1.7.2 version for android.My question is now Titanium sdk 1.7.3 is available. Titanium studio asking for sdk updates.If I update sdk then my application will work without any problem or not?
Does I need to change any configurations? Any one can help me.Thanks in advance.

  • You don't need to change any configuration - just update (and make sure tiapp.xml uses the new SDK). your app should work as it did before without any problems - and you might even find that things that didn't work before (bugs in the sdk) work now.

    In any case, it's always recommended to test your app to see that nothing is wrong with it (it's just good practice) - and, you can always go back to 1.7.2 if you need.
