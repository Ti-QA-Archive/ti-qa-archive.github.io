Insert rows into tableViewSection

Hello,

I have a similiar issue as described here: NBL-34176-722.

I would like to add rows to an exsiting tableViewSection.

var nData = _data[e.section._groupID].slice(4);
                var r = yh.ui.createOverviewRows(nData);
                var p = e.index+(e.section.rowCount-1);
                var j = 0;
                for(var j in r){
                    Ti.API.info(p);
                    var index = Number(p)+Number(j);
                    tV.insertRowAfter(index,r[j]);
                    Ti.API.info(index);
                    if(j == r.length-1){
                        tV.deleteRow(e.index);
                    }
                }

nData is the second part of my data array. These rows shall now be added to the section. If I add only two rows everything is fine, when adding four new rows two are added to the correct section, the other two to the following.

What am I doing wrong?

  • if tV represents your TableView and not your TableViewSection, then that is your problem. I would assume this is the case because insertRowAfter() is a function of TableView, not TableViewSection. Check out the example at the end of the API documentation for TableViewSection to see how you should be adding rows to it:

    http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.UI.TableViewSection-object
