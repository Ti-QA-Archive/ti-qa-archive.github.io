    SOLVED!

    Must use:

    var fb = require('facebook');
fb.appid = '375xxxxx'
fb.permissions = ['publish_stream', 'read_stream'];
fb.

    … fb.AllProperties … Not: MODULE.Facebook.method as you would think the Documentation may suggest here: http://docs.appcelerator.com/titanium/latest/#!/api/Modules.Facebook (Top Class Name)
