Problem - Open android application when clicking on sms

Hey guys,

I'm developing an android app and I need to do some action when the user clicks on a sms received.

Lets say that the user is at the HomeView and after clicking on the message it has to be redirected to the AcceptView. That works fine but the problem is that when the application is minimized (pause) and then resumed, it just shows a black screen. It only happens when I click on that sms, if not, pause and resume works fine.

Also, when that happens, I can see at the Hierarchy View that there are two windows for the app so it seems to be duplicated when it is resumed.

This is what I have in the AndroidManifest:

        <activity android:name="TestActivity"
            android:label="Test" android:theme="@style/Theme.Titanium"
            android:configChanges="keyboardHidden|orientation"
            android:screenOrientation="portrait" 
            android:alwaysRetainTaskState="true">
            <intent-filter>
                <action android:name="android.intent.action.MAIN" />
                <category android:name="android.intent.category.LAUNCHER" />
            </intent-filter>
            <intent-filter>
                <action android:name="android.intent.action.VIEW" />
                <category android:name="android.intent.category.DEFAULT" />
                <category android:name="android.intent.category.BROWSABLE" />
                <data android:scheme="http" android:host="test.com" />
            </intent-filter>            
        </activity>

I tried changing launchMode but it didn't work.

Any help would be appreciated!
