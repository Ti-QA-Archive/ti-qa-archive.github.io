Webview get div id

Dear TI experts,

I am accessing a remote URL using XHR and I am getting the complete html file , now I want to access a particular <DIV> only from the remote html, can somebody pls help me on this , i m using the following code and its unsuccessful. it is throwing

Result of expression 'dhtml.documentElement' [undefined] is not an object.

var xhr = Titanium.Network.createHTTPClient();

xhr.open("GET","http://www.google.com/");
xhr.onload = function(){
var sCode = xhr.status;

if(sCode == 200)
{
    var dhtml = this.responseText;
    Titanium.API.log("html","Response: "+dhtml);
    var myDiv = dhtml.documentElement.getElementById("div1");

    Ti.API.info('result: '+myDiv.innerText);

}

};

xhr.send();

my html is

html….

<div id=div1>
table…….
<div id=div2>
table……..

    this.responseText returns a string, not a DOM object, so you can't really look for getElementById on it, and not documentElement for that matter, only string operations. xhr is NOT jQuery, so you can't maniuplate the content like jQuery does.

    My best bet is looking for that div using string operations (substr and regular expressions).
