  • Do you have a backend server somewhere? You should handle role management remotely via web services. I would:

    1. Create REST web service somewhere since we all love JSON.
    2. When the user logs in to your app, it calls the web service and returns the users role.
    3. Inside your app you have flags per role for what the user can see/access.

    • When the user logs in, you set there role.

      Ti.App.Properties.setString('role', 'admin');

      Then from anywhere in your app you would just use a simple if statement.

      var someRestrictedButton = ...;
    someRestrictedButton.hide();
if (Ti.App.Properties.getString('role') == 'admin') {
    someRestrictedButton.show();
}

      This way, only a user with role admin can see the button.
