Wait for Tableview loading complete

I have a table view that must have time ti populate itself (depending on previous click)

I add it in my window shifted on the left, then i animate it.
On debug i show a warning:
[DEBUG] animated called and we're not ready … (will try again) <TiUITableView: 0x8992f90…..

How is the right sintax to preload it before do an animation?

Thanks
Stefano

  • When you collected all data to populate the tableView (I think you're retrieving that with the Titanium.Network.createHTTPClient method) set the data for the tableView with tableView.setData = data;
    If you are using this method method place tableView.setData = data; in the onload event of your HTTP call.
