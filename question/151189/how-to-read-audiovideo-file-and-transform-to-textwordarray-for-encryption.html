How to read Audio/Video file and transform to text/wordarray for Encryption?

Titanium: 3.0
HTML5

I am currently using sjcl (http://crypto.stanford.edu/sjcl/) to encrypt/decrypt html5 content used on an iOS app. I am embedding source images and encrypting the text content of the html and then decrypting in the app to display, though having tried to embed the video/audio does not work for some reason, not sure if is a size issue. So, I need to encrypt video/audio before build app, then decrypt to temp file or to the 'Documents' folder (since using NSFileProtectionComplete in Entitlements protects the files there when device is passcode locked.

Using iExplore, I can easily copy out the app bundle resources. I REALIZE that the encryption can be broken, but I need to perform at least some form of protection anyway so it's not just sitting out there. Client requirement.

Without using SSL (as there will not be network access on these devices), how can I read from a video/audio file object in Titanium and transform that into plaintext/wordarray that I can pass to the encryption/decryption functions used in sjcl?
