Get Alloy Widget config properties

Hi !

I made a widget for my Alloy application. This widget will be included in some other apps so I want to make a generic plugin and just some parameters to change in the widget.json config file.

But how can access to the plugin config properties in my app ? With the App config file (config.json) I can use the Alloy.CFG. How can I do with the widget config file ?

Thanks for replies !

    You can't access the maifest file or a config file of a widget right now like you can the main config.json. If you'd like that as a new feature, log a feature request ticket here: https://jira.appcelerator.org/browse/TC. Make sure to label it with "alloy" so I can get it in the schedule.

    You'll probably need to have these config values passed in when the widget is created and then process them in the controller until an external config file is supported.
