I made changes to the iOS UDP module so that it can send bytes instead of just a string.

ChrisFJones UDP Module

However when I drop that zip into the root of my project, or in Library/Application Support/Titanium, it unpackages the module correctly when I run my app, however it then gives me the error:

Couldn't find module: chrisfjones.titanium_module_udp

Cleaning the project and deleting the modules folder in the root of my project has no effect. What am I missing?? Here is the relevant part of the log:
