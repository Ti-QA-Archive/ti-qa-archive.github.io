[ERROR] JDK version 'javac' is not recognized as an internal or external command

Hi,

I am trying to build a small app and facing some problem during build process of mobile app. There is some JDK version 'javac' problem. I have installed the 'jdk1.7.0_15' and 'jre7' on my machine. Here is the log details which I am getting in console during build process after that nothing happens :(

[ERROR] Application Installer abnormal process termination. Process exit value was 1
[INFO] logfile = C:\dataStore\RnD\mobileApp\testapp\build.log
[INFO] Building testapp for Android ... one moment
[ERROR] JDK version 'javac' is not recognized as an internal or external command,
operable program or batch file. detected, but at least 1.6.0 is required

ANY HELP WOULD BE REALLY APPRECIATED

  • Accepted Answer

    your system is not able to find the javac because you have not set the path for javac executable.
    Check your "javac" path on Windows using Windows Explorer "C:\Program Files\Java\jdk1.7.0_02\bin" and copy the address.
