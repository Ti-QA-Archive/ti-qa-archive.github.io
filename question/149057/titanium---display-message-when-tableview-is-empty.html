titanium - display message when tableview is empty

I wold like to display a nice label instead of the tableview when there are no items in the tableview. Does anyone have any code for doing this nicely?

  • Hi

    There is no property for it, but that is not a problem as it can easily be achieved.

    The basic principle of this is to use the footerView property with a label inside and then show/hide that as required.

    var viewEmpty = TI.UI.createView({
    height: Ti.UI.FILL,
    width: Ti.UI.FILL
});
var lblEmpty = Ti.UI.createLabel({
    color: '#000',
    font: {
        fontSize: 14
    },
    height: Ti.UI.SIZE,
    left: 50,
    right: 50,
    text: 'Message showing this table is empty\nAdd lines\nand more',
    textAlign: 'center',
    width: Ti.UI.FILL    
});
viewEmpty.add(lblEmpty);
// var data = []; // use to test an empty table
var data = [
    { title: 'One' },
    { title: 'Two' },
    { title: 'Three' }
];
var tbl = Ti.UI.createTableView({
    data: data,
    footerView: viewEmpty,
    height: Ti.UI.FILL,
    width: Ti.UI.FILL    
});
win.add(tbl);
viewEmpty.setVisible(!(data.length > 0));

// !(data.length > 0) returns true or false
// the "!" prefix tells it to provide the opposite result

    Let me know what you think.
