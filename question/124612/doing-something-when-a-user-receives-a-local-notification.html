Doing something when a user receives a local notification

Hello all,

Is there a way to trigger an event or a function when a user slides after having received a local notification on iOS and the app launches?

Local notifications on my app works well but all it do is launching the app, I'd like to be able to do things when the app is launched by a local notification and not by the user just launching it from the dashboard.

Thanks!
