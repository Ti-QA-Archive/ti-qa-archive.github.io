  • Because with Titanium, you DONT have to know Objective C (iOS) or Java (Android). Its all javascript, and for a few minor things that work on one and not the other (which you can easily detect), you oly write the code once.

    For example, look at this code on how to create a button

    –Objective C—

    //create the button
  UIButton *button = [UIButton buttonWithType:UIButtonTypeRoundedRect];

  //set the position of the button
  button.frame = CGRectMake(100, 170, 100, 30);

  //set the button's title
  [button setTitle:@"Click Me!" forState:UIControlStateNormal];


  //add the button to the view
  [self.view addSubview:button];

    –Java—

    public class MyActivity extends Activity {
     protected void onCreate(Bundle icicle) {
         super.onCreate(icicle);

         setContentView(R.layout.content_layout_id);

         final Button button = (Button) findViewById(R.id.button_id);

     }
 }

    Now, look at how you would create it for BOTH Android and iOS

    var button=Titanium.UI.createButton({
width:30,
height:30,
title:'I am a button'

});

    Which would you rather do?
