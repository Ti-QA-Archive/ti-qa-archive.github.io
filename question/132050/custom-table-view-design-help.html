  • You can create array of row objects and assign to data property of table view. Here you have to create a custom row (as you said too).

    You have to play with View, ImageView and Labels only in your designing. You can take a main background view with background image as you have shown in you mockup. Than you can fill that main background view with Image View (Main Image of your row).

    For Displaying test you can use labels. For the separator between that text you have to again use Image View.

    There is no rating bar control in titanium so again you have to manage that using images only.

    You can provide transparency to your view using opacity property.

    Cheers.
