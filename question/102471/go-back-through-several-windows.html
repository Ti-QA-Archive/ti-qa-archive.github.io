Go back through several Windows

I need to do something that seems simple, but I can't find a way to do it :

  • I have a tabGroup with several Windows.

  • I navigate from Window to Window opening them with :
    Titanium.UI.currentTab.open(win,{animated:true});

  • I can go back to the previous Window using the back button BUT I want to go several windows back.

How could I achieve that ?

EDIT:
I'm working on iOS, but my app will need to be ported to Android soon.

EDIT2:
I did something very dirty (and I'm not proud of it):

//creation of a new window identical to the first one
var winNew = Titanium.UI.createWindow({
    url:'tab2.js',
    backgroundColor:'#fff',
    barColor:'#fff'
});
Titanium.UI.currentTab.open(winNew);
//hiding the back button
winNew.setLeftNavButton(Ti.UI.createView({}));

I don't know how it will behave with memory consumption.
I had a lot of answers here telling me to keep the windows I create in an array.
This might be a solution to keep the same behavior in android.

  • One option would be to pass down a list of windows you have opened (either a name or unique identifier), then from your current window you can fire your own custom event……such as 'windowClose' and pass the id of the window you want to close. Each window will have a listener for said event and will close itself if the id provided matches the window id. You may want to do things such as setTimeout() so you aren't trying to close all of your windows at once.
