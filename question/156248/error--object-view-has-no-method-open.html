Error : object #<view> has no method 'open'

I try to create a button on click to redirect me to another view of a controller from the index file
I followed instruction and i read lot of article and i still get the same error 

object #<view> has no  method  'open'

I have in index.js as controller 

function goPlay(e) {  
     var w = Alloy.createController('play').getView();
     w.open();

}

in index.xml :

<Button id="btnPlay"  class="btn btn-play"  bottom="0">play</Button>

the play is the view trying to get it

Note i use alloy framework on windows 7 platform using android emulator

Please help

    Hi Kareem,

    You don't need to call the open() method at all. In order to get the view just needed what you have without the open() method.

    Alloy.createController(&#39;play&#39;).getView();

    • Alloy.createController(&#39;play&#39;).getView();

      will still work - it won't show the XML but it will run the controller.
      So you could still have Titanium Classic code that generates and shows a view in your controller, but that defeats the purpose of Alloy.

      To parse the controller and show the View, ensure the XML has a Window element. (as meenakshi pathak suggests)
