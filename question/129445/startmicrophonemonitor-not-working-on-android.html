startMicrophoneMonitor not working on Android?

Hi. I am trying to create a simple application, monitoring the microphone level. But when I call Titanium.Media.startMicrophoneMonitor(); I get the following error (on Android): TypeError: Cannot find function startMicrophoneMonitor in object [Ti.Media].

It seems to run fine on iOS. The documentation here seems to indicate that the function should work for Android as well.
Any ideas?

  • I have a feeling that's an iPhone only function - though not very well documented as such.

    I found the best way to do audio recording on android waas to use the built-in recorder (which has its own level indicator) using the correct INTENT

    ie : android.provider.MediaStore.RECORD_SOUND

    There's a few post around here somewhere with some demos of this I seem to remember.
