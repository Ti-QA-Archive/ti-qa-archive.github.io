Undefined is not an object error

Hi all, I'm getting an undefined object error, which I'm not sure what's causing it, because each line still returns what it is supposed to and the code continues to run without any issues.

Is there anything wrong with this code? The menu is a view, with each item being a view within it. Each item has rowid added to it.


var menulength = menu.children.length;

for (var c = 0; c <= menulength; c++) {

     // this line gives the undefined object error
     var itemid = menu.children[c].rowid;

     // this line gives no error
     menu.children[c].animate(ewm.Anim.fadeOut);

}

Both lines of code return what they're supposed to be doing.

  • There are few problems here, first is this line:

    for (var c = 0; c <= menulength; c++) {

    It should really be:

    for (var c = 0; c < menulength; c++) {

    Second problem is that in newer versions of Titanium SDK, "children" property is broken (at least for some components). Try to debug that array, you should see that it is empty array no matter how many components you add.
