  • Hi

    I have no experience with JSDOM, but thought the info below may help solve the problem.

    Have you tried YQL? This allows you to parse public (not on the device) web pages and return XML or JSON trees of the whole content or specific branches.

    For example you can get it to return just the news headlines from Yahoo as you provide a YQL (yahoo query language) syntax specify where to start and what to return. You can the very easily traverse this in your app.

    You can query any page as long as it is public.

    • YQL Console

    You call this via the built in Ti.YQL method or directly with httpClient (my preference) as you can also handle errors and time-outs.
