    • One way to know this is to create a global variable using the built-in Alloy globals object and pass the current window reference, whilst there are better methods out there - this is the easiest to explain.

      // add this each time you create and open a new window
Alloy.Globals.currentWindow = newWindow;
// newWindow = whatever name you gave the window variable

      Usage

      var win = Alloy.Globals.currentWindow;
