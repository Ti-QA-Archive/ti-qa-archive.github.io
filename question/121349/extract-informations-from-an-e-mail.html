  • For those suggesting using POP or IMAP access to google, they require you create a secure IMAP or POP3 connection, which is more complex than simply opening a socket and issuing some commands.

    .

    As for accessing recent messages, you can use the Gmail Inbox Feed API.

    .

    http://code.google.com/apis/gmail/docs/inbox_feed.html

    .

    If you're looking to just show some basic data about new messages, you can basically read the new message queue like an RSS feed. But if you want to provide access to read messages or other mail client features… it gets more complicated.

  • You can't do it with Titanium. There is no support for secure protocols.

    There is a unsafe way to do it , however. You can create a PHP webpage which can use Gmail PHP API or SSL sockets. Then you can use unsecured protocol to get the emails from the PHP page.

    So don't try to play with this.
