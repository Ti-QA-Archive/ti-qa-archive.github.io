    Not an official answer, but here's what I know: You'll need an Intel-based Mac (that's an XCode requirement). I run Titanium on an old MacBook 13 with a Core2, 2 GHz. It ran Titanium fine with 1 GB RAM, though pretty slowly. It cruises along pretty nicely with 2 GB. Pretty sure that Snow Leopard (10.6.3) or newer is required. I'm running 10.6.4.
