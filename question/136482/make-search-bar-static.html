Make search bar static?

Hi,

Does anyone know how to make the search bar not move when its above a Tableview?

When its above a tableview, and you scroll down or up, the search bar moves along with it. I would like it always there - not moving.

How would I do this?

I tried building it into a header, but for some reason it didn't work.
Thanks

    Just add the search bar separately to the window / view with the tableview below.

    var win = Ti.UI.createWindow();

var searchBar = Ti.UI.createSearchBar({
    height:40,
    top:0
});

var table = Ti.UI.createTableView({
    top:40,
    search: searchBar
});

win.add(searchBar);
win.add(table);
