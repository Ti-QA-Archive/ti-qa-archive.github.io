    • Okay, this property only change the current dot.

      But you can use a custom module, e.g.

      https://github.com/viezel/NappAppearance

      See the example there:

      pageControl: {
        currentPageIndicatorTintColor:"#CD1625",
        pageIndicatorTintColor:"#ececec"
}

      You might need to recompile the module to support 64-bit binary.

      I am using thie module myself and it works fine with latest SDK.

      — commented February 3rd 2015 by Danny Pham
