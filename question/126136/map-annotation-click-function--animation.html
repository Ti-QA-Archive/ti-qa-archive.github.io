Is it possible to animate the rows of a TableView from clicking on a map annotation? For example, I have a mapview and tableview on a window, the data is being populated from an API. Is it possible from clicking on an annotation to animate the tablerows so that the matching row to the annotation is set to the firstrow visible on the table.

var marker = Titanium.Map.createAnnotation({
