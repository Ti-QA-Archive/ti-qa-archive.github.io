  • It still needs to go through platform evaluation and testing, but there's a pull request into titanium_mobile to address this issue. It will allow you to receive binary data (i.e., images) via Android Intent Filters. All the necessary information for using this, including the source code, API documentation, and a test from the accompanying Jira ticket can be found at this link:

    https://github.com/appcelerator/titanium_mobile/pull/1198
