Multiplatform Development Howto

Hi all,

I am new to Titanium. For a project we are evaluating titanium.
I played a little bit around and have some questions.
I need to write a software which should run on Desktop, iPad and Android tablets.
When I look at creating a project in Titanium I have to decide if I am developing a desktop app or an iPad App or iPhone/Android App. Does this mean that I have to develop 3 different Versions of the same software?

Is Titanium the right decision for me?

  • The iPhone and Android app will cross-develop, there are few things that are os-specific, but check the API and it will tell you. If so, you can always do an

    if(platform="iphone")
    do this

    elseif(platform="android")
    do this

    sort of thing.

    The iPhone app will run on the iPad, but its going to be at the smaller resolution. You should design a separate app for that to take advantage of the larger screen anyway. Your code will mainly stay the same, just the design will need changing.
