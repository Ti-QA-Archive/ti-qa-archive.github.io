App crashing with require function

Well, this morning I got that kind of troubles that you say "WTF! It was working fine yesterday!"
I had my code working fine til yesterday, but now (don't know why) I cant run my app, it is giving me this error: 

[ERROR] Script Error = Parse error (unknown file)
2012-07-26 09:47:07.950 DemoFieldServices[4652:1a903] *** Terminating app due to uncaught exception of class '__NSCFDictionary'
terminate called throwing an exception

So, the first thing I did was create a new project, press "Run" and voilá it works!
But, then I did this in my app.js: 

var foo = require("ui");

and in my ui.js:

function createSomething() {
        return alert("it works!");
    };
    exports.createSomething = createSomething();

And get the same error. So am assuming that the problem is with the "require" function.
I made a search in the Q&A section, but nothing works for me.

Thanks a lot for ur help!

BTW: Am trying in the iOS simulator with Titanium 2.1.0 GA
