ACS periodic delete

Hello everyone,

I'm starting to work on an android app using the Appcelartor Cloud Services.

I'm using the Events object to store data on events my app users create.

My problem is that I need to implement somthing like a delete script that goes over the Events table every X amount of time and deletes expired events.

Does ACS allows me to do such a thing?

    You can use the REST API and write a server application that periodically cleans the old events.

    http://cloud.appcelerator.com/docs/api/v1/events/delete
