Positioning two views in one window underneath another

Im having trouble trying to get two views with dynamically loaded content to display one beneath the other.

Essentially I have a webview that renders dynamic content, and I have a service that returns comments on that content to be displayed in a tableview beneath it.

However, because the height of the webview and the height of the tableview are always changing, I can't hardcode any sizes for them. How do I get them to display nicely on one window?

  • I'd use vertical layout. for example:

    var win = Ti.UI.createWindow({
  // window settings
  layout: 'vertical'
});

var label1 = Ti.UI.createLabel({
  // label1 settings
});
win.add(label1);

var label2 = Ti.UI.createLabel({
  // label2 settings
});
win.add(label2);

    give it a try?
