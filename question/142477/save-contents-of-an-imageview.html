Save contents of an imageview

Hi,

I need to place an image on top of another image, was thinking of using an imageview and add the images to it. Can I save the contents of this imageview then?

Thus, say I have image A and image B, put image B on top of image A then save this new image to file store. Is it just a matter of having an image view, add image A and image B to it and do something like:

var f = Titanium.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory,myfilename);    
f.write(imageView);  // where imageView would be image B on top of image A

Thanks

  • What you need will be done by first using the code in the answer and then in the question.. 

    var container = Ti.UI.createView();
container.add(imageA);
container.add(imageB);
myBLOB = container.toImage(); // toImage converts a UI object to an image blob you can save this data to a file now.
var f = Titanium.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory,myfilename);   
f.write(myBLOB);

    You need to create a container view, and put both images in that view:

    var container = Ti.UI.createView();
container.add(imageA);
container.add(imageB);
myBLOB = container.toImage(); // toImage converts a UI object to an image blob you can save this data to a file now.
