Disabling cross domain security for appcelerator desktop

Hi,

Webkit defaultly has cross domain ajax security where you cannot make a request to another domain. This can be disabled.

Is there anyway to do this in titanium desktop since having this makes oauth extremely difficult/impossible?

Thanks,
Ray

    Ray,

    Without any additional configuration, Titanium Desktop allows cross domain xml http requests.

    If you run the following you should see the contents of Apple's index.html printed to your debug console.

    var req = new XMLHttpRequest();
req.open('GET', 'http://www.apple.com');
req.setRequestHeader("Cache-Control", "no-cache", false);
req.onreadystatechange = function() {
    if (req.readyState == 4 && req.status == 200) {
        Titanium.API.info(req.responseText);
    }
}
response = req.send();
