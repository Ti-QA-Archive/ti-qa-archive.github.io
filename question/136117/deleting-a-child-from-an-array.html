Deleting a child from an Array

Hello,

i create an array in such way:

var arr = [
 {name:'SS', Age:24},
 {name:'aa', Age:22},
 {name:'bb', Age:23}
];

is there any way to delete a specific row from this array ?

— asked April 27th 2012 by Wassim Ghannoum

  • Accepted Answer

    You should use splice() since it modifies the array:

    var arr = [
 {name:'SS', Age:24},
 {name:'aa', Age:22},
 {name:'bb', Age:23}
];

var rowToDelete = 1; // index starts at zero, so row 1 is {name:'aa', Age:22}
arr.splice(rowToDelete, 1);

    Splice can also insert items into the array. Check out the docs for more info: https://developer.mozilla.org/en/JavaScript/Reference/Global_Objects/Array/splice
