openPhotoGallery & showCamera never firing success/error/cancel events

Using API 1.7.2 on an HTC Incredible running aOS 2.2.

(I solved this issue as I was posting this question, but I'm phrasing it in proper Q/A entry in hopes that it will help someone else).

I ran into an issue when calling Ti.Media.openPhotoGallery and Ti.Media.showCamera where either of them would never return and fire off its success handler. Here's a a super basic version of the code that will still reproduce this error:

var win = Ti.UI.createWindow({
    navBarHidden:true

win.open();
Ti.API.info('Ti.Media.openPhotoGallery');
Ti.Media.openPhotoGallery({

    success:function(event)
    {
        Ti.API.info('success');
    },
    cancel:function()
    {
        Ti.API.info('...canceled');
    },
    error:function(error)
    {
        Ti.API.info('error');
    },
    allowImageEditing:false
});

Running this on my device results in the success function never being called. (For those curious, you can run the ddms tool in the android&#x2F;platform&#x2F;sdk&#x2F;tools folder to see the API debug statements that get fired from your device
