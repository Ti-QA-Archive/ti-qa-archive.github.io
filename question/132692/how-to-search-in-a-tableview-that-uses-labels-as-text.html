How to search in a tableview that uses labels as text?

I'm trying to make my searchbar work with a tableview that uses labels. Here's a snippet of my code:

var searchBar = Titanium.UI.createSearchBar({
    barColor:'#000000', 
    showCancel:false,
    height:40
});

searchBar.addEventListener('change', function(e)
{
    e.value; // not working
});

for (var c=0;c<arr.length;c++) 
{
    arrRows[c] = Ti.UI.createTableViewRow({height:40,backgroundColor:'#ffffff',selectedBackgroundColor:'#FFFBD0',hasChild:true}); 

    var label = Titanium.UI.createLabel({
    text:'test',
    font:{fontSize:20,fontWeight:'bold'},
    width:'auto',
    textAlign:'left',
    left:40,
    height:40
    });                

    arrRows[c].add(label);
}

var myTableView = Titanium.UI.createTableView({data:arrRows, search:searchBar});

How can I do this? How can the searchbar detect each label's text? Thanks for any advice! :)

    on each row, add an entry called filter

    row.filter = label.text;

    add filterAttribute: &#39;filter&#39; to your tableView
