1.7.5 Orientation Modes Not Working

After upgrading to the 1.7.5 mobile sdk, windows started ignoring the orientation. I want the app to stay locked in portrait, which was working before and now is not.

Here is the function I use to create heavyweight windows:

function launchWindow(page, opts) {
    var nextWindow = null;
    nextWindow = Ti.UI.createWindow({
        url : page,
        fullScreen : false,
        navBarHidden : true,
        passedData : opts,
        orientationModes : [Titanium.UI.PORTRAIT]
    });
    nextWindow.open();
}

And in the top of every window I have the following:

Titanium.UI.orientation = Titanium.UI.PORTRAIT;

Thanks in advance for any help sorting this out.

    Try moving the orientationModes assignment outside of the object creation, like this:

    function launchWindow(page, opts) {
    var nextWindow = null;
    nextWindow = Ti.UI.createWindow({
        url : page,
        fullScreen : false,
        navBarHidden : true,
        passedData : opts
    });
    nextWindow.orientationModes = [Titanium.UI.PORTRAIT];
    nextWindow.open();
}
