  • Similar to the answer proposed by Joe, I found I could emulate a border on a single side by creating a single pixel wide view with a background set.

    var borderRight = Ti.UI.createView({
    backgroundColor: '#e0e0e0',
    width: 1,
    top: 0,
    bottom: 0,
    right: 0
});

view.add(borderRight);
