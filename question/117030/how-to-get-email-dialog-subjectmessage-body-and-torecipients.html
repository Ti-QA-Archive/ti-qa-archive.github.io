  • Accepted Answer

    Hi Henry,
    getting the text entered by the user in the email dialog it's impossible. It's not a limitation coming from Titanium, but from the underlying OS features.

    Indeed, on iOS Titanium uses the MFMailComposeViewController class which only returns a result code (see here for details), while on Android it uses an intent that fires up the available email application and surely doesn't give you back the information you need.

    Regards

    var email = Ti.UI.createEmailDialog();

    Upon pressing the button 'send', your event handler should copy the values from email.messageBody, email.subject, etc.

    The problem you have is because you are trying to access an empty event object.
