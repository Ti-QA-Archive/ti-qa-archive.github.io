  • Hi Jain,

    Thanks for your concern.

    I've tested your issue and cursor color change from white to black works fine for me. Please follow the steps below.

    • Modify your tiapp.xml using the code below.
    <android xmlns:android="http://schemas.android.com/apk/res/android">
        <manifest>
            <application android:theme="@style/Theme.AppCompat.Light"/>
            <!-- Need to specify at least API level 11 for Titanium SDK 3.2.x and prior -->
            <uses-sdk android:minSdkVersion="14" android:targetSdkVersion="19"/>
        </manifest>
    </android>

    • Create a styles.xml file and a colors.xml file in [Project]/platform/android/res/values, relative to the main project folder.

    • In styles.xml, add the following code.

    <resources>

    <style name="Theme.Custom" parent="@android:style/Theme.Holo.Light">
            <item name="android:editTextStyle">@style/editText</item>
    </style>

    <style name="editText" parent="@android:style/Widget.Holo.Light.EditText">
        <item name="android:textCursorDrawable">@null</item>
        <item name="android:textColorHint">@color/cursor</item>
    </style>
</resources>
    • And in colors.xml, add the following code.
    <?xml version="1.0" encoding="utf-8"?>

<resources>
    <color name="cursor">#000</color>
</resources>

    Hope it'll help you to solve your problem.
