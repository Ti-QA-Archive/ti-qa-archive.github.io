  • You cannot do this

    Ti.UI.SIZE-240

    If the buttons are 80dp wide, I'd do something like this:

    
var wrapper = Ti.UI.createView({
    width: Ti.UI.SIZE,
    layout:'horizontal'
});

b1 = Ti.UI.createButton({width:80});
b2 = Ti.UI.createButton({width:80});
b3 = Ti.UI.createButton({width:80});


wrapper.add(b1);
wrapper.add(b2);
wrapper.add(b3);

  • Complementing the response Dan Tamas.
    You can use percentage to leave the relative position.

    var wrapper = Ti.UI.createView({
    width: Ti.UI.FILL, // < --- note 
    layout:'horizontal'
});

b1 = Ti.UI.createButton({width:'25%'});
b2 = Ti.UI.createButton({width:'50%'});
b3 = Ti.UI.createButton({width:'75%'});


wrapper.add(b1);
wrapper.add(b2);
wrapper.add(b3);
