Conditional Code in Alloy Model

Is there a way to use conditional code in an Alloy Model. Is there some way to say if mobileweb use this while ios and android apps use this other model?

Can I use this structure in a model file?

if (OS_IOS)
{
}
if(OS_ANDROID)
{
}
if(OS_MOBILEWEB)
{
}

    Sorry, no, that's not possible at least at this point in time. You'd have to load different models in the controller based on the platform:

    if(OS_IOS) {
  var book = Alloy.createModel('iosbook', {title: 'Foo'} );
  // or create a collection
  var library = Alloy.createCollection('iosbook');
} else if(OS_ANDROID) {
  var book = Alloy.createModel('androidbook', {title: 'Foo'} );
  var library = Alloy.createCollection('androidbook');
}
