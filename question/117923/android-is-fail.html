    In order to help you with this, we need to check whether your environment is configured correctly. To do this, please copy and paste the following into your terminal/shell so that all the commands execute successfully. Copy the output and paste it to your preferred pastebin service, and paste the link to it here.     clear
cd /
uname -a
echo "$PATH"
echo "$JAVA_HOME"
java -version
javac -version
python -V
echo "$ANDROID_SDK"
cd "$ANDROID_SDK"/tools
./adb version
./android list target
git --version
echo " --- The following is optional ---"
scons -v
echo "$HTTP_PROXY"
echo "$HTTPS_PROXY"
echo "$SOCKS_PROXY"

    • Hi Jeremy

      Your environment looks pretty good, but I can see a couple of improvements that we can make, as follows:

      Regarding your Android packages, can you remove these obsolete and duplicated ones (close your emulators first):

      SDK Platform Android 2.3.1 API 9...
Google APIs by Google Inc Android API 9...
GALAXY Tab Addon by Samsung Electronics Co Ltd Android API 8 (all 6 of them!)

      Then change to the virtual devices screen of the same tool, and delete all the existing AVDs.

      Ensure that you have set the $ANDROID_SDK environemnt variable to the android sdk path. On my system it is &#x2F;opt&#x2F;android-sdk, and this directory contains the tools directory. Note that on your system it may be at a different location, though.

      Then please run the script above again (note that I have edited it since last time).
