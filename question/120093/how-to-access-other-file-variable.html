  • Hi Dave,

    you can use the mechanism with event fire and event listening.

    Ti.API.fireEvent(&#39;datas_are_ready&#39;,{data:[1,2,3]});

    and in other file:

    Ti.API.addEventListener(&#39;datas_ara_ready&#39;,function(e){&#x2F;* do something with e.data*&#x2F;});

    Rainer
