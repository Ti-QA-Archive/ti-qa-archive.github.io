  • To answer my own question.

    Simply run the app from the studio and look at the second or so line in the console.

    it will say something like this

    [DEBUG] /Users/george/Library/Application Support/Titanium/mobilesdk/osx/2.0.2.GA/android/builder.py install {APP_NAME} /Users/george/Library/Developer/android-sdk-macosx /Users/george/code/titanium/{APP_NAME_AGAIN} {APP_ID} {SOME_NUMBER}

    I was missing this number which I guess is some kind of key.

    If you copy paste this in the command line the app will be installed in the device.
