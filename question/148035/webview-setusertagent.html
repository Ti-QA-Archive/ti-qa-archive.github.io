  • setUserAgent does not make the webView open in Mozilla Firefox. It is information that you can send in a web request to identify the type of browser (or any other user agent) that is sending the request.

    • A 'user agent' can be defined as a piece of software that is capable of making web requests. This usually a web browser but can be any other software, for example, curl is a program that can make web requests and retrieve responses, therefore you could call curl a user agent too.

    • I just noticed this is for Android. You can launch an Intent to start Firefox (assuming it is installed on the user's device). Give me a few mins to get you an example..

      — commented February 8th 2013 by Nick Milner 

    • var intent = Ti.Android.createIntent({
            action : Ti.Android.ACTION_VIEW,            
            data : 'www.appcelerator.com',
            className : 'com.android.browser.BrowserActivity',
            packageName : 'org.mozilla.firefox'
        });

        intent.addCategory(Ti.Android.CATEGORY_LAUNCHER);
        Ti.Android.currentActivity.startActivity(intent);
