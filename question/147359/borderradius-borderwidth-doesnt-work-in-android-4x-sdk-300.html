borderRadius borderWidth doesn't work in Android 4.x SDK 3.0.0

I tried to apply borderRadius or borderWidth to View or ImageView I don't see any changes.
It works perfectly with iOS tho.

Here's the code:

var viewBackground = Ti.UI.createView({
    top: '-2dp',
    left: '-2dp',
    right: '-2dp',
    height: '162dp',
    backgroundImage: '../images/general/background@2x.png',
    backgroundRepeat: true,
    borderWidth: '2dp',
    borderColor: 'white',
});

var imgProfile = Ti.UI.createImageView({
    width: '80dp',
    height: '80dp',
    top: '20dp',
    left: '10dp',
    borderRadius: '40dp',
    borderWidth: '4dp',
    borderColor: 'white',
    image: '../images/general/profile@2x.png',
});

    Hi Matteo,
    borderRadius and borderWidth only accept a Number not a string see the the official docs. So changing your code to this

    var viewBackground = Ti.UI.createView({
    top: '-2dp',
    left: '-2dp',
    right: '-2dp',
    height: '162dp',
    backgroundImage: '../images/general/background@2x.png',
    backgroundRepeat: true,
    borderWidth: 5,
    borderColor: 'white',
});

var imgProfile = Ti.UI.createImageView({
    width: '80dp',
    height: '80dp',
    top: '20dp',
    left: '10dp',
    borderRadius: 20,
    borderWidth: 10,
    borderColor: 'white',
    image: '../images/general/profile@2x.png',
});

    will work.
    Hope it will help you .
