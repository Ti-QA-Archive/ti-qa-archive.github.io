  • You should store your parameters into an array and pass it as the second parameter of your execute function. This will avoid weird string concatenations (also very useful when dealing with dates and numbers).

    ####Here is what it would look like:

    var params = [ 
    data[i]id,
    data[i].password,
    data[i].codigo,
    data[i].sucursal,
    data[i].administrador,
    data[i].jefezona,
    data[i].formato,
    data[i].tipo 
];
var insert = db.execute("insert into estructuras (id, password, codigo, sucursal, administrador, jefezona, formato, tipo) values (?, ?, ?, ?, ?, ?, ?, ?)", params);

    Notice that each parameter is represented by a question mark ?. They will be automatically replaced when executing your query. It is imperative that you have the same amount of question marks and parameters in your array though.
