  • If you want to build a web app, then your are here in false cinema.
    If you want to build a native app for iOS you have to use an mac or an virtual mac in net. A good idea is phonegap. With this technology you build your app on your windows, the you commit to github. The cordova builder read your code from github and send you binaries for all platforms, Awesome!
