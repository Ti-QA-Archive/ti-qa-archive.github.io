TableView data binding add model automatically updates UI

I have a TableView that uses a dataCollection to populate it. The collection is being filled with data from a remote server. Each time I make a request, it gives me an array of 20 JSON objects. So I make a loop and for each JSON Object I create a model using this code:

var myModel= Alloy.createModel('model', {
"id": jsonObject.id,
//more stuff…
});

myCollection.add(myModel);

So… I add a new model to the collection 20 times, and for each time I do that, the UI is update, making the app slow…
My question is: Is there a way to add all the 20 models at once, or manually update the UI whenever I want?

    You can use Collection.add to add multiple models at once. I think this only fires a single add event but if it not, you can suppress events by passing silent: true to Collection.add and then manually trigger a single add event in the success callback

    Backbone.js Collection.add
