  • What worked for me was uninstalling both Titanium CLI and Alloy from the command line, and then installing them from Titanium Studio. So, the commands are:

    > sudo npm remove -g titanium

    > sudo npm remove -g alloy

    Then, open Titanium Studio and go to Help -> Check for Updates. It should prompt you to install the latest Node.ACS, Alloy, and Titanium CLI. Go ahead and install from Titanium Studio. Then, close TItanium Studio and open it up again. I had to open and close Studio a couple of times before the changes took effect, although your results may vary.

    Hope this is helpful. Good luck!
