  • Chances are good there is nothing wrong with the code per se, and you just need to force a full rebuild of the project. Titanium tries to optimize the speed of the compile and process just the file(s) changed since the previous compile. Unfortunately, a side effect is if you use a new class (such as SearchBar) for the first time, the support for that module is not yet included in the XCode/Android project.

    Simply switch to the Edit tab and click "Save changes". Even if you do not edit any values, merely clicking the button should force a full rebuild and chances are good the project will work correctly.

    If you are using the iphone and it still does not work, try clearing everything in the (project)/build/iphone folder (but leave the folder there, just empty it).

    The project should now work.
