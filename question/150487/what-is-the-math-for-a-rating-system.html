    Hi Douglas,

    Following is the code will work for calculating average rate.

    var numRates = 0;
var valRates = 0; 

function rate(v) {
    numRates = numRates + 1;
    valRates = valRates + v;

    alert( 'Number of Rates: ' + numRates + '       ' + 'Rate Value: ' + ( valRates /numRates ) )
}

    Let me know, if you require more clarification.
