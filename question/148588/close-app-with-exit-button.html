Close app with exit button

Hiya,

I've made an exit button on my app window(The first window created in app.js, all windows are added to this one). I'd like to close the entire app when someone pressed the "exit" button. However, it doesn't seem to work.

Could someone please take a look: (using Titanium SDK 3.0.0, devving for Android 2.2+ and iOS)

var appWindow = Titanium.UI.createWindow({
    exitOnClose: true,
     navBarHidden: false
});
appWindow.open();


var exitBtn = Ti.UI.createButton({
    width: 100,
    height: 40,
    title: "Sluiten",
    top: 5,
    right: 5
});

exitBtn.addEventListener("click", function(e) {
    appWindow.close();
});
