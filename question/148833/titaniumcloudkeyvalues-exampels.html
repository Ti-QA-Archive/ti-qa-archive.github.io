    Hi Selvam,

    Befor you set and get the key value, you need to authenticate from ACS and then use above cloud Query

    like, 

    Cloud.Users.login({
        login : "test@gmail.com",
        password : "test"
    }, function(e) {
        if (e.success) {
            //set or get Key value
        } else {
            alert("Authentication Failed");
        }
    });
