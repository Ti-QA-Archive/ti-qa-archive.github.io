How to prevent bubbling?

I work on Titanium 3.1.3 on Windows 7 for Android.

I worked today on some code. First I created View and Label (I use 'for' loop, that is why I stash the objects in array):

myObj[buttonNames[i] + 'View'] = Ti.UI.createView({
    width: pWidth*0.8,
    height: '50dp',
    backgroundColor: '#fff',
    borderRadius: 30,
    borderWidth: 0        
});


myObj[buttonNames[i] + 'Label'] = Ti.UI.createLabel({
    text: buttonNames[i],        
    color: '#7a7f83',
    font: {
        fontSize : '25sp',
        fontFamily : myApp.helveticaNL
    }
});

And then I create listeners for the objects:

myObj[buttonNames[i] + 'View'].addEventListener('touchstart', function(e) {
    e.source.backgroundColor = '#5aba3d';
});

myObj[buttonNames[i] + 'View'].addEventListener('touchend', function(e) {
    e.source.backgroundColor = '#fff';                
});

Because of the bubling, listener will be fired on touching the Label, what i want to prevent.

I searched Web and the only advice was to use:

e.cancelBubble = true;

but it does not work.

I found also an answer 4 years old, that it is planned to add bubbling prevention soon. Did anything change in that topic?
