settings adding for android

Hi all,

There is option to show settings in iphone for app (using Settings.bundle ) , how to show setting in android like iphone, is there any way ?

    There is an equivalent on Android yes, they call it Preferences.

    What you will need to do is to create a file named preferences.xml and save it into the platform&#x2F;android&#x2F;res&#x2F;xml directory. There is a very good chance that this directory doesn't exist, so you will just have to create it yourself.

    You will then modify this newly created file to define all your settings:

    ####For example:

    <?xml version="1.0" encoding="utf-8"?>
<PreferenceScreen
    xmlns:android="http://schemas.android.com/apk/res/android"
    android:title="Preferences">

    <PreferenceCategory android:title="Social Networks">
        <CheckBoxPreference
            android:title="Facebook"
            android:defaultValue="false"
            android:summary="Your messages will be published on your Facebook wall"
            android:key="facebook_preference" />
        <CheckBoxPreference
            android:title="Twitter"
            android:defaultValue="false"
            android:summary="Your messages will be published in your Twitter feed"
            android:key="twitter_preference" />
    </PreferenceCategory>

</PreferenceScreen>

    You can have a definition of every available settings control on the Android documentation page.

    Once your settings are defined, you cn invoke the Settings window from your app using the openPreferences function.

    There is no centralised Settings application like on iOS, so it is the app's duty to open that window.

    Hope this helps.
