Make TextField take up the entire width except for a button's size.

I want to have two controls in a horizontal layout view. They are a text field and a button. How do I right justify the button and make the textField take up the rest of the width? I want this to work on more than one screen size obviously so I can't hardcode a width in pixels. Anyone know how this can be achieved?

  • a horizontal layout is just that.

    Take a standard view, no layout specified, make it the width of the device.
    Add your button to the view with a right: 0 adding any padding you need. Width of the TextField would be width of device - width.of.button = padding.required. textfield left:0 + padding

  • Hi

    Do not use layout - not helpful for this.

    var txt = Ti.UI.createTextField({
    height: 50,
    left: 10,
    right: 100,
    width: Ti.UI.FILL
});
var btn = Ti.UI.createButton({
    height: 50,
    right: 10,
    width: 80
});

    The button here is 80 wide and is pushed from the right side by a 10 margin. The text field has a left 10 margin and a right margin of 100 ( which equals the buttons 10 padding the width of 80 plus another 10 padding to the left of the button).

    The magic is the text field and the width of Ti.UI.FILL, this works WITH the left and right if provided to subjtract space.

    So this means the textField will take up any extra space that is available.

  • using layout in a view can make your code complex but still solution for your problem is

    //-------------view is having horizontal layout
var textField = Ti.UI.createtextField({
left:0,
height:30 , //-----------or you can declare it in some percentage say : 10%
width:'60%'     //---------again in % if you are using multiple screens
});
var btn = Ti.UI.createButton({
height: 30 ,//-------------can be in %
width: 'auto' //----------it will make button to occupy remaining space , you can use `TI.UI.FILL` too
});
view.add(textField);
view.add(btn);
