    • Make sure you have the latest Titanium CLI and that your active SDK is 3.1.1.GA. You can do this by running the following:

      sudo npm install -g titanium
ti sdk select 3.1.1.GA

      If that last command fails, then run: ti sdk install –default

      Have the <sdk-version> in your tiapp.xml set to CI release, then run a build from your project directory.
