Ti.Include - Can't use $.AlloyObject anymore ?

At index.js, I can do something like this: $.MyLabel.visible = false
So, I placed on Index.js: Ti.include(''Common/startup.js');
Now I can't do something like $.MyLabel.visible = false
It's returns: Can't find Variable: $ at startup.js

Any light for me ?

  • 1- You are structuring your code the wrong way.
    2- Try using the commonJS functions to return a value and use it in the controller, instead of trying to update UI elements directly from these modules as you are trying to do.
    3- Don't use Ti.include inside commonJS modules because it may show strange behavior. Instead of that wrap the js file into a commonJS module and require it inside the parent commonJS module or controller.
