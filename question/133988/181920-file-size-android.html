  • After 1.8.x, there are two javascript engines being packaged, rhino and V8 (if you choose V8 in TiStudio) The V8 is faster, but have to support those older devices, so the second is included.

    Your decision not to use TiStudio is a personal choice. As for being "stuck" with developer, you have the option to use your own editor and build your packages via command line.
