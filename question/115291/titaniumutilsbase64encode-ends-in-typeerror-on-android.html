Titanium.Utils.base64encode(...) ends in TypeError on Android

I'm trying to call a https url and provide username and password with basic authentication. I found an example as below. But calling the base64encode function ends in following error:

TypeError: Cannot call method &quot;base64encode&quot; of undefined (app:&#x2F;&#x2F;app.js#77)

var xhr = Titanium.Network.createHTTPClient();
xhr.open('GET','https://www.myurl.ch/index.html');
var authorization = 'Basic ' +
    Titanium.Utils.base64encode(username + ':' + password);
xhr.setRequestHeader("Authorization", authorization);
xhr.send();

Similiar problem I had with creating the http client where I forgot to add Android permission for internet access. Is this one the same problem? Do I have to add something in tiap.xml? Where can I find the required settings for specific Titanium Modules?
