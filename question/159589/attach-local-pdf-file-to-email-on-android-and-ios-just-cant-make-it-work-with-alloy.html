    var f = Ti.Filesystem.getFile('vinschema.pdf');

  • Hi

    • You should use the forward slash on BOTH platforms.
    • You should not need to place the file in two different folders.
    • The filename MUST match the case of the physical file, so make sure all letters are lowercase on the physical file.

    For Alloy the file would need to be located here for your code to work;

    {appname}/app/assets/vinschema.pdf

    Take a look in resources folder when you have compiled, does the file NOW appear in the following location;

    {appname}/resources/vinschema.pdf

    If not then the actual compiled code will not see it either. (Remember this is only for confirmation - do not place anything in here manually when using Alloy)
