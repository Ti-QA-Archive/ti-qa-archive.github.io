How can I change the cursor to a hand when it's hovering over a view for MobileWeb?

I tried specifying it in a JSS file and adding an id to the parameter list of Ti.UI.createView. See below:

us.js file:

var view = Ti.UI.createView({
    backgroundColor:'#336699',
    borderRadius:10,
    width:300,
    height:400,
    top:0,
    id : 'handPointer'
});

ui.jss file:

#handPointer {
color: #9f9696;
margin-bottom: 5px;
cursor:pointer;
}

  • There is no API for setting the mouse cursor. Titanium Mobile Web is designed to run on mobile browsers. Desktop browsers are not supported, though many of them do work.
