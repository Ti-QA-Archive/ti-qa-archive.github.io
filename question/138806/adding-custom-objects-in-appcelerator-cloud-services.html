Adding custom objects in Appcelerator Cloud Services

Hello,

I am trying to create a custom object in cloud from my mobile application using following code. However I keep getting error-"You need to sign in or sign up before continuing". I have specified the authentication keys in my tiapp.xml as show below but I dont seem to get past through this error. 

Cloud.Objects.create({
            classname : 'feralobservations',
            fields : {
                latitude : _feralObservation.latitude,
                longitude : _feralObservation.longitude,
                date : _feralObservation.date,
                name : _feralObservation.name,
                documented : _feralObservation.documented,
                hogsSeen : _feralObservation.hogsSeen,
                actionTaken : _feralObservation.actionTaken,
                numberOfHogsSeen : _feralObservation.numberOfHogs
            }
        }, function(e) {
            if (e.success) {
                _feralObservation.id = e.feralObservations[0].id;
                alert(_feralObservation.id);
            } else {
                alert('Error:\\n' + ((e.error && e.message) || JSON.stringify(e)));
            }
        });

Here is my tiapp.XML:

<ti:app xmlns:ti="http://ti.appcelerator.org">
    <property name="acs-api-key" type="string">zzzzzzzz</property>
    <property name="acs-oauth-key" type="string">rrrrrrrrr</property>
    <property name="acs-oauth-secret" type="string">ttttttttttt</property>

Any help is highly appreciated!
