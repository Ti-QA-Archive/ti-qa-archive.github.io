  • Firman

    You will see the "INSTALL_FAILED_MISSING_SHARED_LIBRARY" error when the Android libraries are not available, either because they are not installed on your machine, not selected for the Android Virtual Device (emulator) or not available on your device. For information about the enhanced Google APIs, that includes support for maps, see Android SDK / Target Android Platform (in yellow info box).

    To resolve this:

    • ensure that you have installed, at a minimum, the packages shown in the screenshot in Summary of Installed Android Packages
    • test your application in an emulator (AVD). Ensure that the AVD is running the "Google APIs Android 2.2" SDK, by selecting it in "Run Configurations"

    Note that if your device does not support the enhanced APIs, then you will not be able to run any applications that utilizes the extended functionality.

    Hope this helps
