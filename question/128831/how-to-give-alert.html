  • your alert sentence is ok, the rest of code is not. With titanium, you do not have to use the DOM, so documentis not available.

    Try this and take a look to the Titanium documentation and KitchenSink sample.

    Put this code in you app.js file.

    var win = Ti.UI.createWindow();

var btn = Ti.UI.createButton({top:10, left:10, height:40, width:100, title:'click!'});

win.add(btn);

btn.addEventListener('click', function(){
    alert('working!');
});

win.open();
