Android Version Numbers

Hi Folks,

Im running Titanium 1.2.1 on a Mac (10.6) and am using the latest 1.3 SDK! Im trying to resubmit my app to the Android market place but am having problems. Ive increased the number from 1.0 to 1.1 in the Edit screen, but i understand this isnt reflected for Android version numbering yet.

So i edited the AndroidManifest.xml file with new version numbers, but it appears that when building the app, it overwrites this file!

This seems to be why i cant upload a new app! :(
Anyone got any ideas of a solution to this?

Any help would be greatlyyyyyyyy!!! apreciated!

Scotty
