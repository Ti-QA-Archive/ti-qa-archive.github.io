  • Your server should be able to use a script to detect the IP address where the request is coming from

    eg: in PHP: $_SERVER['REMOTE_ADDR']

    You can get various properties about a device from here : http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.Platform-module

    Be warned that you should be transparent about the data you are retrieving from a user with your app.
