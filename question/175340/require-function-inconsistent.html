    1. Nowhere does it state you can require your own commonJS modules from the assets folder. That only…
      Contains image assets and other files that need to be copied into the Resources directory.
Reference these files in the code without the 'app/assets' path.
    2. The place to put your commonJS files is the app/lib folder and then you reference them without a path or a .js extension
    var mod = require('mymod')

    for a file called mymod.js placed in the app/lib folder
