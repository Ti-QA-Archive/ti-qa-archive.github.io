    This is called the up button (Here's an explanation of the back vs. up navigation patterns: Navigation with Back and Up).

    To achieve this in Titanium, you can use the ActionBar API like this:

    actionBar.setDisplayHomeAsUp(true);
actionBar.onHomeIconItemSelected = function() {
    Ti.Android.currentActivity.finish();
};

    Examples

    If you are on Alloy, check this out: Alloy Action Jackson.

    If you are on Ti Classic, go here: ActionBarSampleTi
