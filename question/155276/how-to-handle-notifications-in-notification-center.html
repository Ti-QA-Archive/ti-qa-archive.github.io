  • Hi

    The badge value should have no effect on the notification centre.

    Push notifications appear in the notification centre based on the rules and choices the user makes, this is not something you choose for them. The badge value changes the app badge display not the notification display in any way.

  • Here is scenario.

    1. Having 2 push notification send. AppBadge = 2 and I can see 2 notification in notification centre.
    2. I click on first notification and it opens my App and show me the notification.

    3.A) Using Ti.UI.iPhone.setAppBadge(null);, will set AppBadge to 0 and all notifications in notification will disappear so, I cant check the second notification.

    B) Using Ti.UI.iPhone.setAppBadge(Ti.UI.iPhone.getAppBadge()-1);, will set proper AppBadge to X-1, but the notification is still in notification centre, so I still see 2 notifications in notification centre. And they are still there until AppBadge is 0. Then they all disappear.
