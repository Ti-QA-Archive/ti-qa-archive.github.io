Save UUID in local directory for later use

Hi,

My app needs to make use of the UUID generated by the API - Titanium.Platform.id, in order to recognize a device. Problem with this API is, once the app is re-installed, the UUID changes. But I need to get the same UUID that was generated first time.
What I am thinking is, if I can save the UUID in a file in any local directory of the device (Intel PC), then I can retrieve it later upon app re-installation.
But since its a web application, I am not sure how to read/write files (.txt) in the local directory of PC from Titanium (say C drive/desktop/My documents). Any help will be much appreciated.

PS: I do not want to store this file in resources/app data directory, since these will be deleted once the app is uninstalled.

Thanks,
Abhisek
