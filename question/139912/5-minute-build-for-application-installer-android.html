  • Mike,

    First and the foremost, reinstalling Titanium or Android SDK will surely not help in this case especially for Android SDK as no matter you install it through Titanium or install independently and provide path in Titanium, its both the same.

    What i feel is that due to previous compilation files it is taking more time to run your app in tablet as sometimes it takes more time doing cleanup though that does not gets shown in the log. I would suggest the following approach which i myself followed to get rid of the situation:

    ####1. In Titanium Studio go to Projects -> Clean… select your project and press OK and then run your app

    ####2. Uninstall your app in tablet first and then run your app as sometimes android takes time shutting down the app and uninstalling it which directly increases the app execution time in titanium

    ####3. If 1 & 2 does not work then universal rule …. Restart Your Machine

    Also, as point to note, if in addition to running your app on tablet you want to run your app on emulator, do ensure to run emulator in advance before running your app in titanium and do not close emulator in successive runs as this saves a lot of time for app execution
