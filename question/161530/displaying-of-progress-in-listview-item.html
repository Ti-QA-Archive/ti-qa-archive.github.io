Displaying of "progress" in listview item

Hi.

in a radio app I try to display a "progress" on android.

First try:

childTemplates[{
    type : 'Ti.UI.ProgressBar',
    bindId : progress,
    properties : {
        min:0,max:0,width:Ti.UI.FILL,height:30
    }
}]

and in listitem:

items.push({
    properties : {},
    progress : {value :Math.random()} 
})

I see the progressbar, but the bar shows 0 (empty bar)

Second try:

childTemplates[{
    type : 'Ti.UI.View',
    bindId : progress,
    properties : {
        left:0,backgroundColor:'black',height:30
    }
}]

and in listitem:

items.push({
    properties : {},
    progress : { left : Math.random()*100} 
})

Also in this case the view is empty.

  • ProgressBar works fine for me in ListView on iOS and Android.

    childTemplates[{
    type : 'Ti.UI.ProgressBar',
    bindId : progress,
    properties : {
        min:0,max:0,width:Ti.UI.FILL,height:30
    }
}]

    bindId should be a string, so &#39;progress&#39; instead of progress.

    min:0 with max:0 looks like an error too.
