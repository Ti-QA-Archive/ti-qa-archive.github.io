Unable to view result

Hi

I have a php file which returns JSON data. I am unable to view the data in my app. The following is my code

var window = Ti.UI.currentWindow;

var xhr = Titanium.Network.createHTTPClient({
    onload: function() {

    var gpn = JSON.parse(this.responseText);

    for(var i=0;i<gpn.length;i++)
    {

    Titanium.API.info(gpn[i]);

      }
 }
});

xhr.open('POST','http://192.168.0.111/mobileapp/viewgp.php'); 
window.open();

I have already studied http://docs.appcelerator.com/titanium/3.0/#!/guide/Working_with_JSON_Data-section-29004905_WorkingwithJSONData-ReceivingandparsingJSONdata

  • Step through your code in the debugger. Put a breakpoint on this line: 

     var gpn = JSON.parse(this.responseText);

    Make sure that your breakpoint is hit; if not, then there's an HTTP error.

    If it does get hit, inspect this.responseText to see what it contains – does it look like valid JSON?

    If it does, step over the line and see what is in gpn. Maybe it's not an array, but an object instead.
