focus event of window in android world

Hi,
I try to catch the focus event. On iOS it works fine, but not in Android:

var win1 = Ti.UI.createWindow({
    title : "A",
    backgroundColor : '#123456'
});
win1.addEventListener('focus', function() {
    Ti.API.log('win1 focused');
});

In docu I found: 'The listener for this event must be defined before this window is opened.'
But in my code the addEventListener is before opening …
