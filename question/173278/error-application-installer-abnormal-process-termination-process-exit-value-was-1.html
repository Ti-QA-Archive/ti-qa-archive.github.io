  • Hello James,
    two things here:

    1. As you can see in the guide doc, windows XP is not supported anymore:

    http://docs.appcelerator.com/titanium/latest/#!/guide/Titanium_Compatibility_Matrix

    So it may randomly fail in Windows XP. You will need to update to Win 7 or 8 in order to run into a supported environment.

    1. The other bug, is more related to another issue, listed here:

    https://jira.appcelerator.org/browse/TIMOB-16073

    Can you please get the fix from the zip in the bug and then apply it to your environment?

    Let us know if we can help more related to this.

      Per your recommendation, I have re-installed Appcelerator on Win 8.1. The installation was successful! The following steps worked on my laptop:

      1. Install all prerequisites: Java SDK 1.6, Node.js v0.10.26
      2. Create new System variables: JAVA_HOME = C:\Program Files (x86)\Java\jdk1.6.0_45
      3. Update PATH: %JAVA_HOME%\bin
      4. PATH added automatically via Node.js installation: C:\Program Files (x86)\nodejs\
      5. Download Android SDK Tools(installer_r22.6.2-windows.exe)
      6. Install Android SDK Tools to C:\android-sdk
      7. Install Android SDK packages Android 4.4.2 and 2.3.3
      8. Exit Android SDK Manager
      9. Create new System variables: ANDROID_SDK = C:\android-sdk
      10. Update PATH: %ANDROID_SDK%\tools; %ANDROID_SDK%\platform-tools;
      11. REBOOT Laptop
      12. Install Titanium Studio: Click "Next" until installation is completed. It will install latest version of Java JDK and Git. The system variable for Java will not be affected. Git variable will be automatically added to PATH
      13. Open Titanium Studio. Titanium Update window will pop-up. Install latest Titanium Update.
      14. Restart Titanium Studio
      15. Go to Window > Preferences > Platforms > Android. Update Android SDK Home to "C:\android-sdk". Galaxy Nexus was the default Emulator after applying the changes. So far so good, I did not received the "[Android SDK Home] No Android SDKs were found" message
      16. Restart Titanium Studio. Click on Dashboard icon. Scroll down to "Configure Native SDKs". Android SDK icon is green!
      17. Using the command "ti info" on terminal shows that I have Android SDK and Platforms were installed!
