  • What if instead of using actual windows for your other content, you used views? You could hide and show those views accordingly?

    • Agreed… You shouldn't have more than one active Window in an application, on any phone OS.

    • I had the same issue as you Daniel. I designed and app an wrongly tried to open multiple windows at the same time. At a certain point, It became difficult or even impossible to correctly manage the layouts. As suggested by Shannon, you should open views inside the windows. As mentioned by Aaron, that does not require so much work. Of course, you'll have to adjust some properties when switching to views but nothing which would require rewriting the whole app.
