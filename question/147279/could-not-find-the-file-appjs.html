  • Accepted Answer

    This error can happen a couple of ways. The most common way is you have a JavaScript syntax error in one of your JavaScript files. Every single .js file in your project's Resources directory (including subdirectories) must be syntactically correct, even if the file is not used in your app.

    I recommend you use the CLI to build your app since it will show you the errors.

    Let's start with making sure you have the correct SDK activated. Run:

    titanium sdk

    From the list, find the 3.0.2 CI build you downloaded. If it doesn't say [active] next it it, run:

    titanium sdk select 3.0.2.<timestamp>

    Next, change into your project's directory and run:

    titanium build -p ios --log-level trace

    If you have any JavaScript errors, they'll show up and the build will fail.

    On a side note, I recommend increasing the log level to "trace" so that you don't have to keep remembering to add it:

    titanium config cli.logLevel trace
