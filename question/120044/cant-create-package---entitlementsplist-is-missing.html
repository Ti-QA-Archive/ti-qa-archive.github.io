  • I know this post is pretty old, but just had a very similar problem trying to run on an iPad device. On the buid.log, it was saying:

    ** BUILD FAILED **

    ProcessProductPackaging Resources/Entitlements.plist build/(rest_of_the_path)/project_name_.xcent
