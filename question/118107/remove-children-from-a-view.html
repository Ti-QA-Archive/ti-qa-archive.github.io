remove children from a view

hi all,
i have to clear a view from all inside elements. Now i use this piece of code

if(viewName.children != null && viewName.children.length > 0){
     var dim =  viewName.children.length-1;
     for (var d = dim; d >= 0; d--) {
    viewName.remove(viewName.children[d]);
     }
}

But when i try to execute it, the console gives me that error:
[NSMutableArray objectAtIndex:]: index 8 beyond bounds [0 .. 7]

(the index is not always 8, but change randomly)

Has anyone solved that problem? Is there another way to do that?

thx

  • This would probably be better for you, otherwise the count and the array may miss each other and some children will not be removed.

    for (var d = viewName.children.length-1;d>=0;d--)
        {
            try
            {
                viewName.remove(viewName.children[d]);
            }
            catch(e){}
        }
