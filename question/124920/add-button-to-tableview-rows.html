    Again, in this case you are calling the last button only. In the previous for loop,when you were adding it, the last reference to buttonrs is the last button.

    try something like this

    1st for:

        for(var tuhg=1;tuhg<data.length;tuhg++)
    {
    buttonrs = Ti.UI.createButton({
        style:Titanium.UI.iPhone.SystemButton.EDIT,
        left:600,
        clickName:'cfgz'
    });

    var rou=tableview.data[0].rows[tuhg];
    rou.add(buttonrs);

    rou.the_button = buttonrs;
    }

    then, on remove, in the 2nd for

    rom.remove(rom.the_button);
