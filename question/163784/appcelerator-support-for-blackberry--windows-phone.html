Appcelerator support for BlackBerry & Windows Phone

Hi,

I wanted to check if Titanium Appclerator has support for the following 2 platforms:

  1. BlackBerry: I see that the SDK documentation mentions this but wants to check if this is still in beta stage or is now officially released and supported by Titanium.

  2. Windows Phone: Is there anyway we can download the same Appcelerator code on a Windows Phone and run it ?

Thanks.

    1. Yes, Blackberry 10 is supported on the current Titanium platform

    2. No, Windows Phone is not supported on the current Titanium platform. However it will be supported on the (under development) Hyperloop based platform. (See here)
