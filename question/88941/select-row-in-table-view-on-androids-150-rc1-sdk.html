Select row in table view on androids (1.5.0 RC1 SDK)

I'm struggling with the row selection in an android application. So I looked to the kitchen sink samples. Same behavior in the table view layout 2 sample: the rows only gets selected if you don't touch an element of the table row. If you touch an element you get the correct row-index in the click event but the row don't get the selected state. Also the labels don't get the highlighted color (this is working on iPads). Unfortunately the function selectRow(index) is not working! So there is no possibility to mimic the behavior by code.

Any ideas?
