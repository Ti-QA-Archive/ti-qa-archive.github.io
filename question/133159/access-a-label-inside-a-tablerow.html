Access a label inside a tablerow

Titanium Mobile SDK 1.6.2
How I can read properties from a label inside a table row? I try this:

table.addEventListener('click', function(e)
{
    if (e.rowData.test)
    {
        var win = Titanium.UI.createWindow({
            url:e.rowData.test,
            title:e.rowData.title
        });
        Ti.API.info(e.rowData.titlelabel.title);
    }
});

But not work. Titlelabel is a label inside a row in the table, and title is the propery of label that I need.

    If you want to access any children of table view row than you can use children property of particular row.

    Check here and here

    You can access row's children like this,

    e.row.children

    For example if your label is first child of your table view row than you can access it some thing like this,

    e.row.children[0].title // To access title property of first children of your table view row.
