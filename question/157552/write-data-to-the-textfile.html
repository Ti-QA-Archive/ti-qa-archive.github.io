    • you can use Titanium.App.Properties.setList and Titanium.App.Properties.setObject as u want

      Sample : 

      var globalArray = [
        {title:'foo',value:'yes'},
        {title:'bar',value:'no'}
]; 

Titanium.App.Properties.setList('globalArrayProperty',globalArray);

//Get value From List : 
var arr = Titanium.App.Properties.getList('globalArrayProperty');
Titanium.API.info('first value with arr: ' + arr[0].value);

      Hope it helps.
