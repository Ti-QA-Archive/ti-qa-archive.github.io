  • Hi

    Try creating a new default Alloy project - do not change and code and try this. This is a test to see if the problem is your code or your development environment.

    Muhammads suggestion was a good one and it does usually resolve things like this. I am going to assume the problem is new and up until now your app worked fine. If this is the case then a few other simple things to try;

    • Reboot your computer and try again - oldie but a goody.
    • Roll back your code to before the error started, either using a back-up or comment out the most recent code block
