How to I position/change size of buttons for different screen sizes?

I have created a window which I launch from my main app screen/window.

I have 4 buttons on it which appear in the correct positions on on the Android emulator, but when I copy my program to my android phone the buttons do not reposition themselves accordingly to accommodate the slightly larger screen on my phone.

I have created my buttons like so:

var b1 = Titanium.UI.createButton({
color:'#fff',
backgroundImage:'../images/BUTT_grn_off.png',
backgroundSelectedImage:'../images/BUTT_grn_on.png',
backgroundDisabledImage: '../images/BUTT_drk_off.png',
font:{fontSize:24,fontWeight:'bold'},
title:'SQUADS',
height:57,
width:301,
top:20
});

the next button (b2) is positioned: top:100 etc.

How do I make sure my apps buttons/labels etc reposition themselves in proportion to the device screen size

  • Instead of using width, use left and right properties:

    var b1 = Titanium.UI.createButton({ color:'#fff', backgroundImage:'../images/BUTT_grn_off.png', backgroundSelectedImage:'../images/BUTT_grn_on.png', backgroundDisabledImage: '../images/BUTT_drk_off.png', font:{fontSize:24,fontWeight:'bold'}, title:'SQUADS', height:57, left:10, right:10, top:20 });

    this will make sure the button stretches given its constraints, no matter the resolution.

    same goes for top+bottom instead of height.
