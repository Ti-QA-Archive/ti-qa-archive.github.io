Ti.nfc License violation detected issue

Hi all,
I'm using Ti.nfc module but I'm getting a "Licence violation" error.
I've downloaded the module from:
https://github.com/appcelerator-modules/ti.nfc/blob/master/android/dist/ti.nfc-android-1.2.1.zip
and installed in Studio via Help > Install Mobile Module
Titanium SDK: 3.1.0.GA
Testing on Android 2.3.6 device
Host OS: MAC Lion 10.8.3
Titanium Studio, build: 3.1.0.201304151600

TEST CASE:

  • Download ti.nfc-android-1.2.1.zip from link above
  • Install it in Titanium SDK via Help > Install Mobile Module
  • Create a new Titanium Mobile Project
  • In tiapp.xml add:
    <modules>
      <module platform="android" version="1.2.1">ti.nfc</module>
    </modules>
  • in app.js delete everything and just put:
    var nfc = require('ti.nfc');
  • Run

After few seconds you will get an Alert with "License Violation detected".
From DDMS log I get:
05-15 14:16:26.469: E/TiVerify(13500): (Timer-1) [620,5620] License violation detected. Unlicensed module(s) detected. You must be a subscriber to use one or more of the modules included with this project.. Please contact Appcelerator Support.

Can anyone help?
Thank you in advance.

    • Apologies for the slow follow-up. According to our Marketplace support team, the module is working for some and not for others. The trick is that if you get the license violation:

      1. Remove it from your project and uninstall from your system (delete the directory)
      2. Remove it from your "Purchased Products" page in the Marketplace
      3. Re-purchase and download it (I know, it's free, but the buttons still say purchase)
      4. Reinstall the module to your system and project
      5. Do a clean-build of your project.

      I hope that helps.

  • NFC is included with Ti 3.1. You don't need to download it from the GitHub repo or Marketplace. If you're going to use it with 3.0.2, you would need to download it from the Marketplace (that will associate it with your developer account and prevent that license violation error). For Ti3.1, all you need to do is require in the module. See the NFC docs page for samples and how-to info
