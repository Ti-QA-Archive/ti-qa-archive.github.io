An Android SDK is missing. Titanium requires Android platform 2.3.3 or above.

Hello. I try to use Titanium in my computer windows 7. But I have this problem: Android configuration details. One or more pieces are missing from the Android SDK. It may be that the Android SDK is already installed and Titanium Studio cannot locate the directory, or it may be that some additional components need to be installed. Items required: •An Android SDK is missing. Titanium requires Android platform: 2.3.3 or above. I installed already many times SDK 2.3.3 and above, but my SDK continues "red". At my SDK Manager I can see that they are instaled, but Titanium dont find it. How exactely can Titanium cate the directory? How can I solution this problem? Please help me, than I try to solution this since one week, Thanks very much.

  • You will find .titanium directory by default created by titanium studio inside your home directory (In my case I am doing cd~/.titanium from terminal). Inside it you will find config.json file which contain all configurations like path to your android sdk etc. Check the path whether it is correct or not. If it is correct and still you studio unable to detect it. Delete the .titanium directory and restart the studio. Titanium will create it fresh one for you.

    Hope this works for you.
