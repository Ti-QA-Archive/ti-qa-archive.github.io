How to install Titanium Module in android on Windows Platform

  • AppData is a hidden folder. You have to change folders view option to show hidden files and folders .

    Module path for windows7 is

    C:\Users\'your-username'\AppData\Roaming\Titanium\modules

//in my case

C:\Users\madnan\AppData\Roaming\Titanium\modules

    you can put your android modules to

    C:\Users\madnan\AppData\Roaming\Titanium\modules\commonjs
or
C:\Users\madnan\AppData\Roaming\Titanium\modules\android

    You have to change tiapp.xml accordingly

    // if you have placed android module under commonjs then use
 <modules>
        <module platform="commonjs" version="2.3.0">ti.cloud</module>
  </modules>

//if you have placed android module under android then use

 <modules>
        <module platform="android" version="2.3.0">ti.cloud</module>
   </modules>

    You can access your module like

    var cloud= require('ti.cloud');
Titanium.API.info(cloud);
