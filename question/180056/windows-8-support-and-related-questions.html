Windows 8 support and related questions

Hi All

Upon reading this:
http://docs.appcelerator.com/titanium/3.0/#!/guide/Getting_Started_with_the_Windows_Phone_SDK

That under "Compatibility and Download" it states that:
Windows Phone 8 SDK
Microsoft only supports Windows Phone 8 development on systems running 64-bit Windows 8. The Pro edition or higher is required to run the Windows Phone simulator. Titanium does not support Windows 8.1.

In speaking with a colleague from MS they said this:
As for Windows Phone & Windows 8, I can only suggest that you target 8.1 as that is the most broadly adopted release of both Windows Phone and Windows 8.

Does anyone know if Titanium will work on 8.1 now or still limited to 8

  • Hello,

    you will be able to create apps for Windows Phones with 8.1, and will be able to run Appc Studio in Windows 8.1.

    The only part not integrated is the uploading with the store. But you can open the project in your Visual Studio 2013 Community edition and will be able to publish.

    I recomment to read this post:

    http://www.appcelerator.com/blog/2014/12/windows-platform-preview/

    Which is the latest preview.

    Just remember: This is a technical preview, i.e., It's a work in progress. Therefore, your mileage will vary.
