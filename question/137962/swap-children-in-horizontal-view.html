Swap children in horizontal view

Hi,
I have a horizontal view and I need to swap the children so that two elements take each other's original position. Now myView.children is read-only , is there some way in which the following can be done?

var temp = myView.children[1] ;
myView.children[1] = myView.children[2];
myView.children[2] = temp;

    (Unfortunately) "zIndex" property can be used to set component's position when "horizontal" or "vertical" layout is used.

    So, when adding child components, set their "zIndex". When you want to swap 2 child elements, swap their zIndexes. ;)
