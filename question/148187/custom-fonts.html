    • I have successfully managed to get custom fonts working on Android. I downloaded your font and I also chose two of my one and added three labels onto the main window, each one with a different font.

      I created a folder called fonts in the resources folder;

      /{appname}/resources/fonts/

      I added each of my fonts into this folder, each of them being a *ttf file.

      I closed the simulator and then used the menu Project > Clean, selected my project and run the clean action.

      I then compiled my app again and it started up using the custom fonts exactly as I hoped it would.

      The label code was obviously very simply;

      var lbl = Ti.UI.createLabel({
    color: '#000',
    font: {
        fontFamily: 'Ubuntu-Title',
        fontSize: 14
    },
    height: Ti.UI.SIZE,
    text: 'Insert your name',
    top: 10,
    width: Ti.UI.SIZE
});
win.add(lbl);

      Obviously the important part is fontFamily: 'Ubuntu-Title',, the name is simply derived from the filename and subtracting the file extension.

      So 

      Ubuntu-Title.ttf

      Becomes

      Ubuntu-Title

      The file is located as follows;

      /{appname}/resources/fonts/Ubuntu-Title.ttf

      No other configuration was required in any other files.

      Works a treat.
