    • i also fell like this.. but in some specific scenario like if i use layout property of parent view and try to set controller using bottom property.. but in normal case it's works as much as expected…

      but alternatively you can use fix height width in "dp". it's working fine in all devices.. also you can calculate it depends on one fix resolution. for example let's consider iPhone4(320x480) one screen layout is working fine and you can calculate according to that..

      e.g

      Alloy.Globals.GetWidth = function(value) {
    var temp = (value * 100) / 320;
    return parseInt((Alloy.Globals.platformWidth * temp) / 100);
};
Alloy.Globals.GetHeight = function(value) {
    var temp = (value * 100) / 480;
    return parseInt((Alloy.Globals.platformHeight * temp) / 100);
};
