Animate View Height, BUT Also Auto Resize Conent

I Created a View and it contains some content in the form of other views and labels.

We want to compress the View vertically (e.g. Animate the height smaller).

When we do this the content of the View just gets cut off.

Is there a way to have everything inside the view resize like in HTML when you use fluid layout?

    Hi

    Matt defiantly you could do this in titanium you use Ti.UI.SIZE like this

    var label1 = Ti.UI.createLabel({
  color: '#900',
  font: { fontSize:48 },
  shadowColor: '#aaa',
  shadowOffset: {x:5, y:5},
  text: 'A simple label',
  textAlign: Ti.UI.TEXT_ALIGNMENT_CENTER,
  top: 30,
  width: Ti.UI.SIZE, height: Ti.UI.SIZE
});

    Here you could see height an width change size with UI
