  • Duc

    Do you mean when a row is selected? If so, the backgroundSelectedColor or selectedBackgroundColor properties of Ti.UI.TableViewRow should do it, although the former only works on Android according to the APIdocs.

    var thisRow = Ti.UI.createTableViewRow({
//  backgroundSelectedColor:"red", // android only
  selectedBackgroundColor:"green",
  className: "grid",
  height: 100
});

    Cheers

  • TableView can have a bg colour (example is navy blue)

    var tableView = Titanium.UI.createTableView({data:array, style:Titanium.UI.iPhone.TableViewStyle.GROUPED,backgroundColor:'#13386c'});

    And so can rows (background for each row is white)

    var fnRow = Titanium.UI.createTableViewRow({height:46, top:46, className:'fnRow',backgroundColor:'#fff'});

    Works for me
