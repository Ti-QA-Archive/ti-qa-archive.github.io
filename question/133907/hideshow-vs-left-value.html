hide/show vs left value

Hi All, I've got an interesting performance-related question. Using .hide() and .show() on a view, which itself contains quite a lot of information gives me a little bit of a slower response to just left-positioning the view off the screen.

So this:


function hide(){
   view.left = Ti.Platform.displayCaps.platformWidth;
}

function show(){
   view.left = 0;
}

runs faster than this:


function hide(){
   view.hide();
}

function show(){
   view.show();
}

Is there any reason I shouldn't be using left value? Is this ok to do?

Cheers!

  • The view will still take up memory, so whatever feels right. If you are reusing the view often this would do, but if you are clearing out the view, you should consider clearing the memory aswell. Could be done like this:

    function hide() {
    view.left = Ti.Platform.displayCaps.platformWidth;
    window.remove(view);
    view = null;
}
