Get an image from a graph

Hi,
I want to set a graph in my application and I understood that the way to do it is by using the canvas or the HTML5 graphs lib within a webview.

BUT, I will also have to add this graph in an email and adding HTML5 code in email won't work…
So, what I have in mind is somehow make an image file from this graph and attach this image to the email.

My question is how can I make such image from a graph like that.

  • I'm trying to do the same.
    RGraph is good for canvas graphs. http://www.rgraph.net/

    I've got this far: (the canvas id is 'graph')

    function saveGraph(){
    var canvas = document.getElementById("graph");
    var context = canvas.getContext("2d");
    var img = canvas.toDataURL("image/png");
    document.write('<img src="'+img+'"/>');
    }

    It'll save the image and display it but I'm not sure how to get to the file so it can be e-mailed, saved etc…
