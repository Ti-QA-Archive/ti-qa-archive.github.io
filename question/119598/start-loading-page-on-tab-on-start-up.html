Start loading page on tab on start-up

Hello

Is it possible to start the loading of a page on the start-up of the application? When I boot my application, it opens the main window on my 'home' tab.

While it is on the home tab, I would like to start loading the main window on my other tab, but I don't want it to open the tab yet.

How can i do this?

Thanks!

    I did this by simply opening the window assigned to the tab you want to load.

    var window1 = Titanium.UI.createWindow({url:"url1.js"});
var window2 = Titanium.UI.createWindow({url:"url2.js"});


var tab1 = Titanium.UI.createTab({window:window1});
var tab2 = Titanium.UI.createTab({window:window2});

nav.add(tab1);
nav.add(tab2);
window2.open(); //will start loading the window, even when it's not active in the navgroup.

nav.open();
