  • Hi

    Ti.Media.showCamera({
    success: function (event) {
        if (event.mediaType === Ti.Media.MEDIA_TYPE_PHOTO) {
            // thumbnail image to 200x200
            var thumb = event.media.imageAsThumbnail(200); 
            // do something with {thumb}
        }
    },
    cancel: function () {
    },
    error: function (error) {
        if (error.code === Ti.Media.NO_CAMERA) {
            alert('Not support on simulator');
        } else {
            alert('Unexpected Error');
        }
    },
    saveToPhotoGallery: false,
    allowEditing: false,
    mediaTypes: [
        Ti.Media.MEDIA_TYPE_PHOTO
    ]
});

    You can also crop and resize the image;

    • imageAsCropped
    • imageAsResized;
    • imageAsThumbnail

    This will let you manipulate the image before you need to upload it to Facebook.
