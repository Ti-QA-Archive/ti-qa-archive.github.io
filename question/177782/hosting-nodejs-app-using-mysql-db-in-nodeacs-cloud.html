Hosting node.js app using MySQL db in Node.ACS cloud

Can I host Node.js application that is using MySQL db and integrated with ACS API in Node.ACS cloud?

Here is architecture of the system.

  1. We developed a Mobile App using Titanium SDK, and integrated with ACS API.
  2. We have Web App that is developed using Node.js and is integrated with ACS API. MySQL is used for for database queries.
  3. Mobile App does not talk to Node.js App server, and it either does not require any data from MySQL db

Can we host Node.js App and MySQL database with Appcelerator Node.ACS? if yes, please provide the pointers to the documentation.

  • Yes, you can host the node.js app portion, however our ACS cloud does not offer MySQL services. There are cloud MySQL providers. I've never worked with any and can't make any recommendations. As for docs, go to the regular docs site and from the drop down next to the "Documentation" heading at the top of the page, choose Cloud.
