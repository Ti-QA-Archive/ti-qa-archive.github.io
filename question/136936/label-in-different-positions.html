Label in different positions?

Hi, I am trying to place a label underneath an image. Here is the code I am using:

var facebookLabel = Ti.UI.createLabel ({
    text:'facebook',
    font:{
        fontFamily:'Verdana',
        fontWeight:'bold',
        fontSize:11
    },

    color:'#fff',
    shadowColor:'#333',
    shadowOffset:{x:1,y:1},
    textAlign:'left',
    width:Ti.Platform.displayCaps.platformWidth,
    height:20,
    top:175,
    left:165,
    opacity:1
})

Unfortunately, In the android emulator it shows the label a little bit to the left underneath of the image while in the iPhone simulator, it is showing the label centered right underneath the label.

How come the labels are in two different positions? Am i doing something wrong in the code that i need to change? Please Help!!!!

  • I am using Titanium Studio 2.0.1.2
  • Using Titanium SDK 2.0.1.GA2
  • Using Google APIs Android 4.0.3
  • Using iPhone Simulator 5.1

  • I wouldn't use the full display width as the width. Just set a right parameter with your left param and if should be the right width.

    The way you have it will be wider than than the width, after you've added on the 165 from the left.
