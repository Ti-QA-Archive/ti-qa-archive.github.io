FLIP_FROM_LEFT background Color

How do I make the background color of the following transition animation black so that you cannot see other views behind it?

Ti.UI.iPhone.AnimationStyle.FLIP_FROM_LEFT

Brief explanation of my setup

//
    application_window_dock = Ti.UI.createView({
        height: 460,
        left: 0,
        top: 459,
        width: 320,

        backgroundColor: '#000'
    });
    self.add(application_window_dock);

    application_window_dock.animate({
        transition: Ti.UI.iPhone.AnimationStyle.FLIP_FROM_LEFT,
        view: preview_detail_window
    });

Currently, when it flips, you can see the previous window behind it. Do I have to create another view with a black background, which will be the parent of application_window_dock?

    Yes, you can make another view with a black background which can either be the parent o application_window_dock, or at the same level sibling, but with lower zIndex. this way you can give the viewer the experience you want them to have.
