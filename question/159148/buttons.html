  • Hi, I think this is because you execute the code every time when you go to this page and everything is always the same. You can try using the Globals like this: 

    if (!Alloy.Globals.check)
    Alloy.Globals.check = 'in'; // Or the default

    and then in your If something like :

    if (Alloy.Globals.check == 'in'){
  $.checkin.setTitle("Check Out");
    Alloy.Globals.check = 'out'; 
}
else if (Alloy.Globals.check == 'out')
// do your stuff

    Be aware that it is not recommended to use the global space like that. When you don't need this anymore set it to null, so the GC can collect this variable: Alloy.Globals.check = null
