translate Z axis

so the translate method offers an argument for an amount to move in the z direction. but when I try to set something for that argument, it seems to be ignored. for example.

var b1 = Titanium.UI.create3DMatrix().translate(100,100,100);
button.animate({transform:b1,duration:2000});

moving the object 100 in the x, 100 in the y and 100 in the z.

I'm pretty new to Titanium, but I've done 3d stuff in webkit, figured that it was supposed to show a similar rendering.

thanks for an clarifications.

    Hi, did you see the documentation guide http://developer.appcelerator.com/guides/en/animations_in_action.html . It has an example of how to use 3d animation.

    I think, what you need to is to set m34 property of the transformation matrix first. This value controls the perspective of the 3d view. So, you can try something like this:

    var b1 = Titanium.UI.create3DMatrix();
b1.m34 = 1.0/-3000;
b1 = b1.translate(100,100,100); 
button.animate({transform:b1,duration:2000});

    Hopefully this should work.
