  • hi Taehoon,

    i also faced the same problem.

    just try the below code. it is a silly solution but worked for me even i have found this solution from Q&A forum.

    var current = {
            latitude : latitude,
            longitude : longitude,
            latitudeDelta : latDelta,
            longitudeDelta : lonDelta
        };

        //set the current region
        mapview.hide();
        mapview.setLocation(current);
        mapview.show();
