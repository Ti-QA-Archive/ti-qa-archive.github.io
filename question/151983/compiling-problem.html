    Hi

    I think you will find the problem is this line;

    win1.add(buton1,buton2,buton3,buton4,buton5,buton6,buton7,buton8,buton9,logo);

    The ability to pass more than one object into the add method.

    Whilst it is possible in iOS it should really be one view per add, this will increase readability and cross-platform compatibility.

    Change to;

    win1.add(buton1);
win1.add(buton2);
win1.add(buton3);
win1.add(buton4);
win1.add(buton5);
win1.add(buton6);
win1.add(buton7);
win1.add(buton8);
win1.add(buton9);
win1.add(logo);

    This should work fine.
