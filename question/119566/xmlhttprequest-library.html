XMLHTTPRequest Library

Hey all,

I made myself a nice library to make HTTPClients much easier to use. It takes way fewer lines of code to get a request up and running. It only uses one instance of a HTTPClient, so it queues the requests in the order that they come in and executes them when the recourses become available. This way, multiple requests don't run each other over. It also keeps track of when it is initialized, so its init() function can be called multiple times without harm. It also includes a little helper function that takes an associative array and parses it into url parameters.

Take a look here: 

https://github.com/justimagin/XMLHTTPRequest-Library

Any suggestions or comments are welcome. I've only tested this on ios, but there is no platform specific code. If anyone could confirm it working on Android, that would be great.
