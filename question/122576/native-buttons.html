Native buttons

I've been spending some time searching - but I just wanted to confirm that it's not possible to create a native iOS button with the arrow on it? For example if you go to the app store, look at an app - at the top left there is a back button - I want to create a button of that style.

Am I right in saying I can't? I thought appcelerator gave us access to native buttons, without simply creating them as images!!

    Those native buttons are only native in tabGroups when a new window is opened in a tab, or in navGroups when a new window is opened in the nav. outside of this they just dont exist even if you wrote the code in Object-C

    Make sense?
