  • Hi Vincenco,
    yesterday I have tested this chart module. Titanium makes a awesome work, but after this testing I suggest a web implementation like highchart or others. Look here:

    var web = Ti.UI.createWebView({
    disableBounce : true,
    url : '/html/index.html'
});
web.addEventListener('load',function(){
    web.evalJS('SCRIPT_TO_START_CHART');
});

    Hope this is an inspiration.
