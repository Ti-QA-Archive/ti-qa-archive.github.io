Installing Modules

I was previously using Sencha Touch in an attempt to create a cross-platform app. Due to performance issues on Android platforms, I'm looking at trying to use Titanium Mobile to rebuild my project. It seems I'm getting tripped up on just getting the module in the right place to be referenced.

I can't find "Navigate to your /Library/Application Support/Titanium/modules directory." that is referenced in the documentation. I've searched the entire MAC for this. Do I need to create it? If so, where? Is this old information and changed in the newer version of the TI Studio? Are these module directories app specific?

I'm trying attempting to use the Ti.Paint module and I'm assuming when I have the module in the correct place, I'll hit the + in the tiapp.xml and I will find the modules there. Right now, it shows ti.cloud.

    Your /Library/Application Support/Titanium/modules can be in two places:

    1. under your user
    2. under the root of the mac (main disk, where the OS is installed)

    In Lion you will have to do a trick to see the Library folder, is hidden by default.

    the easiest way to do this is to open a terminal and run

    open /Library/

    to get the root Library
    or 

    open ~/Library/

    to get the Library folder that is under your user.

    The location may vary depending on how Titanium Studio is installed. I think the latest version has it by default under the current user, while on my mac because I use since a long time ago the stuff in the root :)

  • Make sure you are not looking under your username, but directly from the root level of the HD on the machine. If you do a search for 'ti.cloud', you should find that directory in a commonJS folder inside the modules directoy. You can also install a module locally in one application inside a modules folder at the same level as Resources. Here is a guide, maybe you have already read it, but this walks you through installing a module: https://wiki.appcelerator.org/display/tis/Using+Titanium+Modules
