  • To do this, you would need to create an oAuth adapter (or grab one of the open source ones created byt the community) and manually create a button styled to your liking. You could then assign an event listener to the button to send a tweet to the logged in users account when clicked.

    Long story short, there is no native built in API in android for this, so you would have to create your own.
