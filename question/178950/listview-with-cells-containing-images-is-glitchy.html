ListView with Cells containing images is glitchy

Hey everyone,

I am using Titanium SDK 3.4.0.RC2 and have a ListView with some cells in it that contain both text and some images. I am caching the images from the remote URL and loading from the File System rather than from the URL. This slightly improved my performance and helped eliminate the occasional black images I was getting.

However, when I scroll the ListView, whenever it encounters an image it pauses for half a second before continuing to scroll. This is a glitchy user experience that is not acceptable for a production release. Is there a good work-around for this?

Side question: I have a Label inside each ListView Cell as well and they get ellipses for most content that is one row long. Any content that wraps to the second row, appears to work without the ellipses.
