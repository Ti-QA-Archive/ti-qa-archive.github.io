One Click in a Button and all other events also respond!

I'm new on developing app's. In trying to do an app for iOS and it have multiples buttons and each button leads to a new window. The issue that I'm having is that wen I press one button it open all new windows at the same time.

What can I do so the "button.addEventListener" work independent of each button click?

    Hi

    You may be better off with a re-usable event listener.

    var clickHandler = function (e) {
    var win = Ti.UI.createWindow({
        title: L('Actividades'),
        backgroundColor: 'white'
    });
    switch (e.source.type) {
    case 'type1':
        win.add(Ti.UI.createLabel({ text: 'Only if type1' }));
    }
    case 'type2':
        win.add(Ti.UI.createLabel({ text: 'Only if type1' }));
    }
    self.containingTab.open(win);
};
var btn1 = Ti.UI.createButton({ title:'Button 1', top: 10, type: 'type1' });
btn1.addEventListener('click', clickHandler);
self.add(btn1);
var btn2 = Ti.UI.createButton({ title:'Button 2', top: 100, type: 'type2' });
btn2.addEventListener('click', clickHandler);
self.add(btn2);
var btn3 = Ti.UI.createButton({ title:'Button 3', top: 200, type: 'type3' });
btn3.addEventListener('click', clickHandler);
self.add(btn3);

    This code is VERY simplified but should be clear enough to explain the logic.

    This way each of the three buttons calls the same event handler so the bulk of the code is the same for each. But using the extra custom property type you can let it customise the window for each button.
