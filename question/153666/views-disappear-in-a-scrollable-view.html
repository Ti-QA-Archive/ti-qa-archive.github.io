Views disappear in a scrollable view

I've got a scrollable view with widgets inside each of the views that look like the following: 

<ScrollableView id="scrollableview" showPagingControl="true">
    <View>
        <Require type="widget" src="myWidget" id="myWidget" />
    </View>
</ScrollableView>

I have a total of 11 views with 11 widgets in each views. Randomly a view or views will disappear when a user scrolls or pages through them. I found the link @ scrollable views disappears and I've set the scrollview cacheSize and currentPage properties like so in the JavaScript: 

$.scrollableview.cacheSize = 12;
$.scrollableview.currentPage = 0;

My question is how do I prevent this from happening and why is it happening?

Thanks in advance….
