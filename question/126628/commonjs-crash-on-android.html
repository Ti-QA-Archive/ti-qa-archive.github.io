  • ok. after trying the 1.8 sdk I came to the conclusion it doesn't work (not on the beta 1.8 sdk anyway, and not even with work around).

    BUT(!!!!!) - go to http://builds.appcelerator.com.s3.amazonaws.com/index.html and download the latest 1.7.3 sdk - it works! and without any need to not having to compile your js on android.

    keep in mind that this sdk (1.7.3) is still in beta and wasn't released yet - so it's good for dev mode but I would be careful releasing with beta sdk (and from what I've read it should be released soon anyways).
