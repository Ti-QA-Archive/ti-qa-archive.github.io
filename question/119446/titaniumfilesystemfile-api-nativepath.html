Titanium.Filesystem.File API nativePath

Calling the nativePath method like this:

var tempFile = Ti.Filesystem.createTempFile(Titanium.Filesystem.tempDirectory);
tempFile.write(someString);
var path = tempFile.nativePath();

I get the following error:

TypeError: Cannot call property nativePath in object [object TiFile]. It is not a function, it is &quot;string&quot;

The API docs say that nativePath is a method though. So, either something about my code is wrong or the API docs are. If the latter is true could the docs please be updated?
