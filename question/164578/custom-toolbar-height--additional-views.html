Custom toolbar height & additional views.

Hi!

The aim I have to reach is: customize toolbar with additional views dynamically.

Platform: iOS. I'm using Alloy. Latest Titanium build: 3.2.1.201402041146

My template is (toolbar.xml):

<Toolbar platform="ios" id="toolbar_outer">
        <!-- The Items tag sets the Toolbar.items property. -->
        <Items>
            <Button id="option" systemButton="Ti.UI.iPhone.SystemButton.ADD" onClick="chooseOption"/>
            <FlexSpace/>
            <TextField id="input">
                <!-- Sets the keyboardToolbar property -->
                <KeyboardToolbar>
                    <Require src="toolbar/inner" id="toolbar_inner"/>
                </KeyboardToolbar>
            </TextField>
            <FlexSpace/>
             <Button id="send_pulled" title="????." systemButtonStyle="Ti.UI.iPhone.SystemButtonStyle.DONE" />
        </Items>

        <!-- Place additional views for the Toolbar here. -->
        <Label id="photo_loading_status" text="Photo loading..."/>
    </Toolbar>

Inside styles file (toolbar.tss):

"#toolbar_outer":{
    bottom:0,
    borderTop:true,
    borderBottom:false,
    height: Ti.UI.SIZE
}

// No effect!

Inside controller (toolbar.js):

$.toolbar_outer.setHeight(100); // No effect!

Any attempts to customize toolbar height fail.

Appreciate any help!

    See Apple's docs (scroll down a bit to the Toolbar section). I don't believe you can set the native toolbar control's height. Their docs say you shouldn't, but I think that should be read as you can't. If so, you wouldn't be able to set the Ti.UI.iOS.Toolbar height, as it is simply a proxy object for the native control.
