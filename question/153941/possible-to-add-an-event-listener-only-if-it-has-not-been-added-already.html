    Hi

    One simple method would be to add a custom property to the view and then check against that before adding the event each time.

    var btn = Ti.UI.createButton({
    hasEvent: false,
    title: 'Some Title'
});
win.add(btn);

    Then later on;

    if (btn.hasEvent === false) {
    btn.addEventListener('click', function (e) {
        // do something
    });
    btn.hasEvent = true;
}
