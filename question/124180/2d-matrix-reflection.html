  • A 2D Matrix reflection is it's own inverse. You can use the invert() method of the 2DMatrix object in order to get the inverse of an existing matrix.

  • Ok, the matrix object doesn't keep a stack of changes that you've made to it. In fact, when you make changes to a matrix, it actually returns a new matrix constructed from the original matrix. So if you set a variable to the rotation of the matrix, it will actually return a new matrix and set it to the new variable keeping the original matrix in tact. That seems to be what you are trying to accomplish right?

  • Here is my solution using

    Ti.UI.MaskedImage();
    http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.MaskedImage

    This method DOES require creating a gradient in Photoshop and saved as a PNG

    
var img_url = 'http://developer.appcelerator.com/assets/img/badge_titan.png',
    img_width = 75,
    img_height = 75,
    bgColor = '#000000';

var win = Titanium.UI.createWindow({backgroundColor:bgColor});

var img = Ti.UI.createImageView({image:img_url, top:10, left:10, width:img_width, height:img_height});

var img_reflection = Ti.UI.createMaskedImage({
    image:img_url,
    mask : 'img/grad.png', // my photoshop made PNG with a black and transparent gradient in it 
    // the black with mask the image the transparent shows through
    top: 10 + img_height, 
    left:10, 
    width:img_width, 
    height:img_height,
     mode: Titanium.UI.iOS.BLEND_MODE_SOURCE_OUT
});

img_reflection.transform = Ti.UI.create2DMatrix().scale(1,-1); //flip vertically

win.add(img);
win.add(img_reflection);

win.open();

    I understand people may want dynamic reflections but if your trying to replicate some similar to the iTunes coverflow or just create a reflection this has worked perfectly for me.

    Hope this helps
