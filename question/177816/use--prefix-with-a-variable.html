use $. prefix with a variable?

Hi Everybuddy,

I was wondering if it is possible to use the $. prefix with a variable name?

var pickerType = "SomePicker";
var pickerName = pickerType.toLowerCase()+"Picker";
$.pickerName.setSelectedRow(0, sel);'

//gives an error: Uncaught TypeError: Cannot call method 'setSelectedRow' of undefined

Having searched I cannot find an answer, so any tips appreciated…

Many thanks!

    Hi Brett,

    yes can use dynamics keys of the $ variable like this:

    function setPicker(pickerType){
    var sel = attr.get(pickerType);
    if(sel){
        sel = sel.toString().replace('col','');
        var pickerName = pickerType.toLowerCase()+"Picker";
        $[pickerName].setSelectedRow(0, sel);
    }   
}

//I would like to use it as follows:

setPicker("Main");
setPicker("Secondary");
