  • Hi,
    Create searchBar first for that refer searchBar this link. And then add it to tableView. 

    var search = Titanium.UI.createSearchBar({
    barColor:'#000', 
    showCancel:true,
    height:43,
    top:0,
});

var table = Ti.UI.createTableView({
    data: data
    search: search
});

    Hope this will help you.
