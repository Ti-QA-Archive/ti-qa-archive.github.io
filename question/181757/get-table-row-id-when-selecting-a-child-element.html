Get table row ID when selecting a child element

I have a button inside of a view which is inside of the table row. when selecting that button, how can I get the table row's index?

The table rows are movable, so if it is in the 4th position, when clicking the button must return 4. If they move that row to the second position, when it is clicked, it must return two.

Not too sure on what i should do here as the only thing I know how to do is on selecting the entire row, but that has a different event.

(If they click on the entire row, it takes them to another page, but if they click on the button it removes the row)

Any help will be greatly appreciated :)

Ti.API.info("index ==> " + e.index);

if(e.source == '[object TableViewRow]')
    // To detail page code over here.
else
    // To delete the row over here

});
