How to use buttons from outside a function PLEASE HELP!

Hi all , I want to know if it is possible to create a function that creates a window and all of its buttons and features and then call one of the buttons from outside the function for example 

function HomePage(e){
    var winHome = Ti.UI.createWindow({
    backgroundColor: '#7A7A7A',
    zIndex:0
    });
    var btnMail = Ti.UI.createButton({
           top: '15%',
           width: '35%',
           height: '25%',
           left : '8%',
        backgroundImage:'./Mail.jpg',
        zIndex:1,
    });
    winHome.add(btnMail);
    return winHome
};

//Close the function and then 

btnMail.addEventListener('singletap',function(e)      
{
    var winMail = Ti.UI.createWindow({
    backgroundColor: '#7A7A7A',
});

Is there any way to do something like that

  • You should simply add the event listener inside your function and you should be all set.
    If you really need to access this same button from outside your function, simply declare it outside the function and then assign it inside the function.

    Like so:

    var button;

function  HomePage(e) {
    button = Ti.UI.createButton({ ... });
}

button.addEventListener(...);
