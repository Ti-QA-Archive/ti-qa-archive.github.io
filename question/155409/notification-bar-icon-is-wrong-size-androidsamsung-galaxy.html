Notification bar icon is wrong size (Android/Samsung Galaxy)

Our app uses push notifications for various things and we just noticed when doing QA on a Samsung Galaxy phone that when the notification initially comes in the icon that shows in the status bar/notification area is blown up and distorted. However, if you pull down the notification bar then the icon resizes to be the correct size. I don't think we're explicitly setting this icon anywhere as I believe that the app should just use the appicon. It's also not happening on every Android device — on an HTC Incredible and a Nexus 7 the icon does display properly. Is there any other way to set the notification icon for the app to make sure it's always the right size, or any other best practices here?
