Map view Annotations

I am working with a Mapview and adding some annotations for some buildings. However I have noticed that one of the new buildings is missing on this view. I believe the map view is using Google maps for its maps, is there a way to change this to use Bing Maps as the building appears properly on the Bing?

Can I use or overlay an image the map view and still use annotations to show the users current position?

Is there any other suggestions on how I can get a building and the annotations to show up on the Mapview.

Thanks

  • The inbuilt TiMap for both platforms will only render Google Maps. Your only option to use Bing is to attempt to host it in a WebView. Ive never attempted this personally. Check that the new building is available on Bing first though - it sounds like the satellite imagery is a bit out of date (which can happen)
