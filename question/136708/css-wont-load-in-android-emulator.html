CSS Won't load in Android Emulator

I am using HTML and CSS for some pages in my Titanium Mobile App. On iPhone it works great. But in Android the CSS won't load. The weird thing is that the JavaScript works fine(same folder as the css).

The css and Javascript is located in Resources/
The HTML file is located in Resources/ui/tab/
