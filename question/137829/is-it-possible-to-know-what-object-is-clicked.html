Is it possible to know what object is clicked?

titanium mobile sdk 2.0
iphone sdk 5.1

Hi,

Is it possible to know what object is being clicked without explicitly attaching an eventlistener to the object?

Say, i have hundreds of variables and I don't want to add eventlisteners to each of them. When the user clicks anywhere, titanium will then get the name of the object that has been clicked.

Is this possible?

retVal = el;
break;
}
