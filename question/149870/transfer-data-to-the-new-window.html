  • Hi Tomas ,

    you can pass the link parameter when you create the window , like this : 

    image[i].addEventListener('click', function(e) {

    var subWin = Ti.UI.createWindow({
        url : 'subWin.js',
        backgroundColor : '#ffff00',
    link :    image[i].link
    });
    subWin.open();

});

    next time you have to format your code (and please, write better your code , the manner in which objects are placed in the array is not beautiful to look at) .
