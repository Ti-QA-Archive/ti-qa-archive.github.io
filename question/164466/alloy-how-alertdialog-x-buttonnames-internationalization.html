Alloy, how: AlertDialog x ButtonNames internationalization?

Simple question, how internationalize Button names within Alloy?

<AlertDialog imessage="L('msg')">
    <ButtonNames>
        <ButtonName>L('confirm')</ButtonName> <!-- nop -->
        <ButtonName class="confirm"></ButtonName> <!-- on tss, create a class, no success -->
        <ButtonName title="L('confirm')"></ButtonName> <!-- no success -->
    </ButtonNames>
</AlertDialog>

Any clues?

  • Unfortunately, this is not currently supported. There is an open ticket ALOY-853 requesting this functionality. That ticket is waiting for support at the SDK (TiClassic) level, as described in TIMOB-14763. You might want to add yourself as a watcher on both tickets so that you receive notifications as updates are made.

  • Found a workaround: Create the buttons via TSS file, example:

    "#myAlertDialog":{
    buttonNames: [L('confirm'),L('no')]
}
