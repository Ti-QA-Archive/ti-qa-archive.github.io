The visible property

I have one button under another button.
So I set the first button visible: true, and the second button to: visible: false
Then, inside the listener, I set the first button: visible false, and the second to: visible: true
After the 'click' he first button became hidden, but the second did not became visible.

    Hi

    Do not use visible, setVisible or getVisible to achieve your goal once you have created the buttons.

    Instead use show and hide.

    var btn1 = Ti.UI.createButton({
    title: 'Button 1',
    visible: false
});
win.add(btn1);
var btn2 = Ti.UI.createButton({
    title: 'Button 2'
});
win.add(btn2);
btn2.addEventListener('click', function (e) {
    btn1.show();
    btn2.hide();
});
