    • You are referring the section by index here $.tableHotels.sections[1].add(row[i]); not by its reference as a controller property. You should also be careful with counters in for-loops as they count up and so the references are changing. Passing the counter to self-calling function could be a solution. But I would rather go by a complete different approach when creating the sections / rows as I mentioned in the other comment. Try to split your data in as many arrays as the containing types. Then process each array (forEach) and create the section AND the according rows together. After that you can apply the whole thing onto the tableview unlike appending sections and rows one by one.
