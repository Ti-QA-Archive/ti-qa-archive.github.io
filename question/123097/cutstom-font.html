    1. Add the .otf file into your Resources folder.
    2. Add these lines to your Info.plist file.
      <key>UIAppFonts</key>
<array>
<string>YourFont.otf</string>
</array>

    The &lt;string&gt;YourFont.otf&lt;&#x2F;string&gt; is a case sensitive so watch out for that.

  • I banged my head against the wall for hours trying to get this working on android to no avail. Though I finally got it. If you are struggling to get this working on android as well here are some steps that helped me finally get it with a '.ttf' .

    • Make sure your font is in Resources/fonts
    • Delete android emulator from manager sdk manager
    • Turn off fast dev
    • Make sure the font you are calling is the name of the file name (My_Font.ttf) -> "My_Font"
    • edit tiap.xml to force complete rebuild.
    • I also had my font file named the same name as the font "My_Font.ttf" -> "my Font", thinking that might help. Probably doesn't do a darn thing, but ya never know.
      Other things to keep in mind, Capitalization doesn't matter, it should work on all text objects.

    If you still can't get it it, try downloading AndroidCustomFont Project. Then try your font on that project to narrow down any other possible variables.
    Hope this helps guys!
