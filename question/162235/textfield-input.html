    Your code is updating the label – NaNundefined looks to be the concatenation of createFoodParams.weightPerServing + $.servingPicker.getValue.text.

    Check whatever is generating createFoodParams.weightPerServing to make sure it doesn't return a NaN value. And getValue is a method .. so getValue(). There's no need for the .text part.
