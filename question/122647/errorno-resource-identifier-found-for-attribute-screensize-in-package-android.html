error:No resource identifier found for attribute 'screenSize' in package 'android'

I modified the tiapp.xml to include the android manifest entries - for compatible screens as below
<android >
<manifest>
<uses-sdk android:minSdkVersion="8" />
<supports-screens android:anyDensity="false"/>
<compatible-screens>
<!– normal size screens –>
<screen android:screenSize="normal" android:screenDensity="mdpi" />
<screen android:screenSize="normal" android:screenDensity="hdpi" />
<screen android:screenSize="normal" android:screenDensity="xhdpi" />
<!– large size screens –>
<screen android:screenSize="large" android:screenDensity="ldpi" />
<screen android:screenSize="large" android:screenDensity="mdpi" />
<screen android:screenSize="large" android:screenDensity="hdpi" />
<screen android:screenSize="large" android:screenDensity="xhdpi" />
</compatible-screens>
</manifest>
</android>

But it throws error: No resource identifier found for attribute 'screenSize' in package 'android'
The same error is thrown for screenDensity as well.
Am using Ti mobile SDK 1.7.1 and Android 2.3.3
How can this be fixed?

  • I have done something similar, but never got this to work in the tiapp.xml, I would suggest trying a custom android manifest.

    Here is a great resource for understanding the android manifest in its native form:
    AndroidManifest

    Here is how you implement your custom manifest:
    Titanium Wiki

    Here are the first two lines of my custom manifest.

    <?xml version="1.0" ?><manifest android:versionCode="1" android:versionName="1" package="com.yourapp.name" >
    <supports-screens android:anyDensity="false" android:largeScreens="true" android:normalScreens="false" android:smallScreens="false"/><uses-sdk android:minSdkVersion="7"/>

  • The reason that &lt;tool-api-level&gt;9&lt;&#x2F;tool-api-level&gt; must be outside the manifest element in tiapp.xml is because it is not an AndroidManifest tag. It is a tag who's value is used the to set the api-level-specific tooling that should be used by builder.py.

    More info on this here: http://jira.appcelerator.org/browse/TIMOB-2275
