Alternate For NavigationGroup

I want to add navigation group to my android app. So is there any alternate method for android to add a
navigationGroup in android

  • Hi Mayur Sonawana,

    There is no any inbuilt method or controller to show navigation bar for android but If you want to show navigation bar same like ios then you can do it by using views.. just add 48 or 50 px view at top of window and set properties of that view into app.tss because you no need to set properties to each and every window.

    here i send some code for hint how to do it.

    
<Alloy>
    <Window id="winDemo" class="winLayout"> <!-- Window Layout is vertical depends on requirement -->
        <View class="navBarBackViewProperty">
            <View id="statusBar" class="statusBarProperty"> <!-- It's (0PX initially) view for iOS if it's OS > 7 -->
            </View>
            <View id="navBarView" class="navBarProperty">
                <ImageView id="imgBack" onClick="closeWindow"></ImageView>
                <Label id="lblNavTitle" class="lblNavTitleProperty" ></Label>
            </View>
        </View>

        .......
        ......
    </Window>
</Alloy>

//Here all property defined in app.tss so no need to define in all window tss. Just copy paste xml code in all window

//JS

if (Alloy.Globals.isIOS7Plus) {
    $.statusBar.height = 20;
}
