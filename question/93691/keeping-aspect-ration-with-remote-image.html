Keeping aspect ration with remote image

  • I'm not aware of a good cross-platform solution to this problem, but if you're using an iphone you should check out an undocumented function called imageAsThumbnail. It scales and crops an image blob into a square thumbnail. To use it, convert an imageView to a blob. The function takes one parameter, which is the size of the square thumbnail you want to use.

    // Convert your imageView into a blob
var blob = imageView.toImage();

// Turn blob into a square thumbnail
blob = blob.imageAsThumbnail(64);

// Create new imageView for thumbnail
var thumbnailImageView = Ti.UI.createImageView({
   image: blob
});

    Cheers

    -Brandon
