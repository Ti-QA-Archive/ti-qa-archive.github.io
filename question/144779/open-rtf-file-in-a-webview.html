But with Ti when I try this :

the webView display the content of the RTF file as a text file.

    • Finally it works like this :

      var fileRTF = Ti.Filesystem.getFile(destDir, 'car.rtf');
webView = Titanium.UI.createWebView({
        //height : Ti.UI.SIZE || 'auto',
        top : 5,
        right : 5,
        bottom : 5,
        left : 5,
        data : fileRTF
    });
