TabGroups and Multiple Windows?

Greetings,

I am working on an application that records exercise data and syncs said data with a web application. There are four tabs: dashboard, account, workouts and stats.

Where I am stuck is that each of these tabs will have/need multiple "windows" of content. As a pretend workflow, the dashboard tab, when opened, might perform the following:

  1. Check a local SQLite DB for any existing accounts
  2. Assume multiple accounts are found, it will display a list of the accounts (as buttons) from which the user may select one.
  3. When the user selects their account, a new window should open, load that account data and then show the dashboard for that user's account.
  4. Whoops! The user selected the wrong account. They should be able to select a back button and select a different account.

Step 4 is where I am stuck and am having trouble finding the details in the docs or Q&A. Is there a way that I am missing that I can use multiple windows within a tab? Perhaps I'm missing the point and Views are better for this sort of thing that windows?

If there is a doc that I flat-out missed, I don't need a detailed answer … just being pointed to the right doc(s) would be a huge help … cuz I'm stumped!

Thanks in advance!
Craig

    if you use this from your "dashboard" window/tab:

    
var win2 = Ti.UI.createWindow();

tabBarVariableName.activeTab.open(win2);

    this will open the second window, and give you a back button in the navbar for iOS or let you use the back button in android to go back to the last window :)
