Assign .tss styles to dynamically created objects in the controller

Hello,

I am trying to assign .tss styles to dynamically created objects.

This is my xml View

<Alloy>
    <Window>
        <TableView id="tableView" class="menu-table-view"/>
    </Window>
</Alloy>

This is my tss

".menu-table-view":
{
    backgroundColor: "#fff",
    left: 0,
}
"TableViewRow":
{
    backgroundColor:"black"
}

And this is my controller

var tempData = [],

_.each(jsonObject.categories, function(item)
{
    tempData.push({title: item.name, type:item.type, key:item.key, id:item.id});

});

$.tableView.setData(tempData);

The dynamically created tableViewRows are not getting the black background style. It seems that this only affect to elements defined on the xml view, is that right?

I thought that maybe i could define the view like this:

<Alloy>
    <Window>
        <TableView id="tableView" class="menu-table-view">
            <TableViewRow />
        </TableView>
    </Window>
</Alloy>

and then try to get that element from the view in the controller, and clone it, but it seems Titanium doesn't allow to clone elements.

Any solution to this?

Thank you very much.

Titanium Studio, build: 3.1.1.201306112235
Windows 7

  • Accepted Answer

    Styles aren't going to apply to the shorthand tableviewrows that are created by using a plain Javascript object. You'll need to create a separate controller for your tableviewrows and style them that way. Check out this test app for an example: https://github.com/appcelerator/alloy/tree/master/test/apps/advanced/master_detail

    Specifically pay attention to how the "master" and "row" view/controller/style works.
