  • Hi

    Inside your tiapp.xml there will be a property for status bar style, as you are using Alloy it will probably be inside of the plist information. Here is an example:

    <ios>
        <plist>
            <dict>
                <key>UIRequiresPersistentWiFi</key>
                    <false/>
                <key>UIPrerenderedIcon</key>
                    <true/>
                <key>UIStatusBarHidden</key>
                    <false/>
                <key>UIStatusBarStyle</key>
                    <string>UIStatusBarStyleDefault</string>
            ...

    If you were using the old tiapp.xml then you would change this property

    <statusbar-style>default</statusbar-style>

    These properties need to be set to default in order to allow you to set the colour of your statusbar via your barColor

    You should also remove any other references to status bar styles, such as the one in your tss

    Hope this helps
