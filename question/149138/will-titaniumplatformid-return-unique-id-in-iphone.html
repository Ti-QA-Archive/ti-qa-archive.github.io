Will Titanium.Platform.id return unique ID in iPhone?

I want to uniquely identify logged in users. So if i use install Application ID - Titanium.platform.id can i get unique ID even after user update application though iTunes App store?

Anyone help me

  • Hi

    The short answer is no.

    Apple changed the rules on access to a true unique device id (previously UDID) back in iOS 5.0. The current implementation of Ti.Platform.id provides the best work-around allowed right now.

    The Ti.Platform.id property will give you what is essentially a unique value, HOWEVER if the user un-installs the app and then re-installs it again the returned value will be DIFFERENT.

    So the returned value from Ti.Platform.id is only good whilst the app is on that device uninterrupted.

    If you want a truly unique value then you would have to use a token from your server if logged on means they run through a web-service you have set up. If you mean logged-on as simply running your app then Ti.Platform.id.

    Keep in mind, Apple is not happy about tracking devices without peoples permission.
