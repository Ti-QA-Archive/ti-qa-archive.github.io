Getting started with ACS

I'm trying the ACS and thought it would be a breeze - but the samples I found does not work - so I'm somewhat stumped….hopefully someone can help me.

I pressed the "Enable Cloud Services" button in Titanium Studio, and got the two keys (production / development). I also verified that the ti.cloud module was included in the modules section in the tiapp.xml file.

so I copy/paste this code from the docs into my app.js file:

var Cloud = require('ti.cloud');

Cloud.Objects.create({classname: 'cars',
    fields: {
        make: 'nissan',
        color: 'blue',
        year: 2005
    }
    }, function (e) {
        if (e.success) {
            var car = e.cars[0];
            alert('Success:\\n' +
                'id: ' + car.id + '\\n' +
                'make: ' + car.make + '\\n' +
                'color: ' + car.color + '\\n' +
                'year: ' + car.year + '\\n' +
                'created_at: ' + car.created_at);
        } else {
            alert('Error:\\n' +
                ((e.error && e.message) || JSON.stringify(e)));
        }
    });

When I run the code in the ios simulator, I get an alert saying "Error:\nYou need to sign in or sign up before continuing."

So what am I doing wrong ? I sort of thought the to.cloud module handled the oAuth authentication etc. that is required.

  • Incidently, I figured out that the "magic" requires some kind of user to store information with - there is no "global anonymous" user. So I added a user, and authenticated with that user before creating the object, and everything worked fine!
