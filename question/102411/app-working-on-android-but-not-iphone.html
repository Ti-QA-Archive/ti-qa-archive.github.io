App working on Android but not iPhone

I don't have iPhone SDK, so I developed the app on Android emulator and it works absolutely fine on it, but when my client tried it on iPhone, the visuals were messed up, form elements weren't properly stacked and nothing was working as expected!

I thought Titanium was fully cross-platform; idea being, if it works on one platform it should work on the other one as well, but it doesn't! Or am I missing some crucial step which needs to be taken for iPhone testing/deployment?

Oh and I'm not using any Android-specific code, just basic Titanium code, which presumably should be cross-platform (as it says on the API docs)!

  • Titanium is cross platform, but you Will always need to test and implement platform specific code in your app. There are UI differences in the way that android and iPhone display element and therefore you can not expect it to just work out of the box
