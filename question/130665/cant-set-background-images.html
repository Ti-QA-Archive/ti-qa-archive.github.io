Cant set background images

I can't seem to set background images for my components…

var win1 = Titanium.UI.createWindow({  
    title:'Tab 1',
    backgroundImage : 'images/back.jpg'
});
win1.open();

and my directory structure looks like this:

Resources
         |
         + app.js
         |
         + images
                 |
                 + back.jpg

but the image won't display…. am I missing something?

Thanks

    Have you tried doing a hard re-build? I'm running Lion as well and sometimes when I make a change in the code, things don't always re-compile correctly. Maybe try doing a hard re-build by navigating to your 'Titanium Studio Workspace/project-folder/build' and cleaning out/deleting the contents of android and iphone. Then go back into Titanium and select 'Run simulator'.

    I would think that if you have the above createWindow code added to your app.js file, then you would need to reference the image file at: backgroundImage:&#39;images&#x2F;back.jpg&#39;
