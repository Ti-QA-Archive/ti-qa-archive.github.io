Kitchen Sink on Android Emulator is so so Slow! Unusable.

We just started evaluating Titanium 1.5.1 for doing iPhone and Android application development. We are able to run the Kitchen Sink just fine on the iPhone simulator, but on the Android emulator the performance is horrible.

We keep getting pop-ups saying "Sorry! Process system is not responding" and it takes forever for some screens to show up. Every now and then the emulator will just go black and then the Kitchen Sink dies and the Android desktop shows up.

Please advise…we cannot move forward with Titanium if we cannot do Android development.

Also just so you know I using the latest MacBook Pro i7, 256gig SSD, 8gig Ram. So you cannot say it's my computer.

Best,

Brian

Update: We have found that these issues go away when you use Android APIs 2.2 and higher. Anything below 2.2 has all the issues we describe and more.
