Upgraded to iOS 6.1 on xcode and now cannot build projects

I get this error when building projects through Titanium now which I know is because of my recent upgrade to iOS6.1 on xcode. Can anyone tell me how to get round this? I also have all of the iOS versions installed down to 5.

[ERROR] Invalid –ios-version value '6.0'
Accepted values:
6.1

  • If you run titanium info -t ios from the command line, you can see what Xcode installs Titanium finds as well as each iOS SDK. If you do not see iOS SDK 6.0 in any of the iOS SDK lists, then you won't be able to build for iOS 6.0.

    Titanium tries to find Xcode in the /Applications directory. It will match /Applications/Xcode*.app. If you have iOS SDK installed for an older version of Xcode, you should rename it and move it to the /Applications folder.
