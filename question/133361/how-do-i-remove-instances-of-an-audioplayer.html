How do I remove instances of an AudioPlayer?

Hi,

I created an audio player by using the createAudioPlayer method as follows:

var audioPlayer = Titanium.Media.createAudioPlayer(.. etc.

Can someone kindly tell me how I can erase all instances of this method?

I.e. if an end user comes and starts another audio file, I want to be sure that all previous instances of this audioPlayer have been deleted.

THanks

    I think what you need is:

    var streamer = Ti.Media.createAudioPlayer();

// to remove
streamer.stop();
streamer.release();
