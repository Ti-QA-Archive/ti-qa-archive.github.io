Slide in a View

I have a window that has a banner up top and a tableview below. When a user clicks on a table row, I want to slide in the detail view leaving the banner view in place up top. I dont want to recreate the banner on the detail page, because the animation slides it over the top of the existing one. I want just the tableview to animate to the detail view. Does anyone know how this can be achieved.

Also, do I need to worry about memory management since I will not be opening and closing a new window for the detail view. Any help would be gretly appreciated. Thanks!

  • This is going to require some customization of the native functionality. When a row is clicked in your tableview you can either create a window that is sized appropriately and animate it in, or you can create a view that does the same thing. You may want to create windows as it may help with the memory management issue that you already mentioned.

    The customization part is going to be when creating your windows/views, that the details page usually has a back button in the top nav bar. You're going to have to manage your stack of windows and handle backing up and/or closing your details views on your own.
