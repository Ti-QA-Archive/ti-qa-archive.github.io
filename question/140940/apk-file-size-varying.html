APK file size varying.

Hi All, I have a typical issue regarding the file size of apk file. I am working on an application for Android devices. When I built the application in one of the PC it shows the size approx 4.5 MB but in another PC it was built with sixe approx 12 MB. The build with 4.5 MB works fine in emulator and but not in device while built with 12 MB works fine in both (emulator and devices). So what could be the reason behind this differences and how to fix it. Plz suggest :( I am working on a machine which makes the build with 4.5 MB and for which I am not able to test my code on device.
