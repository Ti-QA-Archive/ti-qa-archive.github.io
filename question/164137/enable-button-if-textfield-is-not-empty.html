Enable button if textfield is not empty

Is their such a way that a button will automatically be enabled if two or more specific textfields have values e.g. listen for a change in a couple of textfields and enable button once they have values

Any help towards this problem would be greatly appreciated.

    Have your change listeners for the textfields call a single function.

    In that function check textfield1.value && textfield2.value then mybutton.enabled=true

    • If as added dynamically, they were put into an array, the function could walk the array and to your checks.
