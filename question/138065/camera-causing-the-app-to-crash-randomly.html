Camera causing the app to crash randomly

When I use the camera-example from the kitchensink, oder own code that access the camera-feature on Android, my app randomly crashes when I take a foto and hit "save" in the built-in camera-app. The screen turns black for a few seconds then I get the error: "App crashed unexpectedly…." and I am kicked out of my App.

var view1_foto = Titanium.UI.createView({
    top:'80%',
    backgroundImage:'button_cam.png',
    title: 'Make a Foto',
    bottom:'5%',
    height:'15%',
    left:'1%',
    width:'30%',
    font:{fontFamily:'Arial',fontWeight:'bold', fontSize:'16dp'}
});

var my_image;

var viewImgShow = Titanium.UI.createImageView({
                top: '30%',
                height: '45%',
                bottom: '25%'
                });

view1_foto.addEventListener('click',function(e)
{   
    Titanium.Media.showCamera({
        success:function(event) {
        my_image = event.media;
        if(event.mediaType == Ti.Media.MEDIA_TYPE_PHOTO) {
               viewImgShow.image = my_image; 
            viewImgShow.show();
        }

    }
    });

});

Like I said, it does not happen everytime, it just crashes sometimes, like 50:50. I use newest Titanium SDK (2.02) + Android 2.2 API

Any idea?
