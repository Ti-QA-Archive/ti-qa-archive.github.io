Adding and Removing Event Listeners for

Hi, I am using Titanium 3.1.1 SDK to develop iOS apps using iOS 6.1 SDK. For various reasons I have one Ti.Media.VideoPlayer object created for the entire app and I am using that one videoplayer to play different mp3s.

I have a table with a list of data and when I click on a row,a proxy common js module creates the "Now Playing" window that contains UI such as buttons and sliders that controls the video player. At the time of construction, the videoplayer object is passed in. I attach event listeners such as "durationavailable", "playbackstate" to the videoplayer as part of construction. Everything works as expected.

When I close the "Now Playing" window, and go through another cycle of instantiating "Now Playing" window, the event listeners doesn't work anymore.

I tried to remove event listeners by passing the same function, but no luck.

Anyone know what I am doing wrong?

Below is when I click on a row, the function below is called. The "audioPlayer" that is passed in is the Ti.Media.VideoPlayer object that was created when the app was started.

function SeeProfileDetailEvent(e) {

    var PlayerWindow = require('/ui/handheld/ios/AudioPlayerWindow');
    var playerWindowObj = new PlayerWindow(e.rowData.params.result, audioPlayer);

    audioPlayer.addEventListener('durationavailable',playerWindowObj.PlaybackDurationAvailable);
    audioPlayer.addEventListener('playbackstate', playerWindowObj.PlaybackStateChange);
    audioPlayer.addEventListener('complete', playerWindowObj.PlaybackComplete);

    playerWindowObj.getProxy().open({modal: true});

    playerWindowObj.getProxy().addEventListener('close', function(e) {
        audioPlayer.stop();
        audioPlayer.setUrl(null);
        audioPlayer.release();
        audioPlayer.removeEventListener('durationavailable',playerWindowObj.PlaybackDurationAvailable);
        audioPlayer.removeEventListener('playbackstate', playerWindowObj.PlaybackStateChange);
        audioPlayer.removeEventListener    ('complete', playerWindowObj.PlaybackComplete);
        PlayerWindow = null;
        playerWindowObj = null;
    });
}

Please help.
