    1. use the httpClient to download the html file
    2. use a regEx to find the portion of the html you want to display
    3. Set the html you extracted on a WebView

    Finally, and most important;

    1. Dont expect it to look anything like the original html file unless all the styling was defined inline on style elements. If external css was used it wont render anything like the original
