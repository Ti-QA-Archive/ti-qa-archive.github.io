Stick view to side of device no matter the orientation

Hey, is it possible to have a view stick to one side of the device no matter how many times you rotate it? I tried detecting the orientation change, but all that does is make an animation that is not pretty. What I am thinking of is possibly getting the previous orientation, detecting how many times to view was turned, and rotate the buttons to the orientation while having the view in the same side of the device. The function would return a 1 being 1 time clockwise, or a -1 being one time counterclockwise etc.

    You can set the visible screen to an opacity of 0, and show it again when the orientationchange event is finished (Or the screen has finished turning, there is no event for that, but you could set a timeout or something).

    That way you probably don't see your view expanding/collapsing. You can beautify it by using the Titanium animation for showing it again (A transition for the opacity).

    Good luck, and if it doesn't work, let me know!
