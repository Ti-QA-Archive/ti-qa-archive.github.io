Setting UI Dimensions Based on Parent Window Percentage *and* Pixel length

I want to set the height of a window to (86% - 50px) of its parent element. I really hope there is a way to do this but haven't been able to figure it out yet. Help appreciated,

Nick

    How about…

    var pWindowHeight = parentWindow.height;
currentWindow.height = (pWindowHeight * 0.8) - 50;
