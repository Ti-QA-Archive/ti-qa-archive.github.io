  • Ok, done.

    In the javascript section I needed something like this (example for Country):

        function loadCountries() {
            $.ajax({
                async: true,
                url: 'api/getCountries',
                type: 'POST',
                dataType: 'json',
                success: function(JSON_output){
                    // Clear the option list
                    $('#country').html('<option value="none" selected="selected">--Select value--</option>');

                    // Fill with Countries
                    $.each(JSON_output.Countries, function (index, value) {
                        $('#country').append('<option value="' + value.id + '">' + value.name + '</option>');
                    });
                },
                error: function (msg, text) {
                    alert('Error loading Countries: ' + text);
                }
            });
    }

    My webservice, called by the $.ajax, is a controller in Node.ACS, and is like this:

    // Return all the Countries
function getCountries(req, res) {
    ACS.Objects.query({
        classname: "Countries",
        order: "name",
        response_json_depth: 1
    }, function(data) {
        if(data.success) {
            res.setHeader('Content-Type', 'application/json');
            res.send(JSON.stringify(data));
        } else {
            res.send('Error occurred. Error code: ' + data.error + '. Message: ' + data.message);
        }
    });
}

    And in the html, I executed the Country load at start, and the other fields in the onchange event:

        $(document).ready(function() {
        loadCountries();
    }
