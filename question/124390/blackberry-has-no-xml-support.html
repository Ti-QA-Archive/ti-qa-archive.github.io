Blackberry has no XML support?

Hello all, it appears that the blackberry system has no xml support. When trying to run a simple parse function 

xmlDoc = Titanium.XML.parseString(_xml);

I receive this error: 

No Module for name XML expected ti.modules.titanium.xml.XMLModule

Is there anything I can do? A hand coded work around? A custom module out there somewhere? Any help would be greatly appreciated.

  • In case anyone else is curious how to make this work.

    This is the code you would have used normally (in iOS or android) to parse.

    xmlDoc = Titanium.XML.parseString(_xml);

    In blackberry you must use

    parser = new DOMParser();
xmlDoc = parser.parseFromString(_xml,"text/xml");

    This changes the syntax of the nodes as well.

    For instance…in iOS you would use the following to access a value

    xmlDoc.documentElement.getElementsByTagName('mynodename').item(0).text

    with this other code you must use 

    xmlDoc.documentElement.getElementsByTagName('mynodename')[0].childNodes[0].nodeValue

    I hope that is helpful.
