    1. I highly recommend using Ticons (either the website or grab the CLI) to generate icons. No muss, no fuss.

    2. If you have a file named properly, and in the correct location, then you may not have actually made a PNG formatted image. Almost-common problem is people save a JPEG with a ".png" extension. Easiest way to tell… open up the file in a text editor. If it's a PNG, it'll say "PNG Image:" along with a file name & size. If it's a JPEG, it'll be a ton of garbage text.
