  • between the time the user clicks 'install' in the app store and first launch, everything is between the user and his device. The first opportunity you have to have the user agree to anything is at that first run. Once the information has been displayed and/or the user has agreed to your terms, you set a property as a flag, then on subsequent runs, you check to see if the flag exists, and if it does, you don't throw up the prompt.
