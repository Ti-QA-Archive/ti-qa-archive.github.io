  • There is absolutely no difference between the installation process between a module on iOS and on Android. The only thing you must keep an eye on is the platform attribute when declaring the module in your tiapp.xml file.

    Here is a module declared for iOS:

    <modules>
    <module platform="iphone" version="1.2">
        com.googlecode.quicktigame2d
    </module>
</modules>

    Here is the same module declared for Android:

    <modules>
    <module platform="android" version="1.2">
        com.googlecode.quicktigame2d
    </module>
</modules>

    And here is the same module with both versions used in the same application:

    <modules>
    <module platform="iphone" version="1.2">
        com.googlecode.quicktigame2d
    </module>
    <module platform="android" version="1.2">
        com.googlecode.quicktigame2d
    </module>
</modules>
