  • Hi

    As your Apis require auth to work there is no way anyone can test the results.

    You are going to have to call the specific customer URL in your own browser and enter the auth info. If this returns more than one record that your problem is there. This function has two loops, one inside the other, this implies it is returning more than one record. If that is the case and you need to filter out info then you need at least one if statement.

    I would start there.. Without a sample data set and better formatted code this is going to be very difficult to determine from this end.
