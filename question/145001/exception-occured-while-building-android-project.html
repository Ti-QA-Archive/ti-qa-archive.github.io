    In $HOME&#x2F;.deltafy you can try renaming the delta file, cleaning your project and rebuilding.

    • This occurs on Windows (I see that you are not on Windows) if there are high ascii chars in a filename, or in the project name. Sometimes, even if those files are renamed, the name seems to persist in the deltas, causing an error. Can't say I'v seen this on non-windows systems before though.
