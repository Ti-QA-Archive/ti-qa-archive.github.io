How to add a click listener to a label in a TableViewRow?

I'm looking at the todomvc sample app on github (https://github.com/appcelerator-titans/todomvc) and it has exactly what I want in terms of a check box on each row of a TableView. I'm using Alloy databinding to populate the table. Each row has a Label containing a check mark character, just like in the todomvc app. I'm trying to attach a click listener to each one of these check mark labels, but I get an error of "can't call method addEventListener of undefined". I haven't found any sample code that will solve my problem. The todomvc has no code (that I've found) that will add a click listener.

Here is my Alloy XML code:

<Alloy>
    <Window title="Trip Legs">
        <TableView id="tripLegTable" dataCollection="tripLeg" dataTransform="legTransform" dataFilter="legFilter">
            <TableViewRow     title="" 
                            font="fontSize:24" 
                            size="Ti.UI.SIZE" 
                            layout="horizontal" 
                            tripLegID="{tripLegID}">
                    <View width="Ti.UI.SIZE" height="100%">
                        <Label id="check" class="unchecked" text="?" checked="false" onClick="checkBoxClicked" touchEnabled="false"/>
                    </View>
                    <View layout="horizontal" width="Ti.UI.FILL" height="Ti.UI.SIZE">
                        <Label text="{destTime}"/>
                        <Label width="5" height="1"/>
                        <Label text="{legSequence}"/>
                        <Label width="5" height="1"/>
                        <Label text="{riderName}"/>
                    </View>
                    <View layout="vertical" width="Ti.UI.FILL" height="Ti.UI.SIZE">
                        <Label text="{destInfo}" left="0"/>
                    </View>
            </TableViewRow>
        </TableView>
    </Window>
</Alloy>

I've tried various ways to get the click listener to attach, such as using postLayout (never seems to fire), but nothing has worked. Any tips?

Titanium Version: 3.0.24
Device: Samsung Galaxy Tablet firmware version 2.3.5, and Android 2.2 Emulator
Host OS: Windows 7 64 bit
Titanium Studio, build: 3.0.2.201302191606

    • Thanks for the link! BTW, for anyone else looking at this, I used the ImageView object to show a checkbox graphic, but when toggling it between an unchecked and checked image it would only update once. Apparently this is a bug on Android. From what I read it works fine on iOS. To get around the problem I used a View and toggled its backgroundImage.
