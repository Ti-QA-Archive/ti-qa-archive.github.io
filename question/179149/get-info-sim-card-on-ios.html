  • You can try if this module still works (few things like the Reachability part are broken):

    https://github.com/benbahrenburg/TiNetworkHelpers/tree/master/iOSNetwork

    You can retrieve the carrier info like this:

    Ti.API.info("Create a new Carrier Object");
var carrier = network.createCarrier();
Ti.API.info("Call the findInfo method to return the carrier details");
var carrierInfo = carrier.findInfo();
