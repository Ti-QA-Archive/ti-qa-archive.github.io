    • A little extra tidbit, you will be doing this a LOT, checking the os and inserting a hack or a change. To save yourself some time, create some variables.

      Simple, non-namespaced example:

          if (Ti.Platform.osname === 'android'){
        isAndroid = true;
        isIOS = false;
    } else {
        isAndroid = false;
        isIOS = true;
    }

      This only splits up android vs iOS and not iPhone vs iPod vs iPad. You can create others to be more specific when breaking out your iOS specific device support. But it can be useful as you now simply write:

      if (isAndroid) {
 // my statements here
}
