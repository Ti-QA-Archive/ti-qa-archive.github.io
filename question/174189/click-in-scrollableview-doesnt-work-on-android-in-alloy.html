  • Hi,

    That won't work on android, see the documentation:

    Only the scroll event exists for the ScrollableView on Android. To support others, child views may be added to pages, and event listeners added to these views instead of the pages themselves.

    http://docs.appcelerator.com/titanium/3.0/#!/api/Titanium.UI.ScrollableView

    So you should add the click event to the pages/views you place inside the scrollableView.
