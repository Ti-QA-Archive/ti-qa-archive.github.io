Text in textField will not wrap

I have a textField that I want the text to wrap in, however, even with the horizontalWrap property set to true, the text will just go on forever on one line. Here is my code for that textfield:

            var messageField = Ti.UI.createTextField({borderStyle:Ti.UI.INPUT_BORDERSTYLE_ROUNDED, autocorrect:true, left: 10, width: '90%', right: 10, height: 100, horizontalWrap: true, layout: 'horizontal', verticalAlign: Ti.UI.TEXT_VERTICAL_ALIGNMENT_TOP});

Is there something that I'm missing?

    Hi

    You need to use textArea instead of textField if you want multiple lines and word wrapping.

    The horizontalWrap property does not set if text wraps, it is used to determine how child views positioned when added to a parent view.

    Here is your code adapted to be a textArea.

    var messageField = Ti.UI.createTextArea({
    autocorrect: true,
    borderColor: '#bbb',
    borderRadius: 10,
    borderWidth: 2,
    color: '#000',
    height: 100,
    layout: 'horizontal',
    left: 10,
    right: 10,
    textAlign: 'left',
    value: 'I am a textarea',
    verticalAlign: Ti.UI.TEXT_VERTICAL_ALIGNMENT_TOP,
    width: '90%'
});
