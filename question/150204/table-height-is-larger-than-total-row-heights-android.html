Table height is larger than total row heights (Android)

I have a screen with a ScrollView that contains a label and a table. The table is defined as:

var table = Ti.UI.createTableView({
  bottom: 0,
  backgroundColor: '#ff0', // for debugging
  contentHeight: 'auto',
  contentWidth: 'auto',
  rowHeight: 'auto',
  top: 0,
});

I dynamically add rows and increment the height of the table by adding the height of each row I add. When I'm through adding rows, The rows are all added correctly and produce a total height of 530 and the table height gets set:

table.height = tableHeight + 'dp';

When I load the screen, the table height extends beyond the last row by about 80 (just an eyeballed guess). If I reduce the size of any row, I impact content so each row is correct in and of itself.

Any idea where I might be going wrong?
