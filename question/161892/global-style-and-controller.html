Global style and controller

In my application the UI is fairly consistent, thus I have created an "app.tss" file to hold all of the necessary information about the layout. I would like to have some "global" functions that can be called when various events happen from many different views. I tried creating a controller called "app.js" assuming that it would go along with anything from app.tss however I was incorrect. How do you create functions that can be easily called from all views and attached in the xml files? (similar to how layout information from app.tss is available)

I am trying to add code to alloy.js but am having issues with scope. My last resort will be to create a js file with the functions in it and include it into all pages and place calls in each individual pages xml file.

Thanks

  • Created a pretty sketchy workaround for this issue. Alloy.js holds all of the code for the menu and each controller js file holds small reference snippets.

    index.js

    function openMenu(e){
    Alloy.Globals.openMenu($);
}

    alloy.js

    var menuOpen = false;
Alloy.Globals.openMenu = function (w) {

    if(menuOpen == false)
    {
        var openWidth = (Ti.Platform.displayCaps.platformWidth*8/10)+ "dp";

        w.mainContentTab.animate({
            left: openWidth,
            duration:200
        }) ;

        w.hiddenMenuTab.applyProperties({
            width: openWidth
        }) ;

        menuOpen = true;
    }
    else{
        var openWidth = "0dp";

        w.hiddenMenuTab.applyProperties({
            width: openWidth,
            duration:200
        }) ;

        menuOpen = false;
    }
};

    it's not pretty but it will do… if there is a more elegant solution let me know
