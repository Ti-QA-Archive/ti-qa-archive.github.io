Many suggested solutions to 'Build process exited with code 1'..Android 4.3, Ti 3.1.2GA,Win 7

I've read most of the 'solutions' for the above mentioned error and the responses indicate that one or more of those solutions works for some people. In particular, what I've seen most is the androidsdk/build_tools has two files and a folder that belong in androidsdk/platform tools (aapt.exe,dx.bat and /lib). It's necessary to copy from build_tools to platform tools. This, they say, is caused by later versions of the android sdk. However, this bug was supposedly fixed in 3.1.1.GA, but it didn't work for me. I've tried most of the suggested solutions, such as:
Ensure Android SDK 2.2 installed
Updated Python to 2.7.0.
Ensure Android SDK 2.3.3 installed
Copy build_tools aapt.exe, dx.bat and /lib to platform_tools.
Set JAVA_HOME to c:\program files\java\jdk1.7.0_40
Add jdk1.7.0_40 to path
Ensure c:\program files\java\jdk1.7.0_40\bin in path
Ensure Android SDK in path

I started having this error in the project I built. Since I couldn't solve the issue, I created a default Ti Studio "Hello World" project, but get the same error with it.
I could really use some help with this. I've seen various of the above solutions work for others, but none work for my simple project. Plus I'm fairly sure I've read all the google search responses to the question of this error. Thank you in advance for your assistance.
