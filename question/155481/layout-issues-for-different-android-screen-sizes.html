  • 1) Set anyDensity="true"

    2) In tiapp.xml set <property name="ti.ui.defaultunit">dp</property>. This way you can use top:10 instead of top:'10dp'

    3) see 2)

    Remember to place density specific images/icons in specific resource folder i.e. mdpi, hdpi, xdpi etc.

    Hope this helps.
