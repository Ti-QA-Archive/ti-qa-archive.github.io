Block click handler on table row?

I have a custom table row, with 2 buttons. The buttons have handlers, which work fine. I don't want clicking the row to do anything, as it serves no purpose on this screen.

I set the row.selectedBackgroundColor = &#39;#fff&#39;;, but still tapping it causes the 2 buttons to flash black, which is confusing to the user.

I tried adding an event handler to the table view with an empty function, but this still happens. How can it be prevented?

SDK 1.7.2, os 10.7, ipad 4.x

    No, adding empty function as event handler would not prevent "click" event. "click" event would happen for all added event handler functions, including empty function (so, adding event handler doesn't replace the old one).

    I'm not sure did I understood your question correctly, but you can try to set table view's property "allowsSelection" to "false".

    var tableView = Ti.UI.createTableView({ allowsSelection: false });
