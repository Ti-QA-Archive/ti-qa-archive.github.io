ScrollView in ScrollableView

Hello,

I want to know if it is possible to put a scrollview in a scrollableView.
I want a scrollableView where each view can scroll vertical.
In my code i add 2 scrollviews to the scrollableView, and the first view is a image with a large height. The problem is that the view won't scroll vertical. When i add only the first scrollView to the window, it CAN scroll vertical, so i don't see why it CAN'T scroll when i add it to the scrollableView. So my question is: Why does my scrollView not scroll in the scrollableView?

this is my code:

var scrollView1 = Ti.UI.createScrollView({
    top: 70,
    contentHeight: 'auto',
    backgroundColor: "blue",
    left: 5,
    right: 5,
    showVerticalScrollIndicator:true,
    layout: 'vertical'
});

var scrollView2 = Ti.UI.createScrollView({
    contentHeight: 1000,
    backgroundColor: "red",
    left: 5,
    right: 5,
    showVerticalScrollIndicator:true,
    layout: 'vertical'
});

var image = Ti.UI.createImageView({
    left: 5,
    right: 5,
    top: 70,
    height: 800,
    image: "../../images/close_img.png"
});
scrollView1.add(image);
var questionsView = Ti.UI.createScrollableView({
    backgroundColor: "black",
    views:[scrollView1,scrollView2],
    currentPage:0,
    height: 1200,
    top: 0,
});

win.add(questionsView);
