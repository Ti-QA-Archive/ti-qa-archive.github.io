Community App crash on Android

This is a critical issue to start a new paradigm (Community App). Well…my test:

1) I downloaded, imported and make it to deploy on device;

2) Launch the app (Community App) and pause it in the background;

2) Start three applications (like WhatsApp, Evernote, Foursquare and/or Twitter), leaving it in the background;

3) Try to access the app (Community App) that was in the background;

4) It's crash!

My environment:

The latest version of Studio Titanium

SDK 1.8.1

V8

Testing done on a Samsung Galaxy GT-S I9000B (android 2.3.3)

    Hello,

    please get a copy of the CI build, this is solved. You can get it from here, select the master branch, copy the url and go back to your Ti Studio and go to help->Install sdk from url and copy the url there. Then go to your tiapp.xml, and change the sdk to 2.x. Then clean the project and recompile.
