  • Hi guys,

    Try this sample code. I just added the color for text color and key Board type. I think KEYBOARD_NUMBER_PAD has some problem .

    var aTextField = Ti.UI.createTextField({
    width : Ti.UI.FILL,
    hintText : 'New Password:',
    left : 10,
    right : 10,
    height : '40dp',
    top : 60,
    color:'#000',
    borderRadius : 5,
    borderColor : '#007F00',
    borderWidth : 1,
    backgroundGradient : {
        type : 'linear',
        colors : [{
            color : '#F2F2F2',
            position : 0.0
        }, {
            color : '#FFFFFF',
            position : 0.50
        }, {
            color : '#FFFFFF',
            position : 1.0
        }],
        backFillStart : false
    },
    passwordMask : true,
    borderStyle : Titanium.UI.INPUT_BORDERSTYLE_ROUNDED,
    returnKeyType : Titanium.UI.RETURNKEY_NEXT,
    keyboardType : Titanium.UI.KEYBOARD_DEFAULT
});

    If you have any question then let me know.

    Thanks.
