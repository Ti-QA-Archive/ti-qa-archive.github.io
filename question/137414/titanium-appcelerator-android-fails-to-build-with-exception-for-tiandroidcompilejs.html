Titanium Appcelerator Android fails to build with exception for ti.android.compilejs

I have a fully working project which is perfectly running on Android and iPhone devices. But when I put &lt;property name=&quot;ti.android.compilejs&quot; type=&quot;bool&quot;&gt;true&lt;&#x2F;property&gt; in TiApp.xml and performs build, it tries to compile each and every js file, that is okay to me and even in these files we don't have any error but suddenly we encounter Exception and build fails.

I don't know why this exception occurs, I have googled about this but go no solution online. Please let me know how to resolve this issue. I need ti.android.compilejs as true.

This all I am doing because I need to distribute an app on market place and I also need to perform certificate signing for security purpose. Please let me know if there is any other solution.

  • This bug is caused by a problem in the javascript compiler (compilejs true). When compiling your code the compiler fails in a way that the Titanium compiler.py script couldn't parse. If you change compiler.py to add the line:

              print "[INFO] so",so,"se",se

    near line 166, in the function compile_javascript, after the line

            so, se = run.run(jsc_args, ignore_error=True, return_error=True)

    you will be able to see which error actually occurred.
