  • Usually you might need to add a height to the element ( label for example ) that is bigger than the fontSize.

    fontSize works only in px and you don't have to specify the "px"

    like this

    
Titanium.UI.createLabel({
   height:20,
   font: {
      fontSize:14,
      fontWeight:'bold'
   }
);
