Array Unique

How to create Below array to unique..?
In JavaScript, We have direct unique() method for ,make array to unique.
But in my array, i have Key and value Pair. Please check the Array below : 

var a = [
    {name : 'abc', address : 'add1'},
    {name : 'pqr', address : 'addre'},
    {name : 'def', address : 'add3'}
]

here , i m using this below code : 

Array.prototype.unique = function unique() {
    var j = 0;
     while (j < this.length) {
          for ( k = this.length; k > j; k--) {
               if (this[k].name === this[j].name) {
                    this.splice(j, 1);
               }
          }
          j++;
     }
     return this;
}

But above code is not working…

  • Hi,
    As array index is one less less than length , thats why you are getting undefined ,
    and if you are assigning k = this.length; so if lets assume array length is 10 so index will start with 0 and goes to 9 , but you are trying to acces 10 index which is not there , therefore you are getting undefined, just try this code :

    Array.prototype.unique = function unique() {
    var j = 0;
    while (j < this.length) {
        for ( k = this.length; k >= j; k--) {
            if (this[k-1].name === this[j].name) {
                    this.splice(j, 1);
            }
        }
        j++;
    }
    return this;
}

  • You'll need to write your own "unique" function because variables that hold object references are the same if they point to the same object, so …

    var obj1 = {};

var obj2 = obj1;

var arr1 = [ obj1, obj2 ];

arr1[0] === arr1[1]; //true, but ...

var arr2 = [ obj1, {} ];

arr2[0] === arr2[1]; //false

    Since array that you use contains objects, you may want to compare different object properties instead of object references.
