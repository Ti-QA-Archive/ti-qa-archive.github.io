Is it possible to replace a window inside a navigation group?

I have 2 windows. The first of which has an option to open another window which shows the location of the image on a map (among other features). On click I'm doing this:

location_btn.addEventListener('click',function(e){
    Ti.App.SMV = new SingleMapView(jsonObject.Latitude,jsonObject.Longitude,jsonObject.Id);
    Ti.App.tabNav.open(Ti.App.SMV,{animated:true});
});

In this second window, there are other points on the map that I want to replace the previous window using different data. I'm attempting to do it with this (e.source._thisid is the id I'm passing back to the view):

Ti.App.tabNav.close(Ti.App.SMV,{animated:true});
Ti.App.tabNav.close(Ti.App.detailWin,{animated:false});
Ti.App.detailWin = new DetailView(e.source._thisid);
Ti.App.tabNav.open(Ti.App.detailWin,{animated:true});

I'm trying to close the map window (with the animation intact) and renewing that data. Is there a way to replace the previous window with a new window so when I close the map, the new data is there?
