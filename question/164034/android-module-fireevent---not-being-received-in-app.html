Android Module fireEvent - not being received in App

Hi, I am writing a Android Module - all is going well except that I cant get the fireEvent in the module to be received in the Javascript side of the app.

In the Java File I have the following
module.java

    @Kroll.method
    public void getNotification() {
        String alert = Module.notificationAlert;
        Map<String, String> extras = Module.notificationExtras;

        Log.d(TAG, "Alert: " + alert + "Extras: " + extras);

        HashMap evt = new HashMap();
            evt.put("payload", alert);
            fireEvent("pushCallback", evt); 

    }

and in the Javascript I have the following

var pushmodule = require('com.module.push');
pushmodule.addEventListener('pushCallBack',function(evt){
        Ti.API.info("Payload:" + evt); 
        alert("Payload:");
    });

I am not getting anything on the API or not even an Alert - the Log.d logs fine.

Any help would be appreciated!

  • Check your spelling – you have both pushCallback and pushCallBack.

    Also, you might want to check to see if there is something listening for the event prior to firing by checking with 

    if (hasListeners("pushCallback")) {
...

    This will save events from being fired if no one is listening.
