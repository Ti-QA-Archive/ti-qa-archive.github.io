showing push notifications in app

I am using Urban Airship to show push notification (when another user likes your content for example) and it works great.

I would now like to show a badge on a tab then link to a page that shows a list of all pushes to you (show who liked your content).

I can show a badge on a tab by using the following 

var moreTab = Titanium.UI.createTab({
    title:'More',
     icon:'images/more_icon.png', 
     badge: true,
    window:more
});

My issue is how I show the actual number of current badges. In theory I'd get the number of active badges set it to a variable then set badge to that var. But I dont know where to pull that info from.

I'm also guessing that I should be able to pull all the badge data from somewhere as a json file then just show that on one page but again not sure where to find that data.

Has anyone done this? If so where do I get the data from? Any info you can share on this would be greatly appreciated.

Thanks in advance.

  • According to Urbanairship documentation.

    "When a push is received in a background, the OS sets the badge number; so the best way to get that is to pull it off the shared application object, like so:

    badge = [[UIApplication sharedApplication] applicationIconBadgeNumber]
