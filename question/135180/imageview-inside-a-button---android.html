  • Accepted Answer

    now you can do for android is

    create a view and add button first then create two imageView with the picture size and add it in the same view

    like

    
var view = Ti.UI.createView({
width : 300,
height : 50
});
var btn = Ti.UI.createButton({
backgroundImage : <path>,
width : 300,
height : 50
});
view.add(btn);

var leftImage = Ti.UI.createImageView({
width : 50,
height : 40,
left : 5,
top : 5,
image :<path>
});
view.add(leftImage);


var rightImage = Ti.UI.createImageView({
width : 50,
height : 40,
right : 5,
top : 5,
image :<path>
});
view.add(rightImage);
