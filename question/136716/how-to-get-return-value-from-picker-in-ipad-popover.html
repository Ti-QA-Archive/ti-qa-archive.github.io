how to get return value from Picker in iPad Popover?

I have a table view with a button in each row that invokes a popover with a picker in it:

It obviously is working, but I need to get what the user has selected and assign that to a variable in the row. Each row might wind up with a different value.

I am calling it like this:


var btn = Titanium.UI.createView({
        width: 100, height: 30, backgroundColor: '#f00'
    });
btn.addEventListener('click', function(){
        showShapePicker(this)
    });

and the handler:

function showShapePicker(v) {
    var popover = Ti.UI.iPad.createPopover({
        width : 200,
        height : 130,
        title : 'Picker',
        arrowDirection : Ti.UI.iPad.POPOVER_ARROW_DIRECTION_RIGHT
    });
    var picker = Ti.UI.createPicker();

    var data = [{
        title : 'Rectangle',
        fontSize : 20
    }, {
        title : 'Triangle',
        fontSize : 20
    }];

    picker.add(data);
    picker.selectionIndicator = true;

    popover.add(picker)
    picker.setSelectedRow(0, 1, true);

    popover.show({
        view : v,
        animated : true,
    });
};

SDK 2.01, iOS 5.1
