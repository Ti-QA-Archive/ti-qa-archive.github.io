Can't fit region when the map view initialized. (Region ignored)

Hi. I've got a mysterious problem using a map when working on a small application.

First, I've tried to show a map like this, just little bit of modify from template created by Titanium Studio.

app.js

var tabGroup = Titanium.UI.createTabGroup();

var win = Titanium.UI.createWindow({  
    title:'Map Inspect1',
    tabBarHidden: true
});
var map = Titanium.Map.createView({
    region: {
        latitude: 35.163760,
        longitude: 136.911134,
        latitudeDelta: 0.01,
        longitudeDelta: 0.01,
    }
});
win.add(map);

var tab = Titanium.UI.createTab({  
    window:win
});
tabGroup.addTab(tab);  
tabGroup.open();

It almost work well, but region property is ignored. It showed world-wide map.

But if I separate source files like this, it works perfect.

app.js

var tabGroup = Titanium.UI.createTabGroup();

var win = Titanium.UI.createWindow({  
    title:'Map Inspect1',
    tabBarHidden: true,
    url: 'map.js'
});

var tab = Titanium.UI.createTab({  
    window:win
});
tabGroup.addTab(tab);  
tabGroup.open();

map.js

var map = Titanium.Map.createView({
    region: {
        latitude: 35.163760,
        longitude: 136.911134,
        latitudeDelta: 0.01,
        longitudeDelta: 0.01,
    }
});
Ti.UI.currentWindow.add(map);

What is the difference between those source code?
