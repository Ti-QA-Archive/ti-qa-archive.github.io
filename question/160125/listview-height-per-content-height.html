Listview height per content height

Hi, I have a ListView with dynamic content pulled in from a web-service. Currently the ListView occupies the entire window/screen height even if there are no content rows. I want to adjust the height of the overall ListView based on the the number of rows in the ListView. I tried using height="Ti.UI.Size" but it doesnt help. Any suggestions?

Thanks.

  • Hi

    The listView is not designed to solve this problem, this is a job for tableView, you can then set the scrollable to false and height to Ti.UI.SIZE.

    The tableView is designed to work with views - the listView is designed to work with data only, so is only designed to present a table of data in a pre-defined way.

    If you have so few rows that the height can be reduced from the original size then there are no performance reasons for using a listView - so a tableView is perfect.
