Calculation over user input or text fields.

Hi Guys,
It's been 2 weeks since I have started using this tool for Mobile Application. I am using Windows 7 and Android emulator as device.
I am stuck in a situation which I am unable to get hold of. I have created 3 text fields. A user will input number in the first two text field. Now, as soon as the second text field is entered I want to make some calculation and display the result in the 3 text field. I ried doing it, for example textfield3.value = textfield1.value + textfield2.value .. But the result for this is it concatenates the two values in the text field. Is there any specific way to do the mathematical calculation on the value of text field.

    That's because text fields "value" property is a string, not a number.

    To quickly cast value to integer you can use "+"

    textfield3.value = ((+textfield1.value) + (+textfield2.value)).toString();

    +textfield1.value - cast to number

    +textfield2.value - cast to number

    (+textfield1.value) + (+textfield2.value) - add number values

    ((+textfield1.value) + (+textfield2.value)).toString() - back to string (because text field value accepts string values)

    You could also check if casting to number produced not-a-number (for example, if text field was empty) and not display anything in that case.
