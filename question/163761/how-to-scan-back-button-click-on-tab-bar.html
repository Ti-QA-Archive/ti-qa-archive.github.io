  • Hi

    This sentence does not make any sense;

    • "how(if possible) can I scan click on Back button added by tab bar to second window"

    What does **scan click" mean? Are you asking?

    • How does the "back" button appear their?

      It is automatically added as tabGroups have built in navigation controllers

    • Can you change the caption of "Back"?

      Yes - set backButtonTitle property of the second WINDOW you create and when opened the button caption changes

    • Can you remove the back button when the second window appears?

      Yes - set the leftNavButton of the second window to null. You can do this in the window properties you create or afterwards in code;

    // assumes "secondWin" is the name of the second window
secondWin.setLeftNavButton(null);

    Anything else then you will need to explain more clearly what you require.
