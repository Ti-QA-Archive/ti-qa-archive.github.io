    Hi Mritunjay,

    put separate resized image in each resolution folder in Resource > android > images > YOUR RESOLUTION for your tab bar image

    or when you using High, medium and low folder for all resolution then put individual sized image for these folders, it also places in above path.

    • Add all folder in following location.

      Resource > android > images > high

      Resource > android > images > medium

      Resource > android > images > low folder

      and create different size of images and put it to all three folder with same name.

      Keep in mind that you have images in your main image folder with same name.
