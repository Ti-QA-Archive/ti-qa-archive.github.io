  • I have got it working… this issue is caused by one or more space characters in the path to the browser executable.

    My Google Chrome is installed in (I'm ussing a 64bits Windows OS): C:\Program Files (x86)\Google\Chrome\Application\chrome.exe . So the path contains multiple spaces.

    Just copy the 'Application' directory of Google Chrome to your C:\ and adjust the browser path in Titanium under Preferences > Web Browser . Titanium starts launching the webbrowser :-)
