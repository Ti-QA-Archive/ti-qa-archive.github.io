    Hi Nick,

    try this way :

    first set the property when you click on button : 

    Ti.App.Properties.setBool("loggedIn", true);

    and check your property using this : 

    if (Ti.App.Properties.hasProperty("loggedIn")) {
    if (Ti.App.Properties.getBool("loggedIn")) {
        // main screen open
    } else {
        // your image and button contain screen open
    }
} else {
    // your image and button contain screen open
}
