Titanium freeze UI when loading Database

I'm building an app for Android.

I have a "Detail" form have a tableview. Tableview retrieve data from sqlite.

In "Main" windows I have the button "Go Detail" to open "Detail" form.

In "Detail" form, I have the "function X()" to create Tableview and "function Y()" to load database.

I call "function X()" first and then I call "function Y()",

When I tap on button "Go Detail", "function X()" run and complete but my app is freeze on "Main" and not show Tableview until "function Y()" complete.

I wanna "function X()" run and show Tableview, then "function Y()" run.

Loading Database make my app freeze.

I need your help.

Many thanks!

  • Hi

    The database calls are synchronous so they will stop your app during them to make sure that any database calls do not crash the database during any writes.

    The solution is to request fewer entries - enough to display on the tableView in the visible portion and then a few more. Then when the user decides to scroll down the tableView you detect the scroll position getting close to the bottom of the tableView and then loads some more - this is called lazy loading - and is the most efficient in this sort of usage.
