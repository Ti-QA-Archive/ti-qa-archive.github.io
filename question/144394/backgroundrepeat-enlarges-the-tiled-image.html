backgroundRepeat enlarges the tiled image

I'm using a lined pattern image to tile in the background of a view for an iphone app. I have the backgroundRepeat property set to true and it works mostly fine. 

var headerText = Ti.UI.createView({
        backgroundImage:'/images/login_header_background.png',
        backgroundRepeat:true,
        width:320,
        height:35,
        left:0,
        right:0,
        top:headerHeight
    });

The issue is the background image seems to be scaled up or enlarged. So a 4x4px image actually looks like its being presented and tiled at 8x8. The problem is present in the iOS Simulator (iPhone3,4,5) and on iPhone 4s and 5 test devices.

Anyone else having this issue? Any potential fixes?

Using:
Titanium SDK version: 2.1.3 (10/02/12 16:16 15997d0)
iPhone SDK version: 6.0
