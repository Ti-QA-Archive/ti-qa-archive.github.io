Eventlistener doesnot return to tableview

Hi, as a newbie i have a problem returning to a tableview after a click event. I have a tableview and click on a row opens a small map with annotation. Now i want to click the map to return back to the tableview. The 'win.close(map view)' close the the complete window (also the tableview) so it isn't working for me.
What i have so far:

inline code

table.addEventListener('click', function(e) { 

  var annotation = Ti.Map.createAnnotation({
        latitude:user.lat,
        longitude:user.lng,
        title:user.name,
        pincolor:Ti.Map.ANNOTATION_GREEN,
        animate:true,
    });

   var mapview = Titanium.Map.createView({
    mapType: Titanium.Map.STANDARD_TYPE,
    region:{latitude:user.lat, longitude:user.lng, latitudeDelta:0.005, longitudeDelta:0.005},
    animate:true,
regionFit:true,
userLocation:true,
    touchEnabled:'true',
    annotations:[annotation],
    height:250,
    width:250,
    borderWidth:1,
    borderRadius: 10,
    });

win.addEventListener('click',function(e){
win.close(mapview)    
});    

  win.add(mapview);

inline code

I hope somebody can give me some advise or code with the solution.

Thank you very much!!

  • This is happening because you are closing the window with win.close(). The close method does not take any parameters. It just closes the window which you've called it on.
    You need to either open the mapview in a new window and close that window, or just remove the mapview from the current window. ie: win.remove(mapview)
