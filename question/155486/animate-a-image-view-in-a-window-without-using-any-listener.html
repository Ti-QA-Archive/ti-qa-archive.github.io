  • You can create an animation object to animate imageview. There is a method given in imageview to set animation object. For Ex.

    var an1 = Titanium.UI.createAnimation();
an1.bottom = 150;
an1.duration = 800;
imageview.animate(an1);

var a2 = Titanium.UI.createAnimation();
a2.bottom = 80;
a2.duration = 800;

an1.addEventListener('complete', function()    {
    imageview.animate(a2);
});

a2.addEventListener('complete', function()    {
    imageview.animate(a1);
});
