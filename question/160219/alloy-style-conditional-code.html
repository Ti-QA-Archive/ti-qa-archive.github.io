    Hello Nico,

    Check out these two related pages that detail conversations on the subject

    1. Custom TSS device queries
    2. Extend custom formFactor

    Expect a near future Alloy revision to provide this logic.

    you can manage in JS temporary like :

    
Alloy.Globals.isIphone5 = (Ti.Platform.osname == 'iphone' && Ti.Platform.displayCaps.platformHeight == 568) ? true : false;

    Use of it for ie.:

    
var view = Ti.UI.createView({
    height :  '100dp',
    width : '150dp',
    backgroundColor : 'red'
}):


if(Alloy.Globals.isIphone5)
{
    view.height = '150dp';
}
