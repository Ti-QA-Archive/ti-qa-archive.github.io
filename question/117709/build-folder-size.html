  • Tjeu

    I think 70MB is huge, bearing in mind that the default app is under 6MB.

    Remember that anything in the Resources folder will be included in the build. Are you sure you haven't inadvertently placed unnecessary images (such as .psd or .xcf files) in Resources?

    The approach I use is to create a folder named source in the root of the project where I keep all my working files, and then move only the resulting, optimized (using pngcrush), .png files into the Resources folder.

    Hope this helps
