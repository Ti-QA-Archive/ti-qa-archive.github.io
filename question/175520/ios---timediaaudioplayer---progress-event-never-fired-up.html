  • Hello Shawn Lan,

    What iOS/Titanium SDK do you use?

    The following code works testing on an iPhone simulator:

    var win = Ti.UI.createWindow({
    backgroundColor: '#fff'
});

var audioPlayer = Ti.Media.createAudioPlayer({
    url: "<Your audio>"
});
audioPlayer.addEventListener('progress',function(e) {
    Ti.API.info('Time Played: ' + Math.round(e.progress) + ' milliseconds');
});

var btn = Ti.UI.createButton({
    title:"Play/Pause",
    width: 200,
    height:20
});
btn.addEventListener("click", function(e) {
    if(audioPlayer.playing) {
        audioPlayer.pause();
    } else {
        audioPlayer.play();
    }
});
win.add(btn);
win.open();

    Outputs:

    [INFO] :   Time Played: 617 milliseconds
[INFO] :   Time Played: 1617 milliseconds
[INFO] :   Time Played: 2617 milliseconds
[INFO] :   Time Played: 3617 milliseconds
[INFO] :   Time Played: 4617 milliseconds
[INFO] :   Time Played: 5617 milliseconds
[INFO] :   Time Played: 6616 milliseconds

    Hope this solves your problem.

    Sincerely,
    Patrick van Vuuren
