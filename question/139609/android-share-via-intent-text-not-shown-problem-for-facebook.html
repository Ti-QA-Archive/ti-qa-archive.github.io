Android Share via Intent Text not Shown Problem for Facebook

I have implemented the intent for sharing text in Android. The text that being passed can be displayed for all other apps (eg. gmail, twitter, etc) except facebook. There is no text being displayed in the facebook dialog. My facebook app has already been updated to the latest version. Does anyone know what is the problem?

The piece of code that I am using is:

var intent = Ti.Android.createIntent({
   action: Ti.Android.ACTION_SEND,
   type: 'text/plain'
});
intent.putExtra(Ti.Android.EXTRA_TEXT, 'Hello world!');
Ti.Android.currentActivity.startActivity(intent);

BTW, except text, how can I pass the redirect URL and image to the intent? I am not yet able to test due to the above problem, but I guess it should use putExtraUri and EXTRA_STREAM.

Testing envrionment:

  • Platform: Android 2.3 & 4.0
  • Titanium SDK: 1.7.6, 2.0.1GA & 2.1.0GA
