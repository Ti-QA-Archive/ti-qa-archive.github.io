Desktop Titlebars

I'm trying to make the titlebar of a desktop app a different color and layout, but I can't:

  1. Get the background color behind rounded corners to be transparent
  2. Allow the window to be dragged when &lt;chrome&gt;false&lt;&#x2F;chrome&gt; is used in tiapp.xml

so how do I get the background to be completely transparent and not just white and how do I get the window to be draggable?

  • You want to use the following in your windows definition in tiapp.xml:

    <transparent-background>true</transparent-background>

    And to move the window around, use the code described in the tiapp.xml documentation:

    var dragging = false;
document.onmousemove = function () {
    if (!dragging) return;

    Titanium.UI.currentWindow.setX(
        Titanium.UI.currentWindow.getX()
        + event.clientX - xstart
    );

    Titanium.UI.currentWindow.setY(
        Titanium.UI.currentWindow.getY()
        + event.clientY - ystart
    );

}

document.onmousedown = function () {
    dragging = true;
    xstart = event.clientX;
    ystart = event.clientY;
}

document.onmouseup = function () {
    dragging = false;
}

    If you don't want to be able to move the window by dragging anywhere, attach the event listeners to the html element you use for the title bar instead of document.
