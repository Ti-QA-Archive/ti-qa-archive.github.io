  • In the PlayListFile.xml first, correct the ending node </FileNam>. It should be </FileName>. And, now before parsing the string into xml document, replace the & character with '&amp;'. File will parsed successfully. :)

    var xmlMessage = blob.text;
    xmlMessage = xmlMessage.replace(/&/g, '&amp;'); // add this line before parsing
    var xmldata = Titanium.XML.parseString(xmlMessage);
