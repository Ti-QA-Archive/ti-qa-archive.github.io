Does audio stream work in a desktop app?

I am trying to stream an audio in a desktop application but it doesn't work

<audio controls preload="auto" autobuffer> 
  <source src="http://xxx/mpd" />
  <source src="http://xxx/mpd.ogg" />
  <!-- now include flash fall back -->
</audio>

while a single file works fine

<audio controls preload="auto" autobuffer> 
  <source src="http://www.jplayer.org/audio/m4a/TSP-01-Cro_magnon_man.m4a" />
  <source src="http://www.jplayer.org/audio/ogg/TSP-01-Cro_magnon_man.ogg" />
  <!-- now include flash fall back -->
</audio>

I found no streaming api for the desktop app.
I have Titanium Studio Community, build: 1.0.6.201110251616.

Am I doing something wrong?

thanks!

  • I dropped your code directly into the default desktop project and the audio seems to work fine for me. I am also using Titanium Studio 1.0.6.201110251616 and am using the Titanium Desktop SDK 1.2 RC6. 

    <html>
    <head>
    </head>
    <body style="background-color:#1c1c1c;margin:0">
        <div style="border-top:1px solid #404040">
            <div style="color:#fff;;padding:10px">
                Welcome to Titanium
            </div>
            <audio controls preload="auto" autobuffer> 
              <source src="http://www.jplayer.org/audio/m4a/TSP-01-Cro_magnon_man.m4a" />
              <source src="http://www.jplayer.org/audio/ogg/TSP-01-Cro_magnon_man.ogg" />
              <!-- now include flash fall back -->
            </audio>
        </div>
    </body>
</html>
