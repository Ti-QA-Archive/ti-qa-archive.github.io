Where are Android databases stored?

Where are databases stored for Android on a mac? I'm trying to update a database but every time I clear the build files the database isn't updated to what's in the resource dir.

I've been searching but can't find the location. I've tried alerting the Ti.Filesystem.applicationDirectory result but that just gives me 'appdata-private://'

    This is more complicated

    You will have to open ddms, select the running emulator then go to the menu Device->File explorer

    browse to:

    data/data/your_app_id/databases/your_database

    select that file and then in the top-left of the file explorer window click on the first icon. It will ask you where to save the file.

    After this u will be able to open it with whatever u want to.

    here you can find your ddms

    \android-sdk-windows\android-sdk-windows\tools
