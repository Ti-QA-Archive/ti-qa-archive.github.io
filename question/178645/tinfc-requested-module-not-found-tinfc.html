ti.nfc: Requested module not found: ti.nfc

Hi,

I am trying to use the ti.nfc module in SDK: 3.4.0.GA and I get:

[ERROR] TiExceptionHandler: (main) [128,93754] ----- Titanium Javascript Runtime Error -----
[ERROR] TiExceptionHandler: (main) [0,93754] - In ti:/module.js:280,9
[ERROR] TiExceptionHandler: (main) [0,93754] - Message: Uncaught Error: Requested module not found: ti.nfc
[ERROR] TiExceptionHandler: (main) [0,93754] - Source:         throw new Error("Requested module not found: " + request);
[ERROR] V8Exception: Exception occurred at ti:/module.js:280: Uncaught Error: Requested module not found: ti.nfc

My understanding was that this was included with the sdk. Do I need to add something to my tiapp.xml?

  • ti.nfc is not included per default. You have to download this module here:

    https://github.com/appcelerator-modules/ti.nfc/tree/master/android/dist

    Unpack it in your module folder (e.g. ~/Library/Application Support/Titanium/modules/android/) or import it through Titanium Studio (Help => Install Mobile Module) and reference it in tiapp.xml

    <modules>
    <module platform="android">ti.nfc</module>
</modules>

    Don't forget to clean your project afterwards.
