Facebook Request with Graph API

Hy,
I'm trying to add an annotation to a mapview using some information taken from an user account after his login.
In more detail: a user make the login, than I want to extract his account information (first name, last name and picture) ancd use them in an annotation.

I tried using this code:


var user1 = Titanium.Map.createAnnotation({
    latitude:43.798145,
    longitude:11.253498,
    title:Titanium.Facebook.requestWithGraphPath('me/name', {}, 'GET', function(e) {
    if (e.success) {
        alert(e.result);
    } else if (e.error) {
        alert(e.error);
    } else {
        alert('Unknown response');
    }
    }),

    animate:true,
    draggable:true,
    image:Titanium.Facebook.requestWithGraphPath('me/picture', {}, 'GET', function(e) {
    if (e.success) {
        alert(e.result);
    } else if (e.error) {
        alert(e.error);
    } else {
        alert('Unknown response');
    }
})
});


mapview.addAnnotation(user1);

but I have this alerts: 


(KrollRuntimeThread) [4868,9851] {"error":{"message":"Unknown path components: \/name","type":"OAuthException","code":2500}}

and:


I/ALERT   (  371): (KrollRuntimeThread) [23,5017] {"error":{"message":"An active access token must be used to query information about the current user.","type":"OAuthException","code":2500}}

How can I resolve this? I thought that information like first and last name didn't need an access token.

Help!!!! :D
