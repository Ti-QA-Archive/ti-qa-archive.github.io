How to save camera orientation on success

According to the documentation, there is a transform key in the dictionary for the camera.

I couldn't get the transform key to do anything when I tried rotating a 2D matrix, but the following code seems to handle camera orientation perfectly in the success function:

success:function() {
  var imageFile = event.media;
  var imageAsTaken = Ti.UI.createImageView({
    image: imageFile
  });
  imageFile = imageAsTaken.toImage();
  // do other processing...
},

I would love to know how this works.
