iPhone Navigation Group sliding direction

Hi,
Does anyone know if there is a way to change the direction of the sliding effect of the navigation group control? I need to have it slide to the right on one option and slide to the left on a different option.

Thanks

— asked October 25th 2011 by Ophir 'Shlomi' Oren

  • I believe the direction of transitions in navigation groups is controlled by the nav group controller itself and cannot be changed. If you need to control the way windows open, you can use the open() function directly on the window itself and assign a transition property. For example:

    win.open({transition:Ti.UI.iPhone.AnimationStyle.FLIP_FROM_LEFT});

  • // New window (homeWin) open at 320 px and move to 0
homeWin.left = 320
var slide_it_left = Titanium.UI.createAnimation();
slide_it_left.left = 0;
slide_it_left.duration = 300;
homeWin.open(slide_it_left);

// Existing window move from 0 to -320 / -640
var slide_it_right = Titanium.UI.createAnimation();
slide_it_right.left = -320; // or -640
slide_it_right.duration = 300;
win.animate(slide_it_right); // Existing showing window.
