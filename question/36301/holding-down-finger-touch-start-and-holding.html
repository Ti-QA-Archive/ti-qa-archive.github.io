Holding down finger (touch start and holding)

Is there any way to find out if the person is holding down the button?

There is touchstart & touchmove but I cant find the right combo to do this.

EXP: while holding down the button a action will happen (realtime)?

  • Start a timer when the touchstart event triggers, and if touchend does not appear in X time ( you decide how much ) start your action.

    setTimeout is a little buggy so you might need to use this

    http://developer.appcelerator.com/question/1881/please-help-on-stopping-the-settimeout

    to solve this issue.
