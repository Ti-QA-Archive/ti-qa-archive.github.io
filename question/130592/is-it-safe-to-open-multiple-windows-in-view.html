Is it safe to open multiple windows in view?

Just wondering… Is it safe to open more than one window at the same time? I'm thinking of having 2 windows open, each covering a certain part of the screen. Is this safe? or should I only really be using one window covering 100% of the screen?

  • Windows always cover all screen (at least on Android). You can have multiple screens open, but in your case you should use View components.

    • Yeah, on Android you can't resize windows, so for cross-platform solution using Views is a way to go. Create Window with "horizontal" layout, and create two Views that have width equal to "50%" and add them to Window. That way they will be positioned one beside other and will take all screen size.
