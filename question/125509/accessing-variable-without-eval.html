Accessing Variable Without Eval

Hi.

I am using Titanium Studio (latest) SDK 1.8 and developing for iPhone and Android.

The issue I have is more a JavaScript Question I think.

I develop a window and assign a set of views to variables FS1 FS2 FS3 etc
in a separate function I set a parameter of 1 2 or 3 depending on a button press.

Everything works great. If I reload the screen the parameter is still available but the variable containing the view has changed still same name.

So I call a function to do something with this as a parameter

eval("FS" + varNum) // varNum is either 1 2 or 3

is there a way passing the content of the variable FS ? 1 2 or 3 without using eval.

Hope this makes sense.

Thanks

T.

    Since your setting the variables with the knowledge of the structure, you should try setting them as an array or object so that you can name them similarly. ie:

    FS[1] = var1;
FS[2] = var2;
FS[3] = var3;

    This way all you need to do is pass the varNum so that you can access what you need via a statement like:

    val = FS[varNum];

    make sense??
