Android: instanceof not working in 1.5 when assigning an Array to a View?

Hi all,

since Titanium 1.5 I run into a strange issue. As soon as I am assigning an Array to a previous created Ti.Ui.View the array will no longer be an instanceof Array:

var x = [];
Log(typeof x);
Log(x instanceof Array);
var xView = Titanium.UI.createView();
xView.x = x;
Log(typeof xView.x);
Log(xView.x instanceof Array);
// [WARN] [10,12458] object
// [WARN] [20,12478] "true"
// [WARN] [23,12501] object
// [WARN] [16,12517] "false"

The 'Log' function is a custom Log function I use for logging.

It works in 1.4.3 but neither in 1.5.0 nor in 1.5.1
