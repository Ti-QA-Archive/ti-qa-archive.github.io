    • When you click the back button while the keyboard is visible, the keyboard will be closed. I don't think there is a way to customize this behavior.

      If you click the back button once more after you have closed the keyboard, you can customize the result because the event is now attached to the window.

      Note that closing the keyboard using the back button works fine on my system.
