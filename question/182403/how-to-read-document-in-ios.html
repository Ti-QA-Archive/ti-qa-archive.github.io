  • Where this test.pdf resides? Try to access it using Ti.Filesystem namespace and pass nativePath to iOSDocument Viewer.
    Below is the sample code,

    
var winButton = Titanium.UI.createButton({
    title:'Launch',
        height:40,
        width:200,
        top:270
});
$.annoucementWin.add(winButton);

var file = Ti.Filesystem.getFile(Ti.Filesystem.getApplicationDataDirectory(), 'test.pdf');

if(file.exists()) {
    var docViewer = Ti.UI.iOS.createDocumentViewer({url :  file.nativePath });
}

winButton.addEventListener('click', function(){
    docViewer.show();
});

    Thanks,

    Narendra Raorane.
