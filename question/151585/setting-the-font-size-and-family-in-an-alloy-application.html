  • On style.tss you should be able to apply to Labels for instance:

    "#headerBold": {
    top: 5,
    left: 10,
    font: {
        fontFamily: "Roboto-Bold",
        fontSize: '14dp',
        fontWeight: "bold"
    },
   color: "#A4A4A4"        
},

"#headerRegular": {
    top: 0,
    left: 10,
    font: {
        fontFamily: "Roboto-Regular",
        fontSize: '14dp',
        fontWeight: "lighter"
    },
   color: "#848484"        
},

    Where Roboto-Bold.ttf & Roboto-Regular.ttf can be under the assets folder. Haven't tried on a SearchBar yet though.

    Hope it helps. Best regards.
