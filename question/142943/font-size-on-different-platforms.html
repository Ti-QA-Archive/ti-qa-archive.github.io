Font size on different platforms

Titanium sdk: 2.1.1
os: mac os
platform: android, ios
the same font size behaves differently in different platfroms, it seems that the font is bigger than the one on android. on both platforms, i use the 'system' as the unit measurement

font:{
    fontSize:16
}

    The only way to be sure with fonts is to use sp (which is like dp, but for fonts).

    font:{
    fontSize: '16sp'
}
