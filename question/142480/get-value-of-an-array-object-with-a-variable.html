Get value of an array object with a variable

How can I get the value of an object in an array with a dynamic variable. I would imagine this would workbut it returns null. I think it's just syntax that I have wrong but I'm stuck.

for (var i = 0; i < testarray.length; i++){

    //The _id is passed to the window from the previous window.  
    //This should equal to something like text1, text2, and so on so  I can use it in the array query below.
    var dynamicvalue = 'text' + _id;

    //The _id value comes through but when I put the dynamicvalue it doesn't work            
    var getthevalue = testarray[i].dynamicvalue;

}

    This is really hard to tell since you haven't provided the original array, but you can try something like this:

    var val = testarray[i][dynamicvalue];
