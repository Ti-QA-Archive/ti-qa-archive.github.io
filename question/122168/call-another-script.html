    Same way you would in any other javascript application. Load script resource. On the click of your button, run the init method of your script.

    • Ti.include('myscript.js');

var win1 = Ti.UI.createWindow({/*args*/});
var btn1 = Ti.IU.createButton({title:'Click Me'});
btn1.addEventListener('click', function() {
    myscript.myInitMethod();
});
win1.add(btn1);
win1.open();

      There are literally 20 different ways to do this in any given app, but this sounds like what you want to do.
