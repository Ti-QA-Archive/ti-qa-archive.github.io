  • This usually happens if you previosly did a device build and did not use any animations.
    Now when you run in simulator, Ti does not performing a full rebuild (to startup faster), therefore the module is not included.
    When performing a clean full rebuild on simulator there should be all modules included, no matter if you used them in your code or not.
