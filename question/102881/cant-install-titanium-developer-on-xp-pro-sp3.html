Can't install Titanium Developer on XP Pro SP3

Hello, Just downloaded the Titanium Developer MSI and tried to run it on my system, and I get the error "Could not query info: Invalid HTTP Status Code (0)"

I see a similar thread but for Windows7, and saw in that thread a recommendation to another member to create their own thread specific to their situation, so that's what I'm doing :)

I'm running Windows XP Pro, Service Pack 3,
I have AVG 2011, and tried installing the package with and without Antivirus and Firewall on. With the same results.
UPDATE: Forgot to mention, I'm on a home network with a router between my machine and the DSL modem… if that would make any difference.

Any help would be very much appreciated.
Thanks in advance,
Tony

  • I fixed it for my situation! On a hunch I went fired up IE to check my settings (I use Firefox as a standard). I almost never use it, and in this case must not have used it for a while because checking my network settings showed it set for a Manual Proxy setup. I removed this setting, re-tried the installation and it worked.

    Hope this helps someone else, but your mileage may vary.
