Titanium Trouble to drag a view with touchmove programmatically

Hi. Developers.

I am going to implement drag and drop with touchmove event in Titanium.

This my code.

front_view.addEventListener('touchmove', function(e) {
        tmp_item.width = 250;
        tmp_item.center = {
            x : e.x ,
            y : e.y  + 70
});

front_view is a background view.

I did it like this because it must be.

But it doesn't work on my phone.

tmp_item always appear at the position that is double to real touch position.

For example when I touch (100,100) it appears at (200,200).

The reason is that touch coordinate differs to view coordinate.

It works fine at my tablet but on phone.

I think there must be a translation from touch position.

Please help me.

Regards. Wang

    Hi Wang Dan,

    try touchmove event of tmp_item view with converting touch point into it's parent view.

    e.g

    var mainView = Ti.UI.createView({
    height : "100%",
    width : "100%"
});
var tempView = Ti.UI.createView({
    width : 150,
    height : 150,
    backgroundColor : "red"
});
mainView.add(tempView);

tempView.addEventListener('touchmove', function(e) {
    var point = tempView.convertPointToView({
        x : e.x,
        y : e.y
    }, mainView);
    tempView.center = point;
});
$.winIndex.add(mainView);

    Hop this is helps you..

    Thanks
