  • Hi Tracy

    I'm afraid it's impossible to troubleshoot an issue like yours without understanding your environment. So that someone can help you, copy and paste the following into your terminal/shell so that all the commands execute successfully. Copy the output and paste it to your preferred pastebin service. It's important that you do not modify it in any way, as this could mask problems with your configuration. Do NOT paste it here! This is because the system will change the formatting.

    clear
cd /
uname -a
echo "$PATH"
echo "$JAVA_HOME"
java -version
javac -version
python -V
echo "$android_sdk"
adb version
android list target
git --version
echo " --- The following is optional ---"
scons -v
echo "$HTTP_PROXY"
echo "$HTTPS_PROXY"
echo "$SOCKS_PROXY"

    Once you have provided this, someone will assist you further.

    Cheers
