get the content width of a scrollView

Hey, How would I get the current width (or height, i'm using a horizontal scrollView) of the content in a scroll view? I've tried using scrollView.contentWidth but this gets the value that has actually been defined when creating the view, which is just undefined.


var scrollView = Ti.UI.scrollView({ width:"100%", layout:"horizontal" });

// add all content to the scroll view here

var scrollWidth = scrollView.contentWidth; // returns 'undefined'

I've also tried scrollView.getContentWidth(); which also returns undefined.

    hi,

    try to set contentWidth:'auto' then after get the width of the scrollview by 

    scrollView.toImage().width;

    it gives the total length of the scrollview.

    use this code after all the content are added.
