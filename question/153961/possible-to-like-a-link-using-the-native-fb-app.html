In my app, I have a few links available for my users to like, however with my implementation the user has to login to facebook from inside my app. THis is too tedious for an average user to do a simple like.

Is it possible for a user to be switched to the native FB app so they can do the like from in there (where they are already logged into presumably) ?

    Hi

    If you mean using the built-in iOS support for Facebook, Twitter and Sina Weibo (very popular in China) - then yes.

    Check out this easy to use module that gives you very fast and easy access to the "Share Sheets" as they are referred to as.

    • TiSocial Framework

    These hook into the iOS facebook and twitter etc support, if the user is logged in they do nothing extra other than confirm or tweak the message before it is sent.
