  • Hello Ehsan,

    Unfortunately, it seems to be a native issue (should I say bug?) which is filed here:

    Google Issue: https://code.google.com/p/android/issues/detail?id=17562

    Rationale explained: 

    It works well on a Wifi connection but on 3G it says : 

ERROR/NoHttpResponseException(5195): org.apache.http.NoHttpResponseException: The target server failed to respond

    You may find it duplicated and a work around posted here.

    Everybody is welcome to build the Titanium core from source using Titanium Advanced Tools.

    I hope inputs above provides you insight enough in order to get over the problem quicker, thanks.
