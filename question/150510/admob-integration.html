  • Hi deepthi

    first of all you new to create an google account:

    http://www.admob.com/login/

    then u have to configure for iphone or android addvertisment. from account you got a unquie code for iphone and android for both different.
    And put that code into your application logic code:

    var Admob = require('ti.admob');

 var addView = Admob.createView({
            width : 320,
            bottom : 0 ,
            height : 50,
            publisherId: 'a150eab012a3a05', 
               backgroundColor: 'black',
               testing: false,
                left:0,
             // location:e.coords,

});
win.add(addView1);

    you will get the google add in your mobile
