Wait process

I need to make a program that prints multiple pages. (Bluetooth Printer) The problem is that I need the program to stop on each page and ask if it was printed successfully. What is the best way to program this?. I tried to use a for loop (per page) but show the question once and the sending to printer code execute "n" times (same as pages).

  • A loop will run through yes. You need to process each page as a separate entity. so print one. ask the question. if ok print 2. if not go back and print 1 etc.

    You can do this by either recursive functions or better still. would be to have a function which when the first page prints calls the question function and then that calls the print function or reprint.
