Any way to remove Plus/Minus zoom tool that pops up in Android Map View when pinching?

I am using a simple Map.View in an iOS/Android application, and when I pinch to zoom on Android, it pops up this ugly little gray tool with a magnifying glass and +/-, which provide a second way to zoom in/out.
