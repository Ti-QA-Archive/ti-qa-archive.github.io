  • Ashish

    Just so that you can see it working, put the html file in the same directory as the js file that creates the window:

    url:'index.html'

    Paths on android and iOS are currently not consistent due to some outstanding bugs. Therefore, try absolute paths:

    url:'/main_windows/index.html'

    (where the first forward slash is equivalent to the Resources folder)

    or

    url:'app://main_windows/index.html'

    (where the app:&#x2F;&#x2F; is equivalent to the Resources folder)
