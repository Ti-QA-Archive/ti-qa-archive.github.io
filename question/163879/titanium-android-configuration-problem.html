Titanium Android Configuration problem

Hello,

I am trying to configure Titanium installed with Android SDK.

I have Windows 7 (64 bit) installed on my machine with Titanium build 3.2.1.201402041146 and Titanium SDK 3.2.2 GA installed.

I have downloaded and installed Android SDK tools 22.6.1, Android SDK platform tools 19.0.1, Android SDK Build Tools 19.0.3 and 19.0.1 and Android 4.4.2 (API Level 19)

I installed Titanium Eclipse Development plugins and ADT plugin as well. under Titanium Window -> Preferences -> Studio -> Platforms -> Android and selected the Android SDK path. Now next I am trying to set default Android Emulator but problem is it doesn't allow me to select my custom android device which I created via AVD and keeps showing all standard device definitions and configurations

Also when I try to run the app via one of the standard deinition emulators, I end up with this error:

" --skip-js-minify was unexpected at this time.
C:\Workspace\Titanium_Studio_Workspace\GuestApp>  "C:\Users\ssangani\AppData\Roaming\npm\\node.exe"  "C:\Users\ssangani\AppData\Roaming\npm\\node_modules\titanium\bin\titanium" --no-colors --no-progress-bars --no-prompt build --platform android --sdk 3.2.2.GA --log-level trace --target emulator --android-sdk "C:\Program Files (x86)\Android\android-sdk" --device-id "7" WSVGA (Tablet)" --skip-js-minify
[ERROR] Application Installer abnormal process termination. Process exit value was 255

I guessed maybe it couldn't locate nodejs so I checked Titanium's path for node.exe and it was correctly set at C:\Program Files (x86)\nodejs

Does anyone know how to solve this?
