Android App works in emulator, fails on device

SDK v. 1.7.5, target Android 2.1, Titanium build 1.0.6.201110251616
I created the initial "hello world" app with 2 tabs and 2 windows. Works just fine in the Android emulator, but when I download to my Motorola Droid Bionic, running 2.3.4, it says the process has stopped unexpectedly, please try again. This is immediately after displaying the red "powered by Titanium" screen.
I apologize if this is a RTFM question - I cannot find any docs on how to debug on the device itself, and setting breakpoints in the js code doesn't seem to make much sense since it works on the emulator.
