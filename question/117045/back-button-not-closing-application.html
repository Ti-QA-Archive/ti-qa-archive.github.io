Back button not closing application

I created a application, which requires login, user can do some actions in different windows after login, at any point they can click the logout through android menu which shows login again.

Question is on click of back button after logout, I need to close the application from the login page(current page), but its going to previously opened page, can anybody help me in fixing this

I am giving navBarHidden:true in all pages except while opening login page

  • On any window you want to exit when the back button is used (typical of a login type screen), simply add this when creating the window:

    exitOnClose:true

    It is ignored for iOS and applies to android only.
