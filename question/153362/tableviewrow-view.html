  • what Nick said is correct, even if sometimes I create little shortcuts to edit a specific object of a row:

    
var myIcon = Ti.UI.createImageView({image:"on.png"});
row.add(myIcon)

row.myIcon = myIcon; //this is the shortcut

row.addEventListener("click", function(e) {
    e.rowData.myIcon.image = "off.png";
});

    this code will change the icon on the row without too much hassle.
