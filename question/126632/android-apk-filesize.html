    The app file contains a javascript runtime which titanium uses to work on the phone.

    I am guessing that you have created a development app, creating a release app produces a smaller file, but it will still be a reasonable size as part of the framework needs to be included.

    The advantage is it doesn't get bigger quickly.

    Hope this explains it.
