  • I had this error, I understood that we can't add an UI item to more than ONE view / window.

    But in your code, I don't see that you add one button to two view… But you add a lot of time (if you tap a lot on the buttons) the same button, maybe the error is there. You may try to add a property "created" to the button you add a lot of time, and do a condition that if created is false you add the button to the view, and if true you only show it…
