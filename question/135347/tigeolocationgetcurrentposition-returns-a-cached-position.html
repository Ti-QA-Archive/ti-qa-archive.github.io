Ti.Geolocation.getCurrentPosition returns a cached position

Hi guys,

As it reads in the title the Ti.Geolocation.getCurrentPosition() is always returning a cache position, it isn't getting my current position. Is there a workaround for it?

Thanks!

    This is what getCurrentPosition does.
    To get the real location you will have to start the gps by attaching a function to the Titanium.Geolocation.location event.

    When this triggers you will get the real position and you can stop the gps by removing the event listener.
    Take care that the event can trigger after a long time (gps needs to lock a position and depending on where the user is, this can take lots of seconds)
