Is it possible to separate (vertically) the tableViewRows in a TableView?

Hi, TableViewRows are separated by a line(separator), is it possible to increase the size of that separator? and put it in like dp or something? I wanna create a 5 o 7 dp margin beetween the tableviewrows! Im using titanium classic, and platform: android. Thanks!

    Hello,

    You can't set line(separator) height bcz it's table property, for your issue you have to set custom…
    first you have to do separatorColor : 'transparent' of tableView.

    
var win = Ti.UI.createWindow({
    layout : 'vertical',
    backgroundColor : 'white'
});

var data = [];

var table = Ti.UI.createTableView({
    height : 450,
    layout : 'vertical',
    separatorColor : 'transparent'
});

var footerView = Ti.UI.createView({
    height : '1',
    backgroundColor : 'transparent'
});

table.footerView = footerView;

for(i=0; i<7; i++){
    var row = Ti.UI.createTableViewRow({
        height : 50,
        title : "Row " + i
    });

    var sepLine = Ti.UI.createView({
        left : 0,
        right : 0,
        height : 5,
        bottom : 0,
        borderRadius : 2,
        backgroundColor : 'black'
    });
    row.add(sepLine);
    data.push(row);
}
table.setData(data);

win.add(table);
win.open();
