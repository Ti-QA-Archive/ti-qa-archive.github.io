    I am confused by this question as I am not sure if you understand what a web sever is and as such how you think it will "add a web server".

    A web server is just a server sitting anywhere on the Internet serving web pages and other documents. You do not add this to an app.

    You can have an app download any file or access any URL from that web server.

      • the type of data you are storing?
      • how many records the data will be to start with?
      • how many records will be added over time?
      • how many devices running the app will be connecting at the same time?

      • is the data dynamic - changed all the time?
