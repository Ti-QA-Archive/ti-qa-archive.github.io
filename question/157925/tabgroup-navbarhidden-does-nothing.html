TabGroup navBarHidden does nothing

I can really get some guidance here. I'm trying to hide the navbar from the tabgroup but it's not working. Code below:

var win = Ti.UI.createWindow({
    title: 'My Window'
});

var nav = Ti.UI.iOS.createNavigationWindow({ window: win});

var tabGroup = Ti.UI.createTabGroup({
    navBarHidden: true // <-- this does nothing...
});

nav.open();
nav.openWindow(tabGroup);

Thanks.

  • Make the window navBar hidden not the tab.Something like this should work

    var tabGroup = Ti.UI.createTabGroup({
    navBarHidden: true // <-- this does nothing...
});
var win = Ti.UI.createWindow({
    title: 'My Window',
    navBarHidden:true
});
var tab=Ti.UI.createTab({
window:win
});
tabGroup.add(tab);
tabGroup.open();

    Thanks
