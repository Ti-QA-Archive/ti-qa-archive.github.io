    Hello Chris.

    What you have is XML string that needs to be parsed and converted to XML document. To do that, use "Ti.XML.parseString" method which returns XML document. On that document you can use standard DOM 2 methods to extract data you need.

    var xml_doc = Ti.XML.parseString(...);
var names = xml_doc.getElementsByTagName('Name');
for (var i = 0; i < names.length; ++i) {
    Ti.API.info(names.items(i).text); //something like this
}
