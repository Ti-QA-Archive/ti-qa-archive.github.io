Titanium Compatibility Matrix

The compatibility matrix states that SDK 3.1.0 requires Xcode 4.6 (max version). While Xcode 5.0 is supported only by SDK 3.1.3 or later. Does this means that if I upgrade Xcode from 4.6 (my current version) to 5.0 I will not able to use older SDK? I have some apps that still use SDK 3.1.0 or 2.1.3 and I do not want to use a different SDK for them.

— asked October 23rd 2013 by alessandro la rocca

  • Hello,

    You can keep both 4.6 and 5 installed, and with xcode-select, you can go from 4 to 5 and back. It's what I do for some projects. After you do xcode-select, you will need to go to the Titanium Studio, select preferences, select iOS and then press the button "select default" and save. Then you can use 4.x or 5, as you want.

    Best,

    Mauro

  • Hi Alessandro,

    SDK 3.1.3 is basically to support iOS7 which is in XCode5. You can still build apps using SDK 3.1.0 or 2.1.3 with Xcode 5. But you can't use both Xcode at the same time.
