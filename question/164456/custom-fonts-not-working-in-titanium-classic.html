Custom fonts not working in Titanium Classic

I have copied the font named "SourceSansPro" to my resources folder, I even created a folders called fonts and copied it there and in addition I also copied it to resources->iphone->fonts.

Here is the font info from font book:

PostScript name SourceSansPro-Regular
Full name Source Sans Pro
Family Source Sans Pro

in my code I have used every variation of the name:

var testLabel = Titanium.UI.createLabel({
width:'auto',
left: '100px',
text: 'test',
color: '#ffffff',
font:{
fontSize:20,
fontFamily: 'SourceSansPro-Regular',
},
});

var testLabel = Titanium.UI.createLabel({
width:'auto',
left: '100px',
text: 'test',
color: '#ffffff',
font:{
fontSize:20,
fontFamily: 'Source Sans Pro',
},
});

It just uses the default font everytime. I've tried other fonts as well and nothing seems to be working. I've tried about 10 different tutorials and all seem to fail. Please help!

  • Hi,

    Have you also made you you added a reference ot the font in tiapp.xml ? 

       <ios>
        <min-ios-ver>6.0</min-ios-ver>
        <plist>
            <dict>
        <key>UIAppFonts</key>
        <array>
            <string>/SourceSansPro.ttf</string>
        </array>
            </dict>
        </plist>
    </ios>

    Hope this helps
