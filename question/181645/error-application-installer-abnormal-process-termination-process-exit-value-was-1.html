[ERROR] Application Installer abnormal process termination. Process exit value was 1

I have a project that was worked perfectly but now is not without changing any things

The other projects in my workspace work fine.

When I run the project it give me this error:

I try that but without result:

  • clean the project and delete the build folder
  • Restart my pc, and Titanium Studio
  • I reconfigure the android SDK and Titanium SDK with the command Titanium setup

Is there any other things that I ca do? How can I trace the log File in order to get the error of the project?

  • My guess is your running this on an emulator and not pushing it to an android phone. The emulator does take a long time to load and build and if you don't unlock it as soon as you can it will fail with that message too.

    Install genymotion if you really want to use an emulator. Otherwise use a real device or try a few times with the default emulator… it will give you that error once in awhile though….its an android thing.
