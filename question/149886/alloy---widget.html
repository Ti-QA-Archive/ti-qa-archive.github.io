Alloy - widget

In traditional Titanium without Alloy, I create a widget with dynamic properties as follows

optionDialog

var _headerTitle = params.title, _options = params.options, _MAX_OPTION_NUM = 6.5, _TITLE_H = _headerTitle ? 50 : 16, _BOTTOM_H = 70, _TABLE_H = Math.min(_MAX_OPTION_NUM, _options.length) * 44, _MARGIN = 10, _key = (params.options && params.options.length > 0) ? params.options[0] : "_key_", _win = null;

var _w = 290, _h = _TITLE_H + _BOTTOM_H + _TABLE_H;


var vWrap = Ti.UI.createView({
            width : _w, 
            height : _h, //will vary depending on the number of options
            borderColor : "#535353",
            borderWidth : 1,
            borderRadius : 10,
            backgroundColor : "#dddddd",
            zIndex : 200
});

With Alloy, to build widget and achieve the same thing as above, how can I do it?

"#container" : {
    width : 200, //how? (As a matter of fact, I can define _w in Alloy.Globals, but I do not think it is a good practice)
    height : 150,
    borderColor : "#535353",
    borderWidth : 1,
    borderRadius : 10,
    backgroundColor : "#dddddd",
    zIndex : 200,
    visible : true
},
