getting (x,y) coordinate from image

using SDK 2.0 android

hi all, i wanna get the coordinate of the clicked pixel in an imageview (the image view is in a scrollView). the image has the size of 640*640 (larger than the screen size),so it must be scrollable . when clicking on a pixel, i got the coordinate like this –> (image.x, screen.y) or ( screen.x,image.y). But what im searching for is like this –> (image.x, image.y).
i think that this problem exist because android can't support scrolling horizontally and vertically at the same time.

so how to get the coordinate of a pixel from this image?!
