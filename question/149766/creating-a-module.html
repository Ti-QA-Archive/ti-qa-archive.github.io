Creating a module

I am trying to create my first module, with the instructions here:

iOS_Module_Development_Guide

Unfortunately something seems to have changed with the CLI scripts, as the parameters in the guide does not seem to create a module framework, only a normal project.

Titanium Command-Line Interface, CLI version 3.0.24, Titanium SDK version 3.0.2.GA

I run 

titanium create --platform=iphone --type=module --dir=/tmp/ --name=test --id=com.example.test

…and is sent into an interactive mode, producing only a normal project.
