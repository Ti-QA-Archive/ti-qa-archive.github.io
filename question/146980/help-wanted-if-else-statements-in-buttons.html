    Hi

    Your event listener for the button click is the problem and the code has commas at the end which is not correct.

    This version will solve your problem and give you a cleaner code sample so you can re-use the model.

    button.addEventListener('click',function (e) {
    if (button.getTitle() === 'Show') {
        label.setText('Basic info about me');
        button.setTitle('Hide');
    } else {
        label.setText('');
        button.setTitle('Show');
    }
});

    This uses the title of the button to determine the toggle state for you and uses the method "setters" to change the values once you have created them.
