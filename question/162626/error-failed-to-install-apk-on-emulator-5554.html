The emulator starts up and I can sign in, but at some point it just throws the error message and then nothing happens after that.

Am I missing something here?

  • The log thrown by Android seems to evidence a timeout error the solution is just increasing the connection timeout for ADB.

    To increase this value you may want to go to the Android Monitor tool (inside /tools directory of your android-sdk folder) then open Preferences > Android > DDMS > ADB Connection time out (ms) > Set a value slightly higher than current one.

    E.G. DDMS

    There is native solution highlighted as the accepted answer here:

    • http://stackoverflow.com/questions/15677640/error-failed-to-install-apk-on-device-emulator-5554-timeout

    Hope it helps.
