    Nicola,

    You cannot modify the connection parameters from Titanium. You would need to write a native module to make the call outs to the OS (iOS or Android).

    I like the idea. Walk into a location, hit up the QR code with network information embedded in it, and connect. No random drive by WiFi stealing, have to be in the location to hit the QR code, yet have an "open" access for clients/customers.
