  • Hello, Carlos, Did you installed Xcode6 in your Mac replacing the Previous Xcode5? If so then you need to Run this Configuration Command in CLI 

     sudo xcode-select -switch /Applications/Xcode.app/Contents/Developer/

    Now if you have both of the Xcode installed in your Mac then, Xcode6 might be installed as Xcode 2.app as in your application directory. Now if you want to use iOS 8.0 and 7.1 simulators you need to configure by the code below.

    sudo xcode-select -switch /Applications/Xcode2.app/Contents/Developer/

    Also Update your CLI to 3.4.0

    Hope that helps. If so, Please mark as best answer. Thanks.
