User string from Alloy.Globals in view.xml

Hi all,

I have a an object in the alloy file like this: 

Alloy.Globals.literals = {
            login : 'Inicia sesión',
            signin : 'Únete a XXX',
            invited : 'Accede como invitado'
        };

And I have a view where I want to use these literals directly but I don't know if it is possible. For example something like this:

<Button id="login" title="Alloy.Globals.literals.login" class="button-default" onClick="goLogin"></Button>

Of course, this isn't working, but if someone knows a way to do it, it will be great!
Thanks.

    In your controller

    $.login.title = Alloy.Globals.literals.login;
