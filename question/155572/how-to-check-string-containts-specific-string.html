  • As well as the regular expression, you can also check for the position of the string (if it exists)

    indexOf will return the start position of the string if it exists and -1 if it doesn't.

    var str = "http://crop.epicnew.welcomeserver";
var match = "epicnew";

if(str.indexOf(match) < 0){
    alert("string does not contain epicnew");
}

  • You can also check if some specific substring is present in a string or not by search method:

    var str="http://crop.epicnew.welcomeserver.";
var n=str.search("epicnew");

    If value of n < 0, it means string not found.
