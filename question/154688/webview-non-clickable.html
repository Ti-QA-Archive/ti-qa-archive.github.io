Webview non clickable

Hi.

Is it possible to make a webview non clickable?

I have a webview with some html content o get from a website and i want to make the links in that content non clickable.

How can i do that? I have tried with this with no luck string is the html content.

var tweet_lbl = Titanium.UI.createWebView({
html : '<div><span style = "color:#111;">'+string+'</span></div>',
touchEnabled: false,
left : '5dp',
top : '0dp',
bottom : '0dp',
height : 'auto',
width : 'auto',
backgroundColor: '#transparent',

});

The content i get look like this

<a href=\"somewebsite.com">

I really dont want to user to be able to click the links in the webview.

Thanx

  • put the following in your HTML

    <script type="text/javascript">
  document.addEventListener("touchmove", function(e){
    e.preventDefault();
    return false;
  }, false);
</script>
