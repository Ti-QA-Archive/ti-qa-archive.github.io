Alloy view, function as controller argument

SDK 3.4
Hello,
I would like to know if there is a way to pass a function as an argument through an Alloy View.
Example of what I look for:
View.xml

<Alloy>
  <View>
    <Require src="mycontroller" type="view" customFunction="myCustomFunction" />
  </View>
</Alloy>

View.js

function myCustomFunction () {};

I don't want to use events (I know the onCustomEvent=&quot;myCustomeFunction&quot; trick )

Thank you for your help.

  • Don't put the require in your xml if you want to pass a function.

    Create the controller programmatically.
    View.xml

    <Alloy>
  <View id="container" />
</Alloy>

    View.js

    $.container.add(Alloy.createController("mycontroller", { myCustomFunction: myCustomFunction }).getView())

    This should work. Let me know.
