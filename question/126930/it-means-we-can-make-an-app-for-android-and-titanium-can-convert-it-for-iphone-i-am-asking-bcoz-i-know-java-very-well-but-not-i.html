  • Hi Hardik

    Titanium does not convert an android app into an ios app, but instead allows you to write apps (from the beginning) in Javascript. This means that you do not necessarily need know the underlying languages, although it can help if you wish to extend Titanium's functionality.

    If you install and configure Titanium, and create a new mobile project, you will see the type of code involved. For example, I just posted an answer to a question here that is all the code required to create an application (although admittedly an extremely basic and not particularly useful one) that runs on both android and ios.
