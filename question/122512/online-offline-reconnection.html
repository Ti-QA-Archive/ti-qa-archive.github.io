  • Holge,

    1) Checking if a network is available

    if (Titanium.Network.online) {
    // Online do something
    }else{
    // Offline do something else
    };

    2) To introduce a delay

    Search 'timeout' in the APIs (http://developer.appcelerator.com/apidoc/mobile/latest)
