    Hi Nick,

    your above condition in never gone true for android devices beacuse, 568 is the height for the iphone5 only and there is android device which have height of 568, you have to put one more condition with you existing condition as below

    if(OS_IOS && Ti.Platform.displayCaps.platformHeight==568){
    //condition for ios 
      home_image.image =  '/img/home-image.jpg';
      home_image.width = '100%';
      home_image.height = '199dp';
      btn_view.top = '263dp';
}else{
    //condition for android
      home_image.image =  '/img/home-image.jpg';
      home_image.width = '100%';
      home_image.height = '199dp';
      btn_view.top = '263dp';
}
