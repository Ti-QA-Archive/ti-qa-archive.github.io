I don't can change the font size of label in ipad

Hi, i have a program with android and runs fine. But in ipad, the font size of the labels, are very small.

i have in my index.js:

        if (Ti.Platform.osname=="ipad") {
        $.LNombre.font="fontSize:200dp";

    }

but no run. How i can change the font size of the label LNombre in my ipad??

thanks,
