How can I change the background color of header area in TableView?

I'm adding tableview like "Table_view_headers.js" in Kitchen Sink.
Now I would like to change the color of header area. How can I set it new color?
Please advise me.

    You will need to create a view and assign it to the table. If you want text in the header also then create a label and add it to the view also

    table.headerView = Ti.UI.createView({
   backgroundColor:'#373',  
   height:35
});
