Can't create an Android Module View

Am trying to make a custom control, so I need to draw a circle in an Android View.
But I can´t "load" the view. The Proxy is working, but the View is never loaded, the method createView() in my proxy is never fired.

MyViewProxy.java

@Kroll.proxy(creatableInModule=TestandroidmoduleModule.class)
public class MyViewProxy extends TiViewProxy{
    public MyViewProxy()
    {
        super();
    }

    @Override
    public TiUIView createView(Activity activity)
    {
        TiUIView view = new MyView(this);
        view.getLayoutParams().autoFillsHeight = true;
        view.getLayoutParams().autoFillsWidth = true;
        return view;
    }

    @Kroll.method
    public void initAction(){
    }

    @Kroll.getProperty @Kroll.method
    public float getMinValue()
    {
        return this.minValue;
    }

    @Kroll.setProperty @Kroll.method
    public void setMinValue(String minValue)
    {
        this.minValue = TiConvert.toFloat(minValue);
        Log.d(LCAT, "minValue" + this.minValue);
    }
}

MyView.java

public class MyView extends TiUIView{
    private static final String LCAT = "MyView";
    public MyView(TiViewProxy proxy) {
        super(proxy);
        // TODO Auto-generated constructor stub
        Log.d(LCAT, "Working...");
    }
}

And in my app.js i got the following code:

var my_module = require('cl.mobdev');


var foo = my_module.createChango({
    minValue: 30
});

var bar = foo.initAction();

As I said, the problem is that MyView is never been load.
Am following the "Titanium-Paint" module as example.
