  • For Android in Alloy it seems the root window tag in index.xml needs to have id="index".

    It's good to clean before build. Alloy will use android assets, controllers, styles, and views when building for android. I tried "#index[platform=android]" in tss files and it didn't work out. Also class attributes don't work either in views.

    class attributes example

    <ScrollableView class="index"></ScrollableView>

    /views/android/index.xml

    <Window id="index"> </Window>

    /styles/android/index.tss

    "#index" : { 
exitOnClose: "true"
}

    controllers/android/index.js

    $.index.open();

    Using Alloy 1.0 , sdk 3.0.1..Thanks to Appcelerator Team for making this amazing toolset.
