touch start and touchend x coordinates

Hi, in my iPad app i need to obtain the difference between the start x coordinate (when the user touches the screen) and the final x coordinate ( when the user finger's leave the screen) how can i do? thanks in advice

  • var startX = null;

win.addEventListener('touchstart',function(e){
    startX = e.x;
});

win.addEventListener('touchend',function(e){
    Ti.Api.info("diffrenece: "+e.x - startX);
});
