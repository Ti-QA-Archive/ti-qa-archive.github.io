Touchmove event on android

I use touchmove event to move image, but it doesn't work on android. How can I use this event on android. Thanks.

ViewControl.addEventListener('touchstart',function(e){
imageViewSelect.center = {
x: e.x,
y: e.y
};
});

ViewControl.addEventListener('touchmove',function(e){
imageViewSelect.center = {
x: e.x,
y: e.y
};
});

    Hi Chen,

    Try this code.

    var X,Y;

viewToDrag.addEventListener('touchstart', function(e) {
    X = e.x; 
    Y = e.y;
});

viewToDrag.addEventListener('touchmove', function(e) {
   imageViewSelect.center = { x: e.x - X, y: e.y - Y }; 

 //   you can also set it's top and left like 

    //    imageViewSelect.left = e.x - X;
    //    imageViewSelect.top = e.y - Y;

});
