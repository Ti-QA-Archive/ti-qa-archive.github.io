iPhone - get country name

I need to get the user's current location's country name.

My current code (the only I've found so far that works) only returns the country code:

COUNTRY = Ti.Locale.getCurrentCountry();

Is there any way to get the country name without making some sort of request to a third party (like the Google Maps API). And, if not, what such API would you recommend?

    You will have to map those yourself unfortunately.

    Using this table, you could build a JavaScript object with the country code as property-name ant the country name as value.

    Another approach would be to have those values stored as internationalized strings with the code as the key. You could then use the L(COUNTRY) function to return the full name stored in the string bundle.
