    Whatever i am righting here is based on xcode 3.2.3

    1) in xcode's left panel (Header is groups & File), click on your project name. New Window will be open named - project "your project name" info. In this window you will find General and Build both tab

    • in general tab at the last select iPhone simulator version from drop down which has a label Base SDK for all labels. It will make you project runnable in simulator.

    and for packaging

    2) in build tab you can file option to select code signing identity. Default selection will be iPhone Developer. Select your own certificate which you obtain from apple. After this you can package your app from xcode.

    by the way. why you are doing this all ? titanium providing option to build your project for titanium it self.

  • There are few things that you have to do to make your titanium project working in xcode.

    1) Select the appropriate SDK version from your project property. (This change will make your project runnable in simulator)

    2) in project property's build section select your code signing certificate. By default iPhone Developer will be selected (After that this will allow you to package your app)
