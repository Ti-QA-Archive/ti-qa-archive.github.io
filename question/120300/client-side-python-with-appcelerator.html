Client-side python with appcelerator

It is written that appcelerator provides "script type=text/python" tag, where I can code in python.
But if I include this tag in my website, how will it work? I mean, browsers can only understand javascript, so in order for it to work user must either use a plugin, or a python to javascript compiler must be included in the website.
So, does appcelerator provide plugin, or compiler, or what?

    The php/python/ruby scripts are executed in the context of the Titanium Desktop. So you will need to run it inside a session of Ti Desktop application, and not on a webpage.

    Ti Desktop is a modified webkit that knows how to handle this scripts.
