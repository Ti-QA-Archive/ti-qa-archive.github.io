  • As you said you have the module so copy the ti.admob folder(i.e the module)

    and paste it at Library/Application Support/Titanium/modules/iphone/

    if there is no iphone folder then create iphone folder and then paste the

    ti.admob folder inside that folder,

    this path is for the Mac OS it may differ if you are using Windows OS

    now open tiapp.xml and place the following code at the end of the xml

        <modules>
        <module version="1.0">ti.admob</module>
    </modules>
</ti:app>

    restart the titanium.
    Now to add the admob view following is the code

    Titanium.Admob = require('ti.admob');

var adsView = Ti.Admob.createView({

    top: 0, left: 0,
    width: 320, height: 50,
    publisherId: youtPublisherId,
    testing: true, //true mean you are in testing mode else keep it false
    dateOfBirth: new Date(1985, 10, 1, 12, 1, 1),
    gender: 'male',
    keywords: '',
});
