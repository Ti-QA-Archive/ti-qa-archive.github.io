  • Alfredo, after some thinking, I'm guessing your problem might be with the command line you are using to launch jsDoc.

    From the JSDoc wiki:

    java -jar jsrun.jar app/run.js -r=4 myscripts/ -t=templates/jsdoc

    The app&#x2F;run.js is actually the script used by jsDocs and is located in the jsdoc-toolkit folder. This is not the entry point of your source code. The error you received saying Titanium or Ti is undefined, is actually the rhino interpreter trying to run YOUR code, not the jsDoc parser.

    A working example would be as follows:

    java -jar /path.to.jsdoc-toolkit/app/run.js /path.to.your.project/ -t=/path.to.jsdoc/templates/jsdoc
