  • Hi

    Please learn to use the content area formatting tags, it will help you get more people commenting if they can better read your question. The most important one is three ~ one a line above and below your code.

    Having taken a quick look I would imagine this is the problem object below.

    var pro_img = Ti.UI.createImageView({
                backgroundImage : 'http://www.swapopolis.biz/uploads/'+json[yGrid].cardimg,

                // 5077df7c410c9.jpg // 1732468_card.jpg
                //image : 'http://www.swapopolis.biz/uploads/1732468_card.jpg',
                //left : 5,
                //top : 5,
                width : 100,
                project_id : json[yGrid].project_id,
                height : 120

            });

    You have two images running here, the first a background image that appears to be different for each loop, so that looks like the main image. You also have image which appears to be static so likely this is meant to be your default image.

    You need to change these; change the background image property to be the image property and change the current image property to be the default image property.

    var pro_img = Ti.UI.createImageView({
                image : 'http://www.swapopolis.biz/uploads/'+json[yGrid].cardimg,

                // 5077df7c410c9.jpg // 1732468_card.jpg
                defaultImage : '1732468_card.jpg',
                //left : 5,
                //top : 5,
                width : 100,
                project_id : json[yGrid].project_id,
                height : 120

            });

    Also default image must be local to work correctly.

    Change the values to suit your needs but should give a starting point.
