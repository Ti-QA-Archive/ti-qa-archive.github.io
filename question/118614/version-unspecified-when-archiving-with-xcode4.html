Version unspecified when archiving with Xcode4

When archiving an app with Xcode4 the version of the app is unspecified.

As a workaround you could manually edit the Info.plist file in the build directory. Just add an additional property of type "Bundle version string, short" with the current version number as value.

Please fix this with the next version, as it is only a small bug.
