Deploying Kitchen Sink

Hi,

I have downloaded the latest version of Titanium and also downloaded the Kitchen Sink project however when I try to deploy it, either on my Android Device or Simulator, I get the following error:

[ERROR] :  Build process exited with code 1
[ERROR] :  Project failed to build after 349ms

Any idea?

I have a Window 7 machine and the latest Android SDK installed.

Thanks!

  • Are you absolutely sure that your whole environment is set up correctly?

    One of the first thing you should verify would be the Android Debug Bridge (adb) Once your emulator is started or your device plugged in, simply type the following at the command prompt.

    adb devices

    If it doesn't return any device, then you know it is related to your setup.

    Keep us posted.

  • Hi, I've had today the same problem and I come to a solution:

    kitchen-sink-not-running

    Delete this tag/line within Android Tag on Tiapp.xml

    <tool-api-level>11</tool-api-level>
