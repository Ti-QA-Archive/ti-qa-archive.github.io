Download file on Android

Hi,

I'm trying to download a pdf from a webservice which works perfectly on iOS. However it seems to be failing/crashing the app on Android. The point in which the app crashes is when it's trying to write the file. Does anyone have any pointers? The file is less than 10MB.

I'm on OSX 10.8 running 3.1.1.201306112235

Phone: Samsung Galaxy S3, OS 4.1.1

See example code below. Many thanks in advance.


function checkIfResourceExists(url, title) {

    var filename = url.split('/');
    filename = filename[filename.length - 1];

    filename = filename.replace('.aspx', '.pdf');

    var file = Ti.Filesystem.getFile(Ti.Filesystem.applicationCacheDirectory, filename);

    if (file.exists()) {

        //open file
        openResource(file, title);

    } else {

        var xhr = Ti.Network.createHTTPClient({
            timeout : 10000
        });

        xhr.onload = function() {

            if (xhr.status == 200) {

                file.write(this.responseData);


            };

        };

        xhr.onerror = function() {

            alert("failed to download resource");

        };


        xhr.open('GET', url);
        xhr.send();

    };

}

  • In iOS by default we have async downloading. In android we don't have such thing by default. I had the same problem before and I fixed it by extend native module for downloading files using asynctask class.
