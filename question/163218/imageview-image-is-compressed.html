  • If you only want to display a rect of the Image, what you can do is a View of fixed height and width, add the ImageView to that View, and set the size of ImageView to Ti.UI.SIZE, so the parent View will only see the top 0,0 of the Image, and from there, you cand adjust ImageView .rect property to set .x and .y with whatever you want..

    Hope it helps!
