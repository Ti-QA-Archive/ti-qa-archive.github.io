Including external js and css in Desktop

Has anyone managed to get external css and javascript included in their desktop app? Here's an example of the code I'm using.

  <link rel="stylesheet" type="text/css" href="css/style.css" />
  <script src="lib/jquery.js"></script>

Neither of these seem to work - only inline stuff works.

I've had a look at tweetanium desktop and it does not seem like it's working for that either. Here's a screenshot of what tweetanium looks like for me: http://cl.ly/3e311v2Y3i2q2d101B33

Is anyone else having this problem, or knows what I am doing wrong?
Thanks

  • First of all "lib/jquery.js" is not external, but relative to your Resources directory. Application will be looking for that file in "YourProject/Resources/lib/jquery.js". External files do load as well (both from local machine and url).
