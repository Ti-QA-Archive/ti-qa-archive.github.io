    Hi

    It is likely your use of include inside a function that is causing problems, the documentation will give you more details.

    Consider moving this (and others) to the now much preferred rout of CommonJS, this provides a much easier and more manageable approach to component re-use.

    A simple example test.js;

    function test(obj) {
    var view = Ti.UI.createView({
        height: Ti.UI.SIZE,
        width: Ti.UI.FILL
    });
    var lbl = Ti.UI.createLabel({
        height: Ti.UI.SIZE,
        text: obj.title || '',
        width: Ti.UI.SIZE
    });
    view.add(lbl);
    return view;
}
exports.test = test;

    Usage;

    var Test = require('\test'); // note the omission of the file extension
var test1 = Test({ title: 'One' });
win1.add(test1);
...
var test2 = Test({ title: 'Two' });
win2.add(test2);

    Using CommonJS should make what you are attempting much easier.
