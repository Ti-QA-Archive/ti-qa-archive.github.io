Hidden window click through

Titanium 3.2.3
Android 4.4
Mac 10.9.3

I have 2 heavyweight windows and I am going to flip from one window to another by orientation change and the only way I think of doing this is by using the show() and hide() functions. Now the problem I noticed is that when the top most window is hidden, click events for the window behind it doens't work. Am guessing topmost window is still there but not visible so to speak. If I use the close() function, click events work fine for the other window, but the problem is I have to recreate the previous window to load and open it again. I can use the close() function but if there's another way to do it without closing a window then I'd actually prefer that.

    Trying to achieve different displays on orientation change on Android using windows is unlikely to ever be satisfactory. It is more common to use to view parents and show/hide those on orientation change.
