Parsing ini or properties files

I was wondering if Titanium has functionality to parse .ini, .properties or other such configuration files. The basic idea is that I want to have a set of properties of my app which a non technical person can modify in this configuration file.

  • Rainer is right when he mentions that the two official formats are XML and JSON. Although you could (and maybe should) use the Properties API and serialize it on the file system.

    The caveat to this approach is that the generated JSON file will not be very comprehensible for a non technical user. As opposed to a properties file.

    The best solution would be to create two functions to load and write the Properties to the file system.

    The code would go something like this:

    /* Read the properties file line by line */

var prop = lineContent.split(':');
var key = prop[0];
var value = prop[1];

/* Populate the Properties object using the key/value */

    The same goes for saving the properties on the file system. You can loop through all the properties using the listProperties method and save the output to your desired format.
