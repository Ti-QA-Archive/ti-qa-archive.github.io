Lookup Table Concept using Alloy Models

Hi all-

Does Alloy / Backbone.js give the ability to utilize a lookup table? I have to fairly basic SQLLite models, one defining a collection of documents, a second defining a collection of companies. A single document can belong to/be linked to many companies. In typical SQL fashion, I would traditionally use a lookup table to link the primary key of the company to the primary key of the document.

For an Alloy implementation though, where I am looking to bind these documents to a table based on a user clicking on the linked company(ies) row, I am not sure where to start with a lookup table, or if the concept with Backbone.js exists so I could have a many to one relationship for models? I am not sure if there are examples out there, or references for model lookups that may be a help.

Thank you so much in advance.
