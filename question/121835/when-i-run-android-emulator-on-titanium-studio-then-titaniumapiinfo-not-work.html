  • Using DDMS works for me. Why on earth Titanium Studio no longer writes Ti.API.info and friends to the console is beyond me. Given up trying to figure it out in favor of DDMS.

    You can find DDMS in the tools directory with the directory when the Android SDK is installed. In OSX, I just double click the application and it boots via a console window.

    Make sure the top item in the list on the top left side of the window is seleceted. On my machine the line reads something like emulator-5560 Online titanium_10_WVGA800

    I then setup a filter 'by log tag' for TiAPI, 'by log level' verbose

    All Ti.API console writes should then begin printing to the console in DDMS.
