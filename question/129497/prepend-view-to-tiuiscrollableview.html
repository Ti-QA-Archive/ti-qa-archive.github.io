Prepend View to Ti.UI.ScrollableView

Is it possible to prepend a view to a Ti.UI.ScrollableView? I want to load a given number of views and as the user scrolls back I want to keep prepending views.

  • try add this method to your scrollableView instance and use 

    scrollableView.prependView = function(view){
    var views = scrollableView.getViews();
    var currentPage = scrollableView.getCurrentPage() + 1;
    views.unshift(view);
    scrollableView.setCurrentPage(currentPage);
    scrollableView.setViews(views);
}
