Slow slider

Hi, I use slider for change image opacity like this:

slider.addEventListener('change', function(e){
    image.opacity = Math.round(e.value) / 100; // Between 0 - 1
});

,but slider is very slow. Exist some better ( Faster ) way ? Thank you very much for reply

  • I would recommend using a timeout to throttle the frequency of the calls to set image opacity:

    var opacityTimeout = null;
slider.addEventListener('change', function(e) {
    if (opacityTimeout != null)
    {
        clearTimeout (opacityTimeout);
    }
    opacityTimeout = setTimeout (function () {
        image.opacity = Math.round(e.value) / 100; // Between 0 - 1
    }, 200);
});

    Now the opacity calculations are only done after the user stops moving the slider for 200 ms.

    The downside is that the user probably won't see the changes until he removes his finger from the slider, unless he moves slowly, pausing a bit at each step.

    If you really need to see the settings change as the user moves the slider, you could still throttle the frequency by only performing one opacity change every X milliseconds (you have to decide what makes sense from a performance and user experience perspective). That would require keeping track of the last time you updated the opacity and computing the elapsed time in your change event listener.
