Read a line from file

Hi, I'm a noobe to Ti SDK. How do you read a line from file? Something like Java's BufferedReader's readLine() method? I'm trying to do something like this.

var eventsFileStream = Ti.Filesystem.openStream(Ti.Filesystem.MODE_READ, 'data/events.txt');
var event_line = eventsFileStream.readLine();
while(event_line != null){
    Ti.API.info(event_line);
    event_line = eventsFileStream.readLine();
}
eventsFileStream.close();

Thanks in advace!

    You can read a file in the following way

    var readContents;
var readFile = Titanium.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory,'data.txt');        

if (readFile.exists()) {
     readContents = readFile.read();
     Ti.API.info('File Exists');  
}

var doc = readContents.text
Ti.API.info('Contents = ' + doc);

    i hope it will help you.
