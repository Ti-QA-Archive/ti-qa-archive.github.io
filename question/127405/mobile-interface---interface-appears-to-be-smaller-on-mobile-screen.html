Mobile interface - interface appears to be smaller on mobile screen

Hi. I am having trouble with re sizing the interface for my mobile application.
When I design the interface, it looks fine on the emulator;fits on the screen.However, when I try to display it on my mobile, the whole interface aligned center, creating some blank wide spaces at the sides and making the interface much smaller than they should (it did not fit the screen).
Could anyone please help me with this? Thank you. :)

  • Basically, try to change all numeric values regarding positions (top,left,right,bottom), font sizes, width and height in your app, by adding "dp" to them. for example:

    replace:

    var b = Titanium.UI.createButton({width:100,height:30,title:"Some Button"});

    with:

    var b = Titanium.UI.createButton({width:"100dp",height:"30dp",title:"Some Button"});

    iOS will ignore it, and on android it will take the resolution into account.
