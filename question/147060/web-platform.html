Web platform

I am looking into Appcelerator for use with mobile, which i know it will handle, but also for web deployment. I researched a bit and found http://developer.appcelerator.com/blog/2011/09/titanium-sdk-for-mobile-web-beta-1.html which shows a mobile web experience. It is not exactly the prettiest web view. Is there a way to perhaps use css or something else to customize the deployment to the web experience have it look different than the mobile or mobile web?

  • Titanium's premiere framework, Alloy, has a CSS-like stylesheet for styling your app. It works with native Titanium mobile apps as well as Titanium Mobile Web. Checkout the Alloy docs here: http://docs.appcelerator.com/titanium/latest/#!/guide/Alloy_Quick_Start.
