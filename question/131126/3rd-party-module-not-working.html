  • If you don't manage, find where Titanium holds the iphone modules on your system
    It can be in the main harddrive of your mac

    /Library/Application\ Support/Titanium/modules/iphone/

    or under the current user

    ~/Library/Application\ Support/Titanium/modules/iphone/

    Either way, unzip the module in this folder and remove the module folder in your project.

    Go to your_project/build/iphone and empty it (empty, don't delete the iphone folder) and then build again.
    It should work.
