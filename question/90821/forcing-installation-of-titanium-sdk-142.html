Forcing installation of Titanium SDK 1.4.2

Hello,

I recently purchased a new Mac and would like to use the 1.4.2 SDK instead 1.5.0 (my app won't work with it).

The app still runs fine on my old Mac when compiled with 1.4.2, so if I can install 1.4.2 on my new machine, I can continue working on my app until 1.5.1 is released. However, I can't figure out where to download the 1.4.2 SDK or how to install it.

Any help?

    I don't know where to download the older SDK's, I would like to know though.

    However, you can easily copy your current SDK over from one machine to the other. The SDK versions are located at either:
    ~/Library/Application Support/Titanium/mobilesdk/osx/ OR
    /Library/Application Support/Titanium/mobilesdk/osx/

    Simply copy the folder with the version number you want from one machine to the other (with Titanium closed of course). You will have to update your project to select the older version once it is available.
