TableView - How to change row's backgroundImage

How can i change one table's row background?

I create my table like this:

var tableData= [];

for(var i=0; i< options.length; i++){

    var rowAux=Titanium.UI.createTableViewRow({
            backgroundImage:"../images/row.png",
               selectedBackgroundImage: "../images/row_selected.png",        
                height : 40,
        });
    ...
    //Use label instead of tableviewrow property title so i can change it's color
        var rowAuxLabel = Titanium.UI.createLabel({
            ...
        });    
    ...
        rowAux.add(rowAuxLabel);

    tableData.push(rowAux);
}
...
var tableAux = Titanium.UI.createTableView({
    data:tableData,
    ...
    backgroundColor: "transparent",
    scrollable:false
});

When the user clicks in one of the table's rows i change it's label color with:


tableAux.addEventListener("click",function(e) {
    tableAux[e.index].children[1].color = "#a55ba5";

});

Now i would like to change the backgroundImage of the row "permanently", not only while the user maintains the row pressed(with selectedBackgroundImage).

tableAux[e.index].backgroundImage = &#39;..&#x2F;images&#x2F;row_selected.png&#39;; doesn't work.
