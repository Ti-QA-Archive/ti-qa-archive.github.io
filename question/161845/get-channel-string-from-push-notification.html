    Firstly, when sending a push notification through ACS you can push to a specific channel, allowing you to only send push notifications to devices subscribed to that channel.

    Secondly, I think that if you want to know what channel you are receiving the push notification from then you would have to create an advanced push notification, where you could create your own payload or JSON object and pass the channel name in the object.

    Hope this helps!
