require() inside include()

Inside my app.js i include a file called ui.js

Ti.include(&#39;ui.js&#39;);

inside that include I have some modules that are required e.g

reportIt.UI.winContact = require(&#39;winContact&#39;).win;

which works fine.

I now have around 10 of these modules and I want to store them all in a subdirectory. I have created a folder called 'views' under Resources and moved the modules there. When I try to access the modules now

reportIt.UI.winContact = require(&#39;views&#x2F;winContact&#39;).win;

they fail and cant be found.

any ideas why?

    I'm always pointing to the root (for includes and modules). Try this (adding &#x2F;):

    require(&#39;&#x2F;views&#x2F;winContact&#39;).win
