Android multiple maps

I need to show more than one map in an android app.
The app is window driven ( so windows with url ) and I need to show a map in more windows.

The app crashes with the error "Only one map per app is allowed".

I tried to remove and set to null the previous map but it seems not to work.

Any ideas?

Thnx

  • Unfortunately, Android doesn't want multiple instances of maps in an application. It's not ideal, but I think the only solution is to have one map instance that's shared between windows. For example, in app.js, you could do something like:

    var mapView = Titanium.Map.createView(options);

    And then also in app.js, you could pass the mapview into each window as such:

    win1 = Ti.UI.createWindow({... 
    map:  mapView
});
win2 = Ti.UI.createWindow({...
    map: mapView
});

    Then in each window, you would have to essentially reset and configure the map for the window's purpose.
