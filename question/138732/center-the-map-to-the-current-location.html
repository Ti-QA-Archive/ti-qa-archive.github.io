Center the map to the current location

Hi,
I am running my app on simulator and I have 2 issues:
1) default "user current location" is placed on San Francisco. Can I change that without using "region:" and set the delta for long lat?
2)How can I center the map on the current location(I want the current location in the center of the map)
Thanks in advance.

    Simulator has no GPS/Cellular provider, so you cannot. To center the map you would need to listen to the location Event and Update the map Region through map.setRegion(...);

      But what you can: center the screen of the map to the blue dot using the functions mentioned above.
      The delta you can control with setRegion (see above), region object consists of 4 properties:

      {
    latitude: ..,
    longitude: ...,
    latitudeDelta: ...,
    longitudeDelta: ...
}

      check the API description for further information.

  • Ti.Geolocation.purpose = 'Location based services for the app';
Ti.Geolocation.accuracy = Ti.Geolocation.ACCURACY_NEAREST_TEN_METERS;
Ti.Geolocation.preferredProvider = Ti.Geolocation.PROVIDER_GPS;

var win = Ti.UI.createWindow();

Ti.Geolocation.getCurrentPosition(function(e){

            var mapView = Ti.Map.createView({
                mapType : Ti.Map.STANDARD_TYPE,
                region : {
                    latitude : e.coords.latitude,
                    longitude : e.coords.longitude,
                    latitudeDelta : 0.0009,
                    longitudeDelta : 0.0009

                },
                left : 0,
                top : 0,
                bottom : 0,
                right : 0,
                animate : true,
                regionFit : true,
                userLocation : true
            });

        win.add(mapView);
        win.open();
});
