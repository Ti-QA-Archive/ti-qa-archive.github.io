  • Ok, dug around, experimented, and basically drove myself insane before I realized (via phpinfo) that the SQLite3 object was available. So, for those using php, if you want to connect to an sqlite db, you'll do something like this:

    class DB extends SQLite3{
    function connect($database_file="path/to/your/data.db"){
    $this->open($database_file);
    }
    }

    For more info on the class SQLite3, go here:
    http://www.php.net/manual/en/sqlite3.query.php
