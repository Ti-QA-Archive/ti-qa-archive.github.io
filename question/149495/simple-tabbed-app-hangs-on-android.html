Simple Tabbed App Hangs on Android

I cannot seem to get a Titanium App to run under any of the 4.x versions of Android (ICS or JellyBean).

If I compile with a 1.x or 2.x version of the Titanium SDK, the app crashes immediately on launch. It doesn't even get to the first line in the app.js.

If I compile with the 3.x Titanium SDK, with analytics set to false in the tiapp.xml, I get the same result. Crashes immediately.

If I set analytics to true, the app will launch, but it just hangs on the launch image.

I'm using a brand new, empty project created right from Titanium Studio, build: 3.0.2.201302191606 I just went File -> New -> Titanium Project -> Tabbed Application. Compile and run.

You can see what happens yourself by visiting this link to the Android emulator:

https://www.manymo.com/share/IU2H2kw-11089-90

Is anyone else having this issue? Has anyone been able to deploy a 3.x app to later versions of Android successfully?
