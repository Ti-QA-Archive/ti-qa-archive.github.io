3.3.0 Android Build Error

Trying to build directly to an Android device gives me these two errors every time with the new appCompat stuff.

No resource found that matches the given name (at 'theme' with value '@style/Theme.AppCompat.Translucent').

No resource found that matches the given name (at 'theme' with value '@style/Theme.AppCompat.Translucent.NoTitleBar.Fullscreen').

Any thoughts on this?

    • I solved the problem according to the doc above.
      Please check files in ./platform/android/res/values directory under your project directory if there's a file named theme.xml, if this file exists please rename to another names, then rebuild the app.

      — commented July 19th 2014 by Ray Woo

    • Yes it work's for me, in my custom theme file, i changed the parent theme name from
