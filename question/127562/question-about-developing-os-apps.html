  • Hi Bahad?r,

    Sorry but you can't develop an iphone app without a mac. You can however create the app for Android using Titanium, then if you can get a mac, you can quickly port that android code into an iphone application.

    If you absolutely can't get access to a mac, you can use MacInCloud. This let's you pay to use a real mac online without needing to buy one.

    Let me know if this works for you. If none of those work out for you, you can always try PhoneGap but it won't be as powerful as Titanium although they will let you build in the cloud without owning a mac.
