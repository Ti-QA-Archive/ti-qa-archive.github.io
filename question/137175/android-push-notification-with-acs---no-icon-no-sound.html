Android Push Notification with ACS - no icon, no sound

Application Type: Mobile

Titanium SDK: 2.0.1.GA2

Platform & version: Windows Vista

Device: HTC EVO with Android 2.3.3

Titanium Studio: 2.0.1

I'm getting push notifications just fine. I'm just trying to refine the user experience and this includes notifications and a custom icon for the status bar in Android.

Consulting the docs for the custom sound, I have created an assets folder under the resources folder and from there a sound folder. I placed a wav file in that folder. Per the docs I specified a custom sound file with just the sound file name and extension (from the Push Notifications in the online App Managment area). No sound.

As far as the custom notification icon is concerned, I am a little confused. The docs, and some QA, reference a folder (res/drawable) as though it should just exist in my project folder structure but it doesn't. I created it and put my custom icon in there and didn't get it to work.

Any ideas on either of these? Do these features work?
