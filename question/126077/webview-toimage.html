Webview toImage()

Hi,
I'm trying to create an Image from a Webview like so:

var webView = Ti.UI.createWebView({
    width:200,
    height:100,
    html:'<div style="font-size:12; font-family=Helvetica">Hi <b>there</b></div>',
    top:10,
    left:10,
    backgroundColor:'#ff0000'
})

app.news.add(webView);

var view = webView.toImage();
var file = Titanium.Filesystem.createTempFile(Titanium.Filesystem.resourcesDirectory);
file.write(view);

var test = Ti.UI.createImageView({
    image:file.nativePath,
    width:200,
    height:100,
    left:30,
    top:10,
});

app.news.add(test);

The webview displays correctly, but the rendered Image only contains the backgroundColor (red).

Thoughts on this would be greatly appreciated!

  • ok solved it. The Webview takes some time to load the HTML. Adding a timeout before calling toImage solved it.
