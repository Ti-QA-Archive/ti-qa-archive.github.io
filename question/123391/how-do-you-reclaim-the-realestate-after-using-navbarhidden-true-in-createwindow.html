How do you reclaim the realestate after using navBarHidden: true in createWindow?

For the first page of my app, I am hiding the navbar. I am trying to place a logo image at the top of the window, but nothing I try will move the image above the bottom of the navbar.

The navbar is not showing, it is hidden, but its like the screen space for it is still being held in reserve.

I am a complete novice at this, and have tried doing searches and checking documentation to no avail. Even tried setting the top position of the image, the window and both at the same time to a negative value. There is probably something obvious I am missing, so any advice on how to resolve this would be most welcome.

The code for the window creation and image are below.

Thanks.

var win_home = Titanium.UI.createWindow({  
    title:'',
    backgroundColor:'#300234',
    backgroundImage:'images/bg.png',
    navBarHidden: true
});
var home_logo = Titanium.UI.createImageView({
    image:'images/logoHome.png',
    touchEnabled: false,
    top: 0,
    left: 0
});
win_home.add(home_logo);
