Debugging on iOS emulator

When debugging emulator for iOS, in the "Variables" tab is not visible structure of objects. At the same time on android emulator structure displayed perfectly.
How to solve this problem?

Sample code:

var win = Ti.UI.createWindow({});
var table = Ti.UI.createTableView({});

for (var i = 0; i < 10; i++) {
    var row = Ti.UI.createTableViewRow({});
    var label = Ti.UI.createLabel({text: 'row ' + i});
    label.addEventListener('click', function(e){
        alert(label.text);
    });
    row.add(label);
    table.appendRow(row);
}
win.add(table);
win.open();
