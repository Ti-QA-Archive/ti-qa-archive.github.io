    • Hi, If you want access remote xml file then you need to store in your remote server(see my gist code) and if you want to access from local (apps directory) then you need to save under Resources directory. 

      var file = Ti.Filesystem.getFile(Ti.Filesystem.resourcesDirectory,'Translation.xml'); 
if ( file.exists() ) {
        var xmltext = file.read().text;
        var doc = Ti.XML.parseString(xmltext); 
}

      For more information please visit: Filesystem Access and Storage

  • It depends if you want to create (an possibily modify) the file programmatically, or you just need to access an read it from your code (with no modification possible).

    In first case, you should create it in (i.e.) applicationDataDirectory; see Titanium.Filesystem for more info on existing storage locations.

    In the second case you could put the xml file in your app/assets directory.

    In both cases you can access and read it using Ti.Filesystem.getFile(), providing the correct path to it.

    To parse it you will want to use Titanium.XML module.
