    • Opening a window with Titanium.UI.currentTab.open only works with an open tabgroup as you can read in the documentaion:

      "currentTab : Titanium.UI.Tab

      The currently active tab, if a tab group is open.
      If no tab group is open, this value is undefined."

      I don't know the rest of your code but it seems that you either never created a Ti.UI.TabGroup or it is not open.

      Maybe you can open your window with win.open() instead.
