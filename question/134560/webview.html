  • Accepted Answer

    Hi Federico,

    Replace this code

    webview2.addEventListener('load', function(e) {
Ti.App.Properties.setString('LastUrl',e.url);

var lastURL = Ti.App.Properties.getString('LastUrl');
alert(lastURL);

activityIndicator.hide();
});
