Rotate and scale an image view from the top left, not center?

Hi I have been working on an iPhone app where by the user takes a photo from their camera, it displays the photo inside a scroll view container, where the user can rotate and change the scale/size of the photo.

I found a good example for scaling and rotating the same image at e same time with this:


function changeImage(type, amount){
  var t = Ti.UI.create2DMatrix(); 
  var anim = Titanium.UI.createAnimation();
  t = t.rotate(degrees).scale(amount);
  anim.transform = t;
  Image.animate(anim);
}

My problem is that the transformation to be from the top left not the center point - the image always needs to be at top:0, left:0 of the scrollView.

I tried adding the translate transform on top of the existing:

t=t.rotate(degrees).scale(amount).translate(0,0)

But this doesn't place it top left, can anyone help me?

    For iOS all views have an anchorPoint to which the view is anchored during animations and such. See the docs here: http://docs.appcelerator.com/titanium/2.1/index.html#!/api/Titanium.UI.View-property-anchorPoint
