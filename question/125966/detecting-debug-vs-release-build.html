  • If you are referring to the Titanium analytics, the only way I know to control whether or not it is used is done via compile time in your tiapp.xml file. So set that to false until ready to distribute, then recompile with it set to true before submitting to the app store.

    If you want to determine from code what build you have, then one easy alternative is to alter your version number to add a suffix like "beta". You can read the current version number using Ti.App.version then act accordingly.

    Just change it and the analytics setting prior to the public build.
