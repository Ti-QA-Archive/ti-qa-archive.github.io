  • Check your path. Depending on how you install Git, its installer can add PATH_TO_GIT\cmd to your system path, but you want PATH_TO_GIT\bin instead. Also, on Windows, it's best to install all the tools to paths without spaces. So, install Git to C:\Git rather than C:\Program Files (x86)\Git.
