  • open cmd and write this codes:

    windows:

    cd %ANDROID_SDK%\tools
mklink adb.exe ..\platform-tools\adb.exe
mklink AdbWinApi.dll ..\platform-tools\AdbWinApi.dll

    linux:

    cd $ANDROID_SDK/tools
ln -s ../platform-tools/adb

  • Hesk

    Please ensure you have the Android SDK Tools and Android SDK Platform-tools packages installed. Then copy adb.exe and its associated dll from the platform-tools\ directory into the tools\ directory. Both of these steps are mentioned in the Getting Started guide under the Preparing for iOS development section.

    Also, note that Android SDK 3.0 is not supported until Titanium SDK 1.7.X, as shown in the Android SDK / Target Android Platform section of the Titanium Compatibility Matrix.

    Hope this helps
