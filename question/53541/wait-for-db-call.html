  • Try storing the database resultset in a variable and do two seperate queries.

    var query = null;
var result = null;

// First query
query = my_db.execute('SELECT * FROM ...');
if (query.isValidRow()) {
    result = query.getFieldByName('my_column');
}
query.close();

if (result !== null) {
    // Second query
    query = my_db.execute('SELECT * FROM ...');
    // Do stuff
}
