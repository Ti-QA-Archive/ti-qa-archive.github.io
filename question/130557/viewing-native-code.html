viewing native code

Is there a way to view the native Java or iOS code at some point during the Titanium dev process?

  • Once you have built your app for the first time check /AppName/build directory. You will see the Android and/or iOS projects.

  • If you are looking for the Java or Objective-C version of the Javascript you will not find it because it does not exist. The Javascript that you write using Titanium is run on an interpreter. Version 1.8 of Titanium for Android allows you to choose between the new V8 interpreter or the older Rhino interpreter.

    As the previous poster said you can look at the /appname/build directory to see the Titanium libraries and customer modules, but you won't see a Java or Objective-C version of your code.
