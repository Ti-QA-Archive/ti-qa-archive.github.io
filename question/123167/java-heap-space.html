  • Accepted Answer

    First, update your heap space limits:

    http://wiki.appcelerator.org/display/tis/Setting+Memory+Levels

    To isolate if it's a particular project:

    1. Close all projects (right-click > close)
    2. Open them one by one, watching the memory level setting to see if one in particular makes the memory jump.

    If you want a very expedient way of doing that, or you still can't open Studio after upping the memory levels:

    1. Find your workspace folder and move all projects out. That will close them all.
    2. Now move them back in and "reopen" them in Studio.

    If you have a large tiapp.xml file in one project (> 1MB), then open it up in a text editor and see if there is gibberish content inside. If so, delete that, but please file a ticket to let us know that happened, especially if you can figure out when it happened.

    — answered August 4th 2011 by Ingo Muschenetz
