Padding for all components in a view?

Hi, is it possible to set padding properties for a view? I have a view with some labels and text fields. all components has width: 100%. This works so far. But now I want to have a padding on each corner for each components. For example 5 pixels on the top, left, right, bottom. I don't want to calculate the size of my components manually. Maybe there is a property which I didn't find to do this.

    Seems like you could always just add an additional view inside of the parent view and set the top, left, right, and bottom all to 5.

    var mainView = Ti.UI.createView( ... );

var subView = Ti.UI.createView({
    top: '5dp',
    right: '5dp',
    bottom: '5dp',
    left: '5dp',
    width: Ti.UI.FILL,
    height: Ti.UI.FILL
});

var label = Ti.UI.createLabel( ... );

subView.add(label);

mainView.add(subView);
