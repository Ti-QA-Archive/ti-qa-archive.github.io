I know how to switch the xcodes but when I select 3.1.2 from the Tiapp.xml and try to build I get an error saying I cannot build 3.1.2 with the CLI I am currently using.

  • Use to uninstall current

    [sudo] npm uninstall -g titanium

    npm generic pattern:

    [sudo] npm install -g <package>@<version>

    So,

    [sudo] npm install -g titanium@<version>
