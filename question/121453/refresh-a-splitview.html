Refresh a SplitView

I'm using different ipad splitviews in multiple tabs and need them to completely refresh each time the tabs are focused (app is being used in a kiosk setting). I know how to reload content on simple views but for some reason I can't get these splitviews to reload. Here is one execution I've tried that didn't work:

app.js

win1.addEventListener('focus',function(e)
{
MainWindows.splitView.close();
MainWindows.splitView.open();
});

    Unfortunately, Titanium doesn't provide something like a .refresh() option (would be much appreciated!). Instead, try to just edit the existing content in your window by using e.g. EventListeners.
