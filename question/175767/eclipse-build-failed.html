Eclipse-Build Failed

While trying to create a android module for titanium,and compiling this in eclipse(Build.xml) it fails with an error
BUILD FAILED
C:\Users\Titanium\mobilesdk\win32\3.2.3.GA\module\android\build.xml:296: The following error occurred while executing this line:
C:\Users\Titanium\mobilesdk\win32\3.2.3.GA\module\android\build.xml:247: Neither the ANDROID_NDK environment variable, or the android.ndk property is not set to an existing Android NDK installation (check your module's build.properties).

i have already added ANDROID_NDK path to the build.properties,still getting the same error

Any help would be appreciated,thanks in advance.

    Hi

    To build your module, you need to set the ANDROID_NDK environment variable:
    ANDROID_NDK=/path/to/android-ndk

    Alternatively, you can set the android.ndk property in build.properties:

    android.ndk=/path/to/android-ndk

      1. Android SDK 2.3.x (API Level 10). This version must be installed to build an Android module.
      2. Android NDK. Add an ANDROID_NDK environment variable pointing to the NDK folder.
      3. Ant 1.7.1 or above must be installed and in your system PATH to build from the command line.
      4. Eclipse and ADT can be used instead of or in addition to Ant.
      5. gperf must be installed and in your system PATH.
