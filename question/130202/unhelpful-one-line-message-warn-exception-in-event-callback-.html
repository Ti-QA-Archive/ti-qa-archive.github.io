Unhelpful one line message "[WARN] Exception in event callback. {"

I used to see these exceptions log details of what is inside the { on multiple lines.

This would usually include the exception details, including the line number and the source file, which in the past has been rather handy :)

Is there any config that will show the rest of the exception details?

I'm using the latest version of Titanium Studio.

  • Always when I search for the keywords I never find it but as soon as I post with tags, other questions posted with simply tags show and I can see the duplicate….

    It's not a Titanium Studio bug. But the default log level was changed to Info.

    To fix it: Right Click on your project > Run As > Run Configurations > Main tab > Set Log Level to Trace. It would bring your log message back.
