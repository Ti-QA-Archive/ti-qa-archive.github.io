Switching Facebook Account

So I'm using Titaniums facebook module and I want to log in with a different account, however every time I log out and back in I get the "you have already authorized" screen with the deprecation warning. How do I switch accounts? I've even tried reinstalling the app.

  • Accepted Answer

    var url = 'https://login.facebook.com';
var client = Titanium.Network.createHTTPClient();
client.clearCookies(url);

    Get rid of the cookies set in the webview used for the login stage if not using SSO.
