  • Hi

    With the very little information you have provided, the best you will get is a guess.

    Your error message includes "java", you say you have performed an "update". These two words usually mean that you have chosen to either;

    • upgraded Java to 1.7 (sometimes known as 7)

    or

    • upgraded to Java 64 bit

    NEITHER of these things should done as neither are currently support. If you have done either, un-install and re-install with the correct version.

    • Java Development Kit
