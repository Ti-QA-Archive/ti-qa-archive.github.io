centered view with horizontal layout within parent view

I have an application that contains of 3 sections: header, content, footer. The footer should contain an image and some text of variable length that is both centered. The footer should look something like this:

+-----------------------------------+
|          (IMG) SOME TEXT          |
+-----------------------------------+

I have already searched through the Q&A section & documentaion and tried a lot of different ways to do this, but nothing works. As soon as I am using layout: &#39;horizontal&#39;, the whole content is aligned left.

All I want is to have a parent view (footer) that contains another view (content) that contains horizontally layouted elements (image, label, …).

Here is my current code:

var footer = Ti.UI.createView({
    backgroundColor: '#ffffff',
           width: '100%',
           height: Ti.Platform.displayCaps.platformHeight * 0.1,
           bottom: 0
});
var content = Ti.UI.createView({
    width: '100%',
    height: 'auto',
    layout: 'horizontal'
});
content.add(Ti.UI.createImageView({
    image: '/images/image.png',
    width: 30,
    height: 30
}));
content.add(Ti.UI.createLabel({
    width: 'auto',
    height: 'auto',
    text: L('some_text'),
    color: '#000000'
}));
footer.add(content);

This code results in the content not being centered (tested on Android 2.3.3, Titanium SDK updated 3 weeks ago -> 'About Titanium Studio' does not work, so I can't provide a version number or build date):

+-----------------------------------+
|(IMG) SOME TEXT                    |
+-----------------------------------+

Does anyone see the error in my code or know how to achieve the desired layout?

  • Thanks Gagan and Mitul for your answers!

    I finally managed to get the layout. It may be a bit of a hack, but I removed the layout: &#39;horizontal&#39; and computed the left values for both the imageView and the label within the window "open" event. I had to do this because they have to be fully rendered before their dimensions can be retrieved.

    Here is the working code:

    var footer = Ti.UI.createView({
    backgroundColor: '#ffffff',
        width: '100%',
        height: Ti.Platform.displayCaps.platformHeight * 0.1,
        bottom: 0
});
var content = Ti.UI.createView({
    width: '100%',
    height: 'auto',
});
var imageView = Ti.UI.createImageView({
    image: '/images/image.png',
    width: 30,
    height: 30
}));
var label = Ti.UI.createLabel({
    text: L('some_text'),
    color: '#000000'
}));
content.add(imageView);
content.add(label);
footer.add(content);

win.addEventListener('open', function(e) {
    // set the position of the footer elements 
    imageView.left = (pu.ui.properties.platformWidth - label.width - image.width) / 2;
    label.left = (imageView.left + imageView.width);
});
