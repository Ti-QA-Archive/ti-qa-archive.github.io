Best practice for isolating scripts for different osnames

titanium mobile sdk 2.0.1

iphone sdk: 5.1

Hi,

I am building an app for android, ipad and iphone. First thing I noticed is that there is no "ipad" folder under Resources to handle images and icons. If I add an "ipad" folder into it will titanium mobile able to detect and use images inside that folder for ipad devices?

At the moment I define my views and windows' dimensions explicitly. Do I have to manually define the images dimensions for ipad?

What is the best practice for identifying ipad devices and assigning correct images built for ipad?

many thanks.

    In code its simple enough with Ti.Platform.osname. (note that Ti.include is not current best practice)

    if (Titanium.Platform.osname == 'ipad'){
Titanium.include('tabInfo2.js');
} else { 
Titanium.include('tabInfo.js');    
}
