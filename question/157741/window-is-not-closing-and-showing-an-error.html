    $.winflow.close();

    Here What u are going to do like, You want to close the index window from winflow.js file but in winflow.js file we have no access of the index file. and Why we are going to do like this.

    If You need to open any window then open from the winflow.js file. No need to close index file. Because Suppose, in index controller You are opening a Winflow Controller. so, Now winflow controller is a heavyweight instead of index controller. So, No need to close that file.

    And if you want to close the that file then you can use Fireevent for that.

    //Winflow.js
function openindex(e) {
    TI.App.fireEvent("closeWin");
}

//index.js
Ti.App.addEventListener("closeWin", function(){
    $.index.close();
});
