Parse XML to Array

Hi there,

The usual way I use XML is using 

<xml>
<item>
<id>1</id>
<body>content 1</body>
</item>
</xml>

However I'm having to pull the content from a CMS which is out of my control which uses 

<xml>
<item id="1" body="content 1" />
<item id="2" body="content 2" />
</xml>

Is there a way to parse and loop through this type of XML formatting easily?

Thanks in advance!

    Something like this…

    var doc = Ti.XML.parseString(http.responseText).documentElement;
var items = doc.getElementsByTagName("item");
for (var i = 0; i < items.length; ) {
   var item = items.item(i)
   var id = item.getAttribute("id");
   var body = item.getAttribute("body");
}
