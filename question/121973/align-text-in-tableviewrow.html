Align Text in TableViewRow

I like to align an label in a TableViewRow "top left 105px". I don't find anything in the Docs.

var row = Titanium.UI.createTableViewRow({className: 'class1', height:'160'});
var myview = Titanium.UI.createView({
});
var mytext = Titanium.UI.createLabel({
    text:'Foo Bar Foo Bar Foo Bar Foo Bar Foo Bar',
    left:105,
    top:5, //this is always ignored :(
    width:210,
    backgroundColor:'FF0000',
    textAlign:'left',
    font:{fontSize:20}
});
myview.add(mytext);

    It is, in fact, very possible. But I understand if you would like to use Views - In some cases I've found them to be very helpful.

    Try this then:

    var row = Ti.UI.createTableViewRow({className:'class1', height:160});

var myView = Ti.UI.createView({
    top:5,
    left:105
});
var myText = Ti.UI.createLabel({
    text:'Foo Bar Foo Bar Foo Bar Foo Bar',
    left:0,
    top:0,
    width:210,
    height:'auto',
    backgroundColor:'#FF0000',
    textAlign:'left',
    font:{fontSize:20}
});

myView.add(myText);

    That way we are positioning the view, rather than positioning the text, because no matter where the view is the text will stay within it at 0,0, which is the top-left corner of the view.

    You could also add a "top:5" property to the view, if you really want to go with a view. But I always prefer just adding children directly to the TableViewRow, rather than adding a View to it then adding children to that.
