Missing Alloy template in Titanium Studio 3

I've downloaded Ti Studio 3 and Alloy, managed to compile a few IOS apps and run it on the simulator. I started those apps with the Alloy template available in Ti Studio 3.

Now the Alloy templates are missing. When I go to File > New > Titanium Project I will not find the Alloy in the Available Templates box. I can still do 'alloy new' from the command line though. How can I fix the missing Alloy template?

    On this topic:

    https://jira.appcelerator.org/browse/TISTUD-3496

    I just removed the package by doing [sudo] npm rm -g [package] and the template is now being shown.

    • as a workaround, what i do now to build an Alloy project is to.

      1. use the Alloy CLI new command to create a project.
      2. import the project into Titanium Studio 3.
      3. open the properties view of the project and under 'Project Natures', checked Titanium Alloy (Beta) as the primary.
        I can then successfully compile and run the project in the IOS simulator.

      It seems that somehow the plugins in my Titanium Studio 3 for the Alloy is somehow screwed.
