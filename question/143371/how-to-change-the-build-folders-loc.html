  • hi all
    I solve it myself. use following step?
    1?use "ln -s" command make link from build folder to the new one in MAC.
    2?edit file "/Library/Application Support/Titanium/mobilesdk/osx/2.X.X.GA/iphone/builder.py",find "if command == 'xcode':"?the next 3 line is "project_dir = os.path …",change it to your real project path,like "project_dir = '/Users/tom/Titanium Mobile Space/testproject'"

    then the Titanium Mobile can run you project with the new build folder:)
