Push notification popup when app is closed

When a user recieves a push notification on iphone and the application is closed he gets an alert and when he clicks on he gets redirect to the application,
Can i detect this redirect so i can redirect him to the inbox after login?

— asked July 11th 2011 by Gerd Koetje

  • You should be able to register for the push notification callback. While that is not an indication that the user pushed the alert button to launch your app, you can set a flag in the callback that will put the user to the inbox the next time that app is opened.
