iPhone emulator - dumb(?) question

On the basis that there are no dumb questions, here's mine!

Can I run the iPhone emulator (only the emulator, no more than that) on a Windows machine? Or is it Mac only?

If the answer's "yes", what do I need to do to set it up?
If the answer's "no", I'll check my bank balance and think about buying a Mac!

I realise that I need a Mac in order to be able to create code to run on the iPhone itself. I just want to see how my app would look on an iPhone by using an emulator. I already have the Android emulator running OK (if slowly!).

    In theory, you need a mac to do that. In practice, the iphone simulator just has to think it is running on a mac. Some people have reportedly run OSX as a virtual machine on their Windows box.

    I don't endorse that method and have never tried it, and it should only be done with a real osx license for the virtual machine. If a person wanted to run both OS on a single box, I think the better approach is a good mac which then runs Windows as a guest, instead of the other way around.

    But if you absolutely need to try and do it with your existing hardware, try web searches for things like hackintosh or "virtual osx".

    YMMV.
