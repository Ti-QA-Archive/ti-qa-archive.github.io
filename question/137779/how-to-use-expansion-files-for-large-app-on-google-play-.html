How to use expansion files for large app on Google Play ?

I have an Android version of my app that is 241MB (mostly images and audio files). While I can download and test it directly to an Android device via USB, I can't upload it to the Google Play site for distribution due to the 50MB apk file size limit.

I have an iOS version of the app running smoothly on iTunes. How would I implement the expansion files in Titanium? I am using the 1.8.2 sdk and am willing to make an Android specific branch of my code should I need to re-write my media path references, say to a dedicated external directory.

I'm not seeing any support for large Android app size in Titanium that would lead to easy distribution on Google Play. Has anyone else come up with a solution for this? Much appreciate any help. Thanks.
