Renew ACS ios Push notification certificate

Hello,

It's time for our app to renew the apple push notification sertificate.

But when I click on the "remove" button in the ACS backend panel i get this image: action Deleting certificate will lose all the push notifications linked to this certificate.

Does it delete all registered push devices and everyone needs to register again? Or does it only remove the logs etc.?

  • No, It will not delete all the devices, as long as you certificate is correct. Those devices subscribed before can be used continually, no need to register again.
