How do I send an object using the fireEvent?

Hey there. I have another (rather dumb) question.
I'm writing for the desktop, and I am trying to get Window 1 to talk to Window 2.
Right now I am using (Window 1):

Titanium.API.fireEvent('go', {e:'some data'});

and then I am listening in Window 2 (Window 2):

Titanium.API.addEventListener('go', function(e){
var data = e.data;
alert(data);
}

The fireEvent and the eventListener are working, because the I actually receive an alert. However, the alert says 'undefined', which would mean that it isn't getting 'some data'. Can anyone tell me what I am doing wrong?

  • Are yes.. slightly confusing as you are using the same variable names..

    in your event listener.

    var data = e.data

    should be

    var data = e.e

    the data is passed as a set of key value pairs. the receiving element function(e) then contains a list of those pairs..

    so e.e would equal 'some data'..

    Hope I explained this ok.
