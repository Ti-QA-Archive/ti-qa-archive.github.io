View Layout with multiple Labels

Hi,
i've been playing around with this for hours and can't figgure this out. I am simply trying to create a view which holds a couple of labels. these labels should be ordered left to right.

Now, when i just create the view and add the labels, they show up. But when I set the layout property of the view to horizontal (or vertical, same effect), the labels do not show up any more.

Funny thing is: If I put two imgviews in there, they do show up ordered correctly after changing the layout of the view.

I'm using Mobile SDK Version 1.8.0.1

Here is my code:

var win1 = Titanium.UI.createWindow({  
    title:'Tab 1',
    backgroundColor:'red'
});


var view = Titanium.UI.createView({
   layout:'horizontal',
   borderRadius:10,
   backgroundColor:'white',
   width:300,
   height:300,
   top:20,

});


var Label1 = Titanium.UI.createLabel({
    text:"Text 1",
    color:'Black',
    font:{fontSize:20},

});

var Label2= Titanium.UI.createLabel({
    text:"Text 2",
    color:'Black',
    font:{fontSize:20},

});


Titanium.UI.currentWindow.add(win1); 

view.add(Label1);
view.add(Label2);
win1.add(view);

Anyone havin an idea / any help? Much apprechiated!

  • you need to set some width on your labels..

    var win1 = Titanium.UI.createWindow({
    title : 'Tab 1',
    backgroundColor : 'red'
});

var view = Titanium.UI.createView({
    layout : 'horizontal',
    borderRadius : 10,
    backgroundColor : 'white',
    width : 300,
    height : 300,
    top : 20,

});

var Label1 = Titanium.UI.createLabel({
    text : "Text 1",
    color : 'Black',
    width : 'auto',
    height : 'auto',
    font : {
        fontSize : 20
    },

});

var Label2 = Titanium.UI.createLabel({
    text : "Text 2",
    color : 'Black',
    width : 'auto',
    height : 'auto',
    font : {
        fontSize : 20
    },

});


view.add(Label1);
view.add(Label2);
win1.add(view);
win1.open();

    also see this documentation on layout for 2.0

    https://wiki.appcelerator.org/display/guides/Titanium+2.0+Layout+Changes
