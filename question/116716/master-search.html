Master Search

Hello All,

I'm creating a tab bar based app with one of the tabs a Search tab. I will have multiple tables containing software names, versions, vendors, etc. and need a 'master search' to search all of my tables at once. I'm able to use the built-in search for each table with no problem but, as you know, it only searches the table it's located in individually. I want to allow users to use a top-level search to find what they're looking for without having to drill down through various tables to get to what they want. Can someone point me to or provide example code for searching multiple tables at once and pulling the search results into a table view?
