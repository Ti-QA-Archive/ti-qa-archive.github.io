Alloy: open and animate new window

Hi guys, I'm looking for a way to open a new controller/view when clicking on a button.

For example in
indexController -> I click a button
-> secondWindow opens in an animated way

Do I need to use the Animation class that comes with Titanium or what's the way to get it right?

    Hi JL

    Try in this way to open new window with animation in IOS,

    casinoDetails as js file name, you want to open

    var casino = Alloy.createController('casinoDetail').getView();

   casino.open({
        transition : Ti.UI.iPhone.AnimationStyle.FLIP_FROM_LEFT
   });
