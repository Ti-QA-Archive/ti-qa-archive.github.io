Scroll View Verticle layout problem

Hi
I am trying to add a text field and a picker in the scroll-view, but they are not aligning vertically, instead they are overlapping on each other.

but i dont want to use height property or top property for the textfield or the picker .
Is there any way around this.
Here is my code 

var FormView = Ti.UI.createScrollView({
        contentHeight : 'auto',
        contentWidth:'auto',
        width : '98%',
        height:'100%',
        top : 110,
        layout : 'verticle'
    });

    var GotTitle = Ti.UI.createTextField({
        hintText : 'Got Title',
        width : 'auto',
        height : 'auto'
    });

    var GotCountry = Ti.UI.createPicker({
        selectionIndicator : true
    });

    for(var i = 0; i < allCountries.length; i++) {
        var rpw = Ti.UI.createPickerRow(allCountries[i]);
        GotCountry.add(rpw);
    }
    FormView.add(GotTitle);
    FormView.add(GotCountry);
    return FormView;

Any kind of help will be appreciated.
Thanks

    • Firstly, With width&#x2F;height set to &#39;auto&#39;, GotTitle wont be rendered. You can test this by comment out this line &#x2F;&#x2F; FormView.add(GotCountry);. So you need to set fix values/percent for width&#x2F;height to make it work.

    • Secondly, without top, GotTitle always be rendered at center of FormView. And you cannot change width&#x2F;height of Picker. So top seems be the only way to get GotTitle rendered.

    var GotTitle = Ti.UI.createTextField({
    hintText : 'Got Title',
    width : '80%',
    height : 40,
    top: 0
});
