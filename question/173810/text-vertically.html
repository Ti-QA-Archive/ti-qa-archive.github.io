    Hi

    @Amimul Hossain has the correct answer which he added in a comment - rather than the correct place as an answer.

    In xml

    // id param only added for next example
<Label id="lbl">Titanium \n appcelerator</Label>

    In controller

    $.lbl.setText('Titanium \n appcelerator');

    And for non-Alloy project;

    var lbl = Ti.UI.createLabel({
    text: 'Titanium \n appcelerator'
});
win.add(lbl);

  • Hi,

    I have tried to solve your problem and used the below code. So please check the following code. I hope, it will help you.

    index.xml

    <Alloy>
  <Window backgroundColor='#fff'>
        <Label id="lbl"> Titanium appcelerator </Label>
  </Window>
</Alloy>

    index.js

    var temp=$.lbl.text.trim().split(" ");
var txt='';

for (var i=0; i < temp.length; i++) {
  txt+=temp[i].charAt(0).toUpperCase() + temp[i].substring(1)+'\n';
};
$.lbl.text=txt;

$.index.open();
