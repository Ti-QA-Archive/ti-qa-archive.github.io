Code Completion

Hi,

I am wondering if there is a way to have code completion in Titanium Studio.

I already saw this and did it.

That get me to autocomplete the Ti API, however, I am looking for autocompleting my objects.

While in this simple lab project bh.db doesn't have that many functions, it is conceivable for it to have much more functions (so it is not so easy to program without code completion).

Note: excuse me if I am using the wrong terminology, I come from the .NET world.

I realize that this might be very difficult for the editor to do considering the dynamic nature of the language.
