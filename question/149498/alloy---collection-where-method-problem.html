Alloy - collection where method problem

Program complains that

Script Error = 'undefined' is not an object (evaluating 'models.length')

index.xml

<Alloy>
    <Collection src="nike">
    <Window class="container">
        <TableView id="shelf" dataCollection="nike" dataFilter="tableFilter" onClick="doUpdateSize" >
            <Require type="view" src="row" />
        </TableView>
        <Require type="view" src="bar" id="bottomBar" />
    </Window>
</Alloy>

index.js

function tableFilter(collection) {
    var result = collection.where({
        year : "2010"
    });

    alert(result.length); //Complaints raised!~
}

Table reflects the effect of execution of tableFilter, but evaluate result.length throws out the error.
Why?

Any idea?

Thanks

  • It doesn't look like you are returning anything from the tableFilter. This will undoubtedly cause a runtime error since you are required to return an array of models from your dataFilter function. Make sure to return "result" and you should be fine.
