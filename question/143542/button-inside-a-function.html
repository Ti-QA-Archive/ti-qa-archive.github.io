    • So you are creating the button and add the function to the window win;
      When you are calling it again all you do is to add it once more to the same window.

      My advice is to create a factory function:

      
function TopFunctions(e){
    var btnMain = Ti.UI.createButton(
            {
                top: 1,
                zIndex:1,
                width: 30,
                height: 33,
                left : 1,
                backgroundImage:'./menu.png',
            });
    return btnMain;
};

      and then

      win1.add( new  TopFunctions() );

win2.add( new  TopFunctions() );
