    in both platforms you can use GET, POST, PUT …

    • Android can only works asynchron
    • if you send on Android a buffer or TiBlob, then the send method modified the raw byte stream a little. It increase 0.4%.

    In normal use cases there is no problem, because you dont want to send synchron and you send object.
