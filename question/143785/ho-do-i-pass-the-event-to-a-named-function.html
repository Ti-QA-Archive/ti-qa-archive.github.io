Ho do i pass the event to a named function?

Hey!

so i have this code:

var someThing = function(e){
   e.source.height = 10;
}

and this code elsewhere:

var button = ...
button.addEventListener('click', someThing());
...

I know that i can inline the someThing function like so:

button.addEventListener('click', function(e){
   ... 
});

and when i do this, e.source will refer to the item that the event was enacted upon. However, when i use the someThing() method, I cannot pass the 'event' ~ if i put in an 'e' (ie someThing(e)) then i get an error because e doesn't exist, and if i don't include the reference then the someThing method gets an undefined on 'e'… what is the way to do this?

  • You want to do this instead:

    button.addEventListener('click',someThing); // pass the function object, don't call the function
