  • I had the same problems with android. What worked for me: Open Android SDK Manager, look at the Virtual Devices, there should be an AVD with the name like 'titanium_2_wvga854'. Click EDIT, verify the target drop-down list is: 'Google APIs (Google Inc.) - API Level 8'. Save and restart TI Studio. Mine was Android 2.2 - Level 8. Kitchen Sink is Googly because of the map example.
