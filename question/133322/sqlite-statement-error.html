  • The second insert depends on the first one (var id = DB.lastInsertRowId;). So make sure the first insert is correct:

    • Make sure data passed to insertRows is correct by:
    function insertRows(data) {
    Ti.API.info( JSON.stringify(data) );
}
    • Make sure you can run below statement directly in your SQLite DB ( any constraints violation ? ):
      INSERT INTO incidents (date, time) VALUES(date, intime)// Replace VALUES by data.date & data.intime in this statement.
