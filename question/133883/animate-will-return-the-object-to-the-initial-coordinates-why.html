animate will return the object to the initial coordinates. why?

I'm trying to move the top of a view with an animation..but when the animation finishes it returns to it's initial top

   var a = Ti.UI.createAnimation({
            top : -220,
            duration : 3000,
            autoreverse : false,
            repeat : 1
        });
        entryView.animate(a);

I've tried to set the top when the animation finishes 

   a.addEventListener('complete', function() {
        entryView.top=-220;
        });

But it still returns to the original top then sets it to the desired top..resulting into an ugly effect …

what I want to acomplish is that the view would just remain at the animated coords.
