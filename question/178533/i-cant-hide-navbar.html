    Hi,

    I've had the same issue and I tried hiding with AndroidManifest with no success.

    Finally I did it with code as mentioned here: http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.Android.ActionBar

        win.addEventListener("open", function() {
        if (Ti.Platform.osname === "android") {
            actionBar = win.activity.actionBar;
            if (actionBar) {
                actionBar.hide();
            }
        }
    });

    On each window you open. Add an EventListener to hide the actionbar once it's open.
    The only problem with this solution is that you can see the actionbar for a few milliseconds before it hides.

    cheers,
    Dan

    • Hey Mario.

      I just got it working via the tiapp.xml. And there's no more showing it for a few milliseconds. It is totally hidden!!

      In your tiapp.xml in the android section add this:

          <android xmlns:android="http://schemas.android.com/apk/res/android">
    <manifest>
        <application android:theme="@style/Theme.Titanium"/>
    </manifest>
</android>

      If you use 3.3.0 and later the actionbar will be hidden as per this documentation:

      http://docs.appcelerator.com/titanium/3.0/#!/guide/Android_Themes-section-34636181_AndroidThemes-TitaniumThemes
