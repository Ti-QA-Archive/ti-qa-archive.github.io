Distribute on device and test on device have different result..

Hello, I'm test on Android about my titanium program.

I have a webview to show web about HTML5 canvas and draw line chart.

I test on simulator and transfer to my android device is correct.

But I distribute apk file to install on my device, the webview content can't show any photo about HTML 5 canvas.

I have test Titanium SDK on 1.8.2~ 2.1.4 , but the result is same. ( the webview is show write page).

have any resolve method?

( I test on android SDK 2.1 and 2.3.3 version)

thank you very much.

  • I didn't face this type of problem till now, I have also implemented pie and bar graph using html5 and java script, but working fine on Android as well as iPhone, what you can do is

    1. just check your device log, You may be getting any crash on iPhone.
    2. Try to run the html file on safari browser, if it works, then it will definitely work on devices as well.
    3. Clean and rebuild the app, and try to run on iPhone simulator.
    4. Go to run app on device, if all the steps are full-filled.
