Can We use underscore js in titanium for android and ios platforms?

Can We use underscore js in titanium for android and ios platforms? if so how can i include it into mobile project and use in android and ios platfroms.Thank you

  • Yes you can. If you are using alloy, you can reference it as part of the alloy tools. If you are using standard titanium, copy underscore.js to your directory and then require it. Remember, titanium uses JavaScript so you can require and use any js library you want.
