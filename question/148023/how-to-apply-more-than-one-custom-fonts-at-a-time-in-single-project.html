  • Hi Sagar,

    Put font folder in your project in resource folder and use the font name display on title of font Viewer.

    For example :

    if you want to use Verdana.ttf then put it in to font > Verdana.ttf and use it like

    var lblTest = Ti.UI.createLabel({
    text : "Testing Fonts...",
    font : { fontSize : 24, fontFamily : "Verdana" }
    color : "red",
});

    If you are using iOS then may be you got problem to set font. please follow following steps if you got trouble in iOS.

    1) Compile project and go to build folder. take a info.plist file from your project build.

    2) Add Property Fonts provided by application in to plist file.

    3) Put info.plist file into your project directory, parallel to resource directory.

    4) put a label in the project and build and clean your project that shows your desired font.

    Best of luck…

    :)

    Note:

    Do not forget to clean and build your project after all changes has been done…
