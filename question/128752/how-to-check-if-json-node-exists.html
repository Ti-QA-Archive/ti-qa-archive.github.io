how to check if JSON node exists?

Hi all;

I have a problem parsing a JSON string.

Basically i'm making an app that fetches detailed information about activities; however, the returned string varies depending on the activity; for instance; some times the string contains a node 'prices' if the activity is paying, but when the activity is free the node is not there (in stead of containing a value 'null').

    var eventItem = JSON.parse(this.responseText).event.eventdetails;
var prices;

if ( eventItem.eventdetail.price ) {
    prices = eventItem.eventdetail.price.pricedescription;
}
