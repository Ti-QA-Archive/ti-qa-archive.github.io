  • Simply create a function that resets all of your checkboxes and invoke it every time your user changes screen.

    ###Here is how the function would look

    function resetCheckboxes() {
    var section = taskList.data[0];

    for (var i = 0; i < section.rowCount; i++) {
        var row = section.rows[i];

        row.hasCheck = false;
    }
}

    Basically what it does it that it loops through every single row from the TableView and then sets the hasCheck property to false.

    Then, all you will have to do is to invoke your newly created function when the user clicks on the next button.

    next.addEventListener('click', function(e) {
    resetCheckboxes();

    // Rest of the code just like before...
}

    Hope this helps.
