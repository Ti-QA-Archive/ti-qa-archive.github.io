  • Somehow adding this line to tiApp is not working:

    <ios>
   <plist>
      <dict><key>UIStatusBarStyle</key><string>UIStatusBarStyleLightContent</string></dict>
   </plist>
</ios>

    Only setting the Window property works:

    var self = Ti.UI.createWindow({
        // ...
        statusBarStyle: Titanium.UI.iPhone.StatusBar.LIGHT_CONTENT,
    });
