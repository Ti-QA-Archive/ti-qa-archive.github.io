Run an update function every 5 minutes

Hi,

I am working on an application on iPad, that has a two-way sync from a website (custom api).

At the moment, everything is pretty manual, so on the settings page, i have two buttons:

"Download Updates" - will basically load data that has been updated on the website, since last sync.

"Send Updates" - will send data that has changed on the device/app to the website, since last sync.

Now, all of this works just fine, but it's a manual process…

I'd like to automate this, so every 5 minutes, the code behind "Download" / "Send" updates will execute by-itself - without interfering with the frontend UI (non-blocking) and works when the app is in background also.

How can I achieve this?
