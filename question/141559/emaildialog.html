emailDialog

Hello

According to the following piece of code I can send mail, but what it does is open the android native application. My question is, I can send an e-push of a button to a predefined email and message?

var emailDialog = Ti.UI.createEmailDialog()
    emailDialog.subject = "Hello from Titanium";
    emailDialog.toRecipients = ['foo@yahoo.com'];
    emailDialog.messageBody = '<b>Appcelerator Titanium Rocks!</b>';
    var f = Ti.Filesystem.getFile('cricket.wav');
    emailDialog.addAttachment(f);
    emailDialog.open();

  • By definition the email dialog is just that, a dialog that allows you to compose an email. There is not built in method of sending an email without composing it first. If you wanted to do something like that, you'd have to send off a request to a service that handled sending the email.
