iTunes connect warnings

I just uploaded an iPad app package. I got three warnings I have not seen before, although the upload did get through. Not sure if this means the app will have problems in review, though. What do these mean, and is there anything I can do about it?

sdk 1.7.3, os 10.7.2, xcode 4.2, ios 5

  • I've seen this occur before. It usually happens because of a function or method was named the same as an Apple private function. ie: exists. The reason its a warning is because its impossible to tell if the actual reference to the function is a reference to the private function or a function that was created as part of your application code. It should not affect your review.
