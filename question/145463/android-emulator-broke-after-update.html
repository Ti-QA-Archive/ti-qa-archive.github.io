  • Ok, i also managed to run the emulator within titanium. I did the following:

    1) Open the console

    2) Locate the mobile SDK folder: cd ~&#x2F;.titanium&#x2F;mobilesdk&#x2F;linux&#x2F;2.1.2.GA&#x2F;android

    3) Open and edit the builder.py file: gedit builder.py

    4) Search for -avd and must get you right to the following:

            # start the emulator
        emulator_cmd = [
            self.sdk.get_emulator(),
            '-avd',
            avd_name,
            '-port',
            '5560',
            '-sdcard',
            self.get_sdcard_path(),
            '-logcat',
            '*:d,*,TiAPI:V',
            '-no-boot-anim',
            '-partition-size',
            '128' # in between nexusone and droid
        ]

    5) Add this element &#39;-force-32bit&#39; to the array to look like the following:

            # start the emulator
        emulator_cmd = [
            self.sdk.get_emulator(),
            '-force-32bit',
            '-avd',
            avd_name,
            '-port',
            '5560',
            '-sdcard',
            self.get_sdcard_path(),
            '-logcat',
            '*:d,*,TiAPI:V',
            '-no-boot-anim',
            '-partition-size',
            '128' # in between nexusone and droid
        ]

    Obviously for all the above, there is an error in 64bit systems.
