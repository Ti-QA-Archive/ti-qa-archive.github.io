Close Application - Move to background in iPhone

Hello everyone, I've read almost all the posts about this. All I want to do is close the app or send it to background if there is no internet connection, my iOS developer fellows do that in iPhone or iPad, but i don't understand how i can do it. Can anyone explain it?

  • You can use the change event of Ti.Network to capture a network change event

    /* When the app start */
if(!Ti.Network.online){
    //do something
}

/* In the code */
Ti.Network.addEventListener('change', function(e) {
    if(!e.online) {
        //do something
    }
});

    By my experience you can't send the app to background.

    I'm pretty sure an app should not terminate itself.

    Stefano
