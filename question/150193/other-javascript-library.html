  • You cannot use any library that accesses the DOM in a Titanium native app because there is no DOM to access. That means you can't use jQuery, YUI, etc. You can use any JS library that is not dependent on the DOM, such as Underscore, Backbone, moment.js, etc.

    The exception would be if you're using a webview to display HTML within a Titanium app. In that case, you could use any of those libraries within the HTML you're displaying inside that webview. You'd load them using the traditional <script src=""> tag.

    Titanium is not a web wrapper like PhoneGap or Sencha. You're manipulating native components with a JavaScript API. If you're not sure of the distinction, I encourage you to read (http://developer.appcelerator.com/blog/2012/05/comparing-titanium-and-phonegap.html)[this blog post].
