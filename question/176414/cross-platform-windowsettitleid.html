    Need to see some code to actually review it. This may be a; bug, syntax issue, misunderstanding of how it works etc. Will not know until we see how you have approached this.

    However I would ask that you test this - just to make sure things tie up together with title and i18n.

    win.setTitle(L('something'));

    Assumes that;

    • you will swap "something" for a valid string entry you have
    • have a variable called win - if not change it
    • if using Alloy then you have an id called win at which point it would look more like this;
    $.win.setTitle(L('something'));

    BTW I know this is not titleid - this is a test only.
