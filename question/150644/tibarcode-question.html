Ti.BarCode Question

Hi all, I am using the Ti.BarCode module, successfully for the most part, but have run into some issues. The scanner works for about 95% of all code 39 format scans (client uses code 39), but if more than a single letter is used in the code, it misreads the barcode. Now, I downloaded the newest ZXing scanner from Google Play and it reads the barcodes fine (and much faster). I am using the 2.3.4 Ti.BarCode module and noticed the scanner I installed from GP was 4.3.1. Are there any pre-compiled, later builds of the Ti.Barcode module I can install? Ones that have some of the latest upgrades from the ZXing project?
I am building for Note 2, Android 4.2.1 in Titanium 3.0.2GA on Windows 7. I appreciate any direction.

Thanks.

    Thank you for your responses. Because I had already downloaded the ZXing scanner from Google play, I made a call to installed scanners in my code. This allows the user to select which scanner he/she would like to use from the installed scanners. If the user selects a scanner from outside the Titanium app, the barcode scanned is placed in the clipbard and the app then pulls the code from the clipboard. This is all pretty seamless. Not a very elegant solution, but the client liked it.
