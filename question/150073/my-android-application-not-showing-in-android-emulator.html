  • If I'm reading your log correctly, your project's name is "alloy". You can't name your project the same as a folder in your Resources folder. Alloy will automatically add this folder to all projects. Name your project something else and it should work fine. This bug is documented in TIMOB-13129.

    • This is a common error with android dev. The emulator can be flaky at times. Here's a few things you can try:

      1. Just trying running it again while the emulator is open
      2. If that doesn't work, close the emulator and try the whole process again
      3. close the emulator and TiStudio and try again
      4. Sometimes the android tooling itself can get hosed. In that case you can manually restart the adb server like this and then try #2 and/or #3 again:
        adb kill-server
adb start-server
