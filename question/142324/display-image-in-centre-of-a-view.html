Display image in centre of a view

Hi,

When I display an image the user has selected from either the camera or the gallery, how do I display this centred in a view?

My code:

    var boundaryview = Titanium.UI.createImageView({image:'/images/Crop Image/Crop_Boundaries.png', top:topadd, left:'10dp', right:'10dp'});//,height:Ti.App.SCREEN_HEIGHT, width:Ti.App.SCREEN_WIDTH});
    view.add(boundaryview);

    var storedimage;
    if(fromCamera) {  // if from camera we take the image from the file system
        storedimage = Ti.UI.createImageView({image:Titanium.Filesystem.applicationDataDirectory + Titanium.Filesystem.separator + pictureName, top:topadd, left:'20dp', right:'20dp'});
    }
    else {
        topadd = topadd + 50;  // needs checking on iOS - 50 is fine on android
        storedimage = Ti.UI.createImageView({image:pictureName, top:topadd, left:'20dp', right:'20dp', height:'200dp'});
    }

    view.add(storedimage);

So I want the storedimage to appear centred inside the boundaryview…

I guess my problem could be if storedimage is larger than the boundary view?!

This is for iOS and Android.
