Log file in Android

Hello,

My costumer is having crashes in an Android app we're developing.

We need to create a log file to trace the action before crashes and understand the problem.

We're planning to pu the log file into the application folder on the device.

1- Is there a log system already in titanium for Android ?

1.1- If so, how do we put it in place and recover the logs ?

2- If not, is it possible to create a log file ?

2.1- Is there any methods that we need to use ?

Any information will be appreciated ?

Kind regards

  • You will need to plug the device in to your development machine, open up a command prompt/terminal

    then, CD to your android-sdk/platform-tools/ directory

    then, type .&#x2F;adb logcat

    This will give you a live debug log of what's going on in the device.
    If you need to filter to only see certain lines, use .&#x2F;adb logcat | grep &quot;something&quot;

    Hope this helps.
