Suggestions to reduce Project Size

Hi TItanium Dev team!

It has been noted that even the smallest XCode Project generated by Appcelerator takes 3MB and it's understandable while it includes EVERY FILE and EVERY FRAMEWORK.

What I would suggest you to split the entire architecture into modules(Similar to DoJoKit etc) where modules are included on demand. There should be a base Module which must be included with every project which contains all the core files required to run the project. Other than that all UI and non-UI modules like Accelerometer etc should be included when needed. This would really help to reduce the project size and decrease build time.
