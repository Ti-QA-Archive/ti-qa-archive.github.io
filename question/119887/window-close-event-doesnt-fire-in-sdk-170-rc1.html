window close event doesn't fire in SDK 1.7.0 RC1

Hi,

In my App I have a window (win1) in which I create and open another window (win2) using a button. In 1.6.2 I could catch the close event of win2 in win1. When I compile with sdk 1.7.0.rc1 the close event is not fired.

I open the window as follows:

but.addEventListener('click', function(e){
    var win2 = Titanium.UI.createWindow({  
        url:'../screens/win2.js'
    });

    win2.addEventListener('close', function(e){
        Ti.API.info('win2 closed');
        RunCode();
    })

    win2.open({modal:true});
});

Closing win2 is done via a click of a button as well.

I have tried creating the window outside the button click event. This doesn't make a difference. Also opening win2 with 'modal: false' doesn't do the trick.

Is it a bug or is it me? Help is really appreciated because I really would like to use the fastdev advantages of 1.7.0!

Thanks
Gino
