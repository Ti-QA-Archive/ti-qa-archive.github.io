Hello friends,
i have been working with Titanium Appcelerator for about 10 days and it's really cool.
i have a view in which i've created 8 subviews .. and i made a function to handle the click event of the small subview .. but i wanna add an id to each subview in order to identify it from other subviews .. something like tags in cocoa UI elements.
And this id shouldn't be visible UI element.
But in Titanium.UI.View i didn't find such thing and don't know how to add it to the view (something like subclassing and adding properties in native Objective-C).

    example, created a property on the view called "_tag"

    view = Ti.UI.createView({
   width:300, 
   height:100, 
   backgroundColor:'gray',
   _tag: 'my property'
});

    Scroll down to passing data on this page from the documentation for a complete description

    http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.UI.Window-object
