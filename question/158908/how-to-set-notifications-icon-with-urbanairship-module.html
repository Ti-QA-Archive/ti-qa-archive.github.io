But the app I'm building is using the urbanairship module. I don't see where I can set the icon there.

  • You need to give the icon name in the JSON string/hash while sending notification like :

    {
  "title": "Example",
  "alert": "Sample alert",
  "icon": "little_star",
  "badge": 3,
  "sound": "door_bell.wav",
  "vibrate": true,
  "score": 51,
  "custom_field": {
    "headlines": "Appcelerator Cloud Services Rocks!"
  }
}

    The icon's file name without extension. The file should be located under app's "res/drawable" directory.
