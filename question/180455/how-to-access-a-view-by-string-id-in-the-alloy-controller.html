How to access a view by string id in the alloy controller

How can you access a particular view with a string id?

I know you can simply use $.theViewId.doSomething()

But suppose you have
var id = "theViewId";

How do you access the view using that var id?
$.id.doSomething() does not work…

    Hi Emil,

    Yes you can…
    see this

    xml

    <Alloy>
    <Window id="winIndex" class="container">
        <Button id="btnTest" title="Hello" top=20></Button> 
    </Window>
</Alloy>

    js

    var strBtnTitle = "btnTest";
alert($[strBtnTitle].title);
