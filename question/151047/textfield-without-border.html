textfield without border

Hi,

Im creating a textfield on android :

here is its style in tss file : 

"TextField" : {
    borderStyle: Ti.UI.INPUT_BORDERSTYLE_NONE,
    left: 5,
    width: Ti.UI.FILL,
    font: {fontSize: '12sp'}
}

But i have still the border : look at this image : http://dzmob.net/textfield.png

as you see i want to make background of textfield but i have the textfield border that i dont need to show..

i tried border : 0 without result.

thanks for help

    Hi,

    Use borderColor

    "TextField" : {
    left: 5,
borderColor : 'white', // border color
    width: Ti.UI.FILL,
    font: {fontSize: '12sp'}
}

    http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.TextField-property-borderColor
