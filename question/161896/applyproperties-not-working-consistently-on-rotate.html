  • Not sure if you are aware but there is Ti.Gesture

    You can use the eventListener for 'orientationchange' to determine the orientation of the device more efficiently. An example for the listener would be like so:

    Ti.Gesture.addEventListener('orientationchange', function (e) {
    if (e.orientation === Ti.UI.LANDSCAPE_RIGHT || e.orientation === Ti.UI.LANDSCAPE_LEFT) {
        // code for if device is landscape
    } else {
        // code for if device is portrait
    }
});

    You can also use 'isLandscape()' or 'isPortrait()' methods to determine whether the device is landscape or portrait as these pass back boolean values.

    By using either of these suggestions you can determine the device orientation and hopefully apply the correct properties everytime! :)
