Disabling hardware keyboard

For an app i'm using a hardware scanner which is connected to the USB.

The problem is, that every time I disconnect and connect the scanner, Android sets the Hardware keyboard to enabled, so the soft keyboard doesn't pop up… I can change the Hardware keyboard back to disabled in the settings, but is there an easier way to do so?

It would be perfect if I could disable the hardware keyboard when I start up my app…!

    • the use this property to show software keyboard

      Titanium.UI.View  ====>> could be your app start view.
Titanium.UI.View.softKeyboardOnFocus = <one of this value>

One of Titanium.UI.Android.SOFT_KEYBOARD_DEFAULT_ON_FOCUS, Titanium.UI.Android.SOFT_KEYBOARD_HIDE_ON_FOCUS, or Titanium.UI.Android.SOFT_KEYBOARD_SHOW_ON_FOCUS.

Default: Titanium.UI.Android.SOFT_KEYBOARD_DEFAULT_ON_FOCUS
