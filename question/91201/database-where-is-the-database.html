Database? Where is the Database?

Hi,

When i do this:

 db = Titanium.Database.install('dataapp.sqlite', 'contacts');

for the first time, titanium understands and i have a database. But if i change the database 'dataapp.sqlite' like adding another table, it seems that titanium is using another database, since i cannot see the new table.

I am assuming Titanium is copying the file 'dataapp.sqlite' to another place. Where it is? This assumption is correct?

I realize that if i change the name from 'contacts' to 'contacts2' titanium grabs the new copy with the new table added.

Someone can point me WHERE is this copied database if this is the case, so i can add tables and such?

Thanks in advance.

    The database is located somehwere on your filesystem.

    Mine is here:
    /Users/W0utR/Library/Application Support/iPhone Simulator/4.1/Applications/0E578C36-6720-4B9C-A9E2-5947FC2A88C1/Library/Application Support/database

    Just check the map Applications for your app and there should be a database.
