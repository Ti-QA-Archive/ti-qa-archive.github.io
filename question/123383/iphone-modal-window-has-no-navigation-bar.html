iphone modal window has no navigation bar

Hi,

I have an app which opens 2 modal windows back to back…and when the second modal window opens, it has no navigation bar !!!!

And I need this navigation bar, because I'm using the leftNavButton to add a "CANCEL" button… !
in summary the code is like

var modalWindow1 = Ti.UI.createWindow(
    {
        url : 'View/modalWindow1.js',
        navBarHidden : false,
        modal : true,
        title: "modal window 1",
        fullscreen : false
    });
modalWindow2.open();
var modalWindow2 = null;

// EVENT FIRED IN View/modalWindow1.js
Ti.App.addEventListener("OPEN_SECOND_MODAL_WINDOW", function()
    {
        Ti.App.removeEventListener("OPEN_SECOND_MODAL_WINDOW", arguments.callee);

        modalWindow2 = Ti.UI.createWindow(
            {
                url : 'View/modalWindow2.js',
                navBarHidden : false,
                modal: true,
                title : "modal window 2",
                fullscreen : false
            });
        modalWindow2.open();
    });

and the thing is that when the second "OPEN" is closed, it slides down the first modal window, and the second modal window slides up…..but without navigation bar…

I even tried to verify by adding a property top : 100 to the second modal window and it really has no navigation bar….

I think this a is a bug !! (Paul or others to confirm it please).

You could say why I do not close the 1st modal window in the code, and then open the second one …..well I first did this way and I got nice exceptions where it says something like a modal window transition was called while another one was not finished, and the app crashes…

Any advices, solutions in order to cope with this issue ??
It is really urgent, I have a demo to present in a very short time, and I need to fix this (and others things, too)…

I'm using SDK 1.7.1 and Mac OS X 10.6.7

Thanks for your help.

    Try working around this problem with normal windows. You can set the leftNavButton of normal windows as well.
