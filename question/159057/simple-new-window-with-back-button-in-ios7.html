Simple new window with back button in iOS7?

This code works just fine with Titanium SDK 3.1.2, but when I try to use to with 3.1.3, there is no nav box at the top at all and the button that is supposed to exist in the top left is non-existent. How can I create that nav bar effect with 3.1.3? Is the only way with NavigationWindow?
I am new at this, thank you for your help.

index.xml

<Alloy>
    <Window backgroundColor="white" modal="true">
        <Label onClick="openWin1">Open Window</Label>
    </Window>
</Alloy>

index.js


function openWin1(){

    var newWindow = Alloy.createController('win1').getView();
    newWindow.open();

}
$.index.open();

win1.xml

<Alloy>
    <Window id="win1" modal="true">
        <View backgroundColor="green" />
<LeftNavButton>
        <Button onClick="closeWin">Back</Button>
</LeftNavButton>
    </Window>
</Alloy>

win1.js

function closeWin() {
    $.win1.close();
}
