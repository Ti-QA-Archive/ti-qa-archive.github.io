    • You can set either the let or right nav button like this;

      var btn1 = Ti.UI.createButton({
    title: 'Left'
});
btn1.addEventListener('click', function (e) {
    alert('Left Pressed');
});
win.setLeftNavButton(btn1);
var btn2 = Ti.UI.createButton({
    title: 'Right'
});
btn2.addEventListener('click', function (e) {
    alert('Right Pressed');
});
win.setRightNavButton(btn2);

    • Notes;

      • This only works if the window is one of the following;
        • windows with modal property set to true
        • Window that is a child of a Navigation Group
        • Window that is a child of a Tab Group
      • If you have a back button normally in the left position setting a new button will STOP this default behaviour

    • Having not seen your code this is a hard answer. But the easiest and most common place would be under your window declaration code.

      var win = Ti.UI.createWindow({  
    backgroundColor: '#fff',
    title: 'Window',
    height: Ti.UI.FILL,
    width: Ti.UI.FILL
});
var btn1 = Ti.UI.createButton({
    title: 'Left'
});
btn1.addEventListener('click', function (e) {
    alert('Left Pressed');
});
win.setLeftNavButton(btn1);
var btn2 = Ti.UI.createButton({
    title: 'Right'
});
btn2.addEventListener('click', function (e) {
    alert('Right Pressed');
});
win.setRightNavButton(btn2);
win.open();
