    Hello Tim,

    From my understanding, You want to open the new window when you clicked on the seetings.
    If I am right then just try below way : 

    function doSettingsClicks(e) {
    var settingWin = Alloy.createController("settings").getView();
    settingWin.open();
}

    Let me know if it helps.
