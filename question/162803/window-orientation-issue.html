Window Orientation issue

Hi,

In my app, there are two windows. Example windowA and windowB. WindowA supports all orientations but windowB supports only portrait mode. And my flow is windowA > windowB.

Now problem is that, when I close the windowB in landscape mode than WindowA successfully get in landscape mode as it supports all orientations. But, when, I again opens windowB than it opens in landscape mode instead of portrait.

Titanium SDK: 3.1.3 (I have also tried on 3.2.0) ,
Platform: iphone ,
iphone OS version 6.0 and 7.0

Pooja

  • Hi Pooja,
    create two js file such as window1.js and window2.js

    code for window1.js

    var win1 = Titanium.UI.createWindow({      
        title:'Test',
        backgroundColor:'#fff',
        layout: 'vertical'
});

win1.open();

    code for window2.js

    var win1 = Titanium.UI.createWindow({      
        title:'Test',
        backgroundColor:'#fff',
        layout: 'vertical'
});

Ti.Gesture.addEventListener('orientationchange', function(e) {
    Ti.Android.currentActivity.setRequestedOrientation(Ti.Android.SCREEN_ORIENTATION_PORTRAIT);
});


win2.open();
