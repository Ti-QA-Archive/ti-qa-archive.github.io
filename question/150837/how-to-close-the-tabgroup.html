mySalesWindow.xml 

<Alloy>
     <TabGroup id="myTabgroup">
         <Tab title="Sales">
             <Window id="salesWin">
                  <TableView id="salesTable" rowHeight="90" /> 
                  <Button bottom="0" left="0" height="8%" width="50%" onClick="backToHomeScre                                    en">Back</Button> 
              </Window>
          </Tab>
        <Tab title="Order">
             <Window id="orderWin">
                  <TableView id="orderTable" rowHeight="90" /> 
              </Window>
          </Tab>
        <Tab>
        - - - - - 
        - - - - - 
        </Tab>
    </TabGroup>
</Alloy>

mySalesWindow.js

function backToHomeScreen(){     

     var homePage=Alloy.createController('index').getView();
     homePage.open();

    $.mySalesWindow.close();         
}

to go back to the homepage i m trying to close the 'mySalesWindow' page, but on clicking the back button my application crashes., My be this is not the correct way to close the Tabgroup.., Pls suggest me how to do. Thanks.

  • Your application is crashing because by default exitOnClose property is True if tab group is opened on top of the root activity. So set it to false on create. And then from your mySalesWindow you can close tab group using $.myTabgroup.close().
