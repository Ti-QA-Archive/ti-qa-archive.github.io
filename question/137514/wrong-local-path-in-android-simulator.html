  • Taking a wild guess because you post is, well, never mind…

    Add a forward slash in front of your file references for Android.

    Examples:

    Ti.include(&#39;&#x2F;libs&#x2F;myfile.js&#39;) instead of Ti.include(&#39;libs&#x2F;myfile.js&#39;)

    require(&#39;&#x2F;libs&#x2F;myfile&#39;) instead of require(&#39;libs&#x2F;myfile&#39;)

    var myimage=Ti.UI.createImageView({&#39;image:&#39;&#x2F;images&#x2F;myimage.png&#39;}) instead of varmy image=Ti.UI.createImageView({&#39;image:&#39;images&#x2F;myimage.png&#39;})
