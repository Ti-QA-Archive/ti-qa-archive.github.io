  • We dont have any apps where we changed the status bar on the fly. All are set in the tiapp.xml (in the case below, white)

    <key>UIStatusBarStyle</key>
  <string>UIStatusBarStyleLightContent</string>

    In our index.js had to add a hack to get the status bar stuff to set up right (you can see a discussion in the napp drawer issues section on github

    $.drawer.open();

//bloody ios7 status bar hack
if (OS_IOS) {
    var tmpWin = Ti.UI.createWindow({
        backgroundColor: 'transparent',
        zIndex: 1000,
        touchEnabled: false
    });
