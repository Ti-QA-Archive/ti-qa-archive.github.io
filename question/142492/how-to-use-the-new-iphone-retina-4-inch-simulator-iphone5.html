How to use the new iPhone retina 4-inch Simulator (iphone5)

Hello,

iOS 6 is out know - should I be happy or should I cry? :-)

How can I start the new iPhone Simulator, which come with the Xcode 4.5? When I choose Hardware -> iPhone (Retina 4-inch) the App and Simulator crashes

I use the RC Titanium SDK 2.1.3
Any hints to start the new Simulator directly from my projects?

    I assume you are using XCode 4.5.

    Open XCode and choose XCode > Open Developer Tool > iOS Simulator and you'll be able to switch between all the different devices from the Hardware Menu without it crashing.

    When you run from Titanium Studio again, it will 'remember' your last Hardware settings.

    Bonus: You can also download previous iOS versions through XCode > Preferences > Downloads and then previous iOS versions will show in Titanium Studio and the iOS Simulator.
