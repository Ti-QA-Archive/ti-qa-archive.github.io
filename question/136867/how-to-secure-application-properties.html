    • I honestly assume nothing is safe on Android. grin

      • iOS and android - as soon as the device is rooted/jailbroken the magic wall between apps disappears and an all access pass is granted around the file system.

      • Android APKs represent some of the weakest 'binaries' in existence. The very nature of byte code make them susceptible to breaching.

      • With API Level 14, the KeyChain Class has arrived. I haven't played with it or tried to wrap a module for it yet. That task is on the whiteboard here, just not high priority yet.
