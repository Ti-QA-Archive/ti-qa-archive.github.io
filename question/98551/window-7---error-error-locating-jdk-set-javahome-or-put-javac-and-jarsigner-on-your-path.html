    JAVA_HOME = C:\Program Files\Java\jdk1.6.0_21

    move the jdk folder to, for example:

    JAVA_HOME = C:\Java\jdk1.6.0_21

  • Spend hours to fix problems on my 64 bit Windows 7. Finally I got Titanium Studio working. The follow are my suggestions for 64 bit Windows 7:

    1. Make sure you are using JDK 1.6 32 bit edition and add JAVA_HOME variable point to your install location and add JAVA_HOME into your PATH variable.

    2. Download Android SDK and install on the path with no space between. e.g. C:\Program Files (x86) won't works. I installed on C:\Android.

    3. When you configure Android SDK directory in Titanium Studio and you see message "Could not locate the Android SDK at the given path", make sure the path is correct (where you can locate SDK Manager.exe). If you sure the path is correct, execute SDK Manager and download SDK Platform Android 1.6, API 4, revision 3.

    Hope this help
