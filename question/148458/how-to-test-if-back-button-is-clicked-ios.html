How to test if back button is clicked iOS

right now I have a window opened by a tab. In that window is a view, in that view is a button. When the button is clicked it opens a new window, and the window replaces the old window with: Ti.UI.currentTab.open(nextWin);. So now in the top title bar a "Back Button is created, with the text of the previous window. What I need is to be able to to test if the "Back Button" was clicked.

Something like:

if('backButtonClicked' == true) {
    //do something
}

  • You cannot do that. However you may intercept the close event on the window you do open like so:

    win.addEventListener('close', function() {
    console.log("closing window");
});

    If you need the "win" variable (like in your case, where you have the tabbar), use Ti.UI.currentTab.window for it.

    You can replace the default back button with your own button

    win.setLeftNavButton(button);

    Drawback is that this button is not shaped like a left-arrow.
