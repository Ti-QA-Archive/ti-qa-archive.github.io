how to change the backgroundGradient on android at touchstart and touchend?

How to change the background gradient on the android platform at touch start?

I set background gradient on the button, it have effect.

But I want to set touch start change other gradient, and touch end back original gradient.

Even I set single color is not work.

my code is below:

var nextBtn = Ti.UI.createButton({
            title : "test",
            width : 240 ,
            height : 160 ,
            bottom : 80 ,
            font : {
                fontSize : '20sp'
            },
            backgroundGradient : {
                type : 'linear',
                colors : ['#6DAAD3', '#5085B6'],
                startPoint : {
                    x : 0,
                    y : '0%'
                },
                endPoint : {
                    x : 0,
                    y : '100%'
                }
            },
            borderWidth : 1,
            borderRadius : 5,
            borderColor : '#456F9A',
            color : "#FFFFFF"

        });
        nextBtn.addEventListener("touchstart", function() {
            e.source.backgroundColor = "#4E89C5";
        });
        nextBtn.addEventListener("touchend", function() {
            e.source.backgroundGradient =  {
                type : 'linear',
                colors : ['#6DAAD3', '#5085B6'],
                startPoint : {
                    x : 0,
                    y : '0%'
                },
                endPoint : {
                    x : 0,
                    y : '100%'
                }
            };
        });

thank you very much!

    Hi

    Try this, your original code missed the e in the event functions, so when you referenced it inside it failed. Also you may need to reset the backgroundGradient, in the touchstart event, I have tweaked your code.

    nextBtn.addEventListener("touchstart", function(e) {
    e.source.setBackgroundGradient(undefined);
    e.source.setBackgroundColor("#4E89C5");
});
nextBtn.addEventListener("touchend", function(e) {
    e.source.setBackgroundGradient({
        type : 'linear',
        colors : ['#6DAAD3', '#5085B6'],
        startPoint : {
            x : 0,
            y : '0%'
        },
        endPoint : {
            x : 0,
            y : '100%'
        }
    });
});
