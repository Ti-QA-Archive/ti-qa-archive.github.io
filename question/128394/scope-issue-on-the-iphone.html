Scope issue on the iPhone

I can get this code to run on both Android and the iPhone emulators:

Ti.App.myApp.UI = 
    { 
        createMyView: function() {...}
    };

But I can't get this to work on the iPhone simulator, only the Android simulator:

Ti.App.myApp.UI.createMyView = function() {...}

The error I receive when the app tries this function is:

Result of expression 'Ti.App.myApp.UI.createMyView' [undefined] is not a function.

— asked November 18th 2011 by Tom Lyne

    I go a little differently:

    MyApp = {};
MyApp.UI = {};
MyApp.UI.createMyView = function(){};
