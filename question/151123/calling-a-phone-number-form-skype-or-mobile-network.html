  • hi Kiran,

    A typo error in above code. Use the following code.

    var url = "skype://"
var username = "kapil130187"
if(Ti.Platform.openURL(url)){
    Ti.Platform.openURL(url + username +"?call");
} else {
    alert("Please install skype");
}

    This will directly open skype and ask you for call.
