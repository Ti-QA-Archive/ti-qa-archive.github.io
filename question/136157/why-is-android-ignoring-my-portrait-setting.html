    the normal code is :

    var win = Ti.UI.createWindow({
fullScreen : true,
orientationModes : [Ti.UI.PORTRAIT]
});

    but for android orientation you have to right it like this:

    win.orientationModes = [Ti.UI.PORTRAIT];

    have you tried this??
