BorderRadius not working on Android versions >= 4.0.0

Hey all.

I've run into an annoying problem I can't solve, and hope to someone has a solution?

I've got lots of buttons and labels in a game I'm making, most of them have a borderRadius assigned, for rounded corners. This works fine on Android versions below 4, but when I run the app on anything newer, there's no rounded corners - just square corners.

Here's a screenshot showing the problem: http://bitfabrikken.dk/borderRadius.jpg. I assume it's a general problem and not something I need to submit any source code for, as it's a simple property that seems to be acting up.

Can anyone help?

I'm using Titanium Studio, build: 2.1.2.201208301612, win32 on a Window 7 Pro running on a 64bit intel cpu.
