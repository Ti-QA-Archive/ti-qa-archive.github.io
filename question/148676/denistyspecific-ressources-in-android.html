Denistyspecific ressources in Android

Hi,
I use the recommanded way of naming folders (res-long-port-ldpi etc..) in Android to put my images.
The problem is that there is a total of 10 folders :
res-long-land-hdpi
res-long-land-ldpi
res-long-port-hdpi
res-long-port-ldpi
res-notlong-land-hdpi
res-notlong-land-ldpi
res-notlong-land-mdpi
res-notlong-port-hdpi
res-notlong-port-ldpi
res-notlong-port-mdpi

and here for instance the real diff between all hdpi folder is the default.png file, the rest image files are the same (and there is a lot of them). Same thing occurs with the mdli and ldli.
the problem is that I have a lot of duplicate files for nothing since the main diff are the default.png.

So my question is how to get the 10 folders only for the default.png and 3 other folders (hdpi, mdpi, ldpi) for the images.
Is such thing possible and how to name the folder ?

Regards

    You might try moving all of the orientation and density independent images directly under Resources/android and only leave the orientation/density dependent default.png files in the appropriate Resource/android/image/res-xxx directory.

    • This documentation for the layout of Android resources suggests that you could store a resource for both portrait and landscape by putting it in a directory named by removing the 'land' and 'port' from the sub-directory name under Resources/android/images. 

      For instance, rather than putting the resource in both:
    Resources/android/images/res-long-land-hdp
    Resources/android/images/res-long-port-hdpi 

you might try putting it under:
    Resources/android/images/res-long-hdpi
