Are there any plans on Titanium implementing support for every type of drawable for Android?

I don't know if this question should be asked here, just wanted to know if support for every type of drawable defined in Android is being considered at all or what is the current state of it.

By Android drawables, I mean the ones defined in the following link Drawable Resources.

The use of drawables reduces much work in my opinion, and the portability is another thing that I really like about them. Currently Titanium supports simple images or 9patches, but if greater support for other kind of resources were to be added I think many of the Android developers using Titanium would consider it like a great addition to the SDK.
