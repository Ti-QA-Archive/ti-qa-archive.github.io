MobileWeb and Pickers...

I was trying to use a date picker in MobileWeb, however, even though the picker renders properly, the values don't seem to be available… (getting errors where the picker doesn't pass the value that is clicked). I imported the latest Kitchen Sink to take a look, but turns out that the Picker is not an object that KS exposes when you run it in a mobile browser… So does that mean that the Picker object is not supported in MobileWeb or has problems?

    Unfortunately browser support for date-picker is really spotty. If you check out Can I Use you will see that only Opera fully supports it. For what it's worth, the Mobile Web platform does support them on browsers that support them natively.
