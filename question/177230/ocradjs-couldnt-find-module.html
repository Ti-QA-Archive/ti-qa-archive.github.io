    Hi

    First things first, have you added the module to your tiapp.xml so that it's available to your app?

    Next, when you require a library you don't include the file extension. So it should simply be:

    var ocrad = require("ocrad");

    Lastly, is ocrad the actual name of the module? ie. no suffix of "com." or domain names
