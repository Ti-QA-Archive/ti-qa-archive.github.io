  • Hey Jay,

    If you can handle this two way.

    First Way : Register click event listener for all rows and open specific windows according to your requirement as you said row1 -> click -> open window 1 etc etc . .

    Row1.addEventListener('click', function(e){
    var win1 = Ti.UI.createWindow({
        url: 'win1.js'
    });
    win1.open();        
});

Row2.addEventListener('click', function(e){
    var win2 = Ti.UI.createWindow({
        url: 'win2.js'
    });
    win2.open();        
});

    Second Way : You can handle click event of table view and check index of row clicked. Based on row index you can open window which you want.

    tablview.addEventListener('click', function(e){
    if(e.index == 0){
        var win1 = Ti.UI.createWindow({
            url: 'win1.js'
        });
        win1.open();        
    }else if(e.index == 1){
        var win2 = Ti.UI.createWindow({
            url: 'win2.js'
        });
        win2.open();
    }
});

    Cheers.:)
