Renaming of workspace

Hi,
how can I change the name. I'm afraid if I change only in filesystem I got an error.
I want to develope an android module and thats why I need a path without whitespace.

Rainer

    Do the following steps:

    • Close all the projects in workspace
    • Close Titanium Studio
    • Move the directory .metadata from ~1 to ~&#x2F;new_workspace
    • Open Titanium Studio and when prompted for the workspace selection set it to ~&#x2F;new_workspace
