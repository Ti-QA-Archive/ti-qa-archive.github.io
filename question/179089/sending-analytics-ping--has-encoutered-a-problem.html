  • Accepted Answer

    delete the folder with the 3.4.1 update in it. you can find this folder in the titanium studio logs - this will revert you to 3.4.0.

    Or use the cli command ti info -t , that will show the location of your sdk installs.

    i think your download of the update might have been corrupted? I don't know, i had problems with 3.4.1 myself, i deleted the 3.4.1.GA folder (which means I'm back to using 3.4.0.GA), and everything is normal.

    Titanium cli has a command to do this, but at least for me, that command didn't do anything.
