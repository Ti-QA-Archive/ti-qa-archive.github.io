  • Jim

    Try one of the following:

    1) in Titanium Sutdio, use Project > Clean function and try launching your app again

    If this doesn't work:

    2) follow these steps

    • close Titanium Studio and all emulators
    • start the Android SDK Manager. Go to Tools > Manage AVDs
    • delete all existing virtual devices
    • close the AVD Manager and the SDK Manager
    • start Titanium Studio
    • use Project > Clean function
    • launch app

    Let us know if either of these resolved it for you.

    Cheers
