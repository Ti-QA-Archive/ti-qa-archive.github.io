Focus - Blur on Window causes problems with view references

Application type: mobile
Titanium SDK: Titanium Command-Line Interface, CLI version 3.4.1, Titanium SDK version 3.5.0.GA
Platform: Android
Device: physical device (all)
Host Operating System: Windows 7
Titanium Studio: Titanium Studio, build: 3.4.1.201410281727

I have a window with a listview and a button, which causes my listview to be removed and added again with another entries. all works fine, as long as my window get blur on it and loose focus. After getting focus again, and I hit the button, the listview won't be removed and a new listview reference will be added behind the current one. It seem to me, that if my window loose focus, my view references will be lost. I have no idea how to handle this problem.
