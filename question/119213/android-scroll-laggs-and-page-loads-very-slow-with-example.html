    • By the way, the workaround is pretty simple. All you have to do is to change Ti.UI.createTableView to Ti.UI.createScrollView, and the same for Ti.UI.createTableViewRow, change that to Ti.UI.createView.

      Next, write a setData function to manually place rows onto the tableView, i.e. 

      tableView.setData = function(data){
     for (var i in data){
          tableView.add(data[i]);     // data is array of views
          // code to add separator lines
          // code to index rows; data[i].index = i
     }
}
