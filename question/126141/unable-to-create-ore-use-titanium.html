    Looks like the Titanium SDK got corrupted in download… Do this:

    1. Open studio
    2. Click the help menu from the toolbar
    3. Click "Install Titanium SDK from URL"
    4. Paste in this link - http://api.appcelerator.net/p/v1/release-download?token=G4skNAN2
    5. Click finish

    This will re-download and install the SDK for you.

    Let me know how it goes!
