setList() error

i found a confusing issue with properties.setList()..

if i have an array and i fill it like that:

var a = [];
a[0] = 10;
a[1] = 20;
a[2] = 30;
a[3] = 40;

Ti.App.Properties.setList('myA',a);
var myA = Ti.App.Properties.getList('myA');
alert(myA.length) // shows 4

but doing like that

var a = [];
a[0] = 10;
a[1] = 20;

a[3] = 40;


Ti.App.Properties.setList('myA',a);
var myA = Ti.App.Properties.getList('myA');
alert(myA.length) // shows 2

i would expect to get myA.length=4 since myA[3]=null. the problem is, that i need to work with the index so i am interested in having some null elements.
