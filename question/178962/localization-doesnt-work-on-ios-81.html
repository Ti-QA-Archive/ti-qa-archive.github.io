  • Hi Fabio Codebue,

    I've tested your issue for iOS 8.1 and it's working as expected for me. Please check my test code along with the screenshot below.

    Test Code:

    var win = Titanium.UI.createWindow({  
    backgroundColor:'#fff'
});

Ti.API.info('Titanium.Platform.locale : ' + Titanium.Platform.locale);

Titanium.Platform.locale = 'it';
Ti.API.info('currentLanguage : ' + Ti.Locale.currentLanguage);
Ti.API.info('Localized string : '+ L('welcome_message'));

var aLabel = Ti.UI.createLabel({
    text: L('welcome_message'),
    color : '#000',
    font : {fontSize:30},
    height : 'auto',
    width : 'auto',
    top : 50,
    textAlign : 'center'
});

win.add(aLabel);
win.open();

    Shreenshot:

    Test Environment:

    Ti SDK 3.4.0.GA

    Ti CLI 3.4.0

    Ti Studio 3.4.0

    iOS 8.1 (iPad & iPhone)

    Clean your project before running and test it on real devices. Also change your language from device settings. Hope this will help you to solve your problem.
