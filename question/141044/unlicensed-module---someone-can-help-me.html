  • Make sure you are logged in to Titanium Studio with the account that has the module license applied to it.

    You can also try creating a new project (still logged in with the account mentioned above) and try the module with it. Assuming it works, you can either continue with using this project, or go to tiapp.xml and copy over the guid to your existing project.
