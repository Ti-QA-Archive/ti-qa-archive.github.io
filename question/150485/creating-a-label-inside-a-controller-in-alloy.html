Creating a label inside a controller in alloy

Using alloy, I'd like to create a label inside index.js and add this newly created label to the window…

I have the following code: 

function createCustomLabel(){

    var createCustomLabel = Ti.UI.createLabel({
        id: "index_label_value",
        top: "15%",
        left: "70%",
        font: Alloy.CFG.fonts.large,
        text: "I'm a label",              
    });

    $.index.add(createCustomLabel);
}

I then call the function in my index.js like so: 

createCustomLabel();

but I keep getting a KrollWrapper: 0xb641630 instead of the actual label on the window…What am I doing wrong?

    Hi Benjamin,

    You need to called open method for index.js

    Following seems working for me.

    function createCustomLabel(){

    var createCustomLabel = Ti.UI.createLabel({
        id: "index_label_value",
        top: "15%",
        left: "70%",
        text: "I'm a label",              
    });

    $.index.add(createCustomLabel);
}
createCustomLabel();
$.index.open();
