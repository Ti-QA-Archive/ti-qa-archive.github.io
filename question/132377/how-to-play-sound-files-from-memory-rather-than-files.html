How to play sound files from memory (rather than files)?

Hi, in my app I have some unique sounds I play using code such as:

var s = Ti.Media.createSound({url:'test.wav'});
s.play();

I would like to ship my app with encrypted .wav files, load them encrypted .wav files into memory, decrypt them in memory, and then play the decrypted bytes of the WAV file from memory. I know how to do all the encryption/decryption steps.

What I don't know is how to use Ti.Media.Sound to play the decrypted bytes (now in memory in the same byte sequence as the unencrypted file would have been on disk). Does anyone how how to do that?

Thanks in advance for your help! Titanium rocks!
