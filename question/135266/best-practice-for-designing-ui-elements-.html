  • I don't think you can actually do the above kind of code which can actually work across all kind of smartphones.. well you can get it to simply work but you don't want to see font that actually look nice in Samsung Galaxy II, look horribly large in Samsung Galaxy Note for an example. And some of them are not having the same screen ratio to another. And DIP unit is only helping just in case if all screens are having the same screen ratio but different density.

    Honestly what I would do is probably to segment the code into this:-
    1) Extra Large screen device like Ipad or Samsung Galaxy Note/Tab
    2) Large like Samsung Galaxy II, Galaxy or probably Sony Xperia.
    3) Smart for some quite old phones.
    So the code can probably look like below:-

    var layout;
if isIphone then
   layout = require('user_iphone');
else if isIpad then
   layout = require('user_ipad');
else if (isLargeScreen)
   layout = require('user_android_large');

var username = Titanium.UI.createTextField({

    color:'#336699',
    top:layout.getTop()
    left:layout.getLeft(),
    width:layout.getWidth(),
    height:layout.getHeight(),
    hintText:'Username',
    keyboardType:Titanium.UI.KEYBOARD_DEFAULT,
    returnKeyType:Titanium.UI.RETURNKEY_DEFAULT,
    borderStyle:Titanium.UI.INPUT_BORDERSTYLE_ROUNDED
});

    There is a bit simplification there, but that is the idea. Probably there are better examples out there which have been tested against real app.
