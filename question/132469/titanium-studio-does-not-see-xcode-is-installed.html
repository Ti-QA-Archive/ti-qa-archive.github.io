Titanium Studio does not see Xcode is installed

I haven't used Titanium Studio on my new MBP since I migrated it from my old MBP. Now,after doing all the Titanium updates and all, Studio says I need Xcode, even though I have 4.2 installed. How do I get it to see this? Clicking "configure" in prefs only opens the web page in Studio with a link to the App Store.

1.0.9.201202141347, OS 10.7.3

    I assume that you can start Xcode on its own and it seems to run ok.

    There is a tool called xcode-select that was designed to let us have multiple Xcode versions and you could switch between them.

    It might be that an environment variable is missing that Ti needs (i can't say, guessing here)

    xcode-select -switch &#x2F;Developer from a terminal might reset your environment (assuming your Xcode install is in /Developer)
