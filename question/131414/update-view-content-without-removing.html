update view content without removing

Hello, I have a simple question

var view = Ti.UI.createView({
    width:'100%',
    height:50,
    backgroundColor:'#000'
    ...
    ..
    .
});
main_view.add(view);

Is there a way to update "view" with another declaration avoid removing and adding as new?

For example 

//this doesn't work
main_view.children[0] = Ti.Ui.createView({...});

  • You can still reference view directly if you're still in the same scope of a scope lower.

    var view = Ti.UI.createView({
    width:'100%',
    height:50,
    backgroundColor:'#000'
    ...
    ..
    .
});
main_view.add(view);

var button = Ti.UI.createButton({
    title: "buttony"
});

button.addEventListener("click", function (event) {
    var label = Ti.UI.createLabel({text:"label text"});
    view.add(label);
});

    If you want to replace the view completely then you'll have to main_view.remove(view) then main_view.add(new_view)
