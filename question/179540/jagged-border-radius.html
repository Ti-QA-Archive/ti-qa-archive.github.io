'#labelName':{
  backgroundColor: '#f1c400',
  borderRadius: '18dp'
}

  • Hello Brandon,

    Thanks for your inquiry. I written a sample code to demonstrate your issue. My recommendation is try using a small borderRadious in label. Screenshot.

    var win = Titanium.UI.createWindow({
    backgroundColor : "white",
});

var view = Titanium.UI.createView({

   backgroundColor:'gray',
   width:200,
   height:100
});

win.add(view);

var label1 = Ti.UI.createLabel({
  backgroundColor: '#FC12dd',
  borderRadius: '3dp',
  text: 'A simple label',  
});

view.add(label1);

win.open();

    Thanks.
