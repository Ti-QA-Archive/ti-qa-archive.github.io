  • Hi I had a similar problem….

    I resolved with adding to the path system variable the directory of the Android SDK….. to do that do the following:

    Go to computer….. right click and select properties
    Click on Advanced system settings….
    Click on Environment Variables…
    On the System variables box…. select Path
    Press edit and on variable value add the end… add a semicolon ; and the path were you installed the Android SDK… for example ;c:\androidSDK
    Press Ok
    Restart Titanium Studio and try again….
