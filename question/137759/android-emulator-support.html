  • Couple of ways to target different screen sizes.

    • If running TiMobile SDK 2.x+, build an AVD to match your hardware's settings
    $ mobilesdk/osx/2.0.0/android/builder.py emulator <project-name> <android-sdk> <project-dir> <project-id> <avd-name>

    • Launch an emulator from Titanium that matches the Android SDK you are targeting (2.2, 2.3.3, 4.0). Then modify the titanium_??? avd that was created to match the screen size and hardware settings you desire.

    • Download sdk add ons from the hardware manufacture that matches the device you want to test on. Motorola is the BEST when it comes to this, with skins and pre made AVDS. HTCDev, not so much. Samsung, not so much. They have a few skins. That being said, Samsung allows you device time in their labs. We have used this to track down some device specific bugs.
