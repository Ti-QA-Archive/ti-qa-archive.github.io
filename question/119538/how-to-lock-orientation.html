How to lock orientation

I'm trying to lock the orientation of my app because I only want it to be used with one orientation (portrait) but I think there is a bug in Appcelerator, I tried using the following:

Titanium.UI.orientation = Ti.UI.PORTRAIT;

and:

window.orientationModes = [ Ti.UI.PORTRAIT ];

Neither locks the orientation to portrait, it just shows the app with black bars on the side when the screen is rotated.

  • Don't think that's the way to go, cause by apple rules you at least need to support the opposite orientation on ipad - which is also automatically handled by appcelerator. So from my knowledge you at least need

    <orientations device="ipad">
   <orientation>Ti.UI.LANDSCAPE_LEFT</orientation>
   <orientation>Ti.UI.LANDSCAPE_RIGHT</orientation>
</orientations>

    or the Portrait orientations for the iPad.

    Apart from that on iPhone / iPad this should work fine in the code of your app / windows:

    Titanium.UI.currentWindow.orientationModes = [Titanium.UI.LANDSCAPE_RIGHT, Titanium.UI.LANDSCAPE_LEFT];

    (or use the orientation modes you need for your window)

    If you're talking about Android it's a whole(!) different story. Are you developing for Android right now?

    • on iPhone the app runs in portrait mode by default.

      On Android you need to do the following:

      • create a "platform/android" folder in the root folder of your project
      • copy AndroidManifest.xml from "build/android" to this folder
      • Edit this AndroidManifest.xml and add

      android:screenOrientation="portrait"

      to each activity entry. So that you have

      <activity android:screenOrientation="portrait" ………..>

      for each activity. Then your Android app should also run in fixed orientation.

  • For the iPhone /iPad all you need to do is have your orientation mode assigned to your main window and it will stay locked.

    win.orientationModes = [Ti.UI.LANDSCAPE_LEFT,Ti.UI.LANDSCAPE_RIGHT];

    This is all I did and it worked.
