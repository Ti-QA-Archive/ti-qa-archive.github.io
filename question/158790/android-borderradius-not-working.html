Android borderRadius not working

Hi!

I can't get borderRadius to work on any view on Android. I've tried Buttons, Views, Windows.. Nothing. It works perfectly fine on iOS. I use it like this:

<View width="35dp" height="35dp" borderRadius="5dp">
    //Content...
</View>

I've tried both on a Galaxy S3 with CM 10.2 (Android 4.3), and in an emulator with Android 2.3.3 and it doesn't work. I've made sure that "Forced HW Acceleration" is off, and I've tried with android:hardwareAccelerated=&quot;false&quot; in tiapp.xml and with android:hardwareAccelerated=&quot;true&quot; but there's no difference.

According to this JIRA this is a known bug but it is supposed to be fixed since 3.1.0.

I'm on 3.1.3 (using Alloy) and I still have the problem.

    I just tried this and it seems that if the borderRadius is specified as an integer then it works. So I then set my Ti.App.xml

    <property name="ti.ui.defaultunit" type="string">dip</property>

    And then this works;

    var win = Ti.UI.createWindow({
    backgroundColor : '#000'
});

var v = Ti.UI.createView({
    width : 195,
    height : 195,
    borderRadius : 25,
    borderWidth : 3,
    borderColor : 'green'
});

win.add(v);

win.open();
