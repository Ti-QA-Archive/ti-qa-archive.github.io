Can't distribute app on application loader.

I am upload app on the application loader.

I'm test on device is correct.

But I'm upload app on the application loader, the application loader show error below.

application executable is missing a required architecture. at least one of the following architecture( s) must be present: armv6

Have any one encounter this problem?

How to resolve this problem?

I'm use xcode 4.6 .

Titanium SDK 2.1.3 .

thank you very much.

    Hi,
    Adding the following to the tiapp.xml allow the app to be validated:

    ios>
        <plist>
            <dict>
                <key>UIRequiredDeviceCapabilities</key>
                <array>
                    <string>armv6</string>
                    <string>armv7</string>
                </array>
            </dict>
        </plist>
    </ios>
