  • @Manuel You have an incomplete or corrupt Titanium CLI configuration. You should remove the existing config file and a new one will be created automatically.

    rm ~/.titanium/config.json

    Make sure that you're the owner of the ~/.titanium folder and that you have permissions to write to it.
