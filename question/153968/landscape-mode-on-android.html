Landscape mode on android.

I want my application in landscape (left and right) only. The whole application.

I've tried orientationModes: [Ti.UI.LANDSCAPE_LEFT, Ti.UI.LANDSCAPE_RIGHT] in the window, orientation in the AndroidManifest, Ti.UI.orientationModes in app.js, window.orientation = …, and another thing I've forgot.

When I start the app while device (launcer) is already in landscape, everything is fine.
When I start the app in portrait mode (or simply auto-rotate isn't on), it just starts, rotates, closes, starts, rotates, closes, and all that stuff again and again until I kill the application.

Any help? (newest titanium studio version, minSDK 8, targetSDK 14. Android 4.1 on both, tablet and phone.)

    On android orientationModes property will work only when your windows are heavyweight. so please include some more property to make window heavyweight like zIndex navBarHidden etc. .
