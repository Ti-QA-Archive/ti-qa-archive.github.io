Size of ActivityIndicator?

How can I change width and height of Activity indicator on iPad. The style is Titanium.UI.iPhone.ActivityIndicatorStyle.DARK. If I set width and height properties, it has no effect.

— asked July 19th 2012 by Tonio Seiler

  • You cannot set the exact size per se.

    But you can make it bigger by adding the BIG attribute to the style.

    var activityIndicator = Ti.UI.createActivityIndicator({
      style: Ti.UI.iPhone.ActivityIndicatorStyle.DARK 
                + Ti.UI.iPhone.ActivityIndicatorStyle.BIG,
      height:'auto',
      width:'auto'
);
