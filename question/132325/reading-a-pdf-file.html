Reading a PDF File

Hello,

Is there an example of how I can read a PDF from an HTML file? I want it to launch automatically.

My webview points to an HTML document, so if I could create the code to launch a PDF document from within there, it would be ideal.

Thanks for any assistance,
E

    You can use webview to render pdf file like this

    var win = Ti.UI.createWindow();
var wv = Ti.UI.createWebView({
    url: 'http://samplepdf.com/sample.pdf'
});
win.add(wv);
win.open();

      here is the solution for android.
      Use Google API for reading the PDF in Android

      var win = Ti.UI.createWindow();
var pdf_url='http://samplepdf.com/sample.pdf';
var wv = Ti.UI.createWebView({
    //Use Google API for reading the PDF in Android
    url : "http://docs.google.com/viewer?embedded=true&url=" + pdf_url
});
win.add(wv);
win.open();
