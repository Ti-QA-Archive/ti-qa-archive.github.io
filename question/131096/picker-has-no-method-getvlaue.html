Picker has no method getVlaue

Hello,
I got the following code to create alert dialog:

    getDateAlertDialog : function(resulTextField) {
        var view = Ti.UI.createView({
            layout : 'vertical'
        });
        var picker = Ti.UI.createPicker({
            type : Ti.UI.PICKER_TYPE_DATE,
        });
        picker.addEventListener('change', function(data){

        });

        view.add(picker);
        var alert = Ti.UI.createAlertDialog({
            androidView : view,
            title : 'Please select date',
            buttonNames : ['Ok', 'Cancel']
        });
        alert.addEventListener('click', function(e) {
            var date = picker.getValue();
            resulTextField.setValue(date);
        });
        return alert;
    }

When I run this code everything works fine until the line

var date = picker.getValue();

On this line i get the following exception:

Uncaught TypeError: Object#<Picker> has no method 'getValue'

Does anyone have any idea why this happens? I even have intellisense on the method name while in titanium studio.

Thanks for the help,

Danny
