Deploying a new version to iTunes Connect

Our application was deployed to iTunes connect using distribution certificate generated on one of the developers computers. The new version was built on another computer. Do we have to use the same distribution certificate or a new distribution certificate has to be generated?

— asked June 17th 2012 by Michael Khaytser

    • Export the private cert used to sign the dev certs.
    • Import to new machine.
    • Download certs from apple to new machine.

    • Install new certs on new machine.

    • Sign apps.
