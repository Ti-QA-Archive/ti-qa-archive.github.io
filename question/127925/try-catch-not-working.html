  • Hi

    Just a thought - why not loop the list of tables inside the database BEFORE you try to update a table that does not exist?

    SELECT * FROM sqlite_master WHERE type='table';

    This will list all of the tables you have - so you can check the one you are after is there.

    You could also look for the specific one;

    // change COMPANY for your table name
SELECT * FROM sqlite_master WHERE type='table' and tbl_name='COMPANY';

    Now if you get a row back - the table exists.

    This way you do not have to worry about the error because you are programming based on looking for it to work - rather than hoping the error will give you the information.
