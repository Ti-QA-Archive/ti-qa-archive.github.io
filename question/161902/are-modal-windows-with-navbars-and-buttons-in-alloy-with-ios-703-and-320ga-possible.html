Are modal windows with navbars and buttons in Alloy with iOS 7.0.3 and 3.2.0.GA possible?

I've been trying to restore the navbars and buttons to my models windows in my Alloy project that disappeared when I updated to the newest SDK. Are these not possible anymore? The titanium docs say this on navBarHidden:

"Beginning with 3.1.3, this is no longer a valid parameter passed to the Window.open method. Modal windows on iOS no longer have a navigation controller and hence do not have a nav bar."

Is there still a way to have a modal window have a navbar with a title and a close button? If so, can someone please show a working example? I can't find one that works anywhere.

    • Try this:

      // modalWin.xml
<Alloy>
    <NavigationWindow id="navwin" platform="ios" modal="true">
        <Window id="win" title="Modal Window" backgroundColor="red">
            <!-- modal window content here -->
        </Window>
    </NavigationWindow>
</Alloy>

      // main.js

var modalWin = Alloy.createController('modalWin').getView();
modalWin.open();

      Generally, you can either define the window ahead as modal or pass the modal property to the open() call, like modalWin.open({modal:true});

    • It works for me: iOS 7.0.3 Simulator, TiSDK 3.2.0.GA

      index.xml

      <Alloy>
    <Window class="container">
        <Button id="btn" onClick="doClick">Click for modal window</Button>
    </Window>
</Alloy>

      index.js

      function doClick(e) {
    var modWin = Alloy.createController('modal').getView();
    modWin.open({
        modal: true
    });
}

$.index.open();

      modal.xml

      <Alloy>
    <NavigationWindow id="nav">
        <Window id="win" title="Modal Window" backgroundColor="white">
            <LeftNavButton>
                <Button id="btn">Close</Button>
            </LeftNavButton>
        </Window>
    </NavigationWindow>
</Alloy>

      modal.js

      var args = arguments[0] || {};
$.btn.addEventListener('click', function(){
    $.nav.close();
});

  • Yes, check out the iOS 7 Migration Docs:

    var win = Ti.UI.createWindow({
    backgroundColor: 'blue',
    titleControl: Ti.UI.createLabel({
        text: 'Enable feature?',
        shadowColor: 'gray',
        shadowOffset: {x: 2, y: 2}
    })
});

var navWin = Ti.UI.iOS.createNavigationWindow({
    modal: true,
    window: win
});
navWin.open();
