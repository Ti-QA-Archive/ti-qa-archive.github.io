Saving Data from a countdown timer picker

Hello,

I am currently working in Titanium Studio Classic on an application for iOS and Android. I am having some trouble saving data from a countdown timer picker. My goal is to have a picker where the user can select a time amount and that selection will be saved for later use in a log and reminder system.
So far, I have a picker that I can select a time from but the information doesn't get saved.
Here is the code I have for the picker and label: 

        //Time estimate Label
        var timeEstimateLabel = Ti.UI.createLabel({
              color: 'purple',

              textAlign: Ti.UI.TEXT_ALIGNMENT_LEFT,
              top: '5%',
              left: '1%',
              width: Ti.UI.SIZE, height: Ti.UI.SIZE
        });    

        var picker = Ti.UI.createPicker({
              type:Ti.UI.PICKER_TYPE_COUNT_DOWN_TIMER,
              top:10
    });

I have looked at the forums and the documentation and I could not find anything that dealt specifically with saving data from the count down timer.

If anyone can provide any advice on how to save data from a count down timer, it would be very much appreciated.

Thank you!

    Hi Jessica,

    You need to use Titanium.App.Properties for saving data in application for later use. you can use this property value in all application.

    for ex:

    var win = Ti.UI.createWindow();
win.open();
var picker = Ti.UI.createPicker({
    type : Ti.UI.PICKER_TYPE_COUNT_DOWN_TIMER,
    bottom : 10
});
Ti.App.Properties.setInt('countDownDuration', 0);
// by default value

picker.addEventListener("change", function(e) {
    Ti.App.Properties.setInt('countDownDuration', e.countDownDuration);
    // set  value
});
win.add(picker);

var btn = Ti.UI.createButton({
    title : 'Get Value',
    height : 100,
    width : 100,
    top : 100,
});

btn.addEventListener('click', function() {
    var val = Ti.App.Properties.getInt('countDownDuration'); // get value
    Ti.API.info('get>>>>' + val);
});

win.add(btn);

    Set your data as you need.

    i hope this helps. thanks.

    Ti.App.Properties.setInt('countDownDuration', e.countDownDuration);

    if (e.countDownDuration >= 3600000)
        {
            hours = Math.floor(Ti.App.Properties.getInt('countDownDuration')/3600000);
            minutes = Math.floor(Ti.App.Properties.getInt('countDownDuration')/60000 - (hours*60));
        }

    else {
        minutes = Math.floor(Ti.App.Properties.getInt('countDownDuration')/60000);
        hours = 0;    
    }
