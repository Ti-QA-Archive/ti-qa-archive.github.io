Android AVD Manager

What is the best way to launch the AVD Manager from within Titanium Studio? I am running version 3.1.

    Hi

    There is a quick way to get the to AVD from Ti Studio.

    • From the dashboard tab in the full Studio (advanced view)
    • Click the Get Started button (under your name)
    • Look for Android and click it
    • Click Update Android SDK
    • Android SDK Manager will launch
    • Tools menu > Manage AVDs…

    Done

    If you cannot see the dashboard, there is a red house (or triangle) on the toolbar - click this.
