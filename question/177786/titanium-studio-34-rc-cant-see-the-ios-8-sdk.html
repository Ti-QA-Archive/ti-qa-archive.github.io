  • There is the 3.4.0.v20140922110915 build that solves this issue.

    1. Update the Studio by using this update site:
      http://preview.appcelerator.com/appcelerator/studio/standalone/update/rc/

    2. Install the latest SDK 3.4.0.v20140922110915

      ti sdk install --branch 3_4_X -d

    3. Make sure that this latest SDK version is selected

      ti sdk select

    4. Restart the Titanium studio.

      as root:
      npm remove -g alloy
      npm remove -g titanium
      npm install -g alloy@1.5.0-rc2
      npm install -g titanium@3.4.0-rc4

      So, tried running a ti build for iOS in command line.
