  • You use Titanium Developer to compile and test your code in simulators as well as install onto devices for testing. You are pretty much limited to tracing in console for debugging and various other logs. You can use any editor to write the code including eclipse. There are some third party tools to check out. There is a beta of Titanium Studio which is a first step at tying together the editing, debugging and testing. You want to look at Getting Started With Titanium as well as the rest of that.
