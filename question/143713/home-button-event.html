    • Anil, 2 things that people don't respond to very well here on the QA forum:

      1. Your urgency - Why should urgency on your part require urgency on anyone elses part
      2. Necessary code examples - Nobody wants to write your code for you.

      Keep that in mind going forward and you'll receive a lot more answers to your questions.

      Having said that though, here is a bit of an expansion on what Aaron already said.

      Ti.App.addEventListener('pause', function() {
    // The app has been paused, grab the current time and store it.
});
Ti.App.addEventListener('resume', function() {
    // The app has been resumed, grab the current time and calculate 
    // the duration between the paused time and the current time.
});
