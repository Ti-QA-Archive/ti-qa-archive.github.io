  • You definitely need to create TiViewProxy and TiView subclasses as described in the iOS Module Development Guid. I haven't used a view controller plus .xib file myself, but it seems like you should be able to create the controller in the TiView initializer method and add its UIView to the TiView. Take a look at the Ti.Paint source code on github to see how a native view is added to a TiView's view hierarchy.
