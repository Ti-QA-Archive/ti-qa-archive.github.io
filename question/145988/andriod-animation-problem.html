Andriod Animation problem

Hi Titaniums ..
i use the example of animation to move box from point to point in my andriod application BUT the box move in wrong direction and after end the animate box back to right position that should move to in first place ..!!!!
any idea !!

var view = Titanium.UI.createView({
    backgroundColor : 'red',
    width : 50,
    height : 50,
    left : 300,
    bottom : 10,
});

var animation = Titanium.UI.createAnimation();
animation.left = 700;
animation.bottom = 200;
animation.duration = 1000;

Titanium.UI.currentWindow.add(view);

view.animate(animation);

  • When you set the animation to left = 700, the animation will move to that point on the screen, not move that direction. So moving a box from left 300 to left 700 will mean that the box actually moves to the right.
