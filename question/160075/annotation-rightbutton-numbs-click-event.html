Annotation rightButton numbs click event

Hello,
I've a problem with the click event trigger on the annotation bubble which has a rightButton.

If I use the default iOS system images for the rightButton then the entire area of the bubble becomes clickable (and I want this to happen).
But when I specify a local image for the rightButton then only the image of the rightButton becomes clickable.

pin = Ti.Map.createAnnotation({
  latitude : e.latitude,
  longitude : e.longitude,
  title: "My title",
  rightButton: Ti.UI.iPhone.SystemButton.INFO_LIGHT, // click on bubble works
  //rightButton: '/images/arrow.png', // disables click on bubble
});

Advices or clues?

Thank you

  • Have you tried this rightView property ? I never tried but, what you can do is

    create a view with height and width and add image to that view. Then add this created view as rightView.

    var rightView = Ti.UI.createView({
  height: '40dip',
  widhth: '80dip'
});

var arrow = Ti.UI.createImageView({
  height: '22dip',
  width: ;22dip;
  image: '/images/arrow.png'
});

rightView.add(arrow);

pin = Ti.Map.createAnnotation({
  latitude : e.latitude,
  longitude : e.longitude,
  title: "My title",
  rightView: rightView
  //rightButton: Ti.UI.iPhone.SystemButton.INFO_LIGHT, // click on bubble works
  //rightButton: '/images/arrow.png', // disables click on bubble
});

    Let, me know how it goes ..!
