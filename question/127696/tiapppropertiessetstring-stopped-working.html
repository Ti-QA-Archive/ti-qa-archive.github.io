Ti.App.Properties.setString stopped working

I really have no idea what I did, but Ti.App.Properties.setString stopped working. On my first screen I have a little tool tip you can close which will keep it closed unless you go into the app settings. It has worked for months now and suddenly it isn't.

function closeTooltip()
{
    demo.animate({ 
        curve: Ti.UI.ANIMATION_CURVE_EASE_OUT, 
        bottom: -100, 
        duration: 500
    });
    if (Ti.App.Properties.setString('showTip',false))
    {
        Ti.API.warn(Ti.App.Properties.getString('showTip') + " success");
    }
    else
    {
        Ti.API.warn(Ti.App.Properties.getString('showTip') + " fail");
    }
}

if (Ti.App.Properties.getString('showTip') == null || Ti.App.Properties.getString('showTip') == true)
{
    demoClose.addEventListener('click', closeTooltip);
}

I noticed that the tooltip kept popping up and when I hit the close button, the closeTooltip method returns 0 fail
