XSLT in WebView?

Hello!

Quick and simple question! If i have i webview, and displays a XML-file in it, where an XSL-file is available for that XML-file, will the XML-file be transformed using the XSL file just like in any normal browser? Will it work on iOS (iPad)? Thanks!

  • Hello,

    if you put your xslt in the xml file, just like here – check the source code and you will see the way to add the xslt, it will display properly in a webview (as you would see it on chrome or firefox) with this code:

    var win1 = Ti.UI.createWindow(); 

var webview = Ti.UI.createWebView({
    url:'http://kal-el.ugr.es/~jmerelo/XSLT/tienda0.xml',
});

win1.add(webview);
win1.open();

    Best,

    Mauro
