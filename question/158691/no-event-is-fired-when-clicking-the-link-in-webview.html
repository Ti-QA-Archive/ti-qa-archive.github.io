  • Hello Kabindra,

    If you are talking about link in html property, it work as an html content in webview and when you click on the webview the webview click event is called, it does not fire any html event.

    If you want to call external function that call from web view then you have to write java script for html - content for html event handling and call you function using Ti.App.fireEvent(&#39;yourEvent&#39;) from html event to run your application code from your application.

    Best of luck.
