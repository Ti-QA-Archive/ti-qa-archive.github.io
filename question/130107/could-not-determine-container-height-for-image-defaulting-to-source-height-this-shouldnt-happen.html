Could not determine container height for image. Defaulting to source height. This shouldn't happen.

So i use this code to scale some images

var left = Ti.UI.createImageView({
        image: 'images/Knop1.png',
       width: widthl,
    height: heightl,

        left:5
    });
    widthl = left.width - Titanium.Platform.displayCaps.platformWidth*0.90;
    heightl = left.height - Titanium.Platform.displayCaps.platformWidth*0.90;

Can anyone tell me what i am doing wrong?

  • First thing, calculated dimension can be found in "size" property (view.size.width and view.size.height).

    Second thing, on Android, dimensions are calculated when window opens, that means that you need to listen to window's "open" event to get calculated dimensions:

    win.addEventListener('open', function() {
    Ti.API.info(view.size.width);
});
