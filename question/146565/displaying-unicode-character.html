    Hi Joe, The Unicode you are talking about is different that 4 character long unicode.

    4 character long unicode is known as SB-Unicode character and most of the symbols has both unicode.

    so here unicode you are talking about is : \u1F4A7 has SB-unicode \uE331

    here is the sample Label to use, i checked and droplets are displayed correctly.

    var sampleLable = Ti.UI.createLabel({
    text : "\uE331",
    color : 'blue',
    height : 200,
    width : 320,
    top : 80,
    left : 10,
    right : 10,
    textAlign : 'center'
});

    one link to have some IDEA.

    Hope this helps you and resolve your query.
