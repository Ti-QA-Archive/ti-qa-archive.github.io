Packing HTML5 as iOS app

Hi,

We have developed a HTML5 application, wanted to create mobile app for iPhone using Titanium Studio 2.1.2 on MAC.

Running the app using iPhone simulator, it runs with no issues.
But on iOS device its loading but no update on the screen.

There is no error or warning.

I figured out that, ipa file is not packed with all the files in Resources > HTML folder. Its filtering un-linked JavaScript files from the HTML.

Please advice me on how to pack the app with all the resources.
