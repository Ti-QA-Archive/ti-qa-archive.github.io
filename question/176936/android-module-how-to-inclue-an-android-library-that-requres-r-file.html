Android Module: How to inclue an Android Library that requres R file

I am attempting to creating a module that uses the Android UserVoiceSDK. The SDK is an Android project and how use it in your Android application is by adding it as a Library (Open project properties -> Android -> Library -> Add…) to your project. The SDK uses res files for the UI. Since modules are built with ant and compiled to an .so file I am at a loss at to how to create a module.

My goal is to use the SDK without change the source because I would like to upgrate the SDK as needed without code changes. However, because of the modules framework I am not sure that is possible. The only thing I can think to do is to copy the source into the module and copy the res files into the modules platform directory. I would need to change all the references to R. to use getIdentifier(...) because the resources would have the applications package name.

In any case, I am wondering if this is even possible or if someone has another idea how to make a module from the SDK.

Thanks,

Tony

    • Thanks Ricardo for pointing me in the right direction. I would say this is a more advanced Android module.

      Here is how I solved it (assume it should be the same for other Android libraries):

      1) Added UserVoiceSDK as an Android project.

      2) As a result it build the library .jar file. In this case, uservoicesdk.jar under uservoice-android-sdk/UserVoiceSDK/bin.

      3) Copied uservoicesdk.jsr to my module's android/lib directory.

      4) Built the module and included the uservoice SDK to the module zip (I know I could just add the res but decided to include the SDK).

      5) Added Mark M's CLI hook to my project. Link

      My hook:

      exports.id = "com.theknot.mobile.pro.aapt";
exports.version = "1.0";

exports.cliVersion = '>=3.X';

exports.init = function(logger, config, cli, appc) {
    cli.on('build.android.aapt', {
        pre: function(data, next) {
            var args = data.args[1];
            if (args.indexOf('--auto-add-overlay') < 0) {
                args.push('--auto-add-overlay');
            }

            var externalLibraries = [{
                javaClass: 'com.uservoice.uservoicesdk',
                resPath: 'modules/android/com.xogrp.mobile.saddle/1.0/Vendors/UserVoiceSDK/res'
            }];

            // --extra-packages can be defined just once
            if (args.indexOf('--extra-packages') < 0) {
                args.push('--extra-packages');
                args.push('');
            }
            var namespaceIndex = args.indexOf('--extra-packages') + 1;

            externalLibraries.forEach(function(lib) {
                if (args[namespaceIndex].indexOf(lib.javaClass) < 0) {
                    args[namespaceIndex].length && (args[namespaceIndex] += ':');
                    args[namespaceIndex] += lib.javaClass;
                }
                if (args.indexOf(lib.resPath) < 0) {
                    args.push('-S');
                    args.push(lib.resPath);
                }
            });
            next(null, data);
        }
    });
};

      6) build app project
