  • Hi Ilya, based on the error message, it appears you have an empty <version> tag in your tiapp.xml. Please make sure you have a <version>X.Y.Z</version> in the tiapp.xml.

    On a side note, –deploy-type is ignored when –target is set to "dist-adhoc".
