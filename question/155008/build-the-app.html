  • Accepted Answer

    This can only be done using the Command Line Interface (CLI).

    There is a --build-only flag (or -b for short) to indicate that you don't want to deploy or run your app.

    ###From the terminal, just go into your project folder and type the following:

    titanium build -p android -b
