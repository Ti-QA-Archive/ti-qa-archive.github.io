Failed to install apk - Lollipop 5.0.1

Hi, I am using Titanium SDK 3.4.1. I recently installed the new android SDK (5.0.1) and also have installed Android Lollipop on my Nexus 5 Phone. However, no matter what combination of Android and Titanium SDK I try to use, I cannot get anything to install on this device. I also, have a Nexus 7 tablet with Android 5.0.1 and nothing will install on that either.

I can, however, install on my Samsung Galaxy 3 running Android 4.3.

Any advice on how I get apps installing on my Nexus Phone/Tablet? The error I get is:
[ERROR] : Failed to install apk on "066e279c0b116c41"

  • Hello Donovan Carthew,

    In tiapp.xml file, add this and try to install in your device, then let me know.

    <android xmlns:android="http://schemas.android.com/apk/res/android">
        <manifest android:installLocation="preferExternal" />
</android>
