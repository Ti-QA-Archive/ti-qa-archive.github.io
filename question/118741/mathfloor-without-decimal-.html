    When the following code is run On Titanium 1.6.1 / Android 2.2:

    var number = (Math.floor(Math.random()*6+1));
Ti.API.info("number = " + number);

    it produces this output:

    (kroll$1: app://app.js) [325,1754] number = 3

    Thus, without knowing what you are using to develop, it's difficult to either advise further or to escalate it as a bug.
