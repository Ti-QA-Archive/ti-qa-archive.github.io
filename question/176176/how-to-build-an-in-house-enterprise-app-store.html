How to build an in-house enterprise app store

Hi all,

I have a requirement to build an in-house enterprise app store for a client of mine. They need a main CRM app that in turn downloads other apps that we build, stores them inside the main app and runs them from a 'launcher' screen within it, similar to another CRM app on the market.

I'm happy building enterprise apps etc, but wondered how I would go about the 'app within an app' requirement and whether anyone had faced a similar challenge. For example is it best to accomplish this using in-app purchasing or is there any other method I could use?

Any advice would be much appreciated as I've been racking my (limited) brains for a while now!

Thanks,
Andy

      2. creating of special script language. This interpreter interpretes JSON, maybe you can use the evil JS command 'evalJS'. In this case you can download new JS-code and start by evalJS
