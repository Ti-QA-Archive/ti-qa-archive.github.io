Ti.App.Properties After re-install or Update

If we have this code

var isInstalled = Ti.App.Properties.getBool("is_installed", false);
if (isInstalled) {
    // show the UI ... etc
} else {
    // we will install the database and some configs...

    // and finally
    Ti.App.Properties.setBool("is_installed", true);
}

The question is: after the update or re-installation of the App. What will Ti.App.Properties.getBool(&quot;is_installed&quot;, false); return?

Will it return false or true?

  • Accepted Answer

    After an update, the property will remain set as it was before the update.
    On reinstall (delete app, install again) the property will be null.
