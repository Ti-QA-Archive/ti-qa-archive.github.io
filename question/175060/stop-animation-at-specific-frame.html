Stop Animation at specific frame.

I have an ImageView with an array of images. After view.start() a frame-by-frame animation is starting. Is it possible to pause or stop the aniamtion at an specific frame?

    Hi Evelyn,

    Yes, this is possible. You just have to count the frames while it's animating. Use the change event to do this.

    I haven't tested this, just quickly wrote it down here, but it should work like this:

    var image = Ti.UI.createImageView({
  images: arrayOfFrames,
  duration: 84 // almost 12 fps
});

image.addEventListener('change', function onFrameChange(e){
  // stop animation at frame 99
  if (e.index === 99){
    image.stop();
    image.removeEventListener('change', onFrameChange);
  }
});
