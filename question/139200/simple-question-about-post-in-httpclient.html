simple question about POST in httpClient

hi , im reading in this page :
HTTPClient and the Request Lifecycle and i have simple questions about POST :

  • in this code :
    ~~~
    var xhr = Ti.Network.createHTTPClient();

xhr.onload = function(e) {

};

xhr.open('POST','http://www.myblog.com/post.php');
xhr.send({
title:'My awesome blog', // <—
body:'Today I met Susy at the laundromat. Best day EVAR!' // <—
}); 

- what dose **title** and **body** means in **xhr.send** ?

var client = Ti.Network.createHTTPClient();
