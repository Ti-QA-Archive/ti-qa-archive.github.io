    hi
    you can show an activity indicator when loading on webview:

    
webview.addEventListener("beforeload", function(e)
{
    // code
}

Then, when the webview has finished loading you can hide the activity indicator with:

webview.addEventListener("load", function(e)
{
    // code
}
