Uncaught TypeError: Object function() getDocumentElement()

Hi guys,

While trying to retrieve online XML data in my Android application I get the "Uncaught TypError"

Here is an example of my code:

xhr.onload = function() {

    var doc = this.responseXML.getDocumentElement;
    var items = doc.getElementsByTagName("days");
    var x = 0;

    for (var c=0; c<items.length; c++) {

        var item = items.item(c);

        var day = item.getElementsByTagName("day").item(0).text;
        var month = item.getElementsByTagName("month").item(0).text;

Any suggestions?

    hi,

    its because getDocumentElement() is a method and you used as a property so

    use as a property like..

    var doc = this.responseXML.documentElement;

    or as a method

    var doc = this.responseXML.getDocumentElement();
