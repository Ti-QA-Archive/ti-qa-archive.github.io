Offset HintText/TextField in TableView & Button Placement

I found a function on these Q&A forums (that I modified a bit) that allowed me to create textFields in a tableView for iOS. It seems to work except the hintText/textField is centered in the tableView for some reason and not left aligned as I stated it. Also when I try to add the two buttons (which should be Calculate and Reset) side by side it doesn't appear. Was wondering if anyone could see anything wrong with the code or at least point me in the right direction. Thank you!

Row adding function

function addRow(hintName)
{
   row = Ti.UI.createTableViewRow({height:45, left:0});

    textField = Titanium.UI.createTextField({
        //font:{fontSize:18, fontWeight:'bold'},
        textAlign: 'left',
        //color:'#888',
        //height:35,
        //top:10,
        left: 10,
        width:'auto',
        hintText:hintName,
        keyboardType:Titanium.UI.KEYBOARD_NUMBER_PAD,
        borderStyle:Titanium.UI.INPUT_BORDERSTYLE_NONE
    });

    textFieldsArray[hintName] = textField;

    row.add(textField);
    row.selectionStyle = Ti.UI.iPhone.TableViewCellSelectionStyle.NONE;
    return row;
}
