  • You will want to attach an eventListener to the table and then pass the event rowData on to a new window. I believe the table is in a tab in this example, which makes it a little tricky. The code I had for this is:

            tv.addEventListener('click', function(_e) {
            var tab = (_captured) ? bh.capturedTab : bh.fugitivesTab;
            tab.open(bh.ui.createDetailWindow(_e.rowData));
        });

    tv is your tableview.

    _captured boolean should have been passed to the createTableView so that it knows which tab is being created. Here you also check it to see which tab (bh.capturedTab or bh.fugitivesTab) to use to open the detail window.

    Finally the details window is built inside your bh.ui.createDetailWindow function, where you are also passing the row data (fugitive name, etc) to it via _e.rowData.

    in createDetailWindow you will set up an arg to recieve that data and then populate your detail window using it.
