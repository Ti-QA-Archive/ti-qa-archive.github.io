Adroid TabGroup "close" event does not work

It seems that the "close" event does not work for TabGroup on Android (but is ok on iOS)

var Tbg_Main = Ti.UI.createTabGroup();
Tbg_Main.addEventListener("close", function(){Ti.API.info("Close !");});
— asked April 20th 2011 by Nicolas GRESSARD

  • Nicolas

    Natively, tabGroups work differently on Android compared with iOS. On Android, it is possible to have only a single tabGroup, and once created and opened, you cannot close it. This ticket has been created to keep an eye on the situation in case anything changes natively, but I doubt it will be possible any time soon.

    I'm afraid the best and only strategy to take is to design your UI within the limitations of how tabGroups are implemented on the respective platform.

    Cheers
