    See if it now works for you

    Titanium.UI.setBackgroundColor('#000');

var win = Ti.UI.createWindow({
    backgroundColor : 'white'
});

var bt1 = Ti.UI.createButton({
    title : 'click me',
    width : 'auto',
    height : 'auto',
    top : 50,
});

win.add(bt1);

var tf1 = Ti.UI.createTextField({
    width : 100,
    height : 20,
    top : 200,
    borderStyle : Ti.UI.INPUT_BORDERSTYLE_ROUNDED
});

win.add(tf1);

win.open();

  • That is how iOS 7 presents its UI components now. Have a look around most of iOS 7 recent apps and you will notice how they use those same controls.

    There is little you can do for the buttons, but you can always set a borderStyle property to to text fields.

    Hope this clears it up for you.
