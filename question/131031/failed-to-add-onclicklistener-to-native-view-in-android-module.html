Failed to Add OnClickListener to native view in Android Module

Hello,

I faced a problem when i have created android module by mobilesdk v1.8.0.1, the problem is occurred when i tried to adding OnClickListener to native view, unfortunately no action is happen when i clicked or touched on the custom module.
can anybody help me, i wasted too much time trying to create a small custom module.

My code

public ExampleView(TiViewProxy proxy) {
            super(proxy);

            TextView textView = new TextView(proxy.getActivity());
            textView.setText("click me to see toast alert");
            final TiViewProxy tiViewProxy = proxy;
            textView.setOnClickListener(new OnClickListener() {

                @Override
                public void onClick(View arg0) {
                    // TODO Auto-generated method stub
                    Toast.makeText(tiViewProxy.getActivity(), " toast alert", Toast.LENGTH_SHORT).show();
                    ((TextView)arg0).setText("onClick worked");
                }
            });

            textView.setOnTouchListener(new OnTouchListener() {

                @Override
                public boolean onTouch(View arg0, MotionEvent arg1) {
                    // TODO Auto-generated method stub
                    Toast.makeText(tiViewProxy.getActivity(), "toast alert", Toast.LENGTH_SHORT).show();
                    ((TextView)arg0).setText("onClick worked");
                    return false;
                }
            });

            setNativeView(textView);
        }
