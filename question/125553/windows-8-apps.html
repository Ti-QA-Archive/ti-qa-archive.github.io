  • In Windows 8 JavaScript is just another language like C#, VB or C++. The only difference is that you use JavaScript along with HTML and the others along with XAML. This has nothing to do with the .NET framework, they build a layer above the Kernel called WinRT, which is installed together with Windows. You can access it with JavaScript the same as with C++! Blend is intergrated into VS and has HTML/CSS-equivalents to XAML. So if you do a so called Metro-style app you get a touch-centric application that runs on every device where Win8 is running, be it a mobile, a tablet or a PC.

    This is the typical Microsoft "We come late, but we come". If you want to jump on that bandwagon TiDesktop is not an option anymore on the Windows platform and there is practically no need that TiMobile compiles to Win8, as you already can use JS and HTML for those apps. So no, I think Appcelerator does not support it.
