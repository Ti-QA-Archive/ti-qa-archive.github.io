Expand view with backgroundImage

I have a View like this: 

var view = Ti.UI.createView({
    backgroundImage: "image.png",
    width: "25%",
    height: 160
});

and I want to expand with a view.addEventListener('click', function(){}); for the user will see better the image. Anyone knows how? Thanks.

  • I'd recommend creating a modal window and using an image view to show a bigger version of the image. Sort of like this:

    
view.addEventListener("click", function( evt ){
    var imgWindow = OS_ANDROID ? Ti.UI.createWindow({ backgroundColor: "#33000000", navBarHidden: true, modal: true })  : Ti.UI.createWindow({ backgroundColor: "#33000000" });

    var img = OS_ANDROID ? Ti.UI.createImageView({ image: evt.source.backgroundImage, width: 250, height: 250 }) : Ti.UI.createImageView({ image: evt.source.backgroundImage, width: 250 });

    imgWindow.add(img);

    imgWindow.addEventListener("click", function(){ 
        imgWindow.close();
    });

    imgWindow.open();
});
