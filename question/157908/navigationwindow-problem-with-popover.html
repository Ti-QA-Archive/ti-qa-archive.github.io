NavigationWindow Problem with PopOver

It appears that NavigationGroup objects are being deprecated in lieu of NavigationWindow objects.

I use NavigationGroups in my popover apps on the ipad.

When I change the code to replace the Navigation groups with Navigation Windows and try to add them to a Popover, I get the message:

Can not add a window as a child of a view.

When reviewing the documentation, it says this: Titanium.UI.iPhone.NavigationGroup may be added as a view in a popover. Do not add top-level view containers, such as a Window or TabGroup, to a popover.

So, how am I supposed to build a Navigation group for a popover on an iPad if you can't add NavigationWindow objects to a popover?

This worked great in 3.1.2 and prior.
