    Hi

    You are almost there, you missed just one thing, as you are referencing e you need to add it to the function brackets.

    So this;

    tableSafari.addEventListener('click', function () {

    Becomes this;

    tableSafari.addEventListener('click', function (e) {

    Your code revised in this section, you will need to check for others.

    tableSafari.addEventListener('click', function (e) {
    //containingTab attribute must be set by parent tab group on
    //the window for this work
    alert(e.rowData.title);
    self.containingTab.open(winKorteSafari);
});

tableKorteSafari.addEventListener('click', function (e) {
    //containingTab attribute must be set by parent tab group on
    //the window for this work
    self.containingTab.open(winSafariDetail);
});
