Application Name and iOS App Store application name different

Hi,

I am creating an application for iOS. The name of the application is HouseSnS (this will be displayed once the application is installed on the device). Now, my question is that while uploading the application to Apple Store, I need to provide the name of the application. In this section can I provide the name as HouseSnS® - Keeping your home safe and secure. Will this be allowed by Apple.

    Yes, and I've done it without issue. Apple's recommendation is that you make sure the app store listing name and your app's on-device name are similar to avoid confusion.
