IPAD Startup Orientation (2.1.0.GA)

Peepz im using this function from the kitchensink app to detect the orientation on startup (using 2.1.0.GA):

function getOrientation(o)
{
    switch (o)
    {
        case Titanium.UI.PORTRAIT:
            return 'portrait';
        case Titanium.UI.UPSIDE_PORTRAIT:
            return 'reverse portrait';
        case Titanium.UI.LANDSCAPE_LEFT:
            return 'landscape';
        case Titanium.UI.LANDSCAPE_RIGHT:
            return 'reverse landscape';
        case Titanium.UI.FACE_UP:
            return 'face up';
        case Titanium.UI.FACE_DOWN:
            return 'face down';
        case Titanium.UI.UNKNOWN:
            return 'unknown';
    }
}

var orientation = getOrientation(Titanium.Gesture.orientation);

The problem is that it will alway give: unknown at startup (also in kitchensink on IPAD not on IPHONE)

How can i fix this for ipad on startup? I only need to now if its in portrait or landscape mode….
