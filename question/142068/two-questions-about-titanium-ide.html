    1. Titanium apps are native apps. So The official market is Apple AppStore and GooglePlay. End if story.
    2. Click implies touchstart/touchend so put ur finger on an element and release it form the same element. Touchend triggers when you release the finger from an element, independent of if you stated the touch on the same element or not.
