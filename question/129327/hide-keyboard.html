Hide keyboard

Hi all. I am a beginner in Titanium Studio. I can able to hide the keyboard when a button is selected.
It works,

okBtn.addEventListener(&quot;click&quot;, function(e) { textField.blur(); });

But, how can I hide the keyboard when I tap/click on the mapView?
I tried this code, Doesn't work,

mapview.addEventListener(&quot;click&quot;, function(e) { textField.blur(); });

mapview.addEventListener(&quot;singletap&quot;, function(e) { textField.blur(); });

So, how can I make it work? Thanks in advance.

  • Add an invisible view to the window with the same position and size of the MapView. Make visible=false by default. When the search field is focused, make visible and when blurred, make invisible. On the click event of the invisible view, blur our the text field and hide the keyboard.

    SearchBar.addEventListener('focus', function() {
    hideKBView.visible = true;
});

SearchBar.addEventListener('blur', function() {
    hideKBView.visible = false;
});
var hideKBView = Ti.UI.createView({
    top: 0,
    left: 0,
    bottom: 0,
    visible: false,
});
hideKBView.addEventListener('click', function() {
    SearchBar.blur();
    hideKBView.visible = false;
    Titanium.App.fireEvent("hideKeyboardToolbar");
    if (Ti.Platform.osname === 'android') {
        Ti.UI.Android.hideSoftKeyboard();
    }
});
win.add(hideKBView);
