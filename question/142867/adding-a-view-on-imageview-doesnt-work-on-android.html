Adding a view on ImageView doesnt work on Android

Hi guys, I am trying to add a new view on an imageview. This view would contain a label with some text in it. The below code perfectly works on iOS but doesnt work on Android. Can you please tell me if something is wrong with it ?

var width = Ti.Platform.displayCaps.platformWidth;
var height = Ti.Platform.displayCaps.platformHeight;

var win = Ti.UI.createWindow({});

var imgView = Ti.UI.createImageView({
    backgroundImage:'images/sr.jpg',
    height:height,
    width:width
});

var infoLabel = Ti.UI.createLabel({
    text : "label text",
    opacity:'1',
    color:'white',
    width:'auto',
    height:'auto',
    textAlign:'center'
});

var infoView = Ti.UI.createView({
    bottom:20,
    height:height/2,
    width:width-20,
    left:20,
    backgroundColor:'black',
    opacity:'0.75',
    borderRadius:10
});

infoView.add(infoLabel);
imgView.add(infoView);
win.add(imgView);
win.open();

  • Android is not so permissive as iOs.
    There are few UI types that don't accept children at all, or they accept only a certain types.

    The idea is to use the common sense when you are building the ui. An image has no reason to hold a child view, is designed to hold pixels and not to be a container.
