Android Camera Problem

Me again, excuse me.
I trigger the camera in a buttom, with onClick evento (in alloy), the camera take the picture and everything, but when i put the bottom "save", the camera closes and my App open again my index, and i really don't know why, when i click save i need the app to continue where i was, not in the Index.

And this is my code:

Titanium.Media.showCamera({
        success:function(event)
        {
            tmpImg=getDate()+".jpg";
            var imageFile=Titanium.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory, tmpImg);
            imageFile.write(event.media);
            alert("Foto cargada correctamente");
            // hideCamera();
        },
        cancel:function()
        {
        },
        error:function(error)
        {
            var a = Titanium.UI.createAlertDialog({title:'Camara'});
            if (error.code == Titanium.Media.NO_CAMERA)
            {
                a.setMessage('Por favor despliegue esto en un dispositivo');
            }
            else
            {
                a.setMessage('Error: No se encontró camara: ' + error.code);
            }
            a.show();
        },
        showControls:true,
        autohide:true,
        saveToPhotoGallery:true
    });
