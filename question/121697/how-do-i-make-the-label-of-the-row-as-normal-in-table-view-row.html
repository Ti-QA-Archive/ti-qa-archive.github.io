    You're currently using a simple table view with no custom row layouts - aside from color, you can't style row titles on simple table views.

    You'll need to create custom row objects with labels set with the desired font properties.

    See the example for programmatically creating a standard table view in the guide.

  • Simply add fontWeight: &#39;bold&#39; to the first (the one with All) entry of your categories array.

    Please also note that since you are going to have two different visual styles (one bold and the others normal). You should specify the className for every row. This will improve performance since the OS re-uses those classNames for a specific look&feel.

    So it would go something like this:

    var categories = [
    {title: "All",hasChild:true,category_name:"All",test:'Show_result.js', fontWeight: 'bold', className:'bold_row'},
    {title:'Cafes & Restaurants',hasChild:true,category_name:"restaurants",test:'Show_result.js', className:'regular_row'},
    {title:'Bars & Pubs',hasChild:true,category_name:"Bars_And_Pubs",test:'Show_result.js', className:'regular_row'},
    // ... And so forth...
];

  • i did the following and it worked.

     //Creating an Array named rows
    var rows = [];

    //Traversing through the Array categories
    for(var v in categories)
    {
        //Creating a row Table View Row
          var row = Titanium.UI.createTableViewRow({
          hasChild:true,
          category_name:categories[v].category_name
          });    //End of 'createTableViewRow'

        //Creating Labels for Table View Row'
          var label = Ti.UI.createLabel({
           text:categories[v].title,
           font:{fontSize:20},
          });//End createlabel

          //Make the first Label All as Bold
          if(Number(v) === 0)
          {
            label.font = {fontWeight:'bold',fontSize:20};
        }//End of if

        //Add the labels Table View Row
          row.add(label);

          //Add the labels to named rows
          rows.push(row); 
    }// End of for loop
