Disable selection/click/touch of single TableViewRow

I have a TableView with multiple rows. The top row we're just using as a heading and thus we don't want users to be able to click/tap it. Here's the code that generates the row:

    var row = Titanium.UI.createTableViewRow({
        height: 43,
        selectionStyle: Titanium.UI.iPhone.TableViewCellSelectionStyle.NONE,
        touchEnabled: false,
        bubbleParent: false
    });
    row.add(Titanium.UI.createLabel({
        text: args.heading,
        font: {fontFamily: "Open Sans [Regular]", fontSize: "13"},
        color: "#ffffff",
        touchEnabled: false,
        bubbleParent: false
    }));

You can see I've tried using selectionStyle, touchEnabled, and bubbleParent to try and disable selection. When this row is clicked (selected) it doesn't change color, but it does un-select the previously selected row, which is the behavior we wish to prevent. How can I achieve this?
