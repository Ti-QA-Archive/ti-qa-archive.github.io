    calendar.feed.entry contains the array of event. you need to loop through that to get the info you need

    try this

    ...
    var labelNews = Ti.UI.createLabel({
            text: calendar.feed.entry[i].THE_TAG_THAT_YOU_WANT_TO_DISPLAY
        });
        win.add(labelNews);
...
