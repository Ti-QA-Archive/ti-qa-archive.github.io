Alloy Controller Inheritance - Possible?

JS has parasitic inheritance as described by this article is I'm guessing not possible in Alloy in, say, the index.js file, because by the time index.js runs it has already been initialised as an Alloy Controller.

Right? Wrong?

Let me show you what I would like to do. Take a basic utils method like this:

var currentPage;

function showPage ( pageID )
{
    if ( currentPage !== null )
    {
        currentPage.close();
    };

    currentPage = $[ pageID ];

    if ( currentPage == null )
    {
        currentPage = Alloy.createController( pageID ).getView();
    }

    currentPage.open();
};

It would be a shame to have to copy and paste that into every Alloy controller I make. Is there a way I can achieve this in a DRY way?
