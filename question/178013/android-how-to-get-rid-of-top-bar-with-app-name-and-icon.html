  • before 3.3.0 sdk released, you can use navBarHidden to true to hide navigationBar but after 3.3.0 released, you can use window's activity to hide actionBar…like…

    before 3.3.0 sdk,

    var win = Ti.UI.createWindow({
    navBarHidden : true
});

    after 3.3.0 sdk

    var win = Ti.UI.createWindow();
win.activity.actionBar.hide();
