    Hi Prashaanth N.G.S,

    Thanks for your concern.

    You are using Titanium SDK 3.1.2.GA, which does not support Android SDK Tools 22.6.2, nor does it support 64-bit Java. You should update to Titanium SDK 3.2.2.GA if you want to use either of those versions. Check the Required Android Packages section, it says Rev 19.0.3 and later requires Titanium SDK 3.2.2 or later.

    You can also take a look at Application Installer abnormal process termination. Process exit value was 8 (Titanium)

    Hope it helps.
