  • There is no parameters for the vibrate function, so its the default length of the device. Why not use:

    var player = Ti.Media.createSound({url:"keypress.wav"});
player.play();

    This way you can set the length & even set the volume (setVolume()).

    Note: If you are using Andriod custom manifest, remember to set the permissions!
