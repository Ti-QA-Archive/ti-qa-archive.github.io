    • According the documents from iOS SDK 5.0, all fixed-resolution ADBannerContenSizeIdentifier constants are deprecated. For example, ADBannerContentSizeIdentifier320x50 says

      >Indicates that the banner view is 320 points by 50 points in size. This size is used on iPhone for portrait advertisements. (Deprecated. Use ADBannerContentSizeIdentifierPortrait instead.)
