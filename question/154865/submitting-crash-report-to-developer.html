  • there's a module for HockeyApp that helps you deal with crashes:

    var Stick = require('hockeystick');

try {
  // Your code
}
catch (e) {
  Stick.reportCrash(e.stack);
  Stick.sendCrashes();
}

  • Hi

    If this is for the Android as you have indicated the Google Play Store already provides full crash report information where possible including any comments the user had when the app crash report was submitted. This can then be read within the Google Play Console and often includes type of device and which version the app was on when the crash occurred.

    If you are also trying to get this to show crash reports prior to pushing the app live then consider checking out the Google Play store beta and alpha testing modes that help you manage the process.

    You do not need any additional libraries to achieve your goal. However I do like Albertos suggestion which can be great for specific; section testing, fail over and logic monitoring which is as important as hard crash testing.
