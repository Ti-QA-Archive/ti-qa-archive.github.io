TableView SeparatorColor iOS

Hi

I am using a TableView with several rows, and I set:

var table = Ti.UI.createTableView({
        separatorColor : "#fff",
    });

And it works fine displaying a white line all across the row's width; but only works on Android. On iOS the line does not is displayed.

Anyone knows how configure a white separator line between tableView's rows on iOS?

Thanks

    For iOS, you need to use the separatorStyle property : http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.TableView-property-separatorStyle

    Bear in mind that the line now doesn't go all the way to the left edge since the new iOS7 styles.

    eg:

    

var table = Ti.UI.createTableView({
        separatorStyle : Titanium.UI.iPhone.TableViewSeparatorStyle.SINGLE_LINE 
});
