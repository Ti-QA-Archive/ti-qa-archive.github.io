    • Ah that's when creating an app. It seems that for some reason the newest version of CLI wasn't installed. You can check that by trying this command:
      ti -v
      if you get a version lower than 3.2.0 you'll need to update Titanium using
      [sudo] npm install -g titanium then try restarting Titanium Studio.
