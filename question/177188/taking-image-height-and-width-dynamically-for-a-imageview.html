Taking Image Height and Width Dynamically for a imageView

Hi All,

In my project i am taking a View and in the View I am taking a image View and assigning a image to it.

I am not giving any layout to imageview. The image is becoming very big and it is out of view. Cant we take the Image height and width dynamically. 

            <View layout="vertical" id="Help_Profile" onClick="loadProfile" class="helpRow topMargin10">

                    <ImageView id="profileimg" image="/images/Profile_tab_a.png"  />
            </View>

Do we need to give the Image height and width of the image manually, If we do that it may effect Andriod sizes. Please let me know how can we do it.

  • Hi

    There are methods for getting the height and width, but from what you are saying the image itself is quite large so you would need to reduce the size of the image anyway.

    You could use: Ti.UI.FILL and Ti.UI.SIZE to set the height and width of the ImageView.

    TSS

    "#profileimg": {
    height: Ti.UI.SIZE,
    width: Ti.UI.FILL
}

    By setting the properties in this way, the ImageView should adopt the width of your containing view and then the height will proportionately size itself. This is assuming that you have defined height and width to your view?

    Hope this helps.
