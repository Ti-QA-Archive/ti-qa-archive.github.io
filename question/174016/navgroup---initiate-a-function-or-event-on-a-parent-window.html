Navgroup - initiate a function or event on a parent window

I have an iOS navigation window with multiple levels (using commonJS). When I close one child window I want to initiate a function / event on that child's parent window.

I've tried adding an 'open' event to the parent, but this only fires when the parent is first opened, not when going back to the parent from the child.

Adding a 'focus' event to the parent does work, but creates an unacceptable result on the screen because the 'focus' event only fires once the window has opened, which is too late (I need to fire the event before the parent window is visible).

Does anyone have any ideas? I guess one way is to pass the parent window onto the child when opening the child in the navgroup. But I'm not sure how to do this, and am also worried about the memory implications of this.

Thanks.
