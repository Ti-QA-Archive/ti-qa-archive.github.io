    The size of the image file does not really matter.
    The decoding of the image works like this:
    Each pixel will have 3 bytes for RGB and another one for Alpha if your image has the alpha channel

    That means that an image 100x100px will have a memory footprint of
    30k(40k with alpha)

    If you have an 1024x1024px image you're kind of doomed as the footprint will be like 3M. Animating this will kill your device for sure.
