Alloy framework doesn't work for me

Hello All,

I am new for Titanium & i want to build up my app on alloy. Just trying to get started – create a project, compile it, and run it. The "run" part doesn't work for me. I am using 3.2.3.GA sdk.

Project will compile successfully but at the run time it says…

/Users/jily/alloy/myproj/app does not contain a valid project

Is there something I'm doing wrong?

    Hello Jily,

    I think the problem has something to do with the Titanium command-line tools… The above command ends up invoking the following Python command, actually it reports the error

    python /Library/Application\ Support/Titanium/mobilesdk/osx/3.2.3.GA/titanium.py run –dir=/Users/jily/myproj/app –platform=iphone
