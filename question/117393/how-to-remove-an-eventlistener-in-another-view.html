How to remove an EventListener in another view

I have my main window with a tabgroup. In the first tab it leads to a map event where I want the shake gesture to open up the built in map for directions. In the second tab I want the shake gesture to show a message, I was wondering if this was possible to do? This is the eventlistener I want to remove in the second tab.

Ti.Gesture.addEventListener('shake', location);

function location() {
Ti.Platform.openURL('http://maps.google.com/maps?daddr=43.72529,-79.60880 &ie=UTF8&t=h&z=16');

};
