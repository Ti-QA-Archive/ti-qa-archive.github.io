  • Hi Stefano de luca,

    previous 'auto' property value divided into two specified behavior options in SDK version 2.0

    This warning caused by Ti.UI.FILL and Ti.UI.SIZE value of property, when you apply this value to any object then it fills appropriates to its parent or its contents

    Ti.UI.FILL - automatically fills the object to its parent and

    Ti.UI.SIZE - automatically fills the object to its contents.

    for more information see this link
