xhtml inside WebView doesn't fire App level event (Ti.App.fireEvent)

I am using a WebView to open a .xhtml file, which is unzipped from a ePub.
It differ from .html by having the html tag as

<html xmlns:epub="http://www.idpf.org/2007/ops" xmlns="http://www.w3.org/1999/xhtml" xmlns:ev="http://www.w3.org/2001/xml-events">

In this xhtml, I fire an event upon user selection, like this:

$(function(){    

/** detect touchend **/
$(document).bind('touchend', function(e){
    var sel = '';
    if(window.getSelection){
        sel = window.getSelection().toString();

        if( sel.length>0 ){
            $('#debug').html( sel );
            Ti.App.fireEvent('EPubReader:selectText', {text:sel} );
        }
    }
});    


});

But the event didn't fire.

However, when I change the file from .xhtml to .html and replace the html tag with simple 

<html>

it can fire the event.

Why is it so?

Although it work after modification, I don't prefer modifying every .xhtml file unpacked from an ePub.
How can I make it work without changing it from .xhtml to .html ??

I am working on iOS
