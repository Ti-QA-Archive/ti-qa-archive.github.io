app update for iphone5: screen height still iphone 4

Hi all!

I'm updating an app for iPhone 5 compatibility. I want the app to run on full screen height, now that the iphone5 packs a larger screen compared to iphone4 and 4S.

When I run the app in the simulator (in iphone 5 mode), there are black bars on top and at the bottom of my app instead of running the app in full height. The window height is set to 100%.

Does anyone know how to solve this? I'm running against the latest appcelerator SDK 2.1.3

Thanks!
