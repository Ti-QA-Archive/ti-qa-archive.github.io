    Please check my test code below. I've tested it and works as expected.

    textField.addEventListener('change', function(e) {
    var temp=aTextField.value;    
    var ok = /^[0-9]*\.[0-9]{2}$/;
    if(ok.test(temp))
    {
        alert(temp);
        //textField.editable=false;
    }
});

    Hope this will solve your problem.

    • The TextField object does not include any property that would let you restrict input to a specific pattern. Instead, you'll need to add a 'change' event listener and use one of these suggestions to reformat the user's input to the desired pattern.

      textField.addEventListener('change', function() {
   textField.value = textField.value.parseFloat().toFixed(2).toString();
});
