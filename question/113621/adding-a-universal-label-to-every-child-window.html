Adding a universal label to every child window

Hi,

If i need to have a universal label on every child window in my program, say "Copyright 2011".

I created a label.js on "Copyright 2011". How do i add this "label.js" to every footer of my child window or whether it is possible to do it?

  • you could try to create a generic container with the label in the footer, then include in it from time to time the different page you're browsing.

    Something like:

    Ti.UI.createWindow({
    url:'genericContainer.js',
    page:'myCurrentPage.js'
});

    and inside genericContainer.js

    Ti.include("pages/"+currWin.page);

Ti.UI.createLabel({text:'copyright'});

  • you could create a base view that has the copyright as a label on the bottom. That view is the container for all of the other windows and views in your app.

    when creating all of the other windows, add them to the base view, but make sure you make the height such that the baseView still appears at the bottom and your copyright label will always be visible
