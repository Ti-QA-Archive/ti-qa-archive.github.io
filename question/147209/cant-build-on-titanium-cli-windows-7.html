    • Perhaps the Titanium CLI doesn't know where the Android SDK is installed. Start by running:

      titanium config android.sdkPath

      If it says "not found" or the path is incorrect, you can fix it by running:

      titanium config android.sdkPath /path/to/android-sdk

    • Hmm, ok, so that looks right. The Android build requires Python 2.7, so let's check if it's installed. From the command prompt, run:

      python --version

      On my machine, it says 2.7.2. If you do not have Python installed or have Python 2.6 or older or Python 3.0 or newer, then go here to download the Titanium-compatible Python 2.7 release: http://www.python.org/download/releases/2.7.3/.

      If you have Python installed, but the command above failed, then it's not in the path. You'll have to add the Python path to the list of paths in the Environment Variables.

      If you have Python 2.7 installed and "python" is in the path, then we need to debug a bit further. Let me know how it goes. Good luck! :)
