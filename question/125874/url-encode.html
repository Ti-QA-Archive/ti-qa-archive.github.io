URL encode

Hi,

I thought this was possible through the Ti.Codec module… but that seems to have changed dramatically with 1.7 to be converting in/out of buffers.

So, to cut a short questions shorter, how can I encode a string to pass as along as a parameter in a URL… basically urlencode type func..

Cheers

SDK 1.7.2, iOS

    Use the encodeURIComponent method of the Ti.Network class module:

    var encoded = Ti.Network.encodeURIComponent('encode this & that');
