    Hi

    The short answer is; not currently possible. Keep readin though, good news does follow.

    Longer answer;

    There are simply not enough parameters available to us to set both the tint colour *focused elements) and the label colour. You will notice the lack of colour information for the caption in the official documentation.

    • iOS 7 Design Resources - Segmented Controls

    OK, so we know iOS does not allow it directly, so how about we cheat? Looks like the other app has - so why not us?

    You have not said if this is Titanium with the Alloy Framework or Titanium without the Alloy Framework - that helps provide suitable answers. Please make sure you do provide this in the future.

    The following code is a quick example that achieves your goal;

    • Titanium with the Alloy Framework
    <Alloy>
  <TabGroup id="tabs" backgroundColor="transparent" barColor="#c60000" tintColor="#fff">
    <Tab id="tab4" titleid="four" icon="KS_nav_views.png">
      <Window id="win4" titleid="four" class="container" translucent="false">
        <TitleControl>
          <View width="200" height="40">
            <TabbedBar platform="ios" width="200" index="0" tintColor="#860000">
              <Labels>
                <Label> </Label>
                <Label> </Label>
              </Labels>
            </TabbedBar>
            <Label text="One" color="#fff" font="{ fontSize: 10 }" left="0" top="0" height="38" width="100" textAlign="center" />
            <Label text="Two" color="#fff" font="{ fontSize: 11 }" right="0" top="0" height="38" width="100" textAlign="center" />
          </View>
        </TitleControl>
      </Window>
    </Tab>
  </TabGroup>
</Alloy>

    *I strongly recommend you convert all of these properties into tss class entries, I have left them in-line purely for my testing needs

    • Titanium without the Alloy Framework
    // create the tab group
var tabs = Ti.UI.createTabGroup({
    backgroundColor: "transparent",
    barColor: "#c60000",
    tintColor: "#fff"
});
// define the title control
var titleControl = Ti.UI.createView({
    height: 40.
    width: 200
});
var tabbedBar = Ti.UI.iOS.createTabbedBar({
    index: 0,
    labels: [' ', ' '],
    tintColor: "#860000",
    width: 200
});
var lbl1 = Ti.UI.createLabel({
    color: "#fff",
    font: {
        fontSize: 10
    },
    height: 38,
    left: 0,
    text: "One",
    textAlign: center,
    top: 0,
    width: 100
});
var lbl2 = Ti.UI.createLabel({
    color: "#fff",
    font: {
        fontSize: 10
    },
    height: 38,
    left: 0,
    text: "One",
    textAlign: center,
    top: 0,
    width: 100
});
// add the tabbedBar and two labels in this order
// to the titleControl view, this gives us a
// single thing to add to the window
titleControl.add(tabbedBar);
titleControl.add(lbl1);
titleControl.add(lbl2);
// create the window and assign the title control
var win = Ti.UI.createWindow({
    titleControl: titleControl,
    titleid: "four",
    translucent: false
});
// create the tab and assign the window
var tab = Ti.UI.createTab({
    icon: "KS_nav_views.png",
    titleid: "four",
    window: win
});
// add the tab
tabs.addTab(tab);
// open the tab group
tabs.open();

    • So how does this work?

      It simply uses the colours we can adjust on the tabbedBar and then sets its labels to empty strings. We then use actual labels and place them over the top of the tabbedBar. These labels can have all the normal styles attached - so we can gain our white text. Your could adjust the tint of the selected or unselected label if you choose - that is a choice for you.
