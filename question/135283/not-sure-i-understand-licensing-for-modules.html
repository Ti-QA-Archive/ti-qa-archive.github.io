Not sure I understand licensing for modules.

When it says X amount per month, what does that mean?

Does it mean that if I use a module and I pay for one month then I won't be able to compile my project 4 months from now? If that is the case, can I just pay for one month, compile and not pay again until I need to recompile?

What does per seat mean? That only I, as a developer can use it? Can I use it in any machine I use to develop on?

  • Per month means you can use/compile/submit to store the app in those 30 days.
    If after 2 month you need to make an update you will have to pay for one month (at least) again.

    Per seat means that one user only can use the module.
    If you have a company with 5 employees and pay for one seat, then only one will be able to use the module.
    You can use it on any machine as long as you login with Ti Studio using the authorized account.

    Makes more sense?
