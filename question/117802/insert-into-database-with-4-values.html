Insert into database with 4 values

I'm trying to insert 4 values into my database, but it gives me a blank array of data as a result.

My insert statements:

db.execute('INSERT INTO DATABASETEST (ID, NAME, TYPE) VALUES(?,?,?)', 1, "Soccer", "Sport")

With 3 values it works.

db.execute('INSERT INTO DATABASETEST (ID, NAME, TYPE , DATO) VALUES(?,?,?,?)', 1, "Soccer", "Sport", "Date")

If I change it to 4 values it doesn't work :(

Does the insert statement take no more than 3 values ?

  • Nicolai

    There is no problem with db.execute in relation to inserting values. The Working with Local Data guide demonstrates inserting 5 values, in fact.

    Depending on your mobile platform and Ti SDK version (which you really should have stated, just like any question you post), the values may need to be in array form. See Ti.Database.DB.execute. That said, I have used the same format as you, when developing for Android, and it works fine.

    If this suggestion fails, please paste your create table statement.
