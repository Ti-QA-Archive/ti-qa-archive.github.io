    I had this issue as well, the block of code you posted is not the code it's going wrong.

    Somewhere in your file (Above the code you mentioned) you probably added a ' somewhere that shouldn't be there. For example: 

    var obj = {
    prop1 : 'hello',
    prop2 : 5,'  << there
}

    The studio doesn't recognize it as a syntax error, this website does though, so my comment is in green.

    Hope this helps you, if you can't figure it out post your code in the comments of this answer and I will try and fix it for you ;).
