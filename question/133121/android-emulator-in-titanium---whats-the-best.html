  • Accepted Answer

    To get the broadest support, target the lowest version that Ti supports. That is currently 2.1

    The only difference between the "Google" version and the "Android" version is that you need to select the "Google" version if you are going to use a mapView, else just use the "Android" version.

    For device screen sizes you can opt to let Ti auto-scale the app (by setting the anyDensity value in your TiApp.xml or use 'dp' values for your control sizes.
