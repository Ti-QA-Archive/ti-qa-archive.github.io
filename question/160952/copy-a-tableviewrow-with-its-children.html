Copy a TableViewRow with its children

Hi all,

I have a tableViewRow that have 3 children (2 labels and a text field), I want to add a new row which has the same children of the current row, I want to add it when the current row is pressed.

Here's my code to explain more

<TableViewRow onClick="insertNewRow">
    <Label class="fieldsDescriptionLables" text="  Primary" />
    <TextField class="emailFieldsTextFields" id="txt_primaryEmail" />
    <Label class="fieldsPrivacylabels" text="Privacy" onClick="openPicker" />
</TableViewRow>

function insertNewRow(e) {
    $.table.insertRowAfter(e.index, e.row);
}

I want the new row to have the same children, but current row is added to the new position as the new row leaving the old row with no children (2 labels and a textfield) but with the same properties.

How to fix this ? or
How can I take a copy (not a reference) of the row's children (with their properties and eventListeners) to put the new copy in the new row ?

Thanks in advance.
