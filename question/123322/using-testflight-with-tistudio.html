    Yes, I have been doing that. In Ti Studio you build to "IOS Device" and it will install it in iTunes. You can upload the *.ipa that gets built in ~/Music/iTunes/iTunes Media/Mobile Applications to TestFlight.

  • Another way to to this to in Titanium Studion (version 3.1.3), is to select "Ad Hoc Distribution". So, right click on the project in Project Explorer, select Publish -> Distribute - Ad Hoc/Enterprise. Make sure to fill out the correct distribution certificate and provisioning profile (for ad hoc), and where you'd like the generated .ipa to be stored on your machine. Once the ad hoc .ipa is generated, you upload that to TestFlight. Hope that helps!
