Android Number Picker

Is it possible to create a standalone widget like this in Titanium, or do I have to create it manually?
I don't want the entire Time Dialog, just a simple number spinner - for a quantity field.

  • You can use a custom picker that is created by the use of Titanium.UI.createPicker's useSpinner: true

    Here is an example:

    var picker = Titanium.UI.createPicker({
    selectionIndicator:true,
    useSpinner: true,        //This will create a customized picker in Android
    type : Titanium.UI.PICKER_TYPE_PLAIN,
    top: 100,
    height: 100,
    columns: [myColumn]
});

    I hope it helps :)
