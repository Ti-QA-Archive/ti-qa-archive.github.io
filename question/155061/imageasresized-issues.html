imageAsResized() issues

I am trying to resize a photo after it is taken by the camera here is my code:

$.takephoto.addEventListener("click",function(_e) {
    Titanium.Media.showCamera({
        success:function(event)
        {
            var image = event.media;
            var org_height = event.media.height;
            var org_width = event.media.width;

            var new_height = 900;
            var new_width = 1200;

            var new_image = image.imageAsResized(new_width,new_height);
        },
        cancel:function() { },
        error:function(error) { },
        allowImageEditing:true,
        saveToPhotoGallery:false
    });
});

However when I do this on Android I receive this:

Runtime Error: 
    Message: 
        Uncaught Error: length=1088606456; index=1089517056
    Source:
        var new_image = image.imageAsResized(new_width,new_height

Any ideas what would cause this?

  • Found out the solution, the problem was with the Android ROM I was using. I downloaded the stock rom for the phone and installed it and now everything works fine.
