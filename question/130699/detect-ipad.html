  • You Can alternatively Trim the OS name and get the first 4 characters like this:

    var str = Titanium.Platform.osname;
var res = str.substring(0,4);
if(res == 'ipad')
{
    //alert('true');
    //put your ipad code here
}

    This way you neglect any other letters that comes after the first 4 letters.
