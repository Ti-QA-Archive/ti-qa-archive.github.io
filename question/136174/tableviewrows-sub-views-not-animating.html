TableViewRows sub views not animating...

First, I am pretty new to Ti, and still getting the hang of programming (been a web designer / developer for 3+ years and have been trying to tackle programming for the past year or so).

I am making an app for a local company and I have a page (window) with a tableView that has more rows than will fit in the given space. That isn't a problem. The problem comes when I have a settings button on the top of the window (it's own object inside of the header of the page, no linking to the tableView), that when clicked makes the arrow view (an arrow icon) slide out to the right, and has an edit button slide in from the right (this is within the tableViewRow).

It works perfect…for the tableViewRows that are currently in the user's viewpoint. However the ones that are hidden below or above the tableView's height dont change in any way. I was wondering if there was a way to get all of them to have the edit button. Anyway, here is the code, any help is appreciated greatly.

btnSettings.addEventListener('click', function(e)
{

    var slideUp = Titanium.UI.createAnimation();
    slideUp.bottom = -6;
    slideUp.duration = 500;

    bottomShadow.animate(slideUp);

    var slideOutRight = Ti.UI.createAnimation();
    slideOutRight.right = '-100%';
    slideOutRight.duration = 800;

    var slideInRight = Ti.UI.createAnimation();
    slideInRight.right = -26;
    slideInRight.delay = 700;
    slideInRight.duration = 600;

    // Slide Out Arrow
    for (var roomTabs = 0; roomTabs < houseRooms; roomTabs++) {
        arrow[roomTabs].animate(slideOutRight);
    };

    // Slide In Arrow
    for (var roomTabs = 0; roomTabs < houseRooms; roomTabs++) {
        btnEdit[roomTabs].animate(slideInRight);
    };

});

  • Its probably because they are not rendered yet, therefore you cannot animate them. You can set the properties directly, without any Animation. Try with element.size.width, whether it returns zero or a small number which is not the width. That means its not rendered yet, didnt test this so gotta play around a bit.
