Choose which android sdk to build against.

Is there a way to choose which android sdk to build against? My project seems to be building against 8, which doesn't allow me to limit my app to handsets only. Trying to do something like this

<supports-screens
android:smallScreens="true"
android:normalScreens="true"
android:largeScreens="true"
android:xlargeScreens="false"/>

But the extra large screen property doesn't exist in 2.2
