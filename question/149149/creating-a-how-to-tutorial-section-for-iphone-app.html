    Hi

    Fonts first, read the Custom Fonts guide which explains what you need to do to include your own choice of fonts.

    The are two common ways of providing simple "How To" guides in an app.

    • Page by Page, use a scrollableView on a new window, which each page providing a single feature for the user to understand. On each page you can add any normal views to tell that story, including labels and images. The idea is the user will slide page by page to understand the app. Make sure you display the page indicator.
    • List Style on a new window create a table with a list of titles, each heading titles focuses on ONE ascept of the app. Selecting a title opens a new window with a webView, you load LOCAL HTML content into this, as this allows a simple display of formatted guide information. You can add a second level of titles before webView if you have a lot of information, just balance the extra information to actual need to explain yourself.

    The choice of these or a variation is up to you, but based on the type of information you want to show.
