    • Hello Sangeetha,

      Here You are using a PICKER_TYPE_DATE. Due to this, You are getting the Value of time is 0.
      Suppose, Please see the Sample

      Ti.UI.backgroundColor = 'white';
var win = Ti.UI.createWindow({
    exitOnClose : true,
    layout : 'vertical'
});

var picker = Ti.UI.createPicker({
    type : Ti.UI.PICKER_TYPE_DATE,
    top : 50
});

win.add(picker);
win.open();

picker.addEventListener('change', function(e) {
     Ti.API.info("User selected date: " + e.value.toLocaleString());
     Ti.API.info(new Date(e.value).getDate());
     Ti.API.info(new Date(e.value).getMonth());
     Ti.API.info(new Date(e.value).getMinutes());
     Ti.API.info(new Date(e.value).getSeconds());
});

      In above sample, i get the date and month when the picker value chaged but i dont get the value of min and sec. The reason is in Date Picker, we only get the Date(year,month,day).
      If you want the time then you need to use a time picker for that.
