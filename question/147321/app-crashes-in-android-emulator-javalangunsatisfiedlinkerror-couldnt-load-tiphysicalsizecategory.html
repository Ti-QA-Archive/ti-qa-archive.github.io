    Hi Anthony,

    to

    /yourWorkspace/yourAppFolder/modules/android/ti.physicalsizecategory

    Could it be that the modules folder is not present in the svn?

    • If you create a new Alloy dummy project, ti.physicalSizeCategory-android-1.0.zip will be present in /yourWorkspace/yourDummyAppFolder

      Make a backup of ti.physicalSizeCategory-android-1.0.zip outside yourDummyAppFolder.

      Copy ti.physicalSizeCategory-android-1.0.zip* to /yourWorkspace/yourAppFolder where the file is missing.

      Try running you project

      — commented January 23rd 2013 by Matthias Kroeger
