Background webservice notification example (gist)

Not a question, just something you mind find useful.
I had thought that network activity was not possible from a 'paused' app in the background. However, I'm happy to say that it does work. (For now)

Think of it more like a 'pull' notification rather than a 'push' notification.

I had to go through a lot of trial and error to make it work without crashing on resume, but I think I've got it stable now.

See this gist code for some example script to put in app.js and also an example bg.js which will poll a server (webservice) for a response.

https://gist.github.com/867404

Hope this helps! :)
