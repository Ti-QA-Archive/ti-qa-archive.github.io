    • I tried a set of latitude and longitude on Titanium.Geolocation. reverseGeocoder , google map api, and yahoo YQL. The address I got from Titanium.Geolocation. reverseGeocoder is exactly the same with yahoo YQL.

      Google states that a daily request limit is 2500. Does the daily limit base on ip address? That means even thousands of requests from my app, it will not hit the limit, right?

      — commented September 30th 2011 by Mel Maxwell

    • If it is in fact using Google, the limit does seem to be based on ip address so only would have an issue if you an error in your code causing thousands of requests or if the user requests location an unbelievably high number of times.

      You may occasionally see the error when you install to device from within Titanium or in the simulator, but on the actual device it is unlikely to be an issue.
