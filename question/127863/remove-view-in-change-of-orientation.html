  • I don't know what is returned by the modules, the view already created or a method to create a view? I say that because I think that when you do

    myApp.main.remove(myApp.portrait());
myApp.portrait = null;

    portrait has parentheses, which I'm pretty sure it shouldn't be a method, but a object to remove from the view.

    Anyway, I would not create two different views. Instead, IMHO would be better to create only one view with properties for landscape and portrait orientations. And, instead of remove the view from the window, hide() or show() it or play with the visible property.
