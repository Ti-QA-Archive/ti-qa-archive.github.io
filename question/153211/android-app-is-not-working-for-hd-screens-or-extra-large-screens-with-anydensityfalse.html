Android app is not working for HD screens or extra large screens with anyDensity:false

Application type: mobile,
Titanium SDK: 2.1.4,
Platform & version: Android,
Device: Samsung S III / S IV / all High density- Extra large resolution screens

I have my app working perfect on android devices with medium and large screen, But it is not working proper (UI) on high density screens(Extra large screens)
In tiApp.xml I've following code

<android xmlns:android="http://schemas.android.com/apk/res/android">
<manifest android:versionCode="6" android:versionName="1.2.6">
<supports-screens android:anyDensity="false"/>
</manifest>
</android>
With this app perfectly works on medium and large screens, Also tested on provided titanium emulator and on real devices.

But, I am not able to detect HD emulator with titanium, Please anyone suggest me how to get titanium emulator for HD support?

Also, if I change the code in tiApp.xml as,
<supports-screens android:anyDensity="true"/>
Then, I also need to change the code into 'DP' format.

is there any other solution which require minimum code change support?

  • Hi Sunita,

    Google says you should not be setting any density to false, changing to dp is probably the simplest solution and is good practice.
    From the Android Docs:

    "For applications that support Android 1.6 (API level 4) and higher, this is "true" by default and you should not set it "false" unless you're absolutely certain that it's necessary for your application to work."

    http://developer.android.com/guide/topics/manifest/supports-screens-element.html

  • Finally, my app is started working on each screens(android including large & HD screens, also on iPhone)

    I have not change any code tiApp.xml.

    <android xmlns:android="http://schemas.android.com/apk/res/android"> <manifest android:versionCode="6" android:versionName="1.2.6"> <supports-screens android:anyDensity="false"/> </manifest> </android>

    You should have your controls/ images, etc. height, width, top, bottom,etc with respect to platform height & width, etc. using Ti.Platform.displayCaps.platformHeight & Ti.Platform.displayCaps.platformWidth resp.

    Also, for images we don't need to worry. with the above platform width - height code, it will use.
