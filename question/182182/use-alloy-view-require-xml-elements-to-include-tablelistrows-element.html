Use Alloy view "Require" XML elements to include "TableListRows" element

One question about using Alloy view "Require" XML element.
I have one TableList view and one TableListRow view by using Alloy "Require" XML element can include TableListRow view into TableList view. Sample code like below :

TableList View (main.js).:
<Alloy>
<Collection src="tblListInfo"/>
<Window id="mainwin">
<ImageView id="headerImg" image="images/header.jpg"/>
<Toolbar top="20dp" height="50dp">
<Items>
<Button id="mapButton" systemButton="Titanium.UI.iPhone.SystemButton.ACTION" onClick="handleAction" />
<FlexSpace/>
<Label>Visit List</Label>
<FlexSpace/>
<Button id="logoutButton" systemButton="Titanium.UI.iPhone.SystemButton.DISCLOSURE" onClick="handleLogout"/>
</Items>
</Toolbar>
<ScrollableView>
<View id="view1" backgroundImage="images/header.jpg">
<Label>View1</Label>
</View>
<View id="view2" backgroundColor="#246" >
<Label>View2</Label>
</View>
<View id="view3" backgroundColor="#48b">
<Label>View3</Label>
</View>
</ScrollableView>
<TableView id="table" dataCollection="tblListInfo" onMove="handleMove" onLongpress="handleLongPress">
<HeaderPullView platform="ios">
<View class="pull">
<Label color="F2F4F4" bottom="25dp">Header pull view</Label>
</View>
</HeaderPullView>
<Require id="listRows" src="custlistrow"/>
</TableView>
</Window>
</Alloy>

TableListRow (custlistrow.js)
<Alloy>
<TableViewRow class="tablerowview" onSwipe="handleSwipe">
<View id="entrypointview">
<ImageView class="enrtypointimage" id="omentrypoint" onClick="handleTouch"/>
<ImageView class="enrtypointimage" id="redentrypoint" onClick="handleTouch"/>
<ImageView class="enrtypointimage" id="eqentrypoint" onClick="handleTouch"/>
</View>
<View id="customerview" backgroundColor="white">
<ImageView id="customerphoto" image="{customerImage}"/>
<Label id="customername" text="{customerName}"/>
<Label id="customeraddress" text="{customerAddress}"/>
<ImageView id="phoneimage" image="/images/phone.png"></ImageView>
<Label id="customerphone" text="{customerPhone}"/>
<ImageView id="customergrade" image="/images/gold.png"/>
<Label id="customergradename" text="Supermarket"/>
<ImageView id="customerflag" image="/images/right.png" onClick="rightAnimate"/>
</View>
</TableViewRow>
</Alloy>

in the controller "main.js" file i would like use below code to get "customerflag" ImageView in custlistrow, but i get the "undefined is not an object (evaluating '$.listRows.getView') at main.js" error.

var view = $.listRows.getView('customerflag');

I search support doc below code should work but the error really confuse me, anyone can help me with some explanation?
