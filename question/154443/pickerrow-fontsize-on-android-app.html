    It's even not as easy as I thought in native way because PickerRows don't support Views. That means adding a label to the PickerRow won't work too. Titanium uses an ArrayAdapter here wich only takes Strings. As you can see here adding / removing views is not supported.

    The only way to accomplish this seems to be a native module for the Picker with a custom Spinner Adapter - for now.
