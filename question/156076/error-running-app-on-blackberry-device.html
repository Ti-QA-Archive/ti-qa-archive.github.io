    It appears the Titanium build command can not find your debug token. I assume since it works in QDE you got your signed keys from BlackBerry and registered them. Also you must in Studio BlackBerry Platform Preferences set the path to the debug token.

    • The issue was with the configuration running.

      In the platform settings the debug token was selected and password was right.

      When I did "Run Configurations…" the debug token was pointing to "bbidtoken.csk" instead of the debug token "debugtoken1.bar". Once I fixed this it worked properly.
