Custom AndroidManifest Woes...

Alright, so I am really wanting to start using tiapp.xml to write into my AndroidManifest.xml so I don't keep having have to build, copy+paste, rename to androidManifest.custom.xml and compiling again, etc.

But whenever I try adding:

<android xmlns:android="http://schemas.android.com/apk/res/android">
<manifest package="ANDROIDPACKAGE" android:versionCode="#" android:versionName="#.#">
<uses-sdk android:minSdkVersion="4" />
</manifest>
</android>

I get the "Error generating R from manifest" error in the console.

Any help? I remember last time I posted about this there was a build that went out that fixed only having properties, but it didn't seem to completely fix the problem.
