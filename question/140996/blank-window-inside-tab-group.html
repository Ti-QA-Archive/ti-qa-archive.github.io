Blank Window inside Tab Group

I have created a tab group in app.js . I Then have created table view and have given url links.
When i click on a new url i create a new tab group and hide the existing tab
It works perfect only when i click back on android it give me blank screen of tab group.

window.js
'

var createtabgroup = Titanium.UI.createTabGroup();

var touristwindow = Titanium.UI.createWindow({
title:"CheckTourist",
backgroundColor:"#fffff",
navBarHidden:true, //Hide the nav bar for the window
tabBarHidden:true //Hide the tab bar for the window
});

var toursttab = Titanium.UI.createTab({
title:"Tours",
window:touristwindow
});

createtabgroup.addTab(toursttab);
createtabgroup.open();
'

Pls provide me the solution

  • You can only have one tab group in an android app. And you can't alter the tabs in a tabgroup once created.
