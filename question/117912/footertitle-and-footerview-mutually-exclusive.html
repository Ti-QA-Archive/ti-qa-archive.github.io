footerTitle and footerView mutually exclusive?

Are footerTitle and footerView mutually exclusive? If I have a footerTitle defined in my TableView, the footerView doesn't show up. Is this expected behavior?

    Hi Joe

    That appears to be the case from the API docs, yes, which states:

    footerTitle (string)

    the table view footer title

    footerView (object)

    the table view footer as a view that will be rendered instead of a label

    uivmm, footerTitle is simply provided as a convenience, to automatically create a label (which is a view object) and add it as a footerView. Obviously you won't have any other control over its appearance other than the text it displays.
