  • Hi

    In order to rule out your code, please;

    • Create a new default project
    • Run it to the emulator

    Does this work? If if does then it is your code, if it does not then it is your setup and the fact that the Android emulator is flaky.

    Also try;

    • Close the emulator if it is running
    • Locate Dashboard tab in Titanium Studio
    • Select Get Started
    • Select Android
    • Select Update Android SDK
    • Wait for Android SDK Manager** to start
    • Menu; Tools > Manage AVDs
    • Find yours, remove it (or remove all, Titanium will recreate as it needs)
    • Close Manage AVDs
    • Close Update Android SDK

    Try the new default project app again and also try your app again.
