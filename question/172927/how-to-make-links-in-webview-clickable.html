How to make links in WebView clickable

Hi,

I am getting JSON data back from a REST web-service and showing the data in a WebView as the returned data contains HTML tags. I am creating the WebView in the following way -

    var localWebview = Titanium.UI.createWebView({
        layout : 'absolute',
        scalesToFitPage : false,
        height : Ti.UI.SIZE,
        top : "0dp",
        left : "10dp",
        right : "0dp",
//totalWriteup is the JSON data I am getting back from the webService.
        html : totalWriteup,
        backgroundColor : '#01FFFFFF',
        touchEnabled : false,
        enableZoomControls : false,
        // To disable phone nos. appearing in the write up section as hyper links
        autoDetect : []
    });

My problem is that sometimes I also get website links back from the backend. These links are displayed in the webView properly but when I click on them nothing happens both in android and iPhone. I want the links to be directed to the respective webpage.

A sample of my back-end data (totalWriteup mentioned above) is as below. I have stripped down the data to show only the 

< a href>

tag which is not being clickable in the WebView

<!DOCTYPE html> 
    <html> 
    <head> 
    <meta name="viewport" content="width=device-width,user-scalable=yes,initial-scale=1" /> 
    <meta http-equiv="content-type" content="text/html; charset=utf-8" /> 
    <ul>
          <li><a href="http://pizzaonline.dominos.co.in/index.php?src=weekend-bonanza&amp;utm_source=Vantage&amp;utm_medium=ROS&amp;utm_term=CPC&amp;utm_campaign=weekendbonanza">Next click here</a></li>
           <li>And apply voucher code while ordering</li>
      </ul>
    </body>
     </html>

I am using Titanium CLI version 3.2.1, Titanium SDK version 3.2.2.GA, Alloy version 1.3.1., Developing mobile app both for iPhone and Android.

Hope some one can guide me in the right direction.

Thanks,
Bably

    • Best way from what I can tell is using this as follows:

      localWebview.addEventListener('beforeload', function(e) {
    if (e.url.indexOf('http') > -1) {
        e.bubble = false;
        e.source.stopLoading();
        Ti.Platform.openURL(e.url);
    }
});
