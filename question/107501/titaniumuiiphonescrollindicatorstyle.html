Titanium.UI.iPhone.ScrollIndicatorStyle

I have dark styled windows and need to use Titanium.UI.iPhone.ScrollIndicatorStyle.WHITE (from the API doc). But this constant is neither defined nor can I find any property that uses it. The doc also relates it to ActivityIndicator objects which seems to be an copy&paste error.

Can somebody tell me whether this feauture is available and what property it is related to? Or: how can I get a light scrollbar for dark background?

  • Not sure if you're still looking for this, but the answer is
    scrollIndicatorStyle: itanium.UI.iPhone.ScrollIndicatorStyle.WHITE,
    as in:

    var    scrollView = Ti.UI.createScrollView({
    contentHeight: 720,
    contentWidth:'auto',
    height: 720,
    width: '100%',
    showHorizontalScrollIndicator: true,
    showVerticalScrollIndicator: false,
    top: 0,
    left: 0,
    id: 'scrollView',
    scrollIndicatorStyle: Ti.UI.iPhone.ScrollIndicatorStyle.WHITE
});
