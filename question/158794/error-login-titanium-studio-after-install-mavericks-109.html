  • This saves me:

    Click this link

    Turns out it's not related to the specific certificates on our server, but rather due to the fact that Mavericks wiped out the existing Java install. To fully resolve the issue:

    1. Open the terminal and run: "sudo rm –rf /Library/Java/JavaVirtualMachines/*.jdk"
1. Download Java 7u45 from http://www.oracle.com/technetwork/java/javase/downloads/jdk7-downloads-1880260.html
3. Use the installer to install fresh. This resulted in good cert files under /Library/Java/JavaVirtualMachines/jdk1.7.0_45.jdk/Contents/Home/jre/lib/security/.
