take user to app store if app is not installed on android/iphone

I am building an app for android and i-phone using titanium. One part of my app starts when user hits a link in browser(e-g when user types the link into browser and press go button, this starts my app) now the issue is if my app is already not installed on the phone i like to send the user to android/apple app store so that user may download the app and install it before hiitng the URL.

How to do this kindly help me as i am new to titanium. Thanks in advance.

  • You cannot do this on iOS. The only way you can launch an App is to use its app protocol string to try and launch it. If the app is not installed the request fails and there is no way to hook into this failure. I suspect you might be able to do something in a native module for Android, but i dont have any firm information on that.

  • For Android we can pass the URL to find the another part app in market

    Exmaple Code :-

    
var win = Ti.UI.createWindow();
win.backgroundColor = 'white';
// Create a Button.
var aButton = Ti.UI.createButton({
    title : 'Link',
    height : 40,
    width : 150,
    top : 300,
    left : 80
});

win.add(aButton);
aButton.addEventListener('click', function() {
    Titanium.Platform.openURL("https://play.google.com/store/apps/details?id=com.cricbuzz.android");
});
win.open();

    If you click the button it will open the cricbuzz app in market. What you need to do is
    Titanium.Platform.openURL("In this place you need to give your URL in market.");
