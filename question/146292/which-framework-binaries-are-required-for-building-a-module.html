    Hi Joseph, you can check your framework library for architecture and its not necessary to build the framework for i386 but when you test your module using titanium run command, then it build for simulator so will give error if symbols are missing.
    if the framework is not for i386 then, just package your module and deploy for iOS device in Titanium app project.
