Appcelerator Studio Install Error

When attempting to install Appcelerator Studio I get the following errors:
"12157 - http://go.appcelerator.com/installer_git_windows.exe", click "continue" and then;
"MSYSGIT mandatory prerequisite was not correctly installed", click "continue" and then;
"The Appcelerator Studio Setup Wizard ended. Appcelerator Studio ended prematurely because of an error."
No further information is given. Any help will be appreciated.

— asked May 29th 2015 by George Jump

    1. Highlight the http:/go.app …. FULL address and copy Ctrl C
    2. Exit the Studio installer
    3. Startup your browser and down load the file from step #1 by pasting to address bar of Chrome Ctrl V
    4. When program downloads, run downloaded GIT Development program
    5. Re-Run Appcelerator Studio
