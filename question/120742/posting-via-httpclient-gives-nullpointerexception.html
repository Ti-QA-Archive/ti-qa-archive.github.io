Posting Via HTTPClient gives NullPointerException

Getting Wrapped java.lang.NullPointerException using the code below.
Is this a bug? Everything seams correct.

  • Using 1.2.2
  • Android 2.2
    ~~~
    var xhr = Ti.Network.createHTTPClient();

xhr.onload = function(e) {
return JSON.parse(this.responseText);
};

xhr.onerror = function(e) {
return 'ooops, ya broke it.';
};

xhr.open('POST','192.168.1.1', false);
xhr.send('stuff');
~~~

Thanks guys!

  • xhr.open('POST','192.168.1.1', false); // <-- false param is ignored
xhr.send('stuff'); //<--- wrong, something like this->> xhr.send({'param1':'stuff});
