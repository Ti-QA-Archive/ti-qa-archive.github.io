Alloy UI problems

I am working on an alloy app (first Alloy project) that needs to have a tabgroup with multiple tabs and a common control panel/bar across all tabs. I found that the only way I can do this is to make the tabgroup the default window in my index.xml and then I add the control panel to the windows in the tabgroup. For the first tab, the rest ofthe tab/window is a listview and the control panel is at the bottom of the window. This displays both the listview and the control panel, however the listview click events are never fired. The click events on the control panel work fine. The control panel is added last. Here is my window XML. How can I get click events working for both the listview and the control panel view?

<Alloy>
<Tab title="Content Links">
<Window title="Content Links">
<ListView id="list" class="contentLinkListView" defaultItemTemplate="contentLinkTemplate" onItemclick="onItemClick">
<Templates>
<Require src="ContentLinksItemTemplates"/>
</Templates>
<ListSection>
<ListItem title:text="My first content link!" subtext:text="I hope this listview works…" />
<ListItem title:text="My second content link!" subtext:text="I hope this listview works again… "
thumbsUp:image="images/thumbsup_checked.png"/>
<ListItem title:text="My first user post!" subtext:text="So and so says this listview rocks!" />
<ListItem title:text="My third content link!" subtext:text="I hope this listview works again…"
thumbsDown:image="images/thumbsdown_checked.png" />
</ListSection>
</ListView>
<Require type="view" src="ControlPanel" id="controlpanelView"/>
</Window>
</Tab>
</Alloy>
