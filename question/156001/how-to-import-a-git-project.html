  • You cannot clone just one project from a git repository. And in this case, this repo has several projects,

    What you can do however is to clone the repository on your machine using the following command:

    git clone https://github.com/appcelerator-training/tcad-code.git

    Once you have downloaded the whole repository on your machine, you can then import your project using the Import feature from Titanium Studio.

    Hope this helps.
