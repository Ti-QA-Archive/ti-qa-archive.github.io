background selected color with label and tableviewrow

Hi all, i have a tableview row that contain a label. The tableiewRow contains a rightImage property ( a custom arrow that tells the user to be redirected to another window). I need this: When the user clicks on the row, all the row background will be highlighted red. I put backgroundSelectedColor : '#960000', and its working when the user clicks in the area around the title of the row ( the title of the row is in the label). But when the user click exactly on the title ( the text of the label ), the row will not be highlighted. I tried to put the backgroundSelectedColor property to the label, its nice but the right side of the row ( where we have a rightImage) its not highlighted. Anyone can help me to let all the row highlighted wherever the user clicks in the row?

    Try this (not tested)

    tableview.addEventListener('click',function(e) {
    e.rowData.backbroundColor='red';
});

  • Hi Anthony,

    Try to set the 'touchEnable' property of the label to false. Doing that, when you touch on the label the event will be forward to the row, applying the backgoundSelectedColor of the row. You can see the documentation of the property
    here.
