Titanium Android module - include a JAR that contains other JAR files

Hi,
i have a native android in-app library, and i want to create a module in titanium so it can be used in titanium apps as well.
in my library, im using other jar files. im using ant build in order to compress my library into a jar file so it can be used in apps.
when im testing the module, im getting errors that the inner jars (the jars im using inside the library), are not recognisable, and the app crashes.
my question is - is what i have described possible? if so, how do i do that?

Application type: mobile,
Titanium SDK version: 3.0.2 (02/15/13 19:02 5a77fe7),
Platform: Android,
Device: Android emulator & physical device model,
Host Operating System: OSX,
Titanium Studio: 3.0.2.201302041757

  • Just put all the jars you're importing in the &#x2F;lib directory as described in theTitanium Guides. They will be automatically packaged by the build scripts.

    Keep in mind that they will be included in the final module zip file and not the jar itself. Java doesn't allow jars to contain other jars.

    Hope this helps.
