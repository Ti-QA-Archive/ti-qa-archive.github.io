Get ListItem by itemId

How can I get a ListItem by itemId? I can get it by index, but didn't find out how to get by itemId. Any idea?

  • Hi Shawn, 

    <Alloy>
    <Window>
        <ListView id="list" onItemclick="onItemClick">
            <ListSection>
                <ListItem itemId="1" title="row 1"/>
                <ListItem itemId="2" title="row 2"/>
                <ListItem itemId="3" title="row 3"/>
            </ListSection>
        </ListView>
    </Window>
</Alloy>

    Given above view, you could do this in the controller 

    function onItemClick(e) {
    // get the clicked section
    var section = $.list.sections[e.sectionIndex];

    // get the clicked item from that section
    var item = section.getItemAt(e.itemIndex);

    // print the item's itemId
    Ti.API.info(item.properties.itemId);
}
