  • Accepted Answer

    Unzip and then copy the files to the modules/iphone|android folder of your SDK folder

    &#x2F;Library&#x2F;Application Support&#x2F;Titanium&#x2F;modules&#x2F;iphone|android

    Then follow the instructions on how to 'require' it and use it.

    (You might want to force a clean build before testing it too)
