    Titanium Studio and Titanium SDK are two different things with different versioning.

    I don't think there is an SDK3.3.0 RC, but you can install one via Help->Install Titanium SDK from here:

    http://builds.appcelerator.com.s3.amazonaws.com/index.html#3.3.0
