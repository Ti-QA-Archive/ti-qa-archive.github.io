get TabGroup container window

What I'm trying to do is open a window when the user clicks a button that exists in a tab.

In android it works ok with window.open() while there is a back button on the device.

On iOS, I'd like that click to spawn a NavigationGroup that will lead back to the TabGroup when the user clicks the back button in the upper left corner.

To spawn a NavGroup you must add it to the current window and then open it but as far as I'm concerned you can't add a NavGroup to a TabGroup, TabGroup.window does not exist and Ti.UI.currentWindow returns null.

Additionally I don't want to add the NavGroup inside the TabGroup.

Is there any way to do that?

  • Tab has an open function
    http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.Tab-method-open

    tab.open(win);

    If you don't know what tab you're in you use Ti.UI.currentTab.open(win);

    on iOS that's visually similar to having a navigation group
