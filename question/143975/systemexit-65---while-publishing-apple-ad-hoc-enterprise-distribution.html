  • The last time I found that the following code would result in the SystemExit error of code 65 while publishing the distribution.

    var consumption = this.mReciprocal ? distance / gasUsed : gasUsed * 100
               / distance;

    I fixed it by not putting the above 1 statement in 2 lines.

    However I found the following code did not cause any trouble

    var distIgnored = this.mRecords[idx].odometer
          - this.mRecords[idx - 1].odometer;
