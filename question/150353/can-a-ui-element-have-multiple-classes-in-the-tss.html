  • While TSS share some common traits with CSS, they do not share the same behavior.

    To my knowledge, the only thing you can do right now is:

    • Use platform queries to have different look depending on platform
    • Override a style with the same name. (for example, you could declare a style in the global tss and override this style in "View specific" TSS)
