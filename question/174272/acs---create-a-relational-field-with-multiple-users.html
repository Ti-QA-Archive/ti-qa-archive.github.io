ACS - Create a Relational Field with multiple Users

I'd like to know if this is possible with ACS.

Imagine an ACS Place Object, that has multiple Users. So when I query Places, each Place would return the ACS Place and all the ACS_Users that are attached to it.

Currently in the documentation Relational Field, it seems that only One ACS_User can be added… is that correct?

  • Hi, not just One ACS_User can be added. You can added it with multiple users or other acs objects as a array.Like this:

    $ curl -b cookies.txt -c cookies.txt -F 'fields={"[ACS_User]Relation_id":"537d481b59112b08170044a6,5350e95615d82732300a7732"}' "https://api.cloud.appcelerator.com/v1/objects/Test/create.json?key=NU6QQdtWjA9DrmxeXSYRaPvrIigTQZuH"
