    Hi

    Be aware that there is an issue with some views and the methods and properties related to visible & setVisible once the view has been created. On some it works, but sometimes it does not.

    If it does work you can create a toggle version that each press will swap the value from true to false.

    botonnavegador.addEventListener('click', function(e){
    webview.visible = !webview.visible;
});

    or 

    botonnavegador.addEventListener('click', function(e){
    webview.setVisible(!webview.getVisible());
});

    These two versions using property and a setter/getter, use the benefit of the ! modifier, this simply swaps the boolean state of the associated boolean to the opposite.

    However if this fails you are better off with the show and hide;

    botonnavegador.addEventListener('click', function(e){
    if (webview.visible === true) {
        webview.hide();
    } else {
        webview.show();
    }
});

    This way always works.
