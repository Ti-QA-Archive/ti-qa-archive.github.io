  • That widget is the TabbedBar widget.

    http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.UI.TabbedBar-object

    More specifically, the style of that particular tabbed bar is set to "bar". You can achieve the same style by assigning the variable "style" to Titanium.UI.iPhone.SystemButtonStyle.BAR.

    Additionally, the background color can be set to further customize the widget.
