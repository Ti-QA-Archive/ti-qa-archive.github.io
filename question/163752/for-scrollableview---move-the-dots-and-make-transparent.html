For scrollableview - move the dots and make transparent

Hi everyone,
I have a scrollableview and now I have the dots showing underneath the images that scroll with a black bar as a background.

<ScrollableView id="scrollableView" showPagingControl="true" onClick="walkthrough">
                     <View id="view1">
                         <ImageView id="image" image="/images/bear.png" />
                    </View>
                        <View id="view2">
                         <Label class="title">Which Phase Are You In?</Label>
                         <Label class="small">An Illustrated Walk-through</Label>
                        <Button class="view" id="two" onClick="two">View</Button>
                    </View>
                ...

My goal is to move those dots up, change the active one to yellow and remove the black background. I can't find any documentation for this. Any help?

    Move it up with
    pagingControlOnTop: true

    Make it transparent with pagingControlColor: &#39;transparent&#39;

    Changing the dots color is not supported but you can achieve it with this hack or by replacing it with a custom component.

    • There is no method to set the top-position of the pagingControl itself but you can instead set the top of the views within the scrollableView to the negative value of the pagingControl height which is 20 by default.

      <ScrollableView id="scrollableView" showPagingControl="true" onClick="walkthrough">
    <View id="view1" class="scrollPage">
        <ImageView id="image" image="/images/bear.png" />
    </View>

    <View id="view2" class="scrollPage">
    </View>
</ScrollableView

      tss:

      ".scrollPage": {
    top: -20
}

      What do you mean by background of pagingcontrols? The background of the pagingControl is set by pagingControlColor (this should stay transparent when you are displaying it on the view)
