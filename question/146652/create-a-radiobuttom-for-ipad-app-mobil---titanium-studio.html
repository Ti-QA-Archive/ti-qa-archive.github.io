    Hi Diego,

    there is no iOS radio button by default.

    You may try to use a TabbedBar instead?

    var win = Ti.UI.createWindow({backgroundColor: 'white'});

var aTabbedBar = Titanium.UI.iOS.createTabbedBar({
    labels:['Female', 'Male'],
    backgroundColor:'#336699',
    top:50,
    style:Titanium.UI.iPhone.SystemButtonStyle.BAR,
    height:25,
    width:200
});
win.add(aTabbedBar);

win.open();

  • Hi. If you're intent on using radio buttons to select female or male, try using two views with custom background images of radio buttons. You would need a default state and a selected state of the radio button images. I would do something like this:

    
/// View file
<View id="maleRadio"/>
<View id="femaleRadio"/>


// Controller file
var radioValue = 'male';
$.maleRadio.on("click", function (e) {
    $.maleRadio.backgroundImage = '/img/radio-button-active.png';
    $.femaleRadio.backgroundImage = '/img/radio-button-default.png';
    radioValue = 'male';
});

$.femaleRadio.on("click", function (e) {
    $.femaleRadio.backgroundImage = '/img/radio-button-active.png';
    $.maleRadio.backgroundImage = '/img/radio-button-default.png';
    radioValue = 'female';
});


// Style file
'#maleRadio': {
    backgroundImage: '/img/radio-button-active.png'    

},

'#femaleRadio': {
    backgroundImage: '/img/radio-button-default.png'    
}

    There's probably a prettier way to do it, but this might be a good start for you.
