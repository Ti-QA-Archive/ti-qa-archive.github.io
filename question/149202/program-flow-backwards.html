Program flow backwards?

Hello. I've been having problems figuring out the flow of my android program based on "alert"s that were inserted for debugging. It appeared that the alert messages were displaying on my S3 in the opposite order of what I thought the program flow should be. So I wrote this little non-sensical code segment to test that theory and I get the same results. Executing this program displays the alert messages in reverse order, i.e., the last alert message is displayed first and so on. I imagine this problem is due to my novice stature in this Ti and javascript world and I'd appreciate any help. Thanks. Win 7, Ti 3.0, Android SDK 4.2

alert("program begin");

//dummy processing
var a = 1.23;
var b = 4.56;
var z = a + b;
alert("a+b = " + z);

//more processing
var g = (a+b-z)/(z * 2.5/6);
alert("(a+b-z)/(z*2.5/6) = " + g);

//more processing
if(g > z)
{
    alert("g>z " + g + ", " + z);
}
else
{
    alert("g<z  " + g + ", " + z);
}
