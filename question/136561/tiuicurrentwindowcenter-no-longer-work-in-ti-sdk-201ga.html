Ti.UI.currentWindow.center no longer work in Ti SDK 2.0.1GA

I am using Ti SDK to 2.0.1GA with Xcode 4.3
My code can be successfully built but prompt error when reaching the line:

var x = Ti.UI.currentWindow.center.x;

Here is the error message:

 message = "'undefined' is not an object (evaluating 'Ti.UI.currentWindow.center.x')";
    name = TypeError;

The same code run without any problem using Ti SDK 1.8 with Xcode 4.2

Just double checked with the latest API doc, the code should be fine.
So, is it a bug?

    That error message likely indicates that the center property is undefined, which would be expected unless you set it yourself earlier. The view layout system changed in 2.0 – take a look at the wiki doc describing the new system. If you need to get the center of your window, you should use the post-layout event as described in the doc instead of the center.x property.
