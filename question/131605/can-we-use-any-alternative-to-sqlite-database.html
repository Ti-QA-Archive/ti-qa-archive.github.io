    • I think the benefit of a NoSQL DB implementation in Titanium is with using object-style documents, which is handy since we are coding in JS.

      A benefit of using SQLITE is that it comes by default on iPhone and Android devices, and is supported by Appcelerator. You don't have to rely upon a 3rd party product.

      After that, I think it comes down to personal preference of SQL or NoSQL.
