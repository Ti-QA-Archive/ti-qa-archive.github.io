  • While the error doesn't give you any clue about the real problem, I was able to solve this one. It turns out it had something to do with the provisioning profile I used to run the app on the device. This profile wasn't valid anymore, so I generated a new one in the Apple Dev Center and added it to the keychain.

    I then removed the run configuration and created a new one with the current profivisioning profile. I hope this helps someone in the future.
