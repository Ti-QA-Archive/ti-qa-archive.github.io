Position an Imageview using X & Y rather than Top and Left?

Hi

Am i able to position an imageView with the X and Y co ordinates rather than just Top and Left? I need to position my imageViews from the center rather than the top, left bottom and right. I can output the Co ordinates with imageview.center.x and imageview.center.y but i cannot set them.

Thank you

    You can use a small function that sets the position based on the center of the view. 

    function setPos(newX, newY, view) {
    view.top = newY - view.height/2;
    view.left = newX - view.width/2;
}

    Thats just one way to do it.
    Hope it helps.

    You need to set the imageView based on the parent Center position. So either you can set the x and y of the image directly based on some calculative result for it parent total height and width or using the Top /left. You can always use the parent view property to center all object.

    And yes you cannot use imageview.center.x and imageview.center.y to set the center position as its readonly properties.
