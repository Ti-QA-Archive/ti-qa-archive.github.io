How to pass JS object by value, and retain its methods?

Hi everyone,

I'm developing a program which generates a layout view using web services, and then applies that view to every row inside a TableView. The problem is that the generated view is passed by reference and when I use it, all the rows have the same data, which I don't want.

I realised that JS passes objects by reference. So I tried to clone my layout view using this method:

    this.createTableRowViewPatternClone = function(){
        return JSON.parse(JSON.stringify(tableRowViewPattern));
    }

However, now I lost all the methods inside view, which I need.

Does anyone know how to pass objects by value in Titanium, but still retaining their methods? Or a workaround to this? Thanks in advance.
