Starting, setting the app.ID

So basic, but how do you set the app.id at the beginning? Is it asking for the address of your website or what? Sorry is this is a basic or covered question.
How do you write the app.ID?

    This is a unique identifier for your app. It needs to be unique throughout the world, so they recommend you use an asset (like your website domain name) that is unlikely to be duplicated by someone else. If you dont actually run a website this is not important. Perhaps use your company name instead. The recommended method is to reverse you domain details eg

    uk.co.mywebsite.myappname

    but you could also use something like

    uk.co.mycompanyname.myappname

    The most important thing is that it is UNIQUE
