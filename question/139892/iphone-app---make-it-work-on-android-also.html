iPhone App - Make it work on Android also??

We have an iPhone App, which is working fine and its on the Apple Appstore, but now we need to make it also work on Android devices.
Initially, we created the app explicitly for the iPhone, and now we would like to expand our customer base by adding Android accessibility.

QUESTION:

How do you make an iPhone App also work on Android devices??

We are currently trying to import our iPhone app into an Android project and trying to run it in the Android Emulator, but with no success.

Any help/advice/support appreciated..

Thanks guys…

  • Accepted Answer

    Easiest way: create new project and copy your Resource folder in the newly created project.
    The recent TiStudio updates also allows you to enable android support afterwards.

    Dont expect to run the app without any issues on android, it depends on your/devs experience of cross-platform coding. If your app was developed targeting iOS only you may run in some issue on android, since its a different platform with different controls etc.

    Therefore you will need to fix the code / write new / extend whatsoever.
