How can I set a loop (e.g. a while loop) for blocking app execution so that httpClient seems to be operating in sync mode?     Hi, When you send the request using createHttp client at that time in case of success it comes in unload callback function and in case of error or failure it comes in the error callback method.

    Therefore you can write the logic in the success or error callback method what you want to preform in case of success and failure. If you also want to preform other http client operation then just write in the success or failure callback function.

    Till you are not getting the response from the server if you also want to block the user operation then you can show activity indicator and you can hide it once it comes in success or failure callback.

    I think this will help you to solve your problem.
