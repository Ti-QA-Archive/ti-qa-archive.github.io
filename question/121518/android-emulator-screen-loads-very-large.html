Android Emulator screen loads very large

When I launch the mobile project HelloWorld with the Android Emulator, the screen is so large that I cannot see the bottom half of the emulator. Has anyone had this problem before? I cannot figure out how to adjust the emulator at all

I'm using Titanium Studio, build: 1.0.1 on Mac OS X 10.6.8

  • If you are using Studio, in the run/debug configuration for your project, change the Screen to HVGA. In Developer, you can do the same under Emulator.
