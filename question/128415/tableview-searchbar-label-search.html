TableView SearchBar Label search

Hello everyone,

I am creating a tableView with a searchbar, but there is one catch..
The data in the table are not pure strings.. in the TableViewRow, there will be a thumbnail, text, and some other stuff..
At the moment I have a search bar appear, but it does not turn up any results.

Thank you

    Use filterAttribute for the table and set it to point to a property of your row that you will set to describe the content inside the row so it finds it on search.

  • When creating the tableViewRow add your own property to it, with the value being the text you want to be searchable;

    Ti.UI.createTableViewRow({filter:'my text to search for…'});

    And when you create your tableView, specify that property as the filterAttribute;

    ti.UI.createTableView({filterAttribute:'filter'});
