    Hi

    You do not use the popover for this simply use the optionDialog, but add the extra view parameter in the show method.

    var dialog = Ti.UI.createOptionDialog({
    cancel: 2,
    options: [
        'Take Photo',
        'Choose Photo',
        'Cancel'
    ]
});
dialog.show({
    view: btnRefresh2
});

    This will ignore the cancel option as it is on the iPad and give you exactly the same as the screenshot.

    btnRefresh2 is the view you want it to point to - change to suit.
