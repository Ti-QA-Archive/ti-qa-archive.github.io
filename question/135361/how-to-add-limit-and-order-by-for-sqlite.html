how to add LIMIT and ORDER BY for sqlite

titanium mobile sdk 1.8.2

iphone sdk 4.3

Error: 'INVALID SQL STATEMENT'

Hi,

This is how I made the query

var max = 5;

var db = Ti.Database.install('db/fielddb.sqlite','fielddb');

var rows = db.execute('SELECT DISTINCT id, cat, reg FROM fields ORDER BY id ASC LIMIT ?', max);

How can I fix this?

thanks

    try:

    var query = 'SELECT DISTINCT id, cat, reg FROM fields ORDER BY id ASC LIMIT ' + max;
var rows = db.execute(query);
