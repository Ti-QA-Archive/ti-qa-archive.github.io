Change the "value" of the iPhone Switch

Hello, how can I change the value of the Switch? by default, it is on ON/OFF but in my case I need to change it to 0/1 or another content, does this possible? or can you suggest me other controls?

    1. This function is part of private Cocoa APIs and AppStore guidelines doesn't allow usage of private APIs in your project. They will refuse your AppStore submission and already did in the past (see Camera+ app for example).

    2. Titanium acts as a wrapper for official Cocoa API and thus this function is not wrapped. Simple as that.
