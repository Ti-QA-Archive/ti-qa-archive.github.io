XCode 4.3.2 and Titanium continuous integration build

According to the blog it should be possible to run XCode 4.3 using the master branch of Titanium continuous integration build (see http://developer.appcelerator.com/blog/2012/03/titanium-and-xcode-4-3-revisited.html ).

I installed XCode 4.3.2 using the App Store. At the end of the installation I've choosen the XCode Installer Option to move the XCode 4.2 installation into trash.

Afterwards I updated the Titanium SDK to the current main branch. Therefore I used "Install Titanium SDK from URL" and provided the following link: http://builds.appcelerator.com.s3.amazonaws.com/mobile/master/mobilesdk-2.1.0.v20120402104839-osx.zip

After restarting Titanium the configuration wizard on the dashboard tells me "An iOS SDK is missing. Titanium requires a version between 4.0 and 5.2". Because of the missing SDK I am unable to run the simulator or

Could anybody please help me how to setup Titanium with XCode 4.3.2?

Thank you for your help!

    Titanium Studio 2.0+ and Titanium SDK are both needed to make this work easily. I would recommend updating to the latest version of Titanium Studio from http://preview.appcelerator.com.
