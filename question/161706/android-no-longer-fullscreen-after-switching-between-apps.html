Android no longer fullscreen after switching between apps

I have the following lines inside my code in order to make the Android app fullscreen.

tiapp.xml

<fullscreen>true</fullscreen>
<navbar-hidden>true</navbar-hidden>

example.js

var winOption = {
    height:winHeight,  
    width:winWidth,
    fullscreen: true,
    navBarHidden: true
};

var win = Ti.UI.createWindow(winOption);
win.open();

The codes works fine on both simulator and device unfortunately when 'home' button is pressed, the mobile back to homescreen then come back to the app again, the status-bar is showing up again.

Another scenario is when try to do Facebook login, it launch the Facebook app and the status-bar will shown. Then when switch back to the app and the status-bar is to hidden as I set it earlier.

Not sure why this happen on android, any help/advice is appreciate.
