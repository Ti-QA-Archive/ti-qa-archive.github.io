How to autoplay Embed Youtube video

I embed youtube video and it works. But I don't know how to auto play.
If i put "&autoplay=1" after url, It works in website. But in titanium webview doesn't work. It is weird.
Anybody knows how to make autoplay?
here is my code

var videoUrl =   json.data.items[e.index].content["5"] +&autoplay=1";
            var htmlheader = "<html><head></head><body style='margin:0'><embed id='yt' src='";
            var htmlfooter = "'type='application/x-shockwave-flash' width='100%' height='100%'></embed></body></html>";
            var htmlmash = htmlheader + videoUrl + htmlfooter;
            var webview = Ti.UI.createWebView({
                html:  htmlmash,
                height: '100%',
                width: '100%',
                top:'10%'
                });
            winVideo.add(webview);
