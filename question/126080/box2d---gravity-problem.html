Box2D - gravity problem

Hi,

I'm trying to get a ball moving on the screen due to gravity in box2D.
The ball is displayed and I have no errors, but the ball stays in the same position.
Can someone tell me what I'm doing wrong?
With following code:

var view = Ti.UI.createView();
win.add(view);

var box2d = require("ti.box2d");

var world = box2d.createWorld(view);

world.setGravity(0, 0);

var ball = world.addBody(Ti.UI.createView({
    backgroundImage : "/images/ball.png",
    borderRadius : 15,
    width : 27,
    height : 27
}), {
    radius : 15,
    density : 11.0,
    friction : 0.9,
    restitution : 0.4,
    type : "dynamic"
});

world.start();
world.setGravity(-5,-5);

Thanks for the help,
Jaan
