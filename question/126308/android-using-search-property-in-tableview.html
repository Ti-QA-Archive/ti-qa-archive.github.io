android using search property in tableview

This is my code (from wiki,appcelerator.com):
var searchBar = Titanium.UI.createSearchBar({
hintText:'type to filter rows',

backgroundColor:"white",
data: weatherData,
headerView:headerLabel,
footerView:footerLabel,
search:searchBar,
separatorColor:"white",
top:10,
width:320 });

In this code the error hide when i remove the "search:searchBar"

  • You probably found the answer already, but this may help for somebody else:

    On Android, you cannot add the searchbar to the window. If you do, the app will force close on opening that window.

    Keep in mind that you still have to add the searchbar to the window on iOS!
