Android: Can't create styles.xml

Hello,
I'm creating a classic Titanium app (not alloy) with Appcelerator Studio and the 4.0.3 SDK .
I need to add a custom theme in the styles.xml file.
I create the file in /Resources/android/res/values/, but when I build the project the file doesn't exist in the res/values folder of the build and I'm unable to package the app because the builder can't find the custom theme and gives me error.

  • Accepted Answer

    Try creating a folder at the same level as Resources called 'platform'. (Not in Resources)

    Then, within that add android/res/values/styles.xml
