Accessing require src property in alloy

I'm trying to dynamically change the src file in a require xml tag. So far i'm just trying to access the property to see what value is stored in it.
In index.xml I have:

<ScrollView id="contentContainer" layout="vertical" height="554px">
    <Require id="menu" src="builder_menu"/>    
</ScrollView>

and in index.js I have:

alert($.menu.src);

The alert box is empty when it pops up. Is there even a way to do what i'm trying to do. Thanks!

    Alloy controller has no property src please refer to docs http://docs.appcelerator.com/titanium/3.0/#!/api/Alloy.Controller
    You can also check properties and methods by simple Ti.API.info

    Anyways, here's probably one way I could solve your problem by toggling views or you can remove/add them as well, and do any kind of other stuff you would normally do with Views:

    //index.xml
<Alloy>
    <Window class="container">
        <Label id="label" onClick="doClick">Change View</Label>
         <Require id="test" src="test"/> 
         <Require id="test2" src="test2"/> 
    </Window>
</Alloy>

//index.js
function doClick(e) {  
    Ti.API.info($.test);
    $.test.testView.visible = false;
    $.test2.testView.visible = true;
}

$.test.testView.visible = true;
$.test2.testView.visible = false;
$.index.open();
