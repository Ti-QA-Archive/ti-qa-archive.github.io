  • Accepted Answer

    Titanium 3.0.0.GA and 3.0.2.GA will only work with Node.js 0.8.x. You can download Node.js 0.8.23 here: http://nodejs.org/dist/v0.8.23/.

    Alternative, you can try out the 3.1.0 RC that does support Node.js 0.10.x. This blog post details what you need to do to upgrade to the release candidate: http://developer.appcelerator.com/blog/2013/04/titanium-sdkstudio-3-1-0-release-candidate-now-available.html.
