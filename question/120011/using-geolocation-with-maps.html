Using geolocation with maps

I want to basically take the coordinates of the device using geolocation, and then display that location on a map using the Maps API.

Here is my code : 


var win = Titanium.UI.createWindow();

var lat = 1;
var lon = 1;

Titanium.Geolocation.getCurrentPosition(function(e) {
        if (e.error) {
            Ti.API.info('error: ' + JSON.stringify(e.error) );
            return;
        }

        lon = e.coords.longitude;
        lat = e.coords.latitude;
        alert(JSON.stringify(lon) + ' ' + JSON.stringify(lat));
});


var mapview = Titanium.Map.createView({
    mapType: Titanium.Map.STANDARD_TYPE,
    region: {latitude:lat,
            longitude:lon,
            latitudeDelta:0.01, longitudeDelta:0.01},
    animate:true,
    regionFit:true,
    userLocation:true

});

win.add(mapview);
win.open();

However, when I run the app, it shows the map at (1,1) coordinates. I assumed there must be some problem with the scoping of lat and lon variables, so I tried all sort of things (from using Ti.App, etc) and nothing seems to work out.
Any workaround for this?
