Is there a way to choose the size of a column in a picker?

Title says it all. I'm using a two columns in a Ti.Ui.Picker object, and I want one column to be larger than the other. And by the way, is it possible to have labels that have bold and normal text at the same time?

  • Accepted Answer

    You can change the width of the individual column like this:

    var col1 = Ti.UI.createPickerColumn({width:120});
var col2 = Ti.UI.createPickerColumn({width:180});

    Works in iPhone, not sure about android.
