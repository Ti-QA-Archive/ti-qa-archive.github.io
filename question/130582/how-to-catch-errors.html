How to catch errors

How do I reliably catch errors in Appcelerator?

Try/Catch doesn't work for me as it would in Java or C# - I have a piece of critical code running on startup, and found that it was failing on Android 4.0. Even though the code is surrounded by a try/catch block, my attempt to delete a directory was failing and causing the whole app to crash (instead it should be reporting the error to me quietly in the background!)

So my question, what do other developers do to avoid this problem? I know it's always best to test for an error rather than rely on try/catch, but this isn't always possible since unexpected circumstances will sometimes occur (e.g. a new OS release).

Cheers,

Marc
