Format a date string to add an event to iOS calendar

I am trying to to add an event to the calendar of the iPhone. I am using the moment library to handle dates throughout my app.

Here is my code as is. 

var defCalendar = Ti.Calendar.defaultCalendar;

var newEvent = defCalendar.createEvent({
        title : $.nameLabel.text,
        begin : moment(items.at(0).get('start_time')).format("ddd MMM DD YYYY HH:mm:ss zZZ"),
        end : moment(items.at(0).get('end_time')).format("ddd MMM DD YYYY HH:mm:ss zZZ"),
        availability : Ti.Calendar.AVAILABILITY_FREE,
        allDay : false,
    });

$.buttonView.addEventListener('click', function(e) {
    newEvent.save(Ti.Calendar.SPAN_THISEVENT);
    $.win.close({animated: 'true'});
    $.destroy();
});

With this code I get the error: "message = "-[__NSCFString timeIntervalSinceReferenceDate]: unrecognized selector sent to instance 0xb0a9880";"

How do I format my date strings to properly pass them?
