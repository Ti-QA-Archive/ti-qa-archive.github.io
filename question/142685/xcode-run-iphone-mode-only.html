  • I find that if I want to make any changes to the simulator settings, I need to launch the simulator from outside Titanium, change the settings, then close it. Next time I launch a debug session from within Ti Studio, the new settings will be in place.

    On my system, the simulator is located here:

    /Applications/Xcode.app/Contents/Developer/Platforms/iPhoneSimulator.platform/Developer/Applications/iPhone\ Simulator.app/Contents/MacOS/iPhone\ Simulator'
