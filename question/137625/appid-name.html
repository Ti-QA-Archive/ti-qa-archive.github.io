AppId name

Hi
I have an appId like this xxxx.com.domain.subdomain but when I try to use the provisioning profile of this app Titanium Studio read only com.domain.subdomain and I can't built the distribution app.
How could I solve this problem?

  • Always start your app id name with com. - not xxxx.

    then do : com.domain.APPNAME

    This will be the "Bundle ID suffix"

    Apple/XCode will then create the actual ID #####.com.domain.appname when you preopare to submit it.

    I think that could be why you're having issues.
