  • You can create an alertDialog by following method:

    var alertDialog = Titanium.UI.createAlertDialog({
      title: 'Title',
      message: 'This is the message of the dialog box.',
      buttonNames: ['OK']
});

alertDialog.show();

      This is the one way to create notification alert box.

      var notification = Ti.App.iOS.scheduleLocalNotification({
      alertBody:"Application put in background",
      alertAction:"Re-Launch!",
      userInfo:{"hello":"world"},
      sound:"pop.caf",
      date:new Date(new Date().getTime() + 3000)
      });
