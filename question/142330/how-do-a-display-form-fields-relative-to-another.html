How do a display form fields relative to another?

Instead of specifying a top, left, right, position for each form field. i would like to use the createTextField and display each field relative to another. How can I do that other than specifying a top, left, right attribute for each field?

— asked September 17th 2012 by Weina Scott

  • Accepted Answer

    Set the layout property of the parent view to either horizontal or vertical. If you set it to vertical, then the top property for each field will be the distance between the previous element and not the distance from the top of the page.
