Encrypted Local Storage in Titanium Desktop

Hi all,

I'm in the process of converting an Adobe AIR application over to Ti Desktop and I reckon I've got 95% of the way there. There's a lot of similarities between the AIR and Ti APIs, and having the Ti source code available has definitely helped too.

There is one AIR feature which I use and I haven't yet found an equivalent in Titanium. Its the EncryptedLocalStorage. Basically, the application I work on needs to store some sensitive user data and I'd like to keep it encrypted. It looks like there are 3 different types of local storage available in Titanium - SQL, Titanium.App.Properties and Filesystem storage (thanks for the docs - http://wiki.appcelerator.org/display/guides/Working+with+Local+Data#WorkingwithLocalData-WhatkindofdatastorageshouldIuse%3F)

None of these options seem secure by default though. Am I missing out on something here or is there no easy way to do this without implementing some custom Appcelerator module?

Sean

    Correct, currently you must provide your own encryption. Search the Q&A for encryption and you should find some threads with links to javascript based routines for a few encryption algorithms.

    However, while that will allow you to encrypt values, using those for a database index or ORDER BY would be problematic.

    For that we'd need an an implementation of the SQLite engine which includes encryption support. So until Ti adds that, yes, it would take a custom module.

    For storing data where the key value can be cleartext, I'd suggest a js implementation of a secure algorithm. Some links are in other Q&A's.
