function is not defined

Hello everyone!
inside app/lib I put the file pippo.js
with written

function pippo(){
    alert("test");
}

inside app/controllers/index.js
I wrote

require("pippo");
pippo();

when I see him in the browser I read
ReferenceError: pippo is not defined at /index.html line 875 > Function (line 58)

how to solve?

— asked December 10th 2014 by iacopo guarneri

  • Accepted Answer

    Try this:

    pippo.js

    exports.pippo = function() {
   alert("test");
};

    index.js

    var pippo = require('pippo').pippo;
pippo();
