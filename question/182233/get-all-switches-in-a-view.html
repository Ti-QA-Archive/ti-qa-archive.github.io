  • If the view has only switches then you can directly access its children array which will have all the switches like this: view.children[switch_index].
    Or if the view also has other components then you can loop through the children array and check for the switches like following:

    var switches = [];
for(var i in view.children){
    if(view.children[i].apiName=="Ti.UI.Switch")
        switches.push(view.children[i]);
}

    Hope this helps.
