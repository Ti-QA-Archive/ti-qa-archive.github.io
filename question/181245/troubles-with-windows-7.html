  • Hi Andres,

    It is always preferable if you write more specific details of your findings or issues. Anyway let's start from the beginning.

    I would recommend you to thoroughly go through the Windows part of Studio Set up. You can download the latest Titanium SDK either from Studio or manually.

    The 32-bit version of the JDK is required regardless of whether Titanium is running on a 32-bit or 64-bit Windows system.Titanium Studio offers to install JDK 7 if no JDK is found.
