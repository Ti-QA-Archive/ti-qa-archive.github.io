    Hi Daniel,

    have you had Genymotion Android Emulator installed on your system?

    If you have Genymotion uninstalled, the CLI still try to call genymotion.js.

    So simply delete that file F:\Programme\Titanium\mobilesdk\win32\3.3.0.GA\node_modules\titanium-sdk\lib\emulators\genymotion.js

    This seems to be a known bug, see Jira TIMOB-16074

    If you still use Genymotion, follow the advice from Chris Barber inside the Jira Ticket.

    Greetings,

    Matthias
