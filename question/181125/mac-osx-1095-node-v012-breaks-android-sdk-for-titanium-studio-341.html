Mac OSX 10.9.5, Node v0.12 breaks Android SDK for Titanium Studio 3.4.1

Hello All,

Is there a recommended upgrade path that will let me use the latest Nodejs with my current Titanium Studio with Titanium SDK 3.4.1.GA ?
