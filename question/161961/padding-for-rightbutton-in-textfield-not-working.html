  • Hi

    Strange as this might sound you may have to set left padding on the text field to 0.

    I noticed this a while back when trying to add a simple button and it never worked as I wanted - then (for reasons unknown to myself) I set left padding to 0 as a test - and then it worked.

    Add this to you TSS for the TextField - and hopefully you will have the same success.
