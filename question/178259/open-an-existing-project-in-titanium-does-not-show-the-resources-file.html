  • Accepted Answer

    Titanium projects can be of two types: Classic and Alloy.

    Alloy projects use our MVC-like framework in which your code goes in the project's app folder. The "root" controller is the index controller, made up of the app/views/index.xml, app/styles/index.tss, and app/controllers/index.js set of files. Generated code goes in the Resources folder. In such projects, Studio will hide the Resources folder (since its contents are generated and you shouldn't work with them).

    For Classic projects, no framework is provided, thus there is no app folder. Your code goes into the Resources folder with app.js as the "root" file.

    Does that explanation help you find the source files for the project you imported?
