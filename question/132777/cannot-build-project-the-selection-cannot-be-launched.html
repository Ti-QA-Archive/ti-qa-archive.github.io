  • Oookay, so I FINALLY figured out how to fix that, if that can be useful to anyone. Just remove the project from the workspace and, under the "local filesystem" list in the project explorer, right-click the folder of the project and click "Promote to project".
    And there you go :D

  • I had exact same problem after importing an old project.
    Solution was:

    1. close project
    2. edit .project file manually and make sure it contains this:
    <natures>
     <nature>com.appcelerator.titanium.mobile.nature</nature>
     <nature>com.aptana.projects.webnature</nature>
</natures>
    1. re-open project

    Voila !

    in my case this part was missing:

    <nature>com.appcelerator.titanium.mobile.nature</nature>
