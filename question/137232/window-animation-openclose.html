Window Animation (Open)/Close

Hi,

I've been searching the Q&A for a while now with no really specific answer for what I'm looking for. I want nothing more but an sliding animation for when the window I'm using closes. To be more exact, here's what I'm working with:

I have chosen not to write everything here, for example "win.add(backBtn);" and the tablerow, which I use to open this window. This is just because I get it working with opening the window with this slideLeft animatio but I do NOT get the close animation to work. Instead it just closes straight away.

If more exact code is needed or explained I will provide it. I work with a movielistview.js and a module for a detailview.js.

    var win = Ti.UI.createWindow({
        height: '100%',
        width: 320,
        left: 320

    var slideLeft = Ti.UI.createAnimation();

    slideLeft.duration = 300;

    var slideRight = Ti.UI.createAnimation();
    slideRight.right = -320;
    slideRight.duration = 300;

    var backBtn = Ti.UI.createButton({
        width: 100,
        height: 30,
    });

    backBtn.addEventListener('click', function(){
        //win.animate(slideRight); - all these animations have been tested without any success..
        //win.hide(slideRight);
        win.close(slideRight);
    });

    win.open(slideLeft);

  • I spent some time analyzing this for you, I have figured out the solution.

    Initially I tried seeing if the negative integer was the problem so i put it in a string:

    var slideRight = Ti.UI.createAnimation();
slideRight.right = '-320';
slideRight.duration = 300;

    This still caused an error, so I figured okay there's something about the negative int that it just doesn't like on close or animation. So the resulting fix is:

    var slideRight = Ti.UI.createAnimation();
slideRight.left = 320;
slideRight.duration = 300;

    That has been tested and officially works with code provided above.

  • I've had the same problem, and found a solution:

    var win = Ti.UI.createWindow({
    height: '100%',
    width: 320,
    left: 320
    });

    **var duration = 300;**
var slideLeft = Ti.UI.createAnimation();
slideLeft.left = 0;
slideLeft.duration = **duration**;

var slideRight = Ti.UI.createAnimation();
slideRight.right = -320;
slideRight.duration = 300;

var backBtn = Ti.UI.createButton({
    width: 100,
    height: 30,
});

backBtn.addEventListener('click', function(){
    win.animate(slideRight);
    setTimeout(function(e){
        win.close(slideRight);
    }, **duration**); 
});

win.open(slideLeft);

    It works for me!!
