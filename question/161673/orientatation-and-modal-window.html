Orientatation and modal window

Hi

I am working on an app (iOS only) that is designed to be alway landscape right in orientation.

All of this works fine until I began adding in a modal window to hold my terms of service. When I add this the simulator jumps back to portrait until it is closed. Ideally I just want it to keep in landscape

I have stripped down to a bare bones project to try and resolve but still no dice. Would appreciate any help.

Index.xml

<Alloy>
    <Window class="container">
        <View layout="vertical">
        <Label id="label" onClick="openModal">Terms & Conditions</Label>

        </View>
    </Window>
</Alloy>

Index.js

function openModal(e) {  
   termsDisplay = Alloy.createController('terms').getView();
   termsDisplay.open(); // Note I have also tried termsDisplay.open({modal:true});


};

$.index.open();

terms.xml

<Alloy>
    <Window id="terms" modal="true" class="container" title="Terms" backgroundColor="white">

        <View layout="vertical">
            <Button id="closer" top="20" onClick="closeTerms">Close</Button>
            <ScrollView width="90%">
            <Label>Terms will go here</Label>
            </ScrollView>
        </View>

    </Window>
</Alloy>

thanks very much in advance

    You need

            <orientations device="iphone">
            <orientation>Ti.UI.LANDSCAPE_RIGHT</orientation>
        </orientations>

    in your TiApp.xml, also specify it in your Alloy xml file

    <Alloy>
 <Window orientationModes ="[Ti.UI.LANDSCAPE_RIGHT]" modal="true"> 
</Window>
</Alloy>

    in classic Ti

    var win = Ti.UI.createWindow({
    backgroundColor : 'black',
    orientationModes : [Ti.UI.LANDSCAPE_RIGHT]
});

var b = Ti.UI.createButton({
    title : 'Terms',
    width : 200,
    height : 60
});

var m = Ti.UI.createWindow({
    modal : true,
    backgroundColor : 'green',
    orientationModes : [Ti.UI.LANDSCAPE_RIGHT]
});

b.addEventListener('click', function() {

    m.open();
});
win.add(b);

win.open();

    • I created a new Alloy project and using your code it works fine for me if i add the terms.js controller code

      var args = arguments[0] || {};

function closeTerms() {
    $.terms.close();
}

      And changed the TiApp.xml

      <ios>
        <plist>
            <dict>
                <key>UISupportedInterfaceOrientations~iphone</key>
                <array>
                    <string>UIInterfaceOrientationLandscapeRight</string>
                </array>
                <key>UISupportedInterfaceOrientations~ipad</key>
                <array>
                    <string>UIInterfaceOrientationPortrait</string>
                    <string>UIInterfaceOrientationPortraitUpsideDown</string>
                    <string>UIInterfaceOrientationLandscapeLeft</string>
                    <string>UIInterfaceOrientationLandscapeRight</string>
                </array>
                <key>UIRequiresPersistentWiFi</key>
                <false/>
                <key>UIPrerenderedIcon</key>
                <false/>
                <key>UIStatusBarHidden</key>
                <false/>
                <key>UIStatusBarStyle</key>
                <string>UIStatusBarStyleDefault</string>
            </dict>
        </plist>
    </ios>
