"Hello World" fails to install on emulator

Hi!
I just installed Titanium, went through "getting started" guide and decided to build my first app for Android. I did not change anything in default app.js. However, it fails to install on emulator.

here is what I got:
[ERROR] Failed installing com.calmforce.calmforce: pkg: /data/local/tmp/app.apk
[TRACE] Failure [INSTALL_FAILED_OLDER_SDK]

The Titanium SDK is 1.5.1; Titanium Developer version is 1.2.2; Android SDK versions for emulator I've tried: 1.5, 2.0, 2.2 - same results.

So what INSTALL_FAILED_OLDER_SDK means?

  • Vladimir

    Make sure you follow the Preparing for Android development guide, particularly with regard to the android packages installed. In addition, note that the "SDK Platform Android 1.5" package currently also needs to be installed due to this bug.

    Lastly, ensure that you have SDK "APIs 2.2" selected on the Run Emulator screen.
