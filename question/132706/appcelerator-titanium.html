  • Hello Sugan,

    I'm sure you'll agree, there is not much to work with in your problem description. I strongly suggest you follow the Guidelines in your next post.

    Now, for your problem at hand. You have to make sure that your html file is located in the Resouces directory. Also, don't forget to add your newly created WebView to your window. Otherwise, you won't see it.

    Here is how the code might look:

    var win = Ti.UI.createWindow();

var webview = Titanium.UI.createWebView({
    url: 'chart.htm',
    height:'auto',
    width:'auto'
});

win.add(webview);

win.open();

    One more thing, to reference resources from inside the web page itself (RaphaelJS script for example). You have to prefix the path with app:&#x2F;&#x2F;.
