  • To define what I mean, when I posted the link to the module, I post some code

    var udid = require('ti.udid');
if (udid.oldUDID == '' || udid.oldUDID == undefined) {
    Ti.App.udid = Ti.Platform.createUUID();
} else {
    Ti.App.udid = udid.oldUDID;
}

    Somewhere in the future, Titanium will not support Ti.Platform.id. This workaround provides you a UDID, when "normal" Ti.Platform.id returns NULL
