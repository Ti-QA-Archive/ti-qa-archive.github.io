Tabgroup Question

I have a question for tabs that I need for iPhone but will also need an android solution later on.

Right now my structure is
Tab > opens main window

Action in window > opens sub window (which has a back button at the top that the tab group creates)

An action in this window > opens another sub window but this one is not in the tab group. It shows up onto of the current window.

Right now when this window that is open onto is done doing what it needs to it closes and shows the sub window that opened it.

My question is how do i with code fire the back button or bring it back the the main window.

Thanks

  • fire and event to the parent window in the tabGroup. that event handler will have access to the currentTab and the tabGroup so you can set the currentTab back to the first one.

  • To close a child window, just do:

    Ti.UI.currentWindow.close();

    It will then focus the window behind it.

    • What I generally do is have a class dedicated to navigation. When I open a window, I call a function within this to not just open the new window, but also to store it in an array (of windows).
