Can we apply control's style in jss dynamically?

Hello,

I want to apply style on control dynamically. for example:

###index.tss

".label1":{
    font : {fontWeight : "bold", fontSize : 16},
    color: 'red'
}
"label2":{
    font : {fontWeight : "bold", fontSize : 26},
    color: 'blue'
}

###index.js

var changeClass = function(e){
    alert("Changing...");
    $.status.class="label2";
    alert("Changed");
};
$.index.open();

###index.xml

<Alloy>
    <Window>
        <Label id="status" class="label1"> Demo for Class </Label>
        <Button id="changeClass" onClick="changeClass">Change Class</Button>
    </Window>
</Alloy>

can we set class like this example?
I did not seen any method or property in the document, so is there way to change this type of functionality possible?
