Help with communication between Titanium App and remote website

Hi, I'm a really novice developer and just started using Titanium at my work recently, so I'm having a really hard time figuring something out. Thing is, I need to make an app that at some point the client will send a confirmation that will be read by a remote website and display a message to someone in a computer. Simply put, when the client presses "ok" on the app on his phone, a window on a remote website will pop up. The problem is that I don't have any idea how to do that, can anyone give me any suggestions how I could do this the best way possible? If you could help me, show me a tutorial or just point me in a direction I'd be really greatful because I have no idea what to do. Thanks a lot.

  • I think the best way of doing what you're trying to do would be to use a service called Pusher, this gives you the ability to do real-time communications, so with this method, you'd post a message from the app to Pusher, and you'd have the website listen out for that message, and then open the page you want.

    Here's the link to the titanium SDK's for pusher:
    http://pusher.com/docs/client_libraries#titanium-ios

    You may want to start with their getting started guide for websites to get a feel for how it works:
    http://pusher.com/docs/javascript_quick_start
