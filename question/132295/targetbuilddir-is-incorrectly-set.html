TARGET_BUILD_DIR is incorrectly set

Hi i'm using Xcode 4, Mobile SDK 1.8.1 (and prev ver 1.7.5), OSx 10.6.8, when i trying to make distribution or installation on iPhone i'm getting error

[ERROR] Your TARGET_BUILD_DIR is incorrectly set. Most likely you have configured in Xcode a customized build location. Titanium does not currently support this configuration.
[ERROR] Expected dir /Users/xxxxxxxx/Work/TEZ-Guide/build/iphone/build/Debug-iphoneos, was: /users/xxxxxxx/Work/TEZ-Guide/build/iphone/build/Debug-iphoneos
[ERROR] Error: Traceback (most recent call last):

Please advice how i can solve this issue? Is it Case Sensative issue regarding User forlder, or problem is in Xcode project file?

  • It's seems that you may have set build directory of your project in Xcode other than Project Directory.

    Try this : Open Xcode, Go to Preferences, and change the option for "Place Build Products in" to "Project directory".

    Or

    Try This : Clean & Build

    Any on this should work fine.
