Layout horizontal causing overlapping issue on iPhone 6

I use Ti sdk 3.4.1 GA and Alloy 1.5.1. I need to place two buttons next to each other and hence using layout horizontal to achieve it. But this causes the Rename button to be placed below the Cancel button. However using it without a layout solves the problem. What is it that i am missing?

<View height='50'  borderColor="red" left='20' right='20' layout="horizontal">
    <Button id='btnCancel' height='Ti.UI.FILL'   width='50%'  title="Cancel" color='#4B443C' backgroundColor="#F4F3F0"></Button>
    <Button id='btnRename' height='Ti.UI.FILL'  width='50%'  title="Rename"    color='#fff' backgroundColor="#f68c00"></Button>            
</View>

  • Hi Adnan Baliwala,

    I've checked your code, you didn't define view width property. Please check my test code below and see if it solve your problem.

    Test Code:

    <Alloy>
    <Window class="container">
        <View top= "50" height='Ti.UI.SIZE' width='Ti.UI.FILL'  borderColor="red" left='20' right='20' layout="horizontal">
            <Button id='btnCancel' height='Ti.UI.SIZE'   width='50%'  title="Cancel" color='#4B443C' backgroundColor="#F4F3F0"></Button>
            <Button id='btnRename' height='Ti.UI.SIZE'  width='50%'  title="Rename" color='#fff' backgroundColor="#f68c00"></Button>            
        </View>
    </Window>
</Alloy>

    Hope it helps.
