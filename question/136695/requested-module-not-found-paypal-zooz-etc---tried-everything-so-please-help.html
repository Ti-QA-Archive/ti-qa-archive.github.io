Requested module not found (PayPal, Zooz etc.) - Tried everything so Please HELP!

I am developing app for Android on Mac Lion and trying to integrate zooz module but facing issue

E/TiJSError( 318): (main) [232,15480] —– Titanium Javascript Runtime Error —–

E/TiJSError( 318): (main) [0,15480] - In ti:/module.js:357,8

E/TiJSError( 318): (main) [1,15481] - Message: Uncaught Error: Requested module not found: com.zooz.ti

E/TiJSError( 318): (main) [1,15482] - Source: throw new Error("Requested module not found: " + request);

E/V8Exception( 318): Exception occurred at ti:/module.js:357: Uncaught Error: Requested module not found: com.zooz.ti

Tried everything from project clean, copying the zip module file to project folder, copying module to Application Support/Titanium/modules but nothing works.

Tried with all SDK version 1.8.2, 2.0.1GA, 2.0.1GA2. Same issue for paypal module also.

Any pointers will be appreciated as this is the last step of my app holding me back from releasing to Google market.

  • Tried few things which eventually resolved issue.

    1) clean the project and manually ensured that there is nothing in build folder of project.

    2) remove existing emulator.

    3) create new emulator with sdk as Android 2.2/2.3.3 etc (not google API for Android 2.2/2.3.3).

    — commented May 19th 2012 by Ratty A

  • One more thing I noticed:
    The manifest file inside the module in version 1.1 of the Andoid was pointing to the wrong version
    Fixing the manifest, solved the problem.
