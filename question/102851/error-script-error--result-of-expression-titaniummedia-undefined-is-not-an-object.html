[ERROR] Script Error = Result of expression 'Titanium.Media' [undefined] is not an object.

I´m trying to add a Video as Startup for a iPad app.
But if i implement the Code from the KitchenkSink Play Embedded Movie Section i receive this error:

[ERROR] Script Error = Result of expression 'Titanium.Media' [undefined] is not an object.

If i start the KitchenSink App the Code works fine.
What´s going wrong? Anyone can help me?

  • Changing the SDK was a red herring here. All that was really needed was a full rebuild using the current SDK. Changing the SDK just made that happen, thought then you needed to change it back.

    The issue here is titanium uses an optimization technique to reduce the compile time, and unless you do a full rebuild will not pick up on the need for a module that is not already included in the project. So the first time you reference some new Ti module/class, it is typical to get an undefined reference.

    Most often all that is needed is to visit the Edit tab and hit "Save Changes", even if you don't change anything (like the SDK or app version number). In rare cases, it can be necessary to do these steps.

  • I ran into this same problem only with the photo gallery.

    In another thread, someone mentioned that it worked in 1.5.0 but not 1.5.1. So I changed the SDK to 1.5.0 and rebuilt. Sure enough, it worked. I then changed it back to 1.5.1 and it still works.

    Maybe give that a try.
