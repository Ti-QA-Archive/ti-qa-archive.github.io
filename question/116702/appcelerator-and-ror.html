somebody know where i can find some tutorials which explain how to use Appcelerator with Ruby on Rails?

Or is it not possible to use the Appcelerator Titanium with RoR?

  • It is very possible. Titanium has no knowledge of your server technology. Instead of using <% ERB %> on a browser client, you are using pure Javascript on a mobile client. The best way is to set up your Rails controller to render JSON. Then handle all of the requests from Titanium using XHR.
