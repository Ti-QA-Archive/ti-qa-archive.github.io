    I think this is due to the fact that you may be low on free contigous memory. Googling this error indicates that the IDE has not enough memory. The problem is that when loading a JAVA program, the runtime needs one whole block of memory, not many small ones. I suggest you try and free up memory on your XP machine by closing all unneccessary programs before trying to start the Ti Studio IDE
