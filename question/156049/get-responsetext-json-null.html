Get responseText Json null?

Hi. I'm using Window7
Titanium 3.1.1

I have been created a project Alloy. And i create a function on index.js

getTodoList();          
       function getTodoList () {
              var xhr= Ti.Network.createHTTPClient({
                      onload:function(e){     
                             alert(xhr.responseText);
                      },
                  timeout:10000, 
              });
               xhr.open('GET', 'http://117.6.131.222:8090/POS/phoneservice/message.txt"');  
              xhr.send(); 
       };

That function get Json
But the alert popup null result.

How can i get data use get?

    there's a typo in the url you are calling and i think you should use this.responseText instead of xhr.responseText

    try this:

    getTodoList();          
       function getTodoList () {
              var xhr= Ti.Network.createHTTPClient({
                    onload:function(e){     
                            alert(this.responseText);
                    },
                  timeout:10000, 
              });
               xhr.open('GET', 'http://117.6.131.222:8090/POS/phoneservice/message.txt');  
              xhr.send(); 
       };
