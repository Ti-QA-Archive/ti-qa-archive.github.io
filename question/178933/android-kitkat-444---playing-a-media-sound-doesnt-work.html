Android KitKat (4.4.4) - Playing a Media Sound Doesn't Work

Playing a basic Media sound doesn't work in Android KitKat. I'm using Titanium SDK 3.4.0.GA to run my code. When I check the logs I get the following output.

[WARN] AudioSystem: AudioFlinger server died!
[WARN] IMediaDeathNotifier: media server died
[ERROR] MediaPlayer: error (100, 0)
[ERROR] MediaPlayer: Should have subtitle controller already set
[ERROR] MediaPlayer: Error (100,0)
[INFO] ServiceManager: Waiting for service media.audio_flinger…

This is the simple code I'm using. I'm running the simulator in genymotion.

var sound = Ti.Media.createSound({
        url : 'i-feel.wav',
        preload: true,
        volume: 1
    });

sound.addEventListener('complete', function(e){
    this.reset();
    this.release();
});

button.addEventListener('click', function(e) {

    sound.play();
    Ti.App.fireEvent('app:fromTitanium', { message: 'event fired from Titanium, handled in WebView' });
});

Any help or workarounds on this issue would be greatly appreciated.

    Hello Eric,

    I have tested the below code, i working in Android 4.4.4. Thanks.

    var win1 = Titanium.UI.createWindow({  
    title:'Tab 1',
    backgroundColor:'#fff'

});

var Button = Ti.UI.createButton({
    title : 'Button',
    height : 100,
    width : 150,
    top : 100,
    left : 50
});

win1.add(Button);

var sound = Ti.Media.createSound({
        url : '/bugsbunny1.wav',
        preload: true,
        volume: 2
    });

Button.addEventListener('click', function(e) {
    sound.play();
});

win1.open();
