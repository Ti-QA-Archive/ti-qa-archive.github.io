    two problems, the way you had the url structured required a callback, and secondly you were parsing the feed improperly.

    var cal = JSON.parse(this.responseText);
Titanium.API.info(cal.feed.entry.length)
var i;
for (i=0; i < cal.feed.entry.length; i++){
   var e = cal.feed.entry[i]
   Titanium.API.info("What: " + e.title.$t);
   Titanium.API.info("When: " +e.gd$when[0].startTime + " - "+e.gd$when[0].endTime);
}

    the code in the pastie functions correctly
