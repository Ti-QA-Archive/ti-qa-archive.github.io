I'm trying to create an app that has to monitor for iBeacons. If an iBeacon is in range it has to notify the user with some information. The code to monitor for iBeacons is working already… The problem is that i don't have a clue how to keep the app monitoring, even if the app is closed.

I know iOS 7.1 is capable to do this, but is it possible in a Titanium app? and Android?

  • Declare your background mode in tiapp.xml
    https://github.com/jbeuckm/TiBeacons-Example-App/blob/master/tiapp.xml

    Build your Service
    https://github.com/jbeuckm/TiBeacons-Example-App/blob/master/app/lib/bgService.js

    When you background, start the service
    https://github.com/jbeuckm/TiBeacons-Example-App/blob/master/app/controllers/index.js#L142
