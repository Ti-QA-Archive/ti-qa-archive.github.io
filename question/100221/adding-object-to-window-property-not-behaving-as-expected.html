Adding object to window property not behaving as expected

Hi guys,

I have an object which does a series of tasks (methods). I have assigned this object to a window, example:

win.object = new ObjectName();

However, the object is not behaving correctly as any properties I set within the object are straight away set back to their original values when I go to access them.

For example, the default value of win.object.test is 0, if I assign a value of 99 to test and try to access it straight after setting it, its value is 0.

Any thoughts as to why this is behaving this way?

If I assign the object to a variable all is fine.
