  • Unless its been fixed in 3.0.22, module creation (As of 3.0.0GA) was not being handled in the CLI.

    I run the titanium.py script in 2.1.4GA to create modules until the CLI catches up.

    alias titaniumpy="~/Library/Application\ Support/Titanium/mobilesdk/osx/2.1.4.GA/titanium.py"
