  • You can use the JavaScript Number object to force the type to integer. For example:

    var bla = Number('321');
alert(typeof bla);

    Or with an object:

    var obj = {
    stringValue : '432'
}

var numberValue = Number(obj.stringValue);
alert(typeof numberValue);

    Typeof should return "number" in the case of a number type and "string" in the case of a string type.
