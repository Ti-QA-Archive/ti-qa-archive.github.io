html property of label is not showing properly

HI All,

I want show the json data in the form of table format.
For this I have used below titanium ui elements

  1. html property of label, but it is not showing as table format.

  2. html property of webview, but some times loading problems.

  3. also I tried with TableView of ui element, but it is not satisfies my requirement.

note: I am developing for android

can you help me out, I want show some data in the form of table format like below html in the browser

Here is html code:

var userInfo = Ti.UI.createLabel({

                        left : '2%',

                        html : "<html><head><meta name='viewport' content='width=device-width,height=device-height,minimum-scale=1.0,maximum-scale=1.0'/></head><body><table><tr><td><b>Nickname</b></td><td>srinivas</td></tr><tr><td><b>Birthday</b></td><td>May 5 1989</td></tr><tr><td><b>Email</b></td><td>abcd@gmail.com</td></tr></table></body></html>"
                        color : '#fff',
                        textAlign : Ti.UI.TEXT_ALIGNMENT_LEFT,
                        font : {
                            fontSize : 10
                        }

                    });

I want show like below:

Nickname   srinivas
Birthday   May 5 1989
Email      abcd@gmail.com

  • The label supports simple HTML formatting only (things like <strong> and <em>); it's not going to do layout-related markup.

    You would be better off taking one of two approaches:

    • build separate labels and explicitly lay them out in your view (in your example, "Nickname" would be in one label, "srinivas" would be in another label, etc. for a total of 6 views)
    • use a WebView and use any well-formed HTML you want to use
