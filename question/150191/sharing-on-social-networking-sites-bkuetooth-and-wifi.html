  • Nirag,

    If you decide to use Bluetooth for your sharing project, you can use one or both of the Logical Labs modules in the Appcelerator Marketplace depending on your need for iOS or Android.

    • BT LE iOS module https://marketplace.appcelerator.com/apps/3834?2063287747
    • BT Classic Android module https://marketplace.appcelerator.com/apps/5221?259328472

    Each module has a demonstrable example app in addition to a full set of documentation. The Android module in particular includes a demonstration of sending/receiving a file.

    You should be aware that you won't be able to share between iOS and Android over Bluetooth. This is because of the current limitations on each platform with respect to the Bluetooth protocols. In short, Android's SDK doesn't natively support the "Low Energy" protocol at this time (even though a few Android devices carry the LE hardware) and iOS doesn't permit interaction with the "Classic" protocol with anything other than certified chipsets (to include that you cannot communicate with other apps on iOS devices using the "Classic" protocol)

    You can use both modules however for building apps that communicate with other-than-smartphone hardware that is designed to communicate over the respective protocols (LE for iOS and Classic for Android). And you can use each module to communicate with other apps on the same platform (iOS to iOS using the LE module and Android to Android using the Classic module)

    I hope this helps guide you.

    John
