File sharing in app

Hi,

I have created a csv file. Now I want to delete the app and reinstall it. Then I want to import the data from the csv file. I think this is done by using iTunes file sharing. Can anyone show me how to include file sharing into my app?

I want it so that if user delete the app, I can still get back the data by importing the csv file.

And also I want to know how to access the csv file. Is it using:

var importfile = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory, 'StockTaking.csv');

I'm building for iPhone and using 3.0.0.

    Build your app. Then copy \build\iphone\Info.plist into the project root folder. Open Info.plist, add a new property, select the key "Application supports iTunes file sharing" and set the value to "YES".

    Your application data directory will be available through iTunes File Sharing.
