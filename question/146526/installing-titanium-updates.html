installing titanium updates

I've installed Titanium, following the directions in the install section of the documentation. I go to Help -> Check for Titanium Updates and it offers me Titanium CLI and Alloy. When I attempt to install I get this error: Error encountered during update NPM command not found. Please check if NodeJS is properly installed.

I'm running Titanium on a Mac with OS X 10.8.2

What do I do to make the updates install?

  • I found the answer. Permissions were turned off to be able to write to usr/bin/local. sudo chmod 755 /usr/local/lib and all is better!

  • My issue was not the permission of the folders.
    Rather, i have more than one version of nodejs installed on my system. (several version via macport)
    I fix the issue by

    1. Uninstall all nodejs version from macport (sudo port uninstall nodejs)
    • It will complain if you have more than one version of nodejs installed from macport. So you need to specify the exact version using "sudo port uninstall nodejs @xxxxxxxxxx". You can get the exact version as macport will list out all the existing version and ask you which version to uninstall
    1. Install nodejs using pkg
