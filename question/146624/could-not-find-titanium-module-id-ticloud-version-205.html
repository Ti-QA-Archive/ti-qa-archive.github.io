  • Ti.cloud comes with the new SDKS.

    You failed to mention which SDK you are building against.

    Ti.Cloud versions sometimes change with the SDKs, so be sure that not only is Ti.cloud referenced in your tiapp.xml but that the correct version in the sdk you are using is listed.
