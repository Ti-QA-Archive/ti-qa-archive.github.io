iAd AdView under NavigationWindow

Is it possible with Titanium Alloy to put an AdView under a NavigationWindow? I have been able to put the AdView inside of a Window that is shown within the NavigationWindow but I would rather have it outside of the NavigationWindow so that it persists during Window navigation. Thanks…

    Yes, just define a separate alloy View for your window with a higher zIndex and open it after your main navigation. In 'classic' something like this;

    var w1 = Titanium.UI.createWindow({
    title : 'Window 1',
    backgroundColor : '#fff'
});

var n = Ti.UI.iOS.createNavigationWindow({
    window : w1,
    zIndex : 1,
});

n.open();

var w2 = Ti.UI.createWindow({
    height : 50,
    zIndex : 5,
    bottom : 0,
    backgroundColor : 'green'
});

var ad = Ti.UI.iOS.createAdView();
w2.add(ad);
w2.open();
