How to set UIApplicationExitsOnSuspend ?

Where do we set UIApplicationExitsOnSuspend value? What file? tiapp.xml ? I know this setting goes to info.plist … but where do we set this parameter on Titanium?

Regards
Joao

    You can define any info.plist property within tiapp.xml. Add the following to tiapp.xml (within the ti:app tag):

    <ios>
    <plist>
        <dict>
            <key>UIApplicationExitsOnSuspend</key><true/>
         </dict>
    </plist>
</ios>

    Then do a clean and rebuild and try it out.
