Edit Mode Cancel Button

I am trying to add a button that cancels the edit behavior in the DashbaordView. I did the following so far:

dashboard.addEventListener('edit', function(){
        var cancel = Titanium.UI.createButton({
            title: 'Cancel',
        });

        //cancel.fireEvent(dashboard.stopEditing());
        //win.leftNavButton = cancel;
        //cancel.hide();
    });

Now I need to use the fireEvent method for the cancel button to invoke dashboard.stopEditing(). I am new to JS and Titanium, I tired the following: cancel.fireEvent(dashboard.stopEditing()); and it did not work. How would I do it? Furthermore, after it cancels, I'd like to to hide using cancel.hide();

Clicking Dashboard Item

In other words, how can I extract the ID of the category that is stored in an array for each Dashboard item?

dashboard.addEventListener('click', function(e){

    });

  • Accepted Answer

    dashboard.addEventListener('edit', function(){
        var cancel = Titanium.UI.createButton({
            title: 'Cancel',
        });

        cancel.addEventListener('click',function(e){
            dashboard.stopEditing();
            win.leftNavButton = null;
        });
        win.leftNavButton = cancel;
    });

  • Please ask just one question per entry. That way it's easier for you and others to keep track of which answers apply. That said, I'll take a stab at your first question.

    You should create the cancel button outside of the dashboard's event listener. You can show it on the edit event and hide it on the cancel event. When firing an event, you have to provide a string name of an event to fire not a function. When firing or listening for events, you can do so at the app or component level. Which you choose depends on your needs: app level events are useful when you want multiple components to react to an event or when multiple contexts would put variables out of scope.

    var cancel = Ti.UI.createButton({
   title: 'Cancel'
});
cancel.addEventListener('click', function() {
   Ti.App.fireEvent('dashboard:cancel');
});

Ti.App.addEventListener('dashboard:cancel', function() {
   // do whatever to stop edit mode such as
   dashboard.stopEditing();
   cancel.hide(); // hide the cancel button
   editBtn.show(); // maybe to show the edit button again
});

    Obviously this is typed in off the top of my head and I don't have your app to fully test this. So, the code will need some tweaking to your situation.
