ACS Use TableView Searchbar to search for friends.

Hello.

I am using the searchbar function for TabelViews. I am wondering how i can get the searchbar to search for the Label i am using as the row title.

The searchbar normally search for the title of the row, ex. title: 'Test' and not the custom added label with the title.

Is it possible to search for the label?

Thanks!

  • Figured it out myself:

    Adding a filter to the table view.

    row.filter = rowlabel;

    in the tableview:
    filterAttribute: 'filter',
