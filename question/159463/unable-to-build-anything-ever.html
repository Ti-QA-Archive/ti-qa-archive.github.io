  • Doing a search around the web, this looks like a windows permissions problem so:

    Are you running as Administrator?

    Does the Node process have the correct permissions on the root folder or sub-folders?

    Try it with UAC turned off…

    — commented November 14th 2013 by Larrie Wilson

  • Thank you for your answer. Yes I am the local admin on this machine, I tried right click->run as admin. I set permissions on everything I could think of as "full control" for "everyone", same thing. UAC is off.

    I looked at where it was erroring out somewhere in the error, and it said permission issues on build/mobileweb/images. So I deleted the 'mobileweb' folder out of the build folder and now I got a good build. We'll see how long it lasts, but why would that happen at all? I searched and can't find anyone else with similar issues.

    So thank you, your response helped!
