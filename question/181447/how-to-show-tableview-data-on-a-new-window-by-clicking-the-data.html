    Hi Mayur

    Try with setting custom property to table view row. and onClick of that pass it to detail screen or where you want use.

    e.g

    for(var i=0 ; i < obj.entities.length ; i++) {
    deal=obj.entities[i];
     row = Ti.UI.createTableViewRow({
                        height : 'auto',
                        hasChild : true,
        deal : deal,
        });
    ...
    ...

tableView.addEventListener('click', function(e){
     viewDetails(e.source.deal);
});

}

 viewDetails(e.row.tableData);
