  • Well, it would depend on the reason for the error.
    If there is no net, there might not be net now, that would change what you would do.
    What if there is net, but the server doesn't answer?
    What if there is net, there is a server, but the JSON 'file' doesn't exists?

    Couple of things:
    You should have your 'loader' wrapped in a function. That way you can recall it.
    You should be checking for net before you create your HTTP call.

    Don't assume anything.
