Dynamically Accessing Labels

I'm developing my first Titanium/Alloy app. I need to know how I can reference a label properties other than using the standard "$.label1.text". My code has a dozen labels. Depending on random events (not on interactions with the user), the text or other properties will change. On the web, I'd use a "getElementById" to access the property, but I'm not sure how to do this in Titanium.

For example, to update the text of the label with the id="Player1Score", I'd like to write something using a variable like "labelName" to serve as a proxy, like this:

if (player==1) {labelName = "Player1Score";}

if (player==2) {labelName = "Player2Score";}

$.labelName.text = "Winner!";

I've searched and have only found suggestions to use the e.source.text to reference the element, but in this case, that wont work because the updates don't correspond to user input.

Thanks for any help.
