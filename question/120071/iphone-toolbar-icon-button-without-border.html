iPhone toolbar icon button without border?

I have been asked to create custom icon buttons for an iPhone toolbar, on the order of the system "ACTION" or "INFO_LIGHT" buttons, except with my own icons and specified color. I created PNG icons 28x28 for this purpose.

However, I see no way to create a button without a border here. I have tried creating a button and setting its backgroundImage, which works but shows the inset bevel border behind/around my image.

I tried "image", which works but makes the icon image white.

I tried creating a view instead of a button, setting the view's size and backgroundImage to my icon, but that ALSO shows the border once added to the toolbar item array.

Any ideas?

  • Had the same need and, after a bit of experimentation, found that this works:

    var infoButton = Titanium.UI.createButton({
    image: 'info.png',
    backgroundImage: 'none'
});
win.setRightNavButton(infoButton);

    A size of 28 x 35 (w x h) seems to look best.
