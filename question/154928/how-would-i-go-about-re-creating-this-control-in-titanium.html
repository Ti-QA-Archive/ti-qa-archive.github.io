  • Seems to me, you are trying to retro-fot a desktop user control paradigm onto a mobile device. That will not fly down the run since touching such small arrows with your thumb would seem pretty hard.

    What you could do however is to use mobile paradigms to offer the same functionality to your application. At first glance, your best bet would be to use a Slider for your user to select a numbered value.

    ###Here is a very simple example of using a Slider component

    var slider = Titanium.UI.createSlider({
    top: 50,
    min: 0,
    max: 100,
    width: '100%',
    value: 50
    });

var label = Ti.UI.createLabel({
    text: slider.value,
    width: '100%',
    height: 'auto',
    top: 30,
    left: 0,
    textAlign: Ti.UI.TEXT_ALIGNMENT_CENTER
    });

slider.addEventListener('change', function(e) {
    label.text = String.format("%3.1f", e.value);
});

  • Guys, just wanted to update this thread for those who might come to this thread in the future…this is what I went with in the XML (notice the unicode character \u25B2 & \u25BC for the up & down arrows): 

    <TableViewRow layout="horizontal">
    <TextField  borderColor="black" width="50 height="40" textAlign="center"/>
    <Button title="\u25B2" left="0" height="40" width="40" border/>
    <Button title="\u25BC" left="0" height="40" width="40" />
</TableViewRow>
