"Install or Update iOS SDK" when latest xcode is already installed

Trying to get started. Installed Titanium, and it asks to Configure Native SDKs. I choose iOS. It asks me to "Install or Update iOS SDK", but I already have the latest xcode 4.5.2 on Mountain Lion.

I searched and found this thread:
http://developer.appcelerator.com/question/132298/an-ios-sdk-is-missing

I followed and did:
sudo ln -s /Applications/Xcode.app/Contents/Developer/ /Developer

but Titanium still can't find xcode. I even restarted my machine, no dice.

Please let me know how do I tell Titanium it's installed.

thanks.

    sudo ln -s &#x2F;Applications&#x2F;Xcode.app&#x2F;Contents&#x2F;Developer&#x2F; &#x2F;Developer <- probably some of the worst advice given in these forums.

    Paul's answer fo sudo xcode-select -switch &#x2F;Applications&#x2F;Xcode.app&#x2F;Contents&#x2F;Developer is a better start towards the correct solution.
