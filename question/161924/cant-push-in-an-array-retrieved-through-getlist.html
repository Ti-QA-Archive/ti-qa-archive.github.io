  • Hi

    We have a few pieces of missing information here;

    • getOldData no idea what it does or what it returns - guessing old data but what and how?
    • getList you mention this but not what it is associated with - getList is a common method name, whilst the assumption is due to xml rather than properties or something else - there is no context here.
    • You have several debug lines that should provide useful information - but they have not been provided
    • loadRssFeed as there is no method using this name in Titanium how and what it does is left to guessing work
    • Rainer asked if you simply wanted to merge arrays which you answered yes - but you state your problem is the push line - and there i no indication here of more than one array being processed
    • JSON/XML - I have no idea which you are using as both are referenced so that either could be passed back to the array.
    • setOldData appears to use the variable donnees as the value but nowhere in the code does anything affect the variable donnees. If this is the case then the initial setting of oldData may itself be the cause due to nothing being passed back.

    If you are simply trying to push something into an array then chances are the problem starts with;

    var oldData= getOldData('rssPress');
    • If this is returning back nothing due to the lack of donnes being set in the calling routine - then oldData does not start as an array so it has no method push on the object as it is likely a string by default.
    • If it is correctly returning information back somehow then if that information is an XML node or XML string then again - it does not start as an array so it has no method push on the object as it is likely a string by default.

    However if you are returning back a JSON object from the returned xml feed having been converted to JSON then it should be working.
