Padding on label

Hi.

I have this code in my app

    var  desc = Ti.UI.createLabel({
            text: win.data,
            font:{
                fontSize:'16dp',
            fontWeight:'bold'
        },
        backgroundImage    : 'images/tablebg.png',
        height:'auto',
        left:'20dp',
        right:'20dp',
        top:'110dp',
        editable: false,
        shadowColor: '#222',
        shadowOffset:{x:0,y:1},
        borderColor: 'transparent',
        borderRadius : '5dp',
        color:'#eee',
        scrollable: true,
        touchEnabled:true
        });

My problem is that i would like to have a padding on the backgrond (5px) on all four sides.
Right now my text starts direct in the corner and thats butt ugly….

And the size of it is auto as i dont know how much tex there will be. Is it possible to set a max size and then have scrolling if the size gets bigger?

Thanx.

//R

  • The easiest hack, and the one which is lightest on the memory, is to do something like this :

    Ti.UI.createLabel({ text: &#39; &#39; + win.data + &#39; &#39; });

    The spaces enclosed within ' ' will become your padding area. Cheers.
