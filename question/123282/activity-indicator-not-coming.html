activity indicator not coming

Hi all,
I am using 1.7.1 titanium sdk and 4.2 iphone sdk. In my app.js i have a background image which covers the whole screen . I want to show an activity indicator on the image.I created an activity indicator but its not showing up.
My code is 

var win=Titanium.UI.createWindow({
     backgroundImage:'logo.jpg'
});
win.open();

var win2 = Ti.UI.createWindow({
    url:'win.js',
    backgroundColor: 'white'
});
var actInd = Titanium.UI.createActivityIndicator({
    height:250,
    width:10
});

setTimeout(function() {
    win.add(actInd);
    actInd.show();
    win.close();
     win2.open() }, 3000);~~

Thanks

  • Try the following:

    setTimeout(function(){
    actInd.show()
}, 1);

    Now you'll tell the system to take a little break to focus on that piece of code. That works for me.
