Remote MySQL and JSON

Could someone please shed some light on why this code only pulls the data from the first field and not the second field? The MySQL db has one table with two fields called 'cat_name' and 'colour_name'. Only the cat_name field data is being returned. Not real sharp on JSON but not sure what's happening here. Thanks in advance!

Titanium.API.info('in here!');
var xhr = Titanium.Network.createHTTPClient();
xhr.onload = function(){
    var json = JSON.parse(this.responseText);
    if (!json) { 
        Titanium.API.info('Error - Null return!'); 
        return;
    }
    var json = json.cats;
    var pos;
    for( pos=0; pos < json.length; pos++){
        //Ti.UI.info(json[pos].cat_name, json[pos].colour_name);
        Titanium.API.info(json[pos].cat_name, json[pos].colour_name);
        //Titanium.API.info('DEEP in here!');
    }
};
xhr.open('GET', 'http://www.lullie.com/catalog.php');
xhr.send();

    John

    I think you just need to replace the comma in your Ti.API.info statement with a plus:

    Ti.API.info(json[pos].cat_name + " : " + json[pos].colour_name);

    Cheers
