change the colour of the button on click in android and iOS

Application type:mobile,Titanium sdk:3.5.1,titanium studio:3.4.1,OS:Mac OSX

I have an array of buttons when the user clicks the button the colour of the button should be changed.

I tried like 

arrayList.addEventListener('click',function(e){
e.source.color='#38c';
});

It is changing the colour of the button but when I click the other button,The colour of the previous button remains '#38c'.

I want to change only the user clicked button color other button should have default colour in both android and iOS.

I have tried with touch start and touchend event listener it worked but I need like it should remain in same colour till the user clicks on other button.

  • Solved the issue.

    button[i].addEventListener('click', changeState);

//Changing the state of the clicked button
function changeState(e) {
    e.source.color = '#38c';
    for (var i = 0; i < 5; i++) {
        if (e.source !== button[i]) {
            button[i].color = '#fff';
        }
    }
}
