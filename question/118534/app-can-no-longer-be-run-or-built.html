App can no longer be run or built.

I have been working on an app for a week, using Titanium Studio (aka Aptana).

I have been running it in the iOS simulator. I needed to test it on my device, and Titanium Studio refused to do this, I I built the project that was in "/builds" with Xcode, and installed it that way.

Now, however, I can no longer run the project in the simulator in either Titanium studio OR Titanium Developer. I get this: http://pastie.org/1795087

  • Steve

    Maybe it is a permissions issue. The easiest way to resolve it would be to simply delete all the directories and files below BrownieMaker2&#x2F;build&#x2F;iphone&#x2F;, click the "Save Changes" button on the "Edit Project" screen of Titanium Developer (to touch tiapp.xml and hence force a full rebuild) and then launch/build your app afresh.

    You may want to make a copy of your build directory first, if you are hesitant to just delete it outright.

    It would be great if you would report back with the outcome.
