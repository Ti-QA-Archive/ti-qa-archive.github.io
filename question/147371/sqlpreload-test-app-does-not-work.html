    The github examples on the master branch are for Alloy 1.0.0 which has not yet been released and contains new functionality not in prior releases. I'm assuming you are using some version in the 0.3.x branch. You can get to that branch of the github repo, and its test apps, here: https://github.com/appcelerator/alloy/tree/0_3_X
