Running Titanium Developer.exe for the first time, SDK is not downloaded, TD does not start

I downloaded and installed Titanium Developer on a Windows 7 64 bit machine. The Mobile SDK was downloaded with the installation (1.2.0, can be found in %ProgramData%\Titanium) but neither the (desktop) SDK nor the runtime were. Starting Titanium Developer.exe, install.exe is called, trying to download the missing SDK, but suddenly stops without any notification.

I am in a corporate environment and have correctly set the HTTP_PROXY and HTTPS_PROXY environment variables. Actually, I don't think this could be the source of the problem, since the Mobile SDK was downloaded successfully.

Also, I have already downloaded the SDK (1.1.0) manually, copied it to %ProgramData%\Titanium and changed the sdk and runtime entries in the manifest file to 1.1.0. When I tried to start Titanium afterwards the installer downloaded all the python/js/ti stuff and then crashed telling me that "kroll.dll" was missing from my system.

I'm pretty much out of ideas on how to get Titanium running. Any help is greatly appreciated.
