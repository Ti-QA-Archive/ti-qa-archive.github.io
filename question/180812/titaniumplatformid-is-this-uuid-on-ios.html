Titanium.Platform.id, is this UUID on iOS?

I believe Titanium.Platform.id on Android would return the UDID, however apparently
the UDID is deprecated on iOS and Apple wants us to be using the UUID (which I believe is an ID that is unique to each application and each USER that uses it instead of the device ID which is deprecated).

My question is: Does Titanium.Platform.id return the UUID on iOS in Titanium SDK 3.5.0?
Can I rely on Titanium.Platform.id for uniquely identifying users?

    In iOS, there is no "the" UUID. Per the Titanium docs for Ti.Platform.id:

    On Android, this may be the UDID (unique device ID). For iOS, this is a unique identifier for this install of the application.

    What this means is that Titanium will generate a UUID for you, the first time the app is run on a device. If the user deletes the app & reinstalls, you get a new UUID. If the user updates the app without deleting it, you get the same UUID.
