Little doubt about google navigation and maps

Trying to be self-explanatory again :P

Due to aesthetical requirements, I need to implement a little instance of google maps inside my application (for both Android and iOS) so that's why I decided to use a webview with an URL, so I have a screen capture like this:

http://i44.tinypic.com/34o3les.png

The idea is have this other page instead that one:

http://i39.tinypic.com/2rp8sjo.png

It would be interesting to implement that little navigation system (or just the map, that's the main idea) with a webview.

Here's my code:

var win = Ti.UI.createWindow({
    fullscreen: false,
    navBarHidden: true,
    modal: true,
    orientationModes: [Ti.UI.PORTRAIT],
    backgroundColor: '#FFFFFF',
    exitOnClose: true,
});

var device = Ti.Platform.displayCaps;

var web = Ti.UI.createWebView({
    width: device.platformWidth,
    height: device.platformHeight*0.90,
    top: 0,
    left: 0,
    url: "http://maps.google.com/maps?t=m&saddr=37.25856118113825,-7.016089179289793&daddr=37.26678165753583,-6.935096220518062",
});

win.add(web);
win.open();
