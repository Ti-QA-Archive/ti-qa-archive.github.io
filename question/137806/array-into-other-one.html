  • You need to make your array structure suitable for JSON encoding.
    So it will be;-

    var tabledata = new Array();
var array1 = new Array("value1",
        "value2",
        "value3",
        "value4");

var array2 = new Array("value1",
        "value2",
        "value3",
        "value4");

tabledata[0] = array1;
tabledata[2] = array2;

    *this is untested, but it is the concept of building a multidimensional array.

    If you are making JSON and you already have javascript then you don't need to do very much.

    Take a look at the Tweetanium app for how to build complex JS objects.

    Good luck,
