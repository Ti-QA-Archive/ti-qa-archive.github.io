Tab Navigation - Multiple Windows

Hello,

I have an app with a number of tabs which requires multiple window navigation in some cases. Example:

Tab 1 - the user can select from a number of options that will open up second window. Within this second window there is a back button which brings the user back.

Tab 2 - again the user is presented with a number of options, each options opens up a second window, where more options are present, upon selecting one of these, a third window is open.

At this point I seem to be falling outside the tabbed navigation. The tab bar is no longer visible. Is this possible ?

I've been looking at the Kitchen Sink example, but don't see anything that goes down three levels.

  • You need to tell people your platform and OS when you're asking. If this is Android (and I assume it is based on experience) then you're out of luck. I've tried EVERYTHING to solve this the way you're trying to solve it.

    Best idea: get rid of your tabs, and when you open new windows, just include the tabs inside of them again. But you'll run into memory problems unless you load your windows properly (IE don't just load one on top of the other forever). Rather, manage them in a wrapper object that tracks all open windows, or better yet do what I did and forget making new windows, just make new views and display inside one main window the way that iOS works.
