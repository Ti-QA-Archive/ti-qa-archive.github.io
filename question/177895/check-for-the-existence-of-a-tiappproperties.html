Check for the existence of a Ti.App.Properties

Hello, I set a Ti.App.Properties.setObject when a certain event happens in my app.
Problem is, in a different area of my app I need to check for the existence of this object.

If I check it this way:

if(myObject) //...

when the object doesn't exist I get:

backtrace = "#0 () at :0";
[ERROR] :      line = 44;
[ERROR] :      message = "'null' is not an object (evaluating 'myObject')";
[ERROR] :      name = TypeError;

How do I check for the existence of this object?

    Ti.App.Properties.hasProperty

    if (Ti.App.Properties.hasProperty('myObject')){
    myObject = Ti.App.Properties.getObject('myObject');
}
