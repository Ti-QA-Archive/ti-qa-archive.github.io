    You dont have to worry about the source size, As the same question was in my mind too and I found that the final outcome of your application will be an.apk for android and .app for iOS which will not big as your source (64MB). Though your application can increase in size due to the images or the other resources.

    As you look deeper in the build directory you will find that your .app file will not be too large what is large will be the your .dSYM file which is a debug symbol file. and As we know that in the build process of Titanium project your JS code is precompiled into a set of symbols. which then maps natively.

    The other folder i.e. appname.build folder will be of big size.

    you should neglect these folders as the final thing in which you will be interested in will be your .app or .apk file.

    Now to reduce the size of your .app file or .apk file make sure the images you choose are of proper resolution. as for higher resolution the size will be more..

    hope that helped
