  • Check this out

    var lblHeader = Ti.UI.createLabel({
    text: 'Custom Header',
    left: 10,
    font: {
        fontSize: 28, fontWeight: 'bold'
    }
});
var hv = Ti.UI.createView({
    backgroundColor: 'green',
    top: 0,
    width: '100%',
    height: 40,
    zIndex: 1    
});
hv.add(lblHeader);

var tableview = Titanium.UI.createTableView ({
    data:[
        { title: 'Row 1' },
        { title: 'Row 2' },
        { title: 'Row 3' }
    ],
    top: 40
});

var win = Ti.UI.createWindow();
win.add(tableview);
win.add(hv);
win.open();

    This's just cosmetic trick - adding a view with zIndex=1 at the top of tableview. Make sure you change tableview.top once you change hv.height.
