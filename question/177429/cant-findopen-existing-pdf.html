Can't find/open existing PDF

Hi, i'm trying to read a PDF file, through my app, but I keep get the same error:

Error opening file. Not exist or can't be read (ThinkFree PDF Viewer)

My code is the following:

var localPath = Ti.Filesystem.applicationDataDirectory;

var doc_path = localPath + "files_"+Aktionserver.GLOBAL.lang +"/";

var file_path = Ti.Filesystem.getFile(doc_path,args.docID);

if(file_path.exists()) {

    if(checkExtension(args.docID,"pdf")) {

        if(android) {

            var intent = Ti.Android.createIntent({
                "action":Ti.Android.ACTION_VIEW,
                "type":"application/pdf",
                "data":file_path.nativePath
            });

            try {

                Ti.Android.currentActivity.startActivity(intent);
            }
            catch(e) {

                popup_pdf_error("no_reader","no_reader_available");
            }
        }
    }
    else {

        popup_pdf_error("no_reader","no_valid_file");
    }
}
