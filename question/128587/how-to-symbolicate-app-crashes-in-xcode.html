  • Your problem also appears to be cross-context function calls. Generally, this is when you create a new window with a file URL, and then call a function that was declared in a different window (execution context). Because of how our JS interpreter runs, this creates unsafe conditions sometimes, so tons of warnings are printed as a result of calls like this, and crashes can frequently occur.
