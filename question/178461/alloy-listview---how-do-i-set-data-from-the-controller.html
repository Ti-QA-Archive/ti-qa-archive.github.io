Alloy ListView - how do I set data from the controller?

Hi,

I'm trying out ListView with Alloy. I was able to set up a ListView with predefined items in it from the view xml. But I can't figure out how to bind items to it from a json object in the controller (eventually what I would like to do is have data coming from some datasource and put it in that list).

My xml looks like this:

<Alloy>
    <Window class="container">
        <ListView id="listView" defaultItemTemplate="template">

            <!-- The Templates tag sets the ListView's templates property -->

            <Templates>

                <!-- Define your item templates within the Templates tags or use the
                     Require tag to include a view that only contains an ItemTemplate -->

                <ItemTemplate name="template">
                    <ImageView bindId="pic" id="icon" />
                    <Label bindId="info" id="title" />
                    <Label bindId="es_info" id="subtitle" />
                </ItemTemplate>

            </Templates>

            <ListSection id='mySection'>

            </ListSection>
        </ListView>
    </Window>
</Alloy>

And here's my controller:

var data = [
    {
        properties: {
            info: 'This is text number 1',
            es_info: 'this is the description for the first image',
            pic: '/1.png'
        }
    },
    {
        properties: {
            info: 'This is text number 2',
            es_info: 'this is the description for the second image',
            pic: '/2.png'
        }
    }
];

$.mySection.setItems(data);

$.index.open();

but when I run this I get an empty ListView.

So how can I make this work? what is the correct way?

Thanks

  • The data array needs to contain the bindids as keys. Each of these keys needs to have an object of the properties you want to set.

    var data = [
    {
        info: {
            text: 'This is text number 1'
        },  
        es_info: {
            text: 'this is the description for the first image'
        },
        pic: {
            image: '/1.png'
        }
    },
    {
        info: {
            text: 'This is text number 2'
        },  
        es_info: {
            text: 'this is the description for the first image'
        },
        pic: {
            image: '/2.png'
        }
    },

];
