Android 2.3 webview events crashing

Hi

I've run into a problem developing an app we're developing where the Android 2.3 emulator will crash if you have a webview that calls Ti.App.addEventListener or Ti.App.fireEvent. It appears to relate to this bug:

http://developer.appcelerator.com/question/126058/custom-event-inside-webview-doesnt-work-on-android-23

We haven't yet tested on a device but as the app is heavily webview based I'm concerned we're going to run into problems (considering the market share of 2.3).

My question is can anyone confirm whether this issue is also present on an actual device? From everything I've read I'm still not clear if it only affects the emulator.

We are using 2.0.1.GA2 tested using V8 and Rhino.

Thanks

    Don't take my word for it, but I had the exact same issues on 2.3 emulator, while everything ran without a problem on the actual 2.3 devices.

    I got around it by switching to Android 2.2 on the emulator, while also testing with 2.3 on the actual device - so just make sure you do use an actual device for testing in case it's not related to this same bug.
