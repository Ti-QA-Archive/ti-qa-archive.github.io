  • Couple of things:

    • Blackberry support is currently in beta and not supported.
    • Blackberry SDK tooling within Eclipse is clumsy and even more when added to TiStudio.
    • On OSX, I put it into /sdks/blackeberry and put the binaries into my path
    • Even with the SDK tools loaded into TiStudio, you will see a menu item in your projects, but because your android project is built out into build, the tools don't really work
    • I have used the command line tools to convert an android APK to a bar and submit successfully to Blackberys App World for the playbook.
