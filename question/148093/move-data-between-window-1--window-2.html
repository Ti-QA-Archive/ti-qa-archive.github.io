    hi,

    instead of passing each value , you can also create a json object and pass that object to another window.

    in your example you miss s in win.FirstServe2 in window(2) so i think that shuould be the problem.

    
var dic = {
    TotalServes2:TotalServes,
    FirstServes2:tfsp,
    SecondServ2:SecSerPer,
    FirstServe:FirstServe2,
    tfsp:tfsp2
}

self.containingTab.open(Ti.UI.createWindow({
    title: L('Match Statistics'),
        thisObj : dic
    });
);

    in window (2)

    var dic = win.thisObj;

var TotalServes = dic.TotalServes2;
var FirstServe = dic.FirstServes2;
var SecoundServe = dic.SecondServ2;
var FirstServePer = dic.tfsp;
