  • Hi Victor,

    You would have to create a WebView within a Window to achieve this. By creating an HTML5 app, you really aren't leveraging anything Titanium has to offer. Titanium uses the javascript language for their API, but everything is converted down to native. So a non WebView Titanium app is actually a native app, where as an HTML 5 version, is just one Titanium Window component wrapped around a WebView component.

    Below is an example of a webview within a window.

    
var window = Ti.UI.createWindow();
var webview = Ti.UI.createWebView({url:'http://appcelerator.com'});

window.add(webview);
window.open();

    Paul
