String Expertise Required

Hi,

Is there a Titanium String expert amongst you that help with this small issue.

I have a variable that contains the following string:

227 Entering Passive Mode (71,126,30,27,148,250)

I need some way to read and store the 1st three characters, and the last 2 digits (148 and 250).

Does anyone have any suggestions?

    Hi,

    try this one..

    var d = "227 Entering Passive Mode (71,126,30,27,148,250)";
var f = d.substring(0, 3);
var l = d.substring((d.length - 8), d.length - 1);
alert("First >>" + f + ", Last >>" + l);
