Using Titanium Studio, where is my build after I Distribute to the app store?

I am trying to find my binaries in order to apload them to the application loader but no luck.

I dont get any error on the console, so I guess it must be somewhere!

    This is what worked for me.

    After doing the build in Titanium Studio, start Xcode(my version is 4.0.2) if titanium studio does not start it. Open organizer and you should see the app you built in there. You can validate and submit the app to app to the app store from organizer.
