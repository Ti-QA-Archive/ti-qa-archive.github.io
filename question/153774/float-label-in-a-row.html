Float label in a row.

Hi..

I'm trying to create a chat app. I'm using table to display the chat messages. I'd like to give alternate styles for chat of sender and receiver. I'd like to display the chat of sender in left and the receiver in right side of rows. Is there any method to float the text to the right side rather than giving fixed positions??

My code is


var tableData = [];
 for(var i =0; i<chats.length;i++){
    var msgID = chats[i].user_id;
    var Avatar  = chats[i].avatar;
    var message = chats[i].message;
    if (userID == msgID){
        img = 10;
        textColor = 'blue';
        left = 70; right = 5;
        textAlign = 'left';
    }else{
        img = 430
        textColor = 'black';
        left = 10; right = 70;
        textAlign = 'right';
    }
        var row = Ti.UI.createTableViewRow({
            height: 'auto'

    var imageView = Ti.UI.createImageView({
          image: Avatar,
            left:img, top:5,bottom :5,
            width:50, height:50
      });
      row.add(imageView);

      var label = Ti.UI.createLabel({
          color:textColor,
            text : message,

            right: right,            
            textAlign: textAlign,                
            width:'auto'

  }

    Hi

    When you want something on the left - do not specify right, when you want it on the right - do not specify the left.

    In your code set the left or right to undefined when you do not want it.

    left: undefined
// or
right: undefined

    This will act as if the value was not set, leaving the other horizontal position to take over.
