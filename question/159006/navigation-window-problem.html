  • Hi

    You already have the syntax in the previous file;

    button.addEventListener('click',function(){
    var newWindow = Ti.UI.createWindow({
        url: 'winyeni1.js'
    }); 
    currentTab1.open(newWindow);
});

    You may have confused yourself by not sticking to the same names for your common code;

    // win1.js
var currentWin1=Ti.UI.currentWindow;
var currentTab1=Ti.UI.currentTab;
// winyeni.js
// this does not end with 1 which is what you were trying
var currenTab=Ti.UI.currentTab; 
var currentWin=Ti.UI.currentWindow;

    Try to keep them the same variable names each - it saves you time.
