Tishadow what can I do with --patch

Hi,

I am new at tishadow, I wonder what I can do with the parameter "–patch".

When I run the command tishadow run –patch it takes
no effect.

What can I do with patch and which setup I have to do
to see changes without relaunching the app ?

— asked April 9th 2014 by Markus Bahl

  • --patch push changes without restarting the app. If the file pushed is a commonjs module it will clear it from the cache. So, e.g. if it is a window, you will need to manually close and open that window to see the changes.

    Most of the time it is not very useful, but if you have an app that needs to login to a slow remote service and you are making small changes to a screen, five screens deep, then you might find it handy.
