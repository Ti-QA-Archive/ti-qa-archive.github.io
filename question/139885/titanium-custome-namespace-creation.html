Titanium Custome Namespace Creation

I'm Having problem with creating a custom namespace. I wanted to provide my own APIs on top of Titanium platform. So that I can call my own methods as APIs when creating applications.

My API hierachy is as below
myFamework - > ui ->createBasicWindow

Below is a sample code done by me.

  • Ti.UI.createWindow( )

// I have this code listed in ui.js file which is in Resource folder 

exports.createBasicWindow = function createBasicWindow(title, backgroungColor) {
                var win = Ti.UI.createWindow({
                                title : title,
                                backgroundImage : backgroundImage
                });

                return win;
};

This is how i call myFramework API from app.js. 

myFramework = {};
myFramework.ui = require('ui');
var win = myFramework.ui.createWindow('New Window','white');

My question is how can I amend my code to expose my own methods as it is done in Titanium. Making it more clear, after I type myFramework. it lists down ui but after I type myFramework.ui. it does not list down my method createWindow just like in Titanium.
Can anyone help me in solving this problem? Thanks.

  • Application type: mobile
  • Titanium SDK: 2.1.0 GA
  • Platform & version:Android 2.3.3
  • Device:Android emulator
  • Host Operating System: Windows 7
  • Titanium Studio: Titanium Studio, build: 2.0.2.201205311912
