Strange behavior when setting landscape mode

Hello, I am trying to create a new window onclick event that will open on landscape mode and show a webview on that window.

Everything seems to be OK, except that when I switch to landscape mode the new window appears on a landscape view but with the size of a portrait window. I can even see the content of the window below it (but the content is shown in landscape!… obviously, half of the content)

Maybe somoene can help, so here is my code.

This is how I call the new window.

Titanium.Gesture.addEventListener('orientationchange',function(e)
                                  { if (Titanium.Gesture.isLandscape(e.to))
                                  {var win = Titanium.UI.createWindow({url:'../chart.js',fullscreen:true});
                                  win.open({animated:false});
                                  } else if (Titanium.Gesture.isPortrait(e.to))
                                  { } });

and this is what I do on chart.js

var win = Titanium.UI.currentWindow;

win.orientationModes = [
                        Titanium.UI.LANDSCAPE_LEFT,
                        Titanium.UI.LANDSCAPE_RIGHT
                        ]; 

Titanium.UI.orientation = Titanium.UI.LANDSCAPE_LEFT;

[..]

var webview = Ti.UI.createWebView();
webview.url = 'http://google.com';
webview.scalesPageToFit = false;
win.add(webview);

Any help will be much appreciated!
Thanks
