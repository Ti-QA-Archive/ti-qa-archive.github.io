  • I have just had the same problem. I don't if my solution will help, but here it is.

    In init_normal_dirs() starting at line 30 in env.py I added to:

    self.install_dirs = [
        xyz,  
        'C:\\Documents and Settings\All Users\Application Data\Titanium',
        script_dir + '\..\..\..'
    ]

    This was needed because env.py hard codes: 'C:\Documents and Settings\All Users\Application Data\Titanium' which on my Win XP installation is not correct.

    Hope that helps,
    Neville Franks
