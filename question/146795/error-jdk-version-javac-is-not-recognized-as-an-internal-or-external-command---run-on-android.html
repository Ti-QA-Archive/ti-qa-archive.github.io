[ERROR] JDK version 'javac' is not recognized as an internal or external command - Run on Android

I am new to Titanium and am having trouble running on an Android device. Every time I try to, I get the error:

[ERROR] JDK version 'javac' is not recognized as an internal or external command

I came across this post that is over a year old regarding this topic.

I understand they recommend changing the installed location of javac. However, when I read the Appcelerator installation guide, it says Oracle JDK should be installed at: C:\Program Files (x86)\Java\jdk1.6.0_xx\bin, so I am hesitant to change it.

I am using Titanium Studio 3.0.
**My questions are:

  1. Does this post still apply?
  2. Is there a way to make it work in its current location?
  3. If not, could you provide information on how to change the location?**

Thanks!

  • Just figured it out. I used this link to see how to Configuring Environment Variables on Windows. I was still confused on what to do with the PATH variable until I read this post. As it says in the answer, "Put a semi-colon at the end of the path as it stands and then type the exact path to the "bin" directory of the JDK". That worked for me.
