iOS: Changing the height of a ListView item?

I'm working with a ListView (for performance) and I'm having issues with iOS not responding to height styling. It seems to only use a default height and not expand to show extra content or react to height properties (either a numeric or Ti.UI.SIZE value) defined in the .tss file.

My example code is based off the Alloy ListViewGuide with an extra "line" of text added.

On an Android device the "Factoid" label is displayed however the iOS does not.

index.xml:

<Alloy>
    <Window class="container">
        <ListView id="elementsList" defaultItemTemplate="elementTemplate">
            <Templates>
                <ItemTemplate name="elementTemplate">
                    <Label bindId="symbol" id="symbol" />
                    <View id="atomProperties">
                        <Label bindId="name" id="name" />
                        <View id="secondLine">
                            <Label class="line2 fieldLabel" text="Number: " />
                            <Label class="line2" bindId="number" id="number" />
                            <Label class="line2 fieldLabel" text="Atomic Mass: " />
                            <Label class="line2" bindId="mass" id="mass" />
                        </View>
                        <View id="thirdLine">
                            <Label class="line3" id="tidbit" text="Factoid:" />
                            <Label class="line3" bindId="subtext" id="subtext" />
                        </View>
                    </View>
                </ItemTemplate>
            </Templates>
            <ListSection/>
        </ListView>
    </Window>
</Alloy>

    • Can you elaborate? I added a wrapper view around the items in the template, gave it a background color and a height. This does nothing to change the height of the individual rows in the list on iOS. The border of the empty rows is still the same height and the content within is just pushed up.

      code:

      <ItemTemplate name="elementTemplate">
    <View id="itemWrapper">
        <Label bindId="symbol" id="symbol" />
        <View id="atomProperties">
            <Label bindId="name" id="name" />
            <View id="secondLine">
                <Label class="line2 fieldLabel" text="Number: " />
                <Label class="line2" bindId="number" id="number" />
                <Label class="line2 fieldLabel" text="Atomic Mass: " />
                <Label class="line2" bindId="mass" id="mass" />
            </View>
            <View id="thirdLine">
                <Label class="line3" id="tidbit" text="Factoid:" />
                <Label class="line3" bindId="subtext" id="subtext" />
            </View>
        </View>
    </View>
</ItemTemplate>

      pertinent styles:

      "#itemWrapper": {
    height: '80dp',
    backgroundColor: '#ddd'
}
