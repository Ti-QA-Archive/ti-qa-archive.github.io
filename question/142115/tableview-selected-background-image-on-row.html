Tableview selected background image on row

Hi All

I am trying to get selected background image to work but it just wont work for me no matter how many options i try?

I am trying below

 var row = Ti.UI.createTableViewRow({
                  backgroundColor:'transparent',
                  selectedBackgroundImage: 'areas/shared/images/left_selected.png',
                  backgroundSelectedImage: 'areas/shared/images/left_selected.png',
                  backgroundFocusedImage: 'areas/shared/images/left_selected.png',
                rowid: item.guid,
                height: 60,
                hasCheck: false
              });

This will just make it flicker i want the background image to stay when it is clicked.

i can make the background stay by setting the background image with a click event on the row like below

catTable.addEventListener('click', function(evt){

           evt.row.setBackgroundImage('areas/shared/images/left_selected.png');

    });

But this just sets it for the clicked row and then when another one is clicked it sets it for that as well so i end up with a selected background on loads of rows instead of just the one that was clicked

  • Hi Samuel,

    Try this.

    lastRowSelected = null;

catTable.addEventListener('click', function(evt){
    evt.row.setBackgroundImage('areas/shared/images/left_selected.png');
    if(lastRowSelected !=null){
        catTable.data[0].rows[lastRowSelected].backgroundImage = "yourImage.png";
    }
    lastRowSelected = evt.index;
});

    make lastRowSelected as globle.

  • the challenge is that the event listener is probably getting called multiple times for each row since the eventListener is at the table level.

    I would suggest finding a way to ensure when you are changing the background image that the event is on the row and not some other item in the row. 

    lastRowSelected = null;

catTable.addEventListener('click', function(evt){
    if ( make_sure_i_am_a_row(evt.source) === true ) {
        evt.row.setBackgroundImage('areas/shared/images/left_selected.png');
        if(lastRowSelected !=null){
            catTable.data[0].rows[lastRowSelected].backgroundImage = "yourImage.png";
        }
         lastRowSelected = evt.index;
    }
});
