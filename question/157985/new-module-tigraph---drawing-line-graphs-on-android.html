new module: TiGraph - drawing line graphs on android

Hello,

started another module cause I didn't want to use a webview :) TiGraph. It allows you to draw (simple) line graphs on android. It will generate a blob so you need to put it into an imageview.

var win = Ti.UI.createWindow({
    backgroundColor : "#000"
});

var g = require("miga.tigraph");

var points = [[0, 0], [10, 5], [20, 25], [30, 5]];
var blob = g.drawLines({
    height : 200, maxY : 50, text : true, textColor : "#444444", circles : true, circleRadius : 5, circleColor : "#ff0000", backgroundColor : "#ffffff", color : "#000000", lineWidth : 2, points : points
});

var img = Ti.UI.createImageView({
    image : blob, top : 0
});
win.add(img);

win.open();

You can set a background color (or leave it empty), line color/width, circles or not, text or not.
Just send an array of points into it and it will render them.

Source,preview image, module: https://github.com/m1ga/tigraph

again: feel free to use, share, hack, edit, fork … it :) Perhaps it will be useful for someone else.
