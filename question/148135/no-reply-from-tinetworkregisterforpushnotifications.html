No reply from Ti.Network.registerForPushNotifications()

I know this has been asked a few times, but a working answer never seems to be provided. When using the Ti.Network.registerForPushNotifications() function, I never get a reply. No callback, no Success, no Error. Nothing. Here is my code:

  Ti.Network.registerForPushNotifications({
    types: [
      Ti.Network.NOTIFICATION_TYPE_BADGE,
      Ti.Network.NOTIFICATION_TYPE_ALERT,
      Ti.Network.NOTIFICATION_TYPE_SOUND
    ],
    success: function(e) {
      Ti.API.debug("token: "+e.deviceToken);
    },
    error: function(e){
      Ti.API.debug("error: "+e.error);
    },
    callback: function(e) {
      Ti.API.debug("callback: "+e.data);
    }
  })

I am using Titanium 3.0 running on OSX and testing on a physical iPhone over WiFi. I know that I have read on here that you may need a 3G/4G connection for this to work. But I am sure that could not be right, could it?

As best I can tell I have everything set up properly in the Provisioning Portal image and I have the certificate's installed on my keyring.

One thing I have noticed is that my phone has never asked me if I want to allow this app to receive push notifications, and I guess iPhones are supposed to do that? My app also does not show up under Settings->Notifications. Maybe I missed a step somewhere that tells the phone my app supports push notifications?

I hope I am just missing something here, I really need to get this working ASAP.

    Hi

    You do not appear to have an errors in your code and as it is very basic code designed to set things up I would have been surprised if you had.

    Just in case this is a simple problem with the debug output (it has been known to happen), change your code as follows;

    Ti.Network.registerForPushNotifications({
    types: [
        Ti.Network.NOTIFICATION_TYPE_BADGE,
        Ti.Network.NOTIFICATION_TYPE_ALERT,
        Ti.Network.NOTIFICATION_TYPE_SOUND
    ],
    success: function(e) {
        alert("success: " + JSON.stringify(e));
    },
    error: function (e) {
        alert("error: " + JSON.stringify(e));
    },
    callback: function (e) {
        alert("callback: " + JSON.stringify(e));
    }
});

    Do you get different results now?
