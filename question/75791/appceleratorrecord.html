AppceleratorRecord

I've been creating tons of Appcelerator projects and I'm always needing database objects, for the first couple of projects I copied my database code around and put it into each Model. This was becoming a pain with the number of projects growing. I decided it's time to move this away from the model and make it as abstracted as I can.

AppceleratorRecord was born. I've implemented it into two projects so far and I'm loving it! If you'd like to help/contribute or use please do so.

You can keep track of the project at http://github.com/wibblz/AppceleratorRecord
