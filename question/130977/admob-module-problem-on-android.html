  • For me, I was able to get it working by having the 'testing' property set to 'true', along with setting the 'publisherId' property to my id (you have done that part, right?) upon creating the adview. I was not able to get it to work when attempting to specifically request a test ad (e.g., adMobView.requestTestAd();) - perhaps because I didn't add my device as a test device (as requested per the log). My device is 0S 2.2, using Ti SDKs 1.8.0.1 and 1.7.5.
