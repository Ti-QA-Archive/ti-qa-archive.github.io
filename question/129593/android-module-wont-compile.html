    • For the the imports failure you have to add some more library to your project, open from Eclipse your project properties, go to "Java Build Path" and select "Libraries", then push on "Add External JARs" and point to "Macintosh HD/Library/Application Support/Titanium/mobilesdk/osx/YOUR_TITANIUM_SDK_VERSION/android/" and add this jar: kroll-apt.jar. At this point all should be recognized by Eclipse. About the KrollCallback, just use KrollFunction instead. If you have some more unrecognized lines, try to add some more library from the same folder, but i think you should be ok just adding the kroll-apt one.
