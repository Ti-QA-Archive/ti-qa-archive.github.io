  • I solved my problem as well. I was putting numbers as the row values, so I had to convert it to a string. An example is below (the i would be a number):

    var row1 = Ti.UI.createPickerRow({title: '' + i});
var row2 = Ti.UI.createPickerRow({title: '' + i});
var row3 = Ti.UI.createPickerRow({title: '' + i});
obj.window.column1.addRow(row1);
obj.window.column2.addRow(row2);
obj.window.column3.addRow(row3);
