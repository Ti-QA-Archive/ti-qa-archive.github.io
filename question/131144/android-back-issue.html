  • Hi Sagar

    Normally there is no direct way for implementing this is something which you have to implement yourself.

    Think of a Stack in mind. And make the views in your windows. Now when you one the view just push the current view in the stack.

    Then when you press the back button listen that back event on the window from where the view is open check over there that which is the current view close that view and then get the view from the stack and open it again.

    I have implemented in many of my projects as it normally required when we work with the custom tab.

    I don't have any specific demo made for that but its something which we can achieve using a perfect logic for that.
