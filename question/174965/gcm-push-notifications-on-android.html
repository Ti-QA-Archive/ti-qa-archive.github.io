  • Hi Djamel ZAHAL,

    Don't mix ACS and GCM. ACS is used for iOS platform and GCM used for android platform.

    There is as such no limit of sending push notification from either ACS or GCM.

    Both of the services needs device tokens. this device token you can retrieve from Ti.register for push
    notification callback.

    You need to retrieve the device token and send to your server. And on your server side you need to do coding to call ACS and GCM service.

    Don't go for urban airship if you have your own server and server developer to code for sending push notification. Even Urban airship is not free.

    If you need further help then let me know.
