Hey,

I'm trying to build my Android app and run tests with behave.js on an Linux Mint machine with Jenkins (iOS version of my app builds fine with Jenkins on a Mac).

I used the build script which was posted here: https://github.com/denvers/unit-test-titanium-alloy-project and adapted it to build the Android version of my app.

cd ${WORKSPACE}
titanium clean
titanium build --platform android --log-level debug &
sleep 240
mv /tmp/junit-buildresult.xml ${WORKSPACE}
sleep 3
killall "emulator64-arm"
