Titanium question about pricing and deployment

Hi,

I want to use Titanium but don't want to be bound to it's ACS and Analytics.
Is it possible to still use titanium but use my own database and servers for storage?

Also I was wondering how I can open a project in xCode?

Thanks in advance!

    You don't have to use ACS or Analytics if you don't want to. None of this is required, its there as a value add.

    Once you compile a project, there is an xcode project that gets generated and added to your project. You can open that file in xcode.

  • Yes, you're not bound the ACS and the Analytics. You can opt to not use either, or both.

    When you build a Titanium project and launch it on the device or simulator, it builds up the XCode project in a folder called /build/iphone. Just open the xproj project file in XCode and off you go.
