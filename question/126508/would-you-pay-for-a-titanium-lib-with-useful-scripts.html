Would you pay for a Titanium Lib with useful scripts?

A collegue and I are thinking about putting together a library of useful functions for Titanium Appcelerator and eventually sell it in the markeptplace.

We are both freelance developers and need to make a living, and since the code in question is purely Javascript there is no way for us to protect it.

If we would be to offer a lib which allows for colored badges on any view for instance, or additional hashing and crypto-methods, simplified database access, as well as debugging and profiling methods, would you be willing to pay for this? If so, how much would you be willing to pay? 5$ or even 10$?

The lib we have in mind is supposed to address common quirks and issues found when working with Titanium and could thus be a potential timesaver.

Please let us know what you think, even if you consider this to be a bad idea.

Best regards,
multimedial.de

    • Ricardo – I assume this 400MB db is downloaded once the app is installed, right? Although the mobile android SDK does not have the "file" property of the httpclient that exists in iOS, there is a work around. See this technique.
