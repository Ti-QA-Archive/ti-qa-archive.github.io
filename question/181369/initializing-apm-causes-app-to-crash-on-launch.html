Initializing APM causes app to crash on launch

I've been able to narrow the cause of my app in iOS crashing to the init function for APM in alloy.js:

Alloy.Globals.apm = require("com.appcelerator.apm");
Alloy.Globals.apm.init();

I have the apm key set up correctly in tiapp.xml:

<property name="com-appcelerator-apm-id" type="string">MY_KEY_EXISTS_HERE</property>

And the only indication that something went wrong in the Titanium logs is a debug statement:

[DEBUG] :  Enabling Crittercism v4.3.7...
[DEBUG] :  App ID required, Crittercism will not be enabled

Disabling APM by removing the init line in alloy.js fixes the issue, but also means that APM is not enabled. What is going on here, and how can I resolve this issue?
