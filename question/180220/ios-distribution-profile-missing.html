iOS Distribution profile missing

All,

I'm trying to prepare my app for internal testing but for some reason none of my distro profiles show up in the list of selectable profiles when building. If I click the 'Manage Settings' link in the run configurations dialog, I see all my profiles but only the development ones show up. Additionally, I see 2 different developer certificates, but only one appears in the run configuration dialog.

Anyone have any thoughts as to what I've done wrong? I am running the latest version of all applications and using the 3.4.1 SDK.

Thanks,

    Are you trying to build an adhoc distribution, or just run in development mode on an attached device? If it's the former, then you will need a distribution certificate, rather than a development certificate. If you have a development certificate selected then the ui may be filtering out distribution profiles?
