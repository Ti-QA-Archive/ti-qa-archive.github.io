Animate view height

Hi all.

Is it possible to animate a view from height 0 to height Ti.UI.SIZE??

someView.animate({ height:Ti.UI.SIZE, duration:1000 });

When i try this, the window just is shown with any animation.

    Hi

    Ti.UI.SIZE is not a height it is a constant to be used during construction of the view.

    Try passing a numeric value for height instead. You can determine the height of views using rect and size once they have finished being displayed.

    • Also animations should be more like this;

      var ani = Ti.UI.createAnimation({
    curve: Ti.UI.ANIMATION_CURVE_EASE_OUT,
    duration: 250,
    left: 0
});
center.animate(ani);
