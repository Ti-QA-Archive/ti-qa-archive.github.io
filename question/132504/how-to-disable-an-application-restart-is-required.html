    • so, you download the source for mobilesdk version 1.7.5, you change this file: android/titanium/src/org/appcelerator/titanium/TiLaunchActivity.java

      the function you are looking for is: protected void alertMissingLauncher()
      the code below creates the message you want to disappear, so you remove either the whole code from that function, or just the part that calls up the popup:

      noLaunchCategoryAlert = new AlertDialog.Builder(this)
           .setTitle(title)
           .setMessage(message)
           .setPositiveButton(buttonText, restartListener)
           .setCancelable(false).create();
      noLaunchCategoryAlert.show();

      Then compile the sdk and you are done.
