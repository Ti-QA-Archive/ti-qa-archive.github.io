  • Hi Alexander Rieger ,

    Thanks for your concern .

    Please follow the steps for Module integration from github :

    1. Download ti.admob-android-2.1.5.zip and ti.admob-iphone-1.9.0.zip from github.

    2. Unzip this .

    3. Copy ti.admob folder following path
      ~~~
      In OSX /Library/Application Support/Titanium/modules/android or iPhone
      In Windows C:\ProgramData\TitaniumSDK\modules\ android or iPhone 

    
4. Add the module  in your application's tiapp.xml  file.
5. Require module in js file

    Example : var Admob = require('ti.admob');
    ~~~
