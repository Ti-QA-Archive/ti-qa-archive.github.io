    Every (well, almost every :D ) Titanium component has a "children" property which contains an array of all components added to it.

    So …

    var button = Ti.UI.createButton();

var view = Ti.UI.createView();

view.add(button);

Ti.API.debug(view.children[0] === button); //should be true
