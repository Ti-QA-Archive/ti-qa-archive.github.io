Adding space between inline elements

I've only just begun learning Titanium and I'm working on an Alloy project. I have the index set up the way I want it except for one part; I have a "remember me" label and switch inside a view with a horizontal layout, like this:

<Alloy>
    <Window class="container">
        <ImageView id="logo" />
        <View id="loginForm">
            <TextField id="usernameField" hintText="Username" />
            <TextField id="passwordField" hintText="Password" passwordMask="true" />
            <View id="rememberMe">
                <Label id="rememberLabel">Remember me next time?</Label>
                <Switch id="rememberSwitch" value="false" onChange="outputState" />
            </View>
            <Button id="loginButton" title="Log me in" onClick="logIn"></Button>
        </View>
    </Window>
</Alloy>

'#rememberMe': {
    height: Ti.UI.SIZE,
    layout: 'horizontal',
    width: Ti.UI.SIZE,
    top: 120
}

This results in the label and switch aligned in the centre stacked next to each other, but there is no space between them. I tried simply setting the "right" attribute of the switch, but that had no effect. Is there any way I can effectively add a margin/spacer/buffer between the two elements?

I'm targeting the app initially toward iOS, but I'm assuming that the layout shouldn't be platform specific.
