if (Ti.Platform.osname == "android") does not work in app.js

Hi,
I'm using Titanium Studio build: 2.1.2.201208301612 and this code:

if (Ti.Platform.osname == "android")

does not work in app.js!
The same code works fine later on in the code, but not in app.js!
Funny thing, when I do Ti.API.log(Ti.Platform.osname) it prints android.

What the hell??

I've tried changing &quot; to &#39; as well as using === instead of ==… nothing!
The only way I was able to run it on my device was by commenting out the if statement and whole else block.

Like I've said. The same statement works fine in other places all around my code.

Any thoughts??

    Hi Krystian. It isn't a fault with Titanium but with your javascript code.

    Here is a test to illustrate.

    if (true) { 
    function hi(e) { 
        console.log("hi")
    } 
} else {
    function hi() {
        console.log("bye")
    }
}
hi();

    Now what do you think hi() should execute? The real answer is the second statement.

    Your function declarations are executed irrespective of the if statement. The last definition of checkCode will be the one that holds (like in your example).

    If you rewrite the above example, separating the declaration and definition you'd get what you want. In your case you should do this:

    var checkCode;
if (Ti.Platform.osname == "android") {
    Ti.API.log("Android")
        checkCode = function(e) {
            Ti.API.debug("Android check code")
        }

        Ti.Android.currentActivity.addEventListener('resume', checkCode)

    } else {
        Ti.API.log("iOS")
        checkCode = function() {
            Ti.API.log("iOS check code")
        }

        Ti.App.addEventListener('resumed', checkCode)
    }
