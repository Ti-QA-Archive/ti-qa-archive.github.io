Displaying a modal window over a TabGroup

I tried several things to get a modal window to display over a TabGroup but cannot get it to work. Why is that? It simply does nothing. If I simply remove modal:true, it shows fine, but without a NavBar (which I want).

My code is a bit too scattered to show, but basically I first create a TabGroup, one of which contains a window that needs to open other windows. I want some of these windows to be modal, so I can rely on the default NavBar (otherwise I guess I have to make a custom NavBar, which is annoying).

It seems to be a platform limitation?

  • Richard

    Paul is correct that you can only have one tabGroup on Android, and it is best if it is in the root context.

    The KitchenSink uses the url property to create heavyweight windows, which I think is the behavior that you are seeking. However, this approach is very resource-intensive for normal apps, so I would recommend you make modal windows using the fullscreen:false property. Let me know if this works for you.

    Cheers
