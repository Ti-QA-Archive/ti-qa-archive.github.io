  • Accepted Answer

    Don't change the plugin version in the tiapp.xml. That is only the version of the alloy plugin, not alloy itself. I have no idea what havoc that might wreak on your app, but fix that first. If you are getting that particular error, you don't actually have alloy 1.2.2 installed. You should probably try reinstalling it clean to make sure you have it.

    sudo npm uninstall -g alloy
sudo npm install -g alloy
