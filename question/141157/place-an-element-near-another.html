Place an element near another

Hey,

i am trying to place a label with a distance of 5 px to another label…

I'm trying something like this:
left:(otherpanel.left + otherpanel.width)

Coming from other languages, this should work. But there seems to be any small thing i am missing …. Any idea how to achieve this?

  • wrap the labels in a view with a horizontal layout. roughly:

    var view = Ti.UI.createView({
    width: Ti.UI.FILL,
    height: '40dp',
    layout: 'horizontal'
});

var label1 = Ti.UI.createLabel({
    left: 0,
    width: '100dp',
    height: '40dp',
    text: 'Label1'
});

var label2 = Ti.UI.createLabel({
    left: '5dp',
    width: '100dp',
    height: '40dp',
    text: 'Label2'
});

view.add(label1);
view.add(label2);
