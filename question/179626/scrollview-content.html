ScrollView Content

Greetings

I am trying to use a ScrollView; with a label inside of it. The problem is that the label has a huge text.
so, I use the:

ScrollView.contentHeight ='auto' ;

And it works just fine on the emulator, but on real device it does not scroll.

Anyone has any method to make the scrollview scrollable?

Thanks

    Hi Hund,

    There is no need to assign content height property to scrollview it will automatically scroll according to its content
    e.g

    var lbl = Ti.UI.createLabel({
    top : 10,
    left : 10,
    right : 10,
    height : Ti.UI.SIZE,
    text : "YOUR TEXT"
});

var scrollView = Ti.UI.createScrollView({
    height : "100%",
    width : "100%",
});
scrollView.add(lbl);

    its working fine for me.
    if you have any doubt then let me know. :)

    Thanks
