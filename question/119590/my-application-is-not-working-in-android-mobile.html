  • I would try breaking it down and simplifying your entry code until you figure it out. It starts out by executing the code in your app.js so if it's erroring before it shows the first window then you know there is something wrong in app.js. If it shows the first window and then errors before it finishes rendering the controls then you know the issue is probably in the first window's code. You can also add in alert("reached point A!") lines into the code so you can see how far it executes and what it trips on. Once you find what is causing the issue then you can post it in a new thread if it's not obvious what's wrong.
