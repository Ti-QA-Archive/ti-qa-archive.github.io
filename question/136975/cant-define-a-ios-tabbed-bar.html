Can't define a iOS Tabbed Bar?

I pulled this directly from Kitchen Sink. It's also on the API docs:

var tb1 = Titanium.UI.iOS.createTabbedBar({
    labels:['One', 'Two', 'Three'],
    backgroundColor:'#336699',
    top:50,
    style:Titanium.UI.iPhone.SystemButtonStyle.BAR,
    height:25,
    width:200,
    index:2
});

It causes my app to fail immediately with
[ERROR] Script Error = &#39;undefined&#39; is not a function (evaluating &#39;Titanium.UI.iOS.createTabbedBar&#39;) at simplecalc.js (line 33).
What's the deal?

SDK 2.0.1, iOS 5.1

    • The reason is that only when a full rebuild (forced by cleaning the project) is requires, does Titanium reparse all the included javascript to se if it need to include a specific native code component (in this case the proxy and implementation of Titanium.UI.iOS.createTabbedBar). If a previously unused TI component is added and the whole project is not reparsed the build process fails to link in the addition component
