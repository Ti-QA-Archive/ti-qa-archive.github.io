  • Just how many threads are you going to have going discussing FastDev and android? Android emulator is notoriously slow (the new emulator release helps, a little) and the only fast way to develop for android is to use an actual device.

    Have you tried issuing a $ titanium fastdev restart-app from the command line instead of running from inside TiStudio?

    FastDev Reference for Android
