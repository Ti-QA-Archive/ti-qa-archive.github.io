  • Hi Joost,

    Its been one year since you posted. No idea you could solve it or not. Still posting a working solution, applicable for both ios and android. 

    var customView = function(Text) {
    var view = Ti.UI.createView({
        backgroundColor: '#222',
        height: 50
    });
    var text = Ti.UI.createLabel({
        text: Text,
        left: 40,
        color: 'white'
    });
    view.add(text);
    return view;
};

    Add the above function to you existing code, to create a custom view with custom label, color, size etc.

    Then while creating ListSection use this to custom your section header as you wish. 

    var section1 = Ti.UI.createListSection({
    headerView: customView('Section 1 Header'), 
});

    Your problem will be solved.

    Hope this helps.
