  • I was bit by this too. npm get shows that something has edited &#x2F;usr&#x2F;local&#x2F;share&#x2F;npm&#x2F;etc&#x2F;npmrc, overriding the prefix setting. In my case, the only thing in this file was an override for prefix, so I deleted the whole file.
