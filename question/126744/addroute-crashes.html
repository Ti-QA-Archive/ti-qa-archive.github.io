addRoute crashes

Hi,

my little project:

var route = [[53.550048, 9.969138], [53.55001, 9.967571]];
// there are 200 points ……
var len = route.length;
for (var i =0; i<len; i++) {
    points.push({latitude:route[i][0],longitude:route[i][1]});
}
var myroute = ({
    points: points,
     color: 'red',
     width : 3
 });
 // following line kills the app:    
 mapView.addRoute(myroute);

To many points or whats going wrong?
