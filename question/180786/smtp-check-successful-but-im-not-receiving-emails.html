SMTP check successful but I'm not receiving emails

Hello,

I've setup email templates and saved my SMTP settings. I receive a "Test email from Appcelerator Cloud Services" saying the SMTP check was successful. However, when I send an email in the app using Cloud.send(), I am not receiving the emails.

If I use my personal Gmail account as the SMTP server I DO receive the emails so I don't think it is an issue with my app code, but I do not want to use Gmail as the SMTP server.

Can anyone give me any tips on what I can check? Thanks.

  • The reason was because I didn't have the 'Set default sender' to true. Once I set that to true and entered a default sender I was able to receive emails.
