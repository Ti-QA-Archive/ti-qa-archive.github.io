Image Identifier

  • Titanium.Media.showCamera({
            success : function(event) {
                var image = event.media;
                var filename = Titanium.Filesystem.applicationDataDirectory + "/" + 'camera_photo' + new Date().getTime() + ".png";
                var f = Titanium.Filesystem.getFile(filename);
                f.write(image);
            },
            cancel : function() {
            },
            error : function(error) {
            },
            allowEditing : true,
            showControls : true,
            mediaTypes : Ti.Media.MEDIA_TYPE_PHOTO,
        });

    here that variable filename give us exact location of that image captured by you.
