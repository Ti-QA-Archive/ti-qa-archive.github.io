Effective memory management, how best to do this?

Hi, i have a simple bit of code that queries the available memory and compares it to the device limit (iPhone 6 = 1GB) in this case. But it frequently goes into minus values which should be impossible.

var memLimit = 1024;

setInterval(function(){
    stats.text = 'Available RAM: ' + (Math.round(Ti.Platform.availableMemory) - memLimit) + 'MBs';
},1000);

    Hello Matt,

    You should do 

    stats.text = 'Available RAM: ' + (memLimit - Math.round(Ti.Platform.availableMemory)) + 'MBs';

    Ti.Platform.availableMemory is always less then memLimit. Hope this clears your problem.

    Thanks.
