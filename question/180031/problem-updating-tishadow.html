Problem updating tishadow

I've updated tishadow to last version then regenerate new tishadow app.

When I try to build the app through CLI with the command ti build -p iphone I've got theses errors :

  • [ERROR] Could not find all required Titanium Modules:
  • [ERROR] id: ti.compression version: 1.1.1 platform: iphone deploy-type: development
  • [ERROR] id: yy.logcatcher version: 0.2.0 platform: iphone deploy-type: development
  • [ERROR] id: yy.tidynamicfont version: 0.2.0 platform: iphone deploy-type: development

I've checked into tishadowapp modules folder and I found theses 3 modules. I tried to reinstall them through CLI using gittio install command, but I always had the message already installed. I tried to delete modules folder and reinstall everything through gittio, but after trying to rebuild the tishadow app, I've got the same 3 error messages.

Any idea ?
