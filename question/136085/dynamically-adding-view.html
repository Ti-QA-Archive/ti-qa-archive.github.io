Dynamically adding view

Hi

I am planning to create a splitview in my mobile application. I have created two separate
js file like leftnav.js and rightNav.js .In leftnav.js file i have created view with 4 images and on the click of that images i want to change view in dynamically on the rightNav.js.what is the best way to active the
same

Thanks

  • A couple of options;

    1. Raise a custom event upon click of an item in leftnav.js and listen for it in rightNav.js and add a view accordingly

    2. Export the rightNav.js view/window object and use the .add method of it to add the neccessary view from leftnav.js
