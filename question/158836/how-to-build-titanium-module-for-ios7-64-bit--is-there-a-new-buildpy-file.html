    Hi

    Currently only 32bit apps are being targeted with the preliminary support in Ti SDK 3.1.3. 64bit apps will be supported as part of Ti SDK 3.2 slated for a December launch.

    • Titanium SDK 3.1.3 - Release Notes read iOS Support, 3rd paragraph
