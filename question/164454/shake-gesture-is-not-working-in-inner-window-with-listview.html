    • Ok so the main problem is that Ti.Gesture is app wide, meaning that when it's called it's always available.

      The solution I would suggest, is to add the shake gesture to your parent window then use Ti.App.Properties to create a boolean which can toggle the gesture on and off, like so:

      Ti.Gesture.addEventListener('shake', function(e) {
    if (Ti.App.Properties.getBool('canShake', false)) {
        alert('shaken at ' + e.timestamp);
    }
});

      When you open your modal window, you would simply set the property to true:

      Ti.App.Properties.setBool('canShake', true);

      Then when you close your modal window, set it back to false:

      Ti.App.Properties.setBool('canShake', false);

      Hope this helps!
