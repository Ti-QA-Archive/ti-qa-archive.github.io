    • You have very basic programming questions, I have to say. I think it's not appropriate to ask that kind of questions here (since this is a place where Titanium related questions are asked) and many programming tutorials can be found on internet.

      But, to answer your question … when using string values (and '4000BC' is a string) in SQL, they need to be in quotes:

      var dbrow1 = db1.execute("Select evnt.Events,evnt.Verse from TimeScaleTV as tv, Events as evnt where tv.EventID=evnt.TimeScaleTVID and tv.Year='" + time + "'");

      The problem could arise if "time" variable contains single quote because that would broke SQL statement. Because of that, they need to be escaped before building SQL statement:

      time = time.replace(/'/g, "''");
