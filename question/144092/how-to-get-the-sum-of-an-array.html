How to get the sum of an array?

How can I get the sum of an array?

I've got an unknown amount of items in an array so I can't do

array[0] + array[1] + array[2]

Is there something like sum.array or something?

  • Accepted Answer

    var my_arr = [1,2,3,4,5,6];

var sum = my_arr.reduce(function(a,b) {return a+b;},0);
