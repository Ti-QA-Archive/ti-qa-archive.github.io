Hi,

The code for this section looks like this (I have put comments on stuff I have tried):

…

win.add(picker);

picker.addEventListener('change', function(e) {

  //var selectedplant = picker.getSelectedRow(0).title; // gives the value not the row number

  //var selectedplant = e.rowselectedValue[0]; // gives the value not the row number

  var selectedrow = e.row;                                      // does not work

});

Can anyone tell me how do I get the row number?
