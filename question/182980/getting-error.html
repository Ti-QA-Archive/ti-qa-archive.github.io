Getting Error

Hi,

I am using appcelerator studio and running my first app using android emulator. But getting different errors while running

##Error 1
[ERROR] Application Installer abnormal process termination. Process exit value was 1
The current timeout is set to 120000 ms
[ERROR] : Emulator failed to start in a timely manner

##Error 2
[ERROR] Application Installer abnormal process termination. Process exit value was 1
TRACE | titanium exited with exit code 1

##Error 3
[ERROR] Application Installer abnormal process termination. Process exit value was 1

Although apk file is created

    This error is just coz of that your emulator is taking very much time to get started.

    This is a usual problem while testing Android apps & Titanium has nothing to do with this.

    Everyone face this issue while running apps on android emulator, even Android developers too….:)

    So try your app on device or increase the timeout of emulator.
