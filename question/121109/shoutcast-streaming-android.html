Shoutcast Streaming Android

Hi,
it seems that streaming audio on android (2.1 and 2.2) does not work with shoutcast, if you do not got a stream mount point (ex: http://somedomain:9020 - the next one would work => http://somedomain:9020/stream.mp3)
It works on iOS but not on Android!

The Problem is, that I can´t control the shoutcast server

What to do????

  • Shoutcast streaming from Android 2.2 onwards is supported natively and works fine via Titanium if you use ipaddress:port/;livestream.mp3 you should have no problems.

    Below 2.2 the Android OS cannot play shoutcast streams natively without using a proxy on the stream end or a stream proxy class on the device to capture the stream and pass it to the audioplayer.
