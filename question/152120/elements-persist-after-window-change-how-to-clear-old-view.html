  • Accepted Answer

    You need to give your windows a backgroundColor otherwise they will be transparent, which would account for your behavior. Also, there's an issue with your index.js:

    index.js

    function openMenu(e) {
    // what is the goal of this line? You should not be attempting to access or close views with Alloy
    // Alloy.getView().close();

    Alloy.createController('mainMenu').getView().open();
}

$.index.open();

    app.tss

    "Window": {
    backgroundColor: '#fff'
}

    index.xml

    <Alloy>
    <Window>
        <Button title="myButton" onClick="openMenu">
            Login
        </Button>
    </Window>
</Alloy>

    mainMenu.xml

    <Alloy>
    <Window>
        <Label>You are now at the menu</Label>
    </Window>
</Alloy>

    • You're not closing the window so nothing is being removed. If you wanted to close the index controller window when you open the mainMenu one, you'd modify the index.js like this:

      function openMenu(e) {
    Alloy.createController('mainMenu').getView().open();
    $.index.close();
}

$.index.open();
