Path to files in file system from webview

hi,
I'm using the local file system to store HTML and CSS files. In the HTML i want to call the CSS but I can't seem to get it to work.

what i do is i download the files true httpClient and store the data in the filesystem using:

var f = Titanium.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory, file);
f.write(data);

I set the Url of the webview with:

webview.setUrl( Titanium.Filesystem.getApplicationDataDirectory() + 'index.html');

when i do document.URL inside index.html i get : "appdata-private://index.html".

The CSS is stored the same way and in the same (root) dir. When i try to call the css in the HTML it does not seem to get the file and load the CSS. I tried setting the path to:

<link rel="stylesheet" type="text/css" href="style.css">
<link rel="stylesheet" type="text/css" href="/style.css">
<link rel="stylesheet" type="text/css" href="/style.css/">
<link rel="stylesheet" type="text/css" href="./style.css/">
<link rel="stylesheet" type="text/css" href="appdata-private/style.css/">
<link rel="stylesheet" type="text/css" href="app:///android_asset/resources/">
<link rel="stylesheet" type="text/css" href="file:///android_asset/resources/">

nothing seems to work. Is there a way to get the file from the filesystem? I'm developing on Android btw.

  • Take a look at this question: loading local image

    you'll see something like:

    webView.setHtml(html, {baseURL: Ti.Filesystem.getApplicationDataDirectory() + '/myresources'});

    I think this will help you. I.e. set the base href to access files from inside a local html doc.
