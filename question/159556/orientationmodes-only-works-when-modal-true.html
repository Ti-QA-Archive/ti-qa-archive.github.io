orientationModes only works when modal: true

Hi,

I have this code:

var self = Ti.UI.createWindow({
        backgroundColor:'#ffffff',
        orientationModes: [Ti.UI.PORTRAIT]
    });

However it doesn't work. Only if I put modal: true then it will work

var self = Ti.UI.createWindow({
        backgroundColor:'#ffffff',
        modal: true,
        orientationModes: [Ti.UI.PORTRAIT]
    });

However if I set modal: true, then the fullscreen mode does not work in android.
How do I make it fullscreen and lock the orientation ?

Thank You.

  • Hi Michael,

    OrientationModes work only with heavyWeight windows. Making it modal:true or fullscreen:true force the window to be heavyweight.
    So try something like :

    var self = Ti.UI.createWindow({
        backgroundColor:'#ffffff',
        fullscreen:true,
        orientationModes: [Ti.UI.PORTRAIT]
    });
