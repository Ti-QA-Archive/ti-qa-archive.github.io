Image / Icon Link

Hello, im fairly new to this coding with JS and i just need some help with an icon image in Android only. I'm making a home splash screen where i will link to other JS files where those files will have tabs. i have all of that set up but for the life of me i cannot figure out how to get an image (in this case is just an icon from the festivus icon pack) onto a single window and have it be a link.

Note: all of the stuff i want to link to is inside the app itself, nothing is external.

Here is what ive been working on already, 

var win1 = Ti.UI.createWindow({
    backgroundColor:'#FFF'
});

var recalls = Titanium.UI.createImageView({
        url:'/android/dark_warn.png',
        width:0,
        height:0,
        top:20,
        align: 'center'
    });


win1.add(recalls);    

var l = Titanium.UI.createLabel({
    text:'Copyright me 2011',
    bottom:2,
    color:'#999',
    height:'auto',
    width:300,
    textAlign:'right'
});
win1.add(l);

win1.open();

any help would be great and please contact me if i was not clear enough.

  • Hello!

    First of all, as this is suggested in the ImageView documentation, the property url is deprecated and image should be used instead.

    Then, I don't know if this is the normal behaviour, but in some of my apps, with the image property of an ImageView, Titanium looks right into the android folder when I specify "/" as the image path. So, you'll probably need something like this:

    var recalls = Ti.UI.createImageView({
    image: '/dark_warn.png',
    width:0,
    height:0,
    top:20,
    align: 'center'
});

    And finally, as suggested by Aaron Saunders, having a width and height equals to 0 will make your image never get displayed.

    Hope that'll help !
