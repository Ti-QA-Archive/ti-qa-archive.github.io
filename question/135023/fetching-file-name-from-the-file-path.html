I want to fetch the file name from the path of the file given.
The file name should be free from its extension.
i.e if i have a file path like /xyz/content/images/abc.png

i want to fetch only abc, how will i do it..

    you can do like this:

    var nameInput = imageUrl[i].replace(/^.*[\\\/]/, ''); // This gives the image name with its extension,
        // here we are removing the extension and just fetching the name
 var name = nameInput.substr(0, nameInput.lastIndexOf('.')) || nameInput;

    in the end your name variable will just hold the name of your file..
