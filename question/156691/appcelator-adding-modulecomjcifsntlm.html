  • Hi,

    just check module name.
    if its com.jcifsntlm then you need to use that name in your tiapp.xml file.

    <modules> <module platform="android" version="android">com.jcifsntlm</module> </modules>

    or else just open tiapp.xml file and there is a Modules section available at right hand side so you can directly add module then you do not need to add <modules> tag. It automatically add that line.
