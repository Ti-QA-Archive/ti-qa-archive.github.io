Destroy/Erase a Window?

Hey there,

i have, of course, several Windows in my application … Now i recognised a bad behavier, so i've started a test…

I've added a setInterval function to one of my windows, like this:


setInterval(function(){
   toDoLater();
}, 1000);

function toDoLater() {
    Ti.API.info( "interval" );
};

After closing the Window, the interval seems to loop further…. Is there any way to really destroy/kill the window, so it stops executing this function or all it's behavier?

Or do i have to implement my own "kill()" function to each window?

Best,
Nico

  • You need to do a clearInterval to stop a repeating setInterval() - It is not bound to any window so closing the window will not stop it repeating

  • Yes. so it could go like this

    var si=setInterval(function(){
   toDoLater();
   clearInterval(si);
}, 1000);
