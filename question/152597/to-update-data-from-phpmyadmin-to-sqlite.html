  • Hello,

    just to be clear, you are trying to have a mysql db in a server synced with a local sqlitedb in any device. Is that right? If so, you need to actually create a mobile web service, within a server in the internet.

    The one way sync (i.e., you sync your local sqlite db with a remote db – this is, at the end the content of the local db is the same that the remote one) is a kind of simple. Doing this bidirectionally (i.e., sync both db ) is more complex.

    In any case, both syncs will be via a webservice. You can start learning about this with this simple tutorial.
