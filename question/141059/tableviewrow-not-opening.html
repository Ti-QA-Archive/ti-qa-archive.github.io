  • This is a duplicate of your last question, one in which I practically wrote the code for you. There is no reason to post the question again. But with the code you posted above, you still have a couple of issues.

    First -
    This line:

    var celebrity = require('Celebrity');

    Should be like this:

    var celebrity = require('celebrity');

    The required celebrity is based on the filename, NOT the name of the function in that file.

    Second -
    This line here:

    if (e.index == 0){

    makes it so that the window will only open if you click on the first row. I'm not sure if that is your intention, but you need to click on that first row to open the window.

    • Ok, more code fixes. In celebrity.js
      change this line:

      var self = Ti.ui.createWindow();

      to:

      var self = Ti.UI.createWindow();

      namespaces are case sensitive.

      Also, in Window2.js it looks like you are opening newWindow twice. One with just .open() and the second time after you create a back button as a modal window. You only need to open the window once.

      Lastly, move the button creation code, to the celebrity module. The idea behind the module is to encapsulate all the code for that window. So if a button belongs to that window, then the code for the button should be in the module.
