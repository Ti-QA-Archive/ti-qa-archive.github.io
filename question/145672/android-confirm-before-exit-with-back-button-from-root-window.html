Android: Confirm before exit with back button from root window

Hi, I want to confirm the user with the alert box with message like ("Are you sure you want to Exit Application?") with confirm and cancel button when pressing back button from root window (currently, app simply exits without any notification to user).
Thank you for your help and advice.

    Hi, just use alert dialogue with two button in there.in android:back event.

    and use win.close(); in the yes button index. use it like:

    var rootwindow= Ti.UI.createWindow({  
  title: 'Root Window',
  backgroundColor: 'white',
  exitOnClose: true
});

var dialog = Ti.UI.createAlertDialog({
    cancel: 1,
    buttonNames: ['YES', 'NO'],
    message: 'Would you like to exit from app?',
    title: 'Confirmation'
  });
  dialog.addEventListener('click', function(e){
    if (e.index === e.source.YES){
      rootwindow.close();
    }
    else {
    Ti.API.info('e.cancel: ' + e.NO);
    Ti.API.info('e.source.cancel: ' + e.source.NO);
    Ti.API.info('e.index: ' + e.index);

    }
  });

rootwindow.addEventListener('android:back',funciton(e){
    dialog.show();
});

    let me know if you have any other problem.
