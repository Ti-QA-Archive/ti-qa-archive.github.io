How would you handle submitting the same app twice (with ads & without)

I'd like to offer a free version of my app with iAds and then offer another one for $.99 without iAds.

How would you deploy this? Is this is what the Appstore toolkit does?

New territory for me here so looking for someone who's done this and took the path of least resistance.

  • I would offer the app for free with the ads, then within the app put a subscription button that sells a subscription trough apple store.

    When the app starts I will check in my server if the user has subscribed and if yes hide the ads.
