    Alternatively, you could just point the web view at the image file itself:

    var w = Ti.UI.createWebView({
    url: 'images/image.png'
});

  • Hello Shektar,

    You can achieve this by doing the following:

    Resources Folder Structure
    -> /html
        -> /index.html
    -> /images
        -> /test.png

// app.js
var win = Ti.UI.createWindow();

var webview = Ti.UI.createWebView({url:'html/index.html'});
win.add(webview);  
win.open();

// Html File:
<html>
<body>
    <img src="../images/test.png" />
</body>
</html>
