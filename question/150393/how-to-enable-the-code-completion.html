  • I think I found the problem, it is caused by some plugins I installed. They confused the Titanium to locate the correct javaScript editor.
    Solution:

    1. Help -> Install New Software -> already Installed -> Uninstall all the other tools have been installed before, and then uninstall the Titanium Studio
    2. use appCleaner to delete all the files of Titanium Studio
    3. delete the workspace folder
    4. reinstall the Titanium
