Updated to TI 3.0 - iOS 6.0 only available SDK?

I updated Titanium last night to 3.0 and then went in today to publish an app update. When I try to publish, the only option I have is for iOS6.0. A week and a half ago, I published an update of the same app and Titanium detected my 4.3 iOS version.

When I run 

titanium info -t ios

on the command line, it returns this 

Xcode
4.5 (build 4G182)
  Install Location                    = /Applications/Xcode.app/Contents/Developer
  iOS SDKs                            = 6.0
  iOS Simulators                      = 4.3, 5.1, 6.0

However in xcode, it lists all the software images above. Can anyone tell me what I need to do to reenable the 4.3 SDK so I can publish to that? I don't want to limit this app to iOS6 users only.

  • Take a look to the Console Log, You'll see the following line: “Building for iOS 6.0; using 4.3 as minimum iOS version”. So your App will be compatible from iOS 4.3 up to 6.0.

    You can also test it running the Configuration and target to 'Simulator Version 4.3', you'll see your App running in the Simulator with iOS 4.3 (or whatever version you choose or have installed).

    I did the 'Books Search App' (Alloy Beta) and targeted to 4.3…. On my iPad 1,1 I have iOS 5.1.1 and worked without a problem.
