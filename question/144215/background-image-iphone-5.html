Background image iPhone 5

Hi everyone, i'm trying to put a background image in a window, i am using a Dashboard.
Well the background image is named:

  • fondo.jpg ( for iPhone 3G -> Screenshot )
  • fondo@2x.jpg ( for iPhone 4 and 4S -> Screenshot )

So far.. everything ok. …

The problem is with:

  • fondo-586h@2x.jpg ( for iPhone 5 Screenshot )

The app still using the fondo@2x.jpg, i know this because the fondo-586h@2x.jpg has different colors.

Now… anyone can tell me how can i fix it ?
Maybe i have to rename fondo-586h@2x.jpg in other way ?

The code to put the background is:

// Imagen de fondo
var imagen_fondo          = "images/fondo.jpg";
// Ventana principal
var win = Titanium.UI.createWindow({
    orientationModes: [ Titanium.UI.PORTRAIT, Titanium.UI.UPSIDE_PORTRAIT    ], 
    backgroundImage: imagen_fondo
});

Thanks in advance….

    Hi, I have the same problem, I tried using the "-586h@2x" suffix but also for me it doesn't work, on iPhone 5 it uses always the background with "@2x" suffix, stretching it. I think that this workaround can work, but I'm not sure that it is the best practice:

    var bg = "images/mybg.jpg";
    if(Ti.Platform.displayCaps.platformHeight > 960) {
        bg = "images/mybg-568h.jpg";
    }
var win = Ti.UI.createWindow({
        backgroundImage: bg
    });
