Erase circles of overlay image(view)

I have an app in mind, but I'm having a problem how to make it.

This will be a little game where the user will try to find something in an image.

Example: I have an image, then I cover all the image with a overlay view (or image), then the user can do erase portions of the otherlay and try to guess what is the image.

Example:
http://oi60.tinypic.com/2j2ycty.jpg

Someone have an idea who to create this holes in the overlay?

  • A few ideas for this:

    1. Instead of one large view covering your image use a grid of square views. Each view would have a solid background that would be replaced with a frame and transparent cut out when clicked. The drawbacks of this would be that your circular cut outs would be in set positions instead of centered at the touch point.

    2. In the ti.paint module, see if there is a way to have the paint view pre-filled, then use a touch to erase. I've never tried this, you may have to hack the module.

    3. See if something similar to 2 can be done with the Ti.Canvas module.
