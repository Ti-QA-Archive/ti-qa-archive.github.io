  • If you meant to ask whether you are able to use the iOS built-in New Person view from Titanium, then no - the only built-in interface currently implemented in Titanium's contacts module is the built-in people picker view that lets the user pick a contact by using the built-in UI.

    However, I'm currently wrapping up a Titanium module that will allow you to use the remaining built-in iOS views from the address book API, including the New Person view controller. Feel free to shoot me a message on twitter (@skledarg) if you're interested in trying it out.

    Alternatively, you will need to build the user interface for adding a contact yourself in Titanium and then use the Titanium.Contacts API to save the new contact.
