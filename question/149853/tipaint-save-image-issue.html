Ti.Paint Save image Issue

Ok, so I know this is a simpleton question, but I have searched and cannot find the answer. I have installed the paint module and created images (I am using it for signature capture so I remarked out most of the features). I am trying to save the resultant image to a file (using a naming convention I need for my app). The problem I have is I do not know the file type to save to. I have tried .png and .bmp and just receive the message: Unable to write to an unrecognized file type. Any help is appreciated. Thanks.

Windows 7

3.0.2GA

Ti.Paint (ti.paint-android-2.0.2)

Android 4.2.1

saveBtn.addEventListener('click', function() { 
    var sigImg = paintView.toImage(); 
    var filename = (Ti.App.currentWorkOrderId + "_signature.bmp");
    var img = Titanium.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory,filename);
    img.write(sigImg);
    Ti.API.info("image " + filename + " loaded at: " + img.nativePath);
    win.close();
});
