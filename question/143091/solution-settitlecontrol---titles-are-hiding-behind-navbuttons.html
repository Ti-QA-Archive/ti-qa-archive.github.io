Solution: setTitleControl - Titles are hiding behind navbuttons

Not a question actually, but a full Q&A. Just trying to help others that may have the same issue right now.

I just installed the 2.1.3.GA version and had some issues with the setTitleControl method on iOS6: Titles were hiding behind left/right navbuttons.

I did declare the window and my custom title this way:

if (!isAndroid) { winFullscreen.setTitleControl(labelTitleFull); };
winFullscreen.open();

And this is what it looked like:

Link to screenshot setTitleContro before window open

I tried to declare the titlecontrol directly to the window instead of using the setTitleControl method and had no success.

Solution:

Eventually I just changed the order to set the title right after the window opening:

winFullscreen.open();
if (!isAndroid) { winFullscreen.setTitleControl(labelTitleFull); };

And it worked like it did before the update :) see the screenshot here:
Link to screenshot setTitleContro after window open

Hope it can help others that have the same issue :)
