    Reid

    Note that this Q&A is not GitHub Gist, which is what is requested in Install Diagnostics for Windows. There are certain problems that the output shown in your question would not help with, due to the way it has been reformated by the system.

    Anyway…

    Your problem is that your JAVA_HOME variable is incorrect. It should be C:\Program Files\Java\jdk1.6.0_23 (without bin). The path to your java executables in your system PATH is correct, however, although it has been duplicated, so you should delete the last occurrence of it.

    Further recommendations:

    The following paths should always appear at the very beginning of your system PATH:

    C:\Windows\system32;
C:\Windows;
C:\Windows\Syste m32\Wbem;

    Hence, move the following to the end:

    C:\Python26\Scripts;
C:\Python26\;

    If you continue to have problems, run the diagnostics commands again, and paste them to gist, using the method explained in the guide. ;)
