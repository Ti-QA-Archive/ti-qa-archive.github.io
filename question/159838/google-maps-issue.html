Google Maps Issue

When adding this to our xml file:

<application>
                <meta-data android:name="com.google.android.maps.v2.API_KEY" android:value="our_key_goes_here"/>
            </application>

The app crashes on start.
If we comment out this line, works fine.

  • We should replace those(name and value) with your google API keys. Look here at documentation how to produce API key http://docs.appcelerator.com/titanium/latest/#!/guide/Google_Maps_v2_for_Android under Obtain and Add a Google API Key section. Before that make sure your device capable of Google Play Services.
    Also, you can't run google maps on Android Emulator. Since, it doesn't have capable of Google play services.

    • I am not aware of this error. Though I can only suggest you this by assuming -

      Looks like you adding this meta data in between activity section. It should be outside the activity section.

       <application android:hardwareAccelerated="true" android:screenOrientation="portrait">
<!-- Replace "PASTE YOUR GOOGLE MAPS API KEY HERE" with the Google API key you obtained -->
<meta-data
    android:name="com.google.android.maps.v2.API_KEY" android:value="my key"/>


<activity
    android:configChanges="keyboardHidden|orientation|screenSize"
    android:label="my-app-hello"
    android:name=".MyAppHelloActivity"
    android:screenOrientation="portrait" android:theme="@style/Theme.Titanium">
    <intent-filter>
        <action android:name="android.intent.action.MAIN"/>
        <category android:name="android.intent.category.LAUNCHER"/>
    </intent-filter>
    <intent-filter>
        <action android:name="android.intent.action.VIEW"/>
        <category android:name="android.intent.category.BROWSABLE"/>
        <category android:name="android.intent.category.DEFAULT"/>
    </intent-filter>
</activity>
</application>
