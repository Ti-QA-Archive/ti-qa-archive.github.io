    • Thanks Everyone for looking into this problem, though this problem i was facing in my early days of module development. Now i have resolved this issue , there are certain things which we might look for this kind of error.

      In my case this error was due to run time linking error with Security Framework and i resolved it by linking security framework for specific version of iOS SDK, i.e. iOS 5.0.

      Syntax for linking with specific version of iOS SDK is generated with Module Template.
