Using a string of binary data as an image?

Is it possible to use a string of binary data as your image source when creating an ImageView?

I'm pretty much going to have a stream of images running as frame in a video, so I don't want the overhead of base64 or creating temp files.

Is it possible to just convert a string to a blob that the ImageView can use?

    The ImageView car accept a blob as a value for the image property.

    Have a look at the BLOB object from the API Documentation. You could simply create a BLOB object from your string and then assigning this newly created BLOB to your ImageView.

    This should get you going.
