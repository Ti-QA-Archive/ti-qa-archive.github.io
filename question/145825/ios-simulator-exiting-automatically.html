iOS Simulator Exiting automatically

Hi,

Whenever i try to execute my iOS project in Simulator, the simulator is getting exited automatically.

That too only in:

  • iOS SDK - 6.0
  • Simulator SDK - 6.0
  • Ti SDK - ALL version including 3.0.0.GA
  • Ti Studio Version -3.0.0+

And following is the error i am getting,

[ERROR] :  An error occurred running the iOS Simulator
[ERROR] :  [DEBUG] Sessiocould not be started: Error Domain=DTiPhoneSimulatorErrorDomain Code=1 "iOS Simulator failed to install the application." UserInfo=0x10014f0e0 {NSLocalizedDescription=iOS Simulator failed to install the application., DTiPhoneSimulatorUnderlyingErrorCodeKey=-1}

More information:

  • The Simulator SDK -6.0 is working fine when i run the same project from xcode (Using the generated Xcode Project in Build)
  • It is working fine when i change Simulator SDK -5.1
  • I tries clean build and everything

I wonder what could be the problem. Please let me know, if u can see the solution for this.

    Manually launch the iOS Simulator, then from the iOS Simulator menu, select "Reset Content and Settings". Rebuild your app and it should install.

    Assuming that does the trick, please build a couple times and see if the error comes back.
