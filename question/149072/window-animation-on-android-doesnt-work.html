Window animation on Android doesn't work

Hello all,

On Android (my Xperia Arc, Android2.3.4), it seems that a window animation like this does not work.

var winObj = Ti.UI.createWindow();
winObj.left = Ti.platform.displayCaps.platformWidth; // trying to slide in from right
var animation = Ti.UI.createAnimation();
animation.left = 0;
animation.duration = 500; // 0.5 sec animation
winObj.open(animation);

On iOS, it works as intended. On Android, the animation does not move and the screen stays blank as a result.

Can anyone advise me to get it work?
Thanks a lot.

  • I still cannot animate windows on Android as I want.

    And I found that it is a device-dependent issue, because the same program running on different Android devices makes different transition animations. For example, my phone pops up a window with a zoom-like effect while another phone makes a slide effect. I could not change that no matter what I do.

    My conclusion: it is impossible to make createAnimation work as intended on Android.
