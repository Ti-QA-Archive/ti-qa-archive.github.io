  • Hi

    Given the very little code you have provided - there is insufficient row information to see what the problem is.

    I would suggest doing the following to narrow down the issue.

    • DO NOT add a tableView to a scrollView, they both scroll - a tableView is a type of scrollView. There MUST be a very good reason for using both and at the very least the tableView should not be scrollable.
    • Remove the backgroundColor from the scrollView.
    • Simplify the tableViewRow, add a title property to the row and ignore all other child elements - is this faster.
    • Delay the actual display until the LOAD event has finished on the window - this will make it look more responsive.
