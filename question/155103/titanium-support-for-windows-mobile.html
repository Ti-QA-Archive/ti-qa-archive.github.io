    Hi

    Windows Mobile is not currently supported as Nick has said, however there is some follow up information.

    October last year there was a blog announcement that said Windows 8 support would be available in the first half of 2013. You probably guessed that that has yet to appear.

    Earlier this year it was announced that Windows Mobile support would be ready for third quarter this year.

    During the last TiConf there was a presentation about Ti.Next which is a whole new way that Titanium will be designed, Windows Mobile is one of the three base supported operating systems as well as Android and iOS. This is likely to be around 1st half of next year.

    So no support right now but it is on the plans and might be soon and then even better later on.
