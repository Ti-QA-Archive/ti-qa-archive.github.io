What's the best way to vertically center a multiparagraph textview?

Titanium Mobile sdk 1.8.2

iphone sdk: 4.3

Hi,

The only way I know is to manually set the top properties but since I am using a dynamic value for the text then this means that some texts do not appear vertically centered. How can i achieve this?

thanks.

    hi Nash,

    you can do it with two way…

    like you can use layout property to verical of your parent control and second thing

    you can dynamically check the textField's height

    something like this

    var txt1 = Ti.UI.createTextField({
    top : 5,
    value : 'bla bla bla',
    width : 300,
    height : 'auto'
});


var txt2 = Ti.UI.createTextField({
    top : txt1.toImage().height + txt1.top + 5,
    value : 'bla bla bla',
    width : 300,
    height : 'auto'
});

var txt3 = Ti.UI.createTextField({
    top : txt2.toImage().height + txt1.top + 5,
    value : 'bla bla bla',
    width : 300,
    height : 'auto'
});

var txt4 = Ti.UI.createTextField({
    top : txt3.toImage().height + txt1.top + 5,
    value : 'bla bla bla',
    width : 300,
    height : 'auto'
});
