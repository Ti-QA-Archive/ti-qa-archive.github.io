Can't include more than one file with Ti.Include()

I'm having the following problem. My statement Ti.Include('filename.js') works correctly. If I have more than one file, it only includes the last file in the Include.

Ex.
Ti.Include('fn1.js', 'fn2.js')

fn2 will can be accessed, but not fn1. If I change the order to:
Ti.Include('fn2.js','fn1.js'), then fn1.js works but it doesn't find 'fn2.js'

Has anything changed??

Thanks

    • Found it - payment.js: This line myapp.ui = {}; will reset myapp.ui namespace, therefore myapp.ui.createApplicationWindow (ui.js) will be undefined.

      Solution: move myapp.ui = {}; to app.js

      // create an object literal to be your app's namespace
var myapp = {};
myapp.ui = {};

// include necessary libraries
Ti.include('windows/ui.js','windows/payment.js');

// instantiate and open the main UI component of our app
myapp.mainWindow = myapp.ui.createApplicationWindow();
myapp.mainWindow.open();
