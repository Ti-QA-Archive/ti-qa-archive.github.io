How to get Image from its native path?

I have store native path of image file to database. Now I need to display that image. For this I am using

imageView.image = Titanium.Filesystem.getfile(rows.fieldByName('path'));

But no effect. Please suggest.

  • If you have stored the nativePath value to the database, you do not need to use the Resourcesdirectory or other directory constants again to access the file. The error code you saw originally contained the reason for the error - there is no need to use the getFile method - just pass rows.fieldByName(path) to the image property.
