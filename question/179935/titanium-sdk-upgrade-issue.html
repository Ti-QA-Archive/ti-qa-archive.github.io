  • Hi BIBHUTI DALEI,

    If Check for titanium updates does not work, try to install it using the command. So open your studio terminal and write

    For Install the latest released SDK.

    sudo titanium sdk install

    For Install the latest released CLI

    sudo npm install titanium -g

    For Install the latest released alloy

    sudo npm install -g alloy

    If you have any concern, then let me know.

    Thanks.
