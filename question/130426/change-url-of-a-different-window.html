Change URL of a Different Window

Hey Guys,

Ok so this question might be a little weird so I'll try to say it as simple as possible (sorry if it comes across confusing).

So, my question is whether or not it is possible to change the URL in a different window? I am mainly asking for iPhone but it applies to Android as well.

For example, if you have taken a look at the Facebook app you can see how they have that window with the menu underneath the window with the content (feeds, user profiles, etc.). Well when you click one of those links in that menu it doesn't just set that button as active, it changes the URL of the content window to point to a different page (ie, from your activity feed to your profile, for example).

The real stumping part is that the header bar never changes it stays right there. It seems almost as though there is a view inside that content window that changes but the window itself doesn't… anyways, just wondering if anyone have any idea how they do that? May be cool to know for the future.
