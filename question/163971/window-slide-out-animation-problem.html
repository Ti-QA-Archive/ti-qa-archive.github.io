Window slide out animation problem

Hey folks!

I've got a weird problem with the animation of windows. First of all: Because of a special customer wish, I cant use tabgroups and have to open each window for itself with an own navigation bar.

I have recorded a video to show the problem: Youtube Video

I set the slide out speed up to 2 seconds to display the animation problem. Normal speed would be 0,3 secs.

Here is my code for the slideout animation and closing the window after completion:

var slideRight = Ti.UI.createAnimation({
    right:Ti.Platform.displayCaps.platformWidth * (-1),
    duration:2000
});
win.animate(slideRight);
setTimeout(function () {
    win.close({
        animated: false
    });
}, slideRight.duration);

Does anyone have an idea, what I can change to animate the window correctly?

For your info: I build the app for iOS and Android with SDK 3.2.2

  • Hi Sebestian,
    I want to tell you that , this functionality you can do with the navigationWindow. Like this

    var navigationBar = Ti.UI.iOS.createNavigationWindow({
    window : Ti.UI.createWindow({
        height : Ti.UI.FILL,
        width : Ti.UI.FILL,
        backgroundColor : 'white',
        title : 'Main Window'
    })
});

navigationBar.open();

    Here you will have a window with its own navigation bar with sliding opening and closing. Just keep your navigation window object reference as global so that you can access that to open and close any window in any file.
    In case of alloy you can save it in 

    Alloy.Global.navWindow=$.navWindow;
//This will be available throughout the application
