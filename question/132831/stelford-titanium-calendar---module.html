Stelford Titanium Calendar - Module

Hello all,
I installed the calendar module, following the step by step instructions in the readme inside the module.
He also fails to see the form, at the time of require:

var ticalendar = require ("com.ti.calendar");

Even if the console is: Detected third-party module: com.ti.calendar/0.1

In a nutshell what I did was:
1 - Copy and paste the form .zip in ~/Library/Application Support/Titanium/modules/iphone/com.ti.calendar/0.1
2 - Add the line in tiapp.xml:

<modules>
<module version="0.1"> com.ti.calendar </ module>
</ modules>

3 - And the last step:

var ticalendar = require ("com.ti.calendar");

I use iPhone and iPad simulator to test this project, with iPhone OS 5.0 and SDK 1.8.1 mobile titanium.
Does anyone know what this is or is unable to resolve this problem?
