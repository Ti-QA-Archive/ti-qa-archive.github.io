Upload progress running to 101%?

Hi all.
I am uploading images to a web API and running the following code to get the upload progess

xhr.onsendstream = function (e) {
    loadLabel.text = 'Uploading, '+(Math.floor(e.progress * 0.5*100)*2)+'% Complete';
};
xhr.ondatastream = function (evt) {
    loadLabel.text = 'Uploading, '+(Math.floor(evt.progress * 0.5*100) + 0.5)*2+'% Complete';
};

The upload starts at 101%, then jumps to 0% and then climbs up to 101% again and ends.
What is the correct way of giving the upload percent?
Its very close but obviously not perfect.

This has been bugging me for ages!
Please help!
Adam.

  • Hi Adam,

    You are using two callbacks, which both update the same label, that might be a problem, i think for the upload monitoring only the "onsendstream" callback is needed. So you have to remove the "ondatastream" callback.
    Also it might be better to use the "toFixed" Method of the Number Prototype to trim the decimals.

    Hope this helps

    xhr.onsendstream = function (e) {
    loadLabel.text = 'Uploading, '+Math.round(e.progress.toFixed(2)*100)+'% Complete';
};
