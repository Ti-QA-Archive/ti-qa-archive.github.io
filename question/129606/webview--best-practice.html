  • 1) I agree with Aaron: Titanium might not be the best platform to develop a webViews-based app.

    2) To answer your question:

    • You can use the 'html' property of the webView to set the value of DOM-elements.
    • To read a DOM-element, you can trap an event in HTML and then fired an app-event (see example below).

    Example:

    // Create client's signing button
html = html + '<ul><li>' + '<button type="button" style="color:black"' + disabled + 'onclick="Ti.App.fireEvent(\'app:signature\', {who : \'client\'});">' + L('client_signature') + '</button>' + '<br>';
html = html + '</li></ul>';
