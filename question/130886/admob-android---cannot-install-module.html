AdMob Android - cannot install module?

I downloaded AdMob module from the market. The iOS version works OK, but I cannot make Android version work.

I followed the tutorial:
http://developer.appcelerator.com/blog/2011/09/admob-for-android-module-tutorial-and-code.html

But I get an error:

[ERROR] /Users/adrian/Documents/Titanium Studio Workspace/Reklama/build/android/AndroidManifest.xml:13: error: Error: String types not allowed (at 'configChanges' with value 'keyboard|keyboardHidden|orientation|screenLayout|uiMode|screenSize|smallestScreenSize').
[ERROR] Error generating R.java from manifest

My tiapp.xml includes:

    <modules>
        <module version="2.0" platform="android">ti.admob</module>
    </modules>

If I change version to 1.1, the app runs but says:

[WARN] Couldn't find app module: ti.admob

and dies with:

Uncaught error: Requested module not found:ti.admob

Any ideas how to solve it?

  • This should solve it (worked for me), thanks to Shawn Lipscomb's answer (in below post)

    For anyone who's having the

    [ERROR] Error generating R.java from manifest
String types not allowed (at 'configChanges' with value 'keyboard|keyboardHidden|orientation|screenLayout|uiMode|screenSize|smallestScreenSize').

    error, all you need to do is add

    <tool-api-level>14</tool-api-level>

    to the "android" section of your tiapp.xml file:

    <android xmlns:android="http://schemas.android.com/apk/res/android">
    <manifest android:versionCode="1" android:versionName="1.0"/>
    <tool-api-level>14</tool-api-level>
</android>

    http://developer.appcelerator.com/question/129947/titanium-1801-and-admob-module-on-android
