  • Hi Jaden Tj,

    Autocomplete search property like native android is not available in titanium. But you can do that various way here is one tableView search. I think it will be little bit helpful for you.

    var win = Titanium.UI.createWindow({
    title : "Table Search",
});

// create table view data object
var data = [];

for (var i = 0; i < 10; i++) {
    var row = Ti.UI.createTableViewRow({
        hasChild : true,
        title : 'Row ' + i
    });

    data.push(row);
};

var search = Titanium.UI.createSearchBar({
    barColor : '#385292',
    showCancel : false,
    hintText : 'search'
});

search.addEventListener('return', function(e) {
    search.blur();
});
search.addEventListener('cancel', function(e) {
    search.blur();
});

// create table view
var tableview = Titanium.UI.createTableView({
    data : data,
    search : search,
    filterAttribute : "title"
});

win.add(tableview);

win.open();

    If you have any concern about this query then let me know. And if this is helpful then mark this as answered so that other can learn from it.

    Thanks.
