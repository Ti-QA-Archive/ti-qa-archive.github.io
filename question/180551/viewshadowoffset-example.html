ViewShadowOffset example

Does anyone have an example of using viewShadowOffset? I want to create a drop shadow 3px on the right and bottom. The docs are a bit light in actual explanation and no matter what I set, viewShadowOffset seems to do nothing. I can control the with of the shadow and color using the other 2 attributes, so I must be doing something wrong with the actual offset param.

Thanks!

    It's a point value, so this should work:

    var view = Ti.UI.createView({
    width: 100,
    height: 100,
    borderWidth: 1,
    backgroundColor: '#fff',
    borderColor: '#000',
    viewShadowColor: '#000',
    viewShadowOffset: { x: 2, y: 2 },
});
