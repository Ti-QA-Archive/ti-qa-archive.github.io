Update All Items in ListView (Slow Performance)

I am using a list view to display a list of about 40 items. I'm trying to implement like a select all concept. This is working however when i loop through all the items int he section to update it the performance is slow. I tested a few variations and it is the section.updateItemAt code which slows things down regardless of what I update. Any ideas on another way to do this?

Here is my code snippet:

var section = $.filterlistView.sections[secindex];
var shows=section.getItems();
for (var i = 0, l = shows.length; i < l; i++) {
Ti.API.info("updating unselect "+i);
var item = section.getItemAt(i);
item.selected.issel=0;
//item.properties.backgroudnColor="blue";
//item.image.image="/images/icon_gray_circle.png";
section.updateItemAt(i, item);
}

    If you have to update all items, setItems(yourItems) is much faster than updating each item in a loop.

    http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.ListSection-method-setItems
