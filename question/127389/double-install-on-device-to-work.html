Hi,

I've noticed one problem when i try to install appecelerator apps on my devices.

The first time i install them, all apps crash on lauching. I have to re-install a second time, to make it work.

    This sometimes happen if you go straight from an emulator/simulator build to a device build without doing a clean on the build path first. Doing a clean before installing to device should alleviate this issue.
