Outlook PST Repair Tool

A pst looks like it needs repairing however the pst file is 7 GB large. Everytime I click ok to fix the pst file after it does the CRC check, it crashed. Is there anyway to reduce the size of the PST file so the repair tool wont crash?

    You can reduce the size of the PST file either by compressing or splitting a PST file.In your case ,issue is corruption in the PST file.so you need to first repair it then apply any compression and splitting process on the repaired file.

    If scanpst fails then you can try any third party repairing tool. For example Outlook Repair Toolbox. The MS Outlook PST repair tool also enables you to convert .OST files to .PST format. https://www.repairtoolbox.com/outlookrepair.html
