    • You CAN have a window set to vertical, and inside that, create a View, and the vertical does not apply to it (I've had to do this when having a mostly vertical layout, but wanting horizontal placement of buttons)

      — commented October 26th 2013 by Josh Lewis

    • Josh whilst you are correct a parents layout will not set that same layout to its child, the grandchild will be constrained to its parent.

      So the constraint still applies as you cannot have one child act as absolute if the parent is set as vertical or horizontal. Whilst a child of a parent can set the grandchildren to have absolute, vertical or horizontal that will not change that child's interaction with its parent.
