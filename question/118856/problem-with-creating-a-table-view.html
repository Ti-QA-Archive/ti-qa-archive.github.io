  • You'll want to take a look at the code in Kitchen Sink, namely table_view_textfield.js for how to insert text fields into rows, and table_view_api_grouped.js on how to do groups. In short, you'll want to use Ti.UI.createTableViewRow() to make rows into which you can insert text fields, and Ti.UI.createTableViewSection() to put those rows into sections. You need only call Ti.UI.createTableView() once, as it will contain both sections.
