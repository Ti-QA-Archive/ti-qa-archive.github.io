Unable to find an iOS Developer Certificate

Hi, I have my iOS Developer Certificate installed and everything seems correct but when I try to install an app in my iPad I get this error. It seems it doesnt' read accents. Whan can I do? Thank you.

Available developer names:
Antonio \303\211vora Gala (E3DDDLG6GV)
[ERROR] : Unable to find an iOS Developer Certificate for "Antonio Évora Gala (E3DDDLG6GV)"

  • Accepted Answer

    There were issues with iOS certs that contained non-ascii letters like É. This was fixed in Titanium CLI 3.0.24 and Titanium 3.0.2.GA.

    Run ti -v to verify you have 3.0.24 installed. If not, run sudo npm install -g titanium.

    Run ti sdk to verify you have 3.0.2.GA installed and it's the active SDK. If it's installed, but not active, run ti sdk select 3.0.2.GA. If 3.0.2.GA is not installed, run ti sdk install --default.

    Lastly, make sure the <sdk-version> in your app's tiapp.xml is set to 3.0.2.GA.
