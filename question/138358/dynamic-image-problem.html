Dynamic image problem

Currently I have created an imageView with an image ( obviously ). The following code works

‘var titelImage = Ti.UI.createImageView({
image: 'images/titel.png',
width:196,
height:49,
top:10,
right: 10
})‘

But now I want to make an imageView where the image path depends on what parameter comes in the function

‘var aaneen = naam.split(' ').join('');
var imagepad = 'images/'+aaneen+'/titel.png'
var titelImage = Ti.UI.createImageView({
image: imagepad,
width:196,
height:49,
top:10,
right: 10
})
‘

This gives the error >[ERROR] invalid image type. expected TiBlob, String, TiFile, was: NSCFString in -[TiUIImageView setImage_:] (TiUIImageView.m:684)

Anyone who can help me ?

  • I don't see anything wrong with your code. I was able to run this just fine:

    var w = Ti.UI.createWindow ({
    backgroundColor: '#fff',
    title: 'foo'
});

var naam = 'foo';

var aaneen = naam.split(' ').join('');
var imagepad = 'images/'+aaneen+'/titel.png';
var titelImage = Ti.UI.createImageView({ image: imagepad, width:196, height:49, top:10, right: 10 });

w.add (titelImage);

w.open ();

    I had to make sure to create a 'foo' directory under my 'Resources/images' directory, and I put an image in there called 'titel.png'.

    I would look closely at the variable 'aaneen'. Have you run it in the debugger to look at what is being put into aaneen and imagepad?

    Also, be sure to use semicolons at the end of all your lines (you left it off the "var imagepad" line). I don't think that was the problem here, but it's a good practice to follow.
