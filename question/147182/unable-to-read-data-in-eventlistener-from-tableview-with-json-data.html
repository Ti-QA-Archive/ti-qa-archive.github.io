    Hi

    Change this;

    row = Ti.UI.createTableViewRow({
    height:'90dp'
});

    To this;

    row = Ti.UI.createTableViewRow({
    height:'90dp',
    FB_ID: SMKs.SMK.id
});

    Then you can access like this;

    var rowdata_fb = e.rowData.FB_ID;

    In order to retrieve from the rowData property - you need to add the data to the row. :-)
