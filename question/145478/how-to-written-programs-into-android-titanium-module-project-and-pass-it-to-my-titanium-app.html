  • If you want to create a native android Module, the first thing would be to familiarize with this section from the Titanium Guides. This will help you get started in terms of development environment and explain the basics of what a module is.

    You have to keep in mind though, that in order to build Modules for Android, you must have knowledge of the Android SDK. The guide assumes you have a good understanding of building applications using the native Android SDK.

    Otherwise, the learning curve can be pretty steep.
