Android Submission - Which Android SDK should i use?

Hi,

Like already read in the headline, i would like to know which of the given Android SDK-versions i should use when uploading to the Google Play store? I have 2.2 up to 4.0.3 but i'm not sure if my app stays compatible to all devices when selecting the 4.0.3.

Thank you!

    From what I understand you are supposed to choose the Google API version according to the version of the Titanium Mobile SDK you have. So if you are using Titanium Mobile SDK 2.0 you would choose Google API 2.2.

    http://docs.appcelerator.com/titanium/2.1/index.html#!/guide/Titanium_Compatibility_Matrix-section-29004837_TitaniumCompatibilityMatrix-PrerequisiteAndroidSDK%2FAPI
