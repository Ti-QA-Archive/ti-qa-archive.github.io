Crash log on device without Play Store

From what I gathered apps can only log 'themself', as in I can not use the logging apps that exist in play store. I couldn't find a hint in the titanium docs.

I want to test my app 'in the wild', so I wanted to know if it is possible to save the debug log, or a similar log on device without being connected via USB, so I can review what happened right before an app crashes?

  • Sadly you can only log rooted devices. So any logger in the play store will fail to log what you want.

    You could try to use Hockeyapp with this module but the logs will be pretty limited to the crash part and java only.

    However is more than nothing :)
