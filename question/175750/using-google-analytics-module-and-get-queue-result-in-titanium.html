  • Hi Silambarasan Raman,

    The above debug is dispatch , the data collected by GA.

    Data collected using the Google Analytics SDK for Android is stored locally before being dispatched on a separate thread to Google Analytics.

    If we have not set the dispatch time then ,By default, data is dispatched from the Google Analytics SDK for Android every 30 minutes.

    you can set the dispatch time using below code example:

    GA.localDispatchPeriod = 10;

    and you can see the dispatch debug due to GA.debug = true
    if you want more info about the dispatch please refer the below link

    https://developers.google.com/analytics/devguides/collection/android/v3/dispatch

    let us know for more help
