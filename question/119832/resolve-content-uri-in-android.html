Resolve content URI in Android

How do I resolve a content:// URI to an actual file?

Hi all,

I have used the example code to record audio on Android. The example launches an Intent, the user records the file and confirms, and then I get an event back with the result of the intent.

The result of the intent in a URI, in the format of content://resource/location/id

I need to access the actual file, so I can upload it using a web service. All the examples of sharing audio do so by using another (built in) Intent, which accept the URI as their input.

I have read up on how this would be done using pure Java, and it seems you would need to use the ContentResolver class.

So my question is: How do I access this class from Appcelerator?

Any help much appreciated.

Thanks,
