  • Hi

    This is a custom view group that the app developer has designed themselves - there is no built in ability to achieve this layout. You can create it yourself but you will have to put some work in to make it happen.

    You will need to use the transform and animate methods in order to achieve this. The transform allows you to make changes to the current presentation of a view - including the angle it is displayed. The animation allows you to bring the view into place with a starting and stoping angle with offset if required, so the it can achieve what I assume is a rotation in to view of the cards.

    There is no code example as this is very custom.

    • First thing to note - you do not use a scrollView, these will be individual separate views - otherwise this would not work.

      This is some starter code to show you how to rotate and how to rotate with animation.

      // define the animation
var matrix2d = Ti.UI.create2DMatrix();
matrix2d = matrix2d.rotate(20); // in degrees
var a = Ti.UI.createAnimation({
    transform: matrix2d,
    duration: 2000,
    autoreverse: true,
    repeat: 3
});
// set the animation to the thing that should animate
box.animate(a);

      var transform = Ti.UI.create2DMatrix().rotate(20);
img.transform = transform;

      Which event you choose is up to you, if you hope this will rotate when you slide your finger over the cards then take a look at an answer I gave a while ago to rotate a view on itself, from this you can work out the angle of a touch - important for this.
