Cannot hide android action bar

Hi,

I'm trying to hide the android action bar in 3.4.1.GA. The documentation says that I need to created a custom theme in platform/android/res/values/custom_theme.xml. Can someone tell me directory what folder should contain platform/android/res/values/custom_theme.xml?

I've tried rooting in in assets and app without any luck. Thanks!

— asked July 11th 2015 by Jim Brazil
  • android themes
2 Comments

  • try this in tiapp.xml :

    <android xmlns:android="http://schemas.android.com/apk/res/android">
        <manifest android:versionCode="9"
                android:versionName="1.0.7">
            <application
                android:label="@string/app_name" android:theme="@style/Theme.AppCompat.Translucent.NoTitleBar"/>
        </manifest>
    </android>
    — commented July 12th 2015 by Ahmed Arafah

  • Thanks that was it!

    The platform directory needs to be rooted your project's home directory (at the same level as the app directory if you're using alloy).
