  • You need to get the urbanairship module from the marketplace. Read the documentation in the package that explains how to use the module. If you are developing on Mac, you need to put the module in &#x2F;Macintosh HD&#x2F;Library&#x2F;Application Support&#x2F;Titanium&#x2F;modules&#x2F;android directory. If you get stuck on some point, read this thread.
