    and your div will be updated. You should be able to get this to work in a plain html file first, then load it into titanium.

  • The trick to using PHP is to understand that Titanium basically blends your PHP code and your JavaScript together when it builds the executable, but you have to include your PHP in the proper manner.

    This page really says it all:
    http://developer.appcelerator.com/doc/desktop/php

    Any PHP code that is wrapped in <?php ?> tags will be preprocessed, which means it only runs once when the program starts. It sounds like you want to execute your PHP code on-demand, in which case your PHP code should be wrapped in <script type="text/php"></script> tags. Whatever functions you declare in there can be called directly from JavaScript/jQuery, just as though you'd written them in JS.

    It's kind of a mind-bender, really, but you'll slowly start to realize that your JS now has access to things like MySQL, and you PHP has direct access to the DOM…
