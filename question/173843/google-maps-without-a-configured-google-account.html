Google maps without a configured google account

Hi, I'm developing an android app that uses a lot of google maps features. Since there is the nice to have to run it in a kiosk like environment I wonder if it's possible to use such features with the play services installed but without a configured google account.

thank you

    Hi

    Short answer - no.

    Longer answer;

    As I understand it, the Google Play Services are only actually enabled once there is a correctly configured account. Having fallen the wrong side of this myself when testing and not remember to set up and account I spent some time trying to debug the map code - only to remember the need for a Google account.

    There is nothing to say the account would need to be a persons account - you could of course set up and account specifically for the purposes of this app. If the app is designed to be distributed - provide the same instructions to the users - "please configure an account - or create a generic one if you do not have one".
