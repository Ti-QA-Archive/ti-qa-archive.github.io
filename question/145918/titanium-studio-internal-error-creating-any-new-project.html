Titanium Studio Internal Error creating any new Project

Mac OS X 10.8.2 Downloaded and installed Titanium Studio (3 times!) 3.0.0.GA Installed Xcode, installed Android SDK. Java version 1.7.0_03 is the default. Have applied latest updates.

Spent several hours searching on Google, Appcelerator Q&A, etc. Did not find similar report.

Every time I try to create a new project get the error:

   Creation Problems

   Internal Error

Opening up the Details has:

      java.lang.reflect.InvocationTargetException

Did not see any other log with a full stack trace (where could I find that if it exists??)

Have tried creating one of the standard templates (single window app, two tab app), have also tried just creating from New. Same result.

On the New Titanium Project screen, have tried creating projects with deployment only to iPad, or only to iPhone, or Android, or Mobile Web. Same result.

Have tried importing sample projects, sometimes works sometimes not. Once imported, cannot Save.

Tearing my hair out and about to give up on Titanium. Like the concept and have heard good things from colleagues, but cannot even get started!

Thanks for any help ….
