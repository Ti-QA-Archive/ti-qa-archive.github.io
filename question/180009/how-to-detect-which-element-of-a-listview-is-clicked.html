How to detect which element of a listview is clicked

I've been looking around for the answer to this question, but I've yet to find anything, so if I missed it I apologize.

I have a listview where each list item has several different elements that can be clicked. Is it possible, using onItemClick to differentiate which element of that list item has been clicked?

In one of my more simple list views, I have a checkbox. If the checkbox is clicked, I want the checkbox to be filled with '\u2713' (the element itself is just a label with a border, I fill in the text, element id is 'check') and if the list item is clicked anywhere else, the app will navigate to another page based on the list item's data. My first attempt utilizes a an onItemClick function with another listener for the checkbox.

function listViewClick(e) {

var section = $.list.sections[e.sectionIndex];

// get the clicked item from that section
var item = section.getItemAt(e.itemIndex);
navigation.open("newPage", {title: 'Page Based On Item'});

}

function checkClicked(e) {

var check = e.source.text || '';
if(check != ''){
    e.source.text = "";
}
else{
    e.source.text = "\u2713";
}
alert(e.source.text);

}

This obviously does't work. On iOS it allows me to check/uncheck but it will also navigate to the new page, due to the two event functions triggering. On Android the check/uncheck doesn't work at all.

Any help would be greatly appreciated.

Thanks!

    clickHandler is List view onItemclick function.And Set bindId=1 for your check box.

    function clickHandler(e){
    if(e.bindId==1){
        alert("Check box clicked");
    }
}
