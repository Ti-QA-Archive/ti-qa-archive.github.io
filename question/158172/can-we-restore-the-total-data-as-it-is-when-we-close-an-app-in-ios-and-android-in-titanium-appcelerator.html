  • you didn't mentioned which platform you are working on ? android ? ios ?

    for Android .. you can listen to the application back button on your top-level activity ( main window ) and then you can save the whole data you want by using SQLite DB or using Filesystem and save the data as json text in case it's small amount of data
    and when the app open you can check if there is a file or records in db so you can restore them

    on iOS you can listen to the resumed / paused events but be-aware they may not fire if the game interrupted by notifications or phone calls
