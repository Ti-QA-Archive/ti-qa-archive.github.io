Position AdMob advertisement outside of NavigationWindow

Hi,
I am using Alloy and and Titanium.UI.iOS.createNavigationWindow to switch between windows. I would like to position the Admob banner outside of the navigation window so it does not disappear when the user navigates.

How to do this?

if (OS_IOS)
    navWindow = Titanium.UI.iOS.createNavigationWindow();

openRootWindow =function(controllerName) {
    var view = Alloy.createController(controllerName).getView();

    if (OS_IOS) {
       navWindow.window=view;
       navWindow.open();
    } else {
        view.open();
    }
    return view;
};
