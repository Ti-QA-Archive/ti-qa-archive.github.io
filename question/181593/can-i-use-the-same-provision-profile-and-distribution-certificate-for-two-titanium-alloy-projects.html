Can I use the same provision profile and distribution certificate for two Titanium Alloy projects

Hi,

Can I use the same provision profile and distribution certificate for two projects built in Appcelerator Titanium Studio for iPhone?

The two projects are actually same so have the same AppID.

The reason for creating two projects is : while trying to do an upgrade to my existing app in Apple App Store the webView stopped working. As a workaround when I created a new project and copy pasted the same code the webView miraculously started working.

So will I be able to use my existing distribution and provision profile certificates to push the upgrade (new Project in Titanium Studio) to AppStore?

Will there be any issues with the guid property as it is unique for every project?

What will I have to do to make it work.

Other details :
CLI version 3.4.2, Titanium SDK version 3.5.1.GA, Mac OS X, Version=10.10

Thanks in advance
Bably

    Hi Babaly Das,

    Yes you can use same development provisioning and distribution provisioning file for same application. it doesn't mater you create new application in studio. it's depends on appID. so you can use it same. you need to change it only in case if you want to upload new app not upgrade existing app.

    if your application is already uploaded to app store then only you need to increase your application version both side application and app store..
