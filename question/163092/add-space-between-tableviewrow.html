How to add space between each TableViewRow for tableview

Window.Js

for(var i=0;i < benefitElection.length ;i++){

        height:(Ti.Platform.osname == 'ipad')?'280':'140',

        left:(Ti.Platform.osname == 'ipad')?'20':'10',
        right:(Ti.Platform.osname == 'ipad')?'20':'10',
        color:'red',

  • Hi, kiran arishanapalli

    You can use a view for each row, then add a spacing view at the top. The issue is that the Row height adjusts automatically for the contents of each row, so you can either set a fixed height, or use a view with layout vertical to add your content to and then add that view to the row.
