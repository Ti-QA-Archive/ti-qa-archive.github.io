filtering our tableview rows via Search Bar

Hi

I did add search bar but I am not finding way to filter data in Tableview. I am filling up tableview via Query. What should be mentioned in "filterAttribute". Is it db column, array field or something else?

    The filterAttribute should be the name of a property of the row. That may be as simple as filterAttribute: 'title' to filter based on the contents of the row title.

    Or more typically, you create another property in each row and place in it whatever you want. This property does not need to be visible in any object.
    Then you name that property in the tableview filterAttribute.
