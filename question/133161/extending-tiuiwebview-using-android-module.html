Extending TiUiWebView using android module

Hi, I am trying to create android module which would extend TiUiWebView with the setSavePassword method or property (http://developer.android.com/reference/android/webkit/WebSettings.html#setSavePassword%28boolean%29).

So far I managed to configure all needed components for android module development and I am able to create simple modules extending TiUiView and also successfully compiled and tested titanium moddevguide module (https://github.com/appcelerator/titanium_modules/tree/master/moddevguide)

I also found one related post in QA (http://developer.appcelerator.com/question/122089/android-titanium-module) but unfortunately could not contact the developer who created very similar module. After that I tried to use classes from Titanium SDK source as it's described in his post, but without any success.

So the question is - maybe someone else has created such a module and could provide some insight in how it should be done? Maybe even share the source of such or similar module?

Basically what I need is some example on how to get simple webview module up and running, from that point I believe I will be able to extend it as needed.

-
My development computer is Windows 7 and I am using Titanium Mobile SDK 1.8.2

  • Just a followup - as a workaround I managed to add
    settings.setSavePassword(false);
    to the Titanium Mobile SDK source - \titanium_mobile\android\modules\ui\src\java\ti\modules\titanium\ui\widget\webview\TiUIWebView.java file (94 line)
    and after that compiled it using scons.

    Not exactly long term solution, but it works for now - I am now using custom 2.0.0 SDK build and there is no more save password dialog when user enters username/password in webview.
