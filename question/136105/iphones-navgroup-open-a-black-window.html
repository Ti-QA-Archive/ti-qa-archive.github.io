iphone's navgroup open a black window

Hey!,

I clone the app structure from appcelerator-titans community app, so here's the: gist.

What I'm doing is this:

  • Create a window with a TableView inside.
  • In this window I create a navgroup, I set the root window as the same main window.
  • When I click on a row I do this: navGroup.open(w, {animated: true});

Then I got a black window :S, wha'ts wrong with this code?.

Thanks for any help.

  • Accepted Answer

    You need at least 2 windows, one as "root" window1 for the app and a second window2 as a "root" for the navGroup. Then you have to window1.add(navGroup); and window1.open();

    All following openings you have to navGroup.open(win);
    The navGroup is not standalone and requries an underlying window.
