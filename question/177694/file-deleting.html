File deleting

Hi there, i'm very new to titanium, and have an extremely basic question:
I started a new hello world project,added a sqlite database to my resource folder an i added also a folder called products as a sub folder in Resource folder. As soon as i start the build process, titanium studio removes all the newly created folders, .js files and even sqlite database? I checked in the project folder (windows explorer, the database was there, but got removed from the file system)

Any idea?

  • You don't say, but I'm guessing that you've created an Alloy-based project (instead of a "Classic" project). With Alloy, the contents of the Resources folder is generated during the compile process, and yes, can be wiped at any time by that process. Your files should all go in the app folder or its subfolders. In the case of a database file, you'd put it in the app/assets folder.

    You might want to take a look at our Alloy Quick Start guide. The Convention over Configuration section describes the purpose of each of the folders.
