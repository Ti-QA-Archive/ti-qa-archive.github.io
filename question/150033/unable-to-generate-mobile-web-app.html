  • That error should not be preventing Studio from building. All Studio is doing is starting a web server, performing the build, and launching the web browser. I recommend you just running nginx or Apache. Studio's web server is pretty basic. Just point the web server's vhost root to the /path/to/your/project/build/mobileweb. You'll be able to develop much faster if you build at the CLI, then switch to the browser and refresh. Takes a few times to get used to it, but it's totally worth it.

    Also, I should point out that Titanium Mobile Web apps should only be served in the root of a web directory. You should NOT put it in http://mysite.com/myapp.
