Which iOS SDK to use when publishing App

Hi all,

I am about to be ready to publish my app to Apple (at last!…) I was wondering now which iOS SDK i should go with ?
I know that selecting 4.3 works perfectly on my 4.0 device as i ve been testing both 4.0 and 4.3 So is it safe to select it ?
Also, I kinda remember that during the publish process we are asked to specify which device we plan to target. Can i safely say 4.0 and above ?

Thanks !

Seb

  • Accepted Answer

    Whatever you've been using to build and test with is what you should deploy with. This is simply because APIs and functionality changes with each update. If you go back to far, then functionality within your app may not exist, if you choose something above what you've tested for, then that is just bad juju.
