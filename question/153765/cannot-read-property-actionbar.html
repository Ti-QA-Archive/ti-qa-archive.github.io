Cannot read property actionBar

I am trying to add an actionbar to my Android app. The window where i want the actionbar to appear is opened with the following code

Alloy.createController('main').getView().open();

Here is the code of my controller "main"

function doOpen(){
    var activity = $.win.activity;

    if (activity.actionBar){
        activity.actionBar.title = "MyApp";
    }

    activity.onCreateOptionsMenu  = function(e){
        var menuItemEvents = e.menu.add({
            title: 'Event',
            showAsAction: Ti.Android.SHOW_AS_ACTION_ALWAYS
        });
    }
}

And the corresponding view

<Alloy>
    <Window id="win" class="container" onOpen="doOpen">
    </Window>
</Alloy>

This gives me the error "Cannot read property "actionbar" of undefined" in main.js.
What am i doing wrong? Why is $.win.activity undefined?

SDK 3.1.0.GA
Android 4.2.2
