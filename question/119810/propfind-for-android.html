PROPFIND for Android

I'm trying to connect to a webdav server. I need to be able to perform a PROPFIND request, but it seems that Android is using org.apache.http. This does not support PROPFIND or some of the other webdav requests. Is there a work around or a different way to do this? 

var xhr = Titanium.Network.createHTTPClient();
xhr.open("PROPFIND","http://mywebdavserver.com");
authstr = 'Basic ' +Titanium.Utils.base64encode("username"+':'+"password");  
Ti.API.info(authstr);
xhr.setRequestHeader('Authorization', authstr);
xhr.setRequestHeader("Content-Type", "application/xml");
xhr.setRequestHeader("Depth", "1");
xhr.send();
