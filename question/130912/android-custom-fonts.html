Android Custom Fonts

I'm having trouble with custom fonts in Android. The code below works in iOS…

I've placed the font file in 'Resources/android/fonts/' and 'Resources/fonts/' with the same unsuccessful results.

I've tried with OTF & TTF with the same result… Am I missing something? Tricks?

var title = $.Label({
    text: titleText,
    left: 17,
    height: 40,
    width: 117,
    color: '#f7f7f7',
    font: {fontFamily:'Museo Slab', fontSize: 13},
    touchEnabled: false,
    zIndex: 2,
    opacity: .6
});

util.androidOnly(function() {
    title.font = {fontFamily: 'Museo_Slab-500', fontSize: '13dp'};
});

Any help would be greatly appreciated :)
