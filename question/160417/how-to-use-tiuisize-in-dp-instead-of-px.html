  • After a lot of search and testing, here's the simplest way, it has an issue but could be handled.

    Whatever object you have, assign Ti.UI.SIZE to it's dimensions, and then access it through the rect property and convert it to dp, here's my code for the imageView.

    imageView.image = url;
imageView.width = Ti.UI.SIZE;
var widthInDP = imageView.rect.width.toString() + "dp";
imageView.width = widthInDp;

    This codes works fine for iOS and has an issue for android, it happened with me but I don't know why, maybe for something in my code, the issue is that Ti.UI.SIZE is generated after some milli seconds, because I tried to alert the width after assigning it Ti.UI.SIZE and it showed the previous width.
    To handle it I forced the app to wait 0.2 second before assigning the widthInDps to the imageView.

    imageView.image = url;
imageView.width = Ti.UI.SIZE;
setTimeout(function(){
    var widthInDP = imageView.rect.width.toString() + "dp";
    imageView.width = widthInDp;
    alert($.img_user.width); 
}, 200); // After 200 milliSeconds this function will execute, you can change it if it's not enough

    It tested it on Galaxy S3 and it works fine, maybe it gives weak user experience but it's the simplest and the best way I found till now.
