Return values from a function in another namespace - how?

Hi!

I have a user namespace at myApp.user in this I have this function:

// user.js

  myApp.user.getUsername = function() {
    return Ti.App.Properties.getString('username');
  };

Now I wanted to just alert the username out from another namespace, like this:

// ui.js

testBtn.addEventListener('click', function(){
  alert(myApp.user.getUsername);
});

But this doesn't work. I get an alert with "<KrollFunction…somthing" (I guess I'm alerting the actual function and not the returned value)

Why is that? Do I have to use a callback function?
Please explain this to me :)

Best Regards
Linus

  • Accepted Answer

    testBtn.addEventListener('click', function(){
  alert( myApp.user.getUsername() );
});

    The way you've set it up its a function and not a property, so you'll need to call it with parenthesis = )
