    • Hi Afeeza,

      ok we will check again.

      I assume that you are still on Titanium Studio 2.1.2.201208301612.

      1. 1) please ensure that you have installed Titanium SDK 2.1.4.GA (if not, install it)
      2. 2) start Android SDK Manager from Titanium Dashboard.
        Dashboard -> click on Android -> click Update Android SDK
      3. 3) check Packages, as minimum Android 2.2 (API 8) SDK Platform and Google APIs are installed
      4. 4) check Packages, Tools Android SDK Tools and Android SDK Platform-tools are installed
      5. 5) start AVD Manager from Android SDK Manager Menu: Tools -> Manage AVDs
      6. 6) delete all Android Virtual Devices (Titanium Studio will create a new one for you)
      7. 7) close Android Virtual Device Manager and Android SDK Manager
      8. 8) open Run Configurations from Titanium Studio Menu: Run -> Run Configurations…
      9. 9) delete all Entries under Titanium Android Emulator and close Run Configurations Dialog
      10. 10) open your tiapp.xml and select Titanium SDK: 2.1.4.GA, Android Runtime: V8, Deployment Target: Android.
      11. 11) click Menu: File -> Save (or Ctrl+S)
      12. 12) in TiApp Editor click configure…
      13. 13) select Default Android SDK: Android 2.2 or 2.3.3 or higher
      14. 14) clean your Projects from Titanium Studio Menu: Project -> Clean…
      15. 15) check Clean all projects and click OK
      16. 16) start Windows Task Manager and search for process adb.exe and kill that process.
      17. 17) open your app.js and paste the code i provided below (only that code!)
      18. 18) save your app.js
      19. 19) in Titanium Studio App Explorer click on the Run icon and select Android Emulator.

      Code for app.js

      Ti.UI.setBackgroundColor('#000');
var win = Ti.UI.createWindow({
    backgroundColor : '#ffffff',
    title : 'Test',
});
var button = Ti.UI.createButton({
    title : "Hello World",
    top : 30,
    left : 20
});
win.add(button);
win.open();

      If any step fail, please comment with step number and description here.

      Hope that helps,
