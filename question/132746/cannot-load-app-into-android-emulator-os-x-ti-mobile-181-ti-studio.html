  • Hey Josh,

    Android emulator needs more memory than iOS simulator. Which emulator are you trying on (I mean 2.2 HVGA, 2.3 HVGA, QVGA etc etc) ?

    And if you want to debug your application from device than also you can do that. You can start DDMS and check your application log in DDMS while you are running application in device.

    If you are having less memory for your android emulators, it will take time to get started and in the mean time titanium application launcher will through time out error. So you can do one thing. Start emulator from AVD manager. Once it get started fully than you can press RUN button to run your application from Ti Studio.

    Basically I want to say is, it's issue with less memory.
