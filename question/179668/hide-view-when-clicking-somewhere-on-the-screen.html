Hide view when clicking somewhere on the screen

Hi all

In my application I have 5 cards, when someone clicks on a card, it will show the contents of the card in the middle of the screen, I would like to hide the contents when the user clicks somewhere on the screen. How do I do that ? And one more thing, when the user clicks on the first card it shows the contents of the card in the view's label, and when the user clicks the second card the contents of the card 1 and the card 2 are getting overlapped. So when the user clicks the first card the contents of the first card should be shown, and again now when the user clicks on the second card, the contents of the first card should be cleared before the contents of the second card is printed. Here is my code

index.xml

<Alloy>

        <ImageView id="coverFlowView" onClick="coverFlowView" top="5"></ImageView>
        <ImageView id="imageView1" onClick="clickImage" image="/images/origins_playing_cards_back.jpg"></ImageView>
        <ImageView id="imageView2" onClick="clickImage" image="/images/origins_playing_cards_back.jpg"></ImageView>

        <ImageView id="imageView5" onClick="clickImage" image="/images/origins_playing_cards_back.jpg"></ImageView>        
        <View id="contentView" visible="false">

        </View>        
    </Window>
</Alloy>

  • Hi Varun,

    for your first question you r did right.. click event of window is fired every time even i click on any of its children.
    by default almost all controller have bubbleParent property is true. so it's transfer event to its parent controller.
    so u r did right code for that.

    and for your second question you need to remove all children from contentView and then add selected label.
    e.g

    function clickImage(e) {
    // Titanium.UI.createAlertDialog({title:'Image View', message:'You clicked me!'}).show();
    var lblContent = Ti.UI.createLabel({
        text: Alloy.Globals.textCollection1[e.source.id].message
    }); 
    $.contentView.removeAllChildren( );    
    $.contentView.add(lblContent);
    $.contentView.visible = true;   
}
