Alloy view xml attribute : ns

I have read "ns" in the example of Alloy mapping.
What does it mean?
here is the code. ("ns" is in line 2.)

<Alloy>
    <View ns="Ti.Map" id="map" mapType="Ti.Map.STANDARD_TYPE">
        <Require src="annotation" title="Appcelerator" />
    </View>
</Alloy>

    As it is explained in the documentation. 

    All UI components specified in the views are prefixed with Titanium.UI for convenience. However, to use a component not part of the Titanium.UI namespace, use the ns attribute.
