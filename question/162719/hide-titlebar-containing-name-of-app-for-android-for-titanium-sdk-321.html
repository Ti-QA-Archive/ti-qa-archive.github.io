Hide titleBar containing name of app for Android for Titanium SDK 3.2.1

Hi,

I am developing a mobile app both for Android and iPhone using Titanium Alloy.

Titanium SDK version 3.2.1.
Alloy version - 1.3.1
Android SDK shown when I check the latest installed version in Titanium Dashboard is 3.0

My problem is that the 'titleBar' containing the name of the app is shown when I run the app in Android.
When I was using Titanium SDK 3.1.3, in the tiapp.xml file, I had done 

 <navbar-hidden>true</navbar-hidden>

and it hid the title Bar. But this is not working for latest version of Titanium SDK.

So how can I hide the titleBar for Android in Titanium SDK 3.2.1?

Sorry if it is a basic question but as I am not being able to find a solution so decided to ask this question in the forum.

Thanks in advance.

    • Hi, Bably Das!
      I'm developing android apps with Ti and navbarHidden property works on Android.

      For example:

      var self = Ti.UI.createWindow({
        backgroundColor : '#FFFFFF',
        layout : "vertical",
        navBarHidden : true,
        width : Ti.UI.FILL,
        orientationModes : [Ti.UI.PORTRAIT]
    });

self.open();

      SDK version: 3.2.0.GA
