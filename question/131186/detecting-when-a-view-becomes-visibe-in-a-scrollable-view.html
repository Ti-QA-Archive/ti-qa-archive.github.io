Detecting when a view becomes visibe in a scrollable view

I have a 3 views in a scrollable view. I want to be able to run some additional javascript but only once the middle of the 3 views is visible.

Is this possible? I can't see how to do it in the documentation.

Thanks

  • Try:

    var sv = Ti.UI.createScrollableView();
var v1 = Ti.UI.createView(); // these would be your views
var v2 = Ti.UI.createView();
var v3 = Ti.UI.createView();
sv.views = [v1, v2, v3];
sv.addEventListener('scroll', function(e){
  if(e.currentPage == 1) {
    // do stuff
  }
  /* OR, IF WORKING WITH INDEX NUMBERS DOESN'T WORK, TRY
  if(e.view == v2) {
    // do stuff
  }
  */
});

    Within the event listener, the currentPage property gives the ordinal (zero-based) index number of the view being showed; e.view is the current view being shown.
