Changing views in a ScrollableView

Hi guys

I have a ScrollableView with 50 remote images.

I want to easily change views from 1 => 50, 50 => 25 and so forth.

From looking at the docs, there is a method called .scrollToView() but what it does is animate from the current view to the wanted view. I want to change views instantly without any animation.

Is this possible?

Thank you in advance.

    Hi John,

    Probably you should check out the currentPage property. Try with 

    myScrollable.currentPage = 50;
