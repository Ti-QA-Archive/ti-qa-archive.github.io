    • @Sam, we've been working a tool that does dynamic code analysis formally called the "code processor". It's reallly powerful and can do all sorts of neat things.

      It's not GA yet, but you can clone the git repo and start playing with it today: https://github.com/appcelerator/titanium-code-processor.

      The code processor by itself can tell you some useful information, but we have also created a bunch of plugins that can perform deep code analysis and code optimizations. These plugins will only be available as apart of the Appcelerator Platform.

      — commented April 9th 2013 by Chris Barber

    • It's also worth noting that the code processor will be released as part of 3.1.0, meaning there will be docs/guides/etc on it.
