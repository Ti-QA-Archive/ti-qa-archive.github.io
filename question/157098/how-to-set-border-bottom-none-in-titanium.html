    Actually what you want if you want to set border in tableView or view simply you can add border property like this but if you want to set border except bottom there is no such kind of property in titanium . 

    var tableView = Ti.UI.createTableView({
    borderWidth :10,
    borderColor:'red',
    data : tableData
});
