How to open a new window with animation?

Hi everyone. I'm new in Titanium and went through the introduction videos. The kitchensink application is not a best practice as we all know but I'm a bit in the dark here about working with different windows.

I'm creating an application that lets users login via Facebook. After successful login I want to animate the window to a new window.

I have this code so far, but that isn't working:

if(Titanium.Facebook.loggedIn)
{
        var windowHome = Ti.UI.createWindow({url:'windows/HomeWindow.js'});
        windowHome.open();
                alert('ok');
}

Inside of "HomeWindow.js" I have the following code:

var winHome = Titanium.UI.createWindow({  
    title:"Home",
    backgroundColor:'#0F0'
});

The alert "ok" shows up just fine so I'm hitting the correct event. Any help is appreciated!

    Try this…

    if(Titanium.Facebook.loggedIn)
{
        var windowHome = Ti.UI.createWindow({url:'windows/HomeWindow.js'});
        windowHome.open({animated:true});
}
