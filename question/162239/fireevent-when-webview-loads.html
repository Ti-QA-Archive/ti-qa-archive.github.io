FireEvent When WebView Loads

HI,

I followed a tutorial on the Wiki so that i am getting data from a index.js controller to a WebView however at the moment, the fireEvent only occurs when a button is pressed:

button.addEventListener('click', function(e) {
    Ti.App.fireEvent('app:fromTitanium', { message: 'data1, data2, data3' });
});

How do i get it to fire only when the web view has loaded?

Thanks.

    There is a 'load' eventListener which you can use to determine when the webview has loaded.

    Here is an example, using your fireEvent, if it is correct:

    webview.addEventListener('load', function () {
    Ti.App.fireEvent('app:fromTitanium', { message: 'data1, data2, data3' });
});
