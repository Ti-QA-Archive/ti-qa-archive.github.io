How to remove a page from a Scrollable View?

Does anyone know how to dynamically remove a page from Scrollable View in Android? The removeView method does not work. Example:

scrollableView.addView(viewA);
scrollableView.addView(viewB);

scrollableView.removeView(1); //the page is not removed

Any ideas? Is this a bug or something?

You can also try like this:

var scrollableView = Titanium.UI.createScrollableView({
    views: [viewA, viewB],
})

scrollableView.removeView(1); //nothing happens. the page is not removed.

Hope someone has a solution.
Thanks

    This code will work only on iPhone :

    Parameters
view : Number/Titanium.UI.View
A Titanium.UI.View object (all platforms) or integer index (iPhone, iPad only) of a page to remove.

    for both to work you can use : 

    
scrollableView.removeView(scrollableView.views[1]);
