[Android] TextArea blur event never firing

Hi all,

I am using latest stable SDK and studio on Mac OS X (latest) and I have a window with a textArea.

When I am clicking on the textArea the focus event is fired and the keyboard pops up.

The strange part is that when I press the android back button to hide the keyboard, the textArea blur event does not fire, and thus I have no handle on that event!

Is there a way to get that event or is it impossible? I mean, I find it really annoying that Android has so many limitations, but this one too?? Seriously??
