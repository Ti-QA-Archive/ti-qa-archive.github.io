    • As stated in the setup guides, Android 2.2 (android-8) is required for the Titanium tooling. That requirement will likely be changed at some point. For now, you must install it. You don't need to build for that level. You can use a tag in your tiapp.xml to specify the tooling level, for example:

       <android xmlns:android="http://schemas.android.com/apk/res/android">
    <tool-api-level>14</tool-api-level>
 </android>

      Sorry if you don't agree with Stephen, but I'm afraid he's right.
