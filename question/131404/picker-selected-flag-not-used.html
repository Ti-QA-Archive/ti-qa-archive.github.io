Picker selected flag not used

I have created a simple picker

var months = [
    Ti.UI.createPickerRow({title:'Jan'}), 
    Ti.UI.createPickerRow({title:'Feb'}),
    Ti.UI.createPickerRow({title:'Mar'}),
    Ti.UI.createPickerRow({title:'Apr'}),
    Ti.UI.createPickerRow({title:'May', selected: true}),
    Ti.UI.createPickerRow({title:'Jun'}),
    Ti.UI.createPickerRow({title:'Jul'}),
    Ti.UI.createPickerRow({title:'Aug'}),
    Ti.UI.createPickerRow({title:'Sep'}),
    Ti.UI.createPickerRow({title:'Oct'}),
    Ti.UI.createPickerRow({title:'Nov'}),
    Ti.UI.createPickerRow({title:'Dec'})
];
var monthColumn = Ti.UI.createPickerColumn({
    rows: months
});
var picker = Ti.UI.createPicker({
    columns: [monthColumn],
    useSpinner : true,
    selectionIndicator : true
});

But the picker doesn't select the value I've told him to.
Using setSelectedRow directly on the picker object is not working too.

I've tried this on Ti 1.8.0.1 and 1.8.1 on iOS, neither of them works.
Any hints why?
The built-in date picker is working by the way.
