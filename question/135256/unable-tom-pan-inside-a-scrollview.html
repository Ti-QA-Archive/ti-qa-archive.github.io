Unable tom pan inside a scrollView

Hi all, In my application (android platform) I have a button. When this button is clicked, a scrollView is showing (double sized scrollView)

btn.addEventListener('click',function(e) {
     win.add(scrollView1);
     var t2 = Ti.UI.create2DMatrix(); 
       var scale = Titanium.UI.createAnimation();
       t2 = t2.scale(2);
       scale.transform = t2;
       scrollView1.animate(scale);
     });

Everything is working fine but the problem is that I can't pan inside the scrollView and here is the code of the scrollView

scrollView1 = Ti.UI.createScrollView({
    top:0,
    left:0,
    showVerticalScrollIndicator:true,
        showHorizontalScrollIndicator:true,
        contentWidth:'auto',
        contentHeight:'auto',
    zIndex:2002,
    height:'auto',
    width:'auto',
        }
    );

Thanks

  • Accepted Answer

    A scroll view will only perform the visual scrolling if the contents shown with the scroll view have a height or width (on android depends on scrolltype setting) exceeding that of the dimensions of the scrollview itself.

    I do not know what contents you have in the scroll view but if we use a very simple example;

    scroll view = 320 x 400 & total width and height of all child views in the scroll view is 320 x 350 then the scroll view will correctly show the contents but will not move up and down. If the contents total was a little bigger 320 x 500 then the bottom portion would initially be hidden and the scroll view would allows the scrolling interaction to move them into view and back again.
