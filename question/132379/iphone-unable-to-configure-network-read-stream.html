iphone unable to configure network read stream

I'm working on an app for an online radio station. To stream I use the Ti.Media Audioplayer. I stream a 128kbps mp3 audio stream. On Android and in the Iphone Simulator all is fine, but since a few days get on on my home network this error message if I try to start the radio stream on my IPod Touch 2G (4.2.1)

File Error: unable to configure network read stream

I tried to increase or decrease the buffersize, but with no success.
The strange thing is, when I'm in my office in my company where I'm also WIFI connected, all is running fine on the same device.

Any Idea what this could be?

Thanks in advance
Frank

  • I found out, if I do a test app only with this code it's playing. But not within my more complex app:

    var streamer = Ti.Media.createAudioPlayer();
streamer.url = "http://54house.stream-balancer.de";
streamer.start();
