    You'll need to modify the XML generated by the ActionBar Style Generator. You can't use the exact XML below, but it will give you an idea of the type of edit you'll make:

    <style name="MyActionBarTitleText"
     parent="@style/TextAppearance.AppCompat.Widget.ActionBar.Title">
     <item name="android:textColor">#000080</item>
</style>

    You'll need to insert the <item> tag noting the color to use into the appropriate style definition. The color notation is RGBA (RGB plus alpha/transparency).

    This guide from Google will provide more details.
