  • Here is all that I had to do to get things working consistently :

    1. Delete all AVD's

    2. Insure that you have the following Installed Packages (others may be ok, but I paired it down to only these)

    • Android SDK Tools, revision 10
    • Android SDK Platform-tools, revision 3
    • SDK Platform Android 2.2, API 8, revision 2
    • SDK Platform Android 2.1-update1, API 7, revision 2
    • SDK Platform Android 1.6, API 4, revision 3
    • Google APIs by Google Inc., Android API 8, revision 2
    • Google APIs by Google Inc., Android API 7, revision 1
    • Google APIs by Google Inc., Android API 4, revision 2
    • Google Market Licensing package, revision 1

    1. Now, from Titanium Developer, under "Run Emulator > Android" click "Launch"

    2. This is very likely fail, but it will create an AVD baseline for you to modify

    3. Go back the the AVD Manager, refresh the list of Virutal Devices and you should now see something like: "titanium_1_HVG…."

    4. Edit the AVD, make sure that the Target is "Google APIs (Google Inc.) - API Level 8

    5. Note, I used WQVGA400 but I think others will work … the key was to use the Google APIs not the Android 2.2 target.

    Now, return to the Titanium Developer and you should be able to launch and see it run.

  • I would like to point out that if you are getting this error and you are installing to a PHYSICAL DEVICE, the solution is to scale back the permissions that you are asking for in your AndroidManifest.xml file.

    You might be asking permission to use hardware that does not exist.
    This error is ultimately caused due to a permissions issue between the DEVICE (or emulator) and the permissions requested in the AndroidManifest.xml file

    I recently ran into this when attempting to build an app on android hardware that is not for sale publicly and defaults to chinese.
    If you work on this kind of hardware be mindful of what you are putting in your AndroidManifest.xml file.
    This issue can still be caused on Titanium_Mobile sdk v 3.0.0.
