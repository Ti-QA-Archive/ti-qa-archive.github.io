  • Manu

    First, close your emulator.

    For the Android packages you have installed, remove the obsolete package, and install all the Google APIs (except for API 9), available under Third party Add-ons / Google Inc.

    Then, delete all your existing AVDs, clear your project build directories, insure the project is using Titanium 1.7.X (bear in mind that you cannot run previous versions of the SDK, as you don't have Android 1.6 installed), and then launch your app.

    Let me know how you get on.

    Cheers
