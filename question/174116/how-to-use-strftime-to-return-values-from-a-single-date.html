how to use strftime to return values from a single date

hi,

my database outputs the following

id: 1| time: 2014-05-12 09:30:57| value: 5
id: 2| time: 2014-05-12 09:35:22| value: 9
id: 3| time: 2014-05-12 09:37:47| value: 5

"value" is user inputs.

I want to create an average at the end of each day, so total "value" then divide by the number of inputs "rows".

So i have some code that I'm trying to run to create a rowCount for dates that are today.

var rows = db.execute('SELECT time FROM logData WHERE strftime("Y-%m-%d", "now")');

but this doesn't seem to output anything. I've been messing around with strftime a bit but can never seem to get it to work correctly. any help appreciated!

Thanks

Mick

    great! worked it out….I needed 2 queries…

    var rows = db.execute('SELECT id, time FROM logData WHERE strftime("Y-%m-%d", time) = strftime("Y-%m-%d", "now")');
