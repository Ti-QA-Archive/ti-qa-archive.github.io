  • Hi,
    It depends whether you are going to target multiple platform? If its the case then definitely Titanium is the best option to go for. As it provides the flexibility to "write once and adapt everywhere". Also if you want to extra functionality to your app that is currently not supported by titanium but it does in native then you can create modules for the same. There is a way provided in titanium to extend the functionality by creating platform specific modules that can be used in titanium to add the functionality.

    http://docs.appcelerator.com/titanium/latest/#!/guide/Extending_Titanium_Mobile

    Hope it helps.
