  • hi,

    i think that question is android related so you should mention that in your question ans also add tag.

    you should use Ti.Filesystem.externalStorageDirectory to store such type of media and you can access them back with the same filename which is used to save that media.

    124534375887.png is the file name which is handled by device os to store images in gallery but you can see your saved image in your application's package folder.
