  • i will give you a little idea, use json to pass data to your webview.

    this is a little gift :D

    app.js
Ti.App.fireEvent('fromwebview',{data:JSON.stringify(myData), month:getMonth(), year:getYear()});

webview.html
Ti.App.addEventListener('fromwebview', function(e){
   var myData = JSON.parse(e.data);
}
