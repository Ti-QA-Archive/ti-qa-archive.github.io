    Though this should be fixed in the latest versions, there are two things to try if this fails:

    1. Project > Clean…
    2. Right-click on project > Properties > Project Build Path and make sure the correct SDK is checked.

  • I've a slightly better solution for the code completion problem. If you imported an existing titanium mobile project into Titanium Studio, it may not have got imported with the correct "Builder" settings. To fix this, close all Titanium Studio instances running on your machine. Go to the directory containing your project. For ex, ~/Documents/Titanium Studio Workspace/MyExistingProject/

    Open the .project file in that folder and add the following lines to the "BuildSpec" section of the xml.

        <buildCommand>
        <name>com.appcelerator.titanium.core.builder</name>
        <arguments>
        </arguments>
    </buildCommand>
    <buildCommand>
        <name>com.aptana.ide.core.unifiedBuilder</name>
        <arguments>
        </arguments>
    </buildCommand>

    Open Titanium Studio and voila! your project now supports code completion for Titanium SDK.
