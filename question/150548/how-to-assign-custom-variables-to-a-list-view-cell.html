How to assign custom variables to a list view cell?

So with the old tableviews I simply did the following to assign a custom variable 'mytitle' to a cell

row.mytitle ='My Custom Title';

And then when the 'click' event listener was fired for that row, I could access the variable via e.row.mytitle

However, for the life of me, I can't figure out how to do the same with the new listView structure. Any ideas?

  • Hi

    Without seeing your actual row code or event listener I cannot check your code or syntax for issues.

    But it is most likely the following, use this instead;

    e.rowData.mytitle

    If you want to see what you get back from the click event data add this line inside the event listener and see the results in the console.

    Ti.API.info('data: ' + JSON.stringify(e));

    All of this assumes the event uses e as the parameter of the listener function.
