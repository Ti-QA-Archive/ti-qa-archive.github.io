iOS Ti.Map (mapview) - Centered position while dragging?

Hi Guys,

I've successfully used the mapview to allow a user to choose a position on a map… However what I'd like to do is have a centered marker which remains in the center whilst you drag the map, showing you where it will land when you let go.

Has anybody achieved similar results?

I've tried using an annotation, but I can only seem to get it to generate in the centre when the user lets go

Thanks :-)

  • You could create a regular View with an image in it and have it float over the mapview, then remove or hide it when you drop a pin on the location.

    Be sure to set touchEnabled to false on the view, so that the user doesn't inadvertently touch it when dragging the map.
