getChildren in 1.8

The following code 

var win1 = Titanium.UI.createWindow({
    backgroundColor : 'red'
});
win1.add(Titanium.UI.createLabel({
    text:"text",
    color:"black",
    backgroundColor:"white"
}))
Titanium.API.info("output1 : " + win1.children[0]);
Titanium.API.info("output2 : " + win1.getChildren(0));
win1.open();

Outputs labels for both outputs in version 1.7.5 but in version 1.8.0.1 it returns nothing. Am I using these wrong? This was tested on a Motorola Droid 1.
