Access Ti.App.Properties from module

Hi
I'm developing an android app that receives push notifications. When my app receives push notification i want to check if a user account has been saved to the app properties. If account is not saved then i want to unregister the user's device from push notification service.
The user account has been saved using Ti.App.Properties and push notifications are received by native module. How can i read properties in native module?
Thank you.

  • I found answer.

    SharedPreferences pref = context.getSharedPreferences(TiApplication.APPLICATION_PREFERENCES_NAME,0);
boolean username = pref.contains("username");

    import org.appcelerator.titanium.TiApplication;
