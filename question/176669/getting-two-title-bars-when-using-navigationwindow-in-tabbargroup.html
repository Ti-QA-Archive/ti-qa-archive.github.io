    Hi kiran,

    this question has been asked many times, tabs have already a builtin in NavigationWindow.
    see this thread

    From the docs:

    On iOS, Mobile Web and Tizen, the tab maintains a stack of windows. On Mobile Web and Tizen, the tab is really just a NavigationGroup. Windows on top of the stack can partially or totally obscure windows lower in the stack. Calling open opens a new window on top of the window stack. When a window is closed, either by the user or by code, the window is removed from the stack, making the previous window visible. The root tab window cannot be removed.

    so the solution is to remove the NavigationWindow tag and use tab.open(newWindow) to open a new window in the stack
