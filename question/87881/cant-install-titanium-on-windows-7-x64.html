  • Thanks for the reply, Hal.
    I've solved the problem by myself a few days ago. Just have rebooted the system in a safe mode with a network support. And the Titanium was successfully installed. I guess the problem was in my ESET antivirus software, but the disabling of antivirus software doesn't resolve the issue when the Windows is in an ordinary mode. I think because the core process of antivirus left and is prevented the normal installation of Titanium.

    So the solution was, the first is to disable the antivirus software, the second is reboot the Windows in the safe mode and install the Titanium.
