Making clock application

I am working on a new project where i need the clock. So i didn't have any problems , until i thought of updating the clock every second.

So i made this:

function settime(){
if(currentTime.getHours()<13){
tekst.text=pad(currentTime.getHours(),2)+":"+pad(currentTime.getMinutes(),2)+" AM";
}else{
tekst.text=pad(currentTime.getHours(),2)+":"+pad(currentTime.getMinutes(),2)+" PM";
}}
setInterval(function(){settime();},500);

And it didn't worked. I've also tried:

function settime(){
if(currentTime.getHours()<13){
tekst.text=pad(currentTime.getHours(),2)+":"+pad(currentTime.getMinutes(),2)+" AM";
}else{
tekst.text=pad(currentTime.getHours(),2)+":"+pad(currentTime.getMinutes(),2)+" PM";
}
setTimeout(function(){settime();},500);
}
settime();

Any ideas why not working?
