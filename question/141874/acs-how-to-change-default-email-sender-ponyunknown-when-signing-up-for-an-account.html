ACS: How to Change Default email sender: pony@unknown when signing up for an Account

I got the SMTP working for users to register for a new account, with the settings I have, users must provide an email address (so they can also have the option to reset their passwords).

When I do get a confirmation email, the sender is: pony@unknown - Would it be possible to change the email address of the sender?

    • Joseph, been messing around with the ACS settings a bit more and got this to work. In SMTP settings set the 'Use Sender' radio button to true and another field appears for 'Sender's Email'.

      Put your send from email address in there and hopefully it should work for you. Did for me.
