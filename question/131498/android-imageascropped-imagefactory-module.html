Android imageAsCropped ImageFactory Module

I am trying to use the Appcelerator ImageFactory module to crop a fullscreen webview to a square and then resize it. The resize works fine, the crop is completely ignored.

I am using SDK 1.8.1 and version 2.0 of the module. My code is as follows:

            var imageData = tab2.toImage();
            if (width > height) {
                imageData = ImageFactory.imageAsCropped(imageData.media,{width:height,height:height,x:0,y:0});
            } else {
                imageData = ImageFactory.imageAsCropped(imageData.media,{width:width,height:width,x:0,y:0});
            }
            imageData = ImageFactory.imageAsResized(imageData,{width:(85 * scale),height:(85 * scale)});
            var f = Titanium.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory,'thumb2.jpg');
            f.write(imageData);
            tabthumb2.backgroundImage = f.nativePath;

Any ideas how to get this to work?

Cheers

  • OK, solved this after some more trial and error. Firstly I noticed this only effecte portrait images, landscape screenshots cropped fine.

    I guessed that the cropping didn't work if the crop size was full width and only reducing the height of the image.

    The code I tried, which works is as follows:

                if (width > height) {
                imageData = ImageFactory.imageAsCropped(imageData.media,{width:height,height:height,x:0,y:0});
            } else {
                imageData = ImageFactory.imageAsCropped(imageData.media,{width:(width - 1),height:(width - 1),x:0,y:0});
            }
