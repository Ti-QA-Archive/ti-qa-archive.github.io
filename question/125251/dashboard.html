  • I worked this one out. Had to go though the Titanium Studio source to find it.
    Follow these steps. It is for OS X, but should be simple to work out for other systems.
    Simply add the following line to the end of this file:
    /Applications/Titanium Studio/TitaniumStudio.app/Contents/MacOS/TitaniumStudio.ini

    -Dtitanium.dashboardUrl=nothankyou

    That said you can get it to open to your favourite page if you like with something like this:

    -Dtitanium.dashboardUrl=http://google.com
