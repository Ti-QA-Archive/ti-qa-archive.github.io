Unable to find iOS Simulator 6.1

Hello,

we've updated an application and it works fine on iOS7 and iOS8. Some of our costumers are still using iOS6 and have some problems. We'd like to use the iOS6.1 simulator in order to detect what is happening on their devices.

We've installed both Xcode 6.1 and Xcode 4.6.3, but we're not able to build the project using the iOS6.1 simulator.

We tried also using CLI:

            ti build -p ios -I 6.1 -C --project-dir /Users... --sdk 3.1.0.GA -S 6.1

But we got the following error:

What is the best solution/configuration (Xcode version, Ti SDK version,…) to develop Apps from iOS6 to iOS8?
