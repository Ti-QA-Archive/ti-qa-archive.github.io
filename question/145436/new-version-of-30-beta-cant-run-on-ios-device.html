New version of 3.0 beta can't run on iOS device

I installed the latest build of the 3.0 beta (3.0.1.201212041920) this morning and now Titanium can't install my app on iOS devices. The Titanium iOS Application Installer lists Available developer names including my name, but in the middle there's an error message:

[ERROR] : Unable to find an iOS Developer Certificate for "My Name (K**W)"

Is there something I can to to troubleshoot this?

  • The command line succeeds where the Titanium Studio Run Configuration fails. 

    titanium build -p ios -f -T device -F ipad -V "<My Name>" -P <uuid> -D test -b

    Then I open the generated XCode project and run on the device from XCode.

  • I found a solution here: https://jira.appcelerator.org/browse/TISTUD-2969.

    It basically says to download the latest build from build.appcelerator.net. I downloaded the latest build from the master branch (mobilesdk-3.1.0.v20121214142601-osx.zip) and that fixed the problem for me.
