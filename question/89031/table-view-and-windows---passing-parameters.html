table view and windows - passing parameters

Hi, if I have a table view and when a row is clicked I want to open a new window and pass a parameter about which row was clicked how do I pass such a parameter? The example code always seems to assume each row creates a different window.

— asked December 13th 2010 by Frank A

    In my experience, the easiest way to pass data to a new window is to add your own custom properties to it between the createWindow() and the open() – or as part of the createWindow() list of properties – then access those properties from the new window.

    See the section "Passing Data" in the docs for the Window object here.
