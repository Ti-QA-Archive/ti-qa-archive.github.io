  • Hi,

    I have the same bug here, it's really annoying.

    It seems like

    Cloud.Chats.getChatGroups({
        where : {
            participate_ids : Ti.App.Properties.getString('userId')
        }
    }, function(e) {
        if (e.success) {
            var group = e.chat_groups;
            o.success(group);
        } else {
            o.error();
        }
    });

    can be a workaround to this bug, but i'd rather a bug resolution.
