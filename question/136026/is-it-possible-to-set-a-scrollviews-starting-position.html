Is it possible to set a ScrollView's starting position?

Hi again,

I have a map of the UK inside a scrollview and would like it to be centred vertically on first launch.

Is there a way you can start a scrollview at a set scrolled height? So that my map will actually sit in the centre of the view and can be then scrolled up or down rather than just down.

This is on iOS only.

Here is my code….

var window = Titanium.UI.currentWindow;

var HeaderImage = Ti.UI.createImageView({
    image: 'images/home_babynames.png',
    width: 320,
    top:0,
    height:50
});

// Create the View
var MainView = Titanium.UI.createView({
   width:320,
   height:416,
   top:0
});

// Add View to the Window
window.add(MainView);

// Create a scroll view for the map area
var ScrollMap = Titanium.UI.createScrollView({
    showVerticalScrollIndicator:true,
    showHorizontalScrollIndicator:true,
    width:320,
    height:320,
    contentWidth:320,
    contentHeight:480,
    bottom:46
});

// Create the image view for the map
var TheMap = Titanium.UI.createImageView({
    height:480,
    width:320,
    image:'images/local_map.png'
});

// Add Scroll View to the main view
MainView.add(ScrollMap);

// Add the Map to the Scroll View
ScrollMap.add(TheMap);



// Add Image Views to the Window
MainView.add(HeaderImage);
