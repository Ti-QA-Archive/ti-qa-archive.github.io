Identify whether the app is installed for the first time

Hi there,

I am working on a cross platform app using Titanium Appcelerator.

I want my app to find out if the application is installed for the first time on that particular device.

How can I do it using Ti.App.Properties?

Can anyone tell how can I do this?

Thanks in advance!

  • Hello Shraddha,

    In app.js file just write code like this…

    if (Ti.App.Properties.getBool('firstTime', false) === false) {

    // Write Your authentification code here...


    Ti.App.Properties.setBool('firstTime', true);
}
