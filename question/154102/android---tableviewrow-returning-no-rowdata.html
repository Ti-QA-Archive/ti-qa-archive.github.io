Android - TableViewRow returning no rowData

I'm trying to get rowData from my tableviewRow but nothing return… it's like it are empty but it isn't.

var row = Ti.UI.createTableViewRow({
    height: 66,
    backgroundImage: 'imagens/atracaoBar.png',
    evento: json[i],
    teste: 'aaaaaa',
    top: 0
})

row.addEventListener( 'click', function(e){
    Ti.API.info( e.rowData )
    Ti.API.info( e )
    Ti.API.info( e.teste )
})

or even if I try to call the rowData from my tableview is the same problem…

tableview.addEventListener( 'click', function(e){
    Ti.API.info( e.rowData )
})

someone can help me?

this code works on iOS but fail on Android…
