How to include iOS Facebook installed tracking code in Titanium?

Dear Titanium comrades,

I would like to add the Facebook Mobile Ads tracking code to my Titanium (iOS) project
From Facebook, we have to add the following code snippet
[FBSettings publishInstall:FB_APP_ID];
to the selector - (void)applicationDidBecomeActive:(UIApplication *)application ).

I found that the selector - (void)applicationDidBecomeActive:(UIApplication *)application is in the TiApp.m file in /Classes/API folder once we compiled the Titanium project into the Xcode project.

However, when I added this line of code into the project, the Xcode project is complaining that I don't have the FBSettings class ("Use of undeclared identifier 'FBSettings.h'")

I tried to find a way to include the FBSettings.h to the TiApp.m but nothing works so far. I tried
downloading a FacebookSDK from developer.facebook (currently version FacebookSDK is 3.5) and added the library to my Resources folder. Once I added the FacebookSDK to my project, I can add the '.h' file import as #import <FacebookSDK/FBSettings.h>. However, when I built the project, it failed with the
following error:
/Library/Application Support/…/mobilesdk-3/mobilesdk/modules/iphone/facebook/3.1.1/libfacebook.a(FBNativeDialogs.o)
/Users/../build/iphone/FacebookSDK.framework/FacebookSDK(FBNativeDialogs.o)
ld: 351 duplicate symbols for architecture i386
clang: error: linker command failed with exit code 1 (use -v to see invocation)

I am assuming the new Facebook resources that I included is crashing with the existing one that Titanium already has.

I would be greatly appreciated if anyone can shed some lights on how to installed this Facebook mobile tracking code.

Thanks so much,
Kavin

Mine
Application type: mobile
Titanium SDK: 3.1.0 (having Facebook SDK 3.1.1)
Platform & version: building with iOS 4.3/5.0 Simulator
Titanium Studio: version, copied and pasted from Help / About
