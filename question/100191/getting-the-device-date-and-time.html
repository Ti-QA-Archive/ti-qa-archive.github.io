Getting the device date and time

I need to get the current date and time and attempted to use

var currentTime = new Date();

This is returning the date and time for GMT 0 which is not my device's correct time. For example, I am on the east coast.

Is there an API for retrieving this info?

    Just use normal javascript date handling as described here such as

    var msUTC = currentTime.UTC();

    to get the milliseconds since Jan 1 1970, in UTC time zone.

  • Hi ! you can use javascript functions curretTime.getHours()…. getMinutes(…) on one variable, like this:

    function getDate(){
    var currentTime = new Date();
    var hours = currentTime.getHours();
    var minutes = currentTime.getMinutes();
    var month = currentTime.getMonth() + 1;
    var day = currentTime.getDate();
    var year = currentTime.getFullYear();

    return hours +":"+minutes + " - "+day+"/"+month+"/"+year;
}
