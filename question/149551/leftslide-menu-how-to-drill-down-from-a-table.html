  • Hi

    The basic idea is that you open a new window as a child of a tab (or navigation controller).

    This is a simple example to show you how it is done, to test create a new default app and replace the contents of app.js with the following code;

    Ti.UI.setBackgroundColor('#000');

var win1 = Ti.UI.createWindow({  
    backgroundColor: '#fff',
    height: Ti.UI.FILL,
    title: 'Window 1',
    width: Ti.UI.FILL
});
var tab1 = Ti.UI.createTab({  
    icon:' KS_nav_views.png',
    title:' Tab 1',
    window: win1
});
var data = [
    { title: 'One', hasChild: true },
    { title: 'Two', hasChild: true },
    { title: 'Three', hasChild: true },
    { title: 'Four', hasChild: true }
];
var tbl = Ti.UI.createTableView({
    data: data,
    height: TI.UI.FILL,
    width: TI.UI.FILL
});
win.add(tbl);
tbl.addEventListener('click', function (e) {
    var newWin = Ti.UI.createWindow({  
        backgroundColor: '#fff',
        height: Ti.UI.FILL,
        title: e.rowData.title,
        width: Ti.UI.FILL
    });
    tab1.open(newWin, { animated: true});
});
var tabGroup = Ti.UI.createTabGroup();
tabGroup.addTab(tab1);  
tabGroup.open();

    The most important line there is this;

    tab1.open(newWin, { animated: true});

    Note it MUST have scope to the tab the current window is in and it opens the new window inside the navigation stack.
