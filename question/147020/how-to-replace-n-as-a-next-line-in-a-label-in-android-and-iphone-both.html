How to replace /n as a next line in a label in android and iphone both

Hi All

I am doing a mobile application, i m using Titanium SDK: 3.0.0 GA,
and my Deployment Targets is iphone, Android, and ipad.

I am gating a question in a label in this format,

Between /n 96,_____,98

how can i got the next line when /n occur in the question getting,for android and iphone both.

like this i want…

Between
96,_____,98

please help for this issue.. Thanks.

    Hi, here the code.

    var label = Ti.UI.createLabel({
    top:10,
    left:10,
    height:50,
    width:300,
    text:"Between"+ "\n" +"96,_____,98"
});

    output

    Between
96,_____,98

    • then use replace method and edit your string like this.

      var str = "Between /n 96,_____,98";

var newSTR = str.replace("/n","\n");

var label = Ti.UI.createLabel({
    top:10,
    left:10,
    height:50,
    width:300,
    text: newSTR
});
