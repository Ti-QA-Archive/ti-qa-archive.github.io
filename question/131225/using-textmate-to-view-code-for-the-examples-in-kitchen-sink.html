Using textmate to view code for the examples in Kitchen Sink.

I'm new to developing in general, and not an expert in java, so I've been trying to figure out how to show the code for the examples given in the Kitchen Sink application in textmate. I saw on an appcelerator video that this was possible, but I can't find out how. Also, is there a free alternative to textmate that will provide this service as well?
