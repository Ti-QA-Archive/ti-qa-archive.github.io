Retain textfield value between parent and child windows(screens)

My parent window and child window belongs two different js files, both have some different textfields. From Parent window, fill a textfield A with value 'xxxx', then click the right navigation button on upper right to the child window, and fill a textfield B with value 'yyy'; Then click the back button to enter the parent window, I see 'xxx' is still there, but click right nav button to the child window again, the value 'yyy' is gone.
Is there a way to keep the value I filled in both window without much effort?

    Hi Bill,

    Every time you select the child view from parent view, it will always create a new instance every time for child view. Once you leave back to parent view from child view, the child wondow get closed and also losing the data as well.

    One way to do is to save temporarily in session. By session, I mean saving in Ti.App.Properties.
