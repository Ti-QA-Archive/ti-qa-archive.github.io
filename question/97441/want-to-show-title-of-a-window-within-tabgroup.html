Want to show title of a window within tabgroup

Hi. I'm currently developing for Android and iPhone, using Titanium. I have a query regarding the tabgroup UI object used for Android.

Basically I'm trying to display the window title, of a window which is a child of a tab. To make myself clear, the following code adds win1 to tab1:

tabGroup = Ti.UI.createTabgroup;

win1 = Ti.UI.createWindow({
   title:'Hello World',
   url:'hello_world.js'
});

tab1 = Ti.UI.createTab({
   window:win1
});

tabGroup.add(tab1);
tabGroup.open();

Ok, I'm typing this code off the top of my head, but its not difficult to get the idea. The problem is, the title of win1, which is "Hello World", doesn't display on the Android phone at all. It seems to be covered by the tabGroup which appears at the top of the screen. If you do a Ti.UI.currentTab.open(win2) for example, the title displays perfectly.

So my question is, is there a simple way to get around this? I have been tinkering over writing a custom tabGroup class which gives me more flexibility in terms of tabgroup placement, but I would really love to use Titanium's UI objects. I've already resorted to spending a load of time re-writing a custom tableview class to allow animations to be displayed on the Android, so I'm hoping to avoid further customization. I mean, whats the point of using Titanium if you end up writing your own classes for almost everything?

Would appreciate any form of help I can get on this.
