Android Back button functionality

I am working on a mobile application for both android and ios and would like to perform following functions when user click back button on android devices-

When user click the back button they should be taken to last screen in the stack. I am able to achieve this functionality by closing the current window. However if the user has keyboard opened, I want the back button to just close the keyboard instead of closing the window.

Is there a way in SDK where I can figure out if the keyboard is active or not? In my current application, I open the keyboard when a textarea gets focus. I also tried following bit of code to hide my keyboard…but it doesnt seem to work-

if(Titanium.Platform.osname == 'android') {
            //dont think there is a way to find if the keyboard is visible or not

                if(win.MainView.commentsTextBox.hasFocus)
                {
                    alert("focus")
                    win.MainView.commentsTextBox.blur();
                }
                else
                {
                    alert("not")
                    win.close();
                }

            });
        }
