Swipe event on window with TableView

Hi, I am using 3.1.3.GA sdk, Alloys and 2.3.3 Android Emulator. I have a created a window with two view one visible and the other one will be at the back of the first view when swiped it will be shown by moving the first view. To achieve this I added swipe event to the Window which results TableView to be very slow in scrolling and jerks. My sample code,

index.xml

<Alloy>
    <Window id="notificationWindow">
        <View>
            <View id="mainView">
                <Require src="header" id="headerWithBack"></Require>
                <TableView id="notificationTableView" onClick="updateNotification" onScroll="lazyLoad">
                    <FooterView id="tableFooter">
                        <View id="footerView">
                            <Label id="loadingIndicator" visible="false">Loading...</Label>
                        </View>
                    </FooterView>
                </TableView>
            </View>
            <View id="slidingView">
                            <View id="SlidingMenu">
                                <View id="test">
                                   <Label>Test</Label>
                               </View>
                            </View>
        </View>        
    </Window>
</Alloy>

index.js

notificationWindow.addEventListener('swipe', windowSwipe);

I ll load my custom rows when Table Loads it has only a label and if I add this swipe event TableView not at all scrolling smooth and If I remove it its moving nicely. I think each time when tableview swiped upwards/downwards, swipe event too triggers, so that it gets lagging. How to avoid this jerk behaviour. Thank you.

    • i just told to use swipe event for tableview instead of window.

      index.xml

      <Alloy>
    <Window class="container">
        <Require src="slidingMenu" id="sld"></Require>
    </Window>
</Alloy>

      index.js

      function openNewWin(direction){
if(direction == 'up' || direction == 'down')
return true;
    if(direction == 'right'){
        var anim = Ti.UI.createAnimation({
            left: '70%',
            duration: 100
        });
        $.tbl.tableView.animate(anim);
    }else if(direction == 'left'){
        var anim = Ti.UI.createAnimation({
            left: 0,
            duration: 100
        });
        $.tbl.tableView.animate(anim);
    }
}
var tbl = Alloy.createController("tableView",openNewWin);
$.index.add(tbl);

      tableView.xml

      <Alloy>
    <TableView id="tableView" onSwipe= "onTblSwipe">

      tableView.js

      var cb = arguments[0];

function onTblSwipe(e){
    if(cb){
        cb(e.direction);
    }
}
