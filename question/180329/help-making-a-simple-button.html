Help making a simple button.

Hi,

I'm new to mobile development, and new to Titanium. I have a simple and maybe stupid problem, but I'm unable to solve it.

I need to put a simple, typical button in a view. And I can't find the way to achieve the typical appearance of a button (background gray, you know, the typical button you can see in a web page or in a mobile app.

I write the following code to make the button:

var but = Titanium.IU.createButton ({tittle:"Button",height:48, widht:128})

I add this button to a view, the view to a window, and when the window opens, I only see a semitransparent rectangle with the word "Button" inside. If I do this in the Android Emulator, the letters foreground color is black, and the background is semitransparent, a little darker than the background of the window; if I do this in the Iphone emulator, the letters foreground color is blue and the background is completely transparent.

What have I to put in the code to get a typical button, with its typical appearance and colors?

    i write below some code for useful to create button for your requirement. in this code only changes in button layout (height,width,color,backgroundcolor,title) as per your requirement.

   title: 'Hello',
   top: 10,
   width: 100,
   height: 50,
backgroundcolor : 'blue'
});
