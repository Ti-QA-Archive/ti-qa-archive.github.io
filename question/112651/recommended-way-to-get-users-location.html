Recommended way to get user's location

We started off with the recommended route by using

Titanium.Geolocation.getCurrentPosition()

we then realized that only the location that is returned in the last known location rather than invoking the gps signal.

We tried the watch position & we time out after 15 seconds. But the users complain that it times out most of the time while other apps in the system are able to get the location.

What is the recommended route?

  • You should be call getCurrentPosition, but you should also have a listener setup to capture events when the location changes.

    As I stated above, if you post what you currently have, someone should be able to help you.

    // event will fire based on this value
// Ti.Geolocation.distanceFilter = 10;
Ti.Geolocation.addEventListener('location',function(e)
{
        var longitude = e.coords.longitude;
        var latitude = e.coords.latitude;
        var altitude = e.coords.altitude;
        var heading = e.coords.heading;
        var accuracy = e.coords.accuracy;
        var speed = e.coords.speed;
        var timestamp = e.coords.timestamp;
        var altitudeAccuracy = e.coords.altitudeAccuracy;

}
