WebView not floating up for Android

I have a window containing a Header, Footer and a WebView. All the three views has fixed sizes in accordance to the device screen size. In my webview, I am opening an HTML form which the user has to fill up. The problem is that, if a user want to enter something into a textbox situated at the end of the form and focuses the textbox, the virtual keypad opens which hides a part of the webview. So the user is not able to see what he is typing in the textbox. I want the textbox to float up when the user starts typing in it.

This is the code for the window:

HtmlForm1 = Titanium.UI.createWindow({
        });

This is to fix the header height:

var header_height=(pHeight-(pHeight/1.1))+pHeight/130;

This is to fix the footer height:

footer_height=header_height+(pHeight/15);

This is to fix the webview height:

web_height=pHeight-(header_height+footer_height+barHeight);

This is to fix the webview top:

var web_top=pHeight-(pHeight/1.1)+2.5;

Code for webview:

var web = Titanium.UI.createWebView({
            scalesPageToFit : true,
            backgroundColor : 'black',
top:web_top,
height:web_height,
url : Ti.App.formurl.burl
        });

Header code:

var header = Titanium.UI.createView({

            backgroundcolor : 'black',
            top : 0,
                    height:headerLength,

            backgroundImage : HomeNav2

        });

Footer code:

var footer = Titanium.UI.createView({


                height:HomeFooter2_height,
                width:pWidth,
                top:footer_top,
            backgroundImage : HomeFooter2

        });

Adding header,footer and webview to window:

HtmlForm1.add(web);

HtmlForm1.add(header);
HtmlForm1.add(footer);
