TARGET_BUILD_DIR error in Xcode 4.2

I'm getting the TARGET_BUILD_DIR incorrectly set error in Titanium after upgrading XCode to 4.2. It's saying -

Expected dir /private/var/root/Documents/Titanium Studio Workspace/my_project/build/iphone/build/Debug-iphonesimulator, was: /var/root/Documents/Titanium Studio Workspace/my_project/build/iphone/build/Debug-iphonesimulator

In the previous version of Xcode I could go into project settings and change the build directory every time I did a build and everything worked fine. But now that I have upgraded, that option is not available.

I've tried all of the solutions I've read in StackOverflow and Appcelerator forums and nothing has worked.

How do I fix this problem so that I can go back to building in Titanium?

Thanks,
Peter.
