How to add 'phone' in Ti.Contacts.Person

Hi folks,

I'm experiencing some problem trying to define the phone property on a Titanium.Contacts.Person object in order to add it on the phone contacts.

I think I get it wrong cuz whathever syntax I'm using, it keeps crashing when I add the phone property.
I've used it in different ways:

Titanium.Contacts.createPerson({firstName:"Test", phone:{mobile:"01010101"}});

Titanium.Contacts.createPerson({firstName:"Test", phone:{"mobile":"01010101"}});

Titanium.Contacts.createPerson({firstName:"Test", phone:{"mobile":01010101}});

or even this

Titanium.Contacts.createPerson({firstName:"Test", phone:"01010101"});

It's weird because te other property works fine.
Thanks for the help!

    Try this

    var contact = Titanium.Contacts.createPerson();
contact.firstName = 'Peter';
contact.lastName = 'Griffin';
var phone = {"work" : ['01999 908777']};
contact.phone = phone;
contact.note = 'Some note text here';
Titanium.Contacts.save();

    Due to their being multiple types of phone in the contacts book you need to define what phone it is.
    Phone label could be any of these.

    home, work, other, mobile, pager, workFax, homeFax, main, iPhone

    The above example uses "work"

    Hope this helps.
