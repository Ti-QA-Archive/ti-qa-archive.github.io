    • what is the purpose of limit and skip parameter?

      limit : Number

      Instead of using page and per_page for pagination, you can use limit and skip to do your own pagination. limit is the maximum number of records to fetch. limit cannot be greater than 1000.
      skip : Number

      Number of records to skip. Must be used together with limit.
