WebView - Use same view - simply refresh with new URL

Hello,

I have a web view which i define as:

var graphWebView = Ti.UI.createWebView({
        url:'bar-basic.html',
        left: 100
});

On this web view Ive placed a button on which I have a eventListener.

On pressing this button I would like the view to be loaded with another url and the view reloaded.

How do i do this? I've tried the following but it doesn't work:

graphWebView.setURL = 'line-basic.html';
graphWebView.reload();

Thanks

    Hi

    You very close and on the right track.

    The "setters" are functions so you would require brackets like this;

    graphWebView.setUrl('line-basic.html');

    If you wanted to use the syntax style you have - that is also possible and very similar;

    graphWebView.Url = 'line-basic.html';

    Note the URL is actually Url when used - first letter is uppercase all others are lowercase.

    Cheers
