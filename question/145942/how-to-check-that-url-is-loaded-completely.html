How to check that url is loaded completely?

Hi Everyone,
It may be a stupid question but I have to ask -:) . I am opening a url using webview with a button click everything is fine , My question is I am showing ActivityIndicator when click on button and i want to hide the indicator when url is loaded completely. I am doing it like this: 

var indicator = Titanium.UI.createActivityIndicator({

    left:10,
    right: 10,
    top:10,
    width:Ti.UI.FILL,
    height: Ti.UI.FILL,
    message: 'Loading Please Wait!......',
    font: {fontSize: 30, fontWeight: 'bold'}

});

win1.add(indicator);
btnLogin.addEventListener('click', function(e) {

    indicator.show();
    var webview = Titanium.UI.createWebView({
        url : 'http://afghanskills.com/login.php'
    });


    win1.add(webview);
    indicator.hide();

});

    Try this:

    webview.addEventListener('load', function() {
    indicator.hide();
});

    And tell me if it helped.
