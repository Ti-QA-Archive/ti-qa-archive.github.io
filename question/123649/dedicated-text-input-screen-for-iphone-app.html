Dedicated text input screen for iPhone app

Hi,

I hope this isn't obvious, I can't seem to find a clean way of doing it…

You know in the Facebook app when you click 'update status' it opens a full-screen text input with a keyboard and 'send' button in the toolbar. Is that just a modal window with a big textarea and auto-focus, or is there a separate object type for a screen dedicated to composing text?

Thanks

    It's just a window with a big textArea (with a border radius or background) in it, which focuses the area on opening (which then pulls up the keyboard).

    It's not necessarily a modal window (which fly up from the bottom afaik), but simply a window which fades in as it opens.

    Also there's a 2D scaling on the textArea (or the background, if that's what it is) as it open too.

    All pretty easy to accomplish in Titanium. ;)
