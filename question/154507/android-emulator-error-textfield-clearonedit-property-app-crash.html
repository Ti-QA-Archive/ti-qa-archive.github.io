  • Hello Alessandro,

    This looks like a bug in the alloy framework. The error description: "java.lang.ClassCastException: java.lang.String cannot be cast to java.lang.Boolean" leads me to believe that when you assign clearOnEdit to "true", that value is getting passed back as a string where it should be a boolean. I will write up a bug report on this and submit it for fixing. Sorry for the inconvenience and thank you for bringing this to our attention. I will respond here with the ticket number so that you may put a watch on it.

    Best,

    Carter
