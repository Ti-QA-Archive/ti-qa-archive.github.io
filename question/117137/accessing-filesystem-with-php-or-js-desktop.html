  • Very Simple. The API Docs will answer most of your questions. Just look at all the apis and your good to go.
    Look at the filesystem API. http://developer.appcelerator.com/apidoc/desktop/latest/Titanium.Filesystem-module

    I was able to come up with this. Your going to have to modify it.

        var homeDir = Titanium.Filesystem.getDocumentsDirectory();
    var file = Titanium.Filesystem.getFile(homeDir);
    if (file.exists()) {
        var fileList = file.getDirectoryListing();
     for(x in fileList){
            throw(fileList[x]);
        }
 } else {
  alert("Error");
