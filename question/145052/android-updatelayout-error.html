  • Instead of using updateLayout, you can set directly the backgroundImage property to the APP.ROOT object.

    APP.ROOT.backgroundImage = App.getImagesPath()+'mainBg.png';

    Also, if you want you can add all these properties directly in the object creation, something like this:

    APP.ROOT = Ti.UI.createView({
   width : Ti.UI.FILL,
   height : Ti.UI.FILL,
   layout : "vertical",
   backgroundImage : App.getImagesPath()+'mainBg.png'
}
