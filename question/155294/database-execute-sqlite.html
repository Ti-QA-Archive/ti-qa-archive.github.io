  • If your three tables have these column names;

    CREATE TABLE "branches" (
    "id" INTEGER DEFAULT (null) ,
    "users_id" INTEGER);

CREATE TABLE "branches_form_types" (
    "branches_id" INTEGER DEFAULT (null) ,
    "forms_types_id" INTEGER);

CREATE TABLE "forms_types" (
    "id" INTEGER);

    (You probably have more columns but check that each table has the ones mentioned above)

    Then your query executes fine.
