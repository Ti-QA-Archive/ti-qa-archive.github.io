  • Accepted Answer

    Run ti info -t ios to see all of the iOS Distribution Certs that Titanium has found. Does the name of your certificate contain UTF-8 characters? What version of Titanium are you using? There was a bug with cert names and Titanium 3.0.0.GA that was fixed with 3.0.2.GA. If you're not on 3.0.2.GA, you should upgrade.
