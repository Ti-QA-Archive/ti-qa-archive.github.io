Efficient way to manage Map annotations

Hi,

I am trying to implement nearby restaurant for a given log/lat and the log/lat delta. This potentially could display hundreds of annotations for a busy city such as Chicago.

My question is there some good way for retrieving additional restaurant annotation for new regions on MapRegionChange and avoid getting the duplicated annotations for overlapped regions?

  • Hi,

    When there is a region change the best way to is to clear all the annotations that you displayed on the map. So that you don't overload the map with more and more annotations.

    There is underscore method that returns the values from array that are not present in the other arrays. which is difference. so that you can compare the new annotations array with the old annotation array.
