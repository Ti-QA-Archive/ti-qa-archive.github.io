In Titanium Android project to Iphone

Lets consider,I created an Android project in Titanium and now after several months I want to test this project towards the iPhone platform but I can't see how I can activate that project for iPhone.How much time will take it to this?

    In your project, open the XML for your tiapp.xml file. You should see a section that looks like this:

    <ti:app xmlns:ti="http://ti.appcelerator.org">
    <deployment-targets>
        <target device="mobileweb">false</target>
        <target device="iphone">false</target>
        <target device="ipad">false</target>
        <target device="android">true</target>
        <target device="blackberry">false</target>
    </deployment-targets>

    All you need to do is change false to true for any platform that you would like to support, provided you have the appropriate SDK installed. You will probably get a build error the first time you attempt to build since your project structure won't include build directories for newly added platforms. Don't worry though, just try it a second time and it should work fine.
