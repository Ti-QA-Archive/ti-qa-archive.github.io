iOS 7 shows both launch images

Migrating my iPad app to iOS 7 due to the new design I have created additional launch images for iOS7. They are named "Default7-Portrait.png" and "Default7-Portrait@2x.png" while the original iOS 6.1 and below images are named "Default-Portrait.png" and "Default-Portrait@2x.png". Building with SDK 3.1.3.GA and iOS 7 the simulator first shows the correct iOS 7 launch image and then - before the first app screen appears - it displays the other iOS 6 launch image.

I added the new images to the tiapp.xml in the ios-plist-dict-section as:

 <key>UILaunchImages~ipad</key>
<array>
        <dict>
                <key>UILaunchImageName</key>
                <string>Default7-Portrait</string>
                <key>UILaunchImageMinimumOSVersion</key>
                <string>7.0</string>
                <key>UILaunchImageOrientation</key>
                <string>Portrait</string>
                <key>UILaunchImageSize</key>
                <string>{768, 1024}</string>
        </dict>
</array>

Is this a bug in SDK 3.1.3.GA or iOS 7 or is my configuration wrong?

Michael

  • To answer my own question, it is the TiRootViewController.m which uses the Default.png (and other variations like Default@2x.png) as the root view image which shows up before the first real app view opens. So using special launch images in iOS 7 does not work as long as the TiRootViewController.m displays something else as well. I modified the TiRootViewController.m and it now uses Default7.png, Default7@2x.png, etc. in case of iOS 7.

    • Sure, but please beware, I am not an Objective-C coder, so the solution is stupid. I am sure this can be done better without all the IFs and some more generic string concatenation, but it suits my needs as a hotfix.

      First I extract the major of the os version:

      NSArray *osVersion = [[UIDevice currentDevice].systemVersion componentsSeparatedByString:@"."];
int osMajor = [[osVersion objectAtIndex:0] intValue];

      Then for each line dealing with a variation of the Default.png I added an IF-statement like this:

      if (osMajor == 7) {
    image = [UIImage imageNamed:@"Default7-Portrait.png"];
} else {
    image = [UIImage imageNamed:@"Default-Portrait.png"];
}
