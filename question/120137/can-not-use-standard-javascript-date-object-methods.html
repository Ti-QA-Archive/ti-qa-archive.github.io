Can not use "STANDARD Javascript Date object methods"

Hi,

I'm currently working on Date stuffs, but Date object in Titanium seems doesn't work properly. Please help me out.

The problem is that I'm try to use standard javascript Date object methods stated in this page: w3schools i.e. getTimezoneOffset(), but the method is not available (not even see in the available method list)

Here's my sample code of create date object from Date class

var currentTime = new Date();
var gmt = currentTime.getTimezoneOffset();

I've tried to do like

currentTime.

and press Ctrl + Space to get available property and methods, most of Date standard methods are not there. Wonder why ? or did I do something wrong ?

P.S. What I really try to do are

  1. Get the week start date - End date of week from current date
  2. Get current local device time and convert to GMT+0

Date stuffs are pulling my hairs out.

Regards,
Dave

  • Problem solved !

    Titanium Studio is using Date JS Core specification v.1.1, that's why the later version methods are not showing in assist list. However, you still able to use them fine.
