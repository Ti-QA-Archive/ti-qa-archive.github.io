Adnroid Context in titanium module

The default constructor in Titanium module is

public ModuleName() {
super();
}

and its beling called in any titanium app as
var module = require('com.module.xyz');

I need to get the Android Application context in The Titanium Module Class.

I tried below construtor (after Removing the default one), but seems like this constructor ie never getting called.
public ModuleName(TiContext tiContext) {
super(tiContext);
}

Can someone give hint how we can get the Android Context in Titanium Module main class?

    The constructor with the TiContext as parameter is deprecated.

    The way to retrieve the current context is done using:

    TiApplication appContext = TiApplication.getInstance();
