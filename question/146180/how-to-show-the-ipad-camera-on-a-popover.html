How to show the iPad camera on a popover?

I'm working on Titanium Studio 2.1 and developing for iOS, currently I'm interested in displaying the camera interface on a popover, the same way Facebook does on its iPad app, but so far I haven't found anything related to how to do this.

Does anybody know a way to display the camera in such fashion?

Thanks for any help in advance.

    Hi

    This works for me, add this to a button event listener and it will show the camera in a popover, you are not able to control the width or height of the popover just a general location based on the popoverView association.

    var btnCamera = Ti.UI.createButton({
    title: 'Camera'
});
btnCamera.addEventListener('click', function (e) {
    Ti.Media.showCamera({
        inPopOver: true, // set this to true
        popoverView: btnCamera, // "view" for the pointer arrow direction
        success: function (event) {
            alert('success');
        },
        cancel: function () {
            alert('cancel');
        },
        error: function (error) {
            alert('error');
        },
        saveToPhotoGallery: false,
        allowEditing: false,
        mediaTypes: [
            Ti.Media.MEDIA_TYPE_PHOTO
        ]
    });
});
win.add(btnCamera);

    The code is tested as working.
