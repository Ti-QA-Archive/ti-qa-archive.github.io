  • Accepted Answer

    Your scrollView should be like this:

            var scrollView = Ti.UI.createScrollView({
            id : i,
            // contentWidth : 'auto',
            // contentHeight : 'auto',
            showVerticalScrollIndicator : false,
            showHorizontalScrollIndicator : false,
            // top : 0, left : 0, width : '100%', height : '100%',
            backgroundColor : 'transparent',
            minZoomScale : 1,
            maxZoomScale : 15,
            zoomScale : 1,
            oldZoom : 1
        });

    Comment out contentWidth&#x2F;contentHeight should fix above error. But without commenting out top&#x2F;left&#x2F;width&#x2F;height, there's no image be rendered - at least with my test code.

  • So, got it working.

    the clue was, to set the height and width 1dp lower than the parent scrollable view.

    here's my code. hopefully, it helps someone

    pictures.photosView = Ti.UI.createScrollableView({
    width : 320,
    height : 367,
    contentWidth : 320,
    contentHeight : 367,
    top : 0
});

var lastItem = null;
for (i = 0; i < galleryImages.length; i++) {
    lightapps.basic.log(galleryImages[i].url);
    if (galleryImages[i].url) {
        var imageView = Ti.UI.createView({
            backgroundImage : galleryImages[i].url
        });

        var scrollView = Ti.UI.createScrollView({
            id : i,
            contentWidth : 319,
            contentHeight : 366,
            showVerticalScrollIndicator : false,
            showHorizontalScrollIndicator : false,
            backgroundColor : 'transparent',
            minZoomScale : 1,
            maxZoomScale : 15,
            zoomScale : 1,
            oldZoom : 1
        });
        scrollView.add(imageView);
        if (i == 0) {
            lastItem = scrollView;
        }

        pictures.viewArray.push(scrollView);
    }
}
