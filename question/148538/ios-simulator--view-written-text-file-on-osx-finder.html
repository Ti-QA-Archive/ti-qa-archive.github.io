IOS Simulator + view written text file on OSX finder

Quick question - If I've written a text file from within my app' . Should I not be able to see it saved on my OSX filesystem in finder.

    Your app should be deployed to

    Users/<your-login>/Library/Application Support/iPhone Simulator/<version>/Applications

    Inside here is a folder for each App. You should find your app here and the file will be saved within the /Documents/ folder inside
