setSelectedRow in multi column picker.

I have a two-column picker where the both columns depend on each other.

When you move column 1, column 2 must be set to the same rowIndex and vice versa.

I tried this:

picker.addEventListener('change',function(e)
{
    if(e.columnIndex == 0)
    {
         picker.setSelectedRow(1, e.rowIndex, true);
    }
    else
    {
         picker.setSelectedRow(0, e.rowIndex, true);
    }
});

What happens is whenever a column is changed, it will trigger the method, which will move the other column, triggering the method again and again forever, eventually crashing Titanium and the Simulator.

How can I fix this?

Thanks in advance!
