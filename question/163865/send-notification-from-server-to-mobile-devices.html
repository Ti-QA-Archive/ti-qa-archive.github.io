  • Hello Touil,

    If you want to send messages from your server to mobile apps then you need to use Push Notifications. Push notifications allow you to send alerts to devices that have your application installed on them, such as notifying the user it is time to download an update or it is their turn in a game. ACS is able to send push notifications to either Android devices using Google Cloud Messaging (GCM) or iOS devices using Apple Push Notification service (APNs).

    Please read following documents link for more help about Push Notifications.

    1. Push Notifications Guide
    2. ACS Push Notifications
    3. Push Notications Example
