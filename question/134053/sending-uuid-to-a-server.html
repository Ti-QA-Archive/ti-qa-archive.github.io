sending uuid to a server

hi, can anyone help me with the proper way to send uuid to a webserver?
I create uuid with 

var uuid = Titanium.Platform.createUUID

I'm trying with

xhr.open('POST','http://myserver/savedata');
xhr.send("id=" + uuid)

but that just sends: function createUUID() { [native code] }.
When i do alert(uuid) i see the id correctly.
