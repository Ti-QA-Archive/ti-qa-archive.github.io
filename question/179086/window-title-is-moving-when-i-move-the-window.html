Window title is moving when I move the window

I'm still very new to Titanium but I'm trying to make a simple "hamburger" or "slide-out" menu. I have a tableview for the menu and I plan on having the window for the selected menu placed overtop the menu, then have the window slide over once the menu button is selected. I am using Alloy to achieve this and I'm targeting iOS 8 on an iPhone.

xml

<Alloy>
    <NavigationWindow class="container" platform="ios">
        <Window id="redWindow" title="Red" backgroundColor="red">
            <LeftNavButton>
                <Button id="menuButton" onClick="menuButtonClicked" title="Menu" toggle=false/>
            </LeftNavButton>
        </Window>
    </NavigationWindow>
</Alloy>

js

function menuButtonClicked(e) {
    if(e.source.toggle == true) {
        win.animate({
            left:0,
            duration:300,
            curve:Ti.UI.ANIMATION_CURVE_EASE_IN_OUT
        });
        e.source.toggle = false;
    } else {
        win.animate({
            left:150,
            duration:300,
            curve:Ti.UI.ANIMATION_CURVE_EASE_IN_OUT
        });
        e.source.toggle = true;        
    }
}

My problem is that when I slide out the window upon clicking the "menu" button, the "Red" title moves so that it stays on screen. I want it to stay in the middle of the titlebar even if it partly goes off screen. I've looked in the documentation and I can't find anything that would lock the alignment or position of the title.

Any help is appreciated! Thanks!

    Hi

    I had a similar issue a while back, I think it was related to the width of the window. With my issue you had to give the window an absolute width as the contents of the window seemed to constrain themselves to what was visible on the screen.

    $.redWindow.width = Ti.Platform.displayCaps.platformWidth;

    Hopefully this will set the window to the devices width.

    Hope this helps
