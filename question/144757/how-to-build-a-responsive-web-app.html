How to build a responsive web app

Hi, there

I am developing a mobile web app and have deployed it to the server thanks to Chris's help.

The app stretches the whole page in the local browser and have different looks on different mobile phones. I am trying to make it responsive and could fit different screen sizes.

I know I could proportional value to set the width and height, instead of the absolute value. Does anyone know any other techniques to achieve the responsive interface?

Sorry for posting a lot of questions recently, I just started using Titanium. It turns out to be a really great experience, just still many stuff to explore.

Thanks in advance!
Best,
Phil

  • I haven't looked into building web apps with Titanium yet but for websites/web apps in general use percentage width values with min-width/max-width values (which can be fixed) so it will size itself to big enough screens but if the screen is too small it will stop sizing and enable horizontal-scrolling (not so nice, but better than certain elements being un-readable or if floated, going over-top of each other.
