  • 1) Check to see if the simulator is running by looking in the Task Manager

    2) Be sure when you look at the "Run COnfiguration" that you have a simulator selected

    3) Verify on the Properties Page that you have a BB SDK Selected

    4) If there is something called "fledgeXXX".exe running in the Task Manager, kill the process

    There is a long list of "challanges" with the beta BB SDK but you can make it work… We did!
