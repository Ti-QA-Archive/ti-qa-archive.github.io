Cannot Disable Wrap of Label

Hi there-

This should be easy, but for some reason, I cannot get it to work.

I have a label, and I am simply trying to disable the wrapping in an iOS device. No matter how I have set the styles, when the text property oversteps the width of the label, the text wraps within the TableView. Has anyone else hit something similar?

<Label class='myLabel' text='This is my label text which always seems to wrap within a table view' />

'.myLabel' : {
 width : 200,
 left : 5,
 layout: 'horizontal',
 horizontalWrap: false
 // I have also tried to set wordWrap to false and set a height but these have not worked 

}

    This works and also 'auto' ellipses by itsself

    var win = Ti.UI.createWindow({
    backgroundColor : '#000'
});

var t = Ti.UI.createTableView({
    backgroundColor : '#000',
    minRowHeight : 40
});

var r = Ti.UI.createTableViewRow({
    height : Ti.UI.SIZE
});

var l = Ti.UI.createLabel({
    text : 'This is my label text which always seems to wrap within a table view',
    color : '#FFF',
    left : 20,
    width : 200,
    height : 40
});
r.add(l);

t.data = [r];

win.add(t);

win.open();
