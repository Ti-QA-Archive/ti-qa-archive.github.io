  • Accepted Answer

    There is an Appstore API:

    https://www.apple.com/itunes/affiliates/resources/documentation/itunes-store-web-service-search-api.html

    Usage:
    https://itunes.apple.com/search?term=Skype&country=us&entity=software

    This returns the current version. You can check this value agains Ti.App.version
