    Hi

    Replace the Button with following code

    var star = Ti.UI.createButton({
        id : i,
            backgroundImage:'/images/star_black.png',
            height:'40px',
            width:'40px',
            backgroundSelectedImage:'/images/star_black.png',
            data:0,
            backgroundDisabledImage:'/images/star_black.png',
            top:'5px',
            right:10
        });

    Also in click listener method you can access the value :

    Ti.API.info(e.source.id);

    Let me know, if it works :)
