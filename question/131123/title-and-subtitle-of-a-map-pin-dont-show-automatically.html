Title and Subtitle of a map pin don't show automatically

The situation is pretty weird. I have a simple map view. A map pin should drop on the map when the map view is launched. My code is exactly the same as that in Kitchensink. But only a pin shows up, and every time I have to touch the pin once to make the title and subtitle of the pin show up. Same situation in both simulator and a real device.

Did I forget some code? Thanks. 

var pizzahut = Titanium.Map.createAnnotation({
    latitude:32.74411,
    longitude:-85.38993,
    title:"PizzaHut ",
    subtitle:'My favorite pizza place',
    pincolor:Titanium.Map.ANNOTATION_RED,
    animate:true,

});

//
// CREATE MAP VIEW
//
var mapview = Titanium.Map.createView({
    mapType: Titanium.Map.STANDARD_TYPE,
    region: {latitude:32.74411, longitude:-85.38993, latitudeDelta:0.01, longitudeDelta:0.01},
    animate:true,
    regionFit:true,
    userLocation:true,
    annotations:[pizzahut]
});

Titanium.UI.currentWindow.add(mapview);

    Yes, you need to select the annotation AFTER you show the map.

    http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.Map.View.selectAnnotation-method.html

    
Titanium.UI.currentWindow.add(mapview);
mapview.selectAnnotation(pizzahut);
