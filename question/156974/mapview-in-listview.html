MapView in ListView

Of course the code without map staff works fine.

    Map Views are not supported in ListView. We're working on better documentation to list exactly which UI components are and are not supported in ListView.

  • Hi,

    The XML tag for Ti.Map.Annotation is Annotation. You should define the tag as follows (just an example)

    <Annotation id="mountainView"
              latitude="37.390749" longitude="-122.081651"
              title="Appcelerator Headquarters" subtitle="Mountain View, CA"
              pincolor="Titanium.Map.ANNOTATION_RED"
              leftButton="/images/appcelerator_small.png"
              myid="1"/>

    instead

    <Ti.Map.Annotation id="mountainView" bindId="mountainView"/>
