Weird error, no idea what to do

Hi!

I am making an app that enables people to order food with their android phones. I have been working on it for quite some time now, and I have made a working app but the coding kind of hurts your eyes since it contained code for testing purposes, as it is my first app.

I am currently rewriting the app, making it a bit more readable. But, for some reason, these errors keep popping up whenever I get past the first window: 

[ERROR][TiJSError(  790)] (main) [1,22068] - In ti:/window.js:321,34
[ERROR][TiJSError(  790)] (main) [1,22069] - Message: Uncaught TypeError: Cannot call method 'extend' of undefined

I have been able to find the block of code that seems to trigger the errors, but I can't find anything wrong, as I have used the exact same code before (I have replaced some values with foo and bar):

}

I am not asking you to write my code, I am simply asking that you push me in the right direction as I want to learn from my mistake.

Thank you in advance,

  • Accepted Answer

    Fixed it myself. Replaced this…

    Ti.UI.currentWindow.activity.onCreateOptionsMenu = function(e) {

    with this…

    var activity = Ti.Android.currentActivity;
activity.onCreateOptionsMenu = function(e) {

    Seems to me like I shouldn't append activities in the current window. Lesson learned.
