    hi,
    try this way…
    Set the windows layout as vertical and Height = Ti.UI.SIZE and Width = Ti.UI.SIZE
    and Views style as,
    "#view1":{
    height:Ti.UI.SIZE
    }

    "#view2":{
    height:Ti.UI.SIZE,
    top:"50dp"//here spacing between two views is 50dp
    }
