    Try using the "feed" feature:

    select * from feed where url='http://www.stationcaster.com/stations/wrks/rss/?c=3041

    • Well, if what you want to do is use YQL to facilitate the process of extracting data from that feed, then that query produces the following JSON feed:

      http://query.yahooapis.com/v1/public/yql?q=select%20*%20from%20feed%20where%20url%3D'http%3A%2F%2Fwww.stationcaster.com%2Fstations%2Fwrks%2Frss%2F%3Fc%3D3041'&format=json&callback=

      However, it's an overkill to parse an RSS with YQL because RSS is already "parse-able" You should grab it as-is and extract data from it. The KitchenSink has an example: https://github.com/appcelerator/KitchenSink/blob/master/Resources/ui/common/platform/xml_rss.js
