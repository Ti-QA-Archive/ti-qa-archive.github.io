How to fill a tableViewRow with another view to 100% width?

Hi folks,

I am trying to add a view to a tablerow that fills it from left to right (mainly because I want some picture in it). But it seems that there is always some margin that cannot be covered. See the minimal example below which I would expect to not show a single pixel red background. But the result is actually that I get around 5 - 7 pixels red on the left and right sides of the table row.

I'm using Titanium Mobile 1.7.2 with an Android 2.2.1 device.

Anyone here can help me to get this to work?

/Andreas

Test case, app.js :

Titanium.UI.setBackgroundColor('#000');

var win = Titanium.UI.createWindow({  
    exitOnClose: true,
    fullscreen: true,
    orientationModes: [Ti.UI.PORTRAIT],
    backgroundColor: 'green'
});

rows = [];
for(i = 0; i < 20; i++) {
    rows[i] = Ti.UI.createTableViewRow({
        height: 40,
        backgroundColor: 'red'
    });
    var view = Ti.UI.createView({
        width: '100%',
        height: '100%',
        backgroundColor: 'orange'
    });
    rows[i].add(view);
}

var table = Ti.UI.createTableView({
    top: 40,
    left: 0,
    right: 0,
    bottom: 0,
    data: rows
});
win.add(table);

win.open();
