  • Hi

    The PayPal module works for UK based accounts, although annoyingly it does not allow credit card payments without signing in.

    PayPal appear to be making massive changes to the back end services that all appear to be US only availability right now.

    My current solution is host the PayPal web page in a webView, not ideal but works, which is an important factor, I could use the module but lack of CC support is a big problem with the module.

    I can only hope that stripe.com speeds up it international rollout as it is the opposite of PayPal; it likes its customers, it's rates are reasonable, it does not lock you out of your account for stupid reasons, it likes developers, it provides many many ways of integrating it to your, site and app.

    We will jump to stripe.com in a heart beat, which is the sentiment of so many respected developers and businesses.
