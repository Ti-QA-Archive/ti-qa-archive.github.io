How can I delete a Calendar Event that I set earlier in Android

I have the following code for setting an event and a reminder.

var eventBegins = new Date(2012, 01, 8, 12, 35, 0, 0);

var eventEnds = new Date(2012, 01, 8, 12, 55, 0, 0);
var details = {
    title: 'blahblah',
    description: "more blah blah",
    begin: eventBegins,
    end: eventEnds,
    hasAlarm:true
};

var event = calendar.createEvent(details);

// Now add a reminder via e-mail for 10 minutes before the event.
var reminderDetails = {
    minutes: 10,
    method: Ti.Android.Calendar.METHOD_DEFAULT
};
event.createReminder(reminderDetails);

I want to remove the event now. How do I do that?
