  • Hi Vinicius Oliveira,

    The default background color for KitKat is white and you are using a similar color to white. If you use the 

    backgroundColor: "#000"

    and 

    color:"#000"

    then you can have the text color to white like,

    var textField = Ti.UI.createTextField({
        paddingLeft : 10,
        height : 40,
        top : 10,
        width : 100,
        hintText : "Hint Text",
        color : '#fff',
        backgroundColor : '#000',
        borderRadius : 4,
        borderWidth : 1,
        borderColor : '#999',
        tintColor : 'black',
        keyboardType : Ti.UI.KEYBOARD_DEFAULT,
        returnKeyType : Ti.UI.RETURNKEY_DEFAULT,

    });

    Thanks
