How to make input box with datalist like HTML5 in Titanium?

Hi, I am trying to make a select input box like the following picture:

Basically it is a input box with some predefined datalist. As user input, the dropdown list will be filtered. However, it is still different from picker since it gives the user power to go beyond this list. (Yes, the list is more like a suggestion or short-cut if it matches something in the list) Imagine gmail app for example.

Basically, I have 2 ideas that might work:

  1. Using listview and textField as it's "search bar". Setting the textField on change event to be changing the searchText of listview can indeed help me with filtering. However, the drawback of this method is listview will fill the whole area and it's height cannot be changed dynamically even when the data is being filtered…I haven't yet find a method tell me how much results matches the criteria in the listview, so I think I might give up this idea.

  2. If abandoning listview, then I believe I can make a container view set to be Ti.UI.SIZE with vertical layout and fill it will a bunch of smaller view rows(just look like listview). It seems feasible.

However, the actual biggest challenge I am facing is how should I position the hint view when there is other UI items below that textField. I only want it pop-up when the textField is focused and it should cover whatever that was originally there without pushing the original view downward. I don't know whether there is absolute positioning to a certain view and it is possible in Titanium.

I really need some inspiration and appreciate any thoughts or possibilities:)
