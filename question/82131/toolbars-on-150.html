Toolbars on 1.5.0?

var search_page = Ti.UI.createWindow({
            title:'Search: '+searchBar.value,
            barColor:'#000'
        });
        var search_webView = Ti.UI.createWebView({
            url:'http://www.website.com/?filter=na='+searchBar.value+';ex=on;ma=1'
        });

        var goBack = Ti.UI.createButton({
            systemButton:Ti.UI.iPhone.SystemButton.REWIND
        });

        var search_toolbar = Ti.UI.createToolbar({
            items:[goBack],
            bottom:0,
            barColor:'#000',
            borderTop:true,
            borderBottom:false
        });

        search_page.add(search_toolbar);
        search_page.add(search_webView);

Using the latest build of 1.5.0, the Toolbar does not show at the bottom of this page. Does anybody know why?

Building on the latest 1.4.3 build works perfectly, it's just 1.5.0 that doesn't work. Did something change that I happened to miss?
