    • take a look in $HOME&#x2F;Library&#x2F;Application\ Support&#x2F;Titanium&#x2F;modules you will find a commonjs folder there.

      You can load your libraries from there as well. It all depends on how you want your work flow to run.

      If you use the commonjs modules folder, you will need to create the appropriate module files and version it in a directory.

    • manifest

      #
# this is your module manifest and used by Titanium

#
version: 1.7.2
description: Appcelerator Cloud Services
author: Tim Wood

# these should not be edited
name: moment
moduleid: moment
guid: 3475fdef-f568-4c6f-b5ea-1f213ec69ae3
platform: commonjs
minsdk: 2.1.0

      var Moment = require('moment');
