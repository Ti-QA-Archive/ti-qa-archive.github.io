Dear Appceleratorists:

    I currently have 3 different xcodes installed.

    One is in /Applications, this is the latest release through the app store.
    On my desktop are 2 more (both 5.x gen).

    I use xcode-select to bounce between them. Since I don't use Studio, my switching is very quick. Studio seems to require that you repoint to the xcode you want to use each time you change.
