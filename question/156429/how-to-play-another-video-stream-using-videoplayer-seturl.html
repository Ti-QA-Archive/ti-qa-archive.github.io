How to play another video stream using VideoPlayer setUrl

Hi I am developing an app for iOS 6 using 3.1.2GA SDK.

I currently have a commonjs module with a window and a videoPlayer that opens and plays a stream when a user clicks on a listitem. I am trying to avoid having to create new window and video player every time the user clicks on the listitem. So I have a reference to this object when the window that contains the listView is created.

When I try to load another stream using setUrl() using the same videoplayer object, it continues to load the previous stream. When I do videoplayer.release() it loads the specified stream but the built-in events for the videoplayer does fire.

Does anyone have any suggestions?

function MediaPlayerWindow()
{
    /**
     * UI: Video Player
      */
    this.audioPlayer = Ti.Media.createVideoPlayer({
        visible: false,
        autoplay: false,
    });
    /**
     * UI: Main Window
     */
    var audioWindow = Ti.UI.createWindow({
        title: 'Now Streaming',
        barColor: 'black',
        backgroundColor:'white',
        tabBarHidden: true,
    });
    this.getProxy = function() {
        return audioWindow;
    };
    this.audioPlayer.addEventListener('durationavailable', function(e) {});
    this.audioPlayer.addEventListener('playbackstate', function(e) {});
    this.audioPlayer.addEventListener('complete',function(e) {});

    return this;
}

MediaPlayerWindow.prototype.setNowPlayingData = function ( streamURL ) {
    /* this.audioPlayer.release() loads the new URL, but the event listeners does not fire when stream loads */
    // this.audioPlayer.release();
    // set url without release plays the 1st stream that was loaded before
    this.audioPlayer.setUrl(streamURL);
};

module.exports = MediaPlayerWindow;

Inside the Item click event handler

parentWin.mediaPlayer.setNowPlayingData( streamURL );
parentWin.containingTab.open(parentWin.mediaPlayer.getProxy(),{animated:true});
