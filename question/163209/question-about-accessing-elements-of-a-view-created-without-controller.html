Question about accessing elements of a view created without controller

Hey guys, first time developing a huge project and it's been a cool ride with Titanium, but now the hard parts, after reading the documentation my app has some user faces which i can preview in little modals, which i create without having a specific controller, just a view with a simple name label.

My function calls any modal like this, from index.js file

profilemodal(persons_id);

which goes to the function, in the same index.js

function profilemodal(userid){
    var profilewin=Alloy.createController('profilemodal').getView();
    $.profilewin.profile_name.text = getUser(userid).name;

    $.index.close();
    profilewin.open({transition:Ti.UI.iPhone.AnimationStyle.CURL_DOWN});
}

the corresponding alloy xml is this one, the problem is i cannot access this profile_name attribute, i've tried without the $, just with profilewin.profile_name.text

<Alloy>
    <Window title="Profile">
        <View layout="vertical">
            <Label id="profile_name">name</Label>
            </View>
        </Window>      
</Alloy>
