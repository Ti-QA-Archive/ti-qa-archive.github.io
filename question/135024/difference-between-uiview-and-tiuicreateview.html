    your ui.View() can be user defined..

    like you created a namespace ui, and inside that namespace you have a function View(). now to access your function inside namespace ui you do ui.View()

    whereas

    Ti.UI.View() creates a view for you. i.e it is a predefined API by titanium to create a view.
    Which will be a placeholder where you can add more subview to it..
