    This happens when creating a project? Does it happen every time? Was there more to the error message?

    What version of Node.js do you have installed? Titanium SDK 3.0.0.GA and 3.0.2.GA will only work with Node.js 0.8.X (download link: http://nodejs.org/dist/v0.8.22/). Node.js 0.10.X is only supported by Titanium 3.1.0 or newer (as of right now, 3.1.0 has not been released).

  • Hello I finaly find the solution of this problem :

    First type on cmd : 'ti info'

    (if you got an error in "throw e" go to this file and comment this line "//throw e;" )

    then check your version of jdk , and node and cli , you find all you need in this page http://docs.appcelerator.com/titanium/latest/#!/guide/Setting_up_Studio-section-37540095_SettingupStudio-CompatibilityandDownload

    then check the environnement variables they should be like this http://docs.appcelerator.com/titanium/latest/#!/guide/Software_Locations_and_Environment_Variables

    and restart titanium , and like magic you find the green ok on Android SDK
