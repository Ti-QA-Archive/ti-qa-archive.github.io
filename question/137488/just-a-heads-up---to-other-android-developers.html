    Note that the page you reference has moved to the new docs site - Installing the Android Development Tools.

    Although it's nice to be able to access the Android tools within the single Studio interface, bear in mind that people may want to avoid the extra installation process by running DDMS directly from the command line. It's available at &#x2F;path&#x2F;to&#x2F;android-sdk&#x2F;tools&#x2F;ddms.

    For viewing adb -e logcat output, I prefer using Colored Logcat with logcat-ti.sh.
