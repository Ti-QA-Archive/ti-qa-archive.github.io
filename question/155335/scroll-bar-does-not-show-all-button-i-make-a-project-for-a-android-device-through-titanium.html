  • Hi

    You have a few problems in your code and there is an easier way to achieve what you are after;

    • You have missed off most of the important scrollView properties
    • You misspelt bottom
    • You should use the layout property on the parent view, it makes it much easier to automatically position view views.
    • Child views of layouts use top, bottom, left & right as margins - this makes it easier to space things.
    • Your for loop was only returning 19 entries, but it looks like you were trying for 20 of them.

    This is an example of how you can improve your code, you will need to tweak the values to get it to what you want, but it is much easier to manage than what you have right now;

    var scrollView = Ti.UI.createScrollView({
    bottom: 20,
    contentHeight: Ti.UI.SIZE,
    contentWidth: Ti.UI.FILL,
    height: Ti.UI.FILL,
    layout: 'horizontal',
    showHorizontalScrollIndicator: true,
    showVerticalScrollIndicator: true,
    top: 33,
    width: Ti.UI.FILL
});
var intButton = 0, intButtons = 20;
for (intButton = 0; intButton < intButtons; intButton = intButton + 1) {
    scrollView.add(Ti.UI.createButton({
        bottom: 5,
        height : 45,
        left: 5,
        right: 5,
        title: intButton.toString(),
        top: 5,
        width: 45
    }));
}
