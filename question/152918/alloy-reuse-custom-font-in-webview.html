Alloy Reuse Custom Font in Webview

I put my custom font (otf and ttf) in assets/font and normally it works when I simply put its name into the tss file.

In order to make it available to webview I have to use this code in the html file

         <style>
            @font-face {
                font-family: "AlexBrush-Regular";
                src: url("../../fonts/AlexBrush-Regular.otf");
            }
        </style>

Where src path is relative from the html file to the assets/fonts folder. If I have many custom fonts in the folder, how to make it available by not specifying one by one like the above? Cant it work just like in tss file where it can detect all font family directly from asset/fonts?
