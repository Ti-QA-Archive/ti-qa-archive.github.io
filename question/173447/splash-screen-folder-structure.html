  • Ray,

    Thanks for the link. I followed the instructions but was still getting wonky and stretched images. I then marked each of them to make them recognizable (from the default image) and was able to see that titanium was using the right image.

    It came down to the one section in the instructions that I overlooked the first time through:

    Example maximum file dimensions for each density (change to suit your own needs)

    ldpi = 242×322
    mdpi = 322×482
    hdpi = 482×722
    xhdpi = 642×962

    It turns out that the images I was feeding the 9 patch generator were too wide, and getting squished regardless. I hope this helps others, the method works but take care in what size image makes up the center patch of your 9 patch.
