UI.View - Move up when keyboard with Toolbar appears

Hello,

I have a Titanium.UI.View with a number of text fields, when the keyboard appears the views move up accordingly so they are always on display.

But I have added a toolbar to the top of the keyboard and this is now covering the text fields. Is there anyway to manually control how much the view moves so I can added the width of the toolbar to it ?

Regards,
Stephen
