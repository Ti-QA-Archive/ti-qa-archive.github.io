App doesn't take code changes into account at launch

In my code, I'm targeting an XML element by its ID.

$.charactersRow.addEventListener('click', function(e) {...});

So, I change my syntax, trying something else in LIST.JS:

And… the project failed to build and shows me the precedent syntax error, just like if my code changes were not taken into account in LIST.JS…

This is strange… what can I do? I have already tried to rename my project…
