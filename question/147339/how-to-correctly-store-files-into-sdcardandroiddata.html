How to correctly store files into SDCard/Android/data?

Hi, in my Android project I download some files to my device SD: after download, I check my SD card and I find the app creates a directory called com.mydomain.myapp and inside there are my files with their proper names and fully working.

Now, I wanted to download the files elsewhere: they should be downloaded in my SD card under Android/data (that is: I want the com.mydomain.myapp under SDCard/Android/data and anything I must download is downloaded inside this folder).

So, I changed one line of code and this one (the destination folder):

Ti.Filesystem.externalStorageDirectory

became:

Ti.Filesystem.externalStorageDirectory+"Android"+Ti.Filesystem.separator+"data"+Ti.Filesystem.separator

After this change, if I check the Android/data directory I find my com.mydomain.myapp, BUT there's a cache folder and, inside this directory a _tmp folder: inside this last one, I find some of my files renamed (eg. tihttp24567574tmp) and some other missing.

What happened to my files? Why, if I directly target Ti.Filesystem.externalStorageDirectory I have no issue? What is the right path to correctly store my files into Android/data folder?
