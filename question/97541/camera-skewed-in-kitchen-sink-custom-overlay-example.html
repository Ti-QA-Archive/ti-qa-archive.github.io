  • Well, the bug wasn't squashed as I said 3 months ago. A clean of my build folder after the update tricked me. Anyways, you work around it by applying a transform to the camera: Ti.UI.create2DMatrix().scale(1)

    Like this:

    Titanium.Media.showCamera({

    success:function(event)
    {
        var imageView = Ti.UI.createImageView({
            image:event.media,
            width:photoView.width,
            height:photoView.height
        });
        win.add(imageView);
        Ti.Media.hideCamera();
    }
    overlay:overlay,
    transform: Ti.UI.create2DMatrix().scale(1),
    showControls:true,
    mediaTypes:Ti.Media.MEDIA_TYPE_PHOTO,
    autohide:false 
});
