Tableview Adjustments for Keyboard

I'm using 1.5.1 for an iPhone app. When the keyboard displays, I'm moving my textbox above it and trying to shorten my tableview and scroll to the bottom.

I'm able to chop the bottom off my tableview with the "bottom" property (height adjustment didn't work), but then attempting to scroll to the bottom via the scrollToIndex isn't working. I've hard-coded the index now (not knowing how to identify the highest index) and still it just chops w/o scrolling.

My hack of a workaround involves setting the "top" property of the tableview to a negative number. If that's the best I can do, that still needs to be sensitive to the current size of the table to determine how big the neg number should be.

My preference is to shorten the tableview and scroll to the bottom. All help greatly appreciated …
