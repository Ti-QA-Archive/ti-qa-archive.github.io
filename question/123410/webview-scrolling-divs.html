  • I add the following code on top of each html page to prevent it from moving and it disables the two finger scrolling as well:

    
    document.ontouchmove = function(e) { e.preventDefault(); return false; }

    Also my clients don't accept two finger scrolling because it's not consistent with older technologies which might create confusion for users.

    (By the way, it works)
    Another solution I tried is using the css for the iOS 5, its something like this: 

     -webkit-overflow-scrolling: touch;

    The problem with this was that it scrolled the entire page not the div only.
