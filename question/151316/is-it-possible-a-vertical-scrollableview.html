  • Hi

    Short answer is no, not using the scrollableView view.

    The property layout your code uses may appear to to give you this option, but it is not for that. The layout is for child views, whilst these are dependant views.

    A child view is one where one view is placed inside another view, for example a label inside a view.

    A dependant view is one that is required by the parent, for example a row to a tableView or in this case a view to a scrollableView.

    It is possible to create your own version using a combination of events and views, by there is no built in way to achieve it.
