Alloy: Toolbar at top list view

Hey all. I'm building an iOS app. I want to have a toolbar at the top of my search results. But I don't want it take up my screen real estate at the top as the user scrolls so I want it to scroll with the list view. I am using listview templates so I figured I could put a toolbar in a listview template and add it as the first item in the list. But I am getting an error thrown. It seems appcelerator doesn't like the toolbar within the listview item.

Here is my searchResults.xml file

<Alloy>
    <Window title="Searching">

        <ListView id="list" defaultItemTemplate="detailTemplate1">
            <Templates>
                <ItemTemplate name="headerTemplate">
                    <View bindID="headerView" id="headerView">
                        <Toolbar>
                            <Items>
                                <Button>Button1</Button>
                                <FlexSpace/>
                                <Button>Button2</Button>
                                <FlexSpace/>
                                <Button>Button3</Button>
                            </Items>
                        </Toolbar>
                    </View>
                </ItemTemplate>
                <ItemTemplate name="detailTemplate1">
                    <View bindID="itemView" id="itemView">
                        <View bindID="imgView" id="imgView">
                            <ImageView bindId="thumbimage" id="thumbimage"/>
                        </View>
                        <Label bindId="subject" id="subject" />
                    </View>
                </ItemTemplate>

            </Templates>

            <ListSection>

            </ListSection>
        </ListView>

    </Window>
</Alloy>

and I add that template at the beginning of my controller file before I iterate through my results: 

var moreItems = [];
moreItems.push({
    template: 'headerTemplate'
});

Any ideas why I'd get that error and any ideas how to fix it? any help is greatly appreciated.

thanks

Dave

    Yeah, you can't do that. ListView ItemTemplates support only the basic UI components. The details should be covered pretty well in the ListView guide: http://docs.appcelerator.com/titanium/latest/#!/guide/ListViews
