Global Variable - Advanced Topic

Hi,

I'm really stuck here.

I have table with a column in a database that requires a unique value (an integer).

In my app, a user adds a row in that table. Each row is meant to have a unique ID. The ID's are meant to start from 2000.

How exactly to I keep track of the row number so that the user, when entering many records, will have records each with unique ID's.

So, record 1's ID = 2000,
record 2's = 2001
record 3 = 2002
record 4 = 2003
record 5 = 2004 etc..

I was playing around with Global variables, but i came to a point where I need to know the latest value added to the table. How do I find that out?

If I could find out what the latest value was - I could simply add a 1 to it and use this value as the INSERT for the next row to be added.

Thanks,

  • IF you are looking for the last item in your table one solution is setting that ID in the row data:

    //The Array where you will place your Rows created when you perform the select on the DB
var TableViewData = [ ];

//when creating initial Rows you loop the results and run something like this. 
var myRow = Ti.UI.createTableViewRow({
    height:20,
    backgroundColor: '#999999',
    record: 2000 // THIS VALLUE IS FROM YOUR SELECT IN THE DB.
});
TableViewData.push(myRow);


//Later on When adding a new row. 
//Get the last item in the Data Array:
var lastRowIndex = TableViewData.length -1; 

//Create a new Row and add 1 to the last record in the array. 
var newRow = Ti.UI.createTableViewRow({
    height:20,
    backgroundColor: '#999999',
    record: TableViewData[lastRowIndex].record +1
});

//Insert this row into your data array
TableViewData.push(newRow);

//Refresh the data for that table:
myTableView.data = TableViewData;

    Hope that helps.
