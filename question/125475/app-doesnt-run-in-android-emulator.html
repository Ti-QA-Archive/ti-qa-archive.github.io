App doesn't run in Android Emulator

Hello all,

I'm trying to get the android emulator to build a demo of an app I've been working on (which does work in the iPhone/iPad, both in simulator/device). I try to run it, and the emulator loads, but I would have to unlock it, and the app itself doesn't install (The console doesn't show anything being loaded).

I'm currently running in OSX Leopard, with Titanium SDK 1.7.2. I have the emulator running Google APIs Android 2.3.3, with HVGA.

Currently the only Android emulator runs the Kitchen Sink without a hitch…but I can't figure out what the issue is.
