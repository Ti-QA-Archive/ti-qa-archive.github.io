Verify file checksum

Hi. I'm trying to verify a downloaded file by comparing MD5 hash of the downloaded file with a hash returned from the server.
However I didn't find a way to generate the local file hash using Titanium mobile.
I've read about Titanium.Utils.md5HexDigest, and tried it, but it crashes the app.
The code is:

var file = Ti.Fylesystem.getFile('myfile.png');
Ti.API.info('local hash: '+Ti.Utils.md5HexDigest(file));

Is there any way to get the raw file content? at least with that I could do an md5 of the string.

Thanks

  • The Ti.Utils.md5HexDigest() receives a blob, not a file.
    Do like this:

    var file = Ti.Filesystem.getFile('myfile.png');
var file_blob = file.read();
Ti.Utils.md5HexDigest(file_blob);
