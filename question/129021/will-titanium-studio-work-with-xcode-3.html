  • I went a different way. I was trying to get around this problem: http://developer.appcelerator.com/question/128919/started-getting—error-traceback-most-recent-call-last . I had the install disk for an older version of OSX so I was going to install on VMWare Fusion with the older tool stack and develop there so I could at least get something done.

    Turns out WMWare Fusion 4 supports Lion so I was able to get Xcode 4 and Titanium Studio loaded on there. My app at least compiles now and runs in the emulator. Crashes when deploying to a real device, but that could be problems with the provisioning. I was also careful to backup the VMWare image in case something goes wrong again.
