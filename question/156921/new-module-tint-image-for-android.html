new module: Tint image for Android

Hi,

not a question but an announcement (hope its ok here :) ):

Since maskedimage is only available for iOS and I need to tint an image on Android I've started a new module (my first one!):

Tint Image, available here

at the moment it will just multiply a blob with a green color and return the tinted blob. I need to add parameters to change color and blend mode. But you can create Hulk-pictures already ;)

Perhaps its useful for other people, feel free to use it and/or contribute to the github project!

  • another update:

    • Shortcut "mask" function to mask two images (see image on github)
    • changed the parameters so its a bit easier to set image, overlay, color and mode

    image overlay

    tint.tint({
image : $.img1.toBlob(), imageOverlay : $.img2.toBlob(), color : "#ff0000", modeColor : "overlay", modeImage : "overlay"
});

    just color:

    tint.tint({
    image : $.img1.toBlob(), color : "#ff0000", modeColor : "multiply"
});

    image mask:

    tint.mask({
  image : img1.toBlob(), mask: img2.toBlob()
});
