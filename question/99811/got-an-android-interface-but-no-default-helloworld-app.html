    Simon

    • Do you have python 2.6 (32bit) or greater installed?
    • Have you followed Preparing for Android development to the letter?
    • Delete all your android avds, using the AVD manager
    • Ensure that all the correct android packages are installed
    • Ensure that you have "APIs 2.2" selected on the Run Emulator screen
    • Delete all the contents of the subdirectories of your build/android/<subdirectory>/* directories
