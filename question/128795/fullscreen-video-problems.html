Fullscreen Video Problems

I am working on an iPhone app ( ios5 ) that has an instructional video within it. Once the app launches and the user clicks on an image, the video is supposed to launch into fullscreen mode. However, I have run into the following problems.

When the video launches in fullscreen mode on the iPad, the controls ( 'Done', 'Pause', 'Play', 'Forward', and 'Reverse' butons do not work. They simply can not be clicked on. However, all of this does work fine on the iPhone. This is extremely frustrating!

Here is a snippet of the code the I am using for the video player:

var activeVideo = Titanium.Media.createVideoPlayer({
    url: videoURL,
    movieControlMode:Ti.Media.VIDEO_CONTROL_DEFAULT,
    scalingMode:Ti.Media.VIDEO_SCALING_ASPECT_FILL 
});


video_thumb.addEventListener('click', function() {
    win.add(activeVideo);
    activeVideo.fullscreen = true;
    activeVideo.play();  

});

activeVideo.addEventListener('fullscreen', function(e){ 
   if (!e.entering){                                     // user pressed "Done" or video finished
        win.remove(activeVideo);
        win.orientationModes = [Ti.UI.PORTRAIT];
    }
});
