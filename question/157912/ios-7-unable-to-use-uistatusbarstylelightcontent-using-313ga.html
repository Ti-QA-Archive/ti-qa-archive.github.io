(iOS 7) Unable to use UIStatusBarStyleLightContent using 3.1.3.GA

Using 3.1.3.GA to build for iOS 7 on Mac OSX 10.8.5, I am unable to override the value for UIStatusBarStyle in my Info.plist. Other properties like UIAppFonts are being copied, but UIStatusBarStyle is overridden with UIStatusBarStyleDefault.

My App/Info.plist:

<key>UIAppFont</key>
<array>
    <string>/fonts/some-font.otf</string>
</array>
<key>UIStatusBarStyle</key>
<string>UIStatusBarStyleLightContent</string>

build/iphone/Info.plist

<key>UIAppFont</key>
<array>
    <string>/fonts/some-font.otf</string>
</array>
<key>UIStatusBarStyle</key>
<string>UIStatusBarDefault</string>

Is this possible to control from Info.plist in 3.1.3.GA?

    Use the <ios> section of the tiapp.xml to do it.

    eg: 

    
<ios>
    <min-ios-ver>6.0</min-ios-ver>
    <plist>
        <dict>
            <key>UIStatusBarStyle</key>
            <string>UIStatusBarStyleLightContent</string>
            <key>UIViewControllerBasedStatusBarAppearance</key>
            <false/>
        </dict>

        <!-- 
        ... 
        any other custom settings required 
        ...
        -->

    </plist>
</ios>
