blurred images inside table rows

Images inside a table view row get blurred vertically by 1 pixel, as if a ghost image was applied 1 pixel up of the real image.

var win1 = Ti.UI.createWindow({
    title : 'Window 1',
    backgroundColor : 'white'
});

var img = Ti.UI.createImageView({
    width : 16,
    height : 16,
    image : 'pixel_test.png'
});

var tr = Ti.UI.createTableViewRow({
    height : 32,
    left : 0,
    right : 0,
});

tr.add(img);

var table = Ti.UI.createTableView({
    data : [tr]
});

win1.add(table);


win1.open();

pixel_test.png is a simple 16x16 png with some bright pixels at the corners to make it easy to spot the problems. The image captured from the Simulator or the device has the right size, but the wrong pixels :( This happens in all the Titanium Mobile version I've tested from 1.8 to 1.9.

You can see the resulting distorted image here

Any idea on how to solve this?
