  • Hello Matej_,

    ###This is not a bug.

    You need to set bubbleParent : false for row.

    it is by default true. so if you longpress on row than tow event fired 1. row's and 2. tableview's.

    If you set Row's bubbleParent : false than only row's longpress event fired.

    i hope this helps. thanks.
