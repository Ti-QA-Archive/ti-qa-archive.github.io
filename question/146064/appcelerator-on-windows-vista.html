  • Hi

    You are right the according to the Titanium Compatibility Matrix Windows 7 is the only windows version actually supported.

    However, supported is not the same thing as not possible. I would suggest trying it and seeing what happens. As Windows 7 and Windows Vista are essentially similar under the hood you may be OK.

    It looks like Android SDK and the emulator will work on Vista so that should not stop you.

    The Q&A is full of people reporting using Vista with Android so that is a positive sign. Hope it goes well, no harm in trying.
