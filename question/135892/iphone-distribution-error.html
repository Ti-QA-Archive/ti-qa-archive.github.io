    That error message means that you don't have a distribution certificate in your keychain. Did you create a distribution profile in the iTunes provisioning portal and download and install both the certificate and the provisioning profile? If you need help doing this, I'd suggest reading the relevant part of the Tools Workflow Guide documentation on Apple's site.
