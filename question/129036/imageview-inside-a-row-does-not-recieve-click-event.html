  • I assume you're using the add() method of the row to add the image view, rather than setting the left image or right image properties. Adding an event listener to the parent row for the 'click' event will give you an event object. If you check the type of the event source, you can narrow it down to a particular type of element and/or, if you have given your elements id's, narrow it down even further if need be.
