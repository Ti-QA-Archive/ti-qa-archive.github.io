  • Hello Narasimha,

    64 bits: Current Status

    Currently, there is no official support for 64-bit simulator, as you can find here. Apple doesn't enforce the 64 bits apps yet, you can safely use a 32 bits app in the iOS7.

    64-bits: The future

    The support for 64 bits will come in future versions of the Mobile SDK, for now we recommend to stick to 32 bits apps.

    Workaround

    Please use the 32 bits ios-sim.

    Thanks for reporting this issue!
