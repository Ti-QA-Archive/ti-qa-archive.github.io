Color of TableViewRow according to an attribute

Hi,

I use a TableView with alloy data-binding for my project. Is it possible to change the color of a row according to the data.

For example, my data contains number. If the number is negative, the color is red and if it's positive, the color is green.

How to achieve that?

Thank you

    Hello Vincent Grivel

    You can use this tricks hope it will help to you.

    var tableData = [];

var win = Ti.UI.createWindow({
    backgroundColor : 'white'
});

var table = Ti.UI.createTableView({
    objName : 'table'
});
var number = [-1, 3, -4, 5, -7, 3, -5, 3, -2, 4];

for (var i = 0; i < number.length; i++) {
    var row = Ti.UI.createTableViewRow({
        height : 100,
        backgroundColor : ((number[i] < 0) ? "red" : "green")
    });

    tableData.push(row);
}

table.setData(tableData);

table.addEventListener('swipe', function(e) {

});

win.add(table);
win.open();

    If you have any concern, then let me know.

    Thanks.
