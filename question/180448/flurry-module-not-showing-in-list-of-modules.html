    • Hi ,

      Do the following steps.

      1.Unzip the zip file

      2.Copy the folder to your Appcelerator titanium's iPhone or Android module folder. If you didn't changed the install path the normal path will be: ~/Library/Application Support/Titanium/modules/iphone or android

      3.Go to the TiApp Editor page of your project. a. If you are using the GUI interface add the module on the right side window. b. If you are using the xml file, add the below lines to your tiapp.xml.

      <module version="$MODULE_VERSION">$MODULE_ID</module>

      EXAMPLE

      <modules>
        <module platform="android" version="2.3.0">ti.cloud</module>
   </modules>

      Thanks.
