Positioning and Image in a ScrollView

I'm making good progress on getting to grips with Titanium, but I have a (probably) simple question. I am putting an image in a ScrollView that I want people to be able to scroll around with their finger. But I want the first view of the image to be centered rather than the top left. I really can't see how to set the starting position of the ImageView though! Here is my code:

// Set up window
var win = Titanium.UI.currentWindow;

var mapView = Titanium.UI.createScrollView({
showVerticalScrollIndicator:false,
showHorizontalScrollIndicator:false,
contentWidth:1000,
contentHeight:750,
height:380,
width:320,
backgroundColor:'#333'
});

var image = Titanium.UI.createImageView({
image:"images/world.png",
canScale:true
});

mapView.add(image);

win.add(mapView);

How do I position the view port over the center of the image?

Thanks in advance.

Sean

PS. Also, the above code works on iPhone nicely, but not on Android. Any ideas?
