  • views/index.xml

        <Alloy>
        <TabGroup backgroundColor="white" >
            <Tab id="tab1" title="Home" icon="KS_nav_views.png">
                <Window id="win1" title="Home">
                   <Label id="label1" color="#999">I am Window 1</Label>
            <Button id="button" onClick="doClick" title="New Window" top="10" width="100" height="50" />
                </Window>
            </Tab>
            <Tab id="tab2" title="Map" icon="KS_nav_views.png">
                <Window id="win2" title="Map">
                    <Label id="label2" color="#999">I am Window 2</Label>
                </Window>
            </Tab>
            <!-- Use the Require tag to include external Ti.UI.Tab views -->
        </TabGroup>
    </Alloy>

    controllers/index.js

    var doClick = function() {
    var win = Ti.UI.createWindow({ backgroundColor: "green"});
    $.tab1.open(win);
}

  • No. You can hide the whole tab bar, but not selected tabs. On iOS it automatically groups them under ellipses(…) tab.

    If you really want the above functionality, you can hide the tabbar completely and add a toolbar at the bottom to provide controls for just the tabs that you want to be accessible.(It would still be a huge hack)

    • You need to open the new window into the home tab navigation group. 

      var win = Ti.UI.createWindow({ backgroundColor: "green"});
$.tab1.open(win);

      Observe the $.tab1.open, not $.win.open(). As a result, the new window will occupy the home window completely and a back button will be added as the left navigation button.
