"101-101 Test" Text is showing as link in iPad.

In HTML if I have a text like "101-101 Test" it turns into hyperlink style with blue colour text and underlinewhile running the app in IOS. How to stop this.

  • Hi Veera,

    Add this after your <html> tag : "<meta name='format-detection' content='telephone=no' />".
    This will disable the telephone number detection in your html page.
