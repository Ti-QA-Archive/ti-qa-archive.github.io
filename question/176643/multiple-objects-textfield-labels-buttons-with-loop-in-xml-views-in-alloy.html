Multiple Objects (Textfield, Labels, Buttons) with Loop in XML (Views) in Alloy?

Hi All,

I am trying to create multiple labels and textfields in one view in xml like:

<Label id="Price1"></TextField>
<TextField id="Amount1"></TextField>
<Label id="Price2"></TextField>
<TextField id="Amount2"></TextField>
<Label id="Price3"></TextField>
<TextField id="Amount3"></TextField>

Can this be declared using any XML looping technique something like:

i=0;
For each
    Create TextfieldObject with id="Price" + i
    Create Label Object with id="Amount" + i
End

Also, how do you declare this dynamically (number of labels/textfields will depend on user input)?

    Hi

    The xml format is simply static so you are hard-coding in the entries. Thus not possible doing it as you would like based on your second code block.

    Dynamic entries are commonly added within the controller in a loop. So you would create a new controller to add the label and textfield and then add that to the parent for as many as you need.

    Keep in mind as the number of entries is dynamic, you have to access the answers dynamically anyway - so this is the best solution.

    pair.xml

    <Label id="Price"></TextField>
<TextField id="Amount"></TextField>

    pair.js

    var args = arguments || {};
// define public functions
$.setPrice = function (val) { $.Price.text = val || ''; };
$.setAmount = function (val) { $.Amount.val = val || ''; };
$.getPrice = function () { return $.Price.text; };
$.getAmount = function () { return $.Amount.val; };

// set initial values based on optional paramaters
$.setPrice(args.price);
$.setAmount(args.amount);

    index.js

    var pairs = [];
for (i = 0, j = 3; i < j; i = i + 1) {
    pairs.push(Alloy.createController('pair', {
        price: 13.13 // get value from where ever you are getting the value
        amount: 13.13 // get value from where ever you are getting the value
    });
    // assumes there is a parent view called parent
    $.parent.add(pairs[pairs.length  - 1].getView());
}

// later you can then access the values;

_.each(pairs, function (pair) {
    Ti.API.info('pair: price: ' + pair.getPrice());
    Ti.API.info('pair: amount: ' + pair.getValue());
});

    This is just one way.
