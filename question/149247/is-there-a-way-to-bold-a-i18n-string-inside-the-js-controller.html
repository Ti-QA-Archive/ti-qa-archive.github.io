  • Hi

    Your variable name tabSystem_textarea_system does not provide any information as to WHAT view this is linked to; textField, textArea, label etc - guessing a text related field.

    There is currently no way to provided formatted values inside a text edit object.

    • If this was Android and this was a label you could use the html property - but the user cannot edit that text of course
    • If this was a label on either platform you could use the styledLabel module from the market place - again though - not editable.

    If your code relates to another control, let me know.
