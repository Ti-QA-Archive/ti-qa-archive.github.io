DatePicker Dialog

Hi guys. I'm fresh new to Titanium Mobile and I'm trying to build a simple datapicker dialog to be shown upon a click to a button.

From what I've experimented and read, the way to do it is by assigning a view to an OptionDialog with the picker inside.

That's all well and good, but I need a "Set" (and maybe a "Cancel") button. And if I add the buttons to the view they overlap with the picker. Any idea how to build this? Should I create another view to hold the button + the view with the date picker? That seems a little nasty…

Also, how could I make the "Set" button to select the date? (close the dialog and store the date on some var for now)

Any help is greatly appreciated!
Thanks a lot!

Here's the code I have so far:

var btn = Ti.UI.createButton({
    title:'Set entry time',
    font:{fontSize:20,fontFamily:'Helvetica Neue'}
});

// Create a time picker for the pulgarize button
var picker = Ti.UI.createPicker({
    type: Ti.UI.PICKER_TYPE_TIME
});

// Create "Set" button
var setBtn = Ti.UI.createButton({ title:'Set' });

// Create a view to put the picker and button in
var pickerView = Ti.UI.createView();
pickerView.add(picker);
pickerView.add(setBtn);

// Create the dialog and assign the pickerView
var timeDialog = Ti.UI.createOptionDialog({
    title: 'Entry time'
    androidView: pickerView
});

btn.addEventListener('click', function(e)
{
    // Show time picker dialog
    timeDialog.show();
});
win.add(btn);
