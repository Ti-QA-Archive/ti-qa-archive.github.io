How to pass parameters to Alloy Require tag from parent controller?

Hi, i have a alloy view which contains a box with image and label. I use this box on many places, so I include it in my alloy views with require tag, many times: 

<Require id="box1" type="view" src="boxItem"></Require>
<Require id="box2" type="view" src="boxItem"></Require>
<Require id="box3" type="view" src="boxItem"></Require>

Depending on other data different boxes have different params for elements in box.
The problem is that i calculate the properties in controller which holds this boxes. Properties depends on custom data which is different for every box. How to send correct parameters to boxes when controller is opened ?

For example every box contains different image. I can not set image in alloy view because image source, comes from backend.

  • Accepted Answer

    In boxItem.js create a "public" method, like the following

    exports.setBoxItemImage = function ( imagename ) {
    $.imageview.image = imagename;
}

    Then to set the images from your controller all you need to do is the following:

    $.box1.setBoxItemImage( "http://www.mysite.com/image1.png" );
$.box2.setBoxItemImage( "http://www.mysite.com/image2.png" );
$.box3.setBoxItemImage( "http://www.mysite.com/image3.png" );
