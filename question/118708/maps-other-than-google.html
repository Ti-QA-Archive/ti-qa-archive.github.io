Maps other than Google

Can appcelerator handle maps other than Google? I want to use a custom image for a map for an app that's just for a small area with lots of detail.

  • Rather than use the full 'interactive' Google Maps, you can still use them to get a simple static image, given lat/long coordinates, which can be pretty useful:

    eg

    

var geo_coords = "50.9078,-2.6515"; // comma separated latitude,longitude

var map_image_url = 'http://maps.google.com/maps/api/staticmap?center='+geo_coords+'&zoom=12&size=300x240&sensor=false&markers=size:small|'+geo_coords;

// see options for other url parameters here : 
// http://code.google.com/apis/maps/documentation/staticmaps/

    hope this helps
