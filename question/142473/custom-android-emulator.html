Custom Android Emulator

Hi Everyone,

I created a custom Android Emulator screen size to the dimensions of the Samsung Galaxy S3 (W - 720 x H - 1280) using the latest Android SDK (r20) and the Emulator starts fine when I launch it from the SDK Manager.

Is there a way to run my custom Samsung Emulator inside of Titanium Studio? I looked in Run Configurations but I could not figure out how to start it.

If anyone could help with this it would be greatly appreciated.

Thank You

    Go the other way.

    Build in Titanium Studio targeting the Android API level you want.

    Then edit that titanium{api}{screen} that was created to match the dimensions you want.
