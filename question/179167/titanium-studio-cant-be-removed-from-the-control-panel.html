  • Hi Lori Baker-Eveleth,

    Please follow the below steps and see if it helps you to solve your problem.

    • First check the C:\Users\username\AppData\Local\Titanium Studio\full_uninstall.txt file for any new info (not all that helpful. Currently it points you to a website that tells you to delete a folder which no longer exists).

    • Delete the folder C:\Users\username\AppData\Local\Titanium Studio

    • Delete the folder C:\Users\username.titanium

    • Delete the folder C:\Users\username\Aptana Rubles

    • Scan your registry for "titanium" and delete the registry folder that refers to the Titanium installation / uninstallation. (You probably did that already)

    • If you fail to delete any of these folders, reboot and try again.

    Check the doc link for details: uninstalling studio from windows

    Hope this will help you to solve your problem.

    Thanks
