Problems installing/running android app on emulator and device

Titanium 3.3, win7, Android SDK 2.2, Java JDK 1.7. This problem is one step in the progression of another post relating to "Errors in Run on …". Finally, when I attempted to Run kitchen sink app on my android device, a window popped up telling me it couldn't find the Android sdk folder. Once corrected, the program loads on the device and emulator. However, there are still problems.The device shows a screen with a title of "Run Time Error" and text box titles Location, Message and Source. Those text boxes contain, "[14,2] app.js", "Uncaught syntax error. Unexpected token 'if'", and if(Ti.version<2.0) {", respectively. On the emulator, it displays the emulator device but ends with "Installing application on emulator, [ERROR] Failed installing com.appcelerator.kitchensink: pkg : /data/local/tmp/app.apk, Failure [INSTALL_FAILED_MISSING_SHARED_LIBRARY\". If anyone has knowledge regarding these errors, please let me know. Also, is there a document somewhere that lists errors and their meanings? Thank you.
