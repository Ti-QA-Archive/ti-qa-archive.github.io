How to change systemButton (tint)color in iOS 8?

Hi there

I'm not able to change the systemButton icon tint color. The bar of the window uses Titanium.UI.iPhone.StatusBar.LIGHT_CONTENT.

var window = Titanium.UI.createWindow({  
    title: 'sometitle',
    barColor: "rgb(153,0,18)",
    statusBarStyle: Titanium.UI.iPhone.StatusBar.LIGHT_CONTENT
});

// add left info button
var btn_info = Ti.UI.createButton({
    systemButton: Ti.UI.iPhone.SystemButton.INFO_LIGHT
});
window.setLeftNavButton(btn_info);

Now the systemButton icon has a blue tintcolor. Using properties like tintColor and color are not working.

Do you know how I may change the tintColor of the systemButtons or do I have to use own icons?

I'm using Titanium SDK 3.4.0.GA on a Mac OS X 10.9.5 developing for iOS8. I'm testing with the iOS Simulator for iPhone 4s.

Thanks in advance!

    The property is called navTintColor and has to set on the window itself.

    var window = Titanium.UI.createWindow({  
    title: 'sometitle',
    barColor: "rgb(153,0,18)",
    statusBarStyle: Titanium.UI.iPhone.StatusBar.LIGHT_CONTENT,
    navTintColor: '#000'
});

    http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.Window-property-navTintColor
