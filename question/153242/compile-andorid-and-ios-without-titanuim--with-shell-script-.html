Compile andorid and ios without titanuim ( with shell script )

Hi,
I created a shell script on my mac os like below:

echo "Removing tmp folder"
sudo rm -R tmp
echo "Copying $name files to tmp folder"
sudo cp -R $name tmp
echo "Changing permission"
sudo chmod -R 777 tmp/
echo "Changing folder to tmp folder"
cd tmp
echo "Removing Resources folder"
sudo rm -R Resources/
echo "Removing all tmp .svn folders"
sudo ~/removeSVNFolder.sh
echo "Removing all tmp project files"
sudo ~/removeProjectFiles.sh
echo "Changing directory to app in tmp folder"
cd app
echo "Removing all application from iphone simulator"
sudo rm -R /Users/user/Library/Application\ Support/iPhone\ Simulator/6.1/Applications/*
echo "Compiling tmp project"
sudo lazyalloy -c
echo "Running project on $platform simulator on $log log level"    

if [ $platform = android ]; then
    echo "Running on android"
    cd ..
    cd ..
    titanium build --platform android --build-only --project-dir tmp/ --sdk /Users/user/Documents/android-sdk --log-level $log 
else
    sudo lazyalloy -p $platform -l $log
fi

I got unable to find android-sdk path error when I try with this command

./run.sh -n myProject -p android -l info

but when I try below I can run my app on iphone simulator

./run.sh -n myProject -p ios -l info

even I replace this:

sudo lazyalloy -p $platform -l $log

with:

if [ $platform = android ]; then
    echo "Running on android"
    cd ..
    cd ..
    titanium build --platform android --build-only --project-dir tmp/ --sdk /Users/user/Documents/android-sdk --log-level $log 
else
    sudo lazyalloy -p $platform -l $log
fi

but I dont have any successfully to run android emulator.
What is my wrong?
