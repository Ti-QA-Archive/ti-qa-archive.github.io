catch for if db fails to load?

var db = Ti.Database.install('/db/appDb.sqlite', 'appDb');

Is there a way to determine if the app failed to load the above database. At the moment the only errors I can get are thrown out in the console log in the Titanium console.

It seems that if the db failed to load the app just fails.

I tried:
var db = Ti.Database.install('/db/appDb.sqlite', 'appDb') || false;

    if(db){
        alert('db loaded');
    } else {
        alert(db + ' aa');
        Ti.API.info(db);
    }
