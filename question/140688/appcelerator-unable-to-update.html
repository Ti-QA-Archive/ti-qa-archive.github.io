  • I just resolved this issue this issue is due to not setting the proxy in the tool. As while establishing initial connection (i.e. First time Login using username and password ) proxy is set only for the HTTPS connection. So one has to manually reconfigure again the proxy for the HTTP Connection.

    One can set Proxy by going to

    • Windows-> Preferences
    • Select the General
    • Select Network Connections
    • check the HTTP with provider type manual
    • click Edit button & provide the respective proxy settings and click OK button.
