    All I can do is point you towards the github page for jasmine, which you've probably already seen:

    https://github.com/guilhermechapiewski/titanium-jasmine

    Here is a blog post about getting setup with it:

    http://www.singingseal.com/dev/titanium-and-jasmine/

    I haven't heard of any official position on unit testing. For a long time there wasn't any advice on how to even setup your project, until Tweetanium came along, and that really changed the game. I have hopes that something will come along for unit testing as well that will become the standard.
