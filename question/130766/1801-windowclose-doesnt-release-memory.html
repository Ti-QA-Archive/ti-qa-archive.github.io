1.8.0.1 window.close() doesn't release memory

Hi there,

Unless this is the expected behaviour I don't understand why the code bellow doesn't release the TiUIWindowProxy on iOS?

var window = Ti.UI.createWindow();    
window.open();
window.close();

Thanks,
Brice

    Try this:

    window.close();
window = null;

var window = Ti.UI.createWindow({
    backgroundColor: 'red'
});  
window.open();

    function() {
        window.close();
        window = null;
    }, 
    2000
);
