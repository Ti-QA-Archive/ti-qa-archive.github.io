app.js is 32kb

But still on the Android market the app shows as 8Mb and it takes 14MB on my device.

  • Has it been released using the V8 engine ? On 1.8.0.1 both V8 and Rhino are included in the binary which is bloating it. I think this is being addressed - check JIRA.

  • I think they main reason why it's larger than "expected" is because Appcelerator has to include which JavaScript-runtime the app is using.

    I've also seen a bug report in Jira (http://jira.appcelerator.org/browse/TIMOB-7472) that it seems like the current SDK is including both the Rhino and V8 runtime, even though it only needs one. So in the future the size will probably reduce a bit.
