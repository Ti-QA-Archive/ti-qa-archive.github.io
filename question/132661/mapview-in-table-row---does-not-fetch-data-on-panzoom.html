MapView in table row - does not fetch data on pan/zoom?

Hi,

I embeded a MapView in a table row. Everything works fine on iOS. On Android the view loads map data on window construction. But if I zoom or pan the map, the map view doesn't fetch the needed map data. I use Ti 1.8.1 and Android 2.2

The map is embeded like this:

var row = Ti.UI.createTableViewRow({
    height: mapHeight
});
var mapview = Titanium.Map.createView({
    mapType: Titanium.Map.STANDARD_TYPE,
    region: {latitude:currentLat, longitude:currentLng, 
            latitudeDelta:0.01, longitudeDelta:0.01},
    animate:true,
    regionFit:true,
    userLocation:true,
    height:mapHeight
});
row.add(mapview);
data.push(row);
var tableview = Titanium.UI.createTableView({
    height: '100%',
    data:data
});

Thanks for help!
