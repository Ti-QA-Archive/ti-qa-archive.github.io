Hi I want to store something related to the app. but it won't be deleted or removed by user until and unless user uninstall that application. I have tried following,

Please suggest me any way to do this.

  • This is the sure shot solution for your case:

    • Use Ti.App.Properties like this
      ~~~
      Ti.App.Properties.setString("username","Prashant"); // username is the property which is a string & holds the value as 'Prashant'

    var name = Ti.App.Properties.getString("username");

    // to reset the property, set the same value again
    Ti.App.Properties.setString("username",""); // username contains nothing now
    ~~~

    • this way you can set/get any property/data type like Ti.App.Properties.setInt(…) or Ti.App.Properties.getInt(…)

    • your properties will be saved unless you delete/modify them or user uninstall the app

    Read more about properties here
