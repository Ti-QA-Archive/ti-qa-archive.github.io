Blob to Image and save to filesystem

Hello,

I'm developing an app that requires saving imageviews into filesystem. So, I want to convert a Blob to an Image and save it to the filesystem.

I managed to do that using this code:

var filename = Titanium.Filesystem.applicationDataDirectory + "/image.jpg";
f = Titanium.Filesystem.getFile(filename);
d = imageView.toImage();
f.write( d );

The problem is that it works perfectly in the simulator but it doesn't work on my iPhone. The Image would be deleted from the filesystem whenever I quit the app, how can I make it save the image even when I quit the app? is it possible?

Thank you very much, I really appreciate all your help.
I look forward to reading you solutions.

BTW sorry for my poor english :-/

  • Instead of using the applicationDataDirectory, try to create a subfolder under it to store your image.

    var filename = Titanium.Filesystem.applicationDataDirectory + "/temp/image.jpg";

    if the code above doesn't work, create a new directory first.

    var dir=Titanium.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory + '/temp');
dir.createDirectory();
var filename=dir + "/image.jpg";
