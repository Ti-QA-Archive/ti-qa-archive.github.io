Error JDK 1.8.0 Android

Hello, I have a little problem and I hope u can help me :)
I installed Titanium for w7 a few days ago, it is the newest version (CLI 3.2.3 and Ti SDK 3.2.3). I installed the latest Android SDK and the latest jdk (jdk1.8.0_05), I setted the ANDROID_SDK (C:\Program Files\Android\android-sdk\sdk) and the same in the Path(C:\Program Files\Android\android-sdk\sdk\platform-tools and sdk\tools). I also setted the JAVA_HOME.
I followed the guide for create the typical "HelloWorld" and I created my personal AVD with Android 4.4.2 when I run the Project HelloWorld I have the error : "[ERROR] : JDK version 1.8.0 detected, but only version <=1.7.x is supported
[ERROR] Application Installer abnormal process termination. Process exit value was 1". Can anyone help me?? thanks a lot :)

    Hi Matteo,

    this error

    [ERROR] : JDK version 1.8.0 detected, but only version <=1.7.x is supported [ERROR] Application Installer abnormal process termination. Process exit value was 1"

    means you have to uninstall jdk 1.8.0 and install jdk 1.6, see the titanium compatibility matrix

    • Titanium creates a native app, all Components inside the Ti. scope are native. If you create a View with Ti.UI.createView() this will be javascript proxy object, which links to native View. But your App Logic still runs in a javasript virtual machine. At the moment Titanium will not compile your Javascript Code to Obj-C or Java, but your JS code will invoke native Components. Sorry this is a very short and vague description, I don't have the time to go into detail at the moment. I cannot say anything to your OpenVPN question, because don't have experience with that.
