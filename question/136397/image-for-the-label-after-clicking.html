Image for the Label after clicking

Hi, i have group of labels which are in for loop. If i click on any particular label the corresponding data should be display for that label, it is working fine for me. But i want to know, when i click on the label, i have particular image ,how can i apply that image on the selected label.

  • Try this:

    var label1 = Titanium.UI.createLabel({
    _image : 'image1.png',
});

label1.addEventListener('click', function(e) {
    // Access to label's image via e.source._image
    e.source.backgroundImage = e.source._image;
});
