    • Also, the latest release of Mavericks causes problems with Java based applications. The menu bar for the application gets disabled/disappears.

      This is because some key files are missing from the Java distribution. If you get this problem download and install Java for OS X 2013-004 from here

  • My Titanium studio stopped working too after updating to Mavericks/OSX 10.9. The log in Console showed 

    com.apple.launchd.peruser.501[161]: (com.appcelerator.titanium.36464[471]) Exited with code: 97

    After Googling I find out this was because Java was not installed or outdated or corrupt. So I installed this Java version http://support.apple.com/kb/DL1572 and Titanium started working again.
