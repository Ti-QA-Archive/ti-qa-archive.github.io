Sort Contacts By Last Name

I simply need to sort a list of contacts by last name. I know about the CONTACTS_SORT_LAST_NAME property, but I have no idea how to use it. I found a sort of answer in Google, but it involved creating groups, which is something I'd like to avoid.

So, after getting the array like this:

var people = Ti.Contacts.getAllPeople();

How do we sort them?

  • Figured it out myself. You have to specifically include which parts of the contact you need within the temp part of the for loop

            var peopleArray = Ti.Contacts.getAllPeople(); //Initial contact array
        var people = []; //Our final array
        for(var i = 0; i < peopleArray.length; i++) {
            var temp = [peopleArray[i].fullName,peopleArray[i].email,peopleArray[i].phone]; //Take parts you need
            people.push(temp);
        }
        people = people.sort(); //Sort by first name

  • CONTACTS_SORT_LAST_NAME is for use with Contacts.Group.sortedMembers

    So, to use that, you'd have to do Contacts.getAllGroups first.

    My suggestion is just use javascript array sorting to re-order all the Contacts however you want… Javascript Tutorial on sorting
