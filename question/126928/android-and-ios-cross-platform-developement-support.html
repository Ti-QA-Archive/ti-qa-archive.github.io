  • Titanium gives you the tools to develop on both these platforms, some functionality is common to both while other functionality is specific to each platform. While it gives you the tools to create cross platform apps it is up to you how you do this and how you structure your projects.

    Titanium also gives you methods to detect which platform you app is running on so you can structure your code to react appropriately. The Q&A has a great deal of information on various ways of doing this as well as the various example projects available to download.

    Also if you follow the Forging Titanium videos they have specific tutorials that deal with your question, in fact Episode 10 is about creating cross platform forms.

    Hope this helps,

    • In Titanium you write your apps in Javascript. You write your app once and you can run both on iPhone and Android - Titanium SDK knows how to convert yout javascript files into native code (sort of). When you get more into it you'll find that there are some differences - and that's ok, because these are different platforms (for example - Android has a menu button on the device which pops up a menu while iPhone does not).

      But lets say your creating a standard button - it's the same command written in JS.
