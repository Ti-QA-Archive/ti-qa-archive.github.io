  • Accepted Answer

    svn import everything from the app except for the build directory. So something like:

    mv MyApp/build /tmp
svn import -m 'initial import' MyApp file:///opt/repo/MyApp/trunk
svn co file:///opt/repo/MyApp/trunk MyApp
mv /tmp/build MyApp
