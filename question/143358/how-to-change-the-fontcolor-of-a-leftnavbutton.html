How to change the fontcolor of a leftNavButton?

Hi,
I wonder how to change the fontcolor of a leftNavButton.

When I make a Button like this: 

  leftnavbutton = Titanium.UI.createButton({
    color: "#3c4de2",
    title: "Test"
  });

And add it as leftNavButton to a Window which is in a tab group the font color changes from blue to white. When I add the Button just normal to the window the fontcolor is as it should be: blue.

Anyone got an idea? :)

Thank you very much!

Jakob

(I´m on SDK 2.1.3 and iPhone iOS 5.1)

  • Accepted Answer

    use backgroundImage: 'none' for button which you are going to set on left navigation

    var tabGroup= Ti.UI.createTabGroup();

    var win1 = Titanium.UI.createWindow({ 

    title:'leftNavigation',

backgroundColor:'pink'

    });

    var tab1= Ti.UI.createTab({
    title:'navbar',
    window:win1

    })

    var edit = Titanium.UI.createButton({

    backgroundColor: '#3c4de2',

borderColor: '#1c1d1c',

borderRadius: 6,

color: '#ffffff',

borderWidth: '2',

height: 25,

font:{size:9, fontWeight:'bold'},

width: 50,

backgroundImage: 'none',

title:'Edit'

    });

    win1.setLeftNavButton(edit);

    tabGroup.addTab(tab1);

    tabGroup.open();
