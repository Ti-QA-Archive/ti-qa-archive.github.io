iPhone tableviewrow padding

I have a tableview with custom tableview rows that contains a single label. Right now the tableview row is equal height to the label, but i want to add some "padding" to the bottom and the top to make it look good.
This is how i create my rows

var row = Ti.UI.createTableViewRow({
    hasChild: true
});


var title = Ti.UI.createLabel({
    text: data[i].title.toUpperCase(),
    color: '#247301',
    left: '10dip'
});
row.add(title);

How do i add "padding" of let say 10dip to the bottom and the top of the label? If i set top and bottom value of the label or the tableview row, each row gets as high as the screen is.

  • 1) Provide a height value to the row

    var row = Ti.UI.createTableViewRow({
    hasChild: true,
    width: Ti.UI.FILL,
    height: '48dp'
});

    2) Nest the label (and other child views of the row) in a view

    var rowView =  Ti.UI.createView({
    width: Ti.UI.FILL,
    height: '48dp' // or Ti.UI.FILL if you specified a height to the row
});
var title = Ti.UI.createLabel({
    text: data[i].title.toUpperCase(),
    color: '#247301',
    left: '10dip',
    top: '10dip',
    bottom: '10dip'
});
rowView.add(title);
row.add(rowView);

    kind of this should work
