Can map callouts be styled by CSS?

Can callout properties be styled by CSS? I'd like to implement a custom map callout such that when a marker is clicked, the title and subtitle are shown in a manner that differs from the default. For example, let's say we need the title and subtitle to be black text on a red background, or to use a custom image as the background. I see by the annotation properties that a custom marker can be implemented, but don't see how I would style the callout.

If not, are there other strategies for implementing a custom callout?
