Have children fill parent equally

Let's say you wanted to create an iOS Style Tabbar with 4 Tabs.

You want the Tabbar to fill the screen horizontally and each Tab to be the same width so they equally fill the Tabbar.

How would you do that? Giving each Tab a width of "25%" and the Tabbar a "layout: horizontal" won't work on an iPhone 6 because of rounding errors (375 * 0,25 = 93,75 -> 94dp). That way each Tab would be too wide.
