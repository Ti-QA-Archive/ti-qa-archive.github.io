  • Accepted Answer

    Titanium is not designed to run HTML for a UI, so unless your talking about using the MobileWeb SDK, then I assume the issue is that your trying to use HTML in your source code.

    Titanium requires you to use JS APIs to generate native UI elements.

    If you wish to use HTML you can, but to do that you must first use the API to create a webview then inject your HTML into the webview
