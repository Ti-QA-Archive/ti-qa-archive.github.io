ButtonBar - label color

How do you change the color of the label on a button bar…?

  • Accepted Answer

    There is NO API exposed to change the label color on Ti.UI.ButtonBar as of now.

    However, you can have a workaround if your absolute need is to use ButtonBar control only. Instead of specifying label's title, specify an image (transparent image with text in color of your choice). Remember, when image is specified it takes precedence over the title property.

    Here is the sample code:

    var btnDone = Ti.UI.createButtonBar({
     labels : [{title:' Done ',image:'done.png'}], 
     height : 30, 
     backgroundColor : 'red',
     style : Titanium.UI.iPhone.SystemButtonStyle.BAR 
     });

    Hope this helps!!!
