Special characters on label

Hi.

I have this label and i want to add ' to the text so it spells " Tro't om du vill."

But you can guess what happends when i do that……

How can i get the ' on the label? 

var quote = Ti.UI.createLabel({
    text    : '"Trot om du vill."',
    color    : '#eee',
    bottom    : 10,
    width    : '90%',
    height    : 'auto',
    font    : { fontSize: 18 },
    textAlign: 'center' 
});
win.add(quote);

    text : &quot;Tro&#39;t om du vill.&quot;

    or

    text : &#39;Tro\&#39;t om du vill&#39;

    Hope this helps.
