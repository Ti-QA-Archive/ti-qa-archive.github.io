MobileWeb development - getting started

I'm trying to develop a mobile web app that needs to read a JSON file from a server - however due to the nature of the XMLHttpRequest restriction of requiring the origin of the request to be on the same server as the JSON file - I can't even get it to load the JSON file…

I tried installing a Turnkey LAMP Appliance and setting up TitaniumStudio to use an External Web Server, but no matter what I do, it seems to always default back to Built-In Server - thus it does not copy the project files to the LAMP Appliance and I can't perform any tests at all.

Can anyone shed some light on how I can get started with this ? I am desparate! Is there some way I can make files available to the Built-In Server for Titanium Studio so I could get started on the development - or even better, get it to work with the LAMP Appliance …
