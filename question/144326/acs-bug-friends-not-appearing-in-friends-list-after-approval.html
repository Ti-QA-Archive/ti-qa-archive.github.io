ACS-Bug, friends not appearing in friends list after approval

###Possible bug on ACS application working with friends (2 way type).###

####Steps to reproduce:####

  1. Configure 2 way friend request type on ACS application.
  2. Create user1.
  3. Create user2.
  4. Login with user1, send friend request to user2.
  5. Receive friend request with user2.
  6. On the ACS console, there is 1 request for user2 (correct)
  7. Approve friend request with user2.
  8. On the ACS console, there is 1 friend for user2 (correct) but when you click to see the list of friends, no friends appear in the list (incorrect).
    Also searchFriends is returning 0 results for user2.
