Database Error

I am attempting to set up my desktop app to run using a sqlite database for data storage. I continue having an error in retrieving data from my database. The first SQL statement I run gets a "Not Found" error (using simple SQL select "SELECT * FROM [table]"). I log into my database using the command line and run the same command and I get results. My database table is created running a SQL statement from my code in Titanium, so I know I have successfully connected to the database.

Any suggestions or advice would be greatly appreciated!

    Hi Today I have found out that there is a bug in TiDesktop Database api. When you query with "*" - all columns it gives error "not found". If you use

    "Select id,title  FROM cw_accounts"

    This will work as expected.
