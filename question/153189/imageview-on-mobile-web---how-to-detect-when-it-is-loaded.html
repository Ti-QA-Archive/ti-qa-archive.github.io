ImageView on Mobile Web - how to detect when it is loaded?

I am trying to load several images into each ImageView and adding them to a parent view. Each of these ImageViews have opacity = 0. When the images are loaded, I want to animate them to opacity = 1.

This works fine in native code, but not on Mobile Web. The animation is trying to run before the image is loaded into the ImageView. So I need some way to trigger when each image is loaded, so I can keep track and THEN fire the animation - however, I can't seem to find any single event that will trigger when the image is loaded….

Does anyone know how to do this ?

  • you either download your images with an xhr:

    xhr.open("GET", "http://imageurl");

xhr.send();

xhr.onload = function() {
    //image is loaded here, you can animate it
}

    or simply use the load event in the image class

    imageview.addEventListener("load", function() {
    //image is loaded here, you can animate it
};
