  • Kumar

    Relative layouts can help with this, which can be enabled using the layout: &#39;vertical&#39; property. Furthermore, on Android, elements may be given percentile dimensions, such as width:&#39;80%&#39;. Read the BNAPP book's Layouts, Positioning, and the View Hierarchy chapter for more information about this.

    You can also obtain the device dimensions from Titanium.Platform.displayCaps, using Ti.Platform.displayCaps.platformHeight and Ti.Platform.displayCaps.platformWidth, an example of which is available in the KitchenSink's hi_res_image_remote.js script.

    Other than that, testing using different emulator skins and physical devices will help you to achieve a schema that works across all Android devices.

    Hope this helps
