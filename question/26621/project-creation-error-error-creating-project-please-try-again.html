Project Creation Error: Error creating project. Please try again

Hello, I'm relatively new to Titanium Developer, I have created only 1 project before the 1.3 update of the sdk.

Since the update I have removed both the kitchen sink project, and my project. But now I cannot create a new project. I fill out all the required information, and click create project, and I get an error in the top right window:

Project Creation Error: Error creating project. Please try again.

I would really like to spearhead a few new apps, but I can't seem to get the developer to get started. Any help is appreciated…

    Hi Chris,

    I ran into this issue earlier, it seemed to be related to my Android install. The below steps addressed the problem.

    1) make sure your ADK is set correctly on your profile page it should point to the root of the ADK (i.e. the folder that contains platforms,add-ons, tools). If you change it, save it then exit developer and restart it. Then try and create the new project.

    2) Make sure you've downloaded the 1.6 platform and 1.6 Google API. Check your platforms directory for android-4 at a minimum.
