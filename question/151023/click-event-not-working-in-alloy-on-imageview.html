Click event not working in Alloy on Imageview

I have an imageView with an image in it, that I want to use for navigation.
I can't seem to make the click event work.

This is what I have, based on the documentation found:

in index.js:

function gotoPage(p_page){
     alert("clicked");
}

$.index.open();

$.imgFullTime.addEventListener('click', function() {
    alert("clicked");
});

in index.xml:

<Button onclick="gotoPage('fulltime')" id="btnFulltime" title="Fulltime">
<ImageView onclick="gotoPage('fulltime')" id="imgFullTime" image="images/btnFulltime.png"></ImageView>
</Button>

This does not trigger the alert. Any ideas what I'm doing wrong?
