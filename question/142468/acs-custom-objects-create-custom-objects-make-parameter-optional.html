ACS Custom Objects: Create Custom Objects make parameter optional

I have function as follows that creates a custom object in ACS. I want to have the photo-upload optional. At the moment I get "[ERROR] 400: Failed to upload photo: Failed to indentify photo file" when I file is null. What is the best way to achieve this (besides creating a second function that doesnt include the photo in the request)?

Cloud.Objects.create({
        classname: userActionsClassname,
        photo: file,
        fields: {
            type: type           
        }
...

    All your doing is passing an object into the Cloud.Objects.create method. So create the object before you send it along. If the photo doesn't exist, then don't add it.

    var obj = {
    classname: userActionsClassname,
    fields: {
        type: type           
    }
};

if (file) {
    obj.photo = file;
}

Cloud.Objects.create(obj, function(e) {
    // callback code here
});
