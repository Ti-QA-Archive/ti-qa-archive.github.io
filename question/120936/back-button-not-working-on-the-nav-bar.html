Back button not working on the nav bar

i want to create a back button in the nav bar , the following code is working fine for other window but not working for another. I m working on 1.6.2 as titanium sdk and this is for iphone .

its giving the following error in the console :

message = "Result of expression 'Ti.UI.currentTab' [undefined] is not an object.";

My code is is

var backBtn = Titanium.UI.createButton({
title:'Back',
style:Titanium.UI.iPhone.SystemButtonStyle.BORDERED
});
win.leftNavButton = backBtn;

backBtn.addEventListener('click',function(e){
    var window = Ti.UI.createWindow({
            url:"Search.js",

        });
    Ti.UI.currentTab.open(window,{animated:true});

});

  • It would be helpful to see what the rest of the window/navigation structure is for this.

    Usually a 'back' will simply close a window to slide back to a previous window. This functionality actually comes 'for free' without need to code it.

    If you currentWindow has the variable name 'win' (as it appears to, looking at your leftNavButton line there) and the name of the window you are wanting to open next is called 'window', then try using win.tab.open(window, {animated:true})

    But I have a feeling this will not give you what you want, since this will open a new window 'forward' and not 'back'.

    It would make more sense for a back button on a window called 'win' to simply call win.close()

  • if the code you have posted above is in app.js then Ti.UI.currentTab is not available. The variable Ti.UI.currentTab is only available in .js files that are associated with creating a window through the url property
