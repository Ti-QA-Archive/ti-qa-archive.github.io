How to set default language (when language i18n/xx does not exist) ?

Hi,

I was trying to set Ti.Locale.defaultLanguage=&#39;en&#39;; at alloy.js but it's not working. At 3.x there's no method or property defaultLanguage.

The problem is that when runnign app for instance in french iOS the app is getting strings from my es folder and not en… Is there any way to set a default strings.xml folder ?

Regards
Joao

    The answer is actually not Titanium specific. There's a info.plist property you have to set

    <key>CFBundleDevelopmentRegion</key>
<string>es</string>

    Use the <ios> section of tiapp.xml to do it.
