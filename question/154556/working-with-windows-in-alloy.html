Working with windows in Alloy

I've two windows, first is main window and the second one is details window.
It has a tableview with custom rows populated by an sql database. I've created a row swipe animation so that whenever the user swipes the row to the left the row vanishes and instantly new Window opens.

In the details window I've created a label (for testing), which when clicked closes the detail window and shows the main window. But the problem is that whenever I came back to the main window the swiped row which vanished stays vanished. How should I get it back.

$.scrollView.addEventListener("dragend", function(e){

    if (rowDraggedPosition > 80) {

        var targetedValue = Titanium.UI.create2DMatrix();
        targetedValue = targetedValue.translate(-400, 0);

        var animation = Titanium.UI.createAnimation({"transform": targetedValue, "duration": 250 });
        $.scrollView.animate(animation);

        animation.addEventListener("complete", function(){
            $.row.fireEvent('detailPageOpen', {
                "row": $.row.getView()
            });

            var detailWindow = Alloy.createController('detailWindow').getView();
            detailWindow.open();
        });
    }

This is the swipe event and opening of the second window. Pls Help.
Thanks in advance
