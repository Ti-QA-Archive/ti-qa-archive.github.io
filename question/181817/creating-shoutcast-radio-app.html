  • Hello Parminder,

    Unfortunately this can't be done with the standard Appcelerator AudioPlayer API which is based on old code from 2008. There are external modules that do this but they are expensive - see http://yakamozlabs.com/products/radiotunes
    Otherwise you'll have to do what I do and pick the songs up via an HTTP call to a sever where the radio station has to dump it's current song as a text file.
