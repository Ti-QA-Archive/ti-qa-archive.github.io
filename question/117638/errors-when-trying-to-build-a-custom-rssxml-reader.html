    there is no tag element called "items" in your XML, I think you want to use "item"

    xhr.onload = function()
{
    var xmlDoc = this.responseXML.documentElement;

    var itemList = xmlDoc.getElementsByTagName("item");
    Ti.API.info('found '+itemList.length+' items in the RSS feed');
    displayItems(itemList); 

    alert(data);

};

    Also the KitchenSink is a set of "examples" of how to use the frame work, you still need to get a basic understanding of the technology. The "examples" will not work in every situation, so be patient and google is your friend
