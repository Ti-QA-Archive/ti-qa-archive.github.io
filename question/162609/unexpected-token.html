  • Simply put…you need to go through and clean your code.

    Lots of silly errors, but to name a few:

    • Missing colons
    • Semicolons when there should be commas
    • Use of parentheses and curly braces
    • Make sure you close off your data structure and your functions
    • JSON not correct (check jsonlint.com)

    Also when you add blocks of code you should wrap it between 3 tildes(~) above and below, this makes it easier for the people who decide to help you, read you code.
