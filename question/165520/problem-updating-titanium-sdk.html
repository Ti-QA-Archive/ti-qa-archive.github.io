  • could always try to update from the CLI.

    Be sure you have the latest cli tools
    [sudo] npm install -g titanium

    Lets go ahead and be sure we have the latest alloy as well
    [sudo] npm install -g alloy

    Now update that sdk
    ti sdk install latest
