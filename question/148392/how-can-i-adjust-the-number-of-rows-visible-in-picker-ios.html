How can I adjust the number of rows visible in picker (iOS)

I have a tight spot to squeeze this picker into … the height you can see is small but it defaults to show 5 rows which is way too big … I changed the Visible items but it still defaults to 5 …. how can I change it to 2 rows … my preference would be 1 row … 

var picker = Ti.UI.createPicker({ top:345, width:125 , left:120, height:25});
            picker.visibleItems,2;
    var column1 = Ti.UI.createPickerColumn({opacity:0});
    column1.addRow(Ti.UI.createPickerRow({title:'1',custom_item:'b', selected:true}));
    column1.addRow(Ti.UI.createPickerRow({title:'2',custom_item:'s'}));
    column1.addRow(Ti.UI.createPickerRow({title:'3',custom_item:'m'}));
    column1.addRow(Ti.UI.createPickerRow({title:'4',custom_item:'g'}));

    var column2 = Ti.UI.createPickerColumn();
    column2.addRow(Ti.UI.createPickerRow({title:'1'}));
    column2.addRow(Ti.UI.createPickerRow({title:'2'}));
    column2.addRow(Ti.UI.createPickerRow({title:'3'}));
    column2.addRow(Ti.UI.createPickerRow({title:'4'}));

    // 2 columns as an array
    picker.add([column1,column2]);


    // turn on the selection indicator (off by default)
    picker.selectionIndicator = true;

    self.add(picker);

  • Hi

    The short answer is no.

    The picker is a fixed UI element, there is ability to change the height, it is fixed and part of the UI standards for iOS.

    You may need to choose a different UI to solve your problem.

  • Hey there,

    As Malcolm mentioned, there is no way to change the height of the default picker.

    Although you might be able to do what you need, using a "dirty" method that I used myself in my App.

    Create a view with a image background or a image view, and place on top of the picker, make sure the image you're using has transparency, and make the view none touchable.
