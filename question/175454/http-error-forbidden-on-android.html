HTTP Error Forbidden on Android

I have an iOS app that pulls json data from a url to populate table views. I have been tasked with now making an android version.

After fixing various UI compatibility problems, I keep running into this error seemingly no matter what I do.

[ERROR] :  TiHttpClient: (TiHttpClient-1) [3608,3608] HTTP Error (org.apache.http.client.HttpResponseException): Forbidden
[ERROR] :  TiHttpClient: org.apache.http.client.HttpResponseException: Forbidden
