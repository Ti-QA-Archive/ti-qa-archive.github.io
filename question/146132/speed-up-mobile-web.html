Speed up mobile web.

Hi,

I want to increase the performance of my mobile web project. Is there any way/tricks to speed up the project performance in the browser ? My app performance gets slow as I have to load lots of components at certain times. I have used 'destroy', nullifying components when not in use.

Any ideas ?
Thanks.

    Using destroy() is good. That will take care of memory issues.

    The first thing you can do is reduce the number of UI elements. Do you really need 3 nested views to position a label? Instead of using a TableView, use a ScrollView and manage the rows yourself.

    Instead of using images for solid or gradient backgrounds, use the backgroundColor and backgroundGradient properties. With some clever gradient trickery, you can do drop shadows.

    If you're using Titanium 3.0, then add <code-processor>true</code-processor> to your tiapp.xml. When the code processor is enabled, it will magically determine all Titanium dependencies in your app. This means if you don't use Facebook or Maps, it'll significantly reduce the size of your app and speed things up.

    This last tip requires a bit of titanium-fu. It involves writing Titanium Mobile Web Modules. The idea is create a new module, copy the original source file from the Titanium SDK's mobileweb folder and heavily modify it. See, Ti.UI.Label has a bunch of overhead to support text alignment and text wrapping. Ti.UI.ImageView has a bunch of code for slideshow functionality. Ti.UI.Button has extra code for putting an icon on the image.

    You must use the old project.py script in the Titanium SDK folder to create the mobile web module. Copy the various Mobile Web source file from the SDK folder that you want to modify, then change the name of the UI element in the source's declare() function to your own namespace. So, "Ti.UI.ImageView" would be "My.ImageView". You must be VERY careful what you change. It is very easy to break MobileWeb.

    I unfortunately don't have a tutorial that covers these techniques, so you're on your own to hack your app into shape.
