    Hi Stephen,

    Titanium mobile applications can be developed using Javascript as your primary development language. You can use webView (HTML/CSS/Javascript) since you mentioned in the question. For local storage you have SQLite support. Let me know if you need any further information.

    Also check the titanium mobile documentation (http://developer.appcelerator.com/documentation) and training materials (http://training.appcelerator.com/)
