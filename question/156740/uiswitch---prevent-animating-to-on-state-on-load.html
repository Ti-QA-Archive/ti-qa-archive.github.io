UI.Switch - Prevent Animating to On State on Load

I've got a switch in Alloy, I don't give it an initial value. In my controller, I set its initial value based on user's preferences from a collection.

It always animates from off to on.

I've seen suggestions of adding the switch before opening the window, but with Alloy, I'm not sure how you would do that.

There must be some way to prevent this behavior. Anyone have an idea? Thanks!
