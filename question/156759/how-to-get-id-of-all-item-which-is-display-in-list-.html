    Well, all you have to do is to add an event handler on the itemclick event. From there you will have access to the row object itself.

    ###Here is what the code would look like:

    listView.addEventListener('itemclick', function(e){
    alert(e.title);
});

    As for you TableView, the process is similar, except that you will have to create a handler for the click event. Inside this event, you will be able to access the row object using the rowData property.

    ###Here is what your code would look like:

    table.addEventListener('click', function(e) {
   alert(e.rowData.title);
});

    Hope this helps!
