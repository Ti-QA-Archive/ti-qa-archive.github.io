No .apk file produced

I've tried just developing a simple "Hello, World!" app for Android

Whenever I try to run it (Both android emulator and physical android device) the emulator device loads but the app does not appear on screen. When I check in the \build\Android\bin folder for the project, there is no .apk file. All the JDK and Android SDK files are up-to-date. This problem is consistent across every emulator platform I've tried.

Contents of app.js:


var win = Titanium.UI.createWindow({});
var button = TItanium.ui.createButton({
    title:'Hello, World!',
    top: 40,
    left: 40,
});

win.add(button);
win.open();

Using:
Windows 7
Titanium Studio, build: 2.1.2.201208301612

  • Open your tiapp.xml and check for problems (yellow triangles with exclamation mark)
