  • HTML needs to be displayed in a WebView not a label. Because it is an HTML fragment, you would need to wrap it in standard HTML code.

    Assuming you parse your xml node <content> into a variable called content:

    content = '<html><body>'+content+'</body></html>';

    pass content as the src to your WebView
