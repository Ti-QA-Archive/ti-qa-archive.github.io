  • Hi

    Unfortunately currently there are no events for the following;

    • You will not be informed that the delete button has been shown
    • You will not be be able to cancel the delete action

    You can ONLY respond to being told about the delete that has already occurred.

    • The delete UI is generated by the OS so it will always appear over whatever you add, it may also affect the width, left and right positions of the contents depending on how you added them.

      As it stands you either accept the default UI and have to ability to intercept the event you hoped would be there or disable the default delete action and create your own UI based on an edit button and/or swipe event.
