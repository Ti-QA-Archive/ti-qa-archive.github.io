  • Although you have put your crash log in here, we don't actually know what you are doing.

    Running an application is not really explaining in detail what you are trying to achieve.

    This could be an issue with your code, the Android SDK installation or something completely unrelated.

    You need to be a lot more specific about what you are doing, what your code does and what you have changed.

    I would suggest closing this Q & A, and creating a new one, with A.. a description of what your app does B.. A simple code example which fails on running. C.. the error log on this.

    Do not polute the question with pasting a lot of code or the error log but use something like gitpaste and put the link in your question.

    Again please be specific: detail what OS your on, what version of studio, what SDK you running against.

    Do not post your whole application but make a test case of it where it is failing.

    Hope this helps
