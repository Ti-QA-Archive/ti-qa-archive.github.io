leftnavbutton not dispalying correctly when using a custom backgroundImage

Hi,
I have a strange problem with leftnavbutton.
I want to use a custom button with a custom image that has the word "back" in it so i don't need a button title.

when running the application in ios ipad simulator the button isn't displayed in the navbar but clicking the area where the button is supposed be triggers the event.

I found that if I do add a title the button is displayed but the button title text is displayed over the button image

any ideas how to solve this?

  • Accepted Answer

    var win2 = Ti.UI.createWindow();
    var officialBtn2 = Ti.UI.createButton({
    left : 10,
    height : 30,
    width : 55,
    backgroundImage : '/images/buttons/en/back.png'
    });
    win2.leftNavButton = officialBtn2;
