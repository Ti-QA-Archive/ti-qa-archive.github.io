Android window close event not firing

For some reason, my app, build with 1.7.2 is not able to add an event listener to a window's close event, on Android.

Here's my code:

Window.addEventListener('close', function() {
 alert('the window was closed');
});

Nothing. Ideas? This only occurs on Android, it seems.

  • Through some trial and error, I found that a very specific combination of parameters passed when creating the window makes the window close event work:

    Ti.UI.createWindow({
    title: 'Window 1',
    // styling parameters...
    modal: true
    // do NOT specify ANY other parameters like "fullscreen" or "exitOnClose"
    // doing that will make the 'close' event NOT fire
});

    I'm using SDK version 2.1.0, by the way.
