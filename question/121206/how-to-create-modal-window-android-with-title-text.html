  • The generated AndroidManifest.xml contains

    <activity android:configChanges="keyboardHidden|orientation" 
android:name="org.appcelerator.titanium.TiModalActivity" 
android:theme="@android:style/Theme.Translucent.NoTitleBar.Fullscreen"/>

    affect all Modal Window without TitleBar, it is the default for android ?

  • did you try hiding the NaVBar?

    var win = Ti.UI.createWindow({
    title:'TestWindow',
    navBarHidden:true
});
win.open({modal:true});

    The documentation states for modal windows, hide the nav bar (true) or show the nav bar (false, default). http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.UI.Window-object
