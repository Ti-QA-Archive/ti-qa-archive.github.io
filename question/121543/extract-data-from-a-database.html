Extract data from a database?

Hi, i want to know how can i extract the data from my database and make them appear in an alert (or a view). For my project it's just a test so i would have just one cell and show it in an alert.

Titanium.UI.setBackgroundColor('#000');

var win = Titanium.UI.createWindow({backgroundColor:'black'});
win.open();

var db = Ti.Database.install('baseTest.sqlite','baseTest');
var rows = db.execute('SELECT * FROM Personne;');
var cpt_row = 0;

while (rows.isValidRow()) {
    cpt_row = rows.field(0); // I count the number of rows
    rows.next();
}

alert(cpt_row); // I want to return the number of rows in an alert here

rows.close();
db.close();

alert(cpt_row); // or here...

    Simply count your number of records using the COUNT SQL function.

    Ti.UI.setBackgroundColor('#000');

var win = Ti.UI.createWindow({backgroundColor:'black'});
win.open();

var db = Ti.Database.install('baseTest.sqlite','baseTest');
var row = db.execute('SELECT COUNT(*) FROM Personne;');

Ti.UI.createAlertDialog({
    title: 'Number of rows',
    message: row.field(0)
}).show();         

rows.close();
db.close();

    • You were on the right track with your loop.

      What you need to do is to store you final value in a variable (message in the example below). You simply "loop" through all the records and add the NOM Column Value to the message using the += operator.

      var rows = Ti.App.db.execute('SELECT * FROM Personne;');
var message = '';

while (rows.isValidRow()) {
    message += rows.fieldByName('NOM');
}

      Then you can use the message variable and display it however you want. This, of course, is a basic example. But it should get you started.
