        id: "index"

    • First clean and build then go to build folder in your project and duplicate AndroidManifest.xml and name it AndroidManifest.custom.xml then open this custom xml and add your theme code in it

       <application android:theme="@android:style/Theme.Holo.Light"/>

      Don't replace the entire application opening tag, just add this to it.

      android:theme="@android:style/Theme.Holo.Light"

      Now don't do clean and build because it will delete your AndroidManifest.custom.xml file. Go ahead run the application on device and now Titanium will use your custom Manifest file.

      Hope this helps.
