win.visible retunrs undefined

The API docs sepcify that the Window object has a visible property. However when I try to view the property value it returns undefined, in SDK 1.4.2 and the continues build from today.
Any idea if this is a bug, or i'm doing something wrong?
This is for iPhone. Don't know if this exists on Android.

The code i'm using:

var info = Ti.API.info;
Ti.UI.setBackgroundColor('#000');

var win = Ti.UI.createWindow({
    title: 'Window 1',
    backgroundColor: '#fff'
});

var label1 = Ti.UI.createButton({
    title: 'visible',
    top: 20,
    width: 100,
    height: 40
});

label1.addEventListener('click', function(e) {
    info('win.visible: ' + win.visible);
});
win.add(label1);
win.open();

Thanks!

    Hi,
    I found the solution. In order to make use of the "visible" property, you have to set it (i.e. define it) while ceating the UI element. Then you can use it in your code:

    var logoutView = Ti.UI.createView({
    bottom : '0dp',
    visible:true,// <-- notice here
    backgroundColor : '#333',
    height : '60dp'
});
tableView.addEventListener('click', function() {
        if(logoutView.visible == true) {
            logoutView.hide();
            logoutView.visible = false;
        } else {
            logoutView.show();
            logoutView.visible = true
        }
});
