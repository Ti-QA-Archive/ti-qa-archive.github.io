Adding ScrollableView currentPage views into another window

Hi,

I am trying to add views from a ScrollableView to show in a new window. More specifically, I need to show the currentPage view in a new window when that view is clicked. All these views are images so its like further browsing or enlarging the image in a new window. The images are pushed into an array from a JSON file so i tried this:

var images = [];
images.push(bunch_of_images);

var num = ScrollableView.currentPage;
some_window.add(images[num]);

This only got the image at the first position since the currentPage when you open the app is indexed at 0. Clicking on another image still showed that first image, of course, and I havent figured out a way to get this number to change…

Any ideas? Everything is appreciated..thanks

I am working in iOS with Titanium 3.1.0

    Hi

    Is this line of code;

    var num = ScrollableView.currentPage;

    In your click event? If not then it will still have the same value as when that line of code was processed, if you put it in the click event it will update to the current value.

    If this does not solve it then please provide the click event code snippet.
