    • Hi,

      Try my test code below in your controllers and see if it works for you. I've tested it and works fine for me.

      $.dialog.addEventListener('click', function(event) {
    if(event.index==0)
    {
        Cancelo();
        alert("Cancelo");
    }
    else
    {
        Copio();
        alert("Copio");
    }
});
