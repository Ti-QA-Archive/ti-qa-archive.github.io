  • Hi,

    It seems that your Provision Profile is missing for some reason. I would do:

    Go to 'Run on Device'
    'Select your provision profile '
    Change this entry to your provision profile. Have in mind that the Provision Profile must be created for this app in question and your provision profile must match with the info you used on the Project information (Edit project > look at the Application ID this must have the same information as your Provision Profile)
