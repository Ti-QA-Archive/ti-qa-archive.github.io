Access global variables directly from .xml files : (ex: Alloy.Globals from index.xml)

Hello,

I want to know if index.xml can access to global variables directly.

For example, I want to do something like this:

alloy.js

Alloy.Globals.myText = "foobar";

index.xml 

<Label text="Alloy.Globals.myText" id="myLabel"></Label>
// or 
<Label id="myLabel">Alloy.Globals.myText</Label>
// of course, this doesn't work. It shows the variable's name: "Alloy.Globals.myText" itself.

Not using index.js like this

$.myLabel.text = Alloy.Globals.myText;

    Hi,

    You can access Alloy.Globals from TSS, not from XML attributes or index.xml file.

    You can make global-style-sheet, the best method is to create a file in the 'styles' folder and name it 'app.tss', this is the top level style, anything you code in here will be read first in the stack.

    You can go through this link, this will be useful to you:

    http://developer.appcelerator.com/question/147815/alloy—access-global-properties-in-tss-file

    Hope this will help you !

    • Welcome :) Yeah, You cannot access the alloy global variable from xml files.

      You can create a global style file, called app.tss, which applies all styles defined inside it to all views, but does not override the non-global styles or property attributes in the markup. 

      //============styles/app.tss==================


"Label":{
    text: Alloy.Globals.myText,
    color: 'gray',
    textAlign: Ti.UI.TEXT_ALIGNMENT_LEFT,
    backgroundColor: 'transparent',

}

      For more details you can refer this link http://docs.appcelerator.com/titanium/3.0/#!/guide/Alloy_Styles_and_Themes

      If this helps you then you can mark answer as accepted.
