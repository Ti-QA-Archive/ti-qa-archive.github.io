Call a number stored in a label inside a table row

I have phone number stored in a label inside a table row. I tried adding an click event to the label so you can call the number but it doesn't work. Any ideeas?

    Capture row click event and look for the source of the click:

            row.addEventListener('click', function(e){
            if(e.source === labelPhone){
                Titanium.Platform.openURL('tel:' + phoneNumber); 
            }else {
                //...
            }
        });

    Share your code if you do not see clear how to implement.
