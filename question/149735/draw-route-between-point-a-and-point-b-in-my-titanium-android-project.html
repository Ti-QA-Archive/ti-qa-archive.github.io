    • Easier than you think;

      If you tweak this section of the code I provided;

      var param = [
    'destination=' + obj.stop,
    'origin=' + obj.start,
    'region=' + obj.region,
    'sensor=false'
];

      To this instead;

      var param = [
    'destination=' + obj.stop,
    'origin=' + obj.start,
    'sensor=false'
];
if (obj.region) {
    param.region = obj.region;
}

      You can then use these different formats;

      • Named locations;
      addRoute({
    map: map,
    region: 'UK',
    start: 'Cambridge',
    stop: 'Birmingham'
});
      • Lat/lon pairs
      addRoute({
    map: map,
    start: '40.7142,-74.0064', // New York
    stop: '41.8500,-87.6500' // Chicago
});
      • Mixed
      addRoute({
    map: map,
    region: 'US',
    start: '40.7142,-74.0064', // New York
    stop: 'Chicago'
});
