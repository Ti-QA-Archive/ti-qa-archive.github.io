  • In short, yes, you can bounce between tabs. Since your first page is going to be 'home' you dont need set the active tab to 0 .. 0 is already going to be your active tab. From there, you can set event listeners on any buttons to call the setActiveTab() on your tab group to target the tabs you want to switch to. Check the tabgroup docs for more info:

    http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.UI.TabGroup-object

    Now, while there are instances when tabs are completely 'pointless', its not always the case. But if they are pointless for your particular situation, then you don't have to use a tab group, you can just as easily create windows and navigation without the tabs.
