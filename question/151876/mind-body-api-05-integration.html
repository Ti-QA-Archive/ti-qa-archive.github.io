  • Hello,

    suds needs to be adapted to your WS response. In this case, the response from the version 5 is different from 4. You will need to get both responses and analyze the differences (you can use hurl for it). As an alternative, you can use a XML to JSON converter like the one in ApiGee.

    Best,

    Mauro
