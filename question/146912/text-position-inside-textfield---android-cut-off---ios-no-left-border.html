  • Hi Michael,
    Try to use dp or percentage for width and height of textField.
    Change your code with this one. Make the top according to your requirements.

    var firstname = Titanium.UI.createTextField({

        top:'auto',
        borderRadius:3,
        hintText:'First name',
        keyboardType:Titanium.UI.KEYBOARD_DEFAULT,
        borderStyle: Ti.UI.INPUT_BORDERSTYLE_ROUNDED,
        width:'84%',
        height:'37dp',
        font: {fontSize: 22, fontWeight: 'bold'},
        left:'8%',
        right:'8%'

    });

    Hope it will help.
