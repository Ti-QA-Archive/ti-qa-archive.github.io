  • Accepted Answer

    'auto': it is a deprecated property. Please use Ti.UI.SIZE, Ti.UI.FILL respectively. On the other hand 'left' and 'right' must be a pixel site. The dimensions above are for 'width' and 'height'.

    Hope this helps.
