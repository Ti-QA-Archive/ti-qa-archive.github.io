SetRegion - Padding

We have set this region for one of our maps using max lat,long values.

var region = {
                                        latitude : (parseFloat(route_points[i].bounds.latmin) + parseFloat(route_points[i].bounds.latmax)) / 2,
                                        longitude : (parseFloat(route_points[i].bounds.lngmin) + parseFloat(route_points[i].bounds.lngmax)) / 2,
                                        latitudeDelta : Math.abs(parseFloat(route_points[i].bounds.latmax) - parseFloat(route_points[i].bounds.latmin)),
                                        longitudeDelta : Math.abs(parseFloat(route_points[i].bounds.lngmax) - parseFloat(route_points[i].bounds.lngmin))
                                    };



                                    map_view.map.setRegion(region);

The problem is that the points are always pinned against the edge, and is there a way to add padding to the points so that they are not right against the edge?
