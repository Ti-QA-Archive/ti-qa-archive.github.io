Thumbnail disapears when new version of app is installed

Hi,

I'm having some problems, when I run the app in Iphone or IPAD, every new version erases my thumbnails, and I don't know why is it happening.

var thumb_filename = Ti.Filesystem.applicationDataDirectory + "/" + c.draft_id + "_thumb.jpg";
var thumb_blob = Ti.Filesystem.getFile(thumb_filename);

What is funny is that I have a .txt in the same directory, and it remains with all info.

If you need more information, let me know.

Thanks

  • Just ran into this issue. I was storing the full path in a database and the files looked like they were missing whenever I installed a new version of my app. The trick is to store "ONLY" the filename (not the path). When you need to get to your image, rebuild the path and append the filename.

    //Filename only
c.draftFileName = c.draft_id + "_thumb.jpg";
//I assume you will write out this file if needed with something like
//Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory, c.draftFileName).write(...)

//...later when you want to use the file
var thumb_blob = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory, c.draftFileName);

    Hope this helps.
