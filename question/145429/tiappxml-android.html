    I might have not understood correctly what are you trying to do, but if you set 'navBarHidden:true' for the window , like this sample, is it what you are trying to achieve?
    Also, if you set the 'fullscreen' to true, you can also hide the top Android status bar.

    var win = Ti.UI.createWindow({
    title:'title window',
    navBarHidden:true,
    // fullscreen:true
});
