Module question

Hello, i've built a module that has the same name for both target(ie com.mycompany.modulename), android and iphone, when i build for android all goes right, when i build it for iphone, titanium tries to import the module from the android module path, ignoring the iphone version. Any idea?

so for instance, i have the 2 module in

titaniumhome/modules/iphone/0.1/modulename
titaniumhome/modules/android/0.1/modulename

Well, anytime i try to build for iphone, titanium looks into the android module folder, ignoring the right path to the iphone module, so it doesn't find the module library.

any idea?

Cheers

  • Specify the platform in tiapp.xml

    
<module platform="android" version="1.3">my.module</module>
<module platform="iphone" version="1.3">my.module</module>
