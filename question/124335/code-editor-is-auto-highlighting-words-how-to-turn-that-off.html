Code editor is auto highlighting words. How to turn that OFF?

I just did an update and now the code editor in Ti Sutudio highlights every word I click on.
Super annoying. How do I shut that off? I looked but couldn't find the editor option where this is checked.

Thanks!

    Go to: Window -> Preferences -> Aptana Studio -> Editors -> JavaScript and uncheck options: Mark occurrences with background color.
