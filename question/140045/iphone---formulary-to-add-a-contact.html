  • I'm afraid you will have to code the form your self.

    Here what I would do (if I were you):

    • Retrieve the Person you want to edit using the getPersonByID function
    • Create your form and assign all the fields with the properties from the Person object you previously found.
    • Once the user is done modifying the form (by clicking the save button for example) You set all the properties to the Person object with the values from the form fields.
    • You update the contacts Database using the save function.

    Hope this helps.
