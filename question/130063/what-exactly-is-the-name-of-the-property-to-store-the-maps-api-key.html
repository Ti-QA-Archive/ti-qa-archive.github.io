What exactly is the name of the property to store the maps api key?

Kitchen sink has "ti.android.google.map.api.key.production", but online documentation has it as "ti.android.google.map.api.key" (without the .production).

Which is it? And what if there is also a "ti.android.google.map.api.key.development"?

        <!-- this will only work with our dev keystore and not on a production device! -->

    <!-- get yours at http://code.google.com/android/maps-api-signup.html -->
    <property name="ti.android.google.map.api.key.production">GET_ME_FROM_GOOGLE</property>

    I believe you can also use just ti.android.google.map.api.key
