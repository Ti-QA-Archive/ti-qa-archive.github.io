Anyone get the camera to work on an iPad 2

I'm using code from my iPhone app to open up the camera and it seems to create the control in what appears like a pop-up item (pointy arrow UI element on its bottom) - as if it wanted to attached to a button in a toolbar. But, that control flies from view within a second. I was able to see it only after recording it with Photo Booth on my Macbook PRO and then watching it frame-by-frame.

Anyone else playing with an iPad 2 yet?

Here's my simple code snippet:

    Titanium.Media.showCamera ({

        saveToPhotoGallery: true,
        allowEditing: true,

        success: function(event){
            var a = Titanium.UI.createAlertDialog({
                title: 'You did it!'
            });
            a.show();
        },

        error: function(error){
            var a = Titanium.UI.createAlertDialog({
                title: 'Uh Oh...'
            });
            if (error.code == Titanium.Media.NO_CAMERA || error.code == Titanium.Media.NO_VIDEO) {
                a.setMessage('Sorry, this device does not have a camera. You knew that, right?');
            }
            else {
                a.setMessage('Unexpected error: ' + error.code);
            }
            a.show();
        }


    });

  • I wasn't able to get to the lighthouseapp.com link for various reasons.. but did find the target I'm sure everyone's looking for.

    https://github.com/appcelerator/titanium_mobile/blob/47df9dd21bf4febfde5e2a17c946bb9c7a590d62/iphone/Classes/MediaModule.m

    You'll need to be sure that you're running a current version of xcode and have a ios >=4.3. Build the app as you normally would– then using xcode, open the project and replace the MediaModule.m code with the one in the link above. I just selected the text of the one I had, and then pasted the new code in its' place, then saved.
    Build and run it.. off you go- the camera now works!
