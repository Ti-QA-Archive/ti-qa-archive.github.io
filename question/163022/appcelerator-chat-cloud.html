appcelerator chat cloud

Hi,

I am interested to implement cloud chat in appcelerator, I checked the doc http://docs.appcelerator.com/cloud/latest/#!/api/Chats-method-create

But i am not getting any idea on how to pass the user id for chat for the below code. if someone helps its we be greate.

 Cloud.Chats.create({
                 to_ids: ids.join(','),
                message: 'Good morning'
            }, function (e) {
                if (e.success) {
                    for (var i = 0; i < e.chats.length; i++) {
                        var chat = e.chats[i];
                        alert('Success:\n' +
                            'From: ' + chat.from.first_name + ' ' + chat.from.last_name + '\n' +
                            'Updated: ' + chat.updated_at + '\n' +
                            'Message: ' + chat.message);
                    }
                } else {
                    alert('Error:\n' +
                        ((e.error && e.message) || JSON.stringify(e)));
                }
            });

  • Like document says, comma separated user IDs of the recipient(s).

    For example:

    to_ids: ids.join(&#39;4ee05d69d9ca726b85000003,4ee05d69d9ca726b85000004&#39;),
    means this chat messge will send to the chat group who has 3 recipients. their user_id are 4ee05d69d9ca726b85000003, 4ee05d69d9ca726b85000004 and the login user_id.

    You can get user_id by query API of user. Ref: http://docs.appcelerator.com/cloud/latest/#!/api/Users-method-query
