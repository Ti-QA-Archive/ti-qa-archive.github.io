  • There is no out of the box API for converting XML to JSON.

    Some approaches you could use are:

    • If your XML data is retrieved from the web, you can use Titanium.Yahoo YQL functions to to the translation for you.
    • If you want to do the conversion on the device itself, you couls use a JavaScript library that does just that. (Here is one that does just that on GitHub)
