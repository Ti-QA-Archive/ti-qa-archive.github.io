[ERROR] Invalid --ios-version value '5.0' Accepted values:

Any project with 3.0.0.0.GA for Iphone Simulator get this:

[ERROR] Invalid –ios-version value '5.0'
Accepted values:

Any idea ?

  • I had the exact same problem with the difference that I am using Mac Lion instead of Snow Leopard. This is not relevant to the problem. The problem is when you update XCode you need to update the iOS configuration in Titanium Studio to point to the latest XCode installation. The iPhone Simulator is defined within the XCode installation. I re-configured the Titanium Studio 3.0 iOS SDK configuration by executing the following command on a Terminal window. Then I restarted Titanium Studio 3.0 and it works by automatically selecting 5.1 as the iPhone simulator version.

    sudo xcode-select -switch /Applications/Xcode.app/Contents/Developer
