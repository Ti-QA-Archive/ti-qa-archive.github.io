Create app only for smartphones, exclude all iPad and android tablets

I want to create a smartphone only app using sdk 3.x (android & iPhone) but I'm not sure how to exclude support for all tablets.

I can set false for iPad:

<deployment-targets>
    ...
    <target device="ipad">false</target>
</deployment-targets>

So I guess I'm looking for something similar which disallows Android tablets.

It seems a thin line between tablet/phone on Android and the only solution I can think of is to use support-screens but I don't think this is viable due to some tablet screen sizes.

Is there a way to define a 'smartphone' only app?

    When you come to deploy to Play, you can further limit the devices that your app is available for. Obviously you cannot prevent an iPad from running an iPhone app though
