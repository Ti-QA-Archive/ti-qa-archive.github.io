Reading File from Resource folder

I have a set of file in 'Resources\images\animation*.png' Now I want to load them using file system so that I can pass the same blob object for animation to multiple image view. Files are with name Frame[index].png

for(var index=0; ; index++) {
var myFile = Titanium.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory, 'images/animation/Frame'+index+'.png');
}

above code does not return me the file. when I check for exists() it returns me false.

    Titanium.Filesystem.resourcesDirectory would help

    var myFile = Titanium.Filesystem.getFile(Titanium.Filesystem.resourcesDirectory, 'images/animation/Frame'+index+'.png');

    • actually applicationDataDirectory is a writable directory that can be used to store applications-specific files.

      On iOS, this directory is specifically designated for user documents, and a separate directory, applicationSupportDirectory is used for other application data.

      and

      resourcesDirectory directory is read-only. If you need to modify any files in this directory, they must first be copied to another directory, such as
      applicationDataDirectory or tempDirectory.
