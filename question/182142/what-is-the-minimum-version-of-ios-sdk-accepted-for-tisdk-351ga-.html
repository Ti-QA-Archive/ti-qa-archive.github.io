What is the minimum version of iOS SDK accepted for Ti.SDK 3.5.1.GA ?

I have updated my project with new SDK 3.5.1.GA but i can't build my project for iOS6 devices (with 3.4.1.GA is working perfectly).

In release notes (3.5.0), the minimum deployment target for module is iOS 6.0 but nothing on a support iOS6 with a new project Titanium.

Trace Log : 

The device is running iOS 6.1.6, however the app's the minimum iOS version is set to 7.0
[ERROR] :  This app does not support the device "iPod of Titanium"
In order to install this app on this device, lower the <min-ios-ver> to 6.1 in the tiapp.xml:
<ti:app xmlns:ti="http://ti.appcelerator.org">
    <ios>
        <min-ios-ver>6.1</min-ios-ver>
    </ios>
</ti:app>

i have edited tiapp.xml with this statement but it's same issue :

<ios>
  <min-ios-ver>6.1</min-ios-ver>
 </ios>

thanks for your help

    Since SDK 3.4.0 minimum target iOS version: 7.1.x

    http://docs.appcelerator.com/titanium/latest/#!/guide/Titanium_Compatibility_Matrix-section-29004837_TitaniumCompatibilityMatrix-iOSSDK%2FTargetiOSPlatform

    From an official blog comment when 3.5.0.GA was released:

    "iOS 6 support is completely removed in 3.5.0. We refactored a number of classes to take advantage of iOS 7+ functionality. You will need to continue using 3.4.0 or earlier if you need iOS 6 support."
