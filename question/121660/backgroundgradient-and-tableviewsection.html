backgroundGradient and tableViewSection

Hi,

I try to specify a backgroundGradient for my TableViewSection but it doesn't work.

My code:

var section = Ti.UI.createTableViewSection({
    backgroundGradient: {
        type: 'linear',
        colors: [
            {color: '#FFFFFF', position: 0.0},
            {color: '#EBEBEB', position: 1.0}
        ],
        startPoint: {x: 0, y: 0},
        endPoint: {x: 0, y: 60},
        backFillStart: false
    }
});
section.add(row1);
section.add(row2);

This backgroundGradient is ok because I use it in another VIEW. Is it a known issue? Anyone has faced this problem?

Thx!

Romain

    Can't be done. Needs to be on the whole table or individual rows.

    Be warned there may be a performance hit if you use gradients on rows.
