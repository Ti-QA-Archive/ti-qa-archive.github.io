  • Sorry, but this can not be done with this component.

    The Click event is called, but if parameters because you can not
    know in advance which way the user will drag.
    the scroll event will be called only after they have exchanged
    full page, so a pointless animation.

    You can even put animation in views, but they would warm only after they have changed the page

    I believe that to work as you want, you have to create a
    customized component.
    I had a very similar problem and had to create a component
    to the effect of changing the book page and avoid a bug
    the memory component.

    http://developer.appcelerator.com/question/93151/flipbook-component
