button in a table row, both have click event

Dear folk
so I've a table with some row, and each row contain a button, which has a click event that will show an alert dialog, while the row itself has a click event too, which open up a window and show some option.
now when I click on the row the window open up, that fine, but when I click on the button the alert dialog show up as well as the window open up, which is not I expected, is there's anyway(or idea) to allow the alert dialog show up while preventing the window open when the button is click.

    There is no way to stop the event to bubble to the parents.

    To overcome this set a flag to the button, and register only the row's event.
    In the event check if the flag exists (the button is clicked) or not

    var row...

var btn= Ti.UI.createButton({
    ...
    is_btn:true
});

row.add(btn);

row.addEventListener('click', function(e){
    if (e.source.is_btn) {
        alert('btn clicked');
    }
    else {
        alert('rest of the row clicked');
    }

}

    Makes sense?
