Custom fonts with different weight from same family

I have problem in adding custom fonts (with different weights) and using them on iOS, when they are of the same family. How can I achieve that?

I have two files:

familyName-200-ExtraLight.ttf
familyName-500-Bold.ttf

Adding them and using on Android is really simple because you just give the name of the font file without .ttf. Something like this:

var customFontExtraLight = 'familyName-200-ExtraLight';
var customFontBold = 'familyName-500-Bold';

On iOS you just declare the family name:

var customFont = "Family Name";

and that is ok if you are not using two weights of the same family. How can I declare those two fonts from my example?

Thanks in advance for any help!

    Just use the full postscript name for the font family.

    fontFamily: &#39;Family Name Bold&#39;

      Yup. This is exactly how it works.

      Double check your PostScript name. Even though you are using 'Family Name Bold' as an example, I suspect your actual font name has a dash, like 'ComicSansMS-Bold'. 

      var myLabel = Ti.UI.createLabel({
    text : 'Hi. Welcome to my home page! (Under construction)',
    font:{
        fontFamily: 'ComicSansMS-Bold',
        fontSize:'20'
    }
});

      var myLabel = Ti.UI.createLabel({
    text : 'Hi. Welcome to my home page! (Under construction)',
    font:{
        fontFamily: 'FontName-Bold',
        fontSize:'20'
    }
});
