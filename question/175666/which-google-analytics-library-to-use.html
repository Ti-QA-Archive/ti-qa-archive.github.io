  • Hi there
    I've played with both, and like the Mark Tuttle one - since it has a "trackScreen" method - which then ties in with google's mobile analytics nicely. I can see activity happening in the live view on analytics, and it's all very satisfying.

    I'm currently having a problem with Android though - I'm getting this weird "module not found" error - so have started another question related to that:

    http://developer.appcelerator.com/question/175754/google-analytics-mark-tuttle-problem-in-android

    I remember reading (but haven't confirmed) that Roger's one has the useragent set to iOS - so you could have to tweak the code to get it to work for both iOS and Android.

    Let us know how you do, either way - it doesn't look like there are too many options out there, and it'd be cool to get more information on the analytics side of things.
