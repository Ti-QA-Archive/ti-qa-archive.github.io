  • Hi

    The keyboard is the only UI element that will display when clicking a textField or textArea, if you need something else then you need to build it as you need for your app.

    First thing to do is NOT use a textField as this display the keyboard and you then have to try and hide it - this never works as you want. Instead display a label, place your value in the text property and add the event listener click to the label. In the even listener display the picker you want to use and then update the label text property as you need to.

    Later on you can extend this by the following;

    • control the position of the picker
    • your even add it to a transparent window and display the window if you need it to cover tabs
    • you can make the view/window animate in from the bottom like the soft keyboard does by just adding an animate method

    Have fun.
