    This is more a regex related question that a Titanium one, but still.

    You could extract all the image tags from your code using the following expression:

    /<img\s+src\s*=\s*(["'][^"']+["']|[^>]+)>/

    By the way, all your img tags seem to have absolutely no quotes for your image's src attribute. Better have that checked out.
