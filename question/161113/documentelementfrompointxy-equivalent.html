document.elementFromPoint(x,y) Equivalent

Is there an equivalent to this not in a webview? If I had four buttons, all with touchmove, touchstart, and touchend listeners, I would want to know on touchmove if I have left the element where the touchstart began.

My code on a webview worked just fine inside a touch event.

        var touches = e.originalEvent.touches;

        for(var x = 0; x < touches.length; x++) {
            var touch = e.originalEvent.touches[x];
            var el = document.elementFromPoint(touch.clientX, touch.clientY);
            touchedIds[x] = el.attr('id');
        }
