Connect and interact with a distant database ?

Hi all !

I'm a new developper on titanium's IDE. I have a project that I started, and i have to connect with a distant database (created with phpmyadmin and mysql). Is it possible with titanium ? Because javascript only do things with database but in the client side, not in the server side. I don't want that the user of my app got some data from a database that is not updated…

Someone know how to do that ?

Thanks for your help :) !

Application type : mobile

Platform : Windows 8.1

Device : Android simulator and blackberry device.

  • Titanium should not interact with your database directly. Actually, NO client should be able to connect to your database directly. You need to expose access to your database via an API.

    Then you will use a sync adapter (Alloy) or an httpClient call (Classic) to interact with the api that you have created.
