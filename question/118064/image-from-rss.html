Image from RSS

I'm working on a project for someone that has an RSS section, and I'm parsing a Wordpress feed.

I'm not sure how to access this part of the feed

<media:content url="http://0.gravatar.com/avatar/a8744b15c67bd76744e982e4a2923f35?s=96&#38;d=identicon&#38;r=G" medium="image">
            <media:title type="html">uspta</media:title>
        </media:content>

I'm using

 Ti.API.info('image :' + itemList.item(c).getElementsByTagName("media:content").item(0).text);

but I'm getting the Title instead of the image.
