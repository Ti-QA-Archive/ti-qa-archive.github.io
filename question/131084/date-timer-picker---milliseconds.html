Date Timer Picker - Milliseconds

Hi,
I would like to know how to get the Milliseconds from a Date Picker time. I had tried use this:

picker.addEventListener('change',function(e)
{

var date = new Date(e.value);
Ti.API.info(" Millis " + date.getMilliseconds());

});

But the Millisseconds is 0. Anyone can help me please?

    I assume you are creating a picker with type: Titanium.UI.PICKER_TYPE_DATE_AND_TIME or something similar. It doesn't appear that there are milliseconds returned with that date value, so what you are seeing is correct, milliseconds is 0.

    But, are you sure that you are actually looking for milliseconds, and not something along the lines of a timestamp, or the number of milliseconds since the epoch? If so, you will use date.getTime() for that.

    Here's an example. Note that the value returned in e.value is already a date object, so new Date(e.value) isn't necessary:

    var win = Ti.UI.createWindow({
    backgroundColor : '#fff'
});

var picker = Ti.UI.createPicker({
    type : Ti.UI.PICKER_TYPE_DATE_AND_TIME,
    selectionIndicator : true,
    value : new Date(),
    bottom : 0
});
win.add(picker);
win.open();

picker.addEventListener('change', function(e) {

    var d =e.value;
    Ti.API.info('Date: '+d.toString());
    Ti.API.info('Epoch: '+d.getTime());
    Ti.API.info('Milliseconds: ' +d.getMilliseconds());
    Ti.API.info('Minutes: '+d.getMinutes());

});
