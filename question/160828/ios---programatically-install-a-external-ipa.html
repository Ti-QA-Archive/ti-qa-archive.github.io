    In the iOS world are two possibilities to install:

    1. via iTunes and USB cable
    2. onAir (iTunes store uses the same method)

    To realize on air you need 3 parts:

    1. http:// - URL (i.e. index.html), in this html a link or javascript links to a itms-service -URL to
    2. manifest.plist, in this plist file is a link to the
    3. IPA file.

    On opening the manifest.plist a dialog appears. Now you can install.

    Here an example

    For linking to first URL you can use a QR-Code. In this case the user starts the scanner app and the install dialog appears immediately.
