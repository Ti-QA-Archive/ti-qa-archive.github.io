Scrolling a webview with evalJS

Hi.
Since a webview in a scrollview is an heavy task for titanium (though it works), I was looking for a way to use only the webview, without the scrollview.

I need one simple thing: scrolling vertically to a pre-determined point in the web document.

My solution would be to call an evalJS in the webview 

var webview = Ti.UI.createWebView(
// url, etc...
);
webview.evalJS('window.scrollTo(0,'+yPos+');');
// yPos is an integer defined in other part

but I'm getting to nowhere (newbie here, sorry), the page remains at the start position, so can someone kindly tell me the best way?
Thanks!

  • I've found the way, and I'm really happy for this HUGE improvement in performance for my app.

    I report the code if someone needs the same, or similar, task:

    var webview = Ti.UI.createWebView(
// url, etc...
);

webview.addEventListener('load', function(){
        webview.evalJS('window.scrollTo(0,'+yPos+');');
    }

    For completeness, yPos is defined in another part or page , for example in a way like this (if you want to know the scroll Y position of the webview):

    var yPos = webview.evalJS('window.pageYOffset;');

    Aloha!
