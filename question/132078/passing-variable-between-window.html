  • you can

    1) Set global variable as application variable between windows,
    for example:

    Ti.App.VariableName = [value];
var variableName = Ti.App.VariableName;

    2) Set values into properties,
    for example:

    Ti.App.Properties.setString('propertyName', Value);
var variableName = Ti.App.Properties.getString('propertyName');

    for more information follow this link:
    http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.App.Properties-module

    3) pass values between windows as property,
    for example:

    var win1 = Ti.UI.createWindow({
    title : "Hello Testing",
    url : "window.js",
    variableName : [value],
    ...
    variableName : [value]
    });
win1.open();

    and another side:

    var win = Ti.UI.currentWindow;
var variableName = win.variableName;
