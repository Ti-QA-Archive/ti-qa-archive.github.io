    • It seems you are mixing up Google Maps JS V3 API and Google Maps Android API. You can not use any of the meta-information and permissions of you manifest from the JS API. So at first, you should remove all the android manifest stuff from the tiapp.xml to get rid of the uncaught reference error. Then you can continue on investigating on the Webview workarround. But I don't think it is a real alternative to a native MapView.
