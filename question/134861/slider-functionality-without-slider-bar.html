Slider functionality without slider bar

I want to implement rating functionality in my app for which I want to display a row of stars in grey colour, as the user slides over the stars, they should appear selected (in colour yellow). So basically I want to implement slider functionality but without displaying the slider bar.

Any suggestions?

    hi Subramanian,

    actually it's not as complex as Gagan said..i have developed this feature before..

    i have used view's touchmove method to set the star images..

    first i have created 5 images like

    1 selected star and other blank

    2 selected star and other blank

    3 selected star and other blank

    4 selected star and other blank

    5 selected star

    then
    create an array with 5 names like var arr=['1.png','2.png',…];

    i have created a view and set the backgroundImage as default then on the touchmove method of the view i checked the condition of the co-ordinates then as per the co-ordinate i have set the value of variable like 0,1,2,3,4 and at last changed the backgroundImage of the view with view.backgroundImage = arr[<value>];

    good luck
