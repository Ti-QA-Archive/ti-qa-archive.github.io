    I'm happy to report that this issue has been resolved with the version 1.2 beta 2.1 bug fix release. Release pushed to marketplace. Please see GitHub project page for quicker access to release updates.

    GitHub Page

    Version 1.2.1.2.1 (1.2 beta 2.1) bug fix

    • project package moved to GitHub
    • Bug fix updates to /core/timvc.js
      1. Update util.dP function to return default px value for iPhone
      2. Update window open url to remove resource directory base path (no longer needed)
    • Bug fix update to /components/controller.js
      1. Added setter function setLayout to allow layout change on the fly
    • tested on 1.8.2.x (V8 engine) with success on Android and iPhone
