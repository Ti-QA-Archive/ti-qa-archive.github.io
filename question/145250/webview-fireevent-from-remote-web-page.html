WebView fireEvent from remote web page?

I'm trying to establish communication between items in a WebView and my main code. I'm able to use Ti.App.fireEvent from html pages that are hosted on the phone (are part of the app), but when I do the same from a remote URL (an html page not hosted on the phone), the events seem not to work.

Is it possible to throw events via javascript in html pages that are remote?

  • Accepted Answer

    Only local html has access to the Titanium namespace. If you need to communicate with a webview, the html must be in the applications resources directory.
