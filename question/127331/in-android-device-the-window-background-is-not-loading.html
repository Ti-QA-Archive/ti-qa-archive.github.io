In android device the window background is not loading

Hi all,

I tried to set background for window... The background is visible in android emulator but when running it on device the background is not visible following code is used to set the backgroung image. Can any one help?

if(Ti.Gesture.isLandscape()) {
myApp.syncWindow.backgroundImage = constant.LANDSCAPE_SPLASH_IMG;
} else {
myApp.syncWindow.backgroundImage = constant.PORTRIAT_SPLASH_IMG;
}

  • Have you checked to make sure your constant is pointing to the correct path on your device? With changes (I think beginning in 1.7) there have been changes to how the path gets resolved. I have changed all paths in my projects to prepend the directory before the path, something like: Titanium.FileSystem.resourcesDirectory+'/images/backgroundImage.png'
