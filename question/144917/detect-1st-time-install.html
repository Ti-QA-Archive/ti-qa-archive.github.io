Detect 1st time install

Any idea how to detect 1st time install an app? Even if the phone has been reset and app is removed, it will not consider as 1st time anymore if previously installed.

  • if (!Ti.App.Properties.hasProperty('firstrun'){
    //Do first run type stuff
    Ti.App.Properties.setBool('firstrun', false);

}
