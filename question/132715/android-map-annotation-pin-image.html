Android Map Annotation pin Image

Am I doing something wrong here ? I don't see the .png image, which I have in my android folder root:

if (Ti.Platform.osname === "android") {
    pinButton = "mapRightArrow.png";
  } else {
    pinButton = Titanium.UI.iPhone.SystemButton.DISCLOSURE;
  }
  allPins = (function() {
    var _i, _len, _results;
    _results = [];
    for (_i = 0, _len = m.length; _i < _len; _i++) {
      value = m[_i];
      _results.push(Ti.Map.createAnnotation({
        id: value.id,
        latitude: value.la,
        longitude: value.lo,
        title: value.text,
        subtitle: value.address,
        pincolor: Titanium.Map.ANNOTATION_GREEN,
        animate: false,
        rightButton: pinButton
      }));
    }
    return _results;
  })();

    It should be in your resource folder. Try to put that image in the folder where your give js file is located(as you have not used related path). You can also use related path like ../images/mapRightArrow.png if you want to use image file in other folder of resources than the folder which contains your js file.
