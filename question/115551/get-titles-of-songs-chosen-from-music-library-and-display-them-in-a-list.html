  • ok looks like you were on the good track by accessing the TiMediaItem. Judging from TiMediaItem.m we can access some information about the tracks that are picked.

    If you do a Ti.Media.OpenMusicLibrary(settings) whereas you define in settings the following:

        var settings = {
      success:function(picked)
      {
          Ti.API.info(picked.items[0].title); // outputs the title of the first selected track.
      }
}

    picked.items is an array with all the picked tracks. You can access the values defined in TiMediaItem.m like so. So wanting to display the title should be as easy as just looping through the items and for example adding a tableviewrow with a text label that refers to picked.items[i].title

    Hope this helps and I sure hope Appcelerator wakes up and adds consistent documentation about these things so we don't need to 'reverse-engineer' all these controller files :)

  • showMusicLibrary() is just as it says it is: show the music library. iPhone and iPad only.

    It opens up a Modal window that shows all the Music in the players "iPod" app on their iPhone, iPod Touch, or iPad. There is no way, as far as I know, to filter this data, display it in a different way, or modify it.
