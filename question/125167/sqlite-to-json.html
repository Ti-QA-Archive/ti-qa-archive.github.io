SQLite to JSON

Hi,
how can I convert to json a table row ?
I have tryed something like this : 

var rows = Ti.App.dbVit.execute(sqltext);
if (rows.rowCount > 0) {
   while (rows.isValidRow()) {
      Ti.API.info (JSON.stringify(rows));
   }
}

But the json array is empty.

    manually?

    but try this…

    if(rows.rowCount) {
  var arr = [];
  while(rows.isValidRow()) {
    var obj = {};
    for(i=0; i<rows.fieldCount; i++) {
      obj[rows.fieldName(i)] = rows.field(i);
      arr.push(obj);
    }

    rows.next();
  }
  Ti.API.info(arr);
}
