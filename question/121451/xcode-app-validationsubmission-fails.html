Xcode: app validation(submission) fails

Hi everybody,
I'm trying to distribute an app for iPad (iOS SDK 1.4!?). I have created the profile in iTunes Connect and I did the packaging through Titanium. When it opens Xcode I opened organizer and I tried to validate my app.

After I entered my info (username and password for iTunes Connect) Xcode gave me the following error:

No suitable application records were found.

Please make sure that you have set up a record for this application on iTunes Connect.

Note: This is the only app I have so one provisioning profile (for distribution) one app id (at all).

Please help!

    Hi Ani,

    You need to sign into iTunes connect and create application and then ready it for binary submission.

    Otherwise, Organzier will not find it when it goes to Validate.

    • Go to Manage Your Application and select the version that you want to upload. View the Version Details and then click the Ready to Upload Binary. Once you do that, your status should change to "Waiting for Upload".

      It is only at this point that you can Validate and Submit your app through Organizer.
