By default the horizontal alignment for imageView is center, how to set this alignment to right (or left) without set width oh this imageView ?

    Ok, you cannot align an image inside of an image view. The image will always be in the center of the image view as it expects the size of the image view to be the size of the image. So if you set the image width to 100%, then the image is treated as 100%. If you want the view to be the same size as the image you put into the view, then set the width to 'auto', this will allow you to set a right or left value to define alignment.

    As an example, this will align an image to the right in the default project.

    imgv = Ti.UI.createImageView({
    width:'auto',
    height:43,
    right:0,
    image:'KS_nav_views.png'
});
win1.add(imgv);

  • Im new with Appcelerator. I wanted to have the image alignment to left right and center in a single row.
    What worked for me was setting right: 0 to right align.
    To have the center property I didn't set a right property.
    To have the left align I set the left: 0

    I got three columns in a single row. Hope this helps for someone.
