  • Hi

    Given your syntax it looks like you are not using Titanium and JavaScript and instead you are using native Java - if so this is not the forum for you. If the syntax is simply an over-copying error then read on.

    This;

    private static final String createRepairStatus = "CREATE TABLE "+ TABLE_REPAIR_STATUS + "(_id INTEGER PRIMARY KEY, status_name TEXT);";

    Would normally be;

    var sql = "CREATE TABLE "+ TABLE_REPAIR_STATUS + " (_id INTEGER PRIMARY KEY, status_name TEXT);";

    This assumes that TABLE_REPAIR_STATUS is a variable and has a string assigned to it.

    Please also note there was a typo in the sql statement;

    TABLE_REPAIR_STATUS + "(
// becomes
TABLE_REPAIR_STATUS + " ( 
// not the extra space after the "
