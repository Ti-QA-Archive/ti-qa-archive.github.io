Titanium cannot find Android SDK

I'm trying to get started with Titanium Studio. I've done a fresh install of Titanium and the Android SDK. Android SDK is installed to ~/lib/android-sdk. I fire up Titanium and tryto create a new project, but when selecting my Android SDK directory it complains that "Could not locate the Android SDK at the given path".

There should not be anything wrong with the SDK because (a) it's a fresh install and (b) it works fine with Eclipse.

I've tried the symlink workaround with ./platform-tools/adb (actually, I've tried symlink, hardlink, and even a copy). Titanium refuses to find it.

What can I do?

  • I managed to fix this myself. I had to install the Android 2.1 SDK. Apparently version 1.7.3 of the Titanium SDK requires the Android 2.1 SDK and won't work with 2.3.3 alone.

    This was mentioned neither in the Quick Start nor the "Installing and Updating Android SDK" link. I happened to stumble across it by chance in the Titanium Compatibility Matrix. (http://wiki.appcelerator.org/display/guides/Titanium+Compatibility+Matrix#TitaniumCompatibilityMatrix-MinimumSystemRequirements).

    This should really be documented more clearly if it's such a pitfall.
