Is it possible to write a try/catch block that catches ALL exceptions?

Ok…I have a database insert statement that fails on trying to insert duplicate rows.

         try{
             insertDetail(itemDetail);

           } catch(e){
                Titanium.API.info('Couldnt insert'); 
           }

I've tried the standard (JAVA) way to catch ALL exceptions…but this doesn't work. Is there a way to ALL exceptions in Javascript so my app doesn't crash?

  • You cannot catch "all" exceptions, but I have seen that in the recent continuous builds more of the SQL exceptions are now being thrown. Previously almost none were.

    So try one of the latest continuous builds and that is about the best you can do.

  • Ok, it looks like you can't catch Database exceptions.

    What I did as a workaround is to use "INSERT or INGORE" instead of just "INSERT". This prevents the error from even being thrown.
