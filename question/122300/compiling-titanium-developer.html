Compiling Titanium Developer?

Hi guys.

I've downloaded Titanium Developer source files from github and am trying to compile it using Titanium Studio (now and then I prefer Developer to Studio - its more lightweight). Anyway, I get this error "this app was created by another user". I was just curious, are we actually not permitted to use Titanium Developer anymore? The binaries aren't available from the downloads page and it doesn't look like we're able to compile it from the sources.

Is there some way around this compilation issue or are we all stuck using Titanium Studio unless we were lucky enough to grab the developer binaries while they were around?
