Refresh Image Without Fade

Hi everyone!
I asked this question about 10 month ago:
Is there a way to stop the alpha fading transition after loading a new image url? I got something like an "webcam" view, just an image that gets refreshed. Something like:

var loadImage = function() { 
   var seed=Math.random();
   camUrl = "http://url.com/image.jpg?"+seed;
   camImage.url = camUrl;
} 

loadImage();
setInterval(loadImage,5000);

Works fine, but there is this annoying alpha fade transition when reloading the image.
Is there a workaround or any setting meanwhile (without a webview)?
