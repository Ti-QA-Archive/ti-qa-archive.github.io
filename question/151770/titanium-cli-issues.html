    Hi suthan

    before installing cCLI, you need to imstall node.js version 0.8 and then install CLI through terminal using command in cmd
    for mac

     sudo npm install titanium -g

    for windows

    npm install titanium -g

    for more help please refer this link to install CLI

    http://docs.appcelerator.com/titanium/latest/#!/guide/Titanium_Command-Line_Interface_Reference-section-35619828_TitaniumCommand-LineInterfaceReference-CLIQuickStart

    • Titanium SDK 3.0.2.GA and older will only work on Node.js 0.8.x. You are running Node.js 0.10.0.

      Appcelerator people fixed the problem https://jira.appcelerator.org/browse/TIMOB-13025 and the upcoming Titanium SDK 3.1.0 will work on Node.js 0.10.0.

      Your options are to either:
      Downgrade to Node.js 0.8.22 http://nodejs.org/dist/v0.8.22/
      Use a CI build http://builds.appcelerator.com/#master. If you use a CI build, note that it's not fully tested and should be used for development only.
