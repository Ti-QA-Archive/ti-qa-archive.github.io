Forcing a button to the bottom of the screen in Android

Hi

I'm trying to position a screen element so that it is always at the bottom of the screen. On the face of it, "bottom=0" would do the trick, but it seems that when using layout=vertical, the bottom=0 thing is always ignored.

I'm quite keen not to use absolute postitioning (given the different sizes of Android screens) - so is there a way to do it?

    Yes, You can Do. like this,

    var win = Ti.UI.currentWindow;
var main_view = Ti.UI.createView({
    height :"100%",
    width :"100%",
    backgroundColor:"transparent",
    layout:"vertical",
    zIndex    :0,// lower then btn
});

var btn = Ti.UI.createButton({
    title :"Button",
    height :50,
    width :100,
    bottom :0,
    zIndex    :1,   // higher then main_view
});
// you can add you other object in main_view. whose you want to add on Window. 
// You can also use ScrollView instead of View.
win.add(main_view);
win.add(btn);
