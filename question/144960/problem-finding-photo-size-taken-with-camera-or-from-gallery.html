Problem finding photo size taken with camera or from gallery

Hi all,

I use this code:

Titanium.Media.openPhotoGallery({
        success : function(event) {
        var image = event.media;
    alert(image.height);
    alert(image.width);     
}       
});

The problem is i always get same values for height and width no matter the image i chose (and there are many different sizes photos). And those values are not in pixels, thats for sure… because i can read the image size from my gallery.

It happens with android, iPhone, both devices and simulators.

Why is this?
