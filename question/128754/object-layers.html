    I will try and explain why …. Try being the word here.

    Firstly I am not sure that adding buttons to labels is what you should be doing. maybe creating a view to add a button to would be a simpler way.

    Secondly, doesn't the second box overlay the first. Creating a scrollview you may need to add layout: vertical.

    Now to explain the ordering of items. as you add an item to a view or a window or any other component then that will add onto the previous component so if you looped through the components the first one added would be 0 in the array then 1 2etc

    This means that adding an event listner to a component then if you have overlaid another component then it will not fire as the second component is on top of the first.

    Best described as a pack of cards, you can only acces one if nothing is on top of if it. and the components in titanium are stacked accordingly.

    Well that is my effort to try to explain how components are added. I hope it helps

    T.
