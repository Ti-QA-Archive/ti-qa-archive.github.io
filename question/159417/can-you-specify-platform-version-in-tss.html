Can you specify platform version in tss?

Is it possible for a style to target a specific platform version?

example: By adding something like the following to my app.tss

".topbar[platform=android]": { top: 0 }
".topbar[platform=ios version=6]": { top: 10 }
".topbar[platform=ios version=7]": { top: 20 }

  • Jaques, while it's true you cannot do it in the manner you asked about specifically, you CAN do something like this…

    alloy.js

    if (Ti.Platform.osname == 'android') {
    Alloy.Globals.topbarTop = 0;
} else if (Ti.Platform.osname == 'iphone') {
    if (Ti.Platform.version >= '7.0.0') {
        Alloy.Globals.topbarTop = 20;
    } else {
        Alloy.Globals.topbarTop = 10;
    }
}

    Then in your .tss

    ".topbar": { top: Alloy.Globals.topbarTop }
