  • I have experienced the same error message and for me it was an illegal character in the Tiapp.xml . We had changed the Tiapp.xml, adding a description which contained enters / new line characters. Although the Tiapp.xml editor does handle the new line characters, the build fails giving the above error message.

    We figured out the exact problem, by opening the Xcode project in the build folder and try to build it from there.Xcode immediately refered to the illegal characters in the description.

    So I'm not sure if this applies to everyone encountering this error, but this was our fix.
