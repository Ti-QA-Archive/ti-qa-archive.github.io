  • Since the date picker uses the native Android picker we must style the text using a theme. The theme is applied to the window containing the picker. Here is an example where I made the text color black (Theme tutorial).

    var win = Ti.UI.createWindow({
    theme: "Theme.BlackText",
});

    <style name="Theme.BlackText" parent="@style/Theme.AppCompat">
    <item name="android:textColorPrimary">#000</item>
</style>
