    Hello, sqlite does not have an array datatype. You could try using in the model's column obj

    "pictures": "text"

    then store a stringified representation of your array, with

    pictures : JSON.stringify(json[0].pictures);

    You will need to JSON.parse() the field value in order to handle it again as an array. hth.
