  • Hi

    As your code snippet does not show that you have displayed the popover I can only assume that you have shown it.

    If you are trying to show the "passcode" on the popover you must actually add it to the popover you created - right now you are adding it to the "familyScreen" which again I must assume is the window/view behind your popover.

    // change
$.familyScreen.add(passcode.getView());
// to
popover.add(passcode.getView());

    • You are using the iPad popover but then not using any of the popover user interface. If you are not using any of the popover interface you should not be using the popover.

      You should create your own using normal views that site over the top of the previous views. You will have to manually place the "custom popover tip" (the arrow) as there will not be any built in function as the real popover does not suit your needs.

      There is a custom module that allow you to modify the visuals of the existing popover - this may suit your needs. I have never tried so I cannot offer assistance in using it - but if the view approach does not work for you - you can try this;
