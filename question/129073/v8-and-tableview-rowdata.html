V8 and tableView rowData

I don't know if I'm doing something wrong, but when I try to run my Android application with V8 (tried several nightly builds) I get a crash, when trying to access a tableview's event's rowData properties.

    controller.categoryView.tableView.addEventListener('click', function(e) {
    controller.blockTableClick = true;
    controller.loadSelection({
        categoryId : e.rowData.id,
        name : e.rowData.name
    });
 });

I use the code above which works fine with rhino, but with v8 I have e.rowData.id and e.rowData.name undefined.

Is rowData somehow not supported any longer? Is there some other way to go about this now? Does anyone else have that problem?
