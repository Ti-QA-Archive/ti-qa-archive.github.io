  • Not sure exactly how to specify this during build with Studio, but in order to do this normally you need to add a CFBundleIconFiles array to your Info.plist file. That array should hold 2 string items which point to both your app icon files. The correct file will be chosen based on the size of the icon and the device. Doing this should fix the validation error.
