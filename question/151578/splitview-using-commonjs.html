  • Hi

    If that code is your CommonJS file then you are missing a few important things;

    • at least one function block (in this case)
    • at least one export statement

    A simple CommonJS for reference;

    win.js CommonJS

    function create(o) {
    var obj = o || {};
    var win = Ti.UI.createWindow({
        height: Ti.UI.FILL,
        title: obj.title || 'Unknown',
        width: Ti.UI.FILL,
    });
    return win;
}
exports.create = create;

    app.js Usage

    var Win = require('win');
var win = Win.create({ title: 'Test' });
win.open();

    You do need to return something in the main function if you want to reference it in the calling script.
