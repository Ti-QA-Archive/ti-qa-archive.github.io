Server Video Download and save in iPhone Gallery

Hi Guys,

I have video at server, i'm downloading it and want to save it to iPhone gallery. I use the same technique as i download image from server and save it to iPhone photo gallery using

Titanium.Media.saveToPhotoGallery () method. Same method i'm using for video to save in gallery. I'm doing right thing?

Please help me how to save video in gallery.

  • You can only save a video in the gallery if you've recorded it from your devices camera. You cannot download and save videos from a remote server into your gallery.

  • After download, you can add it in PhotoGallery with this simple code:

    var file = Ti.Filesystem.getFile(Ti.Filesystem.resourcesDirectory, "video.MOV");
var blob = file.read();
Titanium.Media.saveToPhotoGallery(blob)
