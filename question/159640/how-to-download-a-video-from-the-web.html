  • Hi davide,
    for the download(url) function you'll want to use a Titanium.Network.HTTPClient to make a GET request for the video url. In the success callback save it in desired location.

    You can write files in the applicationDataDirectory (both iOS and Android) or in applicationSupportDirectory (iOS only) or externalStorageDirectory (Android only). ApplicationCacheDirectory, as far as I know, is read-only.

    For more information look also at Ti.Filesystem docs.

    hth.
