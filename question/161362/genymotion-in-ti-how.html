Genymotion in ti, how?

My spec:
Ti 3.2.0.GA

Mac OS X 10.8.5 2.2GHz i7 with 8gig ram

  • To get this working for me I had to make sure that the following from the article that Tim mentions above is done in a terminal.

    First we need to make sure Titanium is identifying your Genymotion emulators, and to do that you need to tell Titanium that you’ll be using Genymotion. Type:

    ti config genymotion.enabled true

    Then when TiStudio was re-started it found the virtual devices I had installed.
