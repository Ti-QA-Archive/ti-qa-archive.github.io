Force a new GPS-Location

Hi

Is there any way to force the iPhone to get a new location? (Not the last one)
At the time I use the following 2 lines, but I don't really know what they are for.

Titanium.Geolocation.accuracy = Titanium.Geolocation.ACCURACY_BEST;
Titanium.Geolocation.distanceFilter = 10;

Thanks

Felix

  • Accepted Answer

    Use the location event to trigger a new gps search.
    Keep in mind that it takes time to trigger.

    Titanium.Geolocation.accuracy = the accuracy you need ( if it's a low accuracy it might as well use the network location instead of gps)

    Titanium.Geolocation.distanceFilter = how often to trigger the location event ( in this case you move 10m, location is triggered, another 10, and so on…)
