    Hi Santiago Aguilar Córdoba 

    Cloud.Objects.create({ 
    classname: 'Example',
    fields: { 
        ids: [1,2,3,4,5],
        title : 'mytitle'
        }
     }, function (e) { if (e.success) {
    } else {
         alert('Error:\n' + ((e.error && e.message) || JSON.stringify(e))); } });

    If this makes problems, you can try to JSON-stringify the array.
