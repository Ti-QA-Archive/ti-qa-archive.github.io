plist files and iCloud

Hi,

I would like to ask if the plist files created with Ti.App.Properties.setObject are being backed up on iCloud or not. I would like to save some user (preferences) data there and the user to be able to retrieve them if he reinstalls the app. Does anybody know for sure? :)

PS: I have already activated iCloud from the apple dev account btw.

iOS 5.x, 6.x
TiSDK 3.1.0

  • Onoma, backing up app preferences to the cloud requires storing them using the NSUbiquitousKeyValueStore which a grep of the 3.1.0 sdk didnt return a hit.

    It probably wouldn't be too hard to modularize this missing ios component though, as it wouldn't require an gui/view proxy support
