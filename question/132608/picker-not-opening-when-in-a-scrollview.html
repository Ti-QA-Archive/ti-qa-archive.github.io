Picker not opening when in a scrollView

Hello everyone,

I am trying to put a picker inside of a scrollView on Android. It gets placed fine, and when you click on it, it turns orange.. But it does not pop up the list of items. Here is the piece of code I am using to do so.

this.barpicker = Ti.UI.createPicker({
  top: 50,
  left: 50,
  zIndex: 2,
});

this.barpicker.add([
   Ti.UI.createPickerRow({
   title: 'bar1',
  }),
  Ti.UI.createPickerRow({
    title: 'bar2',
  }),
  Ti.UI.createPickerRow({
    title: 'bar3',
  }),
]);

this.viewMain.add(this.barpicker);

When I add the picker directly to the window, it works fine (but of course, does not scroll with the view).

Thanks for any help

  • It might have something to do with how you have defined your scroll view. I used your code to create a simple app and it seemed to work fine (SDK version 7.1.5).

    Here is my app.js, hope it helps:

    Titanium.UI.setBackgroundColor('#000');

var win = Titanium.UI.createWindow({  
    title:'Test App',
    backgroundColor:'#fff'
});

var barpicker = Ti.UI.createPicker({
  top: 50,
  left: 50,
  zIndex: 2,
});

barpicker.add([
   Ti.UI.createPickerRow({
   title: 'bar1',
  }),
  Ti.UI.createPickerRow({
    title: 'bar2',
  }),
  Ti.UI.createPickerRow({
    title: 'bar3',
  }),
]);

var viewMain = Ti.UI.createScrollView({ top: 100, left: 0, right: 0, contentWidth: 'auto', layout:'vertical', backgroundColor:'#000' });
viewMain.add(this.barpicker);
win.add(viewMain);
win.open();
