  • Hi

    I found this blog post Augmented Reality Markers with Appcelerator Titanium to be very helpful. Which explains how to use the library tiarmarker by Atsushi.

    It is designed to work with Titanium so you will find it very easy.

    I used this to create a few useful proof of concepts including mapping a video player over a marker in 3d and then playing and pausing when the video comes in frame. The library above makes that sort of thing easy.

    The £d Earth is actually very easy, the example is simply a series of PNG image frames that are cycled through (this can be done with Ti.UI.ImageView), the centre point is anchored to the returned event handler.

    Have fun.
