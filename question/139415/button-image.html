  • If this is for a mobile project, you need to do it this way with image locations relative to the Resources folder :

    Also, use a forward slash.

    If it's a desktop project you need to work out which way the slashes should go using the getSeparator method (but still call the image relative to the Resources folder:

    http://developer.appcelerator.com/apidoc/desktop/latest/Titanium.Filesystem.getSeparator-method.html

    
var spots = Ti.UI.createView({
    image:'images/Jason.png',
    height:'auto',
