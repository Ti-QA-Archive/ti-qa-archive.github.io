Text field Vertical Align

Hello,
I'm using Titanium Studio, build: 3.2.1.201402041146, and my operating system is Windows 8. I have an android alloy mobile application project.
I want to align my text fields content vertically center.

var txtSearch = Ti.UI.createTextField({
    bubbleParent : false,
    height : 30,
    width : '70%',
    top : 111,
    backgroundColor : '#fff',
    hintText : 'Search...',
    font : {
        fontSize : 11
    },
    borderRadius : 5,
    verticalAlign : 'TEXT_VERTICAL_ALIGNMENT_CENTER',

});

Even though I've set the vertical alignment property to center it wont work.
Can any body help me with this?
Thanks in advance!
