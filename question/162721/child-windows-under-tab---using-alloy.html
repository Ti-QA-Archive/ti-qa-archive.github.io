Child Windows Under Tab - using Alloy

I'm using Titanium 3.1.3 and currently testing the application in Android.

My app has got tabgroups. I've 2 tabs and I'm using 'require' to include tabs in the group. My tab windowswould be having many child windows. For example, on clicking a button inside my tab window, it should navigate to another window inside the tab itself.

tabgroup xml file

<TabGroup id="tab" color="green" >
        <Require src="one" />
        <Require src="two" />
</TabGroup>

one.xml

<Alloy>
    <Tab id="one" title="ONE" icon="images/one.png">
        <Window id="win" class="container">
        </Window>
    </Tab>
</Alloy>

For a button click inside this, event in one.js is as follows

var controller = Alloy.createController('inner');
controller.openMainWindow($.one);

inner.xml 

<Window class="container" id="inner">

And in the inner.js controller, 

exports.openMainWindow = function(_tab) {
_tab.open($.inner);
};

With this code, when I click on the button, it's opening the next window; but NOT inside the tab. It is opening as a new screen.

Please advise

  • That's the native behavior of Android platform. If you need the window not to hide the tabs adjust its height/top.
    Are you just passing the tab to the window controller to open it ? you could just do:

    var controller = Alloy.createController('inner');
$.one.open(controller.getView());
