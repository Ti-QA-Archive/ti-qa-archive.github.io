  • I installed titanium on windows a number of times in the last couple of weeks. Here is what I usually do :

    1- install java sdk to a path with no spaces (as you did)
    2- install Android to a path with no spaces
    3- add the jdk bin folder to the path
    4- add JAVA_HOME environment variable C:\jdk1.6.0_24 in your case
    5- I go to the control panel -> java and update the settings to find and use the new jdk
    6- RESTART !

    this worked every time.
