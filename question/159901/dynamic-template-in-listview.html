Dynamic template in listview

I have used a list view in my app. In list view, my template has some child views like below code(ListView ). But for specific reason, i need some dynamic template that means for some specific condition my template will have some views( in childTemplates - see 2nd section code) but for another condition i want to add more views/label in my childTemplates. how can i do this ? i can't put any condition in the plainTemplate. if i use like below -

if(somecondition){
var plainTemplate = {

        childTemplates : [
            //{} views/label
        ]
    }
} else if(another condition){
var plainTemplate = {

        childTemplates : [
            //{} add more views/label with previous
        ]
    }
}

then it will be copy and past of more code that i want to avoid. how can i do this ? is it possible or not to check the condition in the template ? if not then what will be the best solution ? please help me out.
Below is main code -

    Hi

    The ListView is designed to have the rows defined at creation and populated with data as required. You cannot change the layout of the templates on the fly. But you can have more than one template in your ListView and your controller code can determine which template is used to display each row.

    See the following answer I gave on the subject a few weeks ago.

    • ListView - different control combinations for each line?

    • You can also achieve this in the XML directly as weel;

      <ListView id="listView" defaultItemTemplate="subtitle">
    <Templates>
        <ItemTemplate name="title">
            <Label bindId="title" id="main" />
        </ItemTemplate>
        <ItemTemplate name="subtitle">
            <ImageView bindId="icon" id="icon" />
            <Label bindId="title" id="title" />
            <Label bindId="subtitle" id="subtitle" />
        </ItemTemplate>
    </Templates>
    <ListSection headerTitle="Fruit / Frutas">
        <ListItem title:text="Apple" es_info:text="Manzana" icon:image="http://placehold.it/40/000044/ffffff&text=1" />
        <ListItem title:text="Banana" es_info:text="Banana" icon:image="http://placehold.it/40/000044/ffffff&text=1" />
        <ListItem template="title" title:text="John Doe" />
    </ListSection>
    <ListSection headerTitle="Vegetables / Verduras">
        <ListItem title:text="Carrot" subtitle:text="Zanahoria" icon:image="http://placehold.it/40/000044/ffffff&text=1" />
        <ListItem title:text="Potato" subtitle:text="Patata" icon:image="http://placehold.it/40/000044/ffffff&text=1" />
        <ListItem template="title" title:text="Hello World" />
    </ListSection>
    <ListSection headerTitle="Grains / Granos">
        <ListItem title:text="Corn" subtitle:text="Maiz" icon:image="http://placehold.it/40/000044/ffffff&text=1" />
        <ListItem title:text="Rice" subtitle:text="Arroz" icon:image="http://placehold.it/40/000044/ffffff&text=1" />
    </ListSection>
</ListView>
