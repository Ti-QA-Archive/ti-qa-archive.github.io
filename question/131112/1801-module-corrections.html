1.8.0.1 module corrections

The module created by "titanium create" command does not compile and run for Android using 1.8.0.1 SDK.

First in order to build, go to the Project Properties, select "Java Build Path" and add external JAR references to kroll-apt.jar and kroll-common.jar. I could not get Titanium Studio to build at this point, but Ant builds just fine.

Second, I tried creating the template module both from the command line and from Titanium Studio, and then created a test application using the example app.js file (including the <module> statement in tiapp.xml). No code changes were made. This resulted in the require() statement in app.js causing a module not found exception. Turns out that my module filename was camel case - changing the module name to all lower case allowed the example application to run.
