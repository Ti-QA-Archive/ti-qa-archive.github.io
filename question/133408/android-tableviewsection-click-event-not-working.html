Android TableViewSection Click event not working?

I think I've found an issue with TableViewSection on Android. The click event only seems to work when a label is present in the row and only if the label is clicked. Clicking anywhere else on the row doesn't trigger the event.

Here's some code to test this:

Ti.UI.setBackgroundColor('#ffffff');
var win = Ti.UI.createWindow();

var section1 = Titanium.UI.createTableViewSection();

var row1 = Ti.UI.createTableViewRow({hasChild: true});
row1.add(Ti.UI.createLabel({text: 'Label'}));
section1.add(row1);

section1.add(Ti.UI.createTableViewRow({title: 'NoLabel', hasChild: true}));

section1.addEventListener('click', function(e){
            Ti.API.info('Section1 clicked');
            alert('Section1 clicked');
        });

var tableOption = {
    data:[section1]
}
var table = Ti.UI.createTableView(tableOption);
table.addEventListener('click', function(e){
            Ti.API.info('table clicked');
        });

win.add(table);
win.open();

SDK: 1.8.2
Emulator: 2.2 and 4.0.3
Mac OS X Lion

You'll have to click on the label text to trigger the alert.

The tableview click event seem to be working fine so I think I'll change my code to use that meanwhile. On the iPhone it there seems to be no issue at all.
