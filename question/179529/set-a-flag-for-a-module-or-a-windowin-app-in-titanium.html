  • Accepted Answer

    Setting up the flag depends on the requirement:
    For persistance storage of flag you can use:

    Ti.App.Properties.setBool('flagname', '1');

    and you can retrieve the value using 

    Ti.App.Properties.getBool('flagname', '1');

    Other option is if you are using alloy then you can set value using:

    
Alloy.Globals.Flagname = 1; //  this variable can set value
 Ti.API.info(Alloy.Globals.Flagname); //to retrieve in any of the window.

    Another option is you can set value in SQL Lite database and can retrieve the value form it an any of the window.

    I think on of the options is the answer you are looking for.

  • You can use Ti.App.Properties.setBool, there is also a getBool. You can use this to set and get your flag wherever you are inside your app.

    — answered December 9th 2014 by Luke Taylor
    permalink
    3 Comments

    • Here is an example of usage.

      // To set
Ti.App.Properties.setBool('flagName', true);

// To get
var flag = Ti.App.Properties.getBool('flagName');
