  • Hi

    This one has been coming up recently and often.

    Something for you to check and set if it has no current value.

    • Menu File
    • Menu Properties
    • Select "Run/Debug Settings"
    • Select your configuration
    • Choose Edit

    Make sure both iOS SDK and Simulator Version both have values - if not please choose a value, chances are only iOS SDK is affected.

    Once you have done this;

    • Close Simulator (if running)
    • Menu Project
    • Menu Clean
    • Select your project
    • Select OK

    Then re-build.
