TitaniumAlloy Textfield width based on orientation

How do I change Text Field width based on orientation (alternately, have the width auto-expand according to the user's input). Mainly for ios at this point.
One code version I've tried (which does change anything) is:

Titanium.Gesture.addEventListener('orientationchange', function(e) {
    if (Titanium.UI.LANDSCAPE_LEFT || Titanium.UI.LANDSCAPE_RIGHT) {
        $.grTxt.length="240dp" };

Where as grTxt is the id for the TextField.

    Hi

    The usual way is to provide a text field that shows the expected amount of information the user is meant to provide - so an expanding text field based on content provided is really not a good user experience or provides sufficient visual clues to how much information is expected.

    Changing the width during an orientation change is fine as long as the user should have more space with this field. The easiest way to create a TextField that changes size during orientation changes is to use the width of FILL and set a left and right value - these act as margin/padding. When the device rotates the TextField adjusts leaving the same margin/padding;

    Titanium with Alloy framework

    ".yourtextfield": {
    left: 100, // set to a suitable margin
    right: 50, // set to a suitable margin
    width: Ti.UI.FILL
}

    <TextField class="yourtextfield" />

    Titanium without Alloy framwork

    var txt = Ti.UI.createTextFIeld({
    left: 100, // set to a suitable margin
    right: 50, // set to a suitable margin
    width: Ti.UI.FILL
});
