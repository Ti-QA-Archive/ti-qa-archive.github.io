  • Accepted Answer

    You need to set onload and onerror BEFORE calling open. You try to set onload AFTER you open the request.

    And is better to use

    json_data = JOSN.parse(this.responseText);

    instead of 

    json_data = eval('('+this.responseText+')');

    Also this part of the code won't work outside the onload event

    var imageCount = json_data.fighters.length;
    Ti.API.info(imageCount);

    • The xhr request is asynchronous, that means that if you try to get the images somewhere after the xhr you will get nothing.
      images gets populated after the onload event triggers (which happens after the data is downloaded from the internet).

      So if you try to show the images somewhere you will need to do this in the onload event, after you populate the images array, just like the displayImages function.

      Makes sense?
