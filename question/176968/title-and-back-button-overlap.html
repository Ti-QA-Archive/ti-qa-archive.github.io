  • You have a few options:

    • Set the Back button title to something shorter with Window.backButtonTitle property
    • Set the Window's leftNavButton property to an imageview of an icon that represents "back" to your users
    • Use a label in the Window's titleControl and set a smaller font size
    • Use a shorter Window title
