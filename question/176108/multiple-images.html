Multiple Images

I am trying to add a new image each time someone presses a button, the code below is working but it's removing the previous image. I would like it to keep adding a new image which all show side by side until X amount are visible. 

var image= Titanium.UI.createImageView({
   image: "flag.png",
   height:20, 
   width:20, 
   top:10,
   left:5
}); 

function flagIT(){
    $.videoContainer.add(image);
}

  • It's probably not replacing the previous image, just stacking them atop one another. You can put the parent view into horizontal layout mode so that the images are next to each other.
