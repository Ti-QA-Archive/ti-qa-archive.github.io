Script Error = Couldn't find module

Hello, I'm having problems requiring modules in a mobile app (Titanium studio 1.0.7.201112281340, SDK 1.8.0.1) on OS X Lion:

I placed the module (ti.storekit 1.3) in the dir /Library/Application Support/Titanium/ -having in it the subdir modules/iphone/ti.storekit/1.3- then added a node for it in the modules section of tiapp.xml and required in app.js as usual.

The problem is I'm always getting a "Script Error = Couldn't find module…".

I also found different modules usage instructions at https://wiki.appcelerator.org/display/tis/Using+Titanium+Modules, tried them too, but obtaining the same error.
Now I'm a bit confused on what's going wrong, clearly I'm missing something.
Any hint is welcome. TY in advance.

  • try this:

    unzip the module. copy the whole ti.storekit directory directly to /Library/Application Support/Titanium/modules

    so my module is here: /Library/Application Support/Titanium/modules/ti.storekit/iphone/ti.storekit/1.3/..

    and now its working. other modules worked like a charm, with "just put the zip into modules dir", but storekit not.

    hope this helps!
