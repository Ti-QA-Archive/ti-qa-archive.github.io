android: open window in app.js as "leightweight"

hello,
since i've switched to 1.7.1 and Ti Studio, my (android) app opens the window defined in app.js apparently as a "heavyweight" window, e.g. with NavBar visible. before, with Ti Developer it opened the window fullscreen and as a "leightweight" window. any idea how & where i can change that? the problem is that mapviews don't load in heavyweight windows, see also here . thanks for any hint, uno

  • I found that "modal" is the property that is making mapview work. Either set it to true or false and it will work but, the window will behave like a modal window. I've tested fullscreen and navbarhidden properties but those don't work with mapview.

  • The following code is working fine on Android:

        // Create mapView window
    navigationWindow = Ti.UI.createWindow({
        exitOnClose: false,
        fullscreen: false, // -> using fullscreen property -> heavyweight window
        modal: true, // -> using modal property -> heavyweight window too
        title: L('tab_1_item_3_label')
    });

    // Create mapView and add to window
    mapView = Ti.Map.createView({
        exitOnClose: false,
        mapType: Ti.Map.STANDARD_TYPE,
        region: region,
        regionFit: true,
        userLocation: true,
        annotations: annotations
    });
    navigationWindow.add(mapView);

    Using SDK 1.7.2. See also: http://developer.appcelerator.com/question/122172/map-view-bug
