Data binding to the ListSection of a ListView

All,

I would like to use data binding with ListView in Alloy markup. I have looked at the binding_listview Sample but cannot figure out an approach for what I am trying to do.

I have a collection which is a list of stores. Each model has a "state" attribute. I want the list to be sorted by state with the state name as the title to each ListSection title.

Is this possible with Alloy or do I need to do it manually? If possible how do I go about it? I am not looking for someone to write the code for me. Just a nudge in the right direction?

Thanks in advance for any advice you can provide!

  • Yes, it's possible. The Alloy repo example seems a really good place to start. Now that you know it's possible, why don't you follow the example and try to implement it yourself.
