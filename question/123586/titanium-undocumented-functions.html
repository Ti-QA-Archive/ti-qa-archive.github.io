Ti.UI.Window.setTitleControl()

Sets the title control on the nav bar, useful for showing an activity indicator amongst other things.

see tabbedbar.js for an example.

  • Ti.UI.ActivityIndicator has undocumented properties location and type (for Android).

    After a window is open, this code will show the activity indicator and message in the nav bar (even though the code says "STATUS_BAR"). The message will clobber the app name in the nav bar, and the spinner will appear on the right side of the nav bar. You have to have navBarHidden:false for this to work:

    var spinner=Ti.UI.createActivityIndicator({
              location: Ti.UI.ActivityIndicator.STATUS_BAR,
              type: Ti.UI.ActivityIndicator.DETERMINANT,
              message:'Loading something...'
             });
spinner.show();

  • Ti.Filesystem.externalStorageDirectory, which gives you the app's designated SD card directory (on Android only).

    Ti.Filesystem.applicationSupportDirectory, which gives you app's data directory on iPhone (the database subdirectory of this is where your active database is stored).
