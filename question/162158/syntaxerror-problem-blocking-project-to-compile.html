  • OK so you have updated the CLI, Alloy and SDK, unfortunately, there is a problem with compiling using the 3.2 versions with the 3.1 SDK's…

    There not backwardly compatible.

    Two options.

    1) Revert back to 3.1 CLI etc

    2) Move unto 3.2.1 RC which enables using 3.1.3 SDK's

    look HERE for how to revert or update

    T.
