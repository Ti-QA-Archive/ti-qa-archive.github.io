Android giving "Uncaught ReferenceError" when using variable defined in the glboal scope

Hi all,

I have declared an object in my bootstrapper file. On this object, i'm setting various modules that I need to use throughout my App.

My app.js file

Ti.UI.backgroundColor = "#000";

// Define app namespaces in the global scope

MyApp = {};

MyApp.Transactor = {};

MyApp.UI = {};

MyApp.AppLogic = {};

MyApp.AppLogic.ClientInfo = require("applogic/ClientInfo");

var ApplicationTabGroup = require('ui/common/ApplicationTabGroup');

new ApplicationTabGroup().open();

This plays out fine on iOS. I am succesfully using the MyApp.AppLogic.ClientInfo module in the ApplicationTabGroup.

But, on android, im getting: "Uncaught ReferenceError: MyApp is not defined".

I'm running low on ideas on how to fix this :)
