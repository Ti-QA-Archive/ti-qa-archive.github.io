  • If you have a firewall:

    You need to add the following ports to the firewall's white list

    https://api.appcelerator.net (ports 80 and 443) Used for login

    http://studio.appcelerator.com (ports 80 and 443) Used for the dashboard

    http://download.appcelerator.com (ports 80 and 443) Used for software
    updates

    Also, please post a diagnostic log to narrow down the problem, i.e. Help > Titanium Studio > View Log File

    And, you might want to complete the following checklist in order to verify vital information.
