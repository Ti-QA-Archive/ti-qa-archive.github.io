Problem with including module

Hello,

I bought the module BDActivityViewController (https://marketplace.appcelerator.com/apps/3829?2139240552) and can't get any support from the developer. Their website is also down so I try it here.

I can't get the module running it always gives me an error "Couldn't find module: uk.co.betadesigns.titanium.module.bdactivityviewcontroller". I'm using the SDK version 3.0.2. GA.
In The TiApp.xml it's included like this:

    <modules>
        <module platform="commonjs" version="2.3.1">ti.cloud</module>
        <module platform="iphone" version="1.3">ti.admob</module>
        <module platform="iphone" version="1.0">uk.co.betadesigns.titanium.module.bdactivityviewcontroller</module>
    </modules>

In my app it's included like this:

var bdactivityviewcontroller = require("uk.co.betadesigns.titanium.module.bdactivityviewcontroller");

Any suggestions?

    • Library/Application\ Support/Titanium/modules/ in your HOME.

    Then you can select module inside Studio
