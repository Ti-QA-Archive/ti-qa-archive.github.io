invalid IOS-SDK version error

Hi

I am on Snow Leopard. I installed XCode 4.2 and latest Titanium Studio and SDK is 2.x. Whenever I try to compile my project it gives invalid iOS-SDK. On RUN CONFIGURATION I only find Version 5.0 in drop down. Even the version.plist file /Developer/Platforms/iPhoneOS.platform tells that I am using SDK 5.0 but Titanium Studio Keeps saying: [ERROR] Invalid –ios-version value '5.0'

Please help me out!

  • If you run titanium info -t ios from the command line, you can see towards the top all of the Xcode installations that it found as well as each Xcode's iOS SDKs. If you don't see any 5.0 SDKs, then you Titanium didn't find any Xcode installs with 5.0. If you do have 5.0 installed for a version of Xcode that doesn't show up, then you can move that Xcode install into the /Applications directory and make sure the Xcode directory is named "Xcode.app" or "Xcode.app" (where is anything you want). Otherwise you can "xcode-select" the Xcode install that has the iOS 5.0 SDK and Titanium will pick that up.
