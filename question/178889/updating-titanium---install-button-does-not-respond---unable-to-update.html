I've seen this issue reported in Jira several times, but no solution given. I am running Titanium Studio on an iMac OSX Yosemite 10.10. It is quite frustrating. I am happy to provide screenshots and logs if it helps.

  • Thanks Harish, but the links you sent were way too generic. Pointing me to the home page of the Installation and Configuration docs certainly didn't help much. I didn't have the patience to go through it, so I googled a little bit more and ventures to install everything manually. I needed to update for packages:

    • Alloy
    • node.ACS
    • Titanium CLI
    • Titanium Studio

    The first three, I could do it using npm, like this:

    $ sudo npm update -g alloy
    $ sudo npm update -g titanium
    $ sudo npm update -g acs

    Then I reopened Titanium Studio, went to Help->Check for Titanium Updates, and the only remaining item was Titanium Studio itself. I accepted terms and conditions, and this time the install button worked (something is wrong with npm and studio then).

    Hope this helps!!

  • Finally, i got it to work. It seems that two paths for SDK locations caused this problem. I have changed SDK location in studio preferences to /Library/Application Support/Titanium/. This solved problem with updates.
