Problem with video inside webview

Dear All,

I'd like to open the whole webpage which includes videos, and entire of the page by using a Titanium webview. When I click on the play button on the video, It doesn't work on Android. (It works on iPhone) I'm testing using Google 2.3.3 API and Titanium SDK 3.1.1 I tried many ways like:
Adding and add the line 

settings.setPluginsEnabled(true)

to the constructor of TiUIWebView.java and recompiling the Titanium SDK source
or adding 

android:hardwareAccelerated="true"

to the tiApp.xml but they all didn't work.

The sample code looks like below:

var win = Ti.UI.createWindow({
    backgroundColor : '#ffffff',
    fullscreen : true,
    navBarHidden : true,
    exitOnClose : true,
});

var url="http://www.youtube.com/watch?v=V70ruRzEGOI";

var webView = Ti.UI.createWebView({
    url : url,
    scalesPageToFit : true,
    touchEnabled : true,
    top : 0,
    backgroundColor : '#FFF'
});

win.add(webView);
webView.show();
win.open();

I think this is a Titanium bug that Appcelerator has not fixed it so far.

How about your idea?
