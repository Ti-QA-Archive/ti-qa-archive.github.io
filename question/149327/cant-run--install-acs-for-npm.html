  • Okay - I've figured it out - the docs need updating to show that the installation instructions are slightly different for Windows…

    Omit the SUDO command and from a good old fashioned Windows Command Prompt, simply type:

    npm install -g acs

    It will then take a minute or two and run through a huge install batch with one or two non-critical errors (mostly about missing doc files and one about a missing optional item)

    It then completed correctly as far as I can gather and returned me to my c:\ prompt :)
