  • Windows are parent elements, they are the only thing that does not need to get added to anything to be viewed on the screen. Views are like containers within windows that help to organize the view elements such as images, text, and such.

    Besides adding extra pages to your app, windows can be used to force content to show above everything else.

    Windows also can create new Code Contexts when using the URL to load another js file, and in doing so create an app within an app so to speak.

    Hope that helps :)
