  • Judging by the screenshot, you might not have the standalone Titanium Studio installed, but rather as a plugin to your existing Eclipse, or you actually have the Aptana Studio installed instead of Titanium Studio (in this case, check the Aptana Studio preference tab to change the background color of your editor).

    I would try re-installing the standalone version one more time using this procedure:

    1. Uninstall by dropping the entire Titanium folder from your Applications to trash
    2. Go to http://preview.appcelerator.com/studio/ and click the Download for OS X link near the top of the page
    3. Launch and check the Titanium Studio -> About Titanium Studio, the about box should read "Titanium Studio, build: 1.0.5.x" (at the time of writing)

    Let me know if this helps.
