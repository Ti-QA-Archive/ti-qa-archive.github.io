  • If you want to put this js file in the widget, so it can be used only within the widget and the widget can be packaged without having to have other dependencies (like the js in the app assets folder) then it's better to create a lib folder in the widget folder and use WPATH to access it:

    var myJS = require(WPATH('myJS'));

myJS.someMethod();
