  • Unfortunately, you can't in a cross-platform way. :(

    You can hack your way around it referencing internal private API, but it could in the future without notice and break your code.

    I did some quick testing and it appears to only work with color. It's going to take more hacking to see if it's possible to change the font size and such.

    if (Ti.Platform.osname == "mobileweb") {
    mywin._titleControl && mywin._titleControl.color = "#FFF";
}

    • OK, you were pretty close in your sample code. You only need to set the titleControl on the window. Here's a working example:

      var win1 = Ti.UI.createWindow();

var mt = Ti.UI.createView({
    backgroundColor: 'green'
});

var label = Ti.UI.createLabel({
    color: '#FF0',
    text: 'Hello!',
    font: {fontSize:20,fontFamily:'Helvetica Neue'},
    textAlign: 'center',
    width: 'auto'
});
mt.add(label);

var win2 = Ti.UI.createWindow({
    titleControl: mt
});

var nav = Ti.UI.MobileWeb.createNavigationGroup();

win1.add(nav);
win1.open();
nav.open(win2);

      Even though win1 isn't doing much, I'm pretty sure it's required. In the example, you may notice that the green view doesn't stretch the full width… you may have to tinker with it a bit or possibly stuff a view in the left and right title region.
