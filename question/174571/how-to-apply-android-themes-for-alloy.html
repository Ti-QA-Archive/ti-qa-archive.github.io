How to apply Android Themes for Alloy?

Hi guys,

I wanted to create my own theme for android like it's described here: http://docs.appcelerator.com/titanium/3.0/#!/guide/Android_Themes

Somehow the documentation is only for the traditional titanium apps.

How do I achieve the same for my alloy app? I mean this step:

To use the "Theme.MyTheme" theme in your application, modify the Android section of your tiapp.xml file:

<android xmlns:android="http://schemas.android.com/apk/res/android">
    <manifest>
        <application android:theme="@style/Theme.MyTheme"/>
        <!-- For the Titanium SDK 3.2.x and earlier, when using the Holo Theme
             you can only use Android 3.0.x and later -->
        <uses-sdk android:minSdkVersion="11" android:targetSdkVersion="14"/>
    </manifest>
</android>

Where do I have to put the file for the theme "MyTheme"? For traditional titanium apps, it would be here: platform&#x2F;android&#x2F;res&#x2F;values&#x2F;mytheme.xml but for alloy there is no platform folder.

Thanks

    Hi Emre Avsar,

    You can use android theme same process in alloy like traditional titanium.But Platform folder is not created previously, you need to create it in the root project ./platform/android/res/values/xml file. You can also download res zip folder for theme from here. And unpack this res folder paste ./platform/android/thisLocation. After all add the theme name in tiapp.xml file like the sample code. For more info Android Holo Colors Generator

    <android xmlns:android="http://schemas.android.com/apk/res/android">
    <manifest>
        <manifest android:installLocation="preferExternal"
            android:versionCode="1" android:versionName="1.0">
            <application android:theme="@style/AppThemeName"/>
            <uses-sdk android:maxSdkVersion="19"
                android:minSdkVersion="11" android:targetSdkVersion="16"/>
        </manifest>
    </manifest>

    If you have any concern about this query let me know.
