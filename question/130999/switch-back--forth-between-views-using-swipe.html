  • You could handle the swipe event on a window maybe:

    Ti.UI.currentWindow.addEventListener("swipe", function(e) { 
  if(e.direction == "right")
    //close window to show previous window
  else if(e.direction == "left")
    //open new window
}

    Possibly a better solution would be having each of your windows be views that are initialized on load, with the first one showing and the rest being placed off the screen to the right. Then you can move the first view to the left and slide in the next view from the right as they swipe. I'm guessing someone else may chime in with already written code to do this.
