  • ###I find my answer for Alloy, You must:

    Keep the same folder estructure:

    - res-long-land-hdpi
- res-notlong-land-hdpi

    Put the images in a new folder inside those folders, so you has something like this:

    "/app/assets/android/images/res-long-land-hdp/icons/icon.png"

    And then you call the image like this:

        "/image/icons/icon.png".

    It's not like the documentation said, if you don't put the slash before the "image" folder, it won't work.
