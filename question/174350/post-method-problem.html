  • You are sending is JSON so I suggest you change the request header as follows:

    xhr.setRequestHeader('Content-Type', 'application/json');

    If that doesn't work, you need to look at the logs on your server as it states in the errors you are getting:

    [ERROR] : TiHttpClient: (TiHttpClient-1) [3193,3193] HTTP Error (org.apache.http.client.HttpResponseException): Internal Server Error

    In addition, the onload function needs to use this.responseText in the following:

    Ti.API.info("This    " + this.responseText);
