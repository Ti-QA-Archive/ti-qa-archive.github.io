    So you performed an action that precipitated the change.

     I've got Xcode 5.0.2 (5A3005) and version 7 of the iOS Simulator installed

    And Apple removes the older sims. Users of xCode know you can download older versions of the simulator from within xCode.

    Regardless, why doesn't Titanium just use 7?

    Because, were it to do that, some other user would come along and bitz about how titanium changes all their stuff. The project was set for sdk 6.x before it was removed.
