How to control the NetworkActivityIndicator on iOS

Applicationtype: iOS app, 3.4.1.GA, iOS all devices, OSX yosemite, build: 3.4.1.201410281727

On iOS it is possible to control the NetworkActivityIndicator which is the indicator that shows up in the top status bar on iOS devices. In xcode it looks something like this:

(UIApplication *)app = [UIApplication sharedApplication];

if (!app.isNetworkActivityIndicatorVisible) {
    app.networkActivityIndicatorVisible = YES;
} else {
    app.networkActivityIndicatorVisible = NO;
}

I need a way to do this with Titanium, but it seems it is not to be implemented, or maybe (very likely) I just can't find it.

Using the normal ActivityIndicator element it not an option as it is impossible to place correctly in the statusbar because the carrier information varies from user to user.

The root problem I am trying to solve is that a WebView does not show this indicator when loading a page as I think it should, as it has a property to suppress this behaviour (hideLoadIndicator = true);
