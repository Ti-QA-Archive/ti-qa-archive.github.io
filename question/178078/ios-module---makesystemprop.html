iOS Module - MAKE_SYSTEM_PROP

I am curious how you use the MAKE_SYSTEM_PROP work in an iOS module.

Basically I need an Enum for my Module

typedef enum {
    ONE = 1,
    TWO = 2,
    THREE = 3
} kCount;

How would I expose this in the Module?

Also were is the documentation about this sort of stuff? I have read http://docs.appcelerator.com/titanium/3.0/#!/guide/iOS_Module_Development_Guide. But it seems to be just a basic guide.

  • I've not done this, but looking at this class within the iOS SDK, it does it like this (line 69):

    ...
typedef enum {
    kTransformNone = 0,
    kTransformCrop,
    kTransformResize,
    kTransformThumbnail,
    kTransformRoundedCorner,
    kTransformTransparentBorder,
    kTransformAlpha
} TransformType;

MAKE_SYSTEM_PROP(TRANSFORM_CROP,kTransformCrop);
MAKE_SYSTEM_PROP(TRANSFORM_RESIZE,kTransformResize);
MAKE_SYSTEM_PROP(TRANSFORM_THUMBNAIL,kTransformThumbnail);
MAKE_SYSTEM_PROP(TRANSFORM_ROUNDEDCORNER,kTransformRoundedCorner);
MAKE_SYSTEM_PROP(TRANSFORM_TRANSPARENTBORDER,kTransformTransparentBorder);
MAKE_SYSTEM_PROP(TRANSFORM_ALPHA,kTransformAlpha);
...

    Hope this helps
