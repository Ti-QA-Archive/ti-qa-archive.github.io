iPad Local Video

I would like to add a local video to an iPad App. I have used the KitchenSink code from local_movie.js, however when I use this and specify my movie location i receive an error:

Result of expression 'Titanium.Media' [undefined] is not an object for this line:
scalingMode:Titanium.Media.VIDEO_SCALING_MODE_FILL

Does someone have some simple code to use for a full screen movie to just play and then once finished playing to go to a js page I specify?

Any help is appreciated.

    This sounds like you just need to force a full build of the project. In Ti Studio, use menu Project > Clean then try to build the project again.

    You can get undefined object messages when you try to rebuild a project but have added a new class for the first time (Ti.Media in this case). Because Titanium only includes the modules for the classes referenced to reduce the application size, you need a full rebuild to pick up a new class.
