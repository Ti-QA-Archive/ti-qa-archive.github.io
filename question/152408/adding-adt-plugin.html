  • ADT is a plugin for Eclipse and is used by Android Java developers. You do not need it to use Titanium Studio to build & run an app on the emulator or device. The NDK is also not needed for development of mobile apps with Titanium Studio. It is needed for native module development.

    It sounds like you don't have your Java/Android environment set up correctly. I don't have either ADT or NDK installed on my system and I can do Android development with Studio just fine. I suggest you revisit the setup guides to be sure you have everything installed and running properly.
