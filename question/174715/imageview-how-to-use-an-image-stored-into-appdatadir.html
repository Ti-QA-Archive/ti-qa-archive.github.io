    Hi

    Looks like your problem might be more basic than you imagine;

    // you are referencing this file path
Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory, '/img/miniature/' + filename);
// you are getting this error
/img/miniature/8.jpg (No such file or directory)

    This looks as simple as one or both of /img/miniature/ folders do not exist inside the application data folder.

    You will need to create them;

    // this folder needs creating
Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory, '/img/');
// and this folder needs creating
Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory, '/img/miniature/');
