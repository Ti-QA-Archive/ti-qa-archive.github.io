    • Hi

      Feeling nice today, we use this function:

      Central.prototype.pause = function(numberMillis) {
    var now = new Date();
    var exitTime = now.getTime() + numberMillis;
    while (true) {
        now = new Date();
        if (now.getTime() > exitTime)
            return;
    }
};

      Part of a common js module, but easy to call.
