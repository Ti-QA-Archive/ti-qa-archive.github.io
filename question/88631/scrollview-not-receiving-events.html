ScrollView not receiving events

I have a scrollView to which I am trying to add an event listener for click/singletap, but it seems as if the listener isn't working – at least not in the iPhone simulator.

My code looks like this:

var win = Titanium.UI.currentWindow;

var myScrollView = Titanium.UI.createScrollView({
    anchorPoint: {x:0,y:0},
    backgroundColor: '#FFF',
    top: 0,
    width: 967,
    height: 662,
    layout: 'vertical'
});

// child/content view - contains some labels and imageViews
var innerview = Ti.UI.createView({
    anchorPoint: {x:0,y:0},
    top: 0,
    left: 10,
    width: 947,
    height: 'auto',
    layout: 'vertical'
});

myScrollView.addEventListener('click',function(e)
{
    alert('If you see this, it worked.');
});

myScrollView.add(innerview);
win.add(myScrollView);

Has anyone else run into this issue?

  • Possible Answer 1: Set zIndexes

    It could be that the click event isn't firing because you are actually clicking on the view element. By setting the zIndex of the scrollview as 2 and zIndex of the view to 1 it may fix the issue.

    Possible Answer 2: Track Event by AlertDialog or Debug

    It may be that the alert function isn't working. I haven't had any luck with the alert function. It doesn't come up for me, but I've only tested it on Android. So I use Titanium's alert dialog (see docs) or I use the Ti.API.debug(&#39;send message to console&#39;) to track events.
