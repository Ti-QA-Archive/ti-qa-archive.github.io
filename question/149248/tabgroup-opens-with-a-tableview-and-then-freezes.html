  • Hi

    My first thought would be to edit these lines;

    search.addEventListener('click',function(event){
    Titanium.API.info(this);
});
conditonsListings.addEventListener('click',function(event){
    Titanium.API.info(this);
});

    I would get rid of this section as you have tied the searchBar to a tableView and it will then act as a filter to the tableView contents.

    search.addEventListener('click',function(event){
    Titanium.API.info(this);
});

    I would change this as follows;

    conditonsListings.addEventListener('click',function (event) {
    Titanium.API.info('table click: ' + JSON.stringify(event));
});

    BTW save your self some typing and use Ti instead of Titanium, they mean the same thing.

    Example;

    conditonsListings.addEventListener('click',function (event) {
    Ti.API.info('table click: ' + JSON.stringify(event));
});
