Spaces in appname

Just starting a new project with a space in the name will break the compiler in Android. Is there some sort of workaround to have the appname contain spaces?

    As Trevor mentioned, an app name cannot contain spaces. My best guess is that you want to have spaces in the name name that the user will see (under the icon or the store, etc.)

    This is actually a different property on Android that is defined in the AndroidManifest.xml file.

    Simply edit the file and change the label in the following section.

    <application
    android:icon="@drawable/appicon"
    android:label="App Name With Spaces"
    android:name="AppNameWithoutSpaces"
    android:debuggable="false"
>

    • The link was information in regards to where you should pu your AndroidManifest.xml file so it doesn't get overwritten at compile time.

      As for your label problem, here is the code you can use:

      <manifest android:versionCode="2" android:versionName="1.2">
<application android:debuggable="false"
    android:icon="@drawable/appicon"
    android:label="U&Me" android:name="CquotApplication">
    <activity android:configChanges="keyboardHidden"
        android:label="U&Me"
        android:name=".CquotActivity" 
        android:theme="@style/Theme.Titanium">
        <intent-filter>
            <action android:name="android.intent.action.MAIN"/>
            <category android:name="android.intent.category.LAUNCHER"/>
        </intent-filter>
    </activity>
</application>
