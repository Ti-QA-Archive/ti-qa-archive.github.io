[Solved} Large center tabbar tab like gowala

A popular new iphone design trend is the use of a large center tab for the apps core function. Gowala and Path are a good example of this.

I have spent days trying to figure out how to do something like this using titanium using things like the custom tabbar class and such that all involved a LOT of customizing, and resulted in slow performance once they worked and looked how I needed them.

So how can you get results that look good, and have good performance?

Step 1) Create a centered tab with no image or title.

Step 2) add a window from your apps root file (app.js) and open it like so:

var w = Titanium.UI.createWindow({
        height:50,
        width:65,
        bottom:3
});

var a = Titanium.UI.createAnimation({duration:0});

w.open(a);

Step 3) Add a button to the the window with the image you desire
