Can blind individuals write working apps using the technologies provided by this SDK?

Hello all,

I just read about this system for developping apps and it seems very appealing. My only concern is accessibility, though.
I am fully blind, you see. I am currently unable to write a working app for, say, IOS, because the interfaceBuilder tool Apple provides is completely ' inaccessible' . This means screenreaders that visually impaired people use to access their computers, are unable to interact with this tool in a way that is helpful and even useful to them. This SDK appears to be a very good alternative, but I guess thats where my question comes in.
Can blind individuals write apps using this? And also, will these apps be accessible? I'd say Yes to the second question because native UI components are used, but I am quite new to developping and coding in general so I don't know for sure. Any help or feedback on this topic is very much apreciated.

Thanks a lot,
Florian

  • The software you need to work are:

    • Any texteditor for coding
    • Titianium Developer for creating the project and compiling
    • The iOS simulator for testing your application

    The text editor shouldn't be a problem.

    Titanium Developer doesn't look compatible with screen readers and the one built into Windows didn't do a good job. However I think you can work around that and find solutions where you can work with it.

    The really tricky part will probably be the simulator that is made by Apple. You could deploy to the unit every time though.

    I think Titanium Developer is one of your best options available so give it a try.
