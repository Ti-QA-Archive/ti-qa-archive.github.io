Read value from previous window in alloy

Hi
I want to pass some values from one window to another window.
I am passing it like this

//passing argues
var second=Alloy.getController('second',arg1,arg2);

// Reading argues
var arg1=arguments[0] || {},
var arg2=arguments[1] || {},

I am getting value of first argument but unable to get second value.

    it should be like this

    var second=Alloy.getController('second',[arg1,arg2]);

// Reading argues
var args_array=arguments[0] || null;
var arg1= args_array && args_array[0] || {};
var arg2=args_array && args_array[1] || {};

    or what I like to do is

    var second=Alloy.getController('second',{ 
    "args1" : "value one", 
    "args2" : "value two"
});

// Reading argues
var args=arguments[0] || null;
var arg1= args && args.args1;
var arg2=args && args.args2;
