Recreate Window Each Time Tab Clicked

I'm using Ti Developer 1.6.2 and IOS 4.3.

I have a window that is launched by a TabGroup tab. The code inside the window works perfectly, but I would like to completely replace the content each time that tab is clicked. The code inside the window only executes one time, when the window is opened the first time.

In my app.js, I defined a window focus event for the win variable created in my CreateWindow, and attempted to reload controls when that fires, but it doesn't seem to affect my content even in the simplest of examples. I tried other events, all defined in my app.js for the tab and win variables used to create the tab, and although they fire when expected, don't seem to have access to the controls.

I also tried closing that window, in a blur event defined for that same win variable, and that doesn't seem to do anything. I thought that perhaps if I closed the window, the next time the tab was clicked the code inside that window would run again.

Since my content may vary quite a bit each time the tab is clicked, deleting the window and recreating may be a good option. I'm just not sure what the best way is to achieve that.

  • I solved this problem by defining an App event listener in the window that is to be refreshed and in my app.js created an event listener for the tabgroup click and specifically the tab of the window to be refreshed. When that fires, I do an App level fireEvent and launch the entire refresh process.

    Interestingly, I subscribed to the Indie plan last night and watched several training videos, learning about app level events in one of the videos.
