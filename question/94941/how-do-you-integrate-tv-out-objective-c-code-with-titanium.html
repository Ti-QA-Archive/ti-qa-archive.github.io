How do you integrate TV Out Objective-C code with Titanium?

I found this snippet of code that allows an ios objective-c project to have a cloned view on the TV Out for the device. The problem I have is how can you integrate this back into an existing titanium mobile app. Even if I tried to add this code directly to the Objective-C code I still don't know where the Application Controller file is in all these generated files.

  • I found how to manually add this in to the built code for iphone.

    • Download the source code above.
    • Open up your built code with xcode
    • Add in the files TVOutManager.h and TVOutManager.m from the downloaded source to your xcode project
    • Edit the file TiRootViewController.m
    • Add this include after all the other includes in this file #import &quot;TVOutManager.h&quot;
    • Add the line of code [[TVOutManager sharedInstance] startTVOut]; at the bottom of the function windowFocused
    • Compile your app and there it is. (in the simulator you have to have the tv output open before you run the app)
