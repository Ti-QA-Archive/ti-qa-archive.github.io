    You must use a Titanium SDK v3.0 in order to use the new Titanium CLI. Run the following command to update:

    titanium sdk install --branch 3_0_X --default

    • That's not a CLI specific error. You must be using an old version of Titanium Studio and perhaps you manually installed the new Titanium CLI to a non-global location. Try re-installing the CLI:

      npm install -g titanium

      You may need to run that as sudo.
