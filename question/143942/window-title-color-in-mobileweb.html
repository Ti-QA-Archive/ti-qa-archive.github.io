Window title color in mobileweb

Hi, I am developing an application using titanium which uses navigation group in case of iphone and mobile web platforms. While creation I am setting a title to the window. In iphone the title color is white by default. But in mobile web it is black in color. How do I change the window title color in mobile web. I tried changing the title color by defining a label and adding it to a view and assigning as setTitleControl to the window. It is working fine but its not aligning center. Can anyone share the solution to change the title color of the window without using setTitleControl method.

  • Hi,

    which TiSDK are you using? Can you post a runnable sample code?

    I'm using setTitleControl() method in this sample and it's working fine, I can see it also aligned center (run the sample and wait a few seconds for the view to be set in the title).

    Thanks

    Titanium.UI.setBackgroundColor('#000');

var tabGroup = Titanium.UI.createTabGroup();


var win1 = Titanium.UI.createWindow({
    title : 'win1',
    backgroundColor : 'blue'
});

var tab1 = Titanium.UI.createTab({
    titleid : 'tab1',
    window : win1
});

var win2 = Titanium.UI.createWindow({
    title : 'win2',
    backgroundColor : 'orange',
});

var tab2 = Titanium.UI.createTab({
    titleid : 'tab2',
    window : win2
});

var view = Ti.UI.createView({
    backgroundColor : 'yellow',
    color : 'red',
});
var label = Ti.UI.createLabel({
    text : 'changed!'
});
view.add(label);

setTimeout(function() {
    win1.setTitleControl(view)
}, 3000)



tabGroup.addTab(tab1);
tabGroup.addTab(tab2);

tabGroup.open();
