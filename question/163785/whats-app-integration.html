  • Hi,
    WhatsApp Inc. does not provide an open API but a reverse-engineered library is made available on GitHub by the team Venomous on the GitHub. This however according to my knowledge is made possible in PHP. You can check the link here: https://github.com/venomous0x/WhatsAPI

    Also Yowsup provide best solution with example.you can download api from https://github.com/tgalal/yowsup let me know if you have any issue.

    You can also check :
    https://www.whatsapp.com/faq/iphone/23559013

    You need to create a module for titanium to get it worked with titanium.
