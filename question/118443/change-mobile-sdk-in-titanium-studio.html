Change Mobile SDK in Titanium Studio

I can't seem to find a way to change what mobile sdk i want to use while debugging. I see the option when I create a new project, but once that project is created I don't see a way to change it. I've looked through every possible setting and can only find an option for what android api to use.

Any help would appreciated.

    You can double-click to open the project's tiapp.xml file. Alternatively, right-click the file and choose "Open with / TiApp Editor" from the context menu.

    Hope this helps
