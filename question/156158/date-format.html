Date format

Hi,
I have a date format in my application as "2012-11-14T11:57:36+0000". This is part of service response. I just want to display only 2012-11-14 (YYYY-MM-DD) format. I can able to portion of it, but not getting the exact format which I want. How can I achieve this?

date1 = date.split("-");
var new_date = date1[1]+"-"+date1[2]+"-"+date1[3];

What needs to be done to display 2012-11-14 value.

    Hi Ravi,

    You can do as follows

    var date = '2012-11-14T11:57:36+0000';
// replace T with Space
date = date.replace('T',' ');
//output - "2012-11-14 11:57:36+0000"
//getting the first part
date.split(' ')[0]
//output - "2012-11-14"

    You can use the JavaScript Date function
    https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Date
