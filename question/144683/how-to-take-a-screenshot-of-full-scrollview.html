how to take a screenshot of full scrollview?

I want to take screenshot of a scrollview which is taller than the screen

the following code will take a screen shot of only the visible UI on the device

    Ti.Media.takeScreenshot(function(e){
        Ti.Media.saveToPhotoGallery(e.media);
    });

my question is how take a screen shot for the whole scrollview?

I am developing for IOS

    Hi Ahmed,

    You can do it by calling the method toImage(), it'll return a blob that you can save in your device's photo gallery.

    Ex:
    Ti.Media.saveToPhotoGallery(myScrollView.toImage());

    Hope this helps

    • Thank you brother this solved my problem but with some modification!

      if you write it like this

      Ti.Media.saveToPhotoGallery(myScrollView.toImage());

      it will not function as what I wanted since it will capture the scrollview not the view inside it

      so I just changed it to 

      Ti.Media.saveToPhotoGallery(myView.toImage());

      where myView is child element of myScrollView
