another xcode question

I just started seeing questions on testing in xcode and I'd like to do the same. I haven't seen anywhere how to setup my project in Xcode to start testing. Do I have to create a project in Xcode first than import my Titanium code or is there a script I can download that does it for me? This also may make it easier for me to test on my phone but I can't find any steps to get me started on getting my TI project into xcode. Can anyone help?

Greatly appreciated.

  • Hi Kevin,

    Titanium will create a Xcode project during the build process. Keep in mind that this project will be overridden every time you build it again with Titanium. It is okay for testing / running on the device, but you won't be able to make any changes or debug your application code.

    Start Titanium Developer, select your project -> Test & Run -> Launch for iPhone

    Afterwards change to your project directory, then build/iphone/build. Open the xcodeproject with Xcode and you should be able to run and test it on the device.

    This is your generated Xcode project.
