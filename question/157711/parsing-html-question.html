    • Well, parsing HTMl using mobile device could prove difficult since it could be fairly complex and resource intensive. What I usually recommend to people in your situation is to develop a small scree scraping piece of code that would parse the code, extract the information you need and use this extracted information inside your mobile app.

      You can develop the screen scraping part using any language you like and it also makes it more flexible for you if the website were ever to change. (meaning that you would only have to rewrite the scraping code and not have to re-publish a new version of the mobile app itself)
