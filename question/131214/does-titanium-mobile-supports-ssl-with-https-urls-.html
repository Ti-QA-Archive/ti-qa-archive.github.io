  • Mathew -

    
xhr.validatesSecureCertificate =     ;

    It is set to false in the simulator by default and set to true when deployed on device by default.

    Setting it true or false isn't a requirement, setting it to true makes the app require that the certificate is ONLY valid, or exists. I test with SSL on my home server which doesn't have a valid certificate, but I can still talk via https.

    You haven't posted any code, so I'll assume you are using the following formatted url:

    
url: 'https://username:password@yoursite.com/'
