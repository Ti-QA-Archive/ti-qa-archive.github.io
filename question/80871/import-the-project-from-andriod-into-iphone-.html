  • Yonghee

    To clarify, do you have an existing project that was developed using Titanium that you want to boot on the iphone simulator?

    If so, there is nothing you need to do to the project - the same code will launch on both the android emulator and iphone simulator. However, in order to use the apple SDK and run the iphone simulator, you need to have an Apple computer.

    Once you have the necessary hardware, you can follow the Getting Started with Appcelerator Titanium for Mac guide.
