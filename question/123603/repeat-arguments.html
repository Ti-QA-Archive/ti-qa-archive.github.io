    Sounds like you want to run a function every week, day, hour or min.
    If so you need to look at the timeout feature.

    example:

    var min = 60000;

var timeOut = function(){
   alert("ALERT!");
}
setTimeout (timeOut, min);

    If you want it to work when the app is closed you will need to make a background service.
