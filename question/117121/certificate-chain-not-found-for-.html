  • once you generate your key execute below command to find out the alias name

    keytool -keystore formconnect.keystore -list -v

    The first line of execution contains the Alias name: <value>
    If keytool is used as per the Android documentation then alias name might be "mykey"

    Use that alias name while packaging the application.
