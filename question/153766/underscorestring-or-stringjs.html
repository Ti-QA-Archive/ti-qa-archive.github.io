    I believe both are ok (I used underscore - not undrescore.string then moved to LoDash ).
    As long as they don't depend on anything else (like native modules) you can use any library from nodejs.

    If you really want to know which one is better, then you should try to make a test for both in the Ti environment, and use the one that performs faster.
