Can't run a new project created on 1.8.0.1

Hi,

I created a new app, didn't change a thing on the default app. When trying to run on iPhone it won't run.

This is what I see in terminal:

[INFO] One moment, building ...
[INFO] Titanium SDK version: 1.8.0.1 (12/22/11 13:09 fbdc96f)
[INFO] iPhone Device family: universal
[INFO] iPhone SDK version: 4.3
[INFO] iPhone simulated device: iphone
[INFO] Performing full rebuild. This will take a little bit. Hold tight...
[INFO] Skipping JS compile, running from simulator
[INFO] Performing clean build
[ERROR] Error: Traceback (most recent call last):

If I change the sdk to 1.7.5 the project runs without a problem.

What is wrong?

  • when I set debug configuration -> log level to 'trace' then I saw that the problem was that I had no iPhone directory under Resources because this was a source tree that was initially created for Android. Your problem might be different, but setting log level to trace will help explain what is going on.
