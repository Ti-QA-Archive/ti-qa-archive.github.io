Display image from tableview row to imageView in the same Window?

  • Accepted Answer

    Set the image path as a custom property of the table row:

    var prevRow = Ti.UI.createTableViewRow();
prevRow.imagePath = prevImg;

    Then setup an event listener on the table (NOT on the row):

    preView.addEventListener("click", function(e) {
    Ti.API.info("image path clicked: " + e.row.imagePath);
});

    • Ti.App.fireEvent is for firing custom events that span multiple contexts. You currently have this code for firing an event:

      Ti.App.fireEvent('changeImage',{ value:rowImg.image });

      To receive this, you would setup an event listener like so:

      Ti.App.addEventListener("changeImage", function(e) { 

});

      However I highly doubt you want to use Ti.App.fireEvent for your scenario.
