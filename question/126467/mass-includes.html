Mass Includes

Hi,
In CodeStrong lecture Your Apps are Leaking - Controlling Memory Leaks, Rick Blalock demonstrate a code for Mass Includes:

appIncluder.loadObject = function(type, name, params) {
    if(appIncluder[type] == null) { // If the controler type we want to load was not a pre-defined controller
        throw 'Includer error: type \'' + type + '\' not found!';
    } else if(appIncluder[type][name] == null) { // If the child object doesn't exist, include the file and use it
        Ti.include(type.toLowerCase() + '/' + name + '.js');
        return new appIncluder[type][name](params);
    } else { // If the name of the type object already been loaded / exists
        return new appIncluder[type][name](params);
    }
};

I tried implementing it - but it returns a null exception. to be honest - this null exception is expected because in the first 'else if' we include the file but we're not initializing appIncluder[type][name].

What we should really do is initialize appIncluder[type][name] the the function from our file. so my question is - how do I reference to a function that is in another file and that the name of t function is in a variable?

Thanks
