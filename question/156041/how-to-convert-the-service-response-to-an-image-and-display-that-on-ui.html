How to convert the service response to an image and display that on UI

Dear all,

Is there a way to convert the service response results to an image in Titanium.
My requirement is to convert the service output to image file and showing on the mobile UI.
Please provide me the hints on sample code to implement this functionality.

Regards,
Varma

  • I found the solution for this by using base64encode API.

    var picture = Titanium.Utils.base64decode(a.item(i).textContent); 

var image = Ti.UI.createImageView({
              image:picture,
              top:0,
              bottom:100,
              width:470,
              height:550
});

    Issue resolved with above code snippet.
