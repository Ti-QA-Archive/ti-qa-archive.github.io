  • Hi,jitendra kumar

    Install a Database to Internal Storage (Android) : A database, with a filename of mydb1 and located in the same directory as the the running script, is installed.

    var db1 = Ti.Database.install('mydb1', 'mydb1Installed');

    The file is opened in the following default database location, on both emulator and device.

     file:///data/data/appID/databases/mydb1Installed
