Custom Arabic fonts not working with Android 4.0

Hello Experts,

I have developed android application using custom arabic fonts, fonts are working fine with android version 2.2 and 4.1.2 but its not working with 4.0

I am using titanium 3.0

Here is my code

var CustomFont = 'ScheherazadeRegOT';

var lbl_arb = Ti.UI.createLabel({
html : context.Chapter.content[index].arabic,
color : 'black',
font : {
fontSize : 45 dp,
fontFamily : CustomFont
},
left : 5 dp,
right : 5 dp,
top : 15 dp,
bottom : 15 * dp,
height : 'auto',
textAlign : 'center',
verticalAlign : Titanium.UI.TEXT_VERTICAL_ALIGNMENT_TOP});

My font file is located at Resources/fonts/ScheherazadeRegOT.ttf

Thanks in advance

  • Hello Ali,

    Try this : 

    var CustomFont = 'Scheherazade'; //Friendly name for iOs
//var CustomFont = 'ScheherazadeRegOT'; //File name for android

var lbl_arb = Ti.UI.createLabel({
    html : context.Chapter.content[index].arabic,
    color : 'black',
    font : {
        fontSize : 45 * dp,
        fontFamily : CustomFont
    },
    left : 5 * dp,
    right : 5 * dp,
    top : 15 * dp,
    bottom : 15 * dp,
    height : 'auto',
    textAlign : 'center',
    verticalAlign : Titanium.UI.TEXT_VERTICAL_ALIGNMENT_TOP
});

    In iPhone, you also need to use the "friendly name" for the font, rather than its file name. To determine this name, double-click the TTF/OTF file to open it in Font Book. The font's friendly name is shown in the window's title bar.

    Refer this link : https://wiki.appcelerator.org/display/guides/Custom+Fonts

    Thanks
    Jigar
