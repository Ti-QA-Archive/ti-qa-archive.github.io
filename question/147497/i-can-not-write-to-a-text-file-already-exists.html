I can not write to a text file already exists

I tried to write to a text file already exists, but when I run the check to see if the write is successful, always gives me false.
I use the write method of Ti.Filesystem.File.write ().
This is the code I used:

file = Ti.Filesystem.getFile('Testo.txt');

if(file.write('Text added.', true))

{
alert('Text successfully added');
}

else

{
alert('Unable to add text');
}

    Hi

    You are trying to write to files that are in the resources folder, that will not work as that folder is read-only.

    You need to write files in the getApplicationDataDirectory or getApplicationCacheDirectory as two examples.
