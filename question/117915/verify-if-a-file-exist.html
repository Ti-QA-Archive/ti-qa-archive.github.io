verify if a file exist

Hi!

I just want to do something very easy…but there is a problem with methods i use:

I want to check if a files (by exemple 'test.txt') exist. If it does, then we do somthing, if it dont, then we do something else:

var path = Titanium.Filesystem.applicationDataDirectory;
if (Titanium.Filesystem.getFile(path,'test.txt').exists)
    {
        ///////something//////////
    }else
    {
        ///////something else//////////
    }

However it seems the getFile method create the file, so it will ever be true :s

What else can i do?

Thanks for your help.

    maybe you want to call the exists method instead of checking if it has a value?

    var path = Titanium.Filesystem.applicationDataDirectory;
if (Titanium.Filesystem.getFile(path,'test.txt').exists())
    {
        ///////something//////////
    }else
    {
        ///////something else//////////
    }
