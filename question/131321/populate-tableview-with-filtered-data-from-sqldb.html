Populate TableView with filtered data from SQLdb

I have tried a lot of things, but none of them solved my problem…

My app had a homepage with 6 buttons: pub, restaurant, nightshop, etc. All buttons go to the same page with a TableView. This TableView, at this moment, gets populated with the names of all pubs, restaurants, etc. from my table in the db, with an array. My purpose is that, for example, the button 'pubs', only gives a table with the names of pubs. In the db every record has a "type", which is the same as the categories. I want to filter my tableview 'by type'.

Can anyone help me with this issue?

Here the code how my tableView is filled now:

var sendit = Ti.Network.createHTTPClient();
sendit.open('GET', 'http://www.kevinverhoeven.be/read_dirt.php');
sendit.send();
sendit.onload = function(){
var json = JSON.parse(this.responseText);
var json = json.Locaties;

var dataArray = [];
var pos;
for(pos=0; pos < json.length; pos++){
    dataArray.push({title:'' + json[pos].BusinessNaam + ''});
    // set the array to the tableView
    tableview.setData(dataArray);
};

};

  • First: extend your api to e.g.: mypage.com/getItems?type=Pubs
    this will only load the one type. which will Lead to less Network traffic.

    second: if you use table.setData(Array) in a Loop it will replace the last set data, therefore you'll see the last Array. if you wahr addind data, then use .appendRow instead.
