Post/zip code from lat long.

Hi,

I am trying to figure out the user's postcode from their current position.

Is there any way to do this using the google api (from Titanium, not in a web view) as otherwise it is going to be a large database of postcodes and a serious distance calculation formula.

Thanks.

Jez

    Make a httpclient to this link ( user your own lat/lon )

    http://maps.google.com/maps/api/geocode/json?latlng=37.32,-122&sensor=true

    You can see it in a browser as is a json reply, containing an array. The first element is the most relevant.

    Besides this read the links Hal gave you to see the limits and usage rights.
