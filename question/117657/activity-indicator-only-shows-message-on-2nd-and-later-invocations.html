Activity Indicator only shows message on 2nd and later invocations

I see the problem in the Kitchen Sink:

  • Go to Activity Indicators,
  • Click Show Indicator (Message),

Only the spinning wheel shows: no message.

  • Click Hide,
  • Click Show Indicator (Message),

The wheel and the message now correctly show.

  • My work-around:

    
function show_activity_indicator(msg) {
    if (_ai.first_use) {
        debug('**** Activity Indicator First Use ****');
        _ai.message = msg;
        _ai.show();
        _ai.message = null;
        _ai.hide();
        _ai.first_use = false;
    }
    _ai.message = msg;
    _ai.show();    
}
