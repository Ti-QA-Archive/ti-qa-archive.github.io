  • This is of course possible. This could possibly be quite large, but you'd have to put it together to find out exactly how big it would be. I'm not sure exactly what you mean by 'format', the bible is in text format, that would seem to be the only way it would be usable. The only way to know the best way to do something would be to do it. Load it into a database, create static local web pages, pull data from the web but load it into native views .. all viable Im sure, but the best way is the way that works best for you.
