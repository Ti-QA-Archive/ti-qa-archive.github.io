    Hi

    Should be an easy tweak.

    • Select your app in the Project Explorer on the left side of the Studio.
    • Press the Run icon (green circle, white play icon and yellow folder)
    • Select Run Configurations
    • In the iOS SDK box on the right of the screen you should see a version number

    This is likely blank or set to the wrong value. Select the box and set this to the largest number you have shown - very likely 6.1.
