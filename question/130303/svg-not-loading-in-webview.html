SVG not loading in Webview

Hello everyone,

first of all i'm really new to mobile development so please be patient with me. I am trying to develop a mobile android app using Titanium Studio with SDK 1.8.0.1. My problem is that I cannot load a SVG file using webview. Every time I try to run it using the android emulator I get this as a response:
“The web page at file:////android_asset/resources/file.svg could not be loaded as: the requested file was not found. Resources/file.svg"

My SVG file is located in my Resources, as are my images, all of which work fine using webview. As I said, however, I’m new to mobile development. I was researching and it looks like the android browser does not support SVG (am I correct on this?); does that mean that an android’s app’s webview will not be able to load it either? Is this why I am running into this error? Thank you for your help in advance everyone!

Here is a small snippet of the code; apologies for not displaying it properly.

var scale = Ti.UI.create2DMatrix().scale(1);
var map=Ti.UI.createWebView({ url:'map.svg', size:{ width:500, height:300 }, scalesPageToFit:true, transform:scale});

win.add(map);
