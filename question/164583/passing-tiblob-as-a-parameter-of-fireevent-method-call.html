Passing TiBlob as a parameter of .fireEvent method call.

Hi!

Is it possible to make a Ti.App.fireEvent method call with additional data as following:

// event - is camera successful event after taking picture.

var imageBlob = event.media;

Ti.App.fireEvent("app:status", {
    blob: imageBlob
});

Appreciate any help!

  • The way I understand it, fireEvent is more of a pointer to a listener. It doesn't execute code. You can test this by placing a breakpoint on it. The break won't occur. Create a function that processes the blob, add it to a global listener, then fire the event as needed.
