Service access issue from MobileWeb application

Hi All,

I've created one Android/iOS app and is working fine. Now, I wanted to create the mobile web version for the same and I created the UI and is fine.

But, when I tried to consume a simple web service (REST-JSON), it's throwing error. In the console of the browser, I can see that the actual error is
Uncaught SecurityError: Blocked a frame with origin "https://api.appcelerator.net" from accessing a frame with origin "http://127.0.0.1:8020". The frame requesting access has a protocol of "https", the frame being accessed has a protocol of "http". Protocols must match.
Failed to load resource: the server responded with a status of 405 (Method Not Allowed)
XMLHttpRequest cannot load SERVICE LINK. Invalid HTTP status code 405

What could be the reason for this ? Thanks in advance!!

  • Seems like this is Cross Origin Resource Sharing (CORS) related issue.

    If your application uses Ti.Network.HttpClient to communicate with different sites, then either those sites mustenable CORS (Cross-Origin Resource Sharing) or you must set up a proxy on your web server with CORS enabled.
