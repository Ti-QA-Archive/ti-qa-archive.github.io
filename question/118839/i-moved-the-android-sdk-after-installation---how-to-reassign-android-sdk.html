I moved the android SDK after installation - How to reassign Android SDK

MAC OSX 10.6 Snow Lep.

I had to assign the location of the android SDK while I created my first mobile app.

I created the sim link and everything was fine.

I deleted the android sdk and reinstalled it under my Developer folder so all my SDKs would be in one location.

How can I reassign the location of the Android SDK in Titanium.

Titanium cannot find the SDK.

It is not complaining about the adb file. It cannot find the SDK because I moved it.

Thanks for the help.

    I had to download AppDelete to find all the files that were associated with Titanium and delete them.

    One file I found was interesting and it was a .db file.
    I opened it with textmate and found that this is where the SDK is stored and where the errors come from.

    NOTE *: Running AppDelete in the trial mode did nothing for me. I had to delete them manually.

    I deleted all the files associated with titanium and reinstalled and it worked fine.

    You have to find all files. You cant just delete the /Library.. and ~/Library… titanium application support folders. There are more files that have to be deleted in order to get a fresh install again.

    Hope this helps someone out.
