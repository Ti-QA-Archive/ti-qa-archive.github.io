Android Appearance stuck in 2.2?

Hello, I'm new to Titanium. I was checking out the tutorials and loaded the Kitchen Sink example application. However, when I run it on the Android Emulator and on my phone (both which are setup for android 4.1.2) all of the controls render as if they were android 2.2 controls. They don't match the appearance of the latest android components (e.g. sliders, toggles, buttons, etc.).

I thought I had read that Titanium rendered using local controls in the OS, so that it would match the user's expected experience? What am I doing wrong?

Thanks,
Pete

  • You need to set targetSdkVersion="14" which will instruct the Android tools to enable the new UI on capable devices. You probably have just minSdkVersion="8" which pegs it to 2.2. See Android docs. Also check out this excellent post on Android themes
