Swap default content view with a new view dynamically

I have a section of index.xml that looks like this

   <View id="content">
... stuff here ...
</View>

The stuff in there is default content for the view, Now I have a couple different Buttons on the page. How do I swap out the content onclick to the other view chosen by the user's button click? It's kind of like a hand-rolled tab group basically, but I am not using the default TabGroup for a few different reasons.

Should I draw both views and hide and show on click, or is there a way to render a view to a variable in index.js and swap it on the fly with the default hard coded content in the index.xml file?

  • There are many ways to do what you're seeking. For example:

    $.button.addEventListener('click', function() {
   $.index.remove($.content);
   $.index.add(Alloy.createController('yourOtherView').getView());
});

    Or, if you'd prefer:

    $.button.addEventListener('click', function() {
   $.content.removeAllChildren();
   $.content.add(Alloy.createController('newChildView').getView());
});
