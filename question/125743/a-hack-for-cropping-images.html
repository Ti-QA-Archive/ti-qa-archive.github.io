A hack for cropping images

This is not really a question but instead a hack I stumbled upon. I've been wanting to crop imageviews before by placing them inside another view, but there hasn't been an easy solution to this.

I discovered by accident that a imageview inside a view with a border radius will be cropped. You can set the border radius to something small like 0.1 so it won't be visible.
