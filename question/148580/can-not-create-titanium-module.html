  • Accepted Answer

    The Titanium CLI does not support creating modules despite the option being visible. It's not finished. It's hardly started. We probably should have commented it out.

    In the meantime, inside the Titanium SDK's folder is a file called titanium.py that can create a module project for you. Otherwise you can use Titanium Studio to create it.
