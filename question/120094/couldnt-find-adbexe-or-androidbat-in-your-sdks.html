  • Thanks Mark.
    Together with a tech friend we have found a work around. I am on Windows XP and sym links doesn't work on this platform. (I think symlinks work on windows 7 pc and Vista: http://en.wikipedia.org/wiki/Symbolic_link)

    The solution for XP:
    copy the 2 files: "adb.exe" and "AdbWinApi.dll" from the \android-sdk\platform-tools directory and paste them into the \android-sdk\tools directory.

    Make sure you copy them, not move them, the files need to be in both locations.

    Problem solved.
    Lee
