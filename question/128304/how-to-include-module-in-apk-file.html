How to include module in .apk file

I am using java module inside Titanium Android app. When I build app and install .apk into emulator, app is not finding the module file. Please tell me how to deploy\install Ti-Android app along with modules. Thanks

  • You need a "modules" directory below your project directory (at the same level as Resources). Inside of "modules", you need an "android" and an "iphone" directory (if you have modules for both platforms). In your case, just the "android" directory. Inside of that is where your module's subdirectory lives.

    Then, in tiapp.xml, you need to add a "modules" section at the very bottom, that looks something like this (final "</ti:app>" included for orientation purposes):

    <modules>
  <module platform="iphone" version="1.2.3">funkymodule</module>
  <module platform="android" version="6.2">com.othercompany.funk</module>
</modules>
</ti:app>

    Post which module you are using for more specific help.
