  • The location of this option within studio 3.1.0 is preferences > Studio > Platforms > iOS. When Studio 3.1.1 comes out, it will be in run configurations.

    From the command-line, you can use –retina for retina phone or iPad simulator, and –retina –tall for iPhone 5 simulator.

    Example:

    "ti build -p ios -T simulator –retina –tall -Y iphone -F universal -S 6.0"
    This will give you an iOS build run in simulator, with a retina, tall iPhone simulated, running a "universal" app, with your simulated iphone running a simulated OS of v6.0.
