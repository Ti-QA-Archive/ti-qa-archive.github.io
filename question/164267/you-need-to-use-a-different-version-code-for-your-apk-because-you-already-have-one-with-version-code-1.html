    Hi James,

    the versionCode is the internal version code for the playstore, you can assign it through the android manifest tag in tiapp.xml

    this is how to set the versionCode to 2

    ...
<android xmlns:android="http://schemas.android.com/apk/res/android">
        <manifest android:versionCode="2" android:versionName="1.1">

        </manifest>
</android>
....
