changing date format from JSON?

Hi Guys.
IM grabbing a date form a Mysql Server via JSON and then i want to format the date to UK format.

At the moment the JSON result is YYYY-MM-DD and i want it DD-MM-YYYY
I tried

var accd1 = user[c].accd; // GETS YYYY-MM-DD
var accd3 = new Date(accd1); //create a new date but from JSON date?
var day = accd3.getDate();
var month = accd3.getMonth();
month++
var year = accd3.getFullYear();
var accd = day+\\\'-\\\'+month+\\\'-\\\'+year; // SHOULD GIVE DD-MM-YYYY

I just get NaN-NaN-NaN

Any help?

  • You could just to it with string manipulation: splitting up the string by the -, then putting it back together.

    var date_bits = user[c].accd.split(\\'-\\');

    var accd = date_bits[2]+\\'-\\'+date_bits[1]+\\'-\\'+date_bits[0];

    hope this helps. (ignore the backslashes in the code. there seems to be a bug in the Q&A software)
