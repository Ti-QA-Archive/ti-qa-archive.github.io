  • Hi

    As you have not detailed your SDK version or provided information on your target platform - I will need to make some educated guesses.

    • Your use of the word emulator and your issue with Titanium connecting to it - your target platform is most likely Android.
    • The date you become a member implies you are using at least TiSDK version 3.0.2 and given this date you are unlikely to have already generated a Google Maps Key using version Google Maps v1.

    You should be aware that Google has deprecated Google Maps version 1 and no longer issues keys to new requests, so you must use Google Maps version 2, which requires a different kind of key and you MUST use the Google Maps for Android module to achieve your goal.

    I would also suggest testing on a physical device by using a USB cable, the emulator is universally reviled, and the issues you have experienced are no restricted to Titanium - google it.

    More information about the Google Maps for Android Module can be found;

    • Docs - Map Module
    • Guide - Google Maps v2 for Android
    • Google Maps Android API v2 - pre-installed from TiSDK 3.1 or higher (you still need to select it though)

    You MUST be using TiSDK 3.0.2 or higher to use Google Maps for Andoird Module which you require if you have not already obtained a version 1 API key.
