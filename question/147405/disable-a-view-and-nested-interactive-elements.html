Disable a view and nested interactive elements.

Hi,
Is it possible to disable any interactive elements in a view ? including the view ?
I have to handle this behavior to prevent user to interact with the view when animation is in progress.

Thanks

    You can overlay a transparent view over the view you are trying to disable. This will cover all the items in the view. You can even make the overlay a little dark to show the user that the section is disabled.
