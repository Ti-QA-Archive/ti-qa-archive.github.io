    • Don't worry, Amarjeet, we were all noobs once :)

      To answer your question more specifically, you have to log into TiDev at least once but thereafter it should remember your email and password and not ask you for them.

      The only thing I can think of is that your Titanium home directory (in your user profile) is corrupted in some way. Try shutting down TiDev and deleting this Titanium directory and running TiDev again.

      Note that you can find your user profile by running %HOMEPATH% from the run dialog. There should be a Titanium-related directory inside of it.

      Hope this helps
