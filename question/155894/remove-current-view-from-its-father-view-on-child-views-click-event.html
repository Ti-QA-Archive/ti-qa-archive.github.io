remove current view from it's father view on child view's click event

Hi all,

is there any way to remove current view from it's father view on child view's click event?. For example:

var view = Ti.UI.createView();

var childView = Ti.UI.createView();
childView.addEventListener('click',function(){
    view.remove(childView);
});
view.add(childView);

please help

    Hi

    Short answer - Yes.

    Your code example is exactly how to remove a child view from the parent view. Your code would work perfectly.

    You can make the code slightly better an more re-usable by tweaking as follows;

    var removeChild = function (e) {
    view.remove(e.source);
};
childView.addEventListener('click', removeChild);
onotherchildView.addEventListener('click', removeChild);

    This uses a single function to handle the child view removal and it internally uses the source parameter to know which one to remove.
