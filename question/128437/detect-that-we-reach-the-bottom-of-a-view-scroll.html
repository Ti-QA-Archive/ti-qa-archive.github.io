Detect that we reach the bottom of a view? (scroll)

Hi,

Is there a way to detect that the user has reach the bottom of a view?
I would like to fire an event (load more entries) when the bottom has been reach, like the Google Reader is doing.

Let say we have a TableView with many TableViewRows, scrollable.
Is there a way to detect we have scrolled all the way to the bottom?

The event "scrollEnd" fires "when the table view stops scrolling", which can be in the middle of the page if we stop scrolling in the middle, isn't it?

  • Sorry, here is a better listing:

        queryView.addEventListener('scrollEnd', function(e) {
        if ((!updating) && (e.contentOffset.y + e.size.height + 100 > e.contentSize.height)) {
            beginQueryUpdate();
        }
    });

    scroll_view.last_element=UI.createView({height:'6'});

    scroll_view.addEventListener('scroll', scroll_to_bottom_event );

    function scroll_to_bottom_event(e) {
