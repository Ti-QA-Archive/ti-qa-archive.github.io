  • Accepted Answer

    You could try something like this:

    var dashboard = Ti.UI.createWindow({ url : 'windows/dashboard.js', title : "dashboard", your_var:"your_value" });

    In your dashboard.js:

    //your window

var win = Titanium.UI.currentWindow;
alert(win.your_var);

  • Hello Chatura,

    1.you can pass this variable as Patrick suggest you. that is most common method.

    2.if you need to use this variable in many windows then just set

    // set Global Variable
Ti.App.variableName = value ;

//Use
Ti.App.variableName;

    so need to just declare this first time and you can use this into whole application.

    3.you can also use this by setting property

    //set Property
Ti.App.Properties.setObject('variable',value);

//get property
Ti.App.Properties.getObject('variable');

    so this way you can store your full array,dictionary or what ever you want.

    just declare it once and use in full application.

    so just use any of the above method as per your requirement…
