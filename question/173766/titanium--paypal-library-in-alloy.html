    A few questions;

    • Which PayPal library
    • Where did you download it from?
    • What version of the library are you using?
    • Have you confirmed the version you have is the latest you can get?
    • Have looked inside the modules folder to see if both iOS and android are mentioned?

    Alloy is an MVC framework, this process down to Titanium to compile. I mention this to point out that a Alloy is not different from Titanium, Alloy creates Titanium code. So any module requiring you to use titanium code to get it to function will work. The exception is badly written or out of date modules but they would fail for Alloy or non-Alloy projects.
