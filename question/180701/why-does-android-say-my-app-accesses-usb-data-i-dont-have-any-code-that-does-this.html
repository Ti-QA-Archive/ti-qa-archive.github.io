Why does android say my app accesses USB data? I don't have any code that does this!

Hey,

my app - wibblerapp - https://play.google.com/store/apps/details?id=com.wibblerapp.wibbler - is on the android app store.

When you look at the permissions it requires, it lists

Photos/Media/Files
   modify or delete the contents of your USB storage
   read the contents of your USB storage

Along with network permissions.

I'm not aware of doing anything that requires USB storage though. Is there some way to "disable" this permission? I don't have it listed in my ti.app.

I have the modules

analytics.google
ti.cloud
facebook
ti.admob

But I don't think they need USB access either. Is there a list of things that will cause this? Or can i explicitly deny this access and see what breaks to find the offending code?
