    Hi Nadeem, what you understand in little different in reality. Titanium does a mapping for JS files and Native Code for that using Java Script Engine of the platform.
    Titanium SDK contains module and map java Script coding using Kroll Bridge.

    but still there are 2 ways to use native coding in titanium.
    JS –> Java Script
    1: create Titanium Module for what you want in titanium and then use it using JS.
    2: you can do native changes after build form titanium also but you have to be expert in native side then.

    for iOS you can refer by blog where i show case this.

    http://ashishnigamblog.blogspot.in/2013/01/extending-titanium-mobile-application.html
