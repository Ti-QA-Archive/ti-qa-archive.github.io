Getting Dynamic Height

Is there a way to get the height of dynamic view? It is populated with views from a remote request

var entryView = Ti.UI.createView({
                    height: 'auto',
                    layout: 'vertical',
                    left: (g*33+"%"),
                    top: prevHeight,
                    width: "33%",
                    id: g
                });
                var entryHeadline = Ti.UI.createLabel({
                    height: 'auto',
                    left: 5,
                    text: _data[_ident][s][e].headline,
                    top: 5,
                    width: 'auto'
                });
                var entryTeaser = Ti.UI.createLabel({
                    height: 'auto',
                    left: 5,
                    text: _data[_ident][s][e].teaser,
                    top: 5,
                    width: 'auto'
                });
                var entryImg = Ti.UI.createImageView({
                    height: 'auto',
                    left: 5,
                    image: _data[_ident][s][e].leftImage,
                    top: 5,
                    width: 'auto'
                });
                entryView.add(entryHeadline);
                entryView.add(entryImg);
                entryView.add(entryTeaser);

                wrapperView.add(entryView);

                prevHeight = entryView.getHeight();
                Ti.API.info(entryView.size.height);

I am trying to build a multi-column layout. But my getHeight() always returns the same value despite how long my labels are.

    try checking the height of entryView in an open event of its containing window. I believe auto dimensions are calculated til rendered.

    windowThatContainsEntryView.addEventListener('open', function(e) {
    Ti.API.info(entryView.size.height);
});

    • So now I have a solution for calculating the position of elements in columns.

      newsWindow.addEventListener('open',function(e){
    var children = newsList.children[1].getChildren();
    var sum = 0;
    for(var c in children){
        if(children[c].top != 10){
            children[c].top = sum
        }else{
            children[c].top = 10;
            sum = 10;
        }
        sum = Number(sum)+Number(children[c].height)+Number(10);
    }
});

      top=10 marks the first element in column and the way to get var children may differ.
