ActionBar support with Alloy

Does Alloy support Titanium.Android.ActionBar?

<Alloy>
    <Window id="index" title="Window">
    <ActionBar id="abar" title='ActionBar'>
    </ActionBar>
    </Window>
</Alloy>

The above code gave an error saying it could not find createActionBar.

It does work if only the Window tags are in index.xml and the action bar is done via javascript.
