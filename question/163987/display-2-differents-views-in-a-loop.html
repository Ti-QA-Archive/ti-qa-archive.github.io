    Hi

    You can simplify your life by using scrollableView, this provides a controller that displays any number of views side by side.

    You can allow the user to slide between views themselves or you can turn off that option and controll the change yourself.

    You have only two views, so you simply add those to views to the scrollableView and use the built in methods to swap page between them.

    An example from the documentation in alloy;

    <Alloy>
    <Window id="win">
        <ScrollableView id="scrollableView" showPagingControl="true">
            <View id="view1" backgroundColor="#123">
               <Label>View 1</Label>
            </View>
            <View id="view2" backgroundColor="#246">
             <Label>View 2</Label>
             </View>
            <View id="view3" backgroundColor="#48b">
             <Label>View 3</Label>
             </View>
        </ScrollableView>
    </Window>
 </Alloy>

    On android (only) you can use the following methods to navigate;

    $.scrollableView.moveNext();
// and
$.scrollableView.movePrevious();

    I am guessing you can figure out what those do.

    On both Android and iOS you can jump to any page;

    $.scrollableView.setCurrentPage(0); // first page
$.scrollableView.setCurrentPage(1); // second page

// you can also

$.scrollableView.scrollToView(0); // first page

    One scrolls (animates) to the page - the other jumps (swaps) to the page.
