Alloy barColor with iOS7

I'm trying to set the barColor, and I can see my changes regardless of how I implement (xml property, tss, js set function), however since iOS7, my solid colour - say black for example - is no longer a solid black - it's become a dark grey. Even 'red' becomes a pinky-washed-out colour.

Any ideas? I'm still relatively new to Titanium and am now forced to learn it pretty much over again with iOS7.

Thanks in advance.

Dave

SDK: 3.1.3.v20130908095038,
Platform: iOS 7,

  • Accepted Answer

    barColor doesn't work in iOS 7. You need to use tintColor. And 'translucent' is always true for iOS 7.

    The grey background is showing because you've probably got a translucent black over the app's white background. Change the app's backgroundColor to black, and you should be fine.

    See The Migration guide: http://docs.appcelerator.com/titanium/latest/#!/guide/iOS_7_Migration_Guide-section-37533766_iOS7MigrationGuide-Toolbar

      translucent:        false

      statusBarStyle:        1
