Cannot set IOS version to 4.2 with Titanium SDK 1.7.5

Can anyone give inputs on how to set IOS version in Titanium to 4.2? Currently have simulator 4.3 setup in Titanium, Titanium SDK is 1.7.5, X-Code version is 4.1 which provides options for other IOS versions including 4.2.
Is it possible to have multiple versions of IOS SDK and Simulators in Titanium. Please provide inputs/pointers.

  • hi Rahul,

    there is no way to set multiple versions of the simulators but you can do that otherway…

    right click on your main app folder in titanium studio and select run-> run configuration in that you find iOS SDK …select 4.2

    then

    open your simulator, go to Hardware menu and set version to 4.2

    good luck
