Change Ti.UI.SearchBar returnKeyType (iOS)

Hi guys,

The title pretty much says it all. I want to change the returnKeyType for Ti.UI.SearchBar so that it says "OK" instead of "Search" in the lower right corner. Is this possible?

Thanks in advance.

  • Hi

    The short answer is no, the keyboard button have very little scope for change. Choosing your own caption for the return button is not possible.

    Whilst you can tweak the keyboard in text fields using several constants returnKeyType this has a limited set of captions, this is not possible for the searchBar and neither can have a free text value.
