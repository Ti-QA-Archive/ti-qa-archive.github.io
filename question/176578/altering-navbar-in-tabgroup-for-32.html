    Hi

    As you will note from the barColor documentation - this is an iOS only feature. Given your app is for android this would be an a problem.

    I will assume the "NavBar" you are trying to change is actually the "ActionBar", if this is the case then you need to be using Android themes to make these changes.

    • Alloy Action Jackson: An Android Action Bar Demo App for Appcelerator Titanium

    You should also be aware that significant ActionBar changes have been made in Ti SDK 3.3 that you should review. It might make sense to use Ti SDK 3.3 to have those changes already in place.

    • Android changes related to ActionBar in Ti SDK 3.3
