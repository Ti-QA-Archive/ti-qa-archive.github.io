  • It depends on what are you trying to accomplish.

    If you want to see if two variables point to same object (that they reference same object), then you can use strict comparison operator:

    obj1 === obj2;

    If you want to see if those two objects are of same type, you can use "toString" method:

    obj1.toString() === '[object Object]';

    Titanium object should produce similar string in new 1.8.0.1 SDK version, that looks something like this "[object TiUIComponentName]", so if you want to check for different UI components, you need to replace part of that string to corresponding Titanium UI component name.

    If you want to see if two object have same content, but are to different objects, the simplest way is to use JSON.stringify:

    //obj1 !== obj2 is true
JSON.stringify(obj1) == JSON.stringify(obj2);

    Although it won't work in any case, it most cases it will produce right result.
