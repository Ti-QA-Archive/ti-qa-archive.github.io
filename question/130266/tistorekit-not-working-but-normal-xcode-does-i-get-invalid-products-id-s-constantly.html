  • damn. here is the solution:

    evt.invalid comes back as an array.

    instead of this: 

    else if ( evt.invalid) {
            alert('ERROR: We requested an invalid product!');
        }

    need this: 

    else if ( evt.invalid[0]) {
            alert('ERROR: We requested an invalid product!');
        }
