long title getting cut in custom row for tableview

Working Enviroment- Titanium studio-3.1.0, ios- 6.0.

I have created custom rows for table view as:

var RegData = [
{ title:"Spain", hasChild:true },
{  title:"United Kingdom", hasChild:true },
{ title:"United States of america", hasChild:true },
{  title:"France", hasChild:true }
];

var TheTable = Titanium.UI.createTableView({
data:RegData
});

I had set the specific width & height for each row. When I inserted long string on title then it's getting cut.
How to avoid that problem. I want the full title without getting cut. Please suggest solution..

  • Hi Pradip,

    Try below Sample. Hope it helps You.

    var win = Titanium.UI.createWindow({
    backgroundColor : "pink"
});

var tbl_data = [];
for (var i = 0; i < 10; i++) {
    var row = Ti.UI.createTableViewRow({
        height : Ti.UI.SIZE,
        hasChild : true
    });
    var label = Ti.UI.createLabel({
        left : 10,
        text : 'XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX',
        height : Ti.UI.SIZE
    });
    row.add(label);
    tbl_data.push(row);
}

// now assign that array to the table's data property to add those objects as rows
var table = Titanium.UI.createTableView({
    data : tbl_data
});
win.add(table);

win.open();
