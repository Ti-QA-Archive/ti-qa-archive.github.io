Hello to all guys. I wanted to know if it is possible to incorporate within an iOS app the ability to use a webview pointing to a local html file to the app. My aim would be to create a graph like this : http://www.highcharts.com/demo/line-basic , serving me a html file that calls inside the class javascript to create the chart.

    Hi

    You can use web pages and anything associated with web pages in your app, just add a webView and point the url to the local file. Remember that the files will be static if coming from the device so you will need to work out how you populate the data in the chart.

    Titanium does not make or suggest you not use HTML - just use it when it is needed. Think of it like a web page, we all moved away (hopefully) from using table for layout but that does not mean tables should be avoided - just used for tabular data - what they were meant for.

    Have fun.
