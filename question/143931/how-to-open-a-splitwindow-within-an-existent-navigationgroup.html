How to open a SplitWindow within an existent NavigationGroup?

I would like to open a SplitWindow from a window that already has its NavigationGroup, so the SplitWindow would have the back button in its master view labeled with the previous window title.
But all examples I find the SplitWindow creates its own NavigationGroup, any help is appreciated!

  • I found a solution: create a window and put the splitwindow inside it. Then open this new window from its parents navigation group. So you will have a split window inside a window that has a master navigation group with the back button labeled with the parent's window title and a split window below it. You can even have another two navigation groups for each window inside the splitwindow.
