Create variable name from value?

I would like to create a variable name from a input value.

For example if I had param['kitchensink'], I would like to create a variable called pp.api.data.kitchensink, so I could do something like pp.api.data.kitchensink = results.

I know in other languages I can do something like pp.api.data.{param['kitchensink']} = results. How would I do that in Titanium JS?

Since I have been unable to figure out how to do the above I have dropped back to pp.api.data.apiResults['keyName'] = results; however Android is complaining about that method while the iPhone is happy.

  • You can eval() any string that is a valid javascript command.

    eval('pp.api.data.kitchensink = results');
