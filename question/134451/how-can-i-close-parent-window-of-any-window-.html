How Can I close parent window of any Window ?

Hello,
I want to know that How Can I close parent window ?

I have Login window and Registration button inside the window. Now I am showing Registration window as modal when button is pressed.

Now when Registration get successful then I am closing the Registration window by 

var win = Titanium.UI.currentWindow;
// code
win.close({modal:true});

But previous Login screen is still visible. I want to close Login screen too. How to close it ? I don't want to hide it…

    win.addEventListener('close', function(e){
parentwin.close();

    Do something like this.

    
Ti.App.Properties.setBool('LoginFlag', true);
win.close();

    on focus of the parent window do like this.

    parentWindow.addEventListener('focus', function(e) {
    if(Ti.App.Properties.getBool('LoginFlag'))
    {
        parentWindow.close();
    }
});
