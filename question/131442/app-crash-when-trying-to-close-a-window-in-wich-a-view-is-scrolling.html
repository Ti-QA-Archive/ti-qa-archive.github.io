  • Try something like that :

    When the scroll starts, lock the allowed orientation modes to the current orientation :

    Titanium.UI.Window.setOrientationModes(_YOURCURRENTORIENTATION);

    At the end of the scroll (there is a scrollEnd event on tableViews, you should try it on scrollViews), reset the allowed orientation modes to the original value :

    Titanium.UI.Window.setOrientationModes(_ALLTHEORIENTATIONMODESYOUWANT);

    Maybe it'll trigger the orientationChange at that time only ?
