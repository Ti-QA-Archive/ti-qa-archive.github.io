Pass textfield object to a function

Suppose i have a textfield called txtT.
I want to check weder it is blank or not.
So i created a function as below:

var isReq=false;
function RFV(txt)
{
Ti.API.info(txt.value);
if(txt.value=='')
{
isReq=false;
}
else
{
isReq=true;
}

return isReq;
}

and passed the textbox txtT to it as:
RFV(txtT);
but when i checked the value of txt in function i got the value as undefined.
So m doing something wrong or the txtT.value is undefined when it is left blank?
