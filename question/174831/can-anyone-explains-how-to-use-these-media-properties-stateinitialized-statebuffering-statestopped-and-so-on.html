  • Hello,

    When your sound changes, the change event fires

    Do the following to check the state values of the sound on Android:

    /* Android only */
var sound = Ti.Media.createSound({ ... });
sound.addEventListener("change", function(e) {
    switch(e.state) {
        case Ti.Media.Sound.STATE_INITIALIZED: /* Do something */; break;
        case Ti.Media.Sound.STATE_BUFFERING: /* Do something */; break;
        case Ti.Media.Sound.STATE_STATE_STOPPED: /* Do something */; break;
        // etc..
    }
});

    For iOS checkout: Titanium.Media

    Hope this helps,
    Patrick van Vuuren
