iOS Console doesn't log in Titanium Studio [workaround]

For sometime now, I had a problem with the iOS logging in studio with iOS 5.0 simulator and Ti SDK 1.7.5

I found a little workaround:

Open a terminal and:

  • cd &#x2F;Library&#x2F;Application Support&#x2F;iPhone Simulator&#x2F;
  • find the directory of the simulator you're using (for me: 5.0)
  • cd Application
  • find the directory of your application (it's a hashkey so… it isn't very explicit)
  • In this directory: cd Documents
  • Once you're in document you should have a *.log file (we'll call it logs.log)
  • type tail -F logs.log
  • let the terminal opened as you run your simulator and watch your log file changes live.

Hope this could help.
