  • Accepted Answer

    USSD codes are forbidden on IOS and Android by default.
    On IOS there is not much chance as Apples API's to work around the issue are private.

    On Android you can write a Titanium Module which takes a string as an argument and appends the encoded hash sign.

    We developed such a module and it works quite well with all kinds of USSD codes (Android only).
    I you'er interested we can publish it on github although it's quite trivial.
