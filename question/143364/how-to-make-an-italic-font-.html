How to make an italic font ?

Hi

  • on iOS

How can make a italic font?

My code: (doesn't works)

var albumLabel = Ti.UI.createLabel({
        font:{fontFamily:'Georgia', fontWeight:'italic'}, 
        top: 12,
        left: 15,
        text: 'Select an Album'
    });

Thanks

  • Accepted Answer

    set the font family to the correct font name

    var albumLabel = Ti.UI.createLabel({
        font:{fontFamily:'Georgia-Italic'}, 
        top: 12,
        left: 15,
        text: 'Select an Album'
    });

    you have to know the right name for each typography and the type has to support it, so you may find fonts without italic style.
