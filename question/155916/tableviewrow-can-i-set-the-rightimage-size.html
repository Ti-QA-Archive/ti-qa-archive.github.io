[TableViewRow] can i set the RightImage size?

Hi,
I want to know if there is a method to set the size of the RightImage of a TableViewRow.

This is my TableViewRow:

var r = Titanium.UI.createTableViewRow({
               backgroundColor:'#eee',
               title: name,
               textAlign:'left',
               rightImage: 'http://www.mysite.com/images/myimage.png',
               font:{
                 fontSize:24,
                 fontWeight:'bold'
               },
               height: 45,
               width:'auto',
               color:'#333',
            });

I have tried to pass an ImageView but seems to not work!
The rightImage parameter, accept only Urls and Local Paths.

Anyone know a solution?
Thank you

  • Create a view to add to your tableViewRow then place your other UI objects in there (including an imageView of the size you require). Something like…

    
var row = Ti.UI.createTableViewRow();
var container = Ti.UI.createView({height:Ti.UI.SIZE});
var img = Ti.UI.createImageView({
    width: <your size>,
    height:<your size>,
    right:0 
});
container.add(img);
row.add(container);

  • you can use a custom row

    var a = [];

var table = Ti.UI.createTableView();

var custom_row = Ti.UI.createTableViewRow();

var right_image = Ti.UI.createView({
   width : your_width,
   height : your_height,
   backgroundImage : path_your_image
});

custom_row.add(right_image);
a.push(custom_row);

table.setData(a);
