Alloy - open View issues

Hello,,. I have tried many ways to try to get this working with no avail. I am certainly frustrated with the alloy method as I do not understand it. I am writing a login page, at index, then once logged in you get your username and key. I want to forward that through out the app to talk to a server for information. My next page is called landing in which I resend the data to get more information.

My current code is:

if (response.logged == true)
{
var arg = {
uid:response.name,
paired:response.paired
};
//alert("Username: " + response.name + "Paired: " + response.pair);

    var window = Alloy.createController('landing', arg); 
    window.open(window.getView());
    $.index.close();

}

I have also tried:
var window = Alloy.createController("landing");
window.getView.open()

var window = Alloy.createController("landing").getView();
window.open()

and a couple similar variations I found along the way. None have worked obviously. I did test the code the the alert and it worked but did not open the page.

var window = Alloy.createController("landing").getView();
window.open();

    This should work. I am assuming your view file looks like below,

    <Alloy>
    <Window>
        <Label title="New window"/>
    </Window>
</Alloy>
