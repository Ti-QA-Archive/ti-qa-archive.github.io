How can an app detect if the device (iPad) s serving up a Personal Hotspot?

In my iPad app, I'd like to be able to figure out if the device is serving up a Personal Hotspot.

Currently, if "Ti.Network.online" returns true, then "Ti.Network.networkTypeName" returns "MOBILE" if connected via Cellular Data, and "WiFi" if connected via WiFi.

"Ti.Platform.address" reports a valid address only in the WiFi case, however.

If Cellular Data and "Personal Hotspot" are enabled, networkTypeName returns "MOBILE" and "address" returns undefined…

Any advice or hints as to how the app could (1) detect if "Personal Hotspot" is enabled and if so, (2) figure out its WiFi address?

I'm guessing that it would be possible to do this at the Objective C level (via a module)… but is there an Titanium-level solution here?

Thanks in advance for any help…!
