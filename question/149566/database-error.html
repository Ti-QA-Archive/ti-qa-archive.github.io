  • Hi antonio,

    every time you have to execute some query to your database the operations to do are simple : 

    var db = Ti.Database.open("dbname");
var resultSet = db.execute('your query');
db.close()

    if you you have to do multiple query is convenient that you use a transaction : 

    var db = Ti.Database.open("dbname");
db.execute("BEGIN");             // begin the transaction
for(var i=0 ; i < j; i++) {
    db.execute(" your query ");
}
db.execute("COMMIT");
db.close();

    is important that every time you close the database connection , show your code and log error for more support .
