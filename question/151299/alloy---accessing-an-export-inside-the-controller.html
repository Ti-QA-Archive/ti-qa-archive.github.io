Alloy - Accessing an Export inside the controller

I have a function that's an export so it can be called from outside the controller, but I also need to access it from a click event in the same controller, how do I achieve this?

controllers/item.js

exports.showItem = function(itemid){
    ....
};

$.nextItem.addEventListener('click',function(e){
   $.item.showItem($.nextItem.nextItem); 
});

nextItem is a property on the nextItem button that tells the code what the next item id number is

    Nevermind, it was simply to remove the controller name from the $ syntax:

    $.showItem

    instead of

    $.item.showItem
