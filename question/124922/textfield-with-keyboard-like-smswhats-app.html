    I have just pushed an Update, version 2.0

    Please take a look at the documentation in the documentation folder and the sample app.js (which I have cleaned up)

    Make sure you include the module in tiapp.xml

    <modules>
    <module version="2.0">ti.pedro</module>
</modules>

    and in your JS file

    Titanium.Pedro = requre('ti.pedro');

    Also, very important, the smsview.bundle (found in the sample folder) MUST be placed in the Resources directory.

    Open an issue in Github if you run into technical problems.

    Regards, Pedro
