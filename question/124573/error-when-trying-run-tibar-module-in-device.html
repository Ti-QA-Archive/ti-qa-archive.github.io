    • @zozo zozo
      you have to add this line into the "module.xcconfig" file

      OTHER_LDFLAGS=$(inherited) -framework AVFoundation -framework CoreMedia -framework CoreVideo -framework QuartzCore /usr/lib/libiconv.dylib

      Source: Tibar Scan Module – clang: error: linker command failed with exit code 1

      — commented July 17th 2012 by Markus Eschenbach

    • This is the correct answer, it worked for me.
