Check if image still exists

So i have an image for the user's profile photo in my app.

The photo can be taken from their media - no problem., i open up the media and then save to my db the path to the image using the nativePath member.

The problem is, what if the user deletes the image later on? Now my db will have the path to the deleted image, and presumably titanium will crash if it tries to show the image.
I don't want to make a backup copy of the image… that doesn't seem like a good use of the user's available space, and i already have a default image anyway.

What i'd like to do is, when i get the image path from the db, check if the image path points to an image… how do i do this?
