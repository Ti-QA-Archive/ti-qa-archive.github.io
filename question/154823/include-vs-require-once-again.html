  • A simple answer here is if your doing commonJS then you use require …. it isn't really a discussion to be had. You can use include but that is not how commonJS works.

    you basically create a function around what you need to do, export that function and then require the module where you need it.

    making sure you use return to pass back the completed object.

    Hope that makes sense ..
