    Hi Husain,

    You trying to access current window using Ti.UI.currentWindow so may be you trying it before opening root window..

    so just make some change in your code.. e.g add scrollview into your window which you want… for example you want add it in winIndex window then replace this line win.add(scrollView); by $.winIndex.add(scrollView);

    And also remove var win = Ti.UI.currentWindow;.
