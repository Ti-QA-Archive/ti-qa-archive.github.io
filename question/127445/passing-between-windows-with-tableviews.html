Passing between windows with tableviews

I have found lots of questions and answers relating to passing values from a 'parent' window to a 'child' window, but can't see much about passing values the other way.

My scenario is a parent window containing a TableView. I use a label on one of my rows to display some data. When I select this row, a new 'child' window opens. This child window also contains a TableView, and selecting a row opens a picker, where the user can select a value.

When the child is closed and the user goes back to the parent window, I want to display the chosen value from the child's picker on the parent's TableView.

Any help or examples would be much appreciated!

  • If the child window you open is with a new context then you should use global events. you should know that it's not best practice - better way is to use "light weight" windows.

    You should read this: http://developer.appcelerator.com/blog/2010/08/execution-contexts.html
