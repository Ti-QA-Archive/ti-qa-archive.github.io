    Hi,

    1> You can create modules using native java and use them in Titanium

    2> To run code specific to android you have 2 options:

    a. Create a project for android only

    b. In code use 

    var isAndroid = Ti.Platform.osname === 'android';
if(isAndroid){}

    to write android specific code if you are coding for multiple platforms.
