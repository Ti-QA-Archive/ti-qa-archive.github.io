Reading Android Preferences

Hello,

Sorry for the newbie question, but although i've figured out how to open up the Android Preferences window and customize the settings and values, i can't seem to figure out how read those values for use in the app..

i was hoping that the values would be accessible through the Titanium.App.Properties object, but there didn't seem to be anything there. Anyone know of a way to access those values?

  • They should be in the Ti.App.Properties object

    With the following preferences.XML 

    <?xml version="1.0" encoding="utf-8"?>
<PreferenceScreen xmlns:android="http://schemas.android.com/apk/res/android"
    android:title="Settings">
    <PreferenceCategory android:title="Update">
        <CheckBoxPreference
            android:title="Foo"
            android:defaultValue="true"            
            android:summary="Bar"
            android:key="FooBarKey" />        
    </PreferenceCategory>
</PreferenceScreen>

    Now you can check the variable with

    alert(Titanium.App.Properties.getBool('FooBarKey'));
