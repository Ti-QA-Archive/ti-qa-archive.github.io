Center the views in scrollable view Android

Hi,
I am using a scrollable view for an image gallery. It is working perfectly in iOS. The images are of dynamic size and scrollable view auto adjust them to center. But in android it doesn't. :(
Can we align center the views in a scrollable view in Android ?? If yes then how ?? Any help would be great.

  • I would recommend using a view for a wrapper. 

    var imageViewWrapper = Ti.UI.createView({
    width:  Ti.UI.FILL,
    height:  Ti.UI.FILL
});

    Then add your image view to the wrapper.

    Then add the wrapper to the scroll view. It's a little more involved but it works on iOS and Android.
