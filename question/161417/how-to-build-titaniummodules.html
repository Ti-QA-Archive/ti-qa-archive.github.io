    If you have the sources:

    iOS:
    Go in folder and edit the titanium.xcconfig file in the folder. You have to change teh path to SDK. If it is ready you start build.py. Its all. In build folder is a zip.
    Android:
    similar, you start ant.

    It is all verbosely described in documentation: http://docs.appcelerator.com/titanium/latest/#!/guide/Android_Module_Development_Guide

    Rainer
