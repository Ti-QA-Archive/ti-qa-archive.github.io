  • I got the same problem and resolve it.
    The solution is
    1)Un install the titanium studio

    2)Go to user/documents and remove titanium workspace (This will delete your all project so it better to keep copy on hard disk)

    3)Go to macintosh HD/Library/Application support and delete the titanium directory also check in the user/Library/Application support and delete the titanium directory

    4)Restart system

    5)install titanium studio.

    Hope it will resolve this issue , i got this resolved after 3 days of damn trying and doing things.
