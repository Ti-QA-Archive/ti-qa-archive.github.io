  • The Map Module does not say you can only place two annotations at a given coordinate - you can place any number. However, only the last one added will be seen as they will all be on top of each other. Unless you have some other way of showing them, the user will never get to see them - not a good UX.

    If you keep an array of added annotations then you can remove them from the map then delete (null) them and recreate new ones.
