  • Hi Hardik,

    There is no any inbuilt controller for create such kind of alert. so you must need to create with view..

    you can create one view in your xml and set background image which are you want and assign right 0 to this view. and hide and show this view according to your requirement.

    e.g

    window.xml

    <Alloy>
    <Window class="container">
        ...
        ...
        <View id="popupView">
            <Button id="btnAppartment"></Button>>
            <Button id="btnSkill"></Button>>
        </View>
    </Window>
</Alloy>

    window.tss

    ...

"#popupView":{
    backgroundImage : "POP UP BACKGROUND IMAGE",
    top : "DEPEND ON NAVBAR HIGHT, top = NAVBAR HEIGHT",
    right : 0
}

...

    and depends on your requirement hide and show popup view
