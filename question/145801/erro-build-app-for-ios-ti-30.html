    • @Rainer now you need to select 3.0.0.GA as your active Titanium SDK. To verify you have 3.0.0.GA installed, run:

      titanium sdk

      If you see 3.0.0.GA listed and it isn't the active SDK, run this:

      titanium sdk select 3.0.0.GA

      If 3.0.0.GA isn't installed, run:

      titanium sdk install --default --force

      Next be sure your <sdk-version> to 3.0.0.GA in your project's tiapp.xml.

      — commented December 16th 2012 by Chris Barber
