Button inside tableviewrow prevents row from highlighting when pressed

Hi,

Scenario: button inside a row.
Problem: row does not highlight when pressing

If I take the button out of the row, the row change color when pressing.

Any toughts? Is this a bug?

Code below.

var win = Ti.UI.createWindow({
    backgroundColor : 'white'
});

var tableData = [];

var tableView = Titanium.UI.createTableView();

for (var i = 0; i < 5; i++) {

    var row = Ti.UI.createTableViewRow({
        height : 50,
        title: "helloooo"
    });

    var button = Titanium.UI.createButton({
        height : 30,
        width : 50,
        title : 'Click'
    });

    button.addEventListener('click', function(e) {
        alert(1);
    });

    row.add(button);
    tableData.push(row);
};

tableView.setData(tableData);

win.add(tableView);

win.open();

    This is not a bug, row shows highlighting when you click on row. But when you click on button that wont show bcz you are not click the row instead you are clicking the button and that highlighting is related to the row.
    What you can do is you can use the backgroundSelectedColor property to set the color you want on button row(button) click that will solve your problem.
