  • If you can't select which Android SDK to uses using the Preferences Windows, then it is not configured correctly. The best way to be absolutely sure at the point you are at, is to install the Android SDK manually.

    From your Command Prompt, simply launch the android command. (it located in the [ANDROID_SDK_DIR]\tools directory. Then select the necessary SDKs you must install (2.3.3 and any of the latest ones like 4.2 or even 4.3) But 2.3.3 has to be installed for compatibility issues.

    Once this is done (you may need to reboot your windows machine), simply restart Titanium Studio and see if you can select the default Android SDK version.

    Hope this helps
