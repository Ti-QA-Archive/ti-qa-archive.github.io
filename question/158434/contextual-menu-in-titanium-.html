    • Ah, do you want a contextual ActionBar? (Contextual Action Bars)

      This is currently not supported in Titanium as you can read here

      A common context menu can be achieved like this (e.g. on a tableView):

      var table = Ti.UI.createTableView({
    data: tableData
});

table.addEventListener('longpress', function(e){
    var opts = {
        options: [
            'Like ' + e.rowData.title,
            'Share ' + e.rowData.title, 
            'Delete ' + e.rowData.title,
            'Cancel'
        ],
    };
    var contextMenu = Ti.UI.createOptionDialog(opts);
    contextMenu.addEventListener('click', function(e){
        Ti.API.info(e.index + '  selected');
    });
    contextMenu.show();
});
