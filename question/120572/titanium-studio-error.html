  • I just fixed mine this way:

    (BTW, don't have a clue if it's reliable or not)

    1. I got into my workspace folder and removed everything but the folder containing my actual project to work on.
    2. Reopened Titanium Studio.

    Enough so I get no more error message regarding java heap space.

    • Titanium Studio (1.0.2.201107130739)
    • iOS SDK 4.3 (8H7)
    • Xcode 4.1 (48110)
