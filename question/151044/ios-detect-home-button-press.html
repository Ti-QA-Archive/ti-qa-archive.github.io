iOS Detect Home Button Press

In iOS is there a way to detect if the Home Button has been pressed? I know this doesn't turn off the app, so I would like to perform an action on the app when this happens that takes user to a specific window.

  • Accepted Answer

    You cannot change the behaviour of the app on iOs when the home button is pressed.
    All you can do is to catch the Ti.App.pause event and do some fast non UI stuff - like save something in db, etc

    http://docs.appcelerator.com/titanium/3.0/#!/api/Titanium.App-event-pause
