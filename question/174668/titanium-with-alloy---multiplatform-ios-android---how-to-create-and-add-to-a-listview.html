    Hi Benoli,

    You can learn listViw documentation for alloy,here is the doc link. And you can test this code for you expected behavior. I think it will be little bit helpful for you.

    ###index.xml

    <Alloy>
    <Window layout="vertical" title="ListView Test" navBarHidden="false">
        <TextField id="text" width="Ti.UI.FILL" height="Ti.UI.SIZE" top="10" hintText="Enter your Hobby"></TextField>
        <Button id="save" title="Add" width="Ti.UI.FILL" height="Ti.UI.SIZE" top="10" onClick="addsomething"></Button>
        <ListView id="list" top="10">
            <ListSection id="section" headerTitle="Hobby's">
                <ListItem title="Hobby 1"></ListItem>
                <ListItem title="Hobby 2"></ListItem>
            </ListSection>
        </ListView>
    </Window>
</Alloy>

    ###index.sj

    function addsomething() {

    var textValue = $.text.getValue();
    var hobbyData = [
    {properties: { title:textValue}},
            ];

    var item = $.section.appendItems(hobbyData);
    $.list.setSections(item);

    $.text.blur();

}

$.index.open();

    If you have any concern about this query let me know.
