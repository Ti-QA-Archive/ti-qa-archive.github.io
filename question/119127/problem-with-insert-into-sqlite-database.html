Problem with INSERT into SQLite Database

Hi everyone, i hope you can help me with this.

I'm trying to do an insert in my database SQLite rescued by JSON data like this:

                for( pos=0; pos < json.length; pos++){

                    theData = db.execute('INSERT INTO information (cat_id, stock, descuento, fecha) VALUES ("'+json[pos].item_no+'","'+json[pos].stock+'", "'+json[pos].descuento+'", "'+fecha+'")');
                    theData;

                };

And result:

2011-04-29 11:33:59.471 Catalogo[12822:5d03] [ERROR] Error occurred calling next on a PLSqliteResultSet. SQLite error: 'constraint failed' for 'INSERT INTO information (cat_id, stock, descuento, fecha) VALUES ("A002-70000 ","77", "25.33", "2011-04-29")'
[ERROR] The application has crashed with an unhandled exception. Stack trace:

Anyone know why this happens?.

  • You're using the wrong syntax, you have to do it like this (also stated in kitchensink)

    db.execute("INSERT INTO tblName (info, info2) VALUES (?, ?)", var, var);

    Just add a question mark for every field you need to add and the put them after the query like i did
