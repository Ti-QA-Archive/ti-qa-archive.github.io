Check date with 30 day period

Hello!

I can check if a date is smaller or bigger than todays date but how can i check if date is within 30 days from today or more?

For the bigger smaller check

var datum = '2013-09-28';

var date = new Date();
var d  = date.getDate();
var day = (d < 10) ? '0' + d : d;
var m = date.getMonth() + 1;
var month = (m < 10) ? '0' + m : m;
var yy = date.getYear();
var year = (yy < 1000) ? yy + 1900 : yy;
var diff = year + "-" + month + "-" + day; 
Ti.API.info(diff);

if (datum <= diff){
    Ti.API.info('smaller');
} else {
    Ti.API.info('bigger');
}

The bigger part is kind of useless really. The thing i want to see is if it´s more than 30 days ago or not when i change var datum.

    Hello Richard Harrysson,

    Please check this one for checking date.

    

var date1 = new Date(2013, 09, 30);
// set your date

var time = date1.getTime();

Ti.API.info('time' + time);

d2 = new Date();
d2.setDate(d2.getDate() + 30);
// add or subsruct you number of day

if (time > d2.getTime()) {
    Ti.API.info('time' + time + 'd2 ' + d2.getTime() + 'bigger');
} else {
    Ti.API.info('time' + time + 'd2 ' + d2.getTime() + 'smaller');
}
