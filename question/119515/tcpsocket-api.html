TCPSocket API

TCPSocket API of Titanium to connect to our server works for iPhone but errors out for Android. Any help in resolving this would be appreciated?

  • The current mobile SDK for Android does not support TCPSocket, as evidenced by the fact the android icon is not present in the upper right of the module description for TCPSocket. Only an Apple icon is present. (Currently only HTTP network support is present.)

    However, this blog description shows that new socket support is among the things being added in SDK 1.7.0, so you may want to try the latest continuous build and the new modules. I'd suggest looking at the other blog entries from May too because the new socket support is based on some of the other new stuff such as streams.
