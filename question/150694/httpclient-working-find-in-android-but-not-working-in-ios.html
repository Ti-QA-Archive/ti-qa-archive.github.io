    You problem is this I think:

    datos.getElementById(i)

    try to replace it with:

    datos.getElementById(i+''); // '' is empty string, so 2x single quote

    Actually the message says pretty clear that expected: NSString, was: __NSCFNumber in -[TiDOMDocumentProxy getElementById:] - getElementById expects a string and you are giving it a number - the value of i

    @Daniel - I think this should be managed inside the Ti native code, to avoid this kind of headache :) (if I'm right)
