I use WebView as camera overlay. In hmtl/js code of that WebView I use some js libraries that can handle mouse inputs (finger touch). E.g. I click somewhere on that webview and I can handle some things in it's javascript code based on click location. After I added gesture recognizer to that WebView those things stopped working.

webView.addEventListener('dblclick', function() {
    menuDialog.show();
});

After this, my double click gesture is recognized and my menu pops out. But I loose things in WebView's javascript which I need.

Possible solution is to try to recognize double click in JS of webView and then fire an event that I could listen in my app's JS and then show menu. But I'd like it better if I could use both things.
