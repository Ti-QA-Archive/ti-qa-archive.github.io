    Hello Calebe,

    Thanks for your inquiry, I have faced the same problem, I have fixed it following this link. "It may be that you don't have an avd.py script at the location it's looking." The problem is when you update your Ti SDK online, sometimes it doesn't upgrade properly which cause the Android SDK configuration issue. The solution is, you should first download the latest Ti SDK for your platform from this link. Then manually put the SDK to the Ti SDK directory. Now restart the studio.

    Hope this helps. If it does, please mark as best answer. Thanks.
