  • Are you wanting the picker to respond to a click event as opposed to a change event? You currently have change event listeners for the pickers in your code, but as the name implies, those events only fire when the picker selection is changed.

    If you wanted to get the current value of the picker when it is 'clicked', you can try something like this:

    picker.addEventListener('click', function(e){
                Ti.API.info(JSON.stringify(picker.getSelectedRow(0)));
            });
