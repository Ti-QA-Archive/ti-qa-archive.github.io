Label - Number of Lines

Hey Guys,

I was wondering if anyone knows how to calculate the number of lines a string will take up in a label with a specific font? Or if anyone knows a solution to solve this problem.

The issue is that I have a view and I need to make the text fit within the view, wrapping is fine but there are also other labels in there too.

  • First, make you label size to fit the content eg

    var myLabel = Ti.UI.createLabel({width:Ti.UI.SIZE, height:Ti.UI.SIZE, text:'This is some text that will be \n on Two lines....... \n Blah \no Blah....'});

    Next, If you want to ensure you next label doesnt overlap, give your containing view a layout of either 'horizonal' or 'vertical' (depending onwhich direction you want to go. In the 'vertical' layout, the top property of the next label will be relative to the bottom of the first label. eg

    
var myView = Ti.UI.createView({layout:'vertical'});

var myLabel = Ti.UI.createLabel({top:0, width:Ti.UI.SIZE, height:Ti.UI.SIZE, text:'This is some text that will be \n on Two lines....... \n Blah \no Blah....'});

var myLabel2 = Ti.UI.createLabel({top:5, width:Ti.UI.SIZE, height:Ti.UI.SIZE, text:'This label starts 5 pixels after the bottom of the  first label'});


myView.add(myLabel);
myView.add(myLabel2);
