    • Is not that bad, if you organize yourself a little and taking in account it that Titanium speeds up the development of an app with 50% or even more

      1. Make modules for everything is platform agnostic(backend stuff usually) so you will be able to reuse the code across the platforms.
      2. For desktop use a framework build with webkit in mind to keep it as light as possible (zepto.js is one of them) but any other one will do it.
      3. If your app is not heavy you can try to use a webview on the mobile side too. But I STRONGLY don't recommend it cause you'll get into performance issue very fast, especially on android.
