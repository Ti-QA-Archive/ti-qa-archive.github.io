  • I think i have figured it out. After you compile your project, you can check your resource folder, a js file ti.cloud.ProjectName.js is created, in the file content, you will find:

    exports.services_TestFunction = function(data, cb) {
    InvokeService("/TestFunction", "GET", data, cb);
};

    So i only need to call ProjectName.services_TestFunction({data: 'test'},someCallbackFunc)
    From the definition of InvokeService, we can see we are actually using XMLHttpRequest.
