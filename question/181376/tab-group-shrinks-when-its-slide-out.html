    Hi Sonal Jain,

    Just change in animation of tab group add width.

    e.g

    ...
tabGroup.animate({
        left : left,
        right : right,
        width : "100%",  //ADD THIS PROPERTY
        duration : 400,
        curve : toggled ? Ti.UI.ANIMATION_CURVE_EASE_OUT : Ti.UI.ANIMATION_CURVE_EASE_IN
    }, function(o) {

        alert("toggled");
        toggled = !toggled;
        tabGroup.applyProperties({
            left : left,

        });
    });
....

    • Great answer above by Nilesh. Just to clarify what's going on, when you reference absolute positioning such as left=0,right=0 the width of the view is determined by the difference between those two values.

      When you set left=200 it's actually changing the width because the width has not been set. Setting the width value ensures that you get the expected UI.
