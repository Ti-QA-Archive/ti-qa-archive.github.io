Building app deletes local html file

OS: Windows 7
Titanium API: 3.5.1
Mobile App

I make the following webview in my app:

       var win = Ti.UI.createWindow();
        var webview = Ti.UI.createWebView({
            url: 'local.html'
        });
        win.add(webview);
        win.open();

and according to the Titanium WebView guide my local.html file should be placed in the Resources directory. However when I build the app the file gets deleted and the application tells me it cannot find the html file.

  • Accepted Answer

    Another way is to put your html file inside lib directory and access it. If lib directory is not present create it at the controller directory level. Below is the hierarchy strucutre,

    ProjectDirectory

    |
|-- app
    |
    |-- assets
    |
    |-- controller
    |
    |-- lib
    |
    |-- models
    |
    |-- styles
    |
    |-- views

    Hope this will help you.

    Narendra Raorane.
