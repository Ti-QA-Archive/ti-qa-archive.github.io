Why is the resources folder hidden in an alloy project?

I created a new alloy project and the resources folder is hidden. Is there a specific reason why it's hidden? I thought I might have to make some changes and setup controllers from there. Any info would be good.

    The resources folder is hidden because all of the code within it is generated based on the alloy code. When you click "build", the Alloy engine actually compiles your code into best practice CommonJS prior to building the project - therefore if you were to add a controller under the Resources folder, or make changes it would get erased next time you click the build button and replace by the generated code.

    That being said, if you want to "unhide" the Resources folder its pretty easy. From the App Explorer view in Titanium Studio, click the chevron icon (looks like an upside down triangle and the hover text says 'View Menu') and then click Customize View. This will pop a dialog box used to filter the folder structure and file types within the App Explorer view. Find the Resources folder (towards the bottom) and uncheck the checkbox.
