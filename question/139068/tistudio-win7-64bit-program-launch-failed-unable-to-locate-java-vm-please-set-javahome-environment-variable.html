    Take a look at https://wiki.appcelerator.org/display/guides/Installation+Troubleshooting#InstallationTroubleshooting-InstallDiagnosticsforWindows and post the results of that script here.

    If you can't run java or javac from the command line, put the /bin directory that holds those executables from the JRE into your path.
