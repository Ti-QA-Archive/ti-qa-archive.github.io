  • Are you running this in your app.js code? It looks like either the currentWindow context doesn't exist when you are trying to add the button to it, or there is some window cross-mojonation happening when you are trying to add the button.

    If you are trying to add a dialog to tab windows, try targeting the windows explicitly. ie: win1.add(buton); win3.add(buton); etc.
