Window "reactive" event trigger?

I have an application with 2 tabs, each tab associated with a window. I need to do some initialization every time the window is "activated" from background.

I know there's "resume" and "pause", but these event listeners don't apply on my scenario.

Please advise.

    You can listen for the focus and blur events on any window. When that window comes into focus, you can run your init or re-init code. ie:

    win.addEventListener('focus', function(e) {
    // window init stuff here ...
});
