transparent background doesn't work in buttons

Hi,
I have several buttons that are placed within a tableView.

The button consists of an imageView and a labelView. I want the background image of the tableView to show through to the buttons, thus I need the buttons backgroundColor to be transparent. I have not been able to get the background of the buttons to become transparent, instead I get a white background.

The width of the buttonView is 97 while the icon inside the buttonView is only 57 so on either side there currently is whitespace, I expected that space to be transparent. Same goes for the white space around the label.

How to I get the backgroundColor to be transparent? What am I doing wrong?

    var btnView = Ti.UI.createButton({                
        left:ySpacer,
        height: cellHeight,
        width: cellWidth,        
        top: 10,
        mappTitle:this.id,
        backgroundColor:'transparent',
        borderColor:'transparent',
        borderWidth:0,
        backgroundSelectedColor:'#3d3d3d',
        backgroundFocusedColor:'#3d3d3d',
        opacity:1.0,
        focusable:true

    });
    //add an icon    
    var image = Ti.UI.createImageView({                        
          image: this.getRegIcon(),                          
          top:0,                          
          height: 57,
          width:57,
          title:this.id,
          backgroundColor:'transparent',
          backgroundSelectedColor:'#3d3d3d',
        backgroundFocusedColor:'#3d3d3d'

    });
    //create the lable for the current icon object.    
    var testl = this.getLocalLabel();            
    var thisLabel = Ti.UI.createLabel({
        top:58,
        color:'#512907',
        font:{fontSize:12,fontWeight:'bold'},
        text: testl,
        textAlign:'center',
        backgroundColor:'blue',
        touchEnabled:false
    });    
    btnView.add(image);
    btnView.add(thisLabel);

I am currently testing on the iPhone simulator (iOS 5.0), my Titanium SDK version is 1.8.1

This code is contained in a function in a commonJS module.
Thanks

  • I didn't set the style property for the button. So it was defaulting to BORDERED style which was causing the white background to occur as defined in Titanium.UI.iPhone.SystemButtonStyle. Setting the style property to 0 made my background to become transparent.
