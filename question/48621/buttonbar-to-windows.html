    Inside your click event handler, check the e.index property. First button's index is 0 and so on.

    For example:

    bb1.addEventListener('click', function(e) {
    if (e.index === 0) {
        // open window 1
    } else if (e.index === 1) {
        // open window 2
    } else ...
}
