Restoring missing studio features

Hi,

Since the last update to Titanium Studio, I've noticed a couple of things stop working.

  1. jsLint no longer validates any files that aren't in a project. I don't do all my coding in a project, sometimes I just want to edit files. I've turned the jslint validator on for build and reconcile, but no joy.

  2. PHP scripts no longer show an outline for any files not in a project. Again, same as above.

Is there a way to put this back? I really hate it when software updates take away things that were working perfectly before.

This has happened on both my Windows and Mac machines.
