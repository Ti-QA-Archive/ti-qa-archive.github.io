Hide component and remove from layout possible?

So, lets say we have a button in a vertical layout. If you do hide() and show() on the button it simply makes it visible or not. But it still takes up space in the vertical layout if you hide() it. Is it possible somehow to make it not take up layout space when you hide it(). Same as just removing it from the layout… the problem with removing it for us, is that its not that easy to put it back into the layout when we need to show it…

  • You can play with the height of the button. When you hide it, set the height of the button to 0. When you show it again, set it back to it's original value.

    The next example hides a button between 2 other buttons, and shows it again after 2 seconds:

    <Alloy>
    <Window id="win" layout="vertical">
        <Button title="Test 1"></Button>
        <Button id="btn2" title="Test 2"></Button>
        <Button title="Test 3"></Button>
    </Window>
 </Alloy>

    //hide button without taking place
$.btn2.hide();
$.btn2.setHeight(0);

//wait 2 seconds
setTimeout(function() {
    //show button again with the size it needs
    $.btn2.setHeight(Ti.UI.SIZE);
    $.btn2.show();
}, 2000);

//show current window
$.win.open();
