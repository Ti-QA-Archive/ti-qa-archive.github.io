Updates to be done to move from Titanium SDK 3.2.3 to 3.5.0

Hi All,

I have been away from Titanium for the last 4 months. But I am back to developing Titanium apps and I see that there is new Titanium SDK 3.5.0 available for installation.

The installations in my Mac machine are -

Titanium SDK version is - 3.2.3

ios SDK version is - 7.1

Xcode version - 5.1.1

Mac OSX - 10.9.2

I already have an iphone app in the AppStore and now I need to update with some changes.

So my question is what upgrades do I need to do so that my existing iphone app built using
Titanium SDK 3.2.3 app supports ios8 as I am very much aware of the 1st Feb 2015 deadline for the upgrade.

Kindly advise.

Thanks.

  • Hello Bably Das,

    At first you need to update Mac OSX then Xcode because Xcode6 do not run lower version. And ios8 is compatible with Xcode6 or more

    Then update titanium related latested version
    Ti SDK version: 3.5.0GA,

    CLI Version: 3.4.1,

    Alloy Version: 1.5.1,

    Titanium Studio Version: 3.4.1

    And when you will upload your project to the iTunes add this to the tiapp.xml file for supporting minimum iOS version

    <ios>
        <min-ios-ver>7.1.2</min-ios-ver>
    </ios>

    If you have any concern, then let me know.

    Thanks.
