Indefinite animation in Android

Hi,

with SDK 3.2.0 it seems that the repeat parameter with -1 is ignored for indefinite animation:

view.animate({
    transform: Ti.UI.create2DMatrix({ rotate: 360 }),
    duration: 3000,
    repeat: -1
});

This does work with SDK 3.1.3. Now the animation only works one time.

Does anyone knows a workaround now? Using event listener 'complete' and restart doesn't look good as there is a visible lag between end of animation and restart.
