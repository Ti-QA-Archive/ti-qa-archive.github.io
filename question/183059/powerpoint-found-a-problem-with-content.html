powerpoint found a problem with content

I'm actually using office automation with PowerPoint to manipulate PowerPoint presentations. A PPTX was uploaded from the internet and the process failed. I opened the PPTX directly with PowerPoint and got the error "PowerPoint found a problem with content" and asked if I wanted to repair. I said yes, saved it as another PPTX, and everything was ok. However, I can't do this for every PPTX that's uploaded and need to find a way to deal with it when it's using office automation to extract the contents of the PPTX.

I've unchecked all the protected view settings in the trust center, set my computer to authentication and impersonation to connect and identify, enabled macros and activeX controls, added the upload path to trusted settings, right-clicked the PPTX, properties, but there is no "unblock" to unblock. Nothing has fixed the issue.

What else can I check or is this just a bad PPTX since it's the only one we've found like this?

  • Hi
    Although this does solve your problem, it is unsafe to disable the Protected View setting.
    It was specifically added to enhance security of opening Office documents from untrusted sources.
    If you go to file properties of the file, you may see a button called "Unlock". Unlocking the file should also suffice.
    Use PowerPoint Repair Toolbox. The PowerPoint repair tool reads and analyzes the damaged files and exports data the restored data to Microsoft PowerPoint.
    https://www.repairtoolbox.com/powerpointrepair.html
