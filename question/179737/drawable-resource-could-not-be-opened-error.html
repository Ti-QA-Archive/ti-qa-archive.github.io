  • It sounds like you're not providing the version for the display class of your Nexus 7 (xhdpi, I believe). Do you have and xhdpi folder and a default / fallback version provided? Those would go in the app/assets/images/android folder or if using the platform folders, it would go in the res-drawable folder (no qualifiers) or res-drawable-mdpi.

    More info using density-specific images and the Android dev guides
