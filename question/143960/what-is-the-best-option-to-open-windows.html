  • Hi Stiaan,

    We don't really have any way of reproducing your issue without more code, or a test case. Also, are you targeting Android or iOS, or both?

    It also looks like you're not releasing memory by passing new objects to the window every time. Try putting these in global variables and call win.remove(yourObject) and yourObject = nil when you're done.

    Here are some best practice examples which you might find useful:

    You could also try using a tab group and set the 'tabBarHidden' property of the root window to true, or use a navigation group (iOS only).
