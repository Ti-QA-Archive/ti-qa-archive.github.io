    Hi,

    I think you have not set any property for scrollableView that's whiy it overlaps so try to set top for scrollableView.

    permalink
    2 Comments

    • .xml

      <ScrollableView class="scrollableview" id="customerCreateFormWrapper" showPagingControl="true">

      and

      .tss

      ".scrollableview":{
    top : 35
},
"ScrollView": {
    top: 0,
    backgroundImage: 'none',
    backgroundColor: '#E2E5EA'
}
