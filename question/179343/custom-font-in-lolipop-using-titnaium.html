  • Hi Nikunj Munjiyasara,

    Thanks for your concern. To use custom fonts in your android L (lollipop), first you need to copy the font file to your project's Resources/fonts/ directory and use it to your code like below.

    var customFont = 'SpicyRice-Regular';
var label1 = Titanium.UI.createLabel({
   color:'#000',
   text:'I am Window 1',
   font:{
      fontSize:40,
      fontFamily: customFont
   },
   textAlign:'center'
});

    For better understanding check my test case screen shot.

    Hope this will help you to solve your problem. If you have any concern then feel free to ask.
