Help with Callbacks and Parameters

I am trying to better understand callbacks and find that the following code works and is simple to understand. My struggle is how can I add parameters to this? For example if I want to call the callback function with long/lat?

function callback() {
    alert("I am in the callback!");
}

function work(func) {
    alert("I am calling the callback!");
    func(); 
}

work(callback);

    I don't understand what are you trying to do … but beside "apply" and "call" methods, you can call callback function as any other:

    function callback(par1, par2) {
    alert("I am in the callback!");
}

function work(func) {
    alert("I am calling the callback!");
    var long = ...;
    var lat = ....;
    func(long, lat); 
}

work(callback);
