How to center a view in a Window using height="auto" (Android) - In iOS works fine

Hi i tried to center a View in a Window. 

var win = Ti.UI.createWindow({
        title : 'Welcome',
        backgroundColor : '#fff',
        layout:'horizonal' /*On iOS works fine in absoulte and horizonal but not in Android*/
        });

var parantView = Ti.UI.createView({
            backgroundColor : '#eee', /*grey*/
            layout : 'vertical',
            height : 'auto'
        });

var button = Titanium.UI.createButton({
            title : 'Search',
            top : 10,
            width : '90%',
            height : '60dp'
        });
var slider = Titanium.UI.createSlider({
            top : 20,
            min : 0,
            max : 1000,
            width : '90%',
            height : '30dp',
            value : 10
        });
parantView.add(slider);
parantView.add(button);
win.add(parantView);

On iOS the grey parantView is in the center of the window, on Android the complete window is gray, so i guess the height="auto" is not working proberly.

  • the problem is that in IOS everything is attached to the center of the view if not position is specified but on android everything is attached to the top left corner. So to get the center you might have to tell what is the top and left position

  • I faced exactly the same issue with auto height. Setting a fix value to height looks like the only way to solve it. Hope this will be fixed in next Titanium SDK…

    var parantView = Ti.UI.createView({
    backgroundColor : '#eee', /*grey*/
    layout : 'vertical',
    height : '120dp'//, 'auto'
});
