How do I change the background image when in Landscape mode?

Hi,
Can someone tell me how I can change the background image when the user is in landscape mode?

I have a file called Default-Portrait.png

I created one called Default-LandscapeLeft.png but it doest work.

Thanks

    In Titanium.Gesture there's an event-listener called orientationchange that you can use to set the background of your window. For example:

    Ti.Gesture.addEventListener('orientationchange', function(e) {
  if (e.orientation == Titanium.UI.LANDSCAPE_RIGHT || e.orientation == Titanium.UI.LANDSCAPE_LEFT) {
    yourwindow.backgroundImage = "Default-Landscape.png";
  } else {
    yourwindow.backgroundImage = "Default.png";
  }
});
