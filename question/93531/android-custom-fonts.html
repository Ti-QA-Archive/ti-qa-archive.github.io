  • I managed to do what you want to do.
    i have this im my html file

     <style type="text/css" media="screen, print">
    @font-face {
      font-family: "AkzidenzGroteskBE-BoldCn";
      src: url("../../fonts/AkzidenzGroteskBE-BoldCn.otf");
    }

    body { font-family: ""AkzidenzGroteskBE-BoldCn"", sans-serif }
  </style>

    You can see that font path is relative to my html file location!
    Now one thing you have to careful with is that i dont use the font family name in "font-family" but the font FILENAME.
    Hope this helps
