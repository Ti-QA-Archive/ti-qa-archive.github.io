ScrollView with Locking on Items

I'd like to be able to make a scrollview that locks or clicks on the middle or selected item. Similar to the scrolling tabs in kitchen sink, however, having the tabs always center on whichever tab is closest to the center.

An example of this is http://www.choyster.net/wp-content/uploads/2011/03/chow02.png

Where it says Movie & TV Stars, the photo above it is in a scrollview and can be swiped left and right and 'locks' on whichever is closest and centers itself under the triangle (and then fires an event to the top photo to change).

A scrollable view won't work, because only the centered view is 'swipe-able' not the entire view. If this is possible with a scrollable view though, I'm open to suggestions.

The SDK Version isn't really relevant but I'm using 1.8 and I don't have any code since I don't know how to do it. The scrolling tabs example is what I've been going off (https://github.com/appcelerator/titanium_mobile/blob/master/demos/KitchenSink/Resources/examples/scroll_views_tabs.js)

    You could likely do this with a Ti.UI.Scrollview. You could set the scrolling to horizontal, fill it with fixed width views that represent your "tabs", then use the scroll event of the ScrollView to apply your algorithm for determining which tab is closest to the center.

    • This helped a lot. Thanks! Here is the code that I used (my items are 64 wide)

      scrollView.addEventListener('scroll',function(e){ 
    if (scrollView.timeout) { clearTimeout(scrollView.timeout); }   scrollView.timeout = setTimeout(function() { 
        var val = e.x / 64;
        var offsetting = Math.round(val) * 64;
        scrollView.scrollTo(offsetting,0);
    },100); 

});
