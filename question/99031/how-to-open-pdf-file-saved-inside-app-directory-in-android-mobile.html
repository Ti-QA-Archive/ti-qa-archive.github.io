How to open PDF file saved inside App Directory in Android Mobile?

Hi, I have been having this problem and even after checking most of the answers given here, still can't seem to fix this problem.

My app need to download a PDF file from server, and open it inside the android mobile phone. I am not able to open the downloaded pdf file, but using the same method is able to open an online pdf file as it triggers the internet browser.

I know there is no native support for PDF file in android, but I have installed PDF Viewer and ThinkFree Office in Android, anyone has any ideas how to integrate with other application to open? e.g. something like "open with" options?

The codes as follow:

var documentDir = Titanium.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory, 'downloadDir');
var fileName = 'test.pdf';
var webServerPath = 'http://localhost/getFile/';

function saveFile(){
    var retrieveFileUrl = webServerPath;
    var retrieveReq = Ti.Network.createHTTPClient();
    var file = Titanium.Filesystem.getFile(documentDir.nativePath,fileName);
    retrieveReq.onload = function(e){
        file.write(this.responseData);
    }
    retrieveReq.open("GET",retrieveFileUrl);
    retrieveReq.send();
}

function viewFile(){
    var file = Titanium.Filesystem.getFile(documentDir.nativePath,fileName);
    Titanium.Platform.openURL(file.nativePath);
}
