UI.TableView "Editable" customisation.

Hello,

I currently have a UI.TableView with entries and the Editable flag set to true, this enables the swipe-to-delete function which I am fairly happy with.

Is there any way to customise the label on the "Delete" button or add extra buttons into this built in editable "swipe"?

  • var tableView = Titanium.UI.createTableView({
    data: someData,
    editable: true,
    deleteButtonTitle: 'Delete'
});
