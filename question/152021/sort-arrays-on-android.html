  • Hi

    If this is the whole code, then there is no way it worked on any platform.

    • The array is a simple numeric array, yet the sort function expects an object array with a distance property at least.
    • The function you are going to call should ALWAYS be defined before its use. sortafterDistance in this case.

    Your code would need to look more like this;

    function sortafterDistance(a, b) {
    return a.distance - b.distance;
}
var ar = [
    { distance: 1 },
    { distance: 3 },
    { distance: 7 },
    { distance: 3 },
    { distance: 5 },
    { distance: 7 },
    { distance: 3 },
    { distance: 24 }
];
ar = ar.sort(sortafterDistance);
Ti.API.info(JSON.stringify(ar));

    • If you are going to be sorting object arrays you are better off creating a generic sort handler like this one;

      var sortBy = function (prop) {
    return function (a, b) {
        if( a[prop] > b[prop]) {
            return 1;
        } else if (a[prop] < b[prop]) {
            return -1;
        }
        return 0;
    }
};

      Then you can tweak your code as follows;

      ar.sort(sortBy('distance'));
