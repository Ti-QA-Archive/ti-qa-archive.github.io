ScrollViews will not scroll.

I have 2 ScrollViews each taking up half the window. They are vertical columns and need to scroll vertically.

The View XML is like this:

<Alloy>
    <View id="colContainer">
        <ScrollView id="history" class="col">
            <View class="scrollContent">
                <!-- content here -->
            </View>
        </ScrollView>

        <ScrollView id="timeline" class="col">
            <View class="scrollContent">
                <!-- content here -->    
            </View>
        </ScrollView>
        <!-- End Column -->
    </View>
    <!-- End Entire View -->
</Alloy>

The relevant styles are:

"ScrollView":{
    height:Ti.UI.FILL,
    layout:'vertical',
    contentHeight:'auto',
}

/* Columns */
'#colContainer':{
    width:Ti.UI.FILL,
    height:Ti.UI.FILL,
    layout:'horizontal',
    backgroundColor:'#DDD',
}

'.col':{
    width:'50%',
}

".scrollContent":{
    width:Ti.UI.FILL,
    height:Ti.UI.SIZE,
    layout:'vertical',
}

Anybody see any problems here? Neither ScrollView scrolls at all.

    Hi

    Unlike a tableView a scrollView will ONLY scroll if the internal contents of the scrollView exceed the outer size of the scrollView. Think portal window.

    Your scrollContent has a height of size which means it will not exceed the parent size.

    Try these tweaks to check; (not a fix more a test)

    styles

    "ScrollView":{
    height:Ti.UI.FILL,
    layout:'vertical',
    contentHeight: Ti.UI.SIZE // no trailing commas
}

/* Columns */
'#colContainer':{
    width:Ti.UI.FILL,
    height:Ti.UI.FILL,
    layout:'horizontal',
    backgroundColor:'#DDD' // no trailing commas
}

'.col':{
    width:'50%' // no trailing commas
}

".scrollContent":{
    width:Ti.UI.FILL,
    height:1000,
    layout:'vertical' // no trailing commas
}

    • I have the answer - which is weird and a test case so you can see it actually working.

      I would always create a scrollView with a height of Ti.UI.FILL unless I needed a fixed height. Which is what I did here and exactly the same as you did.

      But that seems to be the actual problem; set it to a fixed height it works, set it to 100% and it works. That is VERY strange, I amended my code below to use 100% and suddenly both the motionless scrollViews actually now scroll!

      height: Ti.UI.FILL,

      Becomes 

      height: '100%',

      I played with your code for some time, so changes have been made but the principle above should be the thing you look for;
