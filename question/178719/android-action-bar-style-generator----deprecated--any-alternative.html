  • ActionBar is deprecated starting in Android 5.0. Considering that 5.0 doesn't even have a percentage on the fragmentation chart yet, don't worry about it. https://developer.android.com/about/dashboards/index.html

    Titanium currently only supports the ActionBar, and probably will be like that for a few more versions of the SDK.
