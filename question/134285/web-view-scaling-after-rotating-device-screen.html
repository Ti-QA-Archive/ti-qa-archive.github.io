Web view scaling after rotating device screen

Hi all, I have a table view with multiple words. When u click on a word, a webview will load with an html content aligned to right. The problem is :
If I rotate the screen multiple times, the alignment will be altered ( a white space will be added to the right). I think its a scaling problem, I tried to put this :


var device_width = Titanium.Platform.displayCaps.platformWidth;
<HMTL><meta name="viewport" content="width='+device_width+',initial-scale=1.0, minimum-scale=1.0, maximum-scale=2.0, user-scalable=yes" />

Still having the same problem.
