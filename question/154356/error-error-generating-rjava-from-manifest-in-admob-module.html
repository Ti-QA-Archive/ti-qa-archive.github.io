  • Hello Reseanne,

    I see what you mean.

    You need to do this into your tiapp.xml

    Change this:

        <android xmlns:android="http://schemas.android.com/apk/res/android"/>

    to this:

        <android xmlns:android="http://schemas.android.com/apk/res/android">
        <tool-api-level>14</tool-api-level>
    </android>

    Save the tiapp.xml file and compile it again. Should work. I'm testing it right now in my device.

    Best,

    Mauro
