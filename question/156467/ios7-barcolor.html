iOS7: barColor

I'm trying to change the color in the title bar in iOS7 and does not work, barColor (window) on TabGroup.

function ApplicationWindow(title) {
    var self = Ti.UI.createWindow({
        title:title,
        backgroundColor:'black',
        barColor:'black'
    });

    var button = Ti.UI.createButton({
        height:44,
        width:200,
        title:L('openWindow'),
        top:20
    });
    self.add(button);

    button.addEventListener('click', function() {
        //containingTab attribute must be set by parent tab group on
        //the window for this work
        self.containingTab.open(Ti.UI.createWindow({
            title: L('newWindow'),
            backgroundColor: 'white'
        }));
    });

    return self;
};

module.exports = ApplicationWindow;
  • SDK: 3.1.2
  • xCode: 4.6.2

Test on: iphone 4S with iOS 7

  • Hi

    As far as I am aware this is due to a new property that is required. barTint is the new property for this in xcode but as yet there is no support for it in Titanium as everything about iOS7 (including iOS7) is still in beta.

    Be aware also that some of the things you were previously doing (barColor included) may behave differently when compiled for iOS7 as the some of the UI guidelines and experiences have changed.
