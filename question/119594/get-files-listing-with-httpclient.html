  • Whether or not you can get a file listing returned by naming just a directory path and no web page depends on the server configuration. In general, if the folder has a default web page (such as index.html or index.php or whatever) then the server will return that. In the absence of a default page, whether it returns a file listing depends on the server configuration.

    So if you are asking for a generic method that works via HTTP on all servers, the answer is no. If you care about a specific folder at a specific site, then even if it seems like you get a directory listing returned now, there is no guarantee it will continue to do so. You are at the mercy of the web administrator.

    Also note that just because one folder can be returned that one at a specific server does not mean all folders can.
