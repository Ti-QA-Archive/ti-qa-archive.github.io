My Popover loose my TableView

Hi,

in my iPad app, I use a Popover to display a tableview.

the tableview is created only one time and the same thing with the popover.


categoryPopover = Ti.UI.iPad.createPopover({ 
        width:300, 
        height:250,
        title:'Categories',
        rightNavButton:closeCategoryPopover,
        backgroundColor:'#ccc',
        barColor:'#ccc'
    });

categoryPopover.add(categoryTableView);

myButton = Titanium.UI.createButton({
        title:'Categories'
    });

    categoryButton.addEventListener('click', function()
    {
        categoryPopover.show({
                view:myButton,
                animated:true,
            });    

    });
    win.leftNavButton = myButton;

the first time I show the popover, no problem.
the second time no tableview is visible.
I have to refill the tableview and add again the tableview to the popover.

Am I doing something wrong ?

I'm using 1.6.2.

Regards

  • In popover, I faced also same issue.But when I added view every time before show, it worked. So please try like this,

    don't add table view here // categoryPopover.add(categoryTableView);

    categoryButton.addEventListener('click', function()
    {
    // add table view now
    categoryPopover.add(categoryTableView);
    categoryPopover.show({
    view:myButton,
    animated:true,
    }); 

    });
