  • Hi, Not sure whether this way it can be done for your exact solution but you can try..

    backgroundGradient: {
        type: 'linear',
        startPoint: { x: '0%', y: '50%' },  // your view coordinates as you want
        endPoint: { x: '100%', y: '50%' }, // your view coordinates as you want
        colors: [ { color: 'transparent', offset: 0.0}, { color: 'backColorof the View', offset: 1.0 }],
    }
