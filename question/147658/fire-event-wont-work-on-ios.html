  • Hi

    The first problem is likely just a typo issue for your example;

    fireEvent('test',{myview:theview,sometext:'text'}) // fireEvent is the right way as it is camel case

    The real problem is that fireEvent and addEventListener only pass data. You cannot pass views between them.

    You have to pass a view in another way within your app.

    // all valid
{
    one: 'Hello',
    two: 2.34,
    three: 90
}

    Just not views, buttons, windows, labels etc.
