  • You likely need to create an onClick function for the tableviewrow and then check the source id to see if the user is clicking on the textfield. Then, if the user is clicking on the textfield, you can just focus on it.

    function rowClick(e){    
    if(e.source.id == '<whatever your textfield id is>') {
        textfield.focus();
        return; //this is to prevent the rest of the function from running
    };
