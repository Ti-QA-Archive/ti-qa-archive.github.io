iPhone - event listener for detecting touch/moving of image inside scrollview

Hi

I have a simple window and script set up which needs to detect when the user touches or moves an image inside a scrollview like so:

Setup - iPhone iOS7 - Ti SDK 3.2.1 GA (latest)

var win = Ti.UI.createWindow({backgroundColor: '#fff'});

var imageHolder = Ti.UI.createScrollView({contentWidth:'auto', contentHeight:'auto', height:200, width:200, showVerticalScrollIndicator:false, minZoomScale:1, maxZoomScale:1.3, zoomScale:1.01});

var image = Ti.UI.createImageView({top:0, left:0, image:'img.jpg'});

image.addEventListener('touchstart', function() {
    Ti.API.info('moved image');
});

imageHolder.add(image);    
win.add(imageHolder);
win.open();

But the output is not always accurate - it always detects when i click on the image, but when i drag or move the image it mostly doesn't output anything. I have to click and then move the image for it to output and even then sometimes it only does it on the second time

Am i using the correct event listener and/or on the correct object? I have tried placing it on the imageHolder instead but with the same result
