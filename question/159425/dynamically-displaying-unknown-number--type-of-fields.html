Dynamically displaying unknown number / type of fields

###HELP!!!

I am pretty well versed in developing apps with Titanium but am now faced with a challenging situation:

I need to display an unknown number and type of fields returned as JSON records from a web service.

Basically some records will have 1-10 fields (string, date, array) while others will have 1-15+ fields of different names and types.

I can dynamically loop through the fields key/value pairs no problem, but some fields should be displayed as Labels, while others are a Checkbox or Textfield, etc…

Has anyone come across a similar situation? Are there samples out there like this?
