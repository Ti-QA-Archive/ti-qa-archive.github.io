  • I have the same problem. But, I found out solution.

    We must install with apt-get command.(sun-java6-jdk and "sun-java6-plugin"<-This package is important!)

    I think that sun-java6-plugin is package for web and javascript. So, we must install this package for Titanium.

    But, sun-java was removed repository…So, I use "add-apt-repository" command and "ppa".

    $ sudo add-apt-repository ppa:sun-java-community-team/sun-java6

    (If keyserver refused, You should edit [/usr/share/pyshared/softwareproperties/ppa.py] script.

    Wrong:

     --keyserver keyserver.ubuntu.com

    Correct:

     --keyserver hkp://keyserver.ubuntu.com:80

    )

    $ sudo apt-get update

    $ sudo apt-get install sun-java6-jdk sun-java6-plugin

    I can build Titanium project on this environment.

    Try it.
