That window is loading a new JS file that opens up the photogallery and I can think that this causes the problem? If if close this window and reopen it a couple of times, controls on this window stops to respond?

    Are you opening the new window as modal? If so, I recommend you open using Ti.UI.currentTab instead of w.open(); If you don't have a current tabgroup, create a dummy one with only one tab. You can set the tabBarHidden property on the new window so the window looks like it is standalone.
