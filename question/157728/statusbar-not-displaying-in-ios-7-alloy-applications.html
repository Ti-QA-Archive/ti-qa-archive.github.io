Statusbar not displaying in IOS 7 Alloy applications

The status bar does not display for applications created from the templates.

IOS 7, SDK 3.1.0.GA, Alloy 1.2.2

Steps to reproduce

  1. Open titanium studio

  2. File->New->Module Project

  3. Select Alloy->Default Alloy Project (or tabbed)

  4. Name and finish creating the project.

  5. Build and launch to IOS 7

Result: No status bar after splash screen, only black empty space

Expected: Status bar on window.

Cannot get the status bar to appear for alloy created applications. Please advise. I have tried the fullscreen and statusbar style attributes with no success.
