  • Basically, a popover is just going to give the frame and optionally the title bar. The styling of everything inside is up to the views you place inside it.

    You'd get the frame using something like this (untested and typed into msg):

    var btnTables = Ti.UI.createButton({title:'Tables'});
var btnEmployees = Ti.UI.createButton({title:'Employees'});
var popServers = Ti.UI.createPopover({
  title: 'Servers',
  leftNavButton: btnTables,
  right NavButton: btnEmployees,
  arrowDirection:Ti.UI.POPOVER_ARROW_DIRECTION_DOWN
});

    For everything inside the popover, you'd style things just like a regular view. In this case, you'd create a table, set the left image, title, and add buttons for the stuff on the right.

    Then add that table to the popover before doing the popover show method.
