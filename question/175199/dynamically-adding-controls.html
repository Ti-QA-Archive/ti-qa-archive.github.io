  • Hi

    There are several way, here are two;

    Simple way

    // assumes there is a view with
// the id of "parent"

var txt = Ti.UI.createTextField({
  top: 30,
  value: 'hello world'
});
$.parent.add(txt);

    Smarter way (not always though - simple can be better, depends on needs of that section)

    // assumes there is a view with
// the id of "parent"

var txt = Alloy.createController('example');
$.parent.add(txt.getView());

    example.xml

    <Alloy>
  <TextField id="txt">hello world</TextField>
</Alloy>

    This way has many advantages including; true separation using MVC ideals, ability to take parameters to define the single object.

  • Two ways.

    1. Use traditional Ti javascript code to create the controls you require and add them in the controller.

    index.xml

    <Alloy>
    <Window id="win" />
</Alloy>

    index.js (controller)

    
var t = Ti.UI.createTextField();

$.win.add(t);

    or if your controls are more complicated widgets

    control.xml (view)

    <Alloy>
    <ScrollView>
        <TextField id="title" class="title" hintText="Title" />
        <TextField id="first" class="first" hintText="First name" />
        <TextField id="last" class="last" hintText="Last name" />
        <TextField id="email" class="email" hintText="E-Mail address"  />
        <TextField id="postcode" class="postcode" hintText="Postcode"  />
        <TextField id="promo" class="promo" hintText="Promotion Code"  />
    </ScrollView>
</Alloy>

    index.js (controller)

    
var mycontrol = Alloy.createController('control').getView();

$.win.add(mycontrol);
