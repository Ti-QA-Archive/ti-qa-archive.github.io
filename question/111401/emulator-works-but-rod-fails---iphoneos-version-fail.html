  • Tobias

    At the end of your build.log, errors are shown that you must resolve before it will work properly. For example, "warning: Application failed codesign verification. The signature was invalid, or it was not signed with an Apple submission certificate."

    This is the first way you would troubleshoot these types of errors is:

    • delete all certificates and provisioning profiles
    • reinstall certificates and provisioning profiles
    • ensure the alias is in the correct format
    • ensure the App ID always matches the provisioning profile

    Let me know if this works.

    Cheers
