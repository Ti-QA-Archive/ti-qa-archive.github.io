Facebook module's login event not firing

Just as the title suggests, the Facebook module's login event isn't firing. Works in the simulator however the simulator doesn't appear to be using the iOS 6 built-in Facebook functionality. Here's an example of the code I'm using:

if (!facebook.loggedIn) {
    facebook.appid = "MY APPLICATION ID";
    facebook.forceDialogAuth = false;
    facebook.permissions = [];
    facebook.addEventListener("login",loggedIn);
    facebook.authorize();
} else alert("The user is already logged in.")

function loggedIn(e) {
    alert("The user is now logged in.");
};

The permissions dialog pops up as expected, after accepting/denying it disappears but nothing else happens. I'm testing on an iPhone 4S with iOS 6.1.3 (10B329) using Titanium SDK version 3.1.0GA.

Thanks for checking out my question!
