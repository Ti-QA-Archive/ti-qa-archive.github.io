  • Hi

    The short answer is no.

    In order to create an iOS app; iPhone or iPad you require xcode to compile the app- even when using Appcelerator. In order to use xcode you must have an Apple computer as they are the only ones that run xcode.

    You can find more information in the Titanium Compatibility Matrix.

    There are services that alloe you to rent the use of a Apple in the cloud that you use remotely. They can be expensive so it is often cheaper to buy a Mac Mini and work with that.

    If you want to run a mobile html5 app that could be run on an iOS device you can achieve that without an Apple, but not a native app.

    Sorry
