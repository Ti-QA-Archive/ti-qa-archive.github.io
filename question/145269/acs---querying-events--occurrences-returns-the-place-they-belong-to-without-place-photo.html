  • SOLVED: An important property to add to the request is response_json_depth, but default for Event Occurences it's 3, if 4 is requested, the photo property for Place of the Event shows up :)

    example:

    Cloud.Events.queryOccurrences({
            page: 1,
            per_page: 200,
            where: { start_time : { "$gte" : startDate.format(), "$lt" : endDate.format() } }, 
            order: 'start_time',
            response_json_depth: 4
