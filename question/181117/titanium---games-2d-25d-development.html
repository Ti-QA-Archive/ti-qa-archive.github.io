  • Hello Emanuele,

    Yes, if you want to develop games with Titanium your best option is using Platino. Supports spritesheets, particle effects, orthogonal and isometric maps, camera view, animations and more features for game development.

    In addition I inform you that they have just released a 64 bit update.

    Here you will find all the information about this game engine: www.platino.io
