  • You navigation code is correct.. the only error that you are making is win.add(nav).
    For a navigationGroup you have to define a base window that will hold your navigationGroup. and then you create a window that is your main window..look at this example you will get clear..

    // your base window
var win = Ti.UI.createWindow();
// your main window where you wil show you content
var mainWin = Ti.UI.createWindow({

backgroundColor:'#fff'
});

var nav = Titanium.UI.iPhone.createNavigationGroup({
   window: mainWin
});

win.add(nav);


var btn = Ti.UI.createButton();

mainWin.add(btn);

win.open();

    This will resolve your crash…
    Hope that helped..:)
