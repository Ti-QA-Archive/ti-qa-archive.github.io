Does Not An Event Accept Objects As Data Payload?

I have resorted to using a timeout function for passing data to a webview, because adding a listener to a webview seems to not hear a page load. (At least not as advertised on Android. iOS - of course - works perfectly. Sigh.)

Please someone explain to me why this will work:

setTimeout(function(e){
    Titanium.App.fireEvent('pageReady',{piece:win.piece.title});
}, 10000);

And this will not:

setTimeout(function(e){
    Titanium.App.fireEvent('pageReady',{piece:win.piece});
}, 10000);

I'd like to pass data as object, not have to unpack it before sending. And yes… this is already JSON object. Thank you.
