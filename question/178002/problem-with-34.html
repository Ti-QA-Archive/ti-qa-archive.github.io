  • Hello, Roger, Did you check if your studio is Properly configured with Xcode? in Studio/Platform/iOS/.

    If you have Two Xcode installed "Xcode5" and "Xcode6" then Xcode6 must be anstalled as Xcode 2.app in Application.

    If so then Run This in Terminal

    sudo xcode-select -switch /Applications/Xcode2.app/Contents/Developer/
