Activity indicator not visible on Android2.3

activityIndicator = Ti.UI.createActivityIndicator( {
    color : '#f8f8f8',
    font : {
        fontSize : '18dp',
        fontWeight : 'normal'
    },
    message : 'Loading...',
    style : Ti.UI.ActivityIndicatorStyle.PLAIN
    ,backgroundColor: '#000'
    ,height : '60dp'
    ,width : '170dp'
} );
parentWindow.add( activityIndicator );
activityIndicator.show( );

The code above seems to work fine on Android 4.0 or later, but not on 2.3.5 or before. On older devices, the loading icon (rotating one) is not displayed, while only the text "Loading…" appears. Why and how can I fix that? Thanks in advance!
