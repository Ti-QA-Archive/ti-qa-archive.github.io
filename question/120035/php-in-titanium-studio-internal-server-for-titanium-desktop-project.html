PHP in Titanium Studio Internal Server for Titanium Desktop project

  • Aptana 3.0, and thus Titanium Studio, still have a built-in web server, but I think it's not PHP-enabled anymore, as this was droped in Aptana 2.0 in favor of the PDT Project. "In production" you will need an external webserver anyway.

    Your browser probably just downloads the PHP-file, because the webserver is not configurated to execute them.
