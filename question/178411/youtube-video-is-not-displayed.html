YouTube video is not displayed

Hi, I use this code to load in a webview a YouTube video:

var webView = Ti.UI.createWebView({
    url: 'http://www.youtube.com/embed/MQOt_PQjNV4?autoplay=1&autohide=1&cc_load_policy=0& color=white&controls=0&fs=0&iv_load_policy=3&modestbranding=1&rel=0&showinfo=0',
    enableZoomControls: false,
    scalesPageToFit: false,
    scrollsToTop: false,
    showScrollbars: false
});

but when I test in the simulator I receive the following debug warning and nothing is displayed (neither the videoplayer itself or the view containing it):

[DEBUG] :  New scheme: <NSMutableURLRequest: 0xd7441c0> { URL: http://www.youtube.com/embed/MQOt_PQjNV4?autoplay=1&autohide=1&cc_load_policy=0&color=white&controls=0&fs=0&iv_load_policy=3&modestbranding=1&rel=0&showinfo=0 }

Any idea?

  • Hi Gab, I just tested your code and it works fine on 3.3.0. Make sure you're creating a window first and attaching the view to it. If you did and it's still not working, try cleaning the project and recompiling it.

    var win = Titanium.UI.createWindow();

var webview = Titanium.UI.createWebView({
    url: 'http://www.youtube.com/embed/MQOt_PQjNV4?autoplay=1&autohide=1&cc_load_policy=0& color=white&controls=0&fs=0&iv_load_policy=3&modestbranding=1&rel=0&showinfo=0',
    enableZoomControls: false,
    scalesPageToFit: false,
    scrollsToTop: false,
    showScrollbars: false
    });

win.add(webview);



    win.open({modal:true});

  • Gab,
    You can always attach that webview to a view but make sure the view is attached to the window as well. You can make it anyway you want just by playing with the properties and the layout:

    var win = Titanium.UI.createWindow();

var view = Ti.UI.createView({
    top:20,
    height:100,
    width:300
});


var webview = Titanium.UI.createWebView({
    url: 'http://www.youtube.com/embed/MQOt_PQjNV4?autoplay=1&autohide=1&cc_load_policy=0& color=white&controls=0&fs=0&iv_load_policy=3&modestbranding=1&rel=0&showinfo=0',
    enableZoomControls: false,
    scalesPageToFit: false,
    scrollsToTop: false,
    showScrollbars: false
    });
 view.add(webview);
    win.add(view);



    win.open({modal:true});

    I hope that answers your question.
