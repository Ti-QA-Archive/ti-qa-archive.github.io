  • Without seeing your code I can really only comment in general terms, but here is no "reset" function built into a window by default. What you might try is creating a function which creates a window:

    function createMyWindow() {
  var win = Ti.UI.createWindow();
  //add stuff to window
  return win;
}

    and then every time you open that window, regenerate it with the function:

    createMyWindow().open();

    If you are using windows with url properties, this may require that you refactor your code somewhat.
