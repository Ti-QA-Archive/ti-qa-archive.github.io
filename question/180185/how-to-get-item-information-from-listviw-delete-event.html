how to get item information from listviw delete event

For deleting a item from ListView in iOS, I set canEdit property.
It succeeded in iOS application.
But I cannot get ItemID, so I cannot delete that item from RDBMS.

This code get another one item, because target item has already deleted.

listivew.addEventListener('delete', function(event){
    var item = event.section.getItemAt(event.itemIndex);
}

I think getting itemID before delete event fired, But listview doesn't have touch event, and when swipe a row, itemclick event not fired too.

How to get itemID when delete event fired.

  • Accepted Answer

    You need to have set the itemId as part of loading data into the list view

    items.push({
   someLabel: { text: "Text to show in the list item"},
   properties: { itemId: dbResults.primaryKeyOrID }
});

    Then, it should be available as e.itemId
