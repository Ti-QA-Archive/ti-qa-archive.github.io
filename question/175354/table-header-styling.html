Table header styling

In the following:

<Alloy>
    <View id="secondProducts">
        <TableView id="productsTable">
            <TableViewRow title="Clear Products Field (cancel)" value=""></TableViewRow> 
            <TableViewRow header="USED" title="Standard" value="Standard"></TableViewRow>   
            <TableViewRow title="Other" value="Other"></TableViewRow>                 
        </TableView>
    </View>
</Alloy>

How would one style the generated header row marked header=used?

  • While there turns out to be no method to style the above header using TSS, a minor restructuring of the table with a HeaderView provides the same behavior and allows styling and flexibility:

    <TableView>
    <TableViewSection>                
        <HeaderView>
            <View class="productsTableHeader">
                <Label class="productsTableHeaderText">NEW</Label>
            </View>
        </HeaderView> 

        <TableViewRow title="Standard" value="NEW  Standard"></TableViewRow> 
    </TableViewSection>
</TableView>

    ".productsTableHeader": {
    height: "50dp",
    backgroundColor: "#000"
},
".productsTableHeaderText": {
    color: '#fff',
    textAlign: 'center'
}
