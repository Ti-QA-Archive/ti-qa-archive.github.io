Calculating proportional width and height of images.

Suppose I need a image size maximum of 70*70 so that it can get fit into container , but I download images in my app are of different width and height , but I want to resize image proportionately.

I am also able to find the actual width and height of image but want to convert proportionately, but the problem is how to get proportional width or height near by 70*70 , (width and height) , so that it could fit in the same box , without getting distorted.

    hi,

    you can calculate it something like..

    if (imgWidth > 70) {

    var h = (70 * imgHeight) / imgWidth;
    //this gives you proportiona height as per image width.

}

OR
if (imgHeight > 70) {

    var w = (70 * imgWidth) / imgHeight;
    //this gives you proportiona width as per image height.    
}
