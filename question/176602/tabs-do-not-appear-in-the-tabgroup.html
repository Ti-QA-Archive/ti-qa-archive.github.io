tabs do not appear in the TabGroup

Hello,

I am trying to add tabs to my titanium app. I am running 3.2.3 GA with iPhone iOS 7.1 target. Somehow, no tabs appear in the bar, as if the TabGroup was empty. I tried to add TabGroup and display it as is. Neither seems to make difference.
Please advise: what am I doing wrong?

Below is my code:

var peopleWindow = Ti.UI.createWindow({
backgroundColor:'white'
});
var messagesWindow = Ti.UI.createWindow({
backgroundColor:'white'
});
messagesWindow.add(chatTextArea);

var peopleTab = Ti.UI.createTab({
window : peopleWindow,
title : “People”
});
var messagesTab = Ti.UI.createTab({
window : messagesWindow,
title : "Messages"
});

var chatTabGroup = Ti.UI.createTabGroup({
width : 220,
height : 480,
left : 100,
top : 0,
borderColor : 'blue',
borderWidth : 1,
barColor : 'red',
navBarHidden : false,
tabBarVisible : true,
tabHeight : 30,
tabsAtBottom : false,
tabDividerColor : 'black',
tabDividerWidth : 2,
backgroundColor : 'white'
});
chatTabGroup.add(peopleTab);
chatTabGroup.add(messagesTab);
chatTabGroup.open();

Thank you,

Igor Shmukler

    Hi Igor, 

    chatTabGroup.add(peopleTab);

    should be

    chatTabGroup.addTab(peopleTab);

    see addTab in the docs
