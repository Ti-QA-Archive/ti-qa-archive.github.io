titles to the alert dialogue box on iphone

How do I change the title to an alert dialogue from "webpage.html", to a custom title on the iphone? I've created a game and whenever an event is fired, a box with a title: "webpage.html" along with the message appears.
I designed the games in such a way that they use javascript, css and html. Is there a solution?

    Here is what I do to set an "are you sure" dialog on my reset button. Notice the title property on the alertDialog object.

                resetbutton.addEventListener( 'click', function () {
                var alertDialog = Titanium.UI.createAlertDialog( {
                    title: 'Reset Schedule',
                    message: 'Are you sure you want to reset the schedule?',
                    buttonNames: [ 'OK', 'Cancel' ],
                    cancel: 1
                } );

                alertDialog.addEventListener( 'click', function ( theEvent ) {
                    if ( theEvent.index !== theEvent.cancel ) {
                        myApp.base_win.animate( { view: myApp.entryNav, transition: Titanium.UI.iPhone.AnimationStyle.FLIP_FROM_LEFT } );
                    }
                } );

                alertDialog.show();
            } );

    • Keep in mind that most of the Ti functionality, including alertDialog, is exposed only to the non-HTML javascript (IE, app.js, or a window's url's js file). So if you want to trigger this from an HTML page, look into Kitchen Sink's web_views.js and local_webview.html on how to relay events from HTML to JS. Namely, Ti.App.fireEvent('eventname') on HTML, Ti.App.addEventListener('eventname', function(E){…}) in JS.
