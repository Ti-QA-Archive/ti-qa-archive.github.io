    Hi Xiong,

    I have had this error before when I tried to add Titanium activity indicator. Try the following as it worked for me. Go to Project > Clean. Then select your current project and click okay. Run your app in the simulator again and see what happens.

  • Well this happens sometimes because Ti loads needed libraries on demand, while Simulator gets all modules by default. In this case you got caching issues, removing the build folder will force a full rebuild. I agree this is anoying but it occurs more less than it did in the past and if you know that you can fix it fast.
