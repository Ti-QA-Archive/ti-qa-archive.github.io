  • The solution I found is download the latest android/ios facebook module from github:

    https://github.com/appcelerator-modules/ti.facebook/blob/master/ios/facebook-iphone-3.1.2.zip
    https://github.com/appcelerator-modules/ti.facebook/blob/master/android/dist/facebook-android-3.0.3.zip

    Add them to your project and then you can do the following:

        var fb = require('facebook');
    fb.appid = '1234567890';
    fb.publishInstall();

    Cheers,
    J.
