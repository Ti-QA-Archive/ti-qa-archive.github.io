  • Here is answer
    You can make stroke using shadowOffset. Create four label object and apply shadowOffset x:1, y:1 for first label , second : x:1, y:-1, third : x:-1, y:1, last : x: -1, y:-1.

    example of Alloy .tss 

    
"#labelOutline":{
    text : 'Hello',
    color : '#8C1414',
    shadowColor : '#F0F2B1',
    shadowOffset: {x:-1, y:-1},
    textAlign: Ti.UI.TEXT_ALIGNMENT_CENTER
}

"#labelOutlineRight":{
    text : 'Hello',
    color : '#8C1414',
    shadowColor : '#F0F2B1',
    shadowOffset: {x:1, y:-1},
    textAlign: Ti.UI.TEXT_ALIGNMENT_CENTER
}

"#labelOutlineLeft":{
     text : 'Hello',
     color : '#8C1414',
     shadowColor : '#F0F2B1',
    shadowOffset: {x:-1, y:1},
    textAlign: Ti.UI.TEXT_ALIGNMENT_CENTER
}

"#label":{
   text : 'Hello',
    color : '#8C1414',
    shadowColor : '#F0F2B1',
    shadowOffset: {x:1, y:1},
    textAlign: Ti.UI.TEXT_ALIGNMENT_CENTER
