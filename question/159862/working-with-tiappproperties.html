  • You don't state what your expectations are. So, it's hard to be certain what's "failing" with your code. This line will always return something, hence the first block of your if statement will always execute:

    if (Ti.App.Properties.getString('checkUser','userNotLoged'))

    By providing a second arg to any of the get* functions, you're providing a default value to be supplied if the property doesn't exist or is empty.

    BTW, to continue testing with this app, you'll need to uninstall it. That's because you've set a property named 'checkUser' by running the app once, thus your hasProperty() check will be bypassed on all future runs (until you uninstall it or delete that property explicitly).
