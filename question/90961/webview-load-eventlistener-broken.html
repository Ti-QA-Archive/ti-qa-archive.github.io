  • I am having the same issue. For me both the 'beforeload' and 'load' event listeners have stopped working.

    I use the following code to show a loading indicator when the webview is loading, which worked without any problems before I upgraded to 1.5.1.:

    image.addEventListener('beforeload', function() {
    actInd.show();
    win.add(actInd);
});
image.addEventListener('load', function() {
    actInd.hide();
});

    Anyone got any suggestions or knows whether this is a bug and will be fixed?
