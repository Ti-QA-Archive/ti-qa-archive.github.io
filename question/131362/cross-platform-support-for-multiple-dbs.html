Cross platform support for Multiple DB's?

I currently have an app (Android and IOS) out in the market with one db (db part of app). I was wondering if its possible to create (via scripts) more than one db for the app. Also does iTunes back up both databases?

  • Accepted Answer

    You can have as many databases in your app as you want.. I have 1 per client project, and 1 to manage all the databases that exist so I don't have to loop through a list of files at runtime (and XML was seriously buggy at the time I implemented this bit). You can spawn a new database from an empty shell from the resources folder, or stream one as a normal file download and install it from a server for example..
