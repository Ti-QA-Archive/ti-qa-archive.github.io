How do I encapsulate a createLabel & createTextField in a createView

Hello,

I have a question about formatting. I have a createView and within this view I first want to put a label (createLabel) on the left hand side and then a text field (createTextField) on the right hand side at the same level. Can someone tell me how to do this. I know how to create a view, label and text field but I need to combine them.

This is my code so far

            var view = Ti.UI.createView({
                height:'auto',
                layout:'vertical',
                left:10,
                top:10,
                bottom:10,
                right:10,
                borderWidth:2,
                borderColor:'#203346',
                borderRadius:5,
                backgroundColor:'#E8E8E8',  // a shade of grey //#E8E8E8
                borderStyle:Ti.UI.INPUT_BORDERSTYLE_ROUNDED
            });
            var l1 = Ti.UI.createLabel({
                font:{fontFamily:'Helvetica Neue',fontSize:18,fontWeight:'bold'},
                text:'Name', 
                height:'auto',
                top:10,
                left:15,
                color:'#FF6600',
                //color:'black',
                font:{fontWeight:'bold',fontSize:14}

            });
            view.add(l1);

            var l2 = Titanium.UI.createTextField({
                value:'Bob',
                hintText:'Name',
                font:{
                    fontFamily:'Helvetica Neue',
                    fontSize:14
                },
                height:25,
                top:10,
                left:90,
                width:200,
                borderStyle:Titanium.UI.INPUT_BORDERSTYLE_ROUNDED
            });

            view.add(l2);
            win.add(view);

This code place the text field at the bottom of the view unaligned.

Thanks in advance

    I think some of your view's properties is messing things up, namely layout:&#39;vertical&#39;

    Try setting your view like this:

    var view = Ti.UI.createView({
 //top: 40    // optionally set top property to control view position
 height:80,
 width:Ti.Platform.displayCaps.platformWidth - 20,
 borderWidth:2,
 borderColor:'#203346',
 borderRadius:5,
 backgroundColor:'#E8E8E8',  // a shade of grey //#E8E8E8
 borderStyle:Ti.UI.INPUT_BORDERSTYLE_ROUNDED
});
