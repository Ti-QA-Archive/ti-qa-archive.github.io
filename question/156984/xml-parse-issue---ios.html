    Hi,

    Check the XML, you have not closed the tags properly. Validate your XML here - http://www.xmlvalidation.com

    The way you approached is correct.

    var contents = f.read().text;
var doc = Ti.XML.parseString(contents);// remove this 
var surveyId = doc.documentElement.getElementsByTagName("SurveyID");
Ti.API.info(surveyId);
