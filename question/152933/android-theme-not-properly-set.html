Android theme not properly set

So I'm facing big issues with the android theme using Titanium.

1 - The default theme is always android 2.2 even if I set the minimum and target sdk to 17

<uses-sdk android:minSdkVersion="17" android:targetSdkVersion="17"/>

2 - If I do follow older questions and set it in my tiapp.xml to Holo.Light my splash screen does not appear anymore

android:theme="@android:style/Theme.Holo.Light">

Here is a simple code to test the theme.

Possible Results:

1 - Orange progress Bar = android 2.2

2 - Blue progressBar without background Holo

3 - Blue progressBar with gray background Holo.Light

For 2 and 3 please let me now how you did it, without losing the splash screen like me.

//app.js

var parentView = Ti.UI.createWindow();

// Create a ProgressBar.

    min : 0,
    max : 10,
    value : 0,

    height : '70dp',
    color : '#444',

    font : {fontSize:14, fontWeight:'bold'},
    style : Ti.UI.iPhone.ProgressBarStyle.PLAIN,
    top : 60
});

        aProgressBar.value = val;
        aProgressBar.message = 'Progress: ' + val + ' of 10';
        val++;
    } else {
        aProgressBar.message = 'Progress complete!';
        clearInterval(interval);
    }

  • OK, at the end of the day I found out how it work, but also filled a ticket, as it's a very bad way to do this kind of thing.

    We need to set the tiapp.xml, with application and activity tags like this:

                <application android:theme="@style/LightTheme">
                <activity android:name=".ThemetestActivity"    android:label="ThemeTest" android:theme="@style/LightTheme"    android:configChanges="keyboardHidden|orientation">
                    <intent-filter>
                        <action android:name="android.intent.action.MAIN" />
                        <category android:name="android.intent.category.LAUNCHER" />
                    </intent-filter>
                </activity>   
            </application>

    And we need to create a custom theme.xml so splashs and stuff like that will work..

    platform/android/res/values/theme.xml

    <?xml version="1.0" encoding="utf-8"?>
<resources>
    <style name="LightTheme" parent="@android:style/Theme.Holo.Light">
        <item name="android:windowBackground">@drawable/background</item>
        <item name="android:windowNoTitle">true</item>
    </style>
</resources>

    And the ticket I did: TC-2368
