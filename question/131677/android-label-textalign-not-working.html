Android Label textAlign not working

Hi guys,

I have a table to which I'm trying to add some rows with a view, and labels inside the view:

var tableRow = Ti.UI.createTableViewRow();
var view = Ti.UI.createView({
        layout: 'vertical',
        width: '188dp',
        left: '16dp',
        top: 0,
        bottom: 0,
        id: 'right',
    });
var label = Ti.UI.createLabel({
        backgroundColor: 'blue',
        color: '#000',
        text: text,
        textAlign: Ti.UI.TEXT_ALIGNMENT_RIGHT,
        top: '4dp',
        right: 0,
        left: 0,
        bottom: '4dp',
        width: '188dp',
        font: {
            fontSize: '14sp'
        }
    });
view.add(label);
tableRow.add(view);
tableView.appendRow(tableRow);

…but the labels won't right align at all, and the backgroundColor property isn't working either. Does anybody have an idea about this?

Thank you!
