  • Try not to answer you own questions unless providing an answer. Use the comment system where possible when responding to an answer or providing more information.

    Whenever you update your Google APIs or JDK, you should open up your AVD Manager and delete all of the virtual devices that are shown. Then go back and run Project > Clean from Studio. Last, build your project for emulator again and let the virtual device be re-created from scratch.
