  • Download the Android SDK.You don't have to install the eclipse that comes with the big ADT package, can just pull the SDK folder out.

    Place it in a location WITHOUT SPACES.

    In TiStudio, Window || Preferences || Titanium Studio || Titanium you will need to specify the location of your Android SDK.

    Additionally, you need to setup system environment per http://docs.appcelerator.com/titanium/latest/#!/guide/Software_Locations_and_Environment_Variables-section-29004844_SoftwareLocationsandEnvironmentVariables-ConfiguringEnvironmentVariablesonWindows

    • Jason has just made me think about your question differently from the way I first read it, so here goes.

      All titanium projects are created through that single option, when you fill the details out you have a choice of;

      • android
      • iPhone
      • IPad
      • blackberry
      • mobile

      You can choose as many as you like as long as they are configured and available on your development platform.

      So you will not see a specific menu for each as they get chosen in one place as you could be targeting all platforms with one set of code.
