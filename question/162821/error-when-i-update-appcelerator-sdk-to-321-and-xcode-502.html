  • Hello Raul,

    I have seen some activity on the Q&A related to this, with several different fixes:

    • Fix: Did you open the Xcode 5 at least once? You need to accept the licenses.
    • Fix: Install 3.1.4 and create a mobile project using it. Related question here.
    • Fix: Update NodeJS to 0.10+. Related question here.
    • Fix - user CI build for 3.2.1. Check this bug: https://jira.appcelerator.org/browse/TIMOB-16226.
    • Fix: Check if your CLI is up to date with your Ti Studio. Check this bug: https://jira.appcelerator.org/browse/TIMOB-16232.

    Let me know if any of this helped you.

    Best,

    Mauro
