Titanium can not find Android SDK

Hi,
Ti is complaining:
Titanium requires Android platform 2.2.*
Add-On Google APIs version 8

I've installed Android SDK by clicking on "Install or Update Android SDK" button (via
the Android SDK Manager).
The SDK Manager shows that I have everything below installed:

  • Tools
  • Android SDK Tools - rev 21
  • Android SDK Platform-tools rev 16
  • Android 4.2 (API 17)
  • SDK Platform - api 17
  • Google APIs - api17
  • everything else in Android 4.2

I've checked that Android SDK Home points to the android-sdk path

I have path set as well:
export PATH=${PATH}:/Users/john/android-sdk/platform-tools:/Users/john/android-sdk/tools

restarted Ti and still complains.
Titanium Studio, build: 3.0.1.201211021836

please help, can't even create project w/o this.
thanks,
john

    Not a version higher than 2.2, but 2.2, the current minimum API that Titanium supports.
