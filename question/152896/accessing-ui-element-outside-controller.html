    OK, lots of issues:

    • Don't put parentheses in the "on" attributes. Do onClick=&#39;func&#39;, don't do onClick=&#39;func()&#39;.
    • Don't override the keyword "do", it's reserved in javascript.
    • All you need to do is add a reference to the $ in the controller to make it available externally.
    var view = Titanium.UI.createView();
view.id = "view_id_1";
$.window.add(view);
$.view_id_1 = view;

    but if you want to make it accessible via the getView() function, you may need to add it directly to the internal array

    var view = Titanium.UI.createView();
view.id = "view_id_1";
$.window.add(view);
$.__views.view_id_1 = view;
