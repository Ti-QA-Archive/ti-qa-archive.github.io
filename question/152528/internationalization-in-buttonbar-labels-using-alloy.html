Internationalization in ButtonBar labels using Alloy

Hello.

I have a Map View with a ButtonBar at the bottom. I need the text of the labels in the ButtonBar to change according to the language of the phone. In order to achieve this I use internationalization. But I get an error everytime I run my code.

I'm using the iPhone simulator, Titanium SDK 3.1.0. GA (I also tried with 3.0.2 GA and didn't work either), Alloy 1.0, OSX 10.7.5, Titanium Studio, build: 3.1.0.201304151600.

map.xml

<Alloy>
    <Window id="winMap">
        <View id="viewMap" ns="Ti.Map" animate="false" regionFit="false" userLocation="false" mapType="Ti.Map.STANDARD_TYPE">
            <ButtonBar platform="ios">
            <Labels>
            <Label id="labelOne">Label</Label>
                <Label>++</Label>
                <Label>--</Label>
            </Labels>
        </ButtonBar>
        </View>
    </Window>
</Alloy>

map.js

$.labelOne.text = L('search');

$.winMap.open();

map.tss

"#winMap": {

}

"ButtonBar[platform=ios]": {
    backgroundColor: '#36566A',
    bottom: 0,
    style: Ti.UI.iPhone.SystemButtonStyle.BAR,
    height: 40,
    width: Ti.UI.FILL
}

When I run this code I get the following error:

[ERROR] : Script Error = 'undefined' is not an object (evaluating '$.labelOne.text = L("search")') at map.js (line 55).

I have the i18n files in the right place 'cause they work when I use them in other controllers. They're in the i18n folder at the same level app folder is.

I think I've included all the information needed. Hope you can help me :)

Thanks in advance.
