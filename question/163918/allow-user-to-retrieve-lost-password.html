Allow user to retrieve lost password

I have created a password protected area in my app. It works fine, but what I'm not sure how to handle is if they lose their password. The password is stored in properties. I wanted to be able to send the password to an email, but it appears I cannot send an email without going through a server, and I don't want to maintain a server dedicated to this. Using the email dialog is not an option, because then the message can be seen by anyone who chooses the send password option. Is my assessment correct? Is there another option I'm not thinking of?

  • You should never store the unencrypted version of the password and you should never send it by email. You open yourself up to all sorts of security compromise scenarios if you do either. Instead, create a password reset system by which folks receive some sort of reset key. They would enter that key, along with a new password, to change passwords.
