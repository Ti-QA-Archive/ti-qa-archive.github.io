can not run iphone simulator

I got error as "Accepted values: [ERROR] Invalid –ios-version value '5.0' " when running iphone simulator.
Many of times, I was trying uninstall and install xcode, titanium over again. Also, I did xcode-select command and then refresh many times. But, never running iphone simulator. I'm using MacOSX 10.6.8, xcode 4.2, Titanium Studio, build: 3.0.1. Nothing to help on documentation, help and archive QA.

    Do you have iOS Simulator 5.0 installed? If you run titanium info -t ios from the Terminal, it will display all Xcode installs that Titanium found and for each Xcode install it will list the iOS SDKs found. If you don't see 5.0 for any of the Xcode installs found, then you need to install the iOS Simulator for one of the Xcode versions found. You can do this by launching Xcode, then go into the Preferences, then to Downloads, and install the 5.0 SDK.
