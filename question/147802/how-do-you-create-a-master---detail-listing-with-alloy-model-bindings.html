How do you create a master - detail listing with Alloy model bindings?

I'm using the development version of Alloy, from Git. How do you create a master - detail view using model binding? For example:

I have a TableView, which is bound to a Collection which comes from an SQLite DB, looks a little like this:

<Alloy>
    <Collection src="book" />
    <Window class="container">
        <TableView dataCollection="book" onClick="openBook">
            <TableViewRow title="{title}" />
        </TableView>
    </Window>
</Alloy>

That works fine, I get a nice list of books.

The openBook function looks like this:

function openBook(e) {
    $.navgroup.open(Alloy.createController('singleBookView').getView());
}

The function should open a second controller / view, to display information about the single book that was clicked on the list:

<Alloy>
    <Model src="book" instance="true" id="selectedBook"/>
    <Window class="container">
            <Label text="{$.selectedBook.title}" />
            <Label text="{$.selectedBook.author}" />
        </View>
    </Window>
</Alloy>

Clicking on an item in the list will open the detail ('singleBookView') but does not populate any of the data. I'm not sure how to pass the model / model ID to the singleBookView controller so that it can populate the Labels with the right text.

Any advice very much appreciated as I'm stuck at this point!
