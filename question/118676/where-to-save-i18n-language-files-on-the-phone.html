Where to save i18n language files on the phone

The app I'm making is going to download the needed i18n translation files from a server, so I need to know where I'm supposed to save the files on the phone.

The guide says to save them in a "i18n/en" folder in the app directory, but what directory does this correspond to on the actual phone? (I'm running on the Android.)

I have tried to manually include a translation file (as per the guide), so that I could try to see in which folder it got put into on the phone. I've tried looking in the Titanium.Filesystem's applicationDataDirectory, applicationDirectory and the resources folders, but I can't find anything there.

I hope someone can help me here. Thanks in advance.

    According to iFile, the i18n files are compiled into the xxx.app folder (your resource folder before compilation)

    new directories are created, named "xxx.lproj" ex: "en.lproj". Inside, there is a Localisable.string file.

    Try to create a new folder with the same name and create this file.

    Hope it helps;

    Ivan.
