    you can do a twitter token, you just need to set the values properly. See the REST API documentation here
    http://cloud.appcelerator.com/docs/api/v1/social/loginexternal

    
    var _params = {
        type: "twitter",
        token: accessToken,
        id: twit_data.id_str
    };
