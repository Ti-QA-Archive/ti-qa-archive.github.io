  • I would image that only the owner of that message can delete the message.
    I think an Admin user can delete a message on behalf of someone else.
    Which makes sense to stop someone from just randomly deleting other peoples chat messages.

    chat_id : String
ID of the chat message to delete.
user_id :
User to delete the Chat object on behalf of. The user must be the sender of the chat message.
Current user must be an application admin to send a message on behalf of another user.
