Desktop: how to use json + mootools + pure.js (js template system)

My app just load json data from a remote server, can i use a js template engine ?

Because i make a mistaje will answering the question, here is the answer below ….

    <html>
      <head>
        <script type="text/javascript" src="js/mootools-1.2.4-core-nc.js">    </script>
        <script type="text/javascript" src="js/mootools-1.2.4.2-more.js">    </script>
        <script type="text/javascript" src="js/pure.js"></script>
      </head>
      <body>
        <div class="availableVersion version"></div>
        <script>
          var data = {availableVersion:[
            {version:'version 1'},
            {version:'version 2'}
          ]};

          $$('div.availableVersion').autoRender(data);

        </script>
      </body>
    </html>
