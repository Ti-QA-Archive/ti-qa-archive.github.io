TableView Gradient not working

I am trying to get a background gradient for my table view… below i have pasted my code:


var recentUpdatesTableView = Ti.UI.createTableView({
    data:[],
    borderWidth:1,
    height:130,
    width:280,
    backgroundGradient: {
        type: 'radial',
        startPoint: { x:'50%, y:'50%' },
        endPoint: { x: '50%', y: '50%' },
        colors: [ 'red', 'blue'],
        startRadius: '90%',
        endRadius: 0,
    backfillStart: true
    }
});

I cannot get the gradient to work… i would appreciate any help.

Thanks.

  • TableView doesn't appear to support backgroundGradient, but normal View does. So you can archive gradient style for TableView by wrap it into a View with backgroundGradient.

    Try this:

    var win = Ti.UI.createWindow({backgroundColor: 'white'});

var recentUpdatesTableView = Ti.UI.createTableView({
    data : [{title: 'Row 1'}, {title: 'Row 2'}, {title: 'Row 3'}],
    borderWidth : 1,
    height : 130,
    width : 280,
    backgroundColor: 'transparent'// This will do the trick :)
});
var view = Ti.UI.createView({
    borderWidth : recentUpdatesTableView.borderWidth,
    height : recentUpdatesTableView.height,
    width : recentUpdatesTableView.width,
    backgroundGradient : {
        type : 'radial',
        startPoint : {
            x : '50%',
            y : '50%'
        },
        endPoint : {
            x : '50%',
            y : '50%'
        },
        colors : ['red', 'blue'],
        startRadius : '90%',
        endRadius : 0,
        backfillStart : true
    }
});

view.add(recentUpdatesTableView);
win.add(view);
win.open();
