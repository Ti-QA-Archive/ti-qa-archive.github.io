  • Something like this:

    $.buttonName.addEventListener('click', function(e) {
     return Alloy.createController('viewToOpen').getView().open();
});

    • Ramesh,
      Do you have the button defined in markup? Repost your code using the forum markdown for code so we can see what you're doing. Also the two variations of listener code function the same. The steps are

      1. define an element in the view (xml)
      2. add an event listener to that element $.element.addEventListener in the controller using Alloy.createController(&#39;windowName&#39;).
      3. You can chain the getView() and the open() if you're not passing in parameters.

  • Hello Ramesh :

    Simple Example of alloy :

    index.xml

    <Alloy>
    <Window id="win" backgroundColor="white">
        <Label id="lbl" text="Main Window" color="red" top="5odp"/>
        <Button id="btn" title="Open Second Window" />
    </Window>
</Alloy>

    index.js

    $.btn.addEventListener('click',function(e){
    var testWin = Alloy.createController('test').getView();
    testWin.open();
});

$.win.open();

    test.xml

    <Alloy>
    <Window id="test" backgroundColor="white">
        <Label id="lbl" text="Second Window" color="red"/>
    </Window>
</Alloy>
