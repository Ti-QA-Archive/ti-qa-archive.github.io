    You need to distinguish two things, JavaScript as language defined by ECMAScript specification and environment in which JavaScript runs. DOM and Titanium are JS environments and you can't use DOM features in Titanium (except in Titanium WebView component that is DOM environment), while you can native things defined in ECMAScript spec.

    • HTML5 … well … same thing as with HTML4. :)

      Forget what you know about DOM, this is different. In browser environment there is HTML that is used for document presentation, CSS for document styling and DOM for document manipulation. There, you use JS to call DOM methods and manipulate HTML element's properties.

      Titanium is different environment, there is no HTML, there is no CSS and there is no DOM. What you have here is JS environment (native things describe in ECMAScript spec) and Titanium functions that are described in the documentation.

      In DOM, you can create a button with HTML tag:

      <button></button>

      and with DOM method:

      var button = document.createElement('button');

      Titanium is more like the latter one, to create a button you need to call "createButton" method of UI namespace:

      var button = Ti.UI.createButton();

      You can read more about Titanium environment here: http://zenborgium.blogspot.com/2011/12/about-titanium-framework.html
