How to Scale an Image ( imageView)

Hey,
i wanted to know how to scale an image keeping the aspect ratio:

var image = Titanium.UI.createImageView({
        image:'images/home_men.png',
        width:'auto',
        height:'auto',
        });

This image should be scaled by giving window.width fractum.

After reading this post:
Image not scaling

I figured, that it should work like this:

var image = Titanium.UI.createImageView({
        image:'someimage.png',
        width:mainWindow.width/5,
        height:'auto',
        });

Since it is told, that the image will scale fixed aspect with the SMALES size. In my case the image keeps the original height of the real image. Adding the canScale property doesn't change anything for me.

The real purpose of this question is, that I try to do this for button. But I can't even get it working for an image. If you know any differences for buttons please also let me know.

Thank you for your help!

  • I'm assuming that mainWindow is referencing a variable that you have declared and that it has a valid width object - otherwise that could be part of the problem.

    I would first try to use this example:

    var image = Titanium.UI.createImageView({
        image:'someimage.png',
        width: 300,  // <-- Just using an arbitrary number here
        height:'auto',
        });

    This should scale the image based on the width, maintaining the aspect ratio - if this is working for you you need to ensure that the variable that your using for width is a valid number (not null or undefined).
