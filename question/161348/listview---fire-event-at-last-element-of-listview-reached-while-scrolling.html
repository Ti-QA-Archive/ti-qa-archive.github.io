  • Hi

    If you are using Ti SDK 3.2 (if not read the release notes before upgrading) then you can use the new property setMarker and the assocated event to listen for marker.

    This essentially creates bookmarks of your ListrViewRows so you could add one to the last row and respond when triggerred - then if you add more data you add another one to the new last row.
