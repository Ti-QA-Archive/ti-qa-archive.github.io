How can i test my application on an android device?

I have a Nexus S. I installed the driver, and i'm sure it's ok because i typed "Adb logcat" on command line and i got my output text, so the device is not the problem.
Then i go on Titanium Studio, under "Run"… and i cannot find anything that refers to real devices. Only emulators!
If i press "Run" it opens the emulator.
Why it's all so complicated on Titanium Studio?

    In Project explorer tab, there should be three horizontal icons, one of them (the right one) is used for running the application and it should have several options (I think you need to click on the arrow in that icon), one of them is "Run on device".

    Hope you'll find it. :)
