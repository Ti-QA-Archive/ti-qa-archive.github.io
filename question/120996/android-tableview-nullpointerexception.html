  • I know it's been a while since you posted this but I just ran into it and I've had a similar issue with 1.7. For me I was able to pinpoint the issue to adding an image in the custom row. I posted another Q&A here with what I think is the same issue that never really got any responses.

    • So right after posting this I sort of stumbled on what looks like was causing my problem and it might solve your issue as well. I had my code setup just as you do in the section below.

      var newEventButton = Ti.UI.createButton(
        {
            backgroundImage : '../images/newEvent5.png',
            backgroundSelectedImage : '../images/newEvent5_pressed.png',
            height : 39,
            width : 120,
            right : 0,
            top : 0
    });

      Turns out removing the '..' in url path to the image solved my problem.
      Thus making:

      backgroundImage : &#39;..&#x2F;images&#x2F;newEvent5.png&#39; into

      backgroundImage : &#39;&#x2F;images&#x2F;newEvent5.png&#39;
