  • Looks like you are getting an error finding xcodebuild. Common error installing XCode 4.1 is that the appstore doesn't actually install it (even though it says installing). It just downloads it and creates an 'Install XCode' app in your Applications. Make sure you have run this to actually install XCode 4.1.
