ACS PHP CURL request returns empty?

Hello everyone.

My goal is to create a php page that will send push notifications to Titanium's ACS server. I have been using the REST API provided by Titanium via this documentation.

The problem I am having is that I have yet to be able to get a response from the request. Here is an example of my code below. Keep in mind this is just to get any response at all. Even an error message would be great

<?php
$curlObj    = curl_init("https://api.cloud.appcelerator.com/v1/users/login.json?key=");
$response = curl_exec($curlObj); 
curl_close($curlObj);
echo "Session Response: ".$response;

Now, when you go to that URL I am calling in your web browser, you get a response displayed on the page.
So what am I doing wrong here? No matter the URL or variable I have added, I cannot seem to get a response.

Thank you.
Chris

    This php code works for me. What's the error code of the curl request?

    $errno = curl_errno($curlObj);

    Read this value before closing the curl connection.
