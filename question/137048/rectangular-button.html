  • I would recommend against using an image view or something similar. At least on iOS, using an actual button gives you the benefit of feedback when the user taps the button, a different appearance for disabled state, etc.

    If you set the button style:&#39;none&#39; you will end up with a rectangular button that you can style however you want (Christian's advice of using a backgroundImage is also good).

    The following would create a rectangular button with black background, white text, and text that turns blue when clicked:

    Ti.UI.createButton({
    color: "#fff",
    height: 45,
    width: 120,
    backgroundColor: "#000",
    style:'none',
    font:{fontSize:12,fontFamily:'Helvetica Neue', fontWeight: 'bold'},
    title: "Click Me!",
    selectedColor: "#838B8B"
});
