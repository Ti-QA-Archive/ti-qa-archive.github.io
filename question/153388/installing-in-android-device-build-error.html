    • OMG it worked :) Thanks so much Nadeem for your help! The main context of the problem is that the tools data went outside the folders where Titanium looks into after I updated the Titanium SDK. I also did below. Once the platform tools are in the specified folder (in my case - "developer/sdk", I also did below to ensure that Titanium sees that folder correctly:

      1) tiapp.xml/overview
      2) configure/studio/platforms/Android
      3) at the "Android SDK Home: "point to the correct folder where all the sdk is located"
      …in my case its "/Users/myname/developer/sdk
      4) apply.. then ok..
      5) right click the app.js and install to android device..

      Works perfectly.. Have a nice day!!
