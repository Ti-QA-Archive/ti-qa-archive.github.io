Create a TabGroup that doesn't show a tab's window when the tab icon is click

I have a TabGroup with 4 tabs. For the first 3 tabs, I'd like the normal behavior (when the tab icon is clicked, the TabGroup displays the tab's window).

For the 4th tab, I'd like a special behavior: I'd like to be notified that the tab icon was clicked, but I want the TabGroup to remain in the same tab, and not display the tab that was clicked. In fact, the 4th tab's window contains nothing, so I don't event want to create a window for it (I know that a Tab must have at least one window in it, although I never want it displayed).

Any ideas?

  • Hi Omri,

    this should work:

    var tab4 = Titanium.UI.createTab({  
    icon: 'KS_nav_ui.png',
    title:'Tab 4'
});

tab4.addEventListener("focus", function(e){
    Ti.API.info("click tab4");
    tabGroup.setActiveTab(e.previousTab);
});
