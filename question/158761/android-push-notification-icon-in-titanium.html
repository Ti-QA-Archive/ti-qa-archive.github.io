Android push notification icon in Titanium

I created Android app that uses push notifications using Appcelerator's Urban Airship module.

https://github.com/appcelerator/titanium_modules/tree/master/urbanairship

It works, but I found a peoblem that on devices with hi-res screen (xhdpi, xxhdpi), push notification's icon is not displayed properly. It seems that only a small part of the appicon.png is visible in the notification's list.

How and where can I place the files correctly for the notification icon? Appcelerator's document does not explain that well.

http://docs.appcelerator.com/titanium/3.0/#!/guide/Icons_and_Splash_Screens

Thanks in advance.

  • Hi Sherlock,

    If you want to customize the Titanium Push Notification's icon, then you need to give the icon name in the payload string/hash in notify API like :

    {
  "title": "Example",
  "alert": "Sample alert",
  "icon": "little_star",
  "badge": 3,
  "sound": "door_bell.wav",
  "vibrate": true,
  "score": 51,
  "custom_field": {
    "headlines": "Appcelerator Cloud Services Rocks!"
  }
}

    The icon's file name without extension. The file should be located under app's "res/drawable" directory.

    For more details go through :http://docs.appcelerator.com/cloud/latest/#!/api/PushNotifications-method-notify

    And also: http://docs.appcelerator.com/cloud/latest/#!/guide/android
