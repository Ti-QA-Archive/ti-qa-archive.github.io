Getting a Picker value

iOS SDK 4.2, Titanium SDK 1.5.1

Is there any way to retrieve a Picker's value, without using change on it directly? I want to be able to hit "return" on a keyboard and get a Picker's value.

The only thing I can think of is to fireEvent the change event on the Picker and get the values. Is that it?

  • Nevermind, answered it myself:

    searchBar.addEventListener('return', function(e) {
     var val = e.source.value||"";
     alert("query:" + val 
+ " col1:" + searchPicker.getSelectedRow(0).custom_item 
+ " col2:" + searchPicker.getSelectedRow(1).custom_item);
});
