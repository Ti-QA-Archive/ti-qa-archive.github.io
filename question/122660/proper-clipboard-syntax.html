Proper Clipboard syntax

I'm getting errors when trying to read an image from the Clipboard and I'm hoping you can help. What is the proper syntax for getData when specifying the mime type? I keep getting this error:

[ERROR] Invalid type passed to function. expected: NSString or nil, was: __NSArrayM in -[TiUIClipboardProxy hasData:] (TiUIClipboardProxy.m:168)

I'm using the Ti.UI.Clipboard.hasData and Ti.UI.Clipboard.getData methods. What is the proper way to request and set the mime type for something like an image?

Thanks a million in advance.

  • You should be able to do this if u set the mime type for the image

    Ti.UI.Clipboard.getData('image/png');

    Though I don't know what mimetype they use to copy an image to the clipboard so you might try 

    Ti.UI.Clipboard.getData('image/jpeg');

    as well.
