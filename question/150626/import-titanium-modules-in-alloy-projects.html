Import Titanium Modules in Alloy Projects

Hi everyone, i want to know how to add modules in titanium alloy (mvc) projects. I want to use Pushwoosh Android Module this module for android push notification with pushwoosh. This is a sample of Titanium Traditional Development. I want to use the module in Alloy. I tried by putting the module like ~/Library/Application Support/Titanium/modules/android/com.arellomobie.push and also tried like ~/Library/Application Support/Titanium/com.arellomobie.push but didn't work. The console log says titanium module id not found com.arellomobie.push I'm sure i have added the module reference to xml file's module tag with version and name of module. Even no luck. Any one know how to done with this or how to add module inside alloy project. Please help :)

    hi,

    if you have problem in your path then you can also include your module in your main project folder as stated in this project codeso put it there and include it from tiapp.xml.

    put the module in below path..

    <project folder>/modules/android/

    and also remember that if there is a problem in code then may be you can get success to compile it but you will got the same error at run time so first try to create a sample project to test this module.
