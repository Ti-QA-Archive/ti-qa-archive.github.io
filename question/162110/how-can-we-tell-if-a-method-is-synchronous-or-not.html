I was trying to use Ti.Media.showCamera(), and then run a function upon completion. However, it appears that the method is asynchronous, but this is not documented in the docs. Is there a way to tell which functions are asynchronous without trying to figure it out myself?

    permalink

    • Titanium.Media.showCamera({
    success: function(event){
        // stuff to do if success
    },

    error: function(error){
        // stuff to do it it broke
    },

    cancel:function(e){
        // stuff to do if abandoned
    }

});
