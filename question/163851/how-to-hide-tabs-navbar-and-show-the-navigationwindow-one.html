How to hide Tab's navBar and show the navigationWindow one.

Hello.

I would to ask a little question. I have a NavigationWindow inside a Tab which is inside a TabGroup.
I want to show the navBar of the NavigationWindow and hide the Tabs one but I dont know how. I tried with <TabGroup navBarHidden="True"> but nothing happends. I know i can hide the Window one but I need it. I will be very grateful for your answer.

<Alloy>
    <TabGroup navBarHidden="True">
        <Tab title="I want to hide this navBar">
            <NavigationWindow id="nw_main">
                <Window title="I want to show this navBar">
                    <TextField id="tempo"></TextField>
                        <Button></Button>
                </Window>
            </NavigationWindow>
        </Tab>
    </TabGroup>
</Alloy>

    Hi

    This is clearly an iOS app, so remember that the tabGroup actually contains a navigation controller of its own. There is no need to try and add your own - which is why you are getting this issue.

    <Alloy>
    <TabGroup navBarHidden="True">
        <Tab title="I want to hide this navBar">
            <Window title="I want to show this navBar">
                <TextField id="tempo"></TextField>
                <Button></Button>
            </Window>
        </Tab>
    </TabGroup>
</Alloy>

    • A quick and dirty example - do not judge me on the code ;-)

      xml

      <Alloy>
    <TabGroup id="nav" navBarHidden="True">
        <Tab id="tab1" title="Tab 1">
            <Window title="Window 1" backgroundColor="#fff">
                <TextField id="tempo"></TextField>
                <Button id="btn" color="#000" title="Press Me"></Button>
            </Window>
        </Tab>
    </TabGroup>
</Alloy>

      controller

      $.btn.addEventListener('click', function (e) {
    var win = Ti.UI.createWindow({
        backgroundColor: '#fff',
        title: 'New Window'
    });
    $.tab1.open(win);
});
$.nav.open();
