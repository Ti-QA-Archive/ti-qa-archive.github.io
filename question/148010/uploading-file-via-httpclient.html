Uploading file via HTTPClient?

I can't figure out how to accomplish this. I'm adding the file (Ti.Media.Blob) along with other plain text fields as parameters in a POST to my api. However, the POST is empty when my API receives it. Code below…

api.open("POST","http://mysite.com/api/storedoc");

var params = {
    token: Ti.App.Properties.getString('token'),
    refNumber: refNumber,
    doc: photo, // this is the Ti.Media.Blob
    docType: photoType
};

api.setRequestHeader('Content-Type', 'multipart/form-data');
api.send(params);

This same method with Content-Type "application/x-www-form-urlencoded" works fine for posting plain text parameters, but doesn't work with files.. and multipart/form-data isn't working either. Is the problem in the doc-type or is it not possible to post a file along with plain text fields in a single POST, or is there something else wrong?

Thanks for any help!
