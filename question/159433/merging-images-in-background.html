Merging images in background

Hi,
I'm trying to merge two images into one but I don't want to see the part images. My way was (and this was mentioned in another thread):

var v = Ti.UI.createView({
    left : 0, top : 0, touchEnabled : false
});
var i1 = Ti.UI.createImageView({
    image : $.img1.image, left : 0, top : 0, touchEnabled : false
});
var i2 = Ti.UI.createImageView({
    image : blob.toImage(), left : 0, top : 0, touchEnabled : false
});
v.add(i1);
v.add(i2);
newBlob = v.toImage();

but this only works if v is visible and added to stage. There is blob.append() but it doesn't seem to work they way I want it to. I just want to add multiple image parts into one and asign it to one imageview at the end.

Is there a better way to do this action in background? iOS and Android!
