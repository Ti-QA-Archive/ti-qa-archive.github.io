  • you can do this with creating a new window and add the view with the tabbar height or navbar height like this…

    var customWinBar = Ti.UI.createWindow({
    height : 68,
    bottom : 0,
    zIndex : 444
});

var view = Ti.UI.createView({
    top : 0,
    left : 0,
    width : 320    
});

customWinBar.add(view);

customWinBar.open();

    if you want view above navBar set the window height as 44 and top as 0
