Styling buttons that are not on the toolbar

I've had a difficult time finding info about how to style buttons that are not on the iPhone's toolbar. I haven't found much info about it on the AQ either.

When I create a without any style I get a white button with rounded corners and blue text. If I change the style I get no background and a plus icon next to the text. That's about all I get with the style property. I can't even get that glossy 3D style on the button unless it's on a toolbar. Is that the expected behavior?

If I try to change the background to the original style I only see the background color behind the rounded corners. I can't seem to manipulate buttons much at all. Is it possible to manipulate them in a CSS like way, such as adjusting the border radius, adding gradients, etc? Or do I have to use images?

    • Try the code below, tested in ios sim 4.3 with mobile sdk 1.7.2:

      var win = Ti.UI.createWindow();

var button = Ti.UI.createButton({
    title:'press me',
    width:100,
    height:50,
    top: 60,
    borderRadius: 8,
    borderWidth: 2,
    borderColor: '#fff',
    backgroundImage: 'none',
    backgroundGradient: {
        type: 'linear',
        colors: [
            '#aaa',
            '#00f'
        ]    
    }
});

win.add(button);
win.open();

      it will generate the window seen below, complete with styled button. Note that to use a backgroundGradient on the button you need to set backgroundImage: &#39;none&#39;.
