    What you are after is the Database Reference Guide, which can be downloaded from the API Reference Guides page. Take a look at the sample programs, and the "example using the JavaScript module design pattern", which is based on some nice anonymous closures

    You'll want to use Titanium.Database.install to initially install your db from a pre-existing source. Note that this only happens the first time the app is loaded! (if you change your pre-existing db and want to test it, you'll need to remove it from the device first.)
