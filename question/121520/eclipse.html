    Curtis,

    You should be able to import the project into Titanium Studio. Go to File &gt; import ... and choose the project and type that you want to import. Of course I would suggest backing up your project before attempting this, however, I believe the import will work.

If you did import your project, then you need to look at the "App Explorer" window on the left hand side and from the drop down there, you need to choose the project that you imported, not the project that you created new. Once you choose your imported project, then you can try and run again.
