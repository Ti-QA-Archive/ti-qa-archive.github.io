    Incase of android you dnt only have one splash screen, if you will go to app/assets/android/images there you will find different folder for different densities and screen sizes, so android first try to take the splash screen according to those folders only , if it does not find it there then it takes the one at the root folder. So for your thing to get done please replace all those images too.
