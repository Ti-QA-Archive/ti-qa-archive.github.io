Hide controls of a video

Hi,

I'm trying to display a fullscreen video when a user launches my app.

I don't want to display controls as the video last 5 seconds and then disappears.

I'm using this code that won't work:

// opening video
var videoWin = Ti.UI.createWindow({
    backgroundColor: '#000',
    navBarHidden: true
});

var video = Ti.Media.createVideoPlayer({
    url: 'media/intro.mov',
    fullscreen: true,
    mediaControlStyle: Ti.Media.VIDEO_CONTROL_NONE
});

videoWin.add(video);
videoWin.open({modal: true});

The window opens, the video shows up and play but the controls won't hide whatever option I use.

Is there something I missed or is it a bug?
Could I do that in some other way?

Thanks a lot.

    The docs seem to be wrong about the property name. The name of the property is movieControlStyle, not mediaControlStyle. (As you found, movieControlMode also works but is deprecated.)
