Titanium.App.Properties in 3.4

Hola,
tengo definida una Titanium.App.Properties en tiapp.xml.

<property name="litrosDeposito" type="string">42</property>

Desde la app quiero modificarla:

Ti.App.Properties.setString('litrosDeposito', '58');

Las pruebas que he efectuado me dan error y me comenta que no puede sobreescribir:

Property "screenname_preference" already exist and cannot be overwritten

Alguien me puede ayudar y aclararme si esto que quiero hacer se puede hacer?

Muchas gracias

    Hi Paco,

    don't define your properties in tiapp.xml if you want them to be writable.

    If you want to have some Properties with default options, write something like in your app.js or if you are using Alloy in your alloy.js file

    if (!Ti.App.Properties.hasProperty("litrosDeposito")) { // check if property doesn't exist 
    Ti.App.Properties.setString("litrosDeposito", "42"); // if doesn't exist write the default value
}

    Hope this helps
