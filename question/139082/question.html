    Yes, as long as you have access to their API's. An API is an Application Programming Interface. It's the interface that Facebook, Instagram, etc. provides to developers that gives access to their data.

    For Facebook:

    Facebook API docs

    Appcelerator Facebook Docs

    For Instagram:

    Instagram API Docs

    You can google search "API", "weather API", "Instagram API", or "TheProgramYouLike API" to find out if the program, app, or service that you like has an API.

    • You're not going to be able to find a video for every single thing that you want to accomplish. What you have to learn is, is how to apply what is in the video and your knowledge (or lack thereof) of javascript to attack the exact thing you want to accomplish. Another thing I did was to just go through the Q&A and see what others are doing. Start with small code samples, then build on them. I was very weak with javascript when I started learning Titanium Mobile. So, I had to learn two things at once, both javascript and the API for Titanium Mobile. It's been a year now, and although it didn't happen as fast as I would have liked, my understanding is very strong now.

      As soon as you realize that you won't be able to write the app that you want to write in just a few hours or days, the better off you will be. One last thing…..you may want to look in to a book recently written by a community member. It's more of a cookbook style of book while you learn.

      Appcelerator Titanium Smartphone App Development Cookbook

      The only issue with that book is it may not use some of the newer principles introduced in the last few months, commonJS being one of them.
