Alloy or Classic for dynamic objects

I want to create in alloy some interface objects and after add the on view/ window. This is how i do it on classic:

    // ************************************************************************
    // create the labels of the ui
    // ************************************************************************
    function createLabels(textOfLabel, topPosition) {
        Label = Titanium.UI.createLabel({
            text : textOfLabel,
            top : topPosition + "%",
            textAlign : "center",
            color : Titanium.API.titleFontColorPurple,
            font : {
                fontSize : Titanium.API.smallTitleFontSize,
                fontFamily : Titanium.API.titlesFont,
            },
        });
        //push them on array stack
        Labels.push(Label);
    }

    // ************************************************************************
    // add the labels to the view
    // ************************************************************************
    function addLabels() {
        for (var i = 0; i < Labels.length; i++) {
            viewProfile.add(Labels[i]);
        }
    }

How i can do something like that in Alloy ? Thanks !

        </View>
    </Window>
