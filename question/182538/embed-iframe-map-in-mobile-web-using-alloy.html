Embed iframe map in Mobile Web using Alloy?

I would like to embed this iframe so that I can use maps in Mobile Web. Does anyone know how and where I may do so?
~<iframe width="600" height="450" frameborder="0" style="border:0" src="https://www.google.com/maps/embed/v1/place?key=API_KEY &q=Space+Needle,Seattle+WA"> </iframe>~

  • Finally figured it out. Using a WebView in a Mobile Web project was not immediately sensible to me but seems the answer.

    <WebView id="mapview" platform="mobileweb" url="https://www.google.com/maps/embed/v1/place?key=XXXXXXXXXXXXXX&q=Space+Needle,Seattle+WA" />
