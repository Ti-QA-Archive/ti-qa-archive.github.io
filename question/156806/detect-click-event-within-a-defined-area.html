Detect Click Event Within a Defined Area

I have an image with different regions on it highlighted. I want to detect if a click event is within one of those regions. It will always be a single image (unless there is some way to place transparent images on top of each other and only detect the click on an image where that x,y is not a transparent part of the image!).
Basically I'd like to do something like this:

myImage.addEventListener("click", function(e) {
  var area1 = [{0, 0}, {10, 0}, {12, 5}, {12, 20}, {0, 20}];
  if (area1.contains(e.x, e.y)) {
    dosomething();
  }
});

area1 would be an array of x, y coordinates, with the understanding that the last point given will automatically connect to the first point to create a shape.

What's the best way to do this in Titanium?
