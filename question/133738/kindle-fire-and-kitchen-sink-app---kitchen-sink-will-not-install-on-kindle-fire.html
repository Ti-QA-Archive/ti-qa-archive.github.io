Kindle Fire and Kitchen Sink App - Kitchen Sink will not install on Kindle Fire?

We do a lot of Kindle Fire development. We had some strange issues with apps that worked great on Android Phones and tablets. When we tested on the Kindle Fire the apps had errors. So the next logical step was to install the Kitchen Sink and review the functionality.

We discovered the Kitchen Sink App taken from github as of 3/12/2012 would not install at all in the Kindle Fire.

Are there project settings or a specific environment configuration best practice we should be using for Kindle Fire projects?

  • Kindle FiIre/Mobile Application
  • SDK 1.8.2
  • Kindle Fire
  • Windows 7
  • Titanium Studio, build: 1.0.9.201202141208

  • Kindle Fire doesn't have the standard MapKit and this causes some troubles and stops compiling the app properly. Don't load any module which refers to Maps in order to compile and install the app.
