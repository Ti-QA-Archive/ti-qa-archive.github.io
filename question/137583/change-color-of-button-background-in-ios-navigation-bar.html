Change color of button background in iOS navigation bar?

I have a button I'd like to set as the right nav button in my main buttonbar, but no matter what I do it only takes on the color of the buttonbar itself. Is there no way to override this?


var btnSimpleBegin = Ti.UI.createButton({
    title : 'Estimate',
    height : 50,
    width : 180
});

winSimpleIntro.rightNavButton = btnSimpleBegin;

sdk 2.0.1ga2, ios 5.1

  • hi,

    you can not set backgroundColor of button but you can use buttonBar with backgroundColor

    try this

    var btnSimpleBegin = Ti.UI.createButtonBar({
    labels : ['Estimate'],
    // height : 50,
    // width : 180,
    backgroundColor : 'red'
});

win.rightNavButton = btnSimpleBegin;

  • There's a trick I've used to do this. Create a plain-color image and set as image for the navigation Bar. Then set the BarColor property to the color desired for the button. Voilá!

    win.barColor = "#000";
win.barImage = '/images/bgNavBar.png'; //image with a plain color
