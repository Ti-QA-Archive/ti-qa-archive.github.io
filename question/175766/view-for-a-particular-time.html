View for a particular time

hello,
i want to add a new view{V} in a window {W} and i did it using {W.add(V)} but my originally requirement is that this view {V} should be added for sometime say(for 5 sec), is it possible to do that.if not then please tell me other way to do it.

— asked July 4th 2014 by rahul kag

    var view = Ti.UI.createView();
var win = Ti.UI.createWindow();
win.add(view);
setTimeout(function(){
    win.remove(view);
},5000);

    • you can use a wrapper:

      var view1 = Ti.UI.createView();
var view2 = Ti.UI.createView();
var wrapper= Ti.UI.createView();
var win = Ti.UI.createWindow();
win.add(wrapper);
wrapper.add(view1);
setTimeout(function(){
    wrapper.remove(view1);
    wrapper.add(view2);
},5000);
