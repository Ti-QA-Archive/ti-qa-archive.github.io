  • I have used NativeScript for testing and comparison purpose (I was only using Titanium so far) and simply put, it is A-MA-ZING.

    NativeScript allowed me to directly use native iOS and Android libraries that where not even documented in their documentation in a first place. How beautiful is that. I did the same thing with some random library that I have simply downloaded from Github.

    With NativeScript, it is possible to be using cutting edge platform specific technologies available in the native SDKs without waiting for the NativeScript folks to implement a JavaScript version of it. It lets you access any public methods, functions, properties, constants.

    If you see that you are using some of them often, you can create a multi-platform javascript plugins for it and use it in a unified way.

    I invite anyone to use it. It's great and it's free and open source.
