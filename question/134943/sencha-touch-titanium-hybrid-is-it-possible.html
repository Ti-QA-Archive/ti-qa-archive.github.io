Sencha Touch Titanium hybrid, is it possible?

I realize that one could easily build a sencha touch app and run it from a Titanium webview. What I am uncertain about is whether it is possible to call out to the Titanium api from javascript running within a webview.

Lets say I have a sqlite database in the app and I want to use the titanium database api to load data from sqlite and have it returned to the webview. Is it possible to make an ajax style call from the webview to a local javascript resource that utilizes the titanium db api to query the table and return the data back to my webview?

Is it possible? If so any ideas or better yet code examples on how to do it?
