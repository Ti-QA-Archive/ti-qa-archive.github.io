Javascript OR (||) operator not working

Am I missing something, or is the OR operator inside an if() statement not working in appcelerator?

I have the following check in my script: (I have the string's "null" and "1" in my database, as I have had troubles reading empty fields with Titanium. "null" donates an empty field, "1" donates that I must do something else other than print out a text string.

if((win.events !== "null") || (win.events !== "1")) {
    hospitalityView.add(hospitalityEventsRow);
    Ti.API.info(hospitalityEventsDescr.text);            
}

If I do this, then Titanium will add the word "null" or "1" whichever it is in hospitalityEventsDescr.text. Obviously this is wrong. I am having to construct things like this; which is messy:

if(win.events == "null") {
    // Do nothing    
    Ti.API.info('Event: null');
} else if(win.events == "1") {
    // Add icon
    Ti.API.info('Event Icon');
} else {
        hospitalityView.add(hospitalityEventsRow);
        Ti.API.info(hospitalityEventsDescr.text);
}

Can anyone shed any light on this???
