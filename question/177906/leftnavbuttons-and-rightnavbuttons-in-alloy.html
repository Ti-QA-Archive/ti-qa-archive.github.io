LeftNavButtons and RightNavButtons in Alloy

Hi,

Since SDK 3.3, Titanium has new keywords LeftNavButtons and RightNavButtons. If I use them in Alloy, I get the error 'invalid method (createRightNavButtons) passed to UIModule…'. This is the Alloy xml I used:

<Alloy>
    <Window>
        <RightNavButtons platform=ios>
            <Button title="s" />
            <Button title="i"  />
        </RightNavButtons>
        <ImageView image="/logo.png"/>
    </Window>
</Alloy>

The Window is part of a NavigationWindow. Is this a know bug in SDK 3.3.0? Did I miss(use) something??

Thanks.

    Hi guys,

    For adding a navigation button you have to add a Navigation Window first like this sample code.

    <Alloy>
    <NavigationWindow>
        <Window>
            <RightNavButtons platform=ios>
                <Button title="s" />
                <Button title="i"  />
            </RightNavButtons>
            <ImageView image="/logo.png"/>
        </Window>
    </NavigationWindow>
</Alloy>

    If you have any question about this issue, then let me know.

    Thanks.
