Button click event issue,

Hello everyone!
I am having a issue on Click event on button within a view(Which has also got a click event)

        // View creation.
        var pView = Ti.UI.createView({});

        // Click even for a view.
        pView.addEventListener('click', function(e) {
            var controller = Alloy.createController("p2");
            if(Titanium.Platform.name == 'iPhone OS'){
                Alloy.navigationWindow.openWindow(controller.getView());
            }
            else if(Titanium.Platform.name == 'android'){
                controller.getView().open();
             }
         });

        // Button creation.
        var viewC = Ti.UI.createButton({
            title : "ClickMe"
        });    

        //Click event for a button.
        viewC.addEventListener('click', function(e) {
            alert("ClickMe got cliked!!");
        });

        pView.add(viewC);

When I click anywhere on View it takes me to next controller(i.e.,p2 Its working absolutely fine),
But when I click on a ClickMe button the alert dialogBox doesn't display at all, and it navigates to next controller(i.e., p2)

Is there any way to give some sort of priority to button so that click event on button should not be treated as even of View?

TIA

Maruti.

    Hi Maruti,

    in titanum events a bubbling up by default, this means any event of child of a view also fires as an event of the parent view. To solve your problem, set the "bubbleParent" property of "viewC" to false.

    this should fix your problem:

    var viewC = Ti.UI.createButton({
            title : "ClickMe",
        bubbleParent:false
        });

  • Hello Maruti,

    try this one using Custom Property:

     var pView = Ti.UI.createView({
    object : "View"
});

// Click even for a view.
pView.addEventListener('click', function(e) {
    if (e.source.object == "Button")
        return;
    var controller = Alloy.createController("p2");
    if (Titanium.Platform.name == 'iPhone OS') {
        Alloy.navigationWindow.openWindow(controller.getView());
    } else if (Titanium.Platform.name == 'android') {
        controller.getView().open();
    }
});

// Button creation.
var viewC = Ti.UI.createButton({
    title : "ClickMe",
    object : 'Button'
});

//Click event for a button.
viewC.addEventListener('click', function(e) {
    alert("ClickMe got cliked!!");
});

pView.add(viewC);
