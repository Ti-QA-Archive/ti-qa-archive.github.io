Do stuff on window close from within the window?

I have a lightweight window that is opened from within a tabgroup (not one of the tabs, but a window that is opened from one of the tabs). This results in the window being closable via the navbar. From within that window I would like to intercept the clicking of that close button, and display a dialog prompting the user whether or not they would like to abandon their changes. However during the blur and close events both I'm not able to access the var that I setup to determine whether or not changes have actually been made. A simplified version of my code is below:

var win = Titanium.UI.currentWindow;

var notesHaveChanged = true;

win.addEventListener('close',function(e)
{
if(notesHaveChanged) alert("You have unsaved changes.");
});

Is there a "right" way to do this? notesHaveChanged appears to always be 0 within the context of this event.

    Try accessing the source property of the incoming event parameter:

    e.source.notesHaveChanged;
