    Are you sure you map module is included in your project as it is asked by Titanium :

    • Unpack the module to your Titanium SDK home directory, for example, /Users/…/Library/Application Support/Titanium, which make the module available to all your projects.
    • Alternatively, you can copy the ZIP file to your Resources folder, which makes the module available to a single project.

    Have you included your module in tiapp.xml ?

     <modules>
    <!-- Add this line to your modules section -->
    <module platform="android">ti.map</module>
</modules>
