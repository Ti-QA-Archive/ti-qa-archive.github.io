How do I specify credentials when loading a remote image in an ImageView?

I'm building a TableView with a list of records, and each record has an image associated with it. The list of records is loaded via a web service which requires basic authentication. I'm able to load the list of records using an HTTPClient and specifying credentials with no problem; however, using an ImageView to load the remote image by pointing it to a URL is not working because I haven't found a way to provide credentials to the ImageView for when it makes the request for the image. What's the best way to load remote images into an ImageView when specifying credentials is required?

    • var image = Ti.UI.createImageView();
require('modules/secretimage').get(url,function(_img){image.setImage(_img);});

      and in secretimage.js:

      
exports.get = function(_url,_callback) {
     var xhr = Titanium.Network.createHTTPClient({
        user :user,
        password :password,
        onload : function() {
            _callback(this.responseData)
        };
    });
    xhr.open('GET',_url);
    xhr.send(); // Sending NO data
}
