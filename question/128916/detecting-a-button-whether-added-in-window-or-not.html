Detecting a button whether added in window or not

I am new to appcelerator . How to check whether button is added into a window or not.

  • (Almost) every UI component in Titanium has "children" property which contains array of references of components added to that particular component. So, that array is the key of detection, I'll show you trivial example how to do that:

    var win = Ti.UI.createWindow();

var button = Ti.UI.createButton();

win.add(button);

if (win.children[0] === button) {
    Ti.API.info('Button was added to the window');
}
else {
    Ti.API.info('Button was not added to the window');
}

    Now you can build more complex things.
