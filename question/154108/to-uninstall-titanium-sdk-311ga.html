    Hi prashant,

    Seems build tools got moved to another directory with the latest Android SDK update. Created symlinks to aapt and dx in /Applications/Android-sdk/platform-tools:

    simply copied all the files from build-tool/17.0.0/ to platform-tools and try to run the project

    please refer the below link for more help:
    https://developer.appcelerator.com/question/152497/titanium-sdk-310-error-typeerror-argument-of-type-nonetype-is-not-iterable-on-building-android-app
