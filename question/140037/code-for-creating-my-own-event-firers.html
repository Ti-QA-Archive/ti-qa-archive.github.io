  • You can do custom events like this : 

    
function myHandlerFunction(e){
    Ti.API.info('the event was fired! : '+JSON.stringify(e))
}

Ti.App.addEventListener('myCustomEvent', myHandlerFunction);

// ...

Ti.App.removeEventListener('myCustomEvent', myHandlerFunction);

    See here : http://docs.appcelerator.com/titanium/2.1/index.html#!/api/Titanium.App
