  • Hi

    The most common way is getElementsByTagName, performance is generally not a problem as you should not be dealing with tens of thousands of entries.

    XML in my opinion never lived up to the promise about it making it easier to connect data up. If you have the choice opt for JSON much easier to navigate and faster.
