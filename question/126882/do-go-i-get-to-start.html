  • Related to getting started , this is your first stop:
    http://wiki.appcelerator.org/display/guides/Home

    After this you go to github and download the Kitchensink to see what the app can do and how to do it.
    https://github.com/appcelerator/titanium_mobile/tree/master/demos/KitchenSink

    You will need a Mac to develop for iOs and you will have to be registered as an iOs developer (99$ on apple site) to be able to test on device and upload to appstore.
    You will have to download and install the iOs SDK and the Android SDK but this aspects should be covered in the docs I gave you.

    You will need to know/learn Javascript.

    The last version is always the best :)

    Hope it helps.
