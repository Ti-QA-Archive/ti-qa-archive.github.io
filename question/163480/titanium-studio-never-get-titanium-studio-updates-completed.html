  • finally I get the titanium studio work.
    this is what I did:

    1, install jdk
    2, config path/classpath/java_home
    3, clear npm cache
    4, install node
    5, install titanium from npm command: npm install titanium -g, I failed many times, it stuck after downloading some tgz file, close cmd windows, then try this command again, try several times you'll get all files, then last you'll get it installed.
    6, install alloy
    7, configure android sdk/ndk from preference/platforms/android window, now it recognize sdk folder, if not, please click install android sdk button in the bottom right.

    8, another way I succeed is to configure Eclipse by 'install new software' from help menu, I followed this topic:
    http://rensanning.iteye.com/blog/1413629

    it's really good to write js and generate pure native mobile app, I'm so happy to know this tool, however it took me several days, it worth, I'll learn how to run/debug project in this great tool.
