    Titanium 3.0.2.GA will only work with Node.js 0.8.x. It will not work on Node.js 0.10.x due to a couple bugs (which have been fixed for the upcoming 3.1.0 release).

    Your options are:

    1. Downgrade Node.js to 0.8.23 (http://nodejs.org/dist/v0.8.23/)

    2. Test drive the Titanium SDK 3.1.0 Release Candidate as described here: http://developer.appcelerator.com/blog/2013/04/titanium-sdkstudio-3-1-0-release-candidate-now-available.html. The RC is still going through testing, however the official 3.1.0 release is just around the corner. If you plan to be in development mode for the next couple weeks, I'd go this route.
