Display Base64 image data on mobileweb

My app stores images as base64 data, and decoding them and displaying them on iOS is straightforward:

var blob = Ti.Utils.base64decode(base64String);
var image = Ti.UI.createImageView({image:blob});

However, this doesn't work on mobileweb. I have scoured this forum and other places for an answer, but can't find a way to do it in mobileweb, just talk about how images are already base64 encoded in the browser.

So: how can I draw an image from a base64 string in mobileweb?

  • You need to use a data-uri in the img src tags to use Base4-encoded images on the web. (in HTML or CSS)

    See the syntax here, depending on the original file-type : https://css-tricks.com/data-uris/
