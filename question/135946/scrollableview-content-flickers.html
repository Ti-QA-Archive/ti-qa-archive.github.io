scrollableview content flickers

In my app, I have a section with a scrollable that has about 200 views inside it.

When I scroll, the the content briefly loads the page BEFORE that one, then flickers back to the one I'm looking.

For example, say I have 3 views, with a label that has a 1,2,3, respectively.

I start on view 3 (3 on the label)
I scroll to the left (to view 2, 2 on label)
It flickers with the content of view 1(1 on label), then resolves back to view 2 (label 2).

The code isnt anything complex, just an array of views with some labels, and then my scrollable is set to use that array as the view.

  • 200 views within a scroll view is always going to be a performance nightmare. You might be better off creating a "virtual" list of views where you actually only have, say three views and update the most lefthand/righthand with a new view when you swipe one view closer to it.
