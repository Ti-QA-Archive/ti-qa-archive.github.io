iOS map annotation not showing up

Hi, I tried to use this code from the documentation to set an annotation on an iOS map:

var mountainView = Map.createAnnotation({
    latitude:37.390749,
    longitude:-122.081651,
    title:"Appcelerator Headquarters",
    subtitle:'Mountain View, CA',
    pincolor:Map.ANNOTATION_RED,
    myid:1 // Custom property to uniquely identify this annotation.
});

var mapview = Map.createView({
    mapType: Map.NORMAL_TYPE,
    region: {latitude:33.74511, longitude:-84.38993,
            latitudeDelta:0.01, longitudeDelta:0.01},
    animate:true,
    regionFit:true,
    userLocation:true,
    annotations:[mountainView]
});

mapView.add(mapview);

The result is I can see the map but not the annotation. I tried this code when I noticed my code was not properly working.

Any suggestion?

  • Hi Ree,

    Thanks for your concern.

    Please check my test code below and see if it works for you. It's working as expected for me.

    var Map = require('ti.map');
var win = Ti.UI.createWindow();

var annotations = [
    Map.createAnnotation({
        latitude: 37.389569,
        longitude: -122.050212,
        title: 'Appcelerator HQ',
        subtitle: 'Mountain View, CA',
        animate: true,
        pincolor: Map.ANNOTATION_GREEN,
    }),
];
var mapview = Map.createView({
    mapType: Map.STANDARD_TYPE,
    region: {
        latitude:37.389569,
        longitude:-122.050212,
        latitudeDelta:.05,
        longitudeDelta:.05
    },
    animate:true,
    regionFit:true,
    userLocation:false,
    annotations: annotations
});

win.add(mapview);
win.open();

    Hope it'll help you to solve your problem.

    Thanks
