    Assuming you are using similar mark up to this;

    <Alloy>

        <NavigationGroup id="navigation" platform="ios">
            <Window id="win1Register" title="Register">
                <Label>TESTING BTN</Label>
                <LeftNavButton>
                    <Button id="lbtn" onClick="homebtn" title="home"/>
                </LeftNavButton>
            </Window>
         </NavigationGroup>
     </Window>
</Alloy>

    Have you tried setting the following in the tss?

    "#lbtn": {
    backgroundImage: "someimage.png"
}

      win.close();
