How do I replace double slash with a triple slash?

Hi,

I have a string that contains a double slash (//) - how do I replace this with a triple slash?

var myStringWithThreeSlashes = myStringWith2Slashes.replace(/ // /g, "///")

The above doesn't work.

Can someone send me some documentation on how to change that?

Thanks in advance,
EB
