table view inside scroll view?

I have a window that currently has a scroll view showing some user info a photo and description (think instagram). Now I want to add all comment that have been made about that photo. I have put the comments into a table view. My problem is I can get my table view to show on top of my scroll view but not within the scroll view. My goal is to add the comments as the last item on my page, visible when you scroll past the other info (which has variable height).

Can this be done? If so how? If not, is there another solution to what I am looking to do?

Thanks

  • Having a scrolling table within a view which can itself scroll can be problematic or at least confusing for the user. So the key is to mark as table as scrollable:false then set the tableview height to enough to show the entire thing within the scroll view. Then the user only has to scroll one UI object.
