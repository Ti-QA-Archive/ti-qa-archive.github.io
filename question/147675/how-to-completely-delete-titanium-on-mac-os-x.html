How to completely delete Titanium on Mac OS X?

I want to delete everything related to Titanium so I can reinstall it. This is because I cannot install Titanium CLI 3.0.23, I keep getting this:

Error encountered during update
Failed to install titanium. Please see the Studio log for more information.

    • You might also want to delete the ~/.npm directory before installing Titanium. We've seen where the npm cache directory can act up, especially if the file permissions are messed up.

    • Thad, that message means titanium and alloy were probably installed globally, so add the -g flag to the uninstall command:

      npm uninstall titanium -g
npm uninstall alloy -g

      — commented November 26th 2013 by Alejandro Thorne

  • This also leaves the node modules for titanium and alloy behind ( in /usr/local/lib/node_modules ). To get rid of them from terminal type:

    npm uninstall titanium
npm uninstall alloy
