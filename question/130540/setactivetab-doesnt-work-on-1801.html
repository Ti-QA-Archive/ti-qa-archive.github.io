    — answered January 11th 2012 by Michael Pauley
    permalink
    1 Comment

    • There are 2 tabgroups, in each 5 tabs.

      Tabgroup1 – tab1 to tab5

      Tabgroup2 – tab6 to tab10

      Each time tab 5 or 6 gets focus:

      tab5.addEventListener('focus', function(e) {

        tabGroup2.open({
            transition:Titanium.UI.iPhone.AnimationStyle.FLIP_FROM_LEFT
        });

        tabGroup2.setActiveTab(1);

        tabGroup.close();
    });

          tab6.addEventListener('focus', function(e) {

        tabGroup.open({
            transition:Titanium.UI.iPhone.AnimationStyle.FLIP_FROM_RIGHT
        });

        tabGroup.setActiveTab(2);

        tabGroup2.close();
    });

      On SDK 1.7.5 it works perfect
