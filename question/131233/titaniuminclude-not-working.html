Ti.include() is not working. It always says can't find file. I searched for several options but none were working for me.

My directory structure is:

  • Root

    • Resource

      • app.js

      • functions.js

    • all the standard titanium files

Ti.include(&quot;functions.js&quot;);

But this is not working correctly.

Can anyone help me?

    Try a "clean build" – seems like Titanium sometimes doesn't copy new files into a project. Choose Project, Clean and clean this or all your projects. This issue seems to show up when you add new files to a project…so you build once, later add a file, then build again and the app fails to find your new file.
