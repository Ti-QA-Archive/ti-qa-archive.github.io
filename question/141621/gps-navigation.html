GPS Navigation

Is it possible to create "Get Direction" option in Titanium to function in following way:

    • User click on "Get Direction" button
    • Intent dialog is opened, and user can choose if directions are calculated via google maps or iGo for example.
    • External application is opened with directions

It must work on IOS and android.

  • This sounds like just a button that open a different application. The functionality would depend on the 3rd party app. Some apps don't allow you to pass in data when you call up the app, and other apps do. For google maps on iOS, you could pass a url to the web based google maps and that map will automatically open up the native app. On Android, I believe that just the web map will be displayed. This will only work for a while as the new iOS SDK will use Apple maps and discontinue the use of Google maps. I have no idea how this would work on iGo as I've never heard of it.
