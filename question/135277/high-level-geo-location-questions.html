High level Geo-location questions

Hey all,
Trying to get started on Geo location, I understand the basics, but I have a few questions regarding location services.

  1. is performance affected by listeners constantly checking for changed coordinates?

  2. How to I capture changed settings to location services, say someone turn geo services off and then back on, I may need to do something in those cases.

  3. I do a check if Titanium.Geolocation.locationServicesEnabled, but they may have prevented the app from having location on, so how is it I check for "app location services" because I received kCLErrorDomain error 1, when I had location services on, but app off.

Sorry 3 questions in one.

  • Yes performance is effected by constantly checking for coordinates.

    For the 3rd part here is what I use:

    exports.isLocationEnabled=function(){
    if (Titanium.Geolocation.locationServicesEnabled === false)
    {
        return false;
    }
    else
    {
    if (Titanium.Platform.name != 'android') {
        var authorization = Titanium.Geolocation.locationServicesAuthorization;
        Ti.API.info('Authorization: '+authorization);
        if (authorization == Titanium.Geolocation.AUTHORIZATION_DENIED) {
            return false;

        }
        else if (authorization == Titanium.Geolocation.AUTHORIZATION_RESTRICTED) {
            return false;
        }
        else {
            return true;
        }
    }
}
return true;
}
