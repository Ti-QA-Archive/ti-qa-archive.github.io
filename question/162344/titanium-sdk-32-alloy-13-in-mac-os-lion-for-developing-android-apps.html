  • Accepted Answer

    Titanium SDK 3.2.0.GA had a bug (TIMOB-16130) where Xcode was required even if you just wanted to do Android or Mobile Web development. We fixed it and it'll be available in the upcoming Titanium SDK 3.2.1.

    As far as Mac OS X 10.7 Lion support, we no longer support Lion with 3.2.0 and newer. While Titanium 3.2 may work on older OS X versions, we do not actively test new releases on older versions.

  • Hello Ivan,

    the limits for the MacOS X version on the Ti Studio is usually related to the needed Xcode for running iOS code (i.e., Titanium SDK 3.2. supports iOS 7, therefore you need Xcode 5.x in order to compile for iOS 7, and Xcode 5.x is only supported on 10.8.5+).

    But, if you are going to develop only for Android, sounds like you won't get any issue, as long as the Android Developer Tools are supported in your MacOSX version.

    From the Android Developer Toolkit page, you will need at least: Mac OS X 10.5.8 or later (x86 only). (so, likely you are ok).

    You will need JDK 6 in order to run Ti Studio, but looks like your MacOSX version includes JDK6.

    Let us know how it goes.

    Best,

    Mauro
