Titanium - Android : How to change the color of title text of Tab?

I have a tabGroup which has two tabs having title 1. Login 2. Register. I need to keep a title color (font color) of Login as Green and that of Register as Blue. How can I do that?
I am using Titanium SDK 3.4.0 to develop an App for android platform.

  • Actionbar Tabs must be styled by a theme. Read this guide for more information: Styling the Action Bar

    To create a custom theme you can use this tool: Action Bar Style Generator. Choose appCompat as style compatibility.

    Usually the font is only black or white depending on the base theme. You can still customize the theme, e.g. with something like this:

    <!-- ActionBar tabs styles -->
<style name="MyActionBarTabs"
       parent="@style/Widget.AppCompat.Light.ActionBar.TabView">
    <!-- tab indicator -->
    <item name="android:background">@drawable/actionbar_tab_indicator</item>
    <item name="android:textColor">#00A7F4</item>
    <!-- Support library compatibility -->
    <item name="background">@drawable/actionbar_tab_indicator</item>
    <item name="android:textColor">#00A7F4</item>
</style>

    (source: http://stackoverflow.com/questions/23251994/how-to-change-the-font-color-of-actionbar-tab-to-distinguish-between-selected-an)
