  • Christophe,

    From my knowledge what you are trying to do here is not possible. Please read this documentation on scrollable views:
    http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.ScrollableView

    It says that a scrollable view is a view that encapsulates a horizontally-scrolling set of child views, known as pages, navigable using its built-in horizontal swipe gestures.

    This is a built in native function that cannot be changed to work vertically because neither the iOS or Android OS can recognize such an action.

    Carter
