  • Hi,

    Why dont you use a HashMap and send the data is an object, rather than JSON ?

    you can cast like this:

    public void sendCustomDataQuery(HashMap args) {
    boolean autoSave;
    Object temp;
    temp = args.get("autoSave");
    if (temp instanceof Boolean) {
        autoSave = (Boolean)temp;
    }
}

    That should work.
