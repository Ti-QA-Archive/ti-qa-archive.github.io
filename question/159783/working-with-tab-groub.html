    Hi

    I am hoping that you are trying to open a child window and NOT changing the initial root window.

    Opening a child window of a tab is slightly different to simply open a window without a tab. Try tweaking your code as follows;

    function openResult() {
    myWin = Alloy.createController('resultPage').getView(); 
    $.tab1.open(myWin, { animated: true });
}

    This tells the tab to open the window as a child of this tabs navigation group.
