1.7.2 bug handling certain images on Android device

I have found a bug in the 1.7.2 release with handling certain images. It works fine in the Android Simulator, however when deploying to an Android device, it gives an error - Invalid index 0, size is 0.

Here is the code:

var win = Ti.UI.createWindow();
win.open();

var view = Ti.UI.createView({ top: 0, left: 0, right: 0, height: 33, backgroundImage: "logostrip.png" });
win.add(view);

http://tothsolutions.com/logostrip.png
