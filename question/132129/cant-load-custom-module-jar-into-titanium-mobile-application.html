    you can see the exact location by opening (double-clicking) tiapp.xml of your app, and than clicking "configure…" link at the bottom of the screen.

    1. do not unzip the file! just copy the zip into the folder which is called "Titanium SDK Home" in the configuration screen

    2. delete "modules" folder from your app if you have it

    3. delete "build" folder from your app

    4. click project-> clean project menu in titanium studio

    5. rebuild the app

  • you're using the module wrong.

    here what you do:

    – put the zip file of your module (the file which the build produced) into your titanium installation folder, to the same leve as "modules" folder. do not unzip the file! titanium will do it when your app is built;

    – in your titanium app's tiapp.xml file, registeryour module in the <modules> node, so it should look something like this:

    <modules>
   <module platform="android" version="<your module version>"><your module id></module>
</modules>

    – in your apps' javascript code where you want to use your module, do:

    var mymodule = require('<your module id>');

    now you can use methods and properties from your module - they are exposed by mymodule variable.

    hope it helps
