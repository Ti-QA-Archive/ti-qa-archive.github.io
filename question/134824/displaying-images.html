Displaying Images

I would like to display an image to the left of my text in a TableView. The only images that will display in the emulator are the stock KS_nav_ui.png and KS_nav_views.png. If I change the values to a different image, it doesn't work.

This works fine.

var tbl_data = [
    {title:'Row 1', leftImage: 'KS_nav_views.png'},
    {title:'Row 2', rightImage: 'KS_nav_ui.png'},
    {title:'Row 3', backgroundColor: '#fdd'}
];

This displays nothing.

var tbl_data = [
    {title:'Row 1', leftImage: 'Cal1.png'},
    {title:'Row 2', rightImage: 'Cal2.png'},
    {title:'Row 3', backgroundColor: '#fdd'}
];

The images are all located together.

  • Accepted Answer

    try this:

    var tbl_data = [
    {title:'Row 1', leftImage: '/Cal1.png'},
    {title:'Row 2', rightImage: '/Cal2.png'},
    {title:'Row 3', backgroundColor: '#fdd'}
];

    and also add this in you tiapp.xml :

    <property name="ti.android.fastdev" type="bool">false</property>

    now run after clean and build. This code is working here , should work for you too.
