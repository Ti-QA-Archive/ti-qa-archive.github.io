Ti.Media.getCurrentPosition issues

Hello, All.

I am designing an android app in Titanium 2.0.1 in which I need to get my current location. The issue is that every time I try to use the getCurrentPosition() method, my current position returns as null (or undefined).

My code:

var loc = Ti.Geolocation.getCurrentPosition(MainF());

MainF()
    {
    alert(loc);
    }

I am really stuck.

If anyone can help me, that would be amazing.

Thanks,

Zach

  • Accepted Answer

    the call is made asynchronously and the function is passed a result. Try:

    Ti.Geolocation.getCurrentPosition(function(loc){
    alert(loc);
});

    That's the easiest way to do it.
