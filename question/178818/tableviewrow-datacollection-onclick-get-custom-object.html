TableViewRow dataCollection, onClick get custom Object

I guess I made a mistake, but I don't get how to receive the whole Object.

First of all I have Table

<Collection src="Orders" />

<TableView dataCollection="Orders" separatorColor="#000000" onClick="getRow">
        <TableViewRow id="rowView">
            <Label id="label_colorIndicator" />
            <Label id="label_startTime" text="{startTime}" />
            <Label id="label_woNumber" text="{workOrderNumber}" />

            <View id="label_fullname" layout="horizontal">
                <Label id="label_firstname" class="fontSize" text="{firstName}" />
                <Label id="label_lastname" class="fontSize" text="{lastName}" />
            </View>

            <Label id="label_street" class="fontSize" text="{street}" />

            <View id="label_place" layout="horizontal">
                <Label id="label_plz" class="fontSize" text="{zipCode}" />
                <Label id="label_city" class="fontSize" text="{city}" />
            </View>

            <Label id="label_addition" text="{addition}" />

            <Label id="label_type" text="{woType}" />
        </TableViewRow>
</TableView>

The table fill correctly and the onClick-Event fires, too. On my onClick-Function I want to receive the whole Object. But I don't know how to get it. The data shown in the table are a part of the "Orders" Object.

With "e.rowData" I only get the data stored directly on the "TableViewRow" and with "e.row.children[n].text" I only get the children stored in the "TableViewRow"

Is it possible to get the whole selected object?

    While you could store the whole object (JSON-stringified) in a custom property of your table row, that would be a bad idea memory and performance-wise. Instead, you should store the object's primary key and then fetch that object as needed.

    <TableViewRow id="rowView" key="alloy_id">
<!-- or use the idAttribute field if you've set one -->

    Then in the controller

    var obj = Alloy.createModel('Orders');
obj.fetch({id: e.rowData.key});
