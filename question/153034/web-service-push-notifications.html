  • Hi Michael Massalas,

    did you find a solution?

    i've tried your example and manage to get it work with sucess,

    i changed the channel name in the "sendPush()" function to mach the channel that the device is registered on ACS console.

    Ex in push.html:

    function sendPush() {
    var newtext = document.drop_list.inputtext.value;
    var e = document.getElementById("channel");
    var strUser = e.options[e.selectedIndex].value;
  var data = {
    channel: strUser,
    payload: newtext
  };
  sdk.sendRequest('push_notification/notify.json', 'POST', data, pushCallback);
}

    then changed to:

    function sendPush() {
    var newtext = document.drop_list.inputtext.value;
    var e = document.getElementById("channel");
    var strUser = e.options[e.selectedIndex].value;
  var data = {
    channel:  "channel's name that match's the device subscrived channel",
    payload: newtext
  };
  sdk.sendRequest('push_notification/notify.json', 'POST', data, pushCallback);
}

    and it works :)

    tested only on android with the latest 3.1.3 sdk and android 4.2.2 on sony xperia tablet Z
