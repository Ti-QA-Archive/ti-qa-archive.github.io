How to call a function in a different window.

Hi
I just started with Titanium and I am trying to build a simple To Do List app.

The app is simple – it has two tabs
Tab 1 is the list (list.js) and tab 2 is has a form to add a new item (form.js)

After I add a new item to the list using from.js, how can I automatically refresh the list.js window?
I tried playing with the focus event listener for list.js but I had no luck.
How can I use Ti.App.fireEvent?
Or any other suggestions?

Thank you

    Here's a simple example:

    
// eg: put this in app.js

function doSomething(e){

    Ti.API.info('the event fired: foo = '+e.foo);    
    Ti.API.info('the event fired: something = '+e.something);    
}
Ti.App.addEventListener('your_custom_event', doSomething);

// then put this in some other js file to call it and send it some info

Ti.App.fireEvent('your_custom_event',{foo:'BAR!', something:'or other'});

    hope this helps ;)

    Another thing you can do is:

    You could have a function called anything() in app.js and 'attach' it to another window called win1 by adding win1.anything = anything; after the createWindow method which created it.

    Then in your list.js you can refer to it as var a_function_called_anything = Ti.UI.currentWindow.anything

    Using custom events is easier and very useful. I don't know what the best memory and performance option would be, but passing the values/pointers to things/functions in app.js around other window contexts like that can be very useful to get working too.
