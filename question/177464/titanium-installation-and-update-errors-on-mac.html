Titanium installation and update errors on mac

Hi all,
On installing titanium to a macbook air just updated to OS X Mavericks, when I open titanium for first time it seeks out updates, which is expected. However, when click to install, titanium prompts me for a system admin username and password that are not in use anywhere else on device (fresh rebuild). The user name is readonly, and so I cannot progress the updates, hence cannot use the application. Any suggestions on how to enable use of an alternate keychain username/password combination?

    Some people said that:

    sudo chown -R $USER /usr/local

    will do the trick.
