Localization

Right now there is the possibility of internationalize some text thanks to i18n folder. How can I do the same for images, files…

Thanks very much!

    You can use the strings files in the i18n folder to do that, just create a new string entry and in the value add the path to the image.

    Like this: on i18n/en/strings.XML 

        <string name="path_logo_header">images/logo_en.png</string>

    And in your ImageView use L('string-name') like this:

    var logoHeader = Titanium.UI.createImageView({
           image:             L('path_logo_header'),
           top:            9,
           left:           15,
           width:          163,
           height:         33,
        zIndex:         2
});
