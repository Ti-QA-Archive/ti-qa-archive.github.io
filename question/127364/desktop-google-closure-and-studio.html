    • Basically you pass -j to tibuild.py to have it obfuscate, but it only does simple at the moment.

      My plan was to allow the user to select which files should be obfuscated, so that they could use the header directives for Closure in those files to either include or set params for the obfuscation. Currently it just does simple, on all .js files in the project, for better or worse.

    • I see.

      I did not managed to get Closure to minify CSS files, just JS, so I have a script that copy all files, minify CSS files with Yuicompressor and the JS files with the Closure.

      For JS files:

      "$javaBin" -jar "$compilerJar" --js="$dev/$fl" --js_output_file="$prod/$fl"

      For CSS files:

      "$javaBin" -jar "$yuicompressorJar" --type css -o "$prod/$fl" "$dev/$fl"

      For the rest:

      cp -afu "$dev/$fl" "$prod/$fl"

      I'm putting it here because it may lead you to think about minify CSS files too. Most of then are big and usually have a lot of then when you have themes.
