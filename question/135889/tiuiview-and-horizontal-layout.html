Ti.UI.View and horizontal layout.

Hi,

I am using the latest Titanium Studio version, and the following happened in the iOS simulator, haven't tested on real devices.

My code is very simple:

//The createWindow method is called in a different .js, where I build a tabGroup
var placeFilterWin = Ti.UI.createWindow({
    url:'/Pages/filterScreen.js',
    navBarHidden:true,
    title: 'Place Filter Screen',
    layout: 'vertical',
    fullscreen:true
});

//The code below is in the filterScreen.js file
var win = Ti.UI.currentWindow;

var topBar = Titanium.UI.createView({
   backgroundColor:'#cccccc',
   width:Titanium.Platform.displayCaps.platformWidth,
   height:70,
   layout: 'horizontal'
});

var logo = Ti.UI.createImageView({
    image:'/Images/logo.png',
    width: '50%',
    height: '50%'
})

topBar.add(logo);
win.add(topBar);

When I do this the view is created and is shown, however, the image is not. If I remove the layout parameter the view is shown and the image is shown centered inside the view.

Any ideas on why the horizontal layout does not show the image?

Thanks in advance,

Jay Santos
