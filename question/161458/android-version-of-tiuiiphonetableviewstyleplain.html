Android version of: Ti.UI.iPhone.TableViewStyle.PLAIN

Converting an app to android.

var style = ( osname == 'android') ? 'transparent' : Ti.UI.iPhone.TableViewStyle.PLAIN;
var seperationStyle = ( osname == 'android') ? 'transparent' : Ti.UI.iPhone.TableViewSeparatorStyle.NONE;
var table = Ti.UI.createTableView({
    placeData:{results: [], total: 0},
    backgroundColor:'transparent',
    style: style,
    separatorStyle: seperationStyle,
    scrollsToTop : true,
    showVerticalScrollIndicator: true,
    top: 0,
    layout: 'vertical'
});

This app was created a while ago and as time is of the essence we are sticking with the table for the time being. However, I cannot find an android equiv for the ios: Ti.UI.iPhone.TableViewStyle.PLAIN

What should the default value be?

    Ti.UI.iPhone.TableViewStyle.PLAIN means that you want it not grouped (on iOS), and that's the default setting for android. Long story short, you don't have to set anything on android to make it plain.
