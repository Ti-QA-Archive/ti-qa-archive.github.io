  • You aren't using JSON here, you have an object. You need to turn your object into a JSON string if you want to use JSON. The value property of a Buffer object must be a Number or a String. See docs for more details: http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.Buffer-object
