How to write or read binary data?

Hi, I am trying to determine how to write binary data to a file.

Let's say I have the hex string "1323BD34839EF413AF" (which contains a total of 9 bytes). I want to write these 9 bytes out to a file in binary format so that it only takes 9 bytes of space on the disk. How do I do that? I have browsed around in the Titanium API and I see the ability to read/write text files, and read/write whole blobs (obtained as a result of calling other API methods), but I don't see documentation on how to write out custom binary data a byte at a time…and the reverse capability of reading binary data one byte at a time. Does anyone know if this can be done in Titanium, and if so, could you kindly post some sample code or point to the right documentation? Thank you in advance for your help!

  • Hi

    I use in my project (http://www.draw2d.org) the PHP binding to write binary data to the filesystem and
    this works very well. I retrieve the base64 encoded data from a canvas object and hand over
    this the generated PHP function..works like charm.

     <script type="text/php">
// required for the TideSDK file storage
// normal HTML5 browser ignores this part of code because the type is set to PHP.
// TideSDK with PHP binding run this code
// 
function tideSDK_writeBase64AsBinaryData($fileName, $base64){
   $binary=base64_decode($base64);
   $file = fopen($fileName, "w");
   fwrite($file, $binary);
   fclose($file);
}
</script>

    You must enable and pack the PHP core in the TideSDK Developer to use this method.
