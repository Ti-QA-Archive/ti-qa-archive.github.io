Mobile: Compile with Xcode and include images?

Hi!
I like to use Xcode to compile and install my Titanium iPhone apps, as opposed to using the Appcelerator interface. Mostly because it's faster as it doesn't require a full sync after every install.
However, when I open the Xcode project and compile it (after setting the code signing), the app runs fine. However, all images are missing: I guess they aren't copied into the build.
Is is possible to fix this and use Xcode to compile my apps?
