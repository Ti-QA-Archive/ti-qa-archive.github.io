  • You pass the delegate (callbacks) into the method of the child object…

    Consider this method to be called on the child

    
function DoSomethingReallyInteresting(success, failure)
{
    if (....... something that succeeded.....) {
        success();
    }

    if (...... something that failed......) {
        failure();
    }
}

    It would be called like this from the parent

        childObj.DoSomethingReallyInteresting(function(){
        alert ('it worked!');    
}, 
    function(){
        alert ('it failed');
    });
