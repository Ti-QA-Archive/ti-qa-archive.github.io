Customise buttons in an Option Dialog

Hello,

I'm just wondering if it is possible to customise the buttons with an Option Dialog:

 var dialog = Titanium.UI.createOptionDialog({
        title: 'Choose the station you would like to contact',
        options: ['ButtonOne', 'ButtonTwo', 'ButtonThree'],
        destructive: 0,
        cancel: 1
    });

ButtonOne appears with with a red background and ButtonTwo appears with a black background, whereas any additional buttons have a light grey background.

At the moment I just want to set my own background colours.

Regards
