  • So, to deal with the 2 arrays, you'd probably want to loop through the second array and push the existing ids into a hash-like array. ie: 

    exists[50] = true;

    Then you can loop over the first array and if the id for each iteration exists, then push that data onto a new array.

    However, since you are using databases, it might just be easier to create a query that joins the two tables grabbing only the rows that you need.

    pseudo-statement:

    select * from table1 where id in (select id from table2);
