How to push data to an array for use in another tab

Hi guys,

I'm trying to push data to an array in my data.js. When they click the Add to Favourites button, I want to push the data to the array. var bandWin = Ti.UI.currentWindow;

Ti.include('data.js');

  • I don't think that an include() call is going to do what you want. My understanding is that include() just pulls in the code and runs it in place. It doesn't create any special scope where you could share data across modules.

    You really shouldn't use include() at all; instead, use CommonJS modules and call require() on them. Using a CommonJS module, you could definitely share values between modules.

    module code (Favorites.js)

    var _favorites = [];

function addFavorite (f)
{
    _favorites.push (f);
}

function getFavorites ()
{
    return _favorites;
}

exports.addFavorite = addFavorite;
exports.getFavorites = getFavorites;

    code for your tab1 window:

    var Favorites = require ('/Favorites');
Favorites.addFavorite (bandObject);

    code for your tab2 window:

    var Favorites = require ('/Favorites');
var favs = Favorites.getFavorites ();

for (var i = 0; i < favs.length; i++)
{
    var bandObject = favs[i];
    // ...
}

    • I'm not sure what happens if you pass an object to alert(). It might show as nothing, or it might show as [Object] or something. I haven't tried it.

      Rather than relying on alert() calls, why not use the debugger? Set a breakpoint and inspect the object in the debugger. The nice thing about this technique is that if there is something wrong with bandObject, you can actually inspect some other things and have a chance at figuring out what is wrong. For example, you could inspect the favs array and make sure it's actually an array. If it's not, you know you have a different type of problem.
