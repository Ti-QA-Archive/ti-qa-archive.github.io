SQLite trying to get data range between two dates

Hi, trying to pull data for the last 7 days of my sqlite database

database setup

db.execute('CREATE TABLE IF NOT EXISTS logData("id" INTEGER PRIMARY KEY, "time" DATETIME DEFAULT (CURRENT_TIMESTAMP), "value" INTEGER, "7days" FLOAT, "30days" FLOAT, "fromStart" FLOAT)');

when I output the db to the console it is displaying correctly

[INFO] :   id: 1 time: 2014-04-30 05:58:31 value: 9
[INFO] :   id: 2 time: 2014-04-30 05:59:20 value: 9
[INFO] :   id: 3 time: 2014-04-30 05:59:25 value: 5
[INFO] :   id: 4 time: 2014-04-30 05:59:29 value: 1
[INFO] :   id: 5 time: 2014-05-08 05:34:15 value: 5
[INFO] :   id: 6 time: 2014-05-09 07:45:42 value: 9
[INFO] :   id: 7 time: 2014-05-09 07:45:50 value: 5
[INFO] :   id: 8 time: 2014-05-09 08:20:28 value: 1
[INFO] :   id: 9 time: 2014-05-09 08:20:31 value: 9
[INFO] :   id: 10 time: 2014-05-10 05:34:01 value: 9
[INFO] :   id: 11 time: 2014-05-10 07:33:18 value: 5

the "id", "time" and "value" are the row names from the db, its not actually data in the db.

and here is the query I'm currently using to get the last 7 days data as a resultset and print it to the console

var rows = db.execute('SELECT * FROM logData ORDER BY ABS( strftime( "%Y-%m-%d", time ) - strftime( "%Y-%m-%d", "2014-05-06" ) ) ASC');

    while(rows.isValidRow){
        Ti.API.info(' id: ' + rows.fieldByName('id') + ' time: ' + rows.fieldByName('time') + ' value: ' + rows.fieldByName('value'));
        rows.next();    
    };
    rows.close();
};

and the error I get is:

[INFO] :   id: undefined time: undefined value: undefined

and it prints it indefinitely until I press quit. Thanks for any help.

    Hi

    If you are trying to get the last 7 days worth of rows from the table using the time field to determine the data - then this should do the trick;

    select * from logData where time > date('now','-7 days');

    Personally I always prefer to state the columns I need to use form the result set - this way it saves memory;

    select id, time, value from logData where time > date('now','-7 days');

    Works fine in my tests.
