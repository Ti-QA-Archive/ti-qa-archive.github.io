Filter search by beginnig of word?

Hello,

You can use a searchBar to filter the contents of a tableView by assigning the searchBar's name as the "search" property of the tableView, as explained for example here. But, the contents are filtered so that the matched string can be in any position of the word, and I have not found a way to alter that.

Is it possible that only the elements that match the query string at the beginning of the word are shown? So that, for example, if I write "pa" in the searchBar, only the elements of the tableView whose filterAttribute begins begin with "pa", like "parts" or "paranoia" will be shown, while those that contain this string in the middle of the word, like "spare" or "comparison", will be hidden.

— asked October 31st 2011 by Joan Usecm

  • I'd say since you don't have a filter function to check for validity you'd have to clone the original array you're using, use it as the source for the table view, catch the "change" event for the searchBar, and then clear the array and insert only elements that match your search criteria to it, then add it again as the tableview source.
