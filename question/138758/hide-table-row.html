Hide table row

Hi,

i want do hide/unhide some rows in a table when pressing a button.

In an older thread here, someone suggested to do this by setting the table row height to 0, but this has no effect for me. The minimum height with an effect is 1, which means the row is not hidden completely. I also tried to set the tables minRowHeight to 0, but also with no effect.

Is it not possible to use a height of 0? Is there any other way to hide a row?

I am trying this with Titanium Mobile SDK 2.0.2 on iOS.

Thanks, Joe.

  • The common way is to modify the table's data property. You can maintain an internal array with all the rows and another with the rows shown in the table view.

    You may want also to investigate the setSearch and setFilterAttribute properties, they may help in some situations.
