Android webview - pinch to zoom works only after a scroll occurs

Hi all,
i have a strange behaviour when using webviews in Android.

My purpose is to manage pinch-to-zoom gesture for images, so i chose a webview to do id. Everything works perfectly but only after a scroll.

Il try to explain: when the webview is loaded i see the image correctly scaled inside it. Then i can scroll and finally i can pinch and zoom. If i try pinch my image before any scroll occurs, the pinch is ignored.

This is a little snippet of code to reproduce my issue:

var window = Ti.UI.createWindow();
var webview = Ti.UI.createWebView();
window.add(webview);
webview.url = 'http://path/to/big/image.jpg;
window.open();

The same problem occurs even if i use the html property a bunch of pharagraphs. I've tried also tuning up the webviews's properties in various way, but nothing changed..

Have you ever experienced this ? Any suggestions ?

Thanks!
