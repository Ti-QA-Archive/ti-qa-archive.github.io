How to add eventlistener on controller close

Titanium Command-Line Interface, CLI version 3.2.0, Titanium SDK version 3.2.0.GA

Lo all,

I am trying to add a close event listener on a window without success. I have tried every permutation of id, controller name etc but obv not the correct one. I have a feeling that using a model instance is screwing it up…
No matter what I do, I get some error of the form:

[ERROR] TiExceptionHandler: (main) [7711,7711] ----- Titanium Javascript Runtime Error -----
[ERROR] TiExceptionHandler: (main) [1,7712] - In alloy/controllers/branch.js:44,25
[ERROR] TiExceptionHandler: (main) [0,7712] - Message: Uncaught TypeError: Cannot call method 'on' of undefined
[ERROR] TiExceptionHandler: (main) [0,7712] - Source:     Alloy.Models.branch.on("fetch change destroy", __alloyId3);
[ERROR] V8Exception: Exception occurred at alloy/controllers/branch.js:44: Uncaught TypeError: Cannot call method 'on' of undefined

branch.xml:

<Alloy>

    <Model id="branchinstance" src="branch" instance="true"/>

    <Window>

        <View class="container">

            <ImageView image="{branch.imageurl}" defaultImage="images/cheese.jpg"/>

            <Label text="{branch.branchcode}" class="label_data"/>

        </View>

    </Window>

</Alloy>

branch,js:

var args = arguments[0] || {};


// var branch = Alloy.Models.branch;
var branch = $.branchinstance;

branch.id = args.id;

$.branch.open();

branch.fetch();

$.branchinstance.addEventListener('close', function() {
    alert('close branch');
//     $.destroy();
});

  • John, how about adding the ID to your window object instead of your model object, then add the onclose event to it? This way if you need your window, you have a reference… Something like:

    <Window id="myWindow" onClose="closeFormFunction">

    branch.js

    function closeFormFunction() {
//do something on window close
alert('Close Branch');
}

    Ray

  • Cheers all, I worked it out. The actual error was coming from the view, not the trigger. ie

                <Label text="{branch.branchcode}" class="label_data"/>

    should have been:

                <Label text="{$.branchinstance.branchcode}" class="label_data"/>

    Where branchinstance is the id of my model. Thank you all for your time.
