Android - Uncaught ErrorL Invalid int: "Value" - when Value is Chinese

Hi,

I am only getting this error on Android not on iOS.

Here is the Code

TheButton.addEventListener('click', function(e){
    var value = "??";
// Track Click Value
    Alloy.Globals.tracker.trackEvent({
        category: "Category",
        action: "click",
        label: "TheButton",
        value: value
    });

});

If the value is Chinese I get this error on Android

Uncaught ErrorL Invalid int: "??"

If it is not in Chinese I font get any error??

Is there any way I can convert that Chinese value to something Java would accept?

  • I found this answer on google -

    The 4th parameter of Events must be an integer.

    So use only numbers. No quotes around them. Also don't use floating point numbers. All hits with strings or floating point vars as the last parameters are ignored with no mercy.

    valid event:

    _gaq.push(['_trackEvent', 'Video', 'Percent Watched', 'Video content', 95]);

    invalid events:

    _gaq.push(['_trackEvent', 'Video', 'Percent Watched', 'Video content', 95.7]);
    _gaq.push(['_trackEvent', 'Video', 'Percent Watched', 'Video content', '95.2']);
    _gaq.push(['_trackEvent', 'Video', 'Percent Watched', 'Video content', '95']);

    So seems like we can not use something like Chinese there…
