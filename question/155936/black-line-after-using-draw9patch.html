  • If you're trying to use a 9-patch image, you need to reference it in your TSS or JS code as yourfile.9.png. It sounds like you're using the name yourfile.png or attempting to use a 9-patch image on iOS (where it's not supported).

    If you're trying to convert a 9-patch image back to a standard PNG, simply rename the file without the .9 part. Open it in a standard graphics editor. Crop the outer 1 pixel off all four edges (reduce the canvas size by 1px).

    — answered August 14th 2013 by Tim Poulsen
