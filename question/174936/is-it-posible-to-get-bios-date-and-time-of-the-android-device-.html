  • Devices store the date and time information in UTC and then show it to the user using the proper regional settings. So the best thing for you would be to retrieve the device's time in UTC.

    Moment.js provides such a function: http://momentjs.com/docs/#/parsing/utc/

    If your project is using Alloy, moment.js is already included. So your code would look something like this:

    var moment = require('alloy/moment');

Ti.API.info(moment.utc().format());

    Hope this helps.
