Get the first displayed row of tableview while scrolling

I have a tableview with sections: is there any way to know which of the rows is the first currently displayed while scrolling? Not the first in the entire tableview, the topmost while scrolling. Thanks!

  • You can get this by adding an event listener to the TableView's scroll event. On ios, you could use the event's contentOffset property to calculate the first visible row. On android, you can use the firstVisibleItem property, which gives you the row index of the first visible row.
