Change language to Picker Datetime titanium iOS

Hi, I need change the language to the PICKER_TYPE_DATE, actually the months appear in English: June, Feubrary, March… I need it but in Spanish, for example: Enero, Febrero, Marzo, Abril.

I can do it?

    Hi

    These are automatically set by iOS using the international setting the device (or simulator) is set to when the app is being used.

    To test use the simulator, go into settings > General > International > Region Format > [your choice].

    The user of the app will have the correct settings on their phone (or at least be happy with whatever is currently chosen). So this is dealt with for you.
