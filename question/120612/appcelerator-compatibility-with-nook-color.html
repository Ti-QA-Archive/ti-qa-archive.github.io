Appcelerator compatibility with Nook Color

Does anyone know if the Titanium SDK is compatible with the Nook Color?

According to the B&N website, the "NOOKcolor runs the Android 2.2 Operating System and works seamlessly with the Android SDK (API Level 8). Barnes & Noble provides an Android SDK add-on (NOOK SDK 1.2) to help you get started developing applications for NOOKcolor." (https://nookdeveloper.barnesandnoble.com/tools-services/sdk.html)

Has anyone had success building for a Nook device?

  • Accepted Answer

    Works in Emulator just fine. Here are the steps to test:

    • Register as a Nook Developer – https://nookdeveloper.barnesandnoble.com/sign-up.html
    • Ensure that the Android SDK version 2.2 is installed on your machine: http://developer.android.com/sdk/installing.html
    • Download the Nook SDK
      1. Navigate to the <android-sdk> tools directory
      2. Execute the Android tool command with no options: $ android
      3. In the Android SDK and AVD Manager left panel, Available Packages
      4. Click Add Add-on Site
      5. Enter the URL for the NOOKcolor add-on SDK: https://nookdeveloper.barnesandnoble.com/sdk/
      6. Click Ok
      7. Click the check box for the URL that you just entered
      8. Click Install Selected
      9. Accept the License Agreement and click Install
