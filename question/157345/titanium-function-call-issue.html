I don't know how to call a function from one .js to another .js in tab group.
I have a callback in app.js and i want to call a function from another .js in tab group.
How can i do it?

    Hi Sajid,

    You can use export/require method for use your function in different .js files.

    Example :

    Here is one date function in lib folder.

    exports.getDateAsTitle = function(e) {
    return moment().format('MMMM Do YYYY, h:mm a');
};

    now you can use this in .js file like this way : 

    var util = require('util');

util.getDateAsTitle();

    if you need any other help then please tell me.

    Thanks SP

    You can add event listener on Application level.Like:

    Ti.App.addEventListener('event',''function');

    Alternatively, you can do something like :

    Titanium.include("One.js"); // now you can access all the functions and variables in one.js
