A few beginner questions regarding Appcelerator

Hi,

I am new to Appcelerator and new in the mobile world. I have a few questions regarding this platform.

1) If you build your interface using HTML. Are you limited to mobile web app? By web app, I mean an application that rely on Internet to work.

2) Is it possible to publish to App Store and Google Store?

3) I believe a built-in simulator is inside the IDE. Does it support all the popular events such as slide and double tap?

4) And finally, is there a way to copy it on a real mobile before publishing it somewhere? And if it's possible, does your phone need to be unlocked?

Thank you guys!

    Steve, few things you have to understand (and make use of :) related to the point 1 )

    1. Titanium despite the fact that uses javascript as programming language has nothing to do with the web ( on the iOs and android platforms).
      Javascript is bridged to the native UI elements, so when you create button, you don't get a noce styled html element but a native iOs/Android button like it was written in objC/Java.
      This being said the app will work on the device without the need to have an internet connection.
      Your final app will be a native app, with native UI in a native environment, no simulated with html (like phonegap for example.)

    1. Yes, in both of them.

    2. The simulators (android and iOs ) belong to the native SDKs not to Titanium Studio and tehy do register almost all the events a normal device has.

    3. on iOs you will have to be subscribed to Apple as a developer. On Android u just set the device to accept "Unknown sources" and "USB Debugging" and you're good to go.

    There is 3rd platform (MobileWeb) that indeed uses web technologies (html/css/js) but is for online webapps like any other online mobile web (think jQmobi if you want). The advantage is that with almost the same codebase you get 3 platforms at once.
