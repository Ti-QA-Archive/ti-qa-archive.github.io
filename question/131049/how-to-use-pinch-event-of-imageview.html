How to use pinch event of ImageView

Can any one please provide a sample code that shows how to use pinch event of imageView.

I know how to add the eventListener to the ImageView but not sure ow use the property like scale and velocity.

    Something like this works with pinch:

    var win = Ti.UI.createWindow({
    backgroundColor:'#dfdfdf'
});


var image = Ti.UI.createImageView({
    image:'KS_nav_views.png',
    width:100,
    height:100,
    top:190
});

image.addEventListener('pinch', function(e) {
    var t = Ti.UI.create2DMatrix().scale(e.scale);
    image.transform = t;
});
win.add(image);


win.open();

    As for the velocity property, you could turn the transform into an animation and use the velocity to calculate how quickly it animates, but it wouldn't be necessary in a simple example like the one above.
