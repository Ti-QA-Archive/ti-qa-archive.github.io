Slider - Snap into position

Hello,

I have a custom slider, with a min value of 1 and a max value of 3. I also have three views and depending on the value of the slider I show/hide these views.

I've created an image with 3 boxes, which I've placed on the slider via the thumbimage. Try and image three horizontal boxes that fit within the width of the iphone screen, something like this:

BOX1 | BOX2 | BOX3

Sample code below:

var customSlider = Titanium.UI.createSlider({
        min:1,
        max:3,
        value:2,
        width:533,
        height:70,
        top:105,        
        leftTrackImage:'images/buttons/slider-background.png',
        rightTrackImage:'images/buttons/slider-background.png',
        thumbImage:'images/buttons/slider-button.png',
        highlightedThumbImage: 'images/buttons/slider-button.png'
    });

    customSlider.addEventListener('change',function(e)
    {
        if ( e.value >= 1 && e.value < 2){
            headerLabel.text = 'Information Detail';
            e.value = 1;            
        }else{
            if (e.value >=2 && e.value < 3){
                headerLabel.text = 'Horoscope Detail';
                e.value = 2;
            }else{
                if (e.value == 3 ){
                    headerLabel.text = 'Temperature Detail';
                    e.value = 3;
                }
            }
        }
    });

So from the users point of view, it just looks like there are three different options. When the app opens the slider is in position 2, so the image is nicely centred on the screen. The user can then slide it left or right and the value changes accordingly.

I want the slider to snap into position so if the value is <2, BOX1 should be in the middle of the iphone screen and if the value >2, then BOX3 should be in the middle of the screen.

I hope I've explained myself clearly, any help appreciated.
