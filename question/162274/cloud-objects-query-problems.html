Cloud objects query problems

Hello, I am using android, I just want to ask if there is any possibility that I am able to allow 2 different username to query information among other many more username?
I want to ask if there is any ways to use to allow 2 or 3 username only. Like making use of 'OR' or 'AND', or any other methods. Thanks
For example,

                Cloud.Objects.query({
                    classname : 'Rating',
                    page : 1,
                    per_page : 10,
                    where : {
                        username : "123@gmail.com" || "321@gmail.com",

                        rate: 1
                    }

The part for username :

    It should be username:{$in:["123@gmail.com", "321@gmail.com"]} to query a mongodb record
