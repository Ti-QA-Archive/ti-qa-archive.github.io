  • Hi Joe

    Appcelerator (Titanium) is a rapid cross device development system, that uses javascript as the primary language, creates native apps, has an extensive API that lets you access pretty much all the functionality of the supported devices.

    Provided you have installed one or both SDK (XCODE & Android Java) then within an hour after the installation you can have written your first hello world app.

    There are sample apps, developed Q&A's (this), extensive API documentation, videos, tutorials and a built in debugger. Regular updates adding new functionality all the time.

    It is quick, simple and robust without the lack of features or quality. Well worth the effort.

    Certainly worth you trying it out, I did and I am still here a couple of years later helping out others.
