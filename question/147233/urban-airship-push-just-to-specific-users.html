Urban Airship Push just to specific users?

Hi,

i am building an app with registration and different … let's say groups of users.

I want to use Airship push to notify user of a group, about changes the other users (of this group) make to the app. Like posting a new photo. for example.

How can i send a push just for this user/Group? At the moment i don't have an idea, how to solve it on a cool way.

Any suggestion? :)

  • It all depends on your structure decision…

    In my case I use an external web server (Drupal) where I register all my users and assign their unique ids to urban airship aliases. That way, I can send notifications to specific users using Urban Airship Push API from the web server.
