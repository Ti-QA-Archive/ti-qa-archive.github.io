  • I know this is an old issue, but I was having the same problem with the latest version of inappbilling (2.2.1). This patch did not fix it, but what DID fix it was I called 

    var InAppBilling = require('ti.inappbilling');
InAppBilling.setPublicKey('KEY HERE');

InAppBilling.startBillingService();

    as soon as my application opens. This stopped this error and random crashes.
