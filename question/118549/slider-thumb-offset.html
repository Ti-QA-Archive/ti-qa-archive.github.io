Hi,

On the iPad I have a slider.

I'm listening to the "change" event and I want to show a view above the thumb when it changes.

The value property in the event object isn't sufficient since it shows the value between the min and max property values.

What can I do to make a view follow the position of the thumb in a slider?

  • This is how I accomplished it.

    Slider width = 250
    Slider left offset = 10
    Slider max = 70
    Slider value = 30

    ((Slider value / slider max) x Slider Width) + slider offset = the pixel value horizontally along the slider of where your label needs to be.
