Is there supposed to be a Titanium Folder in Library/Application Support?

Hi,

I've uninstalled and then reinstalled Titanium studio as it would not let me run any projects in ios emulator today (I installed first time and yesterday everything ran ok)

Now I reinstalled it still will not let me run on ios emulator, and I've spotted that the Library/Application Support folder under my user profile no longer has a titanium folder.

Uninstall reinstall does not fix this. Anybody got any ideas?

Oh I'm running OSX 10.6.8 and I have Xcode 4.02

    Depending on your setup, the Titanium folder may not be under your username folder. If installed for all users by an administrator, it might be in the /Library/Application Support/ rather than ~/Library/Application Support/

    You might want to check the console application to see if any errors are logged.
