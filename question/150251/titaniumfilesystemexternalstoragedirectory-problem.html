Titanium.Filesystem.externalStorageDirectory Problem

HEllo ,I am getting a problem creating a file in external storage root.I want to create a file in a folder called Download in external Storage Directory root.Unfortunately ,Titanium.Filesystem.externalStorageDirectory returns appdata:// instead of the external storage directory.I managed to create the folder to the external storage root using 

sdcard_path = "file:///sdcard/";

//lets get download file name
download_dir = Titanium.Filesystem.getFile(sdcard_path,"WapCity_Downloader");

But when I try to write file using the download_dir,file doesnt get written

f = Titanium.Filesystem.getFile(download_dir,file_name);
