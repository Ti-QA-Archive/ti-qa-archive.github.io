Searchbar height

Hi everybody,

I use a search bar in my tableview. The problem is i can't set the height of this searchar. I tried : 

var search = Titanium.UI.createSearchBar({
    barColor:'#000', 
    showCancel:false,
    height:40,
    searchHiddent:true,
    hintText:'blabla'
});


var tableView = Titanium.UI.createTableView({
    search:search,
    data:data,
    rowHeight:55,
    separatorColor:'#d6cdcb',
    backgroundColor:'#f2f2f2',
    fontColor:'#fff',
    filterAttribute:'filter',
    editable:false
});

but nothing..

Any help ?

Thanks :)

  • I don't think you can adjust the height on a searchbar when it is added to a table view. The height property works when you add the bar directly to the window.
