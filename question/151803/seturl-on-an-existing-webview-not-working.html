setURL on an existing webview not working

Is it possible to change the url of an existing webview once its already been added to a window?

The webview loads up our storefront url using a standard webview window:

var webview = Titanium.UI.createWebView({
    url : 'http://shop.com'
});

…then i need that same window to reload a product detail page using a variable that is produced by the barcode scanner which opens up elsewhere and adds its barcode value to a global variable.

I seem to have everything else in place but webview isnt refreshing when i sent it either

webview.setURL('http"//www.shop.com/detail.php?barcode='+Ti.App.barcodevar);

or

webview.url = 'http"//www.shop.com/detail.php?barcode='+Ti.App.barcodevar;

Any thoughts?

thanks in advance!
