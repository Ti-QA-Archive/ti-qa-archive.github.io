one style for multiple elements in TSS file

I saw a thread where you can assign multiple styles to a single element in the view. But nothing about multiple elements with the same style like you can do in CSS. For example, how would I implement this CSS in the TSS file?

h1, h2, h3, h4, h5, h6 { font-weight: normal; margin: 0 0 1em 0; color: #4e4848; line-height:1em;}

    Hi

    There is currently no way to denote multiple selectors for one class block in TSS.

    However you can create generic classes and use them everywhere in addition to your own specific ones;

    example.tss

    ".p": {
    font: {
        fontFamily: 'Arial',
        fontSize: 14,
        fontWeight: 'normal'
    }
},
".h1": {
    font: {
        fontFamily: 'Arial',
        fontSize: 18,
        fontWeight: 'bold'
    }
},
".title": {
    color: '#c60000'
},
".intro": {
    color: '#444'
}

    example.xml

    <Alloy>
    <Label class='h1 title'>Title</Label>
    <Label class='p intro'>Hello World</Label>
    <Label class='p'>Hello Again</Label>
</Alloy>

    Define your main class in the app.tss and then you can use them anywhere. In this case the classes are chosen based on the order so the second amends the first and so on.

    Using the default styles and the app.tss will help simplify the process.
