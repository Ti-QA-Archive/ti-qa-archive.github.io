    • So this can be done using jss files, here is the example:

      Resources/app.js

      Titanium.UI.setBackgroundColor('#000');

var win = Titanium.UI.createWindow({
    title : 'win',
    backgroundColor : '#fff',
    layout: 'vertical'
});

var label = Ti.UI.createLabel({
    text: 'Styled label',
    id: 'styleID'
});
win.add(label);
win.open();

      Resources/global.jss (taken from https://gist.github.com/1127334)

      #styleID {
    width: 250;
    height: 82;
    backgroundImage: 'none';
    backgroundSelectedColor: '#e2ffee';
    borderColor: '#0000ff';
    borderWidth: 1;
    borderRadius:5;
}

      do remember that values in the id will be in camel case instead of hyphen(-) separated.

      I hope this helps. If you feel relevant then mark this as an accepted answer.
