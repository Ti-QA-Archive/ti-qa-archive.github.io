  • lokesh, if this is a remote database exposed only through a web service, as you mentioned, then you cannot directly install it into your app. You need to communicate with the web service as per the communication contract it provides. You'll likely need to use Ti.Network.HTTPClient to communicate over HTTP with the web service to receive and send the data you need.
