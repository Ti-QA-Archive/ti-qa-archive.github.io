prevent going back to window or close main app

I have created a login page whereby it connects to the database and server and if the username and password are correct it opens to a special page (example: page2). My problem is that when the user clicks the device back button, it goes back to the log in page but in fact he is already logged in. How to 1) prevent going back to a certain window page or 2) close the main app after logging in so the user can start fresh and logged in..summary : page 1(consist of login) –> logs in –>(if ok) goes to page2 -> user clicks back button of android device –> goes back to page1 (login again)..Please advise. Thanks.

I am using: Mac MAVERICKS, TI 3.2.3 GA, For android device, SDK 4.4.2.

  • you can use

    page2Window.addEventListener('android:back',function(){return;});

    if you don't want to perform any action just write return in the function.
