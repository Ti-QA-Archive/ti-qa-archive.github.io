  • There are multiple approaches,

    1. Either you preserve the selected row index in array and when clicking on reset button, loop through this indexes and get the row by using tableView.getData[index] api an then unselect the checkbox.

    2. You need to loop through table data and unselect the checkboxes which has check mark.
