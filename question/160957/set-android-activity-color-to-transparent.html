Set android activity color to transparent.

Is it possible to set the android's activity background color to transparent so that the window/activity opened previously could be visible.

I want to make a sliding menu for the application I am working on and so far I have found that there's no way two activities/windows can be opened simultaneously i.e. both be showing. If I animate the window to the right instead of showing the activity previously opened it shows the default black color of activity.

Is it possible to achieve this in Titanium?

— asked December 29th 2013 by Ehtesham Kafeel

3 Comments

  • Yes. but you cannot set the background of activity, you can set the background of the related window.

    var win = Ti.UI.createWindow({backgroundColor:'transparent'});
