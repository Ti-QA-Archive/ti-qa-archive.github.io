  • You should create a standard Titanium button. Titanium.UI.createButton.
    Then using an event listener for 'click' on your button fire Titanium.Facebook.authorize(); after of course you set up your Facebook permissions… see example below.

    
var win = Ti.UI.currentWindow;
Titanium.Facebook.appid = "APP ID";
Titanium.Facebook.permissions = ['publish_stream'];

var myButton = Ti.UI.createButton({
  backgroundImage:'my_button_image.png',
  width:100,
  height:30
});

win.add(myButton);

myButton.addEventListener('click', function() {
  Titanium.Facebook.authorize();
});

  • Hmm.. not entirely true, Matt. (though the code may have come from FB to start with ;) Though also your solution would work )

    The actual image for each type of button is set by the Facebook module in the Titanium SDK.

    The button images used are in /Library/Application Support/Titanium/mobilesdk/osx/[SDK_VERSION]/iphone/modules/facebook/images/

    These are set in the FBLoginButton.m file which exists in:

    /Library/Application Support/Titanium/mobilesdk/osx/[SDK_VERSION]/iphone/Classes/FBConnect/

    I have customised mine this way, but naturally I need to redo this whenever I install a new SDK version.

    Developers/hackers beware.. Your mileage may vary… yadda .. yadda.. disclaimer.. etc :)
