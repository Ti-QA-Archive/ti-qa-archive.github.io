  • I got the same error when deploy to ios device. In my case, I opened xcode project in the myapp/build folder then run from xcode.. Exception threw with parsing error at the last line from code generated from Titanium. Here's in my case:

    (function(exports){exports.construct=function(){var currentUserStr=Ti.App.Properties.getString(MODEL_NAME,JSON.stringify({username:'',password:'',currency:'USD'}));exports.instance(JSON.parse(currentUserStr));return this;}
exports.persist=function(){Ti.App.Properties.setString(MODEL_NAME,JSON.stringify(exports.instance()));return this;}return exports;})({})

    If you open above code with Titanium you would see error
    >missing ; before statement

    To fix this, adding ; after the last exports statement would help.

    exports.persist = function() {
    Ti.App.Properties.setString(MODEL_NAME, JSON.stringify(exports.instance()));
    return this;
};

    Hope this help.
