[Android SDK Home] No Android SDKs were found under the specified SDK location

I tried setting the path from preferences -> Platforms -> Android -> Android SDK Home.
I even downloaded the SDK from developer.android.com, even for that folder i m getting the same error.

  • Hi, Harshad ,Try to follow these steps.

    • Remove the nodejs and upgrade nodejs manually to version 0.10.
    • Check If you have JDK 1.7 installed, which is not supported by Titanium. Please install JDK 6 (aka 1.6).
    • Try specifying your sdk directory as "/Users/your_mac/Documents/android-sdk-macosx

    Than i restart Titanium IDE and Check for updates by "help/Check for Titanium Updates" and "help/Check for updates". Update.

    Hope it helps. Thanks.

    • The problem is solved. It was the problem with genymotion file. i had to delete the file from this path - /Library/Application Support/Titanium/mobilesdk/osx/3.3.0.GA/node_modules/titanium-sdk/lib/emulators/genymotion.js

      I guess somehow by default titanium configured genymotion(a third party emulator instead AVD)…so by removing that file titanium ignored it and used AVD.

      I guess it's better to configure the android sdk on your own than asking the studio to configure it for you.
