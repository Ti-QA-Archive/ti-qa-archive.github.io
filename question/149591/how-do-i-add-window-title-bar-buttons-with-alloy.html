How do I add Window Title Bar Buttons with Alloy?

I'm not sure if it is called a Window Title Bar, although the tss calls it a "barColor" in the Window element, when I change background color.

I would like to add buttons to this using Alloy. When I use TabGroups with Child Windows I can see the back button, but not able to ad my own buttons.

I've tried to create a completely new navbar, but that doesn't seem to work since I can not turn off the current title bar.

I've tried to add buttons using various element names, but nothing has worked yet.

I can not find documentation on how to do this. I'm sure it is simple but could use some help.

Thanks

    Scrap that. I figured it out right after posting… more more google.

    <Window title="BigWindow">
    <RightNavButton>
        <Button id="button1">Click me</Button>
    </RightNavButton>
    <Label>The Window</Label>
    <Button id="button2">Click me too</Button>
</Window>

    You can add navbar button from index.js file also.

    Like : 

    var back = Ti.UI.createButton({
    title : "back"
});
$.index..leftNavButton = back;

    and you also need to set property to window in index.tss :
    navBarHidden:false
