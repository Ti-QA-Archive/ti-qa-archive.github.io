  • You can create an indicator variable that you check before opening the next window. In psuedocode:

    openedWindow = false

function()
    stuff
    if (!openedWindow)
        openedWindow=true
        openWindow

    And then set openedWindow back to false when the user navigates back to the page or using setTimeout
