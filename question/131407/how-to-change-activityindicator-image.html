  • No Dear ..

    You can't change the image of Activity indicator..
    But obviously you can set two three kinds of Indicator style

    If you want to make custom indicator then you can make by following way

    CODE:::

    var images = [];
for (var i=1;i<9;i++)
{
    images.push('load' + i +'.gif');
}

var imageView = Titanium.UI.createImageView({
    images:images,
    duration:100,  
    repeatCount:0, 
    top:100,
    height:48,
    width:48,
    backgroundColor:'transparent'
});

imageView.addEventListener('load', function(e)
{
    // start animation
    imageView.start();
});

win4.add(imageView);

    Note : You have same kind of images which if all are move frquently that seems to be like an Animation

    load1.jpg
    load2.jpg
    load3.jpg

    Same as 8 images in your resource directory

    That is definitely helps you
