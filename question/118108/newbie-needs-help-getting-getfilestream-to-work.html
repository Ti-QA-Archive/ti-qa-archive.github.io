Newbie needs help getting getFileStream to work

Hi,

I'm a very new Appcelerator user, and I need to read a text file line by line. I can read the file using getFile as in the Kitchen Sink example……

var f = Titanium.Filesystem.getFile(Titanium.Filesystem.resourcesDirectory, 'text.txt');
Ti.API.info('file = ' + f);
var contents = f.read();

However, when I try to use getFileStream, the Android emulator always fails with……

>Type Error: Cannot find function getFileStream in [Ti.FileSystem].

At first I thought this might be a signature error in that I could be feeding the method a wrong data type for the file name, but I've tried everything I can think of (text strings, variables), and I always get the same error No example for using getFileStream is in KitcheSink, and there's little online.

Can anyone tell me how to use this method, or another way to read a text file line by line. The file is in the resources directory, and getFile finds it just fine with the Titanium.Filesystem.resourcesDirectory call, but I cannot make getFileStream to work.

The current code looks like….

var fStream = Titanium.Filesystem.getFileStream(f.nativePath);
    fStream.open(Titanium.Filesystem.MODE_READ);
    var line = fStream.readLine();
    while(line != null) {
        // do something with the line
        line = fStream.readLine();
    }
    fStream.close();

where f is the file as in the first code snippet. I've tried f, f.resolve, f.nativePath and other methods to try to correctly feed getFileStream, but nothing works.

Thanks in advance,

Paul

  • function Read()
    {
    var filename = Titanium.Filesystem.getFile(Titanium.Filesystem.resourcesDirectory, 'Work/AA Sample on 26-7-2011 Data.txt');
    var resources = filename.read().text.split('\n');
    var Data = [];

    for(var i=0;i<resources.length;i++)
{
    var line = resources[i];//do something with line

    var row = Titanium.UI.createTableViewRow();

    var lbldata = Titanium.UI.createLabel({
    text:line,
    font:{fontSize:16,fontWeight:'bold'},
    width:'auto',
    textAlign:'left',
    top:2,
    left:40,
    height:'auto'
    });

    row.add(lbldata);
    Data.push(row);
}

var tableView = Titanium.UI.createTableView({
data:Data
});    

window.add(tableView);
window.open();

    }

  • I'm not aware of a getFileStream() method implementation in titanium – or at least I don't see it listed in the docs or a kitchen sink example. I would suggest starting with reading the entire file into a variable, as in your first code block, then splitting that into an array:

    var f = Titanium.Filesystem.getFile(Titanium.Filesystem.resourcesDirectory, 'text.txt');
Ti.API.info('file = ' + f);
var contents = f.read();
var lines = contents.split('\n');
for ( var i = 0; i < lines.length; i++ ) {
    var text = lines[i];
    // do something here
}
