  • Hi Kiran,
    Have you tried doing like:

    var newWindow = Titanium.UI.createWindow({
    title:'My Window',
    tabBarHidden:true,
    backgroundColor:'transparent',
    orientationModes : [
        Titanium.UI.PORTRAIT
    ]
});

    This should set the orientation mode of the window. I hope this helps you.
