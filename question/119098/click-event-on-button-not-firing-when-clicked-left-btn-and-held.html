Click event on button not firing when clicked left btn and held

for 2 seconds.

I have 2 events attached to this button, click and singletap.

If I do left mouse click and hold down it for 2 seconds, none of the two events fire.

If I do normal speed left mouse click, singltap occurs then click event.

How can I fix this so even if it is held for 2 seconds, event works?

  • It would help if you also say what platform you are developing for. Desktop? (OS?) Mobile? iOS, Android?

    A 'tap' is always going to be short.

    A 'click' will always be a 'normal' single click.

    If you want to detect when a button is released, no matter how long for, use the 'touchend' event.

  • Use 'touchstart' event, inside you call:
    touchActive = true;
    setTimeout(callbackToExecute, 2000);

    callbackToExecute = function()
    {
    if(touchActive)
    {
    // do stuff
    }
    }

    on 'touchend' and 'touchcancel':

    touchActive = false;
