  • The primary focus of the training course is on coding mobile applications with Titanium, not necessarily the tooling used to do so. I'd hardly call that useless.
    Studio does everything that Developer did, and the code doesn't change because the IDE did.

    This does not mean that we are not working on new and updated training tools, but I believe that saying that the current training materials along with the current Studio documentation is of no use would be a bit drastic. We will constantly be working on docs and training and updating everything as we go but the core of how things work won't be rendered useless overnight.

    — answered July 10th 2011 by Anthony Decena
