Titanium.App.Properties.getString() on Desktop app

Hello, I have a problem with getString function in a desktop app.

I would to know if a string for example Titanium.App.Properties.getString(&#39;username&#39;); is already saved and if not I have to save the string with Titanium.App.Properties.setString(&#39;username&#39;, &#39;test&#39;);

How I can see if Titanium.App.Properties.getString(&#39;username&#39;); is empty?

I tried with if (Titanium.App.Properties.getString(&#39;username&#39;) == null){ .... } or if (!Titanium.App.Properties.getString(&#39;username&#39;)){ .... } but this code doesn't work.

It work on mobile environment but not on desktop.

Any help?
Thank you, Stefano
