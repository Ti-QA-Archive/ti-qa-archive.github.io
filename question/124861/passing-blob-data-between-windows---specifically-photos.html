Passing BLOB data between windows - specifically photos

Hello,

I have a multi-layer app that has a page that you drill into multiple windows where data is entered. For all of the data, except images, I have been able to pass the data into a App.Properties setting and then reuse it.

How can I accomplish this same thing with images I've taken. I am uploading them to a server and need to do it in bulk. As a summary here's an over view of how the app flows:

  1. Click on an object
  2. A landing page has 7 windows to which a user will supply information
    2.a. A user selects the Images window
  3. The user takes/adds many images
  4. The users clicks Back to the landing page
  5. The user clicks Save on the landing page
  6. Images are uploaded to a server

I've tried the following methods:

  • Creating a string from the images
    • This does not work because Titanium will not properly encode the strings
  • Using the Ti.App namespace
    • This does not work because you cannot modify objects after you created them
  • Using an Event for the Images window
    • This does not work because you can only pass strings

Any advice/direction would be highly appreciated.
