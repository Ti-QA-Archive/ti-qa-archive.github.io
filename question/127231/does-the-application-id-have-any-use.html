    • You still have not identified the platform(s). In the case of iOS, Apple does require you to register as a developer. This is true whether or not you plan to submit the application to the app store. Even those you only want to install and run on your own private device will need a "provisioning profile" from Apple. To get one you need to be a registered developer.

      However, that is not true of a desktop application.

      So like Trevor said, there are differences and it isn't possible to answer your question accurately without knowing more information.
