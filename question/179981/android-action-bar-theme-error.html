Android Action Bar Theme Error

Titanium SDK 3.4.1
Android device API level 17

Trying to change the App Theme but am not able to run the app. Keep getting an error saying


[ERROR] :  Failed to package application:

[ERROR] :  /Users/ZainEsmail/Documents/Titanium_Studio_Workspace/York Region Student Tools/build/android/res/values/themes_apptheme.xml:2: error: Error parsing XML: XML or text declaration not at start of entity

[ERROR] Application Installer abnormal process termination. Process exit value was 1

In my tiapp.xml I have the following

<android xmlns:android="http://schemas.android.com/apk/res/android">
    <manifest>
        <application android:theme="@style/Theme.LightDarkBar"/>
        <uses-sdk android:minSdkVersion="14" android:targetSdkVersion="19"/>
    </manifest>
</android>

and under platform/android/res/values/builtin_themes.xml I have the following

<?xml version="1.0" encoding="utf-8"?>
<resources xmlns:android="http://schemas.android.com/apk/res/android">
  <style name="LightDarkBar" parent="Theme.AppCompat.Light.DarkActionBar"/>
  <style name="Light" parent="Theme.AppCompat.Light"/>
  <style name="Dark" parent="Theme.AppCompat"/>
</resources>

Why is it not allowing me to run the app?

  • Hi Zain Esmail,

    Looks like there is a typo in your tiapp.xml file. Instead of using, 

    <application android:theme="@style/LightDarkBar"/>

    You are using, 

    <application android:theme="@style/Theme.LightDarkBar"/>

    Modify your tiapp.xml file using the code below and it'll work for sure.

    <android xmlns:android="http://schemas.android.com/apk/res/android">
        <manifest>
            <application android:theme="@style/LightDarkBar"/>
            <uses-sdk android:minSdkVersion="14" android:targetSdkVersion="19"/>
        </manifest>
</android>

    Hope it helps.

    Thanks
