Delaying the splash screen - android

Hi. I want to delay my splash screen for 5 seconds, but I can't seem to do it.
I tried this 

setTimeout(function() {
    tabGroup.open();
}, 5000);

but it didn't work for me.

My app.js code is like this.

setTimeout(function() {
    App.open();
}, 5000);

(function() {
    var App = require('ui/AppWindow');
    new App().open();
})();
— asked February 18th 2013 by Andrei Vincze

    You are opening it in your anonymous function…

    I think you want something like this:

    var App = require('ui/AppWindow');

setTimeout(function() {
    new App().open();
}, 5000);
