    • Use takeScreenshot to grab the image of the screen, in the callback function save the media to a file and use the nativePath of that file to share it with the code in the post I gave you.

      Below is some code I didn't test, but you'll get the idea ;)

      

Titanium.Media.takeScreenshot( function(e){

shot_file = Titanium.Filesystem.getFile(Titanium.Filesystem.externalStorageDirectory,'shot.jpg');

       var shareIntent = Ti.Android.createIntent({
                action: Ti.Android.ACTION_SEND,
                type: "image/jpeg"
            });

            shareIntent.putExtra(Ti.Android.EXTRA_TEXT, "Photo By ScreenShotApp");
            shareIntent.putExtraUri(Ti.Android.EXTRA_STREAM, shot_file.nativePath);
            Ti.Android.currentActivity.startActivity(Ti.Android.createIntentChooser(shareIntent, "Share image"));

})

      You will have to tweak the code to make it work for your case.
