publishing to google store/market for webview loading google map from external URL

I am about to publish one android and ios application where I have used webview that loads google map from the external URL. Do I need to add google map API key for this even I am loading google map from the external source? Because it work well in with apk file and in the emulator.

  • Hi, you should test in device first, before publishing to Google store for android and Apple store for iOS. MAP API key is only required when you use the MAP api directly, so in your case it does not seems mandatory.
