    Hi Suraj Shukla,

    From Android SDK Manager, deselect all Android SDK Build Tools numbered above 20. Restart Titanium Studio after the change made with the Android SDK Manager.

    For more details, check the post titanium.jar not found and https://developer.appcelerator.com/question/178514/after-update-to-340-ga-nothing-works-any-more

    Hope it helps.
