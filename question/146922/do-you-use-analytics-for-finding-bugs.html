Do you use analytics for finding bugs?

How do you use analytics for the purpose of finding any possible bugs after you deploy your app? Is there any way I can log any exception or error that my app faces in the wild?

    You can use analytics for making custom analytics entries, but you need to check for each case that you want to push to analytics. If you want to do crash logs and error reporting, you should look into the TestFlight or similar module in the marketplace.
