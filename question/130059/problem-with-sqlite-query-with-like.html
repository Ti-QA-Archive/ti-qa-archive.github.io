Problem with SQLite query with LIKE

After several hours' try, I feel I need help. I tried to write a query against local sqlite database but it did not work. Sometime it seemed to work depending on which word was searched for in LIKE. I suppose it is a encoding problem, but I am not sure much less how to solve it. Did anyone have similar problem and how is it solved? Thanks for any advice.

SELECT ads.id, ads.name, ads.text, ads.region, ads.type, ads.price, ads.images, ads.published 
FROM ads 
WHERE (ads.name LIKE '%canon%' OR ads.text like '%canon%') 
ORDER BY ads.published DESC 
LIMIT 20 OFFSET 1;

  • Problem solved. It turns out that the problem is the offset in the sql query. I set the start page number as 1 and the first offset becames pagesize * 1. So I did not get anything when the total non-paged result is less than 1 page.

    Thank your again for your help. It help me to trace down the bug.

  • I found a possible answer using \ to escape special characters here.

    I modified your code using their example and I also changed your second like to all CAPS….LIKE.

    Your existing code:

    
WHERE (ads.name LIKE '%canon%' OR ads.text like '%canon%')

    Modified:

    WHERE (ads.name LIKE \'%canon%\' OR ads.text LIKE \'%canon%\')

      The code segment is:

      
this.getAds = function (func, searchKeys, pageSize, page) {
    var db = Ti.Database.open(Titanium.App.db);
    var result, sql, where, sort;

    sql = "SELECT ads.id, ads.name, ads.text, ads.region, ads.type, ads.price, ads.images, ads.published FROM ads";
    if (searchKeys) {
        if (searchKeys.category) {
            where = " WHERE categoryId=" + searchKeys.category;
        }
        else if (searchKeys.parentCategory) {
            where = " JOIN categories cat ON ads.categoryId=cat.id WHERE cat.parentId=" + searchKeys.parentCategory;
        }
        if (searchKeys.words) {
            where = ((where)? (where + " AND"):" WHERE") + " (ads.name like '%" + searchKeys.words + "%' OR ads.text like '%" + searchKeys.words + "%')"; 
        }
    }

    sort = " ORDER BY ads.published DESC LIMIT ? OFFSET ?;";
    sql += (where||"") + sort;
    result = db.execute(sql, pageSize, pageSize * page++);

    try {
        if (!result.isValidRow()) {
            return false;
        }
        else {
            func(result, 'SQLDATA', null);
            return true;
        }
    } finally {
        result.close();
        db.close();
    }
}
