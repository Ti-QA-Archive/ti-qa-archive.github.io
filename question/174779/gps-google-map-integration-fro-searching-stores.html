GPS Google map integration fro searching stores?

I have to create an app which should access the GPS to finds the location of user and have to show the nearest company stores[branches of a company] by using google map.is it possible to create it for both android and ios ? anyone know any sample app or document for achieving it?

    Sample app, no, but I can give you an idea on how I see this:

    1. you need to get the current location of the user ( ti geolocation)
    2. find the stores in a specific radius (500m let's say) - this can be done sending the location to a server that replies with a json with the stores ( the need to have the lat/long coordinates for the next point ) or search in a local db using some geo functions you can easily find
    3. the reply of the search ( server or local) should be an array with the locations of the nearest stores and you will iterate this array to add annotations to the map.

    Makes sense?
