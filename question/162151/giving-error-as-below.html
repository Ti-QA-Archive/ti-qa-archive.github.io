  • Create the following:

    searchdetails.xml, which should be placed in '/app/views/'

    <Alloy>
    <Window id="win">
        <Label id="lbl" text="I am a label"/>
    </Window>
</Alloy>

    searchdetails.js, which should be placed in '/app/controllers/'

    $.win.open();

    Finally you should add '.getView()' when calling your controller

    var con = Alloy.createController('searchdetails').getView();

    Hope this helps!
