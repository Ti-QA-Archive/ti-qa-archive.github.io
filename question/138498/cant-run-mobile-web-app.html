  • Sounds like Titanium Studio got a little confused or something. Titanium Studio has a built-in web server that you can directly access by browsing to http:/127.0.0.1:8020/index.html.

    If you just launched Titanium Studio, you need to run preview in browser for the web server to be initially started. We use the nginx web server and leave it running all the time. nginx is always running, better at streaming media, and can proxy requests to other web servers.

    We discourage using the file:/// method of loading Mobile Web apps since it has not been tested.
