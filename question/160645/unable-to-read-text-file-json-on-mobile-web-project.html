  • Mobile Web has no access to the local file system in the same way your desktop browser has no access to your computer's file system. This is standard browser level security.

    If you need to read the JSON data for your mobile web app, you'll need to download it initially from a network endpoint and save it to LocalStorage. You say you're just testing, so that's probably not the route to go down. I'd suggest you test your code on an Android device or investigate the Genymotion emulator (which is enormously faster than the Google emulator).
