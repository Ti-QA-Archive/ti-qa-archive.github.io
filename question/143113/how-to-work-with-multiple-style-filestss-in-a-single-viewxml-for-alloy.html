    Yes, right now the only way to use multiple files style files for a single view would be to take advantage of the global style, app.tss, applied to all views. If you need more flexibility than this, can you provide a use case that the current functionality could not cover?

    We have more Alloy docs coming before the end of the month, including wiki docs and API doc integrations. Until then there is the readme on the github repository which compliments the quick start. In addition, there's good info and feedback at the alloy google group.
