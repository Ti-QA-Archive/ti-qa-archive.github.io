  • For your purposes using the persistent data maybe a good solution. But please don't use app level variables. If you need a few variables then you can create a module with them in and require it where you need it.

    Have a look here. http://www.thewarpedcoder.net/2011/12/apccelrator-titanium-commonjs-global-variables/

    If your using persistent data then you can create a module which puts and retrieves the data and again require it where needed.

    Hope this helps

    T.
