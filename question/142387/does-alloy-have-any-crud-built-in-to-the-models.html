  • Accepted Answer

    Alloy uses Backbone.js model engine, so it's perfect for JSON based models. Alloy also uses adapters to provide an interface to persistent stores which can be local or cloud based. The adapters wrap the lower level CRUD operations. We currently are expanding our adapters and cleaning up the model and collection descriptors but take a look at the sample in alloy/test/apps/ there are several model sample that will get you started.
