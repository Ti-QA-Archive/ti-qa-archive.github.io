Which iOS libraries are included with the Titanium SDK?

I was developing a custom module where I need some iOS libraries (i.e. libz.dylib) and I got really surprised when I saw the module working without these libraries included. A question comes to my mind:

this could be because the Titanium SDK ships with some libraries?

And, if it's that true: how can I know wich are these libraries in any Titanium SDK version?

    Hi Xurde, I opened the Kitchen Sink X-Code project using X-Code and i found libz.dylib and libsqlite3.dylib already added. so if i understood your question correctly then i would suggest you to look for kitchen sink build to find whether a library or framework is included with Titanium SDK or NOT. i generally do this and update my kitchen sink with new titanium SDK,if i found any update for kitchen Sink.

    alternatively you can also check the Titanium.xcodeproj comes with new titanium SDK.
