  • As Kosso said, you would just set an initial position for your TableView and then animate the left/bottom values to the appropriate position using the built in animate() function.

    Try this, not sure off hand if you can specify negative offsets for positioning:

    var table = Titanium.UI.createTableView({
    height:400,
    width:600,
    left:-600
});

    then in your button click handler:

    table.animate({left:0, duration:500});
