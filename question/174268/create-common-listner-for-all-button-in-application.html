  • Hi

    Short answer - no

    Longer answer,

    I do not think this is your actual problem;

    • I want to write click eventlistner for all buttons in the application.

    I suspect you have a different problem that this appears to be the solution as far as you can tell?

    Thoughts;

    • If you are trying to remove every time you are typing addEventListener then not a good enough reason to progress
    • If you have some common functions that ideally could be handled by a generic handler - that is fine - good enough reason
      • Create a CommonJS module and place all the handler logic inside
      • In your Event listeners add
    btn.addEventListener('click', require('yourhandler').coolhandler);
    • If you want to have a group of buttons (or anything you add) handled once even if there are several buttons etc
      • Create an parent view
      • Add each button the this view
      • Add a custom property to each button
      • Add the event listener to the parent view
      • use e.source.customproperty to determine what you want to do for these specific buttons
    • If you want to handle all buttons within one window then;
      • Add buttons with a custom property
      • Add the event listener to the window
      • use e.source.customproperty to determine what you want to do for these specific buttons

    Always many solutions to problems

  • Another solution is to create your own custom button widget that wraps the required behavior and only exposes the functionality you need to the rest of your application. In a lot of our apps we usually create widgets like form-button, form-button-group, form-text-field, etc…they are very generic and can be styled at the app.tss level.
