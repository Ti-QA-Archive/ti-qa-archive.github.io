Is it possible to have click-listener on an element which is included in another element that already has click-listener?

I have a button w/in tableViewRow, the row has click-listener, which opens a new window w/in the same tab. Can I have a listener on the button, that would allow me to do some data manipulation & not be disregarded due to the row-listener?

  • I guess what you will be looking for is a clickName,

    When the tableView click event is fired, you can add clickNames to the row and button.

    So in the eventListener you check if the clickName is the button clickName, and if not you do your normal code for an click row.

    For the code, look at the Kitchensink, go to views > tableViews > tableView ( layout 2 )

    a sample from the example:

    tableView.addEventListener('click', function(e)
{
    Ti.API.info('table view row clicked - source ' + e.source);
    // use rowNum property on object to get row number
    var rowNum = e.index;
    var updateRow = createUpdateRow('You clicked on the '+e.source.clickName);
    tableView.updateRow(rowNum,updateRow,{animationStyle:Titanium.UI.iPhone.RowAnimationStyle.LEFT});
});
