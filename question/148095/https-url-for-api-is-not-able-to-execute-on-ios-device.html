  • Hi

    Obviously this is a difficult question as without knowledge of the service you are using and access to test it, anything offered will be guesses only.

    So here go some guesses;

    • Can you test any of the web services in a browsers, something simple so it returns back valid data? If so - try to run that same test on the devices built-in browser (Safari in this case).
    • Is the service a public web service or private?
      • If private; If it is working in the simulator but not on the device, then it may be network settings on your computers local connection that are not enabled on the device.
      • If public; Can you try on a different device?
      • If Public; Does your WiFi connection have different privileges to your LAN?
    • Test against a different SSL connection, does this work?
