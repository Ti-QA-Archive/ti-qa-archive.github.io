How to do Scrollableview from left to right?

Developing an application in Arabic language.

For Arabic layout is changing text alignment from right to left. Actual scrollable function is right to left. So we need to scroll from left to right for scrollable view.

    After you create and set the views set the currentPage to the last view.
    This way from the Arabic point of view the scrollview will be at the start but from the non-arabic point of view will be at the end.
    Makes sense?

    Of course you will have to set the views in reverse order to keep the normal flow for Arabic.
