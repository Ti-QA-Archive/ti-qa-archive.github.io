NappDrawer module drawer overlaps with status bar on iOS

Hey all,

I am using the NappDrawer module https://github.com/viezel/NappDrawer to create a sliding menu,
I am on Titanium SDK 3.2.2GA and the latest version for module.
The issue that I am facing is that when the drawer opens it overlaps with the status bar of iOS.
I have set the following as suggested by the developer of the module in tiapp.xml
