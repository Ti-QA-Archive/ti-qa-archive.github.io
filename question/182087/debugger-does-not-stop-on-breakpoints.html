Debugger does not stop on breakpoints

In one of my projects the debugger does not stop on breakpoints.
I have tried the same project on two installations of Titanium that other projects work but the result is the same.
I don't get any suspicious messages. I have tried deleting all breakpoints, cleaning, restarting etc but nothing.

  • I have now discovered that if I create a new project and import my source files to the new project without overwriting the application.js files that are generated by appcelerator debuging works. So I think the problem occured due to my modifications to those files.
