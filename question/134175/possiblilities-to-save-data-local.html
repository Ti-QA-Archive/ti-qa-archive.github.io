But when closing the app that means that my data is gone. Is it possible to get the data stored and access-able after closing the app?

  • For XML data (rss feeds, etc) we cache the xml locally, then access the local file for parsing. This permits data to be immediately available to the user on application restart. While this data is 'stale' until refreshed, it offers a better experience, we believe, to the end user.
