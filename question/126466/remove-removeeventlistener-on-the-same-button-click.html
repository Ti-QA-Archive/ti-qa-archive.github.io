  • Why would u want to do this? Is to prevent a second click to execute?
    If yes set a flag to the button itself and check it on the click event

    the_btn.addEventListener('click', function(e){
    if (!e.source.click_done ) {
        //do your stuff
        e.source.click_done =  true;
    }

})
