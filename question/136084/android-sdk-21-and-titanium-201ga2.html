Android SDK 2.1 and Titanium 2.0.1.GA2

Hi everyone, I have to make a project compatible with Android 2.1 and later. But when I tried to install the App in a 2.1 phone, I got this error:

[ERROR] Failed installing com.iphoideas.neoAuto: pkg: /data/local/tmp/app.apk
Failure [INSTALL_FAILED_OLDER_SDK]

I developed everything with the 2.2 Android Emulator and a Galaxy S (that also has Android 2.2), but when I got this error trying to install it on that phone, I tried to set the Run Configurations to use the 2.1 Emulator, but I got an error that said that V8 couldn't be used, so I used Rhino and the emulator started running. But I don't how I can change de target SDK version (I tried in Preferences - > Titanium Studio - > Titanium, but didn't get a chnace)

Does anybody knows a way of using Titanium 2.0.1.GA2 and Android 2.1 ???

Regards!

    • Nick is correct. Be sure to check the Titanium Compatibility Matrix for more information.

      Because it is android, you could pull an older sdk and build/submit your application, with the understanding that there might be some bugs in the sdk.

      — commented April 26th 2012 by Stephen Feather

    • Agree with Stephen but be warned that if you go below version 1.7.x then there are some nasty memory management bugs in the Android implementation that have been since much improved in later versions.
