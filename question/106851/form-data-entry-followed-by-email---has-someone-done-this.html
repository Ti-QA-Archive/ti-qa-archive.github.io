    • Construct your screen with three controls to allow the user to input the minimum fields required to send a valid email. I think they are, "to", "subject" and "body". You could simply use two textFields and a TextArea respectively.

      Use KitchenSink example lines #16 - #60 inside a button click event (probably with the title of "Send"). Pass the values of the above text input fields to the setToRecipients(), setSubject(), setMessageBody() methods of the email dialog.

      This should work. Hope it helps
