Is there a way to resize Image buttons for iPhone/Android?

Basically what I'm doing is building an app for both iPhone and Android, and on the pages there's a button that fits the width of the iPhone screen (about 320px wide), and when I try to run the item in the Android emulator the image doesn't scale and ruins the layout.

Is there a way to have the image automatically scale to the screen size of the device? I don't want to enter a manual size for each android version with a different resolution…

    Hi Marco,

    If I understand correctly, you want to know how to get the device's current resolution, so you can adjust your UI elements accordingly?

    You can do this by reading up about Ti.Platform.displayCaps.* from the following URL: http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.Platform.DisplayCaps-object.html

    In this case, to get the width of the current device, you have to use the following:

    var width = Ti.Platform.displayCaps.platformWidth;

    Hope this helps.

    • This will simply give you the device's screen dimension. For example, let's say you run this on a 480x800 Android device, it will give you 480 back for Ti.Platform.displayCaps.platformWidth. If you run the same app on a 540x960 Android device, it will give you 540 back.

      It will most likely simply stretch the image, if you don't do any extra checking/processing..

      If you want to keep the 1:1 ratio, you will need to do this manually i.e. by creating a function that checks the device's current screen resolution and adjusting the width/height of the image button accordingly.

      Hope this helps.
