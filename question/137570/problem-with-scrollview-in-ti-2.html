Problem with scrollView in Ti 2

Hi,
I have a problem with Ti 2.01 GA2 and that works fine in 1.8.2.
My scrollview should display a lot a small pictures and it should fill the screen. No problem in 1.8.2 and in 2 I have only an horizontal "stripe" of picture that go beyond the iPad screen.
I use this code

albumView = Ti.UI.createScrollView({
        layout:'horizontal',
        left:0,right:0,
        top:0,bottom:0,
        contentHeight:"auto",
        showVerticalScrollIndicator:true
    });

And for adding the thumbnail I use this code :

var container = Ti.UI.createView({
            width:thumbsize, height:thumbsize + 20,
            left:margin,right:margin,
            top:margin, bottom:margin
            //layout:'vertical'
        });


        var image = Ti.UI.createImageView({
            image: imageFile,
            width:thumbsize, height:thumbsize,
            left:0,top:0,
            id:pos
        });


        container.add(image);


        var label = Ti.UI.createLabel({
            text:listName[pos],
            width:thumbsize, height:18,
            left:0,top:thumbsize + 7,
            shadowColor:'#777',
            shadowOffset:{x:1,y:1},
            color:'white',
            textAlign:'center',
            font:{fontWeight:'bold',fontSize:12}            
        });

        container.add(label);

        albumView.add(container);

I have tryed to replace the contentHeight:"auto" by Ti.UI.Size but same problem.
Any idea ?

Regards

    Ti.UI.Size is only for view sizes, not for scrollView content

    So you want to create a kind of thumbnail grid with your images, is correct?

    Not tested, but try this

    
albumView = Ti.UI.createScrollView({
        layout:'horizontal',
        left:0,right:0,
        top:0,bottom:0,
        contentHeight:"auto",
    contentWidth:'auto',
        showVerticalScrollIndicator:true
    });

//this is a big container that will contain all thumbnails. The horizontal layout will receive and allocate all images in line. when a line is completed, will create a new line
var thumbsContainer = Ti.UI.createView({
        layout:'horizontal',
        height:Ti.UI.SIZE,
        top:0, left:0
        });

albumView.add(thumbsContainer);

    And for adding the thumbnail :

    var container = Ti.UI.createView({
            width:thumbsize, height:thumbsize + 20,
            left:margin,right:margin,
            top:margin, bottom:margin
            //layout:'vertical'
        });


        var image = Ti.UI.createImageView({
            image: imageFile,
            width:thumbsize, height:thumbsize,
            left:0,top:0,
            id:pos
        });


        container.add(image);


        var label = Ti.UI.createLabel({
            text:listName[pos],
            width:thumbsize, height:18,
            left:0,top:thumbsize + 7,
            shadowColor:'#777',
            shadowOffset:{x:1,y:1},
            color:'white',
            textAlign:'center',
            font:{fontWeight:'bold',fontSize:12}            
        });

        container.add(label);

        thumbsContainer.add(container);
