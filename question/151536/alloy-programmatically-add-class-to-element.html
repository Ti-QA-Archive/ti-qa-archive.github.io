  • Accepted Answer

    Are you looking to do runtime assignments of classes? Alloy's TSS is unfortunately not the same as HTML+CSS. With Alloy, your XML, TSS, and controller code is parsed prior to the Titanium compile step. That Alloy parsing stage creates a "traditional" project, which is then compiled to the target platform by the build scripts. At runtime, all the Alloy stuff is gone. There are no style sheets, classes, etc. to dynamically access and apply.

  • Hello Bryce,

    Refer this Link :

    http://docs.appcelerator.com/titanium/latest/#!/guide/Alloy_XML_Markup

    i.e :
    //index.xml

    <Alloy>
    <Window class="container">
        <Button id="button" onClick="doClick"/>
        <TextField id="tf"></TextField>
    </Window>
</Alloy>

    //index.tss

    TextField: {
    borderStyle:Ti.UI.INPUT_BORDERSTYLE_NONE,
    left: 5,
    borderColor : 'transparent',     
    width: Ti.UI.FILL,
    font: {fontSize: '12sp'}
},
".container": {
    backgroundColor:"white"
},
"#button": {
    width: "100dp",
    height: "45dp",
    title:"Click me"
}
