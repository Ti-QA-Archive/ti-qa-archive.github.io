Styling TableViewSection "Titanium.UI.iPhone.TableViewStyle.GROUPED"

I'm trying to style the header title of a TableViewStyle.GROUPED section. It seems that any color i put in the tss file it just seems to stay the default color. Any way around this?

<TableViewSection id="MedicalConditions" headerTitle="Weight" >

</TableViewSection>

  • Hi

    This one is an easier answer than you may think.

    You are using headerTitle, which uses the default display method of iOS to render the title Weight you have entered. Using headerTitle is when you are happy with the default look, however, you appear not to be.

    You need to use a headerView INSTEAD of headerTitle (cannot both be used at the same time. Inside this view add any label or other view in anyway you wish with any colour and images choice you like.

    The only thing you have to be aware of is that section headerViews require their height attribute to be an actual value (not Ti.UI.FILL, Ti.UI.SIZE, AUTO).

      <Label id="GoalWeighInformation">Weight loss</Label>
      <View class="chevron-right"/>
      </TableViewRow>
      <TableViewRow class="whiteBackGround" title="Goal weight">
      <Label id="GoalWeighInformation">10st 0lbs</Label>
      <View class="chevron-right"/>
      </TableViewRow>
      <TableViewRow class="whiteBackGround" title="Weight loss rate">
      <View class="chevron-right"/>
      <Label class="otherConditionsInformation">2lbs per week</Label>
      </TableViewRow>
      </TableViewSection>
