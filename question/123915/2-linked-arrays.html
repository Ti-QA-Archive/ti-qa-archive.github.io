  • Accepted Answer

    Why not just make this an array of objects?

    var tel_array = [{
    name: "Joe Blogs",
    title: "Mr Somebody"
}, {
    name: "Person 2",
    title: "Mr Somebody2"
}];

    Then in your event listener loop through the objects:

    for (var i in tel_array){
    new_window.name = tel_array[i].name;
}
