    Parsing XML documents is a non-trivial topic, and one that is not specific to Titanium. I suggest you explore some of the various tutorials posted on the web. In general, you'll use the functions in the Titanium.XML namespace.

    In your case, you're going to have to select the firmName tags (with the getElementsByTagName() method), loop through that array to find the child node of the type you want, and probably loop through those.
