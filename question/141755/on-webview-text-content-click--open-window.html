on webview text content click -open window

HI,
Here i have create one webview which have some html text content :
var webView =Titanium.UI.createWebView({
top:0,
backgroundColor:"transparent",
height:"auto",
width:100,
html:<html><body><p>here are some <b>Example</b> text</p></body></html>,
left:0,
});

window.add(webView);

now on Example text(in html text content) click i want to open new window which have js file as url so can any one help out me for it.?
Thanks in advance

  • You can create a link inside the html that fires an app level event back to the app window. Ti.App.fireEvent. Then in your app window, just listen for the event that you fired off and open the new window. Check out the doc article on Communicating Between Webviews and Titanium
