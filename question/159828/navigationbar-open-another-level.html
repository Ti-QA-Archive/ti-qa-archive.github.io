NavigationBar: open another level

I'm struggling on open another second level in an existing navigationbar.

Out of the root of the navigationbar I open a TableView. But how can I open from the TableView the child in another next level inside the NavigationBar?

index.xml

<Alloy>
    <NavigationWindow id="win1" platform="ios">
        <Window id="win2">
            <Button id="button" onClick="openTableView">TableView</Button>
        </Window>
    </NavigationWindow>
</Alloy>

index.js

function openTableView(e) {
    var win3 = Alloy.createController('tableView').getView();
    $.win1.openWindow(win3);
}

$.win1.open();

tableview.xml

<Alloy>
    <Window id="win3" backButtonTitle="back">
           <TableView id="tableview"></TableView>
    </Window>
</Alloy>

tableview.js

var tbl_data = [];
for(var i=1,j=25; i<j; i++){
    tbl_data.push({title:"Row " + i, hasDetail:true});
};

$.tableview.setData(tbl_data);

I think I need to go through the click event of the TableView in the tableview.js. But I like to keep the navigation which automatically takes care of the navigation.

    Hi

    Common problem and one easily solved using a mechanism built into alloy.

    index.js

    function openTableView(e) {
    var win3 = Alloy.createController('tableView').getView();
    $.win1.openWindow(win3);
}
// cache reference to nav window
// respect globals by NOT referencing everything 
// keep for those times nothing else will solve
Alloy.Globals.nav = $.win1; 
$.win1.open();

    tableview.js

    $.tableview.addEventListener('click', function (e) {
    var winX = Alloy.createController('anotherview').getView();
    Alloy.Globals.nav.openWindow(winX);
});

$.win1.open();
