    • No problem simply change the named locations you have in the URL to lat/lon pairs separate with a comma, example;

      • https://maps.google.com/?daddr=51.5072,-0.1275&saddr=52.4831,-1.8936&sensor=false
      • See this in your browser to test; route

      Based on the code you found;

      button_route.addEventListener('click', function(evt) {
    var link = "https://maps.google.com/?daddr=51.5072,-0.1275&saddr=52.4831,-1.8936&sensor=false";
    Ti.Platform.openURL(link);
});
