emailDialog and HTML File. Can it be done?

Hi. Is there a way to have emailDialog.messageBody use an html file? So it would be something like this:

emailDialog.messageBody URL: mail.html

THANKS.

  • Yes:

    When you open your dialog use: eg:
    ~~
    var emailDialog = Titanium.UI.createEmailDialog({html:true});
    // then
    emailDialog.messageBody = the_contents_of_your_html_file;

    ~~

    hope this helps ;)
