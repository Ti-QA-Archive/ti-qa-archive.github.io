Android GPS increase accuracy

Hi, I am trying to increase the accuracy of GPS in Android to within 5-10 meters.

I first used Ti.Geolocation.Android.accuracy = Ti.Geolocation.ACCURACY_HIGH but this seems to think that 40 meters of real world movement is only 10 meters.

To improve this, I have been experimenting with manual mode. However, when using the code below the event listener fires infrequently and jumps from 10 meters to 150 meters. Has anyone had success customizing these settings to improve accuracy for Android?

Please note I am using the event listener - NOT getCurrentPosition. I greatly appreciate the help.

     Titanium.Geolocation.Android.manualMode = true;

     gpsProvider = Ti.Geolocation.Android.createLocationProvider({
     name: Ti.Geolocation.PROVIDER_GPS,
     minUpdateTime: 10,
     minUpdateDistance: 5
     });
     Ti.Geolocation.Android.addLocationProvider(gpsProvider);

    I set minUpdateTime and minUpdateDistance to zero, and I'm getting updates every second or so. Have you set any filters? They can block aka filter your updates and prevent your app from seeing them.

    • Hi Mark, that did the trick. I removed the filters, set attributes to 0 and it is working. Thanks!
