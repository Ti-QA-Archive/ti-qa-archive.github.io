Is there any way to do a local badge push when app is closed?

Let's say I have an app that at a certain time every day pushes a notification (every day at noon). Is this possible if the app is closed? Doesn't rely on any remote data at all, just need the app to notify the user at the same time every 24 hours. Is this possible?

  • Yes. On iOS, use BackgroundServices. 

    
var service = Ti.App.iOS.registerBackgroundService({url:'/app/notifications.js'});

    notifications.js will launch when user close the app. API is here
