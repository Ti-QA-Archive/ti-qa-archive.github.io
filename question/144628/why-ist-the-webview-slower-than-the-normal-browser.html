Why ist the WebView slower than the normal browser?

Hello,

i try to use a webview. i have a website with many images and some javascript. in the stock android browser it run very fast, with smooth scrolling. no perfomacne problems at all. if i load the same site from a webview scrolling is very slow.

how can this happen? i think the webview use the same engine as the stock brwoser?

thank you

  • On the iOs side is something Apple did, they didn't include in the webview the latest improvements they did in the browser ( like JIT, etc) being concerned for the security or not to allow phonegap-ish apps to run too fast :)

    On the android side, the webview is "linked" to the android app layercause it needs to make callbacks and because of this is slower.
