    Hi

    Firstly make sure you are using Google Maps v2 (the older Ti.Map is no longer supported);

    • Modules.Map
    • Google Maps v2 for Android

    The only reason you would get a blank grid using Google Maps v2 is if the api key and associated values are not set correctly. You say you have copied the values to the tiapp.xml, but have you made sure the API console has the correct value in it for your app id?

    If you have an app id of;

    com.yourcompany.yourapp

    And the SHA1 value looks (something) like this;

    BB:0D:AC:74:D3:21:E1:43:67:71:9B:62:91:AF:A1:66:6E:44:5D:75

    Then you have to combine the two with a semi-colon ; between them on a line of their own;

    BB:0D:AC:74:D3:21:E1:43:67:71:9B:62:91:AF:A1:66:6E:44:5D:75;com.yourcompany.yourapp

    This is the thing many people do not check.

    If this still does not work - make sure you have followed ALL the SHA1 steps in this guide;
