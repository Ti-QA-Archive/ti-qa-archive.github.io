  • Accepted Answer

    The last error is caused because you created a second tabGroup,

    You need to find a way to not use a second tabGroup. Replacing your current tabs with new tabs in the same tabGroup is a solution.

    The white page is caused because you open a window, and in that window you open a new tab. Thus the opened window contains nothing. A tabgroup cannot exicst in a window, so its opening itself in a new context. So when you press back you go back to the empty window.
