Camera Overlay on Android

I'm looking at the latest (or almost latest) KitchenSink using the latest Studio SDK. I noticed that the camera overlays in camera.js are only allowed on iOS, not on Android.

Does anybody know why, specifically, the camera overlays do not work on Android? Could they be made to work? I tweaked camera.js so the camera overlay menu items would appear on Android. I installed KitchenSink on a first generation Xoom tablet, and indeed the overlays fail with some sort of unexpected error.

So, it appears that the camera overlays do not work using Titanium on Android. This is really disappointing as my app really needs camera overlay functionality for Android.

Are camera overlays possible with Android using Titanium, at all?

Thank you!!

    • here is some code from an old project (backfacing camera, android with overlay):
      nothing special. But you have to rotate the image by 90deg because the camera should open in landscape mode. And the camera controls are missing when using an overlay (if I remember correctly):

      var overlay = Ti.UI.createView({
    backgroundImage : "image.png", left : 0, top : 0, right : 0, bottom : 0, touchEnabled : false
});

Titanium.Media.showCamera({

        success : function(event) {
        }, cancel : function() {
        }, error : function(error) {
        }, mediaTypes : [Ti.Media.MEDIA_TYPE_PHOTO], overlay : overlay, showControls : false, autohide : true, saveToPhotoGallery : false
    });
