More 64-bit blues

Needing to use the Titanium Volumeslider for an existing app and finding no 64-bit version I thought I'd have a crack at downloading it from GitHub and rebuilding the module in Xcode using instructions from elsewhere in this forum. I remembered to change the path in the titanium.xcconfig to point to the version of the Titanium SDK I am using and to ask Xcode to update the project to the recommended settings. So far so good.

But on building I immediately get an error that TiViewProxy.h isn't found. Yet it is located at 3.5.0.GA/iphone/include/ as set in titanium.xcconfig. I did notice that since the Titanium Volumeslider module was last updated in 2011 that the former folder 'TiCore' being pointed to in titanium.xcconfig is now called 'JavaScriptCore' so have changed my titanium.xcconfig accordingly. Yet the Xcode build still fails.

Has anybody seen able to successfully build the Titanium Volumeslider module or can give me tips for doing so? It would be a great help.

Cheers.
