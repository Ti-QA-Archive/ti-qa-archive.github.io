Button layout

Hello. I am a newbie. This is my first time creating an Android apps.
On my layout,I have 3 buttons and i want to arrange them on top of the page on one line. How can i do that?
Thank you. :)

    there are a few ways to do this, depends on your code and the result you expect. But a simple way would be using top and left properties

    
var b1 = Ti.UI.createButton({
    top:0,
    left:0,
    height:30,
    width: 60,
    title: 'b1'
});

var b2 = Ti.UI.createButton({
    top:0,
    left:70,
    height:30,
    width: 60,
    title: 'b2'
});

var b3 = Ti.UI.createButton({
    top:0,
    left:140,
    height:30,
    width: 60,
    title: 'b3'
});

var win = Ti.UI.createWindow();

win.add(b1);
win.add(b2);
win.add(b3);

win.open();
