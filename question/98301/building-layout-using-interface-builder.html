Building layout using Interface Builder

I am a visual guy, so I need to be able to layout my labels and textfields using a WYSIWYG editor.

What I've been doing so far is using Interface Builder for layout to get the x/y values. It works fairly well, although I've noticed that where IB says x/y should be, it appears slightly off after Titanium compiles.

That got me thinking if it was possible to build the layout in IB and then somehow, export the XIB into a JS file.

Anyone see a possibility in this or perhaps it has (fingers crossed) already been done??

  • I'm working on a xib file parser / renderer in coffeescript ( which compiles to regular javascript, running titanium )

    I actually got as far as implementing UILabel, UIView, UIButton, UIImageView, UIScrollview, with support nested subviews, background colors, font size, font family, font color, top,left,width,height etc

    you just pass it a regular xib file path and it renders and attaches all the views, and even gives you reference to them using their UI builder labels ( some magic using the xib objectID was required for this)

    this only took a few hours.

    I feel like UIBuilder is the big missing piece to the tititanium development environment. i am sure with their purchase of aptana studio, they are building something similar.
