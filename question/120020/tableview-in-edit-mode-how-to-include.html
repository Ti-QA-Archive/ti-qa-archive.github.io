Tableview in edit mode: how to include

There are several Apps around, where a tableview in edit-mode is extended by an additional line containing a plus-icon an a field to add a new tableview-item.

Example:

I searched my a.. off but still could't find any hint how to achieve this behaviour. At the moment my code looks like this:

// add table view to the window
Titanium.UI.currentWindow.add(tableView);

// add edit categories button
var btnEditCategories = Ti.UI.createButton({
    systemButton: Ti.UI.iPhone.SystemButton.EDIT
});

btnEditCategories.addEventListener('click', function(e){
    tableView.editing = true;
    Titanium.UI.currentWindow.rightNavButton = btnEditCategoriesDone;
});

// add edit categories done button
var btnEditCategoriesDone = Ti.UI.createButton({
    systemButton: Ti.UI.iPhone.SystemButton.DONE
});

btnEditCategoriesDone.addEventListener('click', function(e){
    tableView.editing = false;
    Titanium.UI.currentWindow.rightNavButton = btnEditCategories;
});

Titanium.UI.currentWindow.rightNavButton = btnEditCategories;

This works so far, but i want to extend it by the additional line mentioned above…

Any ideas out there? Any help would be pretty appreciated

  • I'd say you probably won't be successful doing this: Based on my working with tables in edit mode, and the scope of functionality demonstrated by the Kitchen Sink app.

    My suggestion: Achieve the same goal via a different UI. E.g., an "add subcategory" button below the table which displays a temporary modal window. Then, in the future, come back to this if you have time.
