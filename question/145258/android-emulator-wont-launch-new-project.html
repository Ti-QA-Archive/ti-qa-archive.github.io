  • Hi

    To rule things out can you try these in order.

    Clean Project

    • Close emulator if running
    • From Titanium Application; Project > Clean - choose your project
    • Wait for the project to clean - may only show in status bar
    • Try running the app again.

    This will remove any old files in the build folder, which is often a cause for this type of error in my experience.

    Brand new Empty Project

    • Create a brand new project, choose as basic a choice as possible
    • Match the emulator settings
    • Do not add ANY additional code
    • Try running the empty project

    This will test if your project has something that is causing the issue, as this new project will have nothing related to your project.

    Try a different AVD

    • Choose Run Configurations
    • Locate and select the Titanium Android Emulator entry for your app
    • Choose Google APIs Android 2.3.3 instead of the Intel one

    You have chosen an Intel AVD, what happens with an AVD using ARM

    Please try all of these and report back what happens for each.
