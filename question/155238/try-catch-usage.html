  • No. Excessive memory allocations cannot be caught. the OS runtime will kill off any App that has exceeded it memory 'budget'. You need to address the reason why you are consuming so much memory. Without an easily reproducable use-case that demonstrates your issue, it is impossible to offer any useful advice.
