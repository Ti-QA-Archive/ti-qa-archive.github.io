  • math.random() will always use the current time as its "seed" to generate randomness. If this isn't generating random enough numbers for you, you'll likely have to find a more complex method to generate (pseudo) random numbers.

    • The functionality of the the Math.random() is part of the Javascript language, of which Appcelerator has no control. Also, for reference, despite what some sources cite, there is no "seed" parameter for the Math.random() function, which is why you may get duplicate values.
