Call function in module using commonjs

Hi all,

this is my file function_module.js

function myfunction(){

var one = 1;
var two = 2;

var tot = (one + two);

alert(tot);
}

exports.myfunction = myfunction;

In my window1.js

   var function1 = require('ui/function_module');
   var functionrOne = new myfunction();

But doesn't work; what is the correct way for call external module inside a module?

    function myfunction(){

alert(tot);

module.exports = myfunction;

    first.js

    var function1 = require('next');
   var functionrOne = new function1();

  • Hi

    if you are going to reuse the function use this:

    var function1 = require('ui/function_module');
function1.myfunction();

    or you can call it in one line like so:

    require('ui/function_module').myfunction();
