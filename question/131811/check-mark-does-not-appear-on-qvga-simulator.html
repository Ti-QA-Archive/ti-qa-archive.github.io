Check Mark Does Not Appear On QVGA Simulator

I am creating an Android App using build 1.8.0.v2. I have written code that will cause a standard check mark to appear in a tableview row when clicked and than disappear when clicked again. It works properly on screens 320x480 and larger in the simulator but the same code on the 240x320 QVGA does not display the check mark. It does, however, catch the click event and assigns the proper boolean value to the hasCheck property, but than the check mark just doesn't show up.

I am not using a label in the tableview row, I am simply assigning a value to the title property of the row. Here is my code for handling the row clicks.

categoryListview.addEventListener('click', function(e)
{    
    if(e.row.hasCheck)
    {
        e.row.hasCheck = false;
        notify('click false');
    }
    else
    {
        e.row.hasCheck = true;
        notify('click true');
    }


    modifySavedCategories(e.rowData.title);
});
