  • Have you tried downloading from http://docs.appcelerator.com/titanium/latest/#!/guide/Reverting_to_a_Previous_Version? This would be the official link, not the old "guides2" link. If it's still failing, please file a Jira ticket so that we can fix the link (make sure to specify which version you're trying to download).

    The command given by Mridha will install a particular version of the Titanium CLI, not Studio. It also doesn't download the associated SDK.

    # install a specific CLI version
[sudo] npm install -g titanium@<version_number>

# install a specific SDK
ti sdk install <version_number>

# install a specific version of Alloy
[sudo] npm install -g alloy@<version_number>

    With any of those, omit the version number (and @ symbol) to install the current release version.
