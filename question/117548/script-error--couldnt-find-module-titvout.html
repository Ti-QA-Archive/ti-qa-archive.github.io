Script Error = Couldn't find module: ti.tvout

I'm trying to get the ti.tvout module working and I keep getting this a script error indicating that ti.tvout could not be found. I am using Titanium Mobile SDK 1.6.1 with iPhone 4.2 targeting the iPad.

I downloaded the module put the modules folder into my User&#x2F;Library&#x2F;Application Support&#x2F;Titanium&#x2F; directory.

I also added the module node:

<modules>
<module version "0.1">ti.tvout</module>
<modules>

I then created a new project and replaced the app.js file with the one from the examples folder in the module folder.

Everytime I try to run on emulator, it gives me the red error screen with: COULDN'T FIND MODULE: TI.TVOUT

Any suggestions? I also tried deleting the build folder and recompiling with no success.

    Try to put the unzipped module in

    /Library/Application Support/Titanium/modules/iphone

    The Library under the root not your user.
