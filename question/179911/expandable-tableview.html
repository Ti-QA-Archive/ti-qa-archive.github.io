Expandable TableView

  • Create the tableview rows that can be expanded as table view sections. Set your 'row' text in the table view section header. When you click on the header, append the rows to the tableviewsection. When you click it again, remove them.

    Something like: 

    var table = Ti.UI.createTableView({
 // some properties
});
var rows = [];
rows.push(Ti.UI.createTableViewRow({
    // your wanted properties.
});
// add some more if you want

var section = Ti.UI.createTableViewSection({
    headerTitle: "An expandable row",
    open: false,
    // or headerView
});
table.addEventListener("click", function(e) {
    if(e.section) {
        if(e.section.open) {
            _.each(rows, function(row) { e.section.add(row); });
        } else {
            _.each(section.getRows(), function(row) { table.deleteRow(row); });
        }
    }
});
