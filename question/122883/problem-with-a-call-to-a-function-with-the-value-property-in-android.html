  • The problem is that you try to read property from variable that is not object (variable is actually "undefined" as error message says).

    So …

    var obj = { prop : 1 };
Ti.alert(obj.prop); //OK
var notAnObject = undefined;
Ti.alert(notAnObject.prop); //ERROR

    Quick fix is to initialize variable if it is not defined. Something like this:

    motPasse = motPasse || {};
