AdHoc Distribution always builds old version of code (code changes are ignored)

Hello, I have the problem that "suddenly" Titanium Studio started to ignore my code changes in my Alloy project when doing "Distribute AdHoc/Enterprise". It always builds the same old version of the app.

If I start the iOS Simulator, then all changes are there, i.e. the app is correctly built with the latest code version.

Does anybody have an idea what could be the reason? Is there any location where old builds are stored?

Could this have something to do with my latest update of Alloy to version 1.3.0 (using: sudo npm install -g git://github.com/appcelerator/alloy.git)?

Titanium SDK is 3.1.3GA
