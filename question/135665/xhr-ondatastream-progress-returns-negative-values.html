xhr ondatastream progress returns negative values

This is bugging me, why do I get negative values for ondatastream - and nothing for onsendstream when I am trying to send and recieve some data to/from my server. Data comes and goes OK, but would be nice to have feedback for user.

cut-down code:

var xhr = Ti.Network.createHTTPClient({
    enableKeepAlive: false
});
xhr.onsendstream = function(e) {
    Ti.API.info('send progress ' + e.progress);
};
xhr.ondatastream = function(e) {
    Ti.API.info('receive progress ' + e.progress);
};
xhr.open("POST", config.server);
xhr.onerror = function(e) {
    Ti.API.error("API ERROR " + e.error);
};
xhr.onload = function(e) {
    Ti.API.info('Response '  + this.responseText);
};
xhr.send(params);// POST data

Here's what I see…

04-19 07:10:59.203: I/TiAPI(21737): receive progress -945
04-19 07:10:59.210: I/TiAPI(21737): receive progress -5041
... more of the same with increasing negative nums
04-19 07:10:59.671: I/TiAPI(21737): receive progress -98872

Any ideas?

Ti SDK 2.0.1.GA2 - Android V8

Tried on: Android Galaxy Tab with 2.3.6, Galaxy Tab with 3.2, plus Galaxy S2 on 2.3.6
