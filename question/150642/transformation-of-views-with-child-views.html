transformation of views with child views

Platform: ios
SDK: 3.0.2

working example

    var animation3 = Titanium.UI.createAnimation({
        //transform: transform3,
        duration: 500,
        width:200,
        height:190,
        curve: Titanium.UI.ANIMATION_CURVE_EASE_IN_OUT
      });

    priceView.addEventListener('click', function(){
        priceExplainView.show();
        priceExplainView.animate(animation3);
    });

So my view "priceExplainView" is a normal view, that contains 4 imageviews and 4 labels. The initial height and width of the view is 1. When clicking on priceView it shows the view (1x1 pixel) and completes the animation. That works perfectly fine, but the text is not animating nice.

So I though I could come up with a 2D transformation and though this would look nicer:

var transform3 = Titanium.UI.create2DMatrix({
        scale:50,
        rotate:45,
    });

    var animation3 = Titanium.UI.createAnimation({
        transform: transform3,
        duration: 500,
        width:10,
        height:10,
        curve: Titanium.UI.ANIMATION_CURVE_EASE_IN_OUT
      });

Turns out that the transformation does look a bit nicer (I though don't know why I need to rotate, but if I don't the square is 45 degrees turned). The only problem: all the child views are gone.

Any ideas why and how to fix this?
