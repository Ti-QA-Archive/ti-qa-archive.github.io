Hey,

Trying to install a commonjs module without success.

Create a dir structure: /modules/commonjs/rssreader/1.0/
Inside the "rssreader" dir a created a file called "rssreader.js".

The TiApp editor let me install the module; nonetheless Titanium fails to build, throwing "Could not find all required Titanium Modules".

Guess this is not the right way to do this, can anybody help?

    Hi

    If the module is a CommonJS module rather than a compiled module then you simply place the file in the following location;

    {appname}/app/lib/rssreader.js

    Then in your code you would reference it like this;

    // be aware the ACTUAL require line should be in
// whatever documentation you have been supplied
var RSS = require('rssreader');

    If it is a compiled module then there is a step by step for this Using a Module.
