picker in android devices shows white background only.

Picker in android devices shows only white background. So text can be shown until i select it. Is there any option to handle this situation. If no then any other option is present to handle radio controls? OR some UI tweak ?

  • Hi Ketan Kulkarni,

    Thanks for your concern.

    You can use backgroundColor property of picker and that will resolve your problem. Please check my test case below and see how it goes for you.

    var win = Ti.UI.createWindow({
  layout: 'vertical',
  backgroundColor:'#fff'
});

var picker = Ti.UI.createPicker({
  top:50,
  backgroundColor:'gray' 
});

var data = [];
data[0]=Ti.UI.createPickerRow({title:'Bananas'});
data[1]=Ti.UI.createPickerRow({title:'Strawberries'});
data[2]=Ti.UI.createPickerRow({title:'Mangos'});
data[3]=Ti.UI.createPickerRow({title:'Grapes'});

picker.add(data);
picker.selectionIndicator = true;

win.add(picker);
win.open();

picker.setSelectedRow(0, 2, false);

    Thanks
