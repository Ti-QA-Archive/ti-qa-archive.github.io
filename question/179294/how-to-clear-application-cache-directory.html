How to clear application Cache Directory

Hello,

I am using imageView in my app which loads images and stores in cache directory. My application size is growing continuously. How can I clear my application's Cache Directory?

Titanium SDK: 3.2.2GA
Platform: Android and iOS
Host Operating System: OSX Mavericks
