Listview Templates Array - Listview with many templates

Hi,

I have been able to create a Listview just fine. I now want to be able to give every item in the listview a different template. As I will have about 70 items I am creating all the different parts needed in an array, the one problem I am having is the templates setting - how can I replace that with an array with all the values needed.

I can create a simple listview with only one template like this.

    var listView = Ti.UI.createListView({
        templates:{'Template':bookTemplate,},
        sections:[bookSection]
    });

I could then spesify more templates like this

    var listView = Ti.UI.createListView({
        templates:{
            'Template':bookTemplate,
            'Template2':bookTemplate2,
            'Template3':bookTemplate3,
        },
        sections:[bookSection]
    });

But rather than adding in more than 70 lines to define all the templates how would I go about simply replacing that with an array with all the templates defined?

Thanks for the Help!

  • I was able to get this to work. Not sure if using eval is a good solution or not but nothing else would work.

    var templatesObject = {};

for (var index=0;index < 10 ;index++){

templatesObject["Template"+index] = eval("bookTemplate" + index);

};

    and then we set the templates to the templateObeject we just created:

    var listView = Ti.UI.createListView({
        templates:templatesObject,
        sections:[Section]
    });
