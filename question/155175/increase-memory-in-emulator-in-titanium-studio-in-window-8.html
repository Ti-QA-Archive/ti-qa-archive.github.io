    You can't access the AVD Manager directly from Titanium Studio.

    But what you can do however is to access it using the Android SDK Manager. Simply use the android command from your [ANDROID_SDK]&#x2F;tools directory and from there; select the Tools -> Manage AVDs… menu.

    You should see your titanium virtual Device listed there. You can then modify it using the Edit… button.

    Hope this helps.
