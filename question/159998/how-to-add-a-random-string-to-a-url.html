How to add a random string to a url

Hey how can i add a random string to a url something like:

var browser = Ti.UI.createWebView({
    url:'http://apple.com' + '?' + randomString
});

    Hi

    This technique is usually used to force a new version of a url to come down and not a cached version - guess that is what you are after.

    var randomString = Math.random();
var browser = Ti.UI.createWebView({
    url:'http://apple.com' + '?' + randomString
});

    This will generate a random number that includes a very long precision for example;

    0.10404874314554036

    This is the standard way of solving your problem.

  • Just Googled for 'js random string generator', it returned a nice one:

    function randomString() {
    return Math.random().toString(36).slice(2);
}

var browser = Ti.UI.createWebView({
    url:'http://apple.com' + '?' + randomString();
});
