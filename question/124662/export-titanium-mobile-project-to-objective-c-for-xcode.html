Export Titanium Mobile Project to Objective-C for xcode?

This might be a dumb question, but I am wondering if it is possible to take an existing Titanium Project and export all the JavaScript into objective C that you can then import that into an Xcode project.

I know that you can use a module in a titanium project and use Objective-C there, but what i really want to know is if its possible to convert all the JavaScript into Objective-C.

    There is a general misconception on how Titanium "converts" javascript and the Titanium SDK api set into XCode projects. Although Ti projects ultimately build a XCode project when then gets compiled and packaged by Xcode it isn't as direct a conversion as I think you are expecting.

    Some of the best explanations I have seen are here and here.

    The short answer (IMHO) is you don't want to be mucking with the generated XCode source.

    YMMV.
