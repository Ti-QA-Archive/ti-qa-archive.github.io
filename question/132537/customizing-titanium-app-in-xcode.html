Customizing Titanium App in xcode

I am using the Titanium Studio, build: 1.0.8.201201262211 and would like to do some customizing of the Titanium App I built in xcode version 4.2. My specific problem is that I was trying to find the xib,nib files, but I guess that is no longer an option in the new xcode as it has gone to story boards. My question is how can I customize my Titanium App in xcode by using the interface builder? Where are the visual layout files located in Titanium Apps that have been output xcode already?

  • Titanium does not compile your JS to xib files and objective-c… it runs the JS in real time… so to answer your question… no you can't edit your app in interface builder

  • Hello,

    Titanium does certain magic creating windows and stuff dynamically, you will have a Javascript and a native part. So, at running time, Javascript is calling different components as windows, and creating them on the fly.

    So, if you need to do any change, you will need to do the magic in the Titanium side.
