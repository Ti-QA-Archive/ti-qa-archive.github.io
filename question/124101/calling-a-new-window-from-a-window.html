Calling a new window from a window.

Just starting out into mobile development from desktop :).

We experienced that when we open a new window, the execution of the code continues to the succeeding lines (see code snippet below).

Is this how it's supposed to be? Or can execution of the succeeding code run only if the new window is closed?

openThisForm("thisForm"); // this form is opened

doSomething(); // code immediately continues to this line after opening form above
doSomethinElse();

Is this how it is in mobile development? Thanks in advance

    You can use the close event of windows to achieve what you want, but yes windows open asynchronously.

    win.addEventListener('close', function(){
   doSomething();
});

    • Do something like this:

      var myNewWin = Ti.UI.createWindow();
myNewWin.open();

myNewWin.addEventListener('close', function(){
    // code you want to execute after the window is closed and not while it's open
});

      If you want to halt everything like animations and other things, you'll have to stop everything by hand in the open event of the window you open.
