Exporting or Copying tables of one database to another database

Hi I am developing an application using Titanium. I am having a database in my project/Resouces folder. Now I am creating another new database. Instead of creating tables again in the new database I want to copy all the tables of existing database which is in my project/Resouces folder to the new database and able to access the data from the new database. Could anyone share ASAP how to export the tables of one database to another database through coding. Thanks in advnace

    var db = Ti.Database.install('/my_data.sql','new_db_name');
    if (db.file) {
        db.file.remoteBackup = false;
    }
    db.close();
