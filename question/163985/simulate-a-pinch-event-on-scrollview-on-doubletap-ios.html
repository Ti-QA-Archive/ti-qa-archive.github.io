Simulate a pinch event on ScrollView on doubletap (iOS)

Hi there,

I tried to programmatically "generate" a pinch event on a ScrollView, but it does not seem to be handled like I would expect. My log messages both are shown, and the pinch event is triggered, but the view zoom does not change.
Do you have an idea what I am missing here?

this.view.addEventListener('pinch', function(e) {
    Ti.API.log('view pinched ' + e.scale + ' ' +  + e.source + ' ' +  + e.type + ' ' +  + e.velocity);
});

this.view.addEventListener('doubletap', function(e) {
    Ti.API.log('doubletap');
    self.view.fireEvent('pinch', {
        scale: 1.2,
        //source: self,
        //type: 'pinch',
        velocity: 2,
    });
    self.view.fireEvent('pinch', {
        scale: 1.3,
        //source: self,
        //type: 'pinch',
        velocity: 2,
    });
});

  • Yes - you need to update the view's zoomScale property in the doubletap event handler instead of firing the event:

    this.view.addEventListener('doubletap', function(e) {
    Ti.API.log('doubletap');
    self.view.applyProperties({
        zoomScale: 1.2
    });
});

    The pinch event wont be fired as that is only fire programmatically by calling fireEvent or when the user physically pinches
