    Couple of things I don't see mentioned.

    • Can you now, or have you ever, been able to run any project on your genymotion or device?
    • With the genymotion open and started manually, from a terminal run adb devices. Does the emulator show up in the results?
    • If adb can see your genymotion, from inside the project directory run ti build -p android Does it install?
    • Do you have Google Play Services installed on your genymotion?
    • Have you used monitor/ddms to see the logs from your device looking for clues to install failure?
