Ti.UI.FILL not working with vertical layouts

The documentation says that "Defining a child with FILL height behavior in a vertical layout will fill all the remaining vertical space in the layout", however, I cannot get that behavior to work. This is some example code which does not work:

(function() {
    var
        win = Ti.UI.currentWindow;

    var vView = Ti.UI.createView({
        layout : 'vertical',
        width  : '100%',
        width  : '100%',
        left   : 0,
        top    : 0,
    });

    var view1 = Ti.UI.createView({
        width           : Ti.UI.FILL,
        height          : '100dp',
        backgroundColor : 'yellow',
        left            : 0,
        top             : 0,
    });

    var view2 = Ti.UI.createView({
        width           : Ti.UI.FILL,
        height          : Ti.UI.FILL,
        backgroundColor : 'red', 
        left            : 0,
        top             : 0,
    });

    vView.add(view1);
    vView.add(view2);

    win.add(vView);

})();

Whatever I try, and however I define the layouts, the view2 view never appears. If you define a fixed height (eg. 200dp), it appears, but never with Ti.UI.FILL.

Is there something I'm missing, or is this a bug?

BTW, I tried this with the regular 2.0.1 SDK, and both the 2.0.2 and 2.1.0 nightlies, the behavior is the same in all of them.
