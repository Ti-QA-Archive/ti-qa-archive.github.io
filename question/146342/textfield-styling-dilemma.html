TextField Styling Dilemma

I am creating a form in an app that is for iOS and Android, My dilemma is with the styling of (in this example) text boxes. The auto height of the iOS text boxes is not big enough, so I need to set a minimum height of the text boxes so that it looks proper in both iOS and Android. How would I go about doing this? or is there a way to set a height on the text boxes for each OS

  • You can do OS specific code in this manner

    if(Titanium.Platform.name = 'android'){

//Do code for android platform here
txtbox.height = 50;
}
else
{
//Do code for iOs platform here
txtbox.height = 80;
}

      I have tried setting the height to 60 and to auto, the code below is auto.

      var username = Titanium.UI.createTextField({
    color:'#336699',
    top:10,
    left:10,
    width:300,
    height:40,
    hintText:'Username',
    keyboardType:Titanium.UI.KEYBOARD_DEFAULT,
    returnKeyType:Titanium.UI.RETURNKEY_DEFAULT,
    borderStyle:Titanium.UI.INPUT_BORDERSTYLE_ROUNDED
});
if (Titanium.Platform.name = 'android'){
    username.height = 'auto';
}
win.add(username);

var password = Titanium.UI.createTextField({
    color:'#336699',
    top:60,
    left:10,
    width:300,
    height:40,
    hintText:'Password',
    passwordMask:true,
    keyboardType:Titanium.UI.KEYBOARD_DEFAULT,
    returnKeyType:Titanium.UI.RETURNKEY_DEFAULT,
    borderStyle:Titanium.UI.INPUT_BORDERSTYLE_ROUNDED
});
if (Titanium.Platform.name = 'android'){
    password.height = 'auto';
}
win.add(password);

    Why not just use the dp suffix?

    var txt = Ti.UI.createTextField({
   height: '40dp',
   left: '10dp',
   right: '10dp',
   top: '10dp',
   width: Ti.UI.FILL
});
