Retrieving image dimensions

Is there any way to retrieve the dimensions of an image? I'm getting the following error:
E/AndroidRuntime( 1689): java.lang.RuntimeException: Unable to start activity ComponentInfo{com.pvpulse.wip.pv/org.appcelerator.titanium.TiModalActivity}: java.lang.IllegalArgumentException: width and height must be > 0

When trying to create an image view from a local image. I don't know the height or width before hand. Is there a way to determine the image dimensions prior to creating the image view so that the height and width can be set in the create method? Or maybe there is a better way to do it?

  • Hi Corey

    The following code demonstrates what you want to achieve:

    var win = Titanium.UI.createWindow({
  backgroundColor:'#fff',
  layout:'vertical',
  exitOnClose:true
});

var image = Ti.UI.createImageView({
  image:'test.jpg'
});

win.add(image);
win.open();

Ti.API.info('*** Image dimensions = ' + image.toImage().width + ' x ' + image.toImage().height);

    Hope it is useful! :)
