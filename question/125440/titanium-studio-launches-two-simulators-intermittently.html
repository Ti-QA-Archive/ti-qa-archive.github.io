Titanium Studio launches two simulators intermittently

More often than not, when I hit the green Run button, Studio launches two simulators. One will say "Only one iOS simulator can run at a time." and the other has my app.
I'm sure I'm not hitting run twice. Maybe my MacBook Pro is too fast?
It's a minor annoyance since I can still use the simulator.

I'm running Titanium Studio, build: 1.0.4.201108101535

Anybody else seeing this?

    This is a common question and lot of people has reported this issue in Q&A forum. I suggest you to close emulator always before relaunch. Keyboard shortcut Cmd + Q does it fast.
