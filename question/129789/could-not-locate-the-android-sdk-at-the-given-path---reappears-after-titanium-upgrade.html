  • OK, after removing Titanium Mobile, full reinstalling, problem remains the same.
    I notice the link on the Titanium Dashboard, for installing Android SDK Tool.

    One can download and install a dedicate version of Android Tools / Platform, in a new folder.
    Which i do. And it is now OK.

    Hints :

    • Probably an upgrade of Android SDK Tools and Platform is not compatible with Titanium Mobile (or the opposite :) ).

    • My current working version contains SDK Tools Rev 15, SDK Platform-tools Rev 10, SDK API level 7 and 8.

    • Do not upgrade to some higher version of SDK Tools and Platform-tools

    I am happy again now.
