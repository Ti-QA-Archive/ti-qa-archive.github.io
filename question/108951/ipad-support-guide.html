  • I have found programming to the iPad quite similar as to the iPhone. There are some specific iPad functions, as popover and split window which are somewhat described in the Mobile API Reference.

    Apart from this, and the fact commands regarding the phone or SMS will not work on the iPad, it is more or less an iPhone with a large screen to program in my opinion.

    Just create a new project of the iPad type (not the Mobile) and you have at your fingertips a 1024x768 pixels large iPhone with reduced functionality in phone, sms, contacts etc.

    Harald
