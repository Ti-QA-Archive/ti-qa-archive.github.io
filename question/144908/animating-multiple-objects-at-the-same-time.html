Animating multiple objects at the same time

I'm trying to animate two labels - the first moves to top, other one just changes the opacit, thus:

lbl1.animate({top:-10});
lbl2.animate({opacity:1});

However the lbl2 animation sometimes happens few milliseconds later (especially on older iOS devices) and the final animation is ugly.

In objective-c it is possible to setup an animation for multiple objects within a block so this animates nicely:

[UIView animateWithDuration:0.5f
                              delay:0.5f
                            options:UIViewAnimationCurveLinear
                         animations:^{
                             lbl1.frame = top;
                             lbl2.opacity = 1.0;
                         }
                         completion:nil];

or even using begin/commit pattern:

[UIView beginAnimations];
lbl1.frame = top;
lbl2.opacity = 1.0;
[UIView commitAnimations];

Is there a way to achieve this with appcelerator?
