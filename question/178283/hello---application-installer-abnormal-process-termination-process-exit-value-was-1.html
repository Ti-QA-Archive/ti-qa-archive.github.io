  • Hi Calvin,

    Please remove all the SDK build tool higher than 20 from Android SDK manager, as you are using 3.4 GA.

    Always run the emulator and keep it ready before you build/run any of your project. Cleaning project before every rebuilding is a good practice. This may take a few more seconds than testing an app normally, but it can save a whole lot of trouble. Cleaning procedure is surely recommended before testing your app on a device, after switching to a new Titanium SDK version and before submitting your app to the App Store.

    You can use Genymotion, a virtual android device environment for a better/faster testing performance as an alternative to Titanium android emulator.

    Hope this helps you.
