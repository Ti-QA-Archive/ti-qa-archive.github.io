How to build project with 3.1.3 SDK

My iOS app is just rejected by Apple saying my app has bug on iOS 7.
So I now urgently need to rebuild my project using the latest Ti SDK which support iOS 7.

I have download the 3.1.3 SDK from here http://developer.appcelerator.com/blog/2013/09/be-ready-for-ios-7-with-titanium-3-1-3-rc.html

However, after installation, in Ti Studio, inside tiapp.xml, there is no v3.1.3 version for selection in build properties > SDK list.

I have tried edit the tiapp.xml and set sdk version to 3.1.3.v20130908095038, but it end up build fail.

My Ti Studio version is still 3.2. I saw from some blog message that they are using 3.3 already but I have no idea where to download Ti Studio 3.3.

Anyone can help?

  • Maybe. You have to copy this link http://builds.appcelerator.com.s3.amazonaws.com/mobile/3_1_X/mobilesdk-3.1.3.v20130908095038-osx.zip into clipboard. Then you go to Titanium Studio/Help/Install Titanium SDK/ and paste the link into textfield. Then you hit 'Finish' and wait ….
