    • This means that if the length of the value-property (e.value) is greater than 1, you should restrict the value of the textfield to the first character of the current value. Like this:

      textfield.addEventListener('change', function(e) 
{
    if (e.value.length > 1)
    {
        textfield.value = e.value[0];
    }
});
