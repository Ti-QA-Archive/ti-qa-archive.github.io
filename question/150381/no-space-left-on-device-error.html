No space left on device error

I have created a new project (default project) but while running it using Android emulator I am getting the following error.

[ERROR] failed to copy 'C:\Users\user\My Documents\Titanium_Studio_Workspace\lens_alarm\build\android\bin\app.apk' to '/data/local/tmp/app.apk': No space left on device

what i need to do to solve this problem.

    Hi,
    The error which you are getting clearly states that there is not enough memory in the emulator for the application.So, go to the "settings" ->"Applications" -> Manage applications. Now remove some of the unwanted applications. And install your application again.
