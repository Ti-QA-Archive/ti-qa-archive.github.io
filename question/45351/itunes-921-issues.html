iTunes 9.2.1 issues?

Just curious as I have updated to iTunes 9.2.1 today and an now getting an install error from Titanium. I know that we will be having to submit through XCode but did this update block test installs from 3rd party apps? Do we have to test directly through XCode now for device debugging?

  • Solved the issue and thanks to the build.log file. I found that the mobileprovision file was not named the same as Titanium was looking for it. You can find the provisioning profiles installed in the Users/NAME/Library/MobileDevice/Provisioning Profiles/ make sure that you only install your mobile provision through Titanium, otherwise the names won't match and you won't be able to install though iTunes onto device.
