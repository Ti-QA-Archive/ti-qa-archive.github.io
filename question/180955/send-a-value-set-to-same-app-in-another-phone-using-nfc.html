send a value set to same app in another phone using NFC

I want to make a NFC interactive app.

my basic idea is when 2 phone come closer for NFC interaction i should be able to push a value from one phone so that same application will be open in other phone and show the value which send by other phone…

i able to open application using Modules.Nfc.NdefRecordApplication but i cannot send a specific activity or value to other phone.. or its not working…

My NFC function goes as bellow, even sample win which defile in 'lifecycleContainer' is not loading on NFC action

function pushMessageCallback() {
    // This method is called when you have the 'onPushMessage' property set on the adapter.
    // An application can dynmically determine what message to return based on application state.
    var message = nfc.createNdefMessage({
        records: [
            ndefRecord 
            /**
              * The Android Application Record (AAR) is commented out. When a device
              * receives a push with an AAR in it, the application specified in the AAR
              * is guaranteed to run. The AAR overrides the tag dispatch system.
              * You can add it back in to guarantee that the specified
              * activity starts when receiving a beamed message. For now, this code
              * uses the tag dispatch system.
              */
            , nfc.createNdefRecordApplication({ 
                packageName: "com.op.nfc_alloy_1" ,
                lifecycleContainer : "sample"
                })
        ]
    });

    return message;
}
