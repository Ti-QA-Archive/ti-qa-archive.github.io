I am looking for something like
'
aView.clear()
'

or something with which I can rebuild a given view with content…

It is unfortunate that one has to keep record of every object added to a view in order to delete it from it again…

  • Hi Christophe.

    I had similar problem as you now have. I wrote simple container and then used that container for every Titanium component. That container can add, remove and get elements that particular component contains, so it solved my problems (now I can easily add elements to the custom view, and after that I can get particular element, get all elements, remove one, remove all, etc.). That container is basically a wrapper around JavaScript object Object.

    Hope this idea will help you.

  • There is a remove method on the view object. Haven't tested this at the view level, but you could try the below, if you iterate through all of the children (by number, not by having to keep track of them in advance.)

    //put your loop mechanism here...

viewName.remove(viewName.children[i]);

//end your loop

    Or, would it work for you to just remove the view and recreate it?

    window.remove(viewName);//if the view name is constant
window.remove(window.children[x]);//if the order in which the view is added to the window is constant
