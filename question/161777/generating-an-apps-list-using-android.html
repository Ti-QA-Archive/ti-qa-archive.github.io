  • Hello Robert,

    Native alternative

    getting the list name is tricky, since if you only use intents, you don't get all the installed apps, but you get all the actionable items with intent exposure.

    For solving this, you would need a native module. There is some information on how to create this app list here and you need to decide if you want to get a list of the available intents or of the Application info objects.

    Extra info

    Creating a native module is not a trivial task, but might be not that complex for this problem. Please check this documentation about the process of creating an android module).

    Let us know how it goes!

    Best,

    Mauro
