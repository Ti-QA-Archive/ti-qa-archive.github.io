  • Accepted Answer

    You weren't clear in your question whether you're installing to a device or to the emulator. If you're trying to install to the emulator, then the error message is telling you that the adb process has hung. In a terminal window (command prompt), enter adb kill-server then once that's done, enter adb devices The first command shuts down ADB, the second will restart it. You should now be able to build to the emulator.
