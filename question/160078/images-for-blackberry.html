Images for BlackBerry

Hello,

I released a simple app, in Android, everything fine, but when I test with blackberry emulator, I don't see the images,
Where should I put the photos for blackberry???

  • Julien,

    If you enable the blackberry platform when you created the app, it should have made a /blackberry folder under the /assets folder. If it's not there, create it, place images in it, then check that the blackberry config is set in your tiapp.xml (I'm assuming it is because you said you don't see the images on the emulator). I don't have specific experience with blackberry, but that's how it's done with the other platforms. Hope that helps.

    Regards,

    Mike.
