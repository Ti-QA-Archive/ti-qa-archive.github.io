Blob.imageAsThumbnail() is not adding border nor border cuve to image?

First I have to do some workaround since my image is in SVG format so have one WebView that converts it to base64 using canvas element.

Now that I have my base64 representation of image I do this:

var blob = Ti.Utils.base64decode(convertedSVGImage.imageDataURL.replace(/^data:image\/(png|jpg);base64,/, ""));

This blob can be used now as map annotation but it is to big (it's normal size).
I tried calling imagesAsThumbnail and imageAsResized on it but it is not working. Like it doesn't know that this blob is image.

I figured this workaround:

var imageViewBlob = Ti.UI.createImageView({
    image : blob
}).toImage().imageAsThumbnail(200, 30, 30)

ImageView knows how to handle that blob so I convert that ImageView to blobImage and call resize method on it.
Now it works (it is resized) but I don't see any border nor corner curve?

Ideas?

iOS 5, Titanium SDK 2.1.0
