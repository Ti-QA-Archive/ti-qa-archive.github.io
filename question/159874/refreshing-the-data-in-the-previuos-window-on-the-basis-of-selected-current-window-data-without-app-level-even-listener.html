  • The parent window? You meant the current window right?

    Of course, you have other way of doing this.

    Assume you use Alloy

    prev_win.js

    var nextWin = Alloy.createController("next_win", {
    onWinClose : function () {
        //Update
       }


}).getView();

nextWin.open();

    next_win.xml

    <Alloy> 
    <Window onClose="onWinClose">
    ..........
    </Window>
</Alloy>

    next_win.js

    var args = argument[0] || {};

function onWinClose() {
    args.onWinClose();
}

    Hope it helps
