  • It would help if you posted the code showing your ListView's template, rather than the code that populates the data. I assume you're using a Label within your template. Try adding wordWrap=true to your template definition (or to its entry in the TSS). Make sure you're also setting the height of that Label, not the ListItem, to the appropriate height.
