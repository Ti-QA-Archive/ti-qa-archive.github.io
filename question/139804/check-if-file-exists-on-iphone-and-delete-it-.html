Check if file exists on iPhone, and delete it... ??

I create an image file in an app, which is reused during the users activities.
My problem, though, is when this user logs out, the final version of this file still remains on the iphone.

How do I check if the file exists on the iPhone, and then delete it??

    var file = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory, filename);
if (file.exists()) file.deleteFile();
