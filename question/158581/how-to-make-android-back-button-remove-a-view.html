How to make android back button remove a view?

Hi guys, I've been trying to find a solution on how to make the android back button remove a view instead of closing the app…currently the app just keeps closing, any ideas?

    win1.addEventListener('androidback', function(){
    win1.remove(tableView);
});

  • var basicWin = Ti.UI.createWindow({
    backgroundColor : '#f2f2f2',
    exitOnClose : false
});

var t = Ti.UI.createTableView({
    backgroundColor : 'green'
});

basicWin.add(t);

basicWin.addEventListener('androidback', function() {
    basicWin.remove(t);
});

basicWin.open();
