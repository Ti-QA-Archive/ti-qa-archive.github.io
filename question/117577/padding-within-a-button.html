    I haven't worked with buttons, but with a row in a table you could do something like:

    var mylabel = Titanium.UI.createLabel({
    text:'a label',
    top:10,
    right:10,
    bottom:10,
    left:10
});
row.add(mylabel);

    If the row is, say 320x50, then the label's size will not exceed 300x30, thus creating "padding" around the text. At least doing such a thing worked in my experience. Try using the top, right, bottom, left properties with a label and attaching it to a button and see if you get the same effect.
