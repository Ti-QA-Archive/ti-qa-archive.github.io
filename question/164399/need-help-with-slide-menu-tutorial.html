  • I was able to use the import module under help menu. I also added this in the TiApp.xml file after run clean on the project. 

    <modules>
        <module platform="android">nl.fokkezb.drawer</module>
        <module platform="iphone">nl.fokkezb.drawer</module>
</modules>

    I did not change anything else and I ran the application to see if I have an error free application and this is the error I received.

    [WARN] : Could not find Titanium module id=nl.fokkezb.drawer version=latest platform=iphone deploy-type=development
