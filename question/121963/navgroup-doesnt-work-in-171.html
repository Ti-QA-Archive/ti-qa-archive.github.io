Navgroup doesn't work in 1.7.1?

I'm trying to create a navgroup in mobile sdk 1.7.1 even using the default code from the docs:

var win = Titanium.UI.createWindow();
var win1 = Titanium.UI.createWindow({
    backgroundColor:"red",
    title:"Red Window"
});
var nav = Titanium.UI.iPhone.createNavigationGroup({
   window: win1
});
win.add(nav);
win.open();
var win2 = Titanium.UI.createWindow({
    backgroundColor:"blue",
    title:"Blue Window"
});
nav.open(win2,{animated:true});

I get this error:
> Result of expression 'Titanium.UI.iPhone.createNavigationGroup' [undefined] is not a function. at app.js (line 7)

Any idea why this might be happening? It works just fine in the Kitchen Sink with the same code…
