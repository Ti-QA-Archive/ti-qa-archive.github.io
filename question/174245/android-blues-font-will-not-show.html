Android Blues! Font will not show

Hello all,

I'm having the android blues! Been hitting my head against the keyboard for about 4 hours now, would have pulled my hair already if I had any.

I've created a Titanium Alloy Widget that allows for easy use of FontAwesome, a free to use font filled with hundreds of cool icons and it works perfectly on IOS.

Now I know that android goes by the file name without the extension, so I've made sure and triple checked and triple checked it again! The file name is is right!

For the past 4 hours or so, I've been doing a lot of different tests.

  • Used Ti,API.debug to ensure that each step of the process has the correct values stored in used objects, arrays, and variables.

  • Ensured fontFamily uses filename without ext for android

  • Confirmed that the Labels are indeed receiving the proper character codes for the font

  • Used .getFont method just to verbosely ensure that it was indeed set, after I set it.

  • Used applyProperties and SetFont separately, and together, confirming the font has been altered with .getFont as well, but neither work with android.

  • Tried with a seperate copy for assets/android/fonts

  • Tried with just /assets/fonts

  • renamed font file, took out spaces, took out capitilzation (then added caps back after confirming the name change didnt work because it looks better)

  • tried the .otf file instead of the .ttf file - didnt work

  • Tried it with the font file out of the widget all together and just put it in app/assets/android/fonts

Please take a look at my code: https://github.com/MattMcFarland/com.mattmcfarland.fontawesome/blob/master/controllers/widget.js

better yet, can you install the widget and see if it works on your android? Maybe it's the android SDK or API level??

I'm running out of ideas..

Does android have special reservations for the icon attribute?? I originally wanted to use class but could not find an easy way to query elements by class and then change their fonts. Might try a different approach if this is the case.

And remember, this works great on IOS.. I'm havin' the Android Blues

You can get the widget here: https://github.com/MattMcFarland/com.mattmcfarland.fontawesome
