How many shortcuts does Alloy has?

I know that Alloy support some shortcuts for developing. Such as OS_IOS, OS_ANDROID, OS_WEBMOBILE, ENV_DEV, ENV_TEST, ENV_PRODUCTION.

Are there any other shortcuts I can use? I have tried my best to search in the official doc but failed to find any more description. Can anyone give me a link to the list of shortcuts description if there is one? Thanks a lot.

  • Hi

    You appear to be referring to Alloy conditional code constants. This is the list;

    • OS_ANDROID : true if the current compiler target is Android
    • OS_BLACKBERRY: true if the current compiler target is BlackBerry
    • OS_IOS : true if the current compiler target is iOS
    • OS_MOBILEWEB : true if the current compiler target is Mobile Web
    • OS_TIZEN : true if the current compiler target is Tizen
    • ENV_DEV : true if the current compiler target is built for development (running in the simulator or emulator)
    • ENV_TEST : true if the current compiler target is built for testing on a device
    • ENV_PRODUCTION : true if the current compiler target is built for production (running after a packaged installation)

    You can read about them here;

    • Alloy Controllers - Conditional Code

    You do not say what you are hoping to achieve with more of them, but if you have specific app logic to work out then either;

    • You have only a FEW; create them in alloy.js and set them to;
    Alloy.Globals.yourthing = yourvalue;
    • If your have more than a few - create a CommonJS and export them, then when required simply call the require direct to get the value
    // usage
alert(require('yourconstants').whichone);
