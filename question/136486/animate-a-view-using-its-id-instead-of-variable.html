Animate a view using it's id instead of variable

Hi,

I was wondering if there is anyway I could animate an imageView using it's id, instead of it's variable name.
For instance:

var imageview = titanium.ui.createImageView({
});
imageview.animate({whatever});

this is how I'd normally do it. But I'm getting a bunch of images from API and want to animate them when they're tapped. I can give them each a dynamic id like so:

for (var i=0;i<returned.length;i++){
var imageview = titanium.UI.createImageView({
id: 'image'+i
})
}
'image'+i.animate({animation here})

is this possible within titanium? if not what are other ways for me to achieve this?
thanks,
Sam

    Yes, it's possible:

    var imageviews = [];

for(var i = 0; i < returned.length; i++) {
    imageviews.push(Titanium.UI.createImageView({
        id : 'image' + i
    }));
}

// 'image'+i.animate({animation here})
imageviews[i].animate({animation here});// i = 0, 1, ...
