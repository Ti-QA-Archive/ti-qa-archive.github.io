View disappears when moving out of sight

This is strange.
I have a plain Tableview with sections and a few rows. One row contains a switch like so:

var row1 = Titanium.UI.createTableViewRow({title:"Connect Instagram", className: "switch"});
var instagramSwitch = Titanium.UI.createSwitch({
    value:false,
    right:10
});
if (app.loggedIn) {instagramSwitch.value = true;}
row1.add(instagramSwitch);

Now when I move the Tableview so the Row goes off screen and move it back the switch has gone.

mobile sdk 1.8 and iOS5 though it seems to be an issue with iOS5. Someone had the same problem?

best regards, Tom
