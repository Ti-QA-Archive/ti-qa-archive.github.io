Keyboard hides textfields

Hey! I have 10 textfields in my code. these fields shall a user fill in and the press a button.
The problem is that once the keyboard appears 2 textfields are hidden behind the keyboard.
Solutions?
I have tried a scrollview but that moved the textfield up and the layout became a freaking mess.

  • You don't say what platform. Nor have you provided code that will help us see what you've tried. Please provide a more complete question in the future so that people can better help you.

    Putting fields into a scrolling container (such as a ScrollView or TableView) are the classic solutions to this issue. If you're having layout issues, I suspect it's the way you've set heights & positions within that scrollview that are causing problems.

    If you're on Android, you could set the following on the Window:

    windowSoftInputMode = Ti.UI.SOFT_INPUT_ADJUST_PAN
