    To update your sdk for Studio, the easiest I have found is to still download it manually and drop it in the sdk folder. Get the latest sdk here and then extract and drop in to either /Library/Application Support/Titanium or ~/Library/Application Support/Titanium

    More info on updating sdks here (ignore the continuous integration build stuff)
