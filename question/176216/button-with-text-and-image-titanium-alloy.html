  • Hy, thank you for your Query,

    Here is some example of button with image in tiyanium alloy.

    Using Button Image Property

    <Button  class="span2 fblogin" onClick="fbLogin" top="70%" image="/images/global/Social_Sign_In_03.jpg" >Sign in with Facebook</Button>

    Using Button Background Image Property

    <Button  class="span2 fblogin" onClick="fbLogin" top="70%" backgroundImage="/images/global/Social_Sign_In_03.jpg" >Sign in with Facebook</Button>

    Using ImageView as Button

    <ImageView id="image" image="/images/global/Social_Sign_In_03.jpg" height="10%" width="12%" left="5%" top= "10%" click= "fbLogin"><Label id="label1"  text="Sign in with Facebook" /></ImageView>
