Maximum application project size for Apple iPad

Hello, what is the maximum application file size i can deliver for Apple iPad with your platform? - I have to deliver an in-house app which includes PDF and video data up to 4-6 GigaBytes (in total). Target platforms Apple iPad and Android. - Is this possible?

  • Max application size allowed for Apple App Store submission is 2GB. The recommended way to deal with application data is to package an app with the least number of assets possible and then download additional assets upon first run. In this way, you can have a very small app that loads large amounts of data without running into any issues.
