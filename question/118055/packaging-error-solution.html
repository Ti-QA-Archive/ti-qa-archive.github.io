Packaging Error: SOLUTION

If anyone has recently had a "packaging error" while creating an iOS app for distribution, here is your solution. After coming on here and searching for a definitive answer, I started doing some research myself. All it took was a quick look at the build log.

1.) Make sure you have upgraded to the latest Titanium Mobile SDK

2.) Make sure you have updated to the latest Titanium Developer

3.) Go to developer.apple.com and download the latest version of Xcode 4

That's it. After looking at the build log, it showed Titanium is looking for Xcode version >= 4.0 and at the time I stil had Xcode 3 installed (I didn't feel like downloading another 4.5 gigs just yet). Keep in mind, if you have a paid developers account, you can download Xcode 4 for free. If not, you have to pay $4.99 in the app store. Although if you're trying to create a package for distro I'm sure you have a paid developers account.
