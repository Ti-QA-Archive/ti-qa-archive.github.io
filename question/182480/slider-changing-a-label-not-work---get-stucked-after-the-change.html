Slider changing a label not work - get stucked after the change

I've those 2 dynamic created elements: a slider and a label to show the value of the slider.

            var slider = Titanium.UI.createSlider({
                left:"50%",
                min: 0,
                max: maxV,
                value: myValue,
                width: "34%",
                skill: allSkillSets[i].name,
                parent: row
            });
            slider.addEventListener('touchend', function(e){
                console.log(this.value);
                if(this.parent.children[3].text != Math.round(this.value)){
                    this.parent.children[3].text = Math.round(this.value);
                }
            });

When I hold the slider handle, in the very moment that the label get changed, the slider handle just get stucked without able to move until I release the handler.
In other words: It's updates successfully the label and then stops to work. To work again I need to release my finger and put the finger again in the handle - any update on the label cause this very weird behaviour.
If I move very short from one direction to another, not enough to cause a update on the label, I can see the console.log displaying results with no issues. But when the label get update…It's crashes the handler.
