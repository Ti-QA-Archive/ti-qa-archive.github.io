  • Accepted Answer

    You can download it from the command line by running:

    ti sdk install 3.0.2.GA --force

    You can have many installed SDKs, but you must select a default SDK either using the –default flag or explicitly setting it:

    ti sdk install 3.0.2.GA --force --default

    ti sdk select 3.0.2.GA

    • @David, unfortunately Titanium SDK 3.0.0.GA and 3.0.2.GA only works with Node.js 0.8.x. There was a bug that prevents it from working with Node.js 0.10.x that we fixed for the upcoming 3.1.x release. You can download Node.js 0.8.22 from http://nodejs.org/dist/v0.8.22/.
