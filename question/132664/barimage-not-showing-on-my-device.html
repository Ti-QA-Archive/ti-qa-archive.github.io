barImage not showing on my device

So I am still new to Titanium but am loving every minute of it so far. I used a barImage in my code this

barImage: "navBar.png",

It is added to my main window and previews fine on the iOS simulator. However, when I publish to preview on my own device it does not have the bar image displaying and just the default bar. Why would that not show on my device when it shows just fine on the simulator?

  • Hi,
    I had the same problem and figured it out - I'm currently on my iPhone, so can't see my code! Try using:

    yourWindow.barImage('yourImage.png');

    If that doesn't solve the problem try:

    yourWindow = Titanium.UI.createWindow({
// put the barImage-property right underneath the title
title:'',
barImage:''
});

    And if that should fail try:

    yourWindow = Titanium.UI.createWindow({
// put the barImage-property before the title
barImage:'',
title:''
});

    Hope it solves the issue ;)

    /Anders

    PS.
    Always remember to do a clean build when getting errors on change.
