Cannot get/set Textfield value in Alloy

Hi,

I can't get my head around an issue I'm having with an AlertDialog containing a TextField for user input. The platform I'm building for is Android. After pressing one of the buttons, I need to read the value of the Textfield. That field is always empty, which means I can't do anything with the input a user enters.

How should this work and what am I not getting?

The basis code I'm using (in Alloy and using Titanium 3.2.0 GA) is:

<Alloy>
    <Window class="container">
        <Button top="10" width="250" height="50" id="button" onClick="doClick">Open Dialog</Button>
        <Button top="100" width="250" height="50" id="button" onClick="changeTextValue">Change value</Button>
        <AlertDialog id="dialogInput" onClick="doDialogClick" title="title" message="message" cancel="1">
            <ButtonNames>
                <ButtonName>Bel</ButtonName>
                <ButtonName>Cancel</ButtonName>
            </ButtonNames>
            <TextField id="inputTextField"></TextField>
        </AlertDialog>
    </Window>
</Alloy>

function doClick(e) {
    Ti.API.info("Value of Textfield: "+$.inputTextField.value);
    $.dialogInput.show();
}

function doDialogClick(e) {
    Ti.API.info("Value of Textfield: "+$.inputTextField.value);    
}

$.index.open();

  • Accepted Answer

    You have to use View, not AndroidView, and the resulting text field value is pretty buried. This works:

    <Alloy>
    <Window>
        <AlertDialog id="dialog" titleid="del"
            messageid="msg" cancel="1">
            <View>
                <TextField id="mytextfield"></TextField>
            </View>

            <!-- The ButtonNames tag sets the buttonNames property. -->
            <ButtonNames>
                <ButtonName>Confirm</ButtonName>
                <ButtonName>Cancel</ButtonName>
                <ButtonName>Help</ButtonName>
            </ButtonNames>
        </AlertDialog>

    </Window>
</Alloy>

    $.dialog.addEventListener('click', function(e){
    Ti.API.info(e.source.androidView.children[0].value);
});

  • Hi, you cannot add a TextField to an AlertDialog directly. Instead you'll have to the androidView property.

    <Alloy>
    <Window class="container">
        <Button top="10" width="250" height="50" id="button" onClick="doClick">Open Dialog</Button>
        <Button top="100" width="250" height="50" id="button" onClick="changeTextValue">Change value</Button>
        <AlertDialog id="dialogInput" onClick="doDialogClick" title="title" message="message" cancel="1">
            <ButtonNames>
                <ButtonName>Bel</ButtonName>
                <ButtonName>Cancel</ButtonName>
            </ButtonNames>
            <AndroidView>
                <TextField id="inputTextField"></TextField>
            </AndroidView>            
        </AlertDialog>
    </Window>
</Alloy>
