    Hi Idan,

    As per the docs "getOrientation" method for UI.window is for sdk 1.8.0.1 and its not supported by 1.7.5. Thats why you getting this error " Cannot find function getOrientation in object [Ti.UI.Window]"

  • hello Idan Jan,, take a look at this sample code

    var win = Ti.UI.createWindow({
    fullscreen: false
});

var btn = Ti.UI.createButton({
    title: 'show orientation'
});
btn.addEventListener('click',function(e){
    alert(Ti.UI.PORTRAIT + "= Ti.UI.PORTRAIT \n" +
    Ti.UI.LANDSCAPE_LEFT + "= Ti.UI.LANDSCAPE_LEFT \n" +
    Ti.UI.UPSIDE_PORTRAIT + "= Ti.UI.UPSIDE_PORTRAIT \n" +
    Ti.UI.LANDSCAPE_RIGHT + "= Ti.UI.LANDSCAPE_RIGHT \n" +
    "Current Orientation = " + win.orientation);
});
win.add(btn);
win.open();

    Orientation is returned in number format…. try to look the docs here.. Good luck :)
