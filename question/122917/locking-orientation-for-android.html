    • Not tiapp.xml

      Copy the file app_root&#x2F;build&#x2F;android&#x2F;AndroidManifest.xml into app_root&#x2F;platform&#x2F;android&#x2F; ( you will have to create the platform folder and inside the android one)

      Then edit this copy of the file as you did above.
      Remove what you did in tiapp.xml ( this will also force a fresh build ) and build again.

      If it's not working you might want to disable the fastdev and try again.
