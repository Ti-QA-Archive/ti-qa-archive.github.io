Version of Android API

Hello,

I am choosing Android API 10 when make release build because I want to run my application well on old device like Galaxy S. Are my choosing and understanding right? Please give me your opinion.

Thank you very much/ :)

  • Accepted Answer

    This is you entry in tiapp.xml:

    <uses-sdk android:minSdkVersion="11"/>
<uses-sdk android:targetSdkVersion="19"/>

    In the compile option window you select the version 2.3.3.
