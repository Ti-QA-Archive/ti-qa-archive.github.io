Can't set a TableViewRow's height to 0.

I'm trying to re-create the Contact App's display and editor using a TableView, and a custom TableViewRow.

The TableView is styled as: Titanium.UI.iPhone.TableViewStyle.GROUPED

Each row has a view that sets the layout to horizontal, and in that view is two labels.

Now, I want to be able to make each row height = 0 if it is empty, so when the edit button is pressed if will animate open, however setting the height to 0 doesn't do anything. If I set it to height = 1, then it almost hides - but it leaves a 1 pixel border.

Any ideas on what settings I may need to get it to collapse completely? Or does this sound like a bug?

Cheers,
