  • Check in &#x2F;usr&#x2F;local&#x2F;lib&#x2F;node_modules you might find another titanium folder with the 3.1.2 version. Replace that with the already installed 3.2.0 version in &#x2F;usr&#x2F;local&#x2F;share&#x2F;npm&#x2F;lib&#x2F;node_modules&#x2F;titanium.

    When installing titanium globally through the CLI it installs Titanium CLI in the wrong place.
