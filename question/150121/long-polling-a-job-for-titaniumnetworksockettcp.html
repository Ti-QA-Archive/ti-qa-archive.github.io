  • I have used a kind of "long polling" with nginx and HTTP_Push_Module within an Appcelerator app using Ti.Network.createHTTPClient and some minor tweaks.

    http://pushmodule.slact.net

    But I prefer a socket solution with Pubnub for real-time communication or JSON streaming, in this case using Ti.Network.Socket.createTCP

    https://github.com/pubnub/pubnub-api/tree/master/titanium
