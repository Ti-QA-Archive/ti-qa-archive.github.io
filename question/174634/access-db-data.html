Access DB data

I still have some problem using ALLOY models and collections.
this is a part of my alloy.js

Alloy.Collections.temp = Alloy.createCollection('temp');

this part of index.js

var t = Alloy.Collections.temp;

var tm = Alloy.createModel('temp',{
      cod : 1,
      name : 'nick'
  });

t.add(tm);
tm.save();

in another file I've this code

var t =: Alloy.Collections.temp;
var data = t.fetch();

if (data.at(0).get('cod') < 10) {
// do something
}

This last line raises an error: "cannot call method at of undefined".

Is my code correct? What's wrong with it?

Thanks in advance for any help.

    The fetch() call doesn't return data. So, your var data = t.fetch() command isn't putting data into data as you expect. Instead, the data is being loaded into t See the Collections guide as well as the sample Alloy apps for more on this.
