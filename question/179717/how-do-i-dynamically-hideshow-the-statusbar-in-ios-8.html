How do I dynamically hide/show the statusbar in iOS 8?

I'm fairly new to Titanium and I've been trying to find the answer to this for a long time. I want to temporarily hide the status bar in iOS without using the "fullscreen" option. There are ways I can do it when I'm opening a specific window, but that's not my case.

I have a slideout menu using nl.fokkezb.drawer, which is based off NappDrawer. When the user clicks the button to slide out the menu I want the status bar to hide (the Gmail app is a good example of this). All tha is happening is a window moving from left to right revealing the menu below it which is already open.

How can I go about this? I haven't been able to find any way to just dynamically toggle fullscreen on/off using javascript, and as far as I know the fullscreen property is the only way to hide the statusbar. Is there a function like Ti.UI.iPhone.hideStatusBar() or something of the sort?

Thanks in advance!

  • Hi,
    Ti.UI.iPhone.hideStatusBar() is removed in SDK 3.2.0. So its not worked. Only way to hide this using Fullscreen option or you need to set the status bar style with respect to your window colour.
    For white background window you can hide it like this
    var win = Ti.UI.createWindow({ backgroundColor: 'White' });
    win.statusBarStyle=Titanium.UI.iPhone.StatusBar.LIGHT_CONTENT;
    win.open();
