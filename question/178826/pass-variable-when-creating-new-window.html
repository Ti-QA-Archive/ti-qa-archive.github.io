Pass variable when creating new window

So I'm trying to pass a variable when creating a new window. I've read on several sources that Alloy.createController can pass up to 2 variables, which is perfect because I need to pass one variable. This is my current code:

function openNewMenu(e) {
    var myVar = 123;
    var newMenu = Alloy.createController('menu', myVar ).getView();
    newMenu.open();
}

If this is the correct way of passing a variable, how do I catch it in the newly opened menu?

    try below …

    function openNewMenu(e) {
    var myVar = 123;
    var newMenu = Alloy.createController('menu', myVar ).getView();
    newMenu.open();
}

    and in menu.js

    var args = arguments[0] || {};
alert(args);

  • You use arguments.

    var myVar = {whatever:"blah"};
var newMenu = Alloy.createController("menu", myVar).getView();
newMenu.open();

    Then in newMenu view you'd do something like this:

    var args = arguments[0] || {};
alert(args.whatever); //alerts "blah"
