Identify row in a tableview row with a event of a view object inside TableView

I need to identify row number when the user click over an image (view object) inside Table View Row. Is posible?

var row = Ti.UI.createTableViewRow();
...
var photo = Ti.UI.createView({
                backgroundImage:'/images/user.png',
                top:5,
                left:10,
                width:50,
                height:50,
                clickName:'photo'

            });
row.add(photo);
...
photo.addEventListener('click', function(e)
            {
                // Need row to show data 
                alert('Info');

            }

    correction to my other answer. You will need to add that row property manually to the photo object, so if you are iterating through adding the rows, just assign the iteration value to a custom 'row' property, so, like:

    start loop {
var photo = createView...

photo.row = i;  //add a custom property to photo here
row.add(photo);

photo.addEventListener('click',function(e){
alert('Row property sent by event: '+e.source.row);
}

i++
}loop back
