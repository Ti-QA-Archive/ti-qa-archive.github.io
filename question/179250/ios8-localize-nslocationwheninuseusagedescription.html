iOS8 Localize NSLocationWhenInUseUsageDescription

How do I localize the new iOS8 entry in tiapp.xml?

<key>NSLocationWhenInUseUsageDescription</key>
<string>String to be localized</string>

  • Create under your directory "Resources" the appropriate localization folder, e.g. "de.lproj" and "en.lproj".

    Language name according ISO 639-1: http://en.wikipedia.org/wiki/ISO_639-1

    Now in each localization folder create a file with the name "InfoPlist.strings" and with this content:

    NSLocationWhenInUseUsageDescription = "GPS User Location";

    Then reference it in tiapp.xml

    <key>NSLocationWhenInUseUsageDescription</key>
<string>NSLocationWhenInUseUsageDescription</string>

    Instead of "GPS User Location" you can put of course whatever translation you like.

    This method works btw for all keys/values in tiapp.xml
