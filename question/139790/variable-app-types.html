Variable app types

hi all

my understanding with Appcelerator is that we can develop any of the 3 types of apps

a. Native app – true app which you have to download
b. Hybrid – native user interface, application behaves like an app, but also calls HTML and works offline
and online
c. Mobile web – typical website, but mobile friendly

my question is:
if we develop a hybrid app or native app, is it easy through appcelerator to convert it into a 'mobile web' app OR do we have to pretty much start from scratch.

Same if we just develop a mobile web app, is it easy to convert it into a hybrid or native app

Thanks in advance

Sean

  • Once you develop a native app, porting it to Mobile web should be a no brainer (provided that the components used are compatible with the Mobile Web API)

    Hybrid app are whole other beat though. But it should work pretty well also if the html parts integrate pretty well with Titanium (especially for navigation between screens)
