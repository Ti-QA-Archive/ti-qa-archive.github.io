Titanium Studio can't launch Android Emulator after update

Hi Everyone,

Alongside other developers on this forum I can't launch the Android Emulator from Titanium Studio after updating the Android SDK. The error I get is as follows:

invalid command-line parameter: and.
Hint: use '@foo' to launch a virtual device named 'foo'.
please use -help for more information
[DEBUG] signal caught: 3
[DEBUG] calling emulator kill on 5292
ERROR: The process "5292" not found.

I applied the 'no-spaces' fix by removing all spaces from the directory pointing to the android sdk but that hasn't worked. Created a new project as well as re-installed everything, still get the same issue. This is blocking my development and would really appreciate it if anyone could shed some light on how to fix this and/or if the Appcelerator team are working on a fix.

Many thanks.
