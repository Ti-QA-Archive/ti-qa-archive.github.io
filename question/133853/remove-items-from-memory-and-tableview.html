Remove items from memory and tableview

If I want to reuse a tableview in my code, how I can erase his content and release the memory used?

myTable.setData([])

    Yah you can empty a table by setting its data property to an empty array.
    Refer to this:

    table.setData([]);
// or
table.data = [];
