Random Quote or Fact Button. (Similar to iPredict)

So I want to make an app where if you press a button it gives you a fact but every time you press that button I want a different fact. I already have the button and alert part running but how do I make it that every time it will be different?

    On top of my head this should work:

    var facts = ["fact 1", "fact 2", "fact 3"];
var rand = Math.round(Math.random()*facts.length);
alert("Random fact: "+facts[rand]);

    this is basic javascript programming though, you should study a bit more if you want to develop an app.

  • 1) make a database with all of the facts
    2) have a counter so that each time one is used, its incremented
    3) when the button is pressed, use the LIMIT,ORDER BY, and RAND to get one result, ordered by the counter field (ascending)
    4) after its selected, run an update that increments the counter by 1, using the rows ID field
