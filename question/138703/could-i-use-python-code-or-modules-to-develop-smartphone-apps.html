    • As I said in a comment above, this is NOT a full Python interpreter. And as the module itself says, it only "Ports common Python functions to JavaScript". Just the common, built-in python functions are available. You're not going to be able to import other modules and libraries.
      That being said, this wouldn't work on a mobile web app anyway. If you need to gather info about the os, you'll have to run your mobile web project on a remote server and use a server language there to gather what you can about the os.
