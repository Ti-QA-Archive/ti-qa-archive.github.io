Ti.API is undefined

I am creating my first application for iPhone using Titanium and found it really easy and simple to add components and make them work.

I am trying to use Ti.API.info(&#39;hello world&#39;); for printing something in the console. No matter where I put this line it says

Result of expression 'Titanium.API' [undefined] is not an object.

I would like to know where I am going wrong. Is there any prerequisite that I am missing?

Thanks in Advance !!

  • Accepted Answer

    When you run your project, titanium reduce code with only api's you use. Then, if you insert new api's before you compile, this error occurred.

    To correct, open folder 'YourProjectFolder\build\iphone\' and delete all content of that.

    the next time do you run your project, it will do a Full compile, and this error was fixed.
