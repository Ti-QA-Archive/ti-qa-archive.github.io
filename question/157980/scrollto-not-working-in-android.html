ScrollTo not working in android

ScrollTo in ScrollView is not working in android, am using 3.1.3.GA Titanium and 4.2 Emulator

index.xml

<Alloy>
    <Window>
    <ScrollView id="pull">
        <View id="header">
            <Label id="t1">Some Text</Label>
        </View>
    </ScrollView>
    </Window>
</Alloy>

index.js

var pull = $.pull;
pull.scrollTo(0, 40);
$.index.open();

But its working mobile web browser. Any wrong with the code?

  • I got this working by putting the scrollTo code in a window postLayout event handler.

    So view had this code…

    <Window onPostlayout="handlePostlayout" >
    <ScrollView id="scrollView">
    </ScrollView>
</Window>

    and controller had

    function handlePostlayout(evt) {
    $.scrollView.scrollTo(x, y);
};

    Hope this helps
