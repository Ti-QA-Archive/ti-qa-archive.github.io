Launching application from a notification

When launching my app from a notification, if the app is in background, everything works fine - it comes to foreground.
But if the app is not running (no task), it gets started without showing the splash screen. There's a background activity of a few seconds while the views are being set up, and then the app just appears.
How can I make the splash screen appear.

I start the app using the following intent (which I put in a pending intent for the notification):

PackageManager pm = tiapp.getPackageManager();
Intent resultIntent = pm.getLaunchIntentForPackage(mainActivityPackage);
resultIntent.addCategory(Intent.CATEGORY_LAUNCHER);
resultIntent.setFlags(Intent.FLAG_ACTIVITY_SINGLE_TOP);
resultIntent.putExtra("additional_data", data); // just my extra stuff

Application type: mobile (android)
Titanium SDK: 3.2.0
Platform & version: Android 4.4
Device: Samsung S4
Titanium Studio: 3.3.0.201401102007
