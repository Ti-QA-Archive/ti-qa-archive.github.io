Label textAlign did not work

I want to use custom tableview,so I create a Label and put something

here is part of my code.

I using SQLite database to read data , and then show them in tableview

    while(row.isValidRow()){
        var str = row.fieldByName('name');
        var label = Ti.UI.createLabel({
            textAlign:Ti.UI.TEXT_ALIGNMENT_LEFT,
            font:{fontWeight:'bold',fontSize:20},
            width:'auto',
            height:'auto',
        });
        var row_data = Titanium.UI.createTableViewRow({
            width:'auto',
            height:'auto',
        });

        label.setText(str);
        row_data.add(label);
        dataArray.push(row_data);
        row.next();
    }
    depart_tableview.setData(dataArray);

I hava already set the textAlign to Ti.UI.TEXT_ALIGNMENT_LEFT

but when it shows, it's textAlign is 'center', not LEFT

what do I need to do to resolve this problem?

      width:'auto',
      height:'auto',
      left : 0
 });
