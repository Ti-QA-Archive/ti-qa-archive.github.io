    You question is very specific.

    There is no limit on the Open source Titanium Sdk between the free version and paid version.

    the difference is a lot to do with support, ACS etc.

    now to answer the specifics. Titanium uses. JavaScript so, can you do what you are asking, 80% out of the box! Things like document creation, you can but you will either need to write it yourself or put in a third party plugin.

    I think you are misunderstanding a development framework, with an application which does specific tasks.

    however everything you have asked is possible! either via your own code or using a third party plugin.

    • To add to the accepted answer by Trevor, in short you can do all those things with the free version, many using the core solution and a few using some modules (although I would have to check on .doc support - never personally needed it).

      Modules are used to extend the core framework, many modules are free, some are paid, very few of the paid ones are expensive. Only number 2 is a module, there are lots to choose from and there are a few that are free for qrcodes. Creating PDFs requires a js library jsPDF, again free.

      You do not pay a percentage of the revenue or any other money to Appcelerator.
