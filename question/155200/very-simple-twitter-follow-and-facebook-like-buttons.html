Very simple Twitter Follow and Facebook Like buttons

I've looked through the Q&A and have yet to find a simple way to add buttons which link to popular social networks like Facebook and Twitter. I don't want to post from the app, and I don't want to deal with oAuth or authorising the app to post on the users behalf. I simply want add a link to follow or like a particular page. I can't help thinking there is an easier way to achieve this than using the Facebook module. Many thanks for your help in advance.

    • In case anyone else wants to do it this way:

      mainWin.js

          //add the webviews
     var webviewTwitter = Titanium.UI.createWebView({
         url:'/ui/twitter_button.html',
         height: '50dp',
         width: '200dp',
     });

     w.add(webview);

      /ui/twitter_button.html

      <!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="utf-8">
        <title>twitter_button</title>
        <meta name="viewport" content="width=device-width; initial-scale=1.0">
    </head>

    <body>
        <div>
<a href="https://twitter.com/appcelerator" class="twitter-follow-button" data-show-count="false" data-size="large">Follow @appcelerator</a>
<script>!function(d,s,id){var js,fjs=d.getElementsByTagName(s)[0],p=/^http:/.test(d.location)?'http':'https';if(!d.getElementById(id)){js=d.createElement(s);js.id=id;js.src=p+'://platform.twitter.com/widgets.js';fjs.parentNode.insertBefore(js,fjs);}}(document, 'script', 'twitter-wjs');</script>
        </div>
    </body>
</html>
