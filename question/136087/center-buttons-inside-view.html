Center buttons inside view

How do I center buttons inside my View? I don't want to explicitly set left and right. I have 3 small buttons and I just want them centered inside my view, which is 30px high and spans the full width of the iphone.

var vwButtons = Titanium.UI.createView({
   width: '100%',
   height:30,
   bottom: 30,
   layout: 'horizontal'
});

Currently the buttons are aligned on the far left side of the view.

  • var vwButtons = Titanium.UI.createView({
   width: '100%',
   height:30,
   bottom: 30,
});

var btnContainer = Ti.UI.createView({layout:'horizontal'});

var btn1 = Ti.UI.createButton({width:50, height:20});
var btn2 = Ti.UI.createButton({width:50, height:20});
var btn3 = Ti.UI.createButton({width:50, height:20});

btnContainer.add(btn1);
btnContainer.add(btn2);
btnContainer.add(btn3);

vwButtons.add(btnContainer);
