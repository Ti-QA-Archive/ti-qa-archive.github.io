Invalid --ios-version value '5.0'

i have downloaded titanium sdk for MAC.but while running program m getting this error

Invalid –ios-version value '5.0'
Accepted values:
6.0

I am using-

titanium SDK-3.1.0.ga
xcode-4.5.2
MAC-OSX 10.7.5
iOS simulator version -5.0, 5.1, 6.0, 4.3.2

  • Hi Sayali,

    Check your Titanium Preference for iOS, set the SDK to use to 6.0 and it should work then.

    Also make sure that Xcode 4.5.2 is set to use by titanium if multiple Xcode you have in your MAC.
