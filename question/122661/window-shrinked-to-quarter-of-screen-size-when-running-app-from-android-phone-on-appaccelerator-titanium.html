    Without knowing the specifics of your code (without you posting them) I can only guess. The HTC Desire has a high density display. If you had coded your screen sizes in pixels instead of DPs, then it is an expected behavior.

    Read the blog post here for a possible solution:
    http://developer.appcelerator.com/blog/2011/06/new-defaults-for-android-layouts-in-1-7.html
