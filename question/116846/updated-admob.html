  • Okay, in the spirit of not wanting to wait for an updated TiAdMob, I have added my AdSense code to a WebView and put it in the footer of my TableView. Here's my code:

    var footerView = Ti.UI.createView({
    height:50,
    width:320,
    bottom:0,
    backgroundColor:'#000'
});

var googleAd = Ti.UI.createWebView({
    url:'googleAd.html',
    height:50,
    width:320
});

footerView.add(googleAd);

var tableView = Ti.UI.createTableView({
    footerView:footerView
});

    My only problem now is… When you click on the WebView, the ad page opens up in the tiny little box on the bottom. How would I get this to actually open up in Safari?

    • It's possible to catch click events with JQuery, cancel default click event actions (unbind or return false), then fire a Titanium App event to a Ti Platform Open URL. Use the event data param to pass the click event info (e.g. URL or form information). JQuery in your web view will save you many hours.
