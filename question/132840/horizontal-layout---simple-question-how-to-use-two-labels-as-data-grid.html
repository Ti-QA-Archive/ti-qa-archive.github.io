Horizontal layout - Simple question: How to use two labels as data grid

I am trying to display a simple list of keys and values like this:

<key1> <value1>
<key2> <value2>
<key3> <value3>

The keys should be aligned left. The values should be aligned right.

For each row I created a view with horizontal layout. To this view I added the two labels.

Using fixed pixel widths the layout is displayed as expected.

But when using percentage values for the labels of the keys and values these labels are not displayed. Does anybody know why this is happening?

I do need to use percentages for the width in order to comply with screen rotation and multiple devices.

Does anybody have a clue for creating a data grid using percentages?

Here is the code. You can simply add this to an empty app.js and run it.

Thank you for your help!
