    Hello Daryl,

    Try creating a new project by going to file->new->titanium project then select default project, fill out the info and hit finish. the appicon.png should be in the android and iphone folder, depending on what platform you are building for, it will pull in the required appicon.png image. Your folder layout should look like so: screenshot. Let me know if your folder layout looks any different or if this fixes your problem.
