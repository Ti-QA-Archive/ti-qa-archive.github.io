Feasibility: Galaxy Tab 10.1 (or similar Android), and access to GPS and Serial/USB data

Hello. Before proceeding with Titanium Mobile for a new project, I wondered if anyone could tell me if it is possible to do the following using the product:

  • access GPS data (assume so based on the API)
  • access data from USB port (Tab has proprietary-to-USB cable available)
  • post to web (assume so)

Any comments, or your experience would be very much appreciated. Thank you.

  • Accepted Answer

    • Yes, via Titanium.Geolocation
    • You'll need to create a native Android module to do so. Basically, if its possible with Android, you can create a Titanium module to allow you to leverage it in your Javascript code. Here's the docs for creating native module: http://wiki.appcelerator.org/display/guides/Android+Module+Development+Guide. Just a word of warning, according to some research I've been doing, people are having lots of problems getting USB communications to work in native Android with the Galaxy Tab. Check these discussion here and here.
    • Yes, via Titanium.Network.HTTPClient
