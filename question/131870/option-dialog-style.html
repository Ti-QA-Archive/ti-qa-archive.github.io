    There is no other way to Style option dialog , to get this done a simple way is to make one more window object and set size according to your requirement, do not provide window top and open the same window as dialog through win.open and put a button listner on dialog window to close window as win.close and customise it as dialog box.

    This simply mean treat window as dialog window … hope this idea works for you.
