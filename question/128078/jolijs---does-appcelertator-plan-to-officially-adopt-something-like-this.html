  • Hi Bill. You can contact Xavier without hesitation, he is very friendly.

    I'm using Joli ORM for some time. At the beginning it was very cool tool, but soon I found it limited. Because of it's design, it's hard to override some things. Also, I don't like it's model limitation to one database table and even more, model limitation to database models only.

    What I would like to have is database abstraction layer, so SQLing is easier and I would like model part to be generic, so I can work with database models, but also with XML models, JSON models and so on. I'm planing to rework my model part and I think that will be enhancements (among some others).
