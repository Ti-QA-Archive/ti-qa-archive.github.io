Binary size increasing on 2.1.2 / 2.1.3 SDK?

I noticed that my app binary (iOS) increased from 14MB to 25MB going from the 2.1.1 to 2.1.2 SDK.
The new 2.1.3RC2 also gives me a 25MB app binary.

We have worked hard with clever reuse of graphics to make the app stay under 20MB (App Store limit for 3G downloads). Anyone know what happens with the new SDK that inflates the file size so much?

  • Also noticed the increase app binary size and figured out that images in folder /Resources/iphone/ is also copied to main folder /Resources/

    So I just moved the images from /Resources/iphone/ to /Resources/ before final built and binary size was reduced as no duplicated files are now included.
