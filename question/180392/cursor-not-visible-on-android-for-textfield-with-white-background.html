Cursor not visible on Android for Textfield with white Background

Hi,
since the cursor is white on Android by default it is not visible if you set a white background for the textfield. I read a solution here https://developer.appcelerator.com/question/164315/cursor-color-in-a-textbox but I can not figure out where to place those styles.xml and colors.xml in an Alloy project (assets?).

"TextField" : {
    borderStyle : "Ti.UI.INPUT_BORDERSTYLE_BEZEL",
    backgroundColor : "white",
    color: "#336699",
}

Alex

  • hi Alex,
    You need to create Style.xml and colour.xml in this path "platform/android/res/values".
    In Alloy projects platform folder is just outside of your app folder.
