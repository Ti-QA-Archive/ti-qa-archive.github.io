andorid hardware back button (androidback) does not work inside tabgroup

hi there,

I have a tabgroup, with some windows like this:

<Alloy>
    <TabGroup>
        <Tab id="tab_map" class="tab">
            <Require type="view" src="map"/>
        </Tab>
        <Tab id="tab_add_farmer" class="tab">
            <Require type="view" src="add_farmer"/>
        </Tab>
        <Tab id="tab_search" class="tab">
            <Require type="view" src="search"/>
        </Tab>
        <Tab id="tab_more" class="tab">
            <Require type="view" src="more"/>
        </Tab>
    </TabGroup>
</Alloy>

And now, I want to add an eventListener for the hardware 'back' button for android. And I just want to have this event listened inside the 'search' window.

according to the doc, it should be possible to set an eventlistener for androidback for Windows and Tabs.

now, I tried that, and it doesn't fire the event, what am I doing here wrong?

This is my search.xml 

<Alloy>
    <Window id="winSearch">
    <!-- some stuff -->
    </Window>
</Alloy>

and this is my search.js

    alert("android's back button pressed");
});

// });

(note: I'm not opening the search window through any javascript and I added this inside my app.tss to get all windows in modal (i read something about that)):

// app.tss
"Window": {
    modal: true,
    // some other stuff
}

    Hello Emre,

    Set tabGroup's androidback event. like :

    ###index.xml

    <Alloy>
    <TabGroup id="tabGroup>
        <Tab id="tab_map" class="tab">
            <Require type="view" src="map"/>
        </Tab>
        <Tab id="tab_add_farmer" class="tab">
            <Require type="view" src="add_farmer"/>
        </Tab>
        <Tab id="tab_search" class="tab">
            <Require type="view" src="search"/>
        </Tab>
        <Tab id="tab_more" class="tab">
            <Require type="view" src="more"/>
        </Tab>
    </TabGroup>
</Alloy>

    ###index.js

    $.tabGroup.open();

$.tabGroup.addEventListener('androidback', function(e) {
    alert("android's back button pressed");
});

    try this and let me know if any query.

    i hope this helps. thanks.
