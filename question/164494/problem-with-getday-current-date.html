Problem with GetDay()... current date

Hello friends.

I want to get current date in my app using getDay, getMonth and getYear but the titanium returns wrong values. I tried this and check the result:

var currentDate = new Date();
alert(currentDate); //Returns: 2014-04-14 03:49:11 +0000

alert(currentDate.getDay()); //Returns: 1
alert(currentDate.getMonth()); //Returns: 3
alert(currentDate.getYear()); //Returns: 114

Anyone knows what im doing wrong?

Thanks in advance. Regards

    Hello Jorge,

    If you use Alloy than i suggest you to use inbuilt function Moment.

    for ex:

    var moment = require('alloy/moment');
var day = moment("12-25-1995", "MM-DD-YYYY");
Ti.API.info("Date:" + day.format("dddd, MMMM Do YYYY, h:mm:ss a")); // --> Monday, December 25th 1995, 12:00:00 am

    for more information about moment see this Documentation of Moment.

    i hope this helps. thanks.

    • Thanks, I resolved my problem.

      var moment = require('alloy/moment');
var currentDate = moment().format("DD,MM,YYYY");
var currentTime = moment().format("HH,mm,ss");
alert("Current date: " + currentDate );
alert("Current time: " + currentTime  );
