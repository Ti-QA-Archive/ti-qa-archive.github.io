Airplay printing without wifi network

Hi,
I have an iPad app that can print invoices using the AirPrint module.
The problem is that it works only when there is a wifi network (and when you have the wifi password).
The user of the app need at same time intent (via 3G) to synchronise database, and wifi for AirPrint.
Do you know a portable printer that has its own wifi network ? so no need to have a wifi network.
Do you think that a 3G/Wifi portable router will suit the need (having internet with 3G and have airplay) ?
Thanks for your advice.

    Hi

    Whilst many printers now come with the ability to connect to WiFi, the printers themselves do not act as WiFi router. There are some very specialised printers that can do this but the cost is prohibitively pointless for this discussion.

    You mention HP printers but I think you are probably confusing the AirPrint support they have built in that means that configuration for others devices to connect to them is reduced - this does not mean those printers can actually form their own connection to the internet and then share that connection to other local devices.

    There are a couple of solutions to this, assuming the iPad does not have its own network access;

    • If there is an Android device or iPhone in the mix, use it as a hot spot - make sure you have sufficient data allowances though. This will form a local WiFi network using the 3G data of the mobile device and sharing it using WiFI from the same device. This is my personal favourite
    • As Mauro Parra suggests you can get a MiFi (portable WiFi hotspot), but keep in mind you will need an additional data contract to use this device.
