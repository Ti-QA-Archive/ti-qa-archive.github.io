Alloy: Dynamic layout inside TableViewRow

I'm trying to achieve a specific layout inside a TableViewRow:
title and description to the left - below each other, date to the right, centered vertically. Title and description are of variable height, can be 1 or 2 lines. Like below, but without the space between title and desciption:

title
                      datetime
description

<TableView id="theTable" dataCollection="list">
            <TableViewRow class="row" hasChild="true" leftImage={icon}>
                <Label class="title" text="{title}"/>
                <Label class="datetime" text="{datetime}"/>
                <Label class="description" text="{description}"/>                
            </TableViewRow>
</TableView>

I can't get the vertical centering right since the labels are of an unknown height. Is there a clever .tss way of achieving this or do I have to make some calculations. If so, what would be the correct event handler to use on the TableViewRow? Thanks everybody.

    • something like this:

       <TableViewRow class="row" hasChild="true" leftImage={icon}>
        <View layout="vertical" width="200" left="0"  height="Ti.UI.SIZE">
                    <Label class="title" text="{title}"  height="Ti.UI.SIZE"/>
                    <Label class="datetime" text="{datetime}"  height="Ti.UI.SIZE"/>
        </View>
                <Label class="description" text="{description}" width="120" textAlign="right" height="Ti.UI.SIZE" left="200"/>               
            </TableViewRow>

      don't set top to description, it will center itself

    • Thank you very much, the additional view is the right way to do this. With a little bit of additional styling I got the result I need:

                  <TableViewRow class="row" hasChild="true" leftImage={icon}>
                <View layout="vertical" width="200" left="0"  height="Ti.UI.SIZE">
                    <Label width="Ti.UI.FILL" textAlign="left" class="title" text="{title}"  height="Ti.UI.SIZE"/>
                    <Label width="Ti.UI.FILL" textAlign="left" class="description" text="{description}" height="Ti.UI.SIZE" />    
                </View>
                <Label textAlign="right" class="datetime" text="{datetime}" height="Ti.UI.SIZE" right="10dp" />
            </TableViewRow>
