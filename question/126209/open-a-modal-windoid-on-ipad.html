Open a modal windoid on iPad

I am trying to open a smaller, centered window on top of my iPad app, similar to how you see Facebook or Twitter login windows appear. So far, I get a window that covers my whole screen, with my window size and background color centered on black.

I would like it to be a modal window, slide up and float over the app until dismissed (with a button I will place in the window)

var sellWindow = Ti.UI.createWindow({
    width : 400,
    height : 600,
    backgroundColor: '#ff0'
});

function bMoreClickHandler() {
    sellWindow.open({
        modal : true,
        ipad_modal_presentation_style : "UIModalPresentationFormSheet",
        ipad_modal_transition_style : "UIModalTransitionStyleCoverVertical",
        navBarHidden : true
    });
};

SDK 1.7.2, Studio 1.0, OS 10.7.1, iPad 4.x

  • The properties for the modal styles on your window open call should look like:

    window.open({
    modal:true,
    modalTransitionStyle: Ti.UI.iPhone.MODAL_TRANSITION_STYLE_COVER_VERTICAL,
    modalStyle: Ti.UI.iPhone.MODAL_PRESENTATION_FORMSHEET
})

    Check these constants for a list of other acceptable values.
