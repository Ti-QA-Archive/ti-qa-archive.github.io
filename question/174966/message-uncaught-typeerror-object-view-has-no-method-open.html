    • Since this now comes first when googling for error message:

      The tutorial "Creating Your First Titanium App" is broken. Whenever you create new Alloy Controller you need to replace "View" node with "Window" in xml file.

  • The Error happens when you replace Window tag with view Tag in bookdetails.xml file

    the tutorial say : Wrap the two Labelelements within a View element, as shown below.
    Wrap don't replace the Window Tag !!,

    I have the same error and that fix it

    like this 

    <Alloy>
    <Window class="container">
        <View layout='vertical'> 
            <Label id="titleLabel"></Label>
              <Label id="authorLabel"></Label> 
         </View>
   </Window> 
</Alloy>
