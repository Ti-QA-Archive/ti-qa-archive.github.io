My App wont sleep if I'm looping background audio

Application type: Mobile

Titanium SDK: Titanium SDK version 3.1.1.GA

Platform & version: iOS 6.1

Device: iPad 4

Host Operating System: OSX 6.1.3

Titanium Studio: Titanium Studio, build: 3.1.1.201306112235

After adding this code to my app, the iPad no longer goes to sleep:

var player = Ti.Media.createSound({url:  "audio/lobby.wav", looping: true});
player.play();

The screen dims 15 seconds before the 'Auto-Lock' setting (minimum 2 minutes) but it stays dim and does not shut down.

I would like it to shut down.

When I don't include the looping: true then it sleeps just fine.

Thoughts?
