    You will have to act on the click event of the optionDialog.
    Something like this:

    var input_text = Ti.UI.createTextField();

var dialog  = Ti.UI.createOptionDialog({
    androidView:input_text,
    buttonNames:['Cancel','Get it']
});

dialog.addEventListener('click', function(e) {
    if (e.index == 1) { // we read it only if get it is pressed
        alert(input_text.value);
    }

});

    The code is not tested and you might have to use a wrapper view for the textField, but you get the idea :)

    Let me know if you manage to make it work, or I'll try to make a functional code for you.
