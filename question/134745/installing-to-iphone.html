  • Hi Steven,

    I faced that error and found that this error generated by various reasons. and so many people asking for it. you have to diagnosis first fault. try these point first.

    1) If there is any update occurs and there is faulty file is in SDK.

    2) There is faulty image file like default.png and icon.png in images folders.

    3) If there is any special characters in file name.

    4) Spaces in file name and project name in tiapp.xml or any project directory.

    You can try following steps in sequence as it is.

    1) check if lower version build successful.

    2) if newly created project build successfully.

    3) try to find out special characters and spaces in your file name.

    4) remove unnecessary image files from project permanently or temporary.

    5) try to remove old SDK if necessary then replace it by new one.

    if there is still error then please send any changes reflected in your current log.
