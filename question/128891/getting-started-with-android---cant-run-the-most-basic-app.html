I've followed the steps HERE to setup my system for android development on a mac as best I could, but I still can't even run the default 'two tab' app that is created when I start a new mobile android project. (The one that says 'I am window 1', 'I am window 2').

I am able to add the Android SDK and target my app to the android platform, but when I try to run the app in the emulator, the emulator runs and the app never shows up. I just get a phone interface. Any suggestions are welcome. Basically, I need to know why the emulator is running fine, but my app isn't running on it.
