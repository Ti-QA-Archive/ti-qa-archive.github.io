I have a screen with tableview that works fine when visiting the first time, but throws a NullPointerException when i navigate to the next screen and go back to this one again. The stack trace is here

The code is huge but i have tried to explain just the tableView part here.

tableUA : Ti.UI.createTableView({
     separatorColor : appHeaderColor,
     backgroundColor : "transparent",
     data : [],
     top : "190dp",
     left : "10dp",
     width : "260dp"
  })

I set the data to null before populating the rows. 

self.views.tableUA.setData([]);

var cdata = [];
for(var i = 0; i < results.length; i++) {

        height : "40dp"
     });

     var lblName = Ti.UI.createLabel({

      row.add(lblName);
      cdata[i] = row;
}
self.views.tableUA.setData(cdata);
