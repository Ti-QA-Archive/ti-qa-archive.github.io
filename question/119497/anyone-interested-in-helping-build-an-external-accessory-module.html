  • In case there is still interest in this post, Logical Labs has released to the marketplace an External Accessories Module.

    You can build apps with the module that can connect to and communicate with MFi certified devices only per Apple's implementation. It will handle wired and wireless (Bluetooth Classic) connection types via the External Accessory Framework.

    The module includes a working example connecting to a Bluetooth Classic MFi certified Zebra label printer and has a free trial for you test the module out before you purchase. It is also well documented and is fully supported by Logical Labs.
