Deploy to Android device

Hi

I have searched almost every forum and can't find an answer to my problem!

I am new to Titanium and have followed the installation documentation in order to get Titanium up and running on my Windows 7 machine. I began with the "MyFirstApp" tutorial and was able to deploy this successfully onto the Android simulator however when I try to deploy to my device I get the following error:

[ERROR] Timed out waiting for device to be ready, you may try reconnecting the USB cable
[ERROR] :  Build process exited with code 1
[ERROR] :  Project failed to build after 51s 179ms
  • Device: Samsung Galaxy S3
  • Android Version: 4.1.2

I have the phone on Developer Mode with following 'Debugging' settings:

  • USB debugging: enabled
  • Allow mock locations: enabled
  • Select debug app: No debug application set
  • Wait for Debugger: disabled

Windows is able to detect my phone when I plug it in so I'm guessing the drivers are fine. Am I missing something that wasn't in the documentation?

Many thanks for all your help!

  • When you say so I&#39;m guessing the drivers are fine. That doesn't mean that the device will work with the Android SDK. You have to install the drivers manually yourself for Android devices on Windows.

    If you haven't done it already, they usually come bundled with their Kies software. Once you have done that, it should solve your issue.
