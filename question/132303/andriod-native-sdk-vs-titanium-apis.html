  • There is no limitation of titanium and you will not at all limited to use features implemented by Titanium API only .

    Those feature which are not supported in titanium for those feature android module can be made and can be place inside the titanium project . you can use those module using REQUIRE function present in titanium.

    To make module you may reffer :

    https://github.com/appcelerator/titanium_mobile/blob/master/support/module/android/documentation/android_module_dev_guide.md

    You cannot combine java source code with javascript.Only way is to make android module and add in your application.
