  • Accepted Answer

    Try manually updating by downloading the SDK and manually and drop it in the sdk folder. Get the latest sdk here and then extract and drop in to the directory specific to your OS listed here.

    Then, set the sdk version in tiapp.xml. If you don't see your new sdk in the dropdown, try closing and reopening Studio.
