  • So after countless combinations of code, I finally found the answer in the core of Titanium itself. The way Titanium is designed to handle the "scale" event for a scrollView, is that it scales the scrollView itself instead of scaling it's contents.

    So, using the code I posted earlier, what essentially happened is that my scrollView, at zoomScale was 768px wide and 1004px in height. When the zoomScale shifted to 2, and since the scrollView itself was scaled, the new width was 1536px and height was 2008px.

    On reading the docs very carefully, turns out, the minimum value for the contentWidth and contentHeight properties is always the width and height of the scrollView. So at zoomScale 2, the content spanned over 1536px x 2008px.

    Thus, all that extra padding around the content.

    In my case, the solution was simple, since my scrollView has a single child.

    As a simple example, I did this: 

    // t holds the identity matrix provided by the Ti API for transformations.
var t = Ti.UI.create2DMatrix(); 

//In the code below, I assigned the scrollView a custom property "scale" which I set as shown below.

scrollView.addEventListener('dblclick', function(e) {
    if(scrollView.scale == 1) {
        scrollView.children[0].transform = t.scale(2);
        scrollView.contentWidth = w * 2;
        scrollView.contentHeight = h * 2;
        scrollView.scale = 2;
    }
    else if(scrollView.scale > 1)  {
        scrollView.children[0].transform = t;
        scrollView.contentWidth = w;
        scrollView.contentHeight = h;
        scrollView.scale = 1;
    }
});

    Using that example, I only scaled the child views of the scrollView, set the appropriate contentWidth and contentHeight and viola! Everything works as expected.

    I hope someone finds this useful.

    • This is the function I use. First parameter is the width and second is the height.

      function recompsize(w, h) {

    var _w = w, 
        _h = h;

    if ((w /Ti.Platform.displayCaps.platformWidth) >= (h /Ti.Platform.displayCaps.platformHeight)) {

        if (w > Ti.Platform.displayCaps.platformWidth) {
            _h = (h * Ti.Platform.displayCaps.platformWidth) / w;
            _w = Ti.Platform.displayCaps.platformWidth;

        } else if (h > Ti.Platform.displayCaps.platformHeight) {
            _w = (w * Ti.Platform.displayCaps.platformHeight) / h;
            _h = Ti.Platform.displayCaps.platformHeight;
        }

    } else {

        if (h > Ti.Platform.displayCaps.platformHeight) {
            _w = (w * Ti.Platform.displayCaps.platformHeight) / h
            _h = Ti.Platform.displayCaps.platformHeight

        } else if (w > Ti.Platform.displayCaps.platformWidth) {
            _h = (h * Ti.Platform.displayCaps.platformWidth) / w
            _w = Ti.Platform.displayCaps.platformWidth
        }

    }

    return {
        width : _w,
        height : _h
    }

}
