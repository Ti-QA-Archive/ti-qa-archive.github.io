    Hi

    You should have a look at momentjs

    Simply place the javascript in its own commonjs file, then you can use a require on the file to access its functionality.

    var moment = require('/lib/moment');
Ti.API.info('new date = ' + moment(dtpicker.value).format('YYYY MM DD, h:mm:ss');

    Then its a case of just playing around with the formatting to get it how you want it.

    But also note if you are using Alloy it is already included
