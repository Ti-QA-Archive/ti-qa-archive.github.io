    • Hi there,

      If you want to set a background image to a back button, you'll have to use to property
      backButtonTitleImage (click here for documentation).

      In your example, if win is a child of another window or tabgroup, then you'll simple add the following : 

      // ON THE CHILD WINDOW

win.setBackButtonTitleImage('back.png');

      Also if you define the back button image, it'll override the back button title, so you can only use one.

      Hope it helps.
