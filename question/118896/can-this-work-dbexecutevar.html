Hi there,

Can this work:


var sqlInsert = 'INSERT INTO ' + table + ' (' + insertKeys + ')VALUES(' + insertValue + ')';

  • Accepted Answer

    Yes, both can work. In the first example, if any field to be updated is a string, you just have to get the quotes right. Remember the sqlInsert variable must end up with valid SQL syntax including properly quoted values.

    In the second example you are missing a comma before Var1, but you have the right idea.

    I use both methods, depending on what fits better in the particular block of code I am doing.
