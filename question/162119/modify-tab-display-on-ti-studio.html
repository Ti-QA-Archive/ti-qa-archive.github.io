Modify tab display on Ti Studio?

I'm curious if it's possible to modify Titanium Studio to slightly alter the way filenames are shown in the tabs. When I'm developing a cross-platform app, with both iOS and Android-specific files in the same project, it gets confusing which one I'm working on without seeing the folder that the file is in displayed in the tab.

As an example, if you have two files of the same name and they're both open, it looks like the following:

https://www.dropbox.com/s/8a21urckz2ovbc7/Screenshot%202014-01-30%2022.39.01.png

(Apologies, I can't get this image to embed for whatever reason)

That's great when both are open. Is there a way to make all files display with the containing folder displayed in the tab filename?

Thanks!

  • You can't show it in the tab, but if you roll over the tab with your mouse and wait, you'll see the full path.

    Alternatively someone got a different solution here: click on the tab, and have the file be selected in the Project Explorer.
