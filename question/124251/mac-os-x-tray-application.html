  • I have a solution, but it's not perfect.

    1. 1 Create the tray.

    2. 2 Create a new form (new window) set the background in the xml properties to transparent (to remove boarders and topbar).

    3. 3 Event: when the tray icon is clicked display the form at X and Y.

    4. 4 To dynamically position the form, you'll need to use a third-party application to get the mouse's X and Y when the tray icon was clicked.

    I don't know about the arrow pointing the tray icon.
