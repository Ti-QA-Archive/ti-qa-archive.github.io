String with parameter from Localization return null

I' m getting value from user and trying to show it in Toast message. Text of toast added to strings.xml with parameter(entered value by user). Then this toast should show user's entered value with text. But it shows empty toast.

var input_text = Ti.UI.createTextField({
        keyboardType: Ti.UI.KEYBOARD_PHONE_PAD
    });

var toast = Titanium.UI.createNotification({
    duration: 2000,
        message: String.format(L("ToastAlert"), input_text.value)
        });
    Ti.API.info("String.format: " +     String.format(L("ToastAlert"), input_text.value));
    Ti.API.info("input_text.value: " + input_text.value);
        toast.show();

strings.xml

<string name="ToastAlert">Distance is %d</string>

I receive this output in ddms console

11-13 15:33:01.213: I/TiAPI(9928):  String.format: null
11-13 15:33:01.223: I/TiAPI(9928):  input_text.value: 13

I use:
Titanium SDK - 3.1.4
Platform - Android 4.3
Device - Nexus 4
Host OS - Mac OS 10.8.5
Titanium Studio - build: 3.1.4.201310152018
