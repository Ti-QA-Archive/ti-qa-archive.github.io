Searchable TableView with custom layout?

I have a tableview working and I also have a custom layout using createTableViewRow.

When I add a search control to the TableView, however it seems to want to use the "title" property of the row as the search index. The problem being that if you assign a title to the row, then it will appear over the top of the custom layout.

is there a way to assign the searchable text for the row without having it visually display?

    Yes, using the filterAttribute property. You add it to your tableView declaration and it takes in the name of a custom property that you've added to the child rows like so:

    var tv = Ti.UI.createTableView({
   search: search,
   filterAttribute: 'booktitle'
});

var rowBook = Ti.UI.createTableViewRow({
   height:'auto',
   booktitle:book_title
});

    book_title in this case being a variable I declared earlier that holds that data I want to search on.
