  • I corrected few days ago the command line to create the module. It was missing the trailing slash in the target folder

    > titanium create --platform=iphone --type=module --dir=~/tmp/ --name=test --id=com.test

    so instead of

    --dir=./TiTestModule

    try

    --dir=./TiTestModule/
