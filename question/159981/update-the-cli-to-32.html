Update the CLI to 3.2

Hey there,

I'm currently having problems while trying to use the new Ti SDK 3.2.0 (Latest Beta). When selecting the SDK in the Ti Studio (Latest Beta) and build the app, the following error appears and the build fails:

[WARN] :   Requires version >=3.2, currently 3.1.2
[ERROR] : [31m Command "build" incompatible with this version of the Titanium CLI[39m
[ERROR] Unable to find command "build"

I already updated the CLI in the command line with sudo npm install -g git:&#x2F;&#x2F;github.com&#x2F;appcelerator&#x2F;titanium.git but the command titanium -v still says 3.1.2. Is there any way to reset the complete CLI and install a fresh 3.2 CLI?

My workaround:

  • Mac OS X 10.9
  • Ti Studio 3.2 RC
  • Ti SDK 3.2.0.v20131126144841
  • Ti CLI 3.1.2

Thank you! ~ Hans
