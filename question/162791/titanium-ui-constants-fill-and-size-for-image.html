    Hi,

    please check the below code and i hope, you will understand the logic

    var win = Titanium.UI.createWindow({
title:'TEST',
backgroundColor:'#fff' ,
navBarHidden:true,
layout:'vertical'
});

var view1=Ti.UI.createView({
    top:10,
    height:Ti.UI.SIZE,
    width:Ti.UI.SIZE,
    backgroundColor:'red'
});
win.add(view1);

// Create a Label.
var TEST = Ti.UI.createLabel({
    text : 'TEST',
    color : '#000',
    font : {fontSize:20},
    height : 30,
    width : 50,
    top : 0,
    textAlign : 'center'
});

view1.add(TEST);

var view2=Ti.UI.createView({
    top:10,
    height:Ti.UI.SIZE,
    width:Ti.UI.FILL,
    backgroundColor:'green'
});

win.add(view2);
var TEST2 = Ti.UI.createLabel({
    text : 'TEST',
    backgroundColor:'yellow',
    color : '#000',
    font : {fontSize:20},
    height : 30,
    width : 50,
    top : 0,
    textAlign : 'center'
});

view2.add(TEST2);
win.open();

    • Here in view1 height and width is Ti.UI.SIZE and add a level in to the view1 so the view1 size depends on TEST. if TEST width and height increases then the size of view1 will increase, because view1 height and width is Ti.UI.SIZE

      Again, view2 height is Ti.UI.SIZE and width is Ti.UI.FILL. so view2 height depends on TEST2 but width depends on win because width is Ti.UI.FILL and always Ti.UI.FILL depends on parent view. Here view2's parent is win.
