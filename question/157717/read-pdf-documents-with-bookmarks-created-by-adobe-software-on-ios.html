Read PDF documents with Bookmarks created by Adobe Software on iOS

Good day all,

I have a client that has created PDFs using Adobe Bookmarks. Bookmarks in Adobe allow you to create a PDF with links to URLs that open other PDF files. Basically, when you open the document, it is a table of contents with links and the end user has the impression that it is a single file, but when you click on the bookmark it opens a different PDF. The point is to reduce the size of the PDF so you are not opening 300 pages all at once. When we try to open this file using Adobe Reader on any iOS mobile device, the bookmarks appear, but they do not work, and do not open the link to the next document. Does anyone have a workaround solution that will allow us to (1) Read PDF documents with Bookmarks, and (2) Read PDF documents offline with Bookmarks.

Thanks in advance
C.R.
