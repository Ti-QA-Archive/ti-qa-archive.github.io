  • Hi

    I imagine the reason this question has gone unanswered for 3 hours is you never actually asked a question. There is one implied but given you seem very capable of resolving the TableViewRow version, then the View version should have been the same process;

    function SlideShow(response){
    var self = Ti.UI.createView({ backgroundImage:'/img/'+response.imgFocus });
    self.sessionObject = response;
    return self;
};

    You will find this in the scrollView event listener like this (for example only);

    var scrollHandler = function (e) {
    if (e.source.sessionObject) {
        Ti.API.info('sessionObject: ' + JSON.stringify(sessionObject));
    }
};
scrollView.addEventListener('click', scrollHandler);
