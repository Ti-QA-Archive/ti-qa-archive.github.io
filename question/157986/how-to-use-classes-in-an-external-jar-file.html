How to use classes in an external jar file??

I have copied the jar file into the lib/ folder in some module and then built the module for some particular app. Now how do I call some specific method from some X class in the jar file? I have searched online forums and that suggested me to import package and all. But here if i use import it shows Syntax error:unexpected reserved word. What to do?

    If you are developing a module, you will only be able to invoke code from your jar INSIDE the module itself. Since both are written in Java. If you ever wanted to expose a functionality provided by your JAR classes, your module will have to provide a function of your own and inside that same function, you will call the code you require.

    Simply put, you cannot just include a JAR file into a module project and then expect it to be accessible from your JavaScript code. That's just not how it works.
