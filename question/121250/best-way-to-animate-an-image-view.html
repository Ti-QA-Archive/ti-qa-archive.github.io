Best way to animate an image view

Hi community,

Quick question. I have an image view what is the best way so that on click it scales up and then on click of a button it returns to its normal size. I have tried using 2D matrix which works perfectly until I close it and I can't return it to it's normal size, I've tried invert but that seems to scale it to the same ratio the other way which would make sense! I suppose really all I need to know is there a way of returning a scaled image view using the 2d matrix back to it's original state!

  • After doing your scaling animation use the 'complete' event to trigger the reset:

    eg:

    
// lets say you have an imageView called myImage

var animation = Titanium.UI.createAnimation();
animation.scale = 0.5;
animation.duration = 500;

animation.addEventListener('complete',function()
{
    animation.removeEventListener('complete',this);
     // reset the scale to whatever...
     animation.scale = 1.0;
     myImage.animate(animation);
});

// start the first one...
myImage.animate(animation);

  • var image_scale_up_animation = Titanium.UI.createAnimation({width:1024,height:768,center:{x:512,y:334},zIndex:100,duration:500});

var image_scale_down_animation = Titanium.UI.createAnimation({width:512,height:334,center:{x:512,y:334},zIndex:1,duration:500});

var image_scaled = false;

image.addEventListener('click', function(e)
{                
       if (!image_scaled)
        {                        
          image.animate(image_scale_up_animation);
        image_scaled = true;            
    } else {            
        image.animate(image_scale_down_animation);
        image_scaled = false;                              
    }
});
