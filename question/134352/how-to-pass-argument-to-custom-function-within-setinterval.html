How to pass argument to custom function within setInterval()

I have this custom function defined within setInterval() which begins as soon as the application starts and executes every 2 seconds.

setInterval(function() { xhr.open(post,url); xhr.send({userID:1}); },2000)

I want to define an argument instead of using the hardcoded '1' in my userID. How can I do that?

Thank you

    var userID = getUserID();// Get User ID from your datasource such as DB, File, Properties, ...
setInterval(function() {
    xhr.open(post, url);
    xhr.send({userID : userID});
}, 2000);
