    To get the search results for a specific hashtag, you can use :

    http://search.twitter.com/search.json?q=&tag=prayforjapan&rpp=100

    The rpp parameter will give you 100 results, which I think is a maximum.

    To see more Tweets, you use the 'max_id' parameter, where you use the id of the oldest tweet in the returned list and use that return older tweets and therefore the 'next page' of tweets.

    You can get up to 200 tweets using the 'count' parameter.

    See here for further documentation: http://dev.twitter.com/doc/get/statuses/user_timeline
