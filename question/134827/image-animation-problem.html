  • Its because you add them to the same underlying imageView. Transition will Flip the whole view and all its childs. You would need a separate view for each Image inorder to be able to Flip them separately.

    • yeah, but you do imageView.animate();
      this will animate the whole underlying view, which will go for a flip.
      You may want to 

      image1.animate({
view: image2
};

and image2.animate({
view: image1
});

  • hi Lena,
    are you looking for this…?

    image1.addEventListener('click', function() {
    imageView.animate({
        view : image2,
        transition : Ti.UI.iPhone.AnimationStyle.FLIP_FROM_LEFT
    });

});

image2.addEventListener('click', function() {
    imageView.animate({
        view : image1,
        transition : Ti.UI.iPhone.AnimationStyle.FLIP_FROM_RIGHT
    });
});
