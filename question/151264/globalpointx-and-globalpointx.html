globalPoint.x and globalPoint.x

The globalPoint.x and globalPoint.x does not work in Titanium 3.1, whats the substitute for these ?
It is the sample code from Kitchen sink , which does not work anymore:
view.addEventListener('touchstart', function(e)
{
l1.color = 'red';
l1.text = 'touchstart fired x ' + e.x + ' y ' + e.y + " ("+e.globalPoint.x+","+e.globalPoint.y+")";
setTimeout(function()
{
l1.color = '#333';
},200);
});

I am trying to get exact position of controls within the screen(not relative to parent view). I have 2 views view1 and view2. view1 has events touchstart and move, and view2 has no event- I want to show the message when view1 overlaps view2.

Please help me to get the solution

    globalPoint has been deprecated and i ran into this challenge in my own app. What you are supposed to use instead is the convertPointToView. I've dropped an example below. You'll need a reference to the parent view or window that you'll want to get the coordinates of. 

    itemsInfoDrawer.addEventListener('touchmove',function(e){

    var localPoint = {x:e.x, y:e.y};
    var convPoint = itemsInfoDrawer.convertPointToView(localPoint, parentWin);

})
