  • If you only want to customize certain rows, then you need to build each row out, not just pass a data object to the table view. So, create a data array. Build out each row using createTableViewRow so that the first row is how you want it to be and the rest are different. Be sure to use the className property to optimize the rows. Push the rows onto the data array, then set the table data to that.
