    • Yes Sabrina, if you add navBarHidden: true to the createWindow call (see modified code below), Android won't display the default bar with app title above.

      var winWeb1 = Titanium.UI.createWindow({
           navBarHidden: true,
            tabBarHidden: true,
            barColor: 'black',
            backgroundColor: 'white',
            fullscreen: false,
           // title: 'Les concours'
        });
