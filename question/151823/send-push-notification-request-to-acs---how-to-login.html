send push notification request to ACS - how to login?

I sent a REST POST request to:
https://api.cloud.appcelerator.com/v1/push_notification/notify.json?key=jD…my API key

the body is 

"to_ids":"517eb4dda49ecc313e029b30",
"payload":{
    "badge": 2,
    "sound": "default",
    "alert" : "Push Notification Test"
}

I get a 401 response code. Does this mean I have to log in? If I log in, do I get back a token that I have to send together with the request above, or how do I do the login?

  • Actually yes. You need to suscribe the device to a specific channel.

    Cloud.PushNotifications.subscribe({
    channel: 'friend_request',
    device_token: myPushDeviceToken
}, function (e) {
    if (e.success) {
        alert('Success');
    } else {
        alert('Error:\n' +
            ((e.error && e.message) || JSON.stringify(e)));
    }
});

    If ignored please have a wider look here: http://docs.appcelerator.com/cloud/latest/#!/api/PushNotifications. If issue persists you can always post device logs and state your current environment details.
