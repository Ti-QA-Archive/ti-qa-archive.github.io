    Hi Jose

    I think you should have check the console, if the console is saying app is installed and came to focus then you can get the app running. Well if you run any project using TI you will be able to get the project automatically started for you.

    Some tips: dont close the emulator every time, just open it once and then every time you run the project it update your app.

    Keep patience with android development take time to load the app, wait for 2 -10 min to get your app focus in emulator.
