  • Deploying applications on Android is always a long process. So what you are facing is not a problem per se. This is not a Titanium problem since it would take about the same time if you developed with the native SDK.

    Keep in mind though that once it is done. The Fastdev server will push your modifications rapidly. This will save you time. But the very first deployment will always be a long process.
