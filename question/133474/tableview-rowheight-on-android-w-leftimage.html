TableView rowHeight on Android /w leftImage

Hi, I'm trying to figure out how to get my rows to be a specific height in my tableview, but it's not listening to me. I found on a different question to try the min/maxRowHeight, but that's not working either. Here is my code:

var Regdata = new Array(); // I actually add more, but I just added one for example.
Regdata.push({leftImage: 'BennyBeaver.jpg', title:'Fred', color: 'black', hasChild: true});

var table1 = Titanium.UI.createTableView({
    data: Regdata,
    backgroundColor:'white',
    search: searchBar,
    filterAttribute: 'title',
    maxRowHeight: 20,
    minRowHeight: 20,
    searchHidden: true
});

It works on iPhone just fine. My assumptions is the image (BennyBeaver.jpg) is dictating the minRowHeight. Is there a way to tell it what size I would like the image to be? The image is 300 x 300px if it helps anyone.

    You have to create a TableViewRow with a custom view inside that can take any size you like and using this approach you can add labels images buttons and customize your row any way you like

    — answered March 8th 2012 by Arian Caraballo
    permalink
    3 Comments

    • here is an example

          var data = [];
    for (var i = 0; i < 10; i++){
        var row = Ti.UI.createTableViewRow({
            className: 'list-class',
            text:'Apples', 
            price:'1.25', 
            backgroundColor: 'transparent',
            selectedBackgroundColor: 'transparent',
            height: 59
        });
        row.add(Ti.UI.createView({
            backgroundColor: '#000000', 
            opacity: 0.2,
            backgroundColor: 'blue',
            height: 59
        }));
        var contentView = Ti.UI.createView({
            height: 41,
            backgroundColor: 'red',
        })
        row.add(contentView);

        contentView.add(Ti.UI.createLabel({
            text: row.text
        }));
        data.push(row);
    } 

    var table = Ti.UI.createTableView({
        data:data,
        backgroundColor: 'transparent'
    });
