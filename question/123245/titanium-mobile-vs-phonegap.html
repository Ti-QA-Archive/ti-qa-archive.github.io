  • 1) It's difficult to compare Titanium and PhoneGap. To make it short PhoneGap encapsulates mobile sites into web views, whereas Titanium 'compiles' directly into native code (technically this is not entirely true, but this is the general idea).   • My thoughts are …

    Titanium is an exact match up to the mobile SDKs. Titanium is just a way to make native mobile apps using javascript language. Whereas PhoneGap is just a way to wrap your webpage so it looks like a native mobile app. Titanium forces you to conform with the devices native SDK standards.

    Lastly … Titanium pwns PhoneGap all day! :)
