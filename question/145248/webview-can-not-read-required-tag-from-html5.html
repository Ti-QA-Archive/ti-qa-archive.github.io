Webview can not read "required" tag from HTML5?

I have webview renders my mobile web that crated using jquery mobile and html5.
all my input text contain "required" tag and it doesn't seem like webview detect that when I test it with empty text the site still posts to server.

    A webview is just an instance of the on device browser. It doesn't seem that the required attribute even works in the Android web browser. You can test this by firing up the browser on your device or sim and visiting this url: http://mindelusions.com/boneyard/mobile/wv_android/. It will submit even with the required attribute. See that it works correctly in a normal web browser, excluding IE and Safari of course.
