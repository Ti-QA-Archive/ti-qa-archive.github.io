Trouble assigning a value to a custom property

This is a mobile application I am running in Titanium SDK 3.0.2.GA
iOS SDK 6.1 simulator on Mountain Lion

var scoreSelector = Ti.UI.createView({
    top: 10,
    right: 10,
    left: 10,
    height: Ti.UI.SIZE
});
scoreSelector.params = {
    'value': -1
};

var button = Ti.UI.createButton({
    backgroundImage: 'none',
    title: title,
    backgroundColor: '#FFF',
    color: '#366',
    left: left,
    width: width,
    height: height
});

Ti.API.info(button.title); // output = title
Ti.API.info(scoreSelector.params.value); // output = -1

scoreSelector.params.value = button.title;

Ti.API.info(scoreSelector.params.value); // still output = -1

are there any obvious reasons that I would not be able to successfully assign a value to a custom view property?

the actual project is at github.com/otothea/quick7
this code in this post is a simplified version of the code in /Resources/ui/PlayerScrollView.js lines 49 - 138

let me know if there's any more info I can provide…
