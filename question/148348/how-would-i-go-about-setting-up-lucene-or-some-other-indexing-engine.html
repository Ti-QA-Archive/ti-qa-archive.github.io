How would I go about setting up Lucene or some other indexing engine?

I'm looking to build an app that can search a large quantity of text files. My best option at the moment is to process them as paragraphs into a sqlite db, add the fts plugin to my app and use fts queries on the database to return my results. The only issue is sqlite fts doesn't do edit-distance results on individual words. So my app's users will have to spell the words of their query exactly as it appears in the texts they're searching. This limitation might greatly impact the amount of relevant results. This isn't a horrible alternative, but if I could use Lucene or a similar engine it would meet my needs perfectly.

Has anyone had experience using it or a similar engine? How did you get it set up?
