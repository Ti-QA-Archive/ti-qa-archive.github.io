  • Before you start developing for Android, I recommend to read some tutorials, guides and introductions:

    • Android Platform Overview
    • Android UI Components and Conventions
    • Android UI Overview

    When you found out that this bar is called ActionBar, you can start searching the documentation and searching the internet for examples:

    • AlloyActionJackson
    • ActionBarSample
    • ActionBarSampleTi

    Soon, you would have learned that you can hide the ActionBar with a Titanium API or remove it permanently with an Android Theme:

    <application
    android:theme="@android:style/Theme.Holo.Light.NoActionBar" />

// or 

<application
    android:theme="@android:style/Theme.NoTitleBar.Fullscreen" />

    I hope I could point you in the right direction, if Stephen already didn't ;)
