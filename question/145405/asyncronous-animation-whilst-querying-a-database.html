Hi All!

This seems to include any animations. I have an animation on the loading bar, but unfortuantley it freezes when a database call stars and continues the animation when the database query finishes and the app starts loading the layout.

I'm just wondering if there is a way to do an asynchronous animation or database call? so the animation doesn't freeze.
