  • It sounds like you're hard-coding your container view to 768 pixels. Instead of doing that, just set the left/right/top/bottom properties.

    var layout = Ti.UI.createView({ left: 0, right: 0, top: 0, bottom: 0 });

    That way it will always be against the edges of the window.
