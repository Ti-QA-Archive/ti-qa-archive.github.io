opening a window with a view above the window, using negative top values (works in ios, can't find a way in android).

Question:

I am making something similar to the nightshade or android pulldown, or whatever they are respectively called. I got it working perfectly on ios, and nearly perfectly on android. The problem is, getting the window to start in the upper position on android.

code to get the dropdown to start in the upper position in ios (not working in android)

// I added the open window event listener because, it would not work on ios or android otherwise.
// I am guessing the window has to be open before things become assigned to the outside of the window
win.addEventListener('open', function(){
    if(start_down)
    {
            drop_button.addEventListener('click', raise_call);
    }
    else
    {
        drop_button.addEventListener('click', drop);
            // I tried it without animate, didn't work on ios or android
            drop_view_w.animate({
                top: 0 - in_view.size.height - in_view.top,
                duration: 1
            });
    }
});

The dropdown functionality does still work, it just always starts in the lower position (android), which is no good.

Does anyone see why it wouldn't start in the up position? is there another event i should be waiting for? like 'ready' or something?

  • While asking the question, I realized that there may be another event that could work. So i posted the question and continued the search. The answer was yes, the 'focus' event works perfectly for both android and IOS. Here is an updated version of the code: 

    win.addEventListener('focus', function(){
    if(start_down)
    {
            drop_button.addEventListener('click', raise_call);
    }
    else
    {
        drop_button.addEventListener('click', drop);
            // I tried it without animate, didn't work on ios or android
            drop_view_w.animate({
                top: 0 - in_view.size.height - in_view.top,
                duration: 1
            });
    }
});

    I figured I would leave the question up in case some one has a similar problem in the future. Also, I am planning to release the drop down once I get all the features I want in it. If you really want it now email me: 5wooley4@gmail.com
