Quick Tip for calculating label height

So I was looking for a way to find out the height of a label in cases where there's more than 1 line of text and the label height is set at run time.
I noticed that label.height doesn't always return the right value.

The workaround I found (only tested on iPad) is to do label.show(). After you do it, label.height should return the correct value.
