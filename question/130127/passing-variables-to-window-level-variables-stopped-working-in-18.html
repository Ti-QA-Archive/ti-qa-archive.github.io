Passing variables to window level variables stopped working in 1.8

The code below is no longer working in 1.8. Interestingly, when I run in the simulator, the variables are passed to the window. But, when running on a device, the variables are not passed.

Is this a bug in 1.8?

    var w = Ti.UI.createWindow({
    backgroundImage:'images/Background_GlowEdge.png',
        url:'groups_userlist.js'
    });
    w.open({modal:true,navBarHidden:true});
    w.selectedGroupID = selectedGroupID;
    w.selectedGroupName = selectedGroupName;

  • The problem probably is that you add variables after window opens (window opening is asynchronous action). If window manages to open before next statement in JS is executed, then your variables won't be declared.

    Try to put passing of variables to the window before window opens:

    var w = Ti.UI.createWindow({
    backgroundImage:'images/Background_GlowEdge.png',
    url:'groups_userlist.js'
});

w.selectedGroupID = selectedGroupID;
w.selectedGroupName = selectedGroupName;

w.open({modal:true,navBarHidden:true});
