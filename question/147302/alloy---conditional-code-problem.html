Alloy - conditional code problem

Hello,

I'm trying to build a tabbed application for both IOS and Android platforms using the conditional code.
This is my index.xml file:

<Alloy>
    <TabGroup>
        <Require src="tab1" />
        <Require src="tab2"/>
    </TabGroup>
</Alloy>

My tab1.xml file looks like this:

<Alloy>
    <Tab id="tab1" icon="KS_nav_ui.png" title="tab1">
        <Window platform="ios" title="ios window 1" id="win1">
            ...
        </Window>    
        <Window platform="android" title="android window 1" id="win1">
            ...
        </Window>        
    </Tab>
</Alloy>

As you can see, I would like to have different windows on each platform. I expect that, when building for IOS, the second window would be ignored and viceversa. Instead, I get the following error when compiling:

[ERROR] Tab must have only one child element, which must be a Window

[ERROR] Alloy compile failed

I also tried this code for tab1.xml:

<Alloy>
    <Tab id="tab1" title="tab1" platform="ios">
        ...
    </Tab>
    <Tab id="tab1" title="tab1" platform="android">
        ...
    </Tab>
</Alloy>

In this case the error is:

[ERROR] Invalid <TabGroup> child type: Alloy.Require

[ERROR] Alloy compile failed

So it seems that conditional code is not really ignored when building for other platforms. Can anyone please point me in the right direction?
thanks.
