How to use Holo and the old standard theme in the same app?

Hi guys,

Does anybody have an idea if it's possible to use Holo theme in an app but with the old standard theme as a fallback for devices which don't support Holo?

Thanks!

    I believe to acheive this you would use the <uses-sdk> in your Android manifest. You specify the targetSdkVersion as the newest sdk available and your app will use the theme that the device supports i.e. Holo for 3.0+ devices. The minSdkVersion will tell your app what the lowest sdk version it can use, so if it is set lower, than it will use the older Android theme for buttons, activity indicator, textfield, etc…

    Hope this helps.

    Link to Android Documentation

    
<uses-sdk android:minSdkVersion="4" android:targetSdkVersion="15"/>
