    Eventually you'll find it installed in the $HOME&#x2F;Library&#x2F;Application Support&#x2F;Titanium directory, as already mentioned by The Zero. Beware that on Lion the Library directory under your home folder is hidden. Check out this article for making it visible: http://www.mikesel.info/show-hidden-files-mac-os-x-10-7-lion/
