Testing in 1.2.0 SDK after installing iPhone 4.0 SDK

I recently installed iPhone SDK 4.0, and ever since, I've been unable to run the emulator in anything other than Ti SDK 1.3.2 and iPhone SDK 4.0.

1) If I try to use Ti SDK 1.2, when I launch the emulator, it says "Build Failed. Please see output for more details". But I don't see any info in build.log. What output is it referring to?

2) If I try to use Ti SDK 1.3.2 and iPhone SDK 3.2, it launches an iPhone simulator inside of an iPad view (what the heck??)

Does installing the 4.0 iPhone SDK break 3.2? I can't find the 3.2 SDK download anywhere on the Apple developer site to revert to the previous version…
