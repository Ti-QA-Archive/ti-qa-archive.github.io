How do I reference iPhone 5-specific images? (Not the Default-568h@2x.png image)

I'm currently updating apps to support iPhone 5 and in places I have 100% height views with background images. These background images are called welcomeSplash.png and I have a retina version of welcomeSplash@2x.png. Can anyone tell me how to provide a third, iPhone 5-size image so that these images aren't distorted on the new device?

I've tried following the format for Default-568h@2x.png, so I created a welcomeSplash-568h@2x.png but to no avail. Is this supported in the latest Ti SDK? The code looks like this:

var fullPageView = Ti.UI.createView({
    backgroundImage: '../images/welcomeSplash.png',
    height: '100%',
    width: 320,
    bottom: 0
})

Thanks in advance :)

  • I can't find any information on how this might work - if it's even possible - so I created a simple workaround. Hope this helps others with the same issue. Just append this to the above code, but be wary that this property will change if you support device rotation within your app (which I don't):

    if( Ti.Platform.displayCaps.platformHeight == 568 ){
    fullPageView.setBackgroundImage('../images/welcomeSplash-568h@2x.png');
}

    app_bg='/images/bg.png';

app_bg='/images/bg-568h.png';
}
