Error: Missing appid (Facebook)

Hello!
I have an issue. I have added a Facebook login-button to my app, but when i click it Facebook returns an error "An error occurred, please try again later".
In the Titanium console, i get error "Missing appid".
I have added 

<property name="ti.facebook.appid">284575574901135</property>

What could cause this issue, and how do i resolve it?

  • Accepted Answer

    That is to be set in code, not in the tiapp.xml.

    See the api Titanium.Facebook

    Titanium.Facebook.appid = '284575574901135';
