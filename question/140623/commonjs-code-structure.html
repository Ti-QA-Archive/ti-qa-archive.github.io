CommonJs code structure

I have a very basic commonJs piece of code:

TestHome = function () {
    winMain = Titanium.UI.createWindow({  
        title:'Test window',
        backgroundColor:'#C7C7C7'
    })
};

TestHome.prototype.getHomeWindow = function () {
    return winMain;    
}
exports.TestHome;

I am then calling it like so:

    TestHome = require('_tests/TestHome');
    winTester = TestHome.getHomeWindow();

But I get an error "'undefined' is not a function (evaluating 'TestHome.getHomeWindow()'"

Am I doing something wrong?

    Your code is not formatted properly. Your module should define the function and prototype, not set the function to a variable. Like so:

    function TestHome () {
    winMain = Titanium.UI.createWindow({  
        title:'Test window',
        backgroundColor:'#C7C7C7'
    })
};

TestHome.prototype.getHomeWindow = function () {
    return winMain; 
}
module.exports = TestHome;

    Then, in your code, you have to call the function as a constructor. Like this:

    TestHome = require('_tests/TestHome');
winTester = new TestHome().getHomeWindow();

    This becomes much easier if you simply return the winMain in the constructor function though, that way you don't need create/call a function for the window.
