Cannot enlarge images bigger than their original size

Hi fellow developers,

I'm trying to display an image scaling bigger than its original size, but I can't. (it sticks to the original size).
I'm testing on some real Android devices, Titanium SDK 3.1.1.GA.

var image = Ti.UI.createImageView( {
    width : width,
    height : height,
    top : '50dp',
    left : 0,
    images : imageFile,
} );

I set width to like:

  • '100%'
  • '320dp'
  • '1080px'
  • 1080

but none of these values enlarge the image as I want.

Is it simply impossible and do I have to modify the image's blob or something?

Thanks a lot,

  • Same problem, in my case height was fixed (55%) and width had to be set automatically to fit.

    Resolved with this code:

    image.addEventListener('load', function() {
    var ratio = image.toBlob().width / image.toBlob().height;
    image.setWidth(image.rect.height * ratio);
});
