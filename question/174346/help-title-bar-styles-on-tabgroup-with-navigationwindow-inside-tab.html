Help: title bar styles on TabGroup with NavigationWindow inside Tab

Platform: iOS
Project Type: Alloy
Titanium SDK: 3.2.3

My entire app uses a TabGroup with NavigationWindows inside each Tab. I'm trying to set the background color for the titlebar throughout the app (should be the same on each tab). When I set the "barColor" property on the TabGroup, it doesn't affect the title bar background color at all. If I use a Window inside each Tab instead of a NavigationWindow, the barColor works.

Also, I'm trying to hide the title bar for a Tab (it just shows the title property for the Tab) because my NavigationWindow has its own title bar. Currently there seems to be no way to hide the title bar for a Tab, only for a Window (via the navBarHidden property).

You can try this out for yourself with this test index.xml code:

<Alloy>
    <TabGroup id="tabgroup" barColor="red">
        <Tab title="one">
            <NavigationWindow id="navWin">
                <Window title="window one" backgroundColor="white" statusBarStyle="Ti.UI.iPhone.StatusBar.OPAQUE_BLACK">
                    <Label>This is tab 1</Label>
                </Window>
            </NavigationWindow>
        </Tab>
        <Tab title="two">
            <Window title="window two" backgroundColor="white">
                <Label>This is tab 2</Label>
            </Window>
        </Tab>
    </TabGroup>
</Alloy>

    Hi

    Do NOT place a NavigationWindow inside a TabGroup Tab. NavigationWindow AND Tab both use a navigation controller internally. So adding a NavigationWindow inside a tab is unnecessary.

    Each Tab already supports opening child windows, automatic back buttons and everything else NavigationWindow does.

    This is also why the barColor property has failed.

    Simplify as follows;

    <Alloy>
    <TabGroup id="tabgroup" barColor="red">
        <Tab title="one" id="tab1">
            <Window title="window one" backgroundColor="white" statusBarStyle="Ti.UI.iPhone.StatusBar.OPAQUE_BLACK">
                <Label>This is tab 1</Label>
            </Window>
        </Tab>
        <Tab title="two" id="tab2">
            <Window title="window two" backgroundColor="white">
                <Label>This is tab 2</Label>
            </Window>
        </Tab>
    </TabGroup>
</Alloy>

    In your controller;

    var newwin = Alloy.createController('something');
$.tab1.open(newwin.getView());

    This is an example syntax, use whatever you like. The important part is the reference to the Tab id and the 'open' method of the Tab.
