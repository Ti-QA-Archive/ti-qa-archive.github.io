Achieving Fluid Image Dimensions On Android

Hello All,

I'm trying to achieve fluid width (100%) for an image in my app.

When setting width : '100%' for iOS - it works perfectly, Android however won't evenload past the splash screen without throwing an error!

It seems Android won't display an image with either dimensional property set as a percentage or with a value of 'auto' == '0'.

Am i missing a trick? Or is the only way to achieve a fluid image in Android by working out the device' dimensions first?

Appreciate any tips etc…

  • Using the following code I have an image displaying at 100% width on both iOS and Android using Titanium Mobile 1.7.5. See if it works for you. You'll note that if you don't specify dimensions, the ImageView will automatically fill its container, in this case the window.

    var win = Ti.UI.createWindow({
    backgroundColor: '#fff',
    fullscreen: false    
});
var image = Ti.UI.createImageView({
    image: 'org3.png'
});
win.add(image);
win.open();
