Problem parsing XML-string

Hello!
I am trying to use XML in a project, but get's the error

Cannot call method "parseString" of undefined

when running. This is the code

        var readFile = Ti.Filesystem.getFile('compositions.xml');
        var doc = Titanium.XML.parseString(readFile.read());

And this is my XML file

<?xml version="1.0" encoding="UTF-8"?>
<compositions>
    <composition>
        <Image>images/troll.jpg</Image>
        <Artist>Anton Gildebrand</Artist>
        <AboutArtist>Anton Gildebrand är en mycket kreativ människa</AboutArtist>
        <CompositionTitle>Troll</CompositionTitle>
        <Position>
            <longitude>15.630004</longitude>
            <latitude>60.607447</latitude>
        </Position>
    </composition>
</compositions>

I've tried uninstalling the application from the phone and installing it again, touching tiapp.xml, clearing the build folder but nothing works. I've checked that readFile.read() actually contains a value. I get this error on Android (don't know about iOS)

  • I was able to solve this. I edited the dependency.json file for the SDK to make xml required, changed line 27 to this

    "required": ["analytics","android","ui","json","locale","xml"],

    and used readFile.read().text instead of just readFile.read()
    Now it works great =)
