addClass tableviewrow Android

I have this tableview:

<TableView id="menu" onClick="loadView" autoStyle="true">
    <TableViewRow id="A" class="menurow"></TableViewRow>
    <TableViewRow id="B" class="menurow selected"></TableViewRow>
    <TableViewRow id="C" class="menurow"></TableViewRow>
</TableView>

And with this function I can dynamically change class on the clicked row:

var eindex = 1; // holds last selected row
function loadView(e)
{
    if(OS_IOS)
    {
        $.removeClass($.menu.data[0].rows[eindex], "selected");
        $.addClass(e.row, "selected");
    }
    else if(OS_ANDROID)
    {
        // ??????
    }

    // save last selected row
    eindex = e.index;
}

But how do I do the same thing on Android? I can't figure out the correct path to target the clicked row.

    • I use dynamic styling in a similar way by using add/remove class so I don't believe this is the issue.

      Although when I create my tableView's, I tend to create tableViewRow as a seperate controller so that I can loop through an array and build each row without repeating the code. In the data that I pass to the controller I would pass a function property containing the actions I want to perform on row click.

      If you check 'passing arguments' from this link: Alloy_Controllers it shows an example of what I mean.

      So expanding on this by using the example, I would additionally pass a function like so:

      index.js

      var data[];
for (var i=0; i<source.length; i++) {    
    var arg = {
        title: source[i].postTitle,
        url: source[i].postLink,
        action: function () {
            $.removeClass($.menu.data[0].rows[eindex], "selected");
            $.addClass(e.source, "selected");
        }
    };
    var row = Alloy.createController('row', arg).getView();
    data.push(row);
}
$.tableView.setData(data);

      row.js

      var args = arguments[0] || {};

$.rowView.title = args.title || '';
$.rowView.url = args.url || '';

if(args.action) {
    $.row.addEventListener('click', function () {
        args.action();
    });
}

      Sorry if you know all this, I am just trying to explain clearly with examples so others benefit as well.
