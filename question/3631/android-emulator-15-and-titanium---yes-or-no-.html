Android emulator 1.5 and titanium - yes or no ?

Simple question: has anybody succeeded to launch a simple mobile test application (empty project, resulting in the two-tabs iamwindow1 iamwindow2 application) with titanium 1.0 ?

I can launch this app in android 1.6, 2.0 and 2.1 in the respective emulator fine. BUT: Trying this with android 1.5 emulator - 1.5 emulator starts, but won't install or run the app even on repetetive launch tries with emulator open.

Is this normal ?
What am I doing wrong ?

  • Accepted Answer

    From 0.9.3 (or possibly 0.9.1) the minimum spec for Android apps is Android 1.6. The earlier API (0.8.1) is still available for creating apps that run on Android 1.5.
