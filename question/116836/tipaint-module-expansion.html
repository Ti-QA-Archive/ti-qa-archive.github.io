ti.paint module Expansion

Hello People!

Yesterday Titanium Released the awesome painting module and made it opensource!
I've already implemented into my app for a update(was using html5 first)

And even played with the source already.

So while it's a really minor update it is the first of many I hope.
This update to ti.paint allows you to set and edit the Opacity of the brush like you do with the width.

the call for this is
strokeOpacity: 1, // also 0

when you update this value using a slider and a float number it can create a really aquarelly look.

    • Ok so I added a couple of lines, I hope it's a bit in what you need.
      What I did is what I actually just said, instead of drawing the opacity (which is still there) it now sets the opacity of the view you paint on.
      the new argument is : lineOpacity
      so here is an example:

      Titanium.Painter = require('ti.paint');
var painter;

painter = Titanium.Painter.createView({
strokeWidth: 10,
strokeColor: '#2a81df',
strokeOpacity: 1, // you can also remove this it defaults to 1
lineOpacity: 1 // this is where the view get's it opacity value ranges from 0.0 - 1.0
});

      here is the new module (you can overide the 'ti.paint' folder in the modules folder (version is still the same)
      http://www.mediafire.com/?7hh61ge0xj290j9

      To clarify my what I did in the source code:

      TiPainView.h:

      // I added this in the @interface
CGFloat lineOpacity;

      TiPaintView.m:

      // in the init function I added
lineOpacity = 1; //this sets the default value

// in (UIImageView)imageView I added
drawImage.alpha = lineOpacity; // this essentially grabs the value of linOpacity and sets it's alpha

// Added a new call
- (void)setLineOpacity_:(id)alpha2
{
lineOpacity = [TiUtils floatValue:alpha2];
}
