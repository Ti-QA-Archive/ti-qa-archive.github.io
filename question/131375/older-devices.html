  • No. Nor does it support the older BB devices, or for that matter the oldest iOS and Android devices. In fact, using the current 1.8.x SDK level the iOS minimum is 4.2 and the Android minimum is 2.2 though if you use older SDK levels you can create programs for as far back as iOS 3.2 or Android 1.6.

    The full compatibility matrix is here. It does not list BB because it is still in beta. It does not list Windows CE or any Windows Mobile version because there is no support for it period – including WM7 – nor is there a beta for it to my knowledge.
