Close All Windows At Once

Hi,

In iOS there is a default behaviour, when you navigate more than one level in a window that is assigned on a tab as its main window, and then click on the tab, it goes right to the main window of this tab (no matter how deep you have navigated.

I am trying to achieve this to android and although I can close all windows one by one, I do not want to do it because you can see the animation of each tab (even if you do not assign any animation on close) and it is pretty ugly.

Is there any default functionality in ANDROID that does this? If not, any thoughts on how to do this without this ugly animations ruining my application?

Thanks in advance.

    I believe you said it best when you said: In iOS there is a default behavior...

    Android does not have the same default behavior so it would be counter productive to go against the native platform itself.

    That being said, if you really insist on going that route, you could always hide your windows before closing them. That would prevent the animations from being seen.

    But, what I did and works fine is closing them starting from the 2nd to the last (leaving the first one that is the main window on the tab). No animation issues this way and no hiding which also does an ugly flicker. Only animation that you can see is the normal animation of the last window (the one you are viewing the time you want to close all of them [the last one on the line]) when it closes.
