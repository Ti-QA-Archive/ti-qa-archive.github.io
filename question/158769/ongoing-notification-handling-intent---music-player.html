  • In case anyone is looking for this answer also I managed to get it working.
    You create a module with additional Activity. Ass this activity to timodule.xml.
    After importing module we can create Intent which will start dedicated Activity.

    Problems I had to come through:

    1. Remember that module Activity has to be send to back using:
      moveTaskToBack(true);
    2. When you create Intent in js that will refer to your Java Activity from module use:
      Ti.Android.createIntent({
packageName: "tiapp.xml->id"
className: "package name with the class of your Activity from dedicated module"
});
    3. Use KrollFunction.call if you want to run a JS code

    Hope this helps someone:)
