Alloy, how to setup a bunch of same views

Hi,

I'm trying to jump to alloy.

I have a container view with horizontal layout and I'm adding inside 12 identical views, aligning them with top & left properties (thanks to the horizontal layout of their parent and thus creating some kind of grid layout). Until now, I was using a single line of code inside a for-loop to create them:

for (var i = 11; i >= 0; i--){
  myContainer.add(Ti.UI.createView({top: 5, left: 5}));
};

What's the proper and DRY way to do that in alloy? At the moment I'm doing this:

<Alloy>
    <Window class="container">
        <View id="iconsContainer">
            <View id="icon1"></View>
            <View id="icon2"></View>
            <View id="icon3"></View>
            <View id="icon4"></View>
            <View id="icon5"></View>
            <View id="icon6"></View>
            <View id="icon7"></View>
            <View id="icon8"></View>
            <View id="icon9"></View>
            <View id="icon10"></View>
            <View id="icon11"></View>
            <View id="icon11"></View>
            <View id="icon12"></View>
        </View>
    </Window>
</Alloy>

But this seems quite lame. How should I form this kind of layout?

Application type: mobile
Titanium SDK: 3.0.0.GA
Platform & version: iOS 6.0
Device: iOS simulator
Titanium Studio: 3.0.1
