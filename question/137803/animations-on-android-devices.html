Animations on Android devices

Hi,
width this code I try to animate a logo:

var width = Titanium.Platform.displayCaps.platformWidth;
var toplogo = Ti.UI.createImageView({
    image : '/assets/toplogo.png',
    width : width,
    height : width * 320 / 640, // dimensions of image
});
toplogo.center = {
        x : width / 2,
        y : 0
};
toplogo.animate(Ti.UI.createAnimation({
    center : {
        x : width / 2,
        y : toplogo.height/2 +10
    },
    duration : 3000
}));

This works fine on iPhone, but not on Android.
On Android walks the logo from center of parent window to right bottom and on the end it is on right position.

Whats going wrong?
