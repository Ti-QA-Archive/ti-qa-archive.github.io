How to: Create project without Android SDK installed.

Hi

I want to evaluate this tool by building running simple application for iOS. I have no interest in Android at this time and do not want to install Android dev env. In developer, my iOS SDK is recognised and Android SDK is not found. The create project button is disabled. Is there a way around this?

Thanks.

    no, it appears that the tool scans the specified android path looking for specific executables

    this is the path to a dummy directory I use to get around the issue. It has the basic files that tool is looking for so it allows you to create a project

    https://github.com/aaronksaunders/ti.android.dummy

    download it and point the developer property for Android SDK to it
