Change button style in toolbar

Is it possible to update a button style inside a toolbar (e.g. showing which button was clicked/is selected).
When I try to set _btn1.setStyle(Titanium.UI.iPhone.SystemButtonStyle.BORDERED);
or
_btn1.style = Titanium.UI.iPhone.SystemButtonStyle.BORDERED;
it doesn't change anything. Only thing I can change is the title of the button, no color, no backgroundColor, no style.

var _btn1 = Ti.UI.createButton({title: "test",style: Titanium.UI.iPhone.SystemButtonStyle.DONE});
var _toolbar = Ti.UI.iOS.createToolbar({items:[_page1,...some more...],top:0,borderTop:false,borderBottom:true});

Any tips on how I can change the look of the button?

  • As Nick M. said, you have to re-create the button. I had to do this on a table view, where I wanted an Edit/Done button to switch back and forth easily.

    var btnEdit = Titanium.UI.createButton({
    title: 'Edit',
    systemButton: Ti.UI.iPhone.SystemButton.EDIT
  });

  var btnDone = Titanium.UI.createButton({
     title: 'Done',
     systemButton: Ti.UI.iPhone.SystemButton.DONE    
  });

  btnEdit.addEventListener('click', function(e) {
   // Turn ON editing!
   table.setEditing(true);
   win.setRightNavButton(btnDone);
  });

  btnDone.addEventListener('click', function(e) {
    //turn OFF editing
    table.setEditing(false);
    win.setRightNavButton(btnEdit);
  });


 // Add the edit button to the top right of the screen by default.
  win.setRightNavButton(btnEdit);

    Hope that helps!
