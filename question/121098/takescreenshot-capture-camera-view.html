  • Yeah. Unfortunately, this is impossible. I tried a similar thing a while ago to create a kind of webcam app.

    It's to do with the way the screen/video data is available. It used to be possible to do this, but Apple changed some things to stop it happening.

    There is a way around it though using new 'AVFoundation' techniques. If someone knew about all this, they'd be able to easily create a module to accomplish this.

    The TiBar scanner app uses this new way to be able to look at what the camera is seeing to determine a barcode.
