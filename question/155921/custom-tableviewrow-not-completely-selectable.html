Custom TableViewRow not completely selectable!

Hi,
I made a simply custom TableViewRow, but i have a problem.
If i Tap or Click on the row nothing happens, more precisely if i click on the Label or Image that i have inserted in it.
If i click a little less outsite of label or image im able to select the row.

This is the code:

var row = Titanium.UI.createTableViewRow({backgroundColor:'#eee',zIndex:99});

var rating_component =  Titanium.UI.createImageView({
                image:'http://www.mysite.com/images/image.png',
                width:40,
                height:40,
                right:3,
                top:5,
                bottom:5,
                });

var text_row = Titanium.UI.createLabel({
                text: name,
                font:{ fontSize:10, fontWeight:'bold' },
                color: '#333',
                width: 300,
                textAlign:'left',
                top: 'auto',
                bottom: 'auto',
                rigth:2,
                });

row.add(rating_component);
row.add(text_row);

Anyone know how to fix it?
With the zIndex? or what?
thank you.

    • Btw, this is the solution rather than a fix.

      All views, labels etc default to accepting touch events, if one view covers another it may block the bottom one from receiving the event - as in your example. Your requirement was visual benefits only, so you can and should turn off the event processing with touchEnabled: false.

  • I have founded a solution by myself:

    Simply set touchEnabled to false on all the components in the row, except obviously the row.

    touchEnabled:false,

    So the code now is: 

    var row = Titanium.UI.createTableViewRow({backgroundColor:'#eee'});

var rating_component =  Titanium.UI.createImageView({
                image:'http://www.mysite.com/images/image.png',
                width:40,
                height:40,
                right:3,
                top:5,
                bottom:5, 
                touchEnabled:false    // this is the fix
                });

var text_row = Titanium.UI.createLabel({
                text: name,
                font:{ fontSize:10, fontWeight:'bold' },
                color: '#333',
                width: 300,
                textAlign:'left',
                top: 'auto',
                bottom: 'auto',
                rigth:2,
                touchEnabled:false    // this is the fix
                });

row.add(rating_component);
row.add(text_row);
