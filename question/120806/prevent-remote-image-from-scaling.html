prevent remote image from scaling

I am pulling in an remote feed and need to populate a 3x2 thumbnail grid. I want all the images to be 100x100 but I don't want the image to be resized. How can I go about it? I want something like this if it was done in CSS.

div {
height: 100px;
width: 100px;
overflow: hidden;
background: url('http://www.foo.com/bar.jpg');
}

  • var image = Titanium.UI.createImageView({
  url:'myimage.png',
  width:100,
  height:100
});
view.add(image);

    • This works if the remote images are already 100x100, however if the images arent already that size, or that aspect ratio, then the above code will resize the image.

      In the css example, the image doesn't get resized or scaled, it basically gets cropped by hiding the overflow.

      sidenote 'url' property is deprecated, use the 'image' property.
