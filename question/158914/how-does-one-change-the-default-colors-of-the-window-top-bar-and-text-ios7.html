How does one change the default colors of the Window top bar and text? (iOS7)

By default, it's a grayish color…?
I've set "barColor:" to the color i want and it's working…
How do I change the text color of that top bar?

Also, how would I change the color of the operating system text at the top such as "CARRIER" and the TIME and various icons at the top

  • Figured it out…

    Setting this in app.tss:

    "Window": {
  barColor:"#000",
  tintColor:"#FFF"
}

    Would set the top bar color BLACK and the TEXT WHITE for the status bar and the bar itself.

  • Try:

    var win = Ti.UI.currentWindow;
win.barColor = '#000000';

var tit = Titanium.UI.createLabel({
    text : 'Title',
    color : '#ffffff',
    font : {
        fontSize : 20,
        fontWeight : 'regular',
        fontFamily : 'Segoe UI'
    }

});
win.setTitleControl(tit);
