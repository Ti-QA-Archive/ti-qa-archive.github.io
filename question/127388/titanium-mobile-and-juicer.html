  • IMHO, you should try CommonJS & "require" approach. It allows lazy loading of components. Using Ti.include (with or without Juicer) is not so good approach because you can't do lazy loading, instead you need to preload everything you will use in the app. That's not a problem for small number of files, but with file number growth application load time starts to increase considerably.
