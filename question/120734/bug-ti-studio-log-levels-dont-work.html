Bug: Ti Studio Log Levels don't work

This is a big issue for us developers using Ti Studio. The log levels that you set in debug configuration for each simulator don't actually do anything.

In Android simulator, it seems that log level is set to the most verbose setting, as you see all of this junk that you don't care about. It's hard to see the custom debug messages that you have in your app, as you're getting spammed about freed up memory 20 times per second. I tried setting it to the Warning level, yet it still shows Info messages, which proves that it doesn't care what the Ti Studio setting is.

In IPhone simulator, it doesn't show the Info messages at all that you have in your app.
