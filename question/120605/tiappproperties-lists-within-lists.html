Ti.App.Properties Lists within Lists

Hi there,

Is it possible to store a list in properties that contains x amount of lists, which in turn contain x amount of objects? i.e.

var product1 = new product();
var product2 = new product();
var product3 = new product();

var productGroup1 = [product1, product2, product3];
var productGroup2 = [product4, product5, product6];
var productGroup3 = [product7, product8, product9];

var productGroups = [productGroup1, productGroup2, productGroup3];

Ti.App.Properties.setList('productGroups', productGroups);

productGroups = Ti.App.Properties.getList('productGroups');

  • If new product returns an hashtable, yes. If it returns an UI element, no.
    I usually do this storing everything in a JSONed string

    var a = {list:[{a:'A', b:'B'},{a:'AA',b:'BB'}]};

Ti.App.Properties.setString('productGroups', JSON.stringify(a));

productGroups = JSON.parse(Ti.App.Properties.getString('productGroups'));
