SDK config error Windows 7

Titanium Studio is not detecting the Android SDK. Error reads,
"An error was detected with the SDKs configuration. Reason: [Android SDK Home] No Android SDKs were found under the specified SDK location."
It is in fact pointing to the correct location. It looks like this is a common problem on Windows (I am running Windows 7).

  • Hello, Paul, what Titanium SDK you are using, Please check if you have Java 1.7 installed. If so then uninstall it and install 1.6. Also looks like you may don't have an avd.py script at the location it's looking. To solve that please reinstall the current sdk you are using. To do that download titanium sdk jip file.

    • From Studio, select the menu item, Help > Install Titanium SDK
    • The Install dialog appears. Select Install From URL
    • Browse the file path.

    Restart the studio. hope it will solve the problem. Thanks.
