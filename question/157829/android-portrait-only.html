Android Portrait Only

Need some assistance with this issue that I have not been able to resolve.

I have the following in my home screen

    var self = Ti.UI.createWindow({
        navBarHidden : true,
        backgroundImage : Ti.Filesystem.resourcesDirectory + "images/bg.png",
        orientationModes : [Titanium.UI.PORTRAIT]
    });

and have tried the following in my tiapp.xml but the protrait feature is not working.

<activity android:screenOrientation="portrait" android:configChanges="keyboardHidden|orientation" android:name="ti.modules.titanium.map.TiMapActivity"/>

I have on app that is on the Google Play that is fine, but now it's not working, so I assume the update may be the reason for the issue, but not sure. In any case, can someone please assist? Thanks.

Titanium Studio Build 3.1.3

  • You can put this in a functions file and call it at the top of each js file

    function androidPortrait(){
        if (Ti.Platform.osname == 'android'){
                    Ti.Gesture.addEventListener('orientationchange', function(e) {

                      Ti.Android.currentActivity.setRequestedOrientation(Ti.Android.SCREEN_ORIENTATION_PORTRAIT);
                    });
                }
}
