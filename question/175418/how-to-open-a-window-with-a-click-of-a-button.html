  • Hi guys,

    If you want to open a new window which has a login.js file, you have to create a object of that file then open. Here is an example code.
    Suppose that is (ui/common/login. js) location

    function login()
{

    var win = Ti.UI.createWindow({
        backgroundColor: 'red'
    });

    return win;

}
module.exports = login;

    and app.js file will be

    var loginWin = require('ui/common/login');

var login = new loginWin();

loginbtn.addEventListener('click', function() {
    login.open();
});

    If you have any question about this query then let me know.

    Thanks.

    • Thanks for your reply. Here very important part is location where you put the login.js file. You have to require that location correctly otherwise it will show module not found.

      Here you will put the login.js file

      ('ui/common/login');  // ui is a folder and  common is sub folder

      You can directly put the login.js file under the resource folder.
      In this case just change the app.js file to

      var loginWin = require('login');

      And app.js file is 

      var win = Ti.UI.createWindow({
    backgroundColor : 'Green'
});

var loginWin = require('ui/common/login');

var login = new loginWin();

// Create a Button.
var btnLogin = Ti.UI.createButton({
    title : 'btnLogin',
    height : Ti.UI.SIZE,
    width : Ti.UI.SIZE,
    top : 20,

});

// Add to the parent view.
win.add(btnLogin);

btnLogin.addEventListener('click', function() {
    login.open();
});

win.open();
