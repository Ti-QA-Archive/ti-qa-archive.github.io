Alloy TableVIew Link of Class SearchView Failed

On Android(10) and Titanium(3.1.1.201306112235) application fails to open a new window which contains a tableview. Looks like it can't find a serchview widget? The console message displayed is:

Note, this appears to work fine with iOS.

[INFO][dalvikvm( 1815)] Failed resolving Lti/modules/titanium/ui/widget/searchview/TiUISearchView; interface 624 'Landroid/widget/SearchView$OnQueryTextListener;'
[WARN][dalvikvm( 1815)] Link of class 'Lti/modules/titanium/ui/widget/searchview/TiUISearchView;' failed
[ERROR][dalvikvm( 1815)] Could not find class 'ti.modules.titanium.ui.widget.searchview.TiUISearchView', referenced from method ti.modules.titanium.ui.widget.TiUITableView.processProperties
[WARN][dalvikvm( 1815)] VFY: unable to resolve check-cast 1954 (Lti/modules/titanium/ui/widget/searchview/TiUISearchView;) in Lti/modules/titanium/ui/widget/TiUITableView;

index.xml:

<Alloy>
    <Window class="container">
        <Button top="10" onClick="doClick">Click Me</Button>
    </Window>
</Alloy>

index.js:

function doClick(e) {  
       Alloy.createController("win").getView().open(); //called but never opened
}

$.index.open();

win.xml:

<Alloy>
    <Window>
        <TableView>
            <TableViewSection>
                <TableViewRow>
                    Hello!
                </TableViewRow>
            </TableViewSection>
        </TableView>
    </Window>
</Alloy>

    There's a couple things wrong here. For one thing, you can't just put text inside of a <TableViewRow> element. You're probably trying to do this:

    <TableViewRow title="Hello!"/>

    The reason you don't think the "win" controller is opening on Android is because you have no populated the table and you have not made "win" a heavyweight window. This causes your "win" controller to basically appear as a transparent view on top of your index controller. If you look close, you'll see a single grey row separator. If you actually want something to show up, do something like this:

    app.tss

    'Window': {
    backgroundColor: '#fff',
    modal: false // make all Windows heavyweight
}

    index.xml

    <Alloy>
    <Window>
        <Button onClick="doClick">Click Me</Button>
    </Window>
</Alloy>

    index.tss

    '#index': {
    exitOnClose: true
}

    index.js

    function doClick(e) {
    Alloy.createController("win").getView().open();
}

$.index.open();

    win.xml

    <Alloy>
    <Window>
        <TableView>
            <TableViewSection>
                <TableViewRow title="Hello!"/>
            </TableViewSection>
        </TableView>
    </Window>
</Alloy>

    In short, everything is working as expected, you just need to setup the UI correctly so it renders something visible.
