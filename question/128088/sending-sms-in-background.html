Sending SMS in background

How can I send an sms in the background of the app. This is meant for verification of the handset.

Right now I can only seem to open the intent ACTION_SENDTO, but that isn't good enough since the user will have to press send them selves and we're trying to minimize the amount of tampering of the content of the sms.

Any help or tips on how to resolve this will be greatly appreciated.

  • There's no way to send a SMS without user intervention by using Android Intents. The native Android code in Java for doing so is relatively trivial, though:

    found here: http://thinkandroid.wordpress.com/2010/01/08/sending-sms-from-application/

    SmsManager sm = SmsManager.getDefault();
// HERE IS WHERE THE DESTINATION OF THE TEXT SHOULD GO
String number = "6508570720";
sm.sendTextMessage(number, null, "Test SMS Message", null, null);

    You'll likely need to create a simple native Android module that exposes this functionality to the Titanium Javascript API. The following 2 links are to the Android module development documnetation, as well as the most recent episode of Forging Titanium where I show how a simple Android module can be created:

    http://wiki.appcelerator.org/display/guides/Android+Module+Development+Guide

    http://developer.appcelerator.com/blog/2011/11/forging-titanium-episode-14-twisti-part-1-android-module.html
