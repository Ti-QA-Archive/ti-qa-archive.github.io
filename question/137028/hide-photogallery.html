hide PhotoGallery

Hi guys and gals, This is sort of a half question / half answer.

I've noticed that in the Ti Docs you can set autohide to true / false for ''openPhotoGallery". However there is no call to hide the Photo Gallery like there is with the camera (Ti.Media.hideCamera).

I didn't know whether it was undocumented in the API docs so I went searching in the MediaModule.m
in the SDK.

I could not find anything in there for hiding the photogallery. I don't know if there is an official method that i might have missed if anyone can provide me with the details. So for the time being I added my own.

just under -(void)hideMusicLibrary:(id)args in MediaModule.m I added:

-(void)hidePhotoGallery:(id)args
{
    ENSURE_UI_THREAD(hidePhotoGallery,args);
    [self closeModalPicker:picker];
}

So now I can call Ti.Media.hidePhotoGallery to hide the modal picker when I want

Hopefully I have written the Obj C code right lol I am slowly learning. I Hope Titanium can officially add
this so I don't have to keep add this code with every new SDK release.
