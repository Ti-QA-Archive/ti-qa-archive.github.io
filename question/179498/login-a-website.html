login a website

I want to get some html data from a website, problem is I have to go through a login page(username and password only). Is it possible to login a website using Titanium.Network.createHTTPClient() or is there another way? example would be logging in to facebook to get my top news(i am only using facebook as an example, i am using an different url)

  • Accepted Answer

    You can use webview to show login page. If you don't want to show login page set its height/width to 0 and try to login usnig evalJS. Once its login, you will get html using webview.getHtml().

    Using HTTPClient you cant login into web page. For this to work with server, you need one api/web service call which will in return give you html as response.
