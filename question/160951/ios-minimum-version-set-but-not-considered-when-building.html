iOS minimum version set but not considered when building

I already have this in my tiapp.xml

<ios>
        <plist>
            <dict>
                <key>UISupportedInterfaceOrientations</key>
                <array>
                    <string>UIInterfaceOrientationPortrait</string>

                </array>
            </dict>
        </plist>
        <min-ios-ver>5.0</min-ios-ver>
    </ios>

however I get this error when trying to install my app on my iPhone 4 runnng iOS 5.0.1

The device is running iOS 5.0.1, however the app's the minimum iOS version is set to 6.0
In order to install this app on this device, lower the <min-ios-ver> to 5.0 in the tiapp.xml:
[ERROR] :  This app does not support the device "iphone"
<ti:app xmlns:ti="http://ti.appcelerator.org">
    <ios>
        <min-ios-ver>5.0</min-ios-ver>
    </ios>
</ti:app>

How do I set the target version so I can Install the app on my device?
