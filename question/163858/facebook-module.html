    Hello Roger,

    Thanks for raising this issue. You got this error because Titanium.Facebook is deprecated from Ti SDK 3.1.0. In Titanium 3.1, the Titanium.Facebook module on Android and iOS is replaced by the Modules.Facebook add-on module that uses more recent versions of the native Facebook SDKs for Android and iOS. For Mobile Web, continue to use Titanium.Facebook.

    Please test my following code. I hope its solve your problem.

    
var fb = require('facebook');
fb.appid = "xxxxxxxxxxxxxxxx";
fb.permissions =  ['publish_stream'];
fb.authorize();

bsFacebook.postMessage = function(){
    fb.requestWithGraphPath('thevales.net/feed', {message: "Trying out FB Graph API using titanium API and it's fun android phone too!"}, "POST", function(e) {
        if (e.success) {
            alert("Success!  From FB: " + e.result);
        } else {
            if (e.error) {
                alert(e.error);
            } else {
                alert("Unkown result");
            }
        }
    });

    For more information please visit: Modules Facebook and Titanium Facebook

    Thanks

  • Hi, roger vale

    You are using http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.Facebook-method-requestWithGraphPath
    and this part is deprecated with Ti SDK 3.1.0 and above it.

    You need to use this http://docs.appcelerator.com/titanium/latest/#!/api/Modules.Facebook . and use this way

    fb.requestWithGraphPath
