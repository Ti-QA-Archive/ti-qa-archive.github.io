  • Thats very simple

    If you want to have a simple list Kinda UI you simply use row in the tablevie…(allows a simple text with an image either set to the left or right using the property).

    but if you wanna have a table view that has rows with custom design you have to construct the custom row using the Tableviewrow and assign them to the table view…

    A more specific example would be the Kitchen sink itself, all the menus uses a simple table view with a no complex rows thus row in the tableview is used but if you drill down to the codes of Tableview Layout examples you can see that they might have used Tableview rows to construct the rows…
