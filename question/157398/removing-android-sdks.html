    • Since Titanium SDK 3.1.2 the minimum supported Android version is 2.3.3 (api 10).

      As I rewrite the Android build scripts to be Node.js instead of Python, I'm going to see if we can get rid of this ridiculous requirement.

      Pro tip: If you're looking for other places to save space, you can run android avd and delete any AVDs that you're not using. Each one is probably eating up about 140MB. Then in the .titanium folder, each AVD's sdcard is eating up another 64MB. You probably have some sdcard files that don't have a corresponding AVD.

      You'll have to manually audit the sdcard files to see which ones you should delete. You could delete all AVDs and all sdcards, then the next time you build for Android, it will create a new AVD (though this behavior is changing in Titanium SDK 3.2.0).
