date Picker and maxDate

I am using a Titanium.UI.Picker of type Titanium.UI.PICKER_TYPE_DATE to capture the user's birth date.

Obviously, with a birth date, you need the ability to restrict the date to not being greater than today (or not greater than whatever date your youngest user would have).

In the old documentation, I found a property called "maxDate", and I'm using it, and it works.

However, in the latest documentation, "maxDate" has disappeared. This concerns me: is this property deprecated or something? Asking because I don't want my app to rely on outdated API calls.

  • Marc, if you are using the newest version of Titanium Studio, the Console should tell you when you Run/Compile your app if various code in your application is deprecated.

    Something I've found is that Appcelerator's online documentation is pretty spotty…it's usually hit or miss. I would rely more on what Studio outputs.
