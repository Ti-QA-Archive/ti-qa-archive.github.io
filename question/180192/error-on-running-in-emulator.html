  • Hello Rizvi,

    I have researched about your issue. There is some problem in your "tiapp.xml" file. Use the following code in your tiapp.xml file

    <android xmlns:android="http://schemas.android.com/apk/res/android">
    <manifest>
        <application android:theme="@style/Light"/>
        <uses-sdk android:minSdkVersion="14" android:targetSdkVersion="19"/>
    </manifest>
</android>

    And obviously conform that in platform/android/res/values/builtin_themes.xml file is available. For more details see Android Themes. If you have further query feel free to ask.
