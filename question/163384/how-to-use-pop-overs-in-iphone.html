  • Hi

    The UI element called "popover" is native to the iPad only - not even iPhone.

    You can make your own equivalent version though by using a view and styling it to meet your design needs. If you want to mask the window contents then make a view that covers the entire the contents and even give it a background colour that allows a partial translucency - you can add a click event to this to dismiss the popover. Add an internal view to this to add your own UI elements, labels and buttons for example.

    Pretty much all UI elements are made up of views in one way or another - so this is a very standard thing to do.

    If you only require a window to be displayed over the top of another window that you later dismiss - make sure to add the modal property and set it to true. Only this window is closed the one beneath would not be closed until the top one has been.
