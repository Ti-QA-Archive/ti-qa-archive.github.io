Can you not build in the cloud no more?

It has been a year and a half if not longer since I used Titanium and yesterday I downloaded the latest version and started working with it again. I remember in the past when building a desktop project on a pc I could do a build for Mac that was done in the cloud and when finished I could download the .dmg file.

Is this feature not available anymore with Titanium Studio? or am I just not finding it?

  • You can no longer build for desktop in Studio. We have given the desktop project to the community and are no longer officially managing the project. For the latest on the current desktop project, check out the TideSDK website: http://www.tidesdk.org/
