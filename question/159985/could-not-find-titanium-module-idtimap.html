Could not find Titanium module id=ti.map

I was trying to use the ti.map module on iphone and the bellow error shows up on my console:

[ERROR] : Could not find all required Titanium Modules:
[WARN] : Could not find Titanium module id=ti.map version=latest platform=iphone deploy-type=development
[ERROR] : id: ti.map version: latest platform: iphone deploy-type: development

I'm using the Titanium SDK 3.1.3GA.
Inside the folder "modules", there is no map inside the folder "iphone".

How can I fix this?

  • In tiapp.xml:

    <modules>
        <!-- Add this line to your modules section -->
        <module platform="iphone">ti.map</module>
</modules>

    Always clean the project when adding modules and/or new Ti.* portions of the API to ensure your changes are picked up.

  • To try and be a bit more helpful, you can get the IOS ti.map module from:

    https://github.com/appcelerator-modules/ti.map/blob/stable/ios/ti.map-iphone-2.0.0.zip?raw=true

    Then do as above.

    This won't work with 3.1.3GA though, so you need to install the 3.2 SDK using the CLI:

    titanium sdk install –branch 3_2_X

    See https://github.com/appcelerator/titanium for more info on the CLI.

    It won't build via Titanium Studio (gives an error about 7.0.0 being an invalid version, must be 7.0.3), but building works fine from the CLI:

    titanium build -p ios
