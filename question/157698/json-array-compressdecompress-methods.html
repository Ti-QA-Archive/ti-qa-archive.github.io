Json Array compress/decompress methods

Working Enviroment- Titanium studio-3.1.0, iOS-6.1

I have to retrieve json array from a server.I had uses PHP web services. It works fine. The only problem I have, is that I have to retrieve a lot of records to store in local database and it produce a quite huge json file. Is there a way to compress the json in server side (PHP) and decompress it on the app on unload method before opening it and viceversa ? and how ?

Please give me in details about the json array compress/decompress methods for improve the performance of app..

  • If you are using the httpClient (for iOS) then this already supports gzip compression in its request headers, so you just need to configure your web server to allow your json files to be gzipped (via its Mime-Type)
