    You have your strings.xml file in 

    {project_folder}/i18n/es

    Is the emulator set to use spanish language? If not, do you have the english fall back file?

      If you dont want to go to the trouble of an entire en/strings.xml, you can pass a default value to L()

      L('my_tag','My Default Value');

      NOW, that being said, I often don't put the defaults in, so that as we go through testing, we can see which strings are not being replaced and can take note.
