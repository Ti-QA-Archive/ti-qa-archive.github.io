  • Yes, this is called a SplitWindow and exists in the UI.iPad namespace. You can see the API for this in the docs here: http://docs.appcelerator.com/titanium/2.1/index.html#!/api/Titanium.UI.iPad.SplitWindow along with the rest of the APIs here: http://docs.appcelerator.com/titanium/2.1/index.html#!/api

    • You don't have to write 2 different programs, you simply need to write some platform specific code to handle Android tablets. Depending on the rest of your code, its possible the rest of your app may be cross-platform. Remember, one of the most powerful features of Titanium is that it allows you to build your cross-platform apps using a SINGLE code base. So there are very few cases where you'll ever need to write two different apps.
