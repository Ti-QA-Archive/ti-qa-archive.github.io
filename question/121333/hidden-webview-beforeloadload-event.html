Hidden WebView beforeload/load event

Hi,

I'm trying to create a hidden WebView and get some information when it load. This is my testing code:

var win = Titanium.UI.currentWindow;

var button = Titanium.UI.createButton({ title: 'Load' });

button.addEventListener('click',function(e){
    var webView = Titanium.UI.createWebView({ url: 'http://www.google.com' });
    webView.addEventListener('error', function(e){
        Titanium.UI.createAlertDialog({title:'error', message:e.url}).show();
    });
    webView.addEventListener('beforeload', function(e){
        Titanium.UI.createAlertDialog({title:'beforeload', message:e.url}).show();
    });
    webView.addEventListener('load', function(e){
        Titanium.UI.createAlertDialog({title:'load', message:e.url}).show();
    });
    webView.reload();
});

win.add(button);

But beforeload and load events aren't always triggered.
Can anyone help me to resolve this please?

I'm using Titanium SDK 1.7.0 and iPhone SDK 4.3/4.2.

Thanks!
Ishan.

  • Are you doing any event handling in your webview? A webview can only have either Titanium or native js dom event handling, but not both. If you try to do both, you will usually get some cross-mojonation in your event handling.
