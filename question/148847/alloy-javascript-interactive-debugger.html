Alloy Javascript Interactive Debugger

From http://docs.appcelerator.com/titanium/3.0/#!/guide/Alloy_Debugging_and_Troubleshooting

"The Titanium Studio JavaScript debugger can be utilized to debug your Alloy application. "

Can't get it running under 'Titanium Studio, build: 3.0.2.201302191606'

By right-clicking on any of the Alloy Files on a Project with IOS, Android an Mobile as build target, there is no Option 'Debug as Web Preview' or similar, however a 'Run as Web Preview' option is available.
Trying to define such Debug Config on your own as described here
http://docs.appcelerator.com/titanium/3.0/#!/guide/Installing_the_JavaScript_debugger
has no effect either.

Is there any way to use the JavaScript Browser Interactive debugging Tool or the Aptana JavaScript Firefox debugger for Alloy Projects?

    Alloy projects are not web projects, so they wouldn't use the browser-based debugging, they'd use the TiStudio debugger. To launch the debugger just click the debug icon in the project panel.
