How to recognize iPhone 4 and 5 in Alloy Style Sheet (TSS) ?

Hi all.

I'm looking for the way to recognize iPhone 4 and 5 (I mean screen height) in the TSS file because want to change button's background image files and size between iPhone 4 and 5.

Of course I know I can do it in the controller, but I'd love to do it in TSS files.

I'm looking for the way something like this…

index.xml

<Alloy>
    <Window>
        <View id="wrapper">
            <Button id="button_1" />
        </View>
    </Window>
</Alloy>

index.tss (It's JUST example what I WANT)

// for iPhone 4
"#button1[platform=ios formFactor=max-height:480dp]": {
    backgroundImage: '/images/button1-small.png',
    width: 300,
    height: 460
},
// for iPhone 5
"#button1[platform=ios formFactor=min-height:481dp]": {
    backgroundImage: '/images/button1-big.png',
    width: 300,
    height: 540
}

Is there any way to do that?

Thanks.

    With Titanium 3.1.0, iPhone 5 specific images are supported. See the Titanium 3.1.0 ReleaseNote, and more particularly this JIRA : TIMOB-11197.

    Thus, if you only specify the width of your image, the height should be calculated automatically…
