Landscape splash screens Android never display

After a little research I have managed to get different resolution portrait splash screens to display as my Android fires up. I achieved that by adding the following directory structure to my titanium projects root directory.

\platform\android\res\drawable
\platform\android\res\drawable-hdpi
\platform\android\res\drawable-ldpi
\platform\android\res\drawable-mdpi

I add a single splash image named background.png and sized correctly to each directory. This works fine, but when I try to add the landscape and portrait version:

\platform\android\res\drawable-land-hdpi
\platform\android\res\drawable-port-hdpi

The portrait splash screens still display correctly but the landscape images are never used. Instead the portrait splash screens are stretched

I am building on windows using 1.7.6 SDK

Anyone got an idea how to make this work?

  • Try naming the file default.png instead. Also, check that you've enabled rotation by setting win.orientationModes = [Ti.UI.PORTRAIT, TI.UI.LANDSCAPE_RIGHT] (or whatever orientations you need).
