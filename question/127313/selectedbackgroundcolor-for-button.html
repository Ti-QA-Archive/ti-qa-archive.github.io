  • this property(backgroundSelectedColor) don't work

    var button = Titanium.UI.createButton({
    title: 'Button 1',
    height: 55,
    backgroundColor: 'white',
    style: Titanium.UI.iPhone.SystemButtonStyle.PLAIN,
    backgroundSelectedColor: 'yellow'
});
win(button);

    • This should work on Android: 

      var button = Titanium.UI.createButton({
    title: 'Button 1',
    height:55,
    width:200,
    backgroundColor: '#ff0000',
    backgroundSelectedColor: '#000'
});
win1.add(button);

      By removing the style property, the selected color becomes available. This property only works on Android as per the docs.
