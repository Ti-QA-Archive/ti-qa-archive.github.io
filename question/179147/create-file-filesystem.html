    Hello Hund,

    I also did not find this location
    file:///data/data/com.vivelabbogota.idtvivelabbogota/app_appdata/text.txt
    inside my device, but I am sure if we can go to that location, we will get that file. Try to search in your phone memory location.

    But if we create a file in the external storage and write something’s to .txt file. We get this file in the correct location.

    Sample Code

    var win = Ti.UI.createWindow({
    backgroundColor : 'white',
    layout : 'vertical'

});

var log = Ti.Filesystem.getFile(Ti.Filesystem.externalStorageDirectory, 'logfile.txt');

log.write('My log file\n');
for (var i = 0; i < 10; i++) {
    log.write(i + ': new log statement\n', true);
    // Boolean argument causes write() to append
}

alert(log.read().text);
alert("location:" + log.nativePath);

win.open();

    If you have any question, then feel free to ask me.

    Thanks.
