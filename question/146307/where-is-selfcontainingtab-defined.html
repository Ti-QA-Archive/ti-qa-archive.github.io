Where is self.containingTab defined?

I found this property in the solution of the 220 Tutorial Lab, but I can't find this property and what it does in the API. Is not documented? Thanks

    Hi

    self.containingTab.open(new AddWindow);

    Is not an actual property, it is a custom property that was added as a convenience so that a new window could be opened later on as a child of the calling window.

    When you want to open a window inside a tab you need to be able to access the tab variable that was used when created. However determining the variable or a reference to the tab after it has been created and no longer in context is not as easy as it could be.

    When we used to use the windows property url it was fairly easy using the currentTab method.

    So the custom property is simply used to act as a reference to the tab when this window was created.

    • Ok, thought I should mention WHY you might do it one way or the other. If when the view is added to a window or when the window is opened on a tab, you may wish to act upon the parent (change title, change icons, etc) or change the window/view based upon which parent its been added to.

      You can't add the property until after the object is created and you may need the reference before then. (reference during creation vs reference after creation).
