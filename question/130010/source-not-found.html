Source Not Found

Hello-

I am developing an app for Android and I am using SDK version 1.7.5. I am trying to debug my application and I am receiving a page entitled [Eval script] and it says "Source Not Found" and has a button that say's edit Source Lookup Path. I tried adding my project under the "Edit Source Lookup Path" button, but after I click OK it just removes the project I just added. Currently it just shows a folder called "Default" in the Source Lookup Path list. For some reason I can run my application without the debugger but this only happens when I try to debug my application. Also, it was debugging just fine, prior to running the application w/o the debugger. Does anybody have any insight as to what may be going on here? has anyone else experienced this issue?

Thanks

  • This error comes often when you have named file illegally or incorrectly, like your file name is 'createMainWindow.js' and you require that file in another file like this 'CreateMainWindow.js' or other spelling mistake, especially case is main problem. So when you compile it, it gives no error but when you debug, it tells you that source file is not found.
