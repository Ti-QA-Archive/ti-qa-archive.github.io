  • Dear Reinhard,

    As i have experienced, you don't need such permission in android and Ti.Media.showCamera(); can be used in android.

    there are few method of implementing in code

    var getImage;
Ti.Media.showCamera({
            success : function(e) {
                var image = e.media;
                var max_width = 800;
                var max_height = 400;
                getImage = imageGetResize(image, max_width, max_height);
                pictureText.image = getImage; // getImage give you image path
            },
            allowEditing : true,
            mediaTypes : [Ti.Media.MEDIA_TYPE_PHOTO]
        });
