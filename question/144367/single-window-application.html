Single Window Application

I'm trying to understand the new structure behind the single window application template. I can't seem to find any documentation besides the template itself on how I should open and close new views within this new way of structuring an app. I have sucessfully managed to open a new window from within FirstView.js with this code: 

var videoButton = Ti.UI.createButton({
    title:'Watch video',
    width:150,
    height:50,
    top:100,
    left:20
});
self.add(videoButton);

videoButton.addEventListener('click', function(e) {
    var videoView = require('ui/common/videoView');

    videoViewWindow = Titanium.UI.createWindow({
        title:'Video window',
        backgroundColor:'white',
        navBarHidden:true
    });

    var videoView = new videoView(videoViewWindow);
    videoViewWindow.add(videoView);

    videoViewWindow.open();
});

By using the back-button on android I manage to clsoe this window, but I can't see how to do this from a back-button in this new window. Are there any help-files or tutorials out there that describes how I should do this?

  • I am not into the template, but its called single and I am sure it's intended to be so. :)
    The templates are just small helpers for beginners, there is no magic behind them.

    If you want to close a window, then call .close() on its instance.
