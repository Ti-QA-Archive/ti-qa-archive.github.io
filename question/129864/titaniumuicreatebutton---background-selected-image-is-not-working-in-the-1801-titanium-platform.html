Titanium.UI.createButton - background selected image is not working in the 1.8.0.1 Titanium platform

Hello.

I have this code that works just fine in the Titanium 1.7.5 but is not working in the 1.8.0.1 platform. The problem is that the code doesn't show the background selected image when the button is selected (running in the iOS simulator). The code is: 

// Login button.
    var loginButton = Titanium.UI.createButton({
        backgroundImage : "images/login_button_on.png",
        backgroundSelectedImage : "images/login_button_off.png",
        top : "300dp",
        width : "260dp",
        height : "46dp"    
    });

But if you add this lines at the end, it works as it should. 

            borderRadius : 0,
        font : {}

Is there a solution for this problem?

    Hi..in 1.8.0.1, I can make it work by adding both attributes like the followings:

    backgroundImage:'pen.png',
backgroundFocusedImage: 'pressed.png',
backgroundSelectedImage: 'pressed.png',

    The image in backgroundSelectedImage will be shown when pressed.

    I don't know why but this works without adding extra listener.
