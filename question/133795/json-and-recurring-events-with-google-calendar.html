JSON and Recurring Events with Google Calendar

I'm currently working on displaying a Google Calendar in an tableView on an iPhone app. I was able to get single events displayed with no problem, but my real issue is with recurring events. I'm pulling from a public Google calendar as a JSON request (Link). I noticed from the JSON code that the recurring events seems to be stored in iCal format:

DTSTART;TZID=America/New_York:20120209T164500\r\nDTEND;TZID=America/New_York:20120209T174500\r\nRRULE:FREQ=WEEKLY;UNTIL=20120503T204500Z;BYDAY=TH\r\nBEGIN:VTIMEZONE\r\nTZID:America/New_York\r\nX-LIC-LOCATION:America/New_York\r\nBEGIN:DAYLIGHT\r\nTZOFFSETFROM:-0500\r\nTZOFFSETTO:-0400\r\nTZNAME:EDT\r\nDTSTART:19700308T020000\r\nRRULE:FREQ=YEARLY;BYMONTH=3;BYDAY=2SU\r\nEND:DAYLIGHT\r\nBEGIN:STANDARD\r\nTZOFFSETFROM:-0400\r\nTZOFFSETTO:-0500\r\nTZNAME:EST\r\nDTSTART:19701101T020000\r\nRRULE:FREQ=YEARLY;BYMONTH=11;BYDAY=1SU\r\nEND:STANDARD\r\nEND:VTIMEZONE\r\n

I was planning on breaking this code apart and parsing, but I'm not certain how this will display in other situations, like if we were repeating events by weeks. I was wondering if someone else has already developed some sort of solution to this that handles different sorts of recurring event requests or maybe has a different way of accomplishing my goal of displaying events in a table view. Thanks for any help in advance.
