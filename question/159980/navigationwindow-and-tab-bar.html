NavigationWindow and tab bar

Hi, i need to view a static tab bar and use component NavigationWindow for only tab.

With navigationgroup ( release 3.1.2 ) it's possible to add a navigationgroup inside window, but from release 3.1.3 it's not possible because the new component navigationWindow doesn't support this function ( is a top-level window ).

How can write an app where from tab is possible to call a component that have feature as navigationgroup ?

thanks for answer,

Gabriele

    Hi Gabriele, TabGroups have built in support for a Navigating between Windows. There's not need for a extra NavigationWindow. 

    var tabGroup = Ti.UI.createTabGroup();
var win1 = Titanium.UI.createWindow({title: "Hello", backgroundColor:'white'});
var tab = Titanium.UI.createTab({
    window:win1,
    title:'Win1'
});
tabGroup.addTab(tab);

var win2 = Titanium.UI.createWindow({title: "Win2",backgroundColor:'green'});
var nextButton = Ti.UI.createButton({title:"Next"});
win1.setRightNavButton(nextButton);
nextButton.addEventListener('click', function() {
    tab.open(win2, {animated:true});
});

tabGroup.open();
