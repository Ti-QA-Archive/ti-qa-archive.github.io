Trying to make a hover-button, with addEventListener

bullshitButton.addEventListener('click', function({

  • Mathias

    You could use the backgroundSelectedImage property of Ti.UI.Button to change the image on click. However, if you want to do it using your approach:

    First note that app.js is failing with an exception because it contains a syntax error:

    bsButton.addEventListener('click', function({

    should be:

    bsButton.addEventListener('click', function(){

    Try the following code in place of yours:

    Ti.UI.setBackgroundColor('#000');

var win1 = Titanium.UI.createWindow({  
    backgroundColor:'#000'
});

var bsButton = Titanium.UI.createButton({
    image: "static/images/bsbutton.png",
    width: 200,
    height: 200
});

bsButton.addEventListener('click', function({
    bsButton.image = "static/images/bsbuttonhover.png";
});

win1.add(bsButton);

// I dont think this method exists in the Ti API
//win1.hideNavBar();

win1.open();

    Hope this helps
