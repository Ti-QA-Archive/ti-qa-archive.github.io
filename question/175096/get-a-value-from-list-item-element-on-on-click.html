  • I suggest you start by reading the eventing section of the ListView guide. You might also read the ListView vs TableView section of that guide. ListView events do not work like those with other UI components because ListViews are data-oriented, not view-oriented. It takes some getting used to, but is worth it in the end. ListViews are much faster and more flexible than TableViews.
