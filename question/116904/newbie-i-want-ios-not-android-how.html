Newbie: I want IOS, not Android... how?

Hi,

No doubt this is a dumb question but I just installed, tried to create a new project and Appcelerator believes it's detected both IOS and Android SDK's. No idea where the Android SDK came from. I just want to build an app for the iPhone.

So when I try to create a new project for 'mobile', I get to the 'create' button and get the error that the Android SDK isn't set up correctly.

But I don't care! I don't want to build for Android. Just IOS. So how on earth do I do this? I didn't expect to hit a brick wall quite so quickly - this is crazy! Please help!

Jamie.

  • I found out the answer. Apparently you have to create the project using the 1.2 SDK and then update it to 1.6 afterwards.

    This really sucks for new users - I bet some have tried Appcelerator out, hit this wall right away and decided it's just not ready yet. Really frustrating first hurdle - hope things get better…
