How do I lock an Alloy app to a portrait orientation?

I have removed the landscape orientation properties in Tiapp.xml like so: 

        <orientations device="ipad">
            <orientation>Ti.UI.PORTRAIT</orientation>
                <orientation>Ti.UI.UPSIDE_PORTRAIT</orientation>
<!--        
            <orientation>Ti.UI.LANDSCAPE_LEFT</orientation>
            <orientation>Ti.UI.LANDSCAPE_RIGHT</orientation>
-->
        </orientations>

but my app still rotates to landscape…any ideas what I'm doing wrong?

  • Hi

    I would physically remove the entries rather than simply trying to comment them out. This should have worked.

    I would also add the orientationModes inside each window create (MUST be during the creation of the window).

    var win = Ti.UI.createWindow({
    orientationModes: [
        Ti.UI.PORTRAIT,
        Ti.UI.UPSIDE_PORTRAIT
    ],
...
});

    There is a guide to orientation.
