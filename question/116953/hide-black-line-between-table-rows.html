Hide black line between table rows

Hi, is there way to hide black line between table rows on Android?

  • Accepted Answer

    The "separatorColor" property. If your backgroundColor of the TableView is, for example, "#fff", simply set the separatorColor property to "#fff" and it will blend in with the background.
