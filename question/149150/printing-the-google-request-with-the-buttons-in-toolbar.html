  • Hi

    I think your problem is a simple one. You have referenced several properties that do not exist on the webView, they do on a textField and textArea but NOT on the webView.

    var MyWebView = Ti.UI.createWebView({
   keyboardType : Ti.UI.KEYBOARD_NUMBER_PAD, // no such property
   keyboardToolbar : [giveMeKeyboardToolbar()], // no such property
   keyboardToolbarColor : '#999', // no such property
   keyboardToolbarHeight : 40, // no such property
   url: 'http://www.google.com'
});

    There is no current way to add custom buttons into the webViews use of the keyboard inside HTML forms.

    Documentation for the webView view.

    • webView

    Keyboard properties ARE available on these two views, but they will not help you with the webView

    • textField
    • textArea
