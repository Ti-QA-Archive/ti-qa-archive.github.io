dark cursor for textfield in Android

Hi, since the cursor is white on Android by default it is not visible if you set a white background for the textfield. I read a solution here https://developer.appcelerator.com/question/164315/cursor-color-in-a-textbox but I need to change the theme to Theme.AppCompat instead of Theme.Custom and do not know how?

tiapp.xml

<android xmlns:android="http://schemas.android.com/apk/res/android">
     <manifest>
        <application android:theme="@style/Theme.Custom"/>
    </manifest>        
    </android>

styles.xml

<?xml version="1.0" encoding="utf-8"?>
<resources>
    <style name="Theme.Custom" parent="@android:style/Theme.Holo.Light">
            <item name="android:editTextStyle">@style/editText</item>
    </style>

    <style name="editText" parent="@android:style/Widget.Holo.Light.EditText">
        <item name="android:textCursorDrawable">@null</item>
        <item name="android:textColorHint">@color/cursor</item>
    </style>
</resources>

colors.xml

<?xml version="1.0" encoding="utf-8"?>

<resources>
    <color name="cursor">#FFBEBEBE</color>
</resources>

  • Hello Rieger,

    Thanks for your inquery. Please follow the below instruction for creating dark cursor for textfield in Android.

    Step-1: Modify your tiapp.xml using the code below.

    <android xmlns:android="http://schemas.android.com/apk/res/android">
        <manifest>
            <application android:theme="@style/Theme.AppCompat.Light"/>
            <!-- Need to specify at least API level 11 for Titanium SDK 3.2.x and prior -->
            <uses-sdk android:minSdkVersion="14" android:targetSdkVersion="19"/>
        </manifest>
    </android>

    Step-2: Create a styles.xml file and a colors.xml file in [Project]/platform/android/res/values, relative to the main project folder.

    Step-3: In styles.xml, add the following code.

    <resources>

    <style name="Theme.Custom" parent="@android:style/Theme.Holo.Light">
            <item name="android:editTextStyle">@style/editText</item>
    </style>

    <style name="editText" parent="@android:style/Widget.Holo.Light.EditText">
        <item name="android:textCursorDrawable">@null</item>
        <item name="android:textColorHint">@color/cursor</item>
    </style>
</resources>

    Step-4: Finally in colors.xml, add the following code.

    <?xml version="1.0" encoding="utf-8"?>

<resources>
    <color name="cursor">#000</color>
</resources>

    Hope this process solve your problem. If you have any other query feel free to ask.Thanks.
