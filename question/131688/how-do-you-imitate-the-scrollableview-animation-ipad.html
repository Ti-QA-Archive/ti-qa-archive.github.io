How do you imitate the scrollableView animation? [iPad]

I'm trying to create a nav that sits to the left of the content. I want to be able to create a handle on that nav so you can push it to the left to almost hide it and then pull it back out when needed. Basically sliding it left and right similar to the way the scrollableView does but using a window.

Any ideas on how to create this animation and have it work as smooth as the scrollableViews animation does?

  • Have you tried animating a standalone view in and out?

    var win = Ti.UI.createWindow();
var view = Ti.UI.createView({
width:300,
height:300,
left:-280
});
win.add(view);
win.open();
view.addEventListener('click', function() {
var a = Ti.UI.createAnimation();
a.left = 0;
a.duration = 500;
view.animate(a);
});

    And to the inverse when you want to hide the view. is this something you were looking for?

  • I already tried what you suggested but trying to use a touchmove to incrementally move the window off screen. I was looking for something that you could swipe closed yourself just like the way a scrollableview swipe control works. I figured out a simple solution by actually using the scrollableView.

    I created a scrollableView with a second invisible view so that when you swipe it, it looks like it goes completely off screen. Then I set an animation to move that whole window off screen, set the currentPage back to the first view and animated the window back on screen just enough to see that the nav is still there. Then I set a click event listener on the window so that it animates back out if you touch it. simple enough! Thanks though.
