Alloy $.view.animate();... Complete Event?

Hey everybody!

I am running this piece of code:

$.someView.animate( { opacity: 1.0, duration: 500, complete: onSomeViewAnimateComplete } );

function onSomeViewAnimateComplete ( event )
{
    Ti.API.info( 'onSomeViewAnimateComplete...' );
};

The animation works a treat, but the complete function is not called. What am I doing wrong?

  • try this

    $.someView.animate( { opacity: 1.0, duration: 500}, function(){
      onSomeViewAnimateComplete();
} );

function onSomeViewAnimateComplete ( event )
{
    Ti.API.info( 'onSomeViewAnimateComplete...' );
};

    or

    $.someView.animate( { opacity: 1.0, duration: 500}, onSomeViewAnimateComplete);

function onSomeViewAnimateComplete ( event )
{
    Ti.API.info( 'onSomeViewAnimateComplete...' );
};
