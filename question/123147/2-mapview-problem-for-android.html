  • Hello han,

    First, you have to create mapview object in app.js and pass it to both windows which will use it as you can see in begin.js in my comment.
    In map1 and map2 files you are adding annotations specific for that map.
    Then, to be able to draw a map on second window you have to remove it from the first one. I'm doing it on changing focus event of tabGroup.

    tabGroup.addEventListener('focus', function(e)
{
    Ti.API.debug('tab changed to ' + tabGroup.activeTab.title);

    if (tabGroup.activeTab.title=='Map'){
        Titanium.App.fireEvent('removeMap');
    }
    if (tabGroup.activeTab.title=='Branches'){
        Titanium.App.fireEvent('refreshMap');
    }
});

    In map1.js you should have listeners for remove and refresh map:

    Titanium.App.addEventListener('removeMap', function(e){
    if(win.children[0]=='[Ti.Map.View]'){
        win.remove(win.children[0]);
    }
});

Titanium.App.addEventListener('refreshMap', function(e){
    win.map.removeAllAnnotations();
    //your code for adding new annotations
    win.add(win.map);
});

    Hope this helps.
