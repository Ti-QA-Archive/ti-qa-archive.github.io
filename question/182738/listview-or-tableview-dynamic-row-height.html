listview or tableview dynamic row height

hello,
i am using appcelerator studio.i created table view trying to set row height dynamic according to content. but when i set table view height to Ti.UI.SIZE it set height of table to row e.g. if tableview height is 80% the for 1 row it takes that much height. when i set row height to Ti.UI.FILL it takes fixed size to row then if contents are more data is overlapped. i tried heights to auto,SIZE,Ti.UI.SIZE,Ti.UI.FILL but no use. same for list view. If any one knows please help me.

  • This should do the trick:

    Tableview row (View):

    <Alloy>
    <TableViewRow id="row">
        <View class="rowWrapper">
        </View>
    </TableViewRow>
</Alloy>

    Tableview row (TSS):

    "#row" :{
    height:  Titanium.UI.SIZE
}

".rowWrapper" : {
    layout:"vertical",
    height: Titanium.UI.SIZE
}

    Main view:

    <Alloy>
    <Window>
        <TableView>
            <Require src="tableviewRow"></Require>
        </TableView>
    </Window>
</Alloy>
