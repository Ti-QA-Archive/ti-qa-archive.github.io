Save view state

Hi, how can I save a view state?
If i touch and move an object then close the app and relauch it… the object retrieves the last position.

Thanks in advance.

  • Accepted Answer

    The best approach would probably be to use the Titanium.App.Properties

    Once the user has moved the object, simply save its attributes (left, top, width, height, etc.).

    Then, when you create the window (on relaunch), check if there are values for this object in the properties. If there are, simply apply them to the object. If there are no properties (on first launch for example), simply apply the default properties.

    Hope this helps.

  • This is something that you have to handle on your end. The view will always be rendered as you render in on load or relaunch. So, if you want to save the location of something, you can save those values using the Properties API and render your view using those values upon load of your app view.
