  • Just for the history if someone is looking for an iPhone example.

    First on that page u can easliy find the most popular apps u can interact with
    link to IPhone_URL_Schemes

    and here is a little code snipped i'm currently using to check for Facebook:

        var $isIOS = false
    if (Titanium.Platform.name=='ios'){$isIOS=true;};

    if($isIOS==true && Titanium.Platform.canOpenURL('fb://profile') )
        Titanium.Facebook.forceDialogAuth = false;
    else
        Titanium.Facebook.forceDialogAuth = true;
