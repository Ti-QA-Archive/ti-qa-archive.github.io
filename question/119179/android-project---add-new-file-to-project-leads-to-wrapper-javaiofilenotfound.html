    Would you make sure this is the procedure you have followed to force a full rebuild?…

    Delete the contents of the project's build&#x2F;android directory, then touch the tiapp.xml file:

    cd /path/to/root/of/project
touch tiapp.xml

    The above command will work for OSX and *nix.

    If this does not resolve the issue, perform the above procedure again, and see if the project builds correctly if you use Titanium Developer rather than Titanium Studio.
