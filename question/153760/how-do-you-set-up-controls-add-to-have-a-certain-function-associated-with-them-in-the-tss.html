How do you set up controls add to have a certain function associated with them in the tss

I got a # of buttons with a onClick javascript function named "respond" set up for them in my XML like so: 

<Button id="button1" onClick="respond" />

I don't like having to type out onClick="respond" for each of the Buttons and I'd like to make sure that all Buttons have a onClick function set for them in the tss like so: 

"Button": {
    height: 50,
    width: 50,
    onClick: respond
},

This does not work though…does anyone know how I could set this up?

    There's no way to do that in the markup or TSS, but you could do it programmatically if you have predictable button ids. For example, if you had 10 buttons, each with the IDs:

    • button1
    • button2
    • …
    • button10

    you could do this:

    function respond(e) {
    // "e.source" is a reference to the button
}

var numOfButtons = 10;
for (var i = 1; i <= numOfButtons; i++) {
    $['button' + i].addEventListener('click', respond);
}
