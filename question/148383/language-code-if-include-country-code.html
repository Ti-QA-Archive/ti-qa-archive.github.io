Language code if include country code

Hi all,

To localize my App, I put a sub-folder in i18n folder which is named as "zh-TW".
It works for Traditional Chinese on Android.
However, on iOS, the folder name should be "zh_TW" to let it works. (underline between zh and TW).
I tried to prepare both "zh-TW" and "zh_TW" but Android doesn't accept "zh_TW" as locale code and lead to build error.
Ti.Locale.getCurrentLocale() returns "zh-TW" on both Android and iPhone.
This issue is reproduced on Titanium SDK 3.0 and 3.0.2.
Is it a bug?
Any suggestion?
Thx in advance.

  • This issue was solved by modifying Titanium Mobile SDK build script localecompiler.py:

            def compile_for_ios(self,file):
                locale = self.get_locale(file)
>                if locale == 'zh-TW': locale = 'zh-Hant'
>                if locale == 'zh-CN': locale = 'zh-Hans'
                build_dir = self.get_ios_dir()

    However, this is still considered as a workaround, I think Appcelerator should has more official solution.

    • BTW, this solution is for 2.x.

      Since 3.x, CLI is used and the modification is in _build.js:

              compileI18N: function (callback) {
                var data = ti.i18n.load(this.projectDir, this.logger);

                parallel(this,
                        Object.keys(data).map(function (lang) {
                                var locale = lang;
>                                if (locale === 'zh-TW') locale = 'zh-Hant';
>                                if (locale === 'zh-CN') locale = 'zh-Hans';
                                return function (done) {
                                        var contents = [
                                                        '/**',
                                                        ' * Appcelerator Titanium',
                                                        ' * this is a generated file - DO NOT EDIT',
                                                        ' */',
                                                        ''
                                                ],
>                                                dir = path.join(this.xcodeAppDir, locale + '.lproj'),
                                                tasks = [];
