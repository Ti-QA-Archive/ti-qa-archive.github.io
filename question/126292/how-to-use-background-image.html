  • A background image for a view is pretty straight forward. It would have helped if you posted your code and the path of your image, but assuming that you have an image in Resources/images/image.png you should be able to see it as a background image with the following code:

    var view = Ti.UI.createView({
    height: 100,
    width: 100
    backgroundImage: '/images/image.png'
});

    ..and adding the view to your window of course.

    Hope this helps.
