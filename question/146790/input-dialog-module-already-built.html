  • Hi

    As you have not indicated which target platform you are working with and which versions of those platforms - I am going to guess. If this is wrong, provide the details above and we can try again.

    For iOS (specifically 6.0) there is a superb module from Marcel Pociot called titanium social-modul, this provides you with the ability to use the iOS in-built twitter and facebook solutions. You can provide the initial message, link(s) and image(s) then show the dialog, the user can then edit the message as they wish and submit the social dialog.

    It works great.
