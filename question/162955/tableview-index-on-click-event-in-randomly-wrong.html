  • Hi Maxime.

    I got the same issue in my project and I've fixed. If you are creating your rows using JavaScript objects, create using Ti.UI.TableViewRow objects instead. I got this problem in Android only, iOS was working fine.

    Instead of:

    var data = [{title: &#39;My Row&#39;}];

    Do this:

    var tableRow = Ti.UI.createTableViewRow({title: &#39;My Row&#39;});

    data.push(tableRow);

    I tested this on 3.4.1 SDK and Android 4+.

    I hope it works for you too.

    Regards.
