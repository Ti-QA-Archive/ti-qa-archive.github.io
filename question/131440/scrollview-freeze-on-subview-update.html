scrollView freeze on subview update

Hello, I have a scrollView with several view added.
When a "pull to refresh" action is performed, app start to refresh the inside views with updated data from the web.

During this process of views updating, if I try to scroll the scrollview is freezing or the scroll is not fluid..

Anyone know a trick or a best workaround for a problem like this?

Thank you

  • I had a similar problem where I loaded tens of images inside a wrapper (height set to auto) and my ScrollView (with contentHeight set to auto as well) would freeze of sorts (it won't let me scroll until I go to a different view and then "come back").

    I found out that if during the process of adding the images to the wrapper view I set the ScollView's contentHeight to the height of the view wrapper.
