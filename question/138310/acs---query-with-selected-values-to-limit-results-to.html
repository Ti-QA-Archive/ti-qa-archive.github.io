ACS - Query with selected values to limit results to.

I'm implementing an application with ACS as a main data base.
Application type : mobile, Android native application.
ACS with Android SDK

I'd like to implement places search/query function with selected categories. Each place in ACS have a category field representing the place's category such as 'restaurant', 'hospital', 'cafe', etc. And a user can search places by selecting only some categories such as 'restaurant' and 'cafe'.
It's the same functionality defined in Foursquare API https://api.foursquare.com/v2/venues/search. It has an option 'categoryId' which is a comma separated list of categories to limit results to.
But when I checked ACS document, especially places, I couldn't find query options for this.
