After SDK upgrade: "Result of expression 'Titanium.API.debug' [undefined] is not a function"

I just upgraded Titanium Mobile SDK to 1.7.4, and now simple functions like trying to log data returns this error:
"Result of expression 'Titanium.API.debug' [undefined] is not a function"

I searched for similar problems, cleaned my project, manually removed my build directories, restarted Titanium, and so on. Nothing changes.

I'm not sure why this would happen, but as I am on a deadline, I will now treat future Titanium upgrades as dangerous.

Any suggestions or help would be very much appreciated.

  • I solved the problem by going into my project directory, editing .settings/com.appcelerator.titanium.mobile.prefs by hand, and setting MOBILE_PROJECT_SDK_VERSION=1.7.4. It had previously been 1.7.2.
