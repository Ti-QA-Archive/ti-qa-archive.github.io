How to vertically align labels to the text baseline?

I have two labels — each with text of different sizes —horizontally next to each other, and I need to have their baselines match. Like in a standard key: value attribute display. Anyone know how?

  • You didn't mention if you are targeting a desktop app or mobile, and if mobile which platform(s). Since I have never used appcelerator for desktop apps, I'll address mobile apps. :)

    Text fields support a verticalAlign property, as demonstrated in the kitchen sink example textfield_therest.js. To my knowledge, labels don't support that property.

    So here are two different methods you could try:

    1. Put the objects in a view (or table row) and use the same bottom value for both labels and leave off the top property value. This should align the bottom of the labels, and I believe will give you the desired look at least on iOS. I am less sure if it will also align in android.

    2. Use text fields instead of labels, then add verticalAlign:'bottom' and editable:false as properties.

    Both of these methods are untested; they are just ideas off the top of my head.
