  • On android, you can set your gps provider. You can use manualMode with Android and the location provider to 'Network' to use the cell radio instead of the GPS. On IOS it's a little different because you can't select the provider, you can only set accuracy and distance filters. I'm not sure if any combination would 'guarantee' use of the device's radio instead of the GPS, but you could play with the options to figure this out. Information on the GPS capabilities of each platform are explained here.

  • I have heard about GSM triangulation, but in order to work I think that you need some kind of database with the coordinates of the GSM transmitters that you are going to work with. If you have these then it's not that difficult.

    If I am not mistaken you can work this out with two points. The midpoint of a segment defined by two points A(Xa , Ya) and B(Xb,Yb) can be defined as the point (X,Y) where

    X = (Xa+Xb) / 2 
Y = (Ya+Yb) / 2

    but yes, you will need to have in place the coordinates (Xa, Ya) and (Xb,Yb) where (Xa,Ya) is the location of the GPS Transmitter that was found by your mobile before finding the current one which is (Xb,Yb).
