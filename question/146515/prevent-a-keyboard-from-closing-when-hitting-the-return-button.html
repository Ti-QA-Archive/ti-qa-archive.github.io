Prevent a keyboard from closing when hitting the RETURN button

When I create a TextField (default properties), the keyboard auto-hides when I hit the RETURN key. How can I prevent this?

var textField = Titanium.UI.createTextField({
    height : "auto",
    width : 100,
});
myWindow.add(textField);

I tried to add an event listener to the "focus" event, but nothing there seems to help.

Any idea?

  • Hi

    If you want to make sure the textField keyboard does not dismiss when return is pressed then use the following code;

    var textField = Ti.UI.createTextField({
    height: 60,
    left: 10,
    suppressReturn: false,
    top: 10,
    width: 250
});
win.add(myWindow);

    The important part suppressReturn: false, where suppressReturn determines how to handle the return key.
