  • Indie subscriptions (and higher) have access to the Titanium+ modules which currently are:

    Urban Airship (iOS 1.0 / Android 1.1)
QuickLook Module (iOS 1.0)
Paypal Module (iOS 1.0 / Android 1.1)
OpenGL (iOS 1.0)
Magtek Credit Card Reader Module (iOS 1.1)
Charts Module (iOS 1.0)
DoubleClick Module (iOS 1.0)
GameKit Module (iOS 1.0)
Bump Module (iOS 1.0 / Android 1.0) 
Barcode Module (iOS 1.1 / Android 1.0)
Brightcove Module (iOS 1.0)
Omniture Module (iOS 1.0 / Android 1.0)
TV Out Module (iOS 1.0)
Apple StoreKit Module (iOS 1.0)
SMS Module (iOS 1.0)
GreyStripe Module (iOS 1.0)
Compression Module (iOS 1.0)
AirPrint Module (iOS 1.0)
