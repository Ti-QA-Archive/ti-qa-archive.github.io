How to change left position without changing the width

Hi,

I'm trying to build an iOS and Android side drawer menu for our application as a way to learn more about titanium. Currently I have the following in Alloy:

index.xml

<Alloy>
    <Window class="container" navBarHidden="true">
        <View id="movableContainer">
            <View id="headerBar">
                <Label class="element left" onClick="openMenu">Left</Label>
                <Label class="element center">Title</Label>
                <Label class="element right">Right</Label>
            </View>
            <View id="content"></View>
        </View>
    </Window>
</Alloy>

index.tss

".container": {
    backgroundColor:"white",
    layout: 'vertical'
}

"#movableContainer": {
    layout: 'vertical'
}

"#headerBar": {
    top: 0,
    height: 60,
    width: Ti.UI.FILL,
    backgroundColor: "#005e04"
}

".element": {
    top: 20,
    height: 44,
    verticalAlign: Ti.UI.TEXT_VERTICAL_ALIGNMENT_CENTER,
    color: "#ffffff"
}

".left": {
    left: 0
}

".right": {
    right: 0
}

index.js

var openMenu = function() {
    var hideContent = Ti.UI.createAnimation({
        duration: 600,
        left: "90%"
    });
    $.movableContainer.animate(hideContent);
};

$.index.open();

As you can see, initially I created a label that will be clicked on and will make the content "hide" and leave a space to the left so another view will show up (its not in there yet).
My problem is that with the current code after the click the animation happens correctly but the headerBar element shrinks instead of going behind the viewable area, as if the width is decreasing.

Any help with that? I'm sure that the problem is with Ti.UI.FILL and I should have something else there, but I can't find what. I'm been looking at some third party components to do that and the seen to use similar strategies, not sure why mine is not working.

Thanks for the help!

    Set the width to a numeric value. You probably will need to listen for the postlayout event and then use $.movableContainer.size.width

    — answered February 23rd 2015 by Shannon Hicks
    permalink
    3 Comments

    • Thanks!
      This is what I did:

      $.movableContainer.addEventListener('postlayout', function(e) {
    $.movableContainer.setWidth($.movableContainer.size.width); 
});

      It works!
