    Hi Varun Krishna Parthasarathy,

    Seems there is issue with Facebook module. First of all check if your project requires any module or not. If yes, then add the module to your project. If not, the remove unnecessary modules if added to that project, then run your application and see if it works for you.

    If this does not work for you then create a new default project and run the project. Then overwrite the existing resources with new resources, clean the project and rerun your application.

    Hope it'll work for you.
