Build APK without emulator

Hi,

Is there any way to generate the APK without launching the emulator?

Thanks :)

    If you use the CLI to build your app, you can use the option --build-only.

    Cf. the documentation : Only perform the build; when specified, does not install or run the app.

  • Probably the best way to achieve this is to use the Titanium Command Line Intreface (CLI)

    Simply go into your project's directory using your Terminal (or Command Prompt) and type the following:

    titanium build -p android --build-only

    Hope this helps

  • You can easily do this with Titanium CLI using this call:

    titanium build -b -p android

    Run this commend from the project folder.
