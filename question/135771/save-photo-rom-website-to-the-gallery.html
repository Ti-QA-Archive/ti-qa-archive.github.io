  • Accepted Answer

    images[i] is just String, while Titanium.Media.saveToPhotoGallery() requires TiBlob or TiFile. That's why you got that error.

    To fix this, you need to return the full image urls, then use HTTPClient to fetch images, store them to temporary location ( .i.e tempDirectory ). Then pass file or blob to saveToPhotoGallery().

    Hope this can give you some hints.
