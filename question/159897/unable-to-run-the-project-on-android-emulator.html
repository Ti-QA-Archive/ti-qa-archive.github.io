Unable to run the project on Android Emulator

I am using windows os and titanium sdk 3.1.3. Trying run the project on Android Emulator. But for me no run as option is coming. Only Run configuration option coming. When I am trying to configure my project then the below error is coming - Project 'Proj-Name' is not valid for this configuration. Please select another project.

  • Hi, The problem can be project deployment targets. Go to your tiapp.xml and see if the Android Deployment Target is checked. Or manually editing the XML : 

    <deployment-targets>
        <target device="android">true</target>
</deployment-targets>

    Good Luck!
