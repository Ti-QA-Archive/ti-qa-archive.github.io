    First, be aware of this issue with the latest android tools. Athough its now marked fixed in the latest master of 3.3.0

    I'm assuming you also upgraded the npm packages.. Have you reverted them ?

    npm remove -g titanium
npm install -g titanium
npm remove -g alloy
npm install -g alloy

    I also assume you upgrade Studio too ? Have you downgraded that back to the latest stable version too ?
