How to make ONE Android Menu for 5 Heavyweight Windows?

I'm honestly desperate.. trying to get behind this "android activity" but failing hard.

What I am trying is to create 1 Android Menu which works across 5 different heavyweight windows.

I got a simple TableView with 5 rows in it and on click I create an heavyweight window. Now here is where the android menu should get available, which has got the 5 rows in it (just in form of an menu), so the user doesnt need to click "back", but can just click menu and select one of the rows there.

Problem is, do I ALWAYS need to call and create the activity.onCreateOptionsMenu again on each of the 5 windows?

I got it working great when working with lightweight windows, but yeah.. I need them as heavyweight :)

Im following the CommonJS standards btw.

  • Yes, each heavyweight window is an activity by itself, that means each of them has to have the menus initiated.

    You can do this pretty easy creating a module that returns u the activity (window) with the menu setup already so you do this in one place only.
