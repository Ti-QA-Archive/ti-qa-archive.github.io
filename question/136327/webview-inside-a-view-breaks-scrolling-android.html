Webview inside a view breaks scrolling (Android)

Another Android only issue. It appears that if I have a webview inside another view scrolling breaks on Android only:

var win1 = Titanium.UI.createWindow({  
    backgroundColor:'#fff'
});


      innerView = Ti.UI.createView({
        layout:'vertical',
        top:0,
        height:200,
        backgroundColor:'white',
        width:'100%'
        });

    mHtml="<html><body><a href=\"#\">x</a>test<br/><br/>test2<br/><br/>test<br/><br/>test<br/><br/>test5<br/><br/>test<br/><br/>test<br/><br/>test<br/><br/>test<br/><br/>testxxxx<br/><br/>test<br/><br/>test2<br/><br/>test<br/><br/>test<br/><br/>test5<br/><br/>test<br/><br/>test<br/><br/>test<br/><br/>test<br/><br/>testxxxx<br/><br/></body></html>";
        var wView = Ti.UI.createWebView({
                width:280,
                height:200,
                html: mHtml
        });
    innerView.add(wView);
    win1.add(innerView);

If I add the webview to the window directly it works, but as soon as I add to a view inside a view it breaks the scrolling.
