  • I was finally able to work this out by downloading the backdated 1.7.0-RC2 version of the Titanium API. The problem was that the newer versions of the Titanium API don't have the blackberry dir under the titanium/mobilesdk/win32/$TITANIUM_VERSION If that dir (titanium/mobilesdk/win32/$TITANIUM_VERSION/blackberry) isn't there Titanium Studio won't find the blackberry SDK.

    This Answer is where I got the solution.
