  • You are just after the HTML of a page? If the website you are trying to get content from provides an RSS feed then that is the way to get the HTML. However, if that is not possible you will need to try and get the HTML and try and fit into your layout.

    How about something like this.

    var webview = Titanium.UI.createWebView({url:'http://www.appcelerator.com'});

    And then use either the HTML property or the GetHTML method to get the HTML value of the webview.

    See the API documentation for WebView for more details.
    WebView API
