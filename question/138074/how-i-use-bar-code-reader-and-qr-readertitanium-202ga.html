    • Then instead of passing QR_SCAN_MODE you would pass another option, or a comma separated list of options, or do not limit the scan mode at all by simply leaving out the line:

      intent.putExtra("SCAN_MODE", "QR_SCAN_MODE");

      Look at the comments in the Java code for that Intent in the ZXing library, which can be found here. For example, EAN_13,EAN_8,QR_CODE or PRODUCT_CODE,QR_CODE
