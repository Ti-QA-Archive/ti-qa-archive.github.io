Hello

Now when i go back it goes back to the page with the "overview" of the topics (which is only one, so it automatically redirected).

Now I want to check check if i click back on the topic detail, come back into the overview detail, that it redirects to the person overview.

I might have explained it difficult, but I don't quite know how to phrase it!

Hope my question is clear enough. So it's basically how can I get the sending page.
