How to convert X, Y coordinates to latitude and longitude in mapview

I can get X,Y coordinates in mapview click event and to put an annotaion on map view i want to convert these X,Y coordinates to longitude and latitude. If anyone knows about such a method that i can use , plz post the code here.

    Hi,
    in my app "Hochbahn-TV" I have used a linear calculation.
    You can retrieve region (center and delta). The x/y coords are relative to top left corner.

    In this code I simulate movabel annotation – after scrolling/zooming of map the car over over map is replaced:

    map.addEventListener('regionChanged', function(e) {
    hhregion.latmin = e.latitude - e.latitudeDelta / 2;
    hhregion.lngmin = e.longitude - e.longitudeDelta / 2;
    hhregion.latdelta = e.latitudeDelta;
    hhregion.lngdelta = e.longitudeDelta;
    carImageView.animate(Ti.UI.createAnimation({
        delay : 200,
        duration : 300,
        opacity : 1
    }));

});
