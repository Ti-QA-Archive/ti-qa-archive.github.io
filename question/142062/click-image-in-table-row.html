Click image in table row

In my app, I'm looping through my results, and one thing I'm adding in each row is an image, that a user will click to delete.

I have an event listener for my table (as opposed to each row), and I have one added to my delete image individually.

When I click the 'Delete', it does what its supposed to, but it seems to click through to the row, and fire the tablelistener as well.

Anyone know how I can get around this? I could make a lot of row listeners instead of the one tableview, but performance was pretty bad doing it that way.

(Android, btw)

  • Try this code
    var imageview = new Array();
    var btndelete = new Array();
    var data=[];
    var tableview = Ti.UI.createTableView({
    top : 50,
    left : 0,
    backgroundColor : 'transparent',
    separatorStyle : Ti.UI.iPhone.TableViewSeparatorStyle.NONE

    });
    win.add(tableview)

    var counter =5 ; // assign your dynamic values to counter 

    for (var i = 0; i < counter; i++) {
            var row = Titanium.UI.createTableViewRow({

                rowid : i,
                height : 71,
                width : 320,
                rowBackgroundColor : 'transparent',

            });
             imageview[i] = Titanium.UI.createImageView({
                image : // your image path,
                width : 320,
                height : 42,
                left : 0,
                top : 0
            });
            btndelete[i] = Titanium.UI.createButton({
                left : 276,
                top : 15,
                backgroundImage : // set delete button image,

                width : 34,
                height : 31,
                id : i

            });

            row.add(btndelete[i]);
            data.push(row);

            btndelete[i].addEventListener('click', function(e) {
                // your delete code
                });

} // loop over

tableview.data = data;

    As per above code each row of Tableview containing one image and one delete button.
