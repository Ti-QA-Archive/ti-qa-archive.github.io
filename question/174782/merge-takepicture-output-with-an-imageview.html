Now here is where things get complicated:
I want to create another image file that contains the camera output image (preferrably in the original camera resolution) and the contents of a smaller image overlayed on top of it.

I have tried to create a View, insert both images into it via ImageViews, but the output of toImage() comes black.

What other ways are there to perform this operation?
