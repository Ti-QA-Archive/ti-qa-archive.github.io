CommonJS Modules have access to functions defined in app.js?

i am working on an app under Titanium Studio 3.2.x, surprisingly i found that my JS Module could have access to functions which i defined in the app.js ?

here is piece code of my app.js

function swipeWindows(direction) {

    var currentTabIndex = tabGroup.activeTab.myIndex;

    if (direction == 'left') {
        if (currentTabIndex == tabGroup.tabs.length - 1) {
            tabGroup.setActiveTab(0);
        } else {
            tabGroup.setActiveTab(currentTabIndex + 1);
        }

    } else if (direction == 'right') {
        if (currentTabIndex == 0) {
            tabGroup.setActiveTab(tabGroup.tabs.length - 1);
        } else {
            tabGroup.setActiveTab(currentTabIndex - 1);
        }

    }
}

here is some code of my JS module(located at /UI/myModule.js)

    this.mainWindow.addEventListener('open', function(){
        this.addEventListener('swipe', function(e) {
            swipeWindows(e.direction);// it works 
        });
    });

any idea ? or it is supposed to be working this way ? thanks .

    Hi

    Not sure on the actual question.

    Yes it can but it is unlikely to continue doing allowing that in future versions. Different target platforms behave different to this rule.

    Create any function you want to access in multiple places using CommonJS, NEVER rely on this.
