Make a phone call, but ask the user first, with "OK" and "cancel" button

Here's a button "Call Us".
when tapping the button,

Titanium.Platform.openURL('tel:'+1234567890);
enable making a phone call.

I want to do a little better. When tapping the button, I would like a message (or a small view) pop out, the first line is the phone #, and the second line has two buttons, "cancel" and "call". If the user choose "call", then make a phone call. choose "cancel", nothing happens.

Could somebody help? Thanks.

    Hi,

    try this:

    var alertDialog = Titanium.UI.createAlertDialog({
    title: '0049 40 27806982',
    message: 'Are you sure ',
    buttonNames: ['OK','Doh!']
});
alertDialog.addEventListener('click',function(e){
 if (e.index==0) {
   Titanium.Platform.openURL('tel:'+1234567890);    
 }

});
alertDialog.show();
