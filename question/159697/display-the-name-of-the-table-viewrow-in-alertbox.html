  • Your code is poorly readable - Embrace code within ~~~ for nice formatting.

    set Click event listener

    <Alloy> 
<TableView id="tableView" class="container" onClick="onTblClick"> 
<TableViewRow className="menuRow" title="My Profile" /> 
       <TableViewSection headerTitle="FAVORITES"> 
             <TableViewRow className="menuRow" layout="horizontal" page="checkins"> 
                   <ImageView image="/images/check-icon.png"/> 
             <Label>Home</Label> 
             </TableViewRow> .....
</Alloy>

    and then

    function onTblClick(e) {
    Ti.API.info(JSON.stringify(e)); //You will get the answer
}

  • Hi,

    if you want to show text then you need to set that text as a custom property for row.
    like below..

    <Alloy> 
<TableView id="tableView" class="container" onClick="onTblClick">
    <TableViewRow className="menuRow" title="My Profile" myText="My Profile" /> 
    <TableViewSection headerTitle="FAVORITES"> 
        <TableViewRow className="menuRow" layout="horizontal" page="checkins" myText="Home"> 
            <ImageView image="/images/check-icon.png"/> 
            <Label>Home</Label> 
        </TableViewRow> 
.....</Alloy>

    function onTblClick(e)
{
    Ti.API.info("My Text >>>" + e.row.myText);
}
