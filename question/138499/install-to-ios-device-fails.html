    • After much effort I finally narrowed down the source of the problem. It turns out that one or both bits of JavaScript I had used in my html file, in the webview, were not to Titanium's liking.

      To put this into context - I found myself obliged to use a WebView to show a map since I need the full power of Google's infobox annotations which is quite simply not available in Titanium maps.

      The First:, the jQuery ver 1.71, minimized. Given that I was only using jQuery evaluate some JSON and to grab a few DOM elements I simply replaced the JSON parsing with an eval statement (my data are safe) and did the DOM element grab the old fashioned way (getDocumentElement)

      The Second:I was using jscoord.js from, http://www.jstott.me.uk/jscoord/, to convert latitude/longitude values to UTM coordinates. I simply worked around the need to do this inside the webview.

      Result - the IPA now builds! It is interesting to note that simply having those two js files in a subfolder in the project, i.e. merely commenting out the webview bit did not help.
