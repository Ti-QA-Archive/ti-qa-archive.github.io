  • Hi

    Other than the very obvious question why are you hiding the camera action from the user?

    Have you tried using the overlay view feature and covering the entire camera view.

    var viewOverlay = Ti.UI.createView({
    backgroundColor: 'orange',
    height: Ti.UI.FILL,
    width: Ti.UI.FILL
});
Ti.Media.showCamera({
    success:function(event) {
        // called when media returned from the camera
    },
    cancel:function() {
        // called when user cancels taking a picture
    },
    error:function(error) {
        // called when there's an error
    },
    allowEditing: false,
    mediaTypes: [
        Ti.Media.MEDIA_TYPE_PHOTO
    ],
    overlay: viewOverlay,
    saveToPhotoGallery: false
});

    You will need to manually take the picture using takePicture.
