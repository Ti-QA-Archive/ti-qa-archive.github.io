Window barImage orientation change

Hi, I am experiencing a problem in changing the window bar image on orientation change.

Basically, I want the bar image to change to a different image when the user has the iPad in the landscape view however the image never appears.

Here is my code:

var mainPage = Titanium.UI.createWindow({  
    barImage:'image1.png',
    backgroundColor:'white',
    barColor:'#000',
    url:'views/ipad/mainPage.js'
});

// Adjust image position 
Ti.Gesture.addEventListener('orientationchange', function(e){
    if(Ti.Gesture.orientation == Ti.UI.LANDSCAPE_LEFT || Ti.Gesture.orientation == Ti.UI.LANDSCAPE_RIGHT){
        mainPage.barImage = 'image2.png';
            }else{
        mainPage.barImage = 'image2.png';
    }
});

When running this code the window displays the bar image however when you flip the ipad to landscape the image doesn't appear.
