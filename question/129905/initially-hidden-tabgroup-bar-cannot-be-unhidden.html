Initially Hidden TabGroup Bar Cannot Be Unhidden

I believe I've found a bug w/ the tabBarHidden property of Ti.UI.Window but according to the Bug Reporting Lifecycle I must first discuss it in this section before submitting it.

If you have a TabGroup where the first window's tabBarHidden property is set to true and then push a new window onto the tab with tabBarHidden set to false, the tab bar will not become visible as expected. I have included a sample project to further illustrate this bug.

It is also worth noting that the inverse of this DOES work (Initial window's tabBarHidden is set to false, and subsequent window's tabBarHidden is set to true).

Can anyone else confirm that this is a bug?
