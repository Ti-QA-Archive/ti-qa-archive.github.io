Set which Simulator to run

Hi.

Maybe i have totally missed this but i really need help to set which of the simulators to open when i run my iOS apps.

As if now it is ALWAYS the iPhone (non retina) that is opend when i run an app.

And this is what i have to do to get it to run on iPhone 4"

  1. Run the app (iPhone non-retina opens the app)

  2. Press Home and change the device to iPhone 4"

  3. The simulator shuts down.

  4. Open the simulator. (It opens with iPhone non-retina)

  5. Change Hardware - Device to iPhone 4"

  6. Run the app.

  • In your tiapp.xml editor, you will see a list of supported platforms for your application at the bottom left. Under that, simply click the Configure link and you will be shown a dialog window.

    From there, simply select which simulator version you want to run for your target platform.

    Hope this helps.
