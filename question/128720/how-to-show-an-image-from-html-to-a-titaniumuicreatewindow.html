  • Accepted Answer

    So if I'm understanding you right, you want to show a fullscreen image, from a webview, in your Titanium app. The best way to do that would be to fire an event from your HTML when the image is clicked. For example, do something like this in your HTML:

    <img src="http://yourdomain.com/yourSource.jpg" onclick="Ti.App.fireEvent('app:fullscreenImage', {url:'http://yourdomain.com/yourSource.jpg'} )"/>

    then in your Titanium app, you can listen for that event like this:

    Ti.App.addEventListener('app:fullscreenImage', function(e) {
    // e.url now equals the remote URL of your image, which you can 
    // use as the source for an ImageView
});
