Adding a footer to each UI.TableViewRow

Hi all.
I've been trying to build a Table that has multiple lines and each line has it's own footer something like this:

var data = [];
for (var i=0; i<10' i++){
    var title = 'this is row number: ' +i;
    var row = Ti.UI.createTableViewRow({
    title:title,
    footer:'awesome row: '+i,
    width:'auto',
    height:'auto'
    };
    data.push(row);
};
var tv = Ti.UI.createTableView({
    data:data
});

I didn't see any reference to a footer like this, is there no other way but creating a custom label for this?

    The Titanium.UI.TableView will allow you to specify a header/footer. However, the Titanium.UI.TableViewRow does not allow you to specify a header/footer for each row.

    To have a header/footer for each row you would just need to to create the container view your self. 

    //example code has not been tested. use as own risk.
var data = [];
for(var i=0;i<10;i++){
  var row = Ti.UI.createTableViewRow({height:'auto'});
  var container = Ti.UI.createView({layout:'vertical',height:'auto'});
  var header = Ti.UI.createLabel({text:'Header ' + i});
  var content = Ti.UI.createlabel({text:'Content ' + i});
  var footer = Ti.UI.createLabel({text:'Footer ' + i});
  container.add(header);
  container.add(content);
  container.add(footer);
  row.add(container);
  data.push(row);
}
