Can emailDialogs attach PDFs with multiple pages? (iOS)

Hi, I'm having a problem attaching PDFs with multiple pages to an email.

My code looks something like… 

sampleEstimate = Titanium.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory,'document.pdf');

var emailDialog = Ti.UI.createEmailDialog();
emailDialog.addAttachment(sampleEstimate);

emailDialog.open();

Where document.pdf is any PDF with more than one page.

I've tried this, generating my own PDFs (using FPDF), or using PDFs built by others, from PDF version 1.4-1.7.

It works PERFECTLY, if the document is one page. Filesize ranging from 60k-350k.

If the document is MORE than one page, it fails to attach and I just get the icon, and the name of the file it's trying to attach, both in simulator, and in production on the iPad.

Is this a known limitation or am I doing something wrong?
