Can not select iOS SDK 8.0 from Titanium Studio

I would like to use iOS SDK 8.0 from Titanium Studio, but only 7.1 is shown in the drop down list. Neither, iOS 8 simulators are not shown(only iOS7 simulators are shown).

Versions are
Titanium Studio : 3.3.0
CLI version: 3.4.0-rc4
Titanium SDK version: 3.3.0.GA
Xcode: 6.0.1

When I run "ti build" command from console, iOS 8 simulator is launched successfully. But I cannot choose iOS SDK 8.0 on Titanium Studio. What should I do?
