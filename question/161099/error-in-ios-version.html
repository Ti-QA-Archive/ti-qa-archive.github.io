    • Fixed! An Appcelerator rep provided me with the following steps, which reverted me to previous versions of Titanium and Alloy, allowing me to emulate/simulate on both Android and iOS, on both Titanium Studio and Appcelerator Studio:

      In terminal (you may have to enter your password on the first command due to use of sudo):-

      sudo npm remove -g titanium

      sudo npm remove -g alloy

      sudo npm install -g titanium@3.1

      sudo npm install -g alloy@1.2

      Once done you can check the versions you¹ve installed with:-

      titanium -v

      alloy -v

      Should confirm you¹re running Titanium CLI 3.1.4 and Alloy 1.2.x
