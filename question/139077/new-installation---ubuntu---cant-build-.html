  • This is the answer:

    I needed to add the location of my javabinary to the $PATH variable.

    On my system it's: /usr/lib/jdk1.6.0_25/bin

    Add the that path to the $PATH variable defined in the file /etc/environment

    I didn't know how to let the system know about the new $PATH - so I rebooted and it worked

    Got the final clue from this post: http://developer.appcelerator.com/question/123465/cannot-compile-android-application-fedora-15
