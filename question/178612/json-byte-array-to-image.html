    Hello Geervani,

    This object clearly explains that USER_PICTURE contains the base64code {USER_PICTURE: '/9j/4QCcRXhpZgAASUkqAAgAAAAHA..................................................'}
to convert base64code to an image use below util library.
http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.Utils-method-base64decode
Titanium.Utils.base64decode() method it convert the base64code to image.
