Android Heavyset Window top parameter not considered

I am trying to open a Heavyset window in android that is smaller then the screen and align it to the top. 

var win = Titanium.UI.createWindow({ 
    backgroundColor: "white",
    orientationModes: [ Titanium.UI.PORTRAIT ],
    height: Ti.Platform.displayCaps.platformHeight-200,
    top: 0,
    fullscreen:true,
    navBarHidden: true,
    exitOnClose: true
});

The top parameter is not considered in Android while it is in iOS. So the window seems to always appear in the middle of the screen. Anyone knows how to solve this?

Thanks

  • You should use a View instead of a Window. On iOS, Windows and Views are essentially identical components. That's not so on Android, as Windows correspond to Activities.
