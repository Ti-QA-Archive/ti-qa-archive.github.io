  • This is what I ended up doing. Download latest SDK from Google, extract it & then take a look in the Platforms folder which is usually empty (that's why Titanium Studio isn't showing anything).

    Now run 'android' from the tools folder & install a few of the platforms from the Available Packages list.. I did them one at a time, since picking a few didn't work the first time for me. I picked 2.1, 2.2, 3.2.

    After they'd installed I went back to TiStudio and pointed it at the SDK & it picked up the 3 packages I'd installed.
