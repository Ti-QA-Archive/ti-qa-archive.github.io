  • Dear Ronny,

    The only question in your post is 'can somebody help me in this??'. If you're going to expect any type of help from the hardworking people here, you'll need to do something like this:

    1. Clearly phrase your question, letting everyone know exactly what you're interested in.

    2. Choose a more descriptive title that summarizes your question.

    3. Speak in semi-understandable English. If you do not speak it natively, please try a translator, or mention it so that people understand.

    4. When asking a question, don't just say 'I need somebody to code this part of my app'. You need to say 'Ok, I've gotten this far, but I'm stuck on <insert problem>. So far, I've tried to fix it by doing things like <insert attempted solution>, but that didn't work. So, is there any input that somebody has to offer to hopefully solve my problem'

    Hope this helps,

    Jacob
