Alloy 1.3.0 requires Titanium SDK 3.2.0 or higher.

After updating my titanium SDK to 3.2.0, i got the same problem like in this question so i tried to update XCode to v5 but i coudn't do that in my IOS version because the v5 of XCODE requires a newer version than mine.
Then as it wasn't possible for me to update it, i decided to return to the previous of titanium SDK but when lauching my application again i got these errors :

[ERROR] :  Alloy 1.3.0 requires Titanium SDK 3.2.0 or higher.
[ERROR] :  "3.1.3.GA" was found in the "sdk-version" field of your tiapp.xml.
[ERROR] :  If you are building with the legacy titanium.py script and are specifying 
[ERROR] :  an SDK version as a CLI argument that is different than the one in your 
[ERROR] :  tiapp.xml, please change the version in your tiapp.xml file.
[ERROR] :  Alloy compiler failed

which probably means that the Alloy version became the version corresponding to the SDK 3.2.0 and didn't downgrade to the 3.1.3.GA and i don't know how to do to fix this.

Cheers

    Hi,

    you need to execute below command to install any version of Alloy..

    sudo npm uninstall -g alloy

//then

sudo npm install -g alloy@1.0.0

//where 1.0.0 is alloy version
