    • I assume Java JDK 7 is the problem.

      Please uninstall JDK 7 and remove JSDK 7 references from path and so on.

      Titanium needs Oracle Java JDK 6 as you can see in Titanium Compatibility Matrix

      From your build.log:

      [DEBUG] "C:\Program Files (x86)\Java\jdk1.7.0_09\bin\keytool.exe" -v -list -keystore
