Passing data to a Global variable?

Hello.
'm creating a project with titanium Alloy and i need to sharing data between windows by Globale variable.
Can anyone help me please with this and explain to me how i use globale variable for passing data.

  • Hi, it's actually very simple. Just assign it like this in the first window

    Alloy.Globals.my_variable = 'I am a global variable';

    and you can access in the second window, for example

    alert(Alloy.Globals.my_variable);

    Having said that, there is a better way to pass data through controllers.

    var obj = {
    customVar : 'hello'
};
var SecondWindow = Alloy.createController("secondPage",obj).getView();
SecondWindow.open();

    and in the second window you retrieve it like this

    alert(arguments[0].customVar);

    Hope it helps
