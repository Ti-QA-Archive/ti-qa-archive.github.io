  • Hi Mick,

    You can assign your custom property to any controller for example

    if you create several row for table view and you want to some different value for that. like your requirement userNo.

    then

    var tblRow = Ti.UI.createTableViewRow({
    height : Ti.UI.SIZE,
    width : "100%",
    isMoreRow : true,
    userNo : userNo
});

    here isMoreRow and userNo is custom property.
    you can access it directly like all other properties.

    e.g

    $.tableView.addEventListener("click", function(e) {
    if (e.source.isMoreRow) {
        return;
    }
});

    hope this is helps you

    Thanks
