    Hi

    Focus you event on the table NOT the row, this is much more efficient, easier to code and recommended. If you need to determine what you clicked just add a custom property to the imageView and in the click event test for it;

    $.cart.getView('productAddToCart').addEventListener('click',function(e) {
    if (e.source.customproperty === 'image') {
        alert("hi");
    }
});

    You would give each child of the view the same custom property but the value of each would be different, so you can check against it in the click event - like my example.
