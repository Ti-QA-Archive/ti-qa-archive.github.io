    Hi Hamza Najiallah,

    For that you need to set application theme @style&#x2F;Theme.Titanium for android. and you can set it by using androidmanifest.xml and also using tiapp.xml. so better it to add in tiapp.xml. e.g

    add this code into tiapp.xml

     <android xmlns:android="http://schemas.android.com/apk/res/android">
        <manifest>
             <application android:theme="@style/Theme.Titanium">
            </application>
        </manifest>
    </android>

    Make sure this tag is exists or not.. if not then add it.. and if exists then add 

    <application android:theme="@style/Theme.Titanium">
            </application>
