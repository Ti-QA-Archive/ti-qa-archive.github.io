Tableview Row Activity Indicator

Ok, so I have a tableview which shows items from a SQL database, when I click on these I open a modal window that populates over a dozen views with data from the tableview row. Pastie

This works fine, but is slow in opening the modal window. I need to provide some feedback to the user that the window is about to open. I would like to add an activity indicator to the row, I dont mind if I then remove it via a timeout.
But how do I add the activity indicator?

  • Accepted Answer

    var actInd = Ti.UI.createActivityIndicator();

row.add(actInd);
actInd.show();

    And then when you remove it via a timeout, just:

    setTimeout(function() {
   actInd.show();
},TIME);
