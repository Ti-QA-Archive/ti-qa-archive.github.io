  • There's no way to control leading with a Label component. You could try using a TextArea, setting it to editable = false. Not that TextArea lets you set leading properties either, but it might render multiline text better for you. Alternatively, you could try using a custom font. The default leading of Arial might not be sufficient for the fontSize you're using. Perhaps you can find one with a larger vertical size that will lay out better in your labels.
