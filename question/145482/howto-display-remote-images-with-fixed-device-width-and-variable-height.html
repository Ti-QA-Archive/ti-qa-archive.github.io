Howto display remote images with fixed (device) width and variable height?

I am displaying a tableview with remote images (jpg from rss feed), but I can't seem to figure out how to display them nicely with a fixed with (screenWidth) and variable height.

Tried almost everything (screenwidth, 100%, auto height, Ti.UI.FILL/SIZE), but images keep having spaces above and below the image, or they scale down creating a space of both sides left/right.

Also tried the "load" event, grabbing height/width of the remote image, but it doesn't seem to detect the real height of the remote image. 

var imageContainer = Ti.UI.createView({
        backgroundColor: 'red',
        borderRadius: 0    
})
var image = Ti.UI.createImageView({
        image:image_url,
        width: Ti.UI.FILL
        // width: 100% - no effect
        // height: auto - no effect
        // width: devicewidth - effect
})

image.addEventListener('click',function(e) {
        Ti.API.info('loaded ' + image.toImage().width + "x" + image.toImage().height);
})

Help? ;-)
