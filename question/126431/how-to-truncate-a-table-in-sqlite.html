    sqlite doesn't have a truncate command, you must use DELETE FROM table. However, I believe that it is much more efficient to drop the table and recreate it.

    • This question lands more in the realm of sqlite than titanium, however, what you want to do is:

      delete from table_name;
delete from sqlite_sequence where name='table_name';

      Check out SQLite Autoincrement Docs for a bit more info on this.
