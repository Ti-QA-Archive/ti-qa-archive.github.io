Javascript String.replace() not working

Hi,

I can't figure out why this isn't working for me in Titanium!

I have a string with some characters that I want to remove however when I use the javascript 'replace()' function nothing happens!

Here is my code

var myString = "<testing/??>";

var summary = Titanium.UI.createLabel({
    text: myString.replace("</?[^>]+(>|$)","-"),
    width: 100,
    font: {fontSize:14, fontFamily:'Gill Sans'},
    color: 'black',
    height: 'auto'
});

When I do 'summary.text' I get '<testing/??>' instead of 'testing'

Any ideas?

Cheers!

  • replace accepts either a substring (which yours is not) or REGEX (which yours is incomplete)

    myString.replace(/(<([^>]+)>)/ig,"-");
