Running applescript in OSX app

Hi All,

This has been asked before but got no answer. Does anyone know if its possible to run an applescript from a desktop app (mac only). So when a user presses a button it runs the script , then possibly gives "job done" message.

    I haven't tried it, but you should be able to execute an arbitrary external program and read the results back in. JavaScript may not have the facilities for this, but if you program in Python, Ruby or PHP you can probably do it.

    In Python this would be something a simple as:

    import subprocess

rc = subprocess.call(['osascript', '-e', 'filename.scpt'])

    Or if you need more than the return code, use Popen to read back the Applescript's output:

    child = subprocess.Popen(['osascript','-e','filename.scpt'], stdout=subprocess.PIPE)
(stdoutdata, stderrdata) = child.communicate()
child.close()

    Also, have a look at appscript. It's a library that exposes the osascript engine to Python, Ruby, and Objective-C, so you can code things you'd normally use Applescript for natively in those languages.
