I have installed Titanium yesterday but it seems that I can't install CLI and Alloy.
That gives me:
Error encountered during update

NPM command not found. Please check if NodeJS is properly installed.

But nodeJs is installed… I've spent hours googling but I still dont have any clue of what's happend.
Thanks for your help.

Host Operating System: Windows vista 64

  • You need to add the directory containing npm to your PATH. On my machine, npm was installed to:

    C:\Users\Chris\AppData\Roaming\npm\

    The path should be similar on your computer. To add it to your PATH, check out this tutorial.

    On a side note, Titanium 3.0 does not support Windows Vista.
