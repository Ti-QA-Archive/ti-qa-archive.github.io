Desktop cloud woes

  • Update: As Andrew Crook suggested this appears to be a caching problem in the browser. I had bookmarked the link to my apps download page on the Appcelerator server, so this is the likely cause of old versions being downloaded. I'd assumed that the link on the page would always point to the new file on the server which I took to always overwrite the old - so something else to watch out for!

    I also found - although whether this was a coincidence or not - that the last couple of installs were not interrupted by the Windoze Installer throwing an error at me.
