    • Gotcha. When you are exporting functionality, you prefix with exports. When you are actually using the functionality within your controller instance, prefix with the $. In other words, exports for declaration, $ for execution.

      exports.baseController = 'superController';

exports.bar = function(){
    $.foo();
}
