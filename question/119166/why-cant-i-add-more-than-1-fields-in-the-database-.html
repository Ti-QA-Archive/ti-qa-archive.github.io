Why can't I add more than 1 fields in the database ??

db.execute('CREATE TABLE IF NOT EXISTS UTENTE (ID INTEGER, USER TEXT, PASSWORD TEXT, IDDISPOSITIVO TEXT)');

I use the code above to create a table in my database but it doesn't create all the fields i specify above… it creates a table with only the field ID… this problem is in iOS. in Android the thing works fine…

any help ?

    I guess that the db exists on the device with only the id field and because the IF NOT EXISTS statement the app does not create a new table. Delete the app from device or simulator and try it again. Also it is better to put all field and table names in backticks:

    db.execute('CREATE TABLE IF NOT EXISTS `UTENTE`
(`ID` INTEGER, `USER` TEXT, `PASSWORD` TEXT, `IDDISPOSITIVO` TEXT)');
