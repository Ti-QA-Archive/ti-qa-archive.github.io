 tot.setValue(a + b);

    • Follow up tips

      • if your number could have decimal places use parseFloat
      • ALWAYS use the optional second parameter with parseFloat or parseInt. As these parse function can convert numbers other than decimal it will try to guess what sort of number you provided, so to make sure it always selects base 10 (decimal) this is the second parameter.
      tot.setValue(parseInt(a, 10)+parseInt(b,10));
