How can I refresh/initialize my window when I open it?

    Hi there. Welcome!

    This is normal. When a window is loaded (via tab), it is loaded.

    If you need the window to 'recreate' itself (or maybe some content within it), then you should use the 'focus' event on the window.

    eg:

    (in a separate js script which creates your window, called when the tab is hit)..

    var win = Ti.UI.currentWindow;

// display and do stuff... 
// etc..etc..

win.addEventListener('focus',function(){
    // do something
    Ti.API.info('hey! I was focused!');
});
