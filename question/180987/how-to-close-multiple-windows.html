How to close multiple windows

Hi All,

I am developing an android app, in that i have an "add" button when i click on "add" button then add form is displaying in another window.

After filling form data, when i click on "save" button then it's opening a new window with new data which i enter in add form.
when i go back that form will displayed and old data will displayed in two seperate windows , i don't want to display these two windows at the time of back coming
of back coming how to do this . please help me

  • When you save the form and reached new window, close the window which is present behind it. For e.g.

    
var win = Ti.UI.createWindow();

var save = Ti.UI.createButton({
    title: 'Save'
});

save.addEventListener('click', function(e) {
    var newWin = Ti.UI.createWindow();
    newWin.open();
    newWin.addEventListener('postlayout', function() {
        win.close();
    });

});

win.add(save);
win.open();

    Hope the above reference code will help you to understand what I ma taking about.
