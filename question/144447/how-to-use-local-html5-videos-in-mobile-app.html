how to use local html5 videos in mobile app?

hi,

i have a clients who needs to play local html5 videos in his mobile app.

I tested on an Android phone.

In my project the video files are here: /Resources/HTML/video

And my HTML Code is this: 

<video id="v" controls="true" loop poster="../HTML/images/img-cover.jpg"> 
    <source src="../HTML/video/video.webm" type="video/webm" />
    <source src="../HTML/video/video.mp4" type="video/mp4" />
    <source src="../HTML/video/video.ogg" type="video/ogg" />
</video>

But I get everytime this error message (from Dalvik Debug Monitor):
E/(106): Failed to open file '/android_asset/Resources/HTML/video/video.webm'. (No such file or directory)

Is it not possible to use local HTML5 videos with Appcelerator?

  • Meanwhile I found another problem playing HTML5 Videos in Android webviews. It's not possible to play videos with an Android Version lower than 3.0, because it seems you have to active hardware acceleration to see the video. Otherwise you can only hear the video sound.

    It's possible to active hardware acceleration with Android 3.0 and higher.

    I crosschecked it with a native android java app.
