iOS TextArea within ScrollView

I would like to use ScrollView that contains a title (Label) and TextArea. The whole problem is that when I start typing I can't see the last line. Everytime when I type some letter in the bottom or in the middle of the TextArea, ScrollView will scroll to the top… How can I disable this?

var win = Ti.UI.createWindow({
    backgroundColor: "black"
});

var scrollView = Ti.UI.createScrollView({
    width: Ti.UI.FILL, height: Ti.UI.FILL,
    contentHeight: Ti.UI.SIZE
});

var title = Ti.UI.createLabel({
    text: "Title",
    backgroundColor: "white",
    top: 0
});

var textArea = Ti.UI.createTextArea({
    width: Ti.UI.FILL, height: Ti.UI.SIZE,
    suppressReturn : false,
    scrollable: false,
    backgroundColor: "white",
    top: 20
});

scrollView.add(title);
scrollView.add(textArea);

win.add(scrollView);
win.open();

  • Hello, Sko, seems to work fine in SDK 3.4.0.RC, by setting scrollable: false, in TextArea. Try the below code as ScrollView and TextArea. Hope it will solve your problem.

    var scrollView = Ti.UI.createScrollView({
    width: Ti.UI.FILL, height: Ti.UI.FILL,
    contentHeight: Ti.UI.SIZE,
    contentWidth: Ti.UI.SIZE,
    showVerticalScrollIndicator: true
});

var title = Ti.UI.createLabel({
    text: "Title",
    backgroundColor: "white",
    top: 40
});

var textArea = Ti.UI.createTextArea({
    width: Ti.UI.FILL, height: Ti.UI.SIZE,
    suppressReturn : false,
    scrollable: false,
    borderWidth: 2,
      borderColor: '#bbb',
      borderRadius: 5,
      color: '#888',
    top: 60

    Thanks.
