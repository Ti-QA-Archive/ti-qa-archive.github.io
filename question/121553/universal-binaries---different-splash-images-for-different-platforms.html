Universal binaries - different splash images for different platforms

Hello all, my iPhone/iPad app has the exact same source code for the iPhone and iPad version. The only difference is the fact that the splash images for the iPhone are 320x480 px and the iPad ones are 1024x768.

However, when creating a project in Titanium Studio that supports both iPhone and iPad, it still only creates one folder called 'Resources/iphone/' for the splash images. Shouldn't there be a 'Resources/ipad/' folder to store ipad splash images? Or am I doing something wrong?

p.s. there has always been a 'Resources/android/' folder for android splash images, so why isn't there one for ipad?

Thanks in advance!
