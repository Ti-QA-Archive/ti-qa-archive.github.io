TextArea content modification

Is there a way to insert text (via, say, a button event) into a TextArea? The TextArea APIs seem fairly opaque.

  • var textArea = Ti.UI.createTextArea({value:'My value is: 0'});

var button = Ti.UI.createButton({title:'Change text area content!'});

button.addEventListener('click', function() {
textArea.value += 1;
});

    That'll increase the number in the TextArea by 1 everytime the button is clicked.
