How should window be reopened

Hello,

I have an app with multiple windows, the way I thought it works, is when I close the window, all objects defined in window are deleted. But when I open it back, all controls, labels etc, seem to be back where they were.

Question is, should I rely on this, or is this something like GC didn't have to time to clean things up and everything might be gone after a while?

Well basically my question is, what is a best way to reopen a window, without causing posible memory leaks?

So I define window, create all labels and buttons, etc in it. And then I close it using close method.

First scenario:

I simple use win.open to open it back

Second scenarion:

I nullify closed window, then create new one, define all objects in it again and open it.

What is the best choice for heavy window opening/closing applciation?
