    They are two different things.

    You initially download and install Titanium Developer and when you run the application it will prompt you to install the newest SDK. The note to install a new version will be on the top right side of the application where the Appecelerator icon is shown. If you give it a little bit, it will look out on the web and see it needs an SDK and you just need to click the message when it appears.

    BTW - They will have different version numbers. 1.2.1 is the current version of the Titanium Developer and 1.4.1.1 is the latest SDK. When you post questions about API calls, make sure to mention the SDK version number because it does sometimes help when trying to answer questions.
