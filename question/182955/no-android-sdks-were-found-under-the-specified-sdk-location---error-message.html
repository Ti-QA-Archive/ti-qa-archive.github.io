  • Ok, just I fix it at last!
    App SDK only need to install version 5.0.0 acelerator
    Click on this link:
    http://builds.appcelerator.com.s3.amazonaws.com/index.html#master
    Then …
    Branches> Master
    and select the version:
    5.0.0 August 21 time: 14:20:18

    then install it manually or from the Appcelerator.

    After you download the SDK AS, you open .. Helps> Installappcelerator SDK> Install from url … and looking for the sdk you downloaded.
