    • for completeness;

      index.xml

      <Alloy>
    <Window id="win">
        <View id="myview" autoStyle="true" class="green fillWidth sizeHeight leftMargin" />
    </Window>
</Alloy>

      index.js

      var c = $.myview.classes;

for (var x = 0; x < c.length; x++) {
    Ti.API.info(c[x]);
}

$.win.open();

      produces the console output

      [INFO] green
[INFO] fillWidth
[INFO] sizeHeight
[INFO] leftMargin

  • Classes in Alloy are only used to assign properties to an object during the transition from Alloy TSS to the generation of classic alloy code as part of the pre-compile step or since Alloy 1.2.0, applying properties dynamically.

    There is no class property on the object so this is not held.

    You would have to add custom properties to determine what you require.
