How to customize the annotation

I have a map and a custom pin for each location. When I click on a pin I get an area pop-up above the pin (title, subtitle). The text in the area is too close to the borders of the area. The questions is how to add some padding/margin around the text.
Android 4.2, SDK 3.1

  • Hi

    A new property has been provided for Ti SDK 3.1.1;

    • customView

    This allows you to provide your own view for the annotation, however it is limited to presentation only as there appears to be no events fired from it - at least directly.

    Worth a try if you do not need the interaction within the annotation directly.
