GPS Provider Power, what exactly does this mean?

I need to display to the user the signal strength of there gps location. I found this post which shows the values of each property in the gps provider object but still not sure what the power means. 

provider.power; // 0 No Requirement, 1 Low, 2 Medium, 3 High

Is that the amount of power its consuming (dependent on satellite), signal strength?

http://developer.appcelerator.com/question/11311/titaniumgeolocation-gps-or-network

    The power here is the power requirement for the provider.

    For example, using cell tower locations only may provide a lower accuracy location fix, but with a lower power requirement, while using GPS may provide higher accuracy but draw more power.

    In the native Android API, power is one of the criteria you can specify when you request a location provider, in order to allow the system to choose the best provider for your needs. I don't think it's what you're looking for As far as I know, there is currently no way to get signal strength information from Titanium.

    I've opened a bug to clarify the doc on what information is available in the event (TIMOB-7287).
