about Titanium.UI.TabGroup and Titanium.UI.window

I am not sure whether I can create a container(a view or window) for Titanium.UI.TabGroup? Could I put a Titanium.UI.window instance into a new container instant ( may be a view or window)?

  • Accepted Answer

    You can put a Titanium.UI.TabGroup inside of a window, but not a view. Windows are a top-level UI container and unless otherwise specified in the API, should not be nested inside other UI components. An exception to this rule, of course, is the when creating tabs for a TabGroup, which is mentioned in the tab API documentation.
