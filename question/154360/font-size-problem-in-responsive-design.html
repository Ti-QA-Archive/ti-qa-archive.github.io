    • Can you confirm you have set your tiapp.xml file to use dp rather than system. Also make sure you have cleaned the project before trying again.

      Then it should be no more complicated than;

      var lbl = Ti.Ui.createLabel({
    font: {
        fontSize: 14
    },
    text: 'Put something here'
});

      This should provide very similar results in iPhone; non-retina and retina. Android with most devices; keep in mind not all Android devices report back the correct density for the size of the device.

      — commented July 2nd 2013 by Malcolm Hollingsworth

    • So this problem will be the same outside of the app if you view the pages on the built-in browsers.

      Consider adding this meta tag to each HTML page, it may well snap things into place.

      <meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1">

      More details;

      • Using the viewport meta tag to control layout on mobile browsers
