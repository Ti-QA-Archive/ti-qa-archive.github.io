How to lazy load app?

Hello,

Titanium SDK version: 1.6.1
iPhone SDK version: 4.2

I am looking for ways to make my app faster. Every now and then I hear people saying that doing "lazy loading" speeds up the app.

My simple question is, what is lazy loading exactly and how can I use it in an app? It would be perfect with a small example. Some talk about using setTimeout. How is this done in practice?

  • Hi there Daniel,

    One tactic you could use is breaking your app into more contexts/js files, although this would introduce a bit more complexity as you'd need to manage the transfer of data between screens more. The added advantage is it breaks your code into more clearly defined units.

    For example, if you have a tableview which when clicking rows opens a new window, if you create a js file for that window, it won't be run until the user taps. Any pertinent data required by the new window can be attached to it when it is created.

    Basically the processing has to happen sometime, and it's a balance as to how much is done 'up front' and how much is spread over the rest of the app.

    Cheers,
    Chris
