  • yes, for sure taht will work fine.

    only you need to change tiapp.xml, which is in your resources directory of an application.

    <deployment-targets>
    <target device="mobileweb">false</target>
    <target device="iphone">false</target>
    <target device="ipad">false</target>
    <target device="android">true</target>
    <target device="blackberry">false</target>
    </deployment-targets>

    you have to make it only true…
