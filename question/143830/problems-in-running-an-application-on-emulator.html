    • if you want to set path then :

      edit you environment variable.. go to Start > Right Click My Computer > Properties > Advanced > Environment Variables.. then a pop-up window will appear.. Edit the JAVA_HOME to your JAVA location.. in my case i have

      
// C:\Program Files\Java\jdk1.6.0_11

      under the "system variables" then find the "Path" add the location of the BIN

      // C:\Program Files\Java\jdk1.6.0_11\bin

      in my case.. I hope this will help you and get you started
