    Hi

    You cannot write or append to files in the resources folder or sub-folders within the resources folder. This is because resources is read-only.

    var file = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory, "textfile.txt");
file.write('hello');
file = null;
