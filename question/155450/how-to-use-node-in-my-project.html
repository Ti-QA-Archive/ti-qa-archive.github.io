How to use node in my project

I want to use the following library found at github.com:

json2xml

I used the command to install the library, "sudo npm install json2xml". This installed the node module @ /Users/user/node_modules….I get a json2xml folder with a lib folder that contains json2xml.js….

I copy that file into my project in my lib folder. I use the following command to require it: 

var json2xml = require('lib/json2xml');

I have also used the following command as well: 

var json2xml = require('/json2xml');

Both ways of requiring it doesn't work. Any ideas how I can use this library in my project? What am I doing wrong?

    I figured it out, to use the library I had to use the following require statement: 

    var json2xml = require('json2xml/lib/json2xml');
