  • You need to use heavyweight windows if you'd like to be able to navigate back through them with the back button without any additional code. Read this guide for full details on what exactly a heavyweight window is: https://wiki.appcelerator.org/display/guides/Android+Module+Development+Guide#AndroidModuleDevelopmentGuide-HeavyweightandLightweightWindows

    Additionally, the answers here are correct that you can also make your main window close the app by adding exitOnClose to it.
