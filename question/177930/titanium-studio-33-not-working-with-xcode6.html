    Hello, Darren, You need to update your studio,sdk and CLI to do that please folow below instruction.

    *Getting Access to SDK/Studio 3.4.0

    To get access to both Studio and SDK 3.4.0 please follow these 3 steps

    1. Upgrade Studio

    Community Members: Click here to get access to Titanium Studio 3.4.0 RC.

    Appcelerator Platform Customers: Click here to download Appcelerator Studio 3.4.0 RC.

    1. Install Appcelerator/Titanium SDK

    On terminal, Run

    sudo ti sdk install --branch 3_4_X -d
    1. Upgrade alloy and CLI

    On terminal, Run

    sudo npm install -g alloy@1.5.0-rc3
Sudo npm install -g titanium@3.4.0-rc4

    Finally TO configur studio with Xcode6 RUN

     sudo xcode-select -switch /Applications/Xcode6.app/Contents/Developer/

    Now your app will be running using IOS simulator 8.0. Thanks. Hope it helps. Please mark as best answer if it does.
