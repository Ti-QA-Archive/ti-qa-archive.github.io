    • Hi Lalit,

      Change it directly in the SDK folder of Titanium. In Mac it would be:

      HD/Library/Application Support/Titanium/mobilesdk/osx/[sdk_version_you_are_using]/iphone/Classes

      It would be a good idea to back up the entire SDK folder before you change anything.
      And note, that this is only a workaround, solution should be provided by Appcelerator team in next SDK releases.

      Hope this clarifies
