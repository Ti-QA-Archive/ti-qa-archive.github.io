removeEventListener is not working for location event

Hello

I know that for removing a location event listener I have to pass the same function reference to the removeEventListener and addEventListener methods. But in the next code, after removing the listener, the location event is still being listened and the function locationCallback executed.

var locationCallback = function(e) {
    if (!e.success || e.error) {
        ...
    } else {
        ...
    }
}

...

Titanium.Geolocation.addEventListener('location', locationCallback);

...

Titanium.Geolocation.removeEventListener('location', locationCallback);

...

Do I have to do something more for removing my location listener?

Thanks

  • If the remove event listener code is being reached, then it should be removing the listener.

    So if it is not being removed, put a info before and after the statement. Or put it in a try catch, to make sure it is being called and executed.

    Alternatively, if you think you have the listener more than once, it could be adding it twice, with these listeners the identical listener can be added more than once and will be executed more than once.
