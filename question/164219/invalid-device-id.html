  • Hi, There are few can do to solve this issue :

    1. Open Xcode go to Device Manager , delete the old device and add the device again
    2. Clean your project
    3. to make sure a new configuration is created . Open /Users/("YOUR USERNAME)/Developer/CoreSimulator/Devices/ folder , delete the old devices.
    4. Open Titanium Studio/App Studio do to Run->Run configuration and link this new device with your build setup and launch again.
      This works in my case.
