    Hi

    Your logic is correct but your understanding is wrong;

    function add(e) {
  $.c.value = $.a.value + $.b.value;
}

    The + sign is the correct arithmetic symbol to add, but it is also the same to concatenate strings. As text fields returns strings, they are joined, so you need to convert their values to number.

    function add(e) {
  $.c.value = parseFloat($.a.value,10) + parseFloat($.b.value,10);
}

    Because each side of the symbol is a number then addition rather than concatenation is used.
