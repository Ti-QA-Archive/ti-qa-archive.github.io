Recognition of numbers and letters in pictures taken with mobile camera

Hi, I am interested in doing an application for iPhone that will use the camera to take pictures of plastic labels showing part numbers, in order to use them to search in a remote DB the part description, price, etc.. The labels are white background, and the text is black colored, I just need to read this number when the user points the camera.

Is there any module, API or component that can be used in Titanium Appcelarator in order to do this?

Thanks.

    You might be able to tap in to a cloud-based OCR API do to this. eg : http://ocrsdk.com/

    There might also be some heavy JS based solutions around, since I've seen colour and face recognition done with pure JS a while ago, so maybe there's been some development there.

    It's also worth searching any Objective-C out there which might be able to do it. Then you could find something/someone to be bale to build an ORC module.

    good luck!
