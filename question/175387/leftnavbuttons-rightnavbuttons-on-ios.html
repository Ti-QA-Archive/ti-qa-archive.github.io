LeftNavButtons, RightNavButtons on iOS

I've used the Ti.UI.Window properties leftNavButton and rightNavButton before and they work fine.

But I would like to add multiple buttons on the right side of the nav bar. Here is the classic Titanium code I tried, using the rightNavButtons properties (notice the 's' and the end of 'Buttons'):

var buttonDeleteLogs = Ti.UI.createButton({
systemButton: Ti.UI.iPhone.SystemButton.TRASH
});

var buttonToggleLocalRemote = Ti.UI.createButton({
title: "Remote",
style: Ti.UI.iPhone.SystemButtonStyle.DONE
});

$.winNavBar.rightNavButtons = [ buttonDeleteLogs, buttonToggleLocalRemote ];

I ended up with no buttons at all on the right side of the nav bar.

Using this line:

$.winNavBar.rightNavButton = buttonToggleLocalRemote;

or

$.winNavBar.rightNavButton = buttonDeleteLogs;

resulted in the expected button being displayed (but only one, of course).

Is there a bug in the leftNavButtons and the rightNavButtons properties? I'm using Titanium SDK 3.2.3GA and Alloy version 1.3.1.

Also, how can I use the leftNavButtons and the rightNavButtons properties in Alloy?

  • Hi guys,

    If you want to add leftNavButton, rightNavButton or multiple buttons, you have to use navigationWindow as a top level window manager, this is the iOS7 new feature otherwise naveButton will not add to the window.

    Here is an example code of alloy.

    ###index.xml

    <Alloy>
    <NavigationWindow id="navWin" platform="ios">
        <Window id="win2" title="Red Window" backgroundColor="red">
            <LeftNavButton>
                <Button id="left" backgroundColor="red">Left</Button>
            </LeftNavButton>
            <RightNavButton>
                <ButtonBar id="bb1" platform="ios" backgroundColor="#369" height="25" width="100">
                    <!-- The Labels tag sets the ButtonBar.labels property -->
                    <Labels>
                        <!-- Specify text with node text or the title attribute. -->
                        <!-- Can also specify the enabled, image and width attributes. -->
                        <Label>One</Label>
                        <Label>Two</Label>
                    </Labels>
                    <!-- Place additional views for the ButtonBar here. -->
                </ButtonBar>
            </RightNavButton>
        </Window>
    </NavigationWindow>
</Alloy>

    ###index.js

    $.left.addEventListener('click', function() {
     alert('Left was clicked!');
});

$.bb1.addEventListener('click', function(e) {
     alert(e.index);
});

$.navWin.open();

    If you have any question about this query, then let me know.
