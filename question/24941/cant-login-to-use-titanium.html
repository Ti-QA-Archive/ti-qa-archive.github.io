    Omar,
    I have just resolved it, with zero help from Appcelaerator. In my case our corporate ISA Firewall was applying it's rules to my Macpro using a fixed IP address. Due to a re-configuration my Macpro had a new IP address using DHCP and so it was getting blocked. I changed my Macpro to a fixed IP address that matched the one in our ISA server and the rules then let Titanium connect ok.

    Hope that helps you in some way as I did not have much info to go on and no easy way to trace the problem. We also have a proxy server setting and I had looked into that as a problem but it turned out to be the ISA server. Good luck.
