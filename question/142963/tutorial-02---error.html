  • That video is nearly two years old, and from a quick glance looks like it's going to work on only iOS.

    • Windows aren't like divs. Windows are like web pages. Views are like divs. He should be using a view rather than a window for the result he's looking to achieve.
    • On Android, you can't add a window as a child element of a window. You could open the second window on its own (like a modal, or as part of a tab set)

    You should be able to make the rest of what he's showing work if you use views. Something like:

    var subwindow - Ti.UI.createView({
    height:100,
    width:100,
    borderWidth:1,
    borderColor:"#CCC",
    borderRadius:6
});
