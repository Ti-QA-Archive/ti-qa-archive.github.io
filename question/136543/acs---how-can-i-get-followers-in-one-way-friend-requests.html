ACS - How can I get "Followers" in one-way friend requests?

I have one-way friendship system setup using ACS… and I'm successfully able to query who the current user is following and also if I pass a user_id, I can find out who that user is following too. However I need to also find out who is FOLLOWING the current user or the user I provide in the user_id.

How can I do this? Thanks! :-)
