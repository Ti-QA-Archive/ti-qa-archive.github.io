  • Okay, it’s not a demo but the blank project template.

    1. Start app (Dashboard shows up)
    2. Select the Develop tab
    3. Choose a sample app (e.g. GeoCoder) and click Import (New Sample Project window pops up)
    4. Click Finish (which starts cloning the git repo)
    5. Goto the Project Explorer and right-click the GeoCoder project
    6. Select "Run as > iPhone Simulator" or Android

    It’s all the same. It’s the same behaviour like I got before with a new mobile project. These steps are:

    1. File > New > Mobile Project
    2. Default Project > Next
    3. Enter project name and app id > Finish
    4. Goto the Project Explorer and right-click the new project
    5. Select "Run as > Android Simulator" or iPhone
