  • Hi

    There is nothing wrong with that code syntax;

    var Stores = Ti.UI.createButton({
    title: "Stores",
    width: "25%",
    height: "10%",
    bottom: "24%",
    left: "34%",
    id: "stores"
}); // should have a semi-colon but not causing this problem

    Are you SURE this is the line the error is on? Check the problem and make sure this is the problem.
