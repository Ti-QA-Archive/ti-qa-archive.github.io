    • Hey Manu, I believe that Colored Logcat by Jeff Sharkey will only work on Unix operating systems, but not on Windows. For Windows, try Colored Logcat Script for Windows. However, I would recommend ddms over either one of them. (navigate to "C:\Program Files (x86)\Android\android-sdk\tools\ddms.bat" or wherever you have android-sdk installed). The reason i recommend ddms is because that, in addition to nice color-coding, there is a nice little GUI for setting up filters. (so you can filter out only "TiAPI" messages if you want)
