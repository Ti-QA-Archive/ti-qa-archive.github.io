  • Well, from the looks of it, your ProviderService is just a plain old WebService.

    You can use the Suds: A JavaScript SOAP Client Library available on GitHub.

    Your code could then look something like this:

    var suds = new SudsClient({ 
  endpoint: "http://46.18.8.42/FareBids.Service/FareBids.ServiceLayer.ProviderService.svc",
  targetNamespace: "http://46.18.8.42/" 
});

suds.invoke("NameOfMethodYouWantToCall", {}, function(xmlDoc) {
  //Parse XML response (SOAP Envelope)
});
