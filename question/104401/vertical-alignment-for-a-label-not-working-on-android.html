Vertical Alignment for a label not working on Android

Hi,

Trying to achieve a label with vertical alignment so that the text is at the bottom.

It's in a view container of say 50x50. Based off Tweetanium code.

Android 2.2SDK, Ti Mobile 1.6 SDK, Mac OSX

When I try the following (along with some text like "testing") the label just ends up centered in the container view. It doesn't seem like the verticalAlign is doing anything at all.

    tabNavText: {
            color:tt.ui.theme.lightBlue,
            textAlign: 'center',
            verticalAlign: 'bottom',
            font: {
                fontFamily:tt.ui.theme.fontFamily,
                fontSize:12
            },
            height:'auto'
        },

I also tried this 

label1.verticalAlign = Titanium.UI.TEXT_VERTICAL_ALIGNMENT_BOTTOM

Which was copied from Kitchen sink and works for Buttons (where it's copied from) but it doesn't seem to work on labels….

Any idea how to align this label text to the bottom of the holding view container.. basically bottom alignment.

  • I did end up getting this working tho it sure wasn't at the time.

            tabNavText: {
            color:tt.ui.theme.grayTextColor,
            textAlign: 'center',
            verticalAlign: 'bottom',
            font: {
                fontFamily:tt.ui.theme.fontFamily,
                fontSize:10
            },
            bottom: 0,
            height:'auto'
        },
