  • Hello MyungHwan Kim,

    You can easily change API level. When you want to run your apps in Emulator then please follow my instruction for changing API level.

    Steps:

    • Select your project
    • Click on right button menu will appears
    • Select Run As > Run Configurations It will display new window
    • Select Android Emulator
    • Click on left top button New Launch configuration
    • Now in right side will change with few input fields
    • Browse your project if not selected
    • Change Android SDK, Screen and Log Level
    • Now Click on RUN button for apps test.

    I hope this will help you.
