  • Hi,

    you need to put your file in assets folder. Assets folder you can create two types. Either you can create assets folder in your titanium project like project > platform > android > bin > your file or in your module project'.

    When you want to access that file you can try following code: 

    AssetManager assetsManager = this.getActivity().getAssets();

//If you want to create fileDescriptor you can do following
        AssetFileDescriptor afd = null;
        try {
            afd = assetsManager.openFd("abc.pdf");
        } catch (IOException e1) {
            e1.printStackTrace();
        }
        FileDescriptor fd = afd.getFileDescriptor();

//Or if you want InputStream please try following code:
InputStream is = assetsManager.open("m1.map");
