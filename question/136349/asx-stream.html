    Here is some sample code

        var audioPlayer = Ti.Media.createAudioPlayer({
        url: 'http://<host>:<port><optional mount point>',
        preload: false,
         allowBackground: true
    });

    audioPlayer.audioSessionMode = Ti.Media.AUDIO_SESSION_MODE_PLAYBACK;

    audioPlayer.addEventListener('change',function(e)
    {
        Ti.API.info('State: ' + e.description + ' (' + e.state + ')');
    });
    audioPlayer.start();

    • You have to add info.plist with following to the root of the project (iOS only, not sure of android) to play music in background:

      You go into the build directory of the project, and grab the info.plist. Edit it, adding the following

      <key>UIBackgroundModes</key>
    <array>
        <string>audio</string>
    </array>

      Then save it in the Root of the project
