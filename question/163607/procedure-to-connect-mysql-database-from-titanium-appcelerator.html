Procedure to connect MYSQL database from Titanium Appcelerator.

Hi,
My titanium mobile project need to read update delete data on MYSQL database.
Please let me know the procedure to connect to mysql database.
The search items says first need to Use Titanium.Network.createHTTPClient to create a new HTTPClient object.
Then i need to call the URL which contains PHP page, that intern will get the MYSQL connection and get the data.

As i dont want to use PHP, is there any other way to get mysql database connection and do CRUD operations.

    You cannot connect directly to your mySQL database from within Titanium.

    You need to create your API's. Using which ever service language you wish. Although you can use PHP, there are lots of other options. Quite a few people are looking at using Node.js, as it links nicely in with ACS (Appcelerator cloud services) and the programming methodology of Titanium.
