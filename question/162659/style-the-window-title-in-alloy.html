  • Accepted Answer

    Take a look at TitleControl.

    <Window>
  <TitleControl></TitleControl>
</Window>

    You can add a label

    <Window>
  <TitleControl>
    <Label id="myTitleLabel">Tab 1</Label>
  </TitleControl>
</Window>

    Then style it in your tss, color, fonts, etc

    • Sorry for quick comment, I like to add more clarification. The above solutions works in two ways:

      1. The solutions with TitleControl element better works with having window translucent set false and setting bar color. Or,
      2. Defining style of label "myTitleLabel". Set full height, width, backgroundColor, Color

      I think better way is to go with 2 option

      Full code for 1 would be:

      
"Window": {
    backgroundColor: "#fff",
    translucent: 'false',
    barColor: '#2656ac'
},

"#myTitleLabel":{
    backgroundColor: '#2656ac',
    color: '#fff'
}
