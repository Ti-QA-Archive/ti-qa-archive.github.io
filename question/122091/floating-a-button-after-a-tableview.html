Floating a button after a tableview

Hi. My app has a window that displays a tableview. 15px below the last row of data in the tableview (which is of varying length/number of rows) I need to display a button.

I can't seem to get this positioned properly at the end of the tableview, especially when the tableview data is longer than one screen. I'm setting layout:'vertical' in both the tableview and the button, and leaving the top property unset.

Help?

  • Accepted Answer

    The easiest course would probably be to add another row to your table view with the button.

    The row could have a transparent background or some other custom view containing the button so that it appears different to your other rows. If it is a grouped table view, the row would need to be in its own table view section.

    Alternatively, I think the 'official' answer is to use the footerView property, but personally I have had all sorts of difficulties trying to position the table view correctly once the footer view has been added so perhaps stay away from that if you can.
