  • I'm using it. Open the Studio dashboard (red house icon). Under Get Started tab, at bottom is "Configure Native SDK's". Is there a checkmark next to Android SDK or a red X? If red X, you need to fix your Android SDK installation. Follow the steps provided and be sure to install api level 10. You may not use it for your apps, but it has to be installed anyway. I found that out the hard way.

  • Thank you all. I have it running now, it was a java path problem like I thought. For anyone else with similar problems, my error occurred when I tried to move java because it appeared that my location did not match Ti's documentation. I juggled it around and got it all put back just like it installed to begin with and all is well.
    And, BTW, I'm using the java download from the Apple website (which the documentation points to (http://docs.appcelerator.com/titanium/latest/#!/guide/Installing_Oracle_JDK).
    Briefly, if I had left it alone (java) I never would have had any problems.
