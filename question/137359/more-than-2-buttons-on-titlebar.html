More than 2 buttons on titlebar?

I know there is setLeftNavButton and setRightNavButton to add buttons to a title bar. What if I want additional buttons in the title bar (a third title bar button apart from the right and the left nav button)? How do I do that for iPad?

  • One method is to create a ButtonBar with multiple buttons, then set that object as the right (or left) Nav button. If the left Nav button is still a back button, it would be more typical to use a button bar on the right. An example from the kitchen sink is here (see object bb3).

    You can also set any view (including a button or button bar) as a title control, meaning it replaces the title in the center of the navigation group.

  • you can create a custom Toolbar and add any number of buttons in it…take a look at the code below

    var myview = Titanium.UI.createView({
                    height:'251dp',
                     top:'0dp',
                     backgroundColor:'#fff'
                 });

        var cancel =  Titanium.UI.createButton({
                title:'Cancel',
                style:Titanium.UI.iPhone.SystemButtonStyle.BORDERED
                });

        var done =  Titanium.UI.createButton({
                title:'Done',
                style:Titanium.UI.iPhone.SystemButtonStyle.DONE
                });
//custom button added
       var custom =  Titanium.UI.createButton({
                title:'custom',
                style:Titanium.UI.iPhone.SystemButtonStyle.BORDERED
                });
 // flexible space added for auto adjustment while orientation
         var spacer =  Titanium.UI.createButton({
               systemButton:Titanium.UI.iPhone.SystemButton.FLEXIBLE_SPACE
               });

        var toolbar =  Titanium.UI.iOS.createToolbar({
               top:0,
               items:[cancel,spacer,custom,spacer,done]
              });

myview.add(toolbar);

// Add this view to a window and you can see a toolbar with 3 buttons. similarly you can add more buttons to it.

    hope that helps :)
