Key board caps lock

It seems that by default a textfield capitalizes the first letter. How do I turn this off?

— asked February 13th 2011 by Matthew Gagnon

    When you define the field, use one of the following values for the autocapitalization property:

    • Ti.UI.TEXT_AUTOCAPITALIZATION_NONE
    • Ti.UI.TEXT_AUTOCAPITALIZATION_WORDS
    • Ti.UI.TEXT_AUTOCAPITALIZATION_SENTENCES
    • Ti.UI.TEXT_AUTOCAPITALIZATION_ALL
    var example = Ti.UI.createTextField({
    autocapitalization: Ti.UI.TEXT_AUTOCAPITALIZATION_NONE,
    keyboardType: Ti.UI.KEYBOARD_DEFAULT,
    returnKeyType: Ti.UI.RETURNKEY_DEFAULT,
    autocorrect: false
});
