I need a save button

I have a tab group, in one of my tabs I have stacked windows which gives me a back button at the top of the screen. How do I add a save button to that bar at the top? I'm new to this, this is my first app so there may a different way to do this.

I appreciate any help anyone can offer. Thank you.

  • Accepted Answer

    var saveButton = Titanium.UI.createButton({
    style:Titanium.UI.iPhone.SystemButton.SAVE
});

win.setRightNavButton(saveButton);

  • Short answer: create a button object and assign that to window property rightNavButton like this:

    var saveButton = Ti.UI.createButton({
    title:'Save',
    style:Ti.UI.iPhone.SystemButtonStyle.BORDERED
});
Ti.UI.currentWindow.rightNavButton = saveButton;

saveButton.addEventListener('click', function(e) ({
    // do something useful here
});

      • Look in Resources/Examples folder for a *.js file with a name similar to the API module you think may be helpful. In this case perhaps button.js or navgroup.js would be reasonable first guesses.
