Is this an error in the API documentation?

Under iOS Navigation group, the example shown is:

var win1 = Titanium.UI.createWindow();

var win2 = Titanium.UI.createWindow({
    backgroundColor: 'red',
    title: 'Red Window'
});

var nav = Titanium.UI.iPhone.createNavigationGroup({
   window: win2
});

win.add(nav);
win.open();

var win3 = Titanium.UI.createWindow({
    backgroundColor: 'blue',
    title: 'Blue Window'
});
nav.open(win3, {animated:true});

My issue is with the "win.add" and the "win.open" – where is "win"? Do they mean "win1"? Or am I just completely confused here?

SDK 2.0.1, iOS 5.1
