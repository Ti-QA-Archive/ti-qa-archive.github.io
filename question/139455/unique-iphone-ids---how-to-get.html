  • Note that the iPhone UDID is not supported anymore by apple appstoyre policies. Titanium has the function Titanium.Platform.getId() to retrieve a unique identifier for that app. But note that this identifier is unique for each installation, which means that, if the user reinstalls the app, the unique identifier will be different (and will remain the same while the app is not uninstalled).
