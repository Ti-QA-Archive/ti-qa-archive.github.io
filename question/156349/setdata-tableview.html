    Hi

    You have forgotten to include the table view as the prefix to the tableView setData.

    Change your code as follows;

    right.addEventListener('click', function() {
    alert(section[19].lastRow());
    inicio = section[19].lastRow();
    table.setData([]);
    setTimeout(function() {
        table.setData(inicio); // this has changed
    },1000);
});

    However your code does NOT require the setTimeout, you can further improve the code as follows;

    right.addEventListener('click', function() {
    alert(section[19].lastRow());
    inicio = section[19].lastRow();
    table.setData(inicio); // this is different
});
