Galaxy S5 fingerprint scanner?

Hi all,

Some Android devices are equipped with a fingerprint scanner. It is very convenient to replace a password. Titanium handles touchID for IOS, but for Android?

samsung to let developers tap into galaxy s5 fingerprint scanner

I know that a third company can develop a specific module, but it would be good to handle this feature.

Thank you,
olivier

  • If there is no native Android API to access this kind of hardware, there is no chance that Titanium is going to support this via 3rd-party libraries.

    As this is a OEM specific feature your only chance to support this is building your own module using their SDK. The article you've linked describes that Samsung does provide an API for the fingerprint scanner. So at least it is not impossible to make use of it.
