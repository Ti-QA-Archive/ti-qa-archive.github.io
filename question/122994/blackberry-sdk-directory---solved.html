Blackberry SDK Directory - SOLVED

For all of you having this issue like me. Here's the solution.
Took me a lot of time to figure this out. I'm placing this here for people looking for this.

I assume you already have downloaded the BB Keys into a folder in your machine.

STEP 1:

Download Blackberberry Beta 2.
Login to your account and download from here:
(Must have at least Indie account for access)

Download Blackberry Beta 2 Here

STEP 2:

Extract the content on the folder in the same folder where the 'mobilesdk' Directory is.

Where it is?
Go to Window --&gt; Preferences --&gt; Titanium Studio --&gt; Titanium and grab the path from Titanium SDK Directory box

Tipically here (in windows xp) (different place in Windows 7)

C:\Documents and Settings\All Users\Application Data\Titanium\

after unzip, youll get a folder structure like this:

mobilesdk\
  \win32
      \1.7.0.RC2
      \1.7.1
      \1.6.2
      \1.4.0
      \1.2.0

STEP 3:

Point to folder where Titanium executable is. You can track this folder right-clicking on the Titanium Studio Shortcut on the Desktop –> Properties –> Find Target

Tipically here (in windows xp) (different place in Windows 7)

C:\Program Files\Appcelerator\Titanium Studio

In Titanium Studio, go to Window --&gt; Preferences --&gt; Titanium Studio --&gt; Titanium
And place that path in the Blackberry SDK Directory box

STEP 4:

Place the path of the bb signed keys in the Blackberry Keys Directory

OK, Restart Studio

Tested it on windows xp and windows 7, folders are different but it works.
HOPE IT HELPS.
