My problem: When I try to run on iOS through Titanium Studio, I click on the "Upload" link to Upload the Development Provisioning Profile. I select the provisioning profile and click "open." I then wait indefinitely because TS has frozen / hung. I just see the rainbow circle spinning its beautiful wheels.

I've successfully installed/installed my certificate on my keychain. I've entered the appID into apple's dev site and copied the same ID into the tiapp.xml <id> element. (not including the Bundle Seed) Now I'm just using the wildcard * as the appID hoping to get that to work.

I've verified that I can build a test app through Xcode and run it successfully on the iOS device.

The provisioning profile is already installed, so I'm not sure why Titanium Studio isn't recognizing it. Should that "Upload" link even appear if it recognizes the profile?
