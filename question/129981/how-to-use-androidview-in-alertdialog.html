    Hi,

    Please do not type your question in the answer box, put it as a comment. For buttons you can have an onclick eventListener which can check the button index that was clicked.

    
//if dialog has 2 buttons 

dialog.buttonNames = ['Yes', 'No']; 

dialog.addEventListener('click', function(e) { 

     if(e.index===0) 
     {
        alert('Yes');
     }

  • Code

    var view = Ti.UI.createView({
    top:0,
    width:150,
    height:150,
    backgroundColor:'#FFF'
});


var dialog = Ti.UI.createAlertDialog({
        androidView:view
});

dialog.show();
