How to correctly set visibility of a nested view in titanium alloy

Windows 7, Titanium SDK version: 3.1.2 (08/14/13 12:45 5ceaff8), Titanium Studio, build: 3.1.2.201308091617

I am trying to hide and show two views in my xml when respective buttons are pressed. I have the following nested xml code:

<Alloy>
<Window class="container">
    <View id="main">
        <View id="viewone">
            <Label>View 1</Label>
        </View>
        <View id="viewtwo">
            <Label>View 2</Label>
        </View>
    </View>
</Window>

Then i have two normal buttons(openViewOne, and openViewTwo) with onClick attributes in xml to trigger necessary methods,

function openViewOne(e) {
    $.viewone.setVisible(true);
    $.viewtwo.setVisible(false);
}

and for openViewTwo fn,

function openViewTwo(e) {
    $.viewone.setVisible(false);
    $.viewtwo.setVisible(true);
}

i have also tried the following

 $.viewone.visible = true;
 $.viewone.hide();

to no avail, any thoughts? How can i show/hide views when buttons are pressed.
