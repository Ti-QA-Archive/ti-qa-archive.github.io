Database causes crash on 4.3

Hello,

I install a database the first time a user starts my iPhone app, I use the following code and it works perfectly if I use iPhone SDK 4.2, the database is created and I am able to read and write to it. 

Ti.Database.install("/mydb", "mydb");
var db = Ti.Database.open("mydb");
db.execute("CREATE TABLE IF NOT EXISTS test (test_id INTEGER PRIMARY KEY)");
db.close();

But if I use iPhone SDK 4.3, the app immediately crashes on startup. If I use 4.3 and comment out the code above, it works again. Anybody have any idea why it works with 4.2 but not with 4.3?
