  • Hi Matt,

    Where are you placing the button object to? I've tried reproducing the problem with the code you provided and had no trouble building it.

    //Ti SDK 2.0.1.GA2, iOS SDK 5.1

var b = Titanium.UI.createButton({
    systemButton: Titanium.UI.iPhone.SystemButton.ADD
});
win.setRightNavButton(b);

    Just a reminder - systemButton is only used when adding a button to a toolbar, nav bar, or other object which supports system button styling. These icons do not appear for buttons added to a view which does not support this.

    -Jae
