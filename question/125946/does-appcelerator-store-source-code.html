  • Titanium Studio won't correct incorrect code but it will give you some hints if you use the autocompletion feature - switch it on with ctrl + space if you don't see it working while you type.

    For example, start typing TI.a and you should see a little popup appear with a list of suggested methods you could insert, such as API. Hit enter to select the first suggestion and it's typed in for you.
