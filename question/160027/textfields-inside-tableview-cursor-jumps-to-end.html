Textfields inside Tableview. Cursor jumps to end

I've got a TableView with rows containing Textfields. Everything works nicely, except one thing:
When an user edit the text, the cursor moves to the end of the field.
usually, this is the intended behaviour (normally an user is writing at the end of the field, but if he is editing an error, it is very frustrating to see the cursor jumping at the end with every keystroke.

The problem i'm experimenting happen only if the texfield is inside the table view.

Has anyone ever experienced this issue? Is there a workaround, or may be I missed something important with tableViews?

thanks

Alex

  • I just solved this issue by creating a View, and then putting the TextField into the View. I then added the View to the TableRow. This stopped the cursor jumping to the end of the field each time you typed a character in the field.
