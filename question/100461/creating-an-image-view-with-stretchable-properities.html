      var frameB = Ti.UI.createImageView({

    You haven't provided enough information for anyone to help you. Please read Participating in the Q&A to discover what the required information is.

    Furthermore, can you explain what you mean by a "stretchable" png? Do you mean a 9Patch (NinePatch) png, which is only available on android? If so, are you using the Android-SDK 9Patch tool provided? Take a look at this for information. This tool will check to ensure that the file is valid, and also save the file with the correct extension that is required for 9Patch to work.

    By the way, the background*Cap properties should not be used if you are using 9Patch. To see how those properties in action, see textfield_therest.js

    • Hai, I was trying to use this backgroundTopCap as documented in the Titanium Mobile API:

      backgroundTopCap: "End caps specify the portion of an image that should not be resized when an image is stretched. This technique is used to implement buttons and other resizable image-based interface elements. When a button with end caps is resized, the resizing occurs only in the middle of the button, in the region between the end caps. The end caps themselves keep their original size and appearance. This property specifies the size of the top end cap. The middle (stretchable) portion is assumed to be 1 pixel wide. The bottom end cap is therefore computed by adding the size of the top end cap and the middle portion together and then subtracting that value from the height of the image"

      This is the iSO API call that's probably be ultimately called by Titanium:
      UIImage.h:

      • (UIImage *)stretchableImageWithLeftCapWidth:(NSInteger)leftCapWidth topCapHeight:(NSInteger)topCapHeight;

      I finally figured out the problem being that I can't specify both backgroundTopCap and backgroundLeftCap at the same time. It must be one or the other. This might be a Titanium bug?

      Here's my resulting code that works for stretching in one direction only.

      var frame = Ti.UI.createLabel({
        height:170,
    backgroundLeftCap:0.49,
    backgroundImage:'container280.png'
});
