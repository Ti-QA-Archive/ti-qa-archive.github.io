Database file location

I'm working on an Android application that use a sqlite database. The database is deployed using Titanium Studio (is in the resource directory) and I use that using:

var db = Ti.Database.install('myDb', 'mainDb');

With the terminal I see that the database is in this directory (on the device):

file:///data/data/com.appID.appName/databases/mydb

Now I would like to get the database file and copy it to another directory (o send to an http server….is quite small).
How do I get the path to that file? I try with Ti.Filesystem.applicationDataDirectory, and other ways but I can't get to the databases directory.
