  • Sandeep

    There is a good guide called Basic UI Design Tips for Mobile Apps that contains the general best practices in mobile app UI design, although I'm afraid it doesn't cover precisely your question.

    Regarding your issue, using the auto value isn't necessarily the best approach. Ideally the image files will be optimized so that they are already at the resolution they will be displayed on screen. This may be done using the webservice that serves your application.

    Also, use absolute values for imageView widths and leave the height undefined. If you have to scale them, this will ensure the image scales proportionally.

    Hope this helps
