iPhone search bar animate cancel button and disable background color

Hi all.
I've been working on creating an app on Iphone and it has a search bar.
the 2 things I'm trying to do is are:

  1. animate the showing and hiding of the cancel button.

  2. get rid of the background colors for the search bar.

here is a screenshot, and the code for the search bar is this:

var sBar = Ti.UI.createSearchBar({
    hintText: 'Search '+aLocation,
    showCancel: false,
    top: 5,
    left: '5%',
    width: '90%',
    height: 80,
    keyboardType: Ti.UI.KEYBOARD_ASCII,
    returnKeyType: Ti.UI.RETURNKEY_SEARCH,
    barColor: 'transparent',
    borderColor: 'transparent',
    backgroundColor: 'transparent'
});
sBar.addEventListener('focus', function(){sBar.setShowCancel(true);});
sBar.addEventListener('blur', function(){sBar.showCancel = false;});
sBar.addEventListener('cancel', function(){sBar.value = ''; sBar.blur();});
