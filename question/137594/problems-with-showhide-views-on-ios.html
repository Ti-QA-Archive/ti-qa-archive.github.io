Problems with show/hide Views on iOS

I have a button and a eventlistener. Once the eventlistener is triggered, view1 is hidden, view2 is shown. It is working fine on Android Emulator, Android Device and IPhone Emulator. But for some reason, it does not work on IPhone device. Oncethe button is clicked, nothing happens. If I put an altert into the event-listener, the alert message is triggered, but the show/hide commands seem to be ignored (however, there i no errormessage or crash)

This is my code:

var view_1 = Titanium.UI.createView({  
    width: '100%',
    backgroundColor:'#fff',
    backgroundImage:'bg.png'
});


var view_2 = Titanium.UI.createView({  
    width: '95%',
    backgroundColor:'#fff'
});


var view_2_label = Titanium.UI.createLabel({
    color:'#999',
    font:{fontSize:'16dp',fontFamily:'Helvetica Neue'},
    textAlign:'center',
    text: 'Test',
    width:'auto',
    top: '40%',
    bottom: '40%'
});


var button_1 = Titanium.UI.createButton({
    top:'30%',
    bottom:'60%',
    backgroundImage: 'button.png',
    width:'75%',
    color: '#000000',
    font:{fontFamily:'Arial',fontWeight:'bold', fontSize:'16dp'}
    });


button_1.addEventListener('click',function(e)
{
    view_1.hide();
    view_2.show();
});

view_1.show();
view_2.hide();
view_1.add(button_1);
view_2.add(view2_label);
win1.add(view_1);
win1.add(view_2);
win1.open();

I am working with the recent 2.0 and IOs 5.1

  • Hello Marcel,

    Try to set visible property of view instead of show/hide…. see the below code….

    
var view_1 = Titanium.UI.createView({  
    width: '100%',
    backgroundColor:'#fff',
    backgroundImage:'bg.png',
    visible : true
});


var view_2 = Titanium.UI.createView({  
    width: '95%',
    backgroundColor:'#fff',
    visible : false
});


var view_2_label = Titanium.UI.createLabel({
    color:'#999',
    font:{fontSize:'16dp',fontFamily:'Helvetica Neue'},
    textAlign:'center',
    text: 'Test',
    width:'auto',
    top: '40%',
    bottom: '40%'
});


var button_1 = Titanium.UI.createButton({
    top:'30%',
    bottom:'60%',
    backgroundImage: 'button.png',
    width:'75%',
    color: '#000000',
    font:{fontFamily:'Arial',fontWeight:'bold', fontSize:'16dp'}
    });


button_1.addEventListener('click',function(e)
{
    view_1.visible = false;
    view_2.visible = true;
});

view_1.add(button_1);
view_2.add(view_2_label);
win1.add(view_1);
win1.add(view_2);
win1.open();
