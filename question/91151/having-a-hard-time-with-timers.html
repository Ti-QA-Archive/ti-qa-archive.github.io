  • Rui

    Mozilla has an excellent resource, that describes how to use standard Javascript functions, on their MDN. See setTimeout and setInterval.

    Do you know that you can use Firebug's console to run and test standard JS?

    So, using firebug, the following demonstrates the correct usage of the methods:

    var count = 0;
var intervalID = setInterval(function()
{
    count++;
    console.log("Interval fired. Count = " + count);
    if(count === 70){
      clearInterval(intervalID);
    }
},10);

    var count = 0;
setTimeout(function()
{
    count++;
    console.log("Function fired with timeout. Count = " + count);
},3000);

    I have tested these, and they work as expected.

    Hope this clarifies some things.

    Good luck! :)
