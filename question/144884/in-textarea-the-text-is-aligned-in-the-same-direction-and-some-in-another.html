  • In TextArea I downloaded the data from XML -file. Data:

    var name = data.getElementsByTagName("name").item(0).text;
var comment = data.getElementsByTagName("comment").item(0).text;
var isText = comment+name;

var row = Ti.UI.createTextArea
({
    autoLink:Ti.UI.iOS.AUTODETECT_ALL,
    left:5, top: 5, right: 5, height: isScreenheight,
    editable: false, // this needs to be set to false, otherwise data detection type will fail
    value:isText
});

    The essence of the question is, I need to have the text "name" has been aligned to the left, and was written in italics, and the text "comment" is aligned to the right, and is written in normal type.

    I do not know how to implement it.
