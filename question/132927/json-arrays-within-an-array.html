JSON Arrays within an Array

I'm fetching data from a server. The output is in JSON, but one of it has Arrays within an Array.

Example Output:

"infos":[
    [
        {
            "title":"title 1",
            "desciption":"description 1"
        },
        {
            "title":"title 2",
            "description":"description 2"
        },
        {
            "title":"title 3",
            "description":"description 3"
        }
    ]
],

What i'm trying to do:

var jsonData = JSON.parse(this.responseText); 
var info = jsonData.infos

for (var i = 0; i < info.length; i++) {
    var title  = info[i].title; 
    var desc   = info[i].description;
}

if i:

Ti.API.info(info.length);

it will return as 1.

I need to know how to handle Arrays within an Array if it's even possible. Thanks!

    for(var i=0; i<info.length; i++)
    {
        for(var j=0; j<info[i].length; j++)

    info = [ [ … objects … ] ] (there is just one array element)

    info[0] = [ … objects … ]
