Detect if App installed on iPhone

I want to detect if an app like TomTom is installed, and if it is to pass through an address to it to load using the TomTom url format:
http://handleopenurl.com/scheme/tomtom

1) Is there a way to detect if TomTom is installed? Or another way of elegantly dealing with the fact that some devices will have TomTom and others will not?

2) How do I pass through the call to TomTom? (I can probably figure this out myself if the docs talk about url handling…).

Any tips appreciated…

Matthew

    This only works on iOS, but you can use the Ti.Platform.canOpenURL property to see if the current iOS device can handle the url you provide. If you pass it a TomTom style url and it says it can handle that…..then the device probably has TomTom installed (or at least some other app saying it can handle tomtomhome urls).

    TomTom really doesn't provide instructions on how to format the URL, personally I just created a route from within the app and emailed the directions. The URL in the email will launch the app. Basic format seems to be tomtomhome://geo:action=show&lat=#&long=#&name=Place.

    While TomTom is fairly secretive, if you want to support Navigon as well, they offer complete documentation.
