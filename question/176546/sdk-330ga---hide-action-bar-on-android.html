[SDK 3.3.0GA] - Hide action bar on android

Hi all.

I've been looking on here on Q&A and also on Google for a solution to my situation.

Appart of the splash screen, I have a loading window that is shown when the app is gathering remote information. Now with SDK 3.3.0, action bar on Android is always visible and I know I can hide it by creating a custom theme, or by hide it after the window is open with win.activity.actionBar.hide(), but non of this 2 options fits my needs.

Removing the action bar by creating a custom Android theme, will remove completely or I can still use it if I need?

Removing it by hiding after the window is open doesn't work properly, I can still see the action bar showing just before it is hidden, this no only is a solution but it also doesn't give a professional image of the app.

What I'm trying to accomplish:

[loading window] 
// no action bar //

after all data is loaded from the server

[menu window] 
// with action bar to use **onHomeIconItemSelected** to close it or navigate back if other windows are open on top //

    I would suggest you not to use a Window for only showing a loading indication. A better way of doing this is to show a progress indicator within the view that get replaced by the content once loaded.
