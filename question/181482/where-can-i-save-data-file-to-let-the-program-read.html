  • Hi Bobin,

    You can put your data file in Resource directory (Assets if you work with alloy). and you can get data from file using file system.

    e.g

    var file = Ti.Filesystem.getFile(Ti.Filesystem.resourcesDirectory, "textfile.txt");
var blob = file.read();
var readText = blob.text;
alert(readText);

    for more info read this
