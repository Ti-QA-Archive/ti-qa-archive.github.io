  • Accepted Answer

    as a general rule, I would say that there is no need to close() a hidden window, since close() does not release memory nor have any other effect on windows that are not visible. In fact, close() only will hide it. But you should close() and destroy windows you do not need anymore, does not have persistent data (so you can create again) or are part of a navigation group.

    But if you certainly know that the user will comeback to this window, do not close it, unless there is a good reason to do it.
