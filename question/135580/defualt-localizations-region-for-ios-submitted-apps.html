Defualt Localizations region for iOS submitted apps

Hello, i've made an app and wanted to submit to apple, i'm italian and the app should be distribuited in italy only. Now, i've this problem, the binary sent to apple shows as default localization language english and i'd like to set it to Italian otherwise i have to setup the additional language, providing description and screenshots for that language. In my project i've no i18n resources because there is no localization at all, as said, the app is totally in italian language. Is there a way to tell titanium to put italian in the binary description instead of the default english?
Any help is really really appreciated, i'm stuck with the submission…

Regards

    Hi!

    You just need to change the language in your Info.plist which is located in the build/iphone directory. To keep the same language, just copy the Info.plist from build/iphone to your project root.

    • Thank You, that worked perfectly!!! i've copied the info.plist found inside the iphone build folder to the root of the project and then edited the line for the key CFBundleDevelopmentRegion from "English" to "it_IT"!
