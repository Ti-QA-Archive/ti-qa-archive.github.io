  • Logical Labs has released to the marketplace an External Accessories Module which does allow you build apps that can connect to and communicate with MFi devices.

    It will handle wired and wireless (Bluetooth Classic) connection types via the External Accessory Framework.

    The module includes a working example connecting to a Bluetooth Classic MFi certified Zebra label printer and has a free trial for you test the module out before you purchase. It is also well documented and is fully supported by Logical Labs.

    — answered February 11th 2014 by John Pataki
