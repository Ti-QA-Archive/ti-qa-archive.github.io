Can't open controller in Alloy

I'm trying to open a controller but when I trigger the event my console says the controller file is being required but nothing happens and the app freezes.

Here's my controller:

create_comment.js

var args = arguments[0] || {};

create_comment.tss

".container" : {

}

create_comment.xml

<Alloy>
    <Window id="createCommentWindow">
        <View><Label>Testing</Label></View>
    </Window>
</Alloy>

Here's the event:

$.commentBtn.addEventListener('click', function(e){
    Alloy.createController('create_comment').getView().open();
});

Console output:

[INFO] [iphone, 7.0.3, 192.168.1.70] Requiring: file:///Users/Billy/Library/Application%20Support/iPhone%20Simulator/7.0.3/Applications/45FBF15E-7F6F-4FDD-9DEE-411A37D77198/Documents/Motivation/iphone/alloy/controllers/create_comment.js

Then the app freezes.

Building for sim iOS 7.0.3
Ti SDK 3.2.3.GA
Using TiShadow but also tried without and same problem.

    var myWindow = Alloy.createController('create_comment').getView();
    myWindow.open();
});

    Something about the stringing of the windows method to the end of the createController's method.
