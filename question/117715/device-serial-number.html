    Hi Eni

    Unfortunately, there isn't really a workable solution provided natively for Android. The id property of Titanium.Platform will return an id that was supposed to be unique, but there was a mistake made with the Android 2.2 release that causes it to return the same id for all devices. See this page for more info.
