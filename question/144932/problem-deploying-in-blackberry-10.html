  • I had the same issue as well. Did you clean the project at any point? Please confirm that there is a blackberry folder under build (in the StarterApp root folder), and that is contains files.

    Cleaning the project empties this folder and its contents. However, building does not recreate the necessary files. Simply re-download the StarterApp project, and copy the blackberry folder back to your workspace. You should be able to build and deploy to the simulator.
