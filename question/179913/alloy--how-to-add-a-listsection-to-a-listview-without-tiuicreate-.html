Alloy : How to add a ListSection to a ListView without Ti.UI.create ?

Here is what I tried but it didn't work :

index.xml

<Alloy>
    <Window class="container">
        <ListView id="listView"></ListView>
    </Window>
</Alloy>

index.js

var listSection = Alloy.createController('listSection');
$.listView.add(listSection.getView());

$.index.open();

index.tss

".container": {
    backgroundColor:"white",
    layout: "vertical"
}

"#listView": {
    top: 20
}

listSection.xml

<Alloy>
    <ListSection headerTitle="headerTitle" />
</Alloy>

Any idea ?

    You have to call $.listView.appendSection(listSection.getView()). Then it should work
