Disable parent event on child

I am using a TableView to list my data. I added a 'click' event listener to TableView but I dont want that event to be fired when clicked on a Label, added inside a TableViewRow.
How it could be done?

  • Hi

    Nice and easy one this;

    var lbl = Ti.UI.createLabel({
    text: 'Whatever you have'
    touchEnabled: false // this property - set to false
});

    touchEnabled works on most views and controls.

    • Quick follow-up;

      If you use this property on a parent view the child views stop being touchEnabled, so be careful where you add it :)

      Example;

      var parent = TI.UI.createView({
    height: TI.UI.FILL,
    touchEnabled: false,
    width: TI.UI.FILL
});
win.add(parent);
var lbl = Ti.UI.createLabel({
    text: 'Whatever you have'
});
parent.add(lbl);

      Neither parent or lbl will receive events now.

    • New information - always good :-)

      You can use this technique;

      var row = Ti.UI.createTableViewRow({
    height: Ti.UI.SIZE,
    id: 'row', // custom property
    width: Ti.UI.FILL
});
var lbl = Ti.UI.createLabel({
    id: 'lbl', // custom property
    text: 'Whatever you have'
});
row.add(lbl);

      Then in your event listener;

      tbl.addEventListener('click', function (e) {
    switch (e.rowData.id) {
    case: 'row':
        // react to row event
        break;
    case: 'lbl':
        // react to label event
        break;
    }
});

      Try this.
