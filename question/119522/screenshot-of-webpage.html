Screenshot of webpage

Hi,

I've tried a number of methods using blobs, toimage and webviews but have had no luck putting together a string of code that successfully takes a screenshot of an internet page when loaded in a window.

Has anyone had any luck and could provide an example? Would really appreciate it, thanks.

    Miles,

    To do this, you simply need to use the toImage method on the webview.

    Here is a code example:

            var win = Ti.UI.createWindow({
            backgroundColor:'#333'
        });

        web = Titanium.UI.createWebView({
            url: 'http://google.com',
            top:0,
            left:0,
            width:320,
            height:200
        });

        var imageView = Titanium.UI.createImageView({
            width:320,
            height:200,
            top:220
        });

        button = Ti.UI.createButton({
            title: 'make image',
            width:120,
            height:20,
            top:200
        });

        button.addEventListener('click', function() {
            imageView.image = web.toImage();
        });


        win.add(web);
        win.add(imageView);
        win.add(button);
