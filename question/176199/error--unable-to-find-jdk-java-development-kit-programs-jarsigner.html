[ERROR] : Unable to find JDK (Java Development Kit) programs: jarsigner

Hello

I'm new to Titanium Studio and I'm getting this error when trying to run an application on an android device.

[ERROR] :  Unable to find JDK (Java Development Kit) programs: jarsigner
[ERROR] :  Please set the JAVA_HOME is set to the JDK install location and not the JRE (Java Runtime Environment).
[ERROR] :  The JAVA_HOME must point to the JDK and not the JRE (Java Runtime Environment).
[ERROR] :  You may want to reinstall the JDK by downloading it from http://appcelerator.com/jdk.
[ERROR] Application Installer abnormal process termination. Process exit value was 1

I can run an android application on this device (Nexus 4) from eclipse + ADT without trouble but I keep on getting this message from Titanium Studio, and the app does not run.

More info:

OS: Linux Mint Debian (kernel 3.11.8)

Titanium Studio, build: 3.2.3.201404181442

java version "1.7.0_21"
OpenJDK Runtime Environment (IcedTea 2.3.9) (7u21-2.3.9-5)
OpenJDK 64-Bit Server VM (build 23.7-b01, mixed mode)

Any Idea on how to solve this?
