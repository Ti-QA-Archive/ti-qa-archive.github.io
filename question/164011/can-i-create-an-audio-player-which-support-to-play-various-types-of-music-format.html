    In titanium we have Media API which has a method called createMusicPlayer(); this lets you play the music. You can try this one.

    • By AndroidManifest.xml i mean either you specify broadcast receivers inside the <android/> node in your tiapp.xml or create a file in android folder of your application and create a file name AndroidManifest.xml and declare a broadcast receiver there like we do in android native.

      This custom file will be merged with your tiapp.xml at compile time.

      Actually Broadcast Receivers receives any messages from the system and responds accordingly if they match with the data received from the system or any other app.
