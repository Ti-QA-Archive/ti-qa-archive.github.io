Ti.Filesystem not working in 1.5.0

Hello,

As I can't make the Ti.Locale.getString() function work whatever I do I tried to re-create it with a simple xml file and a custom function.

But when I try to load locally my xml file I always get the same error:
> [ERROR] Script Error = Result of expression 'Ti.Filesystem' [undefined] is not an object. at app.js (line 6).

My code:

var file = Ti.Filesystem.getFile(Ti.Filesystem.resourcesDirectory, 'localization/en/strings.xml');

Any idea?

Thanks a lot.

    Look at the Ti.Filesystem API doc and try:

    var file = Ti.Filesystem.getFile(Ti.Filesystem.resourcesDirectory + '/localization/en/strings.xml');
