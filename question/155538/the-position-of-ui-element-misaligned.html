  • Hi

    Assuming that this;

    "#addWin":{
    layout: 'vertical',
    barColor: '#a00'
},

    Relates to the main window the text fields were placed on before you tried the view or the scrollView, then this has a layout property set.

    As your tss appears to not have any styles for #scrollView, the default would not be for the same layout property value.

    I would suggest adding at the very least;

    "#scrollView":{
    layout: 'vertical'
},

    To your tss file and see what happens.
