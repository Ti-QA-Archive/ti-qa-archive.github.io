Ubuntu 11.10

I got Ubuntu 11.10 installed under VirtualBox to test my app on latest Ubuntu. I'm using Desktop SDK 1.2.0RC4 right now. It works alright under Mandriva 2011.0. On Ubuntu, on graphics, it silently does nothing. On terminal it gives:

.../installer: error while loading shared libraries: libcurl.so.4: cannot open shared file: No such file or directory

Since Titanium Desktop Apps comes with this file under runtime/1.2.0RC4 directory, it looks like something is wrong on the installer binary because it is not "seeing" the file there, looks like it just looking for installed one. Any way it's odd because a recent installed ubuntu comes with libcurl3 package installed, so the file is there on /usr/lib/i386-linux-gnu/.
