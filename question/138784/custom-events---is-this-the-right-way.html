Custom Events - Is this the right way?

I am trying to write a custom event which will set the value of a variable in page2.js, to the length of an array which has been constructed and populated in page1.js.

So far, I have not been successful, but here is what I have been trying:

//page2.js:
var list_length = Ti.App.fireEvent('getLength'});
alert(list_length);

and in page1.js:

// var array is set and populated with 5 integers
Ti.App.addEventListener('getLength', function(){ 
    return array.length;
});

I think that I am not using the custom event correctly?

    One thing you can do is pass the variable from one window to another window by making custom variable 

    button.addEventListener('click', function() {
    win.containingTab.open(Ti.UI.createWindow({
        title:"Search by Code",
        customarray:list_length,// made custom variable of array
        url:"page2.js",
        backgroundColor: 'white'
    }));
});

    On page2.js you can retrieve it using :

    window =Ti.UI.currentWindow;
custarray = window.customarray;
var len = custarray.length; // this will retrive length.
