Installing CLI. following docs does not work

I am trying to install CLI as posted here :http://docs.appcelerator.com/titanium/latest/#!/guide/Titanium_Command-Line_Interface_Reference.

following the CLI Quick Start, i have installed node and the titanium CLI 'successfully'.
but after installing the CLI, i run 'titanium login', and it says 'Unknown command: login'.
the only commands available to me are create/run/clean/emulator/help.

Please tell me what I am doing wrong. thanks

    You probably have an alias defined in your bash profile to the old titanium.py CLI. You need to edit your .bash_profile (or .bashrc) in your home directory and removed any aliases for "titanium".
