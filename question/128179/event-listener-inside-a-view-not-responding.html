Event Listener inside a view not responding

Hi

I am having trouble getting an event listener to work in my app for the iPhone.

I have a window which includes a createdView. Within this created view there is a button go_back_btn that has an event listener on it when clicked. The button doesn't respond at all though and i am struggling to figure out why.

Here is the code below:

Window:

// Create FilmStrip to Hold Individual Views
appFilmStrip = S.ui.createFilmStripView({
  height:44, 
  top:0, 
  left:0, 
  views: [S.ui.createFormatView()]
});

// Create FilmStrip Container
  appFilmStripContainer = Ti.UI.createView({
    width:appFilmStrip.views.length, 
    height:44, 
    top:42, 
    left:0
});

View (which includes the EventListener:

(function(){
  S.ui.createFormatView = function(){
    var view = Ti.UI.createView({width:320, height:44}),

    // Create Go Back Button and Define Clickable Function
    go_back_btn = Ti.UI.createButton({
      width:160, 
      height:44, 
      top:378, 
      left:0, 
      backgroundImage:'images/go_back_btn.png'
    }); 

    go_back_btn.addEventListener('click', function() {
      // THIS IS THE EVENT LISTENER THAT IS NOT RESPONDING??
    });

  view.add(go_back_btn);
  return view;
  };
})();

Thank you in advance

    Please ignore the above now, there was an error in the code that i didn't notice at time of posting, all fixed now - the top value of the button exceeded the height in the FilmStrip view
