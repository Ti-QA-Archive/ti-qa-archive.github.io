    Hello Arif Hossen,

    This may happen, if your App has a map and you do not include this to info.list file or tiapp.xml file. You can try this code inside the <plist></plist> tag in tiapp.xml file. 

    <key>CFBundleDocumentTypes</key>
<array>
    <dict>
        <!--Remove both of these key/value pairs -->
        <key>CFBundleTypeName</key>
        <string>MKDirectionsRequest</string>
        <key>LSItemContentTypes</key>
        <array>
            <string>com.apple.maps.directionsrequest</string>
        </array>
    </dict>
</array>

    If you can submit your app to the iTunes using this trick then, let me know.

    Check this link also.

  • Finally i solved my problem .

    I have put below code into tiapp.xml

    <key>CFBundleDocumentTypes</key>
                <array>
                    <dict>
                        <key>CFBundleTypeName</key>
                        <string>MKDirectionsRequest</string>
                        <key>LSItemContentTypes</key>
                        <array>
                            <string>com.apple.maps.directionsrequest</string>
                        </array>
                    </dict>
                    <dict>
                        <key>CFBundleTypeIconFiles</key>
                        <array/>
                        <key>CFBundleTypeName</key>
                        <string>MKDirectionsRequest</string>
                        <key>LSItemContentTypes</key>
                        <array>
                            <string>com.apple.maps.directionsrequest</string>
                        </array>
                    </dict>
                </array>
                <key>MKDirectionsApplicationSupportedModes</key>
                <array>
                    <string>MKDirectionsModePedestrian</string>
                </array>
