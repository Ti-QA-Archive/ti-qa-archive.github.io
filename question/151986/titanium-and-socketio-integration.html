Titanium and Socket.io Integration

Application type: mobile
Titanium SDK: 3.1.0
Platform & version: Android 2.2 and on, iOS 4.x and on
Device: iOS simulator, Android emulator and Physical Devices
Host Operating System: OSX 10.8 (Mountain Lion)

I was wondering if anyone has dealt with integrating Socket.io into a Titanium project. There have been two projects on GitHub that appear to have integrated Socket.io into previous versions of Titanium but as it appears (from the GitHub issue list), they are outdated and will not work with 3.1.0. I was wondering if anybody had experience with implementing a Socket.io protocol on top of Titanium network sockets as this is what we are trying to accomplish at this time. Thank you very much.

  • In order to do proper socket.io with Titanium, you will need to use the TiWS native module. With it, you will be able to use socket.io at the other end ('I assume it's Node.js, but it could be just anything).

    Regular JavaScript libraries won't work as-is with Titanium.

    Hope this helps.
