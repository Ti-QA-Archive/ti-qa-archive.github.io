How can I get the parent objct of another object?

Hi, how it is possible to point to the parent object from its child object? For example, I have a button inside a window:

var winStart = Ti.UI.createWindow({
    ... some properties...
});

var btnStart = Ti.UI.createButton({
    ... some propetries...
});

btnStart.addEventListener('click', function(){
    alert('my parent is: ' + WHAT I HAVE TO WRITE HERE?);
});

winStart.add(btnStart);
winStart.open();

Thanks for the attention.

    You can access your parent by setting a pointer in the child:

    var winStart = Ti.UI.createWindow({
    ... some properties...
});

var btnStart = Ti.UI.createButton({
    parent: winStart
});

btnStart.addEventListener('click', function(){
    alert('my parent is: ' + this.parent);
});

winStart.add(btnStart);
winStart.open();
