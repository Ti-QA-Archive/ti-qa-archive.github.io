DARK and BIG ActivityIndicator

Is there a way to set the ActivityIndicator style to Ti.UI.iPhone.ActivityIndicatorStyle.DARK and Ti.UI.iPhone.ActivityIndicatorStyle.BIG?

If I use BIG the color defaults to PLAIN. I'm showing this over white background content so doesn't work and users are not noticing the smaller spinner.

  • Probably the best way around this would be to create your own loading indicator. You can use an imageView's animation with an array of images to accomplish this. Then you can use imageView.show() and imageView.hide() the same way you show and hide the activity indicator. Hope that helps!

  • As a workaround you can put an indicator inside a view and scale the view. The problem is that the indicator will loose definition.

        var loadingView = Ti.UI.createView({width:50,height:50,transform:Ti.UI.create2DMatrix().scale(1.5,1.5)});
    var loadingIndicator = Ti.UI.createActivityIndicator({
        style: Titanium.UI.iPhone.ActivityIndicatorStyle.DARK,
    });
    loadingView.add(loadingIndicator);
    win.add(loadingView);
