    • Matt … I actually think you may be missing something.

      The directory with the permissions error is :

      OSError: [Errno 13] Permission denied: '/Developer/Platforms/iPhoneOS.platform/Developer/Library/Xcode/File Templates/Cocoa Touch Class/Objective-C class/NSObject subclass.pbfiletemplate/class.h'

      I looked in the install.py file that is causing this error, the code snippet is below.

      xcodedir = os.path.join("/Developer","Platforms","iPhoneOS.platform","Developer","Library","Xcode")
file_dir = os.path.join(xcodedir,"File Templates","Appcelerator")


if os.path.exists(file_dir):
    shutil.rmtree(file_dir)

if not os.path.exists(file_dir):
    os.makedirs(file_dir)

shutil.copytree(os.path.join(template_dir,'Titanium'), os.path.join(file_dir,'Appcelerator'))

      Why would this python script be deleting the directory tree of the STANDARD templates? before adding the titanium templates.

      I would propose that the script needs to be modified to append the titanium specific directories prior to removing.
