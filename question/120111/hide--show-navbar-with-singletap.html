Hide / Show NavBar with singletap

Hi,

I've got a hidden NavBar and when the user taps once on the screen the NavBar shows, but how can I hide it again if the user taps on the screen again??

Is that possible?

  • Accepted Answer

    Set a flag somewhere in the app. Something like:

    var nav_hidden = false;

your_window.addEventListener('click', function() {
    window.navBarHidden =  !nav_hidden;
    nav_hidden = !nav_hidden;
});
