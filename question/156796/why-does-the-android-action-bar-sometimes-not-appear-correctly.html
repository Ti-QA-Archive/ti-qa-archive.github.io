Why Does the Android Action Bar Sometimes Not Appear Correctly?

Using Alloy, I am trying to make a cross-platform iOS/Android app. I have many windows, all of which have a configured action bar.

Controller function bound to window's open event

function init() {
  if ( OS_ANDROID ) {
    var activity = $.dateRangeSelect.getActivity(),
        actionBar = activity && activity.getActionBar();

    if ( activity ) { 
      activity.invalidateOptionsMenu();
    }

    if ( Alloy.Globals.supportsActionBar && actionBar ) {
      actionBar.setDisplayHomeAsUp(true);
      actionBar.setOnHomeIconItemSelected(close);
    }
  }
}

This works most of the time, but occasionally, the action bar looks like this:

If you click on the bottom half of the action bar, everything works fine. If you click on the top half, the app crashes.

What could cause this?
