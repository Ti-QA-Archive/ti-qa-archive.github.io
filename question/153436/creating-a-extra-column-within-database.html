Creating a extra column within database

Hi All

I have hit a bit of a issues i have released my app which has the following database setup.

var db = Ti.Database.open('base');
    db.execute('CREATE TABLE IF NOT EXISTS docs (name TEXT, date TEXT, duration TEXT, uploaded TEXT, actions TEXT)');

Which i have released but i now have some new ideas i wish to add which means adding extra columns to the database, but users will already have files saved within this database so i can just delete it and create a new one which i would love to do.

Is there a way i can add two extra columns (format, lang) so how so my database would be as follows.

var db = Ti.Database.open('base');
    db.execute('CREATE TABLE IF NOT EXISTS docs (name TEXT, date TEXT, duration TEXT, uploaded TEXT, actions TEXT, format TEXT, lang TEXT)');

Is their a UPDATE table or add column how do people deal with this?

Any help would be great

Thanks

    • OR

      You can query the system catalog for the tablename and its DDL statement.

      select sql from sqlite_master where type='table' and name='<tablename>'

      then look at the value of the column sql and find the column name there or not

    • Ok not tested fully yet but this seems to work

              var format = db.execute("SELECT * FROM sqlite_master where sql like('%format%')");
    if (!format.isValidRow()){
        db.execute("ALTER TABLE docs ADD format varchar(244)");
    }
    format.close();
    var guid = db.execute("SELECT * FROM sqlite_master where sql like('%guid%')");
    if (!guid.isValidRow()){
        db.execute("ALTER TABLE docs ADD guid varchar(244)");
    }
    guid.close();
