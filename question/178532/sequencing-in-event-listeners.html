Sequencing in Event Listeners

hey all,
I was wondering how i could make 3 things happen in a specific order when a button is pressed. For example, if i wanted to first display an alert than a sound of cow followed by a sound of a dog. When i put all 3 in the event listener, the two sounds play at the same time.
note: I'm developing on android.
Ahmad

    If you are using Ti.UI.Sound the checkout the complete event. If you are using the Ti.Media.AudioPlayer the use its complete event. In either case, listen on the event before playing the next sound
