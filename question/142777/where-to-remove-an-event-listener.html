Where to remove an event listener?

I have a master/detail application implemented with a navGroup. In my master view I added the below code to simulate back button behavior from the detail window (the back button is then shown in the detailContainerWindow). To avoid memory leaks I want to remove the event listener each time the button has been clicked (i.e. user returns to master view). Is it possible to remove the event listener in the way I do it (inside of the event listener)? If not, what is usually a suitable place to remove it?

// simulate back-button behavior            
var closeWindow = function (){
    navGroup.close(detailContainerWindow, {animated: true});
    custombackbutton.removeEventListener('click', closeWindow);        
}        
//event listener for backbutton click
custombackbutton.addEventListener('click', closeWindow);

    As long ast the custombackbutton was added to the window, it will all get automatically cleaned up in memory when the window closes.

    The only time you need to worry about manually removing event listeners is if you have a Ti.App level event listener. Then you would remove those similar to your proposed code here… You would remove them prior to the navGroup.close() call in your closeWindow method.
