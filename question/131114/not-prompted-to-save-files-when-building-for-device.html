Not prompted to save files when building for device

This is driving me crazy. When I build and run my app to run in the simulator, Titanium Studio prompts me to save any unsaved files before building. When I build my app for a device, I get no such prompt before the build occurs.
I often have to build twice because after building the first time and installing on my device and running my app, I realize that none of the changes got into the build because I forgot to save!
Is there a setting in Titanium Studio that I am missing?
