Moible web app in ios APP / Offline browsing/Mobile web with No server

Is it possible to create an offline version of a mobile web app, no need of a web server? The idea, is to create a webview in a totally different titanium app, for example a portfolio app of web mobile apps with no need of a internet connection.

  • You can do this in a couple of ways. First, if you are interacting with anything on the server side for the live app, ie: a database, then you could always keep a copy of the latest version of the database on the phone and update it when online.
    Second, if you're not interacting with anything on the server, then you can just save the web pages to the phone and point at them in your webview code.
