Positioning a button next to a text field

I have a fixed width graphical button with my text inside (75px X 30px). I want to position the button and text box as follows:

<---------------------------------- X pixles -------------------------------->
|                                                                                               |
|                       N px                           5px         75px      5px       |
||----------------------------------------|  <------->  <------>  <------->   |
||          Textbox                          |                |button|  30px       |           
||----------------------------------------|                        |
|                                                |
|                                                |

The graphic did not come out right but basically I want the button to end 5px from the right side of the window. I set the button's right property to '5px'. Now I want the text box to end 5px from where the button starts.

If the window is X px the button should start at (X-80px) (75px size + 5px space from end). The button should end at (X-5px). The text box should start at 0px and end at (X-85px). I do not know the size of the window, or if I did set it it would be in terms of "100%" for the width and height. I don't think 100%-85px would work

  • If by window you are referring to the size of the screen. You can get it by doing Ti.Platform.displayCaps.platformWidth; or Ti.Platform.displayCaps.platformHeight; then if you still need to use percentage value use this formula 

    var pix = parseInt('100%') * Ti.Platform.displayCaps.platformWidth / 100;// this will give you the pixel value

    hope this helps
