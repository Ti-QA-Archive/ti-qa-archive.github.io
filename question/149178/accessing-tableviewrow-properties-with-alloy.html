Accessing TableViewRow Properties with Alloy

I'm trying to access the properties of a tableviewrow after a click on tableview (using alloy). I thought I'd be able to just do something like this - 

$.someTableView.addEventListener('click', function(e){
    console.log(e.row);
});

but the row object comes back as a string, not an actual reference to the table view row object? How can I make this a real pointer, not a string?

Thanks.

  • Accepted Answer

    I don't think e.row is coming back as a string. It is just getting cast to a string when you put it inside the console.log call. You should be able to use e.row just like a row and we have lots of examples in the alloy test apps that do exactly that.
