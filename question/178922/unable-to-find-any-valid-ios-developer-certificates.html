  • Accepted Answer

    It seems that Apple changed developers certificates names.
    Old names were "iPhone Developer ….". Now it's iOS Development …".

    Current Titanium CLI can't match it : https://github.com/appcelerator/node-appc/blob/master/lib/ios.js#L151
