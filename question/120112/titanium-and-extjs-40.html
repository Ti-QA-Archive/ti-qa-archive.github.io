  • Hi Steve.

    I think you can't use Sencha framework in Titanium framework because Titanium JavaScript environment is different from browser environment (there are no DOM elements and methods in Titanium).

    Maybe you can use Sencha in web view, but then you would not have "native" advantage (your application would be web application).
