  • It seems like something might be messed up with your android environment. Have you attempted to run the emulator by itself? Do this:

    1) Run the AVD Manager (ANDROID_SDK_PATH/tools/android)

    2) Find the name of the emulator you want to test, in my case titanium_9_HVGA

    3) Go to the command line and run ANDROID_SDK_PATH&#x2F;tools&#x2F;emulator -avd NAME_OF_YOUR_EMULATOR_FROM_STEP_2

    If the emulator doesn't start correctly, delete the emulator from the AVD Manager and try to build from Titanium Studio again. If that still doesn't work, chances are you have a problem with your Android installation. You'll likely need to reinstall it.
