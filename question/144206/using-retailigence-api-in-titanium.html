    Just by looking at the Retailigence developer site, it looks like the API should be able to provide you with the information you want, however, the best way to know for sure is to simply register for an API key and find out exactly what you can do with the free version of the API.

    From what it looks like, the information is more than likely behind a RESTful API, so you can easily integrate into Titanium with basic xhr calls out to the endpoints you need.
