  • Here is what I usually do :

    1- install java sdk to a path with no spaces
    2- install Android to a path with no spaces
    3- add the jdk bin folder to the path after removing older jdk entries
    4- add JAVA_HOME environment variable
    5- I go to the control panel -> java and update the settings to find and use the new jdk
    6- RESTART !

    this worked every time.

    I see that %JAVA_HOME%\bin is added twice, you can remove the last one. Just take note of the value of %JAVA_HOME%
