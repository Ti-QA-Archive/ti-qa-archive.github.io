    Hi

    Titanium is only looking for modules during this process so it knows where they are located if you app references one so it can include it in the project.

    Your app could ask for the Facebook app and specify that it uses the very latest version whatever that is, as titanium has located all the modules it can then include the latest version automatically.

    If you had specified a specific version then it would not need to work out which to choice. It will still do the check at the start though as this is done before asking which specific ones are required.

    Only those modules you specify are included in your project.
