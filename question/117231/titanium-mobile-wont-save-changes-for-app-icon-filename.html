Titanium Mobile won't "save changes" for app icon filename?

I'm trying to change the app icon file name on my project in the "Edit" tab of Titanium mobile. Each time I click "Save Changes" on the project details, click on another project, then come back to my original, the app icon file name is set once again to "app_icon_logo.png".

The only way I have managed to get my custom app icon to show up is to just rename it to "app_icon_logo.png"

I've tried dumping my build folder several times with no change. :-(

    Do you self-edit the name or actual click the Folder icon and scroll to your icon and select it? Then when it's in the textbox, click "Save Changes" and then click "Edit" and it should just change to the name of the image.
