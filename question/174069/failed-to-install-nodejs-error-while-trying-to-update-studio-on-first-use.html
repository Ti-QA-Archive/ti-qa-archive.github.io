  • I had a similar problem. For some reason Titanium Studio 3.3 will NOT install Node.js on Mac OSX (Mavericks) very well either! I have a solution. You need to install Node.js first, and other dependencies, then uninstall / re-install Titanium_Studio.dmg again. This worked for me! Follow below:

    Here are the exact steps to get Titanium Studio working properly. Go to http://nodejs.org/ and download/install the pkg for v0.10.31 and this will install node and npm (the package manager) when done, from your command line type 

    node -v

    to see that 0.10.31 version is install correctly.

    Now you need to install Alloy (MVC) framework for Titanium and the Titanium CLI (Command Line Interface):

    > sudo npm install -g alloy
> sudo npm install titanium -g

    Now you need to login to Titanium to fix your SDK paths (Hint, it's your Appcelerator.com account):

    > titanium login

Username: your@email.com
Password: ************

    > titanium setup

    8) for iOS SDK (i chose option 1 and then option 1 again) it had my iOS cert already listed (since I have installed Xcode 5.1.1 already)
    9) update your Android SDK path, my path is here, replace "jimsemple" with your own username
    /Users/jimsemple/Library/android-sdk-macosx

    and I AM NOT using NDK at all, so the next one you can hit [Enter] for "none"

    After you have setup your iOS SDK path, and Android SDK path, you'll need to re-install Titanium Studio 3.3 drag in to your Applications. This time instead of 4 options to download, it'll ask you to install Node ACS (for storing Node.js apps in the cloud) just Install and you can goto "Get Started" tab again, select iOS SDK and click "UPDATE" button on the right, it should have a green check in the box now. Click "Android SDK" do the same click "Update Android SDK" button on the right. The box should be a green checkmark.

    I wanted to share and make it easier for you and others. I was working on this for a few days, its always hard to get development tools setup the right way, but you can breath a sign of relief now, and start coding :-)
