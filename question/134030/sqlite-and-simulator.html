    wait wait i got the point…
    you use 

    var db = Ti.Database.open('hcmc.sqlite','etab');  
var rows = db.execute('SELECT nom_etab FROM etab');

    but when you use open method of the Database you should only use its alias name..

    so try it first..

    Ti.Database.install('hcmc.sqlite','etab');  
var db = Ti.Database.open('etab');  
var rows = db.execute('SELECT nom_etab FROM etab');
