Use of variable indirection with Javascript

How to use indirection for a variable i.e.

var myVar='This is my Var';
var indVar = 'myVar';
alerte('My var's value is '+eval(indVar ));

should display : "This is my Var"

but eval fails with iphone emulator

    Christian, first there is a problem with your alert string construction:
    alerte should be alert
    'My var's …
    should be
    'My var\'s… (you are using single quote in the string delimiters)
    then, why would you "eval" a string to use it as a string?

    alert('My var\'s value is '+indVar);

    Also should work:

    var indVar = "{myString:'my Var'}";
indVar = eval(indVar);

var result = indVar.myString;
