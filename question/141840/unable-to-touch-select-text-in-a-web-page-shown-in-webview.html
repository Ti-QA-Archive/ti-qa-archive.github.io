Unable to touch select text in a web page shown in WebView

I am developing an app for Android.
My app will open a WebView like this: 

var openGoogle = Ti.UI.createButton({
    top: 120,
    title: 'Open Google'
});
openGoogle.addEventListener('click', function(){
    var webview3 = Titanium.UI.createWebView({
        touchEnabled:true,
        url:'http://google.com'
    });
    var window = Titanium.UI.createWindow();
    window.add(webview3); 
    window.open({modal:true});
});
Ti.UI.currentWindow.add(openGoogle);

I can successfully open google page in the Webview and able to click on the hyperlink inside.

But the problem is I can't touch and highlight the text inside the page.
How come?
