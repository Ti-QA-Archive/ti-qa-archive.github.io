Jasmine and alloy

Hi,

it's not really a question but I just wanted to start a discussion on how to integrate Jasmine and alloy in the best way.

Guide to install Jasmine

I placed the tests into the spec folder in the root of my appcelerator app.

Here' a little shell script to do that:
e.g. bin/tests.sh

#!/bin/sh

local=""

if [[ $0 == '/'* ]]; then
  local="`dirname $0`"
else
  local="`pwd`"/"`dirname $0`"
fi

export NODE_PATH=$NODE_PATH:$local/../app/lib:$local/../app/controllers/
jasmine-node spec

To run it just enter e.g. bin&#x2F;tests.sh in your terminal.

You should see something like that in the output:

.

Finished in 0.013 seconds
1 test, 1 assertion, 0 failures
