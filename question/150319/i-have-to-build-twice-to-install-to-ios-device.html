I have to build twice to install to iOS device

Hi,

I'm working on my project using the latest version of Titanium Studio and SDKs, on Mac mini with OS X 10.8 Mountain Lion. Other apps such as Xcode and iTunes are up to date.

I noticed that I have to do "Install to iOS Device" twice, in order to actually transfer the newly built app to my iPhone. If I do it only once, behavior of the app on iPhone does not change, even though I have changed the code and the transfer process has been successfully completed.

I can see no difference among processes each time. It always succeeds and consistently produces the same result (no change in the first time). Does anyone have the same problem or know what is causing this? Thank you very much.

  • Try cleaning the project, then build for device a couple times and see if that fixed the problem. You may also want to manually remove the app from the device and make sure it's not present in iTunes. Lastly, make sure you have Titanium 3.0.2 installed and it's the selected <sdk-version> in the tiapp.xml.
