Facebook (native) permissions on android - reauthorize not available

From my understanding Facebook doesn't like you to request read and write permissions at the same time so on ios you use fb.reauthorize to overcome this which I have working well.

When it comes to android though fb.reauthorize isn't available so I'm a little confused as to what the correct process is to request the necessary read and write permissions for my app.

Am I just misunderstanding and on Android you can just request all the permissions at once? i.e

fb.permissions = ['read_stream','email','publish_actions'];
