Error - "Requires version >=3.2, currently 3.1.2"

Hi,
I have just installed the latest version of Titanium SDK 3.2.0. I am using Mac OS X version 10.8.3

I have created a Test Alloy project and trying to run it using version 3.2.0 but I am getting the error - 

Command "create" incompatible with this version of the Titanium CLI
[WARN] :   Requires version >=3.2, currently 3.1.2

I hope some one can help me overcome this problem and I can start using the new SDK.

Thanks a lot in advance.

    Hello Bably,

    In Ti SDK 3.2.0 minimum required CLI verston is 3.2.

    Your CLI version is 3.1.2. You can update your CLI version 3.1.2 to 3.2.0.

    type following command in titanium for update CLI to latest version.

    npm install titanium -g

    also check this Titanium_Command-Line_Interface_Reference for more information about CLI.

    i hope this helps. thanks.

    • Hi Sanat,

      Thank you so much for your help. After days of trying to install 3.2.0 I have finally been able to install it with your help and the links you provided. I was able to install 3.2.0 SDK of Titanium by -

      1. I ran the uninstall command : sudo npm remove -g titanium
        from the command line within Titanium Studio (Window > Show View > Terminal Editor). It did not work (for me) when I ran the uninstall commands from the Mac terminal window.
      1. I was then prompted to install the latest Node.ACS, Alloy, and Titanium CLI. I went ahead and install it from Titanium Studio. Then, closed Titanium Studio and opened it up again. Then created a sample project using 3.2.0 SDK and it started working.
