  • Hi Vincent,

    You'll need to go to System Properties > Advanced System Settings > Advanced > Environment Variables

    Then add these below (these are placeholders and you will need to use your own paths)

    ANDROID_SDK
    C:\Program Files (x86)\Android\android-sdk

    JAVA_HOME
    C:\Program Files (x86)\Java\jdk1.6.0_29

    PATH
    %JAVA_HOME%\bin; %ANDROID_SDK%/tools; %ANDROID_SDK%/platform

    Note: Java JDK 1.7 does not work. Java JDK must be 32bit version.http://developer.appcelerator.com/question/146919/win8-issues-with-emulator-will-not-work#
