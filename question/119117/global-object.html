Global Object

Is it possible to create an instance of an object like this:

var myObject = new FooBar();

in app.js and be able to use this in child windows? I know I could store data via Titanium.App.Properties, but you can't store methods (functions) in that way.

Any solution?

  • Objects stored on the Ti.App object (or any in the Ti.* namespace) are automatically recalled in every new context. Ti.App.foo = new Foo() should do the trick.
