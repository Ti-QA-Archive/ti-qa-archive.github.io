Adding Caption for instagram

I tried below code to open instagram with caption. But it gives error unrecognized selector sent to instance 0x100451d0

              var filename = Titanium.Filesystem.applicationDataDirectory + 'sample/' + "img.ig";

            var docviewer = Ti.UI.iOS.createDocumentViewer({url: filename});

            var annotationObj = new Object();
            annotationObj.InstagramCaption = "Caption sample";


            Ti.API.info("annotationObj   " + JSON.stringify(annotationObj));

            docviewer.UTI = "com.instagram.exclusivegram";
            docviewer.annotation = annotationObj;

            docviewer.show();

It works fine if I remove annotation //docviewer.annotation = annotationObj;

Anyone able to post caption to instagram?
