Text Field in Table View loses focus

Originally my window had text boxes on it. When I clicked on a text box the keyboard came up and any text typed went into the text field as expected. I added a table view to the window and created custom table row views for each text field. Now when I click the text field the keyboard still comes up, but it loses focus so nothing typed on the keyboard goes into the text fields. If I tap back into the text field a second time all is fine, but that is pretty bad for the user.

I have been playing with settings on the tableView and tableRowView such as allowsSelection, focusable and editable but nothing I can come up with seems to solve the problem. How can I keep focus on my text fields when they a within a table view?

Currently only working on Android. Any and all help greatly appreciated.

Thanks!

  • Try insert a View inside the Window and the TableView inside the View. It works for me.

    var self = new Window();

var viewContainer = new View({
    top : '10dp',
    bottom : '10dp'
});
self.add(viewContainer);

var tableView = new TableView();

viewContainer.add(tableView);
