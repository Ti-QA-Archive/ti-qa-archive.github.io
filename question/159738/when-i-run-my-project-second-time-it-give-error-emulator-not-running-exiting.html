  • The correct solution is actually posted on your other question, though the answer there didn't say how to do so.

    At a command prompt window, enter adb kill-server Then, enter adb devices to restart ADB. You don't need to, and don't want to close the emulator when you do this. If that fails to work, use the Control Panel to kill the adb.exe service, then restart it with adb devices.
