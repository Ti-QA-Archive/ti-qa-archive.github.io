  • Accepted Answer

    Alloy + commonJS.

    Questions and code using url: just get marked -1 from me. No one should be using that crap any more.
    Ti.Include is a bad idea in that it simply 'includes' the contents of the file directly in place, forcing the js engine to load all that code at one time.
    Allow and commonJS allows you to separate out your concerns, only loading them into memory as needed.
