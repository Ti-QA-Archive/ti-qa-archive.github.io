  • Hello, Manish, Try to follow these steps.

    • Remove the nodejs and upgrade nodejs manually to version 0.10.
    • Check If you have JDK 1.7 installed, which is not supported by Titanium. Please install JDK 6 (aka 1.6).
    • Go to this LINK to learn how to install android sdk.

    Than i restart Titanium IDE and Check for updates by "help/Check for Titanium Updates" and "help/Check for updates". Update.

    Now if all of the above is done, still no fix. then i suggest you install the latest Titanium SDK 3.3.0.GA again if you already have.

    Download the sdk zip file ..and from the studio go"help/install titanium sdks/ browse the file"

    Restart studio and open again,Hope it helps. if it does please mark as best answer. Thanks.
