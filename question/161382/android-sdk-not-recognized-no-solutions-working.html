  • Solved:
    I had the same problem with Titanium said "Android SDK not found". I tried everything uninstalling reinstalling, downgrading both java and titanium, nothing worked.
    Finally I found the SOLUTION:

    • When installing Android SDK install it to the predefined path: (C:\program files(x86)\Android\Android-sdk). Instead of the path Titanium suggests (C:\android-sdk-win).
    • When configure SDKs in Titanium browse to the path you installed the SDK into.

    My OS is Windows 8.1

    Good luck!
