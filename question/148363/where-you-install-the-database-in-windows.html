  • Hi

    To copy an existing database into your app you copy your local database into your resources folder, you can locate the file anywhere in the resources folder or in any nested folders underneath the resources folder, examples;

    • /{appname}/resources/yourdatabase.sql
    • /{appname}/resources/database/yourdatabase.sql

    You can the find the physical location of the database you have installed;

    var db = Ti.Database.install('/database/yourdatabase.sql', 'yourdb');
Ti.API.info('Location: ' + db.file.getNativePath());

    To obtain the physical location of the database you have opened after it has already been installed;

    var db = Ti.Database.open('yourdb');
Ti.API.info('Location: ' + db.file.getNativePath());
