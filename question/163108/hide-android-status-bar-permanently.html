Hide Android status bar permanently

I want to hide the status bar permanently. i have several windows in one app, and i removed status bar on all windows, but on one window i open facebook page so when facebook open it that windows, status bar also goes appear with facebook page, but when i close facebook the status bar does not go disappear. I want to hide status bar permanently in android os.

  • In tiaap.xml find android tag , must be like :

    <android xmlns:android="http://schemas.android.com/apk/res/android" />

    Replace it with following :

    <android xmlns:android="http://schemas.android.com/apk/res/android" >
    <manifest>
        <application android:theme="@android:style/Theme.NoTitleBar.Fullscreen"/>
    </manifest>
</android>
