Apple Instruments don't log memory Allocations on simulator

I'm trying to find out why my app crashes on a real device often and I'm using developer Instruments for that, as mentioned here: https://wiki.appcelerator.org/display/guides/Managing+Memory+and+Finding+Leaks

My problem is that I'm not seeing any memory allocations while I'm recording. The Memory Leaks tool gets data, so the process of my app is linked correctly I guess.

Does anyone know how to get this to work? Thank you.

PS: in the guide it says to select "Allocations" but in my (latest) version of the tool it only has "ObjectAlloc" which I guess is the same thing?

    What version of Instruments are you using? I have 4.6.0 installed as part of Xcode 4.6.1. I see the following screen on launch. It doesn't match what you have?

    With the simulator launched, and an app open (Kitchen Sink in my case), from the Processes menu, choose your app – not the Simulator process. Then click the Record button.

    Typically, you have to interact with your app for Instruments to begin displaying data. Click a button or something and memory objects should show up.
