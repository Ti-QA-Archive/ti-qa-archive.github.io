  • Not exactly like the phone app does it. It probably uses a UIMenuController attached to a disabled textfield.

    Titanium doesn't yet support the UIMenuController. There is a issue for it on the issue tracker.

    A solution would be to disable the textfield, bind an click event listener, and when it is clicked, bring up an OptionDialog with a button named Paste, that has an click event listener that takes the number from the clipboard and sets it on the textarea.
