So basically what I wanted to happen was that when the user clicks any of the animation buttons, the background color of the window should animate from its current color to the new color specified. (For example, if the current color was red, and the user clicks on the "Night" button, then the background color should switch from red to purple.) However, what is happening is when I click any of the buttons, the color resets to white, and THEN goes to the color that the button specifies. (e.g. I click the "Night" button, and the color of the background becomes white, then animates to purple.) Any way to fix this?
