Keyboard Toolbar - Buttons Go Missing

Hi All,

I am having some issues with buttons on keyboard toolbars.

I have a window with 6 text fields on it.

Each textfield has a keyboardToolbar that has two buttons on it.

All works well, until I start to tap through the fields. After a few taps of the fields the buttons disappear from the toolbar as it appears for each field.

This is reproducible rather easily.

Anyone else seen this behavior?

Nick

  • Nick, Just in case you're following or still working on this … I found a kludgy workaround. I added 'focus' listeners for my text fields and, when invoked, I simply reapply the keyboard buttons array to the keyboardButtons property. This kept them visible. I'm sure this is a craptacular way to do it but it works!

    // naturally, all the code below is abbreviated significantly :)
var btn1 = Ti.UI.createButton({});
var btn2 = Ti.UI.createButton({});
var keyboard_btns = [btn1,btn2];
var username_field = Ti.UI.createTextField({
  keyboardToolbar: keyboard_btns
});
var next_field = Ti.UI.createTextField({
  keyboardToolbar: keyboard_btns
});
username_field.addEventListener('return', function(e){
  next_field.focus();
});
username_field.addEventListener('focus', function(e){
  username_field.keyboardToolbar = keyboard_btns;
});
