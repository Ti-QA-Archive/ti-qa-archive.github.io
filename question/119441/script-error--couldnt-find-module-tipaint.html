  • Hi,
    in my case, the solution was simple. I think, you dont install the module right:

    For an iOS Module:

    - Navigate in Git repo to module_name/mobile/platform, for example, paypal/mobile/ios
- Click on the zip file that you'll find in the list of files, for example, ti.paypal-iphone-1.6.2.zip
- Right-click View Raw, choose Copy Link Address (or your browser's option for that)
- In Studio, choose Help, Install Mobile Module. Paste the URL you copied into the 'Install From URL' box and click OK
- Proceed with using the module, as described in the docs page linked to above.

    For an Android Module:

    - Navigate in git repo to module_name/mobile/platform/dist, for example, paypal/mobile/android/dist
- Click on the zip file that you'll find in the list of files, for example, ti.paypal-android-2.2.2.zip
- Right-click View Raw, choose Copy Link Address (or your browser's option for that)
- In Studio, choose Help, Install Mobile Module. Paste the URL you copied into the 'Install From URL' box and click OK
- Proceed with using the module, as described in the docs page linked to above.

  • I have fixed the problem:

    I opened the paint.xcodeproj with xcode. After building it comes errors because TiCore.h files not found. The solution:

    => Changing the Titanium_SDK_Version from 1.6.0 to 1.6.2
    => moving the lib.ti.pant.a to &#x2F;Library&#x2F;Application Support&#x2F;Titanium&#x2F;modules&#x2F;iphone&#x2F;paint&#x2F;1.0&#x2F;
    => it runs!
