Scrollable View Paging Control

I have a scrollable view with the paging controls on the bottom. How do I make it so tapping on either side of the page controls scrolls the view? The same way you can click the paging controls on the iphone spring board

    Ronnie

    Create two transparent views, covering either side of the screen, and listen for doubletap (or whatever is your preference) events that run the scrollToView() method of Ti.UI.ScrollableView.

    • ok well this seems to work

      scrollLeft.addEventListener('click',function()
{
    if(scrollView.currentPage != 0)
    {
        var cur = scrollView.currentPage;
        scrollView.scrollToView(cur - 1);
    }
});
scrollRight.addEventListener('click',function()
{
    if(scrollView.currentPage != (categories.length - 1))
    {
        var cur = scrollView.currentPage;
        scrollView.scrollToView(cur + 1);
    }
});
