Image pinch zoom in android

hi, I am using titanium to develop an application. I am trying to zoom the image. I am able to zoom the image well but the issue exists when the image is zooming in. I dont have any problem with zoom out. I specified some properties to an image by default and added the image to a view. I am able to zoom the image to larger size but while zoom in the image it should not get resized less than the default width and height. Below is my code…

var win = Ti.UI.createWindow({

backgroundColor : 'black',

title : 'Tab',

orientationModes : [Ti.UI.PORTRAIT]

});

var view = Ti.UI.createView({

width : 500,

height : 500,

top : '10%',

borderColor : 'red',

borderWidth : 5

});

var baseHeight = 500;

var baseWidth = 500;

var img;

if (Ti.Network.online) {

img = Ti.UI.createImageView({

image : 'http://images.fanpop.com/images/image_uploads/Flower-Wallpaper-flowers-249398_1693_1413.jpg',

height : baseHeight,

width : baseWidth

});

}

view.add(img);

win.add(view);

view.addEventListener('pinch', function(e) {

var t = Ti.UI.create2DMatrix().scale(e.scale);

e.source.transform = t;

});

win.open();

If I execute and check this code I am able to zoom out the image perfectly with in the view`s boundary. But when I zoom in the image it should not get resized less than the image default height and width. Can anyone share a possible solution.

Thanks in Advance
M.R.C.Prasad
