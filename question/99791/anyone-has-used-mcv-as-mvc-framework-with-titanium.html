Anyone has used MCV as MVC framework with Titanium?

Being an MVC and Titanium beginner I'm not sure about which MVC framework combines flexibility with a steep learning curve.
PureMVC seems to need heavy learning and probably unfolds its advantages only in complex projects.

I found MCV from Andrea Ferretti and it seems to me to be a lightweight and "mighty-enough" MVC framework.

Anyone has used MCV with Titanium and can share experiences?
Also any kind of "hello world" example would be appreciated very much!

  • Depending on they type of app you're planning to build, a lot of people would recomend Backbone.js but again it may provide steep learning curve.

    However I would recommend using Mustache for your templating needs. It's really simple.

    It's really a good idea to learn MVC outside of any tool, as it involves separation of concerns. I tend to set up 3 directories and then drill down to the specifics.

    • Controllers
      • Login.js
    • Models
      • User.js
    • Views
      • Login.html

    However you may fee more comfortable with this structure

    • Login
      • Controller.js
      • Model.js
      • View.html

    Ultimately, it doesn't matter what tool you use as long as you are comfortable with it and the organisation of your files.

    The benefit of the first structure is that if you need to use a model from a different controller its logically only one directory away from root making it more accessible. After all:
    &#x2F;Models&#x2F;User.js is easier to read than &#x2F;Login&#x2F;Model.js
