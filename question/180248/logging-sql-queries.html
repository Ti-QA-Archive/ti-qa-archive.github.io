  • Yes you can log SQL queries using Ti.API.info. Below is the sample code.

    
var query = "SELECT * FROM " + tblName + " WHERE EMPID = " + emp_id;
Ti.API.info("query ==> " + query);
var result = db.execute(query);

    Hope this helps.
