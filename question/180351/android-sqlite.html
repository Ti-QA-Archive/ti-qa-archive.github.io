Android sqLite

Hi, I have an error when building my app for Android. It works fine on iOS.

db = Ti.Database.open('SS');
        //--------------
        // create the DB
        // -------------

    db.open(); 
    db.execute('CREATE TABLE IF NOT EXISTS strstd(id INTEGER PRIMARY KEY, date DATETIME, bodyweight INTEGER, benchR INTEGER, benchW INTEGER, squatR INTEGER, squatW INTEGER, deadliftR INTEGER, deadliftW INTEGER, pressR INTEGER, pressW INTEGER);');
    db.close();

[ERROR] : TiExceptionHandler: (main) [59,59] —– Titanium Javascript Runtime Error —–
[ERROR] : TiExceptionHandler: (main) [0,59] - In lib/db.js:7,5
[ERROR] : TiExceptionHandler: (main) [0,59] - Message: Uncaught TypeError: Object #<TiDatabase> has no method 'open'
[ERROR] : TiExceptionHandler: (main) [0,59] - Source: db.open();
[ERROR] : V8Exception: Exception occurred at lib/db.js:7: Uncaught TypeError: Object #<TiDatabase> has no method 'open'
~~~

Thankyou

  • Mick,

    In your code, the first line where you open the database SS returns a database instance (see Database in the documentation). You can find the information on a database instance object here. If you take a look, there is no OPEN method for an instance, that is why you are getting the error "Object #<TiDatabase> has no method 'open' "

    You need to remove the db.open() from your code. It helps if you pay close attention to the errors you are getting. Usually it will point you in the right direction.

    Ray
