  • Valid clicksources I found so far

    • title
    • subtitle
    • pin
    • annotation
    • leftButton
    • rightButton

    rightView and leftView do not fire events themselves when clicked. When they contain something clickable, these clickable objects fire but not through e.annotation.

    So you can handle the above mentioned in the map.onClick event handler with a if or switch statement and buttons in left-/rightView with hteir own handlers.

    Corrections/additions welcome
