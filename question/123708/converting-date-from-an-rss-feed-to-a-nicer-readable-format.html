Converting Date from an RSS feed to a nicer readable format

Hello!

I'm just putting the finishing touches to my latest app, and would like to format the date string i get from my RSS XML feed.

I currently have this:

var datestamp = item.getElementsByTagName(&quot;pubDate&quot;).item(0).text;

Which returns this as an output:

Wed, 03 Aug 2011 18:51:33 +0000

What I'd like it to read is…

Wednesday, August 03

Or something along those lines

Hope someone knows how to do this!

Simon

    This may be overkill, and this isn't exactly an Appcelerator question, but I'm feeling generous. You'll likely want to add additional validation to this.

    var days = {
    Mon:'Monday',
    Tue:'Tuesday',
    Wed:'Wednesday',
    Thu:'Thursday',
    Fri:'Friday',
    Sat:'Saturday',
    Sun:'Sunday'
};
var months = {
    Jan:'January',
    Feb:'February',
    Mar:'March',
    Apr:'April',
    May:'May',
    Jun:'June',
    Jul:'July',
    Aug:'August',
    Sep:'September',
    Oct:'October',
    Nov:'November',
    Dec:'December'
};
var date = item.getElementsByTagName("pubDate").item(0).text;
var matches = /(.+?),\s+(\d+)\s+(.+?)\s+/.exec(date);
var formattedDate = '';
if (matches.length >= 4) {
    formattedDate = days[matches[1]] + ", " + months[matches[3]] + " " + matches[2];    
}
