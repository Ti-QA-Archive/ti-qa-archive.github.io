  • Kristian

    In order to display an html document, a webView is needed. There is an example of how to use it at the bottom of the APIDoc's webView page. However, it sounds like this is what you need:

    app.js:

    var win = Ti.UI.createWindow({
  backgroundColor:'blue',
  exitOnClose: true
});

var webview = Titanium.UI.createWebView({
  url:'index.html'
});
win.add(webview);
win.open();

    index.html:

    <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html >
<head>

This is the title
</head>
<body>
<div id="page-content">
  <h1>Testing Titanium</h1>
  <div id="inner-content">
  <p>This is the content</p>
  </div>
</div>
</body>
</html>

    That said, bear in mind that webviews don't perform as well as other UI views and, by using them for everything, you will not be exploiting the full power of Titanium.

    Cheers
