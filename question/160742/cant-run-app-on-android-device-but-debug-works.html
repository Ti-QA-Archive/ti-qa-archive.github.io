    • UPDATE: We found that manually removing /sdcard/<app id>/deploy.json from the device/emulator SD card solves the problem. You can do this by running:

      adb shell
rm /sdcard/<app id>/deploy.json

      I've created a ticket TIMOB-16132 to remove the dependency on the deploy.json and avoid this issue in the future.

      — commented January 6th 2014 by Chris Barber
