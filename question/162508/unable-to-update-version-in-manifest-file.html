    You will have to keep a separate value for the Android version. For this, simply edit the android section from your tiapp.xml like so:

    <android xmlns:android="http://schemas.android.com/apk/res/android">
    <manifest android:versionCode="11" android:versionName="1.1">
    </manifest>
</android>

    Hope this helps
