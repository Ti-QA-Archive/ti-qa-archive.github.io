I have a two views(x1,x2) in my main window(index.xml). now i want to change x2 view with another view that is in separate file(y3.XML) after that i would have two views in main window(x1,y3).i didn't found any info regarding this in documentation.please help me if it is possible.

  • Hi

    As you have not provided any code to assess some code starting points with you - this answer will only be based on the logic.

    • You can add a view from another file (xml, js, tss combo) into your current view (or any other view that is in scope)
    • You cannot directly swap one for the other as there is no SWAP method.
    • You can however remove one view and add another view
    • The new view will be the last child in the parent to which it was added
    • If the view you wish to add needs to go somewhere specific - add a parent view wrapper around the one you plan on removing, then remove from this instead, then add to that currently empty parent view - this way it stays in the same place as the removed one.
