  • I'll echo what Jacob said above.

    • I hope you aren't doing this on the device.
    • I hope you are planning to do this on a server.

    Big picture to get you started.

    • Pull feed in via php or javascript
    • Parse feed.
    • Each feed entry should have a unique identifier
    • For each entry, compare unique identifier to a data store (Ti.App.Properties or sqlite database)
    • If found, discard.
    • If not found, send push, add to data store
