Mobileweb - Background Images flashing on click

Hi All,

Got a very strange problem,

On my main window I've got a backgroundImage

However whenever I click anywhere on the screen, or try focus in textfields it just flashes

}

    backgroundFocusedImage: "/images/bg.png",
    backgroundSelectedImage: "/images/bg.png"
}

any help would be appreciated, thanks

  • You need add a view with background image, important, you should use this properties for your view: touchEnabled : false

        var win = Ti.UI.createWindow({
        left : 0,
        top : 0,
        navBarHidden : true,
        fullscreen : true
    });
    var viewBG = Ti.UI.createView({
        backgroundImage : '/images/background.jpg',
        touchEnabled : false,
        backgroundColor : 'transparent',
        opacity : 0.6
    });
    win.add(viewBG);
    //After, Do add anything you need to window

    Hope this help

  • On mobile web for some odd reason, when I place a background image on a view it flashes. What I did to remedy this was to remove the background image from the view and make an image view the same size and add it to that view. Hopefully this helps someone.
