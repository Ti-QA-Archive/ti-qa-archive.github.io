  • Well, there are a few things you must consider.

    ##Environment Variable
    The first thing you need to do is to set your JAVA_HOME environment variable.
    Here is what I get when i run the set JAVA_HOME command:

    JAVA_HOME=C:\Java\jdk1.6.0_25

    ##Path
    The next thing on the list is that you need to have the java (as well as the javac) command in your path. So be sure to append the full path with the \bin directory to your PATH variable.

    Here is the output I get when I run the where java command:

    C:\Java\jdk1.6.0_25\bin\java.exe

    ##Titanium Studio
    The last (and probably most important) thing you must set you Android SDK location in Titanium Studio. To do this, simply select the Window -> Preferences menu. And from the Preferences window, Go to the Titanium section and then set your Android SDK Home location.


    Hope this helps
