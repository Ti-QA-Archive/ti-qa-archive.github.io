  • Resolved: I can now run the TestProject app on both the emulator and on a device (Galaxy Nexus). Steps: I uninstalled Titanium. I then downloaded the new Titanium and installed it in a completely different location (C:\Titanium Studio). I ran it and then updated to get the new SDK (1.8). I deleted the existing TestProject and created a new one. I made no changes and just ran it on the emulator and then installed it to a device. And it worked. Interesting note: The TestProject APK is now twice as large as it used to be.
