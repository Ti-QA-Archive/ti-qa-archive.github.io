How to remove eventListener from Table

I have the following code, which adds an event listener to each row of a table. I would like to remove the last eventListener.

How do I do that.

TheTable.addEventListener('click', function(e) {
        var id = e.index +1;

    var win = Titanium.UI.createWindow({
     url: 'movie1.js',
     title: e.rowData.title,
         barColor: 'black',
         backButtonTitle: 'Tutorials'
    });
        win.variableReceived = id;
    win.open({animated:true});        
    });

TheTable.removeEventListener('click', function(e) {

});

EDIT: I have a table view that contains 4 rows. The above Eventlistener is attached to every row so that when you click a row the related video is played.

I would like to remove the Evenlistener from the last row of the table.

  • Since your event handler is an anonymous function, it is difficult to remove it (although there is a way you could remove it from inside the anonymous function itself). I recommend you name the function, and then manually remove it from the object you want to remove it from like so:

    Adding the event listener:

    TheTable.addEventListener(&#39;click&#39;, function onTableClick(e) {

    Removing the event listener:

    TheTable.removeEventListener(&#39;click&#39;, onTableClick);
