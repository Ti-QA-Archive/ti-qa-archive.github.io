  • You can use the getData() and setData() functions of the tableview for the most part. If you have one or moresections you will find the array the comes back ( using tbl.getData() ) as a set of top level array nodes each will have a node 'rows' that will contain the actual tableviewrows.

    If there are not that many rows and/columns you could keep them in variables and then repopulate the data array into the table using setData();
