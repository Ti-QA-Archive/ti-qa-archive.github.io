Using titleImage on createWindow works but text is missing

Hi,

i'm trying to create a windows and adding a image to the side of the title text.

I've once tried to use the setTitleControl() and add a custom view. That works fine but it has a little flick when adding my custom view.

When i did a little research i've found out that there is a property on the createWindow that i can actually set the title image, the "titleImage".

The problem is that when i set the titleImage, the text just disappear.

Is there anyway to add both, the text and the image?

My code atm is

var win = Titanium.UI.createWindow({
                titleImage: 'images/title-view.png',
                title:"Hello World",
                barColor: "#BB0000",
                backgroundColor: "#FFF"
            });

Hope someone can help me :)

  • Hi Rafael.
    You can do this if you change the barImage of the window, having the bagImage an image on the left of the title. Here is a sample code: 

    var win = Titanium.UI.createWindow({
    title : "Hello World",
    barImage : "images/barImageWithPictureOnTheLeft.png",
    barColor: "#BB0000",
    backgroundColor: "#FFF"
});
