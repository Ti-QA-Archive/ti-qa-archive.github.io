Auto size vs Ti.UI.FILL Ti.UI.SIZE in SDK 2.0

I simply don't get it …

  • in your case, you are defining top, bottom and height properties, so the first ones are overwriting the other (unless the labes is in a vertical layout). The easiest way to achieve what you are trying, would be to not defining height nor bottom properties, this way the height label will fit the text, as it is its default behaviour

    var l1 = Titanium.UI.createLabel({
            text:'long message long message long message long message long message long message long message long message',
            width:200,
           // height:'auto',
            color:'black',
            wordWrap:true,
            top:5,
            //bottom:5,
            left:5,
            right:5,
            font:{ fontSize: 14 }
            });

    this is the same as setting height: Ti.UI.SIZE
