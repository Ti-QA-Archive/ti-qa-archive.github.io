    Hi

    You do not need the outer view which you have set as if it was a scrollable view - but not actually defined as one.

    var viewTop = Ti.UI.createImageView({
    backgroundColor: 'blue',
    image: 'bar_grey100x28.png',
    height: 12
    top: 0,
    width: Ti.UI.FILL
});
var web = Ti.UI.createWebView({
    bottom: 12,
    height: Ti.UI.FILL,
    top: 12,
    url: 'http://www.apple.com/',
    width: Ti.UI.FILL
});
var viewBottom = Ti.UI.createImageView({
    backgroundColor: 'green',
    bottom: 0,
    image: 'bar_grey100x28.png',
    height: 12
    width: Ti.UI.FILL
});
win.add(viewTop);
win.add(web);
win.add(viewBottom);

    This creates the two bars with fixed heights and set to the top and bottom of the window, it adds a webView and tells it to take up the WHOLE height, but less the top and bottom settings which equal those two bars we already added.

    Done.
