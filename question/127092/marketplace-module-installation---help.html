Marketplace Module Installation - help

I downloaded the Rotate and Pinch Gesture from the Marketplace, but cannot figure out how to properly install that and use into my project. The documentation shows me how to put the code needed in tiapp.xml but not sure what to do with the rest. I tried to follow through the instructions for developing a module, but wasn't sure how to use this for an existing module that I have downloaded. I feel real dumb on this one. Any advice would be greatly appreciated. THanks!

  • Here is what I did:

    Under your project directory (in studio) create a modules folder. Then create android and/or iphone directories. Then copy the folder that is the module name. So for example I have:

    /modules/android/ti.inappbilling
    /modules/iphone/ti.storekit

    Each of those folders than have a folder that is the version number. Beside that you make the tiapp.xml change, the appropriate "require" statement and presto!
