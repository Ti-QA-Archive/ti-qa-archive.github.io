  • I just ran into this myself and the problem on my end (OSX) was the location where NPM was installed was not on the path available to Titanium Studio, i.e. it was installed to usr/local/bin which is not on the main path. To fix this I just created a symlink to in /usr/bin sudo ln -s &#x2F;usr&#x2F;local&#x2F;bin&#x2F;npm &#x2F;usr&#x2F;bin&#x2F;npm

  • Finally, I have found the solution
    vi a .bash_profile file in ~,

    alias titanium="/Library/Application\ Support/Titanium/mobilesdk/osx/<VERSION>/titanium.py"

    and add the alias to your environment, run the following command:

    source ~/.bash_profile
