Slide window Effect in Alloy

Hi,
I have decided to work on alloy for development. So, I am here to ask my first question. I want the window open with sliding effect from Right to left. I have used the animation effect from left to Right slide window. Which I have implemented using the following code:


$.topRight.addEventListener('click', function() {
    var followUS = Alloy.createController('followUs').getView();
    /*followUS.open({
     activityEnterAnimation : Titanium.App.Android.R.anim.slide_in_bottom,
     activityExitAnimation : Titanium.App.Android.R.anim.slide_in_top
     });*/
    followUS.open({
        activityEnterAnimation : Ti.Android.R.anim.slide_in_left,
        activityExitAnimation : Ti.Android.R.anim.slide_out_right
    });
});

As from the Android Resources. It provides only animation of sliding from left to right . But I want vice versa (from right to left effect). How can I do that? Please Help. Thanks in Advance.

    You seem to have headed in the right direction per the code you have commented out. here is what you have to do.

    1. Create the following folder in your project folder: platform/android/res/anim
    2. Create a file called slide_in_right.xml in the above folder.

    3. Paste the following content into it (You can remove the alpha line if you don't want that effect, but the rest of it is pretty obvious):
      ~~~
      <?xml version="1.0" encoding="utf-8"?>
      <set xmlns:android="http://schemas.android.com/apk/res/android">

      <translate android:fromXDelta="100%p" android:toXDelta="0" android:duration="300"/>
      <alpha android:fromAlpha="0.0" android:toAlpha="1.0" android:duration="300" />

    </set>

    4. change your open statement to the following:

    followUS.open({
    activityEnterAnimation : Ti.App.Android.R.anim.slide_in_right,
    activityExitAnimation : Ti.Android.R.anim.slide_out_right
    });
    ~~~

    Note that the activityEnterAnimation is referencing the XML file you created, while the exit animation is referencing the built in android animation.

    Hope this helps.
