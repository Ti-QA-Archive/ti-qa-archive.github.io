  • Hi

    You have much bigger problems than you think. You are accessing maps using the old deprecated methods you have to move to the correct methods using the maps module;

    • Modules.Map

    Please check the documentation to see the notice at the top of the page;

    • Ti.Map

    One other important fact is that iOS no longer supports google maps natively, iOS now uses Apple maps. You can add google maps but it is a paid for module.
