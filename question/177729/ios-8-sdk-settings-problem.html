    Hi Marini,

    Thanks for your concern.

    At first I got the same problem as yours. I've solved it using the following steps,

    • Install Xcode 6
    • Install Alloy 1.5 e.g sudo npm install -g alloy@1.5.0-beta2
    • Install Ti CLI 3.4.0-beta e.g sudo npm install -g titanium@3.4.0-beta
    • Install Ti SDK 3.4.0 beta
    • Switch iOS SDK home from Studio->Preference->Studio->Platforms->iOS; refresh iOS SDK home and check its ok. Otherwise, switch to Xcode 6 using sudo xcode-select -switch &#x2F;Applications&#x2F;Xcode6-Beta7.app&#x2F;Contents&#x2F;Developer&#x2F;... (it's my example path, use yours)

    Check those steps and see if it helps you.

    Thanks

    • Hi, one of my machine give the same error 2 days ago, i've found the solution. It's because everything was ok except my studio version, it was 3.3.0. For iOS 8.0, everything needs to be >= 3.4.x. So follow my steps below and hopefully you'll solve your problem this time.

      1. Install/ Update Titanium/ Appcelerator studio to 3.4.0

        ( Install Ti Studio, Ti Studio update or AppC Studio update )

      2. Update Ti CLI to 3.4.0 ( npm install -g titanium@3.4.0-rc4 )

      3. Update Alloy 1.5.0 ( npm install -g alloy@1.5.0-rc3 )

      4. Install Ti SDK 3.4.0.RC/RC2 ( ti sdk install --branch 3_4_X -d or Download )

      5. Install Xcode 6

      Thanks
