3.0.2 update break after alloy updates, revert?

I had many errors because I didn't pay enough attention to the update, and ended up with many many issues in my code.
As we are in a time crunch we don't have the time to port all right now.

Can I revert Titanium stack to the previous version?

— asked February 20th 2013 by Fabiano PS

  • (fixed answer)

    Yes we can, just roll back to previous version

    Mac (and hopefully linux) solution pointers:
    First of all close Titanium Studio and regret that you upgraded without researching about it first.
    Then open a terminal window and type one line at a time:

    sudo npm remove -g alloy
sudo npm install -g alloy@0.3.6

sudo npm remove -g titanium
sudo npm install -g titanium@3.0.24

titanium sdk uninstall 3.0.2.GA

    Hopefully you should be good. In case of errors, please create a new question referencing this answer
