Import from photolibrary and camera makes difference

Hi,
my goal is to import in a square image view.

  • If I import from camera and the photo is landscape, then on top and an bottom is transparent strip.
  • If I import from library a landscape photo then the image will stretched to square frame

How can I avoid the streching?

Here my code:

dialog.addEventListener('click', function(_e) {
            switch (_e.index) {
                case 0:
                    Ti.Media.showCamera({
                        success : function(event) {
                            setImage(event.media);
                        },
                        saveToPhotoGallery : false,
                        allowEditing : true,
                        mediaTypes : [Ti.Media.MEDIA_TYPE_PHOTO],
                    });
                    break;
                case 1:
                    Ti.Media.openPhotoGallery({
                        success : function(event) {
                            setImage(event.media);
                        },
                        saveToPhotoGallery : false,
                        allowEditing : true,
                        mediaTypes : [Ti.Media.MEDIA_TYPE_PHOTO],
                    });
                    break;

}
