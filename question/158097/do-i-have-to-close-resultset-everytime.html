Do I have to close ResultSet everytime?

Hi guys I have got a little dilemma do I have to close Result Set everytime before creation a new one?

var db = db.open("..."),
    rs = db.execute("SELECT * FROM table");

// while{ ... }

rs = db.execute("SELECT * FROM another_table"); // Can I do this or I have to close rs before it?

// while{ ... }


rs.close();
db.close();

  • Hi by Matej _

    Close database connections and resultsets as soon as you're done with them. This is the Best Practices

    Because In a mobile app, you're dealing with a single-user, memory constrained environment in which connection pooling is not important. The time to open/close connections is less of a hit that consuming memory by keeping connections open.

    rs.close(); // close the resultset when you're done reading from it
db.close(); // and close the database connection

    Thanks
