How to move to a different tab programmatically

I would like to move to a different tab based on the result of a swipe gesture on the window. I have the swipe event working correctly I can grab the direction property from this event, however I would like the swipe to result in a change in tab. Any ideas if this is possible? Thanks

  • Accepted Answer

    App.js

    Ti.App.addEventListener('gototab',function(e){
    tabGroup.setActiveTab(e.tab);
});

    whatever.js

    Ti.App.fireEvent('gototab',{tab:4});
