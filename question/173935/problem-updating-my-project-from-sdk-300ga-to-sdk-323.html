Problem updating my project from SDK 3.0.0GA to SDK 3.2.3

I have build my project tow years ago with SDK 3.0.0 GA and now i need to make some changes to my source code and build again
But i found problems
First new titanium studio doesn't support SDK less than 3.1.3
Second when i'm trying to build my project on SDK 3.2.3 i got this error "minimum android sdk version must be greater than 10"
Note: in my manifist i use api level 8 and android sdk version 2.1
Any help???

  • Hello Ahmed,

    if you have the SDK installed, you can still use Ti cli in order to compile your old project.

    Example

     ti build -s 3.1.2.GA
Titanium Command-Line Interface, CLI version 3.2.3, Titanium SDK version 3.1.2.GA
Copyright (c) 2012-2014, Appcelerator, Inc.  All Rights Reserved.

Please report bugs to http://jira.appcelerator.org/

Target platform to build for:
 1)  android
 2)  blackberry
 3)  ios
 4)  mobileweb
 5)  tizen

    Basically, use -s and set your version. I tried with 3.1.2.GA and worked just fine.
