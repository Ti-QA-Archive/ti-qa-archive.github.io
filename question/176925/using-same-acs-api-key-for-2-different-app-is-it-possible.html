Using same ACS API Key for 2 different app. Is it possible?

Hi,

I got a question on ACS. Is it possible for 2 different apps to use the same ACS API key, so these apps can access the same ACS database? I am asking this because I would like to create 2 apps, one for end-user, and the other for biz client. For biz client, this app will allow them to access ACS to create, query and update their respective data.

Is this possible? How? Appreciate any advice or suggestion on this matter.

Thanks in advance.

    Did you try to copy your acs keys from one app's tiapp.xml to the other one?

    Do this with 2 brand new apps, so you don't break your app in the case ACS doesn't like this at the server side ;)
