    If you look at the 'Events' for the scrollableView in the docs you will see the only events it supports are;

    dblclick
doubletap
dragend
dragstart
focus
keypressed
longpress
postlayout
scroll
scrollend
singletap
touchcancel
touchstart
twofingertap

    So you will need to pick from these…
