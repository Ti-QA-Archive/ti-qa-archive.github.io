    • Yes, iOS does have "system sounds" and that is an API that Titanium needs but does not yet have… however that blog post outlines a "hack" to get to the system sounds directory that if apple found Titanium using it, would get all Titanium apps rejected as this is not something they "allow"

      The system sounds API plays short sounds in a responsive mannor… like beeps and clicks in the user interface… however that API does not provide the default sounds in the OS… the only way to get those exact sounds is a hack
