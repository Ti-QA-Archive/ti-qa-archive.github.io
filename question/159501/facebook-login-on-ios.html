Facebook Login on iOS

Trying to get the the Facebook dialog working. I'm running it on both the iOS 7 Simulator and device.

On the simulator it opens in the browser, then sometimes when re-directed to the app the dialog is there. On device, i cannot get it to work at all. Even added the FB login button, this does nothing on device either!

I have included <property name="ti.facebook.appid">your_facebook_app_id_here</property > in mt tiapp.xml.

It does login, but the journey with the dialog is, not as documented and quite unreliable.

Others must surely be having these issues? Am i missing a trick or is Titanium still really flakey?!

Ti.Facebook.appid = '6563117977xxxx';
Ti.Facebook.permissions = ['publish_stream']; // Permissions your app needs
Ti.Facebook.addEventListener('login', function(e) {
    if (e.success) {
        alert('Logged In');
    } else if (e.error) {
        alert(e.error);
    } else if (e.cancelled) {
        alert("Canceled");
    }
});
Ti.Facebook.authorize();

  • FYI: I ended up using the native Facebook SDK in an xcode Project and amazingly encountered the same issues… Turns out the native Facebook login (iOS 7) needed to be cleared on device (iPhone 5) e.g Settings > Facebok > Account Name > Delete Account.

    I did this and the Facebook dialogs worked as expected. I have no doubt that this would also be the fix for the problems i was experiencing with Titanium.
