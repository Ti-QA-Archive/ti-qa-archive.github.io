Add an click event on label on the table view

Hi, I have a table view, where I am displaying a title, which should be clickable and opens a URL in a web view and second there is an star icon at the begining of the title, which, when it be clicked it stores the item in a db as favorite.

So my code now:

        for(var i = 0; i < jobsResult.responseData.item.length; i++) {

            //create a table row
               var row = Titanium.UI.createTableViewRow({
                hasChild: true,
                className: 'recipe-row',
                backgroundColor: '#fff',
            });

               //title label
               var titleLabel = Titanium.UI.createLabel({
                text: jobsResult.responseData.item[i].title,
                font: {fontSize: 16, fontWeight: 'bold'},
                left: 35,
                top: 5,
                height: 20,
                width: 270,
                color: '#000'
               });
               row.add(titleLabel);

               // here create the favorite star
               //title label
               var favoriteLabel = Titanium.UI.createLabel({
                text: 's'+i,
                font: {fontSize: 16, fontWeight: 'bold'},
                left: 10,
                top: 5,
                height: 20,
                width: 20,
                color: 'white',
                backgroundColor: 'green'

               });

        row.add(favoriteLabel);

            //add the table row to our data[] object
            data.push(row);
          }

          //finally, set the data property of the tableView
          //to our data[] object
          tblRecipes.data = data;
  1. How to add the event, where to click the title and open an webview?
  2. How to add the event to the favoriteLabel, so I can click and know which title I have to put into the favorites?

Thanks
Nik
