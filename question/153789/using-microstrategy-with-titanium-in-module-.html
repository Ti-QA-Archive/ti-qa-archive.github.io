  • Hi,
    Thanks everyone …
    My issue got resolved .
    Problem was library path of my dylib which I understood how to specify it.
    This was needed in module.xcconfig file .
    OTHER_LDFLAGS=$(inherited) /usr/lib/libiconv.dylib -F"path to MicroStrategyMobileSDK framework" -framework MicroStrategyMobileSDK -framework CoreText -framework CoreVideo -framework CoreData -framework GLKit -framework NewsstandKit -framework QuickLook
