Hi there!

I've got a little problem with this piece of code:

// POINT A

parentView.remove(childView);

// POINT B

if (parentView.children.length === 0) {
    ...
}

What I do is, hiding the parentView temporary if there are no children left.

On Android it is working absolutely fine and without a flaw, but while running on iOS there is the following "misbehaviour":

Let's say at POINT A the value of parentView.children.length is 2, it is still 2 at POINT B. It seems like the array of children does not update.

Titanium Studio Version: 3.1.3.201309132423

iOS SDK Version: 7.0
