How To Update A Tabbed Table

I am trying to understand how to update a specific table with in a specific tab.

Could some one assist me with the Syntax

Code In My XML

<Tab title="Event List" icon="KS_nav_views.png">
            <Window title="Event List">
                <TableView id="eventslisttable" layout="vertical"></TableView>
            </Window>
</Tab>

Wipe table entries

// wipe list view
    eventslisttable.setData([]);

After processing data push to table

        var tableView = Ti.UI.createTableView({
            backgroundColor : 'white',
            data : tableData
        });

        win.add(eventslisttable);

Thanks

    In Alloy, views etc are all referenced in controllers using $. as a prefix to the id of the object. So you need to 

    $.eventslisttable.setData(tableData);

    You dont need to create another table.
