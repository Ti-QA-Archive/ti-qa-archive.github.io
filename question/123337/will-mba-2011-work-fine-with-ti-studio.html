  • studio is based on the eclipse ide which is java based. This is a reasonably resource heavy ide.

    Saying that it works very well on my new mac book pro but I put 8gb ram in it. It might be cheaper to add more ram to your MacBook. Although air does have a solid state hard drive. Which will run it ok.

    Basically before you make your decision look at the requirements and recommendations for the eclipse IDE which should have at least 4gb ram and a nice fast hard drive.
