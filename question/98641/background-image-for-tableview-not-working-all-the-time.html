  • I was able to recreate this in a pretty simple manner:

    Apps.js:

    Titanium.UI.setBackgroundColor('#4c9bcd');


// create tab group
var tabGroup = Titanium.UI.createTabGroup({
    backgroundColor:'transparent'
});

    var tempWin = Titanium.UI.createWindow({  
        title:'Test',
        backgroundImage: 'images/bg1.png',
        url:'windows/test.js'
    });

    var tempTab = Titanium.UI.createTab({  
        icon:'KS_nav_views.png',
        title:'Test',
        window:tempWin
    });

    tabGroup.addTab(tempTab);  


    tabGroup.setActiveTab(tempTab); 
    // open tab group with a transition animation
    tabGroup.open({
        transition:Titanium.UI.iPhone.AnimationStyle.FLIP_FROM_LEFT
    });

    In my test windows/test.js is blank. What I see is that the background image only shows up some of the time. FYI I'm testing on the 1.6 emulator.
