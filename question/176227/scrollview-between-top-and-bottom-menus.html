Scrollview between top and bottom menus

Hi.

I'm trying to do a scrollview with and simple idea (on Android):

BARmenu (a simply view with 2 buttons)
SCROLLview (with variable content)
BOTTOMmenu (a simple text that I want to be always visible)

I've been testing a lot of different possibilities, but any of them worked. The most logic for me was this:

    <View id="win" layout="vertical">
        <View id="topMenu" class="backgroundColor" height="50">
            <View id="leftButton" class="iconButton" left="5"></View>
            <View id="title">
                <Label id="lblTitle" text="title_menu" />
            </View>
            <View id="rightButton" right="5"></View>
        </View>

        <ScrollView id="mainMenu" layout="vertical" contentHeight="auto" scrollType="vertical" showVerticalScrollIndicator="true">
            A lot of boring content, that is bigger than the screen height
        </ScrollView>

        <View id="bottomMenu" class="backgroundColor" height="30" bottom="0">
            <Label id="lblSedet">SEDET</Label>
        </View>
    </View>

This is, I want my bottom menu at bottom of the screen. But the scrollview always go to the end, I never see the bottom menu.

I tried also to do this in code:

$.mainMenu.height = screenHeight - $.topMenu.height - $.bottomMenu.height;

Or even manually do $.mainMenu.height = 380;

but nothing worked.

Some idea what I'm doing wrong?

    Hi

    This can trip people up, especially when the layout property is used as it makes you think it will help (it does in most things) but the answer is fairly simple once you have seen it once.

    I prefer the TSS route as this is where styles should be, but you can add the properties into your xml if you must.

    "win": {
    "height": Ti.UI.FILL,
    // "layout": "vertical", // do not use
    "width": Ti.UI.FILL
},
"topMenu": {
    "height": 50,
    "top": 0,
    "width": Ti.UI.FILL
},
"mainMenu": {
    "bottom": 30, // same as bottom height
    "contentHeight": Ti.UI.FILL,
    "contentWidth": Ti.UI.SIZE,
    "height": Ti.UI.FILL,
    "top": 50, // same as top height
    "width": Ti.UI.FILL
},
"bottomMenu": {
    "bottom": 0,
    "height": 30,
    "width": Ti.UI.FILL
}

    You can add your own properties into these, but this gives you the auto sizing and aspect ratio respecting solution you need.
