  • Hi,

    I suggested to go with Alloy, eventually you end up with creating Alloy project for what you created.

    With the Alloy you will end up writing much lesser code, maintainable code and more over MVC architecture which has better reusability of your code.

    Here is a thread about why Alloy. study into it, you will decide eventually - http://developer.appcelerator.com/question/149865/why-alloy

    Here is the link to documentation if you want to start on Alloy - http://docs.appcelerator.com/titanium/3.0/#!/guide/Alloy_Framework
