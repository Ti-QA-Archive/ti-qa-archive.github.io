showCamera events never firing on Android?

Using SDK 1.7.2, and 1.8 CI (20110811) with the app running on a Galaxy S Android device, Firmware (Android version) 2.2, the success function on showCamera does not fire.

"Showing camera" shows up in the logs, the camera opens, and I'm able to take a picture and hit Save. After hitting Save, however, neither the success nor the error functions get called (that is, neither the log message nor the Alert show up).

Here's some simple code do demonstrate:

function show_camera(e) {
    Titanium.API.info('############ Showing camera');
    Titanium.Media.showCamera ({
        success: function(event){
            Titanium.API.info('############ showCamera SUCCESS');
            var a = Titanium.UI.createAlertDialog({
                title: 'SUCCESS'
            });
            a.show();
        },
        error: function(error){
            Titanium.API.info('############ showCamera ERROR');
            var a = Titanium.UI.createAlertDialog({
                title: 'ERROR'
            });
            a.show();
        }
    });
}

  • Finally got this working. Turns out you have to open a new window using the 'url' method. See this ticket for some background: http://jira.appcelerator.org/browse/TIMOB-1595 .

    Hopefully this saves someone else the two days I wasted banging my ahead against the wall.

    In my app.js

    function save_image(imagefile) {
    imageRow.leftImage = receiptfile.nativePath;
}

function take_picture() {
    var w = Titanium.UI.createWindow({
        url:'/take_picture.js',
        navBarHidden: false
    });
    w.save_image = save_image;
    w.open();
}

    take_picture.js:

    var window = Titanium.UI.currentWindow;

function generate_id() { return (new Date()).getTime() - 1284506183534; }

function show_camera() {
    Titanium.API.info('############ showing camera');
    Titanium.Media.showCamera({
        success: function(event) {
            Titanium.API.info('------------ This is Camera SUCCESS');

            var cropRect = event.cropRect;
            receipt_image = event.media;

            var f = Titanium.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory,'receipt' + generate_id() + '.png');
            f.write(receipt_image);

            window.save_image(f);
            window.close();
        },

        cancel: function() { Titanium.API.info('------------ CANCEL'); },
        error:function(error) {
            Titanium.API.info('------------ showCamera ERROR');

            var a = Titanium.UI.createAlertDialog({title:'Camera'});
            if (error.code == Titanium.Media.NO_CAMERA) {
                a.setMessage('Device does not have camera capabilities');
            } else {
                a.setMessage('Unexpected error: ' + error.code);
            }
            a.show();
        },
        allowImageEditing:false,
        saveToPhotoGallery:false,
        mediaTypes:[Ti.Media.MEDIA_TYPE_PHOTO]
    });
} 

show_camera();
