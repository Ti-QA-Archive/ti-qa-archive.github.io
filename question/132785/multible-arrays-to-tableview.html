  • The easiest is to combine the two arrays, then setData. This is a javascript function.

    The concat() method is used to join two or more arrays.

    This method does not change the existing arrays, it only returns a copy of the joined arrays.

    
allData = Data.concat (Data2); //  concatenate the two arrays in to one array

tableview.setData(allData); // add the concatenated to the table view

    Invest in a few javascript books, or even google can be your friend when looking up how to use javascript. Some good books are "Javascript - The Definitive Guide", "Javascript - The Good and Bad Parts", and "Javascript Patterns". The definitive guide is a good book as a reference. The other two are focused on best-practice.
