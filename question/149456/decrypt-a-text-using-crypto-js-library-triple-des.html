Decrypt a text using 'crypto-js' library (Triple DES)

Hi all,
I'm using "crypto-js" link library to decrypt a text.
I'm having a problem to use the "decrypt" function,
I'm using Triple DES.
So I placed "tripledes.js" under "lib" folder.
And in my controller:

var lib_decrypt = require('tripledes');
var message = lib_decrypt.DES.decrypt(Ti.Utils.base64decode(encryptedtedtext), "secretparaphrase");

I'm always getting the same error: "Cannot call method 'decrypt' of undefined".
So anyone can help me how to use this library to decrypt my text? I need to know how to use the "tripledes.js" library functions because its a big library.
Thank you.

  • It's a old question, but I hope to help someone:

    In the first code from Anthony,

    He use DES, but he needs TripleDES:

    var lib_decrypt = require('tripledes');
var message = lib_decrypt.DES.decrypt(Ti.Utils.base64decode(encryptedtedtext), "secretparaphrase");
alert(message);

    The code must be:

    var lib_decrypt = require('tripledes');
var message = lib_decrypt.TripleDES.decrypt(Ti.Utils.base64decode(encryptedtedtext), "secretparaphrase");
alert(message);

    Is necessary too, add the last line in the file: "tripledes.js"

    exports.CryptoJS = CryptoJS;

    And the file "tripledes.js· from the folder "Crypto/rollups/" must be copied in the app folder: "app/lib/"
