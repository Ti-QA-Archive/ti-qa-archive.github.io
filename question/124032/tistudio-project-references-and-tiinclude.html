TiStudio Project References and Ti.include

Does anyone know whether the project reference ability in TiStudio works for mobile projects?

I want to create a library of js code in a blank project and then use the project reference facility in other projects to access the js files/functionality in the 'library'.

I have given it a try but when using a Ti.include there seems to be no way of referring to the code in the referenced project.

At the moment I am assuming that actually this is not possible due to the way Ti.include works, but wondered if anyone had any suggestions? i.e. is this the right approach or should I really be looking at using/creating a js module?

  • A workaround is that Ti.include accepts any path, not just "current project" :)

    Too include (relative to current project) a js file from a subfolder in a different project, i would write: Ti.include('../../PROJECT_X_COMMON/src/xxxx.js');
