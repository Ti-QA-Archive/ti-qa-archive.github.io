  • … There is no such thing as "Mac iOS 4.1"?

    There is iOS which is the mobile operating system that runs iPhones, iPads, and iPod Touches which is currently on 4.1 (Except for iPad, which is still 3.2.2) and is supported by Titanium Developer.

    Titanium is currently on "release 1.4", but has a beta 1.4.1 that you can build yourself if you have Git.

    Mac OS X is on 10.6.4 and has nothing to do with Titanium, other than you have to have something like 10.5.4+ to run Titanium (Which is usually what every other program has a requirement of).

    PS - If using the simulator for iOS 4.1, you will get strange errors in the console such as:

    Detected an attempt to call a symbol in system libraries that is not present on the iPhone:

    Just ignore those. They're going to be fixed soon and do not stop the App from being accepted by Apple.
