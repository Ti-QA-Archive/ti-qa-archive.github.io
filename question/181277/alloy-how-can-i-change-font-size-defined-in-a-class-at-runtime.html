Alloy: How can I change font size defined in a class at runtime?

  • Application type: Alloy mobile for Android
  • Titanium SDK: 3.5.0.GA
  • Platform & version: Android
  • Device: real S3 Andr. 4.3 and Genymotion API 10
  • Host Operating System: Win x64
  • Titanium Studio: Titanium Studio, build: 3.4.1.201410281727

HI Folks, I've a TSS file containing this:

".myText": {
    font: {
        fontSize: "30dp",
    },    
}

How can I change the 'fontSize' value at runtime in the relative controller? I tried 

$.myText.font = {fontSize: "50dp"};

and

$.myText.font.fontSize = "50dp";

but don't work. Any idea? Thanks.

    You can not dynamically change the definition of classes define in tss file. You can apply the class to the respective Ti object dynamically. For e.g. create two class in tss and apply them dynamically.

    TSS code

    
".myText30": {
    font: {
        fontSize: "30dp",
    },  
}

".myText50": {
    font: {
        fontSize: "50dp",
    },  
}

    Controller code 

    
var element = Ti.UI.Label({
    text: 'Font Change'
});

$.removeClass(element, 'myText30');
$.addClass(element, 'myText50');

        $.title.font = {fontSize:50};
