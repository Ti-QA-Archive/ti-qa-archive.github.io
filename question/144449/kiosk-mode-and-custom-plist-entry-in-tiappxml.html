Kiosk mode and custom plist entry in tiapp.xml

I would like to run a single app in kiosk mode on a iPad for a customer. The Homebutton should be disabled. I read about a uncommented feature in the apps plist.

The plist should contain the following value:

<key>PayloadType</key>
<string>com.apple.defaults.managed</string>

Does anyone know how to make this work. I would like to add this entry to tiapp.xml.

— asked November 7th 2012 by Tonio Seiler

  • in your tiapp.xml

    <ios>
  <plist>
    <key>PayloadType</key>
    <string>com.apple.defaults.managed</string>
  </plist>
</ios>

    It may need a <dict> wrapper for it:

    <ios>
  <plist>
    <dict>
      <key>PayloadType</key>
      <string>com.apple.defaults.managed</string>
    </dict>
  </plist>
</ios>
