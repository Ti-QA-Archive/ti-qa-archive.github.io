    • Nogga,

      Lets make sure that some of the Xcode parts are where they should be.
      Some of the Xcode 4.1 installs failed to set paths properly, leaving the possibility that other parts might go missing as well.

      from a console, please run 

      xcodebuild -showsdks

      It should return something similar to this:

      Mac OS X SDKs:
    Mac OS X 10.6                     -sdk macosx10.6
    Mac OS X 10.7                     -sdk macosx10.7
