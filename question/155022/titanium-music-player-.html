Hi guys

Just wondering if anyone has an example or any information of using Ti.UI.MusicPlayer for playing multiple remote audio files.

For instance pulling in 5 audio file urls and being able to have back/forth functionality.

    The MusicPlayer object cannot play remote files since it only has access to the media library.

    But the AudioPlayer can though. Unfortunately, you can only set one url property at a time.

    One approach you could use would be to keep all your urls into an array and navigate through this array using your own buttons.

    Hope this helps
