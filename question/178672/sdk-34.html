  • Are you using two Xcode in your mac? If so, Xcode 5 may be installed as Xcode and Xcode 6 may be installed as Xcode 2. If so, try running this command in your terminal

     sudo xcode-select -switch /Applications/Xcode2.app/Contents/Developer/

    Restart the studio, see if performance shows Xcode 2. See if can launch iOS simulator. Thanks.
