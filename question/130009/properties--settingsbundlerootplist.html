  • Accepted Answer

    Since sdk 1.8, Root.plist must be located in 'platform' folder instead of 'modules' folder, so the right path from your project folder is:

    &#x2F;platform&#x2F;iphone&#x2F;Settings.bundle&#x2F;Root.plist
