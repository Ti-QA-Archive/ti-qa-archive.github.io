Get xml attribute value

I want to extract the price attribute from the following xml.The xml is returned from the webservice. Can anyone help me?

<Snapshots>
<Snapshot time="2010-12-20T17:49:23" price="170.25"></Snapshot>
<Snapshots>

    Try this…

    var xmlObj = Titanium.XML.parseString('<Snapshots><Snapshot time="2010-12-20T17:49:23" price="170.25"></Snapshot></Snapshots>');

var price = xmlObj.documentElement.getElementsByTagName("Snapshot").item(0).getAttribute("price");

Ti.API.info(price);

    BTW your xml isn't valid, the closing Snapshots doesn't have a /
