Insert geo location coordinates works in database but does not works in device

Below is my code 

var imageID = Ti.App.DataStorage.GetImageCount() + 1;
    var storeResultsInDB = function(e) {

        Ti.App.DataStorage.InsertCurrentLatitude(imageID, e.coords.latitude);

    }

  • My initial reaction is that you should never try and expand the existing Ti.App namespace with your own code. I've seen people get away with it for storing simple property values, but to start storing methods in it is highly likely to be a fragile solution.
