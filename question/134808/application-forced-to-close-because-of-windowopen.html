Application forced to close because of Window.open()

Hi, I am developing an Android 2.3.3 app using Titanium 1.8.2. I am trying to do a simple window opener, but every time I include the line
Window.open(), my app encounters some error and is forced to close.
Code:

var window = Ti.UI.createWindow({
    barColor:'black',
    backgroundImage:'pong.png',
    title:'The Simple Game'
});

window.open();

Do I need to use a fireEvent?

Thanks so much for your help.

    The code seems pretty straight forward (and it should work).

    But the problem could be linked to where the open function is called. If you provide more details, it can give a better understanding of the nature of your problem.

    Just for info, you don't need to use a fireEvent to open a window.

  • Never mind, I figured it out - I need to add an argument to window.open();
    This was my solution:

    
var window = Ti.UI.createWindow({
    barColor:'black',
    backgroundImage:'pong.png',
    title:'The Simple Game',
    exitOnClose:true
});

window.open({fullscreen:true});
