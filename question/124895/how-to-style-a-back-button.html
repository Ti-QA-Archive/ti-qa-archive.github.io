How to Style a Back Button?

Hello all. My app uses a tab group where each tab has a table view menu with single window drill downs. I reskinned everything to match my apps look including the tabgroup drilldown back buttons and I set their click call back to theWin.close() where theWin is a global window variable since each window is programmatically created using includes when the corresponding option is selected.

Of course the problem with this is that any back button will simply close whatever the global theWin window is when its clicked, even if its on another tab. Is there a way to reskin the default back buttons with unselected/selected images and/or is there a predefined back function I can call for each back button? Below is an example of the tableview callback that generates the drilldown window and its back button. Thanks.

tableMenuView.addEventListener('click', function(e)
    {
        if (e.rowData.page)
        {
            var back = Ti.UI.createButton({
                backgroundImage:"images/global/backUp.png",
                backgroundSelectedImage:"images/global/backDown.png",
                width:64,
                height:34
            });

            back.addEventListener("click", function() {
                myWin.close();
            });

            myWin = Titanium.UI.createWindow ({
                backgroundColor:'#fff',
                navBarHidden: false,
                barImage:'images/navBars/blankNav.png',
                backgroundImage:'images/bgs/' + e.rowData.bg,
                rightNavButton:myTarWin.rightNavButton,
                leftNavButton:back
            });

            Ti.include(e.rowData.page);

            tabGroup.activeTab.open(myWin,{animated:true});
        }
    });

The only solution I can think of is to repeat the window creation code in each included file so its in scope (since the local scope doesn't seem to carry over to the includes).
