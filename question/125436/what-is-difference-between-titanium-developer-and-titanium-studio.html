  • Ti Studio is a full IDE with integrated debugger. It replaces Ti Developer which has been end of life'd by Appcelerator.

    With Ti Developer you used the editor of your choice, then Ti Developer let you launch it in the simulator or compile for testing or distribution. There was no IDE nor debugger.

    Most people prefer Ti Studio now. Some have an aversion to Eclipse (the underlying IDE on which it is based) or for whatever reason don't want the full IDE.

    Since Ti Developer was open source, although Appcelerator no longer enhances it or makes it available, some users have given it new life. See this thread for more information and download links.
