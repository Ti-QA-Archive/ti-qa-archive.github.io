  • Remote web pages have no access to the Titanium API. So, a remote page cannot fire an event. However, you can use the webview.evalJS() function to inject javascript into a page. That code could fire an event.

    By the way, it's considered poor form to start your question with a "other projects are a lot better" sort of jab. There are tons of code examples for Titanium from Appcelerator, our wonderful Titans, and the community at large. Our API is pretty sizeable and there's not a code example for every possible permutation of the API.

    In this case, there is a code example for doing what I think you're asking in our Guides. See this page.
