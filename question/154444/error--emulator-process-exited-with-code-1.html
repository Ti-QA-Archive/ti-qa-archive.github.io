  • I installed a fresh copy of Titanium and Android SDK so I had Android 4.2.2 aka build 17 and Titanium 3.1.1 I believe. I copied the files from build-tools to platform tools and that didn't work. Installed 2.2 for Android SDK as well, still didn't work. Installed 2.3.3 for Android SDK after and it finally worked.

    So on a fresh copy just:
    Install Android SDK 2.2 + 2.3.3
    If that alone doesn't work then you will have to copy the 3 files from build-tools to platform tools and that should work. Also of course have Android SDK 4.2.2 installed as well. This is windows 7 64 bits
