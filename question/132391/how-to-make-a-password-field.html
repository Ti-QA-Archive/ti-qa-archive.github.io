how to make a password field

How to specify the type of a text field as a password, like in HTML "type=password".
I searched and found stuff like "passwordMask", and "hintText", but it is not working…still writing in the field appears as visible text.

        return Ti.UI.createTextField({
        top : theTop,
        width : '90%',
        height : '45dp',
        color : '#595959',
        textAlign : 'left',
        hintText : "Password",
        passwordMask : true, // THIS SHOULD DO THE TRICK
        font : {
            fontWeight : 'bold',
            fontSize : '16dp'
        },
        keyboardType : theKeyboard,
        returnKeyType : theReturn,
        borderStyle : Ti.UI.INPUT_BORDERSTYLE_NONE
    });
