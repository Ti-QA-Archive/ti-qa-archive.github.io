    I would add a custom propriety to the view like:

    var MyView = Titanium.UI.createView(
    {
    iseeyou: true
    });

    When you hide that view you can set the propriety to false like

    MyView.iseeyou = false;
    MyView.hide();

    Not sure if thats what you need but… :)
