  • Is this in the emulator or on device? In the emulator, do you see connection bars (in the notification bar) or is there an "X" indicating no connectivity? On my system, the emulator fails to load correctly at least 50% of the time. This is a Google/Android problem, BTW, not a Titanium issue.

    If that's the case, you'll need to close the emulator and kill the adb.exe process (with Task Manager). You might even need to close Ti Developer too.

    Whether on device or in the emulator, you can't use "localhost" type addresses. Either device is essentially another computer on the network. So, you'll need to use IP addresses or network names.
