How to switch to new window

I am developing IOS and Android mobile application. There is a button on window, when user press the button, new window is created. On android, I can do it by using Intent and startActivity method.

On titanium, what kind of code should I write to perform such action.
PS. must be available to iOS and Android

    Hello Edison,

    Its simple way : 

    button.addEventListener('click', function(e){
    //creaet New Window Here
    var win = Ti.UI.createWindow();
    win.open()
});
