Detect TableView last row event

Hi,

Is there any way to detect when doing scroll I've reached the last row?

  • Accepted Answer

    tableview.addEventListener('scroll',function(e)
{
    Ti.API.info(e.contentOffset.y);

});

    this gives you the idea of the content position (usually used to determine the "Pull to refresh" method). There are no direct methods, but the scroll event gives you enough properties to look into:
    http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.TableView-event-scroll
