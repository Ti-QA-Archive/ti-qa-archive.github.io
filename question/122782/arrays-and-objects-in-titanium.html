    • Yes. "{}" means empty object, "{ prop: 'value' }" means object with property "prop" that has string value "value". You can think about object as key-value container. So, you can create container with "{}", add and/or delete properties, loop through properties and so on. Array objects are similar, but they are index based containers. So, instead of key, you use index. The difference is also that objects are unordered, but arrays are ordered. So, if you "push" value to the array, it has new index that is next in that container. So, 

      //create new object with one property
var myObj = { prop1 : 1 };
//add new property
myObj.newProp = true;
//or you can add property this way
myObj['oneMoreProp'] = 'Test';

//loop
for(var p in myObj) {
    alert(p);
    alert(myObj[p]);
}

//create new array
var arr = [];
arr.push(1);
//first element in array has index 0
alert(arr[0]);
arr.push(2);
//second element in array has index 1
alert(arr[1]);
arr.push(3);
//in general, last element has array length - 1 index
alert(arr[arr.length - 1]);

//loop
for (var i = 0; i < arr.length; ++i) {
    alert(arr[i]);
}
