I want to put ads manually in my app. I am creating an image view tab at the bottom of my screen for my app. I have more than one add image and want to have it cycle through the images every like 30 seconds.

How can I go about doing that?

  • Hi

    The best and very easiest way is to use a scrollableView and add each 'page' as a different advert image. Then create a timer using setTimeout to show the next advert using the setCurrentPage method after the interval and to use setInterval again to start the next interval.

    I would suggest not using setTineout even though it looks more convenient, as it can introduce memory leaks if you do not clean things up properly.

    BTW scrollableView accepts ANY view as the 'pages' view array, no need to wrap inside another view. Same is true if you wanted to add a tableView as a page - I know you do not, just illustrating a point.
