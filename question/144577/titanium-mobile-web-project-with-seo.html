  • You can give your Titanium Mobile Web app whatever URL you wish. Since your app is compiled into a single index.html file, you can put it in any directory or setup whatever rewrite rule you want. However, Titanium Mobile Web apps are single page applications. They are not web pages. If you turn off JavaScript, all you see is the startup splash screen.

    If you want to make your app SEO'd, then you're best creating a SEO optimized page that links to the index.html file.
