  • Hi Fenil Jain,

    Please check my test case below and see if it works for you.

    Test Code:

    var win = Ti.UI.createWindow();

var webView = Ti.UI.createWebView({
    url: 'https://www.youtube.com/watch?v=KZcH0AessFY',
    enableZoomControls: false,
    scalesPageToFit: false,
    scrollsToTop: false,
    showScrollbars: false
});

win.add(webView);

win.open();

    Test Environment:

    Ti SDK 3.4.1.GA

    AppC Studio 3.4.1

    Ti CLI 3.4.1

    iOS v8.0,8.1

    Android 4.4.4, 5.0
