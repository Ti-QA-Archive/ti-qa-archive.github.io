  • Hi

    According to the documentation on focusable the use of this property defaults to false. The use of the property appears limited to the hardware navigation keys or trackballs.

    If this determined is a textfield could gain focus (so soft keyboard displays) then the default of false would mean each textfield would not be focusable without changing this default - which is not true all textfields work with setting this property.

    What are you using it for?
