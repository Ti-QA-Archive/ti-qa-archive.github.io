Disable eventlistener temporarily

I want to disable the scroll event listener of an scrollview and activate it later then.

Is that possible?

    Yes.

    Create a separate function to handle the event and then use addEventListener and removeEventListener to enable/disable it

    eg

    
function myClickHandler()
{
    // do something onClick
}

var b = Ti.UI.createButton();

/// add handler
b.addEventListener('click, myClickHandler);

//remove handler

b.removeEventListener('click', myClickHandler);
