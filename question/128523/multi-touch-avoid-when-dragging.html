  • You can reorder the z index of your components with the z-index property. With your code you could do this:

    var maxZIndex = 1;
rec1.addEventListener('touchmove', function(e) {
         newX = e.x + rec1.animatedCenter.x - rec1.width/2;
         newY = e.y + rec1.animatedCenter.y - rec1.height/2;
         rec1.animate({center:{x:newX,y:newY}, duration:1});
         rec1.zIndex = maxZIndex++;
});


rec2.addEventListener('touchmove', function(e) { 
         newX = e.x + rec2.animatedCenter.x - rec2.width/2;
         newY = e.y + rec2.animatedCenter.y - rec2.height/2;
         rec2.animate({center:{x:newX,y:newY}, duration:1});
         rec2.zIndex = maxZIndex++;
});

    • You could use a boolean variable that indicates whether or not an element has already been selected. If this variable is false, allow touches to move elements, if its true, don't allow touches to move elements. You'd likely want to set these values in the touchstart and touchend events.
