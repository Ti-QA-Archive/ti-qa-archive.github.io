SearchBar Blur not working on event firing

I have table view with section and search bar. I want to blur the search bar/table when I click on some button on some other view. The bluring of works well when you click on searchBar cancel button.

But I want to blur the searchBar on clicking some button (Not part of tableview).
I am doing this

button.addEventListener('click', function (e) {
searchBar.fireEvent('cancel');
}

searchBar.addEventListener('cancel', function (e) {
e.source.blur();
});

The above code does not work, though cancel event is getting fired. The same cancel event is fired when i click on search bar cancel button and it blur the searchBar and table but if you fire it programmatically, it does not searchbar and table.

I even try to call blur() directly on searchBar but that does not work too.
button.addEventListener('click', function (e) {
searchBar.blur();
}
