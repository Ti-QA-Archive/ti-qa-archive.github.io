App crashing on launch on Android

Okay, this is a broad issue, but I've tried everything I can think of and
can't figure out why it's happening.

Here's what's happening:
APK downloads (from GMail in this case, to the device), does the install, launches, then crashes at the splash screen.

Here's the information:
Android running 2.2 and 2.3.4 (an HTC Incredible and a Droid X).
SDK 1.7.2 and the CI build of 1.7.3 (I've tested all of them and they all have the issue)
Ti Studio 1.0.4.201108101535

I haven't changed anythign since the last version of the app worked. I've even reverted the code base back to a few builds before I was having this issue to see what was causing the problem and still no luck.

iPhone version works fine and the Android emulator(s) aren't having any problems either.

The one thing I have noticed is that my app's name has changed, although I haven't touched the tiapp.xml file other than to change the version number AND I've run a test with a reverted tiapp.xml to make sure that wasn't the issue.

Any help or ideas would be helpful, as we need to launch soon and, right now, we can't even test our product on a device.
