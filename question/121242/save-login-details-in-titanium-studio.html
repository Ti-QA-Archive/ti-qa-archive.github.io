Save login details in Titanium Studio

Is it possible to save the email and the password in Titanium Studio? Everytime I launch it I have to enter my details again.

Fedora Linux, Titanium Studio, build: 1.0.0.201106101247

    Solved (for me).

    "Window" menu ->
    "Preferences" option ->
    "General" section ->
    "Security" sub-section ->
    "Secure Storage" sub-section ->
    "Contents" tab ->
    "Delete" button ->
    OK in "This action will delete default secure storage…" modal window

    Re-start (twice in my case).

    I have Windows Vista 32 bits but I guess it will work in every plataform.

    I used the information in David Bankier's answer to find my solution.

  • Yes use the persistant data …

    Ti.App.Properties.setString('loggedIn','Y');
    Ti.App.Properties.setString('userID', evt.dataIn[0].USERID);

    This will store the data required then you can write a quick function to control the flow on app start or resume.

    Hope this helps.
