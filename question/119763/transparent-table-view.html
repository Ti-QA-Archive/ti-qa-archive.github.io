  • i found the solution 

    var view1=Titanium.UI.createImageView({
            top:90,
            width:310,
            height:331,
            backgroundColor:'white',
            opacity:0.5,
            borderRadius:10
        });
        win1.add(view1);

tblView=Titanium.UI.createTableView({
        top:90,
        width:310,
        height:306,
        backgroundColor:'transparent',
        borderRadius:10,
    });

    this is works try…..
