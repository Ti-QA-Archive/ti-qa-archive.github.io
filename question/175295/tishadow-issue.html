TiShadow issue

Hi Community,

I've started using TiShadow and I must say it's awesome and huge time saver approach. However, I'm facing a strange issue of Localisation.js file while using the following build command to run Alloy project in simulator.

Surprisingly, If I remove all the localisation calls i.e. L(&#39;&#39;) from the project then all works as expected. This is not the case with Classic Titanium Projects.

CLI Command:

titanium build -p ios -F ipad -T simulator -s 3.2.3.GA --shadow

Error:

Error Domain=com.google.GDataXML Code=-1 "The operation couldn't be completed. (com.google.GDataXML error -1.)" at Localisation.js (line 22)

Other details:

CLI version 3.2.3, Titanium SDK version 3.2.3.GA
Building using iOS SDK: 7.1
Building for iOS iPad Simulator: 7.1
Building for device family: universal

Thanks in advance.
