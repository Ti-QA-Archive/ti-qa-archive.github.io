  • Hi

    They should be automatically, so either you have something else wrong or a setting somewhere is not set correctly.

    Do a test for me, create a new classic "default project" and replace the contents of app.js with the following;

    var win = Ti.UI.createWindow({
    backgroundColor: 'orange'
});
var view = Ti.UI.createView({
    backgroundColor: '#fff',
    borderColor: '#c60000',
    borderRadius: 20,
    borderWidth: 5
    height: 200,
    width: 200
});
win.add(view);
win.open();

    Should be a white square on an orange background with a dark red border and smooth curves.
