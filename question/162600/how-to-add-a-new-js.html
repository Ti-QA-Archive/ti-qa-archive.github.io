    Simply select the File > New > File menu.

    From there, simply select where you want your new file to be created. (usually under the &#x2F;Resources directory) and then name your new file as you choose. Just make sure that your file name ends with .js and it will be considered as a JavaScript file by Studio.

    Hope this helps.
