TI.App variable works only once.

Hello

I am creating a tableView and passing it into a gloable var (TI.App.myVar).

In another window (win2) the tableView is been showed - It works…

But If I close win2, and open it again the table is not been showed. -it is only been showed the first time I open win2.

What is wrong?

MG

  • It's not a good practice to pollute the global scope with data.
    Because of the way the javascript is converted to native code, some weird stuff can happen ( like here :) )

    Why don't you pass the table as a property of the new opened window?

  • let's say in app.js

    var the_table =  Titanium.UI.createTableView({ blah blah});


var win_to_open =  Titanium.UI.createWindow({
   url:'the_file_for_the_win.js',
   my_table:the_table
});

win_to_open.open(); // or how you open the window.

    and in the_file_for_the_win.js

    var win = Titanium.UI.currentWindow;
var the_table = win.my_table;

win.add(the_table);

    Let me know if it's clear enough :)
