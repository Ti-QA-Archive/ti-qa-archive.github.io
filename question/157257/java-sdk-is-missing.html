    Hi renu bhramanna,

    You can try this way i think it would be helpful for you.

    • Right-click the My Computer icon on your desktop and select Properties.

    • Click the Advanced tab.

    • Click the Environment Variables button.

    • Under System Variables, click New.

    • Enter the variable name as JAVA_HOME. Then JAVA-HOME=C:\Program Files\Java\jdk1.7.0_17
      or C:\Program Files (x86)\Java\jdk1.7.0_17

    (note: this is your computer jdk path where you installed it )

    Again click new

    • Again Enter the variable name as path Then path = C:\Program Files\Java\jdk1.7.0_17\bin
      or C:\ Program Files (x86)\Java\jdk1.7.0_17\bin

    • Click OK.

    • Click Apply Changes.

    You might need to restart windows.

    I think JDK problem would not show again. Now you can run your project.
