I have a series of view's, how to target a click eventlistener?

I have a series of views stacked inside of a scrollview.

I have an event listener so that when you click on a view a popup will appear, my question is how do i derive data specific to each view in the eventlistener? It seems to function in a non-targeted way currently.

Is there basicly a way to assign a variable to each of the views that can be accesses in the eventlistener?

THanks!

    In the 'click' event listener, if you set it up with an event parameter, will have your specific view contained in the e.source property. For exmaple:

    view.addEventListener('click', function(e) {
    // e.source is the view you clicked on
}

    You can add any custom properties you want to a View just by adding during creation, or at any other point. Just make sure your property name does not override any existing properties on the View object.

    var view = Ti.UI.createView({
    height:100,
    width:100,
    myCustomProperty:'value',
    anotherCustomProperty:12345
});

// Or later in your code

view.myCustomProperty = 'value';
view.anotherCustomProperty = 12345;
