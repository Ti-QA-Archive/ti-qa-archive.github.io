Alloy - access global properties in .tss file

Let`s say I would like to set backgroundColor to red and use it across my app globally. I do something like this

Alloy.js 

Alloy.Globals.color = "red";

Certainly, I am able to access it in index.js. How can I access it in index.xml file ?

index.xml

<Alloy>
    <view backgroundColor="Alloy.Global.color" />  //This is definitely not gonna work
</Alloy>

Any idea?

Thanks in advance.

    <Alloy>

</Alloy>

  • Accepted Answer

    If you're talking about a global-style-sheet, the best method is to create a file in the 'styles' folder and name it 'app.tss', this is the top level style, anything you code in here will be read first in the stack.
