iPhone - Disabling bounce and hide scrollbars in webviews

Hi all,

This is an answer rather than a question, and thought I'd share it here!
I wanted to be able to disable bounce and scrollbars on my webviews to give a more 'native' look to my app (I had fixed-position elements in my HTML and bounce ruined that illusion so I wanted to disable it).

I found the answer to the 'bounce' effect HERE so all thanks and credit go to the author there, however if you alter the code very slighly, you can also hide the scrollbar.

The file you need to edit is x.xx&#x2F;iphone&#x2F;Classes&#x2F;TiUIWebView.m where x.xx is the version of the SDK you're compiling against

The new code should look like:

#define PAGE_VERTICAL_BOUNCE false
        if (!PAGE_VERTICAL_BOUNCE) {
            for (id subview in webview.subviews) {
                if ([[subview class] isSubclassOfClass: [UIScrollView class]]) {
                    ((UIScrollView *)subview).bounces = NO;
                    ((UIScrollView *)subview).showsVerticalScrollIndicator = NO;
                }
            }
        }

If you wish to hide the horizontal scrollbars, then just add:

((UIScrollView *)subview).showsHorizontalScrollIndicator = NO;

After the vertical scroll indicator line.
