var win3 = Titanium.UI.createWindow({

var buttonalarmlist = Titanium.UI.createButton({

When Im trying to add buttonalarmlist, like in my code, the buttonalarmlist displays under the win3.

    Hi Vladimir,

    when you said header you mean the top toolbar is it? if that's what you mean, try this sample code: 

    var button = Titanium.UI.createButton({
    title: 'My Button'
});

var toolbar = Titanium.UI.iOS.createToolbar({
    items:[button],
    top:0
}); 
win.add(toolbar);
win.open();

    The code above is iOS only. For android you can add buttons to the actionBar. For more information on actionBar look it up http://docs.appcelerator.com/titanium/3.0/#!/api/Titanium.Android.ActionBar
