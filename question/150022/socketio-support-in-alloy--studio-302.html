Socket.io support in Alloy / Studio 3.0.2?

I'm new to Titanium and am working on an app using Studio 3.0.2 and Alloy. It all makes sense to me and I can deploy a test app, etc. I have a server (running locally for now) running a node.js script and socket.io. I'd like to get my Titanium app to use socket.io (or simply web sockets) to communicate with the server. So far, I've not had any luck. Socket.io-titanium was made for Titanium 1.8.x it seems, and I can't get it to play with TI 3.0.x. I've installed the TiWS module and while I can create a websocket client, I can't get it to successfully connect and communicate with my socket.io server. The server simply logs "debug - destroying non-socket.io upgrade" which says to me that it's rejecting the websocket request for some reason.

So, I suppose that ideally, Titanium would support socket.io natively in the near future; is that in the cards? Failing that, is anyone aware of a socket.io-compatible module that works with 3.x? Thanks for any help or guidance.
