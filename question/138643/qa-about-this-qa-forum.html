Q&A About This Q&A "Forum".

This question sounds like I'm being a jerk… because I am. This has got to be one of the worst forums/knowledge base/whatevers that I have, attempted to, use. What makes it worse is that there is no link to find help on using this thing. Can I use booleans? Can I use parenthesis? How do I include tags? How do I exclude tags? How do I ensure that a word has to be in the result? How do I sort my results by votes? How do I sort the results by time? How do I sort results by the number of replies?

I can go on and on.

Please either:

1) Tell me I am stupid and then show me why.
2) Provide some help to make this Q&A thing usable.
3) (My preferred choice) Scrap this crap and get something better.

Trying to learn Titanium is frustrating enough. Having convention defying API documentation and confusing "help forums" makes it not only more difficult, but even more frustrating.

I would love for people to reply with +1 or -1 to let the people in charge know if this is just one idiot complaining or if it is a consensus of the community.
