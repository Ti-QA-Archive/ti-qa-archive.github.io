accesing tableview.data[0].rows crashes the application

I have a simple table and some rows and I'm trying to loop through all the rows..
When I do a for through tableview.data[0].rows my application crashes..

here's my code

.......

var tableData = [];
for (var i in dataDB ) {
 var row1 = Ti.UI.createTableViewRow({
            height:40,
            id:dataDB[i].id
  });

tableData.push(row1);
}

var tableview = Titanium.UI.createTableView({
    data:tableData,
    backgroundColor:'white',
    rowBackgroundColor:'white',
    height:370,
    top:0,
    editable:true,
    moveable:true
});

MainView.add(tableview);
.....

Now if I access the table data for rows it crashes…
Still I'm able to get the number of rows into the default section

Ti.API.info(tableview.data[0].rowCount);
//this actually outputs the actual number of rows

Ti.API.info(tableview.data[0].rows);
//this crashes the app with the following error

//message = "-[__NSCFNumber stringByReplacingOccurrencesOfString:withString:]: 
// unrecognized selector sent to instance 0x9968670";

Please help
