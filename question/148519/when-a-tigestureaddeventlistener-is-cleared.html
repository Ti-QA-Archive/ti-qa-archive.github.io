    You need to use removeEventListener to unhook the listener and release and resources. To do this effectively you need to make your listener a separate function, so;-

    instead of 

    
Ti.Gesture.addEventListener('orientationchange', function(e){

    ..... your code
});

    Do this

    
function handlerOrientationChange(e){
    ... your code
}
Ti.Gesture.addEventListener('orientationchange', handlerOrientationChange);

    Then you can…

    Ti.Gesture.removeEventListener('orientationchange', handlerOrientationChange);
