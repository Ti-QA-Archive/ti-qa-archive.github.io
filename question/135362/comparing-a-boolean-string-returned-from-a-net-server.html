Comparing a boolean string returned from a .net server

Hi guys,

I'm trying to find whether a string I get back from a server running .net code represents true or false. The string returned is simply a bool cast with a call to .ToString(). I'm alerting what's returned, and it shows "True," but I can't find something to compare it to that returns true.

Any advice?

    var result = "True,";
result = (/true/i).test(result);

alert(result);
alert(typeof result);
