The following is the code I used to create textField.

What's wrong ? with me , or Ti 3.1.3 ?

:(

Ti.UI.createTextField({
        value: value,
        height:'auto',
        width:'auto',
        color:'#888',
        keyboardType:Titanium.UI.KEYBOARD_DEFAULT,

        borderStyle:Titanium.UI.INPUT_BORDERSTYLE_ROUNDED,
        font: {
            fontFamily: (osname === 'android') ? 'Droid Sans' : 'Helvetica Neue',
            fontSize: 14
        }
    });

  • The following code CAN enable the Chinese input. 

        var stationField = Titanium.UI.createTextField({
        left: 10,
        width: 300,
        top: ROW_FIELD_PADDING,
        bottom: ROW_FIELD_PADDING,
        hintText: L('input_hint_station'),
        font: INPUT_FONT,
        color: INPUT_COLOR,
        value: stationObj.name,
        color:'#888',
        keyboardType:Titanium.UI.KEYBOARD_DEFAULT,
        returnKeyType:Titanium.UI.RETURNKEY_DEFAULT,
        borderStyle:Titanium.UI.INPUT_BORDERSTYLE_ROUNDED,
        font: {
            fontSize: 14
        }

    });
