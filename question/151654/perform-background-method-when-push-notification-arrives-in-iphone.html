Perform background method when push notification arrives in iPhone

Hi there,

Is there any way to execute one method in the background when a push notification arrives? (not when the user opens the app or clicks the alert).

In android you can do this easily because you can define what is going to be executed in the background when a push notification arrives and also totally customize your alert message there if you want any. More work but much more options if i'm not wrong.

Thanks in advance.

    Hi

    That is how iOS push notifications work, the callback is ONLY fired;

    • if - the app is currently running and in focus
    • if - the app is not running or not in focus then only if
      • the user selects the push notification when it first comes in
      • the user selects the push notification later from the notification panel
