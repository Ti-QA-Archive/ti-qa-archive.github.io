  • Accepted Answer

    If it is a locally loaded html, it can listen to Ti.App events. From the controller you Ti.App.fireEvent() and the html file listens for it.

    Depending on how much interaction needs to take place, I often create an html file as a kind of template with placeholders {--variable1--}. Load that in via Ti.Filesystem and replace the placeholders with actual data.

    I do it this way primarily because it becomes a dance to know if the webview has completely loaded and is ready to listen for an event.
