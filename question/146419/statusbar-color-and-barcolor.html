StatusBar color and barColor

Am I right in thinking that it ought to be possible to set the StatusBar of an iOS 6 app to one of it's standard styles and then be able to set a window's Nav bar's colour (we spell it like that in the UK, mad people that we are) to say dark blue without the StatusBar automatically taking on the same dark blue as the Nav bar?

In iOS 5 the status bar remained the same colour, no matter what the Nav bar is to set to. In iOS 6 it doesn't. And trying to force it by setting the StatusBar to Titanium.UI.iPhone.StatusBar.OPAQUE_BLACK doesn't work either. Should it?

And yes I am using 'color' where appropriate ;)

    Hi

    In iOS 5 and earlier it was possible to change the status bar between a grey variant and a black version, however that changed in iOS 6. Now the status bar bleeds a complementary colour (UK bloke as well) to the bar colour, this is automatic.

    In the past, I used the tiapp.xml property to change the colour/style but I have not tweaked it recently so not sure if it still exits or makes any difference.

    • Update;

      I have played with this and you can have a black status bar, but not the old silver/grey one.

      In tiapp.xml you set the following setting;

      <statusbar-style>opaque</statusbar-style>
  or
<statusbar-style>opaque_black</statusbar-style>

      These both appear to look the same for me.

      Leaving this setting with the detault;

      <statusbar-style>default</statusbar-style>

      Allows the automatic colour bleed.

      So you now have two choices, colour bleed or black. Just remember in iOS 5 and lower default was the silver/grey look.

      Does this answer your question?

    • Quick thought - have you performed a Project > Clean since making the change?

      Make sure you are out of any simulators and emulators, then use the menu; Project > Clean.

      Re-run your code with the following settings in the tiapp.xml;

      <statusbar-style>opaque</statusbar-style>

      Make sure you have removed any other code versions of this call in your code for example (Titanium.UI.iPhone.StatusBar.OPAQUE_BLACK).

      The following settings render as black;

      • opaque
      • opaque_black

      The following settings render as bleed

      • translucent_black
      • gray
      • grey
      • default

      Here are the two variations I have just ended up with

      • Black Status Bar Version
      • Bleed Status Bar Version

      I am using Ti SDK 3.0.0 GA and iOS 6.0 (4.3 min)

      I have also confirmed that the black status bar displays correctly on;

      • iPhone 3.5" Simulator
      • iPhone Retina 3.5" Simulator
      • iPhone Retina 4" Simulator
