  • Hi Arjan,

    you are using Alloy Infinite Scroll widget Version 1.3.1 from Fokke Zandbergen.

    Fokke just released Version 1.3.2: Workaround for regression in Alloy 1.3.0-cr on github.

    This should fix Jira TC-3417 Regression: Widgets in TableView don't get __parentSymbol.

    Would be best to test with Alloy 1.3.0-cr.

    npm install -g alloy@1.3.0-cr

    Greetings, Matthias
