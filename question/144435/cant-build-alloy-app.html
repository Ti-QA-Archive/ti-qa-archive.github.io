  • I'm assuming you are using a version of TiStudio before the 3.0 RC? It has an issue where it does not appropriately pull in your PATH when compiling Alloy. If you upgrade to the 3.0 RC it should solve your problem. If you don't want to upgrade TiStudio…

export PATH=$PATH:ALLOY_PATH
