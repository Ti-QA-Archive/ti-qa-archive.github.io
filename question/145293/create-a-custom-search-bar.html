Create a custom search bar

Hello to all you guys!

Let me explain now.
So, I write on a blog that is part of a community that includes 4 blogs and a forum.
I wanted to create a sort of mini browser, perhaps using a search bar, in order to search for a particular word / phrase in the various community blogs to which they belong.

I could do something like that or is unthinkable?

    Hi

    If you are asking if you can use the search bar to search for things, then yes, if your only experience is using it with a table - it can certainly and easily provide you back a search phrase.

    If you are asking if you can search for content on a page then - yes, you can either;

    • download the content of the four blog pages using http client after each search, the parse the content to match the search phrase.
    • use yql (api is accessible by Titanium), this allows you to query a page and look for specific sections of content that you can then easily return using XML or JSON (my preference). Google YQL console and play with the examples (right side in drop downs)

    You should only fire off the calls after the search return event is called as the downloading and checking will take a few seconds or more.
