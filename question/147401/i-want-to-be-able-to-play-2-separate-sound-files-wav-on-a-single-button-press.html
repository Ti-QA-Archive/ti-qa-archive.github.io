    Hi

    You say you want to have the user press the button to play a single sound and then they press the button again later and it should play a different sound. If that is the case this code should get you started.

    var sounds = [
    "/one.wav",
    "/two.wav"
];
var soundindex = 0;
var btn = Ti.UI.createButton({
    title: 'Open'
});
btn.addEventListener('click', function (e) {
    if (soundindex === sounds.length) {
        soundindex = 0;
    }
    Ti.Media.Sound.volume = 0.5;
    var player = Ti.Media.createSound({
        url: sound[soundindex]
    });
    player.play();
    soundindex = soundindex + 1;
});

    It has an array of sounds (two in this case) and plays the first one on the first button press and then increments the index value, next time a button is pressed it plays the next sound. If the index is greater than the elements it resets to 0.

    Adapt this to suit your needs.
