    Took me a while but i figured it out.

    You declare your Button with the image var before the image var is declared. So your button wants to add a image from a variable that does not exicst in its content.

    What you need to do is to set the button backgroundImage after you know the image variable.

    So in the click even you need to do this

    Remove the backgroundImage initially:

    var show = Ti.UI.createButton({ 
  title:'Show picker', bottom:20, width:50, height:50
});

    Then add the image after you declared the image and the variable has been set with an image.

    show.backgroundImage = image;
