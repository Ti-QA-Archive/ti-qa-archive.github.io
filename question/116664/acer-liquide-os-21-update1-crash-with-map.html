    Andreas

    Are you able to use the Titanium.Map independently of the Geo features?

    Note that the underlying functionality accessed using Titanium.Map is provided by the enhanced Google API, which is shown in the Android SDK under "Installed Packages" as "Google APIs by Google Inc.". On the Run Emulator screen of Titanium Developer, each respective version is listed as "x.y APIs". Unfortunately, not all devices are licensed to use them, and in these cases the enhanced features won't work.

    This is explained in the Preparing for Android development guide.

    It's probably a good idea to inquire with Acer to find out whether this is the problem in your case.

    Cheers
