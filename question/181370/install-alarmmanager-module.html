I want to use the alarmmanager module for Android local notification from benCoding. I downloaded the file from: https://github.com/benbahrenburg/benCoding.AlarmManager/blob/master/dist/bencoding.alarmmanager-android-0.11.zip and compile it too .zip.

In Titanium Studio -> Help -> install new mobile module, but the module is not shown in the mobile moduledialog when i want too add it. What is the problem?

    Hi Joost ,

    Thanks for your concern .

    You can install alarm manager module manually also . Please follow the steps :

    1. Download benCoding.AlarmManager
    2. Unzip it .
    3. dist > bencoding.alarmmanager-android-0.11.zip (unzip it).
    4. modules > android > bencoding.alarmmanager (copy this)

    5. Past bencoding.alarmmanager folder following path

      In OSX   /Library/Application Support/Titanium/modules/android 
In Windows  C:\ProgramData\TitaniumSDK\modules\ android

    6. Add the module in your application's tiapp.xml file.

    7. Require module in js file
    Example : var bencoding = require('bencoding.alarmmanager');
