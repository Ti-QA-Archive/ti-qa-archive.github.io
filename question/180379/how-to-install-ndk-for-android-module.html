  • Hi Bunty Rohra,

    Set Android NDK path using below steps.

    To use the Android NDK with Titanium Studio, the path to the NDK must be set in the Preferences dialog.

    Titanium Studio > Preferences > Studio > Platforms > Android.

    Enter the path to your Android NDK in the Android NDK Home textbox or click the Configure button
