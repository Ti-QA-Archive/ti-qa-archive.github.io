how to remove tab bar header?

Hi All,
Hopefully a simple question that you may be able to help me with.

I have a tabgroup with 2 tabs.Each tab loads in a page.
Each page/tab, by default, has a header title in a bar at the top of the page.
I want to switch off this title bar as it's not required for my design (afterall, i still have the tab titles at the bottom of the screen).

I have tried different options but so far no luck.

My latest attempt, which also does not work ;-) is:

The tabgroup setup in app.js

// create tab group
var tabGroup = Titanium.UI.createTabGroup();


// create base UI tab and root window
var win1 = Titanium.UI.createWindow({  
    title:'isiZulu',
    backgroundColor:'#fff',
    url: 'zulu.js',
});
var tab1 = Titanium.UI.createTab({  
    icon:'KS_nav_views.png',
    title:'isiZulu',
    window:win1
});

var win2 = Titanium.UI.createWindow({  
    title:'isiNgisi',
    backgroundColor:'#fff',
    url: 'english.js',
});
var tab2 = Titanium.UI.createTab({  
    icon:'KS_nav_ui.png',
    title:'isiNgisi',
    window:win2
});

and then the header information in e.g. tab 2, english.js

var win = Titanium.UI.currentWindow({
    fullscreen: true, 
        navBarHidden: true,
})

Any help us much appreciated.
Thanks,
Lee

    You cant set the parameters on currentWindow - only when you create it.

    var win = Titanium.UI.createWindow({
        navBarHidden: true
})

win.open();

  • I just added the code below in my TiApp.xml to hide the action bar that is where the name of my app and logo of Titanium where

    <android xmlns:android="http://schemas.android.com/apk/res/android">
    <manifest>
    <application android:theme="@style/Theme.Titanium"/>
</manifest>

    You can hide the status bar too (where the clock, battery level, notifications, etc. are) by adding the statusbar line

    <fullscreen>false</fullscreen>
<statusbar-hidden>true</statusbar-hidden>
<analytics>true</analytics>
