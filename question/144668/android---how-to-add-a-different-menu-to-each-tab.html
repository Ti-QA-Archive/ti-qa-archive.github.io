Android - How to add a different menu to each tab?

As the documentation says, each window created inside a TabGroup is a heavyweight window.
Although, I tried to add different menu entries to each window, but the menu in every tab is the same as the first tab opened… as if they where the same activity.

Did I get something wrong or is this a bug with the implementation/doc?

  • what i do is (where winNuevoSoporte is the window )

    var activity = winNuevoSoporte.activity;
    activity.onCreateOptionsMenu = function(e){
      var menu = e.menu;
      // Cerrar
      var mnuCerrar = menu.add({ title: "Cerrar" }); 
      mnuCerrar.icon = Titanium.Android.R.drawable.ic_menu_close_clear_cancel;
      mnuCerrar.addEventListener("click", function(e) {  
        winNuevoSoporte.close();
      });
      });
    };

    hope this help…
