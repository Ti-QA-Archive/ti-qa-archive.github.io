  • the problem is in the statement - font: {fontFamily: 'Helvetica-Bold', fontSize: 15}; JSS doesnt recognize this way of defining font properties… it shoud be

    b
    {
    width: 40;
    height: 50;
    color: '#324f85';
    font-family: 'Helvetica-Bold';
    font-size: 15;
    }

    That should work
