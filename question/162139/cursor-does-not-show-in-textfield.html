  • i found the solution !!!

    Here is the explanation

    My app is a TABed one.. so, i declare 4 tabs, but, to make it in iOS 7 style, i use the next code to create the TABS:

    //
//
var color_barra_titulo = '#5daf2b';
var color_tinte_nav = 'white';
//
//
//
var tabGroup = Titanium.UI.createTabGroup( { tabsTintColor: color_barra_titulo, tintColor: color_tinte_nav } );

    That create a white tabs with the icons painted in green…
    But, the textFields takes the tintColor from the tabGroup, making the cursor white, just like the background of them AND THAT'S WHY they look "invisible".

    The only thing i add to my source code was:

    var textUsuario = Ti.UI.createTextField({
  font: fuente_captura,
  autocorrect: false,
  keyboardType: Titanium.UI.KEYBOARD_ASCII,
  borderStyle: Ti.UI.INPUT_BORDERSTYLE_ROUNDED,
  autocapitalization: Ti.UI.TEXT_AUTOCAPITALIZATION_NONE,
  tintColor: 'black', // This make visible the cursor
  value: usuario,
  top: '4%', 
  height: 35,
  left: '35%',
  width: '60%', 
});

    And "ta daaaaaa" … magic!!!

    Thanks for your interest… i hope this can be useful for some one else.
