    You have a few problem here;

    • ScrollAbleView - needs better defaults
    • You have not understood the simplicity of the layout solution
    • Your ScrollViews should have a horizontal layout (trust me)

    Here is some code to gain the logic of the solution - you should be able to work out what do do from this;

    • First we get the scrollableView default to work
    // I would normally place this at the bottom
// after all pages are created, or at least the first batch
var main = Ti.UI.createScrollableView({
    bottom: 120, // leave a gap at the bottom if you need
    contentHeight: Ti.UI.SIZE,
    contentWidth: Ti.UI.FILL,
    height: Ti.UI.FILL,
    top: 80, // leave a gap at the top if you need
    width: '70%'
});

    I hate percentages - this design should use fixed numbers and not percentages in order to work out the margins correctly

    • The pages themselves, these need to be horizontally aligned internally, this may sound weird, however, you layout consists of narrow and wide content and they flow on from left to right.

      If you treat each SEPERATE story as something distinct then simply add each story to each page one after the other then it all works.

      They will follow like this; narrow, wide, narrow, wide, narrow, narrow, narrow, narrow, narrow, narrow - when shown on the page;

    narrow, wide, narrow
wide, narrow, narrow
narrow, narrow, narrow, narrow

    As each one effectively take up either 25% of the space or 50% then each row = 100% of the available space (keep in mind use numbers - percentages are only representative). When the layout gets to the end of that row, it moves down to the next automatically.

    // each page view
var page = Ti.UI.createView({
    height: Ti.UI.FILL,
    layout: 'horizontal',
    width: Ti.UI.FILL
});
    • Then create a function to add each story by itself
    var addStory = function (obj) {
    // you set the based on is wide or not
    var outer = Ti.UI.createView({
        height: 200, // you set this
        width: obj.isWide ? 180 : 90
    });
    var lbl = Ti.UI.createLabel({
        bottom: 10,
        height: Ti.UI.SIZE,
        left: 10,
        text: obj.title || 'No title provided',
        width: Ti.UI.SIZE
    });
    outer.add(lbl);
    // add whatever else you need

    return outer;
};
    • Then when you loop simply add one story per item and choose if you want wide or or not, example usage;
    page.add(addStory({
    isWide: false,
    title: 'Left'
}));
page.add(addStory({
    isWide: true,
    title: 'Middle and wide'
}));
page.add(addStory({
    isWide: false,
    title: 'Right'
}));
page.add(addStory({
    isWide: true,
    title: 'Next line, left and wide'
}));

    Obviously this will be in a loop and pages will be an array.

    • One of the video pages had a scrollView on the right - this is a separate view - not linked to this one in creation, although the scroll event might tie the motion of the two together.

function getCategoryViews() {

    return category_views;

}


function setActiveIndex(index) {
    scrollableView.scrollToView(index);
}

function Init() {

    var Tabs = require('ui/Tabs');

    var categories = Tabs.getCategories();

    var activeIndex = Tabs.getActiveIndex();

    for (var i in categories) {

        /* Load articles from this category here */
       var articles = ['the title'];

        var view = Ti.UI.createView({
            backgroundColor : 'black',
            layout : 'horizontal'
        });

        var featured_articles = featuredArticles(articles, (activeIndex === i) ? true : false);

        var article_list = ArticleList(articles, (activeIndex === i) ? true : false);

        view.add(featured_articles);
        view.add(article_list);

        category_views.push(view);

    }

    scrollableView = Ti.UI.createScrollableView({
        showPagingControl:false,
        height : '60%',
        views:category_views,
        currentPage : activeIndex
    });

    scrollableView.addEventListener('scrollend', function(e) {

        Tabs.setActiveIndex(e.currentPage);

    });


    return scrollableView;

}


exports.init                = Init;
exports.getCategoryViews    = getCategoryViews;
exports.setActiveIndex      = setActiveIndex;
