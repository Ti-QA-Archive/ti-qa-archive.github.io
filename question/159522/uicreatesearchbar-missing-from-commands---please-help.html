  • Hi

    Based on your development platform of Windows you must be using; Android, Blackberry or Mobile Web.

    If you are targetting Android then you should have no problems, but if you are targetting Blackberry or Mobile Web - then you are out of luck as it is not supported on those platforms. You can check this in the documentation;

    • SearchBar

    The top right shows the supported platforms.

    This code is all you need - but it is what you are already using;

    var search = Titanium.UI.createSearchBar({
    showCancel: true,
    height: 43
});
