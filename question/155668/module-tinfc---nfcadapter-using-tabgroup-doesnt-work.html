Module ti.nfc - NfcAdapter using TabGroup doesn't work

Hello,
I have an issue using ti.nfc module with an app that use a TabGroup.

It Works [index.xml]

<Alloy>
<Window title="Tab 1">
</Window>
</Alloy>

It doesn't Work [index.xml]

<Alloy>
        <TabGroup>
            <Tab title="Wally Gym" icon="KS_nav_views.png">
                <Window title="Tab 1" id="gymView">
                </Window>
            </Tab>
        </TabGroup>
</Alloy>

Index.js han been copied from TagForeground Example provided by NFC Titanium example.

The application uses the following code to start NFC adapter: 

$.index.addEventListener('open', function(e) {
    // Must wait until the activity has been opened before setting up NFC
    setupNfc();
});
$.index.open();

If I am using a simple Window the app is able to read the NFC tag and show the content into an "alert" view.

Using a TabGroup the mobilephone asks if this is a new protocol, and the application doesn't "catch" the NFC Tag.

Thanks for any hint

  • Hi,
    I solved the problem changing the method used to get the activity:

    From

    var act = Ti.Android.currentActivity;

    to 

    var act = $.index.getActivity();

    Bye
