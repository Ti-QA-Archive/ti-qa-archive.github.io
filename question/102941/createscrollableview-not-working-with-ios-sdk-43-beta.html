createScrollableView not working with iOS SDK 4.3 beta

UPDATE: This has been fixed already.

Titanium.UI.createScrollableView crushes (and always re-produceable) when iOS SDK 4.3 (now in beta) is selected. Although the same code works fine with other SDKs.

Ti.API.info(view_list);
// output:
// [INFO] (
//  "[object TiUIView]",
//  "[object TiUIView]",
//  "[object TiUIView]",
//  "[object TiUIView]",
//)

var scrollableView = Titanium.UI.createScrollableView({
  views:view_list,
  showPageingControl: false,
  currentPage:0
});

win.add(scrollableView);

See also: Backtrace.
