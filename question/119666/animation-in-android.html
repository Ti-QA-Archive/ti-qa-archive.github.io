Animation in Android

I have animated a view in android. This same view's animation is working perfectly in iphone but when i try to animate in android with same code it's creating some problems.

Animation is performed but when view animation is in progress the background window is displayed in portion of that view.

for example …
I have two java script file (two window) … window1 , window2

on click of window1 i am opening window2 in which i have added one view.
I am animating view (simply changing width from 200 to 1).
At time of this animation the background window's portion is displayed in the part of that view. i mean not the portion of window2 but its displaying window1's portion in the place of that 200 * 200 view rectangle.

any one having idea why such things is happing in android. Same animation working perfectly in iphone.
