  • Since you are using Titanium SDK 3.1.0 and earlier, Android SDK Tools r22 may break your Android builds as described in TIMOB-13944. To workaround this issue, either:

    • Create symlinks between the old and new locations of the test tools:
    cd /Applications/Android-sdk/platform-tools
ln -s /Applications/Android-sdk/build-tools/17.0.0/aapt aapt
ln -s /Applications/Android-sdk/build-tools/17.0.0/dx dx
    • Download the previous version of the tools:
    1. Linux: http://dl.google.com/android/android-sdk_r21.1-linux.tgz
    2. OS X: http://dl.google.com/android/android-sdk_r21.1-macosx.zip
    3. Windows: http://dl.google.com/android/android-sdk_r21.1-windows.zip

    Let me know how it goes, thanks.
