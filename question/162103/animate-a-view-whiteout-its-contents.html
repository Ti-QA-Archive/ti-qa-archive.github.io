Animate a view whiteout its contents

How can I animate a view height without its contents?

On iOS it works just fine. The view is animating and the contents remain at original size, but on Android it doesn't matter how I do it, the contents also get scaled.

I have tried that 2DMatrix.scale and tries in Titanium 3.2.0, 3.2.1, 3.3.0. the problem is the same.

Here is a sample code to demonstrate what I need:

win = Titanium.UI.createWindow({
    fullscreen: true,
    layout: 'composite',
    backgroundColor: 'white',
});

win.open();

var inner = Ti.UI.createView({
    height: Ti.UI.SIZE,
    width: Ti.UI.SIZE,
    top: 50,
});
win.add(inner);
var lbl = Ti.UI.createLabel({
    height: Ti.UI.SIZE,
    width: Ti.UI.SIZE,
    text: 'This Label is HERE',
    font: {fontSize: 50},
    color: 'black',
});
inner.add(lbl);

var a = Ti.UI.createAnimation({
    height: 150,
    duration: 2000,
});

inner.addEventListener('click',function(){
    inner.animate(a);
});

Just to be clear I want the Label (lbl) to stay at original size (50) during and after the animation.

  • The label is sizing in accordance with the parent view. So if you animate the view, it will also resize it's contents.

    You will need to fix the size of the label.

    Also animations really only work if you have the property you are trying to animate set within the object. So in the view you set height to Ti.UI.SIZE, it is bester if you set height to an actual value.

    T.
