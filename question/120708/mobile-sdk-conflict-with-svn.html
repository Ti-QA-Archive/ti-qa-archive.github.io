  • The answer is to edit the file "…/osx/1.6.2/android/builder.py" by adding under the line 468 the following code:

    if orig.find('.svn')>=0:
    trace("FOUND .svn IN NAME: %s" % orig)
    return

    In the solution provided above, there was no ">=0", which broke the build!
