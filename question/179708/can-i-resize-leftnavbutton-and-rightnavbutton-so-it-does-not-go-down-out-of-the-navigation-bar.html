Can I resize LeftNavButton and RightNavButton, so it does not go down out of the navigation bar?

Hi

I am working with navigation on iOS and I am using the LeftNavButton and RightNavButton. My issue is that I really want to change the height of the click area, since it reaches down below the actual navigation bar.

I have tried with height on both the label and the nav button container (LeftNavButton && RightNavButton), which did not work.

Is there any solution for this issue, since ListViews (that goes all the way to the top) have a hard time actually clicking the first list element. 

<Alloy>
    <NavigationWindow>

            <LeftNavButton id="iosNavLeft">
                <Label id="iosLeftNavBtn" class="navigationBarBtn">L('pin_forgot_pin')</Label>
            </LeftNavButton>
            <RightNavButton id="iosNavRight">
                <Label id="iosRightNavBtn" class="navigationBarBtn">L('change_user')</Label>
            </RightNavButton>
