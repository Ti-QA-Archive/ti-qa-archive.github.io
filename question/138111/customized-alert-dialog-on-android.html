Customized alert dialog on Android

Hi,
I try to build a customized dialog with round corners, background image and border.
It works fine on apple but not on Android.
Errors:

  • no border,
  • corners are not transparent, has color of border.

Here my code:

var width = Ti.Platform.displayCaps.platformWidth;
var myalert = Ti.UI.createView({
    borderRadius : 4,
    borderColor : 'silver',
    borderWidth : 2,
    width : width * 0.8,
    height : width * 0.6,
    backgroundColor : 'transparent'
});
myalert.bg = Ti.UI.createImageView({
    image : '/assets/alert/bg.png',
    width : Ti.UI.FILL,
    height : Ti.UI.FILL,
});
myalert.add(myalert.bg)
win.add(myalert);

Any ides?
