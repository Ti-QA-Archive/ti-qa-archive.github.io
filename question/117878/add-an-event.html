add an event

Hi!

I try to figure out why this don't work:

var valid = Ti.UI.createButton({
    width:87,
    height:37,
    color:'blue',
    top:'90%',
    left:'70%',
    title:'valid'
});
valid.addEventListener("valid",function(e){
    alert("you just click submit");
});
win1.add(valid);

Normally, when i click on the submit button, it must show the message….but it doesn't. have I something wrong?

Thanks for your help

    There is no event named "valid".

    valid.addEventListener("click",function(e){
    alert("you just click submit");
});

    In the docs you will find a list of events for each type of UI element.
