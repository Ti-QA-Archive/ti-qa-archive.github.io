  • Accepted Answer

    You can do that with an Intent as described here:

    var intent = Ti.Android.createIntent({
    action: Ti.Android.ACTION_SEND,
    type: "text/plain"
});

intent.putExtra(Ti.Android.EXTRA_TEXT, 'Some text that we want to share');
intent.addCategory(Ti.Android.CATEGORY_DEFAULT);
Ti.Android.currentActivity.startActivity(intent);

    • OK. Let me explain it a little further…

      Starting an Activity with a "sharing" - Intent like this makes the system look for Apps that can receive the data you want to share. If it there is an app like this, it will be started by this Intent. But if the system finds more than one App then it lets the user choose which App he wants to use for this action.

      In case of the emulator, there is only one app (sms/mms). There is no Facebook, Gmail or Twitter App installed on the emulator - that's the reason why the system don't let you choose these apps.

      Just test the same code on a device, where you have those apps installed, it will work as you expect ;-)
