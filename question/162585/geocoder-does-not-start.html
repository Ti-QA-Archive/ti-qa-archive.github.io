  • In the tiapp.xml, change the api version to something that you have installed, like 17 for example. So in the manifest section you'd have

    <uses-sdk android:targetSdkVersion="17"/>

    and make sure you have the appropriate sdk loaded, like

    <sdk-version>3.2.1.GA</sdk-version>
