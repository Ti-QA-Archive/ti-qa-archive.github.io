  • Hi

    I think you are over thinking this;

    • If you are wanting to add jQuery to benefit a web page you load into a webView - then add it as a script into that page - the exact same way as you would in a web site.
    • If you want to use jQuery to access native elements like labels, buttons, views and tableViews then you cannot as they are two VERY different things and are unrelated to each other.

    • If you are asking if you can use sencha touch in a titanium app - the exact same rules apply as I noted above for jQuery. Basically the Titanium API is the only thing that affects Titanium. But if you choose to include local web content as well as your app then all normal web page script libraries will run. Obviously how they look/work inside a mobile browser will depend on how well those libraries normally look/work in mobile browsers.

      Keep in mind your app cannot simply be a series of web pages - the app needs to be more than that.
