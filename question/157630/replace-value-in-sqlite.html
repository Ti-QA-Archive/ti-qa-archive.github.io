Replace value in sqlite

Hi.

I wonder if it is possible to replace a value in a sqlite DB if there already is something in the row.

Ok i try to explain.

If i have a DB with 3 columns city, name and date.

Lets say that i put in 2 rows Detroit, Albert, 9/25 & New York, Peter, 10/24

Next time i make an input i will make Detroit, Albert, 9/26

Can i make it so that doesnt make another row but just replace the date to the new date 9/26 instead of 9/25?

I imagine that it would be some sort of "Check if row exists then replace…" procedure

Thanx!

    This can be achieved using the SQL UPDATE statement.

    Here is a link to the documentation

    ###By head, your SQL code would go something like this:

    UPDATE my_table
SET date = '9/26'
WHERE name = 'Albert';

    Hope this helps.
