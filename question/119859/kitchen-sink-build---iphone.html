  • Derek,

    From the error log you posted, it looks like you are trying to build with an older titanium mobile SDK.

    If you go into your project from titanium developer, and click on the edit tab at the top (between "dashboard" and "test & package"), and towards the bottom of the view you will see a drop down for Titanium SDK.

    From this list you will want to select 1.6.2.

    After you do this, try building the app again.

    The kitchen sink app is designed to demonstrate all the API calls that titanium has available, if you try building a newer version of kitchen sink with an older SDK it will error out.
