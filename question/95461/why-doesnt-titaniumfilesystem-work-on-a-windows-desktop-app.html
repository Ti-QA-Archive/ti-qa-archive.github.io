Why doesn't Titanium.Filesystem work on a Windows desktop app?

I am new (today) to Titanium and I am having a tough time getting the Filesystem to work. if I try:

alert(Titanium.Filesystem.applicationDataDirectory)

I get "undefined". if I try something like:

var f = Titanium.Filesystem.getFile("c:\test.txt");
alert(f);

I get what I expect "c:\test.txt".
But if I try and write to the file and exit the program, there is no file. If I check for the file while the program is running, it does not exist. So at times it 'seems' that things are working but they really aren't.

Any help would be very appreciated.

Thanks,
Jim
