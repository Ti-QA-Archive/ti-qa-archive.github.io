Alloy Nested View Event Propagation Not Working

All,

I am unable to receive events on the proper source in a nested Alloy view. Below is the Alloy code and corresponding controller. If I click on the View, I correctly receive "clicked me: container1". If I click on the Label, I incorrectly receive "clicked me: undefined".

I thought events were supposed to bubble up the chain. I even tried setting the "bubbleParent" property but nothing seems to allow me to receive the correct source.

Any help would be greatly appreciated.

<Alloy>
    <Window id="ratingwindow">
        <View id="container1" width="200" height="200" onClick="doClick">
            <Label id="label1">Hello World</Label>
        </View>
    </Window>
</Alloy>


function doClick(e) {
    console.log("clicked me: " + e.source.id);
}

I am using the following: Titanium: 3.1.0.GA, Target: Mobile Web, Alloy: 2.2.0
