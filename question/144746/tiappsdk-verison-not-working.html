    • Michael: If you read the documentation for Ti.App - and view the list of properties available, you will see that there is no such property called 'sdk-version'.

      Ti.App.version provides the version of the actual app you are building, not the SDK version.

      Go here and scroll down the list of available properties to see which properties you can read from tiapp.xml : http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.App
