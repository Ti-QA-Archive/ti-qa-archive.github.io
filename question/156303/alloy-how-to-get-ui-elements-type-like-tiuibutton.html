  • in Alloy

    <Button class="btncancel" id="btncancelddNew"/>

    and

    $.btncancelddNew.addEventListener('click', function(e){

    Titanium.API.info(e.source);

    });

    The click result is below

    [object btncancelddNew]

    But in The other case

    var btn = Titanium.UI.createButton({
    top : 20,
    left : 20,
    title : 'click me ',
    width : 80,
    height : 30
    });

    win.add(btn);

    btn.addEventListener('click', function(e){

    Titanium.API.info(e.source);

    });

    The click result is below

    [object TiUIButton]
