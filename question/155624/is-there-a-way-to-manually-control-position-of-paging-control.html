    • As the paging indicator is simply a set of dots you could always make your own version and place it over the top of the scrollableView.

      This can be achieved by creating a view and adding the same number of labels as there are pages, each label has a caption of a bullet. During the scrollend event you update the colours of the labels to reflect the current state. Make sure the view and the labels have touchEnabled = false.

      I would set the view to have a layout of horizontal and set height to about 20, width of Ti.UI.SIZE.

  • Hi Amaresh, you have to custom the pagingcontrol by yourself.

    I have an example by my own. Hope it could help you.

                var paging = Ti.UI.createView({
                height:15,
                width: Ti.UI.SIZE,
                left:'45%',
                bottom:15,
            });

            var pageLength = yourscrollable.views.length; //get the number of views
            var page = [];

            for (var i = 0; i < pageLength; i++) {
                var pagingDot = Ti.UI.createView({
                    backgroundColor: '#000',
                    borderRadius: '50',
                    height: 10,
                    width: 10,
                    bottom: 0,
                    left: 20 * i,

                });

                page.push(pagingDot);
                paging.add(pagingDot);
                yourscrollable.add(paging);

            }

            page[yourscrollable.getCurrentPage()].setBackgroundColor("#ddd"); //set the initial colour

            // I Attach the scroll event to scrollableView, so we know when to update things
            yourscrollable.addEventListener("scrollend", function (e) {
                if (e.currentPage || e.currentPage == 0) {

                    // Go through each and reset it's color
                    for (var c = 0; c < pageLength; c++) {
                        page[c].setBackgroundColor("#000");
                    }
                    // Bump the Color of the new current page
                    page[e.currentPage].setBackgroundColor("#ddd");

                }
            });
