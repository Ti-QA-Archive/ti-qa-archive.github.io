How to measure the time between two events?

Hi,

I want to implement a button that when the user holds it, it would record the audio from the user. When the user releases his finger, the audio would then be sent. This comes with two events: touchstart and touchend, thanks to Stephen.

The touchstart event will be triggered once the user taps his finger on the button. I also want to record the audio if the user press the button for a while. Because there are chances that the user would mistakenly tap the button, which is called slip in interaction design.

To prevent that, I want to calculate the time between the touchend and touchstart events, if this period is too short, then I wont record. I wonder is there a way to measure the time between two events?

Thanks in advance!
Phil

    On the touchstart event, grab the current time and store for use later. On the touchend event, grab the current time and then subtract the touchstart time. That is the amount of time that has passed.

  • use this code.. credits: http://favo.net/post/27319446540/titanium-touch-hold-to-trigger-multiple-click-events

    // touch start creates a 250ms interval that fires click events
function touchStartClick(e) {
    if ( !e.source.touchTimer ) {
        e.source.touchTimer = setInterval(function () {
            e.source.fireEvent("click");
        }, 250);
    }
}

// touch end cancels the interval
function touchEndClick(e) {
    if ( e.source.touchTimer ) {
        clearInterval(e.source.touchTimer);
        e.source.touchTimer = null;
    }
}

// assign the touch start & end functions
button.addEventListener("touchstart", touchStartClick);
button.addEventListener("touchend", touchEndClick);
button.addEventListener("touchcancel", touchEndClick);
