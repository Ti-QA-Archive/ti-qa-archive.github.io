get co-ordinates of dynamically created objects

Hi,

How can I get the x,y co-ordinates of dynamically created objects?
for example I want the height, width, and center points of the image placed in a image view created dynamically whose left,top or height,width is not defined in the code?

I tried using getCenter, getHeight , getWidth, getLeft , and getRight methods but all returns as Undifened.

Thanks in advance.

    Hi, you should check the rect property of your dynamic created object, in the postlayout event of the main parent object. It comes with rect.width, rect.height, rect.x, rect.y. hth
