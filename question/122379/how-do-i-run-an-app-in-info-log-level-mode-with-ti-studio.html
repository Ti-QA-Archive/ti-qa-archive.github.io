How do I run an app in 'INFO' log level mode with Ti Studio?

How do I run an app in Titanium Studio (in the emulator or simulator) and only get the INFO, instead of the DEBUG mode stuff.

I have an event firing every few seconds, which makes it difficult to read along with the logs. I tried creating a custom run config with the log level set to INFO and I'm still getting back all of the DEBUG log level data.

Ideas?

  • Hi Alex.

    Have you tried Dalvik Debug Monitor? In it you can create filters and filter out all unnecessary data.

    It's located under "android-sdk/tools/ddms.bat" on Windows XP. After you run it, you can easily create filter (green "+") by log tag ("TiAPI"). Now you'll have only output that is printed from your application. Additionally, you can filter by log level (INFO, DEBUG, …).

  • In Ti Studio you should right click the main Project name in the project explorer. This will give you a menu that should have a Properties option at the bottom.

    Select this option should show you a Properties dialog. You should select the Run/Debug Settings option from the list show. This will show you ALL the currently set up run settings for all the projects (don't know why this is).

    But anyway, select the option that applies to your project, probably something like

    Titanium iPhone Simulator - {projectname}

    You can now select the edit button , select the Log Level you want (on the Main tab) and click apply.

    The next time you run your project it should use the new settings.

    Hope that helps.
