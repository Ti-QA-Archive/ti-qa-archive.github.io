Posted a ScreenCast on Titanium JavaScript Interop

I just did two quick screen casts that cover my solution for
interoperability between JavaScript on a remotely hosted Web page and
Titanium hosting that page in a Web View.

This handles the fact that Titanium does not do it's JavaScript injection on pages loaded outside the application sandbox.

I also briefly cover how to get / mod / build the Titanium Mobile
(iPhone/iPad) source on Mac OS X 10.6

Not exactly scintillating but maybe it will help some Titanium dev out there :) BTW, if anyone has a less "hacky" way to do something similar I am all ears.

Screen casts are here:

http://shaunsullivan.posterous.com/titanium-javascript-interop-on-remotely-hoste
