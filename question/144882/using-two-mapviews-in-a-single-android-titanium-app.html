  • hello Rajeev,
    I had same problem and solved by using a trick

    in android application we can use mapView once so

    at your first windows blur event just remove your mapview and then you can add mapview in second window

    same as first window when you click back then remove mapview on second windows blur event and add mapview on first windows focus event…

    so that's way to use mapview 2 times.

    but if you wants to mapview 2 times in same window then it is not possible.
