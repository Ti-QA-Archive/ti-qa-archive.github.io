Error while reading local file on android

I have a local text file in resources that I am trying to read from my app.

var f=Ti.Filesystem.getFile(Titanium.Filesystem.getResourcesDirectory(), 'testfile.txt');

f.exists() —> returns True

f.size —> returns -1

f.read().text —> Exception

I am also not able to run the kitchen sink app's Filesystem section, its getting the same problem.

I am running Titanium 1.7 on windows.

Any suggestions?

  • Ok, figured it out. There is a bug in Titanium 1.7.2 (found in some other thread).

    It is fixed in 1.7.3, which can be downloaded from http://builds.appcelerator.com.s3.amazonaws.com/index.html and doesn't have this problem.
