    • You should not modify any of the built-in Titanium objects, such as Ti.App. Any examples of that represent very outdated antipatterns.

      Your best option is to create CommonJS modules that you can instantiate as needed. There are various patterns, including Singleton patterns, that will let you set and pass values around in your code. As Stephen says, global variables represent bad-practice in programming.
