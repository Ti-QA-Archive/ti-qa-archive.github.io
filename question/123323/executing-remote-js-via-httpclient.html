Executing remote JS via HTTPClient

I would like to collect statistics in my app by executing a remote JS code (important: without any output, so I don't need anything retrieved), as if invoking the JS from any common web page. But the code below doesn't seem to work as no statistics are counted by the JS from my app:

myJS = 'http://my.server.com/my.js/id=.Ro1Z3.r.u7/param=xnlj/arg=00A55A4/id=1243';
httpClient = Ti.Network.createHTTPClient();
httpClient.setTimeout(10000);
httpClient.open('GET',myJS);
httpClient.send();

    • I was concerned a bit if it's not a waste of resources to create a webview just for this, but I have set it for removal on load event so this is actually a perfect solution. Thought about some similar approach before with a hidden image (I also have access to tracking image, not only JS), but I will stick to this webview method.

      To anyone who bumps into the same issue:

      var countWebView = Titanium.UI.createWebView({
    url:myJS,
    visible:false
});
countWebView.addEventListener('load',function(e){
    win1.remove(countWebView);
});
win1.add(countWebView);
