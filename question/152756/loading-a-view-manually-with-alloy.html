Loading a View manually with Alloy

Hi,

How can i load a view manually with alloy?

If i have this view:

index.xml

<Alloy>
    <Window class="container">
        <Button id="load" title="Load" onClick="loadView"></Button> 
        <View id="content"></View>
    </Window>            
</Alloy>

How can i click the button and load the "NewView.xml"?

NewView.xml

<Alloy>
    <View id="reportsView">
        <Label id="label" text="New View" ></Label>
    </View>
</Alloy>

  • Accepted Answer

    index.js

    function loadView(e) {
    var view = Alloy.createController('NewView').getView();
    // you can now add "view" to your view hierarchy
}
