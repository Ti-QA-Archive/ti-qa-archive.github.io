  • unzip the file and move the files so they exists in this path:

    Application Support &#x2F; Titanium &#x2F; modules &#x2F; iphone &#x2F; com.ti.calendar &#x2F; 0.1

    and check that inside 0.1 folder the file libcom.ti.calendar.a exists.
