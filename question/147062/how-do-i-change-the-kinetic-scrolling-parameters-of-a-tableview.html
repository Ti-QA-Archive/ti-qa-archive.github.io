    Hi

    Sorry but no - there are no parameters available to change the speed of scroll. However the scroll speed should be dynamic based on the users speed of interaction with the tableView, this is true of standard iOS and standard Android.

    If there is a way then it will require you creating a custom module, however this would require the OS actually providing you with this option - if it does then you have a starting point.
