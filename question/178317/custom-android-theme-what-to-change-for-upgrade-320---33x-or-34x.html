  • Simplest thing would be re-creating the Theme at http://android-holo-colors.com/ but selecting AppCompat as Compatibility. Applying this should work.

    Tipp: I had a look into your tiapp.xml - you should remove some lines there:

    // deprecated
<tool-api-level>14</tool-api-level>

// unless you don't really need this, remove it
// Titanium now sets latest targetSdk automatically
// and AppCompat is campatible with minSdk < 10
<uses-sdk android:minSdkVersion="14" android:targetSdkVersion="19"/>

// all activity definitions can be removed
// theme and screenOrientation can be set in <application />
<activity />

    Hope this helps.
