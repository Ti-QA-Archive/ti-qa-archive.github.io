  • Hi

    Yes, but that is not the problem. I can filter correctly, but the search bar appears in the table view when is focused. (see the image that i provide earlier)

    I make a test project and copy the code of this window and the problem comes from the view on the search bar.
    When i delete this upper view, the problems solves. (take a look of the image below)

    application image

    The xml structure is this

    <Alloy>
    <Window id="cartola">
            <ScrollView id="scroll">
                <View id="producto">
                    ....
                </View>
                <View id="detalles">
                    ....
                </View>
            </ScrollView>
    </Window>
</Alloy>

    Where "producto" contains all the views on the search bar and "detalles" contains the tabs, the search bar and the table view.
