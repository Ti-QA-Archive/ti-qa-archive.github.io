Referencing the title of the active tab

Hello,

I want to reference the title of the active tab so I can create a breadcrumb in my header. Something like this:

APP NAME
>> First Tab

In app.js I have the following code:

mainTabGroup.addEventListener('focus', function(e){
    mainTabGroup._activeTab = e.tab
    mainTabGroup._activeTabIndex = e.index
    if ( mainTabGroup._activeTabIndex == -1) return;

    // create property in Ti namespace
    Ti.API._activeTab = mainTabGroup._activeTab;
    Ti.API.info(Ti.API._activeTab.title);
});

No problems here, I can see the title is output depending on which tab is active.

I have a header.js file that creates the header. how do I reference the Ti.API._activeTab.title from the header.js file. At the moment I'm getting an error stating: 'Ti.API._activeTab.title' is undefined.

Any help appreciated…
