  • I had no problem running this code on my Droid X, Android 2.3.3. Are you running on a device or an emulator? if emulator, is GPS Support enabled? Run the Android SDK and AVD manager by running android and click on your virtual device. Click the 'edit' button and look under 'Hardware'. Make sure 'GPS SUpport' is in the list. If it is not, click 'new' and add it from that list.

    var win = Ti.UI.createWindow();
var mapview = Titanium.Map.createView({
    mapType: Titanium.Map.STANDARD_TYPE,
    region:{latitude:33.74511, longitude:-84.38993, latitudeDelta:0.5, longitudeDelta:0.5},
    animate:true,
    regionFit:true,
    userLocation:true
});
win.add(mapview);
win.open();
