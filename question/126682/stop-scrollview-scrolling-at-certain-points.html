Stop ScrollView scrolling at certain points

Hi,
my next app needs a horizontal scrollview where you can select the title (simply a view with a label) that you wish and this title SHOULD stop in the center of the screen. Everyone (they are half a dozen) of these titles should stop, when selected, at the center.

My problems are:

  • if I set a 'click' eventListener on the titles and touchEnabled:0 in scrollview, I can scrollTo to the center BUT the scrollview goes in touchEnabled:true mode and every touch on it make it scroll without centering;

  • if I set a 'scroll' eventListener on the scrollview, apparently I can control the centering of the titles BUT they loops infinitely, firing the event continously; this is an example:

    scrollview.addEventListener('scroll', function (e) {
  Ti.API.info('x: '+e.x)

    if (e.x <= 75)
  {
      scrollview.scrollTo(0, 0);
  }

  if (e.x < 225 || e.x > 75)
  {
      scrollview.scrollTo(150, 0);
  }
});

    The API.info goes on and on getting new x positions even if I touch nothing! I heard this behavior was typical of 'swipe' events, not for scroll.

I tried also with 'touchend' event but it did nothing.

Can someone help me fixing this?
thanks!
