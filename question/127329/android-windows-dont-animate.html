Android Windows Dont Animate

when my button get clicked, then is executed this code:


but_iniciar.addEventListener('click',function(e)
{
      var win = Ti.UI.createWindow({
            url:"home.js"
    });
   win.open({animated:true});
})

but the problem is that the default android activity transition is not triggered, so, everything goes instantly.
How can i fix this issue? Thanks in advance

  • try

    but_iniciar.addEventListener('click',function(e)
{
      var win = Ti.UI.createWindow({
            url:"home.js",
        left: Ti.Platform.displayCaps.platformWidth
      });
      win.open();
      win.animate({ left: 0, duration: 500 });
})
