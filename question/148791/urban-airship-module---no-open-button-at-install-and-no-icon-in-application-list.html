  • Hi

    I think you must be installing bad the module.

    i've just copied the module to Titanium module folders and added the module in tiapp.xml and nothing bad happens.

    For your reference below are the steps,

    1- Download the module from market

    2- Decompress it

    3- Into modules/android folder, copy ti.urbanairship folder

    4- Go to Library/Application Support/Titanium/Modules android an paste it.

    And that's all, in tiapp.xml you can add the module with PLUS button.
