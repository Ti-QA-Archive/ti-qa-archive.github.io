HTTPClient parameters

I'm looking in the docs to find out how to add parameters to an HTTPClient object. The docs don't list any properties or methods for adding parameters, but I'm sure it is possible.

How would this be done? Also, is there somewhere that a more complete set of documents exists? It seems that there are a lot of undocumented properties and methods on objects throughout the framework. I continually come across them when going through sample code and then look for the documentation on things in the sample code and I can't find it.

  • xhr.open('GET',url);
var params = {};
params.first = 'first';
params.second= 'second';
xhr.send(params);

    xhr.open('GET',url);
var params = {
   "first" : "first";
   "second" : "second"
}
xhr.send(params);
