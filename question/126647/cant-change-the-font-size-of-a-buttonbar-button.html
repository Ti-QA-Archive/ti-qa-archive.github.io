Can't change the font size of a ButtonBar button

Hello,

i do 

// create the button
var aBtn = Ti.UI.createButtonBar({
labels: [T(aLabel)],
backgroundColor: aColor,
style: Ti.UI.iPhone.SystemButtonStyle.BAR,
font: {fontSize:24},
width:    "auto",
height: "auto"    
});

yet the fontsize of the button's label is always the same. How does one change the fontSize of the label of a system button in a button bar otherwise?

  • If you don't use that style selector how can you change the properties of the buttonBar? I am trying to use the buttonBar in place of a standard button but I need to be able to edit things like font and backgroundColor… Thanks.

    FOR EXAMPLE:

            var loginBtn = Titanium.UI.createButtonBar({
            labels: [
                {
                    title: 'Login',
                    font: {
                        fontSize: 16
                    },
                    backgroundColor: '#3e82f3'
                }
            ],
            top:10,
            left: 20,
            right: 20,
            height:35,
            backgroundColor: '#3e82f3',
            barColor: '#3e82f3',
            color: 'green'
            //style: Ti.UI.iPhone.SystemButtonStyle.BAR
        });
        view.add(loginBtn);
