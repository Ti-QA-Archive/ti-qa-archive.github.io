How to scale image proportionally?

Hello,

My titanium app loads users avatars via https using xhr. The issue that each avatar has different size and proportion and I need to scale the image proportionally to fit in 50x50 container. The code below makes the avatar to fit all square, but I would like to scale image proportionally.

var frame =Ti.UI.createImageView({
                left: 0,
                top: 0,
                width: 50,//when I use 'auto' it gets more than 50
                height: 50// becouse the images is bigger
          });

...     xhr.onload = function(){
    frame.image = this.responseData;    
}

SDK 1.7.0 IOS 4.3

  • Something like this post might help:

    http://developer.appcelerator.com/question/16461/answer-how-to-dynamically-scale-an-image-from-the-filesystem

    if (image.width > image.height) {
    ratio = 50 / image.width;
} else {
    ratio = 50 / image.height;
}
newWidth = image.width * ratio;
newHeight = image.height * ratio;

// Create image here with newWidth and newHeight

    Obviously this code is a rough layout, but it will get the job done for you with some small modifications.
