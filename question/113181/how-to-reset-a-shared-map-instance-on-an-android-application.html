How to reset a shared map instance on an Android application

Hello,

Since on Android you can only have one instance of a MapView on your application, and the application I'm working on requires 2 different windows to have a mapView Object for different purposes, I need to share the mapView object between the 2 of them (I explained how I did that, here: http://developer.appcelerator.com/question/113171/dealying-with-the-restriction-of-having-a-single-mapview-instance-per-android-application ).

However, a side effect of this is that if you, for example, add on the first window some annotation or define a place where to center the map, and then go to the second window, the annotation or the current location where the map is centered are still present on the second window, because the map object is the same. Is there a way in appcelerator to "reset" the map attributes and configuration?

Thanks!
