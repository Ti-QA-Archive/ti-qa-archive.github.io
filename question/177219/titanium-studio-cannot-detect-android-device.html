Titanium Studio cannot detect Android device

I just reinstalled Titanium Studio and the Android SDK. The adb manager clearly shows my Android device is connected. Yet Titanium is NOT able to detect it. I previously also asked a question about it but received no helpful response.

The Android SDK tools clearly show the Android device is there. Eclipse shows the device is there. Titanium fails to show the device is there when checking the device dropdown in the run configuration menu.

I use Windows 8.1. I just installed appcelerator from the website. I connected a Galaxy Ace 2 with Android 2.3.6. Again: all other programs can detect it.

Does anyone got any helpful tips how to resolve this?
