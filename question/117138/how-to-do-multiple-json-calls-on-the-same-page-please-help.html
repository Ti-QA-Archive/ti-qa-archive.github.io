  • you have two questions here.

    1) it is not recommended that you execute multiple httpClient requests at the same time; so my suggestion is to create some sort of queue and execute the requests one at a time. The completion of each request should/could fire an event that would do the appropriate UI updating.

    2) to save the json results from each httpClient request, just write it to a file and you can reload the file whenever you like.
