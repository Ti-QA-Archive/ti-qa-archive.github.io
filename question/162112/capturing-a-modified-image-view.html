Capturing a modified image view...

I'm building an app where a user can load an image, overlay text and new images over it, then save the new, modified image. I put the original image in the background of an image view, then place new text and images over top that imageview - now I want to capture the result of the whole thing. Is there an easy way to just capture that screen? I'm assuming that putting the original image in the background is probably wrong as well. Please let me know…

I'm working in the latest build of Appcelerator, and this is for an iOS and Android app.

Thanks…Chris

    just do

    var blob = containerView.toImage();

    where containerView is the containing view of the imageView and the labels. Then save the blob to a file and there you have it.

    Or otherwise you could have a go with Ti.Media.takeScreenshot, but that would take the screenshot of the whole screen.
