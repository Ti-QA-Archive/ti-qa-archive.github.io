    Hi

    This assumes this sentence "There is another service to manage the blog feed that allows me to view more?" actually means the opposite "Is there another service to manage the blog feed that allows me to view more?". If not your question already has an answer.

    If this is your website then;

    • you are in control of how many feed entries are shown. A quick look at the source code shows that the site is based on word press - so simply configure the feed length to be a larger number that you already have set (or is the default).
    • create a PHP script that creates the required rss feed with all your parameters and reference that file instead of the public version - more information can be found here Ultimate WordPress RSS Feed Customization, it is old but a good starting point. This information is newer Customizing Feeds.

    If this is someone else's website;

    • ask them for permission before you use their content
    • request access to a longer feed
