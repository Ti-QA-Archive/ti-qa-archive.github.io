How could I get the relative points of 'click' events on overlaped view

I made two views such as

var ViewA = Titanium.UI.createImageView({
image : "A.png"
width : '50x',
height : '50dx',
left:0,
top :'0dx'
});
var ViewB = Titanium.UI.createImageView({
image : "B.png
width : '30dx',
height : '30dx',
left:10,
top :'10dx'
});

ViewA.addEventlistener('click',funcition(e){
//How can I use the "click" position of ViewA?
e.x,e.y returns the position of ViewB
});

ViewB overlapped by ViewA.
When I click on "ViewB", The valiable "e" in ViewA.addEventlistner returns the relative position of ViewB.

I would like to get the ViewA relative position or absolute position.
Do you have any advice?
