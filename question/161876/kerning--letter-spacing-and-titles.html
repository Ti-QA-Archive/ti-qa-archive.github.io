Kerning / Letter Spacing and Titles

Hi All,

I'm using Alloy and have some XML that resembles something like

<TabGroup>
    <Tab title="THIS IS A TITLE"></Tab>
<TabGroup>

How would you go about changing the letter spacing on the tab title?

Thanks

    Hi

    Short answer - not possible.

    Longer answer;

    You cannot do this with the native built in views, labels, buttons, tab and so on - there simply are no properties to allow it.

    The kerning of fonts has been defined by each of the operating systems and is deemed by them to be sufficient.

    If you really must - but keep in mind you may spend more time on this than any other feature in your app.

    • You could create a custom tab solution yourself and use a new label for each letter. This is not recommended.
    • You could look for a custom module to replace the built in tabs - although I do not know of any that support kerning in the tab captions - they usually only support colours, size and image related properties.
