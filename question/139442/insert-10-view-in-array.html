insert 10 view in array

hi, in my android app i need to insert 10 views in an array so as to access to the views using an index, example of use:

view[i].visible=false;
view[i+1].visible=true;

can you give me some help? thanks in advice

  • You just need to push the view reference to an array as:

    var view=[];
view.push(view1);
view.push(view2);

    and so on…

    or you can also use for loop to insert views in array if possible.

  • var data = [];

for(var i = 0; i < 10; i++){

    var view = Ti.UI.createView({ visible : true });
    data.push(view);

}

//then u can access with:

Ti.API.info(data[0].visible);
Ti.API.info(data[1].visible);

// or modify

data[2].visible = false;
