How to get height of a NavBar on Android?

My app is for android and will open a heavy-weight window and then add a view to it by utilizing the whole screen. 

var win = ti.UI.createWindow({
    navBarHidden: false,
});
var view = Ti.UI.createView({
  backgroundColor:'yellow',
  width: Ti.UI.FILL,
  height: Ti.UI.FILL,
  top:0, left:0
});
win.add(view);

My question is, how can I get the actual height of the yellow View?

By the way, using view.getHeight() only give me the value fill

Thanks in advance.

  • Hello Bowie,

    You have to give some time to count it, see this…

    var win = ti.UI.createWindow({
    navBarHidden: false,
});
var view = Ti.UI.createView({
  backgroundColor:'yellow',
  width: Ti.UI.FILL,
  height: Ti.UI.FILL,
  top:0, left:0
});
win.add(view);

setTimeout(function(e) {
    alert(view.size.height + " ~~ " + view.size.width);
    Ti.API.info(view.size.height + " ~~ " + view.size.width);
}, 100);

    Clean the build and run it again, i hope it will help you… Best Luck.

  • Hi Ritesh,

    In your solution, the time to wait is uncertain.
    I found another way which is more appropriate to get the view's size once it's ready.
    That is listen to the postlayout event:

    view.addEventListener('postlayout', function(){
   alert(view.size.height + " ~~ " + view.size.width);
});
