Retain application SQLite database after uninstalling

Hello,
I'm developing an iPad app with titanium.
The application mainly does this: writes data to a local database, downloads a database from a web site, compares the data of databases and displays them to the user.
The application is almost complete but there is a small problem, came out of a customer's request.
What we need is the ability to preserve the local database even after uninstalling the application.
This program was also developed in native code for the Android platform. To remedy the problem on the Android devices we have installed databases in a folder outside the application so that the uninstaller does not delete the files.
Since I could not find a function that allows me to enter the installation path as input using titanium on IOS platform, I started to think that same approach can not be used in this case.
So Is there a way to store the database files even after uninstalling?
