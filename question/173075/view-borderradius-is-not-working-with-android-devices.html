View borderRadius is not working with Android Devices?

Hello, I am using View UI. for display one image.
I have need to use borderRadius properties.

This property is working good with iOS device. but not working with Android.
I am using Titanium 3.2.2 GA.

var view = Ti.UI.createView({
        width : "100",
        height  : Ti.UI.FILL,
        backgroundColor :"transparent",
        borderRadius :"10",
});

this is not working working with Android (borderRadius).
Please Help me, ….!

    Hello Mritunjay,

    Try to set borderWidth and borderColor and check again.

    Try this :

    var view = Ti.UI.createView({
        width : "100",
        height  : Ti.UI.FILL,
        backgroundColor :"transparent",
        borderRadius :"10",
        borderColor : "red",
        borderWidth : 1,

});
