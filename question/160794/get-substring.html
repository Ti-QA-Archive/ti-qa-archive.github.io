Get substring

I am using alloy framework.

How to take sunstring from string?

For example,

var strTest = "Hello1";

Now how can i extract 1 from Hello1?

Please help me.

    Hello Preety,

    Try this :

    var strTest = "Hello1";
var res = strTest.substring(0,5);

alert(res);

    You can also use replace like :

    var strTest = "Hello1";
var res = strTest.replace("1","");
alert(res);
