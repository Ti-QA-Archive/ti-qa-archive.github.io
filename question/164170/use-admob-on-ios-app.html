  • This is the "official" Admob module for iOS:

    https://github.com/appcelerator/titanium_modules/tree/master/admob/mobile/ios

    Last update was 11 months ago and the module is using Admob SDK 6.4.2 which is rather outdated as current version is 6.8.0. But it shouldn't be too much work to update to latest SDK. This is only for banner ads, unfortunately no Interstitial support.
