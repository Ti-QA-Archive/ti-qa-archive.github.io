Pressing back button close the app in Android

Hi when i press the back button in Android simulator my app close down.
Even when i open the two windows below is the test code.

var win1 = Titanium.UI.createWindow({
title:'Tab 1',
backgroundColor:'#fff'
});

var lab = Ti.UI.createLabel({
text:'Hi'
});

lab.addEventListener('click',function(){
var win2 = Titanium.UI.createWindow({
title:'Tab 2',
backgroundColor:'#eee'
});
win2.open();
});

win1.add(lab);
win1.open();

    That's because you don't create heavyweight windows (back hardware button works only with heavyweight windows).

    Here is formal definition:

    • A window is heavyweight if it has any of these properties: fullscreen, navBarHidden, modal, windowSoftInputMode, or if the tabOpen property is true
    • Otherwise, the window is lightweight

    Define your window with "modal" property set to "true", for example.

    var win = Ti.UI.createWindow({
    modal: true
});

    Create more than one window like that and then try to use back hardware button. :)
