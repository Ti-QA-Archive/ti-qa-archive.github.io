How to update position cached in Android

Hi, I'm trying to use Geolocation with manualMode set on false.
I have set ACCURACY_BEST and I would like that, the app get the current position of the user every time the user start this one.
But there's a problem beacuse after the first getCurrentPosition, the following ones returns the same latitude and longitude (Because,I read, there's a position cached in Android).

How can I update the position cached in Android (I would like to have the effective position of the user and no the cached one).
Should I use the Geolocation with manualMode set on true and manage by hand every location provider?

Thank you in advance,

Luca

    • As you discovered getCurrentPosition() only fires once. The location eventlistener returns an accuracy property in the event listener response and fires repeatedly. You should switch off the eventlistener when you have an accuracy that suits you because this will drain the battery quickly
