  • Did you set the Android emulator's location? Usually each time I boot the emulator I run geo fix to make sure it is set to an known location.

    Steps to do this:

    1) run adb to find your emulator device port

    2) telnet into you emulator, for example telnet localhost 5560

    3) Run the following command geo fix -82.411629 28.054553

    This should set your location to someplace in Florida. You can then check through the emulator's maps to verify it is correct. Sometimes you need to re-compile and push your Ti app again for the emulator to push the updates to your App.

    Hope that helps.
