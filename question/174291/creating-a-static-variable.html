Creating a Static Variable

I'm planning to set a fixed variable in order to organize the control flow of the mobile app program. Here's what I did: at the first controller Javascript file (Con1.js), I made it like this, for example:

static var value = 999;
static var listSection = $.section;

Then, when opened another activity window, it will calculate through Con2.js for example:

function dummy() {
    Alert("Value: " + Con1.value);
    Con1.listSection.appendItem([item]);
}

Here's one thing I can't make a static variable: I noticed at Con1.js that there is an error marking (in wavy red underline) when I'm typing a static variable code. Why it's not working? Is the only alternative for this one is to use a Global variable or something to pass this variable? I'm planning to use static variables to add 1 new item from the second window view and a new item has been added to the first window view.

    There is no keyword static in javascript. If you are using Alloy then to create a global variable, assign it to the Alloy Globals namespace like this

    Alloy.Globals.myGlobalVariable = "This is a constant I can use throughout my App";
