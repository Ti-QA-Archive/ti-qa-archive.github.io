Include date in URL

Guys,
anyone know how to include the date in a URL?
So once the APP opens it immediately displays a web view and pulls data via php based on todays date.

So whilst I can include that manually in my app.js file quite comfortable e.g.: var webview = Titanium.UI.createWebView({url:'http://www.myserver.com/info.php?date=2012-01-31'});

….my problem is how can I automatically get the 2012-01-31 part to be automatically inserted based on todays actual date?

any ideas?
regards
Andy

    Try this:

    var now = new Date();
var today = now.getFullYear() + '-' + (now.getMonth() + 1) + '-' + now.getDate();
var url = 'http://www.myserver.com/info.php?date=' + today;
var webview = Titanium.UI.createWebView({url: url});
