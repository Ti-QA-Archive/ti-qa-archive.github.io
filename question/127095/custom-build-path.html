Custom build path

Is it possible to set a custom build path for Titanium Mobile in Studio?

My mobile projects are located in my Dropbox, the problem is that each time I rebuild a project, Dropbox wants to sync the whole build folder, which is quite big. Any way around this?

    I had the same problem. I solved it by going into Dropbox's settings and excluding my /build folder. Note that after you do this, Dropbox wants to remove the folder from your computer. In the case of the /build folder this is not an issue, as the folder will be recreated the next time you compile the project.

  • The SDK has some command line options available, but I don't think you are going to easily get what you want with it.

    You would be much better off in the long run by going with a real source control management system, and Studio supports several of them, including the popular git. You can then use a remote repository, such as github.com, (free for open source projects) to remotely save your code and track changes.
