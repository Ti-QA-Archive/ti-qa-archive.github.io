  • You are splitting a string and asking javascript to traverse it as if it was an Array object. There are ways to make an Array and here is one way.

    var grid_no = new Array("2","5","7","3","6","8");
var arry = new Array("9","10","12");
var arr= new Array("blue","red","yellow");

    Keep in mind when you call 

    var btn_color = arr[i];

    The color would begin with blue since:

    arr[0] = "blue";

    In brakets 0 is blue, 1 is red, and 2 is yellow.

    In your code grid_array:

    grid_array  = grid_no;

    To add value to arry:

    arry.push("green");

    This will add color green at the end of the Array.

    Hope this helps.
