Search Bar Problems In Android

A couple problems:

  1. The search bar always focuses immediately, and I really don't want this to happen. Is it possible to stop the keyboard from coming up immediately?

  2. The search bar just fails to search. If you insert text and press enter, nothing happens at all. The search bar is properly set up and works in iOS

     search : Ti.UI.createSearchBar({
     barColor : '#666',
     height : getHeight(45),
     top : 0,
     showCancel : false
 }),
 filterAttribute : 'filter',

    • Yeah, so I've found. I had to use something like this, if it helps you any:

      var blur = 1;
win.addEventListener('postlayout', function(e) {
    if(isAndroid && blur == 1) {
        searchBar.hide();
        searchBar.show();
    blur = 0;
    }
});

      However, I still can't get it to search.
