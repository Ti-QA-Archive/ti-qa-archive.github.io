Debugging modules

Hi everyone. I'm attempting to write a simple Titamium module for iOS only. I can build it on the command line then add it to my Titanium app and can run it and it works just fine but I want to continue developing the module further and I am at the point where I really need to debug the module (step through line by line) in xcode and I just cant figure out how to do it.

Here is what I have atempted:

  1. Build module on the command line (.&#x2F;build.py)
  2. Add module output to Ti app
  3. Clean then build Ti app
  4. Open the Ti app in xcode from the Ti apps build directory
  5. Drag module project into Ti app project in xcode
  6. Add break point to module code
  7. Run Ti app project on simulator (from xcode)

The breakpoint is never hit.

I even tried modifying the build.py build script for debuggin by changing

rc = os.system("xcodebuild -sdk iphonesimulator -configuration Release")

to 

rc = os.system("xcodebuild -sdk iphonesimulator -configuration Debug")
