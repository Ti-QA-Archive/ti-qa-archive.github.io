Finding item in TableView with dataFilter applied

I'm probably going about this wrong, but I've been searching for an hour and haven't found a solution, so thought I'd try here :)

I have a view:

<Alloy>
    <Collection src="venues">
    <Tab title="Nearby">
        <Window title="MyVenues">
            <TableView id="venuesTable" dataFilter="filterFunction" dataCollection="venues">
                <TableViewRow onClick="showVenue" venueId="{id}">
                    <Label id="title" text="{title}" class="h6"/>
                </TableViewRow>
            </TableView>
        </Window>
    </Tab>
</Alloy>

Which shows a filtered list of venues. The filter function is:

function filterFunction(tableData) {
    return tableData.where({city:'Adelaide'});
}

This all works perfectly. The problem I have is when I try to call showVenue on the click event:

function showVenue(e) {
    var venue = venues.at(e.index);
    var detail = Alloy.createController('venueDetail', venue).getView();
    detail.open();
}

e.index is referring to the item index in the filtered collection (tableData) not the index of the item in the venue collection. I've tried the following but 'venue' comes back as undefined.

function showVenue(e) {
    var venue = venues.fetch({id: e.rowData.venueId});
    var detail = Alloy.createController('venue/venueDetail', venue).getView();
    detail.open();
}

Any ideas/thoughts???

Cheers!

    Don't rely on the index like that, especially when filtering, sorting, or transforming. It's rarely going to match up. Instead, have your row keep track of the model's id and use get() instead. Also, it's more effecient to put the click handler on the TableView itself, not on each TableViewRow. Something like this, based on your sample:

    <Alloy>
    <Collection src="venues">
    <Tab title="Nearby">
        <Window title="MyVenues">
            <TableView id="venuesTable" dataFilter="filterFunction" dataCollection="venues" onClick="showVenue">
                <TableViewRow venueId="{id}">
                    <Label id="title" text="{title}" class="h6"/>
                </TableViewRow>
            </TableView>
        </Window>
    </Tab>
</Alloy>

    function showVenue(e) {
    var venue = venues.get(e.row.venueId);
    var detail = Alloy.createController('venueDetail', venue).getView();
    detail.open();
}
