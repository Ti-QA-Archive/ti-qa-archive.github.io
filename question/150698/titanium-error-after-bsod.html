  • Well, it can be many things. But before doing anything drastic, like uninstalling and re-installing Titanium altogether. You could do the following:

    • Delete your Android Virtual Device (AVD) and relaunch your project. This will create a new Emulator image son it will eliminate the possibility of corruption on that part.
    • Clean your project using the Project -> Clean… menu.
    • Try changing your project's SDK Version to the latest version.
    • Create a whole new project and then copy the contents of your Resources directory into this new project.

    If all that doesn't work, I'm afraid the very last solution is to re-install Titanium altogether.

    Hope this helps.
