  • You didn't say which platform you're building for. Have you tried something like this?

    "TableViewRow[platform=iphone]": {
   selectionStyle = Ti.UI.iPhone.TableViewSelectionStyle.NONE
}
"TableViewRow[platform=android]": {
   backgroundSelectedColor = 'transparent' /* or your row's background color */
}
