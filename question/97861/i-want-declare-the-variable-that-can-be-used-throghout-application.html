  • If all your code is within app.js or included via Ti.include() then any variable you set is global. If you load a window by setting the url:somefile.js property, then that window runs in a separate context and variables set in app.js are unavailable. If you're dealing with constants, you could follow Hal's excellent advice. Or, you could put those constants in a js file and include it using the Ti.include() in all your files. I did that with color settings, for example, prior to the JSS support added with the 1.5 SDK.
