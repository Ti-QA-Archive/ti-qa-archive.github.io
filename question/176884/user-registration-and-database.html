  • Alexander,

    The users cloud object has a 'custom_fields' property that takes a JSON encoded dataset. You can put whatever you want for 'custom' properties for a user there. You can check this out here

    As for the free version of cloud, you get 'Up to 250,000 API calls per day, 5M API calls per month, 20GB of storage'. So the answer is 20GB of storage. You can see the details on the free plan here
