How to move from one window to another

Hi everyone,
I am new to titanium… I am using allow, in that am able to move from one window to another in forward direction but I am not able to come back…. My index.xml code is

<Alloy>
<Window id="home" layout="vertical" class="window">
<ImageView id="logoImg" image="images.jpg" />
<Label id="logo" ></Label>
<TextField id="curvedTextField" hintText="Login" top="20"></TextField>
<TextField id="curvedTextField" hintText="Password" passwordMask="true" top="20"></TextField>
<Button id="signInButton" >Sign In</Button>
<Label color="blue" top="10">
Forgot Password ?
</Label>
</Window>
<Window id="test1" >
<Button id="testButton" >Hello Window</Button>
</Window>
<Window id="test2" >
<Button id="testButton2" >Hello Window</Button>
</Window>
</Alloy>

and my index.js code is
$.index.open();

$.signInButton.addEventListener('click',function(e){
$.test1.open();
});
$.testButton.addEventListener('click',function(e){
$.test2.open();
});
$.testButton2.addEventListener('click',function(e){
$.test1.open();

Plz tell me how to open home window using id "home".

    Hello Husain,

    Thanks for your inquiry. Though it is not clear on which platform you developing. in Android, using system back button can turn you from any window to the previous window. In iOS you can use navigation window. If you want to come back from window "hellowindow2" to your "home" window then try using the below code in "index.js"

    $.home.open();
$.signInButton.addEventListener('click', function(e) {
    $.test1.open();
});
$.testButton.addEventListener('click', function(e) {

    $.test2.open();
});
$.testButton2.addEventListener('click', function(e) {
    $.test2.close();
    $.test1.close();
    $.test1.open();
});

    index.xml

    <Alloy>
    <Window id="home" layout="vertical" class="window">
        <ImageView id="logoImg" image="images.jpg" />
        <Label id="logo" ></Label>
        <TextField id="curvedTextField" hintText="Login" top="20"></TextField>
        <TextField id="curvedTextField" hintText="Password" passwordMask="true" top="20"></TextField>
        <Button id="signInButton" >Sign In</Button>
        <Label color="blue" top="10">Forgot Password ?</Label>
    </Window>
    <Window id="test1" >
        <Button id="testButton" >Hello Window1</Button>
    </Window>
    <Window id="test2" >
        <Button id="testButton2" >Hello Window2</Button>
    </Window>
</Alloy>

    Also you should use $.home.open(); instead of $.index.open(); to first open your home window. Feel free to responce. Thanks.

  • Hi Husain,

    its is best practice if you create separate controller for separate window..
    for example.. you have three more window other then index window. so create three controller
    for example winTest1.js, winTest1.tss and winTest1.xml.

    and if you want to close normal window then need to add button or any controller(for IOS because ios don't have hardware back button) in opened window.
    e.g
    winTest1.xml

    <Alloy>
    <Window id="winTest1">
        <Button id="btnClose" title="Close" onClick="closeWindow">
    </Window>
</Alloy>

    winTest1.js

    function closeWindow(){
    $.winTest1.close();
}

    winTest1.tss

    And more thing if you window within tab group or navigation window then no need to create button because it's provide default navigation bar with back button..

    if you want to learn more about Alloy then refer this
