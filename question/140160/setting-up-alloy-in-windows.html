Setting Up alloy in Windows

hi to all, I wanted to try out the new alloy framework, so I cloned the repo from github, but the instruction there ask to type "[sudo] npm install -g ." to install local copy with executable (I cant get to install in that way in windows). So please help me to get started with alloy. I tried setting the path environment variable to point to alloy folder and then typed "alloy new ." from command line projects folder which just opens the alloy.js in titanium and no "app" folder is added to the project.

  • At this point we only support Alloy development on OSX. That said Titanium supports Windows development and Alloy will as well. We are knocking off a few more important features then will make sure Alloy runs on Windows. Important to note is there is nothing in Alloy that is OSX specific it's more that some build or generate scripts have resource dependencies that need adjusting for Windows.
