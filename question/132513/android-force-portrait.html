Android, force Portrait

Hi,
My app works exclusively in Portrait mode on iPhone.
Would like the same thing on Android. I've done something like this :

<android xmlns:android="http://schemas.android.com/apk/res/android">
        <tool-api-level>8</tool-api-level>
        <manifest android:installLocation="auto">
            <supports-screens android:anyDensity="false"
                android:largeScreens="true" android:normalScreens="true" android:smallScreens="true"/>
            <orientation>Ti.UI.PORTRAIT</orientation>
        </manifest>
    </android>

But it doesn't work (1.8,1 SDK).

Any idea ?

  • One option to get screen in portrait mode is goto android project folder —> build —- > android ——> here you can see AndroidManifest.xml file .

    Copy AndroidMenifest.xml file text and put in AndroidManifest.custom.xml file in same folder where AndroidManifest.xml file is kept and in activity tag write android:screenOrientation="portrait" for portrait mode and replace android:configChanges="keyboardHidden|orientation" to android:configChanges="keyboardHidden" in whole file in each activity .

    Example

    <activity>

    android:configChanges="keyboardHidden"

    android:screenOrientation="portrait "

    </activity>

    This will solve your problem.
