The dark gray (or whatever the flavor of Android the user is running happens to use as a color scheme) tends to clash with the UI of my app and look confusing.

I've tried setting backgroundColor on the tab itself, but that got me nowhere.

Ideas?

  • It's also appears to be possible to style the android tabs by using android theming. The way to go about this would start with copying the titanium_tabgroup.xml from build/android/res/layout to platform/android/res/layout/titanium_tabgroup.xml

    it probably looks like this

    <?xml version="1.0" encoding="utf-8"?>
<TabHost >
    <LinearLayout
        android:orientation="vertical"
        android:layout_width="fill_parent"
        android:layout_height="fill_parent"
        android:padding="0dp">
        <TabWidget
            android:id="@android:id/tabs"
            android:layout_width="fill_parent"
            android:layout_height="wrap_content" />

    </LinearLayout>
</TabHost>

    and add the property style=&quot;@style&#x2F;MyTabWidget&quot; to the TabWidget entry

    then in platform/android/res/values/styles.xml put

    <?xml version="1.0" encoding="utf-8"?>
<resources>

        <item name="android:textAppearance">@style/MyTextAppearance.MyTabWidget</item>
        <item name="android:ellipsize">marquee</item>
        <item name="android:singleLine">true</item>


        <item name="android:textColor">#ff0</item>
        <item name="android:textColorHighlight">#FFFF9200</item>
        <item name="android:textColorHint">#0ff</item>
        <item name="android:textColorLink">#5C5CFF</item>
        <item name="android:textSize">16sp</item>
        <item name="android:textStyle">normal</item>


        <item name="android:textSize">20sp</item>
        <item name="android:textStyle">bold</item>
        <item name="android:textColor">#f00</item>

</resources>

    That should get you started. It would be nice if any of this was documented by appcelerator or even better if the documented styling properties in http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.UI.TabGroup-object actually worked. Get on it Appcelerator!
