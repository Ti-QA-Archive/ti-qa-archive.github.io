  • ###To save you a click or two:
    For command-line, you can use these flags (assuming you are in the project folder - use "-d" flag to specify project directory if not):

    • "–retina": Gets you retina
    • "–tall": combined with retina while targeting an iPhone simulator, gets you iPhone 5 size.

    ###Some examples:

    • "ti build -p ios -T simulator -F universal –retina -Y ipad -S 6.0": This will run an iOS simulator build of a universal app on a retina ipad simulator version 6.0. Please note that the iOS SDK used for compile is based on what you have available in Xcode, and if multiple versions, can be set with "-I"
    • "ti build -p ios -T simulator -F iphone –retina –tall -Y iphone -S 6.1": iphone 5 simulator running 6.1 with iphone-only app.
    • "ti build -p ios -T simulator -F iphone -Y ipad": Non-retina ipad running iPhone app.

    Check "ti build -h" for reference.
