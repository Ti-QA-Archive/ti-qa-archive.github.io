Problems with positioning (changing view's 'bottom' value does not move labels inside)

Hi everyone,

I'm trying to "lock" the position of several labels inside of a view, here's the code for the view:

// Create the variable input view
var toVariableView = Ti.UI.createView({
    bottom: 0,
    left: 38,
    width: 272,
    backgroundColor: '#ccc'
});
toView.add(toVariableView);

And here's the code for one of the labels:

    var toAddress = Titanium.UI.createLabel({
        top: toInput.top+1,
        text: toInput.value.substr(1),
        textAlign: 'center',
        height: 26,
        width: 'auto',
        left: toInput.left+3,
        font:{fontFamily:'Helvetica',fontSize:16},
        backgroundGradient:{type:'linear', colors:['#dbe5f7','#bccef1']},
        borderRadius: 12,
        borderColor: '#a3bdeb',
        recipientCount: recipients_count,
        borderWidth: 1
    });

What I'd expect to happen, is that when I change the bottom value of the toView, that all the labels (represented by the toAddress code above) would adjust as well. However, I change the bottom value and the labels do not adjust. How do I fix the labels inside a view so that they are always X pixels away from the bottom of the view they are contained by? Even if that view moves?
