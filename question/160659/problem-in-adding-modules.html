problem in adding modules

hi i am having problem in adding modules to my apps i build for android i use Titanium 3.1.1.GA (Alloy)

here is what i do
first

i but my module in "MyApp/modules/android/"
in this case it is "net.bajawa.pager" module

after that i add this to my tiapp.xml 

<modules>
        <module platform="android" version="1.2.3">net.bajawa.pager</module>
    </modules>

version number is correct

then in my index.js 

var module  = require('net.bajawa.pager');

i get this in my log 

23:9:38 [DEBUG] [android, 4.2.2, 10.0.3.15] Native module:net.bajawa.pager
23:9:38 [ERROR] [android, 4.2.2, 10.0.3.15] 
Type: Error Message: Requested module not found File: ti Line: /module.js At: Module.require

    Ok, then it is related to your module's installation.

    Try re-installing the module by:

    • Cleaning the contends of your modules directory
    • Remove the module section from your tiapp.xml file
    • Once all this is done, comply copy the module's archive (the zip file) at your project's root directory.
    • Clean your project
    • Rebuild and Launch your project in the emulator or your device (it should fail when trying to load the module, but it should extract the archive and install the module in the right directory)
    • Open your tiapp.xml file with the editor and add the module by clicking the + button from the top right corner of the screen

    This should address any installation issue you might have.
