NavigationWindow without history

Hey guys,

little question - how can i hide the history navigation in the NavigationWindow?
When i open the NavigationWindow and open a new window win.openWindow(newWindow, {animated:true}); - then came the history button in the left-top corner. can i disable that?

  • Hi

    Short answer; No.

    There is no current way to disable the history interaction with the NavigationWindow.

    Keep in mind users will quickly come to expect and anticipate this gesture in app that provide a NavigationWindow - so the current inability to turn it off should not be considered a negative.

  • Use a transparent view as leftNavButton:

    win.leftNavButton = Ti.UI.createView({
    width: 1,
    height: 1
});

    There are other approaches like win.setLeftNavButton(null) or win.backButtonTitle = '', but to get this work in iOS6 and iOS7 I prefer the transparent view.

    • Set the leftNavButton in child window, not root navigation window

      var win = Ti.UI.iOS.createNavigationWindow({
    extendEdges: 0,

        width: "100%",

        zIndex: 10,
        leftNavWindow: Ti.UI.createView({ width: 1, height: 1 })
    })
});
