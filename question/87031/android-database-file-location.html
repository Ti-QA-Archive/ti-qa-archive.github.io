Android Database file location

In iPhone, I can view the current status of my SQLite database by accessing the .sql file at "Library/Application Support/iPhone Simulator", etc. Is there an equivalent location for the database in Android?

    it should be stored as a SQLite file in /data/data/PACKAGE/databases/DATABASEFILE where package name is declared in the AndroidManifest.xml

    Additional Documentation at developer.android.com
