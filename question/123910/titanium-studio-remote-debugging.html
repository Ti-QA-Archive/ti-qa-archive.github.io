Titanium Studio Remote Debugging?

Hello, I was just wondering if Titanium Studio has any kind of Remote Debugging capabilities?
I ask because I am new to iOS development with Titanium, I use a MacBook laptop however I find it much more difficult and tedious to write code on the laptop, so I would like to be able to use Titanium Studio on my desktop (Windows), and have it so if I press Run or Debug etc on the desktop, it would run on the laptop. Is this possible? I have tried remote desktop solutions, but it was not ideal at all.

    I do this all the time; I rarely work directly on my macbook pro because I have three large monitors hanging off my Windows machine and don't have room on the desk for the macbook pro too. I only work directly on the MBP when away from my desk. OSX has built into it what amounts to a VNC server. Enable it at:

    System Preferences > Internet & Wireless > Sharing

    Enter a name where it says Computer Name, and put a check in Remote Management.

    Now on the Windows machine install a VNC client and connect to the MBP using the name you gave above. Any VNC client should do. I use UltraVNC Win32 Viewer and performance is just fine. I only miss the multitouch gestures. :(
