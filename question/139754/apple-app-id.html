    • Did you even try my solution? It does work. (based on your original question).

      
var app_dir = Ti.Filesystem.applicationDirectory;
Ti.API.info('app_dir: '+app_dir);

      Then you can simply work out the part of the path you need using some very simple JavaScript string functions.
