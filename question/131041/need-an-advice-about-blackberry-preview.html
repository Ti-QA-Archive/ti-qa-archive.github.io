Need an advice about BlackBerry preview

Hello,

One of my customers wants a BlackBerry application. As I have a professional account, I know I can try to do it with the BlackBerry SDK Preview.
But I would like first to have your advice about BB stability.
The application is not really complicated (at least for iOS and Android), but can someone tell me if everything should work also for BB ?

  • install small database and sometimes overwrite it with a new version (when I publish a new version of the application)
  • sqlite queries
  • geolocation (not very precise, a few kilometers is enough)
  • webview in order to show html news from a RSS/JSON feed

Also, what about notifications (I will use UrbanAirship for the iOS and Android versions). Is there a solution ?

Thank you for your answers.

Joel

  • I tried to build a Blackberry application a few months ago using Titanium and found it impossible, basic functionality didn't work right and I found it to be unusable. The impression I get is that Appcelerator isn't focused on improving BB, as we haven't heard any new news of improvements in a long time. If you are sure you want to build a BB app, then I'd avoid using Titanium for it.

    That being said, you could advise your client on the rapid decline of RIM and BB, and could maybe suggest that they go a different direction like iOS and Android. Maybe they're set on building a BB app, but it's worth a shot.

  • In my case, I had to forgo even trying the preview because it did not support the range of Blackberry OS versions I needed for the potential project. I don't remember now – and am not finding quickly – what versions it does support but it was a show stopper for me. Thus I never went far enough to even try the SDK.

    I only gave up supporting OS 4.x on my (native) Blackberry projects last month. I need to continue to support 5.x for the foreseeable future, as well as still run correctly on 6.x and now 7.x devices.

    Before you get too invested in a Blackberry project, consider your end user device requirements.

    I'd love for my appcelerator projects to also port to blackberry, but last I checked it did not cover my requirements.

    YMMV.
