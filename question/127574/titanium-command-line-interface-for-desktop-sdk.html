  • Yes, you can reference the packaging doc to see the basics and/or just look at tibuild.py in the sdk folder, as it what is used to manipulate desktop at the command line.

    Just keep in mind that you can't create apps from the command line, there is a small portion that was written in javascript for Titanium Develop as that was ported to java for Titanium Studio. Building and packaging can all be done from the command line.
