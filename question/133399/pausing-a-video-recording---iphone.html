Pausing A Video Recording - iPhone

All my code works for recording a video, uploading it to a server, etc… I also see that there are cancel, re-take and use functionality… I was hoping to add another button on the video recording dialog box that says "Pause"… What this will do is pause the recording in progress and when you decide to start recording again you can click it again…

  • Yes, I did figure out a solution… Probably not the best. But it worked for me. The best solution is to NOT use appcelerator anymore. I have since just started using objective-c and xcode. Appcelerator is buggy and no good… BUT… here is my solution:

    I recorded a video… then created my own pause button. When someone pressed pause… I saved each video segment to an array. So essentially there can be many video segments in an array, then I transfered these to my server and had FFMPEG splice these videos together.

    There is NO other way to do this in Appcelerator.

    Objective-C makes this process easier and more effective. No need for the server.

    Good luck.
