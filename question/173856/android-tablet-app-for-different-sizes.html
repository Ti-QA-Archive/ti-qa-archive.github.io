  • Try using dip as measure unit:

    Ti.UI.createView({
width: '200dp',
height: '200dp',
backgroundColor: 'gray'
});

    This should create a gray square, that will look the same on 10in and 7in tablet, as well as on all phones.
    You can also declare your default unit in tiap.xml

    <property name="ti.ui.defaultunit" type="string">dp</property>

    However it can be tricky.
