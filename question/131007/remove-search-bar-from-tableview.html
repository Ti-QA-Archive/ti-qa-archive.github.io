Remove search bar from tableview

I would like to remove a searchbar from a tableview, I have a tableview like this:

//create search bar
var searchbar = Titanium.UI.createSearchBar({
showCancel:false
});
//create tableview
var themenu = Ti.UI.createTableView({
height:405,
top:55,
search:searchbar
});

I am aware I can set the searchHidden:true but that does not remove the search bar, it only makes it so that I would have to scroll up for it. I also can not use searchbar.hide(); because it will leave a blank space where it used to be. How can I remove it, leaving no access to it, and no blank space where it was?

    Set your searchbar=null; then set the top of your tableview to match where it should start?
