Orientation changes only render/resize active window

Hi, I have couple of windows on a screen, say win1 (which is in front and active) and other win2 (which is in background of top active window).
I am using % way for expansion of these windows. Say win1.width : '50%' and win2.width='75%';

Now on orientation change, i want to change the size of both windows as below
win1.width : '70%' and win2.width='90%';
The issue is on orientation change, it only expands the win1 (which is active) and does not affect win2. In reality, actually the win2 property also gets updated on orientation change but it just does not render the non active windows again. If you close, the win1 (top window) the win2 automatically changes it size as it becomes active by then.

Can anyone suggest me how to adjust the size of inactive windows on orientation changes? Is there anyway, we can get all windows rendered at a time, forcefully active them and render them?
