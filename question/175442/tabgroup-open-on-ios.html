tabGroup open on iOS

Hi,
New to Titanium, I building some little pieces of code to try.
I face a problem using tagGroup on iOS;
Creating a tab group
var tab_group = Ti.UI.createTabGroup();
Then the tab:
var tab1 = Ti.UI.createTab({
title: "Tab-1",
icone: "my_icn.png",
win: my_win1
});
Then declare the window (works fine in other code), add the tab in the taggroup using addTab and so on…

The last function I call is:
tab_group.open();
and I get an error saying "Invalid type passed to function at line 65". When I change the line position, the error follow the tab_group.open() statement. I just check all semi-colon, closed and opened brackets and so on (after 15 years of assembly langage programmation, I know how bug can be viscious!) without any result.
Is there a bug in the createTabGroup() function, create a "non-tabgroup" element, or is there a bug in the open? And how to get a solution?

Best regards to all
Pierre-Louis

  • Hi,

    you have to create the window BEFORE you pass it to a tab.

    Basic example:

    var tabGroup = Ti.UI.createTabGroup();
var win = Ti.UI.createWindow();
var tab = Ti.UI.createTab({window:win});
tabGroup.addTab(tab);
tabGroup.open();
