  • Mattia,

    There is currently no API for this in iPhone for titanium to access, that being said the two options for you would be to write a custom Object-C based module to do that for you, or you can hook into an external API. An example of this external API can be seen in Gennadiy Potapov's sample app found on GitHub https://github.com/TermiT/titanium-gtts
