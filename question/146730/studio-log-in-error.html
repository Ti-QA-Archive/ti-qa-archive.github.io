  • Hi Masaru, If you are behind a firewall you need to add the following ports to the firewall's white list:

    https://api.appcelerator.net (ports 80 and 443) Used for login

    http://studio.appcelerator.com (ports 80 and 443) Used for the dashboard

    http://download.appcelerator.com (ports 80 and 443) Used for software updates
