Newbie Database Question

Hello.
I apologise right from the start if this is a really dumb question. I'm keen to learn how Appcelerator Titanium works with SQLite, have read some of the online resources and watched a couple of tutorials. For the moment I'm looking at this in the context of a mobile iOS project, and I'm coding on a MAC.

I have a few lines of code that installs a database (which I've already setup using the FIrefox plug-in, replaced the .sqlite extension with .db and it is saved in my resources folder). I insert data into the database and then (using a query) read that data back and display it using a simple Alert. So far, so good, that all seems to work.

I say seems to work because if I then re-open the database using the Firefox plugin, I can see the database structure but my data doesn't appear to be there. What's going on? 

var win = Ti.UI.currentWindow;

var db = Titanium.Database.install('test.db', 'dbversion1');

db.execute("INSERT INTO test (name) VALUES ('this is my name')");

var sql = db.execute('SELECT name FROM test');

var name = sql.fieldByName('name');

db.close();

alert(name);

Thanks

Phil

    DCCB126A-BEC8-4EC2-8BF8-2A11F36CD4F9 differs on your system.
