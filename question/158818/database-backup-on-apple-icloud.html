  • Hi

    Support is limited beyond being able to specify which files in your local device will automatically upload to the iCloud.

    This is a link to work someone has done, but the answer is limited to key/value pairs.

    • Using iCloud Key Value Store in Titanium

    Depending on exactly what you are trying to achieve - you could always simply backup to DB file to a location on the device that is marked for iCloud backup - this will backup the file but NOT allow other accounts to access beyond this single device.

    I have my suspicions as to why Appcelerator does not fully embrace iCloud - ACS.
