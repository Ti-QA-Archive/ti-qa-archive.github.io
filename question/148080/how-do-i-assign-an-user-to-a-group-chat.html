    See the documentation http://cloud.appcelerator.com/docs/api/v1/chats/chatgroup

    which states

    If user A sends chat message to user B and C, user A, B and C automatically form a chat group. Use this api to get a list of chat groups the current user belongs to. User must be logged in.

    so by sending a user a message, by default it appears the chat group is created
