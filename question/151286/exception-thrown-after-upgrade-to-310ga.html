  • It turns out it wasn't entirely my array. For whatever reason this error was being thrown because I was trying to change the height of a view while also animating it. I needed to write some basic code that only triggered the animation once the 'postlayout' event listener was complete for the view I changed the height on. That fixed my issue.

    I did go ahead and correct my arrays too, but only when I isolated the issue by logging to the console each step of the function that was executing when the error occurred was I able to narrow down to this view issue.
