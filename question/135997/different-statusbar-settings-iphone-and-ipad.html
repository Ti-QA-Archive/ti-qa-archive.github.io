Different statusbar settings iPhone and iPad

I am developing and iPad and iPhone universal application. So one code base and the deployment on both the iPad and iPhone. For the iPhone I want the statusbar to show. For the iPad I want it hidden.

Anybody a solution how to do this?

    Hello Wienke,

    Try this…

    var osname = Ti.Platform.osname;

if(osname == 'ipad'){
    statusbar.hide();
}
else{
    statusbar.show();
}
