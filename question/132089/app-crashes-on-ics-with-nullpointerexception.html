  • Not a real answer, but I managed to get it installed and running just fine by doing the following:

    1. Compile the app for disribution
2. Start the emulator using the AVD manager.
3. uninstalling the previous version installed in the emulator.
4. Running these commands from the Android platform tools:

    > ./adb connect localhost
> ./adb -e install <path to apk file>

    It ran just fine.
