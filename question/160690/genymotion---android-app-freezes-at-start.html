  • Do you have the Google APIs installed to your Genymotion emulator? If not, they're required. Search the net for the workarounds, since Genymotion doesn't formally support the Google APIs at this point in time.

    Genymotion is also not yet formally supported by Appcelerator. I have been using it for testing and have encountered no issues. It could be a configuration issue. The error messages above seem to indicate some sort of network issue. Can you launch the stock Android emulator?
