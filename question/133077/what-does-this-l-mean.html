What does this "L" mean?

When creating a new sample project, there are some "L" being used in the code. Like:

var win1 = new AppWindow(L('home'))

or

title: L('settings')

What is the meaning/use of the "L"?

    Hello Andre,

    Titanium provides two functions for obtaining a localized string from your resource files. Both take the key of the string requested as their first parameter. The L() macro is a short form for Ti.Locale.getString():

    You check here full documentation about internationalizing your application.

  • L is short for Ti.Locale.getString('ref') which references a string in the il8n/en/strings.xml file.

    It is used to enable localisation on the application.
