Toolbar button style (iPhone) problem

I use a toolbar to display a progressbar

ULtoolbar = Ti.UI.createToolbar({
    top: 0,
    left: 0,
    width: 320,
    height: 44,
    style: Titanium.UI.iPhone.StatusBar.DEFAULT
});
ULtbTitle = Ti.UI.createProgressBar({
    left: 10,
    width: 210,
    min: 0.0,
    max: 100.0,
    value: 0.0,
    message: 'Uploading picture',
    color: '#ffffff',
    style: Titanium.UI.iPhone.ProgressBarStyle.BAR
});
ULtoolbar.add(ULtbTitle);
var btn_annuler = Ti.UI.createButton({
    style: Titanium.UI.iPhone.SystemButtonStyle.BAR,
    image: null,
    right: 10,
    width: 80,
    height: 30,
    title: 'Cancel'
});
ULtoolbar.add(btn_annuler);
ULtbTitle.show();
win.add(ULtoolbar);

The problem is the style of the created button. I can set a PLAIN style button and everything displays correctly, but when I set it to DONE, it shows a blue arrow as a left image and plain text beside it, instead of creating a regular blue button as I use everywhere else in the application, usually in Navigation bar…

Among that, the progress bar doesn't get updated when I set it with onsendstream function…

xhr.onsendstream = function(e) {
    prog.value = (e.progress * 100.0);
};
