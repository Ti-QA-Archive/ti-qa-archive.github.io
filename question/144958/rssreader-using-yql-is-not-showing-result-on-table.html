RssReader using yql is not showing result on table

  • You've got a number of syntax problems.

    1: results is not an array, its an object
    2: it has an array within it called "p"

    Your code should perhaps read more like

    for (var i = 0; i < results.p.length; i++) {
    var rss = results.p[i];
