Is opacity = 0 the same as hide()

Is opacity = 0 the same as hide(), and does one perform better than the other?

  • Hi Marshall

    No, they do not perform the same function. You can demonstrate this on your system, using the following usecase:

    var win = Ti.UI.createWindow({
    backgroundColor:"white"
});

var view1 = Ti.UI.createView({
    backgroundColor:"red"
});
win.add(view1);

view1.addEventListener("click", function(){
    Ti.API.info("*** The view has been clicked ***");
});

win.open();

// toggle each of these lines in turn, to see their effect
//view1.opacity = 0;
//view1.hide();

    Hope this helps

  • Functionally, I am able to get setting opacity to 0.0 to hide objects, so in this sense they "do" the same thing.

    Technically, I cannot affirm that they actually "do" the same thing.

    Performance-ally, I don't know how one would test this. There were no apparent performance issues with one or the other.

    I'm using Titanium Mobile 1.6.1 and Android 1.6.
