ListView performance on Android 2.3.x (Gingerbread)

Hello,

I am facing performance problem with scrolling Listview on Android 2.3. I used TCE Lab 04 (https://wiki.appcelerator.org/display/td/04+-+TableViews+and+ListViews?desktop=true) as source and test it on different devices and OSes.

All seemed to be good until I use device with Gingerbread. On Android 2.3 Gingerbread scrolling is not smooth and begin unusable. Debugger didn't show anthing weird only that app allocate memory when scrolling is performed. I had been testing it a while and doing scrolling and sometimes performance begun good and same time debugger stopped showing anything at all - weird but this time scrolling was good. I used ListView with 20, 200 and 2000 items within it and all cases ran into same results.

Used devices:

  1. Huawei honor, Android 2.3.6 (Gingerbread)
  2. HTC Desire, Android 2.3.3 (Gingerbread)

Sources was compiled with Ti SDK 3.1.3, Alloy 1.2 and also with new Ti SDK 3.2/Alloy 1.3 using Mac OSX Mavericks

I know that Gingerbread does not support Hardware acceleration but is the any workaround how to make ListView to be scrolled smoothly? Or have someone faced the same problem and found the solution?

Thank you very much.
