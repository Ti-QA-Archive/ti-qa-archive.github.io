  • Hi Balakrishna,

    For that you need to maintain one array list of selected record unique id.. and click event of button you can remove all selected record. here i explain you basic step how's achieve it. it's not a compiled code so it's just for hint :) so you can get idea about it

    e.g

    function editList(operation, record_id) {
    if (operation == "add") {
        arrSelectedRecord.push(record_id);

    } else {
        var index = arrSelectedRecord.indexOf(record_id);
        arrSelectedRecord.splice(index, 1);

    }

}

/*
This addToList function is called on click of check box which is in row.
and isSelect and recordID is a custom property of isSelect is initially false to all and recordID is the recordID which is assign when you create the row

*/
function AddToList(e) {

    if (e.source.isSelect) {
        e.source.image = "uncheck.png";
        editList("remove", e.source.recordID);
    } else {
        e.source.image = "check.png";
        editList("add",  e.source.recordID);
    }
    e.source.isSelect.isSelect = !e.source.isSelect;

}

    So after that you have all selected record id. so you can do anything whatever you want.
