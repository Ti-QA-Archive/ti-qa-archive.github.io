Problem with codeboxed AudioRecorder module and SDK 1.8.1

Hello,
my problem is that the following error arise when I try to use the com.codeboxed.audiorecorder module with the SDK 1.8.1:

Uncaught Error: Requested module not found: com.codeboxed.audiorecorder

I am using Titanium SDK 1.8.1 and V8 over a Motorola XOOM with Android SDK 3.1

This is what I did to use this module:

  • Download the code from here and move it to the root folder of the project.

  • Add the module definition to the TiApp.xml:

<modules>
  <module platform="android" version="0.1">com.codeboxed.audiorecorder</module>
</modules
  • follow the instructions from codeboxed to use it, starting with:
var audioRecorder = require('com.codeboxed.audiorecorder');
  • The content of the module manifest is:
    ~~~
    #

    this is your module manifest and used by Titanium

    during compilation, packaging, distribution, etc.

    #
    version: 0.1
    description: My module
    author: Your Name
    license: Specify your license
    copyright: Copyright (c) 2011 by Your Company

these should not be edited

name: audiorecorder
moduleid: com.codeboxed.audiorecorder
guid: 747dce68-7d2d-426a-a527-7c67f4e9dfad
platform: android
minsdk: 1.7.0

.

With the 1.7.5 SDK it was working perfectly but when I changed to 1.8.1, problems start to appear.

The first problem appears when it starts to compile the app:

[ERROR] The 'apiversion' for 'com.codeboxed.audiorecorder' in the module manifest is not a valid value. Please use a version of the module that has an 'apiversion' value of 2 or greater set in it's manifest file

I added this *apiversion* to the manifest:

apiversion 2.0

and I also modified the TiApp.xml with the new version:

<modules>
<module platform="android" version="2.0">com.codeboxed.audiorecorder</module>
</modules

Now it compiles all the app and install it on the tablet but when I try to run the app, the following error appears:

E/TiJSError( 4563): (main) [239,571] —– Titanium Javascript Runtime Error —–
E/TiJSError( 4563): (main) [0,571] - In ti:/module.js:357,8
E/TiJSError( 4563): (main) [0,571] - Message: Uncaught Error: Requested module not found: com.codeboxed.audiorecorder
E/TiJSError( 4563): (main) [0,571] - Source: throw new Error("Requested module not found: " + request);
~~~

I have no idea about the problem and I couldn't find the solution by myself or in Internet. To move to 1.7.5 is not an option for me.

Any idea?

Thank you very much,
Diego.
