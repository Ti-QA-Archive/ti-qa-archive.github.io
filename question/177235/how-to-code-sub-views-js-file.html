How to code sub-views js file?

I created a layout following this example using Require function in alloy, figuring to componentize my views would be smart. I can't code it now!

In my index.xml:

<Alloy>
<Window class="container">
    <Require src="home" id="home"></Require>
</Window>
</Alloy>

In my index.js

$.index.open();

In my home.xml

<View id="body">
    <View id="tabView">
       <Button id="btnA">A</Button>
       <Button id="btnB">B</Button>
    </View>
     <View id="content">
    </View>
</View>

Now in my home.js

var args = arguments[0] || {};
$.btnA.addEventListener('click',function(e){
    Ti.API.info(e);
    alert('A Click'); 
});
$.btnB.addEventListener('click',function(e){
    Ti.API.info(e);
    alert('B Click'); 
});

No events are logged and no alerts appear when clicking. I tried changing home.xml to

<Button id="btnA" onClick="doA">A</Button>

then with this code in home.js 

function doA(e){
    Ti.API.info(e);
}

and even this

var doA = function(e){
    Ti.API.info(e);
};

but it still fails. When I move those functions to above $.index.open(); on index.js then it complains it can't find the functions in home.js. What am I doing wrong?

    Hi guys,

    This is very simple, try this code sample and compare with your code where you mistake.

    index.xml

    <Alloy>
    <Window>
        <Require src="home"></Require>
    </Window>
</Alloy>

    index.js

    $.index.open();

    home.xml

    <Alloy>

    <View id="body">
        <View id="tabView">
            <Button id="btnA" width="50" height="50" top="20">A</Button>
            <Button id="btnB"  width="50" height="50" top="100">B</Button>
        </View>
    </View>

</Alloy>

    home.js

    var args = arguments[0] || {};
$.btnA.addEventListener('click',function(e){
    Ti.API.info(e);
    alert('A Click'); 
});
$.btnB.addEventListener('click',function(e){
    Ti.API.info(e);
    alert('B Click'); 
});

    If you have any question,then let me know.

    Thanks.
