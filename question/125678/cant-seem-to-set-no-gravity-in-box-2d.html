Can't seem to set no gravity in Box-2D

I'm just messing around with the pinball demo that's included in the module. Code looks like:

var world = box2d.createWorld(gameBoard);

        // This will be used to shoot the ball "harder" the longer the user holds the plunger
        var impulseIntervul, impulse = 1, score = 0;

        // Set the gravity so that the ball "falls" correctly
        world.setGravity(0.0, 0.0);

I later add a "ball" to the stage (I've done some minor modification to the code)

app.world.ball = world.addBody(Ti.UI.createView({
            backgroundColor : "#ffffff",
            width : 10,
            height : 10,
            right : 230,
            bottom : 120,         
        }), {
            radius : 10,
            density : 0,
            friction : 0,
            restitution : 0,
            type : "dynamic"
        });

when I load the game, the ball falls straight to the bottom of the screen.

What am I doing wrong?

  • Set the gravity after you start the world.
    This works for me:

    // start the world
world.start();
world.setGravity(0,0);

    Later I have a button that sets the gravity back

    btn.addEventListener('click', function(){
    world.setGravity(0,-9.91);
});

    It's just the modified app.js in the example.
