  • For those who want to use this great extension for Android, here is the User Agent:

    var androidUserAgent = 'Mozilla/5.0 (Linux; U; ' + Titanium.Platform.name + ' ' + Titanium.Platform.version + '; ' + Titanium.Locale.currentLocale + '; ' + Titanium.Platform.model + ' AppleWebKit/533.1 (KHTML, like Gecko) Version/4.0 Mobile Safari/533.1';
