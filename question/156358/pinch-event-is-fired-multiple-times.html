Pinch event is fired multiple times

I have added an event listener for pinch event to a TableView (an alert is generated on pinch). But every time I pinch, the event is fired multiple times (I also observed that the number of times the event is fired isproportional to the pace with which one pinches the screen i.e. the longer you hold on to the screen, more are the number of alerts generated). How can this be controlled to fire the event once?

  • I think this is normal behaviour.

    It fires as your fingers move, providing the difference in scale from start to finish.

    As the documentation states "A pinch is a touch and expand or contract with two fingers. The event occurs continuously until a finger is lifted again."
