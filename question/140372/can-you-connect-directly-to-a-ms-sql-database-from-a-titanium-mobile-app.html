  • Hello, I recommend you to develop a thin business logic layer and build a OData endpoint in .NET 4.0 WCF Data Services. With this approach you can publish only the necessary entities and service methods. After publishing the OData endpoint, you can access it from JavaScript (in this case with JayData).
    You can read my tutorial about accessing OData service from Titanium: Using JayData in Titanium to access remote OData service
