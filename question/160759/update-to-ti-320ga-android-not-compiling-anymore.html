  • Ok, there is nothing like publish a problem to "see the light".

    What the hell was that C:\Users\manuel.conde\AppData\Roaming\Titanium\mobilesdk\win32\3.2.0.GA\node_modules\titanium-sdk\lib\emulators\genymotion.js file? If I don't have Genymotion installed anymore…

    So, what would happen if I delete this file…? Voilá, problem solved. Deleting this file seems to force to use the other existing adv.js file… that works.

    — answered December 23rd 2013 by Manuel Conde Vendrell
    permalink
    3 Comments

    • genymotion.js is an emulator implementation library for Genymotion. It appears it's trying to scan the Program Files directory looking for Virtual Box and stumbles across a restricted directory.

      You can safely remove the genymotion.js file.

      I've created a ticket to fix this: https://jira.appcelerator.org/browse/TIMOB-16074. Feel free to watch this ticket to track progress.

      — commented December 23rd 2013 by Chris Barber
