    • Here are the steps which I found out my issue.

      1. Make sure you use the Titanium updated to install the Titanium CLI 3.2.0

      2. Use "titanium -v" to check the version if it is 3.2.0. It wasn't in my Mac before.

      3. Use "which titanium" to find out where the "titanium" it is .

      4. Use "sudo find / -name titanium -print" to find if there are another "titanium" command installed

      5. execut the commands which you find out to know which one is for 3.2.0.

      6 Then change/link the default "titanium" to 3.2.0 bin.

      — commented January 6th 2014 by Joe Zhu

    • Hi Joe
