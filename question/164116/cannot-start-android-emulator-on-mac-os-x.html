  • Hi,
    Can you please check the version of your ADT tool. As there were issue with ADT version 22.6.1 as this update contains a bug (to be fixed in 22.6.2) that prevents the creation of AVDs from the command line. This impacts Studio and any other Android-related tooling that uses this functionality.

    Please go through the link below for more information :
    http://www.appcelerator.com/blog/2014/03/updating-to-android-22-6-1-tools/
