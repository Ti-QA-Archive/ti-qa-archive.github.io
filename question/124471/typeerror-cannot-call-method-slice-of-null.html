  • Here is function that checks if variable contains string value:

    function isString(string_val) {
    if (typeof string_val === 'string') {
        return true;
    }
    return false;
};

    Now you can check if your variable is string:

    if (isString(myVar)) {
    myVar.slice(0);
}
