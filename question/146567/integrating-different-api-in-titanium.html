    Hi

    These are standard yahoo YQL queries, you can do these on any platform including a standard web browser, so Android should be fine.

    Titanium supports YQL directly; but not the authentician required. YQL, so you will need to use oauth.

    You can play with these in a normal browser to get the queries right before putting them in the app.

    YQL console

    Quick tip, the JSON return format is MUCH easier to process than XML.
