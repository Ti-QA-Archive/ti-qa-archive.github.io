  • My question is: How listing files, to get your name, in lib or asset folder.

    Creating a folder in asset folder, it will be available in the root of project. Then I'm create a myfolder in asset folder and put many files in this folder.

    To get the list of files, I used:

    var myFiles = Ti.Filesystem.getFile(Ti.Filesystem.getResourcesDirectory() + "/myfolder").getDirectoryListing();

    And to run a specific file in this folder, I used a slash before of path of file in require.

    require("/myfolder/" + myFiles[0]);
