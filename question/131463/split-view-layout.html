  • You can specify widths using percentages; eg

    var win = Ti.UI.createWindow();
var leftView = Ti.UI.createView({top:0, left:0, width:'50'});
var rightView = Ti.UI.createView({top:0, left:'50%', width:'50'});
win.add(leftView);
win.add(rightView);

win.open();

  • You haven't even mentioned what platform(s) you need to target. If you really need numerous images plus a label and they need to be big enough to tap (as opposed to notification area icons) then one would surmise you are only targeting tablets.

    And by split view do you mean an ipad split window, or just two views which are side by side?

    Since all UI components are generated dynamically, at run time you can just accumulate the total width required for the left and right side action images then compare those to the device width and calculate what you want for each view width. Then proceed with creating the views.
