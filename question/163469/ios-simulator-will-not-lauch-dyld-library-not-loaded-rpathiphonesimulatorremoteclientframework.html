    • I also had to follow Dan Tamas' advice and update titanium

      sudo npm install -g titanium
sudo mv /usr/bin/titanium /usr/bin/titanium.3.1.2

      And now use the 'ti' command to build

      ti build --platform ios --retina --debug-host localhost:8999

  • Found it!!!

    Apparently /usr/bin/titanium is still 3.1.2 whereas /usr/bin/ti is 3.2.1

    $ which titanium
/usr/bin/titanium

$ titanium -v
3.1.2

$ which ti
/usr/bin/ti
$ ti -v
3.2.1

    My build now succeeds with the new 'ti' command

    I'm moving titanium to /usr/bin/titanium.3.1.2 to get it permanently out of the way

    THANK YOU SO MUCH FOR THE HELP. I think it was a combination of upgrading to 3.2.2 and the npm update of titanium that fixed this issue.
