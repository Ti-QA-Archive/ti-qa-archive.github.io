Displaying a collection of collection elements - Alloy

Hi everyone.
I have a collection of elements, each of it element has a collection too, like this:

"result" : [ {
    "id" : 1,
    "name" : "Animals",
    "childrenCategories" : [ {
      "id" : 7,
      "name" : "fish",
      "childrenCategories" : [ ],
    }, {
      "id" : 5,
      "name" : "dogs",
      "childrenCategories" : [ ],
    } ],
  }, {

    "name" : "Earth",

  }]

I know how to display all attributes of each element, as following:
