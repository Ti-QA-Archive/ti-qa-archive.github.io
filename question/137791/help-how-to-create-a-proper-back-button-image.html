Help! How to create a proper Back Button image?

Hi,

I purchase a theme from AppVaultDesign and it comes with a back button image that looks like this
http://cl.ly/2h3x0j0p2v262k0k2U0e

However, when I add it to the code, the alignment looks wrong. The text is too close to the button.
http://cl.ly/393s2D0f0v3P0m3H150O

var backButton = Ti.UI.createButton({

    I think the root of the problem is the text is being centered between the far left and right edges of the button, including the angled area. If you are not using internationalized strings and always use the word "Back", then one quick and dirty solution is to experiment with adding spaces to the left as in:

      title: ' Back',    or
title: '  Back',

    until you get the desired look.

    Another thing I would recommend with an image like that is using the backgroundLeftCap property to make sure the left side of the image is not stretched if you use the image for a wider button. Unfortunately, I don't think there is a paddingLeft property as there is with fields. Or at least it is not documented by Titanium. You could try it and see what happens.
