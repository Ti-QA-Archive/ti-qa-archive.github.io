Android picker size really small

I have read this thread from 2008 http://developer.appcelerator.com/question/6851/change-height-of-the-picker

Has anything changed since then on the size of the data picker on Android?

It's supported in the documents as far as I have read, however when I add a picker, its so small you can't hardly use it in a Android. I just want to fill the view it's sitting in on another app (not this example code).

My test code:

var win1 = Titanium.UI.createWindow({
    backgroundColor:'#fff',
});

        var picker = Ti.UI.createPicker({
        backgroundColor: '#ff9900',
        });
        // Add the tens
        var column1 = Ti.UI.createPickerColumn({opacity:0});
        for(i=0;i<10;i++) {
            column1.addRow(Ti.UI.createPickerRow({title:i.toString(),custom_item:    i.toString()}));        
        }
        // Add the dollars
        var column2 = Ti.UI.createPickerColumn();
        for(i=0;i<10;i++) {
            column2.addRow(Ti.UI.createPickerRow({title:i.toString(),custom_item:i.toString()}));        
        }
        // Add the change
        var column3 = Ti.UI.createPickerColumn();
        for(i=0;i<100;i++) {
            if(i<10) {
                column3.addRow(Ti.UI.createPickerRow({title:".0"+i.toString(),custom_item:".0"+i.toString()}));
            } else {
                column3.addRow(Ti.UI.createPickerRow({title:"."+i.toString(),custom_item:"."+i.toString()}));
            }    
        }

        // add it up
        picker.add(column1);
        picker.add(column2);
        picker.add(column3);
        picker.useSpinner = true,

        // turn on the selection indicator (off by default)
        picker.selectionIndicator = true;        
win1.add(picker);
win1.open();

  • Found out that you can change the width changing the column's width. Remember to do that after adding some data to the picker.

    picker.add(rRows);
picker.columns[0].width = Ti.Platform.displayCaps.platformWidth;
