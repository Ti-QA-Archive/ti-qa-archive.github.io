  • Hi muhammad

    Appcelerator titanium converts the app in native in the build does'nt mean that it makes the iphone class or java code. You cannot able to view the .java files from the code developed in appcelerator titanium. While you can sure open the project in android and Iphone xcode as of it like the native project only. And also able to make the hybrid implementation of half developed in titanium and other in IPhone or android.

    So make sure before you go in to deep by reading the appcelerator docs. There no way you get the .java using the project developed in appcelerator titanium.
