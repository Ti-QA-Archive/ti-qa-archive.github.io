How do I send SMS on Android 5.0 (Lollipop)

Here is my code that does not work on 4.4+….

var intent = Ti.Android.createIntent({
    action : Ti.Android.ACTION_VIEW,
    type : "vnd.android-dir/mms-sms"
});

intent.putExtra("address", "My text message");
intent.putExtra("sms_body", 1234567890);
Ti.Android.currentActivity.startActivity(intent);

After doing some research and reading links such as this and that, I wrote the following code for my Android 5.0.

var intent = Ti.Android.createIntent({
    action: Ti.Android.ACTION_SEND,
    type: "text/plain"
});
intent.putExtra("Intent.EXTRA_TEXT",SMSMessage);

This code prompts the message picker option, and once I select an option (such as messenger), I get an empty window. What needs to happen is that once the user selects a messaging app from the picker, the window should be populated with the text and the phone number (which I don't know how to pass through my code.

I am hoping that someone has already solved it and can share some code / wisdom. Thanks!

    Hi Dev.

    Try using sms_body instead of Intent.EXTRA_TEXT.

    e.g

    intent.putExtra("sms_body", "HELLO");

    it's working fine for me…

    • Thanks @Nilesh Patel. That worked like a charm. Here is my complete code that works on Android 5.0.

      var intent = Ti.Android.createIntent({
        action: Ti.Android.ACTION_SENDTO,
        data: 'smsto:123456789'
    });
    intent.putExtra('sms_body', SMSMessage);

    Ti.Android.currentActivity.startActivity(intent);
