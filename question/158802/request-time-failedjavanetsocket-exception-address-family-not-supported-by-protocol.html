  • Well, reading from your code, you are trying to access your server using the 127.0.0.1 URL.

    This will not work even in the emulator. The reason for that is quite simple. This url represents the local machine itself. But emulator or device has an IP address of its own. So if you want to access your server from your mobile device, you will have to use the actual server address.

    And as long as they are both on the same network, it should work without a hitch. One little trick is to try to access the URL using the device's web browser. If it can each the URL this way, there is no reason why it won't work from your code.

    Hope this helps.
