    Hi Olga,

    Its possible but you will need to create a module for that. One caveat: Android modules are Android exclusive. IOS modules are IOS modules exclusive. Android compiles Java, IOS SDK compiles obj-c.

    Check this link for Android:

    http://developer.appcelerator.com/doc/mobile/android/module_sdk

    cheers,
