  • Hi,

    If you are referring to the window's nav bar; You need to define a few properties, which you can do at the controller's tss file, or globally in app.tss

    app.tss

    // default window
"Window": {         
     extendEdges: [Ti.UI.EXTEND_EDGE_TOP, Ti.UI.EXTEND_EDGE_BOTTOM]
    ,includeOpaqueBars: true
    ,autoAdjustScrollViewInsets: true
    ,backgroundColor: '#FFFFFF'
    ,navBarHidden: false
    ,barColor: '#1D6CB9'
    ,navTintColor: '#FFFFFF'    
    ,translucent: true
}
