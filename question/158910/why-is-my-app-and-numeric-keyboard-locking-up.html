Why is my app and numeric keyboard locking up?

Hello there. I am trying to make me TextFields accept only positive decimal input. However, entering valid input causes my app and numberic keyboard to lock up. Here is my code:

textBox.addEventListener('change', function (e) {
    var patt = new RegExp('/^\d+(\.\d+)?$/','gi');
    if (e.value.match(patt) == null) {
        textBox.value = loaBox.value.slice(0, -1);
    }
});

Is there any reason why?

  • Actually, I found out a better solution that doesn't lock up:

    textBox.addEventListener('change', function (e) {
    var isValid =  /^\d*\.?\d{0,2}$/.test(loaBox.value);
    if (!isValid) {
        textBox.value = loaBox.value.slice(0, -1);
        textBox.setSelection(loaBox.value.length, loaBox.value.length);
    }
});
