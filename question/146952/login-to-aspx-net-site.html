  • Wee It is solved.

    Just sharing with you, because I think i was not easy.

    Downloaded the program Charles, to see what kind of request was sent if it was a normal login with browser.

    Found out that a lot of the parameters from the webpage should be included in the POST request.
    So I parsed the page and put the info into a postdata string with was included when I sent the request.
    … and succes

    Remember to encode some of the postdata.
