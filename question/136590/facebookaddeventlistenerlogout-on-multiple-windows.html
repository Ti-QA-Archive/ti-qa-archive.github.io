Facebook.addEventListener('logout') on multiple windows?

I use Facebook to login/logout of my mobile app, which works great. But I have multiple windows and on my settings screen, as well as on my main screen, users can login or logout of the application.

I do that by listening to the "login" or "logout" event by using Titanium.Facebook.addEventListener('logout', updateLoginStatus);

The problem is that I don't want the Facebook logout event from my main screen to be triggered in my settings window. I want to run separate application logic depending on which screen a user is logging in or out

Is that possible?

    I'd recommend adding event listeners for each window and checking whether the window is visible before running the logic specific to that window:

    // window initialization:
var win = ...; 
Ti.Facebook.addEventListener('logout', function(e) {
  if (win.visible) {
    // do window-specific logic
  }
});

    If there aren't many windows in your app, it won't be a big deal to have a listener per window. If you have a large number of windows or are concerned with the number of listeners, you could instead add and remove the logout listener as the window is opened and closed (or blurred).
