How do I make a Swipe / Slide from left or right effect / animate using tabs?

Hey guys, here is the thing, I want to make a Swipe / Slide from left and right effect similar to the ones in Tweetdeck or Launcher Pro, obviously I don't expect something as professional as that, but something similar, I researched it quite a lot, and have not found a way, the closest thing I found was the Android View Pager in the Market that only work, in well, Android, so that doesn't cut for me.

I know tabs don't support animation, but windows do, however, I'm lost how to do it since I have several dynamic tabs and windows in the app, anyone know how can I do that?

Tweetdeck swipe effect

I managed to do a transition effect using the standard Ti.UI.iPhone.AnimationStyle, but I didn't like that effect, and also the fact it doesn't work in Android isn't exactly a plus, here is how my code is at the moment after the event is triggered by a swipe.

Ti.App.addEventListener('tabSwipe',function(e){
    var tabIndex = e.tabIndex;
    var lastIndex = tabGroup.getTabs().length - 1;

    var testAnim = Ti.UI.createAnimation({
        left:0,
        duration:250
    });

    if (e.direction == 'left'){
        if (tabIndex == lastIndex){
             tabGroup.close();
            tabGroup.setActiveTab(0);
            tabGroup.open({transition:Ti.UI.iPhone.AnimationStyle.FLIP_FROM_LEFT});
        } else {
            var newIndex = tabIndex + 1;
            tabGroup.close();
            tabGroup.setActiveTab(newIndex);
            tabGroup.open({transition:Ti.UI.iPhone.AnimationStyle.FLIP_FROM_LEFT});
        }
    } else if (e.direction == 'right'){
        if (tabIndex == 0){
            tabGroup.close();
            tabGroup.setActiveTab(lastIndex);
            tabGroup.open({transition:Ti.UI.iPhone.AnimationStyle.FLIP_FROM_RIGHT});
        }
        else {
            var newIndex = tabIndex - 1;
            tabGroup.close();
            tabGroup.setActiveTab(newIndex);            
            tabGroup.open({transition:Ti.UI.iPhone.AnimationStyle.FLIP_FROM_RIGHT});
        } // end else
        } // end if

}); // end mainSwipe
