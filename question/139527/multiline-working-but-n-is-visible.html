  • If it helps anyone, my text coming through a JSON/datastream is interpreted different than if directly set.
    On SDK 3.2.0:

    mylabel.text = 'This is line 1. \nThis is line 2';  // No problem
mylabel.text = messageFromServer; // 'This is line 1. \nThis is line 2';  // Doesn't work
mylabel.text = messageFromServer.replace(/\\n/g, '\n');   // Fixes above
