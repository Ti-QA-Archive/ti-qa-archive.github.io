Add to top of scrollview (with vertical layout)

Is there a method of adding to new view to a ScrollView so that it appears at the top.

assuming view1, view2, view3 are valid views. The below code would put 3 at the bottom. Is there a method or some action I could do so that view3 would appear at the top3? My content is dynamic so this example is quite simplified.

$.scrollview.add(view1);
$.scrollview.add(view2);
$.scrollview.add(view3);

Thanks!

    Two options:

    1. Keep layout=vertical then remove all the views and add them in the correct order

    or

    2. Remove layout=vertical and update the top properties to reposition them based on the size of the ones added before it
