  • Hi !

    I've had the same problem a while ago.

    I've created two subfolders in Ressource folder : en.lproj and fr.lproj each including a file InfoPlist.strings with the following content :

    CFBundleDisplayName = "Volksfürsorge";
CFBundleName = "Volksfürsorge";

    Clear build folder and rebuild your app.
