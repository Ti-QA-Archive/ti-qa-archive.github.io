How to detect if the return or search key is pressed in the iOS keyboard

I have a quick question, probably a stupid one too.

I need to detect whether a user has pressed the return (or in my case 'search') key on the keyboard and in turn then perform a task.

It is attached to a textfield and in my UI I have a 'go' button with an onClick event that actually performs the search.

What I need is for the user to be able to press search on the keyboard and for it to do the same as above.

Here is my texfield

// Search Box
var SearchBox = Titanium.UI.createTextField({
    color:'#00675a',
    height:31,
    top:256,
    left:27,
    width:224,
    borderColor:'#e0e0e0',
    borderRadius:5,
    borderWidth:2,
    paddingLeft:5,     // pad left by 20 pixels
    paddingRight:5,    // pad right by 20 pixels
    autocorrect:false,
    returnKeyType: Ti.UI.RETURNKEY_SEARCH //adds blue search button
});

Here is my onClick event for the button

GoButton.addEventListener('click',function(e)
{
    var myname = SearchBox.value;

    SearchBox.blur();

   var SearchWindow = Ti.UI.createWindow({
           barColor: '#005349',
         titleImage: 'images/title.png',
        url:'search_detail.js',
        id:myname,
        top:0
        });
   Titanium.UI.currentTab.open(SearchWindow,{animated:true});
});

Anyone able to help?
