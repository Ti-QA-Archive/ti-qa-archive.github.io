Where to put window title of tabgroup when using navigation group

I am using an iOS Navigation Group were the first window is a basic window and the second window is a TabGroup. I can't figure out where to specify the Title when the TabGroup is shown. If I show the navbar of the window of one of the tabs, then the Title shows up, but it is in a second nav bar below the one created by the Navigation Group. Any ideas?

The environment is a mobile iOS 5.1 app, Titanium version 2.0.1.201204132053, SDK version 2.0.1 GA2. The project has only been run in the iOS Simulator (5.1 (272.1)) so far.


var firstwin = Ti.UI.createWindow({
    title: 'First Window'
});

var btn = Ti.UI.createButton({
    title: 'cick here'
})

firstwin.add(btn);

var win1 = Ti.UI.createWindow({
    title: "win1"
})

var tab1 = Ti.UI.createTab({
    window: win1,
    title: 'tab1'
})

var win2 = Ti.UI.createWindow({
    title: "win2"
})

var tab2 = Ti.UI.createTab({
    window: win2,
    title: 'tab2'
})

var tabgrp = Ti.UI.createTabGroup();
tabgrp.addTab(tab1);
tabgrp.addTab(tab2);

var navGrp = Ti.UI.iPhone.createNavigationGroup({
    window: firstwin
});

var containerWindow = Ti.UI.createWindow();
containerWindow.add(navGrp);
containerWindow.open();

btn.addEventListener('click', function(e) {
    navGrp.open(tabgrp);
})

    dont open it with navGroup, each Tab has its own navGroup. its Not supposed to be used with navGroup. 

    tabGroup.open();

    when you need to go back

    tabGroup.close();
