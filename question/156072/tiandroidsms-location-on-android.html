  • Doewnload Module from TiAndroidSMS

    Change folder options to show hiddden folders.Module path will be like

    C:\Users\adnan\AppData\Roaming\Titanium\modules\android\ti.android.sms

    add module to tiapp.xml

     <modules>
        <module platform="android">ti.android.sms</module>
  </modules>

    Configured on Win7 using Titanium Sdk 3.1.3
