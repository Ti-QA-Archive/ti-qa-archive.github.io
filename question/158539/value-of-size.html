    Try this;

    var w = Ti.UI.createWindow();
var l = Ti.UI.createLabel({width:Ti.UI.SIZE, height:Ti.UI.SIZE, text:'This is a label'});
w.add(l);
w.addEventListener('postlayout', function(){
    alert('the label is w:' + l.rect.width + ', h:' + l.rect.height);
});
w.open();

  • After an element has been rendered on screen, you can query either of the following properties:

    • view.size() - returns an object with height & width properties describing the rendered size of the view
    • view.rect() - returns an object with x, y, height, and width properties describing the rendered position (relative to the parent) and size of the view

    See the docs for more info.

    Before an element is rendered on the screen, you cannot determine what the values of Ti.UI.SIZE/FILL will become as they have not yet been calculated.
