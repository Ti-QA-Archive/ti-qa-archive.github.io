  • First, you need to define a function that will set the current time to a label (in your case, label1)

    Once this is done, you use the setInterval function to run the previously defined function at an interval of your choice (5 seconds).

    This basically means that the code function will be ran every 5 seconds. 

    var label1 = Titanium.UI.createLabel();

function getTime() {
  var now = new Date();

  label1 = String.formatTime(now);
}

setInterval("getTime()", 5000); // runs every 5 seconds
