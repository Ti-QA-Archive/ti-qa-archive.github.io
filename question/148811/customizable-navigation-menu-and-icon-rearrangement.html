Customizable navigation menu and icon rearrangement

Hi,

We're re-designing the UI in our Titanium app and I was curious if we could accomplish this with Titanium:

We're currently using a tab group navigation at the bottom of our apps screen (our app is only deployed to iPads). As we continue to add new functionality to the app we no longer have room to add new tabs. We've come up with a similar, but more scalable design that we'd like to implement. Basically, we'll have a row of X number of icons (very similar to the tab group currently). There will be an arrow next to the tab group that when tapped will cause the tab group to expand upward and reveal additional rows if icons. I'm fairly certain we can accomplish this (although not with the native tab group control, i'm sure). We'd like our users to be able to set which icons appear in that top row and thus are viewable even when the tab group isn't expanded upwards. To do this we were hoping there'd be a way for the user to drag icons from the bottom to the top or vice-versa. This would be identical to how app icons are rearranged on the iPad.
