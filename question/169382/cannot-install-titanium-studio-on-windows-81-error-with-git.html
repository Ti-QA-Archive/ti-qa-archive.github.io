    Resolved….

    Step 1: download and install this file
    https://github.com/msysgit/msysgit/releases/download/Git-1.9.2-preview20140411/msysGit-netinstall-1.9.2-preview20140411.exe

    Step 2: download and install this
    https://github.com/msysgit/msysgit/releases/download/Git-1.7.11-preview20120710/Git-1.7.11-preview20120710.exe

    Step 3: install Titanium
