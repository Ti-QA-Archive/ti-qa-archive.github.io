    • so you need to set id for login window tag

      login.xml

      <Alloy>
    <Window id="winLogin">
        <Button id="loginButton" onClick="login" backgroundColor="#000">Login</Button>
    </Window>
</Alloy>

      login.js

      function login() {
    Alloy.createController('home').getView().open(); 
    $.winLogin.close(); 
}
