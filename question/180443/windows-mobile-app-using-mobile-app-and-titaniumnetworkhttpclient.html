The documentation says that due to restrictions with Mobile Web applications this won't work and offers two solutions.

Configure the destination server to support Cross-Origin Resource Sharing.
Use Ti.Network.httpURLFormatter in conjunction with a proxy on the server hosting the application.

I can't change anything on the server as its a client web service.

So does anyone have an example of how i might get this example to run on a mobile web app?
