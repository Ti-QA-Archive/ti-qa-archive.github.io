Grouped tables without borders?

I have a grouped table with 4 sections, which I am creating like this:

var dessertSection = Ti.UI.createTableViewSection();
    dessertSection.headerView = header4;
    for(var i = 0; i < dessert.length; i++) {
        dessertSection.add(Ti.UI.createTableViewRow({
            title : dessert[i].title,
            recipeID : dessert[i].recipeID,
        }))
    }

This creates a section, but each item in the row has the typical border.

How would I eliminate this border?

SDK 1.7.2, Studio 1.0, OS 10.7.1

  • Steve,

    My apologies, in reviewing your question, I realize that you were speaking of the separator, not the border of your tables. One of these settings should work, they are properties of the TableView object.

    // sets the color of your seperator
separatorColor:'#fff'
// sets the style, may not work with your GROUPED table
separatorStyle:Ti.UI.iPhone.TableViewSeparatorStyle.NONE
