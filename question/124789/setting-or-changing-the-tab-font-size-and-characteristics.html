setting or changing the tab font size and characteristics

I've spent hours on this and must be missing something How do I change the tab font size, type, etc. on the main tab group. Here's what I have, pulled from the test setup, tried all kinds of things, I just want to change the font size of the word MEETINGS, I've spent a couple hours or more searching, it should be this hard or hard to find. If you can help - how and more importantly - where is the doc for this? and if the doc isn't quite, how can I change it. This isn't the only time waster I've found, I"m willing to donate my time to change things:

// this sets the background color of the master UIView (when there are no windows/tab groups on it)
Titanium.UI.setBackgroundColor('#000');

// create tab group
var tabGroup = Titanium.UI.createTabGroup();

//
// create base UI tab and root window
//
var win1 = Titanium.UI.createWindow({
title:'Tabby 1',
backgroundColor:'#fff',
font:{fontSize:'32', fontStyle:'italic'}
});

var tab1 = Titanium.UI.createTab({
icon:'KS_nav_views.png',
title:'MEETINGS',
font:{fontSize:'32', fontStyle:'italic'},
window:win1
});

var label1 = Titanium.UI.createLabel({
color:'#999',
text:'I am Window #1111',
font:{fontSize:20,fontFamily:'Helvetica Neue'},
textAlign:'center',
width:'auto'
});

win1.add(label1);

//
// add tabs
//
tabGroup.addTab(tab1);

// open tab group
tabGroup.open();
