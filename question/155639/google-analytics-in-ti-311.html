    • i have facing same issue once , but i have solution for you, if you are using MVC architecture then cut and paste these three line in alloy.js , so you can require this module globally , do as below:

      alloy.js

var GA = require('analytics.google');
GA.debug = true;
GA.trackUncaughtExceptions = true;
Alloy.Globals.tracker = GA.getTracker("UA-12356-1");

      and you can access this module in other JS file through below sample code , like this you can use different methods of google analytics. 

      login.js

Alloy.Globals.tracker.trackScreen("login");
