Redraw the screen

Is there a way to redraw a Window with several children views?

Here's my use case:

I have a screen, with a table, lots of graphics, labels, colors, etc.

I want the user to be able to swap to a different theme, which uses a different set of graphics and colors.

All of the theme elements are set as variables, so all that's left to do is "redraw" the screen.

How would I do this? I didn't see anything in the KitchenSink, documentation, or the forums.

Any ideas?

  • You could attach event listeners to your table rows as you create them which would update the UI. Then when you want to change the theme just fire off an event.
