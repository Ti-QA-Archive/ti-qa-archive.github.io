How do I create a split action bar?

In an Android project, how do I create a Split Action Bar? Shoud I emulate it using a view with bottom:0?

    Hi

    Currently a split action bar is not supported using the ActionBar implementation available to us.

    To achieve your goal you will need to create a fake version yourself so that it looks like an action bar bottom but instead it will be made of a view and a series of buttons of other controls.

    There is no news on when the split version option will become available to us.
