  • Hi, go inside you module folder in titanium studio and go in till you find module.xcconfig

    add these lines in that.

    OTHER_LDFLAGS[sdk=iphoneos4.*]=$(inherited) Security
OTHER_LDFLAGS[sdk=iphonesimulator4.*]=$(inherited) Security
OTHER_LDFLAGS[sdk=iphoneos5.*]=$(inherited) Security
OTHER_LDFLAGS[sdk=iphonesimulator5.*]=$(inherited) Security
OTHER_LDFLAGS[sdk=iphoneos6.*]=$(inherited) Security
OTHER_LDFLAGS[sdk=iphonesimulator6.*]=$(inherited) Security

    and then check, it should work, also if any error then inform me about iOS SDK and Titanium SDK information(version).
