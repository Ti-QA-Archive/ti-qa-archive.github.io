Splash full screen on Android

Hi, folks. I saw that many people wish full splash screen in your app. Aparently the only way to do this is change theme.xml in build\android folder. However, any change in this file is lost when Titanium rebuild app. I figured out one way to resolve this problem.
1) Create folders <your project>\platform\android\res\values (see KitchenSink folder structure)
2) Put in folder "values" the file theme.xml modifyed (i.e. )

Done! When Titanium rebuild the app it will put your costumized theme.xml in the build folder. This is working fine with me. I use mobile SDK 1.7.0 and Android SDK 2.2 and 2.3.3.
