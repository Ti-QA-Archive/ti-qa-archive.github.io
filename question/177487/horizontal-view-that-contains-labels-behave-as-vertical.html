Horizontal View that contains labels behave as vertical

So, what I am trying to do is put two labels in a horizontal view. The problem is that I don't really know the width of the labels (It depends on a used text). I would like to achieve this behavior:

[Label1 Label2] // " " is a little gap, [] is parent view with a horizontal layout

["SomeText" "SomeText"]
["Some" "SomeText    "]
["SomeText" "SomeTe.."] // If the second label is too long it will show up just "..."
["SomeMoreText" "So..."]

Basically the first label (text) will be always completely visible ,but the second one will take only the rest of the parent's width.

Currently I have got this… however the second label is below the first one which is not right.

var win = Titanium.UI.createWindow({
    backgroundColor: "white"
});

var view = Ti.UI.createView({
    width: Ti.UI.FILL, height: 15,
    left: 40, right: 40,
    layout: "horizontal",
    backgroundColor: "black"
});

var leftLabel = Ti.UI.createLabel({
    text: "Lorem ipsum dolor sit dwdw",
    width: Ti.UI.SIZE,
    font: {fontSize: 10},
    left: 0,
    backgroundColor: "red"
});

var rightLabel = Ti.UI.createLabel({
    text: "Morbi in sem quis dui placerat ornare. Pellentesque odio nisi, euismod in, pharetra a, ultricies in, diam. Sed arcu. Cras consequat.",
    width: Ti.UI.FILL,
    font: {fontSize: 10},
    left: 5, right: 0,
    height: 10, // Do I really need this?
    backgroundColor: "yellow"
});


view.add(leftLabel);
view.add(rightLabel);

win.add(view);
win.open();

  • The problem is that both objects cannot have their width properties set to FILL. The first object will fill the entire container and wrap the second object. In a horizontal layout, the first object needs to have a fixed width. I've had this exact same issue occur this week.

    So, the fix? Set the first object.width to a "best guess maximum" fixed unit and the second object.width set to Ti.UI.FILL.

    As for whether you need a height property? If I want to vertically center a child object in a container, I set a fixed container height, then set the child object.height to Ti.UI.FILL.

    Good luck. If this answer works for you, please select as best answer.
