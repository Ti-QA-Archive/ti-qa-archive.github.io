By pressing the local notification tab how to redirect to a particular window

Hi I have an application where I get notification upon particular action. Now when I click on that notification my application is relaunching. But instead of that I need to go to particular window. For example if I have 2 windows say win1 and win2. If I open the application by clicking on the application icon I need to go to win1 and it is happening correctly. But when I click on the notification that I received its opening win1 now by default. But I need win2 to be opened. Please help me how to do this. I need to do this both in android and iphone. Currently I am getting notification on both android and iphone.
[I am using titanium 2.0.1, ios5 and android 2.2]
