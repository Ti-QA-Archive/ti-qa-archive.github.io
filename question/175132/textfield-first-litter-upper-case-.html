  • Hi,
    You can use this code to implement this functionality. This line you can call in your text field event listener. This will automatically convert your textfield first letter to capital.

    textField.addEventListener("click",function(e){

 var firstNameVal = e.value;
var propercaseFirstName = firstNameVal.toString().substr(0,1).toUpperCase() + firstNameVal.toString().substring(1,firstNameVal.length);

textField.value = firstNameVal;
});

    Try this code hope this works for u.
