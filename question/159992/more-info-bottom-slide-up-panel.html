  • Hi

    You have not shown any controller code that is used to trigger this line of code;

    $.detailsSlider.height ='80dp';

    Without this and the assumed map code you are not making life very easy.

    However you are better off making the panel height as it should be - do NOT set to zero as this makes everything else harder. Also do not use the property bottom as this will also make things harder for you.

    Your existing code shows a simple jump from 0 to 80, not exactly the slide animation you are wanting. When you do try the slide the height constantly change and the whole panel layout will need to redraw each time the height changes within the animation.

    Instead you should fix the height of the panel and set the top value of the panel to be the exact height of the visible area of the display.

    Then change the top property to the exact height of the visible space minus the height of the panel. This way the whole panel is visible and displayed correctly - just off screen to start with, then when you have it fully open it did not need to redraw anything.

    Once you have mastered this a quick use of the animation object and setting the height as mentioned above with give a nice look.

      "#detailsSlider": {
    bottom: '-80dp',
    height: '80dp',

    textAlign: Ti.UI.TEXT_ALIGNMENT_LEFT

      And when I need slider to appear I call

      exports.showAnnotationDetails = function(annotation) {
$.detailsSlider.
    $.detailsSlider.animate({
            bottom: '0dp'
        });

    $.details_title.text       = annotation.title;
    $.details_type.text        = annotation.subtitle; 
    $.details_description.text = annotation.description;
};

exports.hideAnnotationDetails = function() {
    $.detailsSlider.animate({
            bottom: '-80dp'
        });

    $.details_title.value       = '';
    $.details_type.value        = ''; 
    $.details_description.value = '';
};
