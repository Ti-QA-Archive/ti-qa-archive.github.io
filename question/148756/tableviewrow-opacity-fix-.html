TableViewRow opacity fix ?

Hi,

I've been searching on some forums and websites about the TableViewRow opacity properties (seems like it's documented but don't work) and didn't found anything about a way to fix this bug.

// These tests don't work
var row = Ti.UI.createTableViewRow({opacity:0.5});
var row = Ti.UI.createTableViewRow({backgroundImage:'/images/transparentbackground.png'});

Is there a working way to make a row transparent ?
