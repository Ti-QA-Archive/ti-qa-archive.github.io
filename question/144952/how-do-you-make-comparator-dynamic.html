How do you make comparator dynamic?

I'd like to make the sort field dynamic, because the end user has the ability to change the sort parameter, is there anyway to do this using this model?

exports.definition = {

    configuration : {
        // table schema and adapter information
    },

    extendModel: function(Model) {      
        _.extend(Model.prototype, {
            // Extend, override or implement the Backbone.Model methods                     
        });
        return Model;
    },

    extendCollection: function(Collection) {        
        _.extend(Collection.prototype, {

            // Implement the comparator method.
            comparator : function(book) {
                return book.get('title');
            }

        }); // end extend

        return Collection;
    }
}

  • Hi there-

    I just came across this question, and my guess is you have already answered this yourself. I needed the same thing as you, and what I did was the following (I have implemented a custom datagrid, with sortable columns):

    exports.definition = {

    configuration : {
        // table schema and adapter information
    },

    extendModel: function(Model) {      
        _.extend(Model.prototype, {
            // Extend, override or implement the Backbone.Model methods                     
        });
        return Model;
    },

    extendCollection: function(Collection) {        
        _.extend(Collection.prototype, {


           initialize:  function() {
              this.sortVar = 'title';
           }, 

            // Implement the comparator method.
            comparator : function(book) {
                return book.get(this.sortVar);
            }

        }); // end extend

        return Collection;
    }
}

    In the controller, when implementing the sort then, I simply do the following:

    function sort(e) {

  books.sortVar = e.source.sortVar; // Where sortVar is a custom attribute on a user selected label
  books.sort();

}
