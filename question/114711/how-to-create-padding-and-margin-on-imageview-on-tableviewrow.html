  • Hmm, I've had a similar problem but have only worked out my solution for iPhone thus far (i.e. I haven't tried it on android but should work too), you should be able to do something like this:

    var row = Ti.UI.createTableViewRow();
       row.height = 'auto';

var image1 = Ti.UI.createImageView({
    backgroundImage:'../images/image1.png',
    top:1,
    bottom: 6,
    left:1,
    right:1,
    width:36,
    height:36
});
row.add(image1);

var image2 = Ti.UI.createImageView({
    backgroundImage:'../images/image2.png',
    top:1,
    bottom: 6,
    left:1,
    right:1,
    width:36,
    height:36
});
row.add(image2);

    Side note: from what I've seen so far, I don't think things work exactly like CSS and web pages as such, unless using a WebView but it requires more resources as it would load a lot more stuff on initialisation, than the controls we're using.
