App running in background

Hello,
In my app im getting a json with waypoints and then im pasing these waypoint to the Navigation app with an intent, here is my code

var intent3 = Ti.Android.createIntent({
         action: Ti.Android.ACTION_VIEW,
         data: "google.navigation:q="+activeWaypointLat+","+activeWaypointLon
});
Ti.Android.currentActivity.startActivity(intent3);

When the startActivity starts,Navigation app starts and my app keep running in the background and checking the GPS current position, i want when the user reaches a certain position to fire an alert.
So far i can do this but the alert dont come up in front of the navigation app, if i close the navigation app and go back to my app im seeing the alert.
My question is if there's any way to bring my app (or the alert) to front or if i cant close the navigation app from my app (the same way that im starting it)

thank you

  • I managed to add a notification 

    Titanium.Android.NotificationManager.notify(
    0,
    Ti.Android.createNotification({
        contentTitle: 'Text '+jsonObj.waypoints[activeWaypoint].title,
        contentText: jsonObj.waypoints[activeWaypoint].info,
        defaults: Titanium.Android.NotificationManager.DEFAULT_ALL,
        tickerText: 'Text notification!',
        contentIntent: Titanium.Android.createPendingIntent({
            intent: Titanium.Android.createIntent({
            url: 'app.js'
             })
          })
}));

    Notification works as expected, is there any way to bring my app in the front when the notification fires?
