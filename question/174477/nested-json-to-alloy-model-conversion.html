Nested JSON to Alloy Model Conversion

I am having a nested JSON string as follows:-

var json1 ={
"p1": {
"c1": {
"key1": "val1",
"key2": "val2"
},
"c2": {
"key3": "val3",
"key4": "val4"
}
}
}

I have created 4 Alloy Models viz:-

BaseModel.js
columns:{
"p1" :"p1"
}

p1.js
columns:{
"c1" :"c1",
"c2:"c2"
}

c1.js
columns:{
"key1": "val1",
"key2": "val2"
}

c2.js
columns:{
"key3": "val3",
"key4": "val4"
}

I write the following line:-
var model = Alloy.createModel("BaseModel",json1);
My json is getting parsed and BaseModel is getting generated. I am able to get p1 using:-
bar parentObject = model.get("p1");

But the parentObject is not a model. I cannot do parentObject.get("c1"). the parentObject is a plain javascript object. I can access parentObject.c1 and get the required values.

Is it anyhow possible to create nested mdoel from a json string.

Any help is highly appreciated
