  • Guellai

    There are a number of examples in the KitchenSink, ie:

    • table_view_basic.js
    • table_view_layout.js
    • table_view_layout_2.js
    • table_view_api_basic.js
    • table_view_perf.js

    There are others in Resources/examples (search for "table_view"), although not all of them will be enabled for both platforms. Hence, see table_views.js to determine this.

    It would be best for you to run the KS on your device or emulator, find something that is close to your objective, and then work out which script is being run by checking the code.

    Hope this helps
