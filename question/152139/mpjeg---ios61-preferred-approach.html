MPJEG - iOS6.1, preferred approach

What is preferred approach for playing a MPJEG on iOS…?

I tried videoPlayer on iOS6.1 (simulator and device) with no luck. Soon as it goes to load the mjpeg it closes the window.

    var activeMovie = Titanium.Media.createVideoPlayer({
        autoplay : true,
        backgroundColor : '#000',
        url : oCamera.Url,
        height : Ti.UI.FILL,
        width : Ti.UI.FILL,
        mediaControlStyle : Titanium.Media.VIDEO_CONTROL_DEFAULT,
        scalingMode : Titanium.Media.VIDEO_SCALING_ASPECT_FIT
    });

I have tried a webview as a last resort and it seems to display only a single frame on iOS6.1 (simulator and device).

    var webview = Titanium.UI.createWebView({
        backgroundColor : "#000",
        url : oCamera.Url,
        height : Ti.UI.FILL,
        width : Ti.UI.FILL,
        scalesPageToFit : true,
        loading : true
    });

Mobile iOS6.1
OSX 10.8
Titanium 3.1.0.2
Titanium SDK 3.1
