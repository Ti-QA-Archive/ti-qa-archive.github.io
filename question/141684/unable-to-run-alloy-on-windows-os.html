  • Accepted Answer

    Right now only OSX is officially supported for Alloy development. All of our testing of Alloy's capabilities has been on OSX so far so there will definitely be issues with attempting to use it on any other OS. We were actually planning to start testing and coding for other OSes once this pre-release has had some time in developers' hands.
