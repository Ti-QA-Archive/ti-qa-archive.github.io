XMLRPC login not working

Hi I am trying to request XMLRPC url.
I am using the library of XMLRPC from this url

I am using below code to sent the request

var win = Titanium.UI.createWindow();
var xmlrpc = require('xmlrpc');

var client = xmlrpc.createClient({
  url: url + '/xmlrpc.cgi'
});

client
  .call('myMethod')
  .param({login : username})  // struct
  .param({password : password})  // struct
  .param({ remember: 'YES' })  // struct
  .param({ 'content-length': 0 })  // struct

    if (!res.ok) {

    Hi Dharmendra,

    do you try to login to Bugzilla?

    If yes, you need to use a structure like this, i think.

    <methodCall>
    <methodName>User.login</methodName>
    <params>
        <param>
            <value>
                <struct>
                    <member>
                        <name>login</name>
                        <value>
                            <string>xyz@xyz.com</string>
                        </value>
                    </member>
                    <member>
                        <name>password</name>
                        <value>
                            <string>mypassword</string>
                        </value>
                    </member>
                </struct>
            </value>
        </param>
    </params>
</methodCall>

    From Bugzilla docs, Bugzilla::WebService::Server::XMLRPC

    How XML-RPC WebService Methods Take Parameters

    All functions take a single argument, a <struct> that contains all parameters.

    The names of the parameters listed in the API docs for each function are the <name> element for the struct <member>s.

    Matthias
