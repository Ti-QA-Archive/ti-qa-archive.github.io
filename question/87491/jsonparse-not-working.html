JSON.parse not working

Hi all,

I'm trying to parse some JSON on my app, and JSON.parse seems to be failing to do so. It just returns a {}, when the original 'text' JSON was the one I pasted here http://paste.pocoo.org/show/302607/ (it's quite big :P)

On the other hand, using eval('('+text+')') works just fine. Any ideas? I'm using mobilesdk 1.5 for Android.

  • I had a similar issue with local JSON files.

    Try using something like 

    var f = Titanium.Filesystem.getFile(Titanium.Filesystem.resourcesDirectory, fileSource)
var content = f.read();
content = JSON.stringify(content); // <--- pay attention to this line
parsedContent = JSON.parse(content);
