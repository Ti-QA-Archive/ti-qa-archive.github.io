Create HTML and then display in a web view?

I am trying to take a bunch of string variables and information entered by the user and assemble it into HTML and display it in a web view so it's nicely formatted (and can be emailed).

I can figure out how to construct html from my data, but how do I assign this to a web view?

Ideally, when my window loads, I am doing the html assembly and then want to immediately show in in a full-window webview.

    try this:

    
var html_code = "<html><head></head><body><h1>Hello!</h1></body></html>";

var webview= Ti.UI.createWebView({
   html:html_code
});
