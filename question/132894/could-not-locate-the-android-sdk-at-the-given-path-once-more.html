    Hi Ivan,

    I think you're seeing this issue:

    https://jira.appcelerator.org/browse/TISTUD-1115

    In talking to QA, this should only be an issue if you have a broken SDK build installed. Apparently Studio uses a script in the Ti Mobile SDK to verify that Android is installed, and it always uses the latest Ti Mobile SDK installed. It looks like you're using a CI build, so possibly there was an issue with the build you downloaded.

    I can't see why this would be an issue for only one project.

    You could try removing the current CI build that seems to be causing the problem, downloading a new CI build, or you could try copying the avd.py file from a different SDK.

    Please let me know if this fixes the problem.

    Cheers,
    -Arthur
