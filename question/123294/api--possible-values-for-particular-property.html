API , Possible values for particular property

Dear All,

I have some confusion related to API.

e.g.

Suppose I have Titanium.UI.createView
There is backgroundColor property for setting Background color for view.

Now I want to make it transparent. I don't know how to do it.
From the Q&A I came to know the we can put backgroundColor:'transparent'.

So the question is how to find the possible values for backgroundColor and for all other properties for all other components ? How can I find it in API ?

  • BackgroundColor can be any color you want from the RGB. So you can just look up your color and give it as RGB, for example "#ffffff" is white and "#000000" is black.

    You can search a color by using photoshop for example, and it will tell you the RGB color.

    You can also type the standard values like 'Black', 'White', 'Red', etc.. But I always use the RGB.

    Hope this helps
