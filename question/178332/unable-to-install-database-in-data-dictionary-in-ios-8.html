var db = Titanium.Database.install('MYDB.db', Ti.Filesystem.applicationDataDirectory+'MYDB');

    try it:

    db = Titanium.Database.install('/MYDB.db','MYDB');

    yo do not need to use Ti.Filesystem.applicationDataDirectory if you need more documentation read:
    https://wiki.appcelerator.org/display/guides2/Working+with+a+SQLite+Database
