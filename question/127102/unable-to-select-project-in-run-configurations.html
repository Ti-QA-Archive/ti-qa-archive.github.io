  • In the end I removed the project from Studio and recreated it. I then used the run arrow as suggested and it worked! I think there was a problem with the original project which stopped this working first time around. As far as I can tell the "Browse" button does not work - even with existing fully functional projects.

    In summary, if the 'run' green arrow button does not work then there is probably something about your project that Studio does not like. Try Re-creating the project and pressing run again. It should ask you if you want to create a Run Configuration. Don't waste your time (like me) wondering why the Browse button won't let you select a project - it just won't.
