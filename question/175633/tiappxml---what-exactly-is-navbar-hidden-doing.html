tiapp.xml - what exactly is navbar-hidden doing?

In my tiapp.xml navbar-hidden is set to true. However, I don't find it affecting anything. On iOS, the nav bar is showing. On Android, the action bar is showing. So what does this key do?

Running on SDK 3.2.3, iOS 7, Android 4.4.

  • Back in the day, splash screens had a thin navbar at the top of them. This would remove it.

    Since we can play with android themes now, I prefer to do it there.

    hint: android:Theme.NoTitleBar.Fullscreen
