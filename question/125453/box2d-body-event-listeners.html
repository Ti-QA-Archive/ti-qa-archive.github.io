Box2D body event listeners

Hi,

I've started playing around with ti.box2D and was wondering if there are event listeners for bodies added to the world?

I've noticed on the pinball app that events are being handled by a transparent overlay on top of the whole gamescreen.

Thanks in advance (plus this module is very fun to use, hoping for an android release in the near future)

    There is currently no event listener for adding a body, and as of now, none is planed. The reason for this is that Box2D it self does not have such an event listener. The only events that the module will handel are those generated by the world.

    As for the overlay, the reason it deals with the events of clicking is that if we did not do that if you tapped on a bumper (or any other body) rather then the bg, and the event were registered on the bg then nothing would happen. Using an overlay provides a more consistant experience.
