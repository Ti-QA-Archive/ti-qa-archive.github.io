SearchBar: Show cancel not working

In my IPhone app, the property showcancel:false is not working at all. I am using 1.6.1 SDK. Please help!

  • I had issue with this because it seems to fire a showCancel event on focus. Setting

    searchBar.addEventListener("focus", function() {
    searchBar.setShowCancel(false);
    });

    stopped the Cancel bar appearing completely.
