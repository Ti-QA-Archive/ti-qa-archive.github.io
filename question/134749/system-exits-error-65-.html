  • Hi guys, try this solution first for this kind of problem, Update your Markdown parser

    1: Download the latest release from https://github.com/trentm/python-markdown2
2: Unzip the resulting download*
3: Using the setup.py script in the directory that was created from expanding the zip file you downloaded, install with `sudo python setup.py install`

4:Re-run build.py

    then clean your project and try rebuild, this solution mostly help with 3rd party modules. but may resolve issue with updated Titanium SDK as well.

    • Hi to all,

      please check following error possibility for solution.

      1) Try to find out not printable or non permitted character or special character in the Tiapp.xml file.

      2) Try to remove multiple line break from your Tiapp.xml file and remove linebreak if possible.
      for more information see this thread, it also contain open issue link for jira also.

      3) Remove all image from your project then check if error is still there? sometime it gives error because of faulty image

      4) Try to find out not printable or non permitted or special character in your file name.

      most of time the error is related to Tiapp.xml but try to figure out exact problem in your project. if you got any success then please share it so other can get guidance from your experience.
