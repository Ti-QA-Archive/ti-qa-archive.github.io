com.saqcal

I was hesitant to buy a module with a few free ones on git hub but this one looked nice so for $20 I went for it. I've seen the problem I'm having on here already without any real good answers so I'll ask a detailed question and hope I get an answer to it.

I put the module in the iphone sub folder of the modules folder then added <modules>com.saqcal</modules> to the xml file. I added all the code I copied from the example file to my project, when I compile it I get "[INFO] Detected third-party module: com.saqcal/1.0" but when I run it I get "[ERROR] Script Error = Couldn't find module: com.saqcal." Any ideas???

    The problem is here:

    &lt;modules&gt;com.saqcal&lt;&#x2F;modules&gt;

    should be :

    <modules>
    <module Platform='iPhone' version='1.0'>com.saqcal</module>
</modules>

    Make shure to use the correct version ;)
