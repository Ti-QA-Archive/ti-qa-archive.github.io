Can't change system button

Hi,

I have declared a button object that benefits from iOS's system buttons:


var myButton = Ti.UI.createButton({
    systemButton: Titanium.UI.iPhone.SystemButton.INFO_DARK
});

I wish to change it, at some point to another icon later on:

I have used the following syntax but unfortunately, its either wrong or this does not work with Ti.


myButton.systemButton = Titanium.UI.iPhone.SystemButton.INFO_LIGHT;

This does not work.

Can someone show me kindly how to perform this change, or even more importantly, if this does not work, and will never work, let me know.

The INFO_DARK icon appears, but I cannot seem to change the icon later on by calling any of its properties or attributes.

Thanks,
Eb

  • Hello E B,

    Try to make two system button….

    See this…

    
var myButton = Ti.UI.createButton({
    systemButton : Titanium.UI.iPhone.SystemButton.INFO_DARK
});
win.rightNavButton  = myButton;

var myButton2 = Ti.UI.createButton({
    systemButton : Titanium.UI.iPhone.SystemButton.INFO_LIGHT
});

myButton.addEventListener('click', function(e) {
    win.rightNavButton = myButton2;
});
