  • I'm sure the question is about the focus order in android. As you know, views can become focusable for input methods other than touchscreen, e.g. with a trackball, dpad, keyboard or remote control. This is important for accessibilty, especially for Google TV Apps. When defining layouts in native Android, you can specify which view should receive focus. See this Android guide for more info.

    Unfortunatly I don't know of a solution to do that in Titanium since there is no API like this. I'm looking forward to future releases of Alloy to support this. Maybe you can log a feature request in the Titanium Community Jira Project.
