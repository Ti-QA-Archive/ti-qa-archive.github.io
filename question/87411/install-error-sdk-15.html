    Hey!

    You need to delete the contents of the iPhone build folder. Delete everything in the folder PROJECTNAME > build > iPhone. Dont delete the iPhone folder itself, just whats in the folder.

    The next time you run/build the project in Titanium, it will take a little longer as it will have to rebuild the complete project again but fingers crossed that should solve your problem!
