    In the view xml you can make a view only showing up on android like this:

    <View id="lab1" platform="android" />

    From the controller, you can set show it like this:

    $.lab1.setVisible( OS_ANDROID ? true : false );

    And from the stylesheet you can show it like this:

    "#lab1[platform=android]": {
   visible: true
}
