Graph and Chart on web view

I am using Rgraph api , for drawing a graph on web view . It works fine when i install the Application simply. But when I tried to publish the app for distribution the graph does not appear on the web view. (I have tried Rgraph and Chart.js both have same result )

    rename your graph libraries to .ljs instead of .js so the build scripts do not try to include them as titnaium source files. be sure to change the references in your html files.

    — answered June 4th 2013 by Stephen Feather
    permalink
    3 Comments

    • on distribution builds, the *.js files are encrypted and packed into a single file that is decrypted at run time. the build scripts cannot distinguish between your required webview libraries which should not be encrypted and your app source files.
