Webview in Android and iOS

Hi together,

i have a problem with webviews. But only on Android-devices. I get always a horizontal scrollbar and my html-list-elements go out of the view, so the user has to scroll horizontal.

I load HTML-Blocks from an RSS-Feed in my webview, because i only want to have the text and not the whole page. On iOS it works fine and the text wraps correctly but not on Android.

I already tried "scalesPageToFit" but it doesen't bring the effect i hoped for. Has anyone an idea or workaround? So i only want to have correctly wrapped text without scrollbars. I could take a normal view, but there i have no styling for my li-elements.

Regards, Dennis

    Hi Dennis

    Try adding this line of html code into your web page and then try again.

    <meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1">

    This tells the device to use the same width as the device actually has pixels. You can also substitute a number in place of the device-width, if you want to tweak the scale starting point.
