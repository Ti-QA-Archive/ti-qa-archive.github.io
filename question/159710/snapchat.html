  • As far as I know, Snapchat doesn't have an open API. They don't seem to be very supportive of developers using their closed API either, so my advice would be to make a "proxy" web server where you post your images, and then forward them to snapchat with for example node-snapchat.

    Make sure you'll add good error handling as the Snapchat API could change at any time.
