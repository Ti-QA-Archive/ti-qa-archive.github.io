Fatbooth type features possible in Titanium?

Hi, I'd like to know if the following is doable in Titanium. These are some features found in the Fatbooh app for iPhone and other similar apps.

  1. Ability to choose an image from front camera or load from library.

  2. An outline of a face will be displayed with dots at certain points, user can position their face within this area and be able rotate the image, zoom and move the individual points so the shape of the area matches the outline if their face.

  3. Save only the image that is within the area, so basically cutting the image up at exact coordinates of the outlined area. Will need to be saved as a transparent image (ie PNG).

I'm pretty sure 1 & 3 can be done and the image manipulation (rotate,zoom etc.) with 2DMatrix, but not sure how to handle the moving of the points as outlined in item 2. Actually, the interface would be more like JibJab than Fatbooth. An example of the interface is here:

http://3.bp.blogspot.com/_ShssuEE1cx0/SSm5fM73K5I/AAAAAAAACgE/vjIvj1WS5iw/s1600-h/Elf+Yourself+setup.jpg

Cheers.
