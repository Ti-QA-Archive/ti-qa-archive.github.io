Restrict the auto focus of text field in case of android tablet

Hi, I am using titanium to develop an application which contains a login screen. In that screen I am using a text field and a button below that. When I execute this sample in samsung s3 it is working fine. But when I execute the same in samsung android tablet the text field is getting focussed when the app gets loaded and the key board is getting pop-up. I tested same in motorola android tablet also. I am facing the same issue. I tried using text field blur() method in window focus event. Now the keyboard is not getting pop-up. But to get the focus of text field I uses text field focus,click,single tap events to get the focus of text field. But any of these events are getting fired. Can anyone share a possible solution how can I restrict the auto focus of text field in android tablet and gain the focus when tapped on text field.
