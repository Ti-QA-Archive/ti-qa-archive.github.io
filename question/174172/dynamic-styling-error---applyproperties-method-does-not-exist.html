Dynamic Styling error - applyProperties method does not exist

I'm trying a simple demo to understand how dynamic styling works.

var newHeaderStyle = $.createStyle({
    apiName: 'View',
    font : {        
        fontSize : 28
    }
});
//set dynamic header for this page
$.loginRecoverHeader.headerTitle.text = 'Username \u0026 Password';
// $.loginRecoverHeader.headerTitle.font.fontSize = 12;
if ($.loginRecoverHeader) {
    $.loginRecoverHeader.applyProperties(newHeaderStyle);
}

However, I get an error stating that applyProperties() doesn't exist for that object.

[ERROR] :  TiExceptionHandler: (main) [142,142] ----- Titanium Javascript Runtime Error -----
[ERROR] :  TiExceptionHandler: (main) [0,142] - In alloy/controllers/loginRecover.js:39,50
[ERROR] :  TiExceptionHandler: (main) [0,142] - Message: Uncaught TypeError: Object #<Controller> has no method 'applyProperties'
[ERROR] :  TiExceptionHandler: (main) [0,142] - Source:     $.loginRecoverHeader && $.loginRecoverHeader.applyProperties(newHeaderStyl
[ERROR] :  V8Exception: Exception occurred at alloy/controllers/loginRecover.js:39: Uncaught TypeError: Object #<Controller> has no method 'applyProperties'

I thought a required object inherits these properties from the parent? Or maybe I'm just accessing the method incorrectly. Any ideas?

    Assuming you want to change the properties of the headerTitle.

    $.loginRecoverHeader.headerTitle.applyProperties(newHeaderStyle);
