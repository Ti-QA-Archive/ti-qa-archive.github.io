how to use e.value.toLocaleDateString get date and getMonth.

Hi,
I'm use date picker.
I use datePicker.addEventListener('change', function(e) {}); to get date.
I use 


defaultValue = e.value;

Ti.API.info("defaultValue.getDate():"+defaultValue.getDate());

The date is less one day.

Today is Mon Mar 11 2013.

I select April 2, 1974.

But I use 


datePicker.addEventListener('change', function(e) {

 Ti.API.info("e.value:"+e.value);

});

It's show

Mon Apr 01 1974 23:00:00 GMT+0800 (CST)

I use 


datePicker.addEventListener('change', function(e) {

Ti.API.info("e.value.toLocaleString():"+e.value.toLocaleString());

});

It's show

April 2, 1974, 12:00:00 AM GMT+09:00

The value is correct.

But I don't know how to use e.value.toLocalString() get date and month.

I use e.value.getDate() get value, the date is 1 not 2.

how to get date and month value use e.value.toLocaleString()?

thank you very much

    hi,

    datePicker.addEventListener('change', function(e) {

var dt = new Date(e.value);

Ti.API.info(dt.getMonth() + ' , ' + dt.getDate());

});

    • I'm resolved the problem.
      The reason is createPicker 's initial caused.
      I initial use var intVal = new Date(1960,00,01);
      var picker = Ti.UI.createPicker({
      …
      value: intVal
      …
      });
      It is can't use this method.
      It should use var intVal = new Date();
      intVal.setFullYear(1960);
      intVal.setMonth(0);
      intVal.setDate(1);
