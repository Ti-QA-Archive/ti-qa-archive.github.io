  • Hi Rodrigo Ferreira,

    You got alert message or not?.

    When you create android:back button eventListener you are explicitly overriding the android back button method.You need to close window after you complete your logic.Try below code

    
win.addEventListener('android:back', function(){ 
    alert('boa bla bla'); 
    //Your custom logic here
    win.close();
});

    Best Regards
