Android gradient background color banding

Hi All,

I am trying to use a gradient splash screen for Android (TI SDK 1.8.0.1), but am running into an issue where color bands appear instead of a smooth gradient. I've done some searching around and found this: https://jira.appcelerator.org/browse/TIMOB-4104

Can you set a pixelformat for a splash screen?

  • Hi,

    windowPixelFormat can only be set on window, 

    var w = Ti.UI.createWindow({
    navBarHidden: false,
    exitOnClose: true,
    backgroundImage: 'gradient.png',
    windowPixelFormat: Ti.UI.Android.PIXEL_FORMAT_RGB_565
});

    splash screen is not a part of it. Hence this property wont work for splash screens.
