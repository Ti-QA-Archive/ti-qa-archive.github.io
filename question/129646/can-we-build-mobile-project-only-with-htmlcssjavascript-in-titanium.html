  • You can't build Titanium app using HTML and CSS. Titanium provides JS environment which is different from browser DOM environment and therefore HTML and CSS are not applicable in it (they can only be used in WebView, which provides DOM environment).

    You must use JS and JS API that Titanium provides. For examples check Wiki and official documentation: http://developer.appcelerator.com/apidoc/mobile/latest
