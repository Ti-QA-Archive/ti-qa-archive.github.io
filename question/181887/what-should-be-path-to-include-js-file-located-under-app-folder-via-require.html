What should be path to include js file located under /app folder via require

Hello.

Since Ti.include is deprecated for almost a year I tried to use require in order to "include" file located at &#x2F;app&#x2F;lib&#x2F;setup&#x2F;initialContentSetup.js from &#x2F;controllers&#x2F;index.js. Documentations states that paths starting with '/' or relative paths are pointing to the Resources folder, which puzzles me since files there are erased with each build.

I have tried passing following paths to require:

../app/lib/setup/initialContentSetup
./app/lib/setup/initialContentSetup
/app/lib/setup/initialContentSetup
app/lib/setup/initialContentSetup
../lib/setup/initialContentSetup
./lib/setup/initialContentSetup
/lib/setup/initialContentSetup
lib/setup/initialContentSetup

None of them worked, getting "Requested module not found" error.

What should be correct path for that file?

    Below is the correct path for alloy project,

    
var obj =  require("setup/initialContentSetup");

Ti.API.info("obj ==> " + JSON.stringify(obj));

    Try this one and let me know if it is not working for you.
