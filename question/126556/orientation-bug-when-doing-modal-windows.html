Orientation bug when doing modal windows

Currently I have this responsive app (works on any orientation) but for some reason I want the modal window to be in portrait only. Setting this on the modal window

var window = Titanium.UI.createWindow({
    navBarHidden: true,
    tabBarHidden: true,
    backgroundColor: "#808B9D",
    layout: "vertical",
    orientationModes: [ Titanium.UI.PORTRAIT, Titanium.UI.UPSIDE_PORTRAIT ]
});

Works as expected, but after the window is closed, the whole app is forced into those 2 orientations, it will no longer rotate when moving around the iPhone…

Thoughts?

This is iOS of course and 1.7.2
