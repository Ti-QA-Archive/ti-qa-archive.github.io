  • This is a problem with your Python installation. Try uninstalling Titanium, and all versions of Python you have installed on your system. Remove Python from your PATH and remove the PYTHONHOME environment variable. Then reinstall Titanium. Everything should work then.

    And, yes, it is annoying the Titanium does not play nice with other versions of Python installed on the system.

  • No need to uninstall anything. Just remove the existing python entry in your PATH system variable and restart Titanium.

    You can understand more about the problem this way:

    • Search for python.exe in your Titanium installation folder and see that it's Python 2.
    • Type "python" in Titanium Studio's terminal pane and observe that it runs Python 3 (or some other Python version that expects print("a") rather than print "a") that you already have installed.
    • Understand that when Titanium is trying to run Python, it's not running the Python that comes with it because of your PATH variable. Verify this by typing "echo $PATH" in the Titanium Studio's terminal pane.
