ImageView image not being shown - asynchronous - canScale

In ANDROID

I have an imageView, with the property canScale set to TRUE.

This imageView image is being set asynchronously.

My problem is: if I insert a local image, everything works fine: when I open the imageView window the image is displayed, and I can scale it.

But, when I load this image just after download it, the imageView is just black, and it image is not shown. When I press (click) the imageView, the image appears and I can scale it.

So, the problem is:

Why the image is not showed until I click the imageView?

////

image view :

var imageView = Titanium.UI.createImageView({
    top : 0,
    left : 0,
    right : 0,
    top : 0,
    bottom : 0,
    backgroundColor : '#000000',
    canScale:true
});

an then I set image

file.write(xhr.responseData);
