  • If Metaio supports it you have to create a metaio titanium module (not sure what their sdk offer) and then integrate it. Vuforia has a Android SDK but as far as I know nobody created a Titanium module yet to combine those too.
    But since you are using the create (quote: "In short, the Creator is designed with a simple user interface for those without programming skills") it won't work. You need the SDK not the creator and even then I'm not sure if they support creating custom modules

    You could always just start your AR app from a TI app with custom urlschemes

  • Michael is correct. In order to integrate Metaio into Appcelerator, you're going to have to use the SDK of each specific OS. You'll have to natively code Metaio for both iOS and then for Android and build a module in JavaScript for Appcelerator to call out the native functionality in each. Quite a task to say the least. I'm sorry this isn't the answer you were hoping to get. But, good luck with your endeavors.
