opening a modal view on first opening on app

i'd like to open a modal view on first opening of the app but not on subsequent visits (a brief help type intro) - what would be the methodology of doing this ?

    Probably the best approach would be to save some sort of "flag" using the Properties API.

    For example, on each run, you try to read this flag on every startup.

    Titanium.App.Properties.getBool('isNotFirstRun', false);

    The second parameter is the default value if the property has never been saved.

    If your returned value is false, then you display your modal window. Once you close this modal window, you simply update the flag so the window is not shown again.

    Titanium.App.Properties.setBool('isNotFirstRun', true);
