changing setlocation to refresh map

I have a map showed in a window object. in another tab (tab2) of my tabgroup i input new latitude and longitude and then i refresh the mapview object in previous tab (tab1).
In tab2 i use a button to refresh the map:

var myAddress = NewLocation 

Ti.Geolocation.forwardGeocoder(myAddress,function(evt) 

{

   myLat = evt.latitude;

   myLon = evt.longitude;  

   regionSV = {latitude:myLat,longitude:myLon,animate:true,latitudeDelta:0.01, longitudeDelta:0.01};

   mapview.removeAnnotation(viareggio);

   mapview.setLocation(regionSV);

   alert(myLat);

 });

The alert statement display the correct value but when i go in tab1 are missing the original map and the updated map by setlocation statement.

Any ideas? Thank you.
