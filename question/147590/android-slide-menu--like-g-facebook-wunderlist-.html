Android Slide Menu ( like G+, Facebook, Wunderlist )

I am trying to create a sliding menu like the one on Wunderlist or Facebook, but having troubles with the animation, its either buggy ( flickering ) or really slow on performance ( riding into more flickering )

My Code looks like this:

var alpha, beta, delta,
     point = {},
     t = Ti.UI.create2DMatrix();

view.addEventListener('touchstart', function (e) {

  alpha = parseInt( e.x, 10 );

});

view.addEventListener('touchmove', function (e) {

  point = view.convertPointToView({
    x: e.x
  }, win1 );

  delta = parseInt( (point.x - alpha), 10 );

  view.animate({
    left: delta,
    duration: 0
  });

});

now it works, but as I said its really slow on android, but works like it should on iOS ( even better when I use transform: t.translate(delta, 0) instead of left:

but then I have a really hard flickering ( it looks like the e.x attribute is reset to 0 after every translateToX );

my stack looks like this:

<window>
  <view id="view" zIndex="10" />
  <view id="menu" zIndex="1" />
</window>

the view is above the menu.

Who knows why the performance is so bad?
