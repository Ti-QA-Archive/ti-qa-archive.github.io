Are these instructions in the docs correct?

I was looking to set my mac up to do iPHONE modules… but I ran into a question… I am not sure this instruction is correct.
is the path correct?

http://wiki.appcelerator.org/display/guides/iOS+Module+Development+Guide#iOSModuleDevelopmentGuide-iOSModulePrerequisites

Open the Console application. The Console application is located under /Applications/Utilities/Console.
Edit the file named .bash_profile in your $HOME directory.
In this file, add the following line at the end of the file:
alias titanium='/Library/Application\ Support/Titanium/mobilesdk/osx/1.4.0/titanium.py'

  • Naturally that path was written when the SDK version was 1.4.0

    So, you need to decide which version of the Titanium SDK you need your module to support, make sure it exists in the /Library/Application\ Support/Titanium/mobilesdk/osx/ folder and set it to that version.

    So, it all depends which versions of the SDK you have installed.

    Mkae sure you look in your 'root' Library folder and not the /Users/YOUR_USERNAME/Library/… folder - A common mistake.
