Tab Groups with self calling functions??

Hi Guys.
I've come to update an old app that about six months old.

I need to add new functionality that will require a tab group, however all my current slides are within self-calling functions.

(function(){
    app.ui.mainwindow = function(){
        var win = Ti.UI.createWindow({
            title:'title.png',
            backgroundImage:'Default.png',
            fullscreen: true
        })
...

Then they are opened via: app.mainwindow.open();

How would I use this with tab groups?

Cheers
Jason.

  • Could do a similar setup, just with tab group. You then pass the mainwindow to the tab's window property. Something like:

    (function(){
    app.ui.tabGroup = function(){
        var tg = Ti.UI.createTabGroup();

    var tab1 = Ti.UI.CreateTab({
        title:'tab1'
        window:app.ui.mainwindow()  //pass the window here
    });
    tg.addTab(tab1);

    return tg;
     };
})():

    Then call app.ui.tabGroup.open();
