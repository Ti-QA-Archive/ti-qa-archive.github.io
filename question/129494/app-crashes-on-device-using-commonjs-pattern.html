    I guess you're trying to run that on iPhone/pad

    This has been discussed in other posts.

    You must check that your exports statements are followed with comas.

    i.e:

    exports.someFunction = function(){
    // some code
};

    You should also reformat your code in Titanium Studio with the shift-cmd-F shortcut (or, in the menu bar, Source > Format )

    Worked for me.
