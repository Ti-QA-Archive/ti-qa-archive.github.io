Android justify HTML or text in UI.Label

Titanium SDK 1.7.5, Android 2.2, OSX.

var myLabel = Ti.UI.createLabel({
 top:0,left:0,width:100,height:200,
});
var myHTML = '<p>Some Text Inside P (shortened)</p>Some Text';
myLabel.html = '<html><head></head><body>' + myHTML + '</body></html>';

I need to be able to make my long text body for a label appear with JUSTIFIED alignment, e.g. in CSS: text-align:justify, in old HTML &lt;p align=&#39;justify&#39;&gt;

I've tried a few things, none have worked:

&lt;p style=&#39;text-align:justify;&#39;&gt;Some text&lt;&#x2F;p&gt;
&lt;p align=&#39;justify&#39;&gt;Some text&lt;&#x2F;p&gt;

And of course i've tried this in hope there was an undocumented option:
myLabel.textAlign=&#39;justify&#39;;

It works great if I use a UI.WebView …Problem is, UI.WebView on Android doesn't work too well when you are using the inline webView.html property, it really only works with web pages that have a URL like www.google.com. Basically when you change orientation the whole thing breaks.

However, if someone can verify that the webView works in Android even when you flip orientation to landscape and back, I'd be willing to listen to that. Just doesn't work for me.

Any ideas?
