  • Problem solved!
    I just now found out about the existence of a Titanium Studio log file thanks to your reply. Looking in it, I noticed a couple of errors about files not existing. The reason: I have Titanium use a workspace directory "~/Documents/DEV/", but the old project resided in "~/Documents/DEV/AM/AMbrowser/tiapp.xml", one directory deeper. I moved the AMbrowser directory to the "~/Documents/DEV/" and then did the import again and it worked.
    I would consider that a bug from the importer, as you get no warnings that the project directory should already be a direct child of the workspace directory.
