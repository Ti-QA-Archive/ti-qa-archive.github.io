  • The backButtonTitle is only relevant if the window is in a TabGroup.
    You could use LeftNavButton to make a back button

    In Alloy it would look something like this: 

    <NavigationWindow id="navWin" platform="ios">
    <Window>
        <LeftNavButton>
            <Button onClick="doClickBack">Back</Button>
        </LeftNavButton>
        <WebView id="theWebView" />
    </Window>
</NavigationWindow>

    Initialise the webview and handle the navigation in the controller:

    // Initialisation wrapped in self executing function 
(function(){
    $.theWebView.url = encoded_url
})()

// Handles clicks on the Back Button
function doClickBack(e){
    // Do Back Navigation
}

    (I did not test this, but you will get the idea)
