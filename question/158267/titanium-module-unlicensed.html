Titanium module unlicensed

I am having a problem with a module bought from the marketplace. I am working on a mobile app with another person and for the past few weeks we have not had an issue. Until today, now when I run the app on my phone I get the alert message saying that the module is unlicensed, but for the other person working on it, there is no problem. Can someone enlighten me. I have tried some of the suggestions from other posts, but can't get it fixed

    The module licensing is all kind of crazy.

    You have an account, the module has a GUID, the project has a GUID. The guids all have to match up on the Appcelerator side for a module to be able to run.

    The very first thing I do when i have an issue like this is increase the GUID of the project by a single digit someplace, clean and rebuild. Sometimes, the project while associated with your account, doesnt get associated with the purchased (could be free) module. Maybe the GUID for the project was generated before the module was purchased, etc.

    If that does NOT work, email the marketplace support and have a tech look into it.
