Default behavior of return key on keyboard ends text input. I want to change that behavior to entering a paragraph break (new line) into text. Have not found such setting and have not found how to find out position of cursor in text, if I would like to write my own handler of return key.
Application written for iPad, SDK 1.8.1

  • may be this can help..

    var txt = Ti.UI.createTextArea({
    width : 320,
    height : 200,
    top : 50,
    backgroundColor:'white'
});

txt.addEventListener('return',function(e){
    txt.value = txt.value + '\n';
    txt.focus();
});

win.add(txt);
