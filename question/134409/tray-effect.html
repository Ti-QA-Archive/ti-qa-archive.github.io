Tray Effect

I have this graphic which is the top of the tray.
Then I have this graphic which is the top graphic with some items in a tray, which moves drop.
How can I simulate opening this tray down the screen?

So I have 1 thin graphic, and one larger one, which includes the smaller one.

    • Ok, something like;

      var object = Ti.UI.createView();

var animation = Ti.UI.createAnimation({transform:Ti.UI.create2DMatrix().translate(x, y);});


object.animate();

      Where object is your element (view, graphic etc) you want to move and the values of x and y represent the x/y axis on which to move
