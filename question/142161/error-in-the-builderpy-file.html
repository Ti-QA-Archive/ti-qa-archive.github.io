  • clean your project and run again,if it never gets fixed then
    open tiapp.xml file from you project ,in tiapp.xml file ,look at the tag in most bottom

    <modules>
    <module platform="commonjs" version="2.3.0">ti.cloud</module>
    </modules>

    version shoud be the one located in

    /Library/Application Support/Titanium/modules/commonjs/ti.cloud/'version'

    In my case version was 2.3.0 so tiapp.xml version should be given 2.3.0
