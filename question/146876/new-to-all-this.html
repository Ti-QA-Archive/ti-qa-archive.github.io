  • Hi Brian,

    Titanium was created to allow developers to create native mobile apps for iOS and Android without having to learn Java or Objective-C. Javascript is all you need!

    A good resource to start learning Javascript from scratch is Eloquent JavaScript:
    A Modern Introduction to Programming     by Marijn Haverbeke. It is available free in digital format - if you prefer a paperback version (or would like to support the author) you can purchase a copy from Amazon via the link on his site.

    If you have no prior knowledge of programming, spend a little time learning the basics of Javascript. Afterwards you can jump into learning about Titanium via Appcelerator's online documentation, training guide and video tutorials. Be sure to follow the recommended best practices it will save you a lot of time later on if you learn to structure your code properly from the start.

    Good Luck!
