  • I had this same problem with some views within a view. So I created a class in the .tss file:

    ".hide": {
    height:0,
    visible:"false"
}

    Then I add the class to hide the view:

    $.addClass($.inputFieldsView, 'hide');

    Then remove the class to show it again:

    $.removeClass($.inputFieldsView, 'hide');

  • Hi Cauli,

    In Titanium, Any View have a one property named "layout", which is used for setting the view's element or childrens in horizontal and vertical manner.

    This property directly arrange the children in vertical or horizontal manner.

    Please refer below code for reference . may be it help ypu: 

    var win = Ti.UI.createWindow({
    backgroundColor : '#a8a8a8'
});
var mainView = Ti.UI.createView({
    top:"10dp",
    width : "100%",
    height : Ti.UI.FILL,
    layout:"vertical"  //<----- This property can set the view's children in a vertical manner
});
win.add(mainView);
var button1 = Ti.UI.createView({
    title:"button1",
    width : "200dp",
    height : "45dp"
});
win.add(button1);
var button2 = Ti.UI.createView({
    title:"button2",
    width : "200dp",
    height : "45dp"
});
win.add(button2);
win.open()
