Cannot create Text Field

All I'm trying to do is create a text field and it keeps crashing my app in the emulator. I'm running ios 4.3 and sdk 1.7.2, so all is current. I'm running the iPhone4 emulator. This should be simple but is not working for some reason, and I'm pulling my hair out. Here is my code:

var win = Ti.UI.currentWindow;

var txtFld1 = Ti.UI.createTextField({
color:'#336699',
height:35,
top:40,
width:250,
keyboardType:Ti.UI.KEYBOARD_DEFAULT,
returnKeyType:Ti.UI.RETURNKEY_SEARCH,
borderStyle:Ti.UI.INPUT_BORDERSTYLE_ROUNDED
});
win.add(txtFld1);

What could possibly be the problem? I have tried rebuilding the project a couple of times. Once in awhile, TI Studio console will mention an error about createTextField not being defined in the UIModule, but the text fields in Kitchen Sink work fine. Please help!

    These 'invalid method ' errors on any Titanium method usually point out to something going wrong in the directory build. Clearing the contents of the build/iPhone directory should deal with it. Or, in Titanium Studio, use Project/Clean…
