    • Have you stepped through this in the debugger at all? If you do that, you can examine the variables as the code is running. You would find that when you try to create the window, the variable "e" does not have a property called "rowData". It does have a property called "row". So you can do this:

              var win = Titanium.UI.createWindow({
            url:e.row.test,
            title:e.row.title
        });

      Now like I tell everybody, you're not structuring your application well when you use the "url" property of a window. You absolutely must learn CommonJS module-style programming if you want to build a complex application.
