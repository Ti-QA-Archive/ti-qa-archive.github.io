  • I am not sure Ti.App.Properties informations will be kept if the user reboot the app. If you want to keep the chosen language even when app is closed, you can use Filesystem. At the beginning, check if the file exists or not. If not, ask to your user to choose a language, then write it in the file. If the file is already existing, just read it to know the language. But it will use some kB of Data Storage…
