Variable By Reference

Hello,

I'm trying to figure out how to access a variable by reference in Titanium. I've created an array of views, and I want to swap between them at will. I add say view_array[1] to my window, make changes, and then swap to view_array[2]. When I try re-adding view_array[1], the changes I made from before do not show.

Does anyone know if there is a way to access variables/views by reference in Titanium? I'm trying to handle my own back/forward button navigation so that I don't have to open a bunch of window contexts.

Thanks,

-Kyle

  • Sorry, this was my fault. I was doing:

    var view = require('ui/handheld/android/' + e.page);

    view_array.push(view);
    instead of
    view_array.push(view());

    So it kept re-generating a new view. Once I added (), it worked.
