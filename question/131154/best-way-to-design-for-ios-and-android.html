Something like

if "OS" == iPhone DO THIS{

    Sample from one of my App,

    var isAndroid = false;
if(Titanium.Platform.name == 'android'){
    isAndroid = true;
}

if(isAndroid){
    Android code
}else{
    iphone code
}

  • I think the best advice would be to separate your UI code out from your re-usable cross-platform code. That way you can design for Android and design for IOS separately but re-use the core code. Creating conditional statements within the code for IOS vs ANDROID soon becomes a maintenance nightmare as the parity of support for the UI elements is not quite there yet.
