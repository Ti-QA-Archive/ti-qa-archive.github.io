  • Hello Elizabeth,

    The error saying "invalid username", According to the link, did you tried to put the user name as 

    Try the login command below on the command line:


curl -v -d "un=XXXX&pw=YYYY" https://api.appcelerator.com/p/v1/sso-login


(replacing XXXX and YYYY with your username and password. The username needs to be url-escaped, like user@company.com => user%40company.com)

    So, your user name should be "XXX%40company.com" instead of "XXX@company.com"

    Hope this helps. Thanks.
