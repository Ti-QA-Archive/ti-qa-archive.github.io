Hide Default Activity Indicator in webview

Unable to hide Default Activity Indicator in webview. I have used animate :false but still i cant able to hide loading spinner.

    top:44,
    width:'100%',
    height:'100%',
    animate :false,
    url:"http://www.cam.ac.uk/",
    backgroundColor:'#fff'
});
win.add(extwebview);

  • Open TiUIWebView.m file in your SDK folder (should be in iphone&#x2F;Classes folder) and find only show the loading indicator if it&#39;s a remote URL string (near line 116). There should be if ([self isURLRemote]) code block after it. Comment that block and clean/re-compile your project.
