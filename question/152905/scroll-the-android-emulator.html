scroll the android emulator

Hi,
I created a window with a textarea (as its first control) and then a tableView ( the rows have textarea).
When I load the page, the keypad is automatic opened, and it hides part of table.

  1. Can I hide the keypad in the emulator (not by command but by the mouse).
  2. How can I scroll the view to see the rest of the table?

  • You should be able to add the keyboard if you click on another control that can get the focus (e.g. button). YOu can also set that to be the first control to get focus so the keyboard does not display automatically.

    Also, you can make sure your window is a heavyweight window and add this to your manifest file for that activity: 

    <activity android:configChanges="keyboardHidden" android:label="My Heavy Window" android:name="HeavyWindowActivity" />

    The important section is android:configChanges="keyboardHidden" so you can read up on that here.
