Access child object by name

I want to select a child by a custom property I have set:

var someItem  = Ti.UI.createView({
                   height: 90,
                width: 90,
                backgroundColor: '#000', 
                name: 'some_name'
            });

I am able to do this by looping through the array of children in a 'container' and using an if statement like so:

for(var i = 0; i < container.getChildren().length; i++){

    if(container.children[i].name == 'some_name'){
        //now object is accessible 
    }
}

But I want to access the child like this:

container.children[{"name":"some_name"}];

That latter doesn't work though. Is there any way in javascript to access a child by it's name without having to loop through all the children in the parent object?

How do I select a child object by name? Suggestions….
