Titanium Studio - Ubuntu 14.04 - No menus

I have downloaded the latest Titanium Studio and am running 64 bit Ubuntu 14.04.
I have Oracle Java 8 running:

java version "1.8.0_11"
Java(TM) SE Runtime Environment (build 1.8.0_11-b12)
Java HotSpot(TM) 64-Bit Server VM (build 25.11-b03, mixed mode)

The studio opens fine and loads. The issue is that none of the menus drop down (File, Edit, Source, etc). As a result I cannot use the studio.

Has anyone had a similar problem? Any insight into how to resolve/troubleshoot?

Thanks!

  • The solution was found by adding to the Desktop file, as below
    http://stackoverflow.com/questions/19452390/eclipse-menus-dont-show-up-after-upgrading-to-ubuntu-13-10

    The key line:

        Exec=env UBUNTU_MENUPROXY=0 /home/USERNAME/Dokument/eclipse/eclipse

    Added text: env UBUNTU_MENUPROXY=0

    The menu now works as expected.
