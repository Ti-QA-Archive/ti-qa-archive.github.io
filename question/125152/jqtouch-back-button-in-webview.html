jQTouch back button in WebView

Hi,
I'm using jQTouch in my app which opens html page within a webview(I'm working with single window only).But when I click on simulator's back button , instead of going back to previous page, the application closes.
Any suggestions are greatly appreciated.Thanks.

  • Suggestions:

    1. Get rid of jQTouch. Appcelerator gives you native support so you don't need to use a webview.

    2. You will have to catch the android:back event and send a custom event to the webview to do what you need (in this case go back)

    here is the android:back event - http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.UI.Window-object

    here is how to make custom events - http://developer.appcelerator.com/apidoc/mobile/1.7.1/Titanium.App-module

    But I strongly suggest you to recode the app using the native UI elements.
