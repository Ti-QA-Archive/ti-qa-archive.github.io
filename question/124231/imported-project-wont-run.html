    Fixed it on Lion! This is the wierdest fix you will ever find but it honestly worked for me.

    I updated to OSX Lion, then updated all my applications through Software Update then reinstalled new version of Xcode.

    This error happens in Titanium (see above) but when I try to open Xcode, it just crashes on launch. Every time.

    Then I found this thread https://discussions.apple.com/thread/2787637?start=0&tstart=0 which details the same problem with Xcode.

    Short fix is:

    • Open ActivityMonitor
    • Close Ituneshelper
    • Then download iTunes (honestly stay with me) from here
    • Install it, and then Xcode works.
    • In turn, running projects in iPhone Simulator from Titanium also works.

    Hope it works for you!
