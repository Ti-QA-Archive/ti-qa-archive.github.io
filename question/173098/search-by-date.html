  • you need to set one custom variable in your row object then you need to set that variable name as a filterAttribute in tableview object.read this doc

    var row = Ti.UI.createTableViewRow({
    myDate : [date string]
});

var tableView = Ti.UI.createTableView({
    search : [searchview object],
    filterAttribute : "myDate"
});
