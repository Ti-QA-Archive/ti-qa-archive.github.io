  • Got it to work on my ubuntu laptop and found that the when it from time to time wouldn't start I just had to run the following commands in the terminal, tried the same commands in windows and it worked perfectly: 

    1. appc setup
2. Fill in the credentials
3. Answer yes on "Do you plan on developing Titanium apps?"

    Then start appcelerator studio!

    If it doesn't start, go back to the terminal and create a new project from the terminal. Don't know why this works but it seems to work every time studio freezes or give error on startup.
    To create a new project from terminal just enter: appc new. And then just follow the guide.
