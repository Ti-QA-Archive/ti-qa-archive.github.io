    • yes, but the ipad 3 still decodes the image for retina.

      when the image loads into memory is decoded from the format you are having (jpg,png etc) in a raw image, with 4 bytes per pixel (rgb+alpha).

      it doesn't matter that u have a jpg or png or whatever, the result will be te same.
      so a complete black jpg image with 100x100px will have the file size of few kilos or less while in the memory will have 100x100x4 = 40k.

      For the ipad 3 where a pixel has 2 dips ( so x4) the image will have 160k.

      Makes more sense?
