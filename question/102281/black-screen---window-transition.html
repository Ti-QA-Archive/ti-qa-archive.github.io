Black Screen - Window transition

Strange problem, I have a main window named appWin, when the program launches I open this with appWin.open().

Within the interface I have a button for settings which I open with the transition flip from LEFT. This works fine, settings window is opened as it should.

When I close the window with..

var close_anim = Ti.UI.iPhone.AnimationStyle.FLIP_FROM_RIGHT;
settingsWin.close({
transition: t
});

It flips, but to a black screen and then nothing?? I've debuged to the console if appWin exists which reports in the output window [DEBUG] [object TiUIWindow].

Anyone else had this? Only started on update to SDKiOS4.2/3.2.5 and 1.5.1??

Help!

  • I have found that adding a show() to the first window solves the problem. So, in your case, 

    var close_anim = Ti.UI.iPhone.AnimationStyle.FLIP_FROM_RIGHT;
appWin.show()
settingsWin.close({
transition: t
});

    Note that you don't need to hide the window before. This seems like a hack, and it took me hours of guessing and checking to figure it out, but it should work.
