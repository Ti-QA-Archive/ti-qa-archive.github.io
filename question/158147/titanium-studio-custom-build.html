Titanium Studio Custom Build

I have an app project on Titanium Studio which has to lead to the deployment of 2 different apps (the 2 apps are identical, except for the icon, the launch image and some configurations saved on a js file). Is it possibile to configure custom builds on Titanium Studio (something like Ant's build.xml file where to define which files have to be included when building the app for deployment or distribution)? And if not, did anyone handle this issue in a different way?
Thanks in advance.

  • for icon and launch image you can make a small script depends on your os (e.g. bash in Linux or OSX ) that will package the proper files ( icon , launch image ) to the app directory and then build the app with preferred status ( deploymeny or distrubition )
