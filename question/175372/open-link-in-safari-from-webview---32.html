  • If your HTML comes from locale file system (Resources-folder) you are in the same name space and you can communicate by event/observer pattern. If the HTML comes from remote, you can only work with by calling evalJS() after load event of webview. In any cases you can hook the click event on 'a', return false, grep link and call by openUrl() the new page in safari. In this documenation you will find detailed informations.
