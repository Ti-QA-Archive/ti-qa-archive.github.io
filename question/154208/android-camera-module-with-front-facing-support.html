  • Looks like this isn't supported yet from our side. Here is the JIRA that explains why we haven't implemented it yet. Essentially we would have to rewrite the android camera API without using intents. Unless it's been created as a module by a third party (of which I can't seem to find either), it is not possible at the moment. You could always create the module yourself if you needed it bad enough :)
