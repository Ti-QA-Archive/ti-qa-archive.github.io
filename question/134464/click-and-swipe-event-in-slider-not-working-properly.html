  • To use the swipe feature of slider you can do like this..(remember this only works on iPhone and iPad.. as swipe is not for android)

    swipe.addEventListener('swipe', function(e) {
   if (e.direction == 'left') {
      slider.value=slider.value-10;
   } else if (e.direction == 'right') {
      slider.value = slider.value +10;
   }
});

    hope that help :)
