Universal Modal Activity Indicator for iOS

Hi,

I want to make a activity indicator that i can use multiple places in my app. Kinda like the one in android.

I read in another Q&A that the way to do this is to make a view with opacity:0.5 and place a activity indicator on this.

Anyway when i try this i simply always get an error:
"Result of expression Ti.UI.getCurrentWindow() [undefined] is not an object" ??

Using:

Titanium.UI.getCurrentWindow().add(loaderView);

How come getCurrentWindow is undefined?

Is there another way to make a universal activity indicator?

Thanks

  • Accepted Answer

    If you are using tabs, you can add the indicator to the tab group.

    tabGroup.add(MyActivityIndicator);

    Then as needed:

    MyActivityIndicator.show();

    or

    MyActivityIndicator.hide();
