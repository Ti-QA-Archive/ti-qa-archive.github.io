ACS API Calls

What counts as an ACS API call? I'm testing an app with ACS when I check my usage on my app on https://cloud.appcelerator.com/apps/ there's no monthly usage or if i check my app on https://my.appcelerator.com/app/ there's no API Calls on cloud usage.

Im just querying users, and creating and querying custom objects. Am i missing something, should these values be increasing?

The Appcelerator website states the following:

"API calls are a measure of usage that include any of the following: Pre-built cloud services (e.g. Push notification), Analytics, cloud-based Connectors (e.g. Oracle, Salesforce.com) and Node.ACS. Any and all calls within the mobile app that use these APIs count against the daily and monthly limits"

many thanks in advance
