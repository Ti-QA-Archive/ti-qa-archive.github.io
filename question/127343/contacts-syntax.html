Contacts syntax

Hi,

I try to save contacts to iPhone. The following code snippet kills the app and show error: 

2011-10-25 12:00:52.771 PrivateIslands[99092:207] :
-[__NSCFString countByEnumeratingWithState:objects:count:]:
 unrecognized selector sent to instance 0xa0f2200

[INFO] Application has exited from Simulator

Here the evil code:

var contact = Titanium.Contacts.createPerson({
    URL : {
        "homepage" : 'http://vladi.de'
    },
    email : {
        "work" : 'info@vladi.de'
    },
    lastName : 'Vladi Private Islands',
    kind : Ti.Contacts.CONTACTS_KIND_ORGANIZATION,
    address : {
        "work" : {
            "Street" : 'Ballindamm 7',
            "City" : 'Hamburg',
            "State" : 'Hamburg',
            "ZIP" : '2000'
        }
    }
});

It sounds like a wrong format, but I have used the object description from
http://developer.appcelerator.com/apidoc/mobile/1.7.3/Titanium.Contacts.Person-object.html

Wjast going wrong?

Rainer
