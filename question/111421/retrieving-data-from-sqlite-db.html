    • Renzo

      Regarding your code in your pastebin, I'd make the following suggestions:

      • you have an infinite while loop - you need to use the next() method on your resultset in order to iterate through each record, and cause it to finish
      • the logic of your code does not make sense - you are selecting everything in the table, iterating through every record returned, and then using the last record. Why not select the specific record you want, by using a where condition?
      • you have forgotten to close your database
      • there is no need to wait until the end of your script to close your resultset (or your database). Close it as soon as you have finished with it

      Hence, replace the section of your code with this:

      var db = Ti.Database.install('/mydata/', 'restaurants.db');

var getmyinfo = db.execute('SELECT * FROM Places');
while (getmyinfo.isValidRow())
{
    var lat1 = getmyinfo.fieldByName('Lat');
    var long1 = getmyinfo.fieldByName('Long');
    getmyinfo.next();
}
getmyinfo.close();
db.close();

      Hope this helps
