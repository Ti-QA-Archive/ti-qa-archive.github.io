    • Thx, I managed to make it work with imageAsCropped :)

      But I think I found another (easier) method : I simply use a view with borderRadius:0, and I add an ImageView with 

      width:'100%',
height:((Titanium.Platform.displayCaps.platformWidth * 0.94) / imageWidth) * imageHeight

      Tada, everything is OK. Except if I rotate the screen… so I chose to disable it :) Anyway, thx for your advice

    left:20,
