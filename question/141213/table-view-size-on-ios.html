    Hi Michael

    I have tried your code and the table appears on the screen only taking up the top half of the screen, but as you have only supplied code for two of your six rows - this may have something to do with it not showing up the way I think you are trying to explain it.

    The suggestion of setting height "50%" seems to be the answer but as you have replied "..it shows nothing" I suspect that you need to provide a bit more information.

    Do you mean setting a row to 50% or the table to 50%, why do you need it to go to 50% in the first place - in case the problem lies more in the reason that your 50% solution. Are you trying to leave space for the keyboard?

    Please provide a full set of your code that shows the table so we can really help you as well as what you are trying to achieve, not just I want to make this 50%.

    Please also let us know which SDK version you are using and if you are targeting Android or iOS.

    A couple of things about the code you have supplied.

    • You can simplify your rows by using push instead of explicitly stating an index, this way you do not have to worry about re-indexing them if you change their order, just swap when you add them to the array.
    var data2 = [];
data2.push(row);
data2.push(row2);
    • You can get rid of the section row - if all you are trying to do is add a title in the standard form (note the rows above do not have the section row you added);
    row.setHeader('Post Info');
