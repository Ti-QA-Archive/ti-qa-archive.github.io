  • Hello, I ran into this same problem. The easiest solution for me was to copy the adb from the platform tools to the tools directory. I'm rather new to my mac so I didn't know about the whole link thing that's mentioned elsewhere.

    Just go to your android sdk folder, copy the adb under platform tools directory to the tools directory.

    From what I understand google changed the location of the adb file in their latest release of the SDK and titanium is looking for it in the old place.
