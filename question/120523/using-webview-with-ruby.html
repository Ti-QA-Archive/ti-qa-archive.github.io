Using WebView with ruby

Hello,

I have an web app for iPads wich uses ruby (rubygems and sinatra), html and the sencha js Framework. My app runs fine with 

ruby application.rb

Now, I want to convert this app to an native app with titanium. Therefore I use an app.js file like this:

var win = Titanium.UI.createWindow({
  backgroundColor:'#fff'
});
var web_view = Titanium.UI.createWebView({
  url: 'index.html'
});
win.add(web_view);
win.open();

This works fine if I don't use ruby. But with ruby, I can't call an html file, i have to call the application.rb file to start the app. How can I set up my files so that I can run my app in WebView with ruby?
