  • Hi,

    Have u try this:

    var t = Ti.UI.create2DMatrix();

    t = t.rotate(360);// you can change the rotation angle values from here.
    var a = Titanium.UI.createAnimation();
    a.transform = t;
    webview.animate(a); // animate the web view object
