Titanium 3.1 with Admob 2.1.0, no error, but nothing shows

I'm trying to integrate Admob into my Android app. No error but nothing show up.

My environment:

Titanium 3.1.0 and 3.1.1

Admob 2.1.0

Android SDK: 2.3.3, 4.0.3

Dev machine: win7

I followed instructions from many posts, and here are what I have done:

Downloaded Admob .zip file, put it to root of project. I also tried some other dir like: C:\Users\user\AppData\Roaming\Titanium\XXXXXX.

Added 

<tool-api-level>14</tool-api-level>

to TiApp.xml.

Added 

<module platform="android" version="2.1.0">ti.admob</module>

to <modules> node.

In my app.js, just very simple code as instructed on tutorial, snippet is like:

var admob = require('ti.admob');

var adView = admob.createView({
    publisherId: '12asfd56rtyw45',
    adBackgroundColor: '#666666',
    keywords: 'sports',
    bottom: 0,
    width: 320,
    height: 50,
    borderColor:'#000',
}),

window.add(adView);

It compiled smoothly, no error, but nothing showed on screen. No banner, no view.

Does anyone have any clue? Thanks.

  • Finally, problem solved!
    The reason is I didn't specify the correct height of adView. In my original code I set height of adView as 50:

    var adView = admob.createView({
    publisherId: '12asfd56rtyw45',
    adBackgroundColor: '#666666',
    keywords: 'sports',
    bottom: 0,
    width: 320,
    height: 50,
    borderColor:'#000',
}),

    The problem is: admob requires height of adview to be 75 at least! So I changed it to 75, and it worked!

    I have published my app to:

    https://play.google.com/store/apps/details?id=lingandcs.dishesmap

    But this height value may vary on platforms, e.g. phone vs tablet. I'm still looking into it.

    Hope this helps.
