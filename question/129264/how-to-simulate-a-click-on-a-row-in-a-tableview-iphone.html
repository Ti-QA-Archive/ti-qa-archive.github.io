How to simulate a click on a row in a tableView (iPhone)

Here is my use case : I have app with two tabs :

  • one tab called "new entry" with a form to create a new entry
  • another tab called "my data" listing all previous entries (a tableview). one click on a row displays the detailed view of the entry.

When my user is in the tab "new entry", and saves the entry, I want to display the detailed view of this entry in the other tab "my data".

But I don't know how to programmatically click on the new row created in the list of "my data".

I'v tried selectRow, but it's just hilighting my row, it doesn't enter in the row.

I know I'm not the only one questionning on this, but no solution so far…
http://developer.appcelerator.com/question/117659/fireevent-on-row-error-on-rowdata

any help ?

            index : (MytableViewArray.length) - 1
        });

    • The code is quite simple. In the first tab (form.js), when the user saves the form, I do this :

      Titanium.App.fireEvent('formsaved', {userData: field.id});

      in app.js, I have this code :

      var tableview = Titanium.UI.createTableView({
        data : data,
        style : Titanium.UI.iPhone.TableViewStyle.PLAIN,
});

windowForFirstTab.add(tableview);

Ti.App.addEventListener('quoteCreated', function(data){
   //go to tab2 
   tabGroup.setActiveTab(1); 

   //entering row corresponding to the inserted form, which is always the last one of the tableview by the way

   //tableView.selectRow(data.length-1);
});

      For info, I have alos this code in app.js to always make sure I have an up-to-date tableview

      windowForFirstTab.addEventListener('focus', function() {
    windowForFirstTab.remove(tableview);
    var data = getDataFromDb();
    tableview = Titanium.UI.createTableView({
        data : data,
        style : Titanium.UI.iPhone.TableViewStyle.PLAIN,
    });
    windowForFirstTab.add(tableview);
);
