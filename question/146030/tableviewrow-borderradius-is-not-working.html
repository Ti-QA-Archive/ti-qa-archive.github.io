TableViewRow borderRadius is not working

Hi,
When I use borderRadius style in tableviewrow element , not only it is not working also row cannot display title value anymore . I am using android device to test this feature. Is it a bug or do I cannot use it properly?

".rowStyle1" :{
    backgroundColor:"white",
    height:"50dp",
    rightImage:"/img/general/menu_button_arrow.png",
    borderRadius:10
}

    borderRadius is not supported on TableViewRows, despite its listing as an inherited property in the API. You can however nest another view inide the TableViewRow and give that the borderRadius. So if your row looked like this before:

    <Alloy>
    <TableViewRow borderRadius="10dp">
        <!-- the rest of your row elements -->
    </TableViewRow>
</Alloy>

    You could put a view inside it to give the appearance of a borderRadius, like this:

    <Alloy>
    <TableViewRow>
        <View borderRadius="10dp">
            <!-- the rest of your row elements -->
        </View>
    </TableViewRow>
</Alloy>

    Obviously the borderRadius property would work just as well in your corresponding style file.
