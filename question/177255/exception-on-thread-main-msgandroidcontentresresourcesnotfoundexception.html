  • The error indicates a missing resource for a 'drawable'. Your controller code in many places is specifying image locations like this

     icon:  'images/<one-of-your-images>png',

    change them ALL to absolute instead of relative

     icon:  '/images/<one-of-your-images>png',

    notice the leading &#x2F;
