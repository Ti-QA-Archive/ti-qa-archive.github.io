    I would definitely check this out; titanium module sdk guide

    Though it details building your very own module, it really has all the tools you're going to need in order to actually make it work.

    I've had issues putting stuff in the /Library/Application Support/Titanium/modules folder…I find it MUCH easier to just do an 'ant build' and then copy the resulting zip file (lives in dist folder) into the root folder of your app, and then compile. It will make a 'modules' folder in your app root directory.

    Not sure if that will work for AndroZip but it would sure benefit you to learn how these modules are built from scratch so that you can understand and identify problems. Believe it or not (snicker) the Windows stuff really is bad, but if you're on a MAC and you can use Eclipse, you're in for a treat; Modules in Titanium are its redeeming quality.
