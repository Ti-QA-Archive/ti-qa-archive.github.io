android share intent without always and just once buttons

Hey everyone,

I am looking for a way to create a share button for android that doesn't display the always and just once button, i have seen this behaviour in other applications, it is like the regular share view but it doesnt have the "always" and "just once" button, this way the user always has to make a selection.

so far i have managed to setup the following:

var intent = Ti.Android.createIntent({
            action : Ti.Android.ACTION_SEND,
            type : "text/plain"
        });
        intent.putExtra(Ti.Android.EXTRA_TEXT, data.link);
        intent.addCategory(Ti.Android.CATEGORY_DEFAULT);
        Ti.Android.currentActivity.startActivity(intent);

is there a way to achieve this in titanium?

Thanks in advance

    Yes, it's possible.

    I have not tested this, but please try updating your code to this:

    var intent = Ti.Android.createIntent({
            action : Ti.Android.ACTION_SEND,
            type : "text/plain"
});
intent.putExtra(Ti.Android.EXTRA_TEXT, data.link);
intent.addCategory(Ti.Android.CATEGORY_DEFAULT);
var chooser = Ti.Android.createIntentChooser(intent, "Send Message");
Ti.Android.currentActivity.startActivity(chooser);
