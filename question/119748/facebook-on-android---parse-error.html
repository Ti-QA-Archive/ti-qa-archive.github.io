Facebook on Android - Parse Error

Hello,

I'm trying to port an iPhone App to Android.

The app is connecting to facebook using:
Titanium.Facebook.requestWithGraphPath(&#39;search?type=user&amp;q=&#39;+encodeURI(searchTextField.value)+&#39;&amp;limit=50&#39;, {}, &#39;GET&#39;, function(qFeed)

I'm getting an error:
"error":{"type":"OAuthException","message":"(#601) Parser error: unexpected '\"50?access_token=XXX….' at position 57."}}
This is happening because the Titanium appends the access token with a '?' instead of a '&'.

I read on the forum that I should append a '&' to the end of my path, but that didn't solve the problem and I'm now getting the error:
[INFO] [38,9589] post search 1 - {"error":{"type":"OAuthException","message":"An access token is required to request this resource."}}

Titanium is still appending the access token with '?' as can be seen from the trace:
[TRACE] D/Facebook-Util( 765): GET URL: https://graph.facebook.com/search?type=user&q=dave&limit=50&?access_token=xxxx…

Any help will be much appreciated!
Idan
