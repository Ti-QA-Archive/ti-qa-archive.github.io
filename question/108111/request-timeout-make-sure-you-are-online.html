    In order to troubleshoot your issue, you should restore any changes you have made to the Titanium source code back to the original.

    Then, copy and paste the following into your terminal/shell so that all the commands execute successfully. Copy the output and paste it to your preferred pastebin service. It's important that you do not modify it in any way, as this could mask problems with your configuration. Do NOT paste it here in the Q&A directly! This is because the system will change the formatting.

    clear
cd /
uname -a
echo "$PATH"
echo "$JAVA_HOME"
java -version
javac -version
python -V
echo "$android_sdk"
adb version
android list target
git --version
echo " --- The following is optional ---"
scons -v
echo "$HTTP_PROXY"
echo "$HTTPS_PROXY"
echo "$SOCKS_PROXY"

    Then someone will be able to advise you.
