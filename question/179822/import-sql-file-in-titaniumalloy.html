import sql file in titanium/alloy

I have sqlite dump file with many lines of code.
It's very hardly do it like this:
db.execute(every_line_of_code);
Is it possible to import this file into database something like this:
Ti.db.importSql('file.sql') or Ti.db.loadFromFile('file.sql')

  • Hi

    There is no built in bulk import method.

    You have two choices;

    • If the database is being supplied with this bulk import then created the SQLite database outside of the app and include it as one of the supporting files. The first action your app does when installed is to relocate the database and install it.
    • If the database is being supplied afterwards then you will only need to provide insert statements. Make sure each insert is on a separate new line in a text file then parse that file a line at a time and run the execute command. Use database transactions at the start and end of the import to speed up the insertions.
