Titanium picker behavior for Android

Hi,

I have a form where I want to have a label which says "Select One" which on clicking should show a picker in expanded mode in Android with a list of values. However, when I add a picker to the view on a click event of the label, the picker only comes as a drop down list and I have click once more to expand it. Is there any way I can add the picker to the view such that it shows the expanded list without any futher user interaction, just like in iOS?

  • The following code worked for me and I got a picker which expands on click on the label.

          var optionsDialogOpts = {
            options : options,
            destructive : 1,
      };

     var dialog = Titanium.UI.createOptionDialog(optionsDialogOpts);

     label.addEventListener('click', function() {

            dialog.show();
     });
