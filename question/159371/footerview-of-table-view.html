FooterView of table view

Am using 3.1.3.GA sdk, Alloys and 2.3.3 Android Emulator. Am creating a table view and for it require to add a footeview and center it, but the image or footerview content not aligning at the center. Is it possible to make it center and place a loading image, My code

index.xml

<Alloy>
    <Window class="container">
        <TableView id="mytable">
            <FooterView>
                <ImageView id="testing" />
            </FooterView>
        </TableView>
    </Window>
</Alloy>

index.tss

".container": {
    backgroundColor:"white"
},
"Label": {
    width: Ti.UI.SIZE,
    height: Ti.UI.SIZE,
    color: "#000"
},
"TableView":{
    top: 0,
    bottom: 0
},
'FooterView': {
    backgroundColor:"red",
    width: '100%',
    height: '50dp',
    right: 0
},
'#testing':{
    image: '/c.png',
    left: '200dp'
}

No property on the FooterView applied.

  • I created a view and placed a label for testing, % width doesnt supported and I created another view and kept the imageview inside it and it works,

    <Alloy>
    <Window class="container">
        <TableView>
            <FooterView>
                <View id="viewTest">
                    <ImageView id="testing" />
                </View>                
            </FooterView>
        </TableView>
    </Window>
</Alloy>

    '#viewTest': {
    backgroundColor:"red",
    width: Ti.UI.FILL,
    height: '50dp'
},
