media image

Hello I would like to save this image taken with the camera in the app, this code only takes the picture, but not stored on disk. I have to take another function to store?

cargarImage.addEventListener('click',function() {
    //Abrimos la camara
    Titanium.Media.showCamera({
         success:function(event)
    {
        var cropRect = event.cropRect;
        var image = event.media;

        var f = Titanium.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory,'camera_photo.png');
        f.write(image);
        var imageView = Ti.UI.createImageView({
                width:'100dp',
                height:'100dp',
                image:f.nativePath
                });
            groupCheckbox.add(imageView);

    },
    allowEditing:true
    });
});

  • The lines in your code

    var f = Titanium.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory,'camera_photo.png');
f.write(image);

    are saving the image in the filesystem. You can check the value returned by the write method. If it is false, a problem may have occured saving the file. But, it should be working.

  • These lines right here should save the file to the applicationDataDirectory using the name camera_photo.png:

    var f = Titanium.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory,'camera_photo.png');
f.write(image);

    You should be able to re-access that image at that location after you've taken the picture.
