Hide and show views with transitions on iOS?

Is it possible? I have an iPad app with a view that contains a tabbed bar. The buttons on the tabbed bar set the visible property of subviews on click. Right now, they just "pop" from one view to the next, i was wondering if there was a way to have transitions.

tb1.addEventListener('click', function(e) {
    if(e.index == 0) {
        vMeasure.visible = true;
        vCount.visible = false;
    } else {
        vMeasure.visible = false;
        vCount.visible = true;
    }
})

sdk 2.0.1, ios 5.1, os 10.7.3

  • Absolutely, check out Ti.UI.Animations…

    http://docs.appcelerator.com/titanium/2.0/#!/api/Titanium.UI.Animation

    You could do something like this:

    var show = Ti.UI.createAnimation({
    opacity: 0,
    duration: 2000,
});

var hide = Ti.UI.createAnimation({
    opacity: 1,
    duration: 2000,
});

tb1.addEventListener('click', function(e) {
    if(e.index == 0) {
        vMeasure.animate(show);
        vCount.animate(hide);
    } else {
        vMeasure.animate(hide);
        vCount.animate(show);
    }
})
