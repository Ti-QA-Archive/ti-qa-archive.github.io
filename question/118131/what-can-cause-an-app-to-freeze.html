What can cause an App to freeze?

I have been working on a mobile app, and lately it has started to freeze on the iPhone (simulator and devices) without any errors. It doesn't crash or throw any exception. It just freezes.

I've started to add Ti.API.info logs everywhere, but the freeze seems to be random.

I've disabled analytics tracking, and I still get a freeze.

The App is opening windows on init, then showing and hiding them. When clicking on buttons.

Memory available is still around 22mb when it freezes. The app doesn't respond to any interactions and is just frozen.

I've searched this forum and have seen several users with freeze issues.
Is there any other way to be able to see what can cause this error?

I've debugged using the console in xCode (which seems to be outputting the same as Titaniums log in Debug mode)

I know, it's not an easy task, especially if your code is very complex, but you can give it a try. Have you tried to recreate the project from scratch?
