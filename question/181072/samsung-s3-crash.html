    I think the S3 is a red herring. java.util.HashMap cannot be cast to org.appcelerator.titanium.proxy.TiViewProxy is your error. It means that you are trying to assign a generic object as a View (remember, all UI objects extend Views).

    So, you probably have some code that is either android-only or that-scree-size-only, where you are trying to add javascript to a view instead of a view…

    var wasSupposedToBeAView = {top:0,width:'100%};
win.add(wasSupposedToBeAView);
// will throw the error you shared
