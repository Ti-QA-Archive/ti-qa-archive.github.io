the frm storing clk and its different values.

  • Accepted Answer

    You can use array's "concat" method: https://developer.mozilla.org/en-US/docs/JavaScript/Reference/Global_Objects/Array/concat

    function pushToGlobalArray(arr) {
    frm  = frm.concat(arr);
}

var clk = [0, 0, 0, 0, 0, 0, 0, 0];
pushToGlobalArray(clk);

clk = [];

// etc.
