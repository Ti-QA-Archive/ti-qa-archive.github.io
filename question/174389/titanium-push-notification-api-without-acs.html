Titanium Push Notification API without ACS

Hello, I am developing an app for Android and iOS platform using Titanium. In the app i have to implement push notification for both platforms. My question here is that can i use Titanium Push Notification API without using ACS or i have to use a third party module for that because according to the requirements i have to implement server side for push notification by myself if so then How?

    Of course you can use Push Notifications without ACS.

    Titanium offers an API for iOS: registerForPushNotifications(. On Android, for GCM, you can use a module, such as net.iamyellow.gcmjs (this one works fine for me).

    On the server side you can use your own implementation. There are also good libraries for your prefered technology: PHP, Node.js, whatever you prefer.
