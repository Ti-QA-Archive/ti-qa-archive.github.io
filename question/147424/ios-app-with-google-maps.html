  • Hi

    First thing to note is that google maps ONLY appears in iOS 5.1 and below, iOS and above uses Apple maps.

    For developers there is no difference in how it works but for users there is a very different visual experience. You cannot choose which is used it is dictated by the iOS version the is installed on.

    There is no requirement for any API key to use either google or apple maps within an iOS app. As you correctly note you do for android though.

    If you think this is weird you are right it is weird - but no API key required.
