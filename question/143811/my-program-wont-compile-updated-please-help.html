    • Not sure I;ll have ti to try the code myself, but one thing strikes me. Why do all your objects have zIndexes? There is a very strong chance that your new windows does show up, but because all the other elements are in front of it.

      Keep in mind, this is not html, there is no need to set visual precedence over other items since they are stacked in the same order they are added to the screen.

      To make it simple: Remove every single zIndex from your code.
