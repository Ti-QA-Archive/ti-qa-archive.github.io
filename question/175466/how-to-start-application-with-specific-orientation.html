  • Hi,

    the orientation is controlled by manifest. You can go in build folder. There you will find a manifest.xml the content you copy to tiapp.xml in android section.

    Here you find entries for application and for activities. In Android documentation you can find informations how you can set i.e. orientation. Here an example:

     <activity android:configChanges="keyboardHidden"
     android:name="org.appcelerator.titanium.TiActivity" android:screenOrientation="portrait"/>

    This code forces portrait mode.
