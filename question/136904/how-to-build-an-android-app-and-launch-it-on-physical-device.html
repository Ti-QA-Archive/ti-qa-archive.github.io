how to build an Android App and launch it on physical device

Hello,

I've imported a new git project to create a version for some Android devices. I work on macosx, i have installed the Android SDK, ddms and adb works and Android tools has been added to path. My device is recognized when i run "adb devices". But i'm not able to build for Android :

  • in tiapp.xml > Deployment targets Android is disabled and when i click on the "?" Titanium says me that "the project was created without support for Android Deployment"

So what should i do ? Do you have some recent docs to build and run on a physical Android device ?

    Hi Max,
    Can you check &lt;target device=&quot;android&quot;&gt;true&lt;&#x2F;target&gt; property set true or not in your tiapp.xml file

     <deployment-targets>
        <target device="mobileweb">false</target>
        <target device="iphone">false</target>
        <target device="ipad">false</target>
        <target device="android">true</target>
        <target device="blackberry">false</target>
    </deployment-targets>

  • As you have imported project from github, so right click on the your project root.

    go to properties >> Project natures >> on the right select Titanium Mobile. click OK.

    Now select your project root and clean it.

    In the Project explorer you will see run option(top right). click on it, choose android device.
    Hope that work for you
