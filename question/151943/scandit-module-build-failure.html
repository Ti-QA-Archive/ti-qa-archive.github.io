    Hi Kumar,

    thanks for signing up for our Scandit SDK. It looks like you are running the app on an iphone 3G (armv6 device) and or with an old version of Xcode that still includes the armv6 architecture by default. Apple has stopped supporting the armv6 architecture by default with Xcode 4.5 out last September and our plugins only feature an armv7 and armv7s slice. We do support the iphone 3G with our native Scandit SDK for iOS though.

    I would suggest that you upgrade to the latest version of Xcode and only build for armv7 devices.

    Christian

    Scandit Support
