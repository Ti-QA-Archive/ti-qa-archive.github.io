Problems with animations

Hi everyone,

After few searches about a crossplatform API to build apps and html5 softs I've finally decided to go on with titanium.

I'm just testing the API right now, basically I'm trying go get some results before going forward.

I'm having some issues with animation, basic stuffs, just display a image and trying to move it by clicking on it. My problem is that is not working.

Here's a little test I've settled up :

var win = Ti.UI.createWindow({
    backgroundColor: '#ccc'
});

var view = Ti.UI.createView({
    backgroundColor: 'transparent'
});

var image = Ti.UI.createImageView({
    image:'/images/image.png',
    top: 100,
    left: 20
});

view.add(image);

win.add(view);

image.addEventListener('click', function() {
    var matrix = Ti.UI.create2DMatrix();
    matrix = matrix.rotate(45);
    var animation = Titanium.UI.createAnimation({
        duration: 1000,
        left: image.getLeft() + 50,
        transform: matrix,
        autoreverse: true,
        repeat: 2
    });
    image.animate(animation);
});

win.open();

So here I'm tring to move the image 50 pixels to the left every time I tap on it.
It's perfectly working on my android phone but not on my browser.
The image do rotate, but there's no moving, and there's no autoreverse either, on both my pc's browsers and my android's google chrome browser.
I'm working on a Ubuntu 64 bits and I've tried with both firefox and chromium.
I'm stucked on this for 2 days now, if somebody could help I would really appreciate.

Thanks.
