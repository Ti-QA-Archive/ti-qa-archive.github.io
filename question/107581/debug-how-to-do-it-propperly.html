  • Accepted Answer

    Welcome, Andy.

    JS validation is highly recommended when working with Titanium, and as an android developer it will provide your biggest time-saver. To enable Javascript validation for eclipse, you need to install the wtp plugin (Web Tools Platform).

    The time it takes to boot an Android emulator or launch an app to an emulator is almost entirely dependent on Google's SDK. However, with this in mind and to give you an example for comparison, on my Ubuntu (i7 processor, 6GB memory) machine it takes 40 seconds to launch an application to an already-booted emulator, as this video video demonstrates.

    See the rest of this thread for lots of good advice from other users about improving your development process.

    • Hi Andy,

      I'm in charge of the IDE tooling at Appcelerator, so hopefully I can answer a few of your questions:

      1) Currently for validation in Aptana Studio we use a combination of JSLint (style errors) + the Mozilla Rhino engine (parse errors). We've read that some people consider the validation in WTP so strict as to make it useless, so we want to make sure the errors are relevant. We'll investigate to see if we can add the kinds of validation you're asking about to the current engine.

      2) With regards to debugging, we will be adding that very soon. It's one of the top features we have on our list, and the goal is to let you debug exactly as you expect, setting breakpoints, watching variables, etc. Our current plans are to get a preview of that out in Q1 of this year.

    • You are welcome, George! :)

      I thought js validation and syntax formatting was enabled by default for js files in Aptana. See what you have set in Preferences &#x2F; Editors &#x2F; JavaScript &#x2F; Validation &#x2F; JSLint JavaScript Validator. Use the search field at the top of preferences to search for the words "javascript" and "validat", to see what comes up.

      You should also be able to disable js syntax formatting per code block, as these instructions demonstrate.

      For Aptana advice, it's probably best to post a question to its community, at aptanastudio.tenderapp.com
