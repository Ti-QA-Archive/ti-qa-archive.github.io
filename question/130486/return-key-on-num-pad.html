Return key on Num Pad

Is there a way to have a return key to close the numpad? Can't seem to find it anywhere. I need to use the numpad not another keyboard.

  • you need to add a toolbar to the keypad with a button to dismiss it.

    here is a reusable function that I wrote with help from other posts here. you will have to adjust it for your own use.

    ....
var done = Ti.UI.createButton({
    systemButton:Ti.UI.iPhone.SystemButton.DONE
});

done.addEventListener('click', function(e) { 
        this.activeFld.blur();
});

var flexSpace = Ti.UI.createButton({
    systemButton : Ti.UI.iPhone.SystemButton.FLEXIBLE_SPACE
});

....
createNumTextFieldWithDoneBtn: function(props) {
    props.keyboardToolbar = [flexSpace, done];
    props.keyboardToolbarColor = '#333';
    props.keyboardToolbarHeight = 40;
    var txtField = Ti.UI.createTextField(props);

    txtField.addEventListener('focus', function(e) {
        txtField.keyboardToolbar = [flexSpace, done];
        done.activeFld = txtField;
    });

    return txtField;
}
