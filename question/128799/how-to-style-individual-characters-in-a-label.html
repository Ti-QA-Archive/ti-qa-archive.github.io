  • Hi Nicola

    Thats individual character style may not be possible but you can achieve that in many different ways. Like using the webview where you can add the HTML text as per your need and treat it as label.

    you can also add the label inside the label with the proper formating you need.

    //
// create base UI tab and root window
//
var win1 = Titanium.UI.createWindow({  
    title:'Tab 1',
    backgroundColor:'#fff'
});
var tab1 = Titanium.UI.createTab({  
    icon:'KS_nav_views.png',
    title:'Tab 1',
    window:win1
});

var label1 = Titanium.UI.createLabel({
    color:'#999',
    text:'I am Window 1',
    font:{fontSize:18,fontFamily:'Helvetica Neue'},
    textalign:'left',
    width:'320sp'
});

//labelname
var lblName = Titanium.UI.createLabel({ 
    text:"Cool", 
    font:{fontSize:32},
    color:'#999',  
    textalign:'right',
    width:'auto'
});
label1.add(lblName);
win1.add(label1);

    like the label in the label you can try any other control too.
