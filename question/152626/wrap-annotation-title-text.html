  • On iOS, I think you could use the customView property of Ti.Map.Annotation in order to add a label in it. This property also exists on the Google Map API v2 for android : see Modules.Map.Annotation.

    Thus, I think it should work on the 2 platforms… But I don't know if there is any height/weight limit on this view… Maybe you'll be able to tell us :P
