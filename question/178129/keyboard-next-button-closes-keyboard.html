Keyboard next button closes keyboard

I'm using returnKeyType: Titanium.UI.RETURNKEY_NEXT to display a next button on my keyboard.

When I press the next button the focus moves to the next field, however the keyboard closes, and then needs to be reopened by clicking again. (I don't want this)

If i don't click the next button and click the inside the next textfield, the keyboard does bot close, and allows input into the next textfield. (i want this behaviour)

Question is how can I prevent the keyboard from closing when pressing the next button?
So that the key board only hides when the next field is not a textfield?

Is this possible? I hope so….

Thanks in advance!
