  • Accepted Answer

    Your only option at the minute is to use Ti.API.debug() and Ti.API.info() calls, and examine the output. I would recommend running DDMS, connecting to emulator or device and using that to examine the output from those calls.

    This is hopefully going to improve very soon when the Ti-integrated version of Aptana is released.
