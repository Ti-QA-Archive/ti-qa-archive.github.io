Using Cocos2d or another isometric game engine with Titanium

I'm wondering if anyone knows how to use a 2d/isometric game engine within a Titanium app to produce a Farmville-like game. Is there a way to use cocos2d?

    QuickTiGame2d is a 2d gaming module for Titanium that you can use freely in terms of New BSD license. It supports 3D viewport transformation so you can create Farmville-like games with it.
