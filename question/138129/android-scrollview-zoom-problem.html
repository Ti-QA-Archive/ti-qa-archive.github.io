Android ScrollView zoom problem

Hi guys, I'm having a bit of a problem transfering a working iOS code to Android.
Im using the latest Titanium software and Google API 2.2

In my IOS app I have a image thumb, and when I click on it I get a scrollView with a fullsize image inside
which I can then zoom \ pinch \ scroll etc.

In Android I can't seem to get it to work. I manage to open the scrollview with the large image, but scrolling and zoom just doesnt work.

neither does my dbclick event which changes the scrollview zoomScale property

Any ideas ?

        var fullImage = Ti.UI.createImageView({
           image: bigImgSrc,
           preventDefaultImage:true
        });


        var scrollView = Titanium.UI.createScrollView({
            contentWidth:fullImage.width,
            contentHeight:fullImage.height,
            backgroundColor:'black',
            showVerticalScrollIndicator:false,
            showHorizontalScrollIndicator:false,
            maxZoomScale:2,
            minZoomScale:1,
            zoomScale:1,
            disableBounce:true,
            horizontalBounce:false,
            verticalBounce:false
        });

         scrollView.add(fullImage);

and the dbclick code:

    fullImage.addEventListener('dblclick',function(e){
        (scrollView.scale <= 1.0) ? scrollView.zoomScale = 2.0 : scrollView.zoomScale = 1.0;
    });
