  • Hi Thomas Lynch,

    Thanks for your query.

    You can use mapbox for offline map using titanium.

    iOS offline map:

    Yes, you can use the titanium mapbox for offline map.

    Android offline map:

    Mapbox is actively working on an Android SDK, which in turn can eventually be added to a titanium-mapbox module.

    You can also use leaflet.js. Leaflet is only for web browsers. To use it, you'll have to put it in a WebView.

    Link: A Mobile App with Offline Maps using Titanium

    There is also a offline map in the marketplace and you can have a look if you like.

    Hope this will clear things up for you.
