  • Your problem occurs because the emulator did not launch properly cannot launch because of spaces in the command line parameters.

    You can edit the builder.py file around like 400 or so from:

    self.sdcard ,

    to:

    '\"' + self.sdcard + '\"',

    This wraps the sdcard location param with quotes when called.
