Maps for iphone and android

Hi am using 3.1.3.GA sdk, Alloys and 2.3.4 Android Emulator. I need to use Maps for my Application. I am having an example with Ti.Map and in documentation its listed as depracated. And I have to make maps work on both iphone and android. I have to use maps in two places and in docs its mentioned that for android only one map view can be used and else it will throw an exception.
DOCS Explanation


Android MapView Limitation

On Android, only a single map view is supported per application. Adding a second map view to an application results in an exception being thrown.

I think I can call the same view in both places and I have to know any samples are available for it. Thanks in advance.

  • There is a solution. You use a singleton map and you save the status of the map (region, annotations …) in a persist memory (property). If you want to show the other map, you add this map to an other window. In a past project it works!

    Good look!
