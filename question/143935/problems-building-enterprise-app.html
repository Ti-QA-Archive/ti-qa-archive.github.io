Problems building enterprise app

I created a small app using my personal developer license for my place of work. We are starting to use apps internal and we are hoping to use the app that I created. I've built it for enterprise distribution which created a ipa file and I've also given him the xcode project. However, when my coworker tries to build it on his work mac, he gets errors. The current error is looking for the builder.py script. Is there something that I need to do in order to get him the build files he needs for distribution?
