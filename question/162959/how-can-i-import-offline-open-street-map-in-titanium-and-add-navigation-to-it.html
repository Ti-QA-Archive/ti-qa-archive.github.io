  • Interesting question! I'm going to be talking about a lot of this stuff at this year's tiConf in New York, so grab your ticket!

    While Titanium doesn't do most of what you are asking out of the box (hence the talk), you should check out the following:

    • Mapbox.com, specifically their TileMill product
    • OSM Bright
    • titanium-mapbox, a Titanium module wrapper for the Mapbox iOS SDK

    Notes:

    • Annotation support in the wrapper is still a work in progress. Should have some progress on it very soon.
    • Currently iOS only - Mapbox is actively working on an Android SDK, which in turn can eventually be added to a titanium-mapbox module.
