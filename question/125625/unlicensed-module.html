Unlicensed module

Hello,

Since this morning, I am getting a big red window with written "Unlicensed module(s) detected. You must be a subscriber to use one or more of the module included in the project.".

But:

  • I do have an Indie account
  • I don't use any Ti+ Modules !
  • I have tried to recreate the project a lot of times, to logout / login, to use another Titanium account…

It seems like if I remove the keychain module, it's working back. But I bought this module (http://tinypay.me/~dFAy9Rg) and it has nothing to do with Ti+ Modules.

Would someone have a solution for this issue ?

  • Accepted Answer

    Hello. I had no idea that Appcelerator verified the module GUID against their central service. That caught me by surprise. Since the module I was selling has the same GUID than the module on the Appcelerator Marketplace, the serves thinks you never bought the module. I'm in the middle of vacations so I can't solve the problem right now, but tomorrow I will release new versions of all my modules with a different GUID. Very sorry about all this mess.
