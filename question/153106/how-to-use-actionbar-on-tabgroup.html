How to use actionBar on tabGroup

ActionBar is working for me on normal Window, but when I try it on TabGroup window…

View:

<Alloy>
    <TabGroup id="tabGroup">
        <Require src="prefSrcTab"/>
        <Require src="prefDstTab"/>
    </TabGroup>
</Alloy>

Controller:

$.tabGroup.addEventListener('open', function(e) {
    var activity = globals.tabs.getActivity();

    activity.actionBar.title = L('settings');
    activity.actionBar.displayHomeAsUp = true;
    activity.actionBar.onHomeIconItemSelected = function() {
        alert("Home icon clicked!");
    };

    activity.invalidateOptionsMenu();
});

Now, that 'getActivity' code came from the Titanium Docs. But it rise an error saying that 'globals' is not defined.

And if I change it to

var activity = $.tabGroup.getActivity();

It just rise a alert saying that my app stopped to respond.

What Am I missing here?
It is possible to create an ActionBar within a TabGroup right?

  • This should work:

    $.tabGroup.addEventListener('open', function(e) {
    if(e.activity) {
        e.activity.actionBar.title = L('settings');
        e.activity.actionBar.displayHomeAsUp = true;
        e.activity.actionBar.onHomeIconItemSelected = function() {
            alert("Home icon clicked!");
        };
         e.activity.invalidateOptionsMenu();
    }
});

    Starting with 3.3.0, you could use this sort of XML when defining your tab group:

    <Alloy>
   <TabGroup>
      <Tab>...</Tab>
      <Tab>...</Tab>
   <Menu title="Foo" displayHomeAsUp="true" onHomeIconItemSelect="doFunction" />
   </TabGroup>
</Alloy>

    But maybe you don't have or want a menu. So, starting with 3.4.0, you will be able to use this instead (or addition to)

    <Alloy>
   <TabGroup>
      <Tab>...</Tab>
      <Tab>...</Tab>
   <Activity title="Foo" displayHomeAsUp="true" onHomeIconItemSelect="doFunction" />
   </TabGroup>
</Alloy>
