how to transmit variables between two windows

hi,

i have two windows, the first :

row.addEventListener('click',function(e)
        {
            var winArticle = Ti.UI.createWindow({url:"article.js"});
            winArticle.open();
        });

the second window :

var winArticle = Ti.UI.currentWindow;
var view = Ti.UI.createView({backgroundColor:"red"});
winArticle.add(view);

how to get the variable "e" in the second window ?

Thank you.

    see the question, I believe it will solve your problem

    http://developer.appcelerator.com/question/114241/adding-property-to-the-view

    // the first window :
var winArticle = Ti.UI.createWindow({
   url:"article.js",
   _event: e
});

    // the second window :
var winArticle = Ti.UI.currentWindow;
var eventData = winArticle._event;
var view = Ti.UI.createView({backgroundColor:"red"});
winArticle.add(view);
