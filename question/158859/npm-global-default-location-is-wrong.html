npm global default location is wrong

I am trying to install grunt and can't get it to work globally. I noticed that if I run the command:
npm -g root
I get the location where it installs grunt as:/Applications/Titanium Studio/TitaniumStudio.app/Contents/MacOS/Password:/usr/local/share/npm/lib/node_modules

That doesn't look right!

How do I change it? I've tried a suggestions of running "npm uninstall npm -g" but I can't as it can not find uninstall?

    Try:

    npm set prefix &#x2F;usr&#x2F;local

    More info here and here.
