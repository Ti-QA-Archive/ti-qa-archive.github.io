  • I came across the same issue and was able to store the image name (of event.media) by using a simple php function:

    $theFileName = $_FILES['userfile']['name']; //'name' of image

    When you are writing an image to your server, you first need to extract the name of the .jpg and this is partially how you do it. Hope this helps!
