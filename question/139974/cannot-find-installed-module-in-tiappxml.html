  • Hello Martin,

    Just See below Code which i m using for testing purpose, i m using ti.admob module for testing…

    In tiapp.xml, i write code like….

    <modules>
        <module platform="iphone" version="1.0">ti.admob</module>
</modules>

    In app.js, 

    
Titanium.Admob = Ti.Admob = require('ti.admob');

    It's Working great for me…. I hope it will help you… Best Luck…
