Detecting when "delete" is pressed, even if the textField.value.length = 0

Hi, how do you detect when the delete key is pressed on the keyboard, even if the textField.value.length = 0? It seems change only fires if the field actually changes, rather than if a change is attempted.

Thanks!

    First create an counter or boolean, named previousReciptients = false;

    So put a space in the textfield, the check on change if the value has become '' < empty string. If previousReciptients is false the do the next part.

    If it is an empty string you do your first thing by highlighting the previous recipients. Then set previousreciptients to true

    Then you set the textfield to ' ' < an space again, and when the value becomes '' empty string again and previousReciptients is true, remove them.

    Kinda hard to understand at first but this should do the trick ;)

    
var prevRecip = false;

if( textfield.value = '' && !prevRecip ) 
{ 
--- setPrevRecip(); 
--- prevRecip = true;
--- textfield.value = ' ';
}
else if ( textfield.value = '' && prevRecip ) 
{
--- removePrevRecip();
--- prevRecip = false;
--- textfield.value = ' ';
}
