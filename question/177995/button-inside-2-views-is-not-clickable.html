button inside 2 views is not clickable

Hi, I've got some buttons which appear inside a view which is the child of another view.

The button doesn't work (is not clickable), but when I add the button to the parent view it works fine. So seems like there is some issue with a button inside 2 views.

I've tried setting the zIndex to a high number (didn't work) and creating a blank eventListener for the parent View (which was suggested in another discussion here) and also didn't work.

The hack to get around it would to be to place the buttons inside the first view and line them up manually but it'd be nice to know the solution or if its a bug.

  • Hello, Mick. Here is the Proper code to put a button inside two views.

    var win = Ti.UI.createWindow({
backgroundColor:'yellow'
});

var view1 = Titanium.UI.createView({
   borderRadius:10,
   backgroundColor:'red',
   width:400,
   height:400
});
win.add(view1);
var view2 = Titanium.UI.createView({
   borderRadius:10,
   backgroundColor:'green',
   width:200,
   height:200
});
view1.add(view2);

var button = Titanium.UI.createButton({
   title: 'Hello',   
});
button.addEventListener('click',function(e)
{
   alert("You clicked the button");
});
view2.add(button);

win.open();

    Hope that helps you. Thanks.
