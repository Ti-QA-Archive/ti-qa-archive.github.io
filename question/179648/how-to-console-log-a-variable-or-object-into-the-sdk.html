    You will need to run your app in debug mode on a physical device in order to log any information to the console. Your code doesn't need to change at all either.

    • Ok, firstly make sure to keep replies either in the same thread or as additional comments if it is just extra information. Adding more answers will make it harder for other community members to follow the thread.

      So if you wanted to display a variable in your console you would simply write this:

      Ti.API.info(myVariable);
// or for an object that may be in JSON format
Ti.API.info(JSON.stringify(myVariable));

      Lastly, yes you would need to rerun the debug compile each time. But, you could use TiShadow which will allow you to cast a live version of your app and automatically update as you change your code.

      Some helpful links:

      • http://tishadow.yydigital.com/
      • http://stephenfeather.com/blog/tishadow-getting-started-part-1/

      Hope this helps.
