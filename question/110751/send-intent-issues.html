Send Intent Issues

Using the following code (basically copied from the webinar):

    share.addEventListener('click',function(e)
    {
        var activity = Ti.Android.currentActivity;
        var intent = Ti.Android.createIntent({
            action: Ti.Android.ACTION_SEND,
            type:'text/plain'
        });
        intent.putExtra(Ti.Android.EXTRA_TEXT,'test');
        intent.putExtra(Ti.Android.EXTRA_SUBJECT,'subj');

        activity.startActivity(Ti.Android.createIntentChooser(intent,'Share'));

    });

No chooser appears and the user is brought to a SMS screen. I'd like the user to be able to choose the intent or use email at least.
