Error opening Titanium project

I followed all the steps given here https://github.com/appcelerator/alloy for cloning and running the samples. But now i am unable to open the same project it is giving me an error The project description file (.project) for &#39;alloy&#39; is missing what may have caused this?

  • Your question is lacking a lot of detail so I'm not sure what steps you're actually performing. Which samples are you trying to run? What do you mean by "unable to open the same project" … which project, open in what?

    If you've cloned the repo and are trying to run the samples in the test/apps/* directories, use the jake script like this:

    // from your alloy repo directory
jake app:run dir=basics/simple

// to run for a specific platform
jake app:run dir=basics/simple platform=android

    The test apps are not complete projects. You cannot import them or open them as Titanium projects within Studio. You'll have to create a new Alloy project, then copy the contents of the test app's app folder over the top of your project's app folder.
