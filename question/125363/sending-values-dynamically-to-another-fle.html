    • Ok, are you creating the windows with url property or just with Ti.include? If you're creating with url, so the windows run in different context. So you can use events or add properties. Look this thread: http://developer.appcelerator.com/question/86681/passing-data-between-window

      If not, then it will look like this:

      //file1.js
var checkBoxValue;
//....//

//file2.js
Ti.include(file1.js);

tableview.addEventListener("click", function(e){
        var checkBox = e.row.children[0];
        checkBoxValue = checkBox.value;
    });

      I uncommented "checkboxValue:checkbox[i].value", 'cause it looks like should be working. After this the button[i] will have checkBoxValue = true or false (your default value). And it will remain the same till the end, because you don' change it. Checkbox[i].value is copied to checkBoxValue, but these are two different variables. When you click the switch, checkbox[i].value is changed, but not button[i].checkBoxValue. Try this:

      checkbox:checkbox[i] // instead of checkboxValue:checkbox[i].value
//....//
var k = e.source.checkbox.value // instead of var k = e.source.checkboxValue;
