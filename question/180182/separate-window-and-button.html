Separate window and button

Hello everybody !

I'm just a new user of Titanium Studio, and I'm currently learning how to develop on Android.

More, I'm sorry for my bad English (because I'm French !), and I hope I will be clear.

I've just finished an application which can count the number of click and the number of swipe on the screen.
I want to add a button "Restart", which is able to restart the number of click and swipe.
But, when I click on the button, it counts it like a click ! I've tried to initialize the number of touch at -1 when I restart, but sometimes I can see the value '-1' appears on the screen.
So, I use the event "longclick" for the button Restart.
What I want is to be able to just click on the button, and it restarts the number of click and swipe, without counting like a click on the window.

I give you my code :

var win = Titanium.UI.createWindow({  
    title:'Touch',
    backgroundColor:'#fff'
});


var label = Titanium.UI.createLabel({
    color:'#999',
    text:'Test du nombre de Touch et de Swipe !',
    font:{fontSize:20,fontFamily:'Helvetica Neue'},
    textAlign:'center',
    width:'auto'
});


var button = Titanium.UI.createButton({
    title : 'Restart',
    bottom : 40
});

var click = 0, swipe = 0;

win.addEventListener("click", doClick);
win.addEventListener("swipe", doSwipe);
button.addEventListener("longclick", clickButton);

function doClick()
{    
    click++;
    label.text = 'Nombre de touch : '+ click +'\nNombre de swipe : '+ swipe;
}

function doSwipe()
{
    swipe++;
    label.text = 'Nombre de touch : '+ click +'\nNombre de swipe : '+ swipe;
}

function clickButton()
{
    click = 0;
    swipe = 0;
    label.text = 'Nombre de touch : '+ click +'\nNombre de swipe : '+ swipe;
}

win.add(label);
win.add(button);
win.open();

Thanks a lot !

  • Hi Axel,

    Put the property bubbleParent to false for the button. It will resolve your issue. You can use click listener instead of the long click listener. If it works, please mark as answer/fixed.
