Using dataFilter with passed arguments

Hi!
I'm having a problem filtering data on my TableView.
The thing is that the dataFilter function is being executed previous to the controller which actually holds the variable I need to do the filtering.

Here is my code. Any clues?

TableView:

<TableView id="lstSubjects" search="name" dataCollection="subjects" dataFilter="filterSubjects">
    <SearchBar hintText="Search Subjects"></SearchBar>
    <TableViewRow rowid="{id}" height="40">
        <Label id="lblSubjectName" text="{name}" />
    </TableViewRow>>
</TableView>

Controller:

var subjects = Alloy.Collections.subjects;
subjects.fetch();

var args = arguments[0] || {};
// This is the variable I need to do de filtering
var classId = args.selectedClass.id || null;
var className = args.selectedClass.get('name') || "";

function filterSubjects(collection)
{    
    // Needs the classId variable set at the begining. but It's undefined because this is executed first.
    return collection.where({classId: classId});    
}

    I just found out.
    Running fetch() is what executes all databinding related stuff.

    I just moved the fetch() call to the end of the controller and everything worked smoothly.
