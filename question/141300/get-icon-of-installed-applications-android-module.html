Get Icon of installed applications (Android module)

Hi, I am developing small android module in which I am displaying all installed application on android device. I am able to display package name and application name successfully.. Now I want to display icons (App icons) as well. My kroll method looks like.

ArrayList<Bitmap> App_icon = new ArrayList<Bitmap>();
ArrayList<Drawable> App_icon1 = new ArrayList<Drawable>();
for (int i=0; i<packages.size();i++)
{
    Drawable icon = pm.getApplicationIcon(packageInfo);
    Bitmap bitmap = ((BitmapDrawable)icon).getBitmap(); 
    App_icon1.add(icon);
    App_icon.add(bitmap);
}


@Kroll.method
    public Bitmap[] get_app_images()
    {
        Bitmap[] app_images = App_icon.toArray(new Bitmap[App_icon.size()]);
        return app_images;
        /*Drawable[] app_images = App_icon1.toArray(new Drawable[App_icon1.size()]);
        return app_images;*/
    }

In app.js I called get_app_images()

var App_icon = appnames.get_app_images();

It gives me following error: Unable to convert unknown Java object class 'android.graphics.drawable.BitmapDrawable' to Js value . I tried to send image array in both format bitmap as well as in Drawable format but it's not working… Is this the proper way to pass images from java to java-script? How to pass Image array from Kroll method to java-script in android?

Need Help…. Thank you….

  • ArrayList<TiBlob> App_images = new ArrayList<TiBlob>();

//////////////////////////////////////////////////////////////////////////////////////////
Drawable icon = pm.getApplicationIcon(packageInfo);
Bitmap bitmap = ((BitmapDrawable)icon).getBitmap(); 
App_images.add(TiBlob.blobFromImage(bitmap));

//////////////////////////////////////////////////////////////////////////////////////////
@Kroll.method
public TiBlob[] get_app_images()
{
    TiBlob[] app_imgs = App_images.toArray(new TiBlob[App_images.size()]);

    return app_imgs;
}
