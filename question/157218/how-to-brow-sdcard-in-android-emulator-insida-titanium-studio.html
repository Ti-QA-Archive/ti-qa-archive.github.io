How to brow SDCard in Android emulator insida Titanium Studio

Hi, I'm evaluating Titanium for a customer and come from a background developing Android and iOS app natively, e.g. I have used ADT for Android and XCode for iOS. But it would be sweet if Titanium would be a alternative to developing several versions of the same app.

Ok, now to my question. If I'm using ADT and Eclipse, I can browse the content of the SD Card using 'Window -> Show view -> Other -> File Explorer.

Can I do the same within Titanium Studio? If so, how do I do it?

I'm using Titanium Studio, build: 3.1.2.201308091617, Build: jenkins-titanium-rcp-master-482 (origin/master), date: 09 August 2013, 16:20:52 running on MBPro using OS X 10.8.4 (12E55).

  • HI Jonas,

    for Browse SDCard in Android emulator inside Titanium Studio, you can use DDMS..

    1) switch to DDMS perspective

    2) select the emulator in devices list, whose sdcard you want to explore.

    3) open File Explorer tab on right hand side.

    4) expand tree structure. mnt/sdcard/
