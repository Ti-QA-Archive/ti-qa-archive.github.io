Different android / iphone behavior with titanium-jasmine

I've been using titanium-jasmine in some cross-platform unit tests, particularly the section How to use in your projects.

My app.js file is:

if(Ti.version < 1.8)
{
    alert(L('noVersion1_8'));
}
else if(Ti.Platform.osname === 'mobileweb')
{
    alert(L('noMobileWeb'));
}
else
{
    //require and open top level UI component
    var boot = require('/ui/bootstrap');
    Ti.API.info('Launching as ' + Ti.Platform.osname);
    boot.launch();
}

And bootstrap.js looks like:

var myApp = 
{
    tests_enabled : true
};

exports.launch = function()
{
        if(myApp.tests_enabled !== true)
        {
            // Window stuff here
        }
 };

Ti.include('/test/enabled.js');
Ti.include('/test/tests.js');

which is how the demo code works (I believe).

This works fine for Android, but when I try it in iphone (or ipad) I get:

[INFO] MyApp/1.0 (1.8.1.a24502a)

[ERROR] Script Error = Can't find variable: myApp at enabled.js (line 5).

[ERROR] Script Error = Can't find variable: myApp at tests.js (line 2).

[INFO] Launching as iphone

Any pointers as to where to poke appreciated!

    Please confirm if this is exactly your code structure:

    Resources:
    -> ui/bootstrap.js
    -> test/enabled.js
    -> test/tests.js

    If yes, your code in bootstrap.js should be like this:

    ui/bootstrap.js

    Ti.include('../test/enabled.js');
Ti.include('../test/tests.js');

    • oh did you use myApp in enabled.js/tests.js? If yes then you should move myApp definition to app.js. With current commonjs implementation in SDK 1.8.1, variables defined in app.js will have global scope (on iOS). But I think this is being deprecated since there's no global scope in commonjs approach.

      app.js

      if(Ti.version < 1.8)
{
    alert(L('noVersion1_8'));
}
else if(Ti.Platform.osname === 'mobileweb')
{
    alert(L('noMobileWeb'));
}
else
{
    var myApp = {
        tests_enabled : true
    };

    //require and open top level UI component
        var boot = require('/ui/bootstrap');
        Ti.API.info('Launching as ' + Ti.Platform.osname);
        boot.launch();
}

      bootstrap.js

      //var myApp = 
//{
//    tests_enabled : true
//};

exports.launch = function()
{
        if(myApp.tests_enabled !== true)
        {
            // Window stuff here
        }
 };

Ti.include('/test/enabled.js');
Ti.include('/test/tests.js');
