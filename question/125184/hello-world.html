    It means that there is no default run configuration for your project…

    1. You can try to highlight the resources folder in your App Explorer (tree view on the left side pane) then try to run from the iPhone or iPad simulator from the little icon above your project name (look like a tv screen with a play arrow/button on it).

    By default, Run on the Run menu only launches the last build/run config you have completed.

    1. The other, more complicated option, is to setup a run configuration manually. From the menu at the top, click Run, then Run Configurations… Find the Titanium iOS Simulator section, and add a run configuration there manually. You shouldn't have to do this though unless something broke.
