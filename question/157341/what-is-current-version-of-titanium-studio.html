  • Accepted Answer

    The "Object Team Patch for JDT/Core" error is due to having JDT installed. You would need to go to Preferences > Install/Update > Available Software Sites and make sure only "Titanium Studio Updates" entry is selected, and then run Help > Check for Updates again.

    Hope this helps.
