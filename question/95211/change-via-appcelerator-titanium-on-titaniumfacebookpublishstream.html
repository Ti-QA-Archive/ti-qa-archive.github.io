change "via appcelerator titanium" on Titanium.Facebook.publishStream

Is there a way to change the "via appcelerator titanium" text that is sent using Titanium.Facebook.publishStream? I'd like to change it to the name of my app when people post comments to their wall. Thanks

    when you set up your own application in facebook, you will get API keys to use, when you use these keys, your applications name will appear instead of Appcelerators

    var fbButton = Titanium.Facebook.createLoginButton({
    'style':'wide',
    'apikey':'9494e611f2a93b8d7bfcdfa8cefdaf9f',  // change this !!
    'sessionProxy':'http://api.appcelerator.net/p/fbconnect/', // change this !!
    bottom:10
});

    http://www.facebook.com/developers/#!/developers/createapp.php
