ACS Add Friend oneway not working

I've done a sample app for adding a user as friend. It's of oneway type. So I've added approval_required : false in the code. but still it is sends a request to the user and the user is becoming my friend only after he accepted request. I have also set Friend request type as one way in my ACS settings page. Can anyone tell me what I'm doing wrong?

    Cloud.Friends.add({
        user_ids: user.id,
        approval_required : false
    }, function (e) {
        if (e.success) {
            alert(user.first_name + ' has added to your friend list');
            btnBack.fireEvent('click');
        } else {
            alert('Error: ' + ((e.error && e.message) || JSON.stringify(e)));
        }
    });

    • I encounter the same problem in two-way friends. and I changed my code to: 

      Cloud.Friends.add({
        user_ids: user.id,
        approval_required : "false"
    }, function (e) {
        if (e.success) {
            alert(user.first_name + ' has added to your friend list');
            btnBack.fireEvent('click');
        } else {
            alert('Error: ' + ((e.error && e.message) || JSON.stringify(e)));
        }
    });

      And it works! WOW~
