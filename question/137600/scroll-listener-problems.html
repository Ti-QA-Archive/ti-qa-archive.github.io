  • Hi Matt,

    I have some trick for you that fire event only once. Try this.

    table.addEventListener("scrollEnd", function(e) {
    if(t) {

        clearTimeout(t);

        t = setTimeout(function(ev) {
            Ti.API.log("FIRE ! scroll End");
            t = null;
        }, 1000);

    } else {
        t = setTimeout(function(ev) {
            Ti.API.log("FIRE ! scroll End");
            t = null;
        }, 1000);

    }
});

    It delays scroll view event for 1 second.
