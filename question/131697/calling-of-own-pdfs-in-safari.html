  • You can open PDF using webview container. Try this:

    var w = Ti.UI.createWindow();

var f = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory, 'test.pdf');
if (f.exists()) {
    var wv = Ti.UI.createWebView({url:f.nativePath});
    w.add(wv);    
} else {
    alert('Find not found');
}

w.open();

    Hope that help.
