  • Accepted Answer

    Use result.fieldByName(&#39;fiendName&#39;) to retrieve data from ResultSet.
    Refer to this.

    var db = Titanium.Database.open('onedb');
var result = db.execute('select * from onedb where ID=2');

while(result.isValidRow()) {
    Ti.API.info(result.fieldByName('NAME') + ' - #' + result.fieldByName('ID'));
    result.next();
}

result.close();
db.close();
