  • I figured out the problem. Seems there is something wrong with the Titanium.Facebook API. The way I have my project set up is that the normal "Resources" folder has all my iOS-specific files, then in the "android/" subfolder I have all the Android files. For some reason the regular files (That should've been overwritten by the Android ones) had a Ti.Facebook call so I guess it picked that up? Don't know why it would, judging as how the other file out overwrite that one plus it was commented out.
