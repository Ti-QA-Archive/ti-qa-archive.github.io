Constant crash when updating database

Hey, I'm having trouble updating a database, Every time the code runs the emulator exits with a null error. [DEBUG] Session did end with error (null) - That is everything I get. No SQL syntax error, no variable cannot be found error. nothing… And I can't figure out whats wrong. The error IS with the update query as the code works if I remove that one line.

It is only on the UPDATE query, the INSERT works as it should.

Is there something wrong with the query?:

var db = Ti.Database.open('planner');
if (event_id==null) {
    db.execute('INSERT INTO party_events (id_party,event_title,event_time) VALUES (?,?,?)',party_id,e_name,e_time);
} else {
    db.execute('UPDATE party_events SET event_title = ?, event_time = ? WHERE id_party_event = ?', e_name, e_time, event_id);
}
db.close();

Cheers!

  • Ah! After trawling through my entire code I found one instance of db that hadn't been closed! Even though I had a check in place to close any open databases before trying to perform the update it obviously didnt work and let the open instance through without closing it.

    Thanks everyone for your help, and hopefully this helps anyone else with the same problem… Don't rely on a function to test for open databases, just remember to ALWAYS close a database after execution!
