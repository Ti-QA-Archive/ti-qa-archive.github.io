  • Hello Wan,

    The result of contact reading is an array or collection. A collection is an array ob objects :-)
    If you want to sort this, you must use the javascript function'sort'. This function can use a callback to define the method of sorting.

    myArrayofConacts.sort(function(a,b){
    return (a.lastname>b.lastname)? true : false;
});

    You can read more at W33schools

    Sorting of japanese words are an other theme. I'm not a japanese, but I know there are more then one alphabets (katakana, hiragana, kanji). http://www.localizingjapan.com/blog/2011/02/13/sorting-in-japanese-%E2%80%94-an-unsolved-problem/
