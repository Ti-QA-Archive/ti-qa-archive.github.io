Event swipe in a View with WebView

Hi to all
I have a main View (movableview) that have an event swipe to move it left or right

$.movableview.addEventListener('lswipe', function(e) {
   if (e.direction == 'left') {
        $.movableview.animate(animateReset);
        hasSlided = false;

   } else if (e.direction == 'right') {
      $.movableview.animate(animateRight);
      hasSlided = true;
   }
});

inside this one sometime I add a webview that need to be scrollable.
The WebView scroll perfect if I disable the swipe in main View.
How can I have WebView working if the swipe is not left/right but up/down?
Thanks

  • WebView implements ScroolView. So, by default WebView inherits the same swipe event handlers as the scrollview. No matter what you do, there will always be a conflict in who gets the swipe event.

    In your case its not advisable to just rely on the swipe gesture to move to a different page. Instead, have a button in the header that can trigger the same action as a swipe.
