Android Picker Row font color is White

Hi,

I have upgraded from Appcelerator 3.1.0GA to 3.2.0GA. After the update I am noticing that font color of the picker control has been modified to white, and I am not able to modify the color. Can someone please suggest as to how to modify the color for it. I want to modify the font color of the picker from white to black.

  • Accepted Answer

    <style name="Theme.Titanium" parent="android:Theme">
    <item name="android:windowBackground">@drawable/background</item>
</style>

<style name="Theme.MyTheme" parent="@android:style/Theme.Holo">
    <item name="android:editTextStyle">@style/editText</item>
    <item name="android:spinnerItemStyle">@style/MySpinnerItem</item>
</style>

<style name="editText" parent="@android:style/Widget.EditText">
    <item name="android:textCursorDrawable">@null</item>
</style>

<style name="MySpinnerItem" parent="@android:style/Widget.TextView.SpinnerItem">
    <item name="android:textColor">#000000</item>
</style>

  • Found the solution. This needs a style modification (a custom style.xml for Android). The definition is given below:

        <style name="Theme.Titanium" parent="android:Theme">
    <item name="android:windowBackground">@drawable/background</item>
</style>

<style name="Theme.MyTheme" parent="@android:style/Theme.Holo">
    <item name="android:editTextStyle">@style/editText</item>
    <item name="android:spinnerItemStyle">@style/MySpinnerItem</item>
</style>

<style name="editText" parent="@android:style/Widget.EditText">
    <item name="android:textCursorDrawable">@null</item>
</style>

<style name="MySpinnerItem" parent="@android:style/Widget.TextView.SpinnerItem">
    <item name="android:textColor">#000000</item>
</style>

  • ##Trouble shooting(Update for SDK 3.3.0)

    Starting with Release 3.3.0, the Titanium SDK only supports the AppCompat themes and your target SDK level must be 14 or above. So if you see some error message when running others code as following:

    you need to use a theme.appcompat theme titanium

    You just need to change the parent to appcompat one. (See the following TiDoc)

    Define a custom theme

    There are actually 2 steps for deploying a custom theme:

    1. Define the custom theme
    2. Specify the theme you want to use in Tiapp.xml
    • Define the custom theme

    Create a style.xml file in the platform/android/res/values folder relative to root folder of the project and having the following code in side it.

    <?xml version="1.0" encoding="utf-8"?>

<resources xmlns:android="http://schemas.android.com/apk/res/android">
    <style name="Theme.MyTheme" parent="Theme.AppCompat.Light.DarkActionBar">
        <item name="android:editTextStyle">@style/editText</item>
        <item name="android:spinnerItemStyle">@style/MySpinnerItem</item>
    </style>

    <style name="editText" parent="@android:style/Widget.EditText">
        <item name="android:textCursorDrawable">@null</item>
    </style>

    <style name="MySpinnerItem" parent="@android:style/Widget.TextView.SpinnerItem">
        <item name="android:textColor">#333</item>
        <item name="android:textSize">13sp</item>
    </style>
</resources>

    This theme extended Theme.AppCompat.Light.DarkActionBar by defining a custom cursor, textColor and textSize. You can find the syntax for defining the custom theme in Titanium here and the items available in android developer document

    • Specify the theme you want to use in Tiapp.xml
    <android xmlns:android="http://schemas.android.com/apk/res/android">
        <tool-api-level>14</tool-api-level>
        <manifest>
            <application android:theme="@style/Theme.MyTheme"></application>  //specify the theme to use
            <uses-sdk android:maxSdkVersion="20"
                android:minSdkVersion="14" android:targetSdkVersion="18"/>
        </manifest>
</android>
