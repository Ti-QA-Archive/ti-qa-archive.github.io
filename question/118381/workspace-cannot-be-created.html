Workspace Cannot Be Created

I'm trying to install the new Titanium Studio 1.0 Preview on Windows 7 x64, however there are a number of issues:

  1. It tries to install to a weird directory by default - C:\Users\Justin\AppData\Local\Titanium Studio - it should default to C:\Program Files (x86)\Titanium Studio.

  2. There is a step that says "Select start menu folder", yet the contents of the window has you check the boxes of which file types to associate to the IDE.

  3. After installing and attempting to run the program, it errors: "Workspace Cannot Be Created - Could not launch the product because the specified workspace cannot be created. The specified workspace directory is either invalid or read-only." This is silly because I obviously don't have a workspace, I just installed the app for the first time…

Does it only work for Mac or something?

    This worked for me:

    1. Edit <WhereYouInstalled>\Titanium Studio\configuration\config.ini
    2. Find "osgi.instance.area.default=@user.home/My Documents/Titanium Studio Workspace"
    3. Provide the the full path to where you want the workspace created
      It could be something like this: "=c:\development\titanium\Titanium Studio Workspace"
    4. Fire up Titanium Studio

  • For Mac OSX:
    Right click on Titanium Studio.app > Show Package Content > Contents > MacOS
    Drap TitaniumStudio to Terminal and add args : -data "Your Workspace Path".
    It will look like : …………/TitaniumStudio -data "Your Workspace Path"
