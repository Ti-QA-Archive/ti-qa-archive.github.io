Swipe scrollable view

This is my view:

<Alloy>
    <Window id="index" class="container">
        <View class="top">
            <View class="pager active" />
            <View class="pager inactive" />
            <View class="pager inactive" />
        </View>
        <ScrollableView id="scroller" class="bottom">
            <View id="view1">
                <Label>hi!</Label>
            </View>
            <View id="view2">
                <Label>hello!</Label>
            </View>
            <View id="view3">
                <Label>good morning!</Label>
            </View>
        </ScrollableView>
    </Window>
</Alloy>

I'd like to react to the swipe event on the ScrollableView in order to change color of one of the pager Views, however on Android emulator the below code does not work:

$.scroller.addEventListener("swipe", function(e) {
    Ti.API.info(e.direction);
});

Swipe event seems to almost work for a Window, however the important part is "almost"… It actually does not work if I swipe left to right and right to left, but when I swipe from bottom left corner to top right corner, then the app logs "left", which is obviously not true, because "left" is the direction I started to swipe from. I tested this only under Android emulator, but my goal is to build for Android and iOS. Any help?

  • Hi Marek,

    Thanks for your concern.

    ScrollableView doesn't have any swipe event (check the reference link), but you can use it against Window.

    I've tested your issue and works fine for both android and iOS. Check my test case below:

    index.xml

    <Alloy>
    <Window id="index" class="container">
        <ScrollableView id="scroller" class="bottom">
            <View id="view1" backgroundColor="white">
                <Label color="red">hi!</Label>
            </View>
            <View id="view2" backgroundColor="green">
                <Label color="red">hello!</Label>
            </View>
            <View id="view3" backgroundColor="yellow">
                <Label color="red">good morning!</Label>
            </View>
        </ScrollableView>
    </Window>
</Alloy>

    index.js

    $.index.addEventListener("swipe", function(e) {
    Ti.API.info(e.direction);
});

$.index.open();

    Hope it'll help you to solve your problem.
