Fire titanium event from html offline

Hi all, I'm using Android Titanium, I have a webView built using html offline, I have to fill some fields with runtime values. In this html page, there are some img and, on click, I have to fire titanium event, so that I can go on with my procedure. How can I find the id of the object witch fired the event? Thanks a lot! BR Manuela

    When you fire an app level event, you can pass data with that event as the second argument in the call. So, in your click handler for any given element, you can pass the id of that element in the event fire. 

    el.onclick = function() {
    Ti.App.fireEvent('somecustomevent', el.id);
}
