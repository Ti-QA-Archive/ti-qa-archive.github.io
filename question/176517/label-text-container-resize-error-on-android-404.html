Label text container resize error on Android 4.0.4

Hi there!

I have an application (3.1.2.GA build) with the following problem:

There is a slider which change the label's font size (black text color, red background color).
When I increase the size, it resize the red container, but when I decrease it, the height doesn't change!
I tried setting the label height to a fixed size (100 dip), but it doesn't help…

This is the label:

var tesztText = Ti.UI.createLabel({
    color: '#000000',
    font: { fontSize: "20 dip", fontFamily: Ti.App.customFont },
    shadowColor: '#aaa',
    shadowOffset: {x:"5 dip", y:"5 dip"},
    text: 'test',
    backgroundColor: '#FF0000',
    textAlign: Ti.UI.TEXT_ALIGNMENT_CENTER,
    top: "20 dip",
    width: Ti.UI.SIZE,
    height: Ti.UI.SIZE,
    verticalAlign: Ti.UI.TEXT_VERTICAL_ALIGNMENT_TOP
});

And the slider's eventListener:

sizeSlider.addEventListener('change', function(e) {
    //tesztText.font = {fontSize: parseInt(e.value)};
    Fsize = parseInt(e.value) + "dip";
    tesztText.setFont({fontSize: Fsize, fontFamily: Fpicker.getSelectedRow(0).id});
    textadderTitle_label.text = tesztText.getHeight();
    tesztText.height = '100 dip';
});

It's annoying, that this problem doesn't exists on Android Simulator neither on an old (2.3.6) mobile…

I think, it's a kind of alignment problem inside the label… Or something like this…
