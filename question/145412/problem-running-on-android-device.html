Problem Running on Android Device

Hi everybody,

Running my App in the Emulator works reasonably well ( hehe ) but I wanted to test my App on an Android device aswell. I test on a Samsung Galaxy S2.
Some time ago everything worked fine for android device testing but lately I get an annoying error.

  • Thank you for the link. Unfortunately I get an error message saying "Unable to open log device '/dev/log/main': No such file or directory"
    Although this error is something I am not too happy about and would love to get rid of it I found the solution for my bigger problem not beeing able to run my apps on the device!

    The solution was rather easy as I had to use ExTweaks go to Misc, scroll down to ADB Binary Mode and switch it from "Normal ADB" to "Development ADB".

    Now it is working again. Thank you for your fast response Eduardo!
