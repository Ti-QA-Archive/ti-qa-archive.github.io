Using Android system icons from R.drawable

Is there any way to use Titanium.Android.R.drawable icons (like ic_menu_edit) in the UI, e.g. as icons for the Titanium.UI.Tab?

Assigning Titanium.Android.R.drawable.ic_menu_edit to Titanium.UI.Tab.icon generates exception at runtime (since icon is string and ic_menu_edit is int). Looking at this suggests that it should be fixed, but it still doesn't work.

There is a workaround but it supports only a couple of system icons.
