Window titleControl doesn't show if composite component

Using Titanium 2.0.1

setting titleControl of a Window doesn't seem to display itself for a composite component (View).

If I set titleControl to something simple like the following then it shows as expected

var titleButton =  Ti.UI.createButton({
            title:'Press Me'
        });
myWindow.setTitleControl(titleButton);

If I do something a little more detailed like this then nothing shows in the title.

var myTitleControl = Ti.UI.createView(
    {
         layout: 'horizontal'
     });

var titleButton1 =  Ti.UI.createButton({
            title:'Press Me 1'
        });
var titleButton2 =  Ti.UI.createButton({
            title:'Press Me 2'
        });

myTitleControl.add(titleButton1);
myTitleControl.add(titleButton2);

myWindow.setTitleControl(myTitleControl);

I'm new to Titanium so it's certainly possible I'm making a boneheaded mistake here.

This is all running inside of iOS 5.1 simulator if it makes a difference.
