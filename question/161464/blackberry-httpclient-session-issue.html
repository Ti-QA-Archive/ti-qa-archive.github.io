  • NDK 10.2.0.1155, testing with both a 10.2 and 10.1 emulator.
  • Titanium 3.2.0 GA
  • OSX 10.8.5

Pretty much just using the sample code out of the titanium docs.

Ti.API.info(JSON.stringify(this));

        var sCallback = successCallback;
