    You can use the JavaScript array splice function : http://www.w3schools.com/jsref/jsref_splice.asp

    So, you have two options: 

    data = data.splice(1,2); // this will remove index 1 and 2 ie: 'start at index=1 and remove two elements from the array

//or you can do this twice.. 
data = data.splice(1,1); 
data = data.splice(1,1); // now the original index 2 is now index 1

    hope this helps
