Drawing grid and writing letters

Hey all,
I'm currently working on a crossword app for iPad. I have tried two ways to achieved that and it seems that both have their speed limits. First, creating 20x20 grid of views is really slow on iPad 1. Creating transparent labels is somewhat fast enough but as soon as I specified a background color or border color it is becoming sluggish. I searched but I didn't find a way to draw directly on a single view as in a canvas.
So the other way is loading a webview and filling it with a canvas element. The problem is then drawing letters. I have to use a Ti.APP.fireEvent for each letter clicked and it's translating in a delay between the click and the display in the canvas. Also I wasn't able to preload the htlm file before opening the window and there's a another delay before the canvas is displayed.
The app is pretty simple and should be fast on any devices since it's only ~100 definitions (rows in a tableview), a grid and a keyboard. Is there any way to efficiently draw a grid and update its field (backgroundColor and labels)?
Best regards!
Olivier (English's not my first language, sorry for mistakes)!
