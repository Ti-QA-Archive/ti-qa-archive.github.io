    • The easiest way to update XCode is to open the Mac App Store and download/update it from there. This usually handles the removal and update process for you.
      If you want to make sure you can still test on 5.1, locate the Apple 5.1 sdk that currently installed in your Developer directory and copy it somewhere on your drive. After you have updated to 6.0, you can copy that directory back into the Developer location so that you have 5.1 and 6.0 to test with.
