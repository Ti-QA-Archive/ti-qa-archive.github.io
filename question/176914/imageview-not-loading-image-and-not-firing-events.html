ImageView not loading image and not firing events

I'm trying to load an image in an ImageView, with this code:

var windowsHome = Titanium.UI.createWindow({
   title: 'MyApp', 
   height: Titanium.UI.FULL,
   width: Titanium.UI.FULL
   backgroundColor: '#FFF'  
});

var headerImageView = Titanium.UI.createImageView({
   image:'header_generic.png',
   top: 0,
   left: 0,
   width: 320,
   height: 100
});

windowsHome.add(headerImageView);

var imageViewEventListener = function (e) {
Ti.API.info('I am an event of headerImageView');
};

headerImageView.addEventListener('error', imageViewEventListener);
headerImageView.addEventListener('load', imageViewEventListener);

windowsHome.open();

The image is not loading and no event is fired.

    • a local image should always load as long as

      a) It's in a supported format (.png, .jpg)

      b) you put it in the correct place

      In your example you have not mentioned where in your project the file header_generic.png is placed. Its usually better to provide an application root relative path to your image, so if your .js file in in the /Resources folder and your image file is as well, then make sure you do this

      var headerImageView = Titanium.UI.createImageView({
   image:'/header_generic.png',
   top: 0,
   left: 0,
   width: 320,
   height: 100
});

      note the leading / (slash)
