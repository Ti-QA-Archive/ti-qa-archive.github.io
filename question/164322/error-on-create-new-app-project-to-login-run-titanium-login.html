Error on create new App Project: "To login, run 'titanium login'"

Hi,

I am new to Titanium and just installed it to learn. I already install all updates and Android SDK, but when I try to create a new project, the follow error appears:
"To login, run 'titanium login'"

I am using Windows 7
Titanium build: 3.2.1
Android SDK: 4.4.2

Thank you!
