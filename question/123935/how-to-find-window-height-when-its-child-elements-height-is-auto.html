how to find window height when its child element's height is auto

Hi,

In my app, I have a window which has two views and both the views have their height as auto. Is it possible to find the height of the window in this case? If so, how?

There were many questions asked related to this, but none seem to have the solution.

Thanks in advance.

  • After everything is drawn, you can use the "size" property of most UI elements…

    myWinHeight = win.size.height

    Before it's drawn, you probably want to get the screen resolution and use that.

  • Most probably, you're using Android. On Android dimensions are calculated when window opens, so you need to add event listener to window "open" event:

    win.addEventListener('open', function() {
    Ti.API.info(win.size.height);
});
