Could not open the editor error in Ti Studio

Every time I reboot my mac then load up Ti Studio, opening any file results in it showing the error:

Could not open the editor: The editor class could not be instantiated. This usually indicates a missing no-arg constructor or that the editor's class name was mistyped in plugin.xml.

Closing and reopening the project does not fix the issue. Most of the time I can delete the project then import it again and that will make it work again, however sometimes that doesn't even work.

This is a really annoying bug, please fix it soon… I am using the latest release of Ti Studio and have tried the latest 1.7 and trunk builds.

  • This error occurs when Titanium is not shut down properly i.e. reboot, or kill the process.

    To fix this simply do the following:

    • Remove the project from Titanium
    • Close Titanium (don't kill it, use the normal Quit function)
    • Reopen Titanium
    • Import your project
