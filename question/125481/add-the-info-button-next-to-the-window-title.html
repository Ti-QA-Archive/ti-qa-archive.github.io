Add the Info Button next to the window Title

How do you add the info button next to the title of the window? I have tried to add it to the window and move it around. However, when I try to push it up to the title bar it is limited to the interior of the window.

  • Accepted Answer

    You can do this:

    var infoButton = Ti.UI.createButton({
    systemButton:Titanium.UI.iPhone.SystemButton.INFO_DARK // Or use INFO_LIGHT
});

win.setRightNavButton(infoButton); // Replace win with the name of the window object

var contactadd = Titanium.UI.createButton({
    systemButton:Titanium.UI.iPhone.SystemButton.CONTACT_ADD,

win.rightNavButton = contactadd;
win1.open();
