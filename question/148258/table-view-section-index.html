Table View Section index

Hi,

I have a table view with multiple sections.
I would like to know, how can i fetch the index of a section, when a row is clicked.

Suppose i click on the 2nd row of the 3rd Section, i should get 3.

Thanks
Abhishek

    It's pretty simple Abhishek,

    Just add custom property to the row object you are creating and in the click event fetch the same. Have a look to the following code snippet.

    function getRow(sectionIndex)
{
    var row = Ti.UI.createTableViewRow();
    row._sectionIndex = sectionIndex;

    return row;
}

....

tableView.addEventListener('click', function(e){
    alert(e.rowData._sectionIndex);
});
