  • Accepted Answer

    There is a bug in 1.7.1 and earlier that makes this impossible without creating a custom AndroidManifest.xml TIMOB-4261. It has been fixed but will not be officially released until 1.7.2.

    In the meantime, if you create a custom AndroidManifest.xml under <app_root>/platform/android/AndroidManifest.xml, and add the following to the root (first) activity, &lt;activity ... android:screenOrientation=&quot;portrait&quot; &#x2F;&gt;, see example:

    <activity
        android:name=".AppNameActivity"
        android:label="AppName"
        android:theme="@style/Theme.Titanium"
        android:configChanges="keyboardHidden|orientation"
        android:screenOrientation="portrait"
    >
        <intent-filter>
            <action android:name="android.intent.action.MAIN" />
            <category android:name="android.intent.category.LAUNCHER" />
        </intent-filter>
</activity>

    When the 1.7.2 is released, you should be able to delete your custom manifest and add the following to tiapp.xml:

    <manifest>
    <application>
        <activity android:name=".AppNameActivity" android:screenOrientation="portrait" />
    </application>
</manifest>

    • Try adding android:screenOrientation=&quot;portrait&quot; to each of the other activities in the AndroidManifest.xml.

              <activity
            android:name="org.appcelerator.titanium.TiActivity"
            android:configChanges="keyboardHidden|orientation"
            android:screenOrientation="portrait"
          />
        <activity 
            android:name="org.appcelerator.titanium.TiModalActivity"
            android:configChanges="keyboardHidden|orientation"
            android:theme="@android:style/Theme.Translucent.NoTitleBar.Fullscreen"
            android:screenOrientation="portrait"
        />
        <activity 
            android:name="ti.modules.titanium.ui.TiTabActivity"
            android:configChanges="keyboardHidden|orientation"
            android:screenOrientation="portrait"
        />
        <activity 
            android:name="ti.modules.titanium.ui.android.TiPreferencesActivity" 
            android:screenOrientation="portrait" />

        <service
            android:name="org.appcelerator.titanium.analytics.TiAnalyticsService"
            android:exported="false"
            android:screenOrientation="portrait"
        />
