  • Hi Maxime

    Are you trying to work out the current location of a specific script or the physical location on your machine of where the files are stored whilst being run from the server?

    There is a command getNativePath() you call from the file object that will tell you exactly where the file is physically located on your computer.

    var f = Ti.Filesystem.getFile('yourfile.ext');
Ti.API.info('getNativePath', f.getNativePath());

    If this does not help - please provide a bit more detail.
