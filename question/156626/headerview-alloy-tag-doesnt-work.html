HeaderView Alloy tag doesn't work

Simple example similar to: https://github.com/appcelerator/alloy/tree/master/test/apps/ui/proxy_properties/views

<Alloy>
    <Window>
        <TableView>
            <TableViewSection>
                <HeaderView>
                    <Label height="30dp" backgroundColor="#ff0">Section</Label>
                </HeaderView>
                <TableViewRow title="row 1"/>
                <TableViewRow title="row 2"/>
                <TableViewRow title="row 3"/>
            </TableViewSection>
        </TableView>
    </Window>
</Alloy>

The header doesnt appear only the rows

Android device 4.2.2
SDK 3.1.2
