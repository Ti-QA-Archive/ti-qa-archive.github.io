Hi there,

is there any way to get the request send from an remote url html form?

for example if i search on google inside my webview i want to get the request.

webView.evalJS(&#39;window.location.search&#39;); turns up empty, it seems.

    Hi Jan,

    you can try the "beforeload" event listener, it shows you every request the webView will try to load

    webView.addEventListener('beforeload', function(e) {
        Ti.API.info("beforeload: "+JSON.stringify(e));
});
