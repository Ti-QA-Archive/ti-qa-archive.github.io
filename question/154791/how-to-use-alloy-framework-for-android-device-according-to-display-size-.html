  • You can use the form factor device queries to make a distinction between handheld and tablet: http://docs.appcelerator.com/titanium/latest/#!/guide/Alloy_Styles_and_Themes-section-35621526_AlloyStylesandThemes-Platform-SpecificStyles

    Any more granular than that and you'll need to handle it programmatically in a controller. More granular device queries based on device size and density have not yet been implemented in alloy.
