ScrollView scale bounce

how to disable scale bounce?
i got this scrollView:

var wrapper = Ti.UI.createScrollView({
    disableBounce: true,
    showHorizontalScrollIndicator: true,
    showVerticalScrollIndicator: true,
    contentWidth: 1000,
    contentHeight: 728,
    contentOffset: {
        x: 0,
        y: 0
    },
    zoomScale: 0.5,
    maxZoomScale: 1,
    minZoomScale: 0.5
});

but i am still able to scale over max/min zoom level and on release view ll animate to set max/min zoom scale.
SDK 1.7
iOS 4.2

    I guess the disableBounce sets verticalBounce and horizontalBounce to 0 false, but not the bounces to 0/false.

  • Found the answer for anyone needing this, i changed the TiUIScrollView.m inside the mobileSDK version im using,

    it has a function containing:

    -(void)setDisableBounce_:(id)value
{
    [[self scrollView] setBounces:![TiUtils boolValue:value]];
}

    copy this and make another function like this:

    -(void)setDisableBounceZoom_:(id)value
{
    [[self scrollView] setBouncesZoom:![TiUtils boolValue:value]];
}
    — commented March 20th 2013 by Maikel Tunnissen
