I've got a textarea in a scrollview so that when the keyboard displays the text area is slid up so you can see what you are typing, otherwise it would be covered, Now when the keyboard loses focus the text area stays in it's slid up position. Where as previously it would return to it's original location is there something more I have to do now to return the scroll view to it's home?

    what I usually do is on the event blur of the text field(s), call scrollTo(0,0) on the scrollView to get it reset

    tf.addEventListener('blur',function(){
   theScrollView.scrollTo(0,0);
});
