How to add an imageView on a mapview (with top, bottom, left, right property)?

I need to add a watermark on a mapview so that I can convert the mapview along with the watermark to an image using toImage().

No matter how I change the property of top, bottom, left, right property, the imageView always appear at the top left corner which is {top:0, left:0}.

I know I can set the imageView properly by adding it to the window by win.add(watermark) to get it displayed nicely. But I need to add the imageView on the mapview in order to convert it to an image later.

The same code works on ios, but not android for some reason.

var mapview = Titanium.Map.createView({
    mapType: Titanium.Map.STANDARD_TYPE,
    region: {latitude:33.74511, longitude:-84.38993, 
            latitudeDelta:0.01, longitudeDelta:0.01}

});

win.add(mapview);

var watermark=Ti.UI.createImageView({
    right:10,
    bottom:10,
    opacity:0.5,
    height:104,
    width:100,
    image:'watermark.png'            
});
mapview.add(watermark);

    Hi,

    add the mapView to a container (other view) and add the imageView to the container:

    var container = Ti.API.createView();

var mapview = Titanium.Map.createView({
    mapType: Titanium.Map.STANDARD_TYPE,
    region: {latitude:33.74511, longitude:-84.38993, 
            latitudeDelta:0.01, longitudeDelta:0.01}

});
var watermark=Ti.UI.createImageView({
    right:10,
    bottom:10,
    opacity:0.5,
    height:104,
    width:100,
    image:'watermark.png'           
});
container.add(mapview);
container.add(watermark);
win.add(container);

    Rainer
