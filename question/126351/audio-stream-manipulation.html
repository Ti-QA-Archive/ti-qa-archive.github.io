Audio stream manipulation?

Hello, I'm playing around with Titanium Mobile and will probably use it for my next project.

My application will have to detect in real-time if sounds of certain frequencies get picked up by the microphone. The basic idea would be to pick up the audio stream from the microphone, manipulate it with a band-pass filter, and check if its amplitude is over a certain treshold, then play a sound or fire some other event if that's true.

Unfortunately, I don't know where to find the Titanium functions for sound manipulation in real time. I only found the Ti.Media class, that contains functions and classes to record audio files, but not to analyze them in real time.

What is the best way of doing this in Titanium?
