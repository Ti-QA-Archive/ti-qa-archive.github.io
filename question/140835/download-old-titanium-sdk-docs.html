  • Found a solution: use wget and a terminal emulator:

    wget --no-parent -r -l 2 -p -k http://developer.appcelerator.com/apidoc/mobile/1.7.5/

    You'll get some folders with static HTML files accesible offline.

    • Hi David

      OK, did not realise you meant for off-line use.

      Appcelerator are very good at providing documentation for old version - as you can see.

      However at some point they will need to remove older versions - as they simply stop being relevant, for example some older versions of the SDK CANNOT be used to compile an app to go into the stores.

      So even if they provided these details way into the future they would be off no practical use as there would be nothing you could do with the app besides update it to a newer version.

      That said I do get your point, and I have to guess that leaving old docs up on-line - costs them nothing (practically speaking) so they should survive.
