Location autocomplete similar to google maps

Hi, I am looking to add location autocomplete functionality to my app. What I would like is to use a standard textbox and when the user for example types "Chic" a list of suggestions should pop up with Chicago first.

Anyone has an idea how this could be implemented?

I checked google maps javascript api and it has this functionality but it seems it only works in a webpage and is only allowed when showing a google map and I don't intend to show any map.

  • It's been awhile since this question was asked but this may help anyone else who is still looking for a solution to a similar problem.
    Try
    http://maps.googleapis.com/maps/api/geocode/json?address=Chicago&sensor=true

    change json in the url to xml if you want the results in xml.

    Then just use httpclientrequest or tiquery ajax method to get your data.
