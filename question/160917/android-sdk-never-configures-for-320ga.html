Android SDK Never Configures for 3.2.0.GA

Recently upgraded Titanium Studio to 3.2.0.GA (For Mac) and I've now lost all ability to debug or run for android. Under run configuration it states Please select an Android API version. On the Dashboard under Configure Native SDKs it list android as not installed (red x). Previously was working fine on RC 3.2.0.v20131218114845 until I the upgraded today. Now it's just decided it can't find the android sdk (despite the path its pointing to containing the android SDK).

I have tried running the android configuration multiple times and tried with many different Android SDK locations. After each configuration attempt it says Latest Version Installed: Unknown! Verified that the SDKs had API 19,18, and 10 installed. Tried removing and reinstalling Studio to no avail.

On the bright side, the CLI still works so development at least is not halted. Any thoughts on why its not finding the Android SDK? Gonna try switching to the Beta in the meantime.
