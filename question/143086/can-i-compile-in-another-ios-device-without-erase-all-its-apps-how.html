  • Hi tabe, there is simple solution around it, you create a build for say iPad simulator and open that build using X-Code. now no need to sync your boss iPad with iTunes and just add the developer profile in your boss iPad then deploy app using X-Code directly.

    • Once you run it for iPad simulator, it automatically create a build folder and use that X-Code project.

      — commented October 4th 2012 by Ashish Nigam

    • Hi tabe, ok so here are the things you do,

      1: you run your project for iPad simulator from the run menu.

      2: Then go to your project directory and there you will find a build folder.

      3: go in that and you will see a X-Code project there.

      4: Open that project using X-Code and deploy it into your boss iPad.

      is this solved your problem?
