    • Studio not seeing the 3.4.0 SDK version is a separate issue. The CLI supports SDK install locations in two places. Studio doesn't yet support both locations (fix is in the works, though I'm not sure if that will be included in the final 3.4.0 release version or next version).

      Here's the output from ti sdk select for me:

      Installed SDKs:
   1)  3.4.0.v20140910235714  /Users/tpoulsen/Library/Application Support/Titanium/mobilesdk/osx/3.4.0.v20140910235714
   2)  3.4.0 [selected]       /Users/tpoulsen/Library/Application Support/Titanium/mobilesdk/osx/3.4.0
   3)  3.3.0.GA               /Users/tpoulsen/Library/Application Support/Titanium/mobilesdk/osx/3.3.0.GA
   4)  3.2.2.GA               /Library/Application Support/Titanium/mobilesdk/osx/3.2.2.GA

      Notice how some are in my Library folder and others are in the root Library folder. Check which one you're seeing in Studio and install the new SDKs there. You can configure where they go with the ti setup command.
