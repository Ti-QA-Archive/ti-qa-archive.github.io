  • Oh, I have idea as to what is going on. When we run the security command, if the output is larger than 1024 x 1024 (1,048,576), it essentially does nothing. Try this at the command line:

    TMP=`security dump-keychain`; echo ${#TMP}

    If the number if outputs is greater than 1,048,576, then we found our problem.

    FWIW, my keychain dump is about 175,498 bytes, so when I coded it, I figured at least 5x bigger buffer should pretty much do the job.
