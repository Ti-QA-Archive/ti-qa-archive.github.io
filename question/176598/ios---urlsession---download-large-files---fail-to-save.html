iOS - URLSession - download large files - fail to save

My code works with small files ~ xx mb.

If the files are around 1GB, the download will be finished without a problem, but the data can't be correctly saved to a file.

Here is my code:

Ti.App.iOS.addEventListener('downloadcompleted',function(e){
    var file = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory,"test.mp4");
    file.write(e.data) // e.data is a blob
    file.remoteBackup = false;
});

After download completed, I go try to open test.mp4, the file isn't there!

Looks like some issue when writing large blob to file. Any suggestions? Thanks.

SDK 3.3.0 iOS7
