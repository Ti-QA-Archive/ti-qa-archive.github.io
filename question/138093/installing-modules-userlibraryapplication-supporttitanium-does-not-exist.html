  • Just an example:

    I have a module called com.inzori.flurrymodule version 3.1.0 , so, I unzip the module files so I get this folder structure:

    Mac / Library / Application Support / Titanium / modules / iphone / com.inzori.flurrymodule / 3.1.0 /

    Into this folder (3.1.0) you should see:

    documentation folder
example folder
libcom.inzori.flurrymodule.a
LICENSE
manifest
module.xcconfig

    Then in my Ti project I place this into tiapp.xml:

    <modules>
    <module platform="iphone" version="3.1.0">com.inzori.flurrymodule</module>
</modules>

    Clean and build.

    This should work, if it doesn't you may have to reinstall Ti.

    You can download this module for free to test here:

    Flurry and AppCircle module

    • hi Arun,
      May be late reply but may help you

      In OSX /Library/Application Support/Titanium or ~/Library/Application Support/Titanium

      In Windows 7 C:\Users\username\AppData\Roaming (or C:\ProgramData\Titanium on Titanium Studio 1.0.1 and earlier)

      In Windows XP C:\Documents and Settings\username\Application Data (or C:\Documents and Settings\All Users\Application Data\Titanium on Titanium Studio 1.0.1 and earlier)

      In Linux ~/.titanium
