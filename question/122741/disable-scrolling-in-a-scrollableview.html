Disable scrolling in a ScrollableView

I have multiple views within a ScrollableView that have vertical drag functionality. As such, I want to disable horizontal scrolling of the ScrollableView whilst the vertical scrolling of the visible view is being carried out.

I see in the Apple documentation at http://developer.apple.com/library/ios/, the property scrollEnabled is available for a ScrollView, however I dont't think this is available in the Appcelerator API.

Can anyone suggest a way to lock the ScrollableView?

BTW. I have thought of setting touchEnabled=false, however this then results in no events being fired to the child view so this is not a solution.

Thanks for your advice.

  • Hey!

    I know this is an old post but as I run into it looking for an answer i decided to post a solution in case helps someone.

    It seams to be fixed in the new 2.0… Ti SDK release. check this out: https://jira.appcelerator.org/browse/TIMOB-5811?page=com.atlassian.jira.plugin.system.issuetabpanels:changehistory-tabpanel

    I've tested this out using iOS and it's working ;)

    var scrollableView = Ti.UI.createScrollableView({
  views:[new generateVerifyView(),new generateUserView(),new generateAddrView()],
  showPagingControl : false,
  scrollingEnabled : false
});
