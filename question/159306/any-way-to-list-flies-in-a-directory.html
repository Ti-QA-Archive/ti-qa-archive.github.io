Any way to list flies in a directory?

Hi i would like to be able to dump a lot of images in a directory and have some code that scans the directory and lists all the filenames found putting them into an object i can use later.

Thanks in advance

  • Hi

    The documentation details all of the file handling abilities you will need and loads of other stuff as well.

    • Ti.Filesystem.File.getDirectoryListing()

    This returns an array of strings, each of which represents a file in that location.

    Simply use the file object to identify a folder and then call this method, very simple example;

    var path = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory);
var dir = path.getDirectoryListing();
var intFile = 0, intFiles = dir.length;
for (intFile = 0; intFile < intFiles; intFile  = intFile + 1) {
    Ti.API.info(dir[intFile]);
}

    Simply provide the correct path to start with based on where you are looking. Remember than resource directory is read-only (just in case you were planning on doing anything in there other than listing).
