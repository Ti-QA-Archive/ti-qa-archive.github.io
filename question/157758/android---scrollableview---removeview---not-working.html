  • Just FYI, if anyone runs into the above issue.. was able to find a solution:

    On Android: It appears, you can't really remove the views when it is visible on screen. So scrollToView(0) and then get rid of the other views.

    On iOS: Remove.. and then scrollToView(0).

    Unfortunately same code doesn't work for both..have to go the if-then-else… for OS NAME.. not what I wanted to do, but hey.. it works.

    Thanks

    For Android:

    
homeButton.addEventListener('click', function(e) {
    contentScrollableView.scrollToView(0);

    var currentViews = contentScrollableView.views;

    for(var i = currentViews.length - 1; i > 0; i--) {
        contentScrollableView.removeView(currentViews[i]);
    }
});
