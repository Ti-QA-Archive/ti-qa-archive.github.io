  • Ok, lets walk through this, try to separate studio from titanium problems.

    Open a terminal, outside of studio.

    Lets see your java version:

    java -version

    Lets see your node version:

    node -v

    Lets see your ti version:

    ti -v

    If those both run, lets see your config (remove your user.email entry before posting):

    ti config
