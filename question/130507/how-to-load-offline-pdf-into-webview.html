How to load offline pdf into webView?

I am trying to load offline pdf file into webView.
I have saved offline pdf in my local db, but I am not able to load it into webView.
My code was like below.

myWebView = Titanium.UI.createWebView({
top: 0,
bottom:0
});
myWebView.url = offlinePDFObject;

It gave me error message below:
Invalid type passed to function, expected: NSString, was TiBlob in -[YiUIWebView setURL_:].

Is there any other way to load the offline pdf file into any view?

    Hi! i just placed test.pdf file into my ApplicationDataDirectory. in my case here it is: /Volumes/HD/Artanis/Library/Application Support/iPhone Simulator/5.0/Applications/8B6B689B-3F44-4FD1-90C7-C0C7439D43F7/Documents/test.pdf

    and this is works for me: 

    var file = Ti.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory+'test.pdf');

var webview = Ti.UI.createWebView({    
    data: file.read()
});

    be sure, you have pdf file extension!

    this error ("Frame load interrupted"), because there wasnt any file extension.

    good luck :)

  • //my test.pdf in resource directory at first

//read and write test pdf from resourcedir to application data directory
var newFile = Titanium.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory,'test.pdf');
newFile.write(f.read());

Ti.API.info(newFile);

//get from applicationdatadirectory
var file = Ti.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory,'test.pdf');

var webview = Ti.UI.createWebView({    
    data: file.read()
});
