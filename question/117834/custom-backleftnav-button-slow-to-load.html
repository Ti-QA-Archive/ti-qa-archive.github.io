Custom back/leftNav button slow to load?

I've customized the back button on a window of my app in order to make its color consistent with my custom navbar. My problem is that it takes a moment to load and, therefore shows the native back button for a second before it loads my custom back button. Is there anything I can do to prevent this?

<pre>
var backbutton = Titanium.UI.createButton({
backgroundImage:'offers_button.png',
width:48,
height:27
});
backbutton.addEventListener('click', function() {
win.close();
});
win.leftNavButton = backbutton;
</pre>

  • After looking back over my code for this, I came up with a solution. Instead of loading the custom back button in the separate window code file, I added the back button to the window at window-creation time. This causes the back button to already exist when the window opens.

    Now I never see the native back button style.
