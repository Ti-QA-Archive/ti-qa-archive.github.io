  • Hi. You need to create a web service in the form of a PHP script that receives the data via POST or GET and inserts the data to your DB, just like handling an HTML form. Once you've done that, you have to make an HTTP request from your mobile App. This post can help: http://developer.appcelerator.com/blog/2013/03/consuming-data-from-web-services.html
