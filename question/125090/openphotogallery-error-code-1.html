openPhotoGallery error code 1

What does error code 1 mean on openPhotoGallery? The following code:


    Titanium.Media.openPhotoGallery({
          mediaTypes: [Titanium.Media.MEDIA_TYPE_PHOTO],
        success: function(e)
           {
                Titanium.API.info("gallery success" );
            },
            cancel:function(){},
        error:function(error)
        {
            Titanium.API.info("gallery error" +  JSON.stringify(error ) );
        },
        allowEditing:false
     });

in my multi-tab app, where the gallery is launched from an option dialog, prints out: 

INFO: gallery error{"type":"error","code":1,"source":{},"success":false}

Launching the gallery directly (without the option dialog) works fine, however. I'm trying to reproduce the error outside my app, recreating the windowing set-up, but so far have not been able to.

It would be helpful to know: What's error code 1, in this situation? The API docs don't give much insight.
