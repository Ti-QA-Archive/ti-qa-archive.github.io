Gzipped responses on Android

Is it possible to accept gzipped responses on Android? I believe it is possible to do this on the iPhone.

<http://jira.appcelerator.org/browse/TIMOB-2698> was intended to address this issue - but I don't see any progress there. Does anybody know of any workarounds in the meantime?

Thanks!

    The ticket for this is still open, and currently I don't believe there is really a workaround per se for the encoding headers. However, it looks like Android 'Intents' might be worth looking at for opening gzip files

    http://indyvision.net/2010/03/android-using-intents-open-files/

    Also, here's some info on java.util.zip which supports .zip and .gz

    http://developer.android.com/reference/java/util/zip/package-summary.html
