  • Ok here we go, i can't code it for you due in those days i have no times but:

    1) Create your base of datas.
    // fill it with the movie url. ( remeber to put all the movie in the source directory ex Resources/movie/

    var mymovietoplay=['nameofthefiles1','nameofthefiles2'……]

    // take the counter to move inside the array
    var mycounter =0 ;

    // use the media player

    Titanium.Media.createVideoPlayer with the url to play
    I suggest you to see/check the examples here:

    ( eg. var playcenter = Titanium.Media.createVideoPlayer({url:mymovietoplay[mycounter]};)

    2) use the event completo to go ahead playing

    playcenter.addEventListener('complete',function (e){
    if(mycounter<=mymovietoplay.length-1){
    playcenter.url=mymovietoplay[mycounter]++;
    playcenter.play();

    }

    });

    it should be something like that, i suggest you to install/see kitchensink example here

    http://developer.appcelerator.com/guides/en/kitchensink.html
