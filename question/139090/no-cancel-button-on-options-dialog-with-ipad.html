No Cancel Button On Options Dialog With iPad

Hey! So, I have this code here set as an options dialog with a cancel button at the second index. This button appears on iPhone, but it just shows the normal buttons on iPad. Is this normal? Is there a way to add a cancel button? Thanks!

var homeDialogOptions = {
            title : 'Set Current Page As Home?',
            cancel : 1,
            options : ['Set Homepage', 'Cancel'],
        };
var homeDialog = Ti.UI.createOptionDialog(homeDialogOptions);

  • This is normal. On iPad dialogs the option you mark as 'cancel' does not show up. Simply click outside of the dialog to dismiss the dialog and the cancel option is triggered.
