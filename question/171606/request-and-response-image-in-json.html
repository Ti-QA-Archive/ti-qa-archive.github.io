    A few things of note;

    • You do not pass images to a remote API inside JSON (yes you can technically but this is weird) you pass it in the same way as form data would on a web page
    • You can store an image into a database (again weird) however it is more common to save the file locally and store the file name into the database
    • Displaying a remote image stored as a file (not as an object inside a db) is simply calling a url and using that in the image parameter of the imageView.
