Scrollviews Invisible

Greetings.

I am trying to add a scrollview to my current window, but it never appears after compiling for iPhone.

Here's the code:


var win = Titanium.UI.currentWindow;

var scrollView = Titanium.UI.createScrollView({
  contentWidth:'auto',
  contentHeight:'auto',
  top:0,
  showVerticalScrollIndicator:true,
  showHorizontalScrollIndicator:true
});

var view = Ti.UI.createView({
  backgroundColor:'#336699',
  borderRadius:10,
  width:'auto',
  height:'auto',
  top:10
});

var testLabel = Titanium.UI.createLabel({
  text:"FUNNY",
  height:'auto',
  width:'auto',            
  top:10,  
  left:5,
  font:{fontSize:14}      
});   

view.add(testLabel);

scrollView.add(view);

win.add(scrollView);

  • According to the guide: If your scrollable region fits within the size of the content area the scroll view will not scroll.

    You need to set the view height to make it visible

    var view = Ti.UI.createView({
    backgroundColor : '#336699',
    borderRadius : 10,
    width : '100%',
    height : 1000,
    top : 10
});

  • Minh is right but when you do not know the exact size of the view you can set that by using.

    when you use scrollview you should set the contentHeight or contentWidth as per your requirement because without setting this you can not scroll the view and one more thing you can scroll the view when your scrollview's content is more then your scrollview's height

    var view = Ti.UI.createView({
  backgroundColor:'#336699',
  borderRadius:10,
  width:'auto',
  height:'auto',
  top:10
});

var testLabel = Titanium.UI.createLabel({
  text:"FUNNY",
  height:'auto',
  width:'auto',            
  top:10,  
  left:5,
  font:{fontSize:14}      
}); 

view.add(testLabel);

scrollView.add(view);
view.height = testLabel.top + testLabel.toImage().height;

//or

view.height = testLabel.top + testLabel.size.height;

scrollView.contentHeight = view.height; 

win.add(scrollView);
