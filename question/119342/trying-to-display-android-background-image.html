  • var win = Titanium.UI.currentWindow;
win.backgroundImage = "/img/Background.png";

    It looks like you are not using the right variable. I think you meant to do this: 

    var win1 = Titanium.UI.createWindow({  
    backgroundColor:'#fff',
    backgroundImage:imgPath,
    fullscreen: true,
    navBarHidden: true,
    tabBarHidden: true 
});

    Additionally, I would get rid of the space in the file name.
