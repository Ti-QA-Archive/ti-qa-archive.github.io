How to limit the zoom on Google Maps v2 and iOS maps?

Hi All,

I'm working with maps on both iOS and Android. I've gone through the docs and I don't see anything that appears to address this need. I have as a goal to display locations in a way that supplies strictly high level information- nothing too specific. With that in mind, I'd like to prevent the user from zooming in with the map and getting too detailed of an idea of where the annotations are. Can I prevent the zoom in this way? If not then I'll go with my plan B and just turn off annotations if they zoom in too far, but it'd be nice not to have to game the system and make it appear almost glitch-like.

Any thoughts? Thanks!

  • You can:

    var MINDELTA=0.01;
myMap,addEventListener('regionchanged',function(_e){
    if (_r.latitudeDelta< MINDELTA)  {
        var region = _e.source.getRegion();
        region.latitudeDelta = MINDELTA;    
        _e.source.setRegion(region);
    }
})
