New iPhone Project Will not Compile and Run

I am getting the following error on a clean build for an iPhone mobile app.

error: expected expression before '@' token

error: missing terminating " character

The only thing I had done was fill out the tiapp.xml. I realized that I had used a Multi-line description and this was breaking the main.m file in the build.

Solution! Make your tiapp.xml description only one line with no carriage returns.
