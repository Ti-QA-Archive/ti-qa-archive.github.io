Adding a function to a view

I want to incapsulate the logic for populating my view inside the views itself.
However when I do the following:

// function to create my view
var view = Ti.UI.createView({width:100});
// Add controls to view....
view.label1 = Ti.UI.createLabel(text:'test');
view.add(view.label1);
// Load function - will be gone!
view.load = function(data)
{
    view.label1 = data.text;
};

return view;

My function is not there, and I can't see it in the debugger. I have been using this method to attach metadata to views, so expected functions to work the same way.

Any ideas?

  • a view doesn't have the property load. You can assign parameters (such as 'id' or whatever) to an object, but you can't assign a function.

  • Because the way the js is converted to native you cannot do this. Maybe you could try to encapsulate all in an object

    my_view = {
    the_view:Ti.UI.createView(),
    load: function() {}
};
