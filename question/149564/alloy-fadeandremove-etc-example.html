Alloy FadeAndRemove etc example

I'm a newcomer to alloy and titanium, working on a proof of concept. I've been powering through the basics with a fair amount of success, but I've been having trouble understanding how to put together a multi-page application with animated page transitions. Here's a simple example: UI calls for a new page of type X with data Y, the main window slides (fades, whatever) out the old page and slides in the new page. It looks like I need something like the fadeAndRemove builtin, but there no example in the docs to illustrate how to use it. Can anyone provide a simple example of this type of functionality, either using fadeAndRemove or something similar?

PS: This would be a great addition to one of the test apps. Or an alloy kitchen sink.
