  • hello:
    i'm using ubuntu 14.14 and titanium 3.4.0.201409261227, build jenkins-titanium-rcpmaster-197 from 26 september 2014

    In linux, at least at November 11th of 2014, stills remains an issue with redtty execute permisions.
    You have to run trought directories tree to where you install TitaniumStudio, look for plugins/com.aptana.terminal_3.0.0.1407781207/os/linux/$arquitecture(x86 or x86_64)/redtty and get execute permision to 'redtty' using 'chmod +x redtty'.

    Obviously, you have to change 'com.aptana.terminal_3.0.0.1407781207' to the your actual version of com.aptana.terminal_x.x.x.xxxxxxxxxx.

    Hope this should be usefull for anyone else.
