within a TableView, How to update a row with new content and restore the original content?

I have a Ti.UI.TableView within a Ti.UI.Window. Each row in the table has 80 pixel height and, 3 images and 3 label elements to make it fancy. Here's the thing I'd like to accomplish:

1) When a user clicks on a row in Ti.UI.TableView, the row itself will be switching to the new content with a nice sliding-in effect.

2) And, if the user clicks on the row again, the row will be switching back to the original content of the row. (This is where I get stuck)

[ orig. content ] -> [ some buttons ] -> [ orig. content ] -> …

I know I can do the 1st question with Ti.UI.TableView.updateRow method. But, once updated, how can I restore the original content back?

I saw some apps doing this type of thing to bring up an inline button menu when a user clicks on a row cell. But, I just can't figure out how to get back the original content.

Can someone enlighten me?

    Try this:

    When click on a row save the row content (tableView.row[e.index]) in a var, for example var oldRow = tableView.row[e.index];

    Then create a new row with your new content and set a custom value of this row named restore, restore: oldRow.

    In this way you have the old content in the restore value of your new row.
    When you will come back just update the row with the e.row.restore content.

    Try, I'm not sure if you can assign an object to a custom property :D
