  • Hi James,

    I just tested it now and it worked fine. This is what I did:

    • Unzipped the PayPalModule.zip file.
    • Copied the modules&#x2F;iphone&#x2F;ti.paypal directory to the modules&#x2F;iphone directory of the TiShadow app
    • Added 
      <module platform="iphone" version="1.4">ti.paypal</module>
      to the tiapp.xml file of the TiShadow app.

    • Did a clean build with Ti CLI - i.e. 

      titanium clean && titanium build -p iphone

    • Modified the app.js file of the PayPalModule example app so that all the urls were slash leading.

    • Entered tishadow run
