  • Unless you specifically need to use RedLaser (and pay their fees), you can start by trying out the excellent 'TiBar' module, which will return the codes for all barcodes (and QRCodes etc) scanned, which you would then have to use another API to look up 'what' the product is.

    http://developer.appcelerator.com/question/50531/tibar-zbar-integration-module-for-titanium-mobile

    It works great.
