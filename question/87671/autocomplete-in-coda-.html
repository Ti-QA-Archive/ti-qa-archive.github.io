    Today i've started on a code highlighting template.
    You can download the initial version from RapidShare

    Installation instructions:

    1. Unzip the zip contents, it should contain a 'Javascript-Titanium.mode' directory.
    2. Right click on the Coda application icon -> 'Show Package Contents'.
    3. Navigate to 'Contents' -> 'Resources' -> 'Modes'
    4. Copy the 'Javascript-Titanium.mode' directory to 'Modes'

    Suggestions are welcome :-)
