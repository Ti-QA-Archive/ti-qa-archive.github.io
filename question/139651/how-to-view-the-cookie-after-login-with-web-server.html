How To View The Cookie After Login with web-server

I am Using Titanium.Network.HTTPClient To Login with web-server.

After the successful login, How can I view the saved Cookie?

Here is the code I am using

var loginReq = Titanium.Network.createHTTPClient();
loginReq.setTimeout(10000);
loginReq.open("POST","https://xxxx.com/Account/Login");

var params = {
        uname : loginVal,
        password  : passVal
};

loginReq.onload = function(){
        var response = this.responseText;
        alert(response);
        Ti.API.info("response = "+response);
        App.view.MainWindow.show();
};

loginReq.onerror = function(e){
        App.config.ShowAlert(App.config.TimeOutMessage);
};

loginReq.send(JSON.stringify(params));

Thanks in Advance
