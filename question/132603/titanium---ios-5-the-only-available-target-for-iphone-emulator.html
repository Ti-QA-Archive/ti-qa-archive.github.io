Titanium - IOS 5 the only available target for iPhone emulator

I'm trying to create a run configuration for IOS 4. The options only show IOS 5. I've installed Xcode and it shows targets for IOS 3 and 4 in the Xcode IDE. I only show to have Titanium SDK 1.8.1 installed. How to I develop and test IOS apps for IOS 4 and IOS 3?

  • I guess your Xcode version is 4.2. The default support version of Xcode4.2 is iOS4.3 or later. So if you can not select iOS4.3 when you run on iOS device in Titanium Studio, probably 4.3 simulator is not installed on your system. You can install 4.3 simulator from Xcode-Preference-Downloads.
