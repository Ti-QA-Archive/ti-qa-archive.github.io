When I do publish->package mobile web my code is not minified. I get .js and .uncompressed.js files but they are exactly the same.

How do I get my packaged code to be minified?
Can this be because I use ISO-Latin characters?

Using SDK v2.1.*

  • Hi Dario! There is no log file, but you can manually run a build from the command line. You just need to run the builder.py Python script. The script takes two arguments: the path to your project and the deploy type (production or development).

    On my Mac I run the follow:

    python /Library/Application\ Support/Titanium/mobilesdk/osx/<VERSION>/mobileweb/builder.py /Users/chris/projects/myproject production

    Manually run builder.py for your project and lets see what it says.

  • Hi Dario, I fixed things so now you can do production builds! All you need to do is download the latest CI build from our "master" branch.

    Go to http://builds.appcelerator.com.s3.amazonaws.com/, from the drop down under Mobile Builds, select "master", then download and install the latest build for your platform.

    Remember to select the correct SDK in the tiapp.xml. You should then be able to create production minified builds. :)
