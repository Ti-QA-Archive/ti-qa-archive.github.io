Custom iOS module for AdBuddiz

I am trying to build a simple iOS module to wrap around the AdBuddiz SDK.
I thought (maybe a bit naively) that all I needed to do was to create a module with one single function to call the AdBuddiz show ad function.

From the documentation of AdBuddiz (https://publishers.adbuddiz.com/portal/support/obj_c):

#import <AdBuddiz/AdBuddiz.h>
[AdBuddiz showAd];

What I've done:

  • Create an iOS module
  • Added the framworks as needed by AdBuddiz (from the documentation above)
  • Added the AdBuddiz.h header on top of my module .m
  • In the startup function of my module I've added the method calls to set publisher id and cache

  • Added a new function like this:

    -(void)showAd
{
       [AdBuddiz showAd];
}

  • Built the module and added it to my app

  • Calling the module like so:
    var adbuddiz = require('com.thunapps.adbuddiz');
adbuddiz.showAd();

This crashes the simulator.

My guess is that it is not as simple as I think. Does anybody know how I can solve this, or at least point me in the right direction.
