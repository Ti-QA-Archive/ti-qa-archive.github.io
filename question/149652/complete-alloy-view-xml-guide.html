    The complete TSS guide is right below the XML guide: http://docs.appcelerator.com/titanium/latest/#!/guide/Alloy_Styles_and_Themes

    As far as a naming convention, All Titanium APIs that correspond to a UI component map directly to an XML element. The Ti.UI namespace is used by default, but you can use the "ns" attribute to specify a different namespace.. For example:

    Ti.UI.Label --> <Label/>
Ti.Map.View --> <View ns="Ti.Map"/>
Ti.UI.ImageView --> <ImageView/>
