android:home event listener not working

Running Titanium: 2.1.2
Application Type: mobile
Platform & Version: Android 4.1
Device: Android device

I am trying to add an event listener for the android:home button. I have code that is similar to the following:

window.addEventListener('android:home', function(e) {
    alert("Home button has been pressed")
});

However, this code is not being triggered when the home button is being pressed. I can catch the "back" button being pressed, though, using the following code:

window.addEventListener('android:back', function(e) {
    alert("Back button has been pressed")
});

Why doesn't my code work for catching the home button being pressed?

  • Hi Kodey,

    android:home event listener is not working because Titanium can not catch that event.

    If you look at the Titanium.UI.Window Android Event Docs you will find only 6 android events.

    • android:back
    • android:camera
    • android:focus
    • android:search
    • android:voldown
    • android:volup.

    As of Android navigation philosophy the back button should be used to navigate inside your app.

    Home button brings users to home screen and also should be the last chance for a user to exit an application.

    Greetings,

    Matthias
