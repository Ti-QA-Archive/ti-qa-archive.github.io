    It looks like you may be running an incompatible version of the java SDK. If you check the compatibility chart here, you will see that version 6 aka 1.6 rev 10 is supported, not 7. Your log above shows you have 1.7 rev 40 installed. "Java Development Kit Version = 1.7.0_40 Java Home = c:\Program Files (x86)\Java\jdk1.7.0_40”
