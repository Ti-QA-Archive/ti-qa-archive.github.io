  • We use the web server to do all the PUSH notifications. So for example we currently use ACS for push however it is not work at the moment. We then use a different APNS provider (Google PHP APNS) and make the switch on the server. All existing apps out there still work, and don't care what APNS we use, as long as the API we developed on the server is the same.
