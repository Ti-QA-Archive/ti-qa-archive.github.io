  • I think with this question you need to provide some code, an idea of what device your aiming at..

    But while you get that..

    I would create the views within the window.. Only 3 .. then set their left position to be off screen…. i.e. view1.left = 900;

    Then when you hit the button, you can set the left of the required view to 0…

    Im not sure there is such a thing as a modal view. If you are doing it that way, then you are opening a new window each time, which will kill the app after a while…
