App centered on iPhone5

Hello there,

I have a problem with the screen size on iOS.

If I start my app on an iPhone5 or the iOS-Simulator (4-inch) I have a black space area at the top and at the bottom of my application. It seems the the application is only 3.5-inch large.

I create a window like:

var window = Titanium.UI.createWindow({
        backgroundColor : '#000',
        navBarHidden : true,
        exitOnClose : true
    });

I've tried to set the height property to 100% but it only starts in the size of 3.5 inch (centered on the display).

The SDK-Version is 3.2.0.v20131212122847.

How do I get the screen to the full 4-inch size?
Do I miss some configuration?

Thanks for you help.

Regards,
Marco
