"Masking" or hiding overflow from a view?

Hi I'm trying to create a photo gallery that slides between images, to do this I was hoping I could have an internal view/label that hides content that doesn't fit (i.e. the width of the view is 200, but the actual width of the content inside is 300).

I want to slide the content to the left, while keeping the container fixed.

How would I go about this?

  • Try setting a border radius of 0 for the clipping view.

    var view =  Titanium.UI.createView({
   ...
   borderRadius: 0
});

    Not sure if this is intentional behavior but it seems to be working for me.
