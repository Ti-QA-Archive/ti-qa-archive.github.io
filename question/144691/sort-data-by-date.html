  • You should use the sort JavaScript function and using the date's time as your sort attribute. Assuming your all the objects from your array are Date objects.

    ####The code would look something like this:

    yourDateArray.sort(function(a, b) {
    return a.getTime() > b.getTime()? 1 : a.getTime()< b.getTime()? -1 : 0;
});

    Notice that the getTime function returns the number of milliseconds since 1970/01/01. So every Date would be unique.
