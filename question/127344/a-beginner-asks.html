  • Well, there's not really much we all could do, specially because there's no question in there…

    However, to help a little bit, I gathered a few links that might help you learn JavaScript quick enough.

    [For beginners]

    https://developer.mozilla.org/en/JavaScript/Guide

    http://www.w3schools.com/js/

    ["Advanced" JavaScript]

    http://ejohn.org/apps/learn/

    [JavaScript Coding Conventions]

    http://javascript.crockford.com/code.html

    [Github Titanium Project Repositories]

    https://github.com/search?type=Repositories&language=&q=titanium&repo=&langOverride=&x=0&y=0&start_value=1

    (I'll try to update the list later on)

    Now for learning how to use that in the Titanium environment, I strongly suggest looking at the Kitchen Sink project or some of the projects hosted on Github.

    Try to figure out how it all works and start from the beginning. Starting with a Geolocation project could be tough if you don't have much programming experience, so start with the easy stuff (User Interface and basic JavasCript).

    And eventually, when you have questions, just ask and we'll try to help. :)

    Good luck!
