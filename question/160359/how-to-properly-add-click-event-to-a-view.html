How to properly add click event to a View?

I have a view with several views, imageviews and label elements inside it. I want to add click event to that parent view. Here is an example code:

<View id="user" onClick="userClicked">
    <ImageView id="avatar"></ImageView>
    <Label id="username"></Label>
    <View id="info"></View>
</View>

The problem with the above code is that when user clicks on a child object, "avatar" for example, click event isn't fired on $.user. It happens only when I click outside of every child view but inside the $.user View. How should I add click event so it's fired whenever someone clicks inside the view, despite it's another child view or not?
