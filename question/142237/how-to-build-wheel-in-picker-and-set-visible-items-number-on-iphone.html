  • Accepted Answer

    • The connection of the 2 picker columns is something that only the native DatePicker does. If you wanted to do the same thing, you'd have to listen for the change event on the first column, and if the number is the first or the last number in the sequence, then you could update the number in the second column.

    • You cannot set visibleItems on the iPhone. The native picker is always the same size.
