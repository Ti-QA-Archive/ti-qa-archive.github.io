I have seen many answers but none works.

  • I've found the solution for Android

    Titanium.Filesystem.getFile(&#39;file:&#x2F;&#x2F;&#x2F;data&#x2F;data&#39;, Ti.App.getID() , &#39;databases&#39;, &#39;mydatabasename&#39;);
