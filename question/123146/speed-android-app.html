  • Accepted Answer

    Are you comparing speeds on actual devices, or on the iOS simulator vs the android emulator? Speed comparison must be done on real devices. The iOS simulator is very, very fast (MUCH faster than a device). The android emulator is a dog. This has nothing to do with Titanium – the same is true of native development tools.

    If you are comparing on actual devices and repeating the className property for rows with the same layout, then I'd suggest posting code to see if anyone has a suggestion. Are you using remote images?
