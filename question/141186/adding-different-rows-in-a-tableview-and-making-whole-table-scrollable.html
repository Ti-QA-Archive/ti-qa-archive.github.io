    Hi Nadeem

    If I understand the question correctly, you have done the rows correctly, you just need a label above the rows that will be part of the table when it scrolls.

    var viewHeader = Ti.UI.createView({
  height: Ti.UI.SIZE,
  width: Ti.UI.FILL
});
var lbl = Ti.UI.createLabel({
  height: Ti.UI.SIZE,
  text: 'something',
  width: Ti.UI.FILL
});
viewHeader.add(lbl);
var tbl = Ti.UI.createTableView({
...
  headerView: viewHeader,
...
});

    This will place a label at the top of the table which will scroll up and down with the rows.

    You can add other labels, buttons and more into the headerView and also do the same for footerView which does the same at the bottom.
