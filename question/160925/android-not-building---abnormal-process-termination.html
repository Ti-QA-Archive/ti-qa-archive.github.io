Android not building - abnormal process termination.

I am trying to build my project but now it is not working. Yesterday it was working great with no problems but after I changed the name in the tiapp.xml and the description of the app it would no longer build. I tried to change them back but the problem still persists. I tried adding a target min sdk and all that stuff suggested like restarting my computer and cleaning my project like 100 times. Then I tried to copy everything to a new project, still didn't build. I have turned the console level to Trace and this is all the output that I got:

[DEBUG] :  Processing i18n file: en/strings.xml
[DEBUG] :  Processing i18n file: ja/strings.xml
[DEBUG] :  Merging en strings => C:\Users\Nicholas\Documents\Titanium_Studio_Workspace\Quit Toolbox\build\android\res\values\strings.xml
[DEBUG] :  Writing ja strings => C:\Users\Nicholas\Documents\Titanium_Studio_Workspace\Quit Toolbox\build\android\res\values-ja\strings.xml
[ERROR] :  Failed to package application:
[ERROR] Application Installer abnormal process termination. Process exit value was 1

There was no error log or anything too. If you need more of the console output just ask but this seems to be where the error is occurring.

Additional info: I have installed every api level possible via the studio so that shouldn't be the problem. All the paths are correct ect so where am I going wrong?

  • I fixed the problem!!! For anyone else out there, you need to make sure that the folders in Resources/android/images do not have typos! for example my problem was one of the folders has /re-hdpi rather than /res-hdpi
