Sql no such table?

Titanium Studio: 3.2.3
ios: 7

I created an app with a pre-populated sql file. This is working fine and is now in the appstore in fact. The problem is I forgot to add a version table. So, I added a 'version' table in the next update. The way I plan of doing this is to query for the version table and add/amend old tables if the version is detected as old. This is the code:

var db = Ti.Database.open('tinyDb');
    var result = db.execute('SELECT * FROM version');

    if (!result.isValidRow()) {
        Ti.API.info('update database');
    }

However, I get this error 

Invalid SQL statement at index.js

In the console is says:

com.plausiblelabs.pldatabase.error.vendor.string=no such table: version

  • Ok,

    I found a master table that automatically gathers a list of created tables.

    var db = Ti.Database.open('tinyDb');

var result = db.execute('SELECT name FROM sqlite_master WHERE type="table" AND name="version"');

    if(!result.isValidRow()) {
       // made changes to database
     }
     result.close();
