  • Hi

    I think you are referring to the convenience function ApplicationWindow() this is meant to help you build apps in a more modular way, but can add confusion when you are looking for example code on the net.

    Try creating a default project which does not use and convenience functions - this will make it easier when you are adding code from the net.
