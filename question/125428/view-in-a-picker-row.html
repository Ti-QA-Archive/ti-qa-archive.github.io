View in a Picker Row

I wish to put a label inside of a pickerRow. according to the documentation of picker row, this is possible, it even shows sample code on the bottom; this code, however, doesn't work. Instead of showing the text it just says "[picker row]".

var picker = Titanium.UI.createPicker();
var row = Ti.UI.createPickerRow();
var label = Ti.UI.createLabel({
    text:'text',
    font:{fontSize:24,fontWeight:'bold'},
    color:'#000',
    width:'auto',
    height:'auto'
});
row.add(label);
picker.add(row);

Thank you in advance for your answers.

  • I found the problem, there is the same sample code on the bottom of the picker documentation. It says "Titanium's Android picker control (both the native and the useSpinner variety) does not support/display views that are added to picker rows. Only the title of the row will be displayed within the control." This note should also be added to the pickerRow documentation.
