    I reply to myself hoping that this can help others in the same situation: reading the log file (build.txt in the project folder) I saw that there was, with other stuff, this string:
    [TRACE] adb devices returned 0 devices/emulators
    So, I made another search and I found on this site an answer about this string but for another issue. I followed the solution proposed and I solved my problem. The solution is this:
    1) open a command prompt and go to the android sdk folder containing the adb.exe file (in my case: C:\Program Files (x86)\Android\android-sdk\platform-tools)
    2) launch this command (without quoting): "adb kill-server"
    3) launch this command (without quoting): "adb devices"
    Thanks to Tim Poulsen
