  • okay fellas I figured it out. Kinda whack though, but here goes.

    in index.html for example this is found in almost every answer.

    <html>
    <head>
    <title>Something</title>
        <script>Titanium.include("test.php");</script>
    </head>
    <body style="background-color:#1c1c1c;margin:0">
        <div style="border-top:1px solid #404040">
            <div style="color:#fff;;padding:10px">
            Welcome to Titanium
            <script type="text/javascript">document.write(hello);</script>
            </div>
        </div>
    </body>
</html>

    in test.php do not include PHP Tags as in

    $hello = 'hi';

    When i removed the PHP tags in test.php, the preprocessor semi works, although i havent tested passed this.
