It prompt this message "An application restart is required" and force application restart. I am using Nexus S Android 4.1.2; Titanium Studio build: 3.0.1.201212181159; iOS 6.1; Android SDK 2.2; OSX 10.8.2.

Please advise how to resolve this or skip the alert. Thank you.
