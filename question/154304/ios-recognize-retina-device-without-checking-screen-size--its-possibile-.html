    Hi

    You can do it like this.

    function isRetina() {
    return (Ti.Platform.displayCaps.density === 'high' && Ti.Platform.displayCaps.dpi === 320);
}

    Usage

    if (isRetina()) {
   alert('retina');
}

    I assume you already know you are running on iOS at this point.
