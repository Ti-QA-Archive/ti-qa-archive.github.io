Detecting Google Maps support in Android: how-to

I wasted hours trying to detect Google Maps support in an Android device, because there are a lot of devices without Google applications. I wanted to use a single APK that detects maps support at run time.

If you try to add a map view in a device without Google Maps, Titanium can't catch the exception because tries to use an unavailable library. Didn't find how to avoid this crash, so I tried to detect if the library exists in filesystem, because /system folder is accesible in read-only mode. It worked for Titanium 1.7/1.8 and Android 2.2/2.3/4.0.

First of all, you must add this to AndroidManifest.xml (I use a custom manifest):

<uses-library android:name="com.google.android.maps" android:required="false" />

Then, the code:

try {
    var path = 'file:///system/framework/com.google.android.maps.jar';
    var check = Ti.Filesystem.getFile(path);
    hasGoogleMaps = check.exists();
} catch(e) {
    hasGoogleMaps = false;
}

Hope you find it useful
