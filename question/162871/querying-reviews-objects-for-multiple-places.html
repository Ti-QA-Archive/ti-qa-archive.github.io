Querying Reviews objects for multiple places

Hi I am struggling to write an efficient query to find all queries associated with multiple places.

The docs says to specify a place_id. Which returns the correct results for a single place object. But what if I have multiple places that I want to find all reviews for? Whats the best way to write this query? I am also open to alternative methods as long as its efficient as possible.

Thanks

  • hi Elliott,

    In one query, ACS just allow to query reviews for a single place, so if you want to query reviews for multiple places, you need to make a loop to query.
