  • Hi Gowtham,

    Try this one.

    .xml file

    <View id="viewBatch">
    <Label id="lblValue"/>
</View>

    .tss file

    "#viewBatch":{
    right: '2dp',   //     as required 
    top: '5dp',     //     as required
    height: "20dp",    //     as required
    width: "20dp",    //     as required
    backgroundColor: 'transperant',
    backgroundImage: 'bgImagBatch.png',    //   user image 
    opacity: 1,
    shadow:{
        shadowColor: "black",
        shadowRadius: 5,
        shadowOpacity: 0.9,
        shadowOffset:{
            x: 5,
            y: 5
        }
    },
    borderRadius: "10dp"
},

"#lblValue":{
    height: Ti.UI.FILL,
    width: Ti.UI.FILL,
    text: '0',
    color: 'white',
    textAlign: "center"
}

    This is your same question and same answer for this. Thanks.. :)
