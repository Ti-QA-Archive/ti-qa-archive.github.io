TIP - how to use hideStatusBar/showStatusBar on iOS7

Hello everyone.

This is not an question, it is a help to all who may be experiencing this problem since as I had to face it without any question about this response.

Since the release of iOS 7, Apple changed the functioning of the StatusBar and the Team of Appcelerator deprecated two functions that I used the "hideStatusBar" and "ShowStatusBar".

These functions hide and show the statusbar animating it in runtime.

BUT, unlike what has been said there, this functionality IS PRESENT in iOS 7.

After a long time trying, I managed to do that and I will explain step by step how to solve this.

Assume here that you are using the SDK 3.1.3 GA and Xcode5 to compile applications for iOS 7.

Step One, enable the deprecated functionality on Titanium SDK:

  • Open ~ / Library / Application Support/Titanium/mobilesdk/osx/3.1.3.GA/iphone/Classes/TiUIiPhoneProxy.m in your favorite text editor;

  • Look for "- (void) hideStatusBar: (id) args";

  • On the line below, comment out the code that identifies the use of iOS7 and disables this function:

    
  /*
  if ([TiUtils isIOS7OrGreater]) {
       DebugLog (@ "hideStatusBar method is not supported on IOS7. Ignoring call. FullScreen Use the property of the window");
       return;
   }
   */

  • Do the same on function "- (void) ShowStatusBar: (id) args";

  • Save and close the file;

  • In Titanium, Clear your project by clicking on the menu "Project" -> "Clean …";

  • Run the Simulator.

Right now if you test still will not work.

Step Two, inform iOS7 how to use statusBar:

  • Close the Simulator;

  • Open your project folder, open "build / iphone";

  • Copy the file "Info.plist" to the root of your project (the same folder that contains the "tiapp.xml");

  • Open the file "Info.plist" in your favorite text editor and add the following key:

    <key>UIViewControllerBasedStatusBarAppearance</key>
<false/>

  • Save and close the file;

  • Clean your project again and rerun the simulation.

Ready, functionality back in the statusbar iOS7.

Don't negative this, because isn't an question, give positive votes, so that it becomes this very visible and who knows Appcelerator Team back the functionality to us.
