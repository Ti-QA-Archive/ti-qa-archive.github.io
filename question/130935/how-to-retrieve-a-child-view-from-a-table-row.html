How to retrieve a child view from a table row?

I want to extract the child view from the clicked list item. e.row returns row but how to retrieve child view.

  • You can set the view as a custom property on the row - eg;

    var row = TI.UI.createTableViewRow();

var myView = Ti.UI.createView();

// add view to row to display it

row.add(myView);

// save reference to that view on row custom property

row.myAttachedView = myView;


// Now, when you get a reference to e.row , you can specify

e.row.myAttachedView
