Hi,

  • I located the iOS docs on what the SDK supports in the settings bundle, and it appears to me there is no way to do this using public APIs. The only thing I could find which impacts visiblity is a way to make a given setting only appear if a phone, or only appear if an iPad.

  • Hello Robert,

    You can group all the controls/elements that are conditionally shown within a single view and then use the visibe property of the view to decide whether to show/hide it according to your toggle's status. Alternatively you can simply add or remove from the parent as per your toggle, whatever is easier for you!
