  • HI,

    you can simply get your mobile screens height and width and then you can set controls as per this height and width
    you can use following code to get height and width

    var width=Ti.Platform.displayCaps.platformWidth;
var height=Ti.Platform.displayCaps.platformHeight;

    and you can set controlls as

    var view=Ti.UI.createView({
width:0.2*width,
height:0.5*height,
top:0.1*height
left:0.25*width
});

  • There is no simple answer to "how to build a cross-platform app". You have to take the time to understand the target platforms and design your application so that it adapts to a variety of screen sizes and densities. Raj is pointing you in the right direction, but you will find that it gets even more complicated than what he is showing you. For example, if you have a tabbed interface, you may need to know how much of the screen is taken up by the tabs themselves.

    For an in-depth look at building dynamic cross-platform UIs, have a look at GrillTime, a tutorial app that I wrote recently.
