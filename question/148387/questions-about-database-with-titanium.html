    • Rainer is correct, the suffix of database file is irrelevant to the OS, you do not even need to provide a file extension.

      • something.db
      • something.sql
      • something.random
      • something

      There is no difference between these once installed.

      As to a blank database being possible without structure or data, think about a word document that has no words - you do not have to have written anything in a word document for the OS to know it is a word document.
