  • Hi

    Edit your tiapp.xml file you can find it here;

    {appname}/resource/
{appname}/tiapp.xml <= here

    Change the following settings to the following values;

    <statusbar-hidden>true</statusbar-hidden>
<fullscreen>true</fullscreen>
<navbar-hidden>true</navbar-hidden>

    Define your windows as follows;

    var win = Ti.UI.createWindow({
    backgroundColor: '#fff',
    exitOnClose: true,
    fullscreen: true,
    navBarHidden: true,
    tabBarHidden: true
});
win.open();

    Try this and report back.
