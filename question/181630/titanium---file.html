    • Did you check to see if the storage is available first? 

      if (Ti.Filesystem.isExternalStoragePresent() == true){
 Ti.API.info("THERE IS A SD CARD")
} else {
 Ti.API.info("THERE IS NOT A SD CARD... Now what happens???")
}

      What do you do when the user does not have an SD card?

      Are you creating the sounds folder first and populating it? You should use getDirectoryListing if you want to traverse the files in a directory.

      var folder = Ti.Filesystem.getFile(Titanium.Filesystem.externalStorageDirectory , '/Sounds/');
var dir = folder.getDirectoryListing();
var intFile = 0, intFiles = dir.length;
for (intFile = 0; intFile < intFiles; intFile = intFile + 1) {
        Ti.API.info('file', dir[intFile]);
}
