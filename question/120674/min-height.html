    • After adding a label (detailsLabel is my label name) to my row I checked the height and if it was less than 70, I set the height of my label to 70 (my desired minimum height). My initial row height is set to "auto". For Android you may want to set it to 70dp so it will adjust for screen resolution. 

      
if (detailsLabel.height <70){
    detailsLabel.height = 70;

}
