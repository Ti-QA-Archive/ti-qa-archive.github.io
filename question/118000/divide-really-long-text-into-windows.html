Divide really long text into windows

Hey,

I have a ~100 page book that I want to add to my app.

I would like to load this 100 pages text into multiple pages (one option is to add them to a scrollable view).

Is it possible to automatically add the text to several scrollable views instead of creating everything manually?

Thanks
Idan

  • Idan

    One strategy you could take would be to create a single scrollableView, and add each page to its own view which would then be added to the scrollableView. These views would be programmatically created and added using a loop. You could add all 100 pages together, but I have no idea what performance you'd experience. If it needs optimizing, you could consider adding each block of 10 pages, which you could remove and replace with the next 10 pages when the user scrolls past the 10th or the 1st limit, as required.

    See the Programmatic row creation with Titanium Views section of the Using TableViews guide. Although it's not an example of programmatically creating views for a scrollableView specifically, the general approach is the same.

    Hope this gives you some ideas.

    Cheers
