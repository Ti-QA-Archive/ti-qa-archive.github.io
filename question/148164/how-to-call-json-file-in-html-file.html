  • Since you are in a script tag, you can use the XMLHttpRequest just as you would in a normal html page. Both iOS and Android use WebKit, so you can expect pretty much the same behavior (at least for ajax requests).

    So what you would do is:

    • Create your XMLHttpRequest
    • Invoke this request using your JSON's url
    • In your callback function, you parse the request.responseText using the JSON.parse function. This will transform your JSON content into an actual JavaScript object.
    • Access your object's properties as you would any other object.
