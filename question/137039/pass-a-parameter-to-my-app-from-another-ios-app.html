  • Hi Mitul

    Here's how I retrieve the parameter

    Titanium.App.addEventListener('resumed', function() {
    var args = JSON.stringify(Ti.App.getArguments());  
     //do some stuff to parse the JSON object in order to have clean key:value pairs
});

    On my side I had no need to modify the app plist, regarding the URL Sheme. It worked out of the box.
    I think you might add other EventListeners to cover all users situations.
