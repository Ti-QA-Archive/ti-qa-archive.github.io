  • Hi,

    Use font property of button or textfield.Like :

    var win=Titanium.UI.createWindow({
    backgroundColor:'white'
});

var button = Titanium.UI.createButton({
   title: 'Hello',
   top: 10,
   width: 200,
   height: 150,
   font: {fontSize: '50'},
});

win.add(button);
win.open();

    For more details go to : http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.Button-property-font.
