Undefined when looping trough object

I'm querying all the records from a table in the database, like:

var news_arr = Alloy.createCollection('news'); 
var table = news_arr.config.adapter.collection_name;
news_arr.fetch({query:'SELECT * from ' + table + ' '});

Titanium.API.info(&#39;test1: &#39; + JSON.stringify(news_arr));

returns:

test1: [{&quot;cambuur_id&quot;:123,&quot;date&quot;:&quot;2014-08-04T10:50:06.000&quot;,&quot;title&quot;:&quot;dit is een testpost2&quot;,&quot;content&quot;:&quot;content for testpost2&quot;,&quot;photo&quot;:&quot;&quot;,&quot;is_banner&quot;:&quot;&quot;},{&quot;cambuur_id&quot;:122,&quot;date&quot;:&quot;2014-08-04T10:50:06.000&quot;,&quot;title&quot;:&quot;dit is een testpost1&quot;,&quot;content&quot;:&quot;content for testpost1&quot;,&quot;photo&quot;:&quot;&quot;,&quot;is_banner&quot;:&quot;&quot;}]

When I iterate trough "news_arr" like:

for (var i = 0; i < news_arr.length; i++) {
    Titanium.API.info('test2: ' + JSON.stringify(news_arr[i]));
}

It returns:

test2: undefined

I can't find out what is going wrong. Does anyone see what is going wrong?

    Try the following:

    for (var i = 0; i < news_arr.length; i++) {
    Titanium.API.info('test2: ' + JSON.stringify(news_arr.at(i)));
}
