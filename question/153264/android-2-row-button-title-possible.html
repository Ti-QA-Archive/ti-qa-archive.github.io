    if it doesn't work natively you can do something like this:

    var view = ...
var backgroundImage = ....
var label = ...

view.add(backgroundImage);
view.add(label);

var buttonImage = view.toImage();
var myButton = Ti.UI.createButton({backgroundImage:buttonImage})

    so you still have a button but with a background image with the label on top.
