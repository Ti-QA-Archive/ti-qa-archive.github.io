freeing memory

No doubt my ignorance, but I have a simple question:

If I create an object (i.e. M = new match) containing an array and other variables. During execution the array within M grows (ie. push()). At some time I will free the object with M = null.
Will the array inside M be automatically freed? or should I empty it first?

    You will have to free each of the objects contained in the array if you want to free the memory. Otherwise, you will have to wait for the interpreter to check if the objects have no references left to anything and then free those.

    Hope this clears it for you.
