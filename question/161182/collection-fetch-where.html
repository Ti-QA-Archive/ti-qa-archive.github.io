Collection Fetch Where

Hello,

I am storing GPS data on intervals to SQL Lite. Towards the end, I'd like to fetch that data to do some processing. For some reason, when I do, I keep getting empty collections.

I'm storing the data using the following:

            var m = Alloy.createModel(
                'Coordinate', {
                    lat: lat, 
                    lon: lon, 
                    exerciseInstanceID: exerciseID,
                    registered: new Date()
                }
            );
            Ti.API.info(JSON.stringify(m));
            m.save(m,{
                success: function(){
                    Ti.API.info("SUCCEDED");
                },
                error:  function(){
                    Ti.API.info("FAILED");
                }
            });

And retrieving using the following:

    var c = Alloy.createCollection('Coordinate');

    c.fetch({
        query: "SELECT * FROM coordinate WHERE exerciseInstanceID = ?", 
        params: [exerciseID], 
        success: function(){
            Ti.API.info(">>> RETRIEVED >>>", exerciseID, c.length); // c.length is always 0
        },
        error: function(_e){
            JSON.stringify(_e);
        }
    });

The problem is, upon retrieval, the collection length is always 0, even when I have (allegedly) stored 400+ gps coordinates
