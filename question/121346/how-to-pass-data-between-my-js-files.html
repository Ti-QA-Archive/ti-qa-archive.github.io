how to pass data between my .js files?

Hi everyone,

as we develop application we use alot of files for different reasons.

how do we pass an array of data from one file to the other. 

If i have a tableview containing several rows init and upon clicking on a row the details of that row should be displayed in a separate window. For this we write the tableview in one file and specify the parameters and etc . . . for the window in the other file. Now how do we send a set of data from one file to the other so the details of that row is successfully displayed the next window..

Please help me out with that

Thank you ! ! !

  • You can use Ti.App-events: 

    Ti.App.fireEvent("test", {data: content});

Ti.App.addEventListener("test", function(e) {
    Ti.API.log(e);
});

  • If you scroll to the bottom of the following page you'll see notes on this subject under the heading 'Passing Data' which explains how to achieve what you're after.

    http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.UI.Window-object

    In your case I would try:

    var yourArray = ['item1', 'item2', 'item3'];

var w = Titanium.UI.createWindow({
    url:'foo.js'
});

w.passedArray = yourArray;

    In your foo.js file you could do the following to access the 'passedArray' reference assignment:

    Titanium.UI.currentWindow.passedArray

    Hope that helps.

  • You can store is as a global variable, such as:

    app.js:

    Ti.App.GlobalArray = ['data','data2','data3'];

    foo.js:

    Ti.API.warn(Ti.App.GlobalArray);

    Should dump:

    data,
data2,
data3
