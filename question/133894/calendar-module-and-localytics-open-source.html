Calendar module and Localytics open source!

I just want give something back to the community here at Appcelerator and want to publish the source code for Localytics and a Calendar module I've been using. The Calendar module is not the one in the marketplace.

I want you to continue improving the modules, so feel free to fork and modify as you wish.

Calendar - Appgutta/AGCalendar

Localytics - Appgutta/TiLocalytics

There's also a precompiled version in both repositories, so you dont need any objective-c coding experience to use it. The documentation and example app.js should get you going. If not, dont hesitate to ask questions. Have fun with it!

Please dont republish and charge money for the modules in the Marketplace. Let's keep it available for everyone.

iOS only btw.

  • So, I've added recurring events and "swipe-to-delete" in the calendar module.
    I'll show some examples on how and when to use the additions.

    To be able to use these additions, you need to download the latest build - 1.1.220320122142 or clone the repository.

    Swipe-to-delete - Works for both eventkit and coredata

    // When creating your calendarView, you need to define
// editable: true (default: false) to enable swipe-to-delete
var calendarView = Ti.Calendar.createView({
    top:0,
    editable: true,
    color:"white"
});

    Recurring events - For eventkit only.

    // As recurring events only works when using eventkit 
// as our datasource, make sure its set.
Ti.Calendar.dataSource("eventkit");

// For this event, we want it to repeat every other day in one month.
// Start by creating the end date for our recurrence rule.
var recurringEnd = new Date();
recurringEnd.setMonth(recurringEnd.getMonth()+1);

// Let's make the event last for 3 hours.
var eventEnd = new Date();
eventEnd.setHours(eventEnd.getHours()+3);

// Create and add the event to the calendar.
Ti.Calendar.addEvent({
    title: "Once every other day in one month",
    startDate: new Date(),
    endDate: eventEnd,
    location: "Appcelerator's HQ",
    note: "A note...",
    recurrence: { 
        frequency: "day", // day, week, month, year
        interval: 2, // 1 = every day, 2 = every other day, 3 = every third day and so on...
        end: recurringEnd 
    }
});

    • The module should be placed in the modules directory in the root of your project. 

      - ~/Development/Apps/MyApp/
    - Resources
        - iphone
        - app.js
    - modules
        - ag.calendar
            - 1.2.8
                - assets
                - documentation
                - example
                - libag.calendar.a
                - ......
    - manifest
    - tiapp.xml
    - .....

      Now, in tiapp.xml you should add this line:

      <modules>
    <module platform="iphone" version="1.2.8">ag.calendar</module>
</modules>

      Hope this get you going.
