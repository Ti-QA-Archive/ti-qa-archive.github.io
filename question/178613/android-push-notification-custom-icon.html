Android Push Notification Custom Icon

Hi guys

im having trouble implementing a custom icon for push notifications.

Cant seem to find any examples of this, i did find an online tool for generating the images and im aware that the images go into the /platform/res/drawable folders

however i cant find any examples of how to reference it in either the manifest or style.xml

if anyone has any information on this would be amazing

Gary

    Hello Gary,

    If you are using ACS for push notifications in android you need to put the custom icon on below folder
ProjectName/platform/android/res/drawable/myicon.png and pass the myicon name into the push notification payload.do not pass the image extension into the payload.
Below is an example of the payload which contains the custom icon.
{
    "alert":"i am your alert",
    "title":"i am your title",
    "icon":"myicon",
    "sound":"your_sound.wav"
}
Let me know if it does not work for you.

    Jitendra
