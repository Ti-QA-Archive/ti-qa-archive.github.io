Alloy TabGroup - using Require for each Tab

I have 5 tabs but I don't want to define what feels like the entire application in my index.xml. I've tried to use Require to load in views for each tab but the window remains blank for those tabs.

<Tab title="PROFILE" icon="profile.png">
    <Require type="view" src="profile" id="profileTab" />
</Tab>

And then in my profile.xml I have

<Alloy>
    <Window title="" navBarHidden="true">
        <Label>I am a profile</Label>
    </Window>
</Alloy>

Any thoughts?
