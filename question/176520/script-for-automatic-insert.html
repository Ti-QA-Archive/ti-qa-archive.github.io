    • I use Notepad++ on my hybrid windows + mac combined dev environment, sublime when mac only. I also use the ZenCoding plug-in to make coding faster - particularly xml, where a few shorthand entries can create an entire View;

      So this;

      Alloy>TabGroup#tabs>(Tab#tab1>Window#win1.container>TableView#tbl>TableViewSection>TableViewRow[title="Title of Row"]*5)+(Window#win2.container>(View#header>Label{Header})+(View#bodyLabel{Body})+(View#footer>Label{Footer}))

      Becomes;

      <Alloy>
  <TabGroup id="tabs">
    <Tab id="tab1">
      <Window id="win1" class="container">
        <TableView id="tbl">
          <TableViewSection>
            <TableViewRow title="Title of Row"></TableViewRow>
            <TableViewRow title="Title of Row"></TableViewRow>
            <TableViewRow title="Title of Row"></TableViewRow>
            <TableViewRow title="Title of Row"></TableViewRow>
            <TableViewRow title="Title of Row"></TableViewRow>
          </TableViewSection>
        </TableView>
      </Window>
    </Tab>
    <Window id="win2" class="container">
      <View id="header">
        <Label>
          Header
        </Label>
      </View>
      <View id="bodyLabel">
        Body
      </View>
      <View id="footer">
        <Label>
          Footer
        </Label>
      </View>
    </Window>
  </TabGroup>
</Alloy>
