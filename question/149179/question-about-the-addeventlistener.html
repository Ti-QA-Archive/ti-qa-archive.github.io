Question about the addEventListener

Hello everybody,

I am asking here about a code that I am trying to understand:


Ti.App.addEventListener('toppings',openToppings);
Ti.App.addEventListener('cancelToppings',openCrust);

it saids in the documentation that this line "adds the specified callback as an event listener for the named event", but I can't figure out where should I declare the "toopings" and the "cancelToppings" events? what they are supposed to be?

Thanks in advance!

    Well, if you look at line 27 in Step 1 from the tutorial you are following (I knew I saw this code somewhere…)

    You will see that this line actually creates the toppings event. And the second parameters points to the function that will be called when this event will be fired. In your case, every time the toppings event is fired, the openToppings function will be called.

    Same goes for the cancelToppings event declared at line 16 of Step 4.
