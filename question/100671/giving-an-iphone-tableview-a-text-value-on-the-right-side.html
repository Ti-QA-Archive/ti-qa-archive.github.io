  • To the best of my knowledge, titanium does not expose that iphone table style as a property. But you can certainly create the same effect by adding a view to each table row, then placing labels or any other control type desired in the view. That way you can control the font size and color, etc to achieve the same look.

    This has the side benefit of working under android too, should you ever decide to go cross-platform with the app. So think of it as a "feature". :)
