    This library will help quite a bit: https://github.com/raulriera/XHR

    the api returns a json object like the following:

    {
   "dstOffset" : 0,
   "rawOffset" : -28800,
   "status" : "OK",
   "timeZoneId" : "America/Los_Angeles",
   "timeZoneName" : "Pacific Standard Time"
}

    So, you may access it similar to the following (but don't forget your api key!)

    var XHR = require("/xhr");
var xhr = new XHR();
xhr.get("https://maps.googleapis.com/maps/api/timezone/json?location=39.6034810,-119.6822510&timestamp=1331161200", function( result ){
    alert( result.timeZoneName );
});

    • The document that you point to in your question states:

      To specify a key in your request, include it as the value of a key parameter.

      so I would start when you build your url by appending '&key=[api key]'

      You won't need to put it in your manifest file.
