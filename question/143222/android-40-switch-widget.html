  • The Android switch is an on/off toggle. Small button with On/Off states and a gray bar that turns green in the On state. The switch has been like this for some time.

    I'm thinking that it may have been much quicker to just copy the default createSwitch code into a project and test it out to see if its what you want:

    var basicSwitch = Ti.UI.createSwitch({
  value:true // mandatory for ios
});
