Chaning the exitOnClose propery after window is opened still scloses the app

Hi,
I have a window that's opened with the 'exitOnClose' property. It's all fine and works as it should, but I have a case where I want to close that window without it closing my entire app.

So what I did was set the 'exitOnClose' property to false and then close the window - but it still closes my entire app. How can I solve this?

Thanks

  • You cannot change this property after you construct the window.

    "exitOnClose boolean

    (Android only.) Boolean indicates if the application should exit when the Android back button is pressed while the window is being shown. You can only set this as a createWindow({…}) option. Setting it after window creation will no effect."
