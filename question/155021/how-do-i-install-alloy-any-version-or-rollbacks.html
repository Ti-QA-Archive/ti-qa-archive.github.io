How do I install Alloy? (any version or rollbacks)

DISCLAIMER: This Tony Lukasavage and I'm answering this myself to put it in the FAQ

How can I install the latest stable Alloy? How about the cutting edge version from github? How about a specific version? What about rolling back to the latest stable after testing these other versions?

    NOTE: DO NOT use the sudo prefix on Windows. Only on OSX and Linux.

    latest stable

    sudo npm install -g alloy

    specific version

    for example, 1.1.3

    sudo npm install -g alloy@1.1.3

    cutting edge from github

    sudo npm install -g git://github.com/appcelerator/alloy.git

    rollback to latest stable (clean)

    sudo npm uninstall -g alloy
sudo npm install -g alloy

    All of these installs are trivial. I highly encourage developers to try out the cutting edge version as it will take you seconds to roll back to an earlier version if you find it unstable or not to your liking. It's also a great way to help me shape upcoming versions.
