Toolbar - Prev/Next Buttons for tabbing to txtfields.

Hello,

I have a keyboard toolbar with a number of buttons i.e. previous, next, done. On my screen I have a number of text fields, and I want the user to be able to tab back and forth between the fields via the buttons. The done button hides the keyboard, this bit of functionality works fine.

My code is below, I think I'm in the right area, but I can't get it working. There could be problems with my syntax.

//Setup keyboard toolbar.
var flexSpace = Titanium.UI.createButton({
    systemButton:Titanium.UI.iPhone.SystemButton.FLEXIBLE_SPACE
});

var navButtons = Titanium.UI.createButtonBar({
    labels:['Previous','Next'],
    backgroundColor:'#000',
    top:100,
    style:Titanium.UI.iPhone.SystemButtonStyle.BAR,
    height:25,
    width:'auto'
});

var done = Titanium.UI.createButton({
    systemButton:Titanium.UI.iPhone.SystemButton.DONE
});

navButtons.addEventListener('click',function(e){
    if (nameTxt.focus()){
        nameTxt.hasFocus = false;
        phoneTxt.hasFocus = true;
    }
});

done.addEventListener('click',function(){
    nameTxt.blur();
    phoneTxt.blur();
    makeTxt.blur();
    colourTxt.blur();
    mileageTxt.blur();
    regTxt.blur();
    fuelTxt.blur();
    conditionTxt.blur();
});

Sample text field parameters:

var nameTxt = Ti.UI.createTextField({
    top:7,
    left:100,
    right:10,
    hintText: 'name',
    height:40,
    textAlign:"right",
    borderStyle: Ti.UI.INPUT_BORDERSTYLE_ROUNDED,
    keyboardToolbar:[navButtons, flexSpace, done],
    keyboardToolbarColor: '#336699',    
    keyboardToolbarHeight: 40
});

Any help appreciated, I've been looking into this for awhile and cannot figure it out. Did some googling as well and couldn't get an answer.

Regards,
Stephen

    You can tackle this a couple of different ways, however if you are just want to know which text field has focus, you can simply track it yourself. Once the window is loaded, if you force the focus to the first text field then you know which field has the current focus. From there, all you have to do is listen for the focus event on each of your text fields, and set some variable accordingly. ie:

    var focused;
var nameTxt = Ti.UI.createTextField(/*args*/);
nameTxt.addEventListener('focus', function(e) {
    focused = nameTxt;
});
// force the focus on your first field
nameTxt.focus();
// etc, etc.

    If you wanted to get a bit more crafty with it, you could create a simple factory function that creates your fields then pushes them onto an array. Then you could literally prev and next through the array indexes to change focus.
