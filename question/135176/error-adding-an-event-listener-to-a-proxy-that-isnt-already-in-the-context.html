[ERROR] Adding an event listener to a proxy that isn't already in the context

This code generates an error:

[ERROR] Adding an event listener to a proxy that isn't already in the context

if I quit the gradient background it won't give this error

var priordiag_button = Titanium.UI.createButton({
         title:'Prior Admissions',

             style : Ti.UI.iPhone.SystemButtonStyle.PLAIN,
         backgroundGradient : {
                type : 'linear',
                colors : ['#759ae9', '#2362dd'],
                startPoint : {
                    x : 0,
                    y : 0
                }
            },
        borderRadius : 5,
        color : 'white',
        font : {
            fontWeight : 'bold',
            fontSize : 14
        },

        bottom:"4%",
        right:"3%",
        width:"42%" ,
        height:"9%"   
    });


innerView.add(priordiag_button);

  • A quick and dirty fix for this issue is to set the backgroundGradient AFTER you have created the button and added it to a view. This will put the button into the context and prevent this message from displaying.
