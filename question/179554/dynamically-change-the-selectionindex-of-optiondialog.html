Dynamically Change the selectionIndex of OptionDialog

Hi I want to change the selectionIndex of OptionDialog Dynamically. Means Initially male will be selected but if any one changes the selection and make it like female and then again if any one open the OptionDialog then female should come as selected.

myArray = ['Male','Female','Other'];
var opts = {
      cancel: 2,
      options: myArray,
      selectedIndex: 0,
      destructive: 0,
      title: 'Gender'
    };
 dialog = Ti.UI.createOptionDialog(opts).show();
dialog.addEventListener('click', onSelectDialog);

function onSelectDialog(event){
    dialog.selectedIndex = event.index;
    alert(event.index);
 }

dialog.selectedIndex = event.index; is not working.
