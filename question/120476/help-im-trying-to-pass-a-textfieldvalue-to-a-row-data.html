  • Can you elaborate on what exactly you are trying to do? In the code given, you are creating a tableview row for each record in a SQL result set. For each of those rows you are creating a text field (tf) to hold a quantity but not assigning a default value. Therefore, I think you are intending them to hold a value keyable by the user.

    Since at the time you are creating the tableview rows no qty data exists yet, what exactly are you wanting to "pass" to the the row data[]?

    Are you wanting to retrieve the qty value keyed by a user on a given row? If so, are you wanting to know the value as soon as something is keyed (that is, use an event on the row's text field for instant notification as changes as made)? Or are you wanting to let them key everything, then later loop over all the rows to find the quantities given?

    Please describe better what it is you are trying to achieve. You also didn't mention whether this is a desktop or mobile project, let alone which platform/OS/SDK level you are targeting.
