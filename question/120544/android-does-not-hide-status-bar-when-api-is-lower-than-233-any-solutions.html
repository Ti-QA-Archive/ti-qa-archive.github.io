  • David

    Try "fullscreen:true", which makes the window modal and hides the status bar.

    Please report back if this isn't quite the effect you are after.

    Also, if running on the emulator, use Android 2.2 rather than 2.3.X, as the latter contains bugs, as explained in Android SDK / Target Android Platform.

    Cheers
