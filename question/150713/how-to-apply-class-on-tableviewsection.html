  • Alloy 3.0 doesn't exist, are you referring to 1.0.0? The way to do it would be with the <HeaderView> element. You can see an example of it here: https://github.com/appcelerator/alloy/tree/master/test/apps/ui/proxy_properties.

    Yes, this requires Alloy 1.1.0 as detailed in TIMOB-464. You can wait for the official release which should be in the next few days, or you can start using the candidate release right now by installing like this from the command line:

    sudo npm install -g alloy@1.1.0-cr
