Possible to appendRow to the top of a tableView instead of the bottom?

I have just implemented the Pull To Refresh feature in my app. The only problem I am seeing is that the new appended rows using tableView.appendRow(row) are showing up at the bottom on the tableView. I need them to append to the top of the tableView. Is this possible?

If anyone knows if this is possible please let me know. It really wouldn't make sense to append new rows to the bottom of my tableView. I have it set up so that when the tableView initially loads it pulls the most recent posts from the database and orders them with the most recent post at the top and descends from there.

Any help is greatly appreciated!

  • I figured it out, no need to reload the entire tableView, you just need use insertRowBefore like this tableView.insertRowBefore(0, row); and it works perfectly!
