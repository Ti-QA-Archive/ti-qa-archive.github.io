Resize the picker text in Android.

I want to reduce the height of the Android Picker using Appcelerator Titanium. It allows me to reduce the size but then content is not visible. Moreover I found no way to reduce the font size of the picker data in titanium android. Please let me know if anybody has faced the same problem and found the resolution. Thank you

    I faced the same issue and found no solutions, if you find anyone please tell me.

    I used a tableView instead of a picker.

    For alloy, this is the .xml file

    <Alloy>

    <Label id="lbl" onClick="displayTableViewPicker">TableViewPicker</Label>

    <TableView platform="android" id="tablePicker" height="Ti.UI.SIZE" backgroundColor="brown" visible="false" onClick="changeLabelText">
        <TableViewRow title="Row 1"></TableViewRow>
        <TableViewRow title="Row 2"></TableViewRow>
        <TableViewRow title="Row 3"></TableViewRow>
        <TableViewRow title="Row 4"></TableViewRow>
    </TableView>
</Alloy>

    .tss file

    "#lbl": {
    font: { fontSize: "14" },
}

    .js file

    function displayTableViewPicker(e) {
    $.tablePicker.visible = true;
}

function changeLabelText(e) {
    $.lbl.text = e.row.title;
    $.tablePicker.visible = false;
}

  • Samson,
    You can specify the font size of the picker 'content' (rows) by explicitly specifying row
    properties. But to do this you need to create your rows using Ti.UI.createPickerRow().
    e.g ( Ti.UI.createPickerRow( {option: val, option:val, option:val} ); )

    //real life example below (actually I'm not sure you can mix the font sizes, mine are all the same) but you get it…

    //picker to choose State
    var picker = Ti.UI.createPicker({width: Ti.UI.SIZE , visibleItems: 3});

    //column to add to picker
    var column = Ti.UI.createPickerColumn();

    //add each row explicitly to column zero
    column.addRow(Ti.UI.createPickerRow({title:'– Choose State –',state_code:'', fontSize: 16}));
    column.addRow(Ti.UI.createPickerRow({title:'Alabama',state_code:'AL', fontSize: 14}));
    column.addRow(Ti.UI.createPickerRow({title:'Alabama',state_code:'TX', fontSize: 18}));

    //add column w/data to the picker
    picker.add([column]);
