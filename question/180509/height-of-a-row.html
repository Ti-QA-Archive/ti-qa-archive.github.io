Height of a row

Hi, i'm working with tableview, and i have a problem. Is a dynamic tableview, load the rows from a database. The problem is the different rows has different height.

This is the code.

    var rows = db.execute('SELECT * FROM Termostatos ORDER BY Termostatos.Nombre');
    $.MenuLateral.deleteSection(0);

    var RowsTabla=[];
    var i;
    var sectionTabla = Ti.UI.createTableViewSection({ headerTitle: txtTituloSeccion });
    RowsTabla.push(sectionTabla);
    for (i=0;i<NRegistros;i++){
        TablaNombreTermostatos[i]= rows.field(2);
        var nuevaRow=Ti.UI.createTableViewRow();
        var titulo=Ti.UI.createLabel({
            id: "NombreTerm",
            text: "       "+rows.field(2),
            color: "white" ,
            textAlign: Ti.UI.TEXT_ALIGNMENT_LEFT, 
            left:2
        });
        var imagen = Ti.UI.createImageView({
            id: "BorroTerm",
            image: '/aspa.png', 
            right: 10,
            width: '5%',
            height: '70%'
        });
        nuevaRow.add(titulo);
        nuevaRow.add(imagen);
        RowsTabla.push(nuevaRow);

        rows.next();
    }
    var nuevaRow=Ti.UI.createTableViewRow();
    var titulo=Ti.UI.createLabel({
        text: "End of table",
        color: "white" ,
        left:2
    });
    var imagen = Ti.UI.createImageView({
        id: "BorroTerm",
        image: '/mas.png', 
        right: 10,
        width: '5%',
        height: '70%'
    });
    nuevaRow.add(titulo);
    nuevaRow.add(imagen);
    RowsTabla.push(nuevaRow);

    $.MenuLateral.setData(RowsTabla);

MenuLateral is created in .xml

        <TableView id="MenuLateral" top="0%" height="47%" left="0%" zindex="20" width="0" backgroundColor="black">
        </TableView>

Sometimes one row have a height and another row have another height. How i can fix it??

Thanks

    You can set a fixed height for your row when you create it:

    var nuevaRow=Ti.UI.createTableViewRow(height:100);

    Ray
