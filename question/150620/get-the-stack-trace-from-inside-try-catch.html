Get the stack trace from inside try catch

Hi,

I want to get the stack trace from side the try - catch block for both iOS and Android. Is there any way by which we can get the stack trace. The below code only gives the error message.

var currentWindow = Ti.UI.createWindow({
    exitOnClose : true
});


var obj = null;

try {

} catch(e) {
    alert(JSON.stringify(e.message));
}

currentWindow.open();

  • This code returns the stack trace, but there is not defined way to get the stack trace which will work on both iOS and Android or future platforms.

    var currentWindow = Ti.UI.createWindow({
    exitOnClose : true
});


var obj = null;

try {

} catch(e) {
    alert(e.stack);
}

currentWindow.open();
