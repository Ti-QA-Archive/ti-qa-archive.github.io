  • Simply modify the xml_rss.js from KitchenSink.

    The Facebook Feed is accessible via this Url:
    http://www.facebook.com/feeds/page.php?format=atom10&id=USER-ID-HERE

    To get the User ID go to:
    http://graph.facebook.com/FACEBOOK-USERNAME-HERE

    Replace "FACEBOOK-USERNAME-HERE" in the URL with your Facebook username and you'll get a JSON object, the first property is something like "id": "123456789" - in this example, 123456789 is the Facebook user ID.
