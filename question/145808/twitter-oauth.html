    • No problem.

      The built-in twitter sheet UI cannot be changed, you can only trigger it appearing with optional default values.

      The only way to achieve this with your own UI is to do all the calls to twitter yourself using an oAuth adapter. Fiddly but it works if you need something in your own UI or tweets firing in the background automatically.
