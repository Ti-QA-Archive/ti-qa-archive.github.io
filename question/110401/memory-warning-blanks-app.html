Memory Warning Blanks App

No matter what view I am on while on the iPhone device. Whenever I receive the following:

2011-02-08 23:07:16.750 TestApp[6111:307] Received memory warning. Level=2 [INFO] Due to memory conditions, 0 of 0 images were unloaded from cache.

The window will go solid white. Based on the output logs the app continues to run but nothing displays anymore. I have only run into this issue on the iphone device specifically. I'm using a 3gs. I have not had one single instance where my app didn't run out of memory and have this issue.

Any help would be fantastic,

  • Hello,

    in TiProxy.m you found:

    -(void)didReceiveMemoryWarning:(NSNotification*)notification {
    //FOR NOW, we're not dropping anything but we'll want to do before release
    //subclasses need to call super if overriden
}

    As you see titanium has a problem of memory because the message from system will be ignored.
