Remove border from button

I need to remove a button's border.

In Interface Builder, I can do this by changing the button's Type to 'custom'. This removes the rounded edges and leaves you with a white square.

I tried doing this:

var submitButton = Ti.UI.createButton({
    title:  'Create File',
    width:  '100',
    height:  '100%',
    title:  'Export Data',
    top:0,
    left:0,
    font:  {fontSize:  20,fontWeight:  'bold'},
    borderRadius:  0.0,
        borderWidth:  0

});

I also tried adding borderColor: 'transparent' and that didn't work.

Something interesting to note. When I set the borderColor to red, a square border appeared. The rounded edge lines did not change color.

Thanks

    borderColor: null,
borderRadius:0,
style:Titanium.UI.iPhone.SystemButtonStyle.PLAIN,
