  • Simply user the Properties API. This will save the value on the device even if your application is updated.

    ####Your code could look something like this:

    Ti.App.Properties.setString('yourUrl', userGiverUrl);

    Later in the code, you can retrieve this same value using the following code:

    var savedUrl = Ti.App.Properties.getString(yourUrl);

    Hope this helps
