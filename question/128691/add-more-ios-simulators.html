  • start your emulator, keep it in your tool bar then close it. Then open it not using studio and then in the options you can select which emulator you want to use.

    You cannot switch if you have opened it using studio as it won't remember the settings.

  • In Xcode, choose Xcode > Preferences.

    In the Preferences window, click Downloads.

    In Components, find the legacy simulator version you want to add, and click the Install button.
