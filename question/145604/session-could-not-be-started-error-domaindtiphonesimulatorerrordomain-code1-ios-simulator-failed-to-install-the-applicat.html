  • I solved it by using the following method: http://stackoverflow.com/questions/12932750/ios-simulator-failed-to-install-the-application

    Delete the contents of &#x2F;Users&#x2F;USER&#x2F;Library&#x2F;Application Support&#x2F;iPhone Simulator then rerun your project and it is fixed.

  • In my opinion, this is a memory issue. I run MAMP on my Mac. When it is running and I try to run an Appcelerator app, the simulator crashes with the error mentioned above. When I stop MAMP from running, the crash does not happen, and I can run the simulator over and over again without a problem. I would try looking at what else is running on your machine, and eliminate the things that take the most memory.
