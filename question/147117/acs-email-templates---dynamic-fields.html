ACS Email Templates - Dynamic Fields

I wonder if anyone sent Email Templates via ACS with Dynamic Fields.

I have created a Template (name: test subject: test) on the Web Portal with Template Body:

<!DOCTYPE html>
<html lang="en">
<head>
    ...
<body>
    <div id="box">
        <span>Dear {{first_name}},</span>
        <p>
            Dynamic Property: {{myDynamicProperty}}!<br />
        </p>
    </div>
</body>
</html>

In App: (Send Email Template API Link)

Cloud.Emails.send({
    template: 'test',
    recipients: 'joe@company.com',
    dynamic_fields: { myDynamicProperty: "Surprise!" }
}, function (e) {
    if (e.success) {
        alert('Success');
    } else {
        alert('Error:\\n' +
            ((e.error && e.message) || JSON.stringify(e)));
    }
});

In the Documentation (Send Email Template API Link) it says the property name is: dynamic fields with a space between the words, I think there is a misspell in the API Doc, maybe needs an underscore between the words?
