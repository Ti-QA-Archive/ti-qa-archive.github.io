orientationModes not working on iPhone

iPhone, SDK 2.1.4.GA

I have an app that allows Landscape and Portrait orientations but I want to open a windows that allows only Landscape orientation so I did:

createWindow = function () {

    generalWindow = Ti.UI.createWindow();
    generalWindow.orientationModes = [Titanium.UI.LANDSCAPE_LEFT];

    var win = Ti.UI.createWindow({
        //paramethers
    });
    win.orientationModes = [Titanium.UI.LANDSCAPE_LEFT];

    var masterNav = Ti.UI.iPhone.createNavigationGroup({
        window: win
    });

    generalWindow.add(masterNav);

    //Things...

    return generalWindow;

}

Then I open generalWindow, but it starts on Portrait (if the previous window was on Portrait) and only blocks itselft to be in Landscape after I change the orientation once. I also tried to set orientationModes only for generalWindow, doesnt work. Also only for win, doesnt work. Also tried to set the orientationModes in the createWindow metod, doesnt work. I dont know what else to try…

    You're using a NavigationGroup, which has to be inside of another window. Thus, the windows inside your navgroup will inherit the orientation support of that parent window. Do you require a navgroup? Could you use a different app structure? Maybe tabs or just a set of windows you open with buttons? If you must use the navgroup, maybe you should open your landscape windows as modal widows so that the "break out" of the parent window.

    BTW, is there a reason you're using SDK 2.1.4? Current is 3.1.
