  • SOLVED

    Additional details missing from the install instructions;-

    Put the unzipped module in
    1) /Library/Application Support/Titanium/modules/

    If there is no iPhone directory then create one.

    2) When adding to the tiapp.xml put it right before the closing tag, after both iPhone and Android sections.
    I had put it in the iphone section.
