Draw circle on annotation on a map view

Hi, I want to draw a range circle on an annotation, I was looking at the API but such feature it doesn't exists, Have anyone any idea how to accomplish this?, if I use a webview can I do it with HTML5?

  • If you want to draw lines, circles etc. you have to use a canvas, which is available as an external module. For Android use our module. For iOS use the Appcelerator module. An annotation can have a custom view since Titanium 3.1.
