OSX Lion & Xcode 4.1 - iOS SDK version 4.2 not found

Just upgraded to Xcode 4.1 for Lion to try and get my Titanium projects working again. I now receive the following error when trying to run my iPad app: iOS SDK version 4.2 not found. I've confirmed that SDK in tiapp.xml is set to 1.7.2. I've tried uninstalling and re-installing Titanium to no avail. Is this because Xcode 4.1 only includes iOS SDK 4.3? How do I get this working again?

  • Solved! Had to open the Run Configurations for the project and set the iOS SDK to 4.3 there. Can't believe not many others had this problem?!
