  • Why would you set the controller to null before you're done using the UI components within it? The instance of the window you're opening is contained in the controller. I'm surprised it works at all for you on iOS. Don't free any of your resources until you are done using them.

    Also, you've given extremely little detail. One snippet and saying "it's crashing" gives us very little to go on.
