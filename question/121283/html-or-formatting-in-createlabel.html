html or formatting in createLabel??

Hi All,
Is it possible to have basic html or formatting within the text of createLabel?
As per my example below (which doesnt work) I'd like to add e.g. html linebreaks in the text. Is that possible or should I use something else?

Thanks,
Lee

var l1 = Ti.UI.createLabel({
    text:'Line 1<br>Line 2<br>Line3',
    color:'#fff',
    width:'auto',
    height:'auto',
});

    var l1 = Ti.UI.createLabel({
    text:'Line 1\nLine 2\nLine3',
    color:'#fff',
    width:'auto',
    height:'auto',
});
