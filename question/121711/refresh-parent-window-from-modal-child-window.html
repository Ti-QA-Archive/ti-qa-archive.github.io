  • I'm not sure if you didn't paste your code right, but this is what it should look like:

    Ti.App.fireEvent('updateUser', {userData:"user data goes here"});

    and then

    Ti.App.addEventListener('updateUser', function(data){
   Ti.API.info( "the user information " + data.userData );
});

    this works 100%, I use it every day.

  • Problem solved:

    The problem was this

    Instead of calling Ti.App.Firevent

    I was calling var win = Ti.UI.currentWindow; win.fireEvent

    The correct code would be

    SetButton.addEventListener('click', function() {
    var sday = new Date(DatePicker.value);
    Ti.App.fireEvent('updateDate', {upDate: formatDate1(sday)});
    win.close();
});
