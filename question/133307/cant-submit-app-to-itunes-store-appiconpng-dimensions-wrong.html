Can't submit app to iTunes store: appicon.png dimensions wrong

I have submitted several apps in the past without issue. This is the first with 1.8.1 though. When submitting through Xcode, validation fails:

iPad: appicon.png: icon dimensions (0x0) don't meet the size requirements. The icon file must be 72x72 pixels, in .png format.

OK, I don't get it, as that file IS 72z72, PNG file. Any suggestions?

SDK 1.8.1, OS 10.7.3, Xcode 4.2, Studio 1.0.10.201202241328, IOS 5

  • You need three icons. 57x57 114 x 114 and 72 x 72..

    In the iPhone directory under resources set the 57x57 to icon.png the 114 x 114 to icon@2.png and the 72 x 72 to icon-72.png

    replace icon with your icon image name.
