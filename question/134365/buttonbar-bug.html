  • OK.. here.
    Create a buttonBar:

    

var myButtonBar = Titanium.UI.createButtonBar(
    {
    labels:['A', 'B', 'C', 'D']
});

    Then add this to a window

    win.add(myButtonBar)

    Now - do the following:

    myButtonBar.addEventListener('click', function(e)
{

    switch(e.index)
    {
    case 1: 
    /// do something here
    break;
...
continue until case 4

    case 4:
    Ti.API.info("This is D");
break
}
}

    —> What u will find is that A, B and D work well
    but not C
    If you put a Ti.API.info for case 2 (i.e. C) it will show you that C was pressed. That's great.. you will see that the variables you place in case 2 (i.e. for selection C) will not be treated. They will be ignored.
