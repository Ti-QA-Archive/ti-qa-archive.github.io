  • You can do a slightly hacky workaround

    add a language prefix to one file

    For example

    English

    <string name="en_1">
<string name="en_2">
<string name="en_3">

    French

    <string name="fr_1">
<string name="fr_2">
<string name="fr_3">

    Then in your code simply set up a flag for the language and add it into the localised string

    var defaultLang = "en_"
introText.text = L(defaultLang + "home_text");
