Alloy - Can't set internationalized OptionDialog->Option->Text with alloy

<OptionDialog id="optionDialogt">                
    <!-- The Options tag sets the options property. -->
    <Options>
        <Option text="L('txt_1')"></Option>
        <Option title="L('txt_2')"></Option>
        <Option> Cancel</Option>
    </Options>
</OptionDialog>

I can't set the Option text in tss file cause this property is CO (create only)

Is there any way to use alloy optiondialog with internationalization ?

    You can set it in the TSS and it should set it just fine. Also, XML attribute localization is supported in the upcoming Alloy 1.2.0 release. You can try the alpha now to see if it suits you.

    sudo npm install -g alloy@1.2.0-alpha5
