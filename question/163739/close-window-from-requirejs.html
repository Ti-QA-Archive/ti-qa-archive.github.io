Close window from require.js

Hi!

I have xml file like this:

view.xml

<Alloy>
    <Window id="win" class="container">
        <Require src="require_view" type="view" id="require_view"/>
        <View id="some view">
....
        </View>
    </Window>
</Alloy>

require_view.xml

<Alloy>
    <View id="req_view">
<Button onClick="close">close</Button>    
    </View>    
</Alloy>

require_view.js

function close(){
//need to close window
}

When i push button I need to close window with id="win", but I can't get access to this window:(
Could someone suggest me how can I do that?

    Hi John,
    Please find the below sample on how you can achieve this :
    In view.js you need to pass the win object to require_view controller in order to close it from there. You can do this as mentioned below :

    view.js

    $.require_view.win=$.win;

    Now in your require_view.js file you can close the window as mentioned below :

    require_view.js

    var self = this;
function close(){
self.win.close();
}

  • It is possible by two way either you create global function in view.js or either you add view from the view.js and pass callback function as a argument e.g

    var reqView = Alloy.createController("require_view",$.win).getView("req_view");
    $.win.add(reqView);

    and get argument in require_view file e.g

    var win = arguments[0];

    function close(){
    win.close();
    }
