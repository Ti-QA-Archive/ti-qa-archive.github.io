    You need to host the music file somewhere, AWS is usually a good place to start if you don't have any other place to host. The url property needs the direct url to the mp3 file that you wish to play. Most music websites use urls that redirect a number of times before playing, specifically to stop people from people hotlinking to their music. You really need to either find a place to host your mp3 or to find one that is hosted and can be streamed.

    • The Ti.Media.Sound object is for playing basic audio resources that are locally loaded into memory. You said originally that you were using the Ti.Media.AudioPlayer API, but in your code you are using the Sound object. If you need streaming support, you must use the AudioPlayer API. See the docs here: http://docs.appcelerator.com/titanium/2.1/index.html#!/api/Titanium.Media.AudioPlayer

      Also, if you copy/paste that url into a web browser, you will see that the token is expired and the file is inaccessible. You'll need to go into your AWS console, pull up the file and make it public accessible. Once its public, there will be an option to get a public url. Use that url in your code.

    • Here is an audio example that works. The audio file is an mp3 being served using my own personal S3 bucket.

      
var url = "https://s3.amazonaws.com/com.anthonydecena.media/music/UnwrittenLaw/Elva/06+Seein%27+Red.mp3";

var win = Titanium.UI.createWindow({  
    title:'Audio Test',
    backgroundColor:'#fff',
    layout: 'vertical'
});

var startStopButton = Titanium.UI.createButton({
    title:'Start/Stop Streaming',
    top:10,
    width:200,
    height:40
});

var pauseResumeButton = Titanium.UI.createButton({
    title:'Pause/Resume Streaming',
    top:10,
    width:200,
    height:40,
    enabled:false
});

win.add(startStopButton);
win.add(pauseResumeButton);

// allowBackground: true on Android allows the 
// player to keep playing when the app is in the 
// background.
var audioPlayer = Ti.Media.createAudioPlayer({ 
    url: url,
    allowBackground: true
});           

startStopButton.addEventListener('click',function() {
    // When paused, playing returns false.
    // If both are false, playback is stopped.
    if (audioPlayer.playing || audioPlayer.paused) {
        audioPlayer.stop();
        pauseResumeButton.enabled = false;
        if (Ti.Platform.name === 'android') { 
            audioPlayer.release();
        }   
    } else {
        audioPlayer.start();
        pauseResumeButton.enabled = true;
    }
});

pauseResumeButton.addEventListener('click', function() {
    if (audioPlayer.paused) {
        audioPlayer.start();
    } else {
        audioPlayer.pause();
    }
});

audioPlayer.addEventListener('progress',function(e) {
    Ti.API.info('Time Played: ' + Math.round(e.progress) + ' milliseconds');
});

audioPlayer.addEventListener('change',function(e) {
    Ti.API.info('State: ' + e.description + ' (' + e.state + ')');
});

win.addEventListener('close',function() {
    audioPlayer.stop();
    if (Ti.Platform.osname === 'android') { 
        audioPlayer.release();
    }
});

win.open();
