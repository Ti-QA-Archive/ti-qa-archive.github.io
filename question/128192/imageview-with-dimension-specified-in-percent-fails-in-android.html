ImageView with dimension specified in percent fails in android

Hi,

I have an imageView and I want to set width and height in percent like this

var imageView = Ti.UI.createImageView({
    image:'../KS_nav_ui.png',
    width:'10%',
    height:'10%',
    right : 5
});

This works fine in IOS but fails for android giving this exception :

java.lang.IllegalArgumentException: width and height must be > 0

Is there something missing in my code? or Is this just impossible in Android?

Is there a workaround to set image dimension in percent in android?

Thanks a lot

Ben

    Benjamin

    It's possible to obtain proportional dimensions by only setting one, such as the width, as follows:

    var win = Ti.UI.createWindow({
    backgroundColor:'#ddd',
    fullscreen:false,
    exitOnClose:true
});

var imageView1 = Ti.UI.createImageView({
    image:'KS_nav_ui.png',
    top:10,
    width:50,
    left:5
});

var imageView2 = Ti.UI.createImageView({
    image:'KS_nav_ui.png',
    top:10,
    width:'50dp',
    left:105
});

var imageView3 = Ti.UI.createImageView({
    image:'KS_nav_ui.png',
    top:10,
    width:'50%',
    left:205
});

win.add(imageView1);
win.add(imageView2);
win.add(imageView3);
win.open();

    Hope this helps
