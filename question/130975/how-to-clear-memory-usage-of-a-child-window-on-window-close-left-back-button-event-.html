How to clear memory usage of a child window on window close (left back button) event ?

I have no problem to clear window memory if it's a modal window. But, if it's a window created from another parent one through a tableView click event, I can't get it cleared from the memory, it stays in the TiUIWindowProxy column :

In my parent window, I tried to add a close event in order to clear the child window but it's not working :

tableView.addEventListener('click', function(e) {
    var title = e.row.titleView;
    var newWin = Ti.UI.createWindow({
        title:title,
        url:'childWindow.js',
        backButtonTitle:'Back'
    });

    Ti.UI.currentTab.open(newWin, {animation:true});
    //
    newWin.addEventListener('close', function() {
        Ti.API.info('newWin closed');
        newWin = null;
    });
});

Each time I open the newWin window, it's added to TiUIWindowProxy and never wants to decrease the value of the column when I close the newWin window.

What's wrong with my code ?
