  • You will need to use an older version of the Titanium SDK if you want to build applications compatible with iOS 4.2.

    ###As you can see on the Compatibility Matrix

    Looks like Titanium SDK version 3.0 could fit you bill.

    There is one caveat though. You may encounter problems when submitting the application to the App Store though. Because, if I recall, Apple doesn't accept applications below iOS 4.3 anymore.
