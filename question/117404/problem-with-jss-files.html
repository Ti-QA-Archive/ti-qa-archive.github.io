Problem with JSS files...

So I am trying to implement a "global.jss" file to my app, but whenever I create it and name it "global.jss", it says Dashcode cannot open the file because it is a "JACOSub Subtitle File" file, and the only recommended application to open it is VLC Media Player.

Any help? I have no idea how I get this to work :-S

PS - I'm on a MacBook Pro 13-inch 10.6.6.

  • Select the .jss file in the Finder and use the Finder menu options (from the top of the screen) File - Get Info then look for the Open With section.

    Just choose your text editor for the .jss file format, when you double click the file it will be associated with that program from then on.
