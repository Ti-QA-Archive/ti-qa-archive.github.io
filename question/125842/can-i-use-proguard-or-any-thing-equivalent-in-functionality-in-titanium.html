  • Having spoke to some Ti guys just now I can tell you that Titanium doesn't use ProGuard yet, but it may come along in a future version. Ti will byte encode your JS scripts first, though, so at least all that code is somewhat obfuscated. You could try turning your Ti project in to an Eclipse project, then build and deploy via Eclipse and get ProGuard involved for free. Take a look at this link that the Ti guys shared with me:

    https://github.com/billdawson/tidevtools

    I havent used any of these yet, having just been given the information myself, so I have no idea if they'll work.

    Good luck - let me know if you have any problems
