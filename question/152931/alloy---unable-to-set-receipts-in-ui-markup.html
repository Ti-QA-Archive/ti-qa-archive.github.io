    No, you can't do that. XML attributes are strings, you can't just embed JS objects and arrays in them. You should be assigning this using the TSS files.

    view.xml

    <Alloy>
    <EmailDialog id="emailDialog"/>
</Alloy>

    view.tss

    "#emailDialog": {
    subject: "Receipt Report",
    messageBody: "<b>Please find the attachment for detail</b>",
    toRecipients: ['xxxx@gmail.com']
}
