Air print market place module styling

Hi, we are using the marketplace airprint module (the one from appcelerator)

is there is any way to set colors/styles on the print popup. It’s got that purplish gray color, would be nice if we could get the gray to match our application design. I didn’t see an option in thedocumentation.

    There is no way to change the style on the airprint popup. The popup isn't actually created by the module, it is just passed along from Apples Airprint API and that is how it is styled.
