    • That's how I was trying to do it too - but apparently, it is not working for me :-(
      How did you do it??

      My code was as simple as
      tt.ui.carAnnotation = Ti.Map.createAnnotation({
      latitude : cloc.coords.latitude,
      longitude : cloc.coords.longitude,
      title : 'car',
      image : tt.ui.CAR_IMAGE
      });
      and then
      tt.ui.carAnnotation.setLatitude(currLoc.coords.latitude);
      tt.ui.carAnnotation.setLongitude(currLoc.coords.longitude);
