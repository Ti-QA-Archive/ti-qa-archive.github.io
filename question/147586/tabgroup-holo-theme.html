Tabgroup Holo Theme

Im using Ti. 3 and with the xml down here I can change the tabs into Holo theme. Is it possible to adjust the width of the tabs so it wont get a swipe one if 5 tabs? The same comes to font and the blueish selected color, are they editable?

<android xmlns:android="http://schemas.android.com/apk/res/android">
        <tool-api-level>14</tool-api-level> <!-- usata per compilare -->
        <manifest>
            <application android:theme="@android:style/Theme.Holo"></application>
        </manifest>
</android>

    It's down to the screen width. The system decides wether your navigation tabs are fixed or scrollable depending on the available space. You can not adjust the width. I tested this with the Android API Demos: if the screen is wide enough it will show even more than 5 as fixed tabs (e.g. in landscape).

    The fonts and colors are defined by the Holo Theme. To edit them, define your own theme by extending the holo theme. Check out this style generator for fast and easy customizing
