Bold words inside Label Text

I'd like to make selective words in label text bold, as headings for paragraphs. My workaround is making separate labels, but identifying the 'top' location for each label is terrible, especially when modifying the text. Tried using some escape sequences but none worked.

    I believe that the only options you have currently is to either use a separate label for each text style and for those labels to be contained within a parent view which has a vertical layout (layout:&#39;vertical&#39;), or to put all the text in a webview with styling applied.

    Hope this helps

    • I briefly considered the webview, but prefer to stick with the label. Do you know of any way to avoid specifying an explicit "top" location for each label? When I omitted the "top" specification the labels just stacked up.

      — commented February 21st 2011 by Software Services

    • When you set the layout:&#39;vertical&#39; property on the labels' parent view, then the top property is changed from an absolute to a relative distance between the sibling views (ie the labels).

      — commented February 21st 2011 by Paul Dowsett
