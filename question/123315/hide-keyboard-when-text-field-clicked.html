hide keyboard when text field clicked?

I have a view I show with a date picker when a user clicks a text field. That all works fine but the keyboard pops up too. How can I prevent the keyboard from showing on that text field only. I want the keyboard for my other text fields to work on the rest of the window. Thanks!

    Add a click event listener to the field, and in it do fieldName.blur()
