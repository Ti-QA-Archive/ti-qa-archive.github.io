Hi
I am new to mobile app developement and am building a small app using titanium, where i need to store data in database. I am using sqlite. The code is as follows.

Titanium.Database.install('../Resources/MyDataBase.db','listdb');

var db= Titanium.Database.open('listdb');

db.execute('INSERT INTO AddressBook(name, phone) VALUES(\"'+textField.value+'\",\"'+textField1.value+'\")');
