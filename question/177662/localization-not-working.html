Localization not working

Hello, I am trying to create an app that uses titanium localization as mentioned in the documentation, I used the exact steps suggested in the documentation but still it won't work.

This is the my directory structure:

Here is string.xml code:

<?xml version="1.0" encoding="UTF-8"?>
<resources>
    <string name="welcome_message">Welcome</string>
</resources>

JS that creates the button :


var newBtn = Ti.UI.createButton({
    title : L('welcome_message' , 'testing'),
});
$.mainView.add(newBtn);

I am using Titanium SDK version 3.3.0.GA with an emulator for android version 4.4.2 , and I tried my device I have a samsung galaxy s4 with 4.4.2 installed.

I have looked everywhere to see if I missed something but I can't find anything wrong with what I did. I tried cleaning the builds , I tried changing the folder name from en to en-US because that's the locale of the device, but nothing helps. I still see testing as the title of the button and not 'Welcome'.

I did see a lot of forum pages discussing the localization issue but I couldn't find a solution please help !

and thanks for everyone in advance.

    The i18n folder must go in your project's root directory, not within the app folder. So, move it up one level in the directory tree.
