Android actionbar tabs with fixed width

Hello,
I have a tabgroup which is displayed as an actionbar in Android.

The tab items are scrollable by default, this means when there is not enough space to display all the tabs I have to scroll to the right to see the last tabs.

Does anyone know how to get fixed tabs (all the items have the same width=screenwidth/number of tabs)?
See this description for the difference between scrollable and fixed tabs (http://developer.android.com/design/building-blocks/tabs.html)
