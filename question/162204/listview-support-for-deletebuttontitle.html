ListView support for deleteButtonTitle

In Titanium 3.2.0, the ListView does not seem to support custom titling of the button for deleting items, using the deleteButtonTitle property (which is an undocumented value for the TableView). Is there an alternate way to retitle the button that I am missing?

— asked February 3rd 2014 by Daniel Gump

  • Not all the features of a TableView have been implemented on a ListView yet but it's getting there. I was hoping this might have been included in the recent 3.2.1.GA release. I've created a JIRA ticket with an improvement feature that 'deleteButtonTitle' could be applied to a ListItem rather than the whole of a single ListView, this would allow us to have different custom titles per row… if needed.

    Ticket: https://jira.appcelerator.org/browse/TC-3747
