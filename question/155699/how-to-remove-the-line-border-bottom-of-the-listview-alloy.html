How to remove the line border bottom of the listview alloy?

I have a listview use templates

<Alloy>
    <Window>
        <ListView>
            <Templates>
                <ItemTemplate name="myTem" height="50">
                    <Label bindId="label" class="itemBind"/>
                </ItemTemplate>
            </Templates>
            <ListSection id="section"/>
        </ListView>
    </Window>
</Alloy>

When project builded and run, there is a line border on bottom of each row.

How can i remove them, change style (white color to red color …) or add left, right line border?

  • Now it is supported since 3.2

    http://docs.appcelerator.com/titanium/3.0/#!/api/Titanium.UI.ListView-property-separatorColor

    just set it to transparent like:

    var listView = Ti.UI.createListView({
    separatorColor : "transparent"
});

  • Unfortunately, separatorColor is not yet implemented in ListView, which would be the property you would use to change that. It is being added in the 2nd revision of the ListView API, which is currently slated for Titanium 3.2.0. You can follow the details of the ListView rev2 changes here: https://jira.appcelerator.org/browse/TIMOB-13244
