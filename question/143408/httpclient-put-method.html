HTTPClient PUT method

Hi,

I have to call a PUT request, When call it via php curl it works fine,
Here is my sample php code

$base_url = 'http://xxxx.com/?articleID=392';
$body = '{"comment":"sample text"}';
$c = curl_init($base_url);
curl_setopt($c, CURLOPT_POST, true);
curl_setopt($c, CURLOPT_POSTFIELDS, $body);
curl_setopt($c, CURLOPT_HTTPHEADER, array('X-HTTP-Method-Override: PUT'));
curl_setopt($c, CURLOPT_RETURNTRANSFER, true);
$result = curl_exec($c);
print_r('<br>result:<br>');
print_r($result);
curl_close($c);

But When call the same PUT request via Titanium I am getting error(server not recognize the PUT request)

var url = 'http://xxxx.com/?articleID=392';
var client = Ti.Network.createHTTPClient({
    onload : function(e) {
        Ti.API.info("Response text: \n" + this.responseText);
        alert('success');
    },
    onerror : function(e) {
        Ti.API.info("error text: " + e.error);
        alert('error');
    },
    timeout : 10000 // in milliseconds
});
client.open("PUT", url);
client.setRequestHeader('X-HTTP-Method-Override', 'PUT');
client.setRequestHeader('data', '{"comment":"sample text"}');
client.send();

How to pass "POSTFIELDS" in titanium PUT request?

My Titanium SDK Version 2.1.3. Thanks in Advance
