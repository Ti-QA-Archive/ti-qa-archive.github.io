Android Background Audio

I have a cross-platform application which supports streaming for a couple of sources. I have successfully gotten this to keep playing on iOS even if the app is paused, but unfortunately I have been unable to get this to work on Android.

In my app.js file I have:

Titanium.Media.audioSessionMode = Titanium.Media.AUDIO_SESSION_MODE_PLAYBACK;

and the code for the player is:

var player = Titanium.Media.createAudioPlayer({allowBackground:true});

However, when I press the Home Button on Android the streaming immediately stops. Is there a way to get this to keep playing on Android? Has anybody gotten this to work?

PS. I'm using 1.6 but have been tried with 1.5.x also.

    I found the answer to my own problem. I was originally creating a global player for all my streams that was instantiated at the beginning of my program. However, I had 2 problems with this 1) The Lack of background audio, and 2)the fact that if I tried to change the url of the stream the player would not update and continue playing the original. So, I changed the way this worked by creating a new player each time with the allowBackground set to true and the url passed every single time and voila, it works playing the correct stream and allowing background audio.

    I hope this helps!
