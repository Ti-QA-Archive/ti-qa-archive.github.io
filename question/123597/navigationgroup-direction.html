  • Yes you can, by setting the leftNavButton = null on your window and create a custom button with a click event attached to it where you will close the window, and assign it to the rightNavButton.

    var closeButton = Ti.UI.createButton({
    title:'Back',
    width:100,
    height:30
});

closeButton.addEventListener('click', function() {
    win.close({
        duration: 300
    });
});

win.rightNavButton = closeButton;

    This is just a sample, you can create the look of the button how you want it.

    Hope this helps!
