Titanium cloud service connect to a website application

Is it possible to connect a website application to the titanium cloud service? Say I want to send some information from a website form to my titanium cloud service, store it, and then retrieve that information to be used in a mobile application.

— asked February 20th 2013 by Alexander Cam Liu

    Yes this can be done. Your web page needs to communicate using the REST API
    http://cloud.appcelerator.com/docs/rest

    This will be possible by sending the application key parameter over SSL with your requests
    http://cloud.appcelerator.com/docs/authentication

    more about the API can be found here:
    http://cloud.appcelerator.com/docs/api/v1/acls/create
