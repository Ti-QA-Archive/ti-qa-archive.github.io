  • To manually change a user's password

    1. login to my.appcelerator.com
    2. hit up My Apps
    3. find your app
      4.click 'manage acs'
    4. authenticate again if required
    5. check to see if you are in production/development (top right corner)
    6. Select users from the list of object types in the right pane
    7. Expand details for user in question, click change user password.

    Now to your questions

    Why I can not see password in my user of my project on cloud.

    Because that be pretty farking stupid to be returning a user password BACK to them after they have authenticated.

    Where is the password?

    Securely stored on the backend

    How I can check the password?

    Thank the deity you can't, because with your warped view of security, bad things could happen.

    How code to do work?

    There isn't any. You can change the password manually or via code (Request Reset Password)
