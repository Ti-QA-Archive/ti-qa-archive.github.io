What is the terminal window for?

Hi,

I installed titanium studio 2.1 on win 7. I cannot see any videos or guides re the terminal window and do not really see how to use it and what to use it for.
Can someone help? Indeed, I'd like to find a way to improve the painful debugging experience of an android app and that terminal will certainly have some use here.
Tks.

    The terminal just simply makes it easy to run things like adb or the android device manager from the command line. You can also build using the command line to if you'd like, though its quite a bit easier to use the existing Studio function if you're already in it.

    Some apps may use a local node.js backend for testing and development and access to the command line simplifies that a bit in some cases.

    All the terminal is for is quick and easy access to the command line so you don't have to bounce back and forth between the terminal.
