Opening company facebook page in native facebook app (ios)

OX 10.9, iOS 8.1, Running app on actual iPhone 6

I have a row "Like us on Facebook" in settings page.
When I click on this row, it is opening my facebook app, but it is not opening respective company facebook page. I want to give a user to like my company facebook page.

I am using default facebook module.

Example: var config.fbURL = https://www.facebook.com/appcelerator

rowFB.addEventListener('click', function(e) {
        Titanium.Platform.openURL(config.fbURL);
    });

Any help really appreciate. If you need any more information from my side , please let me know.

    You used to be able to do something like this on iOS (you will need to test Android yourself):

    Ti.Platform.openURL('fb://page/20594753592');

    It was then changed to:

    Ti.Platform.openURL('fb://profile/20594753592');
