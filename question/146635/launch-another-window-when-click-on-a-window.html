launch another window when click on a window

hi everyone,

so i have a tabview in my app, and when people click the fourth tab, they should be redirected to a fullscreen window which has a little game in it, so right now at the fourth tab i have a button which launches the game (lbutton)

    if (osname=="mobileweb")
    {
     lbutton.addEventListener('click', function(e){
      errorwindow.open();
     }) 
    }
    else if (osname=="android")
    {
     lbutton.addEventListener('click', function(e){
      anderrorwindow.open();
     }) 
    }
    else
    {
     lbutton.addEventListener('click', function(e){
      Ti.include('world.js', 'ui.js');
      app.ui.startGame();
      app.ui.createStartWindow().open();
     }) 
    }

does anybody know a way to launch it directly when you click the tab?????

  • myTabGroup.addEventListener('click', function(e){
    switch(e.source.activeTab.title){
    case 'My4rthTabTitle':
        myfullscreenWindow.open();
        break;
    }
}

    My example uses the title, as it allows for tab groups of dynamic lengths. If you are set at 4, then use the index of the activeTab in your switch instead.
