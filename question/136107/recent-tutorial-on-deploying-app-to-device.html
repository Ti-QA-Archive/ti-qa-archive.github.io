  • Accepted Answer

    • Login to developer.apple.com
    • Hit the iOS Provisioning Portal
    • Hit Devices
    • Add Device (will need to get the device id)
    • Now hit up Provisioning
    • If you haven't already, create a Development Provisioning Profile for the App ID you are wanting to use.
    • Under that Development Profile, you check the Device you wish to test on.
    • Download the Development Profile and import into xCode with a double click in finder.
    • Re-build your project using the Development Profile.
