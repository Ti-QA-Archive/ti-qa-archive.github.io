Couldn't start windows simulator

what is the latest version of Titanium SDK . I am trying to have the windows preview .When i try to install titanium latest sdk from github using
'ti sdk install -b master -d' it downloads 4.0.0 automatically.

And then if i try to run any window application then i am getting error :

'[ERROR] Failed to start emulator "8.1-1" [ERROR] The application is not installed.'

  • To properly test the Windows platform, you need both the recent SDK and the newest CLI. The command you listed above installs the newest nightly build of the SDK (though not the master branch from GitHub). You will also need the latest CLI – 'npm install -g git://github.com/appcelerator/titanium.git'

    While you're at it, check the steps from our Windows preview blog post to make sure you have the rest of the requirements set up.

    Also, run 'ti info' and make sure that it can see your Windows emulators.

    If you've run through all those steps (and you're sure you've done so correctly) and you're still having problems, please submit a Jira ticket in which you clearly and completely document your system's environment, the setup steps you've taken, and the error you're getting.
