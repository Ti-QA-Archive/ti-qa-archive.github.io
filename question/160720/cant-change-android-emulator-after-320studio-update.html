  • With Titanium SDK 3.2, you can create and use your own Android Emulator. It doesn't have to be named anything special. You may need to reset Titanium Studio for it to show up, but you should be able to select your custom AVD from the list.

    To see the list of AVDs, you first must select the project you want to build in the "Project Explorer", then from the toolbar, make sure "Run" is selected from the first drop down menu, then under the second drop down menu should be "Android Emulator" that has a submenu of your AVDs. Are you able to see your AVD?
