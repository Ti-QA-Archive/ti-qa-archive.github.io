  • Accepted Answer

    One note is that 3.1.0 is the Titanium SDK version. You can get the Alloy version by typing alloy --version at the command line. You can get the Studio version by going to its "about" menu.

    If that is all you are seeing in your output then there's a problem with Studio's output. I have heard this mentioned on Windows systems before. Mentioning your OS is also helpful when trying to get help. Just to be clear that this is a studio error, can you do the following:

    1. put your code back in the error state
    2. run through studio again and confirm that you are not getting any output past the alloy logo
    3. Go to the command line and go into your project's directory
    4. execute this: alloy compile --config platform=android, assuming you are targeting android

    Do you get better output this way?
