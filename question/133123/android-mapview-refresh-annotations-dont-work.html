Android mapview refresh annotations don´t work

I´ve got this code that´s located in same window:


var mapview = Titanium.Map.createView({
        mapType: Titanium.Map.STANDARD_TYPE,
        //region:region,
        animate:true,
        regionFit:true,
        userLocation:true
        //annotations:anotacionesList
});

self.add(mapview);

var mapupdated = function()
{
 mapview.region = mapa.getRegion();
 mapview.annotations = [];
 mapview.annotations = mapa.getAnnotations();
 mapview.selectAnnotation(mapa.pin);
}
Ti.App.addEventListener('map.updated', mapupdated);

Ok, when an event is fired i need to change annotations in map, i call maps.getAnnotations() and returns me an array with new annotations, in debug mode i see that mapview.annotations fills correctly with new annotations, but map in emulator doesn´t change annotations. however region seems to change in emulator view.

what´s wrong? i need to create a new map everytime i need to change annotations?
