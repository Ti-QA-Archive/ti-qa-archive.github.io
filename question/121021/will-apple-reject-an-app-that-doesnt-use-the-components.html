  • Hi,
    Unfortunately the findme icon/system button is not included.
    but you can easily create one that's identical. I don't think it will cause any issues. (But don't quote me on that ;) )

    Here are some icon images (normal and @2x) : http://mobods.com/test/find_me_icon@2x.png and http://mobods.com/test/find_me_icon.png

    Then use :

    
var find_me_button = Titanium.UI.createButton({
    image:'find_me_icon.png',
    style:Titanium.UI.iPhone.SystemButtonStyle.PLAIN
});
