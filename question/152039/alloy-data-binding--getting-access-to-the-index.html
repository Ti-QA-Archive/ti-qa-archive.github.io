Alloy Data Binding : Getting access to the index

How do I get access to the index number of the row rendered? Take for example the situation below what could I add in row.js or row.xml to allow $.indexNumber.text to equal 1 for first row, 2 for the second row etc…

index.xml

<Alloy>
  <Window id="window">
    <TableView id="table" dataCollection="myModel">
      <Require src="row"/>
    </TableView>
  </Window>
</Alloy>

row.xml

<Alloy>
  <TableViewRow class="row">
    <Label id="indexNumber" text="{index}"/>
    <Label class="rowTitle" text="{title}"/>
  </TableViewRow>
</Alloy>

    I'd try something like the following in the controller:

    var row_index = 1;

    and in the transformFunction(model) 

    transform.index = row_index;
row_index++

    as I understand that the transformFunction is called on any model in you collection, you'd find the row index number in the .index property of the transform object for each model displayed in the textfield.
