Selecting multiple rows and automatically move them to the top of the rows

Hi guys. I have trouble constructing a concept when users click on a row, the row will go to the very top of all rows. When users click on the selected rows, the row will go back to the original position.

Here is my concept that I have developing over a day:

When user select a row,

  1. It will grab the rowIndex(custom property) and push into an empty array and create a tableViewRow based on rowIndex.

  2. Then, remove all the rows(that are not selected) and create new rows for it but excluding the selected ones.

However, I believe that there will tons of crazy algorithm to develop that…

It would be great if you can help me out!

Much thanks,
Harrison

  • It's not as crazy as you think, add the following function to your code:

    function sortBy(a, b) {
    if (a.isGreen && b.isGreen) {
        return 0;
    }
    if (a.isGreen) {
        return 1;
    }
    return -1;
}

    Then the click event needs to be:

    tableview.addEventListener('click', function(e) {
    if (e.source.isGreen) {
        e.source.isGreen = false;
        e.source.backgroundColor = '';
    }
    else {
        e.source.isGreen = true;
        e.source.backgroundColor = 'green';
    }
    var data = tableview.data[0] ? tableview.data[0].rows : [];
    data.sort(sortBy);
    tableview.data = data;
});

    You could use selected instead of isGreen as I wasn't sure if that was for something else.
