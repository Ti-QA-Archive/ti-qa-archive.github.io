Android SearchView icon

  1. Application type: mobile
  2. Titanium SDK : 3.0.2.GA
  3. Platform & version: Android 4.0
  4. Device: Android emulator , Asus transformer
  5. Host Operating System: Ubuntu 11.10
  6. Titanium Studio: Titanium Studio, build: 3.1.0.201303091040

Hi all,

I have implemented the "search view" in my application like this : 


var search = Ti.UI.Android.createSearchView({
    hintText: "Ricerca ead"
});

self.activity.onCreateOptionsMenu = function(e) {

        var menu = e.menu
        var menuItem = menu.add({
            title: 'Table Search',
            actionView : search,
            icon:  Ti.Android.R.drawable.ic_menu_search,
            showAsAction: Ti.Android.SHOW_AS_ACTION_IF_ROOM | Ti.Android.SHOW_AS_ACTION_COLLAPSE_ACTION_VIEW
        })
}//end onCreateOptionsMenu

where "self" is my window. I have a problem with icon because i want it in "holo light theme" but it is showed in "holo dark theme", i have set my tiapp.xml :

<android xmlns:android="http://schemas.android.com/apk/res/android">
   <tool-api-level>14</tool-api-level>
   <manifest>
      <supports-screens android:anyDensity="false"/>
      <uses-sdk android:minSdkVersion="11"  android:targetSdkVersion="14"/>
      <application android:theme="@android:style/Theme.Holo.Light"/>         
    </manifest>
</android>

how can i do if i want search icon in "holo light theme"? Any suggestions? Thanks.
