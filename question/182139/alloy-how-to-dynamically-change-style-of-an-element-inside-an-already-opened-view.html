  • HI Geppoz,

    You can do like this :

    var style = $.createStyle({
        classes: 'label1',  // id of controller
        apiName: 'Label',
        color: 'blue'
    });
$.label1.applyProperties(style);

    for more information refer this documentation Dynamic_Styles.
