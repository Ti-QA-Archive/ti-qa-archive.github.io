  • Hello,

    It seems you don't update everything that requires to develop app on 3.4.1 version. In Titanium Studio, went to Help –>Check for Titanium Updates, and see if anything you missed.

    You need to update all the items below along with the studio and Ti SDK and commands to update those are,

    • Alloy - command sudo npm update -g alloy
    • node.ACS - command sudo npm update -g acs
    • Titanium CLI - command sudo npm update -g titanium

    Then restart studio and check that everything are updated.

    If you want to remove titanium studio, just delete /Applications/Titanium Studio.

    Hope this will help you to solve your problem.

    Thanks

  • After 3 days of looking for the problem I found the solution in the output of sudo npm install -g titanium

    For anyone else having the same issue, the output of sudo npm install -g titanium will give you the correct PATH to include in your environmental variables.

    I did that and everything seems to work now.
