  • Do the following

    in the xml file: 

    <TableViewRow id="row" layout="horizontal" width="Ti.UI.FILL" height="[specify height]">
  <View class="horizontalView" width="25%" height="[specify height here, don't exceed row height]">
  <View class="horizontalView" width="25%" height="[specify height here, don't exceed row height]">
  <View class="horizontalView" width="25%" height="[specify height here, don't exceed row height]">
  <View class="horizontalView" width="25%" height="[specify height here, don't exceed row height]">
</TableViewRow>

    setting width at 25% means that you will not more than 4 items horizontally.
