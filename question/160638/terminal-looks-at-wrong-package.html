Terminal looks at wrong package

Hi All

I have just installed tishadow globally, which it drops the module into
&#x2F;usr&#x2F;local&#x2F;lib&#x2F;node_modules&#x2F; and it is the latest version V2.3.0.

However, when I try tishadow -v, it gives me the wrong version number 1.3.2. Seems tishadow has installed before and terminal looks at the wrong version. which tishadow prints usr&#x2F;local&#x2F;bin&#x2F;tishadow

How can I fix this problem?

    Double check where the symbolic link is pointing to. I.e.:

    $ ls -l /usr/local/bin/tishadow

    • Using npm:

      $ cd /usr/local/lib/node_modules/tishadow
$ sudo npm link

      or directly

      $ cd /usr/local/bin
$ sudo ln -s ../lib/node_modules/tishadow/cli/tishadow tishadow
