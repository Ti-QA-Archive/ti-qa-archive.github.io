Is it possible to use an application generated by Mobile Web SDK RC1 with Desktop SDK?

Hi,

With Titanium 2.0 and Mobile Web SDK RC1 released earlier this week, I wondered if it was possible to build a desktop app using the Desktop SDK using a mobile Web app generated by the Mobile Web SDK.

Thanks.

    Mobile Web SDK has good support for Webkit based browsers. I don't know how old Desktop's Webkit version is, but it's pretty safe to say that yes, a Mobile Web app should work just fine in a Desktop app.
