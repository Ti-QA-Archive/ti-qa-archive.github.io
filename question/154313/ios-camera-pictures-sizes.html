iOS Camera pictures sizes

Hey,

I have some issues with the camera and pictures sizes.

A quick explanation:

  1. I have a button when I press it I activate the camera
  2. The user takes an image and chooses to use the image

Now I have a space in the screen (a view were I want to set the taken image). The space are 320widthx400height.

So what I want to do is:

  1. I want to place the taken in that view but I dont want to destroy or crop the image in any way (I can resize the views width and height, but dont know which width and height to set).
  2. When I save the image I want it to be in the original size from where I took it the first time.

I have tried some solutions but I can´t get it to work.

Does anyone have a clue?

Appreciate your answers!

    Hi

    Easier than you think, just create a new image based on the original image;

    var croppedversion = imagefromcamera.imageAsResized(320, 400);
    • imageAsResized

    The croppedversion version of the image is a new distinct version of the one returned by the camera, so you now have two in memory.

    So you display the croppedversion and save the imagefromcamera.

    • You are saving the cropped image to the gallery - this the problem.

      This line takes the cropped image you supplied from the camera;

      var f = imageView.toImage();

      You need to save the original image to the gallery. You can do this one of two ways;

      • Create an imageView that is hidden and save the ORIGINAL image to that, if the user chooses to save the file use that imageView.
      • Save the ORIGINAL image to a file system file as a cached file (or a temp file), then is the user chooses to save the file use that file
