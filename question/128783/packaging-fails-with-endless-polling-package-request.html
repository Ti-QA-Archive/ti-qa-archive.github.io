Packaging fails with endless "polling package request"

With the latest Studio version (Build: jenkins-titanium-rcp-master-40) from the appcelerator login area it endlessly displays "Polling Package Request". Without any result or error. (On the app-page there is no packaged app found). This problem occurs when packaging for Linux. When packaging for Windows it breaks with the error "Couldn't find release package at C:\cygwin\home\Administrator\titanium_distribution\releases\win32\1.2.0\win32". No chance when transferring the project to another Studio-Installation.

Can anybody help me?

    Andre,

    There is no 1.2.0 on the servers, you must have built that locally as there was never one released to date. I would use the RC6e release candidate, or RC4, both of those are on the packaging servers. You can get RC6e at the following links.

    1.2.0.RC6e for win32

    1.2.0.RC6e for osx

    1.2.0.RC6e for Linux i386

    1.2.0.RC6e for Linux x86_64
