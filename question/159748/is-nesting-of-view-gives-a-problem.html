  • Hi

    The ListView is NOT designed to handle views and click events inside those views. For this you would use a TableView.

    • A ListView is designed to have templates that are updated by data and are not directly interactive
    • A TableView is designed to use views and can be directly interacted with and respond directly to events.

    Directly from the documentation;

    • Because of list view's approach, list view's performance is faster but limits access to items in the list, that is, you cannot directly manipulate the views, set view properties and bind event callbacks the same way as you can with table view rows. The next subsection describes how to use list items.
