  • In a desperate attempt to get it working I stumbled across the answer.

    I had used the "elevate to project" option in the project explorer in order to use some preexisting code for my project. When I created a new project and just added all the preexisting code to the new project manually everything worked. I'm not sure if this is a problem with the "elevate to project" option, or if something else was going wrong. Anyway, just creating a new project and then adding files seemed to work.
