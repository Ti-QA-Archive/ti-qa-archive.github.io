how to reset or reload a window ?

I have some components laid out in my window, a table, text box, few labels.
They have some text after interaction. I have a reset button and if I press that I want to make it reload the window and bring it to a state like it was when the app loaded that window for the first time, with all the text going back to default values.

Is there an easy way to do that ?

My window is part of a tab group.

  • Hi Srihari,

    exports.create = function() {
    var self = Ti.UI.createWindow();
    self.container = Ti.UI.createScrollView();
    self.add(self.container);
    // now you fill all your stuff into cotainer
    self.addEventListener('click', function() {
        self.remove(self.container);
    });    
    return self;
}
