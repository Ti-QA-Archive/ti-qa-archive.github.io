mapView: how to update annotations?

hi, on a mapview, i'm fetching annotations for a visible region from a db per XHR, that works fine. for updating the annotations as the user moves along, i've considered to use the regionChanged event, but this fires way to often to use it a as a trigger for the XHR. i could use a timer, or update every time a location event fires, not sure. what would you recommend? thx, uno

    start a timer when a region change event ends. If the timer completes before it is cancelled, by the region change event starting again, then you know the user has stopped changing the region so now you can update the map
