Hi guys,

    top:0,
    left:2,
    image:'/images/places/photos/'+rows.fieldByName('placesID')+'.jpg',

});

Let's say placesID is for instance 6, but 6.jpg does not exist. Is there any way I can check this and show a different image?

  • var file = Titanium.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory, filename);

if(file.exists()) { Ti.API.info('file exists'); }
