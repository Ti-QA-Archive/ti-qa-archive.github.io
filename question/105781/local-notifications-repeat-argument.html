Local Notifications "Repeat" argument

Hi,

I'm trying to use Titanium.App.iOS.scheduleLocalNotification because it's perfect for my app, and I have read in API docs:

"params object dictionary of key/values that are used to create the notification. the date property is when the notification should fire, repeat is an optional property of one of: weekly, daily, yearly, monthly, …."

My code is bg.js is:

var notification = Ti.App.iOS.scheduleLocalNotification({
alertBody:"Check for new swagswipes every week!",
alertAction:"Launch!",
userInfo:{"hello":"world"},
sound:"pop.caf",
date:new Date(new Date().getTime() + 3000000),
repeat: weekly // 3 seconds after backgrounding
});

However, the simulator gives me an error:

"[ERROR] Script Error = Can't find variable: weekly at bg.js (line 13)."

Maybe it's a dumb problem, but I can't find the solution. I know that Repeat : weekly is the mistakem butm, which would be the correct name for the argument?

Thanks and sorry for my english!
