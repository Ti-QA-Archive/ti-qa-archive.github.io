Android equivalent to "Open (Animation Fun)" example in Kitchen Sink?

Hi everyone. In the Kitchen Sink app in iOS, under "Base UI/Window (Standalone)" there's an example of opening a little pop up window called "Open (Animation Fun)". If I open the Kitchen Sink in Android, that example is conspicuously missing. I'm assuming that means that code doesn't work in Android. Is that correct? Does anyone know of something similar? Thanks.

    Android does not offer the same built in animation effects for windows as iOS. Therefore, Titanium can't offer the same animation methods on both platforms and that example can't run on Android. There are a couple of window open/close animation effects added to recent versions of Android that are exposed to the Ti.UI.Window.open() method. See the openWindowParams page in the docs for more info.
