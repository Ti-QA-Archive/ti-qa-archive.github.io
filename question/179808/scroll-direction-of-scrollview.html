  • Hi ryan tola,

    there is no any event for identifying scroll direction. but you can identify using scroll event

    e.g

    var prey = -1;
$.scrollView.addEventListener("scroll", function(e) {
    if (prey < e.y) {
        Ti.API.info('SCROLL UP');
    } else {
        Ti.API.info('SCROLL DOWN');
    }
    prey = e.y;
});
