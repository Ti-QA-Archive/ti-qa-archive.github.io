Text area for Table view

I am a beginner corder (and I 'm practicing English too),
nowadays I am trying to create an Android application by using Android sdk 1.2.2 version.

I would like to use Text Area for filling to Table View Row.

I tried to make 2 js files.
One of them contains Table View( data setting) and a button which links to another js file.
Another one has Text area and OK button(I made this button for insert text from Text Area).
I could insert row when I clicked OK button, but the text did not appeared.

Is it possible?

    Yes, it is possible (see code example below).

    My example is a single app.js file. If you're using multiple files, you need to have a solid understanding of what you're doing. Make sure you're using CommonJS modules (don't use Window.url or include() – always use require()). And make sure that the code that handles your text area and button has access to the tableview object – make sure everything is scoped properly.

    var win1 = Titanium.UI.createWindow({
    title:"Test",
    backgroundColor:"white" 
});

var rowdata = [];

var ta = Ti.UI.createTextArea({
    top: 10,
    left: 10,
    right: 10,
    height: 80,
    backgroundColor: '#ccf'
});

win1.add (ta);

var btn = Ti.UI.createButton ({
    title: "OK",
    top: 100,
    left: 10,
    right: 10,
    height: 40
});

win1.add (btn);

var tv = Ti.UI.createTableView ({
    top: 150,
    bottom: 10,
    left: 10,
    right: 10
});

win1.add (tv);

btn.addEventListener ('click', function (e) {
    var val = ta.value;
    ta.value = '';
    ta.blur();

    rowdata.push ({title: val});
    tv.setData(rowdata);
});

win1.open ();
