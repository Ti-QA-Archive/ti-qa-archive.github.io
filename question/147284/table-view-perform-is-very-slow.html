table view perform is very slow

Hi all, i added one image view and label in tabelview row.Here i am using more than 100 images and labels using for loop. Every thing is perfect but when i scroll the tableview, it takes much time to show the next rows(during this time images gets loading). How to prevent this and perform better.

  • hi,
    You can use className property of tableview row, this will do lazy loading of table view, means it will reuse the previous row, and this in turn will increase the performance of tableview.

    There is a built in caching method you can invoke by using the property className when you define a row. This tells the OS that it can re-use this row style and just swap data. You should use this on all rows that you add any views to. Also if you have more than one style of presenting the row then have more than one className in the table.

    Example;

    var row = Ti.UI.createTableViewRow({
...
    className: 'standard',
...
});
