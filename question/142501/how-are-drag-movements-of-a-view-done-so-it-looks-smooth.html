how are drag movements of a view done, so it looks smooth

I have a view that contains a text box( top 0, height 20), and a TableListView (top 20, height 300).
the view is set with bottom -280(so nay the text box is visible).
when I create a drag up gesture(using touch start, touch move, touchend), Inside the touchmove event I change the bottom property so tableListView becomes visible(it contains suggestion)

However repeatedly changing the bottom property with the offset of between touch start position.x and touch move position.x creates a not so smooth animation. How can I make it work better.

For example if you open the messages application on iPhone, and you drag down the text box where you write your message, you get a very nice smooth animation. How could I make mine better?

  • Give the view an animation with a duration of 1 instead of just changing the bottom property. This will give it a much smoother look and feel

    view.animate({bottom:bottomValue, duration:1});
