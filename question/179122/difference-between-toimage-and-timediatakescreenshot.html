Difference between toImage() and Ti.Media.takeScreenshot?

What is the difference between toImage() and Ti.Media.takeScreenshot. Do is the image saved in temp folder? If it is saved how do I remove it?

    Basic difference are below…

    • toImage() returns an image of the rendered view, as a Blob.doc
    • Ti.Media.takeScreenshot takes a screen shot of the visible UI on the device. doc

    This will not be saved in temp folder but these return a blob image so you can use it in your app.
