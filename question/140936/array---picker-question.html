Array - Picker Question

I’d like to be able to push each category entry into a separate picker [FullName, Email…].
I’ve been able to convert the csv file to an array and can access each field through the index e.g.

list[0][0] is FullName 1
list[1][0] is FullName 2

list[0][1] is Email 1
list[1][1] is Email 2

list[0][2] is Class 1
list[1][2] is Class 2

How can I go through the list and separate all the entry into their own category, I know how to do it if it’s a single column through the
