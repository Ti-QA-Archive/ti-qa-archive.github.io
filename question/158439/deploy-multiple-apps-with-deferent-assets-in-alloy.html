Hi All,

I've created an app which uses 3 different theme styles. I'll need to distribute the 3 apps separately. I'll need different splash and icons images for the 3 and a different .config file for each. I don't really want 3 separate projects what if i make future developments or have bugs that need fixing. What is the best way the do this with 1 single project?

I'm distributing to both iOS and Android

  • You can use different themes in Alloy, see here: https://github.com/appcelerator/alloy/tree/master/test/apps/advanced/themes.
