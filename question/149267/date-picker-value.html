Hi…!!!
I have got a small problem in my app, please a have a look and see what can be the solution.   • Hi sudeep, first of all you have to set a type in you picker to indicate it'll be a date picker, for example: 

    //First of all you have to create an object of the class date like this
//you can set its default values or do like this to set the current date to the picker

var data = new Date();

//after that you have to associate it to you picker like this

var datePicker = Ti.UI.createPicker({
    type:Ti.UI.PICKER_TYPE_DATE,
    value:data
});

//Our date picker is now created, but you say you need to print its value, for that you can use this 

datePicker.addEventListener('change', function(e) {
        alert(e.value);
    });

    I think this can help you, hope it does :) !
