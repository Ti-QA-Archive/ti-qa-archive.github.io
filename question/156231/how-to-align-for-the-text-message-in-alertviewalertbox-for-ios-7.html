How to align for the text message in Alertview(alertbox) for IOS 7

How to align the text message in alert view for ios

Reading text from the file system

    var file = Ti.Filesystem.getFile(Ti.Filesystem.resourcesDirectory, "MyText.txt");
    var blob = file.read();
    var readText = blob.text;

createAlertDialog box and show the message!

var alertBox = Titanium.UI.createAlertDialog({    
            title:'Title',
                message: readText,
                buttonNames: ['Cancel','Agree'],
        });
    alertBox.show();

I have taken the readText message from the file system.

when i run the application on IOS 6 the alignment for the text is coming left alignment.!
when i run the application on IOS 7 beta version the alignment for the text message is showing center!

I want to align the AlertDialog box message for the ios 7 to the center. Please let me know how to alignment message for Titanium.UI.CreateAlertDialog box message text to the left alignment.

  • Hi

    The alignment for the title and message of the alertDialog is centre. This is true on both iOS5, iOS6 & iOS7.

    I have tried your code in case it is special in any way, but the default is always central.

    However if you are referring to really long sections of text then iOS5 and iOS6 use a text area to display the text which will automatically display the text left aligned. iOS7 has a new default of NOT using a text area to display the text a continues to use the original label but with a scroll ability not previously provided. As this is a continuation of the shorter display mode it maintains the centre alignment.

    There are no settings to change from these defaults available to you. How they present is fixed. This may change in the future but unlikely as they are fixed OS UI elements.
