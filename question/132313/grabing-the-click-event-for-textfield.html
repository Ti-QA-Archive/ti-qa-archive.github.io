grabing the click event for textField

How can we grab the event of first click inside the text Field.

username.addEventListener('change', function(e) {
Ti.App.fireEvent('event', {data:e});
})

this helps in grabbing the event when we insert the text in text field but i want the event as soon as i click on the text field to write the text.

    you can not call click event untill textfield's enabled property is false
    so here is the solution for your problem…

    var txt = Ti.UI.createTextField({
    width:100,
    height : 40,
    top : 10,
    borderStyle: Ti.UI.INPUT_BORDERSTYLE_ROUNDED,
    enabled:false    
});

txt.addEventListener('singletap',function(e){
    alert(' << CLICKED >> ');
    txt.enabled = true;
    txt.focus();
});
win.add(txt);

    instead of singletap you can also use click and touchstart event
