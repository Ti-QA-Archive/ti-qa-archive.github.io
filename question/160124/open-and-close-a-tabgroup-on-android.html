Open and close a TabGroup on Android

Hi,

I'm just wondering how to close and open a tabGroup on Android. I want to open a tabGroup with a button and go back to the main window with the back key on Android. When I do this the first time everything works, but when I press a second time on the button when the tabGroup was already closed once, nothing happens.

Here a simple example:

(function() {
  var label1, label2, mainWindow, tab1, tab2, tabGroup, tabGroupButton, win1, win2;

  Titanium.UI.setBackgroundColor("#000");

  mainWindow = Titanium.UI.createWindow();

  tabGroupButton = Titanium.UI.createButton({
    top: 100,
    width: 240,
    left: 60,
    title: "go to tabgroup"
  });

  mainWindow.add(tabGroupButton);

  tabGroup = Titanium.UI.createTabGroup({
    exitOnClose: false
  });

  tabGroupButton.addEventListener("click", function(e) {
    tabGroup.open();
  });

  tabGroup.addEventListener('android:back', function() {
    tabGroup.close();
    mainWindow.open();
  });

  win1 = Titanium.UI.createWindow({
    title: "Tab 1",
    backgroundColor: "#fff"
  });

  tab1 = Titanium.UI.createTab({
    icon: "KS_nav_views.png",
    title: "Tab 1",
    window: win1
  });

  label1 = Titanium.UI.createLabel({
    color: "#999",
    text: "I am Window 1",
    font: {
      fontSize: 20,
      fontFamily: "Helvetica Neue"
    },
    textAlign: "center",
    width: "auto"
  });

  win1.add(label1);

  win2 = Titanium.UI.createWindow({
    title: "Tab 2",
    backgroundColor: "#fff"
  });

  tab2 = Titanium.UI.createTab({
    icon: "KS_nav_ui.png",
    title: "Tab 2",
    window: win2
  });

  label2 = Titanium.UI.createLabel({
    color: "#999",
    text: "I am Window 2",
    font: {
      fontSize: 20,
      fontFamily: "Helvetica Neue"
    },
    textAlign: "center",
    width: "auto"
  });

  win2.add(label2);

  tabGroup.addTab(tab1);

  tabGroup.addTab(tab2);

  mainWindow.open();

}).call(this);

Any ideas?

    You should remove this part from your code to get it working

      tabGroup.addEventListener('android:back', function() {
    tabGroup.close();
    mainWindow.open();
  });

    Don't override the native back button! Because it does what you expect. That means you don't need to manually close the tabGroup and there is also no need to open mainWindow because it is already open. A TabGroup is always heavyweight. That is why a new Activity gets launched (at native level) when opening the tabgroup. Pressing back finishes the tabGroup activity and you are back in you root window.
