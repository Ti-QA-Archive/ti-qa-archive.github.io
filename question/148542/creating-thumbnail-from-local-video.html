Creating thumbnail from local video

I'm currently having an issue when creating a thumbnail from a local video in iOS (iPhone 5, iOS 6.1, TItanium SDK 2.1.3). After allowing the user to select a video from the media gallery I try to do the following with the selected video info:

var player = Titanium.Media.createVideoPlayer({
    height: 172,
    media: video,
    scalingMode: Ti.Media.VIDEO_SCALING_ASPECT_FILL,
    width: 276
});

var imageBlob = player.thumbnailImageAtTime(1.0, Ti.Media.VIDEO_TIME_OPTION_EXACT);

However, while this code technically "works" without throwing an error it returns an image blob with a width of 0 and a height of 0. The mime type indicates it is in fact a JPEG but apparently has no content to show. Any insight on the matter would be much appreciated!
