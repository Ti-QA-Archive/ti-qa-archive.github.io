Android TextField in TableViewRow does not focus on first touch.

Here is a simple example of the problem. 

var tabGroup = Titanium.UI.createTabGroup();

var win = Titanium.UI.createWindow({
    title:'Tab',
    backgroundColor:'#fff'
});
var tab = Titanium.UI.createTab({
    title:'Tab',
    window:win
});

var row = Ti.UI.createTableViewRow({
});

var textField = Ti.UI.createTextField({
});
row.add(textField);

var table = Ti.UI.createTableView({
    data: [row]
});

win.add(table);

tabGroup.addTab(tab);

tabGroup.open();

If you launch that on an Android device and tap the TextField it will pop the keyboard up but will not be focussed on the field. If you tab the field again it will be focussed and work normally.

iOS does not have this problem.

Android 2.3.3, Titanium 1.8.0 (it was also there on 1.7.2).
