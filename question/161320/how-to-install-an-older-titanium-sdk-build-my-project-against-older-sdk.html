  • Hello Lee,

    I would suggest installing the 2.1.3.GA sdk from the command line interface (CLI).

    This an be done through the following commands in your terminal (You don't have to be in your project's directory):

    Check that the Titanium CLI is installed. See the guide for installing it if it isn't.

    0) ti &#x2F;&#x2F;optional to check if the CLI is installed

    1) ti sdk install 2.1.3.GA &#x2F;&#x2F;install the sdk

    2) Verify that the 2.1.3.GA sdk is available in your tiapp.xml dropdown.

    (Optional)
    If the above doesn't work you can try uninstalling 2.1.3.GA and then reinstalling it.

    ti sdk uninstall 2.1.3.GA
ti sdk install 2.1.3.GA

    For more CLI SDK help type
    ti help sdk

    Hope this helps,
    Seth
