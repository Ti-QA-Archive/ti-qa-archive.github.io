    Hi

    I am going to teach you how to fish - I am not going to give you a fish.

    The multiple select drop down is anything but a multiple select drop down - there is such a UI element that serves this purpose (not pre-created) you have to create some things yourself in your apps.

    This is in fact the combination of several things;

    • Box that holds the 'tags' is a view with a layout set to horizontal and horizontalWrap set to true.
    • it has an event listener on it for click that then pops up a selection, in HTML this is a select field. But for apps it would be an option dialog for android and also for small lists if iOS, or big lists for iOS would use a modal window with a table and each option a row.
    • choices would then be added to the box as a view containing a label and a button (the delete), that view has a layout of horizontal as well but horizontalWrap is set to false.
    • the delete buttons remove the label view parent
    • the choices should remove any previously selected choices when displayed each time
    • each addition or removal from the box should be set to an internal array so those values can then be used elsewhere.

    I say all this in the hopes you really mean - how to achieve the same function in an app, rather than just using jquery which would be a huge shame,
