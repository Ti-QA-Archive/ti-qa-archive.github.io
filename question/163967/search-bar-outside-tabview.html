Search bar outside TabView

Hello!

I am using Titanium 3.1.3, android, Windows 7.

I need create UI like this:

<Alloy>
        <Window layout="vertical">
            <SearchBar/>
            <Label>Label1</Label>
            <Label>Label2</Label>
            <TableView>//some TableRowViews
                       </TableView>
        </Window>
</Alloy>

Could someone suggest me how can I bind searchBar with tableView data? I know that i can paste searchBar inside TableView, but I need to dispose lables between search bar and table view. How can I do that?

  • Hi

    You can place the searchBar outside the tableView, however as the tableView gains additional features if the searchBar is linked to the tableView - having it outside means you have to do things manually.

    The linked seavhBar gives the tableView a filter, you will have to manage this filtering of data yourself by updating the tableView data on each searchBar change.

    As you are on android you could use the android searchView instead, it can be placed inside the actionBar but still be linked to the tableView. This will only work for 4.0 and higher apps though (also 3.x but nobody uses that).
