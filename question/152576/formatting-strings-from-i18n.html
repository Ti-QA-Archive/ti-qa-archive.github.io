    Hi

    iOS has no built in ability to format text using HTML mark-up or other variation. Using a webView as Muhammad has suggested is one solution, but only if you are going to use one per window, the webView is a heavyweight view and should be used only when required. A single page of formatted text is a great use, but lots of smaller ones is not recommended in any way whatsoever.

    You best bet is to use the Styled Label module available via the marketplace. It is now free.

    This allow you to use simplified HTML within a new label view. You should still use this within the guides the module suggests but it is more flexible than the standard webView and more suited to the task you require.
