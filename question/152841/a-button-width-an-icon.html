  • Hi

    There is no property called html for the button.

    There is a property called html on the label, but this is only suitable when used on Android. As you have not specified what your target platform is - it is hard to work out what you are trying to achieve.

    If you really are using a button then try the following (guessing all of this as you provided no details with the question);

    var btn = Ti.UI.createButton({
    title: 'Caption',
    image: '/images/icon.png'
});

    • It is very close, the problem yours had was a button is not a container; view, scrollView & window are examples of containers, but; label, button, switch and more are not containers.

      <Alloy>
    <View id='view'>
        <Button id="btn_home" />
         <ImageView id="img_hm"></ImageView>
         <Label id="lbl_hm">Home</Label>
        </View>
</Alloy>

      Here the View becomes the container and the button is added first, the imageView next (it will sit above the button) and label, it is above button and imageView.

      "#view":{
    height:32,
    top:30,
    width:120
},
"#btn_home":{
    backgroundColor:"white",
    height:32,
    top: 0,
    width:120
},
"#img_hm":{
    backgroundImage:"/images/img_srch.png",   //chage the source
    height:30,
    left:0,
    width:30
}
"#lbl_hm":{
    color:"#FF9900",
    height:30,
    left:32,
    width:85
}

      However; if you are happy with the default layout you can still use the code from above.

      var btn = Ti.UI.createButton({
    title: 'Caption',
    image: '/images/icon.png'
});

      All of this assumes whatever image yo are referencing actually exists and exists in the location you have specified.
