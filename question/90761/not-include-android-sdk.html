  • Are you running OSX? Apple has only released compilers for OSX, so you can't build for iOS using Windows if that is what you are trying.

    Assuming you are running OSX, then in Titantium Developer there should be tabs for both iPhone and Android. The Android SDK is only needed when you are set to Android. Conversely, the iPhone SDK is only needed when you are set to iPhone.

    However, since the iphone emulator and (Apple) SDK is not available for other platforms you must be running OSX to use them. If you don't have an Apple machine, it may be possible to run OSX in a virtual machine, but you still need a valid OSX license.
