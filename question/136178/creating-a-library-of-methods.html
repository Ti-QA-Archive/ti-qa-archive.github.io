Creating a library of methods

Ok, so I know this is more of a JavaScript question, but it's possible that Titanium Studio might have some of their own ideas (or requirements) on the best way to do this, so I hope it's okay if I ask it here.

In this little learning app I'm doing, I have a process that repeats multiple times. Rather than repeating the same five - seven lines of code each time, I'd like to move them into a single method. And, as my app is spread across multiple .js files, I'd like to put them into a "class" (or maybe the right word is module?) which can be accessed from any of my .js files.

I'm a .NET guy up to now. If I were building a .NET app, for example, I would create a basePage class which would include all the methods and objects to which I would want access application wide. Then in each page I would simply inherit from basePage.

I understand JavaScript is a class-less language. So, what's the best method to accomplish this? I've been playing around with a single method which I put into a base.js file, but I can't figure out how to access this method and get it to return a value. Here's what I'm trying.

In base.js, I have tried both of these:

var gCount = function groupCount()
{
    var count;
    // assign a value to count
    return count;
};

and

function groupCount()
{
    var count;
    // assign a value to count
    return count;
};

Then in the original .js file which holds all the elements and logic for a tab, I am trying to access groupCount, but it always comes back Undefined, or throws an error.

var base = require('/lib/base');
var count = base.gCount;
// or
var count = base.groupCount();
Ti.API.info('count: ' + count)
