How do I display 2 views side-by-side?

I've been trying to find the answer to this seemingly simple question for hours. I just want 2 x 50% views sitting side by side. I then plan to add centred text content to each.

Here's a screenshot to show my dilemma: http://d.pr/i/o3lK

My code is as follows:

var win = Titanium.UI.currentWindow;
win.backgroundImage = '../images/ui/background.png';
win.layout = 'vertical';

// BODY
var body = Ti.UI.createView({height:Ti.UI.SIZE, layout:'vertical', backgroundColor:'#fff'});

var bodyView1 = Ti.UI.createView({backgroundColor:'#ff5e64', height:'75dp'});
var bodyView2 = Ti.UI.createView({backgroundColor:'blue', height:'160dp', width:'50%', left:0 });
var bodyView3 = Ti.UI.createView({backgroundColor:'orange', height:'160dp', width:'50%', right:0 });

body.add(bodyView1);
body.add(bodyView2);
body.add(bodyView3);

win.add(body);

Any help is greatly appreciated.

    Hey Glen,

    A couple of things:

    1. I am not sure why you are using "var win = Titanium.UI.currentWindow;" that to me says you may have some structural issues.

    2. The code below is not exactly how I would to it. Stephen is correct, you don't really need to use layouts if you are going to have fixed sizes. You can use top, bottom, left, and right to get the same effect with one less view.

    However I did modify your code to answer your question:

    var win = Ti.UI.createWindow({
    backgroundImage: "../images/ui/background.png",
    layout: "vertical"
});

var bodyView1 = Ti.UI.createView({backgroundColor:'#ff5e64', height:'75dp'});
win.add(bodyView1);

var body = Ti.UI.createView({
    height: Ti.UI.FIT, 
    width: Ti.UI.FIT, 
    layout: "horizontal", 
    backgroundColor: "#fff"
});
win.add(body);

var bodyView2 = Ti.UI.createView({backgroundColor:'blue', height:'160dp', width:'50%', left:0 });
var bodyView3 = Ti.UI.createView({backgroundColor:'orange', height:'160dp', width:'50%', right:0 });
body.add(bodyView2);
body.add(bodyView3);

win.open();

    • Glen,

      I am glad that the post was helpful. The kitchen sink is a great app for seeing what all Titanium can do, but as a guide for learning how to code in titanium it's best practices have been admittedly lacking. You may want to check out the latest version - https://github.com/appcelerator/KitchenSink.

      Without getting into too much detail the method that the kitchen sink uses to manage windows is not considered a best practice. When opening and closing Windows you almost always want to do it within the same context…

      app.js

      var win = Ti.UI.createWindow();

// Do something with your window here...

win.open();

      Instead of:

      app.js

      var win = Ti.UI.createWindow({
    url:'window.js'
}); 
win.open();

      Then in window.js calling:

      var win = Titanium.UI.currentWindow;

// Do something with your window here...

      When you use the second method for opening windows it can be murder on your memory management and you will be forced to use a lot of event listeners.

      I hope that answered your question.

      Keep an eye on http://tidev.io and one of us will do a series soon on Titanium best practices and getting started.
