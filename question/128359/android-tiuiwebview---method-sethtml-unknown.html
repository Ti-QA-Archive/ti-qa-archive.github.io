android: Ti.UI.WebView - method setHtml unknown?

Hi all!

It seems that on android there are no setters or getters for some properties like the method setHtml for Ti.UI.WebView. The android emulator throws an exception "function unknown".

A direct property assignment works i.e. something like myWebView.html = &#39;somehtmlcode&#39;;

So in general: should setters be used for Titanium object properties or not?

Thank you for your help!
Regards.

    Yes, sometimes getters and setters are not defined, but in some cases property is not defined (intuitively enough, at least). For example, table view and "data" property, you can set it via function constructor and via "setData" method, but you can't set it like property (table.data = []).

    The best way is to check how it's done in Kitchensink sink app and API documentation.
