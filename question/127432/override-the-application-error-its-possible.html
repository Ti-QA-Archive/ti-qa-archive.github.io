  • I just found something that might interest you… and since you compile on you're own via xCode it shouldn't be a problem for you to mess with it.

    in xCode open Classes > API > TiErrorController.m

    there you should find 'loadView' method - this method is the on in charged of building that error screen. It's pretty straight forward.
