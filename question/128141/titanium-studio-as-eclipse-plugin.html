  • I found it!
    From the Help menu, select Install New Software… to open an Install pop-up window.
    In the Work with: text box of the Install window, type the URL:

    http://download.appcelerator.com/appcelerator/studio/plugin/update/stable/

    then get titanium as plugin with aptana feature in the air :D

    if this can help you
