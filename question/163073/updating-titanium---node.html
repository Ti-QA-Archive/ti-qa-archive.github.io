    The problem, as mentioned in the links by Chis is NPM don´t support self signed certs.

    It´s recomended that you update npm.

    So, in command line do:

    $ npm config set strict-ssl false

    $ npm install npm -g

    $ npm config set strict-ssl true

    Finally, update Titanium.

    It works for me.
