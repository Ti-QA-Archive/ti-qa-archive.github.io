Application titlebar design change

By default every app I create has small title row (with black background) with application name displayed on it. Then all windows that are created are displayed below this row.

How do I change design or completely remove this top row with application name. What if I need to create top row with a button in it, like in Twitter app.

Application Type: mobile
Titanium SDK: 1.8.2
Platform: Android 2.2
Device: Android (HTC Desire)
Titanium Studio build: 1.0.9.201202141208

    try to set navBarHidden : true

    var win = Ti.UI.createWindow({
url : '',
navBarHidden : true
});
