  • You can listen for the load event on a webview for the activity. If you have control of the html, you can fire an application level event from within the html that responds with the title of the page. If not, then you can try using evalJS on the webview to grab the title via the DOM.

  • To hide/show activity indicator:

    webView.addEventListener('beforeload', function(){
    showActIndicator();
});

webView.addEventListener('load', function(){
    hideActIndicator();
});

    You can catch the title from webView.html property, look for the &lt;title> tag.
