Problem using the RaphaelJS library

Hello everybody,

I am testing the Raphael JS library into my Titanium application: I've created two page:

app.js:


var mywin = Titanium.UI.createWindow({

    exitOnClose:true,
    title:"First Thing"
});

var win = Titanium.UI.createWebView({
    url:"page1.html"
});

mywin.add(win);
mywin.open();

and

page1.html:


<html>
    <head>    
    <script src="raphael.js" type="text/javascript" charset="utf-8"></script>    
    </head>
    <body>
    <h1>Hello World</h1>
    <script>
    window.onload=function() {
        // Creates canvas 320 × 200 at 10, 50
            var paper = Raphael(10, 50, 320, 200);
        // Creates circle at x = 50, y = 40, with radius 10
            var circle = paper.circle(50, 40, 10);
        // Sets the fill attribute of the circle to red (#f00)
            circle.attr("fill", "#f00");
        // Sets the stroke attribute of the circle to white
            circle.attr("stroke", "#fff");
    }
        </script>
    </body>
</html>

The problem is that, in the execution, I've got (Hello World) displayed then the application quit, and into the console output:


[INFO][ActivityManager(   64)] Process org.esprit.tn (pid 820) has died.
[ERROR][InputDispatcher(   64)] channel '40732e18 org.esprit.tn/org.esprit.tn.TestcanvasActivity (server)' ~ Consumer closed input channel or an error occurred.  events=0x8
[ERROR][InputDispatcher(   64)] channel '40732e18 org.esprit.tn/org.esprit.tn.TestcanvasActivity (server)' ~ Channel is unrecoverably broken and will be disposed!

Even If I commented the RaphaelJs code lines, I got always the same error !!!

Any suggestions? thank you in advance!

N.B: I have downloded the raphael.js library from the RaphaelJS official web site.
