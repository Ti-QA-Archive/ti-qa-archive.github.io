    I am not sure what you are trying to achieve here as there is not example of any data to review.

    But if it is simply ordering of the data in your select statement, try this - again no data to test against so you will need to tweak to suit your own needs;

    var sql = db.execute('SELECT * FROM mytable ORDER BY column1, column2, column3;');
