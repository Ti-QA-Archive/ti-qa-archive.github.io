    Titanium Studio compiles your code for distribution to the Apple store. It then puts the compiled binary into xCode.

    This binary is then used to validate and submit to the Apple Store.

    Although you can you do not need to compile using xCode, you just used the Titanium generated binary.

    Hope this helps
