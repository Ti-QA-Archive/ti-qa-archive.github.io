Position BackGround Image

I have troubles with the position of the background image, because I can not position it in the center …. and occupies the entire width of the button

loginButton = Ti.UI.createButton({// Button login
        width:width,
        height:height,
        left:(spacer*2),
        bottom:'0dp',
        backgroundImage: '/images/iconUser.png',
        backgroundColor: '#636363',
        title:'Cerrar Sesión',
        textAlign:'center',
        color:'white',
        borderColor:'none',
        verticalAlign:Titanium.UI.TEXT_VERTICAL_ALIGNMENT_BOTTOM,
    }),

  • Hi

    That is what the background image does - it fills the entire space that the "thing" occupies.

    You have a couple of options;

    • You add an imageView on to the parent first using whatever size it is, then add the button over the top of the image and set the backgroundColor of the button to transparent.
    • You modify the image so that it has a transparent canvas with the actual image centred - the canvas matches the button dimensions - but as the image is now inside that soace it will appear correctly.
