  • I'm not sure I quite follow your question. Are you trying to retrieve the profile image?

    If so, I would recommend against using JavaScript to do that. String parsing is not JS strongest point. My best advice to you would be to develop a short scree-scraping program that retrieves the image and host it on your own server.

    And your application could then use this newly hosted image.
