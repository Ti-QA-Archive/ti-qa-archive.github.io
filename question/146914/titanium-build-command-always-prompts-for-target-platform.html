titanium build command always prompts for target platform?

I've been using 3.0 for about a week now. And I've got a nice command that should just run the entire build and run process, but it always ask "Target platform [android,ios,mobileweb]" no matter what

Here's the command line:
titanium build -p ios -F ipad

Am I missing a flag?

    Hmm, that's strange. Run titanium sdk and it will list all installed SDKs. If you have any 2.2.x SDKs installed, remove them all.

    Next, run titanium -v and you should be on 3.0.22. If not, run sudo npm install -g titanium.
