  • Hi, i'm currently use 1.3.0 (beta).

    You can compile, but the easiest way:

    1- Download https://github.com/appcelerator/titanium_developer/zipball/master. (Trust me… the name of zip has 1_2_2, but this will be release as 1.3.0, if you see inside tiapp.xml, you will read: <version>1.3.0</version>)

    2- Go to your Application folder and duplicate the Titanium Developer App.

    3- Rename this new to "Titanium Developer 1.3.0".

    3- Open the new App (folder) with "Show Package Content", open "Contents" folder, delete "Resources" folder and "tiapp.xml", save here the "Resources" folder and "tiapp.xml" from the source downloaded.

    4- Now run "Titanium Developer 1.3.0" and you be able to create a new project in "Universal iOS" mode.

    If you need, your app can be a "smart", detecting if is iPhone ou iPad, using:

    var device = Titanium.Platform.model;

    • Yes, you can create a "Universal Project".
      This type of App will run on iPhone and iPad.

      All code to iPhone App will run in iPad.
      You need carefull with Ti.UI.iPad, because this code will NOT run on iPhone.

      Except this, the only change is the window size.
      You can use codes to detect and change your layout for each one.

      You can use:

      var device = Titanium.Platform.model;

      or getting the window size.
