Background image rotation

When I rotate the phone from portrait to landscap the background image gets all stretched out. Essentially, I don't want the background to rotate. I've tried doing an animation during the orientation change..

var tra = Ti.UI.create2DMatrix();
tra = tra.rotate(-180);
var picadjust = Titanium.UI.createAnimation();
picadjust.transform = tra;
win2.backgroundImage.animate(picadjust);

But that hasn't worked. Are there any other ways to do this? Help would be appreciated :)

    When you use a background image, it will always take on the dimensions of the background. If you don't want the scale to happen, you need to either use a view or an image view with the dimensions set.
