    Hi Rahul,

    Thanks for your concern.

    Please check my test case below to create buttons using for loop.

    Ti.UI.setBackgroundColor('white');

var win = Ti.UI.createWindow({
  backgroundColor: 'white',
  exitOnClose: true,
  layout:'vertical'
});

var btns = [];

for ( i = 0; i < 3; i++) { // number of time you want button to create
   btns[i] = Titanium.UI.createButton({
       title : "Button"+i,
    top: 10,
    height:'auto',
    width:'auto',
    color:'#000'
    });
    win.add(btns[i]);         
}        
win.open();

    Hope it'll help you to solve your problem. If you have any query about my answer then feel free to ask.

    Thanks
