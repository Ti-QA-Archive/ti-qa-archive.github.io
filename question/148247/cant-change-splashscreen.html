Can't change splashscreen

I just changed Default.png and the related images to the correct ones, however the app is still built using the default splashscreens instead of my new ones. And i have completly removed the default ones from my project. I have tried cleaning the build, and manually emptying the build-folder with the same result.
After a build, the build-folder is populated again and there are the default splash screens instead of my own ones. If i run the app in the simulator however, the correct splashscreen is showing up.
Weird issue, why is it like this and how do i solve it?

iPhone, SDK 3.0.0.GA

  • Hi

    The splash screens should be easy to swap, so half of this may be known to you.

    Your splash images must be placed in the iphone folder;

    /iphone/Default.png
/iphone/Default@2x.png
/iphone/Default-568h@2x.png

    The images and their required dimensions are as follows;

    • Default.png = 320x480
    • Default@2x.png = 640x960
    • Default-568h@2x.png = 640x1136

    You must provide all of them, the last one is required to enable the iPhone tall (iPhone 5) support. 72DPI is fine, the dimensions are the important thing, so do not create higher dpi versions.

    If you have not already close the simulator and then use the menu Project > Clean to clean the build folder out of all previous builds. Then run the build process again, this should create a fresh version and should not include any older files.

    Test on the simulator and a device to confirm all is well.
