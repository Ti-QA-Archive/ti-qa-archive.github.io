    Hello Bowie Poon,

    The Titanium SDK does not support the Android 4.4W SDK (API 20), also known as the Android Wear SDK. Please take a look this Titanium SDK 3.4.1GA release note. So try to package your android module at a supported android SDK.

    If you have any question, then let me know.

    Thanks.

  • Hello,

    We found out that in some cases we get this error immediately after we publish the module. This means that no publishing process is performed. To solve it we installed 2 additional softwares on Titanium/Appcelerator Studio. Here are the instructions:

    Install all the Android Developer tools

    1. Go to help -> Install new software.
    2. Select “add” button.
    3. Add the following URL: https://dl-ssl.google.com/android/eclipse/
    4. In the software tree select the whole “Developer tools” item.
    5. Install it.

    Install Java Web developer tools

    1. Go to help -> Install new software.
    2. Select “add” button.
    3. Add the URL that corresponds to your IDE current eclipse version. For example, http://download.eclipse.org/releases/luna
    4. In the software tree select the following items:
      “Web, XML, Java EE and OSGI Enterprise Development” -> “Eclipse Java Web Developer Tools” and “Eclipse Java EE Developer Tools”
    5. Install them

    That should work for you.

    Bye!
