  • go to new->file
    and make new file with .js extension.For example newfile.js
    write the following code in your first file to access the second one

    var win2 = Ti.UI.createwindow({
    backgroundColor:'white',
    url:'newfile.js'
});
win2.open();

    hope this will work for you.
