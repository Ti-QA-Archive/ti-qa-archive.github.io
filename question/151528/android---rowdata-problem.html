  • Hi

    This is going to be very difficult to assist with because;

    • masteview.js is incomplete
    • detailview.js is incomplete
    • No example of the object mission
    • ui.window not available
    • ui.view not available
    • ui.scroll not available
    • ui.label not available

    You should add the following code to this section;

    x.forEach(function(mission) {
  Ti.API.info('mission: ' + JSON.stringify(mission)); // add this line
  ...

    This will tell you if the properties you expect are being returned

    If they are then append this line for testing purposes;

    var r = ui.row({
  height: 78.5,
  backgroundImage: "/img/missions/rowbg.png",
  className: "missionRow",
  mission: mission,
  sfilter: mission.title+" "+mission.desc+" "+mission.address
});
r.setTitle(mission.title); // add this line

    These will tell you IF the data you expect is being returned and if it is if the label is the issue.
