Play sound even if the side mute switch is on.

Hi,

How can I make the app so it makes sound even if the side mute switch is on.

I created simple player1(as below) and I would like to play even if mute switch i on. Can I? Is there any solution for that case?

var player1 = Ti.Media.createSound({
    url : 'test.mp3'
});
player1.play();

Kind regards,
Tomasz

    Ti.Media.AUDIO_SESSION_MODE_PLAYBACK should do it, but for that you would need the AudioPlayer instead of Sound.

    • OK. This is great tip. And not just only with Audio Player !!!
      We can put just: 

      Ti.Media.audioSessionMode = Ti.Media.AUDIO_SESSION_MODE_PLAYBACK;

      .. and that's it. So, to sum up, my code:

      Ti.Media.audioSessionMode = Ti.Media.AUDIO_SESSION_MODE_PLAYBACK;
var player1 = Ti.Media.createSound({
    url : 'test.mp3'
});
player1.play();
