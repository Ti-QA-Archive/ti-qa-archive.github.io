Is it possible to change App label once installed?

My boss wants to give the user an option to toggle between to app names. For example, let's say the default name is "Cool Kids App". My boss wants to give the user the option to change the display name of the app to "Chill Kids App". Is there anyway to change the label (i.e. the manifest file) from within the app itself (i.e. user clicks button to change name)? Or would the only way be to upload the same app under the alternate name?

    Hi Ronald,

    Unfortunately changing the app name at runtime isn't possible. And as for submitting two of the same apps with a different name, that isn't possible either, especially in the Apple app store - Apple will reject duplicate apps.

    Maybe you could help us understand why you'd like this behavior? Don't forget that in the submission process you can define tags, so users could still find your app with either "cool" or "chill".
