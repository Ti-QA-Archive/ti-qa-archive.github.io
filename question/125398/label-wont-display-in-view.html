Label won't display in view

Hi,
I was trying to create a small widget (switch + title) for both android and ios, simply by adding a switch and label into a view.
However the label won't display in the view ( the swithc does) for iOS, but appears in android build.
I coded a simple test to replicate the error and gave a screenshot of the result on both iphone and android emulator.
The created view is the red rectangle.

Titanium.UI.setBackgroundColor('#000');

var testWin = Titanium.UI.createWindow({  
    title:'Test App',
    backgroundColor:'#111'
});


var viewContainer = Ti.UI.createView({     backgroundColor: '#FF0000',
                                            height: '50', 
                                            width: '300',
                                            layout:'horizontal'
                                            });

var theLabel = Ti.UI.createLabel({
                                      text: "A test label in a view",
                                      color: "white",
                                      touchEnabled : true

                                    });
viewContainer.add(theLabel);



testWin.add(viewContainer);


testWin.open();

So my question is: Did i do something wrong or did i miss some iOs specific code, but i need the label to be displayed?

Regards

Titanium SDK 1.7.2 IOS Simulator 4.3.2 Android 2.1

    • This worked for me:

      var theLabel = Ti.UI.createLabel({
                                      text: "A test label in a view",
                                      color: "white",
                                      touchEnabled : true,
                                      width:"auto",
                                      height:"auto"

                                    });

      — commented September 9th 2011 by Anthony Decena

    • Thanks a lot. It worked when i specified a value for both width and height. I thought that auto was the default value for height and witdh and didn't think it must be set manual. Regards
