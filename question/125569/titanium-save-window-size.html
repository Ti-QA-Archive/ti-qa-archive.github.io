Titanium save window size

I was surprised that Titanium doesn't do this by itself; how can I save the size of the window, once it's been resized? So that, the next time the app is opened, it's the same size as the last time it was used.

  • Yup. What you want to do, is to store the window size (and position) in localStorage each time the resize or move event is triggered. Then, have a few lines that load up the values, and set the window size and position.

    — commented November 9th 2011 by Titanium Developer

  • What you don't want to do is use 1.1.0 on Lion. Get a recent release candidate.

    Check out Titanium.UI.UserWindow.getBounds(), Titanium.UI.UserWindow.setBounds(), Titanium.UI.UserWindow.setHeight() and Titanium.UI.UserWindow.setWidth().

    As TD suggested, store that in localStorage and reset it when the app starts up.

    — commented November 9th 2011 by Alan DuBoff
