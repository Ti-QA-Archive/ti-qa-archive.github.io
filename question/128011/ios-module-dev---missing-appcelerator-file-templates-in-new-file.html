iOS Module Dev - Missing Appcelerator File Templates in New File

Hey, I recently updated to iOS 5.0 SDK/Xcode 4.3. Previously I had been running XCode 3.26. Now, when I create modules there is no Appcelerator templates to pick from (e.g. TiProxy, TiView, etc.), yet it used to be there previously. I did a full uninstall of the iOS SDK before upgrading.

Is there something I can run to re-install them? The module development guide just says they are there and provides no insight in what one needs to do to get them if they are not there.

  • You may need to reinstall Titanium Studio or your mobile SDKs in order to get the templates to show up again. To reinstall the SDK, you'll need to delete your existing copy of the SDK, then restart Studio. When you load Studio back up, it will reinstall the latest SDK.
