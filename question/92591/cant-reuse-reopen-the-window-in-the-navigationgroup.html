can't reuse (reopen) the window in the NavigationGroup

Hi,
if i created a window and opened it using navigation group

navGroup.open(userWin);

if i back to the previous window in the hierarchy and open userWin again, simply the window is opened and all the instantiation code run again.

navGroup.open(userWin/*cached*/);

this behaviour appears in Ti (1.5, 1.5.1) i am using iOS 4.2 SDK and Ti developer 1.2.2

  • If you're creating new execution contexts each time a window is opened, then having the initialization code re-run every time would be reasonable behavior since the context is destroyed when the window is closed.

    If you would like to retain the contents of a window, consider having your application run in a single execution context (include all necessary files in app.js). This should give you a more predictable environment.
