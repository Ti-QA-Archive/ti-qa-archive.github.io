    • This is what I did:

      • Installed Xcode 4.5
      • Installed 2.1.3RC SDK
      • Created the Default-568h@2x.png (640x1136) portrait….
      • On Titanium Studio…. selected the project… Run Configuration and selected iOS SDK 6.0
      • Build on SImulator…. when you build on simulator check on the iOS Simulator… About iOS Simulator and make sure is version 6.0

      and it worked…..
