    Hi

    Assuming this CDATA entry is a node of bigger XML file then yes very easy.

    // access the cdata node the same as yu would another of the other nodes
// this is an example only convert to your own internal xml structure
var original =  xml.getElementsByTagName('thecdatanode').item(0).text;
// as the node returns text you then convert that to its own
// xml document and you then access that as you did the main document
var cdataxml = Ti.XMLparseString(original);
// this is again an example - convert to your own needs
var name =  cdataxml.getElementsByTagName('title').item(0).text;

    As the CDATA returns test - all you are doing is converting that text back into an XML document of its own - from there access things exactly as you would have done previously.
