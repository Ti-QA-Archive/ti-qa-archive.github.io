  • Hi,
    Did you followed the steps mentioned below for push notification :

    http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.Cloud.PushNotifications

    Since Release 3.1.2, GCM is the default method to send push notifications. If you have an existing application that uses the MQTT protocol, you either need to transition your application to use GCM or set the acs-push-type key in the tiapp.xml file to mqtt.
