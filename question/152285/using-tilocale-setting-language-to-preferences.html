Using Ti.Locale: setting language to preferences

Hi, I am developing a multilingual app. In the first screen user selects his language, I save his choice to preferences and according to that I want to be able to load different xml files (standard usage of Ti.Locale). Is that possible? Now I am confused and I think that Ti.Locale responds to language selected on device. Is there a way to select it manually just for that app?

    You are correct Ti.Locale responds to the devices language setting.

    The only way to switch is setting the devices language, which I guess is not what you want.

    I would setup a directory structure like i81n under resources and set the language in their, then select which langauge file required.
