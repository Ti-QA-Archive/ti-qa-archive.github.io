Mimic unload event on a view

I am making a ad system for my app. When the ads are loaded I would like to record how long the ad is shown.

I want the time to start when the image is loaded. And I want to stop it when the image is not longer shown. This will be because the window which holds it will be closed. I would like to solve it like this:

    image.addEventListener('load', function () {
        var start_time = Number(new Date() / 1000);
        ad.logEvent('load', start_time, args.ad_id);
    });

    image.addEventListener('close', function () {
        var start_time = Number(new Date() / 1000);
        ad.logEvent('load', start_time, args.ad_id);
    });

But there is not close or unload event for an image. Any idea if I can mimic this in any way? I would like to prevent to listen to a global event that is fired by the window closing.

  • Why can't you just add the event listener to the window's 'close' event? That's not a global event, it's a view event.

    There's no reason you can't do something like this:

        win.addEventListener('close', function () {
        var start_time = Number(new Date() / 1000);
        ad.logEvent('unload', start_time, args.ad_id);  //assuming you meant "unload"
    });

    I don't know the larger structure of your code, so the module in question may not have access to the window that image is being added to. But you can change that by passing in the window object to the module.
