How to drag and drop [Android]

Hi everyone,
I need a drag and drop effect working in my app. I tried the whole day the different solutions present in here (Q&A) but none worked for me.
I am using Titanium Studio (latest version) and Android (2.3).

Here is the code that I had the closest drag and drop effect (If you have another solution, just go ahead and tell me =] )

mainWin = Ti.UI.currentWindow;

var imageSelected = Titanium.UI.createImageView({
    height:300,
    width: 200,
    anchorPoint:{x:(0.5),y:(0.5)},
    top:10
});
mainWin.add(imageSelected);

var refX = 145;
var refY = 0;


var reference = Titanium.UI.createView({
    height:imageSelected.height,
    width: imageSelected.width,
    anchorPoint:{x:(0.5/2),y:(0.5/2)},
    backgroundColor: "black",
    top:10
});

reference.addEventListener('touchmove', function(e)
{
    imageSelected.setTop(e.y + refY - (imageSelected.height/2));
    imageSelected.setLeft(e.x + refX - (imageSelected.width/2));

});

mainWin.addEventListener('touchend',function(e){ 
    Ti.API.info("Finished!!! -> Win");
    refY = imageSelected.getTop();
    refX = imageSelected.getLeft();
    reference.setTop(imageSelected.getTop());
    reference.setLeft(imageSelected.getLeft());
});

mainWin.add(reference);

I need to click on the black rectangle (reference) every time to update his reference on the screen. Is it possible refresh the reference position automatically as we move imageSelected ?

Thanks
