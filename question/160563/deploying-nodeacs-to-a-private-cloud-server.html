I'm selecting a platform for a new project and came to find Appcelerator, and Node.ACS.

I have been searching for ways to deploy Node.ACS services to our own server, instead of at appcelerator's cloud, but have been unable to find any documentation in that regard.

Are Node.ACS services limited to be hosted at Appcelerator? If not, can you please point me to more info and documentation to achieve that?

  • Node.ACS is simply a node package, which in theory you could install anywhere. In fact, during development, you install it on your local machine. Of course, you'd need an ACS instance with which to interface. At this time (Dec. 2013), ACS runs only in our cloud. That will possibly change in the future. If it does, it will be an Enterprise feature (meaning, you'll have to talk to Sales).
