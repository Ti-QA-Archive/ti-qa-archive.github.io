Video is black, but I hear audio - Android

So on android, 4.0.3 on my ASUS Transformer tablet, I cannot get the video to show up. I can hear the audio and use the controls to skip around. I have this video player inside a modal window. The same code works on iPhone, iPad (and their emulators) as well as on old LG Thrive with android 2.2. What the hell is going on…

var vidPlayer = Ti.Media.createVideoPlayer({
    width:320,
    height:240,
    top:0,
    left:0,
    backgroundColor:'#ffffff',
    autoplay:false,
    allowsAirPlay:true
});
win.add(vidPlayer);

this loads the video

function loadVideo(path)
{
    vidPlayer.url = path;
    vidPlayer.play();
}

Soooo, same code into a new empty project, the video plays fine. In my existing project it has to be conflicting with something, I just can't figure it out and I cannot really display the code as it is proprietary and I am not allowed to show it.

I guess has anyone run into this issue before. I've seen quite a few threads on android video that has audio but no video. All of which seemed to have been video codec issues, which isn't my case.

  • Alright so I have determined the issue here. The video in my actual app opens in a modal window.

    I created a fresh app with the two tabs that comes as a template. I'm sure you all know what I am talking about. I added the code to the first window that you saw in my first post. The video played fine.

    My second test was to remove the code I just put in and add a button to the first window. This button would open a modal window. Upon clicking the button, the modal window opens and I replicate the issue I am having in my actual app. No video, just audio (although the video controls have always been visible up to this point).

    So, next step is start weeding out the minor properties on the window like navBarHidden and fullscreen. Alast, removing those did nothing.

    Final step, I removed the property on the window of modal = true; and viola! Working video and audio.

    I am still not sure as to why the video showed up fine on my older Android 2.2 GoPhone, but did not work with this brand new ASUS Transformer tablet with Android 4.0.3

    Hopefully my findings will help some people in the future and save them some time. I'll submit this as a bug and report back here with updated information
