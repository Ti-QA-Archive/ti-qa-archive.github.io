Basic xml parsing

I have a php webapp emitting valid xml (in the response) and I need my iPad app to be able to (1) consume it; (2) loop over the repeating xml sections. Here is an example of my xml:

<?xml version="1.0"?>
<DailyReport>
  <MainInfo>
    <TransmissionID>1882</TransmissionID>
    <ProjectNumber>1269</ProjectNumber>
    <TempDegreesHigh>84</TempDegreesHigh>
    <TempDegreesLow>73</TempDegreesLow>
  </MainInfo>
  <SubContractors>
    <SubName>Franky</SubName>
    <QTY>7</QTY>
    <Location></Location>
    <Description>Test flush and disinfect fire riser.</Description>
  </SubContractors>
  <SubContractors>
    <SubName>Jerry</SubName>
    <QTY>4</QTY>
    <Location>Chicago</Location>
    <Description>Layout bolts @ east entrance.</Description>
  </SubContractors>
</DailyReport>

I would be grateful if someone could give me the simplest code to loop and consume this xml data. My xml structures will be no more complex than this, and will never go to deeper nodes than this, because it's 2D data pulled right from a database. Thanks.

  • All I needed was a little hint in the right direction. Turned out to be:

    var doc = this.responseXML.documentElement;
var elem = doc.GetElementsByTagName("MainInfo");
var elemText = elem.item(0).text;

    For my situation, that's all I needed. I would have been happy to throw that little bit of info into a post for anyone asking for the xml basics in TI.
