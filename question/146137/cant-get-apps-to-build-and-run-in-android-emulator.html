    Hi

    Have you completed the android section of the "Configure" option of the "dashboard" tab in studio? This should set up everything to connect to the android system.

    You should be able to access the android virtual devices (AVDs), delete any that exist and let Appcelerator re-create a new one as it needs.

    If you have managed to launch an emulator but you still cannot get your app to start then close the emulator and then run for emulator again - this often sorts the problem.
