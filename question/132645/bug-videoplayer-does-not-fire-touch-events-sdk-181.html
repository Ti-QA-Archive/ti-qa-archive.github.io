Bug: VideoPlayer does not fire touch events SDK 1.8.1

VideoPlayer is not firing touch events anymore. This code works with sdk 1.7.5 and 1.8.0.1 on iOS, but is not working with sdk 1.8.1

I tried to capture click, dblclik and touchstart events with same result.

Can someone confirm the bug, so I can open a ticket in Jira?

thx.

var win = Ti.UI.createWindow();

var videoObject = Titanium.Media.createVideoPlayer({
    contentURL:'test.mp4', 
    scalingMode:Titanium.Media.VIDEO_SCALING_MODE_FILL,
    top:0,
    left:0,
    height:'100%',
    width:'100%',
    autoplay:true
});

videoObject.addEventListener('click', function(){
    alert('click!');
});

win.add(videoObject);
win.open();

Tested on ios5, iPhone and iPad simulator, osx Lion.
