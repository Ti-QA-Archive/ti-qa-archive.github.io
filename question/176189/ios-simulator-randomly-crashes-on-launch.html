iOS simulator randomly crashes on launch

I have an incredibly frustrating issue with any apps launched in the iOS simulator randomly crashing as they launch. This seems to be an issue with my system, as it happens with any app, including some which work perfectly for other people. These particular crashes aren't caused by code issues, as I can relaunch the app again a moment later and it'll run fine. Sometimes I can go an entire afternoon without these crashes, sometimes they'll happen 5 times in a row before the app runs perfectly.

I've tried reinstalling the Titanium SDK, using different SDKs, reinstalling Xcode, resetting the simulator… I don't know what else to try, but this is driving me mad!!

I'm currently targeting Titanium 3.2.3.GA, iOS 7.1 and using Alloy.
