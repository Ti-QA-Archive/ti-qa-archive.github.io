Need help on Sqlite database

Hi,

I have following questions on the sqlite database integration in my project.

  1. When I should use install method and open method?
  2. What if I want to create and fill database programmatically upon the xml response?
  3. What is the physical location of the sqlite database file? Can I locate it on my mac?

Thanks in advance.

    I am still learning Titanium as well but I'll give it a try to answer your questions. First of all please read Working with Local Data. But here's a quick summary:

    1. "install"" is typically used for a pre-populated database, meaning you design and create database (possibly populate with data) outside your application. Use "open" if you want to create database programmatically.
    2. If you want to create a database inside your application use "open".
    3. That depends on what platform you are using. For iPhone it is /Users/[user]/Library/Application Support/[iOS version]/Application/[app id]/Library/Application Support/database
