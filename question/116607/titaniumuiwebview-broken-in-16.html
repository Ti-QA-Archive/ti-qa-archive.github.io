Titanium.UI.WebView broken in 1.6

I had several apps using the webview with the following code:

row.add(Ti.UI.createWebView({
  top: 25,
  left: 10, right: 10,
  html: description,
  height: 'auto',
  color: '#444'
}));

Now, in the latest 1.6 release, it's broken the height setting as well as sporadically not even rendering the html content. Scrolling is not possible if the content is larger than the screen. Very frustrating. Has anyone experienced the same results?

I had to revert back to 1.4.2 to getting working again. It seems pretty straight forward but not sure why it's behaving the way it is.
