Does titanium support notification in android? I have implemented notification in iphone using urban airship. But cannot find the code for implementing it in android. In kitchensink, the notification for android requires the app running in background, which seems not quite make sense for real situation. Thanks.

  • There is a module in Ti+ that allows you to use push for android via UrbanAirship ( I did not test it yet) but you have to be a subscriber to have access to it.
