TiSMSDialog fails for SDK 1.7.0

Hello,

I am using the TiSMSDialog module and it works perfectly on 1.6.2 but when I upgraded to 1.7.0 today and tried to recompile the module for this version the build fails.

Have anyone else experienced this issue? Does anyone know how to make it compatible with 1.7.0?

    For me the build works like a charm.

    I have the TiSDK in /Library and not in ~/Library (so in root not in my home folder)
    I edited the titanium.xconfig file and now looks like this:

    //
//
// CHANGE THESE VALUES TO REFLECT THE VERSION (AND LOCATION IF DIFFERENT)
// OF YOUR TITANIUM SDK YOU'RE BUILDING FOR
//
//
TITANIUM_SDK_VERSION = 1.7.0


// 
// THESE SHOULD BE OK GENERALLY AS-IS
//
TITANIUM_SDK = /Library/Application Support/Titanium/mobilesdk/osx/$(TITANIUM_SDK_VERSION)
TITANIUM_BASE_SDK = "$(TITANIUM_SDK)/iphone/include"
TITANIUM_BASE_SDK2 = "$(TITANIUM_SDK)/iphone/include/TiCore"
HEADER_SEARCH_PATHS= $(TITANIUM_BASE_SDK) $(TITANIUM_BASE_SDK2)

    So as you see only TITANIUM_SDK_VERSION is edited, and I even have Xcode4.

    If is not working for you try to download the module compiled for 1.7.0 from here

    http://dl.dropbox.com/u/7744051/com.omorandi-iphone-0.1.zip
