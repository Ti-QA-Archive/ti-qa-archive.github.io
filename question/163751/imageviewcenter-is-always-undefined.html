ImageView.center is always undefined

I have an image view in my window. When I click a button I want to slide that image view offscreen. I've tried using I've tried using ImageView.top to move it up and that works to move it off, but when I try to move it back on nothing happens because the top doesn't update. It just stays 0.

I've had this working in the past using center.y. But since updating to 3.2.2 the center is always undefined. Does anyone know why this is?

  • In case anyone comes across this. In the animation block I have to explicitly set the ImageView's top property to the new top. That seems to solve the problem.
