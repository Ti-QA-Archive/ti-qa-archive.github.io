Unable to Parse JSON Notification iOS

I am sending a test push notification from my computer to my iOS app. Everything is set up and I am receiving the notifications.

Using this code below outputs the JSON string also below.

function receivePush(e) {
Ti.API.info("Push notification is: "+JSON.stringify(e));
}

{
    "code": 0,
    "data": {
        "aps": {
            "alert": "My first push notification!"
        },
        "alert": "My first push notification!"
    },
    "type": "remote",
    "source": {},
    "inBackground": false,
    "success": true
}

The problem is that if I try to parse the data it gives me an error about being unable to parse the JSON. The below code throws an error. What am I missing here?

function receivePush(e) {
var data = JSON.parse(e.data);
}
