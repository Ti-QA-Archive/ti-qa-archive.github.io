Showing Previous, Current and Next Views of ScrollableViews on Android

Is there any way to this this effect of showing a slice of previous view, entire current view and a slice of next view of a ScrollableView in Android?

I've done this in iOS by applying the following style to the scrollableView:

&#39;#scroll&#39;: { width:&#39;60%&#39;, height : &#39;70%&#39;, clipViews: false },

however this clipViews property doesn't work on Android.
Is there any other way to to this?

Thank you
