How to access row by its index

I'm using SDK 3.1.3 and coding in Alloy.
I want to be able to change the attributes of a button. The buttons are inside rows like so:

<TableView id="answers">
    <TableViewRow id="ansA">
        <Button title="A"></Button>
        <Label class="answerText"></Label>
    </TableViewRow>
    <TableViewRow id="ansB">
        <Button title="B"></Button>
        <Label class="answerText"></Label>
    </TableViewRow>
</TableViewRow>

I was hoping to use something like this:

$.answers.data[0].children[0].children[0].backgroundColor = "white";

$.answers.data[0] gets me to the tableview section successfully, but then how do I keep going to get into a row, then the button, using the row's index?

  • This worked for me:

    $.answers.data[0].rows[0].children[0].backgroundColor = "white";
