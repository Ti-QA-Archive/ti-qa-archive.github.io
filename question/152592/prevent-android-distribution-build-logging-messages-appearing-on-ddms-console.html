  • Hello,

    I recommend to create a global variable like:

    var _debug = true;

    So, when you are still debugging your app, you use something like:

    if (_debug){
Ti.App.info("string");
}

    So, when you are building for production, you do:

    var _debug = false;

    and that's it.

    Best,

    Mauro
