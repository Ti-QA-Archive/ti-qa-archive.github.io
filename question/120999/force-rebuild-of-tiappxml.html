Force rebuild of tiapp.xml?

Hi there, I'm trying the simplest change… changing the iOS status bar style. But if I have already run the emulator, I can no longer make changes to the tiapp.xml file. Every time I do, the emulator shows the old settings. Is there any way to force a rebuild of the app using the latest tiapp settings? The file works if I create a whole new project, then change the settings before running the emulator for the first time, but then as soon as I change settings and run again, no changes are shown.

I have tried exciting Titanium Studio and restarting it, I have also tried rebooting my machine. Nothing seems to refresh the app based on new tiapp.xml settings.
