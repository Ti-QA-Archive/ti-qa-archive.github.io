Create Table View in the same window

I am trying to make a table in the same window , i have tabbed ui application when i use Titanium.UI.currentWindow i get the following error Uncaught TypeError: Cannot call method 'add' of null, I also tried using Ti.UI.createWindow(); but then all the tabs get the table i just need the table to show in a particular window under a particular tab

            var win = Titanium.UI.currentWindow;

            var sectionFruit = Ti.UI.createTableViewSection({ headerTitle: 'Fruit' });
            sectionFruit.add(Ti.UI.createTableViewRow({ title: 'Apples' }));
            sectionFruit.add(Ti.UI.createTableViewRow({ title: 'Bananas' }));

            var sectionVeg = Ti.UI.createTableViewSection({ headerTitle: 'Vegetables' });
            sectionVeg.add(Ti.UI.createTableViewRow({ title: 'Carrots' }));
            sectionVeg.add(Ti.UI.createTableViewRow({ title: 'Potatoes' }));

            var table = Ti.UI.createTableView({
              data: [sectionFruit, sectionVeg]
            });

            win.add(table);
            win.open();

  • Ti.UI.currentWindow is only supported in apps with multiple contexts via the url property. For full details on the difference between single and multiple context apps, check this blog post: http://www.appcelerator.com/blog/2010/08/execution-contexts/

    The important thing to note is that multiple context apps are no longer considered best practice. You should be building single context apps with commonJS. In single context apps you can't use Ti.UI.currentWindow. What you should instead do is save references to the windows you need to use and store those references somewhere globally accessible so any part of your app can use them.

  • You need to add a view to your window before you can add anything else:

    var homeView = Titanium.UI.createView({
    top:"0",
    left:"0",
    height:"100%",
    width:"100%"
    });

    Then you can add your table to your view:

    homeView.add(table);
