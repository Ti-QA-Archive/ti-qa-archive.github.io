  • Hi Abhishek

    i don't have source code for that but i explain you how it's better to implement.

    first fetch all question from database and keep it in array of dictionary like 

    var strQuery = "SELECT * from tbl_question";
var sql = Alloy.Globals.db.execute(strQuery);
var arrQuestion = [];
while (sql.isValidRow()) {
    arrQuestion.push({
        qus_id : sql.fieldByName("id"),
        qus : sql.fieldByName("qus"),
        answer : "",  //Leave blank here
        .....
        .... //if you have more field in db
    });
    sql.next();
}

    after this you can create one window for showing question. (according to your ui requirement)
    when user give answer set it into this array of dictionary for example user give answer for question 2.

    arrQuestion[1].answer = "ENTER ANSWER BY USER";

    so after completing all question you can easily identify which question blank or which question having answer.

    hop you understand.. if you have any doubt then let me know.
