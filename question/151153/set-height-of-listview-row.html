Set height of ListView row?

I have a ListView with a custom template for the ListItems, based on the documentation here: http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.ListItem

My questions is how to change the Row height of one item in that ListView?

  • To answer my own questions: when adding list items, set the properties to give it a height:

    var job = { properties:{height: 80}, color: {backgroundColor: '#00FF00'}, title: {text: jobs[i].title}, location: {text: jobs[i].location}}

  • $.ListView.addEventListener('itemclick', function(e){
    var item = $.ListSection.getItemAt(e.itemIndex);
    if(item.properties.height >150) {
        item.properties.height = 70;
    } else {
        item.properties.height = 300; 
    }
    $.ListSection.updateItemAt(e.itemIndex, item);
}
