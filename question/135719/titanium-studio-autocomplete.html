  • What worked for me was this:

    • Go to Window > Preferences > General > Editors > File Associations

    • Select the "*.js" file extension on the file types window and "JavaScript Source Editor" in the Associated editors window. Set it as the default editor.

    • Close the js files that were already open and reopen again.

    Voilá!! =]
    I hope it helps somebody else.
