Titanium.UI.CreateButton vs Ti.UI.CreateButton

Hi all

I know that this question is a very basic question, recently I had started with titanium classic, I came across these two codes. 

var myButton = Titanium.UI.createButton({
title: "click me",
width: "25%",
height: "auto",
id: "myBtn1"
)}

var myButton = Ti.UI.createButton({
title: "click me",
width: "25%",
height: "auto",
id: "myBtn1"
)}

Basically what is the difference between these two syntax for creating a button. So which one should I use ?

Thanks

Varun Krishna. P
