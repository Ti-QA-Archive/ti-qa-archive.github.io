  • Check out the rightNavButton and leftNavButton properties in the Titanium.UI.Window object.

    So if you want to have button b in the titlebar on the right, simply call the following code:

    Ti.UI.currentWindow.setRightNavButton(b);

    Note: These properties are only supported on iPhone.

    • From what I see, you are trying to set the active tab by getting a Tab object. But the setActiveTab methods needs an int as parameter. That might be your problem.

      Try setting the tab using the index of the Tab:

      tabGroup.setActiveTab(1);
