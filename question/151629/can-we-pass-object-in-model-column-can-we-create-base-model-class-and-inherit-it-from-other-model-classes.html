  • The guides for models are here: http://docs.appcelerator.com/titanium/latest/#!/guide/Alloy_Models

    That said, there's not a real good way to do inheritance between models with Alloy notation in it's current incarnation. That said, you don't have to use Alloy's notion of models if you don't want to. You can use straight Backbone models and still leverage all of Alloy's functionality. And if you use straight Backbone models, you can just follow their documentation for inheritance between models and collections. I have an example in the github repo for using data binding in Alloy with a pure Backbone model: https://github.com/appcelerator/alloy/blob/master/test/apps/models/binding_no_persistence/alloy.js
