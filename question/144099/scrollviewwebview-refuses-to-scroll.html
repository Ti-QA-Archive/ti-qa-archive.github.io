ScrollView/WebView refuses to scroll

I am building an app that allows a user to flip through articles just like the pages of an eBook. I have two "article" windows which I toggle between to have the effect of flipping through a series of pages. Each contains a series of header elements and then the remainder of the page is a webview. The issue I'm having is that no matter the length of the content, I do not get vertical scrolling and cannot see the bottom of the article. I have tried to achieve this by placing the webview inside a scrollview, but the result is the same. In a possibly related not, I get this warning

[WARN] couldn't load URL: app://com.vrtx.binderApp/(null)

when I update the HTML content of the webview using articleDetail.setHtml( newHtml )

The code I am using to build the webview is this:

var articleDetail = Ti.UI.createWebView({ top : 0 });

var articleDetailContainer = Ti.UI.createScrollView({
    top                  : 220
    , left                : '6.5%'
    , width                : '87%'
    , scrollType        : 'vertical'
    , layout            : 'vertical'
})

Thanks for any help!

Liam
