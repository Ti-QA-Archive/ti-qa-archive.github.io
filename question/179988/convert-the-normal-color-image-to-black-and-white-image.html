  • I would suggest put one view on top of image view (Size and placing of view is as same as Image View). Play with View opacity and background color property.

            activityIndicatorWithBackgroundStyle: {
            height: 95,
            width: 130,
            opacity:'0.7',
            backgroundColor:'black',
            layout:'vertical'
        }

    Used toImage() method to convert it into BLOB.

  • The Titanium image methods don't support adjusting the saturation / color of an image. You'll need an extension module. I've never used it, but the tintOverlay module might work for your needs. It looks to be Android only. BTW, a great resource for finding modules is gitTio.
