  • This is properly the case of a common mistake in Titanium Studio. When you first started the basic two tab app, you properly pressed the green play button in top menubar in studio and that dreated a new run configuration for that project. If you then close the emolator an switch to a new project and press the green button once again you will automaticly start the same project again.

    What you have to do is press either the arrow next to the green play button and create a new configuration for the KitchenSink project or use the studio play button
