iPhone to Android

UI Window features - there is no "titleControl" feature in Android. How can I add this or set the UI design for the tabs in Android?

The window eventlisteners dont work in Android app.

Can anyone help with any of this? This is only the tip of the iceberg. I am trying to find solutions to a lot more. I have done a lot of research already to no avail.

  • I'm in the same boat as you with all the same issues. My app worked great on Iphone terrible on Android. It seems you can't do anything to move the tabs to the bottom on Android. The Eventlisteners issue I found was solved by not layering views, you have to put views that you want to catch events directly on the window with no views on top of it or under it. As far as the titleControl, are you talking about the title in the NavBar? if so there is no NavBar for Android unless you create your own. My video player still doesn't work, not sure what to do there, my mapView is trashed and My app crashes all the time.

    It would be helpful if Appcelerator offered an overview of all the pitfalls that occur when you make that switch to Android with your code developed on IOS. Hopefully this helps a little, I'm still trying to work out what seems to be an endless list of bugs and it's pretty scary.
