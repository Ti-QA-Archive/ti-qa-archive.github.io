Problem with touch events (1.8.0.1)

Hi,

I have a problem with touch events (touchstart, touchmove, touchend, touchcancel) in 1.8.0.1 v8:

example code 1:

parentView = Titanium.UI.createView({
    width: 8000,
    height: 300,
    // etc ...
});

for(i=0;i<10;i++) {
    childView = Titanium.UI.createView({
        width: 800,
        height: 300,
        left: 800 * i,
        top:0,
        childViewID: anyValue,
        // etc ...
    });
    parentView.add(childView);

    // + many other child view

}

parentView.addEventListener('touchstart', function(){/*code*/}); // listener does not work

parentView.addEventListener('click', function(){/*code*/}); // working properly

example code 2:

parentView = Titanium.UI.createView({
    width: 8000,
    height: 300,
    // etc ...
});

for(i=0;i<10;i++) {
    childView = Titanium.UI.createView({
        width: 800,
        height: 300,
        left: 800 * i,
        top:0,
        childViewID: anyValue,
        // etc ...
    });
    parentView.add(childView);
    childView.addEventListener('touchstart', function(){/* EMPTY */});

    // + many other child view

}

parentView.addEventListener('touchstart', function(){/*code*/}); // work

In version 1.7.5 example code 1 work properly all touch events. Is this correct behavior? I now have add new event listener into each child view? Is that okay?

thanks and sorry for my english
