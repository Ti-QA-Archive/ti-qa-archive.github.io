    Yes, this one may not be immediately obvious.

    • Create a new folder {appname}/app/lib/
    • Place your somethingcool.js file inside this folder
    • Reference code as before

    Example reference;

    var sc = require("somethingcool");

    Then same as you did before Alloy.
