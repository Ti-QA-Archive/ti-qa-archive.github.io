Access other file (about hierarchy)

Index.xml have a view as require.

index.xml

<Alloy>
    <Window>
        <View id="globalNav" platform="ios">
            <Require src="globalNav"/>
        </View>
    </Window>
</Alloy>

And, navigation view have a button.

<Alloy>
    <View>
        <Button id="LeftBtn" onClick="toggleMenu"/>
    </View>
</Alloy>

So, when toggleMenu is clicked, I want add other view(menu.xml) to index.

menu.xml

<Alloy>
    <View class="container">
        <Button>Test</Button>
        <Button>Test</Button>
        <Button>Test</Button>
        <Button>Test</Button>
    </View>
</Alloy>

How can i add function to load menu view?
I can't understand alloy's hierarchy.

Help me.

  • Hi Esteban Kim,

    You can add it by using createController
    e.g

    
function toggleMenu(){

    var menuView = Alloy.createController("menu").getView();
    $.winIndex.add(menuView); //$.winIndex is a window object of window

}

    if you have any doubt then let me know feel free… :)

    Thanks
