    • There is no way to display SVG (scalable vectorial graphical) charts in Android because all versions before the 3.x will not support it.
      But you can try with canvas, I export one of my projects on github which regroup multiple javascript libs on iOS and Android (and for both) like raphael, rgraph, google chart and dygraph.
      You can see my project below : https://sourceforge.net/p/titaniumcharts/code-0/3/tree/Courbes/
