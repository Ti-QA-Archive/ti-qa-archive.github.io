Ti. SDK 3.2.X - TextArea properties NOT working

I test my code in SDK 3.2.X but facing problem with TextArea properties, 'horizontalWrap', 'enableReturnKey' not working. The same code running in SDK3.1.3 is ok. Is this a bug?

    Hello Goh,

    Problem Description

    Differences between the behavior of enableReturnKey and horizontalWrap between 3.1.3 and 3.2.0 in Android.

    Test Case

    We did some testing with this sample code (app.js):

    var win = Ti.UI.createWindow({
  backgroundColor: 'white'
});
var textArea = Ti.UI.createTextArea({
  borderWidth: 2,
  borderColor: '#bbb',
  borderRadius: 5,
  color: '#888',
  font: {fontSize:20, fontWeight:'bold'},
  //keyboardType: Ti.UI.KEYBOARD_NUMBER_PAD,
  returnKeyType: Ti.UI.RETURNKEY_GO,
  horizontalWrap: true,
  enableReturnKey: true,
  textAlign: 'left',
  value: 'I am a textarea',
  top: 60,
  width: 300, height : 70
});
win.add(textArea);
win.open();

    and we obtained the desired behavior in Android and iOS.

    Test environment

    Mobile SDK 3.2.0
    Nexus 4 with android 4.4

    Probable cause of the issue

    Please test with the latest 3.2.0.GA, it is more stable than the daily builds.

    Best,

    Mauro
