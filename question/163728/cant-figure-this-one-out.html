Can't Figure This One Out...?

Hi folks -

Maybe I've been at this for too long today, but…

Please consider the following code from an election app we're building:

aRace.addEventListener('click',function() {
            Ti.API.info('A race selected. lastRaceIndex is ' + lastRaceIndex + ' and raceSelect.children.length is ' + raceSelect.children.length);
            if (lastRaceIndex <= raceSelect.children.length) {
                raceSelect.children[lastRaceIndex].setBackgroundColor('#333332')

    Hi Michael Petit,
    Your code is fine as i see , but you are doing a mistake, You see the raceSelect.children.length is 1, i.e the
    raceSelect has only 1 children and as you know children is an array so, if it has 1 children then its index starts from 0 not 1. This is the reason you are getting an error.

    your code looks for this

    raceSelect.children[1].setBackgroundColor('#333332');
//here the element at index 1 is not defined , so it is throwing error.

    Please feel free to ask if any confusion and mark the answer if your problrm is solved to help others.
