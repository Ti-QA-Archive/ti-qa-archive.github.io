Problem Image background

Hi, i'm new of titanium and i've a problem with background image.
I create a view

var win1 = Titanium.UI.createWindow({  
    backgroundImage: '/Images/bg.png',
    orientationModes: [Ti.UI.PORTRAIT, Ti.UI.LANDSCAPE_LEFT]

});

When i change mode the image was compress. There's a way to focus image only in center when i'm in portrait mode?

Thank you

    • it's because you use Ti.UI.FILL. it's resize according to it's parent view. for that you need to set it fixed height width according to your requirement.. and if you want to set it dynamically according to image height ratio then you need to calculate it after loading image..

      e.g

      
imgViewEx.addEventListener('load', function(e) {
        e.source.touchEnabled = false;
        var imgHeight = imgViewEx.toImage().height;
        var imgWidth = imgViewEx.toImage().width;
        var screenHeight = 100;  //It's Height of parent view of imageView
        var screenWidth = 85; //It's Width of parent view of imageView
        var screenAspectRatio = (screenWidth / screenHeight);
        var imgAspectRatio = (imgWidth / imgHeight);
        var scaledHeight;
        var scaledWidth;


        if (imgAspectRatio <= screenAspectRatio) {
            scaledHeight = screenHeight;
            scaledWidth = (screenHeight * imgAspectRatio);
        } else {
            scaledWidth = screenWidth;
            scaledHeight = (scaledWidth / imgAspectRatio);
        }

        imgViewEx.height = scaledHeight;
        imgViewEx.width = scaledWidth;

    });
