  • Hello. I had no idea that Appcelerator verified the module GUID against their central service. That caught me by surprise. Since the module I was selling has the same GUID than the module on the Appcelerator Marketplace, the serves thinks you never bought the module. I'm in the middle of vacations so I can't solve the problem right now, but tomorrow I will release new versions of all my modules with a different GUID. Very sorry about all this mess.
