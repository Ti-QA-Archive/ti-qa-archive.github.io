Align component on the right (like css float right)

App Type: Android SDK: 3.2.0 Device: Galaxy S3 i9300 Version: Android CM 4.3.1 Host: OSX 10.9.1 Alloy

Simple question, what's the trick to align any component on the right of parent object? Example:

<Window>
    <View width="90%" layout="horizontal">
        <Button title="Do some magic and go to the right of window"></Button>
    </View>
</Window>

Any clues?

    If you are going to give the View a layout of horizontal then you will need to use top and left to position the Button, to position the Button on the right you would need to work out the width of the button and subtract that from the width of the view.

    If you remove the layout of horizontal, then you will be able to use absolute positioning. Meaning you could use right: 0
