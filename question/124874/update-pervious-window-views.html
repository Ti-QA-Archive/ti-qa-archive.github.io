  • It sounds like you are creating windows using the url property, which creates multiple execution contexts. Without seeing more of your code, it's hard to say how this can be resolved. A general advice that would likely help this situation is to use a single execution context for your app, that way all of your windows can share all your data, functions, etc…

    I would highly suggest watching the the 2nd video in the free on-demand training video series, "Cross-Platform Javascript Applications." It explains, in detail, what the difference is between single and multiple context apps, and when you should use each technique. Understanding this will not only help answer your question, but it will make you a better Titanium developer in the long run.
