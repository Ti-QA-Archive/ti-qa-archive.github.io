Module Logs:

    Hi,
    Can you please be clear if you are trying to create a logger module that keeps track of your logs you want to get printed or its just the case that you are not even able to print the logs from the module directly.

    If you create a default module project it uses Log class from org.appcelerator.titanium.util.Log to print the log messages on console. You can try using logs using sample below :

    Log.d(TAG, "printing message: " + message);
