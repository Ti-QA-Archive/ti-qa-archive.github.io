ACS user.logout with device_token not working

In the documentation it says:
Parameters

device_token :

If specified, all push subscriptions associated with this device token are canceled. See Android Push Notifications for more information.

http://docs.appcelerator.com/cloud/latest/#!/api/Users-method-logout

So the code I have is 

Cloud.Users.logout(
{
device_token : token
}, function (e) {
    if (e.success) {
        alert('Success: Logged out');
    } else {
        alert('Error:\n' +
            ((e.error && e.message) || JSON.stringify(e)));
    }
});

But I get the following error:
[ERROR] : line = 49;
[ERROR] : message = "'[object Object]' is not a function (evaluating 'a(d)')";
[ERROR] : name = TypeError;

So it seems like it is trying to evaluate the first parameter as the function, which is in this case a map.
