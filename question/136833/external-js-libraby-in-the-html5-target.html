External JS libraby in the HTML5 Target

Is it possible to use others JS libraries (JQuery, etc) in the HTML5 output ? If yes how to do it ?

  • It's possible, but you do have to be careful. Mobile Web is designed to be UI compliant with iOS and Android, and it achieves this by carefully managing the DOM elements. If you use any of the UI parts of jQuery/Dojo/etc, it can cause lots of things to break.

    If the library in question is in CommonJS or AMD format, you can simply require() the library and use it. I would recommend creating a Mobile Web module to prevent breaking your code's compatibility with Android and iOS.

  • Simply copy the library files in your Resources project folder.

    From there, simply reference the .js files in your html files using the like you would in a regular web page.

    <html>
...
<head>
    <title>Local WebView</title>
    <script type="text/javascript" src="jquery-1.3.2.js"></script>
</head>
...
</html>

    ###Note:
    If you are developing on Android, make sure to set the ti.android.compilejs to false. Otherwise, Titanium will try to convert the JavaScript contained in the library to native code during distrubution (it will certainly fail).

    • Sorry, the UI is too tightly controlled/heavyweight to use 3rd party libraries to style it. If you used jQuery or a similar library to change the styling of a mobile web app, then you will probably break the app. We do have plans to give mobile web a makeover and to introduce the capability to define your own theme.

      We don't really position Titanium Mobile Web as a generic web app for doing typical web things, but rather as a peer of Titanium Mobile on Android and iOS. Our goal is to provide as close to 100% compatibility with those platforms as possible. In order to achieve this goal, we were forced to make the decision to make Ti mobile web apps a closed environment that has total control over its contents. You can use 3rd party libraries to implement functionality that doesn't exist in the Ti API, but you cannot use 3rd party libraries to augment existing functionality.
