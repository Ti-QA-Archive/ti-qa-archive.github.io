Creating HTML pages with the correct size

Hi

I want to create a simeple html page that I can load using the Titanium.UI.WebView for a Samsung Galaxy i9000

My page contains an index.html page and a style.css stylesheet.

I want the page content to display full screen on the phone, so I am setting the main content div to 400px wide by 800px high.

In Titanium when I launch the app I use SDK 2.2 and Screen WVGA800. However the emulator screen does not seem to be large enough to display the content.

Is my html/css wrong or am I using the wrong settings in Titanium?

  • (1) Use a meta viewport tag so your page shows up in full scale to users:

    <meta name="viewport" id="viewport" 
content="width=400, user-scalable=no" />

    (2) set a static width for your body

    <body style='width: 400px'>
 or in your css
 body {
   width: 400px;
 }
