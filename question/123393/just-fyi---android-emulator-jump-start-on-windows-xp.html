Just FYI - Android Emulator Jump Start on Windows XP

Just a simple discovery I made. I am running Windows XP and for some reason I just can't seem to get the android emulator working from Titanium. I have been all over the posts and have tried everything.

Titanium ends up not finding my emulator and timing out. So if all else fails just go to the Android SDK manager, click on the AVD that Titanium created and click the Start button on the right (manually start the emulator). Then recompile your code and your program will load and run on the already opened emulator.

  • Curtis, I had a similar problem just days ago but on Win7.
    I could run the emulator just fine, but TI would NOT see it.

    What worked great for me is: To resolve problem: Open Android SDK Manager and installed ANDROID SDK 2.1-update1 API 7. Using TI Studio open preferences and browse to SDK folder. Now ALL ANDROID SDK versions appear in drop down list and the app runs in the emulator!

    http://developer.appcelerator.com/question/122998/could-not-locate-the-android-sdk-at-the-given-path#214645

    There are also some other great ideas that worked for others.
