view doesn't show correctly on ios

titanium sdk: 2.1.1
os: mac os
platform: ios

it's quite a simple view (in the real world, it may be complicated), but the following right view doesn't show, i don't know why, the left view show correctly.

    this.view = Ti.UI.createView({
        layout : 'horizontal',
        top : 0,
        height : '20%',
        width : Ti.UI.FILL
    });
    var leftView = Ti.UI.createView({
        left:0,
        top:0,
        width:'22%',
        height:Ti.UI.FILL,
        layout: 'vertical'
    });

    var avatar = Ti.UI.createView({
        backgroundImage : '/images/avatar.jpg',
        top : 10,
        left : 10,
        width : 60,
        height : 60,
        borderRadius : 30
    });

    leftView.add(avatar);    

    var rightView = Ti.UI.createView({
        left : 10,
        top : 10,
        width : '78%',
        layout : 'vertical'
    });

    var label = Ti.UI.createLabel({
        text:'good'
    });

    rightView.add(label);

    this.view.add(leftView);
    this.view.add(rightView);

    Probably because there's not enough room, so it's wrapping. You set left & right views to total up to 100% width, but then you put 10px of padding (you set the left property on rightView) in between. That's 10px more than you have, since it's 100% + 10px.
