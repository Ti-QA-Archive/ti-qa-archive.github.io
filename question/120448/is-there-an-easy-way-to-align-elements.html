Is there an easy way to align elements?

How can I automatically align vertically elements (searchBar,buttons,label) without using top,left?

  • Make a wrapper ( a simple view ), set layout:'vertical' to it, and add the elements you want to stack in this view.

    This is what you want?

  • 
var wrapper = Ti.UI.createView({
layout:'vertical';
});

win.add(wrapper);

l1 = Ti.UI.createLabel({
text:'l1',
height:20
});

l2 = Ti.UI.createLabel({
text:'l2',
height:20
});

wrapper.add(l1);
wrapper.add(l2);
