  • Ok, I see the light. The f_ck_ng clean process after change the images folder. If you don't clean it, you can't see all the changes you do.

    Also, I need to put the images in the assets\android\images\res-long-land-hdpi (and so on), because if I put it in other folder, like assets\android\ICONS\res-hdpi then don't works.

    Maybe it's a question of don't having that res-long-blah in images and another different format in icons (res-hdpi, without long-blah).

    Well, I only lost 4 hours with this, not bad… But at least it works now.
