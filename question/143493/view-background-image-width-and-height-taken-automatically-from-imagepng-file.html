View background Image Width and Height taken automatically from 'image.png' file

Hey Folks,

I don't want to specify the width and height of the file explicitly in the code. Can it get width and height values automatically from the image file?

var notification=Ti.UI.createView({
    backgroundImage:'image.png',
    left:25,
    top:25,
});

Using titanium sdk 2.0.2 with Android 2.3.3

    • will the following code work? If yes then is it a good practice to use everywhere in the application? I think the .toImage() function takes some processing time and may slow when used everywhere in the application. What you say?

      var notification=Ti.UI.createView({
    backgroundImage:'image.png',
    height:notification.toImage().height,
    width:notification.toImage().width,    
    left:25,
    top:25,
});

    • yes, this is a working code and i have used this code before many times, if you want to use same height and width every where in the app then you can set these two values in a global variable one more thing you can not set these two properties at creatation time

      just use this as describe below

      var notification=Ti.UI.createView({
    backgroundImage:'image.png',
    left:25,
    top:25,
});

win.add(notification);

notification.height = notification.toImage().height;

notification.width = notification.toImage().width;
