    Hello!

    In general, it's highly complex to decrypt an IPA file. Even if you manage to accomplish it (check this instructions), it's likely that you only get assembler, not the the actual Titanium Project.

    On the side note, if the IPA was made for distribution, you will find that in addition to the IPAD being crypted, JS code will be scrambled (i.e., even if you manage to get the code, the code will be heavily modified to prevent copy or piracy of the app).
