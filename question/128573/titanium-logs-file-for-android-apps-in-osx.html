Titanium logs file for Android apps in OSX

Can someone tell me where is the logs file of titanium in OSX for android apps ? e.g. for iOS what i'm looking for is found in this directory: library/application support/iPhone Simulator/5.0/applications/*myapp*/documents/*.log

    John

    There is no equivalent log for Android, but you should be able to see debugging output shown in Titanium Studio's console when the app launches in the emulator. Alternatively, run this from a terminal window:

    /path/to/android/adb -e logcat

    Personally, I like to use Colored Logcat, listed in the Debugging section of our Third-party Tools page, to make logcat output to the terminal a lot more user-friendly.

    Hope this helps
