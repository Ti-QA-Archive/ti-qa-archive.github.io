  • Hi

    Have you looked in this path;

    • file:///Users/ag/Library/Application%20Support/iPhone%20Simulator/7.1/Applications/21914E03-A059-42FF-B6BF-6240FDA702F5/EndlessAisle.app/

    To see if the app.js file does exist?

    BACKUP your app. To repeat - BACKUP your app.

    Choose from one of theses two;

    • Titanium with Alloy Framework (if you are not sure DO NOTHING)

      • Project > Clean > choose project > confirm
      • Delete the contents of the Resources folder ONLY if using Alloy

    • If this is using Titanium with Alloy

      • Project > Clean > choose project > confirm
        -Do NOT touch the resources folder

    Close the simulator and rebuild for the simulator;

    • Does the app start-up?
    • Does the app.js file appear in the folder reference now?
    • Check app in Studio - are there any red crosses on any files you are ignoring when you compile?
    • Create a new test app and compile for simulator - does this work?
