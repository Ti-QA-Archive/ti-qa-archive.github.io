XSL transformation of XML data in WebView

I'm quite new to Titanium Mobile but have been searching for a while about doing an XSL transformation of XML in Titanium.

My use case is this:
I'm downloading an XML file from a web server, I want to take this XML and apply XSL to it, the XSL file will transform the XML to HTML. I want to show this HTML data in an WebView.

But no luck this far, I have been looking at a few different solutions such as:
http://www.w3schools.com/xsl/xsl_client.asp

And to set the XSL stylesheet directly in the XML data and then adding it to the WebView like this:

webView.html = '<?xml-stylesheet type="text/xsl" href="test.xsl"?><greeting>Hello, World!</greeting>';

But have not been able to make it work, have seen a few other questions like this on the web but none that I have found have been answered with a working solution.

Am I missing something or is there no functionality to use XSLT with either Titanium or the WebView?

I am using Titanium SDK version: 1.8.1 (01/27/12 17:31 a24502a), iOS 5.0 on the iOS Simulator.
Titanium Studio, build: 1.0.8.201201262211 on OSX 10.6.8

  • So I solved this with the help of Mauro's suggestion, had to tweak it a bit though. I made a second layer that I make the request to which is a PHP file on a webserver, the PHP gets the XML from the API and inserts the XSL stylesheet to the XML and then renders it in HTML.

    This successfully displays the converted data in the iPhone. Maybe not the prettiest solution but atleast it works.
