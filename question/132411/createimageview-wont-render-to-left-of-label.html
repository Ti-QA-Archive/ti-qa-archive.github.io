createImageView won't render to left of label

Targeting: iPad, iOS5
Titanium SDK: 1.8.1
Device: iOS simulator
Mac OSX

I have a simple table I'm creating through Titanium.UI.createTableView(). As you can see in the code below, I'm attempting to display a png to the LEFT of a label in each row. I'm only showing the code for 1 row, but there are more. Yet the png always displays in the CENTER of the row no matter what I do. And the label appears right over the top of the image, rather than to the right of it. (But I can still see the image behind the label text.) I have tried different values for the "left" css property, tried changing them both to "right", tried different values. Nothing seems to work. The png is 46x43.

var row = Ti.UI.createTableViewRow();

var image = Ti.UI.createImageView({
    left: 10,
    url: 'KS_nav_ui.png'
});

var label = Ti.UI.createLabel({
    left: 65,
    text: 'My Proprietary Text'
});


row.hasDetail = true;
row.add(image);
row.add(label);
tbl_data.push(row);

What am I doing wrong? I can't find a "horizontal-align" property or similar for the image, in the properties.

// Create an array of explicitly defined custom TableViewRows
var tbl_data = [];
for(var i = 0; i < 10; i++) {
    var row = Ti.UI.createTableViewRow();
    var label = Ti.UI.createLabel({
        left : 40,
        text : 'Row ' + (i + 1)
    });
    var image = Ti.UI.createImageView({
        url : 'KS_nav_ui.png',
        left:1,
        width: 38
    });
    var button = Ti.UI.createButton({
        right : 10,
        height : 30,
        width : 80,
        title : 'press me'
    });
    row.add(image);
    row.add(label);
    row.add(button);
    tbl_data.push(row);
}
