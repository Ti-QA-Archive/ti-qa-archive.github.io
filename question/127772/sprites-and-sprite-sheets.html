Sprites and Sprite Sheets

Is there any way to implement sprite sheets in Titanium as a source for graphics instead of having multiple files? I have an app with lots of assets and I am trying to save space and loading time.

  • There is not a way to handle this natively, no. However you could create some custom component to make this possible. In the end, with the way native components work, it is not clear as to whether or not you'll actually save on load time in the way that you expect. Remember that you're dealing with native components, not a web page.
