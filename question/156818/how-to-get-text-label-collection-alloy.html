How to get text Label Collection Alloy

I have a problems getting data from xml

index.xml

<Alloy>
    <Window id="index" class="Window">
        <View dataCollection="User" >
            <Label id="verifyCode" text="{verify_code}"  />
        </View> 
        <Button title = "Click" onClick = "Click" />
    </Window>
</Alloy>

index.js

var lstUser = Alloy.Collections.User ;
lstUser.fetch({query: "SELECT verify_code FROM User WHERE user_id = 1"});

function onClick (){
     alert( $.verifyCode.text  );
}

Note: lstUser have data.

show error:
Cannot call method text of undefined.

can someone help me !
Thanks.

  • You're only fetching a single record. Do this with a model, not a collection.

    index.xml

    <Alloy>
    <Window id="index" class="Window">
        <View>
            <Label id="verifyCode" text="{User.verify_code}"  />
        </View> 
        <Button title = "Click" onClick = "Click" />
    </Window>
</Alloy>

    index.js

    // this assumes that you created this instance instead of the Collection you had before
var lstUser = Alloy.Models.User ;

lstUser.fetch({query: "SELECT verify_code FROM User WHERE user_id = 1"});

function Click (){
    alert( $.verifyCode.text  );
}
