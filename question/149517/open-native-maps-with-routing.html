  • Hi Nico,

    you can use below code to show route directly in ios native map

    var a = new work;     // put  var 'a' as address you want as destination 

var    latitude = e.coords.latitude ;
var    longitude = e.coords.longitude; 

var str = 'http://maps.apple.com/?daddr='+ a +'&saddr='+latitude+ ',' +longitude

Ti.API.info("string for--->"+str);

Ti.Platform.openURL(str);

    This link directly open ios native map and show route between source and destination address

    Here daddr works as destination address and saddr works as source address (user current location)
