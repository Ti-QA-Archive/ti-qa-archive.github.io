  • I think it might work doing the next:

    1. create a square label of let's say 44x44px with a white background and whatever text number you want
    2. add it to the window offscreen ( left:-1000);
    3. open the window
    4. convert the label to image
    5. set the image you get to the menu item as icon (you might need to make it a file first)
    6. each time you need to change the badge, set a new text to the label, then repeat steps 5 and 6.

    The first steps are requires because on android toImage won't return anything until you actually render the view in the window.

    Let me know how it goes :)
