Portrait to landscape mode, width not stretching

Well i have a comment like pop up window, in here landscape mode is possible.

Now the problem is that when i go to landscape mode, the width is stick to the portrait mode width.

Even tho i set the with to 100% its not automaticly set to a new with for portrait.

How can i use landscape mode properly? Is there an window event listener so i can manually increase or refresh the width?

here is a code example for what i use

var winComment = uiElements.Window('');
                winComment.modal = true;
                winComment.orientationModes = [Ti.UI.PORTRAIT, Ti.UI.LANDSCAPE_LEFT, Ti.UI.LANDSCAPE_RIGHT];

            var comment = Titanium.UI.createTextArea({
                height: '100%',
                width: '100%',
                top: 40,
                font: { fontSize: 14, fontFamily: 'Arial' },
                color:'#000',
                textAlign:'left',
                suppressReturn: false
            });

  • Hi Kami.

    You can lock orientation on global level (in xiapp.xml), or for particular window (see documentation for Ti.UI.Window: https://developer.appcelerator.com/apidoc/mobile/latest/Titanium.UI.Window-object, orientationModes property) or you can manually recalculate dimensions on orientation change (see: https://developer.appcelerator.com/apidoc/mobile/latest/Titanium.Gesture-module).
