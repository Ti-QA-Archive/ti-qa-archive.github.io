Extending primitive types

How can we add methods to existing types like Date and Object in titanium so that it is available everywhere?

  • You can modify native prototypes (which is not recommended practice anyway :) ).

    //this way
Object.prototype.doSomething = function() {
    alert('Did something');
};

var obj = {};

obj.doSomething();

//or even better to avoid creation of enumerable property (won't affect for-in loop)
Object.defineProperty(Object.prototype, 'doSomething', {
    value: function() {
        alert('Did something');
    },
    enumerable: false,
    writable: false,
    configurable: true
});

    There are also alternatives to this approach.

    • Do you use CommonJs modules?

      If so, every CommonJs module is JS context for itself, which means clean JS environment. Because of that, you need to re-setup JS environment of every CommonJs module.

      Because of that, IMO, it is better to write "date" module in which you will create instance of Date object with methods you want to add.
