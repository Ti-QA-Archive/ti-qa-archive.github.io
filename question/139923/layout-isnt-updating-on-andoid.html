Layout isn't updating on andoid

I do some layout changes on orientationchange and on iOS it works fine. On android it doesn't seem to be doing anything. For example:

// orientation change event
case Ti.UI.LANDSCAPE_LEFT:
    vidPlayer.width     = (!Ti.App.isTablet) ? 480 : 1024;
    vidPlayer.height    = (!Ti.App.isTablet) ? 320 : 768;
break;

In portrait mode, the video takes up half the screen. In landscape left, (on iOS, it takes up the full screen). On android, the video is not changing size. I tried using the View.updateLayout() method and that didn't work either.
