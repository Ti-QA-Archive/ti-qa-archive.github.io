  • You probably don't want to set all files to 777 in the ~/.titanium folder. Instead, make sure you're the owner. If you run "titanium" as sudo, then it'll create files as root. To fix this, simply run this:

    sudo chown -R `whoami` ~/.titanium

find ~/.titanium -type d -exec chmod 755 {} \;

find ~/.titanium -type f -exec chmod 644 {} \;

    After doing this, run titanium status to make sure you're logged in and if not, run titanium login.
