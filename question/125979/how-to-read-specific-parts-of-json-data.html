[{"Post":{"id":"1","brand":"Mercedes","model":"E-class Coupe 2010"}},{"Post":{"id":"2","brand":"Mercedes","model":"S-class 550"}},{"Post":{"id":"3","brand":"BMW","model":"X6"}},{"Post":{"id":"4","brand":"Audi","model":"A5"}}];

And i only want to read the data that has mercedes as brand

"brand":"Mercedes"

  • if jsonString represents the remote JSON you've received, you could do so like this:

    var json = JSON.parse(jsonString);
for (var i = 0; i < json.length; i++) {
    var post = json[i];
    if (post.brand === 'Mercedes') {
        // run your code for processing Mercedes data
    }
}
