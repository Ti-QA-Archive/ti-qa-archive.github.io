    I use a javascript version. My Gist for a PhoneFormat.js is here: https://gist.github.com/ckoye/f7a9c987c1f2e9f96d7c

    It is based on libphonenumber.

    Put it into your lib folder, require it and use it. It is fine for usage with some data. 

    // require it:
// in alloy:
var PhoneFormat = require('PhoneFormat');

// use it:
var formatted = PhoneFormat.formatE164("US", "8646978257");
//Returns: +18646978257

    If you want to clean up a lot of phone numbers in a loop the Javascript performance is not so good.
    Of course a native open-source module would be great for performance reasons. Please let me know when there is one!

    I hope that helps!
