Titanium Studio - Git always prompts for Username & Password

Hi,

Titanium Studio works great with Git. I can commit my changes and all is fine … but every time i want to push the commit to GitHub it ask for my username and my password. I enter them and it works. At the moment i use the GitHub.app (working on Mac OSX) to push my commit to the GitHub server … but it is not so effective to switch after every commit to the GitHub.app to push my commits :/

How can i configure the username and password (token) for Titanium Studio so that a push will work without any prompts?

Thank you!

  • I guess this question was asked long time ago, but I am going to add an answer so people can find one.
    If you go the GitHub page of the project you are going to find SSH clone URL.
    Copy that link.

    Usually the link is in this format git@github.com:username/repositoryname.git

    After that using Titanium Studio you need to import your project again:
    Using the option Git Repository as New Project.
    Click in the option for URI and paste the SSH clone URL.
