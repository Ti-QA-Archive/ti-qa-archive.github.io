Input field accept only number

Has some way to create input that accepts only numbers?

If yes, what better way to do this?

— asked October 9th 2014 by Breno Douglas Araujo Souza

  • Accepted Answer

    Add an event listener for the text field change event as follows:

    myTextField.addEventListener('change', checkInput);

    and the function:

    function checkInput(evt) {
    evt.source.value = evt.source.value.replace(/[^0-9]$/, '');
};

    Not forgetting to remove the event listener when the window closes.

    myTextField.removeEventListener('change', checkInput);
