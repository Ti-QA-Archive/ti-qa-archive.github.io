addEventListener 'click' not working sometimes ?

Hey,

I have the following code:

var startButton = Titanium.UI.createButton({
    width: 180,
    height: 50,
    bottom: 0,
    backgroundColor: '#32CD32'
});

startButton.addEventListener("click", function cameraButtonClick (cam) {
    Ti.Media.takePicture();
});

var myOverlay = Titanium.UI.createView();
myOverlay.add(startButton);

Sometimes when I run the app, everything works, and sometimes the app doesn't recognize the click…
What can cause this issue ?

    Change this;

    startButton.addEventListener("click", function cameraButtonClick (cam) {
    Ti.Media.takePicture();

    startButton.addEventListener("click", function (cam) {
    Ti.Media.takePicture();

    I have never used a function name inside a call back in my career or using Titanium.
