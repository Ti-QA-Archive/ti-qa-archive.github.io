How do I access the optionDialog/actionsheet style property?

I want to change the style of my iPhone optionDialog(actionsheet) to 'translucent black' instead of the 'default' one. Anybody know if this can be done? There's nothing in the docs http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.UI.OptionDialog-object

  • Ok, it looks to me like there is no way to access it through Titanium, but since Apples guidelines state that you should use the translucent black UIActionSheetStyle for a black interface I dug a little deeper and found a way to edit it in xCode.

    Within the iphone build folder of the Titanium project, inside the file 'TiUIOptionsDialogProxy.m', below the line that says

    [actionSheet setTitle:[TiUtils stringValue:[self valueForKey:@&quot;title&quot;]]];

    I added the following line:

    [actionSheet setActionSheetStyle:(UIActionSheetStyleBlackTranslucent)];

    This works and makes all of the actionsheets in the app the appropriate style, however you need to build/run the project from within xCode for it to work. Running it from Titanium will overwrite this change and revert the actionSheets to the default style so I figure it's best to do this as a final step before building the app for distribution.
