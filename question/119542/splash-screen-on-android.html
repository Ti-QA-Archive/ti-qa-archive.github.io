Splash screen on Android

I added a custom splash screen to my app and ever since when I try to close the application with the back button the screen goes black, when the phone is rotated the splash screen appears. After pressing the back button again the application closes.

Is this a bug (and are there more people out there with the same problem?) or is there something wrong in my application?

  • Found my own answer: Turns out it had nothing to do with the custom splash screen, I set exitOnClose to true for the main window and the problem was solved.
