TableView's Swipe-To-Delete button behaviour

I have a TableView displaying user messages (a bit like email), where each row has three labels (sender, subject and date, much like the iOS Mail app).

This TableView also has a swipe-to-delete behaviour which displays the delete button on a swipe gesture.

Now, this behaviour displays the button but pushes the date label, which is positioned on the right where the button appears, to the left underneath (or on top of, I can't tell) the other labels which are positioned to the left of the row.

Is there a way to make the button appear over the top of the label or to hide that label when the button is revealed?
