OptionDialog - hangs app after click and event doesn't fire

This is a single view IOS 5 app. I have a button in my view runs this function when clicked:

function changeGrade() {
    var gradeSelect = Ti.UI.createOptionDialog(opts).show();
    gradeSelect.addEventListener('click',function(e){
        alert('you chose ' + e.index);
    })
}

The OptionsDialog opens fine and displays the options (defined outside the function). However, when I click an option, the dialog goes away and nothing happens. After that, the app hangs. Am I doing this wrong? Any help much appreciated.

  • Never mind–I figured it out. It turns out you need to call show() after you add the event listener.

    function changeGrade() {
        var gradeSelect = Ti.UI.createOptionDialog(opts);
        gradeSelect.addEventListener('click',function(e){
            alert('grade level changed to ' + e.index);
        });
        gradeSelect.show();
    }
