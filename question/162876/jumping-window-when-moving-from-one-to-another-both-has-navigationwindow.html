  • Accepted Answer

    Are you only using a navigation window to get the navigation bar? Because normally you would open the child window like this with post being a regular window:

    $.home.openWindow(post);

    Anyway, I do think that the shift you noticed when a NavigationWindow is opened using this animation is a bug you should report on JIRA. Be sure to provide are more simplified test case demonstrating the issue.
