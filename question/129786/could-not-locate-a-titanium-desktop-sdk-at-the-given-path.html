    It appears a filter on the Titanium release list is preventing some Titanium Studio users from being served a Titanium Desktop SDK update. While we work on a fix, a workaround:

    Look here on how to download manually a SDK from a URL, and then use one of the following URLs (this downloads desktop v1.1.0, the latest stable release):

    • 64-bit linux: http://api.appcelerator.net/p/v1/release-download?token=A4pQanA0
    • win 32: http://api.appcelerator.net/p/v1/release-download?token=b4pNB8S6
    • 32-bit linux: http://api.appcelerator.net/p/v1/release-download?token=J4pNAA22
    • os x: http://api.appcelerator.net/p/v1/release-download?token=w4pNA8q6
