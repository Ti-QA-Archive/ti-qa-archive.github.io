  • 4MB is pretty big for any mobile device. Let alone parse the thing!!!

    I remember participating to a pretty long thread about that very same subject a while back. And the consensus was that the only viable option was to split those same files on the server side before parsing them on the device. Also, it was suggested to trim all the unnecessary information from the JSON before parsing it on the device. That may not be much, but it can add up over time.

    Otherwise, you will be confronted to the same problem every time your files will grow.
