Mobile Web Navigation Group bar color?

I can't find a property to change the barColor of the top navigation bar in a Mobile Web Navigation group… How can I change it from the default Gray?

sdk 2.1, studio 2.1, os 10.8

  • The navigation bar is controlled by the title bar of the current window. You can use a barColor or a barImage.

    var yourCurrentWindow = Ti.UI.createWindow({
    barColor:'#000'
});
