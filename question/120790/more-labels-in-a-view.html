More labels in a view

I am building an Android app, with a window that shows an article with title, publish date and body text.

var articleView = Titanium.UI.createScrollView({
    contentHeight:'auto'
});

var articleTitleLabel =  Titanium.UI.createLabel();
var articleDateLabel =  Titanium.UI.createLabel();
var articleBodyTextLabel =  Titanium.UI.createLabel();

articleView.add(articleTitleLabel);
articleView.add(articleDateLabel);
articleView.add(articleBodyTextLabel);

win.add(articleView);

Of course I want the labels to show up under each other. How do I do that?

I can't set any "vertical" on the scrollView, and I can't use an absolute number for height on the labels, because the label text is set dynamically. Setting "top: 0" just makes the text show on top of the others.

  • Although the API documentation doesn't mention it, the View does support the layout attribute.

    So you can just add all your labels to a Sub View and then add this View to your ScrollView:

    var articleView = Titanium.UI.createScrollView({
    contentHeight:'auto'
});

var subView = Ti.UI.createView({
    layout:"vertical"
});

var articleTitleLabel =  Titanium.UI.createLabel();
var articleDateLabel =  Titanium.UI.createLabel();
var articleBodyTextLabel =  Titanium.UI.createLabel();

subView.add(articleTitleLabel);
subView.add(articleDateLabel);
subView.add(articleBodyTextLabel);

articleView.add(subView);

win.add(articleView);
