  • Hi Raju,

    if you are thinking to use a single code for the all the devices than it could be little bit akward. As sanat mamtora mention to use below sample:

    "Label[platform=ios formFactor=tablet]": { 
    backgroundColor: "#0f0",
    text: 'iPad'
},

    its nice but you can also use % instead of dp , if you think any UI element is same for all the devices than use dp or you can use

    if it was height than use platform height
height :(Ti.Platform.displayCaps.platformHeight * 1.2) / 100;

&&

for width
width : (Ti.Platform.displayCaps.platformWidth * 1.2) / 100;

    you can change the value 1.2 as per you need in height and width its for sample only. Try this it may be
    help you . let me know for more help.
