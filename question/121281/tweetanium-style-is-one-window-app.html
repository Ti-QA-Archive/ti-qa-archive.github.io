  • If u have command in the tweetanium approach then it is easy to change views dyanamically when ever u want.
    i did it as like this…

    In Tweetanium App Application window change appFilmStrip Views Property as mentioned below…
    var appFilmStrip = S.ui.createFilmStripView({
    top:0,
    left:0,
    right:0,
    bottom:tabHeight-10,
    views:Ti.App.views,
    custom:0
    });
    then define
    Ti.App.views= [
    S.ui.createView1(),
    S.ui.createView2(),
    S.ui.createView3()
    ];
    S.app.MainWindow = S.ui.createAplicationwindow();
    S.app.MainWindow.open();
    change views dynamically where ever u want.
    don't forget to define in app.js
