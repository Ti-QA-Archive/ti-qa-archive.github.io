Titanium : Trouble cannot get height of tableview.

This is my code.
var table2 = Ti.UI.createTableView({

    top : 70,
    moveable : true,
    bottom : 0,
    // editing : true,
    scrollable : true,
    backgroundColor : 'transparent',

});

newWin.add(table2);

Ti.API.info(table2 .size.height);

It is 0.
I also tried getHeight function;
But the result was undefined.

  • View.size (everything is a subclass of View) is only available after layout happens. So, the window has to open, and all the sizes of everything in the window get calculated. You can listen for the postlayout event on newWin and get the table height then.
