Table or Table Row Offset

How can I determine the offset of a table's row from its original position?

Attempted:

The contentOffset property of the tableView is available only to iOS and the firstVisibleItem is essentially useless, per this post.

Perhaps there is a way to do this by adding a view element on top of the tableView which can track touch events (tableViews can't track touch events, on Android at least). I have attempted to add a scrollView on top of the tableView to no avail.

Usage:

This information can be used to build a tweet-like pull to refresh for Android, see post.

  • Solution Found !!!

    Add the tableView to a scrollView. This requires a set height for the tableView, which can be computed by adding up the row heights. It also requires the scrollView have a higher zIndex than the tableView.

    Then, you can track the offset of the scrollView which now should correlate to the offset of the tableView.

    See the gist of my Tweetie-like pull to refresh for Android for an example.
