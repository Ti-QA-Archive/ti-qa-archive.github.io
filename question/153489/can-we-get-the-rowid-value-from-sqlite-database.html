  • Hi

    You need to create an autoincrementing column, this will create a value automatically in the column.

    CREATE TABLE "combo_calcs" ("id" INTEGER PRIMARY KEY AUTOINCREMENT NOT NULL, "options" VARCHAR,"question" VARCHAR,"grid" VARCHAR,"solution" TEXT DEFAULT (null) )

    You can then use the command;

    // assumes db is the variable name used to access the DB
var lastID = db.lastInsertRowId;

    This tells you the most recent id values associated with the autoincrement column each time you perform an insert.
