Newsstand App Using Appcelerator

It looks like Appcelerator allows Newsstand style alerts in the API but I'm curious if anyone has successfully built a Newsstand app using Appcelerator. If you haven't built one, do you know if it is even possible?

    Jeff,

    Yes and no.

    • It requires a native module that supports the newstand api from apple (happen to have one): https://bitbucket.org/stephenfeather/module-newskit-ios

    • You also have to be able to launch the app on push (doable)

    • Have to be able to do things in the app quickly after the download of assets is completed (not fixed yet)
