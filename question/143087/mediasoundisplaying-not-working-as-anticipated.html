Media.Sound.isPlaying not working as anticipated

Hi all,

I'm trying to hook up a button that turns on and off background music for my app

here's the code

var audioPlayer = Ti.Media.createSound({
        url : "/audio/klfitapp_musicloop_1.mp3",
        looping : true,
    });

if (audioPlayer.isPlaying == true) {
                Ti.API.info("Audio Player status: true");

                // Music playing so switch off and swap button image
                audioPlayer.pause();
                audioButton.setImage("/icons/speaker_off@2x.png");
            } else {
                Ti.API.info("Audio Playing status: false");

                // Music is not playing so start!
                audioPlayer.play();
                audioButton.setImage("/icons/speaker@2x.png");
            }

I'd expect the button to start playing when on touch when app is first run, and the music to stop playing when button is touched again.

However, I can get the music starting but it won't pause.

Any suggestions?
