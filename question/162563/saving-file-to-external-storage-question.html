Saving file to external storage question.

Hi everyone.

I'm trying to save an image to my SDCard. I'm using the following code…

    save.addEventListener('click', function() { 
    var sigImg = paintView.toImage().media; 
    var filename = (manifestNumber + "_signature.bmp");
    var img = Ti.Filesystem.getFile(Ti.Filesystem.externalStorageDirectory,filename);
    img.write(sigImg);
    Ti.API.info("image " + filename + " loaded at: " + img.nativePath);
    $.win.close();
    });

The problem is that my phone has the following directories. "SDCard0" (local storage) and "SDCard1"(actual Sd Card). It's saving the file in the local storage (SDCard0) instead of SDCard1.

Does anyone know how I can specify the external SD card?

Thanks in advance.

Devon Britton.

  • I successfully used this module to save it to an external SD card.

    var mkdfs = require('dk.mikkendigital.mkdfs');
Ti.API.info("module externalStorageDirectory is => " + mkdfs.externalStorageDirectory);
Ti.API.info("module externalFilesDir is => " + mkdfs.externalFilesDir);

var file = Ti.Filesystem.getFile('file://' + mkdfs.externalFilesDir, 'sample.txt');
file.write('Newly created file on SD card');
