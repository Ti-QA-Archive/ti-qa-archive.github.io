TableViewRow height issues

Hi guys,

I have the following labels inside a tableview row:

var tableViewRow = Ti.UI.createTableViewRow({
    height: 'auto',
    layout: 'horizontal',
});

var detailLabel = Ti.UI.createLabel({
    color: '#516691',
    height: 'auto',
    text: text,
    top: 10,
    left: 0,
    width: 75,
    textAlign: Ti.UI.TEXT_ALIGNMENT_RIGHT,
    font: {
        fontSize: 13,
        fontWeight: 'bold'
    }
});

var detailText = Ti.UI.createLabel({
    text: text ? text.replace(/(\r\n|\r|\n)/g, " ") : '',
    top: 10,
    bottom: 10,
    left: 10,
    right: 10,
    font: {
        fontSize: 13,
        fontWeight: 'bold'
    },
    height: 'auto',
    width: 'auto',
});

and here is the result: screenshot

Does anybody have a clue why in the "visit" row doesn't adjust it's height along with the "detailText" inside it? I also noticed this happens when there's a long line of continuous text with no spaces inside (URLs).

Thanks a lot!
