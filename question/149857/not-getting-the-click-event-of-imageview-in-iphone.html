Not getting the click event of ImageView in iPhone

Hi,

I am not able to get the click of ImageView which is in center of my Apps window. I have taken a parent view with horizontal layout and have added 3 ImageViews to it. I am getting the click event of the first and the last ImageViews but not getting click of the one in the center of these two. The main window is part of a TabGroup having three tabs and the issue is with the first window in the TabGroup.

Application type: mobile
Titanium SDK: sdk-version: 2.1.3.GA
Platform: iOS
Device: iPhone simulator, iPad, iPhone

Below is the code:

//Parent view in which three ImageViews are added
var viewSharing2 = Titanium.UI.createView({
height:'80dp',
width:'200dp',
top:'185dp',
left:'54dp',
layout:"horizontal"
});

var imgvFacebook = Ti.UI.createImageView({
image : '/iphone/img_fb.png',
width : GetWidth(48),
height : GetHeight(53),
top : "10dp",
left : '20dp'
});
imgvFacebook.addEventListener('click', function() {
facebookSharing("Hi");
});

viewSharing2.add(imgvFacebook);

var imgvShareThis = Titanium.UI.createImageView({
image : '/iphone/img_share.png',
width :'53dp',
height : '48dp',
top : '10dp',
left : '13dp',
zIndex:100
});

imgvShareThis.addEventListener('click', function(e){
alert('Share button clicked');
});

viewSharing2.add(imgvShareThis);

var imgvTwitter = Ti.UI.createImageView({
image : '/iphone/img_twitter.png',
width : '47dp',
height : '51dp',
top : '10dp',
left : '5dp'
});

imgvTwitter.addEventListener('click', function() {
twitterSharing(lblAgentName.text, Titanium.App.Properties.getString('activation_code'));
});

viewSharing2.add(imgvTwitter);

I am able to get the click events of imgvFacebook and imgvTwitter ImageViews but not getting the click event for imgvShareThis. I don't understand what I am missing.

Any help will be highly appreciated. Thanks in advance.

PS: Above code is working for Android.
