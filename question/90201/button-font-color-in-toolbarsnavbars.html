Button font color in toolbars/navbars

I can't seem to change the font colour of buttons when they are in a navbar OR as left/right navbuttons.

I am setting the color property when creating the button object, but it remains white.

Can anyone confirm this?

    I believe this should solve your problem

    backgroundImage: 'none'

    Example

    var edit = Titanium.UI.createButton({
    backgroundColor: '#000000',
    borderColor: '#1c1d1c',
    borderRadius: 6,
    color: '#ffffff',
    borderWidth: '2',
    height: 25,
    font:{size:9, fontWeight:'bold'},
    width: 50,
    backgroundImage: 'none',
    title:'Edit'
});

    then set up all the other properties for the button
