Updating Alloy UI in run-time ?

Hi guys I wanted to ask if anyone knows a trick to change the UI of an Alloy project without rebuilding. I hate rebuilding the whole project for changing just a text on a label. How do you guys do rapid UI development ?
In a classical environment I used to open TestWindows using URL loading windows which opens the .js file a new context, and hence I ran the app once and while it is live on the Simulator I updated the .js file hosting the UI and it was updated on the fly. This is not working in Alloy, because I think alloy copies the XML files hence a new build is required to re-copy the changes. any ideas ?
