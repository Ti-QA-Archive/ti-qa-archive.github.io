  • hi,

    for Android

    var finalUrl = 'http://docs.google.com/viewer?embedded=true&url=' + <pdfUrl>;    

    var webview = Ti.UI.createWebView({
        url: finalUrl,
        width:'auto',
        height:'auto',
        top:0,
        scalesPageToFit: false,
        loading: true,
        backgroundColor: '#999999'
    });

    win.add(webview);

    for iPhone
    you can download pdf by using httpClient and then show it in webView or else you can directly open it by using 

    
Titanium.Platform.openURL('http://foo.com/bar.pdf');
