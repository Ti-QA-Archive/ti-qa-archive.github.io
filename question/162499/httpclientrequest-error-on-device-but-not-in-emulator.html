  • I see 'xhr.open("POST", _loginUrl, false);' – android doesn't support synchron mode (third parameter).

    If you aspect a JSON answer you must set 'Accept'-header.

    You parse the answer. Please use try/catch.

    In your log I see XML-errors. Perhaps your server gives you the noaspected answer. In any cases gives a network sniffer like 'Burb suite' a good help to analyze the problem.
