    Hi

    You need to set the LeftNavButton and/or RightNavButton;

    <Alloy>
    <Window title="profile">
        <RightNavButton>
            <Button id="add" title="Add" />
        </RightNavButton>
        // Everything else you have
    </Window>
</Alloy>

    This tells the OS to place the button in the right side of the title bar. The left side one if you choose to use it is LefNavButton.

    These will ONLY work if the window itself is part of a TabGroup or NabigationWindow as a navigation controller is required to support the navigation bar, the title, optional left and right buttons as well as the automatic back butyo . Also obviously iOS only.
