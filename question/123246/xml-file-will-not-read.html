  • I actually fixed the problem. I guess you had to have the entire directory. For some reason it would not read the file if I put it in the same folder and the .js I was running

    var file = Titanium.Filesystem.getFile("../Resources/tableWindows/CrossRef.xml");
var xmltext = file.read().text;
var xml = Ti.XML.parseString(xmltext);

    It would always tell me it read the file into a Ti.Filesystem.File but would never go any further. I hope someone else runs across this and it helps because I've searched all of appcelerator for this answer and ended up figuring it out myself.
