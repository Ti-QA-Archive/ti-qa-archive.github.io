  • Accepted Answer

    have you opened signup page with currenttab????? 

    signupBtn.addEventListener('click', function() {
    var win1 = Ti.UI.createWindow({
        url : 'signup.js',
        title : 'Signup'
    });
    Ti.UI.currentTab.open(win1);
});

    using currentTab you will be able to preserve tabGroup

      when you are working in app.js then you cannot use current object of any thing [current win,current tab ,current … etc] current is used when you are in any file except app.js .

    var winNext = Titanium.UI.createWindow({
