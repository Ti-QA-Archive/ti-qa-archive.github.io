Alloy - Check if a record exists in a collection

What's the best way to check if a record exists in a collection?

I use (typeof $.collection.get(id)!=="undefined"). Any better/standard way?

    I'm assuming that you're referring to a backbone collection which contains an array of models. If that's the case, then you could use the where method to search for any models (records) in the collection like so:

    var recordID = "123";
var recordCount = 0;
if(recordCount = $.collection.where({ id : recordID }).models.length){
    alert("I found " + recordCount + " record(s).");
}
