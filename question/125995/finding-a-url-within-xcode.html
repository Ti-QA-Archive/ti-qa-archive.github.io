finding a URL within Xcode

I used a variable in Javascript code. I know the URL but when I try to search text within Xcode project then I don't find it. Why is like that? Does Titanium compile even the value of Variable like URL? I am a bit lazy to recompile everything after making changes. Thanks

— asked September 25th 2011 by Adnan Ahmad

  • You won't find any JavaScript code in your Xcode projects. Your JS files are not referenced statically at run time. Instead, your JS files are compiled into bytecode and referenced as a c variable via JavaScriptCore, directly to UIKit. This enables Titanium to perform operations directly to platform code through an interpreter, thereby achieving the "native" experience that you expect as a native developer.
