Google Maps on Ti 3.1 beta - Hybrid mode not working?

I'm using the Titanium.Map system (which as of 3.1 defaults to the Ti.Map module for Android so you can use Maps Android API v2).

Now, the maps work find on both iOS and Android, however on the Android one, the maps are showing as 'NORMAL_TYPE' (i.e a drawn map). If you try doing mapType: Titanium.MAP.HYBRID_TYPE, on iOS the result is as expected, however on Android it still shows the normal drawn map.

Could someone else who's used this module quickly try this on Android as I cant tell if its a bug or if there's something odd going on just for me.

Code obviously isnt really needed here as its a standard map, but if it makes a few happy:

var mapLabelView = Titanium.Map.createAnnotation({
    latitude:<LAT HERE>,
    longitude:<LON HERE>,
    title:'Some Label Title Here',
    pincolor:Titanium.Map.ANNOTATION_RED,
    animate:true
});

var mapview = Titanium.Map.createView({
    mapType: Titanium.Map.HYBRID_TYPE,
    region: {latitude:<LAT HERE>, longitude:<LON HERE>, 
            latitudeDelta:0.01, longitudeDelta:0.01},
    animate:true,
    regionFit:true,
    userLocation:true,
    annotations:[mapLabelView]
});
win.add(mapview);
