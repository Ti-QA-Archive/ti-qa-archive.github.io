  • Accepted Answer

    You can set the default background for android and iOS to be a transparent PNG.
    Then, assuming you are using Alloy framework, on your index controller, before you open your main application window, you open a window that consists of a web view object that loads the ad.

    similar question was asked here: http://developer.appcelerator.com/question/129572/how-to-remove-splash-screen-its-possible
    In this case, the solution was to create a screen shot of the app window because the person did not want to have a slash screen.
