Activity indicator in a imageView

How can i run an activity indicator inside a imageview (with Android)?

I tryed with:

var actInd = Titanium.UI.createActivityIndicator();
actInd.message = 'loading...';
imageview.add(actInd);
actInd.show();

but nothing seems to happen…

    Hello Matteo Mosconi,

    Add activityIndicator and image to a view, then show activityIndicator, it will work for android like this sample code.

    var win = Ti.UI.createWindow({
    backgroundColor : 'blue',
    layout : 'vertical'
});

if (Ti.Platform.name === 'iPhone OS') {
    var style = Ti.UI.iPhone.ActivityIndicatorStyle.DARK;
} else {
    var style = Ti.UI.ActivityIndicatorStyle.DARK;
}
var activityIndicator = Ti.UI.createActivityIndicator({
    color : 'red',
    font : {
        fontFamily : 'Helvetica Neue',
        fontSize : 26,
        fontWeight : 'bold'
    },
    message : 'Loading...',
    style : style,
    top : 10,
    left : 10,
    height : Ti.UI.SIZE,
    width : Ti.UI.SIZE
});

var image = Ti.UI.createImageView({
    image : 'test.jpg'
});

var view = Ti.UI.createView({
    width : Ti.UI.SIZE,
    height : Ti.UI.SIZE,
    top : 20
});

view.add(image);
view.add(activityIndicator);
activityIndicator.show();

win.add(view);
win.open();

    If you have any concern, then let me know.

    Thanks.

  • Hi Matteo Mosconi,

    Add your activity indicator to the window, that should work.

    ActivityIndicator is a view and, like any view, must be added to a window or other top-level view before it can be shown. Unlike most views, ActivityIndicator is hidden by default and must be shown explicitly by calling its show method.

    Check the link http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.ActivityIndicator.

    Hope it helps.
