2.1.4.GA - Support

We have compiled our app using 2.1.4.GA, and found that a user reported that 4.2.1 of iOS does not run the app.

From the documentation for 3.0:

Titanium SDK Version Min iOS/SDK Version Max iOS/SDK Version
2.1.3 - latest 4.0.x 6.0.x

Seems that it only works for 4.3.0 and above, and nothing below that.
As some iPhones have a limit on what version of iOS it can install.
