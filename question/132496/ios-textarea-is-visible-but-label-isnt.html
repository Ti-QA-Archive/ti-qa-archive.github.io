iOS: textArea is visible but label isn't

Hi all.

I've been working on an app that has a window with some text and a link.
since I want the link to be clickable but have an actual description I wanted to use a text based HTML UI element.

so firstly, just for trial I added a textArea and all went good, but when I tried to add the label to the window it wouldn't show.

here is my code:

function createPostWindow(post){
     //the base window
    var win = Ti.UI.createWindow({
    title: 'post window',
    modal: false,
    navBarHidden: false
    )};

    //the parent view for the text
    var parentView = Ti.UI.createView({
    right: '5%',
    left: '5%',
    top: '10%',
    bottom: '10%',
    backgroundColor: 'transparent'
    });

    //background coloring view
    var backgroundColorView = Ti.UI.createView({
    backgroundColor: 'yellow',
    opacity: 0.3,
    height: '100%',
    width: '100%'
    });

    //text area with the post text
    var text = Ti.UI.createTextArea({
    value: post.text,
    color: 'black',
    top: '3%',
    bottom: '7%',
    font: {fontSize: 17},
    backgroundColor: 'transparent',
    editable: false,
    scrollable: true
    });

    //the link 
    var link = Ti.UI.createLabel({
    html: '<a href=\"'+post.link+'\">read full article</a>,
    bottom: '5%',
    left: 20
    font: {fontSize:20}
    });

    //adding all the UI elements to the window
    parentView.add(backgroundColorView);
    parentView.add(text);
    parentView.add(link);

    win.add(parentView);

    win.open();
}; //end of function

here is a screenshot of the result.

p.s. working on Mac with Titanium Studio 1.0.8.2
