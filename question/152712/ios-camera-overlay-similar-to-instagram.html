    Hi

    Yes - simple;

    You use the property overlay property which you provide a view you have built up to display over the camera.

    You simply add the property as part of the showCamera method, other parameters here; CameraOptionsType

    Take note of the following;

    • To suppress the default controls, set showControls and autohide to false.

    If the showControls option is false, you must control the camera explicitly, using the takePicture and hideCamera methods.
