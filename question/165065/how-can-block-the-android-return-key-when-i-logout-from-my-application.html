How can block the android return key when I logout from my application

  • What you need is this

    function handleBack(evt){
    evt.cancelBubble = true;
};

myWindow.addEventListener('androidback', handleBack);

    and don't forget to:

    myWindow.removeEventListener('androidback', handleBack);

    when the window closes
