include from worker

I'm creating a worker:

var worker = Titanium.Worker.createWorker("worker.js");
worker.start();

And from worker.js I would like to include("ruby.rb") in order to run some ruby scripts. But all I get is:
Error executing worker: (JavaScript.KKJSObject) { message : "Can't find variable: include…

Isn't it possible to run ruby scripts from a worker? I'm using the 1.1.0 SDK.

      You can use the 1.1.0 SDK as long as you use the proper syntax. Titanium.include() was not added until the release candidates. 1.2.0 is not released yet, so if you want to use that syntax you will need to install one of the release candidates. IOW, the 1.1.0 SDK requires the script tags inside the HTML. Here's the blog announcement for the release candidate 4:

  • If you want to include another script inside worker, you can use importScript or importScripts

    var worker = Titanium.Worker.createWorker(function() {
    this.importScript('another.js');
});
