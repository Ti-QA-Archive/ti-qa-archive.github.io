  • OK. I just saw your attachment in the email you sent to me.

    You can see on the alert showing the this.responseText that the server is returning HTML with a body and head tag etc.

    I see you have something like JSON inside the body tag. You need to remove ALL of the HTML surrounding that JSON data.

    JSON data does not contain HTML.

    Your Titanium application code is fine. But your server is set up incorrectly. I'm afraid we can't help you with that. But it should be a very easy fix, if you know PHP.
