Blur in textfield!!

When i open my app, the keyboard appears automatically. I don't want it.
So I tried to call the method blur in the textfield but don't works.
How to solve this?

the code:

//....

var txt_busca = Titanium.UI.createTextField({
        borderRadius:5,
        height:38,
        left:7,
        top:13,
        width:'65%'

win.add(txt_busca);

win.addEventListener('open',function(){
        txt_busca.blur();
    });

  • I've had this same issue, but only in certain newer versions of Android, especially on tablets. This is a total hack, but what I had to do was set a timeout and then blur after 100-200 milliseconds.

    • The TextField has a focusable property that states:

      "Whether view should be focusable while navigating with the trackball. (Android only.)"

      I was hoping that setting this to false would resolve the issue, but it actually disables the entire TextField so you can't select/type in it anymore.

      You could log a bug in jira for this issue. I'm sorry I don't have better news, I spent an hour looking into this a few weeks ago and couldn't see any properties that would resolve the issue.
