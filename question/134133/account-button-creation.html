  • If the "Accounts" button is the back button that comes with the OS when the tab opens a new window all you have to do is to create the window like this:

    
var win = Ti.UI.createWindow({
    backButtonTitle:'Accounts',
    //... etc

})


tab.open(win);

    If is the first window in the tab and you open a new tabgroup you will need to use a backgroundImage for the leftNavButton.
