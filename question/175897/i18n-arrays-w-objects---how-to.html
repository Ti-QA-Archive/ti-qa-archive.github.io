i18n arrays (w. objects) - How to?

I want to internationalize my app and it works fine with i18n until i try to use arrays (with or without objects)

I need to get the length from the array, but its always undefined.

Where am I wrong?

in i18n strings.xml:

<string name="testarray">"Saab", "Volvo", "BMW"</string>

<string name="testarray_2">["Saab", "Volvo", "BMW"]</string>

in my.js

var homeMenuVar = [L('testarray')];
Ti.API.info('homeMenuVar ' + homeMenuVar);
Ti.API.info('homeMenuVar.length ' + homeMenuVar.length);

var homeMenuVar = [L('testarray_2')];
Ti.API.info('homeMenuVar ' + homeMenuVar);
Ti.API.info('homeMenuVar.length ' + homeMenuVar.length);

// here i insert the same array directly (no i18n) in the script and it works
homeMenuVar = ["Saab", "Volvo", "BMW"];
Ti.API.info('homeMenuVar ' + homeMenuVar);
Ti.API.info('homeMenuVar.length ' + homeMenuVar.length);

results in:

[INFO] : homeMenuVar "Saab", "Volvo", "BMW"

[INFO] : homeMenuVar.length 1

[INFO] : homeMenuVar[1] undefined

[INFO] : homeMenuVar ["Saab", "Volvo", "BMW"]

[INFO] : homeMenuVar.length 1

[INFO] : homeMenuVar[1] undefined

[INFO] : homeMenuVar Saab,Volvo,BMW

[INFO] : homeMenuVar.length 3

[INFO] : homeMenuVar[1] Volvo

    Hi Marco,

    a normal String Array ("Saab", "Volvo", "BMW") should work like this (not tested):

    strings.xml

    <string name="testarray">Saab,Volvo,BMW</string>

    js

    var testArray = L('testarray').split(",");

    nested objects in an array will not work with this method.

    Thanks
