swapping picker column data not working on iPhone, but works on android.

I have several different sets in my pickerOptions object that I want to swap the picker values with depending on which button is pressed.

I have this simple code:

selectLocationView.addEventListener('click',function(e) {
picker.columns=[Ti.UI.createPickerColumn({
rows: pickerOptions.locations
})];

})

It works great in android, but doesn't work on iPhone… at least not from within a function. If I make the same picker.columns=[…] call before the view is loaded, it does update the picker values, but it isn't working within a the addEventListener (though if I add Ti.API.info() calls within the listener, they will fire properly. I've tried swapping with picker.setColumns(…) also with no luck in iPhone. Any advice?

    Are you calling picker.reloadColumn()?

    In case that doesn't fix it:

    I remember having some difficulty getting this working cross-platform, as well. Here's my (currently working) approach. Assume that n is the column number I'm reloading, and listData is what I'm reloading it with:

    picker.columns[n].rows = null;
var i;
for (i = 0; i < listData.length; i++) {
    var row = Ti.UI.createPickerRow({
        title: listData[i]
    });
    picker.columns[n].addRow(row);
}
picker.reloadColumn(n);

    (Changed a little for readability and removing unnecessary code; hopefully it's still close to working as given.)
