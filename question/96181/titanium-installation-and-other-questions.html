  • For most people the steps are as follows:

    *Make sure XCode is installed for osx.

    1. Download Titanium from appcelerator.com
    2. Install Titanium.
    3. Run Titanium, it will find an update for the Titanium itself.
    4. It will also download the latest mobilesdk
    5. Create a project.
    6. Compile and run in emulator.
    7. If you want to run on device, this is where you have to follow the on-screen messages carefully. But basically:
    8. Get all your provisioning set up with apple and in your keychain.
    9. Load those files into Titanium via the Run on Device tab.
    10. It will compile and drop in iTunes. For me, I take the .IPA and move it to another computer where I still sync my phone. Sync.

    Now this is for OSX, maybe someone can chime in for Windows but I imagine it's very close to the same. Only a limited few have problems getting it working on the first try I would say.
