  • In order to develop mobile applications with Titanium, you'll need Titanium Studio and the related iOS / Android SDK. So after installing Studio, you'll be asked to download and install these SDK in order to compile for each platform. But for example, if you don't want to develop on Android, you won't need the Android SDK and you'll just have to install the iOS SDK.

    Everything is well explained directly in Titanium Studio after first launch.

  • Once Titanium Studio is downloaded is automatically will download the Titanium Mobile SDK, and then ask you configure the Android and iOS SDKs. As JB mentioned, the dashboard page of Titanium Studio explains this.
