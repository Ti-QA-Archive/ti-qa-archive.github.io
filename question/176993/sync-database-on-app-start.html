Sync database on app start

Hello,

I'm fetching the content for my app via an external source. This content is stored in a local database. Now I want the sync function (this function is checking for new content and saving it to the local database) to be executed on app start. I want the splash screen to be visible as long as this function is being executed. So the index can't be loaded, before the sync function is completed.

Does anyone know how to accomplish this?

    So, the 'splash screen' only shows until the first line of code is executed… So what you want to do is create your own version of the 'splash screen' and pop it up first thing. Then you can execute your database sync and remove your splash screen after the sync is complete.
