    • ok Ivana, there is no project "file" there is a project "folder". In Titanium Studio you have your project list on the left, right click on the Project -> show in -> finder. now copy the downloaded zip files into that folder, there is no need to unzip them, titanium does automatically when you rebuild your project.
      alternatively you can follow this guide

      to the syntax:
      this will not work, because "Password" and "password" are different keys. 

      SecureProperties.setString('Password', password);
SecureProperties.getString("password");

      this will work ;)

      SecureProperties.setString('password', password);
SecureProperties.getString("password");

    • see the documentation of the properties

      secret : String

This is a required parameter. secret is the password used to encrypt and decrypt all property values. The same secret used to encrypt must be used during the decryption process or a null value will be returned.

      you have to initialize the properties with a secret key, then the module automatically en/decrypts with this secret

      var properties = securely.createProperties({
    secret:"this_is_the_secret_key",
    identifier:"myPropertyIdentifier",
    accessGroup:"myAccessGroup",
    encryptFieldNames:false
});
