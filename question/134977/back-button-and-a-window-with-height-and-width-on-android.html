Back button and a Window with Height and Width on Android

I have a root window and have a button on it. When I click on it I create a smaller window as a popup as below.
Problem is when click on the back button, the root window is closed as well. I think this is because the popup window is not a heavyweight window. Is there any way to just close this popup window when I press the back button without closing whole app?

Android SDk 2.2, Ti SDK 1.8.2

var self = Ti.UI.createWindow({
   navBarHidden : true,
   backgroundColor : '#E01BC2',
   exitOnClose : true
});

var btn = Titanium.UI.createButton({
   top : 5,
   width : 33,
   height : 33,
});

btn.addEventListener('click', function() {
   var win = Ti.UI.createWindow({
      width : 200,
      height :300,
      left : 60,
      top : 50,
      backgroundColor:'#fff',
      borderWidth:2,
      borderRadius:3,
      borderColor:'#000'
   });
   win.open();
});
self.add(btn);
self.open();

  • Hi Siva,
    Try this this is working for me i have tested 

    var options = {
            height:0,
            width:0,
            backgroundColor:'#336699',
            bottom:0,
            right:0
        };
    if (Ti.Platform.name == 'android') {
        options.navBarHidden = true;
    }
    var w = Titanium.UI.createWindow(options);

    setTimeout(function(e){
        w.height = 200;
        w.width = 200;
    },1000);

    w.open();

    • It is the problem with navBarHidden again. If you set it to true then you get a heavyweight window. And they cannot be windowed with width and height. They will always cover the fullscreen in Android.

      I think the only solution is to capture the back button event and remove it manually.
