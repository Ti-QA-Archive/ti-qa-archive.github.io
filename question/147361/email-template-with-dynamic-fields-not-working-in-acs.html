Email template with dynamic fields not working in ACS

I cannot get email templates to work with dynamic fields. I tried both encoded and regular JSON. No avail, the tutorial does not show a specific example of how to do it. 

ACS.Emails.send({
                                    template: 'DailyStatus',
                                    recipients: user.email,
                                    // dynamic_fields: JSON.stringify({ 'username' : user.username, 'date' : today })
                                    dynamic_fields: 'date=' + today
                                }, function(data) {
                                    if (data && data.meta && data.meta.code == 200 && data.meta.status == 'ok') {
                                        logger.debug('Email sent successfully to ' + user.email);
                                    }
                                });
