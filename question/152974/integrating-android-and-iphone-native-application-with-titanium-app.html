Integrating android and Iphone native application with titanium app

Hi ,

  I have created an android(java) and iphone(objective-c) native          application .  I want to integrate titanium application with native       application . By creating module for both android and iphone we can       integrate native apps with titanium app is one solution . Im not having    native code for both android and iphone . In this scenario how can i     integrate native application with android apps ? Is this possible?

    No you cannot do that.

    A native module cannot embed another application if you do not have the source code. Keep in mind that native modules are not full-fledged application, but rather components that implement a specific feature.
