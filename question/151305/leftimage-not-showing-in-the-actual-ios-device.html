    Hi

    From your very little information so far, it is possible that you have simply placed the images in the wrong the location or your folder name has a different case to that of the path string.

    Your code suggests your image is located here; image/xyz.jpg you are much safer denoting the location with a leading / in the path, like this - /image/xyz.jpg.

    Your image would need to be located in one of the following locations if it is to be found;

    • {app name}/resources/image/xyz.jpg <- available to all platforms when compiled
    • {app name}/resources/iphone/image/xyz.jpg <- available only to iOS apps when compiled

    Make sure that the file name AND the folder are the same case, for simplicity choose lower-case for all folders and files whilst you are resolving the problem. Different cases will certainly cause different outcomes on simulators and devices - as will the lack or true file paths.
