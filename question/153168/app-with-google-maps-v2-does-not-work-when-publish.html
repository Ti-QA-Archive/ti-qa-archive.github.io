    I would:

    1. clean build and rebuild the project

    2. check that the map module is updated and titanium studio and SDK are updated as well

    3. check the map module on app startup like this for example

      if (MapModule != null || MapModule != undefined)
{
 //do something
}
else
 alert("module is null");

    4. have you updated the this string &lt;meta-data android:name=&quot;com.google.android.maps.v2.API_KEY&quot; android:value=&quot;PASTE YOUR GOOGLE MAPS API KEY HERE&quot;&#x2F;&gt;

    5. is your app signed with the right keystore

    6. and is your keystore registered with google on code.google.com/api/console
