How to avoid adding commonJS file every time to use it.

I have created a simple commonJS file which exports a function.
now to use that commonJS file I do var xyz = require(&#39;myJS&#39;); and I just use it..
now if I want to use my commonJS file to any other JS file, I again have to use the require syntax..

Is there any way to just add my commonJS module one time so that I can use it anywhere without adding it again and again.

I have heard that commonJS has only a local scope..is that true??

    Yes it is only in local scope, but it is loaded and cached globally. MEaning, if you use var xyz = require(&#39;myJS&#39;); it will load module once, and when you use require again, it will use cached version, without loading module again.
