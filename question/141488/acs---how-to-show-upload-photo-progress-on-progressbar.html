ACS - How to show upload photo progress on progressbar?

Hello,

I am using ACS to upload a photo. Is there a way to use a progressbar to show the uploading progresses?

Thankful for all help!

    There are two events you can grab.

    Cloud.onsendstream

    Cloud.onsendstream = function (e) {
    Ti.API.info('Please Wait, Uploading Photo '+(Math.floor(e.progress * 0.5*100)*2)+'% Complete');
};

    Cloud.ondatastream

    Cloud.ondatastream = function (e) {
    Ti.API.info('Please Wait, Uploading Photo '+(Math.floor(e.progress * 0.5*100) + 0.5)*2+'% Complete');
};
