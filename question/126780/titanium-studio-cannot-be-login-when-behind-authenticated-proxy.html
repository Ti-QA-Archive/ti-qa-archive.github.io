Titanium Studio cannot be login when behind authenticated proxy

When i try to login with my email and password,
the message "Logging in" has been displayed for few minutes and i cannot login to Titanium Studio.

Same email and password work fine getting into the website.

In order to login from behind authenticated proxy,
i tried followings but i could not login to Titanium Studio.

  • Configuring the proxy settings in login screen
    http://wiki.appcelerator.org/display/tis/Using+Studio+From+Behind+a+Proxy

  • Configuring the proxy settings in TitaniumStudio.ini
    <Titanium Studio Install Directory>\TitaniumStudio.ini
    Add following lines.
    -Dhttps.proxy=true
    -Dhttps.proxyHost=<proxy host>
    -Dhttps.proxyPort=<proxy port>
    -Dhttps.proxyAuth=true
    -Dhttps.proxyUser=<user name>
    -Dhttps.proxyPassword=<password>
    -Dhttp.proxy=true
    -Dhttp.proxyHost=<proxy host>
    -Dhttp.proxyPort=<proxy port>
    -Dhttp.proxyAuth=true
    -Dhttp.proxyUser=<user name>
    -Dhttp.proxyPassword=<password>

  • Configuring the proxy settings in net.properties
    <Titanium Studio Install Directory>\jre\lib\net.properties
    Set java.net.useSystemProxies to true and input required settings.

  • Configuring system environment variables
    http://wiki.appcelerator.org/display/guides/Installing+on+Windows+7#InstallingonWindows7-CannotlogintoTitaniumDeveloper

It's appreciated if someone can guide me what needs to be done.

P.S.

"Authentication failed. Please verify that the user name and password are correct." is shown and i failed to authentication.
This behavior is different from behavior in Titanium Studio 1.0.5.
