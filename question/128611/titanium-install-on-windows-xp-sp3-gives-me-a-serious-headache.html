    Alex

    First of all, check that you have, at a minimum, all the packages shown in the screenshot of the Summary of Installed Android Packages section of the documentation.

    Install the latest continuous build, by selecting master from the drop down menu, right-clicking and copying the URL of the archive for your OS, and pasting it into Titanium Studio's Help > Install Titanium SDK from URL dialog. Note that you don't have to actually use this version of the SDK for your apps; it just needs to exist on your system.

    Make sure that Studio is configured with the location of the Android SDK, via its Preferences > Titanium Studio > Titanium dialog. See Configuring Titanium Studio with the Platform SDKs.

    Then restart Studio and try again.

    If this does not resolve it, we need to verify that your environment is configured correctly. To do this, please follow the instructions at Install Diagnostics for Windows. Copy the output and paste it to your preferred pastebin service, and paste the link to it here. It's important that you do not modify it in any way, as this could mask problems with your configuration. Do NOT paste the actual code here! This is because the system will change the formatting.

    Also, please create and share a screenshot of your android installed packages, using something like ImageShack

    State the processor and RAM installed to your system.

    Lastly, run the command systeminfo at the commandline and share it in a pastebin.

    Then I will help you further.
