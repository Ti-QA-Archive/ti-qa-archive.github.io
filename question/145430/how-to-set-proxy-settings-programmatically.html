    are you trying to reconfigure the Proxy settings of your android device within your app? In that case, you should read this:

    http proxy with android

    From the Ti Appc API, there is no way to change the proxy settings of the phone within the app.

    If the security allows you, you can create a native module that will be able to do it. Check more about how to create modules here:
