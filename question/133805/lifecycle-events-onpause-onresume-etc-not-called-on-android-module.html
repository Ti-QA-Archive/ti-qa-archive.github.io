Lifecycle events (onPause, onResume, etc...) not called on Android Module

Hi, i'm working on an android titanium module but i have problems getting activity lifecycle events within the module.

I implement the OnLifecycleEvent interface in my TiViewProxy class, and i've added the OnLifecycleEvent methods (onPause, onResume, onStart, etc…).

I put some Log.d in these methods to see if they gets called, but they don't. I'm trying the module on the device (using ant install and then launching it) with adb logcat to see the debug output, but the methods seems that they don't get called.

There is any reason for that ?

I'm using Titanium Mobile 1.8.2 sdk

I hope someone can help me because i don't know where to find helps or clues.

Thanks!

  • i think i just faced the same problem. Add this in your setActivity method

    if (activity instanceof TiBaseActivity) {
            ((TiBaseActivity) activity).addOnLifecycleEventListener(this);
        }

    it worked for me!
