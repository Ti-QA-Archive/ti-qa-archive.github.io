Dynamic styling android using window.applyproperties not working

Hi, I encountered a problem when I try to dynamic change the title of the window. The code is as following:

index.js

function changetitle(){
    $.index.applyProperties({title: "heyheyhey!!"});
}
$.index.open();

index.xml

<Alloy>
    <Window class="container">
        <Label id="label" onClick="changetitle">changetitle</Label>
    </Window>
</Alloy>

When I click the label, I suppose the title should change to "heyheyhey!!", however, it doesn't. One more thing strange is that, when I click the home button and then resume back to the application(the window reopen), the title changed to "heyheyhey!!" actually.

I really wonder if someone has seen this situation before? Do I need to add some property that tells titanium that I need to refresh the page? Thanks!

My SDK is 3.4.0 and I am using the android simulator with API 19. It may be the problem with simulator due to its unbelievable slowness though…

    Hi

    I would give the window an ID of say "win" and then simply call the following inside your function.

    $.win.setTitle("hey hey hey!");

    This does mean you would have to use $.win.open(); to open the window instead of $.index.open()

    Hope this helps
