Site doesn't resize in WebView

Hi all,
when I open a site in this Web View, the page doesn't resize in base of dimension specified in the code.
This happen with every url; why?

var fbUrl = "http://www.site.com/page1.html";

var extwebview = Titanium.UI.createWebView({
     top:0,
     left:0,
     right:0,
     url:fbUrl, 
     height:440,
     width:320,
     borderRadius:12,
     zIndex:2,
     backgroundColor:'#FFF'
});
win1.add(extwebview);

  • Yes, the page contain <meta name="viewport" content="width=device-width, initial-scale=1.0, minimum-scale=1.0, maximum-scale=1.0, user-scalable=no" /> - With iPhone the page is see well.

  • Ok, I resolve with this meta: <meta name="viewport" content="initial-scale=1,maximum-scale=1,user-scalable=no"/>
