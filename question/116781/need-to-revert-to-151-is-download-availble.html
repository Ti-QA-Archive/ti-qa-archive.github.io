  • You should be able to change Titanium SDK version in Titanium developer. Go to Edit tab and there should be option "Titanium SDK" where you can choose what version of Titanium SDK your project will use.

    If there isn't 1.5.1, you can download latest version from 1.5 branch from here:

    http://builds.appcelerator.com.s3.amazonaws.com/index.html

    There is 1.5.2 build.
