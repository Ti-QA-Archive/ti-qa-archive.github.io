TypeError: Cannot find function createDashboardItem in object Ti.UI

Hi, my application is not running on Android emulator as it is yielding a run time error saying " TypeError: Cannot find function createDashboardItem in object Ti.UI "

Can someone help on this, I am using Titanium 1.2.2 SDK with latest mobile SDK.

  • Please observe the big red banner when posting a question, which requests that you set some question tags. ;) It's a good place to put your mobile platform and SDK version numbers.

    If you look at the API for Ti.UI.DashboardItem, note that there is only an apple icon in the top right-hand corner of the page. As it is missing the android icon, like on pages such as Ti.UI.TableView, it means that, unfortunately, Dashboard is an iOS-only feature.
