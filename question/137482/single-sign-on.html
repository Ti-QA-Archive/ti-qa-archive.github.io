hello,
I have a login form in my android app and want to remember user name and password of a user on user's device only same like SSO(single sign on). User need not to login again on his device. How to implement this function in titanium?

  • You can save user name and pasword in the App properties and when you start the app the first this should be to check the values in your app properties for username and password.

    These saved values will be there till you uninstall your app from your device.

    to save your username and password in app properties you can do 

    Ti.App.Properties.setString('Username','YOUR VALUE');
Ti.App.Properties.setString('Password','YOUR VALUE');

    To get the values from there

    Ti.App.Properties.getString('Username','YOUR VALUE');
Ti.App.Properties.getString('Password','YOUR VALUE');

    Apply a check to see that they dont have any undefined values, if not you can bypass your login screen and if those variable has undefined values then you can show your login screen..

    hope that helped :)
