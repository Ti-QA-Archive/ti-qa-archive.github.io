  • Titanium does not compile JavaScript, your JS code is interpreted by JS engine (JSC on iOS, v8 or Rhino on Android).

    In Titanium JS environment, there are host objects provided by environment (they are present and created by methods in "Titanium"/"Ti" namespace), which implementation is native. So, Titanium framework is a bridge to native functionality and native components from JS code. "Date", "String" and other native JS objects are present in JS engine, while for example "Ti.UI.createButton" in the background calls "ButtonProxy" which creates "Button" native element. You can store it JS variable as host object and modifying properties of that host object in the background calls native functionality in similar fashion.
