  • Use vb code to add image to pdf, the image can be inserted at any position of the pdf document, the first page or at the last page:

    Public Sub AddImagetoPage(page As PDFPage, imgPath As [String], positionX As Single, positionY As Single)
    Dim img As REImage = DirectCast(REFile.OpenImageFile(imgPath, New PNGDecoder()), REImage)

    ' If no compression scheme specified. the RasterEdge PDF will decide this.
page.AddImage(img, positionX, positionY)

    End Sub
