Alloy getting a View

Hello everyone,

I have just started using Alloy in my projects, and I am running in a few issues when it comes to getting aview that I have created in the view folder.

So essentially here's what's happening:

I have created a view in a file named: schedule.js. Like so:

<Alloy>
  <Window id="myId">

        ....
                .....

        <View id="datesView"></View>

        <ScrollableView id="ScrollableView"> </ScrollableView>

        ......

  </Window>

</Alloy>

As you see in the code snippet, I also have a ScrollableView to which I am going to add the &lt;View id=&quot;datesView&quot;&gt; &lt;&#x2F;View&gt; into in my controller.

Now, the big issue is on my controller, which is where I fetch the datesView by using $.datesView and push it into the ScrollableView created.

This is how I have been doing it:

....
....
  var totalViews = jsonObject.available.length;

     for( var i = 0; i < totalViews; i++){

          var myView = $.datesView;

            VU.push(myView);

     }

    $.ScrollableView.views =  VU;

.. and I don't seem to be able to have the view showing in the ScrollableView at all!
All I am trying to do is to get the view itself to show. Technically, as you can see, I am getting the view which, I then, put it into myView variable, and then pass to the array which is then pushed to the ScrollableView.

Any ideas on how do I get the view to actually be inserted into the ScrollableView ?

Thanks in advance.

  • Instead of doing this:

    VU.push(myView);

    Try this, directly add to the ScrollableView

    $.ScrollableView.addView(myView);

    Make sure to also get rid of $.ScrollableView.views = VU; I dont think this actually has any effect, views is probably read-only.
