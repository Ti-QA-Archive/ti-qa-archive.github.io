  • Hi

    The likely problem is that the Ti.Map should no longer be used for Android maps.

    As Google has now deprecated the Google Maps v1 platform (accessed via Ti.Map) they are no longer issuing API keys for it. Appcelerator has now provided support for Google Maps v2 via a module plugin which you can access using the documentation;

    • Modules Map
    • Google Maps v2 for Android

    The code is very similar but you now get the better v2 rather than version 1.

    You will need to obtain a SHA1 based API key, the instructions are in the docs I mentioned above.
