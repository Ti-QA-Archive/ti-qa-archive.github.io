Titanium.UI.iPhone.TableViewStyle.GROUPED

When I use the following style for a table view:

style: Titanium.UI.iPhone.TableViewStyle.GROUPED

Then a background with vertical lines appear behing the group.
Is there a way to make that transparent?

— asked May 31st 2011 by Sajid Saiyed
  • tableview
  • tableviewstyle.grouped

  • var win = Ti.UI.currentWindow;

var tableView = Ti.UI.createTableView({
   style:Ti.UI.iPhone.TableViewStyle.GROUPED,
   backgroundColor:'transparent'
});

win.add(tableView);

    • This works great. Only thing I would like to add for anyone trying this in future is, this makes the table rows also transparent. Hence you will need to add row background color if you dont want it to be transparent. Cheers!
