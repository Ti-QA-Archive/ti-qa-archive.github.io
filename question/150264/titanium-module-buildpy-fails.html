Titanium module build.py fails

I'm trying to make an iOS module for Titanium. I selected "File"->"New"->"Titanium Module Project", filled out the info, and it created a directory for me in my workspace. I opened the xcode project and it built fine. According to the instructions at https://wiki.appcelerator.org/display/guides/iOS+Module+Development+Guide I should next run build.py in that directory, but it fails:
