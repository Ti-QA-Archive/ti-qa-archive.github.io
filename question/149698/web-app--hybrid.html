    • Alloy is not translating to HTML. Titanium does not create HTML. This is a MAJOR misconception if I am understanding you correctly. Titanium is very different than most cross-platform frameworks in that it maps to native calls, it is not just wrapping a webview in a native app.

      This is a huge subject, covered very well in a blog post by Kevin Whinnery: http://kevinwhinnery.com/post/22764624253/comparing-titanium-and-phonegap

      Essentially, it's up to you if you choose to use what you consider to be easier in HTML, or if you want true native performance and functionality that is provided with Titanium.

  • In many cases it is a bad idea to use a webview. But if you want:

    
var win = Ti.UI.createWindow();
win.add(Ti.UI.createWebView{disableBounce:true,url:YOUR_URL}));
win.open();

    You can also use phonegap to realize this.
