    I will just give you a quick install guide since I have pretty much had to do it a million times due to unforeseen breaking of things on my boxes. (I did mention that I run arch and have a tendency to h9k at things I should probably learn more about first. But then, thats no way to learn now is it? lol)

    Anywho, here goes:

    Install pre-req's:

    sudo apt-get install libjpeg62 libwebkitgtk-1.0-0 ia32-libs
echo 'export MOZILLA_FIVE_HOME=/usr/lib/mozilla' >> ~/.bashrc

    Get Java:

    Java (Oracle) JDK/JRE available inline link text

    (Just Check the box saying you agree and download the I know that you run a .deb system, but I prefer to get it myself and install it. And it works, but what do I know.)

    Make sure that you download the proper version, i.e. get

    jdk-7u51-linux-x64.tar.gz

    Go ahead and do a number like this to get java home:

    ?( jgalt@l3g10n )?(03:33 PM Sun Apr 06)
??(/opt/Java)?(1 files, 12Kb)?> $ sudo mkdir /opt/Java && cd /opt/Java && sudo tar -xvf ~/Downloads/jdk-7u51-linux-x64.tar.gz


# then,

sudo touch /etc/profile.d/jdk.sh
sudo echo 'export JAVA_HOME=/opt/Java/jdk1.7.0_51' >> /etc/profile.d/jdk.sh
sudo echo 'export PATH=$PATH:$JAVA_HOME/bin' >> /etc/profile.d/jdk.sh

    Go ahead and get and extract the Titanium Studio archive:

    sudo mkdir /tmp/ti && cd /tmp/ti && sudo wget http://titanium-studio.s3.amazonaws.com/latest/titanium.linux.gtk.x86_64.zip

cd /opt && sudo unzip /tmp/ti/titanium.linux.gtk.x86_64.zip

    Start up Titanium

    cd /opt/Titanium_Studio/ && ./TitaniumStudio

    This should get you up and going.
