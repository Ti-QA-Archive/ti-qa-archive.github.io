  • Ok after a long night sleep and having a clear mind i removed all the certificates for a third time, reinstalled them and it worked! Let me describe the process once more for [my] future reference:

    1. Start removing certificates from Xcode Organizer. Remove everything from Xcode
    2. Go to [USER]/Library/MobileDevice/Provisioning Profiles and remove everything from there, do the same for /Archived Applications
    3. Open keychain app and remove all certificates from there as well
    4. Open Titanium and select and delete each certificate you see (use the red X icon next to each combo menu)
    5. Restart your computer (just to be safe)

    Now let's go through all the certificates you will need, i always seem to forget the 4th one:

    1. Development Certificate
    2. Distribution Certificate
    3. Provisional Certificate
    4. Provisional Distribution Certificate [that's the b**]

    Now, each and every one of them you need to double click so that they'll install properly

    Fire up Titanium, give it a few hugs and it'll be ok :)
