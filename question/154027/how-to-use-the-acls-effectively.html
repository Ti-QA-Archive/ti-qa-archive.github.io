How to use the ACLs effectively?

I am having a hard time figuring out how to use the ACS Cloud Access Control Lists (ACLs) effectively. Since I cannot query them or query other objects' associated ACLs, I have to copy the "acl_id" to a custom field and filter search results that way. This seems to defeat the purpose of ACLs as I can just create my own UUIDs for this purpose. Maybe I am missing something or thinking about it wrong but here is the use case I am trying to solve:

  1. Users can "friend/follow" a system generated/controlled "profile".
  2. I am representing this profile as an ACL to create a sort of "stream" for that virtual profile since all content (Statues, Photos) will be created by other users on behalf of the system created profile.
  3. I want the real user to be able to see a "stream/timeline" of content associated with that ACL.
  4. I cant seem to do that without copying the acl_id to a query-able field which seems to defeat the purpose.

Any help about how to go about this would be awesome and I apologize if I am missing something simple since It seems to me this should be already possible. Thanks in advance!
