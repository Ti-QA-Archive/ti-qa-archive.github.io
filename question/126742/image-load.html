Image load

I have a sort of weirdly specific question: exactly what triggers an ImageView to be "loaded"? Does it load as soon as you declare it, or only when you append it to a window or a visible view?

I am having issues with triggering a function tied to a "load" event listener, and knowing exactly when the view is loaded would go a long way towards helping me figure it out!

    The image "loads" when all it's info is present in the control.
    Think to a remote image that needs to be downloaded from the internet. IT will trigger the load event when all the file is on your device.
