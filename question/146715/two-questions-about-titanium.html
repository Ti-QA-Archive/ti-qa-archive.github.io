  • Accepted Answer

    Using external Javascript libraries is not possible.

    Javascript Libraries that require the DOM of a browser cannot be used in Titanium, TRUE.
    HOWEVER, pure javascript libraries seem to work pretty well, including nearly all the node_js modules I have needed.

    Using HTML or CSS is no longer possible as it doesn&#39;t use Web-View for deploying the application.

    Wow, your source is REALLY old, because this change took place like 3 years ago. HTML+CSS can be used within a webview. However, the reason for using Titanium would be access to NATIVE UI objects. Thus the move so far back away from a Phonegapish mess of an implementation towards native.
