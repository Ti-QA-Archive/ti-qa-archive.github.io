  • Accepted Answer

    If you set both labels to an 'auto' width and then set the layout property to horizontal on the TableViewRow, then the labels will stack from left to right.

    • Just a tip as Anthony nailed the answer.

      Play around with using Ti.UI.SIZE and Ti.UI.FILL in place of the &#39;auto' as you will find these will always size consistently.

      • Ti.UI.SIZE = make THIS view the same size as the CONTENTS within it
      • Ti.UI.FILL = make THIS view the same size as the PARENT it was added to.

      When used with the layout property as explained by Anthony - it makes laying things out so much easier - &#39;auto&#39; can be confusing as the meaning is dependant on the view and how it is being used.

      As long as you are using sdk 2.0 or higher of course.

  • Have you tried creating both labels, at say, 320px width, textalign one left, and the other right. :) Make sure to position them in the same top/left as well with having the same height.

    var win = Ti.UI.createWindow({backgroundColor:'#FFF'});

var left = Ti.UI.createLabel({
    text:'LEFT',
    textAlign:'left',
    width:320,height:50,
    top:0,left:0
});
var right = Ti.UI.createLabel({
    text:'RIGHT',
    textAlign:'right',
    width:320,height:50,
    top:0,left:0
});

win.add(left);
win.add(right);
