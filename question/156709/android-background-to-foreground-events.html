Android background to foreground events

Hi,

Need to implement App level pause and resume events.

Requirement:
Need to show a customized alert on launch of the app and also whenever the user comes from background to foreground irrespective of the screen where the user was.( not only on first screen)

Able to achieve this in IOS using app level pause and resume events. Need to achieve the same for Android.

As per docs. " In Android ,activities are started,stopped, paused and resumed. Titanium generates lifecycle events for activities ,but doesn't generate application level start,stop, pause and resume events"

Kindly help out.

  • The following snippet works only on light weight windows. If properties like fullscreen,navBarHidden,modal are added It fails.

    var activity = Ti.Android.currentActivity;
['pause',resume'].forEach(function(e) {
    activity.addEventListener(e, function() {
        alert((new Date()) + " Activity: " + e + " HIT!");
    });
});
