'undefined' is not an object (evaluating 'Ti.UI.currentTab.open')

Hi,

I have a table view in a window and ad trying to open another window on click of a tableview cell click.

This is what I have for the event:

        tableview.addEventListener('click', function(e) {
        self.fireEvent('settingSelected', {
            name:e.rowData.title,
            url:e.rowData.url
        });
    });

    self.addEventListener('settingSelected', function(e){
        var settings_window = Ti.UI.createWindow({
            url:e.url
        });
        Ti.UI.currentTab.open(settings_window);
    });

The error I get is: 'undefined' is not an object (evaluating 'Ti.UI.currentTab.open')

Can anyone guide me to a solution on this? I have searched many websites for the past few hours trying to work out the issue.

Many thanks.

    just open like this

        self.addEventListener('settingSelected', function(e){
        var settings_window = Ti.UI.createWindow({
            url:e.url
        });
        settings_window.open();
    });
