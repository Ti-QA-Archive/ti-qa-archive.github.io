  • Hi guys,

    I have tested date picker. It works well in my environment.

    Android SDK: 4.4.2, 4.2.2,

    iOS SDK: 7.1

    Ti SDK: 3.2.3GA

    Try this code. 

    Ti.UI.backgroundColor = 'white';
var win = Ti.UI.createWindow({
    exitOnClose : true,
    layout : 'vertical'
});
//You need not set current date it will set default.
var picker = Ti.UI.createPicker({
    top : 50,
    type : Titanium.UI.PICKER_TYPE_DATE,

});

picker.addEventListener('change', function(e) {
    alert(e.value);
});

win.add(picker);
win.open();
