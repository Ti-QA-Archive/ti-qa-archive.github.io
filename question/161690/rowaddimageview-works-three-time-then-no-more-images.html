row.add(imageView) works three time, then no more images

I am working on an app, where part of the functionality is to take a picture and show it in a tableView:

(Code is stripped to a bare minimum)

With:

var data = [];
var tableview = Titanium.UI.createTableView({
   data:data
});

var row = Ti.UI.createTableViewRow({
});

// here the image is captured

var imageView = Ti.UI.createImageView({
    image: e.media,
        top: 10,
        left: 10,
        width: 88,
        height: 88,
});
row.add(imageView);

var labelID = Ti.UI.createLabel({
    left: 100,    
    text: "imageID"
});
row.add(labelID);
tableview.appendRow(row);

This works excellent! But, only for the first three imageViews! They are added just like they should be, but from there on, none are added anymore. The labels continue just fine to take their place in teh added rows, just no more images show up.

Any clues, please!? This is the nth night I spend with headaches over Appcelerator problems :(
