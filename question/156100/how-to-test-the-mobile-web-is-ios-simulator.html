  • You can test it by making a server,upload your application and use the link in the iOS simulator web browser.
    Alternatively you can test it through dropbox also. Follow the below steps:

    1. Package your app for mobile app into some folder.

    2. Copy that folder to your dropbox account's public folder.

    3. Just click once on index.html , and choose copy public link option showing on the top of the list.

    4. Copy this link and paste to your iOS simulator browser . It should run now.
