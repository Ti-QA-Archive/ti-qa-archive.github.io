Android - Stop sound when lock

I'm developing a game using Titanium. On iOS, there is no problem.

I play a sound in Alloy.js :

Alloy.Globals.ambiance  = Ti.Media.createSound({url:"/pages/sounds/ambiance.mp3",looping:true,volume:0.8});
Alloy.Globals.ambiance.play();

On android, when phone is locked, sound continues to play. Is there a way to avoid this ?

Thank you.

    Hello, Anthony,

    You need to registers a broadcast receiver to monitor if the device's screen is turned on or off. The broadcast is a system event sent by the Android OS.

    var bc = Ti.Android.createBroadcastReceiver({
    onReceived: function() {
        ambiance.stop();
    }
});

Ti.Android.registerBroadcastReceiver(bc, [Ti.Android.ACTION_SCREEN_OFF]);

    Hope this helps, Thanks.
