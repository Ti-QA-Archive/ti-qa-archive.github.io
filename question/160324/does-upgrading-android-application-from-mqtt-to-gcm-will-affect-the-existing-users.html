So I have changed the application to GCM from MQTT and also changed the TiApp.xml as follows (replaced mqtt using gcm)

<property name="acs-push-type-development" type="string">gcm</property>
<property name="acs-push-type-production" type="string">gcm</property>
<property name="acs-push-type" type="string">gcm</property>

  1. There are 119 android users already subscribed for pushnotification. If anyone among them didn't updated the application to newer version, will they get push notifications from newer version?
