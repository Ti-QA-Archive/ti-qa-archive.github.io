Appcelerator AIR sdk (CocoaFish), queryFiles issue!

I'm using the air SDK, and I managed to login with ease, also I managed to request a file URL with the showFile request..

Now I need to query and find out what files excist. But the query file request does not get me a responce.
(There is no query File request code, on the docs for actionscript)

I tried this:

public function listAvailableUrls( onSuccess:Function = null, onFail:Function = null ):void
{
    if(hasLoggedIn)
    {
        var queryReq:Object = new Object();
        queryReq.where = "{'name':'Appcelerator Cloud Services'}";
        if(onSuccess)
            dispatcher.addEventListener(AppceleratorEvent.URL_SUCCESS,onSuccess);
        if(onFail)
            dispatcher.addEventListener(AppceleratorEvent.URL_FAILED,onSuccess);

        sdk.sendRequest("files/query.json", URLRequestMethod.GET, queryReq, false, queryResponce);
    }
}

private function queryResponce(responce:Object):void {
    if(responce)
    {
        if(responce.hasOwnProperty("meta")) {
            var meta:Object = responce.meta;
            if( meta.status == "ok" && 
                meta.code == 200 && 
                meta.method_name == "queryFiles")
            {
                trace("Files are in the responce object..");
            }
        }
    }

}

The queryResponce is never getting called.

Any ideas?
