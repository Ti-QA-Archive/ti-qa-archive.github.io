Loading YouTube videos.

Hey guys,

I've been trying to load YouTube videos on my App, but without success.
I did a lot of research on the forums and found some code to try out, but none worked.

Here's what I'm trying to do:

var url = '<html><center>' +
                  '<object width="177" height="150">' +
                  '<param name="movie" value="http://www.youtube.com/v/' + data.videos[i].id + '&hl=en_US&fs=1&"></param>' +
                  '<param name="allowFullScreen" value="true"></param>' +
                  '<param name="allowscriptaccess" value="always"></param>' +
                  '<embed src="http://www.youtube.com/v/' + data.videos[i].id + '&hl=en_US&fs=1&" type="application/x-shockwave-flash"' + 
                  'allowscriptaccess="always" allowfullscreen="true" width="177" height="150"></embed></object>' +
                  '</center></html>';

var video = Ti.UI.createWebView({
            url: url,
            visible: true,
            height: 150,
            width: 230
        });

But my WebView is blank. It's not loading anything.
On the Console I get this message:

[WARN] couldn't load URL: app://com.mycompany.myapp/(null)

I've also tried this url instead:

        var url2 = '<html><head></head><body style="padding:0">' +
          '<iframe width="177" height="150" src="http://www.youtube.com/embed/' + data.videos[i].id + '?rel=0" frameborder="0" allowfullscreen></iframe>' +   
        '</body></html>';

But it didn't work either.

My video ID's are VALID.

What's wrong here?

Thanks!

    Turns out I was trying to define the url property of the WebView as an html string.

    This worked:

            var videoView = Ti.UI.createWebView({
            html: videoHtml,
            visible: true,
            height: 150,
            width: 230
        });

    And changed those var names. :)
    Thanks Ronnie Swietek for the heads up!
