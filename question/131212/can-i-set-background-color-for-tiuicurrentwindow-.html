  • Hi, Ti.UI.currentWindow is defined only for windows running in their own context, i.e. those opened through the "url" property. For example in app.js:

    var win = Ti.UI.createWindow({url: 'win1.js'});
win.open();

    in win1.js you can do:

    var win = Ti.UI.currentWindow;
win.backgroundColor = 'white';

    If your app is single context, that value will will be undefined.
    So you need to use a reference to the created window object. For example in app.js:

    var win = Ti.UI.createWindow();
win.backgroundColor = 'white';
win.open();

    Please, consider that using the first approach is lately discouraged by Appcelerator: https://wiki.appcelerator.org/display/guides/Mobile+Best+Practices
