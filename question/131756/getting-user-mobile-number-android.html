  • You can use

    TelephonyManager tMgr = (TelephonyManager)mAppContext.getSystemService(Context.TELEPHONY_SERVICE);
    mPhoneNumber = tMgr.getLine1Number();

    Also requires the READ_PHONE_STATE permission to be set in your manifest.
