    Hi Michael,

    the wiki page Titanium 2.0 Layout Changes is a good source for understanding about Layout and Universal Unit Support.

    From this wiki:

    If no units are specified, a system-default unit is assumed. The system default unit is:

    • pixels on Android
    • DIP on iOS
    • DIP on Mobile Web

    For WVGA, WQVA you will find some info on Using density-specific resources on Android

    Hope this helps,

    Matthias
