How to install nightly build SDK

I downloaded mobilesdk 1.8 to see if i can get an orientation bug to fix. Where do I install this zip file on OSX?

mobilesdk-1.8.0.v20110926132318-osx.zip

    On Mac OSX you extract the contents of the archive into the ~&#x2F;Library&#x2F;Application Support&#x2F;Titanium&#x2F;mobilesdk&#x2F;osx directory

  • The easiest way is:

    1. Copy the download URL for the CI build you desire
    2. Open studio
    3. Click the help menu from the toolbar
    4. Click "Install Titanium SDK from URL"
    5. Paste in this link
    6. Click finish
