Hiding navbar is not possible

Hey Everyone,

I am trying to hide the navigation bar in an android application, i have tried several approaches with no success.

I have tried the following:

    var self = Ti.UI.createWindow({
        title:'Products',
        exitOnClose:true,
        navBarHidden:true,
        backgroundColor:'#ffffff'
    });

and i also have tried the following in tiapp.xml

<fullscreen>false</fullscreen>
<navbar-hidden>true</navbar-hidden>

both of these methods didn't hide the top navigation bar in an android application.

    Hello Fouad,

    Since Release 3.3.0, due to changes to support the appcompat library, this property has no effect. By default, the action bar is always displayed. To hide the action bar, see the Android Action Bar guide.

    for hiding action bar try this code :

    // Wait for the Window (or TabGroup) to render
win.addEventListener('open', function(){
    // Grab the window's action bar instance and call the hide method
    win.activity.actionBar.hide();
});
