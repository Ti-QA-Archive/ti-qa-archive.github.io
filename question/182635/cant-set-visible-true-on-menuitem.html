</Menu>
<button id="button"></button>

in javascript i add eventListener to show the menuItem

$.button.addEventListener('click', function(e){
    $.MenuSave.setVisible(true);
}

the error message is &quot;Uncaught TypeError: Cannot set property &#39;setVisible&#39; of undefined

    var item = e.menu.findItem('99')
