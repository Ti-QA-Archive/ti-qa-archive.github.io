  • You can try to invoke the site with a variable

    Ti.Platform.openURL('http://website.com/index.php?unlock=true');

    and on the server side(pseudo code)

    if ( !$_GET['unlock']) {
    // show overlay
}
else {
    // no overlay
}

    You can also use httpclient to do this without opening the site in mobile safari.
