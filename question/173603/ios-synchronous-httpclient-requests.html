  • Making a synchronous call on IOS is very easy. Go to your HtppClient.open function. The parameters are as follows: open( method, url, [async] ). Set async to false and you're set.

    More info at Methods > Open here: http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.Network.HTTPClient

  • you can simulate a synchronous call by using promises see this post https://gist.github.com/aaronksaunders/f4ecc8c634b035526429

    var Q = require("q");

    function http(method, url, params) {
      var deferred = Q.defer();

      var xhr = Ti.Network.createHTTPClient();
      xhr.setTimeout(15e3);
      xhr.onerror = function(e) {
          deferred.reject(this, e);
      };
      xhr.onload = function() {
          deferred.resolve(this.responseText, this.status);
      };

      xhr.open(method, url);

      xhr.send(params);
    }

    usage

    // simple useage
http("GET",url,params).then(_response, _status) {
  // do sometheing with the _response
}, function(_error) {
  // you got an error
})
