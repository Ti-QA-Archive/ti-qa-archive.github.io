  • When you call createFilDetailWindow with the following code you pass it 5 variables, starting with an image ID.

    var winFildetail = app.ui.createFilDetailWindow(idImage, idTitre, idF, idDate, idDesc);

    but in your definition of createFilDetailWindow, your function signature receives only one variable, index. 

    app.ui.createFilDetailWindow = function(index){

    Now it is totally valid to pass more or less arguments to a Javascript function than are indicated in the signature, but I suspect that you are getting unexpected behavior because you are not passing the appropriate value into createFilDetailWindow with the parameter index. That is why the first time you try to access app.model.FilDatas[] via index you get an undefined.

    Make sure you are passing the appropriate index into createFilDetailWindow.
