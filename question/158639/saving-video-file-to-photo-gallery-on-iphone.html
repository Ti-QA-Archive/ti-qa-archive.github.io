Saving Video file to Photo Gallery on iPhone

I would like to save a pre-existing local video to the PhotoGallery for the iPhone platform. I have used SaveToPhotoGallery for photos successfully but when I try to use this API with a video, using either the path (where path+file) or blob (created from file.read()), nothing happens. Has anyone successfully saved a pre-existing video file to the PhotoGallery with Appcelerator?
Thanks for any help you can provide!

    This works fine with me:

    var savePhoto = function(typ,root,file){ //typ='vid' or 'pic'
    var f = Titanium.Filesystem.getFile(root,file);

        Titanium.Media.saveToPhotoGallery(f,{
        success: function(e) {
            Titanium.UI.createAlertDialog({
                title:'Success',
                message:'Photo saved to your photo gallery'
            }).show();      
        },
        error: function(e) {
            Titanium.UI.createAlertDialog({
                title:'Error saving',
                message:e.error
            }).show();
        }
    });
};

    savePhoto('vid',Ti.Filesystem.applicationDataDirectory + '../Library/Caches/','creat.mp4');
