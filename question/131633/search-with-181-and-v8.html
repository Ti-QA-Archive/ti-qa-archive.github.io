  • I made a some tests and that's the result:

    1.7.5 + Rhino –> work

    1.8.0.1 + Rhino –> not work

    1.8.0.1 + V8 –> not work

    1.8.1 + Rhino –> not work

    1.8.1 + V8 –> not work

    I don't know if it can be useful, but a thing I had to change during the conversion from 1.7.5 to 1.8.1 is a line in the event listener of the click on the table.

    I used (with 1.7.5) the command e.row.children[0].text; and instead now (with 1.8.1) I use e.source.text

    This text is exactly what I want in my search field.
