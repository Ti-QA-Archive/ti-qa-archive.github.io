Synchronous calls createHTTPClient

Hi,

I want to call the URL to way synchronous but SDK Titanium don't run:

var url = "http://www.w3c.com";

var client = Ti.Network.createHTTPClient();
// Prepare the connection.


client.onload = function(){
        alert('1');
};

client.open("GET", url, false );

// Send the request.
client.send();

alert("2");

The request is first the text "2" and after "1" . Why?

  • Network access by necessity takes time – to open the network connection, contact the remote host, download the information, etc. If HTTPClient were synchronous/blocking, your entire app would halt waiting for the network operation to complete. This is rarely desirable.

    If you need alert('2'); to show up before alert('1'), put it inside your onload() callback.
