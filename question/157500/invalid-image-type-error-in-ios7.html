I have a titanium code which run perfectly on ios6.
I use the same code and change simulation to ios7, then I get this error.

* Terminating app due to uncaught exception 'org.pttbluecard.TiUIImageView', reason: 'invalid image type'

How can i solve this?

  • I finally found a solution.
    Before I always create my ImageView like this

    var image = Ti.UI.createImageView({
    width: 270,
    height: 100,
    image: ' '
});
image.setImage("someurl");

    This runs fine on ios6.
    To solve the problem, just remove image property and ti will run fine on ios7
