Facebook post without "via Appcelerator Titanium"

If a user posts something on the Facebook wall, it always adds a little icon by Appcelerator with the text "via Appcelerator Titanium". Do we have an option to change it to another app name like "via myApp" and to replace the little icon?

    You need to set your own App ID instead of using the default one.
    When you register/create your own unique Facebook app, you will get an app id (used to get an api key as well) you can declare in your code.

    Titanium.Facebook.appid = '[YOUR APPID]';

    The API key used to be used when you created your Facebook connect button, but that might have been dropped. It looked like this:

    var fbButton = Titanium.Facebook.createLoginButton({
    'style':'wide',
    'apikey':'9494e611f2a93b8d7bfcdfa8cefdaf9f',  // change this !!
    'sessionProxy':'http://api.appcelerator.net/p/fbconnect/', // change this !!
    bottom:10
});
