Allow scrolling of scrollable view when it is bouncing

Hi,

I have a scrollable view (Titanium.UI.createScrollableView) in my application which can scroll both vertically and horizontally. When an user scrolls up or down of a view, the same view bounces while when the user scrolls left or right, we go to the previous or next view added to the scrollable view.

What we want is: While the view is bouncing because of up or down scroll, if an user tries to scroll left or right, the bounce effect should immediately stop allowing the user to scroll horizontally.

What is happening: Right now, the user is not able to scroll horizontally till the vertical scrolling has completed. We have listened to the "scroll" event of the scrollable view and we found that for both vertical and horizontal scroll the same event is getting fired. There is no way to stop the scroll in vertical direction and allow the scroll in horizontal direction.

Is there any way we can achieve this? We dont want to disable vertical scrolling totally.

Thanks in advance.
