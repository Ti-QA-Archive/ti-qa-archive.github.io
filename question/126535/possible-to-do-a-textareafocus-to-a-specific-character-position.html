Possible to do a textArea.focus to a specific character position?

I have a textarea and with the click of a button I can successfully have the textarea focus (and have the keyboard popup), but it focuses to the the last character of the current textArea.value.

Is it possible to have it focus to a specific character position of my choosing?

    There's not currently a way exposed in the Titanium API, and it can even be a little dodgy attempting to do this with native iOS code. You can put this in as a feature request with the Titanium Community project, or you can attempt to create a module that extends the TextArea functionality yourself.   • I found a trick to do this. Just remove and add any character at the position that you would like and it will focus on that position.

    textArea.value = e.value.substring(0, e.value.length-2);//Second from Last;

    This code would just remove the second from last char and focuses on that position. Just save the char and add it back immediately the focus should stay there.
