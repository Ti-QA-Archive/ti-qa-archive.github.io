  • I'm glad to hear you sorted it out. Would you mind to share solution so other developers find it useful?

    Regarding how to display XML response, it's actually up to you. You can getElementsByTagName to retrieve nodes and load them into a specific UI item. E.g. 

    var items = doc.getElementsByTagName("ShipmentHeaders");
var d = items.item(0).getElementsByTagName("orderNumber");
for (var i = 0, len = d.length; i < len; i++) {
    Ti.API.info('>>>> ' + d.item(i).text);
    //or Render into a TableViewRow 
}

    If ignored, this post might be of much help:
    http://docs.appcelerator.com/titanium/latest/#!/guide/Working_with_XML_Data.
