    • Lets create a small checklist of things that I have done to get the modules to work, this way you can see what I am doing and hopefully it will help you.

      1. Install Titanium and dependencies (Android SDK) in their default paths

      2. Go to File->new Titanium Module Project-> we try to create an android module but it gives us an error saying we need Java Development Tools installed first.
