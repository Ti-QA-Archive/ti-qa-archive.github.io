ti.map annotation.setPincolor() on android is not working?

I've got a simple map and on runtime I want to change the pincolor of an annotation from red to green. This works great on iOS, but not on Android.

I use this to add the marker to the map:

    geodata[i].annotation = Alloy.Globals.Map.createAnnotation({
        title: geodata[i].name,
        subtitle: 'closed',
        latitude: geodata[i].latitude,
        longitude: geodata[i].longitude,
        animate:true,
        pincolor: Alloy.Globals.Map.ANNOTATION_RED,
        myid:'marker_' + i,
    });
    $.mapview.addAnnotation(geodata[i].annotation);

Than when I change the marker every thing works except the pincolor? The subtitle of the balloon is changed correctly.

geodata[i].annotation.setSubtitle('Open');
geodata[i].annotation.setPincolor(Alloy.Globals.Map.ANNOTATION_GREEN);

Again, it works fine on iOS. Is this a bug in Titanium 3.3.0.GA?
Thanks for any help.

    It seems, this is not well documented but I found this comment in the code:

    "customView, image and pincolor must be defined before adding to mapview. Once added, their values are final."
