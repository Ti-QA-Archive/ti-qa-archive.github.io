'New' iPhone/iOS SDK

Hmmm… was waiting today for all the new software to hit, I see now that iOS 4.0 is available for public download…. went into my Apple Dev account and still see the "GM Seed" from June 7th is the latest download

Interestingly, Engadget notes that the build number for what was released today is the same as the seed

"Interestingly, the build is 8A293 – the exact same that was released to developers before as a GM seed – so Apple apparently found no showstoppers pressing enough to necessitate a new build before pushing it to the public at large"

Just thought I would share that in case there are others like me waiting to upgrade the SDK
