Run iPhone app on iPad

Is it possible to run an appcelerator iPhone app in compatibility mode for the iPad? If yes, how?

— asked October 1st 2012 by Nando Cammarota

  • Accepted Answer

    In your TiApp.xml, uncheck 'iPad'. This should give you what you are looking for.

    Clean rebuild required.

    • Yes, as long as you do as Stephen suggested: In your TiApp.xml, uncheck 'iPad'. Doing this tells XCode that the app is designed for iPhone only, but it will run on an iPad in an iPhone-sized window (which the user can make full-screen by pressing the 2x button).
