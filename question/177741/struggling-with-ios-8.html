  • This has been working for me personally as well as our internal testing folks. So I suspect something with your system or steps.

    Did you make sure to update the tiapp.xml to point to the 3.4.0-rc SDK version? Do you have ti sdk select set to the RC version of the SDK? Run ti info and see if there are any warnings or errors that might help you determine what's wrong. If you're running from Studio, make sure you're also using the RC version for it. Or try from the command line:

    ti build -p ios

    • It looks like perhaps you're not using the right CLI or SDK version, as those errors refer to 3.3.0. To build for iOS 8, you will need to use 3.4.0 of both of those.

      Run 'ti sdk select' and make sure to select the 3.4.0 RC SDK. Make sure your tiapp.xml lists that same SDK version.
