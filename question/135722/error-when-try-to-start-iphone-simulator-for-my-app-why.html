  • Titanium Studio 2.0 with lastest SDK for Mobile iOS devices on OS X Lion with lastest Xcode version.

    The term 'latest version' is relative to the time of the posting, the time you updated, etc. Software has actual version numbers for a reason. It is extremely helpful for that information to be added to a question so apples can be compared to apples.

    [INFO] Detected third-party module: ti.admob&#x2F;1.1 [INFO] Detected third-party module: ti.admob&#x2F;1.1 [INFO] Titanium SDK version: 1.8.2 (02&#x2F;23&#x2F;12 17:46 59b3a90)

    TiMobile SDK 1.8.2 (which as of today is NOT the latest) only supports xCode 4.2. If you have the 'latest' xCode as of today, that would be 4.3. You would need to compile against the 2.0.1.

    This may not be an answer to your problem specifically, but is a good starting point.
