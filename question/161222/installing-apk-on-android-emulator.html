Installing apk on Android emulator

Hi,

I'm developping an Android application using Alloy framework.
After the latest updates - Titanium SDK 3.2.0GA and Alloy 1.3.0, I can't run the application on Android emulator.

Installing apk takes a lot of time and finally it throws an error:
Failed to install apk on "emulator-5554"

Do you have any ideas how to work it out?

Thank you in advance,

  • The Android emulator is really, really, REALLY slow. There is no timeout for installing the app in the Android build, however ADB does and I don't know how long it is. You may be able to workaround this by launching the emulator before building your app.

    You will be better off using Genymotion or HAXM. To use Genymotion, you need Titanium SDK 3.2.0 or newer, though Genymotion support hasn't been battle tested yet. HAXM is safe, but you need an Intel CPU.
