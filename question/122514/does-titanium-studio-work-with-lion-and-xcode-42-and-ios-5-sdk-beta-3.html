  • lion gm + latest xcode 4.2 + iOS5b3 + latest titanium studio + titanium mobile continous build 1.8.x working here. i get some crashes in the simulator, don't know whether it's the combination used or simply a buggy nightly build of ti mobile 1.8.x, but overall it works.

    however you can't put apps for approval by using xcode 4.2, you still have to use snow leopard, xcode 4.1, itunes 10.3 (the one before those betas released with iOS5 betas), iOS SDK 4.3.3. i created a virtual box 4.1b2 machine in lion using snow leopard to be able to get apps on the phone and finally to the store. for those interested - once you got it working it's surprisingly fast (in fact it's booting faster than my app-overloaded mac in native mode), as long as it has attached enough memory to it
