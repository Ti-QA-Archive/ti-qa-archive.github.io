Firing events from webview in applicationData directory?

I've been loading a web app from the resource directory to a webview. Now that I'm moving it to the applicationData directory all of the listeners and firing of events don't work anymore. I thought that you couldn't use Titanium events with remote webviews.. I guess that for Titanium a simple applicationData can result a remote url?!

Any workaround?

  • My workaround was to use the simple command:

    webview.evalJS("myMethod()");

    Apparently the applicationData is treated like a remote source.
