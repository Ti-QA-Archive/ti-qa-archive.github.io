  • We encountered the same error, I think it happens if a file name or path contains unicode or non-English alphabet letters.

    In this folder c:\<your user name>\ (this depends where you installed windows, or what windows version you are using) there is a folder named .deltafy. We deleted that and it worked. After too much frustration and a reinstallation…
