  • Accepted Answer

    Take a look in the manifest file within the project folder.
    The default app icon is listed in there (ex: #image:appicon.png)

    The file does not have to be in Resources/. It might be in Resources/iphone (for ios builds) or in Resources/android. If the project was not originally built with android targeted, the default platform specific resources may not have been copied over/generated.
