Constraint failed on Insert

I am getting following error:

2011-04-02 16:00:40.803 APpName[4941:5c03] [ERROR] Error occurred  <br> calling next on a PLSqliteResultSet. SQLite error: 'constraint failed' for 'INSERT INTO q1(sb_title,sb_date)VALUES(?,?)'

I have one Auto Increment Field, rest are Integer,Text and Datetime field. How do I fix it out?

The Table Structure is given below:

CREATE TABLE "q1" (
     "sb_id" integer NOT NULL PRIMARY KEY AUTOINCREMENT,
     "sb_title" text(100,0) NOT NULL,
     "sb_details" text(300,0) NOT NULL,
     "sb_image" text(30,0) NOT NULL,
     "sb_type" integer(4,0) NOT NULL DEFAULT '1',
     "sb_date" datetime NOT NULL
)

    As I can see it, you are trying to insert only 2 columns. But your definition of the table states that even sb_details and sb_image must be inserted as they are NOT NULL. You can also allow NULL for the columns if you don't want to add a value right away.
