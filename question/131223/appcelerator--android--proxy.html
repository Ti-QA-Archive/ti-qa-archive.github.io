Appcelerator + Android + Proxy

Hello,

I'm trying to develop an android application using Titanium Appcelerator. I'm trying to use Ti.Network.createHTTPClient to consume a JSON service that it's avalaible in Internet. But I'm behind a proxy, and I don't know how to set the proxy settings in the Android Emulator.

I try with the Tekila settings, but it doesn't work.

Could anybody help me?

Thanks in advance.

    • Ok, I've found it.

      emulator_cmd = [
            self.sdk.get_emulator(),
            '-avd',
            avd_name,
            '-port',
            '5560',
            '-sdcard',
            self.get_sdcard_path(),
            '-logcat',
            '*:d,*',
            '-no-boot-anim',
            '-partition-size',
            '128', # in between nexusone and droid,
            '-http-proxy http://10.12.54.8:8080']

    • Click on Menu
      • Click on Settings
      • Click on Wireless & Networks
      • Go to Mobile Networks
      • Go to Access Point Names
      • Here you will Telkila Internet, click on it.
      • In the Edit access point section, input the "proxy" and "port"
