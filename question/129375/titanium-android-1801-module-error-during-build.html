Titanium Android 1.8.0.1 module error during build

Hi,

I am using a module in my app. It builds fine in 1.7.x but in 1.8.0.1 it exits build with error 

The 'apiversion' for 'com.mwaysolutions.barcode' in the module manifest is not a valid value.  Please use a version of the module that has an 'apiversion' value of 2 or greater set in it's manifest file

I changed the apiversion in the module manifest from 0.2 to 2 but still shows the same error.

Thanks

  • Try the following:

    1. Change the com.mwaysolutions.barcode manifest to apiversion: 2.0 (not just 2, make sure you add the .0). I just added it after the copyright.
      ~~~

    #

    this is your module manifest and used by Titanium

    during compilation, packaging, distribution, etc.

    #
    version: 0.2
    description: Barcode module
    author: M-WAY Solutions GmbH
    license: Apache v2 license
    copyright: Copyright (c) 2011 M-WAY Solutions GmbH
    apiversion: 2.0

    these should not be edited

    name: titanium-barcode
    moduleid: com.mwaysolutions.barcode
    guid: bc88d841-a015-45a8-8d95-31a4fa924c35
    platform: android
    minsdk: 1.6.0

    
2.  In your TiApp.xml change the module version to 2.0 as well.

    <module platform="android" version="2.0">com.mwaysolutions.barcode</module>
    ~~~
    I'm not sure which change makes it compile, I haven't really messed around with it. But try that it may work for you.

  • I forked the project and built it for 1.8.1.

    The binaries: https://github.com/downloads/RaduCirstoiu/titanium-barcode/com.mwaysolutions.barcode-android-0.3.zip

    In your tiapp.xml make sure to change the version of the module like so:

    <module version="0.3">com.mwaysolutions.barcode</module>

      Your module manifest file should look like this:

      #
# this is your module manifest and used by Titanium
# during compilation, packaging, distribution, etc.
#
version: 0.2
description: Barcode module
author: M-WAY Solutions GmbH
license: Apache v2 license
copyright: Copyright (c) 2011 M-WAY Solutions GmbH

# these should not be edited
name: titanium-barcode
moduleid: com.mwaysolutions.barcode
guid: bc88d841-a015-45a8-8d95-31a4fa924c35
platform: android
minsdk: 1.6.0
apiversion: 2
