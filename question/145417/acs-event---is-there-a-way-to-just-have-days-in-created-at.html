ACS EVENT - IS there a way to just have days in CREATED AT?

I want to use CREATED AT in a tableview but I only need the day it was created, not the time.
Is there a way to do this?

Snippet:

var l_StartTime = Ti.UI.createLabel({
text: "Created at: "+ event.created_at,
left:"120dp",
bottom:"10dp",
font:{
fontSize:14,
fontFamily: "Helvetica neue"
}

            });

    Use moment.js to parse the created_at and then format it:

    moment(event.created_at).format('ddd');

    • Of course you need to require the moment.js module.

      Try something like this:

      // I have moment.js in Resources/moment.js; you may choose 
// somewhere else to put it; adjust your require() call accordingly
var moment = require ('/moment');

var m = moment(event.created_at);
var strCreated = m.format('ddd');

var l_StartTime = Ti.UI.createLabel({ text: strCreated, left:"120dp", bottom:"10dp", font:{ fontSize:14, fontFamily: "Helvetica neue" }
            });

      That way, if you're having trouble with the moment code, you can set a breakpoint and examine your variables.
