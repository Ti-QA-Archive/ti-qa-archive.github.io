Your TARGET_BUILD_DIR is incorrectly set

Ive encounter this problem and i want to share simple solution to those who ever encounter the same problem…

All you have to do is check the path in the error message..
ie :
dir /Users/activate/Documents/Titanium Studio Workspace/wdasg/build/iphone/build/Debug-iphoneos,
was: /Users/activate/Documents/titanium studio workspace/wdasg/build/iphone/build/Debug-iphoneos

Notice case letter difference.. ?

So just rename the folder will solve it..

Another option is edit your builder.py in /Library/Application Support/Titanium/mobilesdk/osx/1.8.1/iphone line 1209..

CHANGE : 
if target_build_dir!=build_dir:

TO:
if target_build_dir.lower() != build_dir.lower():

Hope it help..
