If anyone has some working code that works to pull an image into the right/leftButton or right/leftView of an annotation, I'd be very grateful.

What I'm trying to do is put a text variable (a location name) onto an image and then use that resultant image as the sole contents of the annotation. Users would click on that to navigate to a location specific page.

The reason I don't want to use the title or subtitle (along side a clickable image) is that on Android, the click event doesn't work when you click on the background of the annotation. The user would quite reasonably expect to be able to click anywhere on the annotation.
