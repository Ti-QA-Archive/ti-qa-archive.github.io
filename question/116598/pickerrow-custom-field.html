pickerRow custom field

Hi everybody,

Can i put some custom field in pickerRow. I want to use the title field to display some information but i want to "hide" other.

An example :
var pickerInscrit = Ti.UI.createPicker({
left:centerLeft,
width:'200',
top:'225'
});

var dataInscrit = [];
dataInscrit[0] = Ti.UI.createPickerRow({title:'Inscrit',custom_item:0});
dataInscrit[1] = Ti.UI.createPickerRow({title:'Oui',custom_item:1});
dataInscrit[2] = Ti.UI.createPickerRow({title:'Non',custom_item:2});

pickerInscrit.add(dataInscrit);

I put an event on a button and when it is click I test this : alert(pickerInscrit.getSelectedRow(0).custom_item); and it return undefined (and I have a row selected)

I test with the picker example with Kitchensink and it doesn't work to.

Thanks for your help and sorry if I have a bad english I'm french :-)

  • We found a suitable workaround although we would appreciate that this gets fixed.

    The Kroll Proxy has an extend method which can be used to add custom_items out of javascript without the need to patch the SDK:

    here is a working example which can be tested using the KitchenSink Picker Android Spinner Text:

    var rows1 = [];
for (var i = 0; i < names.length; i++) {
    var r = Ti.UI.createPickerRow({title: names[i]});
    r.extend({
        myidx: i,
        mytest: "a test"
    });
    rows1.push(r);
}

    The test using the check Selection Button looks like this:

    btnCheckSelection.addEventListener('click', function() {
    //showStatus(picker.getSelectedRow(0).title + ' ' + picker.getSelectedRow(1).title + ' ' + picker.getSelectedRow(2).title);
    showStatus(picker.getSelectedRow(0).title + ' ' + picker.getSelectedRow(0).myidx + ' ' + picker.getSelectedRow(0).mytest);

    This works on Android and SDK 1.6.0
