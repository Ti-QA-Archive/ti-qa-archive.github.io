Can I add annotations to the mapp after it's view is drawn?

I have a map but while it is added to the view, asynchronously I call the google map API to convert my addresses in map coordinates. After that I add all the annotations to the map view but it doesn't show any annotation at all. If I put annotations right after creating the mapView object, it works because the view isn't drawn on the screen yet. Otherwise It doesn't work.

But I wouldn't like the user to wait for the app to get all the coordinates of all the annotations before they can see something on the screen.

  • You didn't mention the platform, but on android, I have found that setAnnotations() does not work, while it works just fine on iOS. Instead, call removeAllAnnotations() and then addAnnotations().
