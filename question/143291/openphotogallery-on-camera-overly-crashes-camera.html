openPhotoGallery on Camera overly crashes camera

I have a fairly simple Camera overlay that I'm building and then attaching via the Ti.Media.showCamera() method. A button on that overlay is intended to open the PhotoGallery. The click event handler has this code in it:

Ti.Media.openPhotoGallery({
     mediaTypes : [Titanium.Media.MEDIA_TYPE_PHOTO],
     success : function(event) {
        // event.media
                var win = new PhotoEditWindow(event.media);
                win.open();

                Ti.Media.hideCamera();
    },
    cancel : function() {
    },
    error : function(error) {
        Titanium.API.info("Gallery error" + JSON.stringify(error));
    },
    allowEditing : true
});

As soon as that button is clicked the error callback on the camera (not the PhotoGallery error callback) is triggered and returns an error code of 1. At that point the buttons on the overlay stop working and everything is hoarked.

Any ideas?

SDK 2.1.3

iOS 6

Xcode 4.5.1
