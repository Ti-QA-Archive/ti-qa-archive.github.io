Window close problem

I have a problem closing windows (open 'dynamically'):

Ti.API.info(insideWindow); // [object TiUIWindow]
insideWindow.close();
Ti.API.info(insideWindow);  // [object TiUIWindow] ?!

I hope you can help me. Thank you very much :)

    You need to "nullify" the window if you want to release the memory.

    insideWindow.close();
insideWindow =  null;
