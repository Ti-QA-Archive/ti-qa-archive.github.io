    According to the Apple Developer Docs, you must have a system running Apple OS in order to install the Apple SDKs for building iOS apps. Windows does not have the capabilities of running the Apple SDKs. If you are on Windows and want to build iOS apps, you can try a service called MacInCloud which is basically just a Mac machine that you can rent in the cloud. It comes pre-installed with all Titanium SDKs necessary to build iOS apps.
