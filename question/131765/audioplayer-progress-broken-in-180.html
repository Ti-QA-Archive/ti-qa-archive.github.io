AudioPlayer progress broken in 1.8.0?

Hello all, Im developing an app which makes it possible to listen to music from a server.
Titanium SDK 1.8.0, iPhone 5.0

Anyhow, I've created the audioplayer as said in the documentation:

    var Player = Ti.Media.createAudioPlayer({
        allowBackground: true
    });

var url = "http://url-to-mp3.mp3";

Player.setUrl(url);
Player.play();

That works. But in the console it writes an warning every second, probably becase I cant get the progress-event to work.

eg.

    Player.addEventListener("progress", function(event)
    {
        alert(event);
        Ti.API.info('Time Played: ' + Math.round(event.progress) + ' milliseconds');
    });

It doesnt alert the event at all. But it still writes an warning in the console:

[WARN] Exception in event callback. {

It writes this every second the player is playing. But noting else after tthe {..

FYI, the player plays the track in the iPhone emulator but on on the device..
