Check if other apps are installed

Is it possible to check if other apps are installed on the device (iOS), or have been purchases before?

I've seen this in some other apps but can't figure out if its possible with Titanium. My idea is that my app is in a set of apps, and I want to enable certain features in one, if they have purchases the other.

I can do this with a user account system, but it would be so much easier if I could simply check if the other apps are installed.

  • If the other app(s) have a custom url scheme, then you can use the canOpenURL method to check and see if the app is installed: http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.Platform-method-canOpenURL

    If all of the apps belong to you, then you can always use the Keychain in order to create the access needed for multiple applications and/or service. There is a module in the Marketplace that wraps the iOS Keychain
