To give different sizes for android and ios using alloy

Hi can anyone help regarding height or width to give for IPAD, iphone android phones and tablets.
Suppose I have a view in index.xml as

<View class="view1" backgroundColor="#fff">
</View>

index.tss

".view1":{
top : '10%',
height:"30#",
width:"50%"
}

How to give different dimensions for these in android,iphone,ipad and tablets. I knew using classic. Please help me regarding alloy framework.
Thanks,
Prashaanth

    It's all in the documentation

    ".view1[platform=ios]" : {
}

".view1[platform=android]" : {
}

".view1[formFactor=tablet]" : {
}

".view1[formFactor=handheld]" : {
}

".view1[platform=ios formFactor=tablet]" : {
}

    and so on…
