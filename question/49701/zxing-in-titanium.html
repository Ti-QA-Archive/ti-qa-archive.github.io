    Hello Marcus,

    it is of course possible to include 3rd party libraries into Titanium. However, its a little tricky to start from scretch. To prevent interested developers from trapping into common pitfalls and obstacles we will provide a detailed tutorial which shows how to proceed. For now, we introduced a very similar approach (TiBar) in this post which integrates ZBar, another great barcode library, into Titanium. By using our module it is really easy to use ZBar's barcode scanning capabilities in Titanium. So, check this out and comment if you like. Thanks.
