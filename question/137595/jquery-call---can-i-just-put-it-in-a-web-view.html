    It can bee (may be) a hard work. I would create a unvisible webview and run the logic in it. If your html comes local you can comunnicate with signalR in the same namespace. In other case you can use event firering.

    var signalrContainer = Ti.Ui.createWebView({url:'/html/signalr.html'});
signalrContainer.evalJS(MYJSCODE); // speak to web
Ti.App.addEventListener('webdata',callbackfunction); // read from web

    and in signalr.html:

    try {
Ti.App.fireEvent('webdata',{data:serializeData()}) // send bck to Titanium
}
catch (E) {
// 
}

    Hope this helps.
