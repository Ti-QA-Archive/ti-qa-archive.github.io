    • The animation suggestion does work, I have not tried it in alloy but here is the code I would use.

      var search = Ti.UI.createSearchBar({
    right: 10,
    width: 150
});
search.addEventListener('focus', function (e) {
    Ti.API.info('focus');
    search.animate({
        width: 300,
        duration: 500
    });
});
search.addEventListener('blur', function (e) {
    Ti.API.info('blur');
    search.animate({
        width: 150,
        duration: 500
    });
});

      If you are not succeeding by adding it to the mark-up I would suggest adding the searchBar to the titleBar in code and then attaching the events as I have done.
