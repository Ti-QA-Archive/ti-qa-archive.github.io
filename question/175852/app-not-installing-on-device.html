App not installing on device

I've been having a lot of problems trying to get something to install on an Android device. I had it work a couple times, but almost every time I attempt I get this:

[INFO] : Installing app on device

…but it doesn't show up on the device at all. Other times I'll get this:

[INFO] : Alloy compiler completed successfully

…and nothing else happens. My device is set to enable USB debugging, etc. Is there something else I should be doing?
Thanks

  • If it works sometimes… not really anything else you should be doing. What you can do is take a look at the compiled APK file and copy it manually. It doesn't fix your intermittent connectivity problem, but at least you can copy the file and test on your device.

    If you go to your Application Directory\build\android\bin directory. There will be an apk file there with your project name and you can copy that to your device and test.
