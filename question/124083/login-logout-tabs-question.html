    This example isn't 100% ready for production as you will have to do some work on tab orders and do some checks for if logged out/in already, but it will get you started:

    Make use of some custom events and the addTab and removeTab methods to do the refresh:

    In login.js, change the button events to something like:

    loginBtn.addEventListener('click', function(e) {
    Ti.App.fireEvent('loggedin');

});

logoutBtn.addEventListener('click', function(e) {
    Ti.App.fireEvent('loggedout');

});

    and then in app.js add listeners to handle the tab changes. I also moved the property updates there for clarity:

    Ti.App.addEventListener('loggedout',function(){
    Titanium.App.Properties.setString('LoggedIn', 'loggedout');
    tabGroup1.removeTab(accountTab);
    tabGroup1.addTab(homeTab);
});


Ti.App.addEventListener('loggedin',function(){
    Titanium.App.Properties.setString('LoggedIn', loggedin);
    tabGroup1.removeTab(homeTab);
    tabGroup1.addTab(accountTab);
});
