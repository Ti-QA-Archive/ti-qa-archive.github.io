  • Hi

    I can only assume your question is "how do I stop the shutter noise from sounding when I take a photo using titanium?"

    Short answer - no - you cannot with Titanium.

    Longer answer;

    It "may" be possible to create a native module using Java to turn off the sound but I think this is at the SDK files level - which is not accessible via a module.

    If you could turn off the sound you have two problems;

    • This should be the users choice - which can easily be achieved in the settings. You changing this for a user can be considered an abuse of the device.
    • Many countries have laws requiring any camera device to make an audible sound when taking a picture. This is because of some people using cameras to take pictures of things they shouldn't, especially of young children or in very inappropriate locations including swimming pools and changing rooms.

    If your app is designed to take pictures of birds for example then I understand why you may want it as quiet as possible but again the user can simply change the setting for the phone themselves.
