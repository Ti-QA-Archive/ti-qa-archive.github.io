  • Accepted Answer

    For android, I suggest you install the Java Development Tools and then the Android Development Tools

    This will allow you to open a view that has the DDMS utility in TiStudio. DDMS is a must have for Android development. This will show you all the log activity on the emulator and/or a device.

    You will be able to filter out (using regex, or log levels) the information you do not need.
