  • Dont add it in the first place. Create an array to hold your views and add to them as needed eg

    var views = [];

if (dbValue1) {
    views.push(Ti.UI.createImageView());
}


etc etc etc

then

var scrollView = Titanium.UI.createScrollableView({ 
    views:views,
    top:0,
    width:320,
    height:150,
    showPagingControl:true,
    pagingControlHeight:20,
    pagingControlColor: 'transparent' 
});
