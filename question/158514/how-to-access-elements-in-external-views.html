    • You need to hold a reference to the home controller you created, then simple toggle the values by id with it. 

      // whenever you create the home controller...
Alloy.Globals.home = Alloy.createController('home');

// whenever you want to access it and change its elements
Alloy.Globals.home.getView('profileView').visible = false;
