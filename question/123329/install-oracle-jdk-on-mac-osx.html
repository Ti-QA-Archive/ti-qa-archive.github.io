  • Hey Mike,

    It's true you can find the version of the java engine doing that, but Java Runtime Environment (Java SE) is NOT the SDK or as it is called in the java world JDK. The runtime is the software needed to run java applications, the JDK is needed to make them. Java runtime is installed in every Mac and you can update it by going to Software Update. (I was just helping Joe though, I'm not having problems)
