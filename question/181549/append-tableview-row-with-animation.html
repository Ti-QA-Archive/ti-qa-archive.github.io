Append tableview row with animation

Hi, I have a table view that I made to show comments. When user type new comment and send, it will append a new row to the tableview. I managed to do this but I need animation to go with it.

What I wanted to do is push previous row up while the new row appears below it. So user don't have to scroll up to see what they type in. The tableview have a set height. If the new comment overshoots the table height, I want to scroll it up so the new row is visible without user having to scroll it.

Is this possible to do?

I'm working in iOS and using 3.5.0 SDK.
