Saving Photo Without Preview or Changing Overlay

Hello,

iPhone question here. I'm using Ti.Media.showCamera to take a photo with the following dictionary (with unnecessary code removed):

Titanium.Media.showCamera({
    success: function(event){...},
    cancel: function(event) {...},
    error: function(event) {...},
    saveToPhotoGallery: false,
    allowEditing: false,
    overlay: overlay,
    transform: overylayBugFix,
    autohide: true,
    showControls: true
)};

After pressing the button to take a photo, it brings me to the screen where you can either retake the photo, or "use" it. I'd like to EITHER:

a) remove the overlay

b) skip this step and go ahead with the code in the success:function.

Is there a way to do this? I've looked through the docs and searched like crazy but can't figure this one out.

Thanks everyone!

    I have successfully achieved this by adding a custom overlay with a button containing a click event listener. On the click event, call Ti.Media.takePicture(); which will lead directly to the success function.
