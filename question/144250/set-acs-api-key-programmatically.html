Set acs-api-key programmatically

If my ACS backend is subject to change (new acs-api-key), is it possible to set it programmatically?

So far I set it in tiapp.xml, but cannot change it afterwards (like reading it from external resource at app start):

<property name="acs-api-key" type="string">get_from_app_admin_page</property>
<property name="acs-oauth-key" type="string">get_from_app_admin_page</property>
<property name="acs-oauth-secret" type="string">get_from_app_admin_page</property>

    Note that the ACS keys are Ti.app.Properties loaded from the tiapp.xml.
    You can set them using setString().

    • yes, just found it. Thanks

      Ti.App.Properties.setString('acs-oauth-secret', '123');
Ti.App.Properties.setString('acs-oauth-key', '123');
Ti.App.Properties.setString('acs-api-key', '123');

  • The correct answer to updating the ACS api-key programatically:

    Ti.App.Properties.setString(&#39;acs-api-key-development&#39;, &#39;xxxxxxxxxxx&#39;);
