    • Ok so what you need to do is add a transform function to the table as follows:

      ...
<TableView id="table"  dataCollection="memberinfo" dataTransform="transformFunction">
...

      The label stays as:

      <Label id="name" text="{name}"></Label>

      Then in the JS add the following function

      function transformFunction(model) {
    var m = model.toJSON();
    m.name = 'Name: ' + m.name;
    return m;
};

    • For completeness, if this transformation is to be used in several places, then I suggest you override the model as follows:

              _.extend(Model.prototype, {
            // extended functions and properties go here
            transform: function() {
                var m = this.toJSON();
                m.name = 'Name: ' + m.name;
                return m;
            }
        });

      and change the transform function to:

      function transformFunction(model) {
    return model.transform();
};

      That way you keep all the transformations in the one place and only have to change them once, should they need it

    • The reason this wont work is because you have changed the code from what you originally posted! Plus you are returning o not m from the function.

      However, the SQL statement you have used will return a single name property to the transform function the value of which is set from the spouse.name.

      I suggest you change the SQL to:

      memberinfo.fetch(
    {query:"SELECT members.name AS memberName, spouse.name AS spouseName, members.dom, members.dob,members.business FROM members LEFT JOIN spouse ON members.id = spouse.mid WHERE id="+args.memberid
    });

      Then change the transform function to use these new properties

      function dumpText(model) {
     var m = model.toJSON();
    m.memberName = " Name: "       + m.memberName;
    m.spouseName = " Name: "       + m.spouseName;
    Ti.API.info(m.memberName);
    return m; //m not o
}

      and change the XML to:

      <Label id="name" text="{memberName}"></Label>
...
<Label id="spname" text="{spouseName}"></Label>
