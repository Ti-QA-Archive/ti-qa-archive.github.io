  • I use a process that is database driven. The app consumes a service checking for current version, checks the running version and if different displays a dialog indicating a new version is available. If the user chooses "Install"….

    if (OS_IOS) {
    Ti.Platform.openURL("itms-services://?action=download-manifest&url=" + newURL);
} else if (OS_ANDROID) {
    Ti.Platform.openURL(newURL);
}

    The "newURL" variable is the address to the .apk for android and the .plist for iOS.
