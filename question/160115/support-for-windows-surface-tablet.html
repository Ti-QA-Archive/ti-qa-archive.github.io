  • Hi

    Short answer - no.

    Longer answer; it will do.

    Currently there is no support for the Windows Phone platform or Windows metro style (I know they no longer use that name). There has been talk for the past year about support coming throughout this year.

    However the good news is that as part of the future Titanium development (hyperloop and Ti.next are related phrases) there is going to be a fundamental shift in how the underlying Titanium system works. One of the huge benefits of this will be the ability to support many platforms with a much smaller code footprint.

    So by the middle of next year there should be at the very least beta support for the Windows platform - hopefully production ready.

    Obviously this is all subject to change - but as it stands right now - mid 2014 will give you many more choices than right now.

    Keep in mind you currently have support for;

    • Android
    • Blackberry
    • iOS
    • Mobile Web
    • Tizen

    Which is not bad in any ones book.
