Opening Native Map Application, iPhone

Hey everyone.
I am working on an App that enables the user to get directions using the Native iPhone App, It is currently working but i am facing one problem. The native iPhone mapping application loads with both the currently and destination places plotted but the destination place annotation title has road names instead of the place title name.

How can i enable the annotation to have the place name as it's title instead of the road name as it's title.
Here is my event code below

directionButton.addEventListener('click', function(e){
        Ti.Platform.openURL('http://maps.google.co.jp/maps?saddr='+currentLat+','+currentLong+'&daddr='+placelat+','+placelong);
});

Any help will be highly appreciated, thank you.

  • You're calling out to a map in a webview as shown in your code. So, investigate Google's static maps API, e.g. you can set markers, see https://developers.google.com/maps/documentation/staticmaps/#Markers

    According to this page you can also use maps:// as the URL protocol and that will open the native maps app. I haven't tried that and I'm not sure what you'd use for the remainder of the URL, especially since Apple is moving away from Google for mapping. I'd check Apple's dev docs for more info on that.
