NavigationWindow in Alloy

Hi

I'm using Ti 3.1.3 RC to prepare our Alloy app for iOS 7 changes and I've changed all NavigationGroups to NavigationWindows. This worked okay in the XML but when I try to open another window inside the NavigationWindow I get a "undefined is not a function" error. Here's a sample:

XML:

<Alloy>
    <NavigationWindow id="navGroup">
        <Window class="container">

            <Button onClick="nextPage" top="40">Next page</Button>

        </Window>
    </NavigationWindow>    
</Alloy>

JS:

function nextPage(e) {
    var page2 = Alloy.createController('page2').getView();
    $.navGroup.openWindow(page2);
}

$.navGroup.open();

Now the code looks okay to me so I'm assuming that the NavigationWindow object just hasn't been ported to Alloy yet but could anybody confirm this please?

Thanks

James

Ti 3.1.3RC, iOS 7 beta, Mac OSX Mountain Lion

  • Hello James,

    I tried the non-Alloy sample code in the docs with:

    • SDK : 3.1.3.v20130916153052
    • Alloy : 1.2.2-cr
    • Xcode 5 GM seed
    • iOS 7 GM seed

    I did not encounter a problem. Please let us know if you continue to have difficulty. Here is the sample code I used:

    var win2 = Titanium.UI.createWindow({
    backgroundColor: 'red',
    title: 'Red Window'
});

var win1 = Titanium.UI.iOS.createNavigationWindow({
   window: win2
});

var win3 = Titanium.UI.createWindow({
    backgroundColor: 'blue',
    title: 'Blue Window'
});

var button = Titanium.UI.createButton({
    title: 'Open Blue Window'
});
button.addEventListener('click', function(){
    win1.openWindow(win3, {animated:true});
});

win2.add(button);
var button2 = Titanium.UI.createButton({
    title: 'Close Blue Window'
});
button2.addEventListener('click', function(){
    win1.closeWindow(win3, {animated:false}); //win3.close() will also work!!
});

win3.add(button2);
win1.open();
