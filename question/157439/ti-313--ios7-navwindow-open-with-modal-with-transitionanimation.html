Ti 3.1.3 + iOS7 navWindow open with modal with transition/animation?

Since 3.1.3 and the change to navWindow instead of navGroup, the "slide up"-effect on opening a modal window has disappeared.

Even when I try to open the modal-navWindow with the parameter "animated: true" it doesnt work. I want the modal window to pop up like it did before the move to NavWindow!

Is that possible? 

        notificationsButtonView.addEventListener("click", function()
        {
            notificationsNavWindow.open();
        });

    I just discovered that setting the modal property to "true" for the navigationWindow will use the familiar slideup/down animation for the modal window.

    var modalwin = Ti.UI.createWindow({
});
var self = Titanium.UI.iOS.createNavigationWindow({
    modal: true,
    window: modalwin,
});
