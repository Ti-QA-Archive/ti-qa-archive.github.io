Can not create new project

Hello,

I am new to titanium. After starting the software it turns out, that I can not create a new project, as the button "Create Project is grayed out.

What error might I have made?

Thanks

  • I'm new to Titanium, but I have found that this will occur if you have a "-" (dash) in the App ID field. IOW, com.xxx-yyy.appname WILL NOT work, but com.xxxyyy.appname DOES work.
