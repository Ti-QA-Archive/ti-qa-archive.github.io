Adding a view to a tabgroup in Android

I have an app with 4 tabs. At the bottom of the screen, I need to display a footer on almost every window so I'm creating a view and adding it to the tab group with bottom: 0; height: &#39;45dp&#39;; width: &#39;100%&#39;;. In the iPhone simulator, perfection. In the Android emulator, not so much. In debug mode, I can see that the view is created and has all the right properties, but it simply doesn't appear.

Any ideas?

The footer is created as:

var row = Ti.UI.createTableViewRow({
    hasChild: !Application.isAndroid,
});
var label = Ti.UI.createLabel({
    color: '#fff',
    font: { fontSize: '18dp', },
    left: '10dp',
    text: 'Custom text',
    textAlign: 'left',
})
row.add( label );

data.push( row );
var footer = Titanium.UI.createTableView({
    // borderColor: '#f00',
    id: 'app_footer',
    data: data,
});

And then, of course, added to the tab group in the usual way: myTabGroup.add( theFooterCreatedAbove );

I'd really appreciate any insight. I've been looking for over a day and can't find the problem.

Thanks.
