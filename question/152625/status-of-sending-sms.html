status of sending sms

Hi,

Here is my code to send sms :

if(OS_IOS){
        Titanium.Platform.openURL('sms:' + toShare.sms.content);
    }
    else{
        var intent = Ti.Android.createIntent({
            action: Ti.Android.ACTION_SENDTO,
            data: 'smsto:'
        });

        intent.putExtra('sms_body', toShare.sms.content);
        Ti.Android.currentActivity.startActivity(intent);
    }

is it possible to know when the sms is sent ?

thanks

  • To my knowledge, there are no events or messages returned from those calls. While you could listen for the app's resume event (app would be paused when SMS is opened), there would be no way to tell if the user sent the message or canceled the operation. To my knowledge, those are platform limitations, not Titanium "shortcomings."
