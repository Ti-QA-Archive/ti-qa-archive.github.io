ios Inheriting builtin classes

I'm building a custom Titanium module which extends some of the functionality of the default titanium based view classes. In my case, I want to extend the functionlaity of TiUITableView and TiUITableViewProxy. However, when I try to subclasse these classes, I always get a :

error: cannot find interface declaration for 'TiUITableView', superclass of 'STTableView'; did you mean 'UITableView'?

Even though it seems like xcode can see the header file 

#include "TiUITableView.h"

since when i cmd+click on it, it show the right header.

Does anyone have any ideas how I can subclass the default titanium View and ViewProxy classes?

Thanks

    #define USE_TI_UITABLEVIEW

    and now it works fine.
