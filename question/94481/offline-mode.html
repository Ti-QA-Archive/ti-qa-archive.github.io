Offline mode

Can we please make an offline mode for Titanium Develop. I would like to be able to work on my projects when I don't have an internet connection or at my work where I am behind a company firewall. What is the point of having the software connect to the internet anyways?

I have tried all the hacks that have been posted to work around the login screen but now I am getting an SQL error when I try to create a project. I don't think we should need to use hacks to get a program working just because it isn't connected to the interent.

Nick
