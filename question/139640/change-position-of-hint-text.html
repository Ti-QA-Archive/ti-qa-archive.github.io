  • Hello manu,

    use like this and you can place your hint text as you wish by changing paddingLeft property value

    here is code

    var textfield = Ti.UI.createTextField({
    height : 30,
    width : 150,
    hintText : 'hint Text',
    paddingLeft : 50,
    //backgroundColor : 'yellow'
});
