Titanium.Filesystem undefined in webview

Titanium.API.info("Attempting to log typeof Filesystem");
Titanium.API.info(typeof Titanium.Filesystem);

Filesystem is an object when referenced not in a web view. When I create a web view and reference Titanium.Filesystem or Ti.Filesystem, it is undefined. I noticed a lot of questions on this before. Most said to rebuild/clean the project. That did not seem to help. Some said to start over. That was funny, and also did not help.

Does anyone know why Titanium.Filesystem is undefined in a web view? Titanium is an object in the web view.

Here'e the view creation:

var gameview = Titanium.UI.createWebView({url: '/html/game.html'});
var window = Titanium.UI.createWindow();
window.add(gameview);
window.open();

  • The Titanium.FileSystem namespace isn't accessible from inside a WebView. But if you want to get the application's Resources directory, you simply have use relative paths. Since everything is relative to the Resources directory.

    For example, if you have an image in the Resources&#x2F;sample directory, all you need to do is use the following code:

    <img src="sample/Default.png" />

    Notice that there is no slash in front of the path

    The same goes for your sounds where you could assign them using the following code:

    colors['red'] = 'sounds/red.wav';

    Hope this helps.
