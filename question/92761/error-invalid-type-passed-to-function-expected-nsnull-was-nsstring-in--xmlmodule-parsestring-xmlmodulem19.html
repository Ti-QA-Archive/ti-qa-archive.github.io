  • I bumped into a similar bug and found this page via Google. Not sure if my solution will help you with your problem, and original question was a year ago, but here it is anyway.

    In my case I was trying to pass a function call in directly:

    startupAnimation.addEventListener('complete', doNextThing() );

    When I need to do it like this:

    startupAnimation.addEventListener('complete', function() { doNextThing() } );

    Hope that helps somebody!
