    • Please note that the question asked was concerning MySQL and not MS SQL. We try to keep questions categorized for ease of answering and searching.

      With that said, I'll still answer your off-topic question: MS SQL is NOT supported natively on either Android or iOS.

      On both platforms, middleware is required to interface with that particular DBS. That middleware could be a library/module that extends the language: java/objective-c/javascript or could be actual middleware server software running in the cloud.
