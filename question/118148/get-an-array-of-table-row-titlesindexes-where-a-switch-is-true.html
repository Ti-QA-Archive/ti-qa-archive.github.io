Get an array of table row titles/indexes where a switch is true?

I have a grouped table with a switch in each row, like this:

What I want to do is click the button and get an array of the titles of all table rows (their title or index) where the switch is set to on. I can't figure out how to get either the title or index, from the switch I can only get event.source.parent.

Here's the code for the table: http://pastie.org/1758530

    clickLabel.text = 'row =  '+tableViewRow[0].title;

    try this

    tableView.data[0].rows[0].title

    there is a default section in the table that is holding your rows, and do this to get the table row's switch control

    tableView.data[0].rows[0].children[0].value
