  • Accepted Answer

    Do you need something specific in the newest Android tools? We don't officially support them yet. Google keeps moving components around on us. At the moment, I know that SDK tools R22.3 and Platform tools R19 work with the latest Titanium RC versions. Unless you need something specific, you'll probably need to downgrade versions.

    Speaking of, which RC version of the CLI are you using? We're up to RC4, and the more recent RC builds included fixes specifically for Windows and the Android tooling. On Windows, the following will install the latest RC (on Mac/Linux, add sudo to that).

    npm install -g titanium@3.3.0-rc4

    Finally, is ADB hung or can it see your device? adb devices should show you your connected device. If not, try adb kill-server followed by adb start-server and repeat until devices shows your device connected.
