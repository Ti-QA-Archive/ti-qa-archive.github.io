parse error within nested objects

hi,

i get a parse error in here 

var myObject = {
      name: 'name',
      id: ID,
      probs = {
             color: 'red',
             bla: 'blubb'
      }
}

without the "probs"-part is works fine. i don't want to put it all in myObject directly, so this would be the nicest style.

    It should be:

    var myObject = {
  name: 'name',
  id: ID,
  probs:
    {
      color: 'red',
      bla: 'blubb'
    }
};
