  • Hi Ivana Miserda,

    In order to successfully connect an Android device and a computer running Microsoft Windows, there are a number of things we need to do.

    First, we need an Android device and a computer running Microsoft Windows operating system. Then we need a data cable to connect the Android device to the computer.

    Second, we need to install the device driver software on the windows machine, if not present. This should install automatically from the web. If it does not, you will need to enable the automatic search option in device properties.

    After connecting our android device to the windows machine, we need to do the following:

    First, we need to go to the settings menu on the Android device, then we choose the “Developer Option” after then we can check the” USB debugging” option.

    Next, we check the “Allow installation from unknown sources” option from the Security menu on the Android device.

    Now we run our apps from Titanium studio by selecting the app in the project explorer on the left, and the device name in the drop-down next to the Run icon in the top-left. You can also go to Run > Run as… to select between your various devices and emulators.

    For genymotion setup
    At first enable genymotion option by this code in terminal

    ti config genymotion.enabled true
