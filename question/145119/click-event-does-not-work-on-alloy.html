Click event does not work on Alloy

I'm trying to put a map view and one annotation. I've copied and pasted the code from the documentation:

index.xml

<Alloy>
  <Window id="index">
    <!-- Note the ns attribute is used to change the default namespace. -->
       <View id="mapview" ns="Ti.Map" animate="true" regionFit="true" userLocation="true">
         <Annotation id="mountainView" latitude="37.390749" longitude="-122.081651" onClick="doClick" />
            <!-- Place other child views here, such as an ImageView to add an overlay. -->
        </View>
    </Window>
</Alloy>

And in the controller index.js

$.index.open();
function doClick(evt){
    alert('Hello');
    Ti.API.info("Annotation " + evt.title + " clicked, id: " + evt.annotation.myid);
    // Check for all of the possible names that clicksouce
    // can report for the left button/view.
    if (evt.clicksource == 'leftButton' || evt.clicksource == 'leftPane' ||
        evt.clicksource == 'leftView') {
        Ti.API.info("Annotation " + evt.title + ", left button clicked.");
    }
};

The problem is that the click event is never fired!

Is this a bug? Does anybody knows if it's reported at JIRA?
