  • Hi Nada,

    never use two tableviews, they are memory hogs. instead use a TableViewSection for each feed.

    — answered August 21st 2014 by Moritz Knecht
    permalink

    • Hi Mortiz,

      Thanks for the tip. But how can is there a way to setData() to each table section??

      — commented August 21st 2014 by Nada Dawoud

    • see [appendSection] (http://docs.appcelerator.com/titanium/3.0/#!/api/Titanium.UI.TableView-method-appendSection) you can use this to add a new section to your table,

      otherwise simply use 

      tableView.setData([section1, section2]);

      or even better use a listview, which is faster and has better section support
