How to use element ID as a variable and then use the variable to reflect changes to ID

Hi,

Using latest Ti studio for iphone. I have this in my xml

<View id="currView"></View>
<View id="currView2"></View>
<View id="currView3"></View>

in my js

var s = ['currView', 'currView2', 'currView3'];

for (i=0; i<2; i++) {
    $.s[i].setBackgroundColor('blue');
}

this doesn't work but it doesn't give an error.

  • Hi

    You were very close.

    As Alloy is object based - the normal JavaScript operations can be used to exploit the structure.

    var s = ['currView', 'currView2', 'currView3'];

for (i=0; i<s.length; i++) {
    $[s[i]].setBackgroundColor('blue');
}

    (Notes: I also fixed a logic typo, I<s would have only shown two items.

    In JavaScript then following are logically equal;

    obj.firstname = 'Malcolm';
obj['firstname'] = 'Malcolm';

    Have fun.
