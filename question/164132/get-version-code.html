Get version code

Hi,

I have set version number and version code in tiapp.xml file as follow :

<android xmlns:android="http://schemas.android.com/apk/res/android">
    <manifest android:versionCode="18" android:versionName="1.8">
    </manifest>
</android>

Now how to get this version code/version name using code?

Thanks in advance.

  • Hello Gadhia,
    To get the android version name and code, you have to use android specific property.

    Titanium.App.Android.appVersionName  //this will give you the android version name
Titanium.App.Android.appVersionCode  //this will give you the android version code

    Hope this will help you.please make sure these properties will only work for android and you need to check the platform name as android before using above property.
    Thanks,
    Jitendra
