Compatible layout

I am implementing something like this:

index.xml


<Alloy>
    <Window class="container">
        <Label id="labelTest">Test Label</Label>
    </Window>
</Alloy>

index.js

var screenWidth = Ti.Platform.displayCaps.platformWidth;
var screenHeight = Ti.Platform.displayCaps.platformHeight;

function GetHeight(value) {
    var temp = (value * 100) / 480;
    return parseInt((screenHeight * temp) / 100);
}

function GetWidth(value) {
    var temp = (value * 100) / 320;
    return parseInt((screenWidth * temp) / 100);
}

$.labelTest.height = GetHeight(30);
$.labelTest.width = GetWidth(30);
$.labelTest.fontSize = GetHeight(12);

$.index.open();

Now, above code takes label height, width and fontsize as per device's screen height and width.
But is there any way to implement this for all the label / button instead of calling this function for each and every label/button.
I am developing android app.

  • Hello Preety,

    I suggest you to use dp. it is display same layout in all resolution of android. something like :

     <Label id="labelTest" height="30dp" width="30dp">Test Label</Label>

    if you not write dp just change following line in Tiapp.xml.

    <property name="ti.ui.defaultunit" type="string">system</property>

    to

    <property name="ti.ui.defaultunit" type="string">dip</property>

    Then you can save some typing and do something like :

    <Label id="labelTest" height="30" width="30">Test Label</Label>

    Also see this link for more information in detailing Same Layout.
