  • At the moment, there are no window animations to speak of for android, except the default transition where the current screen slides to the left and the new screen appears underneath.

    With views, you have a scrollable view, which allows the user to swipe left and right and the views will appear to slide in the appropriate direction. See this in action in the KitchenSink / Views / Scroll Views / Scrollable View. The script is scroll_views_scrollable.js.

    I hope this answers your question.
