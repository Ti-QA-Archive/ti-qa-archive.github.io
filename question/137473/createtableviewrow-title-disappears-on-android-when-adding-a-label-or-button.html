createTableViewRow title disappears on android when adding a label or button

The following works perfect for IOS but not on android, the title of the row is not showing:

var oBrandRow = Ti.UI.createTableViewRow({title:'Merk'});

oBrandRow.add(Ti.UI.createButton({
    backgroundImage:'/images/arrow_gray.png',
    top:15,
    right:16,
    width:9,
    height:14
}));

When I don't add the button on android the title is showing fine. I've got the same issue when trying to add other elements to the row. The elements that I add don't overlap with the row title so that isn't the issue.
