License violation detected??

Hi,

Somehow the app that I am developing is now generating an error:

2011-03-02 23:29:43.475 Ocomondo[5880:207] [ERROR] License violation detected. Unlicensed module(s) detected. Unlicensed modules can be used for evaluation purposes only.. Please contact Appcelerator Support.

What is wrong and how can I fix it?

  • You will get that error if you were testing any of the add on modules that require licensing to use. ie. Ti+Plus.

    I was doing the same but I don't know how to remove the error message. I have tried to correct the tiapp.xml by removing the reference to the module, and also removed the module from the Library->Titanium->Modules directory but I have not been able to get rid of the error.

  • My SWAG is you will also need a clean build. For iphone, empty (but do not delete) the project/build/iphone folder. For android, empty (but do not delete) the subfolders of project/build/android.

    Then build the project again and I suspect the error will go away.
