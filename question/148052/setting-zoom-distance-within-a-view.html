  • Hi

    The reason this one is tricky is because the answer is not the same same for different things.

    The zoom scale is based on the ratio of the child to the parent, so if the child size differs for each of your items then the scale will as well.

    If the parent is 500 wide and the child is 250 wide then the scale would be 0.5.

    Simply divide the child size by the parent size. Keep in mind you should take both the height ratio and the width ratio into account as you might want to use the smaller of the two answers if you want to make the child fit within the parent in both directions.
