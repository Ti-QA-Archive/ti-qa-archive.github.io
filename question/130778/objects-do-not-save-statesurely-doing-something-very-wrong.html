Objects do not save state...surely doing something very wrong

Very simple code.
Does not work…
In log there is always 1.
What i am doing wrong?
Tryed this syntax, pure objects without enclosing functions, pure variables without objects… It's never saves the state.
Please help, can not ever start with desktop with this strange behaviour

<html>
    <head>        
        <script>
            var Counter=(function(){
                return {
                    c:0,
                    inc:function(){
                        this.c++;
                        Titanium.API.info(this.c);
                    }
                };
            }());
        </script>

    </head>
    <body style="background-color:#1c1c1c;margin:0">
        <div style="border-top:1px solid #404040">
            <div style="color:#fff;;padding:10px">
                Welcome to Titanium
            </div>
        </div>
        <a href="" onclick="Counter.inc()">
            Test
        </a>
    </body>

    Hi Anton.

    Because you use self invoking function, object stored in "Counter" variable is recreated every time. That's why it does not save state (it's not same object, actually).

    To get that working, remove self invoking function:

    var Counter= {
    c:0,
    inc:function(){
        this.c++;
        Titanium.API.info(this.c);
    }
};
