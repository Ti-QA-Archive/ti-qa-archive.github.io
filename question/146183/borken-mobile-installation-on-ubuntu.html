Borken Mobile Installation on Ubuntu

This is a first time 12.04 Ubuntu installation. All the pieces are installed for an initial installation, including Sun Java 1.6. But, TS cannot find, and wont install the Titanium Mobile SDK. Its unclear to me from the docs exactly where these are downloaded during an initial install, or if they might be actually dl'd and installed during the first launch and set up. Or perhaps there is a separate installation just for the sdk.

The error is:

Titanium SDK Home: Could not locate a Titanium SDK at the given path.

Titanium is installed in app/Titanium_Studio, and the binary launches fine from there. But setting that is the SDK path is no good. And there is nothing in there that looks like a SDK (to me). TS seems to want to use .titanium as the default location, which does exist, but there is nothing there (empty folder). I have another Ubuntu installation and .titanium seems to be the correct location.

Android SDK is configured, and found OK. Going to help and select 'Check for Updates' says 'No updates found'. Then click on 'Check for Titanium updates', and absolutely nothing happens. No popups, no error messages (that I can find) at all. It seems like chicken and egg … no sdk, everything seems fubar. I've looked at every configuration option I can find.

I am probably missing a step, but how to install the mobile sdk?
