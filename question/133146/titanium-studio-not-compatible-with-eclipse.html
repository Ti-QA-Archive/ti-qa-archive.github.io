Titanium Studio not compatible with Eclipse?

I had a perfectly working Eclipse environment, and successfully published 4 smartphone apps, before I decided to try Titanium. No matter what I tried I couldn't get Titanium working. It also failed to recognize the Java, Android, and Apple SDK's that were already installed and working. So I ended up uninstalling Titanium and then discovered my Eclipse environment was corrupted. None of the published apps would compile. Titanium completely modified all the projects when I imported them and left me with corrupt classpath and .project files. Just a head's up to save others the frustration.

  • Hey Michael,
    Titanium is not generating a complete eclipse compatible project like Xcode one. You can run your Titanium iOS application in Xcode too because Titanium is generating whole working project for Xcode. It's not like that in case of android. Java code will be generated run time so a whole working code (eclipse compatible) will not be generated using titanium. Project generated in that build/android folder will contain only few start up file.
