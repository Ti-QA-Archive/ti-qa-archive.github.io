How to make a ListView work inside a TabGroup?

I'm using Titanium Studio 3.2.0 running in OSX Mountain Lion developing for iOS and Android and I'm trying to use ListViews inside different tabs, but when I click any item in the list nothing happens, not even a simple alert box, at most the click sound on my Android device. The same thing happens in the iPhone simulator, clicking an item doesn't trigger any action, at least nothing is displayed on screen. Here is an example code:

<Alloy>
    <TabGroup barColor="purple" id="glosWin">
        <Tab id="perfTab" title="Perforación">
            <Window title="Glosario de Perforación">
                <ListView onItemClick="showPerfDescription" id="perfList">
                    <SearchBar id="perfSearchBar"></SearchBar>
                    <ListSection>
                        <ListItem title=":3" subtitle="wohooo!" searchableText=":3"></ListItem>
                    </ListSection>
                </ListView>
            </Window>
        </Tab>
        <Tab id="prodTab" title="Producción">
            <Window title="Glosario de Producción">
                <ListView onItemClick="showProdDescription" id="prodList">
                    <SearchBar id="prodSearchBar"></SearchBar>
                    <ListSection>
                        <ListItem title="Lulz" subtitle="pululunga" searchableText="Lulz"></ListItem>
                    </ListSection>
                </ListView>
            </Window>
        </Tab>
    </TabGroup>
</Alloy>

I don't know if this is a bug or I'm missing something. I'm also surprised I couldn't find any example of something as simple as this. Do you know if this can be solved? If not, do you know if there is a workaround for this?

I didn't want to use TableViews because I will be likely inserting thousands of items in these lists and I know that the TableView is a very slow and not very efficient tool.
