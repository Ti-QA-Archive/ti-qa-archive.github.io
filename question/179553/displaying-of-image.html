    Hi Sydney,

    There is no need to add images into all folders just put one higher resolution image. (xxhdpi)
    but in this case you must assign image size fixed.. don't use Ti.UI.SIZE for controller.

    but don't worry about it.. it's looking nice in all device just use fix size with "dp".

    and more thing if want to use different image for different resolution (for e.g for annotation image we can't assign its size so that's time higher resolution looks big in smaller device. that time just put annotation image only in different folder and keep it all image in higher resolution folder).

    hop its helps you

    if you have any doubt then let me know.
    :)
    Thanks

    height: 'Ti.UI.SIZE',
    image: '/images/storm_logo.png'
