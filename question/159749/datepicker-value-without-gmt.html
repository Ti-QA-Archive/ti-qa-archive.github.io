Datepicker value without GMT

i want only local date and time from date picker but i got date+time+GMT difference time like " November 21, 2013 at 5:00:40 PM GMT+6 ". i don't want GMT+6. how can i do it ? is it possible ? Below is my code -

var minDate = new Date();
    var curMonth = 0;
    for (var i = 0; i < 12; i++) {
        curMonth = minDate.getMonth();
        curMonth--;
        minDate.setMonth(curMonth);
    }
    var now = new Date();
    var datePicker = Ti.UI.createPicker({
        type : Ti.UI.PICKER_TYPE_DATE,
        maxDate : new Date(2020,12,31),
        value : now,
        minDate : minDate,
        selectionIndicator : true
    });

datePicker.addEventListener('change', function(e) {
        label.text = e.value.toLocaleString();

  • Hello Nuibb,

    Download this file Moment.js. And copy it to your application resource folder.

    Some Example of Moment.js. 

    
var moment = require("moment");

moment().format('L');        // It return    11/22/2013
moment().format('l');        // It return    11/22/2013
moment().format('LL');        // It return       November 22 2013
moment().format('ll');        // It return     Nov 22 2013
moment().format('LLL');      // It return     November 22 2013 6:10 PM
moment().format('lll');        // It return    Nov 22 2013 6:10 PM
moment().format('LLLL');    // It return    Friday, November 22 2013 6:10 PM
moment().format('llll');        // It return    Fri, Nov 22 2013 6:10 PM

    For more information about Moment.js file see it Documentation . I hope this is very useful to you and other and you get solution from here. thanks. :)
