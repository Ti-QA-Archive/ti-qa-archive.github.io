    • two options:

      • set button.touchEnabled to false
      • add a condition to the click listener callback

      — commented August 7th 2014 by Manuel Lehner

    • thanx @Manuel but my button is this way

      <Button bindId="statusButton" left="80%" onClick="show"></Button>

      So how do i access this button and enable and disable a few depending on my conditions?

      — commented August 7th 2014 by Vishnu Khaunte

    • You can bind the condition to the touchEnabled property, e.g.

      <ListItem template="{template}" statusButton:touchEnabled="{status}"/>

      — commented August 7th 2014 by Manuel Lehner

    • example assumes that status is boolean
