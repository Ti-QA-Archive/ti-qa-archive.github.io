i have two tabs tab1 and tab2

tab 1 have win1 and win2
tab2 have win3 and win4

tab 1 loads win1 by default and then it links to win2

    You can either listen for the blur event on the second window and close the window to show the first window. Or you can listen for the focus event on each tab and then check to see if the second window is open and close it if necessary.
