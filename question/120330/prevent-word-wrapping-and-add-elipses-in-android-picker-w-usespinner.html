prevent word wrapping and add elipses in Android picker w/ useSpinner

I'm using a picker with useSpinner in android and for long titles, I'd like to prevent the title from wrapping to a second line (and covering the next title) and instead just add an elipses.

I tried adding:

wordWrap:false,
ellipsize:true,

to the pickerRows with no effect (they're not documented as pickerRow properties, I was just hoping) Is this possible on android?

  • Hi,

    Go down to the bottom of this page

    http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.UI.PickerRow-object

    there is a sample of creating a picker row.you can use the same sample with the following modification 

    var row = Ti.UI.createPickerRow();
var label = Ti.UI.createLabel({
    text:text,
    font:{fontSize:24,fontWeight:'bold'},
    color:text,
    width:100,//we specify constant width value
                    // that will append elipses if the text is more than the width specified instead of auto
    height:50
});
row.add(label);
picker.add(row);
