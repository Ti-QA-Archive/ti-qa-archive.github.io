    • For this task i think fireEvent is best..i had same problem and found this solution..

      let consider you have 3 windows….A -> B -> C

      write this into window B

      Titanium.App.addEventListener('CloseWin', function(e) {
    B.close();
});

      now in window C on button click write this

      Titanium.App.fireEvent('CloseWin');
C.close();

      so this code will close your both window B & C and Redirect to window A.
