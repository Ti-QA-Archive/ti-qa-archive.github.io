Application (Activity) resume event on Android

Any suggestions how you catch the resume of an application on Android?

I tried setting the following event in my app.js:
Ti.Android.currentActivity.resume

but it didn't trigger when I went out to another application then brought my application back to the foreground.

    • In Android, Resume & Pause event fire by following code

      // For resume
win.activity.addEventListener('resume', function(){
    alert('App resume');
});

// For pause
win.activity.addEventListener('pause', function(){
    alert('App pause');
});
