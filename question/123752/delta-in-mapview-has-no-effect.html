Delta in MapView has no effect

Okay … where is the difference in using the MapView since 1.6.2?
I use this code for positioning my map:

if(yh.helper.currentLocation.geoState){
            var _region = {latitude: yh.helper.currentLocation.latitude,
                        longitude: yh.helper.currentLocation.longitude,
                        latitudeDelta: 0.03,
                        longitudeDelta: 0.03}
        }else{
            var _region = {latitude: 50.07605,
                        longitude: 8.253758,
                        latitudeDelta: 0.03,
                        longitudeDelta: 0.03}
        }
        var mv = Ti.Map.createView({
            animate: true,
            mapType: Titanium.Map.STANDARD_TYPE,
            region: _region,
            regionFit: true,
            userLocation: true
        });

SDK 1.6.2 works without problems. Since 1.7. receive the map without any zoom.
