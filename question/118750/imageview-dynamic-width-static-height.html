ImageView: Dynamic width static height

I created a ImageView with dynamic width and a fixed height of 200px. The content comes from a server and could have any size.

iPhone shows the image in correct proportion but android fills the whole ImageView.

How can I tell android to scale the image relative in height and width with a max height of 200?
