Fetch scaledDensity on Android

Is there any way to fetch the scaledDensity value on Android devices from Titanium? I'm trying to create a pixelRatio that will scale text appropriately on all devices and I need this value.

  • Use the sp unit. (which is the same as dp, but for fonts) This will ensure that the size is scales based on dots-per-inch (dpi).

    var label = Ti.UI.createLabel({
    width: 'auto',
    height: 'auto',
    font: {
        fontSize: '10.5sp'
    },
    text: 'Hello world!'
});

    from: http://developer.appcelerator.com/question/130143/font-sizes-and-different-screen-resolution-on-android—how-to-make-it-fit-all-screens
