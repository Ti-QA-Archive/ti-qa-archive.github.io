TableView styling inside NavigationGroup?

app.tss

"TableView": {
    backgroundImage:"style_elements/list-background.png",
    separatorStyle:'none',
    backgroundColor:'none',
    rowHeight:77,
}

test.xml

<Alloy>
    <Window id="document_window" navBarHidden="true">

        <NavigationGroup  id="document_navigation_group">

            <Window id="win1">        
                <TableView id="document_table_view"  />
              </Window>

         </NavigationGroup>            

    </Window>
</Alloy>

Ok, so I populate the document_table_view with this:

// populate top level navigation
$.document_table_view.data = data;

I run this and it works as expected, the TableView is styled according to app.tss.

Now, I create a new tableview and window for the navigation group

    var w = Ti.UI.createWindow({

        backgroundColor:'#ff9900',
        title:e.rowData.title,
        backgroundColor:'transparent',
        barColor:'transparent',
        });

    // create table view
    var tableview = Titanium.UI.createTableView({
        data:data
    });

    // add table view to the window
    w.add(tableview);

So…. when I run this the first level of the NavigationGroup looks great. The second level looses it's TableView styles from app.tss

Why doesn't the TableView styles in app.tss cascade down to all the TableViews?
