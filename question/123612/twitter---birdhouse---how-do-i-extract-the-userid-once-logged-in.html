  • look here
    https://github.com/jpurcell/birdhouse/blob/master/birdhouse.js#L297 and you will see that the value is being set in the private configuration property.

    you can add your own public method to return that data buy doing something like this around line 1140

        this.config = cfg;

    and that will return all of the configuration values including the user_id
