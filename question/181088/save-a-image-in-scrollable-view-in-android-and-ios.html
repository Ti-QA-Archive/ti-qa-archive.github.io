    Hi Aparna,

    Yes it's save only last image as per your code. because you create all views including image and button. and once it's created successfully and then your try to save it into gallery using image object and that it's store last reference only.

    so for solution you need to assign imageView reference to button and then you can get it using btn..

    e.g

    ...
view.add(img);
view.add(btn);
btn.img = img;

photosView.addView(view);
photosView.currentPage = win.myID;
btn.addEventListener('click', function(e) {
    var blobObj = e.source.img.toBlob();
    Ti.Media.saveToPhotoGallery(blobObj, {
                success : function(e) {
                    alert('Saved image to gallery');
                },
                error : function(e) {
                    alert("Error trying to save the image.");
                }
            });
        });

    Try it and let me know..
