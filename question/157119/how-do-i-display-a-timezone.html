How do I display a timezone?

I am using the moment.js library to handle all of the dates in my app and I wanted to add the timezone to my dates. Looking through moment.js I realized the display of timezones has been deprecated. Is there another way I could display the timezone?

Here is an example:

var date = moment(item.get('start_time'), 'YYYY-MM-DDTHH:mm:ssZ');
var dateEnd = moment(item.get('end_time'), 'YYYY-MM-DDTHH:mm:ssZ');

$.date_dow.text = date.format("ddd").toUpperCase();
$.date_day.text = date.format("DD");
if (dateEnd && dateEnd.isValid()) {
    $.time.text = 'from ' + date.format('h:mm a') + ' to ' + dateEnd.format('h:mm a');
} else {
    $.time.text = 'at ' + date.format('h:mm a');

  • Hi,

    You need to download moment-timezone.js file and use this library to work on Timezones.
    http://momentjs.com/timezone/docs/

    This piece of code for a specific time zone. 

    var moment = require('moment-timezone');
moment().tz("America/Los_Angeles").format();

    Check this data builder tool to construct data sets for the time zone that you only need.

    http://momentjs.com/timezone/data/

    You need moment@2.1.0 or greater to use this.
