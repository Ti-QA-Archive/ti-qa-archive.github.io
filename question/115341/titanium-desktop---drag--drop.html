Titanium Desktop - Drag & Drop

Hi,

First of all congratulations on what could well be a massively successful platform!

We are in the process of evaluating your products as a cross platform alternative to Adobe Flex/Air.

However, we are trying to get drag & drop working in a TD application. We are using the JQuery based (http://www.plupload.com/) Plupload library which works perfectly within Safari, Chrome & Firefox. However, the drop seems to be disabled when I package and run the TD application.

We have also tried the following example which again works in the browser but not in a TD application:-

(http://robertnyman.com/html5/fileapi-upload/fileapi-upload.html)

Any suggestions/guidance/pointers would be very much appreciated.

  • This should be possible.

    I just tried it with a simple form text field (so it should be possible using other 'standard' html browser-based drag and drop solutions.)

    Add this text field (for example) to a test app:

    <form><input type='text' name='file' id='file' style='width:100px;height:50px;'></form>

    then drag a file in to it. You'll see that when you do, you get the file path. You could then add an onChange listener to this to tell the app code where the file is.
