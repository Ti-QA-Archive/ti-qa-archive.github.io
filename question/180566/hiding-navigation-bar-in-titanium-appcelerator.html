  • Hi sanyogita gowri,

    Thanks for your concern.

    Release 3.3.0, the tiapp.xml element, <navbar-hidden>, which used to be able to hide the action bar, has no effect on Android applications.To hide the action bar, the application needs to wait for the main window or tab group to open, then call the Activity's actionbar.hide() method; or create a theme that hides the action bar.

    For details please have a look:

    Notice of Feature or Behavior Changes in Release 3.3.0

    Thanks
