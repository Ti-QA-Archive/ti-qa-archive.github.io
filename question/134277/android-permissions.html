Android Permissions

Is there a way to remove the default permissions in the Manifest file. I've seen explanations of creating a custom Manifest file, but unless I'm doing something wrong, the Manifest is overwritten whenever I build.

  • Accepted Answer

    You should read the docs on at how to create and maintain a custom AndroidManifest.xml.

    In short, create the directory platform/android, copy the AndroidManifest.xml from the build/android directory there, and edit it. Before build a store release don't forget to check the differences between build/android/AndroidManifest.xml.gen and your platform/android/AndroidManifest.xml, as you'll have to updated it with any name changes, version number changes etc.

    First you make the duplicate file of your "AndroidManifest.xml" , Rename it With the name "AndroidManifest.custom.xml" and keep this file into same folder where your "AndroidManifest.xml" placed.

    Now change into "AndroidManifest.custom.xml" file whatever you want. Then run your project without clean build..

    So this will make the changes into your original "AndroidManifest.xml" file…
