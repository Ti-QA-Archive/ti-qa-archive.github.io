  • Hi gagan

    The emulator will only show that camera it will not take the snap to test that part you need to install the app on the device and test it. Emulator will not take any picture.
    Adding the function to capture the image and save

    
// function to capture image 
function takePhoto(){

    Titanium.Media.showCamera({

        success:function(event)
          {
            var cameraView = Ti.UI.createImageView({
                width: 320,
                height: 480,
                top: 0,
                left: 0,
                image: event.media
            });
            cameraView.add(overImage);

        var fileName = "/randomimage"+".png";
            var filename1 = Titanium.Filesystem.applicationDataDirectory + fileName;
                f = Titanium.Filesystem.getFile(filename1);
                f.write(cameraView.toBlob());
                Titanium.Media.saveToPhotoGallery(f);

                isChooseImage = false;
            capImg.image = f.nativePath;
            imageLocation = f.nativePath;
          },
          cancel:function()
          {
          },
          error:function(error)
          {
            var a = Titanium.UI.createAlertDialog({title:'Camera'});
            if (error.code == Titanium.Media.NO_CAMERA)
            {
                a.setMessage('Please run this test on device');
            }
            else
            {
                  a.setMessage('Unexpected error: ' + error.code);
            }
                a.show();

},
              overlay:overImage,
              showControls:true,
              mediaTypes:Ti.Media.MEDIA_TYPE_PHOTO,
            saveToPhotoGallery:false,
            allowEditing: true,
            allowImageEditing:true,
        autohide:false   // tell the system not to auto-hide and we'll do it ourself
    });
}

    Hope this will work for you too. And if its work do add a vote and set it as answered.

    Nikunj
