  • With Xcode 6, you must be running the current release candidate (rc) versions of Studio, the CLI, and the SDK. See our blog article for links and install information.

    The specific error you're getting usually indicates that you're trying to use a 3.3 version of Studio with newer versions of other components, such as Xcode or the CLI. They must all be set to a 3.4.0-rc* version.
