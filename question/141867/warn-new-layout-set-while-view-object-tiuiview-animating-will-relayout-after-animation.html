[WARN] New layout set while view [object TiUIView] animating: Will relayout after animation

Hi

For my iPhone app i am needing to animate several views at the same time. Because of this i am getting the following error:

[WARN] New layout set while view [object TiUIView] animating: Will relayout after animation.

I have had a look around the Q&A's and it seems i could put a setTimeout between the animations but i am very reluctant on using them as i've heard they always cause some form of memory leaks, is this correct or am i safe to be using setTimeout?

Is this going to harm my app or cause it to crash as it's only a warning rather than an error?

My code is:

frontView.addEventListener('click', function() {
    animateViewOut(frontView, backView, sideView, topView, bottomView);

    animateViewDown(tyreView);
});

// Functions
function animateViewOut(i, o_1, o_2, o_3, o_4){
    i.animate({left:0, duration:300});
    o_1.left=o_2.left=o_3.left=o_4.left=-52;
}

function animateViewDown(d){
    d.animate({top:282, duration:300});
}
