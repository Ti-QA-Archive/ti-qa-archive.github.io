Dynamic url in webview doesnt change when variable changes

Hi all,

I got 5 tabs that open a webview.
i got the problem that the variables in the url of the webview doesnt change when the variable changes.

my webviews looks like this

var win3 = Titanium.UI.createWindow({  
    backgroundColor:'#ededed',
    navBarHidden: true
});
var tab3 = Titanium.UI.createTab({  
    icon: '164-glasses-2.png',
    title:'Zoeken',
    window:win3
});

var webview3 = Ti.UI.createWebView({
    url:'http://www.domain.nl/?usernaam='+Ti.App.Properties.getString("username")+'&key='+Ti.App.Properties.getString("password")+'&sub=zoeken&net='+Titanium.Network.networkTypeName+'',
    height:'auto',
    width:'auto'
});

win3.add(webview3);

how do i make the variables changes when username or password changes

  • Accepted Answer

    The code you have written above will execute only once(at the time of web view creation), if you want that the url of web view should change dynamically, then use

    webview.url = &#39;http:&#x2F;&#x2F;www.domain.nl&#x2F;?usernaam=&#39;+Ti.App.Properties.getString(&quot;username&quot;)+&#39;&amp;key=&#39;+Ti.App.Properties.getString(&quot;password&quot;)+&#39;&amp;sub=zoeken&amp;net=&#39;+Titanium.Network.networkTypeName+&#39;&#39;,

    in the change listener whenever your username and password changed, and dont forgot to use webview.reload()to refresh your webview again. i hope this will work for you.
