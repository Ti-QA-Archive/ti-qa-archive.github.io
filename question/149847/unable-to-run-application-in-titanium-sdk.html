  • Titanium Mobile Web apps require a web server. Titanium Studio has a built-in web server which it uses when you do a "Preview in Browser" build. Sometimes this build-in web server has troubles starting. Usually building a second time will fix it. Otherwise, try restarting Studio and building a couple times. Alternatively, you could setup Nginx or Apache and configure the web root to "/path/to/yourproject/build/mobileweb".
