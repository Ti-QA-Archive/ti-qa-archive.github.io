Checkbox UI control on iPhone?

I have an app with a series of table rows where I was using switches to determine whether an item in the row was chosen.

Is there a checkbox control that can perform a similar purpose?

  • Accepted Answer

    No, iOS does not have a native "Checkbox" control like Android does. However, I have worked around this and have been using well-placed ImageViews and click events to empty and fill a box while setting properties. It's an annoying walk-around, but necessary because the "On/Off" text didn't really translate perfectly with how my app went.
