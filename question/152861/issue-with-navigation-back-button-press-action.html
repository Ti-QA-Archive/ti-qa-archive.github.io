  • Hi

    It is possible to use something other than a button in the navigation left and right buttons, including; labels, views and images.

    Try an imageView and manage the image toggle yourself on touchstart and touchend. You can emulate the action of the back button with;

    nav.close(win);
// nav = the navigation controller
// win = the window you want to close

    The "hit" point for a none button should be a little smaller.
