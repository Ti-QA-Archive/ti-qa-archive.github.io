  • This woud be a good questions to ask on stackoverflow since folks build upon the answers (for future users to discover). You can post to:

    • http://stackoverflow.com/questions/tagged/tidesdk

    Also note the desktop project is now officially an open source non-profit project called TideSDK:

    • http://developer.appcelerator.com/question/142044/tidesdk—stack-overflow-tag-created-for-q–a

    The short answer is that anything that is not currently possible within the api is possible by using mature libraries together with the available scripting language support in python, php, or ruby. There are libs readily available for working with with IDE in these languages if you cannot find a JavaScript alternative.

    If you would like to see this feature included in the TideSDK API you can file an issue on github for TideSDK devs to consider:

    • https://github.com/TideSDK/TideSDK/issues
