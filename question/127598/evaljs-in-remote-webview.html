EvalJS in remote webview

Is it possible to evaluate js on a remote webview?

Example: How could I load www.google.com and then write to the loaded document with document.write "Hello world"

Based on the docs i would assume this would be the way to do it…

var webview = Titanium.UI.createWebView({url:&#39;http:&#x2F;&#x2F;www.google.com&#39;}); webview.evalJS(&quot;document.write(&#39;hello world&#39;);&quot;); Titanium.UI.currentWindow.add(webview);

However, It did not seem to work for me.

My questions:
1 - Is it possible to evaluate js on a remote webview?

2 - Should this be working or am i wrong about my understanding of this?

3 - If not with evalJS or the above method, is there any way to make this work?

Using iphone, latest stable release version of everything.

Sincerely,

    I needed to listen for the load event.

    var webView = Ti.UI.createWebView({url:"http://www.google.com"});
Titanium.UI.currentWindow.add(webView);

webView.addEventListener('load',function(e)
{
   webView.evalJS("document.write('hello world')");
});
