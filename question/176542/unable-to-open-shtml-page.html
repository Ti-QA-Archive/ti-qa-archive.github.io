  • Hi

    I am going to assume you mean the SHTML file is stored locally on the device and you tried to load that in a webView.

    SHTML file extension is used for html files that;

    • The SHTML file type is primarily associated with 'HTML File Containing Server Side Directives' by Microsoft Corporation.

    Which means the file would need to be on a server not on a local device in order to work. Even if the file itself contained no server side elements - it is unlikely the file extension would be a default type.
