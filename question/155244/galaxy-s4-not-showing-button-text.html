Galaxy S4 Not Showing Button Text

Has anyone else experienced this? It seems to be isolated to my client's Samsung Galaxy S4 as other team member Android devices show the button text fine.

The text and text color are static values and set properly but I cannot figure out why the text isn't showing up on his device.

  • Figured out the problem which seems to be only isolated to new Android devices/phones…the default unit setting in the tiapp.xml under 

    <property name="ti.ui.defaultunit" type="string">dp</property>

    should be set to 'dp' instead of 'system' which is the default. Along with setting values to '20dp' instead of 20, the button text shows up properly. Weird but makes sense.
