Changing image parameter of ImageView and zIndex

I have the following:

var imageView1 = Ti.UI.createView({
    top: 0, left: 0, width: 100, height: 100,
    image: 'http://x.com/image1.jpg',
    zIndex: 10
});

var imageView2 = Ti.UI.createView({
    top: 50, left: 50, width: 25, height: 25,
    image: 'http://x.com/image2.jpg',
    zIndex: 20
});

win.add(imageView1);
win.add(imageView2);

imageView1.image = 'http://x.com/image3.jpg';

So there is two imageviews over eachother (the imageView2 can be any other view or UI object), and I change the image property of the bottom image. This causes Titanium to download the image from the URL, then fade in with an animation. After this the imageView1 will be the topmost image and it covers the imageView2 enterly, no matter how I set the zIndex properties. How can I fix this?
