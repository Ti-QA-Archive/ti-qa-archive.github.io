    • OK. So. If you download one of the m3u files and open in a text editor, you see the last entry eg : http://207.200.96.226:8048

      That is likely to be a ShoutCast stream. A very little know fact/hack, is you can put /7.html on the end of that and see a small bit of text saying what is currently playing.

      eg: http:&#x2F;&#x2F;207.200.96.226:8048&#x2F;7.html

      For a radio app I built, I had a server PHP script polling that and parsing it to something useful like JSON. ;)
