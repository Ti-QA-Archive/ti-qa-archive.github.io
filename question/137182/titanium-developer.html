    Titanium Developer isn't supported by Appcelerator anymore, but there is a community project still working on it: https://github.com/adrianopaladini/titanium_developer

    I personally have started using Ti Studio to set up my projects, then I resize my UI and adjust it to only show the console and the build button, and write my code in Espresso. Just so I easily can access the tiapp.xml editor or other functions when I need them.
