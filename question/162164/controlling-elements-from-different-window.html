Controlling elements from different window

I want to control <View><Label> and that sort of elements from index.js.
I'm trying various ways to do it, but I still can't able to do that sort of things in this time.

Could someone give me a simple sample to solve it?

window2.xml

<Alloy>
    <Window class="container">
        <View id="view_foo"></View>
        <Label class="label_bar"></Label>
    </Window>
</Alloy>

For example, I want to control "view_foo" and "label_bar" in window2.xml from index.js.
How can I do that?

I already wrote this part in index.js. 

$.button_baz.addEventListener('click', function(){
    var win = Alloy.createController('window2').getView();
    win.open();
});

    Hi

    When you call 'window2' it is possible to pass data through the controller. To do this simply pass a second parameter in the createController method, like this example:

    var win = Alloy.createController('window2', {backgroundColor: '#fff'}).getView();

    Inside your window.js file you can then retrieve the data that you pass and set the properties of different ui elements like so:

    var args = arguments[0] || {};

$.view_foo.backgroundColor = args.backgroundColor;

    If the ui you are trying to change already has a property set you can use the set methods to alter them.

    If you have more than one property to change on any single ui element you can use 'applyProperties'

    $.view_foo.applyProperties({
    backgroundColor: args.backgroundColor,
    top: 100
});

    Hope this makes sense and helps!

    • Hi

      Inside your window2.js you would create a function to hold your animation, by using the syntax '$.someFunction' it allows you to export access to this function. You can create as many of these as you need.

      window2.js

      $.someFunction = function () {
    // You can place any code in here to handle your animation
    // that can be initiated by whatever opens this controller.    
};

      Inside your index.js you now need to remove getView() so that you have access to your whole controller as opposed to just the view.

      index.js

      var win
$.button_baz.addEventListener('click', function(){
    win = Alloy.createController('window2');
    win.getView().open();
});

      // This will run the function within window2.js
// this can be placed anywhere after you have set win
// at the same time or whenever you choose.
win.someFunction();

// for example
var win
$.button_baz.addEventListener('click', function(){
    win = Alloy.createController('window2');
    win.getView().open();
    win.someFunction(); // it could be called here
});
