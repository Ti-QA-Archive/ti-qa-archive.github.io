    Ahmed, you can do this by 2 ways

    1) Direct run into device
    2) Run on Emulator and Install .apk to device

    Direct run into device

    1)connect your android device with pc

    2)right click on project

    3)Click on "Install to android Device"

    Wait while installing app into device

    Install .apk file

    1) Run application into Emulator

    2)Right click on project/Showin/Finder

    3)copy that file > Project/Build/android/bin/app.apk

    4) paste this file into device sd card

    5) tap on icon from device sd card and install application

    Feel free to ask again if any problem…
