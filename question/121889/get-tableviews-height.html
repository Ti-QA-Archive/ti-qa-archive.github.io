Get TableView's height

I have a complex window with several views in it. I want to scroll the full page with all the views at once. I have a ScrollView attached to the Window, and a View with auto height in it. In this View are the other two Views and a TableView. I have exact height for the two views, but the table has dynamic height. I set scrollable: false in the table.

If I set auto height to the outer view (which is inside the scrollview), it does not work (the view gets around 100 px in height). If I set exact height, it works, but for this I need to know how much is the height of the tableview. tableview.height says 0 (if I set it to auto). How can I make Titanium to force calculate it?

  • Hi Kristof.

    Titanium calculate component dimensions when window opens. So, you'll have to add event listener to the window and in it read table view height, something like this:

    win.addEventListener('open', function() {
    Ti.API.debug(table_view.size.height);
})
