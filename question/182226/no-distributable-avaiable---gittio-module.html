No distributable avaiable - gittio module

Hi,

I am trying to install a module using gittio, like here http://gitt.io/component/so.hau.tomas.pager
but I get return below:
$ gittio install -g so.hau.tomas.pager
[INFO] so.hau.tomas.pager searching…
[ERROR] so.hau.tomas.pager:commonjs no distributable available
[ERROR] so.hau.tomas.pager:android no distributable available
[ERROR] so.hau.tomas.pager:ios no distributable available

Please, why that is not possible?

Thanks,

Maxwell

    • Maxwell,

      If the module is a 'registered' module in the appc marketplace (https://marketplace.appcelerator.com), whether thats paid/free/trial, if the module is not associated with your account, the TiVerify will throw the error.

      You could search the marketplace for the module, 'purchase' it, and then it should be associated with your account.
