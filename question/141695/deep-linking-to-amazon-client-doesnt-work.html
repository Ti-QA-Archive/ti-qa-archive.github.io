Deep linking to Amazon Client doesn't work

I'm trying to deep link directly to the Amazon Client as described here but it doesn't seem to work.

The code that DOES NOT work is:

var webview = Titanium.UI.createWebView({url:&#39;amzn:&#x2F;&#x2F;apps&#x2F;android?&#39;}); win1.add(webview);

The code that DOES work is:

var webview = Titanium.UI.createWebView({url:&#39;http:&#x2F;&#x2F;www.amazon.com&#x2F;gp&#x2F;mas&#x2F;dl&#x2F;android?&#39;}); win1.add(webview);

Obviously the one that works looks more like a website, but I want it to look more like the native Amazon App Store.

I'm using the following version of Titanium Studio:
Titanium Studio, build: 2.0.1.201204132053
Build: jenkins-titanium-rcp-master-185 (origin/master)
Date: 13 April 2012, 20:56:05

Any ideas?

    You can't open custom urls in a web view. If you want to open the Amazon app using the url scheme amzn:, then you have to use Ti.Platform.openURL()
