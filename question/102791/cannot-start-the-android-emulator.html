  • If you are using Windows, follow the instructions at Titanium Diagnostics for Windows. If you are using OSX or linux, follow the same instructions, but use this code instead:

    clear
echo "$PATH"
echo "$JAVA_HOME"
echo "$android_sdk"
java -version
javac -version
python -V
adb version
android list target
git --version
echo " --- The following is optional ---"
scons -v
echo
