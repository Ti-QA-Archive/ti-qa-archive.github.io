  • The androidsdk.py in 3.2.0 mobilesdk is missing the check for build-tools. The new android sdk moved some files from the platform-tools dir to build-tools dir. Use the androidsdk.py from 3.1.2, either replace the new file with older file or add the code that checks for build-tools dir into the new androidsdk.py file.
