  • Hi Eduardo,
    I had the same problem and couldn't solve it in Titanium. I think it was an error with Titanium not recognising/finding the certificates and/or keys (user error?).

    Anyway the solution for me was to stop using titanium for the final build and try using xcode. I had never used xcode but watched a few tutorials, made a few mistakes, but managed to fix it. When you use xcode, the error warnings are a lot better than titanium so it's easier to trouble shoot. As xcode is an official mac product, it integrates better with the certificates, keys etc than titanium.

    Titanium is definitely better and easier for me when building the app but from now on I will use xcode for the final compile for distribution. Hope this helps.
