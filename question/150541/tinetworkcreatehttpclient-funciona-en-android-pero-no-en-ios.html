    • Create the function creaVentana BEFORE the XHR code. You should be declaring things in the order they are required, not afterwards.

      Also your XHR code is also slightly out of order.

      Change the order of your code as follows;

      var cadena="";
function creaVentana(datos) {
  // existing contents of this function here
}
var id_usuario = Titanium.UI.currentWindow.id_usuario;
var url = "http://neommunications.com/crm/index.php/blog/getCategorias";
var xhr = Ti.Network.createHTTPClient();
xhr.onload = function(e) {
    datos = this.responseXML;
    creaVentana(datos); 
}
xhr.onerror = function(e) {
    Ti.API.debug(e.error);
    alert('error');
}
//  timeout : 5000 // in milliseconds

xhr.open("GET", url); // this has moved
xhr.send({
    idUsuario:id_usuario
});

      This way creaVentana is defined before being used. both onload and onerror are defined before being required by open and send.

      See how that works.
