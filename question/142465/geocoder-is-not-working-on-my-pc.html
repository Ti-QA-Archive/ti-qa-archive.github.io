Geocoder is not working on my pc

I recently install Titanium Studio. i want to run sample Geocoder code on android SDK.i have all Android Sdk software installed(API 8 and API 16). But still it is not working…help please.

    Most times, the [INSTALL_FAILED_MISSING_SHARED_LIBRARY] error means that you're building against the "Android SDK" rather than the "Google APIs SDK". The Google APIs version adds the features you need for building mobile apps (maps support among them). The Android SDK versions are the vanilla Android APIs, which I jokingly call "Android for refrigerators" because it doesn't include APIs for mobile apps.

    For your existing project:

    1. With the project open, choose Run, Run Configuration
    2. Select the project in the list on the left
    3. From the Android API list, select one of the "Google APIs" SDKs

    To set the default for future projects:

    1. Open the tiapp.xml file
    2. Below Deployment types, click Configure
    3. From the Default Android SDK list, choose on of the "Google APIs" versions

    If you don't have the Google APIs SDKs installed, open the android tool (comes with the Android development tools) and update your SDKs.
