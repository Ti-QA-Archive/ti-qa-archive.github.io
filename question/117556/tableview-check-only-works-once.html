TableView Check only works once?

I have a table view with a bunch of rows that may or may not be checked. I have an event listener that should toggle the check. However it only works ONCE per row. I can toggle rowA, and then rowB (and as many other rows as a I want) but can't get toggle rowA again. The listener isn't even fired.

Here's my listener:

    optview.addEventListener('click', function(e)
    {
        Ti.API.debug("optview click occured");
        var tmpData = e.rowData;
        tmpData.hasCheck = !tmpData.hasCheck;
        optview.updateRow(e.index,tmpData);
        Titanium.API.info('Setting '+tmpData.showCode+' to '+tmpData.hasCheck);
        Titanium.App.Properties.setBool(tmpData.showCode,tmpData.hasCheck);
    });

Btw, this code works on Android.

  • Assuming that optview is the TableView, try

    optview.addEventListener('click', function(e)
    {
        if (e.row.hasCheck) {
            e.row.hasCheck = true;
        } else {
            e.row.hasCheck = false;
        }
    });

    That should toggle the check box for the row that is clicked.
