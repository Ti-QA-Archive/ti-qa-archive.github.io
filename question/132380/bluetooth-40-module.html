  • To answer the original poster and the subsequent posters, Logical Labs has released 2 Bluetooth modules for Titanium in the Appcelerator Marketplace.

    • BT LE iOS module https://marketplace.appcelerator.com/apps/3834?2063287747
    • BT Classic Android module https://marketplace.appcelerator.com/apps/5221?259328472

    For clarity, each of these modules are considered BT 4.0 compatible however each uses a different protocol within the BT 4.0 specification.

    To understand more about the differences and ensure success on your project, I suggest reading http://en.wikipedia.org/wiki/Bluetooth#Bluetooth_v4.0

    Contact Logical Labs directly for more information on why each module is using a different protocol and for any other questions or support with the modules.
