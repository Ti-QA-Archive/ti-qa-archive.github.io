Styling ListItem in tss

Is possible to style ListItem in tss file?

I tried:

"ListItem":{
properties:{height:47}
}

It's not working. Any idea?

  • Assign a class name / Id to the ListItem and based on that add properties in tss.

    <Alloy>
    <Window class="container" title="Some things">
        <ListView id="listView">
            <ListSection id="section">
                <ListItem class="listItems" image="images/trash.jpg" title="Take Out the Trash"/>
                <ListItem class="listItems" mage="images/dishes.png" title="Do the Dishes"/>
                <ListItem class="listItems" image="images/doge.png" title="Walk the Dog"/>
            </ListSection>
        </ListView>
    </Window>
</Alloy>

    tss

    ".listItems":{
  height: 47
}
