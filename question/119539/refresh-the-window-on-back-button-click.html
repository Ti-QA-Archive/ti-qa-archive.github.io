refresh the window on back button click

i have two windows first.js and second.js.

There is a table in first.js and one lable at bottom side. second.js is a musicplayer.

when i select any row second.js opens and start the music.

when music start i want to set some values in label of first.js without open it and when i click back button first.js should open with updated label value.

Kindly reply as soon as possible.

  • All you need to do is fire an event when the Back Button is pressed. The event, in this case, would update the label in first.js

  • If first.js can detect what name should go in the label, then just add an event listener for the (first) window's focus event. In it, decipher what the label should contain and update the label's text.

    in first.js:
win.addEventListener('focus', function(e) {
  myLabel.text = 'Now playing ... ' + someValue;
});

    If first.js cannot on its own detect what to display, then second.js needs to either:

    1. Store that somewhere first.js can obtain the value (e.g. use Ti.App.Properties, or extend the namespace and store in an object), or
    2. Have second.js fire a custom event, passing the new label value in the event object data. Listen for that custom event in first.js and update the value during that event, instead of during the window focus event.
