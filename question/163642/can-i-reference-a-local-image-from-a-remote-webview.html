Can I reference a local image from a remote webview?

Hello,

I am building an Android app and I have a webview which loads HTML data from a remote json file. In that HTML data I would like to reference local images that will be shipped with the app. Is that possible? Which path should I use if it is possible?

Thanks in advance for any tip.
Have a nice day

    You'll have to save the remote HTML as a local file and load it as local, then all the resources will resolve to the local filesystem.
