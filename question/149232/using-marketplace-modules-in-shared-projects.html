Using marketplace modules in shared projects

Hello friends, I'm new to Appcelerator development and I'm running into some trouble setting everything up.
I have a project that is hsared through SVN with another person. I'm trying to install a module I "purchased" from the marketplace, and I receive an "Unlicenced module" alert.
For what I been reading, the problem is due to the fact that the project was created by my friend, and I am the one trying to add the module.

What I'm wondering is, is there a way to say that the project belongs to both? or maybe add the module without making that verification?

Thanks in advance for any hint that you can provide.
