Change position of navigation window buttons

I have a navigation window with a two buttons (cancel and submit). To be able to give the submit button a background, I have to wrap it in a view (is this necessary?). Because of that, the button appears further from the right side of the screen than the cancel button does from the left side. How can I adjust the position of the right nav button to move it closer to the edge of the window, say 10 pixels?

Simplified excerpt of my Alloy markup and styles for the button and view:

<NavigationWindow>
<Window modal="true">

    <LeftNavButton>
        <Button>Cancel</Button>
    </LeftNavButton>

    <RightNavButton>
        <View id="navButtonView">
            <Button id="navButton">Submit</Button>
        </View>
    </RightNavButton>

</Window>
</NavigationWindow>

'#navButtonView': {
    backgroundColor: '#96C11F',
    borderRadius: 3,
    height: Ti.UI.SIZE
},
'#navButton': {
    color: '#fff',
    font: {
        fontWeight: 'bold'
    },
    left: 10,
    right: 10,
    height: Ti.UI.SIZE,
    width: Ti.UI.SIZE
}

  • Button backgrounds are gone in iOS7 (at the iOS level, not Titanium). So, wrapping in a view like you're doing is the way to accomplish that effect (or using an ImageView). You're setting a left & right on the navButton. Try setting just the right property. You will probably need to set the right property on #navButtonView also.

    If it comes to it, you could wrap the left nav button in a view too, and set corresponding properties there to get the buttons to appear inset by similar amounts.
