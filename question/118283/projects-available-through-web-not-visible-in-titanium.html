Projects available through web not visible in Titanium

It's been a few months since I used Titanium. If I go to http://my.appcelerator.com/ I can see my projects but they're not showing up in Titanium.

Am I missing something obvious?

    Andrew

    New project/application entries are only created in my.appcelerator.com when you click the "New Project" button in Titanium Developer and complete the resulting screen to create a new project.

    If you still have your original project files intact, you can just import them using the "Import Project" button, which will re-establish the link between the entry on the website and the project.

    Cheers
