Possible to adjust modal window transition speed?

Just wondering if there is a way to adjust a modal windows transition speed? I'm currently using…

m.open({
        modal:true,
        modalTransitionStyle: Ti.UI.iPhone.MODAL_TRANSITION_STYLE_CROSS_DISSOLVE,
        modalStyle: Ti.UI.iPhone.MODAL_PRESENTATION_FORMSHEET
    });

Any help is much appreciated!

  • AFAIK, this is not possible with modal windows (in fact, is not possible to modify any standar iOs transition). But if you use non-modal windows, you could create any animation, maybe this helps you:

        w.opacity = 0;
    w.open();
    w.animate({opacity:1, duration:1000})
