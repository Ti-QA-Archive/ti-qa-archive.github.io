Can't import modules

I am trying to import a barcode module and have tried several already… TiBar, RedLaser, Scandit, etc… no glory

I followed the instructions here and even followed this video and I seem to be doing everything properly

This is my entry in the TiApp.xml

    <modules>
        <module platform="iphone" version="0.4.1">TiBarReaderApp</module>
        <module platform="android" version="0.4.1">TiBarReaderApp</module>
    </modules>
    <sdk-version>3.2.0.GA</sdk-version>

The TiBarReaderApp-0.4.1.zip file is directly under the project root folder and I get the following error when attempting to build:

    Hello David,

    Let's see:

    1. get the zip file with the module into Downloads directory in your mac
    2. unzip it
    3. and you will get a module directory. Copy that module directory to the same level that your Resources directory in your project (not inside Resources)
    4. Go to your tiapp.xml remove the manual entries you added, and get into the graphic UI add a module, it will show you the latest one you added.

    If that doesn't fixes the issue, please drop me a line to mpmiranda@appcelerator.com

    Best,

    Mauro
