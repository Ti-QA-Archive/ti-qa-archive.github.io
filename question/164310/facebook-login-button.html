  • Hello Isleifur Ørn Gudmundsson,

    Thanks for popping this issue. There is a way to make a custom Facebook login button with flat design.
    In alloy view page I added a custom button with Facebook login functionality. When user click on the button, doClick function will be called. doClick function is defined in the controller file (index. js), where I call the authorize () function for login.Please check my code.

    Check Attachments

    Test code:

    alloy.js

    Alloy.Globals.Facebook = require('facebook');

    index.js

    var facebook = Alloy.Globals.Facebook;
facebook.appid = 'XXXXXXXXXXXXXXX';
facebook.permissions = ['publish_stream'];
function doClick(e){
    facebook.authorize();
};
$.index.open();

    index.xml

    <Alloy>
    <Window layout="vertical" backgroundColor="pink">
         <!-- Use the Alloy.Globals.Facebook namespace to create a Facebook module login button -->
       <Button backgroundColor="orange" id="button" onClick="doClick" title="Facebook Login"
            top="50" width="200" height="50" />
    </Window>
</Alloy>

    For more info, please visit Facebook Alloy
