how to close a nav group

Hello,

I have a navgroup that has windows for when the user is not logged in to my app.

Once the user logs in I remove that navgroup, and create a new navgroup that has the windows for when the user is logged on.

When I remove a navgroup, it seems to still exist, it is just not displayed any more.

Is there some way to destroy/close a navgroup?
