  • Hi Vijay,

    We won't get it for free. You have to create a View and inside view you have to add as a child elements what ever you needed. That's how we can achieve the requirement.

    If you look in to documentation http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.AlertDialog it says you can able to have a TextField in an AlertDialog() but to have your create your custom view if you needed one. The iOS 5 and later allow the user to enter plain text. For Ex.

    Ti.UI.setBackgroundColor('white');
var win = Ti.UI.createWindow({
  title: 'Click window to test'
});
win.addEventListener('click', function(e){
  var dialog = Ti.UI.createAlertDialog({
    title: 'Enter text',
    style: Ti.UI.iPhone.AlertDialogStyle.PLAIN_TEXT_INPUT,
    buttonNames: ['OK']
  });
  dialog.addEventListener('click', function(e){
    Ti.API.info('e.text: ' + e.text);
  });
  dialog.show();
});
win.open();
