    To use your example you would turn celebrity.js into a commonjs module. ie:

    celebrity.js

    function Celebrity(_args) {
  var self = Ti.UI.createWindow();

  return self;
}
module.exports = Celebrity;

    then back where you are using celebrity.js as a url in the window property, you'll change the code to this:

    var celebrity = require('celebrity');
var newWindow = new celebrity();
celebrity.open();

    And thats the basics of how to refactor that code.
