Rapid click + button

Hello everybody,

I have a problem, i add the event on a window to open a window but before to open the windo the iphone need to launch a function.

This is ok.

The probleme is that if i click 2 times on the button (added on the navbar), the windows is open 2 times.

how can i block this ?

    Add a flag variable if the user click on the button set it to true and if the user click the second time check for the flag and do nothing

    var flag = false;
......
......
clickEvent{
    if (flag){
        flag = true;
        execute funtion
    }
}
