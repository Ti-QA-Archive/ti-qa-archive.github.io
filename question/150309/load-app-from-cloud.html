    All the applications you create using Titanium Studio will reside on this machine only. There is absolutely no cloud interaction with your source code. Titanium cloud is actually used to develop applications that will make use of those same cloud services.(for messaging, notifications or other) That explains why you see it in your My apps page since every application has it's own cloud ID.

    If you want to share your source code between your different workstations, you will have to rely on a source control management system like Git, Mercurial or SVN.

    Hope this made it clearer for you.
