Modal window - how to put the back button on the navbar..or change the color..

Is it possible to put the back button (the one shaped as an arrow to the left) or change the color of the navbar? in the case of a modal window?

    Back-Buttons can not be added to modal-windows because the native UI of Apple does not allow it in this context (only in Tabs and NavigationWindows). Here is a quick example for implementing modal windows:

    Create the window

    var win = Ti.UI.createWindow({
    title : 'My Modal Window',
    backgroundColor : '#fff',
    barColor : 'green',
});

    Create the Button and the event

    var closeBtn = Ti.UI.createButton({title: 'Close'});

closeBtn.addEventListener("click", function() {
    win.close();
});
win.setRightNavButton(closeBtn);

    Open the window

    win.open({
    modal:true
});
