  • Well, the problem was that although the Android SDK was installed, the javac path was not added into Windows Environment Variables.

    So, a quick 1-2-3 guide would be:

    1. Install Java SDK (1.6, not later!)

    2. Ensure that javac is added into windows environment (right-click "Computer" then "Advanced System Settings", then "Environment Variables", then edit variable "PATH" and append into the list the value "C:\Program Files (x86)\Java\jdk1.6.0_30\bin", or whatever your path to java bin is!

    3. Install Android SDK and follow all documentation of appcelerator on how to configure a android under titanium.

    It's one of the most important how-to's that appcelerator.com should have on the site. Even with a video presentation. If wannabies appcelerator developers cannot figure how to do this they will just consider appcelerator as crap and proceed to an alternative (like as I was ready to do to be honest).
