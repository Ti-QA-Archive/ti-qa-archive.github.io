Log SOAP response? I just want to see what it is

I'm using the SOAP example in Kitchen Sink and seem to be getting a response from a SOAP web service, however, since I don't know much about what to expect in the response, I don't know yet what parts I'll need from it.

So, is there some way to just dump the response to the Appcelerator console?

I'm currently getting a successful .evaluate() like this:

    Not in a human readable format. IMHO you're better off trying to use a client application of some kind to test your web service and inspect the contents. I use SOAPClient on Mac with decent results. You could also manually POST a SOAP envelope using HTTPClient, available in the Mac App Store.

    You might also be able to use apigee.com's console tool, though I've never tried it for SOAP web services.
