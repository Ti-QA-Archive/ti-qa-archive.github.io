Global Scope

I have the following code:

//Add Share to Social Media Window
    var share_win = new ShareWindow('Share');
    //Set up the Application wide variables
    Ti.App.theShare = share_win;

Now if I move the second line under other variable declarations, this global variable stops working in other files. Only seems to work if I set it here in the code. Is this a BUG or am I doing something wrong.

  • Actually, app level global variables are bad, and shouldn't be used unless you have a very very very big requirement for it…

    Instead, you should pass the values and use event handling to control the app flow.

    Have a look here … https://wiki.appcelerator.org/display/guides/Best+Practices

    But if you do need global variables there is a solution here. ..
