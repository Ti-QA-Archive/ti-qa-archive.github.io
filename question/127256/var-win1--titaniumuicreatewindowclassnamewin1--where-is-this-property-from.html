var win1 = Titanium.UI.createWindow({className:'win1'}); -where is this property from

I can't find className in the documenation. what is it about? the documentation for Ti.Ui.Window is here http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.UI.Window-object.html

    You usually set a className property on a TableViewRow to indicate that the structural layout is the same on each TableViewRow with the same className. iOS uses this hint to speed up the rendering performance.
