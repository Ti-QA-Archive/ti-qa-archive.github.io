Use camera in mobile web app (on iOS)??

Hi

I have not yet done any mobile web apps in Titanium - but I would like to keep all my mobile development within this framework as it really ROCKS :-)

I have been asked if it is possible to create a mobile web app that uses the camera? So you can get access to the camera/photos on the device.

My question is then: Can this be done when building a mobile web app using Titanium Studio?

I have tried just creating a simple two tab mobile app for target: Mobile web - and added a simple example using the documentation - but that throws an error:

TypeError: undefined is not a function (evaluating 'Titanium.Media.showCamera')...

I know I have only tried the code in a standard browser and the iOS simulator (as I can only see that I could test it from a device by deploying it to a server somewhere - just referencing the IP of my Mac and port 8020 does not work).

Before going too much into deployment I would really like to know if this is at all possible? Is the "real" cause for the above error message that I cannot use a browser or simulator to view the web page? I have searched for examples and not really found anything. So I would appreciate any experiences/feedback from you ;-)

I am on newest Studio (build: 3.4.1.201410191559), Alloy (1.5.1) and OS X (10.10).

Thanks in advance!
