    winOrderItemDetail.add(tableItems);

what i want to do is loop through the tableItems.data and check for what record the radiobutton is selected so i know what item the enduser wants (name) … but how?

  • If you have only one section in your table view than you can get all rows from the table view using,

    tableItems.data[0].rows

    This will return array of rows,
    You can traverse through that array some thing like this,

    var dataRows = tableItems.data[0].rows;
for (var i=0;i<dataRows.lenght;i++){
    var rowOjb = dataRows[i]; // this is your single row's object, Now you can play with this object. I mean you can get all properties and child of this row object and you can calculate what you want
}
