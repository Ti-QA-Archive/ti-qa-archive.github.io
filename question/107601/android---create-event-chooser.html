Android - Create Event Chooser

More than a question this is an answer to a question I've looked up on the API documentation and elsewhere on the Web, with no luck, and ultimately found on the Appcelerator Mobile source.

If you need to have some action done by another Android app, but there are many that could handle your action, you can use the Intent chooser like this:

var intent = Ti.Android.createIntent({
   action: Ti.Android.ACTION_VIEW,
   data: "your Action's URI"
});

var chooserIntent = Ti.Android.createIntentChooser(
   intent, "Your Title"
);

Ti.Android.currentActivity.startActivity(chooserIntent);
