  • That looks a bit tricky to do. Here's what I would try:

    • create the tab bar exactly how you want it apart from the middle action button
    • for the action button, create a view element with bottom: 0, it should auto-center and sit on top of the tab bar (you may need to specify a higher zIndex to make it clickable), specify its height & width, and see what you can accomplish with that; to get the arch in the 2nd image you may need to set a custom background image, which you may do the same for all three images and just swap what background image you use on a tab 'focus' event

    The reason I would shy away from building your own tab bar is because doing so would require you figure out how to highlight/dehighlight the tab's images and text, ON TOP OF having to figure out how to accomplish what you want with the action button. So, it will save you time in my estimation.

  • Add Button to Tab Group Toolbar/Navbar

    The code:

    var btnEdit = Ti.UI.createButton({
    systemButton: Ti.UI.iPhone.SystemButton.EDIT
});
btnEdit.addEventListener('click',function(e){
    // button was clicked
});
tabWin.rightNavButton = btnEdit;

    You can also add a left button by doing:

    tabWin.leftNavButton = btnEdit;

    Reading these two questions: one and two helped me figure this out.

    Note that this works for iPhones, not just iPads (haven't tested on Android). All the button options are listed here. Also, note that you can do similar things with the icons on the tab bar using these button styles.
