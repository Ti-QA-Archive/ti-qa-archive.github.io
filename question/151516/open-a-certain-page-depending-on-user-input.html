Open a certain page depending on user input

Hi!
I'm working on Ti SDK 3.1.0.GA, trying to code a app for iOS. Using a iOS sim on OSX 10.8.3.

Problem: I have a textfield, and it has a type of num pad (so only numbers are allowed) and I want to open a window based on user input. As in if I input '1' –> go to win1. '2' –> win2

I have created the var txtfield = Ti.UI.createTextField and then I have a button created and an eventlistener.
I have also created a test window, var win1.

In the eventlistener I have: navGroup.open(win+txtfield.value); but it's not working, I get an error saying missing variable win. What do I have to put inside the () to get it to work?

  • Accepted Answer

    You should put your windows into a dict, and use that:

    myWindows = {
  'win1':win1,
  'win2':win2,
}

...

navGroup.open(myWindows['win'+txtfield.value]);
