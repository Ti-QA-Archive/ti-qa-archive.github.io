setBackgroundImage won't work .. any ideas?

I am setting at my app.js, first line, the following background for my app

Titanium.UI.setBackgroundImage(&#39;&#x2F;iphone&#x2F;Default.png&#39;)

It's the same image which I use for the splash. However I only get a black background .. any ideas why this happens?

    • The iphone directory is an override path based on platform. So, when you specify &#x2F;Default.png and you are running on an iPhone, your app will first look in the /iphone path for the file and if it doesn't exist there it will look under the root of the Resources directory. I believe this is true for any files you reference in your code (including CommonJS modules).
