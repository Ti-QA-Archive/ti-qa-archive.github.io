  • Hello,

    The quick answer, could be use a webview inside the app, and just open your website. But that won't solve the resize issues.

    The path to go from your website to a mobile app requires more work. Basically:

    1. You need to create webservices that generates or provides the content of your webpage.

    2. Create the UI/UX using Titanium components.

    3. Create code in javascript that glues the webservices with your UI.

    This in effect can take some time, it depends on the information architecture and implementation of your webpage.

    Best,

    Mauro

  • No, you can't use html5 code as is. Though it speaks JS, Ti is ultimately native.

    And you'll pretty much need to create a new interface from scratch, and set your server up so it spits out raw data where your app can use.

    I'm not sure on the Android side of things, but you can deploy a HTML5 app in iOS with some extra assets and meta tags.
