    Not sure why this fixed my problem but I ended up creating an empty "Single App Application" from the dashboard. Built it using 2.1.1, which worked, then built it using 2.1.3, which also worked. Switched back to my project with the problem and built it with 2.1.3 which now worked and synced properly.
