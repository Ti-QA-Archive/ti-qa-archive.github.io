  • Thank you for your time!

    Indeed, in the process of device orientation change has become of incorrect determined Titanium.Platform.displayCaps.platformWidth/Height. This caused an error.

    I solved the problem a little bit different way.
    I wrote an extra class, which stores the width and height of the devices.

    For all subsequent operations, I use these global variables. Thus, by changing the orientation I do not need to take Titanium.Platform.displayCaps.platformWidth/Height.

    function DisplayController(){
    var self = {}    

    self.setDisplaySizes = function(){        
        self.displayWidth = Titanium.Platform.displayCaps.platformWidth;
        self.displayHeight = Titanium.Platform.displayCaps.platformHeight;        
    }

    self.getDeviceOrientation = function(){
        return self.displayWidth>self.displayHeight ? 'landscape' : 'portrait'
    }

    self.setDisplaySizes();
    return self
}
