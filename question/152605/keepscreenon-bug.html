  • Hello,

    I tested with this code:

    var win = Ti.UI.createWindow();
var view = Ti.UI.createView({
    backgoundColor:'red',
    keepScreenOn:true,
    top:0,
    left:0,
    height: Titanium.UI.FILL, 
    weigth: Titanium.UI.FILL,
});
win.add(view);
win.open();

    and it's working like a charm. Do you have any example? I'm using 3.1.0.GA.
