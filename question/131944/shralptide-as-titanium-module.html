  • I found a solution:

    The points of max an min water comes from a stored list. It need 50 kB / a.

    I calculate the curves with this code:

    api.getTideValues = function(options) {
        var y1 = options.y1;
        var y2 = options.y2;
        var steps = options.steps;
        var amp = Math.abs(y2 - y1) / 2;
        var values = [];
        for(var i = 0; i < steps; i++) {
            var phi = i / steps * Math.PI;
            var value = Math.cos(phi) * amp;
            if(y2 > y1) {// steigend
                values.push(-value + amp);
            } else {
                values.push( value +amp);
            }
        }
        return values;
    };

    For displaying I'am using highcharts in a webview.
