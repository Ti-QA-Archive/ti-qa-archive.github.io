  • Per a recommendation from Danny Pham in the comments of the other answer, I dug into the source. Danny's recommendations were great, but turns out the SDK had all but implemented the event I needed, which was dragstart. So, in the SDK at:

    Titanium&#x2F;mobilesdk&#x2F;osx&#x2F;3.3.0.GA&#x2F;iphone&#x2F;Classes&#x2F;TiUIListView.m

    …I changed:

    - (void)scrollViewWillBeginDragging:(UIScrollView *)scrollView
{
  //Events - None (maybe dragstart later)
}

    …to…

    - (void)scrollViewWillBeginDragging:(UIScrollView *)scrollView
{
  if (![self.proxy _hasListeners:@"dragstart"]) {
    return;
  }
  [self.proxy fireEvent:@"dragstart" withObject:nil withSource:self.proxy propagate:NO reportSuccess:NO errorCode:0 message:nil];
}

    Then in my Alloy code, I just bound to the onDragstart event on the ListView.

      <ListView id="contacts" onItemclick="chooseContact" onDragstart="blurContactSearch">

    You get the gist.
