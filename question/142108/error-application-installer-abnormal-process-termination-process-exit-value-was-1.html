  • For all it's worth, I changed the value of

    <tool-api-level>15</tool-api-level>

    to

    <tool-api-level>8</tool-api-level>

    in tiapp.xml, since I was using Android 2.2, and it solved the appearance of this error for me. A colleague, while developing on OSX, ran into a similar problem when XCode upgraded silently.
