    Can you try solutions here: http://developer.appcelerator.com/question/121993/updating-titanium-sdks-has-encountered-a-problem

    If not, go to http://builds.appcelerator.com.s3.amazonaws.com/, download the SDK you need (20120915120319), then drop the folder in mobilesdk/osx into the following location:

    ~/Library/Application Support/Titanium/mobilesdk/osx
