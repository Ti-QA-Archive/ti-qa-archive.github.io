Titanium Studio 3.3.0 - installation problems with Java 7

Hi,
I try to run the stable version of Titanium Studio 3.3.0 for OSX 10.9.
According to http://docs.appcelerator.com/titanium/3.0/#!/guide/Installing_Oracle_JDK, I have Oracle JDK 1.7.0_65 installed, which is supported. But TS fails to start with either Java 7 and Java 8.
The only solution is installing Apple Java 6, which is EOL and known for its security flaws.
Error is "… libjvm.dylib does not contain the JNI_CreateJavaVM symbol." and seems well known from earlier versions. console says: "/Library/Java/JavaVirtualMachines/jdk1.7.0_65.jdk/Contents/MacOS/libjli.dylib: mach-o, but wrong architecture".
Any help is appreciated
