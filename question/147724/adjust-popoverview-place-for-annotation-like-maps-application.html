Adjust Popoverview Place for annotation like Maps application.

Hello Friends,

I am currently using popover View for showing detail of map annotation.

problem is that my popover is of fixed size so when my annotation is near of any ends of device than half popover cuts down.

i need to adjust it like Maps application.

and also want to hide annotation view with animate like Maps Does.

Detail :

1.Mobile SDK : 1.8.2

2.Platform : IPad

My testing Code is

var win = Ti.UI.createWindow(),
annotationLeftView = Ti.UI.createView({
    backgroundColor:'red',
    height:30,
    width:30
}),
popover = Ti.UI.iPad.createPopover({
    title:'I am a popover',
    height:100,
    width:200
}),
annotation = Ti.Map.createAnnotation({
    latitude:37.33170,
    leftView:annotationLeftView,
    longitude:-122.030733,
    rightButton:Titanium.UI.iPhone.SystemButton.DISCLOSURE,
    subtitle:'I am the subtitle',
    title:'I am the title'
}),
mapView = Ti.Map.createView({
    annotations:[annotation],
    mapType:Ti.Map.STANDARD_TYPE,
    region: {
        latitude:37.33168900,
        latitudeDelta:0.001,
        longitude:-122.03073100,
        longitudeDelta:0.001
    }
});
mapView.addEventListener('click', function(e){
    if(e.clicksource == 'rightButton'){
        var leftView = e.annotation.leftView;
        popover.show({view:leftView});
    }
});
win.add(mapView);
win.open();

Any Help would be appreciated…Thanks !

      i also got the solution to animate popover…

      when you create a popover set height to 1 or something less than actual height of popover then when you show popover at that time set it's actual height height so it animates automatically without using animate method of view.

      just follow the code below

      var  popover = Ti.UI.iPad.createPopover({
    title : 'I am a popover',
    height : 10,
    width : 200
});

popover.show({
    view : v
});
popover.height = 100;
