Date picker size on Androïd

Is there any way to increase the size of a date picker in Androïd ??? It's very small and I really want something bigger…. I try with a transformation… it's working but the touch region isn't following the picker…. Anyone has a solution ??

  • Hi Francois,

    There is a bug.You must be using useSpinner:true while creating picker. Just uncomment this line, you will see the appropriate effect on picker in Android.
    JIRA ticket is already been raised at https://jira.appcelerator.org/browse/TIMOB-11567 for the same issue.

    Thanks.
