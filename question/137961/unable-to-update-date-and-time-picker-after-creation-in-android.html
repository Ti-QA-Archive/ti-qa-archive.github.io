Unable to update Date and Time Picker after creation in Android

I am having problems trying to update the date and time pickers after creation.

I have a date picker and time picker with few buttons displayed on a screen. Each button correspond to a different date/time view. After I create the picker, I tried to update the value of the date or time picker setting the value but it doesn't cause the picker update.

e.g.

timePicker = Ti.UI.createPicker({
      top: 5,
      useSpinner: true,
      type: Titanium.UI.PICKER_TYPE_TIME,
      minuteInterval: 5
});

and when a button is pressed, I use the code but the picker display doesn't change.

timePicker.value = button1Date;

This happens with both the date picker and time picker in Android. I didn't have any problems in iOS with the iOS only setValue function. i.e. timePicker.setrValue(button1Date);
