Hi, in my android app i have a tableview with 20 rows i need to add a contact little image to every row, how can i do? thanks

    Use leftImage Property in the table View…

    Example: var data = [
    {title:'Row 1', leftImage: 'Image'}, //To add Image in the left Side…
    {title:'Row 2', rightImage: 'Image'}, // To add Image in the Right Side….
    {title:'Row 3', backgroundColor: '#fdd'}
    ];

  • Hello Raffaele,

    See this…

    
var win = Ti.UI.createWindow({
    backgroundColor : '#fff'
});

var tableData = [];

for (var i = 1; i < 20; i++) {
    var row = Ti.UI.createTableViewRow({
        height : 40
    });

    var lbl = Ti.UI.createLabel({
        text : 'Contact ' + i,
        left : 5,
        width : 250
    });

    var contactView = Titanium.UI.createImageView({
        right : 10,
        width : 30,
        height : 30,
        image : 'image.png'
    });
    row.add(lbl);
    row.add(contactView);
    tableData.push(row);
}

var tableView = Ti.UI.createTableView({
    data : tableData
});

win.add(tableView);

win.open();
