Hide Action Bar

Hi:

i need to hide the action bar in my app.

i test this:

index.js

    if (Ti.Platform.name=="android"){
        theActionBar = $.index.activity.actionBar; 
        theActionBar.hide();        
    }

Action bar is hidden but the window open with it and then hide. Its wrong.

Now i create the file custom_theme.xml:

<?xml version="1.0" encoding="utf-8"?>
<resources>
    <style name="Theme.NoActionBar" parent="@style/Theme.AppCompat">
        <item name="android:windowActionBar">false</item>
        <item name="android:windowNoTitle">true</item>
    </style>
</resources>

And change Tiapp.xml

    <android xmlns:android="http://schemas.android.com/apk/res/android">
        <manifest>
            <application android:debuggable="false"
                android:icon="@drawable/appicon" android:theme="@style/Theme.AppCompat">
                <activity
                    android:configChanges="keyboardHidden|orientation|screenSize"
                    android:screenOrientation="nosensor" android:theme="@style/Theme.Titanium">
                    <intent-filter>
                        <action android:name="android.intent.action.MAIN"/>
                        <category android:name="android.intent.category.LAUNCHER"/>
                    </intent-filter>
                </activity>
                <activity
                    android:configChanges="keyboardHidden|orientation|screenSize"
                    android:name="org.appcelerator.titanium.TiActivity" android:screenOrientation="nosensor"/>
                <activity
                    android:configChanges="keyboardHidden|orientation|screenSize"
                    android:name="org.appcelerator.titanium.TiTranslucentActivity"
                    android:screenOrientation="nosensor" android:theme="@style/Theme.AppCompat.Translucent"/>
                <activity android:configChanges="screenSize"
                    android:name="ti.modules.titanium.ui.android.TiPreferencesActivity" android:screenOrientation="nosensor"/>

            </application>
               <application android:theme="@style/Theme.NoActionBar"/>
                <!-- Need to specify at least API level 11 for Titanium SDK 3.2.x and prior -->
                <uses-sdk android:minSdkVersion="14" android:targetSdkVersion="19"/>
        </manifest>
    </android>

The file custom_theme.xml is in my styles folder (with index.tss).

But if i run the app:

[ERROR] :  Failed to package application:
[ERROR] :  
[ERROR] :  /Users/carlos/Documents/Titanium_Studio_Workspace_bueno/Termolink Wifi/build/android/AndroidManifest.xml:4: error: Error: No resource found that matches the given name (at 'theme' with value '@style/Theme.NoActionBar').
[ERROR] Application Installer abnormal process termination. Process exit value was 1

Where are i need to put the file custom_theme.xml???

Thanks.
