Add Event Listener Not Working (HUM)

I am posting this not so much as to ask why although it might be nice but more to let know what I have come across.

I have an app, with a unique menu control system. This has a title bar at the top, a screen view area and a menu bar at the bottom.

I had a need to have a popup menu option. This I coded as a view with the menu options included in this and added an event listener to each button.

I called the popup menu from the main menu bar, after building it all. And the popup menu is displayed using a - on the top to display it in the main screen area. But it is actually part of the bottom menu bar.

OK so the issue. Only the event listener which was displayed in the bottom menu bar itself worked, the other which were displayed over the main screen area did not work.

I solved it by recoding the popup menu to display on the main screen not the menu bar and all the event listeners worked as they should.

Oh yes I had set zIndex as well but that didn't resolve the issue, and the popup menu was created and displayed correctly.

I really hope I have explained this ok, and hopefully somebody else may not spend a few hours trying to work it out.
