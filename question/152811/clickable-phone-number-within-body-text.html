Clickable phone number within body text

I have a body of text that I'm pulling from ACS. Within the text could be phone numbers which I'd like to be clickable. Android sees the phone numbers and makes them clickable but iOS does not.

How can I have iOS detect phone numbers within body text and make those numbers clickable to place the call. Also, the ACS data is updated by the client and so the tech level is low, so wrapping the number in a tag would not be an option.

    You should be able to put the text into a TextArea and set the autoLink property to true. You could set the editable property to be false so that the textarea can't be typed in.
