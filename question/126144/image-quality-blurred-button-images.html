Image quality, blurred button images

Hello there,

Today I was making an custom button and when I compiled it on ios simulator I see that my custom button image is a bit blurry compared to what I see on the simulator.
What would cause this problem? 

button = Ti.Ui.createButton({
    width: 85,
    height: 85,
    image: '/images/button.png'
});

The width and height is the same as the image width and height so that is why I can't seems to know what the cause is. I have searched on the Q&A and still can't find any answer to it.

  • Try using backgroundImage instead of image and set the backgroundColor to transparent

    button = Ti.Ui.createButton({
    width: 85,
    height: 85,
    backgroundColor: 'transparent',
    backgroundImage: '/images/button.png'
});

    this works for me
