Check for Titanium SDK Updates - FAIL

Recently installed Titanium Studio and successfully entered proxy info in order to login. Studio immediately begins attempt of download of SDK which fails with error message (at bottom).

It attempts to connect to these two URLs:
http://api.appcelerator.net/p/v1/release-download?token=b4pNB8S6
http://api.appcelerator.net/p/v1/release-download?token=f4tzN4v3

I manually entered both URLs into my browser and they downloaded "sdk-1.1.0.zip" and "mobilesdk-1.7.5-win32.zip".

Clearly, it is not an network access issue. So I assume it is some configuration in Titanium Studio. Yet I can see that the "Window -> Preferences -> General -> Network Connections" contains my proxy info and, of course, it allowed me to login at the start.

  • I had the same issue and was able to resolve this issue by doing the following:

    1) "Window -> Preferences -> General -> Network Conections" under the "Proxy entries" section set the "Host" value for the "Schema" "HTTP" with the same value that should be there already for "HTTPS". The "HTTPS" value should have been set from when you configured your proxy at the logon screen.

    2) "Window -> Preferences -> General -> Network Conections" under the "Proxy bypass" section click the "Add Host…" button and entered your PC's IP address. On a Windows PC you can find this value out by opening a command prompt from your task bar clicking "Start -> All Progams -> Accessories -> Command Prompt" and entering the command "ipconfig". The IP address should be listed for the attribute "IPv4 Address….:".
