  • I'm facing the same issue with android, without removing admobview on window close, admob event doesn't destroy, so it's executing multiple times. For some reason, i can't get admob view on a window, so i can't execute win.remove(win.children….). Instead, one can add admob to a view, then add view to the window, there u can execute view.remove(view.children[0]). So, i managed to remove admob that way.
