    • Of course, the new Twitter API 1.1 REQUIRES that you obtain authentication credentials before any of the APIs will work.

      Basically you need the equivalent of a username and password that is used to enable your access to the API and manage your usage - this way if you end up flooding the API with calls twitter can limit your access by reducing your allowances.

      The allowances are generous so you should be good.
