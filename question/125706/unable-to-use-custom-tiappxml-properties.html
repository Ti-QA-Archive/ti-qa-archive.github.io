I am not getting any runtime errors, just had no luck retrieving any values.
Tried using (Ti.App.Properties.getString("app_owner", 'error'); with no luck.
Tried iterating the Titanium.App.Properties.listProperties(); with no luck.

Here is my custom property…
<property name="app_owner" type="string">"socaltechstaff"</property>
Has anyone been able to store properties in the tiapp.xml file?
