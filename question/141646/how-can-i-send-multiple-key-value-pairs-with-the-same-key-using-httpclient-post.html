How can I send multiple key-value pairs with the same key using HTTPClient POST?

For standard web-browser forms its possible to post the same key with multiple values.

Rails supports a syntax for arrays in a query string: "key1[]=val1&key1[]=val2&key1[]=val3"

I am using HTTPClient.send like this:

client.send({ "key1": "val1", "key1": "val2", "key1": "val3" });

But the param above is a js object that does not support more than one of the same key.

Any ideas about the best (neatest) way to solve this?
