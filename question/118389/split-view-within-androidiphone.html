    Ipad's Split Window

    and for Android tablet, you will need to:

    • Detect is a tablet (beware, some Samsung S3 could be considered as tablet because of the pixel density and resolution, but you should still consider it as phone).
    • Create a window with two views (used as containers) and emulate the split window. For android, AFAIK, there is no native split window component.

  • If you want such a UI I have successfully used two windows with appropriate widths. Then set the "left" window to left:0 and the right window to right:0. You will need to do some width calculation on orientation change. Something like

    Ti.Gesture.addEventListener('orientationchange',function(e) {
        self.width = Ti.Platform.displayCaps.platformWidth-(theme.leftWiniPadWidth+2);  
});

    Seems to work well.
