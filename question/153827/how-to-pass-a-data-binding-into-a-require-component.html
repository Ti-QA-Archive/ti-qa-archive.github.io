How to pass a data binding into a <Require> component?

Hi all, first time poster here.

I'm trying to write a reusable component into which I can pass a bound data element and having problems understanding if this is possible.

I'd like to call it something like:

<Alloy>
    <Model id="address" src="address" instance="true" />
    <Require id="first_name" src="myTextField" type="view" value="{address.first_name}"/>
    <Require id="last_name" src="myTextField" type="view" value="{address.last_name}"/>
</Alloy>

where I imagine the component myTextField would look something like (greatly simplified):

var args = arguments[0] || {};
var value = args.value || {};
var textField = Ti.UI.createTextField({value: value});

When I try this however, the component gets the literal string "{address.first_name}", not the binding to the address' first_name field.

I've also tried to omit the curly brackets, but no luck.

Can this be done? Should it? Is there a better approach than the one I'm trying?

Thanks for your help,
Larry G.
