  • Hi

    The is a built in method of JavaScript to achieve this already.

    var x = 9.567;
Ti.API.info(x.toFixed(0)); // = 10
Ti.API.info(x.toFixed(1)); // = 9.6
Ti.API.info(x.toFixed(2)); // = 9.57
Ti.API.info(x.toFixed(3)); // = 9.567
Ti.API.info(x.toFixed(4)); // = 9.5670

    The single parameter is the number of decimal places you require.
