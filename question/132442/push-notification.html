    Yes you have to write seprate code for both iphone and android it will be achive by C2DM. and iphone by urban Airship.

    For android it needs C2DM mechanism and for iphone it is implemented with urban airship. For implementing C2DM in android, it needs a C2DM module implementation.You can find this module from here

    https://github.com/IGx89/titanium-c2dm

    You can refer the sample code of this module in Example directory. If You will face difficulty in how to use this module then for steps follow this link

    http://developer.appcelerator.com/question/38451/c2dm-in-titanium

    For Iphone you need to implement by urbanairship.

    https://wiki.appcelerator.org/display/guides/Push+Notifications+with+Urban+Airship
