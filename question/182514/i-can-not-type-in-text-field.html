  • Hi Mohammed,

    If you are adding text field inside the table Then use the property softkeyboardonfocus like:

        var text = Ti.UI.createTextField({
        hintText : 'text',
        left : 10,
        top : 40,
        softKeyboardOnFocus : (Ti.UI.Android) ? Titanium.UI.Android.SOFT_KEYBOARD_SHOW_ON_FOCUS : '',
    });

    and also add the property to the window like windowSoftInputMode : (Ti.UI.Android) ? Ti.UI.Android.SOFT_INPUT_ADJUST_PAN : &#39;&#39;

    Hope this helps you.If any issue please let me know.
