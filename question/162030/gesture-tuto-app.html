  • Hi

    Firstly create your view that you want to display as your start up guide, this could be an ImageView or a complex View.

    I would then create a boolean which you could toggle and store in 'Ti.App.Properties' on first use, here is a link to the documentation Titanium.App.Properties-method-setBool

    And here is a more in depth description how-to-persist-complex-javascript-objects/

    In order to remove the View you could simply add a click event listener and add a method to remove the view from the window.

    Here is an example for classic titanium:

    var boolFirstUse = Ti.App.Properties.getBool('boolFirstUse') || true;
if (boolFirstUse) {
    win.add(view);
    view.addEventListener('click', function () {
        win.remove(view);
    });
    Ti.App.Properties.setBool('boolFirstUse', false);
}
