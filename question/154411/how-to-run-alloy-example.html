  • If I didn't get it wrong, I'd say the app directory is "only" the Alloy part of a Titanium project. So you should start by creating a new Alloy Project in Titanium Studio with File > New > Mobile Project and choose a template. You'll be able to run this on your device if everything is configured correctly. After that you can replace the app folder. But your project definitely needs a tiapp.xml to configure the application.

    thanks to Manuel Lehner! The steps for testing NFC Titanium 3.1 Example are:

    1- download ti.nfc module

    2- extract 1 sample project (EG: TagForeground)

    3- Open titanium studio and create new mobile project (File > New > Mobile Project) and select alloy project

    4- Overwrite app folder with the app folder of the sample project

    5- Install ti.nfc module for the project (Help > Install Mobile Module) and select the project

    6- Debug on an android nfc device
