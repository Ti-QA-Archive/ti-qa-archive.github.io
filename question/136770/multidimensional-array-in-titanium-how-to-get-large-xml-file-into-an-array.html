Multidimensional array in Titanium, how to get large xml file into an array?

Problem with arrays in Titanium here.

I have this in my xml response:

<section>
    <id>1</id>
    <category>
        <id>1</id>
        <item>
            <id>1</id>
            <title/>
            <description/>
        </item>
        <item>
            <id>2</id>
            <title/>
            <description/>
        </item>
    </category>
</section>
<section>
    <id>2</id>
    <category>
        <id>2</id>
        <item>
            <id>3</id>
            <title/>
            <description/>
        </item>
    </category>
</section>

And I need to get this type of array from the xml:

myArray[&#39;sectionid&#39;][&#39;categoryid&#39;][&#39;itemid&#39;][&#39;title&#39;]

I define myArray in app.js like this:

var myArray = [];

Then I add values into the array within a function inside example.js like this:

myArray[1][1][1][&#39;title&#39;] = &#39;sometitlehere&#39;;

And then I call a value inside that array within another function in example.js - I get error of undefined variable. Looks like Titanium doesn't support multidimensional arrays?

How do I deal with this, if I need to be able to access values by sectionid, categoryid, itemid and then the key? Please advice.
