    Hi

    You specify the API level in the tiapp.xml file in the root of your app folder.

    An example entry;

    <android xmlns:android="http://schemas.android.com/apk/res/android">
    <tool-api-level>14</tool-api-level>
    <manifest>
        <uses-sdk android:minSdkVersion="14" android:targetSdkVersion="14"/>
    </manifest>
</android>

    I wrote more on the subject this morning on a different answer - but the basic idea is the same;

    • Hide menu button on android?

    This lets you specify the API level you are after.

    • Just a thought - have you been making sure you have the latest Android SDKs installed? These are a separate install from Titanium and NOT managed by Appcelerator in any way.

      • Show the Dashboard tab in Studio
      • Select Get Started
      • Click Android SDK
      • Click Update Android SDK
      • The Android SDK Manager will now display
      • Choose the SDKs/API levels you need and choose Install

      This will take some time, so you might want to be very specific about your needs on the first go.

      If you are stuck at number 4 it may well mean you have simply not installed the more recent versions.
