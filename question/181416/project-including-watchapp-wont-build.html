  • Answering my own question.
    The bundle id you give youur WatchApp target must start with the application id of your Titanium App.
    So if you have set "nl.martijnkooij.myapp" in your Titanium project/tiapp.xml, than you must use "nl.martijnkooij.myapp" as your organization identifier when creating the watchapp target in Xcode.
