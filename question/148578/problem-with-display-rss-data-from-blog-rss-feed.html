  • Hello Chung,

    If I understand correctly, what you want to do is to be able to display the data under the label "Content", but in plain-text. If that's the case, your problem is that this data is in HTML. You will need to strip all tags by yourself.

    If you have control over your data source, I'd suggest to modify the feed to provide the data in plain-text along with the html version. If you don't, then perhaps use a function like this.

    Keep in mind that this level of data manipulation is taking place on the client-side, which means that it may affect the performance of your app, depending on how much data you're transforming.

    Best of luck,
