TabGroup id causes an error

Application worked fine until i specified id for TabGroup. 

<TabGroup id="tab_group">

Now the following error appears during app startup: 

message = "'undefined' is not an object (evaluating '$.index.open')";

What can be the reason of such behaviour?

    In this case the tabgroup is the top level container so its id is by default $.index
    just remove id=&quot;tab_group&quot; or try changing $.index with $.tab_group in the controller
