    Note that only the packaged versions of Titanium Developer are supported by Appcelerator. Are you going to modify the code? How do you need it to change? If you are not going to make any changes, why not use one of the compiled versions?

    Anyway, to compile Titanium projects, simply run scons from the root directory.

      1. use git clone to download the titanium_developer project repository
      2. import this project into your existing installation of Titanium Developer as a desktop project
      3. build and run
