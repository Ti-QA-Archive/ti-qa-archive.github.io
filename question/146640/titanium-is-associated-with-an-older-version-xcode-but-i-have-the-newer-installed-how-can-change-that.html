Titanium is associated with an older version xcode but I have the newer installed. How can change that?

Hi all.
I have a peculiar issue; apparently I have xcode in two versions 3.2 and 4.5 installed. Somehow Titanium is associated with the 3.2 version (but I have exported and submitted an app from Titanium with the 4.5 version….) Now Titanium will only work with the 3.2 which results in my work is not functioning anymore. How can I change this?
Thanks in advance for your help!
Best, Louise

    Hi, You can do switch using :

    sudo xcode-select -switch /Volumes/Xcode/Xcode.app/Contents/Developer/
