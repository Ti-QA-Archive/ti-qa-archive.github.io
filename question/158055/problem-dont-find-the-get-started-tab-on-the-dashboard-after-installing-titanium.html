    Hi

    When displayed this is obvious - when not displayed this is (can be) a pain to know what to do if you have never done it before.

    Locate Toolbar

    Will be at the top of the screen under the menu, should have several buttons including a red "house" with an "a" on it.

    • Found toolbar

      • Press the red house icon
      • The dashboard should appear
      • If "Get Started" is not shown in red, press the caption underneath your name (there are four of them)

    • Not found the toolbar

      • Menu > "Show Toolbar"
      • Start at the first step - "Found toolbar"

    This should get you where you need to be.
