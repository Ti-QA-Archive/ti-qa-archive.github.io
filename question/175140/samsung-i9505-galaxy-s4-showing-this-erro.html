  • You don't specify font style (not even font size) so every device will use its own built-in and that's why you see it differently in different devices.

    First, try to set height as height: Ti.UI.SIZE or even omit title parameter of tab and add a label on top of that, if it is possible.

    Also, try setting backgroundColor to your tab to see how much space this '30dp' covers in relation to the backgroundImage you are using for the tabs. I suspect there is plenty room yet.

    If these doesn't work maybe I would try to install and set same font everywhere.
