TableView Search Issue

Hey guys,

Firstly can you use a textfield rather than a search bar for filtering the tableview?

If not, is there a way of seperating the search bar from the tableview, as i do not want the table view directly below the search bar.

Any help would be amazing.

Thanks!

  • Thomas

    Unfortunately, it isn't possible to separate the searchBar from the tableview at the moment, because there is no way to filter the tableview rows unless the searchBar object is passed to the tableview using its search property. For the same reason, you can't use a standard textField either.

    I am assured that this functionality is on the core team's TODO list, and will be in a future release, but at the moment this release is not known.

    Sorry the news isn't ideal, but I hope it provides an answer, at least.

    Cheers
