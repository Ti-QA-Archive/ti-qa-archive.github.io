  • Hi

    I am not really sure what problem you are trying to solve as you have not provided any code to backup the question. However asking the length of a string does return back line breaks and spaces;

    var x = "One\n Two";
alert(x.length);

    This equals 8, for the following reason;

    One = 3
\n = 1
(space) = 1
Two = 3
// so 
3 + 1 + 1 + 3 = 8

    As \n is a special character sequence it actually only represents one character in a string.
