Horizontal Layout from right to left

I noticed horizontal layout only works left to right. Is it possible to align other way around?

    Hi

    Unfortunately the short answer is "no". However you can obtain right-to-left with a little work-around.

    This example is based on adding child views to a parent view - but can be adapted for most situations.

    var view = Ti.UI.createView({
    height: 300,
    width: Ti.UI.FILL
});
win.add(view);
var inner = Ti.UI.createView({
    height: Ti.UI.SIZE,
    layout: "horizontal",
    right: 0,
    width: Ti.UI.SIZE
});
view.add(inner);
var lbl1 = Ti.UI.createLabel({
    height: Ti.UI.SIZE,
    left: 0,
    text: 'One',
    width: Ti.UI.SIZE
});
inner.add(lbl1);
var lbl2 = Ti.UI.createLabel({
    height: Ti.UI.SIZE,
    left: 0,
    text: 'Two',
    width: Ti.UI.SIZE
});
inner.add(lbl2);

    You will notice the One Two are positioned on the right of the parent view, the order of them is intentionally wrong to point out you still need to reverse the order in which you add items.

    Keep in mind this is a work-around and not perfect in all situations.
