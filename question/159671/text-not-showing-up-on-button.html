Text not showing up on button

I am new to Titanium and JavaScript as whole and I am following some tutorials to learn how to code/use Titanium and JavaScript. I am having trouble getting text to show up on buttons I have created. The buttons respond to the listeners but there is no text showing up. Is there anything I am not including in my code that is causing this?
Side note: It is for an Android Device

Here is what my buttons look like: 

var AvailableJobsButton = Titanium.UI.createButton({
    text: "Available Jobs",
    top: "10%",
    height: "10%",
    width:"100%",
    font:{fontSize: "16"},
    id: "AvailableJobs", //custom property

});

    Hello Richard ,

    Button have no text property. Use title instead of text.

    Try this one : 

    var AvailableJobsButton = Titanium.UI.createButton({
    title: "Available Jobs",
    top: "10%",
    height: "20%",
    width:"100%",
    font:{fontSize: "16"},
    id: "AvailableJobs", //custom property
});
