Check which phone OS (Android or IOS) is being used

I need to determine which phone OS is being used so that I can use phone specific methods depending on which phone OS is being used. I cannot find documentation or information on how to do this. Right now, you are not actually able to build a full app that is both android and iphone native apps, unless they are very very basic, without being able to determine the OS. Any help is appreciated.

    Here is a short code sample that shows how it is done.

    var osname = Ti.Platform.osname,
    version = Ti.Platform.version,
    height = Ti.Platform.displayCaps.platformHeight,
    width = Ti.Platform.displayCaps.platformWidth;

//considering tablet to have one dimension over 900px - this is imperfect, so you should feel free to decide
//yourself what you consider a tablet form factor for android
var isTablet = osname === 'ipad' || (osname === 'android' && (width > 899 || height > 899));
