Using an event from a different Class

I'm new to Titanium and I am trying to create an App for Android. I was just wondering if it is possible to use an event from the Ti.Gestures class in my Ti.Window? Example code:

var window = Ti.UI.createWindow({
    barColor:'black',
    backgroundImage:'pong.png',
    title:'The Simple Game'
});

window.addEventListener('shake',function(e)
        {
        var win2 = Ti.UI.createWindow({
            barColor:'black',
            backgroundImage:'pong.png',
            title:'The Next Tab'
        });
        }

Thanks in advance for the help.

  • No you can only use the events that are listed on the Docs, but you can trigger events

    var window = Ti.UI.createWindow({
    barColor:'black',
    backgroundImage:'pong.png',
    title:'The Simple Game'
});
Ti.Gesture.addEventListener('shake',function(e){
    Titanium.UI.createAlertDialog({title:'Shake',message:'it worked!'}).show();
    var win2 = Ti.UI.createWindow({
            barColor:'black',
            backgroundImage:'pong.png',
            title:'The Next Tab'
        });
});
Ti.Gesture.fireEvent('shake', {});
