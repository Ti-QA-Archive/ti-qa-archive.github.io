Blur not hiding keyboard

In a tableview, I have two rows. Row 1 contains a textfield and row 2 contains a label.

When the user touches the first row, focus is set on the textfield:

customerName.focus();

When the user touches the second row, I remove focus with blur:

customerName.blur();

Focus is removed but the keyboard does not auto-hide.

I don't know if this is a bug or if there is something more that I need to be doing.

Thanks!
