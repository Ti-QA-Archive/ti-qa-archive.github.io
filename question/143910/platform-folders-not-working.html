    • OK, with the additional information and logs, I can now tell you exactly what is wrong.

      You are using one of the nightly builds of TiStudio (3.0.1.X), which makes use of the new Titanium CLI project which is still in development. The new CLI unfortunately has an issue where is sends inconsistent platform values to Alloy when an alloy project is run through TiStudio. This is why your platform specific folders work as expected from the command line, but fail when run through Studio.

      To resolve your issue, you simply need to use a version of TiStudio that is making use of Titanium's traditional CLI, not the new in-development one. To do that, just go to the TiStudio preview download site and download the appropriate RC candiate for your system. The current RC will be a 3.0.0.X version, and will work correctly with your platform folders.

      here's the link: http://preview.appcelerator.com/studio/

      Install the RC, start the 3.0.0.X version of Studio, and you should be good to go.
