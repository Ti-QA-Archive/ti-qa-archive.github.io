SystemButton and SystemButtonStyle

Hi all!

I'm trying to add a button into a "createToolbar" but I can't find a way to make it look like a Navbar button.

I know I can use SystemButton and SystemButtonStyle to style the button but, don't know why, none of the options available are giving me the style I want, in fact, they are giving me a complete different effect.

For example:
style:Ti.UI.iPhone.SystemButton.ADD
This returns a black "i" like for "information" on iPhone

style:Ti.UI.iPhone.SystemButton.CAMERA
This returns nothing…

…and so on.

Am I doing anything wrong here?? I just wanna create a button to add into a createToolbar like the ones we can set on rightNavButton and leftNavButton.

Any ideas?

    You should be using the systemButton property here, not the style property. See:

    docs.appcelerator.com/titanium/2.0/index.html#!/api/Titanium.UI.iPhone.SystemButton

    Also, see the "iOS Platform Notes" for Button, which describe the difference between style and systemButton:

    docs.appcelerator.com/titanium/2.0/index.html#!/api/Titanium.UI.Button

    Hope this helps!

      Also, after a few minutes looking to the documentation, I realised I was adding the buttons to the createToolbar

      I was using:
      myToolBar.add(myButton)

      The right way:

      var myToolBar = Ti.UI.createToolbar({
    items: [myButton1, myButton2, ...]
});
