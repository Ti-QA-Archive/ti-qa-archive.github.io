  • Hi

    First thing to note is that the size property returns the pixel size not the file size when saved. To determine this you will need to save the image to the file system first.

    Try these extra debug tests and report back;

    var successCallBack = function (e) {
    var image = e.media;
    Ti.API.info("original image size:" + image.size);
    Ti.API.info("original image dimensions: " + image.width + 'x' + image.height);
    var newBlob1 = image.imageAsResized(1600,1200);
    Ti.API.info("test 1 image size:" + newBlob1.size);
    Ti.API.info("test 1 image dimensions: " + newBlob1.width + 'x' + newBlob1.height);
    var newBlob2 = image.imageAsResized(160,120);
    Ti.API.info("test 2 image size:" + newBlob2.size);
    Ti.API.info("test 2 image dimensions: " + newBlob2.width + 'x' + newBlob2.height);
}

    What do you now see in the debug console?

    I am hoping these should show different results.
