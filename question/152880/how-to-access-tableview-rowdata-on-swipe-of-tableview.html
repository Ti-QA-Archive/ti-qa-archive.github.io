How to access tableview rowData on swipe of tableview?

Hi

I am not able to access tableviewRow data on tableview swipe event on iOS. but I can access tableviewRow data on tableview click event.

var win = Ti.UI.createWindow({
   backgroundColor:'white' ,
});

// Create a TableView.
var aTableView = Ti.UI.createTableView();

// Populate the TableView data.
var data = [
    {title:'Row 1', hasChild:true, color:'red', header:'First'},
    {title:'Row 2', hasDetail:true, color:'green'},
    {title:'Row 3', hasCheck:true, color:'blue', header:'Second'},
    {title:'Row 4', color:'orange'}
];
aTableView.setData(data);

// Listen for click events.
aTableView.addEventListener('click', function(e) {
    Ti.API.info('onClick RowData- '+JSON.stringify(e.rowData));
});
aTableView.addEventListener('swipe', function(e) {
    Ti.API.info('onSwipe RowData- '+JSON.stringify(e.rowData));
});//

// Add to the parent view.
win.add(aTableView);

win.open();

Output

[INFO] onSwipe RowData- undefined
[INFO] onClick RowData- {"horizontalWrap":true,"color":"orange","title":"Row 4"}

Titanium SDK Version: 3.0.2

Titanium Studio Version: 3.1.0.201303230404

XCode Version: 4.6

Thanks

    Hi

    The swipe event does not provide the row or rowData properties event data object. I know the documentation does say both are provided - but they are NOT available.

    They were available in previous versions of the SDK but not 3.1 or 3.0.2.

    Annoying.

    There is a way to work-around the problem using current SDKs, not ideal but it does work.

    You need to detect there was a swipe event and then use the touchend event to grab the data you need.

    var boolWasSwipe = false;
var objSwipe = {};
tbl.addEventListener('swipe', function (e) {
    boolWasSwipe = true;
    objSwipe = e;
    Ti.API.info('1. swipe/row', JSON.stringify(e.row));
    Ti.API.info('1. swipe/rowData', JSON.stringify(e.rowData));
});
tbl.addEventListener('touchend', function (e) {
    if (boolWasSwipe === true) {
        Ti.API.info('2. touchend/row', JSON.stringify(e.row));
        Ti.API.info('2. touchend/rowData', JSON.stringify(e.rowData));    
        Ti.API.info('2. swipe/object', JSON.stringify(objSwipe));
    }
    objSwipe = {};
    boolWasSwipe = false;
});
