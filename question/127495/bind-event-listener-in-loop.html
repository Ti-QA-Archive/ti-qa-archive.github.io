Bind Event Listener in Loop

I've just finished creating a meticulously generated grid of icons (imageViews) and now I need to be able to do something with them. In the loop I have this snippet:

  var icon = Ti.UI.createImageView({
    left: ( containerW - icoW ) / 2,
    height: icoH,
    image: icons[i].image,
    top: 0,
    width: icoW,
  });

  icon.addEventListener( 'click', function( e ) {
      alert( 'firing the click event' );
  });

  container.add( icon ); // container is a view created before icon
  thisRow.add( container ); // thisRow is also a view created earlier

I'm assuming that there's a timing issue or something similar happening, but I hope there's a way to get this done so I can click on these nicely arranged icons. Help.
