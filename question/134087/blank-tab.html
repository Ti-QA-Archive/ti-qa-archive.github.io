Blank Tab

I have this Tab group I have created.

//Create Tab Group
    var tabGroup = Titanium.UI.createTabGroup();

var home_tab = Titanium.UI.createTab({
        icon : path.R + 'images/tabs/home-tab-off.png',
        title : home_win.getTitle(),
        window : home_win.getWindow()
    });
    home_win.getWindow().containingTab = home_tab;

tabGroup.addTab(home_tab);

Now I have this button which opens another window on a tab page.

home_win.getButton(0).addEventListener('click', function(e) {
        home_win.getWindow().containingTab.open(town_win.getWindow());
    });

But inside that window I want to go back to the home window inside the tab on an event so tried this:

var town_header_image = town_win.getHeaderHome();
    town_header_image.addEventListener('click', function(e) {
        home_win.getWindow().containingTab.open(home_win.getWindow());
    });

This seems ok, but then I clicked the home tab a couple of times, the screen goes black.

Any way to avoid this issue?
