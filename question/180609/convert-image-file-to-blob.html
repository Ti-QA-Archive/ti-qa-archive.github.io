Convert image file to blob

I have an image in resources/images. I want to convert it to a blob and save it to applicationSupportDirectory. I have this code

      var image = Ti.Filesystem.getFile('/images/0.jpg');
     console.log(image.nativePath);
     var blob = image.read();
     console.log(blob.file);

the image part works fine, and the image.nativePath prints into the console, no problem.
image.read() is not returning anything, and so blob.file is throwing an exception.

  • I was able to fix this issue myself. I had to do

    Ti.Filesystem.getFile(Ti.Filesystem.resourceDirectory,&#39;&#x2F;images&#x2F;0.jpg&#39;);

    Everything worked after that.
