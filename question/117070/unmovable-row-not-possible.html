Unmovable Row not possible???

Hi together,

I try to create a tableview, where only the first row is unmovable. The following code in Titanium Mobile 1.6 don't work in the way it should - the first row is still movable after clicking on the Edit-Button. In another post I find this definition:
A row will be movable if

  1. ((row.movable==true) AND (table.editing == true)) OR
  2. ((row.movable!=false) AND (table.moving == true))

Could anyone help me or have another solution? 

var curWindow = Titanium.UI.currentWindow;

var data = [
    {title:"Row 1", editable:false, movable:false},
    {title:"Row 2", hasChild:true, header:''"},
    {title:"Row 3", hasChild:true}
];

var tableview = Titanium.UI.createTableView({
    data:data,
    style: Titanium.UI.iPhone.TableViewStyle.GROUPED,
    editable:true
});

var buttonEdit = Titanium.UI.createButton({
    systemButton:Titanium.UI.iPhone.SystemButton.EDIT
});

buttonEdit.addEventListener('click', function()
{
    tableview.moving = true;
});

curWindow.rightNavButton = buttonEdit;
curWindow.add(tableview);

Thanks for your help!
