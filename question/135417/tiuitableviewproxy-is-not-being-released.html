    • It looks like it was my fault. In my release method I was calling window.close() , but I also have NavigationController which has to be released and it couldn't release resources since I already closed the window. So I did by telling NavigationController only to close window and it released resources.

      Releasing of resources is even more quickly if you make your own release method which you call before closing and in it put you UI building code reverse: this.windowInstanceremove(this.tableView), this.windowInstance.remove(this.scrollView) etc.

      Assuming you are following the CommonJS pattern.
