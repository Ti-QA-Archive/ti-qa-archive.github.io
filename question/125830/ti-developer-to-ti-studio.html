    Actually Titanium Studio is eclipse, so installing titanium studio will install everything. This does include eclipse.

    Basically eclipse is the IDE and it has been customised and packaged as Titanium studio to work specifically the way it is required. But you still have the full functionality of eclipse so you can add all the tools like svn or ruby stuff if needed.

    Also although the developer tool will still be there I don,t think any more work is being done on it, they are purely focusing on studio.

    Hope this explains it.
