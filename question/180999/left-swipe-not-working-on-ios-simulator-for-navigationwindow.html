Left Swipe not working on iOS simulator for NavigationWindow

I have discovered a very strange behavior: the left swipe from the left edge of the iOS simulator when in a NavigationWindow does'n allow to go back to the parent window.
The strange thing is that it works fine on the device.

this a sample code to test it

var win = Ti.UI.createWindow({
    backgroundColor: "white"
});

var navWin = Ti.UI.iOS.createNavigationWindow({
    window: win
});


var win2 = Ti.UI.createWindow({
    backgroundColor: "green",
    title: "ciao"
});

var btn = Ti.UI.createButton({
    title: "ciao"
});

win.add(btn);
btn.addEventListener("click", function() {
    navWin.openWindow(win2);
});


navWin.open();

I am using TiStudio on Mac OS X with SDK 3.5.0.
