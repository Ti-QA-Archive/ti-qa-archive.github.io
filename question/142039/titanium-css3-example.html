  • Where did you read this? Titanium is NOT built on webkit. There is an API that allows you to use a WebView within your native application. This WebView is built on the browser installed on the device you are building for. That would be WebKit of iOS but a version of Chrome for Android devices. Since a webview is just a mini browser inside you app, you can use pretty much any tutorial you find online to use the properties that you want. A quick Google search will provide you with plenty of guides and tutorials for each css property.
