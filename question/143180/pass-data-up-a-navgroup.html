Pass data up a navGroup

I've got a simple three-level navgroup. The 3rd level window has a text field and a keyboard, and when return is pressed on the keyboard the window closes and I go back to the 2nd level window. All good so far…

TextField.addEventListener('return', function(e) {
    navGroup.close(self);
    });

When I close the 3rd level window, I also want to pass the value of the text field back up to the 2nd level window.

I thought I could add an event listener on the 2nd level window, to fire when the 3rd level window closes. But I'm not sure if this is the right thing to do, and I can't figure out how to do it in any case.

Any ideas greatly appreciated!

  • Accepted Answer

    If you have a controller for the navGroup it will be aware of all of the windows in the navGroup stack. Then you would not be passing window objects all through the application.

    the less elegant approach it to assign the navGroup as a property of the window(s) as you add them to the navGroup

    //load component dependencies
    var FirstView = require('ui/common/FirstView');

    backgroundColor:'#ffffff'
});

//construct UI
var firstView = new FirstView();
self.add(firstView);

return self;

    }

    module.exports = ApplicationWindow;
