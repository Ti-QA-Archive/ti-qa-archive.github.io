    • Daniel, I may be misunderstanding your problem, but the this bit of code appears to be attempting to open a remote resource (http://dacheezypuffs.webs.com/tabledata.xml) as a local file, which won't work:

      var file = Ti.Filesystem.getFile(Ti.Filesystem.resourcesDirectory,'http://dacheezypuffs.webs.com/tabledata.xml');

      You'll need to use Ti.Network.createHTTPClient() in order to retrieve a remote resource, see the Kitchen Sink example I linked previously. It is a good example to build from.

      Also, 'http://dacheezypuffs.webs.com/tabledata.xml' does not appear to render as valid XML, so you'll likely have trouble parsing it once delivered.
