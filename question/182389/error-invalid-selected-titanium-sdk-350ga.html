  • There are two solutions to this problem.

    1. Download the Ti SDK 3.5.0.GA from Appcelerator or

    2. Change the SDK version in tiapp.xml file below is the sample code of tiapp.xml that you need to replace it in your tiapp.xml

    
    <deployment-targets>
        <target device="android">true</target>
        <target device="blackberry">false</target>
        <target device="ipad">true</target>
        <target device="iphone">false</target>
        <target device="mobileweb">false</target>
    </deployment-targets>
    <sdk-version>3.5.0.GA</sdk-version>
    <plugins>
        <plugin version="1.0">ti.alloy</plugin>
    </plugins>

    Hope this helps you.
