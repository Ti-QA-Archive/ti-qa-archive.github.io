    Hi

    This method, in picturegallery.js;

    var pictureGallery = PictureGallery.createWindow({
    images: images,
    title: 'Pictures',
});

    Calls PictureGallery but you have not declared that object in this CommonJS file - this is also not declared as a global which you should not do - but either way not declared.

    Also you have a typo;

    module.exports - GalleryWindow;

    Should be

    module.exports = GalleryWindow;

    Note the equals sign.
