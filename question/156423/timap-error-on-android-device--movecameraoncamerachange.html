  • Hi

    Before you go any further, some words of caution.

    If this is a new app or you have yet to obtain an API key for this specific Android app - then you HAVE to move to the Google Maps v2 for Android, Ti.map is v1 and no longer supported.

    • Ti.Map
    • Google Maps v2 for Android guide.

    If you do have an existing api key for an existing appid, then please provide a snippet of code. Otherwise please review the links above as you have no option but to change to Google Maps v2 for Android.
