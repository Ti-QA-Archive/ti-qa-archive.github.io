using border-radius in .tss

I am trying to make a view into a circle. In css i would use

border-radius: '50%',

this doesn't seem to work in tss.

any idea what it should be?

also does anyone know of any good tss documentation. I cant seem to fine any.

    Hi
    If you check the documentation you will see that there is no hyphen used in the keywords - Titanium uses the JavaScript syntax of camel case. So border-radius become borderRadius.

    Three examples;

    borderRadius: '50%',
borderRadius: 15,
borderRadius: '50px',
