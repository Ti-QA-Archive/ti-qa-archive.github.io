Import Xcode Libraries into Appcelerator

Hi,
I have an app written in XCode but want to import it into Titanium. I have some custom libraries written into it too. Can you please suggest how I can import all the libraries and app features into a Titanium app?

  • You can't import your entire Xcode application into Titanium and magically make it a Titanium app (that would be kind cool though), however, you take libraries that you've created in ObjC and create native modules out of them. See the module development guide here: http://docs.appcelerator.com/titanium/2.1/index.html#!/guide/iOS_Module_Development_Guide

    For everything else, you'll have to build out the functionality using the Titanium Mobile SDK.
