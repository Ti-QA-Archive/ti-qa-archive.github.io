Pre-built database

I created a sqlite database that I want to distribute with my iPad app.

Where do I put the file and how do I connect to it? The Kitchen Sink example is using this:

var db = Titanium.Database.install('../testdb.db','quotes');

I can't find API docs that describes the install method, so I'm not sure what exactly it is doing.

I put my database file in the root the Resources folder, along with my js files attempted to open it with:

var db = Titanium.Database.open('myTestDB.sqlite');

But it fails with "Result of expression 'Titanium.Database' [undefined] is not an object.";

  • Yeah it forced the full rebuild so it deleted the last database. It didnt work at first cause you already created a database called myTestDB.sqlite. So when you tried to install a database with that same name it threw an exception. A full rebuild blew away the database you created when you called open. Had the same exact problem as you actually!

  • Jaime

    Pressing the "Save Changes" button on the "Run Emulator" screen, which you had to do to change the SDK version, touches the tiapp.xml file which forces a full rebuild. The full rebuild is what resolved the issue for you.
