Dynamically change Android handset menu option text

I want to change handset menu option text dynamically. Like I want to toggle text depending on the flag I've set. I am using 1.6.2 sdk and TI developer 1.2.2.

— asked July 13th 2011 by Abdul Rehman

2 Comments

  • Here is my code for changing menu option:

    var activity = Ti.Android.currentActivity;
var bmeDisplay = false;
activity.onCreateOptionsMenu = function(e) {
    mnuopt= menu.add({title : 'ShowMe'});
}
    mnuopt.addEventListener('click',function() {        
        if(bmeDisplay == true){
            bmeDisplay = false;
            this.title = 'ShowMe';
        }
        else{
            bmeDisplay = false;
            this.title = 'HideMe';
        }
    });

    It works perfectly on emulator but generates error on device:

    wrapped android.view.ViewRoot$CalledFromWrongThreadException:
    only the original thread that created a view hierarchy can touch its views.
