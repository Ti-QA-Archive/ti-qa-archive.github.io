Cross-platform video editing app

I'm building an online community for sharing a specific genre of video and really want to have an app that runs on both Android and iOS that allows basic video editing similar to Splice, ReelDirector and iMovie and then adds some more features.

Would that be possible in the Titanium framework, or will that have to be done specifically in each SDK using native code?

  • AFAIK, it will have to done specifically for each OS using native code. The Ti.Media module for iOS is built around AVFoundation. Its editing capabilities are extremely limited other than triming either end of a movie.

    For a project I had where I needed to edit videos, I created my own native module using objC utilizing classes in CoreMedia. It allows me to splice together arbitrary segment(s) of source movie(s) and export a new movie. I have not ported that project to android yet.
