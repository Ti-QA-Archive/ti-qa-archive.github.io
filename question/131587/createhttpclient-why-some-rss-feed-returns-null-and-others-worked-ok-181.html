createHTTPClient() Why some rss feed returns null and others worked ok (1.8.1)?

    I found that this.status is 301 - Moved Permanently - so i guess there're some redirect in your feed.

    Open feed url by browser i notice that it redirect to http:&#x2F;&#x2F;developer.appcelerator.com&#x2F;blog&#x2F;feed&#x2F;
    ( Added last / )

    It's very funny but below change would make it work :)

    client.open("GET", 'http://www.watchlearnknow.com/feed/');
