This seems to succeed but right at the end it fails while packaging the sdk into a .zip (this has worked before). Here is what I get:

  • Accepted Answer

    Are you sure you have all of the software requirements installed? Make sure you have current versions of the various python packages, you might need to update easy_update's cache before installing those.

    Do you need the jsca files? You can speed up builds (and maybe bypass the error you're seeing) by using 

    scons build_jsca=0
