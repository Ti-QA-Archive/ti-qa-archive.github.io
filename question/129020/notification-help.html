Notification Help

I'm about to wrap up an app, and the last part is scheduling notifications.

I have dates in my data (yyy-mm-dd), and all I want to do is have a button that says 'remind me', and have it take that date and schedule a notification. After it loads, I want it deleted.

I looked in the kitchensink and around the forums, and cant figure out it out. The Kitchen Sink example keeps crashing on me.

This is for iOS (and Android if possible, but if not oh well).

  • Hi Josh,

    I'm doing something similar in my code. This should work for iOS. Also see bg.js in KitchenSink.

    // First get the date from your data and create a date object
var date = new Date();
date.setFullYear(2011);
date.setMonth(12);
date.setDate(7);
date.setHours(12); // Pick what time of day you want the notification to show up
date.setMinutes(0);

// Somewhere you'll need an array to store notifications
var notifications = [];

// Create notification
var notif = Ti.App.iOS.scheduleLocalNotification({
  alertBody:"You've got mail.",
  alertAction:"View",
  sound:"pop.caf",
  date: date
});

notifications.push(notif);

    -Dylan

  • Try this. This schedules the notification for five seconds after the app launches. It worked for me.

    The W3schools JavaScript Date object reference might come in handy here for working with date objects.

    // First get the date from your data and create a date object
var date = new Date();

// Set date to 5 seconds from now
date.setSeconds(date.getSeconds() + 5); 

// Somewhere you'll need an array to store notifications
var notifications = [];

  alertAction:"View",
  sound:"pop.caf",
  date: date
});

notifications.push(notif);
