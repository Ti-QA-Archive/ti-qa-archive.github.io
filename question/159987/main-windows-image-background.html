How can I put an image in the main windows,

var win = Titanium.UI.createWindow({
    title : 'my Title',
    layout : 'vertical',
    backgroundImage:'background.png'

});

and put background.png in Resources/android/images/res-xxx

But in my emulator, I can only see the splash screen : default.png

How should I do it???

  • You would need to reference it as `'images/background.png' then.

    Everything in android gets moved up into Resources when built for android retaining the directory structure. density specific folders are pooled to pull in the matching assets.
