Looping through multiple textfields values

Hey

I have spend all night looking for a solution but it seems i got something wrong.

I have 20 textfields on multiple views. I have a button, which after I click, I want the values of those 20 textfields to be "".

so, i am not quite sure, whether i am doing this ok but:

var textFields = ["txtF1","txtF2","txtF3","txtF4", etc..];

and then

for (var i in textFields) {
    textFields[i].value = "";
}

but it does not work. The thing is that when i use textFieldsp[i] in alarm call, it returns the name of the field ok.

Could someone help me please?

Thanks

    It will return name of the field ok because you are inserting only name of your text field in the array. You should add object of text field not the name.

    var textFields = ["txtF1","txtF2","txtF3","txtF4", etc..];

    This line will create array of Strings not the text field objects. Because "txtF1" is string.

    So if your text fields name are txtF1, txtF2, txtF3 etc etc…

    than add object of create array of text filed objects. So in short you just have to remove QUOTES and it will create array of objects instead of strings.

    like this,

    var textFields = [txtF1,txtF2,txtF3, etc..];

    and than use your FOR LOOP, it should work now.
