Creating images for various android layouts

Can someone help provide information or simple guide on how to create simple image that could fit in various device layout (portrait, landscape for tablet and mobile). Am not referring to the icons and splash screen under the Resource/Android/Images but in creating new images in a new page inside a view that can fit different device layouts. Some sort of flexible responsive image to the user. Most of the tutorials available are for IOS and Android are for single layout.

  • Just to site an example: I created below 

    var view1 = Ti.UI.createView({
    top:0,
    width: 'auto',
    left:0
});

var backgroundImage1 = Ti.UI.createImageView({
    image:"images/image1.png",
    width: "auto",
    left: 0,
    top: 0
});

win.add(view1);
view1.add(backgroundImage1);

    How and where to tweak the code to adjust the width and height of the image when dealing with Tablet and mobile.
