What is the difference here?

So swiping on my view gui and setting the left property to '90%' works.

gui.addEventListener('swipe',function(event)
{
    if (event.direction == 'right')
    {
        gui.left = '90%';
    }
});

Now I want to animate it. I do:

gui.addEventListener('swipe',function(event)
{
    if (event.direction == 'right')
    {
        gui.animate({
            left:'90%',
            duration:500
        });
    }
});

When I animate it, I'd say the window maybe only moves to 40% left, not 90%. I tried changing the duration to something longer, but that makes no difference. I want the end result of gui to be at 90%, but I also want it animated rather than just setting the left property to 90 without animation

  • It could be that the animation engine is having a hard time incrementing from 0 (which is a number) to '90%' (which is a string). Try calculating 90% of the width of gui's parent, and animate the left of gui to that number.
