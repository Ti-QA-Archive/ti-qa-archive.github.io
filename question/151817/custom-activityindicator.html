Custom ActivityIndicator

We've created a custom activity indicator that uses animation to create an effect where a tail is chasing itself around the company's logo. It uses the following setInterval function:

setInterval(function() {
        var t = Ti.UI.create2DMatrix(), spin = Titanium.UI.createAnimation();
        clicks += 1;
        var degrees = clicks * 10;
        t = t.rotate(degrees);
        spin.transform = t;
        actH.animate(spin);
    }, 50);

Unfortunately, the animation gets stuck when the UI is busy and the page is about to close and the new page is going to open. Is there a way to use a separate UI thread to allow the animation to continue without any "stickiness", similar to how the spinner behaves in the default ActivityIndicator?

I'm using 2.1.3.

Thanks
John
