    Well, that depends.

    You could use the momentJS library to do your conversions.

    Or, you could build an object:

    var months = [];
 months[1] = L('january');
 months[2] = L('february');
 months[3] = L('march');
 months[4] = L('april');
 months[5] = L('may');
 months[6] = L('june');
 months[7] = L('july');
 months[8] = L('august');
 months[9] = L('september');
 months[10] = L('october');
 months[11] = L('november');
 months[12] = L('december');

};

    In this case, its using i18n for localizing the strings, but var myMonthName = months[4]; would return the localized string for 'april'
