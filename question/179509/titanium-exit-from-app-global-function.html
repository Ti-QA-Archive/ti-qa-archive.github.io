Here is my index.js file

var win = $.index;

here is my drawer.js file

var button = $.exit;

button .addEventListener(&#39;android:back&#39;,function(e) {
win.close();
});

When the user click the button the app should exit..Thank you!

  • Hi

    set "exitOnClose = true" property to window and try to close window
    e.g

    xml

    <Alloy>
    <Window id="winIndex">
        <Button id="btnExit"></Button>
    </Window>
</Alloy>

    tss

    ....
"#winIndex[platform=android]":{
    exitOnClose : true
}

...

    js

    $.btnExit.addEventListener('android:back',function(e) {  $.winIndex.close(); });

    Thanks
