  • You can use the free usage tier to run an Amazon EC2 Windows 2008 R2 server and an RDS instance, and it wont cost anything for a year if you don't use a crazy amount of data.

    You could then create lightweight web-services using something like Asp.Net Web services (.asmx).

    Encosia is a great website for how to do this: http://encosia.com/using-jquery-to-consume-aspnet-json-web-services/

    .Net is a very mature platform, easy to add a layer of security and SQL server is the natural database technology to use with it.

    If you want to stay in javascript you could try something like Tedious: http://pekim.github.com/tedious/getting-started.html, but I've never tried it..
