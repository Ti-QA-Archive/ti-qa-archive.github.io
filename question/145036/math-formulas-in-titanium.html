  • Hi

    It looks like your question is specifically about how you can display mathematical formulas visually correct but without resorting to providing images for each one?

    You could use a webView but you will still need to provide the formulas yourself.

    I would consider checking into Math formula fonts and then either use the webView idea or depending on the complexity you could provide labels on a view using the Math custom fonts. See custom fonts on how to build them into your app.

    You will need to source a suitable font though and I would imagine given the limited audience you will need to purchase it - most people are not proper math smart :) (including me). Make sure you purchase a web safe or on-line licence otherwise you will not legally be able to include it.

    • Well, the best way is to create a webview with the simplies markup possible, so it can fit in few lines.

      Rename Mathjax libraries from .js to .lib or something else, so they won't be compiled by Titanium.

      Do not create a html file for you webview, just embed the code in a .js file, so you can generate it on the fly. Add your variables directly into the html code (to avoid events between Ti and the webview).

      On other way could be to have only one (hidden) webview, wiche generate the formulas one by one. Every time one is generated, take a screenshot, save it (if you need to reuse it), and display it as an image. You may loose some time, but you will also save a lot of memory!

      Let me know if I'm not clear enough!
