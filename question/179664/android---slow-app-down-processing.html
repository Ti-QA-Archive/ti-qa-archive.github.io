Android - Slow app down processing

I am creating an app to pull information from a website. I have the code working, but on the initial load, I get errors.

I have created a function to grab the data. In that function, after the data has downloaded, it gets fed to another function, then parsed into system variables. Before the first function completes, the window gets displayed and I get errors because the variables aren't set.

How can I make the app slow down or stop until the download process has completed?

  • Best way to do this is to display a loading window with an activity indicator and status bar, feeding back information to the user to show that something is happening. After the data has all been downloaded, then show the main window.

    In addition, you need to cater for the situation when the user is not online - how is your app going to work then.
