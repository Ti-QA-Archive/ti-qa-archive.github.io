  • Bojan

    Note that it's not necessary to construct the whole Android path, as stated in Doug's code. Instead, do the following:

    var dbName = 'myDb';
var dbPath;
var dbFile;
if ( Ti.Platform.osname == 'android' ) {
    dbPath = Ti.Filesystem.applicationDataDirectory + '../databases/';
    dbFile = Ti.Filesystem.getFile( dbPath + dbName ); 
}
else {
    dbPath = Ti.Filesystem.applicationSupportDirectory + '/database/';
    dbFile = Ti.Filesystem.getFile( dbPath + dbName + '.sql' );
}

    Please let us know if you have success with this.

    Thanks
