Ti.API.info went away

Somewhere there must be a setting or a way to suppress logging via Ti.API.info since it has stopped totally for me sometime today. It outputs to the console fine until it says: "Application should be running." At that point, the app runs but no more messages in the console. I have twiddled every console and debug setting I can find to no avail. Stranger, alerts no longer open either so I am flying blind.

System:

MobileSDK / Android emulator
Ubuntu 10.04
SDK 3.0.0GA
Currrent Android SDK's (tried 3 different versions)

This is a new app just started from one of the stock templates.

Non-functional code at top of app.js:

Ti.API.info('Running Now' );
alert('Running Now');

This does not generate errors or any output either, making life difficult. Other code changes seem to work fine. How to get this back?

Thanks.
