    Hi

    You can achieve the result you are after using the imageView, instead of setting the image property, you use the images property (plural).

    You then need to tell it to play() and stop() as you need.

    Try this;

    var lbl1 = Ti.UI.createLabel({
    text: 'Window 1'
});
win1.add(lbl1);
lbl1.addEventListener('click', function (e) {
    if (img1.getAnimating() === false) {
        Ti.API.info('start');
        img1.start();
    } else {
        Ti.API.info('stop');
        img1.stop();
    }
});

var img1 = Ti.UI.createImageView({
    borderColor: '#666',
    borderWidth: 1,
    height: 46,
    images: [
        'KS_nav_ui.png',
        'KS_nav_views.png'
    ],
    top: 50,
    width: 43
});
win1.add(img1);

    I added the start event to the label, you can add it to whatever you like. The images array can contain a number of images.
