SDK 3.2.0 no longer supports iOS SDK 6.1

I have got the error when trying to compile my project on simulator 6.1 and SDK 6.1

[ERROR] :  The specified iOS SDK version "6.1" is not supported by Titanium 3.2.0.GA

From the release note, it says it does no longer support iOS SDK 5.X? Why is 6.1 not supported?

  • Accepted Answer

    We had to drop support for Xcode 4.x and iOS 6.1 and older. Titanium SDK 3.2.0 has started to use some of the iOS 7 specific APIs and thus Titanium SDK 3.2.0 only supports Xcode 5.0 and iOS 7.0.0-7.0.3.

    It's worth noting that you can build an app using Xcode 5.0 that will work just fine on iOS 6.1 and older.

    Also worth noting, Titanium SDK 3.2.0 does not officially support Xcode 5.1 and iOS 7.1 (currently in beta). There are issues with simulator builds that we're working on fixing, though device and distribution builds appear to work just fine with iOS 7.1.
