  • You could open a picker in a modal window.

    win.open({
    modal: true,
    modalTransitionStyle: Ti.UI.iPhone.MODAL_TRANSITION_STYLE_COVER_VERTICAL,
    modalStyle: Ti.UI.iPhone.MODAL_PRESENTATION_FORMSHEET
});

  • This is the code that opens the modal window

    var subWin = Titanium.UI.createWindow(
            {
                url:'../common/picker.js',
                navBarHidden:true
                }
            );
        subWin.open({
            modal: true,
            modalTransitionStyle: Ti.UI.iPhone.MODAL_TRANSITION_STYLE_COVER_VERTICAL,
            modalStyle: Ti.UI.iPhone.MODAL_PRESENTATION_FORMSHEET
            });

    This is the code of picker.js

    //Current window
var win = Titanium.UI.currentWindow;

var picker = Ti.UI.createPicker({bottom:0});

var data = [];
data[0]=Ti.UI.createPickerRow({title:'Bananas',custom_item:'b'});
data[1]=Ti.UI.createPickerRow({title:'Strawberries',custom_item:'s'});

picker.add(data);

win.add(picker);

    And everything i get is a new window with picker in the bottom, but the background of the modal window is white covering the full screen.

    If i use transaprent png image as modal window background then the window is transaprent only for 1/10 second. Then it goes as the png picture full opacity.
