    Hi

    At the very least it looks like at least one table in your database is not actually in the database prospectos.

    If you have a SQLite database that is installed onto the device, it is possible the original database has the table and your physical device either is missing that table when installed or a newer version has not been used in place of an earlier version.

    If you are building the database on the device then it is possible your physical device is not seeing all the code used to create the schema.

    Either way you need to debug the database creation and/or installation process.
