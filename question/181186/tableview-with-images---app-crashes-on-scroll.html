Tableview with images - app crashes on scroll

Hi ,

I have a tableview which loads some data from a web service along some images( image URL is sent in Webservice) .

there are about 100 records and when I try to scroll the table view, it crashes with the exception 'The application has crashed with an uncaught exception 'NSInvalidArgumentException''.

If I remove the imageview, scrolling works fine. I have given classname for the row as well.

I am using 3.4.0GA SDK. Below is my code


var row = Ti.UI.createTableViewRow({
                        backgroundColor : '#FFFFFF',
                        selectedBackgroundColor : '#dddddd',
                        height : 65,
                        rowData : searchResults[i],
                        className : 'PackageList' + i
                    });

                    var img = Ti.UI.createImageView({
                        image : searchResults[i].DefaultImage,
                        top : 3,
                        bottom : 3,
                        left : 3,
                        width : '20%',
                        height : '90%',
                        defaultImage : 'ui/images/iconrightarrow.png',
                        touchEnabled : false
                    });
                    row.add(img);

Any help on this please? Thanks in advance

Regards

Geervani
