  • This would be the AudioPlayer API which uses Matt Gallagher's AudioStreamer class. The error is a generic one thrown when the stream bombs out or the format is unreadable. It could be that the app is not receiving enough data to keep the buffer happy. It could simply be the state of the connection, rather than anything to do with your app. Can you play the same stream OK using another app?You could try increasing the buffer size on yours to see if it makes any difference.
