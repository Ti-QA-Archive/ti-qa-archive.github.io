How to change backgroundGradient of a button properly

I got some buttons with a backgroundGradient. Depending on there state (completed/new//disabled), they'll get a corresponding color.

  • completed (enabled) = green gradient
  • new (enabled) = blue gradient
  • disabled (disabled) = red gradient

All the buttons also got an other gradients for 'touchstart', 'touchcancel' and 'touchend'.

When you've completed a new button, it need to change from blue to green. The next button, need to change from red to blue.

So far, this works fine when I open the application.

But when a red button becomes blue, they still got a 'red gradient layer' somewhere. So when you push the button, an other 'blue' gradient will show, so you got an 'push-effect'. But when you do that, you'll see a blink of the (old?) red gradient. You do not see it all the time, but most of the time (95%).

How do I solve this? Maybe it's because the 'colors' property of backgroundGradient is an array. But I've tried to empty all first, but also didn't work.

The cause can also be that it get 2 eventListeners of everything. 2 times touchend, 2 times touchcancel and 2 times touchend.

How do I fix this?

        //btntouchend.source.backgroundGradient = ''; //Tried this to clean first

        btntouchend.source.setBackgroundGradient({
            type: 'linear', 
            startPoint: { x: '0%', y: '0%' },

  • I already solved the problem. The cause was that the button did get an extra eventListener. So it got 2 instead of 1. So I had to remove the previous eventListener first.

    The javascript file where it get it first eventlistener:

    if ((sVPS2+1) == button.index){
            //button.setBackgroundImage(ibg2);
            //button.setBackgroundColor('#00f');
            button.setEnabled(true);

            button.setBackgroundGradient({
                type: 'linear',
                startPoint: { x: '0%', y: '0%' },
                endPoint: { x: '0%', y: '100%' },

        } else if ((sVPS2+2) <= button.index){

            button.addEventListener('touchstart', btntouchstartno);
            button.addEventListener('touchend', btntouchendno);
            button.addEventListener('touchcancel', btntouchcancelno);

        } else {
            //button.setBackgroundImage(ibg1);
            //button.setBackgroundColor('#0f0');
            button.setEnabled(true);

            button.setBackgroundGradient({
                type: 'linear',
                startPoint: { x: '0%', y: '0%' },
                endPoint: { x: '0%', y: '100%' },
                colors: [ 
                    { color: '#'+L('knopgoednormaaltop'), offset: 0.0}, 
                    { color: '#'+L('knopgoednormaalbottom'), offset: 1.0} 
                ],
            });

            button.setBorderColor('#'+L('knopgoedborder'));
            button.addEventListener('touchstart', btntouchstartyes);
            button.addEventListener('touchend', btntouchendyes);
