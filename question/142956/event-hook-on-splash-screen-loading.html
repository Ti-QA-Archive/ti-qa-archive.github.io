Event Hook on Splash screen loading

My application requires to fetch data using web services & i want that data to be fetch using API while splash screen is loading at start up to reduce user wait time. can someone tell me how i can achieve this or any event which fires when splash screen is loading so that i can put some code over there to do my work

Any help will be appreciated.

  • So basically you want to show the initial splash image while loading data. To do this the first thing you have to do is an xhr call to you server requesting the data. Once you get the data you will fire and even or simply open the root window / tabgroup of your application either way the splash should stay up aslong as you dont put anything over it.
    On this note you might want to create a transparent window with an activity indicador, so the user knows that the app its doing something and close it when you are ready to show the app content
