Titanium CLI > Install to multiple devices over USB

I'm using the CLI (command-line-interface) to build my apps and install directly onto a single (android) device however I'd really like to be able to install to multiple (android) devices in one command connected via USB. Is this possible?

My current command is:

ti build --project-dir=[SNIP] --platform=android --target=device --no-colors

Actually, I'm using sublime-text with a build-system like so

{
    "cmd": ["ti", "build", "--project-dir", "$project_path", "--platform=android", "--target=device", "--no-colors"]
}

My knowledge of the actual build logic is beyond me (I've only been using titanium for a fortnight) however I was thinking I could use –build-only then replicate the final install command for each of the connected devices…

/path/to/adb -d install -r /path/to/app.apk

However I've no clue where to go from that idea - I'm sure that others would find this useful too.

  • You can build the app once and install it to multiple devices.

    #!/bin/sh
printf "Build the app"
ti build -p android -b

#Device A
printf "Installing the app to Device A..."
adb -s [Device A ID]  install -r /path/to/app.apk

#Device B
printf "Installing the app to Device B..."
adb -s [DeviceB ID]  install -r /path/to/app.apk
