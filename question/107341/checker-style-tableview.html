Checker style tableview

This is my contribution to appcelerator.
Please, don't blame me… it's my first day on titanium!

It's a checker style tableview creator, lets you create a tableview with separate items on every row.

Check the result:

If you want to try it:

save a file named "class.js" with this content: http://pastie.org/1516868

save a file named "checker.js" with this content: http://pastie.org/1516858

Include "checker.js" in your project and use like in the example.

You can also watch for "cellclick" events like this:

table.addEventListener('cellclick', function (e) {
    Ti.API.info("Cell click on ROW = " + e.index + " COLUMN=" + e.column);
});

Hope you like it,
ManuXio
