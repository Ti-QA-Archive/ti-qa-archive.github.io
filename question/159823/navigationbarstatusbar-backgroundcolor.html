Navigationbar/Statusbar backgroundColor

I'm trying to set the backgroundColor of the navigationBar and Statusbar and cannot get there.

index.xml

<Alloy>
    <NavigationWindow id="win1" platform="ios">
        <Window id="win2" title="Red Window" backgroundColor="red">
            <Button id="button" onClick="openBlueWindow">Open Blue Window</Button>
        </Window>
    </NavigationWindow>
</Alloy>

index.js

function openBlueWindow(e) {
    var win3 = Alloy.createController('bluewin').getView();
    $.win1.openWindow(win3);
}

$.win1.open();

index.tss

"#win1": {
    backgroundColor: '#33487e'
}

What am I missing?
