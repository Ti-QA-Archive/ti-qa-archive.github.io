City & Zip Code Search

I am a Titanium beginner and I am currently working on an iPhone application that needs to allow the user to search for their location either by entering their zip code or city. I would like the application to accept a zip code or city in the search bar and then provide a list of matched locations (city, state, zip) in a tableview below the search.

I would like to know if there are any tutorials or examples of this type of search that anyone could direct me to? I'm also open to suggestions about whether to use an API call or a local database search to accomplish this task.

Many thanks.

    I would recommend using the Google Maps API. Unless you're using a smaller subset of cities/zipcodes the database would probably be too large to host locally. Using the API you can do a simple search, parse the results, and render the results in a tableview just as you'd mentioned. Note, the API has some usage limits so you may want to sign up and obtain an API key. Hope this helps!

    • Sorry, I do not. I'd imagine you'll want to first request the data using the HTTPClient and than loop through the results adding TableViewRow's to your TableView. There may be a better way to orchestrate the search but at the very least, when someone's types in your search bar it will fire the change event. You can than use the input to filter (hide or even remove) the rows you'd previously added. I'm sure you've probably already seen it but the documentation (http://docs.appcelerator.com/titanium/latest/) is very helpful. You could also try breaking down the steps needed to accomplish your task and search Q&A for more specific examples, i.e. "how do I search table rows?" Good luck!
