How to hide TabBar (I am on Android)

I have created a tabbar on the main(app.js) screens with 3 tabs and not i want to Hide them on app load. The reason i want to hide the tab bar is because Android donot support Tabs on child windows and i want the tabs to be displayed on all windows andso i am creating a fake tabbar(created them from UI buttons with tab like look and feel). Now in order to show the fake tabbar i need to hide the real one so that the navigation works without the real tabbar being displayed(fake tab bar that i created from UI buttons will still be displayed).

Please let me know if my question is clear, or else i'll try to make 1 more attempt to explain the same.

Thanks a ton in advance.

Ankesh

    By "tabbar", you mean "tabGroup", don't you? Note that there is no "tabbar" supported by Android.

    You could use your custom tabGroup exclusively, rather than the Android tabGroup at the root of your application, which would give you full control over it.

    Alternatively, you could use the Android tabGroup in the root context, and open all child windows using fullscreen:false. This would make them heavyweight, modal, windows that would completely obscure the Android tabGroup in the root context below them.

    Hope this helps
