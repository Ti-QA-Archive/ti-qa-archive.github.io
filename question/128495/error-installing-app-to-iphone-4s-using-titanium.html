  • this

    [ERROR] xcodebuild: error: The directory /Users/bparr/Documents/Titanium Studio Workspace/BusBrain/build/iphone contains 2 projects, including multiple projects with the current extension (.xcodeproj). Specify the project to use with the -project option.

    mean that your TiStudio project name is different from the app name you have specified in your manifest file. This causes a project file to be created with a different name when you are building for a device and when you are building for the simulator.

    you need to ensure the names are the same in both places.

    clearing the build directory will make it go away for now, but it will just come back later
