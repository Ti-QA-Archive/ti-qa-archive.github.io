Paging Control Color SDK Update if u need it

Hi i see the current version of sdk not implement yet the currentPageIndicatorTintColor and pageIndicatorTintColor
i forked the current sdk version ( 3.0.2 ) and added it to the TiUIScrollableView and TiUIScrollableViewProxy

and now you can use 

pageIndicatorTintColor:'gray',
currentPageIndicatorTintColor:'black',

in the createScrollableView

for example


var asteSlider = Ti.UI.createScrollableView({
        top: 48,
        width: 380,
        height: 248,
        backgroundColor:'transparent',
        showPagingControl:true,
        pagingControlHeight:30,
        pageIndicatorTintColor:'gray',
        currentPageIndicatorTintColor:'black',
        overlayEnabled:true,
        pagingControlColor:'transparent'

    });

i make modify in the files

TiUIScrollableView.h
TiUIScrollableView.m
TiUIScrollableViewProxy.m

if you change it in you sdk

( they are in ~/Library/Application Support/Titanium/mobilesdk/osx/3.0.2.GA/iphone/Classes )

and after a clean rebuild the project you can use it

(sorry for my bad english :) )

here the modified files https://github.com/boardmain/TitaniumMods

i hope it can be usefull

bye
Samuele
