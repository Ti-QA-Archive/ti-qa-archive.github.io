  • I found solution.

    I updated my titanium sdk 2.1.3 but my project tiapp.xml's sdk version was still 2.1.1 !!!

    2.1.1 doesn't support iOS6 and XCode 4.5. that's the problem.

    I edited tiapp.xml sdk version to 2.1.3 and re-compiled and succeeded running on device.
