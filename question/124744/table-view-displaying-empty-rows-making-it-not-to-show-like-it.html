  • Accepted Answer

    I don't know if this is what you are talking about, but by default iOS displays empty table view rows to fill up the display. So even if you have 4 small rows, there will be additional rows filling the screen. One way around this however, is to set a blank footer on your table view. As the table view has a footer it doesn't try to add the additional blank rows.

    tableView.footerView = Ti.UI.createView({
    height: 1,
    backgroundColor: 'transparent'
});
