Create a scrollable view filled with a grid of dynamic buttons

So what I am trying to do is:

I am loading data from a sqlite db and using it with a while... isValidRow to dynamically create a set of buttons, that I set a static top and left(or right). What I have found is that if I make the parent of all the buttons a createView() and set the layout to horizontal, this will create a nice grid. 

var brandView = Ti.UI.createView({  
    width: 100,
    height: 100,
    left: 0,
    layout: 'horizontal',
    textAlign: 'center'
});

while(rows.isValidRow()) {
    var dynamicButton = Ti.UI.createButton({
        width: 10,
        height: 10,
        top: 10,
        left: 10,
        title: rows.fieldByName('name');
    });
    view.add(dynamicButton);

    rows.next();
}

The problem I am having is that sometimes there are more buttons than can be displayed on the screen at one time so the buttons extend down below the visible screen. So I turned the view to a createScrollView, and with the layout still set to horizontal (or vertical) it now places the butons side by side extending out beyond the screen to the side ( loosing the grid placements).

So my question is how can I make the view scrollable and have the buttons come out in a grid style formation?

  • Hi

    You are almost there, you just missed a variant of the standard view - called scrollView.

    You can tweak your code;

    var brandView = Ti.UI.createView({  
    width: 100,
    height: 100,
    left: 0,
    layout: 'horizontal',
    textAlign: 'center' // btw there is no property called textAlign for views
});

    To this; (I have made the view fill the window - but you can change these values to suit your needs.

    var brandView= Ti.UI.createScrollView({
    contentHeight: Ti.UI.SIZE, // these lets the internal space fill up greater
    contentWidth: Ti.UI.SIZE,  // than the external space if required
    height: Ti.UI.FILL, // this is the actual height
    layout: 'horizontal',
    showHorizontalScrollIndicator: false,
    showVerticalScrollIndicator: true,
    width: Ti.UI.FILL // this is the actual width
});

    For a more complete example to show you what you can have with a scrollView here is a simple example that displays many remote images that exceed the normal screen to the scrollView will allow scrolling automatically;

    To test this code you should create a new default project and then replace the contents of the app.js file with the code below.

    var win = Ti.UI.createWindow({  
    backgroundColor: '#fff',
    title: 'Scroll'
});
var tab1 = Ti.UI.createTab({
    icon: 'KS_nav_views.png',
    title: 'Scroll',
    window: win
});
//example data
var json = [
    { image: 'http://placehold.it/96x96/ff0000/ffffff&text=Hi' },
    { image: 'http://placehold.it/96x96/ee0000/ffffff&text=Hi' },

    { image: 'http://placehold.it/96x96/cc0000/ffffff&text=Hi' },
    { image: 'http://placehold.it/96x96/bb0000/ffffff&text=Hi' },
    { image: 'http://placehold.it/96x96/aa0000/ffffff&text=Hi' },
    { image: 'http://placehold.it/96x96/990000/ffffff&text=Hi' },
    { image: 'http://placehold.it/96x96/880000/ffffff&text=Hi' },
    { image: 'http://placehold.it/96x96/770000/ffffff&text=Hi' },
    { image: 'http://placehold.it/96x96/660000/ffffff&text=Hi' },
    { image: 'http://placehold.it/96x96/550000/ffffff&text=Hi' },
    { image: 'http://placehold.it/96x96/440000/ffffff&text=Hi' },

var scroll = Ti.UI.createScrollView({
    contentHeight: Ti.UI.SIZE,
    contentWidth: Ti.UI.SIZE,
    height: Ti.UI.FILL,
    showHorizontalScrollIndicator: false,
    showVerticalScrollIndicator: true,
    width: Ti.UI.FILL
});
win.add(scroll);
// provides the ability to offset positions
// also the layout property
var view = Ti.UI.createView({
    bottom: 8,
    height: Ti.UI.SIZE,
    layout: 'horizontal',
    left: 0,
    right: 8,
    top: 0,
    width: Ti.UI.FILL
});
scroll.add(view);
var intImage = 0, intImages = json.length, img;
for (intImage = 0; intImage < intImages; intImage = intImage + 1) {
    img = Ti.UI.createImageView({
        height: 96,
        image: json[intImage].image,
        left: 8,
        top: 8,
        width: 96
    });
    view.add(img);
}
var tabGroup = Ti.UI.createTabGroup();
tabGroup.addTab(tab1);
tabGroup.open();

    Be aware the scroll is automatic, but if the contents fit inside the parents dimensions without any overlap then no scroll action will occur obviously.

    You have not said if this is for Android, iOS or both but this will work for each platform.
