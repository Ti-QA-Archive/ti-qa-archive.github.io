    Hi, Suthan

    At the command line, run 

    titanium info -t ios

    and at the top it will print out the detected Xcode versions. If no versions are displayed, you may have a bad Xcode install or maybe it's in a location that Titanium can't detect. You can also confirm the Xcode location by running

     xcode-select --print-path.
