Local Sqlite database

Development Enviroment- Titanium-3.1.0, iOS-6.

I have created app which contains local sqlite database. after that I had installed app on iPad. After that when update is available on appstore then i have updated app from appstore. After updation what about local sqlite database, Is it remains as previous version or Is it refreshed according to updated version.

I don't know local sqlite database handled at the time of app updating through app store, Please give me detail information about that…

Thanks in advance…

  • The short answer is: The database that was installed the first time on the device will always remain. Meaning that even though you update the application, the database will not be overwritten by the new version.

    If you ever want to force the new version of the database when the user updates the app. You will have to do this by code when the application launched. (if the database file is already present, you simply delete it and then replace it with your new one.)

    Hope this clears this up for you
