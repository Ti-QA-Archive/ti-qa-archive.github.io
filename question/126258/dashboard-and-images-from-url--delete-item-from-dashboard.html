  • To add or remove items from a DashboardView you have to put the dashboard into edit mode. Only in edit mode can items be moved or removed from the view. The removeEventListener is for removing event listeners that have been attached to an object, for physically removing items from an object. For an example of how the DashboardView works, check out the Kitchen Sink demo in the file dashboard.js
