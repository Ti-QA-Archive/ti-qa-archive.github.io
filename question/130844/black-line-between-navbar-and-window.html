Black line between navbar and window

I've noticed that regardless of whether it is a tab window or a navgroup window, there is always a 1px black line between the navbar and window. This can be seen easily by looking at navgroup.js in the Kitchen Sink and adding barColor:'#ffffff' after line 21.

    I think the only way to get rid of is to build the navBar yourself and set the borderBottom to false, like in this example
    or to set an barImage to the window.

    var toolbar = Titanium.UI.createToolbar({
    items:[send, flexSpace, camera, flexSpace, cancel],
    bottom:0,
    borderTop:false,
    borderBottom:false
});
