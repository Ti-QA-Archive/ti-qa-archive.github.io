    • Or can i customise the navigation group chrome..so I can maintain the buil tin stacking?

    Lots of things you can do to your nav controller, as it takes on characteristics of the window its displaying:

    var self = Ti.UI.createWindow({
        barColor: '#00b9e6',
        backgroundImage : '/images/background.png',
        barImage : '/images/v2/navBarBackground.png',
        width : '100%',
        height : '100%',
        name: 'MyDay'
    });

var TitleBar = require('/ui/common/titleBar');
    var titleLabel = new TitleBar.createTitleBar('My Day');

    self.setTitleControl(titleLabel);

    • Title Bar code:

      exports.createTitleBar = function(text){
    var self = Ti.UI.createLabel({
        color: '#ffffff',
        height: 22,
        width: 210,
        top: 10,
        text: text,
        textAlign: 'center',
        font: {fontFamily: 'HelveticaNeue-Bold', fontSize: 20, fontWeight: 'bold'},
        shadowColor:'#262626',shadowOffset:{x:0,y:1}
    });
    return self;
};

    • Also as a note for anyone who finds this post seeking info.

      This is how to do a custom back button:

      Button Creation Code:

      // Set up backbutton
var backbutton = Titanium.UI.createButton({
   backgroundImage:"images/backBtn.png",
   width:30,
   height:30
});

//  Where $.chat is the window element id
$.chat.leftNavButton = backbutton;

backbutton.addEventListener('click', function(){

  Ti.App.fireEvent('app:backToMenu');

});

      And to handle the backbutton event. Example to go on root below. So this example is one level down and goes back to initial window in nav group:

      

// Make this more modular!!! to handle multiple views
