How to change the case of a module name to fix "Couldn't Find Module" Error

I've been burned by the mixed case issue with module names in IOS. My code has been running great in the simulator but now I'm getting the dreaded "Couldn't find module error on an actual device.

I'm in the process converting all of my modules to lowercase (e.g. Menu -> menu) but must be missing a step since my app on my test device will only load the module with the upper case name. Here's what I'm doing:

1) rename all of the .js, .xml and .tss files to the new lower case version (menu.js, etc.)

2) clean the project

3) deleting the app from my test device

4) running the app on my test device.

After all of the above it still will only work with

var win = Alloy.createController(&quot;Menu&quot;).getView();

Can someone tell me what else I need to do to make it work with "menu"?
