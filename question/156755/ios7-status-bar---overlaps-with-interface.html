iOS7 Status Bar - Overlaps with interface

I'm prepping my app for the iOS7 release and I've noticed a few strange things related to the device's status bar.

At first, the status bar was being overwritten by a solid black box. Reading this post (http://developer.appcelerator.com/question/156391/ios-7-status-bar) and upgrading my Titanium sdk to the latest release candidate fixed that issue.

Now, the status bar appears, but my application's interface elements lay behind the status bar. I've tried a few things to help resolve the issue, but haven'd had any luck:

* Set the window's top property to 40px
* Set each page's top property to 20px
* Change the statusbar-style property in tiapp.xml

Any other ideas?
