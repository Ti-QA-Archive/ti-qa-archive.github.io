SQLite-problems since Xcode 4.6.3

Hi,
since I updated my SDK I have this weired problem: I call from my views in modules the same model in module:

var ordnungen = require('module/botanicgarden.model').getFamilien();

Structure of model:

var DBNAME = 'flora1', DBFILE = '/depot/floradb.sql';
var botgartenLink = Ti.Database.install(DBFILE, DBNAME);
exports.getFamilien = function() {
    var sql = 'SELECT familie, count(familie) AS total FROM flora WHERE familie GROUP BY familie';
    var resultSet = botgartenLink.execute(sql);
    var families = {};
    while (resultSet.isValidRow()) {
        families[resultSet.fieldByName('familie')] = resultSet.fieldByName('total');
        resultSet.next();
    }
    resultSet.close();

If I call the method first time, it return with an array. If I call this getFamilien() from an other UI, the the same SQL-query give me as resultSet null.

Last week was not any issue.

Any ideas?
