Titanium Launch Error - Companion Shared Library

Hey guys,

Something very weird has been happening since yesterday.
I got a new iMac, downloaded and installed the lastest version of Titanium, but when I try to run it, I get the message:

The TitaniumStudio executable launcher was unable to locate its 
companion shared library.

I did some research and found a few possible solutions, such as renaming the launcher file in the &#x2F;Applications&#x2F;Titanium\ Studio&#x2F;Titanium\ Studio.app&#x2F;Content&#x2F;MacOS&#x2F;TitaniumStudio.ini from:
..&#x2F;..&#x2F;..&#x2F;plugins&#x2F;org.eclipse.equinox.launcher_1.1.1.R36x_v20101122_1400.jar
to:
..&#x2F;..&#x2F;..&#x2F;plugins&#x2F;org.eclipse.equinox.launcher.cocoa.macosx_1.1.2.R36x_v20101019_1345 (found in /Applications/Titanium\ Studio/plugins/)

However, that did not solve my problem.
Did anyone else run into this? Or perhaps have had this issue with Eclipse or some other IDE based on Eclipse? Or even just have a clue?

Thanks.
