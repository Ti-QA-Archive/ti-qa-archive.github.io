    Do you have android-ndk install in your system if not refer the following link http://developer.android.com/sdk/ndk/index.html.
    If you already have android-ndk, then check in you module, you will be having a build.properties file.
    open it. and check whether the path you added in it are correct,

    for example I have a module in which the build.properties look like this

    titanium.platform=/Library/Application Support/Titanium/mobilesdk/osx/2.0.1.GA2/android
android.platform=/Users/Ajeet/android-sdks/platforms/android-8
google.apis=/Users/Ajeet/android-sdks/add-ons/addon-google_apis-google_inc_-8
android.ndk=/Users/Ajeet/android-ndk

    See I have added the android.ndk path too,
