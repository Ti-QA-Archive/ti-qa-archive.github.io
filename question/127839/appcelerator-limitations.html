  • We try to keep parity with each platform. If you can do it natively, then we strive to make sure that you can do it via Titanium. There are already modules in our marketplace that that will help you with 3d, ie OpenGL. The Oink app is ok, in terms of design, and is something that can absolutely be accomplished using Titanium. In fact, there are a number of apps built on the platform that I believe are superior to the 'design' of Oink. You can see a number of these apps in our Application Showcase.
