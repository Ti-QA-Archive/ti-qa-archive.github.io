    • It really isn't too much to listen for the change in orientation, and adjust the button accordingly.

      
Ti.Gesture.addEventListener('orientationchange',function(e) {
    var o = parseInt(e.orientation);
    if (o == 3 || o == 4) {
        backBt.setHeight(24);
    } else {
        backBt.setHeight(30);
    }
});
