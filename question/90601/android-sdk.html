  • Hi again, Charlotte. Welcome back! :)

    To resolve the adb issue, first you need to ensure that you have the Android SDK Platform-tools package for the Android SDK installed. See Android Packages section of the "Installing on Windows 7" guide, paying particular attention to the section titled "Third-party SDKs".

    Then follow Workaround for a missing adb.

    Really, if you start at the top of the Installing on Windows 7 guide, you are more certain to get a working installation.

    Report back if you have further issues.
