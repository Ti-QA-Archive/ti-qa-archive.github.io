  • In 1.7.1 the following worked:

    var dbRootPath = 'databases';
var file = Titanium.Filesystem.getFile(dbRootPath, fileName);
Titanium.API.log('DEBUG', fileName + ' exists ? ' + file.exists());
var data = file.read().text;

    In 1.7.2, this is the correct way to do it:

    var dbRootPath = Titanium.Filesystem.resourcesDirectory + 'databases/';
var file = Titanium.Filesystem.getFile(dbRootPath, fileName);
Titanium.API.log('DEBUG', fileName + ' exists ? ' + file.exists());
var data = file.read().text;

    dbRootPath now contains the 'app://' prefix and everything seems to work ;)

    Thanks !
