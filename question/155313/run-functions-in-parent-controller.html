Run functions in parent controller

Hi Appcelerators,

Let's say we're using the controller "list". I from there create a new controller by using var win = Alloy.createController(&#39;common&#39;) and calling it as a modal, using win.open({modal: true;}).

Now, this modal needs to run the function getItems() inside our parent controller (list) and also update it's view based on those results.

Step-by-step

  1. Open controller list
  2. The controller fetches some remote data using the function getItems()
  3. The opens the modal controller common
  4. The controller fetches some remote data
  5. The controller requests the function getItems() from the parent controller list which updates the view (list).
  6. User closes the modal using the close button
  7. The parent controller (lists) should now been updated as from step 5.

Could I please get some help on how to solve this problem.
Any help will be appricated.

Thank you!

  • Problem was solved by using events, e.g. Ti.App.addEventListenter(&#39;update&#39;); and Ti.App.fireEvent(&#39;update&#39;);
