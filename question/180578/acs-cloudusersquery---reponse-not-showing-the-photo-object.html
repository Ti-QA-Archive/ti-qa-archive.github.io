    I figured it on my own. I don't know what changes did ACS devs did so that now I've to add the parameter response_json_depth

    Cloud.Users.query({
    page: 1,
    per_page: 10,
         response_json_depth : 2
    where: {
        age: { '$gt': 28 },
    }
},
