  • Evgeniy

    Try:

    Ti.UI.currentWindow.setBackgroundImage('img/background.png')

    If the exception returned is one like "file not found", then use an absolute path:

    Ti.UI.currentWindow.setBackgroundImage('/img/background.png')

    where the first forward slash is equivalent to the Resources folder.
