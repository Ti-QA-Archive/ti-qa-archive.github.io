  • Hello senthil p,

    Install Titanium CLI by using following two steps

    STEP-1: Install Node.js. From Link

    STEP-2 Install and configure the CLI.

    Install the titanium CLI using following command

    npm install titanium -g

    And Download a 3.0 or newer SDK

    titanium sdk update --branch 3_0_X --default

    For more details please see the Titanium document Titanium Command-Line Interface Reference. Thanks
