tabGroup

My program has a tabGroup,there are five tabs in this tableGroup.I have a tab (tab 1) that shows window A.Inside window A something happens and then window A is changed into window B.now ,there is button C in window B,when i click button C,i want to get back tab 1,how do i achieve it?

  • Couple of options, if all your code is in a single execution context you can just stash the tabGroup you create on the App object

    Ti.App.TabGroup = tabGroup;

    and then in your window do

    Ti.App.TabGroup.setActiveTab(0);

    otherwise you can hook up an event on the app and fire that off

    Ti.App.addEventListener('app:changetab', function(e) {
    tabGroup.setActiveTab(e.tabIndex);
});

    and in your window 

    Ti.App.fireEvent('app:changetab', { tabIndex:0 });

  • Try adding an event listener to button C that has this line of code:

    Titanium.UI.currentTab = tab1;

    This should set the current tab to tab1, which has window A assigned to it, so window A should become active.
