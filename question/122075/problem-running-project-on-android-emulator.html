    • Hi Paul, I've found the problem. It turns out that Titanium Studio MUST have the path to <JDK_Home>\bin specified in the environment PATH variable. Without it, the javac command is not found and the build of the project fails. I noticed this by looking through the various consoles that are created when attempting to run the project.

      Question for you… I've not had to add <JDK_Home>\bin path to my environment PATH in the past when working in the Eclipse IDE. Is there a particular reason why this would be the case with Titanium Studio?
