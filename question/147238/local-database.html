    Hi Iacopo,

    If you have large set of data to be stored for offline use then SQLite is the best option. Refer http://docs.appcelerator.com/titanium/3.0/#!/api/Titanium.Database

    If you have few variables to be stored globally then please use the following property to get and set data:

    Titanium.App.Properties.setString('key','value); // set value

Titanium.App.Properties.getString('key'); // get value
