APP rejected because of Dropbox ?

My APP let user to store photos, so I bought the Dropbox module from Appcelerator market ,and put an Dropbox icon on screen for user click to login.

But Apple store rejected it, it seems that I can not use an button link to Dropbox for user to login.

My question is, why Facebook login button is ok no problem, but Dropbox login button is not allowed ?

Below is the reject reason from Apple.

We found that your app provides access to external mechanisms for purchases or subscriptions to be used in the app, which is not in compliance with the App Store Review Guidelines.

Access to external mechanisms via the "Dropbox" feature are seen within the binary.
