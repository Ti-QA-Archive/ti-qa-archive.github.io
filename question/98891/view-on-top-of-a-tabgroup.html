View on top of a TabGroup?

My app heavily relies on the latest updates from a server. While the user is using the app, I want to get the latest updates from my server and update the items later on. To show the user updates are being downloaded, I'd like to make a view that pops up above the TabGroup, like a blue bar saying "Getting updates".

Is it possible to add a View to the root and below that add the TabGroup? So like:

|      UPDATING BAR     |
|-----------------------|
| Tabgroup Window Title |
|-----------------------|
|      content here..   |

Thanks, please answer.
