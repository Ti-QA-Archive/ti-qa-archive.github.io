Change views background on a page with multiple views

Hello

I have created a floorplan based on an XML File. For each booth on the plan I made a boothView.

Now when I have a list of all my booths on a tableview, and I click one and ask for the map, can I access this specific view by it's ID or something like that, so I could highlight it (change the bg)?

Or will I have to re-draw my entire floorplan and set the background for the specific booth?

I'm using Titanium SDK 1.6.1 and iPhone SDK 4.2

Thanks!

  • You shouldn't have to redraw your entire floorplan. If each "booth" is its own view, then you can just change the color of the boothView. I would imagine it would look something like this. You have your views, you add an event listener that gives the clicked view a new background and changes the others back to the original background.

    boothView1 = createView();
boothView2 = createView();

boothView1.addEventListener('click',function(e){
        boothView1.backgroundColor = "#ff0000";
        boothView2.backgroundColor = "#ffffff";
});
