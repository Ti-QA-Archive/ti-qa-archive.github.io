    You can't; or at least there is no way to do this automatically. You could perhaps manually keep track of this as a piece of data associated with the view, something like

    function myFunction(){
    alert('hello');
}

var v = Ti.UI.createView({
    height: 50,
    width: 50,
    _handlers: 0
});

v.addEventListener('click', myFunction);
v._handlers++;

v.removeEventListener('click', myFunction);
v._handlers--;

    Why do you want to keep track of this?
