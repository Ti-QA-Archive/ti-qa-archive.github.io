Scroll label text iOS

Greetings

I am using a label within a scrollview; the text has 3 or 4 paragraphs.

On android the text is displayed vertically and is able to scroll; but on iOS the text is displayed horizontally; and shows a "…" indicating there are more text to be displayed.

Here is my code:

var Scroll = Titanium.UI.createScrollView({
    width : "100%",
    showVerticalScrollIndicator : true,
    top : 135,
    backgroundColor : "#F5E3BF"
});

var informacion = Titanium.UI.createLabel({
    minimumFontSize : 12,
    font : {
        fontFamily : "MuseoSans-300"
    },
    color : "#6F6F6F",
    width : "80%",
    top : "10",
    right : "10%",
    height : Ti.UI.SIZE
});

Scroll.add(informacion);

Anyone knows how to fix it?

Thanks

    Hello Hund,

    I have tested this below code, It's working

    var win = Ti.UI.createWindow({
  title: 'ScrollView Demo'
});

var scrollView = Ti.UI.createScrollView({
  //contentWidth: 'auto',
  //contentHeight: 'auto',
  showVerticalScrollIndicator: true,
  showHorizontalScrollIndicator: true,
  height: '80%',
  width: '100%'
});


var label1 = Ti.UI.createLabel({
  color:'red',
  text: 'Add a long label here',
  width: Ti.UI.FILL, height: Ti.UI.FILL
});

scrollView.add(label1);
win.add(scrollView);
win.open();
