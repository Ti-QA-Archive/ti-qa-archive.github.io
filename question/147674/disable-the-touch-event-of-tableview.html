  • Hello Subhan,

    Try like this…

    
var tbl = Ti.UI.createTableView({
    top : 10,
    bottom : 15,
    width : "100%",
    scrollable : false,
    backgroundColor : 'transparent',
    touchEnabled : false
});
win.add(tbl);
