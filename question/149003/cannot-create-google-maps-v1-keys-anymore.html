Cannot create Google Maps v1 keys anymore

Version 1 of the Google Maps Android API as been officially deprecated as of December 3rd, 2012. This means that from March 3rd, 2013 you will no longer be able to request an API key for this version.

I just created a API key for a Titanium project. I was just in time. But how am I going to do this in the near future? Does Titanium has an update after this weekend where the Mapview works with v2 of the API?
