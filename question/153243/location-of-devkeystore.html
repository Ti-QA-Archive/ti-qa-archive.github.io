Location of dev_keystore

Okay so this all started when I wanted to link Google Play Game App thingy to a game I made. So, this is what it asked me to get:

Signing certificate fingerprint (SHA1): 
(Example: 21:45:BD:F6:98:B8:71:50:39:BD:0E:83:F2:06:9B:ED:43:5A:C2:1C )

So I looked up how to find that in Titanium, and I got this:

For the debug certificate, the SHA-1 certificate fingerprint is stored in a keystore file that comes with each Titanium SDK. The keystore is located at mobilesdk/<platform>/<sdk_version>/android/dev_keystore in your Titanium SDK home directory. For example, to retrieve the SHA-1 fingerprint on Mac OS X, for Titanium SDK 3.0.2.GA in the default install location, run the following command: {some OSX command}

The problem is, I am using Windows, but the Ti.Doc only gives me info for OSX.

So, in short, how can I get the Signing certificate fingerprint (SHA1) for Windows7?
