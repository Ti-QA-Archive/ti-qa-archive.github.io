Flipping one window into another

Okay so I have a modal window that contains a login form under the form there is a button for users that do not have an account. They click the button and are presented with a different form enabling them to register. Currently I have the login window in a modal window and that's great but how can i make it so when the user taps the register button the modal window flips over and the register window is displayed.

I thought by simply creating my login window and making the register button open a new winnow with the iOS modal flip transition and then closing the login modal would work but because i'm using alloy i'm having to call the register window into it. So when i close the login window after opening the register window and then when the user closes that i get a black screen. So how can i achieve this effect? I hope it makes sense.

Here's some code:

Login js

$.cancelLogin.addEventListener('click', function (e) {
    $.loginWin.close();
});

$.signUp.addEventListener('click', function (e) {
    var register=Alloy.createController('register').getView();
    register.open();
    $.loginWin.close();
});
