    I think what you want is a custom theme. Check out: http://android-holo-colors.com/ and choose a color and turn on "Edit Text". You'll see the underscore asset at the bottom in the Output Resources (textfield, textfield_focused, textfield_activated).

    Also, be sure to select AppCompat for "Compatibility" if you are using SDK 3.3.0 or higher.

    Hope this helps.
