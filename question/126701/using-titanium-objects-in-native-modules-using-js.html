Using titanium objects in native modules using JS

Hi,
In this article http://wiki.appcelerator.org/display/guides/Module+Developer+Guide+for+iOS It's mentioned that one can place a js file in the 'assets' directory and use commonJS to write a native modules in JS.

The example works great! I compile my app with the module and I can see I get the alert from the example perfectly!

But what I really want to do is return a Titanium object (with some of manipulations of my own). So I started by writing this simple function:

exports.getMyLabel = function(params) {
    var lbl = Titanium.UI.createLabel(params);
    return lbl;
};

The modules compiles. but when in my programs I write:

var lbl= my_module.getMyLabel({ text: 'test', top: 100, left: 100 });
win.add(lbl);

The application crashes.

How can I use Titanium modules in native js modules?

Thanks

    Just saw it… you cannot start your methods with get or set

    — answered October 11th 2011 by Aaron Saunders

    • exports.getMyLabel = function(params) {
    var lbl = Titanium.UI.createLabel(params);
    return lbl;
};

      should be 

      exports.loadMyLabel = function(params) {
    var lbl = Titanium.UI.createLabel(params);
    return lbl;
};
