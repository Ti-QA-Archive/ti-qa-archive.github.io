Changes in 1.8

Have you tested your application with 1.8?

1.8 seems very cool at the moment and I'm looking forward to final release. Nevertheless, I've seen some changes, so I would like to post here changes that I noticed, you can also post your own observations, so we can all prepare for future full release.

I noticed:

  • SVN files are finally ignored correctly. No need for workaround anymore :)
  • Splash screen functionality seems to be different now, in 1.7.3 I have my own picture, in 1.8 there is Appcelerator's default pic
  • numeric keyboard in Android shows numbers only. Finally!! But it seems that it's not functional yet as it, IMHO, should be. I can enter non-numeric values, there is no key to minimize keyboard.
  • more stable in general

Feel free to add you observations.
