    Hi Abdelali

    If I have read your question correctly, you have a map IMAGE which is not the actual Google Maps object. Which is why you say annotations are not suitable?

    If you wish to add "pins" over the top of your image you could simply create a view and then add an imageView over the top for every "pin" you wish to show. You would have to locate them yourself using top and left to offset them. Provided your "pin" images are PNGs then it would look fine.

    Example code;

    var viewImageMap = Ti.UI.createView({
    backgroundImage: 'yourmapimage.jpg',
    height: 500,
    width: 500
});
win.add(viewImageMap);
var pin1 = Ti.UI.createImage({
    height: 32,
    image: 'pin.png',
    left: 50,
    top: 50,
    width: 32
});
viewImageMap.add(pin1);
var pin2 = Ti.UI.createImage({
    height: 32,
    image: 'pin.png',
    left: 100,
    top: 100,
    width: 32
});
viewImageMap.add(pin2);
var pin3 = Ti.UI.createImage({
    height: 32,
    image: 'pin.png',
    left: 150,
    top: 150,
    width: 32
});
viewImageMap.add(pin1);

    Change image names and dimensions to suit your needs, You could of course add the "pins" in a loop rather than on by one.

    Hope this helps.

    • Hi Abdelali,

      for a simple image it is easy.

      var win = Ti.UI.createWindow({
    backgroundColor : "#fff"
});

var imageView = Ti.UI.createImageView({
    width : 300,
    height : 300,
    top : 10,
    image: 'uk_map.jpg'
});
win.add(imageView);

var pin = Ti.UI.createImageView({
    width : 30,
    height : 30,
    top : 50,
    image: 'map_pin.png' // png for transparent background of the pin
});
imageView.add(pin);

win.open();
