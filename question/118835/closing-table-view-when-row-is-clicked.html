Closing table view when row is clicked

I know that there is an easy answer for this, but I can't seem to find it.

I use a tableview with with a number of rows. When a row is pressed, I want to return to the previous view, just like if I had pressed the "back" button at the top of the window.

How is that done? I've tried to remove the view and a few other things, but none of these does what I want it to do.

Any suggestions?

  • Trigger Ti.currentWindow.close(); which will close the window with the tableView on and 'back' to the previous window (if there is one)
