    • So, in templateVentana, do not return the window, instead return an object, like this:

      return {
    window:self,
    closeAppMain: closeAppMain
};

      Cleate a function to close that other window and attach it to the object being return. Create a reference to that window so that it can be closed, like this:

      var otherWin;
searchBtn.addEventListener('click', function() {
    otherWin = null;
    var appMain = require('appMain');
    otherWin = new appMain();
    otherWin.show();`
});

      And this can be the function to close it:

      function closeAppMain(){
    if(appMain != null){
        appMain.close();
    }
}

      And from "XXX", call it:

      function xxxx(title) {
    var templateVentana = require('templateVentana');
    var self = new templateVentana();
    var win = self.win;

    // close it here!
    self.closeAppMain();
}
