    • Thanks guys, really appreciate your patience!

      I finally got round the problem by creating two different alloy controllers, one to load the iOS window shown in my first post, and another to load the android version which skips out the NavigationGroup and second Window.

      I now call these using this function:

      var onTableClick = function(object) {
    var eventPage;
    if(Alloy.Globals.osname == 'android') {
        eventPage = Alloy.createController('pageEventAndroid');
    } else if (Alloy.Globals.osname == 'iphone') {
        eventPage = Alloy.createController('pageEventiOS');
    }

    eventPage.openMainWindow("data in here");
};
