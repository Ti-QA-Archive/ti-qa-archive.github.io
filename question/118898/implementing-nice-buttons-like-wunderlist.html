    Michael

    Bearing in mind you state you are using CSS, does this mean you are using webViews in your app rather than native controls? If so, I would advise against this if possible, as the app's performance will suffer.

    The native Titanium.UI.Button control can be styled using the background*Image properties, an example of which is available in the KitchenSink's button.js script.

    If this does not help, please provide some more details, such as links to the images or videos that show the buttons you have mentioned.

    Hope this helps
