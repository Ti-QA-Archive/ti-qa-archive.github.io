My understand is that Titanium will get my JS code and compile it down to native code, right?
So if I use a framework like Sencha Ext Core to take advantage of the class structure it provides, will all of that be compiled to native code?
Using Ext Core on a mobile project will decrease the overall performance or JS code is all the same for Titanium?

    Most frameworks out there today require the DOM, Titanium does not have a DOM so they fail… there are several open source titanium frameworks that people have started on GitHub, Ext, jQuery and such will not work.
