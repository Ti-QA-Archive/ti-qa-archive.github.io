    Hi Ronny,

    You can create email dialog using the below code, it will work in both iOS and Android

        //Email Dialog
    var emailDialog = Titanium.UI.createEmailDialog();
    if(emailDialog.isSupported()) {
        emailDialog.toRecipients = ['foo@yahoo.com'];
        emailDialog.subject = "Test subject"; 
        emailDialog.messageBody = "Body text";

        //Attachment
        var f = Ti.Filesystem.getFile('cricket.wav');
        emailDialog.addAttachment(f);
        emailDialog.open();
     }else{
         alert('Please Configure Email');
     }

    Here you can find more about email dialog http://docs.appcelerator.com/titanium/2.0/index.html#!/api/Titanium.UI.EmailDialog
