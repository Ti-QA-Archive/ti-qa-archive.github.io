Android camera custom overlay in portrait mode.

Is this possible? The default view seems to be in Landscape, but our requirement is for Portrait.

There are plenty of native Android apps with custom overlay in portrait mode: Instagram, Facebook, etc.

If this is not supported by Titanium, is there a way to develop around it with a module?
