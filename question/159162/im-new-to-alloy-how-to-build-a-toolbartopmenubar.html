  • Hi

    Your example image can very easily be built using a navigationWindow. However your example image is an iOS only UI element so will only work for iOS.

    You say you want your apps to look the same but they will not, different apps use fundamentally different UI elements for navigation controllers. iOS uses navigation windows and android uses actionBar and menus. You should never try and make two different platforms look the same, the users of android do not want to us an app designed for iOS and neither do iOS users want to use an app designed for android.

    Titanium is a cross platform development tool, this does not mean that your code will look and work the same on every platform and it does not mean every platform shares all elements of each other.
