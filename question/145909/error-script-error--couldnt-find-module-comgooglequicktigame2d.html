[ERROR] Script Error = Couldn't find module: com.google.quicktigame2d.

Hi everyone,

I am trying to develop a mobile-game, using QuickTiGame2D and TiledMapEditor, through Titanium, but I am still stuck in the starting point, because the simulator doesn't find the quicktigame2d module.

I checked all the information, and I have the module "com.googlecode.quicktigame2d" already in /Library/Application Support/Titanium/modules/iphone/com.googlecode.quicktigame2d

In the same way, my tiapp.xml looks like this:
<?xml version="1.0" encoding="UTF-8"?>
<ti:app xmlns:ti="http://ti.appcelerator.org">
…..
<sdk-version>2.1.3.GA</sdk-version>
…..
<modules>
<module platform="iphone" version="1.2">com.googlecode.quicktigame2d</module>
</modules>
</ti:app>

On the other hand, in my app.xml, I included:
var quicktigame2d = require('com.google.quicktigame2d');

Thus, I think I followed all the steps needed to include a new module and use it, but still it doesn't find it.

Any help would be great!
