Facebook module: How to request a new token

Hi,

I'm using Titanium facebook module (tit 3.1) and I can't request a new Token.
The problem is that with this code:

fb = require('facebook');
fb.appid = 'xxxxxxxxx';
fb.permissions = ['publish_stream'];

fb.getAccessToken();

is returning the same token.

Also, if I get fb.getExpirationDate() shows this "Fri Oct 04 2013 18:03:45 GMT+0100 (WEST)" but the token after a fresh login use to expire in 2 hours…

Anyone?

Regards
joao

  • Hi Joao, I think that your problem can be fixed by using
    fb.authorize();
    before using the getAccessToken method.
    The authorize sees if you are already logged in and if you are not it tries to log you in again (with the facebook popup).

    After this, you'll get a brand new access token.

    Hope it helps,

    Eduardo Rocha.
