  • Accepted Answer

    I had the same problem. Turns out I installed KitchenSink before Titanium could download and install the 1.5.1 mobile SDK. Deleting KitchenSink and re-importing after the update fixed the problem. If you haven't downloaded the mobile SDK update, Titanium will have a "New Mobile SDK available" message on the top right of the window with a link that will start the download.
