UI Picker

How is it not possible to move the Picker from the center of the Window? I'm pretty sure I've done it before but I didn't back up my project I can't figure out how I did it. I can't believe that it's stuck in the center and can't be moved. Maybe I'm wrong; if anyone can help I'd greatly appreciate it.

  • Ok, I found the answer…

    var transformPicker = Titanium.UI.create2DMatrix().scale(1.0);
    var picker = Ti.UI.createPicker({
    backgroundColor: '#ff9900',
    top: 244,
    //heigth: 4000,
    width: 320,
    transform:transformPicker
    });

    win1.add(picker);
