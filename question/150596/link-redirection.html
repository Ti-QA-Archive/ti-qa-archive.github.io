  • Hello zaki,

    From my understanding, i think you want to open the controller.

    Just try below sample :

    index.xml

    <Alloy>
    <Window class="container">
        <Button id="button" onClick="doClick"/>
    </Window>
</Alloy>

    index.js

    function doClick(e) {
    var newController = Alloy.createController('sample').getView();
    newController.open();
}

$.index.open();

    *sample.xml

    
<Alloy>
    <Window class="container">
    </Window>
</Alloy>
