Prevent screen off

I need to run a process (updating sqllit) that takes about 10 minutes on an android terminal. However, if during the process, the terminal is locked due to inactivity, the process hangs and the application also hangs. Can prevent it?

Thanks

  • Accepted Answer

    You could be a bad developer and add the WAKE_LOCK permission to your androidmanifest.xml file.

    The problem, is that your app is now on ALL the time. Not a great solution.

    <manifest android:versionName="1.0.0">
     <uses-permission android:name="android.permission.WAKE_LOCK"/>
</manifest>

    Better option is to set keepScreenOn property on a view on the screen doing the update.
