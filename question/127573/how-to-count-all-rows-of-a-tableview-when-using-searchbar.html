How to count all rows of a tableview when using searchBar

I use a tableview with a searchbar. When searchbar change value, I want to count all visible rows of tableview. How do I do?
Please help ! …

  • You can loop trough the rows and check the ".visible" property.

    var tableViewRows = tableView.data[0].rows,
    v = 0;
tableViewRows.forEach(function(val, i, elem) {
    if ( elem.visible ) {
        v++;
    }
});
Ti.API.log('visible rows: ' + v );
