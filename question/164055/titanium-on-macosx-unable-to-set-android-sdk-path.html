  • Adnan,

    The error you posted indicates that you need android-10 along with its equivalent level 10 google apis. This is the minimum level required to get started using Titanium at this time. This does NOT mean you cannot include higher level APIs and set them as your target api level, it just means that the Titanium tooling expects a minimum level of android support to be installed.

    The second error you posted indicates you are trying to build a project that eother has TiSDK 2.1.2 set in nits tiapp.xml (which in your case, this is a new project, so that is probably not the case) or you have 2.1.2 installed an not the newer versions of the sdk.

    Toss us the report generated by ti info, properly formatted, and we can take a look to see what else you are lacking.
