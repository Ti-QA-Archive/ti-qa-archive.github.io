Problem with Android WebView and local files

Hi,
I'm developping an android application that have to show a set of local HTML files (located in the External Storage, but i tried also with files in the resource directory) in a WebView.
Local site has a frameset structure, with a menu on the left and the content on the right; the problem is that android emulator renders the site in a wrong way making the menu frame disappear showing only che content frame. I tried the code on a real device where the initial frameset is showed correctly but, clicking a link on the menu frame that should load another local page in the content frame, make the menu frame disappear as in the emulator and only the new page is showed in the webview.
I noticed that if pages are located on a web server and the webview refear to that external url everything work fine, also if the link in the menu frame refear to an extenal link (google for example) instead a local file the frames behavior is correct and the menu frame don't disappear.

Below the frameset:

<!DOCTYPE html PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN" "http://www.w3.org/TR/html4/frameset.dtd">
<html>
    <head>
        <title>Frameset Example Title (Replace this section with your own title)</title>
    </head>
    <frameset cols="200,*" frameborder="0" border="0" framespacing="0">
        <frame name="menu" src="left.htm" marginheight="0" marginwidth="0" scrolling="auto" noresize>
        <frame name="content" src="right.htm" marginheight="0" marginwidth="0" scrolling="auto" noresize>
        <noframes>
            <p>This section (everything between the 'noframes' tags) will only be displayed if the users' browser doesn't support frames. You can provide a link to a non-frames version of the website here. Feel free to use HTML tags within this section.</p>
        </noframes>
    </frameset>
</html>

Below the left html menu:

<html>
    <head>
        <title>Menu</title>
        <meta http-equiv="Content-Type" content="text/html; charset=iso-8859-1">
    </head>
    <body>
        <a href="link1.htm" target="content">link1</a><br>
        <a href="link2.htm" target="content">link2</a><br>
        <a href="link3.htm" target="content">link3</a><br>
        <a href="http://www.appcelerator.com" target="content">appcelerator</a><br>
    </body>
</html>

I'm using:
Titanium SDK version 2.1.2 GA, Titanium Studio 2.1.2, Android emulator with Google APIs Android 4.0, and the real Android device is a Samsung Galaxy Tab 10.1" (Android Version 3.2).

Thanks for all the help!
