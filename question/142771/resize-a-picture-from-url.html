resize a picture from url

I am new to Titanium Studio.

I cannot seem to find a way to re-size pictures from a url.
All the pictures I want to display in my app are coming from url's and I need a thumbnail version and a version which fits the full screen.
At the moment I am using coverflow and because the images are high res I only see a fraction of the actual image.

Any idea's how I would do this?

    From Kitchen Sink:

        var images = [];
    var prefix = 'http://grin.hq.nasa.gov/IMAGES/SMALL/GPN-2000-0000';
    var start = 38;
    for (var c=0;c<30;c++)
    {
        var name = prefix + (start+c) + '.jpg'; 
        images[c]= {image:name, width:225, height:225};
    }

    // create coverflow view with images
    var view = Titanium.UI.iOS.createCoverFlowView({
        images:images,
        backgroundColor:'#000'
    });

    Also check out the Images and ImageViews API Guide.
