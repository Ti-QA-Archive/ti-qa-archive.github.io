ScrollView inside another ScrollView not scrolling

Am making an app that has a chat, so I need to put an ScrollView and a View inside another ScrollView.
This is a reference image of what am doing. The borders on green is ScrollView1 (containing scrollview2 and another view for the test and a send button).

The first ScrollView (the big one) if for when the user touch the TextArea, the softkeyboard push up the content.

This is what am actually doing:

                </View>

            </ScrollView>
            <View id="writing">

  • I'm not working with Alloy, but this save my day when I had a scrollView and a common View that I wanted to drag and drop.

    parentScrollView.canCancelEvents = false;

    It's seems that the parent view cancels de scroll of his children.
