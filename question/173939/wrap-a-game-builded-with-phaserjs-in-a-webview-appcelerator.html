  • Martin, as per the API documentation here, all local .js files are removed and their content encrypted and obfuscated into one resource.
    For JavaScript files referenced in static local HTML files, these files are ignored. For files not referenced rename the file with an extension of .jslocal

    Is your file directly referenced in the local HTML file?
