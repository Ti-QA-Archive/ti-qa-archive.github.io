    Hello David,
    I hope I understood your question correctly:

    ScrollView:
    Set the scrollingEnabled to false, so manual scrolling is disabled.

    Then use your button to fire the scrollTo method to scroll to a certain point in the view.

    var scrollView = Ti.UI.createScrollView({
    ...
    scrollingEnabled: false
    ...
});

var imageView = Ti.UI.createImageView({...});
scrollView.add(imageView);

btn.addEventListener("click", function() {
    scrollView.scrollTo(0, /* yourY */);
});

    I don't recommend using the ListView for this example, as it doesn't have the scrollingEnabled property to disable manual scrolling.
