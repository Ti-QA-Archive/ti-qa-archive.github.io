No Toolbar Buttons in Titanium Studio

I've just installed Titanium Studio (2.1.0) for the first time. I'm brand new to this. I'm following the "Get Started" Tutorial, and it is referencing the toolbar buttons to click, but there's no toolbar buttons (Example here). There's toolbar buttons on the App Explorer window, Process Explorer window, etc. but no main toolbar buttons. Is there a way to bring them up? Can't seem to find the command to bring it up.

  • Accepted Answer

    This is the result of a new simplified UI–the dashboard should have come up automatically on launch.

    There are two ways to get to the buttons. To just get to the Dashboard, use: Help > Titanium Studio > Show Titanium Dashboard

    To make the buttons reappear: Window > Show Toolbar
