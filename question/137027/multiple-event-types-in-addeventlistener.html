Multiple event types in addEventListener

Is it possible to listen for more than one event type in a single event listener function? For example

search.addEventListener('blur click cancel', function(e) { });
— asked May 16th 2012 by Ian Tearle

    What you need to do in this case is to store in a variable the function you want to run on every event.

    Here is an example:

    var doStuff = function(e) {
    // Same code that will be run for blur, click and cancel
};

search.addEventListener('blur', doStuff);
search.addEventListener('click', doStuff);
search.addEventListener('cancel', doStuff);
