ACS Query tags_array

So My Custom Object Query Looks like this:

var str = "hello,blue"
Alloy.Globals.Cloud.Objects.query({
                classname: 'myObj',
                where: {
                    tags_array: str.toLowerCase(), // all tags are stored in lower case.

I have a custom Object on ACS with the tags: blue,hello,red,green

This ACS Object should be returned, but returns nothing?

  • SOLVED:

    str = str.toLowerCase();   // 1- make sure the queries is lowercased
str = str.split(/[ ,]+/);  // 2- convert string to an Array
//Ti.API.info('Querying tags: ' + JSON.stringify(str)); // An Array now
Alloy.Globals.Cloud.Objects.query({
    classname: 'book',
        where: {
            tags_array: {'$in':str}, // searches for one & multiple keywords!
