Internationalize MenuItem

Im having a problem with internationalization on my android menu.

This is my view:

<Menu>
    <MenuItem id="mnuNew" />
</Menu>

And this is the code that throws and error:

$.mnuNew.title= L('new','New);

This throws the error: Cannot set property 'title' of undefined.
I can't figure out why is my MenuItem undefined.

    Hi

    This can be added to the TSS file;

    '#mnuNew': {
    title: L('new','New');
}

    I am assuming this was a typo;

    L('new','New);
