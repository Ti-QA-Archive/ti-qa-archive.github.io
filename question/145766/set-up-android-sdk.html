      Please following these steps:

      • You tried to install many old API Versions from Android 1.5 (API 3) to Android 2.1 (API 7). If you don't need them, please uncheck and don't install them.

      • For Titanium SDK Version (1.8.x - latest) you need a minimum SDK Platform Android 2.2-update x, API 8, rev y, see Titanium Compatibility Matrix

      • The failing fetch for URL: http://software.intel.com/sites/landingpage/android/addon.xml looks like you checked some System image or Intel x86 Emulator Accelerator from Extras.
        Please uncheck intel system images from APIs and Intel x86 Emulator Accelerator (HAXM) from Extras and see if the update proceeds.
