Geo recording in background

Hi,
I try to build an app, which records geo positiones. I remark: if the app sleeps the recording is interupted. If I go back to app then the recording continue.
How can I solve the issue?

Rainer

  • In your custom Info.plist should do it for you:

    <key>UIBackgroundModes</key>
<array>
    <string>location</string>
</array>

    You might be able to add this to the tiapp.xml now. I'v been using custom files so long, I don't always remember what we can now set there.
