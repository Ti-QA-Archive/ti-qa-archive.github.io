  • Depending on what you are using it for, you can do this. I've got an app in the store that pulls down a .js file from a server I use for syntax highlighting code. Since if I change a keyword, I didn't want to resubmit the app so I have the app able to download the file.

    In my case, I have a 'Click here to update rules' button, so it's not a 'hidden' feature.
