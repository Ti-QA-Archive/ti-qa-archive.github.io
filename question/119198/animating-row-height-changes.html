Animating row height changes

Hi,

I'm trying to add a nice animation to my row. When the user clicks on the row I want it to increase in height to show some extra buttons. I've tried to add an animation to allow the row to increase in height slowly, but when the row is pressed nothing to happens. If I set the height directly on the row, the row height increases as expected, but it's jerky and ugly. Can someone please tell me what I am doing wrong or how to add an animation like this to a row?

function addActionButtonsToRow(row)
{
    var animation = Titanium.UI.createAnimation(); // create the animation
    animation.height = 110;
    animation.duration = 1000;
    row.animate(animation);  // set the animation going on the row

    animation.addEventListener('complete', function(){
        row.actionButtonView.show(); // show the view with the buttons
                                         when the animation is complete
    });
}

  • Ok, I've got a workaround for the time-being.

    I used the JavaScript "setTimeout" function.

    table.addEventListener('click', function(e)
{
    //We are going to increase the row height by '50'.
    //There are 5 iterations. We increase the height by '10' per iteration

    for(var i=0; i<5; i++) { //run the loop 5 times
    setTimeout(function() {
        e.row.height = e.row.height + 10; //this will increase the row height by 10
    },5 //'5' is the delay in milliseconds
)};

    Thanks! :)
