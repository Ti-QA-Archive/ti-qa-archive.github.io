Adding Spacing Between Toolbar Buttons with Alloy

Hi there-

How can I go about adding spaces between buttons on a toolbar using the Alloy framework? If I use the FlexSpace tag, buttons are placed to the far left and far right, and without the FlexSpace tag, they are crowded against one another. Is there a happy medium or margin property to add a nice buffer between buttons on a toolbar?

I am attempting to place buttons as such:

<Toolbar>
<Items>
<Button backgroundImage="/images/home.png" />
<Button backgroundImage="/images/settings.png" />
</Items>
</Toolbar>

    • ok, so if you do not want to use flexible space then the only way is to direct add buttons to toolbar as we used to add child controllers to its parent. 

      <Toolbar layout= "horizontal">
    <!-- <Items> -->
    <Button  left = "5" width="50" height="40" backgroundColor="red" title='hello' />
    <Button left = "5"  width="50" height="40" backgroundColor="green" title='test' />
    <!-- </Items> -->
</Toolbar>

      — commented April 2nd 2013 by Mitul Bhalia

    • That's the ticket…the issue I had prior was I was not commenting out the Items tags, so it was ignoring the left attribute. I used the above solution, and also added a top attribute to place the Button centered on the Toolbar appropriately.
