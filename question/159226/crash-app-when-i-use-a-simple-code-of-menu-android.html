    You have a couple of problems.

    1) If you name your top-level object in the XML file (in this case, that is your window and it has an ID of 'win', then you cannot use $.index to reference the object, must use the name. So your open code at the top of the js file should say $.win.open()

    Another problem you haven't found yet until you get the window open is… invalidateOptionsMenu is a method of the Activity attached to the Window, not the window itself. So, the code should be $.win.activity.invalidateOptionsMenu().

    • If you don't give your top level object an ID, (so in this case, if you didn't include 'id="win"' in your window definition, then you CAN use index to reference it. Since it has a name, you MUST use the name or it won't work. As a matter of practice, I ALWAYS put in the name, just makes it easier to understand.
