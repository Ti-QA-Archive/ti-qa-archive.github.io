Can I access platform/android/res/raw folder when using alloy?

Hi,

I would like to play an android notification sound using Ti.Filesystem.getResRawDirectory() like described here:
http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.Android.Notification-property-sound

However, since I am using alloy, there is no platform/… folder in my project. Is it possible to put my soundfile into the platform/android/res/raw folder when I am using alloy?

Thanks!

    Alloy in no way impacts the existing "platforms" folder. You can use it just like you did with traditional Titanium development. That means don't move it into the "app" folder, just leave it in your project root.
