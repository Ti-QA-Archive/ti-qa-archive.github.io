  • Android apps are never as fast as iOS apps with Appcelerator. The performance loss and bigger filesize is due to the V8 JS engine which has to be packaged with every Android app based on Titanium.

    With the latest SDK 3.2.x the performance has really improved compared to older SDKs (like 2.x). Hoping for Ti.Next to get a performance boost.
