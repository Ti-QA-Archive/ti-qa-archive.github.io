Hide ActionBar to make full screen window in Android

Hi,

My objective is to create a full screen window where the ActionBar will not be displayed. Following are the environment setup:

Titanium SDK : 3.3.0.GA
Android Target (Min): API Level 16 and above

If I create a window like this, then it works perfectly:

var win = Titanium.UI.createWindow({
    navBarHidden : true,
    exitOnClose : false
});

But as soon as I make the window transparent, the actionBar is getting displayed. For example:

var win = Titanium.UI.createWindow({
    navBarHidden : true,
    exitOnClose : false,
    backgroundColor : "transparent"
});

Any idea as to how can I resolve this issue. I have tried to hide the actionBar from code, but that needs to be called after opening the window, but for some moments the user can see actionBar before it gets hidden. I already have a Styles.xml:

<style name="Theme.MyTheme" parent="Theme.AppCompat.Light">
    <item name="android:editTextStyle">@style/editText</item>
    <item name="android:spinnerItemStyle">@style/MySpinnerItem</item>
    <item name="android:spinnerDropDownItemStyle">@style/MyDropDownItem</item>
    <item name="android:windowActionBar">false</item>
    <item name="android:windowNoTitle">true</item>
</style>

But that is not working, or I might have put in the information incorrectly. Can someone from the community suggest as to how to correctly hide the Android ActionBar.

    Hi Soumya Kanti Kar,

    Thanks for your concern.

    Till Ti SDK 3.2.3.GA you can remove ActionBar by setting <navbar-hidden>true</navbar-hidden> in tiapp.xml and setting the window property navBarHidden:true

    Since Release 3.3.0, due to changes to support the appcompat library, navBarHidden property has no effect. By default, the action bar is always displayed.

    For more, check the link

    To hide the action bar you can follow the official doc: Hiding the Action Bar

    Now for your problem, if you use window background color to transparent, then hiding actionbar won't work for you. To do that, add a style:

    <style name="Theme.NoActionBarTransparent" parent="@style/Theme.AppCompat.Translucent.NoTitleBar">
    <item name="android:windowActionBar">false</item>
    <item name="android:windowNoTitle">true</item>
    <item name="android:windowActionBarOverlay">false</item>
</style>

    And when you open the window use: window.open({theme:"Theme.NoActionBarTransparent"})

    Hope this will solve your problem. If so, then mark this answer as accepted.

    Thanks
