Styling Titanium UI

Hi everyone I'm just starting Appcelerator's Titanium Alloy.

How can I add styles to my controller/index.js here's the code:

var title = Titanium.UI.createLabel({
   title: "My Label",
   id: "myLabel"
});

I am trying this one on styles/index.tss

"#myLabel": {
 textAlign: "left",
 right: 15,
 color: '#000',
 font: { fontSize:20 },
 shadowColor: '#aaa',
 shadowOffset: {x:5, y:5},
 horizontalWrap: true
}

but unfortunately it's now working. Can anyone teach me on how to add styles to my Titanium UI Thanks.

  • Hello Allen,

    You can not add JS controller style in tss. you only add xml controller style in tss.

    For ie :

    index.xml

    <Alloy>
    <Window>
        <Label id=myLabel />
    </Window>
</Alloy>

    index.tss

    "#myLabel": {
 textAlign: "left",
 right: 15,
 color: '#000',
 font: { fontSize:20 },
 shadowColor: '#aaa',
 shadowOffset: {x:5, y:5},
 horizontalWrap: true
}

    if you use .tss file stlye in .js. it is possible using Apply Property.

    for ie of Apply Property :

    index.js

    var title = Titanium.UI.createLabel({
   title: "My Label",
   id : 'lbl'
});
title.applyProperties({id:'lbl', class:'myLabel');    // you can use "title" in ID

    See this for more information about applyproperties TIMOB-10346.

    I hope this is helpful to you. thanks.
