    So decided to use another answer, instead of comments, as I know this way you will get an email telling you your answer is here.

    Sorry its taken so long, but been a bit busy at work.

    Basically there is an error. The error is your code.

    here is the corrected version

    var win =Ti.UI.createWindow({
    backgroundColor : "white"
});
var label = Ti.UI.createLabel({
    top:"100dp",
    color:"black",
    text : "Test"
});
win.add(label);

var imageRequired = Ti.UI.createImageView({
        top : "160dp",
        left : "30dp",
        defaultImage : 'KS_nav_ui.png',
        width:Ti.UI.SIZE,
        height:Ti.UI.SIZE,
        borderColor : "red",
        borderWidth : 1,
       image : "https://www.vantagecircle.com/images/jitr/thumb/deals/4226_exotica.jpg"
});
win.add(imageRequired);
win.open();

    If you spot the difference Go for it… If not here is the issue.

    You have a redirect on the URL you were using to point to the actual URL of the image.

    Also you had set top & bottom, and also height, you should only set two of the three.

    Also the default image path was wrong, if your using a base application then that image exists in the main application directory (unless you have changed it).

    Putting this code into an app.js and running it works on Android without issue.

    Hope this helps

    T.
