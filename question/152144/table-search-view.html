    • Actually the search is working as I would expect …

      It is giving you the row selected of the current data within the tableview.

      How to handle this properly is to give your tablerow a custom property.

      i.e.

      
var row = Ti.UI.createTableRow({

    uniqueID: 'ABC'
})

      Then you can do in you event listener

      
e.source.uniqueID
