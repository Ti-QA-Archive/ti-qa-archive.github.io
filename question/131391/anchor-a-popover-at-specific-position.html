Anchor a popover at specific position

Do anyone know how to set the anchor point of a Popover(for iPad) at specific location in a view? What I am trying to do is make the popover arrow point to a specific location.

Besides, is it possible to attach the popover arrow to a slider thumb?
So that when the slider change, the popover will follow the thumb and move.

Appreciate if anyone can help.

  • Accepted Answer

    Normally you would name a view which gets used as the anchor point, and optionally specify a direction for the arrow if you want to force a given direction. The anchor view can be a button or slider, but to my knowledge not a slider thumb.

    To get it positioned on the current thumb location – or any other arbitrary location – I suspect you would need to create a small transparent view at that location then use it as the anchor point view.

    Tapping the slider thumb would normally also dismiss the popover, unless you named the slider in the array of objects for the passthrough view array.

    Even when you do that, it won't follow the thumb.
