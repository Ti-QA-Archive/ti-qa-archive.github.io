Hi, i am using alloy structre and developing mobile apps for ios and android.
I have images for my mobile app.

assets/myfolder/images

var ImageView = Ti.UI.createImageView({

        width : 25,
        heigth : 25
    });

Everything is ok, on adroid and on IOS too.

var CurrentImage = Ti.Filesystem.getFile("myfolder/images/icon.png");

    Hi Kerberos Kerbooo!!!

    Please check the following code and it woks well in my emulator. I hope, it will also work to your pc.

    var CurrentImage = Ti.Filesystem.getFile(Ti.Filesystem.resourcesDirectory,"am/KS_nav_ui.png");

if (!CurrentImage .exists()) 
{
    CurrentImage = "am/empty.png";
} 
else
{
    CurrentImage = CurrentImage.nativePath;
}

var ImageView = Ti.UI.createImageView({
        image : CurrentImage,
        left : 10,
        top : 10,
        width : 25,
        heigth : 25
    });
$.win.add(ImageView);
