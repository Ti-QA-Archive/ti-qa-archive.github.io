Global window background image

I want to set a background image across all windows in my app. So in my app.js I do Ti.UI.setBackgroundImage( '/images/background.png' );

This works great on the simulator but when I deploy it on my iPhone 4 I just get a black background color.

I have a background.png of 320x480px and a background@2x.png of 640x960px in my images directory.

I can get Ti.UI.setBackgroundColor( '#ff0000' ); to work on the device but no go with the image.

Any suggestions? Thank you.

  • I found the bug. Appcelerator doesn't handle paths like this: /images/iphone/background.png. I had to move the file to its parent so that the path would be: /images/background.png. Then it works :-/ Any comments on why before I accept my answer?
