Error on distribute app through Titanium

When I click on "Finish" in the distribute popup in Titanium, I got this error messages :

[ERROR] Error: Traceback (most recent call last):
File "/Users/MAC/Library/Application Support/Titanium/mobilesdk/osx/1.8.0.1/iphone/builder.py", line 869, in main
provisioning_profile = read_provisioning_profile(pp,o)
File "/Users/MAC/Library/Application Support/Titanium/mobilesdk/osx/1.8.0.1/iphone/builder.py", line 207, in read_provisioning_profile
f = open(f,'rb').read()
IOError: [Errno 2] No such file or directory: u'/Users/MAC/Library/MobileDevice/Provisioning Profiles/62BCD0F2-19FC-4170-89AA-CC452D8786C9.mobileprovision'

Any idea ?

  • The script is looking for your provisioning profile from the location /Users/MAC/Library/MobileDevice/Provisioning Profiles/62BCD0F2-19FC-4170-89AA-CC452D8786C9.mobileprovision

    but cannot find it. I suggest you download a new profile, register it with xcode (by double-clicking it after you have downloaded it. Then, update the Distribution screen dropdown with the location of the new one.
