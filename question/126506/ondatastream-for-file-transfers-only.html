ondatastream for file transfers only?

i'm trying to set the width of a progress bar using ondatastream but can't seem to log any info about the event - in fact it crashes my app.

i've checked around and only seem to see folks using ondatastream for up/downloading files/photos, so am i using it incorrectly when just trying show the progress of returning database info that populates a tableview (which works fine)?

dc.functions.page_data = function(data_url, tv, id) {
  var xhr = Titanium.Network.createHTTPClient();
  xhr.setTimeout(dc.data.xhr_timeout);
  // catch error
  var onerror = function() {
    alert('Sorry, could not connect to server');
  }
  // check readystate, then onload
  xhr.onreadystatechange = function(e) {
    if(xhr.readyState == 4) {
      if(Math.floor(xhr.status / 100) == 2) {
        var list = [];
        var data = JSON.parse(xhr.responseData);

        // DO SOMETHING WITH DATA

      }
    };
    //set up stream so we can use the progress property
    xhr.ondatastream = function(e) {
      dc.data.progress = e.progress;
      dc.data.prog_bar.width = parseInt(e.progress * 100, 10); // progress bar
      Ti.API.info(e); <-- THIS CRASHES APP
    };
    // open
    xhr.open('GET', dc.data.global_domain + data_url);
    // send the data
    xhr.send();
}

It looks like it fires once b/c my progress bar goes from 0 to 100 with no incremental progress and doesn't crash as long is i don't try to log the event. Am i missing another xhr component?

Any ideas? Thx!

    try changing this

    Ti.API.info(e);

    to this

    Ti.API.info(JSON.stringify(e));
