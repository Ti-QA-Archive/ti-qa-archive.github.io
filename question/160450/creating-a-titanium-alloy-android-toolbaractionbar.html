Creating a Titanium Alloy Android Toolbar/Actionbar

Hey Guys,

I really appreciate you taking the time to read this. My issue is that I have been trying to create toolbar style navigation with only two items a menu button and a logo, I would also like to stay across all screens.     Yep, I agree with Manuel's suggestion. Sounds like you should be using the native Android ActionBar in your app, and this is a good example of how to do it: https://github.com/adampax/AlloyActionJackson. Also note that if you up the targetSdk in your android manifest that any Ti.Android.Menus that you use will automatically be turned into action bars on appropriate Android versions.
