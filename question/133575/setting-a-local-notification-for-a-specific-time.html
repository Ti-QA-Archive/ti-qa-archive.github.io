Setting a Local Notification for a Specific Time

Is there a simple way to set a local notification (preferably for both Android and iOS, but mostly iOS) for a specific time?

For example if I want to set it to 10am everyday. I understand I can get the current time and then add the amount of milliseconds, but looking for a simpler way to set a local notification and have it repeat everyday.

    If your question is about how to easily set the internal values of the Date object passed in to the notification, try using date.js. It provides you with some nice syntax capabilities like:

    Date.today().set({hour:10, minute:0}); // 10:00 AM
