  • The problem, at least in my case, was due to the upgrade from 3.1.3 to 3.2.0. Here is how I fixed it:

    1. Move Applications/Titanium Studio to the trash
    2. sudo npm remove -g titanium
    3. sudo npm remove -g alloy
      Then download and install Studio 3.2.0
