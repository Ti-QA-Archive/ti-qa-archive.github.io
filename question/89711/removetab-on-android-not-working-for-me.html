removeTab() on android not working for me

So I open my app and I have two tabs. The 'login' tab and the 'new account' tab. Login is the default tab open. Upon successful login I call this:

Ti.App.addEventListener('grantEntrance', function(event)
{
    Ti.API.info(event);

    main.tabBarHidden     = true;
      main.title             = 'Welcome ' + event.name;
    main.url             = 'main_windows/main.js';
    main.name             = event.name;
    main.email             = event.email;
    mainTab.title         = 'Tab Title';
    mainTab.window         = main;

    tabGroup.addTab(mainTab);
    tabGroup.removeTab(loginTab);
    tabGroup.removeTab(accountTab);
});

On the iPhone this works great. The 'login' and 'new account' tabs disappear and now I have no tab bar since I set the main.tabBarHidden property to true. On the droid, it adds the new tab and sets all the properties but one.

It doesn't set the tabBarHidden property to true (I can see login, new account and now 'tab title') and the two tabs I want to remove aren't removed. Any ideas why this would work on the iPhone and not droid?
