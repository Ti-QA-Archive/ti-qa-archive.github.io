How do create 2 RightNavButtons?

Hi,

I am able to create 2 buttons used for a RightNavButton.

The problem is obviously that I cannot have 2 of them.

I tried looking at the createButtonBar feature. Indeed, this would solve the problem except:
a) My single buttons have images, the create Button bar seems to only take text labels.

b) I don't like how it works. For the createButtonBar, I'd have to use a switch command - I don't want to.

(long story).

Any ideas?

    hey E B,

    instead of custom navBar if you want to set two button in the actual navBar then it's possible..

    follow the code

    var view = Ti.UI.createView({

});

var btn1 = Ti.UI.createButton({
    title :'test',
    right : 0,
    width : 50,
    height : 30
});
var btn2 = Ti.UI.createButton({
    title :'test',
    right : 50,
    width : 50,
    height : 30
});
view.add(btn1);
view.add(btn2);

win.setRightNavButton(view);
