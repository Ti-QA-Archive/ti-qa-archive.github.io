private static final String REGISTER_EVENT = "registerC2dm";

    • Hi Joonas, it works with V8! by unzipping the /dist/com.findlaw.c2dm-android-0.1.zip from the c2dm project, and move the unzipped files from the folder /modules/android/com.findlaw.c2dm/0.1 to the project folder of my app: /modules/android/com.findlaw.c2dm.

      Other things I saw:

      1. Use the latest version of the Titanium SDK (1.9, master branch) from: http://builds.appcelerator.com.s3.amazonaws.com/index.html. 1.8.0.1 is missing the method 'createPendingIntent', more info: http://developer.appcelerator.com/question/129961/1801—creatependingintent-missing-from-tiandroid

      2. When you get an error "KrollFunction has no method call", you have to change the file /src/com/findlaw/c2dm/C2dmModule.java:

      from:
      messageCallback.call(getKrollObject(),data);

      to:
      messageCallback.callAsync(getKrollObject(),data);
