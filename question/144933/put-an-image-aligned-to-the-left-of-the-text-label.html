    Hi

    Any easy way to have two views side by side when adding them to a scrollView with the contents being added underneath is the following;

    var scrollView = Ti.UI.createScrollView({
    contentHeight: Ti.UI.SIZE,
    contentWidth: Ti.UI.SIZE,
    height: Ti.UI.FILL,
    layout: 'vertical',
    showHorizontalScrollIndicator: true,
    showVerticalScrollIndicator: true,
    width: Ti.UI.FILL
});

var viewFakeRow = Ti.UI.createView({
    height: Ti.UI.SIZE,
    layout: 'horizontal',
    width: Ti.UI.FILL
});
scrollView.add(viewFakeRow);

var img = Ti.UI.createImageView({
    height: Ti.UI.SIZE,
    image: 'somefile.png',
    width: Ti.UI.SIZE
});
viewFakeRow.add(img);
var lbl = Ti.UI.createLabel({
    height: Ti.UI.SIZE,
    left: 10, // pushes away from the image
    text: 'some caption',
    width: Ti.UI.SIZE
});
viewFakeRow.add(lbl);

    Note the use of the "fake view" and the layout properties to set how the children are position within their parent views.

    • Hi

      I think this is a simple one. Your left property appears as if you may worked out how far you need to be from the left edge of the screen, however when you use the layout property on the parent, left, right, top and bottom become more like margins.

      So you you want to have your label 10 pixels to the right of the image, make the left property 10.

      The order of children being added to the parent is very important when used with the layout property. As they are displayed one after each other - so your second order is best.

      var ev1 = Titanium.UI.createLabel({
    color:"#FFF",
    text:'Giovedì 18 Ottobre 2012\nOre 21.00\nGilles Apap & Mario Brunello',
    font: { fontSize: 14 },
    textAlign: Ti.UI.TEXT_ALIGNMENT_LEFT,
    width: 250, 
    height: 60,
    left: 10, // reduce this as think more as a margin
    top: 5
});

      I would also suggest a second fake view to handle your display. Try and break your layouts down into columns when using layout 'horizontal'.

      • Col 1 = Image
      • Col 2 = Content Container

      Then inside Col 2 you would add

      • Label
      • Button

      Otherwise things go places you would not predict.

      A simple example;

      var viewFakeRow = Ti.UI.createView({
    height: 120,
    layout: 'horizontal',
    width: 320
});

var img1 = Ti.UI.createImageView({
    image:'/images/thumb/8novembre12.jpg',
    top: 10,
    left: 10
});
viewFakeRow.add(img1);

var viewFakeContent = Ti.UI.createView({
    height: Ti.UI.SIZE,
    layout: 'vertical',
    left: 10,
    width: Ti.UI.SIZE
});
viewFakeRow.add(viewFakeContent);

var ev1 = Titanium.UI.createLabel({
    color:"#FFF",
    text:'Giovedì 18 Ottobre 2012\nOre 21.00\nGilles Apap & Mario Brunello',
    font: { fontSize: 14 },
    textAlign: Ti.UI.TEXT_ALIGNMENT_LEFT,
    width: 250, 
    height: Ti.UI.SIZE,
    left: 0,
    top: 5
});
viewFakeContent.add(info1);

var info1 = Titanium.UI.createButton({
    color:"#790000",
    title:'Maggiori Informazioni',
    font: { fontSize: 14 }, 
    left: 0,
    width: 160, 
    height: 20,
    top: 5
});
viewFakeContent.add(info1);

      This should get you closer. You will need to tweak it.

      BTW - I use the phrase 'fake' to point out the view is used a container, you can call them whatever you like.

    • This is my solution:

          var scrollView = Titanium.UI.createScrollView({
            contentWidth: 320,
            contentHeight:'auto',
            showVerticalScrollIndicator:true,
            layout:'vertical'
    });

    var viewFakeRow = Ti.UI.createView({
        height: 60,
        top: 0,
        layout: 'composite'
    });

    var ev1 = Titanium.UI.createLabel({
        color:"#FFF",
        text:'Giovedì 18 Ottobre 2012\nOre 21.00\nGilles Apap & Mario Brunello',
        font: { fontSize: 14 },
        textAlign: Ti.UI.TEXT_ALIGNMENT_LEFT,
        width: 250, 
        height: 60,
        left: 100,
        top: 5
    });

    var img1 = Ti.UI.createImageView({
          image:'/images/thumb/8novembre12.jpg',
          top: 10,
          left: 10
    });

    var info1 = Titanium.UI.createButton({
        color:"#790000",
        title:'Maggiori Informazioni',
        font: { fontSize: 14 }, 
        left: 120,
        width: 160, 
        height: 20,
        top: 10
    });

    scrollView.add(viewFakeRow);
    viewFakeRow.add(img1);
    viewFakeRow.add(ev1);
    scrollView.add(info1);
