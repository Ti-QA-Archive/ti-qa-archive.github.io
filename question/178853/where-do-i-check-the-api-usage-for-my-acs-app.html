where do I check the API usage for my ACS app?

I've been wondering for a while how I can keep track of the APIs I've used in my ACS application.

I've been polling every 5 seconds an API request in my ad hoc app (this is a chat), and as far as the documentation says I only have 250.000 API calls per day for free. Where can I read my current usage?

From what I understand under 'Monthly usage' I only see the push notifications that my app used.

    On the top right corner of your screen you can see a drop down icon with profile icon onto it, click onto that drop down and click on "My App" or directly go to :https://my.appcelerator.com/apps

    there you can see all your apps, click on the particular app for which you want to get details by clicking on "APP Details" there you can see the application details like: Emails, Push, API calls and other details
