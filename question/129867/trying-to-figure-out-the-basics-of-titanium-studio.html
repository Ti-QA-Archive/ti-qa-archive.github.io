    I apologize that the forum software hasn't made this any easier. here's what works for me, just as a point of reference.

    1. Download https://github.com/appcelerator/KitchenSinkiPad/downloads as .zip
    2. Extract the .zip file
    3. Import > Import Existing Titanium Project (though Existing Projects into Workspace seems to work too)
    4. Press the debug or run button in the bar inside Project Explorer or App Explorer.

    Two scenarios to try:

    For no files, have you tried right-clicking on the project in Project Explorer? If you right click on the project, see properties, and find the location on disk, are the files there?

    For "No existing launches", are you clicking the run button in the main toolbar and not the one inside App Explorer or Project Explorer? The project/app explorer run buttons automatically set up default launch configurations for you.

    If you'd like to post an image, most people post them to some external site like http://skitch.com/ and link them in that way.
