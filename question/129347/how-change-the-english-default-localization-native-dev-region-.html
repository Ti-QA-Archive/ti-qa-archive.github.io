  • What you want to do is create a il8n directory in your app root. In there, you create directories for languages… ie. en/ es/ etc.

    Inside these directories you create a xml file named strings.xml

    Here is an example…

    <?xml version="1.0" encoding="UTF-8"?>
<!-- ENGLISH -->
<resources>
    <string name="Cancel">Cancel</string>
    <string name="Date">Date</string>
    <string name="Done">Done</string>
    <string name="Email">Email</string>
</resources>

    Then within your app you can call the strings by…

    var cancel = Ti.UI.createButton({
    title:L('Cancel')
});

    Now based on your devices language settings, the app will pull from the strings.xml :)
