Mapview shows map in emulator, but not on device

Hi, I'm testing my app on an iphone. It makes use of the mapview + annotations. In the emulator all works fine, but on the actual phone I see the annotations, but the underlying map is not shown (instead it shows the grey grid).
I am on an active internet connection, annotations and location services work, my only problem is the map not loading properly for some reason.
Any help would be greatly appreciated!

var mapview = Titanium.Map.createView({
    mapType: Titanium.Map.STANDARD_TYPE,
    region: { latitude: Ti.App.latitude, longitude:Ti.App.longitude, latitudeDelta:delta, longitudeDelta:delta },
    animate:true,
    regionFit:true,
    userLocation:true
});
