    Hi

    Sounds like you have not set the api levels to force the app to use 4.x as a minimum (3.x is the actual minimum though), this is done in tiapp.xml.

    Amend your tiapp.xml to include the following entries if your android section is empty (or essentially empty), be aware that if you have a valid android you should blend the information below with what you have.

    <android xmlns:android="http://schemas.android.com/apk/res/android">
    <tool-api-level>14</tool-api-level>
    <manifest>
        <uses-sdk android:minSdkVersion="14"/>
        <uses-sdk android:targetSdkVersion="16"/>
        <application
            android:screenOrientation="portrait"
            android:theme="@android:style/Theme.Holo"
        />
    </manifest>
</android>
