While it does works fine if you have set up an integration from your iOS6.0 phone to FB, while it fails to fallback to something useful when this has not been done.

This is as easy to reproduce by following the steps on this page:
http://docs.appcelerator.com/titanium/latest/#!/api/Modules.Facebook
