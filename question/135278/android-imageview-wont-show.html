Android ImageView won't show

Hi,
I've made an application which is meant to work on both Android and iPhone. It works on iPhone, but not on Android. The first shown view has nothing but an ImageView. It's setup like this: 

var iV = Ti.UI.createImageView({
    image: 'wpage.jpg',
    width: 'auto',
    height: 'auto'
});

var imageRatioFP=Ti.Platform.displayCaps.platformWidth/iV.getWidth();
iV.setHeight(imageRatioFP*iV.getHeight());
iV.setWidth(imageRatioFP*iV.getWidth());

self.add(iV);

I've put the same images as I got in the iPhone folder in all the folders inside the Android folder including the Adroid folder itself.

    when you set 'auto' to height and width in android you can not get it back by getHeight()

    so use 

    iV.toImage().height;
iV.toImage().width;
