  • Hello Roseanne Carroll,

    For searching on dataToPass array properties you can use my following code. I think, it will helpful to you.

    var tableView = Titanium.UI.createTableView({
    search : searchBar,
    filterAttribute : ‘dataToPass’
});

    And for more information you can read about TableView filterAttribute
