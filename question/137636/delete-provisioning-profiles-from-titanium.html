  • If you mean the TiStudio: it should remove them automatically (the last version of TiStudio) since it is reading them from xcode directly, try hitting "refresh" after "build on iOS device" or check `Settings -> Properties -> Aptana Studio -> Titanium -> iOS Platform" there is also a "refresh" button both for DevCerts and also for Provisioning Profiles.
