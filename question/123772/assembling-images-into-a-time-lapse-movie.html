    If you are looking to do this directly on the iphone, it will require a custom module written in objective C. At this time, Titanium does not have native support for the iOS AVFoundation APIs which are needed to do this. You can read more about creating iOS modules for Titanium here.

  • btw: A quicker way I use for getting files off your app is to enable File sharing for the app, using the Info.plist and setting UIFileSharingEnabled to YES.

    Then, if you get your app to save the photos in the applictionData folder (rather than the album) you'll be able to drag them out via iTunes while it's plugged in.
