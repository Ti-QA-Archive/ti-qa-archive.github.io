  • You need to tune the default shake properties in the tiapp.xml. The stock values require you to use unnaturally hard shaking motions to activate the event.

    Add the following to your tiapp.xml. You might want to tune these more than illustrated here. But, these values work reasonably well on my Galaxy Nexus and other devices I've tried.

    <!-- how hard a shake is needed (seems to have little effect) -->
<property name="ti.android.shake.factor">1.3</property>

<!-- how long must it continue, default is 1000 -->
<property name="ti.android.shake.active.milliseconds">250</property>

<!-- how long between shakes, default is 500 -->
<property name="ti.android.shake.quiet.milliseconds">200</property>

    These are documented in the tiapp.xml and timodule.xml guide.
