Distribute archive in XCode Organizer never finishes

I have an iPhone app (built with Titanium 3.0.0 GA, XCode 4.6.1, iOS SDK version 6.1) ready for upload in iTunes Connect. I have used "Distribute - Apple iTunes Store" from Titanium to create an XCode archive. Selecting this archive in the XCode Organizer, the "Validate…" function works (after logging in to my iTunes account) and updates the status of the archive to "Passed Validation". When I choose "Distribute…" everything looks OK until the final stage, where it seems to be stuck in "Your application is being uploaded" forever (the progress bar looks the way it does before it starts showing completed percentage). Does anyone have an idea of what might be wrong?

The only thing I think I missed when following this guide, was that I tried with "Distribute…" once before using "Validate…".

When I made the app ready for upload in iTunes Connect, there was info about using "Application Loader" (rather than XCode Organizer) for uploading the app, but this app does not seem to want to open an XCode archive. What is the role of "Application Loader" in comparison to using "XCode Organizer"? Is it for doing things that Titanium already has done with the Distribute function?
