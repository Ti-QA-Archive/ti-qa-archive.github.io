    • What I would do is unzip the ti.box2d-iphone-0.9.zip from here to your /library/application support/titanium/ directory. Then add:

      <modules>
    <module version="0.9">ti.box2d</module>
</modules>

      to your Tiapp.xml

      That should get you up and running with Box2d in a jiffy. If you're making edits to the source files and you are wanting to build your own version, I would suggest downloading SmartGit here and pulling down the git source from that. I had trouble originally using the download button from github. SmartGit worked a lot better.

    • Thanks Lance. That is a good starter for anyone getting their feet wet with Box2D on Appcelerator. Since my post 4 months ago, I have come a long way with Box2D and there is a whole bunch of stuff you can do, even with the beta implementation.

      Another small tip is getting access to the view attached to the object. If we create a left wall:

      var leftWall = Ti.UI.createView({

        backgroundColor:‘#fff’,

        width:2,

        height:200,

        bottom:0,

        left:0

});



var leftWallRef = world.addBody(leftWall, {

        density:12.0,

        friction:0.3,

        restitution:0.4,

        type:‘static’

});

      You would gain access to the view for that object by:

      leftWallRef.view.backgroundColor

      Hope this helps anyone working with Box2D for the first time.
