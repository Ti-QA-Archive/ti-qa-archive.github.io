  • I'll leave Stephen to follow up on getting this to work in the command line. But you can change your build target in Studio, which should get Studio builds working again. Two options:

    1) For all projects – choose Run, Run Configurations. In the left pane, select all of the items under the Titanium OS Simulator heading (these are all the run configurations saved from previous builds of your various apps). Click the red-X delete button. Click Close. Then, build your project. A new run configuration will be created to match your system's current configuration.

    2) For a single project – choose Run, Run Configurations. Find the run config for this project and change the iOS SDK and/or Simulator as necessary and click Apply. Click Run to run the project using the new settings.
