Grouped TableView not Rendering Correctly with iOS4

I just updated the SDK to 4.0, and I also flashed my iPhone with iOS4. All of my TableViews that are 'grouped' (i.e. style: Titanium.UI.iPhone.TableViewStyle.GROUPED) are not rendering as such. They are rendering as normal table views.

But, this issue only happens about 50% of the time. So, half the time when I open my app, the tableview renders correctly, and half the time it renders incorrectly.

This is happening on both the Simulator and the device.

This does NOT happen using any SDK 4.0 on an iPhone OS 3.x device.

This does NOT happen using any SDK other than 4.0.

Has anyone else seen this issue? Is there any known fix for this issue.
