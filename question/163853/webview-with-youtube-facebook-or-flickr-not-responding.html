WebView with YouTube, Facebook, or Flickr not responding

Hello,

I am working on a project for a client where they would like their Youtube, Facebook, and Flickr web pages in the app in a webview (I tried to get them to load content in a tableView but there was heavy pushback). I am now running into an issue where the pages will load but you cannot interact with the pages themselves.

The only problem I can think of is that there is a swipe gesture that I have running through the app to access a side menu, will I need to remove this? FYI, I have also tried to set the touch gesture to false but this does not work either.

Please help if possible…Thanks

I am running Titanium 3.2.2, on iOS 7, and all compatible Android Versions on an iMac.

  • Never mind I have solved this:

    For anyone else's benefit if you are using webviews and have multiple custom UI events it will lock up the WebView this is because they are memory hogs (which is why you should always push back against using them). To get around this try to load the event into a modal or new window of any kind. This will allow you to add some custom events in.
