  • Well, your best approach would be to have a parent View with a vertical layout. You will then simply add new Views (one for each brick) that can each have different heights. Hence giving you the effect you are looking for.

    Now this approach will work for a single column as well as multiple columns. Just create additional parent Views and add then to your Window.

    • That is why you will need to a create a whole new main view that will contain your second column elements.

      Something like:

      var mainLeft = Ti.UI.createView({
    layout: 'vertical',
    width: '50%'
});
var mainRight = Ti.UI.createView({
    layout: 'vertical',
    width: '50%'
});

      And you then add your child views to the desired column.

      Hope that made it clearer for you.
