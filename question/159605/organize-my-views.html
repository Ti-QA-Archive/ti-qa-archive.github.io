  • You can adjust for screen size in a number of different ways.

    1. You can have your elements be relative in size to the size of the screen by using percentages or dp for your height and widths.
    2. You could change your primary container to a scrollView so that the items being displayed off screen can seen when the user scrolls up.

    Some reading:

    http://docs.appcelerator.com/titanium/2.0/#!/guide/Supporting_Multiple_Platforms_in_a_Single_Codebase
