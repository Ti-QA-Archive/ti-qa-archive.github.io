Global Font Color?

There are certain elements (Label, TableView and probably a few others) which by default has gray color. I have to explicitly set the color element to black (#00000) so that the color is well.. black. Somehow I feel that there is a global font color somewhere that I can set once and it affects all others.

  • You can set your own style by creating function to create element(Label).
    for example

    function CustomLabel(param){
    var temp = Ti.UI.createLabel();
    // apply default setting.
    temp.color = 'black' // your default setting.

    //apply parameter value to newly created Label.

    return temp;
}

var MyCustomLabel = new CustomLabel();

    or store color variable in to one file and import it into file where you want to use it.

    for example:

    Globle.js

    var MyColor = "Black"

    app.js

    Ti.include("Globle.js");

var temp = Ti.UI.createLabel({
    text  : "Your text",
    color : MyColor
});
