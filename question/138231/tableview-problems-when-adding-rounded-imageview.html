tableview problems when adding rounded imageview

this is my code

// create table view data object
var data = [];

for (var c = 0; c < 100; c++) {
    var row = Ti.UI.createTableViewRow({

    });

    var label = Ti.UI.createLabel({
        text : 'Cell at row ' + (c + 1),
        color : '#111',
        shadowColor : '#900',
        shadowOffset : {
            x : 0,
            y : 1
        },
        textAlign : 'left',
        left : 130,

  • Using borderRadius is known to cause performance issues. You can minimize some of the performance hit though by adding a className to your table view rows. So, your code would become

    var row = Ti.UI.createTableViewRow({
    className: "myRow"
});
