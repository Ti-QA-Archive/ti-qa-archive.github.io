Global Variables

Hi,

I need some guidance with respect to global variables.

I've set the following integer: 

Ti.App.Properties.setInt('myInteger'), 1);

Can someone tell me how to increment this by 1?

Is it myInteger+1?

Thanks

  • Accepted Answer

    Ti.App.Properties.setInt( 'myInteger', 1 );

var myInteger = Ti.App.Properties.getInt( 'myInteger' );
myInteger = myInteger + 1;

Ti.App.Properties.setInt( 'myInteger', myInteger );
