    Are your trying to use this library in a webview? That should be the only time you might experience a problem with the processing of JS files. JS files used within appcelerator should not be having an issue by being packed.

    If this is in a webview, you can change the extension from .js to something like .ljs (local js) and the compiler wont touch them.
