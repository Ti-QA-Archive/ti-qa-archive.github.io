ACS API for push notification

Hi everyone, I develop an App using ACS for push notification. This app manages news from different city and a user can subscribe to a channel. In my case one channel collects news from one city, for example an user in city1 will subscribe to city1_channel, an user in city2 will subscribe to city2_channel and so on.

When I want to push one notification I have to login in "My portal" on Appcelerator, select app, go on "Push notification", go in "Show Console", create notification and then send it.

Is there an API or something similar that I can use to send push notification without go everytime in "My portal"? In this way I can delegate someone in every city to push notification only for his city.
