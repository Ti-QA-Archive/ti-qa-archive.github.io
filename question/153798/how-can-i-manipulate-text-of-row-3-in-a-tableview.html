How can I manipulate text of row 3 in a tableview?

Say I have a tableview with 10 rows, and inside each row are two labels (labelA and labelB).

If I wanted to change the text of labelB in row 3, what would be the call to do that?

Thanks!!

  • You can always do a trick with custom variables inside any object in Titanium. By example:

    // Create a label
var label1 = Ti.UI.createLabel({text:'hola'});
//Create a row and add a reference (in JS level to the label).
var row = Ti.UI.createTableViewRow({
height:'auto',
label1:label1,
});
// Add the label into the row, at the native level (reflected in graphic UI).
row.add(label1);
//then, you can do magic like":
row.label1.text= "modified text";

    The nice thing? It won't be more expensive in terms of memory! You only have an extra reference in the JS level to the same native component.

    Just be careful when you want to remove them from memory, you will need to do:

    row.remove(row.label1);
row.label1=null;
label1=null;
row = null;

    Best,

    Mauro
