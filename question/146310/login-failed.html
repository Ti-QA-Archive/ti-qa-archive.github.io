    • OK - weird.

      If things work with SDKs other than 3.0.0 then it most likely points to the specific SDK that is causing an erroneous error.

      Things to check; (some you most likely have already tried.

      • Menu; Help > Titanium Studio > Run Diagnostic Test… does everything listed look ok? Does the 3.0.0 SDK reside in the location it suggests on your computer?
      • Menu; Check for all updates (both entries)
      • Menu; Help > Install Specific Titanium SDK…, Choose "Install from Update Stream", is 3.0.0 already installed? If not install it.
      • Remove the 3.0.0 SDK and re-install it again.
