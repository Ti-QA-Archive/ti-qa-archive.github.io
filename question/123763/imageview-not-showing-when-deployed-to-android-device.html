ImageView not showing when deployed to Android Device

The below image view shows up fine on an iPhone simulator, but does not show up when the app is deployed onto an Android Phone. The issue doesn't seem to have to do with the particular website. How can this work on Android?

//tb is an object created
tb.select.ui.gamePage.gameImage = Ti.UI.createImageView({
        image: 'http://www.worldwildlife.org/species/photography/Tiger%20Album/WWFImgFullitem4689.jpg',
        top:tb.device.percentHeightToPixels(.01),
        left:tb.device.percentWidthToPixels(.6),
        height:tb.device.percentWidthToPixels(.35),
        width:tb.device.percentWidthToPixels(.35)
    });
