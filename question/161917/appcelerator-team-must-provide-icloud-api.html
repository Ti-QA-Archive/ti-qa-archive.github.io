  • Hello Moiz!

    How to use iCloud?

    You can use iCloud in any supported device (i.e., according to apple's doc here) should be: iPad, iPhone, iPod touch with iOS 7+.

    The support for iCloud has been in the Mobile SDK since version 1.8. In order to use iCloud, you will need to setup your distribution profile to actually support iCloud in your app (you can read more about this configuration in the Provisioning profile configuration here).

    How to sync files and Database on cloud?

    You will need to use remoteBackup. Example:

    // Recursive function to disable iCloud for a folder hierarchy
exports.disableBackupForFolder = function(folderName) {

    function walk(folder) {
        var dir = Ti.Filesystem.getFile(folder);
        var dir_files = dir.getDirectoryListing();
        for (var l = 0; l < dir_files.length; l++) {
            var file = Ti.Filesystem.getFile(folder, dir_files[l]);
            file.setRemoteBackup(false);
            var nativePath = file.nativePath;
            if (nativePath.lastIndexOf('/') == nativePath.length - 1) {
                walk(file.nativePath);
            }
        }
    }
    walk(folderName);
};

    That will recursively set every file inside the directory to be "backup-able". I took this code from the answer here for a question like "how to backup an entire directory". In short: you can't. You need to backup each file.

    How to hide stuff from icloud?

    Just setRemoteBackup to false. Check the docs here.

    Where to put a db?

    Just put it in any readable directory.

    What if I want that db to be synced with icloud?

    Get the file object, and then with that, set the setRemoteBackup to true. See the sample above.

    How to force a restore after a reinstall of the app?

    This is a good one. There is a bug related to this, here. I will ping the Doc team.

    Let me know how it goes with this information.

    Mauro
