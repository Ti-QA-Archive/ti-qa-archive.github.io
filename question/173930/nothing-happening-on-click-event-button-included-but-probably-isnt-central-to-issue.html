    First of all the <button …> and </button> tags should have UCFirst. So <Button …> </Button>. Next to that I'd advice to use the onClick tag inside the button. for example:

    Alloy:

    <Button id='whosit' onClick='someFunction'>Click me!</Button>

    Script:

    function someFunction(e)
{
    alert('hello world!');
}

    but I suspect that:

    $.whosit.addEventListener('click', someFunction);

    should also work.

    Also, your posted code is unreadable, specifically the alloy part…
