Longpress for 3 Seconds

In my application user has to click a button for 3 seconds to view his/her profile. Here i am using onLongpress and onLongclick events but the timestamp is very less. So how can i achieve this ?

    You can achieve this by attaching a touchstart event on the button, and then set a timeout for 3 seconds to do the action for viewing his/her profile

    var timeout;
var viewProfile = function() {
    clearTimeout(timeout);
    ...

};

var longpress = function(e) {
    timeout = setTimeout(viewProfile, 3000);
}

btn.addEventListener("touchstart", longpress);
