Tab Creation Loop

Hello All,

Completely new to Titanium and Javascript development, I'm hoping for a little bit of guidance. I'm trying to build a basic app with 5 tabs, and I'm trying to find a better approach then to brute force the creation of the five tabs. Basically I created an array of 5 items, one is the tab name the other is the tab window file. I want to loop through this array, and create my five tabs. I'm doing something wrong, the program crashes. Just wondering if someone might be able to give me a call pointers, tips for troubleshooting.

Thanks in advance for the advice.

Here is the code:

(function(){
myapp.ui = {};

myapp.ui.createApplicationWindow = function(){
    // create tab group

    for (var i = 0; i < tabArray.length; i++) {

    • even easier.

      When you create a component, you pass a args object to it, i.e:

      var win = Ti.UI.createWindow({
    top:0, left:0, right:0, bottom:0,
    backgroundImage:'/images/whatever.jpg'
});

      the idea is keep that args object separated from component creation. As easy as assign an identifier to these properties.

      var styles = {};

styles.fullWindow = {
    top:0, left:0, right:0, bottom:0,
    backgroundImage:'/images/whatever.jpg'
};

var win = Ti.UI.createWindow(styles.fullWindow);

      If you create a "styles" file with all those definitions, you have separated theme file for your app.

      Another sample you can take a look is my project TitanTricks. Look how is defined the file /app/ui/styles.js, is the same concept explained here, but more sofisticated.
