Admob module in Android not working...help!

I have the Marketplace Admob module, installed like all of my others, but I keep getting the following message.


[ERROR] /Users/admin/Documents/Titanium Studio Workspace/ProjectName/build/android/AndroidManifest.xml:13: error: Error: String types not allowed (at 'configChanges' with value 'keyboard|keyboardHidden|orientation|screenLayout|uiMode|screenSize|smallestScreenSize').

My tiapp looks like

<android xmlns:android="http://schemas.android.com/apk/res/android">
    <modules>
        <module platform="iphone" version="0.1">com.paperwasp.calendar</module>
        <module platform="iphone" version="1.1">ti.admob</module>
        <module platform="android" version="2.0">ti.admob</module>
    </modules>

I'm compiling with Google API's 2.3.3. I read that you have to use a larger API, but then the simulator looks like a giant tablet, and I can't test. Anyone else?

  • I FINALLY got it working. Here are the steps you have to do.

    1) Download the 4.0.3 SDK from the Android tool and install the SDK and API's
    2) Download the latest admob sdk from Android | https://developers.google.com/mobile-ads-sdk/download
    3) In the admob module you downloaded, under the 2.0, replace the JAR file in the "lib" folder with the one you downloaded
    4) In Studio, under Run Configurations, choose "Google APIs Android 4.0.3
    5) In your tiapp.xml file, make your android section look like this

      <android xmlns:android="http://schemas.android.com/apk/res/android">
        <manifest android:versionCode="1" android:versionName="1.0"/>
        <uses-sdk android:maxSdkVersion="13" android:minSdkVersion="8" android:targetSdkVersion="13" />
     <tool-api-level>13</tool-api-level>
    </android>

    After that, all works
