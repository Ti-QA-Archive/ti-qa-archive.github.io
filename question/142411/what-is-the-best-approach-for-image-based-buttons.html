What is the best approach for image-based buttons?

Hi.

I need to create some buttons which will use a background image and foreground image (this is so I can create a generic background and have more specific artwork for the foreground). This is for a iPhone and Android mobile app.

The background image should have a hover (touch) state.

I can see from the different ways of creating UI elements that I could do this several ways, so I want to ask what people think is the best approach - or their preferred approach?

Thanks.

  • I have never used a button with both the foreground image and background image but here how I would try to do it - Create ImageView with the foreground image and add it to the button object that will have the background image. The event listener will be added to the button.
