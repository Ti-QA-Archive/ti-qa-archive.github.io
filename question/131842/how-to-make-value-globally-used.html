    • The code would go something like:

      Ti.App.row_number = 'something';

      This will make the variable global to the entire application.

      — commented February 8th 2012 by Christian Brousseau

    • Although I firmly believe that extending the Ti namespace with your own value is a bad idea as it creates the "potential" for a future clash if Appcelerator choose to use the same name.
