I have no idea what's going on. Do iPads store crash/dump info someplace on the device? Do I have to write exception handling code to wrap all the code in my app to figure this out? What can cause an app to completely unload like this, instead of throwing some kind of error message? If I were to put in a try/catch, would I be able to send error info back to the server in the catch block? I'm assuming so, but puzzled why an ios app would just unload without any kind of message.
