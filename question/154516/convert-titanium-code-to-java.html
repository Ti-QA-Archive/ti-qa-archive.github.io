  • Accepted Answer

    Both iOS and Android compile to native code which you can in theory play with and change. BUT there are two drawbacks

    1. You would lose all your modifications if you needed to return to your original javascript source to change something
    2. they both still rely on the closed-source Kroll runtime so they cant be considered exactly the same as a native app built from scratch
