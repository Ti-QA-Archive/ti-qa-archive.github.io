How to prevent vertical bounce in a horizontal scrollview?

Hi all,

I'm facing a rather peculiar problem and I hope someone can help me out here. I have a scrollview within which I have some views that are scrollable horizontally. Since I require these views and the scrollview to resize within the page based on orientation, I need to use % values or Titanium.UI.FILL. The problem is when I use Titanium.UI.FILL, a vertical bounce is present in the horizontal scrollview. Once I change the height of the scrollview as a definite value the problem seizes. Any way to achieve this without using definite values? Am I doing something wrong here? Following is a sample of talking about.

var rootWindow  = Ti.UI.createWindow({
    layout: 'vertical',
    backgroundColor: 'white'
});

var boxesView = Titanium.UI.createView({
    backgroundColor: '#141414',
    opacity: 0.7,
    top: 50,
    height: '53%',
    width: Titanium.UI.FILL,
    layout: 'vertical'
});

var boxesHorizontalScrollView = Titanium.UI.createScrollView({
    backgroundColor: 'red',
    height: Titanium.UI.FILL,    // 400         //Replacing by 400, a definite value, vertical bounce is solved
    contentWidth: 'auto',
    scrollType: 'horizontal',
    layout: 'horizontal',
    showHorizontalScrollIndicator: true
});

var boxesHeaderView = Ti.UI.createView({
    backgroundColor: 'blue',
    height: '10%',
    width: Titanium.UI.FILL
});

var headingLabel = Ti.UI.createLabel({
    text: "BOXES",
    color: 'white',
    font: {
        fontWeight: 'bold',
        fontSize: 20
    },
    top:10,
    bottom: 10,
    textAlign: Titanium.UI.TEXT_ALIGNMENT_CENTER
});


boxesHeaderView.add(headingLabel);

boxesView.add(boxesHeaderView);
boxesView.add(boxesHorizontalScrollView);

var boxes = [];
for (var i = 0; i < 7; i++){

    // BOXES **********************
    boxes[i] = Ti.UI.createView({
        top: 30,
        left: 12,
        right: 12,
        height: '70%',
        width: 250,
        layout: 'vertical',
        zIndex: 5,
        backgroundColor: 'black'
    });

    boxesHorizontalScrollView.add(boxes[i]);
}

rootWindow.add(boxesView);
rootWindow.open();

If you run the above code you will see that the boxes in the scrollview have a vertical bounce along with the normal horizontal scroll.

Any suggestions would be much appreciated. Thanks!

  • Hey Everyone -

    The problem that you are encountering with horizontal scrollViews is that *its height must be defined outright, so long as its content is not taller than the fixed container's height.

    So instead of defining the height of the horizontal scrollView by the height of its children, it is best to define the height of the horizontal scrollView, and then set the height of its children to '100%'. *

    Here is an example, hope this helps:

    //This will not work, resulting in the inner contents being able to bounce up and down, while scrolling horizontally:
var container = Ti.UI.createScrollView({
  height: Ti.UI.SIZE,
  layout: 'horizontal',
  backgroundColor: '#000000'
});

for (var i=0; i<10; i++) {
  var child = Ti.UI.createView({
    height: 100,
    width: 80,
    right: 20,
    backgroundColor: '#4d4d4d'
  });
  container.add(child);
}

//This will work, resulting in only smooth horizontal scrolling of its contents:
var container = Ti.UI.createScrollView({
  height: 100,
  layout: 'horizontal',
  backgroundColor: '#000000'
});

for (var i=0; i<10; i++) {
  var child = Ti.UI.createView({
    height: '100%',
    width: 80,
    right: 20,
    backgroundColor: '#4d4d4d'
  });
  container.add(child);
}
