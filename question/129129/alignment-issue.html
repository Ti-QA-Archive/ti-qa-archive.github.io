Alignment Issue

Hi,
My application working fine on Android with all controls are positioned well. Now I tried to run the same application on the Iphone and ipad, application works fine but, the controls are not positioned well. My sdk 1.7.2 , Api 2.3.1

Ex: This is my button code:

var button = Titanium.UI.createButton({
    left:250,
    color:'#666666',
    backgroundImage:'img/BUTT_gry_off.png',
    backgroundSelectedImage:'img/BUTT_gry_on.png',
    height:'auto',
    width:'auto',
    font:{fontSize:16,fontWeight:'bold',fontFamily:'Helvetica Neue'},
    top:250,
    title:'Login'
    });

It positioned well in android device, but not in iphone and ipad. Here, my button visible as very large in iphone and ipad.
I changed in tiapp.xml as:

 <deployment-targets>
        <target device="mobileweb">false</target>
        <target device="iphone">true</target>
        <target device="ipad">true</target>
        <target device="android">true</target>
        <target device="blackberry">false</target>
    </deployment-targets>

But no luck, still the problem is same..
Any ideas to solve this problem..

    if (Titanium.Platform.osname == 'android' || Titanium.Platform.osname == 'iphone') 
 {
     telentHomePageImage.image = 'img/header.jpg';
     loginBackGround.image = 'img/login_telent.jpg';
 }else if(Titanium.Platform.osname == 'ipad'){
     telentHomePageImage.image = 'img/header_ipad.jpg';
