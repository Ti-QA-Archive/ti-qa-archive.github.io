  • Korky

    To create a gist, you simply paste your code into Gist, click on the "Create public gist" button, and then copy the address in your browser's address bar here, using the Q&A formatting for creating links.

    Some environment variables won't be automatically created. This is perfectly normal for some packages.

    I believe you have JDK installed, because javac successfully returns the version number.

    Don't worry about the android_sdk variable for the moment - I think this is only required when you are compiling Titanium.

    Once you have added the adb symlinks, you need to ensure that the android tools directory is included in your system's PATH, so that adb runs irrespective of where on your filesystem you are running it from. See these google results for guidance about configuring your path.
