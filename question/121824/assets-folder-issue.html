  • those 2 methods worked fine to me(putting the image in the src package):

    // Method 1
int resourceID = getResources().getIdentifier("IMG.png", "drawable", "it.temp");
// or
int resourceID = getResources().getIdentifier("it.temp:drawable/icon",null,null);
button.setBackgroundResource(resID);

// Method 2  
Bitmap bitmap;
try {
    InputStream is = getAssets().open("IMG.png");
    bitmap = BitmapFactory.decodeStream(is);
    button.setImageBitmap(bitmap);
} catch (IOException e) {}
