  • Stefano,

    You cant set orientation for camera, for alternate you can try this autorotate property for your image view.
    autorotate property indicates whether the image should be rotated based on exif orientation data. By default, this is false on android and true on iOS.

    So in your imageView set autorotate as true, hope this will solve your problem

    More Info:

    http://docs.appcelerator.com/titanium/3.0/#!/api/Titanium.UI.ImageView-property-autorotate

    In Titanium you can use App properties store the data.

    Store a property

    Ti.App.Properties.setString('userName', 'Paul');

    Fetch saved properties

    Ti.App.Properties.getString('userName');

    For more example refer this blog
