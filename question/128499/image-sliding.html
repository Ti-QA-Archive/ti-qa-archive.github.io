  • In reply to how to add an image at the back:

    It looks like the jQuery plugin you linked to adds the image at the back when the animation finishes. If that's the case then you could add this last image when the animation of the last image finishes.

    E.g.

    //The images are out of order here because I'm pretending we've click some already
var images = [image3, image4, image5, image1, image2]
//**Animate the images on click**
//Add handler to the final image
images[images.length-1].animate(yourAnimation, function () {
  //Add the last image (that just zoomed off screen) to the back
  images.unshift(images.pop())
  //Redraw the stack of images
});

    Basically you're just waiting until the top image zooms off screen to add it to the back and draw them all as normal.
