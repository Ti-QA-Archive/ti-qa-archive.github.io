  • Afraid you indeed need a mac / mac mini.

    I've tried the "unofficial" way by trying to install MacOSX alongside a Windows installation, but it's more trouble than it's worth.

    If you're strapped for cash like I was, go to your used stuff listings (ebay / craigslist etc.) and find a Mac Mini. Keep in mind it has to be an Intel based processor, and it's best if you ensure it can run the latest MacOSX Maverick.

    If you have a bit more cash, buy a new Mac Mini, they're awesome.

    Once you get going, I can recommend ShareMouse, which uses your network to transmit mouse coordinates + keyboard presses - this enables you to use the same mouse + keyboard, simple by dragging the mouse between screens. Very cool!
