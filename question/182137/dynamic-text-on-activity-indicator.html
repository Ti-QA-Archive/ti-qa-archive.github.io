Dynamic text on Activity indicator

is it possible to dynamically change the message in an activity indicator or do i need to add 2 on the same controller/view? right now, the 2nd AI message in the activation function is showing the first message.

coupon.xml

<Alloy>
    <Window id="window" class="container">
        <ActivityIndicator id="activityIndicator"/>
    </Window>
</Alloy>

coupon.js

if (uid > 0) {
    $.activityIndicator.message = L("loading_coupons_activity");
     $.activityIndicator.show();
}
.
.
function deactivation(){
    // doesnt emtpy -> $.activityIndicator.message = "";    
    $.activityIndicator.message = L("deactivating_coupons_activity"); //shows the loading_coupons_activity string
     $.activityIndicator.show();
}

  • Accepted Answer

    Try to put the hard-coded strings in the code, & try to use setMessage() method :

    if (uid > 0) {
    $.activityIndicator.setMessage("Loading coupons...");
    $.activityIndicator.show();
}
.
.
function deactivation(){  
    $.activityIndicator.setMessage("Unloading coupons..." ); //if this works then error resides in strings.xml file
    $.activityIndicator.show();
}

    apart from this, i have a question that are you really hiding the indicator somewhere in your code ?

    if not so, then why did you call the show() method twice ?
