How to force an application restart?

In my iPhone app a user may make an in-app purchase that basically requires the application to restart to take effect since tabs that are already rendered need to be re-rendered. Is there a way to do a restart? Or a better way to repaint all of the tabs?

    You cannot force an app restart on ios.

    You could open a new window w with some message and after this close the tabgroup and open it again - of course after you repopulate it.

    • Hey,just run this command.The app will be restarted followed by changes.No need to rebuild it again.
      sudo npm install ios-sim -g
      sudo DevToolsSecurity -enable
      ios-sim "launch" "/Library/WebServer/Documents/testapp/build/iphone/build/Debug-iphonesimulator/test.app" "–devicetypeid" "iPad-2" "–exit";
      for devicetypeid list:
      ios-sim showdevicetypes
