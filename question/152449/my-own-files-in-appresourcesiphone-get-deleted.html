  • Hi

    This is very little to go on, but can I assume you are using Alloy?

    If so then you should not be placing files inside the resources folder at all, they should be in the app/assets folder.

    Just for confirmation, if you are placing files here for Alloy;

    {app name}/resources/iphone/

    This will get deleted and recreated each time you compile. You should instead place files in

    {app name}/app/assets

    See more information in the Alloy Assets section of the documentation.

    If you are not using Alloy, then NO files will be automatically deleted from within the {app name}/resources/… locations.
