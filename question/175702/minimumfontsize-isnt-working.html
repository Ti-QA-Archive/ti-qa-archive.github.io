minimumFontSize isn't working

I'm running build 3.2.3.201404181142 and SDK 3.2.3.GA testing on the iPhone retina (3.5 inch) emulator.

I'm trying to get my label text to resize using minimumFontSize I've tried declaring it in the .tss: 

"#testLabel": {
    text: "This text is too long",
    left: 15,
    top: 100,
    width: 200,
    height: 50,
    font:{fontSize: 50},
    minimumFontSize: 2,
    backgroundColor: "#FFFFFF"
}

And adding it in the .xml. And I've tried adding it in the .js

var jsLabel = Titanium.UI.createLabel({
    text: "So is this text",
    left: 15,
    top: 200,
    width: 200,
    height: 20,    
    minimumFontSize: 2,
    font:{fontSize: 50, fontFamily:'Georgia'},

    backgroundColor: "#FFFFFF"
});

$.index.add(jsLabel);

Neither text fields resize, they both result in the text being cut off e.g.: "This te…." in size 50 font.

What am I overlooking?
