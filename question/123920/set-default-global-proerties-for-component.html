Set default global proerties for component

Hi
It's possible to set default properties for component?
ex. If i set label = font:20 , color:blue; it's value for all created label??

  • Accepted Answer

    No, but you can define your own function constructor:

    function MyBlueLabel(options) {
    var label = Ti.UI.createLabel(options);
    label.backgroundColor = '#00f';
    return label;
}
...
var blueLabel1 = new MyBlueLabel();
var blueLabel2 = new MyBlueLabel();
