Apple push notification certificate password

In ACS when I click Save push certificate changes, I get a message telling me the password is incorect. But there is no password! If you go to apple developer portal and generate a APN certificate, you only need to upload a certificateRequest from keychain. No need to input a password! So what password does ACS Appcelerator settings require me to enter?

  • Hi Razvan, once you get your APNS certificate from apple you need to create a .p12 (this one requires a passsword) to upload to the ACS server.

    You can find more info at http://docs.appcelerator.com/cloud/latest/#!/guide/ios-section-5 in the push notification section
