AVD Emulator Stopped Launching

Application type: mobile

Titanium SDK: 2.1.2.GA? I did not see it in the log after a relaunch

Platform & version: Google APIs Android 2.2

Device: Android emulator 20.0.3

Host Operating System: Windows 7

Titanium Studio: 2.1.2.201208301612

My "run in android emulator" was working fine for most of yesterday. The emulator would launch just fine and I could see my work. The emulator locked up at one point and I shut it down. Since then when I attempt to run in android emulator I get the following log error:

[INFO] Building TiBountyHunter for Android ... one moment [INFO] Titanium SDK version: 2.1.2 (08&#x2F;24&#x2F;12 14:46 ed7f777) [ERROR] Application Installer abnormal process termination. Process exit value was 1 [ERROR] Timed out waiting for emulator to be ready, you may need to close the emulator and try again

However, If I launch the AVD manager and start a device I created separately, my work will show up in the new device I created. However, this new device is not the same device as called out in the TiApp Editor (Different Android SDK and screen type).

Essentially, I have a work around, but clearly something stopped working and I don't know what. I'd like to get back to the way it was working (I could launch the emulator from Titanium).

I have used the AVD manager to delete all devices, Reboted, and tried to create new devices from Titanium. Any time Titanium is required to launch the emulator - it does not launch.

Any ideas are appreciated.

Thanks
