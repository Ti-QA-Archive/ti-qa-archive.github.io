Searchbar without background gradient

Did somebody managed to create a stand alone (no tableview) searchbar which does not have a background gradient?

    var search = Titanium.UI.createSearchBar({
        barColor : '#000',
        showCancel : false,
        height : 43,
        top : 0,
        left : 0,
        width : 340,
        hintText:'Search...',
        backgroundColor:'#EEE'
    });

When I do the above, it still creates a backgroundGradient based on the backgroundColor supplied.

    You need to set the color by using the backgroundImage property combindes with the backgroundRepeat property. Just save a 1x1 file with your color and add the property to your bar. I think there isn't a property "barImage" like for windows but i'm sure thats coming!

    • Thanks! This is the code that worked for me:

      
    var search = Titanium.UI.createSearchBar({
        showCancel : false,
        height : 43,
        top : 0,
        left : 0,
        width : 340,
        hintText:'Search...',
        backgroundImage:'images/ui/pixel333.png',
        backgroundRepeat:true
    });
