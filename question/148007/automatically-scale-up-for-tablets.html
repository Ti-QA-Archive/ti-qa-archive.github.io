Automatically scale up for tablets

I'm currently writing an app. Now it looks fairly ok on a small screen. Then I installed the app for the first time on my Nexus 7… urg. Just not what I expected. Everything was just way too small (font size and also all of the elements could be slightly bigger).

Now I changed from px to dip with the hope that this would magically scale everything by some magic factor… :( well it didn't everything looks still the same. Now I'm wondering if this is possible at all, or if I have to manually add some scaling factor for tablets?

Thanks for inputs.

I'm using SDK 3.0

  • Raphael, Instead of hardcoded values, you can set the position/height/width of a control with a value that depends on device screen width and height. Like

    var  headerLbl = Ti.UI.createLabel({
    width        : Ti.Platform.displayCaps.platformWidth / 0.9,
    height    : Ti.Platform.displayCaps.platformHeight / 0.2,
    text        : 'Header section'
});

    This way all you can get the desired UI throughout all devices with different sizes.
