  • Hi, Fedor,

    I you are working on the alloy frame work then remove 

    userLocation="false"

    this is for removing blue dot on map, that define you current location in map
    you can put custom image instead of pin, there is a properties of annotation

    Titanium.Map.createAnnotation({
    latitude  : json.retailers[i].latitude,
    longitude : json.retailers[i].longitude,
   animate   : true,
    image     :   'img/map_point_h@2x.png',

    you can put you own image here or default image of map
