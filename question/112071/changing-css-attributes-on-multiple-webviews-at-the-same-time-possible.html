Changing CSS attributes on multiple WebViews at the same time. Possible?

Hello,

For a project I need to be able to change the body background color of multiple webviews at the same time.

All of the webviews are on the same .js file and I need that when the user clicks on "black" all of them change to black background.

I know how to do this inside each one of it, but I can't find a way to do it for all of them from outside.

Any clue?

Thanks!

  • I think you should add in each webview an event 

    Ti.App.addEventListener('change_body', function(data){
document.body.style.background = data.bgcolor;
}
)

    and on the control that will change the color to trigger this event

    Ti.App.fireEvent('change_body',{bgcolor:'#000'});

    Each webview will register it's own event, but because they are named the same it will launch on the same trigger.
