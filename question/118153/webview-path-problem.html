Webview path problem

Hello i am using titanium sdk 1.6.2 and android sdk 2.3.3

I am writing simple application imagine one button that when clicked loads a webview -> the content of the web view is in a folder in the Resource section named for example /todor

So in this folder i have 3 html files conected with a static links in index.html i have a href to contacts.html in contacts.html i have a href to index.html and so on.

Here is some coffee script.

b1.addEventListener('click', () ->
  url = Ti.Filesystem.resourcesDirectory + 'todor' + Ti.Filesystem.separator + 'index.html'
  webView = Titanium.UI.createWebView({
    url: url,
    height: 'auto'
  })
  Titanium.UI.currentWindow.add(webView)
)

It loads fine the index.html, but when i click on the other links it goes down and i cant find a solution. Thank you.
