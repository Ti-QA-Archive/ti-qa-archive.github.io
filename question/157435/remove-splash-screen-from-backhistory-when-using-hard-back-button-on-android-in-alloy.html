Remove splash screen from backHistory when using hard back button on android in alloy

I am fairly new to Titanium Alloy and am building an app that will be using the hard back button on an android device. The back button works fine but shows the splash screen before it exits the application. I saw there was an answer for this when creating this in titanium however I am using Alloy and there is no AndroidManifest.xml file for me to add the android:backHistory to. Any help would be greatly appreciated. Thanks.

    Add this to your window: exitOnClose: true (via tss) or exitOnClose=”true” (via xml)
