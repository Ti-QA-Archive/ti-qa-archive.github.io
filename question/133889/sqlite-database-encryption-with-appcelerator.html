SQLite Database Encryption with Appcelerator

Hi,

I was taking a look at http://sqlcipher.net/ and was hoping to use this in my app (to protect the database).

Although the instructions are pretty simple, is this supported by Appcelerator? I get the impression with Appcelerator that if I make any changes to the Xcode Files, they are over-written anyway whenever I compile.

What is the ideal method to use to encrypt an SQLite local database when used in a Titanium project?

Thanks

  • Yes - that's right. Do not modify the files in the build folder - they will be replaced on each clean build.

    You have to create a custom module which links to sqlcipher and the required frameworks.Furthermore you will have to write your own methods to access the database, since sqlcipher needs a key for decrypting.

    See the iOS Module guide in the Docs section and the sqlcipher ios tutorial (http://sqlcipher.net/ios-tutorial/)
