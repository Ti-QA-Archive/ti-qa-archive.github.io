    Hi,
    I suspect there is an issue with the CLI with your setup.
    Ran into the same problem with the CLI before. In my case it turns out the ~/.titanium/config.json file got corrupted with 2 extra curly braces. Removing them fixed the issue.
    delete the file at 

    C:/Users/MyName/.titanium/config.json

    Hope it works.
