Can not get Titanium Developer to start...

I can not get Titanium Developer to start. It was running a month ago when I first installed it.

I am getting this message "Titanium Developer appears to have encountered a fatal error and cannot continue."

I am running:

  • OS X 10.6.6 (Snow Leopard)

  • Xcode 4.01 Build 4A1006

  • Titanium Developer, I don't see a version number but downloaded on 2011-03-26

The only major changes I have made since the last time this was working, was installing some xml lib dependancies to get rvm/ruby running on my Mac.

  • Fixed :)

    It looks like needed to delete the Titanium support files and start the installation over. I do not need any of my previous projects so this isn't a problem for me. Others might want to backup the project files before doing this.

    Anyway to fix, I just deleted the "/Library/Application Support/Titanium" folder and restarted Titanium.

    Hope this helps someone else.

    NOTE: I think I had these problems after upgrading to Snow Leopard and 4.3. This fixed it
