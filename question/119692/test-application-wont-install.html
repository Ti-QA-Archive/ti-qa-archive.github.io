Test Application Won't Install

I followed the Getting Started guide. Installed JDK, Android SDK, and all the stuff from the repositories.

When I click "Launch", the emulator goes to the home screen, and Titanium outputs this:

  • Solved myself: There is a dropdown for SDK version on the bottom-left of Titanium. It was at 1.1. I increased it (to 1.6, arbitrarily), and it worked.
    I figured this out because there is also a "Filter" dropdown that you can use to get more specific error messages.
