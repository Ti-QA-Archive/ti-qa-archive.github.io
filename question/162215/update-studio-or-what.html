  • Hello Richard!

    After reading your error, looks like you added more Software Sources to Titanium Studio and it's trying to solve not only the Titanium Studio update, but the others as well.

    Cleaning the Software Sources

    In order to fix this, follow this instructions:

    1. Go to Ti Studio, click on Help and go to Install new Software.
    2. Click on "Available Software Sites"
    3. Unselect all the sources, except for "Titanium Studio Stable Updates". Check Image here.
    4. Click OK
    5. Request for "Check for updates" in Titanium Studio -> Help -> Check for updates.

    Let us know how it goes!

    Best,

    Mauro
