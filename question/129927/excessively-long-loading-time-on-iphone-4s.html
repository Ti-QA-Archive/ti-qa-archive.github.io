Excessively long loading time on iPhone 4S

We are experiencing up to 40-70 sec delay before the splash screen disappears and opens the first window which brings up a login screen on our app. We are not able to reproduce on iPhone 4. This only happens on iPhone 4S. Normal load time on iPhone 4 is averaging 3-5 seconds every which way you test. This doesn't make sense. The app is working great under iPhone 3 and 4. We have not introduced anything remotely complex or device specific. Just standard views. The only thing remotely advanced would be the SQLlite database which currently only has one table and is deployed with the IPA under the resources directory. Latest build exhibiting this problem was compiled under 1.7.5. Thanks for any help.
