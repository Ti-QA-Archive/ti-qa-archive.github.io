    • I have to check that tomorrow (it was 3.4.1 but I'm not sure if I switched to 3.5.0 with this project). But you can just place the zip file in the root folder and it will get extracted in the right place. Also make sure to make "ti clean" first when you switch to 3.5.0 (think there are some none 64bit config files left, and that gets rid of them).

      But I'm using the official module with 3.5.0 (located here now: https://github.com/appcelerator-modules/ti.compression). But it won't give you the extraction percentage (Marcels module does that). It just gives you a finished event.
