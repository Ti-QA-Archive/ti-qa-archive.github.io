Alloy - Get view to only fill available space?

I have tried searching for this answer and I feel like I must be missing something really obvious.

My Alloy view XML is something like this:

<Window>
    <View>
        <View id="footer">(buttons in here)</View>
        <View id="content">(differing items in here)</View>
    </View>
</Window>

My TSS is something like this:

"#footer": {
    bottom: "0dp",
    width: Ti.Platform.displayCaps.platformWidth,
    height: "44dp",
    backgroundColor: "green"
},
"#content": {
    top: "0dp",
    backgroundColor: "red"
}

How do I make content fill the screen but not extend past my footer? Right now the only way I can get it to work correctly is by setting an explicit height on #content. Like say "300dp" for height. (which isn't the exact amount to get it to fill on the iPhone simulator, but it's short enough that it doesn't hide my footer) I tried Ti.UI.FILL, which just makes it completely take over the screen also.

What's the proper way to do this? If I found the exact height in dp that makes it work on the iPhone simulator, would that carry over to Android and other devices with differing screen sizes?

Thanks!

  • Accepted Answer

    "#footer": {
    bottom: "0dp",
    width: Ti.Platform.displayCaps.platformWidth,
    height: "44dp",
    backgroundColor: "green"
},
"#content": {
    top: "0dp",
    bottom: "44dp",
    backgroundColor: "red"
}
