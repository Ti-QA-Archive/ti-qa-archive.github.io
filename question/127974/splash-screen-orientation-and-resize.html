  • OK what platform are you on ?…..

    iOS you set two splash screens default.png and default@2.png …

    Then if you set in the tiapp.xml file

    <iphone>
    <orientations device="iphone">
        <orientation>Ti.UI.PORTRAIT</orientation>
    </orientations>
</iphone>

    your app will only load in portrait mode.

    Hope this helps
