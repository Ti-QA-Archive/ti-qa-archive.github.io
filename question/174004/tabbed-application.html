Tabbed application

I have a tabbed application. I used tab group and created tabs as follows


var tabGroup = Titanium.UI.createTabGroup();

var win1 = Ti.UI.createWindow({});
var tab1 = Titanium.UI.createTab({
window: win1
});

var win2 = Ti.UI.createWindow({ });
var tab2 = Titanium.UI.createTab({
window: win2
});

tabGroup.addTab(tab1);
tabGroup.addTab(tab2);

tabGroup.open();

Is there any event that is performed only when the tab is changed, and only for the first time . For eg: I need first tab operations to be performed when the tabgroup is opened. But the second tab's operation only when that tab is clicked for the first time.I tried the window's open method, but second window's open event also gets fired when the tabgroup is opened. I tried tab's focus method also, but it gets fired each time the tab is changed.

Any help is appreciated.

  • Hi

    You have the correct event - but you have to do some work. Events will fire each time that trigger is called. You cannot tell to not fire again. The idea is to not respond to the future events being triggered.

    I have not tested this code - but something like this;

    var tabHandler = function (e) {
    e.source.hasShown = !e.source.hasShown;
};
var tabGroup = Titanium.UI.createTabGroup();

var win1 = Ti.UI.createWindow({});
var tab1 = Titanium.UI.createTab({
    hasShown: false,
    window: win1
});
tab1.addEventListener('focus', tabHandler);

var win2 = Ti.UI.createWindow({ });
var tab2 = Titanium.UI.createTab({
    hasShown: false,
    window: win2
});
tab2.addEventListener('focus', tabHandler);

tabGroup.addTab(tab1);
tabGroup.addTab(tab2);

tabGroup.open();

    You may need to set the first tabs hasShown property to true - but again, your testing will determine this.
