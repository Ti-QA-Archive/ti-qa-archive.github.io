  • Hi Pierre,

    This answer might be coming late but I was just trying to find a solution for that same, problem so maybe it helps anyone else too…

    You could store the device token into the app properties module Titanium Poperties Module when the method Titanium.Network.registerForPushNotifications returns you the device token. That way you can get it back anytime with Ti.App.Properties.getString(&#39;deviceToken&#39;));
