  • Hi,

    you have two options.

    • you need to create one function in topBar.js file and export it then in index.js call that function with passing a window object. then on click of button you can close that window.

    • remove require tag from index.xml and add that view in index.js with using Alloy.createController('topBar') then pass window object at the time of creation like

    Alloy.createController('topBar',<window id>);

    then in topBar.js you can store that window object in one variable.
