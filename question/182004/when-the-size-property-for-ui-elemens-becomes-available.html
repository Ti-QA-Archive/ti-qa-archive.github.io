When the "size" property for UI elemens becomes available?

  • Application type: Alloy mobile for Android
  • Titanium SDK: 3.5.1.GA
  • Platform & version: Android
  • Device: real S3 Andr. 4.3 and Genymotion API 10
  • Host Operating System: Win x64
  • Titanium Studio: Titanium Studio, build: 3.4.1.201410281727

HI Folks, I've this simple code:

index.xml

<Alloy>
    <Window id="winIndex">
        <View id="myView" width="500px" backgroundColor="#abc">
        </View>
    </Window>
</Alloy>

index.js

$.winIndex.open();
Ti.API.info($.myView.size.width); // I get 0 instead of 500 !!!!

If I put the statement into an event listener it works:

// index.js

$.winIndex.addEventListener("click", function(){
    Ti.API.info($.myView.size.width); // it prints 500 correctely !
});
$.winIndex.open();

Now, this is my question: when can I access to the "size" property of an UI element? In other words… when "size" property became available? Thanks for attention.
