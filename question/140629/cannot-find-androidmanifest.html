    Hi Sergio,

    By default AndroidManifest file path is Build folder inside android folder if you want to use your custom then you have to copy that and make necessary changes and put it into your ProjectName>platform>android> and after that procedure you have clean build and run again and also you have un-Install from simulator/emulator.

  • Ok, now I have it. The code on the tiapp.xml wasn't my androidmanifest.xml file, i had to go through the file system to the build folder: 

    Documents>Titanium_Studio_Workspace>MyProjectName>build>android>"AndroidManifest.xml"

    and once there copy the file manually to the desired destination:

    MyProjectName>platform>android>res

    so I can make the changes necessaries for the internationalization.
