iADs not showing if not immediately loaded

I have a TableView and I want iAds to be at the bottom, but only give up the space if an iAd actually shows up. An event handler listens for the 'load' event of AdView and then changes the TableViews height.

I have two questions:

1) Can it be determined if an iAd showed up right away? In this code sample, the AdView is overlaid on top of the TableView if the iAd shows up right away. I would like to reset the TableView size at this time, but can't figure out how to detect that the iAd is there.

2) When the iAd is delayed, the 'load' event handler runs. But, there is no iAd. It just shows the underlying window and the console says "WARNING A banner view (0x8e78dd0) has an ad but may be obscured." Where did the AdView go?

Thanks for any help.

Application Type: iPhone,
Titanium SDK: 2.0.2 GA,
Platform: iOS 5.1,
Host OS: OSX,
Studio: 2.0.2.201205311912


    var win = Ti.UI.createWindow({
        backgroundColor: '#00944a',
        title: 'Events',
        modal: true,
        navBarHidden: false,
        layout: 'absolute',
    })

    var tblEvents = Ti.UI.createTableView({
        title: 'List of Events',
        backgroundColor: 'white',
        editable: true,
        top: 0,
        height: '100%'
    });

    win.add(tblEvents);

    var adView;
    var t1;

    adView = Titanium.UI.iOS.createAdView({
        width: '100%',
        height: '50dp',
        top: '366dp'
    });

    t1 = Titanium.UI.createAnimation({height:345, duration:500}); 

    adView.addEventListener('load', function(e) {
        Ti.API.info('iAds load event ');
        tblEvents.animate(t1);
    }); 

    win.add(adView);

    win.open();
