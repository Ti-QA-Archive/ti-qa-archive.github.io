  • Accepted Answer

    It wont fire because is not a global event. in order to fir you need to make the event global or the trigger local

    self.addEventListener('changenav', function(e) {
        Ti.API.info('to push into nav');
    });
self.fireEvent('changenav',{view:'myview'});

    or

    Ti.App.addEventListener('changenav', function(e) {
        Ti.API.info('to push into nav');
    });
Ti.App.fireEvent('changenav',{view:'myview'});
