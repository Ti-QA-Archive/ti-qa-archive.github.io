    hi,

    you can pass your custom variable to another window like

    
var winDetail = Ti.UI.createWindow({
    url:,
    <customVariableName> : <value>
});

    and in that window you can get that back by

    var win = Ti.UI.currentWindow;
var <variableName> = win.<customVariableName>
