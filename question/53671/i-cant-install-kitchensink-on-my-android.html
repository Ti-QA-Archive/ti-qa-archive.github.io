    • Looks like my issue was the USB driver. (I'm using windows BTW).

      Solution was to follow instructions in http://developer.android.com/sdk/win-usb.html

      Small detail is that the instructions have not been updated with the new location of the USB driver in the sdk (USB Driver Revision 4) which is android-sdk-windows\extras\google\usb_driver

      After that worked a treat :)
