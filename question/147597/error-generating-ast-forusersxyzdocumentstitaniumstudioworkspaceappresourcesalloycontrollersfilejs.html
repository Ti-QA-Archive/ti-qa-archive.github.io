    A couple things:

    • If you don't actually show a simple test case (not your whole app) we can run that exhibits the problem, the best anyone can do is make guesses
    • AST errors will generally tell you exactly what line and in what file they occur in. That should show you exactly where the error in your code is.
    • It should be onClick=&quot;clickIt&quot;, not onClick=&quot;clickIt()&quot;

    Again, the best anyone can do is guess based on the single line of code you provided and the fact that you haven't yet shared the full error output.
