  • Rich,

    In my experience, the first things you need to understand are javascript syntax and the basics of object-oriented programming. This will go a long way. I don't believe that code completion is going to be overly helpful, but it may have its uses in certain situations.

    There are many basic javascript tutorials and references, but they aren't necessarily going to be of "great" help. You should reference the API's, the kitchen sink source code, and the Q&A here (which often will contain code snippets). This is how I learned to utilize the features within Titanium. The kitchen sink app is a great source for the basics of how the javascript-style code works in Titanium.

    Again, this is my personal opinion based on my experience. Good luck.

    Clayton
