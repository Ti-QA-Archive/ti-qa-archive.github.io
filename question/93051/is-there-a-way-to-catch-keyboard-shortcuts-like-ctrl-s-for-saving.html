Is there a way to catch keyboard shortcuts (like ctrl-s for saving)?

The title says it all - there a way to catch keyboard shortcuts (like ctrl-s for saving)?

I also noticed that there isn't an obvious to show the keyboard shortcut text in the menubar via Titanium.UI.createMenuItem - is there an unobvious way that i'm missing?

  • Unfortunately, there seems to be no way to attach keyboard shortcuts to menu items yet, at least none that I know of.

    Catching keyboard shortcuts itself is pretty easy, you just need to attach listeners to the DOM key events, for example 'keydown'. This can be done in plain JavaScript or using practically any of the many JS frameworks/libraries. I created a small sample application which demonstrates two different ways. Let me know if it works for you, I only tested on Mac. Speaking of which, remember that Mac OS X has different standard key bindings than Linux and Windows. You wouldn't put save on ctrl+s in a Mac application, but on command+s. Titanium.getPlatform() can be used to implement different shortcuts depending on the OS.
