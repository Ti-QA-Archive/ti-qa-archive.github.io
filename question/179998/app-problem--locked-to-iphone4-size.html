App problem : locked to iPhone4 size

I use the map example (Alloy) and I've got this render when I deploy on my device (iphone 5c). It seems to be the size of an iPhone4, so I've got 2 black stripes on top and bottom.

This is how it's done in index.xml, just in case…

<Alloy>
    <Window class='container'>
        <Module id="map" module="ti.map" method="createView" platform="ios,android">
            <Require src="annotationView" title="Home" latitude="37.47" longitude="-122.12"/>
        </Module>
    </Window>
</Alloy>
