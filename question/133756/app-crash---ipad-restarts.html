    • for that you have to create separate function for web request. when you call a web request you have to decide what to be done after data receive and what is done before data called or before web service called

      for Example:

      var str = "http://webServiceUrl";
performWebRequest(str, 
    null, 
    callBackFunction(responseObj) {
        // performs after data receive.
        /// your code here.
    },
    errorFunction(args){
        // performs when error.
        // your code here.
    }
)

      and another side there is a function that actually performs the web request.

      performWebRequest(url,parameters, callbackFunction, errorFunction){
    // Web Request logic.
    /// on Load event you have to use callbackFunction after data receive.
    postRequest.onload = function() {
        callbackFunction();
    }
    postRequest.error = function() {
        errorFunction();
    }
}

      you can add more functionality as per your need in this function.
