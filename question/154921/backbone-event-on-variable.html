Backbone event on variable

I'd like to bind and trigger a custom event to a local variable using Backbone.js.

//...
var myVal;
_.extend(myVal, Backbone.Events);
myVal.on('myCustomEvent', function() {
    //do something
}
//...
myVal.trigger('myCustomEvent'); //trigger custom event
//...

I'm getting this error message: 'undefined' is not a function (evaluating 'myVal.on')

Did I miss something?

    myVal is not created as an object. Try this:

    //...
var myVal = {};
_.extend(myVal, Backbone.Events);
myVal.on('myCustomEvent', function() {
    //do something
}
//...
myVal.trigger('myCustomEvent'); //trigger custom event
//...
