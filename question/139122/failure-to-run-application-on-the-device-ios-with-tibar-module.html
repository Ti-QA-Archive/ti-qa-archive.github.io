Failure to run application on the device IOS with Tibar module.

Hi, I have a problem with the tibar (barcode scanner http://code.google.com/p/tibar/), when run on my iphone, xcode returns an error exit 65, I've outlined below:

  • I found the solution, you must specify the module tibar frameworks to use. I edited the line in the file /Users/name/Library/Application Support/Titanium/modules/iphone/tibar/0.4.2/module.xcconfig and added the following:

    OTHER_LDFLAGS=$(inherited) -framework AVFoundation -framework CoreMedia -framework CoreVideo -framework QuartzCore /usr/lib/libiconv.dylib
