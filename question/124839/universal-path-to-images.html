Universal path to images

Hello all,

I have a simple question and could not find an answer for this. I have an 'images' folder in my Resources folder and would like to access images like &#x2F;images&#x2F;image1.png but NOT like ..&#x2F;..&#x2F;..&#x2F;..&#x2F;..&#x2F;images&#x2F;image1.png. When I trying &#x2F;images&#x2F;image1.png it does not work.

Does anyone know an universal solution for both Android and iOS? SDK 1.7.0
Thanks.
