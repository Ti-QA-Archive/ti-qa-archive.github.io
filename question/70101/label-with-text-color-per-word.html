Label with text color per word?

In HTML I can set the text per word using the font tag, useful when I don't know the length of text ahead of time, makes alignment easy. I'm in a situation where I have a block of text, I need to change the color of the first few words however I'm having trouble using separate labels to do so because I can't maintain alignment of the labels within a row since the first varies in length. Is it possible to set the color on a portion of the label text?
