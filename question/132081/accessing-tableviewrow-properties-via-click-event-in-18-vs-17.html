Accessing TableViewRow properties via click event in 1.8 vs. 1.7

Hi all,

I'm having an issue with the new 1.8 API for click events. Beforehand, when I set properties on TableViewRows, I was able to see them through the click event. For Example:

var row = Titanium.UI.createTableViewRow();
row['bacon'] = 'heck yeah';
row.addEventHandler('click', function(e) {
    alert(e.rowData);
});
// { height : 30, width: 400, bacon: 'heck yeah' }

However, as of switching to 1.8, this is no longer the case. Is there any way to get to these properties in 1.8?

  • Accepted Answer

    var row = Titanium.UI.createTableViewRow();
row['bacon'] = 'heck yeah'; // or row.bacon='is making you fat :)';
row.addEventHandler('click', function(e) {
    alert(e.row.bacon);
});
