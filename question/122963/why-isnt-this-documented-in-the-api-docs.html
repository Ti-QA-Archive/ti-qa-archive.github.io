  • Using NavGroups and windows with external URL:

    I'm opening a Navgroup from a window. The external URL for the windows contained in the navgroup is in a subdirectory of jsLib of the current window.

    ..setup/currwin.js
    ..setup/add/app.js

    Using a relative URL for a window in the navgroup won't work:

    var winImage = Ti.UI.createWindow({
    url: 'add/image/app.js'
    });

    To make matters worse, you won't get any error messages.

    Just opening winImage (outside the navgroup) works perfectly well.

    The "trick" here seems to be to always use absolute URL:s…
