Help with facebook logout

So, I have an app that needs log in by facebook to use with two tabgroups. The first one has the log in window(win) and the second tabGroup has tree tabs with 1 window each.
At the first log in, I have no problem and all runs well.
To log out, I have an extra button in one of my tabs to do so.
I want that, when the user logs out, the app opens the first tabGroup(with the facebook log in).
Here is it's code:

logoutbtn.addEventListener('click', function(e) {
    Titanium.Facebook.logout();
    labelnome.text = "";
    tabGroup0.open();
    win.open();
    tabGroup.close();
});

tabGroup0.open(); - contains win

win.open(); - opens the window with the log in button

tabGroup.close(); - closes the tabGroup that opens after the log in.

When I use "my button" to log out, the app goes to the first tab ( and window) and the user is loged out as expected. If, after that, I try to log in again, the log in window asking the user and pass is shown but nothing happens after that. If I quit the app and click the log in button, it loges me in without asking user and password. It seems like it is saving the log in data for the next time I want to do so…

  • thank you. In my case it helps doing that:

    Titanium.Facebook.addEventListener('logout', function(e) {
var url = 'https://login.facebook.com';
var client = Titanium.Network.createHTTPClient();
client.clearCookies(url);
});
