  • Hi

    From memory, you can pass extra parameters when calling the method openPhotoGallery from Titanium.Media

    If you look at this link to the docs PhotoGalleryOptionsType it shows the parameters available. The one I think you are after is allowEditing and this needs to be false

    Ti.Media.openPhotoGallery({
    allowEditing: false
});

    Hope this helps.

  • Hello,

    The editing overlay/popup that you see when ‘allowEditing=true’ is the default provided by Apple and is not modified by Appcelerator.

    There currently is not a way to remove the ‘Move and Scale’ label from the iPad’s popup.
    However, as Luke Taylor indicated, you can remove the editing ability completely by setting ‘allowEditing=false’ when calling openPhotoGallery().

    If this is something that you feel should be changed or improved, please write a ticket in JIRA for your request and this will be properly addressed there.

    Thanks!
