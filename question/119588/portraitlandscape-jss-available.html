    • in directory <project_name>/Resources/android/images/, all res-directories are replaced by corresponding drawable-directories like the Android SDK is describing during build fase. Other files are copied to drawable dir. If you have also high-medium-low directories, these directories will be replaced by drawable-hdpi, drawable-mdpi and drawable-ldpi. In the Python builder file: builder.py in the Titanium Mobile 1.7.1 SDK directory you will find all builder information.
