Change visibility of <Label> based on class?

This is kinda in the grey area of my understanding of Alloy.

So, let's say I have a bunch of <Label>'s, maybe 40 in a window, and they all have the same class <Label class="im_a_class">some text</Label>

Is there any way for me, to change all of those elements based on class? For instance if I want to make them all visible:false if the user clicks a button… ? Is that possible?

jQuery would look something like this: $(".im_a_class").hide() if that makes better sense of what I'm trying to do.

Any guidance or help would be greatly appreciated. Thanks!!

  • Accepted Answer

    Sorry, TSS styling and classes don't work that way. The next version of Alloy will introduce dynamic styles through which you'll be able to add/remove classes to elements. But you'll still need to do so individually to each label.

    For now, you could put all the labels inside a View and simply hide that view. Or, you can programmatically loop through a bunch of labels (by ID) and call their hide() method (or set their visible property to false). 

    var kids = $.yourView.children; // this is a built-in array of a view's children
for(var i=0, j=kids.length; i<j; i++) {
  // assuming you've assigned a custom property to each Label
  // in the TSS or XML, say <Label isALabel='true'>
  if(kids[i].isALabel) {
     kids[i].hide();
  }
}
