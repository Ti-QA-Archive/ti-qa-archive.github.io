Send params method GET iOs

Hello, how to pass parameter in the method GET with words with accent on iOS. Ex: the param "Gália". But when sending to the server it is so "G%C3%A1lia".
In android does not have this problem.

Think need to use one set request header but do not know.

thanks.

sorry for bad English.

  • It depends on encoding type and method. Default is urlencode and this case you send: 

    xhr.open('GET', ENDPOINT + '?param=' + encodeUri('Gália'));

    If you send POST and JSON, you have less problems:

    xhr.send(JSON.stringify{param:'Gália'});

  • Thanks for the reply Rainer Schleevoigt. Now it worked. I pass the entire URL.

    var url = 'http://......?param=Gália';
xhr.open('GET', encodeURI(url));
