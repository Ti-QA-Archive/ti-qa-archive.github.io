Missing recommended icon file

I uploaded my app to apple app store and got an email from them. I am wondering how to resolve this:

Dear developer,

We have discovered one or more issues with your recent delivery for "No Physics". Your delivery was successful, but you may wish to correct the following issues in your next delivery:

Missing recommended icon file - The bundle does not contain an app icon for iPhone / iPod Touch of exactly '120x120' pixels, in .png format for iOS versions >= 7.0.

Missing recommended icon file - The bundle does not contain an app icon for iPad of exactly '152x152' pixels, in .png format for iOS versions >= 7.0.

Missing recommended icon file - The bundle does not contain an app icon for iPad of exactly '76x76' pixels, in .png format for iOS versions >= 7.0.

If you would like to update your binary for this app, you can redeliver your binary.

Regards,

The App Store team

    Hi

    The easiest way o solve this is to use the;

    • ticons service

    This is a web app create by one of the Appcelerator community members. It has a single function; provide the icons and splash screen files in the correct sizes with the correct names. You supply a couple of images files and it creates a zip file containing all the required files, pre-sized and in suitable folder names.

    We use this ourselves and I can confirm it supplies you all with the images of the correct size that removes the notice you have received.
