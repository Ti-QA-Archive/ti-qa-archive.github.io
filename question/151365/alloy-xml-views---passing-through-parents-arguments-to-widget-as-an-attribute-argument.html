Alloy XML Views - Passing Through Parents' Arguments to Widget as an Attribute Argument

Is it possible to pass through arguments that were passed to a view from within the XML view to a widget?

<Widget id="test" src="SomeWidget" args="arguments" />

Would be parsed by Alloy as:

$.__views.pagination = Alloy.createWidget("Pagination", "widget", {
    id: "test",
    args: arguments[0],
    __parentSymbol: $.__views.prepare
});

Thus the widget gains access to the arguments that were passed to the parent calling the widget.

  • No, I don't think there's a direct way to do that in markup. You'll have to create the widget programmatically in the parent controller and then pass the arguments to the newly created widget that way.
