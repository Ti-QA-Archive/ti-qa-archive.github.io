addEventListener() should return a reference to the object it was called by

This isn't so much a question as a feature request. I think addEventListener should return a reference to the object that it was called by, so that chaining is possible and therefore I don't need to create single-use variables.

For example, this code…

var dialog = Ti.UI.createDialog({
    title: "Example",
    message: "Hello",
    buttonNames: ["1", "2"]
});

dialog.addEventListener('click', function(e) {
    Ti.API.info("Button " + e.index + " was clicked");
});

dialog.show();

…could be shortened to this instead:

Ti.UI.createDialog({
    title: "Example",
    message: "Hello",
    buttonNames: ["1", "2"]
}).addEventListener('click', function(e) {
    Ti.API.info("Button " + e.index + " was clicked");
}).show();

Now I technically have a one-liner and I didn't need to create the dialog variable.
