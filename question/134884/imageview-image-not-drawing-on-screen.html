ImageView image not drawing on screen

So I am trying to create a simple Pong application, and I realized that unlike Java, Javascript doesn't have any graphics functionality (i.e. g.fillRect()), so I am trying to use.. wait for it.. a strip of bacon for my Pong paddles. Anyways, I have the photo stored in a Ti.UI.createImageView var. This is what it looks like:

var baconStrip = Ti.UI.createImageView({
    image:'bacon.png',
    width:25,
    height:10,
    left:'10%',
    top:'10%'
});

window.addEventListener('open',function(e){
    window.add(button);
    button.addEventListener('singletap',function(e){
        window.add(baconStrip);
        alert(baconStrip.center);
        window.remove(button);
        });

For some reason, the picture does NOT want to draw. I tried adding in 

baconStrip.addEventListener('load', function(e){
    alert('Picture loaded');
});

The result of that was the alert popped up, but I still didn't see the image. If anyone can help me render this picture, it would be a great help. Thanks in advance!
