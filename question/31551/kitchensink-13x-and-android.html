KitchenSink 1.3.x and Android

I've been trying to run the KitchenSink 1.3.x in Android and I keep hitting this issue:

[TRACE] E/PackageManager( 54): Package com.nolanwright.kitchensink requires unavailable shared library com.google.android.maps; failing!

I'm running Titanium mobile SDK 1.3.0, Android SDK 1.6.

  • To use maps, you will need to be using a "Google APIs" version of the Android SDK - this will show up as, for instance, "APIs 1.6" in Titanium Developer.
