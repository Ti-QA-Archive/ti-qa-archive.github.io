  • index.xml

    <Alloy>
    <Window><Button id='likeButton' onClick='onLikeButtonClicked' title='Like' /></Window>
</Alloy>

    index.js

    function onLikeButtonClicked(){
    if (this.title == 'Like'){
        this.title='Unlike';
    } else {
        this.title='Like';
    }
}

    Hello Nick,

    You can also do like this…

    index.xml

    <Alloy>
    <Window>
        <Button id='likeButton' title='Like' width=50 color="red"/>
    </Window>
</Alloy>

    index.js

    $.index.open();

$.likeButton.addEventListener('click', function(e) {
    if (this.title == 'Like') {
        this.title = 'Unlike';
    } else {
        this.title = 'Like';
    }
});
