scrollView not displaying on android device

My app works on the Xcode simulators. Everything displays properly, and the scrollview will scroll.
However the Titanium.UI.scrollView won't display on a physical android device (Galaxy s4).

Anyone else having the same issue or know if I've done something wrong?

var mainWindow = Ti.UI.createWindow({
    layout: 'vertical',
    width: '100%',
    height: '100%',
    backgroundColor: '#3f83bf'
});

var scrollView = Ti.UI.createScrollView({
    top: 0,
    width: '100%',
    layout: 'vertical',
    backgroundColor: 'red', 
    showVerticalScrollIndicator: true,
    scrollType: 'vertical',
    contentWidth: dm.deviceWidth, // the device width
    contentHeight: 'auto'
});

Thank you!
