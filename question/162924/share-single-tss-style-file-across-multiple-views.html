Share single tss style file across multiple views

I have a few views/controllers that will use a baseController for the controllers and I would like to reuse the view and tss files as well. Is there a way to inherit or reuse the code in the these files, similarly to the baseController property for the controller?

    If you're talking about a global-style-sheet, the best method is to create a file in the styles folder and name it app.tss, this is the top level style, anything you code in here will be read first in the stack.
