Books & Tutorials

This is not a question actually, more a reflection.

I've played with Titanium for a couple of months now, targeting both iPhone and Android, creating both dummy-apps and a commercial application. I had some headaches with the android SDK in the beginning but looking back some of them where related to my environment (windows7 - x64 - subversion). Of course, the android SDK lacked a lot of functionality but with 1.5 that has changed to a lot better.

However - to my point. What I missed as a rookie, and still, is nice tutorials and books. Sure I have the Kitchensink, which is the saviour most of the times, but it doesn't give me all the fun ideas to begin with.
When I started doing native android development I read a ton of tutorials and bought some books. Then I used their ideas and enhanced them or twisted them around a little and learned a lot from it.

Perhaps it's just because I'm not a Javascript guru from the beginning. I guess it would have been a lot easier to start developing a side-scroller shooter or a recipe app in Titanium if you had done it on the web before!?

Anyway, keep up the good work, and if you get a book or any advanced tutorial printed, I'll certainly buy it.

Regards
Stefan
