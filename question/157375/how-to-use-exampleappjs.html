    Hello Varun,

    You must need to declare this module in Tiapp.xml

    //tiapp.xml

    <modules>
        <module platform="iphone">module_name</module>
        <module platform="android">module_name</module>
    </modules>

    and also add the modules folder parallel to app directory.

    i.e :

    create folder modules and add your module in this folder like :

    • modules/android/ti.imageFactory

    • modules/iphone/ti.imageFactory

    LEt me know if it helps or need more help.

    Thanks
