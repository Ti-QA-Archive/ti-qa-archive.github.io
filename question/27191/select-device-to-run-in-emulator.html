Select device to run in Emulator

My app is based on SDK 3.2 and will run on iPad simulator if I click on Launch to run the Emulator. How can I change the device to test it on the iPhone simulator?

  • Accepted Answer

    If you created the project as an "iPad" project, then it is targeted to ipad only.
    You might need to recreate the project as "mobile"

    There is no way yet to make an universal app ( both iphone and ipad ) so you might need to make 2 different apps.
