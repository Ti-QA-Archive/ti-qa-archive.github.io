Crash in textarea event selected. There is a way to detect the Cursorposition (Selection)?

Hello altogether,

{
    Ti.API.info("Hello");
});

After selection a crash comes:

  • i tried a simple application for this and this is working fine.

    var win = Ti.UI.createWindow();
var txtArea = Ti.UI.createTextArea({
 value : "hello everyone"
});
txtArea.addEventListener('selected',function(e){
 alert("hello");
 //try to remove this eventlistener otherwise this event will be fired continuosly //untill the text is unselected or other operations performed
 Ti.API.info("hello");
});

win.add(txtArea);
win.open();

  • for getting cursor position u can try the same code by using range and its property returned by event.

    
var win = Ti.UI.createWindow();
var txtArea = Ti.UI.createTextArea({
 value : "hello everyone"
});
txtArea.addEventListener('selected',function(e){
// this will give u the cursor location and length of selected text.
Ti.API.info(e.range.length +","+ e.range.location); 

});

win.add(txtArea);
win.open();
