  • You can just build your Mobile Web app to preview in browser. That will start Titanium Studio's internal web server and launch the default desktop browser. You can ignore the desktop browser and manually launch the iOS Simulator, then open Mobile Safari and enter the URL http:&#x2F;&#x2F;127.0.0.1:8020&#x2F;index.html.
