Titanium Studio will no longer start - "could not create the Java virtual machine" error

Titanium Studio was working fine yesterday, but when I tried to start it this evening, it comes up with an error message "could not create the Java virtual machine. A fatal exception occurred, program will exit". When I click OK, I get a really long error message which starts off by saying "Java was started but returned exit code=1" Titanium Studio does not come up at all. I am on Windows 7.

I recall there being some Titanium updates yesterday, which would have been applied upon startup today, so maybe that could have had an effect?

It's a really long error message and unfortunately it doesn't let me copy and paste the text, so here's a link to screenshot of it: http://s13.postimg.org/qalz7hmvb/Titanium_Error.png

Does anyone know how to resolve this?

  • Hi

    This looks like an easy problem with an easy fix (keep fingers crossed).

    Your error message screen-shot suggests you have Java 1.7 (AKA 7). If that is the case then that is the problem. Titanium only works with 1.6 (AKA 6). Downgrade to 1.6 and you should be fine. Obligatory reboots and project cleans as required.
