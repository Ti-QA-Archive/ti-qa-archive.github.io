ListView renders blank in ScrollableView

Hello,

I am using the Titanium mobile SDK (3.1.1 GA) , IOS 6.1. I tested this issue in the simulator and on the device.

I create a set of ListViews and add this ListViews to a ScrollableView so I can swipe (from left to right or backwards) through the lists.

This works well when I add 3 or less ListViews to the ScrollableView.

When I add more than 3 ListView the fourth ListView appears blank. The ListSection in the blank ListView does have the full collection of ListDataItems and the ListView items are being sized correctly. Even if I click a item in the blank ListView the click event is being triggered correctly, i see a message which shows me the correct data of the item i clicked on.

When I reach the fourth (blank) ListView and I swipe backwards to the third one i still see data in that one, when i swipe further back (second and first one) that ListViews are blank as well.

This seems to be a bug or am I doing something wrong?

Is there another way to do this or does someone have a workaround?

var win = Ti.UI.createWindow({backgroundColor: 'white'});

var listViewArray = [];

var plainTemplate = {
    childTemplates: [
        {
            type: 'Ti.UI.Label', 
            bindId: 'rowtitle',  
            properties: {        
                left: '10dp'
            }
        }
    ]
};

for (lvIdx=0; lvIdx<7; lvIdx++) {
    var listView = Ti.UI.createListView({          
        templates: { 'plain': plainTemplate },       
        defaultItemTemplate: 'plain'               
    });


    var data = [];
    for (var i = 0; i < 10; i++) {
        data.push({
            rowtitle : { text: 'LV ' + lvIdx + ' Row ' + (i + 1) },
            properties : {
                itemId: 'row' + (i + 1),
                accessoryType: Ti.UI.LIST_ACCESSORY_TYPE_NONE
            }
        });
    }

    var section = Ti.UI.createListSection({items: data});
    listView.sections = [section];

    listView.addEventListener('itemclick', function(e){
        alert(e.section.getItemAt(e.itemIndex));
    });

    listViewArray.push(listView);
}

var scrollableView = Ti.UI.createScrollableView({});
scrollableView.setViews(listViewArray);

win.add(scrollableView);
win.open();

Thanks.

Kind regards Stefan
