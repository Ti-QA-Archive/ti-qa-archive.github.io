Help to calculate the map bound using southwest and northeast lat lng

I have a map (Ti.Map.createView) which render routes from google direction api (https://developers.google.com/maps/documentation/directions/#JSON).

In the result json have bounds properties:

  • bounds contains the viewport bounding box of this route.
"bounds": {
      "southwest": {
        "lat": 34.0523600,
        "lng": -118.2435600
      },
      "northeast": {
        "lat": 41.8781100,
        "lng": -87.6297900
      }
}

please tell me how to use this to central the routes in map?

thanks you!

    To center the map in this bounding box you should set the region of the map like this:

    map.region = {
    latitude: bounds.northeast.lat - (bounds.northeast.lat-bounds.southwest.lat)/2,
    longitude: bounds.northeast.lng - (bounds.northeast.lng-bounds.southwest.lng)/2,
    latitudeDelta:(bounds.northeast.lat-bounds.southwest.lat)/2,
    longitudeDelta:(bounds.northeast.lng-bounds.southwest.lng)/2
}
