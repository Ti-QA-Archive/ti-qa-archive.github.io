  • Hi Steve,

    last week I have had the same issue. First I changed the ti.map to an older version (changing in tiapp.xml). Now the message disappears and the map runs. Since three days the error message appears, but I can see the map. There is also a JIRA ticket to this issue and a lot of entries in stackoverflow.

    Next issue is a timing problem: you must add the map after opening the window:

    win.addEventListener('open',function(){
    win.add(mapView);
})

    Hope this helps you and gives you an inspiration.
