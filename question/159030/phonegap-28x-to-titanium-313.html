History:
We developed this app for internal use only and had it working under Phonegap 2.1.x. When mgt decided we needed to make the app public. So, now we're looking at Titanium. I have it installed, I've been able to get Kitchen Sink to build, install and run. I've created a few test projects that basically run.

I don't mind porting everything but honestly, the demos that you have don't resemble our UI at all and I'm not seeing any tutorials on creating a UI (probably missing something).
