Send Titanium.Network.HTTPClient GET request with variables?

This is not working:

xhr.open("GET", 'myUrl');
xhr.send({
    variable : myVariable
});

and this is working:

xhr.open("GET", 'myUrl/?variable=' + myVariable);
xhr.send();

Is this normal behavior or there should be a way to send parameters over GET by filling some object?

  • That is normal: the object you pass to the send() function is converted into form-encoded POST data if the method is POST; with GET, you have to pass the parameters in the URL itself.
