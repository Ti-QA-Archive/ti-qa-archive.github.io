Uploading on desktop with HTTP Client, but no progress

I've created an uploader with the HTTP Client, and it works but the "onsendstream" never fires. It would be pretty nice to be able to use this. I assume the intended use is similar to below:

var hc = Ti.Network.createHTTPClient();
var file = Ti.Filesystem.getFile("somefile.txt");

progressBar.start();

hc.onload = function() { 
    progressBar.finish();
};

hc.onsendstream = function() {
    progressBar.update(this.dataSent);
};

hc.open('POST',url, true);
hc.sendFile(file);

    • In answer to your first problem, I got this working by setting the following request header before the sendFile call:

      hc.setRequestHeader('Connection', 'Close');

      And in answer to your second problem, try this:

      hc.setRequestHeader('user', someValue);
hc.setRequestHeader('password', someEncryptedValue);

      Hope this helps
