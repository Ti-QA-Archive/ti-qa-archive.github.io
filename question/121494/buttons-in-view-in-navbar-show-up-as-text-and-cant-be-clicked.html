Buttons in View in Navbar show up as text and can't be clicked

I want to have two buttons on the right side of my navbar (iPad).
Therefore I created a view and add both buttons to it.

var navbarView = Titanium.UI.createView({
    width: 200
});

var navbarButtonRight = Titanium.UI.createButton({
    title: 'X'
});

var navbarButtonRight2 = Titanium.UI.createButton({
    title: 'Benutzermenü'
});

The Buttons show up as text only and also events such as this aren't fired:

navbarButtonRight.addEventListener('click',function(e){

    You can also create a button inbetween the two buttons to fake a 2 button style.

    var flexSpace = Titanium.UI.createButton({
    systemButton:Titanium.UI.iPhone.SystemButton.FLEXIBLE_SPACE
});

    So create a button bar, add the first button, then the flexSpace, then the other button. Should give you 2 buttons.
