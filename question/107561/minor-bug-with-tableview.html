Minor bug with TableView

Hi,

I have recently noticed a bug with the TableView.

I use to attach some functions to my view elements. For example, with my TableView, I add a function called to get new data.

var tableView = Ti.UI.createTableView();
tableView.getNewData = function () {
   //something;
};

But, I have an unexpected error: when I add a function called 'get*' (here, getNewData).

Indeed, if I do :

Ti.API.info(tableView.getNewData()); 
/*
it prints: null
instead of: a list of 5 elements (result obtained when I call the function directly)
*/

OR

Ti.API.info(tableView.getNewData);
/*
  it prints: [object TiUITableView]
  instead of: the code of my function
*/

The solution to this issue for me was to rename my function with a name which didn't begin with 'get'.

Do you confirm this issue ?

— asked February 1st 2011 by Romain Salles
