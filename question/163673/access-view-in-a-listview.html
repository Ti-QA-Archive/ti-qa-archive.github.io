Access view in a listView

Hi,
I have a listView and on each row I have a button and I need to show a Popover from this button when I click on it.

So I have creates an event like this :

function handleBtnClickEvent(e){
    var item = e.section.getItemAt(e.itemIndex);
};

But how to get access to the button from the item ?
My popover need a "source" view but how can I get access to the button, I can get it's property but not the object (button) itself.

Regards

  • Hi

    As the listView is not made up of "views" in the same way as a tableView is - there are no views to return.

    As the documentation explains this;

    List view is designed for performance. One side effect of the design is that you cannot directly manipulate the views–add children, set view properties and bind event callbacks–as you can in TableView. Instead, you interact with list items indirectly, using templates and data items. The next subsection describes how to use list items.
