Android ActionBar Overflow Menu

I am just finally jumping into the new Android 3/4+ features. I have had no issues changing any properties of the actual actionBar but am unable to get the overflow menu working, it simply defaults the standard bottom menu bar from previous android versions.

win.activity.onCreateOptionsMenu = function(e) {
        var menu = e.menu;

        var item1 = menu.add({
            title : 'item1',
            showAsAction : Ti.Android.SHOW_AS_ACTION_NEVER,
        });
        var item2 = menu.add({
            title : 'item2',
            showAsAction : Ti.Android.SHOW_AS_ACTION_NEVER,
        });
        var item3 = menu.add({
            title : 'item3',
            showAsAction : Ti.Android.SHOW_AS_ACTION_NEVER,
        });
    };

This just doesn't seem right to me, but searching through the Q&A's show that folks got this code to work as the overlfow menu in the actionBar. What am I doing wrong? The documentation on this is pretty limited.

SDK 3.1 GA, Android 4.2.2 on Nexus 4.
I am building against a minSDK of 14

  • Upon further investigation it appears that this is just standard Android behavior. The menu is only available in the actionBar is the device has no hardware menu button.

    So it appears the only way to force the meny in the actionbar would be to create an ActionView to mimic the look of the standard overFlow menu.
