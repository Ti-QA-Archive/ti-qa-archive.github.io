  • Solved the issue by:

    1. Installing the ti.cloud module into the project from https://github.com/appcelerator-modules/ti.cloud/blob/master/commonjs/ti.cloud-commonjs-2.3.3.zip?raw=true
    2. symbolically linking ti.cloud.js into Resources/ui/module/ (cd Resources/ui/module ; ln -sf ../../../modules/commonjs/ti.cloud/2.3.3/ti.cloud.js)
    3. initializing it as var Cloud = require('ui/module/ti.cloud');
