  • I did it this weekend, all my projects worked as usual. I followed this steps:
    1.- Full uninstall of IOS SDK.
    2.- Upgrade to Lion OS.
    3.- Download and install Java.
    4.- Install all Lion updates (Itunes, etc).
    5.- Install from App Store IOS SDK in Lion.
    6.- Run Titanium.

    I modified some code in all projects and run it for Android and Iphone without any errors.
