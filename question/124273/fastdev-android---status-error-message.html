FastDev Android - Status Error Message

I seem to be having a real problem getting Titanium Studio and FastDev to work. I can't seem to get the emulator to connect to the server. I start the FastDev server from my project root: 

$: titanium fastdev start
$: [INFO] [2011-08-15 09:28:12,596] Serving up files for com.burchmedia.HelloWorld at 0.0.0.0:57817 from /Applications/MAMP/htdocs/titanium_apps/HelloWorld

Then open another terminal and here's what I get:

$: titanium fastdev status
$: Error: .fastdev.lock found in /Applications/MAMP/htdocs/titanium_apps/HelloWorld, but couldn't connect to the server on port 57817: [Errno 8] nodename nor servname provided, or not known. Try manually deleting .fastdev.lock.

Someone has asked a similar question, but gotten no response. Not sure if this is the same issue. Can anyone help me out? Studio looks really promising, but if I have to wait 60-70 seconds to recompile each time I change app.js (2-year old MB Pro with 8GB RAM..plenty fast), it's going to keep me from using it.
