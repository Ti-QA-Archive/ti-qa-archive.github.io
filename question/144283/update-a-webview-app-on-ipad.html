Update a Webview App on iPad?

Is there a way to update a webapp (webview loading a webpage) on an ipad? The files are all under myapp.app/content/… ?
I can read the files but I can't write into this folder.

var f = Ti.Filesystem.getFile("content", "file.html");
f.write("test");
Ti.API.info(f.read();) // outputs old values

doesn't seem to do anything.

— asked November 2nd 2012 by Michael G.

  • On iOS (on Android as well) you can only write to certain directories.

    For example this should work:

    var f = Ti.Filesystem.getFile(
  Ti.Filesystem.applicationDataDirectory,
  "file.html"
); 

f.write("test");
f = null; // it's good to free file resources after use

    Read more at http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.Filesystem
