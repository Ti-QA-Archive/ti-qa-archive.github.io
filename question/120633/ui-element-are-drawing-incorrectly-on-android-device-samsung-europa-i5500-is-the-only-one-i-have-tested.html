UI Element are drawing incorrectly on android device (samsung europa i5500 is the only one i have tested)

For example a buttons left and right border will become distorted by about a pixel or so and I also noticed that the border between TableView items is not displayed… however in both examples the UI elements are displayed correctly on the android emulator.

I am new to appcelerator so I was hoping someone could let me know if this is a known issue with my particular phone or an issue with appcelerator in general.

    I think this is caused by Android auto scaling, which can be disabled to allow you to explicitly manage your sizes/densities. This is done by setting anyDensity to true in the tiapp.xml file, as follows:

    <android >
    <manifest>
        <supports-screens 
            android:anyDensity="true"
        />
    </manifest>
</android>

    I hope this helps
