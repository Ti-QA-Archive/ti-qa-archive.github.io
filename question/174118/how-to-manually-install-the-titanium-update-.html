    • The reason for the endless prompts is because you have somehow managed to get onto an automatic update list that goes beyond the normal app - so basically you are on one of the several beta threads.

      You can stop this by;

      • In Studio
      • Help Menu
      • Select "Install New Software.."
      • Select "Available Software Sites"
      • Look down the list and find anything that is prefixed; "http://preview…"
      • Remove these entries
      • LEAVE the "http://download…" entry - that is the one you want

      If you get any further messages to update - simply download the full software from the resourcs page. This should stop them - but the above should have already done that.
