Info about getString

I want to know about the statement

Ti.App.Properties.getString

i looked at the documentation, there it was written that it returns a string value for a property.

But then i tried the following code 


var per= Ti.APP.Properties.getString(" hello");
var tf1= Ti.UI.createField({
    value:per,
    width:250,
    height:40,
    top:20
});

win.add(tf1);

Why it showed me a blank text field.??Why not hello written in the text field.

Thanks for your help.

    Most probably it's because you haven't set that property, you misspelled "APP" and "createField". :)

    Try this:

    
Ti.App.Properties.setString(" hello", "Mathew");

var per= Ti.App.Properties.getString(" hello");

var tf1= Ti.UI.createTextField({
    value:per,
    width:250,
    height:40,
    top:20
});

win.add(tf1);

    So … "Properties" stores and reads values - permanently.

    • "hello" is the key you use to store and retrieve values.

      So, you store values by key:

      "key1" => "value1"

      "key2" => "value2"

      and so on. When you use "setString", you define key which will store value. When you use "getString" you use same key to get value.

      Hope you understand.
