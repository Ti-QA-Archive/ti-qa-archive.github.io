Hi, is it still not possible to response to the iOS's audio remote control events?

I'm streaming some mp3 files from the internet and would like to have a remote control-ability to make users happy.

I have found a almost dead ticket here

and I saw a snippet of Objective-C code that receives the remote control event in blob/master/iphone/Classes/TiRootViewController.m, line 840,

- (void)remoteControlReceivedWithEvent:(UIEvent *)event
