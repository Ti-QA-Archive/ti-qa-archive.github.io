Helvetica Neue on Android?

I can see alot of example code for Android, and most of them use fontFamily:"Helvetica Neue". But that isn't standard for Android afaik?

Do I need to add Helvetica Neue as a custom font and rename it to "Helvetica Neue" in Resources/fonts for Android to also pick up the same font as iOS?

  • Main info on this can be found here

    Basically, add the Helvetica Neue font into the Resources/fonts folder. Make sure the file is named without spaces in the filename (apparently this can cause problems in Android). What you will then need to do is set up a quick check because Android and iOS use different methods to read fonts (filename and font name respectively). something like this works:

    font: {fontFamily: Ti.Platform.osname === 'android' ? 'HelveticaNeue' : 'Helvetica Neue'}
