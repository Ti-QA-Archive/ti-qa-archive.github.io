Backbone Filter works on simlulator but not on device

The follow code is used to filter a collection and works absolutely fine on the simlulator (ios7 & ios8) however when installed on the device the filtered result is always empty. There are no error so a liitle confused as to what the issue is.

$.search.addEventListener('change', function(e) {
    var filterCollection = clonedCollection.filter(function(guest) {
        if (guest.get('firstname').toLowerCase().indexOf(e.value) != -1 || guest.get('lastname').toLowerCase().indexOf(e.value) != -1) {
            return guest;
        }            
    });
    guestlistCollection.reset(filterCollection);
});

  • Found what was wrong, on the simulator is doesn't autocapitalise the search field where on the device is does, so just need to change the search term to lower case

    $.search.addEventListener('change', function(e) {
    var keyword = e.value.toLowerCase();
    var filterCollection = filtered.filter(function(eticket) {
        if (eticket.get('first_name').toLowerCase().indexOf(keyword) != -1 || eticket.get('last_name').toLowerCase().indexOf(keyword) != -1)
            return eticket;
    });
    eticketsCollection.reset(filterCollection);
});
