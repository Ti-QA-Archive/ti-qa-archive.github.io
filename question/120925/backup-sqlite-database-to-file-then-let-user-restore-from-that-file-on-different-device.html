Backup sqlite database to file then let user restore from that file on different device.

Let's say i do a database backup according to (http://developer.appcelerator.com/question/119423/how-to-backup-a-titanium-database) That makes sense but i want my user to be able to backup their database somewhere else and then later (after some device disaster) restore it is this possible? There is only local storage in my app so i can't send the db backup to a server somewhere and then retrieve it it would have to be done by the user locally somehow. I had the idea of sending the backup in an email I think that's possible, but how would i get it back onto another device? Any ideas, anyone have a similar problem?

  • two steps

    (1) allow the user to back up the database by packaging it up, probably in a zip file and then attach it to an email and send it to themselves

    (2) They can get the database back on the device by using iTunes File Sharing basically make sure you set the property on the plist, “UIFileSharingEnabled”
