Languages with alloy

Hi,

im developping with alloy :
i have this code to manage languages :

<Label id="shareOffreLabel" text="Envoyer à une amie" class="button"/>

i want to use L function, like this :

<Label id="shareOffreLabel" text="L('share')" class="button"/>

but it doesn't work..

i think its a bad practice to do it in controller…

is it possible in view ?

thanks

  • You do these in the styles actually. (not the markup)

    ####So in your case, your tss file would look something like this:

    "#shareOffreLabel": {
        text: L('share')
}

    • I finally got it working. I needed to have a locale specific Window title (but was using "text" in my .tss - as mentioned in this example).

      Here is how my .tss looks like now….

      "#Win1": {
    title: L("Win 1")
},
