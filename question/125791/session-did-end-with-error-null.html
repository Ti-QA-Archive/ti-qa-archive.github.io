It is barely an application, just an app.js and and include file for my very first experiment on titanium, however it did not go well. I got the following errors when I start it in iOS SImulator. I don't have a clue. Any recommendations?

    When you use require the methods assigned to your exports variable become the top level functions that you need to call. In your case LoginScreen becomes available, not create.
