  • jquery requires a dom, so you cant require() it.

    js files that need to be loaded into a webview need to not be named *.js, but something else. I use .ljs for 'local javascript). This keeps the library from being munged by the encryption of the build process.

    This works fine. jquery.ljs is in the same directory as the .html file that calls it.

    <script language="javascript" type="text/javascript" src="jquery.ljs"></script>
