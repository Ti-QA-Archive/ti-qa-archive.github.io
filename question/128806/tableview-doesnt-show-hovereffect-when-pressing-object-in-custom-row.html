Tableview doesn't show "hovereffect" when pressing object in custom row

In a normal tableview when i press an item that item turns orange (for example) during the press. The problem is that when i press an object in a custom tableviewrow, it only shows the hovereffect for that particular control, not the entire row. Is it possible to solve somehow?

  • It is not working becoz u r not touching on the row,u r touching or focusing on its child and the perticular child has its own specification.So the solution is to create a view as another child of row and set its height and width according to the height and width of row and make its zIndex higher than another child so that u can focus only on this view.set its selectedColor property to any color whatever u want and opacity.Ive sufferd from the same circumstances so if u need some code than ill.
