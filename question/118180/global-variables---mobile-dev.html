Global Variables - mobile dev

Wondering if there is a way to have global variables in mobile dev?

I have a had look at a few posts one saying you can pass variable between windows:

In main.js

    var win = Titanium.UI.createWindow({
        url:'blah.js',
        a = 1; //custom field
    });

Then in blah.js:

var win = Titanium.UI.currentWindow;

Ti.API.info(win.a)

I have also heard you can do:

Ti.App.Varname = {
    a: 1
}

Can anyone please confirm which is the best way or what other options I have in storing/passing variables with mobile dev?

Thanks in advance.

    You can do either of those things, depending on what you want to do. If you want to have a global variable that is accessible from anywhere in your application (meaning any context, such as the url .js of a window), you can put it in Ti.App, as in Ti.App.myVar = "someValue".

    Or, if it's something you want to pass to a window at creation, you can do it as a property of the window object as in your first example, although the syntax is a: 1, not a = 1.

    Hope that helps.
