  • The THREE YEAR OLD hack is already in the 3.1.0 SDK.
    It is probably NOT in the 2.1.4. You should hack the 2.1.4 SDK (but the binary you build with it will most likely be rejected by Apple) or UPGRADE. Keep in mind that the name of the player entity in that source file changed to movie from [self player].

    -(void)setCurrentPlaybackTime:(id)time
{
    if (movie != nil) {
        movie.currentPlaybackTime = [TiUtils doubleValue:time] / 1000.0f;
    } 
    else {
        [loadProperties setValue:time forKey:@"currentPlaybackTime"];
    }
}
