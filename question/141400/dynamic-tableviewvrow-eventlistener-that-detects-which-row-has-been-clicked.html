    When you create your tableViewRow add a property for the broadcast id:

    Titanium.UI.createTableViewRow({
    hasChild:true,
    broadcasat_id:broadcasts[i].broadcast_id
});

    Then put an event listener on the table and pull the id from the row.

    table.addEventListener('click', function(e) {
    Ti.API.info(e.rowData.broadcast_ie):
});
