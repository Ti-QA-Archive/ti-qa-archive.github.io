Hello,

I am new to Titanium with just basic knowledge of JS. I hope what i am asking is not dumb.
Is there anyway i can add an eventlistener just to a particular area in screen. I have an image on the screen, when clicked on the image an event should fire, but when the rest of the screen is touched the eventlistener should not fire.
Thanks to anyone who can help.

  • Accepted Answer

    set the size of the view, using the same size as the image. For example, if the image is 50 x 80 pixels, do something like this

    Ti.UI.createView({ 
    backgroundImage:'one.jpg',
    width:80,
    height:50,
    top:20, //place wherever you need
    left:20
 });

    tell me if it works

  • This is the code. Thanks for the help guys.

    `var view1 = Ti.UI.createView({
    backgroundImage:'one.jpg'
    });

    view1.addEventListener('click', function(e)
    {
    var sound = Titanium.Media.createSound();
    sound.url = 'one.mp3';
    sound.play();
    });

    var view2 = Ti.UI.createView({
    backgroundImage:'images/2_TWO.png'

    });

    view2.addEventListener('click', function(e)
    {
    var sound = Titanium.Media.createSound();
    sound.url = '2.mp3';
    sound.play();
    });`
