  • When you save an image in your app, you'd most likely do it using something like this:

    var img = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory, 'someimage.png');

    In this case, all you would want to do is write someimage.png to the reference field in your database that way you can access the image by selecting it from the database and then using the Filesystem in the same manner as was used to save the image.
