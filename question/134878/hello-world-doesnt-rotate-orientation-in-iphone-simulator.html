Hello World doesn't rotate orientation in iPhone Simulator

When using the default Hello World application, the orientation doesn't seem to change in the iPhone simulator. Using the iPad simulator it works fine. But the Kitchen Sink application works fine in both simulators. What is the Kitchen Sink doing that the Hello World isn't?

— asked April 3rd 2012 by Scott Blum

    You would need to change the orientation in tiapp.xml for iphone as follows:

    <orientations device="iphone">
    <orientation>Ti.UI.LANDSCAPE_LEFT</orientation>
    <orientation>Ti.UI.LANDSCAPE_RIGHT</orientation>
    <orientation>Ti.UI.PORTRAIT</orientation>
    <orientation>Ti.UI.UPSIDE_PORTRAIT</orientation>
</orientations>

    This is already done for ipad in the default application.
