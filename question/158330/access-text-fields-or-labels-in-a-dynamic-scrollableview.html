Access text fields or labels in a Dynamic ScrollableView

I'm building a small data base application I have added a ScrollableView with some lablles and text feilds in the ScrollableView. i have the ScrollableView updating when i push a new view into the views property so this works fine. This is done by a function that creates a view and adds it to the views property. the problem is i want to be able to edit the data in the view text fields and pass it to the database
My problem is, I can't access the ScrollableView text fields there is no handle to them. I've tried '.children' and '.views' and they both return a null error.
Anyone know how to get the views in a ScrollableView? i printed out the view and the child objects came back empty even tough i added many text filed and labels to the view. whats weird they display correctly as well. I am working with the android emulator on a windows 8 titanium is uptodate

  • You could create an object that holds references to the text fields then assign that object to a property of the scroll view. I've done that before to prevent having to loop through all child objects of a view.
