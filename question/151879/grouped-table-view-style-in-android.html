Grouped Table View Style in Android

I created a table view group in iPhone using the code style:Titanium.UI.iPhone.TableViewStyle.GROUPED. And it was ok in iPhone without problem, displaying a group row with title above it. Can anyone share how to do it in Android? Here is the reference for iPhone:

var data=[

{    title:"Lemon Parsley Prawns With San Marino Tuna Paella", 
    className:"tableRow", 
    hasChild:true, 
    dest:"originalRecipes/123.js", 
    font:{fontSize:12}, 
    header:"Tuna Paella"
}    
    ];

var tableview1=Ti.UI.createTableView({
        data:data,
        style:Titanium.UI.iPhone.TableViewStyle.GROUPED
    });

  • iOS automatically adds a default tableViewSection, android doesn't. So, add a tableViewSection and add your header text to that.
