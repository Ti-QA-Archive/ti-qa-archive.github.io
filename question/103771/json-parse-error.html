    I use this site to check all JSON feeds to see the format, JSON FORMATTER

    When I checked your feed there are errors and that the JSON is invalid. The JSON syntax requires the strings to be wrapped in quotes

    {
  "lhs":"1 U.S. dollar",
   "rhs":"0.735402265 Euros",
   "error":"",
   "icc":true
}

    To check this, you can create your own json file and test that.

  • It's not perfect but inserting the quotes works:

    var objectString = this.responseText.replace('lhs', '"lhs"').replace('rhs', '"rhs"').replace('error', '"error"').replace('icc', '"icc"')

var object = JSON.parse(objectString)
