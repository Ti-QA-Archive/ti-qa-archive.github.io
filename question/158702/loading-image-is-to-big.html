Loading image is to big

Hi,

If I am opening my application on iOS the default image which is shown on the loading screen is bigger than the actual screen size is. If I remove the status bar it is perfectly fine. Am I somehow able to resize the image shown in the loading screen or does it have to be the exact size to fit into the screen WITH the bar?

  • Accepted Answer

    Resize with height and width. Here's an example

    var screen_h = Ti.Platform.displayCaps.platformHeight,
    screen_w = Ti.Platform.displayCaps.platformWidth;

image.height=screen_h-20;
image.top=20;
image.width=Ti.Ui.SIZE; //this will auto adjust width proportionally to the specified height

    Now if the proportional width is greater than the screen width, the pic will overflow and in this case, if you want the whole pic to be shown, you would size by the width, like this:

    image.width=screen_w;
image.height=Ti.Ui.SIZE; //this will auto adjust height proportionally to the specified width
