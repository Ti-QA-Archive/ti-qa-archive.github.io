    @Giovanni - make sure you have the matching version of Node.ACS plugins with the base Titanium Studio. If you have the latest Studio 3.0.2 release, you would need to install Node.ACS 3.0.2 from http://preview.appcelerator.com/appcelerator/studio/acs/update/rc/. If you are on the beta stream of Titanium Studio (3.0.3), the Node.ACS update url to use would be http://preview.appcelerator.com/appcelerator/studio/acs/update/beta/. For the nightly version of Studio (3.1.0), the Node.ACS update site is http://preview.appcelerator.com/appcelerator/studio/acs/update/nightly/.

    Hope this helps.
