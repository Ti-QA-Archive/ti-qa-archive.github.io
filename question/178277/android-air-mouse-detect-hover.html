Android Air Mouse Detect Hover

I am developing an app that will be running on a device that has an Air Mouse. Meaning there will not be any touch events. I would like to make it so that when the mouse hovers over a button, it changes the image. Then if the user wants to click (which is the touch) it will do the action. My issue is that right now it changes the image when I click, but I would like to have it happen when I hover.
