    • If you want to reuse image view then every time create a new image view and deletes or null previous imageview.This will always work and will release memory.
      If you are adding your image view to window or view then first remove them from window or view
      and create new Image view and add them to window or view.

      If you set new property for image then actual image will no longer in memory.
