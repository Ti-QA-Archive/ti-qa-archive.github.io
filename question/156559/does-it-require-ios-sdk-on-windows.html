    • OK - I think you have become a bit confused. In order to test an iPad App you need to use the iOS simulator. As we have said - you CANNOT install the Mac software (xCode and the simulator) on a Windows PC. This is not a short-coming of Titanium, it is Apple who require you to use a Mac to build iOS Apps. PhoneGap is a different product where, as you say, you use their Cloud service to build and get a test IPA file back to install on your device. PhoneGap does NOT build native apps though - It just presents your html/css/js code inside a web view. Titanium builds native apps
