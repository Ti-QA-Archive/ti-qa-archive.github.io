Get text from the pickerWidget in Titanium Alloy

I m trying to use a widget called danielhanold.pickerWidget! By clicking on label the picker should popup and set the selected value as text on the label….i m a newbie to titanium and javascript so can anyone tell me how to get text of the selected value… 

function selectcity (e) {
   Alloy.createWidget('danielhanold.pickerWidget', {
   id: 'mySingleColumn',
     outerView: $.win,
     hideNavBar: false,
     type: 'single-column',
     selectedValues: [20],
     pickerValues: [{10: 'Auburn', 20: 'Bald', 30: 'Black', 40: 'Blond', 50: 'Brown'}],
     onDone: function(e) {
       // Do something
     },
   });
}

    If you look @ the close function on line 373 of the widget.js controller, you will see that it calls onDone with a _callbackParams object. So in your onDone function, the 'e' variable will contain the properties & values so you can access them like this:

        onDone: function(e) {
        Ti.API.debug(e.type);
        Ti.API.debug(e.id);
        Ti.API.debug(e.data);
        Ti.API.debug(e.cancel);
    }

    Hope this helps

        $.label.text = e.data[0].value;
