Ideal size for a windows

which is the correct size for a window that must see correctly in all resolutions?
like a lightbox
my code is:

var popupWin = Ti.UI.createWindow({
        backgroundColor : 'transparent',
        navBarHidden : true,
        id : 'popupWin',
    });
    popupWin.addEventListener("open", function() {
        this.activity.actionBar.hide();
    });
    //Event to close the popup window
    popupWin.addEventListener('click', function(e){
        if(e.source.id != null){
            popupWin.close();
        }
    });
    Ti.API.warn("device width "+Ti.Platform.displayCaps.platformWidth);
    popupView = Ti.UI.createView({
        backgroundColor : '#fbfbfb',
        height : Ti.UI.SIZE,
           width:Ti.Platform.displayCaps.platformWidth/2,
        borderColor:'#009fcd',
        layout:'vertical',
        borderWidth:1,
    }),

is correct use platformWidth/2 ?
the lightbox in tablet seen correctly but in phones of low and middle resolutions dont see fine.

  • Hi Andres,

    Thanks for your concern.

    Yes, it's correct to use Ti.Platform.displayCaps.platformWidth and it'll work for all resolutions. So you can use this feature and it's the best practice to do so. I've tested your issue for the possible number of low, middle and high resolution devices and it's working as expected. Check my test screenshot below.

    Test Code:

    var popupWin = Ti.UI.createWindow({
   backgroundColor : 'transparent',
});
var view = Titanium.UI.createView({
   backgroundColor:'#fbfbfb',
   width:Ti.Platform.displayCaps.platformWidth/2,
   height:Ti.Platform.displayCaps.platformHeight,
   layout:'vertical',
   borderWidth:1,
   borderColor:'#009fcd',
});
popupWin.add(view);
popupWin.open();

    You can also check the link: Why using percentages is bad idea?

    Hope it'll help you to solve your problem. If you have any concern about my answer then feel free to ask.

    Thanks
