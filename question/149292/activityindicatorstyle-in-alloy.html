ActivityIndicatorStyle in Alloy

Hi,

I'm trying to create an ActivityIndicator using Alloy. But I can't specify the style of it inside the .tss file, because the style is different from iOS and Android. From the documentation, I used the following code:

var style;
if (Ti.Platform.name === 'iPhone OS'){
  style = Ti.UI.iPhone.ActivityIndicatorStyle.DARK;
}
else {
  style = Ti.UI.ActivityIndicatorStyle.DARK;
}

It works, but I have to write it in my .js file… is there a way to do the same in .tss file?

Thanks!

    Use a device query in your tss. This assumes your indicator has the id "indicator".

    "#indicator": {
    style: Ti.UI.ActivityIndicatorStyle.DARK
}

"#indicator[platform=ios]": {
    style: Ti.UI.iPhone.ActivityIndicatorStyle.DARK
}
