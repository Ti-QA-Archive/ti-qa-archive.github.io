Animate children of a vertical layout view

I have a parent view set to vertical layout, and inside this I have 3 tiles, all of them are 130h*320w.

//View Structure

Window
    SomeImage
    VerticalLayoutView
        View1
        View2
        View3

What I want is when I clik one of them, animate it's height to '260', and actually it works, the problem is the order things work.

When I call view2.animate(height:260):

  • First the view3 is put down instantly
  • Them after that animation starts

I expected that the view3 would be going down, as the animation is ocurring.

Any ideas on how to fix this?

SDK:3.1, iOS

  • you have to animate both the views, view2 and view3, start animating view3 to moving downside and the view2 to height 260.

    or better try putting time constraint also, use duration:2000 like this.

    view2.animate({
height:260,
duration:2000
}):
