  • I don't know the exact cause of this - but what I did to fix it was copy off the resources directory, blow away the project (delete from Ti studio and remove contents from disk) - Then recreate the project. I then copied the resources back in through Ti studio and it will launch in iOS again. I'm guessing that something got added to source control that shouldn't have between my development machines and then synced into the project.
