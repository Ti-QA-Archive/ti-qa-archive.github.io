Force Landscape Orientation in Ipad only App - SDK 2.1.1.GA

Hello

I am using the latest Titanium Build (2.1.1.GA) and i am looking at building an iPad only app which only works in the Landscape_left or Landscape_Right orientations.

In the tiapp.xml file i have put the following under the GUID field (taken out the iphone option completely and kept just the landscape options for iPad:

<iphone>
        <orientations device="ipad">
               <orientation>Ti.UI.LANDSCAPE_LEFT</orientation>
               <orientation>Ti.UI.LANDSCAPE_RIGHT</orientation>
        </orientations>
    </iphone>

Then in my app.js i stated at the very top before anything else both:

Titanium.UI.orientation=Titanium.UI.LANDSCAPE_LEFT;

and also tried

Titanium.UI.orientationModes=[Titanium.UI.LANDSCAPE_LEFT, Titanium.UI.LANDSCAPE_RIGHT;];

All the above has failed to give me the desired landscape only option. I then read in the docs that Orientation had been depreciated since 1.7.2 - how can i achieve this?

How can i go about forcing my iPad only app to only display in Landscape modes and also start straight away this way (from the splashscreen without any rotating etc in the first few seconds)

Thank you

  • While creating the window set the orientationModes property of Ti.UI.createWindow to limit the orientations.

    Here is the sample:

    Ti.UI.createWindow({
        backgroundColor : "white",
        orientationModes: [
            Ti.UI.LANDSCAPE_LEFT,
            Ti.UI.LANDSCAPE_RIGHT
        ]
    });

    Hope this help!!!
