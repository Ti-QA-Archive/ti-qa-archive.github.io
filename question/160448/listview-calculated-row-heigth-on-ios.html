ListView Calculated Row Heigth on iOS

So I have a ListView which is dynamically added to on view initialization through JS code. I have the templates in the view XML.

Everything works great on Android, the ListView's row heights are calculated automatically using a height property of Ti.UI.SIZE. However, on iOS there is no such luck. I have looked into the forums for other answers, but there doesn't seem to be an ideal solution.

I am wondering if anyone has come up with a solution for this? I tried upgrading my Titanium SDK and Alloy to the latest beta versions to see if that would resolve the issues since they had the newest ListView spec. All this accomplished was nuking my Titanium Studio to no longer work with Android and iOS. I had to literally uninstall everything and reinstall to get working again.

Any help is greatly appreciated.

Thank you.
