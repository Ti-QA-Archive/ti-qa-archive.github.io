  • Aaron is right.

    But since you're getting a static RSS, you could download the HTML with XHR, which support overriding the user-agent and then pass the HTML to a webView.

    var xhr = Titanium.Network.createHTTPClient();
xhr.setRequestHeader('User-Agent','Mozilla/5.0 (Macintosh; U; 
Intel Mac OS X 10.6; en-US; rv:1.9.1.8) Gecko/20100202 Firefox/3.5.8');
