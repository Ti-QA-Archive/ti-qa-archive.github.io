Where to keep local html and javascript file in App?

My app will use webview to display some local HTML files. These HTML file will include some javascript files.

I now place all these HTML and javascript files in folder localHtml under &#x2F;{project}&#x2F;Resources&#x2F;

The problem is when build project, it try to validate the .js file inside localHtml as well, which is not necessary.

Where should I place these local files that I can refer to it inside Webview and skip build validation at the same time?

My app is for both iOS and Android.

    I think if you are putting javascript file inside project it should bydefault consider it for build.

    To exclude it from build you have option in Titanium Studio. Right Click on file and exclude from build.
