  • The first thing that came to my mind was to try to do it with regular expressions. The best I could do was:

    
var re = str.match(/\<li\>(.*?)\<\/li\>/ig);

    Where str is the contents of the description tag.

    This would give you an array in which each element is an li element and its contents. so re[0] would have the value '<li><strong>Artist:</strong> LYNYRD SKYNYRD</li>'.

    There's probably a better way to do this, but this at least what I can think of.
