Switching back to previous view

I have requirement where I want user to back to previous view when he click on a button.
I have two views one Thumbnail and other list(two button). At any given time user can choose one. When user click on thumbnail I am opening a new window with new view. User can switch to list view from thumbnail, reverse is true too. Android OS maintain a view (window ) stack when I use back button of emulator it give me the previous window. But I want it to control through code. If the window view does not exists I will create one but if it is already opened(not in view) I want it to bring it to front and push back the current view.

Please help.

    Hi Avaya

    View and window are totally two different things, views cant work by default as window. The requirement you saying like if the view is currently open then bring in front is partly not possible to do.

    We can only able to add and remove the view from the window at particular event requirement. In order to know which is the current view or the last opened view just maintain one stack or i say array of views. Push the newly open view and pop if the view been closed. so when the user close the current view you can get the entry of last view from the array and take necessary steps to add it again.

    I have done this kind of work in one app where the whole app having only one window and all others are just the view.

    I Hope that this atleast help you to get some clue.
