Titanium Mobile Web

Hi
I developed Titanium app for iPhone and Andriod phone which has some iPhone and
Android specific API used.

I tried to run same app by selecting Mobile web target in Android Emulator 2.0 Browser. But I'm getting black screen.

But If I create another app with some controls its working across platform.

Should i do any change in existing app, so that it will work in Mobile web?

Any one came across this issues in Titanium Mobile Web?

  • It depends on your application. Any references to Android and iOS specific namespaces (such as Ti.UI.Android) will not work on mobile web. In some cases there are equivalents (such as Ti.UI.MobileWeb), but you need to check on a case by case basis.

    There are also some other features, such as Ti.Contacts and Ti.Database that are not currently support on Mobile Web, which you will have to work around or disable.

    The documentation website for the 2.0 SDK is at docs.appcelerator.com/titanium/2.0/index.html and does a good job of listing what is, and is not available for Mobile Web.

    I recommend doing your initial debugging in a desktop browser. This way you can pull up the browser's debugger (command-shift-i in Safari on OS X) and check the console log for errors. It sounds like your application is crashing.
