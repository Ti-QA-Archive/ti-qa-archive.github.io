android map weird behavior when moving it for the first time

Hi,

I create a mapview in the app.js and I change its region later on the creation of the window that contains it ( I do that in order to have one mapview and use it in various windows and thus avoiding the one mapview restriction).

Initially it is centered to the selected (programmatically by me) annotation. It has a weird behavior though. Only the first time, after I touch it in order to move it, it centers to its (initially) top left point. After that it behaves normally centering to where I move it with my fingers. Anybody has an idea what's wrong or experienced this behavior?
