[ERROR] Analytics error sending request: A connection failure occurred

    <analytics>false</analytics>

I haven't written any code yet, I'm just using the wizard code with Tab 1 and Tab 2. Please let me know if this is a bug. I have a requirement that my app can't 'phone home' to any 3rd party service outside of my control. Is there a workaround?

I'm using:

Titanium Studio, build: 1.0.2.201107130739

Titanium Developer, Version 1.2.2

  • After saving your tiapp.xml file, did you do a clean build?
