How to access data of child-elements of a TableViewRow

I add a label to each TableViewRow.
Now I want to read the text-value of these labels.
How can I do this?

    You can access child elements by associating the child with the parent such as:

    row._username_label = Titanium.UI.createLabel({
    //whatever options you want
});

    Or by accessing the children like this:

    row.children[0] // Note the array is numbered so it CAN get messy if you have many elements added to your row

    Note that if you pick the first option, you do not want to do this:

    var username = Titanium.UI.createLabel({
    //whatever options you want
});

row._username_label = username;

    as it can have memory issues releasing the object down the road

  • there are a couple of ways, you could loop through the child objects of the row looking for the object with the id "label_id"

    var label = Titanium.UI.createLabel({
   id:"label_id"
});

    Or you could just associate the label to the actual row.

    row._label = Titanium.UI.createLabel({});
