  • Hello Saamy,

    Did you download older version of simulator? If not yet downloaded then please download via Xcode. Please follow my instruction for downloading older version of iOS simulator.

    1. Open Xcode
    2. Click on preferences
    3. Click on download tab
    4. Select your desired simulator under component section
    5. Then you can select simulator from Ti Studio
