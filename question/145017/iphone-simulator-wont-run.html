iPhone simulator won't run

I am having a problem with my iPhone simulator when working on Mobile apps. When I run it, it starts up, I see the iPhone, then it shuts down.

This is the message I receive in the console:

[INFO] One moment, building ...
[INFO] Titanium SDK version: 2.1.3 (10/02/12 16:16 15997d0)
[INFO] iPhone Device family: universal
[INFO] iPhone SDK version: 6.0
[INFO] iPhone simulated device: iphone
[INFO] Minimum iOS version: 4.3 Linked iOS Version 6.0
[INFO] Launching application in Simulator
144:152: execution error: iPhone Simulator got an error: Connection is invalid. (-609)
[INFO] Launched application in Simulator (4.27 seconds)
[INFO] Application has exited from Simulator

My dashboard says that my iOS SDK is installed correctly, and my iPad Simulator works perfectly fine. I'm really not sure what the problem is. It was working before, then all of a sudden it's automatically shutting down for no reason.

Any help at all would be appreciated. Thank you!

  • Hi Liz

    I am going to assume that it has worked before in the app you are currently working on.

    Rule out test: I would suggest creating brand new project - accept all default and run it. If the simulator starts it is not that - which would be unlikely, but worth checking.

    This can happen often if you are using the CommonJS model (which I recommend) but have an error in the code that it not an obvious structural thing, like missing commas between new parameters (although worth checking).

    If you have recently added some code to a CommonJS module, go back to that section and check that you have named everything correctly, for example;

    • You have a parent view called views
    • but type view.add(something); - note the missing 's' in 'view'.

    This can often trigger one of these weird immediate closures.

    If not 'rem' out the last code you added since the last successful test and start working through it.

    It is worth running your code through jslint.com as you go as it is amazing how accidental errors can occur when you are up against a deadline. If you have never used this before go to the bottom of the page and turn on SOME of the forgiveness features - like messy white space - or you will be there all day.
