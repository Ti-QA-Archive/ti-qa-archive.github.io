    • Sorry for digging this up, but in case he or anyone is still looking for a fix and misunderstood Jesus:

      I think he means, that you have to uninstall the new alloy version and go back to using (in your case propably) 1.5.0.

      Do that by uninstalling alloy and then by explicitly installing alloy 1.5.0:

      (sudo) npm uninstall -g alloy
(sudo) npm install -g alloy@1.5.0

      Use sudo if you are on mac or linux and omit it if you're on windows.

      Doing this fixed an issue, that was very similar to yours, for me at least.
