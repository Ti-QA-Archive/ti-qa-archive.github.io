  • According to JSON site: If the stringify method sees an object that contains a toJSON method, it calls that method, and stringifies the value returned. This allows an object to determine its own JSON representation.

    So I guess Ti.Database implements toJSON which return {}. BTW, you cannot see stringify value doesn't mean it won't work. Try this:

    Ti.API.info(Ti.Database);// Output: [object DatabaseModule]
Ti.API.info(JSON.stringify(Ti.Database));// Output: {}

var db = Ti.Database.open('test');
alert(db.file.nativePath);
