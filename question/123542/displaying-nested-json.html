displaying nested json

I have a json file that gives some info about an author on my blog along with a list of their posts. 

{"author":true,"comment_count":19,"post_count":21,"bio":"Hi and welcome to my bio.","avatar":"4b6c7f4a-55c0-4457-ad5e-130945a3e07a.jpg","latestPosts":[{"Post":{"title":"Dark & Stormy","description":"my first post","notes":"hello world"}},{"Post":{"title":"Voila! ","description":"my second post","notes":"Why not?"}}]}

I send an xhr request to my server to get this data and when its returned I parse the json, set up some vars then add my content to a scroll view.

I've been able to show the authors comment count, post count, bio and avatar but when it comes to showing a list of their latest posts the nested json) I'm not sure how to do it. Any suggestions or examples would be welcomed.
