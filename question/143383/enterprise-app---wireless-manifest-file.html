Enterprise app - wireless manifest file

Hi,
We have developed an enterprise app and now looking to distribute it to our employees via a web server.
I went over Apple's documentation and everything is pretty clear except for one thing - they say I need a wireless manifest file which should be created by Xcode: "The manifest file is created by Xcode, using information you provide when you share an archived app for enterprise distribution.".
Any idea where can I find this file or how can I create it?

Thanks,

  • OK I found an elegant way to do it -
    in titanium - build an app for APPLE ITUNES STORE (not ad hoc)
    in xcode go to archives, click on the app, distribute, save for enterprise or ad hoc. this will create both an ipa file and a plist file
