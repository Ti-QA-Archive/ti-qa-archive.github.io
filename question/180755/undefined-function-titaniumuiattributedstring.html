  • Accepted Answer

    The following line:

    el[conf.attr] = Titanium.UI.AttributedString({

    Should be:

    el[conf.attr] = Titanium.UI.iOS.createAttributedString({

  • Ah I tried with .iOS prefix, however completely overlooked the magical create keyword for the function, thanks

    Additionally I had to swap out some other bits to make it work fully.

    var texts = [
            {
                'id': 'login_email',
                'text' : 'email address',
                'match' : 'email address',
                'attr' : 'attributedHintText',
                'type' : 'ATTRIBUTE_FOREGROUND_COLOR',
                'value': "#fff"
            }
        ];

        _.each( texts , function( conf ){

            var el = $[conf.id];

            el[conf.attr] = Ti.UI.iOS.createAttributedString({
                text: conf.text,
                attributes: [
                    {
                        type: Ti.UI.iOS[conf.type],
                        value: conf.value,
                        range: [conf.text.indexOf(conf.match), (conf.text).length]
                    }
                ]
            });

        });
