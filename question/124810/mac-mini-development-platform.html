Mac Mini Development Platform

I have read that I need a mac for iphone/ipad/etc development but can I use my mac mini using Titanium to develop and deploy for the android world? Will I need to create a Windows VM on the mac mini to do android development? Thanks in advance.

    Sure. You do not need any virtual machine, Titanium Studio for mac can run directly the android emulator, install on an android device and deploy your app. I has been using a Mini Mac for a while developing for both platforms without problems.
