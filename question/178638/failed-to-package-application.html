failed to package application

Hey guys,

I'm getting the error..:

[ERROR] : Failed to package application:
[ERROR] Application Installer abnormal process termination. Process exit value was 1

..when i start running my app.

I tried to upgrade titanium to 3.4.0 - nothing changed.
Downgraded it back to 3.3.0 but its still the same error.
It doesnt metter which project I start, I even tried a new one - same error.
I tried running it on emulators - same problem.

I dont know what else I can do.. any ideas?

Application type: mobile
Titanium SDK: CLI version 3.4.0, Titanium SDK version 3.4.0.GA

Platform & version: Android 4.2.2

Device: HTC One X

Host Operating System: Windows 8.1

Titanium Studio: Titanium Studio, build: 3.3.0.201407100905

Thanks in advance!

  • How to fix it:

    Go to android SDK manager and install 'Android SDK build tools revision 20'.
    Go to Path\android-sdk-win\build-tools and delete all except 20.0.0
