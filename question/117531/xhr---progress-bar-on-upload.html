XHR - progress bar on upload

Hello,

I'm trying to make a progress bar which give the user information about the upload progress of an image (around 40/50KB).

For this, i'm using the onsendstream event, to get the onsendstream.progress value.

I've also added the ondatastream event just to see what it do.

But my issue is that, while the doc says that this value must be from 0.0 to 1.0, I get very strange values.

Everything is okay when I launch the request with the send method (using method "POST" and giving the image as an argument to this method), the onsendstream.progress value goes from 0.0 to approximatively 0.5, and then, as soon as the ondatastream event fires (giving me ondatastream.progress = 1), I get negative value from the onsendstream.progress, very random values like '-0.2335' or '-0.9843'.

My request seems to stop when my ondatastream.progress finnaly gives me a value around 3.000 and 4.000, which is also strange since the doc says this value must also be from 0.000 to 1.000.

I hope someone can understand those weird values, and tell what to do in order to have a normal ondatastream.progress value and so a normal progress bar. Note that the file is successfully uploaded to my server.

Thanks.

  • Ok Aaron, thanks for your help. Here is the code in order to send the photo:

    var sendPhotoReq = Titanium.Network.createHTTPClient({connectionType:'POST'});
 function sendPhotoFunction()
{
var url = 'http://'+ipServer+'/edit/'+spotList[actual_index].id+'?public_key='+Ti.App.Properties.getString("public_key");

 var params = {image:imageToSend};

 sendPhotoReq.setRequestHeader("Content-type", "image/jpg");
 sendPhotoReq.open("POST", url);
 sendPhotoReq.send(params);
}

    Note that this function is launched by a button once the user wants to upload the image, and that imageToSend is a copy of the value event.media returned by the showCamera.success method

    Here are the other methods called by the HTTPClient object:

    sendPhotoReq.ondatastream = function(e)
      {
          Ti.API.info('*****ONDATASTREAM - PROGRESS: ' + e.progress);

      };

sendPhotoReq.onsendstream = function(e)
      {
          Ti.API.info('#####ONSENDSTREAM - PROGRESS: ' + e.progress);

      };

sendPhotoReq.onload = function()
{
    var json = this.responseText;
    var data = JSON.parse(json);
    if (data.status == 200)
    {
            Titanium.API.info(data.response);
            Titanium.API.info(data.status);
    }
    else
    {
    Titanium.API.info(data.response);
    Titanium.API.info(data.status);            
    }
};
