How to target iPhone 6/6+

I am in the process of updating my app for the latest iPhone 6 and 6 + dimensions.

Currently, to differentiate between the iPhone 4 and 5 screen sizes i use Ti.Platform.displayCaps.platformHeight as so:

// if iphone 5
if(Ti.Platform.displayCaps.platformHeight===568)
{
    var image1= Ti.UI.createImageView({
        width:320,
        height:568,
        backgroundImage:'images/backgrounds/img@568h.png'
    }); 
}
// iphone 4 or less
else
{
    var image = Ti.UI.createImageView({
        width:320,
        height:480,
        backgroundImage:'images/backgrounds/img.png'
    }); 
}

On an iPhone 6 and 6+ though - Ti.Platform.displayCaps.platformHeight always comes up as 568 and if i try platformWidth that comes in as 320 on all devices.

How can i determine between the iPhone 4,5,6 and 6 plus so i can serve up different images and heights etc within the app

  • You're still not using Titanium to it's potential.

    Titanium takes the pattern that Apple has established for the splash image filenames, and has applied it to normal images. So your code just needs this:

       var image1= Ti.UI.createImageView({
        width:320,
        height:568,
        backgroundImage:'images/backgrounds/img.png'
    });

    Then, name your files accordingly:

    img.png
    img@2x.png
    img-568h@2x@2x.png
    img-667h@2x.png
    img-736h@3x.png

    Ti will pick them up accordingly.
