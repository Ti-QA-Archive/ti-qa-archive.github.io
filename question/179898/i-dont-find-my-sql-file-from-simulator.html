    Xavier,

    We use a great little utility called SimPholders.
    It will quickly allow you to get to the location of a particular app installed on your iOS Simulator.

    Be sure to get the v2 alpha if you are using xCode 6.x

  • You can use this code:

    var db = Ti.Database.open('DUMMY');
console.log(db.getFile().nativePath);
// or shorter:
console.log(Ti.Database.open('DUMMY').file.nativePath);

    Now you can obtain the path.
