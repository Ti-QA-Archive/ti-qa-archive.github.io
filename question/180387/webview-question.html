Webview Question

When using a webview what rendering engine is used? Is it the same for each OS version or does it depend on what browser the user has selected as the default?

I have an app that is basically a hybrid style app primarily displaying html5 via webview. I'm getting users of the same device describing wildly different experience so I'm trying to understand if their own device settings come in to play when using a webview.

  • Hi Frank,

    Titanium uses the same webview component for the device's undelying OS. For all currently supported platform/OS that component is WEBKIT. So yes the WEBKIT is the engine that actually renders the webpage in web browsers and same goes for titanium webview.
