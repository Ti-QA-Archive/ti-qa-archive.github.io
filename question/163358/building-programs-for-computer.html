    No, no longer. A few years ago we offered desktop app creation tools. We divested that to focus on mobile. You could check out the TideSDK project or NodeWebKit for options to create desktop apps in JavaScript.

    That said, our Hyperloop project will let you write desktop applications for Windows 8. It's in an early stage of development and not production ready.
