  • Hi

    Your description is not very clear, not sure exactly what multiply means.

    But I will guess you mean stretch. If this is the correct - guess read on, if not, please explain exactly what you mean.

    My Guess;

    You want to have an image appear as if it wraps the text you are going to display, like the chat bubble in the message app on the iPhone, the stretch to allow the text to appear as if it inside the space.

    You have to use a view that stretches the image and use that as a backgroundImage.

    This is an example of what you should do, change it to suit your actual needs.

    var outer = Ti.UI.createView({
    height: Ti.UI.SIZE,
    width: Ti.UI.SIZE
});
var back = Ti.UI.createButton({
    backgroundImage: '/mg/my/alert-row.png',
    height: Ti.UI.SIZE,
    style: Ti.UI.iPhone.SystemButtonStyle.PLAIN,
    touchEnabled: false,
    width: Ti.UI.SIZE
});
outer.add(back);
var lbl = Ti.UI.createLabel({
    bottom: 10,
    height: Ti.UI.SIZE,
    left: 10,
    right: 10,
    text: mission.status,
    top: 10,
    width: Ti.UI.SIZE
});
outer.add(lbl);

win.add(outer);

    To make this cross platform compatible I have NOT added the label into an image (or button in this case) as this does not work on Android.

    Notes;

    • You should place a width restriction on the label otherwise it will fill the device width
    • I used the button view as the background as it support stretching backgrounds
    • The button is rendered as a normal view
    • The image you choose must be the exact same size or smaller than the minimum size of the width and height - otherwise it will not stretch correctly.
