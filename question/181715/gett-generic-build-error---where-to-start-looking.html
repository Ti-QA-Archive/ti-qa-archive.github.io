  • Hi Rusty, this error has been asked many times in different scenarios here, but there's no foolproof solution until yet.

    However you could perform some few steps as i have already faced this issue many times in starting when i switched Titanium to different PC:

    • Delete everything related to Titanium, even in temp folder….do not keep any previous files
    • From titanium documentations, follow each step in sequence & keep every software ready with you.
    • Follow each step & have a fresh installation again.

    I know its quite tedious task, as it was for me, but before performing above steps try to change API level to 14 & keep as many API levels installed in Android SDK.

    This issue might be due to JAVA SDK version, so try to check that also..

    Hope you get rid of this error without installing Ti again.
    Good Luck!
