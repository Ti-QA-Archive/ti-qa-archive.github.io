Displaying data in a webview

Working with Android 2.1, MobileSDK 1.7.0

I have a variable : header (it's a string). I need to display this in a webview 

//header has been assigned somewhere
var view = Ti.UI.createWebView({url:"view.html"})
view.addEventListener('load', function()
{
    Ti.App.fireEvent('ready', {header:header})
})
win.add(view)

view.html : <a href = "http://pastie.org/2047016">http://pastie.org/2047016</a>

The problem is, when the webview loads, it first displays a page with black background, and "hello" written. This appears only for an instant, this is then covered by a white page with the header written in black.

Any workaround for this?
Also, is there some better method to pass data to a webview?

  • I would use jQuery to to this with something like:

    <html>
    <header>

            Ti.App.addEventListener('ready', function(e) {
                $('.text').html(e.header);
            })


    </header>
    <body bgcolor="black">
        <pre>
            <font color="white" class="text"></font>
        </pre>
    </body>
</html>

    Or else I would use the javascript replace method on the html in the js side of your app (outside the webview)
