  • The examples provided in the Cloud API doc work as is. Just customize it with your particular values

    Cloud.Users.update({
    email: 'joeuser@mycompany.com',
    first_name: 'joe',
    last_name: 'user',
    custom_fields: {
        favorite_color: 'blue',
        age: 25
    }
}, function (e) {
    if (e.success) {
        var user = e.users[0];
        alert('Success:\n' +
            'id: ' + user.id + '\n' +
            'first name: ' + user.first_name + '\n' +
            'last name: ' + user.last_name);
    } else {
        alert('Error:\n' +
            ((e.error && e.message) || JSON.stringify(e)));
    }
});

    Remember, any custom fields you create must be accessed as a sub-set of custom_fields. So, say you added a phone number field, you can access it in the success callback as user.custom_fields.phone_number. That tripped me up for a little bit.
