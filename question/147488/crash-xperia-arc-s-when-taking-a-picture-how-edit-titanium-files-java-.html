    • Jordann, you seems new to the community. So, please if you want to comment don't add a new answer but use add comment link in the answer given.

      About the source code editing,

      1> You need to download the complete source code and add it as a project in eclipse to edit and compile.

      2> After you are done with that and your code successfully compiles then take a backup of /Library/Application Support/Titanium/mobilesdk/osx/3.0.0.GA/android/modules/titanium-media.jar and replace your compiled file in the jar.

      3> Now run your app just like before and it will pick the changes.

      I hope this helps
