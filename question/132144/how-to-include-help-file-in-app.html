How to include "Help" file in app

Hello all, I've got an app that already has a file browser in it and am wanting to include a "HELP" file in that browser by default for new users to read.

Right now i'm creating the file the first time the app runs and using file.exists() to not run it again. But the problem is that i can't update the file b/c once it exists it will always be ignored.

Anyone have a low-level approach to doing this? One that will allow updates?

Thx!

Ti SDK version: 1.8.1
iPhone Device family: ipad
iPhone SDK version: 5.0

    After first app opening set a ti.app bool variable to true, so you know, that the user already read your help file.

    I would do it in a modal window
