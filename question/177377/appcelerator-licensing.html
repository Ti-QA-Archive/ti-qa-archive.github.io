Appcelerator licensing

I have explored other hybrid framework such as Sencha, Ionic, etc. I want to try out Appcelerator, but have read much about licensing confusion. I saw those post were quite old, and want to know the status now. Other framework's website pricing is clearly stated, but I noticed Appcelerator only have "Free" and "Enterprise" (without price given). I guess I may have been confused by other older posts, and in order not to waste my time exploring this framework that eventually going to be costly, I would appreciate some confirmation regarding pricing. I may develop app using appcelerator and put in appstore.

  • A quick answer.

    If there is just you or a few, then you can use Appcelerator for free. You do not need to worry and there is a good load of API calls and Push notifications available from the ACS, within this free package

    If you are a corporate customer, you may wish to consider an enterprise account, but these are quite expensive, and being honest if you have an app which requires the enterprise facilities then you will be able to afford it.

    If as you say, you want to develop an app then go ahead and do it, using the Titanium Studio and free licenses.

    hope this helps
