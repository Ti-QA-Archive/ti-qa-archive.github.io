  • I downloaded the new mobile SDK using the Titanium SDK and then went to the Edit choice for Kitchen Sink and reviewed each field and noticed on Titanium SDK and it was set to 1.2.1 and I changed to 1.6.1.

    The KitchenSink sample app then launched properly in the iPhone emulator.
