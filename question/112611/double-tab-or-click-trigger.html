Double tab or click trigger?

*RESOLVED - see post by Aaron Saunders

Hi, is there a way to trigger a double tab or double click after opening a WebView? I'm trying to zoom into a local HTML file once it is opened. It has a fixed width and height, and when it is opened it doesn't scale the whole screen. I have the window option scalesPageToFit set to true so when you double tap the page scales to the full screen. You can also pinch zoom.

My concern is about user experience because the page isn't readable until you double tap or pinch zoom. I'm looking to scale without the double click. Thanks.

This is on an ipad by the way.

webview = Ti.Ui.createWebView({
backgroundColor: '#FFF',
scalesPageToFit:true
});
