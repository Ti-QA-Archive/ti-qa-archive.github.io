Picker: can the initial value be blank (none of the choices pre-selected)?

I have a Picker with 3 choices. The user must make a choice, but I don't have any way of knowing what a good default choice is. So I want the unfocused picker to have no value in it at all, but when the user clicks on it to bring up the choices, I only want the 3 valid choices shown. Is there any way to do this?
<br><br>
I've tried:<br>

TheCombo.setSelectedRow(0,-1,false);

but it doesn't work (the first item is pre-selected).

Titanium Studio 1.0.1.201106171427<br>
Titanium SDK 1.7.1<br>
Andriod 2.2 API

  • Theres not really a way to do this the way you want. You kind of have to treat it like an HTML select box. Create an empty row in your data array and set it as the default. Then on selection, you just have to make sure that the selected index is not 0 before continuing.
