Alloy: setting fontSize with SP value for android

hi, I tried to set the fontSize with sp unit in .tss like so:

    font:{
        fontSize: 17sp
    }

and:

    font:{
        fontSize: '17sp'
    }

but both don't work.
What's the proper way to set this?

thanks!!

  • I don't believe Titanium supports "sp" units. You can however use "dp" units. The only difference is that the "dp" won't scale by user font size preference. Both will use density independent pixels though.
