Odd behavior of headerView and table style in a SplitWindow with SDK 1.8.1 - iPad

Hello all!

I have a SplitView and a masterWindow with a tableview (SDK 1.7.5). To customize the look of the tableview's section headers, I defined headerViews pointing to views (viewOne and viewTwo):

SplitViewNav = {};

SplitViewNav.masterWindow = Ti.UI.createWindow({backgroundColor:'#ffffff'});
SplitViewNav.detailWindow = Ti.UI.createWindow({backgroundColor:'#ffffff'});

var viewOne  = Ti.UI.createView({height:'auto'});
var viewTwo  = Ti.UI.createView({height:'auto'});

var settingsTableData = [];
settingsTableData[0] = Ti.UI.createTableViewSection({
    headerView:viewOne
});

settingsTableData[1] = Ti.UI.createTableViewSection({
    headerView:viewTwo
});

...

// create table view
var tableview = Titanium.UI.createTableView({
    data:settingsTableData,
    style:Titanium.UI.iPhone.TableViewStyle.GROUPED
});

SplitViewNav.masterWindow.add(tableview);

SplitViewNav.splitView = Titanium.UI.iPad.createSplitWindow({
    masterView:SplitViewNav.masterWindow,
    detailView:SplitViewNav.detailWindow,
    showMasterInPortrait:true
});

After rebuilding with 1.8.1, I see the masterWindow but the detailWindow is not shown (it's hidden). Surprisingly, if I remove the headerViews I see the detailWindow again:


settingsTableData[0] = Ti.UI.createTableViewSection({
    //headerView:viewOne
});

settingsTableData[1] = Ti.UI.createTableViewSection({
    //headerView:viewTwo
});

I can also see the detailWindow if I enable the headerViews and disable the tableview.style property (or if I set it to Titanium.UI.iPhone.TableViewStyle.PLAIN).

var settingsTableData = [];
settingsTableData[0] = Ti.UI.createTableViewSection({
    headerView:viewOne
});

settingsTableData[1] = Ti.UI.createTableViewSection({
    headerView:viewTwo
});

...

// create table view
var tableview = Titanium.UI.createTableView({
    data:settingsTableData
    //style:Titanium.UI.iPhone.TableViewStyle.GROUPED
    //it also works if "style:Titanium.UI.iPhone.TableViewStyle.PLAIN"
});

I`d like to keep my "GROUPED" tableview and custom headers… :(

Any ideas???

Thank you very much

Luis
