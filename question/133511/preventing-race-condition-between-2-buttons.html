Preventing race condition between 2 buttons

I have 2 buttons, both have event listeners, both does the same thing which is to animate and move up. However, I wanted it in a way where by only one button can move up. The next button has to wait for the button to finish the job first being able to move up.

However there's a race condition. If I clicked both buttons too fast, both move up. I tried a few approaches of preventing it but all failed.

An example :

var isRunning = false;

buttonOne.addEventListener({
execute(buttonOne);
});

buttonTwo.addEventListener({
execute(buttonTwo);
});

function execute(button) {
if (isRunning == false) {
isRunning = true;
runAnimate(button);
isRunning = false;
}
}

function runAnimate(button) {
//Do Something
}
