Function

I have a problem with a function .

I create a function i put it in a file fun.js .

and have several files as app.js / win1.js / win2.js i have included the fun.js in the app files but when i try to call the function from the others files she didnt work i have to include it in the file where im calling it.

any solution to include the function in app file and call it from all my other files ?

    • Based on the definition of the url property of a webview,

      url to a JavaScript file with the windows instructions. this window will create a new JavaScript sub-context that will run in its own thread and global variable space.

      I don't think those files are being included in the same way that Ti.include does.

      You can try including all the files to the app.js file. Then instead of using the url property on the webview, call a function inside the win1.js file to open the window. Here is a working example:

      app.js

      Ti.include('fun.js', 'win1.js');
openWin1(); // found in win1.js

      win1.js

      var win1_window = Ti.UI.createWindow({
    width:'100%', height:'100%',
    backgroundColor:'#CCCCCC'
});

var openWin1 = function()
{
    win1_window.open();
};

win1_window.addEventListener('open', function(e){
    haveSomeFun(); // found in fun.js
});

      fun.js

      var haveSomeFun = function()
{
    Ti.API.info('Good times!');
    alert('Good times!');
};
