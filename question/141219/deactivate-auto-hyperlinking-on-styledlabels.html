Deactivate auto-hyperlinking on styledlabels

Hi,
We are using the styledlabel module on a project, and it keeps auto-hyperlinking sequences of numbers (presumably because it thinks they are phone numbers). These are not intended to be linked and I don't know how to turn that feature off.

Unless it is well hidden, the only documentation for this module (which I downloaded from it's marketplace page) is almost non-existent. It didn't give any insight about the controls properties, methods, or events at all. It is inconcevable to me that a paid-for module could have poorer documentation than the standard API; but that is a question for another time.

For now, can anyone suggest how I can prevent this behaviour?

Thanks

    Hi Nik

    You are obviously correct - this is the standard behaviour and that is by design, but not because of the StyledLabel, it is because the label is rendered exactly as a web page and as such uses the safari html rendering engine in the background.

    Because of this is uses the same method to fix the problem as a normal web page would.

    If you prefix any strings for the label with;

    <meta name="format-detection" content="telephone=no">

    Your issue disappears.

    This is a full example;

    var label = StyledLabel.createLabel({
    height: Ti.UI.SIZE || 'auto',
    html: '<meta name="format-detection" content="telephone=no">1198-206 3115-409 6192-114 2193-213 9014-780 Example Text <b>wait</b>.',
    left: 5,
    right: 5,
    top: 5,
    width: Ti.UI.SIZE
});

    That should solve it. If not - let me know.
