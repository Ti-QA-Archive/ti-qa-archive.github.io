Problem comparing 2 strings

Hi,
in order to detect if the content of a text field is a RTF text (from RTF file) , I do :

var firstChars = win.rtfContent.substr(0,5);
Ti.API.info(firstChars);

if (firstChars == '{\rtf') { // it's a rtf string

}

The problem here is that Ti.API.info(firstChars) shows '{\rtf' but (firstChars == '{\rtf') return always false.
Really strange.. any idea ?
I have also check the length in case of it as a length of 5.

Thanks

    You need to escape the backslash in your if statement like this:

    var firstChars = win.rtfContent.substr(0,5);
Ti.API.info(firstChars);

if (firstChars == '{\\rtf') { // it's a rtf string

}
