How to get the number of children on BlackBerry paltform

I'm developing an app for iOS, Android and BlackBerry.

In iOS and Android i don't have a problem getting the number of children. I just use this snippet

mainWindow.children.length

But in BlackBerry it console logging mainWindow.children returns undefined.
I checked the docs and filtered to only have BlackBerry class members and children RO is there.

Any idea on this? or is there another way how to get the number of children w/o me using the mainWindow.children.length snippet?

    Have you tried using the getChildren() method? Declare a variable to hold its value and then call length afterwards if you are using a loop.

    var children = mainWindow.getChildren();

Ti.API.info(children -->> + children.length);

    Otherwise try debugging out your window object to see how children are stored inside of the object.
