    Hi

    Titanium is a cross platform app development system. It is designed to provide a very simple way to create an app that "could" be used on more than one platform using the same code.

    There are differences for each platform, and you are able to provide code that says; if iOS do this, but if Android do it this way. Those are usually limited to the sections of the platform operating systems that do not have direct equivalents, for example;

    • iOS has a navigation controller that handles windows opening and automatic back button creation
    • Android does not, it does have other ways to achieve the same goal though, including ActinoBar on newer versions of the Android system.

    But in short you write your app in JavaScript, add any logic for platform specific behaviours and then Titanium will compile your app to iOS or Android, or both. You also now work with Blackberry and soon (so we are told) windows phone.

    The are things you have to have to work on specific platforms, iOS requires use of an Apple Mac of some fashion for example.

    All that information can be found here ; Titanium Compatibility Matrix

    You can get up and running in minutes once you have downloaded the Titanium Studio and followed the setup guides for your target platforms.

    Have fun.

    • Having just seen your clarification, here is more information;

      • You cannot take an app created for iOS using ObjectiveC and import it into Titanium
      • You cannot take an app create for Android in Java and import it into Titanium
      • You can create a single cross platform app in Titanium and compile for both platforms and Blackberry and Windows phone (when available)

      You have to start by using Titanium.
