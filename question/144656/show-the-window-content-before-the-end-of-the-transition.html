Show the Window content before the end of the transition

Hi, I'm new to Titanium, so, please, be patient :-)

In my ApplicationWindow I create a root Window and I add to this Window 4 labels, each opening a new Window onclick. The four windows are instantiated this way:

var A = require('ui/handheld/A');
var B = require('ui/handheld/B');
var C = require('ui/handheld/C');
var D = require('ui/handheld/D');

var a = new A();
var b = new B();
var c = new C();
var d = new D();

Each label does the following:

label.addEventListener("click", function(e) {
    var t = Ti.UI.iPhone.AnimationStyle.CURL_UP;
    a.open({transition:t});
});

This perfectly works.

Now, I want to close the previously opened window, so, inside A.js (for example) I have:

var w = Ti.UI.createWindow({backgroundColor:'#00ff00'});
var label = Ti.UI.createLabel({ text: "BACK", top:300 });
label.addEventListener("click", function(e) {
    var t = Ti.UI.iPhone.AnimationStyle.CURL_DOWN;
    w.close({transition:t});
});
w.add(label);
return w;

What happens? When I open this window I can see the content of the window before the end of the transition. But, when I "come back" (that is: I close the window with a similar transition), the content of the "home" (the root window) shows up only at the end of the transition. I know the Window is there because if I close it without passing a transition it shows up immediatly.

Can you tell how could I do to make the window content visible before the end of the transition?
