  • Hi

    The first thing I would do is check the variable you have chosen for the properties;

    var pckValidityDate = Ti.UI.createPicker({
    type : Ti.UI.PICKER_TYPE_DATE,
    height : '150dp',
    color : 'Black',
    minDate : value, // looks incorrect
    maxDate : maxDate,
    value : datevalue // where is this defined?
});

    Maybe it is meant to look like this;

    var pckValidityDate = Ti.UI.createPicker({
    type : Ti.UI.PICKER_TYPE_DATE,
    height : '150dp',
    color : 'Black',
    minDate : minDate, // changed
    maxDate : maxDate,
    value : value // changed
});

    Obviously this code MUST be placed underneath the code used to define these variables.
