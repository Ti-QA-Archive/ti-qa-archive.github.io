ACS Push works in 3G but not in Wifi

I'm building an Android app that uses ACS Push Notifications. The notifications arrive without problem when I'm in a 3G network. But if I'm over Wifi (a Wifi connection that works perfectly for all apps, including the one I'm building), the push notifications don't arrive. They only arrive once I switch back the phone to 3G.

A similar issue was reported a few months back: http://developer.appcelerator.com/question/139916/acs-push-android-switch-to-3g-to-wlan—mqtt-stops-working

I'm using ti.cloud 2.3.0 and ti.cloudpush 2.0.7
