Average memory usage for iphone Titanium apps

Hi

I am currently testing my iphone app in instruments to check for memory usage and leaks. I am using the Tweetanium structure of my app which loads everything at runtime.

My app seems to be pretty stable however the 'Overall Bytes' for all Allocations comes to about 4-5mb in total. Is this a high amount (is the iphone capable of this amount?) and should i be worried? What is the typical 'Overall Bytes' of other apps?

Thank You

  • Obviously, it depends on each app, but I would say from my experience that 4-5 MB is ok. The most important is that this amount do not increase the more you use the app. If you feel the app is stable on device and profile tools says you that there aren't memory leaks, it's ok.

  • "Overall bytes" is not that important. It shows how much space your app has used in it's lifetime, both live and freed. You should rather "Live bytes" and the "Living" columns. Make sure your live bytes doesn't constantly increase. And make sure that the "Living" doesn't increase if you open and close a window or a view.
