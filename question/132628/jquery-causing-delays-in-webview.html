jQuery causing delays in webview

I am passing data to a webview using Events like this,

<script>
    Ti.App.addEventListener('pass_data',function(e) {
        $('#name').val(e.client_id);
    });
</script>

When this executes there is a 2 second delay before the 'name' text box is updated. I've worked out that if I remove the jquery include, the delay goes away. So its something to do with the include I think…

Does anybody have any ideas how to avoid this delay?

Thanks!

  • Don't use jQuery. :)

    I know you'll hate that answer, but no matter how much you (blindly :) ) love jQuery - framework is far from ideal in performance terms, even in desktop environment (company in which I work are now removing jQuery code as much as possible because of performance penalties on slower desktop computers).

    Mobile devices are much more slower than desktop computers and because of that performance penalties are more visible.

    Just use DOM methods and JS, without any library:

    Ti.App.addEventListener('pass_data',function(e) {
        document.getElementById('name').value = e.client_id;
});

    P.S. "name" is not the best choice for "id" attribute because it is name of form element attribute and could potentially brake your code.
