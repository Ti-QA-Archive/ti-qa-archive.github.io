How to open a heavy window with Android

My main focus has been with iOS, I am now getting into the Android side. I'm opening a basic window on a button click- nothing special going on here. I do not receive an error message to my knowledge.

I'm using SDK 3.1.1 for Android with a slightly modified default Alloy project

index.xml

<Alloy>
    <Window id="mainWindow" class="container">
        <Label id="label" onClick="doClick">Hello, World</Label>
    </Window>
</Alloy>

index.js

function doClick(e) {  
    var newWindow = Alloy.createController('test');
    $.mainWindow.open(newWindow.getView());
}

$.mainWindow.open();

test.xml

<Alloy>
    <Window class="container">
        <Label id="label">New label</Label>
    </Window>
</Alloy>
