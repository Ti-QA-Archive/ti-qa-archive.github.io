Get Number of files inside folder (applicationDataDirectory) ?

Hi,
Is there any way i can retrieve the number of files that's currently inside a certain folder (applicationDataDirectory) ?

I couldn't find a way to do this. I don't know if it changes anything, but i have only images inside my folder.

Actually, i'm trying to retrieve the number of images inside the folder to be able to save them with a increasing index inside the folder (like image1,image2,image3,image4,…,…), if there's another simpler way to do this i'd be glad to hear it.

PS: I dont have a limited number of folders, they are created dynamically, so saving this index into "Ti.App.Properties" or something is inviable. That's why i thought into retrieving the number of files from the folder, cause i only have this images on each folder.

Thanks

    This works

    var my_dir = Titanium.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory)

var files = my_dir.getDirectoryListing().toString();

var files_array = Array();
files_array = files.split(',');

var num_of_files = files_array.length
alert("---File Number---" + num_of_files);
