  • Hello Uma,

    Apache Ant (Ant) is a general purpose build tool. Ant is an abbreviation for Another Neat Tool.

    Ant is primarily used for building and deploying Java projects but can be used for every possible repetitive tasks.

    In titanium, You can build the android module from the Command Line and Ant Build.

    If ant is already on your PATH, then simply execute it from the top level directory of your module.

    Just Check Below Link :

    • https://wiki.appcelerator.org/display/guides/Android+Module+Development+Guide#AndroidModuleDevelopmentGuide-Buildingfromcommandline/Ant

    • http://en.wikipedia.org/wiki/Apache_Ant

    • http://www.vogella.com/articles/ApacheAnt/article.html#antoverview_definition

    Let me know if it helps.
