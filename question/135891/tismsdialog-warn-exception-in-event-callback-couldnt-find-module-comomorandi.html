TiSMSDialog: [WARN] Exception in event callback. Couldn't find module: com.omorandi

Application type: mobile. Titanium SDK: 1.8.2. Platform & version: iOS 5.x. Device: iOS simulator. Host Operating System: OSX 10.6.8. Titanium Studio: 1.0.9.

Hello,

I am trying to use what looks like an excellent module to send SMS messages from within an App. The URL for the module is here. I am trying to get it installed, but I keep getting the error: [WARN] Exception in event callback. Couldn't find module: com.omorandi. Here's what I've done so far:

1) I downloaded the original file (omorandi-TiSMSDialog-b0c9c70). Inside that folder I found com.omorandi-iphone-0.3.zip.

2) I moved that .zip folder to: Library > Application Support > Titanium

3) I added these lines to the project's tiapp.xml:

<modules>
    <module platform="iphone" version="0.3">com.omorandi</module>
</modules>

and I verified that once the app is compiled, that zip file is removed, and within the folder: Library > Application Support > Titanium > modules > iphone, a folder called com.omorandi now exists.

4) I added these lines within the app:

var module = require('com.omorandi');
Ti.API.info("module is => " + module);

But every time those lines are executed, I get the "could't find module" error. It seems like I've done everything correctly. What else could I be missing?

Thanks!
