SetActiveTab using Tab id

Hello all.

SetActiveTab takes tab index as its input value. Can SetActiveTab be used with the id of Tab?

Regards,

Shreerang

  • A Tab does not have an id property. SetActiveTab takes a Number (index) or a reference to the Tab itself as argument. So, if you have the reference to the tab (rather to the id) you can pass that too.
