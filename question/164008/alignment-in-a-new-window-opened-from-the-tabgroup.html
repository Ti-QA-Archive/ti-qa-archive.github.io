    Hello Andres Valencia,

    Thanks for writing. I have tested your issue. Can you please test my sample code in your environments? It may help to you.

    Check the attachment

    Test Case:

    index.xml

     <Alloy>
        <TabGroup backgroundColor="white" >
            <Tab id="tab1" title="Tab 1" icon="KS_nav_views.png">
                <Window id="win1" title="Tab 1">
                   <Label id="label1" color="#999">I am Window 1</Label>
                   <Label id="label3" color="#999">I am Window 3</Label>
                   <Label id="label3" color="#999">I am Window 5</Label>

                </Window>
            </Tab>
            <Tab id="tab2" title="Tab 2" icon="KS_nav_views.png">
                <Window id="win2" title="Tab 2">
                    <Label id="label2" color="#999">I am Window 2</Label>
                    <Label id="label2" color="#999">I am Window 4</Label>

                </Window>
            </Tab>
            <!-- Use the Require tag to include external Ti.UI.Tab views -->
        </TabGroup>
    </Alloy>

    index.tss

    "#win1": {
    backgroundColor:"pink",
    layout:"vertical"
},
"#win2": {
    backgroundColor:"orange",
    layout:"vertical"
}

    index.js

    $.index.open();
