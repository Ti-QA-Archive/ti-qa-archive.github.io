How to give index to rows

Hello everyone,

I have a question. I create TableViewRows as bellow, and each rows has name which is inserted
at other window.
Each rows can be created everytime users input rowname, in this case how I can give index to each created rows?
For example row1 [1.2.3.4.5…] row2 [1.2.3.4.5…]

Thanks.

var data = [];

var tableView;
var rowData = [];


var db1 = Ti.Database.open('db1');
var db2 = Ti.Database.open('db2');
var ret = db1.execute('SELECT* FROM database1');
var rowName1 = ret.fieldByName('rowName1');
var ret = db2.execute('SELECT* FROM database2');
var rowName2 = ret.fieldByName('rowName2');

var row1 = Ti.UI.createTableViewRow({
    height: '60dp',
});
var label1 = Ti.UI.createLabel({
            text: rowName1
        })
row1.add(label1);
rowData.push(row1);

var row2 = Ti.UI.createTableViewRow({
            height: '75dp'
        });
var label2 = Ti.UI.createLabel({
            text: rowName2
        })
row2.add(label2);
rowData.push(row1);

var tableView = Ti.UI.createTableView({
  data:rowData
});

    You have a typo on the 2nd row rowData.push(row1); should be row2.
    Why you need Index? they will automatically have Inside table.data[0].rows. However since this is Javascript you can add any property you need.

    app.currentIndex =0;
...var row = ...
row._Index = app.currentIndex;
app.currentIndex++;
