Desktop application / Run as android app

Hi there, I know this might sound counter intuitive but I'm currently looking at packaged software that leverages existing tech (html/css/js) and can deploy to an android platform.

I've tried phonegap and I created a simple 'app' using html5 / css3 / JQuery mobile and phonegap packages it up and can outputs an apk which ive installed on my phone and used, its by no means good but just a proof of concept.

Doing the same with Titanium seems to be impossible, you can create a desktop app OR a mobile app, each are completely different in terms of workflow.

So the question I want to ask really is, can I create a mobile site with html/css/js and package it up as an app using titanium mobile?
