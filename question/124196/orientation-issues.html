Orientation Issues

Hi,

I have my application and it opens up a new window that I want in Landscape mode. However, with the latest version of Titanium it seems that there are issues of which I'm not exactly sure how to describe them. First of all, here is my code that I put in the window that I am loading that I want in landscape mode:


win.addEventListener('focus',function(){
    Titanium.UI.orientation = Titanium.UI.LANDSCAPE_RIGHT;
    win.orientationModes = [Titanium.UI.LANDSCAPE_RIGHT];
});

It looks like it is placing a new version of the window below the old version of the window, but the old window is purely visual and cannot be interacted with. However, it seems to be 'in the way' of the active layer beneath it so the new active layer does not respond to my touches. I can tell this because the old layer seems to be shrunk so that the edges do not reach the edge of the screen and I can see just a bit of the active layer and those areas do respond to my touch.

Also, if I go back to the previous window, my tableview seems to have a black margin beneath and above it. Is this a known issue, or am I doing something wrong?

Thanks so much!

    Without knowing much else though, you should set the orientationModes on a window when you create it, then set the ui orientation immediately after. That will usually take care of the window for the duration of its existence.
