label formating

I have a label on view. I am changing fontsize of a label with a dialog(small,normal,large). my problem is that when i click on small and normal then label appears at specified position but when i click large and after it I click small and normal then label do not apears at specified position.

  • when you want your label will auto resize the text it contains should do so

    var label = Titanium.UI.createLabel({
        text : " ",
        width : 100,
        font : {
            fontSize : 20
        },
        left : 100,
        height:30,
        minimumFontSize:8

    });

    but if you want to change the font manually you should make sure that when you set, for example, the font "large", the width and height properties of the label increases
