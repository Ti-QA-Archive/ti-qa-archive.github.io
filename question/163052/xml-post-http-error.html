    Hi narasimha reddy,
    For your information when you use the post method for http client you have to provide the arguments to the send method of http client. In your code you are using POST method but did not gave the params to the send method like this

    xhr.send(params);

    may be you wanted to send the datastring in the params try this

    xhr.send(dataString);//or like this
xhr.send({'property':dataString});
//The name of the property depends on your web service, i.e what type of object with what properties it accepts

    Please try after these changes and if your problem is resolved then mark the answer to help others, or feel free to ask
