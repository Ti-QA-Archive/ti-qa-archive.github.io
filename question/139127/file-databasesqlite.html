  • If I understand your question correctly, the sqlite database file is located in the following directory:

    • Android: &#x2F;data&#x2F;data&#x2F;[{APP ID?}&#x2F;databases
    • iPhone:
      ~/Library/Application Support/iPhone Simulator/User/Applications/{APP ID?}/Library/Application Support/database
