Blackberry SDK and Titanium 3.1

Hi @all,

i try to get Blackberry development to run on OS X. I have downloaded the App "Momentics" for OS X and with it i also downloaded the SDK. The SDK will installed directly in the App-Folder.

The SDK is named with host_xxxx and target_xxxx. If i setup this in Titanium i'll get an error a directory named with prefix "install" is missing.

But there is no folder named "install_xxx".

Anyone able to help what i am doing wrong?

Thanks in advance.

    I had this problem too, and did my best to research a solution. The best I can tell, and what I did, was create an 'install' folder in the same location as the blackberry SDK (in the Momentics package). I do not know the significance of this, but it did allow me to get past the Titanium error.
