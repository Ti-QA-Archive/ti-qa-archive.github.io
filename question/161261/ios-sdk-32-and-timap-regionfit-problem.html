i'm using Ti.Map module in ios with last sdk version (3.2) in ios

    The regionFit property is no longer supported (see the docs). If you set the region properly, it should fit the map to the view at the zoom level you specify. It could be that your delta values are too small and are being ignored.

    You could check out the MapsDemo in the training labs. It implements the new camera (iOS7 only) which overrides the region and can also set a zoom level. It works on iOS and Android.
