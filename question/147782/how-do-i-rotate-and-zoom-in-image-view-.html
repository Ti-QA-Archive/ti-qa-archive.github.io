    Hi

    The first problem you have is the property autorotate, it is not related to the ability to rotate an image interactively. It tells the OS that is should or should not rotate the image based on how the image was originally taken with a camera.

    The second problem relates to the property canScale this is an Android only feature. As you have not specified which platform you are designing your app for this will not work for iOS. If you want it to work for Android you also need to specify the property enableZoomControls as CanScale is interdependent on it.

    If you want to roate an image around it centre the answer given by Sarafaraz Babi for the question Rotate an image around its center does provide a cross-platform solution.
