Suggestions for Cloud Database code

I am interested in writing a database app using a MYSQL database in the cloud to save my data. I am fairly proficient with JSON and PUT requests so I'm looking for the easiest way to have a front end API to write to a MySQL database I setup to run in "the cloud". I'd like to be able to take a record from the local iPad database (SQLite) and send it to the MySQL in the cloud, retrieve the same way, etc. My preference is JSON rather than XML. I am hoping a pre-cooked API already exists as I am not a PHP programmer (or Python for that matter). If someone knows of a service that already exists to do this, I am not married to the idea of hosting myself.

Thanks!

  • Stephen

    This is a pretty broad question, so you are likely to receive very subjective answers.

    Obviously you know Javascript, so you could consider using Sequelize, which is a MySQL ORM for NodeJS. Check out the video on the NodeJS website, if you haven't heard of it before.

    The problem with NodeJS is that it's not very mature, which is why I would not consider it yet for my own (non-work-related) projects.

    If you need a shortcut to PHP, try a framework. My personal favorite (so, certainly not an official endorsement) is symfony, which comes with the doctrine ORM.

    Hope this helps
