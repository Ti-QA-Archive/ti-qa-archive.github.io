  • Accepted Answer

    Thinking back to my comment above, I realized you may be interested in creating the db programatically, versus providing one in the resources directory. If this is the case, this link should be very helpful to you:
    http://www.codeboxed.com/2011/08/create-database-with-titanium-mobile/

    Try:

    var metakeDb = Ti.Database.open('metakeDb');
var sql = 'CREATE TABLE IF NOT EXISTS my_table(' +
    'id INTEGER PRIMARY KEY AUTOINCREMENT, my_column TEXT, my_column2  INT' +
')';

metakeDb.execute(sql);

    Modify the "create table" statement as necessary for your table.
