    Can you please try again? We deployed a new change that disallow sending push to all users in a channel (when to_ids is empty) unless the sender is an admin. This is due to security concern that it is too easy to spam a channel if developer makes a mistake. But we have reverted this change for now and will make it a configurable setting for developers.
