  • Hi Levani,

    You can hide it using the property Window.shadowImage and by adding an image 1 x 1 px with the desired background color. In this case, you don't need the @2x image because it will create a border with 2px.

    An example using tss: 

    'Window':  {
  backgroundColor: '#fff',
  barImage : '/images/top.png',
  shadowImage: '/images/navBarLine.png' /* a 1px x 1px image with a white background color */,
}

    I believe you can also create an transparent image and set a barColor for the navBar. That way, it will always match the color you set in the navBar (this is just an idea, I didn't try this).

    Hope it helps!
