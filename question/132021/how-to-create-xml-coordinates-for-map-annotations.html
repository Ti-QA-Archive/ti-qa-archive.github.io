    • Here is a example using XML, but i don't really get how your XML-file is designed. This is also for use with XHR, so replace the code in my earlier example within xhr.onload, with the following.

      var annotations = [];
var doc = Titanium.XML.parseString(xhr.responseText);
var items = doc.getElementsByTagName("annotation");
for (i = 0; i < items.length; i++)
{
    var lon = items.item(i).getElementsByTagName('longitude').item(0).text;
    var lat = items.item(i).getElementsByTagName('latitude').item(0).text;
    annotations.push(Titanium.Map.createAnnotation({
        latitude:lat,
        longitude:lon,
        pincolor:Titanium.Map.ANNOTATION_RED,
        animate:true,
    }));
}
mapview.annotations=annotations;

      This assumes that your XML looks like this

      <annotations>
    <annotation>
        <longitude>14.36459</longitude>
        <latitude>92.47432</latitude>
    </annotation>
    <annotation>
        <longitude>41.63459</longitude>
        <latitude>29.74432</latitude>
    </annotation>
</annotations>

      I have not tested this code, but it should work :)
