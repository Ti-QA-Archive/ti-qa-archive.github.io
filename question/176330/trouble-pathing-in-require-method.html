    • For future generations;

      var test = require('ui/js/utility/ActivityIndicatorWithDialog');
      • Never include an initial slash (as Harrison notes)
      • Never add .js (as Stephen noyes)

      To expand on the resources folder comment by Tobias;

      • ActivityIndicatorWithDialog.js (based on above) would be stored in either of these two locations depending on the use of Alloy
      // Alloy
{appname}/app/lib/ui/js/utility/ActivityIndicatorWithDialog.js
// Resources
{appname}/resources/ui/js/utility/ActivityIndicatorWithDialog.js

      All these folders including Alloy "lib** which is not created by default would be created by you as your choice when locating files.

      When you are using the Alloy Framework with Titanium all CommonJS should be placed inside the {appname}/app/lib/ and this is considered the root. When not using the Alloy Framework you can store CommonJS modules anywhere you wish - but try and keep things tidy - for your own sake.
