Block Xtra Large Screens from download app from play store

Titanium SDK: 2.1.0
Platform: Android

Hello There
I have an android phone app and I don't want tablets downloading it form play store. I'm using the code:

          <supports-screens 
          android:anyDensity="false"
            android:largeScreens="true"
            android:normalScreens="true"
            android:smallScreens="false"
            />

and I sucessfully blocked the small screens, but can't block extra large screens using:

android:lxargeScreens="true"

But on building the apk file I receve the following error: No resource identifier found for attribute 'xlargeScreens' in package 'android'

Already seached for posts here but no working solution for now. It seam that I have to force sdk to a higher than 9 version, but no go untill now .. any help please!!!

  • Hi Rodrigo,

    I think that you have to set the API Level up to 9 (or even more).
    Like this here:

    <tool-api-level>10</tool-api-level>
<manifest>
<supports-screens android:anyDensity="true" android:xlargeScreens="true"/>
<uses-sdk android:minSdkVersion="9" />
</manifest>

    Have a look at this ticket here: https://jira.appcelerator.org/browse/TIMOB-10228
