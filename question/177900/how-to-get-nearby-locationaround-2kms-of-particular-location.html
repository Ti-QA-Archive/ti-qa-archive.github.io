    • Sivasankar,

      Just a point of clarification, the MAP object included with Appcelerator is for creating your own map with your own data, so if you had a list of shops and the lat/lon of each, then you could do what you want using the built in map.

      What you seem to be describing is the ability to do a search on public information, then display it on a map. In this case, you want to send a query to the map and have it show you the results. Here is a query for 'Restaurants' near where I live:

      • http://maps.apple.com/?q=restaurant&near=calgary

      So to put this into Titanium, you would pass this in an openURL call like this:

      • Ti.Platform.openURL('http://maps.apple.com/?q=restaurant&near=calgary');

      You can find the apple maps url scheme here

      The Google IOS url scheme documentation is here

      If you are using Android, best way to launch would be using an intent.

      Really, any query you can run at maps.google.com you can create and pass to the URL..

      Ray
