Label with Multiple Links

Hey Guys,

I just had a question that I figured I would ask to make sure I wasn't reinventing the wheel before I built it out myself. This question is relating to the iPhone build…for now.

What I want to do is have a label (or other?) tag that contains a line of text, for example: "I want to go to the store and get cheese". Now, what i want in html would look like:

<p>I want to go to <a href='/store_url'>the store</a> and get <a href='/cheese_url'>cheese</a>.

I do know how to make a label and have it looking great, minus the links. I did see that the facebook iPhone app does this, so just figured I'd throw it out there.

So the question is, is there a way to add links that have their own event listeners and styling INSIDE of a label tag? Or a way that someone already created, to tell add labels back to back in a way that makes them LOOK like they are one single label with links?

Thanks!

    Perhaps a view with 'horizontal' layout?

    var win = Titanium.UI.createWindow({  
    backgroundColor:'#fff'
});

var view = Ti.UI.createView({
    height:300,
    width:320,
    layout:'horizontal'
});
win.add(view);

var name1 = Ti.UI.createLabel({
  left:5,
  width:'auto',
  height:20,
  color: '#ff0000',
  text: 'Mike Fogg',
  font: {
        fontWeight: 'bold',
        fontSize: 12
    }
});
view.add(name1);

var middle = Ti.UI.createLabel({
  left:2,
  width:'auto',
  height:20,
  color: '#00ff00',
  text: ' posted a link to ',
  font: {
        fontWeight: 'bold',
        fontSize: 12
    }
});
view.add(middle); 

var name2 = Ti.UI.createLabel({
  left:2,
  width:'auto',
  height:20,
  color: '#ff0000',
  text: 'Jason Walbeck',
  font: {
        fontWeight: 'bold',
        fontSize: 12
    }
});
view.add(name2);


name1.addEventListener('click',function(e){
    alert('Name 1 eventlistener');
});


name2.addEventListener('click',function(e){
    alert('Name 2 eventlistener');
});

win.open();
