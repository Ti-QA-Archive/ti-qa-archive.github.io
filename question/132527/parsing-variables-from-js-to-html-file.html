  • Is it possible that you're firing the event in the app before the web view has loaded and added the event listener?
    You should add some logging like so:

    <script type="text/javascript">
Ti.API.info("Adding event listener for customEvent.");
Ti.App.addEventListener('customEvent', function(eventData) {
   Ti.API.info("Received customEvent.");
   myFunction('eventData.someStuff[0]'); 
}); 
</script>

    Also, you should log the place where your fire the customEvent, to check that the event listener is added before the event is fired.

    Sometimes Javascript will fail silently and stop parsing the code if there's a syntax error or some other fatal error.
