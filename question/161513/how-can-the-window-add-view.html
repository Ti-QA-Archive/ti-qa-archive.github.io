  • Accepted Answer

    Change the <view> tags to <View> as in the following:

    <Window id="win2" title="Tab 2" exitOnClose="true" layout="vertical">
    <View layout="vertical">
        <Button title="bt1" id="bt1" onClick="bt1onclick" width="700" height="50"></Button>
        <Button title="bt2" id="bt2" onClick="bt2onclick" width="700" height="50"></Button>
        <Button title="bt3" id="bt3" onClick="bt3onclick" width="700" height="50"></Button>
        <Button title="bt4" id="bt4" onClick="bt4onclick" width="700" height="50"></Button>
    </View>
</Window>

    Alloy is case sensitive.

  • Firstly you should use the correct markdown when asking your question so it makes it easier to understand, in your case your code should be wrapped with 3 tilde's (~).

    To add a view to a window in XML, it should look like this:

    <Alloy>
    <Window id="win" backgroundColor="white" fullscreen="true">
        <View id="view" backgroundColor="red" width="50" height="50" />
    </Window>
</Alloy>

    To use a tabgroup, use the example given in the documentation.
