      If you put a properly packaged module in the root of your project, it should be extracted when you first build the project. The modules directory does not magically appear.

      — commented August 23rd 2013 by Stephen Feather

    • If you want to use a module in more than one project, you can place it in :

      ~/Library/Application Support/Titanium/modules
