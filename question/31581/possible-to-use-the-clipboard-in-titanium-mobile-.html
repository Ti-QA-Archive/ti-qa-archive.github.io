Possible to use the clipboard in Titanium Mobile ?

Hi,

Is there a way to copy something to the clipboard using Titanium Mobile ? I have found a clipboard API for Titanium Desktop, but it doesn't seem to be supported in the mobile version yet.

Are there plans to implement it in the near future ?

    We don't yet have programmatic access to the clipboard, but a quick google search shows that it's pretty easy to do on the native side (at least on iPhone). I'm not aware of any immediate plans to add it in the SDK, but you could take a crack at implementing this functionality in a custom module. This will obviously require some Objective-C programming, but should be doable.
