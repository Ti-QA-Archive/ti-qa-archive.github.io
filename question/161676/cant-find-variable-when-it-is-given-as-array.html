    Hi,

    Your while loop creates an array of textField's, however your eventListener assumes there are no array elements and only a single textField. In your eventListener you need to loop through your array to correctly reference your textField's

    Ti.App.addEventListener('orient', function(evt) {
    var intField = 0,
        intFields = textField.length,
        newTop = temptop;

    if(evt.portrait === true) {
        newTop = newTop + 370;
    } else {
        newTop = newTop + 100;
    }

    for (intField = 0; intField < intFields; intField = intField + 1) {
        textField[intField].top = newTop;
    }
});

    Hope this helps!
