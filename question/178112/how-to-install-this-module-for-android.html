How to install this module for android

Does anyone know how to install this module?

https://github.com/harukazepc/TiWebViewUserAgent/tree/master/android

I have the iOS version of the module installed but can't manage to install it for android. There are instructions included, but I can't follow them/find them unclear.

Now I know that if you want to set a custom userAgent string, you don't have to use this module. That you can use the built in titanium method.

I have implemented this method, but it seems to be causing some issues when I load my web views. I would like to be able to install the module to test it against my app with the userAgent method.

Has anyone installed this module that could give some clear concise instructions to install it myself/

  • There is no packaged module included in this repo. So you would have to build it yourself by running the build script with ant.

    But as you already said, the functionality is included in Ti.UI.WebView which makes this module useless. It is also very old and may not work with the current SDK.
