  • Hi

    Here is a suggestion which may help resolve your issue.

    Firstly make a backup of your project and/or code so that any of the following can be recovered in the event of messing up

    • Clean your project and close the project you are working in.

    If you are using Alloy framework go to the root of your project folder and delete any files inside the following folders:

    - build
- Resources

    PLEASE NOTE: This only applies to Alloy Framework

    If you are using classic Titanium then go to the root of your project folder and delete any files inside the following folder:

    - build

    PLEASE NOTE: DO NOT delete your resources if you are using classic Titanium

    Now open your project and compile your app again and see if the issue has been resolved.
