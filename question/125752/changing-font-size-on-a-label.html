Changing font size on a label

I'm trying to change the font size on a label based on the length size. I thought i could do lblname.fontSize = 12; However, that doesn't seem to work. Any thoughts?

  • Try lblname.font = &#39;{fontSize:12}&#39;; instead. Also note that labels support the minimumFontSize property. If you set this, then the font size will be dynamically reduced to fit in your label, unless it needs a size smaller than you set as the minimum. Then the ellipses will still appear.
