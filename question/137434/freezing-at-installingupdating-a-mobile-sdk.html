Freezing at Installing/Updating a Mobile SDK

Hi,

I'm just installing Titanium Studio on my Windows XP machine. The installation worked and after I start it I want to configure Android, then I click the "Install or Update Android SDK" button and it freezes at the dialog box titled "Freezing at Installing/Updating a Mobile SDK".

Could it be the Java version, because the Documentation said it should be at most 1.6.2 and my version is:

    It is highly recommended that you upgrade to Windows 7, due to the problems that XP has with contiguous memory, explained in the info box at Titanium SDK Minimum Memory. If you can't upgrade your OS now, ensure that your system has enough contiguous memory available to run Titanium, by disabling any unnecessary memory-resident software and services (particularly anything like MS SQL), and rebooting before attempting to continue.

    The 32bit version Oracle JDK 1.6 is required (1.7 will not work correctly). See Java Development Kit. Uninstall JDK and replace it with the correct version.

    Ensure that Titanium Studio, the Titanium SDK and Android SDK are all installed to a location within your home directory. This ensures that Studio can update itself and these components without encountering permissions conflicts.

    For a reliable way to configure Titanium, follow the Setting up Titanium Studio guide, followed by Creating Your First Titanium Apps to create your first apps.

    Good luck! :)
