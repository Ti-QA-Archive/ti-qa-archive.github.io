Nested views

Is it possible to add a webView to a view? I would like to display a web page having its top border at a certain distance from the top, but when scrolling down the web view should move all the way to top = 0. I tried to create a view v and then to put my web view w in this, and finally add this to the window, but with no luck.

Does anyone have a suggestion?

w = Ti.UI.currentWindow;

wb = Ti.UI.createWebView({
    url:e.row.link,
    top:100,
    width:320,
    left:0,
});

v = Ti.UI.createView({
    top:0,
    width:'auto',
    height:'auto',
});

v.add(wb);
w.add(v);

  • You didn't set height for wb.. so its container (v) with height: &#39;auto&#39; will have height = 100 (= wb.top) when it's rendered. So comment out height: &#39;auto&#39; would help:

    v = Ti.UI.createView({
    top:0,
    width:'auto',
    // height:'auto',
});
