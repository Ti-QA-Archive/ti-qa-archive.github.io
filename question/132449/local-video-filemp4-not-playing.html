Local video file(.mp4) not playing..

I tried to play a local video file but all i can see a black screen with a activity indicator in it. I dont know what is wrong with my code.
using sdk 1.8.1,

I am showing video when user taps on a tableViewCell.

here is my code.

var win = Titanium.UI.currentWindow;

var activeMovie = Titanium.Media.createVideoPlayer({

    url: "../movie.mp4",
    backgroundColor: "#111",
    movieControlMode:Titanium.Media.VIDEO_CONTROL_DEFAULT

});
win.add(activeMovie);
activeMovie.play();

win.addEventListener('close', function() {
    activeMovie.stop();
});

  • Where do you put movie.mp4? And Where's above code located?

    Let says your movie.mp4 located under yourproject&#x2F;Resources&#x2F;movies&#x2F;movie.mp4, and above code located under yourproject&#x2F;Resources&#x2F;modules&#x2F;player.js, then code will look like below:

    modules/player.js

    var activeMovie = Titanium.Media.createVideoPlayer({
    url: "../movies/movie.mp4",
    backgroundColor: "#111",
    movieControlMode:Titanium.Media.VIDEO_CONTROL_DEFAULT
});
