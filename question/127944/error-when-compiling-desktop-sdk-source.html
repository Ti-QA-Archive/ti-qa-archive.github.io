Do I need to specify the path to the xcode includes or something? I have everything that's needed according to the list of build requirements (although I have xcode4 and not xcode3).

  • I got it to work using xcode4.

    I uninstalled my current xcode4 (not sure if it was needed).
    I then installed xcode3 in /XCode3, opted out of "System tools" and "Unix Development" (I also excluded the 10.4 sdk and iOS since I don't have any interest in those).
    Next I did a full install of xcode4 (in its default folder).
    Did a symlink of the 10.5 sdk from xcode3 into xcode4:
    "cd /Developer/SDKs"
    "sudo ln -s /Xcode3/SDKs/MacOSX10.5.sdk ."

    et viola, I could build using xcode4 \o/

    (source)
