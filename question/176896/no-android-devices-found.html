No android devices found

So I just updated Titanium Studio to 3.3 and now no Android devices are found anymore. It worked before, now it doesn't. Eclipse can find android devices, as can the tools in the Android sdk…but Titanium studio cant'. I tried refreshing devices, I tried different USB cables and rebooting my computers and Titanium Studio but it did not work. I also tried killing and restarting adb.

Does anyone have this problem as well? Can anyone offer any tips to fix this problem? Thanks in advance.

I am developing on Windows 8.1 using Titanium Studio 3.3 (just updated). I attached a Android 2.3 device which previously did work for testing.

    2. must be in developer mode and USB debugging must be allowed
    3. cable plugged
    4. driver (for windows OS)

  • You must restart the adb server with:

    PATH/platform-tools/adb kill-server
PATH/platform-tools/adb start-server

    On the device the debugging mode must enabled and the device must be unlocked.
