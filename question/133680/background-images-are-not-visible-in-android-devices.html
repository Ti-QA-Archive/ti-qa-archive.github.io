Background Images are not visible in android devices

Hi all,

I am making applcation, in that I am adding button to image view, the some buttons are enabled and some disabled.

Now I am trying to add background image to the button with line
addFriendButton.backgroundFocusedImage = '../images/addfriend.png';
and
PendingButton.backgroundDisabledImage = '../images/addfriend.png';

but it is not showing the images for the buttons, Please guide how to resolve this issue

    I remember having issues with images and recall there being a slight difference between Android and iOs. To display images for iOs I would use Gagan Tiwari answer above, but for Android I would lose the first slash; so addFriendButton.backgroundFocusedImage = &#39;images&#x2F;addfriend.png&#39; instead of addFriendButton.backgroundFocusedImage = &#39;&#x2F;images&#x2F;addfriend.png&#39;

    If you still don't see the images and you are sure you have set the image path correctly, try forcing a complete rebuild of your project by deleting the [AppName]/build/android folder.
