  • Hi Seth,
    It happens that titanium looks for some particular files in the android sdk, if it doest find it , it shows this error. So my suggestion is that delete your sdk and install new on from Titanium/preferences/studio/platforms/android/ for Mac else Window/preferences/studio/platforms/android/ for windows.
