Events in Ti.UI.WebView

Hi everyone,

is there a way to directly invoke html event listeners for click, dblclick etc. events when using a WebView? Below example illustrates that neither of the dblclick events is triggered when using the html in a webview (or maybe I'm just missing something).
I've read the posts on the topic 'events/webview', so I know that I could do this by using Ti.App.fireEvent/Ti.App.addEventListener. However this seems impossible to me because what I am finally aiming at is to use an existing Raphaël script that handles creating/resizing/dragging shapes and makes use of events like click, dblclick, touchmove etc. and I want this script to handle all of the events directly.
The notes on Ti.UI.WebView state that you cannot have both Titanium style events against the webview instance and internal Javascript events in the DOM. You must choose between one or the other. This sounds like I could say something like 'bypass Titanium style events, use JavaScript events only'. But how would I do that?

Thanks,
Thomas

app.js:


var win1 = Titanium.UI.createWindow({
    backgroundColor: 'green'
});

var webview1 = Titanium.UI.createWebView({
    width: 768,
    height: 1024,
    top: 0,
    url: 'test.html'
});

win1.add(webview1);
win1.open();

test.html:

<html>
    <head>
        <meta http-equiv="Content-Type" content="text/html; charset=iso-8859-1">
        <title>WebView/Raphael</title>
    </head>
    <body ondblclick="alert('dblclick on body');">
        <p ondblclick="alert('dblclick on p');">
            Hello
        </p>
    </body>
</html>
