  • Hi

    I am not exactly sue which part you are trying to hide as sometimes there are different words for the same thing in this context. However in case it is this own; change following setting in the tiapp.xml file;

    ~~~
    // change this entry
    <navbar-hidden>false</navbar-hidden>
    // to this instead
    <navbar-hidden>true</navbar-hidden>
    ~~

    If you are using studio to edit this file - make source you are not using the visual editor for Tiapp.xml, you can swap using the tabs at the bottom of that window.

    — answered May 2nd 2014 by Malcolm Hollingsworth
