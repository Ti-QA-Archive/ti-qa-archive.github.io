  • I too had the same problem at one point. But your tiapp.xml file is missing the manifest area. This is where you will have to change teh version code and version name. Make sure that you then change the version name on the overview as well. Appcelerator forgot to put this manifest code in there. hope this helps see below…..

    </orientations>
    </iphone>
    <android>
    <manifest android:versionCode="3" android:versionName="1.3" xmlns:android="http://schemas.android.com/apk/res/android"/>
    </android>
    <mobileweb>
    <precache/>
    <splash>
