Snap View Mode for iOS

Titanium SDK 3.2.0
iOS 7

Hello all

I'm trying to make a snapview mode like snapchat video conference.

For those who don't know what it looks like, I explain.
In fact, you can have a snap only when you let your thumb (or any finger) on the screen.

I want to do just the same.

For example, I have a view1, when I make a longpress on the view1, I want to display another view (view2) above. Then when I remove my thumb (or finger) I want to remove the view2.

Is it possible to do so ?

    You mean something like this ?

    var isTouching = false;

var w = Ti.UI.createWindow();

var v1 = Ti.UI.createView({
    top : 0,
    height : '50%',
    backgroundColor : 'red',

    visible : false
});

var v2 = Ti.UI.createView({
    top : '50%',
    height : '50%',
    backgroundColor : 'green'
});

v2.addEventListener('touchstart', function() {

    if (!isTouching) {
        isTouching = true;
        v1.visible = true;
    }
});

v2.addEventListener('touchend', function() {
    if (isTouching) {
        isTouching = false;
        v1.visible = false;
    }
});
w.add(v1);
w.add(v2);
w.open();
