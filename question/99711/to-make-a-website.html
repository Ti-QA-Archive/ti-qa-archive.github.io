  • Titanium is a platform for building native mobile applications in JavaScript. A mobile website is developed the same way a normal website is developed, only with custom mobile styling and fewer visual assets and JavaScript files to facilitate faster loading over mobile networks. A quick Google search turns up many resources out on the web for developing mobile websites - good luck.

  • Technically you could create a single WebView app and point it out to your external web site.

    If you are interested in developing a mobile app using HTML, then take a look at JQuery Mobile and PhoneGap. I started using this SDK, but my client did not like the non-native look of the app. So we switched to Titanium.

    In terms of speed, going Titanium is definitely faster.

    Hope that helps!
