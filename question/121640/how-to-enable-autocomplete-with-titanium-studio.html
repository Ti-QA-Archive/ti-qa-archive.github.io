  • Ahmad

    Try going to Titanium Studio preferences, typing the words "assist" and "complet" (short for "code assist" and "code completion") into the filter at the top, and going through each config screen in the results and clicking on "Restore Defaults" and "Apply". Then restart Titanium Studio.

    If this does not resolve it, create a new, blank, project, and copy into its resources folder the code from the resource folder of an existing project that does not work. Again, you may need to restart TiStudio.

    I'd be interested in hearing from you whether either of these methods resolves it.

    Thanks
