Alertdialog.

Is there a way to use EMOJI in alertdialog?
Also how do you change to new row in alertdialog?

    Hi

    You need to use the Unicode references for the symbols;

    alert('\u2661\u2665\u2764\u2765\u2766');

    The format is \uXXXX, where XXXX equals the Unicode number (may be longer than 4 characters).

    Examples;

    • \u2661 = Hollow heart flat
    • \u2665 = Red heart style 1
    • \u2764 = Red heart style 2
    • \u2765 = Solid heart flat on side
    • \u2766 = Solid heart flat with serifs

    You use \n to break up lines in alerts, labels and other text elements.

    alert('Line 1\nLine 2\n\nLine 4');

    Mixed together

    alert('\u2661\u2665\u2764\u2765\u2766\nHello\n\nMe again');

    • As a follow up;

      You can use the same syntax in labels (and other controls as well);

      var lbl = Ti.UI.createLabel({
    height: Ti.UI.SIZE,
    text: '\u2661\u2665\u2764\u2765\u2766\nHello\n\nMe again',
    width: Ti.UI.SIZE
});
