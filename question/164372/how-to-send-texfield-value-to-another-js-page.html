How to send texfield value to another js page?

Hi!
I have this textfield in app.js:

var phonenumber = Titanium.UI.createTextField({
    top:'50%',
    borderRadius:3,
    hintText:'Your mobile nr',
    keyboardType:Titanium.UI.KEYBOARD_DEFAULT,
    width:'60%',
    height:'auto',
    left:'20%',
    right:'20%',
    touchEnabled: true,
    passwordMask: true  
});
win1.add(phonenumber);

and I wanna be able to get the value of textfield phonenumber in another js page called login.js

This is the button:

var loginTele = Titanium.UI.createButton({  
    title:'Skicka Nr',  
    top:'70%',  
    width:'60%',  
    height:'10%',  
    borderRadius:1,  
    font:{fontFamily:'Arial',fontWeight:'bold',fontSize:14}  
});  
win1.add(loginTele);

This is my addEventListener:

loginTele.addEventListener('click', function (e){
   // alert('Hello \n You have logged in with mobile.nr '+ phonenumber.value +'.');
   var win = Titanium.UI.createWindow({
        url:'login.js',
        title:'Login'
    });
    win.open({modal:true});
});

Know I wanna be able to get that value from the textfield phonenumber.value and use it in my another js page called login.js. How can I do that?

  • Hello Vladimir,

    You can pass phonenumber value like :

     var win = Titanium.UI.createWindow({
        url:'login.js',
        title:'Login',
        phoneNumber : phonenumber.value
    });
    win.open({modal:true});

    ###login.js

    var win = Titanium.UI.currentWindow;

alert(win.phoneNumber);

    i hope this helps. thanks.

  • You can create custom property and pass value using it:

    
loginTele.addEventListener('click', function (e){
   var win = Titanium.UI.createWindow({
        url:'login.js',
        title:'Login',
    phoneNumber: phonenumber.value, // Custom property
    });
    win.open({modal:true});
});

    Retrive value in login.js using following code:

    var loginWin = Titanium.UI.currentWindow;

alert(loginWin.phoneNumber); // fetching cust property.
