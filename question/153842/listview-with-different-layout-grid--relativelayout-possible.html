ListView with different layout (grid / RelativeLayout) possible?

Hi There,

I'm using the latest version of Titanium & Alloy (Git master). In the documentation it refers to an item in a ListView creating a row however is there a way to have more than one item per row? For example in a grid layout. Having previously come from a native Android environment I know this can be achieved with a ListView using a RelativeLayout so is there any way to achieve this with Titanium?

Thanks,

    I don't believe there is. It is meant to be a fast-as-possible standard set of rows. You could, however, submit a feature request to the Titanium Community Jira Project. That way it will be seen by the platforms engineers and you can get direct feedback that way.
