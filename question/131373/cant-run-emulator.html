  • Not sure if this will help anyone (it's a little old now) but I happened to have this issue for a bit too. One thing that seemed to fix it for me was to:

    1. Delete the build (or just build/iphone) folder
    2. Open tiapp.xml and "touch" it (I just go somewhere, press spacebar, then backspace to erase the blank space then resave it).
    3. Rebuild it
