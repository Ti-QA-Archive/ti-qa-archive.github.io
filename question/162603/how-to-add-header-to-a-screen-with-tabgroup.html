How to add header to a screen with tabgroup

How can I add header to the screen with tabgroup ? I tried adding the header to the tab windows, but the header will be coming only under the tabs. I need to get the header on top of the tabs. How this can be acheived ? Please help.

  • Accepted Answer

    TabGroup is a top-level container component, meaning it can't be inside anything. Content must be in windows inside tabs inside the tab group. Thus, using the stock components, there's no way to add a header/banner above the tab group itself. You could create a custom component that looks and operates like a tab group (i.e. write all the code yourself to reproduce that functionality). If there's a stock Android component for this, I'm not aware of it. If there is, it could be exposed for use in Titanium through a custom module.
