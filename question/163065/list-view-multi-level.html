    Hi

    This is not a multiple level listView - it is simple a window with a listView on it and when a row is clicked a new window is created and new listView is added to that window.

    Often the data from the first listView row is used to determine the information that is displayed on the new child window and its listView.

    So each window has its own listView and is opened up whenever the user selects a row.
