Translate system buttons titles (edit, delete...)

Hi !
I'm making my app in french. But some buttons are in english :

  • Searchbar "Cancel" button
  • TableView editing "Delete" button

By editing the "info.plist" file, I changed "Localization native development region" to "France".
It changed the searchBar "Cancel" button to "Annuler". Great !
But the "Delete" button from tableview editing is still in English.

So, can I change that ? And if possible how can I ?

  • I found the answer to my question today !

    You can change the title of a tableview delete button using the deleteButtonTitle property.

    var tableview = Ti.UI.createTableView({
height:300,
deleteButtonTitle:'Supprimer'
});

    I hope this will help !
