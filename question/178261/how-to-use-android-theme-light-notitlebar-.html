How to use Android theme Light NoTitleBar ?

Hi, i upated my titanium sdk to 3.4
i can apply

Theme.AppCompat.Light.NoTitleBar

but this one works

Theme.AppCompat.Translucent.NoTitleBar

    Define a custom Theme which extends Theme.AppCompat.Light

    <resources>
  <style name="Theme.AppCompat.Light.NoActionBar" parent="@style/Theme.AppCompat.Light">
        <item name="android:windowNoTitle">true</item>
        <item name="windowActionBar">false</item> <!-- For 2.x version  -->
  </style>

</resources>

    • The snippet above is already extending it. Put it into your platform&#x2F;android&#x2F;res&#x2F;values&#x2F;themes.xml as described in the Custom Theme Guide.

      In your tiapp.xml you would use it with

       <application android:theme="@style/Theme.AppCompat.Light.NoActionBar">

      The name of the theme is just an example. You can call it what you want.

  • platform/android/res/values/styles.xml

    <?xml version="1.0" encoding="utf-8"?>
<resources>
    <style name="Theme.MyTheme" parent="@style/_MyTheme"/>
    <style name="_MyTheme" parent="Theme.AppCompat.Light.NoActionBar">
</resources>

    In your AndroidManifest.xml

    android:theme="@style/Theme.MyTheme"
