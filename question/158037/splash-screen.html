    Splash screens are found in your &#x2F;app&#x2F;assets&#x2F; folder. &#x2F;iphone&#x2F; for iOS, &#x2F;android&#x2F;images&#x2F; for Android etc.

  • Hi

    Same as non-alloy projects but the root folder is slightly different.

    {appname}/assets/android/images/
{appname}/assets/iphone/

    There should already be some base folders in the android location - you do NOT have to use all of these but you should use as many as your app is likely to need.
