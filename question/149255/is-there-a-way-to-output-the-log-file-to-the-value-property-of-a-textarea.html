    • I don't think you read the docs or my response. Ti.API.log() does not return anything. It prints data to the Titanium console. You give it a string and it prints a string to the console. 

      Ti.API.log('some string');

      That will print "some string" to the console. That's it. I'm not sure what else you are expecting to happen and where your textarea comes into play.
