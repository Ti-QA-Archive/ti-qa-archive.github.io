Setting the Style for the TabbedBar in an Alloy tss

Hi all-

Has anyone had any issues setting the style for the TabbedBar control within the tss file for an Alloy project? When setting a style as thus:

<TabbedBar id='orderEntryBB' height='40px' width='300' backgroundColor='#3E3E3E' style='Ti.UI.iPhone.SystemButtonStyle.BAR'>

The style works as expected, but if moving the style to the tss as follows:

'#orderEntryBB': {
    index: 0,
    height:                '40px',
    width:                '300',
    backgroundColor:    '#3E3E3E',
    style:                'Ti.UI.iPhone.SystemButtonStyle.BAR'
},

The style no longer works (the index does how ever take affect surprisingly). Wondering if anyone has had a similar experience?

Talk soon
Chris

    Hi

    This syntax looks incorrect.

    style: 'Ti.UI.iPhone.SystemButtonStyle.BAR'

    Change this to;

    style: Ti.UI.iPhone.SystemButtonStyle.BAR

    Then try again.

    Ti.UI.iPhone.SystemButtonStyle.BAR is a constant, including it in quotes - turns it into a meaningless string I am afraid.
