Alloy - Align Object from Center of Device display

How can I, using Alloy, align a label, instead of using "left", "right" or "top" to align an object in the device window using:

deviceWidth = Ti.Platform.displayCaps.platformWidth;

Then in my styles code:

"#label1": {
    textAlign: deviceWidth - 50dp
}

I know this doesn't work and doesn't comply with Alloy syntax but I think from this simple example you can understand what I am trying to do.

It seems more logical to align objects this way since different devices have different widths and heights. I could then have a safe zone where all devices would have the objects appearing in the same relative proximity to each other rather than being pulled to the outer edges of the device screen.

  • Depending on your layout. Usually labels are inside views so give the lablel 100% width then
    textAlign: Ti.UI.TEXT_ALIGNMENT_CENTER,

    This will set all the labels in view. Ti.UI.SIZE means height of the text filling the label.

    "Label": {
width: "100%",
height: Ti.UI.SIZE,
textAlign: Ti.UI.TEXT_ALIGNMENT_CENTER,
}

    You can also give it some margin from object above Label;

    "Label": {
top: "10dp",
width: "100%",
height: Ti.UI.SIZE,
textAlign: Ti.UI.TEXT_ALIGNMENT_CENTER,
}

  • Just calculate in the app/alloy.js file

    app/alloy.js

    Alloy.Globals.offCenter = Ti.Platform.displayCaps.platformWidth / 2 - 50dp;

    index.tss

    "#label1": {
    textAlign: Alloy.Globals.offCenter
}
