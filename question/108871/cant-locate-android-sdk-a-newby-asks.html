Hello,

    • Steve

      To resolve your issues, perform these steps in this order:

      • add /path/to/java-jdk/bin to the end of your system path
      • add /path/to/android-sdk/tools to the end of your system path
      • add /path/to/android-sdk/platform-tools to the end of your system path
      • configure a JAVA_HOME variable as /path/to/java-jdk
      • share a screenshot of your android tools "Installed Packages" screen
      • run the script I pasted above again, and paste your new configuration to pastebin, as you did before.

    • Steve

      The correct version of Java is being used by default, which is good. However, have you been able to locate the java and javac executables yet? You have &#x2F;Library&#x2F;Java&#x2F;Home&#x2F; in your path - is this correct? Usually the executables are in a directory named bin, and it is this directory that needs to be in your PATH.

      You have not yet configured your JAVA_HOME environment variable. If &#x2F;Library&#x2F;Java&#x2F;Home&#x2F;bin&#x2F; is the path to your java executables, then JAVA_HOME will normally be set to &#x2F;Library&#x2F;Java&#x2F;Home&#x2F;.

      Have you followed the information in the guide, Preparing for Android development? Read that section through, and take note of the last bit about adb.

      Once you have addressed these issues, pastebin your diagnostics again and I will take another look.
