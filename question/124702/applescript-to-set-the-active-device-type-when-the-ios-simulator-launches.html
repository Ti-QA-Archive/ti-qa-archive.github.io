AppleScript to set the active device type when the iOS Simulator launches

If you frequently switch between device types in the iOS simulator, you probably find Titanium Studio's habit of shutting down the simulator when you select a different device particularly aggravating. Especially so when you discovered you have to run the simulator manually, set the device type, exit, and relaunch your Studio config to make it stick.

So I wrote an AppleScript you can run which will allow you to select the device type and save your selection to the simulator's plist file. It doesn't solve the issue of Studio shutting down the simulator but it makes the switch more convenient.

Also available as a gist

set selectedDevices to choose from list {"iPhone", "iPhone (Retina)", "iPad"} with prompt "Choose device type:" default items {"iPhone"} without multiple selections allowed
if selectedDevices is not false then
    set selectedDevice to item 1 of selectedDevices as string
    set thePListFolderPath to path to preferences folder from user domain as string
    set thePListPath to thePListFolderPath & "com.apple.iphonesimulator.plist"

    tell application "System Events"
        tell property list file thePListPath
            tell contents
                set value of property list item "SimulateDevice" to selectedDevice
            end tell
        end tell
    end tell
end if
