non-DOM, javascript animation library for Titanium mobile?

Has anyone successfully used any animation library or framework with Titanium mobile? The standard Animation class is lacking, and buggy, as it does not allow simultaneous animations to be operating on an object at the same time.

For instance, you can use Animation to alter the top and left at the same time. You have to alter left, and then addListener for 'complete' and start another animation, but even that is buggy as you often get debug errors that the second animation is being ignored because Ti is still "updating the layout", as the log shows.
