iPade setImage from blob

I want to generate an image as follows and then add it to an imageview as blob. It works in the simulator, but on the iPad, the app crashes.

Does anyone have the same problem or did anyone find a solution.

var scaleFactor = 0.5;

        var rb = navController.mapBlob;
        var bx = (_work.mapPosition.x - (roomImage.getWidth() / 2) / scaleFactor);
        var by = (_work.mapPosition.y - (roomImage.getHeight() / 2) / scaleFactor);
        var b = rb.imageAsCropped({
            x : bx,
            y : by,
            width : roomImage.getWidth() / scaleFactor,
            height : roomImage.getHeight() / scaleFactor
        });
        roomImage.setImage(b);

where roomImage is a imageView.
