  • Well, on your first few lines, you have:

    python.check: [echo] Testing for Python [exec] Error: No python executable could be found on your system

    You need Python in order to build modules correctly. So you must install it. If it is installed, make sure it is present in your build path.

    One simple way to check is to launch a Command Prompt and type python. It should be a recognizable command. If not, you need to adjust this setting.
