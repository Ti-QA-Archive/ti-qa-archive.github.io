Ti.App.addEventlistener in webview freezes MobileWebPreview in Browser

Hi. I am working with a webview and local htmls. According to the documentation the way to go in order to communicate with the webview from other parts of Titanium is to add eventlistener like this: 

Ti.App.addEventListener("app:fromTitanium", function(e) {
                alert(e.message);
              });

This works on mobile devices (at least I tested it on android) but it freezes everything when you run it as a Mobile Web Preview in the browser. Since it takes quite some time to install the project to the phone every time I want to test something this is really troublesome. Is there a way to make this work in the Mobile Web Preview on the browser?

I am using Titanium Studio, build: 3.1.0.201304151600 on a windows 7 64bit machine. The browser is Firefox 20.0.1.
