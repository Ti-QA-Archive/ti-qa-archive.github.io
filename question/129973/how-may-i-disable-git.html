How may I disable GIT?

I've never used GIT before and just downloaded the latest Kitchen Sink from GitHub, and somehow I think GIT has latched onto my other project. It added the text [ master*] next to my project name and now all of my project files appear in the App Explorer with a red background and an asterisk to the left of each file name.

How can I get rid of this? I use another source code management product.

  • Accepted Answer

    In the App Explorer Panel, choose the Commands icon (right most icon) for your project, scroll to More and choose disconnect.
