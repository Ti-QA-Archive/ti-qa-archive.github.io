TideSDK - New Site Launched as TideSDK Continues to Move Towards Release

The TideSDK Team has been working hard towards a release of TideSDK 1.3.0-beta. The 1.3.0 release will come with major library upgrades to support OSX Lion, Mountain Lion, Ubuntu 12.04, Windows 7 and 8, updated notifications support for growl in OSX, and bug fixes for improved stability. Updates to WebKit will ensure the latest HTML5 support available.

For current application development, please continue to use Titanium Desktop 1.2.0.RC4 in the interim. We will make an announcement about the availability of the beta. A new Nightly Build site will be launched for downloads so that you can begin to experiment with the beta. Please follow us on Twitter https://twitter.com/TideSDK to keep updated on what is happening with TideSDK and the upcoming release.

The TideSDK Team is currently developing a TideSDK Builder app. It will provide a GUI for creating, running and packaging TideSDK apps. To get developers started faster, TideSDK Builder introduces a new feature - Scaffolds. Scaffolds generate all the boiler plate to instantiate projects using specific patterns of development (such as Backbone MVC). Developers will be able to create, import and share Scaffolds.

The roadmap for TideSDK includes the implementation of CommonJS for 1.4 which aims to provide developers with a more modular development experience using JavaScript.

We have previously reported about our rebranding and documentation efforts. We are pleased to announce the launch of our new site TideSDK.org as we continue to prepare for the upcoming release. Please check it out and let us know what you think.
