    • We have encountered the same problems for Android while trying to implement pinch zoom in the webview. We're using the leaflet javascript library to display offline tiles and json features, and pinch zoom functionality is missing. If I point the webview to the same html code hosted on a website, pinch zoom works fine.

      Roeland–based on your hint, we tried rolling back our SDK from 3.0.0 to 2.0.x and, sure enough, pinch zoom works! Thanks.

      Can anyone explain why this functionality was removed/lost in later versions of the SDK?

      By the way, our current workaround to the missing pinch zoom is to create a Ti.UI slider bar and fire events back to leaflet to zoom in or out. It's not pinch zoom but it does allow users to quickly zoom in and out. We tried a nice leaflet slider library available here but performance in the webview was a little sluggish.
