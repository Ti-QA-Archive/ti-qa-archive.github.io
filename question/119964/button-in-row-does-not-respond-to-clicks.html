    • Try adding a custom property to the button with the index id of the row it's in. Then when the button is clicked, you will know the index of the row to remove.

      eg: add this after your button creation

      button.rowIndex = i;

      then in the button click listener, you will be able to get that using e.rowIndex (if not, try e.source.rowIndex - I forget off hand ; ) )
