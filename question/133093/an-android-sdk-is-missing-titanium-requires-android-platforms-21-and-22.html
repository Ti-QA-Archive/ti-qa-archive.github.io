An Android SDK is missing. Titanium requires Android platforms 2.1.* and 2.2.*

Application Type: Mobile

Titanium Sdk: 1.8.2

Hosting Operating System: OSX 10.6.8

Titanium Studio: 1.0.9

After updating Titanium Studio to the latest version this morning, it is not able to detect android sdk. I get a message:

"An Android SDK is missing. Titanium requires Android platforms 2.1. and 2.2." when I go to "Configure" tab in the studio.

I tried installing updates, checked android sdk manager and the required apis are all installed. Can anyone throw some light on this. I could see that some people have been experiencing the same issue on Windows.

  • my path was not setup correctly after re-installing android SDK a few times. So make sure you have the path for ../android-sdk/tools and ../android-sdk/platform-tools. I would edit .bash_profile to include these paths so that they are available everytime when your shell is started. Also, run Android SDK Manager to make sure all necessary packages are installed, and use the Tools / Manage AVDs to test run your Virtual Android Device. Exit Ti Studio and restart it again. Hope it helps.
