How to remove multiple records in Cloud.Places with a single Web Service Request

I pray that I am not overlooking this, but it seems on Appcelerator Cloud, you can only remove 1 record at a time from the Places Collection.

Based on their documentation online, you would use the following:

Cloud.Places.remove({
    place_id: savedPlaceId
}, function (e) {
    if (e.success) {
        alert('Success');
    } else {
        alert('Error:\n' +
            ((e.error && e.message) || JSON.stringify(e)));
    }
});

In my case, I have 1000+ records that I want to remove, which means I have to submit 1000+ Delete Requests.

As a result of this, I'm receiving timeouts after 1 minute.

Please can someone enlighten me on how you would approach this situation?
