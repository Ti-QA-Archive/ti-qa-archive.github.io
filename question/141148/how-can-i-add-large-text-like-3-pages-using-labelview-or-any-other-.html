    Hi Nadeem

    Can you provide the code you are using to create the labels at the moment?

    If your sdk is 2.0 or higher then you can use Ti.UI.SIZE as the value for the label height for example;

    var label = Ti.UI.createLabel({
...
  height: Ti.UI.SIZE,
...
});

    You can then add your labels to a scrollView with a layout: vertical.
