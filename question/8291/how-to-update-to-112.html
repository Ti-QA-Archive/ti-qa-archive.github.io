How to update to 1.1.2

I'm using 1.0 and just clicked the link in the upper right to update to 1.1.2. Something downloaded but it didn't install (still showing 1.0).
Do I have to update manually? Where did the download go? And the link is gone now so i can't redownload…

    Yep, Tamas is right, you should see 1.1.2 as an option now in the Titanium SDK combobox on your apps Edit screen.

    The app itself is still at 1.0, so that's normal, so the 1.0 you see in the title bar of the app is okay.

    It's sorta confusing, you actually have three different version numbers with Titanium. You have one for the Developer tool itself, one for the mobile SDK, and another for the desktop SDK. The 1.1.2 you downloaded was an update to just the mobile SDK.
