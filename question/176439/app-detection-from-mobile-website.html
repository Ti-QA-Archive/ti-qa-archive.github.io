App detection from mobile Website

Hello,
i built an iOS-App and a website for it. If you visit the Website i want to check if the App is already installed or not. If yes, the user should have the option to open it, if not i want to link in appstore….
My question: What do i have to do inside the app, and what is needed on the website to do that?

Thanks!

    Hi

    There is built-in functionality in mobile safari for this - you simply add a meta tag to each page you want it to appear on;

    <meta name="apple-itunes-app" content="app-id=xxx">

    Replace xxx with your apps id. You get this number from the iTunes Connect page - look for app link and it is the number at the end.

    It displays like this;

    • If the app is installed it says View
    • If not installed it allows the visitor to buy/download the app
