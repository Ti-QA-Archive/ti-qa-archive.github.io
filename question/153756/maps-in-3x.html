Maps in 3.x

Hi,

I have an existing app (written in 2.1) that is using the original maps API. I plan on releasing updates to this app, do I need to upgrade the maps if I'm happy with the current functionality?

Am I correct in understanding that as of the maps v2 I have to handle Android and iPhone differently (with respect to map code)?

    Hi

    As you have already obtained an API key for your app and you know it is working - you do NOT need to change your maps to use the new version.

    • I should have said why;

      The deprecation is currently about not providing new API keys for Android maps v1, if you have one you are fine (for the moment).

      iOS maps do not require API keys as you know.
