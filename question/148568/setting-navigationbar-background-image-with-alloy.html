setting navigationBar background image with Alloy

hi, is it possible to change the navigationBar image with Alloy?
I tried to set the navigationGroup's backgroundImage property but it won't work.
I also tried to pull out the navigationBar in code and call setBackgroundImage, and results the same…

please help, thanks!!

XML:
<NavigationGroup id="nav" backgroundImage = "mbdnavigationbar.png">

in CODE:
var navigationBar = this.getView('nav');
navigationBar.setBackgroundImage ('loginbackground.png');

    Two things:

    1. You want to set the barImage property on the Window contained within the NavigationGroup. Setting the backgroundImage property on the NavigationGroup, as you already know, probably does nothing.
    2. get your navigationbar like this, it's cleaner in the controller:
      var navigationBar = $.nav;
