Error when installing LiveView Plugin

When trying to install LiveView to Titanium Studio, I get this error:

Cannot complete the install because one or more required items could not be found.
  Software being installed: LiveView Feature 1.0.0.201307021617-3--59H5F8B9F9 (com.appcelerator.titanium.liveview.feature.feature.group 1.0.0.201307021617-3--59H5F8B9F9)

Does anybody know how to resolve this?

Titanium Studio, build: 3.1.1.201306112235, Mac OS X 10.8.4

    Hi

    LiveView is an enterprise feature as noted in the documentation Appcelerator Studio LiveView tool and as you will note in the side bar it is in the enterprise section of the documentation.

    You have stated you are using Titanium Studio (the free version - most users) rather than the requires Appcelerator Studio (the paid version). You cannot use LiveView without the paid enterprise version I am afraid.

    You can purchase the paid version if you wish to make use of this feature, be aware the prices are set for the truly enterprising of enterprises.

    Check out TiShadow for an alternative.
