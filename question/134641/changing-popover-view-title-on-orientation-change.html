Changing Popover view title on orientation change

I want to set some property say title for popover view on orientation change. But it's not reflecting the new value.
The scenario : if popover view is open and I change the ipad orientation, at this orientation change, I want to change the title of the popover, but since the popover is already open it does not take new property into account.

Ti.Gesture.addEventListener('orientationchange', function (e) {
popoverview.title = 'abc';
}

    Hi Virendra,

    This code sets the popover title for orientation change. good luck.

    Ti.Gesture.addEventListener('orientationchange', function(e) {

    switch (Titanium.Gesture.orientation) {

        case Titanium.UI.LANDSCAPE_LEFT:
            popover.setTitle("LandscapLeft");
            break;
        case Titanium.UI.LANDSCAPE_RIGHT:
            popover.setTitle("LandscapRight");
            break;
        case Titanium.UI.PORTRAIT:
            popover.setTitle("Portrait");
            break;
        case Titanium.UI.UPSIDE_PORTRAIT:
            popover.setTitle("Upside Portrait");
            break;
    }
});
