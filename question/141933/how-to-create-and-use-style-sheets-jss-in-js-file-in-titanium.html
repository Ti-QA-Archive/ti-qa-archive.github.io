  • Support for .jss files will be going away in future versions of Titanium Mobile. They are only supported by Android and iOS, not Mobile Web or other upcoming platforms. There is no documentation that I'm aware of for .jss files.

    You should take a look at Alloy. It's a MVC framework designed to make things like styling easier.
