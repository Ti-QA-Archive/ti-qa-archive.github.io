  • Accepted Answer

    Personally, I thought getting started was very difficult. And, even after getting started, the development of applications is very difficult as well. However, I am assuming that in order to accomplish developing iPhone and Android applications the overall time will be cut in half by using Titanium Developer because otherwise I would have to learn how to write and use the iPhone and Android SDKs.

    I didn't use the wiki tutorial Andrew mentioned. I googled for some of the installation issues I was having and found a tutorial by Query7 to be helpful.

    I essentially started by using other people's examples:

    • http://cssgallery.info/custom-row-for-tableview-in-appcelerator-titanium/
    • http://query7.com/titanium-mobile-android-development-first-application
    • http://mobile.tutsplus.com/tutorials/appcelerator/titanium-mobile-database-driven-tables-with-sqlite/

    Oddly enough, I have actually found this Q&A to be VERY helpful. Typically you will get a same-day response, if not within minutes. Be sure to view the Participating in the Q&A guide for tips on getting more accurate answers more quickly.

    Hope any of this helps!
