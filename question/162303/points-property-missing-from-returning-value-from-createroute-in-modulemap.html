    • Again the reason you cannot do this line of code;

      testRoute.points.concat([point]);

      Is because you are assuming that;

      var testRoute = MapModule.createRoute({name: 'test', points: testPoints, color: 'blue', width:3});

      Returns the same object you passed into it into the variable testRoute. But it returns an object reference not simply the same data you gave it.

      So;

      testRoute.points // this is not a returned object reference
// so
testRoute.points // is not an array
// so 
testRoute.points.concat([point]); // you cannot perform array functions on it

    • Assuming then that the route is already on the map and you wish to add another point to it - then you have to remove the existing route and add a new one that includes the new point data.

      // initially
var testPoints = [
    {latitude: 38.7137942622212076, longitude:-9.13338303565979},
    {latitude: 38.763495, longitude:-9.093763},
    {latitude: 38.71371054721669, longitude:-9.160258769989014},
    {latitude: 38.71203413379506, longitude:-9.140643775463104},
];
var testRoute = MapModule.createRoute({
    name: 'test', 
    points: testPoints, 
    color: 'blue', 
    width:3
});

      When you want to update the route

      testPoints.push({
    latitude: 38.8, 
    longitude: -9.1
}); // adds a new point

// remove the previous one
MapModule.removeRoute(testRoute);
// create the revised route
testRoute = MapModule.createRoute({
    name: 'test', 
    points: testPoints, 
    color: 'blue', 
    width:3
});

      You could always add a point like this if you prefer

      var point {
    latitude: 38.8, 
    longitude: -9.1
};
testPoints.push(point);
// now remove and recreate as before
