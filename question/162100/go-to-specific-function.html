  • Let win1 is first window from which you open second window. And you want to call specific function when window2 is closed right ?

    add that function in first windows's focus event

    win1.addEventListener('focus', function(e) {
        myFunction();
})
    — commented January 30th 2014 by Muhammad Adnan
