Required jarsigner not found

I usually face this error and this is a model soultion for it
After an hour, this is what worked for me:
` set JAVA_HOME to c:\jdk (or any path with NO spaces)` * set PATH=(previous values);c:\jdk\bin (no spaces and dont use variable names like %JAVA_HOME\bin%)

Close Titanium
Launch Titanium from command prompt with -clean option. eg: c:\&gt;&quot;C:\Program Files\Appcelerator\Titanium Studio\TitaniumStudio.exe&quot; -clean

Works!!!
