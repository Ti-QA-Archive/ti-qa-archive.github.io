  • Do you mean to increase the number in the bubble?

    Although I'm not a ti.cloud user I've read about the internals of push notifications and implemented push by myself in Ti applications.

    I can imagine a way to do that, manually.
    You'll have to keep the number of unread notifications on the server and implement some webservice to decrease the number of those unread notifications once the user opens the app and reads them.

    So, it is possible, but it will also involve the server in keeping a count of unread notifications and implementing the associated webservice for keeping up to date the unread notifcations.
