    for(c=0;c<6;c++) {
        if(c!=n) {
            oLabel = 'l'+c;
            oLabel.color = '#000000';
            oLabel.backgroundColor = '#FFFFFF';
        }
    }

    Of course it doesn't work, "oLabel" is a string, not label's reference. To dynamically get label reference by it's string name, you could "eval" that string, or even better change everything you need in "eval" command. For example:

    eval(oLabel + ".color = '#000000'"); // this would become l1.color = '#000000'; after parsing.

    But, most probably there are better ways of doing this (maybe if you explain bigger picture, we could help you with that).
