What are some ways I can synchronize the Slider as VideoPlayer plays?

Hello,

I am currently working on a custom control for the Ti.UI.VideoPlayer for iOS 6+ using SDK 3.1.0GA and
I need help trying to figure out how I can update the slider as VideoPlayer plays a media. I can get the current time of the media playing by calling:

videoplayer.getCurrentPlaybackTime()

and update the slider accordingly, but I am not sure which event to fire for there is no event in the API that fires every second or so. Any suggestions?

Thanks in advance

    Hi

    You are correct - there is no event that fires during play back to announce a current state. You will have to create a timer and pole the method getCurrentPlaybackTime() as you have noted.

    You can use the standard setInterval JavaScript method an pole inside that. Make sure you clean up the and cancel the setInterval when you do not need it - otherwise you will create a memory leak.
