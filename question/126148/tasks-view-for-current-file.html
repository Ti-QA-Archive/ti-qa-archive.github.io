Tasks View for current file

How do you show the tasks for the current file only?

— asked September 28th 2011 by William Hopkins
  • tasks, tasks view

  • In the upper-right corner of your Task view, you will see a little arrow pointing down.

    • This will show you a menu.
    • Select the Configure Contents… menu
    • This will bring up a configuration window.
    • Select the TODOs item by checking the check-box next to it.
    • From the Scope section, choose the On selected element only option.
    • Click OK to save your settings. (ignore any warning message)

    From now on, you should only see the tasks related to the current file.
