  • Hi

    You can do this in the XML like so:

    <Label textId="test" />

    This will create your label and add the key string identifier for your locale file.

    • The proper attribute name is textid (all lowercase) but otherwise what Luke says is correct. You can also set the localized values in the TSS or controller:

      '#labelID':{ textid: 'test'},
<!-- or -->
'#someOtherLabel':{ text: L('test')}

      $.labelID.textid = 'test';
// or 
$.labelID.text = L('test');
// or if you like to type a lot
$.labelID.text = Titanium.Locale.getString('test');
