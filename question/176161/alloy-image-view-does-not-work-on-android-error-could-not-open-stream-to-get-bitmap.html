    • The issue here was specifically about if you are trying to reuse the icons or splash screens you provide with the app as part of the bundle.

      As these are denoted as system images, they are moved by each operating system to be part of the app bundle and are no longer accessible to use with your own ImageViews or backgroundImages.

      If you are trying to show one of these images then you will need to duplicate the image and use a different name. Then provide the path to that image instead.

      If this is for anything else; create an new question and paste the link back here I will take a look. It will also allow others in the community the ability to provide you with assistance.
