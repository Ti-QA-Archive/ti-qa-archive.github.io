Text is too small when typing into textfield using 3.5.0sdk

Platform : Android
SDK : 3.5.0

Hi I have developed and app with 3.1.3 sdk previosuly but I need to update to 3.5.0 now. When I updated all my textfields in the app are not visible or too small when typed.

here is my textfield 

var f1 = Ti.UI.createTextField({
                value:'',
                borderStyle:Titanium.UI.INPUT_BORDERSTYLE_ROUNDED,
                font:{fontSize:'11sp',fontFamily:'Helvetica Condensed Medium'},
                color:'#000',
                top:0,
                left:(Ti.Platform.osname=== 'android' && (deviceWidth > 899 || deviceHeight > 899)) ?80:82,
                //width:(Ti.Platform.osname=== 'android' && (deviceWidth > 899 || deviceHeight > 899)) ?450: 250,
                editable : true,
                borderRadius:3,
                borderWidth:1,
                borderColor:'#000',
                width:(Ti.Platform.osname=== 'android' && (deviceWidth > 899 || deviceHeight > 899)) ?'185':'28%',
                softKeyboardOnFocus : Titanium.UI.Android.SOFT_KEYBOARD_SHOW_ON_FOCUS 
            });

And one more thing in tiapp.xml, for 3.5.0, the document says support screens for density = 'false' was deprecated and instead we should use this below.

<supports-screens
            android:smallScreens="true"
            android:normalScreens="true"
            android:largeScreens="true"
            android:xlargeScreens="true"    
        />

But when use this and whn I checked my devicewidth and height using 

Ti.API.info('Height: ' + Ti.Platform.displayCaps.platformHeight);
    Ti.API.info('Width: ' + Ti.Platform.displayCaps.platformWidth);

dislayed larger width and height to my original size of my device and all the UI components were disturbed.
Please help me anyone.
Thanks
N.G.S. PRASHAANTH
