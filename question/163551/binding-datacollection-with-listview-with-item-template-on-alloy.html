  • I was able to make a workaround. Instead trying to bind the template I used dataTransform

    <ListSection dataCollection="appointment" dataTransform="doTransform">

    to read an eventType

    function doTransform(model)
{

   var o = model.toJSON();


    if (o.eventType == "type1")
    o.template = ''template1"
    else if (o.eventType == "type2")
    o.template = ''template2"

      return o;
}

    and attach the template name to the model.

    I'm not sure if this is the best way to do it, but its working right now.
