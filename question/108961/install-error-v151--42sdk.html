  • One resolution to this issue may be to delete your build files and force a full rebuild.

    For iOS/iphone, delete the contents of the project's build&#x2F;iphone directory, select Edit Project in Titanium Developer and click the Save Changes button to force a full rebuild. Then launch the app.

    Also, inspect the build&#x2F;iphone&#x2F;build&#x2F;build.log file for errors. If you need help with that, create a pastebin (ie NOT here) and someone will take a look for you.

    Cheers
