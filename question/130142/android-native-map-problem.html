  • Hi Eric.. :).. i have a solution for this now..

    var win = Ti.UI.createWindow({
    modal: true,    //set the window modal first before anything else
    navBarHidden: true
});
var map = Ti.Map.createView();
win.add(map);
win.open();

    i hope this works for u.. :)
