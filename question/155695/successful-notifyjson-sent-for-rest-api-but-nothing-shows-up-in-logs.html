Successful notify.json sent for REST API, but nothing shows up in logs...

So, we use the REST api to send push from our server.
We do a login with an admin user, and then send a push with the notify.json url.

Server output: { "meta": { "status": "ok", "code": 200, "method_name": "Notify" } }

On the appcelerator website in the "Logs" we see (replaced our key with XXX btw):
