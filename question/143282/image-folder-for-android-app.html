Image folder for Android App

Hi,

Im working in a App for Android that saves pictures in Extorage Card (To many pictures). In sd card or similar, i can find my folder named as the application packet, i mean com.something.something, inside this there are my pictures.

I know save in image gallery is a function only for IOS. Android works different but i would like that the name of my folder will be diferent, not the package name, or even better that the pictures can be saved on camera or picture folder in SD, then when user opens gallery always can find the app pictures.

Is that possible?
