How to add files to git ignore list in Titanium Studio?

I have some files in my project that are either programatically generated or are the ones whose changes are not supposed to go back to repository (like config files). I would like to add these files to git ignore list so that it does not ask me for staging those files again. How do I add them to the list from Titanium Studio?

  • Studio will use the .gitignore file from within your project. You can simply flip over to the terminal tab in your project, open the .gitignore file, make you changes, and save the file. When you use the built in git functionality, your new .gitignore file should be respected.

    • You shouldn't have had to create that file. If your project was a proper git repo, then the default file should have already existed. So you can manage your git repo in the same manner as you would a non Studio based project.
