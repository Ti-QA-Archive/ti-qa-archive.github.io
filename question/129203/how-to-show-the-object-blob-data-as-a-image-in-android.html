  • The imageView.image property is supposed to be able to take a blob as an argument. So you shouldn't have to do anything to it. Just set yourImage.image = yourBlobObject.

      var oCardNavOff = Ti.Filesystem.getFile('images/card/CardNavOff.png');
var blob = oCardNavOff.read();

var navOffImg = Ti.UI.createImageView({ top: 30, left: 30, width: 30, height: 30});
navOffImg.image = blob;
navContainer.add(navOffImg);
