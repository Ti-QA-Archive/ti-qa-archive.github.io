Alloy Set Label Text on Window or View Load

All,

I am checking to see if the user is logged in, and if they are, I want to set a label's text to reflect the logged in user. I want to do this when my Alloy View is loaded, not on a button click. I have an Init function that is called when the view is loaded using the onPostlayout event. Whenever I try to set the text with $.lblLoggedIn.text = 'some text' I get an error that the label is not found. I have an id set to lblLoggedIn on the view.

Can anyone kindly point out what I am missing?

Thanks,
B

  • Hi

    I just re-read this and failed to note the post layout (no idea how).

    There is no need to use a post layout event. Just set you value in the controller.

    $.lblLoggedIn.text = 'some text' ;

    This assumes the xml looks like this;

    <Alloy>
...
   <Label id="lblLoggedIn" />
...
</Alloy>
