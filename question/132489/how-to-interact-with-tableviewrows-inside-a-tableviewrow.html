How to interact with TableViewRows inside a TableViewRow

When one of my TableViewRows is clicked, a javascript function is called and I know the index of the clicked row. But I'm having trouble figuring out how to perform the following 3 actions when any given row is clicked:

1- Reach into the ImageView inside the selected TableViewRow and change its image programmatically.

2- Reach over to the Detail View (my table is in the left side of a Split Window) and load/display a form I have (that goes with the row clicked) containing text boxes.

3- Change the background color of the selected row.

Thanks in advance for your help. I apologize for asking 3 questions in one, but I believe they're all quite simple, and all related.

// loop through sections and create rows
var tbl_data = [];
for(var i = 0; i < sections.length; i++) {
    var row = Ti.UI.createTableViewRow();

    var image = Ti.UI.createImageView({
        left: 5,
        width: 38,
        //image: 'check.png'
        image: 'blank.png'
    });


    var label = Ti.UI.createLabel({
        left: 45,
        text: sections[i]
    });

    row.hasDetail = true;
    row.add(image);
    row.add(label);

    tbl_data.push(row);
}

    row.addEventListener('click', function(e) {

                    var tappedRow = e.row;
                e.source.backgroundColor = 'black'; // change background color of row
                tappedRow.children[0].image = 'changeimage.png';
                tappedRow.children[1].color = '#ffffff'; // change background color of label
            });
