    Hi Alexey,

    I am not really sure if this is your problem: CLI: Windows 7 hangs on build during "Powershell" detection.

    But it sounds like as you are affected by this tooling bug.

    Give it a try and replace line 377 inside C:\ProgramData\Titanium\mobilesdk\win32\3.4.0.GA\node_modules\titanium-sdk\lib\windows.js

    make a backup prior changing!

    replace line 377

    if (!selectedVisualStudio) return next();

    with

    return next();

    Greetings,

    Matthias
