Event Bubbling

I have a view to which I'm adding an icon grid. The view contains a view for each row, in each row a container view for each cell to provide the spacing and each container view contains an icon (imageView). Broken down:

TabGroup -> Window -> Grid (view) -> Row (view) -> Cell (view) -> Icon (imageView)

For memory conservation, I've dropped an event listener (click) on the outermost view and I'm trying to identify the source using e.source. If the source is an icon, a custom property provides further instructions. What I'm finding is that the e.source value always seems to be the containing view even though I'm very deliberately clicking on the icons.

Any idea what might make this happen? I swear this was working before, but I don't see anything fundamentally different.

Thanks.
