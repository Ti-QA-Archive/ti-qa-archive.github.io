  • Hi Silvia,

    JSS support is probably removed from 3.0GA and later SDKs. Its better you use Alloy MVC framework in which each xml has its own tss(Titanium StyleSheet) file which is equivalent to jss file.

    For more information about alloy applications please go through Alloy official docs :http://docs.appcelerator.com/titanium/latest/#!/guide/Alloy_Concepts and Titanium style sheets
