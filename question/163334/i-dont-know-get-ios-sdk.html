  • Hi

    I am going to guess that you are not using Apple OSX and instead you are using Windows or Linux. If this is true then you cannot install iOS simulator as it is part of xcode (a required part of iOS development using Titanium).

    • See Titanium Compatibility Matrix

    If you are on a mac using OSX and do meet the minimum requires from the link above - then make sure you have installed XCODE. Which can be done from the iOS Dev Center or from the app store on OSX.

    From there follow the instructions found on the Titanium Studio Dashboard tab > "Get Started"
