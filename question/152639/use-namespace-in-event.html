Use namespace in event

Hi,
I use namespace in declare event on markup language in alloy like this:

Home.xml:

<Button id="orderButton" onClick="Home.onOrderClick"></Button>

Home.js:

var Home = {}

Home.onOrderClick = function() {

}

but when I run my app I got this error:

undefined&#39; is not an object (evaluating &#39;Home.onOrderClick&#39;)

What's my wrong?

  • This is currently a limitation in the way the markup events are handled. They need to be a standard function delcaration at the top-level scope of the controller:

    function onOrderClick() {}

    That said, I think I see how I can make your situation work in Alloy. Can you log a Titanium Community Jira Project ticket with these details? Make sure to give it the label "alloy" so I see it. I'll get it fixed for Alloy 1.2.0. I can also let you know when it's available in the master branch so that you can test it if you like. Thanks.
