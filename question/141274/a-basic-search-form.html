A basic search form

Hi I am wanting a basic search form currently my code looks like this:

 var textField1 = Ti.UI.createTextField({
  borderStyle: Ti.UI.INPUT_BORDERSTYLE_ROUNDED,
  color: '#336699',
  width: 250, height: 60
});

textField1.addEventListener('return',function(event) {
 alert("submitted");
 //do stuff here for the search

});

Am I correct in thinking that the object 'event' would hold what was entered into the text field?

if so how can I convert it to a string because toString does not work on it.

Thanks

    Hi Michael

    If you are after a basic search box there are native controls that solve this for you, rather than using the textField.

    var search = Ti.UI.createSearchBar({
    barColor:'#000', 
    showCancel:true,
    height:43,
    top:0,
});
search.addEventListener('return',function (e) {
    alert(e.source.value);
});

    It will return the result as a string for you already. The e parameter holds a reference to the original view or object in a parameter called source, so if you reference (in this case) e.source.value it will know the current entry of the value parameter.

    You can use a textField as you have in your own example but the searchField has built in extra that can make the experience simpler for you and the user.
