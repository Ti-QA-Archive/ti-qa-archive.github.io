    Hi Devanjali,

    Direct answer to your question is NO. You can not able to customize keyboard type. This is OS feature which gives you different keyboard type, you can not modify that not even in native.

    To give solution to your query, you have to create your own Keyboard layout using Ti View and buttons and manage its functionality so that it can behave as customize keyboard.
