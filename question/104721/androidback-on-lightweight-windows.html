Android:back on lightweight windows

I am opening a lightweight window in order to have the tab group displayed along the top on all windows. However, when I do this, the app closes when the back button is closed. I've tried using exitOnClose on the window when creating it and that does nothing. Also, I've tried adding a listener for the android:back event. This event never gets called. Does that event not get fired on lightweight windows?

I could use a heavyweight window, but the tabBarHidden property does not work either.

I'm using APIs 2.2 and the nightly build 1.6.0

Thanks

win.addEventListener('android:back', function(e) {

  • Brian

    Android hardware button events only function when attached to heavyweight windows, as explained in ticket #1641.

    Note that a tabBar is not related to a tabGroup. Also, it is an iPhone-only feature.

    If you want a "tab group displayed along the top on all windows", then why not use the tabGroup in the standard way, by attaching a window to each tab using the url property?

    It's difficult to know exactly what you are trying to achieve.
