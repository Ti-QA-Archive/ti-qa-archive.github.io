Add event listener to an element loaded in a webview

I am working on a sample app using titanium.

    webview.html = '<div><a href="http://google.com" id="ggle"></a></div>'

is it possible for me to add event listener for the anchor tag specified in the html?

  • You can use Ti.App.addEventListener(), or you can use a <script></script> and write a function there and call that function on the onload of the tag you want in this case like this 

    var webview = Ti.UI.createWebView({ borderWidth:0, paddingRight:10,width:310,top:25, height:210,left:5 });
    webview.html = '<div onload="yourFunction"><a href="http://google.com" id="ggle"></a></div>'
