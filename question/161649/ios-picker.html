iOS Picker

Hey.

I have a quite weird issue with an date type picker.

the picker works wonderful unless the day is bigger than 24 the picker value skips one day.
So when i select the 23rd of Sept it works, same with 24th. But if i select the 25th the value of the picker is 26. (the 30th of Sept is he 1st of Oct)

[INFO] : User selected date: Fri Sep 23 1988 23:00:00 GMT+0200 (CEST)
[INFO] : User selected date: Sat Sep 24 1988 23:00:00 GMT+0200 (CEST)
[INFO] : User selected date: Mon Sep 26 1988 00:00:00 GMT+0200 (CEST) <-- this sould be the 25th
[INFO] : User selected date: Tue Sep 27 1988 00:00:00 GMT+0200 (CEST) <-- this should be the 26th

As you can see theres also a difference with the time (23:00:00 vs 00:00:00) where does this come frome?

i create my picker like this.

var picker = Ti.UI.createPicker({
    type:Ti.UI.PICKER_TYPE_DATE,
    minDate:minDate,
    maxDate:maxDate,
    value:value
});

picker.addEventListener('change',function(e){
    Ti.API.info("User selected date: " + e.value);
});

  • Hello!

    I did some review into this, with:

    Versions

    Mobile SDK 3.2.0.GA
    iOS7

    Test Sample

    var win = Ti.UI.createWindow({
    backgroundColor : 'white,'
});

var picker = Ti.UI.createPicker({
    type : Ti.UI.PICKER_TYPE_DATE,
    minDate : new Date(1988, 0, 1),
    maxDate : new Date(2014, 11, 31),
    value : new Date(2014, 3, 12),
});

picker.addEventListener('change', function(e) {
    Ti.API.info("User selected date: " + e.value.toLocaleString());
    alert("User selected date: " + e.value.toLocaleString());
});

win.add(picker);

win.open();

    Steps to reproduce

    • Create a new mobile project (in Classic Titanium)
    • Paste the code into app.js
    • Run into simulator.

    Chages in the code

    The only difference in our code is the use of e.value.toLocaleString(), which will show you the date/time in your timezone.

    Please try this code and let us know how it goes.

    Best,

    Mauro

    • Try this: 

      var col = Ti.UI.createPickerColumn();
var data = ["foo", "bar", "foobar"];
_.each(data, function(d) {
    var pr = Ti.UI.createPickerRow({
        title: d,
    });

    col.addRow(pr);
});

picker.add(col);

picker.addEventListener('change', function(e) {
    self.selectedValue = e.rowIndex;
});
