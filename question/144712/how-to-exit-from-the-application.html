How to exit from the application

Hello,

I am trying to implement the following code:

var myAlert = Titanium.UI.createAlertDialog({
    title:"Exit",
    message:"Do you really want to quit?",
    buttonNames:["Yes","No"]
});


image1.addEventListener("click",function(e){

    myAlert.show();


});


myAlert.addEventListener("click",function(e){

            if (e.index===0){
                myWindow.close();
            }
            else if (e.index===1){
                return;    
            }

    });

The problem is when I click on "Yes", I got the red Splash Screen, but I want to exit completely from the application!

    title:"Animation",
    backgroundColor:"#FFF",
    exitOnClose:true
