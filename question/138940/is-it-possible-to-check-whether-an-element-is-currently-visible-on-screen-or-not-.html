  • Hi Sahil,

    You can do it using your custom object. Set your custom object while displaying or creating and check while your scrollview scroll.

    Don't forgot to set your customObj in your VIEW. like this way

    var this_label = Ti.UI.createLabel({
    text:"hello",
   customObj: 'true

});

    Then try this code.

    scroll_view.addEventListener('scroll', function(evt) {
    var btns = scroll_view.children;
    for ( i = 0; i < btns.length; i++) {
        if (btns[i].customObj == 'true') {
            alert('visible');
        };
    }
});

  • I haven't done extensive testing on this code, but it appears to be working. It assumes the object you are checking is a direct child of the scrollview. It will require modification if you are checking subchildren.

    function onScreen(obj){
    var scrollPos = obj.parent.getContentOffset(),
        scrollViewHeight = obj.parent.rect.height;


    var aboveViewPort = (obj.rect.y + obj.rect.height) < scrollPos.y,
        belowViewPort = scrollViewObj.rect.y > (scrollPos.y + scrollViewHeight);

    if (aboveViewPort || belowViewPort){
        return false;
    } else {
        return true;
    }
}
