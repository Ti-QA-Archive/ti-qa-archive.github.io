    For example Alloy cannot use internationalization in markup file,
    that means GUI element which are creation only "co" like the optiondialog (http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.OptionDialog)
    cannot use internationalization.

    You cannot access parent element within included/required file,
    the parent element you want to access from inner element has to be global or
    you create the inner part as widget and pass the parent element to the widget.
