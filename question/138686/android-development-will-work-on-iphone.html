  • yes the code will work but make sure any android dedicated API should be called when you are running iphone build. This might cause your app to crash.

    To make such differentiation in your code you can segregate the code like this:

    
if(Ti.Platform.osname == 'android')
{
    //android specific code
}
else
{
    //iphone specific code
}

  • This doesn't help the original poster so much, but it's more intended for anybody who is just getting started and is reading this question.

    It can be quite painful to move an entire application from single-platform to cross-platform. If you have any thoughts of doing cross-platform development, do it from the very beginning. Do not let a day go by where you don't test on both platforms.

    I find that for every 2 hours of coding, I encounter at least one platform inconsistency that I will have to deal with. It's hard enough to deal with one of these issues, the pain of dealing with 20 or 30 at once would make me crazy!
