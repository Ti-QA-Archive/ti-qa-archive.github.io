    • You can test all the resolutions for the iPhone and iPad using Titanium, you do not need to use xcode to achieve that.

      • Select your app in the project explorer (left pane)
      • Find the run icon on the toolbar (green circle white play button and yellow folder)
      • Choose "Run configurations…"
      • You can select device and iOS version here

      Assuming you have selected the iPad and want to change the resolution;

      • Run to the iPad in the current resolution
      • (If your simulator is not normally on your dock - right click it and select options > keep in dock)
      • In the simulator find "hardware" menu
      • Then device and change as you need
      • The simulator will close - this is normal (stupid but normal)
      • Run the simulator manually (from the dock)
      • Change the option again (is still stupid but is still normal)
      • The simulator will now be in the new resolution
      • Run your app from Titanium again
      • It will now run your app in the new resolution

      It seems like a lot of steps - but it takes seconds when you are up and running.
