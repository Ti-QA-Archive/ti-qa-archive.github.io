win.close Crashes Android App

My application is based off of this tutorial - http://mobile.tutsplus.com/tutorials/appcelerator/titanium-user-authentication-part-3/

My app only has the basis of that tutorial, and has a lot of added functionality to it.

Here is the code as of right now:

login.js

loginReq.onload = function()
{
    var json = this.responseText;
    var response = JSON.parse(json);
    if (response.logged == true)
    {
        username.blur();
        password.blur();
        Ti.App.fireEvent('grantEntrance', {
            name:response.name,
            acctId:response.acctId
        });

        win.close();
    }
    else if(response.error == true)
    {
        alert('An unknown error has occured. Please try again later.');
    }
    else
    {
        alert(response.message);
    }
};

app.js

Ti.App.addEventListener('grantEntrance', function(event)
{    
    main.tabBarHidden     = true;
      main.title         = 'Welcome ' + event.name;
    main.url         = 'main_windows/main.js';
    main.name         = event.name;
    main.acctId        = event.acctId;
    mainTab.window     = main;
    Ti.App.acctId     = event.acctId;

    tabGroup.addTab(mainTab);
    tabGroup.removeTab(loginTab);
});

The win.close() part of the login causes the app on Android to crash. If I remove it, it never does anything else. What am I doing wrong here?

I'm running SDK 1.5.1 and Android APIs 2.2

Thanks!
