Webview swipe event incase of mobile web

Hi I am new to titanium. I am developing an application which uses webview. I can add an addeventlistener of type swipe for the webview. I can add another webview to the window in the webview swipe event and I can hide the webview, show the webview in the webview swipe listener. But when I bind some html content to the webview I cannot perform the swipe for webview. First I tried using the below code:

var win = Ti.UI.createWindow({
backgroundColor:'white'
});

var web = Ti.UI.createWebView();
var web1 = Ti.UI.createWebView({
backgroundColor:'black'
});

web.addEventListener('swipe',function(){
alert('1.webview is swiped');
win.add(web1);
web1.show();
});

web1.addEventListener('swipe',function(){
alert('2.webview is swiped');
web1.hide();
});

win.add(web);
win.open();

With the above code I am able to swipe the webview. Later I added some html content to the webview instance 'web' as follows:

var win = Ti.UI.createWindow({
backgroundColor:'white'
});

var web = Ti.UI.createWebView({
width:'80%',
height:'80%',
html:'<html><head>title</head><body><p>This is sample webview</p></body></html>',
borderColor:'black',
borderWidth:5

});

var web1 = Ti.UI.createWebView({
backgroundColor:'black'
});

web.addEventListener('swipe',function(){
alert('1.webview is swiped');
win.add(web1);
web1.show();
});

web1.addEventListener('swipe',function(){
alert('2.webview is swiped');
web1.hide();
});

win.add(web);
win.open();

When I execute the above code I am able to get the title and <p></p> content also in webview. But the webview is not recognizing swipe event. Could anyone share a solution ASAP.

Thanks in Advance

  • The WebView object does not support the swipe event.

    But I must ask, in your title you mentioned "Mobile Web". If you are using Titanium Mobile Web, you should stick to simple views instead of WebViews.
