Button inside TableViewRow. How to get row.index

Hi,
I have a button inside a TableViewRow and would like to get the row index when I click on that button.
When I print out "e.source.parent" inside the button-click event I'll end up with "Ti.UI.TableViewRow" which should be right. But when I try to get "e.source.parent.index" it'll always print out "undefined".

I don't want to use a custom property for storing the ID inside the button because when I delete a row I have to update all custom properties. I want the actual row index of the row my button is in

    put the click event on the tableRow and check for source id to determine if the button was clicked or some other object in the row.
