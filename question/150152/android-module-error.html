  • It means go to the command line to work on your modules, not in studio. I'm not sure module package ever actually worked. Additionally, I'm still building modules using 2.1.4GA until the new 3.0.x CLI has been tested for module builds.

    .bash_profile:

    alias titaniumpy="~/Library/Application\ Support/Titanium/mobilesdk/osx/2.1.4.GA/titanium.py
