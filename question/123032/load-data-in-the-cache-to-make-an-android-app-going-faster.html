  • As Ivan and Paul mention, you can use the internal sqlite database or Titanium.App.properties for key value storage. The cool thing with properties is that you can actually persist a complex object or arrays by JSON.stringify them, and using the Ti.App.Properties.setString(&#39;stringName&#39;,object) method.
