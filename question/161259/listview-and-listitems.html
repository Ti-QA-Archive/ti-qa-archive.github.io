    • Joe, it depends on whether you are using Alloy or not. If not, then you could just do something like:

      newWindow.js

      exports.iWindow = function(itemDetails) {
    var win = Ti.UI.createWindow({
        backgroundColor : '#fff',
        layout: 'vertical'
    });

    //do something with the item you passed in
  return win;
}

      then in your eventlistener, something like:

      listView.addEventListener('itemclick', function(e){
        var item = e.section.getItemAt(e.itemIndex);
    var nw = require('newWindow').iWindow(item);;
    nw.open();
    });

      Make Sense?

      Ray
