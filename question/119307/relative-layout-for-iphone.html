    If you use the default layout format, then you will need to specify the top & left / top & right / bottom & left / bottom & right X and Y coordinates to "pin" the element to the View in that relative location.

    Otherwise, if you use the layout: horizontal format, then you wil need to specify the width & height attributes, and any top & left properties will be applied relative to its previous sibling element – think "float: left" in CSS terms. If you want to do the equivalent of "clear: both" in CSS terms, then just set the width of an element to occupy the remaining space in the row – the next element wil naturally wrap underneath.

    Hopefully this explanation proves helpful :)
