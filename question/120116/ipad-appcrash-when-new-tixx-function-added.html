Ipad AppCrash when new Ti.xx function added

I always need to:

I already try to delete the "build\iphone\build folder", but this is not working :(

Is there a way to do a full rebuild without recreating a new fresh project ?

  • Yoann

    Note that it's not the "build\iphone\build folder" but, rather, it's everything inside the "build\iphone\" folder. Hence, the correct procedure is as follows:

    For iOS/iphone, delete the contents of the project's build&#x2F;iphone directory or, for Android, delete the contents of the project's build&#x2F;iphone directory. Go to the Edit Project screen in Titanium Developer and click the Save Changes to touch the tiapp.xml file, which forces a full rebuild. Then launch the app.

    To understand the reason why you receive the error, and why it isn't such a bad thing, see How does Titanium work?

    Hope this helps
