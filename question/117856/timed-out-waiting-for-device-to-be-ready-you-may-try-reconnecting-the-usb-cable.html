Timed out waiting for device to be ready, you may try reconnecting the USB cable

Hello-

I am trying to deploy my android application on to my samsung acclaim and keep getting the error "Timed out waiting for device to be ready, you may try reconnecting the USB cable". I have USB debugging set to on. I also have Unknown sources set to on to allow installations from non-Market applications. I still get this error. Oddly enough, this error only occurs on and off. I started out the night being able to install the app without any issues. After about 2 hours, I started getting this error.

When getting this error, I powered off my phone and turned it back on. This did not help. I tried restarting my pc, this did not help either. When I look in windows explorer, I do not have a usb device for my phone. Is there a setting in windows that I need to turn on? Does anyone have any thoughts other than turn on USB debugging or Unknown Sources?

Thanks

Josh

  • had same problem with Samsung Prevail on W7; checked that USB debugging & unknown source were checked but still got problem; then i checked the W7 devices and found that the Samsung USB driver was missing; did a troubleshoot and W7 found and installed driver which enabled me to install app
