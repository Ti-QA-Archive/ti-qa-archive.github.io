  • Hi Victor, as suggested HERE in Titanium docs, you cannot stop zooming after setting the page to fit the screen.

    However you can modified the css file of the webpage to fit the content of webpage according to the mobile/tablets/dektops.

    Using webview in Titanium can only display the whole content fit on the screen for once, but user will be able to zoom-in out afterwards, & i think this should be work like this because if a page has a very wide content to display & titanium fits it to the screen, then obviously user will need to zoom-in to view the content more clearly (not the small tiny fonts after fitting the content to screen).

    Hope this helps you :)
