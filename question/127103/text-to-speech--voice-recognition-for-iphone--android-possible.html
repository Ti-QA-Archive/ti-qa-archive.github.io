text-to-speech & voice recognition for iPhone & Android possible?

I have the assignment to make a Native Android and iPhone app with a text-to-speech and voice recognition feature (in Japanese!). I'm now looking around with what I can make such an app for both platforms with just one code.

Is it possible to have text-to-speech & voice recognition features using Appcelerator Titanium? If yes is there a link with information about it?

Thanks a lot for any helpful information about this.

  • There is no built in support. In the past I found an open source iOS library that needs to be imported as a Titanium module - my post about it - if you do write it - I would be happy to hear about it.

    There are API on the web that you can send sounds to and their servers return the words as text (of course the charge for it - google it). you can write an app that records voice and sends it to such API (btw, that what Apples Siri does).
