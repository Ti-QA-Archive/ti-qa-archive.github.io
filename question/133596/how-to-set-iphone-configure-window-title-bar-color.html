How to set iPhone Configure Window Title Bar Color

I'm currently developing an iPhone app with the current version of the Titanium SDK (1.8.2).

I have a TabGroup interface in which all title bar windows are set to green, including the "More"-window. When I now click on the "Edit"-button in the "More"-window the title bar of the "Configure"-window is in the default color. How do I set the color of this title bar and is there a way to globally set the title bar color for all windows insted setting it when each window is created?

  • You set the barColor property. This has the details if you need them;

    …/Titanium.UI.TabGroup.barColor-property.html

    And example code if you need it.

    barGroup.barColor = 'color';

    or

    var tabGroup = Titanium.UI.createTabGroup({
...
   barColor: 'color',
...
})
