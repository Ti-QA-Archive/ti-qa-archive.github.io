Database in mobile web application

Hi, I'm trying to use my current iOS app and make it available as mobile web app. One of the main problem I faced is that my app uses data retrived from database. I know that mobile web doesn't support Ti.Database. My question is that how can I make it so that my mobile web app can read data from a database?

Any example would be helpful.

I'm using latest SDK.

  • Hi Chung,

    You can make an Ajax call using Titanium.Network.createHTTPClient();
    This is an example: 

    var ajax = Ti.Network.createHTTPClient();
ajax.onerror = function(e){
    alert('Error');
};
ajax.onload = function(){
    Titanium.API.info(this.responseText);
    var data = this.responseText; //JSON data from .php file
};
ajax.open('POST', 'url');
ajax.send({
    'var': 'Test Value'
});

    The url would be the route to a .php file which make the database access and it returns your data in JSON.
