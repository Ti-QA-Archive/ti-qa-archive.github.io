    Hi

    This is a common requirement that does not initially have an obvious answer - but it is actually simple when you know how.

    You will need ONE image and that will form the back of a parent view that you place your content inside. By using backgroundLeftCap, backgroundRightCap, backgroundTopCap & backgroundBottomCap to specify how the image can be placed.

    This is a large version of a speech bubble;

    This is the image after repeating sections have been removed

    Image showing cap positions

    Once you have identified the cap positions set those in a view with the reduced image as the source.

    var bubble1 = Ti.UI.createView({
    backgroundImage: 'bubble.png',
    backgroundLeftCap: 43,
    backgroundRightCap: 34,
    backgroundTopCap: 34,
    backgroundBottomCap: 36,
    height: 200,
    top: 10,
    width: 200
});
win.add(bubble1);

var bubble2 = Ti.UI.createView({
    backgroundImage: 'bubble.png',
    backgroundLeftCap: 43,
    backgroundRightCap: 34,
    backgroundTopCap: 34,
    backgroundBottomCap: 36,
    height: 75,
    bottom: 10,
    width: 100
});
win.add(bubble2);

    This will look like this;

    Two different views each with different width and height, but the backgroundImage stretches based on the cap sizes, so it never looks distorted.

    Note; your width & height should never be less than the reduced image width & height or it will not look right.
