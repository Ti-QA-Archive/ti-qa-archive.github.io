Popover on Alloy, not displaying correctly the first time I show

Hey folks. I'm trying to display a popover on button tap. The issue I'm running into is that when I first try showing the popover the view inside of my popover flashes and goes away. When I close and reopen the popover the view inside of my popover shows up just fine. It's just that first time showing the popover that I am running into this issue.

Attached is the code I'm using to create my popover …

In the xml

<Alloy>
    <Popover id="popover" ns="Ti.UI.iPad" width="320" height="478" title="Info" >
        <Require type="view" src="aboutPopup" id="aboutPopup"/>
    </Popover>
    ...
    <Button id="aboutBtn" onClick="togglePopover" backgroundImage="/images/Topbar_info.png"  />
    ...

in the js

function togglePopover() {
    $.popover.show({ view: $.aboutBtn});
}

aboutPopup.xml

<Alloy>
    <View class="container">
        <ImageView id="topBG" image="/images/BG_aboutTop.png" />
        <TabbedBar id="tabbedBar" platform="ios">
            <Labels>
                <Label>About</Label>
                <Label>Version</Label>
            </Labels>
        </TabbedBar>
        <View id="about" class="content" >
            <Label id="aboutLabel">About</Label>
            <TextArea id="aboutText" />
        </View>
        <View id="version" class="content" >
            <Label id="versionLabel">2.0</Label>
        </View>
    </View>
</Alloy>

Like I say, first time I click the button and show the popover the view inside the popover flashes and goes away, next time you open it … works perfect. Any help on this one would be appreciated, still very new to alloy so I'm stumbling through it.
