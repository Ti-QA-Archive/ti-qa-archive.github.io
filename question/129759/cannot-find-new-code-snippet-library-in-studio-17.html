Cannot find new Code Snippet Library in Studio 1.7

I updated to 1.0.7.201112152014 and the developer blog tells me that there should be a new Code Snippet Library in the "Commands" menu.

Unfortunately, there is no such thing in my command menu. I have items like "Bundle development", "Capistrano", "JavaScript" and so on - but no "Titanium Mobile".

Where do I find the new Code Snippet Library?

    Ingo states: 'We just noticed that some users are missing the Commands > Titanium Mobile menu item. We have a fix in the works and should be out within a few days. As a easy workaround, do Commands > Bundle Development > Install Bundle, and select Titanium Mobile from there.'

    Hope this helps Julian.
