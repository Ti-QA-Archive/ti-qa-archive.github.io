Help loading external HTML on a webView (Android)

Hello,

I'm creating a webView on Android because I need to load the HTML contents of an email into my app, but I'm getting a "Web page not loaded" screen with what seems to be the content I'm trying to load but url encoded.

![Here's a screenshot]{http://i51.tinypic.com/28cegya.png}

A little context to help explain:

On the image is loaded a window with the webView, as follows

var my_view = Ti.UI.createView({
  backgroundColor: 'white'
});

var html_content = requestEmail(); //A function that connects to my server and retrieves the email

var my_webview = Ti.UI.createWebView({
  html: html_content,
  width: Ti.UI.FILL,
  scalesPageToFit: true
});

my_view.add(my_webview);

Initially I had the webView loaded on a row of a table, because I need to show other stuff on the screen. But I read something about errors when doing that so I decided to test just the webView without the table, to see if that fixed it but I'm still stuck with the same error.

The code worked (even with the table and other stuff) on iOS, but I haven't been able to figure things out on Android. Please help!

Thanks,

    • Found the solution, though it is weird;

      var my_webview = Ti.UI.createWebView({
  data: html_content, // this line has changed
  width: Ti.UI.FILL,
  scalesPageToFit: true
});

      Note the use of data and not html, Android appears to make use of html coming in as a blob and not plain html. This works correctly for me.

  • I was having the same issue which was being caused by % symbols in the HTML I was loading.

    As per:
    https://code.google.com/p/android/issues/detail?id=4401

    I modified the HTML I was loading

    ie. 

    ratios_html = ratios_html.replace(/\%/g, "&amp;#37;");

    and it seemed to resolve the issue for me.
