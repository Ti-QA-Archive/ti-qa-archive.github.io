How to update the data in a TableView in Alloy?

How can I update the data in a TableView in Alloy?

I've defined my table like this

<TableView id="playersTable" dataCollection="orderedPlayers">

I then have a button that opens a page where filters to the data can be applied. I'm trying to update the data in the close even of the filter page

$.filterWin.addEventListener('close', function(e) {
    // Apply filter

    var result = Alloy.Collections.orderedPlayers.filter(function(p) {
            return p.get("Type") === 3;
    });

    // Update table - handle set to $.filterWin.targetTable
});

How can I update the table in Alloy? I haven't used setData for the initial load so I would assume there is some way to update the table without manually creating each row.

    // force table to update
orderedPlayers.trigger("sync");
