ListItem with custom height doesn't render backgroundGradient correctly in iOS

Hey,

I am trying to implement a gradientBackground through TSS to a ListItem that has a custom height set, but when rendered, the backgroundGradient gets the height of the default height of a ListItem in iOS.

Not sure if this has anything to do with it being bound to a data collection or not…

SDK/Studio

Titanium SDK version 3.4.0.GA
Titanium Studio, build: 3.4.0.201409261227

items.xml:

<Alloy>
  <Collection src="item"/>
    <View class="container">
      <ListView id="listView" defaultItemTemplate="standard">
        <Templates>
        <ItemTemplate name="standard" class="standard">
          <Label bindId="company_name" id="standardCompanyName"/>
        </ItemTemplate>
          <ItemTemplate name="featured" class="featured">
          <Label bindId="company_name" id="featuredCompanyName"/>
        </ItemTemplate>
        </Templates>
        <ListSection dataCollection="item" dataFilter="featuredFilter" dataTransform="featuredTransform">
          <ListItem 
            template="featured" 
            company_name:text="{company_name}" />
      </ListSection>
      <ListSection dataCollection="item" dataFilter="normalFilter">
        <ListItem 
          template="standard" 
          company_name:text="{company_name}"/>
      </ListSection>
        <PullView>
          <View backgroundColor="rgba(0,0,0,0.3)" height="50" layout="horizontal">
            <Label id="pullImage">&#xf078;</Label>
          </View>
        </PullView>
      </ListView>
    </View>
</Alloy>

items.tss:

"#listView": {
  backgroundColor: "transparent",
},
"#listView[platform=ios]": {
  separatorStyle: Titanium.UI.iPhone.ListViewSeparatorStyle.NONE,
},
".standard": {
  backgroundGradient: {
    type: "linear",
    colors: ["#d2d2d2", "#eeeeee"],
    startPoint: {x: 0, y: 0},
    endPoint: {x: 0, y: "50dp"},
    backFillStart: true,
  }  
},
".featured": {
  height: "100dp",
  backgroundGradient: {
    type: "linear",
    colors: ["#fcee89", "#ffffff"],
    startPoint: {x: 0, y: 0},
    endPoint: {x: 0, y: "100dp"},
    backFillStart: true,
    backFillEnd: true,
  }  
},

I have tried searching for this, but have come up empty. Maybe I am using the wrong terminology in my searches? Either way, maybe one of you awesome peeps can help out?

Warm Regards,
Torgny

    Might be a bug in Titanium. You could try to wrap the Label in a View and set to that view a backgroundGradient.

     <ItemTemplate name="standard">
    <View class="standard">
         <Label bindId="company_name" id="standardCompanyName"/>
    </View>
 </ItemTemplate>
