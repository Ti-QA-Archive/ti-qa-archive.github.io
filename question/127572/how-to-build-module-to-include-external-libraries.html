  • During building of a module, any dependencies that the module has (for example, a framework i.e CoreText etc.) must be defined somewhere so that the build tools know to add those frameworks to the commands that it sends to XCode. This needs to be defined in module.xconfig file by setting the OTHER_LD property

    For example:

    OTHER_LDFLAGS=$(inherited) -framework QuartzCore -framework UIKit

    This file is pulled in during application build time to add any additional options to the link command for XCode.
