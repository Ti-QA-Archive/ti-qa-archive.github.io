TableView: background color for row's footer

Hi,
it's possible to set background color for a simple footer (see: "Close section one" footer).
Below a simple example code:

var tbl_data = [{title:'Row 1', header: 'Section one'}, {title:'Row 2'}, {title:'Row 3', footer:'Close section one'}, {title:'Row 4', header: 'Section two'}, {title:'Row 5'}, {title:'Row 6'}];
var tabelView = Titanium.UI.createTableView({
backgroundColor : 'trasparent',
//data: tableData
headerTitle : 'Resources list',
data : tbl_data
});

Thank you.

Lorenzo

    Hi

    Yes, but not when you are using the quick data rows method you have shown. The footer element of the row only allows for a title and that title format is pre-defined.

    You need to add the rows as part of sections, with those section having the header and footer elements.

    • Check out the documentation for tableViewSection
    • See the detailed example in the documentation for tableView to see how this can be done.
