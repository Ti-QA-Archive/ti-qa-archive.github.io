    I prefer Ti.Classic, here is the solution:

    var plainTemplate = {
    ...
    events: {
        click: tableClick
    }
    ...
};

data.push({
    ...
    button: { backgroundColor: '#aaa' },
    ...
    properties: {
        ...
    }
});

function tableClick(e) {
    var item = e.section.getItemAt(e.itemIndex);
    item.button.backgroundColor = 'rgb(255,0,255)';
    e.section.updateItemAt(e.itemIndex, item);
}

    BTW: on Android you don't need to extra define the click event on a button. Use the listview eventlistener "itemclick" to call the tableClick(e) function if e.bindId match.

  • You should be using Alloy to build your apps. Frankly, I never did wrap my head around how to do ListViews in just javascript. Alloy makes it supremely easier.

    MyController.xml:

            <ListView id="myList" defaultItemTemplate="scheduleTemplate">
            <Templates>
                <ItemTemplate name="scheduleTemplate" class="row">
                    <View class="row">
                        <Label bindId="scheduleTitle" class="scheduleTitle" />
                        <Label bindId="scheduleTimes" class="scheduleTimes" />
                        <Button bindId="scheduleButton" class="scheduleOn" onClick="buttonClick" />
                    </View>
                </ItemTemplate>
            </Templates>
            <ListSection id="scheduleSection"></ListSection>
        </ListView>

    MyController.js:

    function buttonClick(e){
    e.section.items[e.itemIndex].scheduleButton.backgroundColor="#f00";
}
