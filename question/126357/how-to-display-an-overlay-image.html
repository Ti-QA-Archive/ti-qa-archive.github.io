How to display an overlay image

Hi there,

I'm trying to display an overlay image (i.e. the word "Paid" or "Overdue") on the transaction details window.

When I use a background image on view, the transaction details table will be on top of the word "Paid".

I need to interact with the transaction so displaying another view on top of the current view won't work either.

Trying to play with the z-index but it didn't work either.

Any helps will be appreciated. Thanks!
