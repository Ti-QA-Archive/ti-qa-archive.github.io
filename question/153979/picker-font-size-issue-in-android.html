  • fontSize works only for iOS in PickerRows. You have to create Labels that you add to the PickerRows. You should also use density-independent units in android: fontSize: &#39;16sp&#39; for fonts and width: &#39;320dp&#39; for your Picker (or all other views). Read more about that here
