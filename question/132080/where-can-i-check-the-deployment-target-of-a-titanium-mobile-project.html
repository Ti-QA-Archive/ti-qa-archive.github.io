Where can I check the "deployment target" of a Titanium Mobile project?

Hello,

I'm trying to see where are the minimum iOS version that I newly created Titanium project is. In Xcode this is called Deployment Target. Where can I find it? And for Android?

  • For Android, you have to set it based on your own testing. You do this in tiapp.xml by looking at hte raw xml content of the file

    <android xmlns:android="http://schemas.android.com/apk/res/android">
        <manifest>
            <uses-sdk android:minSdkVersion="10"/>
        </manifest>
    </android>

    In this example the minimum version is Android 2.3.3. For iOS, this depends on the version of xcode you are running which currently is 4.3.x.
