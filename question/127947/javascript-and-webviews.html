Javascript and Webviews

I'm working on an app (Android-only) in Titanium that consists of two tabs, each with a webview in it. The tabs themselves are fine, and the webviews are fine as long as I don't build for distribution.

My webviews use local HTML, CSS and Javascript, each with its own folder. When I build to distribute, it signs and compiles just fine. But, Titanium tries to compile the Javascript files in the webview folders just like normal "native" Javascript, making it useless to the webviews. Is there a way to prevent Titanium Studio from compiling these files so that my webviews can still work?
