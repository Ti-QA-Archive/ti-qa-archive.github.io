Problem with youtube in a web view

The following code should open a youtube web page in a web view, which it does, but the play button on the video does not work.

If I open the same web page using an intent then the browser allows the video to be played as expected.

This is an Android project and I have tested using Google 2.2 API and Titanium SDKs 1.7.6, 1.8.1, 1.8.2, 2.0.1GA2 and 2.0.2GA all with the same results.

Any ideas anybody?

Cheers,
Richard

var win = Ti.UI.createWindow({
    backgroundColor : '#ffffff',
    fullscreen : true,
    navBarHidden : true,
    exitOnClose : true,
});

var url="http://www.youtube.com/watch?v=o2C8wJwCeLc";

var webView = Ti.UI.createWebView({
    url : url,
    scalesPageToFit : true,
    touchEnabled : true,
    top : 0,
    backgroundColor : '#FFF'
});
