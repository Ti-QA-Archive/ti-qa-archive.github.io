CloudPush.addEventListener('callback', function(evt) {
        if (evt.payload != null) {

<property name="acs-grouped-notification-message-" type="string">You have $number$ unread notifications!</property> in tiapp.xml

  • Two things:

    Your example has a trailing "-" in the property name

    <property name="acs-grouped-notification-message-" type="string">

    should be

    <property name="acs-grouped-notification-message" type="string">

    Also, make sure you are not doing this in your code:

    CloudPush.setSingleCallback(true);

    I believe that if you set single callback, it overrides the grouped notification message property, and you'll get individual messages again.
