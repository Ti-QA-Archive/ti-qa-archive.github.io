  • Hi Ahmed,

    There are many ways to pass data between controllers(javascript pages).

    1) You can pass values during calling time of the new file

    var newWin = Ti.UI.createWindow({
        backgroundColor : 'white',
        barColor : 'black',
        navBarHidden : false,
        url : 'ui/childrenList.js',
        folderId : parentId
    });
    newWin.parentId = parentId;

    newWin.open({
        modal : true
    });

    In above example, we have passed 2 values,

    i)folderId

    ii)parentId

    You can access this values in childrenList.js using following code.

    var self = Ti.UI.currentWindow;
var id = self.parentId;

    2) You can use set methods avaiable in Properties file.

    Ti.App.Properties.setObject('prodInfo',prodInfo);

    and at the receiving end you can use following code.

    Ti.App.Properties.getObject('prodInfo');

  • Hi Ahmed.
    You can work with the local data sources,check the module:

    Ti.App.Properties

    Example whit a simple string:
    app.js

    (function (e){
    var tabGroup = Ti.UI.createTabGroup(),
    //REQUIRE
    win1 = require('ui/win1'),
    win2 = require('ui/win2'),
        tab1 = Ti.UI.createTab({
        title:'win1',
        window:new win1()
    }),
        tab2 = Ti.UI.createTab({
        title:'win2',
        window: new win2()
    });
    tabGroup.addTab(tab1);
    tabGroup.addTab(tab2);
    tabGroup.open();
})();

    Win1.js (on folder UI)

    function Win1(){
    var self = Ti.UI.createWindow({
        backgroundColor:'white',
        title:'win1'
    });
    //create the data Properties
    self.addEventListener('focus',function(e){
        Ti.App.Properties.setString('test', 'Mydata');
        Ti.API.info('Data test created');
    });


    return self;
}
module.exports = Win1;

    Win2.js (on folder UI)

    function Win2(){
    var self = Ti.UI.createWindow({
        backgroundColor:'white',
        title:'win2'
    });
    self.addEventListener('focus',function(e){
        alert(Ti.App.Properties.getString('test'));
    });
    return self;
}
module.exports = Win2;
