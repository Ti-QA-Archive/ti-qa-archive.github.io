Adjusting automatically the height of a table inside another table

Hi, this is a complex problen I think

I have a tableView, and in one row I need to show other table.

I need that the the row that has insed the other table, automatically adjust its height to display all the rows in the table without having to make a scroll.

The code for this sample is

    var win = Ti.UI.createWindow({
        backgroundColor : 'yellow'
    });
    win.open();


    //Create a tableview with one row and a view insed the row
    var maintbl = Ti.UI.createTableView();
    win.add(maintbl);

    var maintblrow=Ti.UI.createTableViewRow({
        height:Ti.UI.SIZE
    });

    var maintblrowview = Ti.UI.createView({
        backgroundColor : 'green',
        layout : 'vertical',
        height:Ti.UI.SIZE // <<< how can grow automatically ?¿
    });
    maintblrow.add(maintblrowview);
    maintbl.setData([maintblrow]);


    //Create inside the row, another table
    var secondtbl = Ti.UI.createTableView({
        data:[ {title: 'Apples'}, {title: 'Bananas'}, {title: 'Carrots'}, {title: 'Potatoes'} ],
        height:Ti.UI.SIZE        
    });
    maintblrowview.add(secondtbl);

and the wrong result is

I need that with this code, maintblrowview automatically adjust its height to display all fruits.

Thanks!!!

Android App
3.1.0 SDK
