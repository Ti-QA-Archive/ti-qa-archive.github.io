Center content in scrollview

I have an ImageView inside a ScrollView. I want to be able to pinch zoom the scrollview to a size, where the ImageView is much smaller than the ScrollView's viewable area (for example halve size of the screen where the ScrollView takes the full). This works fine, except one thing: my content (the ImageView) attaches to the top left edge, and I want it to be in the center of the ScrollView. Is it possible somehow?

— asked June 18th 2012 by Kristof Gruber

  • Here is my (temporary) solution for it:

    scrollView is the ScrollView, and moneyNormalView is the ImageView.

    
        scrollView.addEventListener('scale', function(e) {
            zoomScale = e.scale;

            var moneyWidth = moneyNormalView.width*zoomScale;
            var moneyHeight = moneyNormalView.height*zoomScale;
            if (moneyWidth < Ti.Platform.displayCaps.platformWidth) {
                moneyNormalView.animate({
                    left: (Ti.Platform.displayCaps.platformWidth/zoomScale - moneyNormalView.width) / 2,
                    duration: 400,
                });
            } else {
                moneyNormalView.animate({
                    left: 0,
                    duration: 400,
                });
            }
            if (moneyHeight < Ti.Platform.displayCaps.platformHeight) {
                moneyNormalView.animate({
                    top: (Ti.Platform.displayCaps.platformHeight/zoomScale - moneyNormalView.height) / 2,
                    duration: 400,
                });
            } else {
                moneyNormalView.animate({
                    top: 0,
                    duration: 400,
                });
            }

        });
