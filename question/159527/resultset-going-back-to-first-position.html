ResultSet going back to first position

After looping through the whole resultset, is it possible to go back to the first position?

Will this work?

var originalResultSet = db.execute(.....);

while (randomWhileHere)
{
    var usableSet = originalResultSet;

    while (usableSet.isValidRow())
