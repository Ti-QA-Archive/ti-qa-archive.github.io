createButton: textAlign:right causes vertical alignment of title to be off

I'm seeing the following behavior on iOS only (works fine on Android):

if I add the option textAlign: &#39;right&#39; to my createButton call, then the button text is shifted up by a few pixels instead of being vertically centered. Here's the code:

var saveButton = Ti.UI.createButton({
    title: "Save", style: Ti.UI.iPhone.SystemButtonStyle.PLAIN,
    text: "Save", textAlign: 'right',
    color: 'white', backgroundColor: '#ccc', width: '60dp', 
    height: UI.topbar_height_dp, top:0,
    font: { fontSize:'20dp' },
    right: 20
});

To reproduce the issue try the above call with and without the textAlign parameter - you'll notice the vertical alignment of the button text differs when textAlign:right is present.

There's a verticalAlign parameter available, but only for Android.

How do I right align the text in a button without getting the vertical align shift?

My overall goal is to have the button line up with the other parts of the UI, so the other alternative might be to size the button to be exactly the size of the button text (currently there's padding between the text and the button edges).
