  • Here is a temporary workaround that I'm using. I set the "modal" setting to true in the window has the webview. 

    var webview = Ti.UI.createWebView({url:'insert your url here'});
var window = Ti.UI.createWindow();
window.add(webview);
window.open({modal:true});

    The password dialog pop up in front of the webview rather then a few window behind it.
