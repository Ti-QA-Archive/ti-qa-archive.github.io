Modal dialog (yes/no) without callback functions

Is there any way to build a modal dialog (yes / no) without using callback functions, as the following example?

Android App
Titanium Studio, build: 3.1.1

var response = MyYesNoModalDialogQuestion('¿Sure?');

if (response == <yes>) 
    <something>
else
    <something>;

  • Accepted Answer

    var response = MyYesNoModalDialogQuestion('¿Sure?',
     function(){ // yes
        // do something
    }, 
    function(){ // no
        // do something
    });

    This would be a callback based solution - what's wrong with that ?
