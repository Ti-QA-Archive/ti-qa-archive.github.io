Check if var equals true or false not working

Hi i just need to know why my code always runs regardless of whether the variable equals true or false.


var admob_enabled = false;

if (admob_enabled = !false){
    var Admob = require('ti.admob');
}

Thanks

  • Accepted Answer

    if (admob_enabled = !false){

    should be

    if (admob_enabled == !false){

    first is an assignment, second is a conditional evaluation.
