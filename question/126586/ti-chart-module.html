TI+ Chart Module

Is there a way to change the background color of the chart? I tried to use backgroundColor:'black' but it still displays a gradient grey background.

Thanks.

  • The background of the chart is controlled by the plotArea group of properties.

    plotArea: {
    backgroundColor:'#000',
}
