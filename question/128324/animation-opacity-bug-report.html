animation opacity Bug report

Testing my app and after a lot of tests I solved a little but annoying problem that I have not relieved before.

new_view is a common view with opacity:0

With blank space between opacity: and it's value 1, opacity animation effect doesn't work.

new_view.animate({opacity: 1,duration:1800});

removing blank space it work correctly.

new_view.animate({opacity:1,duration:1800});

Anyone experienced this problem?
Thank you, Stefano.
