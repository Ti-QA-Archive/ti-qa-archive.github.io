Android SDK not found

I'm new to Titanium. When I try to create a new Mobile Project, on the Preferences Form, I browse for the Android SDK directory, I get the error: "Could not locate the Android SDK at the given path".

I installed latest Titanium Studio.(Titanium Studio, build: 1.0.4.201108101535).
I have installed the sdk at c:\Program Files\Android\android-sdk
I have download the latest versions of the Android SDK.
I have developed a working app using Eclipse and the SDK, therefore the SDK must be OK.
I have tried copying the adb.exe file and a dll into the ./tools directory. This doesnt work.
I am using Windows XP.

    Two things:

    1. Place the SDK in a folder that doesn't have spaces in it.

    2. At one point in time, Titanium Studio did its install check based upon an older version of the sdk. So download the older SDK version from inside of android.exe (It was looking for the android-7 folder inside of platforms)
