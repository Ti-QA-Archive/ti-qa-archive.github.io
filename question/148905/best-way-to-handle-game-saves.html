Best way to handle game saves

I'm interested in developing a simple game in Titanium, and just wondering if anyone has an opinion on the best methods for saving game data? We're talking the usual, simple variables for results of levels, a star rating etc, total points, etc.

I am thinking that JSON stored in a simple text file should do the trick, but are there any better ways? Has anyone handled something like this before?

  • Simple to persist the JSON to a Ti.App.Properties object per user. Another way would be to store in a Database. If you are playing across the net then I would setup a web server with a REST interface and post saves there.
