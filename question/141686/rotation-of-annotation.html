Rotation of annotation

I try to rotate annotations:

This doesn't work:

var a = Ti.Map.createAnnotation({
    latitude : v.lat,
    longitude : v.lon,
    image : '/assets/vessels/yellow1_90.png', 
    title : v.name + ' ' + v.country,
    subtitle : v.typeofship
    });
a.transform = Ti.UI.create2DMatrix().rotate(v.orientation);

and this also:

var a = Ti.Map.createAnnotation({
    latitude : v.lat,
    transform : Ti.UI.create2DMatrix().rotate(v.orientation);
    longitude : v.lon,
    image : '/assets/vessels/yellow1_90.png', 
    title : v.name + ' ' + v.country,
    subtitle : v.typeofship
});

In all cases in v.orientaion are values between 0 … 180

Any ideas?
