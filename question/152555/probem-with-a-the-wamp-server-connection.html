    • 10.0.2.2 is an IP address within your local network. For this to work, you should connect using the public IP address of your server. You can use a service like http://whatismyipaddress.com/ to find it.

      Remember that you should make sure that your server has a static IP address (that doesn't change whenever the server is reset), otherwise your app will break eventually.
