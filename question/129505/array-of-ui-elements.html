Array of UI elements?

Hello,

Is it possible to create a dynamically generated array of UI elements?

For example, I'd like to download a list of string and attach a Titanium.UI.Switch to each string.

I tried something like that but it didn't work:

var switchArray = [];
var switchArray[0] = Titanium.UI.createSwitch({
    value:false
});
var switchArray[1] = Titanium.UI.createSwitch({
    value:false
});

Thanks,
Idan

    I think the issue in your code is that once you define the array, you just access the elements like switchArray[0].... No need to use var.

    But, to answer your question, yes you can use arrays to hold UI elements, and do stuff with them like normal:

    var win = Ti.UI.createWindow({
    backgroundColor:'#fff'
});

var switchArray = [];

for(var i = 0; i < 5; i++){
    switchArray.push(Titanium.UI.createSwitch({
        value:false,
        top:35*i
    }));
}

for(j in switchArray){
    win.add(switchArray[j]);
}

switchArray[5] = Titanium.UI.createSwitch({
    value:true,
    top:200
});

win.add(switchArray[5]);

win.open();
