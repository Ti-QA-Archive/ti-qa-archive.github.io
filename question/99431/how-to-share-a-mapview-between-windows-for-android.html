How to share a mapView between windows, for Android

Hello, I'm developing an application for Android and I want to have 2 pages that uses a MapView for different purposes.
Researching this site I've found that on Android you can only use one instance of mapView on the application.

So I would need to know what is the usual way of "sharing" a mapView among different windows on the same application, so that I can for example show the user's location on 2 or more pages on my application.

Thanks in advance!

  • Either passing data…
    Titanium.UI.Window-object , scroll down to passing data section

    or just add it to both windows

    var mapview = Titanium.Map.createView({
    mapType: Titanium.Map.STANDARD_TYPE,
    animate:true,
    regionFit:true,
    annotations:[my_location]
    });

    mapview.selectAnnotation(my_location);

    win1.add(mapview);
    win2.add(mapview);
