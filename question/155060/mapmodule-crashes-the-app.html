MapModule crashes the app

I am using Titanium 3.1.1 on Windows 7 to develop Android app. I wanted to use GoogleMaps so I used MapView. Everything was working fine until distributing it to Play Store. I found this reference Google Maps v2 for Android and followed all the steps but if I initialize the map in window it crashes the app (Android process com.appname has stopped unexpecticly). I don't know how to solve it.

var MapModule = require('ti.map');
var win = Titanium.UI.createWindow();

var mapview = MapModule.createView({
    mapType: MapModule.NORMAL_TYPE,
    region: {latitude:33.74511, longitude:-84.38993,
            latitudeDelta:0.01, longitudeDelta:0.01},
    animate:true,
    regionFit:true,
    userLocation:true,
});


//line below crashes the app
win.add(mapview);

    • You must have configured your need for the map module in your tiapp.xml, in either the file directly or using studio the visual editor for the file.
