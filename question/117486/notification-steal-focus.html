Hello,

I got some problem with the first app I am creating with Titanium.

When I create a notification, it steal focus if from the app if I am typing something in a <input type="text"> element.
What could I do in order to fix this? Or should I just do it the easy way and disable notifications while the window is active?
Code that I use to show the notification:

n.setMessage(msg);
n.setTimeout(5000);
n.setTitle("Oseveno Chat");
n.show();
}

  • Did you try getting the focus back to your field using focus()?

    
function notify(msg){
var n = Titanium.Notification.createNotification(window);
n.setMessage(msg);
n.setTimeout(5000);
n.setTitle("Oseveno Chat");
n.show();
n.blur();
my_text_input.focus();
}
