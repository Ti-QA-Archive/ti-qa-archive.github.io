Unobtrusive text input with a tableview

So I have a tableview that is a standard list of items with a navbar at the top. Normally on the iPhone when you press the + button to add an item it takes you to another dialog where you input all the information about the item and then press 'Add' or 'Save'. In this case though, it is simply a single text input for a list.

I'm wondering if anyone has some less obtrusive way to do this simple input. I was thinking about doing a view that animates down from right below the navbar that has a text field with an 'Add' button next to it. Kind of like the options dialog with buttons, but from the top because of the keyboard. Has anyone seen any apps that do this that I can reference for visual style?

If someone has other suggestions, I'd love to hear them too.

    • Dan,

      I used your suggestion. Thank you! My code looks like this:

              var slide_in = Ti.UI.createAnimation({top:0});
        var slide_out = Ti.UI.createAnimation({top:-40});

        singleInputView = Ti.UI.createView({
            height:40,
            top:-40
        });

        opaqueView = Ti.UI.createView({
            height: 480,
            top:0,
            backgroundColor:'#000',
            opacity:0.5
        });

        duhLabel = Ti.UI.createLabel({
            text:'Tap here to cancel',
            top: 120,
            height:'auto',
            textAlign:'center',
            color:'#ff0000',
            opacity:1.0,
            font:{fontSize:22, fontWeight:'bold'}
        });
        opaqueView.add(duhLabel);

        var newPiece = Ti.UI.createTextField({
            hintText:'Piece or Composition',
            height:32,
            width:240,
            top:10,
            left:10,
            borderStyle:Ti.UI.INPUT_BORDERSTYLE_ROUNDED,
            font:{fontSize:12}
        });
        newPiece.addEventListener('customFocus', function(){
            newPiece.focus();
        });

        var okButton = Ti.UI.createButton({
            title:'Add',
            style:Ti.UI.iPhone.SystemButtonStyle.DONE,
            left: 270,
            top:10
        });

        var inputToolbar = Ti.UI.createToolbar({
            top:0,
            barColor: 'black',
            items: [newPiece,okButton]
        });

        singleInputView.add(inputToolbar);
        var finishAdd = function(){
            newPiece.enabled = false;
            newPiece.enabled = true;
            cancel.enabled = true;
            add.enabled = true;
            opaqueView.hide();
            singleInputView.animate(slide_out);
        };
        okButton.addEventListener('click', finishAdd);
        opaqueView.addEventListener('click', finishAdd);

        dlg.add(opaqueView);
        dlg.add(singleInputView);    
        singleInputView.animate(slide_in);

      Two things of note here, first the customFocus event…trying to find a way to give focus to newPiece, but no matter what I try (settimeout with focus, fireEvent of custom event, etc) it never works. Second is that when you press the add button, doing newPiece.blur() doesn't hide the keyboard, so I found a workaround in the q&a to disable and enable the control which got rid of the keyboard.
