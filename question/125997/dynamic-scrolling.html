Dynamic scrolling

With regards to dynamic scrolling, I have been trying the KitchenSink sample code. I am unclear about the way it does the dynamic scrolling. It seems having an eternal loading…. How can I determine the times and quantity of rows in scrolling? Thanks

— asked September 25th 2011 by Jiale He

  • The loading time in the Kitchen Sink demo is set up to simulate a fetch of the data. The number of rows you load is up to you. Looping over your fetched data and calling appendRow() for each row you want to add is all you must do. The file in the kitchen sink that demonstrates this is table_view_dynamic_scroll.js
