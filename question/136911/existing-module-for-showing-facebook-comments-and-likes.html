  • The Ti.Facebook namespace supports Open Graph API calls which allow you to query and retrieve comment objects, but there is no native module for duplicating the Facebook Comments Social Plugin. That plugin is a web based and I haven't seen anyone duplicate that functionality natively for any other platform.
