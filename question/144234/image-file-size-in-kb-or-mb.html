    suppose i have an image named yourimage.png under resource directory 

    var file= Ti.Filesystem.getFile(' yourimage.png');
var size_bytes= file.getSize();
     Ti.API.info("file size in bytes" + size_bytes);

    once you have imagesize in bytes ,then you have to do simple math calculations for yor MB or KB size and if you need math conversion help then follow http://www.t1shopper.com/tools/calculate/
