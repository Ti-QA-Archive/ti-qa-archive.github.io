  • This error simply says that titanium studio is not able to install the app to the device, though the compilation went fine.

    The ipa/app file is actually generated in the build folder, its just that the studio is not able to open it up with your itunes.

    So for those(like me) who stuck up with this situation for a while the SOLUTION is to:

    Open your build folder, drag the projectname.app or projectname.ipa folder generated under build/Debug-iphoneos or build/Release-iphoneos to itunes and sync it with your iphone/ipad.
