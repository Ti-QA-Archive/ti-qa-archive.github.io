Please wait accessing live radio . . .

And I need the dots to NOT be static i.e. every 2 seconds change the number of dots so it goes 'one dot, two dots, 3 dots, one dot and so on.

I was wondering what is the best way to do this. Should the dots be in an image view? Should I have three images each with a specific number of dots? Or is there a different way to animate this?

    Hi, One option is to use setInterval:

    label.text = "Please wait accessing live radio .";
interval = setInterval(function() {
    //write code to change the text of label or any other component from here...
    if (label.text == "Please wait accessing live radio .") {
        label.text = "Please wait accessing live radio .."
    } else if (label.text == "Please wait accessing live radio ..") {
        label.text = "Please wait accessing live radio ..."
    } else if (label.text == "Please wait accessing live radio ...") {
        label.text = "Please wait accessing live radio ."
    }
}, 2000);

    • @Moiz Chhatriwala @Pooja Laad the reason it wasn't working was because of the 

      streamer.start();

      It was blocking the change of text. So what I did was create 3 images, one with one dot, one with two dots and one with 3 dots. Then in the imageview definition I had

      var dots=Titanium.UI.createImageView({
    images:['1.png', '2.png', '3.png']
});

      And in the event listener above I added in:

       win1.add(dots);
dots.start;

      And it worked
