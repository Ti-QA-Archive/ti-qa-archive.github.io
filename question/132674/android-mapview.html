Android MapView

I have a mapview that I am trying to add buttons to. However when I add the buttons to the map view, no matter what I set the top: and left: to, the buttons always appear in the top left hand corner on top of each other.

This only occurs in the Android emulator, the iPhone works fine.

Has anyone else ran into this problem or have any suggestions on why this occurs only on the android?

  • I fixed the issue by adding the buttons to the window and not the mapview, as shown below:

         // Add map view to window
    Titanium.UI.currentWindow.add(mapview);

    //add buttons to window
    Titanium.UI.currentWindow.add(button1);
    Titanium.UI.currentWindow.add(button2);
    Titanium.UI.currentWindow.add(button3);

    If the buttons are added before the mapview is added, the buttons can not be seen.
