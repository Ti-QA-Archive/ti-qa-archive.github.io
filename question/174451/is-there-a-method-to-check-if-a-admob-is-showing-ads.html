Is there a method to check if a AdMob is showing ads?

Hi, as I wrote before, ads are not shown if the view box hasn't certain dimensions and can happen that AdMob doesn't work because of device screens too small. So, without checking screen dimensions, is there a way to verify if AdMob is working? When it doesn't in the console you read this

[WARN] :   Ads: Not enough space to show ad! Wants:.......

Thanks for the attention.

    Hi mario rossi,

    You can use listener to check, ad is loading or not.Here is the sample code of adMob module. You can try this or see the module example you will get this inside the module named folder example. 

    adMobView.addEventListener('didReceiveAd', function(e) {
        Ti.API.info('New Ad');
        banner.show();
    });

    adMobView.addEventListener('didFailToReceiveAd', function(e) {
        Ti.API.info('New ad failed');
        banner.hide();
    });

    or

    var Admob = require('ti.admob');

adMobView.addEventListener(Admob.AD_RECEIVED,function(){
   // alert("ad received");
   Ti.API.info("ad received");
});

//listener for adNotReceived
adMobView.addEventListener(Admob.AD_NOT_RECEIVED,function(){
    //alert("ad not received");
     Ti.API.info("ad not received");
});

    If you have any question then let me know.
