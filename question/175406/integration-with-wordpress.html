  • Hi

    You need to investigate the HTTPClient and the Request Lifecycle documentation. This gives you the building blocks of sending a remote request to server and retrieve the results of that request if there any.

    These are also must reads;

    • Working with Remote Data Sources
    • Titanium Network HTTPClient

    This is an ambitious first app, I suggest you spend some time learning the basics before attempting to remote connect to third-party services. Start with some tables, windows, navigation, buttons, events. Network calls (especially those to third-party services) can and often are, complicated - understanding the basics makes it easier to understand how to resolve some of the difficulties you will encounter. This is not a Titanium fault - it is just that things like network authorisation, custom headers, session monitoring are not the easiest first time problems to understand.

    Good luck though
