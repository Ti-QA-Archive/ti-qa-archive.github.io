Unable to set view width

I'm unable to set the width of a view window using the width of a label in Android but able to do so with iOS.

This is what I have right now

var btnView = Ti.UI.createView({
            top: 1,
            height: 40
        });

var label = Ti.UI.createLabel({
            text: 'Some Title here',
            font: {
                fontSize: 14,
                fontStyle: 'bold'
            },
            width: 'auto',
            textAlign: 'center',
            height: 40,
            touchEnabled: false
        });

This is where I'm having trouble

btnView.width = label.size.width;

iOS returns the correct value but Android always returns 0.

Does anyone know how to fix this? Any help is much appreciated.

  • just a simple thing , if you are adding label into btnView , then set

    var btnView = Ti.UI.createView({
            top: 1,
            height: 40,
        width:'auto'
        });

var label = Ti.UI.createLabel({
            text: 'Some Title here',
            font: {
                fontSize: 14,
                fontStyle: 'bold'
            },
            width: 'auto',
            textAlign: 'center',
            height: 40,
            touchEnabled: false
        });
btnView.add(label);

    the width of button will automatically adjusted to width of label.
