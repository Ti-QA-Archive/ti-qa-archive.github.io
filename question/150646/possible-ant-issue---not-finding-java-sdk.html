    Hello Colin,

    • Set JAVA_HOME to C:\Program Files\Java\jdk1.6.0_24
    • Set PATH to C:\Program Files\Java\jdk1.6.0_24\bin

    After setting these variable then please open Command Prompt and type javac. If Environment variable is set then these command will complile otherwise it will show the error.

    After setting the variable, just restart the Titanium and run your app again

  • I found the problem - it was my stupid mistake!

    I had set JAVA_HOME to C:\Program Files\Java\jdk1.6.0_20
    but it was actually C:\Program Files\Java\jdk1.6.0_25

    I fixed this, restarted, and now 'ant debug' worked - the build was successful!

    Thank you to those who replied to my post - it was because of you that I found the error.
