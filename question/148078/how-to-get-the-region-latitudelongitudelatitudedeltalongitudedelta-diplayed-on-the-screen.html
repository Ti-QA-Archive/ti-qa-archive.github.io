    Hi

    Assuming you have a simple map view - this is all you need.

    // assumes mapView is the variable used to define the map
var currentRegion = mapView.getRegion();

    Documentation for the getRegion method.

      var mapview = Titanium.Map.createView({

currentRegion.addEventListener('click',function(){
    var region = mapView.getRegion();
    alert(JSON.stringify(region));
}
