  • As error message says, you have passed "undefined" value to "parseString" method of JSON object, instead of "string" value.

    var jsonString = undefined;

Ti.JSON.parseString(jsonString);

    This code produces that error.

    To correct this kind of errors, so a simple check:

    if (jsonString) {
    Ti.JSON.parseString(jsonString);
}
