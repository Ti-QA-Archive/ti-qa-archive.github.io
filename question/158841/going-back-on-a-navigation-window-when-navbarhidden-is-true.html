Hello,

  • Accepted Answer

    Do the following coding:

    var cur = Ti.UI.currentWindow;
var bakbutt3 = Ti.UI.createButton({
    image : '/images/arrow_back.png',
    backgroundImage : 'none',
    backgroundColor : 'transparent'
});

cur.setLeftNavButton(bakbutt3);
