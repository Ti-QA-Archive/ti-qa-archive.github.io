  • Everything you do has a 'reference' variable (if you need it to).

    Think about it that way.

    It's possible to put the 'context' of any variable or object in a wider 'scope'. If you want anything available to you, in any type of window of view: just define a variable for it in the creation of the new window.

    Does that make sense?

    Don't worry if it doesn't. It took a while for me to get to the "ahah" moment of 'wiring' various variables and functions to other windows.
