  • Hi Renato,

    For that you have to put androidmanifest.xml file in your project.And into that file you have to set orientation of application.

    I have already done this.Follow this step :

    • run the application
    • now go to build/android/androidmanifest.xml
    • create one folder in project named platform.
    • copy that manifest file and paste in platform/android folder.
    • platform/android/androidmanifest.xml

    now open the manifest file and change the orientation.
