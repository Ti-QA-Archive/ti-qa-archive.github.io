  • So the main problem is that the source at ti.urbanairship is using an older version of Urban Airship's library (Android is 2.0.3 and iOS is using 3.0.2). The library currently available from Urban Airship is 5.1 Dev Resources

    There have been a number of API changes so its not enough to just update the library. The other issue is the 5.1 branch seems to require JDK 1.7 and according to this question the latest titanium still has trouble with JDK 1.7
