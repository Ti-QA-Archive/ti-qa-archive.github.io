Delete a user's ACS custom objects from cloud.appcelerator.com

Let's say I want to delete all custom objects created by a specific user. How can I achieve this from cloud.appcelerator.com? I know I can filter the user's custom objects, but I can't find a way to delete them all at once. Of course I can delete them one by one, but if the user has 10.000 objects, it will take a while.

— asked March 21st 2015 by Simen Solbakken

  • Accepted Answer

    ACS provides an API end point named batch_delete that allows application admins to delete multiple ACS objects in one operation.

    Create an admin_account for the application and use that account for batch operations

    http://docs.appcelerator.com/cloud/latest/#!/guide/admin_access
