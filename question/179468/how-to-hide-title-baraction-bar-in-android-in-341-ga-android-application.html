  • Hi Deepan,
    Please add this code to your tiapp.xml file(Proper place).
    <android xmlns:android="http://schemas.android.com/apk/res/android">
    <manifest>
    <application android:debuggable="false"
    android:theme="@style/Theme.AppCompat.Translucent.NoTitleBar">
    </application>
    </manifest>
    </android>

    I have tested this with your code and it working without any issues. Hope this helps you.
