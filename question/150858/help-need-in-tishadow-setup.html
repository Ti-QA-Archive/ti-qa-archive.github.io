  • The TiShadow app in the Google Play store really outdated. You need to build the app.

    Do the following (as per the README in the repo):

    This creates the app:

    $ mkdir ~/tishadowapp
$ tishadow app -d ~/tishadowapp

    Then using titanium cli (or the equivalent for iOS):

    $ cd ~/tishadowapp
$ titanium build -p android -T device
