Can't click WebView contents, when bind events(like click, touchstart) to WebView or It's Parent

// this sets the background color of the master UIView (when there are no windows/tab groups on it)
Titanium.UI.setBackgroundColor('#000');

// create tab group
var tabGroup = Titanium.UI.createTabGroup();
var root = Ti.UI.createWindow({
    tabBarHidden:true,
    title:'WebView Testing'
});
var tab = Ti.UI.createTab({
    window:root
})
tabGroup.addTab(tab);
// open tab group
tabGroup.open();

var w = Ti.UI.createWebView({
    url:'http://www.google.com'
});

root.add(w);

//If Comment these lines , contents inside webview could click, 
//and webview could scrolling
w.addEventListener('touchstart', function(e){
    Ti.API.info(e.x);
});

    This is a common issue. If you add a touch EventListener to webView, it does not manage its internal events. Solution: do not use touch events in a webview (you can still listen for other "not-touch" events such as load, beforeload and so on).
