  • You can't combine them into a single sound, but you can play them back to back.
    We did this for a piano type app for special needs children.

    We preloaded all the notes into memory for a full sized keyboard:

    var player = {};

Ti.API.info('Loading Octive 0');
player.a0 = Ti.Media.createSound({url: Ti.Filesystem.resourcesDirectory+'audioLibrary/Piano.ff.A0.mp3', volume: 1});
player.b0 = Ti.Media.createSound({url: Ti.Filesystem.resourcesDirectory+'audioLibrary/Piano.ff.B0.mp3', volume: 1});

    We have the songs in a custom JSON structure, but essentially we had a playNote() function that stops the first sound and starts the second one.

    player[notes[currentNote].step+notes[currentNote].octive].stop();
player[notes[currentNote].step+notes[currentNote].octive].play();

    For our application, there is no noticeable delay between the two. That being said, if you notice delays, take a look at Uri's ZeroLatency Sound module.
