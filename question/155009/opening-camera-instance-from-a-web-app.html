Opening Camera Instance from a Web App

Is it possible to open Camera instance from Web App?

I have a Web App. displays something, and I want the user to be able to Take a picture and send it to the

server. How can I achieve this?

  • Your web app (assuming it is built with titanium) can fire an app level event (see Ti.App.fireEvent, and Ti.App.addEventListener) that can initiate the Ti.Media functionality to access the hardware for the camera (and subsequently upload the file to a server)
