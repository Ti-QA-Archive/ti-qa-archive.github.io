How can I make an IP scanner?

Is it possible to make an IP scanner app to find all the devices' IP and names in the same local network with Titanium?

I have tried with Bonjour but I could only find apple devices with Bonjour services running. For example, I can find my iMac,Macbook which are sharing some folders with afp. service and I can find my iPhone is it is running some apps with Bonjour services like iNet. However, I could not find my iPad or iPhone if it is locked or sleeping and I could not find any Android phones in the same network.

It would be much appreciated if you can give me a solution with Titanium.

  • Bonjour is iOS only and requires that devices be in discovery mode in order to find them. You could scan IPs on your local network using the Socket API, but that probably won't allow you to get information about the device, just whether or not the IP is in use.

    There are a couple of modules on the marketplace that allow you to do some network communications. The SNMP module will let you talk to switches, routers, etc. Which may be helpful if you want to get the names of all the devices on your network. Then there is a UDP module that will allow you to communicate with UDP services.

    If neither of them meet your needs, you may think about creating a module yourself to add the functionality.
