    • Opens a single window:

      var win1 = Titanium.UI.createWindow({   
    title : 'Site',
    backgroundColor : '#fff'
});

win1.open()

      Opens a single window with one webview:

      var win1 = Titanium.UI.createWindow({   
    title : 'Site',
    backgroundColor : '#fff'
});
var webview1 = Ti.UI.createWebView({
    url: '/book/index.html'
});

win1.add(webview1);

win1.open()

    • RTFM to add a button to your window.

      Add an eventListener to the button that changes webview1.url to the second webpage and back

    webview1.url = '/book/chisiamo.html';

      myButton.addEventListener('click', function(e){
    if (webview1.url == '/book/index.html'){
