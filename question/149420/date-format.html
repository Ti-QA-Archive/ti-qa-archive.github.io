Date format

Hi.

I am getting a date from a datepicker but i want to loose the "0" before the month shown.

As if now i get the months as 03.

How do i loose the 0?

This is how i get it

var getDate = function(datetime){
    var currDay = datetime.getDate();
    var currMonth = datetime.getMonth()+1;
    var currYear = datetime.getFullYear();
    var hours = datetime.getHours();
    var minutes = datetime.getMinutes();

    if(currDay <= 9){
        currDay = '0'+currDay;
    }
    if(currMonth <= 9){
        currMonth = '0'+currMonth;
    }

    return currMonth+'/'+currDay;
}

This will get 03/11 as today.

Thanx.
