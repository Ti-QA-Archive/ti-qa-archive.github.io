  • Found it.

    Apparently Titanium is one hell of a lot more lenient when it comes to compiling than xCode is. Titanium continued to compile while one of my files had an unterminated string constant. It wasn't until I looked at the build log that I saw the error message and the offending file.

    Appcelerator folks, perhaps you would want to trap for this condition prior to throwing an obscure build error? It would have saved me hours of snipe hunting not to mention lost code due to rollbacks.
