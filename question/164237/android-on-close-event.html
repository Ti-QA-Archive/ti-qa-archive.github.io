android on close event

I'm trying to execute some code during app closing, but I don't know why…

I have tryed this code:

Ti.App.Android.addEventListener('close',function(e) {
Ti.API.info("do something");
});

but don't work

  • There is no close event for App.Android. I'd suggest looking at:

    Ti.Android.currentActivity.addEventListener('pause', function(e){ /*Do Something*/ });
