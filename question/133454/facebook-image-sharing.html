Facebook image sharing

Hello everybody,
I'm getting this error on the iPhone emulator:
[UIImage length]: unrecognized selector sent to instance

For this piece of code:

#


function send_facebook_stream() {
                log('******************* send_facebook_stream ****************');
                var f = Ti.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory, getImageName(infoBorsa.immagine));
                alert(f.exists());
                var blob = f.read();

                var data = {
                    message : infoBorsa.titolo,
                    picture : blob,
                    link : infoBorsa.link
                };

                log(JSON.stringify(data));
                facebook_dialog = Titanium.Facebook.dialog("feed", data, showRequestResult);
            }

Any ideas?

    I use this to post a photo to facebook

    
var data = {
            name : message,
            picture : photo
        };

var where = 'me/feed';

Ti.Facebook.requestWithGraphPath(where, data, 'POST', callback);

    callback is a function to accept the results of the POST operation
