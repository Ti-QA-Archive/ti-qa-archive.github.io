  • "Now my alloy.js file contains a huge amount of functions to acces them all over the application."

    Then you're doing it wrong. Sorry to be a bit blunt. You do not need nor should you use global variables in most situations. You should be using CommonJS modules to encapsulate functionality and store values, instantiating those modules on an as-needed basis.

    Check out some of these resources:

    • Alloy best practices guide
    • Titanium best practices guide
    • Slideshare: Memory management
    • TiDev.io: Memory management
