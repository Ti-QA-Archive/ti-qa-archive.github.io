    Well, by definition, horizontal would stack views horizontally and vertical would stack them vertically. The composite or absolute would be the default layout based on point system. Everything you want to read more about can be found in the docs: http://docs.appcelerator.com/titanium/2.1/index.html#!/api/Titanium.UI.View-property-layout
