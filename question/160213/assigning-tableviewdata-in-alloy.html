Assigning TableView.data in Alloy

I found this bit of classic code that I wanted to convert to Alloy:

var tableData = [ {title: 'Apples'}, {title: 'Bananas'}, {title: 'Carrots'}, {title: 'Potatoes'} ];
var table = Ti.UI.createTableView({
    data: tableData
});

I tried this XML in the main view (index.xml):

<Alloy>
  <Window>
    <TableView data="tableData">
    </TableView>
</Window>
</Alloy>

And this code in the controller (index.js):

var tableData = [ {title: 'Apples'}, {title: 'Bananas'}, {title: 'Carrots'}, {title: 'Potatoes'} ];

This results in an error "Invalid type passed to function". I also tried creating the Array in Alloy.js and attaching tableData to the 'Alloy.Globals' object, but get the same error.

What would be the recommended approach to this in Alloy ?

  • Hi,

    so i think you need to first learn the Alloy structure from the scratch.

    try the below code….

    you need to fist set tableview's id to use tableview in your index.js file so do this..

    index.xml

     <TableView id="tbl">
    </TableView>

    index.tss

    "#tbl": {
    width : Ti.UI.FILL,
    height : Ti.UI.FILL
}

    index.js

    var tableData = [ {title: 'Apples'}, {title: 'Bananas'}, {title: 'Carrots'}, {title: 'Potatoes'} ];
$.tbl.data = tableData;
