Find and print iOS version dynamically

How can i find the iOS version of the device i am running my application on and print it in a label?
I am using titanium 2.1.3.GA SDK. Thanks in advance.

  • Thank you all. I read about it and did it.
    In my case it was:

    var version = Titanium.Platform.version.split(".");
    var major = parseInt(version[0]);
    if (major < 6){
    tabGroup.addTab(tab2);
    } else {
    tabGroup.addTab(tab3);
    // iOS 6
    }
