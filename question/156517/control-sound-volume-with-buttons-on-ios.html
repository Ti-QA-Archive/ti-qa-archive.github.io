Control sound volume with buttons on iOS

Mobile, Titanium SDK 3.1, iOS, Device

I am trying to allow the user to control the volume with the hardware buttons, of the sound files our app play. This is possible if the iOS setting "Sounds -> Ringer and Alerts -> Change with Buttons" is set to OFF. If it is ON only the ringer signal volume is changed when the user uses the hardware volume buttons.

However, it is possible to change the volume even when that setting is true, but only while the sound is beeing played.

I have seen other apps allow volume change even if there is no sound playing (at least not one that is hearable).

My question: Is it possible to make the hardware buttons work when the setting is ON and no sound is playing?
