Highcharts and Titanium

Hi all,

I need to have advanced charts in our app and highcharts seems to provide everything we need (charts, stocks charts). I have tried a very simple chart in a webview but the chart is not displayed. The "This is a test" text appears fine but nothing else.

Has anybody succeeded in using highcharts in Titanium ? I could not find a proper example to try and reproduce.

Here's the simple code I use.

        var webviewtest = Ti.UI.createWebView({
            height : 300,
            width: 100

        });

var htmlContent =     "<!DOCTYPE HTML>" + 
"<HTML>" + 
"<HEAD>" + 
"<SCRIPT type='text/javascript' src='lib/highcharts/highcharts.js'></script>" + 
"<SCRIPT type='text/javascript'>" + 
"var chart = new Highcharts.Chart({ chart: {renderTo: 'container',defaultSeriesType: 'area'}," + 
"title: {text: 'Chart'}," + "series: [{ data: [12,15,17]}]});" + 
"</SCRIPT>" + 
"</HEAD>" + 
"<BODY>" + 
"<P>This is a test.</P>" +
"<div id='container' style='width:200; height: 100; margin: 0 auto'></div>" + 
"</BODY>" + 
"</HTML>";

webviewtest.html = htmlContent;

Thanks in advance for your help,

Sébastien

Note: Ti 2.1 + IOS 6 SDK
