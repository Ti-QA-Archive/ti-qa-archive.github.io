Create (closed source) components

Is there any way I can create application components in Titanium (js), distribute those for re-use by third parties (license based) without providing them sources?

I do not mean creating custom modules in Obj-C, but pieces of Ti code that can be re-used by others in their application.

— asked July 27th 2011 by Marijn Deurloo

  • Javascript is javascript. You cannot really encode it - only obfuscate it.

    One way would be to take a look on how tiphp and tipy modules are done, and overlay a decode method to provide the js from an encoded content.

    Another one would be to simply license the code for one app use only ( see codecanyon) and trust the people they won't steal it :)
