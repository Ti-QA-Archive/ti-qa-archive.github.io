JAVA_HOME:javac, java and javasigner not found

I'm new to Titanium, trying Android. I'm trying to get the Geolocater template app to run. I'm getting an error message.
[ERROR] Required jarsigner not found
[ERROR] Required javac not found
[ERROR] Required java not found
[ERROR] One or more required files not found - please check your JAVA_HOME environment variable

HOWEVER, I think that the JAVA_HOME environment variable IS correct. If I run java, javac or javasigner from the Titanium Console window, the executibles work correctly. Also, the Java bin directory is in the PATH

JAVA_HOME is C:\Java\jdk1.7.0\bin

I'm using Windows XP.
Java JDK is in C:\Java\jdk1.7.0\bin
Titanium version is the latest Mobile SDK 1.7.2
Maybe the version of JAVA isnt correct

    Remove the \bin to your JAVA_HOME.

    This gives you:

    JAVA_HOME is C:\Java\jdk1.7.0

    If you want to be able to access the javac compiler from the command line. Simply add C:\Java\jdk1.7.0\bin to your PATH.

    • Finally, I managed to get a simple android app working on the emulator. I found it quite tricky. Problems were:

      1) I suggest getting started with the GETTING STARTED GUIDE, read the manual, although it's for Titanium Developer, not Studio

      2) you need java 1.6, 1.7 doesnt seem to be supported.

      3) JAVA_HOME must not point to the ..\bin directory; Test your java/javac exececutibles in a console window to check that the path is ok.

      4) You need to download the earlier versions of the android SDK, Versions 10 and 11. You can download 12 and 13 as well, but Titanium looks for the older versions.

      5) I used the Android SDK manager to start up the emulator

      6) Some versions of android SDK dont like the directory names to have spaces. e.g. Program Files. Either use a different version of SDK or use names with no spaces.
