create an eventListener on a thumbnail image to open a new image view

I'm new to programming and new to Titanium. I'm working on building a basic app with ui only two layers deep. It's a streetfighter II cheat app where the home screen is thumbnails of the various characters and then I want to make each thumbnail clickable. Clicking the thumbnail should open a new image view that has all the special moves for that character. I have all the images for this app designed already so I want to figure out how to do this in this particular way as opposed to styling the new view with css.

I've been searching the help docs everywhere and can't quite figure out how to apend an attribute to make the thumbnail clickable and lead the user to the new image view.

Can you help me?

Thanks,

    You have to add an eventListener for any object or UI control that you want to capture it. The list of events each object can listen is listed in the Titanium API. I. e., imageView can listen for these events, which includes the click event. Once you know the event to listen, implement the event listener this way:

    instanceObject.addEventListener(event, functionToExecute);

    you will usually see implementation like this

    imageView.addEventListener('click', function(event){

    //the image has clicked, put here your stuff

});

    How the new image is opened and showed to user depends on your interface design. You can, for example, launch a new window

    imageView.addEventListener('click', function(event){

    //the image has clicked, put here your stuff

    var win = Ti.UI.createWindow();
    var largeImage = Ti.UI.createImageView({
        top:0, left:0, right:0, bottom:0,
        image: urlToYourImage
    });

    win.add(largeImage);
    win.open();

});

    of course, you will need to add a close button and other stuff.

    From this point, I suggest you read User Interface Fundamentals guide. Good luck with the game!
