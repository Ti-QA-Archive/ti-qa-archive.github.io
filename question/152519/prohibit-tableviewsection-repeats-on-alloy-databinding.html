Prohibit TableViewSection Repeats on Alloy DataBinding

Hi there-

I have what I believe should be a simple question. I am doing a databind to a REST collection. The query works as expected, I receive my results, and the results print to my TableViewRows. I have my rows within a TableViewSection for grouping purposes. What I am ending up with though is that each TableViewRow is placed within a separate TableViewSection (so the TableViewSection is repeated on a per row basis). Is there a way to prohibit this from happening and simply increment the data around the rows within the section?

<Alloy>
  <Collection id='myData' src='myModel' instance='true' />

  <Window id='myWin' title='My Window'>
    <TableView id='myTableView' dataCollection='$.myData'>
      <TableViewSection>
        <TableViewRow title='{Name}' />
      </TableViewSection>
    </TableView>
  </Window>
</Alloy>

…please forgive the implementation of the collection and binding…when attempting to implement per the documentation, the fetch works, but the table never rendered (I finally resolved to the above implementation which works, but totally not sure why the difference in this utilization versus that listed in the documentation…but that is for another thread).

Thanks for any help.
Chris

    Ah, this is a current limitation of Alloy. It doesn't yet support TableViewSection data binding: https://jira.appcelerator.org/browse/ALOY-444

    To create them in sections you may have to do it programmatically for the time being. Or of you are using only a single section, you could just as easily use the "headerView" property of the TableView to display the same thing as your section. The "headerView" does not get repeated for each model instance like the section did.

    <Alloy>
  <Collection id='myData' src='myModel' instance='true' />

  <Window id='myWin' title='My Window'>
    <TableView id='myTableView' dataCollection='$.myData'>
      <HeaderView>
         <View class="classThatStylesLikeASection"/>
      </HeaderView>
      <TableViewRow title='{Name}' />
    </TableView>
  </Window>
</Alloy>

  • I'm using the dataTransform to get the Header:

    <TableView dataCollection="contact"
       dataTransform="transformFunction">
    <TableViewSection headerTitle="{firstLetter}">
        <TableViewRow title="{fullName}"/>
    </TableViewSection>
</TableView>

    var last_letter
function transformFunction(model) {
    var transform = model ?  model.toJSON() : null;
    var firstLetter = transform['lastName'].charAt(0);
    if(last_letter != firstLetter){
      transform['firstLetter'] = firstLetter;
      last_letter = firstLetter;
    }
 return transform;
}
