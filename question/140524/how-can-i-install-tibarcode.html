How can I install ti.barcode?

Hi guys, I'm new on development with Titanium and I'm looking for some plugins to develop some apps. I have downloaded the ti.barcode and I don't know how install it. Can anyone help-me? (My OSx have the Xcode 4.4 installed)

    Hi Raul

    Take the zip file you have and unzip it to a new folder somewhere temporary.

    Copy the folder called modules and then find your apps resources folder.
    Move up one folder from resources and paste your modules folder here. If there is already a modules folder do not worry they will merge.
    Then go to the app you are working on and open up the tiapp.xml file in Titanium studio.
    On the right you will see Modules, press the add button and you should see your module listed, select it and confirm.

    Then use the example code found way down nested inside the modules folder you copied, it will show you how to get it going.

    Quick tip, you need to run it on an actual device and the simulator will not have a camera of course. You can however scan from images so if you have saved any to your simulator you can work with those.

    Happy scanning.
