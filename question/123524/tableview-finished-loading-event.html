Tableview finished loading event?

Hey guys I have a question I don't find an answer to. I'm populating a table view with data from XHR and I have an activity indicator showing up while the data loaded.
My problem is that I close the activity indicator when I have all the data and the table is added to the window but since the table rows are complex (lot's of images, buttons, handlers) it takes much more time for the app to actually draw the table view than to gather the data from a server. So my activity indicator hides when it has the data (approx. 1-2 seconds) but the table view appears much later (4-5 seconds).
Any idea how could I determine when the table view is drawn to the window to extend the activity indicator closure?
Thanks.
