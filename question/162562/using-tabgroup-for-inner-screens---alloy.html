Using tabgroup for inner screens - Alloy

I've been playing around with alloy framework for 2 weeks now. I'm not sure whether this is a blunder or a valid one. Here we go…

The first screen for my app is the login screen (normal screen with a form and not any tabgroups) and if login is successful, I need to navigate to the second screen which has the tabgroups in it. How could I push in a screen with tab groups from a normal screen like the login screen I mentioned.

Thanks!

    Please don't add new answers to comment, just use the comment button to keep the order of conversation.

    First you need to remove the open function from the controller
    In the login function you need to create a tab controller and open it. like:

    function login(){
    // if credentials correct
    var myTabGroup = Alloy.createController('theTabGroupThatYouCreated').getView();
    myTabGroup.open();

}

  • Got this one.

    xml file 

    <TabGroup id="tab" >
</TabGroup>

    css (android back button to take you back to previous screen)

    "#tab": {
    exitOnClose:false,
    navBarHidden : false,
}

    controller

    $.tab.open();
