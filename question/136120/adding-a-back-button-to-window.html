Adding a back button to window

I want to add a back button to the window that will navigate to my previous window. I am having a navigation group with navBarHidden to true. going on next window is easy but how I go back to my previous window

  • Accepted Answer

    Why you are hiding your navbar?

    you declare a navigation group like this:

          var nav = Titanium.UI.iPhone.createNavigationGroup({
      window: win1 //win1 is window defined above

     });

    Now suppose you have win2(another window) on which you navigate.
    so while opening that win2 just do like this

    nav.open(win2,{animated:true});

    by doing this the titanium will automatically add a back button on the top.

    OR

    you can do like this if you dont want that automatic back button

         var win = Titanium.UI.currentWindow;
     var b = Titanium.UI.createButton({title:'Back'});
     win.leftNavButton = b;
        b.addEventListener('click', function()
        {
           alert('I was clicked'); // to confirm its being called
        // do the stuff here
        win.close();
      });

  • Hi Titanium Dev,

    If you set your navBarHidden as true than you have to create custom view look like navBar.Try this code best luck

    var navView = Ti.UI.createView({
    height : 38,
    width : 320,
    top : 0,
    left : 0,
    backgroundImage : '../images/NavBar.png'
});
win.add(navView);

var backbtn = Ti.UI.createButton({
    backgroundImage : '../images/backButton.png',
    bottom : 0,
    left : 0,
    height : 33,
    width : 86
});
backbtn.addEventListener('click', function(e) {
    win.navGroup.close(Ti.UI.currentWindow);
});
navView.add(backbtn);
