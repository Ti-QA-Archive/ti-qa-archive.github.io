Alloy.Globals.* array??

I want to use arrays as global variables and read them from .tss file, but Titanium gives me errors when I declared/used like this:

alloy.js

var array = [];
array[0] = 64;
array[1] = 96;
Alloy.Globals.array = array;

index.tss (gives me error -> syntax error: unexpected token "[" )

"#button1":{
    width:Alloy.Globals.array[0],
    heigth:Alloy.Globals.array[1]
},

How can I use arrays in .tss ? or I can't ??

Thank you.

    May be titanium didn't support arrays in tss . Instead you can use objects to achieve what you need.

    var myButton = {};
myButton.width = 64;
myButton.height = 96;

Alloy.Globals.myButton = myButton;
and then you can access as Alloy.Globals.myButton.width and Alloy.Globals.myButton.height
