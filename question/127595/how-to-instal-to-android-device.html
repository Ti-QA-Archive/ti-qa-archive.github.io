  • Assuming you have the device connected via usb, then the android debug bridge (adb) is not recognizing it. This occasionally happens, and to fix it, you just restart the adb server via command line.

    Open a console or terminal and run adb kill-server, wait a few seconds, then run adb start-server. You can then do adb devices to see if your device is being found. You can see the path to the adb.exe in the log snippet you posted.
