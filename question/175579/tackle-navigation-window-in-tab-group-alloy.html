Tackle navigation window in tab group alloy

I have a two-tabbed alloy application. In each tab i am loading the view using the <Require> tag. Not a big deal. But each of these views have a navigation window present within them so when the application opens, on top the tab bar title is seen and below that the ios navigation bar is shown. Is there a way to handle this. I know i can create a button but i want to know if its possible to do this with navigation window. My titanium sdk version is 3.2.3

        <Tab id="tab2" title="My Jobs">
            <Require type="view" src="myJobsList" id="myJobsListTab"></Require>

</Alloy>

    Hi

    I am having difficulty understanding what you needs are in this section;

    But each of these views have a navigation window present within them so when the application opens, on top the tab bar title is seen and below that the ios navigation bar is shown.

    However more importantly, each Tab already has a Navigation Controller, so adding a NavigationWindow is simply wrong. You should remove the navigation controller and simply remove the root window in each case. Then open a new window as a child of THAT tab - this is the correct way.

    • If it was me I would tweak things a little to make it easier to reference the tab you need to make this work.

      index.xml

      <Alloy>
      <TabGroup backgroundColor="white" >
      <Require type="view" src="allJobsList" id="allJobsListTab"></Require>

      test.xml

      <Alloy>
    <Window id="container1">
        <Label test="New Window" />
    </Window>
</Alloy>

      myJobsList.js

      var newWin = Alloy.createController('text', {});
$.tab1.open(newWin.getView();

      The logic is is as follows;

      • Put the tab into the required file itself - this gives you a reference to it
      • Create a simple window for testing purposes
      • In the controller for the required file, open that controller
      • Reference the tab and open the new window as a child

      This automatically opens the new window as a child of the tab - back button is there automatically as you wanted.
