listview rows very very thin

Hello,

I am starting new with Ti and own a Galaxy S. Just to start playing I created a listview to use as a menu. However all of the rows are very thing. Which of course makes it really hard to click on them.

The rows in the kitchen are perfectly fine, and the rows in some other apps I've downloaded from the market, make with Ti are ok as well. So it has to be something I am doing. I know there is a height attribute but then the rows are huge on the lesser screens. Any help would be appreciated.

Nate

  • I use the following for Ipad versus Iphone

    
alert('My device is: '+Titanium.Platform.model);

// SOME CSS FOR IPHONE/IPAD
if(Titanium.Platform.model == 'iPad')
{    
    myRow.height = 50;
}
else  // IT IS NOT A IPAD
{
    myRow.height = 35;
}
