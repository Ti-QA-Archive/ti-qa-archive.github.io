Alloy and overlapping UI objects

Pretty much I don't see the navgroup at all due to the tableview and my toolbar is covering up the last row of a table

<Alloy>
    <Window class="container" id="mainWindow">
        <NavigationGroup id="navgroup">
            <Window id="win1">
                <LeftNavButton>

                </LeftNavButton>

        </NavigationGroup>
        <View>
            <TableView id="posts">

            <Toolbar platform="ios" bottom="0" borderTop="true" borderBottom="false">

</Alloy>

    Hi

    Your syntax is incorrect in the XML - everything is nested so if you have a window and you want the table inside that window - you place it inside the window markup NOT outside.

    <Alloy>
    <Window class="container" id="mainWindow">
        <NavigationGroup id="navgroup">
            <Window id="win1">
                <LeftNavButton>
                    <Button>Login</Button>
                </LeftNavButton>
                <TableView id="posts">
                    <Widget id="ptr" src="nl.fokkezb.pullToRefresh" onRelease="myLoader" />
                </TableView>
                <Toolbar platform="ios" bottom="0" borderTop="true" borderBottom="false">
                    <Items>
                        <Button id="send" title="Send" style="Ti.UI.iPhone.SystemButtonStyle.DONE" />
                        <FlexSpace/>
                        <Button id="camera" systemButton="Ti.UI.iPhone.SystemButton.COMPOSE" onClick="doCamera"/>
                        <FlexSpace/>
                        <Button id="cancel" title="Quit" />
                    </Items>
                </Toolbar>
            </Window>
        </NavigationGroup>
    </Window>
</Alloy>

    You also appear to have a pointless view - I have removed it, if there was a good reason for it, you can put it back.

    As the table is a child of the window it goes inside the window, the toolbar is also a child of the window so it also goes inside the window.
