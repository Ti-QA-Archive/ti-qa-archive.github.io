How to pass array object from javascript to WebView

I have created database in app.js and storing all database values in array. How to pass this array to webview? I've been going thru the forum for an hour and Im just not finding it. Im using the 1.8.1 sdk.

  • The right way to do this is to add an event listener in the web view:

    <script type="text/javascript">
Ti.App.addEventListener("web:data", function (event) {
    var data = event.data;
    Ti.API.info("Received " + data.length + " new rows.");
});
</script>

    And then in your app you can fire the data into the web view.
    This also allows you to send updated data.

    //We can call the event anything, let's call it "web:data"
//You can pass any object as the second parameter and access it in the listener
Ti.App.fireEvent("web:data", {data: [1, 2, 3, 4]});
