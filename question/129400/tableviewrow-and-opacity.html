TableViewRow and opacity

Hi,
is there a way to use opacity on a TableViewRow ?
There is an opacity property but I see no effect. My tableview background color is transparent.
Regards

    Nope, tablerows dont handle opacity, but the objects you add to your row do. For instance, maybe you have some text or an image that is in the tablerow, you can add opacity to that object and attach it to your row.

    Or you can add opacity to the table object to effect the whole table, rows included.
