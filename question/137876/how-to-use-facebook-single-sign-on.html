How to use Facebook Single-Sign-On?

Hi,
I am having the same problem as posted in stackoverflow regarding Appcelerator-Facebook SSO. Particularly, I set the Ti.Facebook.forceDialogAuth = false. What happened is that my Appcelerator app redirects to Facebook and request permission from the Fb app, however, once permission granted, the flow never return to my app..it just stayed there in FB app. Does anyone knows how to work with Facebook SSO?

Thanks so much for any help!
Kavin

    Hi Kevin,

    add this line to tiapp.xml below <guid>:

    <property name="ti.facebook.appid">134793934930</property>

    if you have a info.plist, then add this right after <key>CFBundleURLTypes</key>:

        <array>
        <dict>
            <key>CFBundleURLName</key>
            <string>com.appcelerator.titanium</string>
            <key>CFBundleURLSchemes</key>
            <array>
                <string>kitchensink</string>
                <string>fb134793934930</string>
            </array>
        </dict>
    </array>

    Of course you have to replace the ID's of this sample with yours.
