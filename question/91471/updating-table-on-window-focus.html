I have a Tab based app where some windows use a Tableview getting refreshed using windows focus.
In the focus function i am using "appendRow".

My Problem with this approach is that every time the same tab gets pressed the screen flickers, before the Table gets created. This flickering aswell affects the win.leftbuttons.

Is there a way to refresh a Table getting its Data from DB without this flickering?
