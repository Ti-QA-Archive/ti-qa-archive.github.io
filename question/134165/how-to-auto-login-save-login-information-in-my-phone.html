    You can do use the Titanium API Ti.App.properties.

    As only storing your username and password I dont think that you actually need sqlLite database.
    When the user uses your app for the first time he/she will register and Login to your app. save that login info into your App properties.

    And when user comes back next time just fetch the login from the App property and the user is auto-logged in..:)

    hope that helped…

  • No need to run web authentication everytime executing the Application.
    You need to use SQLite to save the login response.
    For example create a table in your local database and save the login response like user id, user name, email(if the response is success only)

    My sample code

    var loginReq = Titanium.Network.createHTTPClient();
loginReq.open("POST","http://xxx/users/login");
var params = {
    device: 'mobile',
    email : emailField.value,
    pass  : passwordField.value,
};
loginReq.send(params);

loginReq.onload = function(){
var response = JSON.parse(json);
    if(response.user[0].error == null){
        var db = Titanium.Database.open('mydb');
            db.execute('DELETE FROM users');
            db.execute('INSERT INTO users (u_id, first_name, last_name, email) VALUES(?,?,?,?)',response.user[0].user_id, response.user[0].first_name, response.user[0].last_name, response.user[0].email);
      }     
});

    everytime executing the Application check this condition

    /* Checking whethere user logged in or not */
var users = db.execute("SELECT * FROM users LIMIT 1");
var userCount = users.getRowCount();
if(userCount == 0){
   //open login page
}else{
   //open home page(if user already logged in)
}

    Hope this helps you
