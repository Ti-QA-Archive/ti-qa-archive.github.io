  • Did you add the window itself to your current window?
    Also the name you use (win) may lead to problems, if you use the KitchenSink exemples.

    // this is from KitchenSink...
var win = Titanium.UI.currentWindow;

var myWin = Titanium.UI.createWindow({ backgroundImage:'images/intro.png' });
win.add(myWin);
// then you add your button
