Titanium Desktop APIs not working

I'm kinda new to Titanium Desktop and I was wondering why my app is not working. All of the native browser JavaScript works fine. But when I try to use the Titanium UI API:

Titanium.UI.createWindow("http://www.website.here");

nothing happens. I've basically copied and pasted what was on the API docs page into my app but nothing happens. JavaScript after and before the function works but the API call doesn't. I've also tried packaging the app but to no avail.
Any help appreciated. Thanks.

(I'm using Titanium Studio v1.1.0)

    Ryan,

    By default the window is not opened. Try this:

    <html>
<head>
</head>
<body style="background-color:#1c1c1c;margin:0">
<div style="border-top:1px solid #404040">
    <div style="color:#fff;;padding:10px">
    <script>
        var win = Titanium.UI.createWindow("http://www.yahoo.com");
        win.open();
    </script>
    </div>
</div>
</body>
</html>
