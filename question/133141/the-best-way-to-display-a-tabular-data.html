Hi, I am looking for some help on the best way to display a Table. I know you can create a Table with Titanium but as far as I can work out, it can only have one column.   • To create table view columns, you need to set table view row's layout property to "horizontal" and add view columns to rows. For example:

    var row = Ti.UI.createTableViewRow({ layout: "horizontal" });
var column1 = Ti.UI.createView({ height: "50%" });
var column2 = Ti.UI.createView({ height: "50%" });
row.add(column1);
row.add(column2);

    • It stacks views from left to right ("vertical" stacks them from "top" to "bottom".

      Yeah, sometimes adding view to table view rows brakes things a bit. But, table view row extends a view so you can just use labels with their "width" set to some value, while having table view row's layout property defined as "horizontal".
