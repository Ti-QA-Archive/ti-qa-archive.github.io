  • I didn't found exactly the bug but I'm near.

    It happens when (AND):
    1) You set a breakpoint
    2) the "(x)=Variables panel" or the "Expressions" panel is shown"
    3) One of the variable contains "disturbing content" like binary data or a lot of data.

    Before run try to show the "loaded script" panel instead and check if the "child count update"
