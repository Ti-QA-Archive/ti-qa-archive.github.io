  • Fabio, What you are looking at in the google api docs is for web only, the native map API in iOS does not support this. The fast that you are looking at the javascript API docs is a confusing fact I am sure, but google is making a false assumption in their naming it "javascript api" that the reader is only using JS in the web browser (hence all the recommendations here to use a webview).

    That being said, some have, in pure Object-C, "hacked" a way to do this, an example of such a thing can be found here: https://github.com/mtigas/iOS-MapLayerDemo and someone COULD turn this type of thing into a titanium module.
