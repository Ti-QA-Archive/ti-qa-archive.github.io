How to get the users based on id using $in ?

I want to query cloud.user using $in in the where clause.

Following is my code.


var ids = ['4fd1f5a3002044061500fc25', '4fd0750a002044061e003251'];

Cloud.Users.query({
    page: 1,
    per_page: 100,
    where: {
        id : {"$in":ids}        
    }                                
}, function (e) {
     // here e.users in empty
});

If I try where clause with $in for first_name than it works fine.
