Help me understand module updates

Hi, I had problems getting Modules.TouchId to run. It would always throw an error when trying to call deviceCanAuthenticate:

message = "'undefined' is not a function (evaluating 'TiTouchId.deviceCanAuthenticate()')";

After trying all kinds of tests I had a look at the module code at github and noticed the version is 1.1.0, while version 1.0 is installed on my machine with Titanium SDK 3.4.1.GA installed. I compiled the module, selected the current version and the error went away, it's working as expected.

Modules.TouchId comes preinstalled with Titanium SDK, shouldn't updates to that module also come via Titanium updates? Or is this my own responsibility? Thanks.
