  • I haven't heard of anyone doing a native Android module for AdMob yet, though it certainly is possible. However you should be able to embed AdMob ads in a web view inside your application without a native extension with a bit of extra finagling on your part - you would have to put JavaScript inside the webview to let the native application know the ad had been clicked and what the target URL was. Your native app could then open a web browser or take any other appropriate action.

    There are multiple web view examples (including how to communicate between the web view and native JavaScript with custom events) in the Kitchen Sink.
