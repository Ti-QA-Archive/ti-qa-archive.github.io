Fixed with next to a fluid view

Say we have the following syntax:

<View id="main">
    <View id="sub1">
        <TextField id="tf" />
    </View>
    <View id="sub2">
    </View>
</View>

#main is 100% wide and has a vertical layout

#sub1 should float left

#sub2 should float right

How would you make #sub2 '30dp' wide and let #sub1 fill the remaining space on the left so that it completely fills the view sitting against #sub2?

In a landscape and portrait view #sub2 should have the same width (both '30dp'), sub1 should widen accordingly

Any tips?

    "#sub1":{
    left: '0dp',
    right: '40dp'
}
"#sub2":{
    width: '40dp',
    right: '0dp',
}

    So #sub1 fills up the rest of the window and #sub2 always has the exact same width :)

  • Hi

    The solution is easier than that;

    something.tss

    "#sub1": {
    left: "0dp",
    right: "30dp",
    width: Ti.UI.FILL
},
"#sub2": {
    right: "0dp",
    width: "30dp"
}

    This says sub1 should fill the entire width except a margin of 30dp on the right and anchored to the left side using 0dp. sub2 should only be 30dp and anchored to the right.

    DO NOT set layout to vertical.

    You can extend this technique out to manage three columns;

    "#left": {
    left: "0dp",
    width: "80dp"
},
"#center": {
    left: "80dp",
    right: "40dp",
    width: Ti.UI.FILL
},
"#right": {
    right: "0dp",
    width: "40dp"
}

    This creates three columns - the middle one expands to fill the space after the left and right sides are taken into account. It does this by setting the center columns left side to match the width of the left column and the right side width to match the width of the right column.

    This is a very flexible technique.
