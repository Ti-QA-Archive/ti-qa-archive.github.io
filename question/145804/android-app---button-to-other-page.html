  • If you are using the url property of a window, you don't need to open another window inside the window you just created. Use the Ti.UI.currentWindow to add components to the education file and remove the opening of that window as you will open the window from the main file.

    OR

    Use the commonjs approach and remove the url property completely. When you use the url property you open a new context which is often unnecessary.

      If all you want to do is change tabs, just set the activeTab on the tabgroup to the index of the tab you want to switch to. ie:

      // change to the second tab
tabGroup.setActiveTab(1);
