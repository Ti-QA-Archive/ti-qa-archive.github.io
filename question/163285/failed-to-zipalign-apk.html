    Hi nadia,

    try cleaning your project. From within Titanium Studio menu ->Project -> Clean.. (select your project and click OK)

    And it looks like you are using beta stuff Titanium SDK (\Titanium\mobilesdk\win32\3.2.2.v20140210131652)

    If you need beta, try to use a newer version. If not, go with 3.2.1GA.
