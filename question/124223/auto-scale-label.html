Auto-Scale Label

Hi,

If I have a label with a given width and height, is there a way to autoscale the text to make it as large as possible and still fit within the label's dimensions?

I see minimumSize, but that just doesn't seem to work properly. Is that how it is supposed to be done?

  • It is minimumFontSize, not minimumSize. When you specify a value for minimumFontSize, then if (and only if) the font size given in the font object is too large to fit the alloted space, the OS will reduce the fontsize until it fits. If it still does not fit with the font reduced to minimuimFontSize, then it uses that size anyway.

    So I guess if you want to always make as large as possible, use a large value for the font which should force it to reduce to the extent needed.

    var myLabel = Ti.UI.createLabel({
  left: 10,
  right: 10,
  height: 'auto',
  text: 'John Hancock',
  font: {fontSize: 50},
  minimumFontSize: 8
});
