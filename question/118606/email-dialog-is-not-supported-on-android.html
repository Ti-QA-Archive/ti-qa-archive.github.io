Email dialog is not supported on Android

I used some code:

 var emailDialog = Titanium.UI.createEmailDialog();
if (!emailDialog.isSupported()) {
    alert('not support');
}
else {
    //init object with empty value
    emailDialog.setSubject('dfdsfdsfdsf');
    emailDialog.setToRecipients(['ducnguyen@1112group.com']);
    emailDialog.setMessageBody('gfdgdfgdfgdfgfdg');
    //fill the email form after it open and click the send button.
    emailDialog.addEventListener('complete', function(e){
        //you can alert event handler, only e.source is a emaildialog object.
        //but e.source.messageBody is empty, e.source.subject is empty .... other attributes are empty too.
        if (e.result == emailDialog.SENT) {
            alert('send success');
        }
        else 
            if (e.result == emailDialog.FAILED) {
                alert('fail');
            }
            else 
                if (e.result == emailDialog.CANCELLED) {
                    alert('cancel email')
                }
    });
    emailDialog.open();
}

=> I run it, and result is: "not support".
Can i help you.

  • If you are running on a simulator, you must setup an Email account before trying it. If there is no valid Email account on the device/simulator, the EmailDialog won't be able to run.

    • Dunno if I am too late in the question "How do you setup an email account on an Android emulator?

      I have a quick guide, but with warnings !

      1. Don't use your personal or general development e-mail account; use a free account from GMail

      2. The emulator should retain your email settings, but if you nuke the virtual android device… the settings will naturally be gone.

      So, here we go…

      Launch your emulator and use the Launcher Icon to see your apps

      Select the E-Mail App and Launch It

      Enter the address and password

      You can manually edit your settings, but let Android try to auto config, first.

      Once configured, the Email app will try to verify everything

      Finally, the Email app will ask for a cute name and a first name

      And there you have it – you should be able to check your mail, etc, but more importantly: run your code using email
