Image auto height with fixed width problem

var win = Ti.UI.currentWindow;
var view = Titanium.UI.createView({
    backgroundColor:'blue',
    width:300,
    height:400,
});

var fig = Ti.UI.createImageView({
top:0,
left:0,
width:300,
height:'auto',
backgroundColor:'red',
image:'http://upload.wikimedia.org/wikipedia/commons/4/46/Greenland_scenery.jpg'});

view.add(fig);
win.add(view);

The original size of image is 2560x1446.
And I assume that the height of image will be 1446x(300/2560)=169.xx

but the log code result was diffent like this.

Ti.API.info("width="+fig.width+'/height='+fig.height);
[INFO] width=300/height=247

So, I use load event of image like this.

fig.addEventListener('load',function(){
    Ti.API.info("width="+fig.width+'/height='+fig.height);
    setTimeout(function(){
        Ti.API.info("width="+fig.width+'/height='+fig.height);
    },1000)
})
[INFO] width=300/height=247
[INFO] width=300/height=1446

It still not working.

What I have to do when I want to make image with fixed width?
