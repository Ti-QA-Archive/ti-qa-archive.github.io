showCamera on iOS - disable zoom?

Whenever I use showCamera with an overlay, there's a built-in zoom function if you pinch-gesture.

I cannot find any way to disable this.

Has anyone managed to disable the zoom?

  • Found it - just add a pinch event listener to the overlay, this overrides the built in camera pinch to zoom function

    overlay.addEventListener('pinch',function(e){
    console.log(e.scale);
});
