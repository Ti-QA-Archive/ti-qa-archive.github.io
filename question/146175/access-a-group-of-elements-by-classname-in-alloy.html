Access a group of elements by classname in Alloy?

Is it possible (and how?) to access a group of view elements by their classname? I have a group of elements in a view like this

<Label class="quick">Quick 1</Label>
<Label class="quick">Quick 2</Label>
<Label class="quick">Quick 3</Label>

And I want to attach a similar event listener to each of them. But this isn't valid in Alloy:

$('.quick').addEventListener('click', function(){
    alert(this.text);
});

  • A workaround I have used is to set className or other property to the class style in the .tss file which essentially adds that property to all views with that class and then loop through a view or windows children matching the className. Underscore makes it even easier.

    • _.each($.container.children, function(child) {
    if (child.className === 'btn') {
        child.addEventListener('click', function() {
            // do something
        });
    }
});
