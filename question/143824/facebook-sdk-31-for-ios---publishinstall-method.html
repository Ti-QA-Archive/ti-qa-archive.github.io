Facebook SDK 3.1 for iOS - publishInstall method

I am interested in Facebook's new Mobile Ad offering, especially their ability to try and help target Cost Per Install. In order to get that tracking correctly, their Facebook SDK has to be integrated and they mention calling the function:

[FBSettings publishInstall:appId];

It seems like Titanium is using a modified version of the Facebook iOS SDK, but this function isn't available. Is there a module to incorporate the additional 3.1 functions? Would that have any conflicts with the version of Facebook that is used by Titanium?
