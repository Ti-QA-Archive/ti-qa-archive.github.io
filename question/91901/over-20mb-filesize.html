  • Do you share/submit your application through XCode?
    The build files are rather large, but creating the IPA file through XCode creates smaller files with only neccessary data.

    If you build Android apps, the apk files when signed are also rather small. Create them using Distribute with a proper keystore.

    The builds are downright huge tho.
