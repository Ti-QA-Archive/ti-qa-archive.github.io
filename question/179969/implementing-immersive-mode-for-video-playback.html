  • Hi all,

    I just completed a small Titanium module for create a sticky immersive view. I put it also on Titanium market, but is still waiting for approval.

    You can download it from here (documentation included): www.hinter-software.com/titanium/com.hinter.immersive-android-1.0.0.zip.

    USAGE:

var immersive = require('com.hinter.immersive');
var immersive_view = immersive.createExample(); 

var win = Titanium.UI.createWindow({
    backgroundColor: 'black',
    exitOnClose: false,
    orientationModes:[Titanium.UI.LANDSCAPE_LEFT, Ti.UI.LANDSCAPE_RIGHT],
    fullscreen: true,
    navBarHidden:true,
    customData: 0
});
win.add(immersive_view);

    Hope it helps!
