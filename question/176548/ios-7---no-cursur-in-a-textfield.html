[iOS 7] - No cursur in a textfield

I'm using this code to display some textfields with custom styling. For some reason, iOS doesn't show any cursor in the textfield. Any ideas what is going on?

    pc_label = Ti.UI.createLabel 
        top: 70
        left: "10%"
        height: 40
        width: "80%"
        text: L("postcode", "Postcode:")
        color: settings.colours.navbar 
        font:    
            fontFamily: settings.font()
            fontSize: 16

    pc_input_num = Ti.UI.createTextField 
        top: 110
        left: "10%"
        width: "50%"
        height: 40
        borderWidth: 1
        borderColor: "#999"
        paddingLeft: 10
        paddingRight: 10
        color: "#666" 
        maxLength: 4
        keyboardType: Ti.UI.KEYBOARD_NUMBER_PAD
        font:    
            fontFamily: settings.font()
            fontSize: 16

    Hi Fabian,

    on iOS the cursor color is determined by the "tintColor" property of the textfield, i think you have to choose the right color, then the cursor will be visible again.
