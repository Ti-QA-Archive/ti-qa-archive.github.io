  • Hey Mario.

    I had the same problem, but I found the solution. In my case the problem was this:

    Failure [INSTALL_FAILED_MISSING_SHARED_LIBRARY]

    I googled sometime and found this link: https://github.com/dbankier/TiShadow/issues/272

    So the solution was to have the Google API in my emulator… I was using the HAX emulator for x86, that doen's come with Google API. I changed it to ARM, and I was able to choose the Google API version (this can be done in the AVD manager).

    Now it works.

    Hope that helps.
