How to populate listSection so it fills up the available space

This is working but now, because there are not enough listSection items to fill the screen, I can see the background image appearing below the last item.

Is there a way to make a listSection fill with items so that it adds as many needed to fill up the screen so that the background image does not show through?

I want the background image so if a user swipes the list down fom the top they will see the background image and not a listView backgroundColor.
