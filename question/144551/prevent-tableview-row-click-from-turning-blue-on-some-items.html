    You can use the selectionStyle property for the row in iPhone:

    selectionStyle: Titanium.UI.iPhone.TableViewCellSelectionStyle.NONE
    — answered November 9th 2012 by Matt Berg
    permalink
    1 Comment

    • Keep in mind that the above constant is an iOS only value and will cause errors on Android and Mobile Web. Wrap the code in some conditional logic. For those platforms, you can set the backgroundSelectedColor property instead.
