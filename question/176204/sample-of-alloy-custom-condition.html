Sample of Alloy custom condition

The official doc:
http://docs.appcelerator.com/titanium/3.0/#!/guide/Alloy_XML_Markup-section-35621528_AlloyXMLMarkup-ConditionalCode
mentions that we could use custom "if" attribute to conditionally display UI element based on global statement.

I have the following test condition in alloy.js

Alloy.Globals.own_something = 1;

In a view:

...
<TableViewRow if='Alloy.Globals.own_something' title="Yes I own it"/>
...

The problem is, no matter the globally Boolean value is 0 or 1, true or false, the TableViewRow always shows up. tried if='own_something', same result.

Can't find any working sample so far. What's the correct way to make it work?
