Touble Tap to zoom into a Scrollview (centerd)

Hey guys,

i am building a "huge" Imageslider application, so what do we have:

I've came across with a simple Scrollview, containing all my images ordered in a grid - at the moment (performance tests) we are talking about 25x25 images (each has width:250, height:150)

"#scrollContainer":{
    contentWidth:'auto',
    contentHeight:'auto',
    width:Ti.UI.SIZE,
    height:Ti.Platform.displayCaps.platformHeight,
    top:0,
    left:0,
    disableBounce:true,
    verticalBounce: false,

   minZoomScale:0.2, 
   maxZoomScale:1, 
   //zoomScale:0.3 
}

These are my style settings for the scrollview.

My Problem now is, that i wanna "Zoom in" per toubletab to the tapped area.

At the moment i am using this function, to zoom in after a double tap:

$.scrollContainer.addEventListener('doubletap', function(e) {

    e.source.setZoomScale(1, {
        animated : true
    });
});

This works fine, but sadly it does not zoom based on the tapped point but based on the whole scrollview.

I'm a bit frustrated at the moment cause i have no idea how to solve this. Do i have to move the whole scrollview around?
Help or hints would be great :)

Best,
Nico

ios7, newest SDK

  • Hello Nico Barelmann,

    I have tested your code. In event handler code you set zoomScale 1 that’s means zoom not applied in your scroll view. If you are want to zoom your scroll view then you need to set zoomScale at least 1.1 or greater then 1 or lest then 1 for zoom out.

    Here I have added my test code. Please check this

    index.js

    $.win.open();
 $.scrollContainer.addEventListener('doubletap', function(e) {

    e.source.setZoomScale((1.1 * e.source.zoomScale), {
        animated : true
    });
});

    index.tss

    "#scrollContainer":{
    contentWidth:'auto',
    contentHeight:'auto',
    width:200,
    height:200,

    disableBounce:true,
    verticalBounce: false,
 backgroundColor: '#000',
   minZoomScale:0.2, 
   maxZoomScale:5, 
   //zoomScale:0.3 
}

    index.xml

    <Alloy>
    <Window id="win" backgroundColor="white" exitOnClose="true" fullscreen="false" title="ScrollView Demo">
        <ScrollView id="scrollContainer" >
            <Label backgroundColor="#fff"> Milon</Label>
        </ScrollView>
    </Window>
</Alloy>
