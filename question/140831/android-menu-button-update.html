Android Menu Button Update

In my App I have a place where you can add items to a shopping cart. To acces the cart you press the menu button on Android and the option comes up to "View Cart (9)". The number 9 being the number of items currently in the cart. This works great when I load the window the first time, but when I push my add to cart button and add an item I want it to update this count on the menu option. I can't seem to figure out how to update the text/title of a menu option after the page has been rendered.

Here is the code I am using to render the menu option when the window opens:

win.activity.onCreateOptionsMenu = function(e){
    var cartCount = getCartCount();
    var menu = e.menu;
    var menuItem = menu.add({ title: "View Cart ("+cartCount+")" });
    menuItem.addEventListener("click", function(e) {
        Ti.App.fireEvent('openCart', {});
    });
}

Any suggestions?

    Hi David

    The documentation for Titanium.Android.Menu has a great example of how to show/hide a menu item, from this you can see how changing the properties of a menu item would work.

    Instead of playing with setVisible() in the exmaple use setText() instead.
