Use emoji character in label

Is there a way to use an emoji character in a label?

    Hi B,

    yes it's possible.

    This example is for iOS with unicode:

    var win = Ti.UI.createWindow({
    backgroundColor : '#FFFFFF'
});

var label = Ti.UI.createLabel({
    text:'text with ' + '\uE00E' //thumbs up
});

win.add(label);
win.open();

    For more info read Use Unicode Emoji as Image-Free Icons in iOS Web Apps and Use Unicode Emoji as Icons in Native iOS Apps
