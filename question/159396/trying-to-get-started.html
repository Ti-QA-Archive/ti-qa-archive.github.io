  • with the diagnostic check i found this

    USER=frie
    PWD=/Applications/Titanium Studio/TitaniumStudio.app/Contents/MacOS
    HOME=/Users/frie
    LOGNAME=frie

    in Applications/Titanium Studio/TitaniumStudio.app/Contents/MacOS i found a .ini file and a unix exec file this file has encryption so can not be changed .
