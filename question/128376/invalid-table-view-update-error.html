Invalid table view update ERROR

On iPad2 with iOS5 and Ti SDK 1.7.5 I am having issues with the searchBar, only sometimes. Basically, my modal window opens up with a a tableView that has a search. When I close the modal window, and the search value stays null, the window closes successfully. When I close the modal window, and the search value isn't null, the window crashes with the following error.

*** Terminating app due to uncaught exception 'NSInternalInconsistencyException', reason: 'Invalid table view update. The application has requested an update to the table view that is inconsistent with the state provided by the data source.'
*** First throw

Below is my code, it's simple and straight forward and is pissing me off! :)

searchWhoAreYouVisiting = Ti.UI.createSearchBar({
            barColor:$$._ReflexBlue,
            showCancel:false,
            hintText:'TAP TO BEGIN',
            height:43,
            top:0
        }),
        whoAreYouVisitingTableView = Ti.UI.createTableView({
            width:680,
            height:43,
            top:203,
            borderWidth:2,
            borderColor:$$._ReflexBlue,
            search:searchWhoAreYouVisiting,
            backgroundColor:'#FFF',
            _ARG:'BLUR',
            data:$$._Employees,
            searchHidden:false
        }),
