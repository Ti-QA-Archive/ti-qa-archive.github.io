  • Hi Andres Guerrero,

    It really sounds like the problem is that the deploy.json is not removed from the device's SD card. Could you try manually removing it?

    • Connect device
    • Run "adb shell" from command line
    • Run "cd /sdcard"
    • Run "ls"
    • Locate a directory with your app's id
    • Run "cd <app id>"
    • Run "ls"
    • If a "deploy.json" file exists, delete it by running "rm deploy.json". If it does not exist, then the app should launch just fine.

    Also check that your minimum and target SDK versions are installed or not. If not, then install required SDK. Don't use the Android Virtual Device with too low configuration.

    Hope this will help you to solve your problem.
