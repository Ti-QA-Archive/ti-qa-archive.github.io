    The answer Paul gave should be correct but you also need to take into account;

    • Close the simulator and use project > clean and select you project.
    • you appear to have created a custom plist, if you have and only to achieve this, back it up and remove it before the first step
    • If your custom plist is used for a good reason I would consider re-creating (with your changes) it after the first step but back it up and delete it before you run the first step - this may cause a naming conflict
