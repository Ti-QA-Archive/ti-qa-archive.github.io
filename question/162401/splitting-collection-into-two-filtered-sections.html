Splitting collection into two filtered sections

In Titanium Alloy (SDK 3.2.0) for iOS, what is the best way to display a collection with filtered results at the top and the rest below, similar to stickied threads in a forum?

For now I just have two identical TableViews, both referencing the same collection but with different dataFilter functions. The two functions simply perform a where() function against a single variable to see if it is 1 or 0.

Here is a very simplified snippet of the markup and js:

<TableView dataCollection="myCollection" dataFilter="filterTop">
    <TableViewRow></TableViewRow>
</TableView>

<TableView dataCollection="myCollection" dataFilter="filterRest">
    <TableViewRow></TableViewRow>
</TableView>

function filterTop(collection) {
    return collection.where({ someVariable: 1 });
}

function filterRest(collection) {
    return collection.where({ someVariable: 0 });
}

However, would it be possible to just have one TableView and put the filtered rows into two TableViewSections? Would this be better than two filtered TableViews?
