Insert ASCII "service mark" (or any ascii symbol) into a label

How do I assign ASCII characters into a label text property? I tried HTML &amp;#8480; and UTF U+2120. Neither worked. I tried to escape the UTF like this \U+2120 (error) and \\U+2120, but it rendered as \U+2120.

Thanks.

    • Just a note. For a brief time, an update broke the unicode interpreter in TiSDK. So, if you have source code with \u2120, it would work on iOS, but display as text on android because android expects \U2120.

      This has been fixed. If you experience this problem, try updating your sdk.
