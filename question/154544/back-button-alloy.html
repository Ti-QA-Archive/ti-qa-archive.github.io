Back button alloy

Hi,

I want to implement a back button in alloy.While clicking on back button previous page should be displayed.I have added code for back button as

function back()

{

Alloy.createController('index').getView().open();

}

here index page is previous page

Problem I am having index page contains paginated data and calling this function is reloading page.
I tried to close current window so previous page will come into focus.
But my worry is when memory become tight whether system will remove previous page so that I will end up with blank page…?

Is there any other way to do it better?

Thanks

Vivek P

    You should have a look at the NavigationGroup. When you open Windows with a NavigationGroup there will be a system-provided back button in the navigation bar.

    The system won't remove a page if you don't close it. That means, you don't have to reload data when you come back to the index window. At any rate you should not create a new instance of your index controller without removing the other one. This will be even worse for memory consideration. Closing the current window is the common way to get back to the previous one.
