  • Hi

    Your folder structure should be like this, I have provided the structure I use and example code:

    // location for i18n folder
{appname}/i18n

// location for language folders
{appname}/i18n/en

// I have two files inside
{appname}/i18n/en/app.xml //(file 1)

// example
<?xml version="1.0" encoding="UTF-8"?>
<resources>
    <string name="appname">Example App Name</string>
</resources>

{appname}/i18n/en/strings.xml //(file 2)

// example
<?xml version="1.0" encoding="UTF-8"?>
<resources>
    <!-- app -->
    <string name="appname">Defence Discount Service</string>

    <!-- buttons -->
    <string name="confirm">Confirm</string>
    <string name="cancel">Cancel</string>
</resources>

    Hope this helps
