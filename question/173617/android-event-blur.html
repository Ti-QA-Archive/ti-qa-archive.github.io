  • Hi

    If you only have one text box then it is likely that the blur event will not function as nothing can accept the focus from the text box.

    In what way is this nested stuff helping or hindering?

    loadAudit.inputTxt[x].

    Create a basic plain text box outside of all your normal logic and add it to the first window if your app with an event listener just for it.

    var txt = Ti.UI.createTextField({
    value: 'test'
});
txt.addEventListener('blur',function(){
    Ti.API.log("textbox event fired question id : " + this.id); 
});
// adjust win to suit
win.add(txt);
