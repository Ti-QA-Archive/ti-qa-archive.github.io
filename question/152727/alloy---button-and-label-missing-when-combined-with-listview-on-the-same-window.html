  • Full list view support won't be available until Alloy 1.2.0. Only the <ListView> tag works now. If you want to try out the cutting edge (not fully tested) version, you can get it directly from github via npm:

    sudo npm install -g git://github.com/appcelerator/alloy.git

    you can go back to any version you want at any time like this:

    sudo npm install -g alloy@1.1.2
