    I see in your code, you use tab but you miss TabGroup. Tab is an instance of TabGroup. Without TabGroup tab is now working so you need to add Tabgroup in your code. Please check my following code. I hope it solve your problem.

    <TabGroup id="tabGroup">
        <Tab id="summaryTab" title="Summary">
            <Window id="summaryWindow" title="My TeamCard" class="container" layout="vertical"  >
                <RightNavButton>
                    <Button id="btnLogout" class="logout"   />

                            <Label id="availablePoints" bindId="availablePoints" class="value_text greenText" />

                        <ItemTemplate name="tmpPendingPoints">
                            <Label id="pendingPoints" bindId="pendingPoints" class="value_text blueText" />
                        </ItemTemplate>
