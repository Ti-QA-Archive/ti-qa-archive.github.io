Ti.Blob image width and height issues

Hi guys,

I'm receiving an image through a web service base64 encoded. The based64 encoded string is correctly decoded as a Ti.Blob, and it's displayed correctly in a imageView.

The images that I get are of various dimensions and I need to get their width and height and manipulate them so I can preserve the aspect ratio.

The problem is that the width and height of the Ti.Blob object are 0 and the mimeType property of that same object is application&#x2F;octet-stream. I assume that if the mimeType would be something like image/png, then everything should work.

Is there a way to set the mimeType? Or workaround to get the correct image width and height?

Thanks!

    hi Lucian,

    try to write your image in applicaitonDataDirectory and get back by Filesystem

    then do this 

    var f = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory,<image name>);
var test = f.read();
alert(test.toBlob().width + ' >> ' + test.toBlob().height);
