    Hi Jude,

    Before you click Distribute to Apple iTune in Titanium. Go to your Apple Developer account. Click on Provisioning. Select the Provisioning profile for your application. Select edit and click modify. Then when it shows the Provisioning details, select the Distribution tab.

    Create a Distribution profile and select the application app id. Download the Distribution profile and drag and drop it into Xcode. In Titanium when you click Distribute to Apple iTune, select the Distribution profile instead.

    Let me know if you get it right :)
