Referencing wrapper class from native UI control does not work on iOS

Hello everyone.

I am having a problem in creating wrapper classes. My class looks roughly like this:

function RC_Label(id, uitype, left, top, width, height){

var label = Ti.UI.createLabel({
    color:'black',     

    id:id,
     uitype:uitype,
    left:left,             
    top:top,
    width:width,
    height:height
});

label.wrapper = this;

…

module.exports = RC_Label;

}

I want to have a reference from label (native control) to RC_Label because I made some custom methods I want to use, that is why I wrote label.wrapper = this;

The problem is that on iOS, when I try to fetch the wrapper from the native label, I receive an empty object, and when I JSON.stringify() it it looks like this:

{}

This only happens on iOS, everything works fine on Android. Thanks a lot in advance for your help.

  • Extending host Titanium objects is not a good idea. Instead, use native JS object to store a reference to Ti object:

    this.wrapper = label;
