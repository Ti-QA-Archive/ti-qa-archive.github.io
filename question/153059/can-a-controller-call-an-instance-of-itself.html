Can a Controller call an instance of itself?

I have a navigation group that drills down several layers. The logic that builds the data array for each level is all in one if/else statement.

My question is this…

Can a controller call instances of itself until it satisfies a condition and then call another controller?

Here is a sample of what I'm trying to do

level_one.js

$.some_table.addEventListener('click', function(e)
    {
    // if there is a category id then we go onto the second level controller to show files.
    if(e.rowData.category_id){
        var doc_dep = Alloy.createController('some_folder/level_two',e);
        $.some_navigation_group.open(doc_dep.getView(),{animated:true});    
    }
    // else we need to show another level of menu
    else{
        var doc_dep = Alloy.createController('some_folder/level_one',e);
        $.some_navigation_group.open(doc_dep.getView(),{animated:true});    
    }

    }); // end event listener

The first portion of the if works properly.

When the else fires I get this error:
Script Error = 'undefined' is not an object (evaluating '$.some_navigation_group.open') at level_one.js (line 247).

Any guidance or suggestions would be greatly appreciated.
