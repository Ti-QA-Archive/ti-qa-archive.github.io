Compiling and launching is extremely slow

Hi, I'm trying to develop applications for both iPhone and Android. But I find the launcing procedure extremely slow. I work on both Mac and Windows and have the same problem in both envionments. When hitting "Launch", it takes about 1:20 minutes until I actually see the application running. So everytime I make a simple change in my code, I will have to wait for over a minute. Sometimes I want to make a small change in the code and just preview.

Find it hard to believe that developers are actually working like this. There must be any other way to run the application than in an emulator?

  • Accepted Answer

    for Android, use fastDev:

    http://wiki.appcelerator.org/display/guides/Fastdev+Reference+for+Android

    for iOs, a clean build can take a minute, but following builds shouldn't take more than 10-20 seconds (depending on machine and project complexity).

    If you use internationalization is even worst.
