  • Hi Alex, you can use custom property with window object, like for example:

    Example:

    
var tab2 = Ti.UI.createTab({
    title: L('settings'),
    icon: '/images/KS_nav_views.png',
    window: win2
});

win2.containingTab = tab2;
win2.firstWindow = win1;   // custom property

    now you are having access to win1 function and properties in win2. without use of app level variable or Titanium object customization

    as it is suggested to avoid this.
