WebSocket functionality for Titanium?

I am trying to integrate Firebase into Titanium, and it uses the browser's WebSocket to communicate with the server. That works fine in mobileweb, but doesn't work at all in iOS or Android.

I've tried ti-websocket-client without success:

[DEBUG] application booted in 219.605982 ms

[DEBUG] Queuing up my write!

[DEBUG] mismatch protocol version

[ERROR] While executing Timer, received script error. 'Can't find variable: document at firebase.js (line 49)'

I'm guessing that the Firebase server doesn't support the hybi-07 protocol, which is the only one supported by ti-websocket-client.

Any other suggestions on WebSocket support in Titanium?
