How to Change Titanium SDK Home

Is it possible to change (non-interactively, ie. via scripts) the location where the Titanium SDKs are searched for using the Titanium CLI?

I know you can change it in 3.2.0 interactively via ti setup > sdk, and it'll prompt (#2) to specify another location. However, It would be nice to know if there is a solution that involves no prompts. The –help only specifies way to change which SDK we're on.

— asked January 23rd 2014 by Matthew Tondreau

  • I know you can change it in 3.2.0 interactively via ti setup > sdk, and it'll prompt (#2) to specify another location.

    It doesn't actually change the location, simply the version of the sdk in the default location.

    You can use -s on the command line, but this ONLY changed the SDK used by the CLI to bootstrap. The tiapp.xml still dictates which sdk is used for the building of the app.
