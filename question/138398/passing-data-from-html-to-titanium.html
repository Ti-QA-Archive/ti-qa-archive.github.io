Passing data from HTML to Titanium

Hello-

I am working on a mobile application with a webview that has an HTML file opened. This HTML file has google maps api JS. I want to be able to pass the location of the place where user clicks on map from this HTML file to my Titanium map controller class. I have been using the code specified in

http://developer.appcelerator.com/question/118693/addeventlistener-on-webview-stops-hyperlinks–onclick-events

However I have not been able to pass the data from HTML back to my application. Here is the HTML file code

<html>
    <head>

        <meta name="viewport"
        content="width=device-width, initial-scale=1.0, user-scalable=no">
        <meta charset="UTF-8">
        <style type="text/css">
            html, body, #map_canvas {
                margin: 0;
                padding: 0;
                height: 100%;
            }
        </style>
        <script type="text/javascript"
        src="https://maps.googleapis.com/maps/api/js?sensor=false"></script>

        <script type="text/javascript">
            var map;

            function initialize() {
                var myOptions = {
                    zoom : 8,
                    center : new google.maps.LatLng(37.5, -77.3),
                    mapTypeId : google.maps.MapTypeId.ROADMAP
                };
                map = new google.maps.Map(document.getElementById('map_canvas'), myOptions);

                google.maps.event.addListener(map, 'mousedown', function(e) {
                    //alert('clicked');
                    Ti.App.fireEvent('mapclicked');
                });
            }

            //alert(m);

            google.maps.event.addDomListener(window, 'load', initialize);
        </script>

    </head>
    <body>
        <div id="map_canvas"></div>
    </body>
</html>

And here is my controller class code:

    Ti.App.addEventListener('mapclicked', function() {
            // Pass data to the webView
            Ti.API.info("yes");
            alert("yes");
        });

I dont see an alert box with meesage "yes" when I click the map. Is there anything I am missing?
We really need this functionality and any help will be really great!

Thanks
