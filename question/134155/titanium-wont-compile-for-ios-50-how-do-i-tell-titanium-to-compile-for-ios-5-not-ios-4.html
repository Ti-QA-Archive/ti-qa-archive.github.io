Titanium won't compile for iOS 5.0. How do I tell Titanium to compile for iOS 5 not iOS 4?

I've got Titanium SDK 1.8.2. I've got XCode 4.2 on Leopard. But Titanium won't compile for iOS 5.0. How do I tell Titanium to compile for iOS 5 not iOS 4? (Titanium builds an xcode project into the Resources/build/iphone directory that specifies 4.0 as the target!). I want the target to be 5.0 so I can deploy my app to my iPad running iOS 5.0

— asked March 20th 2012 by Scott Williams

  • If you develop with Titanium Studio, open your project and select menu below.

    Project -> Properties -> Run/Debug Settings -> choose your project iPhone/iPad simulator -> Edit button -> Main tab -> iOS SDK -> select 5.0.
