  • Photo requires to have a photo, therefore you should go with customeObjects + photos.
    And if you are using voting you should consider Reviews(http://docs.appcelerator.com/cloud/latest/#!/api/Reviews).

    For query on custom fields you can read docs on http://docs.appcelerator.com/cloud/latest/#!/guide/search_query.
