  • Eugen

    It's really important to state your mobile OS version and Titanium version, whenever you post a question, otherwise people are only able to make random guesses with their suggestions. The best place for this information is in your question tags.

    Indeed, your usecase is good, so sorry if I sounded harsh before.

    It seems like a bug, because the code runs fine on Android. The approach I would take is to remove properties incrementally until you find the ones that cause the issue.

    Start with this:

    var window = Titanium.UI.createWindow();
Titanium.UI.setBackgroundColor('black');
var data = [
{title:'Row 1', color:'red'},
{title:'Row 2', color:'green'},
{title:'Row 3', color:'blue'},
{title:'Row 4', color:'orange'}
];

var tableview = Titanium.UI.createTableView({data:data});

window.add(tableview);
window.open();

    Then, although it's a long shot, change the colors to their hex values, ie:

    {title:'Row 1', color:'#f00'},

    Once you have found the cause, please report back.

    Cheers
