Fetch single data from database and push to multiple labels?

Hi guys,

is it in any way possible to fetch words from a single textfield from the database and push them to multiple separate labels?

For instance, let's say I would enter these three words in to a single textfield in a database, separated with a comma:

iPhone,titanium,development

I want these three words to be pushed to three separate labels. Like this:

Ti.UI.createLabel({
    text: //iPhone goes here
}); 

Ti.UI.createLabel({
    text: //titanium goes here
}); 

Ti.UI.createLabel({
    text: //development goes here
});

Anyone any solution for this?

If possible, would it also be possible to have not three but a lot more words in a single textfield, but only have the three first words created in to labels?

Thanks!

    Hi

    You can use the standard JavaScript command split, example;

    var str = 'iPhone,titanium,development';
var keywords = str.split(',');
alert(keywords[2]);

    This will show the third item (index = 0 to 2).

    • Using your own example;

      var str = 'iPhone,titanium,development';
var keywords = str.split(',');

if (keywords.length >= 1) {
    Ti.UI.createLabel({
        text: keywords[0]
    }); 
}
if (keywords.length >= 2) {
    Ti.UI.createLabel({
        text: keywords[1]
    }); 
}
if (keywords.length >= 3) {
    Ti.UI.createLabel({
        text: keywords[2]
    }); 
}

      This will show the first 3 keywords AS long as they exist.

      Your str could have contained 50 keywords, this will only work for the first 3.
