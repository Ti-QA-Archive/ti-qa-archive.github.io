1.8.2 - Android - Problems with multiple EventListener on Same event

Very simple use case :


function func1() {
    Ti.API.info("in func1");
}


function func2() {
    Ti.API.info("in func2");
}

Ti.App.addEventListener("toto",func1)
Ti.App.addEventListener("toto",func2)

Ti.App.removeEventListener("toto",func1)

This will result in this error popping up:

E/TiJSError(29824): (main) [179,537] ----- Titanium Javascript Runtime Error -----
E/TiJSError(29824): (main) [0,537] - In ti:/events.js:241,23
E/TiJSError(29824): (main) [0,537] - Message: Uncaught TypeError: Cannot read property '0' of undefined
E/TiJSError(29824): (main) [0,537] - Source:     
    if (list.listener[i] === listener ||
E/V8Exception(29824): Exception occurred at ti:/events.js:241: Uncaught TypeError: Cannot read property '0' of undefined

When there is only one event listener set up, all is good.

Sounds like a bug to me.

1.8.2 - tested on Rhino and V8
Android 4.0.3

Thanks.
