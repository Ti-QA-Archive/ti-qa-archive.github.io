Going from Android to iPhone

Since I only have a Windows box (right now) I built my first app on Android. I'm happy to say its being successfully used on the Android Market (see: http://goo.gl/ZBGLz if interested). I'm going to be getting a Mac soon but wanted to get a head start preparing my app for iPhone. Any recommendations? What things should I look out for in my app? Besides basic functionality are there other aspects that might make the app not get accepted by Apple?

Any thoughts/comments/suggestions are greatly appreciated.

  • You will be amazed simply how excellent and wonderful the iPhone simulator works compared to the Android emulator. Using the sluggish Android emulator it was somewhat a relief to test on the real device.

    The iPhone, especially the 3G is slower than the simulator, so it is important to test on your phone frequently.

    One piece of advice which i have never seen written somewhere, is to get yourself an iPhone and start using it. There are considerable differences in the user interface with no menus, or back button on the iPhone.

    I have never used iPhone before starting to develop to it, and I did many mistakes thinking Android in the beginning. iPhone users are familiar with their device, and adding Android lookalike functionality is not the way to go i feel.

    And also, be prepared to have Titanium pointing out a massive amount of Javascript errors on your iPhone project on the Mac, compared to a Android project on a PC.

    Why it is like this, I really cant explain, but I believe the compiler to objective C for iPhone needs more correct Javascript than the Javascript to Java compiler for Android.

    There are also quite a few differences between iPhone and Android, so expect to check if your code run on Android and have different code quite a few times if you develop one project for both platforms.

    Beeing an Android user for years, I tend to like iPhone more and more… and developing for iPhone I also find easier.

    Well this is my 5 cents of opinions anyway,

    Good Luck!

    Harald
