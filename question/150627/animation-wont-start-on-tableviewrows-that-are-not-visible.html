animation won't start on tableViewRows that are not visible

Hey guys,

I'm experiencing something very strange. When I build a tableView with many cells and add animations to each cell, the cells that are currently not visible (as in "you would have to scroll to see them") won't animate.

I guess this is iOS memory management, but still a bummer. Is there a way around that?

best, Achim

  • Sure. Create an empty window, add the tableView. Only the first labels will be animated red.

    var data = [];
for (var i=0; i<10; i++){
    var r = Ti.UI.createTableViewRow({
        height:100
    });
    var label = Ti.UI.createLabel({
        text: 'Animated teststring'
    });
    r.add(label);
    label.animate({color:'#ff0000', duration:300});
    data.push(r);
}

var tableView = Ti.UI.createTableView({
    data:data
});
