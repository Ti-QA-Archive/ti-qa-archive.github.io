  • Hi Alex, i've checked that the alloy module didn't send the variable "theme" when it's trying to compile. So, the workaround that i found was:

    sudo vim /usr/local/lib/node_modules/alloy/Alloy/commands/compile/index.js

    locate the line 572: 

    var themeGlobal = path.join(appPath,'themes',theme,CONST.DIR.STYLE,CONST.GLOBAL_STYLE);

    add an IF to validade theme like this:

    if(theme)
var themeGlobal = path.join(appPath,'themes',theme,CONST.DIR.STYLE,CONST.GLOBAL_STYLE);

    Try to compile your application and smile!
