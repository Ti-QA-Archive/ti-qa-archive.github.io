  • Hi

    Ray is correct this is a reserved word, here is a full list;

    • Reserved Words

    However if you REALLY must use this word then for EVERY single occurrence that is a table (or column if you also use it for columns) enclose the reserved word in square brackets so SQLite can use it as a caption and not get it confused.

    INSERT INTO order (itemnumber) VALUES ("123456")

    Becomes;

    INSERT INTO [order] (itemnumber) VALUES ("123456")
