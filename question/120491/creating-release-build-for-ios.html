I am ready to release an app. I have done many dev builds, so now I choose "Distribute - App Store" in the App Explorer.

[DEBUG] Detecting modules in /Library/Application Support/Titanium/modules
[INFO] Performing clean build

Then nothing at all. What is supposed to happen next?

  • I think the output is a bit cryptic as it seems to indicate that it is doing a build but that means that the build has finished. I have been using this method to do ad hoc distributions for a while now and it works OK for me.

    You should find the .app file in the dir that you specified in the Distribute dialog. You can drag and drop this straight onto iTunes for installation onto the phone ( as long as you have also installed the .mobileprovision file into iTunes as well).
