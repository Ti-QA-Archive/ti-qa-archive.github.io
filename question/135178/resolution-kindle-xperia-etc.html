    Hi Eduardo, For knowledge purpose I wrote this answer, as Ivars says that use dp values is the best choice now but knowledge base is most important thing that why I choose this answer,

    There are 10 folders in the images in Adroid folder. each have its owen images for images folder in resource, its serving all type of resolution image for an image placed in resources > images folder. this concept for high, medium and low resolution screen.

    for example:

    If you have one image image.png then all folder contains image.png file. 

    res-long-land-hdpi 
    >image.png
res-long-land-idpi 
    >image.png
res-long-port-hdpi 
    >image.png
res-long-port-idpi 
    >image.png
res-notlong-land-hdpi
    >images.png
res-notlong-land-mdpi
    >images.png
res-notlong-land-ldpi
    >images.png
res-notlong-port-hdpi
    >images.png
res-notlong-port-mdpi
    >images.png
res-notlong-port-ldpi
    >images.png

    Each image differ from another one in sense of resolution. It automatically renders in appropriate resolution in device to device. For more help you can refer this link

    otherwise you can create three folder inside android\images directory.

    android > images 
>high
>medium
>low

    these three folder contains same images for images folder in resource with different resolution. for more this link helps you.
