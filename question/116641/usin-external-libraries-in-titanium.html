usin external libraries in Titanium

Is it possible to use external libraries in Titanium when developing mobile applications?

For example I have a compiled custom library that I would normally use in Eclipse Android project to call this library's APIs. Is it possible to do the same thing in Titanium?

  • It is possible, but you have to write your own module for this library.

    The module gives you the ability to call the "compiled custom library"-functions within JavaScript. Have a closer look at the module guide in the docs section.

    Basically you have to forward your parameters from the module to the compiled library like this:

    @Kroll.method
    public void myJavaScriptMethodName(KrollDict options){
        super.handleCreationDict(options);
                customLib.myMethodName(options);
       }

    afterwards you will be able to call the function within your app.js

    myModule.myJavaScriptMethodName('foo');
