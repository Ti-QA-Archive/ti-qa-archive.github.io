Capture user idle in iOS

I want to logout a user if user stays idle longer that period of time. As far as i know titanium do not expose functionality to capture this. These are the options I encountered so far,

  • Capture global touch events (which doesn't work )

    Ti.App.addEventListener('click', function(e) {
 alert('Touch!');
});

  • Using timer and resetting on user interaction (click / touch / swap etc …)

Does anyone found a way to do this, or any suggestions would be appreciated.
