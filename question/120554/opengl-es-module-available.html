  • I wouldn't consider Titanium for writing any real game except the most basic. You should look into frameworks like Corona, Cocos2d or Unity3D if you're serious about writing a decent game on iOS/Android.

    Titanium is great for building data entry/multimedia apps that tap the existing API's but as I've written a game using OpenGL directly in Obj-C on the iPhone I wouldn't see any benefit of bothering to hook up under Titanium..

    If you're after JS for scripting a game, use Unity3d.
