Picker is not getting populated by data in rows

I am simply trying to populate a picker with data from a database. I can see the data coming into my console through Ti.API.info() call. I cannot though get it into my picker on my UI.

Here is my code

var picker = Ti.UI.createPicker({
top: 43
});

picker.selectionIndicator = true;

rows var here is getting the result set from my query which is not listed.

var rowCount = rows.rowCount;
var picker_data = [];

Ti.API.info(picker_data);
This brings me back all my data in my database call.

for(var i = 0; i < rowCount;i++) {
picker_data.push(rows.fieldByName('countryName'));
rows.next();
};

picker.add(picker_data);
picker_view.add(toolbar);
picker_view.add(picker);

When I simulate, the picker is empty however, I need my picker_data array data to fill up the picker.
Thoughts?

    The reason you don't see anything in your picker is because all you are doing is populating it with strings. (Which is not that the Picker object requires). What is required though, is PickerRow objects.

    So you will have to modify your loop so it creates a new PickerRow on each iteration.

    ####Like this:

    for(var i = 0; i < rowCount;i++) { 
    picker_data.push(Ti.UI.createPickerRow({ 
        title: rows.fieldByName('countryName') 
    }));
    rows.next();
}

    Hope this helps.
