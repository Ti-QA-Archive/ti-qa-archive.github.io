  • You can use the following code:

    annotations = Titanium.Map.createAnnotation({
    latitude:lat,
    longitude:lon,
    data:data
    title:'',
    subtitle:'',
    image:'../images/category1.png',
    animate:true,
});

    Here data property is custom which holds whatever value you give.

    • Both Zarir and the StackOverFlow answers are offering you the same solution. Store your annotation specific data on the annotation object and reference it in the click event.

      — commented December 16th 2011 by Nick Milner

    • Solution:

      mapView.addEventListener('click', function(e) {
    if(e.clicksource == 'rightButton'){
        var detailwindow = Titanium.UI.createWindow({
            fullscreen: false,
            url:'detailwindow.js',
            data:e.annotation.data
        });
        detailwindow.open();
    }
});
