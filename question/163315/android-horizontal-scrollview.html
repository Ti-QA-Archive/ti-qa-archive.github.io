Android Horizontal ScrollView

I have been trying, very unsuccessfully, to create a horizontal scrollview that works in android. I have the following implementation of a scrollview in my application

"#horizontalScrollBar":{
    left: "0px",
    width: Ti.UI.FULL,
    height: Alloy.Globals.bottomBarHeight,
    contentHeight: Alloy.Globals.bottomBarHeight,
    scrollType: "horizontal",
    backgroundColor:"orange",
    layout: "horizontal"
}

The id "horizontalScrollBar" is used on a "<ScrollView>"

I am adding buttons like the following…

    this.button = Ti.UI.createButton({
            left: "1%",
            top:"3%",
            title: this.title,
            entry: this,
            width: Ti.UI.SIZE,
            height: "94%",
            callBackFunction: this.callBackFunction
        });

    this.button.addEventListener('click',function(e)
        {
            this.callBackFunction(this.entry);
        });

It works fine in the web and in iOS but on android it only displays the first few elements, and wont allow scrolling.

I have tried various combinations of "width = ", "contentWidth=" and am getting nowhere.

Any help would be appreciated.

  • Hi

    A couple of quick observations;

    • Tweak width
    // this should be
width: Ti.UI.FULL,
// should be
width: Ti.UI.FILL, // note the simple difference
    • Width & Height control the outside of the scrollView
    • contentWidth & contentHeight control the inside of the scrollView

    So I would try contentWidth as Ti.UI.SIZE or a specific width.

    Sorry not near my dev machine to test your code with the modifications, but give it a go.

  • If anyone ever has an issue with this, the fix that finally solved all of my issues was a simple boolean (5 hours of unnecessary frustration later and i finally decided to thoroughly read the layout documentation… derp). I needed to add the flag horizontalWrap: false, to my div id.

    "#horizontalScrollBar":{
    left: "0px",
    width: Ti.UI.FULL,
    height: Alloy.Globals.bottomBarHeight,
    contentHeight: Alloy.Globals.bottomBarHeight,
    scrollType: "horizontal",
    horizontalWrap: false,
    backgroundColor:"orange",
    layout: "horizontal"
}

    with this div id i can now add as many horizontal components as I want and scroll through them.
