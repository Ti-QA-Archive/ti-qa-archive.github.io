TiBlob to Android native Bitmap, Drawable or ImageView?

Hello, i haven't found any useful info or reference to develop modules except a very basic one. I need to know how to convert TiBlob passed to an android module method, to one of the native format like Bitmap, Drawble or ImageView. I'm doing a module to apply a fisheye filter to images and, after deformed the image i want to return it to the javascript.
Any help is appreciated!

-Giuseppe

  • Hey there,

    If you take a look here:
    http://docs.appcelerator.com/module-apidoc/latest/android/org/appcelerator/titanium/view/TiDrawableReference.html

    You can create a Bitmap from a TiBlob using the TiDrawableReference.fromBlob function to create a TiDrawableReference, then use the getBitmap function to get a bitmap for it.
