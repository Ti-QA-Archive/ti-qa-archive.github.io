  • Accepted Answer

    Umut, I had the same problem and solved it by going into the Terminal and enabling the Spotlight Server with the launchctl option.

    I found the post here: http://osxdaily.com/2011/12/10/disable-or-enable-spotlight-in-mac-os-x-lion/

    Once enabled, the error went away and the emulator ran as expected.
