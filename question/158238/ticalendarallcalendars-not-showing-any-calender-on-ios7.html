Ti.Calendar.allCalendars not showing any calender on ios7

I am trying to add an event in calendar for iPhone.
The below code is working on iphone simultor 6.1 & iphone device 6.1 but fails on ios7, iphone ios7 or ipad ios6.1.3.

  • I've only tested on iOS 7, but you need to call the authorise method before fetching the calendars.

    

if (Ti.Calendar.eventsAuthorization == Ti.Calendar.AUTHORIZATION_AUTHORIZED) {

    showCalendars(Ti.Calendar.allCalendars);

} else {
    Ti.Calendar.requestEventsAuthorization(function(e) {
        if (e.success) {
            showCalendars(Ti.Calendar.allCalendars);
        } else {
            alert('Access to calendar is not allowed');
        }
    });
}

    I hope this helps
