    Hi

    Easier when you know how - so lets see if I can show you.

    There are two main ways - either should sort you out.

    Global

    • Select the Dashboard tab in Titanium Studio (should be large panel on right)
    • Select Get Started button (with a wrench icon - should be under your name)
    • Select Mobile Web
    • Click here (middle of panel)
    • You should have a list of one or more browsers - first one being default system web browser
    • If **Firefox is not listed
      • choose new
      • Enter Firefox in Name
      • Select Browse
      • Locate Firefox in the Applications folder (select applications on left and then look for FireFox
      • Select FireFox
      • Select Open
      • Leave Parameters blank
      • Select OK
    • If Firefox is listed - choose Edit
      • Use Browse to confirm it points to the right place
      • Select OK (in the Edit box)
    • Tick Firefox
    • Choose OK

    Per App

    • Select your project in project explorer
    • Press the run icon from the top (green circled white play button with yellow folder)
    • Select Run Configurations
    • Select Web Browser form the list on the left (at or close to the bottom)
    • If the triangle is pointing right - click it
    • Select Firefox
    • Confirm Firefox is shown in browser executable
    • If not use the Browse and find it in the same way as shown previously
    • Select Apply
    • Choose Close to dismiss window, or
    • Choose Run to dismiss the window and run the app in Firefox.

    That should do you - I think I got all the steps right.
