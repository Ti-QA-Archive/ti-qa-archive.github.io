  • With all the naysayers, my success may muddy the information stream. Have successfully rebuilt android and iOS binaries for 3 different projects on:

    • Lion (installed today)
    • using Xcode 4.1 (installed today)
    • and TiStudio 1.0.2.201107130739 with the 1.7.1 sdk (something updated today)

    Accurate information:

    • xCode 4.0.x is not compatible with Lion and should not run.

    • xCode 4.3 may run, but is not approved for app store distribution

    Now, walking back through that code in builder,it seems to indicate a failure when checking to see if it can find the simulators sdk to build against.

    So, with that in mind, lets check some things:

    • Can you launch the simulator by itself? (/Developer/Platforms/iPhoneSimulator.platform/Developer/Applications/)
    • Can you run a "hello world" app from Xcode on the simulator?
    • Does the simulator sdk actually exist where it is supposed to? ((/Developer/Platforms/iPhoneSimulator.platform/Developer/SDKs/iPhoneSimulator4.3.sdk/)
