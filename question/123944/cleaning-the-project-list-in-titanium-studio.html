Cleaning the project list in Titanium Studio

How do I clean that list so it offers me only the existingremaining (existing) projects ?

In Titanium Studio, we can switch between projects using the App Explorer. Once a project has been created, it resides in this list, even once it has been totally deleted from the workspace folder.

As we can see in the following screenshots, on the left image is the project list from the App Explorer, and on the right one, the workspace folder as it was when the left screenshot was taken:

So here is the question (again):

How do I clean that list so it offers me only the existingremaining (existing) projects ?

    Just go to the Project Explorer tab (next to the App Explorer), it lists all your projects then right click on the one you want gone and choose delete.
