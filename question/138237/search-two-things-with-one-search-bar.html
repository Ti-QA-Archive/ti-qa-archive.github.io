Search Two Things With One Search Bar

So, in a basic overview, I have a search bar, a table, and two labels in the table. I want the search bar to search both of these labels. My table -kind of- looks like this:

Note: The search bar is called searchBar

var mainTable = Ti.UI.createTableView({
    search : searchBar,
    filterAttribute : 'filter',
    separatorStyle : Ti.UI.iPhone.TableViewSeparatorStyle.SINGLE_LINE
});

Then, down in the part where I create my rows, I have this:
row.filter = label1.text;

But, I also would like to search label2. How would one do that?

    Did you just try adding the other label to the filter?

    row.filter = label1.text + ' ' + label2.text;

    Pretty sure the search looks throughout the whole string for results.
