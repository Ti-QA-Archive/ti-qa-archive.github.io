    • Titanium does not output html.

      Titanium outputs a native, Objective-C application for iOS, or java for Android.

      Your javascript code, written against the Titanium API drives the native controls/interfaces/features of a native app.

      — commented January 10th 2013 by Stephen Feather

    • Clarification, it WILL generate html/javascript code if your target is mobileweb.

      But again, you are writing javascript against the Titanium API.

    • I believe Stephen is referring to the Titanium API is the way you make an app within Titanium. If you wanted to write HTML/CSS, then you would have to use a WebView within Titanium, which is pretty much a web browser within your app. If you go down the path of creating a WebView then you won't be making anything "native", you will simply be wrapping a web page and compiling it for a mobile device, which is more like a PhoneGap than Titanium. Titanium main goal is to compile down to native code, it just uses Javascript as its language of choice. Hope this helps.
