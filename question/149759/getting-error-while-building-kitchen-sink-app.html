  • I get stuck with same problem. And looking around for solution that worked for me is. add path of java sdk in your system environment variables.
    Just add the following path:

    • variable name : JAVA_HOME

    and

    • Variable Value: C:\Program Files (x86)\Java\jdk1.6.0_31

    In my case my Java environment was installed in the mentioned directory.
