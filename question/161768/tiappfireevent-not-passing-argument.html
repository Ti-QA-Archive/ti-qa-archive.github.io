Ti.App.FireEvent not passing argument

I want to fire an event and send an object as a parameter:

Ti.App.FireEvent('custom',{custom:object});

When trying to use that object inside Ti.App.addEventListener I get an 'Undefined' reference!

Can I NOT pass an object as a parameter to Ti.App.FireEvent?

  • FireEvent should be fireEvent. So your event is likely not firing if your code uses the uppercase F.

    Why nest your object in another object? Why not:

    Ti.App.fireEvent('custom', yourObject);
// instead of 
Ti.App.fireEvent('custom', {customProperty: yourObject});

    Either should work, just why unnecessarily nest objects.

    Rainer is correct in that you cannot pass objects with methods. Objects have to be JSON parseable.

  • Your syntax is correct. You can only send object that are stringifyable.

    In your case:

    Ti.App.FireEvent('custom',{custom:{name:'name'}});


Ti.App.addEventListener('custom',function(_e){
    alert(_e.custom.name);  // shows name
});
