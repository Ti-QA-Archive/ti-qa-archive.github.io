Opacity problems

Hi.

I wonder if it is possible to have a text area with backgroundColor : &#39;#111&#39;, for instance and change the opacity so it will be semi transparent?

Same thing in tableView?

If it is possible how do i do it? All i have manage to do is change opacity on the whole thing. Background text etc…

Thanx.

    I faced this same problem a while ago in a project I was working on. I ended up just using a semi-transparent .png as the background image for the view and it works fine. Anything added to the view is full opacity but the view itself is semi-transparent.

  • A basic color hash is as #ffcc00

    To set it to 70% opacity, do something like this: #70ffcc00.
