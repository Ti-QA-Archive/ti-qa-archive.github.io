Math in .tss

Doesn't look like Math is working in .tss files for Alloy development. Shouldn't I be able to do something like this:

'#value[platform=android]' : {
    top         : 90,
    font        : {
        fontFamily  : 'Roboto-Thin',
        fontSize    : '68 * (Ti.Platform.displayCaps.density/240 + Ti.Platform.displayCaps.platformWidth/800) + dp'
    }
}

  • Accepted Answer

    No, you can't do calcuations in TSS files. You can however assign calculations to properties on the Alloy namespace (CFG or Globals) and then use those in your TSS files:

    app/alloy.js

    Alloy.CFG.someCalculation = Ti.Platform.displayCaps.displayWidth / 2;

    index.tss

    "#someLabel": {
    width: Alloy.CFG.someCalculation
}
