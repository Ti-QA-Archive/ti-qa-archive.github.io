    James,

    We have a feed (xml) with the following:

    <description><![CDATA[<a href="http://gbcga.com/sermons/the-kingdom-of-heaven-5/"><img src="http://gbcga.com/images/2011-10-02-1330_200.jpg" width="200" align="left"></a>]]></description>

    If we access it as follows within a loop:

    var description = item.getElementsByTagName("description").item(0).text;

    When we apply that description to the .text value on a label we get what you see below the "summary" label:

    (In this particular case, someone was messing with the feed and it contains actual html tags and it shouldn't. But no additional parsing was done to remove the extended data wrappers from the feed)
