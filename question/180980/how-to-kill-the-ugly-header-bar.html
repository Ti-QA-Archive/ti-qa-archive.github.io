How to kill the ugly header bar

So I'm playing with the framework for my next project, using V3.5.0.GA.

I got everything working, using the basic template in the default project. Deleted all the code in app.js, and just added a blank window with in ImageView (which I mean to be my header).

But Titanium seems to insist on putting it's own header bar up there.

I have researched this, and added to my window definition the navBarHidden : true. Have edited the tiapp.xml file, found the same property and set it to true. Then tried calling win.hideNavBar() - all that did was throw an exception telling me that said method does not exist. And it keeps showing it's ugly navbar. How do I get rid of it?

My current app.js file:

// this sets the background color of the master UIView (when there are no windows/tab groups on it)
var win = Ti.UI.createWindow({
    backgroundColor : '#FFFFFF',
    layout : "vertical",
    navBarHidden : true,
    width : Ti.UI.FILL,
    orientationModes : [Ti.UI.PORTRAIT]
});

var headView = Ti.UI.createImageView({
  top:0,
  width: '100%',
  image: 'CTC_banner.jpg'
});

win.add(headView);

win.open();

    Without all the details, I can only guess…

    Not Titanium's header bar, but the Android ActionBar. You'll want to change or use a custom Android Theme. Specifically, Theme.AppCompat.Translucent.NoTitleBar.

    See Hiding the Android ActionBar for code and even more ways to do it.
