  • Hi

    Yes and possible in many ways and more than one using XML - depending on your specific needs, here is the most likely ;

    VERY simple example;

    index.xml

    <Alloy>
    <Window>
        <View id="outer">
            <Label>One</Label>
            <Label>Two</Label>
            <Label>Three</Label>
        </View>
    </Window>
</Alloy>

    index.tss

    "#outer": {
    height; Ti.UI.FILL,
    layout: "vertical",
    width; Ti.UI.FIL
}

    index.js

    $.index.open();

    test.xml

    <Alloy>
    <Label>Four</Label>
</Alloy>

    Then inside the controller you wish to add items to - but from an external XML, you amend as follows;

    index.js (amended version)

    // this takes the xml from the 'test.xml' file and appends it
// $.outer is the id of the parent you wish to add to
$.outer.add(Alloy.createController('test').getView());
$.index.open();

    Let me know if this is what you meant.
