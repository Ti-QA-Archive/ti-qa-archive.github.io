  • Hi

    Unless I'm missing something, it should be as simple as this:

    checkbox.addEventListener('change', function (e) { 
    if (e.value) {
        strikethrough.setVisible(true);
    } else {
        strikethrough.setVisible(false);
    }
});

    Hope this helps!
