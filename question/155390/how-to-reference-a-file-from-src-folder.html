how to reference a file from SRC folder?

I am trying to reference a html file through appcelerator as shown in the code below, this code is written in (app.js):

var webview = Ti.UI.createWebview({
    url: 'D:\files\files\javascriptparser01.html',          // path to the html file.
});

But that did not work, and I think the file (html file) should be placed in the SRC folder to be invoked.
My Question is: how to invoke a file from the SRC folder, with example.

  • the file has to be under the Resources folder of your project, to be accessible by titanium.

    Resources \
    javascriptparser01.html
    app.js

    Then you can include it like this:

    var webview = Ti.UI.createWebview({
    url: '/javascriptparser01.html',          
});

    in case it's in the root of your project.

    I suggest you to use a structure like 

    var webview = Ti.UI.createWebview({
    url: '/ui/webview/javascriptparser01.html',          
});
