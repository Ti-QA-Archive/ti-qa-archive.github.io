eventListener if( Statements

Titanium Mobile 1.2.2 // iPhone SDK 4.0 // Platform iPhone

Im having a bit of trouble trying to set the index values of items (TabbedBar) to Text that is saved as a preference.

I don't want 0's or 1's to be printed in the email form, so i THOUGHT it would be easy to capture the index value, and apply an IF statement to save it to a YES (index==0) or NO (index==1)

When compiling, it warns with this: [WARN] JavaScript compiler reported "Expected an identifier and instead saw ')'."

code:

tb1.addEventListener('click', function(e){
  if(e.index == 0) // Yes
  {
    Titanium.App.Properties.setString("Preference1",Yes); 
  });

  if(e.index == 1)  // No
  {
    Titanium.App.Properties.setString ("Preference1",No);  
  });

});

I copied the above format from another supposedly working piece of code, however im now at a loss.

If there is a better way to capture this info, my ears are open.

Thanks to all who have helped! Its a great community!

    The problem is you have an extra ) in two places, where you try to close each if {} block, just after each closing } brace. Once you solved that you would discover the Yes and No in each setString() call appear to be variable names since they are not quoted.

    I would suggest something like this:

    tb1.addEventListener('click', function(e){
  var yesNo = ( e.index == 0 ) ? 'Yes' : 'No';
  Titanium.App.Properties.setString("Preference1",yesNo); 
});
