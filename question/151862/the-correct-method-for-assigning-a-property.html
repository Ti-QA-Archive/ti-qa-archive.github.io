The correct method for assigning a property

Some objects has properties and a pair Set/Get methods for this property.

b.e: 

Titanium.UI.Button.title 
Titanium.UI.Button.setTitle()
Titanium.UI.Button.getTitle()

Is there any difference between using the property to assign a value or the method set in this example? and generally what's the correct way?

Titanium.UI.Button.title = 'test';
Titanium.UI.Button.setTitle('test');
— asked May 2nd 2013 by Xabi Palmer

    You should use the properties at creation time of views.
    When the view is already displayed then you should use the get-/set- methods to change its properties.
