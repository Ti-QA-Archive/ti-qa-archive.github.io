Blurry text fix?

Hi there,

While width:"auto" is a great way of getting a label to center the text, it also introduce a new bug, blurry text.

I figured out that if the label is placed on a float number due to the automatic calc of width:"auto" than the font is blurry. e.g. a view is 100px and the label with auto width is 89px, than it will be placed 10.5px from the left of the view container, making the font blurry.

This is a problem. How do I fix this?

And I guess this is a bug, due to Appcelerator should have all views on pixelperfect placement, right?

  • Accepted Answer

    Perhaps a little hacky but this seems to work.

    var wnd = Ti.UI.createWindow({
    backgroundColor: "#FFF"
});

var viewA = Ti.UI.createLabel({color: "#FFF", text: "$", backgroundColor: "#F00", left : 0, width: 14, font: {fontSize: 20}, textAlign: "center"});
var viewB = Ti.UI.createLabel({color: "#FFF", text: "3000", backgroundColor: "#00F", left: 14, width: "auto", font: {fontSize: 40}});

var wrapper1 = Ti.UI.createView({width: (viewA.width + viewB.width), height: 50, top: 10});

wrapper1.add(viewA);
wrapper1.add(viewB);
wnd.add(wrapper1);

var viewC = Ti.UI.createLabel({color: "#FFF", text: "$", backgroundColor: "#F00", left : 0, width: 14, font: {fontSize: 20}, textAlign: "center"});
var viewD = Ti.UI.createLabel({color: "#FFF", text: "20", backgroundColor: "#00F", left: 14, width: "auto", font: {fontSize: 40}});

var wrapper2 = Ti.UI.createView({width: (viewC.width + viewD.width), height: 50, top: 70});

wrapper2.add(viewC);
wrapper2.add(viewD);
wnd.add(wrapper2);

var viewE = Ti.UI.createLabel({color: "#FFF", text: "$", backgroundColor: "#F00", left : 0, width: 14, font: {fontSize: 20}, textAlign: "center"});
var viewF = Ti.UI.createLabel({color: "#FFF", text: "1999999", backgroundColor: "#00F", left: 14, width: "auto", font: {fontSize: 40}});

var wrapper3 = Ti.UI.createView({width: (viewE.width + viewF.width), height: 50, top: 130});

wrapper3.add(viewE);
wrapper3.add(viewF);
wnd.add(wrapper3);

wnd.open();

    • The trick is to nest and play with the lefts and rights. For greater flexibility I'd suggest using percentages but I used fixed numbers here to get a working example.

      var wnd = Ti.UI.createWindow({
    backgroundColor: "#FFF"
});

var viewA = Ti.UI.createLabel({color: "#FFF", text: "$", backgroundColor: "#F00", left: 0, right: 91, font: {fontSize: 20}, textAlign: "center"});
var viewB = Ti.UI.createLabel({color: "#FFF", text: "3000", backgroundColor: "#00F", left: 14, right: 0, font: {fontSize: 40}});

var wrapper = Ti.UI.createView({width: 105, height: 50});

wrapper.add(viewA);
wrapper.add(viewB);
wnd.add(wrapper);

wnd.open();

      As a point of note, elements are usually centered and full height / width unless you tell it explicitly.
