Data to be passed from one window to another

Hello,

I have a problem in getting the data.
In my window(win1) i have 3 buttons, on click of each button it navigates to a new window. there i connect to the server and store the data in a row of tableview.
On the click of table view, i need to populate that selected data into win1 and make it persistent data. so that when i click the second button, get the data from the server and follow the same process, the previoulsy fetched data must also be present.

Please help me out.
Thanks in advance.

  • As Alexander Bauer stated, there are just a couple of different possibilites to pass data between windows:

    1. With the global Ti.App
    2. Using events with data payload (fireEvent + addEventListener)
    3. Persistent storage via models (saving and loading via sqlite)

    The easiest way is:
    4. Don't use windows unless you really need to.

    Have a look at some great examples like Aidori or Tweetanium, both of them use one main window (and create a tab bar which switches between views). Using views instead of new windows makes your life way easier since every object will be created in the same scope (and therefore be available in any view).

    Using windows is very useful if you want to run something in a new thread or if you need a clean scope.

    Correct me if I am wrong :-)

  • In your win1:
    var win2 = Ti.UI.createWindow({url:'win2.js', _parent: this});

    in your
    sub window(2) you could:

    Ti.UI.currentWindow().addEventListener('close', function() {
    _parent.data = dataOfWin2;
    _parent.fireEvent('dataArrived');
    – or –
    _parent.updateData(dataOfWin2);
    });

    if you need the same data more than once
    think about using Ti.App.myGlobalData = dataOfWin2;
