    Hi

    At first glance you have three stand out problems;

    var client = Ti.Network.createHTTPClient({
    onload : function(e) {
        // problem 1
        var news = eval('(' + this.responseText + ')');
        // problem 3 a
        news = news[i].posts;
        for (var i=0; i<=20; i++){      
            // problem 2
            var data = [];
            var row = Ti.UI.createTableViewRow({
                backgroundColor:'transparent',
                hasChild: true,
                // problem 3 b
                title: news.title,
                fontSize: 24,
                color:"#fff",
                height:110
            });
            data.push(row);
        };
        tableView.setData(data);
     },
     onerror : function(e) {
         Ti.API.debug(e.error);
         alert('error');
     },
     timeout : 5000  
});

    • Problem 1

      Do not use eval to process this data, use this instead;

    var news = JSON.parse(this.responseText);

    I could spend some time explaining why you should not try your way, but the internet people will do not for you. Eval should be avoid when possible and always when interacting with remote data. Plus the method I showed above is the right one, and easier to type - bonus.

    • Problem 2

      As excited as you might be to clean out variables and set them up correctly - this is not usually performed inside the look that you then add the newly created row to.

      Change this;

    news = news[i].posts;
for (var i=0; i<=20; i++){
    var data = [];
...

    To this;

    news = news[i].posts;
var data = [];
for (var i=0; i<=20; i++){
...

    You were resetting the array to empty inside each loop - the result should have only provided one entry - the last one on the table.

    • Problem 3

      You set news to an array outside the loop and refer to the loop iteration i, but inside the loop you assume there is only ever a single node - despite the json node stating posts.

      To figure this out I would need to see the actual data coming back, but main points;

      • Cannot use i outside of the loop that defines and especially not before it is defined
    ...
news = news[i].posts;
for (var i=0; i<=20; i++){   
...
    • It is more likely this should be as follows;
    ...
for (var i=0; i<=20; i++){   
    news = news[i].posts;
...
    • However the posts word implies an array - so you HAVE to check what is coming back to conform if this is correct or not. So this code will not work if news is an array.
    var row = Ti.UI.createTableViewRow({
    backgroundColor:'transparent',
    hasChild: true,
    title: news.title, // will not work if "news" is an array
    fontSize: 24,
    color:"#fff",
    height:110
});
    • Add this debug line to check what is returned and fix if required - revised internal code included;
    var client = Ti.Network.createHTTPClient({

    1. please try to use listviews instead of tableviews

    module.exports = function(_options,_callback)
  var xhr = Ti.Network.createHTTPClient({  
     onload : function() {
    try{
        var posts = JSON.parse(this.responseText).posts;
        _callback(posts);
    }
    catch(error) {}
}});
xhr.open();
