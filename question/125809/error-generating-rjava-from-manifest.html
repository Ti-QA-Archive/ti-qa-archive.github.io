Error generating R.java from manifest

I got this earlier - Could not locate the Android SDK at the given path
and found solution of creating empty folder android-7 under android_sdk\platforms
but now getting this error:

  • Accepted Answer

    This is typically caused by not having the minimum default SDK versions installed, as detailed here in the wiki. You will need to have Android SDK platform 2.1 (API level 7) in order for Titanium to run your project.

    Getting Android set up can be a pain though.
