    Note that only Ubuntu is officially supported. See Installing Titanium. I have no problems at all using Titanium on either Ubuntu 9.10 or 10.10.

    There really should be no need to compile from source, unless you are patching one of the components or building a module.

    You should use a network analyzer like Wireshark to see whether the request to the server is being made, and whether anything is preventing it from proceeding.

    Cheers
