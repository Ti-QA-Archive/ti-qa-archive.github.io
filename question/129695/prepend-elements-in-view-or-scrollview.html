Prepend elements in view or scrollview

Hi,

I am trying to develop an app which contains a part similar to the iphone sms app: a message feed with bubbles, when you open the app you arrive at the very bottom of the feed where is the most recent message, if you scroll up to the top and if there are many messages you can tap on a button like "show earlier messages", and the earlier messages are prepended to the feed.

What I am trying to figure out is how to prepend elements(messages) to a view/scrollview above existing elements, because when I use view.add(message) the message is add at the very bottom of the view and I can't find methods like "prepend" or "insert before"…

Thanks a lot in advance for any help.

  • You can't do that for regular views unless you implement that functionality (something similar can be found here: https://gist.github.com/1485751) because regular views can't add some view at specific position.

    Table view component has functionality that you need, so I would recommend that you start using table view instead of regular view. http://developer.appcelerator.com/apidoc/mobile/1.8.0.1.RC1/Titanium.UI.TableView.insertRowBefore-method.html
