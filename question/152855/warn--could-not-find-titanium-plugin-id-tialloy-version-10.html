[WARN] : Could not find Titanium plugin id: ti.alloy version: 1.0

I am new to Titanium but familiar with the Eclipse environment. I pulled down various projects and they all worked fine except for one, which gives me this error on run:

[WARN] : Could not find Titanium plugin id: ti.alloy version: 1.0
[ERROR] : Could not find all required Titanium plugins:
[ERROR] : id: ti.alloy version: 1.0

So I am assuming that this message is accurate, and that I am using a later version of Alloy for the projects that work. (They do use Alloy.) So how do I specify Alloy v 1.0 for the one that does not work? (In Flex you would set the Flex SDK in the project properties. There is nothing in "Linked Resources" in any of the projects, and plugins seem to live on the IDE-level, not the project level).

Mobile
Titanium Command-Line Interface, CLI version 3.1.0, Titanium SDK version 3.1.0.GA
iPhone
Hosted on OS X

  • So this only happens on one specific project? What project? The project is probably corrupted somehow. You can try reinstalling the plugin for the app by doing the following:

    • open the command line
    • go to the folder that contains your project
    • run alloy install plugin

    Then you can try running the app again, assuming it is an alloy app
