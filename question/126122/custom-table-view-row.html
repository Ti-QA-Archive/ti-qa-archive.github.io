  • The approach I would TRY is to appendRows to the table and then after each append, animate the height of the table to be equal to row_height * row_count if row_height is less than whatever you set as the bottom of the screen.

    I say that because you have buttons and tabGroup down there that would need to be accounted for.

    • var win = Ti.UI.createWindow({
  backgroundColor: 'blue'
});

var table = Ti.UI.createTableView({
  height: '100%',
  width: '100%',
  backgroundColor: 'transparent',
  data: [
    {
      title: 'Hi',
      backgroundColor: 'white'
    },
    {
      title: 'Hello',
      backgroundColor: 'white'
    }
  ]
});
table.footerView = Ti.UI.createView({
  width: 0,
  height: 0
});

win.add(table);

win.open();
