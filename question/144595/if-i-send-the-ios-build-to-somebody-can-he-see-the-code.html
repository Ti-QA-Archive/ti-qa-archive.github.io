  • FWIW, your JavaScript code is ONLY minified and encrypted when packaged for the App Store or Adhoc distribution. The new Titanium CLI (which does the actual iOS application compiling) will now encrypt by default for device builds too so that you can see the real world performance of your app.

    Additionally, only JavaScript files are encrypted. HTML files, text files, images, etc are NOT encrypted. That might change in the future since a lot of people would love that feature.
