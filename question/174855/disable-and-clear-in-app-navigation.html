    Hi

    So I would change your login.js to bring back a view rather than a window. This is to sit over the main container, and also so that there is no weird animations when removing a root window.

    Unsure if you want to be able to only logout on the root window (index.js) or anywhere in the app. But I will assume the former.

    I would create a couple functions to call when you open/close your login view. These will be helpful later on to handle everything cross platform. Here are a couple examples:

    $.disableNav = function () {
    if (OS_IOS) {
        $.win.setLeftNavButton(undefined);
    }
    if (OS_ANDROID) {
        $.nav.activity.actionBar.displayHomeAsUp = false;
    }
}

$.enableNav = function () {
    if (OS_IOS) {
        $.win.setLeftNavButton($.leftNavBtn);
    }
    if (OS_ANDROID) {
        $.nav.activity.actionBar.displayHomeAsUp = true;
    }
}

    This again assumes that there is only a left navigation button on both iOS and Android, and that you are using an Android version of 4.x or higher and finally that you have set up the ActionBar for Android also.

    See here for ActionBar

    Hopefully this all makes sense and points you in the right direction, any questions just ask!
