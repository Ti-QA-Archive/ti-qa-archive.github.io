Android: Use Ti.Android.R.drawable_ as button background image?

Is it possible to use the Ti.Android.R.drawable_* resources as the background image of a regular button (not the ActionBar Menu)?

Sample code:

<Alloy>
  <View>
    <Button id="theButton"/>
  </View>
</Alloy>

$.theButton.setBackgroundImage(Ti.Android.R.drawable_is_menu_send

    Hello Mark Olsen,

    Android resource ID works only "icon" property 

    icon = Ti.Android.R.drawable.ic_menu_send

    At regular button, it's not possible.
