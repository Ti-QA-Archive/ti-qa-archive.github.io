Displaying a placeholder in picker ?

Hi guys I am using picker as a dropdown to display the contents of the list from my backend (rails) server. I just want to show some dummy value in the picker like "Please select …" or something else. Here is the code that I use for the picker. 

var pickerTitle = [];
var my_title= new Array();
var win = Ti.UI.createWindow({
        title:title        
});
var myPicker = Ti.UI.createPicker({
            selectionIndicator: true,
            width: "100%",
            top: "2%"                                            
});
win.add(myPicker);
win.addEventListener("open", function(){
// ...
    for(var i=0; i<data.length; i++){
        title.push(data[i].my_title);                        
        pickerTitle[i] = Ti.UI.createPickerRow({title: my_title[i]});
        myPicker.add(pickerTitle[i]);
    }

});

Everything is working fine, I would like to display some text on the picker like "Select a value." Is this possible in titanium's picker on android/ios devices ?

P.S.
Node.js
Node.js Version = 0.12.1
npm Version = 2.5.1
