Resize Image taken with iPhone camera?

I have an app here that Ive been working on which takes a photo and then uploads it to a server.
All of this is fine, but the image taken by the iphone is very large.

QUESTION:
How do I resize this captured image down to 25% or 30%, before the upload process occurs?

    You can use the free ImageFactory module:

    https://marketplace.appcelerator.com/apps/1184?1927520424

    They have examples in their documentation but here is a basic example you could use in the 'success' callback of the showCamera or openPhotoGallery function.

    success: function(e) {

    var ImageFactory = require('ti.imagefactory');

    var w = e.media.width / 2,
        h = e.media.height / 2;

    var resized = ImageFactory.imageAsResized(e.media, { width: w, height: h });

},
