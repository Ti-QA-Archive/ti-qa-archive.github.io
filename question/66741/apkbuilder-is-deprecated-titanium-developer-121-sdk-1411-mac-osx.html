  • I get the same error on Windows. It showed up when I updated my android SDK recently. While it's an "[ERROR]" it does not appear to be causing any problems. As you say, the app still compiles, installs, and runs in both the emulator and on my phone. I suspect that the apkbuilder script is outputting a notice that it's deprecated (shouldn't be used any more going forward). Titanium is probably programed to interpret any output from that script as an error. In this case, it's a deprecated notice not a real error.
