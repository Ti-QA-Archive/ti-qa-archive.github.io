Different media format supported by Titanium.Media API

I am trying to build a very simple video player and here is my code

var vidWin = Titanium.UI.createWindow({
    title : 'Video View Demo',
    backgroundColor : '#fff'
});

var videoPlayer = Titanium.Media.createVideoPlayer({
    height: '400',
    width: '300',
    url: 'http://www.tools4movies.com/dvd_catalyst_profile_samples/The Amazing Spiderman bionic.mp4',
    backgroundColor:'blue', 
    autoplay: true,  
    mediaControlMode:Titanium.Media.VIDEO_CONTROL_EMBEDDED,
    scalingMode:Titanium.Media.VIDEO_SCALING_ASPECT_FIT,
       sourceType:Titanium.Media.VIDEO_SOURCE_TYPE_STREAMING
});


vidWin.add(videoPlayer);
vidWin.open();

As this code runs fine in android, I can see video playing but it dont work in iOS device.

When I tried a different url which streams .m4v, it works in iOS devices.

Can anyone tell that which video format are supported and is there is any way to create a generic player that runs all the format or most common formats.
