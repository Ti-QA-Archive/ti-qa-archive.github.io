Application Close / Stop / Exit

Hi I think this may be a little stupid question but after searching the forum etc I have not found an answer.

I want to close my mobile application. Actually stop it running.

If I use ExitOnClose in the window then on Android it closes the application but does not stop it. It is still running.

On iPhone doesn't actually do anything.

The only way to exit the app on iPhone is to use the hard close key. On Android using the back button again just exits and not stops the application.

Is there a simple way to stop the application on pressing a button.

Thanks

T..

  • Hi ,

    You can quit the app on iPhone using this.,

    1. copy the info.plist from [project dir]/build/iphone/ to [project dir]/
    2. Then edit the info.plist to add the following at the last.
      <key>UIApplicationExitsOnSuspend</key>
<true/>
    3. Delete the contents of [project dir]/build/iphone/ and rebuild.

  • On iPhone you are not allowed to close the app other way then pushing the Home button on the device.
    It's an Apple rule.

    On Android exitOnClose in the main window should close the app. If you still see it when you press the home hardware button is because there is a list with the last apps and not the running ones.

  • On android clicking back button, the app is still running (I saw it in a process explorer) but when you click on the app icon, the application starts from the beginning. BUT if you press home button and the you click the app icon the application resumes.
