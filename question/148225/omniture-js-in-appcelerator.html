Omniture JS in Appcelerator

Hi,

I want to include the JS script tagging from omniture in appcelerator. Is this possible? or can u suggest an alternative..

— asked February 13th 2013 by saga s

  • If you are building a native app, then there is no such thing as a script tag, and no way to include remote scripts within your app. In a normal case the alternative would be to add the script locally to you're app and then turn it into a commonjs module that you can use within your app, however, I do not believe this will be possible as I think the omniture scripts include more remote scripts within their original remote js.

    The only way you may be able to make this happen is to use a webview and add the scripts there just as you would a normal web page.
