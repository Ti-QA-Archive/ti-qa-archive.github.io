title in table view row is not coming fully

Hi ,

I have a table view row with long title but my title is not coming full , after few words its coming dots.How to show the full title so that its height is adjusted according to the title to show the full title.

for(var i=0;i<title.length;i++){
 data[i] = Ti.UI.createTableViewRow({title:title[i],className: "row_" + i});

}

var table = Titanium.UI.createTableView({data:data});
win.add(table);

Thanks.

  • The title property uses a standard font which you cannot override, and which does not wrap. Leave off the title property and add a label to each row. This will let you have full control over the formatting. With a label you can change the font and size, plus allow the label to wrap if you prefer. Or you can use the label.minimumFontSize property to have the system reduce the fontsize until it fits or your minimum size, whichever occurs first.

    Take a look at how table rows are formatted in the Using TableView guide, or some of the kitchen sink examples.
