    Hi

    Yes Titanium creates Native Apps. All APIs create true platform specific UI elements and calls all internal APIs natively. There is some cool internal magic going on but everything is native.

    As you appear to know PhoneGap is a different thing - it has a wrapper shell and then uses web pages internally to pretend to be a native app - Titanium is so very different.

    BTW Titanium can show web pages - but when you want to show web pages as actual web pages in your app - not to pretend to be something they are not.

  • Titanium does not require a webview to run like phonegap does. Titanium builds a native application, with native components. The difference would be that the logic behind the manipulation of those native components is driven by JavaScript instead of java(android) or objectivec(ios).
