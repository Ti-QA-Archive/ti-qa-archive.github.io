Install problems and solutions

I'm developing on win 7 machine and encountered many issues in installing Titanium studio and making it work.

Hope these solutions may help other developers:

Reset Titanium

Delete .titanium, .android, .eclipse from C:\<User>\<your PC Name>
It returns titanium to “factory” state

Check installation:

http://docs.appcelerator.com/titanium/latest/#!/guide/Installation_Troubleshooting

Solve Error: Unable to find Android SDK tools: zipalign

Copy the zipalign tool from "android-sdk-folder/build-tools/19.1.0" to "tools" folder:

Copy c:\android-sdk-win\build-tools\19.1.0\zipalign.exe c:\android-sdk-win\tools\

Solve Error: Target Android SDK 11 is not installed

Set in tiapp.xml <uses-sdk android:targetSdkVersion="19"/>
or any valid value from c:\android-sdk-win\platforms\

Solve Error: Android Emulator fails to launch app after timed out

The emulator launch timeout is set to 2 minutes and the app launch timeout is set to 2 minutes. If your computer is slow, then you must increase these timeouts.

ti config android.emulatorStartTimeout 300000

ti config android.appStartTimeout 300000

You may also need to restart the ADB server:

adb kill-server

adb start-server

Solve Error: Failed to install apk on "emulator-5554"

This is a bitch with no clear solution. Titanium doesn’t give the error why it happened.
My workaround is to have a constantly open emulator and use CLI (command line interface) to do the run.

Once: open cmd window and start your emulator with your chosen AVD:

emulator -avd avd-name

i.e. emulator -avd S4

Create a small batch file deploy.bat with:

`call titanium build –platform android –project-dir <project path> -b –log-level info

adb install –r <path of apk file>`

for example:

call titanium build --platform android --project-dir c:\Users\Yair\Documents\Titanium_Studio_Workspace\Todo List -b --log-level info

adb install -r c:\Users\Yair\Documents\Titanium_Studio_Workspace\Todo List\build\android\bin\Todo List.apk

Open a second command window.

After every code change in studio, run in the cmd window deploy.bat and test immediately the updated app in the emulator.
