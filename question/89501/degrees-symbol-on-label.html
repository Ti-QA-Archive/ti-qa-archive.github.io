Degrees Symbol on Label

I keep getting a StringEncoding error when adding a "degrees" symbol to a label.

°C / °F for example.

Any ideas if its possible?

    Craig

    You can use the unicode for the symbol. For example, this code works:

    var window = Titanium.UI.createWindow({  
    backgroundColor:'red'
});
var thisLabelTemp = Ti.UI.createLabel({
    color:'white',
    font:{ fontSize: 20, fontWeight:"bold" },
    text:"99.9F"+"\u00B0"
});
window.add(thisLabelTemp);
window.open();

    Hope this helps.

  • I'm currently on SDK 1.2.2 (about to update to 1.5.0) below is parsing JSON from Yahoo! Weather API and just wanted to add the degree's symbol to a label.

    var tempTitle = Titanium.UI.createLabel({
    color:'#000000',
    text: json.weather_condition+' '+json.weather_temp+' °C',
    font:{fontSize:13},
    textAlign:'left',
    width:165,
    height:25,
    left:73,
    top:30,
    backgroundColor:'transparent'
});
weatherLabel.add(tempTitle);
