How to get dynamic scrolling table to work on Android

I am referring to the following file in the KitchenSink app: table_view_dynamic_scroll.js

What I'm trying to do is to use the same functionality in Android. However, Android does not support the contentOffset.y property in the TableView scroll event.

Does anyone know a way to get around this, or to achieve this functionality in a different way?

If you're not aware of what this code does, it allows you to display N records at a time in a TableView, and as the user scrolls down, it will fetch additional info and add N rows at a time.

Thanks!
