  • Ti.Platform.openURL(urlapp) this is the only line you need, if you require to open not only in the market, if you do not find in market (that is out of line or whatever) you can use the same line to open on web.

    if (Ti.Platform.canOpenURL(urlapp)) {
//for market
            Ti.Platform.openURL(urlapp);
        } else {
//for web url
            Ti.Platform.openURL(urlweb);
        }
