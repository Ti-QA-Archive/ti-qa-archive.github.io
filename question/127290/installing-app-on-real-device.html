Installing app on real device

Hi! I developed an application for android using Titanium. Now I want to install and run this app on real android device.i don't know how do it.Any one please tell me the deployment process i.e what are the prerequisites etc. Also is there any way to install app on real device from command line tool on windows 7.I am using Titanium sdk version 1.7.2.

  • First of all prepare your Android device by allowing unknown sources and USB debugging in the settings (usually in the applications and applications/development section of the settings).

    Then connect your device via USB and in your project in TiStudio right click on your project (App Explorer or Project Explorer tab) and select "Install to Android device".

    Hope this helps.
