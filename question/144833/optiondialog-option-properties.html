OptionDialog option properties

In reviewing the documentation it seems as though you can only set the options property to a string array. When a user selects an option I need to know a numeric ID # associated with the string that's presented in the option dialog. How would I go about this if I can't pass in custom properties with each option?

This is for an iPad application.

  • Hi Shawn

    As you have noticed the labels are set as a string array, but you can solve the problem very simply.

    var data = [
    { title: 'Option 1', id: 51 },
    { title: 'Option 2', id: 22 },
    { title: 'Option 3', id: 38 }
];
var opts = [];
var intOption = 0, intOptions = data.length;
for (intOption = 0; intOption < intOptions; intOption = intOption + 1) {
    opts.push(data[intOption].title);
}
var dialog = Ti.UI.createOptionDialog({
  cancel: 2,
  options: opts,
  selectedIndex: 2,
  destructive: 0,
  title: 'Delete File?'
});
dialog.show();

dialog.addEventListener('click', function (e) {
    alert(data[e.index].id);
});

    Using this simple technique you can work out the id from an associated/lookup array.

    Hope this helps.
