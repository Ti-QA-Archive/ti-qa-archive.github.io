    I think you are asking how to store an array of values instead of a single one ? If so, you can use a different Ti.App.Properties methods getList and setList.

    To read/write to and from the list you do this

    
// get the current list , or an empty array if there is nothing set yet)
var myList = Ti.App.Properties.getList('myValues',[]);

myList.push(22); // add the value 22 to the list

// re-save the list into the property
Ti.App.Properties.setList('myValues',myList);
