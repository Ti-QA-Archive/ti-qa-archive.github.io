Return specific information from XML data?

I've been searching through Q&A to try and find an example of what I'm looking for with no luck.

Given this XML data pulled from an HTTPclient:

<inventory>
    <car>
        <image>http://images.idealer1.com/getimage/640/9502293.jpg</image>
        <vin>1G1YY34U655118812</vin>
    </car>
      <car>
        <image>http://images.idealer1.com/getimage/640/9494274.jpg</image>
        <vin>1GNDS13SX32369978</vin>
    </car>
</inventory>

How would I return the contents of the <image> tag associated specifically with 1 vin (say <vin>1GNDS13SX32369978</vin>)?

  • //after getting a response
var xml = this.responseXML;
var cars = xml.documentElement.getElementsByTagName("car");

//loop through the cars
for (var i=0,c=cars.length; i<c; i++) {
    var vin = cars.item(i).getElementByTagName("vin").item(0).text;
}

    • //after getting a response

var cars = xml.documentElement.getElementsByTagName("car");

//loop through the cars
for (var i=0,c=cars.length; i<c; i++) {
    var vin = cars.item(i).getElementByTagName("vin").item(0).text;

        var image = cars.item(i).getElementsByTagName("image").item(0).text

      Just perform an equality test (===), and pull out other values in the same node based of the results. Make sure to wrap it in single/double quotes like above.
