  • Look like the problem is on the 'auto' property you are setting for the height and width. This property is been deprecated for version 2.0xxxx, instead you have to use Ti.UI.SIZE for the object to get the size of its content or Ti.UI.FILL for the object to get the size of its parent

    — answered June 18th 2012 by Arian Caraballo

    • substitude the on your code where you have a width: &#39;auto&#39; by width: Ti.UI.SIZE this is the equivalent of auto on sdk2.0xxxx. and if by any change you need your object to be the size of the parent container you do a width: Ti.UI.FILL. you can also use height
