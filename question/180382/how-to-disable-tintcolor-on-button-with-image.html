How to disable tintColor on button with image?

I have a <Button> and set as image for it, but this image have multiple colors I want that this colors appear exactly as the image.

In Android it works ok, but in iOS I need to set a tintColor and this breaks my design.

Can I disable tintColor?

<Button class="phone" onClick="call"></Button>

".phone":{
    image:"/images/telefone.png",
    height:Ti.UI.FILL,
    width:50,
    backgroundColor:"transparent",
}
".phone[platform=ios]":{
    style:Titanium.UI.iPhone.SystemButtonStyle.Plain
    //tintColor:'blue' // I want to disable tintColor
}
