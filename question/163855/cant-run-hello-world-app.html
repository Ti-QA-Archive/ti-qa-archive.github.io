  • The NDK is ONLY used if you're building native Android modules for Titanium.

    Your error says no AVD device available. You should go to your Android SDK folder, go under Tools and run the "android" program. It'll launch and show you the list of Android SDKs. You should have at least one and your version of Android SDK Tools MUST NOT BE 22.6. Now go to the Tools menu and select Manage AVDs. There you -should- have at least one, since Titanium Studio automatically creates them if you don't have them. Start there and report your progress.

    Also, if your computer is -slow-, you should try and use either a device or the Genymotion emulators (these are way better than the Android-provided ones).

    Best of luck,

    R
