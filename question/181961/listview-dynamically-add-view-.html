    • I have no example. But I know this often asked question. listview is a compromis: you win performance and lost references. It is very heavy to use spinner/progress views inside a listview item. Perhaps an other titan knows more.

      Way (no tested):
      In you template you have type:

       childTemplates : [{
       type : 'Ti.UI.ScrollableView',
       bindId : container,
       properties : {
        views : [],
        top : 0    
 }]

      Now you can determine the object after itemclick and can set the property;

      container.views = [require('de.marcelpociot.circularprogress').createProgressView()];

      Hope this helps you.
