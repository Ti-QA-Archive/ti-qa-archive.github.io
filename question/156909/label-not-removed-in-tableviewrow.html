label not removed in tableviewrow

Hi Guys,

I have problem removing labels inside a tableviewrow. Please see below for the code.

var catnewwindow = Titanium.UI.createWindow({title: 'Add Category'});

var rows = [];
var row = Ti.UI.createTableViewRow({title: 'Parent Category', font: {fontSize: 16, fontWeight: 'bold'}, color: '#355DAC', hasChild: true});
var lblHintText = Titanium.UI.createLabel({text: 'Optional', right: 5, color: '#808080', font: {fontSize: 14}});
row.add(lblHintText);
rows.push(row);

var cattableview = Titanium.UI.createTableView({ data: rows, minRowHeight: 50, style: Titanium.UI.iPhone.TableViewStyle.GROUPED});
cattableview.addEventListener('click', function(e2){
    e2.row.remove( lblHintText ); // label not removed
    e2.row.remove( e2.row.chilren[0] ); // same thing on this one.

    var label = Titanium.UI.createLabel({text: 'test 123', backgroundColor: '#fff', right: 5});
    e2.row.add(label);
});
catnewwindow.add(cattableview);
catnewwindow.open({modal: true});

This is in iOS running Titanium SDK 3.1.2 GA. The remove function doesn't seem to work when we add a label after it is executed.

 e2.row.remove( lblHintText ); // label not removed
 e2.row.remove( e2.row.chilren[0] ); // same thing on this one.

Any ideas? I've applied all workaround that I can think but it still not working. Any help is very much appreciated.

Thank you - Mark

  • Hi

    Without actually testing your code the first things I would think of are;

    • lblHintText scope may not work the think it does inside a tableViewRow
    • You spelt children incorrectly, if this is not a typo getting the code in here try correctly that and see if that fixes things;
    e2.row.chilren[0]
// should be
e2.row.children[0] // missing "d"
    • Consider using the source attribute if the second point fails;
    var lblHintText = Titanium.UI.createLabel({
  text: 'Optional', 
  right: 5, 
  color: '#808080', 
  toucheEnabled: false,  // makes sure this will not be the source
  font: {
    fontSize: 14
  }
});
...
cattableview.addEventListener('click', function(e2){
    e2.source.remove(e2.source.children[0]);
});

    Worth a try.
