  • Here is method to use global variable and global array:

    Ti.App.myVar=0;
Ti.App.myString="";
Ti.App.myArray=new Array();

    But these global variables are not very good method to use you would always find an alternative but at initial stage they would be easy to use

  • Hi, You can use this to store global list.

    var props = Ti.App.Properties.listProperties();

for (var i=0, ilen=props.length; i<ilen; i++){
    var value = Ti.App.Properties.getString(props[i]);
    Ti.API.info(props[i] + ' = ' + value);
}
