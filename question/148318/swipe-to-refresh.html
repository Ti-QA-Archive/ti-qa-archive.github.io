Swipe to refresh

Hi,
in a video portal I show the latest videos in a scrollable view. My idea: if the user swipes in the first view to right, the model retrieves new entries. How can I realize a text on left side of scrollable view, like in tableview.

Rainer

  • Self solved:

    self.container.addEventListener('scroll', function(_e) {
        if (_e.currentPage == 0 && _e.currentPageAsFloat == 0) {
            self.actind.show();
        } else {
            self.actind.hide();
        }
    });
