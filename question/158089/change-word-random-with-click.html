    Hi

    This might be easier than you think;

    var getRandomInt = function (min, max) {
  return Math.floor(Math.random() * (max - min + 1) + min);
};
// shortened list for example purposes
var word = [
    'One',
    'Two',
    'Three',
    'Four',
    'Five'
];
var win = Ti.UI.createWindow({
});
var lbl = Ti.UI.createLabel({
    text: '',
    top: 50
});
var btn = Ti.UI.createButton({
    title: 'Press Me',
    top: 100
});
btn.addEventListener('click', function (e) {
    lbl.setText(word[getRandomInt(1, word.length) - 1]);
});
win.add(lbl);
win.add(btn);
win.open();

    • This uses the getRandomInt function to gain a random number between 1 and the total number of array elements we have created to choose from. As the array returns items using a zero based system we simply remove 1 from the random result to get the index of the array.

      Above is a fully working code example, but the only real points of interest to look at are the function getRandomInt and the btn event listener.
