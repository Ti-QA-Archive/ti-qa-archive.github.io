html property of Label on android

Hi, I need to use html string for a label in android (Ti SDK 3.0 GA). I am trying with below code. The label did get updated as per the <b> <i> and <a> html tag. However, I need to call the openAndroidLinkedWebView function when you click on <a> tag. This is not working and error saying that no Intent/Activity found.

Anyone clue? 

$.infoText.html = '<html><body><b> Title </b> <i> Paragraph</i><a href="#" style="color:#01B0F1;" onclick="openAndroidLinkedWebView(this)" id="' + subString + '">' + subString + '</a></body></html>'


function openAndroidLinkedWebView(e) {
    log(JSON.stringify(e));
    alert("e.id = " + e.id);
    log("e.href = " + e.href);
}

  • You will need to add an event listener to the label and open the target URL from there.

    $.infoText.addEventListener('click', function() {
   Ti.Platform.openURL('http://wherever.com');
});
