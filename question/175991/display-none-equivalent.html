Display: none equivalent

In Titanium, one can hide an element by assigning visible: false, however, this still displays the space this object uses.

Any way to completely hide the element without removing it and without putting height/top/bottom/left/right at 0dp?

    Hi

    Short answer; not really.

    Longer answer;

    You already know web pages and mobile apps are two different which helps in answering this.

    The most common solution is to set the height of the thing you wish to "hide" to zero. This however has the problem that the layout of the child objects within that parent may change if Ti.UI.SIZE, Ti.UI.FILL or percentages are used. When the parent is then expanded there may be layout flickers as it adjusts. However smart planning can reduce or remove this sometimes annoying side effect.

    You can (as mentioned) remove the view from its own own parent. An important thing to remember is that it is the variable that contains the view so removing it, adding it back or even removing it and putting it somewhere else are perfectly fine as you are not destroying the view in this process.
