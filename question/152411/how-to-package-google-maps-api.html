How to package Google Maps API?

I have an Android Titanium module project which uses the class MapActivity from the Google APIs. By default, maps.jar (Android SDK level 8) was already referenced in my module project. I added

<uses-library android:name="com.google.android.maps" />

to timodule.xml to ensure the library is loaded but still the class MapActivity cannot be found at runtime in an application that uses this module. In the run configuration of the app, I explicitly selected "Google APIs (Android 4.2.2) [armeabi-v7a]".

How can I reference maps.jar (Google Maps APIs) in my application project? Because if I copy maps.jar to the module project's "lib" folder, the whole thing works – so the solution would be to just reference maps.jar in the application somehow, right?!
