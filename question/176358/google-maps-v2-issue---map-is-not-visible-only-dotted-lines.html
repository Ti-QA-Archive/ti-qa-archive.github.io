  • I think for static maps you use an other key. I have build a lot of android map apps with Titanium. Always if issues appears, then it was a typo by me.

    Summary:

    1. signing the app with own key
    2. in google api console you must switch on API and create a API key. (also wildcard, it works)
    3. API key comes with other entries in tiapp.xml
    4. in you adk you must install google play
    5. you must test if the right version of google play is instaleld on device
    6. connectivity!
    7. ignoring of warning in console about missing google play id
    8. testing on real device
