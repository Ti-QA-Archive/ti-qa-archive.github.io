  • Aptana Studio 2 or 3 is a free IDE, but be aware that it has a rather large footprint.

    On the Mac, the kind of default standard is TextMate. It's really nice and has a built-in text expander for shortcuts, which allow you to code very quickly without having to go to a bundle kind of thing. It's also a smaller footprint. There is no syntax checking, but there is color highlighting for most common languages. However, it's $56 US.

    Smultron is a very functional, kind of minimal editor, as is TextWrangler (BBEdit's little brother). Both are similar, have a reasonably small footprint and are free of charge.

    Hope this helps!

    Cheers,
    Korky

  • As others have mentioned Aptana Studio is good for this kind of of development already, and we'll be bringing a lot more Appcelerator-specific features to it. Some early highlights:

    1. Titanium-specific code assistance while editing
    2. Compatibility with the ADT Eclipse plugin
    3. Integrated Titanium packaging and deployment for desktop and mobile platforms
    4. Debugging
    5. All-in-one package so no need to download and configure a bunch of separate development environments.

    Aptana Studio currently has a pretty robust extensibility model, and you can import a number of TextMate bundles into it, so it's relatively easy to customize it to suit your needs.

    We'll of course be adding more as time goes on, but we think the features above alone should help smooth and enhance Titanium Development. If there are things people would love to see, please let us know.
