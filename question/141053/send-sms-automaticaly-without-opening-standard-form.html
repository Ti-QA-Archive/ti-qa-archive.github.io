  • Most important part of Java code which is needed to be in main java module file looks like this:

    import android.telephony.SmsManager;

@Kroll.module(name="Smsapp", id="id.name-of-module")
public class SmsappModule extends KrollModule {

    @Kroll.method
    public void sendSMS(String phoneNumber, String message) {                
        SmsManager sms = SmsManager.getDefault();
        sms.sendTextMessage(phoneNumber, null, message, null, null);
    }

    After you export module add it in your app. 

    var sms = require('id.app-name');

    Next you need to add this line into tiapp.xml: 

    <manifest>
    <uses-permission android:name=”android.permission.SEND_SMS”></uses-permission>
</manifest>

    From titanium just create button which trigers event on click:

    sms.sendSMS("phone number", "any text");

    I supose that you know how to use this code it so i hope that it will help.
