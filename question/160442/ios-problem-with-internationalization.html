[iOS] Problem with Internationalization

Hello guys, I have a problem with the internationalization of my App

The app allows you to change the language of the app from within it, to do this use the following module : Locale

I only have a problem, that is, when trying to change the text of a series of label dynamically created and added to a row which is added, of course, to a tableviewrow

my tablewviewrow is created in the following way

var array_row_icona = ["/images/ipad/icone_impostazioni/connessione.png", "/images/ipad/icone_impostazioni/lingue.png", "/images/ipad/icone_impostazioni/titolo.png", "/images/ipad/icone_impostazioni/sfondo.png", "/images/ipad/icone_impostazioni/sicurezza.png"];
var array_row_testo = [L('label2'), L('label3') , L('label4') , L('label5') , L('label6')];

for (var m = 1; m <= 5; m++) {

    var row = Titanium.UI.createTableViewRow({
        backgroundImage : "/images/bg_barra_.png",
        height : 80,
        id : m
    });

    var icona_row = Ti.UI.createView({
        width : 60,
        height : 70,
        left : 10,
        backgroundImage : array_row_icona[m - 1]
    });

    var nome_row = Ti.UI.createLabel({
        text : array_row_testo[m - 1],
        left : 90,
        color : 'white',
        font : {
            fontSize : 40,
            fontFamily : 'Helvetica Neue Lt Std',
        },
        top : 20,
        height : 50
    });

    row.add(icona_row);
    row.add(nome_row);

    array_row_impostazioni.push(row);

}

table_impostazioni.data = array_row_impostazioni;

when the app start , the app change the text of the label/button/etc etc according to the language of the phone, and this is not a problem, and the labels of the tableviewrow are created with the correct language

the problem occurs when I change the language of the app from within it. if I try to do this with the normal label created for example in this way there are no problems , the language of the label change correctly even when I change it from within the app

var test_label = Ti.UI.createLabel({
        text : L('label1'),
        left : 90,
        color : 'white'
    });

However, this does not happen with the label created by this array of texts

var array_row_testo = [L('label2'), L('label3') , L('label4') , L('label5') , L('label6')];

Any solutions ? thanks

  • Taken from a project done last year, languages.js

    https://gist.github.com/sfeather/7950948

    Load the module:

    var Languages = require('languages');
languages = new Languages();

    Initialize it one time:

    languages.init();

    Replacement L() function:

    languages.L('word_tag', 'Word Tag');

    Set the default language to use (pass it the index value for the array in languages.js)

    languages.setLanguage(1); // es

    pairs are stored in JSON (sample in the gist)

    Use the functions that pull from the properties to get your values, there is extra code in the file that I never used, left over from dev cycle like 20 months ago.

    • Pull in the list of languages, throw up a dialog, then set the new language, fire an app level event to trigger listeners that would update already displayed elements

      daButton.addEventListener('click', function(e){
    var temparray = [];
    for (var j = 0; j < languages.languageList.length;j++){
        temparray.push(languages.languageList[j].title);
    }
    temparray.push(languages.L('cancel'));
    var dialog = Ti.UI.createOptionDialog({
        title: languages.L('select_language'),
        options:temparray,
        cancel: temparray.length-1
    });
    dialog.show();

    dialog.addEventListener('click', function(e){
        // add if not cancel
            if (e.source.selectedIndex !== (temparray.length-1)){

                // set the default language
                console.log(languages.setLanguage(e.source.selectedIndex));

                //change menu title
                menu.items[0].title = languages.languageList[e.index].title;
                // refresh the tables and ui
                Ti.App.fireEvent('switch_language');
            }
    });
});
