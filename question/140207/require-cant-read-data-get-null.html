    I looked at your code, and everything looked correct. So I ran the code, and it worked fine with no changes.

    [INFO] getSomething

    If you're still having trouble, I might suggest breaking up your click event listener so that you can examine things more closely in the debugger:

    btn.addEventListener('click', function(e){
    var api = require ('api');
    var strval = api.getSomething ();
    Ti.API.info(strval);
});

    After the require() call, api should contain an object, and you should be able to see that it has a getSomething() function.
