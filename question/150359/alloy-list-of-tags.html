  • You can refer to the Titanium Guides where the XML Markup syntax is explained.

    There are no complete list per se, but all the Titanium UI elements abide by the same convention:

    • Ti.UI.Window or <Window>
    • Ti.UI.TabGroup or <TabGroup>
    • Ti.UI.iPad.SplitWindow or <SplitWindow>
    • And so on…
