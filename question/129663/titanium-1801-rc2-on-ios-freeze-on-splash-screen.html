    1. If you used distribution to release the app, and it freezes on device, debug/run the project immediately after the distribution is done, this seems to run the project using distributed code, it may allow you to see the error causing the freeze in debug window

    2. In my cause, the error is something like this: http://developer.appcelerator.com/question/85671/invalid-method-createtoolbar-passed-to-uimodule-on-device, I didn't find out why this is happening, but this fix: http://developer.appcelerator.com/question/85671/invalid-method-createtoolbar-passed-to-uimodule-on-device#answer-212573 does work for my case and fixed the freeze permanently on sdk 1.8
