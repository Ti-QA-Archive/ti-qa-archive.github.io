    Hi Matteo, to get started with xml parsing you could begin with this ParsingXML tutorial.

    I never explored solution for XML-tableView databinding in Alloy, and I guess it is not as immediate as with alloys' backbone collections, but possibly achieved by extending the collection with a customized parser method.

    So my first try would be to write the parser code to build up an array of tableViewRows objects and assign it to the data property of a tableView instance. Having this working, I'd try defining an alloy model then extend its collection with that parser function, as final step I'd implement Alloy's databinding capabilities with the tableView.
