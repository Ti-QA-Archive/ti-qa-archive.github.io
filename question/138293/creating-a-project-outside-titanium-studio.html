Creating a project outside Titanium Studio

Hi everyone,

I'm working on a project that require creating Titanium applications on the fly, based on an existing model.
Basically, I will have all my 'ready-to-tune' app in a git repo that I will clone, generate a GUID/Bundle ID/…, customize it a little and voila here is a new Titanium app.

But doing this seems to cause trouble with licensed modules (license validation error) and the 'my account' page on titanium, not reporting all the apps.

So I'm wondering what is the incidence of Titanium Studio in creating a Titanium project ? Does creating a new project require to send data to Appcelerators servers, and if it does, is there a command-line way to do the same ?

Thanks !

  • The project creation step in inside Titanium Studio makes an API call to create the application in your account page. You can create projects using the command-line, but they won't create a corresponding entry in my.appcelerator.com. There isn't a command-line way exposed to do that piece at the moment, but I do believe that the new command-line SDK that is under discussion/development will assist there.
