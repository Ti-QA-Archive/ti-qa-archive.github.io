    i was not sure if this question should be answered with more detail than yes as you have not provided even the most basic information, for example; Alloy or non-Alloy?

    The solution is to create a container view with no top, bottom, left or right set but has the layout property set to vertical.

    • Titanium without Alloy
    var container = Ti.UI.createView({
  height: Ti.UI.SIZE,
  layout: 'vertical',
  width: Ti.UI.SIZE
});
var btn1 = Ti.UI.createButton({
  bottom: 5, // acts as padding
  height: 40,
  title: 'One',
  width: 100
});
var btn2 = Ti.UI.createButton({
  height: 40,
  title: 'Two',
  top: 5, // acts as padding
  width: 100
});
win.add(container);
container.add(btn1);
container.add(btn2);

    • Titanium with Alloy Framework

      example.tss

    ".container": {
  height: Ti.UI.SIZE,
  layout: 'vertical',
  width: Ti.UI.SIZE
},
".btn": {
  height: 40,
  width: 100
},
".btntop": {
  bottom: 5 // acts as padding
},
".btnbottom": {
  top: 5 // acts as padding
}

    example.xml

    <Alloy>
  <View class="container">
    <Button title="One" class="btn btntop" />
    <Button title="Two" class="btn btnbottom" />
  </View>
</Alloy>

    Having just painfully typed this on an iPad I am hoping there are no errors, but forgive a typo or two.
