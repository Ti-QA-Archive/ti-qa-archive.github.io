Opening a window as a Popover

I have a small login window that, on iPhone and Android, opens as separate window. On iPhone it's part of a NavigationGroup, and on Android it's modal.

On iPad, however, the appropriate thing would be to open the window in a (smaller-than-the-screen) Popover.

Essentially I'd like to not have to rewrite the code, and be able to reuse my existing loginwin.js, which currently is found in:

    var login = Ti.UI.createWindow({
        url: 'loginwin.js',
        [etc.]
    });

For the iPhone version, opening this as navGroup.open(login) works fine. On Android, it's simply login.open();

But for the iPad version, adding login to my popover via popover.add(login) doesn't work – the login window is never shown: there's just an empty Popover.

Is this possible, or am I looking at rewriting this (and several other similar controls)?

    It may be worth considering using a view rather than a window, adding it last to the window to ensure that it appears above other objects (or by using zindex) depending on what you intend to achieve.

    However, in this case, you need to make the window "modal" and set the window size to be less than the screen size. If you want the parent window to have the blur effect when the child opens, use modal:true, otherwise use fullscreen: false:

    var login = Ti.UI.createWindow({
  url: 'loginwin.js',
  fullscreen:false,
  height:300,
  width:300,
  [etc.]
});

    If the point of your question is more about conditional code depending on the platform, see the example in version.js

    Cheers
