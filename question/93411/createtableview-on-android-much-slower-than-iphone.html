createTableView on Android much slower than iPhone

We are creating about 850 rows for display in a table that users can search.
var rows = [];
for (i = 0; i < 850; i++) {
var row = {title:'test'};
rows[i] = row;
}

Titanium.UI.createTableView({
top: 0,
height: 300,
data:rows,
});

setting the data to rows takes 3-4 seconds on my emulator in Android where as in iPhone it takes less than 1 second.

setting className on the row didn't decrease the time and may have even increased it by 1 - 2 seconds.

Is there a way to make this run in less than 1 second?
