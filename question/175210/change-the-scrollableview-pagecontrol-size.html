Change the scrollableview pagecontrol size

Is there a way to make the page control dots bigger for scrollableview?

  • Hi

    Short answer - yes.

    Longer answer;

    You have to create them. The indicator dots (paging control) are designed to work within the UI guidelines. Their placement and style are predetermined. But should you wish; turn of the existing ones and make your own - it is very easy.

    You only need to create a parent view for them, add a view inside for each page and make the borderradius half the width of each (do they appear circular). Then each time the scrollable view event for scrollend occures update the colours of the circle views. If you add the views using an array then it is even easier as the scrollend event gives you the page index which will match the dots.

    // not complete code
var dots = [];
var page = 0, pages = 3; // change to suit
for (page = 0; page < pages; page = page + 1) {
    dots.push(Ti.UI.createView({
        backgroundColor: '#fff',
        borderRadius: 25,
        height: 50,
        width: 50
    }));
}

    Then on the event change the background color of the current page to something else and the rest to unselevted.
