TitanTricks: sample project with extended components

Just a few lines to introduce you TitanTricks, an open source project for Titanium Mobile with code samples and reusable and extended Titanium components you can use in your projects.

By now, these are the samples includes:

  • Graphic switch button
  • LongClick button event
  • Custom progress bar
  • Flip Image
  • Badge Image
  • Create image reflection fx
  • Resizer/optimizer image
  • Table with columns samples

More on the way. Hope you enjoy it, don't hesitate to participate in the project if you like, and subscribe to this question to get updated about new components.

https://github.com/jaraen/TitanTricks
