    Hi Nicola.

    Well not everything works the same across both platforms. I cannot comment on this issue as I have never tried it, but generally if it doesn't show on the emulator it won't show on the phone.

    But most of the components properties do work across both platforms in certain cases there are differences.

    What os are you using and the vernon of studio and SDK… The latest nightly build 1.8 is much more stable for Android development.

    And can be found here.

    http://builds.appcelerator.com.s3.amazonaws.com/index.html

    Hope this helps.

    • Hi Nicola.

      You can change the sdk being used in the tiapp.xml file.

      The nightly build is the latest beta sdk, you should be on 1.7.5/6 as the release build but they are working on 1.8 this is stored in /Library/Application Support/Titanium/mobilesdk/Osx

      You can setup studio to automatically download the latest nightly builds. It should automatically update the release builds.

      T.
