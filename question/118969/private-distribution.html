Private distribution

Is it possible to distribute an App for private company use only (e.g. not to appear in the Markets)?

  • Yes, it is very possible. Please see my reply to a similar question here. There are two basic licensing methods, but in either case you may install over the air (if the device is running OS4) or install via iTunes or iPhone Configuration Utility.

  • Android you can just create the app under distribution and send it to the user to install (over the internet download link).

    iPhone you have to have a developer license and create the apps through ad hoc distribution. Within the organizer you can email the apps to the user to add within iTunes. You can also sign up as a enterprise company and distribute the applications that way.
