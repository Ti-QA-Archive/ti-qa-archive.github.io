  • You are probably using Node.js v0.10. Titanium 3.0.2 and older does not support Node.js v0.10. You must either use Node.js v0.8.x (http://nodejs.org/dist/v0.8.22/) or use a Titanium 3.1.0 CI build (http://builds.appcelerator.com/). Note that CI builds are not fully tested and should be used for development only.
