How & When to Close Windows?

I want to make sure I am setting up my app architecture in such a way that I am not creating memory leaks. My app (which is my first) is very simple:

  • It has only three tabs;
  • Each tab has one window;
  • Each window has multiple views;
  • Each view has buttons, tableviews, and a few other UI elements
  • Based on user interaction, I make the views 'visible' property true or false to show and hide the various groupings of UI elements

This is the entire text of my app.js file:

Titanium.UI.setBackgroundColor('#000');

var db = Titanium.Database.install('/data/MyDB.sqlite', 'MyDB');
db.close();

var nsWins = {
    win1: Titanium.UI.createWindow({ title:'Win 1 Title', backgroundColor:'#fff', barColor: 'orange', url: 'tab1/tab1.js' }),
    win2: Titanium.UI.createWindow({ title:'Win 2 Title', backgroundColor:'#fff', barColor: 'orange', url: 'tab2/tab2.js' }),
    win3: Titanium.UI.createWindow({ title:'Win 3 Title, backgroundColor:'#fff', barColor: 'orange', url: 'tab3/tab3.js' })
};

var nsTabs = {
    tabGroup: Titanium.UI.createTabGroup(),
    tab1: Titanium.UI.createTab({ icon:'/images/icnTab1.png', title:'Tab 1', window:nsWins.win1 }),
    tab2: Titanium.UI.createTab({ icon:'/images/icnTab2.png', title:'Tab 2', window:nsWins.win2 }),
    tab3: Titanium.UI.createTab({ icon:'/images/icnTab3.png', title:'Tab 3', window:nsWins.win3 })
};

nsTabs.tabGroup.addTab(nsTabs.tab1);  
nsTabs.tabGroup.addTab(nsTabs.tab2);  
nsTabs.tabGroup.addTab(nsTabs.tab3); 
nsTabs.tabGroup.open();

In each individual .js file, all the variables / elemets that would be global are all grouped under a single namespace so as not to pollute the global namespace.

My question is this: Am I doing this right? And I am adequately preventing memory leaks? At no point in my app am I closing any windows, tabs or tab groups. Is this bad? Will all the buttons, tableviews etc be destroyed along the way?

(Hopefully I've given enough info to answer my question.)

I'm new to all of this, so any help is appreciated!

Thanks
