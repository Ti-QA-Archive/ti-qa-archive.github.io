How to open Password protected PDF files inside the app

Hi,
I want to develop an app where the PDF's are password protected and the passwords for the PDF are stored inside the database.

If the user opens the PDF, I want the app to fetch the password associated with that PDF from the database and than open the PDF without prompting the password dialog box.

I want to do this so that if the user hacks the app and get the PDF's they still dont have the passwords. (to protect the content)

Is this possible?

  • iOS or Android?

    Native iOS viewer does not allow open protected files, so you need a module pdf viewer to do this. I've developed my own, contact me if you're interested. There are also some good modules in the marketplace.

    For android, you can a) use a native module or b) I've never tried, but I think you could launch an external app using Android intents and specify the password for the file.
