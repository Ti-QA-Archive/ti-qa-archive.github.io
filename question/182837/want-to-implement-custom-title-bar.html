    Hello Bunty Rohra,

    Title bar of the app is bit of tricky i found and finally i find out how to do it.

    to the first you have to overwrite default theme

    1. create a new theme under
      yourProjectfolder/platform/android/res/values/MyTheme.xml

    and MyTheme.xml should content if you are running titanium sdk bellow 4.0

    <?xml version="1.0" encoding="utf-8"?>
<resources>
    <style name="Theme.NoActionBar" parent="@style/Theme.AppCompat">
        <item name="android:windowActionBar">false</item>
        <item name="android:windowNoTitle">true</item>
    </style>
</resources>

    if you are running titanium sdk 4.0 or above, then your MyTheme.xml should contain 

    <?xml version="1.0" encoding="utf-8"?>
<resources>
    <style name="Theme.NoActionBar" parent="@style/Theme.AppCompat">
        <item name="windowActionBar">false</item>
        <item name="windowNoTitle">true</item>
    </style>
</resources>

    then open tiapp.xml
    replace android area

    <android xmlns:android="http://schemas.android.com/apk/res/android">
        <manifest>
            <application android:theme="@style/Theme.NoActionBar">
                <!-- Need to specify at least API level 11 for Titanium SDK 3.2.x and prior -->

            </application>
        </manifest>
    </android>

    In your window object

    var win = Ti.UI.createWindow({
        backgroundColor : '#EEEEEE',
        exitOnClose : true,
        fullscreen : true,
        title : 'MyTitle',
    });

    Hope this may help you..

    good luck..
