  • I don't think anybody has ever gotten scons to work on Windows. :) According to your error, you're probably missing the ANDROID_SDK environment variable. Even if you set that, you'll probably still run into issues.

    FWIW, we're in the process of replacing scons with something awesome, but there's to show today.

    Also FYI, the Titanium SDK builds for Windows are actually built on a Linux box. Eventually we'll get that new scons replacement done and Windows builds will actually be built on a Windows box.
