Get height of a label created with height property set 'auto' in Android

How is possible?

var text1 = Titanium.UI.createLabel({
    text: 'sample',
    color: 'white',
    textAlign: 'left',
    width: 300,
    height: 'auto',
    font:{fontSize:14, fontWeight: 'bold'},
    top: 30,
});

h1 = text1.height;

In iOS h1 get the size but in Android return 'auto'
