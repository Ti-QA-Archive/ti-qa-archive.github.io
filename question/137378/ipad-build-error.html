  • Try this!

    1. 1) First, backup Resource fold of your project
    2. 2) Remove the project in Titanium Studio (make sure delete project contents on disk)
    3. 3) Create new Mobile Project as a sample.
    4. 4) Important!! You must build the Sample Project immediately.
    5. 5) Confirm the Sample Project has build without an error.
    6. 6) Now, copy origin resources to the Project and Rebuild!
