Where can I download the Android SDK Tools rev 22?

I am trying to set up the latest version of Titanium Studio for the first time but am having issues. I see that the setup documentation indicates that I need Rev 22 of the Android SDK. From the Android SDK tools website (http://developer.android.com/sdk/index.html), I only see rev 23 made available and no way to download Rev 22.

Here is the install guide that notes you need 22: http://docs.appcelerator.com/titanium/latest/#!/guide/Installing_the_Android_SDK-section-29004851_InstallingtheAndroidSDK-RequiredAndroidPackages
