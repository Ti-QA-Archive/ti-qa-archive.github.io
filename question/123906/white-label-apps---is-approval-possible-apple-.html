White-label Apps - is approval possible (Apple) ?

Is it possible to submit different versions of same App to Apple Appstore?

To clarify, we would create a framework with Appcelerator, but content (images) would be different for each App. Think of Car image App. One App shows images about Audi, another one shows images about Toyota.

Thanks for any help

    • And don't submit them all at the same time. Apple has rejected batch submissions because they don't represent uniqueness. Couple of guys made "college" apps, with information about various schools. They were not rejected for copyright issues (which would have been my first thought), but that they were too similar.

  • In my previous job last year, I did exactly this. We built an app concept with a core set of features and re-branded it for different clients in the same type of business. There are at least 3 apps in the App Store based on this and I had no trouble getting them accepted.

    They DO have unique IDs, of course.
    And we did submit them over a period of a few months, but I do feel like Apple allows these in certain contexts.

    THere is another app developer group who has done the same thing and I found at least 50 iterations of their app, rebranded for a client each time.
