How to add an Event Listener to an Array?

How may I add an event listener to an array, note this array is being randomly executed with Math.Random. Each array contains objects that are buttons, and I'd like to add an event listener to each one.
Here's my code example:

 buttonQ1[0] = Ti.UI.createButton({
    title: 'Hello',
    font:{fontSize:15,fontFamily:'Helvectica Neue'},
    textAlign: 'center',
    top: 160,
    width: 'auto'    
});

buttonQ1[0].addEventListener('click', function(e){
  var dialog = Ti.UI.createAlertDialog({
    title: 'Enter text',
    style: Ti.UI.iPhone.AlertDialogStyle.PLAIN_TEXT_INPUT,
    buttonNames: ['OK']
  });
  dialog.addEventListener('click', function(e){
    Ti.API.info('e.text: ' + e.text);
  });
dialog.show();

When the addEventListener is removed the code runs, and when the addEventListener is present I receive a parse error on the last line of the document. How can i fix this issue? Please & Thank you.
