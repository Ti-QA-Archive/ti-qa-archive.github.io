Margin and padding ( Left and right)

Hello,

How can i put margin or padding in may main view ( ScrollableView)
If I put : 

var form = Titanium.UI.createScrollView({
    // backgroundColor : 'transparent',
    top : 10,
    left : 10,
    right : 10,
    width : '100%',
    height : '100%',
    layout : 'vertical',
    contentHeight : 'auto'

});

all my fields overfill 10 pix on the right of the screen
I should put width to 90%, but I don't like it.

I can't resolve it neither putting : 

paddingLeft:10,
paddingRight:10,

it happens the same thing

How should I do???

  • Hi

    The easiest way is to create an inner view and give that the "margins" you need - it handles the layout and you add items to that instead of the "outer".

    var form = Ti.UI.createScrollView({
    height: Ti.UI.FILL,
    contentHeight: Ti.UI.SIZE,
    contentWidth: Ti.UI.FILL,
    width: Ti.UI.FILL
});
var inner = Ti.UI.createView({
    bottom: 10,
    height: Ti.UI.SIZE,
    layout: 'vertical',
    left: 10,
    right: 10,
    top: 10,
    width: Ti.UI.FILL
});
form.add(inner);

var something = Ti.UI.createLabel({
    text: 'Add things to inner'
});
inner.add(something);

    See how that goes.
