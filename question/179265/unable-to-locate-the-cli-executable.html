Unable to locate the CLI executable

I recently installed node usin nvm and since then I'm unable to lunch simulator. I'm getting the following error:

Launching simulator process failed
Unable to locate the CLI executable. Please restart Studio or use Help > Check for Titanium Updates to install Titanium CLI.

Tried to restarts titanium studio several times but didn't help. Also Help &gt; Check for Titanium Updates says that no updates were found…

  • Studio installs node for you. If you've reinstalled or installed a new version using nvm, you've likely confused your system. Global/executable modules are typically installed to /usr/local/bin. Perhaps nvm has put them elsewhere? You could create a symlink. Or, under your new version of node, install titanium and alloy with:

    sudo npm install -g titanium
sudo npm install -g alloy

    Symlinking is the better solution so that you don't have to maintain multiple versions of your modules.

    Another, perhaps more drastic solution is to uninstall/reinstall Studio so that it detects your new nvm/node environment and installs the required modules for you.

    If you're using Windows (you didn't say), leave off the sudo. Also, Windows installs global modules someplace else that I'm forgetting right now and doesn't offer symlinking without an add-on utility.
