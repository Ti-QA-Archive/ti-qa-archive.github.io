Activity - how to pass a constant to startActivityForResult

I'm looking at the Android documentation to start a request to enable Bluetooth and it gives me this:

startActivityForResult(enableBtIntent, REQUEST_ENABLE_BT);

But the Titanium function is:

Titanium.Android.Activity.startActivityForResult(intent, callback);

Where the callback is a function. How do I set the REQUEST_ENABLE_BT in titanium?

    activity.startActivityForResult(intent, function(e) {
    // The request code used to start this Activity
    var requestCode = e.requestCode;
    // The result code returned from the activity

    var resultCode = e.resultCode;
    // A Titanium.Android.Intent filled with data returned from the Activity
    var intent = e.intent;
    // The Activity the received the result
    var source = e.source;
});
