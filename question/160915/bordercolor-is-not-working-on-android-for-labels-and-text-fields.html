borderColor is not working on android for labels and text fields

Hi all,

The property borderColor is not working on android, neither for labels nor for textFileds.

<Label borderColor= "#2279bc" borderRadius= "4">Test</Label>
<TextField borderColor= "#2279bc" borderRadius= "4"></TextField>

It's working normally on ios but no visible border on android.

  • Hello Ahmed,

    We did some testing on this, and we got the same conclusion than you.

    Test case

    var win = Ti.UI.createWindow({backgroundColor:'white'});
var label = Ti.UI.createLabel({
    borderColor:"pink",
    top:"40",
    left:"40",
    borderRadius:"4",
    text:'Test',
});
var tf = Ti.UI.createTextField({
    borderColor:"pink",
    top:"80",
    left:"40",
    borderRadius:"4",
    width:'40',
});
win.add(label);
win.add(tf);
win.open();
