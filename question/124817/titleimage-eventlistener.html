titleImage eventListener?

I'm building an app and I need to know how to create a window when the title image is tapped. I've been trying everything I know and I haven't had any luck so far.

At first I thought it wasn't possible, then I noticed that the new LinkedIn app logo in the nav bar opens a window when it is tapped.

Does anyone have any tips or code that could help me?

I would really appreciate it.

    On a window you can set a titleControl, which is a view that can be set in place of the window title. You can create a view to use as the titleControl and attach listeners to that view in the normal manner.

    • Thank you very much. With that information I found this …

      var tb4 = Titanium.UI.createButton({
      title:'Button',
      height:40,
      width:200
      });

      Ti.UI.currentWindow.setTitleControl(tb4);

      I knew there had to be a way, but I wasn't sure how to do it.
      The code above is working.
