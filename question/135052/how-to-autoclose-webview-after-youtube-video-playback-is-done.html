How to autoclose webview after youtube video playback is done

Titanium Mobile SDK 1.8.2

iPhone SDK 4.3

Hi,

I have 2 things that i'd like to achieve with this script. First, the autoplay doesn't seem to work as I have yet to manually click play to play the video. Second, How can I autoclose the webview after youtube video is done?

here's my code. I created a function because I have lots of videos links in the window.

function playThisVid(view, video){
    // Set the window orientation 
    view.orientationModes = [ Ti.UI.LANDSCAPE_RIGHT ];

    // Build the movie URL. 'fs' and 'autoplay'
    var movieId = video;
    var movieUrl = "http://www.youtube.com/embed/" + movieId+ "?fs=1&autoplay=1";

    webview = Ti.UI.createWebView({
        url : movieUrl,
        zIndex: 10
    });

    view.add(webview);
}

//event listeners for videos

    Hi Nash,

    create one view and add button as well as your webview and on click event of the button close view.Like this..

    
function playThisVid(view, video){
    // Set the window orientation 
    view.orientationModes = [ Ti.UI.LANDSCAPE_RIGHT ];

    // Build the movie URL. 'fs' and 'autoplay'
    var movieId = video;
    var movieUrl = "http://www.youtube.com/embed/" + movieId+ "?fs=1&autoplay=1";

var Viewweb = Ti.UI.createView({
//== your code for position i.e height width top left 
});

var backButton = Ti.UI.createButton({ 
//== your button position i.e height width top left
});
backButton.addEventListener('click',function(e) {
     Viewweb.close();
 });

Viewweb.add(backButton);

    webview = Ti.UI.createWebView({
        url : movieUrl,
        zIndex: 10
    });

    Viewweb.add(webview);
}
