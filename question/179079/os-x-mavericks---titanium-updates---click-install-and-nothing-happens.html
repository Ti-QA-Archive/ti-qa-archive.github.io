OS X Mavericks - Titanium Updates - Click Install and nothing happens.

Hello, I just installed Titanium Studio, this window comes up called Titanium Updates which shows Titanium CLI, Node.ACS, and Alloy are all required, however, when I click the install button, I can see the install button color change, but absolutely nothing happens.

  • Hi Jeffrey Roberts,

    Looks like your update is broken. Use below commands to update,

    • Alloy - command sudo npm update -g alloy
    • node.ACS - command sudo npm update -g acs
    • Titanium CLI - command sudo npm update -g titanium

    Then reopened Titanium Studio, went to Help –>Check for Titanium Updates, and the only remaining item was Titanium Studio itself. Update studio, then it should work.

    For more information, check the link: Titanium Studio hangs on Update

    Hope this will help you to solve your problem.
