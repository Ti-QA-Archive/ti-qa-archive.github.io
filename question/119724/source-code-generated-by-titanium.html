Source code generated by Titanium

Hi all,

I would like to know if Titanium, when compiling a project, translates my JavaScript code into Java/Objective-C code. In order to make further improvement to the final code it would be useful to access that code. If it does, do you know where I could find the generated code?
Because in the build folder there are only data files (images, texts etc…) and a few Java classes (in case of Android) containing simple Titanium references and not the logic of the application.

Thank's for your help!

  • Since the Java/Objective-C code is only an intermediate I suspect it is deleted after the application binaries have been generated from it. I am also not sure if this code would allow for much optimization. As far as I know only the API stuff is actually compiled, i.e. all your logic remains JavaScript and is interpreted at runtime on the phone.
