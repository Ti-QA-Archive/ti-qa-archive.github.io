  • Backing up Stephen's answer, There really isn't any difference between the two studios, for developing the application. The paid for version does allow some additional features but not in coding.

    So to answer your question, pressing ctrl+space will give you code completion.

    I have tried moment in the editor and it works by default, however moment.js is a third party module, not part of the Titanium tools.

    Appcelerator Studio is a customised version of the eclipse IDE, and has built in code snippets which should appear at the bottom left of the IDE. If its not then from the top menu select window->Show View and the snippets & samples options are there.

    When asking a question please follow the guidelines and let us know what platform and version of the tooling your using. As you may have something setup wrong but from the info you have provided, we cannot guess.

    Hope this helps
