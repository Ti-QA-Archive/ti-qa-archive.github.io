  • Accepted Answer

    To get this working, you'll need Titanium 3.3.0 (the Release Candidate was just announced in the Blog: Announcing the 3.3.0 Release Candidate of Titanium SDK and Studio)

    Next thing you need to do is: remove this from your tiapp.xml:

    <uses-sdk android:minSdkVersion="11" android:targetSdkVersion="19"/>

    You simply don't need that. Titanium will automatically target the latest Android Version and you do not have to define a minSdk (it will be compatible with API Levels lower than 11 with AppCompat)

  • Benoli,

    Android themes are a combination of files and information placed in your tiapp.xml file. Since you are using Alloy, your theme file should be in the directory platform\android\res.

    Then in your tiapp.xml file you need a line like the following:

    <application android:label="@string/app_name" android:theme="@style/NAME_OF_YOUR_THEME_FILE">

    Check to make sure both files are setup properly. You also need to generate your theme. I have used this theme generator in the past and it works well.

    So the proper steps are:
    1) Generate the theme file. See the Theme Generator link.
    2) Unzip the theme files and copy into platform\android\res directory. Take note of the name of the file in platform\android\res\values\NAME_OF_YOUR_THEME_FILE.XML
    3) Put the theme reference in your tiapp.xml file.

    Hope this helps.

    Ray
