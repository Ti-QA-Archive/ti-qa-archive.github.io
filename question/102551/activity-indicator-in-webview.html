Activity indicator in WebView

Hello I need some help.

In the event 'beforeload' of a webview show a Activity Indicator and hide it on 'load' event. This is Ok. the problem came when the webview redirect to another page and prompt an alert to remember the password. At this moment the activity indicator is over the alert and I can´t do nothing.

Suggestions are welcome

  • While I don't have a solution for you, one thing I do is clear the activity indicator after a certain amount of time. I find that most pages have loaded fully in 8 seconds & if they haven't usually it's a minor element that is taking forever.

    function getWebView(url){
    var loading = false;

    var vwWeb = Ti.UI.createWebView({
        left: 0,
        right: 0,
        top: 0,
        bottom: 0,
        scalesPageToFit: true,
        url: url
    });

    vwWeb.addEventListener('beforeload', function(e){
        if (!loading) {
            loading = true;
            //Start Activity Indicator Here
            setTimeout(function(){
                if (loading) {
                    loading = false;                    
                    //Hide activity indicator here
                }
            }, 8000);
        }
    });

    vwWeb.addEventListener('load', function(e){
        if (loading) {
            loading = false;            
            //Hide activity indicator here
        }
    });

    return vwWeb;
}
