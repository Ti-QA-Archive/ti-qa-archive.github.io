    Perhaps you could use this on Android?

    Ti.Android.currentActivity.addEventListener('pause', function(e) {
   // your code
});

    You can watch for any of the lifecycle events: pause, resume, destroy, etc. See http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.Android.Activity-object.html as well as the Android developer docs.
