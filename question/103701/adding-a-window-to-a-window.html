Adding a window to a window

Hi,

I'm developing a simple app with Appcelerator these days and it's working fine on IPhone / IPod. Because I wanted to make a completely custom bottom navigation, I didn't use a tab group but put my own buttons on the bottom.

So I have a main window and then add 3 other windows (which are connected to the 3 buttons on the bottom) to that window.
Code looks something like this:

function openWindows() {
window1.open();
window2.open();
window3.open();
mainwindow.add(window1);
mainwindow.add(window2);
mainwindow.add(window3);

window3.visible = false;
window2.visible = false;
}

And when a button on the bottom is clicked, I make the wanted window visible and the others invisible.

This works perfectly on IPhone / IPod but on Android it seems that you can't add windows to windows. When I try to run it in the emulator the traceback says something like "error: trying to call getView on a Window".

So is my way of adding windows to a window simply wrong or is this not implemented on Android?

    A window cannot be added to another window, as this is not supported. You should either open, rather than add, the child window, or add a view to the parent window instead.
