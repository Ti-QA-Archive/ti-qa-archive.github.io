  • Yes, you can. One way to do it through a user generated event:

    var win = Ti.UI.createWindow({
    backgroundColor : '#fff'
});

var button = Ti.UI.createButton({
    height:100,
    width:100,
    backgroundImage:'http://example.com/fake.gif'
});
win.add(button);

button.addEventListener('click', function(){
    button.backgroundImage = 'http://example.com/anotherfake.gif';
});

win.open();

    If you are wanting to change it remotely, I suppose you could just replace the image with another file of the same name. The next time the app retrieves the file, it should get the new image.

    I don't think that jpgs would work as background images, though that is a guess. Png or gif should. Also, you might want to try using a a View in place of a button, as that can work better in many cases. Same would apply, just change backgroundImage.
