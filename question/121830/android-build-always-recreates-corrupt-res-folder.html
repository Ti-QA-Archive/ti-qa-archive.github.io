Android build always recreates corrupt "res" folder

Maybe this is normal, but when my app gets run (in Titanium Studio) the 'res' folder under 'android/build' is recreated with only two kinds of images folders and my custom images (i.e. tab icons) are left out completely. The consequence is, obviously, that the images aren't displayed.

I must delete the content of 'build' folder to make all files populate correctly.

Is there a solution for this behavior?
Thanks!
