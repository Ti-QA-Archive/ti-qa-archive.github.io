  • HI Vasilis,

    in my projects I'm using sqlite. In most cases the DB is smaller then ca. 2MB.
    In this case I test at start time if the remote DB is newer as the local (a JSON object on web server says me this).
    You can transfer the newer sqlite from net and change after downloading the file. It is a little with magic. ;-)) ut it worls like a charm. SQLITE is everytime smaller then a transfered JSON and you avoid the calculation for parsing.
