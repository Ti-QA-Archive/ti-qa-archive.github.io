Save PNG to Photo Gallery

Hello,

I have a .PNG file in the applicationDataDirectory and want to save the image in the iOS Photo Gallery.
I use Titanium.Media.saveToPhotoGallery(); and it always saves the image as .JPG and not as .PNG

I have tried to set the extension manually, but if I do the image won't be saved to the gallery?!

WORKS:

Titanium.Media.saveToPhotoGallery(image);

DO NOT WORK:

Titanium.Media.saveToPhotoGallery(image + '.png');

Running the latest XCode, iOS Simulator and Ti-SDK

Any suggestions? :-)

    Hi Fabian,

    saveToPhotoGallery() always saves with same type as your source image, I tested it and it works as expected.
    Are you sure your source image is really a png?

    You can check the mimeType of your image with this code

    var image = Ti.Filesystem.getFile(Ti.Filesystem.resourcesDirectory, "/images/someImage.png");
if (image.exists()) {
    Ti.API.info("mimeType = "+image.toBlob().mimeType);
    //Ti.Media.saveToPhotoGallery(image);
}
