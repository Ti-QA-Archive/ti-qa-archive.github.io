where clause on Alloy Collections

Hi,

function filterFunction(collection) {
    return collection.where({author:'Mark Twain'});
}

This will return the models which has "'Mark Twain" author.
but what if want models which has "'Mark Twain" OR "John Twain" as authors.
In short i want result same as following SQL query

"select * from library where author="'Mark Twain" OR  author="Mark Twain""
