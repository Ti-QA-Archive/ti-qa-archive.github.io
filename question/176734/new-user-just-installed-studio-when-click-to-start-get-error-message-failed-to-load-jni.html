  • This appears to be a Windows Java setup issue. Some pointers:

    • http://stackoverflow.com/questions/23165372/eclipse-failed-to-load-the-jni-shared-library-c-windows-system32-jre-bin-c
    • http://stackoverflow.com/questions/7352493/failed-to-load-the-jni-shared-library-jdk

    A few things to check:

    • Are you using the proper JRE/JDK with your system. i.e. Don't use a 64-bit JDK on a 32-bit machine?
    • You might try using a different version of Java. i.e. Use Java 6 instead of 7, or vice versa.
