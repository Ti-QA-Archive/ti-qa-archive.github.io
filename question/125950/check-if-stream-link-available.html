  • The VideoPlayer object has 2 events that you may want to look into. First is the error event which will fire in the result of a playback error, and the second is the loadstate event which fires when the network loadState changes. You can listen for the changes on the loadState for any given period of time and use that as the timeout. Check out the VideoPlayer Docs for more info on those events.
