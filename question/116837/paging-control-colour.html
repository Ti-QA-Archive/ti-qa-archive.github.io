Paging Control Colour!

This has been asked a lot on this forum, but i wonder if anyone has found an answer?

How do you change the colour of the "DOTS" on the paging control both on the Dashboard and on the scrollable view.?

I've seen it on many apps "facebook" app for example, so it definitely can be done.

Thanks for taking the time.

    The paging control does not have implemented by Apple any kind of method/property to change the dots color.

    http://developer.apple.com/library/ios/#documentation/uikit/reference/UIPageControl_Class/Reference/Reference.html

    You can try to implement one by yourself :)
