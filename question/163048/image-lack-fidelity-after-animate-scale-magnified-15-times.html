Image lack fidelity after animate scale magnified 1.5 times

Hi all, I faced a problem while change from portrait to landscape using animate.
like below codes:

var matrix = Ti.UI.create2DMatrix();
matrix = matrix.rotate(rot);
matrix = matrix.scale(1.5, 1.5);

var anim = Ti.UI.createAnimation({
    top : posn.top,
    left : posn.left,
    transform : matrix,
    duration : 300,
    curve : Ti.UI.ANIMATION_CURVE_EASE_OUT,
});

$.frame.animate(anim);

The images like below, you can find out that second image had lack fidelity within red rectangle after animate.

Is there any method to solve this?or need to submit issue to Titanium?
Thanks very much.

    Hi

    This is expected - you have stretched the image by 1.5 times which means each single pixel is now 1.5 pixels. As there is no such thing as 1.5 pixels - it has to be either 1 or 2 pixels. So the normal solution is to interpolate the pixel rather than simply stretch or compress the pixel.

    Interpolation will simply choose to either make an extra black pixel or instead make a grey pixel so it appears visually correct for the strength of the 0.5 pixels. The logic here is the background is white the foreground is black 50% between those two is grey.
