Ti.UI.createButton makes iOS App crashing.

Dear Appcelerator users,

I'm working on a simple app and my wish is to add a default UI Button to the window. When a create an instance of the UI Button, the app stops immediately at startup. After four days of trying things. I've created a minimal setup for the UIButton, but it still don't start on the iPhone Simulator. I'm confused. This is my minimal setup:

var button = Titanium.UI.createButton({title: 'Hello'});
var root = Titanium.UI.createWindow();
root.add(button);
root.open();

The App opens in the simulator, shows the Appcelerator screen and quits directly. But it's essential part of the App. My question: Does any one know why the app is crashing and how to solve it? It still looks like a bug.

Thank you!

Titanium Studio version: 1.0.4;
Titanium Mobile version: 1.7.2

  • Accepted Answer

    This kind of error is normally caused by Titanium not picking up a dependency on a new type of object used in the project and not including the needed code, try cleaning the project (MainMenu->Project->Clean) and rebuilding.
