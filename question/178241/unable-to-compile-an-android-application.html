  • Hi guys,

    Your main problem is that you are using the JDK version 7. Titanium requires JDK Version 6 (as explained in the compatibility matrix).
    Also, please make sure that you install the 32-bit version. Even if you are running on a 64-bit version of Windows.
    Hope this helps

  • Hi Cédric,

    it maybe a problem with the accented character é inside your workspace-path C:\Users\Cédric\Documents\Titanium_Studio_Workspace

    This is a known issue with accented characters (non ASCII) inside the App name or also the path.

    It is documented in Jira Ticket TIMOB-17270 CLI: App name with accented characters fails to build

    Simply relocate your Titanium_Studio_Workspace to a path without any accented characters and do not use accented characters inside app names.

    Hope that helps.

    Greetings,

    Matthias
