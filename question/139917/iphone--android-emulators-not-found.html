    Following are the scenarios wherein this can happen:

    ###1. If you are using Windows Machine, then:
    a. You don't have an option of iPhone Simulator as it is available only on Mac.
    b. To test Android Apps you need to have Android SDK installed. Refer Android site for details http://developer.android.com/sdk/index.html
    c. For Blackberry, Titanium Studio currently does not supports it

    ###2. If you are using Mac Machine, then:
    a. You have to install XCode based on the version of your Mac
    b. For Android, procedure is same as for windows machine
