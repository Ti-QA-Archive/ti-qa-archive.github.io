    Hi

    It is not possible to change the color of the UI standard switch control. You also cannot change the text - in case you go there next.

    You create your own using images, views and animations though, not ideal and if you are creating a multilingual app you will need to sort the captions translations yourself (they are currently achieved by the OS).

    You can also buy a switch from the marketplace Slider Switch. This is not an exact match for the standard one but provides certain customisations including main color.
