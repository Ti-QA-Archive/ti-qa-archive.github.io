StoreKit returns empty products array

var array=['com.mysite.id1', 'com.mysite.id2', 'com.mysite.id3'];
Storekit.requestProducts(array, function(e) {    
   Ti.API.info('Response:' + JSON.stringify(e));                
   if (!e.success) {
      //no success
   } else if (e.invalid) {      
      // invalid
   }else{
      // code...
   }
});

This is the Response I get after long long time:

{"type": "callback", "products": [], "source": {}, "success": true}

Why is products empty? All my settings (appID, certificates, provisioning) are OK and my array doesn't have any Invalid product.
