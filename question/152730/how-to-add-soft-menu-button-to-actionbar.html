How to add soft menu button to Actionbar?

Hi.

I have implemented an actionBar as in the Ti.Photogram example.

The problem is that I can't figure out how to add a soft menu button (that 3 little squares icon) to open the menu options (that has showAsAction: Ti.Android.SHOW_AS_ACTION_NEVER) on devices that does not have a hardware menu button (as in my Galaxy Nexus).

How can I add the menu icon on the actionbar or anywhere else?

Thank you.

    Now that I understand the overflow menu button, I can tell you that it is easy. :)

    • First you create a Menu.
    • If you have MenuItems with SHOW_AS_ACTION_NEVER, the menu button will show up always.
    • If you have MenuItems with SHOW_AS_ACTION_IF_ROOM, the menu button will show up if there is no space on the actionBar to the MenuItem to show its icon
    • If all your MenuItems are SHOW_AS_ACTION_ALWAYS, the menu button will never show up.

    That's it
