touchEnabled anb webview error

Hi, in IOS i need to create a webview and disable the scrolls, for this i change the property touchEnabled to false, but when i change this the event dont work, for example:


var web         = Titanium.UI.createWebView({
        top : 0,
        left : 0,
        right : 0,
        backgroundColor : 'Transparent',
        height : Ti.UI.SIZE,
        showScrollbars : false,
        willHandleTouches : false,
        touchEnabled : false,
});
web.url         = '/html/index.html';
web.addEventListener('beforeload',function(){
    Ti.app.info('webview befor load');
});
contentLeft.add(web);

If i change touchEnabled to true, the event works fine, but in false no. Any solution?

I am using the titanium version 3.1.1

  • Setting this willHandleTouches to false allows the user to interact with the native web view events like scrolling,zooming and click on the link and not interact with normal titanium touch events.
    For more details go to : http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.WebView-property-willHandleTouches

  • You should try the following:

    <script type="text/javascript">
    document.addEventListener("touchmove", function(e){
        e.preventDefault();
        return false;
    }, false);
</script>
