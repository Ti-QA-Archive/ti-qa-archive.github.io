Facebook Login only works with custom info.plist

Hi,

I just implemented a simple Facebook login based on the documentation found here.

Unfortunately the additional xml node 

<property name="ti.facebook.appid">MY_FB_APP_ID</property>

isn't copied correctly into the info.plist file for iOS.

There it writes following xml node:

<string>fb[Object]</string>

instead of

<string>fbMY_FB_APP_ID</string>

Is this a known bug, or do I need an additional configuration?

Regards,

Christoph

    Refer Kitchen Sink App, and its explained there how to use FaceBook Login.

    custom info.plist is there and with this URL format.

    <key>CFBundleURLTypes</key>
    <array>
        <dict>
            <key>CFBundleURLName</key>
            <string>com.appcelerator.titanium</string>
            <key>CFBundleURLSchemes</key>
            <array>
                <string>kitchensink</string>
                <string>fb495338853813822</string>
            </array>
        </dict>
    </array>

    what is bug in that?
