Change "Back Button" Text in Navigation Group

Greetings,

I am working on an iOS app with the latest SDK and my app includes a navigation group with several windows. Each window has a left navigation button that goes back to the previous window in the stack. The problem is that the button pulls the title from the previous window for its title. What I would like to do is simply change the back button title to "Back". Perhaps I haven't looked hard enough, but I can't find this in the documentation. Everything I've tried so far hasn't worked. Any suggestions would be appreciated.

Thanks!

    Hi

    This should be VERY easy to achieve, chances are you are not looking in the right place as it has nothing to do with the navigation group.

    • backButtonTitle

    Usage example;

    var win = Ti.UI.createWindow({
    backgroundColor: '#fff',
    backButtonTitle: 'Back', // choose your own title for the back button
    height: Ti.UI.FILL,
    title: 'Title Bar',
    width: Ti.UI.FILL
});

  • Here you go my friend suppose navGroup is the name of navigation group

    if (Ti.Platform.osname == 'iphone' || Ti.Platform.osname == 'ipad') {
        var btnBack = Ti.UI.createButton({
            title : 'Back',
            style : Titanium.UI.iPhone.SystemButtonStyle
        })
        self.setLeftNavButton(btnBack);
        btnBack.addEventListener('click', function(e) {
            navGroup.close(self);
        })
    }
