22 * 2.05 = 45.0999999999996 and it should be 45.10 (this without any rounding).

  • The problem is not related with iOS, it's related to floating point numbers: http://en.wikipedia.org/wiki/Floating_point#Accuracy_problems

    For solution of that problem, you can try this library: https://github.com/dtrebbien/BigDecimal.js if calculation precision is important to you, or you can round the result you get using "Math.round" method.
