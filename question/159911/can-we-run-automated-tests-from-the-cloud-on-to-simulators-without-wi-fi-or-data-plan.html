Can we run automated tests from the Cloud on to simulators without wi-fi or data plan

As part of continous integration setup using Jenkins-Soasta Plugin, can we run automated tests (composition) from Cloud on to iOS simulator (on build server) without wi-fi and data plan.
In our company, the build server is going to be in data center with no wireless access.

  • Well, the iOS Simulator actually used the host computer's network connection, whether it's wired, wifi or else. So technically, as long as the machine running the simulator can access the network, your code will be able to access the cloud.
