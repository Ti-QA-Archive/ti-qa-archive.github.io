Animating multiple objects

Titanium SDK 1.5.1, iOS 4.2

I'm trying to animate two different objects at the same time using a "doubletap", however I can't seem to get it to work. Here's generally what I'm trying to do…

// for this example, product contains two child elements, "card" and "thumb"

product.addEventListener('doubletap', function() {
    toggleDetails(card, thumb);
});

function toggleDetails(card, thumb) {
   var t = Ti.UI.create3DMatrix(),
            a = Ti.UI.createAnimation();
if(card.state == 'min') {
            t = t.scale(1);
            t = t.rotate(180,0,0,1);
            a.transform = t;
            a.duration = 500;
            thumb.animate(a);

            card.state = 'max';
            card.animate({ height: 500, duration: 500, curve: Ti.UI.ANIMATION_CURVE_EASE_IN_OUT });
        }
        else {
            card.state = 'min';
            card.animate({ height: 80, duration: 250, curve: Ti.UI.ANIMATION_CURVE_EASE_IN_OUT });            
        }
}

If I just pass the "e" variable into that function alone I can get the card to animate. However, when I try to pass multiple objects in there nothing happens.

Anyone with any idea?
