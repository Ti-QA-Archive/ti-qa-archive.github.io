var db = Ti.Database.open('TiBountyHunter');
db.file.setRemoteBackup(false);

  • I think I got it (!?) - add this function to the alloy collection:

    var collection = this;
db = Ti.Database.open(collection.config.adapter.db_name);
db.file.setRemoteBackup(false); //important! Apple rejected app due to this flag missing on several files!!! (valid for this alloy db file too?)
db.close();
