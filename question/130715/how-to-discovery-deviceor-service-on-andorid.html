    • I tried to use this module and it's workable.

      But notice that the describe of discover method usage on web is wrong.

      After mail to the author, I got the correct usage.

      Discover method should be used as below:

      var jmdns = require('com.looprecur.jmdns')

      jmdns.discover(service_name, domain, function(service) {
      Ti.API.debug("It found "+ service.name + " " + service.host + " " + service.port);
      });

      The service name should be something like "__xbmc-web._tcp" or "_workstation._tcp" and the domain is usually "local."

      If anyone need to discovery device could use the module.
