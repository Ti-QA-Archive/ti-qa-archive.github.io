  • I'm having no problem using Ti.Filesystem functions with Android using mobile SDK 1.7.2. here's an example of me testing for the existence of a file in my Resources path

    var win = Ti.UI.createWindow();

var path = Ti.Filesystem.resourcesDirectory + '/app.js';
Ti.API.debug('path: ' + path);

var file = Ti.Filesystem.getFile(path);
if (file.exists()) {
    Ti.API.debug('file exists!');
}

win.open();

    This will print the expected file path and 'file exists!' when run on the android emulator. Can you copy this into a fresh project and test it? If you still have issues with this sample let me know.
