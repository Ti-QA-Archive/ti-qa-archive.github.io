Underscore - blur multiple text fields with _.each() to hide keyboard

Maybe someone can help me understand why this use case isn't working. I can imagine it conceptually, but having problems implementing it.

I want to solve the ever-present problem of hiding the keyboard when a textfield is blurred (loses focus). I haven't found a satisfactory solution in this forum. But I thought that maybe underscore.js might provide acleaner way of achieving the goal of hiding the keyboard.

So, in my window event listener I have this:

$.nameOfWindow.addEventListener('click', function(e) {

    var arr = $.parentViewName.getChildren();
    _.each(arr, blur);

});

However, blur is undefined in that context. Now, if I use alert instead of blur, it works because alert doesn't really care what you pass it.

Am I headed in the right direction? Will this work? If not, can someone post a viable solution for hiding the keyboard when any textfield loses focus? If I'm close, maybe someone can fix what I've posted.

Thanks.

  • blur is not a variable/ property. It's a method. Where you need to pass the right UI element to blur the keyboard Where here it might be TextFiled, TextArea and so on..

    Okay here is answer How to blur the text filed when it looses focus -

    We have blur event listener. You can do something when the TextField loses its focus by calling the method you want. Let you have a 

     <TextField id="userInput" left="5dip" width="250dip" onBlur="textLostFocus"/>

    function textLostFocus(event) {
    event.source.blur();
}

    you can use for loop to blur all your textFields…..

    var arr = $.parentViewName.getChildren();
var objTxt = null;

for(var i in arr){
    objTxt = arr[i];    
    objTxt.blur();
    objTxt = null;
}
