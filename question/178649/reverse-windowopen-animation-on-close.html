Reverse window.open() animation on close

This might be a dumb question, so forgive me in advance :)

When using an animation to open a window, is there a way to reverse that animation when it closes?

Example, let's say you open a window as such:

win.open({
    transition: Titanium.UI.iPhone.AnimationStyle.FLIP_FROM_RIGHT
});

Is is possible to have the window remember how it was opened, and use FLIP_FROM_LEFT when closing? Or, do you have to pass the state along?

The reason I ask is that I have a window controller that's invoked in a couple ways. In one instance, it's opened as a modal, the other, it's opened with the aforementioned transition. It'd be awesome if the window remembered how it was opened, but I'm not sure if that's even remotely possible.

Thanks!
