Importing an Image from Gallery

Hi, I can open my phones gallery now I want to select an Image and import it into my app how can I do this ?? Please help

  • Accepted Answer

    Simply put:

    Titanium.Media.openPhotoGallery({
    mediaTypes:[Ti.Media.MEDIA_TYPE_PHOTO],

    success:function(e) {
          var image = e.media;
          filename = 'someFileName.jpeg';
          var f = Titanium.Filesystem.getFile(Titanium.Filesystem.tempDirectory, filename+ ".jpeg");
          f.write(image);
          file = f.getNativePath();

        }
});

    You might want to adapt the success block as per your requirements.

    In the above example, you will notice that I have stored the file in the temporary directory of the application. This is mostly suitable, unless you wish to cache the file for a long period of time. If you wish to keep the file for the purpose of the session only, use the temp directory.

    Hope this helps.
