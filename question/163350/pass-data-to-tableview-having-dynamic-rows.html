Pass data to tableview having dynamic rows

Hi All,

I have a tableview like below:
question.js

<TableView id="tabView" onClick="questionClick">
    <Require src="questionRow"></Require>
</TableView>

questionRow.js

<Alloy>
    <TableViewRow>
        <Label id="question"/>
        <Label id="answer"/>
    </TableViewRow>
</Alloy>

I have a json data to insert into this table –

data = (
{
answer = yes;
question = "Was the driver drunk while driving?";
},
{
answer = No;
question = "Was the vehicle insured?";
}
)

Can u please help me with the way to do it? Can we do following way?
$.tabView.data = data;

///// ios platform, titanium sdk 3.2.0, mac OS, Alloy framework ////

    Hello Harsad,

    Try this :

    // your data is :

var data = [{
    answer : yes,
    question : "Was the driver drunk while driving?"
}, {
    answer : No,
    question : "Was the vehicle insured?"
}];

    question.xml

    <TableView id="tabView" onClick="questionClick"/>

    question.js

    var arrRow = [];
for (i=0;i<data.length;i++){
    var row = Alloy.createController('questionRow', {
            question : data[i].question,
            answer : data[i].answer
        }).getView();
        arrRow.push(row);
}

$.tabView.data = arrRow;

    questionRow.xml

    <Alloy>
    <TableViewRow>
        <Label id="question"/>
        <Label id="answer"/>
    </TableViewRow>
</Alloy>

    questionRow.js

    var args = arguments[0] || {};

$.question.text = args.question;
$.answer.text = args.answer;

    you can create row dynamically doing something like above.

    i hope this helps. thanks.
