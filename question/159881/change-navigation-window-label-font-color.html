Change navigation window label font color

I have a navigation window set up. If I want to go back to a previous page, there is the navigation label at the top left of the screen. This is a blue label and has an arrow. Is there a way that I can change the font color of this label and perhaps the color of the arrow? If the arrow cannot be changed, is there a way I can replace it with my own image and position it how I want?

  • Hi Matthew,

    For now, there is no official support to change the back button font color.

    But in 3.2.0 SDK release, you can use navTintColor to change the color. this feature is for iOS7 only.

    You can check this link : https://jira.appcelerator.org/browse/TIMOB-14911
