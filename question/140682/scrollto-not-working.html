I have some problem with this code, it's supposed to load some views and a scrollView, then scroll it to a certain point to hide the png icon (myIcon) but it just doesn't scroll at all.

Any ideas?

 * A basic starting point for your application.  Mostly a blank canvas.

    hi Andres, If you want that your scrollview open with some scrolled position initially then you can use this
    myScrollView.setContentOffset({x:0,y:200}); in place of scrollTo().This will work fine. If you want to use scrollTo then write it in some eventlistener.
    i hope this will help you.
