Titanium API : tried on 1.6.2 and 1.7.2

Please advice. Thanks in advance.

var login = Titanium.Facebook.createLoginButton({
    top: 10, style:'wide'

win.add(login);

    It's hard to tell from the small snippet if you've done the necessary configuration for the button. Have you included your appid and permissions in the Titanium.Facebook module?

    Titanium.Facebook.appid = '[your appid]';
Titanium.Facebook.permissions = ['publish_stream'];
var login = Titanium.Facebook.createLoginButton({
    top: 10, style:'wide'
});

win.add(login);

    • Hi Tony, thank you for your help.

      I figured out that using the below code will force the Facebook login dialog to popup instead of using the official Facebook app dialog.

      Titanium.Facebook.forceDialogAuth=true;

      I set that to false from ios because I didn't want the ios to use safari as the login media. Since I couldn't set a "back" button on safari.
