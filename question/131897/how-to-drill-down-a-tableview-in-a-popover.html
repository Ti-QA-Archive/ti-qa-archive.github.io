How to drill down a tableview in a popover

What I wanna do is to open a tableView in a popover and all tableViewRow has child. Whenever a row is clicked, it slide to the left and open a the child View.
Is it possible?

In Window, I used to open the child like this:

var win2 = Ti.UI.createWindow({
        url: 'some.js',
    });
Titanium.UI.currentTab.open(win2,{animated:true});

But how to do it in a popover? because in a Popover, no more currentTab.

I have searched the web for almost a whole day but still cannot find any sample on this.

Appreciate if anyone can help. Many thanks.

  • I created two separate TableViews and I placed the second one to the right of the first one (outside of the popover width). For example, if the popover has a width of 300, then the second TableView has a property of left: 300. When you click on a TableRow on the first table, you will populate the second table and nimated the first left: -300 and the second one left: 0 so it seems like it's sliding. You also need to create a back button yourself to go back.
