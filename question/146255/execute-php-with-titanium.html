    Le me as clear as possible here:

    • You cannot code a mobile Titanium application using PHP. Titanium only supports JavaScript (CoffeeScript also works, but it's still JavaScript under the hood).
    • The part where you configured the MAMP server in Titanium Studio has nothing to do with PHP development. It is actually used when you want to develop Titanium Mobile Web applications. (still using only JavaScript)
    • Although, if you find the transition from PHP to JavaScript pretty steep. You can use the TiPHP Module that will give you a lot of functions that mimic the more popular functions you will find in PHP.
      • This is still JavaScript though. Meaning that you will have to learn (and use) JavaScript to develop your applications.

    There you have it.

    I hope I made this clear for you.

    • if you want to right PHP within the Titanium then this is not what Titanium really does. It allows you to write JavaScript and this is translated into native code in compile time
    • if you want to write a PHP web service and you want somehow to execute this and get the result into your titanium application then all you want is to check out the documentation of Titanium.Network.HTTPClient
