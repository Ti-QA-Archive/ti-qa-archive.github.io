iPad-Popover: how can I detect hiding/closing

Hi,
in my opentok-project (face2face-video) I'm using a popover with videoview in it. If I click outside the popover, then the popover is hiding, but I can listen the talk inside the current hidden popover view.

My plan: killing of video session after hiding. In the API I cannot find an event. 'blur' and 'close' doesn't work. Or how can I avoid the autohiding of popover?

Best regards
Rainer

PS here a code snippet:

var OpenTokModul = require('module/opentok_view');
    var OpenTokContainer = Ti.UI.iPad.createPopover({
        width : 480,
        height : 540,
        title : 'Videokonferenz',
    });
    OpenTokContainer.show({
        view : actionbutton
    });
    var OpenTok = new OpenTokModul();
    OpenTokContainer.add(OpenTok);
    OpenTokContainer.addEventListener('hide', function() {
        OpenTok.finishSession();
        OpenTokContainer.remove(OpenTok);
    });

    You can prevent the popover from auto-hiding by calling the setPassThroughViews method on the surrounding views. Then the user must use a 'close' button to dismiss the popover
