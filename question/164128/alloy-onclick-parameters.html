Alloy onClick parameters

Hi,

is it possible, in alloy, to pass a parameter to the function called in the onClick ?

I would like something like:

<View onClick="writeMail('a@b.com')">

Thanks!

    Try something like this:

    <View onClick="writeMail"  toEmail="a@b.com">

    and in the controller

    
var writeEmail = function() {
    my_send_email_method(args.toEmail);
}
