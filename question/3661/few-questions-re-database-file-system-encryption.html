Few questions re database, file system, encryption

Have just discovered Appcelerator and joined this forum. Very impressed.

I have a few questions about present capabilities and futures.

  1. Does the API in Titanium 1.0 support read and/or write access to the file system on the mobile phone (e.g. calendar and contact information) ?

  2. Is there support for encryption/decryption of data stored in the database or the file system ?

  3. How about support for Symbian and Java ME (a significant subset of the mobile phone market in Europe and Asia/Pacific region). Is this on the roadmap for a future release of Titanium ?

Thanks

Raju Karia

    1. you can read and write files on the file system, but access to calendar and contact information is different). we do have a contacts API (it's still being improved) - look at contacts.js in Kitchen Sink

    2. we don't have any encryption right now for databases

    3. our next target platform is blackberry. we are in discussions about some additional platforms but there are no firm plans
