  • From what I can understand about your requirements, you should be able to add custom data fields onto the standard user object (e.g., 'company', 'team', 'normalPayment', etc.), then query and update based on those fields going forward. Just make sure that the current query capability suits your needs (see link below and test).

    http://cloud.appcelerator.com/docs/api/v1/users/query
