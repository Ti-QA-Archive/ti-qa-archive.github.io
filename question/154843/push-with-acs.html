  • When you set the icon in the JSON object you're sending to ACS, or using the ACS push dashboard, make sure:

    1) You give the name of the image, without the extension. I always use .png files

    2) You copy the image into your resources/android/images/DIRECTORY LIST WITH RESOLUTION

    So, under the resources/android/images directory, there are a bunch of directories listed by resolution type where you can put resolution specific images that would be included when you compile the app. I've always been too lazy to figure out whether I NEED to put the appicon image file in each directory, or whether I just need to put it in the resources/android/images directory, so I have always just put the icon in all of the directories… but it works! Likely it could just be in the images directory, so you could try just putting it there and see if your icon shows up.

    Good luck.

    Ray
