Titanium 1.2.2 can't find adb

I've just installed Titanium on my Mac, and it wants me to tell it the path to the android sdk in the Profile page. I navigate to the sdk folder, but Titanium complains that it can't find adb or android. android is in the Tools folder where it is supposed to be, and adb has been moved in this version of android sdk to the platform-tools folder (there is a file named adb_has_moved.txt in Tools that explains that). Apparently Titanium isn't happy with that. Is there a fix?
