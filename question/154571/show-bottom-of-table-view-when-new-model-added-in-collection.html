Show bottom of table view when new model added in collection

Hi Friends,

  i am developing an android app using alloy framework, i have table view which used collection view data binding to display rows. When a new model added to collection it automatically displays in view, thats fine, but when new rows comes in table it is not scroll  down automatically, i want scroll down automatically and  show last row when new model added to collection, anyone please help


     <TableView dataCollection="restModel" dataTransform="transformFunction" id="table">
            <TableViewRow id="row" title="{title}" onClick="openChats" accessibilityLabel="{id}" selectedBackgroundColor="#D8D8D8">
                <Label id="title" text="{title}"/>
                <Label id="date" text="{date}"/>
                <ImageView id="avatar" image="{image}"/>
                <Label id="careprovider" text="{careprovider}"></Label>
            </TableViewRow>
       </TableView>

Thanks

  • Add an event listener to the collection to which you are adding models for any relevant events (add, reset, remove, etc…). In that listener you can then use the scrollToIndex() function of the TableView to go where ever you want.
