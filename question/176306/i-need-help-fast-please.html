    • I am concerned you actually mean Can I develop on windows to create my app and then send the files to someone/somewhere else so the iOS version will magically appear. If this is the real question - then very much no.

      There will be no way to debug for iOS, no way of making sure you haven't left any Android specific code in for the iOS version. Most importantly you will not be able to test it yourself properly.

      If you are exporting it so you can do the iOS stuff elsewhere - then as Stephen says - this is a "… truly sad and inefficient approach…". It may be a necessity but it is never going to be anywhere near usable, convenient or sensible.
