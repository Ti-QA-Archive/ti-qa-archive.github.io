  • Well, you are trying to access localhost as your target server.

    While that might work on your desktop PC, it will not work in a mobile app. The reason behind it is quite simple. Even though you are running an emulator on that same PC, the emulator was build to replicate an actual physical device. Meaning that it will have its own IP address. Therefore, if you want to access a web service running on your desktop, you will need to enter the full IP address in the URL.

    One easy way to test this is by entering the url in the mobile browser. If it can reach the server, then it is safe to assume your code will also be able.

    Hope it clears that up for you
