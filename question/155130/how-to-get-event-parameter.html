first mode:
alloy view code:

<TableView id = 'tv' onClick="changeDate">
    <TableViewRow header = "Start Date">
        <Label left="5" id = "lblStartDate"></Label>    
    </TableViewRow>
</TableView>

alloy controller code:

function changeDate(e)
{
    ......
}

second mode:

tv.addEventListener("click", function(e){

    var w = Titanium.UI.createWindow({
        title:e.rowData.title,
        backgroundColor:"#FFFFFF"
    });

});

use first mode,the parameter 'e' is null,but use second mode , I can get event parameter.I want to know the how to get event parameter by first mode

  • Accepted Answer

    In your changeDate function you would use

    e.source.something

    For something use any of the items listed here for tableview. be sure to use exact camel case name if it's two word property:

    http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.UI.TableView-property-data
