Require custom tss files

Hi,
I would like to break my app.tss file to many smaller files like, icons.tss, fonts.tss, etc'.
My questions is how can I require custom CSS.
JS is not runnable on tss file and I'm not sure I can require a tss file from my index.js / alloy.js files.

I'm using alloy so it should support the alloy xml class mapping.

Thanks in advance.

    Alloy doesn't support this directly. You could log a ticket for a feature request with the Titanium Community Jira project. Be sure to link this ticket, include all the details, and label it with "alloy".

    That said, you could probably implement this yourself using an alloy.jmk file to preprocess your specific TSS files that you want to make up the final app.tss in the pre:compile phase. Check out the docs for alloy.jmk files here: http://docs.appcelerator.com/titanium/latest/#!/guide/Build_ConfigurationFile(alloy.jmk))
