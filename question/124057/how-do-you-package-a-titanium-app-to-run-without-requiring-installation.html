How do you package a titanium app to run without requiring installation?

I'm wondering if it's possible to package a titanium desktop application to run directly from an exe without requiring installation first.

For example, assuming we're distributing on Windows, how would I package the application so that it just runs from an exe file without requiring an install process (usually the package is an installable msi). I realize that the exe size would be large, since it would have to include the platform runtime, but for the type of application I'm trying to develop requiring installation might be deal breaker.
