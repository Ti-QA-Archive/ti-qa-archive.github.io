How to run through my array?

So how can I run through my array and get "name" and "age" ?

var array = []

array.push( { name: "Douglas", age: "20" } )

Ti.App.Properties.setList( "favoritosProperties", [array] )


var buscaArray = Ti.App.Properties.getList( "favoritosProperties" )

Ti.API.info( buscaArray )

my Ti.API.info( buscaArray ) returns this:

(
        (
                {
            age = 20;
            name = Douglas;
        }
    )
)

so how can I walk to "name" and "age" fields?

Thanks

    Because it is returning an array, you need to treat it that way. First you wouldn't initially set the property in square brackets like that, its already an array, so doing it that way makes it an array within an array. Second, you just need to reference the array element you want. In this case it would be at position 0. The below code should help you.

    var array = []
array.push( { name: "Douglas", age: "20" } )
Ti.App.Properties.setList( "favoritosProperties", array )

var buscaArray = Ti.App.Properties.getList( "favoritosProperties" );

Ti.API.info( buscaArray[0].name)    
Ti.API.info( buscaArray[0].age)

    and will output correctly

    [INFO] Douglas
[INFO] 20
