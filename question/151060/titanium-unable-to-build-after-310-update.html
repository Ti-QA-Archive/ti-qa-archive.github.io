    To reiterate what Chris Barber said: you need to be running node 0.8.x. 0.10.x only works for the SDK that was released yesterday. Anything older than that requires 0.8.x.

    If you don't want to uninstall your newer version of node.js, you can use the n npm page to manage your node.js versions.

    • @Allan, as I stated in my answer, there is a bug in the "longjohn" module where it doesn't work properly in Node.js 0.11. The Titanium CLI says to use any version of longjohn and we should have locked in the version to prevent this.

  • Titanium is designed and tested for Node.js v0.8.x, though the just released Titanium CLI v3.1.0 supports Node.js v0.10.x.

    You're using Node.js v0.11.0 which is unstable and we haven't tested it yet otherwise we would have found this: https://github.com/mattinsler/longjohn/issues/12.

    I highly recommend downgrading to v0.10.x: http://nodejs.org/download/.
