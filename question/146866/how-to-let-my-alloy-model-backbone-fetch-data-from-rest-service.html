How-to let my Alloy Model (Backbone) fetch data from REST service

Hi, I am currently working on a Titanium Alloy project. I want to let the app fetch data from a RESTful webservice.

I created a Model and a Collection:

var Friend = Backbone.Model.extend
({
    urlRoot: "http://url-to-rest.com/service"
});

var FriendCollection = Backbone.Collection.extend
({
    model: Friend,
    url: "http://url-to-rest.com/service"
});

Now I want to get data from: http://url-to-rest.com/service/all – But how?

1) I tried:

var friendCollection = new FriendCollection();
friendCollection.fetch(); // which ends up in the debugger telling me something went wrong...
friendCollection.all(); // which also ends up in the debugger ...

2) Backbone uses routes which I could use to map fetch to the all method at the REST webservice, but I can't find any info about that using routes in my Titanium project.

Who is willing to help me out?
