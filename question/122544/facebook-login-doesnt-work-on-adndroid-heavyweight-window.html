Facebook login doesn't work on Adndroid heavyweight window

When using heavyweight window on android - the facebbok login window cannot be shown!

var win = Ti.UI.createWindow({navBarHidden: true});
win.open();

Ti.Facebook.appid = '113869198637480';
Ti.Facebook.permissions = ['publish_stream'];
win.add(Ti.Facebook.createLoginButton());

Clicking the "Connect" button won't result in a facebook login window.

Ommiting the navBarHidden: true on the first line will solve the problem.

The point is, I do need the window to be heavyweight.

Any ideas?
