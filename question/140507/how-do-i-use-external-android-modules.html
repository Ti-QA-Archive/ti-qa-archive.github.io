  • Hi Joshua

    Provided the code you refer to is in the titanium 'module' format then, navigate to the resources folder your app.js folder is in. Go up one folder and create a folder called modules at this level. Look for the folder calledandroid` that will be in the archive and paste into the folder you just created.

    In studio, open up tiapp.xml and on the right you should see the modules interface. Press the plus button, provided the module was valid and placed correctly it will be listed - select it.

    Save the changes and close the file. Run project > clean. And you should have access to the module at code level. You will need to use the require keyword as directed in the modules example app.js.
