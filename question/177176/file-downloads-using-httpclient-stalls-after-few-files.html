The size of the folder can go from 2Mb to 100 Mb in some cases. We are firing an http GET request for each file in the folder (we already have a list of file names). The requests are fired in parallel and are all asynchronous requests. The files start downloading successfully initially. After some number of requests (typically after request ~2000), I see this error in the logs ''-[NSThread start]: Thread creation failed with error 35". No file downloads after I get this error. Does anyone know what causes this error and how to resolve this?
