  • What he said about memory leaks is all true. Titanium continues to have serious issues with memory leaks. Unfortunately, there is no easy solution for these, and it doesn't look like they are getting fixed any time soon. Any substantially large application in Titanium is at risk of becoming unusable through these leaks. Overall, it all comes down to what you want your app to do. If it is small enough, then Titanium may be a good solution for you.

    Keep in mind that an app developed in the device's native code will almost always perform better. The only real advantage to using Titanium is saving time. But even then you may have to backtrack and redo the app in native code if it becomes too complex.

    Weigh your options carefully…
