Add more rowData to TableView

I want a TableView that shows TableView A’s rowData as well as TableView B’s data. So basically append or “concatenate” them, I think.

I tried this:

var dataFruits = [
        {title:’Apple’, x:1148, y:1056},
        {title:’Banana’, x:1218, y:1076},
        {title:’Pear’, x:1216, y:961},
    ];

var dataVegs = [
        {title:’Carrot’, x:1148, y:1056},
        {title:’Peas’, x:1218, y:1076},
        {title:’Broccoli’, x:1216, y:961},
    ];

var foodTable = Ti.UI.createTableView({
  data: [dataFruits, dataVegs]
});

The TableView shows up but there are no rows.

I also tried:

var dataFood = [];
dataFood.push(dataFruit, dataVegs);

What is the correct way? I guess I can make a new data array, but that doesn't seem very… smart.
Any help would be appreciated!

Michael Deyzel

iOS, 3.2.2.GA
