can I throw an exception in one module and catch it in other?

I have my modules representing objects. I have one method of an object throwing an exception if a condition is not true(it tries to call a web service, and if here was a problem it fires an exception).
I also have another object(another module) which calls the method in the previous module which was fireing the custom exception. However I get a error saying uncaught exception, with the location being the first module(the one which fires the custom exemption exception). So can I fire a custom exception in one module and cats it in another?

    Assuming you're using CommonJS modules and using require(), the answer is "yes". The following example shows an exception thrown from within a module and caught in the main app.

    MyModule.js:

    function MyModule ()
{
    throw { message: 'foo' };
}

module.exports = MyModule;

    app.js:

    // this sets the background color of the master UIView (when there are no windows/tab groups on it)
Titanium.UI.setBackgroundColor('#000');

var win1 = Titanium.UI.createWindow({  
    title:'Tab 1',
    backgroundColor:'#fff'
});

win1.open  ();

var MyModule = require ('MyModule');

try
{
    var mm = new MyModule ();
}
catch (e)
{
    alert (e.message);
}
