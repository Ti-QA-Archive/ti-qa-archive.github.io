Launch third-party apps from your app

Hi,

I'm currently working on an app thats showcases different third-party apps. From my app I would like to have an "Launch"-button that will open the third-party app. In native development this wouldn't be any hassle, but I can't find any information on how to do this in appcelerator. Does anyone have an idea of how this can be done ?

FYI. I use iOS 5.0, Android 2.3.3 and Titanium Mobile 1.8.2.
This is an cross-platform app, that should work on both tablets and phones.

  • I found the previous answers lacking, for opening a 3rd party app in Android. This is how I did it:

    First get the package name and activity name using adb (http://developer.android.com/tools/help/adb.html).

    Grab the apk of the app you want to start, this can be done by(make sure you have it installed on the connected android device):

        You don't need root to pull the apk files from /data/app. Sure, you might not have permissions to list the contents of that directory, but you can find the file locations of APKs with:

    adb shell pm list packages -f
    Then you can use adb pull:

    adb pull <APK path from previous command>
    and then aapt to get the information you want:

    aapt dump badging <pulledfile.apk>

    Source: http://stackoverflow.com/a/12700063/686884

    This will give you the package name, eg "com.aperations.foo" and the activity class eg "com.aperations.foo.NeatActivity".

    You can then fire an intent to launch the app.

    
    var intent = Ti.Android.createIntent({
            action: 'null',
            data: 'null',
            packageName: 'com.aperations.foo',
            className: 'com.aperations.foo.NeatActivity'
        });

    Ti.Android.currentActivity.startActivity(intent);

    This will then launch the app for you.
