  • I see you're working on windows…

    I was not able to successfully build modules on Windows with 1.8.0.1, so I'm doing it on Mac.

    I'd suggest to try to do it on Mac, if you can, or to drop to 1.7.6 SDK version.

    A couple of things:

    – I assume you've looked at this document:
    https://wiki.appcelerator.org/display/guides/Android+Module+Porting+Guide+for+1.8.0.1

    – make sure you have Android NDK installed ("…To accomodate the new V8 runtime, all 3rd party modules now require the Android NDK for building V8 bindings…")

    Good luck.
