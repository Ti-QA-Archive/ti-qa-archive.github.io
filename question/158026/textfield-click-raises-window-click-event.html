TextField click, raises window click event

Hi!
I'mt trying to hide the keyboard when the user taps outside any textfield.
I do txtField.blur() on the window/view click event.

The problem is that when I click on the TextField, the window/view click event is also raised, so the keyboard never appears (because it immediatelly receives blur method).

How can I prevent the textfields from raising parent events? Thanks!

    You could either set the bubbleParent property to false, or in the eventlistener for the text field click event, set the e.cancelBubble property to true as per documentation.
