  • @Aaron and @Ivan,

    I had been tested using allowsSelection:false and/or touchEnabled: false but without success!

    My TableView now has the configuration below:

    var tableView = Titanium.UI.createTableView({
    data: data,
    backgroundColor:'white',
    rowBackgroundColor:'white',
    top:10,
    height:'auto',
    width:300,
    borderRadius:10,
    separatorStyle: 0,
    bottom: 10,
    allowsSelection: false,
    touchEnabled: false
});
