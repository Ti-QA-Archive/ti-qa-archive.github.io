Problem installing ipad app with sdk 1.8.2

Hi,
Recently I faced some problems in installing my iPad app generated with sdk 1.8.2. It is working in the simulator(both Studio and Xcode) but when i tried to install in the iPad it just hung in the splash screen itself. When changed to sdk 1.8.0 it got installed. Is there any known isuue with the sdk version?

    Hi Maheish,

    I also faced this problem few days ago, and I conclude that there is an image problem in iphone folder of resource folder, may be it conflict some property of pervious version or something else but I resolve it.

    Just create new empty folder and copy paste your project windows file in to new project, remember that do not paste iphone folder in to new project. replace only splash screen image and try to run.

    If you are using SVN then separate that copy for updates only, for development you should use newer one and remove hidden folder of svn form developing copy of the project.
