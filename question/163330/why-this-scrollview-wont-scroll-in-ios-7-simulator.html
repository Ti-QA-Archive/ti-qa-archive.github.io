Why this scrollview won't scroll in IOS 7 (simulator)

Hello,

I've started building a new app, first time with the last ti SDK (3.2.1.GA) and IOS 7 SDK.
I'm building with Alloy 1.3.1.

I've done this simple screen that contains a window and a scrollview with various squared views inside. Enough for the scrollview to allow scrolling. In my Android phone it works perfect but in the IOS 7 simulator, no scrolling! (Code below)

I didn't have this problem before (Haven't made any IOS 7 app yet), can someone tell me my mistake or any solution?

Thank you.

index.js

$.getView().open()

index.xml

<Alloy>
    <Window id="WIN" class="container">
        <ScrollView id="containersSCROLL">
            <View class="box" />
            <View class="box" />
            <View class="box" />
            <View class="box" />
            <View class="box" />
        </ScrollView>
    </Window>
</Alloy>

index.tss

".container": {
    backgroundColor:"white",
    layout: "horizontal"
},

"#containersSCROLL": {
    layout: "vertical",
    height: Ti.UI.FILL,
    width: Ti.UI.FILL,
    backgroundColor: "darkgray"
},

".box": {
    width: Ti.UI.FILL,
    height: "200dp",
    backgroundColor: "blue",
    borderColor: "black",
    borderRadius: 5,
    borderWidth: 5
}

    Hi

    Try this tweak;

    "#containersSCROLL": {
    contentHeight: Ti.UI.SIZE,  // new
    contentWidth: Ti.UI.FILL, // new
    layout: "vertical",
    height: Ti.UI.FILL,
    width: Ti.UI.FILL,
    backgroundColor: "darkgray"
},

    Any difference?

    I would also suggest removing the following entry as unless you have a very good reason for it - dump it.

    ".container": {
    backgroundColor:"white",
    layout: "horizontal"  // remove this
},

    • You thought you had stumped me didn't you? Nope I have the answer - and I must say I thought the problem this fixes would have been fixed by now.

      You need to wrap the children of the scrollView you currently have inside the scrollView inside another view. Here is a quick example;

      <Alloy>
    <Window id="WIN" class="container">
        <ScrollView id="containersSCROLL">
            <View id="outer">                
                <View class="box red" />
                <View class="box green" />
                <View class="box blue" />
                <View class="box orange" />
                <View class="box yellow" />
            </View>
        </ScrollView>
    </Window>
</Alloy>

      ".container": {
    backgroundColor: "#fff"
},

"#containersSCROLL": {
    backgroundColor: "darkgray",
    contentHeight: Ti.UI.SIZE,
    contentWidth: Ti.UI.FILL,
    height: Ti.UI.FILL,
    width: Ti.UI.FILL
},
"#outer": {
    height: Ti.UI.SIZE,
    layout: "vertical",
    width: Ti.UI.FILL
},

".box": {
    backgroundColor: "blue",
    borderColor: "black",
    borderRadius: 5,
    borderWidth: 5,
    bottom: 10,
    height: 200,
    left: 20,
    right: 20,
    top: 10,
    width: Ti.UI.FILL
},
".red": { backgroundColor: "red" },
".green": { backgroundColor: "green" },
".blue": { backgroundColor: "blue" },
".orange": { backgroundColor: "orange" },
".yellow": { backgroundColor: "yellow" }

      Tested and working.
