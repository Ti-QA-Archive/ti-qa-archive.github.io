  • You can perform a 2D matrix rotation of the view setting the anchor point. The trick is that you set that anchor point differently on iOS and Android.

    var view = Ti.UI.createView();
var t = Ti.UI.create2DMatrix();
t = t.rotate(360);

var a = Titanium.UI.createAnimation({
   transform: t,
   duration: 300
});
// set the anchorPoint on the animation object for Android
a.anchorPoint = {x:'50%', y:'50%'};
// set the anchorPoint on the view for iOS
view.anchorPoint = {x:'50%', y:'50%'};

view.animate(a);

    • It would help if I read the documentation! For anchor point, you specify a fraction where 0,0 is the upper left corner, 0.5, 0.5 is the middle, and 1,1 is the bottom right. Also, on iOS, the default anchorPoint is the middle apparently. This worked for me:

      var win = Ti.UI.createWindow();

var view = Ti.UI.createView({
    backgroundImage : 'star.png',
    backgroundColor: 'white',
    height: 100, width: 100
});
win.add(view);
var t = Ti.UI.create2DMatrix();
t = t.rotate(180);

var a = Titanium.UI.createAnimation();
a.transform = t;
a.duration = 3000;
a.autoreverse = true;
a.repeat = 3;
view.anchorPoint = {x:.25, y:.25};

view.animate(a);
win.open();
