Tutorial on migrating your tiapp.xml file to the newer 2.1 format, and setting custom Info.plist values

I noticed that the 2.1.0 changelog mentioned the ability to set custom Info.plist values in the tiapp.xml file. As part of the changeover a number of tiapp.xml settings are now deprecated.

If you want a working example of how to migrate your file to the newer format, and how to set custom key/value pairs to be included in the Info.plist file when the project is built, I have written up a quick tutorial…

http://matthew.delmarters.com/weblog/appcelerator-titanium-upgrading-tiapp-xml-to-the-newer-format-and-define-custom-info-plist-values/
