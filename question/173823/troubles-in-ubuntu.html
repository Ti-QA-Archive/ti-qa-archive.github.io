  • same issue with Oracle Java 7 64 and openjdk 7 64 bit on a quad core atom n3520.
    The package downloaded may have been for ARM. Is that the issue?
    Problem is solved by adding this:
    -Dtitanium.hideDashboard=true
    to TitaniumStudio.ini
    Once the dashboard is clicked, it crashes again.
    It also seems to fail to recognise nodejs while it is installed.
    Works fine on another machine with similar setup
