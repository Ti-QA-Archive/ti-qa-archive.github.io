current_section = Ti.UI.createTableViewSection({headerTitle:&#39;OLD TITLE&#39;});

current_section.setHeaderTitle(&#39;NEW TITLE&#39;);

… i get a runtime error. If I try to set it via: current_section.headerTitle = 'NEW TITLE'; nothing happens, the 'OLD TITLE' is the one showing.

    //Section must be created anew every time its data changes
        self.browserTable.searchSection = Ti.UI.createTableViewSection({

        //Get all products matching the user string
        var rows = self.getSearchProductRows(withAFewArguments);

        //Add search results to section

        //If results exist, show those
        if (rows.length > 0) self.browserTable.searchSection.setHeaderTitle(rows.length + " results found");
