  • Hi, mi original question wasn't correctly focused. The client accessed to server but the communication channel was broken by some reason that I couldn't figure out (I was something related with the Application server, WAS, and the authentication using JAAS).

    Anyway, I managed to fix the problem adding the header "Connection": "Keep-alive" to the XHR object. Maybe this can help you.

    Add next code before you call send() method:

    xhr.setRequestHeader('Connection', 'Keep-Alive');
