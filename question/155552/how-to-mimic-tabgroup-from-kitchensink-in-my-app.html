  • I've realized that the reason mine looks different is because I had to apply the Holo theme to it. I added this to the tiapp.xml file. A minimum API version of 11 is required for the Holo theme to work but I want to target older devices too using minSdkVersion. Also, if you're using a TabGroup, don't set navHidden="true" or the tabs will have the default style, not Holo.

    <android xmlns:android="http://schemas.android.com/apk/res/android">
        <tool-api-level>14</tool-api-level>
        <manifest>
            <uses-sdk android:minSdkVersion="10" android:targetSdkVersion="14" android:maxSdkVersion="18" />
            <!--<supports-screens android:anyDensity="false"/>-->
            <application android:theme="@android:style/Theme.Holo"></application>
            <activity android:name=".BabyTracker" android:theme="@android:style/Theme.Holo">
                <intent-filter>
                    <action android:name="android.intent.action.MAIN"/>
                    <category android:name="android.intent.category.LAUNCHER"/>
                </intent-filter>
         </activity>
        </manifest>
    </android>
