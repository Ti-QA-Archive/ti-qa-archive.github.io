Opening popup window and blurring/graying out parent window

Is it possible to open a window in an iPad app that's a set width and height (smaller than the parent window) and have it be the focus while the main window behind is grayed/blurred out? I'm looking for an effect that's similar to what's commonly found in websites with modal/popup DIVs.

    I've never tried this, but if I were, I would do something like this:

    Fully cover your main window using a new window with a black background and drop the opacity of that window to your liking. Add a view to that window without the opacity containing the 'popup' content that you are trying to display. Register some event or button to remove the window when complete.
