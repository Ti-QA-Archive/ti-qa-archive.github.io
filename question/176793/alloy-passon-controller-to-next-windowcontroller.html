Alloy Passon Controller to Next Window/Controller

I need to pass on this (current/parent controller) to the next screen (child) I want to open. I need to call some method of parent controller from child controller.

On Parent controller, I am calling it like as below

Alloy.createController('child', {
        parent: this,
});

function abc(){
}
exports.abc = abc;

On Clild, I call it as 

var args = arguments[0] || {};
args.parent.abc();

But this does not work. I get args.parent is undefined.

    Hi Virendra,

    try parent: $ inside your Parent controller.

    Alloy.createController('child', {
        parent: $
});

function abc(){
}
exports.abc = abc;

    I found this tip in an older answer from Tony Lukasavage. Accessing Parent Object in Alloy Applications

  • Put your export into a library (in the libs folder) and then access it via require

    function abc(){
}

    In your Child

    var args = arguments[0] || {};

var util=require('util');
util.abc();
