  • Accepted Answer

    Here ya go :)

    var splash = Ti.UI.createImageView({
    image:'logo.png',
    height:480,
    width:320
});

tabGroup.addEventListener('blur', function() {
    tabGroup.add(splash);

    setTimeout(function() {
        tabGroup.remove(splash);
    }, 500);
});

    Though I must agree with Daniel that it is better to not do something like this if you can.
