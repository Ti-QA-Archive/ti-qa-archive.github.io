Cannot start fastdev on SDK 3.0

I tried to start fastdev using the following command:

titanium fastdev start

I get:

Titanium Command-Line Interface, CLI version 3.0.18, Titanium SDK version 3.0.0.v20121105141612
Copyright (c) 2012, Appcelerator, Inc. All Rights Reserved.

Please report bugs to http://jira.appcelerator.org/

[ERROR] "fastdev" is an unrecognized command.

I tried to make a different project using Titanium SDK 2.1.4 GA, but I still get the same answer and the SDK 3.0 is still used.

How do I start fastdev on SDK 3.0 ?

  • When you install the CLI, the titanium command becomes aliased to the CLI instead of the titanium.py file that it used to be aliased to. So you need to execute the full python script in order to start the fastdev server. Try this:

    python /Library/Application Support/Titanium/mobilesdk/osx/2.1.4.GA/titanium.py fastdev start
