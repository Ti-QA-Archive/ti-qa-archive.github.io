    Well, as the documentation for HTML5 Builder clearly states, you application will still be a web app running in a PhoneGap container. This will allow you to use some features like Camera, Geoloc, etc. But all your UI will still be displayed using HTML. Which won't give you the same performance as a native application.

    Now I understand that the prospect of using a graphical IDE sounds appealing. You can have the same flexibility using VisualUI which is allows you to build your user interface without code. (it even supports the new Alloy MVC Framework if you want to use it)

    Have fun!
