Alloy + External Module

I'm using a third party module in my Alloy project and am running into an issue - the events from the module seem to be firing inconsistently, or sometimes not at all.

  • If the module is called in the index controller, the event will fire three or four times, then quit.
  • If the module is called from any other controller, no event will ever fire.
  • When the module is called from any controller, it's internal events always appear to be working (verified by log messages from the module).
  • If I implement the module into a project which does not use Alloy, it works properly.

The implementation of the module is simple, and looks like this -

var audiosync = require('com.audiosync');
audiosync.addEventListener('timecode', function(e){
    console.log(e.timecode);
});

I can go into more details of how the module works and what it does if necessary, but I didn't want to confuse things.
