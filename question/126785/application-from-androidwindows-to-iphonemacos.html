  • Accepted Answer

    There is a problem with 'vertical' layout for iphone. If you have specified this you need to specify Height for each controls that you are adding to this view.

    var table = Titanium.UI.createTableView({ color:&#39;#000&#39;, data:details, height:300 });

    OR

    var win1 = Titanium.UI.createWindow({title:&#39;Fugitives&#39;, backgroundColor:&#39;#fff&#39;});

    Replace one of the above code should work for you.
