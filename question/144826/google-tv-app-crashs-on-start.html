Google TV App crashs on start

Hello,

My Developement System

Titanium Studio: 2.1.2. on Mac OS X 10.7.5
Titanium SDK 2.1.4.GA
Goolge APIs Android 4.1
Google TV Firmware 3.2
Google TV Device NSZ-GS7/GX70

I try to code a App that will run on Google TV. I can upload the App to the Device but right after the start the App crashes with
"Restart Required".

A little bit later "Acivity testgtv1 (in application testgtv1) is not responding."

Thats it. Maybe I am using the wrong SDK or miss something else.

This is the code I am using:

var win = Ti.UI.createWindow();

var testbutton = Ti.UI.createButton({
    title: 'Hello'
})

win.open();
win.add(testbutton);

Any suggestions?

    • Because Google TV doesn't support NDK. Even recent update 4.2.2 by Google isn't going to help because device manufacturer used processor and parts that won't allow ndk apps to work.

      Try changing Android runtime to rhino in tiapp.xml..

  • Any suggestions?

    Connect to the device using DDMS and see what the actual error is?

    As for the 'restart required' this is a long standing bug/feature/pain-in-the-a$$/issue/problem that some (engineering) claim 'cant' be fixed, but others (developers) lean towards 'wont' be fixed. Something to do with the way apps are launched from the Play Store on a different intent or something.
