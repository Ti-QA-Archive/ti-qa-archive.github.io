Unable to use 'Require src' with TabGroup.

Hi,

I am developing a mobile application both for Android and iPhone using the latest version of Titanium Alloy.

Titanium SDK version is 3.1.3

My index.xml file is -

<Alloy>
    <TabGroup id="tabGroup">
        <Require src="home" id="tabHome" />
        <Require src="list" id="tabList" />
    </TabGroup>
</Alloy>

But when I run my app I get an error -

[ERROR] :  Invalid <TabGroup> child type: Alloy.Require
[ERROR] :  All <TabGroup> children must be <Tab> or <Menu>
[ERROR] :  Alloy compiler failed

I am not sure why I am getting this error.

It might be a very simple question but being relatively new to Titanium I am stuck for days on this problem.

Any help on how I can get rid of this error will be much appreciated.

Thanks a lot in advance.

    try this one.

    index.xml

    <Alloy>
    <TabGroup id="tabGroup">
    <Tab id="tab1">
            <Require src="home" id="tabHome" />
    </Tab>
    <Tab id="tab2">
            <Require src="list" id="tabList" />
    </Tab>
    </TabGroup>
</Alloy>
