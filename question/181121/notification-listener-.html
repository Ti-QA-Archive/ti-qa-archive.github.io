notification listener .

I have issue in local notification listener when app is open , it revecied twice at same second , if there are method to ignore the second .

Ti.App.iOS.addEventListener('notification', function(e) {
Ti.API.info(e);
});

  • You can temporarily remove the event listener after first call, or add a flag to ignore multiple calls in a short span of time

    
var notificationAlreadyCalled = false;
Ti.App.iOS.addEventListener('notification', function(e) { 
    if(!notificationAlreadyCalled){
        notificationAlreadyCalled = true;
        Ti.API.info(e);
        setTimeout(function(){notificationAlreadyCalled=false}, 3000);
    }
});
