Force to use gps when getting the current location using getCurrentPosition

I'm completely new in developing apps using titanium so forgive me if i'm missing something obvious. I'm trying to get the exact current position(lat and long) of my device. But it seems like the coordinates are being provided by the network instead of via gps. I was under the impression that using getCurrentPosition, it will use gps by default. Anyways, i also changed the Location settings of my device and unchecked the Use wireless network option so that the device would only use gps when getting the coordinates/location data, but it produces an error and cannot get the location. Another problem(actually my real problem) is that the retrieve lat and long doesn't seem to change even though I changed location. Here's a snippet of my code:

Titanium.Geolocation.preferredProvider = Titanium.Geolocation.PROVIDER_GPS;
    Titanium.Geolocation.accuracy = Titanium.Geolocation.ACCURACY_BEST;
    Titanium.Geolocation.distanceFilter = 5;

    Titanium.Geolocation.getCurrentPosition(function(e)
            {
                if (e.error)
                {
                    alert('Cannot get your current location');
                    return;
                }
                var longitude = e.coords.longitude;
                var latitude = e.coords.latitude;
               alert('longitude: '+longitude+', latitude: '+latitude);
            });
    Titanium.Geolocation.addEventListener('location',function(e)
        {
            if (e.error)
                {
                    alert('Cannot get your current location');
                    return;
                }
                var longitude = e.coords.longitude;
                var latitude = e.coords.latitude;
               alert('longitude: '+longitude+', latitude: '+latitude);

        });
