    Try deleting your dev provisioning profile, then create and install a new one. Make sure that xCode is not using the old one. Also, does your project name have a space in it?

  • I get this all the time. I correct it by reloading my provisioning profile in Ti, then click save.
    Sometimes takes a couple tries but always seems to correct itself. Just go back and forth to the Edit Project settings, switch API versions back and forth a couple times, click Save and it may correct itself. Always works for me.

    Good luck.
