SDK version for app store

Hello

I'm using titanium 3.1.3 and SDK 7.0 to build my app.
When I create the package for distribution in app store which version of ios will the app support?

My question is if an iphone/ipad with ios 6.0 will support my app.

Is it possible to limit the minimum ios version?

I want my app to support both 6 and 7 but no less than 6. And I want to use SDK 7.0 to create the distribution package.

Thanks

  • try adding the following in your tiapp.xml

    <ios>
    <min-ios-ver>6.0</min-ios-ver>
</ios>
