[ERROR] : Failed to compile Java source files:

While building, I am getting this generic error message on the console, as usual with no further details.

[ERROR] : Failed to compile Java source files:

Searching around for this error, I find posts from various points over the past 5 years that seem to indicate that I need to install java 1.6 instead of 1.7 32 bit. However, this is a fresh install of Titanium, and Java 1.7 32bit is what Titanium installed itself during the install process. Is this really still a problem, or do I need to look elsewhere for the issue? Is there a log or someplace I can get more information to pursue?

Titanium Studio, build: 3.4.1.201410281727
Titanium SDK 3.4.1.GA
Windows 7
Java 1.7.0_40

  • The problem was that there was a random .ds_store file in one of the module directories, caused by somebody else working on the project with a Mac. Although this doesn't cause problems with Titanium on Mac, it does cause Titanium on Windows to fail to build.
