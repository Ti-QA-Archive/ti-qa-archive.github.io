Open Window to Front

Short Question: Can I make a open window (in the background) come to the front?

I have 3 windows during the initial launch phase where the user can navigate to each window from any of the 3 windows. I've found that if Window A has been opened, and the user navigates to Window B, I cannot simply use window_a.open(); to bring that window to the front.

I know that if I close Window A after Window B was opened, I can easily use window_a.open(); to bring Window A to the front again. But, I'm hoping there is a method to bring the window to the front but for the life of me I cannot find anything in the docs to do it.

Reason: By design, one window has a back button and I do not want to close the previous window opened.

  • Hmmmm….well without seeing any code its a bit hard to help, and i'm not sure if you are doing this on iOS or Android but there are several things you could do to either solve or workaround that.

    One method would be to simply call the open of one while calling the close of the other one. I haven't tried that, but i'm pretty sure it would work.

    //In window A
button.addEventListener('click', function(){
    window_b.open();
    window_a.close();
});

    The other solution would be to just make them views instead. Buttons would simply show and hide views. Obviously with this techniuqe you would have to create an array of some sort to keep track of shown and hidden views, but it would work. 

    //In window A
button.addEventListener('click', function(){
    view_b.show();
    view_a.hide();
});
