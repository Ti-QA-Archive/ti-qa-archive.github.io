  • Accepted Answer

    Want you want to do is override the build-in localization. You can do that by placing your language files in the resources folder and write your own function to read them, which should override the L() function. Also you COULD use XML in the language files, but using JSON will make things a lot easier.
    I've build such a system myself and it works quite well.
