Multiple SearchBars

I'm having a hard time with multiple search bars, Currently i have 2 tabs, both having searchbar. On the first tab, the searchbar is working, but on the second tab, it's not, you can't even fire-up the 'focus' event.

Any ideas?

Thanks!

  • When you add searchbar without height or set height to "auto" then it fills whole window and becomes un-focusable. you have to set height or width to some fixed value height = &#39;100&#39; or height = &#39;100%&#39;.

    another reason is that the searchbar goes behind some Views and become un-focusable so find out if there is any view over the searchbar or not? set background color of all transparent view.
