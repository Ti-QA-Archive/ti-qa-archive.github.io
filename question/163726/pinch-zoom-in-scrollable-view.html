  • From what I understand you cannot pan the image after zooming in a particular direction because the outer scroll view takes the event and scrolls instead of the image being panned.

    I have used the following:

    When a touch start event is received on the image, then set the scrollingEnabled property of the outer scroll to false - this will stop the scroller moved when the pan is in effect.

    When a touch start event is received on the outer scroll, set the scrollingEnabled property of the outer scroll to true.

    However, the user will have to remove the finger and then scroll again which isn't very user friendly, but I couldn't find another way of doing it on Android without building a module which I haven't got time for.
