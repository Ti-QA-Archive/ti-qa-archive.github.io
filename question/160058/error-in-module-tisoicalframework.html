  • Looking at the stack trace you downloaded the wrong package. I'm looking at that '-master' suffix, which is automatically appended by GitHub when you download the source of a repository. If the module you're talking about is Nyvra/TiSocial then you need to download the source and build it yourself, I couldn't find a precompiled package.

    To do this you need to

    1. download the code in a different location
    2. from the command line move (cd) to the unzipped source
    3. run build.py from there
    4. install the resulting MODULENAME-iphone-VERSION.zip as you did for the wrong zip
    5. enjoy.
