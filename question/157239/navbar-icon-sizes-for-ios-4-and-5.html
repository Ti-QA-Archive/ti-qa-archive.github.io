      You can find the KitchenSink project here.

      The navbar icons are here.

      As you can see from the image below (that you can find here ), there's a little padding

      If you want to use retina images, you have to put them the double of the size with the suffix @2x. Eg: if your image is named KS_nav_platform.png you have to put in the same folder an image named KS_nav_platform@2x.png, double the original size.

      Not sure what your devs are doing wrong unless you show me bits of the code or the images that are causing problems.
