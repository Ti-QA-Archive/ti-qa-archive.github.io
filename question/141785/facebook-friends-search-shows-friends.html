Facebook friends search shows friends.

Hello,

I am building an iOS app and I am using ACS. There is a Facebook Friend Search where you can find Facebook friends that is also members of the app. I can use this fine but after I befriend a user he still appears when searching Facebook friends. Is there a way to only include users that the current user has not already befriended?

Also is there a way to know if current user is already friend with a user when searching for user the normal way?

  • You'd have to do this manually. The user_id for friends should be the same. So if you keep a list of user_ids for people you've already friend requested, then you can check that list while you are printing out the list of facebook friends and only show the ones that don't exist in the list.
