Universal App for iphone/ipad

Is it possible to create one binary that contains both the ipad/iphone versions of an app (a.k.a a Universal App) with Titanium Studio? If so is there anywhere I can go for more information?

  • Yes, in the Titanium Studio where you edit the app's properties select both ipad and iphone and the app created will be universal.
    From here you will have to detect the device using Ti.Platform.model and act as needed.
