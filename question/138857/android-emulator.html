  • I find the run/debug UI elements in Ti Studio to be VERY confusing. They're inconsistent between the App Explorer and the main toolbar.

    The button in the main toolbar runs whatever the last debug configuration was that you ran, regardless of what project you have open in the App Explorer, unless you click on the little "down arrow" to the right of the green bug. That will let you choose the configuration to run.

    In the App Explorer, the debug button always drops down a menu, and lets you select the platform – it always launches the current app on the selected platform.

    I suspect that you might be clicking the debug button in the main toolbar, which would do exactly what you suggest. If you go back to the app explorer and click the debug button (the one with the "bug + folder" icon), you should definitely be able to launch the desired app.
