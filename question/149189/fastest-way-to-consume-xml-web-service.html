  • Accepted Answer

    One of the most important things to add to a tableView row to improve performance is to add a className property for each row that is built using the same layout.

    Having said that - 400 items is too many for a tableview to load in one go and expect it to perform well. Break your content down into smaller chunks and load-on-demand
