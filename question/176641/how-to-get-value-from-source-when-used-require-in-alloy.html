How to get value from source when used Require in Alloy?

Hi Guys,
Application type: mobile
Titanium SDK: 3.3.0,
Platform & version: Android 4.2.2
Device: Android emulator
Host Operating System: Windows 8.1
Titanium Studio: 3.3.0

I have declared View in index.xml

<View backgroundColor="white" id="PeriodView">
    <Require src="MainTable" id="MainTable" />
</View>

I am trying to get text of Label from TableView declared in MainTable.xml

<View>
    <TableView>
            <TableViewRow>
                <Label text="Value1" id="Value1"></Label>                   
        </TableViewRow>
    </TableView>
</View>

But how do you get the handle to Label's value in index.js?
I tried following in index.js:

alert($.Value1.text);

alert($.MainTable.children[0].children[0].text);

(I suppose above isnt a good method but both doesnt work)

Any inputs appreciated.

  • return value pre getView() (method is attached to the controller)

    $.getTableRowLabel(){
    return $.Value1.text;
}

    return value post getView() (method is attached to the primary view)

    $.mainTable.getTableRowLabel(){
    return $.Value1.text;
}
