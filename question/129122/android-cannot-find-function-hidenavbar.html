Android Cannot find function hideNavBar

I have an app looking solid in the iPhone simulator and now I'm ready to start down the Android path. The app has a tab group (hidden) so that I can navigate (primarily back) on the iPhone, but I hide that nav bar option on my home screen. On Android, the app doesn't even open because I get this:

TypeError: Cannot find function hideNavBar in object [Ti.UI.Window]

I can't find any indication that this method (or it's "related" window property) is iPhone only, but I think that the nav bar itself is an iPhone construct so in a way, I guess this makes some sense.

My question, I suppose, is simply this: what are others doing to display a navbar on the iPhone and avoiding it on Android? Is a simple platform test the right way to go or is there a more sophisticated approach?
