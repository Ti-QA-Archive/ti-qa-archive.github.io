  • These problems are reported , so no problems should come. Try 1.6 , 100% width is fixed.

    Also , launching applications is possible using Android's Intent. Same as Java. You need the package and class names and create an Intent.

    var intent = Ti.Android.createIntent({
    action: Ti.Android.ACTION_MAIN,
    className: 'com.app.app.Activity',
    packageName:'com.app.app'
});
intent.addCategory(Ti.Android.CATEGORY_LAUNCHER);
Ti.Android.currentActivity.startActivity(intent);
