Is it possible to get the width of the screen of your smartphone?
I tried screen.width but that doesn't work for mobile apps i guess.

I'm using Titanium SDK 1.6.1 and iPhone SDK 4.2

    Tjeu

    Don't worry, it's not a stupid question. :)

    You need to use Titanium.Platform.displayCaps.platformWidth and Titanium.Platform.displayCaps.platformHeight, listed in the apidocs under Titanium.Platform. Also, there is an example in the KitchenSink in the platform.js script.

    Hope this is useful.

    Cheers
