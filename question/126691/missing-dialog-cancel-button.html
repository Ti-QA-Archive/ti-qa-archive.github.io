Missing Dialog "cancel" button?

I am using some code in my iPad app which I already have successfully used in a prior iPhone app. It's to present a dialog when the user clicks an action button asking if they want to email a recipe:

var emailDialogOpts = {
    options : ['Create Email', 'Cancel'],
    cancel : 1,
    title : 'Email This Recipe?'
};
var emailDialog = Titanium.UI.createOptionDialog(emailDialogOpts);

However, I get only this dialog – no cancel button:

How come?

Studio 1.0, SDK 1.7.2, iPad 4.x, os 10.7

  • Hello, this is not an error or defect, is so. Apple set to the iPad, the cancel button is not necessary, since this action is performed by tapping outside the window dialog. So this is a feature of iOS on iPad, not an error of Titanium.

    • This post shows a bug logged, and if you add an "extra" button as the author says, you will get a cancel button.
