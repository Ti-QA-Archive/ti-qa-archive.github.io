KitchenSink does not appear on my Android emulator

I am using a windows 7 professional computer. I have downloaded all the android sdk and am using titanium 1.6.3. I can launch the android emulator but it just sits and flashes android and does not bring up kitchen sink. Also i get the java screen that says can not load preferences.

  • Seth

    The "load preferences" message does not adversely affect Titanium. I get it too, in fact. You can safely ignore it.

    Bear in mind that you must launch the KS in an emulator that uses the enhanced Google API. From the SDK drop-down menu of Titanium Developer's Run Emulator screen, choose one of the items with the "APIs…" prefix.

    Failing this, are you able to create and launch a default project, as explained in Test a default project?

    Cheers
