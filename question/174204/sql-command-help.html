    • The use of ? is used as a place-holder, so you simply place it in-line of the string.

      You can use more than one place-holder.

      // simple example ~ (NOT your code)
var user = 'user1';
var pass = 'pass1';
var sql = "SELECT firstname, lastname FROM users where username = ? and password = ?;";
var data =  db.execute(sql, user1, pass1);

      As you see yu simply add the ? place-holders where you need them and suffix the actual values one after the other.
