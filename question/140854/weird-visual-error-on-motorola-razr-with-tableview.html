  • Hi Felipe

    As there is no code to check this - I am going to guess.

    Chances are the tableView you have has no backgroundColor property or if it does it is set to transparent.

    Look for the following line, if it exists update it to something suitable, if it does not add it with something suitable.

    var tbl = Ti.UI.createTableView({
...
  backgroundColor: '#ff0000' // this line
...
});

    You can set this to any colour you wish. You could also use a backgroundImage instead.

    var tbl = Ti.UI.createTableView({
...
  backgroundImage: 'someimage.png' // this line
...
});

    If you have already done either of these, or this does not work, please provide your tableView code.

    Cheers
