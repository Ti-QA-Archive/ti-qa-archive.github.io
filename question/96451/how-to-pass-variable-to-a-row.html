How to pass variable to a row?

I am still learning, and finding appcelerator to be a pretty good and robust platform.

I have run into one issue so far…
how can you pass a variable to a row like so:

if( f.exists() )
{
var hascheck = 'hasCheck:true';
}

var data = [
"{title:'Row 1'," +hascheck+ ", hasChild:true, color:'red', selectedColor:'#fff',}",
];

var tableview = Titanium.UI.createTableView({
data:data
});

Titanium.UI.currentWindow.add(tableview);

I have tried all kinds of variations of this, and so far none work. Essentially, I want to change a setting in a row (check, color, title, etc) dynamically based on some criteria.

Bill

    var data = [ 
   {title:'Row 1', color: (hasCheck == true ? 'red':'white')},
   {title:'Row 2', color: (hasCheck == true ? 'red':'white')}
 ];
