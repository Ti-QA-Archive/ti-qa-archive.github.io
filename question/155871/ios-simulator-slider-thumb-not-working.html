The label doesn't change either but stays fixed at the value given to it. I don't know why the thumb isn't moving. The KitchenSink App sliders dont work either.

    Your tags indicate that you are attempting to use iOS7, if this is true you need to remember a few thingsl

    • Latest xcode that must be used is in beta
    • iOS7 is in beta and unreleased
    • Ti SDK 3.1.2 GA that must be used is in beta

    Please provide your code snippet just in case it is code and not a beta issue yet to be resolved.

    • This code sample works correctly using iOS 6.1 so try this if is fails it is a bug or you are not using one of the required version combinations mentioned above.

      var slider = Titanium.UI.createSlider({
    top: 50,
    min: 0,
    max: 100,
    width: '100%',
    value: 50
});    
var label = Ti.UI.createLabel({
    text: slider.value,
    width: '100%',
    height: 'auto',
    top: 30,
    left: 0,
    textAlign: Ti.UI.TEXT_ALIGNMENT_CENTER
});
slider.addEventListener('change', function(e) {
    label.text = String.format("%3.1f", e.value);
});
win.add(slider);
win.add(label);
