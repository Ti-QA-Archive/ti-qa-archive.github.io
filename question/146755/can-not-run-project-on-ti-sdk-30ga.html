  • Accepted Answer

    Titanium SDK 3.0 is only supported on Windows 7 or newer. Windows XP is not supported by Titanium SDK 3.0. Also, you should verify that you have Python 2.7 installed.

    Please refer to the Titanium Compatibility Matrix for the complete list of system requirements.
