  • Guys, thank you so much for your answers. Ok, so particles[0].obj.img is a simple ImageView. I've discovered that, while animating positions, at end of the animation coordinates are update to their final location, and properties like autoreverse or repeat are totally useless. So, if you want to play back your animation you have to add a listener for the 'complete' event and, inside the function defined for the event, you just put your image to its original position and than set back the same anim. here you are the code,

    var a = Ti.UI.createAnimation();
a.center = {x:100,y:100};
a.duration = 1000;
a.addEventListener('complete',function(e){
    if (e.source.center.x==100) {
        e.source.center = {x:0,y:0};
    } else {
        e.source.center = {x:100,y:100};        
    }
    myView.animate(e.source);
})

    It's a really naive way but i did not find any other technique to do a repeating anim for moving objets.
