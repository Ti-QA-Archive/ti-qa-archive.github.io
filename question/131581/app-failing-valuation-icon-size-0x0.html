  • I came across this issue and it almost took a day for me to fix it.
    Simple solution:

    1. open your project in Xcode,
    2. import the icons (if not already in the project),
    3. change the File Type to "icon" [select the images(s) and then see the 'Utilities' panel].

    I used Xcode 4.2.
