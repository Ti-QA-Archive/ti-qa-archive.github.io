Android Window opacity

Hi,

I have the following window:

var win = Ti.UI.createWindow({
        modal: true,
        navBarHidden: true,
        opacity: 0.7,
    });

I'm testing my app in the simulator and I've notice opacity only works in Android 2.3.3, while in Android 2.2 and 4.0.3 it isn't working. Any thoughts about this?

Thank you!

  • Can't tell you the difference, why it works, but it usually with opacity, or visibility problems I use animation as fix. I know not the best solution, but usually it works:

    var animation = Titanium.UI.createAnimation();
animation.opacity = 0.7;
animation.duration = 1;
win.animate(animation);
