    • I did it by backing up my builds folder in android sdk, then deleting the whole thing and getting titanium to download the files again. Titanium automatically downloads version 21.1 however the update client will still prompt you to update to 22 in the future and it is ticked by default so beware. Also in order to delete you android sdk folder you will need to kill any processes by adb.exe. Hope this helps
