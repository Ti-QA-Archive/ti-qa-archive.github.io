Best way to stack images, swap them using JS

I'm using Titanium Studio to do cross-platform development. I need to show a single very tall image that's made up of slices of smaller images. The user needs to be able to scroll along the height of the composite image (width is fixed to Ti.UI.SIZE). Also, I need to be able to swap out the content of these image slices on the fly.

####My questions:

  1. What's the best way to structure the XML components to achieve this?

  2. In my index.js, how will I target specific components and swap out the image src.

Thanks!

  • Posting what code you have would help more. Are you doing this only in landscape mode? Can you do this in portrait mode? What size are your images to swap out? When are you swapping images out?

    Make a scrollview and set the height to Ti.UI.SIZE, make a wrapper view (layout:vertical), add an array of images to multiple views (Ti.UI.SIZE, layout:horizontal), add the views to the wrapper view, add the wrapper view to the scrollview, on the scroll event of the scrollview change the images?

    • Heres an example of flipping an image and having some image on the other side… Pretty much you will have to multiple views inside of a horizontal view and then wrapper view and do your flipping based on some sort of action. My guess is from scrolling

        var flipWindow= Ti.UI.createWindow();
var testView = Titanium.UI.createView();
var flipView = Titanium.UI.createView();

var myImageView = Titanium.UI.createImageView({ 
            image: '/appicon.png',
            height:60
        });

flipView.add( myImageView );
testView.add(flipView);

testView.addEventListener("click", function(e) {

testView.animate({

view: flipView,
transition:Ti.UI.iPhone.AnimationStyle.FLIP_FROM_LEFT,
duration:1500

});

myImageView.image = '/Default.png';

});

flipWindow.add(testView);
flipWindow.open();
