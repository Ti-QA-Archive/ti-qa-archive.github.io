XCode is not installed. Can not find SDK

Hi

I try to use Titanium for the iPhone. I installed the latest iPhone SDK 4.2.1 with the XCode version 3.2.5.
But always, when I try to create a project and set the type to mobile, it tells me:
XCode is not installed. It is required for iPhone.

I use the Titanium Developer 1.2.2 version and the mobile SDK 1.5.1.
I already tried to remove it completely, restart my mac and install an old version. (doesn't work either)

If I type "xcodebuild" the terminal tells me: command not found.
If I ype in the whole path:

user$ /Developer/usr/bin/xcodebuild 
xcodebuild: Error: the directory /Users/name does not contain an Xcode project.

Would be great if someone could help me
Thanks

Felix

    It would appear that your system path was not set up correctly when the iOS SDK was installed. Placing the &#x2F;Developer&#x2F;usr&#x2F;bin directory on your system path should solve the issue, but if not, you may need to re-install the iOS SDK and Xcode tools.
