    You need to check whether the device in which you are running your application is a tablet or a mobile manually for time being. You can do this by :

    var deviceWidth = Ti.Platform.displayCaps.platformWidth,
var deviceHeight = Ti.Platform.displayCaps.platformHeight,
var isTablet = (Ti.Platform.osname=== 'android' && (deviceWidth > 899 || deviceHeight > 899)) ? true : false

    And regarding images for android this link might help you :

    https://wiki.jasig.org/display/UPC/Configuring+and+Customizing+Your+Appcelerator+Titanium+Project

    • use if - else statements by checking the boolean varaible isTablet and write the code individually for mobile and tablet based on UI and orientation modes.

      And regarding the code that I pasted above you can use it anywhere in any file and access them within the same file or alternatively you can paste them app.js and then pass them through out the application using :

      var deviceWidth = Ti.Platform.displayCaps.platformWidth,
var deviceHeight = Ti.Platform.displayCaps.platformHeight,
var isTablet = (Ti.Platform.osname=== 'android' && (deviceWidth > 899 || deviceHeight > 899)) ? true : false ;
Ti.App.Properties.setString('IS_TABLET' , isTablet );

      And to receive the value in any file when required use :

      var isTablet = Ti.App.Properties.getString('IS_TABLET');
