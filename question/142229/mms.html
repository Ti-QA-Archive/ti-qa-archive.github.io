  • You can use standard intents

    tiapp.xml add this lines to main activity

    <intent-filter>
    <action android:name="android.intent.action.SEND" />
    <category android:name="android.intent.category.DEFAULT" />
    <data android:mimeType="image/*" />
</intent-filter>

    your share listner can look like this

    if(Ti.Filesystem.isExternalStoragePresent()){
    var imageBlob = Ti.Filesystem.getFile(Ti.Filesystem.externalStorageDirectory+'share.png');
}else{
    var imageBlob = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory+'share.png');
}

intent = Ti.Android.createIntent({
    action: Ti.Android.ACTION_SEND,
    packageName: 'com.android.mms'
})
intent.setType('image/png');
intent.putExtraUri(Titanium.Android.EXTRA_STREAM, imageBlob.getNativePath());
intent.putExtra("sms_body",'Your predefined sms text body ');
Ti.Android.currentActivity.startActivity(intent);

  • Hello Jigar,

    With iOS it's impossible, you can't send sms/mms programmatically and I think that is not possible in android too, you can use this code for send a SMS:

    var intent = Ti.Android.createIntent({
        action: Ti.Android.ACTION_VIEW,
        type: 'vnd.android-dir/mms-sms',
        });
        intent.putExtra('sms_body',sms_content);    
        intent.putExtra("address", "4800"); 
        try {
            Ti.Android.currentActivity.startActivity(intent);
        } catch (ActivityNotFoundException) {
            Ti.UI.createNotification({ message : "Error" }).show();
        }

    The code mentions mms, but I'm not sure if is possible attach media or something.
