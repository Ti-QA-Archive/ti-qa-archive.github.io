  • Mobile Web does not support Windows Phone 7, however we do support Windows Phone 7.5… to a degree.

    As of v2.1.0, we don't support animation and instead snaps to the end position. A new animation system is almost complete and will be available with v2.1.1.

    Mobile Web does not support any touch events on WP7.5 because of limitations of Internet Explorer on WP7.5, so that means NO scrolling of ScrollViews, ScrollableView, MapView pinch-zooming, dragging, touchstart/end, etc.

    We haven't tested WP8.
