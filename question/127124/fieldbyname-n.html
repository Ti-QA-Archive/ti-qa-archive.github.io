  • Ok, I solved with the replace. Before I replaced in the field value '\ n' with '**'

    UPDATE name_table SET nome_field REPLACE (nome_field, '\n', '** ')

    The value in my field msg is 'hello ** Charles' then:

    ...
var msgRep = result.fieldByName('msg');
msgRep = msgRep.replace(/** /gi, '\n');
alert(msgRep);

    I's work!
