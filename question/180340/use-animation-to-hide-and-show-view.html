Use animation to hide and show view

Hello everybody,
I try to use animate to hide and show view.
the problem is that I can't resize the view to his original height.
this is my code:

'$.branchDetailsView.addEventListener('click', function(e) {
    if (branchDetailsOpen == false) {
        $.moreDetails.animate({
            height : "0%",
            duration : 200
        });    
        branchDetailsOpen = true;
    } else {
        $.moreDetails.animate({
            height : $.moreDetails.getSize().height,
            duration : 200
        });        
        branchDetailsOpen = false;
    }
});

I tried to set the height to inline code $.moreDetails.getSize().heigh but it set it to 0, when i use getHeight() it set it to 'size' because i used 'Ti.UI.SIZE'.
someone know what else can i do?
Thank's!

    Hello Yair Cohen,

    I got a tricks, but it works only for iOS. Just set height property ".01" not "0" like this sample code

    var branchDetailsOpen = false;
$.branchDetailsView.addEventListener('click', function(e) {
    if (branchDetailsOpen == false) {
        $.moreDetails.animate({
            height : .01,
            duration : 200
        });
        branchDetailsOpen = true;
        Ti.API.info('1');
    } else {
        $.moreDetails.animate({
            height : $.moreDetails.getHeight(),
            duration : 200
        });
        branchDetailsOpen = false;
        Ti.API.info($.moreDetails.getHeight());
    }
});
$.index.open();
