processing CSV files on the device

Hey guys!

I am currently working on a very small app that reads CSV files and displays the results in a table.

The parsing is done by this function: http://www.bennadel.com/blog/1504-Ask-Ben-Parsing-CSV-Strings-With-Javascript-Exec-Regular-Expression-Command.htm

My first try was to convert and generate the table in one step.
It works in the simulator, and it works on the device but a file with 100 entries is processed very slow and a file with 600 entries crashes the device with a low memory warning.

After that I changed it from processing the file in step 1 and generating the table in step 2.
Step 1 takes ages, step 2 is really fast (105 entries).

The processing of the data is always the bottleneck and I have no idea how to solve it.

Currently I am reading all the data from a CSV file stored on the filesystem.

I know that SQLite can import CSV files (at least on the console) so that might be worth trying.
