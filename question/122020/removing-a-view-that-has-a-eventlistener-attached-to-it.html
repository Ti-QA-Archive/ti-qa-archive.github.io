Removing a view that has a EventListener attached to it

Hi all

Is removing a view that has an event listener attached to it supported? or do I have to remove the event listener before deleting the view itself?

How do i remove the event listeners for views that have been dynamically generated. 


// Clean any previous sub views before adding new ones

for (var d in masterView.children) {
    masterView.remove(masterView.children[d]);
};


// Event handler for each sub view
var mySubViewClick_handler = function(e) {
    ... do something
};


// Create the sub views which each have an event listener.
//      myViewList tells me how many views I have to create
for (var i=0; i<myViewList.length; i++) {

    var mySubView = Ti.UI.createView({
        top: myViewList[i].top
    });

    mySubView.addEventListener('click', mySubViewClick_handler);

    masterView.add(mySubView);
};

Thanks

  • You can remove a view without removing the EventListener, yes. I believe the EventListener is then removed automatically, unless I am mistaken. But I usually never remove EventListeners, and my apps run perfectly fine.
