Is there an example of loding more table rows with Alloy?

I have an Alloy app with a TableView that is bound to a Collection which is being loaded via a custom sync adapter from an http server. There may be very many rows, and I will need to page in the JSON. Is there an example of loading more rows when the table scrolls to the bottom, or keeping a "Load More" row at the bottom of the table with a link to get the next page on touch?

  • Here is part of the answer. This will sense if the user pulls the top down and releases (refresh) or if they pull the bottom up and release (load more):

    var scrollOffset = 0;
var contentHeight = 0;

$.tableView.addEventListener('scroll',function(e) {
    scrollOffset = e.contentOffset.y;
    contentHeight = e.contentSize.height;
});

$.tableView.addEventListener('dragEnd',function(e) {

    if ((scrollOffset + $.tableView.size.height - contentHeight) > 20) {
        alert('load more');
    }
    else if (scrollOffset < -20) {
        alert('refresh');
    }
});
