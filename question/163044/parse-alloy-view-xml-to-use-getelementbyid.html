  • You don't. Not if you're using TableView as you say. I think you're in the HTML/web page mindset here. Titanium is not the same.

    The iterating over table rows section in the guides might help.

    If you need to access a row during an event (e.g. user taps a row), then you'd do something like this:

    myTable.addEventListener('click', function(e) {
   Ti.API.info('you tapped row ' + (e.index+1));
   Ti.API.info('some property of your row = ' + (e.rowData.yourProperty));
});
