  • You have to logout the current user to log in a new one. At some point, you'll need to decide to call Ti.Facebook.logout(); to clear the credentials on the current user.

  • This worked for me…

    if (Titanium.Facebook.loggedIn)
        Titanium.Facebook.logout();

    var url = 'https://login.facebook.com';
    var client = Titanium.Network.createHTTPClient();
    client.clearCookies(url);
