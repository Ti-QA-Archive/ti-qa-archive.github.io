Array to JSON

I have created an array of values that is passed into a method. How do I get that into a JSON.stringify format so I can send it to an API?

Example:

var dataArray = Array();

dataArray['username'] = 'user123';
dataArray['password'] = 'testing';

    hi Levi,

    try this

    var dataArray = {};

dataArray['username'] = 'user123';
dataArray['password'] = 'testing';

    JSON.stringify(dataArray);
