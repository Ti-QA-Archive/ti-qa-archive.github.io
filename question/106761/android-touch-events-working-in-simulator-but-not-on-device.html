Android: Touch events working in simulator, but not on device.

Hi,

I am using Titanium 1.2.2 with SDK Version 1.5.1.
My Android Simulator SDK is 1.6 and on my device I have version 2.1 installed.
So now whats going on:
I need to react fast to a touch event in a WebView. Since 'onclick' triggers only after 500ms, I was using following code to get the event faster:

var element = document.createElement(...);
var tempTouchPosition = new Position(0, 0);
    element.ontouchstart = function(e)    {
            tempTouchPosition.x = e.changedTouches[0].pageX;
            tempTouchPosition.y = e.changedTouches[0].pageY;
        };
    element.ontouchmove = function(e)    {
            tempTouchPosition = new Position(-1, -1);
        };
    element.ontouchend = function(e)    {
        if (tempTouchPosition.x == e.changedTouches[0].pageX && tempTouchPosition.y == e.changedTouches[0].pageY)    {
                // do stuff
            }
        };

This works just fine in simulator, but not on the device. Using 'onclick' it was working.
It also works on iPhone Simulator and Device…

Any help is much appreciated :)
