Mapview with Height parameter doesn't zoom on region.

If I set the height parameter in the mapview, it stays zoomed out. Can't figure out why? Is anyone else having the same issue? I am putting the map as a header of a table.

Here is the code

var region = {latitude:10, longitude:-10, latitudeDelta:0.05, longitudeDelta:0.05}
    var mapview = Titanium.Map.createView({
        mapType: Titanium.Map.STANDARD_TYPE,
        height: 200,
        region:region,
        animate:true,
        regionFit:true,
        userLocation:true,
        annotations:[pin],
        width: '100%',
        top: 0
    });
