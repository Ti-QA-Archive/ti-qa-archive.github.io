How to implement a "Request new feature here" functionality in your app?

Before downloading an app in iTunes, I usually read some users reviews.
Quite often, 1-2 stars reviews are not because the user didn't like the app, but because they want some new feature to be implemented.
If, inside the app, the user were able to request a new feature, I think the number of 1-2 stars reviews would be reduced.

Therefore, I would like to ask if you provide a "Request new feature here" kind of functionality inside your app? And how do you do it?
