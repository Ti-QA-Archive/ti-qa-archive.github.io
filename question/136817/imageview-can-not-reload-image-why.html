    • its simple titanium uses cache image so that you can not see other image with the same name

      so you can do one more thing…

      try to download image with imageurl using httpClient and set that image on the success like

      var xhr = Ti.Network.createHttpClient({
onerror : function(e) {

            alert('Error | No internet connection detected.');
            Ti.App.fireEvent('hide_indicator');
        },
        onload : function(e) {

            if (this.status == 200) {

            imageView.image = this.responseData;

            }
        }
});

xhr.setTimeout(120000);

xhr.open('GET',<image url>);
xhr.send();
