Image Size and Image Scaling

Hi guys,
I have an finished android project(I will call it project 1) and I am using the ui layout defined in tweetanium. So I copied and pasted the code(from finished project) to the new one(with the ui layout addressed above and I will call it project 2). But somehow,the images all got scaled.(My android device is 480 by 800). For example,

var view = Ti.UI.createView({
    left:0,
    right:0,
    height:29
});

This view looks like its width is "fill_parent" in both projects but its height is bigger than that of project 2.
Does anyone know what is going on?

    Are the projects using different versions of the Titanium mobile SDK? I'm guessing the old project is using a version prior to 1.7 and the new project is using 1.7 or later.

    Version 1.7 of the SDK is using new defaults that you should be aware of that may have caused this behavior. Check out the blog post on that.
