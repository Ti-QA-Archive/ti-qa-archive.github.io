If the right button is not available, can anyone suggest another way to achieve the same function? Can I use a TabGroup inside a NavigationWindow?

    • Sorry I should have been clearer, when I said set the button on your window - I did not been place the button inside the window, but rather use the properties to set it as the nav button.

      As I do not know if you are using Alloy or Classic I am going to guess, if I am wrong let me know I will show you the other way.

      // buttons
var btnLeft = Ti.UI.createButton({ title: 'left' });
var btnRight = Ti.UI.createButton({ title: 'right' });
btnLeft.addEventListener('click', function (e) { alert('left'); });
btnRight.addEventListener('click', function (e) { alert('right'); });
// main window
var win = Ti.UI.createWindow({
    backgroundColor: 'red',
    leftNavButton: btnLeft, // one way to set a nav button
    title: 'Red Window'
});
win.setRightNavButton(btnRight); // other way to set the nav button
// nav window
var nav = Ti.UI.iOS.createNavigationWindow({
    window: win
});
nav.open();
