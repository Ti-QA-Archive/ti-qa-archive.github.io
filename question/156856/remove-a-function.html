Remove a function

Hi.
I have a function that uses singletap to do stuff.

The view is both a pic and some text but sometimes it wont be a pic so in that case i dont want the text to be able to singletap to change props.
Make sense..?

Anyhow here is my code

scview.addEventListener('singletap', function() {
                if(descriptionview.height === '100dp') {
                    descriptionview.applyProperties({height: '0dp'});     
                } else{
                    descriptionview.applyProperties({height: '100dp'});    
                 }
                });

if(win.object === null){
ta1.applyProperties({height : '300dp'});  
}

So when win.object is null i dont want the singletap to work.

How do i set that?

  • Hello,

    if the scview is not being added to the current window, or not in the screen, it won't work. On the other hand, you can always remove the events. You need to do something like:

    function f () {
return;
}
scview.addEventListener('singletap',f);

    so you can do something like:

    scview.removeEventListener('singletap',f);

    Best,

    Mauro
