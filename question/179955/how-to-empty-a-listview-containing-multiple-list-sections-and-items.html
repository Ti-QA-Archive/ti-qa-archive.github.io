How to empty a ListView containing multiple list sections and items?

I dynamically add list sections and populate each of them with items. I have a button that refreshes the list. How can I first empty all the content of ListView before re-populate it with list sections and items?

    Below is the sample code for your reference,

    
var listView = Ti.UI.createListView();

/* Empty each section */
for(var i=0, arr=listView.getSections(); i < arr.length; i++) {
    arr[i].setItems([]);

}

/* Addeding new section to list view */
var fishSection = Ti.UI.createListSection({ headerTitle: 'Fish'});
var fishDataSet = [
    {properties: { title: 'Cod'}},
    {properties: { title: 'Haddock'}},
];
fishSection.setItems(fishDataSet);
listView.appendSection(fishSection);
