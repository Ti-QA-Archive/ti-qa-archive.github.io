    Have you tried without adding the require statement. var Alloy = require(&#39;alloy&#39;). It's not necessary to add it explicitly.

    I hope you added $.singup.open() in your signup page.

    Also, the following statement is wrong var view_signup = Alloy.Controller.getView();.
    It is - var view_signup = Alloy.createController.getView(); or var view_signup = Alloy.createController.getView().open();

    Here is the Alloy documentation for reference - http://docs.appcelerator.com/titanium/latest/#!/guide/Alloy_Framework
