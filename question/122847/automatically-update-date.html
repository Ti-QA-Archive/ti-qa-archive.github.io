Automatically update Date

Hi,
I have an application where I need to update the date while the application may be running in the background.

Currently, the date update properly if I kill the application and re-launch it.
But if the user has the app running in the background, and if the date changes, I would like to update the date in the app.

What would be the best possible way to do it?

Do I need to constantly keep checking the date? Will that use up a lot of memory?
I am no tsure how it should be done.

Thanks and Regards

  • I'd work with an interval. It triggers a function every x seconds.

    For example:

    var interval = setInterval(function(){
    //Do something with the date
}, 1000);

    In this example the application will do something with the date every second.

    if you would like to stop the interval you just do clearInterval(interval);

    Hope this helps
