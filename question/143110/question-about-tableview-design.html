Question about TableView design

Hi guys.

First of all, sorry if the question is unclear because I can't make it short.

I have a TableView with few sections categorized by month. Each month has data inside it and I want to know if it's possible to show / hide the data by pressing on the section headers.

So let's say, I create a TableView with 2 sections with headers (January and February) and I want to show only January and hide February data. So I can press on the February header and hide the row of February section. Then, when I press the February header again, I want the section to expand and show the data back.

I only know that TableView has hasChild property but correct me if I'm wrong, in iOS it will open a new TableView. I don't even know how hasChild works in Android because I couldn't find a working sample for it.

I tried the latest KitchenSink but it doesn't have what I wanted. Is this possible? If anyone has a tip on how to make this happen, I would love to hear it. Thanks in advance =)

  • Check out the Verbs app for iPhone. T has something similar. Maybe you can use it and try to understand the logic behind it.

    My guess, add a single tap/click event listener to the section headers. Add a custom property index to every section. The first section will be 0, the second will be 1 and so on.

    When you detect a click, grab the index. And then, you can do something like this:

    tableView.data[sectionIndex].rows =null;

    Always make sure you have this same data stored else where during the app's session.

    This isn't tested, but my guess is that this would work. Worth a try.

    — commented October 5th 2012 by Nikhil Nigade
