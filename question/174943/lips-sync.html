Lips Sync

I want to make an application that moves the lips of a character according to a recorded sound.
If the sound volume increase the lips should be fully opened, if it is low it should be closed.

Something like this application:-
https://play.google.com/store/apps/details?id=com.digeebird.funnymouth

So is it possible with appcelerator or not, and how ?

    Hi

    You use this event;

    • Titanium .Media event volume

    Then depending on the level of animation you need use the animate method on any of the views/images you have - or simply change the top and left properties for quick changes.

    Before you ask this is an obscure in frequent request - it is unlikely anyone will have a sample set of code - however those two things will have you up and running in 10 minutes.
