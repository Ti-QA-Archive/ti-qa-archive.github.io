CDATA html parsing error

Hello Developers,

I have a question. Im having a problem with my feed, because of html styling.
Normal html works great in the xml parser, e.g:

<p>title</p><h1>this is a test</h1>

But styling does not, and returns an error:

<p style:"color:red; font-size:20px;">title</p>

The parser cannot accept the ; sign. If I do a replace of this sign, the functionality of the styling will not work.

How do you guys solve this in your apps?

  • I found the answer. Just wanna share with the community.

    Both insert statements works with normal strings, but if html tags are included, use the second one. 

    //dont work with html
db.execute('INSERT INTO articles ("title", "html") VALUES ("'+title+'", "'+html+'")');

// WORKS
db.execute('INSERT INTO articles (title, html) VALUES(?,?)', title, html);
