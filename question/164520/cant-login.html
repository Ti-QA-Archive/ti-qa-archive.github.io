  • OK I've got it working now, deleted all my Java installs and reinstalled the latest version that I found at https://developer.apple.com/downloads/. Some of the other posts I saw said to download an older version of java but that didn't work for me. Or it might have been that I had 2 java versions installed, I saw another post that said if there is more than one that will cause problems. Seems like a lot of people have had problems after updating to Mavericks. Hope this helps someone, good luck!
