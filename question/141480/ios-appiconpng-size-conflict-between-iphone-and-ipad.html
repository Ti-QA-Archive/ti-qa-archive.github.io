iOS appicon.png size conflict between iPhone and iPad?

I am trying distribute my app to itune.
I initially have appicon.png ( in iphone directorr) size 57x57 pixels. When I validate it, it complains the size must be 72x72 pixels.

So I changed the size to 72x72 pixels. Then the validation complains the following:

iPhone/iPod Touch: appicon.png: icon dimensions (72x72) don't meet the size requirements. The icon file must be 57x57 pixels, in .png format

There is only appicon.png file in Resource/iPhone directory. How do I make it work with the two different icon size?
Thanks
