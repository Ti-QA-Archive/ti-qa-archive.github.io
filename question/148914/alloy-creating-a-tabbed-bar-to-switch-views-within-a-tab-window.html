Alloy Creating a Tabbed Bar to switch views within a Tab Window

Hi. I am making my very first iOS and Android app. using Titanium Alloy and I need you guys' help!

I want to make an app with 4 tabs and each "tab view/window" should have two or three tabs that you can switch back and forth.

index.xml

<Alloy>
    <TabGroup>

        <!-- Tab included via <Require> tag -->
        <Require src="one"/>        
        <Require src="two"/>        
        <Require src="three"/>        
        <Require src="four"/>

    </TabGroup>
</Alloy>

one.xml

<Alloy>
    <Tab id="one" title="ONE" icon="KS_nav_ui.png">
        <Window id="win1" title="ONE" backgroundColor="white">
            <View id="header">
                <TabbedBar>
                        <Labels>
                            <Label>TAB1</Label>
                            <Label>TAB2</Label>
                            <Label>TAB3</Label>
                        </Labels>
                </TabbedBar>
            </View>
            <!--<TableView id="myTableView">
                    <TableViewSection id="section1" headerTitle="Date???">
                        <TableViewRow title="Some Text"></TableViewRow>
                    </TableViewSection>
            </TableView>-->            
        </Window>
    </Tab>    
</Alloy>

If I run this code for iOS, I get an app with 4 tabs at the bottom and the first tab will have 3 tabbed bar buttons you can click on. Which is fine and I want this.

Now, I wanted to add table view so I can have some list of events. So I added my code that is commented out. If I run this code, with the comments removed, my first tab view changes to the table view and my 3 tabbed bar disappears.

How can I have both of them showing up on my app? Also, I read in the document that <TabbedBar> is only for iOS. What is the way around this for Androids?

Thanks, any help would be appreciated.
