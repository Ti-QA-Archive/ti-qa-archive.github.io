    Hi

    This is an example sql statement that will limit your results;

    select * from tablename LIMIT 20 OFFSET 20;

    This will retrieve 20 records starting from the the offset of the 20th record. The following example will get the 20 records from the start.

    select * from tablename LIMIT 20 OFFSET 0;

    This part is the important bit - add it to the end of the select statement you have right now.

    LIMIT 20 OFFSET 0

    • Here is a helper function to make it MUCH easier to use this technique.

      function asLimit(size, page) {
    return 'LIMIT ' + (size) + ' OFFSET ' + (size * (page - 1));
}

      You would use it like this;

      asLimit(20, 1); // page 1  = LIMIT 20 OFFSET 0
asLimit(20, 2); // page 2  = LIMIT 20 OFFSET 20
asLimit(10, 10); // page 10  = LIMIT 10 OFFSET 90

      Usage with a sql statement;

      sql = "select * from tablename " + asLimit(20, 2);
// select * from tablename LIMIT 20 OFFSET 20;
