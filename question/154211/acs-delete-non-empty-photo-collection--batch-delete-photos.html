ACS Delete Non-empty Photo Collection / Batch Delete Photos

With ACS is there a way to delete a non-empty photo collection (without manually emptying it first), or a way to batch delete photos?

I've gone through the ACS docs and it seems that you can't:

http://docs.appcelerator.com/cloud/latest/#!/api/Photos
http://docs.appcelerator.com/cloud/latest/#!/api/PhotoCollections

// If photos belong to my_collection_id this will always fail with "can't delete non-empty collection"
Cloud.PhotoCollections.remove({ collection_id: my_collection_id }, callbackFunction);

// Will only accept a single photo_id (prohibiting batch deletion)
Cloud.Photos.remove({ photo_id: my_photo_id }, callbackFunction);

So if a PhotoCollection contained 30 photos I would have to send off 30 separate Cloud.Photos.remove requests - which is not very efficient.

Ideally I would be able to either set a flag to Cloud.PhotoCollections.remove to force emptying it's child photos, or pass an array as the photo_id argument to Cloud.Photos.remove
