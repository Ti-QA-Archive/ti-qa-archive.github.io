    Check your Titanium studio configuration setting. In platform/iOS check if the iOS configuration is correct.

    if you have both Xcode5 and Xcode6 installed in your mac then Xcode6 be installed as Xcode 2. To use Xcode6 for iOS 8.0 you need to configure the studio with that Xcode

    Run 

    sudo xcode-select -switch /Applications/Xcode2.app/Contents/Developer/

    in terminal. hope that solves your problem. Thanks.
