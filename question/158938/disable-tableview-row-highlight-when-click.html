Disable tableview row highlight when click

Hi guys,

I have a view inside a table view row with a backgroundColor. When i click on that row, I want the view to remain the same colour.

I know that in iPhone, I can use selectionStyle:Ti.UI.iPhone.TableViewCellSelectionStyle.NONE. But this won't work in Android. I need it to work in iPhone and android.

I have tried setting selectedBackgroundColor to transparent but it has no effect.

Anybody can help? I'm using the latest version of the SDK.

    HI Chung ,

    Try this one,

    tableView :{
    backgroundColor:'transparent',
    separatorStyle:'none',
    selectionStyle:'none',
    separatorColor : 'transparent',
}
