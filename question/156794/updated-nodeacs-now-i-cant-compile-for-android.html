    The support within the QA section is terrible so I figured I would come back and give the solution. This appears to be a Java related issue.

    This should fix the problem:

    1. Download the latest version of the Java JDK here:
      http://download.oracle.com/otn-pub/java/java_ee_sdk/6u4/java_ee_sdk-6u4-jdk7-windows.exe

    2. Within System properties edit your environment variables changing your JAVA_HOME system variable to C:\Program Files (x86)\Java\jdk1.7.0_25

    3. Restart Titanium Studio and try to compile for distribution again.
