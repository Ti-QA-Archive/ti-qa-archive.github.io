Assign Event listener to Button

Hi ,

If i use the following hierarchy , then how can i assign an event listener to the button.

var view = Ti.UI.createView({
    // some parameters here...
});

view.add(Ti.UI.createButton({ // Want to give event listener to this button
    // parameters go here...
}));

// ... later
win.remove(view);
view = null; // deletes the view, button, and their proxies

  • var view = Ti.UI.createView({
    // some parameters here...
});

var button = Ti.UI.createButton({ // Want to give event listener to this button
    // parameters go here...
});

view.add(button);

button.addEventListener('click', function(e) {
    alert('Button clicked! ('+e+')');
});

// ... later
win.remove(view);

    You would normally need to create a reference to that button in order to attach an event listener to that button. However, you could access the button using the children property of the view once its been attached. I don't ever suggest doing this as it can become much more confusing and harder to track than just simply creating a variable for the button.
