[iOS] information about "set a minimum OS version"

Hello to all.
I wanted some clarification on the fact of being able to set a minimum OS for people installing my app ..
currently the SDK apple support as a minimum OS iOS 4.3.
If I wanted to, with the same SDK, how can I set the minimum OS 6.0 using the ti.app.xml with this code

<min-ios-ver>6.0</min-ios-ver>

?

thanks

    Hi

    As follows;

    • Step 1: Edit your tiapp.xml and look for the following section (you may need to swap the the XML view (see tabs at the bottom of page in studio editor);
    <ti:app xmlns:ti="http://ti.appcelerator.org">
...
   <ios>
   ...
   </ios>
...
</ti:app>
    • Step 2 You found it, check the current version number
    <ios>
    <min-ios-ver>4.3</min-ios-ver>
...
</ios>

    • Step 3; adjust

      • Change 4.3 (or whatever value to 6.0.
      • Jump to Step 5

    • Step 4 Does not exist - add the ios section as below;

    <ios>
    <min-ios-ver>6.0</min-ios-ver>
...
</ios>

    • Step 5 Finishing Up

      • Save the file
      • Close the file

    • Step 6 Restarting, remember to close the simulator if running and then use;

      • Menu
      • Project
      • Clean
      • Select your project
      • Confirm

    (Newer SDKs do this automatically but as you have not provided your SDK I have included it).

    Important Note: Ti. SDK 3.1.1 has raised the iOS minimum to 5.0.
