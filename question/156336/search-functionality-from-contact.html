  • Hi

    Without the full question this is only a guess - but you can easily search the table data you have added by associating a table and a searchBar together;

    Adjust your row creation to include a custom property so you can filter the tableView;

    var row = Titanium.UI.createTableViewRow({
    filter: people_name // set this to the variable you wish to filter
});

    Add a searchBar and assign it to the tableView;

    var search = Ti.UI.createSearchBar({});
// you do NOT add the search bar to the parent window or view
var table = Ti.UI.createTableView({
   data: tableData,
   filterAttribute: 'filter', // this links the variable with the table and search
   search: search, // this links the search  bar with the table
   top: '20%',
   backgroundColor: 'white'
});
