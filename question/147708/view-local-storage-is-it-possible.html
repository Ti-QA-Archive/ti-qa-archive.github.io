View local storage, is it possible?

Hello.

Is it possible in titanium to store a view in local storage. I want to recover full view (properties, childs, events,… ) if the app restart or crashes.

I've tried with properties.setObject, but it doesn't work.

Any way to do this?

Thanks.

    Hi

    The short answer is no. The windows, views, buttons, labels, imageView etc are more than the sum of their parts. So whilst you can store the same properties that you used to create them you cannot store their current state to the point the they simply come back to life again after retrieving them.

    This is true for the Properties which you tried and the file system.

    Sorry.
