Mobileweb: alertDialog message gets truncated

Hi,

I've just noticed that mobileweb truncates the message from alertDialogs when it's bigger than a certain size.

Here's some code:

Ti.UI.setBackgroundColor('#000');

var win = Ti.UI.createWindow();


var alertDialog = Titanium.UI.createAlertDialog({
            title: "this is a title",
            message: "this is a long message that will be trucated on mobileweb"
        });
alertDialog.show();


win.open();

Tested with the nightly build from 19th May. If you open a full desktop browser it shows fine but in a frame of 320x480 or mobile browser it gets truncated.

Thanks
