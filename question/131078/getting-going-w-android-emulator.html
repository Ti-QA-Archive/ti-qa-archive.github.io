getting going w/ android emulator

Hi everyone. I am just having a little trouble getting started. My son is successfully using titanium to write javascript on his apple lap-top. I have eclipse on my desk-top at home and am in the process of learning Java. I am trying to put titanium on my wife's PC lap top which we have with us while visiting my son.
I have downloaded JDK and the android SDK. I have downloaded titanium version: Titanium Studio, build: 1.0.7.201112281340. I am running windows 7. With the help ofthis forum, I added a line to the path and I added a JAVA_HOME variable. Again withyour help, I changed the JDK version to 1.6 and the JRE to 6, which helped me along. I imported the hello World sample code and did not change it in any way.
So, Everything seems to work fine until I hit the run button. I chose the android emulator and the emulator comes onto the screen just fine, populated w/ several icons. It does not contain the Hello World icon or bar that shows up in the first on-line turorial. I have tried doing this on my son's computer and it seems to work fine. Any thoughts? Advice? Thankyou

    It's hard to guess at the issue without you posting the log, however here's a guess. In Titanium Studio, edit your Run Configurations. In the Android Emulator section, make sure the Android API is set to a "Google APIs" version such as "Google APIs Android 2.3.3". Do a clean, touch tiapp.xml, then run again. If it doesn't work then post the log here.

    • That tells you that the path to your jdk bin isn't included in your PATH variable.

      If this is Windows 7:

      1. Controls + pause (opens up system window)
      2. Advanced settings - Advanced tab
      3. Environment Variables
      4. System Variables
      5. Edit Path - add to the end of it <existing path>;C:\java\bin (or wherever you installed the jdk to)
