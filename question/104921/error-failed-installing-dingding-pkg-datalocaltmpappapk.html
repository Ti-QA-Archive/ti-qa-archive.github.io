  • According to your trace, you are using Ti SDK 1.5.1 from 16 Dec. I think this is a continuous build. Either use a major release, or install the latest 1.5.X continuous build using the instructions in the Continuous Builds guide (note the info about the hash!).

    Also, delete your build files, and make sure you have SDK "APIs 2.2" selected on the Run Emulator screen.

    Hope this helps
