How to extract the string from the xml file given as sever response

I want to print the server response ,When i am doing this

~~
alert(xhr.responseText);
~~

It is just showing the whole xml file given as server response.
My xml as sever response is as follow

<?xml version="1.0" encoding="utf-8"?>


    <HelloWorldResponse >
      <HelloWorldResult>string</HelloWorldResult>
    </HelloWorldResponse>

I just want to print the string in the hello world result .

  • Accepted Answer

    see http://wiki.appcelerator.org/display/guides/Working+with+Remote+Data#WorkingwithRemoteData-RetrievingandparsingXML

    var doc = Titanium.XML.parseString(xhr.responseText).documentElement;
var products = doc.getElementsByTagName('HelloWorldResult');
alert(products.item(0).text);
