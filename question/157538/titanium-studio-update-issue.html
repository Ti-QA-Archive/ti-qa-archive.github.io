    • Team, i solved the issue 4 days before, but, it now work

      Steps

      1. Unistall Titanium SDK
      2. Delete folder 'Users/<user_name>/Library/Application Support/Titanium'
      3. Download the latest version of Titanium SDK and install
      4. Download the latest version of node from http://www.nodes.org and install

      5. Open a Terminal and make the following commands:

        sudo su
export PATH=$PATH:/home/<user_name>/local/node/bin
curl https://npmjs.org/install.sh | sh
chown -R ec2-user /home/<user_name>/local/node
exit

      6. Open a Titanium and make the update to Titanium Cli, Alloy and node

      7. Make the following instructions to configure Titanium CLI http://docs.appcelerator.com/titanium/3.0/#!/guide/Titanium_Command-Line_Interface_Reference
      8. Done

      Regards
