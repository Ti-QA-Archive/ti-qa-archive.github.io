evalJS not working get error

for evalJS i used following example

app.js

var window = Ti.UI.createWindow();
var webview = Titanium.UI.createWebView({url:'message.html'});
window.add(webview);
var str = "Hello world!";
webview.addEventListener('beforeload',function(e)
{
webview.evalJS("var message="+str+";");
});
window.open();

As the JS ‘message’ variable is assigned before loading the webview, it is also available in the html page.

message.html

<html>
<head>
<script type="text/javascript">
alert(message);
</script>
</head>
<body>
</body>
</html>

but after running i get the following error
getJSValue:javascript:_TiReturn.setValue((function(){try{return var message=Hello world!;+"";}catch(ti_eval_err){return '';}})());
