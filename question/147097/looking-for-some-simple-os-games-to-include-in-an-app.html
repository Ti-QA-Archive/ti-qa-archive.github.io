    I know there might be a few you can find if you search GitHub. With the launch of Lanica, people have been waiting for its release to start creating their Titanium games. Another simple way to get some light games into your app would be to find some web based, or local html5 based games and drop them in for use in a webview. Back in the day, people did this all the time. See Golingo. Its an older example, but it should give you an idea of what Im talking about.
