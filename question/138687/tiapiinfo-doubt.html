    This is just for logging purpose. in your code if you want to log something on your console window of Titanium studio.
    for example
    I have a variable x i.e.
    var x

    and after doing some operation on x I want to check the value of x and I dont want it to show it on screen so I will do Ti.API.info(&#39;value of x is&#39; + x);

    This will print the value of x in the console window.
