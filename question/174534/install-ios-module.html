  • Hi Shekhar Saraf,

    After downloading your ios module zip file you can install it using studio, go to Help>Install Mobile Module then browse your zip file and select output location then press ok button. If you want this module will get all application select output location Titanium SDK or if you want specific project select mobile app project. Another way of module installation create a folder named modules in root project then paste the unpack module in this folder and go to the tiapp.xm file to check it.
