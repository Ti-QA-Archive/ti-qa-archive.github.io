Enable Android Byte Code Compile

I remember hearing in last weeks Titanium webcasts that as of 1.5 when compiling for distribution they are now compiling down to an almost byte code level for android. They mentioned there was a TiApp.xml flag that could be set to enable this in debug/run on device mode.

Does anyone know the flag?

    The bytecode compiler is enabled by default when building for distribution, but you can enable it for development mode with this property:

    <property name="ti.android.compilejs" type="bool">true</property>
