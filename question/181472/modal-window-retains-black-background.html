How can I fix this?

  • Hi Amresh Beuria,

    Earlier i also tried to make transparent background to modal window but i didn't find any solution for that.

    But if you want to open like pop up then you can do it by using views with nice animation.

    and here also you doing mistake. If you assign opacity to any parent view it's children views opacity also set according to that.. instead of that use background color with transparency. more info about transparent background read this.

    And here i send you some sample code for pop up. hop this is helps you

    //xml

<Alloy>
    <Window id="winIndex" class="container">
<!-- HERE YOU REST OF CODE FOR WINDOW 

WINDOW LAYOUT MUST BE NON VERTICAL AND NON HORIZONTAL -->

...
<Button id="btnPopup" onClick="openPopUp"></Button>
        <View id="popupBackView" onClick="openPopUp">
            <View id="popView">
                <Label id="lblTest"></Label>        
            </View>
        </View>
</Window>
</Alloy>

//tss
"#btnPopup":{
    title : "PopUp"
}
"#popupBackView":{
    height : "100%",
    width : "100%",
    opacity : 0,
    backgroundColor : "#80000000"  //If you want to change transparency. i sent you one link refer that
}

"#popView":{
    height : 300,
    width : 280,
    backgroundColor : "white",
    borderRadius : 5,
    touchEnabled : false,
}

"#lblTest":{
    color : "red",
    touchEnabled : false,
    text : "TEST"
}


//js

var isOpen = false;
function openPopUp(){
    if(isOpen){
        $.popupBackView.animate({
            opacity : 0,
            duration : 300
        });
    } else {
        $.popupBackView.animate({
            opacity : 1,
            duration : 300
        });
    }
    isOpen = !isOpen;
}

  • To make any window or view transparent, just add two more hex digits before the hex color code to the background property like this backgroundColor = "#aarrggbb", where aa=transparency values, rr=red, gg=green,bb=blue.

    All values can be from 00 to ff(in hex format)
