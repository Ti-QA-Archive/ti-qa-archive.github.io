Apply font size to Label using varible

I need to apply a font size to a Label using a variable defined in the app.

Example of coding without variable:

var iTamano = 20;
var lblMensajeCentral = Titanium.UI.createLabel({
    color:'#400040',
    text:'CAS Móvil 2012',
    font:{fontSize:20,
          fontFamily:'Helvetica Neue'},
    textAlign:'top',
    top: '2%'
});

    Leo,

    I'm going to answer this question for you, but if this is something you're really having a problem with, I'm going to have to suggest that you hit up some basic javascript tutorials. I usually suggest the w3schools javascript section when first starting out.

    Here is your example, using the variable:

    var iTamano = 20;
var lblMensajeCentral = Titanium.UI.createLabel({
    color:'#400040',
    text:'CAS Móvil 2012',
    font:{fontSize:iTamano,
          fontFamily:'Helvetica Neue'},
    textAlign:'top',
    top: '2%'
});
