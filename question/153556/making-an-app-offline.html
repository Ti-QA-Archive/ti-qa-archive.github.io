  • Accepted Answer

    • how do I set up the database and primary contents at installation :
      you bundle the database with the files or download it when the application starts and save it for later use. Downloading is done via Ti.Network.createHTTPClient(); sqlite best practices here
    • how do I detect 'online' and 'offline'/'unstable' status of the device?
      the fastest way is to use if (Ti.Network.online)

    As I said, the app can download the sqlite file from your web server and install it using the db commands.
