  • If you check out the device console using Xcode's Organizer all of your API.info() type calls and javascript errors end up there.

    The JS is not actually natively converted. As I understand it Ti creates a javascript runtime environment then when you call a javascript function starting with Ti. it executes that call against native code. So when you compile your app you will see javascript errors in the consol and native crashes in the crash log.
