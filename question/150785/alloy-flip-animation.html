Alloy Flip Animation

Hi

I want to create an iOS flip animation using Alloy. I know that in the old way of doing it you would create a view for the back side of the animation but not add it it's parent and then show it as follows: 

parentView.animate({view: backView, transition:Ti.UI.iPhone.AnimationStyle.FLIP_FROM_LEFT});

but is it possible to define all of the views in the Alloy View and then flip between them, something like:

// View
<View id="parentView" >
    <Button title="btn" onClick="flipScreen" />
    <View id="frontView" />
    <View id="backView" />
</View>

// Controller
var flipped = false;

function flipScreen() {
    if ( !flipped ) {
        flipped = true;
        $.parentView.animate({view: backView, transition:Ti.UI.iPhone.AnimationStyle.FLIP_FROM_LEFT});

    } else {
        flipped = false;
        $.scrollView.animate({view: frontView, transition:Ti.UI.iPhone.AnimationStyle.FLIP_FROM_LEFT});

    }
}

… or is it necessary to just create the backView the old fashioned way within the controller, as Alloys seems to add all the views automatically? I hope that makes sense.

Thanks,
James

  • Hi James,

    I have tried flipping of animation using transition effect on view. Hope it might help you. You can apply 2 way transition effect by creating 2 views.

    var casinoDetail = Alloy.createController('casinoDetail', {
    _detail   : data,
   }).getView();

   casinoDetail.open({
    transition : Ti.UI.iPhone.AnimationStyle.FLIP_FROM_LEFT
   });
