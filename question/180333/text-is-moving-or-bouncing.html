Text is moving or bouncing

Platform : IOS
SDK : 3.3.0

Hi, Iam adding a label to the scrollview. My problem is the text is bouncing or moving left and right when scrolling, here is my code.

var scrollview = Ti.UI.createScrollView({
        contentWidth:'auto',
        contentHeight:'auto',
        top :'25%',
        left : '15%',
        height : '55%',
        width : '70%'
        layout:'vertical',
        showVerticalScrollIndicator:true,
        showHorizontalScrollIndicator:false,
    });
    win.add(scrollview);

    var label2 = Ti.UI.createLabel({
        text : 'Iam a text'
        font : {fontSize : '18',fontFamily:'Helvetica Neue'},
        color : '#000'
    });
    scrollview.add(label2);

    Hi Prashaanth,

    Assign width property "100%" to label.
    e.g

    var label2 = Ti.UI.createLabel({
    text : 'SAMPle TEXT',
    width : "100%",
    font : {
        fontSize : '18',
        fontFamily : 'Helvetica Neue'
    },
    color : '#000'
});
