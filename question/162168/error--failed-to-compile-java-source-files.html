  • Not sure where to start.

    Have all the new updates for every platform, newest SDK and etcetera..

    Folks in the past have claimed to be running the latest, yet upon further investigation, install problems actually had them running older versions, or pieces were missing. Such statements also tend to betray a laziness regarding actually posting a complete question as carefully instructed to on the question submission form.

    E:\Övriga program\

    In the past with the python build scripts (yes, we have moved towards node), spaces in paths caused problems.

    E:\Övriga program\

    In the past, high ascii characters in paths and file names caused problems.

    E:\Övriga program\

    In the past, on windows, installs on drives other than C: caused problems.

    C:\Program Files\Java\jdk1.7.0_51

    Referencing Installing Oracle JDK it should be noted that 1.6 32 bit is listed as the supported version of the JDK. With 3.2.0 some changes were made in an attempt to work around the 1.7 JDK keytool changes.

    How about you run ti info and paste the results here so we can take a look at your setup.
