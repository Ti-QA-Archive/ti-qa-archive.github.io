Websocket in MobileWeb in Titanium 3.0.x

Is it possible to implement a websocket in MobileWeb? Ti.Network.socket is not supported so I can't use NodeJS implementations.

…and don't tell me to use a WebView, because as of Titanium 3.0 the Titanium object is undefined, so I can't use Ti.fireEvent to send the incoming messages to the Titanium app.

    If you're using Titanium Mobile Web, you can also use whatever APIs the web browser supports.

    if (Ti.Platform.osname == "mobileweb" && "WebSocket" in window) {
    // yay, you can use WebSockets!
}
