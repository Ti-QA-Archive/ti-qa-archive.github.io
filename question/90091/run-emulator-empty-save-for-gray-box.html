"Run emulator" empty save for gray box

I just created my first project.
During creation, I see a "Android SDK found" with a green check. So far so good.
Project gets created, and from reading here: http://developer.appcelerator.com/guides/en/getting_started.html it's my understanding that I should now have some kind of "Default app".

When I go to "Test & Package" though, the "Run Emulator" area is blank, unlike the KitchenSink which has a subtab "Android. "Run on device" has this android subtab, but also says "This project is not configured to run for Android".

I don't understand what I'm missing. Running on Linux, 64-bit.

  • Couple of things to check:

    • In the profile perspective of Titanium Developer (third button in strip of buttons at the top), make sure the Android SDK is pointed to the top-level directory where you installed the Android SDK.
    • If you run the 'android' command in the Android SDK tools directory, you should have a number of 'installed packages'. This should include the android tools and at a minimum, the 1.6 Android SDK
