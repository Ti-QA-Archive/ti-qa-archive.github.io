iPhone Retina Simulator 4-inch - cant see home button.

OK so I hope there is an easy solution. When I test my app with the iPhone Retina 4-inch simulator, I cant see the home button because its below the bottom of my laptop screen. Its hard to test all the functionality since I can't make the window smaller. Any suggestions?

    You can use shortcut:

    ?+?+H to simulate Home button click

    btw, you can resize the iOS simulator:

    ?+1 for 100%

    ?+2 for 75%

    ?+3 for 50%
