how to disable scroll table view in android

When scroll by mouse, i see table view row is focused, i'd like change background color of the table view row , but can not

  • Duc

    Do you mean when a row is selected? If so, the backgroundSelectedColor or selectedBackgroundColor properties of Ti.UI.TableViewRow should do it, although the former only works on Android according to the APIdocs.

    var thisRow = Ti.UI.createTableViewRow({
//  backgroundSelectedColor:"red", // android only
  selectedBackgroundColor:"green",
  className: "grid",
  height: 100
});

    Cheers
