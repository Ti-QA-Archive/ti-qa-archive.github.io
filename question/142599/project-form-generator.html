[project] Form Generator

Hey guys,

cause form creation in Titanium i such a pain and there isn't a "really cool" gui Editor out now (addional for free) i've createt a class to convert "complex" arrays to forms.

JEAH :P

In the current version the "source Array" can be look like this:

var formGroups = ['Persönliche Angaben', 'Sonstiges'];    
    var fields = [
        { title:'Name', type:'text', id:'name', group:0  },
        { title:'Password', type:'password', id:'name', group:0  },
        { title:'E-mail', type:'email', id:'email', group:0 },
        { title:'Are you sure?', type:'boolean', id:'einverstanden', group:0, startValue:true },
        { title:'Really?', type:'boolean', id:'einverstanden', group:0, startValue:false },
        { title:'Gehalt (€)', type:'slider', id:'gehalt', group:1, minValue:0, maxValue:300, startValue:100 },
        { title:'Gehalt (€)', type:'slider', id:'gehalt', group:1, minValue:0, maxValue:500, startValue:100 },

        { title:'State', type:'picker', id:'state', group:1, data: [
            'Alabama', 'Alaska', 'Arizona',    'Arkansas',
            'California', 'Colorado', 'Connecticut', 'Delaware'
        ]},

        { title:'Time', type:'picker_time', id:'zip', group:1 },
        { title:'Date', type:'picker_date', id:'phone', group:1 },
        { title:'Date & Time', type:'picker_datetime', id:'password', group:1 }
    ];

Screenshot: http://s1.directupload.net/file/d/3023/2zb7d2hf_png.htm

I am currently working on more "customize" features on the different possible items. For the UI i've choosed the "basic ios layout style", so everything looks "nativ" :) Allthough i am working on 2 addional styles, so you can choose from either of them.

In a future Version i wanna implement the function to change colors etc. like you want.

I am posting this, cause i wanna ask you for ideas, "Must have" Features etc. The current Build is 80% ready and will be published next week.

Best regards,
Nico

Any kind of feedback is welcome!
