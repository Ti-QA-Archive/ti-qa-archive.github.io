Is it possible to set make the title text go to a new line?
i.e. title: 'Hi XXX\n\rHow are you?',

    Yes you can. The way is to build an costume optionsdialog:

    var optionsdialog = Ti.Ui.createView({
    bottom:-300,
    backgroundImage:'background.png'
});
// put someting in this view …
someElem.addEventListener('click'.function(){
    optionsdialog.animate(Ti.UI.createAnimation({
        bottom:0
    }));})
