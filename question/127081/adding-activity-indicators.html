Adding activity indicators

I am building a comic reader style application and am wanting to have a coverflowview loading images dynamically (from a database eventually…array at the moment) and am wanting to have an activity indicator in the code which will show in place of the coverflowview while the images are loading.

I have images loading but am struggling with the activity indicator, here's what i have got so far.

//CoverFlow holding comic covers 
var comicFlow = Titanium.UI.createCoverFlowView({
     images: imagesUrls2,
    top: 0,
    animate:true
});

setTimeout(function(){
    comicFlow.selected = 5;
},300);

// click listener - when image is clicked
comicFlow.addEventListener('click',function(e)
{
    Titanium.API.info("image clicked: "+e.index+', selected is '+comicFlow.selected);    
});

// change listener when active image changes
comicFlow.addEventListener('change',function(d)
{
    Titanium.API.info("image changed: "+d.index+', selected is '+comicFlow.selected);    
});

where imagesUrls2 is the portion of the array being loaded if that's of any help/relevance to anyone.

Look forward to hearing from anyone..
