  • Accepted Answer

    You can manually install Node.js by downloading and compiling it from http://nodejs.org/dist/v0.8.23/.

    If you're ok with using only Titanium SDK 3.1.0.GA or newer, then you can run the following at the Terminal:

    sudo apt-get install nodejs
