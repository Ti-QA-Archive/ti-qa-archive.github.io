Syntax error: Unexpected token "delete"

Hello,

I am trying to use events delete function but Ti Studio does not allow me to do that.
I cannot use Cloud.Events.delete because then I get the error:

Syntax error: Unexpected token "delete".

How can I fix this?

    There is no such method as "delete". The method for removing an event is "remove". Please check the docs: http://docs.appcelerator.com/titanium/2.1/index.html#!/api/Titanium.Cloud.Events
