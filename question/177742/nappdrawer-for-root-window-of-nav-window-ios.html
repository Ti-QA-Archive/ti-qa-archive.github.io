NappDrawer for root window of Nav Window iOS

I have a iOS NavigationWindow and i have given that as the centre window in my drawer, and added a root window to it
When i click this window another window opens with a back button. In it i don't want the drawer to be seen however when i swipe its there.
Can i apply the NappDrawer only to the root window of the nav window ?

    According to the documentation, there is an API property OpenDrawerGestureMode to set gesture support.

    var mainWindow = NappDrawerModule.createDrawer({
    ...
});

...
// create and open child window
...

mainWindow.setOpenDrawerGestureMode(NappDrawerModule.OPEN_MODE_NONE);
