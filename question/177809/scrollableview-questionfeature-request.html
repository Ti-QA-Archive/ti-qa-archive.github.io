ScrollableView Question/Feature request

I always liked the ScrollableView component.

Now, I have 4 views (pages) inside and I need to go from page 0 to page 3 WITHOUT showing page 1 and 2. I know there is a method "scrollToView(numberOfTheView)" but this one doesn't do what I'm looking for. What's the best way to achieve a similar behavior? Has anyone ever tried?

Feature request: add a jumpToView(numberOfTheView) method.
