    By library do you mean one of the Ti+ modules available to Indie plan subscribers? If so, then AFAIK there are no restrictions to publishing an app which uses one or more modules. Be aware that if you let you Indie plan expire, app versions already published will continue to work but you can't recompile a project using a Ti+ module without a current license.

    That is, you need to have an active subscription whenever you (re)create a project.
