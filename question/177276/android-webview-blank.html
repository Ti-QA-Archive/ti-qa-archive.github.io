Trying to build an in-app viewer for external documents (url supplied by an API)

var args = arguments[0] || {};

  • There's no need to encode the URL before setting the web view's URL property. From your snippet, I see no reason why you're setting args.url = 'the_url' then setting the web view's url property equal to args.url. There's no need for that extra assignment, and would lead only to unnecessary memory consumption. Maybe that's just a result of posting only a snippet of your code.
