Popup with a textfield cusor position

Hello All,

I have a problem with a popup with a textfield. I want that the cursor stands on the back of the input. But as you can see on the picture the cursor always appears before the input. Does anyone know a solution to this problem?

Thanks in advance

var input_text = Ti.UI.createTextField();
input_text.value = '123456789';

var dialog = Ti.UI.createOptionDialog({
    androidView: input_text,
    id: e.source.id,
    buttonNames: ['Opslaan', 'Annuleren'],
    title: 'Telefoonnummer',
    jsonTitle: e.source.jsontitle
});

    You can set the cursor position using TextField.setSelection(start,end) and setting start & end to the same value.

    After you open your dialog, try:

    input_text.setSelection(input_text.value.length,input_text.value.length);
