Show/Hide the keyboard when search bar clicked

Hello all, i have a searchbar. When the application starts, I want that the keyboard will not pop up, only it will pop up when user clicks on the searchbar. I saw many posts and comments ( Ti.UI.Android.hideSoftKeyboard(), blur()…) but always the same problem, the keyboard always shown on the screen! I'm testing on my samsung galaxy tab 10.1. Please i need your help :)

  • try this one…

    var search = Titanium.UI.createSearchBar({
    barColor : '#000',
    backgroundColor : '#fff',
    showCancel : true,
    focusable : false
});

search.hide();
setTimeout(function(e) {
    search.show();
    search.blur();
}, 500);

  • try this one

    var search = Titanium.UI.createSearchBar({
    barColor : '#000',
    backgroundColor : '#fff',
    showCancel : true,
    focusable : false
});

currwin.addEventListener('focus', function(e) {
    search.hide();

    setTimeout(function(e) {
        search.show();
    }, 500);
});
