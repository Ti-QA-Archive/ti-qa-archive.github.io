  • You can open html files in webview. Suppose the above code is in file index.html then in your app.js write the following code:

    var win = Ti.UI.createWindow({
    backgroundColor: '#fff'
})

var webview = Ti.UI.createWebview({
    url: 'index.html',            // path to the html file.
});

win.add(webview);
win.open();

    You can see the html output in the webview.
