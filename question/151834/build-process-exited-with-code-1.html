    • In tiapp.xml file the required SDK can be set using the following. If nothing is specified then it seems to default to 2.3.3

      <uses-sdk android:targetSdkVersion="11"/>
<tool-api-level>11</tool-api-level>

      — commented October 2nd 2013 by Vello Pi

    • Thx man, after installing 2.3.3 everything worked great!

      — commented December 2nd 2013 by Jakob Lehner

    • For me replacing the android line: 

      <android xmlns:android="http://schemas.android.com/apk/res/android">
</android>

      with this worked

      <android xmlns:android="http://schemas.android.com/apk/res/android">
    <tool-api-level>19</tool-api-level>
    <abi>all</abi>
    <manifest>
      <uses-sdk android:minSdkVersion="8" android:targetSdkVersion="19"/>
    </manifest>
</android>
