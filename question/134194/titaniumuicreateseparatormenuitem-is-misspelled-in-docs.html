Titanium.UI.createSeparatorMenuItem is misspelled in docs

Not a question so much as an alert to those encountering the same problem I did with this method. In the docs, the method is spelled createSeperatorMenuItem. After scratching my head for 20 minutes, I realized the method name is just spelled incorrectly.

The correct spelling and usage is: var separator = Titanium.UI.createSeparatorMenuItem();
