Read Widget Version Number?

Is there a way to programmatically read the version of an Alloy Widget? I have a QA widget that would do well to report the versions of every versionable part of an app.

— asked May 29th 2014 by Joseph Beuckman

  • Accepted Answer

    Widgets are part of the dependencies of your project, and a available in the Alloy.CFG object. 

    _.map(Alloy.CFG.dependencies, function(version, widgetName) {
    Ti.API.info(widgetName + ' is at version ' + version);
});
