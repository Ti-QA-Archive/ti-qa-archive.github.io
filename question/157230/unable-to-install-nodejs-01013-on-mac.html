Unable to install nodeJS 0.10.13 on Mac

I've had problems updating a many months-old version of Ti Studio, so I uninstalled the app and reloaded it. That fixed most of the update problems, though the "Provide your system password" prompt still fails to accept my system password every single time (fails 3 times, then opens a terminal window that does accept the password). However, I cannot get nodeJS 0.10.13 to install. Aside from the 3 pwd failures, in the terminal window, it executes this command: sudo npm install -g, which fails with "Could not install .". Shouldn't it be naming the nodeJS package name after the -g? If so, can someone tell me what the package name is (under Mac OS X)?
