Listview - first Listitem is blank (white)

We are having a apparently relatively rare random issue where some users are reporting that the first listitem in a listview is just white. We are not able to reproduce this issue when testing nor when using the build in App Store, but have received several screenshots from our users that prove this. Some report that they can get a correct listview rendering, if they restart the app, but the problem will resurface on another restart. I'm not posting any test case, since I assume this is a more generel listview issue. Anyone else experiencing this, or advice on steps to take to avoid this is appreciated.

Titanium SDK: 3.2.2
Platform & version: iOS 7
Device: iPhone
