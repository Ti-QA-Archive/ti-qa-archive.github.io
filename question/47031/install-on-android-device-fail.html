  • Have you installed the latest Android on your HTC Hero?
    You have to update to Android 2.1!

    First install the ROM update provided on HTC support site.
    http://www.htc.com/de/SupportDownload.aspx?p_id=283&cat=2&dl_id=775

    Then sometimes you have to set the Hero's internal calendar to 2011 before it will prompt you with a message to update your firmware.

    Try googling for "HTC Hero Android 2.1".
