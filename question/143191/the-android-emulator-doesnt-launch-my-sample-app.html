The Android Emulator doesn't launch my Sample App.

Hello.

I think it is some permission problems.

  • It often takes the first run on the Android emulator quite some time to launch. Try launching the Android emulator, then while the emulator is running, launch the app. The app will be installed into the already running emulator.

    Also, make sure that you are using the Google APIs version of the Android sdk.
