Downloading and updating an XML or JSON file

I'd like my iOS and Android app to occasionally download an updated version of an XML file and save it to disk, overwriting the file that was there before. This is a file full of content that may need to be adjusted from time to time. Will I run into any permission issues doing this, or is it pretty straightforward?
