    We do this by having a system which stores the messages on a cloud database and serves them up in the app.

    We also handle the push notification, which if the app is closed opens the app and using specific links within the notification, we then handle navigation within the app.

    However if the notification isn't selected then nothing will happen, and that is a user option, which is why we handle a messaging system on top of the push notifications, as the user can also switch off notifications for the app.

    Hope this helps, but the solution is to code it.

    T.
