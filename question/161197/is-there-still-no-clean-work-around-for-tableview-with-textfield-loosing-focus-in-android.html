Is there still no clean work around for tableview with textfield loosing focus in android?

Created an app that works fine in ios but when used in android the textfield looses focus on click, read that scrollview should be used for android but already using a lot of tableview functions and would like to avoid remaking code.

Read one with event on focus then blur it will refocus the textfield, but the keyboard switches from number type to normal and to number type again that is not good to look at.

Also tried putting textfield in view and adding, it to row did not work.

Is there a cleaner way of doing it?

  • after some more searching found another solution SOFT_INPUT_ADJUST_PAN

    sample code:

        var self = Ti.UI.createWindow({
        backgroundColor:'black',
        //ADD THIS IN YOUR WINDOW        
        windowSoftInputMode: Ti.UI.Android.SOFT_INPUT_ADJUST_PAN,
    });

    hope this also help some others that are having the same problem
