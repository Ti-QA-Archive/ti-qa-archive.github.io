    Hi

    I see you still do not have an answer for this, in case it is still unresolved you should try this;

    clear.addEventListener("click", function() {
    message.hide();
    tableData = []; // you have not been resetting this
    table.setData(tableData);
});

    Because you have not been resetting the the array you place the rows into it simply stays set and gets appended too.
