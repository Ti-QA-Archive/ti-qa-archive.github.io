How can I change the location of or remove the remote-image-cache folder on the SD card?

All my apps automatically create a folder named after the ID of the app on the SD card, and within it a folder named "remote-image-cache". I'm already creating a separate folder for app files because I don't like using the app ID as the folder name. How can I make the app use my own created folder for the "remote-image-cache" folder, or how can I make it not create these folders automatically (to my knowledge they are not being used)?

I'm using Titanium SDK 1.7.5.
