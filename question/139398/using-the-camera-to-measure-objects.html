Using the Camera to measure objects?

Would it be possible to use Titanium to build an app that measures objects using geometery from the camera and accelerometer, similar to this app (Smart Measure) in Google Play:

https://play.google.com/store/apps/details?id=kr.sira.measure#?t=W251bGwsMSwxLDIxMiwia3Iuc2lyYS5tZWFzdXJlIl0.

From the description the app estimates the height of the camera, the distance to the object, and uses the accelerometer to measure the angle and then calculates the objects height.

I know how to use Titanium to access the camera and accelerometer, beyond that any suggestions how one might approach building an app like this with Titanium? Any thoughts or suggestions would be greatly appreciated, thanks.
