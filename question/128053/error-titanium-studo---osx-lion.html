  • I think this is because you no longer have XCode and the iOS SDK installed. When you upgrade to Lion, you need to re-download iTunes, XCode (iOS SDK included) Did you do these steps?

  • Often times you can get more descriptive errors from XCode itself when encountering this error with the Titanium build scripts. Go to your project's build&#x2F;iphone directory and double click on the xcodeproject file contained in it. This should open your Titanium project, prepared for iOS, in XCode. Try to deploy the project to a device using XCode. If you still encounter an error, there should be a more descriptive one in the XCode build logs. This may aid your troubleshooting.
