  • It's nothing special

    var mapview = Titanium.Map.createView({
mapType: Titanium.Map.STANDARD_TYPE,
region:{latitude:win.lat, longitude:win.lon, latitudeDelta:0.01, longitudeDelta:0.01},
animate:true,
regionFit:true,
userLocation:true
});win.add(mapview);

    Then I add a few annotations

    for(i=0;i<response.length;i++){
        var myPin = Titanium.Map.createAnnotation({
        latitude:response[i].lat,
        longitude:response[i].lon,
        title:response[i].title,
        pincolor:Titanium.Map.ANNOTATION_GREEN,
        animate:true,
        rightButton: 'mapbutton.png'
        });
        mapview.addAnnotation(myPin);
}//end for

    It could be 1 pin or 20, I still get the same error, and if I tell it to wait, it works fine. I also have some routing code from a module, but I took that out and it doesnt change anything.
