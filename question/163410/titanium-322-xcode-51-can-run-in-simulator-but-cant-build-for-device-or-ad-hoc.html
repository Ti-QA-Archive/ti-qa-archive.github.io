Titanium 3.2.2, XCode 5.1: can run in simulator but can't build for device or ad-hoc

Configuration:
XCode 5.1
Titanium SDK: 3.2.2
OS X 10.8.5

Things were working fine on previous versions of both (XCode, SDK 3.2.1 GA). Now, today, after install of latest XCode/Titanium SDK bits, this is the situation:
1: Am able to build and run simulator builds
2: Attempts to build for device, or ad-hoc, apparently do nothing. Seems like there's some disk activity, but no other apparent action. Nothing in the console, etc.

Other evidence:

  • Going into the "Configurations" option (an option on the "Distribute" drop-down), and attempting to view any of the existing configurations shows a IDE "Problem Occurred" dialog ("java.lang.NullPointerException").

Anyone else seeing this? Any hints as to how to debug this?

Many thanks in advance.
