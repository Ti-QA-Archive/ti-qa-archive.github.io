  • Hi

    When you say "There is no appcelerator icon on the toolbar to open the dashboard", I am not sure if you are having trouble starting the app, if you;

    • Spotlight search
    • start typing Titanium
    • Enter

    Titanium creates mobile web, iPhone/iPad and Android (and Blackberry) apps all in the same app.

    • File
    • New
    • Titanium Project (top one)
    • Default Project (you can be ambitious later)
    • Next
    • Enter Project Name (Enter Test)
    • Choose a location (or leave it where it is)
    • Enter an app ip (usually reverse domain, examples; com.microsoft.someapp, com.yourcompany.yourapp)
    • Choose WHICH platforms you are designing for
    • Finish

    You are app template will be created, start editing;

    When you are ready;

    • Choose the Run button (green circle on the left of the screen with white play button and yellow folder)
    • iPhone Simulator (or whichever one you want)

    You are up and running.
