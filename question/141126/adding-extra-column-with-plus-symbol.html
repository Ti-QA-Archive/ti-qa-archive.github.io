  • Hi Rohith

    If you are trying to replicate the iOS contacts app 'add address' screen by having a green plus button to the left of a row then - I am afraid there is no built in way to achieve this.

    So you have two routes;

    • Add a button using an image and no border on the left of the row - it will will appear inside the row however.
    • you could create a hybrid view using a scrollview and lots of fixed non scrollable tableviews with only one or two tableviewrows each. This way you can make the width of the table with the grouped style small enough to fit an imageview to the left - which will appear outside of the tableview row.

    The second solution will solve the problem perfectly but it is a lot of work though - so I would question the benefit of the extra effort.
