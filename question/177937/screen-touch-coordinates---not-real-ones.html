screen touch coordinates - not real ones

I am trying to get the coordinates of the main window using touchstart event:

win.addEventListener("touchstart", function(e) {

    console.log(e.x);
});

But if i'm touching a view or label inside the window, it gives the views/labels coordinates, how can I bypass that, in order to get only the main window coordinates?

    As the document states for the event, "X coordinate of the event from the source view's coordinate system."

    What you need to do is translate this view's coordinate system to the window's coordinate system using convertPointToView

    win.addEventListener("touchstart", function(e) {
    var point = { x: e.x, y: e.y };
    if (e.source != win) {
        point = e.source.convertPointToView(point, win);
    }

    console.log(point.x);
});
