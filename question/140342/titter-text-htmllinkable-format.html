Titter Text HTML/Linkable format

Is there a a way to take the text from a twitter feed and make the links click-able. If possible I would rather skip paying for a 3rd party module/plugin.

    Do you have access to the ti.styledlabel module that comes free with some Appcelerator account levels?

    If you do then not only will this solve your problem, it also comes with a code example that does exactly what you need - each status message is parsed and the resulting links are clickable.

    I have checked the market place for the module to see what it costs if you do not get it for free and whilst the cost per seat is not a problem - the monthly bit seems a bit steep.
