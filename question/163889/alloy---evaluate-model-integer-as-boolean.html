Alloy - Evaluate model integer as boolean

I'm using binary integers pulled from a database to determine whether to show certain elements. For instance, the table might have a column 'open' with a value of either 0 or 1. This value is stored in the Alloy model as an integer with the same name.

Then in the markup I would ideally do something like this to either show or hide the element if the model was "open" or not:

<View visible="{open}" />

However, this doesn't seem to work. I tried using several varieties of ternary operators, like {open} ? true : false, but for some reason {open} doesn't evaluate to an integer that can be used in the operator.

I ended up having to include an additional model attribute and do a string compare on {open}:

/* Excerpt from model exports
...
config: {
    columns: {
    ...,
    "open": "integer",
    "openBool": "text"
},
...
*/

var foo = Alloy.createModel('foo', {
    open: open,
    openBool: open == '1' ? 'true' : 'false',
});

<View visible="{openBool}" />

But I don't want to have to create a supplementary attribute for each binary integer. Is there a way I can use the integer to evaluate to a boolean?
