  • I had this same problem. It seems the value can be only one string. If your minimum API is 9 you can use "sensorPortrait".

    Portrait orientation, but can be either normal or reverse portrait based on the device sensor. Added in API level 9.

    More info can be found here:

    http://developer.android.com/guide/topics/manifest/activity-element.html#screen
