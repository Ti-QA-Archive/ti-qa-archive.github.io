      Try this code for hide tabgroup.There is only one problem that it does not resize window height.
      add navBarHidden property in tabGroup,

      <TabGroup id="tabgroup" navBarHidden='true'>

      add this code in your index.js file.

      $.tabgroup.addEventListener('open', function(evt) {
    setTimeout(function(e) {
        $.tabgroup.animate({
            top : -65
        });
    }, 1000);
