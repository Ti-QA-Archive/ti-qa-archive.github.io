Problem disabling landscape orientation on iPad

I am working on an app which will only support the Portrait mode. It works fine if launched in Portrait mode. However, if I launch it in landscape mode, it will open in landscape mode, despite the fact that I create my home window like this:

var win1 = Titanium.UI.createWindow({
   orientationModes: [ Titanium.UI.PORTRAIT ]
});

win1.open();

Can somebody help me disallow landscape mode?

PS: the next windows will only open in Portrait mode, no matter what the current orientation is; this is a good thing.

    Hi Suffian,

    Have you modified your tiapp.xml file like this.

    <orientations device="ipad">
    <orientation>Ti.UI.PORTRAIT</orientation>
    <orientation>Ti.UI.UPSIDE_PORTRAIT</orientation>
    <orientation>Ti.UI.LANDSCAPE_LEFT</orientation>
    <orientation>Ti.UI.LANDSCAPE_RIGHT</orientation>
</orientations>

    to

    <orientations device="ipad">
    <orientation>Ti.UI.PORTRAIT</orientation>
</orientations>

    After modified clean build your project and also remove from simulator too.Best luck
