Considering Appcelerator - Browser Question

I understand that app is developed using HTML5 and JS (among others). The app that would be distributed - does it open in it's own "browser" or does it use safari or chrome?

If in Safari or Chrome, is there a way to open in full screen mode on iOS 7?

  • Hi

    Appcelerator Apps do NOT get created in HTML in anyway shape or form. Unless you are talking about web apps.

    Whilst Javascript is used as the development language that does not mean this is related to HTML development. An app designed for the Apple AppStore or Google Play Store are native apps that use native components to give users of these apps native experiences.

    You can choose to include web pages in your app, but they'd are used to show web content NOT building blocks for the app UI.

    If you are talking about a web app then that is compiled down into suitable HTML, CSS and JacScript from the original code using Titanium APIs. These apps are not provided inside a web browser either as that is not how web apps are distributed.

    All app created using Appcelerator Titanium can display in full screen mode if you choose on one or all windows, that is an app design choice. But there is no faking of web pages made to look like a real app - these are real apps.
