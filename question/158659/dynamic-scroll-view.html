  • First method to try: set the content height to auto after you add all the content. There's a good chance that won't work in all cases. So, the more complete technique would be to listen on the postlayout event (try on the scrollview, but it might work better on the window) and in that event's handler set the content height to auto. Make sure to remove the postlayout event listener within its handler function or you'll create an endless loop – postlayout fires every time the layout is rendered.

    If those fail to work, you could add a "Show More" button to the scroll view, position it from the bottom of the scroll view and at a higher zIndex than your other content. In its click handler, increase the content height of the scroll view by some amount (say 500). The user can then expand the scroll view's content as needed.
