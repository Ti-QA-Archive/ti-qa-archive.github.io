    • You have to define title at time of Making window object

      var win = Ti.UI.currentWindow({
title:'window2'
});

      You can get retrive title by 

      win = Titanium.UI.currentWindow;

win.getTitle();

      You can access all the property of window using

      win.property name ….

      So there is no need to get object actual name , only reference is required , which you in win as current window object as win in above code.
