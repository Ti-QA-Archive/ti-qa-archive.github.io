  • inside of the 1.7/android folder are the python scripts for the fastdev server. I believe you actually want to run the fastdev.pyc. To call it you need to pass it the directory of your project that you want served up not the mobilesdk directory. Something like this: "python fastdev.pyc start /path/to/my/cool/mobile/app/" This is the root of the entire project not the resources directory.
