  • Ok, run

    $ titanium build --platform android --build-only -f

    And in the proyect

    project
   + platform
          + Android
                + bin
                      + app-unsigned.apk ---> The app unsigned
                      + <name the app>.apk ---> The sign with titanium

    Run the signed with Jar

    $ jarsigner -verbose -sigalg SHA1withRSA -digestalg SHA1 -keystore release.keystore  app-unsigned.apk alias_name
$ zipalign -v 4 app-unsigned.apk <name the app>.apk
