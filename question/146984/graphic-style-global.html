graphic style global

Hello everyone! I am creating an app for iphone.
I created some windows with text.
is possible, with only one rule to a graphic style uniform to all pages?
for example, black background, white text, font arial etc …
or I have to declare the style of a window and cut and paste all of the others?

ps: I'm not using the web view
thanks :)

    You can use variables pulled from a common file:

    styles.js

    exports.window = {
    backgroundColor: '#ffffff',
    barColor: '#262626'
};

    In file:

    var styles = require('styles');
var myWindow = Ti.UI.createWindow(styles.window);
myWindow.title = 'Styled Window';

    • Building on Stephens example, if you have a few properties to over-ride from the defaults you can do this;

      var styles = require('styles');
var myWindow = Ti.UI.createWindow(styles.window);
myWindow.applyProperties({
    barColor: '#c60000', // over-ride
    title: 'Styled Window' // new property
});

      As long as you are using Ti SDK 3.0.0 or higher. This will perform all the operations in one go and be more familiar to you from the existing method.

      The only thing to keep in mind is there are SOME properties than can only be set at creation, examples for the window include;

      • exitOnClose
      • windowSoftInputMode
      • url
