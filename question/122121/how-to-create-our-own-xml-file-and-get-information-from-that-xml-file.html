  • To create XML file, you need to create file like any other. Only difference is "xml" extension (my.xml for example) and file content that should be valid XML string, at least XML declaration and one root node.

    To read XML from file, you need to read whole file like any other, and use Ti.XML.parseString to parse file content that you read. Then you can use DOM methods to manipulate XML data (read data from some node, for example).
