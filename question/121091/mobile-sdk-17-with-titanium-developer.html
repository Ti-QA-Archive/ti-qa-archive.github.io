  • Of course my friend!

    I'm currently using mobileSDK 1.8.0 with Titanium Developer 1.3.0

    .

    But the 1.8.0 mobileSDK is more usable then 1.7.x
    In 1.8.0 Ti.API.info, modules and other things run properly.

    .

    To get the mobileSDK 1.8.0:

    .

    -Go to Continuous Builds, select master branch and download it.
    -Put 1.8.0 folder in /Library/Application Support/Titanium/mobilesdk/osx/.

    .

    To get the Titanium Developer 1.3.0:

    .

    -Duplicate your Titanium Developer app, and rename it like "Titanium Developer 1.3.0".

    -Go to Github and download source from master branch, or directly from https://github.com/appcelerator/titanium_developer/zipball/master.

    (Trust me… the name of zip has 1_2_2, but this is a discontinued 1.3.0 version, if you see inside tiapp.xml, you will read: 1.3.0)

    -Open the duplicated App (folder) with "Show Package Content", open "Contents" folder, delete "Resources" folder and "tiapp.xml", save here the "Resources" folder and "tiapp.xml" from the source downloaded.

    .

    .

    Enjoy

    .

    PS.: I'm making a new version of "TiDev" with many things, such project folder files and Code editor.
