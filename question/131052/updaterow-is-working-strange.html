I used updateRow for updating row! after updating instead of this row display empty row ( displaying only hasChild but without any word )

if(CustomData[6].name == 'Starts'){

var time = p.value;

  • First of all, you don't need to set test to a new Object, you can just do:

    var test = {name:"Starts", status:time, hasChild:true};

    Secondly, updateRow should pass in a TableViewRow, not the data source. So do something like this:

    var newRow = Ti.UI.createTableViewRow({ height: 60 });
newRow.add(Ti.UI.createLabel({ text: "test", height: "auto" });
newRow.test = {name:"Starts", status:time, hasChild:true};;
TheTable.updateRow(e.index, newRow);
