    *I think your problem is a simple one of not enough properties and maybe you could do with a nested layout.
    Let's try the properties one first.

    Your inner view does not have a width set so it is likely defaulting to size of contents which would explain the location you are seeing.

    Try adding property width="Ti.UI.FILL" to the inner view and see what happens.*
