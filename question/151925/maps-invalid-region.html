  • This seems to be a timing bug still existing in the latest SDK: I solved it by implementing a timeout, as mention in JIRA: setTimeout(function(){
    $.companyMap.region = {latitude:jobLat, longitude:jobLng, latitudeDelta:0.71, longitudeDelta:0.71};
    $.companyMap.regionFit = true;
    }
    , 300);

    https://jira.appcelerator.org/browse/TIMOB-13102
