    • The JavaScript has basically been replaced by the REST section and it hasn't changed that much since then. It's just that the new term is more accurate since you can call REST services using any language.

      And since Appc acquired CocoaFish (the company that originally created ACS) it made much more sense to integrate this directly into the Titanium Framework.
