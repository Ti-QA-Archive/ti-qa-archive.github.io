  • You perform builds in Ti.Studio against a run/debug configuration. Any of these can target a specific sdk version. You can see the defaults for a new run/debug config by clicking

    Titanium Studio->Preferences->Studio->Platforms->Android

    and the default sdk version is there.

    You can see what has been set in an existing run/debug config by choosing

    Run->Run Configurations and select your app in list on left

    or do the same for

    Run->Debug Configurations
