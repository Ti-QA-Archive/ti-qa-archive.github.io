Divide android screen in horizontal space

How to divide android screen in horizontal 3 equally proportional size which can be supported by all android devices?

I need to design app which has 3 tabs like button and I want to make them of equal width then How to achieve that in android where width of device varies most of time.

NOTE: I don't want to use % of total width.

  • I'd calculate a third of the screen-width like this:

    var screenWidth = Ti.Platform.displayCaps.getPlatformWidth();
var columns = 3;
OS_ANDROID && (screenWidth /= Ti.Platform.displayCaps.logicalDensityFactor);
var tabWidth = Math.floor(screenWidth / columns);
