How would I edit my code (see below) to slide it all up so the box is visible whilst typing and then slides back to it's original position on keyboard hide?

Many thanks,

Simon

    change your mainView from a simple view to scrollView like this:

    var MainView = Titanium.UI.createScrollView({
   width:'//as per needed',
   height:'as per needed',
   contentHeight:'//as per needed',
   contentWidth:'//as per needed'
});

    scrollView scrolls when keyboard comes up.
