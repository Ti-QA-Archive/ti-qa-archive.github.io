Application Failed to Install Error

I'm trying to install an application to an Ipad2 but it keeps telling me "Application X Failed to Install" where X is the name of the application.

I've set the provisioning profile to run in all registered devices and I doubt it's the code as I ran it with the most basic code, and still couldn't install it. I've been stuck with this error for a couple of days, and have been unable to make progress. I'd appreciate any help that can be given

    Hi

    The first thing I would check is that the certificate and the app id match, so it you have the app id of com.somecompany.someapp that needs to be used as the app id in your tiapp.xml file and in your provisional profile certificate that you must have installed on your machine.

    The only exception to this would be if you created a testing provisioning profile with a wild card, for example; 

    com.somecompany.*

    This is only useful for testing and would not be allowed in an app store app of course.

    I would also check that you do not have MORE than one version of the certificate installed and your build process is using an old one that does not include your device information.

    • Type xcode in the spotlight search and then press enter.
    • Menu Window > Organizer
    • Locate your provisioning profile list
    • Delete ALL that match your app id
    • Re-download the provisioning profile from apple
    • Double-click it to have it appear in your provising profile list
    • Close the simulator if it running
    • In Titanium us the menu Project > Clean and then select your app and confirm clean
    • Re-build your app for your device, and you will be asked to choose your provisioning profile again as it is unlikely to know which to use.

    Does this work?

    If not, create a brand new app with a brand new app id, create a new provisioning profile. Then compile for device. If this works then it is your code, if it does not then it is how you are creating provisioning profiles, your installation and unknown.

    Let me know how you go.
