When I try to change the width of the labels to Ti.UI.FILL they move to different lines. I expected them to fill the space evenly. What is the best way to fit these four items in the TableViewRow so the image is on the left, the button is on the right and the two labels take up half of the remaining space.

  • Show us what you have so far. Without more detail it'll be tough. That said, here's a snippet of XML and TSS for organizing a "row" in a similar fashion to what you suggest (untested):

    snippet.xml

    <Alloy>
  <TableViewRow id="row">
    <ImageView id="image"/>
    <Label id="title">the title</Label>
    <Label id="subtitle">the subtitle</Label>
    <Button id="button>press me</Button>
  </TableViewRow>
</Alloy>

    snippet.tss

    'Label': {
  left: '70dp',
  right: '70dp',
  color: '#000'
}

'#row': {
  height: '60dp'
}

'#image': {
  left: '10dp',
  top: '5dp',
  bottom: '5dp',
  width: '50dp',
  image: '/path/to/image.png'
}

// You'll probably need to play with the title and subtitle settings to get it to look right
'#title': {
  top: '5dp',
  height: '30dp',
  font: {
    fontSize: '24dp',
    fontWeight: 'bold'
  }
}

'#subtitle': {
  top: '40dp',
  bottom: '5dp',
  font: {
    fontSize: '16dp',
    fontWeight: 'normal'
  }
}

'#button': {
  top: '5dp',
  bottom: '5dp',
  right: '60dp',
  width: '50dp'
}

    <Collection src="KKJWishlist">
    <Tab id="wl_tab">
        <Window id="winWishList" title="Wish List">
            <View class="container">
                <View id="viewButtons">
                    <Button id="btnSend" />
                </View>
                <TableView id="tblWishList" dataCollection="KKJWishlist">
                    <TableViewRow id="tblrowWishList" ItemID="{ItemID}">
                        <ImageView id="imgItem" image="{CacheThumbnail}" />
                        <Label id="lblItemName" text="{ItemName}" />
                        <Label id="lblItemPrice" text="{ItemPrice}"  />
                        <Button id="btnDelete" ItemID="{ItemID}" />
                    </TableViewRow>
                </TableView>
            </View>
        </Window>
    </Tab>
