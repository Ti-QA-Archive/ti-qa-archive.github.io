Trouble passing data between to webview

Hello,

I'm having trouble passing data to a webview, my JS file has the following code:

 var myWebview = Controls.createWebview({
        url:'html/data.html', 
        top: 110 
    });
    win.add(myWebview);

myWebview.addEventListener('load',function(e){        
        Ti.App.fireEvent('webPageReady', { 
            data: content        
        });
    });

The variable CONTENT does contain data, I have verified this. My webview calls data.html file which has the follow:

<!DOCTYPE html> 
<html> 
    <head> 
    <title></title> 

    <meta name="viewport" content="width=device-width, initial-scale=1"> 

    <link rel="stylesheet" href="css/jquery.mobile.min.css" />
    <script type="text/javascript" src="js/jquery.min.js"></script>
    <script type="text/javascript" src="js/jquery.mobile.min.js"></script>
</head> 
<script>    
    // Fire event to load data.
    Ti.App.addEventListener('webPageReady', function(e) {                                        
        alert(e.data);        
    });


</script>

<body> 
</body>
</html>

I have managed to pass 'Hello World' and display in an alert statement, so I know its being executed correctly. I'm just having a problem passing variable, below is an example of some of the content in the variable. The problem may just be with the way I am referencing the variable contents in data.html.

data = "[{\"idnoticia\":\"6900\",\"title\":\"Pastor rechaza la ley navarra que crea un registro de sanitarios objetores al aborto\",\"created\":\"Mon, 11 Apr 2011 06:00:00 +0200\",\"prioridad\":\"\"}

Regards,
S

  • Finally resolved it……I wasn't parsing the JSON as I read it from the file, in my JS file I change the read statement to:

    var content = JSON.parse(file.read());

    This allowed me to pass the variable 'content' to data.html and access the values I wanted correctly.
