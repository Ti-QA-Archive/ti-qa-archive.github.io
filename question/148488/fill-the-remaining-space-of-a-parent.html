Fill the remaining space of a parent

Hi there! I have a parent view (with horizontal layout) and 2 child views: one has specified with and I expect another one to fill the remaining space of it's parent. Is there an easy way to achieve that? If I try to set width: Ti.UI.FILL to the second view - it fills the whole size of a parent (but not the remaining, as I need).

It works well if I set specified width to both child views (second_view.width = screen.width - first_view.width), but I would like to avoid such workaround, if possible.

Thanks!

    • I just did a little test with the code below and width:Ti.UI.FILL for the second view works for me.
      (tested on iOS simulator)

      And yes, it should as defined per specification: Layout Limitations Spec and Layouts, Positioning, and the View Hierarchy

      var win = Ti.UI.createWindow({
    backgroundColor:'white'
});

var viewParent = Ti.UI.createView({
   backgroundColor:'yellow',
   layout:'horizontal' 
});

var view1 = Ti.UI.createView({
    backgroundColor:'blue',
    width:'20%'
});

var view2 = Ti.UI.createView({
    backgroundColor:'green',
    width:Ti.UI.FILL
});

viewParent.add(view1);
viewParent.add(view2);
win.add(viewParent);
win.open();

      <view layout="horz">

            <view>

            </view>

    <view size="30">

  • In case anyone stumbles upon this in the future, if the views are on top of each other, the first one's height should be set to Ti.UI.SIZE so the height will adapt to the content. The second can be set to Ti.UI.FILL to fill the remaining part of the screen.

    This question is for the views side by side. I haven't tested using the width instead of height, but it's probably worth a shot
