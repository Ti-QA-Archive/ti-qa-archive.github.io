  • To differentiate between the different iOS or Android versions you can try something like:

    if (Ti.Platform.osname == 'iphone') {
    if (Ti.Platform.version == '4.0') {
       alert('This is IOS4!');
    } else {
       alert('Please upgrade your device to iOS 4.0!);
    }
}

    And just change the else statement to an if else and check for iOS 3 or iOS 5. Here the is documentation for Ti.Platform.version. Switch out iphone with android to check android version OS.
