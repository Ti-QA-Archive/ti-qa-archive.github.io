Is there a GetProperty (or similar) command

Morning all,
Can you help with the following?
In certain languages/platforms you have a GetProperty command which you can use to find a property of an object so e.g. to find the X value (in pixels) of a button location, you could write getProperty(myButton, _x)

Does something similar exist in Appcelerator?

Reason being: Imagine that I have 2 buttons. Button 1 is placed randomly onto the stage (so I don't know it's X or Y position) and I would then like Button 2 to be placed on stage 20 pixels to the right of Button 1.

To do that I need a command to first return the X and Y value of Button 1 so e.g.
(Button2, _x) = getProperty(Button1,_x) + 20; << old skool actionscript ;-)

Any thoughts?
Thanks,
Lee

    • This only works if Button1.right has been set otherwise Button1.right is undefined.

      Here is a solution that calculates the position slightly differently:

      var window = Ti.UI.createWindow({
    backgroundColor: "#FFF"
});

var button1 = Ti.UI.createButton({
    left: 15,
    width: 100
});
window.add(button1);

var button2 = Ti.UI.createButton({
    left: button1.left + button1.width + 20,
    width: 80
});
window.add(button2);

window.open();

      A general rule of thumb is that you can't use an object's attribute if you didn't define it earlier. Often there are no default values you can rely on.
