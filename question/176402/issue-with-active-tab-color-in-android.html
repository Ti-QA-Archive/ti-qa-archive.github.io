    • Hi Wahib, you have already defined an onFocusListener on the tabGroup. I don't know how it looks like but you can change title and icon like this:

      function setTabNames(e){
    e. previousTab.setIcon( unselectedIcon );
    e. previousTab.setTitle( unselectedTitle );

    e. tab.setIcon( selectedIcon );
    e. tab.setTitle( selectedTitle );
}
