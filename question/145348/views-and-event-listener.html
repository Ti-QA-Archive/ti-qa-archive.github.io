Views and Event Listener

How do I add event listeners to my views that triggers my app to open another window?

    Hello Jay…

    Try like this…

    
var view = Ti.UI.createView({
    top : 10,
    left : 10,
    right : 10,
    bottom : 10,
    backgroundColor : 'red'
});

view.addEventListener('click',function(e){
    var viewWin = Ti.UI.createWindow({
        url : 'viewDetail.js',
        title : 'New Window',
    });
    Ti.UI.currentTab.open(viewWin,{animated:true})
})

win.add(view);
