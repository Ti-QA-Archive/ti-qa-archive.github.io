  • It sounds like you are creating your app with multiple contexts using the window's url property. Managing data across multiple contexts becomes more difficult because each context is a clean slate.

    You'll likely want to consider refactoring your app to use a single context. Check out the "Global Scope" and "Execution Contexts" sections of the following wiki guide: http://wiki.appcelerator.org/display/guides/Tabbed+Application+Template+%28Single+Context%29

    Also, for a simple example of using a single execution context in a Titanium app, check out this wiki guide, which includes a corresponding Forging Titnaium video: http://wiki.appcelerator.org/display/guides/Tabbed+Application+Template+%28Single+Context%29
