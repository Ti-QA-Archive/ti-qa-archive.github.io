  • Hi,

    You can refer http://docs.appcelerator.com/titanium/3.0/#!/api/Titanium.Platform-property-model

    This basically is an identifier of the hardware model of the device.

    There are other API to find wether device is iPad or iPhone :

    
if (Ti.Platform.osname == 'iPad'){
// if iPad controls comes here
}

if (Ti.Platform.osname == 'iphone') {
        //if iPhone control comes here
    }
