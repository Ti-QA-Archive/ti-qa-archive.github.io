message is showing.

"The current timeout is set to 120000 ms
[ERROR] : Emulator failed to start in a timely manner

Android SDK is installed in my Titanium Studio. My operating system is Windows 7. Now What can i do.Please help me….

  • This is because your emulator timeout is 2 sec. Try to increase it by using below command,

    titanium config android.emulatorStartTimeout 0 // 0 means emulator will wait forever. It won't cause timeout.
