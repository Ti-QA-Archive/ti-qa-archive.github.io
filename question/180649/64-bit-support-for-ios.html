To enable 64-bit in your project, we recommend using the default Xcode build setting of “Standard architectures” to build a single binary with both 32-bit and 64-bit code. After you’ve corrected the issues, you can use Xcode or Application Loader to upload a new binary to iTunes Connect. Regards, The App Store team.
