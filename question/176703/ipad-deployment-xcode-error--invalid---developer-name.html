  • [ERROR] :  Invalid "--developer-name" value "iPhone Distribution: THE NAME OF MY APP"

    From ti help build:

    -R, --distribution-name <name>   the iOS Distribution Certificate to use; required when target is dist-appstore or dist-adhoc

    -V, --developer-name <name>      the iOS Developer Certificate to use; required when target is device

    You said its trying to run iPhone Distribution: which should only be used with --distribution-name yet is being passed to the cli with --developer-name. Check the build settings in your setup.
