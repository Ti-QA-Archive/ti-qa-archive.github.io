Possible to remove an SDK version?

    Yeah, you just need to delete the SDK folder. I'm not sure what OS you are on, but for mac you would just delete ~&#x2F;Library&#x2F;Application Support&#x2F;Titanium&#x2F;mobilesdk&#x2F;osx&#x2F;SDK_YOU_WANT_TO_DELETE
