Ti.UI.SIZE is not working for View's width

I have a Table with a View inside a TableRow. This View contains a Label and Images. When I set
View.width = Ti.UI.SIZE
expecting it should shrink to Label+Image width - it doesn't work and View's width takes the whole width of the screen.

Here is code example

var tablerow = Ti.UI.createTableViewRow({
    className: 'content',
    height: 'auto',
    layout: 'absolute'
});

var label_view = Ti.UI.createView({
    backgroundColor: "red",
    width: Ti.UI.SIZE,
    height: Ti.UI.SIZE,
    layout: 'horizontal',
    right: "50dp",
    top="15dp"
});

var label = Ti.UI.createLabel({
    backgroundColor: "blue",
    text: "Hello World!"
});

label_view.add(label);

tablerow.add(label_view);

Here is what I get:

as you see, View's width wasn't adjusted to Label's width. Am I understand the concept of Ti.UI.SIZE wrong? What should I do to adjust View's width to Label's width automatically?

  • Accepted Answer

    There is a difference between auto and Ti.UI.SIZE. auto means Ti.UI.SIZE on some controls, but it means Ti.UI.FILL on other controls, including TableView. Ti.UI.SIZE will give you the behavior you want. auto has been deprecated as well, so you should use Ti.UI.SIZE anyways.

    (Side note: auto always defaults to Ti.UI.SIZE on Mobile Web).
