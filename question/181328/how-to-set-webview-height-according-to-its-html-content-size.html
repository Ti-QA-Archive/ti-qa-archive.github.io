  • You're thinking about it the wrong way around. You dont size the webview to the content height of the document (what would happen if it was 100,000 lines long!??). You need to ensure the web view content displays at its maximum size and then pinch/zoom to view sections of it. This is controlled by the Viewport. Read more here for iOS and here for android
