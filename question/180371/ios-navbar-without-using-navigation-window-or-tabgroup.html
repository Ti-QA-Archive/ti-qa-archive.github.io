IOS NavBar Without Using Navigation Window or TabGroup

Hi,

Is there a way to show the navBar on IOS without using a Navigation Window or TabGroup ? I just want to have windows that show the navBar.

Thanks,
Victor

  • Yes create a customNav using view

    var win = Ti.UI.createWindow({
    backgroundColor : 'white',
});
var navView = Titanium.UI.createView({
    top : 0,
    backgroundColor : 'red',
    width : Ti.UI.FILL,
    height : 50
});
win.add(navView);
win.open();
