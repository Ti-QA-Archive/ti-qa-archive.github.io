Unable to animate view with height 0

  • Application Type: mobile
  • Titanium SDK: 3.1.0GA
  • Platform: iOS 6.1
  • Device: iOS Simulator
  • Host OS: Max OSX 10.8.3
  • Titanium Studio: 3.1.1.201304291957

Our app contains a view whose display will be toggled by the user. When toggled visible we need the view to slide down. When closed it should slide up. In order to achieve this I'm animating the height to 0 and 210 (full-size). I have the height property of the view initially set to 0 in the TSS file. In this scenario Titanium will not display the view and the animation never executes. If I set the view's initial height to 1 in the TSS file everything works just fine. Why is this, and how do I work around it?

view.tss:

"#index": {
    top: 83,
    width: 1024,
    height: 0,
    backgroundImage: "image_gallery_background.png",
    zIndex: 9999
}

view.js:

var args = arguments[0] || {};
var open = false;

exports.toggle = function() {

    var newOpen;

    var animation = Titanium.UI.createAnimation();

    if(open) {
        animation.height = 0;
        newOpen = false;
    } else {
        animation.height = 210;
        newOpen = true;
    }

    animation.duration = 150;
    animation.setCurve(Ti.UI.ANIMATION_CURVE_EASE_IN_OUT);

    $.index.animate(animation, function() {
        open = newOpen;
    });

};
