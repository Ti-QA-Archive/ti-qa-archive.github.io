  • I had a similar problem – no blinking underscore, but I got the exception during an Android build and the "error device not found". This was the first time I had built this particular project for Android (although I created the project months ago).

    It turns out that I was using a CI build of Titanium SDK 2.0.0. When I changed the SDK version to 2.0.1.GA2 (edit tiapp.xml), it built and installed fine on my android device.
