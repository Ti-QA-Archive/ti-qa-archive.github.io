Is it possible to export activites in Titanium apps so they can be launched by other applications?

Using java, it is possible to export an activity so that other applications can launch it using an intent.

I have successfully written an activity in java, exported it in AndroidManifest.xml, and then created an intent to launch it from a Titanium application. However, I have been unsuccessful at writing an activity in the Titanium application that can be launched from another application (either another Titanium application or a native java appliction).

Here is an example of my activity written in java:

import android.app.Activity;
import android.os.Bundle;
import android.widget.TextView;

public class ActivityA3 extends Activity {
    /** Called when the activity is first created. */
    @Override
    public void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.alt);
        ((TextView) findViewById(R.id.text)).setText("Welcome to Activity A3!");
    }
}

Here is the corresponding AndroidManifest.xml (snippet) to export the activity:

<activity android:name=".ActivityA3"
    android:label="@string/app_name">
<intent-filter>
  <action android:name="android.intent.action.VIEW" />
  <category android:name="android.intent.category.DEFAULT" />
  <data android:scheme="file" android:host="*" android:pathPattern=".*\\.txt" />
</intent-filter>
</activity>

Thanks in advance.
