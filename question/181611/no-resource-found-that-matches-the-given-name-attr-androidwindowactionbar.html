  • I was able to get it to run by adding 

    <uses-sdk android:minSdkVersion="10" android:targetSdkVersion="21"/>

    into the tiapp.xml file inside 

    <android xmlns:android="http://schemas.android.com/apk/res/android">
    <manifest>
    </manifest>
</android>

    Also, I found through the Titianium CLI that it wasn't able to locate the path for my ndk due to it looking for RELEASE.txt but the release file was named RELEASE.TXT, so I renamed it and set the ndk path.
