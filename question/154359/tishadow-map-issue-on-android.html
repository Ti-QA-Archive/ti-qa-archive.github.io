TiShadow map issue on android

Hi, I suppose this issue is not related directly at titanium SDK, I'm under TiShadow and I have a simple view with a map in. I can not show it under Android, to be more accurate I get this error

Could not find class 'ti.modules.titanium.map.TiMapActivity', referenced from method ti.modules.titanium.map.ViewProxy.createView

but if I install the app manually on the device map is correctly shown, I have this issue just with TiShadow.
Any hint?
Under the xml's view.

<Alloy>
    <View>
        <View id="mapview" ns="Ti.Map"
            animate="true" regionFit="true" userLocation="true"
            mapType="Ti.Map.STANDARD_TYPE" >

            <Annotation id="mountainView"
              latitude="37.390749" longitude="-122.081651"
              title="Appcelerator Headquarters" subtitle="Mountain View, CA"
              pincolor="Titanium.Map.ANNOTATION_RED"
              leftButton="/images/appcelerator_small.png"
              myid="1"/>

        </View>
    </View>
</Alloy>

    I don't know if this is the best solution, but I just found a way to get this work. If you modify the tiapp.xml of the tishadow app by inserting this

    <application android:debuggable="false" android:icon="@drawable/appicon" android:label="TiShadow" android:name="TishadowApplication">
    <activity android:configChanges="keyboardHidden|orientation" android:launchMode="singleTask" android:name="ti.modules.titanium.map.TiMapActivity"/>
    <uses-library android:name="com.google.android.maps"/>
</application>

    maps will work on android.

    I think when titanium generates the android manifest, it only includes this if the app is using maps, but tishadow does not by default. But hopefully this quick workaround helps you to go on
