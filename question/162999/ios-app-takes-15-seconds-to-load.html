iOS app takes 15 seconds to load?

Hi everyone. I've recently updated some apps to iOS7. The first screen in the app is 4 scrollViews with images of products within each scrollView. In iOS5 and 6, I would open the app, and everything would work instantly. Since updating to iOS7, on the first open, or any time you open it after fully quitting, it takes around 15 seconds to be able to click anything. You can still scroll left and right through the images, but the arrows I have to indicate whether you can scroll left and/or right aren't active until that 15 seconds is up. Is this an iOS7 thing?

I'm using 3.2.1 GA. Here's a sample of the code.

var win = Titanium.UI.currentWindow;
win.backgroundColor = '#ffffff';

win.orientationModes = [Ti.UI.PORTRAIT];


var 1LeftArrow = Ti.UI.createView({
    backgroundImage:'../images/leftarrow.png',
    height:70,
    width:35,
    top:225,
    left:5,
    visible:false
});
win.add(1LeftArrow);

var 1RightArrow = Ti.UI.createView({
    backgroundImage:'../images/rightarrow.png',
    height:70,
    width:35,
    top:225,
    right:5
});
win.add(1RightArrow);


var scrollView = Titanium.UI.createScrollView({
    //contentWidth:1610,
    //contentHeight:240,
    contentWidth:'auto',
    contentHeight:'auto',
    top:170,
    height:158,
    width:690,
    //borderRadius:10,
    backgroundColor:'#ffffff'

});

scrollView.addEventListener('scroll', function(e)
{
    Ti.API.info('x ' + e.x + ' y ' + e.y);

    if (e.x > 50)
    {
        1LeftArrow.show();
    }
    else
    {
        1LeftArrow.hide();
    }
    if (e.x < 750)
    {
        1RightArrow.show();
    }
    else
    {
        1RightArrow.hide();
    }

});


win.add(scrollView);


var view1 = Ti.UI.createButton({
    backgroundImage:'../images/image1.jpg',
    width:211,
    height:158,
    left:10,
});
scrollView.add(view1);



view1.addEventListener('click',function(e)
{
        var win = Titanium.UI.createWindow({
            url:'../products/product1.js',
            title:'Product1',
        });
        Titanium.UI.currentTab.open(win,{animated:true});

     });

Any ideas?
