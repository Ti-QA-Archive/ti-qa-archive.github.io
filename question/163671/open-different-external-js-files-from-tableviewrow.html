    Hello Neville Chimuka,

    Your data variable is global. so you can simply use it in the row like below :

    try this :

    row.addEventListener("click", function(e) {
    //2. Add a listener to the table and use the e.source property to figure out what event fired the event

    var w = Ti.UI.createWindow({// (MAINLY THIS AREA)
        title : data[e.index].title,
        url : data[e.index].js,
    });

    tab.open(w);
});

    i hope this helps. thanks.
