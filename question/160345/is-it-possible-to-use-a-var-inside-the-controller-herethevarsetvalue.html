Is it possible to use a var inside the controller ($.hereTheVar.setValue())?

Hi all,

I am just starting with Titanium and so far so good, but I am running into a problem which I can't figure out and which I cannot find on here or on the web.

I am developing a mobile app (using alloy) for Iphone on OS X 10.9 running Titanium SDK build: 3.1.4.201310171935.

So let's say I want to change the value of the TextField with id="thisID", I would use the following js right?:

$.thisID.setValue('newValue');

This is working fine but now i have a var with an ID in it. So for example 

var currentID = 'thisID';

This var is holding the ID of the TextField I want to change, but the id is dynamic so it will change in some cases. Is it possible to use this var inside the $.HEREVAR.setValue(&#39;newValue&#39;); ??

I am trying to figure this out but so far no luck.

I hope my question is clear.

Thank you in advance!
Tim

    $[TextField id].setValue('newValue');
