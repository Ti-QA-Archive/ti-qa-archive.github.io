Android Emulator - Application Installer abnormal process termination. Process exit value was 1

Hi,

I'm new to Titanium and i'm trying to run the Android Emulator, on Titanium build 3.2.1.
I have Android plataforms 10 and 14 to 19 installed aswell as the SDK Tools rev 22.6.2 and Sdk Plataform tools 19.0.1.

I've changed tiapp.xml to target a specific version:

        <android xmlns:android="http://schemas.android.com/apk/res/android">
              <manifest>
                <uses-sdk android:minSdkVersion="10" android:targetSdkVersion="10" 
android:maxSdkVersion="19"/>
            </manifest> 
        </android>

And the console log, on trace level:

`[INFO] : Alloy compiler completed successfully
[INFO] : Launching emulator: Galaxy Nexus
[ERROR] Application Installer abnormal process termination. Process exit value was 1
[INFO] : Running: C:\android-sdk-win\tools\emulator.exe "-avd" "Galaxy Nexus" "-port" "5554" "-no-boot-anim" "-partition-size" "128"

Whatelse do i need to configure?

Thanks in advance
`

    Hi,
    Try creating an avd through ave manager first (whatever setting/device you like) for example galaxy nexus and then after it gets created try installing the app in this newly created emulator.

    For creating AVD you need to open the avd manager by opening the file android kept in /android-sdk/tools/android

    then click New

    and create your device

    After it gets created it will automatically get listed on your device list in titanium studio.

    Now install the app in it.

    Hope it works.
