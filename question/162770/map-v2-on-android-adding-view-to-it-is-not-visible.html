Map V2 on Android, adding view to it is not visible.

Hi,
I'm porting my app to Map V2 on Android.
I have a problem on IOS I can add controls to the Map View, while on Android they don't appear.

I use this code

    var testBtn = Titanium.UI.createButton({
            left : "136dp",
            top : "136dp",
            width : "32dp",
            height : "32dp",
            title: "test"
        });

    mapView.add(testBtn);

Any idea ?

  • Try this:

    var container: Ti.UI.createView();
