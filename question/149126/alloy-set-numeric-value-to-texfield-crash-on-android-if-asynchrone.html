Alloy. Set numeric value to Texfield crash on Android if asynchrone.

This little code crash on Android :

var myTf = Ti.UI.createTextField({
    value:5
});

setTimeout(function() {
    $.index.add(myTf);
    myTf.setValue(56);
}, 2500)

$.index.open();

This one works fine :

var myTf = Ti.UI.createTextField({
    value:5
});

setTimeout(function() {
    myTf.setValue(56);
    $.index.add(myTf);
}, 2500)

$.index.open();

myTf.setValue("56") works

All my TextFields are set in the XML markup and filled in my controller asynchronously in my project.
I gonna burn my pc, please let me know if the is a work around…

Ti SDK : 3.0.2.GA
Android : 4.4.2 And 2.3.6
Titanium CLI Version: 3.0.24
Alloy Version: 1.0.0
