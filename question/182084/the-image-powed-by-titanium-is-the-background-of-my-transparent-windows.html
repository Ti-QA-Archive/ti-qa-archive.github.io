  • Hello Wafae,

    You can try this code, by setting Titanium.UI.setBackgroundColor and using opacity in window.

    Titanium.UI.setBackgroundColor('#BF0A13');

var win1 = Titanium.UI.createWindow({  
    title:'Tab 1',
    backgroundColor:'#fff',
    opacity : 0.5
});


var label1 = Titanium.UI.createLabel({
    color:'#999',
    text:'I am Window 1',
    font:{fontSize:20,fontFamily:'Helvetica Neue'},
    textAlign:'center',
    width:'auto'
});

win1.add(label1);

win1.open();

    Hope this helps, Thanks.
