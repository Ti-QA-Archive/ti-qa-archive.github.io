How to get data back from window upon close

Hi all,
I use 

Var window = Ti.UI.createWindow({
       data = dataToSend   //  data from current window
)}
window.open()

to send data from Current window to the new window, so that i can perform some computation on that data in the new window. How can I get the some data back from the opened window to previous window upon its close.

window.close()

I am a native android developer, and I am trying to get a similar functionality as 

startActivityForResult()

in titanium. Currently I am using app level event listener to fire an event from the window that is opened along with the data and catch it in the previous window, but I read that the app level listeners scope is through out the app and are costly, so please provide a work around to this.

  • Hi Rajesh, though you read correctly and app level events are to be used with care, but that does not mean to avoid them where they are necessary, you can use app level event and do not forget to remove after your app level event work is done, else that will be a memory leak.
