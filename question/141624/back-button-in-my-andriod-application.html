  • It is my understanding that you want to create an extra back button in your application. (since you have to support the Android device's back button anyway)

    You can first implement the physical back button:

    win.addEventListener('android:back', function(e) {
    // Back Logic Here...
});

    And fire this same android:back event from your custom button:

    backbutton.addEventListener('click'. function(e) {
    win.fireEvent('android:back');
});

    This will avoid code duplication.
