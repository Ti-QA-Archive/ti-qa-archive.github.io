      And also if you want navbar, then you can override androidback event as like this,

      index.xml

      <Alloy>
    <Window navBarHidden="true" exitOnClose="true">
        <Label onClick="test">Test</Label>
    </Window>
</Alloy>

      index.js

      function test(){
    var win2 = Alloy.createController('win2').getView();
