  • Hi li,

    If you are using Beta 3.4.0 then you must update Studio, SDK and CLI.

    And you need Xcode 6 on OS X Mavericks or Yosemite to build for iOS 8.

    Please read Titanium SDK 3.4.0 Beta for iOS 8 Now Available
