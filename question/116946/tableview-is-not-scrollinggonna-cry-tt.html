  • You forgot to create the tableview row. There are some other things that should be done differently, but I've included an example that you can try out yourself.

    Hope this helps!

    //create tableview data object
var data = [];

for (var c = 0;c < 52;c += 1) 
{
    //create the table view row
    var row = Ti.UI.createTableViewRow({height: 60});

    //create a label for the title text
    var title = Ti.UI.createLabel({
        text: 'Row ' + (c+1),
    });
    //add the title to each tableview row
    row.add(title);

    data[c] = row;
}

//create table view after the for statement
//add all of the rows of data to the tableview
var tableview = Titanium.UI.createTableView({
    data: data
});

//add the tableview to the current window
Titanium.UI.currentWindow.add(tableview);

  • you are doing this:

    win.add(listechants);
var view = Ti.UI.createView({background:"#000"}); 
win.add(view);

    without some dimensions, the views will take the max amount the have ( full window ) and in your case you have the view above the table. So your finger does not "reach" the table, but the view.
