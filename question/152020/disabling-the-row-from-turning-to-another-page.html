    Hi

    Try changing this line;

    if(e.rowData.hasChild){

    To this;

    if(typeof (e.rowData.hasChild) !== 'undefined' && e.rowData.hasChild === true){

    The first one return true if the property hasChild exists, which it can be even if the value is not true for the property.

    The second one makes sure the property exists and is set to true.
