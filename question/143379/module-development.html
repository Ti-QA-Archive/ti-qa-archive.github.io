    Hi Shobit,

    Though this is a very trivial question and you should do some homework before posting such a question. So, here are few pointers which you should consider:

    First, go and study Android Module development document from link:

    http://docs.appcelerator.com/titanium/2.1/index.html#!/guide/Android_Module_Development_Guide

    After you are through with development document and is familiar with API lingo, go and check some sample modules:

    https://github.com/appcelerator/titanium_modules

    If you still face issues, then community is happy to help you.
