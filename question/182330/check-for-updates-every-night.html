Check for updates every night

I am working on an app, that needs to run some code every night. It basically calls an api and checks for updates, and acts accordingly on the response.

I have no issue writing the code, and it works perfectly when called manually with the app in the foreground, but I need the app to do this automatically every night.

Can someone please point me in the right direction? Background services only run for a few minutes on iOS, so I can not rely on those.

  • Its very simple, and even it doesn't require the firing of background service, firstly what you need to do is to store the last updated date and time whenever updation is taking place, secondly on the application resume event just check when application was lat updated, and if its haven't updated for that day call the update service or right the application update logic in the resume event. As application resume event is fires when either application is launched from background or if application is opened for the first time,

    So all your conditions will get handle, if you implement in this way. Hope this will give you some approach to resolve your problem.
