  • Hello R Singar,

    It's work for me. I think keyboard is not set up for this simulator. When the simulator will open just go to the Hardware menu then keyboard and click Toggle Software Keyboard like Hardware > Keyboard > Toggle Software Keyboard. Hope it will help.

    If you have any concern then let me know.

    Thanks.

    • If "Toggle Software Keyboard" is disabled at first enable it. Just select "connect hardware keyboard."
      Hardware > Keyboard > connect hardware keyboard.

      Thanks.

      — commented February 16th 2015 by Motiur Rahman

    • Bingo! That was it. Phew.. It didn't occur to me to check the hardware menu. The 'toggle software keyboard' setting didn't stick (i.e. it lost the setting when simulator restarts). So I unchecked 'connect hardware keyboard' so that the soft keyboard shows up all the time.

      Not sure why soft keyboard was off for me (and that too just for 5s and 4s). Anyway, all is good now. Thanks much for your help.
