  • I was able to come up with a solution but its not very pratical. I'm still hoping someone come up with something better.

    I basically bind the field to an event and then use this code to get its value

    <TextField id="emailField" hintText="Email" onChanged="updateEmail"></TextField>

    var newEmail = '';
function updateEmail(e)
{
    newEmail = e.source.value;
}

    then when I need the value I just check the newEmail variable.
