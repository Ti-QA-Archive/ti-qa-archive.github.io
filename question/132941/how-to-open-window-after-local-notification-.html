How to open window after local notification ?

I'm working on ios application that have local notification, If the user click on the notification the application is lunch and open specific page depending on the notification massage (userInfo).
bg.js codes:

var notification = Ti.App.iOS.scheduleLocalNotification({
    alertBody:"notification",
    alertAction:"Re-Launch!",
    userInfo:{"path":"files/fav.js"},
    sound:"default",
    date:new Date(new Date().getTime() + 9000)
});

Ti.App.iOS.addEventListener('notification',function(e)
    {

        var win = Ti.UI.createWindow({
            url:e.userInfo.path,
            title:'quote',
            barColor:'#444',
        });

        win.open();
    });

When the application lunch from notification, The window (win) open and close directly.
What is the problem ?
