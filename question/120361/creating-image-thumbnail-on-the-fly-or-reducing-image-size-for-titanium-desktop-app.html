Creating image thumbnail on the fly, or reducing image size for Titanium Desktop App

Hi,

I am trying to create a desktop app to look into local image files inside a directory, display it, and allow the user to rename the images. I wanted to create an image gallery like interface.

My problem is, if I try loading image using just the path to the image like file:&#x2F;&#x2F;&#x2F;location&#x2F;image.jpg it will cause the problem to run very slowly as all the images are high-res camera images around 4-5MB each.

Is there a way or a trick to reduce or create a thumbnail version of the images, so I can just display the thumbnail in my application, so it will run faster?

  • Hi Harris,

    you can try to use the command mogrify from ImageMagick.

    mogrify -geometry 80x80! *

    Hope it helps.
