tabGroup and Android

Hi, that's not a question. I want to share with you 2 non documented function i found in the SDK.

  • how to hide the title bar in Android

    var tabGroup = Titanium.UI.createTabGroup({
  navBarHidden : true
});

  • how to prevent orientation changes in app using a tabGroup in Android:

    var tabGroup = Titanium.UI.createTabGroup();
tabGroup.setOrientationModes([Titanium.UI.PORTRAIT]);

I'm usinng 3.1.3 GA SDK and in the official documentation those two features are not described.

I only want to be helpfull for you all guys, and hope titanium developers let enter in the official documentation what i found.
