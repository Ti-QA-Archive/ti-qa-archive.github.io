SQLCipher Module (dMarie) Available?

I noticed that the dMarie module implementing SQLCipher for the Titanium platform is no longer available. Are there other options for full SQLite encryption, or do I need to roll my own?

    Hi Brian,

    The SQLCipher (by dMarie) module was acquired by Appcelerator a few months ago. It is now known as "Encrypted Database Module" and available through Appcelerator Platform Subscription. As far as I know, there aren't any other SQL encryption module available through Appcelerator's Marketplace
