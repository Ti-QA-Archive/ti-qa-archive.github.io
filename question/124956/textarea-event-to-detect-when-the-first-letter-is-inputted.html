textarea event to detect when the first letter is inputted?

Is it possible to setup an eventlistener to know when the very first letter is inputted into a textarea?

  • Accepted Answer

    Add a 'change' event listener to the text area, then check the length to see if it is one character. Be aware this can occur more than once if they backspace then start again. If you don't want that, initialize a variable to false when you open the window, then set to true in the 'change' event. Only perform the action when the event listener starts with that variable equal to false.
