    I have run into situations on android where accessor functions don't work, and I've had to resort to using the property (example: TextField.getValue() sometimes crashes and burns on me).

    Try accessing the property directly. So instead of

    var latitude = foo.getLatitude();

    do this:

    var latitude = foo.latitude;

    Maybe this will work?
