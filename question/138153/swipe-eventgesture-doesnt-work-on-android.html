Swipe event/gesture doesn't work on android

Hi,

with this simple snippet I want to build a day switcher:

container.addEventListener('swipe', function(_e) {
    var diff = (_e.direction == 'left') ? 1 : -1;
    var datetext = date.add('days', diff).format('DD.MM.YYYY');
/* Changing of calendar text:  */
    dateLabel.text = datetext;
});

This is the output of console:

D/View    ( 1009): onTouchEvent: viewFlags: 0x18204080
D/View    ( 1009): onTouchEvent: isFocusable: false, isFocusableInTouchMode: false, isFocused: false; focusTaken: false

On iPhone it works fine …

Any ideas?

Rainer

  • This code works, but not so perfect as swipe event on iPhone. Time to time the calendur jumps more then one day ;-((

    var x_start;
container.addEventListener('touchstart', function(e) {
        x_start = e.x;
});
container.addEventListener('touchend', function(e) {
        if (e.x > x_start) {
            container.fireEvent('swipe', {
                direction : 'right'
            });
        } else {
            container.fireEvent('swipe', {
                direction : 'left'
            });
        }
});
