  • Rainer,

    You can use the HLS tools from Apple to create an appropriate stream.

    They are only available to developers.
    If you find that you cannot download them, its because some goofball (see below) associated them with IOS 5 beta and when IOS 5 left beta, they changed the perms for the download.

    Call the !-800 for apple dev services, and ask for permission.

    Also, the tools will not run on Lion. Will run on Snow Leopard Server or Client (allows you to virtualize a session on Lion, check online for low cost Snow Leopard Server licences)

    The tools are apparently under the purview of a single grumpy dev at Apple who does not play well with others.

    We use these tools under a Snow Leopard Server virtual machine on Lion to chop up our audio. Only used the tools for video as a proof-of-concept for a client.

    BTW: The only problem I have EVER had with Apple as a developer has been in my interaction with the guy who handles these tools.

  • Hallo Rainer,

    basically, that is not possible. HTTP Live Streaming is not a file format, it is a server architecture and protocol. There are several streaming servers, that have this protocol implemented. Examples are the Adobe FMS, Wowza or Darwin/Apple Server.

    You'll finde more information (in German) on that topic in my Bachelors thesis.
