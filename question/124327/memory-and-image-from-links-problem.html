  • Accepted Answer

    Its hard to say exactly, as the question is very broad however let me see if these details can help you:

    • There are many very large, and very well known apps that successfully run with Titanium Mobile. NBC’s flagship iPad app and the Late Night with Jimmy Fallon apps are all built on Titanium. Wunderlist, a top todo list app for Android, iOS, and desktop is built with Titanium.

    • There are, things you can do to free up memory – the big ones in the 1.x product are closing windows (which releases all UI resources associated with the window) and setting references to a proxy object (like one returned by Ti.UI.createXXX) to null, which will release the resources associated with that object. Also the 1.7.x release fixed a number of memory management issues.

    • Titanium is not idiot proof, if a developer builds an app poorly, it will perform poorly. Also we know there are bugs in the platform, just like any other platform. If you find a bug, we have a JIRA bug tracker here that if a bug is submitted to we will do our best to fix it as fast as we can.

    With all that said, many developers have successfully used Titanium unmodified to deliver applications. Many more have extended Titanium to fit their needs. Titanium can and does work for many developers, and I hope you will count yourself among them soon.
