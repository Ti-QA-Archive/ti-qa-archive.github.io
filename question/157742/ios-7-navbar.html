  • If you are trying to change the title-text color (and the status bar foreground color), then set the statusBarStyle property. The navTintColor property controls the buttons but not the title text.

    var win = Ti.UI.createWindow({
    translucent: false,
    barImage: 'none',
    title: 'Title Text',
    barColor: '#08f',
    navTintColor: 'white',
    statusBarStyle: Titanium.UI.iPhone.StatusBar.OPAQUE_BLACK
});

    The above code should set the title text and navbar button titles to white.
