Issue in Viewing A Controller

Hi,

Am a beginner here so an elementary issue.

I'm facing an issue and have a question:

In my index.xml I have the following label:

<Label id="label" onClick="doClick">Hello, World</Label>

Then my index.js has:

function doClick() {
    var secondView = Alloy.createController("secondPage").getView();
    secondView.open();
}

However on clicking the label i get an error "undefined is not a function (evaluating 'secondView.open()')

Any ideas why?

Also the second question - is the best way to move through screens or is there a concept of pushing and popping views?

Thanks.

  • Accepted Answer

    secondPage.xml should look like this:

    <Alloy>
    <Window>
        <View class="container">
            <Label id="label">SECOND WINDOW</Label>
        </View>
    </Window>
</Alloy>

    That will allow you to call secondView.open();

    • As a follow up, if you want to pass any data through to your controller then you would pass a second parameter:

      var secondView = Alloy.createController("secondPage", data).getView();

      You would then use 'args' inside of your controller to retrieve the data you passed.

      You could also tidy up your XML by using the text property in Label

      <Label id="label" text="Second Window"/>

    • Do you have any stylesheets in your project? Try setting 'backgroundColor' for each window, it might be a transparency issue?
