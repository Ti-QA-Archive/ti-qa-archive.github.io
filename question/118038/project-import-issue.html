Project Import issue

I came across an issue where if you make an Android
project on a PC and then do an "import project" on a
Mac it doesn't show "iPhone" options.

Normally there are two tabs for "iPhone" and "Android"
on the Mac when making a mobile project.

We had to recreate the project to get "iPhone" to show up again.

— asked April 1st 2011 by Chris Flowers

  • In the future, I think you can get it to show by creating an empty "iphone" folder under (project root)/build/iphone, then switch away from the project in titanium developer and switch back again.
