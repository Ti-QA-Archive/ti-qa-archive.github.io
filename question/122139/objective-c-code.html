Objective C code

Titanium Mobile creates Objective C code out of your app. How does that work.
Does it only compile the Javascript Code into Objective C code or does it also compile the CSS and HTML files into Objective C

  • HTML and CSS are handled differently between mobile and desktop however they aren't compiled as ObjC code. However Titanium is a bridge that translates your Titanium javascript objects into their ObjC equivalents. Check the Titanium Architecture page on the wiki for a better explanation: http://wiki.appcelerator.org/display/guides/The+Titanium+Architecture

  • To get into more details:

    • Titanium maps JavaScript calls at the lowest possible level for each supported OS (Rhino for Android, JavaScriptCore for iOS, etc.);
    • Doing so allows Titanium to get rid off webviews and WebKit, which are both time- and resources-consuming;
    • The calls are then embedded into Objective-C, Java, etc., but it's JavaScript code that is actually being executed.
