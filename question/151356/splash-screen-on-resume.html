Splash screen on resume

Hi. I am developing app on iOS. I have the following problem: I need show splash screen and login screen every times when user press home button and he want start app again. When user press home button, app still running in background. I saw the following link http://developer.appcelerator.com/question/40081/ios4—make-app-not-run-in-background, but there is a problem with lock screen. When I unlock iphone, app still running and buttons not working. What I can do ? Thanks

— asked April 22nd 2013 by Andrej Lukes

  • I don't believe you can force the splash screen to show, but you could setup event handlers for Ti.App.resume or Ti.App.resumed events and display a view with your login screen. This lets you detect each time the app is closed and opened.

  • You could achieve this using the resume event.

    And every time the application is resumed, then you could show the login screen, thus forcing your users to authenticate themselves again.
