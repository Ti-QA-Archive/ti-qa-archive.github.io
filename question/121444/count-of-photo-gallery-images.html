Count of photo gallery images?

I'm trying trying to select a random collection of images from the iOS/Android photo galleries to import into my application. To do so, I need two things that I can't seem to track down:

1) A count of the images in the photo gallery to use as a top bound of the random number

2) The ability to import the random file selected automatically instead of the manual openPhotoGallery

Is there a way to do this with Appcelerator?

Thanks for the help!

    Todd,

    There is currently no API that allows you to enumerate through the photo gallery without user interaction. I understand that iOS 4.0 introduced the AssetLibrary Framework which is supposed to allow you to do something along those lines, but currently that Framework is not exposed via any API.
