touchend event not fired after touchmove

The touchend event does not seem to be fired after touch move. If i for instance use the code:

win.addEventListener('touchstart', function(e){    
    Ti.API.info('Touch start');
});

win.addEventListener('touchend', function(e){    
    Ti.API.info('Touch end ');
});

win.addEventListener('touchmove', function(e){    
    Ti.API.info('Touch move');
});

If i touch and then move i get:

>[INFO] Touch start

>[INFO] Touch move

>[INFO] Touch move

>[INFO] Touch move

If i just touch with out moving i get:

>[INFO] Touch start

>[INFO] Touch end

It seems like the touchend never gets called after a touchmove. This seems strange to me, did i misunderstand something or is it perhaps a bug ?.

  • you probably want to use touchcancel to.
    If you like move outside the area of you object, you receive a touch cancel event and not an touch end event.

    win.addEventListener('touchcancel', function(e){
    Ti.API.info('Touch move');
    });
