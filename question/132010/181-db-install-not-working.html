[1.8.1] DB install not working

All,

Using Ti Mobile SDK 1.8.0.1 on OSX, I'm trying to install a db using the following code:

var db = Titanium.Database.install('client_db', 'client_db');

When executed, I get the following error:

[ERROR] Script Error = invalid database install path. /Users/kipsta/Library/Application Support/iPhone Simulator/5.0/Applications/E9F9D340-085F-42A1-96F8-EDADBBD7BD08/DBTest.app/client_db  in -[TiDatabaseProxy install:name:] (TiDatabaseProxy.m:120) at app.js (line 8).

The code is being executed from my app.js and in the main Resources folder, with the app.js, is a file called client_db.sql

Shouldn't this be trying to install to /Users/kipsta/Library/Application Support/iPhone Simulator/5.0/Applications/E9F9D340-085F-42A1-96F8-EDADBBD7BD08/Library/Private Documents ?
