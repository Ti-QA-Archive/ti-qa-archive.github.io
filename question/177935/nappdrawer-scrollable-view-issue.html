Nappdrawer Scrollable view issue

Titanium SDK 3.3.0, for iOS …

I am using NappDrawer…i am using a scrollable view as the centerwindow.

Heres my code : 

var view1 = Ti.UI.createView({
            backgroundColor : '#123'
        });
        var view2 = Ti.UI.createView({
            backgroundColor : '#246'
        });
        var view3 = Ti.UI.createView({
            backgroundColor : '#48b'
        });

        scrblvwHome = Ti.UI.createScrollableView({
            top : '0dp',
            left : '30dp',
            width : Ti.UI.FILL,
            backgroundColor : 'pink',
            views : [view1, view2, view3],
            showPagingControl : false,
        });

        NappDrawerModule = require('dk.napp.drawer');

        drawer = NappDrawerModule.createDrawer({
            leftWindow : require("leftPanel").createLeftPanel(), //createAPIExampleWindow(),
            centerWindow : scrblvwHome,
            closeDrawerGestureMode : NappDrawerModule.CLOSE_MODE_ALL,
            openDrawerGestureMode : NappDrawerModule.OPEN_MODE_ALL,
            leftDrawerWidth : '200dp',
            statusBarStyle : NappDrawerModule.STATUSBAR_WHITE, // remember to set UIViewControllerBasedStatusBarAppearance to false in tiapp.xml
            orientationModes : [Ti.UI.PORTRAIT]
        });
        drawer.setAnimationMode(NappDrawerModule.ANIMATION_PARALLAX_FACTOR_3);
        drawer.setOpenDrawerGestureMode(NappDrawerModule.OPEN_MODE_BEZEL_PANNING_CENTERWINDOW);

         drawer.open();

However when i am at view one i on swiping i cannot get the Menu to become visible.
This seems so basic.
What am i doing wrong ?
Any help much appreciated…..

  • As per documentation the leftWindow and centerWindow must be Windows, not Views.

    You haven't provided code for require("leftPanel").createLeftPanel(), but I can see from the createDrawer parameters centerWindow, scrblvwHome, is a ScrollableView, not a window.
