Error with database installation / open on Iphone6

Hi,

I had just test my app on an Iphone6 but an error occur when i launch the application : 

- invalid SQL statement at file.js

var win = Titanium.UI.currentWindow;


    var db = Ti.Database.open('mydb.sqlite');

This is how i call my data base in file.js

I already tryed to install my data base in my app.js : 

var db = Ti.Database.install('mydb.sqlite', 'mydb');

And open it with var db = Ti.Database.open(&#39;mydb.sqlite&#39;);

It didn't work eather, the same error was display.

When i try to open the db with var db = Ti.Database.open(&#39;mydb.sqlite&#39;); the sql request doesn't return anything.

Do someone have an idea to fix it please ?

  • Accepted Answer

    Per the documentation, Ti.Database.install(databaseFile,databaseName)

    After you install the database, you reference it with the name that you gave it in the install() method. In your case, var db = Ti.Database.open(&#39;mydb&#39;);
