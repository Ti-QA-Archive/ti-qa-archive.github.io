createController: pass model as a argument does not work on Android

Hello,
I have a question about passing data to the controller by using the args parameter of the Alloy.createController method.
I try to pass a model to the controller, this my code:

var selectedModels = Alloy.Globals.productsCollection.where({ID:e.itemId});
var testController = Alloy.createController('test', {data: selectedModels[0]});

In test controller i do the following:

var args = arguments[0] || {}
alert(args.data.get("description"));

The result:
IOS: I get a message showing me the description.
Android: I get an error: Cannot call method get of undefined.

Any ideas what the cause can be?

Thanks
