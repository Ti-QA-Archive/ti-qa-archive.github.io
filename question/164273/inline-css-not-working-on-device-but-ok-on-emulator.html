  • Hello,

    I am not using any external CSS file as shown in the code : it's all inline CSS. I'm not sure which information you think was missing in the question : I would be glad to hear an example.

    I finally found where was the issue. Titanium does not seem to handle several classes very well. Indeed, although class="label label-info" didn't display anything, using only one style at a time worked (example : class="label" or class="label-info").
