Change the font color of title text of Option Dialog Box

Hi
I want to change the font color of title text in OptionDialog.
Can anyone help me out with this.

var opts = {
  cancel: 2,
  options: ['Confirm', 'Help', 'Cancel'],
  selectedIndex: 2,
  destructive: 0,
  title: 'Delete File?'
};
var dialog = Ti.UI.createOptionDialog(opts).show();

I want to change the font color of "Delete File".

Thanks in advance..

  • Hi

    As @Moiz said there is no property available to change the title colour for the standard OptionDialog.

    This doesn't mean that you couldn't create your own custom view to replace the OptionDialog you are using. Yes, it means more effort on your part but it would allow you the freedom to create what you want. Creating it as a Widget or Controller would also allow you to expose and re-use it throughout your app.
