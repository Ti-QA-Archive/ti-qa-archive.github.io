  • I had this problem, this is how I fixed it: On Windows, if you have been changing the default images in the assets folders (like splash or icon, or even adding new images) you need to delete the thumbs.db file in the folders where the images are saved. This requires turning the system files to visible in the folder settings.

    There is a way to disable the creation of these thumbs.db settings just search how online.
