  • Accepted Answer

    Titanium does not handle CoffeeScript natively, but you can use the Titanium CoffeeScript build plugin written by Bill Dawson to pre-compile your CoffeeScript files to JavaScript at build time. The only drawback to transforming .coffee files to .js files is that error messages give the line numbers of the JavaScript source, not the original CoffeeScript.

    • Titanium Studio is built as an extension to the Eclipse platform and inherits quite a bit of functionality from it. The Commands menu is part of Eclipse, not Titanium per se. The CoffeeScript menu and its submenus are mainly for editing and formatting CoffeeScript source files.
