Set values for top, right, left, bottom so that the position remains same in all the mobile devices.

  • hello Varun,

    Try using "Relative measurements" unit "%" to set the UI element position on their parents. As Different Android device varies in sizes like

    • small screens are at least 426dp x 320dp.

    • normal screens are at least 470dp x 320dp.

    • large screens are at least 640dp x 480dp.

    • xlarge screens are at least 960dp x 720dp.

    Thanks.
