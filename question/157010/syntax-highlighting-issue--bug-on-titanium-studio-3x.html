  • old post, but i had the same issue, and i found a possible solution. Find where exactly the syntax highlighting stops, and start removing and replacing characters. most likely it has been caused by a copypaste which included an unknown character, in my case it was a space that caused this and as soon as i removed it the file was fixed.

    Then you can just retype the character and it will not break the file highlighting.
