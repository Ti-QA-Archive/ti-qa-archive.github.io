    Without a code sample, it's a little hard to help. However, I'm guessing you're using appendRow() and deleteRow().

    Instead, create an array of your new rows and set them to the tableview's data property.

    var myRows = [];
for (var x in someData) {
  myRows.push({... table row creation goes here. ..});
}
myTableView.data = myRows;

    • Furthermore, unless you specifically need a TableViewRow instance, use a generic object rather than calling createTableViewRow() for each row. Using generic objects saves "trips over the bridge" (communication between the JavaScript and native layers) and can save measurable time.

      var myRows = [];
for (var x in someData) {
  myRows.push({
    moveable: true,
    editable: false,
    title: x[0],
    ...
  });
}
myTableView.data = myRows;

      Also, don't set an index property. One is assigned automatically.
