  • Accepted Answer

    Your TSS file is off, you need to use the font object just as you do in traditional titanium dev

    "Window": {
    backgroundColor: "#fff"
},
"#mainLogo":{
    image:"/images/asdalogo.png",
    width:250,
    height:100,
    bottom: "50%"
}
"#labelTest": {
    top: "75%",
    color: "#fff",
    font: {
       fontFamily: "AsdaScript",
        fontSize: "20"
    }  
}

    index.js:

    $.index.open();

    index.tss:

    "Window": {
    backgroundColor: "#fff"

    height:100,
    bottom: "50%"
}
"#labelTest": {
    top: "75%",
    color: "#fff",
    fontFamily: "AsdaScript",
    fontSize: "20"    
}

    index.xml:

    <Alloy>
    <Window fullscreen="true" backgroundColor="#7cc242">
        <ImageView id="mainLogo"></ImageView>
        <Label id="labelTest">This is a test label</Label>
    </Window>
</Alloy>

    Relevant section of info.plist (sitting in application root folder):

        <key>UIAppFonts</key>
    <array>
        <string>/AsdaScriptBold.otf</string>
        <string>/AsdaScriptRegular.otf</string>
    </array>

    Fonts are sitting in root of "assets" folder, font "friendly" name is AsdaScript

    config.json:

    {
    "global": {}, 
    "env:development": {}, 
    "env:test": {}, 
    "env:production": {}, 
    "os:ios": {}, 
    "os:android": {}, 
    "dependencies": {}
}

    alloy.js only contains commented lines (as per default). The rest of the folder structure is as per standard alloy layout - nothing has been altered. All this does when run, is give a green screen and display both the image and the label. The label font, however, is "standard".

    Smallest Titanium test case which does work

    This is a white screen with a label in the middle.

    app.js:

    Titanium.UI.setBackgroundColor('#000');

var win1 = Titanium.UI.createWindow({  
    title:'Window 1',
    backgroundColor:'#fff'
});

var label1 = Titanium.UI.createLabel({
    color:'#000',
    text:'I am Asda Script',
    font:{
        fontSize:40,
        fontFamily: 'AsdaScript'
    },
    textAlign:'center',
    width:'auto'
});

win1.add(label1);

win1.open();

    Relevant section of info.plist (sitting in application root folder):

        <key>UIAppFonts</key>
    <array>
        <string>/fonts/AsdaScriptRegular.otf</string>
        <string>/fonts/AsdaScriptBold.otf</string>
    </array>

    Fonts are sitting in "fonts" folder in root of "resources" folder, font "friendly" name is AsdaScript
