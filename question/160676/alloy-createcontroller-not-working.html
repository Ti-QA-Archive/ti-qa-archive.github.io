Alloy createController not working

I am trying to open a new window in alloy project when i say
new_window = Alloy.createController("about").getView();

This is my about.js page code:

var formFactor = "handheld";
if (Alloy.isTablet) {
formFactor = "tablet";
}
function closeModal(e) {
$.modal.fireEvent("removeClose", e);
}

This is my about.xml page code:

<Alloy>
<View class="container">
<View id="overlay" />
<View id="modal">
<View id="vertical">
<View id="header">
<Label id="title">About</Label>
<Require id="done_btn" type="view" src="done_btn" onClick="closeModal"/>
</View>
</View>
</View>
</View>
</Alloy>

So please guide me why I'm not able to opening new window.

Thanks in advance.

  • Accepted Answer

    Your about controller is not a window. Change the top-level element and make it a window.

    <Alloy>
    <Window class="container">
        <View id="overlay" />
        <View id="modal">
            <View id="vertical">
                <View id="header">
                    <Label id="title">About</Label>
                    <Require id="done_btn" type="view" src="done_btn" onClick="closeModal"/>
                </View>
            </View>
        </View>
    </Window>
</Alloy>

    You can now open it

    var new_window = Alloy.createController("about").getView();
new_window.open();
