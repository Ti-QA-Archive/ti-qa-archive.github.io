    • I'm also with the same problem.

      I did the workaround of getting the activity after the window is open but like Michael said, it first shows the navigation bar and then it disappears which is not a solution since it looks very unprofessional.

      I don't want to remove it from the entire project, I just don't want the user to be able to see it on the loading window.

      if(OS_ANDROID){
    $.win.addEventListener('open', function winOpenHandler(){
        $.win.removeEventListener('open', winOpenHandler);
        if(!$.win.activity){
            Alloy.Globals.error('android activity', "Can't access action bar on a lightweight window.");
        } else {
            var actionBar = $.win.activity.actionBar;
            if(actionBar){
                actionBar.hide();
            }
        }
    });
}
