Checkboxes: how to use?

Hello

In my app i made a popup window so you can choose some criteria to filter a tableview. Now i can sort by type for examle. I've created a pop-up window with a checkbox and the name of type.

The checkbox is made with an image (I Assume thats the only way?), but now I want to check which checkboxes are checked in my code. How can i loop through all the checkboxes?

Is there a way to find a control in my code? Or is there a better way to use checkboxes?

Here's how i create my checkbox: (perhaps I should create a field 'name' or so?)

 var typeCheckBox = Titanium.UI.createButton({
        image: '../images/unchecked.png',
    checked:false,
        width: 20,
        height: 20,

    typeCheckBox.addEventListener('click', function(e){
        if (e.source.checked == true){

    TJeu

    If you are using Android, you can use the checkbox widget that has been extended from a UI switch, as follows:

    var win = Ti.UI.createWindow({backgroundColor:"white"});

var checkbox = Ti.UI.createSwitch({
    style:Ti.UI.Android.SWITCH_STYLE_CHECKBOX,
    title:"Sound Enabled",
    value:true
});

win.add(checkbox);
checkbox.addEventListener("change", function(e){
    Ti.API.info("The checkbox has been set to " + e.value);
});
win.open();

    But without more information about your development objectives, it's difficult to say whether this is suitable for you. ;)

    • OK, Tjeu, no worries, I got a bit sidetracked, anyway! ;) Still, your platform and environment is almost always important, so it's worth stating.

      The approach you need to take is to declare an array outside of any eventListeners, and then add each of your checkboxes to it. Then, when you want to determine the checkbox state, you simply iterate through this array. The following code has been tested and works, although it uses the Android checkbox UI widget mentioned before. The technique you will use will be identical, though.

      var win = Ti.UI.createWindow({
    objName: "mainwindow",
    backgroundColor:"white",
    layout:"vertical"
});

var checkboxArray = [];

for (var i = 0; i<6; i++){
    var checkbox = Ti.UI.createSwitch({
        objName:"checkbox-" + i,
        style:Ti.UI.Android.SWITCH_STYLE_CHECKBOX,
        title:"Sound Enabled",
        value:true
    });

    win.add(checkbox);
    checkboxArray.push(checkbox);

    checkbox.addEventListener("change", function(e){
        Ti.API.info(e.source.objName + " has been set to " + e.source.value);
    });

}

win.addEventListener("click", function(e){
    if(e.source.objName === "mainwindow"){
        for (var i=0, ilen=checkboxArray.length; i<ilen; i++){
            Ti.API.info(checkboxArray[i].objName + " has been set to " + checkboxArray[i].value);
        }
    }
});
win.open();

      Good luck with it! :)
