Is there a guide to using Modules?

I see a few modules I'd like to use, but can't exactly figure out how to use them, due to lack of documentation for the individual module.

How do you install them to use in your app?

  • You can check out the module development guide for the specifics on developing and installing modules. However the quick of it is to take the module zip file (com.test-iphone-0.1.zip) and place it in the Titanium directory .. (/Library/Application\ Support/Titanium/ or ~/Library/Application\ Support/Titanium/). Then add the module to your tiapp.xml file, see the wiki entry for details.
