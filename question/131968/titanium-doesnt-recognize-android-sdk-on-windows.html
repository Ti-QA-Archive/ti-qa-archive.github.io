  • Go to Preferences and select Android. Doublecheck the Path for SDK Location. Point it to your D:\ti-android-sdk, click apply, and see if any SDK Targets get listed in the table. Should a minimum platform of 2.1 (2.2 for Ti SDK 1.8.1 support, I think).
