Update imageView width based on new image

I got an imageView in a tableRow. I want to update the image property to an other image on a button click.
It works, but the new image still got the size of the old image. If I flip my screen to landscape it will get it own size. But I want it to have it's own size when the new image is in the image property.

var tableRowQuestionImage = Ti.UI.createTableViewRow({
        width: Ti.Platform.displayCaps.platformWidth,
        backgroundColor: '#000',
        selectedBackgroundColor: '#000',
        layout: 'vertical',
    });

    var tableRowQuestionImageView = Ti.UI.createImageView({
        image: '/images/btnbgok.png',
        height: ((Ti.Platform.displayCaps.platformHeight/100)*30)
    });

    tableRowQuestionImage.add(tableRowQuestionImageView);

tView[0].children[0].data[0].rows[1].children[0].setImage('/images/test2vraag4afbeelding.jpg');

So is it possible to get the images dimension of test2vraag4afbeelding.jpg before loading it in the view, so I can set the image to a new width? The height is always the same ( ((Ti.Platform.displayCaps.platformHeight/100)*30) ) so I can calculate the new width If I've the dimension of the image.

  • Put the new image into an offscreen ImageView, then get the dimensions from that ImageView (either directly or by using ImageView.toImage()). This may not be fast, but it should work.
