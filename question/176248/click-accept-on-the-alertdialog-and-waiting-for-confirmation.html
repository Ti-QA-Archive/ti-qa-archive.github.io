  • Hi!
    Most of the time this is done by using Push Notifications. Ofc, you could build your won service where you can ping your server to check if the user have accepted, but that will give you to many unwanted httpRequests.

    I'm using Parse.com to handle push Notification, but you can also take a look at Appcelerators Cloud Service : http://docs.appcelerator.com/cloud/latest/#!/api/PushNotifications

    So the idea is to have you PHP service with DB connection that you normally call to the that user list. A user click on a person to request "friendship" you can send out a push notification to that user and inform them that someone want to add them as friends. When that user accepts the request, then you send a Push to the first user. In your App you can listen for Push callbacks and update your views based on that.
