Confirm that someone entered a number (and only a number) into a field?

I have an iPad application with input fields. Some of the require a number be input, such as 6 or 10.25. I then do math with the inputs when the user clicks a button, by casting the value to Number().

How can I check that the user entered a valid number (as opposed to a word or something like 1'6"?

SDK 2.01, OS 10.7.3

    You can test the input's value like this:

    //Your input
var yourInput = Ti.UI.createTextField({ ... });

//Test the input's value
if(isNumber(yourInput.value)){
    //It's a number
}else{
    //It's not a number, throw an error
}

//Number testing function
function isNumber(n) {
    return !isNaN(parseFloat(n)) && isFinite(n);
}

  • You can try to cast value to number and check if it is numeric value:

    function isNumber(num) {
    return !isNaN(+num);
}

    +num - cast to number
    isNaN - true, if value is not-a-number
