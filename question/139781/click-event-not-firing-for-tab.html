Click event not firing for tab

Hi everyone,

I'm trying to setup a click event listener for a tab. The documentation lists click as a valid event for tabs, but I can't seem to get it to work.

Here is the chunk of code I'm having difficulty with:

var someWindow = Ti.UI.createWindow({ ... });
var someTab = Ti.UI.createTab({
    icon: 'someiconpath.png',
    title: 'Title',
    window: someWindow
});
someTab.addEventListener('click', function(e) {
    alert('Some tab clicked');
});

I'm using 2.1 and attempting to run this code in the iOS simulator. I've read similar questions on here and they all mention using the focus event. I actually need to know when the user clicks the tab versus having that tab come into focus. Is there something I'm missing here?

Thanks in advance,
Brandon

  • Currently, that is not supported: https://jira.appcelerator.org/browse/TIMOB-6499

    As a workaround, you can listen to "focus" event of window inside tab and from there fire "click" event (or put there your logic). Basically, when tab is clicked, window inside that tab will gain focus.

    someWindow.addEventListener('focus', function() {
    someTab.fireEvent('click');
});

    Just take care of situation when tabgroup is loaded for the first time (focus will be fired for first window, but first tab won't be clicked).
