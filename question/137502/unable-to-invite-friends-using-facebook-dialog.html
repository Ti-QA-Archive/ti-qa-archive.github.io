Unable to invite friends using facebook dialog

I am developing an app for iPhone. In this app I am trying to have a facebook dialog for enabling the end user to send invitation of this app to his facebook friends. For the same I am using the below line of code.

Ti.Facebook.dialog('apprequests', {message: 'my custom message'}, function(data) { 
  Ti.API.info(data); 
});

When I run this on simulator, I find the facebook dialog popping up and I am able to see my friends list. After I select the friends and click send, the popup closes and in the console I see

{
  cancelled = 0;
  result = "request=xxxxxxxxxxx&to%5B0%5D=xxxxxxx";
  source = "[object FacebookModule]";
  success = 1;
  type = result;
}

And when I go to the profile of my friend whom I have invited, I see that he hasn't received any invitation. What could be going wrong?
