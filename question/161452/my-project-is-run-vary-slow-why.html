  • Well, first of all, you have to know that testing applications on the Android simulator is very slow. This not a Titanium issue, but actually related to the Android simulator itself. This problem is even greater when developing on a middle-end machine.

    The best recommendation I could give you would be to test on an actual device.

    Hope that clears that up for you.
