    Hi J W, i have modified your code a little..though i was not having some files like main.js which you are using..
    so resolved your one issue related to popup window and its tool bar at bottom and nav bar at top…

    There is a bug in Titanium, that it does not show Navigation Bar on single widow until you use navigation Group

    here is the modified and tested code….

    I just added this code after your poplin creation

    var baseWin = Ti.UI.createWindow();
    var navGroup = Ti.UI.iPhone.createNavigationGroup({
        window : popwin
    });

    popwin.navGroup = navGroup;
    baseWin.add(navGroup);
    baseWin.open();

    complete code
