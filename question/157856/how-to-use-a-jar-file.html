How to use a jar file

  • You cannot directly use classes from a JAR file inside a Titanium application. Titanium relies on JavaScript, so it makes sense that Java classes contained in your JAR file won't work.

    That being said, there is a way to use native Java classes inside a Titanium application. You can achieve that by developing a native extension module. But doing so requires deep knowledge of the Titanium platform as well as the native platform itself.

    So, to sum up, you need to code your functionality using JavaScript if you want it in your app.
