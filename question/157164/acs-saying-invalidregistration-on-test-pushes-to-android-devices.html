    • Thanks for the response.

      What we found was the problem is this - we changed which version of cloudpush we were using (went back to 2.1.0) due to the bug mentioned here: https://jira.appcelerator.org/browse/TC-2870

      When we did that, we didn't realize that GCM was not supported in that version. (not sure if documentation for previous versions of cloudpush is available anywhere - I tried to search for it) Eventually I found a release note saying GCM was just added August 1 2013, so we then just tried MQTT. It is working now that way!
