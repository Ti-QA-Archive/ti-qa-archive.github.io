  • Vimeo player is playing HTML5 Video content. For supporting this you need to create a Titanium Android
    Module which can possibly return a View layout to Titanium project. You can see HTML5WebView which
    is using custom WebChromeClient in this path.
    But this is failing to play Video files when we tried in Android 5.1 OS version.

    It is returning error MediaPlayer Error(0,1)

    Below content is already there in Android API documentation.

    HTML5 Video support

    In order to support inline HTML5 video in your application, you need to have hardware
    acceleration turned on, and set a WebChromeClient. For full screen support, implementations
    of onShowCustomView(View, WebChromeClient.CustomViewCallback) and onHideCustomView()
    are required, getVideoLoadingProgressView() is optional.
