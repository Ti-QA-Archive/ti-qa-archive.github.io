How to send push notifications in diferent languages?

Hi, I'm trying to figure out how to send push notifications in diferent languages for iOS.
I tried sending it with "loc-key" but the application can't decode it.

Is there any other method?
If not, where do I need to place the file in the project so that the app can decode it?

Thanks in advance.

  • Hi

    Whilst I have not needed to do this myself, so I cannot attest to having tried it; I believe it works like this;

    Current parcel has the message like this;

    "alert": 'Hello World'

    You expand it like this;

    "alert": {
    "loc-key": "hellow"
}

    This then would look up your i18n strings local folders for the key 'hellow' and uses the returned string from that for your push.

    If you needed to personalise the message then you use normal i18n string formatting and placeholders and add the properties in an array;

    "alert": {
    "loc-key": "hellow",
    "loc-args": ["Malcolm"]
}

    Our localised string in this case would look like;

    En = "Hello %@"
Fr = "Bonjour %@"

    The key here is that the strings used in the push parcel are ALREADY in your app.

    Keep in mind, I have never tried this - you are getting my understanding of what you do do. This info is based on using Urban airship and making your own JSON object to send.

    Good Luck
