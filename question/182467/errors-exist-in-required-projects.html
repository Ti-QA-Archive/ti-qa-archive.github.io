  • in the console; at the end of the build process there is cryptic error when trying to load .apk into emulator:

    [INFO] : Installing app on device: Genymobile
    [ERROR] : Failed to install apk on "192.168.56.101:5555"
    [ERROR] Application Installer abnormal process termination. Process exit value was 1

    there are intermittently a ton of file overwrite warnings like this:

    [WARN] : Overwriting file E:\www\UvisitApp\build\android\bin\assets\Resources\appicon.png

    but other than that; no console errors.

    there is also a ton of java errors in the Studio log:

    java.io.IOException: Cannot run program "d:\Git\cmd\git.exe" (in directory "E:\www\UvisitApp"): CreateProcess error=206, The filename or extension is too long
