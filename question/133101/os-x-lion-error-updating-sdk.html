OS X Lion Error updating SDK

Hi everyone,

I made an app for iPhone and iPad using Titanium a few months ago, and everything worked fine. I recently got a new computer at work, and had to re-download and install Titanium. When I opened it for the first time, it started running all the updates, etc. but at the end I had an error message saying it can't unzip the file because there's no such file or directory. I continued on as normal, thinking it wasn't a big deal, but now when I try to run my app in the simulator, I get the message "Titanium mobile SDK not found".

I found a few other questions on here regarding this problem, but none of the solutions are working for me. When I try to find the folder to check permissions, the folder doesn't exist. Shouldn't Titanium have created this folder for me? I also tried renaming my user name to my short user name, but no luck there either.

Is there a way I can download and install the SDK manually, or can someone explain step by step how to make the automatic update work?
