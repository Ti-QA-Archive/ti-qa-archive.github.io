  • Hi

    I can only assume that you are trying this on the simulator as your code works fine on a physical device.

    The documentation does suggest that the pattern parameter will work for iOS, it actually suggests; Android, Mobile Web and Tizen can accept the pattern, iOS is not specified.

    So the pattern will be ignored, and these examples will do the same thing on iOS;

    Ti.Media.vibrate();
Ti.Media.vibrate([0,500]);
Ti.Media.vibrate([0,500, 250, 500, 1000]);
