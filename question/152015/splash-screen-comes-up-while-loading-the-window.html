splash screen comes up while loading the window.

hi…. I m trying to use the android back button so that i can go back to the home page.. but on clicking back button, i m able to see the previous window but only for some sec. and the splash screen comes up…. pls help me out..

this is the code…

index.js

function customerClick() {
    var customerListView=Alloy.createController('customerListWindow').getView();    
    customerListView.open();
        $.index.close();
}

customerListWindow.js

if (Ti.Platform.osname == 'android')
{    
   $.customerListWindow.addEventListener('android:back', function() 
    {                
        var homeWindow=Alloy.createController('index').getView();
         homeWindow.open();
         $.customerListWindow.close();
    });
};

  • Android might be struggling having no window (activity) at all at some point between opening the index-Window and closing the customerListWindow.

    Why not leave the index-Window open all the time?

    Then you can also properly set exitOnClose on that window so the user can close the app by pressing back on the main window.
