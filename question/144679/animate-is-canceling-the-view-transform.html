animate is canceling the view transform

Application type: mobile
Titanium SDK: 2.1.3GA
Platform & version: Android ICS
Device: Sony Xperia

Hello,
i have a view with 2D matrix transformation applied to it

var t = Ti.UI.create2DMatrix();
t = t.scale(0.95, 0.95);
topView.transform = t

but when i execute an animation on the same view, the previously applied transformation gets cancelled automatically, scaling the view to 1

var closeAnimation = Ti.UI.createAnimation({
            left : this.navPagingView.size.width,
            curve : Ti.UI.ANIMATION_CURVE_LINEAR,
            duration : 200
        });
topView.animate(closeAnimation);

any idea why this is happening?
