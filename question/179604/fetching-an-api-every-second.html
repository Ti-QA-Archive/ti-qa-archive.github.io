    Hi

    I would say your biggest problem here is that you are using the wrong approach to your solution. You shouldn't be calling an API every second, for one if you are calling your API asynchronously you can't guarantee that the responses will return in the same order and using it synchronously still wouldn't be the solution.

    This is quite a general question. You need to explain what it is you are trying to achieve so that we can provide you with the best solution. But at this moment in time it would seem that you should be using a websocket so that you have a constant connection to your data. This is something that you would need to spend some time researching if you don't already know about it.
