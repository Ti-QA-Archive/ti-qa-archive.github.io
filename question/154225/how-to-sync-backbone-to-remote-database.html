    • Nothing in Alloy/Titanium/Backbone is going to help you sync with JBOSS as that doesn't tell us anything. The sync adapters in Alloy, documented here, are meant to simply synchronize the memory state of a model/collection with a data store, be it local or remote. In theory you could also have multiple storage points. It is not meant to directly solve the problem of sync across devices, but you can use it as a stepping to to sync to a data store, which in turns syncs to any number of devices.

      again, starting with the REST adapter as a connection point to your PHP API is a good start.
