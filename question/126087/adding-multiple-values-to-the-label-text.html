Adding multiple Values to the label text.

Hi,
Is it possible, to adding of multiple values to the label Text property.I read values from Array and assign to the Label Text,
Here is my snippet:

var lblTitle = Titanium.UI.createLabel({
                text:dataArray[i].content,
                font:{fontSize:12,fontFamily:'Helvetica Neue',fontWeight:'bold'},
                color:'white',
                width:'75%',
                textAlign:'left',                
                height:'auto',
                left:0,
                touchEnabled:false                
                });

above, I got the value from the Array, i.e dataArray[i].content..
Now I want to add another value from the array i.e dataArray[i].category(Appending both the values as the Label text property. Can any one helping me on this..
Thanks in advance..
Lokesh

    Simply use the + operator.

    var lblTitle = Titanium.UI.createLabel({
    text:dataArray[i].content + ' ' + dataArray[i].category
});

  • Also, if you didn't want to keep repeating your variables everytime you added a new one, you could use the += operator and simply add another variable ontop. Such as…

    var lblTitle = Titanium.UI.createLabel({
 text:dataArray[i].content,
 font:{fontSize:12,fontFamily:'Helvetica Neue',fontWeight:'bold'},
 color:'white',
 width:'75%',
 textAlign:'left',               
 height:'auto',
 left:0,
 touchEnabled:false              
});

lblTitle.text += ' '+dataArray[i].content2;
lblTitle.text += ' '+dataArray[i].content3;
