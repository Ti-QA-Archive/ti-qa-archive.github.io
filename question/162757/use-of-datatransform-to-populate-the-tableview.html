  • dataTransform only applies if you are using data binding and it defines a function to modify the model used to bind the UI with the model data. This function is executed automatically when the model is modified, so you don't have to care about calling, it will be called if the model or the collection binded is modified. As far as you update or force to sync the data binded, it will be called.
