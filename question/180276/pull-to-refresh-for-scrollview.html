"Pull to refresh" for Scrollview

I need a "Pull to refresh" component for a scrollview instead of a tableview. Therefore I added a view with a negative top value. My problem is that the scrollview will only let the user scroll down if the content heigt is greater than the screen height. How can I force the scrollview to scroll down even if contentheight is very small?

                <ScrollView class="rootScroll">

</Alloy>

  • Hi Alexander Rieger,
    Thanks for your concern. You can also try to use “scrollend”
    event for scrollview. scrollend Fired when the view has stopped moving completely.
    For details please see the Documentation

    Code :

    scrollview.addEventListener('scrollend', function(evt) {
    if([checking if ended on 0 location]) {
        //code for refreshing
    }
});
