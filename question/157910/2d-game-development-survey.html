2D Game development survey

Hi,

My name is Sergey Nosenko, and I still continue development of QuickTiGame2D engine originally developed by Kota Iguchi. QuickTiGame2D now is Platino Engine by Lanica.

I made fork QuickTiGame2D under BSD license and created engine called TiGame2D very similar to Platino, but with lot of features that unavailable in Platino.

I published four interactive books for kids based on the engine and you can look at them here:
https://itunes.apple.com/us/app/tri-porosenka-interaktivnaa/id623858130?mt=8
and
https://itunes.apple.com/us/artist/ipublisher-ua/id468043785

Notice: only "The Three Little Pigs" is available in English yet

This question is for developers who want to develop 2.5D games using Titanium. I'm going to publish Ti.Game2D on GitHub but I'm not sure that it is interesting for you. I need to apply some efforts (documentation, samples, code cleanup, etc) before publish it and I have to know is it needed for anybody? :)

Also I'd like to co-operate with Java and Objective-C developers to make the engine much better.

I'm open to new any ideas and contributions.

I'd like to write blog post to developer section where describe how it works with samples, but I don't have access to it. :(

— asked October 1st 2013 by Sergey Nosenko
