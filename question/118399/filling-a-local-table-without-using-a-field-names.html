  • You does not need the field names. With …

    SELECT * FROM table_name

    … you can get all data from the source table and with …

    INSERT INTO table_name VALUES (value1, value2, value3,...)

    … you can store this data in the SQLite database.

    Hope that is what you mean.

  • If you don’t need the field names separately, the simplest way is to send all values for a record in one line. On the server side you get all values with SELECT * FROM and put the values for each record in a comma separated string. Then you send this strings as JSON array and on the client side you can put this string in the INSERT INTO line.

    var values = json[pos].values;
var sql = "INSERT INTO table_name VALUES ("+values+")";
