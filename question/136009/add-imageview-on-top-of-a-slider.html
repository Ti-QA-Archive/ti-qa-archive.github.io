Add imageView on top of a slider

Hello,

I'm facing a problem while trying to add add an image on top of a slider.
On IOS it all works just fine, but on Android it doesn't work(does things ever work on Android for the first time)..
This is my code:

var slideImage = Ti.UI.createImageView({
        image: '../../images/kruisje.png',
    });
    var slider = Titanium.UI.createSlider({
        min: info.leftvalue,
        max: info.rightvalue,
        top: 0,
        width: Titanium.Platform.displayCaps.platformWidth-10,
        thumbImage: "../../images/blank.png",
        leftTrackImage: '../../images/slider-left3.png',
        rightTrackImage: '../../images/slider-right3.png'
    });
    var leftAnker = Ti.UI.createImageView({
        image: "../../images/vertical-line.png",
        left: (info.leftpos/100)*slider.width+5,
        width: 1,
        height: 30
    });
    slider.add(leftAnker);
    var rightAnker = Ti.UI.createImageView({
        image: "../../images/vertical-line.png",
        left: (info.rightpos/100)*slider.width-5,
        width: 1,
        height: 30
    });
    slider.add(rightAnker);

I add this slider somewhere else to the window, so this is not the problem aight.
It WORKS on IOS but not on Android, so does anybody knows what i'm doing wrong here?

  • If you are working for android then you should use min and max properties with slider.For Example:

    var slider = Titanium.UI.createSlider({
    top: 50,
    min: 0,
    max: 100,
    width: '100%',
    value: 50
    });

    for further details reffer the following link:
    http://docs.appcelerator.com/titanium/2.0/index.html#!/api/Titanium.UI.Slider
