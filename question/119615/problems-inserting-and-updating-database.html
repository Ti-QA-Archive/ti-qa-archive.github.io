  • Alex

    The way to access databases with Titanium is explained in the SQLite Databases guide. Using the information provided here, there are a few changes you could make to your code:

    • there is no need to quote table names in your CREATE TABLE statement, if they do not contain a space
    • string substitution will, at least, simplify your code and, at best, fix your problems. Hence, try changing your code to the following…
    db.execute('CREATE TABLE IF NOT EXISTS gameroster ( gamerosterid INTEGER PRIMARY KEY AUTOINCREMENT NOT NULL, gameid INTEGER NOT NULL, teamid INTEGER NOT NULL, playerid TEXT NOT NULL, position TEXT NOT NULL, jerseyno INTEGER NOT NULL, status TEXT NOT NULL, post TEXT NULL )');
db.execute('UPDATE gameroster SET status=? WHERE playerid=? AND gameid=?', status, playerID, Ti.App.gameID);

    Hope this helps
