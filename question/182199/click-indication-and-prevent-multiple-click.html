Click indication and prevent multiple click

Hello everybody.
There is a default tool in titanium to set click indication?and someone tell me if there is a module for prevent multiple click? because when I click fast enough on button that open new window, it can open me more than one window. I saw some solutions in the forum to set in the beginning of the function some lock and release it in the end. the problem is that sometimes I have async function that can be ended after the function ended, like http request and other stuff..there is a solution for this?
Thank's!

    You can use underscore library. It has two methods (throttle/debounce) to prevent multiple click. Choose one as per your requirement. If you are working on Alloy app then it is by default available to you. No need to import it separately unlike titanium classic project.
