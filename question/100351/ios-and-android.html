iOS and Android

Hi,

I'm currently adapting my iOS app for Android. I am wondering if I should create a new copy of the app and just modify it, or put in a bunch of if Iphone statements.

My question is do those if statements that check if it is iOS or Android stay in there when the app is compiled to a specific OS? For example, does the iphone specific app still check if Android or iOS is running, and once I compile an Android version will it still check if it is an iPhone? I don't want this checking to take up resources on the actual phone.

Thanks.

  • CZ

    Yes, the if statements will remain in the code when the app is run on both platforms. You should not be concerned about the extra resources used; this approach to programming with Titanium is standard practice, and the extra overhead will be minimal.

    Note that there is no need to create a separate copy of the app for each platform - this really would defeat one of the main benefits of using Titanium in the first place.
