  • Hello I searched the forum and found the answer :)
    here is the solution

    I added this line in the file tiapp.xml

    <android xmlns:android="http://schemas.android.com/apk/res/android">
    <manifest>
        <application android:theme="@android:style/Theme.Holo.Light.NoActionBar" />
    </manifest>
</android>

    More info you can follow this link:

    http://developer.appcelerator.com/question/161730/how-remove-topbar-android—help
    http://docs.appcelerator.com/titanium/latest/#!/guide/Android_Themes
