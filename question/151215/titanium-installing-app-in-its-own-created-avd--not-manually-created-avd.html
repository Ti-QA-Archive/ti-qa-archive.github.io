  • Accepted Answer

    When Titanium launches the Android Emulator it takes the Android SDK's "avd id" and "avd skin" and checks if there's an existing AVD named "titanium<avd id><avd skin>". The "id" is machine specific. You can see all the ids as well as your AVDs by running android list.

    The default avd id is "7" and the default avd skin is "HVGA", so you could delete the existing "titanium_7_HVGA" AVD and make a new one with whatever Android SDK and settings that you want and name it "titanium_7_HVGA" and everything should work.

    Note: this may change in the upcoming Titanium SDK 3.2.0. We're actually going to make this easier. :)
