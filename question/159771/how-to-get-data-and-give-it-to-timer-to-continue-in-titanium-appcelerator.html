how to get data and give it to timer to continue in titanium appcelerator

Hi am having the following code

var s =02 ;
var s1 = 13;
Ti.API.info("S"+s);
Ti.API.info("S1"+s1);
var my_timer = new countDown(s,s1, function () {
}

if am giving in this way the timer is working.

am getting the value and it is stored as

var lasttimer = Ti.App.Properties.getString('lasttimercount').split(":");
    var last1 = lasttimer[0];
    var last2 = lasttimer[1];

when am giving in the following manner it is not working that is

var s =last1 ;

    Hi,

    you are actually storing string value so you need pass integer as per your first code.

    so try this one..

    var s = parseInt(last1);
var s1 = parseInt(last2);
Ti.API.info("S"+s);// output is 02
Ti.API.info("S1"+s1);//output is 13
var my_timer = new countDown(s,s1, function () {
}

    • This code;

      var s = parseInt(last1);
var s1 = parseInt(last2);

      Should be;

      var s = parseInt(last1, 10);
var s1 = parseInt(last2, 10);

      You should ALWAYS provide the second parameter radix as 10. If you do not then the number passed may be confused as something other than a base 10 number.

      The same is true for parseFloat.
