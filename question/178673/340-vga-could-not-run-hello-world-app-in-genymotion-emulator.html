  • I had this problem too. It hangs after 'Making sure the adb server is running'. I solved it by making genymotion point to the same Android SDK that Titanium uses.

    In genymotion, access Settings > ADB and select the 'Use custom Android SDK Tools' checkbox. Select the SDK that is under '/Users/<username>/Library/android-sdk-macosx'.

    It resolved the issue in my case.
