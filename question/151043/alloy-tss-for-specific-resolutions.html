Alloy tss for specific resolutions

Hi,

I want to tweak visuals based on an iPhone form factor vs larger. Right now .tss files allow formFactor=tablet / formFactor=handheld, but I've noticed that some android devices (e.g. the default emulator titanium_2_WVGA854) have a wider resolution, yet their formfactor is set to "tablet".

Is there a way I can create tss that affects this devices (or devices larger than 320 width) and not devices of 320 width or below?

In general I see this as a common use case, so if it's not supported by alloy, it probably should be, IMHO. :)

Thanks,
Dave

    You're talking about pure pixel dimensions? Not accounting for density? That likely to lead to some misleading results. That said, you can always override the 2 builtin properties to use them how you want. Special care was taken to make these properties as accurate as possible, particularly on ios and android, but you can do this to override in your "app/alloy.js" file:

    // isTablet and isHandheld are properties, not functions for the sake of speed.
// Be sure when you override that you remember this.
Alloy.isTablet = Ti.Platform.displayCaps.platformWidth > 320 ? 'tablet' : 'handheld';
Alloy.isHandheld = !Alloy.isTablet;

    Again, going purely by pixels on Android is dodgy, but this is how you could do it.
