Set Properties of All Table Rows?

I want to be able to write the properties of each button once (for obvious reasons), like this:

var btns = [
    {title:"Pay Your Bill"},
    {title:"Make A Claim"},
    {title:"Manage Account"},
    {title:"Customer Service"}
];
btns.color = "#333";
btns.left = "50";

var list = Ti.UI.createTableView();
list.data = btns;

What's the proper way?

    The best example of programmatically creating Titanium objects is in the TableViews guide. See Programmatic row creation with Titanium Views. Reading that tutorial from the top, even though TableViews are admittedly a little more complicated than buttons, will give you a good idea of the underlying approach that you can apply to all areas of Titanium development.
