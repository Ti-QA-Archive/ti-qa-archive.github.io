How to change chat server ?

Hi

I am working on a chat app in android.I need to store my chats in an sqlite database not in Cloud server provided by appcelerator.Can I do that ? If I do so will the chat api's work in my app ? If somebody has some good knowledge about this, Please help me.Thanks in advance :-)

  • The chat actually routes through ACS, if you want to store the messages, that is fine, just fire up your sqlite database and insert messages as they come in, or you can push them to your server and store them.

    If you actually want to use your own server for chat, then you'll have to create your own chat implementation. I'd try looking at a simple nodejs server that you can connect to using the Socket API and just integrate your message storage system on server.
