This is the code i use in the controller

index.js

var ents = Alloy.Collections.entries;

var num = Math.floor(Math.random()*(ents.length-1));

This just gives me a null object (selected) and if i print out ents to the console the app crashes.

any ideas?

    You need to pass the alloy_id value of the object to collection.get() to get the model. Try using collection.at() to get an item at a given index.

    var ents = Alloy.Collections.entries;
var num = Math.floor(Math.random()*(ents.length-1));
var selected = Alloy.Collections.entries.at(num);
alert(JSON.stringify(selected.item));
