Configuration section

Is there a way to setup a configuration section that can hold various variables? ie….In php, I have a config.inc.php file that has my database settings as well as the website name and any other variable I need to use on any other page. This config page is linked(required) on all pages of the website.

Is there a way to do this using Appcelerator?

  • Chris

    The place for configuration settings is in Titanium.App.Properties, where key/value pairs are saved to a file, handled seamlessly by the respective platform.

    For a demonstration of how to use this functionality, see the properties.js script in the KitchenSink.

    Cheers
