  • Hi Chung,

    Try to use Custome Property of rows. like this :

    
var sideScroll = Ti.UI.createScrollView({
    height:229,
    width:'100%',
    top:40,
    scrollType:'horizontal',
    layout:'horizontal'
});
row.add(sideScroll);

row.scrollView =  sideScroll;      //   SET ROW's CUSTOM PROPERTY

rows.push(row);
library_view.appendRow(row);

    and Access is like this :

    
for(var k = 0; k < rows.length; k++)
{
    var empty = Ti.UI.createView({
        width:10,
        height:50,
        touchEnabled:false,
        left:49,// + (rows[k].child*180),
        backgroundColor:'red'
    });

//    rows[k].children[1].add(empty);  
    rows[k].scrollView.add(empty);    // USE OF CUSTOM PROPERTY
}

    I hope this is solved your problem. Thanks..:)
