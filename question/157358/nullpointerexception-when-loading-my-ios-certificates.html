  • I'm not sure if this is a work-around but it seems to be working for me.

    1. In Overview pane of the TiApp editor, select the Titanium SKD (in the drop down) as: 3.1.3.GA (or anything just prior to 3.2.0.GA)

    2. In the studio terminal enter: titanium setup

    3. Then enter: sdk

    4. Then enter: 2 ( this should select the previous sdk)

    5. In Titanium Studio > Preferences, select Platforms > IOS (the certificates should now load)

    6. In Overview pane of the TiApp editor, select the Titanium SKD (in the drop down) as: 3.2.0.GA

    6 In Titanium Studio > Preferences (once more), select Platforms > IOS (the certificates should now still load)

    I hope this helps…..
