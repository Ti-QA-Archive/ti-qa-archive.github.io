Lazy load images

Hello,

Titanium SDK version: 1.6.1
iPhone SDK version: 4.2

How can I "lazy load" images into a table? When using for example the Titanium build getGlue the images are loaded as I scroll down the page. How is this done? I also see that they are using image placeholders, is this a feature of TI?

  • Accepted Answer

    The images by default are loaded as you scroll down the page. To use a "placeholder" image, use the defaultImage property of an ImageView. For my purposes, I use a 1x1 white image for my defaultImage so it looks as though the image just animates onto the view as it loads.
