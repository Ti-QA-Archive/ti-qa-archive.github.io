Update button text from optionDialog selection

Hi there

I am very new to Titanium so be gentle. I am creating a very simple app and want to update the title of a button with the selection made from an optionDialog. I am assuming I can do this but cant for the life of me figure out how. Any advice would be greatly appreciate. This is the code I have for the button and the optionDialog and it works as it should. I just need to update the button 'title' with the option selected from the optionDialog. So the button 'title' will then read either One, Two or Three…

//Create button Field
var button = Titanium.UI.createButton({
        width:395,
    title:'Select a Number',
        top:300
});

//Add button to currentWindow
scrollView.add(button);


var opts = {
    title: 'Category',
    options: ['One', 'Two', 'Three''],
    selectedIndex: 0
};

button.addEventListener('click', function(e){
  var dialog = Ti.UI.createOptionDialog(opts).show();
});

Thanks

    Hi

    Welcome, this should help you out;

    button.addEventListener('click', function(e){
    var dialog = Ti.UI.createOptionDialog(opts);
    dialog.addEventListener('click', function (e) {
        alert(opts.options[e.index]);
    });
    dialog.show();
});

    • button.addEventListener('click', function(e){
    var dialog = Ti.UI.createOptionDialog(opts);
    dialog.addEventListener('click', function (e) {
        // alert(opts.options[e.index]);
        button.setTitle(opts.options[e.index]);
    });
    dialog.show();
});

      Small change to the previous code, this uses the same response and then using the 'setter' updates the property text with the result.
