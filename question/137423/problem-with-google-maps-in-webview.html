Problem with Google Maps in WebView

Hello,
I want to display google maps via a webview. Here is the code

<!DOCTYPE html>
<html>
  <head>
    <meta name="viewport" content="initial-scale=1.0, user-scalable=no" />
    <title>Maps</title>
    <style type="text/css">
      html { height: 100% }
      body { height: 100%; margin: 0; padding: 0 }
      #map_canvas { height: 100% }
    </style>
    <script type="text/javascript"
      src="http://maps.googleapis.com/maps/api/js?key=MYKEY&sensor=false">
    </script>
    <script type="text/javascript">

      function initialize() {
        var myOptions = {
          center: new google.maps.LatLng(49.29344, 8.6418),
          zoom: 19,
          mapTypeId: google.maps.MapTypeId.ROADMAP
        };
        var map = new google.maps.Map(document.getElementById("map_canvas"),
            myOptions);

      }
    </script>

  </head>
  <body onload="initialize()">
      <h4>Maps</h4>
    <div id="map_canvas" style="width:100%; height:100%"></div>
  </body>
</html>

However, when I run this in my browser via an .html file outside Titanium, it works fine.
Inside my app in Titanium, I can only see the heading (so the .html is included correctly), but no map.

Any suggestions?

Thanks,
Martin
