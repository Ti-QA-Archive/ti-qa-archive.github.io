Close all the windows in a stack before a particular Window

Hi Folks,
After logging off my app,I need to close all the windows in the stack and open a new one(login window) so that the new one is the only one in the stack.One approach which I have thought of is to write a listener for all the Windows in the app and then fire event for each one of them.But is there any easier and cleaner way to do this?

    Hi Mukund, you can pass window as custom property and where you want to close all window you just make a loop to close a window.

    for example : 

    var parent = Ti.UI.createWindow({
    title : "parent",
    url : "parent.js"
});

var child = Tio.UI.createWinow({
    title  : "child",
    url : "child.js",
    _parent : parent
});

    at child you can access parent

    var win = Ti.UI.currentWinodw;

alert(win.parent.title);
