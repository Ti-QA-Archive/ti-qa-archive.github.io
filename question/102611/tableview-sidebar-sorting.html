  • You need to set the index attribute in the table with an array of objects with title and index keys.

    To get the magnifying glass, use {title: &#39;{search}&#39;}. The problem is that it shows the magnifying glass but it is not linked to the tableView's searchBar.
    If you are interested in such feature, please watch the ticket 2710.

    Also bear in mind that indexes don't work if the table doesn't have headers. See ticket 2652.
