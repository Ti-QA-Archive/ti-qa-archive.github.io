Having problems to make androzip module working

Hi all,

I wanted to test the androzip module (more infos here) but i'm facing some problems…

After downloaded the module on git, i builded it. Then, added the com.websiteburo.androzip module in /Library/Application Support/Titanium/modules/android/ and also added this lines in tiapp.xml : "<modules> <module version="0.1">com.websiteburo.androzip</module> </modules>"

When i want to test the module in my project, i've got an error : java.lang.IllegalArgumentException: size must be >= 0 (i've read that's because of the require() function, isn't it ?)

Some precisions : i'm on Mac OSX 10.6 with Titanium 1.0.4 (SDK version : 1.7.3) I've also a friend who has tested the module on Linux Ubuntu and got the error that the extract() function doesn't exist in (Ti.Androzip) (???).

So, what are we doing wrong ? Did we forget to do something ? Please help. Thx :)

  • The built zip file should be copied to the /Library/Application Support/Titanium folder. Titanium will extract the module to the correct modules subfolder when you build your project the next time (best after a clean).

    Hope this helps.
