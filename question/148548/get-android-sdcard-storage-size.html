Get Android SDCard storage size

Hi! I'm building an app that will store alot of data, and i need get the free sdcard's space,
how can i do this?

  • I found!
    the space comes as bytes

    the code is:

    var oneGigabytes = 1073741824;
var sdcardSpace = Ti.Filesystem.getFile(sdcard).spaceAvailable();
var spaceLeft = sdcardSpace/oneGigabytes;
