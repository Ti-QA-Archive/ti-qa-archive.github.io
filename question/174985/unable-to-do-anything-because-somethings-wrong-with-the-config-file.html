    On your mac, the titanium config is in $HOME/.titanium

    Only in extremes do I tend to mess with this file, but it can become corrupted, usually by 3rd party tools that add hooks or directories.

    I'd back it up before doing anything extreme. Nuke it, then from a terminal run ti setup and see if you can get a clean one rebuilt for you.

    Or you can delete this file and again restart titanium , problem will get solved.
