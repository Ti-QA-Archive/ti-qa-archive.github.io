    • This is a quick example that will work with or without the Alloy framework.

      Each of these two will take the data provided and flatten the values from the multiple nested branches from each contact. You can then use this with your option dialog.

      This version only returns the phone numbers.

      var contact = {
  phone: {
    work: [
      '123',
      '456'
    ],
    home: [
      'abc'
    ]
  }
};
var numbers = [];
for (var type in contact.phone) {
 for (var phone in contact.phone[type]) {
   numbers.push(contact.phone[type][phone]);
   //numbers.push(type + ': ' + contact.phone[type][phone]);
 }
}
Ti.API.info(numbers);

      This version also takes the email address into account.

      var contact = {
  phone: {
    work: [
      '123',
      '456'
    ],
    home: [
      'abc'
    ]
  },
  email: {
    home: [
      'abc@xyz.com'
    ]
  }
};
var choices = [];
for (var section in contact) {
  for (var type in contact[section]) {
    for (var entry in contact[section][type]) {
      choices.push(contact[section][type][entry]);
      //choices.push(section + ' ' + type + ': ' + contact[section][type][entry]);
    }
  }
}
Ti.API.info(JSON.stringify(choices, null, 4));

      The commented alternatives also include the captions to explain where the values came from/

      The contact object examples included in each above are based on contact structure you provided in the structure. Just change the references to match your own loops.
