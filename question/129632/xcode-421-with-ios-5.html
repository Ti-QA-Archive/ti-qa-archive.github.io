Xcode 4.2.1 with IOS 5

Actually I have Xcode 4.1 with IOS SDK 4.3 and Titanium mobile SDK 1.8.0.1.RC3

Is safe to upgrade Xcode to 4.2.1 with IOS 5 ?

Wich is the best upgrade approach to upgrade Xcode as long as now we have to use AppStore.

Thanks in advance

  • I've been running XCode 4.2 and iOS 5 with Titanium for the last 3-4 weeks without any issues. I would recommend using the last 1.8 release as it fixes an issue with pop-overs on iOS 5.

    You should just be able to upgrade XCode and then clean your Ti projects. This will force a re-build using the new version of XCode.
