iOS7 TableView with SearchBar and Ti.UI.EXTEND_EDGE_TOP does not position properly.

Running into the following problem when displaying a TableView with a SearchBar and the Ti.UI.EXTEND_EDGE_TOP property on the window.

The TableView no longer seems to add an inset (spacer) at the top so the searchbar and first row are shown under the navigation bar when the window is rendered:

If I take the search bar out, the tableview seems to automatically add the inset spacer, so that row 1 is shown below the navigation bar:

You can see the issue, by removing and added the searchbar markup in the xml. The searchHidden property does not seem to have an effect on this.

Here's the code:

index.xml

<Alloy>
    <NavigationWindow>
        <Window>
            <TableView id="tv">
                <SearchBar />
            </TableView>
        </Window>
    </NavigationWindow>
</Alloy>

index.tss

"Window": {
     layout: 'composite'
    ,backgroundColor:"#fff"
    ,extendEdges: [Ti.UI.EXTEND_EDGE_TOP, Ti.UI.EXTEND_EDGE_BOTTOM]
    ,includeOpaqueBars: false
    ,autoAdjustScrollViewInsets: true    
    ,navBarHidden: false
    ,barColor: '#156bbc'
    ,navTintColor: '#ffffff'
    ,translucent: true
    ,statusBarStyle: Ti.UI.iPhone.StatusBar.LIGHT_CONTENT
    ,title: 'Test'
}

index.js

var tableData = [];

for (var i=1; i<21; i++) {
    tableData.push({ title: 'Row ' + i });
}

$.tv.setData(tableData);

$.index.open();

Tested with iOS Simulator 7.

  • This seems like a similar issue to:

    http://developer.appcelerator.com/question/158466/setting-content-offset-for-tableview

    This seems to work:
    (note, the translucent titlebar image can be downloaded from here)

    var win = Ti.UI.createWindow({
  title:'TableView extend edges test', 
  backgroundColor:'white',
  barImage: '/images/titlebar_gradient.png',
  extendEdges: [
    Ti.UI.EXTEND_EDGE_TOP
  ],
  autoAdjustScrollViewInsets: true,
  statusBarStyle:Titanium.UI.iPhone.StatusBar.LIGHT_CONTENT
});

var table = Ti.UI.createTableView({
  height: '100%', 
  data: [
    {title:'row 1', header:'Header 1'},
    {title:'row 2'},
    {title:'row 3'},
    {title:'row 4', header:'Header 2'},
    {title:'row 5'},
    {title:'row 6'},
    {title:'row 7'},
    {title:'row 8'},
    {title:'row 9'},
    {title:'row 10'},
    {title:'row 11'},
    {title:'row 12'},
    {title:'row 13'},
    {title:'row 14'},
    {title:'row 15'}
  ]
});
win.add(table);

var navWin = Titanium.UI.iOS.createNavigationWindow({
   window: win
});
navWin.open();
