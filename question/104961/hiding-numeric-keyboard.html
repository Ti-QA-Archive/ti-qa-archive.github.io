  • Although the name may not be intuitive, the blur() method on a field object will close the keyboard. So in your button code do something like:

    myField.blur();

  • I place my form fields on a view. I then add an event listener on the view like this:

    // REMOVE FOCUS FROM TEXT FIELDS
viewMain.addEventListener('click',function(e){
    firstNameField.blur();
    lastNameField.blur();
    emailField.blur();
    passwordField.blur();
    passwordVerifyField.blur();
});

    I think this is a bit clunky, but it works. I'll streamline it later.
