createImageView on Click Event issue

Hello,

I'm having some trouble with createImageView. My app creates an image very very well indeed, I'm just trying to create an event so that a url or an html page is launched when someone clicks on presses on that image. How do I do this?

var av = Ti.UI.createImageView(
            {

                image: Var_picture.replace(/^\s+|\s+$/g, '') || 'images/myimage.jpg',
                left: 0,
                top: 0,
                height: 110,
                width: 110,
                defaultImage: 'images/default.png',
                backgroundColor: '#000',
                touchEnabled: true
            });

What do I add? I tried eventListener, with no luck.
THanks

    • If you're trying to load a window, you need to create the window inside the event listener.

      img1.addEventListener('click', function() {
    var window = Ti.UI.createWindow({
        modal: true,
        url:'computersystems/computersystems.js'
    });
    window.open();
});

      There's some really great examples in the KitchenSink that you can look at and play with too that show how to do this and some more advanced stuff.
