How to run an application through Titanium Studio on older iOS Simulator version?

Hi. The question in its header.

I can't found a way to run application through Titanium Studio on older iOS Simulator version. Just choose older iOS sdk into Titanium "Run Configuration…" settings is wrong, because as I know SDK version setted must be always latest one from available. Simulator menu "Hardware" -> "Version" not helps in this – after choosing 4.3 version simulator closes and after recompiling and second run you again get 5.1 version of iOS. Fortunately switching to Retina works for testing, but not switching the iOS version.

Only way what I found to run on older simulator version is to run the app directly from XCode. This is only way to do this?

  • Accepted Answer

    Go to ~/Library/Application Support/iPhone Simulator/5.1/Applications and find the directory of your app.
    Copy it to ~/Library/Application Support/iPhone Simulator/4.3/Applications, run the iOS Simulator and switch to iOS 4.3.
