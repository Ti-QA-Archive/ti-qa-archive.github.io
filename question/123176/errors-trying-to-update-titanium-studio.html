  • Accepted Answer

    Make sure you have at least one update site checked in the Preferences Window.

    1. Select the Window -> Preferences menu.
    2. From there, Select the Install/Update -> Available Software Sites branch on the left.
    3. Select the update site depending on your needs. (usually Titanium Studio Stable Updates) at URL http:&#x2F;&#x2F;download.appcelerator.com&#x2F;appcelerator&#x2F;studio&#x2F;standalone&#x2F;update&#x2F;stable&#x2F;
    4. Re-try Update.

  • This helped me… https://wiki.appcelerator.org/display/tis/Errors+While+Updating

    "No updates were found"
    If there is a new version of Studio available but your current Studio shows "No updates were found" when you select Help > Check for Updates, you could try the following:

    Open Preferences > Install/Update > Available Software Sites and make sure the appropriate "Release Updates" entry is checked and enabled for what you want to install.
    Select the entry and click Reload to make sure it returns success.
    Try Help > Check for Updates again.

    Ensure you are clicking the reload button…
