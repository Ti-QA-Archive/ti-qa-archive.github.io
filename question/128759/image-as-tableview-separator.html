Image as TableView separator

Hello!

Is it possible to use an image as a separator in a tableview? If so, how?

  • on iOS, you set the table like so

    tavbleView.separatorStyle = Titanium.UI.iPhone.TableViewSeparatorStyle.NONE;
tavbleView.separatorColor = 'transparent';

    and then just add the imageView to the row as the last item, then you have it

    var separator = Titanium.UI.createImageView({
   left : 0,
   bottom : 0,
   width : 320,
   height : 1,
   touchEnabled : false,
   image : "THE IMAGE URL"
});
row.add(separator);
tableView.appendRow(row);
