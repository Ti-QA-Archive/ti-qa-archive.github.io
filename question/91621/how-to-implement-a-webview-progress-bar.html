How to implement a WebView progress bar

I am trying to implement a progress bar for a WebView, similar to what we have on Traditional browsers, showing how much is left for a webpage to load.

Does anyone tried before?

    Not possible, unless you're actually downloading the content locally to display.

    The only events we have for webViews are 'beforeload' (if you want to show a custom spinner while loading, for example) and 'load' to remove it once page is loaded.

    To do what you want, we'd need an ondatastream event with a progress property, which is only available in the HTTPClient module, used for communicating with a server and uploading or downloading data of files.
