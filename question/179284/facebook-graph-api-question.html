    for e.g this string stored in obj variable

    if you want to access all batter content then you fetch like this

    var arrBatter = obj.batters.batter;

for(var i=0;i<arrBatter.length;i++){
    Ti.API.info('Type = ' + arrBatter[i].type);
}

    Regular
