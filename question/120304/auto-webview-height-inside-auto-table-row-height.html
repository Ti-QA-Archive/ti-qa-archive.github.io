Auto webview height inside auto table row height.

I am trying to have my table row height automatically set with the auto height of my webview but the height auto of the table row seems null or something because it's giving me default.

how can I fix this?

I'm developing on an iPad2

var row = Ti.UI.createTableViewRow({height:'auto'});

var blog_post = Ti.UI.createWebView({
    html:blog_body,
    left:72,
    top:5,
    bottom:5,
    right:5    ,
    height:'auto'
});

row.add(blog_post);

  • For whatever its worth… It would not be recommended to have many web-views in a table view… web views and tables are memory intensive… they are just fine to have one, or a small number… but the more you add the slower the app will run. This holds true not just for Titanium, but native development too.

    To get the height though, you would just need to do something like:

    row.height = blog_post.height;

    after the webview was loaded using a load event listener (before its loaded the height would be 0)
