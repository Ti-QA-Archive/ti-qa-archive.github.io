Page Load Activity

Hi,
i'm facing the following problem, i'm writing an iOS app that loads the content of almost every page from xml files and put it in table views with custom tableViewRows.
This operation seems to be expensive and i'm facing some delays (in the order of one or two seconds depending on the number of the rows).I had a look in the window API looking for a onload event to put an activity indicator but i did'nt find it helpful.There is someone has a solution to this kind of problems?Thank you very much!!

  • Use window "open" event:

    var win = Ti.UI.createWindow();
actInd.show();
...
//expensive operation
...
win.addEventListener('open', function() {
    actInd.hide();
});

win.open();
