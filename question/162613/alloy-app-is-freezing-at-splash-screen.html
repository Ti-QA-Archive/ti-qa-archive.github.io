  • This might help…

    Tim is almost there. Look at the index.js file nothing is actually being called when the app launches so it hangs at the splash screen. I got it working by adding $.navGroupWin.open(); after the book.save command, then adding <NavigationWindow id="navGroupWin"> before the Window tag in index.xml (remember to close the NavWindow tag too), so in index.xml you should have:

    //index.xml
<Alloy>
    <Collection src="books"/>
    <NavigationWindow id="navGroupWin">
        <Window class="container" title="My Books">
            <TableView dataCollection="books" id="bookTable">
                <TableViewRow title="{title}" author="{author}"></TableViewRow>
            </TableView>
        </Window>
    </NavigationWindow>>
</Alloy>

//index.js
var myBooks = Alloy.Collections.books;

var book = Alloy.createModel('books', { 
   title : 'Great Expectations', 
   author: 'Charles Dickens' 
});

myBooks.add(book); 
book.save();

 $.navGroupWin.open();

    I'm only using iOS sim, but it works fine.
