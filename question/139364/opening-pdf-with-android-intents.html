  • Hi Ronnie,

    You can open pdf in webview in android using this code just place your remote PDF Url in url variable.

    var finalUrl = 'http://docs.google.com/viewer?embedded=true&url='+ url ;
        var webView = Ti.UI.createWebView({
            left : 0,
            url : finalUrl,
            top : 10,
            width : 320,
            height : 350,
        });
win.add(webView);
