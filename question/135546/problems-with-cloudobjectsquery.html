Problems with Cloud.Objects.query

function loadData(student_id) {
        Ti.API.info(student_id);
        Cloud.Objects.query({
            classname: 'Students',
            page: 1,
            per_page: 10,
            where :{
                id:student_id
            }
        }, function (e) {
            if (e.success) {
                  Ti.API.info(e)
            } else {
                alert('Error:' +
                    ((e.error && e.message) || JSON.stringify(e)));
            }
        });

This returns 0 results, I've verified that the object's ID is correct. I've been able to query using where on this object using other values, but not the ID.

Using 2.0.0 with OSX Simulator
