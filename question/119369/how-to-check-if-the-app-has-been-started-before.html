How to check if the app has been started before?

Is there any way to check if the app (iPhone) has been started before and hasn't been closed by the user?

I have an image that I want to show only the first time the app is started, just after the splash screen is shown.

    You can use these two things to determine which kind of start the app is experiencing:

    • true start after app is shut done: app.js is executed.

    • relaunch after being put into the background: The resume event is fired.

  • You can set a property on the first run of the app that will be a flag if the app is launched for the first time or not. If this property exists, then it means you don;t have to show your image.

    Read more about here:

    http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.App.Properties-module

    This properties are persistent across relaunches and even reboots.

  • app.js:

    if (Ti.App.Properties.getBool('hasStarted') == false) {
    // Show image
    Ti.App.Properties.setBool('hasStarted', true);
} else {
    // Don't show image; Start App
}

    If you want it to reset everytime you update the app, add the App Version the bool, like so:

    if (Ti.App.Properties.getBool('hasStarted'+Ti.App.getVersion()) == false) {
    // Show Image
} else {
    // Start App
}
