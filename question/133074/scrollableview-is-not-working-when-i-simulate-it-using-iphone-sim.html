ScrollableView is not working when i simulate it using iphone sim

var win = Ti.UI.currentWindow;

win.backgroundImage = 'Laogo.png';
var handMade = Ti.UI.createView({backgroundImage:'hand.png'});
var natural = Ti.UI.createView({backgroundImage:'natural.png'});

var scrollView = Ti.UI.createScrollableView({
views:[handMade,natural],
showPagingControl:true,
clipViews:false,
top:180,
left:30,
right:30,
height:180,
opacity:0
});

win.add(scrollView);
win.open();

I create a new window in the app.js, and set url parameter to the .js file above. But i dont understand why it will not scroll through the 2 images i have in the scrollview when i run the program!?!?!?!!?
