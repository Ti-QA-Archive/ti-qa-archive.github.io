Cannot Click Facebook Button into a TableView Row

When I try to add Facebook button to the table view row. It is visible but I cannot click on it, adding it to the window like kitchensink works.

Problem only with iOS. Works fine with Android.

Appreciate any help. 

var  fbRow = Ti.UI.createTableViewRow({
});

var fbLabel = Ti.UI.createLabel({
    text: "Facebook",
    left: 12,
});
fbRow.add(fbLabel);


var fbButton = Titanium.Facebook.createLoginButton({
    right: 10,  
});

fbButton.row = fbRow;
fbRow.add(fbButton); 
tableview.appendRow(fbRow);

    Add an empty event listener to the tableview and it will work. It is kind of a weird workaround, but what is happening is that the tableview is receiving the click event and not propagating it down to the button. It seems, when the tableview has it's own event handler, then it knows which events it should keep and which it should pass on.

    Add the following line:

    tableview.addEventListener('click',function(){});
