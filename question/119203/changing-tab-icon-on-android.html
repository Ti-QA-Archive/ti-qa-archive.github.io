Changing Tab Icon on Android

Is it possible to change Tab Icons in Android? The Kitchen Sink makes it look easy, but since I can't get that to run….

Here's my code. Using trace statements I can see that it's getting executed, but the icons never update. What am I missing???

tabGroup.addEventListener('focus', function(e) {
    //  Reset the Icons
    tabSchedule.icon = 'images/busIcon.png';
    tabMap.icon = 'images/mapIcon.png';
    tabFAQ.icon = 'images/faqIcon.png';
    tabMore.icon = 'images/moreIcon.png';

    // Set the icon for the selected tab
    switch(e.index) {
        case 0:  // Schedule
            tabSchedule.icon = 'images/busIconActive.png';
            break;
        case 1:  // Map
            tabMap.icon = 'images/mapIconActive.png';
            break;
        case 2:  // FAQ
            tabFAQ.icon = 'images/faqIconActive.png';
            break;
        case 3:  // More Info
            tabMore.icon = 'images/moreIconActive.png';
            break;
    }
});

  • I have had problems with the tabgroup not responding to eventlisteners. If it doesn't work (try to show an alert or something) you can work around it by triggering a custom event on every tab-page. On the app.js you can add an eventlistener so it can switch the tabIcon every time another tab is opened.

    Perhaps not very effective but it works great for me. If anyone would know why the eventListener doesn't work please let it know!

    Hope this helps you a bit.
