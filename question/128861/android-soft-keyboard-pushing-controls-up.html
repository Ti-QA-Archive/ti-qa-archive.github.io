  • Accepted Answer

    Hi Adam

    I believe you put your controls into a view. If yes, try to set the height of the view.
    If no, try set this at the time of window creation

    
    windowSoftInputMode : Ti.UI.Android.SOFT_INPUT_ADJUST_PAN

    To avoid such behavior the only solution i am aware is not to have any TextFeild in the later Half of the screen(keypad view area).
