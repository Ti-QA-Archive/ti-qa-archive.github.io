iPhone3 and iPhone3S support

In Titanium SDK, it shows iOS6.1 while build, will this support iOS5.0 on older version of iPhone3,3GS?

    Hi

    You should always use the latest version of the Titanium SDK to give you the maximum compatibility.

    Taking note of that; this means your apps will work on iOS 4.3 or higher. If you have any version specific iOS features you can include but make sure they fail back gracefully.

    As iOS 4.3 works on all iPhones since the 3GS - you can expect it to work with little or no issues. Apple themselves now ONLY support the 3GS or higher.

    If you want to support earlier devices, you will need to use an earlier titanium SDK and combined with an earlier xcode SDK as the current versions ONLY support iOS 4.3 or higher. Still possible but a bit more work for you to do.

    The compatibility chart @Paul references is the right place to refer too.
