    Hi Iacopo,

    you have two problems:

    1. nodeJS delivered all modules in little moduls, Titanium uses one modul. It is an other callaing syntax. With Titaniumifier you can convert. The tools uses grunt.
    2. nodejS is Javascript. It is same as Titanium. But nodeJS uses „native“ modules too. i.e. 'net'. This module is not pure javascript and you cannot use in Titanium hosting.

    Hope I can help you

    Rainer
