Image Dimensions Android without opening a window

Hi there - I'm aware of the techinique that's been suggested to measure an image in Titanium, which involves creating an ImageView with an image, then adding it to a window, then opening he window, then measuring the image using imageView.toImage().

The problem I have with this is that it actually opens a window - even closing the window immediately leaves the image displayed for a second or so. I have an aplication where I need to display various images at their original dimensions. It's not acceptable for this solution to have the image pop up breifly whenever I measure the dimensions. Is there an alternative to this method - or some way to open the window off screen?

My current measure image code:

var map.measureImage = function(imageName) {
  var win = Titanium.UI.createWindow({
    backgroundColor:'#fff',
    layout:'vertical',
    exitOnClose:true
  });

  var image = Ti.UI.createImageView({
    image:imageName
  });

  win.add(image);

  win.open();
  var dim = {width: image.toImage().width, height: image.toImage().height};
  win.close();
  return dim;
}
