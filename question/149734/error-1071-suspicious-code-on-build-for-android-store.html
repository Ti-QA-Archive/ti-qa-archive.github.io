Hello,
Just about ready to submit (iOS seemingly OK) and on Android App Store build I get this:
[WARN] /<path to folder> Android/build/android/bin/assets/Resources/alloy/controllers/index.js:1071: WARNING - Suspicious code. The result of the 'ne' operator is not being used.
flag != 0;
^

TI 3.0.1 and using SDK 2.1.3

    This is a warning given by the closure compiler when an Andoid app is built for production. As it says, its only a warning and the closure compiler has found some code that appears to do nothing. I suspect this is just a side-effect of the alloy framework and how it produces templated code. I wouldnt worry as long as your app functions as designed. See this stackoverflow post for more info
