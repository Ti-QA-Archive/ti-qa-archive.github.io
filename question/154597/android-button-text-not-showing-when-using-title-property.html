    Martin,

    Hmm, the app works for me:

    A couple of thoughts:

    • Do you have entries in your i18n/en/strings.xml file for the button label values (<string name="capture">Capture</string>)? Try just putting a string for the value of the title property of the buttons rather than the L() function to be sure your buttons are working as they should.
    • Have you tried setting an explicit color for the buttons (default is gray text so it might not be showing up)?

    The default color scheme on Android is a bit dull. You can set backgroundColor and other properties to pretty them up. But the title should work either way.
