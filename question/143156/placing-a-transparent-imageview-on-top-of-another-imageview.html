Placing a transparent ImageView on top of another ImageView

I have a semi-transparent PNG file that I am trying to overlay on an existing image. Here's what I am trying:

var overlay = Ti.UI.createImageView({
  image: 'overlay.png'
});

var image = Ti.UI.createImageView({
 image: 'actual_image.png'
});

image.add(overlay);

I then add image to the window. However, only the image specified in image gets displayed and overlay is not displayed at all.

I have also tried setting the zindex property of overlay to be a large number but that didn't help.

Any suggestions?

  • Accepted Answer

    Set the actual image to the background image of a regular view. Then overlay the transparent png by adding it to the view.

    var image = Ti.UI.createView({
  backgroundImage:'actual_image.png'
});
var overlay = Ti.UI.createImageView({
  image:'overlay.png'
});
image.add(overlay);
