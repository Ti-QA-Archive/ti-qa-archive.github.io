  • Hi

    Ti.UI.currentWindow

    Only works if the app you are creating uses the url property of the window object. This is not something you will be using with Alloy and in fact the url property usage is discouraged in favour of better techniques.

    However it does mean you need to keep a handle on which window you are on. This can be done by passing references or events.
