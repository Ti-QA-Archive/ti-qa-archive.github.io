  • var intent = Ti.Android.createIntent({
        action: Ti.Android.ACTION_SEND,
        type: 'text/plain'
    });
    intent.putExtra(Ti.Android.EXTRA_SUBJECT, 'My subject!');
    intent.putExtra(Ti.Android.EXTRA_TEXT, 'My text!');
    try {
        Ti.Android.currentActivity.startActivity(intent);
    } catch (ex) {
        /* Handle Exception if no suitable apps installed */
        Ti.UI.createNotification({ message : 'No sharing apps installed!' }).show();
    }

    You need to do something like that in android it will show you all email apps installed in your phone. You can also add pre filled email address in "EXTRA_EMAIL" which requires an array not a single string.
