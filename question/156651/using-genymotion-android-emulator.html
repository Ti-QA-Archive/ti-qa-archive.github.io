Using Genymotion Android Emulator.

Not a question, so much as a tip…

I find the Eclipse Titanium development environment to be among the worst development environments I've used in 25 years of development, especially if you couple it with running debug apps in the standard Android eumlator. That said, it is MUCH easier to use a good Javascript dev environment (I use WebStorm, it's awesome), the command line for builds, and the Android monitor for Android, and the command line console for IOS, and Genymotion of Android emulation.

The end result is that I can edit much more efficiently than I can in Eclipse, I can find bugs more easily by watching generated exceptions in the monitor, and I can more easily control my build process for both emulators and devices from the command line.

And best of all, I can run my app on the blazing fast Genymotion emulator.
