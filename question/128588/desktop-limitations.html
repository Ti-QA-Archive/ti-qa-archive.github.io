  • Accepted Answer

    Titanium Desktop does have printer support, but it is basically a single function that will pull up the printer dialog on a system through the webkit. I have not used it to date, so not sure how well it works, but my experience in the past working with printers says that it is sometimes difficult to get the output exactly as you want, and requires dwiddling to get how one wants it. I don't know how difficult that would be or if it would work well as-is, since I haven't used it much at all.

    The webkit has some graphics capability, such as SVG. Currently we use 3 flavors of the webkit on osx/linux/win so that type of graphics could take some tweaking to get correct also.

    Any devices you might want to talk to that are not available would need a module written for them.
