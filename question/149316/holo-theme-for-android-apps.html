Holo Theme for Android Apps

I am trying to figure out how to use the holo theme for android devices. I have tried several different ways and they either change nothing or break the application.

Site 1: The first I tried was the first response to this thread: http://developer.appcelerator.com/question/130719/how-do-you-build-your-app-with-holo-theme

site 2: As well as this http://developer.appcelerator.com/question/140548/using-themes-and-fallback-themes-in-android

For the first one i added the code in the first code box to my tiapp.xml file and created the directories and files for the other code boxes, without the extra themes.xml files the app would start but stop because if could find the resource specified in the tiapp.xml, add the extra files and folders and i get this.

Application Installer abnormal process termination. Process exit value was 1
[ERROR] Error generating R.java from manifest

and this for the site 2

[INFO] logfile = /Users/chadgregory/Documents/Titanium_Studio_Workspace/Holo App Test/build.log
[ERROR] Application Installer abnormal process termination. Process exit value was 1

I am also updated to the latest version on the SDK.

Does anyone know how to do this?

    this code in tiapp.xml worked for me:
    no need for making a custom AndroidManifest.xml, it falls back to the default theme on 2.x automatically 

    <android xmlns:android="http://schemas.android.com/apk/res/android">
        <tool-api-level>17</tool-api-level>
        <manifest android:installLocation="preferExternal"
            android:versionCode="3" android:versionName="1.0.2">
            <uses-sdk android:minSdkVersion="7" android:targetSdkVersion="17"/>
            <application android:theme="@android:style/Theme.Holo">
                <activity
                    android:configChanges="keyboardHidden|orientation"
                    android:name="org.appcelerator.titanium.TiActivity"
                    android:screenOrientation="portrait" 
            android:theme="@android:style/Theme.Holo"/>
            </application>
        </manifest>
    </android>
