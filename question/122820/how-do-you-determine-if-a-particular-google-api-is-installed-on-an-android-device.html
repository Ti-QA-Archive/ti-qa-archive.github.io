How do you determine if a particular google api is installed on an android device?

I put an app Im working on on a unit without google maps, and it crashed. I discovered it was due to the unit not having google maps api installed. Is there a clean way to pre-test for that sort of thing?

  • I believe in the Applications manifest file, an app that uses the google map api needs to declare that it uses the library:

    
<application android:name="MyApplication" >
    <uses-library android:name="com.google.android.maps" />
</application>

    This should prevent your app from being installed on a device that does not have the apis installed.
