    Hi Deepan ,
    The best way is to use the array splice method. Like this

    resultnumber.splice(1,1,{'title':'Titanium Studio'});

    The splice method has three arguments
    1.) the index(in your case it would be 1) from which you want to remove or add to the array.
    2.) Number of elements you want to remove.
    3.) This one is optional- The elements you want to add to the array.
