  • To fix it there are two things that must be done (from my testing). This works on Windows 7 and Mac OS X 10.6.

    1. Within Titanium Studio bring up the "TiApp Editor" and be sure to set up the correct naming for Application Id: com.<Company>.<App Name> – example com.Company.demoapp

    2. Go to your Workspace for the Titanium project you are working on within your computer (Mac Finder or Windows 7 File Explorer) and place the App icon in the following folder: <path to workspace>/Titanium Studio Workspace/<project name>/build/android/bin/assets/Resources (may need backslashes \ for Windows). I copied the "appicon.png" into that folder and you may have to create that directory.

    3. Re-build/Run the application and be patient for it to load the icon into your emulator (it may take up to 40+ secs).

    4. Done.
