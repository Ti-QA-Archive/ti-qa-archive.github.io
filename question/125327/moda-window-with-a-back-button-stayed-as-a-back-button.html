Moda Window with a back button, stayed as a back button.

For the iPhone/iPad, is there any way to have a window with a back button styled as a back button without having the Window in a Tab/TabGroup?

I just want to have a simple modal window that I want to pop up and then provide a back button in the NavBar to go back. I don't want to have to set up a TabGroup and Tab and then figure out how to hide them just to get a Back button to appear in a window… Or am I missing something fundamental here?

    You just need to assign a properly styled button to the leftNavButton property of your window:

    var win = Ti.UI.createWindow({
    // style your window
});
var button = Ti.UI.createButton({
    // style the button to your liking    
});
button.addEventListener('click', function(e) {
    // handle your navigation when the button is clicked
});
win.leftNavButton = button;

  • Found a solution.

    1. You can only get a real pointy back button if the window is part of a tab-group. Not sure why this limitation is in place, but there you go.

    2. Second issue is that you are limited in the way you can style the button in the Navigation bar.

    The solution is to add a ButtonBar instead of a Button to the NavBar

    var navBackButt = Ti.UI.createButtonBar({
  labels:           ['Back'],
  backgroundColor:  '#ae4041',
  color:            '#ffffff'
});

win.setLeftNavButton(navBackButt);

    I found many people with the same issue.

    Still if you want to style the back button as a classic back button, you need to use an image. Seems like a weird restriction - but there you go.
