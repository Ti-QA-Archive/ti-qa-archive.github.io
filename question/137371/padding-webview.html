Padding WebView

web_content = Ti.UI.createWebView({
            top : 100,
            left : 0,
            color : "#2D2D2D",
            font : {
                fontSize : 14,
                fontFamily : 'Arial'
            },
            html : css,
            width : 'auto'
        });

When I add this to a window, the web view has padding to the left, and right sides of about 10px I think.
I need to handle padding myself, as I need to place an image edge to edge on the web view.

Any ideas how I can acheive this, or what is causing the padding.

— asked May 24th 2012 by Steven Marshall

  • Try forcing the view to the edges like this;

    web_content = Ti.UI.createWebView({
            top : 100,
            left : 0,
            right: 0,
            color : "#2D2D2D",
            font : {
                fontSize : 14,
                fontFamily : 'Arial'
            },
            html : css
        });
