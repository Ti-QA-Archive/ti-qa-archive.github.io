  • Sometimes Titanium does not detect XML module. That should be fixed. I'm not sure what version of SDK do you use, but I fixed this issue by adding explicitly XML modules to list of module dependencies. Go to "Application Data\Titanium\mobilesdk\win32\1.7.5\android", open "dependency.json" and modify this line

    "required": ["analytics","android","ui","json","locale","xml"],
