Sticky footer

Hello, I am trying to achieve a sticky footer in a non-tabbed window, I have read through the forums and people suggest adding a view with bottom: 0, however this is not working for me, it just places it beneath the other elements in a non-sticky fashion.

Could anyone please point me in the right direction, please?

Many thanks!

  • Hi

    The actual answer will depend on how you have coded the rest of the items and if you are using something like a scrollView. But the key here is to have a container that everything goes inside and it must have a bottom value set to the height you need to set aside. Then create the footer and set its height to the same value as the space you left in the containers bottom property.

    This is a quick example if you are using classic Titanium, the key settings are commented below.

    var win = Ti.UI.createWindow({
    height: Ti.UI.FILL,
    width: Ti.UI.FILL
});
var scrollView = Ti.UI.createScrollView({
    bottom: 50, // leave space
    contentHeight: Ti.UI.SIZE,
    contentWidth: Ti.UI.FILL,
    height: Ti.UI.FILL,
    layout: 'vertical',
    showHorizontalScrollIndicator: true,
    showVerticalScrollIndicator: true,
    width: Ti.UI.FILL
});
win.add(scrollView);
var view1 = Ti.UI.createView({
    backgroundColor: 'red',
    borderRadius: 10,
    bottom: 10,
    left: 10,
    right: 10,
    top: 10,
    height: 300,
    width: Ti.UI.FILL
});
scrollView.add(view1);
var view2 = Ti.UI.createView({
    backgroundColor: 'green',
    borderRadius: 10,
    bottom: 10,
    left: 10,
    right: 10,
    top: 10,
    height: 300,
    width: Ti.UI.FILL
});
scrollView.add(view2);
var fixed = Ti.UI.createView({
    backgroundColor: 'blue',
    bottom: 0,
    height: 50, // match space from above
    width: Ti.UI.FILL
});
win.add(fixed);
win.open();
