Bug in Tableview

It seems that you have to call:

tableview.moving = true;

before 

tableview.editing = true;

otherwise moving won't appear.

So this is correct:

tableview.moving = true;
tableview.editing = true;

this is incorrect:


tableview.editing = true;
tableview.moving = true;
