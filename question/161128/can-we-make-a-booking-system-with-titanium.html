  • Hi

    As Titanium is a full software development system - you can create any type of app you are capable of creating. It supports the vast majority of views/controls can store data, call APIs, receive input - all the things you would need.

    You do need to be a programmer - have a basic understanding of JavaScript (which will need to get better as you go). You will also need to understand how each phone UI works by having actual experience with each so your users do not get confused.

    Titanium will not create apps for you - but it will allow you to create apps with it.
