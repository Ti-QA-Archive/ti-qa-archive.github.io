app boot sound?

Hi, I am very new to appcelerator, still learning the basics and I was wondering if it is possible to play a small sound on bootup after the splashscreen. This will be on the iOS SDK by the way. It would also be nice if you could share some example code. Thx. :)

    Hi Daniel,

    A great place to start on something like this would be the kitchen sink, it has some great examples for stuff just like this.

    That said, creating a boot sound is quite straight forward.

    I would do something like this in my bootstrap file. So for starting out or a really simple app you could place this in your app.js.

    var sound = Titanium.Media.createSound({
    url: 'ding.wav'    
});
sound.play();

    Obviously make sure that your sound file is in your "Resources" folder somewhere and the url points to it. Using createSound works great for local small sounds but it does not work well for remotely hosted files.

    I hope this helps.
