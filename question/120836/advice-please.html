  • If you are dealing content within with windows…

    Views are like divs. Check out the parameters here:
    http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.UI.View-object.html

    Labels can be used like spans, paragraphs, etc. They are essentially a block or link of text.
    http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.UI.Label-object

    Once you get an understanding of those two you should be good for any type of text design (more complicated versions at least).

    For what you are talking about with info pages, I like to place these pages on a server and then set the CSS to look just like an iPhone/Android application. I just open a webview for the info page. Using this setup it gives me the freedom to make changes after the app is in the app store.
