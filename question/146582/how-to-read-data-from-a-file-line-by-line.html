How to Read data from a file line by line

Hi,
I want to create an application which reads data from a text file line by line. Each lines contains a unique string and I want to move that string to a database. But I couldn't get any methods to read data from file line by line. I used the following code to read data from file but it reads all data and return it while I want each line to be returned

function readFile(){
    var fileName = txtName.value;
    file = Ti.Filesystem.getFile(Ti.Filesystem.resourcesDirectory, fileName);
    var blob = file.read();
    var readText = blob.text;
        //Adding the file content to the label placed
    lblFileContent.setText(readText);
    Ti.API.info(readText);
    // dispose of file handle & blob.
    file = null;
    blob = null;
}

Can anyone tell what I'm doing wrong here? Is it possible to read each line in Titanium? Is there any method for that? Can anyone help me please.
