Google Maps V2 in Android: Empty Map View

SDK 3.1.0.GA,
T-Mobile G2 device

I created debug and release API Keys using the Google Console, installed the maps module successfully, added the API Keys to tiapp.xml, and now when I create a map view it is empty. The only things visible are the zoom controls and the "my location" button. I have Google Play installed on this device (which is my personal phone) and I am assuming this means that the "Google Play Services" are installed. Could someone verify that this is the case?

  • The SHA1 for Titanium 3.1.0.GA is CC:E3:7F:08:FA:03:9C:88:07:BC:CB:AB:7B:88:61:F4:75:9D:47:9F

    You obtain it by executing :

    keytool -list -v -keystore ~/Library/Application\ Support/Titanium/mobilesdk/osx/3.1.0.GA/android/dev_keystore

    You should have declared in your android console :

    CC:E3:7F:08:FA:03:9C:88:07:BC:CB:AB:7B:88:61:F4:75:9D:47:9F;com.zerowastehome.bulk

    Are you sure the com.zerowastehome.bulk matches the id specified in your tiapp.xml ?

    <id>com.zerowastehome.bulk</id>
