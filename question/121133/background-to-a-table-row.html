Background to a table row?

How do I add a background to a row in a tableview?

    Setting the background image of a TableViewRow should look something like this:

    var tableViewRow = new Ti.UI.createTableViewRow();

tableViewRow.backgroundImage = 'images/myimage.png';

    or set it during instantiation like this:

    var tableViewRow = new Ti.UI.createTableViewRow({
     backgroundImage: 'images/myimage.png'
});
