  • I know this topic is nearly one year old, but for those who are still searching for the answer:
    You have to create a tab group with only one tab and hide the tab bar in the windows constructor.

    Example:

    var tabGroup = Titanium.UI.createTabGroup();

var win = Titanium.UI.createWindow({
  title:'My Window',
  tabBarHidden: true
});

var tab = Titanium.UI.createTab({  
  title:'My Window',
  window:win
});

win.add(label1);
tabGroup.addTab(tab);
tabGroup.open();
