How to go from Photoshop design to Titanium?

I've looked on the web and around the forums here but couldn't find a clear answer on how to go from a design in photoshop to design a titanium app. So let's say we are using a GUI kit like http://www.rawapps.com/849/ipad-gui-kit-in-psd-format-is-here/

How do you go about designing the app in Titanium, do you start by creating default UI controls like a Titanium.UI.createTextArea and then apply e.g. Titanium.UI.TextArea.backgroundImage which is a PNG exported from the Photoshop?

Is that the way to go or is there a better way? I've read something about jss, but I've also read that titanium is dropping support for that. I'm new to Titanium so forgive any stupid questions please.

Thanks for helping out!

    > do you start by creating default UI controls like a Titanium.UI.createTextArea and then apply e.g. Titanium.UI.TextArea.backgroundImage which is a PNG exported from the Photoshop?

    Yes, you basically hit the nail on the head. If you have a completely custom interface, slice your PSD up and use the PNG images as ImageViews or BackgroundImage's for Views, TextFields or whatever field you happen to need. You should of course use native components wherever possible (Nav-bars and TabBars for example).
