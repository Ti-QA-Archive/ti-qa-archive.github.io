Open webview in modal window from within webview

I have a web app that we are converting to a native app.

With in the app there are links to view a pdf, what I'm trying to do is when clicking the link the pdf opens in a modal window, I tried doing this with a div containing a iframe that had the pdf but the scrolling wasn't intuitive.

So what i have is is in my app.js

Titanium.App.addEventListener('viewPDF', function(e) {
    var pdfView = Titanium.UI.createWebView({url: e.url});
    var pdfWin = Titanium.UI.createWindow({modal: true});
    pdfWin.add(pdfView);
    pdfWin.open();
});

Then i have a page that is loaded in via ajax a few pages deep that has the following javascript on it. (this page is loaded into the main window, which is just a webview)

$("#viewPDF").click(function(){
    Titanium.App.fireEvent('viewPDF',{url: 'pdfs/ef23134.pdf'});
});

When clicking on the viewPDF element nothing happens. Not sure why not.
