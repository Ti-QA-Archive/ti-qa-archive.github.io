    You need to be using TiSDK 3.2.2 for ios 7.1 support.

    — answered April 4th 2014 by Stephen Feather
    permalink
    1 Comment

    • from the command line, first check to see if you have 3.2.2 installed

      ti sdk

      will give you a list of the installed sdks. If 3.2.2 is listed, then change your tiapp.xml. If it is not listed, then do

      ti sdk install

      Will probably still need to edit your tiapp.xml
