  • They will animate in android automatically IF (and it seems only if…) the new window is a "heavyweight" window. That means one of the following is true when the window is created:

    modal
    navBarHidden
    fullscreen
    windowSoftInputMode

    However, you don't appear to be able to change the animation. Android is not so good for that kind of stuff.
