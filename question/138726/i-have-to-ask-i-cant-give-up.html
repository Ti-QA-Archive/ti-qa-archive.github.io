  • Accepted Answer

    The Desktop-SDK root path /users/johnbexxxxxx/library/applicationsupport/titanium/sdk/osx' is not a directory

    This is really a PITA. The Desktop SDK was discontinued, but the install still requires it. Many people post here in these forums with this problem.

    Look at http://developer.appcelerator.com/question/129786/could-not-locate-a-titanium-desktop-sdk-at-the-given-path.
