Android - how to search for all files on SD card with given file extensions and display them in an OptionDialog?

I was wondering, does anyone know how I can search for all files on the SD card with given file extensions and display them in an OptionDialog? I would like to search for all files on the sdcard with an .mp3, .m4a, .aac, .wav, .aif, or .aiff extension and display them in an OptionDialog. Does anyone know how?

  • You will achieve it using methods of Titanium.Filesystem, as

    • isExternalStoragePresent();
    • getExternalStorageDirectory();
    • getFile(String path);

    and Ti.Filesystem.File's:

    • getDirectoryListing();
    • isDirectory();
    • isFile();
    • extension();

    You will want to iterate on the SD dir-list, recursively for subdirectories, checking file for extensions and adding targets to an array or object accordingly to your needs.

    Try some code, if you get stuck post it to let someone help you with the debug.
