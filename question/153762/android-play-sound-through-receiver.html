Android: Play sound through receiver

Hello!

I'm trying to get audio to play through an Android device's receiver. I've been able to do this just fine with no issues on iOS using Ti.Media.createSound but can't with Android.

Doing some research I determined the way to do this will be using a notification because that's where Titanium will allow me to make use of Android's STREAM_VOICE_CALL flag.

Here's what I'm doing…

var notification = Titanium.Android.createNotification({
                                sound:mediaURL, 
                                audioStreamType: Titanium.Android.STREAM_VOICE_CALL,
                                contentTitle: "Playing Message",
                                contentText:  "Playing Message",
                                tickerText:      "Playing Message",
                                contentIntent: pIntent
                            });
        Ti.Android.NotificationManager.notify(1, notification);

On the Android Emulator (2.2) I'm able to hear the audio (mediaURL is a remote URL) however if I try running the application on my Galaxy Nexus running 4.2 only the notification shows in the status bar; there is no audio.

Any help would be awesome! Thanks!

Funfacts:
Developing on: OSX
Titanium SDK: 3.1.0
Emulator: Android 2.2
Device: Android 4.2

Also, for whoever lands here from a Google search that may find it useful, here is what I'm doing for iOS…

Titanium.Media.audioSessionMode = Ti.Media.AUDIO_SESSION_MODE_PLAY_AND_RECORD;
        var player = Ti.Media.createSound({
                            url:mediaURL,
                            allowBackground:false            
                    });
