  • I had the same problem a week ago, and this is what I did. I got this from the skeleton code generated by studio.

    ...

var win = Titanium.UI.createWindow(...);
var tab = Ti.UI.createTab({window:win,..});
win.containingTab = tab;

...


var tableView = Titanium.UI.createTableView(...);
tableView .addEventListener('click', function(e){
    var win2 = Titanium.UI.createWindow(...);
    //Instead of Ti.UI.currentTab.open(win2);
    //use 
    win.containingTab.open(win2);
});
win.add(tableView);

    This worked for me. I hope you fine it useful.
