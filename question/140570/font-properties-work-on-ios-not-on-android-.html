  • Hi Nico

    There is a small issue with your code related to color. I have tweaked it below. There is much smaller list of fonts available for Android, that list differs between pre 3.0 and post 3.0.

    You can read more about standard parameters and how to use custom fonts in the docs.

    Revised code 

    var mainTitle = Titanium.UI.createLabel({
    text: pdfFiles[i].title,
    'color': '#000',  / move color up a level it is part of the main properties rather than the font subset.
    font: {
        'fontSize': 13,
        'fontFamily': 'arial',
        'fontWeight': 'bold'
    },
    top: 5,
    left: 85
});
