tableView - Update and pass tableView data between two windows

Hi there

I am having trouble figuring out how to pass data between tables across two windows.

The first window has a table that has some data in it and a button that takes you to a second window where you can edit the table data.

This is the code for the table on window1:


/ Create array to hold table object

var table_data = [
        {title:"One", hasCheck:true},
        {title:"Two", hasCheck:true},

];


//Create tableView to hold ingredients and assign data

var table1 =  Titanium.UI.createTableView({
    data:table_data,
    width: '70%',
    top: 50,
    bottom:90,
    backgroundColor:'#B6B6B6'

});

view.add(table1);


//Create second button field and set to open window 2
var button1 = Titanium.UI.createButton({
        width:295,
    height:60,
    title:'Edit Table',
        bottom:5,
    backgroundColor: '#E6E6E6',
    borderRadius: 8,
    borderColor: '#E6E6E6'
});

view.add(button1);


//Create event listener to chaneg to window2
button1.addEventListener('click', function(e){
    Titanium.UI.createWindow({
    fullscreen: true,
    url: 'window2.js'
    }).open();
});

The second window allows you to add/delete items from the table then go back to the first window and the table will be updated with the new data.

It does not ned to be stored in a database, just stored as variables then forgotten on exiting the program.

The table works OK on window 1 and I can open window 2 with the button but I am not sure what is supposed to happen on the second window to enable the data to be passed between windows, updated and passed back.

Any advice or pointers greatly appreciated

Thanks

JC
