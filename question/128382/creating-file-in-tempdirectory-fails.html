Creating file in tempDirectory fails

Mobile 1.7.5 with Android SDK on HTC Incredible 2.2 (no sdcard):

var tempFile = Titanium.Filesystem.getFile(Titanium.Filesystem.tempDirectory, app.reportName);

tempFile.write('test');

if (tempFile.exists()) {
    alert('not there');
} else {
    alert('there');
}
