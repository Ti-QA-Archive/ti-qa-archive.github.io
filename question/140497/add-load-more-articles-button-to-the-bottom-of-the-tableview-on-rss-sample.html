add 'load more articles' button to the bottom of the tableView on Rss Sample

Hello, I work with RSS Samble, on Android and iOS. App load 25 last articles on table. Does anyone know how to add to the bottom of the table ‘’Load More Articles’’ and when the user push the button the table view load the next 25 articles.
I work with Mac Book Pro 10.6.8, Titanium version 2.0.2 , Android Emulator 4.0.3 and iOS emulator 5.0.
Thank you!

  • Hi

    The load more row is nothing more than a row that says "load more", the only difference is you detect the click event as being on the last row and then call whatever code you use to gain more records and then "push" them into the table using insertRowBefore or insertRowAfter whilst including the row index that the before or after is associated with.

    You could use setData but you would have to reload everything back in and re-position the table, the inserts will probably work best.
