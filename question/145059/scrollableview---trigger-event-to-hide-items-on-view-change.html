    I am not sure if I fully understand the question, but in my case I needed to add a logic when I swipe to particular view. So event for this is "scrollEnd"

    var currentPage;
//alloy
$.scrollableView.on("scrollEnd", function(evt){
    currentPage = evt.currentPage;
    //do stuff
});
//Ti
scrollableView.addEventListener("scrollEnd", function(evt){
    currentPage = evt.currentPage;
    //do stuff
});

    • dragStart or dragstart is only supported for mobile web. If you look in documentation, you can see badges indicate what events are supported by which platform. The only thing that I can recommend is to listen for touchstart event

      scrollableView.addEventListener("touchstart", function(evt){
    Ti.API.info(evt.x + " : "+evt.y);
});
