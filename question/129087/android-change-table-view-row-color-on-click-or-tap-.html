ANDROID Change Table View Row color on click or tap ?

Hi,

the following refers to android only.

I cannot find any way to change the table view row default color on clicking or tapping. They keep getting the default orange color for the emulator or green for my htc.

What I try to do is change their color when they are selected. On selection it works with the relevant property BUT they first get the default color I described above before they are selected.

I tried everything. Adding listeners (both to table and tablerows) on click, tap, touchstart, touchend etc (in order to change their background colors on start of the click and resume it to default on release), touchenabled false etc. No result :(
I googled it and it seems that with native android code this is possible by adding a listener to the table and use a similar approach I just described. But somehow here these are overriden.

Has anyone managed to change the background color of the tableviewrows on clicking?
