Remove border on tableViewRow inside grouped tableview

Hello

How can I get rid of the border of a tableViewRow?

I have an image inside a row, and I want to make it seem it's just floating there without the border of the row.

I have set the borderColor and background to transparent, the background is transparent now, but the color is still black.

I've also tried to set the borderWidth to 0 but it doesn't help either. There is still this small border around the row.

here's the code I used:

var row = Titanium.UI.createTableViewRow({
    hasChild: false,
    height: 'auto',
    backgroundColor: 'transparent',
    borderWidth: 0,
    borderColor: 'transparent',
    header: " "
});

// the Image
var image = Titanium.UI.createImageView({
    height: 'auto',
    image: "../images/image.jpg",
    top: 0,
    left:0,
    right:0,
    bottom: 0
});

row.add(image);
row.className = 'image';
data.push(row);

var tableView = Titanium.UI.createTableView({
    style: Ti.UI.iPhone.TableViewStyle.GROUPED,
    data: data
});

I'm working on iPhone SDK 4.2 and Titanium SDK 1.6.2

Thanks in advance!

Tjeu

    Try a different value for the style property e.g.

    style: Ti.UI.iPhone.TableViewStyle.PLAIN,

    Then experiment with settings

    separatorStyle: Titanium.UI.iPhone.TableViewSeparatorStyle.NONE
separatorColor: 'transparent'

    It's a while since I've experimented in this area, but hope that's useful/relevant.
