    Have you seen the Animations in Action guide, which is pretty cool. Download the code from the link at the end of the guide and run it to see what it looks like (and how well it works) before you try to understand it. The reason is that, in the past, Titanium's Android support for animations has lagged behind iOS, so some animations would not work. That said, there have been a lot of enhancements in that area lately, so try it with the 1.6 branch of the latest Continuous Build first before trying 1.5.X.

    Cheers
