    You have to treat the numeric key as a string:

    json.data.items[e.index].content["5"];

    I suppose if you're looping through values, it might make sense to do something like this:

    var idx = "" + 5;  // convert to string
json.data.items[e.index].content[idx];

    For more discussion of numeric keys in Javascript objects, see this thread.
