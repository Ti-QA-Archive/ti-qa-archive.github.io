Ti.App.fireEvent not accepting paramaters

I'm trying to pass an object to an App level event without success, but a window level event will accept the parameters. I have both examples below. Am I missing something? The App level event param is an object that has the variable type with a value of 'hello'.

Window Event that works:

var win = Ti.UI.currentWindow;
var testBut = Ti.UI.createButton({
    title:'Visit House',
    top:390,
    height:50,
    width:200,
    action:function(){alert('here')}
});

win.addEventListener('hello',function(e){
    e.action()
})
win.fireEvent('hello', {action:testBut.action});

App Event that does not work:

var win = Ti.UI.currentWindow;
var testBut = Ti.UI.createButton({
    title:'Visit House',
    top:390,
    height:50,
    width:200,
    action:function(){alert('here')}
});

Ti.App.addEventListener('hello',function(e){
    e.action()
})
Ti.App.fireEvent('hello', {action:testBut.action});

    • Arguments of "fireEvent" are JSON stringified and functions cannot be stringified. Therefore, you can't send objects that contain function references.
