How can i get the datatype of a tablefield in sqlite?

How can i retrieve the information about a field of an table in sqlite? With recordSet.fieldName(i) I can get the name. But I need the datatype of a field. I want dynamically create my insert and update statement. That's why I have to know the datatype. Because for some type I have to quote the value.

    You will need to do a seperate query for that using the following:

    pragma table_info(<TABLE_NAME>);

    This will return you something along those lines.

    cid         name        type        notnull     dflt_value  pk
----------  ----------  ----------  ----------  ----------  ----------
0           COL_A       TEXT        0                       0
1           COL_B       VARCHAR     0                       0

    From there, you simply loop through each column and get its type.
