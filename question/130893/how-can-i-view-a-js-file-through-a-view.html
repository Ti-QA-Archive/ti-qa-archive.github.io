  • Sorry, that's not how Appcelerator works. You cannot just add a file of javascript to a window and expect it to interpret the content. the runtime my execute the javascript and consume the TI objects returned from that process

  • Nick is right is that respect. You cannot pass a file and expect it to be interpreted by Titanium.

    If what you are trying to do is actually show the user the actual content (source code) of the JS file. Then I would recommend you read the file's content using the Titanium.Filesystem.File.read method and print it out in your view.

  • In fact, .js are compiled, so you can't read them with the filesystem (except on the simulator). If you are trying to run code, you can use eval(&#39;...&#39;);, but you must use every library before (example: you want to use a scrollableview, you must have used it before).

    If you just want to read, rename it to .lib!
