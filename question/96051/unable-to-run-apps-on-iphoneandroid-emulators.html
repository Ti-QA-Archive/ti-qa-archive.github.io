  • So, I had the same issue but it was caused from upgrading to Titanium SDK 1.5.1 (at least I believe so, thats all that I believe had changed)

    anyway, I had to follow the steps that was linked to above - which I've since copied here for clarity & ease.

    The Following Steps Worked

    1. Backup the Resources folder.
    2. Delete project in Titanium
    3. Close Titanium
    4. (Finder) Go to [Project Directory]/build/iphone/build
    5. Delete all contents inside (don't delete build folder but every sub-folder)
    6. Go to /Library/Application Support/Titanium/mobilesdk/osx/
    7. Remove latest folder (1.5.1 currently)
    8. Re-open Titanium and it will say there's an update, so do that
    9. Create a new project with same details as original
    10. (Finder) Go to that projects Resources folder and delete contents
    11. Extract your backup you made in step 1 to this folder
    12. In titanium, compile in emulator
    13. If above successful, try compiling for device
    14. If all this fails. I'd go as far as re-installing Titanium itself
