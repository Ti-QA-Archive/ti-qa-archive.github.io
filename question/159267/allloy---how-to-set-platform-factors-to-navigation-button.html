    Hi

    Short answer - Yes. You already had the reference yourself.

    <LeftNavButton platform="ios">
// and
<RightNavButton platform="ios">

    Full revised code

    <Window class="APP_WIN" title="product_code" id="holder" onClose="onWinClose" onAndroid:back="onExitWindow">
    <Widget src="logisapp.david.customalert" id="AD"/>
    <LeftNavButton platform="ios">
        <Button id="lBtn" onClick="onExitWindow" style="Titanium.UI.iPhone.SystemButtonStyle.PLAIN" />
    </LeftNavButton>
    <RightNavButton platform="ios">
        <Button id="rBtn" onClick="onRNavBtnClicked" systemButton="Ti.UI.iPhone.SystemButton.DONE" />
    </RightNavButton>
    <TableView id="rec_det_tbl" class="APP_GROUP_TABLE" onScrollend="onTblScroll" onClick="onTblClick"/>
</Window>

    For reference the android version is almost identical;

    <View platform="android">
  <Label>Android Only</Label>
</View>

    You can also use formatFactor;

    <View platform="android">
  <Label formFactor='tablet'>Android Only Tablet only</Label>
  <Label formFactor='handheld'>Android Only Handheld only</Label>
</View>

    You can read more about it in the documentation;

    • Alloy XML Markup - Conditional Code

    There are new additions scheduled for (ideally) Ti SDK 3.2 which will allow even greater control.
