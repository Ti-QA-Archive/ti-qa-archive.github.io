    • The download above is so you can develop using Titanium - once installed you can then develop for; Android, iOS, Tizen, Blackberry and Mobile Web - provided you have the minimum requirements for each platform.

      Basically speaking - if you are running on OSX (Mac) then you can develop for all, otherwise you can develop for all but iOS in most cases.

      — commented December 1st 2013 by Malcolm Hollingsworth

    • According to the minimum requirements you must be on Mountain Lion minimum on a Mac, this is partly because of xcode requirements and partly because of supporting software requirements.

      • Titanium Compatibility Matrix

      Additionally you will find;

      • Deploying for iOS 7.0 requires Xcode 5.0, and Mac OS X 10.8.4 and later.

      You will need a newer version of OSX to develop properly. Remember that 10.7 is two versions behind the current version available.

      — commented December 1st 2013 by Malcolm Hollingsworth

    • If you wish to attempt coding for iOS6 and NOT iOS7 then;

      • xcode 4.3
      • Titanium SDK 3.1.1 or 3.1.2 - you will NOT be able to go higher

      This may not work but try the only OSX Titanium download there is from the link already provided. You will have problems though as your operating system is now deemed out of date for development needs and even if you do manage to get everything installed it is likely that in less than a year you will no longer be able to submit apps that do not support iOS7.
