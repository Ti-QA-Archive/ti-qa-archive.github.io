orientationModes did not work on tabgroup based windows (Android)?

i am using Titanium Studio 3.3.0, i have a tabGroup and several windows , but i could not lock my windows Portrait Mode , any advise ? thantks .

var winNewHouse = Titanium.UI.createWindow({,
    backgroundColor : '#fff',
    navBarHidden : false,
    fullscreen:false,
    layout : 'vertical',
    orientationModes:[Ti.UI.PORTRAIT]
});

var tabNewHouse = Titanium.UI.createTab({
    title : 'blabla',
    window : winNewHouse,
    myIndex:0
});

  • In a tabgroup driven app only the tab group is an activity. This tabgroup activity you can control in manifest.xml.

  • Your solution: modifying of your android manifest file like this:

      <activity android:configChanges="keyboardHidden"
                    android:label=""
                    android:name="ti.modules.titanium.ui.TiTabActivity" android:screenOrientation="sensorPortrait"/>
