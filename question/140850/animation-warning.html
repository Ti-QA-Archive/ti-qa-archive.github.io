Animation Warning

I keep getting this warning when running two simultaneous animations.

[WARN] New layout set while view [object TiUIView] animating: Will relayout after animation.

My code is as follows: 

view.animate({
    transform: t1,
    duration: 300
});

view2.animate({
    transform: t2,
    duration: 300
});

// t1 and t2 have been defined before this block as individual 2D Matrices

Should I be worried about this warning? My animations work exactly as expected.
If I should be worried, what am I doing wrong? :S

Note: I have narrowed it down to this: The warning comes when the animation for view2 is fired. When I comment that bit out, I no longer get this warning.

META: Ti.SDK = 2.1.1.GA
iOS 5.1

Thank you.

    Hi Nikhil

    I have also seen these 'warnings', having reviewed it a little bit I think you will find it is simply a warning, but more information based. It is simply stating the something (possibly) the animation itself has tweaked either; left, right, top, bottom, width or height. But it will not update the layout because the animation itself takes priority.

    I never noticed any ACTUAL errors myself, so I now take these as information only.
