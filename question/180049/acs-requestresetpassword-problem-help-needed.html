ACS RequestResetPassword problem. Help Needed.

Hi,
I am trying to use ACS RequestResetPassword but facing some problem. Appreciate if anyone can advice on this matter?

I had tried:

  1. I had setup up an 'Email Smtp settings' and tested. It replied an email with 'SMTP Check Successful'. So I presume the email is correctly setup.
  2. Then when I use the following code:
        Cloud.Users.requestResetPassword({
            email: 'tester123@gmail.com',             //This is a valid ACS 'registered' user email address!!

        }, function (e) {
            if (e.success) {
                alert('Success: Reset Request Sent');
            } else {
                alert('Error:\n' +
                    ((e.error && e.message) || JSON.stringify(e)));
            }
        });

it response with:

{"success":true,"error":false,"meta":{"status":"ok","code":200,"method_name":"requestResetPassword"}}

BUT no email was received!! WHY??

3.AND Then I tried with custom ‘Email Template' with following setting:

Note: (AppKey) is my ACS Application Key, and I used 'testing' as my reset_password_token.

name: ResetPassword

subject: Password Reset Request

body: https://cloud.appcelerator.com/users/reset_password?key=(AppKey)&reset_password_token=testing

and with following code:

        Cloud.Users.requestResetPassword({
            email: 'tester123@gmail.com',             //A valid ACS 'registered' user's email address!!
            template: 'ResetPassword',

        }, function (e) {
            if (e.success) {
                alert('Success: Reset Request Sent');
            } else {
                alert('Error:\n' +
                    ((e.error && e.message) || JSON.stringify(e)));
            }
        });

I get an Email replied with the body URL content but when I clicked on the URL link, it said:

Invalid Key

Please reacquire reset password token.

Anyone had encounter similar problem? Appreciate if anyone can advice on this matter.
Thank you in advance.

I am using Ti.SDK 3.4.1 GA and with android device with SDK 4.4.2
