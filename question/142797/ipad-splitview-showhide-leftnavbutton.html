iPad SplitView show/hide leftNavButton

Hello!

I'm working with a SplitView in an iPad app. Seems pretty basic, but I'm having trouble figuring out how to show/hide the leftNavButton based on orientation. I've looked around quite a bit, and I'm finding a lot of the same code, so I'm thinking I'm missing something silly.

I started with this…

SplitViewNav.splitView.addEventListener('visible', function(e) {
    Ti.API.log('View: '+e.view);
    if (e.view == 'detail') {
        e.button.title = "Categories";
        SplitViewNav.detailWindow.leftNavButton = e.button;
    }
    else if (e.view == 'master') {
        SplitViewNav.detailWindow.leftNavButton = null;
    }
});

That shows the button, but it doesn't hide it when I switch to landscape. The log only happens once, so I'm guessing that the "visible" only gets fired when the app launches?

So I tried to listen for orientationchange…

Titanium.Gesture.addEventListener('orientationchange', function(e){
    Ti.API.log(e);
    if(e.orientation == 3 || e.orientation == 4){
        SplitViewNav.detailWindow.leftNavButton = null;
    } else {
        e.button.title = "Categories";
        SplitViewNav.detailWindow.leftNavButton = e.button;

    }
});

Of course the issue here is that the button is not passed in "e". I'm a little stuck. I'm thinking I should be firing an event in the orentationchange listener that would have a listener that might resemble the visible listener , but I'm just not sure how to get my button in there.

Any guidance is much appreciated!!
