scrolling background image?

Forgive me if this is obvious but can a background image scroll along with content?

For instance if I give my scrollView a background shouldn't it move with my content as I scroll down the page? Also, when I have reached the end of my background (lets say its 600 tall and my view is double that) will it stop scrolling or should I see a background color (like you would on an html page if you set a background image to no-repeat)?

This is my sample scrollView code.

var scrollView = Titanium.UI.createScrollView({
    backgroundColor: '#ffffff',
    backgroundImage :'../images/my_bg.png',
    contentWidth:320,
    contentHeight:'auto',
    layout: 'vertical',
    top:0,
    showVerticalScrollIndicator:true,
    showHorizontalScrollIndicator:false
});

Thanks

  • To make the background scroll, create a view and put imageView inside of it, set it to the size you want. Then, put the view in to a ScrollView, make sure that imageView and View are both bigger than scrollView
