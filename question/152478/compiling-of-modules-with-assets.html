  • HI Rainer,
    That strenge as assets in the assets folder generally work with out issue,but still as a workaround you can create a platform folder parallel to module folder and put assets there while packaging module.

    Process:

    1: Create module zip.
    2: Unzip it.
    3: Create a Platform foder parallel to module foder now.
    4: create android and iphone folder parallel to module folder.
    5: package module manually now and create a zip with the same name as earlier.
    6: now it should work in Titanium App the way it should.
