  • My hunch is that the width of the image or the container it's in is what's causing you the troubles. Make sure to set widths as well as left values for the image and its parent container (e.g. View). If it's inside the table row, this shouldn't be an issue but sometimes the auto-calculated position/dimension values can be a bit non-intuitive. A trick I like to use is to set background colors to garish values – pink, red, green, etc. That way, you can easily see dimensions and positions.
