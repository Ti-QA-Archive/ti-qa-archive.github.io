    • Instead of string you can use boolean property :

        Ti.App.Properties.getBool('loaded first', true);
    if (Ti.App.Properties.getBool('loaded first') == true) {
        alert('loaded first time');
        Ti.App.Properties.setBool('loaded first', false);
    }

  • Hi Zozo,

    You can do it by setting your custom property like below set this code in your app.js file and check it.

        Ti.App.Properties.getString('loaded', "No");
    if (Ti.App.Properties.getString('loaded') == 'Yes') {
        alert('loaded');
    } else {
        alert('now Loaded');
        Ti.App.Properties.setString('loaded', "Yes");
    }
