AdMob on iPad and iPhone: iPad version shows iPhone formatted ads

Not sure where to go from here. AdMob's site doesn't seem to have controls for it, and I don't know what else I could do in my file. Using the ti.admob module, v1.1.

Here's what I have in my admob.js file, can anyone help?


if (Ti.Platform.osname == 'ipad') {
    // iPad
function create_admob(win) {
  var ad;

  ad = Admob.createView({
      top: 0, left: 0,
      width: 1024, height: 50,
      publisherId: XXXX
  });

  win.add(ad);  
}}
else {
    // iPhone
function create_admob(win) {
  var ad;

  ad = Admob.createView({
      top: 0, left: 0,
      width: 320, height: 50,
      publisherId: XXXX
  });

  win.add(ad);  
}}

  • From Dawson:

    You weren't requesting a valid size for an iPad ad.

    Please read https://developers.google.com/mobile-ads-sdk/docs/ios/intermediate

    The supported sizes are:

    320x50 Standard Banner Phones and Tablets
    300x250 IAB Medium Rectangle Tablets
    468x60 IAB Full-Size Banner Tablets
    728x90 IAB Leaderboard Tablets
    By specifying 1024x50, the next smallest fitting size was used, which is 320x50.

    By specifying width: 728, height: 90, the IAB Leaderboard Tablet ad size is properly displayed.

  • Accepted Answer

    AdMob will automatically use an ad size that fits within the bounds you've given it. Take a look at the sizes specified on their website. By specifying a height of 50, the only size that fits is 320x50. Boost that height to 90 and you'll see a bigger ad.
