  • Your quotes are missing. All search expressions need to be contained between quotes (even with wildcards)

    If you want your query to be For example:

    select * from users where username like '%lokesh%'

    You will have to escape the extra quotes in order to avoid syntax errors though.

    Like so:

    select * from users where username like '\'%lokesh%\''
