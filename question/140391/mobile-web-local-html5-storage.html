Mobile Web: Local HTML5 storage?

When my app was on iOS, I wrote my data to the App's Properties, as in

function setArr(name, value) {
Titanium.App.Properties.setList(name, value);
}

Now that I am doing the app as a mobile web html5 app, how would I store these arrays in HTML5 local storage?

sdk 2.1, studio 2.1, os 10.8

    On Titanium Mobile Web, Ti.App.Properties actually use local storage behind the scenes. You can store whatever you'd like in there. Keep in mind that the name/key takes up space, so keep those short and sweet. Also, most devices appear to have a limit of around 5MB of local storage and that's shared between Ti.App.Properites, Ti.Filesystem, and Ti.UI.Clipboard. When you run out of space, some browsers will truncate the data and others will prompt to increase the local storage limit.
