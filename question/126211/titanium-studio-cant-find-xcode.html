Titanium Studio can't find Xcode

I've installed Titanium Studio and Xcode on a new iMac and when I try to launch an app in in the iPhone simulator Titanium Studio says that Xcode is not installed on my system. Is there any way to show it where Xcode is?

    We choose the location using 'xcode-select -print-path'. If you type that on the command line in terminal, what does it print? In one case where we saw this, it turned out to be a bad install of Xcode. A uninstall and reinstall fixed it here.
