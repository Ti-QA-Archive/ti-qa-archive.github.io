Create XCode project on linux/windows - possible?

First: I know that a mac is needed to compile the xcode project!

But the latest Unity3d windows version allows the user to create the xcode project in windows:

Changelog:
...
Ability to build iOS target in Windows! It's still necessary to compile resulting Xcode project on a Mac.
...

Of course you still need a mac to compile it but you could have a remote machine that will just compile the xcode projects (drop folder or web server with interface).

Would this be possible with Titanium,too? Or does the osx build process already need osx-system tools to create the xcode project?

    I'm scratching my head trying to figure out what the value of an xcode project without xcode would be.

    So, how are you going to test this build out? Every build would have to be adhoc to throw it on a device.

    So the work flow is: add a label, build xcode project, upload to remote server, build, push binary to adhoc distro, download to device, run. Change label color, …, run?

    Using Tishadow, you should be able to push a classic project.
    Unless something has changed, using TiShadow, you should be able to push an alloy project.

    TiShadow only pushes js and assets, so no local compiling. Just have to compile the TiShadow client app with the modules you will want to use in it.
