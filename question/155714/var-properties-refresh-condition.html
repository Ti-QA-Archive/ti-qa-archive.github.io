Var properties refresh condition

Hello guys,

i have a problem in my code for an android application.

However if i click on the button, the event its still working, althought its suppose no possible, the condition is false with the new value of the variable.

  • Simple example:

    var condition = Ti.App.Properties.getInt('condition',false);

var btn = Ti.UI.createButton({
    backgroundColor: '#020202', 
    color: '#fff',
    height: 20,
    width: 20,
    top: 50,
    textAlign: 'center'
});

btn.addEventListener('click', function(e){    
    if(condition === false)
    {
        Ti.App.Properties.setList('condition', true);

        alert('Hello World');
    }            
});

    With this condition, my alert isnt supposed to work a second time, but it does.
    If i change window and back on this window, it will be work.
    What i want is a direct refresh of the window, window keep directly the new value of my variable condition, here "true".
