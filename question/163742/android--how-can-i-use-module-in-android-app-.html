    • The unzipped folder would need to be in the root of your app folder and nested like this;

      {appname}/modules/android/ <- like this here
{appname}/resources/ <- for structural reference only

      If you are using studio then remove the manual entry in your tiapp.xml and use studio to add it - this way it will confirm it has been found.

      • In Studio
      • Edit tiapp.xml
      • Confirm you see the visual view (not the text based view)
      • On the right side is the list of modules
      • Press the green plus
      • Look for the module
      • If you have it in the correct location it will be listed
      • Select it and confirm it
      • It will appear in the modules list for this app

      • Save and confirm

      • Menu > Project > Clean > Choose Project > Confirm

      Then try again.
