    • Did your path is correct in Titanium Preferences?

      Is there path is OK and does not display any error in that path then please delete all emulator from ADB and clean build your project again.

    • Yes the Android Sdk path was correct. How do I delete all emulators from ADB?

      — commented June 7th 2012 by Alfonso Velasco

    • Right click on your project in project explorer and select Run as > Run Configuration

      Window pops up. There is Titanium Android Emulators are there in the window.

      Remove all android emulators.

      There is another path that store emulator is [Android SDK folder] > Tools > android

      Android SDK Manager pops up, select Tools (from menu) > Manage AVDs

      delete all android emulators.
