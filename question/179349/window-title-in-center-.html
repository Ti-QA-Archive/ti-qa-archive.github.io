  • Hello Arafah,

    If we want to customize the title of a window we need to make sure navBarHidden is
    true then we can create a customized label for the title.

    To achieve this we can to follow the code below.

    var win = Ti.UI.createWindow({
     backgroundColor : 'white',
     navBarHidden : true
});
var navBarView = Ti.UI.createView({
     top : 0,
     left : 0,
     height : 45,
     width : '100%',
     backgroundColor : 'black'
});
win.add(navBarView);

var lblTitle = Ti.UI.createLabel({
     text : 'Window title',
     top : 0,
     left : '0.75%', //Customize
     height : navBarView.height,
     width : '62.5%', //Customize
     textAlign : 'right',
     color : 'White',
     font : {
          fontWeight : 'bold',
          fontSize : 22
     }
});
win.add(lblTitle);
win.open();

    Thanks.
