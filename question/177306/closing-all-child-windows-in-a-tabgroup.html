    HI Virendra,

    Create an array and push opened window object and when you double clicked on tab get that array and close all window using for loop or while loop.

    var myWinOpenedArray = [];
myWinOpenedArray.push(winObj);
//set this  myWinOpenedArray array in properties 
// http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.App.Properties-method-setList
//and get that array using getList
