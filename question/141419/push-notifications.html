Push notifications

Hi.

I am developing a small app and i want to implement push notifications.

I think that 199$/month with urban airship is a little expensive.

Is it possible to create push with the ACS that take the rss feed of a certain WP page that i have for example?

Say that i have a page that i use only for push. So when i edit the content of that page it will create a push.

I am a total noob at ACS so any pointers would be great.

Thanx.

    I don't think ACS itself can do this, but you can "order" to ACS to send pushes from the same server that holds the WP when a new post is online for example.

    There are some php wrappers that could help:

    • https://github.com/EDF-RetD-ICAME/titanium-acs-php-api
    • https://github.com/benedmunds/PHP-Appcelerator
