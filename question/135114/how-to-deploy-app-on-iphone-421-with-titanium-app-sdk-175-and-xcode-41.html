How to deploy App on iPhone 4.2.1 with Titanium App SDK 1.7.5 and Xcode 4.1

Hi,

I have developed a app using Titanium App with SDK 1.7.5 and have XCode 4.1 (SDK 4.3).

In Titanium, I can only set the iOS version to 4.3. So I am using XCode to deploy and test the app on iPhone. The App works great in simulator (4.3)

In XCode I selected 4.2 as target as the target iPhone device is running 4.2.1, but when I try to deploy, I get the below error.
