    Hi

    According to to the comparability matrix you will need to use much older unsupported versions of the SDK as well.

    This means you will have to avoid all new functionality that arrived in the 1.8 branch and 2.0 branch - so say goodbye to T.UI.FILL & Ti.UI.SIZE for a start.

    You should consider running side by side versions of XCODE and Titanium to keep them sand-boxed from each other. SO make sure you copy each to folders OTHER than the default ones. You may also find you need to run a terminal command to let the system know which xcode you are using at any one time.
