    Please note the following:

    As you are using the Titanium SDK 3.2.1 you do not have to specify an android: targetSdkVersion because Titanium will automatically use the latest Android SDK to build your app: http://docs.appcelerator.com/titanium/release-notes/?version=3.2.1.GA#android_sdk.

    That means you should get the Holo Theme by default. Make sure you have selected the latest Titanium SDK in your tiapp.xml
