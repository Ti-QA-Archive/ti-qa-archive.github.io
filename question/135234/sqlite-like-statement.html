SQLite "LIKE" Statement

Hi,

Can someone tell me the sql syntax for a like statement where the "like" is given by a variable?

for example, i know how to write it if we are talking about a simple "Where" clause:

var dbrows = db.execute('Select item1, item2, item3 from ITEMS_TABLE where item1=?', variable_X)

So, what exactly would the syntax be if I wanted to put a 'LIKE' instead of a 'WHERE' clause where the LIKE would be a variable (variable_x)?

Thanks

    try this

    var dbrows = db.execute('Select item1, item2, item3 from ITEMS_TABLE where item1 like ?', variable_X)
