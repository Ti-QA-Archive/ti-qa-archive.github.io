    • Well, lets say you have HTML string in some variable:

      var htmlString = '<label id="myLabel">My Text</label>';

      You can create webview in standard Titanium way:

      var webView = Ti.UI.createWebView({
    html: htmlString
});

      Now you can use "evalJS" method to use DOM methods.

      var labelText = webView.evalJS("document.getElementById('myLabel').innerText");

      "labelText" should be "My Text".
