No android sdk were found under specified path

I am using 64 bit windows 7 laptop. titanium studio is not detecting android sdk even after trying many versions of android sdk manager. I am using titanium studio version 3.4.1 and titanium sdk 3.5.1 GA and android installer23.0.2.
Can anyone please solve my problem.

  • Hi Naimika Senghani ,

    Thanks for your concern .

    It may be that you don't have an avd.py script at the location it's looking. To solve that, just install the latest Continuous Build, ensuring that you choose the master branch from the drop-down menu of the download site, and restart Studio.

    For derails please have a look :

    Could not locate the Android SDK at the given path.

    Steps

    1. Download the latest Ti SDK.

    2. Manually put the SDK to the Ti SDK directory.

    3. Restart the studio

  • I solved that issue, changing the main sdk in titanium command line

    1. titanium setup
    2. Select option "s"
    3. Select another sdk than 3.5.1
    4. Restart studio
