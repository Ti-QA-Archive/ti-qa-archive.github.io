Could not find the file app.js

When making a build for the iPhoneSimulator, then transferring the .app file to another machine and running on the simulator (via ios-sim) an error message appears "could not find the file app.s". Note, the .app runs fine on the machine it was built on.

The build command we're using is:
titanium build –build-only –platform ios –project-dir myproject –device-family iphone –ios-version 7.0 –sim-version 7.0

Doing some investigation I do notice absolute paths in the binary myproject.app/myproject so I'm guessing it's not portable as it is now.

Is it possible to make a portable simulator build?

Ti CLI 3.1.2

    Personally, I would Export the project from the first machine, and Import onto the other.

    Titanium Studio->File-Export->General->FileSystem
