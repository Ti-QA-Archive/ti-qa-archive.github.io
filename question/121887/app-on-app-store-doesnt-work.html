App on app store doesnt work

Hi,

I recently updated my app on the app store, everything seemed to go well and it was approved on the first try. Now however it doesn't work, when I press certain buttons nothing happens etc.

It worked fine in both the simulator and when I tested it on my devices. So my questions is;

What could be the reason for this? Does the code alter between the distrubution build and the test build and if so, can I test the dist build easily wihtout uploading it to the app store?

  • The code does alter between test build and a distribution build, the best way to test is to build for distribution but instead of uploading to the app store, get the .ipa file from your build iphone-release folder and install to your device through itunes. You'll need an AdHoc distribution profile first, which you'll need to drag into itunes before the app.

  • For iPhone you can do a ad-hoc distribution, you need to upload a .plist which has a link towards the .ipa file. This way you can test you app without cables etc. You go to the link where the .plist is, this will redirect towards the .ipa and install on your phone.

    You should try this for sure.
