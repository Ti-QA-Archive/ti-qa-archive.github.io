facebook error trying yo upload a photo

Hi,

I'm developing a simple iPhone application that can upload a photo to a user's Facebook profile. However, I get a "com.facebook.sdk error, code 5" . The authorization steps seem to work fine but when I get back to the application I get the facebook error code 5.

I'm using Titanium sdk 3.1.2.GA and Titanium Studio, build: 3.1.1.201306112235.

Here's the code snippet:

actionBtn.addEventListener("click", function(){
        sound.play();
        var dialog = new OptionsDialog();
        dialog.addEventListener('click',function(e) {
            sound.play();
            if (e.index == 0) {
                if (fb.loggedIn) {
                    fb.reauthorize(['publish_stream'], 'me', function(e) {

                        if (e.success == true) {    
                            var data = {
                                message: "testmessage",
                                picture: www.domain.com/url,
                            };            
                            fb.requestWithGraphPath('me/photos', data, "POST", function(e) {
                                if (e.success) {
                                    alert(L("Successfully posted the link to your profile!"));
                                } else {
                                    if (e.error) {
                                        alert(L("There was an error posting to Facebook") )
                                        alert(e.error);
                                    } else {
                                        Ti.API.info(e);
                                    }
                                }
                            });    
                        } else {
                            alert(e);
                            Ti.API.info(e);
                            alert("Hmm, something is wrong...");
                        }
                    });    
                } else {    
                    fb.authorize();
                }    
            }
});

I read that Facebook needs to reauthorize to upload pictures so I implemented this but without any luck. Did I do something wrong with the above implementation? I do not have a info.plist in the root of my directory. I see the fb appid correctly in my info.plist file in the build folder. I have the facebook app id in my tiapp.xml file.

I also read something about manually getting an access token but I haven't tried that.

Any suggestions or links to docs are greatly appreciated.

Regards,

Aksel

    Hi A E,

    Try this.

    fb.appid = 'xxxxxxxx';
// your app id
fb.permissions = ['publish_stream', 'read_stream'];
var file = Ti.Filesystem.getFile('your file path');
// your file path
var blob = file.read();
if (file.exists()) {
    var data = {
        caption : 'Some text ',
        picture : blob,
        name : 'application name',
    };
} else {
    alert('file does not exists');
}

fb.request('photos.upload', data, function(e) {
    if (e.success) {
            alert("photo uploaded");
    } else {
        if (e.error) {
            alert(e.error);
        } else {
            alert("Unkown result");
        }
    }
});
