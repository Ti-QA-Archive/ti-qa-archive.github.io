  • Here is the link to the Android module development guide: http://wiki.appcelerator.org/display/guides/Android+Module+Development+Guide

    You want to load an entire app in a Titanium window? You could potentially load an Android UI component that represents your app, using a native module, but not load the apk itself into Titanium. You might also want to consider using an Android Intent to launch from your Titanium app to launch your separate APK. here's some relevant links on using Android Intents and Intent Filters with Titanium:

    http://developer.appcelerator.com/blog/2011/10/forging-titanium-episode-9-android-intent-cookbook.html

    http://developer.appcelerator.com/blog/2011/08/android-intents.html

    http://developer.appcelerator.com/blog/2011/08/android-intent-filters.html

    http://developer.appcelerator.com/blog/2011/02/launching-activities-and-using-content-uris-in-android.html
