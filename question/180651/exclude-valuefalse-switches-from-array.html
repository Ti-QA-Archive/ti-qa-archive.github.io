  • I would use underscore to filter your array. You can use the _.filter function to filter out the values you don't want.

    Something list this should work:

    var trueOnly = _.filter(FIELDS, function(fldVal){ return fldVal != 'false'; });

    This will return only the values that are not equal to True.
