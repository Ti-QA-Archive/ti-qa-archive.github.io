Setting image for a button issue

I am setting image for a button with a background color. The image color is white with a transparent background. When i set this image for a button it appears in blue color. When i am setting this as background image there is no issue

"#directDebit" :{
width: "50%",
height: "10%",
borderWidth : 1,
borderColor : "#000000",
top : "70%",
left : 0,
backgroundColor: "#2a3950",
image : 'SubmitaReading.png'

}

  • Hi Arunraj,

    Thanks for your concern.

    The button image property is for displaying image left of its title. Check the link for details..

    If you use image property, then it renders some space for button title ( take space for all sides, strange! ). You are using a blue background, that's why you are facing it, use backgroundColor to 'transparent'. And if you want just image for button, then use the backgroundImage property.
