  • What you need to do is wrap an IF statement around the require, something like:

    var labelFactory = null;
if (OS_IOS) {
    labelFactory = require('com.deko.label');
}
else {
    labelFactory = somethingElse;
}

    or have seperate controllers for android and iOS depending on how you are using it.
    Please post your code where you "require" the module so we can understand better.
