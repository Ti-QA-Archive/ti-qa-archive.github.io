iOS 6 Style Tabs

I am pretty new to Ti, but I am curious about the way this platform works. My understanding is that Ti creates native elements for things such as buttons, tabs, etc. If that is correct then running an application on iOS 6 should create the iOS 6 style tabs. Tabs look quite different in the new version of iOS.

I don't recall a change this significant in the iOS UI before, so I am not sure how Ti is going to handle this. But it seems if the native iOS tab has a certain style, then an app running on an different version of iOS would look different, but it doesn't.

Any thoughts on how this might get updated in the future. Will we have to select a tab style? It doesn't seem like Ti should need to be updated unless the native calls have changed in some way.
