Any way to put google maps standar buttons on App

Hi everybody

I have an app which uses ti.map module to show a mapview. I want to display the map buttons for zoom in/out go to current location, etc.

my code to show the map is this


var map = require('ti.map');

var mapview = map.createView({
    top: '0 dp',
    bottom: '0 dp',
    left:'0 dp',
    right:'0 dp'
});

Ti.Geolocation.purpose = 'Find my house';
Ti.Geolocation.accuracy = Ti.Geolocation.ACCURACY_BEST;
Ti.Geolocation.distanceFilter = 10;
Ti.Geolocation.getCurrentPosition(function(e)
{

    mapview.setLocation({
        latitude : e.coords.latitude,
        longitude : e.coords.longitude,
        latitudeDelta : 0.01,
        longitudeDelta : 0.01
    });

    refreshProperties(e.coords.latitude, e.coords.longitude,1000);
    //Ti.API.info('se cargaron las propiedades');
});

$.win.add(mapview);

Is there anyway I can accomplish this?

    I assume your question is about Android as iOS has no integrated map buttons (besides the controls in the navBar and tabedBar).

    For Android, there are the properties userLocationButton and enableZoomControls to display / hide the native map buttons. They are true on default, so I wonder why you don't see them …
