  • A couple of things…

    Can you get any app to work? Try creating a brand new app and running it. Their "empty" new app should create two tabs and some simple output labels. If that doesn't work, I'm wondering if you have paths and environment variable problems.

    I'm not familiar with Linux-specific issues, but there could be something going on with that. The Ti folks seem to be Mac exclusively. I use both Mac and Windows and there are some platform differences due to the OS differences.

    Are you using the right version of the KitchenSink for your Titanium SDK? (Set that on the project description tab, not the launch in emulator tab). The newest KitchenSink won't work with an older SDK, for example.

    Have you tried clearing out the PROJECT/build directory? Crud builds up there and causes problems.
