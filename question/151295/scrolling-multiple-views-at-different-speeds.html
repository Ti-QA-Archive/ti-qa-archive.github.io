scrolling multiple views at different speeds

Hello everyone,

I'm attempting to make an effect where background scrollViews will scroll at different speeds than a foreground scroll view. The foreground scrollview is the only one which the user will interact with, but I'd like the background scrollViews to fire at different speeds with it. I've done multiple different solutions but I haven't been able to get enough performance to make anything visually viable.

  • scrollview inside of scrollview, and bubbling events
  • foreground scrollview manually fires events to background scrollviews
  • on scroll events, manually move background items

All of these attempts technically work, but its just so choppy it looks awe-full. I was hoping someone might have another idea that will give this visual effect, and still have a good frame-rate. If not, it looks like jumping to native sdk might be last option.

Thanks in advance, looking forward to any possible ideas!

-Daniel

  • Hi

    The main problem you are going to have is that Android does not handle or cache scrollView images efficiently and earlier the device - the worse it gets.

    This is an example of using three layers, front layer controls the other two, I have added a simple smoothing routine to get you started.

    You should make sure that you compensate for partial pixels and also adjust the offset of each layer to adjust evenly given the shift. So if you move 1 pixel but the layer is 1.5 times the height your resulting shift MUST be an even integer.

    Try this code to get you started - this will need you to play with it to meet your needs but will give you a starting point. Start a new default project and replace the contents of app.js with the code below.

    Ti.UI.setBackgroundColor('transparent');

var win = Ti.UI.createWindow({
    backgroundColor: '#fff',
    height: Ti.UI.FILL,
    width: Ti.UI.FILL
});
win.open();

var layerBack = Ti.UI.createView({
    height: Ti.UI.SIZE,
    layout: 'vertical',
    opacity: 1,
    width: Ti.UI.FILL
});
var layerMiddle = Ti.UI.createView({
    height: Ti.UI.SIZE,
    layout: 'vertical',
    opacity: 0.75,
    width: Ti.UI.FILL
});
var layerFront = Ti.UI.createScrollView({
    contentHeight: Ti.UI.SIZE,
    contentWidth: Ti.UI.FILL,
    showHorizontalScrollIndicator: true,
    showVerticalScrollIndicator: true,
    height: Ti.UI.SIZE,
    layout: 'vertical',
    opacity: 0.5,
    width: Ti.UI.FILL
});
function asPixel(p, s) {
    var t = p * s;
    var f = Math.floor(t);
    var c = Math.ceil(t);
    var a = (f === c) ? f : c;
    return a;
}
layerFront.addEventListener('scroll', function (e) {
    Ti.API.info('scroll', e.y);
    layerBack.setTop(-asPixel(e.y, 3));
    layerMiddle.setTop(-asPixel(e.y, 2));
});
var intItem = 0, intItems = 20, imgBack, imgMiddle, imgFront;
for (intItem = 0; intItem < intItems; intItem = intItem + 1) {
    imgBack = Ti.UI.createImageView({
        bottom: 5,
        height: 300,
        image: 'http://placehold.it/200x300/c60000/ffffff&text=Back' + (intItem + 1).toString(),
        left: 10,
        top: 5,
        width: 200
    });
    layerBack.add(imgBack);
    imgMiddle = Ti.UI.createImageView({
        bottom: 5,
        height: 200,
        hires: true,
        image: 'http://placehold.it/200x200/009900/ffffff&text=Middle' + (intItem + 1).toString(),
        top: 5,
        width: 200
    });
    layerMiddle.add(imgMiddle);
    imgFront = Ti.UI.createImageView({
        bottom: 5,
        height: 100,
        image: 'http://placehold.it/200x100/000099/ffffff&text=Front' + (intItem + 1).toString(),
        right: 10,
        top: 5,
        width: 200
    });
    layerFront.add(imgFront);
}
win.add(layerBack);
win.add(layerMiddle);
win.add(layerFront);
