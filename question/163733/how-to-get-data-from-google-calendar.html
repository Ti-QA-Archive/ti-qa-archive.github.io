  • Ok, I haven't actually tried this myself, but I would imagine you should be able to do a call to the calendar API using a HTTP request.

    Here is a link to the documentation which shows you how to do a simple GET request using the HTTP Client: Titanium.Network.HTTPClient

    The URL will need to be swapped out for the calendar API, which you will need to build up correctly yourself. I think it will be along the lines of this:

    https://www.googleapis.com/calendar/v3/users/me/calendarList

    Here is the documentation which shows you how to build the URL for the API call:

    https://developers.google.com/google-apps/calendar/firstapp#rest

    And here is additional information about the API references which should also help:

    https://developers.google.com/google-apps/calendar/v3/reference/

    Hope this helps!
