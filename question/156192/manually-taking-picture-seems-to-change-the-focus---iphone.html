Manually taking picture seems to change the focus - iPhone

Hello,
I have a camera with an overlay button and showControls set to false.

When the users presses the button then I call Ti.Media.takePicture() but, since the user has touched the screen, the iPhone start to focus on the exact spot where the button lays.
The result is a blurred picture. I do not want the autofocus to kick in! If this cannot be avoided, then I just do not want the auto focus to detect pressure on the overlay layer.

What do I do?

Thanks

  • Try to set bubbleParent to false to the buton. This way the event should be isolated to the button only.
