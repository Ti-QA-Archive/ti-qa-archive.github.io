  • Try adding JAVA's directory to the PATH directly on .bashrc:

    export PATH=$PATH:&#x2F;path&#x2F;to&#x2F;java:&#x2F;path&#x2F;to&#x2F;java&#x2F;bin

    then restart the shell or do

    $ source ~&#x2F;.bashrc
