Android: Center 'auto' sized views horizontally in a row

Hi,

I'm trying to create a menu like this:

Week | Month | Year

The group of menu entries must be centered.
Each label have a undefined width because text size is variable (supporting internationalization).
All the labels must be in a row.

On iPhone I am able to do this with the following code, but on Android
the same code put all the labels and views left aligned.

How can I accomplish this on Android?

        var menuHeight = 30;
        var elementSpacing = 5;
        var separatorWidth = 1;

        var menu = Ti.UI.createView({
            left : 0,
            right : 0,
            bottom : 0,
            height : menuHeight,
            layout : 'vertical',
        });

        var elements = Ti.UI.createView({
            bottom : 0,
            height : menuHeight,
            width : 'auto',
            layout : 'horizontal',
        });


        var createElement = function(_elementData, _separator) {
            var elementView = Ti.UI.createView({
                top : 0,
                height : 'auto',
                width : 'auto',
            });

            if(_separator) {
                var separator = Ti.UI.createView({
                    width : separatorWidth,
                    height : menuHeight,
                    backgroundImage : '/images/separator.png',
                    left : 0,
                    touchEnabled : false
                });

                elementView.add(separator);
            }

            var elementTitle = Ti.UI.createLabel({
                text : _elementData.title,
                height : 'auto',
                width : 'auto',
                left : elementSpacing,
                right : elementSpacing,
                top : 0,
                color : 'white',
                touchEnabled : false,
            });

            elementView.add(elementTitle);

            elements.add(elementView);
        }

        if(_elementsData) {
            var separator = false;
            for(var i = 0; i < _elementsData.length; i++) {
                createElement(_elementsData[i],separator);
                separator = true;
            }
        }

        menu.add(elements);

SDK: 1.8.0.1
Platforms: iPhone, Android

    Hi Hugo

    The height and width with the android is not accepting 'auto' try to give the fixed height and width for this and check it again. In the label you can use auto but not for the views..

    if the width is about 100% then you can use the 

    height:Ti.Platform.displayCaps.platformHeight,
    width:Ti.Platform.displayCaps.platformWidth,

    So try this out. Hope it help
