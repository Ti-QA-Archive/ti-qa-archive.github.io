  • "recreated my APP ID on Apple Dev Center" << You can't "re-create" an appID.

    See "Deleting an app" and "Transferring an app"

    If the previous app was deleted from iTunes Connect, it's gone: there's no coming back. If it was just removed from sale, you'll need to have ownership transferred, or access to that developer account.

  • You will need to use the same app id for your new project. (don't forget to use the same GUID also)

    Now, if you really wan to create a new project using Titanium Studio, you could just create your project with some new temporary ID. Then you could just close your project, edit the tiapp.xml file and replace the IDs with your old ones. Once this is done, simply re-open your project in Studio an you should be all set.

    Hope this helps
