How to send a parameter from app.js to html?

Hi all, I have an issue about send parameter from app.js to my local html file.

I searched for Q&A and found "evalJS" that I can pass some description to html, but when I using evalJS("showmsg(\"test\");"); to call function showmsg(msg) in html it didn't work.

Does there any mistake I made? or maybe who can tell me how to pass a parameter to html from app.js in another way.

Thanks for your help.

    • I'd suggest you to try this sample:

      app.js

      var win = Ti.UI.createWindow({
    backgroundColor : 'white'
});

var webview = Ti.UI.createWebView({
    url : 'html/index.html'
});
win.add(webview);
win.open();

Ti.App.addEventListener('html_loaded', function(){
    var a_string = 'test';
    webview.evalJS('showmsg("' + a_string + '");');
});

// setTimeout(function() {
    // var a_string = 'test';
    // webview.evalJS('showmsg("' + a_string + '");');
// }, 1000);

      html/index.html

      <html>
    <head>
        <script type="text/javascript">
            function loaded() {
                Ti.App.fireEvent('html_loaded');    
            }

            function showmsg(msg) {
                alert('msg = ' + msg);
            }
        </script>
    </head>
    <body onload="loaded();">
        <p>
            HTML Content
        </p>
    </body>
</html>

      Please noticed that you can only call webview.evalJS(&#39;showmsg&#39;) after html loaded. In above code, when html loaded, it'll trigger Ti.App.fireEvent(&#39;html_loaded&#39;); to inform Titanium code that it's ready to accept calls using evalJS.

      Another solution is use setTimeout() as I commented above.
