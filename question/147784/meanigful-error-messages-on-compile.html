Meanigful error messages on compile

(sorry for my bad english) On windows 7 (64bits) everytime I get an syntax error on a Alloy Controller javascript (mobileweb), when I try to run the app i get the error:

Unable to generate Mobile Web App. Error:

[INFO] :  Compiling "development" build
[INFO] :  Reading Titanium Mobile Web package.json file
[INFO] :  Validating theme
[INFO] :  Found Alloy app in C:\DEV\Titanium\TestApp\app
[INFO] :  Executing Alloy compile: alloy.cmd compile C:\DEV\Titanium\TestApp\app --config platform=mobileweb,version=0,simtype=none,devicefamily=none,deploytype=development

An exemple that cause this error is a missing " that I have on a script…
Is it possible to get a more meaningfull error message, maybe showing the line where the problem is?

  • Try rebuilding with a log level of "trace".

    From Titanium Studio, you can pick that from the console's icons on the right side. From the command line, you can run from your project's directory:

    titanium build -p mobileweb --log-level trace
