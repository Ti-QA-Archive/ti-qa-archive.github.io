    • In this section of code;

      var image=Ti.UI.createImageView({
    image:json.item[i].photo,
    width:60,
    height:60,
    top:5,
    left:5
});

      You reference the photo node here;

      image:json.item[i].photo,

      The console.shows the following example entry;

      1368000791_15.jpg

      So this means your code translates to;

      var image=Ti.UI.createImageView({
    image: '1368000791_15.jpg',
    width:60,
    height:60,
    top:5,
    left:5
});

      This then means that your code is trying to load 1368000791_15.jpg from your resources folder. You say you want to load the image from a remote server so it should look much more like this;

      var image=Ti.UI.createImageView({
    image: 'http://somedomain.com/somefolder/' + json.item[i].photo,
    width:60,
    height:60,
    top:5,
    left:5
});

      Change http://somedomain.com/somefolder/ to be wherever your images are stored.

      Basically you need to tell the image property that your value is suffixed to a URL that reaches your specific image.

  • Atlast the issue is solved:

    function storeImages(filename) {
                var xhr = Titanium.Network.createHTTPClient();
                 xhr.onload=function() {
                    var f = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory,filename);
                    f.write(this.responseData); // write to the file
                 };
                 xhr.onerror=function(e){
                 alert("check the connection");
                 setTimeout(1000);
                };
           xhr.open('GET','url'+filename, false);
           xhr.send();
  }
