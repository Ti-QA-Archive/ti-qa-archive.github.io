  • The DoubleClick module is now only available to enterprise customers. However, I recently got DFP working with the version 1.3 of the Titanium Admob module. You replace the publisherId from Admob with the Adunit id from DFP. Its a bit of work to get Admob backfill working, but I've documented how here:

    http://jameslow.com/2013/01/08/google-dfp-with-appcelerator-titanium/
