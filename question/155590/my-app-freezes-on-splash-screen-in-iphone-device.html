My app freezes on Splash Screen in iPhone device

It's very strange, my app works fine on iOS Simulator and freezes when i test in my iPhone. When i run with the Debug mode, runs normal.

I have others projects that run normal in both cases, but the new projects have this problem. For a another test, i create a blank project, run on Simulator but it freezes on device.

After that, i think that was my phone that has a problem, and i try to run it on the ipad, same problem, freezes at start.

Does anyone knows a solution?
