how to deal with the blank space when there is no Ad

I am using iAd and Admob as a backup. I tried to use a simple table view with two rows to do this. so the top row is for Ad, bottom row is the content. the height is set to auto, I am expecting the top row not show when there is no Ad, however, somehow, the blank space is always there.

Can anyone please help me out here?
This should be a very common issue for everyone.

  • You need to use an event that catches on iad to show or hide it, check the iad module for available events, there is one indicating whether or not it is loaded. Off hand I'm not sure if admob has I similar event but you should be able to use the same process.
