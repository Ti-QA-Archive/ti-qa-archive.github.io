  • Fix: sudo chmod 777 /usr/local/lib/node_modules
    Reason: If you check the contents of the folder /usr/local/lib/node_modules, the folders within are owned by nobody:staff regardless of what user you did the npm install with, even with root or sudo. Fixing the permissions of this folder allowed acs to install with no further event.

  • Fix: sudo chmod 777 /usr/local/lib/node_modules

    Reason: If you check the contents of the folder /usr/local/lib/node_modules, the folders within are owned by nobody:staff regardless of what user you did the npm install with, even with root or sudo. Fixing the permissions of this folder allowed acs to install with no further event.
