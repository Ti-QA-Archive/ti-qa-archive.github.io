  • can we stop the script to run, and exit the code

    You cannot exit an iOS program without generating a crash which would prevent your app from passing apple app review.

    You CAN exit an Android application if you close the main window and has exitOnClose: true.

    In either case, closing the app on the user is really crappy design and UX. Particularly since network connectivity is a varying state that could have changed as soon as your force closed the app.
