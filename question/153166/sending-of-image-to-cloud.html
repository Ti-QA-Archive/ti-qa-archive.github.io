Sending of image to cloud.

Hi,

in my app the camera take photo and then I want to send to cloud.

I see three ways:

  1. I connect the result of camera with a (preview-) imageview and send it from view like:
    var scaledphoto = self.photoview.getImage();
require('vendor/mensa.cloud').postComment({
 post : {
     photo : scaledphoto 
     }
}),
    It works fine, but it need long time because of imagesize. Idea: I send the blob from Image view like:
    var scaledphoto = self.photoview.toImage();
require('vendor/mensa.cloud').postComment({
 post : {
     photo : scaledphoto 
     }
}),
    It works fast, but the photo from cloud is emty (white).

Next idea:

var scaledphoto = self.photoview.toImage(function(_photo) {
    require('vendor/mensa.cloud').postComment({
        post : {
            photo : _photo 
        }
    })
});

In this case the Cloud function says me: "wrong data type".

Any ideas?

iOS, SDK 3.2

  • Now I'am using imageFactory. It works.

    require('vendor/mensa.cloud').postComment({
            post : {
                vote : parseInt(self.slider.value),
                comment : self.comment.value || '',
                dish : _title,
                photo : (self.photo) ? ImageFactory.imageAsResized(self.photo, {
                    width : 320,
                    height : 320,
                    quality : ImageFactory.QUALITY_MEDIUM
                }) : null
            },
            id : (self.id) ? self.id : null
        });
