Back Button Event for Camera Overlay

Is there a way to listen for when the user presses the hardware back button in Android when the camera overlay is showing? I know you can listen for the 'android:back' event on windows, but I wasn't sure how to detect such events while the camera overlay is displayed. Thanks.

  • Hi David

    The camera overlay is actually a view and we cannot directly listen the back event inside the view. The window which opens the view for camera able to listen the back event and from that you can perform other logic.

    So from my perspective if the overlay is done using view then we cant handle the back event inside the view.

    Nikunj
