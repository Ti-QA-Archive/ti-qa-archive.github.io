animation won't go to the right....

Hi All
I hope someone can help me out with this animation below. The label scales and animates fine to the top but it won't move to the right only to the left. Can anyone explain why? I am trying to scale the label and move it to the center of the screen - and I succeed in the scaling and moving upwards parts but not to move it to the right…. Thanks a lot for your help ;)

var myLabel = Ti.UI.createLabel({
        bottom:windowHeight * 0.1,
        width:windowWidth * 0.22,
        height: windowWidth * 0.22,
        left:windowWidth * 0.05,
        borderRadius: 35,
        backgroundColor: 'white',
        color:'black',
        textAlign:'center',
        text:'Setup'
    });

function animateRound (label) {  
    var t2 = Ti.UI.create2DMatrix().scale(3);
    b = Ti.UI.createAnimation({ 
    duration:400,
    transform:t2,
    top:windowHeight * 0.4,
    right:windowWidth * 0.1,
    curve:Ti.UI.ANIMATION_CURVE_EASE_OUT
    });
    label.backgroundColor = 'red';
    label.animate(b);
};

myLabel.addEventListener('click', function(e){
        animateRound(myLabel);
    });

Best, Louise

  • Accepted Answer

    I think the answer is that when you create the view object, you set the left property.

    You then try and move it by the right property.

    Try setting the right property within the initial object creation you can then set the right property going forwards.

    Hope this makes sense

  • Try to be consistent with what you declare in the constructor of the label and what you animate.
    If you set the left property then animate the left property.
    If you want to animate the right property then set it first in the constructor.
