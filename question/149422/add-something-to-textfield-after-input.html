Add something to textField after input?

Hi.

I wonder if it is possible to add some value to a textfield after the input has been made.

For ex:

var input = Ti.UI.createTextField({
height:'20dp',
left:'50dp',
color : '#181818', 
width:'100dp', 
top:'113dp', 
hintText:'Write Here' ,
keyboardType: Ti.UI.KEYBOARD_PHONE_PAD, 
value: inputValue + ' Something',
});

So i want to see the hintText and after i made an input like "I like" it will read: "I like Something".

Or do i have to make separate text fields and take the value from the first, put it in a string and then remove the first textField and add a second with 

value: Titanium.App.Properties.getString("input") + ' Something',

If so how do i add an Eventlistner to do that?

Thanx.

    Probably the best way would be to modify the value right when the user leaves the textfield using the blur event.

    ####Your coude could look something like this:

    input.addEventListener('blur', function(e) {
    e.source.value = e.value + ' Something';
});

    I would advise against doing the same thing on the change event. Otherwise, you could could generate an infinite loop. (every time the value changes, you append something, thus changing the value again and so forth.)

    Hope this helps
