Ti.Include

I wonder if it's normal that this code does not work:

var label = 'Demo';
Ti.include(demo.js);

demo.js

alert(demo);

Script Error = Can't find variable: demo at demo.js
There are solutions?

  • Yes, it is normal. You haven't declared "demo" variable, "demo" is your file name (which has to be in quotes btw - "demo.js", because it's a string).

    Try this:

    //app.js
Ti.include('demo.js');
alert(label);

//demo.js
var label = 'Demo';

    This example is not the best one, because "label" is global variable and you should avoid using them if that's not necessary. You should read about JavaScript best practices.
