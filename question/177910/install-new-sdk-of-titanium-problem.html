install new SDK of Titanium, problem

Hi, I need to update Titanium SDK 3.4.0. but i can't make it.

In page http://preview.appcelerator.com/appcelerator/studio/standalone/update/rc/ says:

Titanium Studio Update Site

This site is designed to be used through Titanium Studio update manager.

Update Titanium Studio

  1. From the Help menu, select Install New Software… to open an Install pop-up window.
  2. In the Work with: text box of the Install window, type the URL http://preview.appcelerator.com/appcelerator/studio/standalone/update/rc/ for the update site, and hit the Enter key.
  3. Close the Install window.
  4. From the Help menu, select Check for Updates.
  5. In the new window, click the Next button and then click the Next button again.
  6. Choose the option to accept the terms of the license agreement, and click the Finish button.
