    an alternative solution would be to convert your view into an image (view.toImage() ) and send it to your server. Then the server would create the PDF (you can use php open source libraries like FPDF, and send it back to your app.
    IMHO what meenakshi suggested above would work better because you wouldn't have to manage all of the back and forth to the server.
