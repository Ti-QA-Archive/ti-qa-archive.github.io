  • You could use e.source to figure out which of the buttons that have been clicked. If I remember correctly, you should also be able to get custom properties from there as well, so you could do something like this:

    var button = Ti.UI.createButton({
  title: "Test",
  myId: 1
});
button.addEventListener('click', function(e) {
  if (e.source.myId == 1) {
    alert("Yep");
  }
});
