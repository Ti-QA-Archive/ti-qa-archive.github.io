change button text at runtime...

Is there a way to change the text on a button at runtime? I'd like to select a date from a picker and then display said date on the button. Any ideas?

    Sure. Most any property of any object can be changed dynamically at runtime using the notation:

    objectName.propertyName = newValue;

    In the case of a button, the text is called the title, so just:

    someButton.title = &#39;new title&#39;;

    If you used a specific width for the button the title may not fit as well, if you replace with a longer title.

    One notable exception to the ability to change nearly any property at runtime is the "layout" property of a view or window. These MUST be given when the object is created and cannot be changed at runtime.
