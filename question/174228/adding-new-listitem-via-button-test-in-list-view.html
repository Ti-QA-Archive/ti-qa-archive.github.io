  • Accepted Answer

    The ListSection items property expects an array of ListDataItems (here).

    So instead, try the following:

    var tmp = {
    properties: { title: "Sample" }
};

section.setItems( [tmp] );
list.setSections( section );

    I don't know the format of your template, but that could be set up wrong too. Something like the following might be more correct (complete guess)

    var tmp = {
    location: {text: "RESERVE Y"},
    template: "location_template"
}
