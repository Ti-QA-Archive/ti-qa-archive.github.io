write/update a xml file fails

hello,

I'm trying to change the nodeValue of a node. I think this works, but when i'm trying to write the updated document to a file, it does not seem to be updated. First i read out the xml file:

var settingsFile = Ti.Filesystem.getFile(Ti.Filesystem.resourcesDirectory,"settings.xml");
var text = settingsFile.read().text;
var xmlDoc = Ti.XML.parseString(text);

then i want to change a value of a node with this method:

Settings.prototype.setUrl = function(url){
    xmlDoc.documentElement.getElementsByTagName("eyequestionurl").item(0).nodeValue = url;
    Ti.API.info(xmlDoc.documentElement.getElementsByTagName("eyequestionurl").item(0).text);
    settingsFile.write(xmlDoc);
    Ti.API.info(settingsFile.read().text);
}

As you can see in this method i first change the nodeValue of a tag and then i print out the text of the node. this works just fine, because it prints out the new url.
but after that i write de document to the file, and after that i print out that file, but in this print the nodeValue is not changed, so writing the xmlDoc to the file goes wrong.
Does anybody knows how i can update a node and write the document to a file?
thnx in advanced.
