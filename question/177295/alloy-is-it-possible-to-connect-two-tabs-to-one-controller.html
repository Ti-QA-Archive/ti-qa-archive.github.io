Alloy: Is it possible to connect two tabs to one controller?

So my question is - is it possible to connect two tabs to one controller in alloy?
I have the following structure in index.xml:

<TabGroup>
    <Require type="view" src="tab1" id="tab1" />
    <Require type="view" src="tab2" id="tab2" />
    <Require type="view" src="tab3" id="tab3" />
</TabGroup>

And in the "tab1" controller I have the Tab element with "Window" element inside.
What I want is to open tab1 when user clicks tab2.
I've tried to add second "tab" element to "tab1" controller, but it doesn't help :(

  • There's a one-to-one correspondence between XML and JS files that together make up a "controller." You could possibly add code within index.js to do what you're seeking, though you'd probably have to not use the <Require> tags and put the code directly into index.xml.

    Having said that, I don't understand what you're trying to accomplish. Tabs have a particular purpose from a user-experience point of view, which you seem to be trying to break. But perhaps I'm misunderstanding what you mean by "open tab1 when user clicks tab2".
