    Googling brought you almost to the right place to get started with Titanium API :)

    To add menu items and other desktop functionality to your Titanium Desktop app, you will need to interact with the Titanium Desktop API from within your code (in your case from JavaScript).

    To do this, you simply use various Titanium API classes and call their methods from JavaScript, similar to how you would interact with the window or browser objects in a web application.

    As a quick example, add an html file with these contents to your application:

    <html>
    <head>
        <script type="text/javascript" charset="utf-8">

            var mainMenu = Titanium.UI.createMenu({});
            var myMenu = Titanium.UI.createMenuItem("My Menu");

            myMenu.addItem("First menu item", function() {
                alert("You clicked the first menu item");
            });

            myMenu.addItem("Second menu item", function(){
                alert("You clicked the second menu item");    
            });

            mainMenu.appendItem(myMenu);
            Titanium.UI.setMenu(mainMenu);

        </script>
    </head>
    <body style="background-color:#1c1c1c;margin:0">
        <div style="border-top:1px solid #404040">
            <div style="color:#fff;;padding:10px">
                Welcome to Titanium
            </div>
        </div>
    </body>
</html>

    Take a look at the JavaScript, especially the lines which use the Titanium.UI namespace.

    To get familiar with the Titanium Desktop API, take a look at the API reference at http://developer.appcelerator.com/apidoc/desktop/latest.

    If you want to learn from examples, a good example of a great application written using Titanium Desktop API is Wunderlist. Source for their desktop client is available on GitHub here - it can be a good learning resource until Titanium Desktop API gets some more attention on Appcelerator Wiki.
