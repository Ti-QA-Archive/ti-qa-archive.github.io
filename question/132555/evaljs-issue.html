evalJS Issue

Hi everyone,

I am trying to load a javascript function before a webpage is launched. Here is my web view:


webview.addEventListener('load', function() 
    {
        var a_string = webview.myElement;
        Ti.API.info("I am right now in webview");
        Ti.App.fireEvent('FromWebview',{id: a_string});
        webview.evalJS(Bobby(a_string));
    });

My Javascript Function is called "Bobby(a_string)"
As you can see, I've also tried to send the SAME javascript function the a_string parameter using the FireEvent function, but with no success.

I just prefer to see if evalJS works.

Is my syntax correct?

My Javascript function looks like this:

function Bobby(IneedAString)
{

.... my code here

}

Many thanks in advance for any help you can provide.

Regards
