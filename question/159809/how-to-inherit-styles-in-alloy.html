How to inherit styles in Alloy?

Hi, please is possible somehow extend/inherit styles in Alloy?

Example:

".niceButton":{
    width: 100, height: 50,
    borderRadius: 10,
    borderWidth: 3,
    borderColor: "pink"
    // etc...
}

".nB_redStyle":{
    backgroundImage: "red.png",
    backgroundSelectedImage: "red_selected.png"
    // etc...
}

".nB_greenStyle":{
    backgroundImage: "green.png",
    backgroundSelectedImage: "green_selected.png"
    // etc...
}

… and I need to implement all properties from .niceButton into redStyle, greenStyle… I have about six different colours and I don't want to use everywhere same properties (Duplicates…).

I have tried also something like this (Obviously it didn't work…).

<Button class="niceButton" class="nB_redStyle">

    Put the base style in the app.tss (global style), then create the more complex style in whatever controller will use it. The styles from the app.tss and the controller style (i.e., index.tss) will be merged.
