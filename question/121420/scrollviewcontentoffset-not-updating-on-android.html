scrollView.contentOffset not updating on android

There's a problem with scrollview.contentOffset - it doesn't change after scrolling the view.

This is my code:

var win = Ti.UI.currentWindow;

var view = Ti.UI.createScrollView({
    layout:'vertical',
    contentHeight:1000,
    top:0,
    bottom:0,
    right:0,
    left:0,
    contentOffset:1
});

win.add(view);

var label;

for (var i = 0; i<10; i++){
    label = Ti.UI.createLabel({
        text:'row '+i,
        font:{
            fontSize:30
        },
        height:100
    });

    view.add(label);
}

var interval = setInterval(function(){
Ti.API.info(view.contentOffset);
}, 1000);

If contetntOffset is not defined in createScrollView, it's just undefined all of the time…

This is only an android problem.
Has anyone seen this or has any suggestions?

thanks.
