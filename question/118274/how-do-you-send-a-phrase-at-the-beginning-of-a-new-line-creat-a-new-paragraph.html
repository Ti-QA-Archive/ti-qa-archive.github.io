How do you send a phrase at the beginning of a new line (creat a new paragraph)?

I have been building a Table View app and every description of the items I wrote in html. Some of the descriptions are long and to break them up a bit (easier to read) I clicked Enter, so phrases wouldn't start one right after another - but they still so. Basically, my question is: how do you create a paragraph? How do you break the phrases so they start from the beginning of the line?
Thank you and sorry for my English.

    Sorin

    You can use the newline character, which is \n. There is an example of this in use in the Improving the data and view layout section of the "Using TableViews" guide.

    Of course, if you are using webViews in your tableViewRows, you would use the html line break tag:

    <br />

    Cheers
