Titanium Release for Android (Roadmap)

Hi,

I'm about to port my app to Android. It works ok in iPhone (iOS).

I tried to compile it for Android, and I had a load of error messages.

Is there a Titanium/Appcelerator release in the works that handles these exceptions (porting issues) better so we don't have to make loads of code changes to suit the different OS types?

I checked on the website and found no mention of any new features addressing this issue.

Is there any chance of it compiling for Windows Mobile also?

Thanks for any updates you can provide on this,

— asked April 15th 2012 by E B

  • I would say no. Sorry, but if your app is crashing on Android, it most likely needs some refactoring. The platforms are very different, so you can't expect a whole app written and optimized for iOS to work perfectly (or at all) on Android.

    I'd definitely recommend testing on both platforms during development. I've done the same mistake before, and since I started testing everywhere all the time, life has certainly gotten easier.

    I've heard that they are looking into Windows Mobile-support, but I don't think it's on any roadmaps yet so it probably won't be available anytime soon.
