    Hi

    As you have not provided any code - this will have to be a guess.

    If the error message you have provided is the exact wording shown then the problem could be that you have set an id property on the Window tag. This means that the assumed index id swaps to the id you have entered. index is assumed normally as the is the name if the controller.

    <Alloy>
  <Window id="thisthing">
    <!-- other code -->
  </Window>
</Alloy>

    The root id here is thisthing, so the controller would say;

    $.thisthing.open();

    If this foes not solve it provide the contents if index.js and index.xml.

    BTW yes Alloy is most certainly the fight thing to do.
