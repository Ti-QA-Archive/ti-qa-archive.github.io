  • I would recommend using the https://gist.github.com/1287727 twitter library that sits in top of jsOAuth. Then you can simply perform

                client.request('1/statuses/update.json', {
                'status' : 'This is a tweet'
            }, {}, 'POST',  function(){
                alert('Tweeted!');
            });
