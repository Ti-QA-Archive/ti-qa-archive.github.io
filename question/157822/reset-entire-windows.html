Reset entire windows

So what I want is a window that after you back out of it, it completely resets. All the variables are removed and when you call require() again it goes in a remakes the whole window. I've tried nulling the variable that holds the whole thing, but when I call require again it just skips it. How can I accomplish this?

  • require statements are cached at the system level for performance purposes, so there is no way you can go around that. Than you could do though is to create a function that you want to run every time the Window is created. For example:

    ####MyWindow.js

    function MyWindow() {
    var self = Ti.UI.createWindow();

    self.init = function() {
        // Reset everything you want here...
    }

    return self;
}

module.exports = MyWindow;

    ####app.js

    var MyWindow = require('MyWindow'); // Will be cached

var win = new MyWindow();

win.init(); // Will be called every single time
