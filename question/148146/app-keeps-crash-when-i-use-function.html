I am making a calculator app for android(currently testing on 2.2) and im working on the operations but every time I call the function my app will crash without telling me what went wrong.

Here is my function:

function Operation(op){
    var ReadOut = parseFloat(output.value);
    if(flagNewNum && pendingOp != '='){}
    else{
        flagNewNum = true;
        if('+' == pendingOp)
            Accumulate += ReadOut;
        else if('-' == pendingOp)
            Accumulate -= ReadOut;
        else if('/' == pendingOp)
            Accumulate /= ReadOut;
        else if('*' == pendingOp)
            Accumulate *= ReadOut;
        else
            Accumulate = parseFloat(ReadOut);
            output.value = Accumulate;
            pendingOp = op;
    }
}

And here is how im trying to use it:

addBtn.addEventListener('click', function(){

    Hi

    Have you tried it using the correct syntax for if statements with multiple lines?

    function Operation(op){
    var ReadOut = parseFloat(output.value, 10);
    if(flagNewNum && pendingOp != '='){
    } else
        flagNewNum = true;
        if('+' == pendingOp) {
            Accumulate += ReadOut;
        } else if('-' == pendingOp) {
            Accumulate -= ReadOut;
        } else if('/' == pendingOp) {
            Accumulate /= ReadOut;
        } else if('*' == pendingOp) {
            Accumulate *= ReadOut;
        } else {
            Accumulate = parseFloat(ReadOut, 10);
            output.value = Accumulate;
            pendingOp = op;
        }
    }
}

    Titanium uses standard JavaScript, not a special kind of JavaScript, it does have additional methods but all commands are the same.

    The code above will now correctly use the statements in the right place, let me know if your problem persists.
