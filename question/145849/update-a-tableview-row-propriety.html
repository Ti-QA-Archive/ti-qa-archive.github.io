Update a TableView Row propriety

Hi, i have this tableView:

    var tabellaGare = Titanium.UI.createTableView({
    top: '0',
    separatorColor: 'black',
    backgroundColor: '#27344F',
    editable: false,
    data:elencoGare,
    search: searchbar,
    hideSearchOnSelection: true,

});

and i append rows in this way:

         for (var i=0; i < elencoGare.length; i++) {tabellaGare.appendRow({title: elencoGare[i].idgara+") "+elencoGare[i].nomegara+"   "+elencoGare[i].giorno,color: 'white',hasChild: true,});};

how can i add the hasCheck propriety without edit the rest of the row? i only want it to appear.
i tried like this:

tabellaGare.updateRow(rowid,{hasCheck:true});

but then in the table i see only the check and no more the id and the name. i want just to add the check without touching the rest

thank you so much

  • Accepted Answer

    var rows = [];
for (var i=0; i < elencoGare.length; i++) {
    rows[i] = Ti.UI.createTableViewRow({
        title: elencoGare[i].idgara+") "+elencoGare[i].nomegara+"   "+elencoGare[i].giorno,
        color: 'white',
        hasChild: true});
}
tabellaGare.data = rows;

    Now you can modify the rows addressing by index.

    ie: rows[7].hasChild=false;
