  • I've narrowed it down to the backgroundGradient. If anyone else has this error, declare your gradients before you add them to the object should get rid of that error message. Sample code below:

    var bgGradientHeader   = ({type:'linear',colors:[{color:'#fefefe',position:0.0},{color:'#e7e7e7',position:0.50},{color:'#c8c8c8',position:1.0}]});

var header = Ti.UI.createView({
...
});

header.backgroundGradient = bgGradientHeader;
