Cloud querying on array properties

Hi!

I'm using ACS and I have some users and some teams (custom objects). Every team has a members array which contains objects like {user_id: 12345, role: "user"}. I'm trying to do a query that gives me every team that a specific user is a member of. Like:

"Give me teams where team.members contains an object where user_id = my_user_id".

Is this possible with the built in cloud queries?

Thanks!

    Cannot be done, you probably shsould create a relationhip between the teams and the players to make the query easier.

    From documentation: http://docs.appcelerator.com/cloud/latest/#!/guide/search_query

    In addition, you can query the custom fields on any object. Note, however, that you can only query simple values, such as Strings, Dates, Numbers, or Booleans. If a custom field takes an array or object as a value, you can't query any of the values stored inside the array or object.

    Using Relational Fields
