    • In OSX /Library/Application Support/Titanium or ~/Library/Application Support/Titanium

      In Windows 7 C:\Users\username\AppData\Roaming (or C:\ProgramData\Titanium on Titanium Studio 1.0.1 and earlier)

      In Windows XP C:\Documents and Settings\username\Application Data (or C:\Documents and Settings\All Users\Application Data\Titanium on Titanium Studio 1.0.1 and earlier)

      In Linux ~/.titanium
