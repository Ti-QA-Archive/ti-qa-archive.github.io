    • Just keep the Ti.Gesture event listener in app.js and when it executes, fire an app event which you listen to in your other js files. This is pseudo code but should give you the right idea.

      app.js:

      Ti.Gesture.addEventListener("onorientationchange", function(e) { 
  Ti.App.fireEvent("OrientationChanged", e);
});

      otherfile.js:

      Ti.App.addEventListener("OrientationChanged", function(e) {

});
