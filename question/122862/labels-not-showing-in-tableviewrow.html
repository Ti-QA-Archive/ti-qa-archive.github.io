What I am trying to do is very simple: create ONE table row with two labels. The labels however, are NOT showing up and I am really frustrated. In Cocoa Touch, this takes 2 seconds with Interface Builder and just 5 seconds if written programaticaly in objective-c.

Here is my code:


function createView()

    var row = Ti.UI.createTableViewRow({
        layout: 'vertical',
        leftImage: '/images/moreArrow.png'
    });

    var title = Ti.UI.createLabel({
        text: '??????',
        textAlign: 'right',
        top: 0,
        right: 0

        text: '????',

    win.add(table);

  • Use horizontal layout and remove positioning properties (left, right, top, bottom). You'll see that labels are there, but you misplaced them or they have been hidden because combination of vertical layout and small container (~table view row height) can make things disappear. :)

    Anyway, I think you need to play a bit more with layouts and positioning to see how they work. Unfortunately, there is no good documentation regarding this.

    If you need help, please explain what you want to do precisely.
