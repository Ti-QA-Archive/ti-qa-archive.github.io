    Hello perception system,

    For numpad or decimalPad, please update my following code: - 

    Ti.UI.createTextField({
        value : "",
        left : 10,
        width : "90%",
        hintText : "Enter Amount",
        color : "gray",
        top : 10,
        height : 35,
        clearButtonMode : Ti.UI.INPUT_BUTTONMODE_ONFOCUS,
        keyboardType : Ti.UI.KEYBOARD_NUMBER_PAD,
        font : {
            fontFamily : "Open Sans",
            fontWeight : "regular",
            fontSize : 14
        },
        backgroundImage : "txtbackgroundImage.png", // Please remove this
        leftButtonPadding : 5,
        bubbleParent : false,
        maxLength : 10
    });

    I think this will solved your problem.

    Thanks
