Camera Image Size

I cannot seem to get the width and height of an image taken by the camera. I know that the image taken is 2048x1536, however I cannot seem to get anything that will report that back to me.

For example:

success:function(event)
    {
        //var cropRect = event.cropRect;
        var image = event.media;
        Ti.API.debug('Our type was: '+event.mediaType);
        if(event.mediaType == Ti.Media.MEDIA_TYPE_PHOTO)
        {
            var imageView = Ti.UI.createImageView({

                //width: 'auto',
                //height: 'auto',
                image:image,
            });
            var d=new Date();
            var filename = d.getTime() +"pb10.jpg";
            var f = Titanium.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory,filename);
            f.write(image);
            var fileImage = imageView.toBlob();

//tell me how big the file is?
            vblob = imageView.toImage();
            var iw=vblob.width;
            var ih=vblob.height;
            alert('Original width='+iw+'  height='+iw);
}

I have tried setting the imageView height and width to 'auto' and it never reports the correct height and width…

  • Accepted Answer

    Ti.API.info("success! event: " + JSON.stringify(event));

    will show you this, which contains the dimensions you are looking for

    success! event: {"height":2448,"cropRect":{"y":0,"x":0,"width":3264,"height":2448},"y":0,"mediaType":"public.image","x":0,"width":3264}

  • Use the event.media object to get the values.

    var height = event.media.height;
var width = event.media.width;
alert("Photo height = " + height + " , width = " + width);
