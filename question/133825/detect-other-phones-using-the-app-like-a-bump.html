I'm looking at building an app that will need to detect others phones around it that are using the same app. Does Titanium support this for iPhone and or Android? And please point me in the right direction as I have no idea what to call it or what it is.

An example of this would be an ability to detect other phones in a range, feet or inches, so one could text chat with that device.

    Some thoughts. Devices would have to be in an open/receiving mode (potential security issues) for a connection if we use bluetooth or wifi to determine fellow users. It could be possible to spam local wifi with a tcp packet with data from the app looking for users.

    Other options is for each device would have to report its location to a central service. and the service do the math on distance and then msg each user with a proximity alert.
