Map view window has no other elements?

I seem to be opening my map view window just fine. I'm even getting the location pin and annotation just fine. When I touch it, it shows the little hover box with the title and address.

My problem is: How do I get a back button on this window? And is there any way to also invoke the directions from current location?

Titanium.Geolocation.forwardGeocoder(dealer_address,function(evt)
{
    lat = evt.latitude;
    //Ti.API.info('latitude: '+evt.latitude);
    lon = evt.longitude;
    //Ti.API.info('longitude: '+evt.longitude);
});

var locationView = Titanium.Map.createAnnotation({
    latitude:lat,
    longitude:lon,
    title:client_name,
    subtitle:dealer_address,
    pincolor:Titanium.Map.ANNOTATION_RED,
    animate:true,
    leftButton: '../images/appcelerator_small.png',
    myid:1 // CUSTOM ATTRIBUTE THAT IS PASSED INTO EVENT OBJECTS
});

var mapview = Titanium.Map.createView({
    mapType: Titanium.Map.STANDARD_TYPE,
    region: {latitude:lat, longitude:lon, 
            latitudeDelta:0.01, longitudeDelta:0.01},
    animate:true,
    regionFit:true,
    userLocation:true,
    annotations:[locationView]
});


Titanium.UI.currentWindow.add(mapview);

    Its not the map view, its how you're creating the window that HOLDS the map view.

    For example, say your code is called "map.js". In the page that loads it, you would either use the currentTab.open('map.js'), or you would create a window with map.js as the url, and do a myWindowName.open();

    If you're doing the 2nd route, you just need to create your own button, and use the setLeftNavButton(yourButtonName). Add a click listerner that is 'Titanium.UI.currentWindow.close()' and you're fine.

    If you're using the currentTab method, it generates a back button automatically.
