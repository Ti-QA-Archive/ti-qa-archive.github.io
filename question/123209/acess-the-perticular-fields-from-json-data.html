    • If it is valid, you should have regular JavaScript object stored to "Jdata" variable. Now, you can access to your data same as to any other object's property:

      Jdata.PropertyName
//or
Jdata["PropertyName"]

      If you need to loop over object's properties, you're doing it wrong way. Object has "length" property, but it's not the same as for arrays (google what's the difference between object and array in JavaScript). This is the way to loop through object's properties:

      for (var prop in Jdata) {
   Ti.API.info('Property name: ' + prop);
   Ti.API.info('Property value: ' + Jdata[prop]);
}
