Hi,

Has some one here run into an issue to keep the android screen on ie. not go into idle or lock when the phone is left idle. Would appreciate a code sample or some directions.

In 1.8.0.1 I believe the keepscreenon property for a window works. But 1.8.0.1 and CI builds on 1.8+ are causing other bugs to crop in my application.

ti Version: 1.7.5
Android: 2.2

  • Accepted Answer

    keepScreenOn was added in 1.8.0, so I think you will need to go with that version if you want to use it. You use it in android when you create the window, add the property: keepScreenOn:true. The Jira Ticket has some code samples in the comments.
