Targeting particular element in row of Tableview (android).

Hi,I want to change particular element of row in table view.My row structure like this:

var row = Ti.UI.createTableViewRow({height:'auto',className:"row",backgroundColor:'transparent',layout:'vertical'});

mainview
{
 sub view
 {
    label1{}
    label2{}        
 }
 sub view2
 {
    label{}
    image{}    
 }    
}
row.add(mainview);

description
{

}
row.add(description)

I tried table.data[0].rows[e.index].children[0]but its gives me null values.When I tried e.row.children it gives me some value like [Ljava.lang.Object;@441455.I want to target image in sub view2. Any solution to this problem.Thank you

  • Accepted Answer

    Good practice is to put reference on row object to its child elements like this:

    row.Label = label;

    Then, in event listener, you can get it this way:

    e.row.Label

    That way your code is not dependent on row child order. :)
