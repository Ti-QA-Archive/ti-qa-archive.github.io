Does same app, different iOS device require different App Id

If I wanted to take an iPad app and create an iPhone version from it, can I keep the same app ID or do I change that?

For example, do I need to do this:

com.redgreyhound.myproduct (iPad)
com.redgreyhound.myproduct-iphone

I was going to create a new mobile project in Developer, copy the Resources folder from the iPad app and then compile.

It's the same product, but I wasn't sure if it requires a different ID because of the different devices.

    This does not require a different id.

    You could even package it as a Universal App and not even need a second app at all for it.
