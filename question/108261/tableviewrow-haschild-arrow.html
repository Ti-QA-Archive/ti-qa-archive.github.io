TableViewRow hasChild arrow

Hi,

Does anyone know if there is a way to render the arrow that is displayed as a result of the hasChild property to be rotated? Specifically, I'm hoping to be able to point it up or down instead of to the right. Thanks!

Matt

    Instead of using hasChild, create an image which you want to show, then use 'rightImage' in the tableViewRow construction.

    See : http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.UI.TableViewRow-object
