  • I've had my best luck with storing a separate array that acts as a stack reference to the windows in the hierarchy. When a window is added, I push a reference to that window onto the array. When a window is remove, I pop a reference to that window off the array. This also makes it easy to drill back to the root window of the navigation stack, since you can run a while-loop that counts the length of the array and pops off a reference for each windowClose until nothing is left.
