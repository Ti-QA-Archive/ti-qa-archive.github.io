Ti.include() not including my JS file (or so it would seem)

    Using 

    function foo()
{
    function bar()
    {
    }
}

    makes bar a local function of foo that is not accessible from outside foo

    using

    function foo()
{
     this.bar = function()
    {
    }
}

    works as you expect.
