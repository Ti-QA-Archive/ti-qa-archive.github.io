    What you want exactly. explain more.

    If you want draw line between imageview then use VIEW.

    var hrLine = Ti.UI.createView({
    width : whatever you want,,
    height : 1
})

    for horizontal line : use width : 'auto' and height : 1

    for vertical line : use width : 1 and height : 'auto'
