Collapsable ListView items (similar to iOS weather app)

Hi,

I would like to create a collapsable ListView - so when an item is clicked it will expand to a view with more details on the screen. Like done in iOS weather app.

I found a similar thing done in iOS at Github here: https://github.com/Antol/APPaginalTableView

    Implementing something like iOS weather app listview functionality without going native looks complex.

    Anyway, it is possible to implement the collapse behavior. I had the exact same need, and basically what I did was something like this:

    1. Setup two listview templates, one for the collapsible header and another for the items 'inside' it.
    2. While adding my collapsible header listview items, I set up two additional properties on them: isExpanded and id
    3. When the collapsible header list view item is clicked, check if it is expanded. If it isn't, load data (if this is the case) and insert items immediately after our header index. If it is, remove the listview items that have parentHeader (another custom property) equal to our collapsible header.

    And it works for Android too :)
