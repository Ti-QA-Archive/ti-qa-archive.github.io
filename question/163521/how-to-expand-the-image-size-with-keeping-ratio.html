    If I understand you correctly, you need to approach the problem from the opposite direction. Start with a high-resolution image and shrink it. Also, auto is not a supported property value for height/width (since sdk 2). You need to use Ti.UI.SIZE or Ti.UI.FILL depending on your requirement. See this link for details.
