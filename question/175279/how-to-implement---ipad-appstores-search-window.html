How to implement - iPad Appstore's search window

iPad Appstore uses tabgroup. When you search, it opens a window with a title bar. That window seems not to belong to any tab, but tabs are still showing underneath. If you click "Cancel," the window closes and it goes back to the tabgroup. If you click any of the tabs below, you go back to the tabgroup, too.

How do I implement the same?

Thanks.

  • Hi

    These are distinct methods of opening the window if on iOS devices and additional changes when on iPad.

    Alloy syntax

    $.window.open({
  modal:true,
  modalTransitionStyle: Ti.UI.iPhone.MODAL_TRANSITION_STYLE_FLIP_HORIZONTAL,
  modalStyle: Ti.UI.iPhone.MODAL_PRESENTATION_FORMSHEET
})

    Non-Alloy syntax (so close might not have been worth stating)

    window.open({
  modal:true,
  modalTransitionStyle: Ti.UI.iPhone.MODAL_TRANSITION_STYLE_FLIP_HORIZONTAL,
  modalStyle: Ti.UI.iPhone.MODAL_PRESENTATION_FORMSHEET
})

    These are described in the Ti.UI.Window documentation under iPad Modal Windows.

    You find more constants to play here;

    • Titanium.UI.iPhone - constants section
