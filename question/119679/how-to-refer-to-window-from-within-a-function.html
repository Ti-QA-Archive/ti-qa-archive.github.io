How to refer to window from within a function

Hi,
Pseudo code:

App.js file creates a window.
There is a button in this window.
The button calls a function.
Inside the function, I write the code to create a modal window.
When the modal window close button is clicked, I want to change a variable in the root of the main window.

I am not able to understand how the events propagate in this case.

How can I access variables in different windows that are created using functions.

this?
root?
whats the way to access these things.

For example, I created a window, then added a button to it.
Now when I do:

Ti.API.info(myWindow); &#x2F;&#x2F; this works and shows [object TiUIWindow]

but
if i do:

Ti.API.info(myWindow.myButton); &#x2F;&#x2F; this works and shows &lt;Null&gt;

  • In app.js create your function or variable

    eg:

    
function doSomething(something){
    Ti.API.info('doing something : '+something);
}

    Then, when you create your window in app.js, add this function name as a property of the new window

    eg:

    
var myWindow = Ti.UI.createWindow({
   url:'somescript.js',
});
myWindow.doSomething = doSomething;

    Then, in your window js, you can do this:

    var win = Ti.UI.currentWindow;

win.doSomething('hello!');

    that should work.
