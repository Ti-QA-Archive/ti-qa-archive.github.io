  • Hope you don't load 100 images all at once regardless of what platform you are using and never use full size images for tableview rows. Couple things you can do here. 1) Provide thumb size images for the tableview row if the images are too big. Somewhere in the range of 150x150 then Using Paging to replace rows or load more to append more rows.
