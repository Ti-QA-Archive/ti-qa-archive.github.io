About module installation

Hi

I tried to install module which I downloaded from github and gittio.

However, I have no idea to use them because I just followed instruction of documentation to install

module however, in tiapp.xml cannot find out module after I installed and I just find out module

folder in C:\ProgramData\Titanium this path when I use module install wizard to unzip the module.

Then I thought other market place modules are placed in C:\ProgramData\Titanium\modules

this path under right platform I hope to integrated such as android, so I move folder to the path, but
it was still not working.

I hope to know what I need to fix for using modules.

Thanks.

    Hi JH,

    Try unzip the module directly to your project.
    i.e.:

    ~{projectFolder}&#x2F;modules&#x2F;android&#x2F;{module_name} for Android

    ~{projectFolder}&#x2F;modules&#x2F;iphone&#x2F;{module_name} for iOS

    Then refresh your project and you should see them in tiapp.xml
