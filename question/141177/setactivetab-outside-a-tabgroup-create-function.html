setActiveTab outside a TabGroup create function

hi,
I've been using the fugitive demo from appcelerator as a template for my app, whereby i have the following function in ui.js:

af.ui.createApplicationTabGroup = function() {
        var tabGroup = Titanium.UI.createTabGroup();
        var blog = af.ui.createBlogWindow(false);
        var me = af.ui.createMeWindow(true);
        af.blogTab = Titanium.UI.createTab(
        {
          title: L('blog'),
          window: blog
        });
        af.meTab = Titanium.UI.createTab(
        {
          title: L('me'),
          window: me
        });
        tabGroup.addTab(af.blogTab);
        tabGroup.addTab(af.meTab);
        return tabGroup;
    };

within the createBlogWindow function i have a button, when pressed I want the set active tab to af.meTab.

The code i have to do this is: tabGroup.setActiveTab(2); but not too sure how to do this outside of the createApplicationTabGroup.

many thanks - adam

    Hi Adam

    I am assuming you are calling the function into a variable like;

    var something = af.ui.createApplicationTabGroup();

    So you should be able to call;

    something.setActiveTab(1);

    BTW tabs are zero indexed, so 'meTab' is actually 1, with 'blogTab' would be 0.

    If this does not work, please provide the code you use to call the function createApplicationTabGroup.
