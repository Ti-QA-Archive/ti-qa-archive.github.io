    • "setTimeout" is a function, not property. Here is simple example that shows how it's used:

      var win = Ti.UI.createWindow({
    modal: true,
    backgroundColor: '#000'
});

setTimeout(function() { win.open() }, 3000);

      So, this code will open window after 3 seconds.
