Extending alloy styles

Hi,
I want to extend my globel styles with specific ones. For example I have style in app.tss called rowStyle1 and in specific examples i want to override some of the values. So what i want is in another tss file , I need to rowStyleX extends rowStyle1. Is it possible in alloy?

    app.tss styles merge with all styles in your app. So your index.tss merges with app.tss, with the index.tss taking precedence. Unless I'm misunderstanding your question, there's no need to use some kind of explicit "extend" syntax.

    app.tss

    ".rowStyle1": {
    backgroundColor: '#fff'
    height: 50
}

    index.tss

    ".rowStyle1": {
    height: 60,
    left: 20
}

    A view in the index controller that was assigned the rowStyle1 class would then get the following generated style:

    {
    backgroundColor: '#fff'
    height: 60
    left: 20
}
