Application Loader: Where is the IPA located to upload to app store?

I'm trying to upload a new version of my app to the app store using Application Loader. I packaged the app in Titanium Studio using "Distribute - App Store". Then in Application Loader, I need to choose the file to upload. But I can't find the file to upload, where does Titanium Studio save the IPA file to upload to the Application Loader?

  • Startup XCode and open Menu->Window->Organizer, there is an Archives button at the top and you will find your apps in there.

  • If you can't find the IPA file, open Xcode Organizer, select your app from the list of Archived Applications. You can generate the IPA file by clicking 'Share…'
