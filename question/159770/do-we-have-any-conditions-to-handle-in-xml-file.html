    Hi

    There are currently two conditional parameters that can be used; platform & formFactor - but I am working on the basis that you both know about these are are not talking about those.

    Currently there are no logic based conditional code that can be applied - however that might be about to change;

    Check out these two related pages that detail conversations on the subject

    • Custom TSS device queries
    • Extend custom formFactor

    Expect a near future Alloy revision to provide this logic.

    • That syntax will never be used in the XML files, but this will be in either the next version of Alloy or the one after. Adding this logic is currently being discussed for TSS but there is no reason why xml files should not gain it as well.

      <View if="Alloy.Globals.isIOS7">
    <Button title="Conditional Appearance" />
</View>
<View if="true">
    <Button title="Conditional Appearance" />
</View>
<View if="false7">
    <Button title="Conditional Appearance" />
</View>
