Build fails on Snow Leopard

Hi,

I'm new to Titanium.
I'm using Titanium Studio build: 3.0.1.201212181159
I've installed Xcode 4.2 for Snow Leopard.
I created a test app and it builds and runs fine for Android Emulator.
But when I try to build for iPhone Simulator I get error as follows:

[ERROR] Invalid –ios-version value '5.0'
Accepted values:

Any idea how I can fix this?

  • Hi

    Try this;

    • Select your project on the left side - project explorer
    • Click the run icon (green circle play button, small yellow folder)
    • Click Run configurations…
    • The right side should show iOS SDK
    • This is probably blank, change this to 6.0 or the highest it provides
    • Apply
    • Run

    Report back either way. If this does not work or the iOS SDK field has no choices, let me know if you have had XCODE on your machine before.

    • Another thing to try;

      This can happen if you have either installed two different versions of Xcode in different locations in the past or chosen to install Xcode in a location that is different from the standard location.

      Type this in the terminal:

      > sudo xcode-select -switch /Developer

      Or

      sudo xcode-select -switch /Applications/Xcode.app/Contents/Developer/

      Checking the path matches the location of your Xcode. Re-start studio afterwards and try again.

    • This is also how I fixed it. I could not find a link to 2.1.4.GA, so I snagged the link to the 2.1.4 RC here at the blog, and followed their instructions, slightly modified.

      1. Help > Install Titanium SDK …
      2. Paste the link to the SDK into the "Install from URL" box.
      3. Click Finish.

      I think had to go tell my example application which version of the Titanium SDK to use. I went to the TiAPP Editor associated with the application, and selected the 2.1.4.vNNNN under Build Properties / Titanium SDK:. I then saved the file (Apple-S) and voila! The iPhone simulator now builds and executes fine.

      I am running OSX 10.6.8 Snow Leopard, Titanium 3.0.2, XCode 4.2 with iOS SDK 5.0 and now I have shoehorned in Titanium SDK version 2.1.4. Hopefully someone will read this and provide an updated link to 2.1.4 GA or inform us when Titanium 3.x and SDK 3.x work with Snow Leopard?
