I cannot make TextField use custom font when passwordMask is set to true:

    var passwordEntry = Ti.UI.createTextField({
        font: {fontFamily:"Roboto-Light", fontSize:'16dp'},            
        hintText:'Password',
        passwordMask:true,
    });

Is it some bug in Titanium or do I get something wrong? (The font works perfectly well when passwordMask is set to false)
