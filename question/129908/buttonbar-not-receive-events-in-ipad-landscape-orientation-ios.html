ButtonBar not receive events in iPad landscape orientation (iOS)

A strange case here.

this code works fine on emulator (iPhone/iPad) and runs fine on device (iPhone) in landscape and portrait.

But, on an iPad in landscape orientation, the buttonbar does not receive any event (buttons does not change at all and touchstart nor click events are fired). Rotate the iPad to Portrait orientation and it works fine.

Haven't found any workaround, any idea how to solve?

Can someone confirm if it is a bug?

var win = Ti.UI.createWindow({backgroundColor:'#fff'});


var bb1 = Titanium.UI.createButtonBar({
    labels:['option 1', 'option 2'],
    backgroundColor:'#f6c26e',
    bottom:0,
    style:Titanium.UI.iPhone.SystemButtonStyle.BAR,
    height:35,
    width:'100%',
    zIndex:100
});
win.add(bb1);

bb1.addEventListener('click', function(e){
    alert('click');
});


win.open();

Device: iPad 2, iOS 5.0.1
Titanium: sdk 1.8.0.1
