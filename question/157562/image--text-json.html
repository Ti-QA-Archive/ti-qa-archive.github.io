    Your image values are in an array, so you need an index to identify which one you want

    nickLabel2 = Ti.UI.createImageView({
        image:images[0].value,
        width:64,
        height:64,
        left:'85dp',
        top:50,
        });

    Or change your variable declaration to

    var image = json.data.current_condition.weatherIconUrl[0];

    Or just remove the array, (and object actually), and have weatherIconUrl contain the actual URL as it's name suggests.
