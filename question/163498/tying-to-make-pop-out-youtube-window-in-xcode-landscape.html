Tying to make pop-out youtube window in Xcode landscape

Hello,
I have this code to load my youtube video when I click on my UIWebView:

        - (void)viewDidLoad
        {
            [super viewDidLoad];
        // Do any additional setup after loading the view.

            // This is the video content for the organ donor video:


            [self.exampleAdvert setAllowsInlineMediaPlayback:YES];
            [self.exampleAdvert setMediaPlaybackRequiresUserAction:NO];


            [self.view addSubview:self.exampleAdvert];

            NSString* embedHTML = [NSString stringWithFormat:@"\

                    <html>\

                    <body style='margin:0px;padding:0px;'>\

                    <script type='text/javascript' src='http://www.youtube.com/iframe_api'></script>\

                    <script type='text/javascript'>\

                    function onYouTubeIframeAPIReady()\

                    {\

                    ytplayer=new YT.Player('playerId',{events:{onReady:onPlayerReady}})\

                    }\

                    function onPlayerReady(a)\

                    { \

                    a.target.playVideo(); \

                    }\

                    </script>\

                    <iframe id='playerId' type='text/html' width='%d' height='%d' src='http://www.youtube.com/embed/%@?enablejsapi=0&rel=0&playsinline=0&autoplay=0' frameborder='0'>\

                    </body>\

                    </html>", 300, 200,//<size of the video

                    //this is the video link on the end of the youtube video

                    @"videocode"];

        [self.exampleAdvert loadHTMLString:embedHTML baseURL:[[NSBundle mainBundle] resourceURL]];

}

Everything works perfect, however I wanted to know if there is a way to add a landscape mode when the video is playing? As the whole app is restricted to portrait, I thought there must be some code to allow just this certain function in landscape as well?
