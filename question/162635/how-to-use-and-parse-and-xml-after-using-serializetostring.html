how to use and parse and XML after using 'serializeToString'

Hello folks,
I have an rss and I want to use it another place therefore,
in onload(); I am serializing to String then I will use it in another place 

        var stringedXML = Titanium.XML.serializeToString(this.responseXML);

however;
I don't know how to open it back and parse according to tag or id names

eg. if I do var name = stringedXML.getElementsByTagName(&#39;title&#39;).item(0).text;
I receive and error

How I can parse it after converting this xml to a string

    Hi

    As you have converted the XML into a screen for later use - it is still a string when you use it on. So you need to convert it back to xml to use as XML. You use the parseString method;

    • parseString
    var xml = Ti.XMLparseString(stringedXML);
var name =  xml.getElementsByTagName('title').item(0).text;

    The first line creates an xml object from the parsed string. From there you suse any XML command you would have normally done previously.

    So serializeToString converts a XML object to a string and parseString converts that string back to an XML object.
