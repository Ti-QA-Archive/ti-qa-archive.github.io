[ERROR-SOLUTION] Can’t make name of 4 into type Unicode text. (-1700)

Hello to all boys !

As written in the title of the question I would like to show you the solution that I've taken to resolve the error

/Users/your_user/Library/Application Support/Titanium/mobilesdk/osx/3.1.1GA/iphone/itunes_sync.scpt:[INFO] iTunes sync initiated
 execution error: Can’t make name of 4 into type Unicode text. (-1700)

It is not the first time that I find myself in front of this error, and I know that many others have the same problem happened. I saw a question here on the forum about this issue just last week, but unfortunately can not find it anymore, so I decided to open this "new answer-solution" to solve once and for all this problem.

  1. Go here 

    /Users/your_user/Library/Application Support/Titanium/mobilesdk/osx/3.1.1GA/iphone/

    or

    /Users/your_user/Library/Application Support/Titanium/mobilesdk/osx/(sdk that gave you the error)/iphone/

  2. Locate the file itunes_sync.scpt

  3. Open the file itunes_sync.scpt
  4. Edit the file

original file

tell application "iTunes"
    repeat with n from 1 to the count of sources
        if the kind of source n is iPod then
            try
                update n
            on error errorStr number errorNumber
                log "The device " & name of n & " was unable to sync (" & errorStr & ":" & errorNumber & ")"
            end try
        end if
    end repeat
end tell

edit file

tell application "iTunes"
    repeat with n from 1 to the count of sources
        if the kind of source n is iPod then
            try
                update n
            on error errorStr number errorNumber
                log "The device " & name of source n & " was unable to sync (" & errorStr & ":" & errorNumber & ")"
            end try
        end if
    end repeat
end tell

(just add the word source)

  1. Compile and Run the file
  2. Clean your titanium project
  3. Run your titanium project
  4. No Error :)
