iPad: Split View with a ToolBar giving a menu look

I am trying to create a splitview with a toolbar at the top, which will have multiple buttons. Each button will have its own detail view. With some information. Its like any menu in windows. For eg: File, Edit, View etc. Is it possible with a splitview. If not, then which api will be appropriate in such cases. Also, is it possible to have a cross platform independency with android?

  • Hello,

    Please check the Apple's HIG guide so you can take a look into what kind of UI is expected. The thing that you said like "any menu in windows" might not be accepted in the Apple Store because of Apple's policies.

    You will need to work in order to get a UI that looks and feels like a native one.

    Being said that, you don't get a split view in Android. So, the best way to do this is implement this "split view" with different windows being shown in the app, with custom events handling all the UI communication.
