  • Assuming you are trying to run on android here….

    I had this when my app size went over the 64mg which is the default sdcard size created by titanium. You can use the tool @ <AndroidSDK>/Tools/mksdcard to create a larger sdcard and replace the default one created by titanium in <user>/.titanium/android2.sdcard (on os x, not sure where it is on windows). Alternatively you could probably hack the android source to create a larger one.
