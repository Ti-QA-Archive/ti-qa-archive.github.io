Change color of the tab indicator in an Android Actionbar

Hello,
Does anyone have an example of how to change the color of the tab indicator in the Android actionbar.

This is the style.xml of my own custom theme:

<resources xmlns:android="http://schemas.android.com/apk/res/android">
    <style name="Theme.Creaim" parent="android:style/Theme.Holo.Light">
        <item name="android:actionBarStyle">@style/Theme.Creaim.ActionBar</item>
        <item name="android:actionBarTabStyle">@style/Theme.Creaim.ActionBarTab</item>
        <item name="android:actionMenuTextColor">#ffffff</item>
    </style>
    <style name="Theme.Creaim.ActionBar" parent="android:style/Widget.Holo.Light.ActionBar">
        <item name="android:background">#211e4d</item>
        <item name="android:titleTextStyle">@style/Theme.Creaim.ActionBar.Text</item>
    </style>
    <style name="Theme.Creaim.ActionBar.Text" parent="@android:style/TextAppearance">
        <item name="android:textColor">#ffffff</item>
    </style>
    <style name="Theme.Creaim.ActionBarTab" parent="android:style/Widget.Holo.ActionBar.TabView">
        <item name="android:background">#a80532</item>
    </style>
</resources>

The error message i got is:

/Documents/Titanium_Studio_Workspace/myApp/build/android/res/values/style.xml:14: error: Error retrieving parent for item: No resource found that matches the given name 'android:style/Widget.Holo.ActionBar.TabView'.

What am i doing wrong?
