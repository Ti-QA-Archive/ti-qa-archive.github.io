  • The keyword is this.. AFAIK you cannot change the font of the table row, but you can add a label and change the label's font.. sample code:-

    var row = Ti.UI.createTableViewRow();
var label = Ti.UI.createLabel({font: { fontSize: 20, fontColor: '#444'}});
row.add(label);

    I didn't check whether the code above compiles.. but that is the idea.
    Hope it helps.
