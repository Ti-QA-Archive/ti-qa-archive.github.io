Parameterized Query for Sqlite DB to avoid Sql Injection in Titanium

Can anyone give me example of using Parameterized Query for Sqlite DB to avoid Sql Injection in Titanium.

    Simply replace the values you want replaces by ? in your query. Then simply pass the values you want to search as extra parameters to the execute function.

    Here is an example that searches a person named Bob of age 17:

    var db = Ti.Database.install('mydb1', 'mydb1Installed');

var rows = db.execute('SELECT * FROM people WHERE name = ? AND age = ?', 'Bob', 17);
...
