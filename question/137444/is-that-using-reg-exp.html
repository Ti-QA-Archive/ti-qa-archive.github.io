Is that Using REG-EXP?

dear all. i've JSON from my client like this

{
id: 1,
price_str: "555<sup>,400</sup>",
currency_code: "IDR",
currency_sym: "Rp",
provider_name: "accorhotels.com",
provider_code: "ibishotel.com",
is_direct: true
},

i've parse the output of json, but it become IDR 555<sup>,400</sup>

how to parse that json so the output will IDR 555,400???

please need help

    You could use some RegEx like this :

    var price_str = "555<sup>,400</sup>";
price_str = price_str.replace(/<[\/]?(sup|SUP)[^><]*>/g,"");
// Output should be "555,400"
