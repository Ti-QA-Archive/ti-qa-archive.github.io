Android landscape orientation

Hello, i need to create landscape only oriented app. For ipad i specify 

  <orientations device="ipad">
            <orientation>Ti.UI.LANDSCAPE_LEFT</orientation>
            <orientation>Ti.UI.LANDSCAPE_RIGHT</orientation>
  </orientations>

All works great. App always start in landscape mode.

With android i try to do the same with

<manifest android:versionCode="1" android:versionName="1.0">
        <supports-screens android:anyDensity="true"/>
        <application>
            <activity android:name="org.appcelerator.titanium.TiActivity" 
                      android:screenOrientation="landscape" 
                      android:configChanges="keyboardHidden|orientation">
            </activity>
        </application>
</manifest>

But if device is in portrait mode app start in portrait and only after showing me portrait splash screen switches to landsacape.

How to start app in landsacape mode from launch?
Thanks.
