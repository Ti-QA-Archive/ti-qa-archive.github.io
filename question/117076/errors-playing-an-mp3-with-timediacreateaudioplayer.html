Errors playing an .mp3 with Ti.Media.createAudioPlayer()

Hey everyone,

I'm trying to use a very simple code snippet to play a streamed mp3 file from freesound.org:

var url = "http://media.freesound.org/data/21/previews/21409__greyseraphim__heart_beat_preview.mp3";
var player = Ti.Media.createAudioPlayer({url:url});
player.start();

However everytime I try to run it I get an alert that says Unable to configure network read stream and the following in the console:

2011-03-10 18:39:01.289 ... Error '!dat' trying to set the (null) audio devices' sample rate
2011-03-10 18:39:03.762 ... Audio queue start failed. err: ???á -66681

This happens with any mp3 on any server I throw at it. The same error happens in the emulator and on the phone. I'm using the lasted 1.6 dev kit.

Any ideas?

  • Try this:

    var url = "http://media.freesound.org/data/21/previews/21409__greyseraphim__heart_beat_preview.mp3";
var player = Ti.Media.createAudioPlayer();

player.url = url;
player.start();
