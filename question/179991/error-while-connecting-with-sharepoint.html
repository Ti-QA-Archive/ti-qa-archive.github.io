  • In my opinion a login should never be a get request, for safety reasons.

    Make it a post and set your username and password in a JSON object. And then on send add it as argument

    Like:

    var postData = {
    username: 'username',
    password: 'password'
}
xhr.open("POST", url);
xhr.send(postData);
