JSON parsing problem

How is it possible that this code absolutely works (i.e.: JSON parsing is done without problems):

      var f = Ti.Filesystem.getFile(Titanium.Filesystem.resourcesDirectory,'/lib/data.json'); 
     var content=f.read();
     alert(JSON.parse(content.text));
     var parsed = JSON.parse(content.text);

while this code with the same JSON file gives error:


     var url = "/lib/data.json";
     var xhr=Ti.Network.createHTTPClient({
    onload: function(e) {
       alert(JSON.parse(this.responseText));
      }
      });
      xhr.open("GET", url);
      xhr.send();

The error description is: "JSON Parse error: Unexpected identifier \"no\""
Not able to fix it…

  • You can only request http resources ( eg http://www.somesite.com ) using the httpClient. so this statement

    var url = "/lib/data.json";

    is not a valid web address and what you are probably getting is the first few lines of a response saying 'not found'
