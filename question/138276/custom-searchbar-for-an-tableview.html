Custom searchbar for an tableview

Hello,

I'm trying to do a custom searchbar for a tableview.

I need 2 things: first since my application is in french, I want to remove the accent.
So if I have a tableviewrow with the word Étienne, and someone search etienne I want my row to show (in this example, I replace the é with an e).

And second, if the user type multiple partial word like "App ru" I want the row containing "appcelerator rules" to show.

I looked at the event of both searchbar and tableview but didn't find anything.

I try to make my own searchbar and removing / adding result in the tableview, but it's too slow. Each tableviewrow have 3 views, and 6 labels, and I can have up to 300 rows in my tableview.

I did try to use the function .hide() on the tableviewrow but it only work if I scroll the tableview for the view to be outside of the window and then it get hided, but the height of the row stay.

I did try too to set the height to 0 of my tableviewrow, but it only take some bottom padding off.

So anyone have an idea?
