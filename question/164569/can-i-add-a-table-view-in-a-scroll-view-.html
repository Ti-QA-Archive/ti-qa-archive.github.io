Can i add a table view in a scroll view ??

I am adding a table view in a scroll view, its smoothly working in case of iOS but in case of Android the table view not scroll.
Is there any solution so that i can add a table view in a scrolling window in case of Android ??

  • Hi Neeraj Goel,

    Thanks for raising this issue. In Table View, there is a default scroll property.
    This is not recommended natively in android to put a table view in scroll view.
    I have tested a sample code for your issue.
    You can scroll the table view in android without adding any scrollview. Hope this helps.

    Check Attachment

    Test Case:

    var win = Ti.UI.createWindow({

    backgroundColor:'white'
});

var tableView = Titanium.UI.createTableView({});

var rows = [];
for(var i=0; i<50; i++){
    rows.push(Titanium.UI.createTableViewRow({
        title:'Test ' + String(i)
    }));
};
tableView.setData(rows);
win.add(tableView);

win.open();

    For more info please visit tableview-in-scrollview-doesnt-scroll-android

    And tableview-and-scrollview-problems
