Accessing an ImageView object from addEventListener to Row

Hi,

I have a tableview.
In that tableview, I have a tableviewRow.
In that tableviewRow, i have an image that is added in each row.

I would just like to know how I can access the image object from the TableView Event listener?

OK.. i'll give an example:


var row = Ti.UI.createTableViewRow({});

var imageViewVar = Ti.UI.createImageView(
{
    image: 'images/pic.png',

});

row.add(imageViewVar);

.. ...etc

And then I have the TableView Listener:

tableviewVar.addEventListener('click', function(e)
{

////? I need to access : imageViewVar.image property from here? 

});

Now, my guess is to assign the imageView a property like "is_image: yes"

and then do a 

e.source.is_image.image = ''?

Any ideas? Sorry guys, I should be beyond this issue.

    Hello EB,

    Well assuming that you have the complete code, you are near to what you want to do. Just remember that a row is a actually a view, attached to a tableView. A row, in your case contains more views (i.e. one is the imageView). So, back to your question, yes, e.source is where you can find information on view that it's been clicked. Try the following:

    tableView.addEventListener('click', function(e) {
alert(e.source); 
});

    this will give you the details of the view clicked.
