Really simple Android program behaves unexpectedly!

Hi guys,

This is the simplest piece of code running on Win Vista and Android 2.0.1 V8
Also tried it on my SE Xperia Arc S and it behaves the same way.

I would have expected to see a blue screen with white numbers counting very fast up to 2000 but all I get is the Titanium 'splash' screen and then 'Done!' at the very end!

Using a setTimeout() doesn't work as it doesn't seem to actually pause the program at that point - it seems to merely delay when that line is executed but carries on to do the rest…

Can anyone explain why it behaves like this please?

Many thanks!

var counter = 0;

color:'#fff',

});

win.add(label);

win.open(); 

do
{
    counter+=1;
    label.text=counter + ' ';
} while (counter<2000);

  • Hi Robin,

    Look at this code, I modify your code little bit.

    var counter = 0;

var win = Ti.UI.createWindow({backgroundColor:'#00f'});

var label = Ti.UI.createLabel({ color:'#fff', text:'<blank>' });

win.add(label);

win.open();

// do
// {
    // counter+=1;
    // label.text=counter + ' ';
// } while (counter<2000);
var t = setInterval(function(e){
      counter+=1;
    label.text=counter + ' ';
    if(counter>99)
            clearInterval(t);
},50);

label.text = 'Done!';

    In Titanium javascript runs in single thread. so your program runs sequentially.

    This is for your understanding. In this program you can not see the 'Done' anymore.
