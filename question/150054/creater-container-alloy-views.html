Creater Container Alloy Views

Hello there, i'm trying to create an Alloy view to contain other views, much like a TabView or something like that.

So what I want to do is something like

mywindow.xml

<Alloy>
    <Window>
        <Label>All of my windows should have this label</Label>
        <!-- Added elements go here -->
    </Window>
</Alloy>

and then in another file do something like

<Alloy>
    <Require type="view" src="mywindow" id="window">
        <Button>OK</Button>
    </Require>
</Alloy>

Additionally, if this is possible, I need the ability to access the added elements from the mywindow controller, something like

$.elementsAddedFromOutside

to do things like accessing those widgets properties, such as title and so, that may be setted in a tss.

Can this be done? If so, how?

Thanks in advance for any help you can provide.

Cheers.
