  • To see if the file was actually written to the simulator/emulator during testing:

    for iOS, open Finder and browse to $HOME/Library/Application Support/iPhone Simulator/{sdk-version}/Applications/{application-guid}/Documents

    for android, start ddms from android-sdk-location/tools. At the top, choose Device, File Explorer. Browse down to data/data/{app id}/app_appdata

    The other half of your question, as Aaron stated, requires more information.
