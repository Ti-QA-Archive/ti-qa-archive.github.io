Auto height view

I want to create a view with height auto I am using height:'auto', though its working fine in ios but bot wokring in android . In android its creating a view with 100% height . Please suggest how can i create a view with height auto

  • Hi Gaurav,

    put your view in scrollview and set height and width of scrollview try below code.

    var scrollView = Ti.UI.createScrollView({
    width : 320,
    height : 416,
    contentHeight : 'auto',
    contentWidth : 'auto',
});

  • If you are using Titanium SDK 2.0.x or later, then use Ti.UI.SIZE/Ti.UI.FILL based on your requirements. Check this documentation for the layout changes from SDK 2.0 @ (https://wiki.appcelerator.org/display/guides/Titanium+2.0+Layout+Changes).

    Other approach is to use left,right,top & bottom property with value 0(zero), which will stretch your view to Fill the area occupied by its parent view.
