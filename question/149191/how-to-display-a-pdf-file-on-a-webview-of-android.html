How to display a PDF file on a WebView of Android?

on Ipad i can display a PDF file on a webView, but on Android i can't. Is there a way to do something similar with Android? Thanks foy your help.

this code works fine on Ipad, but does not display anything on android

var popWeb = Ti.UI.createWebView({url:'http://…..file.pdf'});

    var w = Ti.UI.createWindow({
            barColor: 'orange',
            //barImage: '/images/navbar.png'
            title: e.rowData.titulo
    });
    w.add(popWeb);

    hi,

    try this one

    var pdfUrl = 'http://docs.google.com/viewer?embedded=true&url=' + <your pdf url>;

var popWeb = Ti.UI.createWebView({url:pdfUrl});

    var w = Ti.UI.createWindow({
            barColor: 'orange',
            //barImage: '/images/navbar.png'
            title: e.rowData.titulo
    });
    w.add(popWeb);
