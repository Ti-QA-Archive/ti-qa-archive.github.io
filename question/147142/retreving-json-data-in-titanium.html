    Hi

    You made a common mistake, you think the array is in one place but it is in another.

    Change these two lines;

    for ( i = 0; i < json.nodes.node.length; i++) {
    fighter = json.nodes.node[i];

    To

    for ( i = 0; i < json.nodeslength; i++) {
    fighter = json.nodes[i].node;

    As your feed starts like this;

    {
  nodes: [
    {
      node: {
        nid: "10",
        title: "(Chinese)",
        ...
    },
    {
      node: {
        nid: "6",
        title: "Cui Jugis Nostrud Venio",
        ...
    },
    ...
  ]
}

    It is nodes that has the array; nodes: […], so that is where you request the length and use the index. Whereas node is a child of that array element.
