  • FYI change is going to fire every time you make a selection change to the picker.

    There is no DateTime / Date type in SQLite, so I store them as SQL strings in the format YYYY-MM-DD HH:MM:SS

    What you want to do is convert the date to a string using momentjs and then you can hit your database to bring results back.

    You can also use momentjs to parse the string result from the db into a date object for you.
