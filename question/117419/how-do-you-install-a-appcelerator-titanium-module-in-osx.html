How do you install a Appcelerator Titanium Module in OSX

I cannot find any instructions on how to install modules into Titanium

I have downloaded a trial version of the TVOut module from the Appcelerator webpage

https://developer.appcelerator.com/package/28/tv-out/media

however it doesn't say how to install this into Titanium?

I know it needs to end up in /Library/Application Support/Titanium/Modules but just unzipping into this location is not enough.

What is the command line script which I can run on the zip which will allow me to use it?

thx

    Hi there,
    put the folder to:

    /Library/Application Support/Titanium/Modules/iphone/Your.Module.Folder

    To use it add to tiapp.xml the following codes:

        <modules>
     <module version="1.0">Module Name</module>
    </modules>

    be careful, the version number has to be the same of the module's version number.

    cheers

    Andrea

  • As Andrea says, add 

    
<module version="0.1">ti.tvout</module>

    to your tiapp.xml file (in the modules tag near the bottom)

    Then see the app.js example in the 'examples' folder within the module directory where you unzipped it, to see how to 'require' it in your scripts, so you can use it.

    There is also some documentation in the documentation folder. ;)
