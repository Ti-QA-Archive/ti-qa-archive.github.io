Open new window with animation also animates its children

I want to slide a new window in from the right side, if the user clicks a button. There is no Tab Group involved and I do not want to use a tab group. Therefore I created an animation (as shown in the code below). The new window (var self) contains some children (views, labels).

As soon as the function slideOpen() is being called, you can see the animation effect, which is a window sliding from right into the device screen. However, there is an undesired effect. All the children of that window are sliding into the device screen from the top right. What I want is that all children of the window are already at their place, before the window slides in from the right side.

How can I fix this?

var self = Ti.UI.createWindow();
self.left = Titanium.Platform.displayCaps.platformWidth;

self.slideOpen = function() {
    var slide_it_left = Titanium.UI.createAnimation();
    slide_it_left.left = 0;
    // to put it back to the left side of the window
    slide_it_left.duration = 1000;
    self.open(slide_it_left);
}
