    • Hi Oliver,

      If you create the empty table with blank data it won't have the 1 x 3row height there.

          $.table = Titanium.UI.createTableView({
        data: [{}]
    });

      Then when you setData later its fine..

      This allows you to create the empty table.

      It's still a bug in 1.7.x as well as the 1.8.x branches from nightlies I've tested..

      But its a workaround.
