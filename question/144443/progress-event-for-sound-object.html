  • Since the sound object doesn't have any events to know how much time has been played so far. You'll have to check the current time at a specific interval (every second should do it) and update the progress bar appropriately.

    I can't give you the code precisely, but here is how I would go about it.

    • Create your sound object
    • Set you progressBar max property with the sound's duration
    • Once you started playing your sound
      • Check the time property of the audio player and assign it to the progress bar's value.
      • Repeat the operation every second by using the setInterval JavaScript function.

    Hope this helps
