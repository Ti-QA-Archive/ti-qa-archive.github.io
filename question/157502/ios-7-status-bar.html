  • Hello Samuele Coppedè ,

    On iOS 7, the status bar and navigation bar are translucent by default. On iOS 6 and prior, they are not translucent.The status bar is controlled by the currently displayed window and can no longer be controlled on the fly in iOS. On iOS 6 and prior, setting the statusBarStyle property to new Titanium.UI.iPhone.StatusBar.LIGHT_CONTENT constant behaves the same as setting it to Titanium.UI.iPhone.StatusBar.TRANSLUCENT_BLACK. On iOS 7 and later, setting the statusBarStyle property to either Titanium.UI.iPhone.StatusBar.TRANSLUCENT_BLACK or Titanium.UI.iPhone.StatusBar.OPAQUE_BLACK behaves the same as setting it to the new Titanium.UI.iPhone.StatusBar.LIGHT_CONTENT constant.

    For more information, please visit following:- iOS7MigrationGuide

  • If you are using Titanium SDK 3.1.3 GA, set the following properties when you create the window.

    var window = Titanium.UI.createWindow({
   statusBarStyle : Titanium.UI.iPhone.StatusBar.LIGHT_CONTENT,
   fullscreen:true
});

    But this code does not give the same output if the app is built using Titanium SDK 3.1.2 GA or lesser.
