    You need to use an Android SDK bigger than 2.1. I think you are trying to install on a 2.1 emulator.
    Am I right?

    permalink

    • Yes, that did the trick.

      For anyone else reading this… The settings are in the drop down menu you used to launch the emulator. Select "Run Configurations". Double click on "Titanium Android Emulator" and create a new emulator with a higher SDK. I was trying to do it from within the Preferences window which doesn't do the same thing.
