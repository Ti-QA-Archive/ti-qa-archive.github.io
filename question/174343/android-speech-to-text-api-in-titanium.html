  • You can use bencoding.Utterance https://github.com/benbahrenburg/Utterance

    Examples here https://github.com/benbahrenburg/Utterance/tree/master/android/example

    When using Android, make sure you sent the voice or language with the correct locale as by default it will use the device locale when speaking.
