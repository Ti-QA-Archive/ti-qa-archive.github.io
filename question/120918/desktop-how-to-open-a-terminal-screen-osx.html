[Desktop] How to open a Terminal screen (OSX)

I wonder how I can open a Mac OSX Terminal screen with a SSH session. I tried a few things but cant get it working.

var vProc = Titanium.Process.createProcess({
     args: ['/usr/bin/ssh', '-p 22 root@10.0.0.1']
});

vProc.launch();

and

Titanium.Platform.openApplication('/usr/bin/ssh -p 22 root@10.0.0.1');

Using the first one, I get no results at all. But using the second one I get this in Titanium Studio:

2011-06-12 22:20:56.962 MyProject[7433:b07] LSOpenFromURLSpec() returned -43 for application (null) path /usr/bin/ssh -p 22 root@10.0.0.1.
2011-06-12 22:20:56.965 MyProject[7433:b07] LSOpenFromURLSpec() returned -43 for application /usr/bin/ssh -p 22 root@10.0.0.1 path (null).

  • Kosso,

    When using this line of code:

    Titanium.Platform.openApplication("open -n Terminal");

    I see this in my debugger:

    2011-06-14 21:20:51.322 SSH Manager[14266:b07] LSOpenFromURLSpec() returned -43 for application (null) path open -n Terminal.

    When using 

    Titanium.Platform.openApplication("Terminal");

    It works and opens the Terminal, somehow it seems that it is not possible to provide any parameters to the 'application' which I want to execute.
