Missing architecture

Hi,
in my Titanium studio I have since a lot of days problems with provisioning files (I cannot see it). Thats why I have started the compilation process for store from XCODE. Device testing makes no trouble.

I start Product/ARCHIV, no problems. No I start the uploader in organizer then the apple server answered:

"iPhone/iPod Touch: application executable is missing a required architecture. Al least one of the following architecture(s) must be present: armv6"

I have no modified the from Titanium build project. Any ideas?

Rainer

    Adding the following to the tiapp.xml allow the app to be validated:

    <ios>
        <plist>
            <dict>
                <key>UIRequiredDeviceCapabilities</key>
                <array>
                    <string>armv6</string>
                    <string>armv7</string>
                </array>
            </dict>
        </plist>
    </ios>

    or this

    <ios>
    <min-ios-ver>4.3</min-ios-ver>
</ios>
