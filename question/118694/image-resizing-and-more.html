Image resizing (and more..)

THIS IS A GUIDE

I came across few questions regarding this issue for manipulating images for decreasing resolution or what not - and none of the solutions i saw was perfect so i decided to give it a try.

For image resizing:

You can always use the built-in function for TiBlob.

here is the function's signature:

[TiBlob] imageAsResized([int] width, [int] height)

For example:

var blob = myimage.toBlob();
// blob.width = 1936
// blob.height = 2592
// blob.size = 5018112
var smallBlob = blob.imageAsResized(80, 80);
// smallBlob.width = 80
// smallBlob.height = 80
// smallBlob.size = 6400

Sadly appcelerator dev team didn't finish writing the entire API for Titanium components so what i usually do when i want to see the entire API is visit the *.h/m files.
For the complete list of TiBlob properties and methods see TiBlob.m/TiBlob.h

Here's a preview for some of TiBlob functions:

  • [TiBlob] imageWithAlpha()
  • [TiBlob] imageWithTransparentBorder([int] size)
  • [TiBlob] imageWithRoundedCorner([int] cornerSize, [optional] [int] borderSize)
  • [TiBlob] imageAsThumbnail([int] size, [optional] [int] borderSize, [optional] [int] cornerRadius)
  • [TiBlob] imageAsResized([int] width, [int] height)
  • [TiBlob] imageAsCropped([dictionary] config)

* where config is of the form:
{ width: [float], height: [float], x: [float], y: [float] }

Happy coding
