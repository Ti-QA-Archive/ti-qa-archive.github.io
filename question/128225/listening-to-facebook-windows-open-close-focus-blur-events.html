    • I don't see how having to wait just a second is a problem?

      If you read the Facebook module docs, you'll see that the login event object has a cancelled property you can check. This property is true if the event was cancelled, like for example by clicking the X button. I tried it myself, and it works correctly. The login event is fired when the X button is clicked.
