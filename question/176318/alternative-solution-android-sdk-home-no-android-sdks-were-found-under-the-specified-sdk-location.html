Alternative solution: [Android SDK Home] No Android SDKs were found under the specified SDK location

After spend the whole friday trying to solve this problem with common solutions (reinstall SDK, check paths, unistall&reinstall titanium…) I finally found the real problem in my case: Genymotion.

To solve the problem I only had to delete the genymotion.js file at C:\Users\manuel.conde\AppData\Roaming\Titanium\mobilesdk\win32\3.3.0.GA\node_modules\titanium-sdk\lib\emulators and instantly, the SDK was fine again.

To see more details, check the JIRA I opened.
