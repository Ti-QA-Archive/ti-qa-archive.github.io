How do you monitor mic level on Android?

Does anyone know how to monitor the mic level on Android (0.0 - 1.0) without the recording app popping up?

It's simple on iPhone:
volLevel = Titanium.Media.peakMicrophonePower

But that doesn't work on Android.

I tried this:

var RECORD_SOUND_ACTION = "android.provider.MediaStore.RECORD_SOUND";

var intent = Titanium.Android.createIntent({
        action: RECORD_SOUND_ACTION
    });

    Titanium.Android.currentActivity.startActivityForResult(intent, function(e) {
        if (e.error) {
            Ti.API.info('Error: ' + e.error);
        } else {
            if (e.resultCode === Titanium.Android.RESULT_OK) {
                soundUri = e.intent.data;
                Ti.API.info('Audio Captured.  Content URI: ' + soundURI);
            } else {
                Ti.API.info('Canceled/Error? Result code: ' + e.resultCode);
            }
        }
    });

But this pops up the audio recording app. I need it to stay in the background and just report the mic level.

Thanks!
RJ

  • AFAIK, that's not possible on Android. The Android Audio recorder is only possible using the new 'intents' stuff and your exmaple forces open the new window for the audio recorder.

    Maybe there's some other 'intent' app out there (google 'open intents') which can do exactly what you need.
