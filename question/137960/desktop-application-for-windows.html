    Hi Raju. The TideSDK project is close to launching our first release candidate quite soon. We are continuing with our build infrastructure. You can keep appraised of our progress using some of the links above. We are quite excited about the future

    I think the first think you can do to get started is to get a reasonably current Desktop SDK and documentation to start. Here are some links for you:

    http://builds.appcelerator.com.s3.amazonaws.com/RC/desktopsdk-1.2.0.RC4-win32.zip
    http://builds.appcelerator.com.s3.amazonaws.com/RC/desktopsdk-1.2.0.RC4-osx.zip

    http://developer.appcelerator.com.s3.amazonaws.com/release-candidate-apidocs/desktop-api-docs-1.2.0-rc4.tar.gz

    You can install this and developer with Titanium Studio for the time being. We are available at #tidesdk on irc on freenode so feel free to drop by. Please subscribe to our mailing list as well at https://groups.google.com/forum/?fromgroups&hl=en#!forum/tidesdk

    Hope that helps

    David
