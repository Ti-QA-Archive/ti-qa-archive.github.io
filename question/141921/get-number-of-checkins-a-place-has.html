Get number of Checkins a place has

Hi. I'm trying to find a method for getting the number of checkins a place has via ACS!?
I'm doing a query on all my places, and I would like to sort them after how many checkins they have!?
I'm trying to make a list of the most popular places.

Is that possible??

  • Hi Jan,

    If you're still looking for an answer

    you can use the following:

    Cloud.Checkins.query({
                place_id : placeID,
            }, function(e) {
                if (e.success) {
                    var checkins = e.checkins.length;
                    checkinCount.text = "There have been " + checkins + " checkins here.";
                };
            });

    • The reason I want this is for saving API calls. If I for example want to know which of my places have the most checkins I first need to get all my places and then check each of them for their checkin count. So something that could be done with one API call ends up with n calls, where n is number of places you have in your ACS (could be 1000+) I ended up adding a custom field to my places which holds the numer of times someone checks in and manually update to this for each checkin. This way it results in one extra API call for each checking in so I guess that isn't to bad :-)
