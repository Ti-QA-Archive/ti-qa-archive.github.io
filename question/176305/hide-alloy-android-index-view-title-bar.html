Hide Alloy Android index view title bar

Hi,

I've created a alloy Project and would like the index window to be shown without a title bar. I've read a couple of answers here, but nothing seems to work for me. This is what I have done so far:

Set navbar-hidden and fullscreen in tiapp.xml

<fullscreen>true</fullscreen>
<navbar-hidden>true</navbar-hidden>

Created a global tss file called app.tss and set navBarHidden

"Window": {
    navBarHidden: true,
}

I've done the same in index.tss with the container class selector.

Tried to disable it directly in index.xml 

<Window navBarHidden="true" class="container">

I'm new to developing with TitaniumStudio and this is my first alloy app. It might be the simplest thing in the world to achieve what I want, but I just can't get there without you guys.

    I'm assuming you are using the latest version 3.3.0.GA - This version changed the way that the title (Action) bar is displayed because of the introduction of the appcompat library. So in the latest version you now have two options for a full screen (no Action Bar) window as described here
