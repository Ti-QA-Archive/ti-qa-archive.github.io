I'm looking for a working example of someone using the popover control with the alloy framework. Please post if you have something. Thanks.

    Here is a basic example for a popover that contains a view.

    popover.xml

    <Alloy>
    <Popover id="popover" ns="Ti.UI.iPad">
        <View id="popview" />
    </Popover>
</Alloy>

    popover.tss

    "#popview": {
    height: 100,
    width: 100,
    backgroundColor: 'pink'
},
"#popover":{
    height: 100,
    width: 100,
    title: 'Hello World'
}

    index.xml

    <Alloy>
    <Window class="container">
        <Button id="popbutton" title="Popover Open" onClick="togglePopover"/>
    </Window>
</Alloy>

    index.js

    var pop =  Alloy.createController('popover').getView();

function togglePopover() {
    pop.show({view: $.popbutton});
}

$.index.open();

    I hope this helps :)
