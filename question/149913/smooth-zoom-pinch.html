Smooth zoom (pinch)

This is the code I use to zoom an image view. It works but the zoom is not smooth. After some fingers movements, it resizes abruptly, allowing very little control to the user. Any workaround?

    function pinch(e) {
        var _width = Math.round(e.source.width * e.scale);
        var _height = Math.round(e.source.height * e.scale);

        if (_width < 100 && _height < 100)
            return;

        e.source.width = _width;
        e.source.height = _height;

        e.source.animate({
            width: e.source.width,
            height: e.source.height,
            duration: 1,
        });
    }
