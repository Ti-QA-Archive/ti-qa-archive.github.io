Set System Volume

I'm creating an app for iOS 4.2 that when the user puts it into the background, I'd like to be able to play a sound. All that works fine, but if the user has the system volume off, it will not be heard.

Is there a way to set the system volume?

I've tried Ti.Media.setVolume(1.0) but have not had any luck.

Any help or suggestions would be greatly appreciated. I know the functionality is supported by the OS because I've downloaded other apps from the Apple App Store that do automatically set the system volume.

Any help would be greatly appreciated.

Thanks in advance

  • I'm pretty sure the Apple don't allow apps to set the volume. It has to be done by the user.

    There's no such method called setVolume in Titanium.

    One thing you could do though, is detect if the user has the audio muted or not using Titanium.Media.AUDIO_MUTED
