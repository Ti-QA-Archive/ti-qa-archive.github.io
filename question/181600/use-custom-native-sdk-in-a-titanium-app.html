Use custom native sdk in a titanium app

Application type: mobile
Titanium SDK: 3.5.1.GA
Platform & version: iOS & Android
Host Operating System: Windows 8.1
Titanium Studio: 3.4.1.201410281727

Hi,
I'm currently developing an app for a client that has provided me a custom SDK that takes care of some tasks the app have to do. The sdk is write in both java for android and objective c for ios, but I don’t know how to implement them in the titanium app. That sdk doesn’t have to return anything to the app, the only thing the titanium app has to do is launch it with a few parameters so it does its work. I've been looking at the module creation, but I don’t know if that's exactly what I need to do. The client has given me a "Hello world" android code with the sdk working, so i know what functions I have to call and what parameters are needed.

Any suggestions? Thanks in advance.
