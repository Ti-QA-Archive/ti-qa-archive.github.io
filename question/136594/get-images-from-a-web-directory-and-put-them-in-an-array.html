  • Hi Matt,

    I created a little php script for you that will pull all the image files from the directory then turn that into a JSON string.

    https://gist.github.com/2629102

    So what I would do is save that gist into a index.php file. Then put that file in this folder, http://imattic.com/paper/. Then you can call http://imattic.com/paper/ from your app and it will return a JSON string with all of the latest files in it.

    You could then call JSON.parse(data object here) in your app and you will have an array with all of your images in it automatically.
