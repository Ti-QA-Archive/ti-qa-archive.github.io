    • Ok, step by step (but no code examples, you'll have to do it yourself :) )

      • you declare the images var before you start the httpclient
        ~~~
        var images=[];

      // start here the httpclient

      - You server needs to return a valid json, something like

      '[{ "image":"http://a1969.phobos.apple.com/us/r30/Purple/v4/1d/88/55/1d885533-e23f-5a71-856d-7e80691f21e0/Icon.png", "title":"AIG"},
      { "image":"https://lh5.ggpht.com/yRUw11DbeZ-XgJG-a1lBViQxH7UjwBsy8QhH0ur9VejpLZK6tNLLarpuRqestCiB6W4=w300", "title":"SEGA"}]'

      - The load event of the httpclient will parse that json with

      onload: function(){
      images = JSON.parse(this.responseText);
      // start here to build the table
      }
      ~~~

      At his point you will have the images variable populated, however only here is where you will be able to start the population of the table or else it will still be an empty array

      Makes sense?
