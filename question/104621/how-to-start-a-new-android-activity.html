How to start a new Android Activity?

I'm trying to start a new custom Android activity with Appcelerator and having some trouble. I'm not sure where to put the java file in my Appcelerator project and how to correctly link to it.

Here's the HelloAndroid.java activity I'm trying to run from my app.js
I've tested this code in a simple Android project (no Appcelerator) and it runs:

package Resources;

import android.app.Activity;
import android.os.Bundle;
import android.widget.TextView;

public class HelloAndroid extends Activity {
    /** Called when the activity is first created. 
     */
    @Override
    public void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        TextView tv = new TextView(this);
        tv.setText("Hello thar!");
        setContentView(tv);
    }
}

Right now I just threw it in the Resources folder, but I'm sure that's not right.

Here's what's in my app.js: 

Titanium.API.info("Testing");

var intent = Ti.Android.createIntent({
               action: Ti.Android.ACTION_MAIN,
               packageName: 'Resources',
               className: 'HelloAndroid.java'
});
intent.addCategory(Ti.Android.CATEGORY_LAUNCHER);
Ti.Android.currentActivity.startActivity(intent);

Again, not sure what do for the packageName. I think I want the java file to end up in my 'src' folder in the 'Build' folder, but I don't know how to make Appcelerator put it there.

I've also added the following to my Tiapp.xml so Titanium will put it in the AndroidManifest.xml:

<android xmlns:android="http://schemas.android.com/apk/res/android">
               <manifest>
               <application>
                              <activity android:name=".HelloAndroid" />
                              </application>
               </manifest>
               </android>

I'm pretty sure the full packageName has to go before .HelloAndroid, but I'm not sure what to put there.

When I test the project with Titanium, I get a runtime error telling me that it can't find my activity and it asks if I linked to it in my AndroidManifest.xml.

Does anyone know how to do this? @_@
