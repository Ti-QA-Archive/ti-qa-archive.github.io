  • Its because the nextwin is being opened at the top of the previos window. Creating always new windows is in this case a bad idea. Instead make a function which gives you the correct answer window and one which changes the Content of the question instead of creating new window. Why would you like to create Windows, what Kind of benefit you expect ? Also you may find this module useful
