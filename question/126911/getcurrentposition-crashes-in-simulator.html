getCurrentPosition crashes in simulator

I try to do something like this in the simulator with 1.8.0

var currentPosition = Ti.Geolocation.getCurrentPosition();

What I get is this:

[ERROR] Invalid type passed to function. expected: KrollCallback, was: (null)  in -[GeolocationModule getCurrentPosition:] (GeolocationModule.mm:514)

Anyone an idea of what is happen here? documentation says, no parameter needed.

    you need the callback… something like this

    
Titanium.Geolocation.getCurrentPosition(function(e) {

   var currentLocation;

   if(e.error) {
      currentLocation = 'error: ' + JSON.stringify(e.error);
      return;
   }

   var longitude = e.coords.longitude;
   var latitude = e.coords.latitude;
   Ti.API.info('long:' + longitude + ' lat: ' + latitude);
   return;
});
