Label Width

var lbl = Ti.UI.createLabel({
        text : txt,
        top : position,
        height : 20,
        left : left_postion,
        font : {
            fontSize : 15,
            fontWeight : bold ? 'bold' : 'normal',
            fontFamily : "Arial"
        }
    });
label_end = lbl_obj.toImage().width;

On Andriod this returns 100, but on iPhone the correct label width is returned?
Any reason why?
or just bug?

  • If you are using SDK 2.x, then Ti.UI.SIZE is used as default value. To get the width you will need to listen to the postlayout Event of the label and retrieve the width with lbl.size.width.
