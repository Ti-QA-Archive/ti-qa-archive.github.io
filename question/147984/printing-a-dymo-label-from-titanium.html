  • I've been through a couple of ideas. One of them was to use the android libraries from http://dymodevelopers.wordpress.com/2011/07/19/dymo-label-mobile-sdk-for-android-beta/ and create a Titanium Module but it is only in beta and have not been updated in a long time, so I'm not too sure about that.

    Another idea, and probably the best so far, but requires a lot of work, is to use the regular DYMO SDK and use one of the high-level samples to create some kind of server software that can communicate with the app by regular TCP. But I have not any experience in any of the programming languages that the samples provides. That's why I wanted to hear if someone has found a simpler solution.
