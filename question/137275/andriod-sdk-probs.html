  • I installed, on Win 7, to the following locations to get rid of white-space

    Python to C:\Python27

    Android SDK to C:\Android

    java and javac to C:\Java\jdk1.7.0_04\bin

    Git to C:\Git

    and I added the following environmental variables

    JAVA_HOME to C:\Java\jdk1.7.0_04

    JAVA_HOME\bin to C:\Java\jdk1.7.0_04\bin

    Path to C:\Android\android-sdk\tools; C:\Android\android-sdk\platform-tools; C:\Java\jdk1.7.0_04\bin
