Connect to device via USB or Bluetooth?

I have an external device (Lego NXT brick) that I'd like to communicate with for sensor reading and motor control, but it can only be connected to over USB or Bluetooth. Is this possible to do with either the desktop or mobile versions of Titanium? I know I can use intermediary software on my laptop to bridge the connection, but that's a huge hassle that I'd rather not have to deal with.

  • You can choose to embed Python or Ruby into a Titanium Desktop application, so have the full power of their libraries at your disposal, then call them from within your application (from either Python/Ruby/Javascript)

    I started writing an NXT library for ruby a few years ago, it's probably still on google code somewhere, although I'm sure other have taken it a lot further than I did by now.
