Getting the current table row ?

I have a new app which has functionality wherein you tap a row to toggle a checkmark on or off, as well as a variable of "chosen". I have done this before with a previous app, so I was trying to just paste in my code again:

// when row is tapped, toggle check mark
tv.addEventListener('click', function(e) {
    var rowNum = e.index;
    Ti.API.info('clicked on ' + rowNum);
    var currentRow = tv.data[0].rows[rowNum];
    Ti.API.info('current row is ' + currentRow);

    var currentRow = tv.data[0].rows[rowNum];
    if(currentRow.children[0].backgroundImage == 'images/checkmark-off.jpg') {
        currentRow.children[0].backgroundImage = 'images/checkmark-on.jpg';
    } else {
        currentRow.children[0].backgroundImage = 'images/checkmark-off.jpg';
    }
});

However, for some reason I can't understand, this time currentRow is coming up as "undefined". What am I missing?
