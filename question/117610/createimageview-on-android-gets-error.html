createImageView on Android gets error

Application has stopped unexpectedly when i try to add the logo to container.

Where can i get trace to see actual problem?

My traces works but tell me nothing, is there a log of the error?

Code works fine on iPhone, just trying to run it on Android.

Is there something special about images?

Here is code from app.js:

var viewContainer = Titanium.UI.createView({
top:60,
width:screenWidth,
height:screenHeight
});

var vh_im_jhsLogo = Titanium.UI.createImageView({
bottom:10,
height:42,
image:'images/im_header_jhsLogo_BW.png',
left:10,
opacity:1,
width:160
});

viewContainer.add(vh_im_jhsLogo);

Thanks!

    Try using logcat, if you find it easier than the Titanium Developer Run Emulator screen. Hence, at the command line:

    adb -e logcat

    This will tell you why it's failing.

    Note that your path to the image images&#x2F;im_header_jhsLogo_BW.png will be relative to the JS context.

    Cheers
