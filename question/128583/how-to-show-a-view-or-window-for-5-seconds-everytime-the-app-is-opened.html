How to Show a View or Window for 5 seconds everytime the App is Opened

Hi

I need to be able to open and show a created view or window everytime the user opens up the app. I know i can automatically use the Default.png image when the app is first opened but i need to be able to bring up a view or image if it is put in the background and the user re-opens or resumes the app. Is this possible? I would then use the below function to remove it after 5 seconds:

setTimeout(function() {
  // close window or remove view
  win.close();  
}, 50000);

p.s i am using the Tweetanium structure of loading everything at runtime

Thank you

    Here's some code for you. It will open the modal window when the app loads, open it when the app resumes, and close the window after five seconds each time.

    //Create our modal window (I'm assuming the window is in a file)
var win = Titanium.UI.createWindow({
    url:     'splashscreen.js',
    title:    'Splash Screen',
    modal: true
});

//Start the timer when the modal window opens
win.addEventListener("open", function () {
    setTimeout(function(){win.close();}, 5000);
});

//When the app resumes we want to open the modal again
Ti.App.addEventListener("resumed", function () {
    win.open();
});

//Open the modal window when it first loads
win.open();
