    Ti.UI.ImageView has a globalPoint property for most of its events, that will allow you to access the x,y coordinates that are returned. See view_events.js and view_events_2.js for a demonstration of this in the KitchenSink.

  • I am doing something similar and am doing this:

    view.addEventListener('touchstart', function (e) {
    Titanium.API.info('touchstart fired x: ' + Math.floor(e.x) + 
                      ' y: ' + Math.floor(e.y));

    if ((e.x >= 80 && e.x <= 260) && (e.y >= 80 && e.y <= 160)) {
        soundA.play();
    }
    else if ((e.x >= 20 && e.x <= 300) && (e.y >= 200 && e.y <= 400)) {
        soundB.play();
    }
});
