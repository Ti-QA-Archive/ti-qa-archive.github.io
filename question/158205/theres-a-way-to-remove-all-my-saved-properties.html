There's a way to remove all my saved properties?

I'm trying to remove all my saved properties from my app but I don't know how since the example provider from docs show every properties of my app including the app uuid.

example from docs:

var props = Ti.App.Properties.listProperties();

for (var i=0, ilen=props.length; i<ilen; i++){
    var value = Ti.App.Properties.getString(props[i]);
    Ti.API.info(props[i] + ' = ' + value);
}

there's a way to get all properties that I have created ?

Thanks

3.1.3 GA iOS Android

  • If you format your own property names in a unique way then you can easily identify them.. (untested but hopefully you get the idea);

    Ti.App.Properties.setString('myapp:badgecount', 10);

for (var i=0, ilen=props.length; i<ilen; i++){
    if(props[i].indexOf('myapp:') == 0){
        alert('my property!');
    }
}

    eg Ti.App.Properties.setString(&#39;a_property_name&#39;, null);
