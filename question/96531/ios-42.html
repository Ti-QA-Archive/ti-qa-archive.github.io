  • Same issue here… Ryan Gartin suggested this, didn't work for me but you could try:

    *Make sure XCode is installed for osx. 1. Download Titanium from appcelerator.com 2. Install Titanium. 3. Run Titanium, it will find an update for the Titanium itself. 4. It will also download the latest mobilesdk 5. Create a project. 6. Compile and run in emulator. 7. If you want to run on device, this is where you have to follow the on-screen messages carefully. But basically: 8. Get all your provisioning set up with apple and in your keychain. 9. Load those files into Titanium via the Run on Device tab. 10. It will compile and drop in iTunes. For me, I take the .IPA and move it to another computer where I still sync my phone. Sync.
