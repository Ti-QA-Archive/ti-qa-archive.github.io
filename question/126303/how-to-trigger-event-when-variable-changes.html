How to trigger event when variable changes?

Hello!
I have i TableView, where the tableviewrows are created 'manually'. I wan't to fire the event 'selected' when the rows variable 'selected' changes. How can i accomplish that? I know how to fire an event, but not how to listen for variable changes.

  • To fire an event on variable change you would have to poll the variable use a setInterval method, but this would be expensive (use a lot of memory)

    If your only goal is to see when a row is clicked, then you would want to add an event listener on the table for clicks, and when the table is clicked see if the click was on a row, then if it was, get the row ID.
