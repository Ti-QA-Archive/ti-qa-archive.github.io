Android Windows Changing Problem

I have 2 diferent .js files that are my "windows", when the project opens i call for "login.js" window, and then i have a button that leaves me to window "menu.js". the problem is that when close the second window, to return to the first one, i can't interact with the controls in the login page.

    Try using this adapted version of your code as a starting point:

    app.js:

    var login = Ti.UI.createWindow({
  backgroundColor:'red',
  exitonclose:true,
  layout: 'vertical',
  title:'Login',
  url: 'login.js'
});
login.open();

    login.js:

    var win = Ti.UI.currentWindow;

var labelLogin = Ti.UI.createLabel({
  color : 'black',
  text : 'Welcome',
  top : 50
});

var btnlogin = Ti.UI.createButton({
  title : "Login",
  textAlign : "center",
  top : 20,
  height : 75,
  width : 150
});
win.add(labelLogin);
win.add(btnlogin);

btnlogin.addEventListener('click', function(e) {
  var menu = Ti.UI.createWindow({
    backgroundColor:'green',
    title:'Menu',
    url : 'menu.js'
  });
  menu.open();
});

    menu.js:

    var win = Ti.UI.currentWindow;

var btnprocurar = Ti.UI.createButton({
  title:"Procurar",
  textAlign:"center",
  top:200,
  height:75,
  width:150
});

win.add(btnprocurar);
win.open();

btnprocurar.addEventListener('click', function(e){
  win.close();
});

    Hope this helps

  • I think the problem is that you open too many windows.

    In your btnLogin event listener you create a window with the url menu.js and open it. In your menu.js you create another window with a button and you open that window as well. So now you have two windows stacked where you only wanted one. When you click the button in your menu.js you'll probably end up in the empty window that was opened before.

    Hope that was understandable..
