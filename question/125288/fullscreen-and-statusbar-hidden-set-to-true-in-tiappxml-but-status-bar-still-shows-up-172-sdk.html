Fullscreen and statusbar-hidden set to true in tiapp.xml, but status bar still shows up (1.7.2 SDK)

Not sure why this isn't working, but setting this:

<statusbar-hidden>true</statusbar-hidden>
<fullscreen>true</fullscreen>

does not do what it should. I have to manually hide the status bar with Ti.UI.iPhone.hideStatusBar(), which results in an inconsistent and janky feel to the app. I want the status bar to be hidden as soon as the loading screen opens. What can I do to acheive this?

FYI: I'm only focusing on this being an iPhone app. Android methods / specifics are not needed.

  • Figure this out myself. I swapped the settings in tiapp.xml after the first build/run, so the changes were not being picked up by subsequent builds. To remedy this, I had to go Project > Clean, then rebuild. This solved the problem 100%, and the app now works as expected.
