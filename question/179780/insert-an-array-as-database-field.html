    • Alloy.js is the initializer file which is use to preconfigure methods and variable before controller is executed.. so you can fetch all information from database which is loaded earlier in alloy.js file and store it into global variable like Alloy.Globals.someVariable = YOUR CONTENT WHICH IS FETCHED FROM DB . and you use this information in any screen before load

      e.g

      window.js

      ....
...
function loadPreloadedData(){

    for(var i=0;i<Alloy.Globals.someVariable.length;i++){
        //LOAD VIES HERE
    }

}

if(Alloy.Globals.someVariable.length > 0){
    loadPreloadedData();
}
