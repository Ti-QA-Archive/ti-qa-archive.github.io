Auto height elements

how can I set the automatic height of the elements (not to propisovat for each element separately), so that they display correctly.

Example:

element1 = Ti.UI.createLabel({text:'Element 1'});
element2 = Ti.UI.createLabel({text:'Element 2'});
element3 = Ti.UI.createLabel({text:'Element 3'});
w.add(element1);
w.add(element2);
w.add(element3);
— asked March 29th 2011 by Zhuk Vitaliy

    Try this:

    w.layout = 'vertical';

element1 = Ti.UI.createLabel({text:'Element 1', height:'auto'});
element2 = Ti.UI.createLabel({text:'Element 2', height:'auto'});
element3 = Ti.UI.createLabel({text:'Element 3', height:'auto'});
w.add(element1);
w.add(element2);
w.add(element3);
