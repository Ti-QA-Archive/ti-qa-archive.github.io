Accessing data in a nested JSON response

Im trying to access data that appears to be an object in an webservice call. The structure of the returned data has me stumped on how to actually pull the values out. Any help would be appreciated. The response information is formatted as such

{} JSON
 |
 |_ feedback{}
 |
 |_communications{}
 |                |
 |                |_communincationID: "123"
 |                |
 |                |_communicationAuthor: "Jim Tester"
 |                |
 |                |_ communicationDate: "01/01/1991"
 |                |
 |                |_communicationTitle: "Walk in the Park"
 |
 |                                            etc..etc
 |_primaryKeys[]
              |_ 0: "123"
              |_ 1: "456"
              |_ 2: "789"
              |_ 3: "012"
              |_ 4: "345"

I am able to take the response data and parse it but im only able to traverse through the primaryKeys array because its an actual array. How would I be able to see the communications{} data??

If my response information is myCommunications and I am only able to traverse through the primary key array.
Ex. myCommunications.primaryKeys.length or myCommunication.primaryKey[0] which would return me value "123"
…. How would I get the communicationAuthor information?
