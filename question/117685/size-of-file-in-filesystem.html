size of file in filesystem?

hello, is there a way to get the size of a file somewhere in the filesystem (sdcard etc.)? thought it must be a method or property of Ti.Filesystem.File, but can't find it. any idea? thx, u.

    From the kitchen sink example filesystem.js:

    var f = Titanium.Filesystem.getFile(Titanium.Filesystem.resourcesDirectory, 'text.txt');
Ti.API.info('size = ' + f.size);

    In general, the kitchen sink is a good place to look for examples, even when the docs don't suggest such a property exists.
