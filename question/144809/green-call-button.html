    Actually IT IS an image in the Apple's apps :)

    Download this project and run it with xcode against the simulator, then do a search in the results for green, you will see a condensed artwork (suitable for background set with backgroundLeftCap )

    https://github.com/0xced/UIKit-Artwork-Extractor
