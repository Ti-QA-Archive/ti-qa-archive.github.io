Ti.API.currentWindow is null

Hi

I'm creating a game with two windows, one main menu window and the game window. From the menu window I open the game window like so:

var btnPlay = Titanium.UI.createButton({id:"btnPlay"});
btnPlay.addEventListener('click', function() {
    var win1 = Titanium.UI.createWindow({
        url:'game.js',
        fullscreen: true,
        navBarHidden: true
    });
    win1.open({animate: true});
});
win.add(btnPlay);

This works fine the first time.

In the game.js I have this to go back to the menu window:

var win = Titanium.UI.currentWindow;
var btnMenu = Titanium.UI.createButton({id:"btnMenu"});
btnMenu.addEventListener('click', function() {
    win.close();
});
win.add(btnMenu);

This also works and I get back to the menu window. But when I press the play button again Titatinum.UI.currentWindow is null (in game.js).
