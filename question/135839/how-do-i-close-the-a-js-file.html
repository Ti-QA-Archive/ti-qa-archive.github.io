    First: you do not close files, but Elements.
    you can only close those which are standalone like window/tabgriup .. all others you can hide().
    Second: your view creation is a bit wrong, instead of new PageView you should new page or new page.PageView it depends on your Module. It works for now because iOS/TiSDK is a bit buggy in this case.

    So to remove the view: win&#x2F;view.remove(pageview);
