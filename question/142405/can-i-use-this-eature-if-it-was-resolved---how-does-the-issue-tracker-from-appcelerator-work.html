    This means that its fixed in the current 3.0.0 CI build. You can grab nightly builds from the CI Server. Please keep in mind that continuous integration builds are NOT officially supported. If you have issues and ask questions regarding CI builds, very few people are going to be able to help you, and will probably tell you to use a supported version.
