Help with scrollable view please

Hi All,
Can you please help with a scrollable view problem.

In a nutshell, I have dictionary application. Each view, in the scrollable view, contains 1 word with the explanation. So Word1 is View1, Word 2 is View2 etc.

The scrollable view works fine but I have many,many words in the dictionary. So as I have that many views, I'd like a way to jump directly to a particular word/view, without scrolling through all the words/views.

So to do that, I have a picker. The picker contains all the words/views, so that you can ddirectly jump to that word. Well, that's the theory anyway ;-)

My question is: what do I need to stick into the eventlistener of the picker to control the scrollable view function, so to jump from e.g. View10 to View2 without scrolling?

I think the issue is that I don't know how to make the one function speak to the other… or at least pass the selected value from the picker to the scrollable view code

Scrollable view code (which works)

// Add scroll view
var scrollView = Titanium.UI.createScrollableView({
    views:[view1,view2,view3,view4],
    showPagingControl:true,
    pagingControlHeight:30,
    maxZoomScale:2.0,
    currentPage:1
});
win.add(scrollView);

var i=1;
var activeView = view1;

Picker event listener code (I think the code goes here)

picker.addEventListener('change', function(e) {
    // the value returned from the picker is 'e.selectedValue[0]' 
    Ti.API.info(e.selectedValue[0]); // this works BTW


    //now what?
    // something like:  activeView = e.selectedValue[0]  ?
});

Thanks in advance
