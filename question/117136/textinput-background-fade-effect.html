  • Uhmmm never saw that thingy , but if you think making it look like everything but not text field is blurred and black:

    You could create a view with black background and 0.4(your choise) opacity and make it 500 zIndex. Then configure the text field to have 501 zIndex , so everything but not the element you want to be shown will be faded and will look blurry.

  • Vitali, I would use the focus event listener:

    MyTextField.addEventListener('focus', function(){

    MyTextField.backgroundColor = '#333333';
})
MyTextField.addEventListener('blur', function(){

    MyTextField.backgroundColor = '#FFFFFF';
})

    within the listener handler, you can also add some special effect
