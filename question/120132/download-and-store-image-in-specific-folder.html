    use setFile in the HTTPClient object:

    var imageDownloader = Titanium.Network.createHTTPClient();
imageDownload.open("GET", "path to image", false);
imageDownload.setFile("/path/to/save/your/image/and/filename");
imageDownload.send();

    • from the api:

      Titanium.Filesystem.applicationDataDirectory - readonly constant where your application data directory is located. this directory should be used to place applications-specific files. on iPhone, this directory is also backed up.

      Titanium.Filesystem.applicationDirectory - readonly constant where your application is located

      Titanium.Filesystem.resourcesDirectory - readonly constant where your application resources are located

      Titanium.Filesystem.tempDirectory - readonly constant where your application can place temporary files

      choose one.
