Android image folders

I have a background image for the main/home screen. In developing for Android's multiple resolutions and densities, I'm trying to take advantage of Titanium's density-specific folders with no luck:

Resources
    /android
        /images
            /res-notlong-port-hdpi
                main.png
            /res-notlong-port-ldpi
                main.png
            /res-notlong-port-mdpi
                main.png

The code for my home window:

var self = Ti.UI.createWindow({
    backgroundImage:'main.png',
    title: 'Home',
    navBarHidden: true
});

From Titanium's documentation, I'd think I should be able to simply use the image's base name (image_name.png) and Titanium would select the proper one. However, I'm getting nothing but a black background (buttons and everything else work). Oh, and using HVGA emulator.
