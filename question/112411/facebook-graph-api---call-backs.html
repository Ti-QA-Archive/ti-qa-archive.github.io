  • you can just listen for an event that if fired, will pass the access_token

    or

    you can set a global variable that holds the access_token

    or

    you can set a property

    Ti.App.addEventListener('GOT_FACEBOOK_ACCESS_TOKEN', function(data){
   Ti.API.debug(data.access_token);

   Ti.App.Properties.setString("ACCESS_TOKEN",data.access_token);
});

    or

    with success_callback

    function success_callback(xhr) {
   var data = JSON.parse(xhr.responseText);
   Ti.API.debug(data[0].access_token);
}
