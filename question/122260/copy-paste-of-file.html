  • This is just keyed into my reply off the top of my head. So it is untested and could have errors, but should give you the general idea:

    if ( Ti.Filesystem.isExternalStoragePresent == true ) {
    var fromFile = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory,'myFile');
    var toFile = Ti.Filesystem.getFile(Ti.Filesystem.externalStorageDirectory,'Downlaods','myFile');
    if ( toFile.exists() == true ) {
        toFile.deleteFile();
    }
    toFile.write(fromFile.read());
}

    • Change the alerts to show fromFile.nativePath and toFile.nativePath, and you should see the complete path including path delimiters. You don't want a slash at the end of download in the getFile statement – when you name multiple arguments in getFile, it auto adds the proper delimiter for you based on the underlying OS. In general, multiple arguments are better than using + or trailing / in getFile() calls.
