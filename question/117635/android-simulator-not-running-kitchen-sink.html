  • Make sure that you have the required Android SDK packages installed, as per the Prerequisite Android SDK / API section of the Titanium Compatibility Matrix.

    Also, ensure that you are choosing the correct OS version to use for the virtual device, by using the SDK menu on the Run Emulator screen of Titanium Developer. For example, Titanium SDK 1.6.X supports up to "APIs 2.3", as per the table in the Android SDK / Target Android Platform section.

    Lastly, always choose the SDKs with the "APIs…" prefix. This is because the enhanced API is required for the Kitchensink application, as it uses Maps.

    If these suggestions don't help, please post a screenshot of your Android packages to ImageShack, and I will try to assist further.

    Cheers
