Ti.Admob - Android doesn't work

Hello.

I have installed correctly Ti.admob for Android developing. This is my code.

var Admob = require('ti.admob');

// then create an adMob view
var adMobView = Admob.createView({
    publisherId:"<<here i put my id>>",
    testing:false, // default is false
    //top: 10, //optional
    //left: 0, // optional
    //right: 0, // optional
    bottom: 0, // optional
    adBackgroundColor:"FF8855", // optional
    backgroundColorTop: "738000", //optional - Gradient background color at top
    borderColor: "#000000", // optional - Border color
    textColor: "#000000", // optional - Text color
    urlColor: "#00FF00", // optional - URL color
    linkColor: "#0000FF" //optional -  Link text color
    //primaryTextColor: "blue", // deprecated -- now maps to textColor
    //secondaryTextColor: "green" // deprecated -- now maps to linkColor

});

win.add(adMobView);

The App stops and this is the message:

java.lang.NoSuchMethodError: com.google.ads.AdRequest.setTesting

Somebody help me?

Thanks.

    I had a thought: you need to have a height/width set on your AdMob ad view. You are only setting the bottom.
    Maybe the AdMob module is returning some other kind of object when you don't specify height and width. It's that other object type that has no setTesting() method.

    When you're setting your view size, you should know that if you want to run a 300x50 ad unit, you have to set the height and width to 300 display pixels, or AdMob will refuse to fill the position.

    So if you're on an hdpi Android device, you need a view that is 450x75 physical pixels.
