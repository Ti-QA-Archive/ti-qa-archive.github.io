tableView Search

Hi,
I have used the default search of tableView(by creating a searchBar and assigning it to search property of tableView) and it works fine when i have the title attribute of the tableView rows. But when i use Custom tableView where i have labels in tableViewRows added to the tableView(i am not using title property since it's custom), the Search functionality is not working. Can anyone help me fix this? Thanks in advance :)

    Here is a sample code:

    var row = Ti.UI.createTableViewRow({
    height: auto,
    searchFilter: data[i].title
});

var searchBar = Ti.UI.createSearchBar({
    showCancel: true,
    hintText: "search for whatever"
});

var tableView = Ti.UI.createTableView({
    data: rowData,
    search: searchBar,
    filterAttribute:"searchFilter"
});
