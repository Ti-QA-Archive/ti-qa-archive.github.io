Run/debug on Android device?

Hi,

I'm trying to debug or just run an app on an Android device attached via USB. I can't see any options that let me do that- my debug perspectives only have the Emulator. Seems like it ought to be simple to do but I can't find any advice anywhere- can anyone point me in the right direction?

  • If in Titanium Studio you right-click in the App Explorer pane or right click on your project in the Project Explorer pane you should see the option "Install on iOS Device" and "Install on Android" device.

    Your Android device should have USB Debugging activated and then you should be set.

    You cannot use the Titanium debugger for debugging on device, but you can debug via logging by running the "ddms" tool which is located in your android sdk in the subfolder "tools".
