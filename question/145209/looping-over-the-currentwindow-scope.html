    • I m looking if a specific variable exists and if it contains a certain value. This variable is not always the same, it depends on certain conditions.

      anyway, for those looking something similar, here's the answer.

      //'win' is my currentWindow

for (var key in win) {
   var obj = win[key];
      alert(key+'='+obj);
}
