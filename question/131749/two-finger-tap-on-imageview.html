Two finger tap on ImageView

Hi, I can't seem to make the 'twofingertap' event work for an ImageView. I use the following code, which I think is the standard. I've searched on the Web to no avail, so it seems I'm missing something obvious.

var win = Ti.UI.createWindow();
var img = Ti.UI.createImageView({
    image: 'myImage.png',
    width: '100%',
    height: '100%',
    top: 0,
    left: 0
});
win.add(img);

img.addEventListener('twofingertap', function(e) {
    alert('Hey! '+JSON.stringify(e));
});    
win.open();

I'm using Ti Studio 1.0.8.201201262211 on OSX 10.6.8, SDK version 1.9.0.v20120121223134 (I can't switch to an earlier SDK because this has a bugfix I definitely need).

Thanks in advance!
