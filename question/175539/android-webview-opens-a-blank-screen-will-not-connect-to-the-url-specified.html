Android webview opens a blank screen, will not connect to the URL specified.

I run a website that I am currently redesigning (due to non use for a while), as part of the re-design I would like to run a application on android for users to open rather than having to navigate through chrome, all I need is for the app to load the website, the website will do the rest and show the mobile version of my site. webView.addEventListener('load', function() { webView.visible = true;

});

    You forgot to add the webview to the window, and you're trying to open the webview, whilst it has no method open. win1 should be opened. Like this:

    //creates Win1 
var win1 = Titanium.UI.createWindow();
//creates webView 
var webView = Titanium.UI.createWebView({
    url:'http://www.ryanhammill.co.uk', 
    visible:false
}); 
win1.add(webView);

webView.addEventListener('load', function() { 
    webView.visible = true;
});

win1.open({fullsreen:true, navBarHidden:true});

    Then it works,

    Also a tip for your html:
    On the devices it takes by default the resolution, if you add a meta tag for viewport and set it to width=device-width it will pick the device width, so it scales properly:

    <meta name="viewport" content="width=device-width">

    Just a tip I use when developing mobile web apps.

    Hope this helps,
    Patrick van Vuuren
