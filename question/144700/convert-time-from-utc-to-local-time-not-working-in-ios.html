Convert time from UTC to Local Time not working in ios

Hi, In my app I need to take the date from the cloud and display it in a label as follows. 

//messageThreads is an array that contains all the threads of the current logged in user. 
var date = messageThreads[index].created_at;
alert(date);//displays 2012-11-14T06:57:36+0000
//Formatting the date to required format
date = FormatDate(date);
alert(date);//displays Nov 14 2012 12:27 **Required format**

and the function body is 

function FormatDate(date)
{    
    var newDate = new Date(date);
    newDate = newDate.toString("MMMM");
    return (newDate.substring(4,21));    
}

it works well in android platform and gives the correct output.
But in ios the function returns a value as "invalid date"

Can anyone help me to get the date time in required format?
