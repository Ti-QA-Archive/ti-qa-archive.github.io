CreateFile failing on Android 2.1?

Hi,

I've got an app that works nicely with iOS, but is giving me two major issues (so far) getting it to run with Android. The first is record audio, which I've found sample code for. The second is creating a file. I use createfile() with no issues on iOS, but I'm getting an error that the function is missing when I run the app in the android emulator.

Does anybody know what the workaround for this is? Saving preferences and user info is a key first step for my app.

  • Following on Tommy. You can 

    var oF = Ti.Filesystem.getFile( Ti.Filesystem.tempDirectory, 'myfile.txt' ) // the args ',' add the '/' for tempDirectory + '/' + fileName
    oF.write('123') // write serves as a create 'myfile.txt' if it doesn't exist and fill it with '123'
