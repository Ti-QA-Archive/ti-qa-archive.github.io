Automated Login / Single Signon for Moodle Site

Does anyone know if its possible to create some sort of Automated Login / Single Sign-on to a Moodle Site (in Titanium). My Titanium app links to a Moodle site via a button but I'd like if possible to store the username and password in my app so that the user does not have to login to the Moodle site when they hit my button. Have looked at the Moodle documentation and am a bit confused so I wondered if anyone has done the same thing? Thanks.

  • Looking at the Moodle docs, it doesn't look like Moodle supports an SSO out of the box, but it seems that you can add via plugin: http://docs.moodle.org/dev/Authentication_plugins. However, if the goal is to simply go from your app to the site without having to login and not to use any Moodle functionality within your app, then it might not even be worth it. You could just ask the user if they'd like to store the credentials winthin your app and then just pass them along if they do.
