    • try this code

      var win = Ti.UI.createWindow();

var txt = Ti.UI.createTextField({
    width : 200,
    height : 40,
    bottom : 400,    
    backgroundColor : 'gray'
});

txt.addEventListener('change',function(e){
    var letters = /^[0-9,]+$/;    

    if (e.value.match(letters)){
        e.source.oldValue = e.value;
    }else{
        e.source.value = e.source.oldValue;

    }
});

win.add(txt);

win.open();

  • Hi Michael,
    keypressed ever gets fired only when focusable property is set to true.

    
keypressed
Fired when a hardware key is pressed in the view.

A keypressed event is generated by pressing a hardware key. On Android, this event can only be fired when the property focusable is set to true.

    try with focusable property and it should work.
