    Of course you do!

    But for now it is a manual procedure.

    For this you must do the following:

    1 - download the last mobile build (1.8.x) from http://builds.appcelerator.com.s3.amazonaws.com/index.html

    2 - save in your ˜/library/Application Support/Titanium/mobilesdk/osx/

    3 - edit ˜/library/Application Support/Titanium/mobilesdk/osx/1.8.0.xxxx/iphone/Classes/NetworkModule.m

    In this file make these changes: https://github.com/rubenfonseca/titanium_mobile/commit/3135267350e667cfb4a8b721270239b05622cabc

    4 - now create your application with this new mobile version.

    5 - run first time.

    6 - copy the file info.plist from YOUR_PROJECT_FOLDER/build/iphone/ to YOUR_PROJECT_FOLDER.

    7 - Edit the info.plist from YOUR_PROJECT_FOLDER including:

    <key>UINewsstandApp</key>
<true/>

    and

    <key>UIBackgroundModes</key>
<array>
<string>newsstand-content</string>
</array>

    change the structure of icons for:

    <key>CFBundleIconFiles</key>
<array>
<string>appicon.png</string>
<string>appicon@2x.png</string>
<string>appicon-72.png</string>
</array>
<key>CFBundleIcons</key>
<dict>
<key>CFBundlePrimaryIcon</key>
<dict>
<key>CFBundleIconFiles</key>
<array>
<string>appicon.png</string>
<string>appicon@2x.png</string>
<string>appicon-72.png</string>
</array>
<key>UIPrerenderedIcon</key>
<false/>
</dict>
<key>UINewsstandIcon</key>
<dict>
<key>CFBundleIconFiles</key>
<array>
<string>icon_newsstand.png</string>
</array>
<key>UINewsstandBindingEdge</key>
<string>UINewsstandBindingEdgeLeft</string>
<key>UINewsstandBindingType</key>
<string>UINewsstandBindingTypeMagazine</string>
</dict>
</dict>

    8 - Create the new icon for your app to the newsstand in the root of your project with the name icon_newsstand.png, being 130x190 px.

    9 - Now in your project add the NOTIFICATION_TYPE_NEWSSTAND EventListner with a function to download new contents in background.

    Voila!
