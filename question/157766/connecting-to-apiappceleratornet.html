  • Ok, SDK version: 2.1.1.GA
    Connection:
    In my app.js:
    var db = Ti.Database.install("mySeizureDb.sqlite", "mydb");
    db.close();

    Then in the file I try to use it in I open it with:
    var db = Ti.Database.open('mydb');

    db.execute('SELECT * FROM tblSeizureInfo');

    I get the results I'm looking for but I get the warning above. Thanks again.
