  • Hello Bojaraj,

    There is no way to do this from your windows environment. You have to have a apple desktop or laptop and you should have installed Xcode on that. Than after you just have to configure and install your titanium and just import your project over there. You will need a little changes in code (That depends on your code what controls and what properties you have used If you have used some thing specific to android than that you have to change over here to run it on iPhone). Other things will be same. Launch your project after importing and it will generated Xcode project for you.

    Cheers.
