Slide windows in/out for a tableview WITHOUT using a tab?

I am trying to setup a simple two-page app where the main window has a tableview, and clicking on each item goes to a detail view. Simple stuff that a tableview-based app should be able to do.

The problem I am having at the moment is that I can't figure out any way to just seamlessly animate the detail view in, with all the structure in place to slide the main window out, and the detail window in, then provide the back button to go back to the main view when you're done. The examples I've seen use Ti.UI.currentTab to do this, or something like that.

But what if you aren't using a tabbed view of any sort? How do you get this functionality, short of hand-coding it yourself?

I've searched this forum until my eyes are red, but I just can't find the simple answer to this. Any help is appreciated.

  • You would have to use the Ti.UI.iPhone.NavigationGroup module, or else make a TabGroup but just hide the tab.

    Or you would have to go into some intense self-animating with views and the like, which I doubt would be worth it.
