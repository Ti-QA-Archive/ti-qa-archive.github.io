Email template when creating users

Hello,

I'm creating a mobile application with Titanium (SDK 2.1.3.GA) which is making use of the Appcelerator Cloud Services.
In my app, users are being created in ACS. I would like to have the created users having their e-mail address confirmed using the ACS email verification. Getting newly created users confirmed by e-mail is working without problem.

Unfortunately, I can't find anywhere in the ACS documents how to specify the e-mail template to use when sending the e-mail verification. After checking the specification of the resend_confirmation method, I figured the configuration option had to be either template or confirmation_template.

I tried both these configuration parameters when creating an user. This is what I've found out:

  • template : doesn't work - default Appcelerator's verification e-mail is send
  • confirmation_template : custom e-mail is send using the specified template but the {{key}} and {{confirmation_token}} in the template aren't replaced with the actual values.

Can someone help me with this?

Thanks!

    some modifications for the previous answer:
    when creating an user, you can pass template and confirmation_template parameter at the same time.

    when creating an user , if you pass template parameter , then the user will receive an email, the email just notify that the user has been created successfully.

    when creating an user, if you also pass confirmation_template parameter , then an verification email will be sent to the user with the template specified by confirmation_template parameter.
    could you give me your confirmation_template body string? i will check it.
