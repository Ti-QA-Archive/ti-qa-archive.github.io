currentWindow

I am using the code from the kitchen sink XML RSS and I get to the point on line 60 where it says current window is not valid.
I have used this file as an included file in my app.js but still telling me currentWindow is a no go. What is the work around for this. I have changed the currentWindow to createWindow variable and that still doesn't help. I know this has to be a quick simple answer. Please be as layman as possible with your answers as I am an idiot.

    app.js doesn't have a 'currentWindow'.

    You usually load windows 'from' app.js (via tabs etc. or direct, as in KitchenSink where they have variable names win1, win2 .. )

    Assuming your app JS has a createWindow in it somewhere, with a variable name, replace Titanium.UI.currentWindow with the variable name.
