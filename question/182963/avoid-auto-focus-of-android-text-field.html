Avoid Auto focus of android text field

Hi All,

I have text fields in my app. When the value in textfield1 changes, the value in the textfield2 should get cleared. This is working fine simply by setting textfield2.value = '';

But the problem is the focus event of the textfield2 gets fired when value is cleared. how to avoid this.

Actually there are more than 10 input textfields in the same page. and whenever value in textfield1 is changed everything else should be reset. But focus event on textfields is flickering the screen and behaves weird.

Any solution to avoid focus event being fired when value is cleared.

Using Ti SDK version 3.5.1
Checking on nexus 7 device with Lollipop OS

Thanks in advance.

Geervani

deregisterFocusEvent();
