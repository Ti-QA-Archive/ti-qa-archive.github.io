    • Hi Levi - thanks for the update.

      The reason for the message:

      Failure [INSTALL_FAILED_MISSING_SHARED_LIBRARY]

      …is that the KitchenSink is being launched using the standard Android SDK rather than the enhanced one (that includes support for maps, that the KS uses).

      To resolve this in Titanium Studio, select Run Configurations from the run button, select the KitchenSink configuration, and ensure that the Android API selected is Google APIs Android 2.1 rather than Android 2.1.

      Also, you should install the packages for Android 2.2. Make sure you have at least the packages shown in the screenshot in Summary of Installed Android Packages.

      Let me know if this helps
