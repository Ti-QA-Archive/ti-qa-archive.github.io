Prevent Users from spamming the UI

Hi,

What would people here suggest be the best way to prevent a user from spamming the UI?
In my example I have a navigation group, and I click a button which does some processing (creates a new listview and shows it in a new screen). On the old iPhone 3G this takes about a second, if the user presses the button multiple times then the navigation group goes a bit crazy and eventually shows the window multiple times.

All I can think to do is to have some sort of lock, i wondered if anyone else had any ideas?

  • When the user clicks the button set a flag to true and when you operation finishes set the flag back to false.
    This way on the click event you check for this flag, if is false do your stuff, else do nothing.
    Makes sense?
