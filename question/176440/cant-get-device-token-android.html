  • Hi ILIBERI,
    Well i had similar problems as well . Let me share my ordeal !!!.
    My sdk version was 3.3.0 but when i used the latest cloud push module , I used to get the device token but the app used to crash when i used to go to the background with the error = "Unable to instantiate receiver com.appcelerator.cloud.push.Push BroadcastReceiver" .
    So i had to go to the tiapp.xml and change the version of the imported module to 3.2.1.
    Something like this <module platform="android" version="3.2.1">ti.cloudpush</module>
    And then things finally started working . Got the device token as well as no crash on going to background and trying to listen for push notifications.

    Coming to ur situation . I think it would be best to upgrade ur titanium sdk to 3.3.0 as i see ur still using 3.2.3 and in the tiapp.xml use ti.cloudpush version 3.2.1. I am pretty sure that should do the trick. Let us all know !!!
