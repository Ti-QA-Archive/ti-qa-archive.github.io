  • U should try to use 'scroll' event for scroll view:

    scrollview.addEventListener('scroll', function(evt) {
    if([checking if ended on 0 location]) {
        //code for refreshing
    }
});

    or with 'scrollend' / 'scrollEnd':

    scrollview.addEventListener('scrollend', function(evt) {
    if([checking if ended on 0 location]) {
        //code for refreshing
    }
});
