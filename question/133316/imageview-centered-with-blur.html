ImageView centered with blur

This is strange but using ImageView with center property, or percentage, blures the image as if it was stretched or something. Using the same image with left/top and it looks sharp and as it should. Are there any issues with ImageView or with center/percentage properties?

I’m using sdk 1.8.2, ios5

Thanks.

  • Found some more information. It seems that if the ‘left’ property is a whole number, say 75, the image is rendered in a correct way. If it is a number with a fraction, such as 75.5 then it gets blurred.

    In my case {x=’55%’} returns a fracture and therefore looks blurred.

  • That's simple maths, your image width/height must be an odd, so when Ti calculates the position, it returns the half of the width, that's not round anymore.

    The best workaround is to edit your image, and add one pixel to it, or wrap it into a view that's 1px larger.
