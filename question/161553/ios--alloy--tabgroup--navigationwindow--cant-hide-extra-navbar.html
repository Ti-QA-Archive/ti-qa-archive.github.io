  • Accepted Answer

    Windows of Tabs automatically become NavigationWindows. You can open new windows like:

    $.myTab.open(newWindow);
// or
$.myTabGroup.activeTab.open(newWindow);

    If you use NavigationWindows in TabGroups you have to set navBarHidden to true on your TabGroup.

      Is there a way to globally access myTabGroup.activeTab? Am I right that $.tabGroupId would only be available in index.js where it's created?

      — commented January 16th 2014 by Malcolm Christie

    • Yes. In alloy.js add a Global referecne:

      Alloy.Globals.tabgroup = undefined;

      Then, in your index.js assign the tabGroup:

      Alloy.Globals.tabgroup = $.tabGroup;

      Now you can access the tabgroup via Alloy.Globals.tabgroup from every controller :)
