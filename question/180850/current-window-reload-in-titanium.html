Current WIndow reload in titanium

Hi All,

I am developing an android app,in that i am deleting some elements in click event,after deleting i want to reload current window with remaining elements how to achieve this please help me

  • Reloading the entire window would be a bit of an ugly experience for the user, if it wasn't in the normal flow of transitioning between 2 contexts or something.

    Instead, consider loading the elements on to a view, and removing and re-adding that view, or a portion of it as needed. This is a good bit cleaner and less excessive than removing and reloading a window.

  • Although I agree with Stephen's answer above, sometimes you might need to reload the entire window. In that case, write a function to remove all the elements from that page using (e.g.: window.remove(element) and add only what is required(e.g.: window.add(element) and call that function after your delete function.
