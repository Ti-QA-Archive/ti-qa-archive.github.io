Android app resume / resumed events

I am trying to get my app events (specifically resume/d) to work in Android, but have had little luck. I have had this working in iOS for some time now, just the Android side seems to handle app events differently.

This code exists in my index.js file:

  • Ti.App.addEventListener('pause', pauseApp);
  • Ti.App.addEventListener('close', closeApp);
  • Ti.App.addEventListener('resume', initApp);
  • Ti.App.addEventListener('resumed', initApp);

I then tried:

  • Ti.Android.currentActivity.addEventListener('pause', pauseApp);
  • Ti.Android.currentActivity.addEventListener('close', closeApp);
  • Ti.Android.currentActivity.addEventListener('resume', initApp);
  • Ti.Android.currentActivity.addEventListener('resumed', initApp);

It still appears to not execute the app events at all on resume/d.

Any help or direction will be greatly appreciated, thanks in advance.

  • OSX 10.9
  • Ti Studio 3.2.0.201312191547
  • Ti SDK 3.2
  • Android 4.1 - 4.3
  • Genymotion GS3/4 4.1 - 4.3
  • HTC One gpe 4.4.2
