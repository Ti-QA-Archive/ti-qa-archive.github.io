registerForPushNotifications undefined error

I am calling registerForPushNotifications and the error callback
is calling unfortunately the error is undefined.

Titanium.Network.registerForPushNotifications({
types: [
Titanium.Network.NOTIFICATION_TYPE_ALERT
],
success:function(e)
{
alert("ok");
},
error:function(e)
{
alert("Error: "+"e.message");
}

    According to the documentation The error function callback has two properties on the object; error and type. You are trying to read e.message. Change this to e.error
