Missing controls after reopen window in navigation group

I have created a two windows (home and add) and embedded them in a navigation group. I am opening the add window with help of a button on the home window.

As soon as I use the back-button on the add-window I return to the home window again. Now I am clicking on the button (on the home window) again to open the add window.

Now the add window is entirely empty without any controls. How do I avoid this behavior? Thank you.

–

UPDATE with code (only relevant parts):

app.js

Ti.include('includes/layout.homeWindow.js');
Ti.include('includes/layout.addWindow.js');

// create Navigation group to hold the application windows
var navGroup = Ti.UI.iPhone.createNavigationGroup({
    window:homeWindow
});

Ti.include('includes/events.homeWindow.js');
Ti.include('includes/events.addWindow.js');

// create main window and add navgroup to show it
var main = Ti.UI.createWindow();
main.add(navGroup);
main.open();

events.homeWindow.js:

homeWindowAddButton.addEventListener('click', function(e) {
    navGroup.open(addWindow);
});

addWindow has a couple of controls and events.addWindow.js is (with regards to a back-event) empty. I am using built-in back functionality of thr navGroup.
