"Update iOS info" - iOS8, Ti SDK 3.4.0GA

Hello,

I updated to the newest Xcode 6.0.1 and Titanium SDK 3.4.0GA.
When I start the Titanium Studio I get the following error:

An internal error occurred during: "Update iOS info".
org.json.simple.JSONObject cannot be cast to org.json.simple.JSONArray

Also in Titanium Studio I can set the SDK to 3.4.0GA in my projects but can not run it.
The Simulator iPhone 6 and iPhone 6 plus are still not visible.

With the workaround in terminal it works fine

ti build -p ios -C iPhone 6

But how can I start my projects through the Titanium Studio?
My Titanium Studio is in Build 3.2.3.201404181442

Thanks for any help.

Cheers,
Fab

  • I solved the problem by myself.
    My Titanium Studio was not updated within the automatically updates…

    Please read the following site to update the Titanium Studio from Titanium Studio
    http://download.appcelerator.com/appcelerator/studio/standalone/update/stable/

    Attention! Deselect all checkboxes in the Update-Menu, the newest Titanium Studio installation pops up, select them -> update -> all works fine now :-)

    Cheers!
