Ti SDK v1.8.2
iPhone SDK 4.3

Hi,

I have a tab with a window and on the top of that window I created a box to hold my scrollview. The box is rendered but the external js isn't seem to be loaded.

this is how I call the external js

var imageScrollView = Ti.UI.createView({
    top: 30,
    height: 50,
    width: 230,
    borderWidth: 1,
    url: 'scrollImages.js'
})

I can see the border of the box and it is rendered ok but no external js. I created an alert on the external js but this isn't fired.

How can I display the external js inside this window? thanks for your help.
