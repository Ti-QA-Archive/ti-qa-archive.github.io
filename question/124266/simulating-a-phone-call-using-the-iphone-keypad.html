Simulating a Phone Call using the iPhone Keypad?

I'm looking at the possibility of using titanium for a project I'm doing.

In short, I'm making a demo that simulates the process of an automated call center menu. "Press 1 for help, press 2 for accounts" etc. The demo would ideally work on a phone, hence the idea to make an App that looks and feels just like you were making a call and answering options.

So I'd need to be able to press different keypad buttons and have an audio file played back.

Is this something achievable through titanium? Or am I barking up the wrong tree?

Any advice is helpful!

  • There are a bunch of keyboard types supported by Ti.

    That said, here's an example that should work (it's copied from the kitchen sink)

    var win = Titanium.UI.currentWindow;

var tf1 = Titanium.UI.createTextField({
    color:'#336699',
    height:35,
    top:10,
    left:10,
    width:250,
    keyboardType:Titanium.UI.KEYBOARD_NUMBER_PAD,
    returnKeyType:Titanium.UI.RETURNKEY_DEFAULT,
    borderStyle:Titanium.UI.INPUT_BORDERSTYLE_ROUNDED

});

win.add(tf1);

tf1.addEventListener('click', function(e) {
    Ti.API.info(e.source.value);
    // play the audio or do what ever else you need to do...
});
