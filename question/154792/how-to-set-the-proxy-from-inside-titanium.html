How to set the proxy from inside Titanium

I am using " Titanium Studio, build: 3.1.1.201306112235 " and i am behind a proxy and i do not know how to set the proxy.
Please help me to solve the problem

  • editing the following file: "%Titanium_Studio_Path%\jre\lib\net.properties".
    You simply have to edit the following lines:

    http.proxyHost=yourProxy.com
http.proxyPort=8080
http.nonProxyHosts=localhost|127.0.0.1

https.proxyHost=yourProxy.com
https.proxyPort=8080

  • In the initial splash screen when you launch Titanium Studio, there is a "Proxy Setup…" button where you could select to set up the proxy information before logging in.

    Hope this helps.
