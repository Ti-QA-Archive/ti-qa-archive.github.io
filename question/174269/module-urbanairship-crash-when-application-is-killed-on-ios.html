Module Urbanairship crash when application is killed on iOS

Hi !

I use the module Urbanairship in my application (on iOS) and everything is working, the module works perfectly.

When my application is killed, i do some actions :

Ti.App.addEventListener('close', function (e) {
    // some actions ...
    });

But, when i kill my application, the module Urbanairship crash and I get the following error :

<Error>: *** Terminating app due to uncaught exception 'InstanceNotExists', reason: 'Attempted to access instance before initializaion. Please call takeOff: first.'

And, because of this crash, the actions in event "close" are not executed.

If i remove the Urbanairship call in my application, i do not have this crash and my actions are executed.

My Titanium SDK version is 2.1.4.GA (this is an old application and i have to deal with it).
I used this to implement the module and this is working.

Just the case I kill the application.

Any suggestions ?

TY
