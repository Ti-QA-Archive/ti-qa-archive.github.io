    • If you are on an iOS device (or simulator) that is running iOS 6 or higher then it will ONLY display the Apple maps.

      The module I linked to above is Apple maps ONLY, whilst that page explains how to add maps to both Android devices and to iOS devices the modules are specific to each platform and WILL not run on the other platforms;

      • Modules.Map for Android will only work on Android
      • Modules.Map for iOS will only work on iOS

      If you device is running iOS 5 or lower then that does run Google Maps - however the Modules.Map module will NOT work.

      So if you have used Modules.Map on an iOS device then it can ONLY be Apple maps.
