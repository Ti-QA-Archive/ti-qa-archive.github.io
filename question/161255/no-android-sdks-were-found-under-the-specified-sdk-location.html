  • It's already a know issue in the titanium studio TISTUD-6007

    To work around it, open the terminal and execute this command :

    "ti config android.sdkPath /path/to/your/android/sdk"

    then configure the same path in Studio preferences.

    P.S. you may need root access.

    — answered March 29th 2014 by Abdallah Zidan
    permalink
    2 Comments

    • Thanks to your answer I could notice that in Windows environment, Titanium stores the configuration for android.sdkPath with 2 slashes… I mean: "C:\android-sdk-path" (if your sdks are in the root). So, I wrote in the cli:
      ti config android.sdkPath "C:\android-sdk-path" (note the one slash)
      And it worked.
      Hope this helps!

    sudo apt-get install nodejs-legacy
