    Hi

    Take at look at this previous example - it was given before v2 maps but the route points methods should be the same;

    • Android generate route on mapview

    iOS may be more forgiving and accept a simple lat/lon pair with a comma but the array is best served as an object array with latitude and longitude as parameters.

    Whilst you have attempted this yourself in your code - you have actually ended up with a string array - as all you are doing is concatenating a string. You could try this;

    if(isAndroid){
    androidwaypoints.push({
        latitude: data[i].poi_lat,
        longitude: data[i].poi_lon
    });
}

    This would be the correct syntax. If not refer to my previous answer for more examples on the way to build the array.
