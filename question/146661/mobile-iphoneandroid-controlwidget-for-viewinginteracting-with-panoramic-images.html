Mobile (iPhone/Android) control/widget for viewing/interacting with Panoramic images

Hi all,

I would like to incorporate a "Google Street View"-like aspect to a game I am planning to make using the Appcelerator SDK. For example, to present a (pre-prepared) panoramic photo that will look "correct" when being panned in a 360 degree motion (left/right/up/down).

Does any such functionality exist within the Appcelerator SDK? (primarily I'm targeting the iOS platform, but would like to cater for Android later)

I know I could use a simple Image control, but that would not present the correct perspective when panning. I think the image needs to be "projected" onto a spherical surface (or using the old "Sky Box" principle?).

Either way, would be grateful if anyone has any suggestions in this area.

Many thanks! :)

    This functionality does not exist out of the box in Titanium or either mobile platforms. You're going to need to find an sdk or other library that you can use to build a native module in order to add this functionality. See the Module Development Guides for info on creating modules.
