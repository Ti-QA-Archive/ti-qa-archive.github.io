Empty initial value for picker

Is there a way to set the initial value of the Picker to nothing? I'm creating one for gender, and I don't want to have to declare an empty string at data[0]. However, when the picker loads, it always automatically chooses the first element.

  • This is a real shortcoming. I have resorted to using a non-editable text box and a '…' button alongside that fires up an OptionDialog. You could give the button a down arrow graphic if required.
    That way you get full control and can populate the textbox and a selectedIndex variable in the click event of the dialog.
