Stuck on basic database operation

I'm working on a mobile application using titanium (Titanium Studio, build: 3.4.1.201410281727).

I've created a simple sqlite database,

CREATE TABLE "messages" ("message_id" INTEGER PRIMARY KEY NOT NULL ,"message_type" VARCHAR(20),"message" VARCHAR(255));

INSERT INTO "messages" VALUES(1,'START','Test message');

It's in my Resources folder under a subfolder called db/

The following code runs with no error:

var f = Ti.Filesystem.getFile(
Ti.Filesystem.applicationSupportDirectory+'/database', 'fitquitdbNM.sql');
//If it's there, delete it and reinstall the DB
if(f.exists() == true){
f.deleteFile();
Ti.Database.install('/db/fitquitdb', 'fitquitdbNM');
}else{
//Otherwise, install it for the first time
Ti.Database.install('/db/fitquitdb', 'fitquitdbNM');
}

db = Ti.Database.open('fitquitdbNM');

However when I use the iPhone Simulator I get the following error: "file is encrypted or is not a database." when running a simple query:

var tablesResultSet = db.execute("SELECT message FROM messages;");

Any help would be appreciated, cheers.
