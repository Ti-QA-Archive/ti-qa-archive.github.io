How to get window.title if resumed from background?

IOS4+

I try to get the title of the current window if I resume my application from the background. But Titanium.UI.currentWindow is empty. Any ideas how to get it?

var service = Ti.App.iOS.registerBackgroundService({
    url:'bg.js' //empty file right now..
});
Ti.App.addEventListener('resumed',function(e){
    Ti.API.info("app has resumed from the background");
    var backwin = Titanium.UI.currentWindow;
    Ti.API.info(backwin);
});

  • In apps that require this functionality, I write some of the information to a Titanium.App.Properties when a new window is opened.

    We have one app that can have multiple windows in a tab's stack.
    Tabs > List > Details > Additional Details

    We save the current tab number to a property.
    List window has a value of 1
    Details has a value of 2
    Additional Details has a value of 3
    Set that to a property.

    On app startup, either from resume or fresh start, read the property value.
    It gets passed down through each of the window creation functions until the appropriate window level has been reached .
