tableView : get the previous row of the selected one

Hi,
In a tableView I want to modify the previous row of the selected one.
If I select the third row I want to modify the backgroundColor of the second row.
So how can I get the previous row in ?

tableView.addEventListener('click',function(e){
    Ti.API.info(tableView);
    Ti.API.info(e.index); //index of the selected row
    Ti.API.info(e.index - 1); //index of the previous row
});
