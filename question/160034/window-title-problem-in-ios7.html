    Hi

    The problem is almost certainly due to the way that the label positions its width inside the titleControl space.

    Normally you would apply a width to the control placed inside the titleControl to make sure it know how much space to take up.

    Try this;

    win.titleControl = Ti.UI.createLable({
    color: "white",
    text: "title",
    width: 200
});

    See what happens.

    • Firstly; try to stop using auto this should be swapped to Ti.UI.SIZE (fits to contents) or Ti.UI.FILL (fits to parent). These tell Titanium and more importantly YOU what you intend.

      Using auto is the same as not setting anything at all as that is the default - actually Ti.UI.SIZE should be considered the default.

      The flickering is a different thing altogether. This depends on everything else you have going on in the app title area or underneath. If you are allowing content to go underneath the navigation bar then you will likely not easily get rid of it.

      But I would suggest either trying backgroundColor of transparent on the label or if the navigation colour is linear (not a gradient or showing the content below through it) then set a fixed color to the the background.
