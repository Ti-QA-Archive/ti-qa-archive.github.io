How do I allow a user to pinch-to-zoom a square image without extra space at the top-bottom?

I'm building a photo detail screen similar to Facebook's where a user can tap on an image and view it on a black background. The user needs to be able to pinch to zoom in on the image. I've tried using a ScrollView which works to a certain extent; the problem is that the image is square, so when the image is not zoomed in there are black areas of the screen above and below it (I'm in portrait mode). When I zoom the ScrollView, not only does the image zoom but the area around it as well, which enables the user to scroll to those black portions of the screen that are now so large it's possible to lose where the image is. Is there a way to allow a user to pinch to zoom an image so that as the image gets larger the black portions on the top and bottom go away so the user can't scroll beyond the image?

    • myImageView.addEventListener('pinch', function(e) {
    var t = Ti.UI.create2DMatrix().scale(e.scale);
    myImageView.transform = t; 
});
