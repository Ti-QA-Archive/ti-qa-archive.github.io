  • The Titanium Framework does not provide you with the functionality to read mails out-of-the-box. The only way I can imagine this possible is by implementing yourself the Mail protocol in JavaScript using Sockets

    You could also develop a module that reads the emails. (that would probably be easier since the code to read email in Objective-C or Java already exists.)

  • To receive email, an app must talk directly to the SMTP server. If you are not familiar with SMTP, then I suggest you first start by learning how SMTP works. Do a web search on something like SMTP commands to see how exchanges with a mail server work.

    Once you understand that, you need to implement those same commands using sockets, as Christian already pointed out.
