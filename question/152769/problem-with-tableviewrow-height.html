  • Accepted Answer

    You can use the following. Set minimum row height for having a minimum height when less text.

    //tableview
tableView = Ti.UI.createTableView({
        style : Titanium.UI.iPhone.TableViewStyle.PLAIN,
        backgroundColor : "#ffffff",
        minRowHeight : 50
    });

//create row with following height attribute.
tvow = Ti.UI.createTableViewRow({
            height : Ti.UI.SIZE
    });
var lbl = Ti.UI.createLabel({
            text : 'text',
            height : Ti.UI.SIZE,
            font : {
                fontFamily : "Arial",
                fontWeight : "bold"
            },
            left : 5
    });
    tvow.add(lbl);
tableView.setData([tvrow]);

    This is working for me. Let me know how it is on your side.
