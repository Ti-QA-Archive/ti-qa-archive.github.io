Create a new class based on View

Hi,
I would like to create a new class based on view.
this view will have 2 buttons, with label1 and label2.
So I need something like :
myNewView = Titanium.UI.createNewView({
height:"50dp",
left:"10dp",
label1:'text1',
label2:'text2'
});

and in the create process of this class, I create the 2 buttons and set their label.

Any idea of how to do such thing in Ti.Mobile ?

Regards

    I would create a function that generated that object. Something like;

    function customViewFactory(title1, title2){
 var containerView = Ti.UI.createView();
 var btn1 = Ti.UI.createButton({title:title1});
 var btn2 = Ti.UI.createButton({title:title2});
 containerView.add(btn1);
 containerView.add(btn2);
 return containerView;
}
