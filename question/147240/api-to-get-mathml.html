  • There is a solution called WIRIS that does precisely what you request. You can convert from a formula in text format “1 plus 2” to its MathML as part of its accessibility features.

    There is also an API to do that.You can find a live demo of WIRIS editor. The buttons of the demo that you might be interested are “Get accessible” and “Set accessible” (this last is the good one).

    The API’s are described at the WIRIS editor reference pages. Follow the link Services class reference.
