TableView Get Text From Row With Specific ID

Hi,

I have a TableView with a onClick method.

What I would like to obtain is the text in the row with the id my left number - this is set parsing a JSON file and placing the data into the Table.

<TableViewRow>
        <Label id="leftNumber" left="1px" />
</TableViewRow>

So I can get the row number using

function openDetail(e) {
    alert('row index = ' + JSON.stringify(e.index));
}

But how do i get the actual text?

Thanks

    Hi

    You could use the 'getChildren()' method to obtain the child of the tableViewRow?

    You could then do something like this:

    function openDetail(e) {
    var child = $.row.getChildren();
    Ti.API.info(child[0].text);
}

    This specific example will only give you back the text from the first child, which for your use should work ok. But you would have to adjust it if you are going to create a more complex TableViewRow.
