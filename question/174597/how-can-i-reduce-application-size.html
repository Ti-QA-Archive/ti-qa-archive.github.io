    Hi

    There are many answers in the Q&A on this subject, here is one of mine;

    • apk file is too large size 9.3MB

    Once you understand the payload point from the answer I gave, you now need to start working out how much your code and assets add to the size. Count all the images you have in every dependant folder - including splash screens. Add the total size required for all other files, then add the amount id code you have.

    One if the more obvious things to do is make sure you only add as many images as you actually need, optimise the images, compress if possible and use one of the many optimisation tools you can google for - these can shave 5-60+% from the original size. Include only what required. If you are not supporting tablets - remove this splash screens, no provision for ldpi devices - get rid of this as well, no landscape - delete.
