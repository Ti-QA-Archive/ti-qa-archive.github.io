Creating Android native module fail creating module template

Hi,
I'm trying to create a native Android module using this guide: http://wiki.appcelerator.org/display/guides/Module+Developer+Guide+for+Android

  • It is a tiny little error in androidsdk.py located in the Titanium SDK's directory

    • go to the Titanium sdk directory (on Mac OS &#x2F;Library&#x2F;Application Support&#x2F;Titanium&#x2F;mobilesdk&#x2F;osx&#x2F; )
    • select the version of the sdk which is referenced by the alias 'titanium' in your ~&#x2F;.profile . On my computer, the latest is 1.7.5 in the first post it's 1.7.2
    • open androidsdk.py in your favourite editor
    • in the lines 94 and 96 check that the string passed as second argument of os.path.join matches the format of the directories of the google-apis you installed with Android SDK.

    In line 96

    self.google_apis_dir = self.find_dir(self.api_level, os.path.join('add-ons', 'addon_google_apis_google_inc_'))

    I changed &#39;addon_google_apis_google_inc_&#39; into &#39;addon-google_apis-google_inc_-&#39;

    watch the hyphens (-) and underscores (_)

    Hope this helps
