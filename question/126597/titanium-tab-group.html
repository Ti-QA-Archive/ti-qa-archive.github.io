Titanium tab group

Hi guys! I am building an app where I need 2 tab groups in the same window. Is it somehow possible? Because in the examples they only work with one and I don't know how to add the second one. Thanks

— asked October 8th 2011 by Szilard Mandici

  • Windows don't contain tab groups, but rather a tab group contains windows, so the short answer to you question is no. However, you can have a window for any given tab that contains a tabbed bar or a toolbar with buttons on it that can control aspects of that window within the tab group.
