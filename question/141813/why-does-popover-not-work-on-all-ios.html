    Mark,

    First off I recommend explaining your issues a little more clearly and then also reading the documentation on platform support. The documents clearly state that the popover is only for iPad and thats because in the native code thats all thats supported. Titanium does not try to "fix" the underlying platform but rather just give access to it from JS.

    If you need something for iPhone you can try this module I made - https://github.com/mattapperson/TiPopover. It's only been tested on iPhone but with some minor tweaks it should work for android too.
