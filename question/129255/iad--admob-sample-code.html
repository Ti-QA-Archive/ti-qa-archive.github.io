    Try something like this:

    
x_time=5000; //(5 seconds)

wait_timeout = setTimeout( function(){
    your_create_the_admob_ad_function();
}, x_time );


the_iad.addEventListener('load', function(){
    clearTimeout(wait_timeout);
});

    So you start a timeout that waits for 5 seconds for the iad to appear. If the load event does not trigger, you launch the admob banner.
    If the iad loads before the 5 secs you kill the timeout and u show the add.

    you will have to manage somehow the case when the iad loads later than your timeout, but is not that hard.
