  • Accepted Answer

    On iOS, you can use an AttributedString to set it as the Label.attributedString property

    On Android, you can use the labels html property to style your label with HTML / CSS.

    Another very nifty solution (which also works fine in cross-platform usage) is the use of this module:
    ti.styledlabel which also allows you to use html.

  • Hi

    If your questions are VERY simple then these would be the answers;

    • Bigger
    var label1 = Ti.UI.createLabel({
  color: '#ffffff',
  font: {
    fontSize: 40 // change this
  },
  text: 'A simple label'
});
    • Bold
    var label1 = Ti.UI.createLabel({
  color: '#ffffff',
  font: {
    fontSize: 20,
    fontWeight: 'bold' // add this
  },
  text: 'A simple label'
});
    • Red
    var label1 = Ti.UI.createLabel({
  color: '#ff0000', // change this
  font: {
    fontSize: 20
  },
  text: 'A simple label'
});
