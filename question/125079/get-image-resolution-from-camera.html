Get image resolution from camera

Hi All,

I am trying to resize an image. Landscape or portrait depending on what comes from the camera. I'm trying something along the lines of this. But I cannot get any values from 'event.media.height' or image.height. This is on Android / mobile sdk 1.7.2 and 1.8.?

Any help would be much appreciated.

Titanium.Media.showCamera({
        success:function(event)
        {

          var image = event.media;

          if (event.media.width > event.media.height){

           resize = Titanium.UI.createImageView ({
                width:600,
                height:450,
                image:image }); 

                } else {
                          resize = Titanium.UI.createImageView ({
                          width:450,
                          height:600,
                          image:image });
                           }



           FinalImage =resize.toImage().media;
           filename = "Android.jpg";

        },
        error:function(error)
        {
            if (error.code == Titanium.Media.NO_CAMERA)
            {
                alert('Device does not have video recording capabilities');
            }
            else
            {
                alert('Unexpected error: ' + error.code);
            }
        }
    });

    You can get the image height and width using the following code in your showCamera Success event:

    var eventInfo = Titanium.JSON.stringify(event);
var parsedData = Titanium.JSON.parse(eventInfo);

alert ('Photo width = '+parsedData.cropRect.width+' height ="+ parsedData.cropRect.height);

    Hope that helps,

    Ray
