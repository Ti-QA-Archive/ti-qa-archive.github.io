Platform: iphone and ipad
Titanium SDK: 3.0.0.GA

I want to let the user know, that information has been saved succesfully.

But there is no action he needs to take. So no need to click OK or cancel.

How do I implement this for iOS

Also it should be only displayed for a brief moment, say 1 second.

    Just create a view that has the desired message in it, display it for 1 second, then close it. If it must look like the iOS confirmation dialog you'll just have to style it that way. I'm sure you can find a hi res image of the dialog box that you can use as the background of the view.
