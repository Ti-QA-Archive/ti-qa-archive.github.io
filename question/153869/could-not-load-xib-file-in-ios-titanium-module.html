Could not load xib file in iOS Titanium module

Hi,

I am trying to load a xib file named "HALNumberPad" in a iOS module from inside a Titanium app. I have followed the steps mentioned at https://wiki.appcelerator.org/display/guides/iOS+Module+Development+Guide to create the iOS module and access it from Titanium.

Basically the IOS module has a TiUIView which will load a xib file on init. However, when the view is created in titanium after module load, I get an error saying there is no nib file present in the app.

Here is the error message I get:

Could not load NIB in bundle: 'NSBundle </Users/biswarup.dasgupta/Library/Application Support/iPhone Simulator/6.0/Applications/8C6BD140-D2C8-4580-906B-8E7024C48A5E/paint.app> (loaded)' with name 'HALNumberPad'";

I have put the xib file in my Titanium app under Resources/iPhone folder but still it didnt pick up. But if I change the extension from .xib to .nib, the app picks up the file but obviously complains that the data format is not correct.

So, my questions are:
1> Can we load a xib file inside an iOS module when the corresponding Titanium view is created?
2> If not, how to convert a xib file to a nib file.

Thanks in advance!
