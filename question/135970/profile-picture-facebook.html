  • See my reply below.

    The reason you're seeing 'weird characters' is because that's the 'text' 'code' of the ACTUAL IMAGE.

    You do not need to make that requestWithGraph at all to show the Facebook picture.

    If you know the Facebook UID (which you get via Ti.Facebook.uid) you can simply show a an image using its URL, which is simply :

    https:&#x2F;&#x2F;graph.facebook.com&#x2F;&#39; + Ti.Facebook.uid + &#39;&#x2F;picture

    ie:

    
var imagem = Ti.UI.createImageView({
    image : 'https://graph.facebook.com/' + Ti.Facebook.uid + '/picture'
});

win1.add(imagem);

  • No. It helps if you read the Facebook documentation.

    To get the JSON from Facebook which tells you the username, do a request to :

    &#39;https:&#x2F;&#x2F;graph.facebook.com&#x2F;&#39;+Ti.Facebook.uid

    Which will return a simple bit a JSON which you can parse to get the info you need, like the username.

    Try it with your own UID. (Or even your Facebook url name) :

    eg : https:&#x2F;&#x2F;graph.facebook.com&#x2F;zuck simply try that url in your browser and you'll see the user info for Mark Zuckerberg.
