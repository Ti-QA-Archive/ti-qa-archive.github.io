    I think you should use absolute display pixel (dp) sizes. Calculate the desired width using the display width. Then set height accordingly.

    So make sure your project has this in the tiapp.xml:

    <property name="ti.ui.defaultunit" type="string">dip</property>

    Then calculate the row width like this:

       var row_width = parseInt (Ti.Platform.displayCaps.platformWidth / 3);

    You can now use this row_width for the width and the height of the rows.
