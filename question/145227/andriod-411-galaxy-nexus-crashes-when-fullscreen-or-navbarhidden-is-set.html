Andriod 4.1.1 (Galaxy Nexus) crashes when fullscreen or navBarHidden is set

Updating an iPhone app to run on android. App crashes on my Galaxy Nexus. It appears to be a heavy weight window issue.

This code will crash the app on the phone:

var self = Ti.UI.createWindow({
navBarHidden:true,
});

Removing navBarHidden allows app to run fine.

Same thing happens if I add fullscreen:true (or false)

Is 4.1.1 not supported?

Note running on andriod 2.3 there isn't an issue.

Thanks,
-Rick
