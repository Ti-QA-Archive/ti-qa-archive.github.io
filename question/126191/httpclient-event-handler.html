HTTPClient event handler?

Hi Guys.
Yet another question for the community (I hope to return the favour one day).

Any'hoo…

I've got a HTTPClient sending data over to our remote web service, and I think its working okay. However the response obviously isn't immediate… is there a event handler that I could use that would fire when the response comes back in?

Many thanks for your help.

    The event you are looking for is the onload function. From there, you will have access to the responseText property.

    Here is a short example:

    var xhr = Titanium.Network.createHTTPClient();

xhr.onload = function()
{
    alert(this.responseText);
};

xhr.onerror = function(e)
{
    alert(e.error);
};

// open the client
xhr.open('GET','http://www.myserver.com');

// send the data
xhr.send();
