'Like' Button

Hello. Is it possible for me to create a 'like button' in which when the button is clicked, it turns into 'unlike button'? And is it possible if the button is clicked by a lot of users, it collects or shows the number of user that clicked the button (increment)? and is it also possible when a user clicked the 'unlike button', it will decrement by one.

  • You need a web server to store the number of likes (a simple php script will work like a charm).

    Then, create the Like button. When the user cliks it, save with Application Data that the user liked, and do a web request to add a like.

    Do the oposite to unlike…

    If you need code sample, just ask!
