Is there code assist for Canvas context elements?

Sorry if this question has been asked before; I've tried searching but couldn't find anything definitive.

I know there's code assist for the <CANVAS> HTML tag, but is there any way to get code assist for the canvas 2D element in JavaScript (the one with methods like fillRect()). I'm using the latest Titanium Studio (build date 17 June 2011).

I try doing something like this:

var cvas = document.getElementById("my_canvas");
var ctx = cvas.getContext("2d");

If I then type cvas. and press CTRL-SPACE, I get no method suggestions. I also don't get the 'getElementById' as a suggested method if I type cvas. + CTRL-SPACE. The code I'm writing works though.

Am I missing something in the project settings or is the assist not there to that level?

    Matthew,

    There is currently very minimal support for Canvas completion. While we are talking about adding more properties/methods, there is no ETA for such additions. However, it is all based on community request, so your best bet is to file a feature request and be very specific about any particular methods or snippets of code that you would like to see added.
