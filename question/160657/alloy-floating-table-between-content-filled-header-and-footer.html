Alloy: Floating table between content filled header and footer

I'm completely new here and to Appcelerator and Alloy. This seems logical to want to do, but perhaps it can not be done? I have a header and footer view which are both sized to their content. I would like to have a tableview sandwiched in between utilizing what ever vertical space remains. I put all 3 within a view with layout=vertical. And the header and footer height=Ti.UI.SIZE, and the footer bottom=0. Yet when setting height of the tableview to Ti.UI.FILL it extends to the bottom of the screen with the footer overlayed.

I can see how to do this using % on the 3 vertical views, but then I loose the ability to make the header and footer sized by content.

I'm using Titanium Studio build 3.1.3.201309132423, running on Windows 7, deploying only to Android and web for now, but eventually will be adding iOS.

  • Instead of having a separate header and footer view you might want to consider the headerView and footerView components of the tableView instead. Also, if your layout is going to be fairly simple, you probably want to consider the newer listView component instead

    • FooterView seemed like it should do the job, and I had explored it, but it also is not behaving as anticipated. Table extends to the bottom of the screen, and the footer is not visible.

      index.xml

      <Alloy>
    <Window title="Eljay Oil Co. Inc. - Invoices" class="container">
        <View layout="vertical">
            <View id="mainHeader">
                <Label text="mainHeader"></Label>
            </View>
            <View id="mainSelector">
                <TableView id="selectorTable">
                    <TableViewRow title="row 1"/>
                    <TableViewRow title="row 2"/>
                    <TableViewRow title="row 3"/>
                    <TableViewRow title="row 4"/>
                    <TableViewRow title="row 5"/>
                    <TableViewSection>
                        <FooterView id="mainFooter">
                            <View left="10" top="10" layout="horizontal">
                                <Button title="Undelivered">
                                <Button title="Delivered">
                                <Button title="All">
                            </View>
                            <View right="10" top="10" layout="horizontal">
                                <Button title="Upload Invoices">
                                <Button title="Download Invoices">
                            </View>
                        </FooterView>    
                    </TableViewSection>                    
                </TableView>
            </View>
        </View>
    </Window>
</Alloy>

      index.tss

      ".container": {
    backgroundColor:"white"
},

"Label": {
    width: Ti.UI.SIZE,
    height: Ti.UI.SIZE,
    color: "#000"
}

"#mainHeader": {
    width: "100%",
    height: Ti.UI.SIZE,
    borderWidth: 2,
    borderColor: "red"

}

"#mainSelector": {
    width: "100%",
    backgroundColor: "yellow",
    borderWidth: 2,
    borderColor: "green"

}

"#selectorTable": {
    width: "100%",
    separatorColor: "grey"
}

"#mainFooter": {
    width: "100%",
    height: Ti.UI.SIZE,
    bottom: 0,
    borderWidth: 2,
    borderColor: "blue"
}
