  • This is a show stopper for me too. However, I did a "temp" solution that comes close to achieve this.

        var flag = false;
    mapview.addEventListener('click', function(e){

        if(flag){
            flag = false;
            mapview.deselectAnnotation(e.annotation);
            //DO SOMETHING HERE
        }
        else{
            flag = true;
            mapview.selectAnnotation(e.annotation);
        }
    });

    Of course is not the same than clicking the popup and fire an event but at least is something temporary to work with. Hope this get fixed soon.
