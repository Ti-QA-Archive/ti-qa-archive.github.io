iOS sdk home not found on Lion

Hello,

I have installed the latest version of Titanium Studio and the latest version of xCode on my Mac with an up-to-date verion of Lion installed.
When I try to create a new project, I see the message 'iOS SDK Home' is unavailable.

Some notes:

  • From xCode I can launch the iOS simulator.
  • I already tried to reinstall xCode.
  • The xCode install doesn't create a /Developer directory, as in the earlier versions. It is now located in /Applications/xcode.app/Contents/Developer.

Someone an idea of what the problem could be?

Thanks!
