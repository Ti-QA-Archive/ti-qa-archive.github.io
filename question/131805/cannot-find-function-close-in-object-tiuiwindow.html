cannot find function close in object [Ti.UI.Window]

Experiencing an issue when attempting to close an array of Windows in Android whereby the function close cannot be found in object [Ti.UI.Window]

At the start of app.js I initialize the global Windows array as follows

globals.windows = Ti.App.Properties.getList('windows', []);

Each time I open a Window, it is added to a global array like so

globals.windows.push(win);
Ti.App.Properties.setList('windows', globals.windows);

Next block of code is how I am iterating the global Windows array and closing each Window.

for(var i = 0, l = globals.windows.length; i < l; i++) {
  globals.windows[i].close();
};

Any ideas why the error "cannot find function close in object [Ti.UI.Window]" is happening?

Additional Information

  • Application type: mobile
  • Titanium SDK: 1.8.1
  • Platform & version: Android V8
  • Device: Android emulator
  • Host Operating System: OSX
  • Titanium Studio: 1.0.8.201201262211
