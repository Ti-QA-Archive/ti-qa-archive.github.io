Email template not found for Cloud.Users.requestResetPassword

I have created an email template on ACS called forgot_password.

Any ideas what I am doing wrong?

When I run this code:

   this.config.Cloud.Users.requestResetPassword({
      email : usersEmail,
      template : "forgot_password"
    }, function(e) {
      if (e.success) {
        alert('You will receive an email soon with your password details.');
      } else {
        Ti.API.error(e);
        _callback && _callback({
          success : false,
          model : null,
          error : e

    });
