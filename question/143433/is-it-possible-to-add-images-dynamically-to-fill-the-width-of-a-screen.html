Is it possible to add images dynamically to fill the width of a screen?

Regardless if your using an iPhone or iPad, is there a way to add images to a view based on the width of the screen? For example, if you open up the Apple photos app and view all the photos on your device at once you see 4 images across in portrait mode. If you rotate to Landscape you see 6 images across. Is something dynamic like this possible with appcelerator?

— asked October 12th 2012 by Nando Cammarota

  • Yes. When you build your layout knowing the width of the screen, you can simply set those thumbnails to the size that will fit the number you want across the screen. Then you listen for orientation changes and re-layout the images for the wider screen size.
