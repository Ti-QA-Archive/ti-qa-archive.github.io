URL Parameter Not Showing In WebView

Hi,

I have a web view that i'm successfully able to load using:

<View class="rightSection">
            <WebView id="webview" url="radar-basic.html" />
</View>

However when i pass some variables i.e.

<WebView id="webview" url="radar-basic.html?somevalue=abc" />

On looking at window.location The parameters are not shown.

How do i get them to appear?

    Hi

    According to your code and its use of the URL property - your web page is located locally. However your additional use of the ? indicates you expect your parameters to affect the page, is this correct?

    If so - html pages on mobile devices do not process the parameters you pass in the URL as there is no server side scripting language - as your mobile device is not on a server.

    So either work without parameters or host the page on an actual server.
