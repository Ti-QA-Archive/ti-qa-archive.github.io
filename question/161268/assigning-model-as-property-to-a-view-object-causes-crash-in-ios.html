Assigning model as property to a view object causes crash in iOS

I'm trying to assign a model as a property to a view object I'm creating as a quick and easy way of referencing the model when the view is clicked. I can assign simple strings and objects as properties on my view object, but when I try to assign a model, iOS crashes. Works fine in android.

Thoughts? Am I missing something painfully obvious in the docs?

Alloy.Collections.instance('photo').each(function(photo) {
    var image = Ti.UI.createImageView({
        image: photo.get("filePath")
    });
    image._props = {photo: photo};
    image.addEventListener('click', function(event) {
        //...do something with the view-attached model here...
    });
    $.getView('media').add(image);
});
