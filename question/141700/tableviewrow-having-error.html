  • Your filenames are all over the place – you have a file called "locationthree.js" with class called "location1"? This is not good for maintainability (or for having others help you with your code).

    At any rate, your error is this:

    var location1 = (Mods.LOCATIONONE);   // missing require call
var newView = new location1('args');
newView.open();

    should be

    var location1 = require (Mods.LOCATIONONE);
var newView = new location1('args');
newView.open();

    (this appears in multiple places). Without require, you're setting location1 to the string that you put into Mods.LOCATIONONE. You can't call a string like a function, obviously.

    Note that I have seen your other posts, so I have a general idea what you're doing with these Mods, but keep in mind that many others would have no idea what you're talking about.
