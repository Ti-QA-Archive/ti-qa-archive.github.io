Resource Already Exists

I received a project from our vendor and am trying to open it in Titanium. I did so and then closed it. Now when I try to import this project again I am getting an error that "Resource /Application name already exists". I have deleted any reference to "Application name" folders on the hard drive but no joy. This is iPad app development for reference.

Thanks.

JR

  • John, if you want to clean things up then check the .metadata directory in your workspace directory as there are project references there.

    I have seen a suggestion to just delete the .metadata directory. You will need to re-import all your projects if you do that.
