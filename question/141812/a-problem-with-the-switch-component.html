A problem with the switch component

Hello everyone,

I wrote a simple code for testing the switch component:

var window = Titanium.UI.createWindow({

    title:"Switches",
    exitOnClose:true,
    backgroundColor:"#FFF"

});


var switch1 = Titanium.UI.createSwitch({

    top:100,
    value:true,

});


var text1 = Titanium.UI.createLabel({

    text:"click on the switch",
    height:80,
    width:"auto",
    top:40

});

var PLATFORM = Titanium.Platform.osname;

if (PLATFORM === "android"){

var switch2 = Titanium.UI.createSwitch({

    top:150,
    value:false,

});    


var text2 = Titanium.UI.createLabel({

    text:"click on the switch2",
    height:80,
    width:"auto",
    top:160

});    

}


if (PLATFORM === "android"){

    switch2.addEventListener("change",function(e){

        alert("HELLO WORLD");
    });
}


if (PLATFORM === "android"){

    window.add(switch2);
    window.add(text2);
}


window.add(switch1);
window.add(text1);
window.open();

The problem is when I click the second switch, the alert didn't work !!! I didn't know why? :(

Anyone have an idea for this?

Thanks ;)

  • Accepted Answer

    This is happening because of your text2 label. You've made it so large that it overlaps the switch above it. So when you think you are clicking on the switch, you're actually clicking on the label. You can test this by adding a border and border width to that label, or by just clicking the very top of the switch. The label doesn't cover the whole switch, just the bottom 2/3s of it.

    Use layout properties and proper sizing to avoid this issue in the future.
