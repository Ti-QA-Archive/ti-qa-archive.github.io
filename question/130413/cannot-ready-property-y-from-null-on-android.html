Cannot ready property "y" from null on Android

Hi all,

I'm trying to implement drag and drop on the Android and used TiDrop (https://github.com/mcongrove/TiDrop) but the first error I get when I touch the screen (using the demo) is:

&quot;TypeError: Cannot read property &quot;y&quot; from null (file:&#x2F;&#x2F;&#x2F;android_asset&#x2F;Resources&#x2F;examples&#x2F;TiDrop.js#46).

This happened with another example (https://gist.github.com/1073811) and I can't seem to tell why? I'm using APIs 2.2 with SDK 1.7.6.v20111228170134 on a OSX developing for Android.

Thanks!

  • Version 1.7.x broke support for e.globalPoint.y on Android, which is where you error is. You will need to implement the method convertPointToView to achieve the same thing.
