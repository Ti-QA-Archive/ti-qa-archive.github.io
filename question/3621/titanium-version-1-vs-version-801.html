Soon after I learn a few tricks in 8.0.1, here comes version 1 with a totally new approach…

I am in a dilemma now as to how to proceed..

Should I abandon 8.0.1 and start picking up 1.0?

The API Docs are catered to both versions?

How do I get started on 1.0?

Anyone having the same dilemma as me and care to share a bit of your experience?

  • I personally suggest you to check out the video tutorials and the Kitchensink project as a start. Using the Kitchensink as your first hand reference while the API doc hasn't been completed. A lot of APIs are currently missing from the official API reference doc.

    1.0 uses a very different approach on structuring the UI and functionality. So it's definitely worth-while to erase your old knowledge and start embracing the new way.

    Good luck.
