  • Hello J W,

    The active window (var win) is associated with the current executing JavaScript context. Window objects created with a url value run in a separate JavaScript context and thread. Then you may reference them with Ti.UI.currentWindow.

    On this case near the bottom of box6.event listener use: contactswin.open(); instead of Titanium.UI.currentWindow.open(contactswin,{animated:true}); as you still have a single context application.
