Mp3 AudioRecorder / or Facebook post with Mp4

Mobile
Titanium 3.1.1
iOS 6.0, 6.1, 7.0
Mac OS X

Hello - I am having a problem with audio recording and /or posting to facebook, depending on how you look at it. What I would like to do is to make a recording on device using audiorecorder, which I can do successfully. However I would like to also post this to facebook along with a couple of other parameters. I can get a successful post to facebook complete with text and picture, but I cannot get an audio file to post. I read on the facebook feed dialog info page that they will except mp3 audio files. Currently my audio is being recorded in mp4 on both android and ios. (android using a separate module). So I figured the problem was my audio file format, right? So I tried changing audiorecorder.format from Ti.Media.AUDIO_FILEFORMAT_MP4 to Ti.Media.AUDIO_FILEFORMAT_MP3. Now I can't get a recording at all. My recorder doesn't even work properly. I have tried literally every single compression format available in the docs to see if that was the problem, but so far none of them made any change. That is the only change I have made and if I change it back to mp4 it works perfect. Help?! Is there a way to record a file as mp3 in titanium? Or better yet, is there a way to post to facebook using mp4 file format?

Thanks!

  • Hello Jenna,

    If you want to save recorded audio file as .mp4 extension, you need to set "compression" as "Titanium.Media.AUDIO_FORMAT_AAC". According to this link, it's for audio format MPEG4 AAC encoding.

    recording.compression = Titanium.Media.AUDIO_FORMAT_AAC;
recording.format = Titanium.Media.AUDIO_FILEFORMAT_MP4;

    I have tested this with KitchenSink sound recording example. It's working. Hope this helps. Thanks.
