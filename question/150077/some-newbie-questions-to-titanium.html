Some Newbie Questions to Titanium

So, I'm planning to develop my first own app for IOS. I am now considering which platform or concept to use.
As I'm pretty good in Javascript and HTML and have no idea of Objective C - I tend to start using Phonegap or Titanium.
When looking closer the concept of Titanium looks somehow smarter as it is compiled to "real" Native App instead and not only running as local website and in the webview widget.

However I have some questions where I need to get help:

First: Using external JS libraries with Tianium:
I want to use Javascript Libraries. One is used to draw charts and graphs. There are several options out there, e.g. Highchart.js. The other is a library to collect transaction data for German bank accounts. There is a protocol called HBCI / FINTS which is mainly used from all German / Austrian banks and there exists a JS library that enables the communication to the banks.
Is it possible to use these libraries? How will Titanium integrate them when compiling. Didn't really see an example that makes use of external libraries.

Second: Storing SQL, respectively database files as not readable objects
The data in the app should be stored on the device. For the beginning I am not planning an online connection. However, I would appreciate that the files are encrypted, so that nobody - not even the user is able to view them in an editor as soon as he get file access to it.
Does Titanium provide features to realize this? (I couldn't find hints in the documentation guides.)

Third: SQL or NOSQL
As NOSQL databases provide some very nice features, I am also considering of using a NOSQL database like Couchbase Lite. I found a Git Hub repository for TouchDB (aka Couchbase Lite) using Alloy.
https://github.com/pegli/ti_touchdb/wiki/Alloy-and-TouchDB-Tutorial. But it's not clear to me how to use that code. Do I have to first compile it in XCode?

So these are the most important questions to me. Thanks for anybody giving me some hints to make the correct decision.

Best regards, Bernd

  • Hello,

    First: Using external JS libraries with Tianium - Yes, you can use them. Just one thing: remember that Titanium doesn't have a DOM or similar. You will have to adjust some of them.
    Second: Storing SQL - Go to the marketplace you will find alternatives to store your db encrypted.
    Third: SQL or NOSQL - As you mention, there are alternatives nowdays. The link you provided looks quite interesting, you only need to learn how to add modules to your titanium app.
