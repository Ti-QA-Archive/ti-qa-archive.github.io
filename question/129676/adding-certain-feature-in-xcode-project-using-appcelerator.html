Adding certain feature in xcode project using appcelerator

I am having a xcode project and i need to add some feature in it, so is it possible to extend that xcode project or convert that xcode project into appcelerator project? And after the required feature i build it again to get my xcode project back.

  • You can get xcode project from your titanium project as it generates native source code for iOS. Even you can run that in xcode and publish from xcode. But you can not generate titanium project from your native xcode as Titanium is to build native apps using java script not to build java script using native.
