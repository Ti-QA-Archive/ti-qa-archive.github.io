  • So I was able to get this to work - for all those out there who have the same problem.

    1. Check that you have python installed - if not

    http:&#x2F;&#x2F;www.python.org&#x2F;getit&#x2F;

    1. Add python to your path so you can use it in the cmd - I went for Method 3: using a small .bat file on this page

    https:&#x2F;&#x2F;code.google.com&#x2F;p&#x2F;tryton&#x2F;wiki&#x2F;AddingPythonToWindowsPath

    1. Start the fastdev server with the FULL path command to the Titanium.py script
      In my case:

    T:\TestApp C:\Users\Username\AppData\Roaming\Titanium\mobilesdk\win32\ 3.1.3.GA\titanium.py fastdev start

    Then it finally works!!
