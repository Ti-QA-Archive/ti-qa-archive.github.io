Verifying iOS push cert

Hello Community,

I have been trying for two days to get the cloud console to accept my .p12 cert. I continually get this error message : Fail to verify development certificate file: XXX.p12. Please make sure the password is correct. I replaced the name with XXX. I have followed the instructions here: http://docs.appcelerator.com/cloud/latest/#!/guide/ios, as well as several other tutorials. I have gotten an app id and generated the CSR, got the cert, and exported the .p12 several times. Each time removing the old cert from the app id at the Apple dev center.

If anyone could help me out or point a noob in the right direction I would appreciate it.

Thanks in advance.
