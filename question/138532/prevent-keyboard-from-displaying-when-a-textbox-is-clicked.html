    • My requirement is to display a Picker, instead of the keyboard, when this particular textbox is clicked.

      PS: I am doing the following tutorial on how to build a picker, however it currently displays the keyboard on click, which is what I am trying to prevent:

      http://cssgallery.info/making-a-combo-box-in-titanium-appcelerator-code-and-video/

      The following change is what I have now made to the tutorial, which worked nicely:

      var slide_in =  Titanium.UI.createAnimation({bottom:0});
    var slide_out =  Titanium.UI.createAnimation({bottom:-251});

    my_combo.addEventListener('focus', function() {
        my_combo.blur();
        picker_view.animate(slide_in);
    });
