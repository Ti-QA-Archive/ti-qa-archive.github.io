Solution for font size in titanium

Hi
I have come across a problem for font size in titanium.All other things i give in the percentage form and it works perfect on all screens,but i am unable to find a solution for label font size in titanium.Here is my code example:

var l2 = Titanium.UI.createLabel({ 
text:'Appcelerator',
top:'2%',
left:'5%',
 height:'auto',
 width:'auto', 
shadowColor:'#aaa', 
shadowOffset:{x:5,y:5},
 color:'#900',
 font:{fontSize:48}, 
textAlign:'center'
 });

Could you give me a solution how can i adjust the label to support all screen sizes

    Hi, you can give font in some percent of platform height like

    var tmp = (Titanium.Platform.displayCaps.platformHeight*10)/100;

    font:{fontSize: tmp},

  • HI,
    The best way is to set your font size to '48dp' instead of 48. If it looks good on a medium-density device, it will look the same relative size on devices of all densities.

    font:{fontSize: '48dp'},
