  • Well, let me start by saying the android emulator is a crap shoot no matter what API its trying to run. At least a lame horse has value as glue or dog food, the emulator ranks well behind a lame horse.

    Second, lets refer to android levels as API and Titanium versions as SDK so that no one gets confused.

    Third, in all Android development, DDMS (now the monitor binary) is your friend as it will let you know whats going on behind the scenes with a device or emulator.
