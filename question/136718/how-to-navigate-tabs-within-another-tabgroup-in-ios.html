How to navigate tabs within another tabgroup in iOS?

I have a tabbed iPad application. On one of my tabs, I have a button bar and a view. I need to display a different view depending on which button is active in the button bar.

I know how to get the event index of the bar, and I've done this with a web view by swapping the URL property, but what's the best way to manage this with views?

SDK 2.01, iOS 5.1
