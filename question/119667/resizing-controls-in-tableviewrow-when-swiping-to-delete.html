resizing controls in tableviewrow when swiping to delete

I have a progressBar inside a tableViewRow and it's 240px, left 70px, so it basically holds the entire iphone screen width.

when swiping to delete, the "Delete" button hides a part of the progress bar. how can I make the progress bar auto-resize to it's new available width?

    ok, so I've found the answer myself. For anyone else who wants to do it, just position the progress bar (or any other component) using left AND right:

    var progressBar = Titanium.UI.createProgressBar({
    left:10,
    right:10
});

    when the "Delete" button appears, it actually changes the row width, so the "right" constraint kicks in, and shorten the progress bar :)
