I can't run my project in the simulator anymore

I've been successfully running my iPad app in the Titanium simulator for some time now. Last night I shut down my Mac. This morning when I started everything up again, there is no "Run/Debug" icon, and I can't find ANY way at all to simply launch my project in the iOS simulator. My project is there, code is all there, everything is good.

Side note: I configured a keyboard shortcut for Run (a week ago), and that shortcut no longer works. It's as though running my app has become disabled for some reason. Please help.

  • Fixed it! The Run button disappeared because somehow the "Launch" Command Group got deselected and was thus unavailable. I certainly didn't do this, but somehow it happened on its own, maybe as a side effect of something else I did. I fixed it by right-clicking on the toolbar and choosing "Customize Perspective". There was a "Launch" section that was deselected.
