Database Migration from Classic to Alloy

My app was done in Classic. Now I redo my app in Alloy. This time I also make some changes to the database.

What can I do to migrate previous users to the new database structure? Since my app wasn't done in Alloy, Alloy's Migration method seems not working in this case….

What should I do?

    I achieved this by writing my own migration module, making sure that this part only runs once, using the following steps written in psuedo code:

    1. Create the new database using Alloy models;
2. If the old version SQLite file does not exist, exit;
3. Read the old tables data and store in the new models;
4. Delete the old version database;

    That covered it. The key is only doing this once, and not doing it if the old DB doesn't exist, which either means it is a new install or it's already run once.

    Hope this helps
