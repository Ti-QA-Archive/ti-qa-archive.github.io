  • Hi

    If I read this question correctly it boils down to wanting a scrollView or tableView occupying the entire display except for a fixed area at the bottom that includes a textField and a button. The scrollVoew or tableView should be allowed to scroll but the textField and button remain locked at the bottom. I also assume you need this to be capable of displaying on any aspect ratio.

    • XML
    <Window>
    <ScrollView id="scroll">
        <!-- content goes here -->    
    </ScrollView>
    <View id="ledge">
        <TextField id="txt">Hello World</TextField>
        <Button id="btn">Press Me</Button>
    </View>
</Window>
    • TSS
    "#scroll": {
    bottom: 50,
    height: Ti.UI.FILL,
    width: Ti.UI.FILL
},
"#ledge": {
    bottom: 0,
    height: 50,
    width: Ti.UI.FILL
},
"#txt": {
    left: 0,
    right: 100,
    height: 50,
    width: Ti.UI.FILL
},
"#btn": {
    right: 10,
    height: 50,
    width: 80
}
