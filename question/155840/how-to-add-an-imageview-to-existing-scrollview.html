How to add an ImageView to existing ScrollView

I have a button that opens the camera for the user to take a picture. After taking the picture I want to open a different view/page "attachPhoto.xml" which will show the photo and a text field to enter a description. How can I do this with Alloy? Currently, the imageView doesn't appear. Any help is appreciated. I've tried the following in the success method of showCamera:

        success : function(event) {
            var attachPhotoWindow = Alloy.createController('attachPhoto').getView();

            // called when media returned from the camera
            if (event.mediaType == Ti.Media.MEDIA_TYPE_PHOTO) {
                var imageView = Ti.UI.createImageView({
                    width : "200dp",
                    height : "100dp",
                    top : "5dp",
                    image : event.media
                });

                attachPhotoWindow.add(imageView);
                attachPhotoWindow.open();
            } 
        },

And attachPhoto.xml:

<Alloy>
    <Window id="AttachPhotoWindow" class="container" modal="true">
        <ScrollView id="AttachPhotoScrollView" height="100%" width="100%">

            <!-- How can I put the imageView here -->

            <TextField id="PhotoDescription" hintText="Enter Description"></TextField>
        </ScrollView>
    </Window>
</Alloy>

    Hi

    I would normally approach this by adding the imageView mark-up directly in the attachPhoto.xml where you are indicating and assign an id to it. Set the image used to be a placeholder image - so the user know this is where the image will go or at least that something is progressing. Then in your success callback instead of adding your new image - you simply access the imageView you already creating and now assign the image property with the event.media response.

    This will make the app look and respond well.

    • Excellent they are visible - so now we just have to fix what I said earlier because I gave you the wrong stuff - sorry!

      You need to have a controller for the view, so where are you may have the attachPhoto.xml here; app/views/attachPhoto.xml you also need the following;

      app/controllers/attachPhoto.js

      var args = arguments[0] || {};
$.imageView1.image = args.image || '';

      Then change what I told you before;

      var attachPhotoWindow = Alloy.createController('attachPhoto', {
    imageView1: event.media
}).getView();

      To this instead;

      var attachPhotoWindow = Alloy.createController('attachPhoto', {
    image: event.media
}).getView();

      This "should" pass the custom arguments containing a property image that the controller code reads and uses to set inside the the view.
