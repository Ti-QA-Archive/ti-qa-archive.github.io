    There is Native Class, Known As NSUserDefaults.
    all the properties set in tiapp.xml are set in NSUserDefaults, so you can get the properties in your module using this kind of implementation.

    
NSUserDefaults *defaultsObject = [[NSUserDefaults standardUserDefaults] retain];

// will return string value for key
[defaultsObject stringForKey:key];

    i hope that helps you, and give you some idea to proceed.
