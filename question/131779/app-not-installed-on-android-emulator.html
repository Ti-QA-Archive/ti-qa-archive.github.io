  • Oke, i now got it working. This is what i did.

    1. Make sure the following android version (Google Android API 2.1 / 2.2) is installed.

    2. Configure it in Titanium (https://wiki.appcelerator.org/display/guides/2.+Getting+Started+with+Titanium+Studio) to set a default android location. Make sure it is the same as it is described as on the page.

    3. Create a new project like it is shown here (https://wiki.appcelerator.org/display/guides/3.+Creating+your+First+Mobile+and+Desktop+Applications). Make sure everything is the same (the appID).

    4. Run the app under android. When you see the lock screen of Android wait for 1 minute. The unlock the screen and you see the app.

    I recommend to read this tutorial (https://wiki.appcelerator.org/display/guides/Titanium+3-Step+Setup) , thats how i got it working.
