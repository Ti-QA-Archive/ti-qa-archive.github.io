  • try this terminal command

    xcode-select -print-path

    returns /Applications/Xcode.app/Contents/Developer and not /Developer. If it is instead returning /Developer, you must switch the Xcode path with the following command:

    sudo xcode-select -switch /Applications/Xcode.app/Contents/Developer
