Titanium Studio deploy with resources

Hello,

When I deploy my iOS app in Titanium Studio things don't work as I expect. In Titanium Studio I click the deploy button, I select the mobileprovision file and deploy the app. After that I start XCode and open Organizer to share the archive and make an ipa file. I get a file of like 4 mb. Once I start that app on my iPhone I don't see any images. I have the idea the resources aren't added to the deployed app. Am I doing anything wrong?

Thanks in advance.

My kind regards,
Rick.

    I found out by myself already. The pictures were in the 'iphone' directory in the Resources directory. Apparently that folder won't be added to the .app

    My solution: I created a folder called 'images' in the Resources directory. Added all the images there and it works.
