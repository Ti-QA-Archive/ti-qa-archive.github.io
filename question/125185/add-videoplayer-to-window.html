Add VideoPlayer To Window

Hello. I've been through documentation and examples literally for a few days and nothing seems to work for me. I'm just trying to add a simple video to a window and I keep getting this error: "Wrapped Java.lang.IllegalArgumentException: Expected org.appcelerator.titanium.proxy.TiViewProxy type for argument child In add, but got [Ti.Media.VideoPlayer]".

My code is really simple here it is:

var videoWindow = Titanium.UI.createWindow({
title:'Tab 1',
backgroundColor:'#fff'
});
var movie = Titanium.Media.createVideoPlayer({
url: 'movie.mp4',
backgroundColor: '#000',
movieControlMode: Titanium.Media.VIDEO_CONTROL_DEFAULT
});

videoWindow.add(movie);
//movie.addEventListener('complete', function (e) {
 //alert("movie is finished");
    //movie.close();  

//});


movie.fullscreen = true;    
movie.play();

All I want to do is open a project and play a video. What am I missing? Thanks in advance.

  • For Android, you don't add the videoPlayer to the window as you do in iOS. (If you look at the KiSink examples, there is an IF statement - IF iOS, add to Window, otherwise not. See 'movie_local.js' or 'movie_remote.js' in KiSink or see this pastie with another example.
