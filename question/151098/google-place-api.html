  • Google places requires an api key and key for google maps also.

    https://developers.google.com/places/documentation/#Introduction

    
1. Visit the APIs Console at https://code.google.com/apis/console and log in with your Google Account.
2. A default project called API Project is created for you when you first log in to the APIs Console. You can use the project, or create a new one by clicking the API Project button at the top of the window and selecting Create. Maps API for Business customers must use the API project created for them as part of their Places for Business purchase.
3. Click the Services link from the left-hand menu.
4. Click the Status switch next to the Places API entry. The switch slides to On.
5. Click API access from the left navigation. Your key is listed in the Simple API Access section.
