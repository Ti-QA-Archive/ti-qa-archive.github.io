  • If I understand the question correctly, are you asking if ellipsis can be added to the label text when it does not have enough room to display using the font requested?

    If so, the iphone should do that automagically, but you also have a second option which I like even better. If you add the minimumFontSize property, iOS will automatically reduce the font size to fit the label width unless that requires a font size smaller than minimumFontSize. Then it only reduce to that point and adds the ellipsis.

    All of these presumes you are not using 'auto' for the width.

    If I am not understanding the question right, please elaborate.
