change row font in a tableview

Hi, in my iPad app i have a split view with a tableview in the master window, in this tableview i have to
change the font of the second and of the third row, what can i do?

    You can archive that by creating custom row, then adding label with your desired font.

    — answered April 24th 2012 by Minh Nguyen
    permalink
    4 Comments

    • some thing like below:

      var row1 = Ti.UI.createTableViewRow({title: 'Row 1'});

var row2 = Ti.UI.createTableViewRow();
row2.add( Ti.UI.createLabel({text: 'Row 2', font: {fontSize:25, fontWeight:"bold"}}) );

var row3 = Ti.UI.createTableViewRow(); 
row3.add( Ti.UI.createLabel({text: 'Row 3', font: {fontSize:27, fontWeight:"bold"}}) )

    • Thank you, i need to change font color too, how can i do?

    • Use label.color property

      var row2 = Ti.UI.createTableViewRow();
row2.add( Ti.UI.createLabel({text: 'Row 2', font: {fontSize:25, fontWeight:"bold"}, color: 'red'}) );

var row3 = Ti.UI.createTableViewRow();
row3.add( Ti.UI.createLabel({text: 'Row 3', font: {fontSize:27, fontWeight:"bold"}, color: 'green'}) );
