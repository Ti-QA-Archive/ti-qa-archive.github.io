  • Hi

    Can you try this tweak please on your table;

    var tableView = Ti.UI.createTableView({
    height:Ti.UI.FILL,
    // this extra property
    softKeyboardOnFocus: Ti.UI.Android.SOFT_INPUT_ADJUST_PAN, 
    width:Ti.UI.FILL
});
    • softKeyboardOnFocus documentation

    This should compensate for the tableViewRow containing a textField which Android has never really been happy about.
