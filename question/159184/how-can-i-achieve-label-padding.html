How can I achieve label padding?

I have looked at several previously asked questions, and the consensus seems to be wrapping the label with a standard view, and then adding top, left, right and bottom properties to the label view. I have done this successfully with two of the four (left & bottom, for example), but when I combine left and right it causes the label to fill the entire parent view.

I have also tried adding a borderWidth, but this renders inclusively without increasing the size of my label view, thus obstructing the text. What am I missing?

<Alloy>
    <View id="message">
        <View id="labelContainer">
            <Label id="messageContent" />
        </View>
    </View>
</Alloy>

"#message": {
    width: 220,
    height: Ti.UI.SIZE
},
"#labelContainer": {
    borderRadius: 4,
    height: Ti.UI.SIZE,
    width: Ti.UI.SIZE,
    top: 11
},
"#messageContent": {
    left: 4,
    bottom: 4,
    font: {fontSize: 12}
}
