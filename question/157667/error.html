    Hi Kusuma,

    I have also faced this issue and got the solution as follows:

    => "The issue is due to the builder.py file that is being used by Titanium Studio to build the Android app and the code is specific to Windows. We have an updated version of the builder.py file that you can download here:
    https://download.newrelic.com/titanium/Titanium_Studio_3.1.3_Windows_Fix.zip

    All you need to do is replace the old builder.py file with the new one and you should be able to successfully build now. That file is located at:
    C:\Users\<username>\AppData\Roaming\Titanium\mobilesdk\win32\<version>\android\"

    Follow the steps in README.txt file from the zip. It has solved my issue. All the best..
