  • Short answer:

    You'll need to upgrade to the latest Xcode, or at least one supported by that version of Titanium.

    Log into Apple's developer site and visit https://developer.apple.com/downloads/index.action?name=Xcode

    Long answer:

    The latest as of this writing is Xcode 4.6.1, and is highly suggested. An app built with this version of Xcode will be backwards compatible all the way back to iOS 4.3, but also take advantage of things in iOS 5.x and 6.x. Because of how SDKs work, if you build with an older version of Xcode, it is likely that Titanium will not be able to compensate for iOS 6's changes, and not behave properly.

    The __bridge you're seeing was something Apple added in iOS 5.0's SDK in order to allow for ARC to understand non-ARC code (It was added so ARC-based modules could work with Titanium). From the log, it looks like your copy of Xcode is 2 years out of date.

    • There is a version of Xcode 4.2 (which has iOS 5.0 SDK support) for Snow Leopard, but I don't think that's been tested.

      I'd really suggest upgrading your OS (It's only $20), as an app built with an XCode before 4.6.1 might have issues with orientation and the like, even with the latest Titanium build (For example, iOS 6 suddenly dropped support for the old style orientation and introduced new methods. Unfortunately, since these new methods only came with the iOS 6 SDK, we gracefully degrade, but only so far).
