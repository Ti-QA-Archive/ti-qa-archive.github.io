ListView in Alloy

Hi,

Does anyone know how to use the new ListView in the 3.1.0 release candidate with Alloy? All the examples I could find in the docs are straight-up normal Titanium code, understandably. I've tried using different combinations since now ListView expects ListView, ListSection, and ListItem. Thanks!

Titanium Mobile SDK 3.1.0-cr iOS 6.1 Simulator OSX 10.8 Titanium Studio 3.1.0

    In short, full markup integration of ListView won't be in Alloy until 1.2.0, but you can reference the ListView in markup just like any other cpomponent and then follow the existing Titanium guides.

    index.xml

    <Alloy>
    <Window>
        <ListView id="list"/>
    </Window>
</Alloy>

    index.js

    // $.list is now a reference to the ListView

    Watch this ticket to follow the progress of ListView integration: https://jira.appcelerator.org/browse/ALOY-585
