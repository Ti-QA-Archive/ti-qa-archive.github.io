"Application Installer abnormal process termination" after update on 3.2 when deploying Android application

I recently updated to Titanium 3.2. After the update I got serious problems running my Android applications on a physical device. I even cannot run sample applications.
I get the log messages:

[INFO] : Installing app on device: GT-I8160
[ERROR] Application Installer abnormal process termination. Process exit value was 1
[ERROR] : Failed to install apk on "6DFA5E7C6692FD4F40B15B0F8FE0072"

I am using Titanium 3.2 (build: 3.2.0.201312191547) on a 64bit Windows 7 with a Java 1.7 32bit SDK, my JAVA_HOME is set to this JDK. I am using Android SDK Level 16, which is declared in my tiapp file. I reinstalled my Titanium several times, did several cleans, but nothing helped. I even tried several Android ddevices. Before updating on 3.2 everything worked perfectly.

Any help would be appreciated.
