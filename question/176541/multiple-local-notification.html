Multiple Local Notification

How can i set multiple LocalNotification using Ti.App.iOS.scheduleLocalNotification

For example i need to set Local notification for daily 3 Times.How can i implement?

    I remember that there was a bug when scheduling multiple notifications on iOS. The solution is keeping a reference to the notification as returned from scheduleLocalNotification. Here's a snippet out of an app. It's old but it still should work.

    if (OS_IOS){
        var _notifications = [];

        for (var i=0; i < 4; i++){

            _notifications.push((function(value){
                var date = nDate.add('minutes', 30);
                return Titanium.App.iOS.scheduleLocalNotification({
                    alertBody: alerts[value],
                    alertAction: L("alert_action"),
                    sound: sounds[value],
                    date: date.toDate()
                });
            })(i));

        }
}
