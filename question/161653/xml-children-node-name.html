XML children node name

I have an xml like this

<element id="id1" >
    <name>
                    <![CDATA[<b>Folder 1</b>]]>
    </name>            

    <element id="id2" >
        <name>
                    <![CDATA[<b>Folder 2</b>]]>
        </name>
    </element>

</element>

I can't change the xml structure or space or others because become from external source.

I'm trying to get the text in the <name> value, but I'm not be able.

I've tried something like :

var folders = xmlSource.documentElement.getElementsByTagName("element");

var test1 = folders.item(1).text; 
var test2 = folders.item(1).text;

the result of test1 is blank;
the result of test2 is <b>Folder 1</b>

but I don't want to use a for because I will take all the "name" TAG in xml if I will use getElementsByTagName

My question are:

1. How I can directly see the text of the first "name" tag of <element> ?

2. there is some commands for ignoring white space in TI.XML.parseString() ?

Any help will be appreciate.

    var folders = xmlSource.documentElement.getElementsByTagName("element");

var test1 = folders.item(0).getElementsByTagName('name').item(0).text;

    you are getting test1 result is null because you have inner element tag of element tag. so when you use 

    xmlSource.documentElement.getElementsByTagName("element")

    the above line then it will return the array of element tag .
