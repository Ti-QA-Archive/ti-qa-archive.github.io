    Hi Sebastian

    if the apple stop the support for the UDID the world will bring something in the alternate solution at that time so dont worry and simply use it for now..

    And creating the own UDID is not bad but think of the scenario when the user uninstall the app and again install it then you again generate a new UDID even thought the user is already used the app before. Well in that case the UDID of the device is the best way to track the unique user.
