Will this work on multiple (Android) devices?

Hi,
I'm still trying to get a feel for building nice UI's with TI. I was looking at some sample code from Oil Reporter and saw:

var seeView = Ti.UI.createView({
  top: 10,
  left: 10,
  width: 300,
  height: 160,
  backgroundColor:'#333',
  borderRadius:6
});

In the app this builds a nicely centered curved background with padding of 10 on both sides, but will this work properly on devices that have a resolution that is not 320?

    Hi Frank

    If you wish to have a view that is centered with 10px on both sides, then set left and right to 10, and leave width at the default of 'auto'. This will allow the view to expand and contract depending on the width of the screen. Hence, your code will be:

    var seeView = Ti.UI.createView({
  top: 10,
  left: 10,
  right: 10,
  height: 160,
  backgroundColor:'#333',
  borderRadius:6
});

    Hope this helps
