How to add attribute in uses-permission tag in custom android manifest

Hi i am using custom manifestfile in my app using bellow link

Custom Manifest

I am using CAMERA option in my app.so it shows bellow permission in build manifest file by default when run my project.

<uses-permission android:name="android.permission.CAMERA"/>

but i wantted to add attribute android:required="false" like bellow

<uses-permission android:name="android.permission.CAMERA" android:required="false"/>

in my manifest file i added like above,but i did not get that attribute in build manifest file.

how i can able to add this? please find any solution..

Thank You in Advance.

  • How to add attribute in uses-permission tag in custom android manifest

    You don't. You can add it to <user-feature/>

    One says I want permission to use this. The other says I need this, or, Id' like this but don't have to have it.
