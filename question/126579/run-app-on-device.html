Run app on device

I've registered at apple developer center, I paid my yearly fee, I can install my app on my iPhone.

How do I run the app directly on the iPhone without having to sync it or re-installing it ?

As we can do on Xcode.

  • In the application view is a "play" button which, when clicked, gives you multiple run options. Pick run on iOS device. It will be compiled, sent to iTunes, and synced with the connected device. You will not need to resync each time to run it. So, yes, there is a way and it's quite simple. :)
