  • Your code to get the data is doing things in the wrong order and missing a few steps:

    var datosbadge = Ti.Network.createHTTPClient();

datosbadge.onload = function() {
    var json = this.responseText;
    var response = JSON.parse(json);

    //TODO

};

//Always recommended to have an error handler as well
datosbadge.onerror = function(e) {
    Ti.API.debug(e);
    Ti.API.debug(this.responseText);
};

datosbadge.open("GET",'http://test.com/folder/file.php');
datosbadge.send();

    Depending on the data coming back from the server, the TODO is the position in the code where you need to process the data, maybe looping around the objects/properties in the response object.

    Hope this helps.
