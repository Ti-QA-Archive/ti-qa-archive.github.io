How to pass parameters from parent controller to Alloy <Require> tag children?

I would like to pass couple of values to a children from the parent controller.
Here is my scenario;
parent.xml

<View id="childrenView">
    <Require src="children" id="rChildren" />
</View>

And here is what I've tried, but it did not work :(. source

parent.js

$.rChildren.init({
    key1: 'value1',
    key2: 'value2'
});

children.js

var args = arguments[0] || {};
alert (args);

it alerts
{id=rChildren}

Can anyone please give me some pointer here??

    Hi Himalay Sunuwar,

    Please find below the solution for you,

    parent.xml

    
<View id="childrenView">
    <Require src="children" id="rChildren" />
</View>

    parent.js

    
$.rChildren.init({
    key1: 'value1',
    key2: 'value2'
});

    children.js

    
var args = arguments[0] || {};

$.init = function(obj) {
    console.log(JSON.stringify(obj));
}

    Hope this helps you.

    • You are welcome Himalay.

      You can not access those parameter outside init directly. In order to do that you need to create local variable in controller like below,

      
var args = arguments[0] || {};
var key1;

$.init = function(obj) {
    key1 = obj.key1;
    console.log(JSON.stringify(obj));
}

function useVariable() {
    alert("This key is send from parent controlller ==>  " + key1);
}

      Note - If you are using this obj for displaying something on UI then that you can do it like below,

      parent.js

      
$.rChildren.txtField.value = "{ key1: 'value1', key2: 'value2' }";

      Hope this explanation helps you.

      Thanks,

      Narendra Raorane
