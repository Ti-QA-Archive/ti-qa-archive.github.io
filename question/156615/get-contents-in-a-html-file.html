Get contents in a html file

Hi everyone. I was wondering if there's a way to grab contents of a html file, just like with an xml file. For example with an xml file I could create a httpclient when I was grabbing content. The url I need information from is:

http://50.7.242.154:8070/7.html

All that's in there is some numbers and the name of a song and artist. Is there any way I can do this?

  • Hi

    Short answer - yes;

    Use httpClient, same as you have with XML, but use responseText instead of responseXML, then treat it as plan text.

    You add whatever parameters you need in the URL in the same way you would do anything else.
