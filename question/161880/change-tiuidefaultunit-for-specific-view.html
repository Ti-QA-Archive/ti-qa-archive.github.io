    • There is no way to choose a different default unit for a specific view or window. But as @Moritz Knecht shows here you can choose which unit of measure is being used at any time.

      To extend the example;

      // assuming the tiapp.xml has dp (or dip = the same thing) for defaultunit
var view = Ti.UI.createView({
    top: 10, // is 10 dp
    bottom:'10dp', // is still dp
    left:'10dip', // is still dp
    width:'10px', // is 10 pixels
});

      You can suffix;

      • dp or dip
      • px
      • mm
      • cm
      • in

      I would suggest avoiding; mm, cm & in - unless you have a very good reason.

      If you use something other than the default unit you must enclose the number and the suffix as a string - shown in each example here and above.
