    • If you want to completely hide all children ( including children of children recursively ) - The best way is to hide their top parent - in your case is mainContainer by mainContainer.hide();

      • If you want to show them later - just call mainContainer.show(); to show it and all its children ( recursively )

      • If you dont have the need to re-show it later, you should completely remove it:

      mainContainer.hide();
window.remove(mainContainer);
mainContainer = null;

    • ah I see. Yah you can assign id to A letters when creating it, then looping through mainContainer's children ( recursively ) to find element with a specific id.

      Try this sample code:

      var win = Ti.UI.createWindow({backgroundColor: 'white'});

var mainContainer = Ti.UI.createView({width: '100%', height: '100%'});

for(var i = 0; i < 4; i++) {
    var rec = Ti.UI.createView({height: 50, top: i*60 + 100, borderColor: 'red'});

    var lbl = Ti.UI.createLabel({
        text: 'A', 
        textAlign: 'center',
        id: i%2 ? 'even' : 'odd' // A letter with id even | odd
    });

    rec.add(lbl);
    mainContainer.add(rec);
}

win.add(mainContainer);
win.open();

mainContainer.addEventListener('click', function(){
    alert('mainContainer clicked');

    for (var i = 0, len = mainContainer.children.length; i < len; i++) {
        var rec = mainContainer.children[i];

        for(var j = 0, lenj = rec.children.length; j < lenj; j++) {
            var a = rec.children[j];

            if (a.id == 'even') { // Hide letter with id even
                a.hide();
            }
        }
    }
});

  • Here a possible solution, this example remove recursively any child (view):

    function removeChildrens(objeto) {
    for (i in objeto.children) {
        var child=objeto.children[0];
        removeChildrens(child);
        objeto.remove(child);
        child=null;
    }
}

    You can change the remove() sentence by the hide().

  • Until recently I've always used the looping method above..

    If you find you have a lot of child Views you can skip the need to loop each time by keeping an array of pointers to the Views:

    var Views = [];

//(loop code that adds your views)

parentView.add(childView);
Views[your index] = childView;

//(close loop)

    Now you can call:

    parentView.remove(Views[your index]);
delete Views[your index]; // remove from array too

    Or:

    Views[your index].hide();

    Enjoy ;)
