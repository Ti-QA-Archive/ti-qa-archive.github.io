How to tell when an Android app has been backgrounded?

Is there any way to do this? To determine when the app has been paused/suspended/backgrounded (by the Home key, or otherwise switching to another app) so that I can close any windows and exit instead?

There's this answer, which suggests there isn't, and that to do it (or something like it) is not the Android way.

But seeing as I'm trying to work around a worse Titanium bug, this is a starting point for yet-another-Android-hack.

(Basically what I want to do is save the application state and exit gracefully, and the resume from state when the application is restarted so it looks like the app didn't exit at all – instead of letting Titanium botch the exit when the backgrounded app gets forced out of memory.)

  • Im not sure but this works on iPhone to see if your app is pauzed/resumed, you should test this out on Android.

    Ti.App.addEventListener('resume', function(e) {
    // When the app is resumed do something
});

    Ti.App.addEventListener('pauze', function(e) {
    // When the app is pauzed do something
});
