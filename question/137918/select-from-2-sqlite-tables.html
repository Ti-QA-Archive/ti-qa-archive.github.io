    When you say 2 tables, are they in the same database file or two different files? If just 2 tables in the same database, just use standard SQL syntax to join them. The SELECT statement syntax in SQLite is pretty much what you'd expect from any SQL datababse.

    If they are in two different databases, then you need to "attach" them as Nick mentioned.
