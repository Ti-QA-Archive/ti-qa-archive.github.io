Current position of a view

Hello,

I need to get the current position of a view. I tried using the left and right properties but it seems they only hold the value, specified when rendering the view. But my view can be dragged to another position. Is it possible to retrieve this current position of a view?

Thanks in advance.

Rob

  • I solved it by setting the left and top property at the end of dragging (touchend) manually to the new position.

    win.addEventListener("touchend", function(e)
{
  //Set new left and top value
  e.source.left = e.x;
  e.source.top = e.y;
});

    This solution does not answers the question in general, but it may help others when having the same problem with drag and drop functionality.
