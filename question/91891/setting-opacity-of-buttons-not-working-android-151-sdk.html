Setting opacity of buttons not working (Android 1.5.1 SDK)

I'm trying to set the opacity of a button in code and end in a system crash (Android 2.2, 1.5.1 SDK) after setting the opacity from 1 back to 0.3.

Titanium.UI.setBackgroundColor('#FFF');

var win1 = Titanium.UI.createWindow({  
    title:'Test',
    exitOnClose: true
});

var button1 = Ti.UI.createButton({
    opacity: 0.3,
    top: 10,
    left: 10,
    width: 220,
    height: 80,
    title: 'Button'
});
win1.add(button1);

var button2 = Ti.UI.createButton({
    top: 100,
    left: 10,
    width: 220,
    height: 80,
    title: 'Set Opacity'
});
win1.add(button2);


win1.open();
button2.addEventListener('click', function(){
 if (button1.opacity != 1) 
    button1.opacity = 1;
 else 
    button1.opacity = 0.3;   
});

Also the button background is not painted when setting the opacity.

  • This is a confirmed issue. The value of 1 does not work, only values less than 1. Instead of 1, use 0.99. See error ticket here,

    https://appcelerator.lighthouseapp.com/projects/32238/tickets/2853-android-creating-button-or-imageview-opacity-to-1-causes-null-reference-exception
