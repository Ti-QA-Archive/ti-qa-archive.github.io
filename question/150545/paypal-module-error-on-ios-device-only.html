PayPal module error on iOS device only.

When I try to use the PayPal module in sandbox mode in the iOS Simulator, everything works fine and as expected. When I try to use it on an iOS device when the PayPal button is created, I get the alert: "System error. Please try again later." I don't see any further information in the XCode console about why I'm getting the error.

I tried running the example code that is bundled with the module and I experience the same problem. I only changed the appID to a valid one. Runs fine in the Simulator, error alert on device.

Any ideas?

Application type: mobile
Titanium SDK: 3.0.2.GA
Platform & version: iOS 6.0
Host Operating System: OSX 10.8.2
Titanium Studio: 3.0.2.201302191606
Test case: https://github.com/appcelerator/titanium_modules/tree/master/paypal/mobile/ios/example

  • It would appear that: Paypal.PAYPAL_ENV_SANDBOX does not work on an actual device.

    Changing it to: Paypal.PAYPAL_ENV_NONE resolved the problem for me. It would be nice to actually use the PayPal sandbox on the device to test the flow of money moving between accounts.
