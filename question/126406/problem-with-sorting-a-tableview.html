Problem with sorting a tableview

Hello!

I am trying to sort a tableview during runtime in my program. I am using this code to do that

            tableViewMain.data.sort(sortM);
            function sortM(ob1,ob2)
            {
                if (ob1.name > ob2.name)
                {
                    return 1;
                }
                else if (ob1.name < ob2.name)
                {
                    return -1;
                }
                return 0;
            }

The problem is when im running the code, i get "TypeError: Cannot find default value for object" on the line where i call data.sort. I thought this was the way i'm supposed to sort in, what is wrong?
