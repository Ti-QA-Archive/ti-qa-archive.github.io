  • Accepted Answer

    save() also takes similar parameters to set(). Your code:

    function test() {
    var mod1 = fighters.at(0);
    mod1.set({
        name : "I Love this game"
    }, {
        silent : true
    });
    mod1.save();
}

    could be more easily expressed as:

    function test() {
    var mod1 = fighters.at(0);
    mod1.save({
        name : "I Love this game"
    }, {
        silent : true
    });
}

    and it will fire none of your change events.
