    • Sorry i seems like i fixed my own problem.

      It turns out, I "did not assign password" to my mac user account (due to my laziness) and the "sudo command" which used to point to new xcode (an action automated in the latest Titanium Studio) requires me to have a password to execute properly.

      Once, I assign it, i manually use the console to execute "sudo xcode-select -switch /Applications/Xcode.app/Contents/Developer" which prompts a password.

      restarted the Titanium and it works now. Just want to say it here, so that it might helped some one else later :)
