When Opening a New Tab (or Window) - the Bar Flickers

I'm using a TabGroup with 3 windows. Each window is a separate .js file. At the beginning of each file, I am setting a custom bar color, a few icons, etc. Now - I do the same exact thing on each of the 3 main Windows (and sub windows from there). Every time I open a tab though, initially the bar is the standard blue, then it changes quickly to my custom background.

Is there a way I can fix this? Is there a way I can do this on a JSS level? Or - if I'm using a background image for the bar, am I stuck?
