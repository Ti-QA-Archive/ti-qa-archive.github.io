Need to Delete TableViewRow data through delete button located at each row

I am fetching data from a table "MyItems" (Titanium.Database) and adding In TableViewRow of TableView. Also, I have Delete button on each row.
Now, I want to delete each row data (from database & TableView) when user clicks Delete button.
Please advise/suggest solution for the same.

Thanking you in advance.

    Add "click" event to the table view component and in the event handler function check if button is clicked (and not table view row) by checking the source of the click.

    Something like this:

    
tableView.addEventListener('click', function(e) {
    if (e.source.toString() == '[Ti.UI.Button]') {
        tableView.deleteRow(e.index);
    }
});

    If you have more than one button in one table view row, add custom property to the delete button and check for that property (and not result of "toString" function).
