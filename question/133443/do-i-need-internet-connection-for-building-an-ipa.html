  • I don't know if you need internet connection, but you can also build your IPA from xCode opening your project from build folder.

    In xcode select Build for Archive and when the process is complete select Archive.

    Your app will be builded and archived, then in the next window you can select "share" option to save as ipa.

    Stefano
