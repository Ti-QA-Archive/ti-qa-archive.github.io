    • @mitul 

      Prior to Titanium SDK 2.1.3, applications were always built for both ARMv6 architecture (used by iPhone 3G and earlier devices) and ARMv7 architecture (used by iPhone 3GS, iPod touch 3rd generation, and newer devices).

Starting in Titanium SDK 2.1.3, if min-ios-ver is 4.3 or higher, the application is not built for the ARMv6 architecture, which may result in smaller application sizes. If min-ios-ver is not set, or is set lower than 4.3, the application is built for both ARMv6 and ARMv7 architecture, as before.
