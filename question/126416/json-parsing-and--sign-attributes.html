JSON Parsing and @ Sign Attributes

Hopefully someone can help me - I'm pulling my hair out! I'm using an XHR connection to get back JSON data. I'm using JSON.parse(this.returnText) to correctly parse the JSON data. However… I'm having an issue. All my end-nodes in JSON have the '@' sign at the beginning. '@Title' and '@Description'. 

var returnArray = JSON.parse(this.returnText);
var myTitle = returnArray.@Title;
var myTitle2 = returnArray.Title;

In that example - I get an error when I try to run the app (appcerator red screen). And 'myTitle2' is always null.

Am I missing something painfully obvious here? Thanks!

  • I thought I would answer this myself (if anyone else runs into this problem).

    Using this works:

    returnArray['@ReturnCode']

    will correctly get the data. Thanks for the help!
