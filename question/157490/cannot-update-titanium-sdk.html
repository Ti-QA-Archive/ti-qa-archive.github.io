  • Hi Angela

    Firstly please check you have run xCode. If it hasn't been run for the first time since installation, then you get errors like this.

    If xCode has been run and is all fine, then go to you home directory, ~/Library/Application Support/Titanium/mobilesdk/osx and check you have 3.1.2.GA installed you should also have 3.1.3.GA installed.

    If you don't have then instal them, if you do and your having problems delete the directories and start again. 3.1.3.GA should install automatically through Studio updates. You can get all the builds from here if you want to instal any previous SDK's.

    But if your using xCode 5 you really want to be using 3.1.3

    T.
