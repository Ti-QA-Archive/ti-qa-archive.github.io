  • Simply use the openURL function withe the sms: prefix.

    ####Like so:

    Titanium.Platform.openURL('sms:' + phone_number);

    This will open the message dialog window with the phone number already pre filled with the phone number.

    Note that this will only work using physical devices (makes sense when you think of it). this will not work on an emulator.
