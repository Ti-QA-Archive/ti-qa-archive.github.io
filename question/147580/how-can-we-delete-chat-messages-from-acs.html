    • I got the solution for the delete chat messages. We have to edit the following code in the
      cloud.js file

      {
            property : "Chats",
            children : [{
                method : "create",
                verb : "POST"
            },
            // Added this code in side the Cloud.js file in chats functionality,
            // which is beta version in Appcelertor as of January 31st 2013.
            {
                method : "remove",
                restMethod : "delete",
                verb : "DELETE"
            },
            //--------------------
            {
                method : "query"
            },
}

      It is working fine with one side chat deletion which I have messaged.
