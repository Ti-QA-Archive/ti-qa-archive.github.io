iPhone 5 support

What are the things, i have to do to give my app to iPhone 5 support.

  • Your app will still run the same it just wont use all the screen.

    The only difference is the screen size, so you must adjust in code for that (try using Ti.UI.FILL instead of absolute values for height) if your app is driven by TableViews you shouldn't have to change much.

    Also, you will need to add Default-568h@2x.png a 640x1136px image in your iPhone image assets folder, this is just the splash screen size of the new phone.

    Also, you need to have the latest versions of XCode, the command line tools, and Titanium Studio and SDK.
