  • Hi Nico, welcome to Titanium, here you have an amazing tool for mobile development and a great community. You are very lucky to study Titanium on school.

    You have a lot to learn, let me suggest to explore the docs page, just on the top menu of this page. Try to run KitchenSink, which will be your main reference of sample code and take a look to the guides available here

    http://wiki.appcelerator.org/label/guides/mobile

    Titanium works with javascript, but is not browser based nor DOM based, so html tags does not have sense here (there is a webView component to do that, but avoid to use it until you learn the rest of the components).

    The screen you have sent is not very difficult to reproduce, explore kitchensink and you will find how.
