    hi,

    so you want to save image in Android…right..?

    you should mention that in your question first.

    now if you want to save any media then you should save it in sd card instead applicationDirectory.

    and also for android you can get blob with myView.toImage().media;

    var Blob = myView.toImage().media;
var file = Titanium.Filesystem.getFile(Titanium.Filesystem.externalStorageDirectory,'image_1.png');
file.write(Blob);

  • Hi, I'm using ImageFactory module to convert png images to jpeg format on Android and iOS and works fine.

    Example:

    // read source png image
var sourceImage = Titanium.Filesystem.getFile(Titanium.Filesystem.resourcesDirectory, 'source.png').read();

// resize it using imageAsResized() if needed
var resizedImage = sourceImage.imageAsResized(160,120);

// choose desired compression level (from 0.0 to 1.0) 
var compression_level = 0.75; 

// load ImageFactory module
var imageFactory = require('ti.imagefactory');

// convert png to jpeg
resizedImage = imageFactory.compress(resizedImage, compression_level);

// create destination file
var destFile = Titanium.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory, 'destination.jpg');

// write jpeg image to destination file
destFile.write(resizedImage);
