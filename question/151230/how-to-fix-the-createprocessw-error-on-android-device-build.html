    @Matt, is adb in your PATH? In Titanium SDK 3.0.2.GA, you must have adb accessible in your system PATH, which adb is located in C:\path\to\android-sdk\platform-tools directory.

    In Titanium SDK 3.1.0.GA, we fixed it so that it finds it automatically based on your CLI config settings.

  • @Matt, you need to make sure you have Python 2.7 installed and in the system environment PATH. You can verify Python is properly installed by opening the Command Prompt and run:

    python --version

    If it says bad command, then you must download and install Python from here: http://www.python.org/download/releases/2.7.4/.

    Re-verify Python is installed in your path, then try again to build your project.
