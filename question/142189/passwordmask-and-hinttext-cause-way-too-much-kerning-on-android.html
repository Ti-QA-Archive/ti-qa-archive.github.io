passwordMask and hintText cause way too much kerning on Android

I am testing on Android 4.0.1 and the kerning for the hintText is quite a bit when used with passwordMask

var password = Ti.UI.createTextField({
    color:'#336699',
    top:148,
    width:330,
    height:48,
    hintText:'Password (\'example\' for tutorial)',
    passwordMask:true
});

If you use that as a text field, you'll notice that the hintText kerning is quite a bit. If you set passwordMask to false, the kerning is normal. Seems like a bug to me. Workaround for me was set a focus event listener on the text field and set the hintText to an empty string. Anyone else experience this?
