  • A:

    Yes, Xcode runs only on Macs. But I know that it is possible to run OS X (with Xcode) in a VM on Linux or Windows - I can't tell you much about that because I never tried out.

    B:

    There's a free iOS Developer Program that you can use to develop Apps for iOS. But to publish Apps on the App Store (or even to debug on your own devices) you need that paid developer program for 99$ / year.

    C:

    Apple get 30% of every sale in iTunes, exactly as Google does on Google Play.
