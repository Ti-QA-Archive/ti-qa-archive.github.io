returning data from a dialog box?

Hi again ;-)
I am normally used to programming in PHP and not so fluent in javascript yet, so I hope you bare with me asking a lot of questions.
Below is my dialog box with a multiple choice answer to my question stored in the variable q. How do I log the users answer into an array of various questions (which will be posed along the way)? In PHP I would just use the GET property and store the result in an array, but I cannot find any documentation on how to do that in javascript. Can anyone in here help me? Is it via an eventListener and then making a function? I have tried some poor attempts, using various ways of return, leading to nowhere…

var dialog1 = Ti.UI.createOptionDialog({
title: q,
options: [A1, A2, A3],
cancel:-1
});
dialog1.show();

Dearest Louise

    Just add a click listener to the OptionDialog. The event object parameter will include a property named index that will give you the selected index of your options array.

    dialog1.addEventListener('click', function(e) {
    alert(e.index);
});
