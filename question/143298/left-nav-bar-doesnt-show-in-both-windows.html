  • Accepted Answer

    If you're using a TabGroup, you need to open any new windows through the TabGroup's active tab. In other words, you can't just call

    october.open ();

    Instead, you need to open the window like this:

    _tabgroup.getActiveTab().open (october);

    (this assumes you have access to the _tabgroup). There are various ways to keep track of the tabgroup. You can pass it into the constructors to your window classes and store it in an instance variable within the class.

    var tg = Ti.UI.createTabGroup ();

...

var sc = new SeasonContent(tg);

...

    Then in SeasonContent.js:

    function SeasonContent (tabgroup) {
    var _tabgroup  = tabgroup;

    ...

    _tabgroup.getActiveTab().open (october);
}

module.exports = SeasonContent;

    A lot of people like to add a property containingTab to their windows. So instead of passing around the TabGroup, you just pass the individual tab to the window when you add the window to the tab. Then the window would pass that to any windows that it creates, and so on.
