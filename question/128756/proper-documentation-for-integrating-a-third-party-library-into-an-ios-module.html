  • Kwasi, did you resolve this? I know your question was specific to integrating a 3rd party library, as opposed to generically creating a module (which is what the Mod Guide shows). I happened to notice in the Codestrong Demystifying Modules video at ~11:35 in, one tip is briefly mentioned on 3rd-party libraries. Essentially, it says that they go in the module 'Assets' folder. I haven't done it, but would guess that from there you reference it in your Obj-C almost as you would in a straight Obj-C project, and therefore it becomes transparent to Ti and your Ti module that you are referencing an external library. I think this topic would be a good Blog Post candidate for AppC though.

  • This is the iOS Module Guide: http://wiki.appcelerator.org/display/guides/iOS+Module+Development+Guide

    That will show you how to create a module. If you want a working example you can find it here: https://github.com/appcelerator/titanium_modules

    The 'moddevguide' example in that repository is a perfect starting example.
