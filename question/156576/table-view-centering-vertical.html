  • Hi

    If you want your table to sit in the middle of the window with a gap at the top and a gap at the bottom;

    var tableview1=Ti.UI.createTableView({
    data: data2,
    separatorColor: 'transparent',
    backgroundColor: 'transparent',
    height: Ti.UI.FILL,
    top:"5%", // change to suit
    bottom:"5%" // change to suit
});
win.add(tableview1);

    If you want a table to sit in the middle of the window with a gap left and a gap right.

    var tableview1=Ti.UI.createTableView({
    data: data2,
    separatorColor: 'transparent',
    backgroundColor: 'transparent',
    width: Ti.UI.FILL,
    left:"5%", // change to suit
    right:"5%" // change to suit
});
win.add(tableview1);

    If you want to put a view above the table with a fixed height and the table taking the space left;

    var view = Ti.UI.createView({
    height: 50, // same value as the table top
    top: 0,
    width: Ti.UI.FILL
});
win.add(view);
var tableview1=Ti.UI.createTableView({
    data: data2,
    separatorColor: 'transparent',
    backgroundColor: 'transparent',
    height: Ti.UI.FILL,
    width: Ti.UI.FILL,
    top:50 // same value as view height
});
win.add(tableview1);
