Getting cell network name from iOS and or Android

Hi Guys.
Is it possible to get the name of the cell network back from iOS and Android?

Jason.

  • I don't think there's any way to directly get the information you're looking for. The only thing I can think of is to use a web service that looks at the device's IP address and does a reverse lookup to get the hostname, and use the domain of the host to determine the carrier.

    Here's a simple PHP script:

    <?php
$ipaddr = $_SERVER['REMOTE_ADDR'];
print gethostbyaddr ($ipaddr);
?>

    I've made this available at http://smorgasbork.com/revdns.php if you want to try it.

    I tried hitting this with my Sprint phone, and I got "108-108-19-121.pools.spcsdns.net".

    I guess you could compile a list of known domains used by carriers (or maybe that's available online somewhere).
