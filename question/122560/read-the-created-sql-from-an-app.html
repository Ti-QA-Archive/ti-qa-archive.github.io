Hello

Is it possible to read an .SQL file that is created from an SQLite database when you run the app? I would like to have the entire database (with the data that is inserted from the app).

This doesn't get inserted in the .sqlite database, only in the .sql file.

How can I read this entire database? (With the changes made from using the app?).

    On iOS, the answer is no you can't (as far as I know). On Android, you can use the adb command to either open a shell and execute SQL commands against SQLite, or pull the .sql file.

    See StackOverflow or Android docs
