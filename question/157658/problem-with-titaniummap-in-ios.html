Problem with Titanium.Map in iOS

Greetings (and sorry for any mistakes in my english),

I have in my application a couple of tabs. One tab open a window with a Titanium.Map to show a map with some locations of interest. When I updated to the new SDK 3.1.3 and iOS 7, I started to have a problem: When I navigate throughout the tabs, all thing are fine, but when I navigate to the window with the map and navigate to another window, the bar color of all windows turn to black! The thing is I don't use black in any bars of my application. I tried taking off the map from that problematic window and thing gone perfectly fine, but when I turn back the map, I have the black bar color problem… In the simulator I don't have this problem, just in my iPhone 4 (with iOS 7).

My map:

var mapView = Titanium.Map.createView({
    mapType : Titanium.Map.STANDARD_TYPE,
    region : {
        latitudeDelta : 0.1,
        longitudeDelta : 0.1
    },
    animate : true,
    regionFit : true,
    userLocation : false,
    width : TiPerc.getPerc(98),
    height : TiPerc.getPerc(35),
    top : 10,
    left : 8,
    borderWidth : 1
});

Any ideas?

Thanks in advance!
