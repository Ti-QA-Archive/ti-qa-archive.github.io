Upload image to server

Hi guys,

I was hoping someone would be able to assist me with this issue I am having with my Android App. I an the option to upload images to our server with some code that displays and uploading message followed by a completion message. The problem I am having is that the "Image Uploading Please Wait" is getting stuck even after the image is uploaded. I verified on the server that the image is there but on the device the message stays stuck and does not display "Done Select Next Image or Submit". Can someone please assist with this? Thanks.


    var timeOut;
    var changeProgress = function() {
        var uploaded = 0;
        for (var x in blocks) {
            uploaded += blocks[x].uploaded;
        }
        pb.setValue(uploaded);
        timeOut && clearTimeout(timeOut);
        if (uploaded >= pb.getMax()) {
            pb.setMessage("Done Select Next Image or Submit");
            timeOut = setTimeout(function() {
                pb.hide();
            }, 500);
        } else {
            pb.setMessage("Image Uploading Please Wait");
        }
    };

  • Are you using a HTTPClient to upload you image to the server?

    If so you can check it's state of uploading by checking the event onreadystatechange

    See http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.Network.HTTPClient-property-onreadystatechange

    When the state is DONE (http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.Network.HTTPClient-property-DONE) it should have finished uploading and you can set your progress
