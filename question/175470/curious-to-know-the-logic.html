    Well, looking at the URL you are calling, you are returning the following:

    {
    location_name: "Central Parking",
    address: "500 W. Monroe",
    city: "Chicago",
    state: "IL",
    zip: "60606",
    type: "commercial-garage",
    timezone: "America/Chicago",
    lat: 41.880776963202,
    lng: -87.640315890312,
    eticket: 1,
    directions: "Look for the green Central Parking post on the sidewalk near the     garage entrance. The entrance is on the left on W. Monroe Street between Clinton     and Canal Street. There is an alternate entrance on Canal Street.",
    description: "Perfect spot for the Civic Opera House as it is very close. Indoor     parking lot completly sheltered from the elements. Heated during the winter and     well lit in a class A office building. This is a 24/7 parking garage.",
    recommendations: 318,
    accepting: null,
    listings: [ ]
}

    And there is no results element in this response. So in order to display the actual parsed object, you would have to use the following code:

    var response = JSON.parse(this.responseText);
Ti.API.info(response);

    Also, use the GET action when invoking your http request instead of Get.

    Hope this helps
