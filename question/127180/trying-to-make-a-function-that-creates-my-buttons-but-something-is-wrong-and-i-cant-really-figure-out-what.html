Trying to make a function that creates my buttons... But something is wrong and I can't really figure out what

Hi All.
I am trying to make a function that will generate buttons that I can custom place around the screen. The code is below. When I try it out in the simulator it crashes, and I can't really figure out what is wrong. I am passing in two variables and below I am constructing my button. Can anyone help me? Maybe this isn't the way at all to do it. I just wanted to find some smoother way to creating various similar buttons instead of writing all the same code for each. Does it make sense what I am trying to do?

function myQbuttons (posTop, posRight) {
var myButton = Ti.UI.createButton ({
top:posTop,
width:20,
height:20,
right:posRight,
Color:'black'
});
};

var button1 = myQbuttons(posTop=20, posRight=20);

Obviously this code below works, but because I will need to have varying amount of buttons, I think there might be an easier way.

var button1 = Ti.UI.createButton ({
top:20,
width:20,
height:20,
right:20,
Color:'black'
});

  • Accepted Answer

    In your myQbuttons function, you forgot to return the newly created button.

    The final function should look something like this:

    function myQbuttons (posTop, posRight) { 
    var myButton = Ti.UI.createButton ({ 
        top:posTop, 
        width:20, 
        height:20, 
        right:posRight, 
        Color:'black' 
    }); 

    return myButton;
};
