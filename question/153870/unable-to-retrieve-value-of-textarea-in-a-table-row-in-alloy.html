  • Accepted Answer

    You're trying to reference a field by id, but you are giving the same id to every instance of the comments textfield. That isn't going to work. When you are trying to reference specific fields in a databound model, there's a couple things you can do:

    • Use the events built in to the textarea and react to them as opposed to trying to get them directly.
    • Hold a reference to particular UI elements in your models to access the corresponding text area directly.

    How and when are you trying to access these textareas so I can better understand which option you should use.
