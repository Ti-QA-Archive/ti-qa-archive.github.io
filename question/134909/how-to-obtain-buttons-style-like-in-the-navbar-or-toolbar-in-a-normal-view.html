  • Hi Cristian,

    try this, it will help you to make style button.

    var btnClear = Ti.UI.createButton({
    title : "Clear",
    style : Ti.UI.iPhone.SystemButtonStyle.PLAIN,
    right : 10,
    height : 31,
    width : 52,
    backgroundGradient : {
        type : 'linear',
        colors : ['#759563', '#1c5202'],
        startPoint : {
            x : 0,
            y : 0
        }
    },
    borderRadius : 5,
    color : 'white',
    font : {
        fontWeight : 'bold',
        fontSize : 14
    }
});

    • If you really want to create a button that looks like original button and did not want to decorate anymore then try to button bar. It creates button Same as iOS creates.

      var btn = Ti.UI.createTabbedBar({
    labels : ['Ok'],
    style : Ti.UI.iPhone.SystemButtonStyle.BAR,
    right : 10,
    height : 31,
    width : 100,
    top : 10,
    backgroundColor : "blue"
}

      It will be the solution that you want. but you have to code little more for that or change your code.
