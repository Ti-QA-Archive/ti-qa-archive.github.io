Problem when playing video on webview

Dear All,

I'd like to open the whole webpage which includes videos, and entires of the page by using a web view.
When I click on the play button on the video, It doesn't work on Android. (It works on iPhone)
I'm testing using Google 2.3.3 API and Titanium SDK 3.1.1
I tried many ways like: Adding and add the line settings.setPluginsEnabled(true) to the constructor of TiUIWebView.java, or adding android:hardwareAccelerated="true" to the tiApp.xml… but they didn't work as well.

Can anyone help?
Thank you,
Dang

    fullscreen : true,
    navBarHidden : true,
    exitOnClose : true,

var url="http://www.youtube.com/watch?v=V70ruRzEGOI";

var webView = Ti.UI.createWebView({
    url : url,
    scalesPageToFit : true,
    touchEnabled : true,
    top : 0,
