How to clean a spot and make it transparent?

I would like to draw a spot over an image and allow the user to remove it using the finger.

You can see the effect on this video: http://www.youtube.com/watch?v=p_WLu9lfpVs (after 10 seconds).

Any suggestion how to achive it?

    Hi

    Understanding your problem is actually essentially the same as scratching a silver panel of a lottery card, the silver panel being the top layer, the scratching being the reveal and the card being the layer reveal. It can be "easyish".

    The key

    The key here is an iOS only view Titanium.UI.MaskedImage, this allows you to create a layered image that can have a top layer than can be combined with a bottom layer. The top layer is a mask that you can set to expose the bottom layer.

    The only problem is that you need to provide an image mask that contains transparent sections that reveal the bottom layer. Obviously you can achieve the position of the etching out with touchmove but you are left with creating a an image for each position and updating the MaskedImage with the new mask with the new transparent section from the latest touch position.

    Good Luck.

    • This teddy is a layer (an image with transparent "thick border"). On event:

      teddyView.animate({
    opacity:0,
    duration: 1000,
    transform : Ti.UI.create2DMatrix({scale:0.1})
});

      This blurs and shrink the teddy after an event i.e. touchend
