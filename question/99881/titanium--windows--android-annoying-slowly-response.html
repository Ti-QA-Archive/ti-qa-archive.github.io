  • Mario,

    The time it takes to boot an emulator or launch an app in an emulator is almost entirely dependent on Google's SDK.

    However, so that you have a guide:

    An application launched in TiDev running locally on an Ubuntu machine (i7 processor, 6GB memory) takes 40 seconds to an already-booted emulator, as this video shows.

    An application launched in TiDev running on a Windows 7 virtual machine (VirtualBox), on the same Ubuntu host machine, takes 120 seconds to an already-booted emulator, as this video shows. Bear in mind that in this case the guest only has access to a fraction of the resources of the host, and so it is expected that it will take longer to launch the app.

    Hope this helps

  • One thing that is not clear in the developer tool is that while you need to stop the iOS emulator before launching a new build, stopping the Android emulator requires a full restart – which takes a very long time. But once the Android emulator is started you can just relaunch new builds. This still takes about a minute but is still much faster than stopping and restarting the emulator each time.
