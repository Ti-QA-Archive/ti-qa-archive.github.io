    • Yes i did.
      The reason for my crash was that in every call to the 

         Ti.Storekit.purchase(product, function(r) { })

      i was calling the 

         Ti.Storekit.requestProducts(['prodID1','prodID2'], function(e) { })

      so what i did was create an array and populate it with the products that was returned by the Ti.Storekit.requestProducts function

      var iap_products = []; // at the top of my code;
...
...
// so for the first purchase click to get all the products i did
if (iap_products.length==0) {
    Ti.Storekit.requestProducts(['prodx','prod2'], function(e) {
        if (e.success) {
           for (var i = 0; i < e.products.length; i++) {
            iap_products.push(e.products[i]                       }
           // call the purchase function
           buyProduct()
        }
    }
}
else {
 buyProduct()
}

      then my buyProduct function() is something like this:

      function buyProduct() {
  var product = iap_products[ //here you past the itemIndex of the product selected and previously returned by Ti.Storekit.requestProducts ];
  // and then you call the normal Ti.StoreKit.purchase function
  Ti.Storekit.purchase(product, function(r) {
     .... 
  });

}

      hope it helps. It worked for me since the cause of my crash was the repetion of the Ti.Storekit.requestProducts
