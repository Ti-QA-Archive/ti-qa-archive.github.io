Annoying Home/End on OSX keyboard in Titanium Studio

I am really annoyed of the OSX handling of the Home/End keys. Instead of moving to beginning/end of line as in Windows, it jumps to the beginning/end of the document. This is worthless for coding.

I have tried to change my OSX keyboard layout and it works for many other applications (including code editors, e.g. Komodo Edit), but it is just ignored by Aptana.

Is there any way to get the usual Windows behavior for the Home/End-Keys?
