How to build an app with two tab groups

Hi guys,

I'm migrating my app to CommonJS. My app has two tab groups: the login one with one window, and the app one. I just need to close the Login one if I get valid login credentials, and then open the App tab group. What's a best practice to do this?

I'm only using a tab group for the Login window because I can have that nice title bar at the top of the tab where I can add a button for options.

Thank you!

    You could create two instance of the commonjs module (here is a sample of such a module https://gist.github.com/1672333 )

    This would give you two instance of the tabgroup so you can open / close similar to how you are doing it now.
