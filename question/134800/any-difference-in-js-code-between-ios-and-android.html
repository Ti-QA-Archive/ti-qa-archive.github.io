  • There are many structure differences , but it's 10x easier to convert Android JS to iPhone JS than Java Android to xCode iPhone.

    Main differences:

    1. Navigation bars
    2. Type of windows (heavyweight/lightweight is Android-only)
    3. iPhone has no "activities" nor "intents"
    4. iPhone has special kind of navigation icons
