Detect which TableViewRow has been swiped

I have a dynamically generated TableView and want to be able to swipe the row to pull up extra options that will be specific to that row.

TableViewRows don't seem to be able to receive swipe events, and while TableViews do i can't find out how to determine which row fired the event. There seems to have been an option in a previous version of the SDK to see row or rowIndex, but that doesn't seem to be supported anymore.

Has anyone worked this out? Using SDK 1.8.3

Thanks
