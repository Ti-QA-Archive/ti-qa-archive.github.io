What is a compiled Titanium consist of?

Hello,

I'm considering using Appcelerator Titanium for an iPhone and Android app I'm working on. I understand that it compiles JavaScript down into some native format. My question is what is that format? Is actual Objective C code generated so that I could edit that later if I wanted to?

Thanks!

    I think it may be more accurate to say generating actual obj C code is a qualified yes. You do get an xcode project created, and can manipulate that and recompile in xcode but in general you don't want to since any changes there would be lost if / when you regenerate from the titanium project source.

    In the xcode project you can see / modify all the obj c source for all the titanium classes. But the *.js files which have "your" source are not strictly speaking converted to equivalent obj c source code. I'd suggest you look at the top answer on this question.

    However, being open source, you can modify what you want. That said, IMHO you are better off creating your own "module" to add extensibity to the titanium framework so you can continue to take advantage of any new SDK features /fixes as titanium evolves.
