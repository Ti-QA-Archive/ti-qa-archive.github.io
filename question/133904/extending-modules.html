Extending Modules

I tried to create a basic module like so:

function MyWin(win_title) {
this.win = Ti.UI.createWindow({
title : win_title,
// navBarHidden: true
});
};

module.exports = MyWin;

So I saved this to say mywin.js

Now what I want to do is extend this base window, and add more functions to it say:

MyWin.prototype.getWindow = function(index) {
return this.win;
};

How do I do this in a separate file, so they I can build a complex set of functions for this class.
Or can I do a class extension some how.

— asked March 16th 2012 by Steven Marshall

  • You should keep related functions in the same file.

    So mywin.js should look something like this:

    function Mywin(title) {
    this.win = Ti.UI.createWindow({
        title: title
    });
}

Mywin.prototype.getWindow = function() {
    if (typeof(this.win) != "undefined") { 
        return this.win; 
    } 
    else {
        Ti.API.info("No window created.");
    }
}

module.exports = Mywin;
