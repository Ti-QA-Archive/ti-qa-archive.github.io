Open window from index.js, cant reference window using $. in the controller of the open window to open webview

Hello,

I am opening a window on starting the my app, opening the login window via the index.js.

index.js


var login = Alloy.createController("login").getView().open();

This window opens fine. However, I would like to open a couple WebViews in login.js via a button click, but I can't get a reference to $.login to do

var webView = Ti.UI.createWebView({
    url: "http://www.domain.com"
})

$.login.add(webView);

How am I losing the reference to $.login?

Thank you in advance!

  • Accepted Answer

    You probably changed the id of the window. $.login references an element with the id "login". By default, this is the top-level element of your XML, referenced by its controller name.

    Have a look into your xml, if you have given another id like

    <Window id="loginWindow" >
  ...
</Window>

  • ah – need to add a window first, then add the webview to the window, add the new window to the current window, open new window.

    
$.hrefUsername.addEventListener('click', function() {
    var win = Ti.UI.createWindow();    
    var webView = Titanium.UI.createWebView({
        url: "https://www.domain.com/"
    });

    win.add(webView);
    $.login.add(win);
    win.open();
});
