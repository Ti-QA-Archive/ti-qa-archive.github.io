Titanium.UI.ScrollView add() returns NullPointerException on Android

Generally, I try to avoid a complete redraws of objects to the screen as much as possible. I have a horizontal ScrollView that I append standard Views onto. New Views are generated by polling a remote server.

Similar code to this works great on iOS

someScrollableView.add(newlyCreateView);

On Android I get the following error:
[ERROR][TiApplication( 2457)] (main) [3565,4447] Sending event: exception on thread: main msg:java.lang.NullPointerException; Titanium 2.1.2,2012/08/23 21:31,fe81dcf

sdk-version: 2.1.2.v20120823213110

Any work around for this? The docs allude to the fact that this doesn't work on all devices, but the docs aren't explicit about Android / iOS compatibility.
