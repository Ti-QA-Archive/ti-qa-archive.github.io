    • Just an update as there has been no activity on this question, 9-patch will work with Android for standard images as I read about it today when checking the docs.

      ImageView has a couple of paragraphs, with links to Android Developer documentation. So that should help you if you are stuck.

      If this isn't enough, @Malcolm Hollingsworth did a write up on TiDev about 9-patch splash screens, where the same logic applies. android splash screens using 9-patch images

      Some crucial things to point out are the naming conventions, so the name would look like this

      imagename.9.png

      The documentation tells you to place your 9-patch images in the your "/images/" folder.

      Also when referencing the 9-patch images you don't reference the "9", simply "imagename.png"

      Hope this additional information helps further.
