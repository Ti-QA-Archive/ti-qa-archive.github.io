  • Hi Dario, I'm sorry you're having problems.

    I was able to confirm the behavior, and it is definitely a bug. TIMOB-8275 fixes this bug, for the most part, except that you still can't use the mouse scroll wheel to scroll tableviews in Firefox. I filed bug TIMOB-9095 to address the mouse scroll wheel issue.

    TIMOB-8275 is mostly complete, but there are a few kinks that need to be ironed out before it can be merged into master. It should be merged into master in the next week or so. This fix is slated for the 2.1 release, although if you need it sooner you can always use a Continuous Integration (CI) build of the master branch. I must warn you though that master is not considered stable.

    I don't think there is a way to work around this with the 2.0.1 codebase unfortunately.
