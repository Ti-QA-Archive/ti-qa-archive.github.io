Best practices for deleting data

Hi, I have a fairly simple application but implemented using the wrong approach, that is creating windows using the js url like the kitchensink.
I am aware of the tweetanium approach and I'm planning to change the app that way as soon as I can.

There's about 10 windows in my app. The good thing is that all the windows are independent and there is no need to move data or communicate between them at all.

So in the meanwhile I'd like to make sure I clean up as much as I can while navigating the app. What I'm doing is going backwards when I close a window.

So I use window.remove any views I might have added to the window + set the variables of any objects to null. I know we can't trust that everything is deleted but is there anything else I should take care of?

Thanks.
