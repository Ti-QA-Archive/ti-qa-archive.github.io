Best method to refresh all Window colors/themes?

In my current application I have 5 tabs each with their own windows. Each window can also be in its own level of navigation (3 windows deep for example). I am trying to add "Theme" functionality, where the user can select their own color combinations. I am adjusting things like nav bar color, tableview row color, etc.

On my Settings window, after the user selects their new theme. What is the best way to update all other windows? Would I just change them onfocus? Would I attempt to loop through and update them all? Maybe even close the TabGroup and reopen it?

Any help would be appreciated.

(This is for iPhone and iPad and I am running the lastest Titanium SDK)

  • I'd add in each window an Ti.App event, that would change the theme for each particular window.
    If the window it's opened, the event is listened and executed.
    Don't forget to remove the event when you close the window to avoid multiple triggers and memory leaks.

    Makes sense?
