    Hi

    Yes is the short answer.

    Longer answer;

    • You can use sound at the most basic level you can work with input level. Using this you detect the loudness of sound and react to that, something low - do first thing, something high do something else.
    • You can record snippets and process for patterns - MUCH harder

    As you have not said what platform you are using this example is iOS only but you can get the idea from this of what you would do.

    Using the first answer;

    // turn on monitoring
Ti.Media.startMicrophoneMonitor();

// retrieve this information is a interval of some kind
var peak = Ti.Media.peakMicrophonePower;
var avg = Ti.Media.averageMicrophonePower;
// react to the level information returned and fire events as required

// turn off monitoring
Ti.Media.stopMicrophoneMonitor();

    This technique is used in apps such as Credit Card swipes (using headset socket), music apps that rely on blowing into the microphone and more.

      if (avg > 0.1 && avg < 0.5) {
