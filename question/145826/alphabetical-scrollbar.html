  • Hi

    This should help you;

    var data = [
    { title: 'Adam' },
    { title: 'Andrew' },
    { title: 'Bob' },
    { title: 'Brian' },
    { title: 'Charles' },
    { title: 'Chris' },
    { title: 'Curtis' },
    { title: 'Daniel' },
    { title: 'David' },
    { title: 'Don' },
    { title: 'Duncan' },
    { title: 'Edward' },
    { title: 'Evan' }
];
varindex = [
    { title: 'A', index: 0 },
    { title: 'B', index: 2 },
    { title: 'C', index: 4 },
    { title: 'D', index: 7 },
    { title: 'E', index: 11 }
];
var tbl = Ti.UI.createTableView({
    data: data,
    height: Ti.UI.FILL,
    index: index,
    width: Ti.UI.FILL
});
win.add(tbl);

    The index property of a table uses an object array to have pointers to the positions within your table, these are set by the index property and are simply index points to the rows. So E points to the 11th 0 based entry in the table in my example. The title property is what you want disabled.

      OK, have you tried adding labels to the table? If so you could then make the index bar yourself.

      Try this test, this should scroll to the top when you press a, if this works you can then repeat with other letters.

      var data = [
    { title: 'Adam' },
    { title: 'Andrew' },
    { title: 'Bob' },
    { title: 'Brian' },
    { title: 'Charles' },
    { title: 'Chris' },
    { title: 'Curtis' },
    { title: 'Daniel' },
    { title: 'David' },
    { title: 'Don' },
    { title: 'Duncan' },
    { title: 'Edward' },
    { title: 'Evan' }
];
var lblA = Ti.UI.createTable({
   height: Ti.UI.SIZE,
   text: 'A',
   width: Ti.UI.SIZE
});
lblA.addEventListener('click', function (e) {
    tbl.scrollToIndex(0);
});
var tbl = Ti.UI.createTableView({
    data: data,
    height: Ti.UI.FILL,
    width: Ti.UI.FILL
});
tbl.add(lblA); //add this LAST to the table
win.add(tbl);

      var data = [
    { title: 'Adam' },
    { title: 'Andrew' },
    { title: 'Bob' },

var viewAZ = Ti.UI.createView({
    backgroundColor: 'orange',
    height: Ti.UI.SIZE,
    layout: 'vertical',
    right: 10,
    top: 10,
    width: 20
});
var lblA = Ti.UI.createLabel({
    color: '#000',
    height: Ti.UI.SIZE,
    text: 'A',
    top: 10,
    width: Ti.UI.SIZE
});
viewAZ.add(lblA);
lblA.addEventListener('click', function (e) {
    tbl.scrollToIndex(0);
});
var lblB = Ti.UI.createLabel({
    color: '#000',
    height: Ti.UI.SIZE,
    text: 'B',
    top: 10,
    width: Ti.UI.SIZE
});
viewAZ.add(lblB);
lblB.addEventListener('click', function (e) {
    tbl.scrollToIndex(2);
});
var tbl = Ti.UI.createTableView({
    data: data,
    height: Ti.UI.FILL,
    width: Ti.UI.FILL
});
win.add(tbl);
win.add(viewAZ);

      Do not try my first suggestion of placing the label in the tableView - this may work well in iOS but not in Android.

      What we are doing is placing a view OVER the table and placing a couple of letters inside that view. Then each letter is told to jump to the correct index in the table. Scroll to the bottom of the table and then press the letters.

      You can extend this by auto adding letters as you need them of course.

      You are welcome to remove the orange background - just there so I knew where it was displaying.

      Not a quick as the iOS version - but works just as well.
