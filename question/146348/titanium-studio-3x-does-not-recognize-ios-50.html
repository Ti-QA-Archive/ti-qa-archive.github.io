Titanium Studio 3.x does not recognize iOS 5.0

I recently updated Titanium Studio from 2.1.2 to 3.0.0 (and now 3.0.1) on Mac OSX 10.6.8. When using Titanium SDK 2.1.4.GA I am having no trouble launching the iPhone/iPad simulator. However, when I switch to Titanium SDK 3.0.0.GA, on trying to launch the iPhone/iPad simulator I am given the error "Invalid –ios-version value '5.0'". I've checked the iOS SDK Home path and it correctly points to the parent directory of the iPhoneOS5.0.sdk directory. I'm at a loss as to how to resolve this issue. Any help would be much appreciated.
