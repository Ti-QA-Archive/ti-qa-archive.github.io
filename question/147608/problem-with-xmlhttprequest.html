problem with XMLhttpRequest

I am trying to get the user information from google. The authentication works perfectly fine. But when I try to get the user data, the XMLhttpRequest returns an error. But if I copy paste the URL on a webbrowser manually I get the data! What could be the problem in accessing the data using XMLhttpRequest?

i.e.

    var xhr = Titanium.Network.createHTTPClient();

    xhr.open("POST", "https://www.googleapis.com/oauth2/v1/userinfo?alt=json&access_token="+ ACCESS_TOKEN);
    //xhr.setRequestHeader("Content-Type", "text/plain");

    xhr.send(params);
    xhr.onload = function(e){ };

This gives an error onload. but if I manually copy paste the url in a browser it works…

  • I notice that you're using POST. When you paste it in a browser, you are using GET.

    It's quite possible that the web service at google is ignoring your GET parameters when the method used is POST.
