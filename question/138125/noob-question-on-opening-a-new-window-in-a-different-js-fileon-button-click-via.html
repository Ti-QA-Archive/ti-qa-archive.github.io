  • In your second file, it looks like you are trying to add a button to a tab and then add a tab to a window, which isn't going to work out for you. Also, if you are only using windows, you shouldn't be doing anything with tabs or tab groups.

    You were close with the commented out items, just try:

    win1.add(metric_button_female);
win1.open();

    No views necessary, unless you want them.

    Like Aaron said, the eBook is probably a place where you want to begin.
