  • Sergey

    The safest solution to this would be to delete all the build files and force a full rebuild.

    Thus, for iOS/iphone, delete the contents of the project's build&#x2F;iphone directory, select Edit Project in Titanium Developer and click the Save Changes button (to touch tiapp.xml which forces a full rebuild). Then launch the app.

    There is no need to raise this as a bug. Clearing your build files should be considered your first step in typical troubleshooting process, in which case it wouldn't have caused you as big a problem as it did.

    Hope this helps
