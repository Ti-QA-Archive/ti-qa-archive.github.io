How to parse "description" element from an RSS feed?

I want to present the different elements from an RSS feed in my app. It works fine, besides that the <description> -tag is cluttered with more than just the actual text (which i want to display):

<description><div class="field field-name-body field-type-text-with-summary field-label-hidden"><div class="field-items"><div class="field-item even" property="content:encoded"><p>The actual text i would like to display in my App!<...

So my question is, does anyone know if there is such a description-"parser" method in the API somewhere that i can use to trim whats in the description tag? Or any other good solution?

    You can get the whole tag content using the textContent property.

    And then strip all the html tags using regular expressions.

    Example:

    var regex = /(<([^>]+)>)/ig;
var body = "<p>test</p>";
var result = body.replace(regex, "");
alert(result);

    Will show an alert containing only the word test.
