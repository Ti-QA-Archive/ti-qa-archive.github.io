Loading android images

I have created a new mobile project, and can see the folders in the images directory like so:
res-long-land-hdpi
res-long-land-ldpi
etc

I have the following simple bit of code:

var image1 = Titanium.UI.createImageView({
    top:0,
    left:0,
    width:100,
    height:100,
    image : "test.png"
});
win1.add(image1);

When i place test.png in the root of the "Resources" folder, it loads, when i move it to any of the android image folders, it doesn't. How do i load an image specifically for an android screen size/density?

  • Fixed it, needed to have the path of "images":

    var image1 = Titanium.UI.createImageView({
    top:0,
    left:0,
    width:100,
    height:100,
    image : "images/test.png"
});
win1.add(image1);
