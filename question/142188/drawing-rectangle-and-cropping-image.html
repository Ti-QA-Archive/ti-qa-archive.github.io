Any ideas?

  • So lets break down the functionality -

    1) you need to be able to set an image in a view
    2) draw a rectangle on top of the image
    3) add a listener to the pinch event to capture the pinch event (http://docs.appcelerator.com/titanium/2.1/#!/api/Titanium.UI.View-event-pinch)
    4) Change the size of the image based on the gesture made by user..
    5) once done save the image on to the device.

    Which part are you having problem with? Between the Kitchen Sink and the documentation, this should be pretty straight forward but it is highly unlikely that someone will code the whole thing for you on this QnA forum.

    If you have any specific questions, please ask.

    One last thing - In the snippet above, screenShot just really have to be any blob. So if you have an image loaded from device into a view, you can get a blob for that file to crop it.
