Alloy widget with multiple windows?

I'm looking for an example of a Widget with multiple views that the controller can decide which one to display when called from the main app. For instance, a widget that has a logged-in and logged-out view.

I want to have the widget determine which view to display and open it. It might look like this:


var x = Alloy.createWidget('xxx');
x.openCorrectWindow();

    Late response, but wanted to put answer here if others came looking

    To use another view-controller besides widget.js/widget.xml, use the createWidget(widget_name, controller_name) method to create a new instance of the controller and the getView() method to access the Titanium proxy object.

    http://docs.appcelerator.com/titanium/latest/#!/guide/Alloy_Widgets
