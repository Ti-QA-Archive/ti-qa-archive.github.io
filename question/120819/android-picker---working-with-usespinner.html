Android Picker - Working with 'useSpinner'

Titanium Mobile 1.7 RC1 - Titanium Developer 1.2.2 - Mac OS X.6.7

I'm posting this to help others who may run into problems using Android picker views & the useSpinner option. It is quite handy the Titanium team added this option, especially for multi platform development. However, like many things in Android, usage of useSpinner is not very intuitive and takes some trial and error.

To successfully use 'useSpinner' avoid adding your spinners columns to the picker object once the picker is initialized. You must add the picker columns on initialization or the app will crash/force close.

Use:

mPicker = Ti.UI.createPicker({
    height:mPickerBaseHeight,
    width:"100%",
    top:0,
    useSpinner:true,
    type:Titanium.UI.PICKER_TYPE_PLAIN,
    selectionIndicator:true,
    columns:[mMajorUnitColumn,mMinorUnitColumn,mUnitTypeColumn]
});
