Switching Windows

/*
 * Navigate to window on the stack
 */
NavController.prototype.navigateTo = function(title) {
    var window_index = stackIndex(title);
    // open_windows[window_index].show();
    Ti.API.info("window_index:"+window_index);
    var window_count = open_windows.length;
    var i;
    for( i = (window_count - 1); i > 0; i--) {
        if(i > window_index) {
            open_windows[i].close();
            central.pause(1000);
        }
    }
};

At the moment this code works fine, but on iPHone the user can see the windows closing.
Would be better if I could switch to the open window instead of closing all those ahead of it, and causing a visual slide show for the user.

  • You cannot switch to a previous window without closing the current one. The navigation group does not work this way. You can use a scrollable view to switch views (back a forth) if this is what you're looking for.

    If the scrollable view does not fit your needs, because it moves, I have create a small component that can help you. You might need to tweak it a bit for your purposes, but it can serve as a starting point. Gist

    Also, if needed, a window in a navigation group or a tab group, can be closed and open without animation.
