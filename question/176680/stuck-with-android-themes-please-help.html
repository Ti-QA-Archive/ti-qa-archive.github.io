  • I hope that "titanium sdk 2.3.2" is a typo. If not, why are you using such an old SDK?

    Starting with SDK 3.3.0, all themes must be based on the AppCompat themes, which yours are not. That could be the problem. You should not use the tool-api-level tag (by default, the highest installed will be used, which is what you want). Also, do not modify the androidManifest.xml file. All customizations should be done in the tiapp.xml file. Compare your steps to what is described in our using Android Themes guide.
