MusicPlayer Query - Issues

Core.prototype.musicPlayer = function(index) {
    if (isAndroid) {
        var player = Ti.Media.createSound({
            allowBackground : true
        });
    } else {
        var player = Titanium.Media.systemMusicPlayer;
        /*
         * Query DB
         * 
         * Select all songs in the device, and queue them.
         */
        var query = {}
        query['title'] = {value:'', exact:false};
        var picked = Ti.Media.queryMusicLibrary(query);
        player.setQueue(picked);
        player.setRepeatMode(Titanium.Media.MUSIC_PLAYER_REPEAT_ALL);
        player.setShuffleMode(Titanium.Media.MUSIC_PLAYER_SHUFFLE_SONGS);
    }

    return player;
};

player = core.musicPlayer();
player.play();

Now this plays the first song in the list fine, but then repeats it again, and again.
I have checked the 'length' of the 'picked' result, and it says there are 195 songs, but it can only play the first song.

I have also tried player.skipToNext(), and this will not play anything.

AM I doing something wrong, or does it just not work??

SDK 2.1.4.GA
Mac OS X

  • This took me a good while to figure out, but finally figured this thing out!

    Change 

            player.setQueue(picked);

    To

            player.setQueue({items: picked});
