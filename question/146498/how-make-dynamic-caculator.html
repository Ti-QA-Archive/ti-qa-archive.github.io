    Hi

    These should get you started - I have created two functions; convert2Joules & convert2Fps.

    • convert2Joules, 2 parameters; weight in grammes and the FPS value
    function convert2Joules(g, fps) {
    var mps = fps * 0.3048;
    var kg = g / 1000;
    return 0.5 * kg * (mps * mps);
}
// usage example;
Ti.API.info('convert2Joules: ' + convert2Joules(0.2, 350));
// 1.1380622400000002
    • convert2Fps, 2 parameters; weight in grammes and the joules value
    function convert2Fps(g, j) {
    var kg = g / 1000;
    return (Math.sqrt(2 * j / kg) / 0.3048);
}
// usage example;
Ti.API.info('convert2Fps: ' + convert2Fps(0.25, 1.86));
// 400.20867931697916

    Returned values in both includes full precision although the calculations are usually rounded to different levels, in those case the usage becomes;

    Ti.API.info('convert2Joules: ' + convert2Joules(0.2, 350).toFixed(2)); // 1.14
Ti.API.info('convert2Fps: ' + convert2Fps(0.25, 1.86).toFixed(0)); // 400

    Enjoy
