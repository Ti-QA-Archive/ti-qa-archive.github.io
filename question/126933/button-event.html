  • Hi Bilel

    You must bear in mind that people like Trevor spend a lot of their own time in this Q&A helping people like you, and so are understandably-frustrated if some don't follow the guidelines in the Participating in the Q&A guide, and especially the Q&A Question Checklist.

    That said, and unusually, I think I can help with this question without all the information being present.

    The code below should do what you are trying to achieve. Also, it's worth running the KitchenSink button.js example, for various other ways that a button may be coded.

    var win = Titanium.UI.createWindow({
  backgroundColor:'#fff',
  layout:'vertical',
  exitOnClose:true
});

var button = Titanium.UI.createButton({
  top:50,
  width:200,
  height:60,
  title: 'Button',
  backgroundColor:'blue'
});
win.add(button);

var buttonToggle = true;

win.open();

button.addEventListener('click', function(e){
  Ti.API.info('*** Button Clicked ***');

  if(buttonToggle){
    e.source.backgroundColor = 'green';
  } else {
    e.source.backgroundColor = 'blue';
  }

  buttonToggle = !buttonToggle;
});

    Hope this helps! :)
