StatusBar color in tiapp.xml doesnt work? (iOS7, Ti 3.1.3)

Hello, I want the text in the Statusbar to be White instead of black.

I've tried to change it in the tiapp.xml several times, with all possible combinations.

DEFAULT, GRAY, GREY, OPAQUE_BLACK, LIGHT_CONTENT, TRANSLUCENT_BLACK.

None seems to change the color. What am I doing wrong?

    Make sure you're using the new way. Works for me.

    <ios>
    <plist>
        <dict>
            <key>UIStatusBarStyle</key><string>UIStatusBarStyleLightContent</string>
        </dict>
    </plist>
</ios>
