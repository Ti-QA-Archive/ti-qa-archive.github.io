"Add Row" row in editable tableview

I have an editable tableview with an Edit button in the window title bar that sets the table to edit mode when I click it. I would like it to also show one additional row at the end of the table that acts as an "Add New Row" function with a green + icon next to it, just like when I edit a contact in the Contacts app. I've seen other applications do this, but can't find any documentation on having the table present an "Add Row" row when going into edit mode.

I've attached an image of another app that does this. Is this possible in Titanium? Thanks!

  • Ended up extending the interface to add the ability to set an editable row cell style to UITableViewCellEditingStyleInsert. Had to edit the TiUITableView.m file to expose a new interface for tableViewRow, called styleInsert (boolean).

    Will put this in a blog posting at www.codexlabs.com and will submit to Appcelerator as a pull request for their next version. I'll update this post when we've got the blog post out there!

  • Contrust the 'add' row you need, then when you set the table as editable, do an appendRow

    Then when you don't need it, do a deleteRow

    That should do the trick.
