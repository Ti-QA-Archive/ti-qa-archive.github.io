  • The fix that I found:

    1. Go to /Users/<username>/Library/Application Support/iPhone Simulator
    2. mv 7.0 7.0.backup (just keep it with any name you wish)
    3. ln -s 7.0.3 7.0
    4. In Titanium Studio settings set Default Display for iOS platform = "Retina & Tall"
    5. Run your code in iOS 7.0.3 iOS Simulator

    That's it!
