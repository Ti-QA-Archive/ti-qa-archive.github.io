Setting up Android Emulator in OS X

I'm using OS X 10.8.5 and was able to install Android SDK through Titanium.

But when I'm trying to run my application, it's giving me error

###[ERROR] : Emulator exited with error: 1

###[ERROR] : PANIC: Could not open: Galaxy Nexus

###[ERROR] Application Installer abnormal process termination. Process exit value was 1

I went to Manage AVD's under Preferences and created new AVD and started it and the emulator showed up. But I'm not able able to run the application into the emulator.

Am I missing something here. Please help.

    Hi,
    I came across the similar error when I try to install the application on Galaxy Nexus emulator. To overcome it I opened the AVD manager from ../android-sdk/tools/android and created a new emulator with device selected as galaxy nexus and other properties.
    After creating, it gets updated in my device list on titanium studio and then I selected my created emulator and it worked fine.

    Hope it helps.

      I got this working like the following. May be this will help somebody else.
      I went to Manage AVD's under Preferences and created new AVD (you can create any available one in your sdk) and started it and then the emulator showed up. The select Run as - "sdk" under android emulators list. This will run the application in the created AVD.
