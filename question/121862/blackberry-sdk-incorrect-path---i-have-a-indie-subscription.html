BlackBerry SDK: Incorrect path - I have a Indie subscription

Hi everybody!

I have purchased a Indie subscription to get blackberry working on my titanium studio.

It takes like 3 lives to download the Blackberry SDK but at the end changes the link from "Get SDK" to "Update SDK", and says that i have all installed.

Then i go to the place where he "downloaded" the sdk and cant run the emulator and even in the Help -> About Titanium, appears the blackberry logo as installed plugin.

But I CAN'T build ANY app in blackberry, and the part of the Get SDK says that i have an incorrect path to the sdk and never activates the plugin right.

I haver search around the Q&A, follow the Help Desk guide and everything i can found without success, so the question its simple:

  • Check to see that the Blackberry enabled sdk is the most recent and you haven't auto updated to 1.7.1

    That caused me problems and deleting the 1.7.1 fixed it.
