Prevent click two or more buttons at the same time

Hi Everyone!!! How I can prevent a user from click two or more buttons at the same time (and their event listeners). Right now I'm having an issue with this, because each button (the event listeners) do some animation. But If more than one is clicked at the same time or press one and holding and the click another and hold it and then releasing both, it cause the same effect (the animation gets pretty weird and the buttons move, and the images also moves, everything gets misplaced; it's a mess). Any help will be appreciate!!!

  • you can fire an event to indicate you have started animation, then have all the other buttons disabled… on completion of the event, fire another event to enable the buttons

  • Use a flag variable, ex: isAnimating, inside every function event listeners use a piece of codes like that:

    if (isAnimating)
    return;
    else
    isAnimating = true;
