    • Hi

      I have done some digging and found some evidence to back up my belief that the iOS SDK is the upper limit and not the lower limit - meaning you can compile using iOS 6.0 and you can still use that to install to a device as low as 4.3, it says this in the console when you build the app.

      • [INFO] : Building using iOS SDK: 6.0
      • [INFO] : Building for iOS iPhone Simulator: 6.0
      • [INFO] : Building for device family: iphone
      • [INFO] : Building for iOS 6.0; using 4.3 as minimum iOS version

      A comment by Chris Barber @ Appcelerator

      "Also, there was a bug in older versions of Titanium Studio where the iOS SDK dropdown would incorrectly list all the iOS Simulators, not the actual installed iOS SDKs. That means if you have iOS 6 SDK installed with iOS 5.1 and 6.0 Simulators installed, it would allow you to pick 5.1 or 6.0. Picking 5.1 would then tell the build scripts to use 5.1, but the build script wouldn't find 5.1 and would actually just use 6.0 anyways. Crazy, I know." Chris Barber

      Just make sure to not include any 6.0 specific features of course.
