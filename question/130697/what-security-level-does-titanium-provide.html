What security level does Titanium provide

I want to develop an application on Titanium using cross platform so what security level does it provide, because if my application got decrypted or can someone de-compile my code since it is a web application converting code into native app, and how can I decompile the code to further use it as a native application for iOS and Android.

  • Hello,

    It depends. Basically if your app is having a web app embedded in a webview in Ti, you will get as much security as the offered by your server. If you are creating a Ti app with webservices, there is several factors here: your webservice, if you use ssl or not, etc. In the case of the app, your code is converted into a native app, with the same level of security that any other native app.

    Additionally, you are able to generate obfuscated javascript code before passing thru the Ti compilation, for extended security.
