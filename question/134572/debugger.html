If we are using Windows then…
is it possible debugging line by line in appcelerator if yes, then please help me…….

  • Yes, you can start a "debug" session by clicking the "bug" icon above the application explorer. You can then set breakpoints in the debug perspective by clicking on the border to the left of the line at which you want to set a breakpoint. Then, you can single step using the options in debug perspective.
