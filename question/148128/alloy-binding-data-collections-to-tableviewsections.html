Alloy: Binding data collections to TableViewSections

I have a TableView in XML which I want to contain 3 distinct sections. Each of those sections are broken up using TableViewSections. I would like to bind a different collection to each of the view sections. Is this possible?

Here is what I tried:

<Collection src="patient" />

<TableView id="patientsTable" width="100%" height="100%" style="1">
    <TableViewSection headerTitle="All Patients" id="allPatientsSection" dataCollection="patient">
        <TableViewRow title="{first_name}" />    
    </TableViewSection>
</TableView>

Given this didn't work, I'm guessing that dataCollection cannot be applied to a TableViewSection. Is there any other way to bind a collection to a view section instead of the entire table?

Thanks.
