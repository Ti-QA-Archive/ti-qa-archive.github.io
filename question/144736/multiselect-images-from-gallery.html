Multiselect images from gallery

Hi, I'm trying to make a multiselection of images from the photo gallery. If I open the gallery app, the menu button shows some options (capture pictures, multiselect, settings…), but if I open the gallery with my app using Titanium.Media.openPhotoGallery, that menu only lets me to capture picture. How can I open that option menu? Or there's a better way to do it (multiselect of images)?

Thank you.

    The UIImagePickerController that Titanium.Media.openPhotoGallery is built upon does not have the ability to select multiple images. For this to work you'd need to build your own picker module using the Asset Libraries API.
