    Welcome to the confusing world of Andoid!

    Your first question about the 'errors'. These are log messages from the android OS itself. They are completely normal and should be ignored. Android tends to be very verbose when logging. The only ones that should concern you usually start with TiActivity (or anything starting Ti….), those represent errors within your app.

    Regards API version. You need to make a decision about which Android devices you want to support. Currently Titanium only supports up to 4.0 (so if you choose 4.2.2 then it may not work correctly). The main thing to know about Android is that the vast majority of devices out there were released with much earlier versions of the API. You will massively limit the amount of devices that can install your app if you dont target those earlier versions of the API. The API is (supposedly) backwards compatible so a device running 4.2.2 can also run apps written for (eg 4.0). I would suggest you target the lowest Appcelerator supported API, which is currently 2.3

    If you are going to include a MapView in your app you need to alway pick the API that says (Google) next to it. That will include the neccessary libraries to support Maps.
