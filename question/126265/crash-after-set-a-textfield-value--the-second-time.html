Crash after set a textfield value ... the second time.

I have a strange behavior. I assign a value to a textfield, this time works good. But in another part of the code, in the same .js, i assign a new value to the same textfield, and the program crash.

Example:

var dblVariable;
dblVariable = 5.5;

txtTestTextField.value = dblVariable;
...
// in another part
dblVariable = 8.5;
txtTestTextField.value = dblVariable;

  • The non-visibility is more than likely the issue. You're casting a variable to an instance that probably doesn't exist in the current scope. You might try keeping a properties or global variable reference that can be applied to your text area upon focus of the window in which it exists.
