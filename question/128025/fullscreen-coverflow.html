Fullscreen coverflow

Hello all,

Does anybody know about any workaround to make a coverflow object look like a fullscreen slideshow?

Increasing coverflow width/height does not enlarge it enough to show fullscreen images… and if I load larger images into a coverflow, it will crop them…

Thank you for any help you can provide,

Luis

    The coverflow will be the appropriate height and width if you don't specify values for them. Also, you should set your containing window to be fullscreen as well. Here's a simple test. Your images should likely then be scaled/cropped to fit the dimensions of your screen if that is your aim.

    var win = Ti.UI.createWindow({
    backgroundColor: '#fff',
    navBarHidden: true,
    fullscreen: true
});
var coverflow = Ti.UI.createCoverFlowView({
    backgroundColor: '#fff',
    images: [ /* your images */ ]
});
win.add(coverflow);
win.open();

    One note is that if you are aiming to cover the whole screen with your images, perhaps you might consider the ScrollableView component as well.
