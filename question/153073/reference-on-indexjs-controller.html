Reference on index.js controller

Is there a way to get a reference to the index.js controller from Alloy?

Something like "this" in Java.

public class ClassName{
    public ClassName getReference(){
        return this;
    }
}

    • No, it's not wrong. The problem is that the $ is not fully populated with functions until after its controller is executed, meaning after any code that you put in it. So when you reference arg1 in the test controller, $ does not yet have the assignment from exports.scream. The easiest thing to do would be to assign your scream function directly to $:

      $.scream = function(){
    alert('yessss');
};
