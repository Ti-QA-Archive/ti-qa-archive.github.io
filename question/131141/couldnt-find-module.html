Couldn't find module

Hi-

I am launching an app on a new machine that I just put Titanium on. However, I get the "Script Error = Couldn't find module:" error. I've tried cleaning out my build with a fresh one, restarting the studio, etc to no avail.

Yes, I have the modules in the root Library directory:

/Library/Application Support/Titanium/modules/iphone/

and I have my tiapp.xml file looking like this, with the proper version:

<modules>
<module version=".1">ti.flurry</module>
</modules>

It was working on the old system. One other note, and I think this is important, is that I had no titanium folder in the Application Support folder, and I had to manually create it.

Any help would be appreciated.

  • You are missing the platform attribute.

    <modules>
        <module platform="iphone" version="0.1">ti.flurry</module>
        <module platform="android" version="0.1">ti.flurry</module>
    </modules>

    Also, try putting the module's dist zip in the project root instead of in the Library directory. On build, it should remove the zip and create instead a folder for modules/platform/modulename/version.
