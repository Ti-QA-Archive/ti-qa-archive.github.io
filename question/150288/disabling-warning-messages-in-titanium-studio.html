Disabling warning messages in Titanium Studio

Hi, I trying to develop an application for iPhone using Titanium Studio. I have a table within a table's row.
I have set the scrollable property of the parent table to false. While scrolling the inner table at specific point of time, I want to change the top of the parent table.

Below is the code example:

      $.innerTable.addEventListener('scroll',function(e)
      {
         var offset = e.contentOffset.y;
         if($.parentTable.top <= 100 && $.parentTable.top >= 0)
         {
            var newTop = $.parentTable.top - offset; 
            if(newTop > 100)
                   {
              newTop = 100;
            }
            if(newTop < 0)
            {
               newTop = 0;
            }

               $.parentTable.animate({top: newTop, duration: 50});
        }
       }

As I'm animating the parent table inside the scroll function of inner table, I'm getting the following warning on each pixel of scroll:

 [WARN] :   New layout set while view [object parentTable] animating: Will relayout after animation.

Which makes the scroll flicker. Can any one tell me if it is possible to disable the warning logs in Titanium Studio. Or is there any other way to implement the above functionality. Thanks for your help in advance.
