  • Hi Tony. I solved it. In one of the file I include I've created a json object like this:

    var obj = {
    Name: txtName.value,
    Phone: txtPhone.value
};

    I changed it to:

    var obj = { };
obj.Name = txtName.value;
obj.Phone = txtPhone.value;

    and it seems to work and makes the error disappear.
