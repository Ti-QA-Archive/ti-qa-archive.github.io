  • Hi Wei,

    You need to create a default user in Cloud Console, and then login from the app first to use any Cloud service

    After creating the the user put this code in your app and insert your Custom object creation code in a function(createobject() in the below code)

    Cloud.Users.login({
        login: 'user',
        password: 'password'
    }, function (e) {
        if (e.success) {
            alert("login success");
            createobject();   //Insert the Custom Object creation code in this function            
        } else {
            alert('Error: ' +((e.error && e.message) || JSON.stringify(e)));
            }
        });
