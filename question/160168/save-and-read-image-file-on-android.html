Save and read image file on Android

Am trying to read an image already saved en Android (in iOS my functions works fine), but when am trying to read the image, Android says "Error, <FILE Object> has no method .toBlob()"

This is the way am saving the image:

function saveDataFile(content) {
    var fileName = "someImage.jpg"; //Am making a random name really...
    var f = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory, fileName);
    imageLocation = Titanium.Filesystem.applicationDataDirectory + fileName;
    f.write(content);
    f = null;
    return fileName;
}

And am reading it on this way:

function readDataFile(fileName) {
   var f = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory, fileName);
   return f.toBlob();
}

The image is saving well, but I cant read it in my readDataFile function. I've already try returning the image like: f.toImage() and f.media.

What am doing wrong? Or, how is the right way to save and read an image file on Android?
Am using Ti SDK 3.1.3.GA on a Mac. My Android phone for test has Android 2.3.6

  • function readDataFile(fileName) {
   var f = Ti.Filesystem.getFile(Ti.Filesystem.applicationDataDirectory, fileName);
   return f.read();
}

    Titanium.Filesystem.File#read() : Titanium.Blob

    Returns the contents of the file identified by this file object as a Blob.
