Number Keypad not working

Hi,

In our Android app, we have a requirement for a Number keyboard to open up on click of a textfield, we used the snippet below. 

txtField = Titanium.UI.createTextField({
               color:'#336699',
            top:16,
            height:40,
            keyboardType:Titanium.UI.KEYBOARD_NUMBERS_PUNCTUATION,
             passwordMask:true,
            returnKeyType:Titanium.UI.RETURNKEY_DONE,
            borderStyle:Titanium.UI.INPUT_BORDERSTYLE_ROUNDED,
        });

Alpha-numeric keyboard shows up when we add the above textfield. But number keyboard opens when the passwordMask property is removed.
Is there any other keyboard type that shows up Number pad, even with the passwordMask set as true?

We are using : Android 2.2, Ti SDK - 1.7.2
