  • I've not seen a label interpreting html before, so I'd be really interested to see some example code. Without this, I imagine I would have done something like the following:

    var regex1 = /<br\s+\/>/gi;
var regex2 = /<(p|\/p)>/gi;
var str = "<p>This is the first paragraph.</p><p>This is the second paragraph,<br />followed by the next bit of the sentence.</p><p>And this is the last paragraph</p>";
var newstr = str.replace(regex1, "\n").replace(regex2, "\n\n");
Ti.API.info(newstr);

    or, alternatively, for the label:

    thisLabel.text = thisLabel.text.replace(regex1, "\n").replace(regex2, "\n\n");
