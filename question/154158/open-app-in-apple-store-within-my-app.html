  • Why don't you want to use Ti.Platform.openURL()? That's the way you'd link to an app in the store.

    // for example, when a button named 'btn' is clicked:
btn.addEventListener('click', function() {
  Ti.Platform.openURL('http://itunes.apple.com/us/app/YOURAPPURL?mt=8');
}

    Get the proper link from the http://itunes.apple.com/linkmaker/ site.
