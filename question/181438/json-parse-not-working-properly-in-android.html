  • Hi,

    If you are using GET method, you can try your API url with http://jsonlint.com/. If this website cannot parse your data, meaning your data absolutely have trouble because of your server returning incorrect data format.

    If you are using POST method, you can use Titanium code: Ti.API.error(JSON.stringify(your_data)); to see the result of your data in console, then validate the result with http://jsonlint.com/

    Hope it help!

  • Call JSON.parse(xhr.responseText); when xhr.responseText is an invalid
    JSON the app might crash without any error.

    The app even crash when JSON.parse is surrounded with try/catch

    Before call JSON.parse validate JSON,
    I suggest to use regular expression:

                    if (/^[\],:{}\s]*$/.test(xhr.responseText.replace(/\\["\\\/bfnrtu]/g, '@').
                    replace(/"[^"\\\n\r]*"|true|false|null|-?\d+(?:\.\d*)?(?:[eE][+\-]?\d+)?/g, ']').
                    replace(/(?:^|:|,)(?:\s*\[)+/g, ''))) {

                      responseJSON = JSON.parse(xhr.responseText);
                }
