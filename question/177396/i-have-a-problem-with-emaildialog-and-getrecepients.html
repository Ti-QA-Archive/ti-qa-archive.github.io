I have a problem with emailDialog and getRecepients

Hi, i want to send mail to some emails adress retreived from remote database. Here is my code :

var emailDialog = Ti.UI.createEmailDialog();
        emailDialog.subject = "Confirmation";
        for (var i = 0; i < json.length; i++) {
        emailDialog.toRecipients = [json[i].mail];
        };
        emailDialog.messageBody = "About Confimation";    
        emailDialog.open();
        };

when i run my app only one email adress will be on the recepient zone. I tried to change the json[i].mail with this code :

var emailDialog = Ti.UI.createEmailDialog();

emailDialog.toRecipients =recipient;

emailDialog.open();
