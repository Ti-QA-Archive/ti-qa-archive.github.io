Open modal in single context

Hello,

I am developing an iOS app with Titanium Studio and SDK 1.7.2.
I am working in a single context way of doing things (like tweetanium).
I know how to switch between windows but how can I open a modal window?

This kind of works but I cannot add a button to the navBar. It also
does not feel right at all.

 var wind = UI.Win({

  title: 'Twitter',
  modal: true

 });

 wind.open(Window.ShareTwitter());

Thankful for all input!

  • in the same way tweetanium does, create a view with all the window content and add to the window before open.

    The view will be in the same execution context, but note that you can't interact with other windows from a modal win.

    var wind = UI.Win({

  title: 'Twitter',
  modal: true

 });

 var _view = UI.createYourModalView();
 wind.add(_view);
 wind.open();
