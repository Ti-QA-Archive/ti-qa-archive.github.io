  • Hello,

    you need to use transport.py, which is useful to share your xcode project with people without having Titanium.

    Steeps to do this:

    • In your Titanium machine, build your project to run in the simulator. This will generate a build directory, which include an iphone directory with your xcode project.
    • Open a terminal and go to your project's directory (where build directory is).
    • Then you run something like this:
    /Users/mauroparra-miranda/Library/Application\ Support/Titanium/mobilesdk/osx/3.1.1.GA/iphone/transport.py . 3.1.1.GA

    where . is the current directory in the terminal, and 3.1.1.GA is the SDK that you are using. Please adapt according to your username, Mobile SDK version, and so.

    • Then you can just zip your build directory and share it with your friend or coworker. The project will be able to run in any xcode, even if they don't have Titanium installed.

    Best,

    Mauro
