Close nl.fokkezb.drawer window on Android

Hi All, I'm using nl.fokkezb.drawer widget both on iOS and on Android. I works good, but I have one problem on Android. My app has drawer logic (left slide menu) on some of inner screens, not on the first screens with which the app starts. So I start the app, goes some screens forward, and open 'base' screen with slide menu. Then I can open next screens and close them. But I cannot close drawer screen when I want to go to first screen. I need to close it programmatically via javascript. I Use the drawer like this:

<Alloy>
<Widget id="drawer" src="nl.fokkezb.drawer">
<Require src="slideMenu"></Require>
<Window module="xp.ui" platform="android" role="centerWindow">
<View id="viewContainer">
</View>
</Window>
</Widget>

</Alloy>

So, how can I close the drawer window using javascript on Android? Thanks.

  • I finally closed the drawer widget on Android. I added close function to the widget like:

    exports.close = function() {

    $.instance.close();

    };

    So it became available to close via calling $.drawer.close();
