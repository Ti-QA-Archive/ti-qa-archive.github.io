Remove TabGroup gloss effect in iOS

Is there a way to remove the "gloss" overlay effect that appears over a TabGroup in iOS?

tabsBackgroundColor -> changes color, but gloss still there

  • Hi

    There are no exposed properties to achieve your goal, however there are many custom modules that provide this functionality.

    • TiCustomTab

    Extends the existing tabGroup to include many customisation features which include the select and unselected state images, tab background and font changes. Worth taking a look.
