Button doesn't take backgroundGradient colors

I have this code making a button:

var menuBtn = Titanium.UI.createButton({
        image:'images/menuIcon.png',
        backgroundGradient:{
            type:'linear',
            startPoint:{x:'0%',y:'50%'},
            endPoint:{x:'100%',y:'50%'},
            colors:[{color:'#8ca61b',offset:0.0},{color:'#9bb81e',offset:1}]
        }
    });

I'm exporting to iOS but it doesn't take any of the backgroundGradient properties. It compiles fine, but the buttons still show the default blue colors.

Any idea whats wrong?

  • On iOS, there's a bunch of default button decoration that you can suppress. Setting backgroundImage to null is one way–the other way is to set the the button style to PLAIN:

    style: Ti.UI.iPhone.SystemButtonStyle.PLAIN,

    See the section "iOS Platform Notes" in the Ti.UI.Button API reference for more information on button styling.

    With the style set to PLAIN, you won't have any border drawn by default and won't radius the corners, so if you want rounded corners and/or borders like the standard iOS buttons, you'll need to add that yourself:

    borderRadius: 10,
borderColor: '#000'

    Hope this helps.
