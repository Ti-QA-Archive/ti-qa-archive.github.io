  • Howdy! We are aware of the issue and I have fixed it, but I still need to get it tested and merged into master tomorrow.

    Titanium SDK 3.2 and newer will REQUIRE Titanium CLI 3.2 or newer. You will not be able to use Titanium CLI 3.1.x to build with Titanium SDK 3.2. To get the bleeding edge (which is NOT stable and has bugs, but you can build with SDK 3.2), then run:

    sudo rm -rf ~/.npm
sudo npm install -g git://github.com/appcelerator/titanium.git

    If you using a Titanium SDK master CI build, then you'll be good to go after my upcoming fix.

    If you are building Titanium SDK from source, then you need to specify the new scons parameter "node-appc-branch" like this:

    scons node-appc-branch=master
