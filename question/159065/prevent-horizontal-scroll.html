  • Hi

    This looks like a typo as you have provided the wrong spelling and type upper/lower case of the property. Also the property does NOT require quotes unless part of alloy and only then if part of the view.

    // change this
showhorizentalscrollindicator="false"
// to this (if alloy view)
showHorizontalScrollIndicator="false"

    In your code this would be shown as;

    // classic with no object
showHorizontalScrollIndicator: false
// classic with object example
var scrollView = Ti.UI.createScrollView({
    showHorizontalScrollIndicator: false
});
// classic with property after creation
scrollView.showHorizontalScrollIndicator = false;
// alloy tss
"ScrollView" : {
    showHorizontalScrollIndicator: false
}
// alloy property
<ScrollView id="scrollView" showHorizontalScrollIndicator="true" />
