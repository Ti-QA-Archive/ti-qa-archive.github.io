Titanium.App.exit() is undefined.

Hi,

I found Titanium.App.exit() in the API, but when calling this function in my application, all I get is the following error: 

Result of expression 'Titanium.App.exit' [undefined] is not a function.

Why doesn't this work? I simply want to exit the iOs app. Ti version 1.7.5.
