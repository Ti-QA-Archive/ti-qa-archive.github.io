  • Hi, lets say that win is the id of the window you want to open. So in the controller you write something like this:

    $.win.left = 800;
var slide_it_left = Titanium.UI.createAnimation();
slide_it_left.left = 0;
slide_it_left.duration = 500;
$.win.fullscreen = true;
$.win.open(slide_it_left);

    This should open your Window sliding.
