Hi
I'm using SDK 3.1
I'm trying to fire an event in a webview and detected it at app level. I've followed all the post I have found on this subject but I cannot make it work.

I need to attach an event listener to all images, which are generated from the app in the web view.
my problem is that at the following code lines

        window.onload = function() {
            var images = document.images;

            for (var i = 0; i < images.length; ++i) {
                images[i].addEventListener('click', function(e) {
                    Ti.App.fireEvent('openMenuImage');
                    alert('cliccato');

If I leave in there the line Ti.App.fireEvent('openMenuImage') my click stops working and it does not fire any event. Also the alert do not work with the Ti.App.fireEvent('openMenuImage') line in there.

Actually not events are fired, neither at webview level or app…

Here is the complete code. I have also tried to remove jquery and iosslide library and the problem persist…

Thank you
