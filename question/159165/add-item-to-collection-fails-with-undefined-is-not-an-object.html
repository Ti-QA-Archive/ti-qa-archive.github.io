Add item to collection fails with undefined is not an object

when i do this: myScannedItems.add(scanItem);
he failed with "undefined is not an object" - what am i doing wrong?

Ti.App.addEventListener('updatePoints', function(data) 
{
    var myScannedItems = Alloy.Collections.ScannedItems;
    var scanItem = Alloy.createModel('scannedItems', {
        barcode : data.code,
        points : data.pointsDelta
    });
    myScannedItems.add(scanItem);
    myScannedItems.save();

    showMessageTimeout("+" + data.pointsDelta +" Punkte. <br>Du hast jetzt insgesamt: " + data.pointsValue + " Punkte :-)",1500);
 });

    You have to replace this part

    myScannedItems.add(scanItem);
myScannedItems.save();

    with

    scanItem.save();

    I assume, you have defined your collection as

    Alloy.Collections.ScannedItems = Alloy.createCollection(&#39;scannedItems&#39;); in alloy.js where scannedItems is the collection_name of your model
