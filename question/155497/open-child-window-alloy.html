Open Child Window Alloy

Hi, This is probably an easy one but I just can't figure it out, I have an app which has a tabgroup with a number of tabs, within one of the tabs I have a tableview with an onclick event.

One of these is called 'Settings', when you click on the settings i want it to completely replace the current window with settings, but the android back button should still go back to the table view on the tab window.
When I try it now, the window is opened on top of the current window but the back button closes the app ccompletely, here is my code:

— list.xml —

<Alloy>
    <Window>
        <TableView id="table">
            <TableViewSection backgroundColor='#F00' headerTitle="Options" onClick="eachItem">
                <TableViewRow>
                    <Label class="title" text="Settings"/>
                    <Label class="subtitle" text="These are the settings for teh application" />
                </TableViewRow>
            </TableViewSection>
     </Window>
</Alloy>

— list.js —

var eachItem = function(e) {
    debug(e);
    var settingsWindow= Alloy.createController('settings').getView();
    settingsWindow.open();    
};

— settings.xml —

<Alloy>
    <Window>
        <Label text="This is some text" />        
    </Window>    
</Alloy>

What am I doing wrong?

Thanks

    Hi,

    you need to open it with modal property.

    settingsWindow.open({
    modal : true
});

    because back button only supports heavy weight window to close.
