Set an iOS table view to only the height/number of rows of its data?

I have a table view with one row at the start, with a button for the user to append rows. I want the view to only have one row at the start, and get "taller" every time a row is added, pushing the button below it down it the process. Right now, the table view is the height of the enclosing view, with empty rows.

SDK 2.01, iOS 5.1
