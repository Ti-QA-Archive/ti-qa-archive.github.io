NodeJS third party module usage

Hi,

I have downloaded braintree nodejs module npm install braintree for my app but when I make a call to it using the following statement var braintree = require(&#39;braintree&#39;) it gives me Couldn't find module error.

Can anyone help me out on how to use third party node modules in our app?

  • Accepted Answer

    node.js is an entirely different Javascript engine than Titanium. In most cases you can't just start using node.js modules in Titanium apps. There are some node.js modules that also have browser-based versions, which often work with Titanium. You can also try using browserify to turn a node.js module into something that might work with Titanium. Additionally, but more experimentally, titaniumifier tries to solve this problem in a more Titaium-specific way.

    All of this is contingent on the type of modules and dependencies the node.js module in question uses. The simpler the module, the more likely it is to work with Titanium with the help of one of the above methods.
