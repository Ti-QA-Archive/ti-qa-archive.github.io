  • I've honestly not had too much success using the in-built forward geocoder functionality. I generally just access Google's API directly:

    http:&#x2F;&#x2F;maps.googleapis.com&#x2F;maps&#x2F;geo?output=json&amp;q=Lisbonne

    You can do this in an XHR request. It returns JSON, so it's easy to pass right into your callback function.
