Licensing files

Almost ready to publish my first paid-for App!

Question: Is there a standard license file I can use to populate the license.txt file or that at least I can send to my client, so he can in turn, send it to his lawyer?

There is also a License file (no extension) that has Appcelerator and Apple legalese. Should I leave this file in the App?

Thanks to all those who have helped!

    The license file is for the code, not for the app. So unless you are giving the code away or making it public, then the license should be something specific between you and your client. This is usually agreed upon before coding on the app even begins. If you're just looking for some boilerplate stuff, then swing by LegalZoom or similar place and grab some template text that you throw in there. If you are sharing the code, then you can just grab standard text from the license you plan to use. MIT, Apache 2.0, GPL, etc. Check out this entry for more info: http://en.wikipedia.org/wiki/Software_license
