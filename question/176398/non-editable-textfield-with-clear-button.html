    You have told it that it is not editable, so clearing the text is editing the text - so it will not work. You will need to create a button of your own and set the value to and empty string.

    • This is the problem; your description of a text field is no user entry, no OS clipboard access - just a cancel button. The only thing that will set a value are your own buttons not directly linked to the text field.

      So what you are talking about is a label. But for logical reasons your UI will likely want a consistent style. However that does not mean that a text field used as a label can gain things from the text field when not asked to be used as a text field. That become complicated.

      So; no.
