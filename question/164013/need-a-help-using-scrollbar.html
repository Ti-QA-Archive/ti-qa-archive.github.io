  • Hi

    If this is for a ScrollableView then you want to show the paging control. This can be defined in any of three places:

    XML

    <ScrollableView id="example" showPagingControl="true">
...
</ScrollableView>

    JS

    $.example.setShowPagingControl(true);

    TSS

    "#example": {
    showPagingControl: true
}

    This will show (X) page indicators for (X) views you have inside of your ScrollableView.

    If this is for a scrollview, then in the same way as before you would alter showHorizontalScrollIndicator and showVerticalScrollIndicator to suit your needs.

    Hope this helps!
