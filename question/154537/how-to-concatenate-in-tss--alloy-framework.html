    Hi,

    you can not concatenate like this but you can declare global variable in alloy.js file and then you can use it in.tss

    alloy.js

    Alloy.Globals.fbUser =  'https://graph.facebook.com/'+ Ti.Facebook.uid+ '/picture?type=square';

    .tss

    image : Alloy.Globals.fbUser

    or else you can set that image in your .js file also without setting any global value.
