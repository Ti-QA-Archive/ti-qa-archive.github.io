Android Menu Item icon is not displayed

I follow the example on the API page to create a menu for my Android app.

It works fine but the icon in the menu item is not displayed at all.
Here is my code:

activity.onCreateOptionsMenu = function(e){
    var menu = e.menu;
    var home = menu.add({title: "Home", icon: '/images/home.png'});
    home.addEventListener("click", function(e) {
        controller.home();
    });
};
