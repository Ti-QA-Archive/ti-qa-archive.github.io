Integrate appcelerator into my current app(ios)

Can anyone tell me the best way to or if it's even possible to integrate appcelerator into an existing ios app? I was interested in creating some web forms via titanium

  • Titanium isn't a library that you integrate into an existing application. It is a platform on which you would build your application. Moreover, applications built on Titanium are NOT web apps, they are fully native apps. If you are looking to do web forms then I would suggest using a web framework.
