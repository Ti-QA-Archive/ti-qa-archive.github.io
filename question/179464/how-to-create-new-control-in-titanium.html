For example, creating a simple Icon control, derived from Image (or wrap image or a view), adds little extra functionality.

  • Hi,

    you can create using controller same like we create window
    e.g

    cutomeRow.xml

    <Alloy>
    <TableViewRow id="row">
        <ImageView id="imgCheck"></ImageView>
        <View id="imgCheckBackView" onClick="AddToFavourite"> </View>
        <ImageView id="imgCategory"></ImageView>
        <Label id="lblTitle"></Label>        
        <ImageView id="imgArrow"></ImageView>
    </TableViewRow>
</Alloy>

    cutomeRow.tss

    "#row" :{
    width:"100%",
    height:80,
    backgroundImage : Alloy.Globals.path.imgNewsRowBg,
    selectionStyle : 0
}
.....

    cutomeRow.js
