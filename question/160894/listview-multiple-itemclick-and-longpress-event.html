  • It is way easier than it looks like.

    The fact is that you cannot set the events on the ListView or on the ListItem(Data). Events can be set in the template as follows:

    var listView = Ti.UI.createListView({
  // …
  templates: {
    "simple": {
      // …
      events: {
        "longpress": function (event) {
          Ti.API.warn(event.itemId); // Works! ;)
        }
      }
    }
  }
});
