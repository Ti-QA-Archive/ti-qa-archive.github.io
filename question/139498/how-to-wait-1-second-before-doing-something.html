function x(){    
    //  < pause for 1 second
    alert("waited 1 second before alerting!");
}

Is this possible??

    Hi Donal,

    Try this way,

    
function x(){   
    //  < pause for 1 second
    setTimeOut(function(evt){
//=== do your stuff here
    alert("waited 1 second before alerting!");
    },1000);

}
