    Hi Shubham Jain.

    The rhino was the JS engine used a REALLY long time ago now. The new engine is V8. Somewhere, your old project is trying to use support for rhino that no longer exists in the current SDKs (unless there is a hidden flag laying around).

    Check out the link for details: Getting an error while building an Android Module

    Hope this will solve your problem.
