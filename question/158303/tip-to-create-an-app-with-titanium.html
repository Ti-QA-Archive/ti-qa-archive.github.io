Tip to create an App with Titanium

Hi all.

This is only question to select Titanium for the project or not.

I want to create a mobile app that can receive push notifications. This client is so easy: only a list of received notifications, and the posibility to receive more.

The server will be a simple web page for a user, where he will have the sent message and also can write a new message to broadcast to all the clients.

Just this, it's a simply concept. So I suppose it can be done with Titanium technology, using ACS for clients (mobile apps) and Node.ACS for the web server. It's right? Can I have a little database in Node.ACS to store the messages or I need another approach?

    Regarding Node.ACS, you can have your own database if you so desire that is not the problem.

    The only caveat that I foresee with such an application is that your app will simply display a webpage from the server. Apple has rules against that sort of app that basically "embed" a website. Meaning that your application can be refused publication on the app store.

    The best way to avoid this is to build an application that brings added value to a service. Remember, application are not the web.
