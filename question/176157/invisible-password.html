  • Hi

    If you just want to show a password field without the mask characters - then create a textfield and do not set passwordMask to true.

    If you mean can the user be given the choice to view the password they are typing in or not then yes, for Android just toggle the passwordMask property when the user selects a choice you add. For iOS you create two fields, place own over the top of the other, only one has the passwordMask set. Add a button to toggle between them. Then use an event listener to set the other one to the value of the first. This way whichever one they are typing into sets the other one.

    If you are hoping your app can somehow access other apps passwords as they are being typed - not helping you solve that.

    If you mean something else - more details needed.
