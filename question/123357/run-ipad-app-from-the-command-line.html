run iPad App from the command line

Hi,

I'm still figuring out what the ideal workflow is to build mobile apps (iphone, ipad and android) on a Mac using Titanium.

I just started using Titanium and downloaded Titanium Studio. That works great, but I really like to use TextMate as it is my favorite editor.

I managed to use the command-line to run the project (using titanium run –platform-ihpone), but cannot run the ipad version.

Is this the way to go? Then how can I run the iPad simulator from the command-line? If not, then what is a good workflow using TextMate (or any other editor) to test the apps?

Thanks!

  • This is a crude approach, but it seems to work: in /Library/Application Support/Titanium/mobilesdk/osx/1.7.3/iphone/builder.py, on line 410, change

    devicefamily = 'iphone'

    to

    devicefamily = 'ipad'
