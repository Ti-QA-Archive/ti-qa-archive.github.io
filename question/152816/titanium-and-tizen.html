  • I also faced same issue. I tried 2 solutions that worked for me. Please try these:

    1. Delete Tizen emulator instance from emulator manager and create new instance. Now Install the app.

       OR

    2. 2.1 Repeat step 1.
      2.2 Remove all tizen specific preferences from titanium studio.
      2.3 Delete certificate file from Tizen-SDK folder and create new one.
      2.4 Now Give all settings in titanium studio.
      2.5 Launch emulator
      2.6 Install application

    But after 3 or 4 run, again same issue appears. So repeat it. It is very inconsistent behavior.
