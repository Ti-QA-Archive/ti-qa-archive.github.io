Screenshot to gallery in Android

Hi there.
I have read some post about the topic, where it seemed that you could use toBlob() instead of toImage, in android, but i haven´t figured it out.
Here my thing, I have made a view into a scrollview and it can take screenshot of the whole screen (included the scrollview) and it works fine on IOS, but it won´t take any screenshot, when im testing it on Android.
Is there a solution to this, or do you have to do this in another way?

here´s some code.

var butnToImage = Ti.UI.createButton({
    title:'Gem CV',
    width: 100,
    right: 10,
    height: 34,
    top: 3
});
footer.add(butnToImage);

butnToImage.addEventListener("click", function(e){
    //gem footer
    footer.visible = false;

    var gemtImage = Ti.Media.saveToPhotoGallery(view.toImage());

            skrivFile = Titanium.Filesystem.getFile(
            Titanium.Filesystem.applicationDataDirectory,
            'mitcv.jpg');
        skrivFile.write(gemtImage);

});

      var gemtImage = view.toImage();
      var f = Ti.Filesystem.getFile(Ti.Filesystem.externalStorageDirectory, 'test.png'); f.write(gemtImage.media); Ti.Media.Android.scanMediaFiles([f.nativePath], null, function(e){});
