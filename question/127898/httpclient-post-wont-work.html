httpClient Post won't work!!!

Hi guys, I'm trying to make a simple app send some values through post to a php file I have on my local web server. Then that php script will insert into a mySQL database, but looks like I have something wrong on my application's code cause it gives me a 404 error code. Here is the code:

    var xhr = Ti.Network.createHTTPClient();
            xhr.onload = function(){ alert(xhr.responseText); }
            xhr.open('POST', 'http://localhost/test/submitdata.php');

            var data_send = {
              name:'data1',
              password:'data2'
            }
            xhr.send(data_send);

Can someone help please!

  • Unless you are running PHP on your device, which I seriously doubt, you need to give HTTPClient the IP address or hostname of the server that is actually running the webserver with PHP. This remains the case even if you are using the Android emulator or iOS simulator.
