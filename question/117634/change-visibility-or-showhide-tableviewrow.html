    • It appears that show/hide and the visible property don't do what I want, but changing the height property does. Aaron, you're right. Here is an example that show/hides all 'myclass' rows:

      if (showDates) {
    for (var i = 0; i < theTable.data[0].rowCount; i++) {
        var row = theTable.data[0].rows[i];
        if (row.className=='myclass') {
            row.height = 20;
        }
    }
} else {
    for (var i = 0; i < theTable.data[0].rowCount; i++) {
        var row = theTable.data[0].rows[i];
        if (row.className=='myclass') {
            row.height = 0;
        }
    }
}

      Note that I'm able to use theTable.data[0] since I only have one section in the table.
