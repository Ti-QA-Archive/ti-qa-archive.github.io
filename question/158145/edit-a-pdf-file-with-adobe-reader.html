Edit a PDF file with Adobe Reader

In my app the users can download files from a server and I want to know whether there is a way to open and edit a downloaded PDF file with Adobe Reader. With Adobe Reader you can add personal notices which are stored in the PDF. Can my app access the PDF with the personal notices? This is imortant, because when the user opens the same PDF again the personal notices should be visible.

  • you can use the PSPDFKit module. There use to be a module on the marketplace, but I think you have to buy the binaries from his website now. That's the only module I know that let you highlight and add personal notes. They'll be saved inside the PDF, so every time the user will open the PDF, he will find the same notes.
