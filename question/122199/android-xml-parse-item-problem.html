Android XML Parse Item problem

Hi all,

I have a problem to get the string of the received XML. It works on iPhone, but not on Android.

The XML that I receive is:

<?xml version="1.0" encoding="utf-8"?>
<string _xmlns="http://server.com/">[{"poiId": 1, "name": "Test", "lat": 57.570999, "lng": 11.785639},{"poiId": 2, "name": "Test2", "lat": 58.569237, "lng": 11.058598}]</string>

(Strange bug with posting the XML on this forum, remove the _ from the string _xmlns)

Titanium Code:

var doc = this.responseXML.documentElement;
var items = doc.getElementsByTagName("string");

arPOIData = JSON.parse(items.item(0).text);

I receive this error in Android:
Wrapped java.lang.NullPointerException

Hope someone could help or have a better solution to get the string from the XML.

Thanks.

Louis

    Hi Louis.

    What does this mean:

    var doc = this.responseXML.documentElement;

    we can't know what is "this" referring to without context.

    Anyway, have you tried to use "Ti.XML.parseString", and then use DOM methods? I've tried to load your XML string and everything works fine:

    var xml_string = '<?xml version="1.0" encoding="utf-8"?><string _xmlns="http://server.com/">[{"poiId": 1, "name": "Test", "lat": 57.570999, "lng": 11.785639},{"poiId": 2, "name": "Test2", "lat": 58.569237, "lng": 11.058598}]</string>'

var doc = Ti.XML.parseString(xml_string);
var items = doc.getElementsByTagName("string");

Ti.alert(items.item(0).text);

    (my real example is without underscore, I've put it here because Q/A software removes attribute if underscore is present)
