Thank You Appcelerator Q&A

We have completed our first app using Appcelerator and could not have done it without using the Q&A section. From researching older answers and posting new questions we were able to accomplish everything that we set out to do. We got approved to the appstore on the first try and it took 7 days total. We have entered our app into the EPA's Apps for the Environment Contest, if you would like to check it out (and vote :) ) - http://appsfortheenvironment.challenge.gov/submissions/4608-the-green-gumshoe

Thanks again and we look forward to using Appcelerator on our future projects!
