Disable click event on annotation deselect

Is there a way to disable the default disable click event so it doesn't fire the event associated with 'click'?

    Check the event's clicksource property. I filter out spurious events like this:

    mapview.addEventListener('click',function(e) {
    if (e.clicksource == null || e.clicksource == 'pin'
        || e.annotation == null)
    {
        return;
    }

    // do whatever you want to do when you click on the annotation
});

  • The click event itself will always fire on a click, but if you don't want anything to happen, just don't attach an event handler to it. If you have one attached and you don't want it to fire, then you can either remove it, or set a flag to be read inside the handler to check to see if the handle code should run or not.
