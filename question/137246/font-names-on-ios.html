Font names on iOS?

How is a font called for iOS, e.g. "fontFamily : 'HelveticaNeue-Bold'," or fontFamily : 'Helvetica Neue', fontWeight: bold"?

This site has them listed as one word, but I have been using the CSS method of 2 words. Which is correct?

sdk 2.01, ios 5.1, os 10.7.3

  • Hi Steve,
    on iOS you have to register the font in Info.plist with the name in file system. It is a good idea to save the ttf in subfolder.
    In application you have to reference the font with ist "native" name. On your Mac you can doubleclick and you see the name in title of window or you use fontforge or other toools to explore the name.

  • If you need a font like "HelveticaNeue-Bold" you call it "HelveticaNeue" - without the bold. Then you specify the fontWeight - this is the same style as you would do it in CSS for the web.

    font: {
    fontFamily: 'HelveticaNeue',
    fontWeight: 'bold',
    fontHeight: 22
}
