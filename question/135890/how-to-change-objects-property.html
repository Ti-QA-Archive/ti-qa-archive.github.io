How to change object's property

titanium mobile sdk 1.8.2

iphone sdk: 4.3

Hi,

I added an object from FirstView.js like this:

var ListView = require('ui/ListPage');

var listview = new ListView();
self.add(listview);

this is what's inside the ListPage.js

function ListPage(){

    var list = Ti.UI.createView({
        width: 320,
        height: 391,
        left: 320,
        top: 89
    });

    return list;
}
module.exports = ListPage;

how can I change ListPage's list left value from within the FirstView?

thanks for your help.

    Since you are returning the actual list object, you can access its properties directly.

    var ListView = require('ui/ListPage');

var listview = new ListView();
listview.left = 100;
self.add(listview);
