  • Hi

    Unfortunately this is a know problem if you are on Mavericks;

    • OS X 10.9: No output from warn, info, debug when debugging

    You do not say what platform you are on but if it is Mavericks then this is likely the reason.

    It has been fixed for the next version - 3.2 which you can get in beta version right now - but it will likely be released around the next 7 days from beta for general availability.

    Here's the quick fix:

    clean your project, rebuild, log should be working
