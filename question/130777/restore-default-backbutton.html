restore default backButton

Hi, I've been wondering this for a while.
Is there a way to restore the leftNavButton as the default back button?
I have an app that needs to load some stuff before the default back button appears, so my workaround is to set it as null and then set the left button like the default one.
Is there a function to do that?

    You can restore the default backButton by saving the initial value of LeftNavButton before you change it. Here is an example where I hide the nav button by covering it with an emptyView and then restore it using it's original value.

    function toggleBackButton(hideBackButton) {
  var tempLeftNavButton, emptyView;
  if (hideBackButton) {
    // store value of LeftNavButton temporarily while hiding button
    tempLeftNavButton = mywindow.getLeftNavButton();
    emptyView = Ti.UI.createView();
    mywindow.setLeftNavButton(emptyView);
  } else {
    // restore LeftNavButton
    mywindow.setLeftNavButton(tempLeftNavButton);
  }
}

    Hope that helps!
