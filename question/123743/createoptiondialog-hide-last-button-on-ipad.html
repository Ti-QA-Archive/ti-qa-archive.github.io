createOptionDialog hide last button on iPad

Hi,
since 1.7.2 and 1.7.3 the last button is not visible in createOptionDialog.
For instance using this code will show only the 2 first buttons, the third one is not visible.
Noticed it on iPad simulator

var dialog = Titanium.UI.createOptionDialog({
          title:'Add a photo to this post',
          options:['Take Photo', 'Choose From Library', 'Cancel'],
          cancel:2
        });

Any idea of how to fix this ?
Regards
