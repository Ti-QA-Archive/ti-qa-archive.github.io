  • Here's my solution to drag and drop but the logic is pretty much the same

    var win = Titanium.UI.createWindow({
    backgroundColor: 'white',
    height: Ti.UI.FILL,
    width: Ti.UI.FILL
});
var viewSize = 100, viewX = 145, viewY = 0;
var view = Titanium.UI.createView({
    height:viewSize,
    width:viewSize,
    backgroundColor:'#7f8c8d',
    opacity:0.3,
    top:150
});
var view2 = Titanium.UI.createView({
    height:viewSize,
    width:viewSize,
    backgroundColor:'#c0392b',
    top:150
});
view2.addEventListener('touchmove', function(e) {
    //view2.hide();
    view.setLeft(e.x + viewX - (viewSize/2));

});
view2.addEventListener('touchend', function(e) {
    viewX = view.getLeft();;
    view2.setLeft(view.getLeft());
    //view2.show();
});
win.add(view);
win.add(view2);
win.open();

    In this example I create two views, where I attach all the event listeners to view2 and apply the positioning to view. On touchend I then set the positioning of view2 to be the same as view.

    note if you don't want to see both views at the same time then hide view2 as you move view, which are commented out

    • I have revised the code again and removed the viewX and viewY variables.

      var win = Titanium.UI.createWindow({
    backgroundColor: 'white',
    height: Ti.UI.FILL,
    width: Ti.UI.FILL
});
var viewSize = 100;
// view to handle touch events
var view = Titanium.UI.createView({
    backgroundColor:'#c0392b',
    height:viewSize,
    left: 100,
    top: 150,
    width: viewSize
});
// view to mirror the first view and show the animation from the touch events
var view2 = Titanium.UI.createView({
    backgroundColor:'#c0392b',
    height:viewSize,
    left: 100,
    top: 150,
    width: viewSize
});
view.addEventListener('touchmove', function(e) {
    view.hide(); // hide so that it looks like only one view is being moved
    //  calculate x and y of view based on where you touch
    view2.setLeft(e.x + e.source.getLeft() - (viewSize/2));
    view2.setTop(e.y + e.source.getTop() - (viewSize/2));
});
view.addEventListener('touchend', function(e) {
    // set the positioning to match view2 and show the view again
    view.setLeft(view2.getLeft());
    view.setTop(view2.getTop());
    view.show();
});
win.add(view2);
win.add(view);
win.open();

      This code allows you to move the object along both the x and y, if you simply want to move left and right then remove anything related to 'setTop'

      The logical density factor doesn't need to be taken into account in this example.
