    This module solves it at least for android: sc.roundedavatar

    I'm not aware of a similar ios module.

    But I think, you can achieve this on both, iOS and Android, with the ImageFactory module. It provides a method ImageFactory.imageWithRoundedCorner that should be able to create rounded images.
