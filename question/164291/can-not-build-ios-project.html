    • longjohn errors often come from missing or malformed config files. Things like missing ios certificates/profiles, bad android AVD listings, etc.

      Both Apple and Android have mucked with their configurations in recent months, making changes that Appcelerator's team have had to chase down.

      Apple changes sdk versioning, xcode removes profiles during upgrade that need to be redownloaded. Android goes and changes the way AVDs are reported. Apparently fake stubs report as fully functional AVDs when they are not.

      So, if you get a longjohn, work back through the native tooling to see what might be missing.
