  • Hi Gema Fernadez,

    You can create tabgroup for android like iOS. Android not support this which you want.. it always open new window as a full screen(without tabbar). for more info read this.

    but you can create it by custom tab group (But in this case you need to create view instead of window (excluding root window)). just create one window and create two views. one for tab bar which is height 40 (according to your requirement) and set it top or bottom whatever you want.

    And remaining view is full view (excluding tab bar height). and when you want to open new window then require
    view and add it into view.. and parallely you need to manage array (for handling view hierarchy).

    And on back button just remove top most view from mainView.

    winHome.xml

    <Alloy>
    <Window>
        <View id="tabbarView" top="0" height="55">
            <!-- ADD TABS -->    
        </View>
        <View id="mainView" top="55" bottom="0">
            <!-- INITIALLY ADD ROOT SCREEN OF TAB -->    
        </View>

    </Window>

</Alloy>

    winTab1.xml

    <Alloy>
    <View>
    </View>
</Alloy>

    winTab2.xml

    <Alloy>
    <View>
    </View>
</Alloy>
