How to reauthorize contact permission

Hi,

I am looking at the kitchensink example how to authorize access to contact information.

var requestPermission = function(e) {

        var privs = Ti.Contacts.contactsAuthorization;

        if (privs===Ti.Contacts.AUTHORIZATION_RESTRICTED){

            Ti.API.info('Contact authorization restricted. User can not grant permission.');
        }
        else if (privs===Ti.Contacts.AUTHORIZATION_DENIED){
            Ti.API.info('Contact authorization denied. User has disallowed contacts use.');
        }
        else if (privs===Ti.Contacts.AUTHORIZATION_UNKNOWN){
            Ti.API.info('Contact authorization unknown. Request permission from user.');
        }
        else {
            Ti.API.info('Got unknown value for Ti.Contacts.contactsAuthorization');

        }


    };            

        if(isiOS6Plus() && Ti.Contacts.contactsAuthorization !==Ti.Contacts.AUTHORIZATION_AUTHORIZED){

            Ti.Contacts.requestAuthorization(requestPermission);
        }

I denied it once and now the permission popup won't show up again. How do I make it so that it will display the authorize pop up every time I try to access the contact until I finally authorize allowing contact access?

  • Once a user has denied access to a function from within an App (eg Contacts, Facebook etc) then only the user can re-allow it from the Settings App. All you can do is test for the authorization failure and prompt the user to go to Settings and re-allow it. This is the default behavior of all permissions in iOS
