  • you should:

    • download the whole project from github
    • execute sudo .&#x2F;build.py inside the root folder from Terminal and if all goes well, it'll create a zip file
    • Install the module: Titanium Studio->Help->Install Mobile Modules, select the zip file and the current project
    • make sure it's added in your tiapp.xml
    • follow the instructions on how to use it here
