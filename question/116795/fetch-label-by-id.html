Fetch label by id

Hey Guys,

Can i by any chance fetch the following label with its ID?

var l1 = Titanium.UI.createLabel({
id:'FN,',
text:'text',
width:100,
height:25,
top:10,
color:'#336699',
textAlign:'center'
});

The purpose is to change the text on the label dynamically. Im aware that it can be done with l1.text = '…'; but i would rather try to fetch it by the 'id' attribute set in the label code.

Thanks in advance.

  • you could loop through all of the children of the view and look for a specific id to find the object

    function findById(w, id) {
  for ( var x in w.children ) {
    Ti.API.info( 'Number of children: ' + w.children.length );
    Ti.API.info(  w.children[x].id );
    if (w.children[x].id == id) {
        return w.children[x];
    }
    return null;
  }
}
