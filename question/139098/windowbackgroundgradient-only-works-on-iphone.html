Window.backgroundGradient only works on iPhone

Hi *,
width this code 

exports.create = function() {
    var w = Ti.UI.createWindow({
        navBarHidden : true,
        backgroundGradient : {
            type : 'linear',
            colors : [{
                color : '#31BEEF',
                position : 0.0
            }, {
                color : '#006492',
                position : 1.0
            }],
            startPoint : {
                x : 0,
                y : 0
            },
            endPoint : {
                x : '100%',
                y : 0
            },
            backFillStart : false
        },
    });
    w.orientationModes = [Ti.UI.PORTRAIT];
    return w;
};

I tried to make a nice background. But it doesn't work on Android.

How can I do – workarounds?

Best regards Rainer

  • Hello Rainer,

    I found out that I couldn't put a gradient on a Window background, but it was OK to put it on a view.
    There is a bug that forces us to set the colors offset.

    See more details in this JIRA link: TIMOB-10015
