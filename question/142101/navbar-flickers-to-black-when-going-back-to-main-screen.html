Navbar flickers to black when going back to main screen

Okay. For iOS, I have my home screen's navbar hidden:
navBarHidden: true
and all child screens' nabbers shown:
navBarHidden: false

Navigation Bar

Black Bar

This transition does not appear when going back from a tab group, only from a single window.

While I am using an iOS navigation group on the home window to control navigation throughout the app, I don't believe that is the problem. Then again, I don't know if this is simply a quirk of Titanium or iOS in general. Any suggestions would be very much appreciated!
