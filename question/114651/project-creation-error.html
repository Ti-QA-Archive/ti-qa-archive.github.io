  • It is possibly a problem with environment variables. I had exactly same issue but after some time I discovered that my environment variable android_sdk was not correct. The problem was this:

    I had two variables set as follows:

    ANDROID_SDK = D:\DEV\Android\android-sdk-windows

    ANDROID_HOME = D:\DEV\Android\android-sdk-windows

    In the path variable I had %ANDROID_HOME%\tools

    That is where the problem was. You must have %ANDROID_SDK%\tools instead of %ANDROID_HOME%\tools in your path variable.

    It works even without Python and Git.

    I hope this will help some people on windows.
