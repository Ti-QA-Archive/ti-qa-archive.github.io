ACS - Push Notify Friends sends to everyone

Ti SDK 3.2.3

I've been wanting to send a push notification to All Friends of the current user:

Cloud.PushNotifications.notify({
       channel: 'friendsNotice',
       friends: 'all', // <-- This is not sending to All current user Friends but Everyone
       payload: {
           "badge": 1,
           "sound": "default",
           "alert" : 'A new notice has been shared'
      },
}, function (e) {
....

This comes directly from the documentation: PushNotifications to current User Friends

Doc mentions Param:

friends : Any
If this parameter is specified (regardless of the parameter's value), push notifications are sent to any of the user's Friends who are subscribed to the identified channel.

  • Cloud.PushNotifications.notify({
            channel : Ti.App.Properties.getString('CITY', 'Hamburg') + _channel,
            friends : true,
            "to_ids" : "everyone", // <==
            payload : _payload
        }, _callback);
