How to associate file extension with android app?

Hello everyone!

I would like to associate a custom file extension with my android app.

I've found a SO posting that shows an example using an intent-filter ( http://stackoverflow.com/questions/3760276/android-intent-filter-associate-app-with-file-extension ).

I've tried that and tested it by trying to open a file from Googles Mail App but it did not work.

Can anyone point me a good direction or provide me maybe with an example on how he did this?

Thank you!
Mario

  • This I did try:

    <intent-filter>
    <action android:name="android.intent.action.VIEW" />
    <category android:name="android.intent.category.DEFAULT" />
    <data android:scheme="file" />
    <data android:host="*" />
    <data android:pathPattern=".*\\.pdf" />
 </intent-filter>

    Added it to tiapp.xml within the <android> section so that it will be included in the manifest.
