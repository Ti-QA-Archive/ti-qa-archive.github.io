Titatnium Studio on Windows - Script error - windowLocalStorage is null and Alloy and CLI are not getting installed

Titatnium Studio on Windows 7: After starting Titanium studio, an error pop-up is appeared that says windowLocalStorage is null or not an object before displaying onto the dashboard. Dashboard is completely empty with this error pop-up. Also, when checking for updates, Alloy and CLI are not getting installed.

I am a first time user and would like to get rid of this for building a mobile app on windows.

  • HI Usha,

    First of all you need to install Alloy and node.js in your system and titanium
    If you are using windows then open cmd as administrator and type command:

    npm install -g

    you can directly install node.js from website from below link
    http://nodejs.org/

    If you are using mac then type this command:

    sudo npm install -g

    And prefer this link this will help you:
    http://docs.appcelerator.com/titanium/latest/#!/guide/Titanium_Compatibility_Matrix
