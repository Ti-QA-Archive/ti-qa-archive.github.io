Performance issues while swiping between WebViews in Tabs / Viewpager-Module

Hello! I´m using the "Android View Pager" by Bajawa Labs to create Tabs and display a bunch of Webviews and to swipe back and forth between them. This is the Module: https://marketplace.appcelerator.com/apps/2228?1903717439#overview

The webviews are mostly a big image (JPG, 6MP, pages of a Newspaper) with some styling, absolute positioned DIVs etc. When I test the swiping between the webviews on a Nexus 7, it works like a charm. But on a Nexus 10, the response of the swiping is painfully slow, and there´s no fun in using it.

I tried to reduce the filesize, that didn´t help much. Reducing the imagesize took performance to an acceptable level, because the page did not need to scale down to fit in the screen (webview´s "scalesPageToFit" is set to true"). But this isn´t helping cause I need the big images to display their content properly and readable.

Are there some general hints how to optimize the performance when using such big images in combination with webviews and especially the "Android View Pager"?
