Open Facebook page link

I am using Ti.Platform.openURL to open up the link to a Facebook page. The Facebook app opens up, but the page isn't opened. Here is an example of a function call i'm making

Ti.Platform.openURL("https://www.facebook.com/microsoft")

How can i make the Facebook app open the page?

  • First you need to get the Microsoft user id using : https://graph.facebook.com/microsoft

    Which provides the user id : 20528438720

    Then, use the Facebook app 'url scheme' fb:// to open the profile

    Ti.Platform.openURL("fb://profile/20528438720");

    This worked for me.
