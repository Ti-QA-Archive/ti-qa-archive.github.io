Writing apps that are running on iOS and Android with the same code

I've had some trouble getting my app to run both on iOS and Android with the same code. I've experienced performance impacts when calling openWindow under Android and different behaviours of the orientationchange event. But I find a solution how I can avoid these traps. But this solution has some influences on the way I structure my code and where I place my variables.

To share my conclusions I've written an article in my blog. Maybe you will find it helpful:
