Import in Alloy an external class

Hi there,

I am looking into Alloy and I'm having a problem to import an external class :

  • lib/externalclass.js

    var gExternalClass = {

     // --- global vars & flags
     globs :{
         nited: false,
     },

     init: function ( ) {
     },

     public_function_1: function ( ){
      return 'xxx';
     },

     public_function_2: function ( ){
     },
};

  • index.js

    require('externalclass');
...
var test = gExternalClass.public_function_1();

On execution I get the error 'Can't find variable: gExternalClass'…

I believe the problem is in the syntax to include an external class, would anybody could help me on that ?

Best Regards,
Clem

  • Check out this wiki article for an exhaustive reference on commonjs: http://docs.appcelerator.com/titanium/latest/#!/guide/CommonJS_Modules_in_Titanium

    lib/externalclass.js

    module.exports = {

       // --- global vars & flags
       globs :{
           nited: false,
       },

       init: function ( ) {
       },

       public_function_1: function ( ){
        return 'xxx';
       },

       public_function_2: function ( ){
       },
};

    index.js

    var gExternalClass = require('externalclass');
// ...
var test = gExternalClass.public_function_1();

  • For this particular case, since your class doesn't abide by the CommonJS pattern, you should use Ti.include instead.

    ###In your index.js file:

    Ti.include('externalclass.js');
...
var test = gExternalClass.public_function_1();
