create map annotations from array?

Hi everyone,

Been playing with appcelerator for maybe two weeks now and having a great time learning the ins and outs. I was wondering if someone could please help me create map annotations from an array like the one below. I tried this but didn't work.

var addr = [ { latitute: 18.93231,
    longitude: -70.55923,
    title: 'Loma de la Calentura',
    pincolor: 'Ti.Map.ANNOTATION_GREEN',
    animate: 'true' },
  { latitute: 18.54213,
    longitude: -71.02084,
    title: 'Loma de la Zurza',
    pincolor: 'Ti.Map.ANNOTATION_GREEN',
    animate: 'true' },
  { latitute: 18.87938,
    longitude: -70.01084,
    title: 'Arroyo Zunbado',
    pincolor: 'Ti.Map.ANNOTATION_GREEN',
    animate: 'true' } ];

_.each(addr, function(addr) {
    Ti.Map.createAnnotation({
        addr

    });
});

Thanks

    Create an array of annotations and use addAnnotations methods to add your array of annotations. Then, add it to your map view.

    Creating array of annotations.

    var addrAnnotations = [];
_.each(addr, function (addr) {
  addrAnnotations.push(Ti.Map.createAnnotation({
    title: addr.title,
    pincolor: addr.pincolor,
    latitute: addr.latitute,
    longitude: addr.longitude
  }));  
});

    Add array of annotations to your view.

    $.myMapView.addAnnotations(addrAnnotations);
