    add this to your tiapp.xml file

    
<ios>
    <min-ios-ver>4.3</min-ios-ver>
</ios>

    • sorry, I sent before finish. Add those lines if you want to use xcode 4.5 and iOS 6. Obviously, you will not able to deply to 4.2 and below. Since armv6 now is not supported by xCode, I understand that is not possible to deploy for old devices with this xCode version and, if you use older xCode versions, you will have also to use old titanium sdk versions.

      — commented October 4th 2012 by Javier Rayon

    • Yes Javier said correct you cannot build your app for Architecture armv6 as Xcode 4.3 supports only for Standard armv7 architecture
