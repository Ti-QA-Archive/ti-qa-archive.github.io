Solution for a simple multilanguage app

Hello

I am quite new to Titanium and also no veteran JS coder so please be patient with me.

I have a simple app which I am looking into translating to english and german. I have so far not quite figured out the best way to do it.

There is of course the "proper" way of doing it using i18n and externalize all strings. What I don't understand is, would it then be possible for me to let the user set the desired language from within the application or is it depending on the system locale and language? For example I need someone with ie. a Danish locale to use the english version of my app.

Is there perhaps a simpler way of dealing with multiple languages?

    Hi Johan. In fact, i18n files are quite easy. While in other languages and platforms internationalization is a nightmare, in Titanium is really really fast and easy to maintain.

    Regard your question, by default the user will not be able to select which language to use if their local language is not supported by your app. You can easily emulate other languages changing iPhone or Android settings (on iPhone emulator, goto settings > general > international > languange). This way, you can see what the user will see.

    You can create your own locale resource system, but I suggest you to use i18n and try supporting the most popular languages for your users.
