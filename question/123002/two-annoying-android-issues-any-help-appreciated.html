  • I volunteer to try to help you solve one issue. I needed to modified the standard Android manifest (it was about modifying 'android:theme'). The best way I found to do that was:

    1) To create a 'platform' folder in my project's directory ('platform' folder in same directory as 'Resources' folder).

    2) To create an 'android' folder inside the 'platform' folder.

    3) To copy the original 'AndroidManifest.xml' file (see 'build/android' folder), to paste it into the 'platform/android' folder, and to modify it as needed.

    4) The new 'AndroidManifest.xml' file takes precedence over the generic one.
