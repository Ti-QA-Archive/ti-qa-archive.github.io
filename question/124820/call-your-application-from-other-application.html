Call your application from other application

Hello im interesting in calling my application from web browser, for example when i surf in internet with my mobile when i click on any link i wanna point that link to my application content inside and open that item from web site in my application. for example when i post something on facebook and when someone click on my post app should run and open that content inside? can i do it with titanium?

    Yes, you can do it. Ti automatically sets up a protocol name you can use. By default I believe it is just your application name with no spaces and in all lowercase.

    For iOS, you can see the name (and even override it) by looking in the Info.plist file in the project/build/iphone folder. Look for key CFBundleURLSchemes and see what the associated string value is.

    In browser links, you use that name followed by :// and optionally other text. You can retrieve the other text on the link using Ti.App.getArguments
