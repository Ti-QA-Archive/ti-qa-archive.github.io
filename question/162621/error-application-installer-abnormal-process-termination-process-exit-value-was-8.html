  • I have the same problem with windows 32-bit. I did many tests. I gave up, I installed the emulator Bluestak, compile the program with the command "titanium build - android-platform b" in the terminal and install it on the emulator, which by the way is very fast.
    On windows 7 64 bit I have no problem.
