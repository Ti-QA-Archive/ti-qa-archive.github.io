Android status bar and nav bar not hidden

I'm trying to hide status bar and nag bar from my android app. But so far its unsuccessful.

Here is my code:

function createStoreTab() {
    var storeWin = Titanium.UI.createWindow({  
           fullscreen: true,
           navBarHidden: true,
           exitOnClose: true,
           modal:true,
           url:'main_windows/store.js',
        orientationModes: orientationAllow
    });
    var storeTab = Titanium.UI.createTab({  
        icon:'tab_images/store.png',
        title:'STORE',
        window:storeWin
    });


    return storeTab;
}

This is one of my win in a tab group. In the xml, I've even set status bar hidden, navbar hidden and fullscreen to true. But, other than the splash screen, when the window open, the status bar and nag bar is still the.

I'm using 3.1.1 and building android sdk 2.3.3 running on device with 2.3.6.
