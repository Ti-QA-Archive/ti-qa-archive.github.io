Metaio "Cloud Plugin" as a module?

Ti SDK 3.1.3
Platform: iOS

My understanding of module development is limited. To be honest, it is over my head. But I have successfully created apps with titanium, and I would love for my next one to utilize Metaio AR technology. I realize that setting this up so that I could utilize all the "extras" of the Metaio SDK would be extremely difficult/impossible - but Metaio offers a "cloud plugin" Metaio Cloud Plugin that I suspect could be turned into a module.

Right now, I can export a cloud plugin app from Metaio Creator to xcode, and all it does is show the AR viewer - which accesses content uploaded to the Metaio cloud. I would love to be able to put this in a tab of a titanium app, then be able to build the rest of the app the only way I know how, with Titanium!

I know appcelerator has module development documentation, but I am completely lost in the regard of turning the default auto-generated cloud plugin xcode project into something I can use in titanium.

Does anyone have any suggestions, tips, existing Metaio modules, tutorials, etc that would get me headed in the right direction to achieve this goal.

PS - I am aware there is a wikitude AR module. My company has already decided on Metaio for AR needs. It would be a great benefit to us to not pay an outside developer for each app that would utilize the tech.
