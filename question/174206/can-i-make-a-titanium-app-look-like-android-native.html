  • OK, the easiest solution is upgrading to the latest Titanium SDK. If you build your app with sdk version 3.2.0 (or greater) it will automatically target the latest Android sdk and as a result you'll get the default Holo theme. If you have a newer version of titanium already installed then just select it in tiapp.xml (under Built Properties). Otherwise you can update Titanium with Titanium Studio (Help > Check for Titanium Updates…) or via npm with npm update -g titanium :)
