Show image from sd-card in webview

Hi,

I want to show an image from my sd-card in a webview.
(The path to the image is a variable)

I tried this, but it doesn't work.

var location = _storage+'nl.tomwebs.blurry.friends/100000573103540/1458662928';
var f = Titanium.Filesystem.getFile(location);
var temp = f.read();
var encodeData = Ti.Utils.base64encode(temp);

$.web_game.html = '<html><head></head><body><img src="'+encodeData+'"/></body></html>';

Now I get a html error (this image doesn't exists).

Can someone help me?

(Android and iOS)
