How do you test on iOS6?

I need to test that my app runs on iOS 6 and it's quite urgent (i.e. I can't wait until it's fully released).

I've installed the GM of iOS6 and I have tested XCode and written and run an app from there.

However, when I try to build my Titanium app, Ti Studio reports the following:

[INFO] Launching application in Simulator
Unknown or unsupported SDK version: 6.0
[DEBUG] Simulator SDK Roots:
[DEBUG] 'Simulator - iOS 5.1' (5.1)
    /Applications/Xcode.app/Contents/Developer/Platforms/iPhoneSimulator.platform/Developer/SDKs/iPhoneSimulator5.1.sdk
[INFO] Launched application in Simulator (80.36 seconds)
[INFO] Application has exited from Simulator

The simulator opens and then crashes straight away. I've tried cleaning the build directory with no success.

Can anyone assist in getting my app running in the iOS6 Simulator?
