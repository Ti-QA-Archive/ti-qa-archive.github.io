how to parse this soap response

Hi all,

how to parse this soap response ,

<s:Envelope xmlns:s="http://schemas.xmlsoap.org/soap/envelope/">
   <s:Body>
      <GetFacebookUserCredentialsResponse xmlns="http://tempuri.org/">
         <GetFacebookUserCredentialsResult xmlns:a="http://schemas.datacontract.org/2004/07/AlameenDAL.Model" xmlns:i="http://www.w3.org/2001/XMLSchema-instance">
            <a:UserName>ramesh</a:UserName>
            <a:UserPassWord>ramesh</a:UserPassWord>
         </GetFacebookUserCredentialsResult>
      </GetFacebookUserCredentialsResponse>
   </s:Body>
</s:Envelope>

usually respose will be like this 

<GetAllBusStationResult><![CDATA[[{"Title":"Rashidiya Metro Station 1","StationID":"12801","Area":null,"ZoneNo":"128","Distance":30.........

    Hi, if you want to retrieve the UserName value for example, you can use

    var name = getElementsByTagName('a:UserName').item(0).textContent

    The name variable now contains "ramesh"
