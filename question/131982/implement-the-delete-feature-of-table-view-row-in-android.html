  • Hi mathew,

    There is no built in editable table in android like in iOS. So can not implement this functionality just by setting 1-2 attributes.

    What you can do is, create your custom table view row and in that row add this two button,
    1) first button with that MINUS sign on click of which Delete button will display and
    2) Second button is that delete button on click of which you can perform delete functionality of your table view row. (You can keep this button hidden and only show when user clicks that minus sign button)

    And on the click of that minus sign button you can change background image so that it will give effect like, in starting it was displaying in vertical and than on click of that minus sign when Delete button is displayed, you can change background image of delete button to that horizontal minus sign image.

    Hope this will help.

    Cheers.
