    In javascript, array content is accessed by the corresponding index:

    var myArray = [ 'apple', 'orange', 'pear'];
console.log(myArray[1]); // apple
console.log(myArray[2]); // orange

    I normally for my way through arrays, so that I know they always come up in the order I expect them to:

    for (var x = 0, max = myArray.length; x< max; x++){
  console.log(myArray[x]);
}
