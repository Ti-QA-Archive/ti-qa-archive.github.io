Help with evaluating Javascript object

So this seems like a dumb question, please bear with me. When I'm debugging Titanium objects, arrays, etc. I would like to see what an object contains using Ti.API.info. This works great on most simple variables until I hit objects like the Contacts object. How can I see the values of an object if I don't know what the child properties are?

  • Try:

            function printObj(object) {
            var output = '';
            for(property in object) {
                output += property + ': ' + object[property] + '; ';
            }
            Ti.API.info(output);
        }

    From SO.

  • Adam's example looks fine, but you need to use the full Titanium.API.info on desktop, as Ti is not aliased like it is on mobile. This minor fix should allow it to work on desktop. DISCLAIMER: I haven't actually tested this code in an app…

    function printObj(object) {
            var output = '';
            for(property in object) {
                output += property + ': ' + object[property] + '; ';
            }
            Titanium.API.info(output);
        }
