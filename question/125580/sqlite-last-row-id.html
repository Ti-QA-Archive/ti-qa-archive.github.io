SQLite Last Row ID

I am trying to get the value of the last row Id in a table. I am trying to use the last_insert_rowid() syntax but with no luck. Does anyone have any sample code on how to get the ID of the last row added?

      — commented September 14th 2011 by Doug Handy

    • Just to clarify, using db.lastInsertRowId in only appropriate after you have done a db.execute with a sql INSERT statement. It is a fast way to get the id of the last row inserted when you are using auto increment or whatever. Sometimes this is useful for additional processing.

      When you want to know the highest id in a given table but did not just do an INSERT, then you would use a SELECT like above to obtain the value.

      — commented September 14th 2011 by Doug Handy

    • Just a side note. If you use the method described by Doug, be sure you have AUTOINCREMENT on the column, otherwise you might have recycled IDs back. Check SQlite3 documentation for lastInsertId.
