  • Accepted Answer

    First go to /Library/Application Support/Titanium/modules/iphone/
    and paste the ti.paint folder here.

    tiapp.xml you should have this:

    <modules>
<module version="1.0">ti.paint</module>
</modules>

    then in your app use this code:

    Titanium.Painter = require('ti.paint');

//then create the paintview
painter = Titanium.Painter.createView({
    top: 0,
    left: 0,
    right: 0,
    bottom: 0,
    strokeWidth: 45,
    strokeColor: '#000'
    zIndex: 3
});

    This should be enough, if it still won't work, throw away your build folder and try again
