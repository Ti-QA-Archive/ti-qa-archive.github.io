iOS: Animating the textfield which is added to a window causes the keyboard to flicker

When i added a textfield into a simple window and when the textfield is animated upon focus, the keyboard appears and immediately disappears. This would happen if you click on the textfield about 2-3 times and clicking/focusing the textfield several times would cause the keyboard to " flicker animation".
This is ocurr on Titanium SDK 3.4.0 GA; 3.4.1 GA; 3.5.0 GA.

Steps to reproduce:

  1. Add my code block to app.js

  2. Focus on the textfield

  3. Focus on the textfield several times continuously

    var textfield = Ti.UI.createTextField({ 
 value: "",
 borderWidth: 1, 
 borderColor:"red", 
 color: '#537CA6', 
 height: 30, 
 top: 5, 
 left: 5, 
 paddingLeft: 5, 
 paddingRight: 5,  
 width: 200
});
var winMain = Ti.UI.createWindow({ backgroundColor: 'white' });
winMain.add(textfield);
winMain.open();

    Has anyone that problem? Or Is there a way for resolve this problem?

I found out an issues for same this problem when add Textfield into Tableview control on SDK 3.4.1 but it had not still resolve on SDK 3.5.0(Please see by link: https://jira.appcelerator.org/browse/TIMOB-18127)

I waiting release new fix version for this problem.
