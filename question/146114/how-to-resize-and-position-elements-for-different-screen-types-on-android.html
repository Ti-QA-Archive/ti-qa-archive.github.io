  • Hi

    Check out the guide Transitioning to the New UI Layout System and in particular Universal Unit Support.

    If you use dimensions with 'dp' and the end you will find most things fall into place.

    Try;

    var view1 = Ti.UI.createView({
    backgroundColor: 'orange',
    height: 100
    width: 100
});
win.add(view1);
var view2 = Ti.UI.createView({
    backgroundColor: 'orange',
    height: '100dp',
    width: '100dp'
});
win.add(view2);

    The first one will change in size on different resolution devices, the second one should stay very close to the same on all.
