    You can't prototype Titanium objects, you can only prototype native JS objects.

    You can, for example, create function that creates custom Titanium view:

    function MyBlueView(config) {
    config = config || {};
    config.backgroundColor = '#00f';
    return Ti.UI.createView(config);
}
