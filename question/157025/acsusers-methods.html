ACS.Users Methods

We are using Node in a app that uses ACS. We have used ACS.Users.login and ACS.Users.logout without an issue, however, we cannot use ACS.Users.resendConfirmation - it does not exist.

The API: http://docs.appcelerator.com/cloud/latest/#!/api/Users-method-resend_confirmation seems to indicate that the method is there, but when I look at the Users object, this is what I see:

{ create: [Function],
login: [Function],
show: [Function],
showMe: [Function],
search: [Function],
query: [Function],
update: [Function],
logout: [Function],
remove: [Function],
requestResetPassword: [Function] }

Is there a reason that the resend confirmation method is not there?

Thanks,
JT

  • Hello James Mudge,

    As per cloud documentation, Resend Confirmation method is not yet supported in the ti.cloud module. So you cannot use in Titanium SDK. This API is available for only REST API, IOS SDK and Android SDK.
