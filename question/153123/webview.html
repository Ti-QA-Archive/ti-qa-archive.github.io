WebView

Hello,

I am trying to change the innerHTML of a locally loaded HTML file. I create my webview, then pass in the local HTML via the url property, I add a load listener and when the HTML is loaded I am trying to use the evalJS method of the webview to change the innerHTML of the body tag.

The local HTML file loads fine, I just cant get it to change its content.

This fails and I cant seem to find a way to dynamically change the content on the fly. I am using Android.

I appreciate any advice, Thanks in advance.

  • Hello,

    you can always create a webview with the actual html. I recommend to get the HTML file with an httpclient connection, save it to a file, and then load it into the webview. That way, you will have the source code and will be able to change it and reload the webview.
