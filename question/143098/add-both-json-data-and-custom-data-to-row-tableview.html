Add both json data and custom data to row tableview

Is it possible to have some rows in a table use standard json data and other rows use a custom element?
For example:

    var settingsData = [{title: 'Edit', hasChild: true, header: 'Tools' },
                     {title: 'Option1', hasChild: true },
                    // { INSERT a CUSTOM ROW HERE? }
    ];

    var table = Ti.UI.createTableView({
        data: settingsData,
        style: Ti.UI.iPhone.TableViewStyle.GROUPED,
    });

I'd like to add a 3rd row to this table where INSERT CUSTOM ROW is that has a label to the right so I can display a version number

    Yes, you can create custom rows and mix them in with "non-custom" ones. TableView.data just takes an array of TableViewRow objects

    • TableView.data also accepts a dictionary representing a TableViewRow, which is what you're taking advantage of in your example. You can do either, or both. Just be sure to set the className on custom rows that have the same layout.
