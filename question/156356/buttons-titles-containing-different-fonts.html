    Hi

    The bad news is that the font is only available to some things, with buttons you can change the font to a built-in or a custom font - but if the button is in a toolbar, titlebar, nav bar - then you cannot change the font.

    You can combine views, buttons, labels but this will not get around the problem your specific needs have, a custom symbol font will be unusable on the system buttons.

    I recently went through this with iOS and made the decision that it was easier (not easy) to render the icons as images in the different sizes required.

  • here this how you could make custom font buttons

    //custom - bug?
var customBtn = Ti.UI.createButton({
    title:"BUY",
    borderRadius: 3,
    borderWidth: 1,
    borderColor: "#5d7731",
    color:"#000",
    backgroundColor: "transparent",
    backgroundImage: 'images/btn_grad.png',
    font: {
        fontFamily:'Caecilia LT Std', //custom font
        fontSize:18,
        fontWeight:'bold'
    },
    height:32,
    top:100,
    right:0,
    width:50
});
