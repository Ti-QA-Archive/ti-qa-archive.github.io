Regex a label created by a data array

I have a table view with a label that's created from some data. That data is an array so the text comes out as a comma separated array string.

However, as the commas have no spaces after them it isn't as human readable as I would like it to be.

How could I go about REGEXing this data so it adds some a single space after each comma?

   var routesLabel = Ti.UI.createLabel({
      text : 'ROUTES ' + data.stop.routes,
   });
