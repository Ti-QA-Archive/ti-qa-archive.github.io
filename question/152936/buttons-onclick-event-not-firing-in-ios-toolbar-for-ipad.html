Button's onclick event not firing in iOS Toolbar for iPad

I have a Window in a Tab Group which contains a TableView with records. I'm adding a Toolbar to be displayed at the bottom of the Window. Inside it I have 2 buttons, each with an onclick event.

When clicking on the button, nothing happens. At the moment I'm trying a simple alert. Please see example code below.

var win = Ti.UI.createWindow();

var orderType = Titanium.UI.createButton({
    title:'Order Type',
      style: Titanium.UI.iPhone.SystemButtonStyle.DONE
});

orderType.addEventListener('click', function(e){
    alert('Clicked');
});

var toolbar = Titanium.UI.iOS.createToolbar({
    items:[orderType],
    bottom:0,
    borderTop:true,
    borderBottom:false,
    barColor:globals.appGlobals.defaultRed
}); 

win.add(toolbar);

The toolbar loads perfectly, but it's like all the Touch Events are disabled.

The following is my environment:

  • iOS 6.1 for iPad only
  • SDK 3.1.0 GA
  • The above code does not exist in the app.js, but in its own commonJS Module.
  • OSX 10.8.3

Many thanks.
John

  • Ok. I figured it out more or less, but this might not be an answer for everyone.

    Instead of creating a Toolbar Object, i simply created the buttons, added them to an array and pushed the array to the win.toolbar property. The button's click event is now firing. See below code:

    var win = Ti.UI.createWindow();

var orderType = Titanium.UI.createButton({
    title:'Order Type',
    style: Titanium.UI.iPhone.SystemButtonStyle.DONE
});

orderType.addEventListener('click', function(e){
    alert('Clicked');
});

var cancel = Titanium.UI.createButton({
    systemButton: Titanium.UI.iPhone.SystemButton.CANCEL
});

flexSpace = Titanium.UI.createButton({
    systemButton:Titanium.UI.iPhone.SystemButton.FLEXIBLE_SPACE
});

var buttonArray = [];
buttonArray.push(orderType);
buttonArray.push(flexSpace);
buttonArray.push(cancel);
win.setToolbar(buttonArray);

    This will default the style of the Toolbar to the same as the Menu Toolbar. It also defaults the Toolbar at the bottom. I need to investigate how to override these properties, but for now this should do.

    I hope this helps a bit.
