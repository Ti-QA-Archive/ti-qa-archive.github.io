Paper Printout

We want to create form for users to fill in via their mobile device, so that once completed they have a ready to printout, filled-in application that conforms to the A4 paper standard size.

  1. 1). User enters data via fields which is saved in variables.
  2. 2). These variables are used to populate various visual elements to be placed on the form like 'Your Address'.
  3. 3). All the final customised elements are placed inline with the same proportions as that of an A4 piece of paper.
  4. 4). User hits submit which triggers a capture function saving the layout to a jpg/png.
  5. 5). User now has an image to send as an email or print from the phone.

In theory is it technically workable? What would be the relevant 'capture' function using Appcellerator/Titanium Studio?
