Google Maps v2 is not available on SDK 3.1.0.GA

I installed Titanium SDK 3.1.0.GA but it's unavailable to use Google Maps API v2.

require('ti.map'); is not working, and I can only see 'ti.cloudpush', 'facebook' inside '~/Library/Application Support/Titanium/modules/android'

I reinstalled the SDK several times but it didn't work.

What is the problem?

  • I assume you are using the Android emulator. You need to download the Google APIs for the Android SDK you are interested in using. Run the android command to view all Android packages, expand the SDK and select Google APIs. Then you'll have to tell Studio to use that Android SDK version WITH the Google APIs, then Google Maps will work.
