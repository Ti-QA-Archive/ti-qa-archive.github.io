Replace words in string

Hi.

If i have a string : I am a mobile rockstar.

Can i replace for ex one word in that string?

Something like:

var rock = 'I am a mobile rockstar.';

var star = rock.replace('am', 'aint');

var mobile = star;

var btn = Ti.UI.createButton({
        title: star,
        style: Titanium.UI.iPhone.SystemButtonStyle.PLAIN
        });
win.setLeftNavButton(btn);

Yes, The button says: I aint a mobile rockstar.

But if i want to do it with a JSON respons where i have a lot of html and i want to change the color on something it wont work.

For ex i have this in my returned code: #00ccff and i want to have #ffffff

if i try

 var wordpressd = JSON.parse(this.responseText);
        var clean = wordpressd.replace('00ccff', 'ffffff')
        var wordpress = JSON.parse(clean);

it wont work. It wont show me anything.

What have i done wrong?

Thanx

    The problem is that you are trying to replace after you have parsed the responseText using the JSON.parse function. Once it is parsed, it is a fill fledged JavaScript object (not a string), so invoking the replace function on it won't achieve much.

    You code should look something like this:

    var wordpressd = this.responseText;  // See what I did there?

var clean = wordpressd.replace('00ccff', 'ffffff'); 
var wordpress = JSON.parse(clean);

    Hope this helps
