Test and distribute same code in iOS7 and iOS8

Hi,
I would like my app to work both on iOS7 and iOS8.
So far the code is working fine with both, however I need to switch between SDK 3.3 and 3.4 to deploy on either.
How do I handle testing (packaging and ipa) and down the track distributing my app so that it works with both?

NB: The Titanium Compatibility Matrix (http://docs.appcelerator.com/titanium/3.0/#!/guide/Titanium_Compatibility_Matrix
) says "Minimum target iOS version 7.1.X" for SDK 3.4.0 however when I test it on my iOS7 device I get an error 

Invalid "--ios-version" value "7.1"

    For Titanium SDK 3.. 4.0 you have to install minimum iOS SDK version 8.0.x then you can set minimum target iOS SDK version 7.1.1 or later.

            <ios>
        <min-ios-ver>7.1.2</min-ios-ver>
    </ios>
