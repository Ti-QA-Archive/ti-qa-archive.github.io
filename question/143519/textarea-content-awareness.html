Textarea content awareness

Hi.

I have a win in my app that reads feeds from FB.

Sometimes a feed only concist of a picture and no text.

When that is the case i get an empty textarea and a picture below it.

Is there any way to get the textarea not to appear if there is no text?

For instance i have this to show the text and pic from a feed.

 var ta1 = Titanium.UI.createTextArea({
    value : win.data ,
    height : 70,
    width : 300,
    top : 5,
    font : {fontSize:15},
    textAlign : 'center',
    backgroundImage    : 'images/tablebgg.png',
    borderRadius: 4,
    borderColor: '#050505',
    scrollable : true,
    editable : false

});
win.add(ta1);

var pic = Ti.UI.createImageView({
    top: 80,
    height: 85,
    width: 'auto',
    image    : win.pic,
    borderColor: 'transparent',
    borderRadius: 6,
    borderWidth: 3,
    zIndex    : 0
});
win.add(pic);

Something to get the ImageView to be

top: 5,
height: 140

And the textare to be

height: 0,
width: 0

if win.data is empty.

    • if ( win.data === "" ) {
    pic.updateLayout({top:5, height: 140});
} else {
    pic.updateLayout({top: 80, height: 85});
}

    pic.updateLayout({top:5, height: 140});

    pic.updateLayout({top: 80, height: 85});
}
