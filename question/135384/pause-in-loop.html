pause in loop

Hi,

I have an array MyArray[], and i'm running through it using a for loop (typical scenario).

It works, no issues..

Within the loop, i'm doing the following

for (…)
{
MyFunction(MyArray[c].myVar);
}

Everything works great.

(it really does. )

However, what I need is to put a pause in that loop… for a few milli seconds.. its just going so fast that some bad stuff is happening.

ANy ideas how to do this?

I know in C/C++, there were functions like sleep(5), or pause(4); etc..

Is there anything similar in JS?

Cheers

    There's no built in sleep function available in JS, but you can archive it this way:

    function sleep(milliseconds) {
    var start = new Date().getTime();

    while((new Date().getTime() - start) < milliseconds) {
        // Do nothing
    }
}
