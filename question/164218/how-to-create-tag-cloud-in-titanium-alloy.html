    Create a view to use as your container. 

    View id="tagsContainer"/>

    Set its layout to horizontal and its height to SIZE, so it will adjust as needed

    layout: 'horizontal', height: Ti.UI.SIZE

    Create a custom view for your tags, and add them to the container.

      I have been able to create a basic TagCloud View and labels following your suggestion and also by using an algorithm I found on Stack overflow, I have been able to vary the font size according to the weight of the tag.

      function calculateFontSize(weight){
    Ti.API.info("Weight is : "+ weight);
    var max = 100; // Should be computed
    var min = 0;   // Should be computed

    var fontMin = 10;
    var fontMax = 20;

    var size = weight == min ? fontMin
            : (weight / max) * (fontMax - fontMin) + fontMin;        
    return size;
}
