SMTP Check Failed

Hi everyone!

I configured smtp settings on Appceleretor's site, but i got that message:

SMTP Check Failed.
554 5.7.1 <ec2-204-236-171-146.us-west-1.compute.amazonaws.com[204.236.171.146]>: Client host rejected: Temporary Blocked

I have my own domain and an email box in that domain. I have follow the tutorials on the web and seems to be all right, but i got that message instead.

Any ideas?

  • Do the settings like below:

    Username : your email
Password : email password
TLS : true
SMTP : smtp.gmail.com (use your email it may be yahoo or hotmail )
Port : 587
domain : gmail.com (similarly here use yours)
authentication : login
user sender : false
test mode : false

    This will help you, I got the same error and this solved the issue. In my case my mistake was, I wasn't using the password of my email I was using a dumy password.
