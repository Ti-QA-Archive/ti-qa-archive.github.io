  • Hi Bhagyashree

    It's really difficult to answer your question accurately without having all the information that is necessary. Please read Participating in the Q&A, paying particular attention to the Q&A Question Checklist, before posting any more questions.

    At a guess, you should do something like this:

    picker.addEventListener('change',function(e){
  Ti.API.info("You selected row: " + e.row + ", column: " + e.column + ", title: " + e.row.title + "row index: "+e.rowIndex + ", column index: " + e.columnIndex);
  eth_combo.value = e.row.title;
});

    However, look at the scripts with the word "picker" in them in the KitchenSink for more examples. Start with picker_basic.js and picker_basic2.js.
