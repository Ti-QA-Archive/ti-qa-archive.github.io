Show window title on window.leftNavButton but not on window itself

This might be a bit odd but the client would like to hide the window title on the actual page itself but display the title on the next window's leftNavButton object.

As far as I've found this is a bit tricky because the only way to hide just the window title in a nav is to set it to nothing but doing so hides the leftNavButton on the next page.

Is there a straightforward way of going about this? Maybe setting a global variable to something like currentWindowName and on window load set that to the leftNavButton's text before overwriting the variable with the new current window title. Is that doable?

Thanks for any help you can give.
