    Hi,

    You can add this views in your js file.

    For that you have two options….one is you can handle array with storing all the image names….

    like…

    .js file

    var arrImages = ['LS30135512-lois-eyewear_sun.jpg','LS30135513-lois-eyewear_sun.jpg'];
var views = [];

for(var i in arrImages){
    var view = Ti.UI.createView({
        backgroundImage: "eyewear/collection/images/elems/" + arrImages(i)
    });
    views.push(view);
}
$.scrollView.views = views;

    or if you do not want to store images in array and you know the total number of images then you can rename your images in a sequence so at the time of for loop you can use that number like below….

    rename all your images like… lois-eyewear_sun_0.jpg,lois-eyewear_sun_1.jpg,lois-eyewear_sun_2.jpg,etc..

    then use for loop like below

    var views = [];
for(var i = 0; i < 10; i++){
    var view = Ti.UI.createView({
        backgroundImage: "eyewear/collection/images/elems/lois-eyewear_sun_" + i.toString() + ".jpg"
    });
    views.push(view);
}
$.scrollView.views = views;
