How to turn UglifyJS OFF

I am trying to wrap my head around all the new features of Alloy which means I get to learn CommonJS, Node.js, Underscore.js, and Backbone.js. To accomplish this, I need to do a LOT of code-stepping to see how each line impacts everything, and UglifyJS is making this more trouble than ANY benefit I might get out of its use.

How do I turn off the automatic "uglifying" of my code while I am developing?

  • Uglifyjs does a little more behind the scenes than you might be aware. It doesn't just minify and "uglify" code, it makes a number of performance optimizations and makes core functionality of Alloy possible. Specifically things like the construction of the runtime controllers and compile time checks based on target platform or deploy type (development, test, production) all rely on uglifyjs's abstract syntax tree parsing methods. While uglifyjs can't toally be disabled, we have disabled all of the "uglify" features and more aggressive comporessions methods so that the generated code is far more readable.
