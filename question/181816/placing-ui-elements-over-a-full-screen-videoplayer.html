Placing UI elements over a full screen videoplayer

Hey there guys, I'm trying to place a button over a fullscreen video that is displayed as soon as my app loads.

I already tried playing with zIndex properties, and adding the button element to my video player, and main window as well… No success so far.

Does anyone knows how to do this? 

var bgMovie = Ti.Media.createVideoPlayer({
    url:'bg-movie.mov',
    background:'#111',
    movieControlMode:Ti.Media.VIDEO_CONTROL_NONE,
    scalingMode:Ti.Media.VIDEO_SCALING_MODE_FILL,
    repeatMode:Titanium.Media.VIDEO_REPEAT_MODE_ONE,
    autoplay:true,
    fullscreen:true,
    setZIndex:-100,
    zIndex:-100
});


bgMovie.add($.enterButton);
$.index.add(bgMovie);
$.index.open();

Im using alloy, the button style is the following:

"#enterButton":{
   bottom: 100,
   width: 200,
   height: 40,
   borderColor:'#FFFFFF',
   borderWidth: 1,
   borderRadius : 10,
   zIndex:100
}

thanks in advance!
