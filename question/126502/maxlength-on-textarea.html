Maxlength on TextArea

Is there a way to set a MaxLength on a TextArea (not the html control in a webview, the Ti control)? I know I could setup an event to capture key strokes and then remove them once they go past the length, and then also figure out how to handle a paste which could go over, but come on, we shouldn't have to reinvent the wheel…

  • This actually worked best for me so far:

        textField.addEventListener('change', function(e){        
        if (e.value.length > 9) 
            e.source.value = e.source.oldValue;                
        else
            e.source.oldValue = e.value;
    });

    You can set a listener on the textArea 'change' event (each time the characters change) and do a character count to force a textArea.value, if it goes over a given length. (To chop it off each time)
