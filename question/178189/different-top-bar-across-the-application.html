Different top bar across the application

I am developing iPhone app using the classic way, my SDK version is 3.1.3 GA in the home screen and splashscreen the top bar (where the battery and carrier is displayed) is not showing while in all other windows it's showing, I need to fix it across the app because it looks inconsistent, any idea ?

    Hello Nohamed : You can fix this issue by adding following properties to your window.
    navBarHidden : true,
    tabBarHidden : true,
    fullscreen : true,
    you can also set this property in tiapp.xml file for splash screen.
