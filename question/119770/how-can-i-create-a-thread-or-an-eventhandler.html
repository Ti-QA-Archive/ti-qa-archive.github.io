How can i create a thread or an eventHandler

I need to 'watch' my app so in certain condition my app does something instead of something else….

basically:

i have a webView and i want to check the url the webview is pointing to every once in a while. I don't want to use neither of the events that the webview element can handle because for some unknown reason they all disable it at all. you can't fallow clicks n stuff…

i want to create a thread or something that every 2 seconds checks for the webview url.

any idea ?
thanks in advance

    Set an interval for every 2000 milliseconds using the built-in method: setInterval
    (There's more info on google)

    You can then access the url of the webview through it's url property and monitor this with the function you assign to the interval.

    Remember to clear the interval when you're finished (add a close event listener to the window if necessary)!

    Good luck :)
