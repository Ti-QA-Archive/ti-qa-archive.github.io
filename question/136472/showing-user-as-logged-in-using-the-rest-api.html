  • Well it depends on your system, if you have an apiKey/token authentication you could retrieve that through the httpClient and if the response is successful you can then open the window containing the secret stuff
    e.g.

    
var httpClient = Ti.Network.createHTTPClient({
    onload: function() {
        var response = JSON.parse(this.responseText);
        if(response.success && response.apiKey) {
            ... // 

            var win = createWindowWithSecretContent()
            win.open();
        } else {
            alert("Login failed");
    }
});

    some kind of that.
