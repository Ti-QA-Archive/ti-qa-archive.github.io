How to know if a window is open?

I have a tabgroup with windows, and in one of the tabs, I wrote this:

var one_tab = Titanium.UI.createTab({
   window: Titanium.UI.createWindow({
      backgroundColor: '#fff',
      onload: function() {
         // do something
      }
   })
});

Basically I want the window to do something when it's loaded but apparently, windows have no onload attributes. I know I could just put a url: &#39;some_folder&#x2F;some_file.js&#39; inside the window JSON but using an external JavaScript will prevent me from using some variables defined in the scope of the tab above. Help?

  • Yes, I miss "isOpened" function of window object, too.

    Maybe you can use something like this:

    function Window(options) {
    var win = Ti.UI.createWindow(options);
    win.isOpened = false;
    win.addEventListner('open', function() {
        this.isOpened = true;
    });
    win.addEventListner('close', function() {
        this.isOpened = false;
    });
    return win;
}

    Now you can read window property to see if it is opened.

    var myWin = Window({ modal: true });
Ti.API.info(myWin.isOpened);
myWin.open();
Ti.API.info(myWin.isOpened);

    (code is not tested, but it can give you an idea :) )

        this.isOpened = true;

    win.addEventListner('blur', function() {
        this.isOpened = false;
    });
    return win;
}
