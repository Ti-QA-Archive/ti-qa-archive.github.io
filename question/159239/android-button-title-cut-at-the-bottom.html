android: button title cut at the bottom

Hi, in an android app, I'm just creating a Button with a simple title but it doesn't fit the button's size. How could I fix it?
I'm using Titanium SDK 3.1.3.GA, emulator Android API Google APIs Android 4.2.2, Titanium Studio 3.1.3 on a Mac OSX 10.8.5.

This is the code:

var winks = Ti.UI.createWindow({
        backgroundColor:'#ffffff',
        navBarHidden : false,
        title:'title'
});

var b1ks = Titanium.UI.createButton({
        title:'I am a Button',
        height:40,
        width:200,
        top:10
});
b1ks.addEventListener("click", function(e) {
    winks.close();
});
winks.add(b1ks);

If someone could explain me how to attach here an image, I will show you the button.

Thanks in advance for help!

  • Hi Daniele , set button's height Ti.UI.SIZE. for ie: 

    var b1ks = Titanium.UI.createButton({
        title : 'I am a Button',
        height : Ti.UI.SIZE,
        width : 200,
        top : 10
});

  • Hi Daniele Grassi,

    You can use Titanium.UI.SIZE property in a button height and width to fix this problem and can set background Image like this code just change image location. 

    var b1ks = Titanium.UI.createButton({
        title : 'I am a Button',
        height : Titanium.UI.SIZE,
        width : Titanium.UI.SIZE,
        backgroundImage : 'image location',
        top : 10
    });
