  • If you use HTML with CSS in a webView, you can pretty much make anything you like.

    Example: http://www.stillbreathing.co.uk/experiments/gantt/

    Recreating this using Titanium.UI components is however slightly more complicated.

    I would go for the solution to use Titanium writing the html/css needed to show the Gant Charts as you want locally in the appCache or directly to the webView. Seems to be the shortest route to achieve the goal for me.

    Hope this helps,
