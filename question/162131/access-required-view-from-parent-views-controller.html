Access required view from parent view's controller

Inside the Alloy app is it possible to access required view from the parent view's controller?

Here is a simple example:

index.xml

<Alloy>
    <Window>
        <Button onClick="setBgColor" title="Change Background" />
        <Require type="view" src="some_view" />
    </Window>
</Alloy>

index.js

function setBgColor() {
    $.someView.setBackgroundColor('red');
}

some_view.xml

<Alloy>
    <View id="someView" />
</Alloy>

As you can see someView is an id of a view that is required inside index.xml. I'm trying to access that view from index.js.

When I click the button error is thrown instead of change the background color…

  • Add an ID to your require tag

    <Alloy>
    <Window>
        <Button onClick="setBgColor" title="Change Background" />
        <Require type="view" src="some_view" id="requiredSomeView" />
    </Window>
</Alloy>

    then 

    function setBgColor() {
    $.requiredSomeView.setBackgroundColor('red');
}

    • In some_view.js, add:

      exports.setViewBackground = function(id, color) {
   $[id].backgroundColor = color;
};

      Then in index.js

      var sView = Alloy.createController('some_view');
sView.setViewBackground('someView2', 'red');

    • Agreed, I generally wouldn't both require the view in via the XML and then instantiate it in the code. I'd just instantiate it in the code and add the child to the parent view. But there's no reason you couldn't do both.

      Require and in controller

      index.xml

      <Alloy>
    <Window>
        <Button onClick="setBgColor" title="Change Background" />
        <Require type="view" src="some_view" />
    </Window>
</Alloy>

      some_view.xml

      <Alloy>
    <View id="someView1" />
    <View id="someView2" />
    <View id="someView3" />
</Alloy>

      index.js

      var sView = Alloy.createController('some_view');
sView.setViewBackground('someView2', 'red');

      some_view.js

      exports.setViewBackground = function(id, color) {
   $[id].backgroundColor = color;
};

      Only in controller

      index.xml

      <Alloy>
    <Window id="index">
        <Button onClick="setBgColor" title="Change Background" />
    </Window>
</Alloy>

      some_view.xml

      <Alloy>
    <View id="someView1" />
    <View id="someView2" />
    <View id="someView3" />
</Alloy>

      index.js

      var sView = Alloy.createController('some_view');
$.index.add(sView.getView());
sView.setViewBackground('someView2', 'red');

      some_view.js

      exports.setViewBackground = function(id, color) {
   $[id].backgroundColor = color;
};
