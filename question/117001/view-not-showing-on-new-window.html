View not showing on new window

Just started working with appcelerator for mobile development, so this may be a simple question. I am trying to develop an app that has multiple windows. When I create the new window I can add buttons directly to the window, but I can't add a view. The view never shows up, just a blank window. Here is the code where I launch the new window and add the view.

w_ViewSnappy = Titanium.UI.createWindow({
    backgroundColor:'#fff'
});        

xView = Titanium.UI.createView({
    id:'footerview22',
    height:50,
    width:'100%',
    borderRadius:0,
    backgroundColor:'#336699',
    opacity:1,
    bottom:0,
    touchEnabled:false
});    

w_ViewSnappy.add(xView);
w_ViewSnappy.open();

Thanks for any help.
