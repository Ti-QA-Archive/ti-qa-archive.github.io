'Cloning git repo' has encountered a problem.

The workspace log file shows the following message:

!ENTRY com.aptana.core 4 128 2012-06-26 15:58:44.123
!MESSAGE Cloning into C:\Titanium\workspace1\Todo List…

I have the corporate proxy setup in Titanium Studio - all other network access is working fine. I see Appcelerator installed 'portable git'. Do I need to update a config file with the proxy info to get Git to work?
