    • Thanks Priyanka,

      This TiSDK 3.0.2 is a bit obsolete. You may want to have a try at 3.1.3.GA which is latest stable and fixes numerous problems while building apps to Android emulator.

      To update just: go to Ti Studio Help option menu > Install Titanium SDK > Install from Update stream > 3.1.3.GA.

      Also, have latest Ti Studio. Go to Help option menu > Check for Titanium Updates … (install what Studio IDE prompts).

      Restart Studio, and Clean build to start over. If problem persists please post OS information and Studio Diagnostic log. You can also retrieve it by clicking the Studio Help menu > Studio > Run Diagnostic …

      Just as a side note, please post your feedback as comments instead of adding an answer. It will allow to keep track while narrowing down the problem.

      Thanks,
