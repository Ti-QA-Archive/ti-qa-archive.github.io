Yql query in titanium is not giving me the expected result

Hi

I am trying to retrieve the xml data from rss feed using yql query.Below is the link for rssfeed.Actually the data which i want to retrieve is the url of mp3.But when i try this query on yahoo console.It gives me null in the link tag.I am unable to figure out the problem.

http://www.stationcaster.com/stations/wrks/rss/?c=3041.

My yql query is as follows

select content from html where url="http://www.stationcaster.com/stations/wrks/rss/?c=3041" and xpath="//link"

Thanks

    You have to use xml. 

    select content from xml where url="http://www.stationcaster.com/stations/wrks/rss/?c=3041" and xpath="//link"

    On the other hand you have problems with yql:

    • timeout 10 sec.
    • a part of web sites denied access from yahoo

    A better idea is to use httpclient and convert the xml result to object with xml2json-lib.
