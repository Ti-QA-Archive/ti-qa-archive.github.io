  • AFIK, iOS only supports turning on the LED in conjunction with a videocapture mode. That does not necessarily mean you have to store the video, or even have a preview window. But only the video camera can turn it on. See this StackOverflow thread for sample code on how to do it from native iOS. You'd have to put this in a module though to perform it from Titanium.
