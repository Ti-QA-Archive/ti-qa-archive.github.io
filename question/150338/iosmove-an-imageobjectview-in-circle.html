[iOS]Move an image(object,view..) in circle

hello all.
I want to do is this: I have an image that has to be able to move only in a circle, according to the drag on the window. Practically according to drag that run on the window (which can be in all positions of course), my image must also move, but not in all directions, but only in a circle.
I hope I was clear!
Thank you!

  • It is a true challenge. If the radius is fixed you can only rotate a png with transparency. Please note: on iPhone you can only rotate +-180°. To realize a true rotation you need a special module named path.animator.

    Hope this helps.

    Rainer

      — commented April 2nd 2013 by Rainer Schleevoigt 

    • var foo =  require('path.animator').createView({
    backgroundColor: '#f00',
    width: 100,
    height: 100
});
foo.addEventListener('click', function() {
    foo.rotate({
        angle: 720,
        duration: 3000,
        delay: 1000,
        timingFunction: Ti.UI.iOS.ANIMATION_CURVE_EASE_OUT
    });
});

      Its all.
