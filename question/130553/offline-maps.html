    • Openlayers allows you to set markers, so all you would have to do is to get the gps position form the device and set a marker "your_position" using this values.

      About the map tiles simply make some tiles for the map area (or save them from the net ) and provide to openlayers where to get them from.

      As I told you, is a little frustrating but doable.
      Take a deeper look at openlayers to see how it works.
