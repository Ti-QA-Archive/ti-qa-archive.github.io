  • Hi

    It looks like you are trying to send two arrays of data; id & value. I can only assume you hope to loop those arrays in PHP. If this is the case you need to parse the data first;

    Example of payload object.

    var payload = {
    "myArray[0]" : 1,
    "myArray[1]" : 2,
    "myArray[2]" : 3
};

    This passes myArray as an array PHP can iterate.

    So your data will need to be passed in the same style;

    var payload = {
    "id[0]" : 1,
    "id[1]" : 2,
    "value[0]" : 10,
    "value[1]" : 20
};

    Create a simple loop to create the restyled array before passing it to the send method.

    • Unfortunately you solution will simply create an id & value properties - which will equal the last one.

      You need to be a little more creative. This code will encode the array as it need to be;

      var data=[{"id":"1","value":"10"},{"id":"2","value":"20"}];
var payload = {};
for(var i=0;i<data.length;i++){
    payload['id[x]'.replace('x', i)] = data[i].id;
    payload['value[x]'.replace('x', i)] = data[i].value;
}
Ti.API.info('payload:\n' + JSON.stringify(payload, null, 4));
