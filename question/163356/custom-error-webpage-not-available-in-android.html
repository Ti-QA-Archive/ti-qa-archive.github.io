  • Hello Mahesh Rocks,

    Thanks for your question. Here I have added my sample code for your issues. Before running the code please check your wifi connection and make sure it's enabled. It may help to you. If the connectivity is available, above will be displayed, else show custom dialog for No connection available.

    Check the attachment

    Test Code:

    if(Titanium.Network.networkType == Titanium.Network.NETWORK_NONE){ 
          Ti.API.info("no internet connection");
          var alertDialog = Titanium.UI.createAlertDialog({
              title: 'WARNING!',
              message: 'No Internet Connection in your Device.',
              buttonNames: ['OK']
            });
         alertDialog.show();
       }
  var webview = Titanium.UI.createWebView({url:'http://www.google.com'});
  var win = Titanium.UI.createWindow();
  win.add(webview);
  win.open({modal:true});
