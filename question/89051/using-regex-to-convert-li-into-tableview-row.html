Using Regex to convert <li> into TableView row

EDIT: Post title should read…

"Using Regex to convert &lt;li&gt; into TableView row"

I'm pulling data from a WordPress site into my app using JSON.
Quite logically, posts stored in my database contain tags like &lt;ul&gt; and &lt;strong&gt;. For example:

<ul>
<li>Item 1<\/li>
<li>Item 2<\/li>
</ul>

I want to extract "Item 1" and "Item 2" and place them in a TableView row.
Normally I would just use a WebView to display HTML, but I need usable rows of data.

I already have a half-working expression: 

\<li\>(.*?)\<\\/li\>

This grabs the contents of the list-item, but the &lt;li&gt; tags too.

Where would I go from here?
