  • Provided your new system has the Titanium SDK/Studio installed, all you should need to do is import the Titanium project into Titanium Studio and everything should be fine. This is a pretty common workflow when collaborating on projects. For example, it doesn't matter whether a Titanium project on github was created on a Mac, Windows, or Linux box. Importing the project should set you up fine.

    If it does not, post the specific problem you are having.

    • To access the project in XCode, do the following:

      • Import the project in Titanium Studio, as stated above
      • Build/run the project for iPhone and/or iPad in the simulator, which will generate your build&#x2F;iphone directory
      • In the build&#x2F;iphone directory, open your project's generated .xcodeproject file in XCode. So if your project was named MyProject, you would open the build&#x2F;iphone&#x2F;MyProject.xcodeproject file in XCode

      Doing that will allow you to run your Titanium app through XCode.

  • Hello,

    you will need to follow the next steps:

    1. Run/compile your project in Ti Studio for iOS.
    2. Let the simulator open and see your app.
    3. Close the simulator.
    4. Open Terminal
    5. Go to yourproject directory
    6. Execute this (in the terminal, adapt to the SDK version you using):
    /Library/Application\ Support/Titanium/mobilesdk/osx/1.8.1/iphone/transport.py .

    Then go to build/iphone directory. Copy the project and share it with anyone, even if they don't have Titanium.
