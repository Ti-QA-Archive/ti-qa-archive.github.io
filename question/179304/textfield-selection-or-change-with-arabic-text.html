  • Hello Arafah,

    We have tested this issue for Arabic text inside textfield and try to select sub of this text and It's working as we expect. Try to use normal textfield code from the Titanium document TextField.

    App.js

    var win = Ti.UI.createWindow({
  backgroundColor: 'white'
});

var textField = Ti.UI.createTextField({
  borderStyle: Ti.UI.INPUT_BORDERSTYLE_ROUNDED,
  color: '#336699',
  top: 10, left: 10,
  width: 250, height: 60
});

win.add(textField);
win.open();
