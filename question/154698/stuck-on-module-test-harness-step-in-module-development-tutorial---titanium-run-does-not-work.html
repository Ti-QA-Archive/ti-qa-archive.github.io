  • I've got this error too and in the new documentation (3.x) the executable titanium.py is mentioned.
    This doesn't work either in my installation so I searched for it and found it under:

    /Users/<yourusername>/Library/Application\ Support/Titanium/mobilesdk/osx/3.1.2.GA/titanium.py

    If you're in your module directory, you have to run the test as follows:

    /Users/<yourusername>/Library/Application\ Support/Titanium/mobilesdk/osx/3.1.2.GA/titanium.py run

    The better way is to add the path to your PATH environment variable in .bash_profile (located in your home directory):

    export PATH=/Users/<yourusername>/Library/Application\ Support/Titanium/mobilesdk/osx/3.1.2.GA:$PATH

    So this has worked for me.

    Regards,
    Christian
