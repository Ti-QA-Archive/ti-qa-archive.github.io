Can't hide/show tab group within navigation group

I've spent many hours trying to figure this out and believe this to be a bug, as tabBarHidden=false and then tabBarHidden=true on the currentWindow and/or base window of navigation group should show/hide the tab bar. It doesn't.

I found a hack at TIMOB-1621:

var thisWindow = Titanium.UI.currentWindow; 
thisWindow._base.tabBarHidden = true;
thisWindow._tabGroup.setActiveTab(1); // hack to hide tabs
thisWindow._tabGroup.setActiveTab(0); // and update the screen
Ti.API.info("tabBarHidden=" + thisWindow._base.tabBarHidden);

Basically, you pass in your base window and tab group to subsequent windows, and do the operations above. This sloppy solution works to hide the tab bar.

I was originally just passing the base window and setting tabBarHidden to true on it in subsequent windows, but the tab bar wouldn't hide unless I switched tabs and returned to the tab I was on (so annoying). The hack above does this switching of tabs programmatically.

The problem for me now is that I can't get the tab group to show again after hiding. (e.g. go back in the navigation group). This doesn't work:

var thisWindow = Titanium.UI.currentWindow; 
thisWindow._base.tabBarHidden = false; 
thisWindow._tabGroup.setActiveTab(1); // hack to hide tabs
thisWindow._tabGroup.setActiveTab(0); // and update the screen
Ti.API.info("tabBarHidden=" + thisWindow._base.tabBarHidden);

Ideally, the bug would be fixed and toggling tabBarHidden would work (preferably on the current window), but I'm desperate for any solution at this point.

PLEASE HELP! :)
