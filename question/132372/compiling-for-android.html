  • no, actually android simulator takes more time then iPhone simulator….
    but if you want fast compilation then set this line in the tiapp.xml file then fully rebuild the app and run again but sometimes this fast dev gives more pain. so my advice is to let the android compiler compile the app as it does.

    <property name="ti.android.fastdev" type="bool">true</property>

    so if you sometimes get something weired by setting this line just set false instead of true
