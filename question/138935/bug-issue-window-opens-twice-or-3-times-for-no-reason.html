  • Example:
    If you want to open a new window from table view after clicking and you build the content of new window dynamicly, then time to time the window is opened more the once. I think in the time of building the new content the event more then once fired.

    Workaround: in a state variable you save the state and avoid twice firing:

    var active = false;
tableView.addEventListener('click',function(_e){
    if (active==true) return;
    active = true;
    setTimeout(function(){active=false},500);
    var win = require('mywindow').create(options);
    win.open();
});

    Hope this helps
