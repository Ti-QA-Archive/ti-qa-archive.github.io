    • You have not said with the Alloy framework or not, so here is the principle;

      • Add View "parent"
      • Add Label "lbl1" to parent
      • Add Label "lbl2" to parent
      • Add ImageView "image" to parent

      Then add the event listener to the parent - all events are bubbled up.

    • Yes I was trying to make a point ;-) check the note I am about to add the comments for both you and others that follow - the more details - the better the answer. Thus my initial answer was very simple.

      var view = Ti.UI.createView({
    action: 'view',
    height: Ti.UI.FILL,
    layout: 'vertical',
    width: Ti.UI.FILL
});
var lbl1 = Ti.UI.createLabel({
    action: 'lbl1',
    height: Ti.UI.SIZE,
    text: 'Label 1',
    width: Ti.UI.SIZE,
});
var lbl2 = Ti.UI.createLabel({
    action: 'lbl2',
    height: Ti.UI.SIZE,
    text: 'Label 2',
    width: Ti.UI.SIZE
});
var img = Ti.UI.createImageView({
    action: 'img',
    height: Ti.UI.SIZE,
    image: '/images/example.png',
    width: Ti.UI.SIZE
});
view.add(lbl1);
view.add(lbl2);
view.add(img);
view.addEventListener('click', function (e) {
    switch (e.source.action) {
    case 'lbl1': 
       alert('Clicked lbl1');
       break;
    case 'lbl2': 
       alert('Clicked lbl2');
       break;
    }
});
