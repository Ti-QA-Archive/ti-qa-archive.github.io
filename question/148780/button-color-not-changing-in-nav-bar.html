  • Hi

    If you just want a blue button which is considered the done button then the answer Ishan gave you is the right one.

    If you want to play with colour and also make it something other than blue - there is a way.

    • TabbedBar

    If you add one of these instead of a button and only add one segment you can have any colour you wish. To add the click colour effect - set the initial index to -1 and then also set the index to -1 in the click event. This gives it active/inactive visual states. I use this in a commercial to allow red, green and blue buttons.
