Module build and find

I am trying out the stelford and tzmartin Calendar modules, but with both of them I can build them using the build.py, but when i try to use them I am given the error of can't find the file.

so what I did:

  1. build the py file
  2. copy the zip to the Application Support/Titanium folder
  3. create a new application
  4. add the line 
    var ticalendar = require("com.ti.calendar");
    to the app.js

  5. add the lines

    <modules>
      <module version="0.1">com.ti.calendar</module>
</modules>

    to the tiapp.xml

  6. run the application (at this time the file is unzipped and placed into the module folder)

  7. get the error [ERROR] Script Error = Couldn't find module: com.ti.calendar.

  • I think you just put the library in the wrong directory or you didn't decompress the zip file.

    Make sure it's decompressed (just the com.ti.calendar folder) and inside the iphone directory here:

    &#x2F;Library&#x2F;Application Support&#x2F;Titanium&#x2F;modules&#x2F;iphone

    If that doesn't work, try putting it in your project's module directory here:

    (PATH TO MY PROJECT)&#x2F;modules&#x2F;iphone

    Let me know if that works for you. I compiled and have the modules working just fine on my computer.

  • Open titanium studio, preferences->Titanium Studio->Titanium and check where the Titanium SDK is installed. Let's call this directory (TITANIUM SDK DIR). Now make sure you unzip the module and put it in the (TITANIUM SDK DIR)&#x2F;modules&#x2F;iphone directory so in the end you should have: (TITANIUM SDK DIR)&#x2F;modules&#x2F;iphone&#x2F;com.ti.calendar

    Let me know if that works for you. That worked for me.
