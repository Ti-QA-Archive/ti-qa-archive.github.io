    Hi

    If you actually mean tab bar then the link actually says:

    About 50 x 50 (maximum: 96 x 64) // (for Retina)

// and

About 25 x 25 (maximum: 48 x 32) // (for Non-Retina)

    Also make sure when you create the images, you label them correctly by adding the '@2x' suffix to the name for Retina images like so image@2x.png and image.png for standard.
