Loading image in ti.paint from mobile gallery

I tried to search everywhere for this but no luck.

is it possible to load any image with ti.paint and make the user be able to draw in top of the image? (e.g. mark notes)

    This is the way to do it:

    // Have a base view
var view = Ti.UI.createView({ 
// Dimension as you wish
});
// Add your image
var image = Ti.UI.createView({ 
// Dimension it and source it with the image
});
view.add(image);
// Add the paint view on top of the image
var paint = require("ti.paint");
var paint_view = paint.createPaintView({
// Dimension and configure
});
view.add(paint_view);

    The advantage is that then you can do this to export the image with annotations to a new image/blob:

    view.toImage();
