Error while creating new "Map Annotation"

I'm getting the following error:
cannot call method "createAnnotation" of undefined …

var apple = Titanium.Map.createAnnotation({
    latitude:37.33168900,
    longitude:-122.03073100,
    title:"Steve Jobs",
    subtitle:'Cupertino, CA',
    pincolor:Titanium.Map.ANNOTATION_GREEN,
    animate:true,
    myid:2 // CUSTOM ATTRIBUTE THAT IS PASSED INTO EVENT OBJECTS
});

  • Accepted Answer

    If you are referencing Titanium.Map module in your project for the first time, then Titanium may not have caught that it needs to include this module in the build. Try clearing the build directory and rebuilding. /projectname/build/iphone (or android) <- clean out the directory and rebuild.
