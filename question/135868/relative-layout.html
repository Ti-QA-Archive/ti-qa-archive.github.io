Relative Layout

Hello to everybody, I was wondering if there's a way to put a view (TextField, Label, etc.) beside, above or below another view. Like in Android you can do "toRightOf: txtSth" when you have a RelativeLayout.

Thanks!

    You can place objects (labels, texfields, etc) inside another view that has the layout property set to 'vertical' or 'horizontal'.

    for example:

      var win = Ti.UI.createWindow({layout:'vertical'});
  var lab = Ti.UI.createLabel({text:'label 1'});
  var lab2 = Ti.UI.createLabel({text: 'label 2'});

//will place labels directly beneath each other. You can set properties like lab2.top=20 which would set the top relative to the previous label.
win.add(lab);
win.add(lab2);
win.open();
