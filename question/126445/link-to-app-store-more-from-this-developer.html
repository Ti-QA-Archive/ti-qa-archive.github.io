  • The link itself on the question is problematic…

    For the developer profile you can use this:

    https://itunes.apple.com/artist/idYOUR_ARTIST_ID?at=10l6dK

    You can get YOUR_ARTIST_ID from your iTunes Connect or from the link on one of your apps (there is a "View more by this developer" and copy the number after the "id" on the link itself).

    Pay attention there is no country code there that could make the link not to work on other stores (like german on this example).

    The "at=…" would make the App Store opens with your developer account. not using that could open first a web view (safari) then user would have to click there to really open the store.
