    Hi Amasidda,

    I have also face same issue.. but if i set image by directly without using Ti.Filesystem.resourcesDirectory.

    e.g

    $.imgView.image = &#39;&#x2F;images&#x2F;indicator.png&#39;;

    here images is my image folder in assets

    it's works for me… hop it's work for you.. :)
