Setting a Button's Font Properties with Alloy Styles

How can I set the font properties of a button using Alloy styles? I am trying the below, but the styles do not apply to the button (as they do in the Label control). Is this also possible inline within the button control, and if so, how so?

Thanks in advance for any help.
Chris

"Button": {
  font: {
    fontSize: 18
    fontWeight: 'bold'
  }
}

  • You're missing a comma separating the fontSize and fontWeight properties. Should be:

    "Button": {
  font: {
    fontSize: 18,
    fontWeight: 'bold'
  }
}

    • I am have this issue as well. If the buttons at in the Items section the styles are ignore, if the items section is removed the style is applied.

      <Toolbar id="toolbar" class="Toolbar">
    <Button>Today</Button>
    <Button>Annual</Button>
    <Button onClick="openSettingsWin">Settings</Button>
</Toolbar>

      In this case styles work.

      <Toolbar id="toolbar" class="Toolbar">
    <Items>
        <Button>Today</Button>
        <Button>Annual</Button>
        <Button onClick="openSettingsWin">Settings</Button>
    </Items>
</Toolbar>

      In this case styles DO NOT work.

      STYLE for reference

      "Button": {
    style: Titanium.UI.iPhone.SystemButtonStyle.PLAIN,
    color: "FF3B30",
    font: {
        fontFamily:'Helvetica Neue',
        fontSize: '18dp',
        fontWeight: 'Normal',
        textStyle: Titanium.UI.TEXT_STYLE_BODY
    }
}
