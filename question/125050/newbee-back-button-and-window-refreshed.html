  • The back button should have an event listener that you can make do pretty much whatever you want. I think the standard back buttons simply close the current window.

    Also, creating a new window doesn't close others as far as I know. It would just open a window on top of the current window. Closing it would simply expose the original again.

  • Found the answer here. So easy, I can't believe it but would have never guessed it.

    In short just add a listener to your window with the function to refresh when it's focused. 

    win.addEventListener('focus', refreshWindow);
