  • To disable completely ESET firewall, I had to go in Advanced Setup / Personal Firewall / System Integration and select "Personal firewall is completely disabled" and then reboot. Then Titanium installed correctly.
    Selecting "Personal firewall is inactive" is not enough.

  • I'm using Windows 7 Home Premium. After disabling both my firewall and anti-virus I as well could not get Titanium to install. I was getting to the same point in the installation process and it would fail. I received an error stating "Could not query info: Invalid HTTP Status Code (0)". Finally what I did in order to get Titanium to install was: Control Panel -> Network and Internet -> Internet Options -> Connections -> LAN Settings -> and then I unchecked the "Automatically detect settings" and applied the new settings. After I made those changes and while my firewall and anti-virus were still off ( not sure if having them off made a difference ) I went through the installation process again and I'm not too sure why this worked but it did.
