Alloy ListView with Switch in template

I have a simple ListView using Alloy xml. It contains a switch (checkbox) and a label. How can i get all the items the user has marked (or the underlying data-index or something else) ?

I am using Titanium 3.x anddeploying on Android

simple sample:

<ListView id="myList" defaultItemTemplate="myTemplate">
    <Templates>
        <ItemTemplate name="myTemplate">
            <Switch bindId="checked" value="false"/>
            <Label bindId="title" left="40dp"/>
        </ItemTemplate>
    </Templates>
    <ListSection headerTitle="A Section">
        <ListItem title:text="A Title one" checked:value="false"/>
        <ListItem title:text="A Title two" checked:value="false"/>
        <ListItem title:text="A Title three" checked:value="false"/>
    </ListSection>
</ListView>

  • Hopefully this will get you started in the right direction:

    <ListView id="myList" defaultItemTemplate="myTemplate">
    <Templates>
        <ItemTemplate name="myTemplate">
            <Switch bindId="checked" value="false" onChange="doSwitchMe"/>
            <Label bindId="title" left="40dp"/>
        </ItemTemplate>
    </Templates>
    <ListSection headerTitle="A Section">
        <ListItem title:text="A Title one" checked:value="false" checked:myID='one'/>
        <ListItem title:text="A Title two" checked:value="false" checked:myID='two'/>
        <ListItem title:text="A Title three" checked:value="false" checked:myID='three'/>
    </ListSection>
</ListView>

    function doSwitchMe(e) {
    alert('Switch ' + e.source.myID + ' was clicked and is now ' + ((e.source.value===true)? 'on' : 'off'));
}

    • As pointed out in that ticket, the (should have been obvious to me) solution is to define a custom property for each list item.

      Here's a technique for adding items in the controller. You should be able to adapt this to add them in the XML if you really need to do it that way:

      var items=[];
for(var i=0; i<4; i++){
    items.push({
        title: {text: 'Title one'}, 
        properties: {itemId: 'switch ' +i},
        checked: {value: false}
    });
};
$.mySection.setItems(items);

function doSwitchMe(e) {
    alert('Switch ' + e.itemId + ' was clicked and is now ' + ((e.source.value===true)? 'on' : 'off'));
}

$.index.open();

      Tested, works on both iOS and Android.
