Parsing JSON with hyphenated key names

Just figured this out yesterday and wanted to pass it along in case other devs are struggling with the same issue:

If the JSON object you are parsing has key names with hyphens like this:

{ "customer-list":
  [{"fname":"John", "lname":"Doe", "title":"Code Monkey"},
   {"fname":"Jane", "lname":"Smith", "title":"Project Manager"}]
}

You can't use dot syntax to reference the hyphenated keys:

myData.customer-list[0].fname.text // Undefined variable 'list'

Instead you use this:

myData["customer-list"][0].fname.text // 'John'

Hope it helps.
