Couldn't find module

I'm an Appcelerator newbie and trying to write something very simple to get my feet wet.

I watched Tim Poulsen's "Cross-Platform Javascript Applications" and was playing with the require.

I created a simple two-tab application using the wizard. In /app/alloy.js I have:

var UserObj = require('UserClass');

In /app/UserClass.js I have:

var User = function() {
Ti.AP.info("Instantiating the user class.");
this.FirstName = 'Timothy';
this.LastName = 'Johnson';
this.Username = 'tjohnson';
};

exports.User = User;

When I build and run it complains of not being able to find UserClass…I'm sure I'm doing something waaay stupid but can't figure it out.
