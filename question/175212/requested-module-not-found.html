Requested module not found

I know this seems like a really obvious question, but I've used lots of modules before, and Titanium can't seem to find this module.

I just purchased the Android SmsSender module, installed it (both locally and globally), but when I require it, I get the following error:

V8Exception: Exception occurred at ti:/module.js:280: Uncaught Error: Requested module not found: co.il.webe.smssender

I've installed and used several modules in the past, so I'm pretty confused as to what's gone wrong here. This is what my code looks like:

var sms = require('co.il.webe.smssender');
sms.Send(num, text);

Pretty standard, and I've checked and double-checked that I've got the module name correct. Any suggestions for what may be going wrong here?

The only lead I've found is this line from the build output:

Module co.il.webe.smssender version 1.1 does not have a main jar file

Before that, the build output seems to have found the module:

Found Titanium module id=co.il.webe.smssender version=latest platform=android deploy-type=test path=&#x2F;Users&#x2F;ap&#x2F;code&#x2F;mobile&#x2F;parse_model&#x2F;modules&#x2F;android&#x2F;co.il.webe.smssender&#x2F;1.1

Thanks,
