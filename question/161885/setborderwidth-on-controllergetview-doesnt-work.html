setBorderWidth on controller.getView() doesn't work?

Hi I'm writing a piece of code where when an entry gets added to a list, the borderWidth of the previous entry must be set to "1px" and the new entry gets its borderWidth set to "0dp". Whatever I try though, the borderWidth stays at zero, while I can change the backgroundColor. So I know I've selected the correct element, the border just doesn't get set.

the following DOES change the backgroundColor, but it DOESN'T change the borderWidth. Am I doing something unbelievably stupid, or is it just broke?

if($[previousRow]) $[previousRow].getView().setBorderWidth("10px");
if($[previousRow]) $[previousRow].getView().setBackgroundColor("#0F0");

$[previousRow] gets generated as follows (this is what basically happens when a row gets added):

var versionId = e.row.versionId;
$[versionId] = Alloy.createController("Row");
$[versionId].getView().setBorderWidth("0dp");
if($[previousRow]) $[previousRow].getView().setBorderWidth("1px");
previousRow = versionId;

but it just doesn't change the borderWidth back to "1px"

I'm on Android by the way, 4.3, using Ti SDK 3.2.0.GA on Windows 8.1
