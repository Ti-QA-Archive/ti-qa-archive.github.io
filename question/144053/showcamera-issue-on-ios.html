showCamera issue on iOS

I'm using Titanium and testing the camera on iOS. I'm getting an issue where code after the camera page is being run before I've gone through the camera page. In the below code, the last line &#39;alert(&#39;Picture uploaded successfully.&#39;) is executed before the camera screen has even opened up.. Any ideas?

    showCamera is an asynchronous event it does not stop the execution context. If you want an alert on the successful capture of an image, put the alert in the success event.
