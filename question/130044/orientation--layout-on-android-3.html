Orientation / Layout on Android 3

My app is designed to only work in portrait mode. I have the orientation locked to portrait in my custom manifest and in each window. When testing on the phone emulator (2.3.3) when the app is launched and the device is in landscape mode the app properly is shown in portrait mode and calls to Ti.Platform.displayCaps.platformWidth return the width as if it were held in portrait mode. However when testing on Android 3.0 the app is displayed in portrait mode but it appears that Ti.Platform.displayCaps.platformWidth is returning the width as if it were held in Landscape mode. Is this an expected behavior?

  • Basically I want to always treat the device as if its in portrait mode so I'm doing:

        if (styles.ui.platformWidth>styles.ui.platformHeight){
        var tmp = styles.ui.platformWidth;
        styles.ui.platformWidth = styles.ui.platformHeight;
        styles.ui.platformHeight = tmp;
    }
