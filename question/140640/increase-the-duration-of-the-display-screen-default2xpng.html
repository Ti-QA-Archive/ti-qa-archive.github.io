    Hi nicolò monili,

    Create window and add imageview and set your default image in that imageView and open that window and on setTimeout function close your splash window and do your procedure further.

    — answered August 7th 2012 by Nitin Chavda
    permalink
    5 Comments

    • var win = Titanium.UI.createWindow();
 var imgSplash = Ti.UI.createImageView({
            image : 'default.png',
            top : 0
        });

win.add(imgSplash);

win.addEventListener("open", function () {
    setTimeout(function()
        {
            win.close();
            //====== open you window or tabGroup here
        }, 5000);
});
win.open();
