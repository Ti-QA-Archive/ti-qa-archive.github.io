  • I answer myself.

    What I did was check my new plist file, since it is XML, the new version of the system and compare it with the current version of my App.

    If the new version is larger than the old one, is that I need to update.

    :D

    The code:

    var xhr = Titanium.Network.createHTTPClient();

xhr.onload = function()
{
    var doc = this.responseXML.documentElement;
    var elements = doc.getElementsByTagName("string");
    var version = elements.item(3);

    alert("The new version is: "+version.text);
    alert("actual version: "+Titanium.App.getVersion());


};

xhr.open('GET','http://url/file.plist');
xhr.send();
