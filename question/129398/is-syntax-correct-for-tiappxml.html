Is syntax correct for tiapp.xml?

Using Ti.sdk 1.7.5

 <modules/>
    <modules>
        <module version="1.0">ti.admob</module>
        <module platform="iphone" version="0.2">com.omorandi</module>
    </modules>
</ti:app>

    Well - you are including the <modules> element twice. Once as a self closing tag <modules /> and then again as a start/end tag <modules></modules>

    I would remove the <modules /> tag
