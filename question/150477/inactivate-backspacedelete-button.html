  • Hello,

    the only way to accomplish this is creating your own keyboard component. This doesn't make any sense for the general component (i.e., the native one). If you want to do it, you can simply create a view with different views (the keys) and then catch each event, reflecting that information in your textfield.

    Best,

    Mauro
