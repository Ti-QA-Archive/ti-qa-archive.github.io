How I made Titanium worked with BlueStack Android Player

I am sorry if I am posting at the wrong place,I accidentally made BlueStack worked with titanium and I fee like sharing it. First of all ,I installed bluestack ,After that,I created a new run configuration ,Create a new configuration and since Titanium will only make you select the android version so that it will create the emulator automatically ,I selected android version 2.2 .After I saved the configuration.After run the configuration for the first time,Android 2.2 emulator will open.After open blueStack. Close the android 2.2 emulator and run your configuration.Dont close bootstrap. If it didnt run from bluestack ,go to run configuration and select the configuration you made,run it from there.All this while,make sure bluestack emulator is running. Thank You
