Dates don't compare correctly in titanium mobile

Take the following code:

var first = new Date("2011-09-17T15:12:04.331Z");
var second = new Date("2011-09-17T19:55:19.269Z");
alert(first < second);

In a browser, it alerts true. In Titanium Mobile, it alerts 0, or false. Why is this? I cannot get the dates in a different format from the source.
