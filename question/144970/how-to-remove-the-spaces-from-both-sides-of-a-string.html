How to remove the spaces from both sides of a string

Hi, I have to enter a name in a textField which can contain one blank space only between the words and it can contain any character, if the user enter any spaces at the beginning or end of the string, it should be automatically removed(just like Trim method). Can anyone help me

    Hi

    A trim method.

    String.prototype.trim = function() {
    return this.replace(/^\s+|\s+$/g, "");
};

    It extends the String object.

    Place this in your app.js but you will want to determine a more suitable place for your project later.

    Use like;

    var strName = txtName.getValue().trim();

    Where txtName is the name of the textfield.
