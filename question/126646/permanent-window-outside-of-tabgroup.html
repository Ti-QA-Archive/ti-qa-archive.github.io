Permanent window outside of tabgroup

Is there a way to have a permanent window outside of tabgroup?

I want to create an audio player window that'll still play the audio while I'm navigating away from the window, but it's not part of the tabgroup.

Example: audible app

I've tried create a modal window, but setting visible: false or playerWin.hide() the navigation bar of that window still there, and the main screen just black… a non-modal window doesn't have the navigation bar (even with navBarHidden: false so it's impossible to navigate away from that window…

thoughts?

    Create a view with the audio player and add to the tab group. 

    tabGroup.add(playerView);

    I did this way in this app (look the button for opening the player, is always in the same position over the windows)
