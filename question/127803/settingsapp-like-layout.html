Settings.app like layout

I want to create TableRows similar to the Settings.app, where there is a label in the left side and a value label in the right side aligned right.

The problem is that I can't figure out how can I avoid overflowing of the two labels on eachother without restricting the with of both of them (e.g. to 50-50%). I want the labels to have dynamic width. If the first label is short, let the second to be long and the other way too.

Is it any layout or technique to do this without realigning them after the window opens (with object.size.width)?

  • Maybe like this?

    // assuming we have row already

var l1 = Ti.UI.createLabel({
    text:'setting one',
    width:'auto',
    left:0,
    textAlign:'left'
});

var l2 = Ti.UI.createLabel({
    text:'value one',
    width:'auto',
    right:0,
    textAlign:'right'
});

row.add(l1);
row.add(l2);
