    • Even longer answer;

      If you are actually after the ability to have the whole screen read out to the user without their interaction then this is not possible using the built in capabilities as the operating system sticks very clearly to the guidelines of those with disabilities to wish to have the text spoken out with a means to control when it starts and when it stops. If the device simply read everything out then the user would not be able to stop at a point and choose to navigate or select that thing. Also if they needed to start again then it would have to read it all over again.

      Have you tried the accessibility settings in iOS or Android and actually used them - they show you how they are used and how they are accessible.

      If you really need to be able to read everything out without taking the navigational need if the user into account - basically a book reading solution then you need to look for text-to-speech as this is not a normal accessibility feature of modern mobile devices.
