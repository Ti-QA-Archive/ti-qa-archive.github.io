Editing the Objective-C Code in Titanium

Is it possible to edit the Objective-C code that is generated from Titanium or does that break Titanium? If so, what are easy ways to trace what classes are for what etc…?

    If you mean can you alter the Titanium SDK… Yes, you can always do this by simply branching on GitHub or downloading and altering the code.

    If your question is can you alter the Object-C of the final build… no… thats not exactly how Titanium works, it still processes JS at runtime, though that JS is compiled into the app, not just loose JS files floating around.
