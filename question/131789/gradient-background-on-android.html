Gradient background on Android

Hi,
I use a lot of gradient background in my windows on IOS.
I have to port the app to Android, but it seems that gradient background is not supported on Android.
What is the best alternative ? Adding an imageview with a gradient image inside then adding all controls to this imageview instead of adding them to the windows ?
Regards

    Use the technique is used on the web. Make an image with the gradient 1px width and use it as backgroundImage for your view (window, whatever ui element has the gradient). It should stretch to cover the whole background.
