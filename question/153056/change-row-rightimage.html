Change row rightImage

Hello,
Im creating dynamically a table with a contents of a json file, here is my code:

        for (var i=0; i<jsonObj.waypoints.length; i++){
        var row = Ti.UI.createTableViewRow({
                color:'white',
                height:'60dp',
                rightImage: 'marker.png'
          });
         var labelUserName = Ti.UI.createLabel({
             color:'white',
             left:10,
            text:i+1+": " +jsonObj.waypoints[i].title 
          });
          row.add(labelUserName);
          tableData.push(row);
     }
     table = Ti.UI.createTableView({
           color:"white",
           separatorColor:'white',
          data: tableData,
          top:130
        });
        win.add(table);

Is it possible to change after creating the table the rightImage of a row?

thank you

  • Accepted Answer

    You can change the rightImage of a Ti.UI.TableViewRow with setRightImage( String rightImage ).

    To retrieve all rows of a Ti.UI.TableView just use getData() - it will return an array of TableViewSections / TableViewRows
