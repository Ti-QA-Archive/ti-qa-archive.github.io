  • Hello,

    Please open a terminal and run android by itself, and share with us the error code. My theory here: in the update, you lost the shared32 bits libs or similar.

    This is an error with your Native Android setup. Please check that and see if that solves the problem or give you more info. In addition to that, I recommend to install the x86 support from android.
