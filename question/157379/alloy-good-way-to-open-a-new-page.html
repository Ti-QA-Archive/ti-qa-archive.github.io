  • Close the window and open the new controller using var
    arg - data that if you needed to send to other view

    var arg = {username:'xyz'}
Alloy.createController('home', arg).getView();

    get this passed data using 

    var arg = arguments[0] || {};

    Look at this documentation for more information - http://docs.appcelerator.com/titanium/latest/#!/guide/Alloy_Controllers
