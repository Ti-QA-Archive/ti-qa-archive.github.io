    Hi

    I have looked at this one a couple of times and I think I am either missing some extra info or some extra code - just not sure which yet.

    So let's start with the basics, a navigation controller is added to a window AND has a window of it own to start with, you can then add child windows to the navigation controller each of which get a back button.

    This example is taken from the documentation but the variable names have been re-written to make more sense and the order of the code is placed so that it should flow a little better.

    To test this create a new default classic project and replace the contents of app.js with the code below.

    // never actually seen as it is filled by the nav controller
var winParent = Ti.UI.createWindow(); 

var winRoot = Ti.UI.createWindow({
    backgroundColor: 'red',
    title: 'Root Window'
});
var navGroup = Ti.UI.iPhone.createNavigationGroup({
    window: winRoot
});

// define child window
var winChild = Ti.UI.createWindow({
    backgroundColor: 'blue',
    backButtonTitle: 'Back',
    title: 'Child Window'
});

// create a button to add to the root window
var button = Ti.UI.createButton({
    title: 'Open Child Window'
});
button.addEventListener('click', function(){
    navGroup.open(winChild, {animated:true});
});
winRoot.add(button);

winParent.add(navGroup);
winParent.open();

    If you can understand this then hopefully we can nail down the issue in your code. It is very likely that you are simply not adding the correct window to the nav controller.
