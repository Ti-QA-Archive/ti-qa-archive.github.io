Prevent a window from going to the foreground while opening

I have a situation where multiple windows are opened at the same time. The order in which they are opened is dependent on responses from my server. Currently the last one that is opened will appear in the foreground. This is not what is supposed to happen in my app.

One window should always be in the foreground and the other in the background. I tried using the zIndex on the windows but everytime the last window to open appears on top.

Anybody familiar with this and has a solution?

  • Hi Wienke, just Use the code like this, if i understood your problem correctly ::

    Titanium.UI.setBackgroundColor('#000');

    var win1 = Titanium.UI.createWindow({
    title:'Tab 1',
    backgroundColor:'#fff'
    });

    var button1 = Ti.UI.createButton({
    title:'win1',
    });
    win1.add(button1);
    win1.open();

    var win2 = Titanium.UI.createWindow({
    title:'Tab 2',
    backgroundColor:'#fff'
    });

    var button2 = Ti.UI.createButton({
    title:'win2',
    });
    win2.add(button2);
    win2.hide();
    win2.open();

    var win3 = Titanium.UI.createWindow({
    title:'Tab 3',
    backgroundColor:'#fff'
