Using Underscore from a Widget

I am trying to use Underscore from a widget and I keep on getting the error: 

Uncaught TypeError: Cannot call method 'extend' of undefined

I have tried using var _ = require(&#39;alloy&#x2F;underscore&#39;)._; and several variations of that but haven't found the right path that works.

Can anyone share the correct path to use for this require in a widget? Thanks!!

    You can use underscore functionality. You don't need to use the require in a widget as this will cause an error. What method from the underscore are you trying to use? Remember appcelerator ships with an older version of underscore.js.
