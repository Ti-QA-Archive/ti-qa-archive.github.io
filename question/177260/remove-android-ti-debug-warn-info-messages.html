  • Change the log level on your console. On the standard Ti Studio, the console window has an icon on the far right that lets you set the log level. Click and change to 'Error' or 'Warning', that will suppress the Info messages. Basically, whatever you set the level to, the message has to match that level or higher to be shown. If you were to set to Warning, then in your listing above, you would only see the two messages with [WARN] beside them. If you set to Error, then none of the messages above would be shown.
