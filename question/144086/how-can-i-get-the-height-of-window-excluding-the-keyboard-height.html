how can i get the height of window excluding the keyboard height

Application type: mobile
Titanium SDK: 2.1.2
Platform & version: Android 4.0.3

when i focus on a textarea i want to get the height of the window visible area excluding the softkeyboard height. i tried just get the keyboard height but didnt find a solution.
the idea is to resize and position some contents in the window after opening the softkeyboard. any ideas?

  • my problem was that when i click on the textarea the entire window gets pushed up. therefore, i was thinking of explicitly managing the window height. but after digging around the API i found that i can set the window "windowSoftInputMode" property to Ti.UI.Android.SOFT_INPUT_ADJUST_RESIZE which solved my problem.
