Python not configured message in project tree

Hi, I've restarted my machine, and out of the bloom I have a "Python not configured." message on top of my project tree in Application Explorer window. I have it only on one specific project and not others. It has a red X sign next to it.

What does it mean? How do I get rid of it?

I'm not using Python in my app!

Thanks

  • I found a solution (I love asking and answering myself - NOT!). I'm not sure if it's the right way to get rid of it but here it is if anyone needs to know in the future:

    In the app root directory there are 2 hidden files (I'm using mac - it might not be hidden on windows or linux). first, I deleted .pyproject file. Then I edited the .project file and removed from it's xml the settings that refer to python. restarted my app and the message was gone.

  • Just click on "Project Explorer",
    Right Click on Error Project > Select PyDev >
    Click on Remove PyDev Project Config
    Close Project and open it
    Done.
