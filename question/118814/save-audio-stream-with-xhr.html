Save audio stream with XHR

Hello everyone, I'm trying to save an mp3 stream and then play it locally, but obviously does not work. Can anyone help me? This is the code:

var stream = Ti.Media.createSound();
var xhr = Titanium.Network.createHTTPClient();

xhr.onload = function(){
    var filename = 'stream.mp3';
    var f = Titanium.Filesystem.getFile(Titanium.Filesystem.applicationDataDirectory,filename);
    f.write(this.responseData);
    stream.url('stream.mp3');
};
xhr.onerror = function(e){
    Ti.API.info('XHR Error ' + e.error);
};
// open the client
xhr.open('GET','http://shoutcast.unitedradio.it:1413/');

// send the data
xhr.send();

setTimeout(function(){stream.play();}, 6000);

I know that I could use the method createAudioPlayer, but unfortunately does not work on the devices of the family of Samsung Galaxy (with Froyo), and I need to make it compatible with these.
Sorry for English.

  • The mp3 local url is not correct.
    You shouldn't call 'stream.mp3', because that would make your app look into your Resources path.
    Instead try something like stream.url(f);

    Something else: why do you play it after 6 secs you call the xhr.send()? you should place that code in the onload listener, waiting the mp3 to be fully loaded.
