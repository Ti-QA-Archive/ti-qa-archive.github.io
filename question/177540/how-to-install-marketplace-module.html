  • Hi Alexander,

    you are trying to connecting using the wrong protocol, of course this cannot work, so the websocket closes.

    instead of "http"

    websocket.open('http://localhost:49892');

    use the "ws" (=websocket) protocol

    websocket.open('ws://localhost:49892');

    Hello, Alexander. Here is the link to your Desire module in Appceleratoer Marketplace LINK

    Click "view Profile"-"Download Now"..The Module will download. Go to your Purchased Product and you will find the module. Click on the module will download a zip file in to your pc.

    Extract the file will be a module folder. Now Put the module to your project directory. In the module there is example folder which contains a "app.js" file which explains how to use the module in your project. Also add the module to your "tiapp.xml" file. on overview page click + in module, you will see the module in the list. add it.
