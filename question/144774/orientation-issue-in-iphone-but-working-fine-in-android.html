  • iOS6? or earlier?

    There are some issues around this which started to happen in iOS6 with orientation and fullscreen movies.

    Try adding the win.orientation settings in the 'fullscreen' 'entering' event.

    
activeMovie.addEventListener('fullscreen',function(e){
    //Ti.API.info('fullscreen event : '+e.entering);

    if(e.entering==true){
        win.orientationModes = [ 
            Titanium.UI.PORTRAIT,
            Titanium.UI.LANDSCAPE_LEFT, 
            Titanium.UI.LANDSCAPE_RIGHT
        ];                            
    } else {
        win.orientationModes = [ 
            Titanium.UI.PORTRAIT
        ];                        
    }
});
