    • backgroundfetch

    This appears to be for the smaller stuff. But in this documentation is does specially note what to do for larger content;

    "If you are downloading large content, initiate the download using a urlSession background download task, then call the endBackgroundHandler method. Listen for the downloadcompleted event to prepare the data for use."

    • URLSession

    This is meant to handle the bigger stuff. I found this site with a better description - ignore the ObjectiveC code though.
