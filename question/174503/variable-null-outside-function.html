variable null outside function

Hello,

I'm very new to titanium and javascript so please excuse me if my question is a simple one.

I have the following lines, can someone please help me understand why console.log(arr); gives me a null value.

sendit.onload = function() {

    json1 = JSON.parse(this.responseText);
    var json2 = _.first(json1,5);

    for ( var i=0;i<json1.length;i++){
            _.each(json1[i].receivers, function(json) {
                // console.log(json);
                arr.push(json);
                // console.log(json1[i].receivers);
            });
        };
    console.log(arr); <----- THIS WORKS
};
console.log(arr); <-------- THIS DOES NOT

Thanks in advance.

    Scope

    If this

    arr.push(json);

    is your first declaration, then your variable can only be seen within the scope of the sendit.onload function.
