    Hi Sylvain,

    Did you try checking your variable before casting?
    Or casting the nb variable to a string with toString() instead of +''?
    Which platform are you targeting? (They have different JS Engines..)

    function BlocInfoFlux  (nb, image) {
    Ti.API.info("nb = "+JSON.stringify(nb)+" typeof nb = "+typeof nb);
    if (!nb) {
        Ti.API.info("nb has invalid value");
        return;
    }
    var nombre = nb.toString();  // or String(nb);
    // now ouput again
    Ti.API.info("nombre = "+JSON.stringify(nombre)+" typeof nombre = "+typeof nombre);
    ....

    if everything fails, post the stringified output of "dataRss" variable

    eg:

    Ti.API.info(JSON.stringify(dataRss, undefined, 2));
