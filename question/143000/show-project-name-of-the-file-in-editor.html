Show project name of the file in editor

Hi guys.

First of all, sorry if the title is not clear because I couldn't come up with better name.

I want to know if it's possible to show project name of the file in editor tabs, I hope you know what I mean. It's something like when you open similar file name, let's say app.js and app.js but they from different projects, the tab in editor will show something like this, "app.js | Project 1" and the other one would be "app.js | Project 2".

Since I usually copy some parts of code to other projects, sometimes I edited the code in wrong projects. I just want to avoid this kind of mistake in the future. If anyone maybe has tips to avoid this, I would love to try. Thanks in advance =)

  • Many people here gave more detailed answers, so I can't give more details than that.

    But a shorter answer would be:

    The project's name will be displayed in the tab only when two files with the same name are opened. There is no setting to change that. Of course, you could customize Studio using Rubles. That would be the best approach.
