Is there any way to retrieve the UUID of an iPhone using Titanium?

The API documentations says that Titanium.Platform.getId() returns the UDID (unique device ID) for android but for iOS this is a unique identifier for a particular installation of the application. It also says that Apple has restricted access to UUID.

However, if I need to retrieve a unique device ID for an iPhone, is there any way to do it? Or is it not possible at all?

  • Apple no longer allows developers access to the devices UUID. For the documentation check this: Deprecated UIDevice Methods.

    Only devices running iOS 2.0 to iOS 6.1 can have their UUID pulled by the developer.
