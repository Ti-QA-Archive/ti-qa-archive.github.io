    • Your English is fine, we had the same problem.

      Developer Options in settings

      • Turn on the USB debugging option
      • Set developer mode to true (if available)

      In Applications (2.x) or Security (3.x and higher)

      • Allow unknown sources

      When you connect to your computer

      • Choose media device
      • Not camera or similar

      I also run on macs and have several different Android Phones and tablets and once these settings are selected it works fine - MUCH better than emulator.

    • OK next thing to do is DELETE every AVD you that has ever been created, you do this by going in to the manage Android and then Manage AVD menu option.

      Titanium will recreate AVDs as it needs them.

      When choosing a new Android API I always select those ending with [armeabi-xxx]

      Be aware that 4.2 is not regarded as supported yet.
