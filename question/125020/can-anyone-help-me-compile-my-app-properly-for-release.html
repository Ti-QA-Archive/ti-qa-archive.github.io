    Adam,

    Xcode 4.2 currently can't be used to submit applications to the App Store, since it's still in beta, along with iOS SDK 5.

    To package and submit your app to App Store, you will need to remove Xcode 4.2 and install Xcode 4.1 (current build 4B110) with the iOS SDK 4.3.
