remove titleBar from android application

Solution :

on the root folder of your project create this stucture : 

resources 
plateform
android
   res
     values
    theme.xml

and inside the file theme.xml :

<?xml version="1.0" encoding="utf-8"?>
<resources>
<style name="Theme.Titanium" parent="android:Theme.NoTitleBar">
<item name="android:windowBackground">@drawable/background</item>
</style>
</resources>

NOTICE :

android:Theme.NoTitleBar
android:Theme.Light.NoTitleBar
…

http://developer.android.com/guide/topics/ui/themes.html#PlatformStyles
