How to Zoom in & out a background image of a view on pinch gesture

Hi,
I have a view with a background image as given below:: 

var view5 = Ti.UI.createView({
    backgroundImage:'view5_background.png',
    borderRadius : 10,
    width : 1030,
    height : 600,
    top : 0

});

On pinch gesture by the user i want the view to zoom in & out. Is it possible to develop such a thing in Titanium?? If not, what are the other options i need to consider keeping in mind that pinch event is compulsory?
Thanks.

  • var webview = Ti.UI.createWebView({

    borderRadius : 10,
width : 1030,
height : 600,
top : 0,
backgroundColor:'#00000000',
scalesPageToFit:true,
touchEnabled: true,

    });

    wdt = 100;

    hgt = 200;

    img_pth = 'view5_background.png';

    webview.html = '<html><body> <img src="images/'+img_pth+'" width='+wdt+' height='+hgt+'></body></html>';

    Webview automatically listen pinch gesture and by default provide zoom in and out functionality , for this you don't want to put any efforts . All this is possible in titanium using webview. Accept answer if you find usefull … This things work for me.
